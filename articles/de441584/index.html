PHP Digest Nr. 150 (11. - 25. Februar 2019)


Neue Auswahl mit Links zu Nachrichten und Materialien. In der Ausgabe: Die Regeln für die Abstimmung für RFC in PHP Internals wurden geändert, die Annahme von Anträgen auf Berichte für PHP Russia 2019, eine neue Erweiterung für die Implementierung der parallelen Codeausführung, neue Schulungsmaterialien, Videos, ein Teil nützlicher Tools und vieles mehr wurden gestartet.

Viel Spaß beim Lesen!



Nachrichten und Veröffentlichungen


  • PHP Russia 2019 - Der erste Star-Sprecher, der in Kürze bekannt gegeben wird. In der Zwischenzeit ist die Annahme von Berichtsanträgen in vollem Gange. Bis zum 25. Februar können Konferenztickets zum besten Preis gekauft werden.
  • 16. März Badoo PHP Meetup: Tests und Codequalität. Die Registrierung ist offen

PHP-Interna


  • Der Vorschlag, die 50% -Schwelle für die Abstimmung abzuschaffen, wird angenommen, und jetzt benötigen alle RFCs eine 2/3 Mehrheit.
  • PEAR wird seit PHP 7.4 nicht mehr standardmäßig installiert .
  • Ein ausgezeichneter Überblick über nicht akzeptierte RFCs - Warum Generika, kurze Lambdas, Methodenüberladung, Anmerkungen usw. nie realisiert und was ist die Prognose.
  • Xdebug Derick Rethans Autor spricht über einen kleinen Fehler in PHP <7.3.2

Die Werkzeuge


  • krakjoe / parallel - Eine neue Erweiterung des Autors von pthreads mit einer vereinfachten Schnittstelle für die parallele (Multithread-) Codeausführung. Lesen Sie mehr über die Motive zum Erstellen einer Erweiterung in einem Beitrag .
  • willdurand / Hassoas 3.0 - Eine beliebte Bibliothek und ein Paket zur Implementierung von HATEOAS in REST-Services.
  • liip / serializer - Ein Paket zum ( De- ) Serialisieren von Daten. Kompatibel mit jms / serializer und schneller als letzterer. Lesen Sie mehr über die Leistung in der Post .
  • TBPixel / xml-streamer - Ein Paket zum Streaming von XML-Daten, das mit PSR-7 kompatibel ist.
  • phpstan / phpstan-doctrine - Eine PHPStan-Erweiterung zur Unterstützung von Doctrine ORM, einschließlich DQL-Analyse .
  • Spatie / Enum - Eine Implementierung eines Aufzählungstyps und zur Unterstützung eines Beitrags über Enum in PHP .
  • kriswallsmith / Buzz - Leichter HTTP-Client.
  • thecodingmachine / graphqlite - Ein weiteres GraphQL-Framework für PHP. Einführungsbeitrag mit einer Überprüfung zur Unterstützung.
