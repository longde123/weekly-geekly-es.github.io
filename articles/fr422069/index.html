PHP Digest n ° 138 (20 août - 2 septembre 2018)


Nouvelle sélection avec des liens vers des nouvelles et des documents. Dans la version: la dernière version bêta de PHP 7.3, Laravel Nova, un site avec des idées pour les prochaines versions de PHP, des nouvelles de Yii 3, une partie d'outils utiles, des vidéos et bien plus encore. Bonne lecture!



Nouvelles et communiqués


  • PHP 7.3.0 Beta 3 - Dernière beta de la boucle. La prochaine version sera RC 1, qui est attendue le 13 septembre .
    • Parmi les changements:
      • Amélioration de la syntaxe RFC Heredoc et Nowdoc
      • Les fonctions json_encode () et json_decode () lèveront une exception en cas d'erreur et l'indicateur RFC correspondant
      • Possibilité d'utiliser une virgule après le dernier argument de la fonction RFC
      • Nouvelle fonction hrtime (), qui renvoie un temps de haute précision et ne dépend pas de l'heure système PR
      • Ajout de la fonction is_countable () pour vérifier l'énumération RFC
      • Utilisation de liens dans la liste () du RFC
      • Certaines fonctionnalités dépréciées RFC 1 , RFC 2
      • Ajout de la prise en charge RFC du même site pour les cookies
      • Amélioration significative du ramasse-miettes PR
      et d'autres améliorations. Liste complète des changements .
      Vous pouvez en savoir plus sur les changements dans une analyse détaillée des nouveautés de PHP 7.3 ou consulter rapidement les différences .
  • Exécution de code à distance sur packagist.org - Vulnérabilité qui a permis au code d'être exécuté sur le serveur de référentiel Packagist PHP en passant une chaîne spécialement conçue au lieu d'une URL au formulaire pour ajouter un nouveau package. Déjà réparé .
  • Highload fwdays'18 - Le 15 septembre à Kiev sera une conférence à grande échelle sur la highload. Il n'y a pas de rapports directement sur PHP, mais le programme propose des présentations intéressantes sur la mise à l'échelle, l'architecture et l'optimisation. Code de réduction: PHPDIGEST_HL .
  • php-vote.com - Qu'aimeriez -vous voir dans la prochaine version de PHP? Sur le site, vous pouvez ajouter une idée ou discuter et voter pour celles déjà ajoutées.


Les outils


  • nbs-system / snuffleupagus - Une extension de sécurité pour PHP 7 inspirée de Suhosin . Empêche l'exécution de code potentiellement dangereux lors de l'exécution.
  • adsr / phpspy - Profileur léger pour PHP 7.
  • adhocore / phint - Un outil pour générer rapidement un squelette d'application à partir d'un modèle.
  • phpstan / phpstan - Puissant analyseur statique. Afficher les nouveautés de la v0.10 dans le rapport de l'auteur.
  • unreal4u / telegram-api - Bibliothèque asynchrone en PHP 7 pour travailler avec l'API bot Telegram.
  • nette / utils - Un ensemble de fonctionnalités utiles pour tous les jours. Poster à l'appui.
