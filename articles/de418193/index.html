Wie fühlt es sich an, 2017 ein Spiel für Game Boy zu erstellen?


Jeder hatte einen Kindheitstraum. Persönlich habe ich davon geträumt, ein Spiel für meine erste Konsole zu entwickeln: Nintendo Game Boy. Heute wurde mein Traum wahr - ich habe das erste Spiel für Game Boy auf einer echten Kassette veröffentlicht : Sheep It Up!

In diesem Artikel werde ich über die von mir verwendeten Tools und die Hindernisse sprechen, die ein Anfänger überwinden musste, um ein Projekt zu erstellen. Der Artikel ist sehr lang und daher in zwei Teile gegliedert:

  • Teil 1: Verwendete Werkzeuge / technische Schwierigkeiten / Grafikschwierigkeiten
  • Teil 2: Klangbeschränkungen / Kassettendesign / Player-Bewertungen

Das Spiel


"Schaf es auf!" - Dies ist ein Arcade-Spiel, bei dem die Schafe auf die fliegenden Klettstreifen klettern müssen. Das Konzept ist einfach, aber das Spiel wird schnell ziemlich kompliziert: Wie weit kann man klettern, ohne zu fallen?


Ich selbst bin ein Spielesammler, daher wollte ich nicht, dass Game Games bei der Erstellung meines Projekts unter Vintage-Spielen leiden. Daher wurde alles speziell für das Spiel erstellt: Leiterplatte, ROM, Hülle, Schutzhülle und sogar ein Aufkleber! Wir haben auch versucht, einen angemessenen Preis beizubehalten, damit jeder das Spiel genießen kann: $ 15 (+ Versand) . Es läuft auf jedem Game Boy-Modell, vom ersten bis zum GBA SP, einschließlich des Super Game Boy.


Wenn Sie noch einen Game Boy haben, können Sie eine Kassette auf der Website des Herausgebers kaufen:

Klangbeschränkungen


Wie wir aus dem ersten Teil des Artikels verstanden haben , weist Game Boy offensichtliche Einschränkungen hinsichtlich CPU-Leistung und Grafik auf. Das Gleiche gilt für den Klang. Und im Gegensatz zu modernen Maschinen kann Game Boy keine MP3-Musik abspielen oder WAV-Audiodateien lesen. Um Sounds und Musik für den Game Boy zu erstellen, müssen Sie diese mithilfe der vier in der Konsole integrierten „Soundkanäle“ dynamisch generieren. Jeder Kanal hat seine eigene Funktion:

  • Kanal 1 - Tone & Sweep: Wir können Sound (Ton) mit zunehmender oder abnehmender Tonhöhe abspielen.
  • Kanal 2 - Ton: Gibt einen Ton ohne Spezialeffekt wieder.
  • Kanal 3 - Wellenform: Kann sehr einfache Wellenformen abspielen (in 4 Bit codiert).
  • Kanal 4 - Lärm: Kann Explosions- oder Motorgeräusche erzeugen.

Wiederholen wir es noch einmal: Um Sound in Game Boy abzuspielen, müssen Sie ihn manuell generieren und diese vier Soundkanäle direkt steuern . Glücklicherweise ist dies ganz einfach: Ändern Sie einfach die Werte mehrerer Hardwareregister. Jeder Soundkanal wird von 4-5 Hardware-Registern (8-Bit-Variablen) gesteuert, die Lautstärke, Tonhöhe, Spezialeffekte usw. bestimmen. Durch Einstellen unterschiedlicher Werte für die Register, die den Kanal steuern, können wir Game Boy dazu bringen, eine breite Palette von Sounds abzuspielen. Hier sind zum Beispiel drei Sounds von Sheep It Up! das ich mit Soundkanal 1 und den entsprechenden Werten der Hardware-Register erstellt habe:

Wie Sie sehen können, können wir durch Zuweisen von 5 verschiedenen Werten zu Variablen sehr unterschiedliche Soundeffekte erzeugen. Sie könnten denken, dass Sie zum Erstellen von Sounds Versuch und Irrtum durchlaufen müssen: Fügen Sie zufällige Werte in die Register ein, bis Sie einen schönen Sound erhalten. Aber das wäre langweilig und sinnlos. Daher können moderne Werkzeuge unser Leben erleichtern . Lassen Sie mich Ihnen den GB Sound Sample Generator vorstellen.


Zuerst stellen wir bequeme Parameter ein, um den richtigen Sound zu finden (drücken Sie START, um den aktuellen Sound zu überprüfen).

Dies ist ein Programm für Game Boy, das vollständig mit echter Hardware kompatibel ist. Sie können die Werte der Hardware-Soundregister auf eine für den Benutzer bequeme Weise ändern. Anstatt rohe NR10_REG-Werte festzulegen, können wir die Schallfrequenz, den Lautstärkepegel, die Dauer des Sweep-Signals usw. angeben. Sie können dieses Tool als SFXR für Game Boy betrachten. Um den Sound zu exportieren, zeigt der GB Sound Sample-Generator die entsprechenden Hardwareregisterwerte an, anstatt die WAV-Datei zu speichern. Um den Sound in Ihrem Spiel wiederzugeben, geben Sie diese Werte einfach in den Quellcode ein.


Dann müssen Sie A + SELECT gedrückt halten, um die entsprechenden Rohwerte der Hardwareregister anzuzeigen

Um an Sheep It Up zu arbeiten! Ich habe den GB Sound Sample Generator für Game Boy Advance SP verwendet (ein hintergrundbeleuchteter Bildschirm ist praktischer) . Das ROM-Programm wurde auf Everdrive GB gestartet. Ich habe ziemlich viel Zeit damit verbracht, die Parameter direkt auf der Konsole einzustellen, und als ich endlich die richtige Kombination gefunden habe, habe ich einfach die Werte vom GB-Bildschirm in den Quellcode auf dem Computer kopiert.

Chiptune Musik!


Trotz der begrenzten Klangfähigkeiten wird Game Boy heute als Musikinstrument verwendet. Viele talentierte Komponisten der Chiptune-Szene erstellen mit Programmen wie LSDJ oder Nanoloop energiegeladene Tracks auf Game Boy. Zum Beispiel haben Sie vielleicht von der irischen Legende Chip Chip Tun Chipzel gehört , der die Musik für Super Hexagon geschrieben hat , und von dem französischen Zauberer Sidabitball , der viele Hits der Popmusik der 90er Jahre remixt hat.


Chipzel und Sidabitball

Unglücklicherweise für uns Spieleentwickler verbraucht das LSDJ- Tool, mit dem diese großartigen Songs komponiert wurden, fast 100% der Hardwareressourcen von Game Boy. Daher können Sie wunderbare Musik darin schreiben, aber Sie können das Spiel nicht starten, während diese Musik auf der Konsole abgespielt wird. Glücklicherweise gibt es weniger ressourcenintensive Alternativen zum Erstellen und Abspielen von Musik auf Game Boy.

1) Eigener Musikplayer


Die minimalistischste Lösung besteht darin, einen eigenen „Musik-Player“ zu erstellen.
Vergessen Sie nicht, dass die einzige Möglichkeit, Sounds auf Game Boy abzuspielen, darin besteht, die Werte der Hardware-Register festzulegen, die die vier Kanäle der Klangerzeugung steuern. Daher ist es ganz einfach, einen eigenen "Musik-Player" zu erstellen und die "Noten" in einem Array zu speichern. Jede "Note" ist eine bestimmte Kombination von Werten für jedes der Hardwareregister. Wenn Sie das Tempo festlegen, mit dem wir dieses Array durchlaufen, um „Noten“ abzuspielen, erhalten wir eine sehr einfache Möglichkeit, Musik abzuspielen. Genau das habe ich in Sheep It Up getan! Hier ist ein kurzes Gameplay-Video mit Sound und Musik, damit Sie das Ergebnis hören können:


Tatsächlich werden meine „Noten“ nur auf Soundkanal 2 abgespielt. Ja, die vier Soundkanäle sind für Soundeffekte und Musik gleich. Wenn wir also eine Note in einem Kanal spielen, kann sie nicht gleichzeitig zum Abspielen eines Soundeffekts verwendet werden . Und da jeder Kanal etwas „Besonderes“ ist, ist es manchmal vorzuziehen, einige Kanäle nur für Musik zu verwenden, während andere für Soundeffekte übrig bleiben. Zum Beispiel wird auf Kanal 4 ein „Knarren“ abgespielt, das zu hören ist, wenn ein Schaf an einem fliegenden Klettverschluss festhält (Geräusch). Da dies ein sehr häufiger Sound in meinem Spiel ist, habe ich beschlossen, diesen Kanal nicht für Musik zu verwenden. Gleiches gilt für Kanal 1, den ich für alle anderen Soundeffekte verwendet habe. Da ich kein großer Musikspezialist bin, habe ich mich entschieden, die Wellen von Kanal 3 nicht zu verwenden und stattdessen mit den Tönen von Kanal 2 zu arbeiten.

2) GBT-Spieler und Carillon-Spieler


Viele Homebrew-Spiele verwenden ihren eigenen Musik-Player. Dies ist jedoch nicht die einzige Lösung. Es gibt eine Bibliothek namens GBT Player , die Musikmodule (im Impulse Tracker ".it" -Format) für die Wiedergabe auf Game Boy konvertieren kann. Natürlich gibt es viele Einschränkungen hinsichtlich der Größe der Samples, der Anzahl der Kanäle und der Noteneffekte. In der Bibliothek können Sie jedoch echte Musik auf einem echten Musik-Tracker, z. B. OpenMPT , komponieren und dann auf Game Boy abspielen. Am wichtigsten ist, dass die Bibliothek nur 7-8% der Game Boy-CPU-Ressourcen verbraucht und eine Menge Rechenleistung für die Ausführung des Spiels übrig bleibt. Darüber hinaus ist es sehr einfach, es in Ihren Code zu integrieren, wenn Sie ein Spiel in C mit GBDK oder in Assembler mit RGBDS schreiben möchten. Eine gute Alternative zum GBT-Player ist der Carillon-Player . Sie müssen jedoch den Carillon-Editor verwenden, um Musik zu komponieren.


Geometrix ist ein Homebrew-Spiel des GBT-Spielers AntonioND. Für Musik wird diese Bibliothek verwendet.

Fit werden: Ein Spiel auf Kassetten aufnehmen


Nach langer Arbeit an Code, Grafik und Sound ist das Spiel endlich fertig. Zu diesem Zeitpunkt haben wir jedoch nur die ROM-Datei. Natürlich kann ich viele verschiedene Gameboy-Modelle aus meiner Sammlung aktiv testen (ich habe sie alle außer dem AGS-101) . Ich hatte jedoch nicht das Gefühl, dass das Projekt fertig war: Meiner Meinung nach sollte das Spiel eine eigene Kassette haben, um als „abgeschlossen“ zu gelten .

Wie ich im ersten Teil des Artikels sagte, ist das gesamte ROM des Spiels Sheep It Up! Es wiegt nur 32K, wie bei früheren Game Boy-Spielen wie Tetris und Alleyway . Tatsächlich sind die meisten Spiele für den Game Boy viel größer, von 128 KB bis 512 KB, und das größte Pokemon Blue and Red- Spiel benötigt monströse 1024 KB (1 MB). In den 90er Jahren stellte sich heraus, dass die Kassetten umso teurer waren, je größer das Spiel war: Um große Spiele zu speichern, wurden teurere Chips benötigt. Aber heute stehen sie alle fast gleich. Warum bin ich auf ein 32-Kilobyte-Spiel beschränkt?

Die Antwort ist, dass ich das Spiel auf den heute hergestellten Kassetten veröffentlichen wollte. Wie Sie sich vorstellen können, hat Nintendo die Produktion von Kassetten für Game Boy (Game Pak-s) längst eingestellt. Um heute Ihre eigene Patrone zu erstellen, gibt es zwei Lösungen:

1) Modifikation von Patronen (Cartmodding)


Die erste Option, "Cartmodding" genannt, besteht darin, alte Game Boy-Spiele zu recyceln. Spiele, die während der kommerziellen Ära von Game Boy veröffentlicht wurden, wurden auf nicht wiederbeschreibbaren Kassetten gespeichert. Um das darauf gespeicherte Spiel zu ersetzen, müssen Sie sie öffnen, den Speicherchip entfernen und durch einen anderen Speicherchip ersetzen, auf den das ROM Ihres Spiels geladen ist. Mit dieser Lösung können Sie Spiele jeder Größe erstellen. Aber als Spielesammler wollte ich vorhandene Game Boy-Kassetten für die Produktion meines Spiels nicht zerstören.


Ein Beispiel für Kartenmodding. Ein neuer Speicherchip mit einem Homebrew-Spiel wird in eine alte Kassette eingelötet

2) Eigene Patrone


Die zweite Möglichkeit besteht darin, eine neue Patrone von Grund auf neu zu erstellen. Da ich kein Elektronikexperte bin, begann ich im Internet nach Leuten zu suchen, die Game Boy-Kassetten zusammenbauen konnten. Nach mehreren erfolglosen Versuchen hatte ich das Glück, Catskull zu finden. Dies ist ein erstaunlicher Elektronik-Assistent, der in seinem Catskull Electronics- Geschäft eine Menge Game Boy-Geräte herstellt und verkauft. Unter anderem entwickelte er ein Gerät zur Verbindung von Gameboy mit der MIDI-Schnittstelle, mit dem Musiker den Game Boy mit echten Instrumenten und Synthesizern verwenden können. Er entwickelte auch seine Game Boy-Patrone. Für diese Patrone wurde alles neu gemacht : die Leiterplatte, Chips, Schalen, sogar Aufkleber und neue Teile - nichts wurde aus früheren Spielen übernommen. Es war die perfekte Lösung für mich.


Catskull hat eine Originalplatine entworfen, die ich für Sheep It Up!

Aber dann wurde alles noch besser: Nach einer kurzen Korrespondenz sagte mir Catskull, dass ich lange Zeit eine Firma für die Veröffentlichung von Homebrew-Spielen gründen wollte und dass mein Spiel eine großartige Gelegenheit dafür wäre! Deshalb haben wir uns zusammengetan: Ich mache Software, er beschäftigt sich mit Hardware . Die einzige Einschränkung bei dieser Lösung besteht darin, dass nur 32 KB auf einer Catskull Game Boy-Kassette gespeichert werden können . Warum?

Ursprünglich sollte die Game Boy-Konsole nur auf 32 KB ROM-Daten zugreifen können. Um Spiele mit mehr als 32 KB zu erstellen, müssen Sie durch alle Daten auf der Kassette navigieren und gleichzeitig auf nur einen Unterabschnitt aller ROM-Daten zugreifen. Diese Methode wird als Bankwechsel bezeichnet : Alle ROM-Daten sind in Banken mit jeweils 16 KB unterteilt, und wir können jederzeit (mit bestimmten Einschränkungen) zwischen ihnen wechseln. Für die Bankumschaltung muss die Kassette jedoch über einen zusätzlichen Chip verfügen : „Memory Bank Controller“ (MBC). Es gibt mehrere Chips (MBC1, MBC5 usw.) , die das ROM vergrößern und es Ihnen ermöglichen, RAM zum Speichern von Spielen mit Akku usw. zu verwenden. Leider gibt es derzeit keine vorgefertigte Lösung, um neue Patronen mit einem solchen Chip von Grund auf neu zu erstellen.

Daher verwenden Leute, die Homebrew-Spiele mit mehr als 32 KB erstellen, normalerweise Kartenmodding, um den MBC-Chip der alten Game Boy-Kassette zu verwenden, den sie „recyceln“. Eine andere Lösung besteht darin, gefälschte Kopien von echten Game Boy-Spielen zu kaufen, gefolgt von einem Blinken. Vor vielen Jahren haben chinesische Hersteller gelernt, einen MBC-Chip zu „klonen“, um Raubkopien großer Spiele wie Pokemon zu erstellen. Und da solche Raubkopien-Kassetten EEPROM-Chips anstelle von ROM-Chips alter Spiele verwenden, können sie normalerweise "neu geflasht" werden, um das auf der Kassette gespeicherte Spiel zu ersetzen. Verwenden Sie dazu einen Kassetten-Flash-Recorder wie BennVenns Joey . Mehrere Homebrew-Spiele, die größer als 32 KB sind, haben diesen Ansatz verwendet.


Eine Piratenpatrone hergestellt in China. Sie können es erneut starten, indem Sie Ihr Spiel darauf schreiben, aber einige andere können nicht erneut gestartet werden (mehr dazu hier ).

Um Sheep It Up zu machen ! Ich wollte völlig neue Kassetten erstellen, um die ursprünglichen Game Boy-Spiele nicht zu zerstören. Ich hatte großes Glück, dass ich mich mit Catskull zusammengetan habe, das seine eigenen Kassetten herstellt, obwohl ich mich auf 32 KB Shared Memory beschränken musste.

Patronen backen!


Wie stellen Sie Ihre eigenen Patronen her? Ich bat Catskull, Fotos von den einzelnen Schritten der Patronenmontage zu machen. Wie Sie sehen können, wird jede Patrone manuell zusammengebaut:

1) Die Leiterplatte wird in die Hardware eingesetzt.


2) Zum Auftragen von Lötpaste wird eine Edelstahlschablone auf die Leiterplatte aufgebracht. Mit dieser Paste können Sie den Speicherchip auf die Platine löten.


3) Normalerweise mache ich Chargen von 18-20 Patronen gleichzeitig. Auf diese Charge wurde bereits Lötpaste aufgetragen.


4) Ich weiß, wie man Kondensatoren und Widerstände manuell auswählt und auf die Platinen legt. Dann habe ich Flash-Chips auf jedes Board gelegt.



5) Die Bretter werden auf dem Gitter des Reflow-Ofens ausgelegt. Wir "backen" buchstäblich jede Platine, um einen Chip darauf zu löten. Jetzt kann ich jedes der Bretter nehmen und das ROM des Spiels auf den Speicherchip schreiben.



6) Anschließend wird das Board mit einem schönen Aufkleber auf einem professionellen Drucker in die Hülle der Spielekassette eingelegt.



7) Nach einer gründlichen Überprüfung der Patrone auf der realen Ausrüstung kann das Spiel an den Spieler gesendet werden, der es bestellt hat!
