学校：一首关于主要事物的老歌


分组


平均而言，一个国家的学校是教育中非常有趣的元素，尤其是现在。 在其中完全不同的世界越过：

  1. 年龄较大的老派教师大多不准备接受新的现实和教育形式，也不愿意听学生的话；
  2. 来自90年代的年轻且漠不关心的老师，由于绝望和无法入读另一所大学（由于准备水平或缺乏资金）而极少例外地去教学时；
  3. 父母的年龄范围从70年代到90年代，即从苏联人民到所谓的“迷路的一代”的疯狂代表；
  4. 15-17岁的孩子（大多数情况下，我们会谈论它们）-数字时代的孩子，自动化，计算机化，内向和虚拟，具有自己的思维以及对心理和记忆的特殊组织。

所有这四个群体之间和群体之间都在与其他群体进行斗争，在这样的社区中，存在很多误解以及主要的权威教育者互联网的无形之手。 你知道我会告诉你什么吗？ 这非常好，您只需要一种特殊的方法即可。 我要说的是，世代之间的冲突是永恒的，就像学童的懒惰一样，只有风景在改变。

小学生有什么问题？


  • 知识与实践完全脱节。 学校课程不提供与实践相关的信息。 这就是为什么人们会遇到有关程序员是否需要数学或选择哪种编程语言来规避数学问题的问题。 在同一个代数上，您可以接触到神经网络，机器学习和游戏开发的问题（估计发现您最喜欢的游戏世界的英雄根据物理定律运动是多么酷，并且每个轨迹都由一个数学公式描述）。 理论和实践的融合可以提高学生的兴趣，克服课程中的无聊感，同时有助于初级职业指导（6-9年级）。 同时，不需要昂贵的材料基础，足够的需求，一块板和一个粉笔/马克笔就足够了。
  • 真正的知识水平与日记和证书中的评估不符。 充斥，鼓励和挫败评估的永恒问题，竞争导致学生追求珍贵的身材，而父母和老师则鼓励这场比赛。 毫不奇怪，在高中的第一年，优秀的学生在高等数学中滑落到三级，而三级的成绩很强，四级-他们对科目有所了解，而不是考试后就出现的锯齿状部分。
  • 实际上， 免费获取信息是一个大问题。 无需记忆，搜索和分析-只需打开Wikipedia或google即可，仅此而已，信息就在您眼前。 这很不好，因为记忆功能确实下降了，所以没有形成正确的教育基础。 教您抓住问题，找到缺失的难题然后使用目录或Internet的基础。 简而言之，在不断使用Google的情况下，学生不会学会了解Google的必要条件。 同时，它是未来职业基础的主要教育基础，是分析和综合技能的平台。
  • 在学校里没有多余的知识 。 也许，阅读这篇文章的老师现在想找到作者并把它撕成碎片，但是学校越凉爽，更多，对不起，垃圾塞满了课程。 从我遇到的游戏来看：拉丁语的4年，外国文学的7年（不断加深），4年（！）的生活史，2年的哲学以及各种文学，希腊语，体育文化理论，数学史等。 当然，一般的博学，学校锦标赛上的“什么？ 在哪 何时？”，保持对话的能力是无价的，甚至是非常愉快和有用的，但是在如此大量的课程中，上课的时间使学生的大脑从与他相关的主题以及最重要的教育部分（您看的是现代拼写，但至少在同一个哈伯瑞上！） 。 有一个解决方案：使此类科目为选修科目而无需评分。
  • 自从学校开始存在以来， 接受教育的艰难步伐就一直存在，这是一个很难找到的解决方案。 在一个班级中，即使是“强者”或“弱者”，学生对材料的掌握率，解决问题的速度和“摇摆”的速度也不同。 最后，您要么必须去练级并失去潜在的强者，要么得分较弱，并使他们变得更弱。 我有一个学生能很好地解决数学统计学中的问题，但是做得很慢，因为 他一直在寻找最佳解决方案，优化了解决方案。 结果，在五个任务中，我设法解决了三个。 你命令他放什么？ 是一样的 同时，您可以找到一个较小的解决方法：给有能力的人更多的任务来独立解决，让他们有权在老师的监督下进行辅导和培训同学-这显着增加了责任感，减少了对错误的恐惧，并使学生能够展示团队合作的基础。
  • 社会化问题是一个痛苦而严重的问题，拖累了许多其他问题。 虚拟交流环境，游戏互动，社交网络和即时通讯程序使儿童（是的，18岁以下的儿童，儿童以及可惜的是儿童）失去了交流和社交互动的能力。 没有解决问题的能力，没有团队合作，没有一群人之间的关系，没什么-对等社交网络，简单的对话。 在这里，学校的任务是展示“人对人”系统的酷感：组织团队游戏，组织交流。

如何选择专业？


到目前为止，在俄罗斯的大多数学校中（莫斯科的情况更好），针对学童的职业指导只限于关于未来职业的论文，而职业指导测试还不够充分，其中一些测试可以大致确定学生对某个领域或其他领域的倾向。 但是，没有讨论诸如生物信息学，医学计算机科学等专业。 -也就是说，多才多艺和高级儿童的热门和有前途的地区。 学生本身主要还是儿童，浪漫主义者和梦想家。 今天，他们想待人或在紧急部任职，明天要成为企业家，一个星期后，他们就是要建造未来汽车的程序员或工程师。 重要的是要倾听，思考选择原因的原因是豪斯医生的魅力，埃隆·马斯克的魅力或真正需要和年轻人的呼唤。

如何评价一个专业？


透视也许是最困难的指标。 在高中和大学毕业之前，现在看来有希望的东西可能会变成最过热的领域（在2000-2002年间招呼律师和经济学家，招生！），或者完全消失。 因此，您需要向您的孩子说清楚，并意识到自己应该有一个基础，可以反复更改您的专业。 例如，拥有C / C ++的软件工程师可以轻松进入神经网络开发，工业发展，科学等领域，但是作家（应用信息学）可能会在五年内被淘汰出局。研究。 再次，与银行或房地产评估相比，在财务管理方面具有专长的经济学家更有希望 。 为了评估未来，研究未来的职业清单，查看编程语言的等级（如果是关于IT的），阅读专业出版物（例如15-17年前，在医学期刊上），科学界积极讨论了眼显微外科手术，医学界的机器人，腹腔镜操作以及今天，这已成为日常现实）。 另一种方法是查看过去2-3年中哪些大学开设了大学，通常，这是您可以进入的最高职位。

实际收益是一个更简单的指标。 打开“我的圈子”或“猎头公司”，按专业评估平均收入水平（有时也提供现成的分析）。 商业领域的薪资指数每年高达10％，而公共部门的指数每年高达5％。 这很容易计算，但是请不要忘记，在N年内，需求深度将有所调整，领域的格局也会发生变化等。

每个领域的发展速度和职业发展速度都不同。 而且，它并不处处，也不值得浪漫：有时最好是横向移动，学习新的专业并且不在工作簿上工作，而是在实际收入水平上工作（这是充满希望的，但在下一个系列中会更多）。 最主要的是要向学生传达他不会立即成为负责人，他将需要工作，而真正的专业人士有时会比他的头部昂贵。

增长和专业发展是先前指标的重要延续。 专业人士不断学习，直到工作的最后一天（有时甚至是之后）。 因此，您需要将学生的学习倾向与所需职业的要求联系起来（ 例如，男孩梦想成为一名医生，化学和生物学领域有5位，但是懒惰的学习表明他将来可能会遇到专业发展方面的问题 ） ，但请不要挂在嘴上：通常在大学学习后，成年人很乐意学习并继续接受教育，在学校里没有懒惰，但讨厌繁琐的故事和无聊的地理女孩。

要考虑什么？


选择职业，您应该帮助您的孩子，但不要为他决定（我保证您不会得到“谢谢”）。 同时，重要的是不要错过任何一个细节，也许甚至要严格客观地从外部观察亲密的亲爱的人（相对而言，在Lambada舞姿下扭转赃物的能力尚不是B级舞厅舞，但无论您想要多少， ）

  • 儿童的总体倾向与我们上面讨论的职业指导基础相同：“人”，“自然”，“机器”，“信息系统”。 没有人没有倾向，没有希望的载体，因此重要的是要认识到哪种机制占上风。 即使是通才，也可能在一个方向或另一个方向上发生某些变化。 请注意以下事实：学生说出什么科目以及为什么他会更容易，在对话中会引起他的注意，他是否具有算法思维，如何发展逻辑或想象力。 而且，这种对非自愿反应的观察比测试要准确得多，因为一个13-17岁的学生可以轻松地猜测如何回答，以便获得当时想要的结果并欺骗系统和成年人：-)
  • 必须考虑并鼓励学生的愿望 ，甚至有机会怀着职业梦想“生病”-这样才能更快地确定职业。 绝对不要让他放弃自己的选择，不要以负面的眼光来揭露这个职业（ “所有程序员都是书呆子，”“一个女孩在汽车部门中没有位置”，“哈哈，心理学，你自己是一个心理医生，你会请离婚之类的吗？” “是出租车司机？是的，他们会杀了你”-基于真实事件 ）。 如果可能，请给孩子尝试专业知识，或至少尝试其中的一部分：尝试在夏季做一份兼职工作，寻求与该专业有关的帮助，请朋友雇用几天。 如果有这样的机会，它将毫无瑕疵地发挥作用：要么降温和失望，要么充满热情并批准未来的计划。
  • 家庭特征不能排除在我们的困境之外：如果整个土木工程师和女儿的家庭将童年时期的水泥品牌与众不同，知道加固的厚度，区分砌体的类型，并且在7岁时就可以解释暖气的工作原理……这并不意味着建筑物正在等待它，不，但是您不应该等待与Akhmatova和Petrarch的早期作品相恋，那不是她的环境。 虽然有例外。 但是，裙带关系不应对学生施加压力，迫使他成为某人，因为他的父母就是这样。 是的，您的好处显而易见：教育，帮助，找到工作等更加容易。 但是好处是您的，孩子的生活，以及朝代的选择可能不适合他。

碰巧的是，父母确定自己的孩子什么都不想要，没有志向和爱好，不寻求选择大学，不考虑未来。 实际上，这不会发生，总会有您喜欢的东西-您需要以此为基础。 如果您认为确实有困难，请与老师交谈，听取他们的建议，然后咨询社会心理学家，他为青少年提供职业指导（有很多很酷的私人商人，以下是他们的更多内容）。 我同学的女儿是15岁，是个很小的孩子，我的母亲是一位没有受过教育的惰性家庭主妇，她把女儿看成“一无所有”。 这个女孩端上一杯美味的冲泡咖啡，优雅地弯下餐巾，递给她自己做的蚁丘蛋糕。 “卡蒂亚，你认为她应该尝试糕点师还是在咖啡馆工作？” -车，她不是全民服务的人，我会让她去找会计师。 窗帘。


学生应该了解什么专业？


当您是一名男生时，您总是试图隐藏自己行为或选择的真正动机，以免显得不成熟或缺乏动力。 因此，父母很难弄清对特定职业的渴望来自何处，尤其是在突然的情况下。 这是不值得的，最好传达一些游戏规则。

  • 任何工作都包括例行程序的一小部分（最多占整个工作的100％）-学生必须理解，他将获得一些例行任务或视觉属性，他们将收到许多例行任务，这些任务的执行可以构成大部分工作：程序员不编写完整的程序（ （如果他不是企业主或自由职业者），但在其代码部分工作； 即使他是救护车或外科医生，医生也必须填写大量文件； 宇航员训练了很长时间，研究了很多东西，不得不完成太空中的大量任务等。 您需要了解，没有这样的细节就没有专业，这不值得浪漫化。
  • 工作是专家的日常工作。 如果您将生活与职业联系起来，那么可能性很大，它将永远存在：每天，短暂的假期，老板，星期一，复杂的下属等等。
  • 该行业的时尚和声望可能会发生变化-在他毕业之前。 然后将有两种方法：更改资格或成为该行业的佼佼者，以保证劳动力市场的需求。
  • 您无法将对人的态度转移到对整个活动领域的态度-如果您喜欢该职业，因为它是由父亲/叔叔/兄弟/电影英雄所有的，那么这并不意味着您会感到自己像以前一样自在。 每个人都应该选择自己喜欢的东西和为自己准备的东西。 例子可能是，偶像不应该。
  • 必须喜欢作品，必须喜欢它的组成部分。 每项工作分为几个部分：主要活动及其目标，同事，工作环境，基础结构，工作的“客户”，外部环境及其与活动的关系。 您不能接受一件事而拒绝其他任何事情，否认存在外部因素。 为了使工作顺利并获得满意，重要的是要在所有这些组件中找到积极的方面，并关闭警报，然后知道为什么现在将其关闭（为此，除了钱以外）。
  • 漫长的道路始于一连串的小步骤-您不能立即成为伟大和成名，经验丰富和领导。 会有错误，责备，导师和竞争对手，第一步似乎不起眼，微不足道。 但实际上，每一个这样的步骤背后都会有一个突破-经验的基础。 出于微不足道的原因，不必担心下班或匆忙上班：石头在原地生长，道路将被行走的人压倒。



  • 职业生涯的开始几乎总是很无聊-没有人会委托新手来完成复杂而有趣的任务，您将不得不从外围，从基础到学习，掌握，每天重复一些非常无聊的事情。 但是正是由于这些东西的精通，这位年轻的专家才能够将自己沉浸在该行业的深厚基础中。 这种无聊是不可避免的，因此您将需要学习如何在其中找到粉丝。
  • 管理资金也是可行的。 我们的父母没有通知我们这个论文，我们离它还有一段距离。 重要的是不仅要赚钱甚至存钱，而且要能够理财并能够在这段时间里维持自己的生活水平，这一点很重要。 这是一项宝贵的技能，除其他外，它教会您尊重专业的自我和技能，不花一分钱，但也要充分说出您的价格。

结果是这样一个有点哲学的部分，但这恰恰是学生对父母的职业指导的支持，这是他作为未来专家的自尊心的最初基础。

什么，谁将提供帮助？


职业指导是决定生活的过程，因此您需要依靠第三方方法和专业人员的帮助。

  • 职业指导方面的私人专家是一个真正能够找到孩子最隐藏的愿望和能力的人。 通常，这些人不仅是社会心理学家，而且还是人力资源专家，数百名求职者都通过这些专家，他们可以清醒地评估您的孩子为什么做好准备以及应该依靠什么视野。
