Buka data. Roscosmos. Mari kita lakukan seperti di NASA

Kita semua tahu benar bahwa semakin banyak informasi, semakin sedikit rumor.
Data terbuka adalah negara terbuka.

NASA memiliki portal data terbuka open.nasa.gov/open-data , yang berarti ... dan mereka memahaminya.

Portal ini berisi kumpulan data reguler, yaitu file terpisah (Kumpulan data), yang menawarkan format output pilihan dari CSV, PDF, XML, JSON, dan dalam bentuk perpustakaan yang diakses menggunakan API (format output) JSON).

Dari set yang disajikan dalam bentuk biasa, lebih detail tentang keduanya.

EVA Kegiatan Ekstra-Kendaraan (AS dan Rusia) , bekerja di luar kapal.

Set ini memberikan informasi lengkap tentang spacewalks (USSR / USA / Russia), dimulai dengan yang pertama dibuat oleh Alexei Leonov.



Set berisi informasi dalam bentuk:
Yang keluar - Tanggal - Durasi keluar (Waktu) - Target - Negara - Pesawat Luar Angkasa.

Sangat mudah untuk mendapatkan sampel statistik dari set. Pekerjaan selanjutnya dengan data terbuka adalah persis seperti yang dimaksudkan untuk itu. Karena setiap pengguna data terbuka, tanpa membuat perjanjian dengan pemiliknya, dapat menggunakan (termasuk berulang kali) data terbuka secara bebas, bebas, tanpa batas waktu, gratis dan tanpa membatasi wilayah penggunaan, termasuk hak untuk menyalin, menerbitkan, mendistribusikan data terbuka, memodifikasi data terbuka data dan menggabungkannya dengan informasi lain, gunakan data terbuka untuk tujuan non-komersial dan komersial, gunakan untuk membuat perangkat lunak dan aplikasi.



Misalnya, dalam grafik ini kita melihat semua output berdasarkan tahun, dan di bawahnya kontribusi satu negara (US kuning, CCCP biru)

Dengan demikian, Anda dapat melihat informasi tentang negara, penerbangan, dan astronot atau astronot tertentu.



Faktanya, NASA telah menyiapkan set data ini untuk portal Roskosmos.
Anda hanya perlu memeriksanya (dan mungkin mengembangkannya pada saat bersamaan), dan menerjemahkannya ke dalam bahasa Rusia.
Anda bisa mulai ...
