Pemrograman


  • Kembali pada tahun 2017, peserta yaugenka diminta untuk menambah kemampuan iD untuk "memutuskan" seluruh rentang penggunaan lahan dari garis. Quincy Morgan baru-baru ini mengimplementasikan fitur ini dan membuat gif singkat tentang cara kerjanya.
  • Fitur hebat untuk melihat geometri ditambahkan ke program PgAdmin4, dan OpenStreetMap bertindak sebagai latar belakang. Saat menguji skrip PostGIS, programmer cukup beralih antara melihat data dan melihat geometri, sekarang mereka tidak membutuhkan alat visualisasi lainnya.
  • Anggota jguthrie100 menyarankan permintaan tarik untuk editor web iD untuk memperingatkan jika alamat yang sama digunakan beberapa kali.
