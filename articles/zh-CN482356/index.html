从常规Windows用户的角度来看，在没有资源管理器的情况下使用Windows Server

在没有资源管理器的情况下，在Windows Server下向所有人“生存”致以问候


今天，我将测试用于非常规Windows的常规程序。

我将从头开始


当您打开计算机时，将显示标准的Windows启动，但是启动后并没有打开桌面，而是打开了命令行。



从命令行通过Internet下载文件


由于我不知道向普通用户全新安装后下载文件的其他方法（除了将另一个硬盘驱动器与其他系统连接之外），我使用了内置的bitsadmin.exe实用程序

首先，创建一个文件夹，我们将在其中上传文件 

md c:\download

然后，我将所需的文件上传到我的网站。

例如，安装标准浏览器的类似物-Explorer ++ 

 bitsadmin.exe /transfer "Download" https://_/files/Explorer++.exe C:\download\Explorer++.exe

下载过程：



下载完成：



回顾MS-DOS的时代，我们在命令行中编写Explorer++.exe 。
我下载的资源管理器++打开



我还尝试通过将其从系统中拉出来启动普通指挥，但尽管我非常要求，但它拒绝工作。 通过Powershell，您不仅可以安装标准资源管理器，还可以安装MMC，Eventvwr，PerfMon，Resmon和Powershell ISE，但这将消耗资源，并且不适合本出版物的主题。



软件测试


首先，停止使用命令行下载文件，将其替换为Internet Download Manager。

作为文件管理器，我使用资源管理器++，您可以下载其他任何文件，而这只是我的手。

我安装了压缩程序7zip，它也可以正常工作。
我没有安装Firefox浏览器，所以我选择了Chromium Portable。 您将无法通过浏览器下载文件，因此我们将继续使用Internet Download Manager。
VLC媒体播放器正常启动
qBittorrent在启动时也没有引起问题
雷鸟给出与Firefox相同的错误
我稍后再讨论办公软件。

游戏


不幸的是，通过Steam进行的现代游戏无法正常工作。 在启动时，客户端崩溃并显示错误。



办公软件


Microsoft Office不适合资源消耗。 Libre办公室没有启动，这非常可悲。

Web版本的Microsoft Office通过浏览器工作，因此您可以轻松地编辑文档，但是不能将它们保存到本地计算机上。 我尚未测试过Google文档，但我认为它也可以工作。

决赛


在没有资源管理器的情况下使用Windows Server作为“桌面”没有任何意义。 如果资源用尽，最好选择一种轻量级Linux发行版。

在此出版物中，我展示了一个测试示例，它不是培训手册，而是娱乐内容。 感谢您的阅读。
