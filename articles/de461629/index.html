0. Schreiben eines Drahtgitter-Lebenslaufs


Um einen Lebenslauf zu schreiben, müssen Sie einen Lebenslauf schreiben :-) Dies sind die ersten Entwürfe Ihres Lebenslaufs. Geben Sie zunächst alle Ihre Arbeitsorte an, welche Position sie besetzt haben, welche Aufgaben sie ausgeführt haben. Beschreiben Sie ALLE Aktionen, die an jedem Arbeitsplatz ausgeführt wurden, auch wenn dies für Sie unbedeutend erscheint.
Wenn Sie sich für eine Stelle bewerben, sollten Sie immer wichtige Ereignisse in Ihrer Arbeit aufzeichnen. Seitdem können Sie sie vergessen und im Lebenslauf werden sie für einen regnerischen Tag aufbewahrt.
Wenn Sie an etwas teilgenommen haben - groß oder klein -, nehmen Sie es im Lebenslauf auf. Dies kann die Ausführung einer Aktion (Schreiben eines Plug-Ins zum Versenden von E-Mails, Organisieren einer Aktionärsversammlung, Einrichten eines Mailservers usw.) oder das Bestehen einer Schulung oder die Teilnahme an einem Seminar sein. Wenn Sie vor oder während der Arbeit zusätzliche Schulungen absolviert haben, nehmen Sie diese unbedingt in den Lebenslauf auf.

Geben Sie Ihre persönlichen Erfahrungen mit jeder Technologie oder jedem Programm an. Oder ein Buch über Zeitmanagement, das Sie auswendig kennen (letzteres wird am besten im Abschnitt "Info" angegeben).
Wenn Sie als Designer arbeiten möchten und gleichzeitig Ihr Kunstwerk fertiggestellt haben, geben Sie dies unbedingt an! Für kreative Berufe empfehle ich außerdem, ein Portfolio aller meiner Werke zu füllen, absolut alle, einschließlich derer, die ich von Hand gemalt oder zum Spaß am Computer gemacht habe.

Achten Sie sehr auf den Abschnitt über sich. Beschreiben Sie, was Sie von dem neuen Job erwarten, welche Teamgröße Sie haben, welchen Tätigkeitsbereich Sie haben und was Sie Ihrem Arbeitgeber anbieten möchten.

Schreiben Sie jetzt alles so, wie SIE es sehen.
