无线数据传输的世界纪录：每11公里40 Gb / s

2019年8月，他们首次在俄罗斯（是的，这是事实）在俄罗斯完成了一个商业项目，以实现40 Gb / s中继光缆的无线冗余。 诺里尔斯克镍矿的子公司Unity运营商通过这种渠道通过叶尼塞转发了11公里的无线备份。



媒体和哈布雷杂志不时有关于无线通信世界记录的记录 。 从技术进步的角度来看，它们很有趣，但是它们始终是研究测试。 这是一个真正的商业项目，不是在硅谷或欧洲大学的条件下，而是在北极圈的针叶林中。 令人惊讶的是，这是一个巨大的国家，地理和气候条件艰苦，为为最佳研究实验室带来优势的项目创造了先决条件。

最新无线记录的时间表：

  • 2013年5月 ，由卡尔斯鲁厄技术学院，辐射计物理有限公司和弗劳恩霍夫应用固体物理研究所的科学家联合进行的实验，在240 GHz实验频率下， 每1 km为40 Gbit / s 。 信号频率不可用于商业用途。
  • 2016年5月 ：在70/80 GHz频段中， 在37 km处以6 Gb / s的速度运行，属于同一团队，但作为分配给商业项目的频率的新实验，
  • 2016年11月 ：每13公里20 Gbps，Facebook Connectivity Lab研究中心，
  • 2019年1月 ， 每1.4公里40 Gbit / s ，德国电信在爱立信串行设备上的训练场，于2019年5月将同一训练场上的相同链路连续扩展到8个，产生了约100 Gbit / s，
  • 2019年8月 ， 每11公里40 Gbit / s ，Unity运营商Norilsk在DOK LLC（圣彼得堡）的串行设备上。

实际上，如果不是在叶尼塞（Yenisei）上的冰漂，北极圈条件下可能没有无线记录。 该项目的背景如下：2017年，三大运营商拒绝在Taimyr方向发展通信之后，MMC Norilsk Nickel PJSC使用自有资金，在距墨西哥城956公里处建造了一条巨大的光纤干线（FOCL）。 Novy Urengya至Norilsk的容量为40 Gb / s。 这是一条非常艰难的道路，穿越了艰难的地形，其建造者因这项工作而获得了政府奖。

操作过程中的问题之一-一条40吉比特的FOCL电缆在没有桥梁的情况下通过叶尼塞河穿过，决定沿着河底铺设，并铺设了几根电缆以提高可靠性。 但是，冰漂移很容易损坏光学器件。 此外，叶尼塞（Yenisei）上的冰块漂流已经不是一天的事了，而且由于对人类的高度危险，一直以来都不允许对水进行维修工作。

除叶尼塞河底部的其他电缆外，这条路线还由位于伊加尔卡和Priluki村的河两边的无线电塔的无线电中继信道1 Gbit / cc保留（该无线电信道在顶部照片中可见-一个大盘子）。 但是，如果光学器件受损，则整个诺里尔斯克工业区需要提供1 Gbit / s的速率-眼泪。 因此，在2018-2019秋冬期间，作为PJSC MMC Norilsk Nickel结构的一部分的Norilsk运营商Unity开始着手设计通过Yenisei的无线通道的建设，其容量不低于光纤链路。

令Unity的专家惊讶的是，世界上没有一个电信品牌接受了为11公里外的40 Gb无线信道提供设备的提议。 这里的重点是高通道容量和范围的复杂组合。 在70/80 GHz范围内，容量为10 Gb / s或更高的现代串行设备具有范围非常有限的功能。 这是由于以下事实：使用诸如QAM128或QAM256之类的复杂编码方案，并且它们单独可以提供10 Gb / s或更高的带宽，因此很难确保显着的发射机功率。 3-5公里的路线很容易，但在11公里处，信号衰减会变得不必要地大，并且无法获得10GE标准的连接。

DOK公司在圣彼得堡的一家本地开发商接受了此电话。 她已经开发了无线电桥，并提供了必要的射程。 在此项目之前，他们在其4 km测试地点以4个共同运行的10 Gbit / s无线电桥的形式测试了40 Gbit / s信道，并确保可以获得这样的容量。 但是实际上，电信行业中从未有人尝试在11 km的距离上同时构建4个同时运行的10 Gb / s无线电桥。



在收到来自全球品牌的拒绝之后，由Unity LLC代表的客户也不确定家用设备是否可以应付该项目。 因此，一开始就决定将每11 km仅一个10 Gbit / s的无线电桥作为试验阶段。 如果他推荐自己很好，那么已经可以将任务扩展到4个并行运行的无线电桥。







从技术角度来看，完全可以选择通过空中和通过光缆在一个通道中传输40 Gbit / s。 通过多个并行的10 Gb / s“字符串”传输数据要容易得多。 10GE网络比40GE交换机更便宜，更实惠。 另外，并行“线程”为整个通道提供了更高的可靠性。

但是有一个问题是，与光缆不同，沿平行光纤的信号不会以任何方式相互影响，无线电信道会导致相互干扰，直至完全通信失败。 他们正通过施加不同的信号极化并在频率上扩展信号来解决这一问题。 但是说起来容易些，“在硬件中”实现起来要困难得多。 Petrograd在基于砷化镓的大型微波电路（MMIC，单片微波集成电路）上制作了电路，并对他们的电路解决方案充满信心。

“全世界的10GE标准无线电桥都是在购买的微波电路上制造的。 在这个领域，当所有技术过程都在一家公司中进行时，进行垂直集成开发效率很低-从微波芯片的沉积到组件到最终产品的组装。 这与许多公司基于Intel和AMD的芯片生产计算机板的情况相同。 但是，与用于PC的大规模母板不同，它需要特殊的专业知识来配置微波芯片，然后放大信号并将其馈送到天线，这实际上是公司专有技术的主题，”项目经理Valery Salomatov评论说。 LLC DOK。

PPC-10G-E-HP 10 Gb / s导频无线电桥成功地在叶尼塞河沿岸的铁塔上工作了几个月（2019年5月至6月）。 夏季降雨是毫米波无线电通信最困难的时期，因为 雨滴与波长（约4毫米）相当，这会导致信号衰减。 在冬天，不会发生这样的问题，因为 雪花，雾气和烟雾对于70/80 GHz频段的无线通信是透明的。





DOK LLC的10 Gbit / s无线电桥应对了天气条件和距离，此后，根据通信线路的可用性统计，Unity运营商决定扩展为4个并行无线信道，每个信道的容量为10GE。 安装工作由“ Unity”公司的专家完成，他们根据设备说明独立地弄清了调试的复杂性。 在2019年7月底，无线电桥
在DOK公司的安装监督团队的陪同下，通过Yenisei传输的40 Gbit / s（4×10 Gbit / s）进入了商业运营。
