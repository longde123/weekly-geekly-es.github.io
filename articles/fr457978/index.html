18 astuces pour travailler avec Visual Studio

Que vous soyez débutant ou que vous utilisiez Visual Studio depuis de nombreuses années, il existe de nombreux conseils et astuces pour vous aider à devenir plus productif. Pendant un certain temps, nous avons partagé des conseils sur Twitter en utilisant le hashtag #vstip , et aujourd'hui, nous avons compilé une sélection des meilleurs d'entre eux aujourd'hui.

Débogueur


L'utilisation de F10 au lieu de F5 pour générer, exécuter et attacher le débogueur est automatiquement interrompue la première fois que vous exécutez votre propre code. Les points d'arrêt ne sont pas nécessaires.



Pris en charge depuis Visual Studio 2005

Rattacher au processus (Maj + Alt + P) est une fonctionnalité extrêmement utile qui vous aide à rejoindre le même processus encore et encore.



Pris en charge depuis Visual Studio 2017 v15.8

Un point bleu sur le champ indique un changement de thread pendant le débogage.



Pris en charge depuis Visual Studio 2013

Suite sous la coupe!

Solution


Améliorez les performances de chargement de la solution et réduisez le bruit visuel en désactivant la récupération des extensions de nœud dans l'Explorateur de solutions, ainsi que la réouverture des documents lors du chargement de la solution .



Pris en charge depuis Visual Studio 2019

Pour une navigation rapide, utilisez Ctrl + T pour trouver quoi que ce soit dans votre solution - fichiers, classes, etc.



Pris en charge depuis Visual Studio 2017

Attribuez un raccourci clavier pour effectuer le «git pull» afin que vous n'ayez pas à utiliser l'interface CLI ou Team Explorer pour cela.



Pris en charge depuis Visual Studio 2019

Donnez à l'Explorateur de solutions la possibilité de sélectionner automatiquement le document actuellement actif afin de ne jamais perdre son emplacement dans le projet.



Pris en charge depuis Visual Studio 2010

L'éditeur


Mettez facilement en surbrillance les éléments HTML avec une balise div à l'aide de la combinaison Maj + Alt + W. Le div inséré, vous pouvez facilement le modifier pour n'importe quelle balise de votre choix, et la balise de fermeture changera automatiquement.



Pris en charge depuis Visual Studio 2017

Copiez tout fragment JSON dans le presse-papiers et collez-le en tant que classes .NET fortement typées dans n'importe quel fichier C # ou VB.



Pris en charge depuis Visual Studio 2013

Vous n'avez pas besoin de mettre des guillemets doubles autour des noms de propriété JSON vous - même - entrez simplement deux points et Visual Studio les placera automatiquement.



Pris en charge depuis Visual Studio 2015

Rendez IntelliSense et les info-bulles translucides tout en maintenant la touche Ctrl enfoncée.



Pris en charge depuis Visual Studio 2010

Au lieu de ressaisir '(') pour afficher les informations sur les paramètres dans les signatures de méthode, utilisez Ctrl + Maj + Espace pour afficher la surcharge actuellement utilisée.



Pris en charge depuis Visual Studio 2010

Autre


Jouez du son lorsque certains événements se produisent dans Visual Studio.



Pris en charge depuis Visual Studio 2010

Créez vos propres dispositions de fenêtre pour des scénarios de développement spécifiques ou des paramètres de moniteur, et passez facilement de l'un à l'autre.



Pris en charge depuis Visual Studio 2017

Spécifiez les composants Visual Studio nécessaires pour toute solution et Visual Studio invitera l'utilisateur à les installer s'ils sont manquants. Pour plus d'informations, consultez l'article « Configurer Visual Studio dans votre organisation avec .vsconfig ».



Pris en charge depuis Visual Studio 2019

Extensions


Vérificateur orthographique de Visual Studio . Une extension d'éditeur qui vérifie l'orthographe des commentaires, des lignes et du texte brut. Il peut également vérifier l'orthographe d'une solution entière, d'un projet ou d'éléments sélectionnés. Des options sont disponibles pour définir plusieurs langues pour la vérification orthographique.



Pris en charge depuis Visual Studio 2013

Ajouter un nouveau fichier . Une extension pour Visual Studio qui vous permet d'ajouter facilement de nouveaux fichiers à n'importe quel projet. Appuyez simplement sur Maj + F2 pour créer un fichier vide dans le dossier sélectionné ou dans le même dossier que le fichier sélectionné.



Pris en charge depuis Visual Studio 2015

Git Diff Margin . Git Diff Margin affiche les modifications Git actuelles dans le fichier modifiable sur le terrain et dans la barre de défilement de Visual Studio.



Pris en charge depuis Visual Studio 2012

Ce ne sont que quelques-uns des milliers d'extensions disponibles. Pour en savoir plus, accédez à Visual Studio Marketplace .

En conclusion


Ce ne sont là que quelques-uns des conseils publiés sur le hashtag #vstip sur Twitter. Il existe de nombreux autres conseils dignes d'attention. De plus, si vous avez de bons conseils, partagez-les avec le hashtag #vstip.
