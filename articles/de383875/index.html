HTML5-Version von StarCraft



Das ursprüngliche StarCraft-Spiel (mit Brood War) wurde auf HTML5 portiert . Jetzt wird dieses Meisterwerk von Blizzard Entertainment direkt in Ihrem Browser gestartet. Das ist zwar noch kein komplettes Spiel, sondern nur Demo-Levels, aber es sieht ziemlich würdig aus. Der Entwickler hat hart gearbeitet und den Code aus den Programmiersprachen C, C ++ und dem Assembler nach Javascript portiert.



Ein Benutzer mit dem Spitznamen Ryuta hat den Quellcode auf Github veröffentlicht. Wenn Sie das Spiel auf Ihrem PC platzieren, der als Server fungiert, können Sie offline spielen, indem Sie StarCraft auf localhost ausführen.

Die Entwicklung der HTML5-Version ist eine besonders gute Nachricht für Linux-Benutzer, da das Spiel zuvor nicht direkt im kostenlosen Betriebssystem gestartet wurde, sondern nur über Wine. Jetzt funktioniert die Demo ohne Emulatoren. Die Kultstrategie in Echtzeit wurde vor 18 Jahren veröffentlicht, kann aber bisher als Qualitätsstandard im RTS-Genre bezeichnet werden. Wie auch immer, es ist schön, sich an die guten alten Zeiten zu erinnern, die dank moderner Webtechnologien zurückgekehrt sind. Vielleicht nicht lange. Das Repository auf Github enthält proprietäre StarCraft-Grafiken, für die Blizzard Entertainment zum Autor fliegen kann.







Übrigens wird auch daran gearbeitet, Starcraft 2 auf WebGL zu portieren .
