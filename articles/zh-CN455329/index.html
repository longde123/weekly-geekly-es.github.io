Azure IoT Edge工具扩展（预览）公告

我们很高兴宣布Visual Studio 2019的新Azure IoT Edge工具扩展（预览版）的预览版本可用。 该扩展提供了丰富的功能集，以支持使用Visual Studio 2019开发IoT Edge解决方案：



  • 针对不同平台（Linux amd64，Linux arm32v7，Windows amd64）的新Azure IoT Edge项目
  • 向解决方案添加新的IoT Edge模块（C＃/ C）
  • 在Visual Studio计算机上本地编辑，构建和调试IoT Edge模块
  • 构建和推送IoT Edge模块的Docker映像
  • 在本地或远程模拟器中运行IoT Edge模块
  • 将物联网解决方案部署到物联网边缘设备（使用Cloud Explorer ）


先决条件


Visual Studio 2019 ：安装了“ .NET桌面开发”和“ Azure开发工作负载”工作负载； 如果您计划开发C模块，则需要“使用C ++进行Windows桌面开发”。

Docker桌面 。 您需要正确设置以Linux容器模式或Windows容器模式运行的Docker CE。

若要设置本地开发环境以调试，运行和测试IoT Edge解决方案，您需要Azure IoT EdgeHub开发工具 。 安装Python（2.7 / 3.6） ，然后通过在终端中运行以下命令来安装iotedgehubdev。 确保您的Azure IoT EdgeHub开发工具版本大于0.8.0。 

pip install --upgrade iotedgehubdev

安装方式


有两种方法可以安装新的扩展程序：


  • 从Visual Studio Marketplace下载并安装新的扩展。
  • 或者，您可以使用菜单扩展->管理扩展直接在Visual Studio 2019中安装扩展 。 在“管理扩展”窗口中，从左侧面板中选择“ 联机” ，然后在右上角的搜索框中输入边缘 ，以搜索和下载“用于VS 2019的Azure IoT Edge工具[预览]”。


如何使用此扩展名？


请参考以下教程以开始使用：

→ 使用Visual Studio 2019为Azure IoT Edge开发和调试模块（预览）
→ 使用Azure IoT Edge工具轻松开发和调试Azure IoT Edge C模块
→ Visual Studio Azure IoT边缘工具文档存储库


请不要犹豫， 尝试一下 ！ 您的反馈和建议对于我们不断改进并使其更易于开发IoT应用程序非常重要。 建议功能或在Visual Studio Azure IoT Edge工具仓库中报告问题，与我们分享您的想法。
