Cómo hacer sitios en 2019

La web está cambiando rápidamente, ¡mantente al día!



Escribí un artículo similar en 2018, pero la web está cambiando tan rápido que esas pautas están irremediablemente desactualizadas.


Echemos un vistazo a los principios más importantes del diseño web en 2019 paso a paso.


Asegúrese de informar al usuario que el sitio tiene una aplicación móvil


Este es un principio esencial, y hay razones para ello. Los grupos focales, para los cuales ya hemos gastado mucho dinero, han demostrado que lo primero que un usuario desea para llegar a su sitio es instalar una aplicación móvil.


Bueno, el camino más corto para esto es mostrar una ventana modal considerable:



Como opción opcional, puede agregar un botón o un enlace para cerrar esta ventana, pero es fundamental que el texto en él sea tenue y esté formulado como si el usuario tuviera vergüenza de rechazarlo o de lo contrario quiere instalar la aplicación en el futuro.


Consejo: si no tiene una aplicación móvil, ¡no importa! ¡Simplemente atiborre todo el sitio en la vista web con la seguridad deshabilitada y escriba que esta es una aplicación móvil!


Desarrolle una estrategia para No rastrear


La mayoría de los navegadores modernos (¡para que lo hagan!) Admiten el encabezado DNT HTTP, que significa No rastrear. Este encabezado indica que el usuario no desea ser rastreado y, desafortunadamente, este encabezado está habilitado de manera predeterminada en algunos navegadores como Brave ( navegador de código abierto con bloqueo de anuncios del creador de JavaScript Brendan Eich ).


Sin pánico! Resultó que la mayoría de los usuarios apagarán el encabezado tan pronto como les pregunte, y para que todo funcione como un reloj, adjunte instrucciones detalladas sobre cómo hacer esto:



Consejo: no pregunte si el usuario desea deshabilitar DNT en el manual para deshabilitar DNT; de lo contrario, seguirá aceptando.


Desarrollar una estrategia para Cook


No es necesario preguntar acerca de las cookies si se usan en el sitio por razones adecuadas, pero es mucho mejor solicitar permiso para todas las cookies a granel, ¡entonces podemos rastrear todo lo que queremos sin temor a consecuencias!



Consejo: No cargue páginas sin cookies, incluso si no son necesarias para el trabajo, ya que podrá rastrear mucha menos información personal que luego podrá vender.

Haz un diseño lo más posible


Los días de los diseños de 800 píxeles han quedado atrás. Ahora hay más y más pantallas panorámicas, así que trate de usar la mayor cantidad de espacio posible.



Consejo: ¡Los estudios muestran que las ventanas modales sin un botón de cierre se cierran con mucha menos frecuencia!


Usar notificaciones


Los navegadores modernos admiten notificaciones, lo que significa que deben estar activadas, pero si no funciona, siempre puede intentar suscribirse a una lista de correo o sms de forma predeterminada.



Consejo : También puede usar estas notificaciones para informar a los usuarios sobre una nueva versión de una aplicación móvil.
