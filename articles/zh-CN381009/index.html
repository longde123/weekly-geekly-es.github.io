Windows 10通过业余爱好者的眼睛

Windows 10已经拖延了很长时间，每个人都在谈论它。那么，为什么不尝试谈论新的Microsoft产品呢？绝对不会使任何人变得更糟，对吗？在发布新版本的Windows之前已经有一点点时间了-一个多月，因此已经可以进行某种预览审查并告诉您您的使用印象，因为该版本几乎没有任何重大变化。该审查是在目前的最后一个版本10130上完成的。

也许值得从安装/更新过程本身开始。从头开始安装时，什么都没有改变，只有过程的设计有所改变。现在，我们在下面有一个包含百分比和安装步骤的圆圈。安装本身与安装“八乘七”的时间大致相同。

如果从W7 / 8.1升级，则安装过程将花费更长的时间。可以理解，备份已完成，Windows 10本身已安装，旧Windows中的数据已还原。例如，我的更新过程大约花费了五个小时，因此请做好准备。

安装后，建议连接到网络，使用Microsoft帐户登录，设置PIN码。然后将安装标准应用程序，配置文件，PC将重新启动，我们将看到Windows 8熟悉的锁定屏幕。我们输入PIN码或密码，最后进入桌面。



立即注意到数十种新设计。极简主义都一样，但现在也带有深色调。他们在某个地方添加了新的动画，更改的声音，图标以及许多您不会立即注意到的小东西。但是首先是第一件事。

让我们从“开始”开始。随着Windows 8的发布，许多用户对面向平板电脑的图块的全屏Start外观极为否定。现在，“经典”开始与新的结合。在这里，我们既有图块又有经典的应用程序列表。没错，开始还没有全力以赴。没有指向用户文件夹的链接，依此类推。取而代之的是，有一些用于快速访问设置的按钮，这些按钮应代替控制面板和资源管理器。如果愿意，可以在“开始”中删除磁贴。取消全部固定，并调整“开始”本身的大小。



指挥也改变了一点。现在，它具有一个快速访问页面，该页面显示常用的文件和文件夹以及您自己“附加”的文件和文件夹。 Win + E组合现在发送到这里，而不是打开“我的电脑”。如果需要，可以在设置中更改此组合的功能。

现在介绍一下设置本身。在这里，“控制面板”中的参数和一些新参数正在这里缓慢地收集，但是不幸的是，仍然不够，Microsoft只留下了“控制面板”的链接。也许然后一切都将移到这里，或者也许只有最基本的参数。



在“开始”旁边是新图标：圆形和矩形。 Small-Gentle Cortana和观看任务是如此地赞美和吹捧它。

也许值得从一个较小的功能入手-虚拟桌面和查看任务。当您单击“任务视图”按钮或使用“ Win + Tab”时，将打开一个窗口，您可以在其中查看所有正在运行的窗口和桌面。您可以将桌面与浏览器窗口进行比较，将应用程序窗口与选项卡进行比较。创建新的桌面时，与打开新的浏览器窗口时会发生相同的事情-您将获得一个干净的工作空间。只是不要忘记您的RAM不是无限的，是的。



微软还更新了Windows 7附带的一项出色功能-快照视图。最有可能的是，许多人经常使用它。将屏幕分为两个工作区并使用信息而无需在窗口之间切换很方便。现在，您可以轻松地将其分为至少三个，至少四个部分。只要考虑使用此功能的方案。



现在关于Cortana。重要说明：到目前为止，Cortana仅适用于少数国家/地区，到目前为止，俄罗斯不在其中。因此，下文所述的所有内容都谈到了美国Cortana。她出现在Siri和Google Now这样的巨头之后，是最新的，因此到目前为止，她的功能还很差。通常，Cortana会按名字给您打电话（不管是您喜欢的还是其他方式），提供上班和回家的路线，显示选定类别的新闻，识别播放的音乐，提醒您日历中的笔记和事件，通常几乎所有已经谷歌现在。尽管Cortana的助手不是很好，但是Google Now和Siri并没有立即学习如何做所有事情。但是启动应用程序和用声音切换音轨已经很有趣了。



有了这些，在Windows 10中就会弹出一个通知栏，就像在Android，iPhone和Windows Phone上一样。这里没有超级革命性的东西，只是来自通用应用程序和普通桌面应用程序（Win32）的通知都将落在这里。好的，这里有一些快速设置，但它们主要用于平板电脑（您不会在固定PC上打开飞行模式）。



值得谈论通用应用程序。这些是在Windows新平台上构建的应用程序。 PC，平板电脑，智能手机，Xbox，物联网设备以及神秘的Hololens-如果开发人员做出最少的手势操作，则可以在任何地方运行该应用程序。虽然，接口的适配当然没有被取消。例如，现在，已经有了Office的“通用”版本。它具有简化的功能，但是非常适合于编辑小型文档，并且已经可用于PC和移动设备。此外，它是完全免费的。

这些相同的“通用”应用程序的使用也有所变化。如果在Windows 8中只能在全屏模式或分屏模式下使用它们，则现在它们是普通窗口，可以像其他窗口一样移动和调整大小。

微软还更新了他们的应用程序：Xbox，照片，邮件，人，日历，计算器等。更改了设计，更加强调了键盘和鼠标。但是触摸设备的所有者不必担心，按钮变小了，但不足以错过它们。对于平板电脑，他们添加了一种特殊模式，大约在稍后。只是在许多应用程序还没有声明的功能的时候，我希望它会在发行版之前出现。如果更新的应用程序的功能比以前的版本差，这将很奇怪。

在缺点中，我可以注意到，Charms侧边栏被剪掉了几十个，有时在“旧” Windows应用程序中是必需的。并且以程序头中的三点形式替换它不是很方便使用。令人遗憾的是，他们在“ OneDrive”中卸下了“插头”。该芯片的本质是用户不必从云中下载所有文件，但是它们是可见的，并且在打开时便从网络中加载。仅仅是一些用户是愚蠢的，因此Microsoft（暂时）放弃了这种方法，并返回了整个云或选定文件夹的常规同步。以及如何忘记错误。他们是。到目前为止，我还没有发现任何紧急情况-系统没有崩溃，驱动程序稳定，应用程序（几乎）没有崩溃。到目前为止，新行的翻译与Promt无关，但肯定会解决。

平板电脑。 Windows 8专门针对触摸屏设备进行了大幅改进。 Windows 10稍微改变了偏见，现在正尝试取悦所有人。但是我们知道这几乎是不可能的。但是微软正在尝试。因此，有一个数位板模式，当您从数位板变压器上断开键盘连接时，可以从通知中心手动或自动激活。开启后，任务栏会丢失正在运行的程序的图标，并变得更粗一些。剩下的只是状态图标，启动，时钟，通知栏，cortana和用于查看任务的按钮。借助它，您只需选择必要的窗口之一，这些窗口现在已变为全屏。但是随后出现了后退按钮，在最后两个窗口之间切换。开始几乎就像Windows 8中一样，带有附加菜单，称为左侧滑动或角落的汉堡按钮。



最后但并非最不重要的一点是性能。 Windows 10进行了优化。它的工作原理，至少不比七八分差。一切也很快，反应灵敏且良好。在功能强大的PC上，在7年前的笔记本电脑上（我在其上编写此预览）。 3D应用程序也没有失去敏捷性，一切都像以前一样流畅。
