Não por muito tempo a música tocou ... ou como o OS Elbrus nunca se tornou livre

Alguns dias atrás, alguns meios de comunicação relataram a possibilidade de baixar gratuitamente o sistema operacional Elbrus. Os links de distribuição foram fornecidos apenas para arquiteturas x86, mas mesmo nesta forma, esse pode ser um marco muito importante no desenvolvimento desse sistema operacional.

Uma das manchetes da mídia: OS Elbrus tornou-se gratuito. Links para Download
O desenvolvedor da linha de processadores domésticos Elbrus atualizou a seção sobre software especializado em seu site. O Elbrus OS para processadores de arquitetura x86 padrão estava disponível gratuitamente para download. Em breve, os desenvolvedores planejam abrir seus códigos-fonte.
Outra manchete para as mesmas notícias: O sistema operacional Elbrus já pode ser baixado!

Sim, isso pode realmente ser um marco muito importante no desenvolvimento do sistema operacional Elbrus. Poderia se tornar, mas infelizmente ainda não foi (espero que a palavra seja a palavra-chave por enquanto )
