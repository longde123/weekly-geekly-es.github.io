Zusammenfassung der IT-Ereignisse im November (Teil 1)

Effiziente Seleninfrastruktur

Wann: 2. November
Wo: Jaroslawl, October Lane, 3, Büro 207
Teilnahmebedingungen: 5000 Rubel.

Die Schulung soll Webentwicklern und Testautomatisierungsingenieuren zeigen, wie sie Selenoid und Ggr effektiv einsetzen können, um eine skalierbare Infrastruktur für die Browserautomatisierung zu erstellen, was den Reiz von Containern ausmacht und wie ausfallsichere Cluster erstellt werden. Erfahrungen mit Selenium, der Linux-Befehlszeile und Docker sind willkommen. Es wird außerdem empfohlen, Docker und einige Selenium-Tests zu installieren. Insgesamt dauert das Training acht Stunden, die auf die drei Blöcke aufgeteilt werden. Zunächst lernen die Schüler die Geschichte und das Wesen des Selenium-Projekts kennen, lernen, wie es installiert und gestartet wird, und sortieren die Hauptfunktionen. Der nächste Block konzentriert sich auf die Einrichtung eines Selenium-Clusters und Ggr sowie auf die Clusterverwaltung. Abschließend werden die Organisatoren über erweiterte Funktionen von Selenoid sprechen.

Maschinelles Lernen in BigData-Projekten

Wann: 2. November
Wo: Jaroslawl, October Lane, 3, Büro 207
Teilnahmebedingungen: 8000 Rubel.

Gleichzeitig werden am selben Ort Schulungen für diejenigen abgehalten, die mit dem Backend, Java / Scala, Python / R oder C # / C ++ vertraut sind und ihr Wissen über BigData für zukünftige (oder aktuelle) Projekte vertiefen möchten. Die Anforderungen für all diese Personen sind dieselben - Grundkenntnisse in Linux, Java und SQL. Der theoretische Teil der Schulung umfasst die Grundkonzepte des maschinellen und tiefen Lernens, Java-Bibliotheken, die Integration in das Python-Ökosystem, die Arbeit mit dem Spark-Cluster, maschinelle Lernketten, Kafka, Flink und Storm. Jedes Modul wird von Übungen begleitet - Aufgaben erledigen, Live-Codierungssitzungen durchführen und die Werkzeuge näher kennenlernen.
