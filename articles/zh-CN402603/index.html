一台频谱分析仪的历史



有时我想知道为什么我对某些事物非常感兴趣而对其他事物完全不感兴趣，因为兴趣可以帮助您持续数天而不感到疲倦，专注于感兴趣的主题，同时，如果它消失了，就很难强迫自己去做某件事。 不清楚它来自哪里，它离开哪里，只是很清楚，当他在那儿时，您不能浪费时间，您需要做一些有趣的事情，因为兴趣迟早会留下来，而经验会保留下来。

所以是这次，前一段时间是一名电工，我对这个问题很感兴趣-“我怎么知道报警控制台是否工作以及它们之间有何不同？”

这个故事如何结束，请仔细阅读。

时间的流逝，我改变了工作，但是兴趣和理解的欲望依然存在。
再一次，我在Internet上走来走去，遇到了rf Explorer，这是我所需要的，虽然价格让我感到困惑，但它似乎是一个“显示器”，并且没有针对电源进行标准化，但售价为130美元。

我不喜欢这样的购买，最好推迟这笔钱，以后再购买一台成熟的分析仪，然后自己尝试收集，因为从互联网上的照片来看，它仅基于一个si4313芯片构建。 到那时，我已经具有显示器和MK的经验，因此我在Ali上订购了si4313并开始阅读数据表。

芯片到货后，花了很多时间才能启动它。 根据书面的TK，第一阶段通过，第二阶段是对设备操作和电路开发的描述。 它适中，因此可以快速设计电路，绘制并组装电路板。 对我来说，最困难的时刻到了-这是界面的开发，尽管它是什么样的开发，但有必要决定如何在屏幕上排列元素，使它们看起来很漂亮，并选择颜色。 由于多次尝试和错误，此接口被证明。



图片中所示的设备准备就绪后，我很满意，有些窗帘开了，显影停止了。 通常，当我为自己做某事时，这种情况总是会发生，有一段时间我会尽可能地专注于开发，直到获得令我满意的结果，而忘记这个项目一两个月。

几个月后，我决定在工厂订购一块板子并将整个东西包装在一个箱子里仍然是一件好事。 在Ali上发现了一个铝制的盒子，为此重新绘制了木板，然后订购了该盒子和木板。 在订单进行期间（即一个月），开发再次停止了一段时间。

当包裹到达时，事实证明箱子不适合高度，我不得不使用3D打印机上打印的插入物对其进行升级。 在接下来的几天里，设备组装完毕，人们的兴趣得到了充分满足，并且开发又被放弃了。 当时，他具有以下特征：

  • 频率范围240-960 MHz
  • 最小跨度-3.18MHz
  • 最大跨度-636MHz
  • 可调输入滤波器带宽-10.6 kHz / 24.0 kHz / 56.2 kHz / 112.1 kHz / 225.1 kHz / 335.5 kHz / 420.2 kHz / 620.7 kHz
  • 可调扫描步长-10kHz / 50kHz / 100kHz / 500kHz / 1MHz / 1.5MHz / 2MHz
  • 灵敏度--98 ... -118dBm
  • 电流消耗-170mA
  • 将屏幕截图保存到SD卡

使用一块3.2英寸的电阻式触摸屏和一个编码器进行了控制，尽管该板上有放置机械按钮的地方，但不适用于这种情况。













一段时间后，我意识到可以截取屏幕截图非常方便，几周后，我将sd卡拧紧到分析仪上，所幸我提供了在PC接线阶段连接它的可能性。 有几个简单但重要的问题，功率读数中的误差是什么，以及它是否在整个范围内均有效。 在此之前，我用它来检查警报中的钥匙扣，即在315和433 MHz的频率下，我不知道它在其余范围内是否正常工作。

为了回答这些问题，组装了dernuss的RF发生器， 在此进行描述并进行检查。 我对结果感到满意，分析仪在整个范围内都能看到它，但我无法弄清楚功率，结果证明发生器的功率未标准化，并且在其设置中，您只能从它可以发出的最大值中选择信号的衰减。

当我发表有关示波器的最后一篇文章时，我以个人电子邮件形式写了engine9并愿意从事驴接口的设计工作，但是后来我有了其他计划，但我拒绝了，说我正在计划303芯片上的另一个版本，因此在这里继续进行研究。 过了一段时间，我们注销了，不得不开始研究驴子。 在一次对话中，我提到了分析仪的工作，并希望他以他的创造性外观来观察它，并提出可以解决的问题，因为我绝对不喜欢设计的某些方面。 他看了看然后开始……总体而言，我们开始着手分析仪的设计，这就是最后发生的事情。


当然，您不能将此设备称为频谱分析仪，它很可能是“显示仪表”，并且永远不会成为分析仪，因为这种设备的开发人员几乎不会影响si4313芯片的操作，也就是说，如果该芯片在某个范围内褪色并给出错误的结果， ，请解决此问题。 幸运的是，到目前为止，尚未发现此类错误，但是测试人员是其中之一））
