Mikrotik mAP精简版

我一看到，便获得了我要分享的新微型模型Mikrotik RouterBoard mAP lite。


我已经有了Mikrotik RouterBoard mAP 2n，但是mAP lite的尺寸给我留下了深刻的印象：48x49x11mm
MAP Lite图像









设备规格和测试可在制造商的网站上找到。与mAP 2n简要比较：
  • 处理器频率650 MHz，高于mAP 2n-400 MHz
  • RAM 64 MB，如mAP 2n
  • 像mAP 2n一样快闪16 MB
  • 1个以太网端口（10/100）-mAP 2n有2个
  • Wi-Fi QCA9533 802.11b / g / n-at mAP 2n AR9331

没有超越自然，除了尺寸48x49x11mm对68x68x19mm在地图2N。
该设备在网站上的建议价格为25美元，我设法以2700卢布的价格购买该产品。
第一个惊喜是包装盒的尺寸（长包装盒中配备了相同的mAP 2n）
包装mAP lite和mAP 2n



mAP lite随附电源（5V-1A），金属安装，橡胶脚。
套餐捆绑




MAP Lite本身









我拆开婴儿的第一件事。原来不容易打开。非常紧固的按扣。
里面








设备内部有磁铁，可让您将路由器安装到任何金属表面上，但最重要的是将其安装在其随附的正方形上。
挂载




检查PoE供电（802.3af / at（+ 4.5 / -7.8或+ 3.6 / -1.2））
PoE供电


在mAP lite上，有一个新的固件。标准，常规RouterOS。一切正常，没有问题。工作很热情（尽管如果您通过它吸引更多的流量，情况可能会发生变化）。
该设备将在下一个组件中组装。
组装方式





其实质如下。我将具有4G和Wi-Fi的路由器和调制解调器-华为E8372（MTS 8211F）连接到了移动电源（Romoss solo 3 PH30 6000mAh）。路由器紧贴以太网（如果未连接），然后紧贴特定的Wi-Fi网络，如果找不到，则紧贴调制解调器。接下来，将L2TP / IPsec VPN到家庭服务器的SoftEther。配置的Web代理以重定向到家庭代理。现在，我将遍历这些脚本，最后，如果有趣的话，我将在新文章中发布它们。

愿望清单：
  • USB接口 单独的端口。根据情况判断，制造商也会考虑。可以直接连接调制解调器。
  • 802.11ac
  • 哔声-您真的需要它
  • 与mAP 2n一样的LED控制

另一个以太网端口并通过它为其他设备供电，但这已经导致尺寸增加，我不希望这样做。
这是我得到的东西。
