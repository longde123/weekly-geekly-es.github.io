尽管带有电子墨水的设备可以让您手写手写的笔记和绘图已经存在了很长时间，但相对较近的笔触到笔迹与痕迹的出现之间就已经达到了可接受的延迟（已知的是Sony DPT-S1 / RP1和reMarkable）。 自1990年代以来，我有很多不同的设备-都带有笔输入（Palm，Newton MP2000，HP TC1100，Raon Everun，Asus EA800，Boogie Board和仅电子墨水阅读器（Kindle Paperwhite，PocketBook 740，良好的电子阅读器6.8），即各种解决方案的利弊思路，以及“理想”设备的某种个人形象。

在我看来，它应该具有带背光的电子墨水显示器，具有按压力识别功能的磁共振数字化仪（通常称为“ wacom”），绘图时无法察觉的延迟，10英寸（或更多）的电子墨水屏幕，足够选择的第三方应用程序。

但是，正如我所说，相对较近的时间，这些技术问题已得到解决。 另外，出现了或多或少的最新版本Android的E-ink屏幕驱动程序-由于缺少E-ink阅读器，直到最近，其Android版本均未高于4。

第一款至少能正式满足我的要求并同时满足我要求的设备在俄罗斯出售（我不想重复一周在美国购买的Good e-Reader的体验）-Onyx Boox Note Pro。

在哈布雷（Habré）上已经对该设备进行了广告审查 ，因此，我将不对其进行详细描述-我将重点介绍对我来说重要的东西以及对做出购买决定的人有用的东西。

简单来说，Onyx Boox Note Pro是一款Android 6平板电脑，带有10英寸带背光和wacom笔的电子墨水屏。技术上：1872x1404x16，Cortex-A17 1.6GHz四核，4GB RAM，64GB闪存，Wi-Fi / BT 4.1，USB Type-C，音频/麦克风，390克，返回和电源按钮，Android 6。

Android的功能在某种程度上受到限制（在某些情况下它是可逆的，在某些情况下不是可逆的）-我想这样可以将其定位为电子书。 从营销的角度来看，这是合理的，因为将其作为平板电脑（即平板电脑）出售意味着对“制动黑白屏幕”会产生负面反应，与此同时，电子墨水书的用户也非常熟悉。 另一方面，设备的成本是如此之高（我认为主要是由于E-ink和wacom庞大），Android提供的附加功能对于证明价格是必要的。 但是，使用Android意味着运行时间很短，而且由于更新速度慢，它在E-ink上根本没有得到加强。 结果，我们有很多不同的权衡。
