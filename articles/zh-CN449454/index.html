微软Kaizala如何允许印度铁路将300万医护人员汇集在一起

印度最大的雇主印度铁路公司（Indian Railways）将使用Microsoft Kaizala将其遍布全国的员工与优质的医疗设施联系起来。 Microsoft Kaizala应用程序将使员工和退休的铁路工人能够利用125家铁路公司和133家私家医院的医疗服务。 Kaizala的团队由中央南部铁路的医生领导，将由有针对性的医生，护士人员和护士组成的团队进行补充。



在医疗服务机构注册时，印度铁路公司的员工将可以搜索Microsoft Kaizala，最近的医院和医生，诊断中心和医疗设施的列表。 员工可以与医生预约，直接与医生共享诊断实验室报告，并在Microsoft Chat Meiz Meiz中保存数字记录。 他们还可以访问重要的公告，通过内置的动作卡分享他们的反馈意见，以提高医疗质量。

Kaizala允许印度铁路公司与其员工进行直接沟通，包括成就公告，教育公告和新闻通讯，以传播有关心肺复苏，急救，免疫，疫苗接种等方面的信息。 Kaizala小组的医生可以查看员工的病史，检索病历并帮助及时做出决定。 Microsoft Kaizala是仅限移动设备的产品，通过将计算机用户和仅使用移动设备的用户召集在一起，组织可以轻松地进行通信，协作和执行任务。

印度铁路铁路理事会主席Vinod Kumar Yadav先生在评论这一事件时说：“ 作为印度最大的雇主之一，我们努力为员工提供丰富的经验。 为我们的员工提供优质的医疗设施是关键领域之一，因此它与Microsoft Kaizala的集成将帮助我们轻松地为所有员工（包括印度偏远地区的员工）实施这一理念 。”

“ Microsoft Kaizala是一个简单且安全的聊天应用程序。 它使组织可以轻松访问组织内外的所有员工，以共享数据的方式做出明智的决策，并为员工提供易于使用的解决方案，以实现有效的协作并提高生产率。 凭借其Microsoft云平台，移动性和公司安全性，Microsoft Kaizala非常适合满足印度最大雇主之一的独特需求。 我们希望与私人和​​公共部门的大型客户合作的经验将有助于我们有效地为数百万名铁路员工提供服务 ，” 微软体验与设备事业部公司副总裁Rajiv Kumar说。

Microsoft Kaizala为铁路等大型组织提供了一种方便快捷的方法，可以共享见解以做出有见地的决策，并为员工提供易于使用的有效解决方案。
