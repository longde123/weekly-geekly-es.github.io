在覆盆子的俄罗斯学校中进行计算机科学课程的现代化：便宜又开朗

世界上没有比普通学校的俄罗斯IT教育更糟糕的故事了

引言


俄罗斯的教育体系存在许多不同的问题，但是今天我将考虑一个不经常解决的话题：学校的教育。 在这种情况下，我将不涉及人员主题，而只是进行“思想实验”，并尝试解决为计算机课配备少量血液的问题。


问题所在


  1. 在大多数中学（尤其是在各省），计算机科学课程很长时间没有更新，这有多种原因，我将重点介绍财务原因：市政预算缺乏针对性的投入，或者学校本身的预算不允许现代化。
  2. 除了时间以外，还有另一个影响设备状态的因素-学生。 通常，系统单元位于学生的附近，因此在无聊的时候，没有人看到时，有些人可以踢SB或以其他方式获得乐趣。
  3. 学生正在忙碌的不受控制的计算机。 例如，在一个20人的班级中（实际上，这个数字达到30个或更多），在计算机图形学或编写程序上给出了一项任务。 在这种情况下，如果老师有机会观看屏幕上正在发生的事情，而又没有在教室里跑来跑去，看着监视器中的每个人，然后停下来检查5分钟，那么本课将更加生动。

树莓的解决方案


现在：从抱怨到生意。 您可能已经了解，针对上述问题我将提供的解决方案是raspberry pi，但让我们看一下要点。


  1. 设备价格将从主要的联邦零售商的站点获取-这样做仅仅是为了方便，自然在实际情况下（购买设备时批发价会更低）。
  2. 在我的虚拟课堂中，我将做一个假设：老师准备坐下来学习一些与更新设备和扩展老师本人能力相关的细微差别。

因此，让我们开始吧。 与使用覆盆子有关的整个想法是基于它们的主要优点：紧凑，相对可用性，降低了功耗。


身体水平


基数


  1. 让我们从需要购买多少树莓开始。 让我们以该班的平均汽车数量为例：24 + 1（为什么我以后再告诉你）。 我们将使用Raspberry Pi 3 Model B + ，即约3.5千卢布。 每台 或8.75万卢布。 为25件。
  2. 此外，要放置板子，我们可以拿一个电信柜子，例如Cabeus的平均费用约为13000卢布。 同时，我们解决了第二段中描述的问题，即可以随时从学生身上移除部分设备并对其进行物理控制。
  3. 在大多数学校中，值得教育部肯定的是，安装了必要的网络设备：交换机，路由器等，但是，为了结构整洁，我们将这些内容包括在需求清单中。 让我们进行一个简单的切换，最主要的是应该有足够数量的端口-从26（24个学生，一个特殊，一个用于老师），我会选择D-Link DES-1210-28 ，它增加了7.5千卢布。 由我们承担。
  4. 我们还选择了一个简单的路由器，因为对我们来说，最重要的是，它以适当的速度拉出许多汽车，因此请选择Mikrotik-这是+4.5千卢布。
  5. 进一步的小事情：3个正常的HAMA 47775电涌保护器 +5.7千卢布。 跳线25条 用于从开关2 m。Greenconnect GCR-50691接线 = +3.7千卢布。 用于在树莓上安装OS的存储卡，至少为10类Transcend 300S microSDHC 32 GB的存储卡+ 1万卢布。 为25件。
  6. 如您所知，从不同的并行学习几十个类将需要超过32 GB。 到工作场所，这样就可以与学生一起共享资源库。 为此，请使用Synology Disk Station DS119j + 8.2千卢布。 和一个十亿字节的驱动器东芝P300 + 2.7千卢布。

总费用 ：142 800卢布。 （考虑零售价时）。


周边设备


我必须马上说，考虑到已经有键盘，鼠标和显示器，该清单还在继续-仅解决了将它们连接到远程计算机的问题。 另外，我假设底座在同一房间内，距离不超过5-10米，因为如果距离较长，则必须购买带中继器的HDMI电缆。


  1. 如前所述，要将显示器连接到树莓派，我们需要HDMI电缆。 拿5米的FinePower HDMI +1.92万卢布。 24个。
  2. 要连接鼠标和键盘，我们需要USB延长线Gembird USB + 5.2千卢布。 和拆分器DEXP BT3-03 + 9.6千卢布。

总费用 ：34 000卢布。 （考虑零售价时）。


组件总数 ：176 800卢布。 （考虑零售价时）。


程序级别


作为面向学生的操作系统，我认为值得选择标准的Raspbian，因为即使现在大多数学校仍使用Linux发行版（值得一提的是，它可能是由于资源有限，而不是因为他们知道这样做很有用）。 此外，在raspbian上，您可以安装学习培训计划所需的所有东西：libre office，geany或其他代码编辑器pinta，通常来说，已经使用的所有东西。 安装最重要的是Veyon或类似的软件，因为它可以从第三点解决问题，使您可以控制学生计算机上正在发生的事情，还可以使教师显示屏幕，例如进行演示。


总体而言，教师所需的软件与学生所需的软件没有太大区别。 关于老师最重要的事情是为什么您需要第25个覆盆子pi板。 实际上，这不是强制性的，但对我而言，其目的很重要。 我认为它应该在上面安装pi hole-特殊软件，可以帮助老师控制学生的网络活动。
