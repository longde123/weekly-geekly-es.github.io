电子产品中的3D打印



3D打印机无疑是一种出色的设备，对于许多人来说，它已成为不可缺少的设备。但通常，人们在寻找3D打印新应用程序时走得太远。今天让我们谈谈使用3D打印机生产印刷电路板和REA外壳。

首先，考虑所有用于制造印刷电路板（以下称为PCB）的方法。

PP生产


电子发烧友经常梦想在家中生产印刷电路板的设备。当然，还有伟大的技术“ LUT”（激光熨烫技术）以及经典的光刻胶。但是我常常懒得把它们弄乱，我希望有人为你做所有的事情。

还有一些创建软件的区域：

  • 去除导电层。研磨或刮擦玻璃纤维铜层
  • 去除玻璃纤维表面的保护层
  • 光刻胶亮点
  • 保护层在玻璃纤维上的应用
  • 在不同材料的表面上直接印刷导电和电介质轨道
  • 用导电材料填充空腔

去除导电层。研磨或刮擦玻璃纤维铜层


这种方法是最简单的，甚至具有非常摇摇欲坠且便宜的铣刀来完成此任务。许多发烧友甚至在他们的3D打印机上放了一个小型雕刻机，并成功地研磨了PP。 （展望未来，假设基于3D打印机的铣床仅能够铣削软件，仅此而已。尽管许多制造商都在尝试将它们组合在一起，但铣床和3D打印机却是不同的设备）。

» Roland EGX-350

工作区域尺寸：305 x 230 x 40毫米；主轴功率：50 W，20,000 rpm;轴数：3


EGX-350是最受欢迎的Roland雕刻机系列。 EGX-350体积小巧，能够在有色金属上进行全尺寸雕刻，具有高达20,000 rpm的高速主轴以及在曲面上进行雕刻的功能，因此在印刷版制造，广告纪念品雕刻，不仅可以在珠宝首饰中创建蜡质模型，还可以在机械工程中用于打标，雕刻印刷电路板和电极以进行电腐蚀等。

» Roland EGX-400

工作区域尺寸：407 x 305 x 42.5毫米；主轴功率：72 W，30000 rpm;轴数：3


EGX-400是专业的Roland雕刻机，专为高速雕刻而设计。罗兰EGX-400得益于高速主轴，最高30,000 rpm的速度，全轴伺服电机和FFP（前馈处理）技术，可向前扫描程序几步，并实现高速和平滑的曲线移动，因此是同类产品中的佼佼者


» Roland EGX-600

工作区域尺寸：610 x 407 x 42.5毫米；主轴功率：70 W，30000 rpm;轴数：3

EGX-600是一款功能强大且高精度的Roland CAMM-2 PRO雕刻机，带有伺服驱动器，以可承受的价格进行专业2D / 3D雕刻和铣削。 EGX PRO是一系列雕刻机，配备了先进的工业技术，具有专业雕刻机的功率，工作台尺寸和速度，但价格实惠。 EGX-600配备70瓦主轴，转速为30'000 rpm。 EGX-600机器非常适合制造用于压印和浮雕压印的高精度印章。此外，机器还刻有铭牌，标签，纪念章，三维题字，仪表板，印刷电路板等等。

如您所见，这些机器并不是最便宜的，而且很少有无线电爱好者能够负担得起。但是，本文中为铣削准备电路的说明仍然是：“ 在家中使用印刷电路板，或如何在家中使用CNC机床制造印刷电路板。”

此方法不仅限于一台铣床，还可以使用足够强大的激光来燃烧轨迹或用针刮擦轨迹。并且也要使用电蚀法。

REA外壳


如果使用软件或多或少地了解了一切，即使您没有用于自动创建板的复杂设备，也可以随时返回到LUT。设备外壳通常会出现问题。当然，有标准的REA盒，但它们并不总是适用于自制产品，因为必须严格观察电路板和元件的尺寸，这并不总是很方便。同样，不要忘记问题的美学方面。在这种情况下，普通的FDM 3D打印机可以为我们提供完美的帮助。唯一的问题是，是否有任何打印机适合这些任务。

在撰写本文时，老虎友善地分享了他的经验。进一步说：

“一旦我有了彻底了解此问题的经验。该公司的任务是开发瓢虫形式的REA外壳，并进一步制造用于TPA的模具。当您了解问题所在时，该盒子看起来应该像一头牛，并且有必要在盒子内部安装标准的正方形板。建模过程花了很长时间，但是在创建原型阶段出现了意外问题。那时，我们的stratasys尺寸破裂，我们不得不在侧面订购印刷品。他们用不溶性支持物和厚层的PLA塑料为我们印制了这种情况。我不得不腻子并手动完成模型很长时间。

然后，当然，我们在翻新的stratasys上打印了其余模型，并完全完成了机箱的设计。随后，3D模型进入工厂。模具已经制成的地方。
