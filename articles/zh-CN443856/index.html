21世纪的金属加工-挑战与机遇


根据分析报告 ，俄罗斯工程产品的特点是在世界市场上的竞争力很低。 原因在于该行业正在遇到的问题。 我们列出了主要的：

  • 2014年开始的国内消费下降；
  • 增加磨损机器的比例；
  • 数控机床的份额落后于发达国家；
  • 企业生产和业务流程的优化和自动化水平较低。

同时，如同一份报告中所述 ，工业化国家的技术举措主要旨在加速向基于物联网（IoT）技术的新一代产品生产过渡，引入自动化系统和大数据分析。 。

该行业的领先企业正在技术（先进的设备和处理技术）以及过程控制系统（MES系统）上进行投资。

在过去三年中，对工业部门最有效的投资是对以下方面的投资：

  • 多功能加工中心；
  • 精密加工市场中使用的五轴/通用加工；
  • 快速更换工具/夹具；
  • 高速加工（HSM）；
  • 用于创建，模拟和检查CNC机器的控制程序的软件（软件）（以下简称UP）。

工程的数字化和向生命周期合同的过渡将使企业能够在解决增加竞争性产品份额的问题方面取得进步。 据估计 ，采用新方法，到2025年，此类产品的产量将从目前的16％增加到30％，到2030年至少增加到50％。

高度自动化的CAD-CAM系统，用于解决工程问题


全球工业界关注的西门子公司正在使用Siemens PLM Software的软件来实施其数字化战略。 根据后者的专家，机械制造企业应解决以下问题以提高竞争力：

  • 确保设备的最大负载并减少设置时间；
  • 基于模板，结合产品零件来实施有关产品和过程的信息收集，以监视和管理工具及配件；
  • 进行3D模型处理路径的仿真，以模拟机床的运动学和对刀具路径进行建模；
  • 减少编程时间，引入用于创建标准元素（例如孔）的UP创建阶段的自动化；
  • 减少处理时间，引入新的处理策略。

正如行业领先公司的做法所表明的那样，始终如一地解决这些问题是效率低下且漫长的过程。 需要集成方法和CAD-CAM系统的实施，该系统控制产品制造的各个阶段-从设计到成品。

生产过程数字化的一个关键特征是不仅能够根据市场需求设计产品的技术和功能特性，而且还能设计生产和运营过程。 为此，正在开发以下产品：物理产品，用于管理产品生产和自动监控的数学（软件）模型（所谓的数字孪生，数字孪生）。

由于引入了系统，开发过程变得更加灵活：设计工程师改进产品，专家优化控制过程，程序员检查策略并选择制造产品的最佳方法。

考虑主要驱动程序，这些驱动程序可减少编程的复杂性，减少CNC机械的处理时间和磨损，并因此导致输出增加。

Solid Edge + Solid Edge CAM Pro：西门子PLM Software提供的CAD-CAM系统


在接下来的网络研讨会上，您将能够看到如何在Solid Edge CAM Pro中创建用于车削和铣削的控制程序。

接下来，我将概述CAM系统的功能和优点。
基于NX CAM的Solid Edge CAM Pro是带有Solid Edge的Siemens PLM Software产品线的一部分。 该软件解决方案提供了广泛的功能-从两轴铣削和高速加工到编程多功能机床和五轴铣削。

CNC机器编程人员可以使用Solid Edge CAM Pro解决具有不同加工要求的问题-铣削，钻孔，车削和EDM。

使用同步技术，您可以直接编辑零件模型，并准备它们以创建CNC机床程序，包括处理盲孔和间隙，偏置曲面以及调整零件元素的大小。
Solid Edge CAM Pro使用主模型的概念，通过将所有CAM功能链接到定义零件几何形状的单个模型来提供NC程序的端到端设计和开发。 结果，程序员无需等待设计人员完成工作就可以开始为CNC机床开发程序。 当模型几何形状更改时，完全关联性为CNC机床提供了控制程序操作的后续更新。

Solid Edge CAM Pro的主要功能


  • 使用PMI-3D模型的设计和技术信息。

    PMI产品制造信息-与CAD中的三维产品模型相关的生产数据。 PMI数据包括几何尺寸和公差（GD＆T），三维标注（文本标记），表面处理质量要求和材料规格。 PMI数据在许多用于共享和可视化产品数据的文件格式（例如PDF和JT）中受支持。 如果设计工程师将其嵌入到模型中，则该数据将与几何数据一起从Solid Edge转换为Solid Edge CAM Pro。 因此，CNC机器的程序员会从设计工程师那里收到所有必要的信息。 这样，您可以避免与2D图纸使用相关的错误和延迟，可以使用端到端产品描述来优化生产过程，并可以基于此数据自动创建控制程序。


  • 基于特征的加工。

    基于元素的处理模块可识别孔，腔，平面（包括从其他CAD系统导入的模型）并创建处理它们的策略。 识别是通过构造元素的参数及其拓扑来执行的。 该模块极大地加快了棱柱形零件的编程，提供了加工的优化，需要较少的操作员技能。 分配加工技术时，该模块会自动识别设计和技术产品信息（PMI）-公差，3D注释，表面清洁度参数。 例如，对于精确的孔，除了钻孔，镗孔或铰孔之外，还将自动添加（此外，您可以配置首选的操作类型）。
    基于元素的处理是编程自动化的主要示例，它可以显着减少创建控制程序所需的时间。


  • 后处理和模拟。

    Solid Edge CAM Pro包含自己的后处理系统，该系统与CAM系统的核心紧密配合。 这使得为​​大多数类型的机器和控制器配置轻松生成所需的控制程序代码。
    该程序包括PostBuilder实用程序，可用于创建和编辑后处理器。 使用该实用程序的图形用户界面，用户可以设置CNC机床所需程序代码的参数。

    后处理器库是Internet资源，其中包含许多支持大量不同机器和工具的进程。 Solid Edge CAM Pro还包括经过优化的Sinumerik后处理器，该后处理器可根据过程操作数据自动选择基本控制器设置。


  • 在机器上进行建模处理。

    Solid Edge CAM Pro系统的主要优点之一是集成的仿真和加工验证功能，使专家可以在CNC机床编程期间检查刀具路径。 同时，可以使用多级验证过程。 例如，基于G代码的仿真显示了运动，该运动受内置NX后处理器上CNC机床程序代码输出的控制。 机器的3D模型以及零件，固定装置和工具会根据G代码根据工具的运动进行运动。


  • 五轴加工。

    主要优点：先进的加工策略，具有用于控制刀具轴的灵活选项，可变轴轮廓可根据底部几何形状自动加工复杂的壁，使用倾斜刀具进行Z型轮廓加工可减少刀具偏斜，从而获得更好的表面光洁度。 对于航空航天和能源行业中使用的复杂零件，Solid Edge CAM Pro提供了灵活的方法以及用于控制五轴加工的刀具轴的一系列选项。 例如，当对带有设计好的壁的多个腔的零件进行编程时，您需要选择一次腔的底部，系统将为壁创建精加工路径。


  • 高速加工-HSM。

    Solid Edge CAM Pro中的高速粗加工支持高速材料去除，同时管理工具负载。 有效的高速铣削HSM策略可以减少加工时间并提高模具，棱柱形和复杂零件的加工表面质量。 用户可以选择多种高速加工策略，以有效铣削淬硬零件，以确保平稳的刀具运动和恒定的切削力。


  • 应用编程。 铣削涡轮组件。

    Turbomachinery Milling模块设计用于编程处理多叶片和多轴旋转零件的CNC机床。 提供了加工带有底切刀片的可能性。 此外，还支持处理多个分隔线，从而可以更有效地处理CAD数据，而无需考虑创建分隔线的系统。 刀片可能包含一个或多个表面。 曲面和曲面叠加层之间的间隙会自动校正。 该系统允许您使用不兼容的参数线在相邻曲面上创建平滑的刀具路径。 为单轮或叶轮的一个元素定义加工操作，然后将其自动应用于零件的其余部分。



使用Solid Edge CAM Pro的数字车间


Solid Edge CAM Pro是“建立”数字车间或计划升级设备的制造商的工具。 使用此系统，用户可以在CNC机床上为Solid Edge中创建的模型和第三方CAD格式创建的模型创建最佳处理程序，从而降低生产成本并提高产品质量。

Solid Edge + Solid Edge CAM Pro的推出是迈向数字化业务流程并提高产品竞争力的重要一步。
