我们用冷信给招聘人员打分

今天，我将告诉您，我如何评价那些用所谓的“冷字母”（这就像“ 冷电话 ”，仅是字母）和工作机会给我写信的招聘人员，以及一些示例。


怎么了


因为我真的很想最终阻止垃圾邮件的传播。 也许这篇文章甚至会以某种方式改变普通招聘人员的工作方式。 好吧，至少我希望我个人会收到质量稍好的信件。 （ 显然，作者希望招聘人员在写这篇文章之前先在他的博客上阅读这篇文章-大约翻译。 ）


如果您碰巧在IT行业工作了一段时间，那么很可能会遇到来自不同规模的招聘人员的连续“冷酷信件”。 这些电子邮件大多数都是通过LinkedIn获得的。 有时，招聘人员会以某种方式找到您的电子邮件并直接写信。 在大多数情况下，冷字母是非常标准的，看起来像这样：


你好％用户名％！
我从事员工招聘，我们的职位空缺很大。
您的个人资料对我来说似乎很有趣<...>
如果您有兴趣，请回答我<...>附上简历<...>，我们将讨论详细信息

实际上，可能有很多选择，但是比喻地讲，所有选择都患有相同的疾病。 因此，为了评估与招聘者之间的沟通，我决定开发一个积分系统，以便在同一类型的信件流中找到对我感兴趣的同一雇主。 这是我的方法：


奖励积分


+ 1使用直接电子邮件，而不使用LinkedIn内部Messenger。 人们为什么重新发明交流方式？ 我不明白 不，我了解LinkedIn希望访问者在其网站上停留更长的时间，但是坦白地说，使用我的工作电子邮件并不难，这表明招聘者已经研究了至少一点我没有对角查看LinkedIn个人资料（ 但就我而言，它的味道-大约翻译 ）；


+ 1每当有一封信提及我在GitHub上的项目或任何我的文章时，我都会给1分。 业余时间我会做开源，因为我喜欢开源。 在一封信中提到一个开源项目，就像对一个小孩的父母说：“上帝，他只是一个奇迹！” -简单，但有利于沟通；


+ 3对于我的每个项目，招聘公司都认为这可能对自己有用 。 这里的一切都很认真：他们不仅认真地看了我的项目，还欣赏了其中的好处。 因此，他们不仅要雇用我来关闭某种过往职位，而且还要出于某种目的。


+ 2如果招聘人员是我曾经与之共事的人的推荐。 这里的计算是，这位前同事最有可能知道我感兴趣的工作是什么，并且不会提出不好的建议。


+ 2该信至少包含有关招聘公司的一些细节（不仅是姓名），还是对拟议工作的详细描述；


+ 1这封信是由招聘公司的员工写的（ 不是由公司的招聘人员写的。人力资源部门的员工是否包括在此类别中-待解决的问题-大约需要翻译 ）；


+ 3这封信是由CTO或招聘公司工程部门负责人亲自写的。


罚分


是的，除了奖励积分之外，一封信件可能还会受到处罚。


- 10字母包含模板字符串 。 例如，由于软件错误，可能会发生这种情况。 好吧，精神上的东西：


您好<％FirstName％>，
我对您的个人资料非常感兴趣...

<% FirstName %>忘记输入我的名字了。


- 1数据已正确插入模板中，但是消息的模板部分和手写部分之间的差异太明显；


- 5与招聘人员的往来信件包含“我对您的个人资料感兴趣”以及“发送您的简历和指向LinkedIn的链接”。 嘿伙计，这是怎么回事？


- 1冗长的“在快速成长的公司中提供了很棒的职业机会”，而不是公司名称。 我了解到，雇用公司的名称是隐藏的，以保护自己免受其他招聘人员的侵害，但是老实说，我不在乎。


- 1如果信件包含库存图片或视频。 这是我浪费时间阅读一封信。 此外，它清楚地表明缺乏个体方法；


- 1他们在第一个字母中要求放弃当前的简历。 听着，您很长的半模板文字使您的履历表很荣幸。 为此，我有一个LinkedIn个人资料和我的个人网站。 与静态PDF相比，它以更完整，美观和互动的方式包含有关我的信息；


- 2如果冷封信中包含“需要学士学位”的精神。 看，我是计算机科学博士学位。 我并不是要炫耀它，也不是让别人自称为“格莱布医生”。 目前，我什至不从事科学工作-我不写文章，也不从事研究。 但是，任何熟悉我的个人资料的人都不会在信中包含有关学士学位的信息。 对我来说，这就像在喊“模板！” 就在耳边；


- 1如果一组必需的技能包括对我来说显然不有趣的技术。 好吧，说真的，您只需查看我博客中的标签云（右，右）就可以了解我感兴趣的内容（ 指向原始文章的链接的右侧确实有一个标签云-大约是Transl。 ）；


- 1如果一组必需的技能包括我很擅长的技术（在我的情况下为NodeJS，AngularJS，JavaScript，QA等）；


- 2如果我在第一个字母的底部发现有人推荐这个职位，招聘人员将获得最后的罚款。 这与邀请一个女孩约会，然后在不等待答案的情况下，询问她是否有任何准备好去的朋友一样，以防该女孩本人不同意。 如果招聘者等待我拒绝，然后才要求推荐空缺，那么他将获得-1分而不是-2 。 我不会做别人的工作，我当然不会急着问我知道的每个人是否愿意利用“惊人的职业机会”。


关于最后罚款的简短评论。 如果您只是希望我好运并请我保持联系，而不是迷路，那么可以重播此刻并获得+1 。 至少这是合乎逻辑的：每个人都将换工作，而成功的招聘则是建立和维持关系。 因此，除了征求建议外，请记住并记住我的信，并在确切的时间写信给您 。


即使礼貌地说“谢谢，不感兴趣”，我也会回复您的消息吗？ 如果您没有获得至少5分，那是不可能的。 难吗 让我们看几个例子。
