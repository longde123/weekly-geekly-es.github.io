Windows XP ist jetzt endlich offiziell tot


Jeder mochte den XP-Suchhund, oder?

Die meisten Benutzer haben Windows XP vor mehr als 5 Jahren begraben. Aber treue Fans und Geiseln des Ökosystems zusammen nutzten dieses Betriebssystem weiterhin und versuchten verschiedene Tricks, um seinen vegetativen Zustand aufrechtzuerhalten. Aber die Zeit verging und Windows XP erreichte endlich das Ende der Straße, da seine letzte noch unterstützte Option - POSReady 2009 - nicht mehr offiziell unterstützt wird.

Der Punkt ohne Rückkehr wird überschritten.


Screenshot von neowin.net .

Windows Embedded POSReady 2009, das, wie der Name schon sagt, in erster Linie dazu dient, Anwendungen zu starten, die die Aufmerksamkeit der Kunden mit Ausrufen wie "Free Cash!" Im April 2019 verlor das Unternehmen endgültig seine offizielle Unterstützung, was das absolute Ende einer beeindruckenden Lebensdauer für eine derart große Familie von Betriebssystemen bedeutete.

Der britische Einzelhändler Boots zeigt einen alten Windows XP-Anmeldebildschirm auf einem Self-Service-Terminal in seinem Geschäft in Islington:

Foto eines Windows POSready 2009 Outlets theregister.co.uk

Das vom Register-Reader entdeckte POS-Terminal zeigt freudig die alte XP-Anmeldeseite an, obwohl die Mitarbeiter einen umgekehrten Korb vor die Maschine stellten, damit die Kunden ihn nicht berühren konnten.

Windows XP wird nicht mehr unterstützt. Einige Veröffentlichungen verzögerten sich jedoch um Jahre nach dem offiziellen Todesdatum. Embedded Standard 2009 wurde im Januar endgültig deaktiviert und die erweiterte Implementierungsunterstützung in Form von Embedded POSReady 2009 endete am 9. April.

Einige Tage zuvor, am 5. April 2019, veröffentlichte Microsoft das neueste Update für „Last of the Mohicans“ mit KB4487990, mit dem Zeitzonen für Sao Tome und Principe sowie Kyzylorda in Kasachstan festgelegt wurden.

Danach herrschte totale Stille. Das Unternehmen hat alle lebenserhaltenden Systeme ausgeschaltet. Der Patient ist tot und wird nie wieder aus dem Koma kommen.

Die weltweite Unterstützung für die meisten Windows XP-Varianten endete leider bereits 2014 unter lauten Schreien und Zähneknirschen, als Unternehmen erkannten, dass sie plötzlich von ihrer üblichen Plattform irgendwohin gehen mussten. XP ist seit 2001 für die Installation verfügbar. Aufgrund der Tatsache, dass viele das katastrophale Vista verpasst und den Trend zur Ablehnung der Aktualisierung festgelegt haben, blieb eine erhebliche Anzahl von XP-Workstations bis heute am Leben.

Glücklicherweise war Windows 7 so gut, dass die meisten Leute einfach damit anfangen konnten. Aber nicht alle. Windows XP war das erste Betriebssystem, das ernsthaft unter einem Problem namens "Zu gut zum Beenden" litt, das dazu führte, dass Endbenutzer das Upgrade lange genug verweigerten, nachdem das Betriebssystem seine goldenen Jahre hinter sich hatte.

Dies war teilweise eine Reaktion auf Windows Vista und teilweise das Ergebnis, dass XP im Flaggschiff-Status ungewöhnlich lang war. Als Bill Gates beschloss, sich auf die Sicherheit in Windows XP Service Pack 2 (SP2) zu konzentrieren, hat Microsoft trotz einer Reihe bedeutender Änderungen keine neue Version des Betriebssystems umbenannt oder veröffentlicht. Vor XP sortierte Microsoft schnell seine Consumer-Betriebssysteme: 1992 - Windows 3.1, dann - Windows 95, 98 und Windows 2k und ME im Jahr 2000. Die Amtszeit von XP erwies sich als viel länger - es war mehr als fünf Jahre lang (2001-2006) das Hauptbetriebssystem von Microsoft und war bis zur Veröffentlichung von Windows 7 im Jahr 2009 weiterhin auf dem Markt beliebt.

Diese Langlebigkeit kann manchmal nur ärgerlich sein. Nachdem Windows XP in den letzten Tagen der Popularität von Disketten zutage getreten war, litt es sofort unter einem der inhärenten Mängel - der Unfähigkeit, die Disk-Controller-Treiber manuell von etwas anderem als einer Diskette zu laden, was bereits am Ende der Lebensdauer dieses Betriebssystems bei Verwendung von Controllern mit AHCI oder sehr ärgerlich war RAID (Ja, es war möglich, Treiber in neue Kopien von Windows XP zu integrieren, die auf die Festplatte geschrieben wurden. Es war jedoch ziemlich unangenehm, ständig neue Kopien von Windows XP zu schreiben.)

In der XP-Ära hat Microsoft einige Lektionen darüber gelernt, wie schwierig es ist, Benutzer dazu zu bringen, von der Version des Betriebssystems zu wechseln, die sie geliebt haben, und die sie weiterhin auf die neue Version verwenden möchten. Höchstwahrscheinlich war es diese Erfahrung, die dazu führte, dass Windows 10 als ewiges, ständig aktualisiertes Produkt betrachtet wurde - obwohl Microsoft auch mehrere Schritte unternommen hat, um diese Richtlinie zu schwächen und den Benutzern zusätzliche Kontrolle zurückzugeben.

Einige große Benutzer, wie die britische Regierung, haben die Flammen des Lebens im Sterben von Windows XP nach 2014 aufrechterhalten und Microsoft erhebliche Pfund Sterling für den personalisierten Zugriff auf Updates gezahlt, während andere festgestellt haben, dass sie das veraltete Betriebssystem ihres Computers „maskieren“ Unter "POSReady" können Sie mit Hilfe bestimmter Änderungen in der Registrierung für einige Zeit Sicherheitsupdates erhalten.

Trotz der Tatsache, dass veraltete (aus Sicherheitsgründen) Computer mit Windows XP ein fruchtbarer Boden für die Verbreitung von Viren blieben, haben Computer mit diesem Betriebssystem in einigen Fällen die Pläne von Angreifern tatsächlich vereitelt. Zumindest war dies während eines der jüngsten Ausbrüche von WannaCry-Malware im Jahr 2017 der Fall, als festgestellt wurde, dass Computer mit XP zu oft in das BSOD fielen und „ vorgaben, tot zu sein “, was die Verbreitung des Virus verhinderte, dessen Exploit nicht wie gewünscht funktionierte ".

Ungepatchte Computer mit Windows 7 wurden zum Hauptziel für Hacker, die besonders von Marcus Hutchins verwirrt waren, der den globalen „Schalter“ der WannaCry-Epidemie fand.

Es sei daran erinnert, dass Microsoft für Windows 7 einen Termin für die Ausführung im Jahr 2020 festgelegt hat, der gleich um die Ecke liegt.

Microsoft freut sich, ein Upgrade auf Windows 10 oder Windows 10 Pro für POSReady 2009-Computer vorschlagen zu können. Es ist jedoch unwahrscheinlich, dass die für Benutzer verfügbare Hardware davon profitiert, da sie aufgrund der gestiegenen Systemanforderungen ersetzt werden muss.
