Que tengas una buena lectura!



Noticias y lanzamientos


  • PHP 7.3.7
  • PHP 7.2.20
  • PHP 7.4.0 alpha 3 - La última versión alpha de lo planeado . La congelación de funciones debe ser el 22 de julio, por lo que la lista actual de cambios solo puede complementarse con aquellos para los cuales la votación comenzó a más tardar el 8 de julio. Total ahora en la votación:
    • Desaprobaciones para PHP 7.4
    • Escape PDO "?" marcador de posición de parámetro
    • Desaprobar la sintaxis de llaves para acceder a elementos de matriz y compensaciones de cadenas
    • Agregue str_starts_with (), str_ends_with () y funciones relacionadas .

    También un par de cambios nuevos sin RFC separados:
    • proc_open aceptará argumentos en forma de matriz, no es necesario que se escapen y el proceso comenzará directamente sin un shell intermedio: 
     proc_open(['php', '-r', 'echo "Hello World\n";'], $descriptors, $pipes);

    • En PHP 7.4, se generará un aviso cuando se intente utilizar tipos escalares como matrices incorrectamente. Anteriormente, tales llamadas simplemente devolvían null : 
     $null = null; var_dump($null[0]); // NULL $int = 42; var_dump($int['foo']); // NULL var_dump(true[0] === false[0]); // true

Las herramientas


  • humbug / php-scoper : la herramienta le permite agregar un prefijo a todos los espacios de nombres en la carpeta especificada en un solo paso. Puede ser útil cuando necesite usar varias versiones de un paquete, un ejemplo de uso para los complementos de WordPress .
  • CaptainHookPhp / captainhook - Git hook manager para proyectos PHP.
  • sebastianfeldmann / phpbu : una herramienta para crear copias de seguridad de archivos y bases de datos. Capaz de cifrar copias de seguridad y enviar al almacenamiento.
  • localheinz / http-method : un conjunto de constantes con los nombres de los métodos HTTP.
  • spiral / roadrunner : servidor de aplicaciones para PHP en Golang. Se agregó soporte para FastCGI, configuraciones CORS listas para usar, métricas integradas en Prometheus. Una revisión detallada en el informe del autor .
  • leocarmo / circuit-breaker-php - Implementación del patrón Circuit Breaker para PHP. Alternativa: ackintosh / ganesha .

PHP asíncrono


  • ReactPHP 1.0 LTS - Hace 7 años, se lanzó la primera versión de ReactPHP v0.1.0. Y ahora, el equipo del proyecto ha anunciado las versiones 1.0 de los paquetes meta reactphp / dns , reactphp / cache y reactphp / react . Por lo tanto, todos los paquetes principales han alcanzado una versión estable. El lanzamiento está marcado como LTS y, por lo tanto, será compatible durante al menos dos años.
  • Escribimos una API RESTful usando ReactPHP: Agregar rutas , Refactorizar controladores , Probar puntos finales , Agregar manejo de errores
  • Usando ReactPHP para ejecutar los comandos de Drupal
  • swoft-cloud / swoft 2.0 : un marco basado en Swoole para implementar microservicios inspirados en Spring Cloud de Java. Implementa PSR-7, PSR-14, PSR-15 y usa AOP. Publicación introductoria en soporte.

Seguridad


  • nbs-system / snuffleupagus : la extensión bloquea la ejecución de código potencialmente inseguro en tiempo de ejecución. Acerca del uso para prevenir vulnerabilidades publicadas recientemente.
  • paragonie / argon2-refiner : la herramienta lo ayudará a elegir los parámetros óptimos para el hash utilizando el algoritmo Argon2id agregado en PHP 7.3 .
  • Utilizamos los matices del analizador de consultas en PHP para evitar los sistemas y prevenir / detectar intrusiones en la red.
  • Magento 2.3.1: XSS almacenado no autenticado a RCE : análisis de vulnerabilidades en Magento (<2.3.1) que permitieron a un usuario no autenticado secuestrar una tienda y redirigir los pagos.
