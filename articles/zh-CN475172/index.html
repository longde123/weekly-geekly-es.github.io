在第二部分中 ，讨论了使用Raspberry Pi的5种方法。 事实证明，这种微型计算机还具有更多功能。 现在，我们希望考虑Raspberry Pi的另外5个有趣的功能。

首先，我将回答上一部分提出的问题-最好选择哪种型号的Raspberry Pi。 实际上，不同的版本具有不同的性能，在Google中很容易找到测试结果：



如您所见，在现代机型中，最快的是Raspberry Pi 3 B +和最新版本的Raspberry Pi 4.Pi4的唯一缺点是它具有较为罕见的Micro HDMI连接器，因此3B +机型在这方面更为通用。 如果您需要超紧凑型机型，则可以考虑使用Raspberry Pi Zero W，但总的来说，我不建议这样做。 首先，从测试中可以看出，它明显较慢，其次，Zero W具有（再次！）另一个视频连接器（Mini HDMI），没有以太网，并且需要USB适配器连接键盘或鼠标。 仅当您确切知道为什么需要超紧凑型机型时，才应采用零瓦-常规版本（Pi 3B +或Pi 4）既更快又更通用。

现在，我们将继续。 因此，有5种使用Raspberry Pi的方法。

1.信息亭



来自newscrewdriver.com/2019/08/10/raspberry-pi-web-kiosk-boots-raster-on-raspbian-than-ubuntu-core的照片

最新版本的Raspbian OS具有完全现代化的Chromium浏览器，可以正确显示甚至复杂的网页。 因此，如果您需要在Jira上显示Jira的票证，公司股票的报价，火车时刻表或至少天气预报，Raspberry Pi将完全解决此任务。

它非常简单，不需要任何编程，您只需要在信息亭模式下运行Chromium，隐藏鼠标光标并关闭屏幕保护程序即可。

对于那些感兴趣的人，有一些教程：

  • pimylifeup.com/raspberry-pi-kiosk
  • blockdev.io/raspberry-pi-2-and-3-chromium-in-kiosk模式

2. LoRaWAN网关




如果您需要LoRaWAN无线网络以通过低功率LoRa无线电调制解调器接收数据，则可以基于Raspberry Pi制作数据。 有不同价格类别的选项，单通道网关更便宜，而8通道则更贵。 根据网关的“技巧”，发行的价格大约为50到500美元。

对LoRa的详细研究超出了本文的范围，无法进行更详细的研究：

  • learn.adafruit.com/lora-and-lorawan-radio-for-raspberry-pi
  • www.cooking-hacks.com/documentation/tutorials/lorawan-for-arduino-raspberry-pi-waspmote-868-900-915-433-mhz
  • www.thethingsnetwork.org/docs/gateways/start/build.html

3.互联网广播和互联网电视




Raspberry Pi上的音频播放主题非常广泛-您可以将3.5美元的中文USB声卡，35美元的带有镀金连接器的HiFi屏蔽和一个带扬声器的板连接到该设备，如上图所示。 也很容易找到带有光输出的板，这将为旧的音频系统甚至增加现代的“网络”功能。

至于软件，也有很多选择。 您只能硬编码并播放一个Internet广播电台，为您心爱的祖母像个“无线电点”，您可以添加LCD显示屏，按钮等。可以从Spotify进行订阅。

有兴趣的人可以查看以下教程：

  • www.jan-holst.dk/pi-radio/pi-radio.html
  • www.hackster.io/Granpino/raspberry-pi-internet-radio-and-mp3-player-with-bluetooth-1aa591
  • learning.pimoroni.com/tutorial/sandyj/streaming-spotify-to-your-pi
  • pimylifeup.com/raspberry-pi-spotify

对于IP-TV，建议使用专门用于媒体中心的Kodi软件。 我个人没有尝试设置IP-TV，也没有付费订阅，但是对于那些住在国外但想看俄语电视的人来说，这可能是有意义的。

有兴趣的人可以参考以下教程：

  • pimylifeup.com/raspberry-pi-kodi
  • www.raspberrypi.org/documentation/usage/kodi

您还可以将IR接收器连接到Raspberry Pi，这使您可以使用遥控器。

4.调频发射机



图片来自circuitdigest.com/microcontroller-projects/raspberry-pi-fm-transitter

鲜为人知的是，但是通过在Raspberry Pi上以编程方式更改PLL频率，您可以在不使用任何其他硬件的情况下传输高达500 MHz范围内的无线电信号-如图所示，只需将一根电线连接到该引脚，然后启动程序即可。 当然，这里将没有HiFi品质，输出功率仅为10mW，但是对于测试任何无线电设备或仅用于与朋友或邻居玩耍，这可能会派上用场。 10mW的功率足以在房间内的FM收音机上播放音乐。

几年前，我在有关geektimes的文章中更详细地描述了该技术，尽管今天该文章已经过时，但可以在此处找到更好的传输程序。

5.无人机飞行控制器




Raspberry Pi的下一个项目同样有趣。 许多人可能知道多旋翼飞机的控制器，例如MultiWii，Arducopter或DJI。 可以基于Raspberry Pi制作类似的控制器。 NAVIO2项目是一种特殊的防护罩，可以控制电机调节器，并在板上装有必要的传感器（加速度计，气压计，GPS等）。

很难说这样的一堆要比“普通”的Arducopter / Pixhawk好多少，但是对于许多任务，这个想法可能是非常有前途的。 首先，您可以通过3G / 4G调制解调器从此类直升机控制或发送遥测（甚至是图像），这大大增加了链接的范围。 其次，Raspberry Pi可以运行非常复杂的算法，例如卡尔曼滤波器或其他一些数学运算。 我个人没有尝试过使用此控制器，所以我无法更详细地说，任何人都可以在Youtube上搜索航班视频。
