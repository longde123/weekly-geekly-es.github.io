So erstellen Sie einen QR-Code für die Zahlung in einem Zahlungssystem

Heute möchte ich echte Geschichten aus dem Leben erzählen. Ich hoffe, dass viele der Leser, die mindestens einmal Stromrechnungen oder Quittungen bezahlt haben, meine bescheidene Arbeit mit ihren Sorgen teilen und, ohne sie als Arbeit zu betrachten, eine zum Karma hinzufügen. Ich möchte die anderen Glücklichen warnen, dass dieses Ereignis in Ihrem Leben immer noch besteht vor.

Quittungen für die Zahlung aller Arten von Utensilien warten buchstäblich bei jedem Schritt auf einen modernen Menschen. Vor relativ kurzer Zeit ist ein neuer Dienst erschienen, bei dem Sie Quittungen anhand ihrer Nummer bezahlen können. Mit angehaltenem Atem habe ich mir diese Datenmengen immer angesehen, denn damit dies funktioniert, muss die Stelle, die die Quittung ausgestellt hat, diese Daten der Bank oder einer API zur Verfügung stellen. Nach den neuesten Trends zu urteilen, wurde diese Technologie durch QR-Codes ersetzt, die alle erforderlichen Informationen enthalten und keinen vorläufigen Datenaustausch von Institutionen erfordern. Eines schönen Tages erhalten Sie eine Quittung.



Sie müssen lediglich die Bankanwendung öffnen und den QR-Code scannen. Dann füllt das Programm alle notwendigen Details für Sie aus und Sie müssen nur Ihre Zahlungsabsicht bestätigen. Angesichts der Tatsache, dass es viele Belege gibt, insbesondere unter Bedingungen, in denen Versorgungsunternehmen dazu neigen, in den Punkt-zu-Punkt-Kommunikationsmodus zu wechseln und jeder Dienst seine eigene Rechnung ausstellt. Der Gesetzgeber verkompliziert sein Leben durch die Einführung neuer Standards - Kapitaleinlagen erfordern ein separates Konto und dergleichen - alles auf dieser Welt beginnt sich um eine Quittung zu drehen. In Erwartung einer solchen Situation beginnen die relevanten Organisationen darüber nachzudenken, wie sie mit solchen Dienstleistungen Kapital verdienen können. Und an einem der folgenden schönen Tage kommt eine Quittung an.



Neo ist es gewohnt, am letzten Tag alles wie gewohnt zu bezahlen und bemerkt plötzlich, dass sich etwas in der Matrix geändert hat. Ohne auf einen bitweisen Vergleich zurückzugreifen, stellen wir schnell fest, dass der QR-Code verschwunden ist. Wenn wir uns an den Vorsitzenden der HOA wenden, erhalten wir eine Erklärung, dass der QR-Code eine kostenpflichtige Dienstleistung ist, die auf 50 Rubel geschätzt wird. Das heißt, Damit die Quittung mit einem QR-Code versehen werden kann, müssen Sie für jede Quittung 50 Rubel bezahlen. In der Geschichte wird nichts über die Namen realer Organisationen gesagt. Testen Sie diesen Code also nicht - er ist fiktiv, aber lassen Sie uns herausfinden, was in diesem Code enthalten ist. Dieser Standard wurde von der Non-Profit-Partnerschaft des National Payment Council entwickelt, die vom Technischen Komitee für Normung TK122 „Financial Transaction Standards“ eingeführt wurde und von der Verordnung der Bundesagentur für technische Regulierung und METROLOGIE genehmigt und eingeführt wurde (ich unterstreiche das Wort NON-PROFIT-Organisation). Der Code enthält obligatorische und optionale Parameter.
