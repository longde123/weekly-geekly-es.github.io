乔恩·冯·泰纳（Jon von Techner）：Google故意将新的Vivaldi浏览器放在轮子上，因为它曾经“粉碎”了Opera


von Techner认为，这些链接是应Google的要求添加到该站点的。

Opera网络浏览器的创建者Jon von Techner 提出了针对Google的指控 。 据他说，这家搜索巨头特别阻碍了其新浏览器Vivaldi的普及。 这位工程师说，由于垄断了搜索和广告，谷歌“显示出无法抵制滥用权力的诱惑”。 乔恩·冯·特纳（Jon von Techner）在正式的维瓦尔第（Vivaldi）博客上发表的文章“ 我在Google上的朋友们：现在该回到不再邪恶的时代 ”中解释了他的立场。

乔恩·冯·特纳（Jon von Techner）写道，他认识Google已经很长时间了。 他回忆说，在Opera中，他们是第一个将Google搜索添加到浏览器界面的人。 当时，Google确实是一个极客公司，其创始人拉里（Larry）和谢尔盖（Sergey）给真正的热情家伙留下了愉快的印象。 从某种意义上说，Opera甚至帮助Google成为了一个相当流行的浏览器，而Google –一个新的未知搜索引擎也开始了。

但是随着时间的流逝，情况发生了变化。 Google开始发展，与Mozilla接近并集成到Firefox，并推出了诸如Google Docs之类的新服务。 von Techner写道：“然后，我们看到了Google的阴暗面。不仅这些新服务与Opera不兼容，而且它们直接鼓励用户更换浏览器。” 为了数百万用户，Opera必须隐藏其用户代理，这是Google服务在挪威浏览器中正常运行的唯一方式。 但是情况只有在Google发布Chrome浏览器时情况才会恶化。

2011年，由于与董事会和管理层的分歧，Jon von Techner离开了Opera Software，并于2014年8月创立了Vivaldi Technologies，后者启动了新的Vivaldi浏览器。 它承诺包括版本15之前Opera浏览器中提供的许多功能。 许多功能（例如侧栏和网页上的图像显示模式）将在Opera 12（Opera的最新经典版本）之后直接建模。 不幸的是，仍未实现所有计划，因此Vivaldi在许多方面都类似于常规的Chrome克隆。

Vivaldi浏览器基于Chromium，并在WebKit和KHTML引擎上运行。 这些都是Google领导的免费项目。 您会认为，现在下载Google服务不会出现问题。 但是现实是不同的。 von Techner认为，他们仍然必须隐藏用户代理，而Google的情况“已达到新的谷底”。

现在，谷歌开始向竞争对手施加压力，不仅封锁了浏览器本身的服务，而且还封锁了维瓦尔第的Google AdWords帐户，而没有发出警告。 为了公平起见，Google代表表示，为了恢复Google AdWords中的帐户，Vivaldi必须更改其网站的内容（如第一个屏幕截图所示）。

经过近三个月的协商，AdWords帐户已恢复，但是Jon von Techner认为这些事件并非偶然。 他怀疑Google出于各种原因故意将其放在轮子上并阻塞了竞争对手。

这位冰岛程序员甚至猜测为什么Google会在5月底封锁他们的帐户。 巧合的是，这是在他在奥斯陆自由论坛上发表演讲两天后发生的。在那次演讲中，他批评Facebook和Google过度监视用户并大量收集个人数据。 然后他呼吁禁止位置追踪 ，该追踪由Facebook和Google实施。 两天后，谷歌禁止了他的帐户。 “这仅仅是一个巧合吗？ -他问一个问题。 “或者是故意向我们发送消息吗？”

Vivaldi的创始人向《连线》记者展示了他们从Google发给他的信的副本。 他们要求Vivaldi在“下载程序的每个按钮内”添加指向EULA（用户许可协议）的链接。 von Techner写道，这是一个相当奇怪的要求，因为Google本身并不总是遵守它 。



公平地讲，应该注意的是Google没有义务遵守AdWords的这一要求，因为其页面上没有AdWords广告，并且确实由于这个原因，许多AdWords帐户被阻止了，包括那些不以任何方式与Google竞争的公司的帐户。

乔恩·冯·特纳（Jon von Techner）说，问题不在于特定的要求本身，而在于Google提出要求的形式：激进地，无警告地进行阻止，即使Google本身没有满足要求。 程序员遗憾地指出，现在的Google不再像以前那样令人讨厌，积极向上。
