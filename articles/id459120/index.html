Komputer Tertanam Modular Seri UNO-1000/2000



UNO 1000/2000 adalah komputer X86 lengkap untuk sistem embedded, dalam bentuk faktor dari modul yang siap pakai, dengan kemampuan untuk menginstal di atas meja, dinding, DIN-rail, atau VESA. Platform ini mudah digunakan untuk tugas-tugas tertentu, berkat berbagai pilihan modul ekstensi iDoor eksklusif yang memungkinkan Anda untuk menambahkan antarmuka yang diperlukan ke perangkat.

Fitur utama garis:

  • Pendinginan yang sepenuhnya pasif - tidak adanya bagian yang bergerak, kipas dan lubang ventilasi radiator. Debu tidak menumpuk di dalam perangkat, mereka cenderung membutuhkan perawatan.
  • Arsitektur X86 lengkap - tidak perlu mem-port program-program lama, seperti halnya arsitektur MIPS atau ARM. Dimungkinkan untuk menginstal versi desktop Windows.
  • Port I / O terintegrasi - Port I / O digital terintegrasi untuk menghubungkan periferal
  • Antarmuka RS-232/485/422 built-in - tidak perlu menghubungkan adaptor serial eksternal
  • Modul ekspansi IDoor - lebih dari 30 jenis kartu ekspansi bermerek memungkinkan Anda untuk memasang konfigurasi yang tepat
  • Tegangan pasokan dari 9V ke 36V DC - berbagai tegangan pengoperasian memungkinkan Anda untuk memasang perangkat di sistem yang ada, tanpa konverter.

Seri UNO-1000




Seri UNO-1000 dirancang untuk pemasangan pada rel DIN. Di jajarannya ada dua model dengan prosesor Cortex A8, berdasarkan arsitektur ARM. Tujuan utama dari seri ini adalah untuk menginstal sebagai server kompak di lemari kabel untuk pemrosesan atau otomatisasi data lokal.

UNO-1251G




Perangkat paling ringkas di jajaran berdasarkan arsitektur ARM. Ini memiliki layar OLED yang dapat diprogram untuk menampilkan pesan status sewenang-wenang, bus CAN untuk menghubungkan ke peralatan otomotif, serta 2 port LAN dan 3 port serial.

UNO-1251G / 1252G dapat dilengkapi dengan modul nirkabel iDoor Wi-Fi / 3G / 4G dan memiliki port kartu SIM yang dapat diakses dari luar.

UNO-1372G



Penampilan komputer UNO-1372G
Komputer memiliki dua input daya, untuk koneksi primer dan cadangan. Dalam hal terjadi kegagalan daya utama, transisi ke cadangan terjadi secara instan, tanpa me-reboot. Didukung beberapa port LAN ke dalam satu antarmuka tim yang didukung, untuk meningkatkan toleransi kesalahan. Jika salah satu antarmuka gagal, jaringan akan terus beroperasi secara normal.

Port I / O



Input dan output port digital untuk menghubungkan relay, sensor, dan periferal eksternal lainnya.
UNO-1372G memiliki 4 input dan output digital untuk mengendalikan periferal eksternal, seperti sensor, relay, reed switch. Anda dapat bekerja dengan port I / O dari sistem operasi, di tingkat program pengguna.

Seri UNO-2000




Seri UNO-2000 dirancang untuk pemasangan di sistem tertanam, objek bergerak, mobil, kereta api. Casing dirancang untuk pemasangan pada permukaan datar dengan sekrup pengunci. Komputer memiliki prosesor yang lebih kuat daripada garis UNO-1000 . Beberapa model memiliki enkripsi perangkat keras bawaan dan modul penyimpanan kunci TPM untuk melindungi aplikasi penting.

Memori cadangan FRAM


Memori cadangan FRAM memungkinkan Anda menyimpan data penting untuk melindungi dari kehilangan jika terjadi crash atau macet. Bekerja dengan memori terjadi dengan cara yang sama seperti dengan RAM, tetapi nilainya tidak hilang jika terjadi pemadaman listrik atau kegagalan fungsi lainnya. FRAM dapat menyimpan penghitung penting. Setelah pemulihan sistem, program akan terus bekerja dengan data yang sebelum kegagalan.

Modul Platform Tepercaya (TPM) 2.0


Komputer dari seri UNO-2000 , misalnya, model UNO-2484G , memiliki modul TPM 2.0 terintegrasi untuk perlindungan perangkat keras dari data penting. TPM dapat digunakan untuk boot OS tepercaya (Boot Tepercaya), penyimpanan kunci enkripsi, serta generator nomor acak.

Watchdog untuk perlindungan terhadap pembekuan


Untuk melindungi dari pembekuan program dan sistem operasi, komputer UNO memiliki pengatur waktu yang dapat diprogram perangkat keras (watchdog). Penghitung waktu terus-menerus mengharapkan sistem untuk melaporkan bahwa ia beroperasi secara normal. Setiap pesan baru me-reset timer. Jika penghitung waktu tidak menerima pesan dalam waktu tertentu, sistem dianggap beku dan dapat di-boot ulang secara otomatis.

Cara bekerja dengan anjing penjaga


Windows menggunakan driver level kernel (KMDF) dan perpustakaan untuk menggunakan API dari ruang pengguna untuk bekerja dengan timer. Juga, timer dapat beroperasi sebagai layanan sistem untuk memantau sistem operasi.

Perangkat pengawas pengawas Windows

Saat menggunakan watchdog sebagai layanan sistem, konfigurasinya tersedia di panel kontrol. Anda dapat mengatur interval polling dan mode operasi.

Layanan Watchdog tersedia di panel kontrol


Mengatur mode operasi dan interval pemungutan suara dari Watchdog timer

Pabrikan menyediakan pustaka untuk bahasa C ++, yang memungkinkan menanamkan dukungan anjing penjaga dalam program untuk melacak kegagalan di dalam perangkat lunak yang tidak terlihat oleh sistem operasi. Misalnya, anjing penjaga dapat membantu Anda melacak ketika suatu proses telah dibekukan, tetapi belum selesai.

Modul Ekstensi IDoor



UNO-1251G dengan Modul Ekspansi Port COM Terpasang
Fungsionalitas komputer UNO dapat diperluas menggunakan modul ekspansi iDoor eksklusif. Modul-modul ini adalah papan mPCIe, dengan loop untuk menghasilkan antarmuka ke panel depan. Tidak seperti kartu PCIe konvensional untuk casing yang kompatibel dengan ATX, modul iDoor dipasang dengan kokoh pada case dan menahan beban mekanis yang signifikan pada port antarmuka, karena mereka dipasang di beberapa titik.

Modul Komunikasi 3G / 4G / GPS



Modul PCM-24S34G adalah 2G / 3G / 4G dalam format modul iDoor, dengan unit antena eksternal. Ini mendukung pita LTE B1, B2, B3, B4, B5, B7, B8, B17, B20, dan kompatibel dengan jaringan 2G (GPRS / EDGE). Bluetooth dan Wi-Fi


Modul PCM-24S2WF-BE menambahkan dukungan untuk protokol nirkabel Bluetooth 4.1 dan Wi-Fi a / b / g / n / ac nirkabel dengan dukungan untuk pita 5 GHz.

Antarmuka Industri



Dalam faktor bentuk modul iDoor, ada banyak opsi untuk antarmuka industri yang dipatenkan, seperti PROFIBUS, EtherCAT, PROFINET, dan lainnya.

Kesimpulan


Komputer modular UNO-1000/2000 memungkinkan Anda untuk mengonfigurasi serangkaian opsi secara mandiri, tergantung pada kebutuhan pelanggan, setelah menerima perangkat yang lengkap dan ringkas. Arsitektur penuh X86 memungkinkan Anda untuk menjalankan perangkat lunak yang ada dalam sistem embedded yang kompak. Tidak adanya kisi-kisi pendingin dan ventilasi aktif meningkatkan masa pakai perangkat dan tidak memerlukan perawatan yang sering.
