Microsoft Menghabiskan Dua Serangan Paten di Linux dalam Seminggu



Pekan lalu, banyak outlet media menulis tentang inisiatif open source Microsoft yang luar biasa - pengumuman SQL Server untuk Linux , rilis OS untuk perangkat jaringan berbasis Debian (yang sebenarnya merupakan ulangan dari berita lama ), dan masuknya ke Eclipse Foundation .

Secara kebetulan atau tidak, tetapi acara-acara ini menaungi diri mereka sendiri dan mengabaikan inisiatif Microsoft lainnya yang sama sekali tidak terlihat menawan. Ini adalah perjanjian dengan Acer untuk melakukan pra-instal program Microsoft di bawah ancaman gugatan paten, serta upaya untuk mematenkan teknologi yang sudah diterapkan di Ubuntu. "Microsoft sangat mencintai Linux sehingga akan mencekiknya hingga mati dengan klaim paten, sementara media tidak memperhatikan, tetapi sebaliknya mengatakan bahwa Microsoft sekarang" berteman "dengan GNU / Linux," TechRights mengomentari situasi tersebut.

Diketahui bahwa berkat paten, Microsoft telah mengubah Android yang beroperasi menjadi "sapi perah" dan setiap tahun menerima sekitar $ 3 miliar dari 74 produsen perangkat Android, Samsung sendiri membayar Microsoft satu miliar dolar per tahun . HTC, General Dynamics, Wistron, Onkyo, Acer, ViewSonic dan banyak lainnya membayar Microsoft $ 5-10 untuk setiap perangkat Android yang terjual . Di bawah ancaman litigasi, produsen yang merilis perangkat Android menyetujui persyaratan Microsoft, termasuk pra-instal program Microsoft pada ponsel cerdas dan tablet mereka, seperti yang terjadi dengan Acer minggu lalu .

Korporasi Redmond telah mencoba untuk mengeluarkan sekitar skema yang sama dengan Linux sejak 2006 (gugatan paten terhadap Novell). Litigasi secara tradisional berlangsung secara tertutup, dan transaksi diselesaikan dengan persyaratan NDA.

"Ini adalah pemerasan," kata Mark Shuttleworth (pendiri Canonical) beberapa tahun yang lalu, "dan kita harus menyebut sekop sekop ... Kita tidak dapat ditarik ke dalam permainan ini." Kemudian, dia juga mengatakan: "Microsoft menuntut untuk membayar paten mereka, tetapi tidak mengatakan yang mana. Jika seorang pria datang ke toko dan mengatakan: "Ini adalah area berbahaya, mengapa Anda tidak membayar saya 20 dolar dan saya jamin semuanya akan beres", ini ilegal. Ini raket. "

Situasi ini diperburuk oleh fakta bahwa Microsoft tidak malu untuk meminjam ide dari GNU / Linux (dalam hal ini, dari Ubuntu) dan mematenkannya, menyamar sebagai penemuan mereka. Sekarang Microsoft telah mengajukan aplikasi paten untuk teknologi Continuum untuk konvergensi smartphone dengan perangkat periferal - monitor, keyboard, mouse - secara nirkabel, seperti di Ubuntu Phone. Orang hanya bisa berharap bahwa kantor paten akan mendeteksi keberadaan penemuan sebelumnya. Jika tidak, Microsoft akan memiliki senjata lain dalam portofolionya untuk memeras pengembang perangkat Linux.

Sebenarnya, sisa paten dari portofolio Microsoft adalah tentang rencana yang sama. Meskipun ada transaksi dengan ketentuan NDA, pada 2011, daftar paten Android masih tersedia untuk umum.yang dibawa Microsoft terhadap Barnes & Nobles. Produsen tablet Android Nook. Pengacara Microsoft kemudian mengatakan bahwa perusahaan mereka memiliki lebih dari 60 ribu paten. Namun, mereka mengajukan sepuluh langsung ke kasus ini.

Barnes & Nobles menganggap semua paten ini "sepele, baik dalam hal kepentingan komersial maupun dalam hal kesesuaian untuk paten." Spesialis perusahaan memberikan komentar rinci tentang masing-masing paten. Misalnya, dalam kasus "paten 372" untuk mengunduh gambar di browser web, mereka mengingatkan akan keberadaan browser Netscape Navigator 2.0b3, di mana itu diterapkan (prior art). Dalam hal “522 paten” untuk tab di sistem operasi, mereka memberikan contoh panduan pemrograman untuk sistem operasi OS / 2 2.0, yang menjelaskan tentang tab tersebut. Anotasi untuk dokumen elektronik dari Patent 233 dijelaskan dua tahun sebelumnya (1997) dalam Panduan Pengguna Perangkat Lunak Adobe Acrobat. Dan seterusnya untuk sisa paten.

Tapi taktik membawa efek.

Minggu lalu Microsoftmemperbarui perjanjian lisensi dengan Wistron, yang menurutnya wajib melakukan pembayaran dari setiap tablet, smartphone, pembaca elektronik atau perangkat lain yang menjalankan Android atau Chrome OS. Transaksi semacam itu tidak hanya membawa keuntungan tambahan bagi Microsoft tanpa banyak usaha, tetapi juga secara artifisial meningkatkan biaya gadget dengan OS berbasis pada kernel Linux.

Microsoft melakukan litigasi paten melalui anak perusahaan Microsoft Technology Licensing LLC, yang beroperasi dalam tradisi troll paten terbaik. Pekan lalu , perjanjian paten lisensi silang lainnya diumumkan , kali ini dengan perusahaan Jepang Rakuten Inc., syarat-syarat transaksi bersifat rahasia, tetapi itu memengaruhi "perangkat Linux dan Android."

Pada saat yang sama, media massa terus menulis"Microsoft baru", yang bersahabat dengan Linux , dengan hampir tidak menyebutkan serangan besar-besaran Microsoft terhadap GNU / Linux dengan klaim paten.

Mengenai inisiatif open source terbaru Microsoft, termasuk rilis SQL Server untuk Linux, TechRights menulis : “Microsoft menegaskan bahwa Microsoft adalah properti intelektualnya di pengadilan. Sekarang dia bertingkah seperti Linux adalah miliknya. "
