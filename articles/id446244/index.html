Ekstensi Chrome untuk pengembangan web dan bekerja dengan GitHub



Halo semuanya. Saya ingin berbagi dengan Anda beberapa ekstensi untuk Chrome yang saya gunakan saat mengembangkan, menganalisis situs, bekerja dengan GitHub dan hanya dalam kehidupan sehari-hari. Saya yakin bahwa bagi sebagian besar pembaca, daftar ekstensi ini tidak akan menjadi pembuka, tetapi mereka yang belajar untuk pertama kalinya dan menginstalnya tidak diragukan lagi akan menikmati bekerja dengan alat baru.

Saya pikir itu tidak layak membuat tapak panjang dengan deskripsi mendalam tentang alat, mereka cukup sederhana dan dapat dimengerti. Oleh karena itu, saya akan mencoba berbicara secara singkat tentang keuntungan penggunaannya.

Untuk bekerja dengan GitHub


Octotree , menurut saya, merupakan ekstensi yang paling berguna dan nyaman. Ini memungkinkan Anda untuk melihat kode sumber dalam bentuk pohon, yang ditampilkan di sisi kiri halaman. Navigasi baru jauh lebih nyaman daripada standar. Octotree akan sangat berguna jika Anda bekerja pada proyek skala besar dengan sejumlah besar direktori bersarang.

GitHub Repository Size adalah plugin yang nyaman dan sederhana yang menampilkan ukuran repositori individual dan ukuran setiap file di dalamnya.

OctoLinker adalah ekstensi yang menyediakan navigasi kode cepat dengan mengubah elemen kode sederhana menjadi tautan yang dapat diklik. Dengan mengikuti tautan ini, Anda dapat dengan cepat menemukan sumbernya. Bergantung pada bahasa pemrograman, ekstensi menunjukkan file sumber atau membawa pengguna ke situs eksternal. Elemen yang dikonversi ke tautan ditandai dengan titik-titik merah muda sehingga Anda dapat langsung melihat di mana ada tautan interaktif.

GitHub Hovercard menyediakan akses instan ke informasi pengguna, repositori, pertanyaan, dan komitmen dari mana saja di situs web GitHub. Arahkan kursor ke atas informasi.

npmhub Saat melihat repositori di github.com yang memiliki file package.json , ekstensi ini akan menganalisis dependensi dalam paket dan menampilkan tautan dengan deskripsi untuknya tepat di bawah file README.

statistik github npm menunjukkan statistik unduhan paket di situs NPM, jika ada.

Github yang disempurnakan menampilkan brunch yang baru saja diterbitkan, dan juga memungkinkan Anda untuk bekerja kembali dengan permintaan dan menampilkan avatar pengguna yang telah menanggapi komentar.

Kontribusi Isometrik - ekstensi ini tidak berguna seperti yang sebelumnya, tetapi sangat menarik). Ini memungkinkan Anda untuk beralih di antara bagan kontribusi reguler di GitHub dan versi seni piksel isometrik.

Untuk bekerja dengan situs


Eye Dropper adalah alat yang hebat untuk melihat warna yang digunakan di situs web dan membuat palet Anda sendiri. Saya menggunakannya hampir setiap hari)).

Tangkapan Layar Layar Penuh . Bayangkan bahwa Anda telah mengembangkan pendaratan di 3-5 layar dan tidak tahu bagaimana menjadikannya layar penuh. Tekan layar cetak 5 kali dan buat satu di editor satu di bawah yang lain? Oh baik! FPSC adalah alat yang hebat yang mengambil tangkapan layar dari halaman saat ini dalam beberapa detik. Ini sangat berguna jika Anda perlu memeriksa optimasi situs tanpa mengunduh layanan khusus, serta untuk sepenuhnya menangkap situs yang Anda buat.

WhatFont - dengan bantuan ekstensi ini, Anda dapat dengan mudah mengetahui font apa yang digunakan di situs hanya dengan mengeklik ekstensi dan memindahkan kursor ke tempat yang Anda butuhkan. Sederhana dan elegan. Dinonaktifkan dengan menekan tombol "Esc". Juga menunjukkan warna teks, tinggi, ukuran font dan parameter lainnya.

PerfectPixel - ekstensi ini akan membantu Anda mengembangkan situs Anda dengan akurasi piksel demi piksel.

CSS Peeper adalah alat analisis CSS yang dirancang khusus untuk desainer. Tujuan dari ekstensi ini adalah untuk memungkinkan para desainer untuk fokus pada desain dan menghabiskan waktu sesedikit mungkin menggali ke dalam kode. Coba saja dalam tindakan dan Anda akan mengerti segalanya.

Untuk pekerjaan sehari-hari


Google Translator adalah alat lain yang hampir tidak ada hari kerja saya bisa tanpanya. Ini bukan halaman, ini adalah ekstensi. Dengan itu, Anda dapat dengan cepat menerjemahkan teks yang dipilih langsung pada halaman tanpa pergi ke halaman penerjemah. P.S. tidak menerjemahkan teks yang panjang, tetapi itu tidak masalah.

Saya juga meletakkan informan-SEO, tapi itu agak bodoh). Itu saja untuk saat ini. Saya harap Anda menganggap ulasan ini bermanfaat.
