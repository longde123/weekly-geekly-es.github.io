我想立即进行预订：将出售两种Transformer 3配置，其中包括没有键盘的键盘和没有键盘的手写笔。我在审查中有第一个选择，因此除了触控笔可以识别1024度的挤压力之外，我无法对触控笔说什么，这对于艺术家和设计师来说非常有用。

在设备本身的套件中也有充电，一块擦拭屏幕的布和一个迷你基座。它具有三个接口：USB 3.1 TYPE C / Thunderbolt 3，HDMI和USB 3.0。但是，请注意，标准设备中没有扩展坞；您必须单独购买。





在扩展坞的底部，一块类似橡胶的材料制成的板可防止您插入和拔出插头时打滑。



在Transformer Mini中，封闭形式的锁定键盘仅保护设备的屏幕，在这里，它集成到了成熟的外壳中，平板电脑被完全包裹起来。







外壳由粗糙的防滑材料制成。带有触点连接器的架子上装有强力磁铁，字面上附着在平板电脑上。持有非常安全。









键盘的外形非常薄，类似于超级本。按键行程仅为1.4毫米。光滑的触摸板与粗糙的表面形成鲜明的对比。因此，通过触摸找到它比大多数笔记本电脑容易得多。





键盘配备了具有多个亮度级别的白色背光。就像在笔记本电脑上一样。



盖子上有很多折痕，您可以将其折成不同的形状，从而为平板电脑选择最方便的位置。开发人员提供两个标准位置，但在该位置可以折叠，而以其他方式可以折叠。





变形金刚3的外壳是全金属的铝制零件，其中许多复杂形状的型腔在CNC机器上进行加工以容纳内部零件。表壳的颜色不是灰色的铝，而是带有淡淡的金色。背面有精美的虹彩公司图案。



还将出售灰色钛金属版本。



主摄像机镜头从外壳表面略上方突出。





镜头左侧是相机操作指示灯，右侧是光传感器，边缘有麦克风。



变形金刚3的端部圆润，倒角围绕四周。



左侧是两个扬声器和一个组合音频插孔的网格。



顶部有一个音量控制和一个电源按钮，以及一个指纹扫描仪。





右侧是USB 3.1 TYPE C / Thunderbolt 3接口，用于指示设备的运行情况，并带有两个扬声器。









展示架


12.6英寸的显示器具有出色的色彩还原效果（85％NTSC）和近180度的视角。它的分辨率也很不错-2880x1920，像素密度达到275 ppi。屏幕占据了设备正面的80％。

在某些人看来，显示器周围的框架过宽。但是在我看来，这已经是一个人的解剖特征的问题：当您将平板电脑握在手中时，您需要足够的空间以舒适地抓握，否则手指不会意外进入非触摸区域。毕竟，我们没有手持像智能手机这样的平板电脑，而平板电脑大部分只是放在我们的手掌中；您通常必须用手指抓住平板电脑的边缘，例如书本或纸。可以使框架更薄，但随后屏幕边缘会出现随机的咔嗒声，我们会为该设备感到傻瓜而烦恼。就像智能手机的大小一样-纯粹是从技术上讲，很早以前您就可以使它们更薄，只有使用它才完全不方便。

音效


最近，Harman Kardon徽标经常出现在ASUS设备上。鉴于这家公司的声誉，这不能不令人高兴。变形金刚3支持这一新传统



，平板电脑具有四个扬声器和一个内置的SmartAmp放大器。系统会根据平板电脑的位置（横向或纵向）调整声音输出，因此无论您如何扭转设备，立体声效果都不会消失。在我看来，这四个扬声器产生的声音更加坚实或类似，从而在空间中充满了声音。好像听起来很完整，没有任何单独的声音来源，也没有指向听众的“成功”方向。听起来非常不错。

性能表现


Transformer 3的核心是Intel Core i5-7Y54处理器，主频为1.6 GHz。RAM的数量-4或8 GB，双通道。对于图形符合集成处理器芯片的Intel HD Graphics 615。

我不能说《变形金刚3》有点体贴。为了进行测试，我有一个具有8 GB RAM的模型，它足以通过几十个打开的选项卡快速运行OS本身和浏览器，可以以高比特率查看全高清视频，甚至可以使用Lightroom CC和Photoshop CC（可以通过订阅获得）。当然，复杂的效果（例如在30兆像素的照片上使用重抗锯齿功能）的速度要比在台式机上慢得多，但是Transformer 3的重量类别却完全不同。小工具的其余部分显示出非常差的速度，特别是考虑到非常不错的显示分辨率。

顺便说一下，在长时间的重负载下，变压器会明显发热：它具有被动冷却功能，因此将其放在枕头上放置数小时，而用WoT斩波可能不值得。

在压力测试过程中，电池在4.5小时内放电至22％。同时，《变形金刚3》支持快速充电模式：大约50分钟即可达到60％的电量。



结论


这种变压器-部分与超级本竞争。相当大的紧凑性，与大型显示器水平相当的分辨率，相当不错的处理器。同时，它可以用作带有“桌面”操作系统和强大音频系统的平板电脑。通过连接扩展坞可以解决端口不足的问题，您可以单独购买具有各种连接器的较大版本。结合手写笔，变形金刚3将首先引起艺术家，设计师和仅仅是运动鉴赏家的兴趣，例如摄影师，他们的背包已经被笨重的摄影器材所堵塞。
