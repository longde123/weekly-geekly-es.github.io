电动车不适合我

关于更好的讨论，电动汽车或ICE汽车已经有好几年没有安静了，随着知名品牌推出的每款新车型都一再引起人们的注意。

也许，对于每个打算购买新车的人来说，是否应该认真考虑将电动车视为替代品的问题也出现了同样的问题。 不可能不注意到有越来越多的要约，但这是问题的吸引力。 在本文中，我试图回答自己问的问题，希望这些问题的答案对您有用。

由于需要购买自己的新车，一年多以前，我遇到了，这真是出乎意料。 我的任务是找到公司车的替代品。 同时，我不想失去舒适感，与此同时，我的财务机会也不是无限的。

根据与沃尔沃和三巨头的亲身经验，选择了沃尔沃。

首先，我建议比较一些一般的技术（官方）特征：



没有仅柴油的数据，但是根据在Youtube上进行的各种测试，混合动力汽车的能耗至少比D5高出一升。

比较时，我使用了具有最相似常规配置的相同版本。 同时，总价差接近12,000欧元。

为了进一步评估，我们提出了以下尽可能接近实际的情况：

  • 车主每周四天上班，每天行驶130公里以上（一年190天，每年190 * 130 = 24700），其中大部分沿着不太繁忙的高速公路行驶（可减少少于5升的柴油消耗每100公里）
  • 由于车主喜欢旅行，因此他每年因个人事务至少行驶10,000公里。
  • 因此，年里程超过35,000公里。 2018年的平均油耗为5.4 L / 100 km。 要覆盖35,000的距离，您将需要大约（35,000 x 0,054）1,890升柴油，价值超过2,300€（2019年前六个月德国西部每升柴油的平均价格为1.22）

为什么插电式混合动力车无法替代


  1. 所有者住在公寓楼中，无法在家中或附近给电池充电。 没有工作的机会。
  2. 即使有可能在家中为Plug-In-Hybrid充电，一次充电也不足以进行充电。 因此，然后我们在一辆重2吨以上的汽车上使用日光浴室。
    • 从130公里的电力减去40公里的电力后，剩下90公里。 而这已经是6.4 L / 100 km的速率，而不是5.4 L / 100 km。 因此，仅在评估上下班道路时，平均消耗量才增加一升，结果如下：
      每年190个工作日* 90 = 17100 km
      17100公里* 0.054公升/ 1公里= 923公升x 1.22欧元/公升= 1127欧元
      17100公里* 0.064公升/ 1公里= 1094公升x 1.22欧元/公升= 1335欧元（每年增加208欧元）
    • 为电池充电11.2 kWh：（11.2 x 0.30 Cent / kWh德国绿色能源正常价格）= 3.36€
      €3.36 x 190天=每年€638
    • 因此，在计算“机器人屋，家庭作业”道路的成本时，我们发生了什么：
      仅限柴油：每年1,627欧元（190 x 130 x 0.054 x 1.22）
      柴油插电式混合动力车：每年1973€（1335 + 638）
      每年加上346€-这是我们的混合动力车额外费用
    • 充电需要3.5到7.5个小时（家用插座）。 将充电时间减少到三个小时需要追加800欧元的投资。

  3. 混合公路税仅便宜12欧元（228欧元对240欧元）
  4. 剩余价值。 相同配置（2017年型号）的两年混合动力和柴油的成本几乎相同（与mobile.de相比）。 因此，沃尔沃V60插电式混合动力车的价格损失更大。 由此我们可以得出结论，购买者没有看到这种配置的特殊优势。 与其他厂商的情况如何，我不知道。
  5. 如您所料，混合动力车的维修费用与柴油发动机的费用没有太大的不同。
  6. 在许多模型中，电池部分位于后备箱中，从而减小了后备箱和油箱的体积，从而降低了汽车对旅行爱好者的吸引力。 需要额外的车顶行李架，结果，这又增加了燃料消耗和购买成本。 是的，油箱减少22.5升会增加停车次数和行驶时间。

基于以上所述，我可以说，在我的情况下，Plug-In-Hybrid的东西毫无意义，而且绝对不切实际。 我相信这种计算将适合许多插电式混合动力汽车制造商。 只有价差（+11880）可以购买9738（11880 / 1.22）升柴油，并可以行驶18万多公里！

现代沃尔沃车型已放弃柴油电车，转而使用汽油电车。 这种配置大大增加了拥有成本，因为即使根据官方数据，汽油消耗也比柴油发动机高两升以上。 由于汽车重量更大，实际消耗量甚至更高。 此外，汽油（在德国）比柴油贵0.15欧元。

必须认识到，价值差异已大大减少。 油箱和行李箱尺寸等指标已得到改善。 但是，充电仍仅足以满足40（正式为50）公里的行驶需求，因此该车不适合长途旅行。 各种YouTube博主保证的实际花费超过10升/ 100。 就我个人而言，柴油曾经是并且仍然是唯一的选择。

德国汽车业的未来是什么？


令我惊讶的是，德国经济的所谓机车-汽车工业热衷于梦想着光明的未来，这将使他们看到德国车队的“电气化”。 各方都有不同的预测，不同的专家做出各种假设，到哪一年（2025年，2030年），德国汽车车队中有多少将变成电动汽车。 提醒拍卖，谁会提供更多。 例如，大众汽车表示，到2030年，其生产的电动汽车中约有40％（约70款）。 2018年，该工厂生产了超过1100万辆汽车。 令人好奇的是，这种发展是部分被迫的，因为欧洲立法者要求到2030年将二氧化碳排放量减少37.5％。

参观了欧洲几乎所有的汽车博物馆之后，我不禁欣赏奥迪和戴姆勒等品牌在30年代已取得的发展和进步。 当时，这些品牌的汽车能够克服400 km / h的速度线。 也许我只是从小就喜欢机械师，但是如果以自动变速箱为例，那么现代变速箱就是机械师的成就。 我仍然很欣赏8/12/16缸发动机。 而且，ICE的声音...



我不得不同意，内燃机的效率大大低于电动机的效率。 但是，三相异步电动机并没有引起我太多的激动，就像费迪南德·保时捷（Ferdinand Porsche）在1900年创造的电动轮一样！



但是，让我们回到思考未来。 因此，如果实际上十年内在德国注册的新车中有40％至60％是电动汽车，那么我们所说的是超过200万辆汽车。 该预测表明汽车行业的整体结构将发生重大变化。 知道所有为以下系统生产零件的人今天将做什么（铝铸件除外）将是非常有趣的：

  • 气体分配系统：凸轮轴，气门，摇臂，弹簧等。
  • 曲柄机构：曲轴，连杆，活塞，环等
  • 冷却系统：散热器，风扇，泵
  • 排气系统：谐振器，催化剂，消音器
  • 其他小物件，过滤器，泵，蜡烛，喷油器等。
  • 变速箱：变速箱，离合器（不需要大多数组件）



大幅减产不仅会影响上述组件的直接制造商，还会影响其盟友，设备制造商等。我最终了解到，许多工厂将尝试恢复生产。 但是电动汽车的组件太少，无法“雇用”所有人。 我不认为那些生产排气管的人正在对电动机进行重塑。 毫无疑问，一切都不会在一天内发生，但是变化会以一种或另一种方式影响许多事物。

电气工程没有历史或传统。 例如，如果梅赛德斯（Mercedes）对其汽车进行了100多年的改进，那么这主要涉及发动机制造。 购买汽车时，买家的主要问题之一是发动机变速箱的耐用性和可靠性问题。 在电动汽车市场上竞争为他们制造4、6、8、12或16缸柴油或汽油发动机以及自动变速箱的经验并不能带来竞争优势。 特斯拉成功地证明了这一点。
