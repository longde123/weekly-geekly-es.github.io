Konferensi Industri Game GAMEDEV.HOUSE

Pada 22 September 2019, sebuah konferensi dari Gamedev.House akan diadakan di Sekolah Tinggi Informatika Bisnis dari Universitas Riset Nasional, Sekolah Tinggi Ekonomi (HSE) tentang penciptaan dan peluncuran proyek permainan. Acara ini akan diadakan dengan dukungan program pendidikan di industri game dari Sekolah Tinggi Teknik Komputer - "Manajemen Proyek Game" dan "Dasar-dasar Pembuatan Game".


Dari pukul 12:00 hingga malam hari, para tamu acara akan dapat mendengarkan laporan dari perwakilan industri, mengajukan pertanyaan kepada dosen, berkenalan dengan proyek-proyek pengembang indie dan lulusan Sekolah Tinggi Ekonomi di Sekolah Tinggi Ekonomi, dipresentasikan di showcase, dan juga menghabiskan sepanjang hari di perusahaan orang-orang yang berpikiran sama - berkenalan dan berguna. komunikasi.

Dalam program acara


  • Laporan dari perwakilan industri;
  • Meja bundar tentang topik paling relevan untuk membuat dan meluncurkan game di berbagai platform;
  • Menampilkan proyek game dari pengembang independen dan game lulusan program pendidikan di industri game di HSE; HSE;
  • Perpustakaan game dengan permainan papan yang menarik;
  • Stand perusahaan game;
  • Berdiri dengan proyek dalam realitas virtual;
  • Hadiah menarik dari mitra acara dan banyak lagi!

Pendaftaran Mulai: 11:00
Lokasi: st. Trifonovskaya, w. 57, hlm. 1 (stasiun metro Rizhskaya), gedung pertemuan.
