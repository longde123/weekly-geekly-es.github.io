Pemblokiran telegram memicu peningkatan biaya startup lokal


Sumber foto: Kommersant

Biaya layanan cloud untuk beberapa pengembang perangkat lunak domestik meningkat dua kali lipat atau lebih. Alasannya adalah "pemboman karpet" Roskomnadzor, yang dilakukan sehubungan dengan seluruh kumpulan alamat IP di Rusia. Adapun peningkatan biaya, masalah ini hanya mempengaruhi startup kecil yang dipaksa untuk beralih ke analog Rusia Google Cloud dan Amazon. Tetapi perusahaan pengembangan besar dan menengah tidak merasakan perubahan signifikan.

Pemblokiran telegram menyebabkan peningkatan 100% dalam biaya layanan cloud, dan, sebagai akibatnya, peningkatan biaya perangkat lunak dalam negeri. Ini dilaporkan ke Kommersant oleh direktur umum perusahaan Equio Alexey Vagin. Dia mengatakan bahwa di antara perusahaan kecil, mereka yang berada di luar negeri, tetapi mengembangkan produk cloud untuk Rusia, paling menderita. Sebelum memblokir messenger, perusahaan seperti itu menggunakan layanan cloud Rusia hanya untuk menyimpan data pribadi pelanggan, informasi umum diposting di server layanan asing seperti Google Cloud dan Amazon Web Services.

Setelah 16 April, beberapa perusahaan IT mendapati diri mereka dalam kondisi yang tidak menguntungkan bagi mereka, karena banyak alamat IP layanan cloud asing diblokir di wilayah Federasi Rusia.
