Recherche: Linux est toujours le système d'exploitation le plus populaire dans le cloud

Nous discutons des statistiques des fournisseurs IaaS étrangers, fournissons des chiffres pour notre cloud et parlons des raisons qui ont influencé une telle distribution de système d'exploitation open source.


Photos - Ian Parker - Unsplash

Distribution d'actions


Selon IDC, en 2017, 68% des serveurs d'entreprise internes et basés sur le cloud utilisaient Linux. Depuis lors, ce chiffre a augmenté - cette tendance est tracée par de nombreux fournisseurs IaaS.

En 2015, les responsables de Microsoft ont déclaré que chaque quatrième instance du cloud Azure s'exécute sous Linux. Deux ans plus tard, leur nombre était de 40%. Cette année, le nombre de machines Linux a dépassé 50% . L'entreprise informatique elle-même est également devenue un utilisateur actif d'un système d'exploitation ouvert. Par exemple, les réseaux définis par logiciel (SDN) d'une organisation sont construits sur sa base.

Une image similaire est observée dans les nuages ​​des autres fournisseurs IaaS. Par exemple, dans le cloud 1cloud.ru, Linux exécute 44% des machines virtuelles. Dans le cas de Windows, ce chiffre est de 45%.


Partages de systèmes d'exploitation sur des serveurs actifs dans le cloud 1cloud

"Nous nous attendons à ce que dans un avenir proche, Linux puisse devenir un leader et contourner d'autres systèmes d'exploitation", commente Sergey Belkin, chef du département de développement du projet 1cloud.ru . «Étant donné qu'il y a quelques années à peine, plus de la moitié des machines virtuelles déployées dans notre cloud fonctionnaient sous Windows.»

La prévision est confirmée par les statistiques d'autres fournisseurs IaaS. Par exemple, dans le cloud privé de l'un des plus grands fournisseurs occidentaux sous Linux, plus de 90% des instances fonctionnent .

Dans le même temps, le système d'exploitation open source reste la plateforme la plus populaire pour l'hébergement Web. Selon l'agence analytique W3Techs, 70% des dix millions de sites les plus populaires sont déployés sur des serveurs Linux (selon le classement Alexa ). Les 30% restants appartiennent à Windows.

Pourquoi Linux


Les experts identifient au moins deux facteurs qui affectent la distribution du système d'exploitation dans le cloud.

Flexibilité de l'architecture. Ce facteur est considéré comme l' un des facteurs déterminants de la Linux Foundation. Linux est adapté pour effectuer diverses tâches et fonctionne sur des plates-formes de différentes tailles: des appareils mobiles aux superordinateurs. Par exemple, en 2017, 498 des 500 premiers supercalculateurs exécutaient ce système d'exploitation open source. Mais à la fin de cette année, Linux a commencé à fonctionner à 100% des ordinateurs au sommet.

Le supercalculateur le plus puissant à ce jour - IBM's Summit - est géré par Linux . Le premier supercalculateur exaflops aux États-Unis, qu'ils prévoient de terminer en 2021, fonctionnera également sur la base de ce système d'exploitation open source .

Vaste communauté. La base de code Linux est mise à jour toutes les dix semaines environ. Depuis 2005, plus de 15 000 ingénieurs ont contribué au développement du cœur. Parmi eux, des employés de 200 grandes entreprises. Ce n'est qu'en 2017, 3% des modifications de la base de code ont été apportées par les développeurs de Google et Samsung. Plus de 13% des changements sont «responsables» dans Intel.


Photos - Ian Parker - Unsplash

Les grandes entreprises informatiques sont activement impliquées dans le développement de Linux lui-même et des produits open source basés sur celui-ci. Microsoft propose la plate-forme Azure Sphere pour les applications IoT, qui est basée sur le noyau Linux. Intel a lancé le projet cloud Clear Linux , dans lequel les ingénieurs optimisent le système d'exploitation open source pour travailler sur leurs processeurs. HPE propose ClearOS pour être livré avec son matériel. IBM a acquis RedHat et développe actuellement l'une des distributions les plus populaires du marché.

De nouveaux produits open source sont activement déployés dans le cloud, ce qui a un effet positif sur la distribution de Linux.

Et ensuite


Les chiffres exacts concernant la popularité d'un système d'exploitation particulier dans le cloud doivent être traités avec un certain scepticisme. L'infrastructure informatique moderne des fournisseurs de cloud est complète. De nombreux hyperviseurs peuvent être appelés «imbriqués» et des situations surviennent lorsqu'un système d'exploitation est entouré par un autre.

Mais même avec ce fait à l'esprit, il est sûr de dire que Linux est de plus en plus utilisé dans le cloud.
