Tipps und Tricks für Visual Studio

Unabhängig davon, ob Sie neu sind oder Visual Studio seit Jahren verwenden, gibt es eine Reihe von Tipps und Tricks, mit denen Sie produktiver arbeiten können. Wir teilen seit einiger Zeit Tipps auf Twitter mit dem Hashtag #vstip , und dies ist eine Sammlung der besten, die es bisher gab.

Debugger


Wenn Sie F10 drücken, um den Debugger anstelle von F5 zu erstellen, auszuführen und anzuhängen, wird dies automatisch unterbrochen, wenn Ihr eigener Code zum ersten Mal ausgeführt wird. Keine Haltepunkte erforderlich.



Unterstützt von Visual Studio 2005

Das erneute Anschließen an den Prozess (Umschalt + Alt + P) ist äußerst hilfreich, wenn Sie immer wieder an denselben Prozess anhängen müssen.



Unterstützt von Visual Studio 2017 v15.8

Ein blauer Punkt am Rand zeigt einen Threadwechsel während des Debuggens an.



Unterstützt von Visual Studio 2013

Lösung


Verbessern Sie die Leistung beim Laden der Lösung und reduzieren Sie das visuelle Rauschen, indem Sie die Wiederherstellung von Knotenerweiterungen im Projektmappen-Explorer deaktivieren und Dokumente beim Laden der Lösung erneut öffnen.



Unterstützt von Visual Studio 2019

Verwenden Sie für eine schnelle Tastaturnavigation Strg + T, um alles in Ihrer Lösung zu finden - Dateien, Klassen usw.



Unterstützt von Visual Studio 2017

Weisen Sie eine Tastenkombination zu, um einen "Git Pull" auszuführen, damit Sie weder CLI noch Team Explorer verwenden müssen, um sicherzustellen, dass Ihr Repo auf dem neuesten Stand ist.



Wird in Visual Studio 2019 unterstützt

Lassen Sie den Projektmappen-Explorer automatisch das aktuell aktive Dokument auswählen, damit Sie den Überblick über dessen Position im Projekt nicht verlieren.



Unterstützt von Visual Studio 2010

Herausgeber


Umgeben Sie HTML-Elemente einfach mit Umschalt + Alt + W. Das eingefügte Tag wird ausgewählt, damit Sie es problemlos zu einem beliebigen Tag bearbeiten können, und das End-Tag stimmt automatisch überein.



Unterstützt von Visual Studio 2017

Kopieren Sie ein beliebiges JSON-Fragment in die Zwischenablage und fügen Sie es als stark typisierte .NET-Klassen in eine beliebige C # - oder VB-Codedatei ein.



Unterstützt von Visual Studio 2013

Sie müssen keine Anführungszeichen um JSON-Eigenschaftsnamen schreiben. Geben Sie einfach einen Doppelpunkt ein, und Visual Studio fügt die Anführungszeichen automatisch ein.



Wird in Visual Studio 2015 unterstützt

Machen Sie IntelliSense und QuickInfos für die Dauer, in der Sie die Steuertaste gedrückt halten, halbtransparent.



Unterstützt von Visual Studio 2010

Anstatt '(' erneut einzugeben, um Parameterinformationen in Methodensignaturen anzuzeigen, verwenden Sie Strg + Umschalt + Leertaste, um die aktuell verwendete Überladung anzuzeigen.



Unterstützt von Visual Studio 2010

Verschiedenes


Spielen Sie einen Sound ab, wenn bestimmte Ereignisse in Visual Studio auftreten.



Unterstützt von Visual Studio 2010

Erstellen Sie benutzerdefinierte Fensterlayouts für bestimmte Entwicklungsszenarien oder überwachen Sie Setups und wechseln Sie einfach zwischen diesen.



Unterstützt von Visual Studio 2017

Geben Sie an, welche Visual Studio-Komponenten für eine Lösung erforderlich sind, und Visual Studio fordert den Benutzer auf, sie zu installieren, falls sie fehlen. Weitere Informationen finden Sie im Blogbeitrag Konfigurieren Sie Visual Studio in Ihrem Unternehmen mit .vsconfig .



Unterstützt von Visual Studio 2019

Erweiterungen


Visual Studio Rechtschreibprüfung . Eine Editorerweiterung, die die Schreibweise von Kommentaren, Zeichenfolgen und einfachem Text während der Eingabe oder interaktiv mit einem Werkzeugfenster überprüft. Es kann auch die Rechtschreibprüfung einer gesamten Lösung, eines Projekts oder ausgewählter Elemente durchführen. Es stehen Optionen zum Definieren mehrerer Sprachen zur Rechtschreibprüfung zur Verfügung.



Unterstützt von Visual Studio 2013

Neue Datei hinzufügen . Eine Visual Studio-Erweiterung zum einfachen Hinzufügen neuer Dateien zu jedem Projekt. Drücken Sie einfach Umschalt + F2, um eine leere Datei im ausgewählten Ordner oder im selben Ordner wie die ausgewählte Datei zu erstellen.



Unterstützt von Visual Studio 2015

Git Diff Margin . Git Diff Margin zeigt Live-Git-Änderungen der aktuell bearbeiteten Datei am Visual Studio-Rand und in der Bildlaufleiste an.



Unterstützt von Visual Studio 2012

Dies waren nur einige der Tausenden verfügbaren Erweiterungen. Weitere Erweiterungen finden Sie im Visual Studio Marketplace .
