我如何通过Google Cloud Professional数据工程师认证考试


没有推荐的三年实践经验


*注意：本文专门介绍Google Cloud Professional Data Engineer认证考试，该考试有效期至2019年3月29日。之后，发生了一些更改-在“ 高级 ”部分中进行了介绍*


Google运动衫：是的。 表情严肃：是的。 本文的视频版本在YouTube上的照片 。

是否想要像我的照片中那样的全新运动衫？

或者，也许您对Google Cloud Professional Data Engineer证书感兴趣，并试图找出如何获得它？

在过去的几个月中，我完成了几门课程，并与Google Cloud并行工作，为Professional Data Engineer考试做准备。 然后我参加了考试并通过了考试。 几周后，一件运动衫到来了-但是证书来得更快。

本文将提供一些您可能会有所帮助的信息，以及为获取Google Cloud Professional Data Engineer证书而采取的步骤。

翻译成Alconost

为什么要获得Google Cloud Professional数据工程师证书？


数据围绕着我们，无处不在。 因此，如今急需专家，他们知道如何创建可以处理和使用数据的系统。 Google Cloud提供了用于构建这些系统的基础架构。

如果您已经具备Google Cloud技能，该如何向未来的雇主或客户展示它们？ 有两种方法可以做到这一点：拥有项目组合或通过认证。

该证书告诉潜在的客户和雇主您具有一定的技能，并且您已努力获得他们的正式确认。

这在考试的官方说明中有所说明。

展示您在Google Cloud平台上设计和构建数据处理系统和机器学习模型的能力。

如果您还没有适当的技能，那么在学习认证培训材料时，您将学到有关如何使用Google Cloud创建最高级别的数据处理系统所需的一切。

谁需要Google Cloud Professional数据工程师证书？


您看到了数字-云技术领域正在不断发展，它们与我们在一起已有很长时间了。 如果您不熟悉统计信息，请相信：“乌云”正在上升。

如果您已经是数据处理或分析专家，机器学习工程师，或者想进入数据处理行业，那么您需要Google Cloud Professional数据工程师认证。

使用云技术的能力已成为所有处理数据的专业人员的强制性要求。

我是否需要证书才能成为处理，数据分析或机器学习方面的专业人士？


不行

您可以使用Google Cloud无需证书即可使用数据处理解决方案。

证书只是确认您现有技能的一种方法。

多少钱？


通过考试的费用为200美元。 如果您使他失败，您将不得不再次付款。

此外，您将不得不花钱准备课程和使用平台本身。

使用该平台的成本是使用Google Cloud服务的费用。 如果您是它的活跃用户，那么您将对此有所了解。 如果您是初学者，并且刚刚开始学习本文中介绍的培训材料，则可以创建Google Cloud帐户并完成所需的所有工作，并在注册时将Google记入您帐户的300美元。

我们将立即处理所有课程的费用。

证书有效期有多长？


两年 在此期间之后，必须再次参加考试。

而且由于Google Cloud在不断发展，因此认证要求也可能会发生变化（这是在我开始撰写本文时发生的）。

您需要准备什么考试？


对于专业认证，Google建议拥有三年以上的行业经验，并拥有一年以上使用GCP开发和管理解决方案的经验。

我什么都没有。

在每种情况下，相应的经验大约为六个月。

为了填补空白，我使用了一些在线培训资源。

我选了什么课程？


如果您的案例与我的案例相似，并且您不符合建议的要求，那么要提高自己的水平，可以从以下课程中学习一些课程。

我用它们来准备认证。 它们按通过顺序列出。

我为每个人指出了通过认证考试的成本，时间和有用性。


我在考试前用来提高技能的一些很棒的在线学习资源是有序的： A Cloud Guru ， Linux Academy ， Coursera 。

Google Cloud Platform专业化（Cousera）上的数据工程


费用：每月49美元（7天免费试用后）。
时间： 1-2个月，每周超过10个小时。
有用： 10之8。

Googlera平台上的课程数据工程 Coursera上的Specilization是与Google Cloud合作开发的。

它分为五个嵌套课程，每个课程每周大约需要10个小时的学习时间。

如果您不熟悉Google Cloud上的数据处理，那么此专业知识将为您提供所需的技能。 您必须使用称为QwikLabs的迭代平台完成一系列实践练习。 在此之前，将会有使用Google Cloud的专家进行有关如何使用各种服务（例如Google BigQuery，Cloud Dataproc，Dataflow和Bigtable）的讲座。

Cloud Guru Google Cloud Platform简介


费用：免费。
时间： 1周4-6小时。
有用性： 10之4。

公用事业费率低并不意味着该课程通常毫无用处-并非全非如此。 评级如此之低的唯一原因是因为它不专注于专业数据工程师认证（顾名思义）。

在完成Coursera专业化工作后，我通过它来刷新自己的知识，因为在某些情况下我使用了Google Cloud。

如果您以前曾与其他云服务提供商合作，或者从未使用过Google Cloud，那么本课程对您可能会有所帮助：这是对整个Google Cloud平台的出色介绍。

Linux Academy Google认证的专业数据工程师


费用：每月49美元（7天免费试用后）。
时间： 1-4周，每周超过4个小时。
实用性： 10 分之 10

在通过考试并考虑了课程之后，我可以说Linux Academy Google认证的专业数据工程师是最有用的。

视频教程以及Data Dossier电子书 （该课程提供了出色的免费培训资源）和实践考试使该课程成为我有史以来最好的课程之一。

考试后，我什至在Slack的笔记中为团队推荐了它作为参考。

松弛中的注释


  • 考试的某些问题未在Linux Academy课程，A Cloud Guru或Google Cloud Practice考试中涵盖（这是预期的）。
  • 在一个问题中，有一张数据点图。 通过什么方程式可以将它们分组（例如，cos（X）或X²+Y²）。
  • 您必须了解Dataflow，Dataproc，Datastore，Bigtable，BigQuery，Pub / Sub之间的区别，并了解如何使用它们。
  • 考试中的两个具体示例与培训中的示例相同，尽管在考试期间我根本没有阅读它们（问题本身足以回答）。
  • 了解SQL查询的基本语法非常有用，尤其是对于BigQuery问题。
  • Linux Academy和GCP课程中的实践考试的形式与考试中的问题非常相似-应该通过几次以发现自己的弱点。
  • 请记住， Dataproc可与Hadoop ， Spark ， Hive和Pigs一起使用 。
  • 数据流可与Apache Beam一起使用 。
  • Cloud Spanner是最初为云开发的数据库，它与ACID兼容并且可以在世界任何地方使用。
  • 了解“旧版本”的名称（与关系数据库和非关系数据库（例如，MongoDB，Cassandra）等效）非常有用。
  • 服务的IAM角色稍有不同，但是很高兴了解如何划分用户查看数据的能力和设计工作流（例如，您可以以Dataflow Worker的角色设计工作流，但看不到数据）。

到目前为止，这也许就足够了。 每次考试将以自己的方式进行。 Linux Academy课程将提供80％的必需知识。

在学习过程中，某些服务可能看起来很复杂，因此很高兴能在短时间内看到如何描述特定服务。

课后我做了什么？


课程即将结束，我预定了一个星期的考试。

截止日期的存在是对所学知识进行审核的绝佳动机。

我几次通过了Linux Academy和Google Cloud培训考试，直到我始终如一地获得超过95％的成绩。


首次通过Linux Academy培训考试，分数超过90％。

每个平台的测试都是相似的； 我写下并整理出我经常被误解的问题-这有助于消除弱点。

在考试本身中，主题是使用两个示例开发Google Cloud中的数据处理系统（自2019年3月29日以来考试内容已更改）。 整个考试都有多项选择题。

通过考试花了两个小时，在我看来，这比熟悉的培训考试困难了大约20％。

但是，后者是非常宝贵的资源。

如果我再次参加考试，该怎么办？


更多练习考试。 更多实用知识。

当然，您总是可以做得更好。

推荐的要求表明使用GCP已有3年以上的经验，而我还没有，所以我不得不处理过去的经验。

选配


考试已于3月29日更新。 本文中的材料仍将为准备工作提供良好的基础，但重要的是要注意一些更改。

Google Cloud Professional数据工程师考试部分（ 第1版 ）


  1. 数据处理系统设计。
  2. 建立和维护数据结构和数据库。
  3. 数据分析和机器学习连接。
  4. 为业务流程建模以进行分析和优化。
  5. 确保可靠性。
  6. 数据可视化和决策支持。
  7. 设计时注重安全性和合规性。

Google Cloud Professional数据工程师考试部分（ 第2版 ）


  1. 数据处理系统设计。
  2. 数据处理系统的构建和运行。
  3. 机器学习模型的操作（大部分更改已在此处进行） [NEW] 。
  4. 质量保证解决方案。

在版本2中，版本1的第1、2、4和6部分合并为第1部分和第2部分，在第5部分和第7部分合并为第4部分。版本2中的第3部分已扩展为现在涵盖Google Cloud中的所有新机器学习功能。

这些更改是最近发生的，因此许多培训材料没有时间更新。

但是，如果您使用文章中的材料，则这应该足以覆盖70％的必要知识。 我还将使自己熟悉以下主题（它们出现在考试的第二版中）：

  • Google机器学习API（ML）。
  • Google Cloud Machine Learning Core。
  • 适用于Google Cloud的TPU （由Google专为机器学习开发的设备）。
  • 带有机器学习条款的Google词汇表。

如您所见，考试更新主要与Google Cloud中的机器学习功能有关。

自2019年4月29日起更新。 我收到了Linux学院课程老师（Matthew Ulasien）的信息。

仅供参考：我们计划在5月中旬或下旬更新Linux Academy的Data Engineer课程，并反映其中的新目标。

考试后


通过考试后，您将获得“通过”或“未通过”的结果。 在培训考试中，建议至少争取70％，因此我的目标是90％。

通过考试后，您将通过电子邮件收到激活码以及正式的Google Cloud Professional Data Engineer证书。 恭喜你！

激活码可以在Google Cloud Professional Data Engineer独家商店中使用，在这里您可以赚钱：有T恤，背包和运动衫（在交付时，可能无法提供）。 我选择了一件运动衫。

获得证书后，您可以（正式）展示自己的技能，并返回自己最擅长的工作-构建系统。

两年后再见-重新认证。

P. S.非常感谢上述课程的出色老师以及Max Kelsen为学习和准备考试提供了资源和时间。
