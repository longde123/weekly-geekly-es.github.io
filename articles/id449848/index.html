Hubungkan adaptor WiFi WN727N ke Ubuntu / Mint

Saya punya masalah menghubungkan adaptor wifi wn727n ke ubuntu / mint. Saya mencari Google untuk waktu yang lama, tetapi tidak menemukan solusi. Setelah memecahkan masalah, saya memutuskan untuk menulisnya sendiri. Segala sesuatu yang ditulis di bawah ini dirancang untuk pemula.

PERHATIAN! PENULIS PASAL TIDAK BERTANGGUNG JAWAB ATAS KERUSAKAN APA PUN!

Tetapi, jika Anda melakukan semuanya dengan benar, tidak akan ada konsekuensi. Bahkan jika terjadi kesalahan, tidak ada hal buruk yang akan terjadi. Mari kita mulai.

Pertama-tama, buka terminal menggunakan tombol Ctrl + Alt + T dan masukkan perintah berikut: 

lsusb



Kami melihat adaptor kami Ralink RT7601 (disorot). Anda mungkin memiliki adaptor Ralink RT5370. Driver pada adaptor yang berbeda dipasang secara berbeda. Saya akan menjelaskan bagaimana melakukan ini untuk dua kasus.

Manual untuk Ralink RT5370


Ikuti tautan dan pilih RT8070 / RT3070 / RT3370 / RT3572 / RT5370 / RT5372 / RT5572 USB USB. Unduh arsip dengan driver.

Buka folder tempat driver disimpan dan membongkar arsip bz2. Untuk melakukan ini, klik kanan pada file dan klik "Ekstrak di sini".

Setelah itu, arsip tar akan muncul. Buka kemasan lagi. Klik kanan pada file dan klik "Ekstrak di sini".

Selanjutnya, ubah nama folder menjadi sesuatu yang lebih pendek, karena kita masih harus menulis pathnya ke konsol. Sebagai contoh, saya menyebutnya Driver.

Kami masuk ke folder yang belum dibongkar dan membuka file /os/linux/config.mk dalam editor teks

Temukan baris berikut dan ubah huruf n menjadi y:

# Mendukung Wpa_Supplicant
HAS_WPA_SUPPLICANT = y

# Mendukung WpaSupplicant Asli untuk Network Maganger
HAS_NATIVE_WPA_SUPPLICANT_SUPPORT = y

Setelah itu, simpan file tersebut. Buka terminal dan buka folder yang belum dibongkar. Perhatian! Saya memiliki nama pengguna sergey. Anda memasukkan nama pengguna Anda! Di masa depan, ubah sergey ke nama pengguna Anda. 

 cd /home/sergey//driver/

Selanjutnya, jalankan perintah: 

 sudo make sudo make install sudo modprobe rt5370sta

Dan itu saja! Astaga! WIFI berfungsi, nikmati kesehatan Anda.

Manual untuk Ralink RT7601


Untuk mendapatkan adaptor ini (Ralink RT7601), Anda harus memiliki versi kernel 3.19 atau lebih tinggi. jika perlu - perbarui kernel (jika Anda tidak tahu caranya - google akan membantu).

Selanjutnya, ikuti tautan dan unduh driver:



Selanjutnya, kami memindahkan arsip yang diunduh ke folder beranda dan membukanya (klik kanan, "ekstrak di sini"). Ganti nama folder mt7601-master yang dihasilkan hanya menjadi mt7601.

Setelah itu, masukkan perintah: 

 cd mt7601/src

Sekarang kita berada di direktori yang benar. Anda dapat membangun driver, jalankan perintah: 

 sudo make

Sistem akan meminta kata sandi - masukkan (kata sandi tidak ditampilkan).

Selanjutnya, masukkan perintah: 

 sudo mkdir -p /etc/Wireless/RT2870STA/ cp RT2870STA.dat /etc/Wireless/RT2870STA/

Dan perintah terakhir yang akan mengaktifkan adaptor kami: 

 insmod os/linux/mt7601Usta.ko

Semua !!! Sekarang ubuntu melihat wifi.

Tapi itu belum semuanya! Sekarang, setelah setiap reboot, Anda harus memasukkan perintah terakhir, jika tidak, sistem tidak akan melihat adaptor (khusus untuk Ralink RT7601). Tapi ada jalan keluar! Anda dapat membuat skrip dan menambahkannya ke startup. Di bawah ini tertulis cara melakukannya.

Pertama-tama, kita perlu memastikan bahwa sistem tidak meminta kata sandi saat menggunakan sudo. Untuk melakukan ini, masukkan perintah: 

 sudo gedit /etc/sudoers

Jendela ini akan terbuka:



Kami mencari garis:

% sudo ALL = (ALL: ALL) ALL

Dan kami mengubahnya menjadi:

% sudo ALL = (ALL: ALL) NOPASSWD: ALL

Simpan perubahan - klik "Simpan."

Masukkan perintah: 

 sudo cp -R mt7601 /etc/Wireless/RT2870STA/

Setelah itu, masukkan perintah: 

 sudo gedit /etc/Wireless/RT2870STA/autowifi.sh

Editor teks kosong terbuka. Di dalamnya kita menulis atau menyalin:

#! / bin / bash
insmod /etc/Wireless/RT2870STA/mt7601/src/os/linux/mt7601Usta.ko

Klik "Simpan" dan tutup.

Kami memasukkan perintah: 

 cd /etc/Wireless/RT2870STA/ sudo chmod +x autowifi.sh

Selanjutnya, buka menu Dash dan cari program seperti pada gambar di bawah ini:



Kami membukanya. Klik "Tambah."



Sebuah jendela akan terbuka. Di seberang bidang "Nama" yang kami tulis:

autowifi

Di seberang bidang “Tim” yang kami tulis:

sudo sh /etc/Wireless/RT2870STA/autowifi.sh

Klik tombol "Tambah" dan tutup program. Mulai ulang. Setelah reboot, semuanya berfungsi. Sekarang Anda dapat memilih jaringan di baki.



Ini melengkapi instruksi "kecil" untuk adaptor Ralink RT7601.
