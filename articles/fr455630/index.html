à propos de: cloud


  • 15 juin (samedi)
  • Tolstoï 16
  • gratuitement
  • Pendant environ: cloud, vous pouvez discuter avec ceux qui créent Yandex.Cloud, donner des commentaires aux développeurs et aux gestionnaires de services.
    Cette fois, nous parlerons de ces services:
    Le service géré Yandex pour Kubernetes est un environnement pour une gestion fiable, pratique et sécurisée des clusters Kubernetes dans l'infrastructure Yandex.Cloud.
    Yandex Monitoring est un service de collecte de métriques sur l'état des ressources avec la possibilité de les visualiser.
    Yandex Instance Groups est un service de déploiement et de mise à l'échelle de machines virtuelles, qui vous permet de créer des groupes du même type de machine virtuelle dans l'infrastructure Yandex.Cloud.
    Yandex Message Queue est un service de mise en file d'attente distribué qui vous permet d'organiser une messagerie fiable, évolutive et hautes performances entre les applications.
