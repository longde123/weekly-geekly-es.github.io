Produksi elektronik dalam jumlah kecil di Cina, pengalaman pribadi seorang awam

Terlibat dalam elektronik "untuk jiwa" dan tidak mengklaim kemenangan "profesional" dan "spesialis", namun, kadang-kadang dimungkinkan untuk membuat sesuatu yang menarik tidak hanya untuk diri sendiri, tetapi juga untuk orang lain. Dan di sini muncul pertanyaan membuat seri kecil. Tentang urutan papan sirkuit cetak telah ditulis berkali-kali, banyak yang telah mencoba dan, pada prinsipnya, di sini kurang lebih jelas apa dan bagaimana. Batch pertama disegel, dijahit dan diberikan kepada pemilik yang bahagia, semuanya baik-baik saja. Tiba-tiba ternyata Anda perlu membuat beberapa papan lagi, dan terutama tidak ada waktu untuk duduk di malam hari dengan pinset dan pengering rambut, dan saya merasa kasihan pada mata saya. Waktunya telah tiba untuk mengambil langkah selanjutnya - coba layanan PCBA (Majelis PCB).



Artikel ini akan berbicara tentang adaptor yang sudah dijelaskan di atas, yang ditujukan untuk penggemar simracing. Artikel ini bukan riset pemasaran, saya menggambarkan pengalaman pribadi saya.

Untuk mulai dengan, saya mengingatkan Anda apa yang Anda butuhkan untuk memesan pembuatan papan sirkuit cetak kosong. Jika Anda sudah memiliki papan cerai, Anda perlu mengekspor file Gerber dan Bor. Setiap lapisan PCB membutuhkan file Gerber yang terpisah. Nama file Gerber harus cocok dengan lapisan PCB. Saya menggunakan DipTrace untuk papan kabel dan ketika mengekspor file Gerber, file dengan nama berikut dibuat:

BoardOutline.gbr
Bottom.gbr
BottomMask.gbr
BottomPaste.gbr
BottomSilk.gbr
Melalui.drl
Top.gbr
TopMask.gbr
TopPaste.gbr
TopSilk.gbr

Tampaknya cukup jelas? Tetapi dengan nama file seperti itu, arsip hanya akan diterima oleh SeeedStudio. Untuk semua file lain Anda harus mengganti nama:

Lapisan teratas: pcbname.GTL
Lapisan Dalam 1: pcbname.GL1 (untuk papan 4-lapisan)
Lapisan Dalam 2: pcbname.GL2 (untuk papan 4-lapisan)
Lapisan bawah: pcbname.GBL
Topeng Stop Solder atas: pcbname.GTS
Solder Stop Mask Bottom: pcbname.GBS
Tempel Teratas: pcbname.GTP
Tempel Bawah: pcbname.GBP
Silk Top: pcbname.GTO
Silk Bottom: pcbname.GBO
NC Drill: pcbname.TXT
Lapisan garis besar: pcbname.GKO atau pcbname.DO

Perhatikan fakta bahwa saat memesan papan sirkuit cetak, lapisan dengan pasta solder biasanya tidak diekspor, tetapi saat memesan PCBA, itu berguna bagi kami. Juga ingat untuk mengekspor file bor.

Kami juga membutuhkan BOM (bill of material) dan file pick-and-place, yang juga dapat diekspor dari DipTrace (atau perangkat lunak yang Anda gunakan). Selain itu, BOM dapat dibuat dari file pick-and-place dengan mengelompokkan posisi yang sama dan mengecualikan posisi yang tidak Anda rencanakan untuk dipasang. Sebagai contoh, saya menyediakan tempat untuk memasang konektor JTAG, tetapi konektor itu sendiri disolder hanya ke papan yang digunakan untuk debugging.

Format file BOM berbeda untuk semua orang, tetapi bidang berikut hampir selalu ada di dalamnya:

1. Kuantitas
2. Pabrikan
3. Produsen nomor

Persyaratan dapat beragam, jadi yang terbaik adalah memeriksa titik ini terlebih dahulu.

Juga, jangan lupa untuk memeriksa kapabilitas Design Rules / PCB. Sebagai aturan, sebagian besar layanan memungkinkan untuk mengunduh file yang sudah jadi untuk perangkat lunak yang sering digunakan, tetapi untuk DipTrace Anda perlu memeriksa semuanya secara manual. Di sisi lain, dengan aturan yang berlaku “out of the box” di DipTrace, sebagian besar perusahaan manufaktur PCB mengatasi pasokan tersebut.

Dengan sirkulasi kecil (10-20 buah) sangat mungkin bahwa komponen akan ditempatkan di papan tulis secara manual. Anda perlu memastikan bahwa label semua komponen di papan sesuai dengan file BOM, bahkan jika Anda memiliki file pick-and-place di mana lokasi masing-masing komponen juga ditunjukkan.

Misalkan Anda menangani ini, maka Anda perlu melakukan hal berikut. Pertama, hubungi layanan dan minta mereka untuk mengirim perhitungan biaya proyek dengan mengirimkan mereka gerberas, file bor, BOM dan pilih-dan-tempat. Biasanya jawabannya datang dalam satu atau dua hari. Perhatikan dengan cermat biaya per item baris. Barang-barang berikut akan dimasukkan dalam faktur:

- Produksi papan sirkuit cetak
- Membuat stensil untuk menempelkan pasta ke papan sirkuit
- Pemilihan komponen dari antara analog lokal (*)
- Instalasi dan pengujian (**) (***)
- Pengiriman

(*) - memungkinkan Anda untuk mengurangi biaya papan dengan mengganti komponen asli dengan yang lokal, tetapi juga tidak gratis, biayanya $ 20. Biaya pemilihan adalah biaya satu kali, pemesanan ulang dari desain yang sama tidak dikenakan biaya.
(**) Prosedur pengujian dapat dinegosiasikan terlebih dahulu, di perusahaan kecil mereka dapat memenuhi kemauan apa pun, tentu saja, tidak gratis.
(***) Jika Anda memesan batch kecil, komisi kecil untuk sirkulasi kecil dapat ditambahkan, misalnya, di Makerfabs - dengan batch kurang dari 20 buah.

Membuat stensil juga, secara umum, prosedur satu kali, ketika memesan ulang, Anda tidak perlu membayar, janji Cina untuk menyimpannya.

Berikan perhatian khusus pada metode pengiriman. Kantor kecil sering tidak memiliki pengalaman dengan Federasi Rusia, mereka mungkin, untuk alasan yang baik, menyarankan menggunakan semacam layanan kurir, yang namanya tidak akan memberi tahu Anda apa pun. Pada saat yang sama, layanan kurir di wilayah Federasi Rusia ini dapat bekerja melalui, katakanlah, semacam EMS. Dan Anda menderita bersamanya hanya karena kesalahan Anda sendiri ...

Sekali lagi, jangan lupa untuk menentukan nomor wajib pajak (NPWP) dan minta orang Cina untuk mencantumkannya di formulir alamat agar tidak mendapat masalah dari bea cukai kami. Orang Cina juga dapat bertanya apakah akan menunjukkan harga penuh pada kotak atau meremehkannya. Saya diminta untuk menunjukkan secara penuh, pajak dari jumlah ini belum dipungut.

Setelah harga disepakati dan pesanan dibayar, tahap pembuatan biasanya memakan waktu 1-2 minggu. Pada akhir tahap, Anda dapat meminta saya untuk mengirimkan foto hasil sehingga tidak membosankan menunggu selama pengiriman :) Juga minta laporan saat papan lulus pengujian setuju dengan Anda. Jika semuanya baik - tunggu, akan segera datang.

Ya, ada installer yang terlatih khusus yang melakukannya lebih cepat dan lebih baik. Tapi ini bukan tentang aku ...

Kapan itu pasti tidak layak menggunakan majelis kontrak? Misalnya, jika Anda tidak sepenuhnya yakin bahwa desain Anda berfungsi penuh. Karena ini semata-mata masalah Anda, pengumpul tidak bertanggung jawab atas ini dan pesanan akan dirakit sepenuhnya terlepas dari kinerjanya. Yah, bahkan pada akhir Januari, Anda juga tidak boleh memesan. Anda akan melihat pesanan dilakukan setelah 20 Januari kecuali pada pertengahan Maret. Karena semua China sedang merayakan tahun baru ...

Mengenai kebocoran kekayaan intelektual - layanan ini menyatakan integritas kekayaan intelektual Anda. Bagaimana kekebalan ini dijamin tidak terlalu jelas, tetapi untuk proyek-proyek sumber terbuka atau hobi ini tidak begitu penting. Pada akhirnya, untuk hobi elektronik, sirkuit biasanya cukup sederhana, tetapi firmware mungkin tidak disediakan untuk pengujian. Tapi terserah kamu.
