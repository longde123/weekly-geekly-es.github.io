Kotak Alat untuk Peneliti - Edisi Ketiga: Mencari dan Bekerja dengan Sumber

Bekerja pada proyek penelitian apa pun melibatkan pencarian dan studi banyak sumber informasi. Organisasi dari proses ini bukanlah tugas yang mudah. Hari ini kita akan berbicara tentang alat yang dirancang untuk mengoptimalkan berbagai komponennya.

Vendor perangkat lunak akademik sering bekerja berdasarkan prinsip kolaborasi dengan lembaga pendidikan. Jika organisasi Anda belum memperoleh produk yang Anda minati, mengatur akses ke sana mungkin sulit. Jika ada rencana individual, tidak semua orang siap membayarnya dari kantong mereka sendiri. Di sisi lain, produk terbuka atau gratis yang dibuat oleh akademisi sering kali menderita desain yang tidak nyaman dan kurangnya dukungan.

Anda dapat mencoba berbagai alat sebelum memutuskan untuk menggunakan versi yang berfungsi. Kami memutuskan untuk berbicara tentang Hacker News dan koleksi GitHub yang paling terkenal.

Sumber


DOAJ adalah katalog kuratif jurnal ilmiah terbuka. Dalam basisnya - lebih dari 4 juta artikel dari 13 ribu publikasi yang mewakili 130 negara. Dalam waktu dekat, situs ini berencana untuk secara signifikan meningkatkan jumlah majalah kemanusiaan, sangat terspesialisasi, dan non-Inggris. Menariknya, DOAJs menyediakan API terbuka untuk mengotomatisasi pencarian pada berbagai parameter.

Peerus - membantu Anda melacak publikasi topikal terbaru. Setiap hari, arsip diperbarui dengan rata-rata 7 ribu materi baru. Peerus menyediakan integrasi proxy universitas untuk unduhan sekali klik.

Crossref adalah arsip terbuka untuk penerbitan dan pencarian karya ilmiah. Tujuan dari proyek ini adalah untuk menyatukan para ilmuwan dari seluruh dunia di satu lokasi sehingga mereka dapat bersama-sama memecahkan masalah penelitian. Situs ini telah mengumpulkan seratus juta entri. Anda dapat menemukan konten yang Anda butuhkan dengan metadata dan kata kunci.

SciRate adalah aplikasi web penelitian penelitian di arXiv.org . Ada filter berdasarkan bidang ilmiah dan tanggal publikasi. Selain itu, sistem peringkat pengguna telah diterapkan, dengan dasar setiap artikel diberi peringkat.

ArXiv Sanity Preserver - Mempercepat pencarian Anda di arXiv.org menggunakan algoritma pembelajaran mesin. Halaman awal menampilkan bahan-bahan terbaru bersama dengan anotasi. Anda dapat mencari publikasi dengan kata kunci - di bilah pencarian, dan juga dengan tanda-tanda serupa - menggunakan tautan tampilkan yang serupa, yang terletak di sudut kanan atas dekat setiap dokumen.

OSF adalah hub terbuka untuk mencari dan membuat proyek ilmiah. Di sini Anda dapat menyusun karya penelitian Anda - untuk membangun kontrol versi dan menambahkan tag sehingga orang lain dapat menemukan publikasi. Platform ini melibatkan bekerja di cloud, yang memfasilitasi akses ke proyek-proyek mereka. Pekerjaan dapat dilakukan secara mandiri dan dibagikan dengan anggota masyarakat.

Organisasi dan kutipan


doi2bib adalah mesin pencari untuk bekerja dengan daftar sumber. Pengidentifikasi objek digital (DOI) dimasukkan ke dalam bilah pencarian, dan sistem menampilkan metadata untuk bahan yang ditemukan untuk mengatur daftar bibliografi dalam format BibTeX .

JabRef adalah aplikasi yang mengoptimalkan kerja dengan bibliografi dalam format BibTeX. Daftar yang dihasilkan diekspor ke HTML, Docbook, BibTeXML, MODS, RTF, Refer / Endnote, OpenOffice dan LibreOffice. Anda dapat menjalankan utilitas pada Java VM ( versi 8 ), di bawah Win, Linux, dan OS X.

Mendeley adalah daftar manajer bibliografi dan platform cloud yang memungkinkan Anda untuk mengakses file proyek dari perangkat apa pun. Mendeley juga menawarkan peluang konsultasi dan alat komunikasi.

Docear adalah alat terbuka untuk mengatur sumber dan membangun karya ilmiah berdasarkan pada mereka. Pengguna perlu menentukan folder yang akan memainkan peran "perpustakaan" bahan untuk proyek tertentu. Program akan secara otomatis memindai dan menambahkan file yang ditemukan ke basis datanya. Basis data dapat disusun dalam bentuk "pohon", menggabungkan pekerjaan dalam satu arah dalam satu cabang. Komentar dan sorotan dalam file pdf diimpor secara otomatis dan juga ditampilkan di antarmuka. Ketika pekerjaan dengan sumber-sumber selesai, Docear akan membantu untuk membentuk "peta pikiran" terakhir dan memberikan kutipan yang diperlukan.

Kredit ganda - Anda dapat mengutip tidak hanya teks, tetapi juga kode. Jika program Anda meminjam algoritme atau metode dari karya ilmiah, perpustakaan ini akan membantu Anda merujuknya dengan benar. Perpustakaan saat ini tersedia dalam Python.

Bekerja dengan dokumen


Sphinx adalah alat untuk "mengkompilasi" dokumen di reStructuredText dalam HTML, ePub, Texinfo, halaman manual dan teks biasa. Ini mendukung 50 ekstensi , pengindeksan otomatis komponen perangkat lunak, membuat tautan untuk fungsi, kelas, kutipan, istilah, dan sebagainya. Untuk pemula, pengembang telah menyiapkan tutorial dan panduan untuk memulai dengan cepat . Selain itu, kami sarankan Anda melihat topik berikut untuk bekerja di Sphinx: rtd - mendukung Sphinx dan Read the Docs ; Bootstrap adalah tema di mana templat CSS dan ekstensi JavaScript dari kerangka Bootstrap terintegrasi dengan fungsi Sphinx: navigasi, menu hierarkis, dll.


Foto pernikahan freddie - Unsplash

Git-scribe adalah alat sumber terbuka untuk membuat buku elektronik. Menyediakan fungsionalitas untuk penataan teks, pengeditan, proofreading, koreksi, kolaborasi, serta penerbitan dan ekspor ke PDF, mobi dan epub.

Jupyter Notebook adalah editor teks dengan lingkungan bawaan untuk pemrograman interaktif dengan Python. Sebelumnya dikenal sebagai iPython Notebook. Ini dapat diinstal secara lokal dan VPS, ada versi cloud dari produk.

Tekstur - antarmuka editor ini memberikan akses ke berbagai fungsi khusus - seperti membuat grafik, formula, dan berbagai metode kutipan. Program ini menggunakan format file khusus, yang, jika diinginkan, dapat "dibongkar" dan dibongkar menjadi komponen. Menurut pencipta Texture, ini memungkinkan penerbit untuk menyederhanakan proses pengeditan dan peninjauan.
