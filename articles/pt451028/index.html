Guia: tipos de ensaios em inglês e dicas para escrevê-los



Escrever um ensaio é uma tarefa comum enfrentada por quem estuda inglês. Muitas vezes, os ensaios são usados ​​como lição de casa, parte de testes e, é claro, como um dos testes de admissão em instituições de ensino.

Encontrei um post interessante que fala sobre que tipos de ensaios são usados ​​com mais frequência e dá dicas para escrever esses textos. Apresento a sua atenção uma versão adaptada (e aumentada) deste material em língua russa.

O que é um ensaio em inglês


Em inglês, um ensaio é um dos gêneros de obras escritas que se assemelham ao "raciocínio de ensaio" habitual no espaço pós-soviético. Ao escrever um texto, a tarefa do autor é muitas vezes transmitir sua opinião e justificá-la; em outros casos, investigar uma pergunta para fornecer ao leitor espaço para suas próprias conclusões.

Isso deixa uma certa impressão no estilo narrativo. Muitas vezes, é conduzido na primeira pessoa do singular, as palavras conectivas são amplamente usadas no texto, o vocabulário emocional e as metáforas são encontradas.

Existem vários ensaios de vários tipos - em várias fontes eles são distinguidos de vários principais, até dez ou mais. Hoje vamos nos concentrar nos tipos básicos de ensaios.

Ensaios Narrativos


Um dos tipos mais populares de ensaios. Ao escrever um texto, o autor deve contar uma história baseada na experiência da vida real. Tais histórias são geralmente escritas em formato de primeira pessoa, e a narrativa leva o leitor a conclusões independentes.

Ensaios descritivos


Nesse caso, o objetivo principal é desenhar uma imagem para o leitor usando palavras. Normalmente, o autor descreve situações, pessoas ou objetos, apresentando uma descrição simples de um significado mais profundo. Normalmente, esses textos afetam as emoções dos leitores, a tarefa também é levá-los a algumas conclusões com base no que lêem.

Ensaios Analíticos (Ensaios Expositivos)


Textos desse tipo geralmente incluem a análise de um determinado tópico, que é realizado usando fatos, estatísticas, links para vários estudos e relatórios. Ou seja, os fatos são mais importantes que as emoções em ensaios descritivos. Portanto, mesmo em uma apresentação em primeira pessoa, o autor deve evitar expressar sua própria opinião ou expressar suas emoções.

Ensaios Persuasivos


Como o nome indica, a principal tarefa desse tipo de ensaio é convencer o leitor de algo usando fatos, lógica e descrições de situações relacionadas ao tópico. Nesses textos, é bom usar opiniões de especialistas, exemplos práticos etc.

Como dito acima, existem significativamente mais tipos de ensaios em inglês, mas esses são os principais. Agora, vamos às dicas para escrever textos desses tipos.

Etapa 1: formular uma ideia


Escrever pelo menos um texto normal não é tão fácil quanto parece, mas sem uma ideia claramente articulada, a tarefa se torna praticamente impossível. Portanto, antes de começar a escrever, você precisa formular claramente o tópico. Para fazer isso, você precisa decidir sobre pares de teses "argumento-argumento".

Uma tese é um pensamento completo que você deseja transmitir ao leitor. Normalmente, um ensaio deve incluir pelo menos três resumos. Por exemplo, você deseja escrever um ensaio sobre o tópico “O que é ciúme perigoso”, teses para ele podem ser assim:

  • uma pessoa invejosa se torna má, é difícil para ele usufruir dos sucessos dos outros, ela não quer ajudar ninguém;
  • no mundo moderno do gloss e do Instagram, será difícil para essas pessoas se comunicar com outras pessoas, a inveja provoca isolamento e promove a solidão;
  • uma pessoa invejosa se alegra com os fracassos dos outros, não procura combater a injustiça.

Pode haver uma tese, comum a todo o texto. Em apoio a cada tese, é preciso captar argumentos, isto é, fatos e exemplos que os provem. Dependendo do tipo de redação, pode ser uma situação da vida real, uma cotação, um link para um estudo etc.

Etapa 2: desenvolver uma estrutura básica


Simplicidade é difícil. Entre outras coisas, isso significa que, para criar um bom ensaio, você não precisa preenchê-lo com uma enorme quantidade de fatos, seções, belas técnicas literárias. Um dos principais segredos do sucesso é uma estrutura bem pensada.

Ao criá-lo, é conveniente confiar nas teses e argumentos prescritos no parágrafo anterior:

  • introdução - nesta parte são mencionadas as principais teses do texto;
  • a parte principal - aqui estão parágrafos com argumentos, fatos de apoio à tese, que são apresentados de forma adequada para um ensaio de um determinado tipo.
  • conclusão - nesta seção, você não repete a idéia principal novamente, mas tira conclusões com base nos argumentos apresentados.

3. Etapa 3: se necessário, use ferramentas auxiliares


Quando você precisa escrever algo agora, pode ocorrer um problema chamado "medo do lençol branco". Mesmo que o autor saiba sobre o que precisa escrever, dar o primeiro passo pode ser o mais difícil.

Pode haver muitas razões para o medo de um lençol branco: dos efêmeros, como “sem humor”, aos muito específicos, como um medo paralisante de ter uma marca ruim nos trabalhos de casa ou no perfeccionismo excessivo. Obviamente, a falta de prática também não se reflete da melhor maneira.

As experiências mostram que esse problema pode ser resolvido usando modelos de texto. A ideia é simples: é difícil para uma pessoa escrever pelo menos uma frase para que a página não fique mais vazia. Se já existe algum texto que precisa ser alterado, é muito mais fácil fazer isso psicologicamente.

Etapa 4: Subtrair texto


Depois de superar o medo de uma folha em branco e escrever um texto, é importante relê-lo mais algumas vezes e corrigir os erros. Se você escrever on-line, é uma boa ideia usar ferramentas especializadas para revisão e verificação gramatical.
