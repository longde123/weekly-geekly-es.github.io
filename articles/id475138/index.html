5 Cara Menggunakan Raspberry Pi Menguntungkan Bagian dua

Hai, Habr.

Pada bagian pertama , 5 cara untuk menggunakan Raspberry Pi dibahas. Topiknya ternyata cukup menarik, dan hari ini saya akan mempertimbangkan beberapa opsi lagi tentang bagaimana menggunakan komputer mikro ini dengan manfaat.


Foto dari learn.adafruit.com

Seperti pada bagian sebelumnya, saya akan mempertimbangkan metode-metode yang tidak memerlukan pemrograman.
Bagi yang berminat, lanjutkan di bawah cut.

1. Kamera Pengintai



Sumber: www.raspberrypi-spy.co.uk/2017/04/raspberry-pi-zero-w-cctv-camera-with-motioneyeos

Raspberry Pi dapat digunakan dengan hampir semua kamera CCTV.
Dengan Raspberry Pi dapat bekerja:

  • Webcam USB (mis. Logitech C910)
  • Kamera IP (Axis, dll.) Dengan injektor PoE (daya 48V disuplai ke kamera tersebut melalui kabel jaringan, yang memungkinkan mereka untuk dibawa keluar gedung)
  • kamera yang terhubung langsung ke konektor RPi (seperti pada foto di atas).

Ada beberapa opsi konfigurasi perangkat lunak. Anda dapat menggunakan paket Motion , yang memiliki pengaturan yang cukup fleksibel. Anda dapat menulis langsung dari konsol melalui ffmpeg, Anda akhirnya dapat menulis handler Anda menggunakan Python dan OpenCV. Anda dapat menyiarkan streaming video, menggunakan detektor gerakan, mengirim foto melalui email, dll.

Mereka yang tertarik dapat melihat tutorial berikut:

  • www.bouvet.no/bouvet-deler/utbrudd/building-a-motion-activated-surity-camera-with-the-raspberry-pi-zero
  • hackernoon.com/spy-your-pet-with-a-raspberry-pi-camera-server-e71bb74f79ea
  • medium.com/@gonzalovazquez/raspberry-pi-and-motioneye-setting-up-up-your-own-video-surveilance-95444e0faad

Penting : pada bagian sebelumnya sudah disebutkan, tetapi lebih baik untuk mengulang. Untuk tugas intensif sumber daya apa pun (yang mencakup pemrosesan video) pada Raspberry Pi, diperlukan catu daya 2.5A bermerek berkualitas tinggi dan heatsink pasif pada CPU diperlukan (Anda bisa mendapatkannya dengan harga murah di Cina seharga $ 1-2, dengan mengetikkan raspberry pi heatsink dalam pencarian). Jika tidak, perangkat dapat membeku, kesalahan salin file, dll. Dapat muncul.

2. Rekaman audio




Dengan mikrofon USB, Raspberry Pi dapat digunakan sebagai perangkat bug yang cukup ringkas untuk perekaman audio. Di sini, sekali lagi, ada banyak opsi untuk mengatur perangkat lunak - Anda dapat menulis file secara lokal ke kartu SD, Anda dapat melakukan streaming ke PC lain atau melakukan siaran melalui jaringan.

Beberapa tutorial untuk diulang:

  • iltabiai.github.io/2018/03/17/rpi-stream.html
  • blog.mutsuda.com/raspberry-pi-into-an-audio-spying-device-7a56e7a9090e
  • www.heidislab.com/tutorials/audio-streaming-with-raspberry-pi-zero-w-and-adafruits-i2s-mic

Omong-omong, dengan mikrofon, Raspberry Pi dapat digunakan dengan Amazon Alexa dan menggunakan perangkat untuk perintah suara.

3. Prof. fotografi


Jangan bingung n3 dan n1. Paragraf pertama berurusan dengan kamera pengintai, tetapi Raspberry Pi juga dapat mengontrol kamera profesional dari Canon, Nikon, Sony, dll. Cukup untuk menghubungkan kamera ke Raspberry Pi melalui USB.


Foto dari www.movingelectrons.net/blog/2017/08/09/Camera-Time-lapse-Controller-with-Python-and-Raspberry-Pi.html

Pustaka gphoto2 dan libgphoto2 memiliki kemampuan untuk bekerja dari baris perintah dan antarmuka untuk Python dan C ++, yang memungkinkan Anda untuk menggunakan Raspberry Pi untuk mengontrol "mirror", misalnya, untuk fotografi selang waktu. Daftar kamera yang didukung cukup besar, dan mencakup hampir semua model, dari yang modern hingga yang lama 10 tahun yang lalu. Libgphoto2 memiliki API yang cukup berkembang , dan tidak hanya dapat mengontrol rana, tetapi juga mengubah pengaturan, mengunggah file, dll.

Ulangi Tutorial:

  • pimylifeup.com/raspberry-pi-dslr-camera-control
  • medium.com/@cgulabrani/controlling-your-dslr-through-raspberry-pi-ad4896f5e225

Omong-omong, Anda dapat menulis gambar baik di kartu memori kamera atau langsung di Raspberry Pi, yang memungkinkan, misalnya, untuk mengunggahnya secara otomatis ke "cloud". Ada juga perpustakaan untuk mengendalikan tidak hanya SLR, tetapi juga kamera astronomi (mis. ZWO ASI), termasuk bahkan pemandu otomatis .

4. Stasiun cuaca


Raspberry Pi "tahu caranya" tidak hanya menjalankan program-program Linux, tetapi juga memiliki periferal yang cukup berkembang - serial, I2C, SPI, GPIO. Hal ini membuat perangkat ini hampir ideal untuk mengumpulkan dan mengirim data dari berbagai sensor - dari sensor suhu dan kelembaban ke dosimeter berdasarkan penghitung Geiger.


Foto dari www.raspberrypi.org/blog/build-your-own-weather-station

Omong-omong, jika Anda benar-benar malas , Anda dapat mengambil data tidak hanya dari sensor Anda tetapi juga dari web-a, opsi ini juga memiliki hak untuk ada. Namun, papan dengan sensor untuk Raspberry Pi mudah dibeli secara terpisah .

Tutorial untuk dipelajari:

  • www.raspberrypi.org/blog/build-your-own-weather-station
  • www.instructables.com/id/Build-a-Raspberry-Pi-SUPER-Weather-Station
  • www.cooking-hacks.com/documentation/tutorials/geiger-counter-radiation-sensor-board-arduino-raspberry-pi-tutorial

5. Konsol game




Menggunakan proyek RetroPie , Anda dapat mengubah Raspberry Pi menjadi emulator "retro" untuk konsol game yang berbeda, dari Atari ke Gameboy atau ZX Spectrum. Anda juga dapat membeli berbagai case, joystick, dll.

Saya jauh dari permainan, jadi saya tidak bisa mengatakan lebih, siapa pun bisa mencobanya sendiri. Beberapa tutorial untuk dipelajari:

  • www.pcworld.com/article/3190347/how-to-build-a-raspberry-pi-retrogaming-emulation-console.html
  • thewirecutter.com/reviews/raspberry-pi-game-console
