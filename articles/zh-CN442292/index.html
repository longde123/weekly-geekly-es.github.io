我们的Sberseasons实习计划有多好？ 我们有许多很棒的有趣项目可供选择。 它们与现代技术体系捆绑在一起，然后您可以将它们放入自己的产品组合中。 另外，它可以与学习相结合。 当然，我们付钱。

立即在18个IT领域提供实习机会。 我们将更详细地讨论其中的一些。



数据工程


对于仅对机器学习有基本了解但对数据库开发和管理感兴趣的人，数据工程是一个很好的机会，可以开始从事数据工作。

在实习期间，学生将创建店面原型，以开发机器学习模型和数据分析。 与受训人员联系的项目示例：

  1. 分析有关公司客户的数据，创建用于查找公司客户的ML解决方案，分析有关投资交易的数据，创建决策算法。
  2. 开发客户体验数字化的概念。
  3. FastGraph图形平台-完善和优化现有算法，实施新的图形算法，参与解决方案体系结构的开发和优化，模拟分析，与外部工具集成，替代解决方案的原型设计。



UX / UI设计


任何产品都必须具有清晰的设计。 我们正等待您一起为客户端应用程序创建用户友好的界面。 这些任务取决于您将参与的项目，但是这里有一些示例：

  1. 根据业务需求设计IOS移动应用程序接口；
  2. 根据分析数据开发用户界面脚本；
  3. 分析和优化现有接口；
  4. 用于可用性测试的原型开发；
  5. 可视化三维空间中的强连通图；
  6. 创建各种UI组件和动画，实现测试模块和动态自定义机制。 创建视觉效果以显示公司的各种财务指标。 互动设计。



数据科学


最抢手的目的地之一。 主要重点将放在原型制作，数据实际工作，参与问题设置，为自动化传输仿真结果以及为实际飞行员做准备方面。 我们拥有一个完整的实验室集群，这使我们能够做得非常非常多，在实习之后，您将能够在简历中记录以下项目之一：

  1. 开发描述顾客偏好的模型，该模型通过交易活动预测顾客的行为；
  2. 分析上诉案文，包括调性分析，上诉案聚类等
  3. 检查ML模型的质量，评估模型对银行业务的经济影响，开发数学测试并进行研究；
  4. 开发新的欺诈检测模型；
  5. 开发反欺诈算法（产生新想法，识别新趋势）；
  6. 分析银行渠道中被阻止和成功欺诈的案例（互联网银行，处理业务，内部欺诈等）。
  7. 进行人员分析项目-根据人力资源数据建立预测模型，
  8. 文本挖掘和模式识别项目。

Python开发


Python是世界上最受欢迎的语言之一。 它通常与数据科学相关，但是我们在银行使用它来开发业务应用程序及其后端。 如果您选择此方向，它将与我们一起完成以下任务：

  1. 创建一个创新的房地产平台；
  2. 研究用于构建银行对账单的QlikView工具；
  3. 语音识别
  4. 分析视频（对视频进行人数统计）；
  5. 开发用于应用ML模型的微服务。
