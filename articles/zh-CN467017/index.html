什么是HED / MRSD，其用途是什么？

对于正常健康的人，原则上讲，以下信息仅是开阔视野的必要条件，但是，如果您遇到严重的健康问题，而科学家只是在尝试有效地治疗这些问题。 还是您只是冒着足够的鲁re风险而想要获得快速的成功（例如在体育运动中），或者只是想给自己塞上学习不好的药物，在猫之下？

首先，值得介绍一些概念：

人等效剂量（HED）是一个人为达到与动物研究相似的结果而需要施用的某种物质的量。

最大建议起始剂量（MRSD）-在主要人体试验中使用的建议最大起始剂量。 MRSD不会对成人和健康人产生副作用。

NOAEL（未观察到不良反应水平）是副作用发生率或严重程度（与对照组相比）没有明显增加的最大剂量，

NOEL-完全没有效果的剂量，

LOAEL-引起副作用的最小剂量，

MTD是证明可接受的毒性水平的最大剂量。

那么一个人应该服用什么剂量的物质？ 最合乎逻辑的答案是剂量取决于体重，但是为什么我们不能只将动物的剂量mg / kg推算给普通人呢？

在某些情况下，例如，如果NOAEL剂量mg / kg对于广泛的物种而言是相同的，则可以计算重量剂量。 在大多数情况下，无疑适用于人类的大型动物也具有较低的新陈代谢速率，其生理过程较慢，即 较大的动物每体重需要较低的剂量。

Freireich等在20世纪中叶有许多研究人员。 （1966）和Schein等。 （1970），建议剂量不应降低至mg / kg，而应降低至mg / m 2，即，不降低至物体的质量，而是降低至物体的表面积。

通常，为了将剂量降低至mg /m²，采用了变氧指示剂（b）等于0.67，但是一些研究（包括Frerich获得的数据）显示系数0.75更适合计算MTD，例如，它是种间致癌性研究中使用的因素 $ inline $ W ^ {0.75} $ inline $ （EPA 1992）。 但是，迄今为止，尚无数据选择将NOAEL转换为HED的最佳方法。 使用以下方法计算了各种动物和人类体重的赔率

$ inline $（W_ {animal} / W_ {human}）^ {0.33} $ inline $

或

$ inline $（W_ {animal} / W_ {human}）^ {0.25} $ inline $

评估这些参数对初始剂量选择的影响。



总结一下总结：

  • 异种指标从0.67更改为0.75对小物种的影响最大，例如，对于老鼠来说，系数几乎变化了两倍。
  • 指标b = 0.75的使用导致使用更高剂量，更具侵略性且可能更具毒性的药物。
  • 有限的数据表明，在各种抗癌药物的研究中，使用b = 0.75是最合理的。
  • 除非另有证明，否则HED将基于指标b = 0.67进行计算
  • 在研究范围内，没有观察到动物体重对HED计算的显着影响。

但是HED并不是终点，因为如上所示，这种剂量因物种而异，所以最终如何计算​​一个人的剂量？

我们采用每种情况下针对最合适物种计算的HED。 简而言之，它通常是最合适的物种，这是为其计算的HED最小的物种，在上面的示例中是一只老鼠。 但是，如果我们进行更深入的研究，我们将面临一个事实，即有一些因素会影响该物种的选择：（1）吸收，组织分布，代谢，药物排泄的差异，以及（2）假设或先前的研究表明该特定物种在这种情况下，视图是最优选的。

一些制剂的决定性方面可能是特定物种表达相应受体或表位的能力。 然后，将选定的HED除以安全系数，得到MRSD。 从历史上看，安全系数是10，但可以根据可用数据增加或减少。

例如，我们可以服用抗癫痫药普瑞巴林（Pregabalin），辉瑞（Pfizer）在其MSDS中为猴子提供NOAEL剂量100 mg / kg，相当于32 mg / kg（HED）或3.2mg / kg（MRSD）。 值得注意的是，欧洲医学机构为临床试验的第一阶段提供的剂量范围为5-300mg，但我找不到报道提到剂量低于25mg的报告。

这就是FDA推荐的用于在人体研究中确定MRSD的方案。
