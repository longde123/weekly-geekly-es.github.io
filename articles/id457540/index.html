Memori Tetap Intel Optane DC, setahun kemudian



Musim panas lalu, kami mengumumkan Optane DC Persistent Memory di blog kami - Memori Optane berdasarkan modul 3D XPoint dalam format DIMM. Seperti yang kemudian diumumkan, pengiriman strip Optane dimulai pada kuartal kedua 2019, pada saat ini cukup banyak informasi yang terkumpul tentang mereka yang sangat kurang saat itu, selama pengumuman. Jadi, di bawah cut - spesifikasi teknis dan model penggunaan Optane DC Persistent Memory , serta segala macam infografis.

Jadi, seperti yang telah disebutkan, modul Optane DC Persistent Memory (Optane DC PM) dipasang di slot DDR4 DIMM standar, namun, penggunaannya memerlukan dukungan oleh pengontrol memori, sehingga memori jenis ini hanya dapat digunakan dengan prosesor Intel Xeon Scalable Gold atau Platinum generasi kedua . Secara total, satu modul Optane DC PM dapat dipasang per saluran memori, yaitu hingga 6 modul per soket, yaitu total 3 TB atau 24 TB per server 8-soket.



Optane DC PM hadir dalam 3 ukuran: 128, 256 dan 512 GB - jauh lebih besar dari DDR DIMM yang saat ini tersedia. Ada dua cara utama untuk menggunakannya dan berinteraksi dengan memori tradisional.

  • Mode memori - tidak memerlukan modifikasi aplikasi apa pun. Dalam mode ini, Optane DC PM digunakan sebagai RAM utama yang dapat dialamatkan, dan jumlah DRAM tradisional yang ada digunakan sebagai cache untuk Optane. Mode memori memungkinkan Anda untuk menyediakan aplikasi dengan jumlah RAM yang signifikan dengan harga yang jauh lebih rendah, yang dapat menjadi penting saat hosting mesin virtual, database besar, dan sebagainya. Harus diingat bahwa dalam mode ini, Optane DC Persistent Memory bersifat volatile, karena data di dalamnya dienkripsi dengan kunci yang hilang saat reboot.
  • Mode akses langsung - aplikasi dan perangkat lunak memiliki kemampuan untuk secara langsung mengakses Optane DC PM, yang menyederhanakan rantai panggilan. Juga dalam mode ini, Anda dapat menggunakan alat penyimpanan API yang ada, yang memungkinkan Anda untuk bekerja dengan memori sebagai SSD dan, khususnya, mem-boot darinya. Sistem melihat Optane DC PM dan DRAM sebagai dua kumpulan memori independen. Keuntungan Anda adalah penyimpanan yang besar, tidak mudah menguap, cepat, dan andal untuk menuntut aplikasi pertukaran data dan kebutuhan sistem.

Pilihan perantara juga dimungkinkan: beberapa bilah Optane DC PM digunakan dalam mode memori, dan beberapa dalam mode akses langsung. Slide berikut menunjukkan manfaat menggunakan Intel Optane DC Persistent Memory untuk hosting mesin virtual.
