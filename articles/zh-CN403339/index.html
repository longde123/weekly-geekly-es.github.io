有微软纪律吗？ Windows 10将在三月和九月严格更新



直到今天，很难猜测微软何时会针对Windows 10推出具有新功能的更新（功能升级）。从2017年9月开始，发布时间表将变得更加稳定：该公司宣布将对Windows 10，Office 365 ProPlus和System Center Configuration Manager进行大规模更新。根据既定的时间表，每年两次外出：9月和3月。 由于经常将Office Pro Plus和Windows 10一起安装和更新，因此更新Office套件和操作系统的组合使许多人的生活变得更轻松。

因此，Windows 10和Office 365 ProPlus的下一次更新将在2017年9月进行，下一次更新将在2018年3月进行。

建立了更明确的规则。 每个版本的Windows 10都将提供18个月的服务和支持。 微软写道，从原则上讲，这与当前的做法是一致的，但是与Office同时退出，官方的政策解释将简化企业系统管理员的工作。 事实证明，为了保持稳定的支持，一年半就可以清楚地安装此更新就足够了。

Microsoft希望软件包中包含半年更新，并且System Center Configuration Manager也将简化按计划进行定期更新的过程。

在批准了新时间表之后，现在已经知道Redstone 3版本的更确切的发布日期（最多一个月）：这是9月。 以前，微软只使用一个指示性术语：他们说这将在“ 2017年秋季”发生。

以前尚不清楚Windows 10更新的开发和发布情况。 微软表示计划在一个日历年内发布两到三个更新。 但实际上，2016年只有一个问世-Windows 10周年更新。 也许现在开发人员已经把事情整理好了，现在他们将纪律地开始遵守时间表了吗？ 尽管微软对9月的截止日期并不十分有信心。 该博客使用的措辞是，下一个Windows 10更新将针对2017年9月。 经理们似乎并不完全确定自己会做什么。

对于Office 365 ProPlus，更新一年发布3次。 此客户端部分可用于许多Office 365订阅程序包，现在还将每年更新两次：9月和3月。 反过来，Office 365在线更新也在不断推出：每个月或更频繁地添加新功能。



更改不会影响具有永久许可证或相同单个应用程序的Office“脱机”版本的用户。 它还不会影响Office应用程序的Web版本，例如Word Online，Excel Online和PowerPoint Online。 六个月的计划仅适用于连接到Office 365服务的Office 365ProPlus。Office365 ProPlus包括以下应用程序：

  • 展望
  • 字词
  • 电子表格
  • 简报
  • OneNote
  • 访问权限
  • 发行人
  • Skype for Business；
  • 商业用OneDrive
  • Office移动应用程序。

顺便说一句，微软警告说，从2020年10月13日开始，Office 365 ProPlus版本将仍然是连接到Office 365的唯一功能齐全的Office客户端。对于永久许可证的用户，将不再保证其完整功能。 该公司提前3.5年警告说，到2020年，Office 2010，Office 2013和Office 2016的脱机客户端可能会与联机Office 365服务断开连接。

许多Microsoft产品也逐渐发生了相同的变化：所有用户都逐渐被逐渐转移到具有固定费用的订阅中。 永久版本将越来越少地更新。 例如，现在通常不知道何时发布Office 2016“永久”版本的下一个更新，此软件包于2015年9月发布，从那时起已经过去了一年半。 如果公司遵循旧的原则，那么我们可以预计在发布后的两到三年内进行更新，即在2017年底或2018年期间进行更新。

微软在单独的博客文章中发布了具有永久许可证的Office 2010，Office 2013和Office 2016的Office 365 ProPlus和脱机版本。

Windows 10和Office 365 ProPlus的更新时间表明确的新闻专门与功能更新有关。 安全更新将按照旧的时间表与功能更新分开发布。

微软表示，Windows 10已经安装在超过4亿台活动设备上，在企业领域，新版本的Windows展示了创纪录的实施速度，而Windows 7和其他先前版本则没有。 如果更新可预测，那么这只会对Microsoft及其产品的形象有所帮助。
