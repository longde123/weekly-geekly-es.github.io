Azure Cloud Shell sur Windows Terminal

Vous pouvez maintenant vous connecter à Azure Cloud Shell via le terminal Windows!



Pour ce faire, nous avons ajouté un nouveau profil par défaut - Azure Cloud Shell, qui vous permet d'accéder aux répertoires / clients Azure via l'application Windows Terminal.

Si vous avez déjà installé Windows Terminal


Vos paramètres Windows Terminal ne seront pas automatiquement mis à jour vers le nouveau profil (car le fichier ne se met pas à jour à chaque fois que vous ouvrez Windows Terminal), voici donc comment l'ajouter manuellement:

  1. Lancer le terminal Windows
  2. Ouvrir les paramètres (à l'aide de la liste déroulante)
  3. Ajoutez ce profil à votre liste de profils (profils d'élément): 

{"acrylicOpacity" : 0.6, "closeOnExit" : false, "colorScheme" : "Vintage", "commandline" : "Azure", "connectionType" : "{d9fcfdfa-a479-412c-83b7-c5640e61cd62}", "cursorColor" : "#FFFFFF", "cursorShape" : "bar", "fontFace" : "Consolas", "fontSize" : 10, "guid" : "{b453ae62-4e3d-5e58-b989-0a998ec441b8}", "historySize" : 9001, "icon" : "ms-appx:///ProfileIcons/{b453ae62-4e3d-5e58-b989-0a998ec441b8}.png", "name" : "Azure Cloud Shell", "padding" : "0, 0, 0, 0", "snapOnInput" : true, "startingDirectory" : "%USERPROFILE%", "useAcrylic" : true}

Une fois cela fait, l'onglet Azure Cloud Shell sera disponible pour vous.

Réglage de base


  1. Cliquez sur l'onglet «Azure Cloud Shell».
  2. Vous serez invité à accéder à «microsoft.com/devicelogin» et à saisir le code affiché.
  3. Après avoir entré le code dans le navigateur, vous devrez vous connecter sous votre compte. Assurez-vous également que vous êtes connecté avec un compte disposant d'un répertoire / client Azure actif.
  4. Retournez au terminal. Après quelques secondes, vous verrez le message «Authentifié».
  5. Seulement dans certains cas : si votre compte a plusieurs clients, vous serez invité à en sélectionner un. Entrez simplement le numéro du client auquel vous souhaitez vous connecter.
  6. Vous serez ensuite invité à enregistrer ces paramètres de connexion. L'enregistrement des paramètres de connexion vous permettra d'entrer dans le système à l'avenir sans passer par les étapes 1 à 5.
  7. Ensuite, l'application commencera à établir une connexion avec Cloud Shell (cela peut prendre un certain temps, soyez patient).
  8. Vous êtes maintenant connecté à Azure Cloud Shell!

Maintenant que j'ai enregistré les paramètres de connexion, voici à quoi ressemblera le processus de connexion à l'avenir (j'appuie sur '0' pour accéder aux paramètres de connexion enregistrés):



Vous pouvez également vous connecter à l'aide d'un autre compte / client en appuyant sur «n» ou supprimer des connexions enregistrées en appuyant sur «r». Ces paramètres seront enregistrés pendant les sessions, donc même si vous démarrez le terminal après quelques jours, vous pouvez toujours vous connecter avec les paramètres de connexion enregistrés sans avoir à rouvrir le navigateur.

J'espère que vous apprécierez la possibilité d'accéder à vos ressources Azure via Windows Terminal. De plus, comme toujours, veuillez signaler tout bogue / problème que vous avez remarqué à notre référentiel sur Github .
