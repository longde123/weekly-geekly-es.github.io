Inovasi entry-level array HPE MSA: flash array yang lebih terjangkau dan integrasi solusi yang kaya

Bagaimana HPE berinvestasi dalam mengembangkan kemampuan array MSA entry-level dengan dukungan SSD untuk perusahaan dengan anggaran dan kebutuhan penyimpanan yang berbeda?

Semakin banyak berita baik untuk pemilik usaha kecil dan menengah yang mencari disk array entry-level - mereka menjadi lebih produktif dan mudah dikelola. HPE terus memperluas susunan MSA untuk membantu Anda mendapatkan lebih banyak kecepatan, ketersediaan data, dan mobilitas. Saya sarankan untuk melihat lebih dekat pada inovasi yang akan memungkinkan Anda untuk mendapatkan lebih banyak dari MSA.



Array entry level yang terbukti dengan ekosistem kelas bisnis sejati dan fitur-fitur baru.

Bukan rahasia lagi bahwa penyimpanan HPE MSA bukan pemain baru di pasar. Selama hampir dua dekade, kami telah meningkatkan dan meningkatkan lini produk ini. Tahun lalu, generasi baru MSA 1050, 2050 dan 2052 diperkenalkan dengan kinerja dua kali lipat dengan harga yang sama dengan generasi sebelumnya. Saat ini, dengan lebih dari setengah juta unit dikerahkan, HPE MSA adalah FC SAN Array industri terkemuka.

Jadi apa yang baru dalam MSA?

HPE Storage File Controller menawarkan layanan akses data blok dan file terbaik di kelasnya. Ambil perangkat HPE MSA 1050, 2050, atau 2052 dan tambahkan Pengontrol File Penyimpanan HPE untuk membuat solusi penyimpanan file dan blok tunggal yang efisien untuk bekerja dengan data terstruktur dan tidak terstruktur. Dengan layanan data canggih dan konfigurasi yang fleksibel, Anda sekarang dapat mengoptimalkan investasi SAN Anda dengan HPE Storage File Controller dengan memanfaatkan kapasitas yang tidak terpakai dari array MSA Anda untuk file Windows bersama. Pengontrol File Penyimpanan HPE dapat digunakan dalam konfigurasi tradisional atau berkelompok untuk ketersediaan data yang tinggi. Pengontrol File Penyimpanan HPE melengkapi lingkungan file terutama berdasarkan Microsoft, terutama ketika server file ketinggalan zaman.

HPE dan Zerto: Replikasi Berbasis Hypervisor

HPE dan Zerto, penyedia cloud, telah bergabung untuk menyediakan Anda dengan layanan yang tak tertandingi yang memberikan solusi on-the-go yang hampir sinkron dengan mobilitas lingkungan kerja dan fleksibilitas teknologi multi-cloud.

Sebagai bagian dari ketersediaan data, kita berbicara tentang mereplikasi MSA atau array lainnya ke MSA untuk menciptakan solusi tahan bencana - dengan RTO dan RPO yang sangat rendah. Ini memberi Anda kesempatan untuk menggunakan array yang lebih murah sebagai penyimpanan korporat yang digunakan untuk toleransi bencana, yang, pada gilirannya, mengurangi biaya keseluruhan. Anda dapat menggunakan HPE MSA dan solusi Zerto untuk meningkatkan produktivitas dan mobilitas lingkungan kerja aplikasi Anda antara dua larik MSE MSE dan / atau sistem penyimpanan HPE lainnya. Aplikasi dan beban kerja yang berjalan pada MSA juga dapat direplikasi ke pusat data primer Anda atau direplikasi ke cloud tanpa risiko. Untuk informasi lebih lanjut tentang solusi ini, unduh White Paper Replika HPE MSA dan Zerto Hybrid.

Platform MSA 1050: Lompatan Kinerja dengan Pembaruan Firmware Baru ke v270

Berkat pembaruan baru, model 1050 termuda mendapatkan lebih dari 30% peningkatan kinerja pada operasi tulis dan baca acak. Ini adalah berita bagus jika Anda menggunakan SSD di sistem Anda. Pembaruan baru sudah tersedia untuk sistem MSA 1050 baru dan yang sudah ada.

Penyederhanaan interaksi dengan hybrid dan array flash.

Kami telah memasukkan jadwal IOPS dalam Konsol Manajemen MSA, yang memberi pengguna gambaran visual tentang bagaimana tingkat penyimpanan pada SSD memengaruhi I / O untuk aplikasi yang dihosting dalam larik. Karena berbagai aplikasi membuat hotspot dalam array pada waktu yang berbeda, mekanisme penskalaan otomatis MSA mentransfer blok atau data terpanas ke level media tercepat (biasanya SSD) secara real time. Bagan IOPS baru memberi pengguna wawasan tentang dinamika ini, yang membuat alat ini sangat berguna ketika Anda bekerja untuk memaksimalkan manfaat berinvestasi di media flash.

Dukungan LDAP: Sederhanakan Otentikasi Pengguna

Kami telah menyertakan dukungan LDAP pada model MSA 1050/2050/2052 dengan rilis firmware V270. Setelah grup pengguna dikonfigurasikan pada semua MSA di organisasi Anda, pengguna dapat diautentikasi pada MSA apa pun melalui Active Directory. Ini memungkinkan Anda untuk memusatkan aturan otentikasi dan menyederhanakan akses bagi pengguna yang memiliki banyak akun untuk beberapa sistem MSA.

Encrypted Drives: Menyebarkan Solusi Hybrid pada SSD / HDD

MSA 2050 sekarang mendukung enkripsi dengan disk spindle yang dapat dienkripsi sendiri dan kapabilitas manajemen kunci lokal di dalam array. Belum lama ini, dukungan ditambahkan untuk dua solid state drive dengan kapasitas 800GB dan 1.6TB. Karena itu, ketika Anda perlu menerapkan solusi yang membutuhkan enkripsi, Anda dapat dengan mudah membangunnya di HPE MSA.

Contoh singkat tentang bagaimana penerapan HPE MSA dengan fitur-fitur baru memungkinkan kami untuk menyelesaikan tugas klien.

Dalam industri apa pun, sangat penting untuk memiliki sistem penyimpanan data yang efisien dan andal - terutama dengan semakin populernya virtualisasi, yang membutuhkan penyimpanan bersama yang terpusat.

Salah satu pelanggan kami, Longview Public School di Washington, menunjukkan bagaimana HPE MSA Flash memenuhi kebutuhan yang terus meningkat akan keandalan, kapasitas, dan kinerja. Beberapa tahun yang lalu, mereka membeli satu HPE MSA SAN Storage array dengan kapasitas total 28,8 TB untuk mendukung tiga-host cluster Hyper-V mereka. Dan Jones, spesialis teknologi di Longview Public Schools, menceritakan kisah penyebaran MSA:

“Kami menghargai HPE untuk keandalan. Kami berdebat tentang apakah akan memilih MSA atau segera menginstal HPE 3PAR StoreServ Storage. Pada akhirnya, kami menyadari bahwa untuk kebutuhan kami, sistem MSA adalah solusi yang tepat dengan kapasitas yang tepat dan dengan harga yang tepat. ”

Jones mencatat bahwa array dengan drive spindle standar dipasang terlebih dahulu. Di masa depan, ketika diputuskan untuk menambahkan rak kedua agar sesuai dengan jumlah disk lebih besar dari 24, untuk memulai virtualisasi server Microsoft Exchange, diputuskan untuk memilih solusi yang lebih produktif.

4 SSD dengan volume 1,6 TB dan perangkat lunak untuk meningkatkan produktivitas secara otomatis dibeli untuk menempatkan beban kerja dengan benar tergantung pada portabilitas. Thin provisioning pada MSA (Thin Provisioning) mengalokasikan sumber daya fisik hanya saat aplikasi mengkonsumsi untuk meningkatkan efisiensi ruang disk yang digunakan, sementara cache SSD meningkatkan kinerja pembacaan acak, memungkinkan Anda untuk menggunakan SSD sebagai perpanjangan untuk cache controller MSA.
