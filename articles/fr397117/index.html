Puces Sailfish OS non évidentes

Bonjour lecteur!

Aujourd'hui, je voudrais souligner certaines des puces non évidentes Sailfish OS. L'article est plus axé sur les utilisateurs de Sailfish OS, mais aussi sur les «adhérents» des autres plates-formes, j'espère que cela sera également intéressant, ils pourront apprécier le degré de commodité et de préparation du système pour l'utilisateur final. Si vous êtes prêt, nous irons sous la coupe!



Saviez-vous que:


1. Si vous maintenez votre doigt sur l'application et retournez le smartphone, l'application ne changera PAS d'orientation.
Imaginons un exemple: vous avez navigué sur Internet à partir de votre smartphone en position debout, puis vous avez décidé de vous allonger (le verrouillage d'orientation est désactivé), afin que l'orientation ne change pas, maintenez simplement votre doigt sur l'écran pendant que le smartphone change de position dans l'espace.

2. Double-cliquez sur le bouton de verrouillage du matériel pour déverrouiller votre appareil et vous serez immédiatement redirigé vers l'écran d'accueil, sans passer par l'écran de verrouillage.
Veuillez noter que le double-clic ne fonctionne pas si vous avez un mot de passe lors du déverrouillage.

3. Depuis la version 2.0.2, une fonction d'enregistrement des appels est apparue dans Sailfish OS.
Pour l'activer, allez dans Paramètres (Applications) - «Enregistrement d'appel».

Remarque:
En général, dans la section Paramètres de la section Applications, vous pouvez configurer de nombreuses applications créées par Jolla lui-même, ainsi que réaliser certaines actions sur les applications Android (vider le cache, arrêter le programme, etc.).

4. Un double-clic sur le bouton de verrouillage, si le téléphone n'est pas verrouillé, verrouillera le téléphone avec des vibrations ultérieures, indiquant que le téléphone est verrouillé.
Exemple d'utilisation: le téléphone est dans votre poche et vous ne savez pas s'il est verrouillé, mais vous ne voulez pas l'obtenir. Ensuite, il vous suffit de toucher le bouton de verrouillage et d'appuyer deux fois jusqu'à ce que vous ressentiez la vibration.

5. Vous pouvez quitter l'application Tutorial tout de suite. Pour ce faire, cliquez avec le bouton droit sur les coins au tout début, en partant du coin supérieur gauche.

6. À partir de la version 2.0.2, un simple navigateur de fichiers «prêt à l'emploi» est apparu, pour le trouver, allez dans Paramètres (Stockage) - Mémoire externe / Carte mémoire (Données utilisateur) - faites glisser de haut en bas - option Gestionnaire de fichiers (Fichier gestionnaire).

7. Depuis la version 2.0.2, il est possible de prendre des captures d'écran, pour cela, effectuez une pression d'une demi-seconde sur les deux boutons de volume.

8. Pour les connaisseurs du Web sémantique : Sailfish OS utilise certaines technologies du Web sémantique:

[nemo@Jolla ~]$ tracker-
tracker-control  tracker-info     tracker-sparql   tracker-tag

Quelques choses relativement évidentes, mais pour une raison quelconque, certains ne les connaissent pas:


-dans l'application Appareil photo (Appareil photo), vous pouvez prendre des photos simplement en cliquant sur le chariot de volume (bien que l'application ne mettra pas à jour la mise au point avant la photo);

-dans l'application Appareil photo (Appareil photo), vous pouvez régler le zoom en rapprochant deux doigts sur l'écran et en vous éloignant l'un de l'autre; :)

-Dans Paramètres, dans l'onglet Gestes, il est agréable d'activer la "Fermeture rapide de l'application" afin que vous puissiez immédiatement fermer le programme d'un glissement de haut en bas;

-ici, dans les gestes, vous pouvez activer svayp de gauche à droite pour accéder à la bande d'événements (événements), au lieu de l'écran d'accueil (accueil);

-si le téléphone est verrouillé, puis double-cliquez sur l'écran, vous serez redirigé vers l'écran de verrouillage (écran de verrouillage);

-Changez les fonds d'écran à votre propre image uniquement à partir de la galerie. Pour ce faire, ouvrez l'image dans la galerie, balayez de haut en bas, puis sélectionnez l'option «Créer une ambiance». Vous serez redirigé vers la page des paramètres d'atmosphère, où vous pourrez ajuster les couleurs à votre goût ou tout simplement la laisser telle quelle et en profiter;

-Dans Paramètres (Paramètres) dans la "Saisie de texte" (Saisie de texte), vous pouvez activer la séparation du clavier en mode paysage;

-dire de mettre Warehouse- client pour Openrepos. Tout d'abord, autorisez l'installation d'applications à partir de sources non fiables: Paramètres - «Logiciels non approuvés», puis téléchargez le fichier .rpm à partir du lien ci-dessus et installez-le à partir de la console ou à l'aide de l'application Navigateur de fichiers. Openrepos est un référentiel dans lequel il existe un grand nombre d'applications qui ne sont pas dans le Jolla Store pour diverses raisons (par exemple, le Jolla Store interdit l'utilisation de démons, voir la FAQ ). Il n'y a pas d'AQ dans Openrepos, ce qui comporte certains risques, mais donne en même temps une liberté totale au développeur.

Bonus


Plusieurs applications qui peuvent vous sembler utiles (à mon avis):
- SearchNemo - recherche de fichiers / texte spécifique dans des fichiers texte à travers le système de fichiers;

- LLs vPlayer (uniquement dans Openrepos) - un lecteur vidéo omnivore avec une quantité décente de goodies qui ne sont pas dans un lecteur régulier;

- Message d' écran - affiche le texte tapé à l'écran (par exemple, peut être utilisé pour les notes);

- Sidudict est un bon dictionnaire hors ligne basé sur QStarDict. Moins: si vous ajoutez un grand nombre de dictionnaires, l'application se bloque ou démarre trop longtemps, car chaque fois que vous commencez à réindexer les dictionnaires à partir de zéro;

- navigateur de fichiers- Un excellent navigateur de fichiers pour Sailfish OS;

- Listes - ToDo-organisateur. À mon avis, l'auteur a battu de façon très intéressante la fonctionnalité de la couverture;

- Phare - moniteur système pour Sailfish OS;

- Battery Log - une application pour surveiller la consommation de la batterie, il peut construire un graphique;

- Unplayer (uniquement dans Openrepos) - un excellent lecteur de musique avec la possibilité de lire de la musique directement à partir de répertoires;

- Kat est un client VK en développement actif;

- Superposition de la batterie - s'il est inhabituel pour vous que l'indicateur de charge se trouve uniquement sur l'écran d'accueil, cette application résoudra le problème;

-Enregistreur - enregistreur vocal;

- Textracteur - OCR;

- TAO Translator - un client pour les dictionnaires Google, Yandex, Bing;

- Analyseur WiFi - affichez le nombre et la puissance des réseaux WiFi;

- TinyEdit - un éditeur de texte avec un minimum de fonctionnalités;

- Papocchio - vous permet de dessiner sur l'écran.
