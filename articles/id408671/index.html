Mail.ru meluncurkan beta "Disk-O:" untuk menghubungkan awan seperti disk

Dalam waktu kurang dari dua lima tahun, ketika mereka akhirnya merilis opsi yang nyaman untuk menghubungkan Mail.ru Clouds (bukan WebDAV) dan cloud perusahaan lain. Belum ada berita resmi, dan karena ini mungkin menarik bagi banyak pengguna perangkat seluler, saya memutuskan untuk menerbitkannya di sini.



Di habragike sudah ada banyak publikasi tentang Cloud Mail.ru. Saya merekomendasikan untuk mencatat bahwa Cloud Mail.ru dapat dihubungkan sebagai WebDAV (+ dapat mengunduh file besar dan enkripsi), dan juga menggunakannya melalui plug-in yang sangat nyaman untuk Total Commander .

Sejauh ini, beta disk-o.cloud hanya mendukung Cloud Mail.ru (gratis; berjanji untuk mendukung sejumlah perangkat dan disk ke-1) dan Yandex-disk (gratis hingga 1 Februari), tetapi mereka berjanji untuk segera menambahkan layanan lain. Jadi, di antara tugas utama, dukungan ambulans untuk Dropbox, Google Drive, dan OneDrive diumumkan.
