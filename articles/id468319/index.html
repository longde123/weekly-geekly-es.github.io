Tema Gelap Baru Tersedia di Visual Studio App Center

Kami berkomitmen untuk membangun Pusat Aplikasi Visual Studio untuk Anda. Berkat Anda meluangkan waktu untuk meminta fitur melalui repo Github kami, kami dengan senang hati mengumumkan bahwa tema gelap tersedia di App Center.

Di beberapa layar berikutnya, Anda bisa mendapatkan gambaran tentang bagaimana tema gelap App Center terlihat:


App Center Distribute in Dark theme


Uji App Center dalam tema Gelap


App Center Analytics dalam tema Gelap

Artikel ini di blog kami .

Cara Memulai dengan Tema Gelap


Tidak pernah semudah ini! Untuk penggemar tema gelap itu, Pusat Aplikasi akan secara otomatis default ke pengaturan tema mesin Anda. Namun, Anda dapat mengubahnya di App Center kapan saja. Buka pengaturan akun Anda dan klik Antarmuka Pengguna . Di halaman itu, Anda dapat memilih tema Gelap / Terang sebagai pengaturan yang Anda inginkan.


Konfigurasi tema gelap

Berikan Tanggapan Anda


Meskipun kami telah menghabiskan beberapa bulan terakhir dalam pratinjau pribadi mengidentifikasi dan memperbaiki bug untuk tema gelap kami, perubahan ini menyentuh semua layanan di seluruh produk. Jadi, kami menyadari ada kasus tepi untuk konfigurasi unik yang mungkin belum ditemukan. Kami mendorong komunitas untuk melaporkan bug apa pun yang mungkin kami lewatkan saat menggunakannya! Untuk melakukannya, Anda dapat menghubungi kami melalui dukungan atau repo Github kami.
