Save File Me - layanan cadangan gratis dengan enkripsi sisi klien

Setiap administrator tahu betapa pentingnya membuat cadangan harian.
Bagi saya, layanan cadangan "ideal" adalah:

  • Kurangnya pendaftaran, mengikat ke alamat surat, dll.
  • Kemampuan untuk membuat akun independen untuk setiap proyek
  • Enkripsi sisi klien, termasuk nama file
  • Penyimpanan sejumlah "versi" file yang tidak terbatas
  • Sumber terbuka klien sehingga Anda dapat mempercayai enkripsi
  • Satu file biner, tanpa dependensi, sehingga nyaman untuk digunakan
  • Jika mungkin gratis :)

Saya memutuskan untuk membuat layanan seperti itu, dan inilah yang saya dapat:

Setelah memuat klien, Anda dapat segera mulai menyimpan file yang diperlukan. Layanan ini sepenuhnya anonim dan tidak memerlukan pendaftaran apa pun.

Di bawah "kap" pada saat ini, klien membuat kunci unik untuk akun dan untuk enkripsi file. Dan file itu sendiri dienkripsi dan diunggah ke server: 
savefileme put < > Active subscription until 2021-01-20T17:08:16Z. put: < >: 8.1MiB (1.2MiB/s) done

Memulihkan file semudah mengunggahnya: 
 savefileme get < > get: < >: 2KiB (41.2KiB/s) done

Anda dapat menyimpan folder dengan file dengan mentransfer data melalui stdin: 
 tar cz <> | savefileme put --stdin folder.tar.gz Active subscription until 2021-01-20T17:08:16Z. put: folder.tar.gz: 8.1MiB (1.1MiB/s) done

Lihat daftar file yang sudah diunduh: 
 savefileme ls total 1 id size updated datetime name (#files) 64 8539136 2019-01-21T14:13:05Z < > (< >)

Secara default, klien "menciutkan" versi file, tetapi Anda dapat melihat semua versi: 
 savefileme ls -a total 6 id size updated datetime name 44 8539136 2019-01-19T07:49:53Z < > 45 8539136 2019-01-19T12:18:50Z < > 46 8539136 2019-01-20T16:17:42Z < > 47 8539136 2019-01-20T16:24:31Z < > 48 8539136 2019-01-20T16:31:25Z < > 49 8539136 2019-01-20T16:39:19Z < >

Setiap versi diberi pengenal unik, Anda dapat mengembalikan versi tertentu dengan menentukan pengenal: 
 savefileme get --id 46 get: < >: 2KiB (38KiB/s) done

Untuk mengekspor akun ke server lain, cukup salin baris inisialisasi dengan perintah ekspor akun : 
 savefileme account export savefileme account import --secret <  >

Layanan gratis memungkinkan Anda untuk menyimpan file hingga 3 hari, setelah itu mereka dihapus secara otomatis. Biasanya ini cukup untuk cadangan harian untuk pulih dari bencana atau hanya ketika memigrasi data ke server lain. Langganan tahunan memperpanjang periode penyimpanan file hingga 30 hari dengan $ 0,99 per bulan (termasuk PPN).

Saya berharap layanan Save File Me akan bermanfaat bagi Anda, juga bagi saya.
