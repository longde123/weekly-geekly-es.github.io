提供者的“检查器”-俄罗斯联邦中监视网站拦截的系统的硬件模块

今年秋天，俄罗斯联邦联邦政府下属企业中央联邦区FSUE射频中心（FSUE GRCHC）发起了招标，以创建一种系统来控制电信运营商对网站拦截的执行。 3家公司参加了比赛，MFI Soft LLC成为获胜者。同一家企业是SORM的开发商。该组织要求的预算为8400万卢布。Roskomsvoboda 写道

，“检查器”系统被称为监控站点阻止的平台，应于2015年12月21日至25日投入运行。因此，在不久的将来，在俄罗斯运营的大多数电信公司将不得不在家中安装硬件监视设备。什么是“审查员”？

该系统的全称是“用于监视电信运营商是否遵守2006年7月27日第149-号联邦法律“关于信息，信息技术和信息保护”（AS“审计员”）的规定的自动化系统。”

锁定控制。硬件模块“ Agent Inspector”

控制锁的安装说明。硬件模块“ Agent Auditor”的通行证

同时，据一些用户称，该系统是TP-Link便携式无线路由器的完整模拟。而是以该路由器为基础创建了“检查器”。
