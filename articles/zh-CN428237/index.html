Scrum-mitap与棋盘游戏：邀请参加Scrum Values游戏

我和AgileVerse将于11月13日在棋盘游戏中度过一个温暖的夜晚。 Scrum Values游戏将代替您D＆D，Munchkin或其他经典游戏，而是一款等待您的新游戏，它将帮助您了解Scrum框架的价值并将它们与普通案例联系起来。 （是的，不要搞乱真实的项目-游戏是一样的！）



并发之后， AgileVerse会随游戏分发 100个盒子，以便您可以将其展示给同事和朋友。 细节剪下。

Scrum价值观游戏从何而来？ 一切始于这样一个想法，即Scrum实施中的许多问题都与以下事实有关：不尊重方法论的一个或多个价值-勇气，专注，奉献，开放，尊重。 AgileVerse的团队一直把这个想法孵化到2017年11月，当时在敏捷社区的下一次会议上，他们制作了一个原型游戏，其中价值观的状态取决于团队在某些情况下的决定。

然后是新的Scrum Values Game原型。 当第一批技术人员安定下来时，有必要让游戏充满内容。 为了产生情况，他们收集了丝巾，这有所帮助。

然后，在Agileverse的一次日常会议中，突然发现了灵感，并且极大地简化了机制。 这样游戏就变成了现在的形式。 它接受了来自不同公司的志愿者的测试，并在国际敏捷会议2018敏捷岩石会议上召集了一个研讨会。在所有情况下，他们都收到了很多反馈并纠正了一些小障碍。 经过一个月的艰苦努力，更新后的Scrum Values Game于11月13日到达会议。

如您所见，开发历史已经很长了，游戏已经成熟并且可以发布了-我们希望您喜欢这个结果。 在mitap上，我们将在20张桌子上同时进行游戏。 我们认为这对于已经开始学习该框架的人和将要学习该框架的人都将很有趣。
