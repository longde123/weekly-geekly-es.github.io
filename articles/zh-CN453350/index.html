RIT ++ 2019大会堂公开广播

RIT ++是举办互联网专业人士的节日。 就像在音乐节上一样，我们有很多信息流，而不是音乐流派-IT主题。 作为组织者，我们试图猜测趋势并找到新的声音。 今年是“质量”和QualityConf会议。 我们不会在新的解释中忽略我们最喜欢的动机：整体切割和微服务，Kubernetes和CI / CD，CSS和JS，重构和性能。 当然，我们会介绍新的热门话题。 一切都像人一样，包括成群的先进设备，商品和饮料！

最后两个仅适用于节日嘉宾。 但是这些设备将投入广播业务。 按照良好的传统，正厅-也就是最受欢迎的“表演者”-我们在youtube频道上免费播放 。



连接到5月27日9:30的广播，您将看到并听到很多有趣的事情，时间表已缩短。

这是仅一个流的计划，所有这些都在RIT ++ 9（九个！）并行报告流中。 节日结束后几乎所有会议参与者以及一年中的其他所有人都将获得所有录音。 我们建议您订阅新闻通讯 ，以便在其他人之前获得访问权限。

10:00- CSS的状态 / Sergey Popov（HTML学院，联赛A）

16:00- 每年进行5千万部署-亚马逊上的DevOps文化的故事 / Tomasz Stachlewski（Amazon Web Services）
在我们讨论 DevOps文化在 亚马逊 发展中 的作用之后 。 了解 亚马逊 如何以及为什么 从单一方式转向微服务。 我们将看到使用哪些工具和方法来确保新服务的开发速度并在第二次部署中保持灵活性。

17: 00-《前端的新冒险》，2019年版 / Vitaliy Fridman（Smashing Magazine）
