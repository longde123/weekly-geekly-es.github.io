IBM LTO-8 - Maneira fácil de armazenar dados frios



Olá Habr!

Segundo as estatísticas, 80% dos dados estão desatualizados em 90 dias e não são mais usados ​​ativamente. Toda essa matriz de dados precisa ser armazenada em algum lugar e, de preferência, armazenada ao menor custo. E, ao mesmo tempo, tenha acesso fácil e rápido, se necessário.

Recentemente, eles discutem principalmente a transferência e o armazenamento de dados na nuvem, sugerindo que isso resolve o problema de armazenar dados e backups subutilizados. Ao mesmo tempo, esquecendo indevidamente as bibliotecas de fitas. Afinal, são as tecnologias de fita que podem otimizar significativamente o custo do armazenamento de dados. Em 2018, a IBM anunciou uma nova geração de unidades de fita - IBM LTO-8, e hoje quero compartilhar com você uma das opções para o gerenciamento de dados competente.



As unidades de fita continuam a ser uma solução barata e eficiente para armazenar dados frios. O IBM LTO-8 permite armazenar o dobro de dados (em comparação com a geração anterior), usando menos cartuchos e ocupando menos espaço. E, em combinação com o IBM Spectrum Protect, temos a capacidade de gerenciar arquivos, backups e podemos ter certeza de que os dados estão protegidos.

Provavelmente, você não deve repetir mais uma vez que seus dados são seu ativo mais importante. Que eles estejam sempre com você.
