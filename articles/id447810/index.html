Analytics untuk Layanan Azure DevOps sekarang tersedia untuk umum

Laporan adalah peluang penting bagi pengguna Azure DevOps yang mengandalkan Analytics (Layanan Azure Analytics) untuk membuat keputusan yang didorong data.

Hari ini, kami dengan senang hati mengumumkan bahwa fitur Analytics berikut akan disertakan dalam Layanan Azure DevOps tanpa biaya tambahan. Dalam waktu dekat, pelanggan akan melihat perubahan ini di akun mereka.



Fitur Analytics Sekarang Tersedia di Layanan Azure DevOps


  • Widget Analytics - modul khusus yang menampilkan data di dasbor dan membantu melacak kemajuan.
    1. Burndown and Burnup - memantau kemajuan pekerjaan di area tertentu untuk periode waktu tertentu.
    2. Cycle Time dan Lead Time - memvisualisasikan siklus pengembangan produk di tim Anda.
    3. Cumulative Flow Diagram (CFD) - melacak item pekerjaan saat melewati berbagai negara.
    4. Velocity - melacak bagaimana suatu tim menambah nilai pada beberapa sprint.
    5. Tren Hasil Tes - melacak tren tes, mengidentifikasi kegagalan dan durasi tes untuk satu atau lebih jalur pipa (Pipa Azure).

  • Dalam Pengalaman Produk - Analytics berjalan di dalam DevOps Azure dan di luar dasbor, yang menampilkan data dan analitik.
    1. Laporan Uji Gagal Teratas - dapatkan gagasan tentang tes gagal paling serius di pipa Anda.


Kami akan terus menawarkan Integrasi BI Power melalui Tampilan Analytics dan akses langsung ke OData Endpoint dalam mode pratinjau untuk semua pelanggan Layanan Azure DevOps. Informasi lebih lanjut tentang model penetapan harga untuk mengintegrasikan Power BI dan OData harus diharapkan pada Juni 2019.

Pelanggan Layanan Azure DevOps saat ini yang memiliki ekstensi Analytics diinstal di Marketplace dapat terus menggunakan Analytics seperti sebelumnya, dan mereka tidak perlu mengambil langkah tambahan apa pun untuk mendapatkan Analytics. Karenanya, kami akan mengutuk ekstensi analitik Marketplace untuk pelanggan yang diinangi.

Azure DevOps Server 2019


Untuk Server Azure DevOps, fungsi Analytics akan tetap dalam mode Pratinjau sebagai ekstensi yang dapat diinstal di pasar lokal dan akan tersedia untuk umum dalam rilis utama berikutnya.

Azure DevOps Analytics adalah masa depan pelaporan, dan kami akan terus berinvestasi dalam fitur-fitur baru yang disediakan oleh Analytics. Untuk mempelajari lebih lanjut tentang Analytics dan peluang yang saat ini disediakannya:

  • Lihat dokumentasi tinjauan umum analitik kami.
  • Lihat fitur utama: Widget Analytics , Laporan Uji Gagal Teratas , Integrasi BI Power, dan OData Endpoint .
  • Tonton video ini di Saluran 9 tentang Azure DevOps Analytics .
