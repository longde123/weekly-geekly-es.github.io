JetBrains Night Moscow: bagaimana keadaannya

Pada bulan April, kami mengadakan JetBrains Night Moscow - ternyata pertemuan yang hebat dengan teman-teman: topik menarik, banyak komunikasi yang hidup dan, tentu saja, pidato oleh pembicara JetBrains. Lebih dari 700 tamu mendatangi kami. Terima kasih kepada semua orang karena membuat malam ini sangat keren!


Seperti yang dijanjikan, kami berbagi laporan:

  • Menghapus Hambatan terhadap Produktivitas - Hadi Hariri, Kepala Advokasi Pengembang
  • Coroutines di Kotlin pada server - Roman Elizarov, pengembang bahasa Kotlin
  • Pengembangan dukungan Java di IntelliJ IDEA: tidak ada batasan untuk kesempurnaan - Tagir Valeev, Direktur Teknis Java Support Team di IntelliJ IDEA
  • IntelliJ IDEA Tips Pengembangan Lebih Berguna - Trisha Guy, Advokat Pengembang
  • Bagaimana kami mengembangkan WebStorm - Denis Ushakov, Manajer Pengembangan WebStorm; Ekaterina Prigara, Manajer Produk WebStorm
  • Aplikasi tumpukan penuh di Kotlin melalui mata pengembang front-end - Mikhail Krainov, pengembang
  • Evolusi TypeScript: Bahasa dan IDE - Andrey Starovoit, Pengembang Senior WebStorm
  • Buka diskusi dengan tim JetBrains
