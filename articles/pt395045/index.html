11 ótimos PCs de placa única compatíveis com Linux de especificação aberta

O preço de todos os modelos está abaixo de US $ 200. O



HackerBoards.com vem catalogando computadores de placa única há vários anos . Agora, 81 modelos únicos estão representados lá. Escolhi 11 deles que atendem aos seguintes critérios: o custo da placa deve ser inferior a US $ 200, o software deve estar aberto (Linux ou Android). As placas devem ser fornecidas com especificações estendidas, incluindo um diagrama detalhado. Cada diretoria deve ter uma comunidade ativa e suporte técnico para desenvolvedores individuais. Uma placa ideal para um desenvolvedor pode ser chamada de sistema, frequentemente mencionado em fóruns e outros recursos dedicados ao desenvolvimento no campo da eletrônica.

86 Duino Zero



  • Empresa / Recurso - DM&P, 86Duino.com
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - DM&P Vortex86EX (1x x86 a 300MHz)
  • Memória - 128MB RAM
  • Preço - $ 39

É uma plataforma compatível com Arduino de código aberto baseada no processador SoC x86 Vortex86EX de 32 bits. É compatível com sistemas operacionais Windows, Linux, RTOS de 32 bits. A placa é equipada com um barramento PCIE, DDR3, controlador ROM, xISA, I2C, SPI, IPC, Fast Ethernet, FIFO UART, USB2.0 e controlador SD / SATA. O modelo de US $ 69 é muito semelhante, mas já possui suporte para áudio HD e mais extensões de E / S. Há também uma versão do mini-PC EduCake.

A20-OlinuXino-Lima



  • Empresa / recurso - Olimex, OlinuXino, Mouser
  • Página do projeto
  • Processador - Allwinner A20 (2x Cortex-A7 @ 1GHz); GPU Mali-400
  • Memória - 512 MB de RAM DDR3
  • Porta USB-OTG;
  • Conector SATA + conector SATA de 5V;
  • Conector HDMI com suporte para FullHD (1080p);
  • Conector Ethernet de 100 Mbps
  • Conector para baterias LiPo;
  • Conector LCD compatível com os displays Olimex de 4,3 ", 7,0", 10,1 ";
  • Slot para cartão MicroSD;
  • Soquete de depuração DEBUG-UART;
  • 4 soquetes GPIO (160 contatos);
  • Indicadores LED GPIO, bateria, energia;
  • Armazenamento de endereços MAC EEPROM 2K
  • Preço - $ 37


O mini-PC é um computador Linux de placa única bastante compacto, com uma plataforma de hardware aberta e um conjunto muito bom de interfaces periféricas. A versão 84 x 60mm está equipada com uma interface SATA, o que é bastante raro em placas-mãe desse tipo. Existem microSD, Ethernet, HDMI e três portas USB. Você pode conectar um monitor LCD, há suporte para baterias e 160 GPIO.

Borda da caixa de Andrômeda



  • Empresa / Recurso - Marvell
  • Página do projeto
  • Processador - Marvell IAP140 (4x núcleos Cortex-A53 a 1,2 GHz); GPU Vivante GC7000UL
  • Memória - 1 GB de RAM LPDDR3; EMMC de 8GB
  • Preço - $ 75

Este PC em miniatura está executando o Brillo OS, baseado no Android. O conselho possui uma comunidade bastante grande. Há uma especificação completa do sistema, esquemas, documentação, distribuição Brillo. O sistema suporta os padrões de rede Google Weave e Thread. A placa inclui WiFi, Bluetooth, três portas USB, uma porta HDMI e microSD.

Arduino Industrial 101



  • Empresa / Recurso - Arduino Srl
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - Qualcomm Atheros AR9331 (1x MIPS24k a até 400MHz)
  • Memória - 64 MB de RAM DDR2; Flash SPI de 16 MB
  • Preço - $ 39


A placa é feita no formato sanduíche e é semelhante ao Arduino Yun. O Inside é um microcontrolador, processador Qualcomm Atheros AR9331 MIPS com sistema operacional Linino e módulo WiFi IEEE 802.11b / g / n 1x1 2.4 GHz. O sistema é alimentado via mini-USB. Há um LED.

Arduino Yún Mini



  • Empresa / Recurso - Arduino Srl
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - Qualcomm Atheros AR9331 (1x MIPS24k a até 400MHz)
  • Memória - 64 MB de RAM DDR2
  • Preço - $ 60


O quadro é uma versão menor do Yún original. O sistema é baseado no OpenWrt (Linino), com um processador AR9331 de 400 MHz da Qualcomm. Chip - Atmel Atmega32U4. A placa está equipada com Ethernet, USB e microSD.

Arduino tian



  • Empresa / Recurso - Arduino Srl
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - Qualcomm Atheros AR9432 (1x MIPS24k a até 560MHz); Atmel SAMD21G18 Cortex M0 + MCU (48MHz)
  • Memória - 64 MB de RAM DDR2; 256 KB para SAMD21G18
  • Preço - $ 99

Comparado às duas placas anteriores, esta é mais produtiva. Funciona com o rápido SoC Atheros AR9432 WiFi. Há suporte para Bluetooth EDR / BLE 4.0a. Outros módulos incluem uma porta micro-USB, 20 pinos DIO, 12x PWMs e 6 pinos analógicos.

Arndale octa




  • Empresa / Recurso - InSignal, Pyrustek
  • Página do projeto
  • Processador - Samsung Exynos 5420 Octa (4x Cortex-A15 a 1,8 GHz e 4x Cortex-A7 a 1,3 GHz); GPU Mali T628 MP6
  • Memória - 3GB de RAM LPDDR3e
  • Preço - $ 199


Uma das placas mais caras da seleção. Baseia-se no Exynos 5420 SoC da Samsung, existem portas HDMI, eDP e MIPI DSI, além de portas USB. A placa suporta câmera MIPI-CSI, existe um módulo sem fio opcional. Existe um conselho ainda mais produtivo, o ArndaleBoard-K, embora já exija US $ 259.

Banana pi m2



  • Empresa / Recurso - SinoVoip
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - Allwinner A31 (4x Cortex-A7 @ 1GHz); GPU PowerVR SGX544MP2
  • Memória - 1GB de RAM DDR3
  • Preço - $ 58


Em 2014, o projeto Banana Pi se dividiu em SinoVoip e LeMaker. Como na maioria dos outros projetos Allwinner, o suporte ao código aberto foi ligeiramente reduzido. O Banana Pi de núcleo duplo é um clone do RPi Modelo B, enquanto o Banana Pi M2 é um análogo do Modelo B + com um conector de 40 pinos. A placa está equipada com GBE, WiFi, cinco portas USB e várias interfaces para conectar um monitor ou câmera.

BeagleBone Black, Rev C



  • Empresa / Recurso - Beagleboard.org
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - TI Sitara AM3358 (1x Cortex-A8 a 1 GHz) com chips PRU MCU; GPU PowerVR SGX530
  • Memória - 512 MB de RAM; EMMC de 4 GB
  • Preço - $ 48


Este fórum compatível com o Debian apareceu há muito tempo. Destina-se principalmente à indústria. Não é muito adequado para trabalhar com conteúdo de mídia. No entanto, o conselho é bastante popular e, no ano passado, vários de seus clones apareceram ao mesmo tempo. Alguns projetos foram exibidos no Indiegogo.

Osso Beagle verde



  • Empresa / Recurso - SeeedStudio de Beagleboard.org
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - TI Sitara AM3358 (1x Cortex-A8 a 1 GHz) com chips PRU MCU; GPU PowerVR SGX530
  • Memória - 512 MB de RAM; EMMC de 4 GB
  • Preço - $ 39


O conselho é posicionado pelo desenvolvedor como um mini-PC para a Internet das coisas. Quase imediatamente após seu aparecimento, tornou-se popular e agora adquiriu uma grande comunidade. BeagleBone Green (BBG) - Com base na clássica plataforma de código aberto BeagleBone Black (BBB), foram adicionados dois conectores Grove. Ao contrário do BBB, a plataforma BBG não possui uma porta HDMI, o conector do host USB foi movido e o conector do cliente USB foi atualizado para Micro USB. O software BBB é totalmente compatível com o BBG.

Bubblegum-96



  • Empresa / Recurso - uCRobotics
  • Descrição no LinuxGizmos / HackerBoards
  • Página do projeto
  • Processador - Ações Ações de semicondutores S900 (4x Cortex-A53 a 1,8GHz); GPU PowerVR G6230 a 600 MHz
  • Memória - 2GB LPDDR3 de RAM; EMMC de 8GB
  • Preço - $ 89


O conselho é baseado em um protótipo mais ActDuino. As especificações do Linaro 96Boards CE são usadas aqui. Infelizmente, o chip GPS DragonBoard não está aqui, mas há um SoC rápido e 2 GB de RAM. A GPU PowerVR G6230 também é bastante poderosa, portanto, esta placa também pode ser usada para trabalhar com mídia. Dentro, há uma porta HDMI, um slot microSD, uma porta micro-USB e duas portas USB, uma das quais é USB 3.0. Wi-Fi e Bluetooth também estão disponíveis.
