Penyalahgunaan smartphone Meizu Pro 5 Ubuntu Edition

Ini bukan ulasan tentang smartphone atau sistem operasi Ubuntu Touch yang dijalankannya, melainkan deskripsi tentang bagaimana menggunakan teknologi Linux yang ditetapkan pada smartphone untuk tugas-tugas tertentu (yang biasanya digunakan laptop atau komputer desktop).



Sekitar setengah tahun yang lalu (pada hari ulang tahunnya), smartphone Meizu Pro 5 dibeli dalam konfigurasi kelas atas. Pada minggu pertama, Flyme Android Android dihancurkan dan OS Ubuntu Touch OTA-12 diinstal. Saya harus mengatakan segera bahwa OS pada waktu itu (dan sekarang juga) masih mentah, tetapi smartphone melakukan fungsi telepon dengan benar. Panggilan, SMS, distribusi 4G Internet ke laptop melalui WiFi - semuanya berfungsi.

Sekitar waktu yang sama, Raspberry Pi 3 dikonfigurasi dengan Ubuntu Mate - dan percobaan dilakukan untuk menggunakannya sebagai komputer desktop rumah, yang, sayangnya, tidak berhasil - di satu sisi, Pi 3 ternyata agak lemah, dan saya ingin memiliki sesuatu- itu seperti laptop pribadi - karena laptop kerja (disediakan oleh majikan, yang harus bepergian tanpa berpisah) dan iPhone karya digunakan hanya untuk kebutuhan pekerjaan.

Akibatnya, kartu microSD dengan Ubuntu Mate yang terinstal bermigrasi ke smartphone Meizu Pro 5 dan dipasang di folder / home / phablet / PI, dan kemudian, menggunakan perintah sederhana, server VNC dibesarkan dalam lingkungan chroot.

Dengan demikian, setiap saat Anda dapat terhubung ke desktop Ubuntu Mate menggunakan vncviewer atau Microsoft RDP Client dari laptop atau perangkat lain (misalnya, iPhone). Dan jika Anda menghubungkan keyboard dan mouse Bluetooth ke Meizu Pro 5, dan menjalankan vncviewer dari wadah Libertine, maka Anda dapat menggunakan desktop Ubuntu Mate secara langsung di smartphone Anda (gambar yang sangat kecil, tetapi sangat cocok untuk kasus darurat).

Sumber daya smartphone berubah menjadi kira-kira dua kali lipat dari Raspberry Pi 3 - 8 core, 4 Gb RAM - itu sebabnya semua aplikasi "kantor" bekerja dengan sempurna. LibreOffice (artikel ini sekarang sedang ditulis menggunakan Writer), Chromium, Firefox, GIMP, Inkscape, Keepass2 dan sebagainya.
Tentu saja, semua aplikasi terminal ada - pasangan-terminal, konsole, xterm, dll. VPN dengan otentikasi dua faktor (openconnect) - dari lingkungan chroot - juga berfungsi.

Aplikasi Windows. Ternyata, ini juga mungkin.

Pengembang PL / SQL (+ Oracle Instant Client) dan AutoCAD - bekerja melalui wine dan Exagear (emulation i686). Dan cukup cepat - gambar GT ini dibuat di AutoCAD pada smartphone dalam 5 menit - menggambar dengan nyaman!

Microsoft Office (2007) - memulai dan menjalankan, diinstal jika terjadi keadaan darurat, jika Libre Office tidak dapat mengatasinya.



Pembaruan sistem operasi - baik Ubuntu Touch maupun Ubuntu Mate dilakukan menggunakan apt-get.

Di bawah ini adalah deskripsi dari beberapa hal berguna yang telah dicoba.

Pengalihan SMS jika perlu (dengan filter dan penyortiran)


Karena sms disimpan dalam sqlite, Anda dapat memilih sms menggunakan SQL, misalnya: 

phablet@ubuntu-phablet$ echo "select timestamp, message from text_events where message like 'Example%' order by timestamp; " | sqlite3 /home/phablet/.localhost/share/history-service/history.sqlite

dan kemudian beri makan skrip untuk dikirim melalui pipa: 

 /usr/share/ofono/scripts/send-sms/ril_1 555555555 $message_string 0

Menyalin foto, video, dan file lain melalui sftp (dari ponsel ke ponsel)


Anda mungkin tidak tahu di saku mana smartphone itu berada, atau di ruangan mana laptop pasangan berada, tetapi jika mereka berada di jaringan yang sama, Anda dapat mengakses telepon melalui ssh dan menyalin gambar ke laptop: 

 phablet@ubuntu-phablet$ scp -p *.jpg laptop:/Users/wife_login/Pictures/

Enkripsi file


Terutama dokumen berharga dan Keepass dengan kata sandi terletak di folder Cryptkeeper yang terpasang.

Menghubungkan monitor / TV eksternal


Bekerja dengan Microsoft Wireless Display Adapter (Miracast) - Penerima USB terhubung ke TV atau monitor. Pada saat yang sama, Ubuntu Touch masuk ke mode Desktop, semua aplikasi telepon menjadi jendela terpisah pada desktop. Ini jarang digunakan, karena kualitas gambar hanya cocok untuk menonton video, dan bekerja dengan teks kecil sangat tidak menyenangkan - untuk ini, vncviewer digunakan dari laptop.

Distribusi internet dari smartphone


Ini berfungsi cukup baik, kadang-kadang digunakan untuk menghubungkan iPhone untuk berbicara di Skype, atau untuk menghubungkan laptop ke Internet melalui WiFi.

Pemisahan lalu lintas Internet pribadi dan kerja


Jika ada keinginan untuk tidak menghabiskan lalu lintas Internet dari majikan (selama jam kerja, saat di jaringan lokal), Anda dapat menghapus gateway default ke WiFi Tamu di telepon, dan kemudian semua aktivitas jaringan akan terjadi pada jaringan 4G, dan ditampilkan di layar laptop melalui vnc (atau xrdp).



Menghubungkan keyboard, mouse, hard drive USB, drive USB, printer


Koneksi ada menggunakan USB-C ke adaptor USB, di mana Anda dapat menghubungkan hub USB, dan ke sana - mouse, keyboard, dll.

Printer laser warna Samsung ditemukan di jaringan WiFi rumah dan secara otomatis diinstal menggunakan CUPS, dokumen dicetak dengan benar.

Pemutar konsol di terminal untuk mendengarkan musik


Yang menyukai mplayer, Anda dapat mulai memutar musik di konsol: 

 user@ubuntu-phablet$ cd Music; mplayer *.mp3

Dan di tab lain dari konsol - mixer (kontrol volume) 

 phablet@ubuntu-phablet$ padsp aumix

Repositori Mercurial


Menggunakan TortoiseHg, repositori utama ada di smartphone, klon repositori dibuat di beberapa komputer rumah / laptop. Untuk menyinkronkan pada telepon pintar, server Web mulai menggunakan TortoiseHg Workbench.

Mengisi daya ponsel lain dari Meizu Pro 5


Smartphone dapat bekerja selama beberapa hari dengan sekali pengisian daya, sekitar 2 kali lebih lama dari iPhone (saya tidak mengukurnya dengan tepat, pendapat subjektif saya) - yang cukup menyenangkan. Selain itu, ia dapat berbagi muatan dengan telepon lain melalui USB-C ke adaptor USB.

Ringkasan


Kesenjangan kinerja antara ponsel cerdas dan komputer satu dekade yang lalu secara bertahap menyempit, sehingga sebagian besar tugas yang dilakukan pada komputer pribadi sekarang dapat dilakukan pada telepon pintar.
