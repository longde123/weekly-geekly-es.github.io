2003年跳蚤市场的热敏打印机：2019年能做什么？



几个月前，我在Habr上发表了有关2003年发布的HP PDA的评论 ，以及20年前对笔记本电脑的另一篇评论 。 在撰写文章时，我试图了解设备是否可以在2019年使用。 事实证明，这两种设备几乎都是过时的。 顺便说一句，我在西班牙跳蚤市场上买了这一切，非常便宜。

这个星期六，我在那买了另一台设备，一台现代PDA-Brother mw-140bt热敏打印机，售价3欧元。 让我们看看它是哪种小工具。

设计，硬件，功能



特点

  • 打印类型：热敏打印；
  • 打印机尺寸：100 x 160 x 17.5毫米;
  • 连接器：mini-USB 1.1，充电器端口；
  • 纸张格式：A7；
  • 重量：300克
  • 打印分辨率：300 dpi；
  • 指示灯：工作模式指示灯和充电指示灯；
  • 电池：内部，可移动，780 mAh（7.4 V），锂离子电池。





它看起来不像打印机。 看着他在跳蚤市场上，我没有立即意识到这对谷歌当场有所帮助。 打印机出售时带有纸张，文档，磁盘，微型USB电线和电源，但没有包装盒。



该设备是日本制造的，这是不言而喻的。 该表壳由铝制成，带有塑料插件，16年后没有任何反应。



纸张放置在内部，这是一个特殊的纸板盒，里面装有50张A7格式的热敏纸。 为了打开打印机盖，您需要向后滑动一点。

左侧是设备的电源按钮，蓝牙模块的激活按钮，充电器的端口。

前面板顶部的两个LED指示蓝牙是否已打开以及电池已充电。 当电池电量耗尽时，第二个二极管开始闪烁。 如果将设备连接至充电状态，则LED指示灯将以稳定的橙色灯点亮。



设备中电池的出现通常令我感到惊讶。 事实证明，该小工具完全不需要电线即可工作-如果您选择了蓝牙模式，则无需充电或连接到PC。

A7格式很小，如果是一张照片，则打印一张纸大约需要15秒。 只需几秒钟即可打印文本。

问题开始


电池没电

连接到网络后，充电指示灯开始闪烁。 事实证明，这表明电池丢失。 我取下盖子，发现电池已装好。 他取出电池并测量触点上的电压-很明显，电池已放电至零。 显然，自发布以来就没有使用过打印机-内部的密封纸盒也证明了这一点。

锂电池完全放电后（例如，躺卧数年后仍未工作），在大多数情况下不会死亡。 但是电池控制器不允许通过常规方法对其进行充电，充电器通常与打印机的连接，甚至尝试使用外部电源进行充电（将其与电池触点连接）都不会导致任何结果。

恢复电池最安全有效的方法是使用智能充电器IMAX B6 。 我在视频中拍摄了电池恢复情况，以使其更加清晰。 将已充电的电池插入打印机后，电源将打开。 将设备连接到电源后，我看到设备正在充电-充电指示灯以稳定的橙色指示灯点亮。

为了使打印机正常工作而进行了还原。 它可以与电源一起使用，但是我想充分利用移动性。


无纸抓

用纸给打印机充电后，我用USB电缆将其连接到计算机。 Windows 10找到了必要的驱动程序并安装了它们，打印机在系统中变得可用。 磁盘上不需要驱动程序。

当我尝试打印某些东西时，打印机发出了10秒钟的嘈杂声，此后指示灯开始以红色闪烁。 事实证明，纸张没有被捕获，经过多年的不活动，负责拾取纸张的橡胶辊上已经积满灰尘，因此只是滑落了。 解决方法是拿走胶带，将其粘贴在滚筒上，然后撕下。 因此，您需要重复5-6次。 通过反复试验，可以正确地进行所有操作，然后辊开始抓纸，然后进行打印。


蓝牙设定


对于无线连接，已经需要Brother驱动程序。 首先，使用标准的Windows 10方法，找到一个新的蓝牙设备，将其连接（对于我的打印机，PIN码是其序列号的后四位）。

接下来，转到蓝牙设置，选择“高级”→COM端口，创建一个传出端口，然后选择一个将挂在该端口上的设备。

然后，我们安装Brother驱动程序，在安装过程中，应用程序将要求您选择之前创建的端口。 我们选择它，安装完成，仅此而已-打印机可以使用了。

设置手机打印

您也可以将打印机连接到智能手机，但只能连接到Android。 不幸的是，在Apple手机上，蓝牙功能被剥夺了，因此iOS甚至找不到该小工具。

Android会立即查看并拿起设备。 要打印，您需要一个应用程序。 事实证明，Brother仍然支持该打印机，并且它具有自己的应用程序。 下载并配置后，会发生以下情况：







我们加载图片，然后在出口处-Habr页面。 该应用程序允许您打印条形码，图像和文本。



每张纸10美分，以及当前的打印机功能


我将打印机的可能用途分为有用的和纯粹的娱乐性。

有用的：

  • 您可以在发送包裹或信件时打印出地址。 这种格式非常适合粘贴货件。
  • 打印条形码（商店，办公室等）；
  • 从手机的待办事项列表，购物清单等快速打印
  • 为企业家打印支票；
  • 学习语言时，打印带有外来词的卡片。

Sun-ami提出的另一个有用的列表，他对此表示感谢：

  • 如果事先不知道参加者的名单，则在露天活动中列出徽章；
  • 打印房屋附近区域的站点区域的地图，其位置在旅行开始之前是未知的
  • 印有搭便车和公共交通工具的目的地名称的印版；
  • 列出长途旅行中遇到的旅行者名字的车牌；
  • 在没有电源的地方与受邀厨师野餐时列出当前菜单。

一般来说，贸易，运输和仓库企业都使用热敏打印机。 大多数情况下，它们用于打印条形码。

小纸可以用于标记电缆或在物品，书籍等上粘贴库存标签。

娱乐性：

  • 为自己和亲人打印黑白照片。 例如，您可以在早上在冰箱上或直接在海滩上留下漂亮的信息，以便为友好的朋友陪伴您的朋友的所有代表进行黑白照片打印；
  • 列出社交网络中的消息和评论（2014年，专门为这些目的发布了该打印机，价格超过了200美元）。

尽管该打印机于2003年开始生产，但在道德上并没有过时。 现在可以很好地利用它。 制造商仍支持该设备，易损件可在亚马逊，eBay和Brother网站上购买。 50张墨盒的价格为5欧元。 事实证明，印刷成本为每片10美分。

起初，我想进一步转售该设备，但在弄清楚之后，为了风扇，我将其留在了那里。
