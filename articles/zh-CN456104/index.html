Ezblock Pi-无需编程即可编程，这次是针对Raspberry Pi爱好者

不编写代码而编写代码的想法（是的，写作的-现在时态的现在分词从动词的写作，现在就付诸实践）在聪明的人和懒惰的人中已经出现了不止一次。 图形界面的梦想，在其中可以将一个骰子放在另一个骰子上，绘制互连并从可爱的下拉列表中选择对象的属性，然后通过按下神奇的“编译”按钮来获得与另一个等效的工作代码（当然不是那么聪明）一个使用过时的手动打印方法的程序员总是像企业老板一样闷闷不乐，他们梦想着将昨天的学生介绍给编程人员，他们的才智使他不会错过洗手间，所以 初创希望为适当的价格把快乐带给整个世界。 今天，请您注意：

众筹项目 ：Ezblock Pi。
该项目的实质是 ：Raspberry Pi与扩展板一起使用的图形化编程环境。
平台 ：Kickstarter。
项目地址 ： kickstarter.com/ezblock 。
作者 ：张建Ge，刘慧卿。
地点 ：美国，特拉华州，威明顿市。



开发严肃的图形化编程环境的尝试已慢慢变得没有了。 甚至最高领导也意识到编程过程太复杂了，无法放入彩色立方体的Procrustean床上。 幸运的是，仍然有业余程序员，在正在考虑的众筹项目中，有树莓派爱好者。 为了不推广裸机软件，作者使用扩展板对图形开发环境进行了补充，该扩展板旨在促进连接到外部设备的过程。

在项目页面的标题视频中，向我们介绍了两个机器人程序员Robert和Emily。 与每个自重自大的领带和眼镜佩戴者一样，罗伯特（Robert）正在使用显示器和键盘以老式的方式编写Python。 艾米（Amy）充满爱心的双手在框架的边缘上方飞过，拿走了键盘，显示器甚至鼠标，用漂亮的白色平板电脑取代了所有这些。 平板电脑又运行一个名为Ezblock Studio的程序，该程序使您能够以Drag-n-Drop-n-happy的风格为时下流行的IoT编写程序。

自然，虽然罗伯特（Robert）尝试失败（可能是由于使用了游戏键盘），但艾米丽（Emily）的机器人成功地从杯子上浇灌了植物，女孩自己也从机器人那里收到了通知，直接发送到了手机，甚至使用语音控制命令了响应命令。

由于仍然需要使用某种逻辑将正方形粘合在一起，因此在视频结尾处，最终宣布了对编程语言的支持，这些语言是Python和Swift（视频的主要特征是平板电脑，具有苹果徽标）。 直到现在，Amy仍需要敲击屏幕键盘，因为没有人归还她的正常键盘。 Ezblock Studio宣布支持iOS，Android，Linux，Windows和macOS。 大家都很开心 恩，也许吧，除了罗伯特（Robert）在视频中间消失了。 可能陷入狂暴或戒烟。

好吧，我认为，文学就足够了。 让我们不用开玩笑地看看开发者为我们提供的$ 35美元的价格。

最低配置的Ezblock Pi项目包括三个部分：

  • Ezblock Pi板本身，用作Raspberry Pi的扩展板；
  • 基本的一组15个模块（还有一套用于IoT的模块，以74美元的价格出售，价格更贵一些）
  • 访问Ezblock Studio，该软件使您可以通过拖放操作为Raspberry Pi编写软件；
  • 用于组装Raspberry Pi + Ezblock Pi的塑料盒；
  • 指示。

有了案例和说明，我认为一切都清楚了，让我们仔细看一下前三点。

只能通过提及“由STM32控制器支持”和第一个原型的模糊照片来判断Ezblock Pi板的填充物。 显然，板上有TQFP32封装的STM32微控制器。 此封装中最便宜的微控制器STM32L010K4T6（ARM Cortex-M0 +）的批量为100或更多时，价格为€0.737； 最昂贵的STM32F334K8T6（ARM Cortex-M4）-2.79欧元（鼠标价格）。 从外观来看，采用SOT-223封装的3.3 V线性稳定器负责电源，而用于蓝牙的现成模块则类似于ESP12E。 为了与外界联系，在板中心有两个20针连接器和一个面包板区域。

老实说，即使仔细观察了该项目的插图，对我来说，这15个基本模块的基本构成仍然是一个谜。 如果对IoT的整套模块进行完整的拍照和命名，那么包含在初始包装中的基本模块就比在大型汽车展览会之前的新车设计更为机密。 基本集允许您“创建15个不同的项目”，但是插图中有10个纸板箱，其中似乎包含某种电子组件，但是基本集的完整集在任何地方都没有破译。

至于Ezblock Studio，我已经在新闻开头分享了我的怀疑态度。 我认为，可以真正掌握所有选项的系统（我记得：（块编程+ Python + Swift）*（iOS + macOS + Android + Linux + Windows））可能会开发，但是我会提出即使使用某种多功能工具（例如Electron），也要由5个人组成的团队进行5个工作年或一年的工作（您会给多少钱？）。 考虑到开发人员只要求支付10,000美元（该项目看起来非常愉快，所以现在已经收取了其中的400％），所以目前尚不清楚该团队在整个开发期间将吃什么。 值得称赞的是，我必须补充说，Ezblock Studio的第一个版本已在Google Play上提供。

不要误会我的意思，我完全不会因为对中国开发商的消极态度而淹死。 通常，这是既成事实-首先，中国程序员在Google Play和Apple App Store中占了很大一部分，现在他们通过众筹平台获得了成功。 众筹就这么好，几乎所有拥有互联网和银行卡的地球人都可以向世界介绍他的发展，有时甚至可以从中赚钱。 否定性只能由重点过分地从项目的技术部分转移到彩虹营销而引起，这时，[可能的]设计缺陷被掩盖了，而令人愉悦的一面过分富裕。 这是Ezblock Pi演示文稿中的另一个插图：



就像视频博主Evgeny Bazhenov又名BadComedian所说的那样，“作者编辑”已保存。 您是否知道在Raspberry Pi和“振动模块”的帮助下如何保持清醒的头脑和声音记忆来构建IT？ 还是仍在召唤我们集体无意识的人：“看多么酷，买得更快！”？

服用还是不服用？ 首先，我提醒您，已经有509人参与了41,000美元的筹款活动（要求的10,000美元），直到活动结束将近3周。 人们喜欢它。 也许，如果您是Raspberry Pi的粉丝，您还会发现所建议的设计中的积极方面超过了不愿意将价格从$ 35降至$ 179的部分。 也许您也像宣传视频中的Robert一样，厌倦了“编写重复的代码行”。 或者，也许您只是认为这些家伙正在朝着正确的方向前进，并希望通过他们的资金注入来支持他们。 只需记住，相当于35美元的价格就出售了Raspberry Pi本身（我什至不会细说Raspberry Pi Zero和Raspberry Pi Zero W），工程师团队确实必须使用它来创建，而ARM Cortex-A53时钟速度为1.4 GHz，1000 Mbps以太网，Wi-Fi 802.11n和蓝牙4.2。
