Instal BigBlueButton di Ubuntu 16.04

Selamat siang, hari ini saya memutuskan untuk menulis pada akhirnya posting pertama saya.


Saya tidak akan berbicara banyak tentang BigBlueButton, lalu BBB, saya hanya akan mengatakan bahwa BBB adalah:


  • Gratis (Lisensi Publik Umum GNU).
  • Untuk Konferensi Audio Video.
  • Untuk presentasi.
  • Untuk webinar.

Mari kita mulai.


Sedikit latar belakang.


Dalam kasus kami, kami menggunakan Ubuntu 16.04 x64 dan saya menyarankan Anda sama)


Untuk fungsi BBB yang lengkap, yang berikut ini diperlukan:


  1. IP eksternal statis.
  2. Nama domain terikat.
  3. Sertifikat SSL.
  4. Port yang diteruskan.

Bagian satu


Script instalasi cepat


Diperkirakan waktu pemasangan adalah 15-20 menit.
Instalasi sederhana dan tidak mengganggu.


Dalam hal ini, instalasi menggunakan IP eksternal. 


wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200

Menggunakan nama domain eksternal bbb.example.com 


 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com

Menggunakan nama domain eksternal bbb.example.com dan memperoleh sertifikat SSL dari
Mari Enkripsi menggunakan Email info@example.com 


 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com

SSL + Klien HTML5 terbaru 


  wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com -t

SSL + GreenLight 


 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com -g

Semua hal di atas 


 wget -qO- https://ubuntu.bigbluebutton.org/bbb-install.sh | bash -s -- -v xenial-200 -s bbb.example.com -e info@example.com -t -g

Masih menunggu instalasi selesai.


Kemudian saya berpikir tentang bagaimana untuk tidak menyiksa pembaca dan tidak menulis bagian kedua bersama dengan yang pertama, tetapi saya ingat bahwa ada bagian lain dengan instalasi manual.


Bagian Dua


Instal Docker Cepat


Situs web BBB memiliki daftar apa yang tidak akan kami dapatkan di versi buruh pelabuhan


  • Nginx tidak memiliki sertifikat SSL, jadi Anda perlu menggunakan FireFox untuk menggunakan audio / video WebRTC
  • Proses dimulai menggunakan supervisord (bukan systemd)
  • Klien HTML5 mendukung video tetapi tidak berbagi desktop dengan WebRTC

Artinya, Anda tidak akan mengirimnya ke produksi.


Tapi saya akan jelaskan, tiba-tiba seseorang akan tertarik untuk menguji.


By the way, jika Anda menyembunyikannya di belakang proksi terbalik dengan SSL, maka semuanya harus mulai saat itu (saya belum memeriksanya sendiri). 


 docker run -p 80:80/tcp -p 443:443/tcp -p 1935:1935 -p 5066:5066 -p 3478:3478 -p 3478:3478/udp bigbluebutton/bigbluebutton -h LOCAL_IP_ADDRESS

Yah, seperti dulu. Artikel saya berikutnya adalah tentang menginstal semua kesenangan ini secara manual, kecuali tentu saja mereka melewatkannya.


Sirip.
