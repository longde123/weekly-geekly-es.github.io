LED灯的耐用性和减少的光输出

我进行了一个实验，比较了不同LED灯在打开后四十分钟如何降低光通量。


在进行任何测试之前，我一定要对LED灯预热至少30分钟。 事实是，在此期间，任何LED灯都会随着LED和电子设备的预热而降低其光通量。

我使用自制的Lamptest-1黑匣子，测量了打开后的四十分钟内十二种不同LED灯的光通量如何变化。 为了清楚起见，他选择了其中的九个：

宜家Ryet 603.979.60 LED1741G6 ;
欧司朗LEDSCLA60 10W / 827 230VFR E27 ;
爱仕达LED-A60-std 7W E27 3000K 630Lm ASD ;
驼色LED ULTRA LED6-A60-FL / 830 / E27灯丝;
Lisma SDF-6 A50灯丝；
东芝00101315012B ;
产生01327 ;
高斯102802115石墨烯灯丝;
Saffit SBA6010 55004 。

这是发生了什么事。


首先引起您注意的是，Saffit灯的亮度下降幅度更大（以下我将用这个词，不是专业术语，而是家庭概念）。 25分钟后，该灯的光通量下降到初始灯的近70％。 我注意到这是一个非常非常糟糕的灯泡（标明为10 W，实际上为5 W，表示为CRI 80，实际上为72，尚未测量光通量，但我认为它也处于所声明的一半范围内）。 这样的下降表明LED最有可能在过载模式下运行。 这样的灯泡将长时间不工作。

第二个有趣的点是三个白炽灯的曲线图之间的差异。 它们升温更快，稳定更快。 在Camelion灯处，光通量在接通后五分钟内稳定下来，在35分钟内稳定下来仅1％。

一个有趣的图表是带有重型散热器的旧欧司朗灯（该灯重达117克）。 它加热得如此缓慢，以至于即使40分钟也不足以完全稳定光束。

IKEA Ryet的波浪图是由光线的涟漪引起的（其涟漪系数约为13％）。

也许可以通过光通量的降低程度来预测灯的寿命，如果在15分钟内亮度下降了20％以上，则灯过早失效的可能性就很高。

到目前为止，我还没有机会以这种方式测试每盏灯，但是如果您为Lamptest-1制作了特殊的固件并将设备本身做成几块，则可以自动进行处理。
