迷你桌面自己动手做。3.0



自从我第二篇关于UCFF PC外形的个人电脑机箱设计的文章发表以来已经过去了一年多。在这段时间里，我设法启动了生产中的第一批产品，不仅开始了案例的在线销售，还开始了基于该案例的配置的在线销售。特别令人高兴的是，该项目大部分是在Habrahabr社区读者的支持和关注下进行的。

首先，该项目的有趣之处在于其采用非标准的冷却系统方法，以及外壳设计本身的构造以及系统主要组件的放置。我不得不从字面上颠倒一切。这适用于沿相反方向吹的空气，倒置的主板，从散热器吹出的风扇，进出气流的清晰分离。因此，可以在如此紧凑的外壳中放置和冷却电源，并可以安装功能强大的台式机处理器，从而使该解决方案与网顶笔记本电脑和其他基于超低压（U系列）处理器的小型计算机区分开来。

跟随第一版出版物的许多人对于该系列的塑料版本感到失望。从今年年初开始，我就开始进行更新工作，并且由于有了一个成功的平台，要更新外观，仅添加一个细节就足够了。不想进行弯曲，冲压或焊接金属。这将违反外观的完整性，并与设计的基本概念-坚固性相矛盾。因此，我决定返回唯一的真实方法：用一块铝在CNC机床上进行铣削。使用这种重量为5.5 kg的钢筋将产生更新的设计：



目前，在用于计算机自组装的案例行业中，根本不存在类似的方法。基本上，这是铝的挤压，然后在CNC机床上进一步钻孔，或更简单的金属弯曲操作。我们考虑了注塑成型和后续精加工的选择，但是由于铸造中使用的铝成分，该选择不能保证整个零件的表面光滑，也不能保证更漂亮的阳极氧化。用于研磨的材料是铝5052-T32。由于其最佳强度和更可预测的阳极氧化效果，该组合物可用于大多数情况下的产品，计算机，平板电脑和电话。另外，零件的上部也没有遗漏747个通风孔。他们的任务是排出主板下方的热空气，从而冷却了外壳的顶部。将使用单独的高精度机器来钻这些孔。



通常，电子制造商可以节省零件内表面的表面处理。对于专有解决方案，这是有道理的，但在可以自行组装系统的情况下，这是不可接受的。这不再是设计领域的东西。这来自情绪领域。就像一件衬里很烂的漂亮夹克。每个人都看到外面有什么，但您知道里面有什么。



不久前，我对俄罗斯和国外的买家进行了调查。下图中的平均反馈率。如您所见，主要的投诉归结为噪音水平。这是一个非常有趣的功能：购买如此小尺寸的计算机时，我们当然希望安静运行。即使在那里安装了Core i7台式机处理器。但是，此问题必须纠正。



在更新的版本中，将使用Akasa风扇以优化的速度曲线和Enter轴承技术（简单来说是衬套）使用，运行时间长达40,000小时。在以前的设计版本中，使用了带有滑动轴承的风扇，这种风扇产生了一定程度的噪音，无论晚上还是晚上都可以听到。并且以最大速度-独特的嗡嗡声。 Enter Bearing技术没有这些缺点。另一个优点是，铝制部件从风扇发出的噪音要比塑料制件少得多。也就是说，即使使用功能强大的台式机处理器，也可以在基于超低压处理器的Nettop级别上实现噪声特性。主观上，在最大转速下，噪音降低了近2倍。



在新设计中，添加了用于M.2无线适配器的内部双频天线。为了使铝盖不充当屏幕，天线位于外壳的下部。支持以2.4G / 5G频率连接。



测试配置：


主板：华硕H110T（4 x USB3.0，HDMI，DisplayPort，2 x以太网1Gb，音频）
处理器： Intel Core i7-6700（8M，3.40 GHz）
内存：金士顿2 x 4Gb DDR4 2133MHz
固态硬盘：金士顿M.2 2260 SATA III 128G（530/360 Mb /秒）
SSHD： 1TB希捷（ST1000LM014）WIFI
： M.2 WIFI 802.11AC | BT 4.0 Broadcom

Skylake处理器不像其前身Haswell那样炙手可热，并且具有相同的TDP计算值，因此发热更少。现在，不再需要降低处理器电压的建议，而只是希望这样做。在这种模式下，系统消耗的能量少于额定CPU电压下的能量，并且最大负载下的风扇速度也较低。



在此应该注意的是，随着CPU负载的增加，系统其他组件的温度没有变化，相反，它们的散热效果只会更好。这直接影响主板和热组件（例如M.2 SSD PCI-e驱动器）的生命周期。当然，还有内置电源的寿命。顺便说一下，即使在最大负载下，其温度也不会升高到50℃以上。



我对Skylake处理器的游戏功能感到惊讶。大多数具有1366 x 768分辨率和平均图形设置及更高分辨率的游戏都可以玩，并且每秒速度不低于30帧。即使在GTAV中也能正常播放。虽然，我同意IGP HD 530不适合此类游戏。



我们仍然设法在尺寸和速度上都保持UCFF PC尺寸。通过更新的设计，我们能够使外观更具吸引力，降低噪音特性，并在新配置中增加了对Skylake处理器，高速SSD驱动器和M.2无线适配器的支持。在这种情况下，只有2升的体积，所有这些效果都很棒。
