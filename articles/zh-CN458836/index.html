罗宋汤指数。 评估，比较，确定价格/质量比的系统方法

罗宋汤指数BI（borsch index）的主要目的是创建一个公正，全面的系统，用于评估和比较不同地方和机构的罗宋汤。
此外，下面介绍的方法是一种通用方法，不仅可以对罗宋汤进行全面评估，还可以对任何其他产品，设备，产品，工作和服务进行全面评估，最后提供的“公平价格”方法将提供另一个工具来检查您为产品设置的价格或服务 。

BI指数包括对罗宋汤的整体评估（以点表示）以及罗宋汤的成本。

第一阶段 整体评估


为了在第一阶段确定罗宋汤BI的指数，有必要对罗宋汤进行综合评估。

整体评估IA（整体评估）用于对罗宋汤的口味和视觉品质进行全面评估。

整体评估的结果是将一定数量的点分配给罗宋汤（可能最多500点）。 罗宋汤评分的分数越多，就参数的总和而言越好。

第二阶段。 罗宋汤指数


在第二阶段，了解罗宋汤的积分估计，我们确定了罗宋汤指数。

6.我们考虑罗宋汤指数

BI-罗宋汤指数（borsch index）被认为是罗宋汤成本与其综合评估（考虑到通货膨胀）的比率。

罗宋汤指数的数学含义是按通胀调整后的积分评估的1卢布成本。

您也可以说罗宋汤指数是数字化的价格/质量比。

BI =价格/（IA×Infl）

其中BI是罗宋汤指数；
价格-罗宋汤的价格，擦。
IA-（综合评估）根据综合评估的结果，罗宋汤获得的分数；
Infl-（通货膨胀率）当年的官方通货膨胀率，以2018年12月为基准。

相对于2018年的通胀率是固定的，计算方法如下：

1.数据取自俄罗斯联邦中央银行网站的“消费市场统计/通货膨胀”部分，网址为： www.cbr.ru/statistics/macro_sub

2.在整个2019年，Infl为1。
整个2020年通货膨胀率将等于2019年12月至2018年12月的通货膨胀率。
整个2021年的Infl将等于2020年12月至2019年12月的通货膨胀率乘以2020年Infl。

例如，2023年的通货膨胀率Infl将等于：
Infl 2023 = Infl 2022×Infl 2021×Infl 2020，
其中Infl 2022是2021年12月至2020年12月的通货膨胀率，
Infl 2021-2020年12月至2019年12月的通货膨胀率，
2020年通胀-2019年12月至2018年12月的通胀率。
例如，如果这些值分别为4.5％，3.9％，5.7％，我们将获得一个校正因子
Infl 2023 = 1,045×1,039×1,057 = 1,148

然后，对于将在2023年进行评估的罗宋汤，其成本为300卢布，总积分为330分，Infl = 1.148，BI罗宋汤指数将等于：

BI =价格/（IA×Infl）
BI = 300 /（330×1.148）= 0.792

对于2019年所做的所有估计，Infl = 1，因此，如果罗宋汤的价格为300卢布，而其整体评分为330分，我们将得到：

BI =价格/（IA×Infl）
BI = 300 /（330×1）= 0.909

在此示例中，罗宋汤指数的数学含义是，积分评估的1分成本为0.909卢布。

BI罗宋汤指数越低，进行积分评估的费用就越便宜。 也就是说，在具有相同评级的罗宋汤的情况下，在BI指标较低的地方订购该产品更有利可图。
