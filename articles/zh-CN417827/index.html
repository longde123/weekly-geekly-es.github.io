免费Wi-Fi：德国法院废除了咖啡馆侵犯顾客版权的惩罚措施

今年7月，德国联邦最高法院终止了Störerhaftung 的法律概念（ 可以翻译为“服务网络失败”）。 该法律规定，公共Wi-Fi用户下载盗版内容的责任在于提供访问网络（咖啡馆，酒店等）的组织。

法院的裁决应导致该国免费无线上网的积分增加。

细节剪下。


/ 照片 / PD

Störerhaftung-有什么意义？


2010年，德国最高刑事法院判处无线网络所有者负责下载盗版内容。 该法规称为 Störerhaftung。 它说，向其中一位顾客提供免费Wi-Fi的餐馆（或其他场所）的所有者，如果其中一位访客下载盗版内容（侵犯他人版权），将被处以 100欧元的罚款 。

正是由于Störerhaftung，德国拥有免费Wi-Fi 的网吧数量是英国或挪威的一半 。 在全国各地，每10万。人有自由访问网络只有两点。 例如，在少数这样的观点中，可以注意到在柏林开设了许多新教教堂的所谓的“神殿”。 项目目标：通过网络在三千个教堂中分发wifi，以覆盖全国最大的区域。

为什么法律被废除


取消Störerhaftung的原因是TOR网络的两个节点的运营商发生的情况 。 他有几个Wi-Fi点，有人用这个网络分发了盗版的《死岛》。 由于在这种情况下几乎不可能计算出攻击者，因此版权所有者，Deep Silver的开发人员已对网络所有者提起诉讼，要求赔偿一千欧元 。

被告提起反诉，并认罪。 结果，德国联邦最高法院对此表示支持。


/ 照片 / PD

并且，经过一段时间后，联邦议院批准了该法律的修正案，根据该修正案，热点所有者无需承担网络用户侵犯版权的责任。

最初，Wi-Fi提供商担心法律不符合指令2004/48 / EC “关于实现知识产权”。 它指出，版权所有者必须为因侵犯版权而遭受的损害赔偿。 然而，德国联邦最高法院已经消除了这些担忧 。

这应该鼓励企业家发现更多的免费Wi-Fi点。

剩余部分罚款


Wi-Fi提供商不能仅仅提供免费的Wi-Fi。 他们需要适当地保护此网络。 例如，我们将回到“死岛”的“长期游戏”案例。 尽管德国联邦最高法院撤消了针对该热点所有者的所有侵犯版权的指控，但他因未提供足够的网络保护而受到罚款。

这是德国新法律和欧盟指令中的强制性条款。 必要的保护措施包括：阻止对匿名资源的访问（例如，文件共享），所有用户的注册，密码加密等。

接下来是什么


2017年的法律尚未在欧洲法院进行测试，但这只是一种形式：在2016年，该法院已经审议了类似案件，并裁定 Wi-Fi提供商不对第三方侵犯版权负责。 因此，我们可以假设现在的企业家将更愿意组织免费的热点。

请注意，在欧盟，原则上他们寻求增加Wi-Fi覆盖率。 为此，他们启动了WiFi4EU项目 ，该项目必须将免费Wi-Fi “带”到公共场所。 欧盟委员会主席让-克洛德·容克（Jean-Claude Juncker） 承诺 ，到2020年，整个欧洲都将拥有热点地区。
