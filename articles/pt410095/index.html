Entusiasta do computador montou um processador de 8 bits a partir dos materiais disponíveis

Os processadores mais modernos são baseados na arquitetura de 64 bits. Isso, na maioria das vezes, significa que as informações no sistema são transmitidas em blocos de 64 bits. No entanto, processadores com uma arquitetura diferente ainda não foram para outro mundo. Existem até novas instâncias - é verdade que os entusiastas as criam para algum propósito. Anteriormente, um dos mais famosos processadores de oito bits era o Intel 8008 , que apareceu no início dos anos 1970.

Obviamente, esses processadores são bastante primitivos em comparação aos modernos, mas eles fizeram seu trabalho perfeitamente. Recentemente, a rede obteve informações sobre um processador de 8 bits montado a partir de materiais improvisados, que funcionam sem problemas, embora pareça um pouco estranho. Foi criado pelo usuário do YouTube, Paulo Constantino.

À primeira vista, o desenvolvimento de Konstantino parece um monte de fios (à segunda vista também). Mas a estrutura da estrutura inclui não apenas fios. O processador é montado em uma placa de circuito, o que permite criar modelos de uma ampla variedade de circuitos - do simples ao sofisticado.

Segundo Konstantino, ele inicialmente desenvolveu o esquema no papel, e levou dois dias. Depois de iniciar a instalação, ele teve que passar cerca de uma semana no trabalho. Durante a instalação, ele usou todos os tipos de switches para garantir que o circuito criado funcionasse como um processador real.


Seu processador pode reproduzir o tema de Mario, mas programar o sistema é bastante difícil. O problema é que o teclado não está conectado ao sistema, portanto, tudo deve ser feito manualmente. Depois de algum tempo, o desenvolvedor planeja adicionar um teclado para torná-lo mais conveniente. Em seguida, o processador poderá mostrar tudo o que é capaz.

Um processador de 8 bits é um projeto interessante, mas o trabalho mais ambicioso desse tipo é o Megaprocessador . É um processador de 16 bits com quatro registradores de uso geral. Painéis separados exibem circuitos para diversos fins, incluindo memória, unidade aritmética, entrada / saída e outros. Este sistema usa uma unidade ALU de 16 bits e um somador de 16 bits. O megaprocessador é equipado com 10.000 LEDs, cuja intermitência mostra cada estágio do processo de cálculo, mostrando a direção do movimento dos dados de acordo com o esquema. A capacidade da memória é de 256 bytes.

O circuito usa mais de 27.000 transistores - isto é apenas para o bloco de memória. O número total de transistores neste sistema é superior a 40 mil. A massa do processador, que opera com uma frequência de 20 kilohertz, é de 500 kg.


O autor desse desenvolvimento é um engenheiro eletrônico do Reino Unido. O nome dele é James Newman. De alguma forma, ele queria entender os mínimos detalhes no princípio do processo. Bem, a melhor maneira de fazer isso é montar você mesmo o processador.

“Os computadores são opacos. Olhando para eles, não se pode entender como eles funcionam. O que eu queria fazer era entrar e entender o que estava acontecendo. O problema é que isso não pode ser feito com processadores de tamanho regular. Mas podemos construir um grande processador com nossas próprias mãos - e então ficará claro o que está acontecendo lá dentro. Para exibir os fluxos de dados, vale a pena adicionar também LEDs, o que foi feito ”, diz James Newman. Agora o Megaprocessador pode ser assistido em tempo real, e qualquer um pode entender em quais elementos o processador consiste e como cada unidade funciona.

Você pode usar o processador para vários propósitos, mas o engenheiro escolheu o jogo Tetris para mostrar o princípio de operação de todo o sistema. Durante o jogo, todos os processos que ocorrem durante a operação do sistema são exibidos em tempo real usando indicadores. O número de vários compostos no sistema excede 100.000.

Newman gastou quase 40.000 libras em seu projeto. Agora, talvez, já mais, uma vez que ele continua constantemente a melhorar o sistema.

Inicialmente, parecia ao engenheiro que ele poderia implementar seu projeto em um ano. Mas tive que passar mais de cinco anos, apesar do Megaprocessador continuar melhorando.
