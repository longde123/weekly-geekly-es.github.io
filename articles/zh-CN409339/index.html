具有GPS信息或AdvoCam-FD Black-II GPS + GLONASS的DVR概述

直到2016年，俄罗斯AdvoCam DVR的最愚蠢的价格要比同类竞争对手的价格高得多。 此外，该品牌基本上没有生产价格最低的超级预算机型。 就像，用适度的预算来确保可靠性和高质量拍摄是不可能的。 2014年初，该公司发布了AdvoCam-FD Black-GPS-当时最实惠的设备。 好吧，那就是，负担得起的-价格在模型然后成本近130美元。 按今天的标准，这几乎是8000卢布。 那就是制造商当时关于新颖性的内容： “来自优质DVR的预算模型，其中完全保留了视频质量和制造水平，这是更昂贵的AdvoCam黑盒的典型代表。”



您好，Geektimes的普通读者和第一次来这里的人！ 您又是亚历山大·舒布（Alexander Shub），白俄罗斯的不倦记者，还是汽车DVR领域的俄罗斯领先专家。 我不会马上研究有关我自己的故事。

两年来，AdvoCam根本不是“溢价”，或者不仅仅是溢价。 因为该品牌的设备“有效”，包括价格范围“最高5,000卢布”。 AdvoCam-FD Black-GPS怎么样？ 这是需求量很大的第三年，断然不想“退休”。 总的来说，结识-AdvoCam-FD Black-II GPS + GLONASS的版本于2017年底更新为5,600卢布。 猜三遍，有什么了不起的？

没错，添加了GLONASS支持，并且它们还“拧紧”了基于坐标库的关于固定速度控制组合的警告的支持。 其他所有内容-设计和组件均保持不变。 因此，如果您已经熟悉AdvoCam-FD Black-GPS模型，请随时跳过此材料。 如果没有，在您的存钱罐中添加一个体面的中型预算注册商（相当大的预算-例如，这是AdvoCam FD-4的价格为3500卢布 ）。

顺便说一句，今年7月，我的资料发表了“我如何前往苏兹达尔，或几乎是三年来热门影片的概述-AdvoCam-FD Black DVR” 。 因此，对于那些想要了解苏兹达尔之美并在当前评论的前任英雄的参与下阅读评论之旅的人，建议您使用该链接。

该模型如何获得大众认可？ 坦白说，答案并不明显。 实际上，从功能上讲，这是一个不起眼的模型。 为此，市场上充满了全高清摄像机，甚至具有通过支架的直通电源。 我只能假设用户喜欢较高的拍摄质量（当然，对于这个价格段而言）（顺便说一下，到2017年底仍不算丢人），再加上较高的可靠性。 正如AdvoCam工程师告诉我的那样，在发布之后，该设备进行了几次硬件修订，以消除所有潜在的问题区域。 在整个2017年，AdvoCam-FD Black-GPS及其不带GPS的版本始终每月收集约6,000个搜索查询。 为了进行比较：我问了Ash Wordstate ，根据Yandex版本，我最喜欢的最受欢迎的注册服务商的搜索频率TOP-12 在哪里 ？ 其中只有三个超过了4,000个搜索查询。 然后这两个设备是混合的。

包装及设备


在盒子设计方面，AdvoCam具有令人羡慕的恒定一致性 。 我在这里没有看到任何根本上全新的内容-记录时间与不同分辨率的存储卡的容量之间的对应表以及每个控制按钮的说明。 如果您至少有一点与DVR进行“通信”的经验，那么无需学习完整的手册，这种微型说明就足以进一步使用该设备。



打开包装盒后，我发现：

•DVR本身；
•用户手册；
•真空吸盘上的支架；
•使用USB连接器从点烟器充电；
•miniUSB电缆；
•带有软件的磁盘（编解码器，播放器以及PDF中的说明）
•简短的小册子，带有法律解释-发生事故时如何处理视频。 作者是臭名昭著的电视和广播节目主持人，车主法律保护学院院长Victor Travin。



套件中没有完整的存储卡-它们已保存。 制造商为点烟器电源适配器提供了USB接口，以便您可以将智能手机，导航器或平板电脑与记录仪同时连接至一个充电器。 但是，这里的总安培数仅为1 A-仅适用于智能手机，但无法为任何iPad快速充电。

外观和控制按钮


AdvoCam-FD Black-II GPS + GLONASS采用塑料制成的略带细长外壳，手感柔软。 该模型的特点在于其微型参数和低重量-分别为87 x 45 x 17mm和62 g。 但是，尽管整体紧凑，但该设备仍具有令人印象深刻的2.7英寸显示屏。 这里的关键是16 :: 9的宽高比，观看视频时，视频底部和顶部不会出现黑色的长条。 这与具有4：3 2.4英寸屏幕的典型型号形成对比。



自2014年以来，该模型的设计获得了广泛的欢迎，几乎每秒钟中国工厂仍然提供具有这种外观的设备。 我不会讲这个故事的细节，但是AdvoCam-FD Black的“外部”开发是俄罗斯AdvoCam专家的优点。 后来，中国人开始免费使用国内产品。

屏幕几乎占据了设备背面的所有空间。 控制键不在录音机的末端。 因此，开发人员可以腾出空间来进行宽屏显示，同时保持记录仪的较小尺寸。 设备顶部面板上的四个按钮连续排列。 凹槽左侧的两个键负责更改模式和手动开始/停止视频录制。 右侧的按钮具有其他功能-菜单调用，快速拍照和打开/关闭电源。



外壳底部是HDMI，miniUSB端口和麦克风孔。



microSD卡插槽位于刻录机右侧。 导航按钮（“向上”和“向下”）位于左端（从显示屏侧面查看时）。 他们负责浏览菜单项，锁定和解锁屏幕，打开和关闭麦克风。





在设备的背面，我们看到一个镜头，上面有光滑的塑料插件。 “眼睛”右边是扬声器格栅，在下面-重置按钮。 通常，注册服务商的设计是中性且谨慎的。 绝对-不会引起外界的注意，并且不会在旅途中分散驾驶员的注意力。 尽管这显然与大多数其他型号的品牌政策背道而驰，但带有鲜艳的色彩覆盖层。



DVR安装


AdvoCam-FD Black-II GPS + GLONASS包装中还提供了AdvoCam型号的标准吸盘安装座。 支架由通过铰链互连的两个部分组成。 尽管具有明显的移动性，但该设计仍可将录音机牢牢固定在适当的位置，并防止其悬挂。 同时，如有必要，可以通过向正确的方向旋转摄像机来快速更改摄像机的位置。 例如，为了给驾照不足的人拍照，要遵守第20.1条的主题。 CAO RF亵渎。



通过支架向记录器供电，该支架配备有用于将电缆连接到网络单元的适当的连接器。 此解决方案使车载小工具可以移动-您可以随时轻松地将其从支架中取出，而不必担心断开电线的麻烦。 但是，电缆从支架的最高点出来，并且不能将支架安装在挡风玻璃上边缘的旁边，这有点令人沮丧。



录像录像


现在介绍主要内容-录音质量。 有了这个，“新手”（本质上讲的是2014年款）就已经准备就绪（再次，如果我们考虑价格-5,600卢布）。 录制以1920 x 1080像素的最大分辨率和30 fps的频率进行。 硬件没有太大变化-美国Aptina AR0330矩阵，台湾Novatek 96650处理器，对不起，不知道带有五个玻璃镜片和对角线170度视角的镜片是从哪里来的。 老实说，我认为没有任何理由要详细说明填充内容-很难找到从未发布过大量Novatek 96650 + Aptina AR0330模型的注册服务商品牌。 尽管最终结果在设备可靠性和“追求最高的拍摄质量”方面都可以，但仍然取决于工程师的直接性，甚至更多地取决于软件程序员。



在我看来，该模型的起价为5600卢布。 图像足够清晰，中等亮度和对比度。 没有“像素乱七八糟”，车牌在多云的天气中可以从10到12米处看到一条直线。 我怀疑在晴朗的日子里还会摔出几米。

















到了晚上，唯一会让您困惑的不是理想的白平衡，几乎总是会出现特征性的黄色。 这不会影响车牌的清晰度。 反过来，数字的可分辨性更多地取决于相机抑制眩光的能力。 我不能说AdvoCam在100％的情况下都会这样做。 但是，目前以相同的价格制定模型要比许多竞争对手好得多，更不用说价格低于5,000的超级预算员工了，我个人显然吸引了我的眼球。 所有视频示例均以1080p HDR模式拍摄-如果注册服务商原则上具有最大分辨率的Full HD，那么我认为没有理由考虑是否启用扩展的动态范围。

其他选项


当然，首先要注意的是名称中GPS和GLONASS的下划线。 让我提醒您，以前的型号仅配备了GPS模块。 从理论上讲，更新后的设备可以更快地捕获信号，然后在诸如隧道，天桥，车道或密集的季度建筑物等困难情况下更稳定地保持通信。 我之所以写“理论上的”，是因为我在寻找卫星或定期失去联系时也没有注意到前任注册服务商的“体贴”。 但是，如果有人对这些观点提出了要求，那么现在他们（要求）肯定会消失。

除了支持GLONASS，第二个值得注意的创新是速度-AdvoCam-FD Black-II GPS + GLONASS将警告驾驶员接近摄像机和固定式交警雷达。



屏幕显示汽车的当前速度和允许的速度，以及到指定物体的距离。 此外，设备将发出声音信号，说明需要降低该区域的速度。 为了正确操作SPEEDCAM通知者功能，建议不要忘记更新数据库。 “新事物”将大约每月发布一次。



与几乎所有现代DVR（甚至最便宜的DVR）一样，AdvoCam-FD Black-II GPS + GLONASS也具有震动传感器（G传感器）。 G传感器的目的是防止紧急情况剪辑被覆盖或删除。 震动传感器可记录急转弯，旋转，震动，制动和其他紧急情况。 在这种情况下，当前文件会自动从循环记录中排除，并移动到存储卡上的特殊文件夹中。

结论


我再说一遍： AdvoCam-FD Black-II GPS + GLONASS-2014年极受欢迎的模型的修订和改进版本 。 明显的改进-增加了对GLONASS的支持。 混合信号接收模块（GPS + GLONASS）提高了在困难条件下确定坐标的准确性。 并且它可以基于上传的摄像机和雷达坐标库来确保Speedcam的平稳运行。 以前的AdvoCam-FD Black-GPS型号没有Speedcam。

但是事情仍然保持不变。 像“老人”一样，新一代FD Black即使在弱光下也能提供清晰的图像。 然而，以较低的拍摄质量（5,600卢布），它很可能会与更昂贵的“商店里的同事”竞争。
