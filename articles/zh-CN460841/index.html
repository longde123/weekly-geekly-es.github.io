TOP-23语言学习应用

有数百种语言学习应用程序以及数百种评论和比较。 实际上，大多数比较都是关于同一程序的。 这些应用真的有用吗，还是仅广告和市场营销？ 是的，是的。

五年来，我一直在使用各种方法和资源来学习英语。 语言学习不是我最大的才能，但我仅使用智能手机和PC就达到了B2（从A2）的水平。 我发现了一些功能可以真正帮助您学习外语。 其中一些至关重要，另一些则非常有用。 通过切割，您会发现我通过分析这些功能组成的语言学习应用程序的等级，即尽可能客观 。

方法论


如果我们要获得客观评价，则需要客观指标。 评级不是关于精美的设计或可爱的辅助动物，正如我之前提到的，应用程序中有一组功能可以帮助您学习语言。 每个功能都可以通过多种方式实现，但并非每种方式都有用。 因此，可以根据下表估算每种方式：



让我们详细了解每个功能。

1.最重要的功能


1.1。 重复系统

您每天都需要重复单词，重复系统至关重要。 只有一种方法有效： 间隔重复 （50分）。 不幸的是，许多应用程序为您提供了重复您随机知道的所有单词（15分）的功能，这很浪费时间。 任何重复系统都可以帮助您学习1000个单词，但是如果您需要8000个或更多，则必须使用空格重复。

1.2。 准备好的卡片

如果您想说一口流利的外语，那么您需要在应用程序中知道数千个单词和相同数量的单词（50分）。 书籍和电影不如卡片有效，因为稀有单词很少，只有应用才能强迫您接受教育。 许多流行的应用程序仅为您提供3000个单词（25分），因此请当心。 某些应用程序根本不包含卡，它们是无用的（-100分）。

1.3。 多种含义

“绿色”一词不仅是绿色，而且是绿色和绿色。 好的应用可以带给您所有流行的含义。 最好的应用程序应该在较高的级别（50分）为您提供稀有的含义。 不幸的是，许多应用程序只为一个单词提供一种含义，而这仅对初学者来说就足够了（10分）。

1.4。 单词的例子

在大多数情况下，使用单词的示例非常有帮助。 对于我来说，“收购”和“美德”是无法理解的词，没有附带短语。 好的应用包含了最多单词含义的示例（25分）。 这里最好的应用程序是Lingvist：单词总是示例（50分）。

1.5。 习语和短语

仅仅知道单词是不够的，您需要具备说话的能力，并且某些应用程序对此技能非常有用。 短语和习语（卡的翻译）方面的培训使我无需思考即可讲话（50分）。 此功能至关重要。

1.6。 语法-语法

您可以使用教科书或应用程序中的卡片学习语法。 第二种方法通常更有效，因为您真的不需要知道什么是“副词”，而只需要正确使用它（25分）即可。 这与带有短语的卡片非常相似，但侧重于语法。

1.7。 语法-形态

有时，您需要练习如何从单数形式生成复数形式，这种类型的测试很有帮助（15分）。

2.负面因素


2.1无用的教训

真痛苦！ 单词和短语的轻击测试非常流行，而且绝对没有用（-100分）。 如果您需要学习如何写单词，则只需使用标准键盘输入即可。 如果您需要翻译短语，则只需在脑海中思考一下，然后检查结果并在卡片中找到答案即可。 不幸的是，只有少数应用程序不包含点击测试或允许您将其禁用（-10分）。 最好的应用程序必须包含所有类型的测试（甚至是轻击），但允许您禁用其中的任何一个（0分）。

2.2。 声音-单词，短语

没有声音，该应用程序将毫无用处（-100分）。 许多应用使用机器人代替专业演讲者，这足以容纳单词（0分）而不是短语（-10分）。 机器人将以相似的方式读取单词中的单个单词和相同单词。 这不会让您为现实生活做好准备。 理想情况下，每个单词都需要来自不同扬声器（包括非专业扬声器）的几种声音。 收集所有声音非常昂贵，但对于亚罗不是。

2.3自定义开始级别

如果您懂一点语言，则无需学习琐碎的单词和语法。 该应用程序应自行调整至您的水平（0分），否则将浪费您的时间（-25分）。

3.不错的功能


3.1。 口语-单词，词组

说话者的语音通常足以进行语音训练（0分），但是如果应用程序还可以听您说话，则非常有帮助（10分）。 最好的应用程序应该检测并指出您的口音（20分）。

3.2。 统一训练

当您上一堂语法课，另一堂新单词课和第三堂复读课时，总有机会今天忘记其中一个（0分）。 一些应用程序为您提供了所有这些课程的一个按钮：您会随机获得语法，新单词和已知单词，并且不会丢失任何内容（5分）。

3.3。 用户词

有时您遇到一个单词，您需要立即学习。 如果该单词支持此功能，只需将其添加到您的应用程序中，并立即学习（10分）。

3.4。 特别课程

您出于某种原因需要语言：旅行，工作，在线游戏等，因此您需要特殊的词汇。 如果申请很好，它将为您提供其他专业课程（10分）。

3.5。 学习内容

单词和语法不足以促进您的进步，甚至会变得无聊。 外语书籍和电影可以帮助您省下心情，一个好的应用程序会告诉您在哪里得到它们（5分）。 最好的应用程序包括对所有内容的用户评分，以节省您的时间（10分）。

3.6。 老师们

语言学习应用程序可以节省您的预算，但有些事情只能由私人老师向您解释。 您可以在italki.com上或在您的申请表中找到老师（5分）。

3.7。 社交网络

这不是Facebook，而是学习语言的人脉网络。 他们可以互相交谈（5分）或检查您的文章（10分）。

3.8。 离线学习

每天上班途中在地铁上做什么？ 当然要学习语言！ 但是您的应用程序在脱机工作时应该稳定（10分）。

3.9。 网络教育

PC键盘比手机键盘好得多。 如果该应用程序为您提供了网络培训（10分），则您可以更快地学习。

关于一些有趣的应用的几句话：

  • Duolingo是最受欢迎的应用程序。 显然，这是因为Duolingo确实是免费的（带有广告），并且其得分相对较高，因此非常有用。
  • Anki是免费的（适用于Android和Desktop），得分很高，但并不那么受欢迎。 怎么了 它缺乏可用性。 这是我现在用于学习的程序，但是我是一位经验丰富的IT专家，并且使用Python脚本使Anki对我有好处。 你知道Python / Java / etc吗？ 如果是，那么Anki就是给您的。

  • Rosetta Stone使用图片而不是文字进行培训。 对于孩子来说学习英语可能是完美的选择，但对您而言却不是（因为您可以阅读）。 Rosetta Stone是唯一可以检测到语音中口音的应用程序：这是提高英语发音的最强大的程序。
  • Yarrow是最高分数为400分的最佳语言学习应用程序。 唯一的问题是它尚不存在。 这是我自己的项目，如果您对完美的语言学习应用程序感兴趣，请访问该项目的网站yarrow.club 。 Yarrow的目标是像Duolingo这样免费 （付费或说），像TOP-5一样高效，并且根据方法要更好，它具有完善的语音引擎（如Rosetta Stone），但适用于所有流行语言，并且不像Anki那样易于使用 。 我只能在您的帮助下创建此应用程序。
