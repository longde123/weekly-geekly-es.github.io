Intinya adalah bahwa biaya klik secara artifisial sedemikian rupa sehingga tidak ada pengiklan dapat mengiklankan aplikasi pemesanan taksi mobile mereka.

Sedikit latar belakang.
Beberapa waktu lalu, saya mengembangkan layanan yang mirip dengan Yandex.Taxi, dan setelah saya menerima aplikasi seluler, saya memutuskan untuk masuk untuk iklan dan apa yang akhirnya saya lihat membuat saya sangat sedih. Di bawah ini saya akan memberi tahu Anda lebih detail.
