XYZprinting 3D打印机审查与Formnext 2017



在之前的出版物中，我们讨论了来自Formnext 2017的两台XYZprinting全色3D打印机，本文中介绍了该公司的其他新产品。


在XYZprinting展位上，您可以看到大型工业SLS机器MfgPro230 xS，这是一种配备30瓦CO2激光器的大型粉末印刷的经济实惠的解决方案。

XYZprinting MfgPro230 xS


  • 技术：SLS
  • 工作腔，mm：230x230x230
  • 层厚度，微米，从：100
  • 尺寸，毫米：1460x1890x740
  • 重量，公斤：310
  • 二氧化碳激光，W：30
  • 打印分辨率，dpi：1600x1600
  • 电源：220V / 20A，2.5 kW
  • 支持的材料：PA，TPU，PMMA
  • 软体：SLS Build / SLS Ware
  • 印刷速度，升/小时：1
  • 支持的操作系统：Win7 / Win8 / Win10
  • 档案格式：STL，3MF
  • 价格：约60,000美元

一个显着的特点-该设备非常便宜，它将以约6万美元的价格交付，同时具有相当严重的特征：摄像头尺寸为230x230x230 mm，CO2激光和大量支持的材料-尼龙，复合材料，高温尼龙，TPU和PMMA。

主要用途：重工业，汽车工业，航空航天领域。

XYZPrinting Dent Pro 100 xP


  • 技术：DLP
  • 工作室，毫米：6440120
  • 层厚度，微米，从：25
  • 生产国：中国
  • 作业系统：Windows 7，8 10
  • 沿X和Y轴的分辨率，微米：50
  • 价格：约20万卢布


XYZ的下一个重要新产品是DLP 3D打印机，该打印机可以高分辨率打印，满足珠宝商和牙医的需求，并具有高质量3D打印，而价格却很便宜。



大约20万卢布，这台打印机将在俄罗斯购买。 这是最近宣布的新产品，目前尚未发货。



在合适的光敏聚合物中，还有用于燃烧的特殊材料，可用于铸造。



唯一的缺点是相对较小的打印区域，此处为6x3厘米。 但是，对于牙科和珠宝需求，这可能就足够了，尤其是考虑到打印的高精度时。 非常适合打印注射模型。

XYZPrinting MFG Pro600 xT


  • 工作室，mm：580x400x500
  • 细丝横截面，mm：1,75
  • 两台挤出机
  • 印刷温度，℃，至：245
  • 加热平台，℃，至：100
  • 加热相机，℃，至：65

它通过Wi-Fi，互联网连接，可与所有传统印刷材料和异国情调一起使用。


它看起来非常高品质且庄重。 它将花费不到200万卢布，对于具有这种功能的设备来说，这是非常合理的价格。 非常适合打印大型物体。
我们一定会测试这台打印机，然后单独讨论。


多亏了所提供的照片，我们可以说打印机很不错-它可以打印一种具有不同硬度和熔点的材料的模型，并可以提供高质量的大尺寸照片。
