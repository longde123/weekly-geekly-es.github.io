IMTS 2018上XYZprinting的新功能：3D打印机和机器人



在芝加哥举行的IMTS 2018上，XYZPrinting展示了自己最新的内部3D设备：活页夹喷射PartPro350 xBC打印机， MfgPro230 xS SLS打印机以及MfgPro 180 xPf和220 xPF超快打印设备 。


全彩PartPro350 xBC


PartPro350 xBC全彩3D打印机基于Binder Jetting技术，该技术是复合粉末与粘合剂的逐层粘合。


特点

  • 技术：粘结剂喷射
  • 电源：100-240V，2.5 A
  • 尺寸（毫米）：350 x 222 x 200
  • 重量，公斤：254
  • 软体：PartPro Suite
  • 档案格式：STL，WRL，OBJ，3MF
  • 工作腔室：138 x 77 x 130毫米
  • 打印分辨率：1600 x 1600 dpi
  • 专业化：原型制作，原型制作，教育，艺术，广告
  • 层厚度从：100微米
  • 支持的材料：石膏
  • 打印头数量：1
  • 打印速度：18mm / h
  • 兼容的操作系统：Windows 7 /8.1 / 10（64bits）


完整的生产周期，垂直打印速度高达每小时18毫米，可移动的成品托盘，低廉的耗材成本-所有这些使该设备成为具有高生产率的重要工具，将有助于使全彩色3D打印具有成本效益。


在美国，一台3D打印机PartPro350 xBC的价格为3万美元。


XYZPrinting在俄罗斯的官方代表Top 3D Shop计划明年开始销售新产品。

聚酰胺MfgPro230 xS



特点

  • 技术：SLS
  • 电源：220B，20A，2.5 kW
  • 尺寸，毫米：1460x1890x740
  • 重量，公斤：310
  • 软体：SLS Build / SLS Ware
  • 档案格式：STL，3MF
  • 激光：CO2，30瓦
  • 工作腔室：230x230x230 mm
  • 氮气系统
  • 打印分辨率：1600 x 1600 dpi
  • 专业：航空航天，汽车
  • 层厚度：100微米（0.1毫米）
  • 支持的材料：PA（包括柔性，复合材料，涂漆和高温），TPU，PMMA
  • 打印速度：1升/小时
  • 支持的操作系统：Win7 / Win8 / Win10


XYZprinting MfgPro230 xS SLS打印机已经是单色SLS打印机，用于使用聚酰胺进行打印，模型建立速度达到每小时1升。 开发人员承诺与EOS和3D Systems的3D设备的功能相当的高重复性和质量。


该设备价格低至6万美元，显示出出色的印刷效果。


超快系列：MfgPro xPF
MfgPro 180 xPF和MfgPro 220 xPF




特点

  • 材质：光敏聚合物
  • 打印速度：最高10毫米/分钟
  • XY分辨率：70μm
  • 分辨率Z：（层厚度）：25-50-100-200微米
  • 接口：USB（电缆或闪存驱动器），以太网
  • 建筑面积-

MfgPro 180 xPF ：180x140x200毫米
MfgPro 220 xPF ：220x120x380毫米

MfgPro xPF系列基于最新的LSPc打印技术，该技术与Carbon 3D的著名CLIP技术类似。 加利福尼亚Nexa3D的壁内开发了润滑剂亚层光固化。 目前，两家公司已经联手营销基于它的技术和设备。


这些超快打印机的广告中说：“快40倍，准确2倍，便宜5倍。” 这种设备的打印速度达到每分钟1厘米。


3D打印机MfgPro180和MfgPro 220 xPF将在需要高速制造定制零件的应用中找到应用，这些定制零件的小批量生产无法使用传统方法获利，这是一个广泛的行业-从修复工作和珠宝首饰到汽车工业和航空航天研究。 这些设备的起价为2万美元。


除3D设备外，XYZprinting还以AI N.ERA Robotics品牌开发自己的机器人产品线。

AI N.ERA机器人


在芝加哥的一个展览会上，该公司介绍了两种带有集成导航系统的移动机器人模型。

C01



特点

  • 语音接口系统
  • 人脸识别和面部表情
  • 15英寸触摸屏平板电脑
  • 读卡器和扫描仪：NFC，智能（带芯片），MSR（带磁条），OCR（字符识别），条形码扫描仪
  • 内置热敏打印机（如收银机和帮助终端中的打印机）
  • 15英寸后面板信息


C01辅助机器人可用于教育-作为视觉辅助，在展览会，商店，饭店和电影院，酒店，医院和银行中-作为电视节目主持人，顾问，指南甚至销售员。


内置的界面和软件使他不仅可以看到周围的空间并进行导航，还可以识别人的脸，响应请求并进行交互-机器人具有集成的NFC设备，智能卡读取器和磁条卡读取器，打印机，条形码扫描仪，3D -摄像头，当然还有一个15英寸的平板电脑作为终端，位于正面，背面还有一个信息屏幕。


C01与云服务同步并独立分析收集的数据。



此外，该机器人还可以在夜间巡逻无人居住的建筑物，并向陌生人发出信号。


同样令人感兴趣的是M01模型-多功能机器人运输车。

M01



特点

  • 有效负载：最多100公斤（可选）
  • 导航：LiDAR，超声波测距仪（7个），红外传感器（2个）
  • 从“云”自动安装更新
  • 电池容量：34.8 Ah
  • 工作时间：8〜10小时
  • 充电时间：4〜6小时
  • 电压和电池类型：25.9 V，7S12P锂离子电池
  • 最高 速度：2 m / s
  • 驱动：2个驱动轮
  • 转弯半径：0
  • 温度工作条件：0〜+ 45°
  • 尺寸：宽520 x长580 x高355毫米
  • 重量：40公斤
  • 包装内容：充电座
  • 可选：手动操纵杆


M01能够搬运高达100公斤的负载。 它配备了激光雷达（雷达的激光模拟），超声波测距仪和红外传感器，使他能够在10,000平方米的区域上绘制定向地图，并在其通常的磁性标记和无线电信标之外行驶长达一个小时。


将类似的设备与机械臂结合使用，可以为3D打印机，CNC机器提供服务，从工作站组装成品，在企业内部运输货物和文件而无需人工干预。

适用范围：工业企业，旅馆，饭店，医院以及需要沿永久性路线运输货物的任何地方。
