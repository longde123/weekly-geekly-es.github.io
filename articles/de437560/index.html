Durchscheinender Hintergrund

Ich möchte zeigen, wie man schnell einen durchscheinenden Hintergrund erstellt. Wenn Sie den Befehl Deckkraft in CSS verwenden, wird der Hintergrund sicherlich transparent, aber damit auch alles andere, was darauf steht.

Wie kann ich dieses Problem beheben? Ich stöberte im Internet und fand einen wunderbaren Weg, von dem ich Ihnen jetzt erzählen werde.

Angenommen, dies ist Ihre Website (HTML): 

<html> <head> <meta charset="UTF-8"> <link rel="stylesheet" href="style.css"> <title>   </title> </head> <body> <div class="mid"> <p> </p>  .  . </div> </body> </html>

Zeile "" enthält ein CSS-Stylesheet.

Hier ist es (es heißt style.css): 

 body{ background: "   " ; * background-attachment: fixed ; background-position: top center; } .mid{ width: 1250px;** margin: 0 auto; background: rgba(161,21,207, 0.5);*** }

In der mittleren Klasse 3 legt die Linie die Farbe der durchscheinenden Farbe im RGBA-Format fest. Seine 4 Ziffern, in diesem Fall 0,5, sind der Transparenzwert von 0 bis 1.

So einfach. Probieren Sie es aus, Sie werden es sicherlich schaffen!

* - URL-Bilder hier einfügen
** - Stellen Sie die Seitenbreite ein (dafür haben wir die Klasse mid im HTML-Code erstellt)
*** - rgba ist die Farbe im RGB-Format. Der letzte Buchstabe (a) gibt die Transparenz des Hintergrunds an. Sein Wert ist eine Ziffer von 0 bis 1. 0 ist ein vollständig transparenter Hintergrund, 1 ist das Gegenteil.
