6年后，传奇的崩溃发行版Hiren's BootCD的新版本



今天是与铁工，修理工，系统管理员和enikeyshchikov一起度过的假期！
长期遗忘之后， 恢复了传说中的Hiren的BootCD启动盘的发行 。

Hiren的BootCD （也称为HBCD ）是一种可引导映像，带有可写入CD \ DVD \ USB闪存的软件，其中包含许多诊断实用程序，例如系统性能测试，格式化和布局程序，克隆和映像创建工具，数据恢复工具，使用MBR的工具，用于刷新和更新BIOS的工具以及用于解决各种计算机问题的其他工具。

这次，该分发基于64位Windows PE 10.0，并且仅包含合法的免费分发程序，但前提是您仅将HBCD用于诊断和消除计算机问题，而不将其用作计算机的主要软件。

新的正式版本的发布是由分发工具包的一群特别敬业的爱好者执行的 ，他们可以访问官方网站（网站的旧版本，直到发布新版本），而原作者早已退休。

历史背景


Hiren的BootCD灾难恢复分发工具包的历史始于2004年。 最初，Windows 98 Second Edition和MS-DOS 7.1被用作操作系统。 后来，在9.9版中，引入了Mini Windows XP SP3作为操作系统质量。

Hiren的BootCD长期以来一直被warez分类，因为它实际上侵犯了单个计算机程序发行者的版权。 尽管分发工具包有95％被废弃或可以免费使用，但它还包含Microsoft和其他公司的程序，需要特殊许可。 因此，六溴环十二烷使用者必须考虑到其使用可能会违反知识产权法（由于其构成中有大量专有许可程序）。 在某些国家/地区，分发Hiren's BootCD被视为非法。

Hiren的BootCD中包含许多受版权保护并需要特殊许可的商业软件产品，直到11.0版为止。

例如，在版本10.6中，发行说明中指示了17个商业程序。 首先，这些是Acronis True Image，Norton Ghost，Acronis Disk Director和Paragon分区管理器（Partition Magic Pro在版本10.5中已被替换）。 为了能够运行商业应用程序，用户必须发誓自己是许可证的所有者，但实际上并未进行任何检查。

在11.0版的商业产品中，只有三个：

  • 迷你Windows 98
  • 迷你Windows XP
  • MS-DOS实用程序集合

一些粉丝只喜欢旧的发行版形成策略，因此，他们创建了一个替代的Restored Edition分支，其中包括（返回）需要商业许可证的工具，该工具已由原始发行版作者删除。

原始作者发布的最新版本是Hiren的BootCD v15.2 ，该版本于2012年11月9日发布，此后，创作者出于未知原因对他们的创造力失去了兴趣。

一段时间以来，Hiren的BootCD v的一系列非官方发行。 15.3-v.16.x

使用Windows PE 10.0是否合法？


Windows PE不是通用操作系统。 除部署和恢复外，它不能用于任何其他目的。 不应将其用作瘦客户端或嵌入式操作系统。 为此，还应使用其他Microsoft产品，例如Windows Embedded CE。

为防止被用作主要操作系统，Windows PE会自动关闭外壳程序，并在连续使用72小时后重新启动。 此时间段无法通过常规方式更改。
来源

Windows PE不包含Windows许可证密钥。 此外，Hiren的BootCD与Windows PE不冲突，并且保留了相同的72小时连续使用限制。 因此，从Microsoft的许可协议策略的角度来看，在Hiren的BootCD中使用Windows PE是合法的。
