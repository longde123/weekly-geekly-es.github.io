Tidak sepenuhnya upaya yang berhasil untuk mengembalikan masa lalu atau PKC dari pasar loak



Kenalan saya dengan komputer pribadi genggam ( PDA ) dimulai beberapa tahun yang lalu. Suatu ketika, pada 2004-2005, di tangan seorang teman saya melihat perangkat kecil dengan layar warna yang indah yang segera menarik perhatian saya. Kemudian saya sudah mendengar sesuatu tentang PDA atau "handheld", jadi saya memutuskan untuk mempelajari perangkat. Seorang teman memberi saya sedikit kerja dengan perangkat ini, dan saya menyadari - inilah yang saya butuhkan. Maksud saya, bukan gadget yang dimiliki oleh seorang teman, tetapi setiap PDA dengan level yang sama. Sayangnya, pada waktu itu saya tidak mampu membeli model yang relatif murah, jadi saya harus menunggu beberapa tahun.

Satu atau dua tahun kemudian, mereka memberi saya PDA, dan model yang sama seperti teman pernah. Mulai saat ini mulai pemulihan hubungan langsung dengan handheld. Saya kemudian mendapatkan model HP iPAQ H2210, yang dilengkapi dengan serangkaian fungsi dan modul perangkat keras yang baik. Dalam beberapa minggu, saya menguasai hampir semua fungsi ini, belajar cara menghubungkan PDA ke Internet melalui telepon, terhubung ke jaringan lokal di rumah dan melakukan lebih banyak lagi. Salah satu fitur yang paling saya sukai adalah kemampuan untuk menggunakan PDA sebagai remote control.

Untuk melakukan ini, perlu menginstal program khusus (tentangnya, tepat di bawah), pilih dari daftar model perangkat yang ingin Anda gunakan, dan itu saja - remote control layar sentuh sudah siap. Itu sangat nyaman, karena program ini dapat bekerja bahkan dengan gadget yang tidak ada dalam basis data utama. Itu hanya perlu untuk "melatih" dia dengan menyalakan penerimaan sinyal inframerah dan secara berurutan menekan tombol pada remote control perangkat yang perlu dikendalikan. Semuanya bekerja dengan sempurna.

Setelah beberapa saat saya menjual PDA itu dengan membeli smartphone yang bahkan lebih fungsional.

Bertahun-tahun telah berlalu, banyak perangkat berbeda dengan kendali jarak jauh IR terakumulasi di rumah. Dan entah bagaimana saya ingat PDA saya yang setia, mampu melakukan fungsi remote control yang cerdas. Ada ide untuk menemukan PDA yang sama, dan membelinya, mengubahnya menjadi remote control.

Di Internet, saya memutuskan untuk tidak memesan perangkat, karena pasar loak bekerja di sebelah saya pada hari Sabtu, di mana sejumlah besar pecinta teknologi menjual semua jenis perangkat seharga satu sen. Entah bagaimana saya sudah melihat PDA di tempat ini, jadi saya memutuskan untuk pergi melalui baris untuk tujuan tertentu - untuk menemukan PDA dengan port inframerah, dalam fungsionalitas kira-kira sama dengan handheld yang saya gunakan sebelumnya.

Saya beruntung - segera saya menemukan persis model yang saya cari. Saya bahkan tidak percaya pada keberuntungan saya segera. Tentu saja, keadaan gadget meninggalkan banyak yang harus diinginkan - misalnya, bantalan karet di samping tampak sangat tua. Tapi saya tidak harus memilih, yang utama adalah perangkat itu bisa dikerjakan. Masalah lain muncul di sini - tidak mungkin untuk memverifikasi pengoperasian perangkat di pasar loak.

Yang positif adalah bahwa perangkat itu dijual seharga satu sen - saya membelinya seharga 4 euro (saya tinggal di Spanyol). Oleh karena itu, diputuskan untuk membeli beberapa genggam lagi sehingga semuanya dijamin bekerja. PDA lain menarik perhatian saya, dan jauh lebih tua dari model sebelumnya. Ini tentang Compaq iPAQ H3100. Masalahnya adalah dinding perangkat sedikit bengkak (tidak ada baterai yang dapat dilepas di sini). Benar, mereka menjualnya lebih murah, jadi saya memutuskan untuk mengambilnya. Sayangnya, tidak ada charger atau stasiun docking, untuk PDA saya ditawari begitu banyak kabel dengan pengisian untuk mobil. Konektor pas, jadi saya mengambil apa yang ditawarkan.



Beberapa saat kemudian saya menemukan PDA lain, sudah lebih modern. Ini adalah model HP Pocket PC iPAQ h5455. Perangkat itu terlihat lebih baik dari yang sebelumnya. Benar, baterainya bengkak, tetapi saya memutuskan untuk mengambil risiko dalam kasus ini. Ke depan, saya akan mengatakan bahwa PKC khusus ini ternyata tidak beroperasi. Saya tidak beruntung dengan dia, meskipun saya mengeluarkan biaya pembelian kecil - hanya 3 euro.
