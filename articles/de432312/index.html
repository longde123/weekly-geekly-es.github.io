Erstellen Sie eine Formkarte der RF-Karte in Power BI

Hallo allerseits! Heute werden wir über "räumliche" Analysen in Power BI sprechen. Informationen zur Plattform für Business Intelligence finden Sie hier . In einem Beitrag werde ich einen Aspekt analysieren - wie man eine Karte der Russischen Föderation in der Formkarte zeichnet.


In Power BI kann die räumliche Analyse mithilfe der folgenden benutzerdefinierten visuellen Elemente implementiert werden:


  • Karte
  • Abgeschlossene Karte
  • ArcGIS Maps für Power BI
  • Formkarte

Jedes hat seine eigenen Vor- und Nachteile, daher ist es besser, sie anhand einer bestimmten Aufgabe zu betrachten .


Der Artikel berücksichtigt das visuelle Element Shape Map . Warum haben wir ArcGIS Maps für Power BI nicht verwendet ? Weil der Administrator seine Verwendung blockiert hat!



Die Verwendung der Formkarte war nicht so einfach. Die Probleme hängen mit der Tatsache zusammen, dass die Standard-Desktop-Version von Power BI keine integrierten Karten der Russischen Föderation enthält.



Die Figur. Benutzerdefinierte Karte


Optionen zur Lösung des Problems:


  1. Zeichnen Sie selbst online eine Karte. Dafür ist schon alles da . Der Code der generierten Karte wird rechts angezeigt. Prozess hinzufügen Eigene Daten können automatisiert werden, indem ein vorbereiteter Code mit den Koordinaten der Objekte eingefügt wird.
  2. Zeichnen Sie eine Karte der Russischen Föderation gemäß den Anweisungen . Die Implementierung erfordert Eingabedateien = Shapefile . Laden Sie Shapefile RF unter dem Link herunter.
  3. Verwenden Sie den vorgefertigten Bericht "Karte der Russischen Föderation (Regionen)". Sie können die Berichtsdateien über den Link herunterladen (Power BI-Bericht - "Map.pbix", Originaltabelle zum Markieren - "file.txt", RF Map im * .json-Format - "RF.json map"). Wenn Sie beispielsweise Regionen zu Gruppen zusammenfassen müssen, muss die Karte nicht neu gezeichnet werden. Es reicht aus, die Spalte "desc" in der Datei "file.txt" so zu ändern, dass die Regionen derselben Gruppe denselben Wert (Gruppenname) haben.
