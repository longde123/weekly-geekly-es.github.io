Pembaruan Pembuat Model ML.NET

ML.NET adalah cross-platform, belajar mesin kerangka kerja untuk NET pengembang, dan Model Builder adalah UI perkakas di Visual Studio menggunakan klausul Automated Bahwa Machine Learning (AutoML) Mudah memungkinkan Anda untuk melatih mengkonsumsi dan model kustom The ML.NET. Dengan ML.NET dan Model Builder, Anda dapat membuat model pembelajaran mesin kustom untuk skenario seperti analisis sentimen, prediksi harga, dan lainnya tanpa pengalaman pembelajaran mesin!

Pembuat Model ML.NET


Rilis Model Builder ini dilengkapi dengan perbaikan bug dan dua fitur baru yang menarik:

  • klasifikasi citra skenario - lokal model klasifikasi citra kereta dengan gambar Anda sendiri
  • Coba model Anda - buat prediksi pada data input sampel langsung di UI



Skenario Klasifikasi Gambar


Kami memamerkan fitur ini dalam .NET Conf untuk mengklasifikasikan cuaca dalam gambar sebagai cuaca yang cerah, berawan, atau hujan, dan sekarang Anda dapat melatih model klasifikasi gambar secara lokal di Model Builder dengan gambar Anda sendiri!



Misalnya, Anda memiliki dataset gambar anjing dan kucing, dan Anda ingin menggunakan gambar-gambar itu untuk melatih model ML.NET yang mengklasifikasikan gambar baru sebagai "anjing" atau "kucing."

Dataset Anda harus berisi folder induk dengan subfolder berlabel untuk setiap kategori (misalnya folder yang disebut Hewan yang berisi dua sub-folder: satu bernama Dog, yang berisi gambar pelatihan anjing, dan satu bernama Cat, yang berisi gambar pelatihan kucing):



Anda dapat menggunakan kode Berikutnya Langkah dan proyek-proyek yang dihasilkan oleh Model Builder untuk dengan mudah mengkonsumsi model klasifikasi yang dilatih gambar dalam aplikasi pengguna akhir Anda, seperti dengan skenario teks.

Coba model Anda


Setelah melatih model dalam Model Builder, Anda dapat menggunakan model untuk membuat prediksi pada input sampel langsung di UI untuk skenario teks dan gambar.

Misalnya, untuk anjing vs kucing klasifikasi citra contoh, Anda bisa masukan gambar dan lihat hasilnya dalam Mengevaluasi langkah Model Builder:

Berikan Tanggapan Anda


Jika Anda mengalami masalah, merasa ada sesuatu yang hilang, atau sangat menyukai sesuatu tentang ML.NET Model Builder, beri tahu kami dengan membuat masalah dalam repo GitHub kami.

Model Builder masih dalam Pratinjau, dan umpan balik Anda sangat penting dalam mengarahkan arah yang kami ambil dengan alat ini!
