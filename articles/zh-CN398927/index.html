展示架


ZenFone 3 Laser配备5.5英寸显示屏，分辨率为1920x1080。有了这样的对角线，就不需要更多了-图像看起来完全平滑，没有像素级。色彩渲染质量非常不错，尽管不如ZenFone 3系列中较昂贵的型号便宜，但是，只有将智能手机放在旁边，才能注意到这一点，但是在正常使用情况下，Laser Display没什么可怪的。顺便说一句，他的亮度也不错-500 cd / m2。

音效


作为“扬声器系统”，ZenFone 3 Laser使用五磁铁扬声器。SonicMaster 3.0专有软件和硬件组合负责音质，而NXP Smart Amplifier负责音量。该放大器可能与ZenFone 3 Ultra具有相同的型号。总之，它听起来真的很大声，也很不错。

摄影机


“激光”一词强调了主要功能之一-即时聚焦系统。它的运行速度非常快，我从来没有遇到过这样的情况，因此在单击屏幕上的一个对象后，会有一个痛苦的停顿，之后智能手机终于徘徊了。

该相机使用13百万像素Sony IMX214 CMOS矩阵。光圈-f / 2.0。没有光学稳定性，只有电子。这意味着系统分析图像沿矩阵的运动并尝试在一定程度上进行补偿。稳定对照片和视频拍摄均有效。智能手机使用位于闪光灯附近的RGB传感器来测量光线的色温，并在拍摄时进行校正，从而改善了色彩还原。

与旧型号一样，ZenFone 3具有动态HDR模式：您无需在先拍摄再观看即可在屏幕上立即看到HDR帧的外观。事实是从矩阵获取的图像不是由智能手机的中央处理器处理，而是由摄像头模块的内置处理器处理。由于处理器性能的提高，动态HDR模式的引入成为可能。此外，“超分辨率”模式对某人很有用：相机拍摄一张照片，然后对其进行推断，并将最终帧的分辨率提高4倍，最高可达52兆像素。

对于使用ZenFone 3 Laser自拍照的爱好者来说，出现了一个有用的功能：如果从上到下滑动指纹扫描仪，则会激活前置摄像头。要拍摄照片，您无需改变握力并将手指拖到屏幕上，只需单击同一台扫描仪一次即可。

绩效和工作经验


ZenFone 3 Laser基于Qualcomm Snapdragon 430 64位四核处理器。 RAM-2或4 GB。对于数据存储，根据型号，可用32或64 GB。如果事实证明这还不够，那么您可以将SIM卡插槽之一与存储卡一起使用。在这种情况下，您只能插入一张SIM卡：智能手机不允许同时使用两张SIM卡和一张存储卡，只能同时使用两张SIM卡或一张SIM卡和microSD。



ZenFone 3 Laser的性能足以应付几乎所有任务，我没有遇到刹车。只有一些大的，超载的JavaScript页面正在滚动并大量滚动。

再次返回指纹扫描仪：从主观上讲，其位置比与主页按钮组合起来更方便解锁。当您将智能手机拿在手中时，与用拇指按下正面的按钮时相比，食指的移动更为自然。此外，扫描仪可用于接听来电，因此请不要再次用拇指触摸显示器。您还可以将最多五个常用的应用程序分配给不同的手指。尽管此功能在我看来颇具争议：请尝试用小手指或无名指舒适地按下扫描仪。而且没有理由考虑换手选择；更容易在屏幕上显示快捷方式。

结论


ZenFone 3 Laser属于ZenFone产品线的中间价格范围。它没有最高的相机分辨率，处理器更简单，而且收尾也不那么丰富。但是同时，性能足以应付日常任务，出色的显示效果，并且电池也不会比价格更昂贵的ZenFone 3差。该相机配备了非常快的对焦系统和RGB传感器，支持与ZenFone系列中更复杂的型号相同的拍摄模式。
