工具


  • mougrim / php-xdebug-proxy- PHP Xdebug代理：当Xdebug的标准功能不够用时 。
  • php-defer / php-defer - Go的defer函数的类似物。 当执行线程离开当前（外部）函数/方法的范围时，作为此函数的参数传递的回调将起作用。 析构函数仅在引擎盖下使用。 这个想法很老，几年前已经作为phplang / scope-exit的一部分实现了。
  • sensiolabs-de /deptra0.6-静态代码分析工具，用于确定应用程序层之间的依赖关系。
  • graze / parallel-process-让您轻松地并行运行多个symfony / process 。
