CTT dalam solusi server - seperti apa bentuknya?


Seperti ini Ini adalah bagian dari penggemar yang ternyata berlebihan dan dihapus dari dua puluh server di rak uji yang terletak di pusat data DataPro. Di bawah cut - traffic. Deskripsi ilustrasi sistem pendingin kami. Dan tawaran yang tak terduga untuk pemilik peralatan server yang sangat ekonomis, tetapi tidak kenal takut.


Sistem pendingin peralatan server berdasarkan pipa panas kontur dianggap sebagai alternatif dari sistem cair. Sebanding dengan efisiensi, lebih murah untuk diimplementasikan dan dioperasikan. Pada saat yang sama, bahkan secara teori itu tidak memungkinkan kebocoran cairan di dalam peralatan server yang mahal.


Tahun lalu, rak eksperimental pertama kami dipasang di pusat data DataPro. Terdiri dari empat puluh server Supermicro yang identik. Dua puluh pertama dari mereka dengan sistem pendingin standar, yang kedua puluh - dengan yang dimodifikasi. Tujuan dari percobaan ini adalah untuk menguji penerapan sistem pendingin kami di pusat data nyata, di rak nyata, di server nyata.

Di bagian atas rak adalah server yang biasa. Di bagian bawah ada bus pertukaran panas dengan perangkat penjepit untuk server tanpa kipas (hampir) yang tidak biasa. Kipas hanya dibiarkan untuk mendinginkan memori. Panas ditransfer dari prosesor ke penukar panas menggunakan pipa panas loop kami. Dan dari penukar panas, panas melalui bus cair pergi ke tempat lain.
