Petunjuk untuk menghindari pemblokiran Telegram. Cara mengaktifkan SOCKS 5

Saluran telegram tnews_ru menerbitkan instruksi tentang cara mem-bypass kemungkinan pemblokiran Telegram, messenger populer yang dikembangkan oleh tim Pavel Durov. Artikel ini menjelaskan pemecahan masalah berdasarkan penggunaan protokol jaringan SOCKS5. Protokol ini secara mulus meneruskan paket data dari klien ke server melalui server proxy.

SOCKS adalah protokol jaringan yang memungkinkan aplikasi klien-server untuk secara transparan (tidak terlihat oleh mereka) menggunakan layanan di balik firewall (firewall). Opsi ini sudah tersedia di Telegram Desktop dan Mac, dan klien seluler akan menerimanya dalam pembaruan berikutnya.

Tip: aktifkan opsi pembaruan otomatis dalam pengaturan.

Bagaimana cara mempertahankan akses ke Telegram setelah messenger diblokir di Rusia?

Nyalakan SOCKS5 terlebih dahulu sebelum memulai kunci.

Jika karena alasan tertentu Anda tidak punya waktu untuk melakukan ini, maka Anda harus kencang, karena dari Google Play dan AppStore Rusia, pelanggan Telegram cenderung menghilang. Vpn dan Tor akan membantu di sini, di mana Anda dapat mengunduh Telegram di toko-toko Amerika. Tapi jangan menyanjung diri sendiri - di depan hukum adalah larangan cara melewati kunci, diadopsi dalam bacaan pertama di Duma Negara Federasi Rusia.

Kiat: dalam pengaturan Android, izinkan unduhan pembaruan bukan dari toko resmi, dan unduh pembaruan Telegram dari situs messenger telegram.org .

Konfigurasikan SOCKS 5


Kami memberikan instruksi dalam bahasa Rusia:

1) Buka pengaturan desktop Telegram dan klik "Secara default (sekarang: TCP)."

2) Beralih di menu "Jenis Koneksi" ke "TCP socks5 proxy"

3) Masukkan alamat server proxy (misalnya, 192.254.68.37), port yang sesuai (1080). Lebih baik menggunakan alamat proxy Amerika, Jerman, Swedia atau Inggris. Kemudian periksa "Gunakan IPv6" dan simpan.

Bagaimana cara menemukan server proxy?


Cukup ikuti tautan ini:

→ proksi AS
→ Inggris proksi
→ Jerman proksi
→ Swedia proksi

Ambil saja server untuk SOCKS5, jangan campur dengan HTTP.

Anda dapat mengonfigurasi dari daftar gratis ini (socks - sockslist.net ).

Sekarang, secara default, Telegram Anda akan terhubung melalui server proxy yang dipilih.

Tambahan: Jika Anda mengalami masalah dengan koneksi, Anda perlu mengkonfigurasi firewall Windows - tambahkan Telegram ke aplikasi yang diizinkan. Ini mudah dilakukan, tetapi Anda harus memiliki hak administrator.

  • Pergi ke "Control Panel" dan pergi ke "Windows Firewall".
  • Klik pada item "Izin untuk berinteraksi dengan aplikasi atau komponen ...". Sebuah jendela dengan sejumlah besar program akan terbuka, tetapi sepertinya Anda tidak akan menemukan Telegram di sana. Karena itu, Anda perlu menambahkannya di sana.
  • Klik pada "Ubah Pengaturan" dan kemudian pada tombol yang lebih rendah "Izinkan aplikasi lain."
  • Di jendela pop-up "Tambah aplikasi" melalui Ikhtisar tambahkan Telegram.
  • Itu saja. Konfigurasikan koneksi ke server proxy lagi.
