11 de julio, seminario web del Grupo IB "Análisis de malware para principiantes: enfoques básicos"



El trabajo de un analista viral requiere un conocimiento profundo y una amplia experiencia, pero existen métodos básicos de análisis que están disponibles incluso para expertos novatos. A pesar de su simplicidad, dichos métodos funcionan de manera eficiente y le permiten identificar una porción significativa de software malicioso.

Los especialistas de CERT-GIB prepararon un seminario web sobre el tema "Análisis de malware para principiantes: enfoques básicos" . El seminario web se llevará a cabo el 11 de julio de 2019 a las 11:00 (hora de Moscú) , y será organizado por Ilya Pomerantsev, especialista en análisis de malware del Grupo IB.

¿Qué será interesante en el seminario web?


En el seminario web aprenderá:

  • Qué enfoques básicos utilizan los analistas de virus para determinar la nocividad de un archivo;
  • Cómo realizar un análisis estático de un archivo malicioso;
  • Cómo preparar un entorno virtual para el análisis dinámico de malware y llevarlo a cabo;
  • ¿Cuáles son los pros y los contras de ambos enfoques y cómo elegir el mejor?
