Les mathématiques et le jeu "Set"


Quiconque trouve un «set» ici recevra une barre de chocolat de ma part.

Set est un jeu génial dans lequel nous avons coupé il y a environ 5 ans. Crier, crier, photographier des combinaisons.

Les règles du jeu disent qu'il a été inventé en 1991 par le généticien Marsha Falco, prenant des notes lors d'une étude sur l'épilepsie chez des bergers allemands en 1974. Pour ceux dont le cerveau est suffisamment épuisé par les mathématiques, après un certain temps, on soupçonne qu'il y a ici des échos avec la planimétrie et le tracé de lignes droites à travers les points. (Pour deux cartes données, il y a une et une seule carte qui va avec elles dans le même ensemble.)


Marsha Falco demande: "Eh bien, vous n'avez pas trouvé le" set "?"

N'oubliez pas les règles



Seth est un jeu de cartes. Toutes les cartes ont quatre paramètres, chacun prenant trois valeurs (3 x 3 × 3 × 3 = 81 cartes au total).



Les types et valeurs des paramètres sont les suivants:

  • figure :: = ellipse | losange | "Snot"
  • couleur :: = rouge | vert | pourpre
  • remplir :: = blanc | rayé | solide
  • nombre :: = 1 | 2 | 3

Le but du jeu est de trouver des combinaisons spéciales de trois cartes. Trois cartes sont appelées «ensemble» si, pour chacun des quatre attributs de la carte, toutes sont identiques ou toutes sont différentes.



En d'autres termes, nous pouvons dire que trois cartes ne constitueront pas un ensemble si deux cartes ont une valeur de paramètre et que la troisième est différente. Vous pouvez voir que pour deux cartes, il y aura toujours une troisième (et une seule) avec laquelle elles seront un ensemble.

Progression du jeu: le leader met sur la table 12 cartes. Quand quelqu'un trouve un ensemble, il crie "Ensemble!" puis ramasse calmement les cartes constituantes. S'il n'y a pas de jeu dans les cartes présentées (très probablement, il semble juste qu'il n'y en ait pas), l'hôte sort trois autres cartes.

Le nombre maximum de cartes sans set est de 20. Le tour continue jusqu'à ce que le jeu soit épuisé. Celui qui a collecté le plus de sets gagne.

Les mathématiciens s'affairèrent et présentèrent une combinaison de 20 cartes. Qui se considère Chuck Norris, peut oublier cette photo et essayer de collectionner soi-même le «solitaire».
