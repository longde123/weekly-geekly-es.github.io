Red Hat Enterprise Linux 8 Beta - Bersamaan Dengan Dunia

Buka Red Hat Enterprise Linux 8 Beta - pada saat yang sama dengan seluruh dunia!



Dan sebelum kita sampai pada titik memperkenalkan perubahan penting dan fitur baru, katakanlah hal utama - pengembang dapat mulai menguji keajaiban baru kami menggunakan Program Pengembang Red Hat .

Wadah Linux, Kubernetes, kecerdasan buatan, blockchain dan sejumlah terobosan teknologi profil tinggi lainnya dalam beberapa tahun terakhir memiliki satu kesamaan - Linux. Linux yang telah menjadi pekerja keras untuk sistem produksi mission-critical selama hampir dua dekade. Hari ini, kami menyajikan visi kami tentang Linux sebagai dasar untuk inovasi yang memperluas batasan dan mengubah sistem TI perusahaan ke masa depan. Temui Red Hat Enterprise Linux 8 Beta.

Ketika kami menciptakan Red Hat Enterprise Linux 8 Beta, kami bekerja keras untuk menjadikannya basis universal untuk lingkungan komputasi tradisional dan infrastruktur TI inovatif. Generasi baru dari platform Linux perusahaan kami dirancang untuk membantu menerapkan strategi transformasi digital di lingkungan cloud hybrid yang memungkinkan organisasi memanfaatkan wadah Linux, Kubernetes, dan inovasi teknis lainnya untuk menghadirkan produk dan layanan unik ke pasar. Pada saat yang sama, Red Hat Enterprise Linux 8 Beta akan membantu profesional TI mengoptimalkan dan meningkatkan efisiensi sistem TI yang ada, serta memastikan stabilitas dan produktivitas dalam implementasi inovasi.

Dalam empat tahun sejak Red Hat Enterprise Linux 7 kami keluar, dunia teknologi IT telah banyak berubah - kami mendukungnya. Hari ini, Red Hat Enterprise Linux 8 Beta sekali lagi menetapkan standar sistem operasi baru sebagai dasar untuk inovasi TI. Red Hat Enterprise Linux 8 Beta telah menerima ratusan perbaikan dan lusinan fitur baru, termasuk yang memungkinkan untuk menggunakan platform ini secara efektif untuk transformasi digital dan transisi ke teknologi cloud computing hybrid, tanpa mengorbankan stabilitas sistem TI yang ada.

Ups! Red Hat Enterprise Linux 8 Beta memperkenalkan konsep aliran aplikasi Application Stream , yang menyediakan pengiriman paket userspace yang lebih mudah dan lebih fleksibel. Komponen userspace sekarang dapat diperbarui lebih sering daripada paket OS utama, yaitu ini dapat dilakukan tanpa menunggu rilis versi baru dari sistem operasi. Dimungkinkan juga untuk menginstal versi berbeda dari paket yang sama, misalnya, bahasa pemrograman atau DBMS, yang meningkatkan fleksibilitas pengaturan dan penyesuaian Red Hat Enterprise Linux dengan tetap menjaga stabilitas dan keandalan platform secara keseluruhan.

Lebih jauh lagi, selain arsitektur kernel yang diperbarui, peningkatan dan inovasi besar lainnya telah muncul di Red Hat Enterprise Linux 8 Beta.

Jaringan


Red Hat Enterprise Linux 8 Beta mendukung IPVLAN dan memungkinkan kontainer pada mesin virtual untuk berkomunikasi dengan host jaringan dengan overhead minimal dalam hal bandwidth dan latensi jaringan. Tumpukan TCP / IP baru dengan pelacakan kemacetan Bandwidth dan Round-trip propagation time (BBR) meningkatkan kinerja, meminimalkan latensi, dan mengurangi hilangnya paket untuk streaming video, penyimpanan cloud, dan layanan Internet serupa lainnya.

Keamanan


Melanjutkan tradisi versi sebelumnya, Red Hat Enterprise Linux 8 Beta menawarkan kode yang disempurnakan, perbaikan keamanan rutin, dan pengalamannya yang luas dalam menciptakan perangkat lunak yang aman. Salah satu tujuan utama kami dengan merilis Red Hat Enterprise Linux 8 adalah untuk meningkatkan keamanan lingkungan cloud hybrid dengan menyediakan sistem operasi yang aman secara default.

Dengan dukungan untuk OpenSSL 1.1.1 dan TLS 1.3 , aplikasi server berbasis Red Hat Enterprise Linux 8 Beta sekarang dapat menggunakan standar terbaru untuk perlindungan data. Kebijakan kriptografi sistem-lebar Kebijakan Sistem kriptografi Sistem menyederhanakan kepatuhan dengan peraturan dan persyaratan perlindungan data, memungkinkan mereka untuk diterapkan secara terpusat, dan tidak dengan menyesuaikan atau menyelesaikan aplikasi individual.

Kontainer Linux


Kami memiliki dukungan bawaan untuk wadah Linux di Red Hat Enterprise Linux 7, dan sekarang wadah adalah salah satu komponen terpenting dari transformasi digital yang terkenal, membuka jalan bagi semua aplikasi perusahaan yang lebih portabel dan fleksibel ini.

Red Hat Enterprise Linux 8 Beta mendukung penuh dan termasuk perangkat manajemen wadah ringan dan sumber terbuka Red Hat, yang mencakup alat-alat seperti Buildah (pembuatan wadah), Podman (peluncuran wadah) dan Skopeo (pencarian dan kolaborasi) penggunaan wadah). Karena arsitekturnya yang terdistribusi dan penghapusan sifat tanpa daemon, alat ini membantu pengembang mencari, meluncurkan, membangun dan bertukar wadah lebih cepat dan lebih efisien.

Manajemen sistem


Dengan semakin meningkatnya peran Linux dalam pusat data perusahaan, ada peningkatan kebutuhan akan alat manajemen yang lebih sederhana dan lebih terjangkau untuk membantu administrator sistem pemula bekerja secara efektif dengan lingkungan hibrid dan melakukan tugas-tugas tidak biasa lainnya. Red Hat Enterprise Linux 8 Beta dirancang untuk menyederhanakan kehidupan seorang administrator keterampilan apa pun, terutama karena konsol manajemen terpadu Red Hat Enterprise Linux Web Console, yang antarmuka yang sederhana dan intuitif memungkinkan Anda untuk mengelola server Red Hat Enterprise Linux secara efektif baik secara lokal maupun jarak jauh , termasuk mesin virtual.

Alat Komposer akan membantu pengguna baru dan berpengalaman Red Hat Enterprise Linux membuat dan menggunakan gambar OS di cloud hybrid, termasuk lingkungan fisik dan virtual, cloud pribadi dan publik. Antarmuka grafis Komposer yang mudah digunakan menyederhanakan akses paket dan perakitan gambar yang dapat digunakan, membantu Anda dengan cepat dan mudah mengkonfigurasi Red Hat Enterprise Linux untuk memenuhi kebutuhan spesifik Anda, apakah meminimalkan ukuran gambar atau membuat konfigurasi khusus, serta lingkungan penyebaran target, termasuk mesin virtual dan cloud Rabu

Yum 4 , versi baru dari manajer paket Red Hat Enterprise Linux in-house, menawarkan lebih banyak kinerja, lebih sedikit ketergantungan selama instalasi, dan lebih banyak pilihan ketika bekerja dengan versi paket.

Sistem File dan Pergudangan Data


Kebaruan lain dari Red Hat Enterprise Linux 8 Beta adalah Stratis , sistem file yang berorientasi volume. Stratis memungkinkan administrator untuk melepaskan diri dari rincian teknis implementasi penyimpanan melalui penggunaan API dan dengan cepat membuat sistem file yang mudah dikelola dan efisien.

Snapshots Sistem File snapshot secara signifikan mempercepat tugas tingkat file, seperti mengkloning mesin virtual, dan juga membantu menghemat ruang disk dengan menulis ke penyimpanan hanya ketika data berubah. Dukungan LUKSv2 untuk mengenkripsi data disk dengan Enkripsi Disk Bound Jaringan (NBDE) meningkatkan keamanan dan menyederhanakan akses ke data yang dienkripsi.

Masa depan siap untuk pengujian.


Pada artikel ini, kami tidak hanya ingin berbicara tentang mengapa Red Hat Enterprise Linux 8 Beta adalah dasar dari sistem TI perusahaan di masa depan. Kami benar-benar ingin Anda bertemu dengannya secara langsung. Kami mengundang semua pelanggan kami untuk mulai menguji Red Hat Enterprise Linux 8 Beta dan untuk mengevaluasi kemampuannya dalam hal meningkatkan fleksibilitas, keandalan, dan pengelolaan saat meningkatkan aplikasi. Adapun pengembang perangkat lunak, mereka juga dapat melihat masa depan platform Linux kelas enterprise terkemuka menggunakan Program Pengembang Red Hat .

Nah, jika Anda tidak terbiasa dengan Red Hat Enterprise Linux, buka di sini Red Hat Enterprise Linux 8 Public Beta , lihat file README untuk instruksi dan memulai.
