Mentransfer kontak dari Windows Phone ke iOS

Hai Geektimes!
Saya ingin berbagi hack kehidupan, yang menyederhanakan impor kontak dari WP ke iOS dari tingkat "tidak mungkin" menjadi "beberapa tapas."



Jadi, mari kita langsung ke pokok permasalahan:

Bagaimana itu bekerja


  1. Pertama, pada iPhone Anda harus masuk ke akun Outlook.com Anda, Anda harus memilih kontak sebagai sinkronisasi;
  2. Selanjutnya, unduh program yang akan membuat cadangan semua kontak Anda di memori telepon;
  3. Setelah itu - lepaskan akun Outlook dari iPhone;
  4. Kontak menjadi kosong. Kami memulihkan kontak dari cadangan menggunakan program (p. 2.) ke memori iPhone lokal;
  5. Untuk sinkronisasi selanjutnya dari semua kontak dengan iCloud - aktifkan pengaturan yang sesuai di iCloud (beri tanda centang di depan item "Kontak");
  6. Keuntungan!

Sekarang kita tinggal lebih detail pada setiap item.

Masuk ke akun Outlook Anda


Pada artikel ini, saya mempertimbangkan untuk mentransfer kontak dari akun Microsoft Live, yang data pribadi di Windows Phone (kontak, surat, catatan ..) disinkronkan secara default, sebenarnya itu adalah analog dari Apple ID di iOS. Jadi untuk mentransfer kontak, sinkronkan terlebih dahulu di Windows Phone Anda dengan akun Microsoft Live Anda.

Di iPhone, buka "Pengaturan", "Mail, alamat, kalender", "Tambah akun", pilih "Outlook.com".
Setelah login berhasil, centang kotak di sebelah "Kontak".



Memilih program untuk membuat cadangan


Saya menggunakan program Simpler . Pengembang ini memiliki banyak program yang hampir sama, jadi pilihlah yang gratis. Kami hanya tertarik pada tab "Salinan Salinan". Jika ya, maka programnya cocok.



Cadangkan


Semuanya sederhana di sini. Dalam program ini, buka tab "Salin salinan" dan klik tombol besar "Cadangkan sekarang". Setelah itu, jika diinginkan, Anda dapat mengunggah cadangan lebih lanjut ke cloud.



Penyelesaian prosedur


Agar tidak ada kontak ganda dan semuanya berfungsi dengan benar (termasuk agar kontak tidak dihapus dari Microsoft Live), Anda perlu melepaskan ikatan akun Outlook.com dari iPhone. Untuk melakukan ini, kita kembali ke "Mail, alamat, kalender", pilih akun Outlook yang baru dibuat dan klik "Hapus akun", dan semua data yang disinkronkan dari iPhone dihapus. Kontak menjadi kosong. Setelah itu, buka program yang membuat cadangan, buka bagian cadangan dan pulihkan yang terbaru.



Voila!

Setelah itu, Anda dapat mengaktifkan sinkronisasi kontak di iCloud dan kontak Anda tidak akan pernah hilang lagi.

Latar Belakang Posting


Saya sendiri tidak pernah memiliki smartphone di WP, setelah Nokia dan Sony Ericsson saya mulai menggunakan iPhone dan selalu bersukacita pada kesederhanaan sinkronisasi kontak antara semua perangkat. Tetapi kisah itu tidak terjadi dengan iPhone saya, tetapi dengan telepon istri saya.

Pada titik tertentu, pada iPhone yang disumbangkan istri, alih-alih Lumia, kontak mulai menghilang. Mereka menghilang tidak sekaligus, tetapi secara selektif, dan pada awalnya itu tidak tampak sebagai masalah besar. Tetapi ketika sekitar setengah dari kontak penting sudah pergi, dan bukannya mereka yang misterius "Lida", "Suami" (dan kami belum menikah saat itu), "Tetangga" dan lainnya - air mata pertama tumpah. Yang menarik dari cerita ini adalah bahwa setelah hadiah saya berbicara banyak tentang kelebihan iOS, khususnya, bahwa sekarang semua informasi terlindungi dengan aman, dan sinkronisasi dengan cloud mencegah kemungkinan kehilangan data, bahkan jika Anda kehilangan iPhone itu sendiri ...

Menyadari bahwa saya perlu segera menghentikan ini, saya mulai mempelajari pengaturan. Saya mengklik tanda centang di iCloud - semuanya tampak berfungsi. Saya mencoba mengubah atau menghapus kontak - semuanya seperti biasa. Saya memanjat semuanya naik dan turun sampai saya menemukan bagian "Mail, alamat, kalender" di pengaturan.

Jawabannya adalah bahwa kontak diikat ke akun Microsoft (Live), dan istri memberikan Lumia kepada ibunya, sebagai imbalan untuk Nokia sederhana. Akun dengan kontak di telepon tidak diikat, dan sang ibu, pada gilirannya, melihat kontak putri yang tidak dikenal di telepon, memutuskan untuk "membersihkan" mereka, yang dia ambil 10-15 menit sehari.

"Hore!" - kami bersukacita. Poltergeist terdeteksi, ia tetap mentransfer kontak dari Live ke iCloud dan melupakannya sebagai mimpi buruk.

Tapi itu tidak ada di sana.

Anda hanya dapat menghapus kontak dari akun Microsoft dalam format CSV, tetapi pada Mac saya, saya tidak dapat mendekripsi file scrawl yang dihasilkan sehingga saya selanjutnya dapat mengimpornya ke iPhone. Saya mencoba melalui program "Kontak" bawaan, melalui layanan online, konverter offline, tetapi programmer Microsoft melakukan yang terbaik, dan tidak peduli seberapa keras saya mencoba, ada masalah dengan pengkodean dan informasi tidak dibaca.
Kami merasakan musuh! Bagaimanapun, metode ini jelas tidak ditemukan untuk orang yang tidak tergoda oleh pengetahuan khusus dalam kriptografi.



Dukungan teknis Apple tidak dapat membantu, seperti halnya dukungan teknis Microsoft, apalagi berbagai forum.

Akibatnya, setelah beberapa waktu, saya teringat tentang sebuah utilitas yang telah lama diunduh untuk menghapus duplikat di buku catatan, mengaduk-aduk perangkat lunak yang diunduh di AppStore dan menemukan celah pada pengembang Microsoft yang cerdik ini.
Untuk nefig, buka platform lain!
