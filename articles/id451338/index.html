Pengembang distribusi Linux yang populer berencana untuk go IPO publik dan berurusan dengan solusi cloud

Canonical, perusahaan pengembangan Ubuntu, bersiap untuk penawaran umum. Dia berencana untuk berkembang di bidang komputasi awan.



/ foto NASA (PD) - Mark Shuttleworth di ISS

Bicara tentang IPO Canonical telah berlangsung sejak 2015 - maka pendiri perusahaan, Mark Shuttleworth, mengumumkan kemungkinan penawaran umum saham. Tujuan dari IPO adalah untuk mengumpulkan dana untuk membantu Canonical mengembangkan produk untuk cloud dan sistem IoT perusahaan.

Sebagai contoh, perusahaan berencana untuk lebih fokus pada teknologi kontainerisasi LXD dan OS untuk gadget Ubuntu Core IoT. Pilihan arah pengembangan ini ditentukan oleh model bisnis perusahaan. Canonical tidak menjual lisensi dan menghasilkan uang dari layanan B2B.

Canonical mulai mempersiapkan IPO pada tahun 2017. Untuk menjadi lebih menarik bagi investor, perusahaan berhenti mengembangkan produk yang tidak menguntungkan - shell desktop Unity dan OS ponsel Ubuntu Phone. Canonical juga berupaya meningkatkan pendapatan tahunan - dari $ 110 menjadi $ 200 juta. Oleh karena itu, sekarang perusahaan berusaha menarik lebih banyak pelanggan korporat. Untuk ini, paket layanan baru diperkenalkan - Ubuntu Advantage for Infrastructure.

Canonical tidak memerlukan biaya terpisah untuk perbaikan komponen infrastruktur berdasarkan berbagai teknologi - OpenStack, Ceph, Kubernetes, dan Linux. Biaya layanan dihitung berdasarkan jumlah server atau mesin virtual, dan paket termasuk dukungan teknis dan hukum. Menurut Canonical, pendekatan ini akan membantu pelanggan mereka menabung.

Langkah lain dalam menarik pelanggan adalah memperpanjang periode dukungan Ubuntu dari lima menjadi sepuluh tahun. Menurut Mark Shuttleworth, siklus hidup yang lebih lama dari sistem operasi penting bagi organisasi keuangan dan telekomunikasi, yang, jika dibandingkan dengan perusahaan lain, kecil kemungkinannya untuk beralih ke versi baru dari OS dan layanan TI.

Tindakan Canonical membantu menjadikan Ubuntu lebih populer di antara organisasi "konservatif" dan memperkuat posisi pengembang di pasar cloud. Upaya perusahaan dapat segera membuahkan hasil. Ada kemungkinan bahwa penawaran umum saham Canonical akan terjadi pada tahun 2020.

Ada apa dengan pasar ini


Analis percaya bahwa dengan transisi ke status Canonical publik akan dapat menjadi pesaing penuh untuk Red Hat. Yang terakhir telah mengembangkan dan menerapkan prinsip-prinsip monetisasi teknologi open source, yang sekarang digunakan Canonical.

Untuk waktu yang lama, perusahaan lain dengan model bisnis serupa tidak dapat tumbuh dengan ukuran Red Hat. Dalam hal skala, ini secara signifikan mengungguli Canonical - pendapatan tahunan Red Hat sendiri melebihi semua pendapatan Ubuntu. Namun, para ahli percaya bahwa dana IPO akan membantu Canonical tumbuh ke ukuran pesaing.

Pengembang Ubuntu memiliki keunggulan dibandingkan Red Hat. Canonical adalah perusahaan independen yang memberikan pelanggan korporat kemampuan untuk memilih lingkungan berbasis cloud apa pun untuk menyebarkan aplikasi. Red Hat akan segera menjadi bagian dari IBM. Meskipun raksasa TI berjanji untuk mempertahankan kemandirian anak perusahaan, ada kemungkinan Red Hat akan mempromosikan cloud publik IBM.


/ foto Bran Sorem ( CC BY )

IPO juga diharapkan membantu Canonical mendapatkan pijakan di pasar IoT dan edge computing. Perusahaan ini mengembangkan produk baru berbasis Ubuntu yang akan membantu menggabungkan perangkat tepi dengan lingkungan cloud menjadi satu sistem hybrid. Meskipun arah ini tidak membawa keuntungan bagi Canonical, Shuttleworth menganggapnya menjanjikan untuk masa depan perusahaan. Alat IPO akan membantu mengembangkan teknologi untuk IoT - Canonical akan dapat mengalokasikan lebih banyak sumber daya untuk pengembangan produk tepi.

Siapa lagi yang masuk IPO


Pada April 2018, Pivotal menempatkan sebagian sahamnya di bursa saham. Dia sedang mengembangkan platform Cloud Foundry untuk menyebarkan dan memantau aplikasi di lingkungan cloud publik dan pribadi. Sebagian besar Pivotal dimiliki oleh Dell: raksasa IT tersebut memiliki 67% saham di perusahaan dan peran yang menentukan dalam pengambilan keputusan.

Penempatan sekuritas di bursa seharusnya membantu Pivotal memperluas kehadirannya di pasar layanan cloud. Perusahaan berencana untuk menghabiskan hasil pengembangan produk baru dan menarik perusahaan terbesar di dunia sebagai klien. Harapan Pivotal menjadi kenyataan - setelah penjualan saham, dia berhasil meningkatkan pendapatan dan jumlah pelanggan korporat.

IPO lain di pasar akan segera dilaksanakan. Pada bulan April tahun ini, startup Fastly, yang menawarkan platform untuk komputasi periferal dan solusi penyeimbangan beban untuk pusat data, mengajukan penawaran saham. Perusahaan akan mengeluarkan uang dari IPO untuk mempromosikan komputasi tepi di pasar. Dengan cepat berharap investasi akan membantunya menjadi pemain yang lebih menonjol di industri layanan pusat data.

Apa selanjutnya


Menurut Wall Street Journal (sebuah artikel di bawah paywall), saham organisasi teknologi B2B mungkin lebih menarik daripada sekuritas di sektor TI B2C. Oleh karena itu, IPO di segmen B2B biasanya menarik perhatian investor serius.

Tren ini juga relevan untuk bidang komputasi awan, karena perusahaan IPO seperti Canonical memiliki peluang besar untuk sukses. Hasil dari penjualan saham akan membantu industri cloud untuk lebih aktif mengembangkan teknologi yang sekarang ada permintaan khusus di antara klien korporat - solusi multi - cloud dan sistem komputasi periferal.
