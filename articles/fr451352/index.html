Le modèle Visual Studio C ++ IntelliSense se remplit en fonction des instanciations de votre code

Depuis que nous avons annoncé Template IntelliSense , vous nous avez tous fait d'excellentes suggestions. Une suggestion très populaire était de faire en sorte que la barre de modèles remplisse automatiquement les candidats en fonction des instanciations dans votre code. Dans Visual Studio 2019 version 16.1 Preview 2 , nous avons ajouté cette fonctionnalité via une option «Ajouter toutes les instanciations existantes» dans le menu déroulant de la barre des modèles. Les exemples suivants proviennent de la base de code SuperTux .



Le menu déroulant de la barre des modèles contient désormais une nouvelle entrée, «Ajouter toutes les instanciations existantes».


Cliquez sur l'option «Ajouter toutes les instanciations existantes» pour remplir la liste déroulante, vous n'avez donc plus besoin de taper manuellement les entrées.



De plus, similaire à Rechercher toutes les références , une fenêtre en bas de l'éditeur vous montre où chaque instanciation a été trouvée et quels étaient ses arguments.
