Les satellites de Dauria Aerospace ne contactent pas

Le 14 juillet 2017, le lanceur Soyouz-2.1a a été lancé depuis le cosmodrome de Baïkonour avec une charge utile du grand satellite Canopus-V-IK et 72 petits engins spatiaux, établissant ainsi un nouveau record pour le programme spatial russe en termes de quantité. vaisseau spatial lancé dans l'espace.

Ainsi commence l'article «Nous avons trouvé le phare en orbite» sur GT. L'article est consacré à l'un de ces 72 petits satellites - le vaisseau spatial Mayak de type 3U CubeSat (Université polytechnique de Moscou). Ce satellite a été décrit par l'article «Une étoile artificielle appellera dans l'espace» sur GT avant même son lancement, et dans les commentaires il n'y avait pas quelques exemplaires cassés. Le satellite a été lancé sous le contrat de Glavkosmos OJSC. Toujours dans le cadre du contrat, le vaisseau spatial russe Iskra-MAI-85, type 3U CubeSat (Moscow Aviation Institute) et le vaisseau spatial Equateur UTE-South-West State University, type 1U CubeSat (Southwest State University) ont été lancés pour ce lancement.

Deux satellites de ce groupe ont été lancés dans le cadre de contrats fédéraux:

  • SC «MKA-N n ° 1», type 6U CubeSat (Russie, Dauria, sur ordre de ROSKOSMOS);
  • SC «MKA-N n ° 2», type 6U CubeSat (Russie, Dauria, sur ordre de ROSKOSMOS).
