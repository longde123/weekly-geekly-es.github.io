Lupa Total.js?



Publikasi ini adalah untuk tujuan informasi, karena saya ingin mendukung kerangka kerja yang, menurut pendapat saya, diremehkan dalam CIS dan perlu perhatian. Hal pertama yang saya coba cari tahu adalah seberapa produktifnya dibandingkan dengan pesaing utama koa.js, express.js, dll. Setelah membaca beberapa artikel dan forum, menjadi jelas bahwa total lebih cepat dan sulit untuk menyangkal telah melakukan tes semacam itu.





Total.js telah menjadi platform keseluruhan, apalagi, gratis dan sekarang menggunakan alat ini untuk meluncurkan aplikasi Anda akan menjadi lebih mudah dan lebih cepat.

Saya akan bercerita tentang tumpukan utama, yang saya gunakan sendiri.

SuperAdmin adalah sistem manajemen aplikasi node.js (tidak hanya dengan total).

  • Buat sertifikat SSL dengan pembaruan
  • SMS dan email
  • Pengidentifikasi Akses Pribadi API
  • Integrasi yang mudah dengan GIT
  • Backup otomatis FTP
  • Browser file
  • Statistik
  • Menggunakan NGINX sebagai proxy terbalik
  • Template Aplikasi

Alat ini bagi saya adalah hal yang mudah dan fungsional. Jika ada banyak aplikasi di server dan tidak ada keinginan untuk main-main dengan buruh pelabuhan untuk membuat saluran pipa untuk meluncurkan pembaruan, ini adalah pilihan yang sangat baik.

Eshop adalah toko lengkap yang dibangun di atas platform alat lain dari total CMS .
Dari cms, ada juga set widget, editor pos, dan semua barang serupa yang tipikal dari cms. Seperti di toko lengkap lainnya, fungsionalitas untuk mencari dan mengedit barang dan pesanan dilaksanakan di sini, ditambah pembayaran untuk pembayaran. Saat ini, penulis hanya mendukung database nosql-nya sendiri, ini adalah level yang layak, tetapi saya ingin yang lain) Penulis menjawab bahwa ia mempunyai rencana untuk mengimplementasikan postgresql, mysql, dan walaupun ini bukan rencana utamanya, ia tidak menentang melakukan ini dengan permintaan dan dukungan yang tepat (mungkin satu sen) dari masyarakat. Jadi letakkan bintang, ajukan pertanyaan, buat keributan (setidaknya pada github yang sama) atau hubungkan ke pengembangan.



Flow adalah editor kode visual yang nyaman, yang pada gilirannya dapat dihubungkan sebagai modul ke aplikasi total lainnya. Ada keinginan untuk menggunakan untuk hal-hal Internet, ya tolong, ambil sekelompok alat Dashboard lain untuk menampilkan indikator dan banyak lagi.



OpenPlatform adalah aplikasi sederhana yang dapat mengelola aplikasi pihak ketiga dan menyediakan layanan dasar, seperti manajemen pengguna dan peran. Setiap aplikasi berjalan dalam konteks OpenPlatform dalam HTML-iframe. Bergantung pada hak akses, aplikasi yang berjalan dapat membaca daftar pengguna dan aplikasi platform, membuat notifikasi dan berkomunikasi dengan aplikasi lain melalui karyawan layanan. Administrator memiliki kontrol penuh atas hak akses pengguna dan aplikasi. Masih dalam versi beta, tetapi sedang aktif berkembang. Ini sangat berguna jika Anda membangun infrastruktur, misalnya dalam hubungannya dengan aplikasi di atas.



Kesimpulan
Ini hanya setengah dari semua alat total yang dapat Anda temukan di sini . Jika Anda tertarik pada platform total, maka Anda dapat mulai belajar dengan membaca dokumentasi atau wiki dan, jika mungkin, mendukung pengembangan. Dimungkinkan untuk menjual aplikasi Anda sendiri berdasarkan total di Marketplace lokal, meskipun untuk ini Anda harus menjadi premium, berikut detailnya. Sedangkan bagi saya, ini adalah tumpukan open souce yang paling layak pada sebuah node, yang untuk beberapa alasan tidak sepatutnya di tempat teduh.
