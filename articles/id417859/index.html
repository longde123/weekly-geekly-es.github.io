3CX Integrasi dengan Microsoft Dynamics 365

Dalam artikel ini, kami akan menunjukkan kepada Anda bagaimana menghubungkan integrasi Microsoft Dynamics 365 CRM ke 3CX IP PBXs.

Saat ini, 3CX sedang menyelesaikan dukungan integrasi di sisi klien 3CXPhone dan mentransfer semua integrasi CRM ke sisi server PBX menggunakan teknologi REST API. Ini memungkinkan Anda untuk mengonfigurasi integrasi satu kali untuk semua pengguna di antarmuka manajemen 3CX. Administrator tidak perlu mem-bypass semua workstation dan bahkan menginstal aplikasi apa pun di komputer pengguna - cukup gunakan klien web berbasis browser 3CX.

Dengan panggilan masuk, kartu mini dari kontak dari CRM terbuka dengan nama, nama keluarga dan nomor telepon di antarmuka klien web. Ada tombol kecil di kartu, dengan mengklik di mana Anda akan membuka kartu lengkap langsung di antarmuka Microsoft Dynamics 365 CRM.

Ingatlah bahwa semua integrasi CRM disertakan secara gratis dalam lisensi 3CX Pro .

Konfigurasikan Microsoft Dynamics 365


Masuk ke akun Microsoft Dynamics 365 Anda, dan kemudian pergi ke portal Azure untuk membuat aplikasi yang Anda butuhkan untuk otentikasi. Buka Azure Active Directory> Pendaftaran aplikasi, tombol pendaftaran aplikasi baru.



Tentukan parameter aplikasi:

  • Nama: Sisi Server 3CX
  • Jenis aplikasi: Asli
  • Redirect URI: https: // 3CX_URL / api / oauth2crm . 3CX_URL - alamat web server 3CX


Setelah mengklik tombol Buat, Anda akan melihat data aplikasi, khususnya, ID Aplikasi, yang kemudian ditunjukkan di sisi 3CX.



Selanjutnya, Anda perlu menentukan izin untuk mengakses informasi di Microsoft Dynamics 365. Buka Pengaturan> Izin yang diperlukan, tombol Tambah.



Klik Pilih API dan pilih API Dynamics CRM Online.



Tetapkan semua izin yang tersedia, lalu tekan Pilih dan Selesai.



Anda dapat menghapus izin Direktori Aktif Windows Azure karena itu tidak diperlukan untuk pekerjaan integrasi. Aplikasi otentikasi dibuat.

Sekarang Anda harus mendapatkan ID Tenant, yang juga digunakan di sisi server 3CX. Buka Azure Active Directory> Pendaftaran aplikasi, tombol Endpoints.



Komit string penyewa yang ada di URL titik akhir. Misalnya, OAUTH 2.0 AUTHORISATION ENDPOINT memiliki URL login.microsoftonline.com/dedc4ebb-183f-42e2-9192-92eb425a027d/oauth2/authorize . Inilah garis tenant, dedc4ebb-183f-42e2-9192-92eb425a027d.

Mengkonfigurasi 3CX CRM Server Integration


Buka Pengaturan> Integrasi CRM, tab sisi-Server, dan unduh integrasi Microsoft Dynamics 365.


Kemudian pilih DynamicsCRM dari daftar drop-down.



Tentukan ID Aplikasi dan parameter Penyewa yang disalin pada langkah sebelumnya. Di sini Domain adalah alamat web portal Dynamics 365 Anda, lalu klik Otorisasi. Tab baru terbuka meminta izin untuk aplikasi.



Klik Terima - token otorisasi akan muncul di bawah tombol Otorisasi. Simpan perubahannya.

Pada akhirnya, Anda dapat menguji integrasi dengan mengklik tombol Uji.
