PHP Digest No. 136 (24 Juli - 6 Agustus 2018)


Seleksi segar dengan tautan ke berita dan bahan. Dalam rilis: PHP 7.3.0 Beta 1, PhpStorm 2018.2, Composer 1.7 dan rilis lainnya, standar PSR-17, review Yii 3.0, implementasi alternatif dari API API untuk aplikasi asinkron, sebagian dari alat yang bermanfaat, dan banyak lagi yang diterima.

Selamat membaca!



Berita dan Siaran


  • PHP 7.3.0 Beta 1 - Dengan rilis beta pertama, fase pengembangan aktif berakhir, yang berarti bahwa daftar fitur baru dan perubahan di cabang 7.3 dapat dianggap final. Beta berikutnya diharapkan pada 16 Agustus.
  • PhpStorm 2018.2
    • Pencocokan namespace yang ditingkatkan
    • Pencarian dan penggantian struktural
    • Pola tambahan postfix khusus
    • Antarmuka yang diperbarui dan dukungan untuk panel sentuh
    • serta inspeksi baru dan peningkatan lainnya .
  • PSR-17: Pabrik HTTP - Sebuah standar telah diadopsi secara formal yang mengatur antarmuka pabrik untuk membuat objek HTTP yang kompatibel dengan PSR-7 .
  • PHP 7.2 tersedia di Google Cloud Platform
  • PHPUnit 7.3 - Di antara perubahan adalah kemampuan untuk menjalankan tes pertama yang jatuh pada menjalankan sebelumnya.
  • Komposer 1.7.0

PHP internal


  • [PHP]: Cookie Situs yang Sama - Kemampuan untuk melewatkan array opsi, termasuk dukungan untuk standar Cookie situs yang Sama , telah ditambahkan ke setcookie() dan fungsi lain untuk bekerja dengan cookie. Sudah diterapkan di PHP 7.3.
  • [RFC]: Typed Properties 2.0 - Proposal untuk properti yang diketik ditunda hingga versi utama PHP berikutnya, sementara Dmitry Stogov menerbitkan hasil benchmark untuk menilai overhead fitur ini.

Alat-alatnya


  • spiral / roadrunner - Server aplikasi berkinerja tinggi, penyeimbang beban, dan manajer proses untuk PHP yang diimplementasikan pada Go. Dapat digunakan untuk menjalankan PHP di AWS Lambda .
  • codeplea / ahocorasickphp - Implementasi dari algoritma Aho - Korasik untuk mencari beberapa substring dalam sebuah string. Lebih cepat daripada memanggil strpos() beberapa kali, dan jauh lebih cepat daripada memanggil preg_match_all() .
  • chekalskiy / php-bank-db - Perpustakaan untuk menentukan bank dengan nomor kartu. Dikirim oleh Ilya Chekalsky .
  • igniphp / framework - Kerangka kerja ringan dengan dukungan untuk PSR-15, PSR-7, serta berjalan di Swoole .
  • spatie / phpunit-snapshot-assersions - Alat untuk mengimplementasikan pengujian snapshot pada PhpUnit. Membandingkan hasil tes dengan menjalankan sebelumnya.
  • elgentos / masquerade - Alat untuk menganonimkan informasi dalam database. Out of the box mendukung Magento 2.
  • zendframework / zend-problem-details - Menerapkan dukungan untuk " Rincian Masalah RFC 7807 standar untuk API HTTP " untuk aplikasi PSR-7, yang mengatur respons kesalahan.
  • formapro / pvm - Perpustakaan yang kuat untuk menggambarkan alur kerja, proses bisnis, dan hanya mesin negara. UI yang tersedia untuk visualisasi.
  • Rican7 / masuk - Perpustakaan dirancang untuk mengubah input mentah dari sumber apa pun menjadi struktur yang diketik dengan kuat.
  • rezozero / mixedfeed - Perpustakaan memungkinkan Anda untuk menggabungkan umpan berbagai layanan sosial menjadi satu umpan.
  • KikApp - Alat ini memungkinkan Anda untuk menulis aplikasi asli untuk iOS dan Android dalam PHP. Adakah yang punya pengalaman menggunakannya?
