Himbeer Pi 2 Bierkühlschrank



Für viele Bierliebhaber ist es wichtig, eine ausreichende Menge des Getränks im Kühlschrank zu halten. Zwar müssen Sie neben Bier auch Lebensmittel im Kühlschrank aufbewahren, damit die Kapazität eines herkömmlichen Kühlschranks stark abnimmt. Trotzdem ist dieses Problem ganz einfach gelöst - Sie müssen einen Kühlschrank ausschließlich für Bier erstellen.

Es war dieser Kühlschrank, den David Sulpy mit dem Raspberry Pi 2 kreierte, um das System intelligent zu machen. Außerdem wurde ein anderes Gerät für das Design verwendet - das Wii Balance Board.

Das Ergebnis ist „ Fridge of Awesomeness “ - ein Bierkühlschrank, der Daten an eine PC-Anwendung, ein Smartphone oder ein Tablet überträgt. Innerhalb des Systems - eine große Anzahl verschiedener Sensoren sowie Rasperry Pi 2, ein Mini-PC, der den gesamten Prozess steuert. Sie können die Parameter des Kühlschranks von überall auf der Welt steuern.



Die Verwaltung erfolgt über das Admin-Panel, in dem der Benutzer Informationen über die Temperatur, die Anzahl der Flaschen im Inneren und die Anzahl der Flaschen sieht, die vom ersten Tag des Systems an aus dem Kühlschrank genommen wurden. Alle Informationen werden in digitaler Form und in Form von Diagrammen bereitgestellt.

Der Benutzer kann auch sehen, an welchen Tagen das Bier am aktivsten herausgenommen wird. Vorsichtshalber können Sie sehen, wann die Kühlschranktür zuletzt geöffnet wurde.
