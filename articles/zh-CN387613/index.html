Raspberry Pi零的4个USB端口？没问题



微型PC Raspberry Pi Zero于最近发布，但“树莓”的粉丝已经设法与开发板合作。因此，其中一位用户向迷你PC添加了WiFi模块。另一位则从棋盘上制作了一个微型游戏机，将其直接集成到游戏控制器中。尽管无法将这些项目称为复杂或精美的项目，但它们引起了树莓其他所有者的一定兴趣。

现在，已将另一个项目添加到指示的项目列表中。因此，另一位购买Raspberry Pi Zero的买家认为板子的一个USB端口不够用，因此决定再增加一些。同时，他可以使用USB集线器LogiLink UA0160，其大小几乎与主板完全匹配。



食物第一


最初，有必要使用集线器为Pi Zero供电。为此，决定使用USB设备。LogiLink UA0160摆脱了这种情况，开始了工作。

连接如下：
  • Pi Zero PP1-5V集线器；
  • Pi Zero PP6-GND集线器;


使用万用表，用户可以确定所有操作均已正确完成。





现在USB


下一步是将Pi上的USB OTG连接到集线器的USB端口。做到了：

  • Pi Zero PP22-D +集线器（在微型USB连接器上）；
  • Pi Zero PP23-集线器集线器（在微型USB连接器上）；






我们合并两个板


为了将两个板连接在一起，决定不打扰，并使用通常的松紧带。好了，为了避免连接两个板的触点，使用了薄泡沫橡胶。







所有这一切都非常简单，但是复杂的项目，如往常一样，将在稍后出现。现在，用户正在提供简单的解决方案，以及扩展电路板功能的方法。此选项肯定比焊接到无线通信模块的“树莓”更好。现在，您可以添加一个模块，以及更多。
