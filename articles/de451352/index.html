Visual Studio C ++ - Vorlage IntelliSense wird basierend auf Instanziierungen in Ihrem Code ausgefüllt

Seit wir Template IntelliSense angekündigt haben , haben Sie uns alle großartige Vorschläge gemacht. Ein sehr beliebter Vorschlag war, dass die Vorlagenleiste Kandidaten basierend auf Instanziierungen in Ihrem Code automatisch ausfüllt. In Visual Studio 2019 Version 16.1 Vorschau 2 haben wir diese Funktionalität über die Option "Alle vorhandenen Instanziierungen hinzufügen" im Dropdown-Menü "Vorlagenleiste" hinzugefügt. Die folgenden Beispiele stammen aus der SuperTux- Codebasis.



Das Dropdown-Menü Vorlagenleiste enthält jetzt den neuen Eintrag "Alle vorhandenen Instanziierungen hinzufügen".


Wenn Sie auf die Option "Alle vorhandenen Instanziierungen hinzufügen" klicken, wird die Dropdown-Liste ausgefüllt, sodass Sie keine Einträge mehr manuell eingeben müssen.



Ähnlich wie bei " Alle Referenzen suchen" zeigt Ihnen ein Fenster am unteren Rand des Editors, wo jede Instanziierung gefunden wurde und welche Argumente sie hatte.
