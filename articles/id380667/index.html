Matrix Flare - mainan untuk memperkenalkan anak-anak ke dunia elektronik



Proyek Matrix Flare dimulai sebagai mainan khusus untuk satu anak, dan secara bertahap berkembang menjadi desainer elektronik pengajaran untuk semua orang. Penulis meluncurkan kampanye di Kickstarter untuk mengumpulkan dana untuk proyek tersebut, berharap dapat meningkatkan jumlah untuk produksi massal mainan.

Proyek ini tidak mungkin menghasilkan jumlah uang yang tepat, tetapi cukup menarik, jadi kami memutuskan untuk menggambarkannya di blog Medgadgets. Penulis ide tersebut berharap Matrix Flare akan membantu anak-anak (dan bukan hanya mereka) untuk memahami dasar-dasar pemrograman dan elektronik.

Apa itu Matrix Flare?


Ini adalah kubus di mana wajah terdiri dari elemen individu yang diterangi oleh LED dengan warna berbeda. Untuk mengontrol setiap kubus, perangkat lunak khusus yang berjalan di Visual.net digunakan, kode untuk Arduino digunakan.

Input data disimpan dari Pixel Maker App ke file anim.h yang digunakan oleh Arduino IDE Trinket Jewelry Sketch dari Adafruit.com. Sketsa dan instruksi program dapat digunakan untuk setiap "matriks". Pada saat yang sama, Arduino IDE adalah alat perangkat lunak gratis.



Jika mau, Anda dapat memesan program yang sudah jadi dengan kubus Matrix Flare Anda.

Semua urutan animasi dapat diubah dan dimuat dengan cara baru. Setelah akhir kampanye, sejumlah besar animasi untuk kubus akan diposting di situs web pabrikan.



Majelis Matriks Flare


Secara default, animasi dengan kuda hadir dengan setiap dadu, sehingga semuanya berjalan di luar kotak. Kasing Matriks Flare dibuat pada printer 3D, ada berbagai pilihan.

Adapun karakteristik Matrix Flare, mereka adalah sebagai berikut:
  • Antarmuka USB tipe B mikro menggunakan Arduino IDE;
  • Baterai 3,7 VDC 150mAH, pengisian melalui port yang ditunjukkan di atas;
  • Jika diinginkan, kubus dapat tetap terhubung ke USB;
  • Ketika kubus sederhana dimatikan, dan untuk memainkan animasi, Anda harus mengklik tombol Reset;




Case dadu dicetak pada printer 3D, sementara ada beberapa model case.
