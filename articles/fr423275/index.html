Webinaires Skillbox Friday: Tout pour les débutants



Les webinaires sont un bon moyen d'obtenir des informations et de l'expérience en dehors des murs d'un établissement d'enseignement. Pour visualiser et écouter, vous n'avez besoin que d'un ordinateur portable, d'un appareil mobile et, probablement, d'un ordinateur portable pour enregistrer des thèses intéressantes. Nous espérons que vous avez tout cela, car il est temps de publier une nouvelle collection de Skillbox. Aujourd'hui, c'est tout pour les développeurs.

Quel langage de programmation choisir


Mikhail Ovchinnikov, ingénieur logiciel en chef de Badoo, explique comment et quelle langue choisir pour construire la carrière d'un développeur. De plus, il discute du choix de l'orientation dans l'informatique - du jeu à l'IA - et explique comment développer et construire une carrière.


Comment devenir développeur Java


Daniil Pilipenko, développeur professionnel et directeur du SymbioWay Center, partage ses réflexions et son expérience personnelle sur la façon de devenir programmeur Java.

Le webinaire parle de la syntaxe du langage et de la pratique de développement de programmes Java avec une discussion sur les erreurs possibles, et en parallèle, sur les composants du professionnalisme du développeur et les éléments clés qui composent le chemin du développeur Java du débutant au pro.

Comment programmer en Python


Vadim Shandrinov, programmeur-mentor à l'Université Skillbox, explique comment travailler avec Python, sans oublier le potentiel d'un développeur Python sur le marché du travail.

Dans le processus, vous ferez un travail pratique en créant votre premier produit Python.
Cours en ligne pour développeur Python à partir de zéro


Comment développer une application pour iOS


Nikita Arkhipov, PDG d'Anvics, parle et montre comment l'application iOS est en cours de développement. La partie centrale du webinaire est la création d'une application ToDo en ligne. Expliquant comment procéder, Nikita parle des principes de travail avec Swift et Xcode, donne une idée de la manière d'associer une interface au code et enseigne comment afficher les données et traiter les actions des utilisateurs.
Cours pratique "Mobile Developer PRO"


Lifehacks pour les programmeurs débutants


Le directeur de la production numérique bêta, Gleb Mikheev, partage des astuces pour les programmeurs débutants. Il les a personnellement collectées pendant son travail. Entre autres choses utiles sont considérées:

  • Détails sur le travail et la vie de l'agence numérique Frontend Developer à Moscou.
  • Démystifier les mythes sur la proximité des programmeurs.
  • Ce dont vous avez besoin pour devenir un bon codeur.
