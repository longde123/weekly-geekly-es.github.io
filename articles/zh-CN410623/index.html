对于尚未与我们在一起的人：什么是SMD？


从英语翻译起，SMD是表面安装的设备，即“表面安装的设备”。 与在小型化的下一阶段之前的技术不同，SMD元件占用的空间要少得多。 SMD使设备非常紧凑。 只需查看任何智能手机的主板即可了解所面临的风险。

SMD有多种尺寸。 矩形元件（例如LED或电阻器）沿边的长度进行测量。 例如，在Arduino上安装了0805 LED，从英寸系统转换为公制系统，相当于2 x 1.25毫米。 而且，同一板上的大多数陶瓷电容器的尺寸为0603 = 1.6 x 0.8 mm。



常规二极管具有不同的尺寸。 例如，SOD-123二极管的尺寸对应于3.68 x 1.17 x 1.60 mm。 这是一个三足晶体管的示例：SOT-323 = 2 x 1.25 x 0.95 mm。 通常，SMD外壳的类型和尺寸种类繁多。

LED手表套装


众所周知，该套件包含各种尺寸的SMD和一块印刷电路板，所有这些都需要焊接在上面。 板上已经有一个微控制器和一个石英谐振器，我们不敢单独提供（至少在此版本中）。

手表的核心是微控制器MSP430G2553。 这个选择似乎很奇特。 但是，时钟并未定位为教学编程的支架。 我们应用了在这种情况下最佳的方法。

为了使时钟正常工作，您需要在板上安装61个LED，一些电阻器和陶瓷电容器。 训练是逐步进行的，并逐渐复杂化。 首先，您需要焊接12个最大的LED 1206，然后焊接49个，但较小的LED-0805。最后，仅需焊接一点点最小的电阻和电容器0603。

综上所述，您将获得一只虚拟手可以正常工作的LED手表！

该套件包括：

  • 带有预安装且已编程微控制器的印刷电路板；
  • LED尺寸为1206和0805;
  • 电阻0603;
  • 电容器尺寸0603;
  • 电池座；
  • CR2032电池。

现在使用该工具。 通常，大多数学童被迫用爸爸或最有可能是祖父的60-80瓦特厚的烙铁焊接。 熟练地，您可以使用这种设备焊接SMD，但这是一种暴力形式。

您还将需要镊子，嘴唇要薄而均匀。 一般情况下，镊子应单独放置。 一些疏忽大意的人不断努力用锋利的镊子采摘东西，他的嘴唇弯曲而钝了。 我强烈建议在妥善保管的地方保留用于SMD的镊子！
