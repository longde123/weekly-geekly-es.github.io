Eine neue Pflanzrunde für Windows 10: Der Hintergrundprozess überwacht Registrierungsänderungen

Am 29. Juli 2015 veröffentlichte Microsoft das Windows 10-Betriebssystem. Kostenlose Updates für das neue Betriebssystem sorgten für ein gutes Wachstum der Benutzer. Auf Computern von Benutzern von Windows 7 und 8 / 8.1 begann ein Dutzend zu verschütten. 24 Stunden nach der Veröffentlichung wurde es auf 14 Millionen Computern installiert, und vier Wochen später erreichte die Zahl 75 Millionen. Um neun Zahlen (einhundert Millionen) zu erreichen, brauchte das Betriebssystem nur zehn Wochen. Die neuesten Zahlen aus dem Microsoft-Blog besagen, dass Windows 10 200 Millionen Geräte steuert.

Die Zahlen sind beeindruckend, aber weit vom Ziel entfernt. In Redmond wollen sie in 2-3 Jahren an einer Milliarde Geräten arbeiten. Diese Zahl ist nicht so fantastisch, wenn man bedenkt, dass sie PCs und Laptops, Xbox-Spielekonsolen, Smartphones und Tablets umfasst. Aber das Wachstum verlangsamt sich immer noch. Dies ist wahrscheinlich der Grund, warum die Benutzer von G-7 und G-8 bereits begonnen haben, die viel größere Beharrlichkeit von Microsoft zu spüren.

Im Oktober des vergangenen Jahres ein Upgrade auf Windows 10 schlug in der empfohlenen Liste. Für viele Benutzer werden empfohlene Updates automatisch heruntergeladen und installiert. Wie von Microsoft angegeben, geschah dies versehentlich. Bereits im November wurde bekannt , dass dieses Update wieder in die Liste der empfohlenen Updates aufgenommen werden soll, diesmal jedoch offiziell.

Für ein kostenloses Upgrade auf Windows 10 ist das Update KB3035583 verantwortlich . Außerdem wird das Programm GWX (Get Windows 10) installiert. Letzterer überprüft den Computer auf Kompatibilität mit Windows 10. Wenn auf dem Computer die "Top Ten" ausgeführt werden können, bietet das Programm die Installation an. Das Beharren auf diesem Vorschlag überschreitet allmählich die Grenze: Im letzten Monat verschwand der "Ablehnen" -KnopfJetzt können Sie nur noch das Fenster schließen.

Nicht jeder möchte auf die Top Ten upgraden, und es kommen immer wieder Erinnerungen. Im Oktober versprach Microsoft , dass es möglich sein würde, Angebote für ein Upgrade vollständig abzulehnen, obwohl eine solche Möglichkeit noch nicht erschienen ist. Einige Benutzer haben völlig befreit von zanudlivym Vorschläge Werte Bearbeitung AllowOSUpgrade, DisableOSUpgrade, DisableGWXund ReservationsAllowedin der Registrierung. Es wäre genauer zu sagen, dass eine solche Lösung von Microsoft selbst vorgeschlagen wird: Ein Unternehmensvertreter empfahl, die Registrierung im offiziellen Forum zu bearbeiten.

Um das Zähmen von GWX und das Aktualisieren von KB3035583 zu vereinfachen, wurde das Dienstprogramm GWX Control Panel geschrieben(früher GWX Stopper). Es legt die erforderlichen Werte in der Registrierung fest, entfernt Symbole aus dem Benachrichtigungsbereich und Dateien, die schwer manuell zu löschen sind, und gibt die von den Aktualisierungsdateien belegten Gigabyte frei.

Woody Leonhard stellt fest, dass die neue Version des Updates KB3035583 Hintergrundprozesse einführt, die einige Registrierungswerte zweimal täglich überprüfen. Sie versetzt sie dann in ihren ursprünglichen Zustand, wenn sie geändert wurden. Josh Mayfield , Entwickler des GWX Control Panels, erklärte, wie dies geschieht.

Es gibt verschiedene Versionen von KB3035583. Alle von ihnen werden auf den neuesten Stand gebracht, sobald sie verfügbar sind. Zusätzlich zum auffälligen Benutzersymbol im Benachrichtigungsbereich enthält das Update mehrere Hintergrundprozesse, die bestimmte Aufgaben nach einem Zeitplan ausführen.

Die Werte können sich mit neuen Versionen des Updates KB3035583 ändern. Das Entfernen eines Updates funktioniert nicht immer, da es zurückgegeben werden kann, auch wenn Sie es ausblenden. Die GWX-Systemsteuerung vereinfacht das Bearbeiten dieser Registrierungswerte und führt mehrere andere Aufgaben im Zusammenhang mit einem unerwünschten Upgrade auf Windows 10 aus.

Laut Mayfield ist die Beständigkeit von Übergangsempfehlungen nicht nur auf die Werte in der Registrierung beschränkt. Unter Bezugnahme auf Bewertungen von nicht genannten IT-Beratern und Spezialisten für technischen Support deutete der Entwickler des GWX Control Panels auf einen größeren Umfang des Geschehens hin.
