Nintendo a demandé de supprimer l'émulateur JavaScript de Game Boy Advance de Github



La société Nintendo a demandé la suppression de github Emulator Game Boy Advance, écrit en JavaScript. Apparemment, la raison en était la présence de plusieurs jeux originaux de la console de jeu avec l'émulateur d'images. Actuellement, la page de l'émulateur n'est plus disponible. La demande a été envoyée par l'entreprise la semaine dernière.

L'histoire du jeu électronique portable culte Game Boy a commencé en 1989. Il est devenu le premier système de jeu populaire et les concurrents ne pouvaient pas répéter son succès. Nintendo a périodiquement publié de nouvelles variétés du jeu - en 1996, une version plus petite de Game Boy Pocket est apparue, en 1998 - Game Boy Color, la première version couleur du jeu. Et en 2001, Game Boy Advance est apparu, le dernier jeu à avoir une grande popularité.


Game Boy Advance

Bien que le dernier préfixe de cette série, Game Boy Micro, soit sorti en 2005, beaucoup de gens se souviennent encore avec nostalgie de leur jouet préféré et profitent de l'occasion pour jouer à des jeux familiers sur l'émulateur. Et, bien que la page de l'émulateur avec des jeux dans le kit soit fermée, l'émulateur sans jeux peut être trouvé à une adresse différente . De plus, github a une bonne liste d'émulateurs de différentes plateformes écrites en JavaScript.

Ce n'est pas la première fois que Nintendo s'attaque à l'émulateur Game Boy - l'année dernière, le même sort est arrivé à l'application mobile alors populaire pour iOS, GBA4iOS.
