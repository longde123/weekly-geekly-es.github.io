Microsoft akan meyakinkan pengguna tentang manfaat Edge



Pengguna yang bekerja dengan salah satu pratinjau build Windows 10 terbaru (build 10568) memperhatikan bahwa ketika mencoba menginstal browser default selain Edge pada sistem, sistem operasi menampilkan kotak dialog yang meminta Anda untuk berpikir dan masih memberikan browser Windows 10 kesempatan. Sumber daya Verge menarik perhatian pada keadaan ini .

Pengguna, menurut Microsoft, harus menghargai keunggulan Edge berikut: kemampuan untuk mengambil dan menyimpan catatan di halaman web, kehadiran mode "baca" khusus yang menyembunyikan elemen asing, dan integrasi dengan asisten suara Cortana. Dilihat oleh statistik lembaga analitik StatCounter, yang diterbitkan pada awal September, langkah seperti itu oleh Microsoft tampaknya tepat. Memang, meskipun Windows 10 memulai dengan sangat sukses, ternyata Edge tidak hanya tidak termasuk dalam peringkat browser sama sekali, tetapi juga pemilik sistem operasi baru itu sendiri lebih suka browser pesaing. Pangsa terbesar Edge - sekitar 20% - dicapai pada 30 Juli (yaitu, sehari setelah rilis Windows 10) dan setelah itu hanya berkurang. Selama satu setengah bulan terakhir, situasinya belum membaik sama sekali..

Karena kita hanya berbicara tentang pratinjau build Windows 10, tidak diketahui pasti apakah fitur yang diperhatikan akan muncul di versi final sistem operasi. Jika ini benar-benar terjadi, maka mungkin ada skandal yang terjadi pada awal Agustus ketika CEO Mozilla Chris Byrd menyatakan ketidakpuasannya dengan fakta bahwa ketika menginstal Windows 10, pengaturan default pengguna untuk aplikasi berubah dan dengan demikian Firefox kembali berada di belakang Edge. Kembali pada tahun 2009, regulator Uni Eropa mengakui dominasi Internet Explorer sebagai ancaman persaingan yang sehat dan mewajibkan Microsoft untuk menampilkan dialog khusus untuk memilih browser alternatif selama lima tahun saat memasang sistem operasi. Pada akhir tahun lalu, jangka waktu resep lima tahun berakhir .
