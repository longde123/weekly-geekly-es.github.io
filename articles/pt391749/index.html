O amplificador de som original - faça você mesmo

Neste artigo, falaremos sobre uma das opções para um amplificador de potência caseiro com base nos módulos do Master Kit. O projeto é implementado pelo usuário de nossos produtos.



Estruturalmente, o dispositivo consiste em quatro blocos funcionais principais:

1) o próprio amplificador de potência ( MP3106S , classe D, 2x40W);
2) controle de volume digital e nível de balança ( MP1231 );
3) indicador de intensidade do sinal digital MP1054 ;
4) uma fonte de alimentação de 12V para uma corrente de pelo menos 4A (o autor usou uma fonte de alimentação de computador).

Como acústica, foi utilizado o conhecido sistema de alto-falantes Radio System S-90 (estéreo - dois alto-falantes).

O esboço da conexão dos módulos é mostrado na figura:



Deve-se lembrar que o nível de ruído de todo o sistema depende da qualidade da fonte de energia. Cuidados especiais devem ser tomados ao usar fontes de alimentação do computador. É melhor usar fontes de alimentação ATX de marca de boa qualidade, pois na maioria das fontes de alimentação sem nome baratas, em vez de bloqueadores de filtros, estão instalados jumpers comuns, o que afeta extremamente negativamente a qualidade do som.

Existem algumas sutilezas nos módulos de conexão que podem causar dificuldades para engenheiros eletrônicos com formação insuficiente. Por exemplo, o amplificador de potência MP3106S, por padrão, implica uma maneira equilibrada de conexão, mas geralmente é mais conveniente conectar o sinal de entrada de acordo com o circuito clássico com um "terra" comum.

Para fazer isso, você precisa modificar levemente o diagrama de conexão do amplificador:



Além dos módulos básicos acima, além disso, tive que comprar uma “coisinha”: conectores de entrada e saída, chave liga / desliga, fixadores.

O protótipo do caso futuro era uma caixa de sapatos de papelão comum.



Nesse caso improvisado, todo o sistema foi depurado. Após a depuração, todos os componentes foram movidos para uma caixa de madeira.

O sistema parece digno: sem chiado, sem quedas visíveis na faixa de frequência. Obviamente, os amantes da música podem encontrar algumas falhas, mas, para clientes tão exigentes, são produzidos dispositivos prontos e sistemas de som HI-FI (com os mesmos preços do HI-FI). Mas a experiência mostra que, com a audição às cegas, 90% das pessoas não conseguem distinguir o som de um aparato valioso de tubo de um amplificador econômico e, em caso afirmativo, por que pagar mais? Além disso, um design faça você mesmo sempre parece o melhor!
