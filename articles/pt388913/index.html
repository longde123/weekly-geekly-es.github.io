Microsoft está se preparando para lançar seu próprio cartão SIM



Um programa gratuito chamado Cellular Data apareceu na loja de aplicativos móveis da Windows Store e fornece acesso à Internet móvel, independentemente dos contratos com os provedores móveis. Como segue a descrição, para que o programa funcione, você precisa usar um cartão SIM especial da Microsoft e uma versão móvel do sistema operacional Windows 10.

Supõe-se que os usuários paguem pelo acesso à Internet usando sua conta da Microsoft. Ainda não foi especificado em quais países e a que preço um serviço semelhante estará disponível.

Aparentemente, a Microsoft se tornará uma operadora móvel virtual que fornece serviços de dados em redes celulares alugadas.

O serviço relacionado já é oferecido pela Apple para seus tablets. O cartão SIM da Apple permite alterar as operadoras de telefonia móvel cujos serviços o usuário deseja trabalhar no momento.

As operadoras móveis virtuais que operam com as capacidades arrendadas por provedores existentes, mas com seus planos tarifários, começaram a aparecer em 2000. O primeiro a fazer isso foi o Tele2, com base no provedor dinamarquês Sonofon. Uma das divisões da Virgin, a Virgin Mobile, tornou-se a primeira operadora virtual no Reino Unido.

Atualmente, na Rússia, existem três dezenas de operadoras virtuais com base nas capacidades das três grandes e de alguns provedores regionais.
