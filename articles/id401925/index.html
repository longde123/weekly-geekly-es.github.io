Diperkenalkan oleh Pi Zero W



Memperkenalkan versi baru komputer papan-tunggal Rapberry Pi Zero - Pi Zero W dengan wi-fi dan bluetooth

Antena adalah bagian dari papan sirkuit:



Kasing resmi baru dengan biaya. Penutup atas tersedia dalam beberapa versi - dengan potongan kabel GPIO, lubang untuk modul kamera di tengah, dan benar-benar tertutup.



Beberapa spesifikasi:

  1. Dimensi: 65mm x 30mm
  2. Berat: 9 gram
  3. Prosesor: BCM2835 dengan VC4
  4. Frekuensi CPU Default: 1GHz dengan overclock cache L2
  5. Memori: RAM 512MB
  6. Modul Nirkabel: chip BCM43438 untuk WiFi / BT
  7. WiFi: 802.11 b / g / n
  8. Bluetooth: 4.0 (UART penuh dengan kontrol aliran)
