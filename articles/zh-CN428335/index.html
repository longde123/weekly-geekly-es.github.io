Siri快捷方式更新



除了发布有关新iPad和Mac的公告外，苹果还发布了Siri Shortcuts的新版本。 该应用程序最初是在WWWC上推出的，并随iOS12一起提供，可通过创建自己的团队和宏来释放Siri的潜力。 现在，应用程序可以执行新操作，例如设置时间和计时器，获取最新的天气预报等。

天气预报对于为早上的例行创建团队的人来说尤其有用。 使用此选项，您可以请求当前或某个特定时间段或某些特定参数（例如“降水机会”，“空气中的湿度百分比”等）的天气预报。

闹钟和计时器的新功能也将有所帮助：“创建闹钟”（“ Create Alarm”），“ On / Off Alarm”（“ Toggle Alarm”），“ Enable Timer”（“ Start timer”）。 以前，人们对存在这些功能感到困惑，但无法使用它们。 好吧，此更新还旨在解决此问题。

“测量”和“转换测量”功能使您可以在不同的测量单位之间进行转换。 也可以使用“获取最新导入”功能从“照片”应用程序中获取最新导入的照片。 您也可以要求使用“拍摄视频”功能立即开始录制视频。

有趣的是，Apple没有命名更新令牌之一。 很可能是因为所有新闻媒体都在谈论她。 我们正在谈论“控制”功能（在原始“警察”上）。 此功能允许您设置一系列事件。 示例链：向朋友发送消息-停止所有媒体-降低亮度并开始录音。

Siri Shortcut 2.1版是自iOS12以来的第一个主要更新。 但是，今天的应用仍然吸引了许多iOS用户。
