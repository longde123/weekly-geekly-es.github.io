适用于iOS Beta的3CX v16 Update 1、3CX应用程序和新版本的3CX Call Flow Designer

介绍最新的3CX产品评论。 会有很多有趣的事情-不要切换！

3CX v16更新1


我们最近发布了下一个3CX v16 Update1。此更新包括新的聊天功能和3CX Live Chat＆Talk网站的更新的通信小部件。 更新1还引入了新的呼叫流程服务，该服务向PBX添加了呼叫控制脚本接口。 脚本“引擎”与Call Flow Designer开发环境一起使用，并允许您创建任何复杂程度的呼叫处理脚本。

在Web客户端中更新了聊天




现在，更新的聊天功能使您可以灵活地管理对话。 此外，它与3CX Live Chat＆Talk通讯小部件可以很好地交互。

  • 站点访问者可以与3CX操作员队列开始聊天。 这将创建一个聊天组，其中包括所有队列操作员和此访客。
  • 将来，队列操作员可以将群聊切换为自己，并继续与访问者进行个人交流。
  • 另外，如有需要，话务员可以将聊天内容转移给其他话务员或PBX普通用户。
  • 要卸载Web客户端界面，可以将所选对话转移到存档中（但是不会将其删除）。
  • 现在，不同类型的聊天（来自站点，群组等）具有不同的图标，以便于识别。
  • 现在，您可以通过单击聊天窗口中的电子邮件，快速向站点访问者发送电子邮件。

新功能在“聊天指南”和视频 中有更详细的描述。

更新了3CX Live Chat＆Talk小部件




更新的3CX Live Chat＆Talk小部件提供了界面的其他自定义功能，并与在WordPress CMS和其他技术上建立的网站增强了集成。

  • 设置聊天窗口图标-您可以为聊天窗口标题设置合适的图像。 例如，这可能是您的公司徽标。
  • 设置操作员图标-您还可以设置聊天操作员图标，例如他的照片。
  • 窗口小部件的位置-“位置”参数确定窗口小部件在网页上的位置-右下（默认）或左下。
  • 移动浏览器中的视图是一个很小但重要的补充。 现在，从移动设备访问站点时，默认情况下，聊天窗口将最小化。
  • 聊天弹出窗口-在3CX Live Chat＆Talk小部件的3CX v16 Update 1窗口中，“弹出窗口”在一个单独的窗口中，使访问者可以在站点中自由移动，但同时可以随时与聊天联系。

脚本界面呼叫流程服务


3CX v16 Update 1引入了新的Call Flow Apps Service脚本接口。 它支持3CX语音应用新标准。 但是， 可以在新版本的Call Flow Designer中对现有应用程序进行转换或修改 （请参见下文）。 呼叫流程应用服务的体系结构已在今天完成。 该应用程序服务器在Debian / Raspbian Linux和Windows的3CX上运行。

适用于iOS Beta的新3CX应用


我们很久没有更新我们专有的3CX iOS应用了。 一些用户甚至抱怨文件传输不正确。 但是在下一个更新中，所有问题都已修复！ 这次，重点是集成聊天功能。 现在，移动应用程序中的聊天实际上并不逊于3CX Web客户端中的聊天。



该应用程序可以创建群聊并为其分配名称。 还添加了聊天存档。 要归档对话，请在其上向左滑动（您可以用相同的方式从归档中恢复对话）。

同样在该应用程序中，通过3CX实时聊天和对话通信小部件与网站的访问者进行了聊天。 现在，它是如何实现的：

  • 向左滑动即可聊天，以发现其他选项：自己动手，转移，结束和删除。
  • 该站点中的聊天图标与常规聊天的图标不同，因此您可以轻松区分它们。
  • 来自网站消息的PUSH通知会显示访问者的姓名和消息内容。
  • 为方便起见，发送到座席队列的聊天包含队列的名称。

新版本3CX呼叫流程设计器


就在本周，我们发布了3CX CFD语音应用开发环境的新版本。 它引入了新的C＃组件，完善的用户界面，改进的错误处理以及开发环境的自动更新。 需要新的CFD才能为3CX v16 Update 1和更高版本创建新型的语音应用程序。


更新的开发环境（IDE）CFD为开发人员提供了其他工具：

  • 新组件执行C＃文件和代码。 它们替代了旧版的“启动外部脚本”组件。 组件可以直接从CFD应用程序运行C＃代码文件或内联代码段。
  • 新的“设置扩展状态”组件实现了CFD应用程序中扩展参数的简单设置。
  • 改进的错误处理。 新的表达式编辑器还会检查这些值，并在编译阶段检测错误。

除了在开发过程方面进行了改进之外，我们还添加了一些功能来提高应用程序本身的可用性：

  • 自动更新应用程序。 CFD现在会自动检查新版本并在发布更新后立即安装更新。
  • 新的菜单项“项目另存为”允许您使用新名称或在其他位置保存CFD项目。
  • 新的上下文菜单“打开音频文件夹”，用于使用音频文件的组件。 它将打开资源管理器，以方便地浏览项目的音频文件夹。
  • 方便显示禁用组件。 它们现在显示为灰色，以不同于CFD的活动组件。

新的CFD版本涉及3CX V16 Update 1的使用。 下载CFD并从《 Call Flow Designer安装指南》中进行安装 。
