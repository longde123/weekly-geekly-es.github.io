Python dalam Visual Studio Code - rilis Juni

Kami senang mengumumkan bahwa ekstensi Python untuk Visual Studio Code dari Juni 2019 sudah tersedia. Anda dapat mengunduh ekstensi Python dari Marketplace atau menginstalnya langsung dari galeri ekstensi di Visual Studio Code. Jika Anda sudah menginstal ekstensi Python, Anda juga bisa mendapatkan pembaruan terbaru hanya dengan me-restart Visual Studio Code. Untuk mempelajari lebih lanjut tentang dukungan Python di Visual Studio Code, lihat dokumentasi .

Dalam rilis ini, kami membuat perbaikan yang terdaftar di changelog kami, menyelesaikan total 70 masalah, termasuk yang terkait dengan penampil grafik dengan jendela Python Interactive dan pengujian paralel dengan pytest. Baca tentang semua perubahan di bawah kucing.



Plot Viewer dengan Jendela Python


Grafik biasanya digunakan untuk memvisualisasikan data. Salah satu fitur paling populer dari jendela interaktif Python adalah kemampuan untuk melihat lebih dalam pada grafik yang dibuat, misalnya, penskalaan, panning dan ekspor mereka. Pembaruan Juni mencakup penampil yang sama sekali baru yang dapat digunakan untuk bekerja dengan gambar bagan apa pun, seperti dari perpustakaan matplotlib.

Anda dapat membuka penampil dengan mengklik dua kali pada grafik atau mengklik tombol “maksimalkan gambar”, yang ditampilkan ketika Anda mengarahkan kursor ke gambar grafik di jendela interaktif Python:



Dengan menggunakan penampil grafik, Anda dapat memindahkan grafik, memperbesar dan memperkecil, beralih di antara grafik dalam sesi saat ini dan mengekspornya ke PDF, SVG atau PNG.

Tes paralel dengan pytest


Kami telah membuat peningkatan dalam keandalan statistik yang ditampilkan untuk menjalankan tes, khususnya, untuk menjalankan tes secara paralel dengan pytest.

Anda dapat menjalankan tes secara paralel dengan pytest dengan menginstal paket pytest-xdist dan menambahkan "-n <jumlah prosesor>" ke file konfigurasi. Misalnya, untuk 4 prosesor, Anda dapat membuat file pytest.ini di folder proyek dan menambahkan konten berikut ke dalamnya: 

[pytest] addopts=-n4

Sekarang, ketika Anda menjalankan dan men-debug tes, mereka akan berjalan secara paralel.



Anda dapat merujuk ke dokumentasi kami untuk mempelajari lebih lanjut tentang menguji dukungan di ekstensi Python.

Indent pada start selektif di terminal


Dalam repositori GitHub kami , fungsi VS Code Python sangat populer , yang menghapus lekukan umum dari kode yang dipilih sebelum mengirimkannya ke terminal ketika Run Selection / Line dalam perintah Terminal Python dijalankan. Dimulai dengan rilis ini, perintah akan mengirim kode terminal dengan lekukan umum yang dihapus, berdasarkan baris non-kosong pertama dari kode yang dipilih.
