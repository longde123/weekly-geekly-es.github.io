Entrando no DevOps do Azure usando suas credenciais do GitHub

Em toda a Microsoft, estamos focando em capacitar os desenvolvedores a criar aplicativos melhores, mais rapidamente. Uma maneira de conseguir isso é fornecendo uma gama de produtos e serviços que cobrem todas as etapas do ciclo de vida de desenvolvimento de software. Isso inclui ferramentas de IDEs e DevOps, plataformas de aplicativos e dados na nuvem, sistemas operacionais, soluções de Inteligência Artificial e IoT e muito mais. Tudo isso é centrado em desenvolvedores, tanto como indivíduos trabalhando em equipes e organizações, como como membros de comunidades de desenvolvedores.


O GitHub é uma das maiores comunidades de desenvolvedores e, para milhões de desenvolvedores em todo o mundo, sua identidade no GitHub se tornou um aspecto crítico da sua vida digital. Reconhecendo isso, temos o prazer de anunciar melhorias que ajudarão os usuários do GitHub a começar mais facilmente com nossos serviços de desenvolvedor, incluindo Azure DevOps e Azure.



Suas credenciais do GitHub agora podem fazer login nos serviços da Microsoft


Hoje, estamos permitindo que os desenvolvedores façam login com sua conta GitHub existente nos serviços online da Microsoft, em qualquer página de logon da Microsoft. Usando suas credenciais do GitHub, agora você pode entrar via OAuth em qualquer lugar que uma conta pessoal da Microsoft, incluindo Azure DevOps e Azure.


Você verá a opção de fazer login com sua conta do GitHub clicando em "Entrar com o GitHub".


Após entrar no GitHub e autorizar o aplicativo da Microsoft, você receberá uma nova conta da Microsoft vinculada à sua identidade do GitHub. Durante esse processo, você também poderá vinculá-lo a uma conta existente da Microsoft, se já tiver uma.


Entrar no Azure DevOps


O Azure DevOps oferece um conjunto de serviços para desenvolvedores para ajudá-los a planejar, criar e enviar qualquer aplicativo. Com o suporte à autenticação GitHub, estamos facilitando o início de serviços como Integração Contínua e Entrega Contínua (Pipelines do Azure); planejamento ágil (Azure Boards); e armazenamento para pacotes particulares, como módulos para NuGet, npm, PyPi etc. (Artefatos do Azure). O pacote Azure DevOps é gratuito para indivíduos e equipes pequenas de até cinco.


Para começar com o DevOps do Azure usando sua conta do GitHub, clique em "Iniciar gratuitamente usando o GitHub" na página DevOps do Azure .



Depois de concluir o processo de entrada, você será levado diretamente para a última organização do Azure DevOps que você visitou. Se você é novo no DevOps do Azure, acessará uma nova organização criada para você.


Acesse todos os serviços online da Microsoft


Além de acessar serviços de desenvolvedor, como Azure DevOps e Azure, sua conta do GitHub pode ser usada para acessar todos os serviços online da Microsoft, do Excel Online ao Xbox.


Ao se autenticar com esses serviços, você pode encontrar sua conta do GitHub depois de clicar em "Opções de login".



Nosso compromisso com sua privacidade


Quando você usa sua conta do GitHub pela primeira vez para entrar na Microsoft, o GitHub solicitará permissão para liberar as informações do seu perfil.


Se você concordar, o GitHub compartilhará os endereços de e-mail da sua conta do GitHub (públicos e privados), além de informações de perfil, como seu nome. Usaremos esses dados para verificar se você já tem uma conta conosco ou para criar uma nova conta, se não tiver. Conectar sua identidade do GitHub a uma da Microsoft não concede à Microsoft acesso aos seus repositórios no GitHub. Aplicativos como o DevOps do Azure ou o Visual Studio solicitarão acesso a seus repositórios separadamente se precisarem trabalhar com seu código, com o qual você precisará consentir separadamente.


Enquanto sua conta do GitHub é usada para fazer login na sua conta da Microsoft, elas ainda são entidades separadas - uma apenas usa a outra como método de login. As alterações feitas na sua conta do GitHub (como alterar a senha ou ativar a autenticação de dois fatores) não mudarão a sua conta da Microsoft e vice-versa. Você pode gerenciar a conexão entre as identidades do GitHub e da Microsoft na sua página de gerenciamento de contas , na guia Segurança.


Comece a explorar os DevOps do Azure agora


Vá para a página DevOps do Azure e clique em "Iniciar Grátis com GitHub" para começar.
