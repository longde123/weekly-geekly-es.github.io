VMmanager 6: Ein neues Tool zum Erstellen virtueller Maschinen aus dem ISP-System



Am 5. Februar haben wir den Beta-Test der Cloud-Version von VMmanager 6, dem neuen Virtualisierungspanel, eröffnet. Wir laden die Habr-Community (alle, die mit der Erstellung und Verwaltung virtueller Maschinen konfrontiert sind) ein, diese zu sehen und Ihr Feedback zu hinterlassen.

Die erste Version von VMmanager 6 ist Cloud. Es muss nicht heruntergeladen und auf dem Server installiert werden. Gleichzeitig werden virtuelle Maschinen, Betriebssysteme und Benutzerabbilder auf Ihren Servern gespeichert. VMmanager 6 On-Premise wird später veröffentlicht.

Sie können das Produkt jetzt testen. Gehen Sie dazu auf my.saasvm.com . Zum Testen benötigen Sie einen dedizierten Server mit einer weißen IP-Adresse und CentOS 7.

Kostenloser Server zum Testen

Wenn es keinen kostenlosen Server gibt, nutzen Sie den kostenlosen von unserem Partner Selectel. Sie können die Geräte des Anbieters zwei Tage lang nutzen. Um einen Server zu erhalten, schreiben Sie uns über my.ispsystem.com oder chatten Sie auf der Website .

Weitere Informationen zu VMmanager finden Sie in den Nachrichten auf unserer Website.
