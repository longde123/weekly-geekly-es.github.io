Steam在三年内推出了1,600多种与Linux兼容的游戏



对于对游戏行业感兴趣的人，以及对游戏玩家来说，Steam电子游戏发行商店的开发商Valve都非常忠于Linux平台，这已不是什么秘密。该公司的家伙认为，在运行这些操作系统的计算机上有足够的游戏空间。但是，就像其他主要开发人员在使用开源程序时一样，Valve的声明也不是空话。通过在Linux系统上推广游戏（以及间接在Linux系统本身上推广）的课程，该公司在三年内实现了Steam商店中兼容的1600多种游戏并可在运行* unix系统的计算机上运行。

朝这个方向迈出的主要步骤之一是Valve加入Linux Foundation非营利组织，使Steam能够与unix系统一起工作并更新自己的项目：Valve现在不仅发行游戏，而且还开发了诸如Team Fortress之类的世界著名游戏2，Dota 2和CS：GO。所有这些游戏已经可以在Linux下成功运行。对于Valve自己的控制台，SteamMachine也被开发出来了，它是基于Linux内核的操作系统。

过去三年，Valve积极投资于Linux开发项目，该项目由Collabora LunarG和先进的跨平台多媒体库SDL2赞助。

Valve对新的用于2D和3D图形的Vulkan API寄予厚望。它最初在2015年GDC上展示。根据开发人员的说法，Vulkan将成为DX12的重要竞争对手，因为它将具备DX12加Vulkan不能局限于一个平台的所有功能。

在8月的SIGGRAPH 2015大会上，Valve的Dan Ginsburg 对 Vulkan API 进行了评论：

如果您不打算今年在DirectX 12上发布游戏，则没有使用DX12的严肃理由。事实是Vulkan可以在Windows 10上完成相同的工作，而不仅限于一个平台。Vulkan将竭尽所能。不仅是Windows 10，还包括Windows 7，Windows 8和Linux。

主要问题是需要说服开发人员使用Vulkan而不是DX12。如果新的API得以普及，这将是游戏产业从Windows转向跨平台的又一步。
