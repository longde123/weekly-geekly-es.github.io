Windows 10即使通过限制连接也会下载一些更新



根据初步数据，随着Windows 10 Creators Update最终版本的发布，用户将不再能够使用自定义限制连接来规避强制安装更新。 此外，操作系统现在不仅能够通过自定义方式，而且默认情况下还可以通过限制连接（例如，移动Internet连接）接收重要更新。

Windows Insider计划的参与者之一Richard Hay得出了这个结论，他研究了上周初由Microsoft提出的Windows 10 Build 15058更新的初始版本的汇编。 在熟悉此构建过程中，他发现Windows Update与先前版本的设置措辞不同。


版本14393.953中更新中心设置中的文本


程序集15058中更新中心设置中的文本

内部程序的任何成员都可以相信这一点：如果转到Windows设置>更新和安全性> Windows Update，则在标题下可以看到更新参数，现在可以通过限制连接下载该参数。

这两个版本的Windows 10之间的主要区别是第二句话，内容如下：“在这种情况下，我们将仅自动下载维护Windows正常运行所必需的那些更新。”

Windows 10中的可配置限制连接是一种广泛使用的方法，用于控制自动下载和绕过操作系统主版本中所需更新的安装。 现在，使用蜂窝连接进行数据传输或任何其他默认收费频道的用户在下载更新并用尽所有其余流量时将无法执行任何操作。

当Hay看到更新中心的修订说明时，他决定找出那些相同的更新意味着什么，以确保操作系统的正常运行。 他与Microsoft支持中心联系，以更好地了解属于此类别的更新类型。

发言人的回应并没有澄清这种情况：“我们不打算通过限位连接发送大型更新，但将来会在必要时使用它们进行重大更正。” 由于没有有关消耗流量的详细信息，因此Haye要求对此进行仔细研究。

Windows 10 Creators Update将于4月11日发布。 为初始版本编写的文本可能仍会更改。 另一方面，大的最终更新在发布前不久会发生巨大变化的可能性非常小。 毕竟，工作不仅限于编辑文本：开发人员设置了基本的网络功能，以允许在收费流量的通道上安装某些更新。

如果我们谈论在Creators Update中设置限制连接的细微差别的变化，则可以在网络连接设置页面上将有线连接标记为限制。 除此之外，用户将可以在任何方便的时间延迟已下载更新的安装，或者使用“睡眠”模式将其自动传输三天。

此外，新程序集将进行更重要的更新。 最有趣的问题之一是与多任务处理有关：借助紧凑叠加功能，用户可以切换到“画中画”模式。 使用它，您可以在一个小窗口中运行应用程序，该窗口将显示在其他打开的选项卡的顶部。 这不仅对于粉丝在执行单调任务的同时观看视频是有用的，而且对于例如即时通讯程序的启动也将是有用的。

另一个有用的功能是，当断开通过蓝牙连接的智能手机时，计算机会自动锁定。 如果您离开蓝牙范围，则30秒钟后，计算机将进入锁定模式。

此外，将支持增强和虚拟现实设备，以及与其他3D功能的集成，例如与其他三维模型创建者共享内容。 微软游戏玩家承诺将优化对4K分辨率游戏的支持以及视频广播的发布。 游戏模式将通过关闭不必要的后台进程和其他技巧来在游戏过程中优化计算机，但是，显然，您不应期望此功能带来更多好处。

Windows 10 Creators Update也将更加关注安全性。 操作系统安全中心将通过高级威胁检测功能和分析工具得到增强。

Windows 10用户与所有更新的进一步了解将于4月11日进行。
