Pemindaian otak dapat mengganti kata sandi suatu hari

Kata sandi teks sulit diingat dan mudah dipecahkan, sehingga banyak produsen bekerja dengan biometrik - sensor sidik jari pada laptop ThinkPad dan ponsel Apple, identifikasi mata pada ponsel cerdas ZTE. Sidik jari tidak perlu diingat, dan lebih sulit untuk memalsukannya - meskipun, tentu saja, itu tidak mustahil.

Para ilmuwan telah mengusulkan cara baru : ensefalogram.



Teknologi Brainprint baru dari para ilmuwan di Binghamton University melibatkan penggunaan satu set elektroda untuk ensefalogram. Sistem menunjukkan seseorang gambar 500 dan merekam respon otak untuk mereka. Teknologi dengan akurasi mutlak menebak satu dari tiga puluh orang. Para ilmuwan tidak mengusulkan pengiriman awal laptop dengan penutup untuk ensefalogram. Teknologi Brainprint, jika terus menunjukkan keefektifannya, dapat digunakan di fasilitas dengan tingkat keamanan tinggi, termasuk bank dan pangkalan militer.

Gambar dipilih dengan tujuan memperoleh respons emosional tertentu. Ini adalah foto sepotong pizza, perahu, sushi, aktris Anne Hathaway, berbagai kata. Para peneliti mencatat bahwa mereka memilih gambar sedemikian rupa sehingga setiap orang akan merespons secara berbeda. Ini meningkatkan akurasi.

Untuk menguji keamanan teknologi, penciptanya mencoba menipu Brainprint. Salah satu metode termasuk aksi lampu berkedip untuk "menyinkronkan" kerja otak dua subjek eksperimental. Metode ini tidak berhasil - tidak berhasil memecahkan Brainprint.

Gagasan utama dari teknologi ini adalah untuk mengenali seseorang dengan pemikirannya - aktivitas otak tidak terlihat oleh siapa pun. Adalah penting bahwa pengguna tidak harus memikirkan identifikasi, yaitu teknologi yang dapat digunakan sehingga pengguna tidak mengetahuinya, tulis pengembang.
