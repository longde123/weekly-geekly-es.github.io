NASA telah membuka akses gratis ke perangkat lunaknya

NASA telah membuka akses gratis ke katalog perangkat lunak 2017-2018 . Sebagaimana dinyatakan dalam pengumuman resmi , semua produk dan algoritme perangkat lunak dapat digunakan sepenuhnya gratis.



Katalog berisi sejumlah besar berbagai jenis perangkat lunak.



Meskipun akses terbuka, ada batasan lisensi. Misalnya, beberapa perangkat lunak hanya tersedia di AS.



Pada saat yang sama, beberapa sumber terbuka tersedia di github. Misalnya, proyek alat penelusuran dan pengunduhan pencitraan citra satelit Worldview dirancang untuk secara interaktif melihat dan mengunduh gambar dari satelit NASA.



Selain perangkat lunak, situs web NASA berisi set model 3D, gambar, dan tekstur yang dapat digunakan secara gratis untuk keperluan pribadi atau pendidikan.
