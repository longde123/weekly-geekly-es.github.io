Mikrotik mAP lite

Segera setelah saya melihat, saya memperoleh model miniatur baru Mikrotik RouterBoard lite tentang yang ingin saya bagikan.


Saya sudah punya Mikrotik RouterBoard mAP 2n , tapi pETA lite sangat terkesan dengan dimensinya: 48x49x11mm
MAP gambar lite









Spesifikasi dan pengujian perangkat dapat ditemukan di situs web produsen. Secara singkat dibandingkan dengan peta 2n:
  • Frekuensi prosesor 650 MHz yang lebih dari mAP 2n - 400 MHz
  • RAM 64 MB seperti mAP 2n
  • FLASH 16 MB seperti mAP 2n
  • Satu port Ethernet (10/100) - mAP 2n memiliki dua
  • Wi-Fi QCA9533 802.11b / g / n - di mAP 2n AR9331

tidak lebih dari alami, kecuali untuk ukuran 48x49x11mm dibandingkan 68x68x19mm untuk pemetaan 2n.
Di situs itu, harga perangkat yang disarankan adalah $ 25, saya berhasil membelinya dengan pengiriman untuk 2.700 rubel.
Kejutan pertama adalah ukuran kotak (peta 2n yang sama muncul dalam kotak panjang)
Packing mAP lite dan mAP 2n



Lite mAP dilengkapi dengan catu daya (5V - 1A), dudukan logam, kaki karet.
Paket bundel




PETA lite sendiri









Hal pertama yang saya ambil adalah bayi itu. Ternyata tidak mudah dibuka. Pengencang snap yang sangat cepat.
Di dalam








Di dalam perangkat terdapat magnet, yang memungkinkan Anda memasang router ke permukaan logam apa pun, tetapi yang paling penting ke kotak yang menyertainya.
Gunung




Memeriksa kekuatan atas PoE (802.3af / at (+ 4.5 / -7.8 atau + 3.6 / -1.2))
PoE


Pada pETA lite ada firmware baru. RouterOS standar dan reguler. Semuanya bekerja tanpa masalah. Hangat di tempat kerja (meskipun bisa jika Anda mengarahkan lalu lintas yang lebih besar melewatinya, gambar akan berubah).
Perangkat ini dirakit di majelis berikutnya.
Majelis





Intinya adalah sebagai berikut. Saya menghubungkan router dan modem dengan 4G dan Wi-Fi - Huawei E8372 (MTS 8211F) ke bank daya (Romoss solo 3 PH30 6000mAh). Router menempel ke Ethernet, jika tidak terhubung, maka ke jaringan Wi-Fi tertentu, jika tidak menemukannya, maka ke modem. Selanjutnya, L2TP / IPsec VPN ke server rumah ke SoftEther. Web-Proxy yang dikonfigurasi untuk mengarahkan ulang ke proxy rumah. Sekarang saya akan membaca skrip dan, pada akhirnya, saya akan menerbitkannya di artikel baru, jika itu menarik.

Daftar Keinginan:
  • USB Port terpisah. Dilihat dari kasusnya, produsen juga memikirkannya. Seseorang dapat menghubungkan modem secara langsung.
  • 802.11ac
  • Bip - Anda benar-benar membutuhkannya
  • Kontrol LED seperti pada mAP 2n

Port Ethernet lain dan daya ke perangkat lain melalui itu, tetapi ini sudah akan mengarah pada peningkatan ukuran, yang saya tidak mau.
Ada satu hal yang saya dapat.
