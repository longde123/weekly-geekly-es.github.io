Ruang Server atau Kabinet Membatalkan Kebisingan Aktif

Halo Ini adalah posting pertama saya, jadi tolong jangan menilai dengan ketat.


Saya adalah pemilik perusahaan kecil, dalam kombinasi saya mengkonfigurasi semua server saya. Saya memiliki arsitektur jaringan yang cukup sederhana dengan empat 19 "server fisik yang baru saja kehilangan colocation di server lessor.

Muncul pertanyaan tentang mengalokasikan ruang server dari area kantor total dengan segel tim atau memasang kabinet diam, prinsip yang telah dijelaskan pada Habré.


Selanjutnya, saya akan mencoba menentukan sendiri: untuk membangun ruang server paling sederhana atau memasang kabinet diam.



Dari perhitungan saya akan membuang pekerjaan yang sama untuk kedua kasus, seperti: catu daya, switching dan pekerjaan identik lainnya.


Perhitungan biaya kepemilikan server

Input data untuk perhitungan adalah:

- area server minimum sesuai dengan standar ANSI / TIA / EIA-569-B harus setidaknya 14 sq.m., tetapi standar asing bukan keputusan untuk kita. Kami mengambil dasar 8 meter persegi. m sehingga lemari dapat didekati dari kedua sisi;
- tarif sewa per sq.m. - 1000 rubel / bulan;
- pekerjaan konstruksi minimal untuk pembangunan server: partisi drywall, isolasi suara, pintu lebar, alarm kebakaran, pencahayaan, perubahan langit-langit;
- Sistem pendingin paling sederhana untuk pendingin udara rumah tangga tanpa redundansi;
- kabinet server CMO 42U yang terjangkau dengan modul ventilasi;
- masa kerja adalah 3 tahun, karena di negara kita pemimpin mana pun dari bidang usaha kecil dan menengah tidak melihat lebih jauh daripada selama 3 tahun.


Perhitungan:

- biaya kepemilikan selama 3 tahun - 1000 rubel / sq.m. * 8sq.m. * 36bulan. = 288.000 rubel.
- Biaya membangun server - 50.000 rubel.
- Biaya pemasangan AC - 60 000 rubel.
- biaya kabinet CMO dengan ventilasi adalah 66.500 rubel.

TOTAL selama 3 tahun: 464 500 rubel.


Perhitungan biaya kepemilikan kabinet diam

Input data untuk menghitung biaya kepemilikan kabinet Silentium aktif menggunakan teknologi pengurangan kebisingan aktif:

- area yang ditempati oleh kabinet adalah 1 sq.m.;
- tarif sewa per sq.m. - 1000 rubel / bulan;
- AC di kantor sudah terpasang;
- masa kerja - 3 tahun.


Perhitungan:

- Biaya kepemilikan selama 3 tahun adalah 1000 rubel / sq.m. bulan * 1sq.m * 36 bulan = 36 000 gosok.
- Biaya kabinet AcoustiRACK VIST khas diam adalah 420.000 rubel.

TOTAL selama 3 tahun: 456.000 rubel.
