Análise do Android Remix OS para computadores pessoais

Mais recentemente, em 14 de janeiro de 2016, foi lançado um sistema operacional para um PC Android chamado Remix OS da Jide . A empresa foi fundada por ex-funcionários do Google. O sistema é gratuito e existe uma versão alfa disponível gratuitamente para desenvolvedores.

Na página do produto, você pode encontrar uma descrição do novo sistema operacional, requisitos do sistema e fazer o download do kit de distribuição com um utilitário para gravar em uma unidade flash. Há também um link para os serviços do Google GMSInstaller.apk , que não são pré-instalados no sistema. Sem serviços, você não terá um Play Market e simplesmente não poderá baixar aplicativos.

Portanto, para experimentar o novo sistema operacional, usei o netbook ASUS X102BA de baixa potência com uma unidade flash Transcend USB 3.0 de 16GB.



O sistema se inscreveu para o flash em 2 minutos. Após reiniciar e selecionar a inicialização do BIOS a partir do flash, o processo de verificação da unidade quanto à velocidade de gravação e outras verificações foram executadas.

A primeira inicialização demora um pouco mais do que a seguinte, e o SO solicita o idioma do sistema. Você pode escolher entre inglês, chinês e outro asiático. Lemos o contrato, que é bastante volumoso, então não li, apenas rolei a página e cliquei em OK. Existe uma proposta para conectar uma rede Wi-Fi disponível.

Em seguida, nos encontramos na área de trabalho, cujo design é muito semelhante ao design do Windows 10, vale a pena dar uma olhada na barra de tarefas e na área de notificação para ver isso. O que o Android possui uma barra de notificação apareceu mais cedo do que no Windows 10, portanto, já está claro quem a emprestou de alguém.





Início minimalista, nada mais. Mais uma vez, os serviços do Google não estão pré-instalados. Por padrão na lista inicial:

  • Navegador padrão do Google
  • Calculadora
  • Camera
  • Relógio
  • Transferências
  • Gerenciador de arquivos
  • MX player com anúncios, provavelmente você precisará comprá-lo ou removê-lo
  • Música
  • Configurações
  • Widgets



O vídeo é reproduzido pelo MX player normalmente.



Também com o som, tudo está em ordem. O teste é um programa para criar música chamada Caustic, disponível no Android e no Windows.



O sistema operacional é multi-janela, não é surpreendente, pois Ele foi projetado para PC, seria tolice se fosse diferente. Em algumas conchas do Android, isso tem sido implementado em tablets.



O navegador funciona, tanto o padrão quanto o Chrome. Eu não uso outro, acho que também não haverá problemas com os outros.



Talvez o suficiente, o gerenciador de arquivos e outros aplicativos funcionem por padrão. Dificuldades podem surgir com aplicativos instalados, por exemplo, com escritório ou jogos e com outros. Os documentos, folhas e outros editores de documentos do Google podem ser reduzidos por si mesmos ou estúpidos. Talvez produtos de outros fabricantes funcionem melhor, mas "como o tipo de aplicativos do Google" funcionará mal. Obviamente, não são más aplicações, mas apenas um sistema úmido.

Existem aplicativos, incluindo jogos para Android, que funcionam ou baixam conteúdo via WI-FI, ou seja, "eles não sabem" sobre o lançamento de um novo sistema operacional baseado no Android no qual agora você pode inserir um fio de rede. Assim, de fato, os dados são transmitidos pelos mesmos protocolos simplesmente em ambientes diferentes. Agora, cabe aos desenvolvedores que removerem essa limitação para igualar os dois tipos de conexão no nível do software.

A conexão de rede através do fio está ativa, mas todos os aplicativos são amigáveis? Por exemplo, o jogo Asphalt 8 pede para ativar o Wi-Fi, não deseja baixar conteúdo por cabo. Quantos programas sombrios? Esta é uma pergunta difícil. A propósito, o Asphalt 8 não funcionou para mim, tudo terminou na imagem de carregamento do estúdio GAMELOFT.



Foram dois jogos no total, asfalto 8 e necessidade de velocidade sem limites. O último agradou que funcionou, e funcionou normalmente. Não mergulhei nas configurações, mas não acho que exista uma configuração para o teclado. Felizmente, há uma tela sensível ao toque no meu netbook e ela acabou sendo reproduzida clicando na tela com os dedos.





Verificamos as configurações e sobre. Vemos o que podemos configurar neste sistema operacional e em que ele é construído. Sua base é o Android 5.1.1.





Os resultados do benchmark Antutu não são animadores, embora seja difícil dizer quão adequada é essa classificação de desempenho. É improvável que o programa seja projetado para PC. Talvez o computador esteja fraco ou os drivers estejam úmidos, depois de atualizar os drivers e do lançamento de uma versão estável do sistema, ficará claro.



Agora vamos falar sobre o mal. Os aplicativos costumam travar em uma reinicialização rápida ou em uma tela preta. Eu acho que no primeiro caso, apenas algo deu errado, não de acordo com o esquema. Eu imagino isso como uma ramificação incorreta no código do sistema operacional, ou seja, se o sistema "não entender" o que aconteceu, ele não terá uma opção para esse desenvolvimento de eventos - ele será reiniciado rapidamente. A tela preta provavelmente é causada por uma reinicialização do driver de vídeo. É claro que esse é apenas o meu raciocínio; de fato, o motivo pode ser diferente.

Concluindo, gostaria de observar que este sistema operacional tem futuro. Para os proprietários de PCs antigos ou de baixo consumo de energia, isso será um presente gratuito, você poderá jogar jogos Android, se não houver recursos suficientes para os exigentes jogos do Windows. Haverá uma oportunidade de atualizar um pedaço de ferro com o novo software. Com os drivers já na versão alfa, nem tudo é tão ruim, a rede, o som, o toque, o mouse funcionam e você não precisa procurar nada, tudo se apega a si mesmo. Se o Remix OS ganhar popularidade, obviamente afetará o mercado de trabalho em TI, novas vagas aparecerão para aqueles que já estão escrevendo sob o "balde" ou apenas começando. Obviamente, sempre há contras, por exemplo, nada substituirá o Microsoft Office no Windows; ele provavelmente pode ser entregue usando um fio parecido com uma muleta como o vinho, mas esse tipo de "entretenimento" não é mais para o usuário comum.

Se você deseja contribuir para o desenvolvimento do sistema ou apenas discutir este produto, participe do grupo do Google - Remix OS for PC.
