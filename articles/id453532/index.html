Xamarin API Docs: Buka Sumber dan Tersedia Sekarang

Hari ini, kami dengan senang hati mengumumkan rilis semua Dokumentasi Xamarin API sebagai Sumber Terbuka! Dokumentasi API mendorong pengalaman IntelliSense sambil menjadi salah satu cara terbaik untuk membantu pengembang mencapai tujuan mereka.


Xamarin API Documents


Selain itu, kami juga telah memindahkan hosting berikut dari rumah lama Xamarin mereka ke docs.microsoft.com:


  • Xamarin. Bentuk
  • Xamarin. Android
  • Xamarin.iOS & Xamarin.Mac
  • Skiasharp

Situs Dokumentasi Baru


Situs docs.microsoft.com memberikan pengalaman presentasi, pencarian, penelusuran, lokalisasi, dan versi yang sama dengan yang digunakan di platform Microsoft .NET. Sekarang, Anda dapat memfilter berdasarkan platform dan versi, semuanya sambil mendapatkan hasil pencarian yang cepat dan akurat untuk ruang nama, nama kelas, dan tanda tangan anggota.



Xamarin API Documents sekarang Open Source

Perwakilan Github


Ada repositori Github terpisah untuk:


  • Xamarin.Forms API Documents
  • Xamarin.Android API Documents
  • API Damar Xamarin untuk iOS dan Mac
  • SkiaSharp API Documents

Dokumentasi API dirilis di bawah Lisensi Creative Commons Attribution 4.0 dan semua kode sumber terkait di bawah Lisensi MIT . Repo Github tunduk pada Kode Etik Open Source Microsoft . Halaman dokumentasi Xamarin.Android API adalah modifikasi berdasarkan pada karya yang dibuat dan dibagikan oleh Android Open Source Project dan digunakan sesuai dengan istilah yang dijelaskan dalam Lisensi Atribusi Creative Commons 2.5 .


Dibangun dengan Mono


Dokumentasi Xamarin API dibangun dengan open-source toolchain mdoc proyek mono. Toolchain ini mengubah komentar kode sebaris menjadi direktori khusus file dokumentasi XML. Ketika Anda memperbarui rakitan sumber Anda, kelas dan anggota yang ditambahkan / dihapus disinkronkan ke repo dokumentasi. Dengan alat ini, Anda dapat memisahkan masalah teknis dan dokumentasi dengan rapi.


Berkontribusi


Bantu berkontribusi ke dokumentasi Xamarin API, atau gunakan mdoc toolchain di proyek Anda sendiri hari ini! Edit XML secara langsung atau gunakan alat DocWriter untuk menghindari kompleksitas format XML .
