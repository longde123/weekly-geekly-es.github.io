4xiDraw: um plotter de caneta caseiro no Arduino



Os plotadores de caneta e caneta e caneta já foram extremamente populares. Com o tempo, sua produção começou a diminuir. Mas você pode usar esses sistemas em vários campos, incluindo corte e costura, engenharia, desenho, etc. Você pode encontrar um plotter de caneta no mercado, mas é mais interessante fazer isso sozinho, certo?

E um usuário chamado Miguel Sanchez decidiu fazer um plotter por conta própria. Como plataforma de controle, ele escolheu o Arduino Uno. O sistema também usa motores de passo NEMA 17 e um servoconversor auxiliar para levantar e abaixar a alavanca.

Além disso, tubos de metal, correias e várias peças impressas em uma impressora 3D são usadas. Todo esse sistema é bastante simples e, com uma impressora 3D, não é particularmente difícil. Curiosamente, Miguel decidiu inicialmente usar o corte a laser para criar as peças necessárias, mas depois decidiu trabalhar com uma impressora 3D.

Sanchez decidiu criar sua própria plotadora, inspirada no modelo AxiDraw, desenvolvido pela Evil Mad Scientist .



Aqui está o primeiro modelo da base da plotadora, criado a partir de peças cortadas por um laser:



Depois que o desenvolvedor decidiu finalizar o sistema, use também uma impressora 3D para criar peças para sua plotadora.



Todos os modelos necessários para a impressão de peças são enviados pelo usuário ao público .

Para enviar arquivos para "imprimir", o artesão utilizou o programa Universal Serial Sender . Este é um programa java que lida perfeitamente com sua tarefa. E aqui está o resultado final do trabalho:
