Consola BLE: una forma completamente nueva de interactuar con dispositivos BLE

La primera aplicación de consola del mundo en Microsoft Store.



La idea de este único (no tengo miedo de esta palabra) surgió, literalmente, espontáneamente. No voy a "extender un árbol sobre un árbol": si usted es un desarrollador que se ocupa de dispositivos BLE (BlueTooth Low Energy), intente encontrar un análogo funcional de esta aplicación en código abierto o aplicaciones pagas, sin mencionar las tiendas especializadas para ciertos sistemas operativos. Admito completamente la posibilidad de que bajo Linux (bueno, o una de las múltiples implementaciones) haya algo similar, pero no lo he visto. Sí, si viste algo similar publicado anteriormente, avísame.

¡Me complace presentarle a la comunidad BLEConsole , la primera aplicación del mundo para profesionales (y aficionados) para trabajar con dispositivos BLE desde la consola estándar de Windows 10!

Probablemente, algunos "estetas" (que no han estropeado una sola línea de código para el público en general en la vida) encontrarán el código fuente de esta aplicación "terrible"; no me molesta en absoluto :) Lo principal es que esta aplicación se vuelve útil y familiar para usted. Y si alguien adquiere algún conocimiento del código fuente, ¡mucho mejor!

Y después de eso: BLEConsole se convirtió en la primera aplicación de consola en la historia de Windows 10 publicada ... en Microsoft Store ;) Esto se hizo "solo una prueba de concepto" en lugar de perseguir un objetivo específico, pero si querías comprarme cerveza, de nada ! ;) Naturalmente, la versión "de prueba" no es diferente de la "completa", y más aún, del nabo autoensamblado en github;)
[ACTUALIZACIÓN] Los comentarios mencionaron la aplicación de consola de Ubuntu (de los creadores de este sistema operativo), que estaba por delante de la consola BLE. Pero, sin embargo, "BLE Console" sigue siendo la única aplicación de consola para que Windows 10 funcione con dispositivos BLE.
