Hacer un temporizador de cuenta regresiva en las tablas de google

Si no tiene nada que hacer mientras espera un nuevo proyecto, y solo tiene a mano las tablas de Google, puede hacer un temporizador de cuenta regresiva, por ejemplo, para averiguar cuánto tiempo libre le queda.

Para hacer un temporizador, solo use un par de funciones:

TDAT (AHORA) : muestra la fecha y hora actuales en el formato de "fecha".
DATEDIF: cuenta el número de días, meses y años restantes entre dos fechas. Por defecto, el día actual se tiene en cuenta.
DÍAS : cuenta el número de días entre dos fechas.
HORA : devuelve el componente de hora en un momento determinado y también puede calcular la diferencia en horas entre dos fechas.
MINUTO : funciona de manera similar a la función HORA, solo por minutos.

Digamos que queremos saber cuántos días quedan hasta el cumpleaños del habr - 26 de mayo según Wikipedia.

Configuraciones iniciales


Para que el temporizador se actualice constantemente y muestre información relevante, es necesario cambiar la configuración de cálculo en la tabla.

En la tabla abierta, seleccione el elemento de menú "Archivo", vaya a "Configuración de tabla" y en la pestaña "Cálculos", seleccione "Actualizaciones de cálculos" - "Cuando cambia cada minuto".



Fórmulas de temporizador


En el ejemplo, hice un temporizador de dos tipos:

para mostrar los días, horas y minutos restantes,
también
para mostrar meses, días, horas y minutos.

La única diferencia es que para calcular el número de meses y días completos restantes es necesario usar la función RANDATE, mientras que la función DAYS es suficiente para mostrar solo el número de días.

También hay una SEGUNDA función que le permite mostrar segundos, pero debido al hecho de que el rango de tiempo del cálculo se limita a la lectura "una vez por minuto", el temporizador de segundos simplemente no tendrá tiempo para actualizarse.

Escribimos fórmulas para contar el tiempo.


El ejemplo usa celdas:

A2 : la fecha de finalización a partir de la cual contamos
B2 es la fecha actual, también puede usar la función AHORA.

Meses
=DATEDIF(B2;A2;"M")
Número de días excluyendo meses completos:
=DATEDIF(B2;A2;"MD")
Numero de dias en total
=DÍAS(A2;B2)
Reloj
=HORA(A2−B2)
Minutos
=MINUTO(A2−B2)

En la realización más simple, mostramos los valores del temporizador en celdas separadas:



Usando la función CONCATENAR, todos los datos se pueden mostrar en una celda.

Para este ejemplo, la fórmula tiene la forma

=CONCATENAR(DATEDIF(B2;A2;"M");"mes";":";DATEDIF(B2;A2;"MD");"días";":";Hora(A2−B2);"hora";":";Minuto(A2−B2);"min")



o contar solo días

=CONCATENAR(DÍAS(A2;B2);"días";":";Hora(A2−B2);"hora";":";Minuto(A2−B2);"min")





Opcionalmente, también puede instalar complementos en la tabla que enviarán notificaciones bajo ciertas condiciones, por ejemplo, cuando se alcanza un cierto número de días. Pero solo funcionan si la mesa está abierta en una ventana.
