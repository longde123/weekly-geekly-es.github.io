Cara menghilangkan bau menggunakan gadget modern



Bau tak sedap bisa membuat orang tidak seimbang. Dari mereka muncul lekas marah, kepala bisa sakit. Bau busuk tidak akan memungkinkan untuk berkonsentrasi pada pekerjaan. Dari bau di rumah, tamu-tamu kami bisa menilai kami. Karena itu, saya tertarik untuk menguji gadget FreshON.

Deskripsi




Ukuran. Perangkat ringkas ini karena ukurannya pas di rak apa pun di lemari es. Ini merupakan nilai tambah yang besar. Dari dimensi berikut nuansanya: "FreshON" tidak dirancang untuk menangani volume udara yang besar. Misalnya, sulit baginya untuk menutupi seluruh ruangan (baca hasil percobaan saya di bawah).



Nutrisi. Gadget ini cocok untuk penggunaan offline. Sudah berjalan selama lebih dari dua bulan pada tiga baterai AA. Atur - dan lupa.



Prinsip operasi. "FreshON" adalah ozonizer udara rumah tangga.

Alat. Gadget hanya memiliki dua tombol: aktifkan dan aktifkan mode. Ada juga dua mode operasi: tinggi dan rendah. Dalam mode tinggi, perangkat bekerja selama 30 menit, lalu - 1 menit setiap jam. Ini berjalan selama 20 menit dalam mode rendah dan kemudian selama 40 detik setiap jam. Seperti yang ditunjukkan oleh pengalaman, ini cukup untuk menetralisir bau. Meski ada nuansa, tapi soal mereka nanti.

Penggunaan berbeda


Jadi, setelah menerima "FreshON", saya mulai mengujinya dalam kondisi yang berbeda.
Kulkas.
Hal pertama yang saya pergi ke lemari es. Bukan berarti baunya kuat, tetapi karena perangkat diposisikan terutama sebagai penyegar untuk kulkas, taruh di sana.



Anda harus meletakkan gadget lebih tinggi. Saya telah menetapkan sebelumnya bahwa ozon lebih berat daripada udara, sehingga akan turun, menghancurkan bau dan bakteri. Jadi, di malam hari saya meletakkan perangkat di rak paling atas, nyalakan mode tinggi. "FreshON" mulai terdengar gemerisik dan mengedipkan lampu hijau. OK, kita lihat besok pagi.

Di pagi hari, saat membuka pintu kulkas, aku berharap mencium aroma ozon. Saya buka - saya tidak merasakan apa-apa. Apa? Gadget sedang berdiri, berkedip. Tidak ada bau. Hanya setelah sedetik kebingungan aku mengetahuinya: lagipula, aku seharusnya tidak merasakan apa-apa! Tidak ada bau susu asam, bawang putih dari rak bawah tidak berbau. Itu tidak berbau apa-apa. Umumnya. Saya pikir ini adalah hasil yang saya tunggu-tunggu. Yah, itu tidak akan berbau seperti aster. Perangkat menghilangkan bau. Peringkat saya 5 dari 5.

Saya memperpanjang percobaan selama satu hari. Selalu ada susu di lemari es kami. Kami membelinya setiap hari. Itu berdiri dalam paket terbuka, kadang-kadang rampasan. Dan selama dua hari itu rusak dalam hal apapun. Saya membeli lebih banyak susu dan satu paket akan bertahan lebih lama dari biasanya.

Suatu malam "FreshON" sudah bekerja. Saya memeriksa susu dalam sehari - semuanya beres. Setelah satu malam lagi - tidak diasamkan. Pada pagi ketiga, susu itu dimasukkan ke dalam makanan - saya memasak bubur di atasnya. OK, saya suka hasilnya. Saya tidak akan memiliki ilusi bahwa gadget akan selamanya memperpanjang kesegaran produk, tetapi untuk beberapa hari ini cukup realistis. Peringkat saya 5 dari 5.

Tempat Sampah
Kami beralih ke prosedur air, percobaan dalam kondisi lain. Saya memutuskan untuk bertahan hidup dengan sampah penuh (jangan ulangi! Ini berbahaya bagi kesehatan orang lain).



Jadi, setelah lemari es, sesuai dengan instruksinya, Anda perlu mendapatkan baterai dan membiarkannya dan alat memanas. Saya melakukannya, dan kemudian meletakkannya langsung di tutup tempat sampah. Di sampingnya, tentu saja, ada bau tidak sedap, dan sampah harus dibuang, tetapi demi percobaan saya siap mengorbankan kenyamanan.

Jadi, di malam hari saya pergi ke dapur dan mengendus: tidak ada bau yang tidak enak. Saya melihat ke alas di mana ember itu berdiri: tidak berbau ozon, "FreshON" bekerja. Ada bau yang tidak enak, tetapi tidak banyak. Sesuatu mengatakan kepada saya bahwa, jika bukan karena perangkat, situasinya akan berbeda.

Pada pagi hari, tidak ada yang berubah secara mendasar. Saya mengeluarkan sampah, tetapi memutuskan bahwa gadget ini bukan untuk kasus seperti itu. Dan sampah harus dibuang secara teratur. Peringkat saya adalah 4 dari 5.

Kamar mandi
Kami bergerak di sekitar apartemen. Saya meletakkan gadget di kamar mandi, atau lebih tepatnya di kamar mandi gabungan. Ada banyak aroma, dan FreshON memiliki sesuatu untuk dikerjakan.



Saya meninggalkan gadget selama sehari. Ini bekerja dalam mode tinggi. Tetapi hasilnya tidak sepenuhnya terlihat. Ini, tentu saja, mempengaruhi ukuran perangkat dan volume ruangan. Saya pikir pemurnian udara sedang berlangsung, tetapi ini tidak terlihat di beberapa meter kubik udara. Pintu terbuka terus-menerus, kelembaban berubah. Secara umum, dia berdiri dengan indah, tetapi tanpa hasil nyata. Peringkat saya 2 dari 5.

Lemari pakaian
Kami masuk ke kamar. Ada ruang tertutup: lemari, misalnya. Selama operasi, berbagai aroma dapat menumpuk di dalamnya. Tidak heran wanita membeli wewangian untuk lemari. "FreshON" dalam pengertian ini jauh lebih baik, karena aroma beberapa lavender tidak meresap ke pakaian Anda. Gadget, seperti yang kami definisikan di atas, tidak menambahkan bau sama sekali. Dia menghancurkan mereka.



Jadi, saya meletakkan perangkat di rak paling atas sehingga ozon turun (yah, Anda ingat). Sniffing - Saya harus ingat bau lemari. Saya menutup pintu dengan erat. Percobaan telah dimulai.

Setelah sehari saya periksa - saya tidak bisa mengatakan bahwa baunya telah hilang. Meskipun mereka pasti semakin lemah. Saya mematikan gadget, lalu menyalakannya lagi sehingga berfungsi selama 30 menit dalam mode tinggi. Saya memeriksa di malam hari - well, hampir tidak berbau. Sekali lagi, siklus "nonaktif", ia bekerja dalam mode tinggi selama 30 menit. Sekarang di dalam lemari hampir tidak ada bau.

Ternyata untuk kabinet saya membutuhkan pekerjaan gadget yang lebih intensif. Jelas bahwa "FreshON" berbau hal-hal berbau sangat (kotor atau diresapi dengan eau de toilette) tidak akan menghilangkan bau - Anda perlu memahami ini. Tapi dia bisa menghilangkan bau khas lemari pakaian. Peringkat saya adalah 5 dari 5.

Sepatu kets
Menguji sepatu kets. Saya berlari mereka di jalan. Mereka basah, kering, tetapi baunya, tentu saja, tetap ada. Saya baru saja memasukkan FreshON, high-end, di sepatu lari saya. Setelah setengah jam saya memasukkan yang lain. Dan dari yang pertama masih menarik dengan ozon ...



Setelah putaran pertama saya harus mengulangi operasi, karena hasilnya tidak cocok untuk saya. Jadi, sekali lagi saya menempatkan mode tinggi - dan di sepatu kets.

Setelah ronde kedua, sepatu kets bisa diletakkan berdampingan tanpa rasa takut. Bau itu tidak terlihat, kecuali jika Anda mendekatkan hidung Anda kepada mereka. Peringkat saya adalah 4 dari 5. Poinnya berkurang karena kerumitan memindahkan perangkat.

Ransel,
apa lagi yang akan saya menyegarkan, pikir saya, dan mata saya tertuju pada ransel. Berbau api unggun, dia berbaring di dapur. Anda tidak bisa membawanya keluar ruangan - aroma api unggun dan rawa yang terus-menerus akan menyebar.



Seperti halnya sepatu kets, saya meletakkan gadget di dalam selama setengah jam. Lalu saya nyalakan lagi selama setengah jam. Hasilnya - praktis tidak ada bau di dalamnya, tetapi di luar tetap ada. Bagaimana cara menghapusnya dari luar? Cuci ransel Anda, tentu saja. Pekerjaan ini bukan untuk FreshON. Saya tidak akan mengevaluasi, karena gadget tidak cocok untuk menghilangkan bau dari ransel.



Kesimpulan:
1. "FreshON" bagus di ruang tertutup kecil.
2. Dalam kasus di mana Anda perlu berkelahi dengan bau secara intensif, ada baiknya menyalakan mode tinggi perangkat beberapa kali.
3. Harus dipahami bahwa "FreshON" menghilangkan bau. Karena itu, hasil karyanya akan sama sekali tidak ada bau, tetapi tidak aroma kesegaran.
4. FreshON membantu menjaga makanan tetap segar. Tetapi pada saat yang sama lemari es Anda harus bekerja :) Dan Anda tidak harus menunggu peningkatan umur simpan di kali. Beberapa hari adalah waktu yang sebenarnya.
