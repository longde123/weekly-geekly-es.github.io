孟山都许可的CRISPR / Cas9基因编辑技术


孟山都在圣路易斯的温室。照片：Sirton / Getty

跨国公司孟山都公司（Monsanto Corporation）是转基因种子和除草剂生产的全球领导者，但尚未通过有前途的CRISPR / Cas9技术。该公司很快意识到了这种新的基因编辑方法的商业前景。 2016年9月22日，众所周知，孟山都公司从哈佛大学和麻省理工学院的私立布罗德研究所获得了CRISPR / Cas9技术的许可，该技术以其在基因组学领域的研究而闻名。

在CRISPR / Cas9的帮助下，孟山都计划开发新品种的农作物，包括玉米和大豆，它们对疾病，病虫害和恶劣天气条件的抵抗力更强。多亏了转基因文化，人们希望即使面对全球气候变化和世界人口增加，人们也不会死于饥饿。

CRISPR / Cas9


CRISPR / Cas9基因编辑方法的独特之处在于，借助免疫系统的自然机制，人体可以独立地在指尖切除掉错误的DNA部分。由于自然进化，生物体基因组中可能发生类似的变化。根据基因编辑的结果，无法确定它是由于人为干预还是自然突变而产生的。

瑞典遗传学家最近说服监管机构如果植物不含“外源DNA”，则根据欧洲联盟的要求未正式承认其为转基因生物。因此，目前并非所有欧盟国家都禁止进行 CRISPR / Cas9基因实验和使用CRISPR / Cas9方法进行转基因植物的种植。孟山都公司希望CRISPR / Cas9的特定功能将有助于更广泛地应用这种基因编辑技术，从而绕开适用于GMO的一些限制。

CRISPR / Cas9技术可进行精确的DNA编辑。它比标准的基因修饰技术更有效，并且肯定比过去几个世纪以来通过选择方法进行的“盲目”基因组修饰的古老方法有效一个数量级。多亏了CRISPR / Cas9，科学家无需选择数百代即可获得所需的结果-抗虫性或所需的食物参数。

转基因-人类的希望


孟山都生物技术总监汤姆·亚当斯（Tom Adams）解释说：“增加农业用地面积来增加产量显然对人类至关重要。” “基因编辑是另一种工具，可以帮助我们加快这一过程。”

多年来，人们试图生产具有所需特性的蔬菜和水果。例如，开发对病毒和真菌具有抗性的西红柿和辣椒品种。基因编辑是处理负责这些特性的植物基因的出色工具。

Broad研究机构是一个独特的研究中心，以亿万富翁Eli Broad的捐赠为生。科学中心将哈佛医学院的临床专业知识与麻省理工学院和哈佛大学的学术和工程科学相结合。自2004年成立以来，他一直在基因组学和化学生物学领域进行大规模研究，包括对小分子进行大规模筛选，对各种哺乳动物的DNA基因组进行解码以及其他遗传研究。 CRISPR / Cas9基因组编辑技术也是Broad Institute的研究领域之一。

该研究所传统上会与科学界和非营利组织共享研究成果。与孟山都公司达成的非排他性协议通常是该机构首次许可将基因技术用于农业。

这个决定并不容易。他长期通过伦理与安全委员会认为研究所。当相应的DNA片段在人群中迅速传播时，科学家将潜在的威胁视为基因不受控制的传播（基因驱动）。另一个值得关注的-该公司可以利用该技术对不育株的生产，迫使农民购买他们的产品，每年

到这并没有发生，Broad研究院为使用CRISPR / Cas9技术设置了许多条件。特别是，禁止在种群中故意分布基因和生产不育植物。另外，对烟叶的某些修饰类型是有限的-这种修饰可以增加吸烟的流行度（道德委员会指出，吸烟使患肺癌的风险增加25倍）。出于科学目的和用于生产医学蛋白质的目的，可以编辑烟草基因组。

孟山都的终结


在“转基因生物”的狂热反对者中，“孟山都”公司的名称几乎已经成为家喻户晓的词。在互联网上进行简单的搜索即可获得数百张带有该跨国公司名称的漫画。数百人参加海报集会，孟山都被描绘成是毒害儿童的怪物。该公司还因开发美军在越战期间使用的除草剂-特工橙（Agent orange）而闻名。



很快，这些讨厌孟山都的人会发现生活变得更轻松。两周前，德国拜耳公司宣布以587亿欧元的价格收购孟山都，即以每股128美元的价格收购（市场价格的+ 44％）。两家公司的董事会一致批准了合并。顺便说一句，这是最大的交易在德国经济的整个历史上。

经过监管机构的必要检查，预计该交易将于2017年底完成。可以认为，德国人的关注可以摆脱声誉极差的“孟山都”品牌。

当然，孟山都品牌的消失不会影响转基因产品的分销。如果有30个国家的监管机构批准合并，拜耳和孟山都将共同控制全球种子市场约30％的份额。
