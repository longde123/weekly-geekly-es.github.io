参加者的眼中的HolyJS 2019莫斯科



HolyJS 2019莫斯科最近通过。 这是本次会议的第二次。

去年，我设法写了一个初学者的小文章 ，一年后，我想重复我的实验，并对会议和报告进行简短回顾。

对于有经验的会议参与者和初学者来说，这篇文章都会很有趣，因为今年有很多创新。

会议审查。 革新性


这次有很多娱乐活动：大量的看台，各种任务和测验，编码问题。 起初，人们不知道该怎么办:-)好像我想去报告并解决问题。

徽章贴纸


我遇到了一个奇怪的创新，使我想起了Morpheus，他从电影《黑客帝国》中选择了一些药丸。 注册时，每个参与者都可以粘贴两个贴纸之一：绿色或红色。 通过将其粘贴在徽章上，您可以明确是否要与其他参与者进行交流。 与其他参与者讨论了该主题之后，事实证明每个人对此都模棱两可。 在会议的两天中，我的任何一个朋友都没有看到带有红色标贴的人。

在我看来，会议似乎一直是关于报告和交流的。 毕竟，如果只报告的话，那么您可以放心在家观看广播。 而且，既然出现了红色徽章，也许沟通中的一切都不会那么顺利吗？ 还是关于开发人员推动组织者的陈规定型观念？

也许这个话题将发展成为明年有关软技能的报告。 这个季节，它们比以往任何时候都受欢迎。

工作坊


研讨会已成为另一项创新。

讲习班是实践练习，可以解决您所遇到的问题。 通常持续几个小时。

他们的参加者说，研讨会对他们来说似乎很潮湿。 但是，在我看来，举办研讨会非常困难……而这仅仅是开始，也是第一次实践。



会议摊位之一的比赛

闪电谈话


我的发现是闪电谈话-在聚会中间进行的5至10分钟的微型报告。 在第一天，任何人都可以尝试作为演讲者。

主题很有趣，思路敏捷，最重要的是-整个晚上都有一些要讨论的话题。

报告概述


因为有英文报道，所以HolyJS很可能被认为是一次国际会议。 今年大约有俄罗斯人的两倍。 有时，这会给不太会说英语的人带来不适。 在报告中，演讲者说话非常快或带有重音，有时甚至是全部。 如果选择英语报告，请记住这一点。

TC39团队是著名的演讲者。 这是一个开发JavaScript的委员会。 他们进行了采访和个人表演。

我将重点介绍一些有趣且非同寻常的报告。



Michael Saboff的报告

  • Michael Saboff-“ JavaScript Core，许多编译器使该引擎执行”


    Michael概述了用于WebKit的开源JavaScriptCore虚拟机。 他讲述了JavaScriptCore包含哪些级别，以及它用于优化的内容。 演讲者是TC39委员会的成员。

  • Ivan Popelyshev-“以PixiJS为例，我们如何为从<div />到WebGL的过渡支付费用，以及我们从中获得什么（利润！）”


    Ivan展示了将应用程序切换到WebGL的利弊。 他列举了DOM / CSS / SVG，Canvas 2d和WebGL的比较示例。 剧透！ 仅仅因为您不能使用WebGL，您需要了解很多。 我为这份报告提供建议，发言人精辟地解释。

  • Dmitry Patsura-“基于LLVM在TypeScript上为TypeScript开发编译器”


    关于编译器，解释器，各种引擎，虚拟机，一些汇编器，编写自己的编译器的示例的相当精明的报告……大脑最终从中爆炸了一点。

  • Prashant Palikhe-“从代码到闪烁的像素”


    Pranshant谈到了在地址栏中输入URL时发生的浏览器进程。 我检查了Google渲染引擎Blink的内部组件。

  • NicolòRibaudo-“ @ babel /使用方法”


    Nicolò解释了babel的组成，内部的工作原理，JavaScript的编译方式以及对新ECMAScript的支持。 在报告的结尾，他还实时编写了一个插件。 聆听是一种乐趣，最重要的是-每个字都清晰。

  • 丹尼斯·拉丹（Denis Radin）-“计算机可以为所有生物的利益祈祷吗？ JavaScript和WebGL的祷告。”


    很奇怪，并同时报告。 事实证明，这是最昂贵的，因为演讲者前往尼泊尔。 整个报告充满了以下问题：“为什么全部？ 您是怎么想到这个的？” 丹尼斯（Denis）保持着好奇心，并只在最末尾进行了解释。 如果您喜欢奇怪的报道，那么这份报告适合您。



NicolòRibaudo报告

我将分别突出显示TC39小组的报告，因为对我而言，这是这次会议最大的失望。 宣布这是向TC39委员会询问JavaScript的过去，现在和未来的机会。 也许没有一个与会者能完全理解该报告仅基于他们的问题-在线采访，并且他们询问发生在他们身上的第一件事，因此该报告具有更一般的管理意义（格式：“您好吗？”。 ，“委员会是如何工作的？”）

总结


今年的会议上有很多人，但是每个人都很舒服。 总的来说，我比去年更喜欢这次会议。 也许有可能完全融合到conf的气氛中。 而且那里没有传统的小鸭礼品。
我希望这篇评论对您有所帮助，并且您可以在报告中学到一些新东西。

参加会议，太酷了！

PS我的评论是真诚的，希望是公正的。 原谅组织者:-)

PPS一旦发布了视频报告，我将把它们附加到文章上。
