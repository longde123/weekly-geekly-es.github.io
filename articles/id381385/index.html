Mesin penggilingan CNC 3-sumbu yang sederhana dan murah

Pada bulan Juli, tepat satu tahun telah berlalu sejak Master Kit menciptakan printer 3D pertamanya . Kami mencoba membuatnya sesederhana dan semurah mungkin bagi pengguna kami. Ini pertama dan semua model berikut dari printer kami dikendalikan oleh perangkat lunak Pengulang-Host yang terbuka. Bersama Anda, kami terbiasa dengan antarmuka ini. Dan sekarang menurut kami hampir sempurna untuk printer 3D.

Printer 3D adalah fenomena yang relatif baru dalam pemodelan. Sebelum mereka, mesin penggilingan CNC biasa digunakan. Sebenarnya, sekarang mesin penggilingan 3D bekerja di semua perusahaan manufaktur. Saya benar-benar ingin memiliki alat seperti itu di bengkel rumah. Tidak lebih cepat dikatakan daripada dilakukan!

Master Kit memperkenalkan perancang mesin penggilingan 3D yang sederhana dan murah. Tidak ada mesin - mesin. Mesin
penggilingan router 3D CNC dirancang untuk penggilingan dan pemotongan 3D. Semua Repetier-Host yang sama, file model 3D yang sama - * .stl. Tapi dia tidak mencetak model, tetapi memotong semua yang tidak perlu. Semacam 3D negatif!

Persiapan untuk bekerja

1. Memasang pemotong penggilingan

Mesin siap dioperasikan.


Menginstal perangkat lunak

Untuk menginstal perangkat lunak, jalankan file setup.exe dari arsip Installer Repetier Frezz

. Ikuti instruksi program selama instalasi.

Program-program berikut akan diinstal:
- RepetierHost: program untuk mengendalikan pemotong frais
- Skeinforge: program untuk menerjemahkan model 3d ke dalam kode untuk pemotong frais
- Arduino: program dan driver untuk menginstal router

. Koneksi pertama dari router

Koneksi pertama akan menginstal driver.
Untuk menghubungkan printer, lakukan langkah-langkah berikut:
- keadaan awal: daya dimatikan, kabel USB hanya terhubung ke komputer
- sambungkan kabel USB ke printer
- instalasi otomatis driver untuk port COM board akan dimulai
- jika instalasi berhasil, sistem akan menampilkan pesan tentang kesiapan peralatan untuk bekerja; jika windows tidak dapat menginstal driver, sistem akan menampilkan pesan yang menyatakan bahwa driver tidak dapat diinstal secara otomatis.

Jika driver diinstal secara otomatis, Anda perlu menentukan nomor port COM, untuk ini:
- buka "Control Panel / System / Device Manager" (gunakan bantuan Windows untuk membuka "Device Manager").
- pada tab â€œPorts (COM and LPT)â€, tulisan akan ditampilkan: Arduino Mega 2560 (COMâ„–), dengan â„– adalah nomor dan merupakan nomor port COM pemotong penggilingan Anda.



Jika driver belum dipasang secara otomatis, hubungi layanan dukungan pabrikan untuk bantuan .

Operation

Connection
- colokkan adaptor 12V ke jaringan 220 V dan sambungkan kabel daya ke mesin
- sambungkan kabel USB ke mesin dan komputer
- jalankan program Repetier-Host (baik dari pintasan desktop atau dari menu Start Windows / Programs / Repetier-Host / Host Berulang



Jendela program terbuka


Saat menghubungkan printer untuk pertama kalinya, perlu mendaftarkan nomor port COM yang ditentukan dalam ayat 5 di profil program printer.
Untuk melakukan ini, pilih menu "Printer Settings" di tab "Configuration".



Atau klik tombol.



Jendela


penting terbuka : ketika Anda memilih port COM, printer harus terhubung.

Pilih port COM dari daftar "Port".



Jika daftar tersebut tidak berisi port yang diperlukan, klik "Refresh port" dan coba lagi.
Setelah memilih port yang diinginkan, periksa kecepatan dalam "Baud rate".
Nilai harus "250000".
Jika nilainya berbeda, maka pilih 250.000 dari daftar.

Parameter yang tersisa tidak perlu diubah. Profil printer dikonfigurasikan sepenuhnya.

Setelah kita klik tombol "Terapkan".

Setelah selesai, klik "OK."

Penggilingan

Program kerja untuk penggilingan

File sumber untuk penggilingan dapat berupa:
- Model 3D (format STL) - untuk memproses permukaan tiga dimensi
- Grafik vektor - untuk pemrosesan datar benda kerja

Ketentuan umum untuk penggilingan.

Penggilingan dilakukan dengan gilingan yang memiliki karakteristik sebagai berikut:


Parameter berikut ini penting untuk menyiapkan program:
- diameter pemotong: dalam program, nilai "Diameter pabrik (mm)"
adalah panjang ujung tombak: dalam program nilai "Tinggi pabrik (mm)"

PENTING: kedalaman pemrosesan model 3D dari bagian atas benda kerja tidak boleh lebih panjang dari panjang tepi potong.

Penggilingan dilakukan dari atas benda kerja ke bawah.
Karena tidak ada posisi manual di sepanjang sumbu XYZ, maka perlu untuk mempersiapkan dan memposisikan model 3D dengan benar dalam program pemodelan 3D.

Sebelum mulai bekerja, pemotong secara otomatis diatur ke posisi 0, yang sesuai dengan nilai minimum di sepanjang sumbu. Bagaimanapun, benda kerja akan lebih tinggi dari ujung pemotong.

Oleh karena itu, dimensi model 3D dan nilai absolutnya dalam koordinat XY harus sama persis dengan lokasi benda kerja untuk penggilingan.

Menggunakan model 3D untuk milling.

Buka tab "Penempatan objek", tambahkan model yang diinginkan.

Jadi, seperti pada saat milling, material dikeluarkan dari benda kerja. Hal ini diperlukan untuk memposisikan model dalam program dan benda kerja di atas meja.

Kami mulai dengan lokasi model.
