Was ist neu in Upsource 2018.2?

Hallo allerseits!

Kürzlich wurde eine neue Version von Upsource 2018.2 veröffentlicht, die das Arbeiten mit Codeüberprüfungen vereinfacht. Die neue Version hat die Automatisierung verbessert, damit Sie sich auf die Codeanalyse konzentrieren können, anstatt administrative Aufgaben auszuführen. Jetzt werden wir genauer erzählen.



Aktualisiertes Design


Wie andere Serverprodukte von JetBrains haben wir Upsource auf die neueste Version von Ring UI aktualisiert. Jetzt ist die Upsource-Oberfläche moderner und bequemer geworden!



Der Name des Commits in benutzerdefinierten Workflows


Jetzt können Sie den Namen des Commits in den Workflows-Einstellungen angeben. Wenn das Commit den relevanten Kriterien entspricht, wird die Überprüfung automatisch erstellt.

Dieses Update erweitert die Funktionalität benutzerdefinierter Workflows und ermöglicht es Ihnen, sich auf Ihren Code zu konzentrieren, ohne den Kontext zu wechseln, um Überprüfungen zu erstellen.



Die Fähigkeit, sich als "nicht im Büro" zu markieren


Wenn Sie in den Urlaub fahren oder einen Tag frei nehmen möchten, können Sie jetzt den Abwesenheitsmodus für eine bestimmte Anzahl von Tagen aktivieren. Wenn der Modus aktiviert ist, werden Ihnen nicht automatisch Bewertungen und Erinnerungen per E-Mail zugewiesen.



Tooltip überprüfen


Jetzt können Sie die Details der Überprüfung anzeigen, ohne die Überprüfungsseite selbst aufzurufen. Sparen Sie wertvolle Zeit, während Sie auf dem Laufenden bleiben.



Gleichzeitiges Entfernen mehrerer Revisionen aus einer Überprüfung


In früheren Versionen mussten Sie mehrere Revisionen einzeln auswählen, um sie aus einer Überprüfung zu entfernen. Jetzt können Sie mehrere Revisionen gleichzeitig aus der Überprüfung auswählen und alle zusammen löschen.



Möglichkeit, Termine für die Überprüfung automatisch zu verschieben


Jetzt wird die Frist für die Überprüfung automatisch verschoben, wenn Sie neue Änderungen hinzufügen oder neuen Überprüfern beitreten. Um die Frist zu übertragen, können Sie die gewünschte Anzahl von Tagen einstellen.



Ab Java 9


Upsource 2018.2 unterstützt die Syntaxhervorhebung und Codeüberprüfung in Projekten mit Java 9 und höher.



Wir haben auch die Leistung von Upsource verbessert und Unterstützung für neue Versionen von Mercurial (4.6+) hinzugefügt.

→ Hier können Sie die gesamte Liste der Aufgaben anzeigen, die in der Version enthalten sind .

Probieren Sie Upsource 2018.2 aus und teilen Sie uns mit, wie Sie die neue Version erhalten! Vergessen Sie nicht, dass Upsource für kleine Teams (bis zu 10 Personen) kostenlos ist.

Abonnieren Sie das Twitter-Konto von Upsource oder unseren Blog, um die Produktneuheiten zu verfolgen.
