Cherchons à nouveau «notre» film à Kinopoisk


Sur le chemin du travail, je me suis souvenu d'une merveilleuse méthode mathématique et j'ai décidé d'examiner de plus près les étapes de la recherche au cinéma et de découvrir ce que les robots de cinéma attendent le plus cette année.

Il était presque évident pour moi que lors de l'inscription, Kinopoisk donne l'ordre des utilisateurs dans l'ordre. Les nouveaux utilisateurs ont un nombre légèrement inférieur à 15 000 000. Autant de pages utilisateur que le site ne me permettra pas de télécharger. Eh bien, vous pouvez essayer de tordre les pages dans un ordre aléatoire et de regarder des utilisateurs aléatoires.

Un paramètre assez intéressant est la date d'enregistrement.

Un peu de code chinois nous indiquera le nombre de jours du 1er janvier 0000 à l'inscription 
function [ DateNumber ] = find_datenumber( date_reg ) m(1).m = strfind(date_reg,''); m(2).m = strfind(date_reg,''); m(3).m = strfind(date_reg,''); m(4).m = strfind(date_reg,''); m(5).m = strfind(date_reg,''); m(6).m = strfind(date_reg,''); m(7).m = strfind(date_reg,''); m(8).m = strfind(date_reg,''); m(9).m = strfind(date_reg,''); m(10).m = strfind(date_reg,''); m(11).m = strfind(date_reg,''); m(12).m = strfind(date_reg,''); for jj = 1:12 if ~isempty(m(jj).m) month = jj; end m(jj).m=[]; end year = str2num(date_reg(end-3:end)); day = str2num(date_reg(1:2)); DateNumber = datenum(year,month,day); end
