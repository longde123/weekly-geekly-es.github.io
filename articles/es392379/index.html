Convierta Raspberry Pi 3 en la baliza URL de Eddystone



Raspberry Pi 3 mini PC se lanzó recientemente, y en muy poco tiempo después de la aparición del sistema en el mercado, comenzaron a crear proyectos interesantes sobre la base. Una de ellas, muy específica, es la transformación de "frambuesas" en una baliza URL de Eddystone.

Eddystone es el propio formato de baliza abierta de Google (especificación aquí ), y esta tecnología es compatible con Android OS e iOS. Para que esto funcione, debe mantener Bluetooth LE constantemente encendido en su teléfono inteligente; esta es quizás la única condición. Todo lo que se requiere para convertir RPI 3 en una baliza URL de Eddystone es la mini PC en sí, una fuente de alimentación y una tarjeta SD. El resto está debajo del corte.

Entonces, un poco más sobre Eddystone. Su ventaja es la capacidad de enviar tres paquetes en lugar de uno. Estos son UID, URL, telemetría.
  • Eddystone-UID. Este es el mismo identificador: un análogo de iBeacon (UUID, mayor, menor). Consta de 2 partes: ID de espacio de nombres (10 bytes), ID de instancia (6 bytes).
  • Eddystone-URL. Ahora las balizas traducen URL de hasta 18 bytes. Los acortadores de URL nos ayudarán a resolver el problema de las URL largas. Por ejemplo, un servicio del mismo Google goo.gl. Desde esta dirección touchin.ru/portfolio/squawk puede obtener una versión corta de goo.gl/PidTf3. Esta es la principal innovación, explicaré por qué un poco más tarde.
  • Eddystone-TLM (Telemetría). Los siguientes datos de baliza están disponibles:

  • nivel de carga
  • la cantidad de paquetes enviados desde que se conectó a la fuente de alimentación
  • Tiempo de actividad de la baliza desde el momento de la conexión a la fuente de alimentación.
  • temperatura


El teléfono que recibe estos paquetes puede advertir sobre la recepción de información de cierto tipo o realizar alguna acción.

Configuración de "frambuesas"

1. Descarga Raspbian ;
2. Instale la imagen ;
3. Estamos buscando "frambuesas" IP. Esto se puede hacer usando el " Raspberry Pi Finder " o nmap.

$ sudo nmap -sP 192.168.2.0/24
o
$ arp -a | grep "b8: 27"

4. Conéctese a Raspberry:

$ ssh pi @ <the_ip_of_your_pi>

5. Buscamos información sobre el comando hciconfig

$ hciconfig -h

6. Encienda el dispositivo bluetooth

pi @ raspberrypi: ~ $ sudo hciconfig hci0 arriba

7. Configure el modo "anunciar y no conectable"

pi @ raspberrypi: ~ $ sudo hciconfig hci0 leadv 3

8. Ingrese los datos para Beacon

pi @ raspberrypi: ~ $ sudo hcitool -i hci0 cmd 0x08 0x0008 17 02 01 06 03 03 aa fe 0f 16 aa fe 10 00 02 77 65 62 67 61 7a 65 72 08 00 00 00 00 00 00 00 00
