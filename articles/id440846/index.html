Timer kinerja

Dalam berbicara di depan umum, itu ide yang baik untuk mengontrol waktu pidato.
Bagi saya, tidak adanya stres tambahan itu penting.

Banyak penghitung waktu dapat ditemukan di Internet, tetapi semuanya mengandung milidetik dan detik, atau berkedip titik dua. Beberapa dari mereka dapat disesuaikan, tetapi saya belum menemukan satu pun yang diperlukan. Mereka terus memperbarui gambar layar dan membuat saya khawatir. Saya hanya ingin melihat berapa menit telah berlalu, dan saya akan melihat perangkat. Saya juga menyadari bahwa hitung mundur memberi tekanan lebih pada saya daripada menampilkan berapa banyak waktu telah berlalu.


Jadi saya ingin melihat berapa menit yang telah berlalu sejak awal pertunjukan

Ketika saya melakukan salah satu pidato pertama saya, saya memiliki telepon Apple dan saya tidak dapat menemukan apa yang saya inginkan. Saya bahkan membeli beberapa timer untuk pertunjukan. Tapi bukan itu. Saya seorang pengembang web dan tidak ingin menghabiskan lebih dari setengah jam untuk alat ini. Mudah membuat halaman web, tetapi tidak mudah membuat perangkat tidak mematikan layar. Yah, saya ingin itu berfungsi tanpa internet. Pengepakan dalam cordova atau sesuatu yang serupa dan penerbitan ke pihak juga bukan bagian dari rencana saya (terutama dengan kemungkinan bahwa apel akan menolak aplikasi ini).

Kemudian teman saya yang sedang mengembangkan untuk ios menulis aplikasi seperti itu. Dari semua fungsi ada start / reset, dan larangan mematikan layar selama operasi. Kami tidak pernah menerbitkannya.

Sekarang saya mengubah ponsel ke platform lain dan sekali lagi saya harus berbicara. Dan saya pikir, yang diperlukan hanyalah merekam video semacam itu. Dan itu dia. Video ini hanya menunjukkan berapa menit telah berlalu sejak awal.

Ini dia:


Di sini Anda dapat mengunduh. 160 Mb. yadi.sk/i/VyeWx1jxfLBZ2Q

Ada 130 menit, ini cukup untuk mengadakan kuliah dua jam yang agak berlarut-larut. Sebenarnya, Anda tidak perlu memberikan kuliah seperti itu, tetapi bagaimana jika seseorang membutuhkannya?

Hellish over-engineering, tapi hell, it works, it just berfungsi, lebih mudah daripada aplikasi.
Di telepon, laptop, tablet, perusahaan apa pun, bahkan di TV.

Ya, saya membuatnya dengan cara yang sangat mengerikan. Saya terlalu malas untuk memahami editor video, dan saya hanya mengambil Open Broadcast Studio dan merekam halaman web dengan alat bawaan. Saya memfilmkan halaman web dalam 5 menit. Ya, saya merekamnya selama dua jam (well, lebih tepatnya, itu ditulis di latar belakang itu sendiri).

Maafkan saya atas kejahatan mengerikan terhadap teknologi. Mungkin Anda harus menunggu hingga hari Jumat. Tetapi sekarang menjadi sedikit lebih mudah untuk berbicara dan tidak terlalu khawatir.
