Kami melihat format yang tersedia di YouTube

youtube-dl -f 11 https://www.youtube.com/watch?v=

Memeriksa laju bingkai file

ffprobe -v error -select_streams v -of default=noprint_wrappers=1:nokey=1 -show_entries stream=r_frame_rate input.mp4

Storyboard

ffmpeg -i "input.mp4" -start_number 0 -vf fps=25 "convert/capture-%002d.png"

Selanjutnya, prosedur standar untuk meningkatkan frame menggunakan jaringan saraf.

Ikatan dari png + audio

  • tanpa izin - ffmpeg -framerate 25 -i convert/capture-%002d.png -vb 20M -i audio.webm -vcodec mpeg4 -y "ffmpegtest.mp4"
  • dengan resolusi yang ditentukan - ffmpeg -framerate 25 -s 900x720 -i convert/capture-%002d.png -vb 20M -i audio.webm -c:v libx264 -pix_fmt yuv420p -crf 23 -y "ffmpegtest3.mp4"
