STAX历史：静电消除器与市场状况

我很少想为任何品牌写赞歌；通常，我对创建公司或引进先进技术的个人感兴趣。 对于人们来说，我不会ski草。 但是，此材料将是规则的例外。 我们历史周期的下一篇文章将讨论一家公司，该公司的名称已成为静电耳机一词的同义词-这就是STAX。 关于公司创始人的信息很少，而关于工程师的信息则很少。



同时，Stax产品确实吸引了我。 在该品牌的设备中实施的技术解决方案仍然很受欢迎，这并不是由于市场规律，而是与之相反。 似乎这家公司不知道或不希望知道在任何形式的大规模音频妥协中，营销人员早在上世纪八十年代就已经赢得了胜利。 几乎无处不在，但STAX中却没有。 因此，在削减之下，静电声音再现的真正武士的故事-STAX公司。

未知的年代和突然的开始


STAX成立于1938年。 该公司由林直直组织，他从一家小型家族企业成功创立了一家举世闻名的公司。 这时，日本与中国发动了一场激烈的战争，这场战争始于1937年，顺利地进入了第二次世界大战，并于1945年结束。

关于公司在此期间发布的设备，几乎一无所知。 我想当时没有一家日本公司能够逃避国防工作。 当时的日本工业奉行“一切为前，一切为裕仁天皇的胜利”的原则。 考虑到STAX战后发展的概况，我假设该公司从事日本军用耳机的开发，例如以下所示。



对于非军事专业人士使用的电子设备，公司名称在战后时期已经存在。 正式地，第一个STAX设备是：电容式麦克风，独特的静电拾音头CP-20和同样独特的弯曲LA-24音臂。 所有这些设备在成立后的12年（1950年）几乎同时出现。


产品一次发布了多个版本，高级细分版本也进行了修改，并且设备更简单。 例如，CP-20AD / N弹药盒带有钻石针，CP-20AN带有蓝宝石。


换句话说，STAX就像袖子上的卡片一样，得到了几行流行且独特的高级设备，可以与当时世界知名制造商的最佳典范竞争。 似乎该公司已经在电声，材料科学，电物理领域进行了12年的基础研究，并且数十年来一直发展到突然成为（如果不是市场领导者的话）那么就占据了高级音频领域。

那时，该公司已经选择了发展方向-电声学中的静电设备。 所有发布的CP-20拾音头都是静电的。


在20到18000 Hz的可复制频率范围内，它们的范围非常广，对于那个时间而言，这是很大的。 CP线的逻辑延续是最著名的STAX产品之一，未直接连接到耳机CP-X静电盒式磁带。



STAX这个名字也清楚地暗示了公司简介。 顺便说一下，目前尚不清楚该公司何时收到此名称。 显然，名字タクス（Yugen-gaisha Sutakkusu）最初被认为与静电有关。

好被遗忘的新


1954年，STAX受到了另一项发展的推动，更加接近于未来的狭profile轮廓。 该公司已经发布了静电高音扬声器。 应当指出的是，他们试图从上世纪20年代开始将声辐射的静电原理用于声学系统。

特别是在电影院的音响系统中，通常使用静电驱动器，但是由于缺乏合适的聚合材料，这种类型的发射器在当时是不可靠的，到30年代中期动圈式扬声器几乎完全取代了静电技术。 可以说，随着STAX在1954年的发展，它使该技术复活并确定了其发展方向。


1960年，STAX举行了具有划时代意义的活动，该公司发布了世界上第一只静电耳机SR-1。 时至今日，静电原理已被认为是确保使用耳机播放最大保真度的参考。 Onotaka Hayashi放弃了常用的“耳机”，并称该设备为一个全新的术语“ Earspeakers”，从字面上可以翻译为“入耳式扬声器”。


这种耳机需要自己的放大器，1960年该公司设法只发布了两种型号SRD-1和SRD-2。 两者均具有偏置电流控制。 放大器是具有相对强大输出变压器的电子管放大器。 它们的输出信号具有高电势差，这对于实现静电声发射是必要的。 有趣的是，工程师从150 V的电压开始，该电压对于静电发生器来说相对较小，在接下来的修改中，该电压增加到230 V以增加辐射表面的幅度。



耳机膜使用约6微米厚的聚合物膜。 当时，它是世界上声音发射器中最薄的膜。 很大程度上归功于此功能，全球首款静电耳机具有以下特征：

  • 重量：320克
  • 孔径：大约5微米。
  • 灵敏度：95 dB / 100 V rms
  • 阻抗：130 kOhm / 10 kHz
  • 频带：20-20,000 Hz
  • 基本电压：150V / DC
  • 零售价：¥6,000 / $ 50，今天等于$ 432。

最初生产的静电消除器的逻辑延续是SR-3的改进版本。


该公司将静电耳机的设计和制造作为其主要业务。 同时，STAX工程师正在努力为全尺寸落地式扬声器创建发射器。 因此，在1964年，出现了一系列ESS静电宽带扬声器；该产品系列包括ESS-3A，ESS-6A和ESS-12系统。



设备具有其时代令人印象深刻的特征，即：

  • 频率范围：40 Hz至20 kHz。
  • 声压输出电平：84 dB / W / m。
  • 最低建议功率：15瓦起。
  • 电阻：8欧姆（400欧姆-OTL端子）。
  • 电源：100 V，50 Hz / 60 Hz

尽管指标值得信赖，但演讲者并没有成为畅销书，即使在日本本土也被认为是异国情调的。 来自STAX的绝大多数演讲者都遭受同样的命运。 直到现在，一个稀有的音乐爱好者已经准备好为静电扬声器买单了，类似的情况发生在60年代。 ESS系列在60年代延续了另一款ESS-4A装置，此后从商业角度看，它作为一个毫无希望的方向被遗忘了。


黄金时期


从60年代后期开始，STAX开始了密集发展和增长的时期。 他们的耳机在日本以外的地区变得越来越流行，这导致产量的急剧增加和静电技术领域的研究日益加强。 因此，从1968年到1970年，大约出现了40种系列产品，其中几乎是70年代普通发烧友所需要的全部产品。 特别是耳机，单独的静电HF部分，拾音头，音调臂（尤其是碳纤维部件，例如UA-9），放大器（包括A类）以及根据双重单声道原则）。

UA-9

1979年新产品特别丰富。 那时，该公司发布了CA-X PRO双单声道放大器，即传奇的CP-Y / TYPE 2拾音器，这是一种用于将静电耳机连接到标准立体声系统SRD-X的特殊适配器。


同样在1979年，传奇的SR-Lambda全景耳机问世。 耳机膜由3微米厚的薄膜制成，并且经过精心设计的声学设计使其有可能获得全景声并消除“头内声音”的影响。


拉姆达

耳机已成为当今最流行的系列之一的开始，它在8 Hz至35 kHz的频率范围内再现声音，其灵敏度为102 dB / 100 V rms，并且能够产生高达109 dB的声压。


增强型1986 SR-Lambda PRO

80年代STAX变得更加饱和。 该公司能够开发和准备75多种产品以进行批量发布。 其中，发布了工作室版本的SR-Lambda，称为Stax SR-Lambda Pro。 除了已经建立的静电耳机外，Stax还通过发布CDP Quattro CD播放器而成为数字音频的先驱之一。



播放黑胶唱片的设备的开发仍在继续，放大器的分配方向是分开的。 在1980年代，总共设计并推出了76款设备，包括12款静电耳机。

传统主义濒临犯规


在1990年代中期，该公司开始经历财务困难。 事实是，在90年代，发烧友的主要趋势，即复制保真度的观念在商人和“全能”音频媒体的信息压力下破裂并改变。 易用性和实用性备受青睐，在飞利浦和松下这样的巨头每年将自己推向市场的背景下，STAX产品显得有些多余且价格过高。


许多STAX设备的基础已经几十年没有改变，例如，在POD-X（70年代的发展）中，清晰可见各种多样，昂贵的听觉“绒毛”和传奇的Telefunken 12AU7灯
鉴于上述情况，到2000年代初，该公司的财务用尽，濒临破产。 尽管经济形势困难，大多数部门仍在继续开发新产品。 1996年，公司进行了重组，并以更新的结构继续开展活动。 到底发生了什么变化，公司资源却保持沉默，林直直继续像以前一样管理公司。 但是，在所有官方资料中，1996年之后的时期都称为“新税制”。 可能是，该公司取消了所有无法使用耳机的部门。

重组对公司没有太大帮助，STAX的财务困难加剧，达到零中值。 经过这段时期的巨大努力，该公司推出了64种产品，主要包括耳机，用于它们的放大器，适配器和其他配件。 90年代创新的一个突出例子是世界上第一个带有便携式放大器的通道内静电耳机STAX SR-001。 新项目没有保存，系列减少，收入继续下降。


STAX SR-001 mk2-高级修改

节省吸收


一些信息表明，Naotake Hayashi公司的创始人和永久领导人在90年代后期离开了职位。 从网络上的零碎数据中得知，1991年他已经84岁了，他仍然经营着这个企业。 显然，他的离职对改善公司事务没有任何帮助。


2011年12月，STAX被中国扬声器制造商Edifier收购。 这家中国公司表示，已以象征性的价格1.2亿日元（约合100万美元）收购了Stax的100％股权。 新的所有者决定继续为其生产静电耳机，放大器和DAC。 对公司的进一步投资使她免于破产，并使继续开发和批量生产成为可能。

总结


去年，该公司成立80周年。 在周年纪念日，STAX发行了限量版传奇SR-009BK耳机，仅发行了180册。



如今，该公司的其他产品已批量生产数千件。 尽管经历了财务困难，并且转向了更大的Edifier，但STAX却没有丢脸，正如他们在日本所说的那样。 他们没有跟随市场，而是找到了顺应潮流的机会。


如今，他们忠于多年前选择的方向，并且对他们的产品以及对电声原理本身的兴趣正在恢复。 有趣的是，该公司拥有一个维修车间，甚至可以处理20年前生产的模型，这对于他们的产品质量和经营方式都可以说很多。
