Emulator konsol game terbaik untuk Android



Konsol gim muncul sejak lama, banyak dari kita memulai perjalanan kita ke dunia gim dengan perangkat kecil ini. Beberapa game yang populer di SNES, Sega, PlayStation, tetap populer hingga sekarang. Versi baru dari beberapa hit muncul sekarang di konsol game terbaru. Misalnya, Tempur Mortal yang selalu hidup. Sekarang mereka akan menghidupkan kembali Crash Bandicoot.

Kadang-kadang Anda ingin bermain bukan dalam hal baru, tetapi dalam game lama - yang kami mainkan bertahun-tahun yang lalu. Ini dapat dilakukan - di pasar dan jaringan ada awalan bekas yang digunakan, aksesori untuknya, permainan. Tetapi ada opsi yang lebih terjangkau - emulator konsol game. Berkat emulator, Anda tidak dapat membuang-buang uang untuk konsol, kartrid, disk lama dengan game. Ada banyak program semacam ini, mereka dikeluarkan untuk berbagai platform. Hari ini kita melihat yang terbaik, diuji oleh waktu dan banyak versi emulator konsol game untuk OS Android.

Emulator konsol game adalah shell perangkat lunak yang mengemulasi operasi perangkat keras perangkat itu sendiri. Mengembangkan emulator tidak sesederhana itu - ia membutuhkan pengalaman pemrograman dan pengetahuan informasi yang akurat tentang prinsip-prinsip operasi perangkat yang ditiru itu sendiri. Kebanyakan emulator dapat dibagi menjadi dua kelompok, termasuk program "sederhana" dan "majemuk". Emulator memberikan fungsionalitas yang lebih banyak daripada konsol lama (misalnya, menyimpan game untuk Dandy dan Sega).

Sederhana itu mudah - Anda perlu menginstal program, mengunduh gambar gim, dan hanya itu - Anda dapat menikmati proses passing. Ada pengaturan minimum dalam program tersebut. "Compound" memerlukan komponen tambahan - ini adalah BIOS konsol (biasanya file .bin) dan berbagai plugin dll. Untuk sebagian besar emulator konsol Android, plug-in tidak diperlukan. Tetapi BIOS dibutuhkan. File-file ini dilindungi oleh hak produsen, sehingga pengembang emulator tidak dapat menanamkannya - jika tidak, emulator seperti itu tidak akan masuk ke direktori program Google. BIOS perlu dicari di internet - ada banyak situs tempat file-file ini dibagikan.

Kami membutuhkan gambar permainan. Pengembang emulator mereka juga tidak memiliki hak untuk menyematkan perangkat lunak mereka. Gambar, atau ROM, tersedia di Internet. Untuk Sega atau PlayStation, jumlah ROM mencapai ratusan. Sekarang di Internet Anda dapat menemukan gambar bahkan game yang sangat langka yang sebelumnya sulit didapat di toko.

Sistem tertua


Colecovision


ColEm Deluxe



Konsol permainan video ini tidak diketahui semua orang. Ini pertama kali muncul di pasar pada tahun 1982. Ini telah menjadi populer dengan permainan seperti Lipan, Adipati Hazzard, dan Buck Rogers.

Emulator ColEm Deluxe berfungsi mulus dengan gambar semua game untuk konsol ini. Anda dapat menggunakan fitur paling canggih, termasuk dukungan untuk Wi-Fi atau multi-pemain. Program ini kompatibel dengan gamepad dan keyboard Bluetooth eksternal. Seperti dalam semua kasus lain, gim ini membutuhkan gambar gim - ada cukup banyak di jaringan.

Commodore 64


C64.emu



Ini adalah emulator berbayar, biayanya $ 3,99. Salah satu emulator
Commodore 64 yangpaling stabil,bekerja dengan sejumlah besar format file, ada dukungan multi-disk. Emulator didasarkan pada VICE 2.4.20, porta asli, tanpa menggunakan SDL. Diuji pada perangkat seperti Xoom, Galaxy S2, Nexus 4, Nexus 7, NVidia Shield, Xperia Tablet Z dan Xperia Play.

Dimungkinkan untuk menyimpan game, mendukung NTSC / PAL. Keyboard dan pengendali nirkabel didukung. Gambar gim tidak disertakan dengan emulator - sebagai aturan, pengguna emulator harus mencarinya sendiri.

Frodo c64


Versi gratis dari emulator sistem Commodore 64. Penulis, sayangnya, tidak lagi mendukung program ini. Meski begitu, semuanya berjalan stabil, tidak ada masalah. Jika Anda tidak ingin membayar $ 4, Anda harus memilih emulator ini.

Marvin

Ini adalah salah satu emulator ZX Spectrum terbaik. Ini memberikan emulasi penuh 48 / 128k sistem. Kecepatan permainan sesuai dengan aslinya, frame rate adalah 50. Program ini mereproduksi suara berkualitas tinggi. Ada database ribuan game. Dekompresi otomatis file zip.

NES dan SNES


Nostalgia.NES Pro



Nostalgia.NES Pro adalah salah satu emulator delapan-bit terbaik di pasar. Benar, program dibayar, harganya $ 1,99. Tetapi hampir semua game yang dikenal didukung oleh emulator. Semuanya dapat diubah - mode video, pengontrol grafis, suara. Cheat, pengendali nirkabel didukung.

Ada fitur yang menarik untuk mengembalikan karakter ke masa lalu. Jika selama berlalunya permainan Anda "terbunuh", maka Anda dapat mengaktifkan mundur, dan kembali waktu tertentu kembali. Anda juga dapat menghindari konsekuensi fatal dari kematian dengan menyimpan game di titik-titik penting dalam proses game, sehingga jika terjadi masalah itu akan dimuat, tetapi ini membutuhkan waktu. Dan di sini - satu klik pada tombol, dan pahlawan yang sudah meninggal kembali.

Ada emulasi pistol ringan - Anda dapat menggunakan jari Anda alih-alih tembakan. Tentu saja, ini sama sekali bukan seperti menembak dengan pistol ringan pada bebek, tetapi lebih baik daripada tidak sama sekali.

Superretro



SuperRetro16 adalah salah satu emulator SNES terbaik dari pasar. Biayanya $ 3,99. Dukungan untuk hampir semua gim, termasuk yang bermasalah untuk emulator nama lainnya, misalnya, Super Mario RPG. Di antara fitur-fitur lain dari emulator: save "on the fly", dukungan untuk gamepad eksternal dan curang. Ada juga multipemain melalui jaringan.

Sega


GENPlusdroid



GENPlusDroid adalah emulator Sega Genesis open source. Bekerja dengan Sega Master System dan game Sega Mega Drive. Tingkat kompatibilitas yang tinggi dengan game. Performa bagus. Jika Anda menginstal program di ponsel atau tablet dengan adaptor grafis yang kuat, Anda dapat menggunakan shader - dalam hal ini, kualitas gambar meningkat secara signifikan.

Kemampuan:
  • Mendukung mode permainan dengan 6-tombol Sega controller;
  • Kemampuan bermain bersama;
  • Menyimpan dan memulihkan game yang disimpan;
  • Panggilan telepon tidak memblokir mode permainan;
  • Kemampuan untuk mengubah pengaturan manajemen;
  • Kemampuan untuk mengubah pengaturan video dan audio;
  • Dukungan untuk arsip dari ROM;
  • Pindai memori utama perangkat dan kartu memori untuk mengidentifikasi gambar yang disimpan.

Program ini diuji pada Droid / Milestone, Xoom, Galaxy S2, Nexus 4, Nexus 7, NVidia Shield, Xperia Tablet Z, dan Xperia Play. Para pengembang menjanjikan gameplay normal pada perangkat lain dengan karakteristik serupa.

Kemampuan:

  • Kemampuan untuk menyimpan game dan mengunduh dari save "on the fly";
  • Dukungan untuk format .bin, .smd, .gen, dan .sms, termasuk arsip;
  • Dukungan untuk gamepad dan keyboard nirkabel.

Sega dreamcast


Reicast



Sega Dreamcast adalah konsol game kelima dan terakhir Sega, penerus Sega Saturn. Sebagai upaya untuk menaklukkan pasar selama generasi keenam sistem game, Dreamcast dirancang untuk bersaing dengan PlayStation dan Nintendo 64.

Reicast adalah satu-satunya emulator untuk konsol game Android ini. Ini tidak sempurna, namun dapat menjalankan sebagian besar game. Persyaratan sistemnya tidak terlalu tinggi, ambang bawahnya adalah Cortex-A9 dual-core dengan frekuensi 1 GHz.

Sebuah emulator muncul pada 2013, pengembang aktif mengerjakan proyek sekarang. Berkat ini, emulator mendukung sebagian besar game yang dirilis untuk Sega Dreamcast. Saat Anda memulai beberapa permainan, artefak grafis muncul di layar. Dilihat dari ulasannya, emulatornya produktif, tidak ada masalah dengan "rem".

Sony Play Station


ePSXe



ePSXe adalah salah satu emulator PC PlayStation tertua. Versi saat ini adalah port untuk Android. Emulator dibayar, harganya $ 3,75. Program ini dianggap sebagai salah satu emulator konsol game terbaik untuk perangkat seluler. Tingkat kompatibilitas yang tinggi dengan game (99% game didukung), kinerja luar biasa. Dimungkinkan untuk mengubah tombol kontrol, pengaturan untuk output video, audio. Joystick dan keyboard eksternal yang didukung terhubung melalui Bluetooth atau USB, seperti WiiMote, Sixaxis, Xbox 360, Moga, Ipega.

Ada kompatibilitas dengan plugin OpenGL, yang memungkinkan Anda untuk meningkatkan grafik dan kinerja gim. Kode cheat yang didukung, simpan. ePSXe secara alami mendukung ARM dan Intel Atom X86. Ini adalah emulator untuk ponsel cerdas dan tablet (1-4 pemain), termasuk gim untuk dua orang di layar yang sama.

FPse bekerja dengan hampir semua gim. Ada dukungan untuk plugin OpenGL, yang memungkinkan Anda untuk menyesuaikan output video untuk berbagai permainan.

FPse adalah program yang lebih kompleks daripada ePSXe, emulator ini memiliki lebih banyak pengaturan. Jika Anda hanya perlu bermain, Anda harus memilih ePSXe. Jika Anda ingin mengonfigurasi perangkat seluler Anda seefisien mungkin, saya sarankan FPse. Bahkan ada dukungan multipemain untuk beberapa game, termasuk Tekken3.

Seperti pada kasus sebelumnya, ada dukungan untuk Xbox 360, PS3, PS4 dan pengontrol game lainnya.

Informasi terperinci tentang prinsip-prinsip bekerja dengan emulator ini dapat ditemukan di sini di tautan ini.. Ini adalah tutorial video dari pengembang.

RetroArch



RetroArch adalah shell lintas-platform gratis berdasarkan pada Libretro API. Lebih dari 40 inti emulator telah diangkut ke libretro. Hanya emulator open source yang porting di sini, terutama di Linux. Tidak ada emulator eksklusif di shell umum. Banyak core hanya bagian dari RetroArch.



Kemampuan:
  • konfigurasi otomatis / pemuatan profil kontrol untuk perangkat input eksternal;
  • kemungkinan akselerasi / deselerasi / mundur;
  • penyesuaian tampilan pengontrol grafis;
  • pembaruan online (kernel, overlay, kode cheat);
  • dukungan shader; bekerja dengan arsip 7-Z / ZIP;
  • dukungan untuk menyimpan cepat, kode cheat;
  • multipemain oleh wi-fi.


Pengembangan telah berlangsung sejak 2013.
