Menggunakan Astra Linux pada Komputer Tertanam ARM


Tren baru di bidang substitusi impor memaksa perusahaan-perusahaan Rusia untuk beralih ke sistem operasi domestik. Salah satu sistem tersebut adalah OS Rusia yang berbasis pada Debian - Astra Linux. Di bidang pengadaan publik, persyaratan untuk menggunakan perangkat lunak dalam negeri dengan sertifikat FSTEC, serta masuknya ke dalam daftar perangkat lunak dalam negeri, semakin banyak dipenuhi. Meskipun perlu dicatat bahwa di bawah hukum ketersediaan sertifikat FSTEC tidak wajib.

Sebagian besar sistem operasi Rusia dirancang untuk digunakan dalam mode "Workstation", yang sebenarnya merupakan analog dari solusi arsitektur x86 untuk tempat kerja karyawan. Kami memutuskan untuk menginstal OS Astra Linux pada arsitektur ARM, untuk dapat menggunakan OS buatan Rusia di bidang industri, yaitu di komputer yang tertanam AntexGate (kami tidak akan mempelajari kelebihan arsitektur ARM lebih dari x86 sekarang).

Mengapa kami memilih Astra Linux?


  • Mereka memiliki distribusi khusus untuk arsitektur ARM;
  • Kami menyukai kenyataan bahwa mereka menggunakan desktop bergaya Windows, bagi orang-orang yang terbiasa dengan Windows, ini adalah keuntungan penting ketika beralih ke Linux;
  • Astra Linux sudah digunakan di perusahaan milik negara dan di Kementerian Pertahanan, yang berarti bahwa proyek akan hidup dan tidak akan mati dalam waktu dekat.

Mengapa kami memilih ARM arsitektur PC tertanam?


  • efisiensi energi dan pembuangan panas yang lebih rendah (perangkat yang didasarkan pada arsitektur ARM mengkonsumsi lebih sedikit energi dan panas yang relatif lebih sedikit selama operasi);
  • ukuran kecil dan tingkat integrasi yang tinggi (sejumlah besar basis elemen terletak pada satu chip, yang menyederhanakan desain motherboard dan menghilangkan kebutuhan untuk membeli sejumlah besar komponen tambahan);
  • redundansi perintah dan instruksi (dalam arsitektur ARM, persis jumlah perintah yang diperlukan untuk pekerjaan disediakan)
  • Tren RF di bidang Internet hal (karena perkembangan teknologi cloud, persyaratan untuk komputer akhir berkurang, tidak perlu menggunakan workstation yang kuat, semakin banyak komputasi bergerak ke cloud, cukup banyak perangkat seperti "thin client").

1. Mendapatkan distribusi AstraLinux


Untuk menerima kit distribusi, Anda harus menulis surat pertanyaan kepada mitra resmi resmi dari RusBiTech NGO. Selanjutnya, Anda harus menandatangani perjanjian kerahasiaan dan non-pengungkapan informasi dan kesepakatan tentang kerja sama ilmiah dan teknis (jika perusahaan Anda adalah pengembang perangkat lunak atau perangkat keras).


Fig. 2 - Catatan Rilis AstraLinux

2. Menginstal AstraLinux pada perangkat AntexGate


Setelah menerima kit distribusi AstraLinux, Anda harus menginstalnya pada perangkat target (dalam kasus kami, ini adalah PC tertanam AntexGate). Instruksi resmi memberitahu kita untuk menggunakan OS Linux apa pun untuk menginstal AstraLinux pada komputer ARM, tetapi kami memutuskan untuk mencobanya pada Windows. Jadi, kami melakukan urutan tindakan berikut:

1. Unduh dan instal perangkat lunak untuk sistem operasi Windows.

2. Sambungkan perangkat melalui Micro USB ke komputer.

3. Hidupkan perangkat, sekarang Windows harus menemukan perangkat keras dan menginstal driver.

4. Setelah instalasi driver selesai, jalankan program.

5. Setelah beberapa detik, drive eMMC muncul di Windows sebagai perangkat penyimpanan USB.

6. Unduh utilitas Win32DiskImager dari halaman proyek Sourceforge dan instal program dalam mode normal.

7. Jalankan perangkat lunak Win32DiskImager yang baru diinstal.

8. Pilih file gambar AstraLinux yang Anda terima sebelumnya.

9. Di bidang perangkat, pilih huruf drive kartu eMMC. Hati-hati: jika Anda memilih drive yang salah, Anda dapat menghancurkan data dari hard drive komputer Anda!

10. Klik "Rekam" dan tunggu rekaman selesai.

11. Mulai ulang perangkat.

Mem-boot ulang perangkat harus menyebabkan perangkat mengunduh gambar sistem operasi AstraLinux dari eMMC.

3. Menggunakan Astra Linux


Setelah perangkat menyala, layar otorisasi akan muncul. Di bidang login, masukkan "admin", kata sandi juga merupakan kata "admin". Setelah otorisasi berhasil, desktop akan muncul (Gbr. 3).


Fig. 3 - desktop AstraLinux

Hal pertama yang menarik perhatian Anda adalah bahwa desktop benar-benar mirip dengan Windows, semua elemen dan dialog dipanggil dengan cara biasa ("Control Panel", "Desktop", "Explorer", "My Computer" di desktop). Yang penting, bahkan Solitaire dan Minesweeper diinstal di Astra Linux!

Perlu dicatat bahwa untuk digunakan sebagai solusi tertanam terdapat akses melalui SSH, melalui konsol Linux, juga dimungkinkan untuk menginstal paket Debian favorit Anda (nginx, apache, dll.). Jadi, untuk mantan pengguna Windows ada desktop yang akrab, dan untuk pengguna Linux yang berpengalaman dan solusi yang tertanam - konsol.


Fig. 8 - Konsol AstraLinux

Optimalisasi AstraLinux


1. Untuk perangkat dengan kinerja perangkat keras yang rendah, kami sarankan untuk menggunakan monitor dengan resolusi rendah, atau secara manual menurunkan resolusi dalam file /boot/config.txt menjadi 1280x720.

2. Kami juga merekomendasikan menginstal utilitas untuk secara otomatis mengontrol frekuensi prosesor: 

sudo apt-get install cpufrequtils

Kami memperbaiki nilai berikut di /boot/config.txt : 

 force_turbo=1

3. Secara default, repositori standar dinonaktifkan di sistem. Untuk mengaktifkannya, Anda perlu menghapus komentar dari tiga baris dalam file berikut cd / etc / apt / nano sources.list
