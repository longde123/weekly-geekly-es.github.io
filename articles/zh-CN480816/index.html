Lamoda上的Dynamics 365和Power Platform聚会-公告

更新资料 广播链接： www.youtube.com/watch?v=kytvkUVJP_8&feature=youtu.be

大家好！ 我叫Stanislav Gots，是Lamoda ERP系统实施和支持的负责人。 我想邀请您参加将于2月11日在我们办公室举行的Dynamics 365和Power Platform聚会 。

五位演讲者将在集会上发言，在讨论部分中，我们将讨论有关ERP实施及其支持的项目管理主题，我们将讨论测试和业务分析的方法，我们将涉及新版本的开发细微差别和技术方面。 午餐和办公室游览也在等待您。

在临时会议计划下，提供指向注册表格的链接以及有关如何找到我们的更多信息。

报告书


“亲身体验如何与Dynamics 365集成。” 拉莫达Stanislav Gotz ：
“ 在我的报告中，我将讨论在项目上与Dynamics 365 F＆O进行集成时，我们采取了什么方法，遇到了哪些问题，Azure API Management如何帮助解决了这些问题 。”

“今天是明天。” Tatyana Gorgul，拉莫达 ：
“我将分享我在Dynamics 365中的开发经验。我将告诉您有关在项目上进行操作时所犯的错误。
让我们谈谈对对象使用模型，扩展和命名约定。
我将向您介绍应用程序的体系结构和更新版本。”


“ Dynamics 365 FO和Power Platform。 如何使最终用户的生活更轻松。” Awara IT业务应用实践主管Ilya Karjakin：
“在我的演讲中，我想谈谈使我们能够快速有效地解决业务运营任务的工具。 我们将从业务角度讨论Power Platform，无需使用繁重的开发工具即可使用哪些工具解决哪些任务。 让我们讨论如何将PowerPlatform服务与Azure服务结合。 我还将根据我们的经验提供统计信息-使用这些工具可以实现哪些财务效果。”


“使用Dynamics F＆O + Microsoft Azure Services + Power Platform的最佳实践。”
Awara IT执行合伙人Alexander Ermakov：
“使用Azure和Power Platform服务来提高Microsoft Dynamics 365业务应用程序的效率是一个单一的生态系统以及创建相关ERP解决方案的实际示例和业务案例。 展望世界以及俄罗斯的技术未来。”
