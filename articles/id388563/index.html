Red Star adalah sistem operasi tertutup dari negara tertutup. Beberapa detail + video



Pada konferensi 32c3 , yang diadakan di Hamburg, beberapa rincian menarik tentang fungsi sistem operasi Korea Utara, yang sudah ditulis di HabrÃ© dan Geektimes, diumumkan. Sistem ini disebut Red Star. Ternyata, para ahli Korea Utara belum menghabiskan begitu banyak waktu mengerjakan OS ini (total sekitar 10 tahun). Terlepas dari kenyataan bahwa sistem ini didasarkan pada salah satu distribusi Linux, adalah mungkin untuk menjadikannya sebagai tertutup dan buram mungkin.

Ini bukan hanya salinan pucat sistem operasi Barat, karena banyak dari Anda mungkin memutuskan. OS itu sendiri dianalisis oleh spesialis dari perusahaan Jerman yang bekerja di bidang keamanan informasi. Menariknya, OS diunduh dari situs di luar Korea Utara (seperti yang Anda tahu, tidak ada jaringan di dalam negeri, hanya ada intranet terbatas).

Versi terbaru Red Star ditulis pada 2013, dan terus digunakan di negara ini. Desain visual dibuat dalam gambar dan rupa OSX - ini mungkin dilakukan atas instruksi Kim Jong-un, yang bekerja pada Mac (biasanya sistem ini di sebelah pemimpin Korea Utara).

Warga Korea Utara menciptakan OS mereka, menjadikan Fedora sebagai basis. Pada saat yang sama, banyak yang "selesai" sendiri, termasuk sistem enkripsi file. Ini dilakukan agar kekuatan eksternal (kita membaca orang Amerika) tidak dapat menemukan celah dalam sistem. Untuk meningkatkan kontrol keamanan, Korea Utara juga telah mengembangkan sistem yang memonitor file. Jika ada perubahan, sistem menunjukkan kesalahan atau crash. Ini terjadi, misalnya, ketika Anda mematikan antivirus atau firewall - sistem tidak memungkinkan Anda untuk mengkonfigurasi parameter sistem dasar. Red Star juga membangun sistem perlindungan terhadap file media pihak ketiga (film, musik, buku). Jika di Korea Utara mereka mengubah data, maka pada flash drive, dan tidak melalui intranet. Dan sekarang, misalnya, menggunakan USB flash drive dengan file asing juga gagal - sistem "melihat2 dan mengendalikan semua ini."Red Star memberi tag atau menandai file media yang mudah dilacak.

Sekarang di Korea Utara, sebagian besar komputer sudah cukup tua, dengan Windows XP on board.



Sistem itu sendiri, menurut para ahli yang mempelajarinya, adalah sebagai buram bagi pengguna mungkin. Semuanya di sini dikonfigurasi untuk mengontrol dan melacak tindakan pengguna. Himpunan aplikasi minimal: editor teks, kalender, program untuk membuat musik dan beberapa program lainnya.

Persyaratan sistem:
  • Pentium III 800 MHz
  • RAM 256MB
  • 3GB hard drive
