一个云的故事：华为+ 3data =云

三年前，我们采用了一项新战略，根据该战略，公司发展计划中的领先地位是云技术。 2018年，华为与3data一起在俄罗斯推出了云服务。 我国是中国以外的第一个以自己的品牌部署云的国家。


3data HP-26数据中心是华为云生活的地方之一

俄罗斯云远非华为使用自己的硬件和软件构建的唯一云。 在中国，华为正在开发三种非常大型的云解决方案：华为自己的公共云，中国移动最大的电信运营商之一的公共云以及用于政府机构的云。 此外，在中国，我们为客户提供的服务范围可与三大西方公司（Amazon，Microsoft和Google）媲美。 其中许多服务已经在德国，法国，西班牙和许多国家/地区运营，但都以当地合作伙伴为品牌。

华为在俄罗斯的市场一直很特别，它开始了我们在中国以外的成功故事。 毫不奇怪，在这里推出了自己品牌的第一个外国云。 应将其视为开发集成到公司许多产品中的云技术解决方案的战略的一部分：针对电信，企业部门，甚至最终用户。

如何开始


当出现考虑在俄罗斯启动公共云的可能性的问题时，我们要做的第一件事是对2016-2017年俄罗斯云服务市场进行深入而全面的分析。 分析结果使我们充满信心，市场是充满活力和动态的，但与此同时，市场还远未达到饱和状态。 出现了许多实际问题，尤其是法律和商业性质的问题：提供此类服务需要哪些许可证，如何最好地获得它们，如何与客户建立正式关系等。



结果，云服务被部署在合作伙伴站点上：3data向我们提供了具有工程基础设施和通信渠道的数据中心，这就是其特征层3。 ，数据存储系统，通信设备以及软件-都是自己生产的。

该设备位于莫斯科，并以“ Huawei 3data Cloud”品牌提供服务：合作伙伴为云提供第一线支持，第二部分提供部分支持。 3data还是与3data签订合同的客户的前台。 自己的数据中心数量已接近两个，该公司完全有能力进行如此大规模的项目。

我们的云软件基于华为FusionCloud解决方案，而该软件基于OpenStack。 使用KVM虚拟化。 值得注意的是，华为是OpenStack项目的八个白金赞助商之一。



我们使用基于KVM的Huawei FusionSphere虚拟机管理程序。 需要特别注意的是，华为是首批为其管理程序获得FSTEC证书的公司之一。

首先推出了什么


俄罗斯的云服务市场不同于中国或欧洲。 它的体积要小得多，并且俄罗斯企业仍然不像西方国家接受云技术。 此外，我们在2017年所做的研究表明，尽管有上升趋势，但高级PaaS级云服务的需求不是很大。 （对于提供商）也存在积极的差异：市场距离饱和还很远，俄罗斯的竞争也较少。 最困难的事情是为部署的第一阶段选择产品。 在第三方承包商的参与下进行了认真的市场分析，在俄罗斯的初始阶段，仅选择了IaaS服务：

  • 虚拟数据中心（VPC）
  • 虚拟服务器（ECS）
  • 块存储（EVS）
  • 支持S3协议的对象存储（OBS）
  • 自动缩放
  • 负载均衡器（ELB）
  • VPN即服务
  • 防火墙即服务
  • 安全小组
  • 专用服务器（裸机服务）和其他一些服务器。




第一个可用区于2018年3月正式开始运营。

我们的云还提供了免费的云基础架构监视服务（Cloud Eye服务），基础架构更改日志记录服务（云跟踪服务）和简单消息通知服务，可显着提高云基础架构的安全性和管理能力。

我们很快就收到了客户要求GPU虚拟化的请求。 特别是对他们来说，这项服务很快就从中国云迁移了。 到2018年底，配备Nvidia Tesla V100 GPU的服务器已经为这些客户所用。 在2019年第二季度，我们计划进一步扩展具有GPU的服务器集群。

Direct Connect服务已证明其价值所在-客户物理数据中心与华为云中虚拟数据中心的暗光纤连接。 在这种情况下，客户端将在其物理网络和云网络之间接收具有高度安全性的IP级别路​​由。

2018年10月，首个PaaS类服务以测试模式启动：

  • RDS，关系数据库服务（数据库即服务）。 现在在云中有可用的产品，例如MySQL，PostgreSQL和Microsoft SQL Server。
  • DCS，分布式缓存服务，它实现了用于在RAM中缓存数据的系统。 您可以选择基于Memcached或基于Redis的服务。
  • DDS，文档数据库服务：基于MongoDB的基于文档的数据库管理系统。

这是截至2019年2月的产品清单：



小型客户通常对基本的IaaS服务感到满意，而中型和大型公司则需要更广泛的服务，包括PaaS：容器管理，构建数据仓库，消息代理。

客户的实际需求如此之大，以至于在2018年第三季度，我们用尽了当时使用的数据中心的功能，并开始寻找进一步扩展的机会。

路线图


华为的俄罗斯云为客户提供了超过两个半的服务，到2019年第一季度末，他们的数量将增加到50个以上。我们计划推出约30种新服务，其中包括：

  • 云容器引擎-基于Kubernetes的容器管理;
  • 弹性文件服务-共享文件存储；
  • 域名服务；
  • MapReduce服务-使用基于Apache Hadoop的大数据；
  • Data Lake Insight-基于Apache Spark的大数据处理和分析；
  • 分布式消息服务-各种类型的高性能消息队列，尤其是Kafka，ActiveMQ等。

此外，在2019年第一季度末，访问区域的数量将增加到三个-计划在三月推出三个新站点，我们将从现有数据中心悄悄地为我们的客户迁移到该站点中。 莫斯科的两个最大数据中心租用了两个站点，第三个站点由我们的3data合作伙伴提供。 由于我们国家/地区的特定业务集中度，我们目前专注于在俄罗斯中部地区站稳脚跟，但是在未来几年中，我们计划将业务扩展到我们国家/地区的其他地区。



在云产品的基础上，将为客户提供完整的交钥匙解决方案：基于Commvault RBaaS的云备份解决方案，以及从基于Hystax Acura的其他环境自动迁移到华为云的解决方案正准备推出。

在开发项目时，我们将在2019年特别注意合作伙伴渠道。 一方面，我们计划扩展我们的销售网络和联合集成服务组合。 其次，我们的目标之一是建立这样一个生态系统，以便俄罗斯技术公司可以基于华为云产品创建自己的解决方案并将其提供给客户。 第三个合作领域是服务合作伙伴，它们将帮助我们为客户提供一些专业服务，例如，在我们的云中启动和支持Oracle DBMS。



人工智能


正如我们已经说过的，应该将俄罗斯项目视为整体背景的一部分。 华为的战略涉及全球云平台的开发，未来，基于机器学习和人工智能技术的服务将被添加到服务列表中。 在中国，现在有大量的产品可供客户使用，即：

  • 文字识别（OCR）；
  • 阅读文字（文字转语音）；
  • 语音识别
  • 音乐识别；
  • 高亮显示视频中的对象等等。

根据我们的战略，华为计划成为人工智能服务在云服务市场的领先者。 2018年10月，为解决AI任务而优化的Ascend系列芯片在上海的Huawei Connect上亮相。 它们可以用作辅助处理器来解决新华为服务器中的特殊任务。 使用12纳米技术创建的Ascend 310已经可用，而更先进的Ascend 910将在2019年第二季度发布。 使用7纳米技术创建的芯片将成为与Nvidia Tesla V100最接近的竞争对手，功耗稍高，它将具有256 TFLOPS的性能。

在俄罗斯出现基于硬件的AI服务器解决方案的确切日期尚不清楚，但这将在2019年下半年左右发生。预计带有Ascend 910芯片的服务器将在我们云的硬件组件中占据重要位置，并且相应的服务将依赖于它们。云计算。

竞争者


在全球范围内，华为云与亚马逊网络服务，谷歌云平台，微软Azure和阿里云竞争，我们为自己设定了成为全球五大云提供商之一的目标。 我们尊重竞争对手，但是我们绝对可以做到这一点。 如今，西方三国提供的服务范围比俄罗斯的华为要广，但我们的俄罗斯云拥有强大的捐助者。 华为在中国的服务已经运营了多年，为用户提供了一百多种服务以及基于它们的数百种合作伙伴解决方案。 这具有特别的价值，因为我们不必从头开始创建新产品，它们从父云的迁移过程仍在进行中。 此外，亚马逊，谷歌和微软没有俄罗斯数据中心，这给参与个人数据处理的客户带来了困难，并大大限制了合作机会。



在国内主要竞争对手中，可以提到的是Mail.ru Cloud Solutions和Yandex.Cloud，但华为在这里有其优势：它包括数千名R＆D员工，自己生产的全套设备（包括服务器，网络设备和数据存储系统）以及所有相同的已开发中文云，可以在存在客户请求的情况下从那里轻松转移新服务。 俄罗斯的云服务市场确实还远未达到饱和状态，每个人仍然有足够的空间，但这不会无限期地持续下去。 另一个全球参与者的出现应加快这一进程。
