Software und Formate


Testen von Player-Programmen für macOS X gegen reguläres iTunes
Interessantes Material, das auf ixbt.com veröffentlicht wurde. Es ist die bemerkenswerteste Software von iTunes bis BitPerfect, Vox und Fidelia - einen Versuch wert, wenn Sie die macOS-Plattform verwenden.

Wie ich einen VHS-Videorecorder für die Tonaufnahme verwendet habe
Eine erstaunliche Geschichte von Stereo.ru darüber, wie man aus einem alten Videorecorder ein gutes Audio-Tool macht. Das Beispiel Grundig GV-464 zeigte, dass eine gesamte Diskographie auf einer einzigen E-240-Videokassette aufgezeichnet werden kann.

Der Kampf um das Format: Rolle gegen Kassette gegen Vinyl gegen CD gegen HiRes
Rückblick auf den „Kampf der Formate“, in dessen Rahmen fünf verschiedene Audioaufnahmeformate vergleichend angehört wurden: Magnetband auf einer Rolle und in einer kompakten Kassette, Vinyl, CD und einer 24-Bit / 96-kHz-HiRes-Datei.

Spotify bietet HiFi-Streaming mit verlustfreiem Musikzugriff
Der Dienst, über den wir in unserem Blog über Geektimes wiederholt gesprochen haben, begann mit dem Testen von Spotify Hi-Fi. Unternehmensvertreter haben diese Informationen bestätigt.

Panasonic wird 2016 in einigen TV-Modellen Unterstützung für das HLG-Format hinzufügen
Der HLG-Standard wurde von BBC und NHK gemeinsam entwickelt, um hybride HDR- und SDR-Signale über Fernsehsender zu übertragen. Das Endergebnis ist der Standard für die Fähigkeit eines Hybridsignals, sowohl HDR- als auch SDR-Informationen gleichzeitig zu übertragen, ohne zusätzlich Bandbreite zu beanspruchen.
