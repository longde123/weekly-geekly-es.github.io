SAP no Microsoft Azure Tour em Moscou

Convidamos você para a etapa russa da turnê mundial dedicada à SAP no Microsoft Azure - um evento no qual ofereceremos uma nova visão da localização das soluções SAP na nuvem do Azure e compartilharemos as melhores práticas de migração.

Novotel Moscow City, Presnenskaya nab., 2, Moscou
