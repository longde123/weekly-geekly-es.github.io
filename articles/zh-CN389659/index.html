2016年预测：以人为本的技术

再次回顾过去一年中实现的预测，我们考虑下一年的前景。在2015年，仅明显的是该技术没有显示出任何发展放缓的迹象。非接触式支付系统（特别是Apple Pay）的泛滥，首批智能手表的出现，以及3D打印机的购买价格越来越便宜-这些只是其中的一些创新。

而且，有关企业数据泄漏和黑客攻击各行各业的新闻也开始越来越频繁地出现……企业领导人一致认为，没有公司，组织甚至国家都可以完全安全，用户也越来越担心通讯可能会丢失个人信息。

那么，2016年将为技术世界做些什么呢？我相信将特别关注个人用户，而不是公司。已经投入了大量资金来开发可带来新商机的技术，但现在该关注普通用户的需求了。新技术和软件不仅将成为全局的一部分，而且还将为未来的创新奠定基础。这就是2016年等待我们的地方。

1.提高网络速度并进入HTTP / 2协议市场


HTTP / 2协议于今年在市场上推出，它将提高万维网上浏览器和服务器之间的信息传输速度。新协议是HTTP / 1.1协议的更新版本，该协议于1997年开始运行。很明显，从那时起，站点已进行了足够的更改。现代用户喜欢类似于FlipBoad的网站，这些网站具有许多GIF，Flash动画，图形等。

2.保护用户，而不是设备


这并不是什么新鲜事物，但是我们几乎已经达到了大多数公司停止为其设备提供保护并重新定位用户安全的地步。BYOD概念将在2016年继续发展，但安全策略将不会与单个设备绑定，而是与所使用的用户，应用程序和数据捆绑在一起。对于公司来说，这是完全合理的步骤，因为 员工将越来越多的设备用于商业目的。

3.修订安全行动


当我们观察到通过网络对Dev-Opps流程的控制越来越多时，我认为在未来，我们将在信息安全领域看到类似的变化。从这个意义上讲，凭借安全和开发技能，他们将开始使用自动化技术来交付软件和基础架构。同样，我们将看到安全分析的使用增加。将用户连接到应用程序时，将检查数据点流以获取更明智的安全决策。这些附加信息不仅可以增强用户的功能，还可以使您更好地了解应用程序使用的原理及其速度。

4.受过教育的用户


每次，我们一次又一次地了解到，无论大小，任何组织都可能成为攻击者的目标。今年，我们目睹了从电信和零售公司到政府机构和物联网的网络攻击数量空前增加。通常，由于这种攻击，用户数据受到攻击。在2016年，用户将更加关注他们传递给公司的信息。

在2016年，我们正在等待世界上许多新技术。但是我们一定不能忘记教给我们的错误以及我们得出的结论，特别是有关在移动模式和网络基础结构中工作的网络安全的结论。借助于现代技术的强大功能，我们周围的世界正在迅速变化。您可以轻松地摆脱创新，而忽略现有风险和用户的实际需求。有必要预见未来，并制定明确的策略，以确保您的业务的技术竞争力。
