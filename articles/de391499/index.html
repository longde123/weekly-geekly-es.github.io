Remix OS für Beta-PC: So organisieren Sie Dual-Boot mit Windows und beginnen mit dem neuen Betriebssystem



Geektimes hat wiederholt über Remix OS geschrieben, ein Android-basiertes Desktop-Betriebssystem. Am 1. März erschien eine Beta-Version dieses Betriebssystems, mit der man sich bereits gut auseinandersetzen kann. Im Allgemeinen handelt es sich hierbei um eine angepasste Version von Android, die wie ein vollwertiges Betriebssystem für einen PC aussieht und funktioniert. Es wurde erstmals im Jahr 2015 mit minimaler Unterstützung für mehrere Geräte veröffentlicht. Im Januar wurde die Alpha-Version von Remix OS veröffentlicht. Die Entwickler stellten sie allen zur Verfügung und eröffneten die Möglichkeit, das System von einem USB-Laufwerk zu starten.

In der Beta-Version erschien Dual-Boot mit der Möglichkeit, zwischen zwei Betriebssystemen zu wechseln - Remix OS selbst und Windows. Sie können das Betriebssystem sowohl auf der Festplatte als auch auf der SSD installieren, was eine relativ hohe Plattformleistung bietet. Jetzt unterstützt das Remix OS-Installationsprogramm für PC die Arbeit mit Windows 7 und höheren Versionen dieses Betriebssystems. Eine Version für den Mac sollte bald verfügbar sein. Wenn das Betriebssystem müde ist, ist das Entfernen noch einfacher als das Installieren.



Updates werden jetzt regelmäßig bereitgestellt. Die Unterstützung für drahtlose Updates ist integriert. Es gibt weitere Neuigkeiten, darunter die Umstellung auf den Linux-Kernel Version 4.4 für eine bessere Kompatibilität mit neueren Prozessoren, die Unterstützung von 1080p-Videos von YouTube und Netflix sowie die Arbeit mit USB-Laufwerken mit bis zu 64 GB und mehr.

Warum nicht? Unterstütze Play Store. Da Remix OS nicht von Google zertifiziert ist, gibt es keinen Standardzugriff auf den Markt. Aber Sie können es hinzufügen, mehr dazu weiter unten. Lassen Sie uns zunächst über die Installation von Remix OS selbst sprechen.



Möchten Sie Remix OS nicht auf einer SSD oder Festplatte installieren? Kein Problem. Sie können mit dem Betriebssystem direkt von einem USB-Flash-Laufwerk aus arbeiten und es so bootfähig machen. Trotzdem ist die Leistung des Betriebssystems in diesem Fall geringer als auf einer Festplatte oder SSD.



So installieren Sie Remix OS auf einer Festplatte oder SSD



1. Laden Sie zunächst die neueste Version des Betriebssystems von hier herunter . Es gibt 32-Bit- und 64-Bit-Builds für die entsprechenden PCs.

2. Entpacken Sie das Archiv und erhalten Sie drei Dateien: ein Lehrbuch mit Anweisungen, ein Installationsprogramm und ISO, ein Disk-Image.

3. Starten Sie Remix OS Installation Tool.exe und überprüfen Sie, ob der Installationsort korrekt ist (Laufwerksbuchstabe usw.).



4. Klicken Sie auf Weiter und wählen Sie das gerade geladene Disk-Image aus. Es wird so etwas wie Remix OS für PC_x86_64-B2016xxxxxx.iso sein.

5. Wir starten die Installation und warten einige Minuten, bis eine Meldung angezeigt wird, dass der PC neu gestartet werden muss, um die Installation abzuschließen.

6. Nach dem Neustart wird ein Bildschirm mit dem Vorschlag angezeigt, zwei Systeme auszuwählen.



7. Wir warten.

Der nächste Schritt dauert je nach Leistung Ihres PCs zwischen 15 und 30 Minuten. Remix OS erstellt einen eigenen Bootbereich von 8 GB und installiert das Betriebssystem in diesem Bereich.

8. Nachdem alles fertig ist, wird Remix OS zum ersten Mal gestartet und Sie können loslegen.

9. Wenn Sie zu Windows zurückkehren müssen, wählen Sie das Symbol zum Herunterfahren und dann die Option zum Neustart. Klicken Sie danach anstelle von Remix OS im Auswahlbildschirm auf Windows.

Ist es möglich, mehr als 8 GB zu bekommen?


Das offizielle Remix OS PC-Installationsprogramm benötigt 8 GB für seine Anforderungen. Dies reicht aus, um verschiedene Anwendungen zu installieren, die Sie möglicherweise für die Arbeit oder Unterhaltung benötigen.



Wenn Sie jedoch plötzlich mehr Platz benötigen, können Sie ihn hinzufügen .

So entfernen Sie Remix OS


Wenn plötzlich etwas schief gelaufen ist oder nur Remix OS entfernt werden soll, kann dies sehr schnell erfolgen.

Führen Sie das Installationsprogramm zunächst erneut aus. Es zeigt, dass das System bereits auf dem PC installiert ist, und bietet an, Remix OS zu entfernen. Wir sind uns einig und nach einigen Sekunden wird das Betriebssystem gelöscht.

Sie können nicht mit dem Installationsprogramm arbeiten, sondern alles manuell löschen. Wählen Sie in der Adressleiste des Dateimanagers C: \ Remix OS \ aus und führen Sie die Datei UninstallRemixOS.exe aus . Dann passiert alles im automatischen Modus.
