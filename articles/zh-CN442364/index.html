Snom M300 Base DECT站和M215套件概述



在秋天，我们讨论了用于创建DECT网络的Snom的M系列设备的整个系列 。 这里描述了一个大型的M700站，但是这样的设备对于小型办公室/商店/咖啡馆和其他小型企业是多余的。 但是在这个利基市场，Snom M300工作站（该文章被遗忘了）和全新的Snom M200新奇之处完美地解决了。

胜诺​​M300


由于Snom M300是M700的“弟弟”，因此比较它们的某些参数是合乎逻辑的。



因此，M300旨在在相对较小的区域中创建DECT网络。 像第700个型号一样，“三百个”的最大室内距离可达50 m，而开放空间的最大距离可达300 m。 但这是在理想条件下，实际上，范围取决于墙壁和地板的材料以及厚度。

同时，M300最多可支持20个Snom M25，M65和M85 DECT管（M700-最多30个）。 同时有20个受支持的SIP帐户，同时M300无法合并到多网状网络中，这与最旧的模型有很大的不同。 如果M700可以成千上万的相互连接，在非常大的范围内建立覆盖范围，并且可以为多达1000部手机提供服务，那么M300就是“孤独的英雄”。 但是，您可以借助M5中继器扩展该站的覆盖范围，该中继器最多可以连接三块。 同时，M300最多可以处理5个呼叫（M700-最多5个宽带或最多10个标准。M300还支持宽带）。



为了连接到以太网，M300配备了RJ45端口。 不会忘记硬件重置按钮。

配置和更新是通过Web界面通过网络执行的。 M300中没有内置的订户簿（在M700中-用于200个订户）。

该站支持所有必要的协议：

  • SIP（RFC3261）
  • DHCP，NTP
  • HTTP / HTTPS / TFTP
  • LDAP

支持的编解码器列表甚至比M700大：

  • G711 A-law，µlaw
  • G722（宽带）
  • G726
  • G729（A / AB包括VAD）

无线DECT加密，TLS，SRTP（RFC3711），SIPS和站管理Web界面的密码保护支持M300的安全性。 Snom M300与所有主要的IP PBX系统兼容，包括基于Broadsoft和Asterisk的系统。

该工作站由5 V电源供电；其总体尺寸为178 x 120 x 30 mm。



此外，可以使用M325套件购买该电台，该套件除了M300本身外，还包括Snom M25 DECT电话。 在专门针对M系列设备的先前出版物之一中对此进行了详细描述。

索姆m215


该套件包括Snom M200SC基站和M15SC DETC管。



M200的功能比M300适中。 最多可以同时连接6个DECT管；它支持相同数量的SIP帐户。 该分机最多可以同时处理4个呼叫。

M200SC的范围是标准范围，在理想条件下最长可达50 m。 它既可以通过电源供电，也可以通过PoE供电。



前面板上有一个用于搜索DECT听筒的按钮：单击它时，电话开始响铃。


背面是RJ45连接器，用于连接10/100 BASE-T网络。


M200SC有一个内置电话簿，其中包含200个号码，与呼叫日志中的条目数相同。 通过受密码保护的Web界面，可通过网络完全配置该设备。 该站支持无线固件更新，具有内部通信（在本地网络内）和三方电话会议的功能。 能够转接和保留呼叫。

M200SC与GAP设备和Asterisk平台兼容。 M200的音质并不比支持音频编码解码器的更高级M300的音质差：

  • G.711 a-law，u-law
  • G.722
  • G.726
  • G.729
  • 白细胞

支持的通信协议列表：

  • SIP（RFC3261）
  • HTTP / HTTPS / TFTP / FTP
  • DHCP，NTP，SNTP，STUN，VLAN
  • RTP / SRTP
  • IPv4 / IPv6
  • LDAP

安全也不是“有偏见的”：200SC支持多种安全工具，包括TLS，AES加密，LLDP-MED，802.1x和802.1pq。

M215随附的M15SC是最简单的Snom产品之一。 这是一种可靠的“主力军”，适用于任何组织，甚至适合在家中使用。


该手机配备了对角线为1.7英寸，分辨率为90 x 65像素的显示屏。 通常使用顶行来放置信息图标，而在显示屏正下方的两个功能键的值指示下给出底行。


小键盘配备了背光灯，电话还配备了一个LED，在接收语音邮件时会亮起。 该电话支持呼叫转移和保持，可让您分配10个号码以快速拨号。



背面有一个可拆卸的夹子，用于将手机挂在皮带上。 右侧有一个2.5毫米的耳机插孔，上面盖有一个橡胶塞。



电池是镍氢电池，容量为550 mAh。 一周的待机时间和长达7个小时的通话时间可能就足够了。







手机的整体尺寸为174 x 49 x 31毫米。
