Actualizar automáticamente la foto de perfil de Vkontakte

Código para actualizar la foto de perfil de Vkontakte utilizando la API de VK. Considere trabajar con captcha y cargue el código en el servidor para actualizar automáticamente la foto.



Bibliotecas requeridas


Instale la biblioteca vk_api para trabajar con la API de VK para Python: pip install vk_api

Ejemplo de trabajo de vk_api 
import vk_api # ( )      vk_session = vk_api.VkApi('login', 'password') vk_session.auth() vk = vk_session.get_api() #       'Hello world!' vk.wall.post(message='Hello world!')


Importa las bibliotecas necesarias 

 import os import time import random import requests import vk_api from vk_api.utils import get_random_id from urllib import urlretrieve

Iniciar sesión


Autorización en VK usando API 

 vk_session = vk_api.VkApi('+7999132****', '*********') vk_session.auth() vk = vk_session.get_api()

Subir imágenes


Suba imágenes al servidor VKontakte.
photos.getOwnerPhotoUploadServer () devuelve la dirección del servidor para cargar la foto principal en la página del usuario. 

 url = vk.photos.getOwnerPhotoUploadServer()['upload_url']

Para cargar la imagen en el servidor VKontakte, debe transferir el archivo a la dirección upload_url obtenida en el párrafo anterior generando una solicitud POST con el campo de foto. 

 #    images images = os.listdir("images") photo = [] for image in images: request = requests.post(url, files={'photo': open('images/'+image, 'rb')}).json() photo.append(request['photo'])

Guarde el servidor y los valores hash. Los necesitará para actualizar su foto de perfil. 

 server = request['server'] hash = request['hash']

Actualización de foto de perfil


photos.saveOwnerPhoto () guarda la foto del usuario. 

 #       x = random.randint(0, len(photo)-1) vk.photos.saveOwnerPhoto(server = server, hash = hash, photo = photo[x])

Después de actualizar la foto, se agrega una publicación con esta foto al muro. Si no elimina estas publicaciones, el feed de amigos se obstruirá con publicaciones sobre la actualización de su avatar. 

 posts = vk.wall.get() post_id = posts["items"][0]["id"] vk.wall.delete(post_id = post_id)

Si ejecutamos el código, la foto de nuestro perfil se actualizará.
Para la automatización, solo agregue un bucle sin fin y haga un minuto de retraso después de cada actualización de fotos.

Captcha: se necesita Captcha

Veamos cómo trabajar con captcha en VK API.

Trabajar con captcha


El método vk_api.VkApi () ya ha implementado el trabajo con captcha. Además del inicio de sesión y la contraseña, es necesario transferir la función de procesamiento captcha captcha_handler.

Cambiar vk_session 

 vk_session = vk_api.VkApi('+7999132****', '**********', captcha_handler=captcha_handler)

Agregamos la función captcha_handler (captcha), que toma la dirección captcha, envía la imagen captcha a los mensajes del usuario y espera el captcha del usuario. 

 def captcha_handler(captcha): #   captcha_url = captcha.get_url() #     urlretrieve(captcha_url, "captcha.jpg") #  (send_captcha() . ) key = send_captcha(captcha_url) #  ()  return captcha.try_again(key)

Función para enviar mensajes con una imagen captcha al usuario.

El usuario no puede acceder a los métodos de mensaje desde el servidor, por lo que debe crear un grupo y obtener un token.

  2. Gestión => Trabajar con API => Crear una clave
  3. Elegir:
    • Permitir que la aplicación acceda a mensajes de la comunidad
    • Permitir el acceso de la aplicación a las fotos de la comunidad.
  4. Copia la llave
    También debe habilitar Mensajes en la configuración del grupo (Control => Mensajes) y habilitar mensajes (en el menú del grupo)
