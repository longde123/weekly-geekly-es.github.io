Yota - atau bagaimana Anda bisa mengetahui semuanya

Semua tindakan dalam artikel ini hanya untuk tujuan informasi. Semua karakter karya itu fiktif, kebetulan apa pun dengan orang sungguhan adalah acak.

Pendahuluan


Semuanya berawal dari fakta bahwa saya membaca artikel-artikel ini tentang Megaphone.

  1. Bagaimana Megaphone tidur di langganan seluler
  2. "Konten seluler" gratis, tanpa SMS dan pendaftaran. Detail penipuan megafon

Saya adalah pelanggan perusahaan ini selama 12 tahun, saya tidak pernah secara pribadi menangkap lelucon ini dan apa yang saya baca menuntun saya ke keadaan bahwa saya memutuskan untuk mengganti operator.

Pilihan saya tidak hebat ( MTS, Beeline, Tele2 ), tetapi saya menyadari bahwa perusahaan-perusahaan ini berdagang persis sama dengan perusahaan yang disebutkan di atas. Dan kemudian saya sadar bahwa ada perusahaan dan namanya adalah Yota .

Tetapi Anda berkata, bagaimanapun juga, Yota adalah milik Megaphone dan saya akan menjawab, biarkan lingkaran setan itu dimulai.
Semuanya baik-baik saja, saya datang, saya membeli kartu SIM dan mulai menggunakannya.

Semua untuk apa kita di sini


Saya perlu mengubah nomor telepon di akun pribadi Russian Post
Dan saya mengalami masalah bahwa nomor ini (dibeli dari Yota ) sudah diambil.



Saya punya pertanyaan - mengapa begitu? Dan saya ingat bahwa dengan Yot Anda dapat dengan mudah mengubah nomor telepon tepat di aplikasi. Dan mungkin mantan pemilik nomor saya hanya lupa untuk "melepaskan" dia.

Cuplikan layar dari aplikasi Yota


Dan saya tertarik dengan keingintahuan untuk kegiatan sosialisasi yang buruk , dan jika saya bisa masuk ke akun orang yang memiliki nomor telepon ini dilampirkan.

Dan saya dengan mudah menerima kode konfirmasi dan dapat memasukkan kata sandi baru.
