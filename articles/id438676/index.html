Plugin 3CX Webinar untuk Situs WordPress Anda

Memperkenalkan Plugin WebCars 3CX Baru untuk WordPress!

Plugin ini memungkinkan Anda untuk mempublikasikan daftar webinar di situs Anda yang dibuat di klien web 3CX . Pengunjung ke situs mendaftar untuk webinar melalui formulir online plugin. Tampilan plugin dapat dikustomisasi melalui templat (Javascript dan HTML). Webinar sendiri diadakan di platform WebMeeting, diintegrasikan ke dalam PBX 3CX.

Kami mencantumkan manfaat plugin:

  • Registrasi cepat - pengunjung situs melihat daftar webinar dan mendaftar dengan cepat menggunakan nama dan email tempat tautan ke koneksi datang.
  • Koneksi yang mudah - peserta yang diundang terhubung ke webinar menggunakan tautan yang membuka jendela browser dengan antarmuka konferensi. Teknologi konferensi WebRTC terintegrasi ke dalam browser, sehingga para peserta tidak perlu menginstal aplikasi tambahan.
  • Konferensi WebMeeting yang Luas: Rekaman Percakapan, Flipchart, Berbagi Layar, Presentasi, dll.



Sudah cukup bagi pengelola webinar untuk menjadwalkan webinar di antarmuka klien web 3CX . Dan pengunjung sumber daya Anda dapat mendaftar, menerima pengingat dan terhubung ke acara dari mana saja di dunia. Namun, Anda dapat menawarkan webinar tertentu hanya kepada pengguna situs yang sudah terdaftar sebelumnya. Penyelenggara juga dapat melacak pendaftaran peserta di bagian "Jadwalkan Konferensi" Klien Web 3CX.

Manfaat 3CX WebMeeting untuk Webinar


Konferensi video dan webinar telah memantapkan diri mereka sebagai alat e-learning. Menggunakan 3CX WebMeeting untuk pelatihan teknis, perusahaan dan lainnya memberikan penghematan biaya yang signifikan Itu tidak memerlukan berlangganan dan pembelian peralatan, kecuali untuk webcam dan headset. Cukup membuat webinar di klien web 3CX v16 Pro dan informasi tentangnya akan segera ditampilkan di situs Anda. Rekaman konferensi internal semakin meningkatkan nilai solusi, memungkinkan Anda untuk kembali ke webinar sebelumnya dan mengarsipkannya untuk penggunaan di masa mendatang.



Instal dan konfigurasikan plugin


Untuk menginstal plugin, buka antarmuka manajemen WordPress CMS dan buka Plugins> Add New . Cari plugin 3CX Webinars dan klik Install Now .



Lalu pergi ke antarmuka manajemen 3CX (v16 dan lebih tinggi) di bawah Pengaturan> WordPress> Plugin untuk Webmeeting / webinar dan klik Hasilkan untuk membuat token API Webmeeting baru untuk mengesahkan permintaan plugin untuk menerbitkan daftar webinar di situs. Klik Perlihatkan untuk melihat token API dan Salin untuk menyalinnya ke clipboard.

Membuat objek formulir Webinar


Setelah menginstal plugin di situs, buat objek formulir Webinar. Mereka digunakan pada Halaman dan Blog Entri tertentu atau di seluruh situs untuk menempatkan tabel daftar webinar terjadwal.

Untuk membuat objek, di panel admin WordPress, buka bagian 3CX Webinar , klik Tambah Baru dan tentukan nama objek.



Di bidang FQDN WebMeeting, tentukan FQDN dari server WebMeeting, seperti yang ditunjukkan di sisi kanan antarmuka manajemen 3CX di bagian Beranda.

Parameter berikut digunakan untuk menyesuaikan tampilan Formulir:

  • Token API WebMeeting - masukkan token yang dihasilkan di bagian antarmuka 3CX Pengaturan> WordPress> Plugin untuk Webmeeting / webinar .
  • 3CX Extension Number - tentukan nomor ekstensi dari pengguna 3CX yang akan menerbitkan daftar webinar.
  • Negara / bahasa webinar - pilih bahasa / negara webinar.
  • Batasi pelanggan untuk - jumlah maksimum peserta (tempat) yang dapat mendaftar untuk webinar.

Memposting formulir webinar di situs WordPress


Untuk menempatkan formulir webinar di tempat yang tepat di situs, salin kode pendek yang sesuai dari daftar objek formulir Webinar.

Penempatan pada semua halaman situs: pergi ke Penampilan> Widget dan seret widget Teks ke area widget yang muncul di seluruh situs, misalnya, di bagian bawah (footer) dari tema WordPress. Kemudian tambahkan kode pendek, misalnya, [3cx-webinar id="9" title=" "] di area konten dan klik Simpan.



Posting pada halaman yang dipilih: buat Postingan atau Halaman Blog, lalu tambahkan kode pendek ke area konten. Klik Pratinjau Perubahan untuk melihat halaman dengan formulir webinar yang disematkan.
