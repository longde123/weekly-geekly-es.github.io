小米Mi5-开箱，基准



小米最新推出的智能手机是Mi5，白色版本拥有32 GB的内存（玻璃版本，是的，最简单的版本）！

特点



对于那些不遵循新产品的人：

  • 处理器：高通Snapdragon 820 1.8 GHz，4核
  • 视频：Adreno 530，频率510 MHz
  • 屏幕：5.15英寸1920x1080,428 PPI
  • 内存：3 GB LPDDR4
  • 储存空间：32 GB，UFS2.0
  • 主摄像头：16百万像素，4轴稳定
  • 前置摄像头：4百万像素
  • 蜂窝：2个NanoSIM，支持TD-LTE（通道B38，B39，B40，B41）
  • 无线协议：802.11 a / b / g / n / ac（2.4和5 GHz），蓝牙4.2，NFC
  • 导航：GPS，AGPS，格洛纳斯，北斗
  • 电池：3000毫安时
  • 其他功能：物理“主页”按钮，指纹扫描仪
  • 案例：金属框架，背面为圆形玻璃
  • 尺寸：144.55 x 69.2 x 7.25毫米
  • 重量：129克


选件：手册，2.5 A充电器，USB-USB Type-C电缆，保修卡，用于打开SIM卡隔层的固定夹。

芯片概述


指纹扫描仪工作正常。要注册一根手指，您需要大约12次触摸。解锁几乎是即时的。有些人担心要进行激活，就必须上下滑动手指-这一点都不正确，只需触摸一下就可以了。的确，如果在注册过程中手指保持非常笔直的12次，那么当以45度或更大的角度触摸时，手指可能无法识别。一般来说，工作非常舒适。



其他


一个令人不愉快的惊喜是，在30分钟内，我无法通过电缆将其连接到Windows 10 PC。计算机顽固地表示“无法识别该设备”，重新启动计算机，电话也无济于事，只有一条USB Type-C电缆。无奈之下，他试图将手机连接到MacBook，然后他连接了。之后，连接到Windows-一切开始按预期进行。希望它是新固件的重要组成部分。

结论



它可以很好地放在手中，尤其是在Galaxy Note 3之后的情况下，它的外壳很轻但很滑-最好使用它。外观可能会引起争议：一个同事称他为iPhone和三星的“私生子”。指纹扫描仪效果很好，锁被快速移除且没有错误。没有关于速度的抱怨。相机的表现尤其出色：与Mi4不同，一切都可以立即发生。性能下降仅在40-50帧后的连拍模式下开始-由于缓冲区溢出，降低了拍摄速度。否则，一切都很好。

当前版本没有Google Play和欧洲语言，但是在巴塞罗那进行了演讲并在Svyaznoy网站上提示后，人们可以希望该固件的“国际”版本。

是的，价格不免让人欣喜：在中国，初始型号的价格为1999元人民币（约305美元）。
