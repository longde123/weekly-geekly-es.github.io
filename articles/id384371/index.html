Windows 10 build 10547 tersedia untuk orang dalam di PC

Dalam posting sebelumnya tentang build baru untuk Windows 10 seluler, survei dilakukan pada apakah GT membutuhkan berita tentang build baru pada PC, hasil pemungutan suara ini adalah penampilan posting ini.
