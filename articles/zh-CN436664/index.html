.NET社区站立系列介绍

我们喜欢我们的.NET社区，每个社区都有许多出色的开发人员，他们每周撰写出色的博客，库，演示文稿和请求请求。 我们一直在寻找突出这项出色工作的方法，并且四年多来，Microsoft 的ASP.NET团队一直在YouTube和Twitch上直播他们的ASP.NET社区演讲。

在活动期间，他们将展示团队最新和最伟大的贡献，以及团队一直在做的所有出色的开源工作。 随着.NET社区的扩展，社区支持也应扩展，这就是为什么我们很高兴地介绍其社区支持的扩展，我们将其正式称为“ .NET社区支持”系列。 这些社区代表在.NET领域中跨越多个团队和产品，并展示了社区正在做的出色工作。

社区站立


每周我们都会覆盖.NET几个不同领域的内容。 这是我们的日程安排，以及一些即将到来的站立式演讲将涉及的主题：


  • 每个星期二：ASP.NET-ASP.NET Core，SignalR，Blazor和MVC
    • 太平洋时间上午和晚上之间交替（通常是太平洋时间上午10:30和太平洋时间下午3:30）
  • 第一个星期四： 移动 -Xamarin，移动Azure，.NET库和绑定
    • 太平洋时间上午9:30开始
  • 第二个星期四： 语言和运行时 -.NET Core，C＃/ F＃/ VB，.NET Framework和MSBuild / CLI
    • 太平洋时间上午9:30开始
  • 第三周四： 工具 -Visual Studio和Visual Studio for Mac，VS Code，Roslyn和生产力
    • 太平洋时间上午9:30开始
  • 第4个星期四： 云 -Azure，AI，机器学习，容器等
    • 太平洋时间上午9:30开始

在哪里看


这些节目都在Twitch和YouTube上直播，这是全球开发人员与Microsoft的.NET团队进行交互的一种好方法。 您可以在两个频道上设置提醒，以在直播开始时得到通知。 如果您真想念他们，请放心，您可以随时在YouTube上的 .NET Community Standup存档中按需观看它们。
