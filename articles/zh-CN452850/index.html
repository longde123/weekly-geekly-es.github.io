卡匣内的系统：工程师如何扩展游戏机的功能


在上一篇文章中，我们讨论了三个具有有趣功能的墨盒：它们有一个连接器，其他墨盒都插入其中。 其他一些盒带通过允许将各个视频游戏机插入其控制台连接器来发展这种想法。 这些不是普通的硬件添加：它们需要自己的墨盒，这些墨盒与主系统不兼容。

今天，我们看了来自三个截然不同的制造商的三个截然不同的“墨盒内控制台”。 所有这三个系统的出现都有所有技术和战略原因，它们可以告诉我们一个有趣的故事。 让我们从最神秘的系统开始-这是未经许可且不是由控制台创建者生产的三个系统中的唯一一个。

阿拉丁甲板增强剂


在1990年代初期，Codemasters办公室一定有一个令人印象深刻的研发实验室。 该公司最初的创建目标是为Commodore家用计算机开发游戏，但是几年后，它开设了一个部门，为Nintendo Entertainment System（NES）创造无牌游戏和硬件附件。 至少在游戏机玩家中，该公司最著名的产品可能是Game Genie 1作弊设备，但事实证明，这并不是NES的无牌设备的唯一例子。

视频游戏卡带最惊人的功能之一就是其卓越的多功能性。 在技​​术开发过程中，盒式磁带变得更强大，并补充了控制台本身的功能。 NES设备是在1983年中期开发的，到90年代初，他已经需要所有可能的帮助。 后来，NES盒中出现了各种组件，包括RAM，用于切换存储库的控制器和特殊的硬件映射器2 。

在用于不同游戏的某些弹药筒内，有相同的设备，唯一的区别是存储在ROM芯片上的游戏数据。 Codemasters联合创始人Richard Darling注意到了这一点，并提出了一个绝妙的主意，为NES创建一个附加组件，使游戏更容易，更便宜。 这个想法的结果是1992年11月发布的Aladdin Deck Enhancer。 增强器是一个NES卡式盒，其中包含游戏数据以外的所有内容。 它具有用于连接所谓的“紧凑型墨盒”（Compact Cartridge）的连接器。 这些紧凑型墨盒应该比普通的NES墨盒小得多，并且仅包含记录在ROM芯片中的游戏数据。


紧凑型墨盒上的冒险家头昏眼花。

Codemasters希望Enhancer可以成为有兴趣规避任天堂官方许可计划并使用紧凑盒带发行游戏的发行商自己的可行目标平台。 由于尺寸更小，而且常规弹药盒所需的大多数电路已经直接内置在Enhancer中，因此紧凑弹药盒的售价可能比普通NES游戏便宜得多。 考虑到它是在NES（SuperNES）的后继产品上市后发布的，很明显，Enhancer的目标受众仅限于资金购买者，他们不仅对新技术感兴趣，而且对游戏也更加感兴趣。

对于Codemasters来说不幸的是，这个想法从未成功。 尽管承诺在1993年底发布24款游戏，但仅发布了7款游戏，而且数十年内仍有许多未售出的设备库存。 令人难以置信的是，即使在发行25年后的3年，仍可以在Internet上找到未打开的全新增强器。

超级游戏男孩


超级游戏男孩是超级任天堂娱乐系统（Super NES）的补充，为游戏男孩提供了绝佳的玩游戏机会。 Super Game Boy的大小不超过Super NES的常规墨盒，在顶部边缘有一个连接器，可以插入标准Game Boy游戏。 通过插入Game Boy盒带并将Super Game Boy连接到控制台，玩家第一次有机会在电视上玩自己喜欢的游戏。 鉴于Game Boy因其颗粒状，模糊和暗淡的屏幕而臭名昭著，这似乎是一个了不起的改进。

现代玩家习惯了旧游戏与新系统的兼容性，但在90年代，这个想法还很新鲜4 。 90年代的游戏机功能不足以用于其他系统的软件仿真，因此Super Game Boy墨盒实际上包含了常规Game Boy的所有内部设备。 尽管事实上他只是一个墨盒中的Game Boy控制台，但Super Game Boy还是有一些窍门。


超级游戏男孩的大金刚（Donkey Kong）具有特殊的镜架。

用户必须在大型电视屏幕上而不是在手持设备的显示器上玩Super NES游戏，因此其分辨率高于Game Boy游戏。 这意味着在电视屏幕上显示Game Boy游戏时，其图像应已插入到框架中。 玩家可以选择许多内置装饰框架之一，也可以使用虚拟绘图工具创建自己的框架。 此外，玩家可以在游戏中添加颜色，用自定义调色板替换Game Boy颜色的单色阴影。 某些后来的Game Boy游戏甚至发布了它们自己的内置框架，并可以使用Super Game Boy硬件功能为游戏显示更多颜色5 。 不幸的是，对于游戏玩家来说，这样的改进没有扎根，但是那些仍然发布的改进看起来令人惊叹6 。

超级游戏男孩（Game Game）于1994年发布，距游戏男孩（Game Boy）生命周期开始五年，而超级NES（Super NES）发布三年。 在许多方面，《超级游戏男孩》的发行都是非常明智的举动。 首先，他极大地扩展了Super NES可用的游戏库，并立即引入了三百种其他游戏的访问权限7 。 其次，他再次提请注意失败的Game Boy，那时Game Boy似乎已经过时了。 任天堂并没有迫不及待地想给Game Boy施加第二次机会：1996年在日本发布了第一款Pocket MonstersPokémon游戏，引起了整个流行文化现象。到1990年代末，任天堂又发布了Game Boy 8控制台的另外三个版本。 最终，Game Boy系列已成为历史上最畅销的视频游戏系统之一9 。

《超级游戏男孩2》于1998年在日本发布。 他更新了自己的风格，并与90年代后期的“透明塑料”时尚相对应，并且进行了许多改进。 超级游戏男孩的第一个版本与原始便携式游戏机的CPU速度不完全匹配，因此游戏的运行速度提高了2.4％。 在《超级游戏男孩2》中，此错误已得到修复。 此外，还添加了一个Game Boy链接电缆连接器，该连接器允许两个与第二个控制台10一起玩。

总体而言，超级游戏男孩可能是最成功的“弹药筒系统”。 任天堂在2003年使用相同的想法发行了Game Boy Player，这使其可以玩Game Boy Advance，Game Boy Color和第一个Game Boy的GameCube游戏。

世嘉32X


在我们考虑的三个系统中，Sega 32X最不像传统的游戏墨盒。 插入Sega Genesis墨盒连接器的32X为该逐渐老化的系统增加了显着的图形功能。 不难想象，世嘉的设计师们试图通过涡轮增压器在买家协会中唤起一辆强劲的油车，尤其是考虑到该系统的主要目标受众-十几岁的男孩11 。

与前两个系统不同，32X不是简单的即插即用设备。 在连接32X之前，玩家必须在Genesis内安装金属钉书钉。 然后，您必须将32X连接到电视（不是Genesis），然后用另一条视频电缆将32X连接到Genesis。 最后，32X有自己的电源，电源插座伸出了一个大型变压器。 连接整个系统后，Genesis / 32X设计的后部杂乱无章的电线12 。

32X于1994年发布，距Genesis发行已经超过五年了，当时Genesis受到来自任天堂和其他具有强大下一代控制台的竞争对手的越来越大的压力。 尽管下一代Sega控制台几乎已准备好发布，但它将以下一代价格出售，是Sega Genesis 13当前价格的四倍以上。

Sega希望有一个巨大的潜在市场，希望希望以相对少量的价格购买到更接近下一代游戏的消费者14 ，而且这种策略似乎行得通。 但是，在日本于北美发行Sega 32X的第二天，便出现了一个真正的下一代游戏机，称为Saturn。 吸引开发者同时支持两个新系统的尝试是一项艰巨的任务，消费者对下一代游戏机“半步走”的需求每次都下降了，因为下一代游戏机已经非常接近了。 这给世嘉带来了压力，并使土星在计划推出之前（在32倍飞行后仅六个月）就在北美推出。 那是一场灾难。

但是，我们很容易想象32X将会成功的世界。 如果32X发行得较早，或者Saturn发行得较晚，则32X可能有时间达到流行程度15 。 即使考虑到Genesis的价格，32X的价格仍仅为土星的一半，并且在硬件改进方面可以为玩家提供很多东西。 不幸的是，32X没有成为Sega即将成功的基础，反而预示着Sega逐渐退出游戏机开发市场。

附件补充


弹药筒连接器用于游戏，而不是用于系统，但是勇敢的工程师仍然扩大了可能性的范围，并在此基础上创造了令人印象深刻的设备。 从盒式磁带到光盘，再到数字内容下载，这意味着盒式磁带插槽的时光飞逝。

硬件的增加也出现在现代视频游戏的历史中-例如索尼，请记住，但是其使用的复杂性并未改变。 附加组件始终是轮盘赌：它们可能共同的市场受到主系统用户的现有用户的限制，并且当附加组件发布到市场上时，参与者通常已经在等待下一代技术。 大多数附加组件都不会产生重大影响。 但是只要玩家有很大的优势，他们就可以 “投篮”-超级游戏男孩的成功证明了这一点。
