Forkney it: 8 proyectos Go que son interesantes para cavar en el código fuente

Nosotros en el Distrito Binario nos estamos volviendo intensivos en Go. En dos días, puede aprender a comprender la ideología y el interior de Go, escribir código limpio y comprobable. Pero este es el primer paso, y luego, practicar, practicar, practicar: leer el código de otra persona, escribir y editar el suyo.

El código abierto es ideal para esto:

  • El código abierto aumenta la visibilidad. Puede estudiar de forma no selectiva la fuente.
  • El código abierto ofrece un incentivo para escribir código lo más limpio posible y documentar la lógica.
  • Contribuir al movimiento de código abierto mejora el karma.



Junto con Sergey Tolsi, un graduado de nuestro curso Go, hicimos una lista de proyectos de código abierto de varios niveles y orientaciones que son adecuados para aprender a escribir un buen código y al mismo tiempo hacer del mundo un lugar mejor.

Estrategia Tolsi :
“Si desea utilizar el código abierto para la capacitación, primero decida sobre un área específica de trabajo y establezca un objetivo, por ejemplo, hacer un sitio web o un juego, algún tipo de servicio, y luego encuentre un proyecto similar o un marco con ejemplos. Por ejemplo, trabajo en el campo de las criptomonedas y descubrí que están escribiendo en Go una implementación de la red Lightning para acelerar las transferencias en la red Bitcoin y el nodo más estable para Ethereum es geth ".

1. Docker


Comencemos con uno de los proyectos más complejos y voluminosos conocidos más allá de la Comunidad Go. Desde 2013, Docker se distribuye libremente bajo la licencia Apache 2.0 y es un sistema para crear máquinas virtuales aisladas con cualquier entorno necesario para probar aplicaciones. Su componente clave, libcontainer, es el estándar para este tipo de sistema. La mejor implementación del "sandbox", por ejemplo, no se encuentra.

2. Kubernetes


Otro proyecto interesante, que apareció a mediados de 2014, está estrechamente relacionado con Docker: Kubernetes , software para la gestión grupal de contenedores en una gran flota de dispositivos. Es posible que ya hayas tratado con él, pero ¿has mirado bajo el capó de este proyecto?

Si estudia Go, vale la pena hacerlo solo porque Kubernetes fue escrito por programadores de Google, y entre los contribuyentes del proyecto hay especialistas de RedHat, IBM y Microsoft. El código base de Kubernetes se proporciona con una buena guía para colaboradores de Kubernetes , y puede chatear con el equipo de desarrollo en Slack durante las sesiones regulares de preguntas y respuestas .

3. Sincronización


Syncthing es un proyecto bien diseñado sin problemas técnicos notables y errores críticos, creado teniendo en cuenta las prácticas de Effective Go .

Esta implementación de un sistema de intercambio de archivos distribuido como BitTorrent Sync se basa en el Protocolo de intercambio de bloques (BEP) de código abierto.

En comparación con Kubernetes, Syncthing tiene una arquitectura simple que consta de solo tres capas principales. El código del programa corresponde a la biblioteca Go estándar y lo combina con LevelDB y varias otras bibliotecas. Se utilizan para cifrar conexiones de red, calcular hashes y almacenar datos. Comprender su código base ayudará a la guía desde el sitio oficial y un foro animado .

4. G3N


¿Qué pasa con el motor 3D creado por programadores para programadores? Este título lo posee el G3N , y también está completamente escrito en Go. El motor es adecuado para juegos, pero se posiciona como una herramienta para escribir aplicaciones interactivas.

5. GoLearn


El aprendizaje automático y Python son casi sinónimos, pero Go también es útil para los científicos de datos. La biblioteca de aprendizaje automático simple y flexible de GoLearn está escrita en este idioma. Utiliza el shell Gonum BLAN y admite algoritmos básicos de aprendizaje automático, que incluyen: método de vector de soporte, regresión lineal, redes neuronales, bosque aleatorio, árboles de decisión y un ingenuo clasificador bayesiano.

6. Gorgonia


La solución de aprendizaje automático de bajo nivel de Go, Gorgonia , también es notable. Esta biblioteca, que recuerda el concepto de TensorFlow , utiliza el concepto de representar ecuaciones en un gráfico y está empaquetado en un solo archivo binario.

7. acra


Estudiar la base de código de acra , una herramienta de cifrado y almacenamiento seguro de bases de datos diseñada para microservicios en plataformas móviles y en la web, puede ser una experiencia interesante. Esta es una oportunidad para aprender código verdaderamente seguro y aprender de la experiencia cómo detectar y proteger contra fugas de datos, integrar la criptografía en un proyecto.

Además, la biblioteca del cliente AcraWriter está disponible no solo en Go, sino también en Ruby, Python, NodeJS, iOS (Objective-C / Swift), Android Java y PHP : puede comparar implementaciones y aprender algunas de las sutilezas del software de portabilidad en la práctica.

La última versión de la documentación y tutoriales de Acra está disponible en el servidor oficial de Cossack Labs .

8. ConnectorDB


ConnectorDB es adecuado si no solo desea aprender el código de otra persona, sino también hacer algo útil para usted personalmente. Esta es una plataforma de código abierto para desplegar su propio segmento de Internet de las cosas en función de los dispositivos que ya están disponibles. Agrega datos de computadoras, teléfonos inteligentes de navegadores y los procesa.

Una vez que comprenda el código, puede escribir nuevos módulos y conectar casi cualquier dispositivo a ConnectorDB, configurar el monitoreo de su propia vida, comenzando por monitorear la actividad física y terminando contando la productividad y evaluando el estado de ánimo.

Y también ...


Un lugar ideal para comenzar su búsqueda es la lista Awesome Go , donde la comunidad recopila enlaces a las mejores bibliotecas y aplicaciones de código abierto en Go. Además, el Wiki es útil para repositorios de idiomas. Tiene listas de artículos , cursos y libros potencialmente útiles .

Al analizar el código del proyecto y contribuir, puede desarrollar las habilidades adquiridas y actualizar el conocimiento. Existe la posibilidad de involucrarse en el movimiento de código abierto, y solo hay una forma de averiguar qué tan profundo y largo. Y para aquellos que van a comenzar con lo básico y desean unirse de inmediato al desarrollo, el 8 y 9 de diciembre se llevará a cabo una práctica programación de intensidad en Go en el Distrito Binario.
