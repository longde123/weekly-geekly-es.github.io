La antena Biquad Yagi




Extraño a primera vista, el trabajo casero del modelista inglés no es más que una antena direccional de Yagi . El autor es ampliamente conocido en círculos de guías de drones extranjeros gracias a videos detallados sobre la efectividad de varias antenas. Este proyecto está dirigido a la operación simultánea en dos frecuencias: 2.4 y 5.8 GHz (control y transmisión de video). El diseño promete una ganancia de aproximadamente 18 dB. Muy buena antena para vuelos quadrocopter de largo alcance.

Whoa board




Una plataforma abierta para la creación de prototipos de dispositivos electrónicos portátiles utilizando componentes electroluminiscentes. El conjunto se basa en la placa del mismo nombre, en la que se colocan microUSB para la comunicación con una computadora, un interruptor y 4 conectores para conectar cintas, placas y sensores electroluminiscentes. Dependiendo del propósito del proyecto, pueden actuar como sensores o indicadores luminosos.

El trabajo con periféricos se realiza en 4 canales independientes y no requiere software ni hardware adicional. Para una mayor flexibilidad, Whoa Board es compatible con Arduino , admite el trabajo con UART, SPI e I2C. Y a través de microUSB puede emitir una señal MIDI. Los códigos fuente del proyecto están disponibles en GitHub .

Teensy 3.5 y 3.6




La continuación de la línea bastante conocida de plataformas compactas en el Cortex M 4, con soporte para Arduino IDE y bajo consumo de energía. A bordo hay:

  • 62 pines de E / S;
  • 25 entradas analógicas / 2 salidas;
  • 20 PWM;
  • microUSB (12 Mbps);
  • Ethernet (100 Mbps);
  • microSD;
  • I2S;
  • 6 puertos seriales;
  • 3 SPI;
  • 3 I2C;
  • reloj en tiempo real

La versión 3.5 está equipada con un procesador de 120 MHz y funciona con 5 V. La versión 3.6 tiene un procesador de 180 MHz más potente, pero requiere una potencia de 3,3 voltios. Ambas opciones son compatibles con las generaciones anteriores del proyecto y (según el desarrollador, por supuesto), le permitirán implementar proyectos existentes.

Respeaker




Otro proyecto interesante para desarrolladores de electrónica inteligente y entusiastas del bricolaje: una plataforma para desarrollar dispositivos domésticos controlados por voz. En combinación, un controlador IoT que controla los otros componentes de una casa inteligente.

La principal diferencia entre ReSpeaker y muchos competidores es su plataforma abierta y soporte para todas las API de reconocimiento de voz más populares. Además, el dispositivo puede funcionar tanto en línea a través del módulo Wi-Fi (MT7688) como en modo fuera de línea. En el primer " listo para usar ", el trabajo está disponible con el Servicio Cognitivo de Microsoft , Amazon Alexa Voice Service , Google Speech API , Witai y Houndify. Sin conectarse a una red de Internet, el proyecto procesa la voz utilizando la placa integrada PocketSphinx .

El sistema incluye tres componentes: una placa principal ReSpeaker Core, una adicional con una variedad de micrófonos digitales, una tarjeta de expansión con interfaces adicionales y un altavoz para crear un proyecto completo. La principal alberga un micrófono analógico para recibir comandos, un microUSB de depuración y un DAC con un amplificador ( Cirrus Logic WM8960 ) , una ranura para una tarjeta microSD y varios bloques para conectarse a otros dispositivos. Una placa adicional con micrófonos permite el reconocimiento de voz a una distancia de hasta 10 metros.

Para simplificar el proceso de programación y configuración, los desarrolladores planean lanzar su propia aplicación móvil para administrar la placa.
