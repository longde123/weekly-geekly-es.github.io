让我们从Bay Trail平台开始。 它于2013年第三季度宣布，是一种高度集成的系统（SoC）。 它基于22nm三栅极技术。 所有处理器均分为3组：

  • J-用于台式机（J1xxx-赛扬，J2xxx-Pentium）；
  • Z-用于平板电脑（Z37xx-四核型号，Z36xx-双核型号）。

在价格高达15,000卢布的负担得起的笔记本电脑中，最常见的处理器是基于Bay Trail平台（2013）的处理器，特别是Intel Atom Z3735G和Intel Atom Z3735F，它们是双胞胎，但可惜的是，它们本人并不相同。 事实是最后一个字母表示存储通道的数量。 制造商的网站提供了规范中的干燥数字和事实：

  • 4核/ 4线程，2 MB L2缓存；
  • 英特尔爆裂
  • 支持64位操作系统；
  • SSE4.1矢量指令；

顺便说一句，介词“做”并不总是反映现实。 例如，Intel Atom Z3735G仅支持1 GB内存，最大吞吐量为5.3 GB / s，而Intel Atom Z3735F已经具有2 GB内存，吞吐量分别增加到10.6 GB / s和卷。 英特尔凌动Z3735G包含四个核心并处理四个线程。 推荐费用为$ 17。 处理器的基本时钟频率为1.33 GHz，最大频率为1.83 GHz，高速缓存为2 MB，最大额定功率为2.2瓦。 英特尔凌动Z3735F的售价为18美元。

除Atom之外，还有赛扬处理器，编号为N2840，N3050，N3060和N3350。 N2840型属于Bay Trail平台，包含两个核心。 工作频率为2.15至2.58 GHz。 最大功耗为4.5-7.5 W，具体取决于任务的类型。 该芯片支持两个通道的DDR3L 1333内存，最大容量为8 GB。 通常，笔记本电脑的普通入门级处理器。

通常，随着笔记本电脑价格的上涨，制造商会尝试从模型系列中投放高端CPU。 事实证明，在现实世界中，一切并非如此，去年的型号与本年度型号的笔记本电脑可以和平共处。 因此，我们将再次警告不专心的客户：即使在价格更高的细分市场中，也有带有上述处理器的型号。 并且新的添加到了它们，它们并不总是比旧的更快和更好。

英特尔


打开三个英特尔奔腾处理器的列表，编号分别为N3540，N3700和N3710。 第一个是指Bay Trail平台（2013年），其余是Braswell（2015年）。



仅当您想使用功耗更低（4-6 W而非4.5-7.5 W）的14 nm技术制造处理器时，才需要在Braswell停下来。 结果，降低了冷却系统的热量和噪音。 由于过于紧密的特性，没有必要谈论一个在另一个方面的显着优势。 它们全部具有四个内核，2 MB的高速缓存，并且N3540的频率范围为2.16-2.66 GHz，N3700的频率范围为1.60-2.40，N3710的范围为1.60-2.56。 Braswell处理器支持1600 MHz DDR3内存，而Bay Trail被限制为1333 MHz。 而第一个重要的区别是所有所述处理器的双通道内存。

奔腾中有名称为U的CPU，它们是Intel Pentium 3558U和Intel Celeron 2957U。 两种处理器均来自Haswell体系结构桌面系统。 尽管Celeron和Pentium的名称有所不同，但实际上它们都只有两个处理核心，2 MB的高速缓存和15瓦的功耗。 它们之间的区别仅在于工作频率。 英特尔Celeron 2957U在没有Turbo模式的情况下以1.40 GHz的频率运行，英特尔奔腾3558U在1.70 GHz的情况下也没有增强功能。 根据笔记本电脑制造商的意愿，在具有这些处理器的系统中，双通道模式下最多可以有16 GB的普通DDR3 1600 MHz SoDimm内存。 事实证明，这些处理器最接近台式机系统，并为改善笔记本电脑的性能提供了最大的机会。

AMD公司

笔记本电脑的价格为20至25,000卢布


根本不能以不同的方式花费五千卢布。 有人会选择中等配置，更喜欢好的矩阵或紧凑的尺寸。 其余关心处理器性能的人将告诉您他们的期望。



打开我们的全功能奔腾N4200英特尔清单，该清单具有阿波罗湖家族的双通道内存（2016年）。 我们已经见过他的弟弟赛扬N3350。 它们之间的区别在于涡轮频率略高，四个核心而不是两个。 该平台虽然基于14 nm，但仍是新鲜的。

末尾带有字母“ U”的处理器属于第三代（Haswell，Broadwell，SkyLake）。 可以说，最古老的是具有Haswell架构的Intel Pentium 3556U。 它没有最低的工作频率-无Turbo模式时为1.7 GHz，但它是根据22 nm制程技术制造的，并且与DDR3存储器相邻，更确切地说，它是在1.35 V电压下运行的节能型LDDR3。

基于Broadwell架构的处理器正在逐步发展。 在这里，您拥有先进的14 nm制造工艺，并且在某些情况下还具有超线程技术。 多亏了它，系统上的可执行线程数量从两个增加到四个，尽管实际上处理器仍然拥有两个内核。

较年轻的Intel Celeron 3215U和Intel Celeron 3205U CPU的工作频率有所不同（1.7 GHz与1.5 GHz），其余规格完全相同：两个内核，2 MB高速缓存内存，高达15 W的功耗以及对LDDR3内存的支持频率高达1600 MHz。 同样，返回到内存数量而不是其通道的重要性，在所有列出的处理器中，使用了两个内存通道。 对于处理器而言，这是一个物理机会，但并非所有制造商都能利用此机会，通常会在单个通道上为笔记本电脑提供4或8 GB的RAM。

英特尔奔腾3825U和3805U也非常相似，除了一个重要事实，即3825U支持超线程，而3805U没有。 两者均以1.9 GHz的频率运行，而没有Turbo模式，内存最高可达1600 MHz。

最后，Broadwell处理器的最后代表是Intel Core i3-5005U。 尽管属于i3系列，但与Pentium 3825U几乎没有什么区别。 首先，频率上升到2 GHz。 其次，缓存大小已从2 MB增加到3 MB，仅此而已。 为此多付钱是值得的，显然不是，因为附近有几个SkyLake处理器。

英特尔奔腾4405U的性能接近奔腾3825U，但工作频率为2.1 GHz。 但是，这里的频率并不是特别重要，因为 您应该注意这一方面：SkyLake有了一个新的控制器，该控制器一次与两种类型的内存兼容-DDR4和LPDDR3。 根据制造商的诚实程度，笔记本电脑可以是一种，也可以是另一种。 自然地，使用DDR4，性能会稍高，这是值得考虑的。 但是使用LDDR3，SkyLake具有比Broadwell更高的优势，因为它具有更高的工作频率-1866 MHz，而不是1600 MHz。

从所有迹象来看，英特尔酷睿i3-6006U的正式形式应优于奔腾4405U，但经过仔细检查，其频率降低了100MHz。 尽管增加了高速缓存，但它无法与奔腾4405U竞争。 再次，使用的内存类型将成为决定因素。

AMD公司


为了应对快速增长的廉价笔记本电脑行业，AMD和英特尔一样，有一些雄心壮志来抢占一席之地。 在图形方面，其APU领先于竞争对手，这不能说主处理器内核的处理能力。 尽管展望未来，但一些APU在英特尔处理器的背景下表现得相当不错。 另一方面，在这个细分市场中，过时的APU某种程度上不适合现代现实。 但是，我们将从非常有趣的示例开始。

AMD A9-9410仅具有两个内核，动态范围为2.9至3.5 GHz。 通过在双通道模式下支持最高8 GB的DDR4内存和最高2133 MHz的频率，可以补偿少量的高速缓存（1 MB）。 这是Stoney Ridge平台上唯一具有挖掘机核心的处理器。 其图形核心（取决于笔记本电脑中大量RAM的可用性）能够与最简单的独立图形卡竞争。

下面列出的所有四核APU都是过时的处理器，但是它们可以很好地满足需求不高的客户的需求。 例如，在2.2-2.5 GHz频率下运行的AMD A8-7410或在1.8-2.4 GHz频率下运行的AMD A6-6310，甚至具有1.5 GHz的AMD E2-6110都能胜任。 但是，它们都仅支持一个内存通道，并且它们将无法充分显示集成图形核心Radeon的潜力。 而最新的AMD Brazos QC-4000 APU就像是恐龙：它不仅具有1.3 GHz的适度频率潜能，而且更专注于平板电脑而不是笔记本电脑。

笔记本电脑的价格从25到3万卢布


在某些方面，最后一台笔记本电脑确实很有趣。 在这一类别中，我们首次遇到了最新的处理器和灵活的APU型号，并带有不错的图形。 为了继续进行对话，我们将所有内容都放在了书架上。 首先，我们说所有系统都支持双通道内存，并且大多数处理器都支持更快的DDR4内存。

英特尔


蓝色阵营的代表通过三代处理器满足了这一需求：从Haswell到最新鲜的Kaby Lake。


英特尔Kaby-Lake。 图片：pcworld.co.nz

当然，在更多现代CPU的背景下，英特尔酷睿i3-4030U看上去已经褪色，但它也具有良好的特性，包括1.9 GHz的频率，3 MB的高速缓存，对超线程的支持，DDR3L / LPDDR3内存（最大容量为16 GB）以及1600 MHz。
但是Skylake在各个方面绝对比他更好。

Core i3-6100U的固定频率为2.3 MHz，并与三种类型的内存结合使用，最大容量为32 GB：DDR3L高达1600 MHz，LPDDR3高达1866 MHz，DDR4高达2133 MHz。 您会在笔记本电脑中找到哪一个是一个反问。 最好使用DDR4内存找到它，然后在下拉列表中找到优先级。 Core i5-6200U的动态核心频率范围为2.3至2.8 GHz，其他特性与Core i3-6100U相同。

在呼吁的最上方，Kaby Lake架构上的Core i5-7200U停止了。 对于给定价格范围内的潜在买家，他是优先选择。 对内核进行了优化和改进，频率增加了，图形内核又得到了改进。

AMD公司

结论


普通买家同时出现的大量平台最初会令人恐惧。相似且难以区分的处理器名称也会使他感到困惑。同意，谁将立即回答有关Intel Pentium 3825U和Intel Pentium 3805U之间的区别的问题？还是用APU AMD解决难题？那么哪些处理器在四个价格类别中值得考虑？

如果您有10到1万5千卢布，就不能指望运气了。所有处理器都有缺陷，要么是四个内核和一个低频与单通道内存配对，要么是两个内核和一个相对高频但具有双通道内存。甚至都不考虑购买这两种选择，到达后立即会让您感到失望。可以接受的选择是Intel Celeron N3350处理器，但只能使用LPDDR4内存。

价格从15到2万，申请人数正在增加。这些是Intel Pentium N3710，Intel Pentium 3558U和带有咬牙的AMD A6-9210。

上一步是成熟的Intel Core i3-6006U处理器，广义上来说是Intel Pentium 4405U。也许它们会为您提供必要的性能和相关性。对于怜悯的图形组件，我们要求AMD接受AMD中的APU A9-9410。

我们的评论以价格在25至3万卢布的一度最受欢迎的笔记本电脑结束。几年前，对于这个数量，您可以找到具有IPS矩阵，节能处理器和转换可能性的良好配置。现在，充其量您会得到一个中等的系统。尽管我必须说，即使是最新的Intel Core i5-7200U处理器也设法到达了这里。我们建议您搜索它，因为有这样的机会。AMD代表了挖掘机内核时代的最后代表-A10-9600P。
