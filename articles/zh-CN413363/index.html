工匠将佳能EF 70-200毫米长焦镜头固定在Game Boy相机上



Game Boy掌上游戏机在90年代非常流行。 俄罗斯发行于1994年，当时只有有钱父母的后代才能负担得起这种奢侈品，而普通孩子只能在电子产品上抓鸡蛋。

Game Boy已成为世界上第一个流行的游戏系统。 这款设备令人叹为观止，配备了160×144像素的单色LCD屏幕，立体声和GameLink功能，如果游戏支持此模式，您可以将两个游戏机连接在一起并一起玩。

在90年代后期，Game Boy的受欢迎程度开始下降，大约在1998年的那个时候，任天堂为其发布了一个有趣的配件：Pocket Camera，这是一个带有128×128像素CMOS传感器的小型相机模块。

传感器使用Game Boy系统的四色调色板记录了128×112像素的数字黑白图像。 甚至可以插入Game Boy打印机并在热敏纸上打印照片。 总的来说，真正的日本奇迹是：90年代的数码摄影和便携式打印。

当然，任天堂掌上型相机并没有广受欢迎：很快，市场上以1-2百万像素的价格在500-600美元之间的数字“肥皂盒”就进入了市场，然后出现了带有内置相机的廉价手机，因此一般公众不再需要单个相机了。 。

专业摄影器材仍然是专业人士的命运。 其中之一绝对是设计师，摄影爱好者和杂工Bastiaan Ekeler。 他的新项目结合了看似不兼容的东西：旧的Game Boy控制台，带传感器的摄像头模块，可拍摄128×112像素的黑白图像，以及... 70-200 mm的专业佳能EF镜头。



巴斯蒂安（Bastian）计算得出，使用1/4“传感器格式，Game Boy Camera的裁切因子约为10.81。如果在1.4倍增倍镜上连接70-200 f4镜头，则最大等效焦距为200×1.4×10， 81 = 3026.8毫米，您将获得功能强大的远摄镜头，可以开始一个有趣的项目。

言归正传。 工匠在Rhinoceros 3D程序中设计了镜头适配器，并以相当低的分辨率（用于速度）在采用PLA塑料的Monoprice Select Mini v2 3D打印机上进行了打印。



经过一些文件调整后，Game Boy Camera印刷电路板可安装在打印毛坯的背面，并与适配器一起安装在部分拆卸的Game Boy Camera外壳中。



到目前为止，发烧友还不认为该适配器足够完美，无法在Thingiverse上的公共领域中发布。 他说，如果其他人对这项工作表现出足够的热情和兴趣，那么他将最终确定模型并将其发布。

虽然如此，但即使在当前状态下，适配器也能执行其功能-并将整个结构与相机传感器和70-200 mm佳能EF镜头结合在一起。

巴斯蒂安·埃克勒（Bastian Eckeler）已经在大自然中漫步，并以最大变焦拍摄了一系列照片。

如您所见，在这种情况下，镜头可以像真正的长焦镜头一样正常工作。 有趣的是，传感器尺寸和最大焦距与其他参数的这种组合理想地将满月放置在最大帧尺寸为128×112像素中。 巴斯蒂安（Bastian）表示，通过不带背光的游戏控制台的深色显示屏（每秒更新一次）来观察月球，在等效于3000毫米镜头上的月亮并非易事。 但仍然成功拍摄了几张照片。

图片通过Arduino下的Game Boy打印机模拟器从游戏机复制到计算机。 有必要以5美元的价格购买专用电缆，然后将其直接焊接到旧Arduino Duemilanove平台上的Adafruit原型开发板上。 控制台中的数据为HEX格式，但是一个小的JavaScript脚本将其转换为PNG（作者今天想知道，现在不用JavaScript可以做到）。
