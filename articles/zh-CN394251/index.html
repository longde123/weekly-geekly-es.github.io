戴尔XPS 12：无需在笔记本电脑和平板电脑之间进行选择

二合一设备的销售，即结合了平板电脑和笔记本电脑的质量，正在逐年增长。 2015年，此类联合收割机在全球销售了超过4,200万。增长的原因很明显：以相同的价格买到这两者真的很诱人。但是，二合一设备类别的定义仍然非常模糊。以此为幌子，您还可以出售最普通的笔记本电脑，它的盖子可以旋转180度，还有一个简单的平板电脑，其键盘上的字面上是一块抹布。



全新的Dell XPS 12经过精心设计，使两个实体都拥有比笔记本电脑和平板电脑更强大的功能。坦白说，这项任务并不简单：可拆卸的屏幕通常会迫使您将过多的组件转移到其中，这至少会破坏系统的平衡。但是戴尔工程师设法以最小的折衷解决了问题。而且大多数用户甚至不会注意到它们。

碰巧的是，戴尔XPS 12来我这里学习时没有任何初步的虚拟认识。通常，我会访问Dell网站，查看有关Amazon和Best Buy的评论，然后进行几次商务旅行，坦率地说，这绝对没有任何好处。我只知道有一个有趣的新产品，剩下的一个。因此，当时间到了时，我打开了包装盒-很小，比iPad还要薄-在最初的五分钟里，我确定我正在处理一款紧凑型的超极本。当我更仔细地看着键盘和屏幕的交界处时，我更加用力地拉了后者-而他仍然留在我的手中，引起了一些疑问。



认真地说，设计很棒。折叠后，您将无法理解顶盖没有拧紧。键盘和主机与屏幕的连接处的角度突出表明了这一点，但它可能被误认为是设计师的架子。此外，当您抬起屏幕时，它会随着磁铁平稳而可靠地漂浮。所以在这个阶段，我并没有想到我正在处理二合一。展开后，Dell XPS 12可以同时戴在键盘和屏幕上。磁铁很坚固，什么也不会掉下来。



但是，我们付出了一点努力-只有平板电脑可以拿在手中。没羽毛，790克。但是在厚度为8毫米的情况下，可以用一组不错的接口来填充真正的个人计算机。而且只有一个很棒的屏幕。

让我们更详细地讨论配置。我必须马上说，其中一个顶级版本落入了我的手中，但是有一些更简单的选择，我将在结尾处进行介绍。

顾名思义，Dell XPS 12的屏幕对角线为12，即12.5英寸。分辨率为3840 x 2160像素，或者使用电视领域的术语为4K Ultra HD。您可能会猜到，清晰度超过-352 dpi，RGB调色板的覆盖率达到100％。屏幕真的很好。 Windows 10已学会明智地使用高分辨率，因此对象不仅很小，而且非常清晰。 IPS IGZO矩阵的视角接近180度。至少，即使完全从侧面看图片，我也没有发现阴影有任何失真。考虑到屏幕的对角线，此功能特别有用：如果您必须集中看待它，则不必面对额头寻找舒适的观察点。



表壳材料-带有软触涂层的镁合金。屏幕由大猩猩玻璃NBT保护-一种专门为平板电脑和笔记本电脑设计的涂层，可防止刮擦表面。平板电脑零件的制作就像是真正的平板电脑。我非常擅长拆卸基于x86架构的笔记本电脑和平板电脑，但是我承认在这里，我做不到。细节紧密地结合在一起，我只是害怕寻找秘密的闩锁，以免损坏边缘。

由于这是一台真正的平板电脑，因此没有主动冷却功能。是的，没有螺旋桨。它没有散热器，其作用是屏幕的整个背面。这要归功于Intel Core m5-6Y57处理器。它只有4.5瓦的热量，两个核心的标准频率为1.1 GHz（支持超线程），并且可以在负载下加速到2.8 GHz。我知道有关Core m性能的巨大偏见，但是我自己在不同的系统中尝试了第一代产品，因此我个人就足够了。尽管很难称呼我为需求不高的用户：例如，对我来说，每天晚上以RAW速度处理一百张照片是我的典型任务。新一代Core m似乎出于一致性被称为第六（事实上，它是第二），根本没有性能问题。在实际应用中，Core m5-6Y57绰绰有余。图形核心Intel HD Graphics 515并不令人失望，对于4K分辨率而言，这并不构成任何困难。



如果我们仔细研究第六代移动Core i5和Core m的特性，我们会发现Core m中的内存带宽略低，这可以通过“额外”兆字节的Smart Cache进行补偿。有一种观点认为，在日常任务中，缓存将更加有用。

只是为了比较。在WinRar基准测试中，英特尔酷睿m5-6Y57得分为2813分，而酷睿i5-5200U的基本频率为两倍，仅为2405。因此，请确定谁更快。

好消息是XPS 12的最小存储容量为8 GB。但是碰巧她是最大的。处理器本身最多可以寻址32 GB，但是在这种紧凑的情况下（记住大约8毫米），很难找到另外4-8个芯片的位置。另一方面，8 GB是用于移动PC的最佳选择，因为我的需求是我购买具有该容量的设备。

我的Dell XPS 12副本的存储容量为256 GB，但从128 GB到512不等。我的Samsung SSD的性能令人满意。

12英寸笔记本电脑键盘舒适吗？是的，当然。这个空间允许它，问题仅在于工程师的资格。在这种情况下，不仅可以将主键制作成正常大小，而且还可以找到许多功能键的位置。他们并没有忘记箭头。手指立即落在键盘上，无需上瘾。我注意到按键的过程非常熟悉，舒适-1.9毫米。没有便宜的平板电脑和一些高端笔记本电脑，可以缩短行程的棘手设计。另外，出色的背光，对于此类设备而言不是很典型。

唯一需要抱怨的是固定屏幕的一个可能角度。对于中等身高的人来说，这挺舒适的，但是我曾经长得不错，因此，当我跪在XPS 12上时，我希望屏幕再偏离我自己7度，这不是一个严重的缺点，因为您仍然可以工作（尤其是在桌子上）。但是一旦我注意到-我需要告诉。值得注意的是，除了完整的键盘外，戴尔还生产另一款轻薄的键盘。按键的行程较小，为1.3毫米，但可以调整视角（最大150度）。同样出于这些目的，您可以使用提供的手提箱文件夹。



Dell XPS 12具有两个扬声器，可以绘制相当宽的立体声全景图（但没有强大的低音，还没有禁止物理定律），一对麦克风可以在Skype上即使在大风时也能舒适地进行通话，还有两个摄像头-主摄像头（8兆像素）和前置摄像头（5百万像素）。作为DSLR的替代品，我不建议您使用此组合机，但是，对于日常通信而言，相机的功能已绰绰有余。许多智能手机拍摄的照片质量较差。



在官方网站上阅读此设备的说明后，您可以欣赏接口的数量-两个Thunderbolt 3，两个USB 3.1 Gen.2，甚至还有一个读卡器的地方。确实，所有这些都是存在的，但是，正如他们所说，这是有细微差别的。 Thunderbolt 3和USB 3.1端口组合在一起，就USB 3.1而言，我们的意思是USB-C。通过Thunderbolt连接，您将获得大约四倍的速度（通过USB达到40 Gb / s而不是10 Gb / s），但实际上这些是相同的连接器。其中之一也用于连接充电器。

在我看来，这比连接器动物园更方便，尤其是考虑到USB-C基础架构的发展速度。如果可用的接口不够用（这是完全可能的），则可以购买具有USB Type-C和HDMI / VGA /以太网/ USB 3.0的Dell适配器。但是请不要忘记，可以通过Thunderbolt轻松连接新的显示器和投影仪。



读取器不支持全尺寸SD卡，但支持microSD。因此，对于摄影师来说，通过适配器在相机中使用它们是有意义的。最好的microSD样本已经支持以高达100 MB / s的速度记录数据，并且过渡到这种形状系数似乎是不可避免的。我记得我们对在数码单反相机中支持SD持怀疑态度。与正确的CompactFlash相比，它们能为我们提供什么？但是这个CompactFlash现在在哪里？



Dell Wireless 1820A适配器支持802.11ac（2x2，最高866 Mbps）和Bluetooth 4.1。将来计划发布带有LTE模块的版本，在microSD插槽旁边保留SIM卡的位置。

平板电脑的电池电量为30 W / h。它是不可移动的，并且没有提供选择更大容量版本的可能性。在正常的平板电脑模式下充满电大约足以使我持续五个小时左右（约15分钟）。当然，在严重的负载下，结果会更少-我指望四个小时。我想要更多，但是值得一提的是，我们拥有a）非常紧凑的结构，b）强大的结构以及c）混合设备。实际上，这是我们在开发期间必须做出的唯一妥协。对于所有其他文章，混合性和紧凑性不会影响用户体验。好吧，这是一个令人愉快的时刻：电池在一个半小时内从零开始充满电。在半小时内达到50％。
