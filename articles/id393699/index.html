Cara menonaktifkan pemutaran video otomatis di browser populer



Dengan perkembangan Internet dan peningkatan kecepatan akses rata-rata, banyak pemilik situs web menganggapnya sebagai praktik yang dapat diterima untuk secara otomatis mulai memutar video segera setelah memuat halaman.

Namun, tidak semua pengguna setuju dengan kebijakan ini. Mereka yang tidak setuju dengan ini, yang menganggap situs-situs dengan musik yang diputar buruk (dan mereka berusia 90-an dan seterusnya) menjadi buruk, sekarang percaya bahwa video tidak boleh mulai secara otomatis, dan mulai hanya atas permintaan pengguna.

Memulai video secara otomatis dapat menyebabkan pemborosan lalu lintas (yang mungkin ingin dihindari pengguna), penurunan bandwidth saluran, dan tampilan suara keras yang tidak pantas. Dan meskipun bahkan di tab Firefox sedang ditandai yang saat ini menghasilkan suara, perilaku browser ini tampaknya tidak nyaman bagi beberapa pengguna.

Khusus untuk mereka, blogger Martin Brinkmann menyiapkan pilihan cara untuk melarang pemutaran video secara otomatis untuk browser yang berbeda. Termasuk video yang menggunakan opsi Flash, dan HTML5.

Firefox dan HTML5


Yang menyenangkan, Firefox memiliki pengaturan untuk memblokir video autorun tanpa perlu plugin. Dalam hal HTML5, pengaturan tentang: config akan membantu Anda. Anda perlu memuat halaman ini, temukan pengaturan media.autoplay.enabled dan alihkan ke false



Firefox dan Flash


Untuk mencegah video berbasis Flash dari memulai secara otomatis, Anda dapat mengunduh halaman ekstensi, pergi ke halaman plugins di sana dan ubah pengaturan Shockwave Flash ke "meminta untuk mengaktifkan" (selalu bertanya sebelum memulai).



Chrome dan HTML5


Chrome tidak memiliki pengaturan untuk menonaktifkan autorun. Plugin dapat membantu dengan ini, tetapi hanya dalam versi desktop - karena mobile Chrome tidak mendukung plugins, penggunanya masih tidak bisa menghilangkan kemalangan ini.

Di desktop, Anda dapat menggunakan plugin Nonaktifkan Autoplay HTML5 dan Video Autoplay Blocker .

Chrome dan Flash


Untuk menonaktifkan video flash, Anda perlu memuat chrome: // pengaturan / pengaturan konten, temukan bagian plugin di sana dan alihkan pengaturan ke "biarkan saya memilih kapan harus menjalankan konten plugin" (pilih kapan memulai plugin).



Opera dan HTML5


Opera didasarkan pada mesin Chromium / Blink, jadi situasinya mirip dengan Chrome. Untuk memblokir video berbasis HTML5, Anda dapat menggunakan plugin Nonaktifkan Autoplay HTML5 dan Video Autoplay Blocker .

Opera dan Flash


Untuk menonaktifkan video flash, Anda perlu mengunduh opera: // pengaturan, beralih ke "situs web" di menu di sebelah kiri, dan beralih ke opsi "klik untuk memutar" di bagian "plugin".



Internet Explorer / Microsoft Edge


Di IE, tidak ada cara untuk mengkonfigurasi tampilan otomatis video - Anda hanya dapat sepenuhnya menonaktifkan Flash. Diharapkan bahwa ketika Edge datang dengan dukungan plug-in, plugins dengan Chrome akan porting untuk itu.

Vivaldi


Browser Vivaldi baru didasarkan pada mesin Chromium / Blink, jadi di dalamnya semua pengaturan dan plugin bekerja persis sama dengan di browser Chrome.

Pembaruan: sayangnya, di browser Firefox, jika pemutaran video otomatis dimatikan, itu mulai membisukan pemutaran video YouTube (dalam mode HTML5).
