Workshop Elektronik: Bagian 1 - Ikhtisar Solusi

Bagaimana saya memutuskan untuk membuat laboratorium portabel.
Bagian 1: mengapa dan apa yang serupa ditemukan di Internet ...
Bagian 2: Wishlist



Saya memiliki hobi kecil: pada hari Senin, berkeliling di satu toko dan membantu memahami berbagai omong kosong elektronik, memperbaiki gadget yang terbang, dll. dll.
Apa jenis toko dan apa jenis gadget - sayangnya, rahasia dagang ...
Tetapi sebagai fakta, kita secara berkala membutuhkannya:
setrika solder,
senapan panas,
pengering rambut (sebagai hasil dari paragraf terakhir, saya memiliki masalah dengan SMD untuk saat ini, tetapi untuk menghilangkan lem panas yang dikuasai oleh orang Cina oleh orang-orang China - paling banyak it!),
tester,
komputer,
programmer,
untuk beberapa waktu sekarang - osiloskop,
secara berkala - "radio tester-interceptor" untuk pita 315 dan 433 MHz,
catu daya,
kaca pembesar, lampu, dll. dll.

Fakta bahwa membawa alat setiap waktu adalah kemalasan, karena tidak setiap kunjungan diperlukan untuk semua ini, jadi mereka memanggil saya terlebih dahulu dan mengatakan bahwa saya perlu sesuatu yang spesifik, atau saya pulang kerja.
Dimungkinkan untuk menyimpan set alat kedua di toko, tetapi meja, yang dianggap sebagai "milikku," secara berkala gagal karena benang atau sampah, karena butuh tempat, dll. dll., pada akhirnya, Anda tidak dapat membusuk dan Anda perlu mengumpulkan dan meletakkan semuanya di rak setelah bekerja ...

Saya membuat kesimpulan yang paradoks:
"Tetapi bagaimana jika saya membuat kotak alat yang saya letakkan di atas meja dan membukanya, voila - laboratorium portabel!? "

Pencarian sumber daya memberikan beberapa solusi yang â€œ1 in 1â€ tidak cocok untuk saya, tetapi beberapa pemikiran berpikir ...
