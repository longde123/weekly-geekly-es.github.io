Ultimaker 3扩展3D打印机视频审查



关于荷兰3D打印机Ultimaker的较旧型号之一的视频回顾-Ultimaker 3 Extended。 这是对Ultimaker 3型号的改进，使用一个挤出机时扩大了打印面积-215x215x300 mm，使用两种材料打印时扩大了197x215x300 mm。




Ultimaker 3扩展版



  • 技术：FDM
  • 挤出：双
  • 校准：自动
  • 重量：11.3公斤
  • 尺寸：342 x 505 x 688毫米
  • 建筑面积：使用一台挤出机打印时为215 x 215 x 300毫米，使用两台挤出机为197 x 215 x 300。
  • 喷嘴：0.4毫米，可选0.8。
  • 层数：200-20微米
  • 平台加热：最高100˚C，4分钟
  • 喷嘴加热：180-280˚C，2分钟
  • 打印速度：30-300 mm / s
  • 建立速度：最高16mm³/ s（使用0.4 mm的喷嘴）
  • XYZ精度：12.5-12.5-2.5微米
  • 噪音水平：50dB
  • 灯丝直径：2.85毫米
  • 材料：尼龙，PLA，ABS，CPE，CPE +，PVA，PC，TPU 95A，PP
  • 价格： 349,000卢布








从授权经销商处订购 Ultimaker 3 Extended，并提供保修和免费奖金。
