Winamp回来了-讨论他在流媒体服务时代是否有机会

Radionomy计划在中断五年后发布新版本的Winamp。 播放器应成为流媒体服务，播客和有声读物的聚集者。

我们正在讨论前景如何。


图片Jarle Refsnes / CC

Winamp的回归


上一次更新（如果您不考虑版本5.8的话 ）于2013年发布。 但是最近，Radionomy在2014年从AOL收购了Winamp，宣布了Winamp 6。

它应该成为听音乐的通用平台。 除了计算机上的曲目外，用户还可以收听互联网广播，流媒体服务，播客或有声书。 同时，Winamp 6将在Android和iOS设备上可用。

Radionomy首席执行官Alexandre Saboundjian 表示 Winamp是创建统一音频服务的理想基础。 该程序仍然有很多粉丝：每月有1亿人使用该播放器。

Winamp有机会吗？


产品的普及可以起到“重启”的作用。 许多曾经使用过该播放器的人都极有可能至少尝试使用新版本。

对于移动平台，还有另一个因素可以帮助Winamp 6“解决”问题。 iOS和Android的音频市场支离破碎。 它还没有聚合器，可以让您（不仅限于）从大量不同的来源收听音乐。 Sabunjan认为Winamp将能够解决此问题。

通过对社交网络和主题站点的评论来判断，用户自己很高兴听到有关该应用程序新版本的发布。 但是，尽管采用了其他格式，但仍然有人对重启产品的可取性表示怀疑。

部分怀疑与将一个站点上的多个流服务组合在一起的想法有关。 根据TechCrunch的说法，相同的Apple Music和Spotify可能拒绝与Winamp合作，因为他们不想与竞争对手分享观众。

如果您仍然同意大型流服务，那么开发人员将面临另一个问题。 Winamp的主要王牌是怀旧和皮肤的感觉。 当开发人员开始适应当前市场需求时，应用程序可能会发生巨大变化。

竞争对手太多


Winamp 6似乎不太可能成功的另一个原因是众多竞争对手。 我们正在谈论的是相同的流媒体平台-Spotify，Apple Music，Deezer，Tidal（我们在此做了单独的帖子 ）以及其他。 最近，Spotify的订户数量已达到 8300万，而在Apple Music上，这一数字已超过5000万 。

有一种观点认为，人们不会将其通常的服务交换为充当中介的聚合器。 Winamp不太可能成功占领成熟的流媒体平台的利基市场。 市场部分已经完成，对于新来者来说，很难楔入（即使拥有大量用户）。 即使是像Google这样的巨头，其YouTube音乐服务也存在问题 。


图片维基百科 / CC

但是，即使您忘记了流媒体平台的存在，而将注意力转移到经典音乐播放器上，也一切都是模棱两可的。 当Winamp不在时，其他应用程序的开发吸引了听众。 这里只是其中一些：

  • Foobar2000：它的创建者是波兰自由程序员Peter Pavlovsky。 顺便说一下，Peter之前曾与NullSoft合作，后者创建了第一个Winamp。 Foobar因其简约的界面和丰富的功能集而获得赞誉，可提供高保真的声音。 它仍然是Windows （以及AIMP） 最受欢迎的播放器之一 。 去年，针对Linux发行了foobar2000。 您可以在此处下载。
  • AIMP：开发由俄罗斯程序员Artyom Izmailov开始。 播放器在自己的声音引擎上运行。 该应用程序支持一些有趣的功能，尤其是“智能”播放列表 ，这些列表会根据指定的选择规则（流派，年份，专辑等）自动收集。 同时，AIMP有一个适用于Android的版本-年初，它获得了29个频段的均衡器 。
  • QMMP：适用于Linux和Windows的多平台播放器。 它的主要优点之一可能是可定制的UI。 特别是，QMMP支持原始Winamp2的皮肤。 您可以从官方网站上的存储库中下载播放器。
  • 大胆：一个开源播放器。 最新版本已于今年8月发布。 它支持插件，因此可以扩展其功能。 有些模块可以转换输出声音或提供将信息直接广播到last.fm的功能。 还支持Winamp2的皮肤。
  • Windows Media Player： Microsoft的媒体播放器也找到了它的观众。 该播放器因其媒体目录的便利性以及与指挥的整合而受到赞誉 。 另外，它是标准的Windows播放器，因此您无法安装其他应用程序来播放音乐。

而不是结论


关于即将发布的Winamp版本的细节并不多。 甚至不知道旧的口号“ 真的鞭打了美洲驼的屁股 ”是否会被保留。 因此，很难理解Winamp 6的期望。

Engadget 写道 ，如果Radionomy可以与流媒体平台进行谈判并提供真正的通用服务，那么用户将获得一个尚无类似产品的便捷应用程序。 否则，它将只是播放器的新版本，这将使Winamp的粉丝感到满意。
