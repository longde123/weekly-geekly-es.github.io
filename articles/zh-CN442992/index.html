Windows Calculator现在开源

我们很高兴地说，上周我们的Windows计算器成为了OpenSource。 现在可以通过MIT许可在GitHub上使用它。 这包括源代码，程序集结构，单元测试和产品结构。 我们的目标是与社区合作，创造出更好的产品。 我们鼓励您提出新想法并积极参与确定Windows计算器的未来。



作为开发人员，如果您想知道Calculator应用程序的各个部分如何工作，可以很容易地将Calculator的逻辑或用户界面集成到您自己的应用程序中，或者直接为Windows中可用的功能做出贡献，现在您可以做到。 与所有其他应用程序一样，计算器将继续执行所有常规测试过程，包括检查合规性，安全性，质量和检查内部数据。 您可以在GitHub的文档中了解有关此内容的更多信息。



共同发展


除了在自己的应用程序中重用和调整代码外，每个人都可以参与Windows计算器的开发 。 参与很容易。 随行复制项目及其开发将根据标准GitHub流程进行 。 在所有阶段，开发人员都可以做出贡献：


  • 参与讨论
  • 报告或错误修复
  • 提供新功能
  • 原型新功能
  • 与我们的工程师一起设计和创建产品

培训课程


学习计算器代码是了解最新Microsoft技术的好方法，例如通用Windows平台，XAML和Azure Pipelines 。 由于有了这个项目，开发人员可以从Microsoft的整个开发周期中学习，以及重用代码来创建自己的产品。 这也是创建Fluent应用程序设计的一个很好的例子。 为了使这一过程变得更加容易，我们将提供自定义控件和API扩展，供我们在计算器和其他应用程序中用于Windows Community Toolkit和Windows UI Library等项目 。


我们很高兴邀请大家加入Windows计算器团队！ 首先，请转到GitHub上的Windows计算器项目。
