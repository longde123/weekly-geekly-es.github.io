作为水壶，遥控器读取

长期以来，我一直想将一个IR接收器（1838B）和一个IR发射器（5毫米IR二极管，940 nm，大约80度的发射角）连接到Raspberry，以了解它们将如何工作以及如何使用它们。这样的一天来了。

在研究了包括Habr的材料在内的问题之后，我意识到除了IR二极管和IR接收器之外，您还需要一个晶体管和几个电阻，这些电阻有点压抑。接收器更容易-它随套件一起提供，但晶体管不足以容纳二极管，但仍然决定尝试，但是如果能解决的话该怎么办？原来...

基本上，我在浩瀚的地方找到了所有必要的信息，但是在每种材料中都遗漏了一点，因此我必须自己思考一下。这种情况特别复杂，因为我是编程的茶壶，所以有些时候并不容易，所以我将尝试与我逐步描述所有内容并尽可能详细的相同的那些。

因此，IR接收器模块已连接到GPIO23，gnd和3.3V-上面有相应的符号。在GPIO和3.3V之间有一个10k的电阻。LED-通过一个1kΩ电阻连接到GPIO22（阳极）和gnd（阴极）。

接下来-安装LIRC：

sudo apt-get install lirc

编辑文件/ etc /模块：

sudo nano /etc/modules

将行添加到文件中：

lirc_dev
lirc_rpi gpio_in_pin=23 gpio_out_pin=22

其中：
gpio_in_pin = 23-接收器连接的引脚。
gpio_out_pin = 22-发送器的引脚。

编辑文件/etc/lirc/hardware.conf：

sudo nano  /etc/lirc/hardware.conf

应该是这样的：

/etc/lirc/hardware.conf
########################################################
# /etc/lirc/hardware.conf
#
# Arguments which will be used when launching lircd
LIRCD_ARGS="--uinput"

# Don't start lircmd even if there seems to be a good config file
# START_LIRCMD=false

# Don't start irexec, even if a good config file seems to exist.
# START_IREXEC=false

# Try to load appropriate kernel modules
LOAD_MODULES=true

# Run «lircd --driver=help» for a list of supported drivers.
DRIVER=«default»
# usually /dev/lirc0 is the correct setting for systems using udev

MODULES=«lirc_rpi»

# Default configuration files for your hardware if any
LIRCD_CONF=""
LIRCMD_CONF=""
########################################################

编辑文件/boot/config.txt：

sudo nano  /boot/config.txt

将行添加到文件中：

dtoverlay=lirc-rpi,gpio_in_pin=23,gpio_out_pin=22

其中：
gpio_in_pin = 23-接收器连接的引脚。
gpio_out_pin = 22-发送器的引脚。

重新启动树莓：

sudo reboot

设置完成，您现在可以检查所有工作方式。

需要为每个遥控器执行此操作。配置文件包含一个重要字段-名称。这是lirc用来确定从哪个文件中接收命令的设备的名称（在我的情况下为Samsung_TV）。

生成的文件被复制到某处。我放入/ etc / lirc / rc文件夹，并将该行添加到/etc/lirc/lircd.conf文件中：

include "/etc/lirc/rc/aa59-00741a.conf"

因此，lirc配置文件将从/ etc / lirc / rc文件夹中的aa59-00741a.conf文件获取有关我的遥控器的信息。我认为，这是将有关控制台配置的信息添加到程序设置的最简单方法。

重新启动lirc：

sudo /etc/init.d/lirc restart

现在，lirc可以使用远程控制数据通过发送器进行发送，或者识别来自远程控制的命令并启动任何响应于覆盆子的操作。

检查Lirc和IR发射器的操作

为此，我们将发射器放置在尽可能靠近设备的位置（以进行验证，然后有可能通过实验确定其工作距离），并在终端中编写命令，例如：

irsend SEND_ONCE Samsung_TV KEY_POWER

irsend是用于控制发送器的程序；
SEND_ONCE-一次性信号发送的属性；
Samsung_TV-配置文件中的设备名称（在名称字段中）；
KEY_POWER-要从配置文件发送的命令的名称。

这样，电视可以打开或关闭。使用此命令，我可以打开或关闭电视。

再次检查IR接收器，以及是否正确识别配置文件

在终端中，输入命令：

irw

之后，按下遥控器的按钮时，将显示配置文件中的名称。
