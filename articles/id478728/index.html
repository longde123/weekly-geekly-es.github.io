9 sumber open source terbaik untuk november 2019

Selamat musim dingin, tuan dan nyonya. Saya telah menyiapkan untuk Anda pilihan penemuan paling menarik dari sumber terbuka untuk November 2019.


Untuk daftar lengkap alat, artikel, dan laporan baru yang bermanfaat, Anda dapat menghubungi saluran telegram saya @OpensourceFindings ( cermin tautan jika dokumen asli tidak terbuka).


Dalam edisi hari ini.
Teknologi di dalam: Rust, TypeScript, JavaScript, Go, Python.
Topik: pengembangan web, alat untuk QA, bekerja dengan data, alat pengembang dan administrator.


Masalah terakhir .


githistory


Mesin waktu untuk github. Memungkinkan Anda benar-benar melihat seperti apa suatu file pada waktu tertentu. Nyaman, cantik.
Itu ditulis dalam JavaScript.


Tautan



Qawolf


Satu npx qawolf record example.com untuk merekam tes browser. Ini terlihat sangat mudah.
Ditulis dalam TypeScript.


Tautan



plynth


Dan sekarang sedikit hiruk-pikuk. Anda dapat menulis aplikasi web yang cepat, asli, utas aman dalam HTML / CSS / Python. Apa? Python Ya, Python.
Anda bahkan dapat menggunakan Vue.js secara internal. Dan menulis kode dengan Python. Di sini, lihat!
Itu ditulis dalam Python.


Tautan



lazydocker


TUI minimalis untuk mereka yang banyak bekerja dengan Docker dan suka menggunakan mouse.
Ditulis dalam Go.


Tautan



tui-rs


Omong-omong, tentang TUI. Berikut adalah perpustakaan yang berguna untuk membuat aplikasi Anda. Di dalam banyak widget yang sudah jadi.
Ditulis dalam Karat.


Tautan



easydb


Database dalam satu klik. Tanpa registrasi dan SMS.
Klien untuk Python, JavaScript, Ruby, Bash.


Tautan



transform.tools


Layanan yang sangat nyaman, memungkinkan Anda untuk secara otomatis mengkonversi satu format ke format lainnya. Sebagai contoh:


  • HTML dan SVG di jsx
  • json di Kotlin, TypeScript atau Go Struct
  • Penurunan harga dalam HTML, yaml di json, dll.

Tautan



Dovpanda


Linter untuk pandas . Menemukan kesalahan dan konfirmasi. Sangat berguna bagi mereka yang banyak bekerja dengan data. Atau mengajari mereka cara bekerja.
Itu ditulis dalam Python.


Tautan



mengalir


Perpustakaan untuk membuat diagram alur, berjalan di desktop dan seluler. Bagus!
Itu ditulis dalam JavaScript.


Tautan

Itu saja untuk hari ini. Bagi mereka yang menyukai pilihan - berlangganan saluran. Ada banyak proyek menarik lainnya. Saran untuk perbaikan, tautan ke proyek, umpan balik - tulis di komentar.
