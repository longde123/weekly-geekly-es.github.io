Die 9 besten Open Source-Funde für September 2019

Gutes Haktoberfest, meine Damen und Herren. Ich habe für Sie eine Auswahl der interessantesten Funde aus dem Open Source für September 2019 vorbereitet.


Eine vollständige Liste der neuen nützlichen Tools, Artikel und Berichte erhalten Sie von meinem Telegrammkanal @OpensourceFindings ( Spiegellink, wenn das Original nicht geöffnet wird).


In der heutigen Ausgabe.
Technologien im Inneren: Python, C, Rust, Ruby, JavaScript, Go.
Thema: Webentwicklung, Administration, Entwicklertools.


Letzte Ausgabe .


Sampler


Ein Dienstprogramm zum Ausführen und Visualisieren von Shell-Befehlen. Sammeln Sie in 2 Minuten Ihre eigene Überwachung.
Geschrieben in Go.


Link



hyperfein


Ein Tool zum Messen der Geschwindigkeit von Konsolenanwendungen.
Geschrieben in Rust.


Link



Fliesenleger


Tool zum Erstellen von Bildern aus anderen Bildern und Vorlagen.
Es klingt wild - es sieht cool aus!
Es ist in Python geschrieben.


Link



bic


REPL für C. Ja, das tut es.
Geschrieben in C.


Link



Webhint


Linter für Ihre Websites. Mit SSL-Zertifikaten, browserübergreifender Unterstützung, Barrierefreiheit (a11y) und vielen anderen können Sie sicherstellen, dass alles in Ordnung ist. Überprüfen Sie jetzt Ihre Website!
Geschrieben in TypeScript.


Link



pychubby


Ermöglicht das Ändern des Gesichtsausdrucks von Personen (und nicht nur!) Auf einem Foto.
Es ist in Python geschrieben.


Link



Hypothese-Auto


Generiert Hunderte von eigenschaftsbasierten Python-Tests basierend auf der mit Anmerkungen versehenen Funktionssignatur.
Geschrieben in und für Python.


Link



Trockeneffekte


Eine Bibliothek zum Schreiben deklarativer Geschäftslogik in Ruby. Verwendet grundsätzlich algebraische Effekte, über die wir das letzte Mal gesprochen haben .
Geschrieben in und für Ruby.


Link



wemake-python-styleguide


Der strengste und härteste Python-Linter der Welt. Im Inneren gibt es mehr als tausend Regeln zur Überprüfung des Stils, der Komplexität, der Sicherheit und der Lesbarkeit Ihres Codes. Voll kompatibel mit flake8 und als einziges Team in Legacy implementiert.
Geschrieben in und für Python.


Link



Bonus!


Großartiger (und unterhaltsamer!) Artikel über Speicherverwaltung in C und Rust. Ein Beispiel zeigt, vor welchen häufigen Fehlern Borrow Checker von Rust Sie retten wird. Sie können sicher jeden beraten, der Rust aus irgendeinem Grund nicht ausprobiert hat.


Das ist alles für heute. Für diejenigen, die die Auswahl mögen - abonnieren Sie den Kanal. Es gibt viele andere interessante Projekte. Verbesserungsvorschläge, Links zu Projekten, Feedback - schreiben Sie in die Kommentare.
