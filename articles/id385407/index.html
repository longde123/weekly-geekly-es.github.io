YouTube Gaming telah membuka kemungkinan untuk menyiarkan game-nya di ponsel ke Network



Banyak dari kita menikmati menonton orang lain bermain. Untuk melakukan ini, bahkan ada seluruh layanan - Kedutan, dibeli oleh Amazon tahun lalu untuk beberapa miliar dolar. Google memutuskan untuk tidak ketinggalan, tetapi tidak mengulanginya sendiri, oleh karena itu, memperkenalkan layanan YouTube Gaming asli.

Secara umum, layanan ini bukan hal baru, tetapi sekarang perusahaan telah membuat sejumlah pembaruan, termasuk yang sangat besar - YouTube Gaming untuk Android . Aplikasi ini, di antara fungsi-fungsi lainnya, membuka kemungkinan untuk menyiarkan gameplay Anda di Web. Dengan kata lain, jika diinginkan, pemutar dapat menyiarkan video game di Angry Birds, dan pengguna lain dapat mengamati semua ini.

Aliran video tidak bodoh, jika diinginkan, Anda juga dapat mengalirkan aliran audio, memberikan komentar suara selama pertandingan. Banyak pengguna Web mungkin akan menghargai layanan Google. Jika Anda ingin mencoba menyiarkan game Anda sendiri, menunjukkan kepada dunia kemampuan Anda dalam membunuh babi, maka Anda dapat melakukan ini tanpa masalah dengan menginstal aplikasi yang sesuai.

Benar, ini membutuhkan Android 5.0 atau lebih baru. Jika Anda suka menonton bagaimana orang lain bermain, maka daftar gamer dapat disederhanakan dengan menetapkan prioritas. Berbagai aliran dapat ditandai sebagai favorit, agar tidak terlihat lama di masa mendatang. Jika mau, Anda bahkan dapat memberi hadiah kepada gamer dengan transfer uang.

Pengguna YouTube Gaming dapat mengimpor langganan YouTube mereka.
