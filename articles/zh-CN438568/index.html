关于量子计算机：不同国家如何开发这项技术

我们正在谈论谁已经在云中使用量子计算，以及政府正在推动哪些举措来支持量子计算行业。


/ Flickr / Steve Jurvetson / CC BY /照片编辑

谁在云中使用量子计算机


诸如Google，IBM，Intel之类的大型组织以及D-Wave和Rigetti初创公司等许多小型IT公司正在创建自己的量子计算机。 他们中的一些人为大学和研究机构提供了访问其解决方案的途径。 云基础架构可以帮助他们。 例如， IBM和阿里巴巴分别开放了对20位和11位计算机的访问权限。 要与安装一起使用，请使用IBM Q和阿里云云平台。 这些计划的目的是帮助工程师使用复杂的机器学习模型和各种实际现象模型： 从气候模型到金融市场建模。

加拿大公司D-Wave Systems也推出了其云服务。 她的客户每月每分钟可获得一次量子计算。 这似乎有些不足，但是仅几毫秒就足以使量子机器完成许多任务。 我们谈论的是大量分解和构建路径-传统计算机在这种计算上花费数天和数周。

几家公司已经在使用D-Wave系统的功能。 例如，空中客车公司使用基于量子位的计算机来制造新飞机-完成所需计算的速度是传统计算机的四倍。 大众转向量子计算机来开发无人车辆控制算法。

参与量子计算开发和推广的不仅是IT公司（例如，IBM 成立了一个用于量子计算机的软件开发人员社区）。 许多国家的政府开始促进旨在发展量子产业的特殊金融计划和法律。 稍后我们将讨论其中的一些。

美国量子计划


去年12月中旬，美国参议院通过了建立国家量子计划（NQI） 的法案。 新法案的目的是支持致力于创建量子计算机和网络的公司和研究机构。 NQI计划的设计时间为十年，将为实施计划拨款12亿美元。


/ Flickr / IBM研究 / CC BY-NA

该项目计划建立两个到五个新的量子技术研究中心。 美国能源部将对此负责。 它还将为研究提供资金-每个中心每年将获得2500万美元的拨款。 这笔钱也将用于开发量子开发领域专家的教育计划。 美国国家科学基金会将为大学生准备一项特殊的教育计划。

负责计划项目实施的是一个专门创建的机构-国家量子协调办公室。 其工作人员将监督研究计划并处理咨询问题。

来自其他国家的倡议


在欧盟，中国和俄罗斯也创建了量子计算机产业发展计划。 2016年在欧洲启动了“量子旗舰计划”。 政府和欧盟成员国已拨款10亿欧元支持量子计算领域的20个项目。 而且，这笔钱将不仅用于创建计算机。 量子旗舰包括理论物理学，量子计量学和通信领域。

资助项目的一个例子是牛津仪器公司的QMiCS 。 其目的是在量子设备之间创建无线网络。

此外，两个Atos项目也获得了资助。 首要任务是创建一个基于离子阱的50量子位量子计算机， 离子阱是最常见且最稳定的量子位类型之一。 第二个项目的目标是模拟 500个量子比特的系统。 迄今为止，仅对60比特的计算机进行了建模。

量子计算领域也在中国发展。 2016年，中国开始建立新的研究实验室-中国政府为此拨款约10亿美元。 科学中心将于2020年开始运营。

政府先前的财政支持帮助中国科学家在量子计算领域取得了首个成功。 2016年，一组研究人员将一颗卫星发射到了轨道上，其中的通信信道受到量子加密的保护。 这项技术还用于地面网络：在北京和上海之间铺设了一条长度约为2,000公里的量子通信通道。


/ Flickr / IBM研究 / CC BY-ND

他们还参与了俄罗斯的量子计算。 2018年2月，高级研究基金会，Vnesheconombank，莫斯科国立大学和数字经济ANO等多个组织创建了一个量子财团，其任务是到2021年开发出50量子比特的计算机。

莫斯科国立大学之前一直与量子计算相关联。 2017年，该大学获得了建立量子技术中心的赠款。 其目标是开发一种量子数据加密系统。 ITMO大学也参与这一领域：2017年，政府拨款 1.6亿卢布用于大学和电信公司的联合项目。

金融项目的前景


美国政府的法案得到了大型IT公司（例如Microsoft）的支持。 据该公司的代表说，政府的支持对于创建可靠的多量子位芯片是必要的，这些芯片不允许计算错误。

“这是业界最重要的任务之一。 1cloud.ru的开发主管Sergey Belkin说，处理器包含的量子比特越多，它们开始相互影响彼此的工作并产生干扰的机会就越大 。 -到目前为止，还没有人成功达到零错误值。 这阻止了我们提高量子机器的计算能力。”

同时，国家资金将帮助那些无法为研究量子技术所需的所有设备付费的组织。

在政府的支持下，中国公司在量子加密方面取得了长足的进步。 预计不久后，量子密码术将在美国开始密集发展。 据预测 ，新的美国法律将加速量子加密算法标准的开发，到2024年该技术将开始在Web服务中实现。

政府资助将有助于在俄罗斯发展量子技术。 可以预料，由国家机构赞助的项目最终将建立大规模的量子网络并设计多量子芯片。
