Pilih objek


Semuanya dimulai dengan pemilihan objek. Di Windows File Explorer dengan seleksi, semuanya secara alami buruk - sulit untuk memilih banyak objek dan tidak sengaja kehilangan pilihan. Pilihan pilihan dengan kotak centang tidak terlalu membantu. Google Drive adalah tentang sampah yang sama, tetapi setidaknya Anda tidak dapat kehilangan pilihan dengan mengklik di antara ubin secara acak.


Saya percaya bahwa objek harus dibuka dengan 1 klik (setelah semua, mungkin perlu untuk mengklik begitu banyak sehingga jari sakit), dan mode pemilihan diaktifkan dengan tekan lama (seperti pada telepon), atau dengan menyorot satu objek menggunakan ikon pilihan (terutama untuk gerobak dorong). Tindakan dan Antrian


Hanya yang malas tidak mengolok-olok memperkirakan waktu salin di Explorer, saya suka opsi xkcd .



Dalam File Explorer terakhir, dialog dengan kemajuan penyalinan / pemindahan dibuat tampak dapat ditoleransi, antrian berfungsi dengan baik, tetapi perilakunya tidak benar - pertama-tama menyalin bagian dari file, kemudian mengajukan pertanyaan, seperti konflik nama, kemudian melanjutkan, kemudian bertanya lagi ...


Saya benci Windows karena aplikasi apa pun dapat tiba - tiba muncul (memotong input keyboard) di atas jendela yang sedang Anda gunakan! Anda dapat secara tidak sengaja menyodoknya tanpa melihat dan menimpa informasi penting. Sangat nyaman Pernahkah pencipta menemukan ini? Ya, tentu saja mereka lakukan.

Pembatalan tindakan yang dilakukan hanyalah neraka. Cobalah untuk membatalkan 5 bagian mereka yang berbeda dan kemungkinan besar Anda akan mengerti apa yang terjadi jelas bukan yang Anda harapkan. Resolusi Konflik Nama


Ini yang paling banyak hhana bagian gila. Secara umum, secara historis ini adalah dialog paling liar dan paling berbahaya sepanjang masa. Ini seperti berjalan di ladang ranjau setinggi lutut dengan granat tanpa cek. Di bawah tembakan musuh. Di malam hari. Satu langkah yang salah dan file yang tak ternilai harganya akan menguap.


Saya tidak punya keinginan untuk mempelajari sejarah perang ini, saya hanya ingin menunjukkan apa yang bisa Anda peroleh dengan baik. Silakan baca panduan dari Microsoft tentang cara membuat salah satu dialog resolusi nama terburuk - Merancang pengalaman tabrakan nama file Windows 8 .


Di sana mereka pertama-tama membuat dialog yang baik, dan kemudian merusaknya dengan melakukan penelitian tentang slide menggunakan metode RITE. Hanya n - c. Itu harus dilestarikan untuk generasi mendatang! Makalah ilmiah dapat ditulis tentang ini.



Jadi, jika Anda tiba-tiba tidak tahu, dalam versi terbaru dari OS paling luas di dunia, Anda harus mengklik minimal 4 kali (dengan akurasi tinggi!) Untuk menyimpan objek bergerak dan yang ada di folder. Tidak ada tombol sederhana "Simpan keduanya".



Ini hanya Masterpiece of Shit . Tetapi mereka bangga dengan pekerjaan mereka! Bahkan video memang - " Memperbaiki dasar-dasar manajemen file kami ". Mengapa perusahaan terbesar di dunia tidak bisa merekrut desainer papan atas dan bukan dinosaurus ini? Jawabannya ada di bab pertama - Microsoft tidak peduli dengan desain, seperti orang lain.
