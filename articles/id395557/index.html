Motor digerakkan oleh EasyDriver . Nema 17



motor stepper . Kami mengontrol kecepatan dengan potensiometer . Daya terorganisir berdasarkan ukuran baterai 18650 . Kali ini, konektor untuk menghubungkan mesin telah ditingkatkan . Alih-alih Uni Soviet menggunakan MIC 4 PIN . Fungsi .
























Pelanggan hanya meminta satu mode operasi - gerbong yang tak terbatas bolak-balik, kecepatan dapat diubah dengan potensiometer setiap saat. Kami menghentikan carriage dengan sakelar sakelar bila perlu.

Sketsa:
#define xy A0
int Distance = 0;  
int x = 0;
void setup() {                
  pinMode(2, OUTPUT);     
  pinMode(3, OUTPUT);
  digitalWrite(2, LOW);
  digitalWrite(3, LOW);
  pinMode(xy, INPUT);
  
}

void loop() 
{
   x = analogRead(xy); 
x = map(x, 20, 700, 10, 4000); 

  digitalWrite(3, HIGH);
  delayMicroseconds(x);          
  digitalWrite(3, LOW); 
  delayMicroseconds(x);
  Distance = Distance + 1;   
  
  
 if (Distance == 1600*13.8)
  {
  
    if (digitalRead(2) == LOW)
    {
      digitalWrite(2, HIGH);
    }
    else
    {
      digitalWrite(2, LOW);
    }

    Distance = 0;
   
    delay(2500);
  }
}
