Nah, jika Kaspersky Lab tidak dapat menemukan cacat, para prajurit akan menemukannya. H harus mencarinya sendiri.

Konfigurasi layanan Windows disimpan di cabang registri: 

 HKLM\System\CurrentControlSet\services\

Tidak ada yang berguna disimpan di sistem file kecuali file biner.

Bagaimana cara kita memonitor registri? Alat paling serbaguna adalah Process Monitor dari Sysinternals .

Apa yang salah dengan Process Monitor? Sangat sulit menemukan di dalamnya jika Anda tidak tahu persis apa yang Anda cari.

Pada saat yang sama, ada utilitas dari perusahaan Nirsoft yang tidak begitu terkenal. Ini menghasilkan lusinan program unik - mulai dari memantau koneksi perangkat USB hingga membaca kunci produk dari registri. Jika Anda belum pernah mendengarnya, saya sangat merekomendasikan mengunjungi situs ini dan mengevaluasi koleksinya. Ketika saya pertama kali mengetahui tentang mereka - seolah-olah saya membuka sebuah kotak berisi mainan.

Utilitas www.nirsoft.net/utils/registry_changes_view.html akan berguna untuk pekerjaan kami
