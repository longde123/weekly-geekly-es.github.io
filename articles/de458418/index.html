5 Browser-Erweiterungen für die Arbeit mit englischem Text



Hallo allerseits! Ich teile weiterhin meine Berufserfahrung im Bereich der Arbeit mit englischen Texten. Heute werden wir darüber sprechen, welche Browsererweiterungen verwendet werden können, um die Erstellung hochwertiger Materialien zu erleichtern und um zu lernen, besser auf Englisch zu schreiben. Lass uns gehen!

Hinweis : Der Text enthält Erweiterungen für Google Chrome und Mozilla Firefox, da ich nur diese Browser verwende. Nicht alle genannten Produkte haben eine Version für beide Browser.

Linguix


Dieser Service ähnelt Grammatik - er findet auch Tippfehler und Grammatikfehler in den Texten, kostet aber fast die Hälfte des Preises. Es gibt auch Unterschiede - zum Beispiel eine Datenbank mit Inhaltsvorlagen, mit denen Texte verschiedener Kategorien schneller geschrieben werden können (z. B. Pressemitteilungen oder Unternehmensnachrichten).



Von den Funktionen, die mir gefallen - das Statistikberechnungsmodul. Das System analysiert die „Lesbarkeit“ des Textes und zeigt an, wie komplex er ist. Nützlich, um der Versuchung entgegenzuwirken, Text zu sperrig zu machen.



Eine weitere praktische Sache ist der Pastebin-Bearbeitungsmodus in der Webversion, der wie ein geheimer Chat im Messenger funktioniert. Nach dem Schreiben und Bearbeiten des Textes wird dieser vollständig gelöscht.

Version für Chrome | Firefox-Version

Schriftsteller


Eine Masthead-Anwendung für alle, die mit einem Browser mit englischen Texten arbeiten. Mit dieser Erweiterung können Sie in Chrome einen äußerst praktischen, hochgradig anpassbaren und nicht ablenkenden Editor öffnen. Es sieht sehr cool aus:



Linguix arbeitet im Editor - so können Sie sofort nach Fehlern suchen

Aus erfolgreichen Entscheidungen - Die Erweiterung speichert den Text von Zeit zu Zeit automatisch. Sie können Ziele festlegen, auf einfache Weise Statistiken über die Anzahl der Zeichen im Text abrufen usw. Trotz der Tatsache, dass die Anwendung eine separate Seite im Chrome-Erweiterungsspeicher hat, wird sie tatsächlich zur Webversion des Dienstes umgeleitet, und Sie können sie von jedem Browser aus verwenden.

Writer Browser Version

Google Wörterbuch


Mit einem Wörterbuch von Google können Sie ein unbekanntes Wort übersetzen. Dies kann hilfreich sein, wenn Sie mit englischsprachigen Materialien zur Textvorbereitung arbeiten. Mit dieser Erweiterung können Sie leicht die Bedeutung jedes Wortes lernen, auf das Sie auf einer Webseite stoßen. Sie müssen nur ein Wort markieren, und ein Popup mit einer Übersetzung wird angezeigt.



Eine bequeme Sache - das Programm kann Wörter laut aussprechen, so dass Sie die Aussprache aufrufen können. Es stellt sich heraus, dass das Google-Wörterbuch nicht nur für die direkte Bearbeitung des Textes nützlich ist, sondern auch als Werkzeug zum Erlernen der Sprache.

Chrome-Version

Dunkler Leser


Um ehrlich zu sein, sitzen viele von uns bis in die Nacht ständig am Computer, auch in völliger Dunkelheit. Einige, zum Beispiel ich, können sogar in diesem Format arbeiten, auch an Texten.

Es ist unnötig zu erwähnen, dass die meisten Web-Texteditoren hellweiße Seiten mit schwarzen Buchstaben sind. Im Dunkeln ist dies eine äußerst schwerwiegende Augenbelastung, daher verwende ich die DarkReader-Erweiterung. Es verdunkelt den Hintergrund auf Webseiten, so dass es viel angenehmer ist, abends mit ihnen zu arbeiten. So sieht beispielsweise die Medium-Blogging-Plattform mit dieser Erweiterung aus:



Version für Chrome | Firefox-Version

Konzentrieren Sie sich


Aufschub ist einer der Hauptfeinde aller, die mindestens einmal versucht haben, an Texten zu arbeiten. Das Schreiben eines besonders langen Textes, ohne von sozialen Netzwerken, E-Mails und Nachrichten abgelenkt zu werden, ist eine äußerst schwierige Aufgabe. Die StayFocusd-Erweiterung hilft mir, das Problem zu lösen.



Damit können Sie verhindern, dass Sie eine bestimmte Site für eine Weile verlassen. Wenn Sie also einen Blog-Beitrag schreiben müssen, werden Sie bis zum Schreiben nicht zu Facebook gehen. Mit dem Beginn der Verwendung dieser Erweiterung hat sich meine Produktivität deutlich erhöht. Ich empfehle es.
