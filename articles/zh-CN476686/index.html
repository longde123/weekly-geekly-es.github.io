为什么选择覆盆子？


Raspberry Pi是一种小型且便宜的通用微型计算机，可以灵活配置以完成任何任务。 自2014年以来，它已作为独立的计算模块计算模块发布，即从熟悉的具有各种接口和连接器的板上发布，仅保留最重要的部分：处理器，ROM和RAM。 这种设计使您可以将该设备用于任何可能的和难以想象的任务，这完全取决于是否有可能在模块周围创建线束。 还值得注意的是，自从发布设备的第一个版本以来，已经发布了三个版本的模块，并且在发布Raspberry Pi 4之后，您可能会在不久的将来看到第四个版本。 所有这些都表明开发人员正在积极开发其产品，以提高其功能和速度，并且其设备在人们中非常受欢迎。 这种流行并不是没有根据的：在所有这些时间里，他们已经将自己确立为能够在任何条件下甚至在太空中解决各种任务的可靠设备。 在Raspberry Pi上编程也是有条件的简单，因为Raspberry Pi在世界范围内拥有大量的集成商。

开放的Linux操作系统允许您根据所需的任务在设备上绝对安装任何软件。 例如，对于智能家居领域的解决方案，可以与OpenHab，Home Assistant，iRidiumMobile，NodeRed等对接（可以在此处找到将设备用于智能家居的示例）。 对于工业来说，可以安装SCADA系统，例如CODESYS，Rapid SCADA，OpenSCADA，并能够使用该设备作为网关，使用MQTT，http，REST API或CoAP协议将数据传输到上层。 也可以与各种云服务集成。

有哪些接口？



图2-俯视图

执行最少的设备支持以下接口：

  • RS485 x 2;
  • RS232 x 1;
  • CanBus x 1;
  • 1-线x 1;
  • USB x 1;
  • 以太网x 1;
  • SMA x 2;
  • SIM卡x 1;
  • miniPCIe x 2;
  • HDMI 4k x 1;
  • MicroUSB x 1;
  • MicroSD x 1;
  • GPIO x 1。
  • LED x 1（可编程）

以上接口允许您在任何项目中实现设备。 第三方制造商的其他用于在Mini PCI-e插槽中安装的硬件模块也解决了设备上通信和Internet可用性的问题。 我们没有选择安装通信模块的方法，因为USB适配器（所谓的“哨子”）的存在不是一个非常可靠和高质量的选择，并且安装工业路由器（如Robustel R2000-3P）非常昂贵（大约12 tr ）。 因此，我们决定为Mini PCI-e模块安装两个插槽，可以随意使用：

  • 3G，LTE，GPRS模块（华为MU709s-2，价格：2.5 tr）;
  • 具有定向天线功能的Wi-Fi模块；
  • NB-IoT模块；
  • 用于构建物联网网络的LoraWan模块。

因此，在为特定任务选择正确的通信接口时存在一定的可变性和灵活性。
