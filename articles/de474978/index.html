IBM Watson Visual Recognition: Objekterkennung jetzt in IBM Cloud verfügbar



Bis vor kurzem wurde IBM Watson Visual Recognition hauptsächlich zum Erkennen eines Bildes als Ganzes verwendet. Das Arbeiten mit einem Bild als Ganzes ist jedoch alles andere als richtig. Dank der neuen Objekterkennungsfunktion haben IBM Watson-Benutzer jetzt die Möglichkeit, Modelle auf Bildern mit markierten Objekten zu trainieren, um sie anschließend auf beliebigen Frames zu erkennen.

Wir zeigen, wie das jetzt geht.

Wenn Sie früher mit Hilfe von IBM Watson ein beschädigtes Auto von einem unbeschädigten Auto unterscheiden konnten, können Sie jetzt nicht nur das Vorhandensein eines Schadens erkennen, sondern auch dessen Position und Größe bewerten. Dieser Ansatz ist sehr viel informativer und ermöglicht es Ihnen, Vorhersagen über die Kosten notwendiger Reparaturen zu treffen.
Natürlich ist die Liste der Optionen für die Verwendung dieser Funktionalität viel breiter als eine einfache Prüfung der Fahrzeugintegrität. Jetzt können Sie Watson Visual Recognition verwenden, um:

  • Zählung der Anzahl Personen in Linien oder Autos im Verkehr
  • Identifizierung von Waren in Handelsregalen
  • Erkennen von Logos in Fotos
  • Analyse von CT-Scans und MRT auf Anomalien
  • Andere Aufgaben im Zusammenhang mit der Arbeit mit bestimmten Objekten in den Bildern

Sie müssen keine Monate mit der Auswahl und Markierung von Daten verbringen - unser Modell ist bereits auf mehrere Millionen Stichproben trainiert und bietet eine relativ hohe Vorhersagequalität ohne Änderungen. Bei Bedarf können Sie es jederzeit neu trainieren, damit das neuronale Netz den Besonderheiten Ihres Tätigkeitsbereichs entspricht.

Kennzeichnen Sie Bilder und trainieren Sie das Modell mit Ihren Daten schneller mit Watson Studio


Normalerweise ist es die schwierigste Aufgabe bei der Erstellung eines Computer-Vision-Systems, Ihr eigenes Modell auf das genaue Erkennen von Objekten zu trainieren. Watson Studio beschleunigt diesen Vorgang und hilft, die Zeit bei der Arbeit mit großen Datenmengen zu verkürzen. In Verbindung mit dem kostenlosen Add-On Auto Label können Sie alle Bilder im Datensatz schnell markieren.

Erste Schritte


Stellen Sie nach dem Aktivieren und Erstellen der Visual Recognition-Anwendung in der Cloud eine Verbindung zu Watson Studio her und erstellen Sie ein Modell im Fenster "Objekte erkennen" im Abschnitt "Benutzerdefinierte Modelle".



Laden Sie Ihre nicht zugewiesenen Daten in Watson Studio hoch (Sie können ein JPEG-, PNG- oder ZIP-Archiv verwenden, das diese Bilder enthält).



Wählen Sie ein Bild aus, markieren Sie das Objekt, das Sie erkennen möchten, geben Sie ihm einen Namen und speichern Sie es. Wiederholen Sie diesen Vorgang, bis Sie alle erforderlichen Objekte in diesem Bild ausgewählt haben.


Sobald Sie einige Bilder markiert haben, können Sie Ihr Modell trainieren und testen.



Sie können auch weitere Bilder hinzufügen, um die Qualität des Modells mit der Funktion "Automatische Beschriftung" zu verbessern, mit der Sie alle Ihre Daten kennzeichnen können. Um diese Funktion zu verwenden, wählen Sie alle erforderlichen Bilder aus und klicken Sie auf die Schaltfläche „Auto Label“ (Automatische Beschriftung), damit Watson die Daten unabhängig nach den angegebenen Klassen kennzeichnet.



Nachdem Sie die Genauigkeit Ihres Modells überprüft haben, können Sie eine fertige Lösung in Ihr Produkt einbetten.



Testen Sie die Objekterkennung noch heute kostenlos mit IBM Watson Visual Recognition!
