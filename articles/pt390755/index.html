Grupos no Slack



No Slack , como em qualquer outra sala de bate-papo, há uma oportunidade conveniente de entrar em contato com alguém @userpor nome ou apelido. Para todos, o apelido se torna um link e, quando clicado, um menu é aberto para esse usuário. E o apelido é destacado para o usuário e uma notificação é enviada automaticamente.

Mas e se você precisar entrar em contato com vários usuários ao mesmo tempo? Existem várias maneiras de fazer isso.

A primeira maneira: mudar para o Slack pago


Profissionais de Slack pago


  • Existe uma capacidade embutida de se dividir em grupos.
  • Integração ilimitada.
  • Hospedagem ilimitada de arquivos.
  • Histórico de mensagens ilimitado e pesquisa neles.
  • Estatísticas detalhadas.
  • Acesso para convidados.
  • E assim por diante

Contras de folga paga


Apesar do grande número de vantagens, também existem desvantagens para a versão paga:

  • Caro O plano tarifário mínimo é de US $ 6,67 por pessoa no primeiro mês e US $ 8 no seguinte.
  • A personalização da interface não é fornecida, exceto pelo esquema de cores.
  • Nenhuma vídeo chamada incorporada (em breve).

A segunda maneira: imitar grupos por meio de palavras destacadas


Para fazer isso, nas configurações de notificação no campo Destaque palavras, você pode listar todos os grupos necessários com uma vírgula.



Para descobrir quais grupos existem ou adicionar um novo, você pode usar o bot interno adicionando palavras-chave para que ele responda a elas e poste ajuda sobre ele.



A resposta do bot para -groups:

Type `@` with one of these groups: `manager`, `react`, `js`. Example: `@example`. It will send notifications automatically when you add necessary groups in Highlight Words: https://slack.com/account/notifications#highlight_words_div

A resposta do bot para -channels:

Type `#` with one of channels in Slack. Example: `#example`. It will send notifications automatically when you add necessary channels in Highlight Words: https://slack.com/account/notifications#highlight_words_div

Então o bot responde no chat.



Diferença de grupo


Grupo começando com:

  • @ - um grupo independente, parece texto comum, mas também pode ser um link se já tiver sido usado pelo apelido de alguém.
  • # - um grupo com pessoas desse canal se torna um link para o canal, geralmente usado para acessar outros canais.

Os benefícios de destacar palavras


  • De graça.
  • Pode ser dividido em grupos de usuários a partir de canais.

Contras de destacar palavras


  • É necessário que cada usuário prescreva para si todos os grupos necessários.

A terceira maneira: delegar toda a rotina ao bot


Se algo está faltando no Slack grátis, um bot vem em seu socorro.

Os grupos podem ser armazenados de qualquer maneira conveniente, por exemplo, na configuração ou, se você precisar adicionar do bate-papo por meio de comandos, no banco de dados.

Ao acessar um grupo específico, se houver um no dicionário, o bot o procura na descrição do perfil de todos os usuários e publica uma mensagem com uma lista de nicks, e todos esses usuários recebem uma notificação já no nível da folga.

Vantagens do bot


  • De graça. Você pode implementar recursos pagos.
  • Personalização aprimorada de mensagens.
  • Você pode implementar qualquer recurso e ainda mais poderoso.

Contras bot


  • É necessário escrever um bot (mesmo usando soluções prontas).
  • Bate-papo de spam com mensagens intermediárias. Obviamente, você pode excluir a mensagem imediatamente, mas será apenas irritante.
  • Depende do conteúdo da descrição do perfil com um comprimento limitado de 249 caracteres.

Conclusão


Quando há muitos usuários no Slack, surge a necessidade de se dividir em grupos para aqueles que são mais competentes e se especializam estreitamente em qualquer campo. Os recursos de um Slack pago sempre podem ser implementados de maneiras alternativas: através de recursos internos e através de um bot.
