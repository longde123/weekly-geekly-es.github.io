VS Code - Cuplikan Wayang

Ekstensi Visual Studio Code ini menambahkan potongan kode bermanfaat yang telah ditentukan sebelumnya untuk Puppeteer. Periksa API untuk informasi lebih lanjut tentang opsi yang tersedia.



‚Äç Bagaimana cara menggunakan?


Instal ekstensi terlebih dahulu.


Faktanya, ada dua cara untuk menggunakan cuplikan:


1. Mengetik langsung


Mulailah mengetikkan awalan atau hanya sebagian dari fragmen. Kemudian temukan fragmen dalam daftar kalimat dan tekan TAB atau ENTER .


2. Command Palette


Buka Command Palette dan ketik Insert Snippet . Kemudian cukup pilih cuplikan yang disorot.


Daftar cuplikan


Di bawah ini adalah daftar cuplikan yang didukung.
