Internes PHP


  • [RFC] Deprecate Backtick Operator (V2) - En PHP, vous pouvez utiliser des guillemets ` ` comme alias pour shell_exec() . Autrement dit, la ligne qui y est écrite sera perçue comme une commande et lancée dans le shell. La proposition de déclarer cette opportunité obsolète a provoqué une tempête de controverses , tout comme d'autres propositions de dépréciation.
  • [RFC] Implémenter de nouvelles API DOM Living Standard dans ext / dom - Il est proposé d'implémenter la prise en charge de la nouvelle norme DOM dans l'extension du noyau. En savoir plus sur l'offre dans le podcast. PHP Internals News # 31 avec l'auteur de RFC Benjamin Eberlei .
  • Un sondage de Sarah Goleman sur Twitter sur l'idée de blocs d'essai au niveau de la fonction, inspiré d'une fonctionnalité similaire de C ++:

    Blocs d'essai au niveau de la fonction en PHP:

    fonction f () essayez {
    // code
    } catch (Throwable $ t ) {
    // code
    } enfin {
    // code
    }

Les outils


  • mougrim / php-xdebug-proxy - Proxy PHP Xdebug: lorsque les fonctionnalités standard de Xdebug ne suffisent pas .
  • php-defer / php-defer - Un analogue de la fonction de report de Go . Le rappel passé en paramètre de cette fonction fonctionnera lorsque le thread d'exécution quittera la portée de la fonction / méthode actuelle (externe). Un destructeur est simplement utilisé sous le capot. L'idée est ancienne et a déjà été implémentée il y a plusieurs années dans le cadre de phplang / scope-exit .
  • sensiolabs-de / deptra 0.6 - Outil d'analyse de code statique pour déterminer les dépendances entre les couches d'application.
  • graze / parallel-process - Vous permet d'exécuter facilement plusieurs symfony / processus en parallèle.
