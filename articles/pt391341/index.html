Espionagem de alta tecnologia: gravando o som de uma impressora 3D em funcionamento, você pode reproduzir uma peça impressa anteriormente



Uma equipe de pesquisadores da Universidade da Califórnia (Irwin, EUA) descobriu que, ao gravar o som de uma impressora 3D em funcionamento, é possível reproduzir uma parte impressa anteriormente por esta impressora. É verdade que, para que tudo dê certo, o "espião" deve ter um modelo de impressora semelhante. Ele deve ser usado para calibrar o software "spyware" . Após a conclusão do programa de treinamento, você pode usar o som de uma impressora funcionando com um modelo semelhante. O resultado é uma cópia do que a impressora imprimiu - o "doador" do som.

Os autores testaram sua idéia na prática usando uma impressora FDM. Como se viu, a precisão da reprodução de detalhes é de cerca de 90%. É verdade que os detalhes não foram impressos da maneira mais complexa. Presumivelmente, a precisão de reprodução de partes mais complexas será menor.



Os pesquisadores chamam o método de copiar detalhes usando o som "ataque acústico". Esse método pode ser usado para espionagem industrial, e é uma maneira bastante eficaz e barata - apenas um smartphone é necessário para gravar o som.



É verdade que o método em si tem várias desvantagens significativas, uma das quais é a necessidade de colocar um smartphone perto de um sistema em funcionamento. Além disso, o próprio algoritmo não funciona muito bem em operações curtas da impressora - o som é muito curto.

Para se proteger de um "ataque acústico", dizem os autores do estudo, será suficiente introduzir operações intermediárias que não carregam uma carga semântica especial. Além disso, o controle de acesso regular à área de trabalho também ajudará.
