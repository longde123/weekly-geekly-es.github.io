智能手机的演变。第1部分：1994年-2010年。版本1.1

智能手机的文章演变。第1部分：1994年-2010年，有关遗漏的移动设备的话题（制造商的产品线和特定型号）收到了大量评论。我考虑了这些评论，并在其中增加了第一篇文章中未包含的最有趣的模型，以及您在评论中谈到的模型。

在评论中，我再次请您分享您的回忆并提供指向您自己的评论的链接。

如果文章缺少特定的模型，请在带有图像的评论中写出。

棕榈Treo 180


2002年，著名的PDA制造商Palm推出了其第一款智能手机-即具有在电话中通话能力的PDA。该设备以翻盖形状尺寸接收了分辨率为160x160的单色显示器。该小工具由Dragonball 33 MHz处理器提供动力，并具有16 MB的内部存储器。

从此设备开始，Palm专注于智能手机的发布。







Palm Treo 600


Palm Treo 600智能手机于2003年发布。它可以在Palm OS v5.2.1H上运行，具有32 MB的内存和一个SD插槽。该设备的显示屏为2.5英寸，分辨率为160x160，但它是触摸感应的，显示4096种颜色。该设备重168克。



摩托罗拉MPx200


摩托罗拉于2003年推出了使用Microsoft Smartphone 2002操作系统的MPx200智能手机，该翻盖式显示器接收了一块2.2英寸的9行内部显示屏，分辨率为176x220像素，内存为8兆字节，并支持高达1 GB的MMC和SC。户外显示器的分辨率为80x48像素。



索爱P900


索尼爱立信P900于2003年推出。这是一个带触摸屏的小工具，分辨率为208x320像素，内部内存为16兆字节，操作系统为Symbian 7.0。他有一个时尚的铰链盖，可以将其折叠成按钮智能手机。



索爱P910


P900的后继产品是2004年发布的P910。该设备的设备盖上有完整的QWERTY键盘。内存从16兆字节扩展到64兆字节，最大支持2 GB的Memory Stick Duo。该设备在159 MHz ARM9处理器上工作。此模型在HTML支持方面也不同于其前身。



诺基亚7710


2004年11月2日发布的诺基亚7710智能手机已成为该制造商最令人难忘的小工具之一。这是制造带触摸屏智能手机的首次尝试。外形有点不寻常-但是在当时很平常。



诺基亚E60


2005年10月，诺基亚发布了诺基亚E60-基于Symbian OS 9.1的设备接收了64 MB的内部和RAM，这是一块2.1英寸的显示屏，分辨率为352x416。17毫米厚的小工具重117克。该浏览器支持WAP 2.0 / xHTML和HTML。





以太m600


台湾电信供应商和制造商E-Ten于2005年推出了Eten M600智能手机。这款具有2.8英寸TFT显示屏，分辨率为240x320像素的设备采用了手写识别技术，64 MB RAM和128 MB内置1.3 MB像素带LED闪光灯的摄像头。该小工具基于Samsung 2440 400 MHz处理器。操作系统-Microsoft Windows Mobile 5.0 PocketPC。

在1990年代，这家制造商占领了中国PDA市场的85％。



索爱M600


2006年初，索尼爱立信M600在Symbian OS v9.1上发布。这个小工具配备了QWERTY键盘，2.6英寸TFT屏幕（分辨率为240x320像素）以及中英文手写识别软件。处理器-32位Philips Nexperia PNX4008 208 MHz。该设备的内部存储器为64 MB，但是可以借助容量最大为4 GB的Memory Stick Micro对其进行扩展。





诺基亚E61


诺基亚E61i智能手机于2006年问世，一年后，其继任者诺基亚E61i进入市场。这个小工具支持3G，并且可以在Symbian OS上运行。



三星SGH-i320


2006年2月，三星发布了Samsung SGH-i320-一种在Microsoft Windows Mobile 5.0操作系统上带有QWERTY键盘的智能手机。带有2.2英寸屏幕的设备仅重95克，厚度为11.5毫米。极其紧凑的设备。



华硕p505


2006年3月，华硕发布了P505。这款智能手机重165克，具有2.8英寸的屏幕，分辨率为240x320像素，带有铰链式盖子，64 MB的RAM和闪存。这个小工具可以在PDA的Microsoft Windows Mobile OS操作系统上工作。



索爱W950


Symbian OS v9.1上的智能手机索尼爱立信W950于2006年发布。他重112克，表壳的厚度-15毫米。小工具具有4 GB的内部存储器。显示屏为2.6英寸，分辨率为240x320像素。设备支持手写笔输入。


图片： 1xaoc2



诺基亚3250


诺基亚3250于2006年第一季度开始销售。其追随者是传奇的诺基亚5700 XpressMusic。这个小工具收到2 GB的内存，足以存储500首歌曲。他重115克。该设备可在Symbian OS 9.1上运行。



富士通西门子Pocket LOOX T810


2006年，发布了富士通西门子Pocket LOOX T810智能手机。它支持3G，具有2百万像素的摄像头，QWERTY键盘，并具有支持手写笔输入的触摸屏显示器。



华硕p525


华硕P525于2007年发布。它是运行2.8英寸显示屏，分辨率为240x320像素，128 MB内存，Wi-Fi和蓝牙的MS Windows Mobile 5.0操作系统的设备。和一个2百万像素的相机。

这个小工具的一则评论是：“将设备放在裤子口袋里，甚至少穿衬衫，不是很方便。他很棒，而且会掏腰包。”



i-mate SP3i


i-mate SP3i建立在HTC Typhoon平台上，并于2007年初开始销售。这是运行用于Smartphone SE的Windows Mobile 2003的直板游戏中的小工具。分辨率为176x220像素的彩色屏幕最多可显示65,000种颜色。称重该小工具102克，并在处理器TI OMAP 730 200 MHz上工作。

一个ServiceChip用户 记得该小工具甚至没有3.5毫米耳机插孔-而是2.5毫米插孔带有非标准接线，因此我不得不从另一部飞利浦电话上重做该耳机。



东芝Portege G900


2007年2月，东芝推出了东芝G900商务智能手机。该小工具的厚度为21.5毫米，重198克，可以在Windows Mobile 6 Professional上使用。该设备接收了40 MB的内部存储器和一个用于miniSD的插槽，128 MB的RAM，一个分辨率为480x800像素的3英寸显示器和一个QWERTY键盘。以及两个摄像头-主要的2百万像素带LED闪光灯和VGA的视频通话。

这个小工具已经有了指纹扫描仪。







诺基亚E90 Communicator


2007年2月，诺基亚发布了3G智能手机诺基亚E90操作系统Symbian。这个小工具更像是一台小型笔记本电脑，并且以相同的方式显示。第一版于2007年5月17日在印度尼西亚以5,000美元的价格出售。

内部屏幕的分辨率为352x800像素。





索爱P1


索尼爱立信P1是P990和最新P系列小工具的后继产品。它采用M600尺寸（上面已写过）制造，但就功能而言它接近P990。该设备在带有UIQ 3.0的Symbian OS 9.1上。与M600相比，该产品略厚，但比P990薄。该设备重124克，配备3.2像素像素相机和3倍变焦。小工具的显示屏为2.6英寸，分辨率为240x320像素。闪存-256兆字节，其中160兆字节可供用户使用。RAM-128 MB。作为一个不错的奖励，该套件包括一个512 MB的Memory Stick Micro卡。

该小工具已于2007年5月8日发布。



新1973


2007年7月，基于开源和基于Linux的免费Openmoko平台的Neo 1973智能手机开始销售。该设备配备了一个2.8英寸的TFT显示屏，一个既可以使用手指输入又可以使用手写笔的触摸屏，三星266 MHz处理器。RAM-128 MB，内置-64 MB。称量小工具184克。

另外，手写笔还包含激光笔，手电筒和钢笔。







棕榈Treo 750


2007年12月，Palm开始销售解锁版的Palm Treo 750-现在，不仅AT＆T订户可以使用它。



HTC的优势


2007年，HTC在Intel XScale PXA270 624 MHz处理器上发布了Advantage。这个小工具已经收到8 GB的内存和128 MB的640x480像素5英寸可操作显示屏。然后他只是巨大的。但是他真的很重-400克。



新自由奔跑者


Neo 1973的后继产品是2008 Neo FreeRunner。这个小工具可让您运行各种基于Linux的操作系统-Android，Debian，Gentoo，Arch Linux，GameRunner，Slackware，OpenWrt，OpenMoko，SHR和QtMoko。



三星SGH-i900 WiTu（Omnia）


2008年6月，三星宣布以WIndows Mobile三星SGH-i900 WiTu（Omnia）为基础的通讯器。该智能手机配备了一块3.2英寸的TFT触摸屏，分辨率为240x400像素，128 MB的RAM和256 MB的内部存储器，一个具有自动对焦和闪光灯的5像素主摄像头。



Palm Treo Pro


在2008年，Palm推出了最新的Treo智能手机。它变成了Palm Treo Pro-它拥有触摸屏，GPS，3.4毫米耳机插孔（而不是2.5毫米）。顺便说一下，HTC制造了这款智能手机，我在上一篇文章中提到过很多型号。

小工具的屏幕是3.3英寸的TFT，分辨率为320x320。操作系统-Windows Mobile Professional 6.1。

使用microSDHC可以将256 MB的内部存储器扩展到32 GB。

该设备具有Wi-Fi模块





东芝图书馆


2009年发布的Toshiba Biblio滑块使用了一个有趣的Morphpad键盘：它既可以显示普通的电话键盘，也可以显示为QWERTY。设备的触摸屏为3.5英寸，分辨率为480x960像素。小工具还配备了具有自动对焦和稳定功能的5百万像素相机。内部存储器为7 GB，但是由于microSDHC，可以扩展它。





HTC Desire Z


HTC Desire Z智能手机获得了一个有趣的外形尺寸-它不仅是侧面滑盖，而且还以字母Z的形式打开的滑盖-从那里得到了名字。该设备配备了Android 2.2.1“ Froyo”，然后收到了对2.3的更新。这个小工具具有1.5 GB的闪存和512 MB的RAM，一个3.7英寸的屏幕，一个具有LED闪光灯和面部识别功能的5百万像素相机。



罗孚PC Pro G7


RoverPC Pro G7智能手机由Marvell PXA310 624 MHz和Qualcomm 6280处理器提供动力，具有128 MB的RAM和256 MB的内部存储器，具有2.8英寸的触摸屏，分辨率为480x640像素。该小工具可在Microsoft Windows Mobile 6.1 Professional操作系统下运行。该设备具有3G和Wi-Fi模块，重119克。

该小工具最有趣的功能之一就是电视输出端口的存在。





连结一


2010年初，HTC和Google推出了Nexus One智能手机。这是第一款使用Android 2.1的设备，并于当年夏天将其更新为2.2。自2011年以来，他从事2.3.3的工作。该设备可在带有Adreno 200的Qualcomm QSD 8250 Snapdragon ARM 1 GHz处理器上工作，该处理器具有512 MB的RAM和512 MB的闪存。这个小工具最多支持32 GB的存储卡，并且智能手机随附4 GB的卡。

该设备具有480英寸x 800像素的3.7英寸显示屏，带有闪光灯和自动对焦功能的5百万像素摄像头，支持Wi-Fi，蓝牙和HSDPA。
