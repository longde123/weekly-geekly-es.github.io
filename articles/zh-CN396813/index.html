打印管理解决方案概述

你好，Giktayms！解决与创建和提供公司打印基础结构相关的问题的方法始于其监视。强烈建议跟踪打印设备的状态，即使公司规模较小且所有需求都由一个网络共享MFP满足。即使在这种情况下，有一天您也可能根本没有打印机，寻找“实际上”的修复方法。如果组织拥有由几台（无论是10台还是100台）打印机和MFP组成的完善的基础架构，那么审核这样一个机群并非易事。在这里，您离不开特殊的软件解决方案和服务，将在讨论中对其进行讨论。



根据Gartner或Nuance的分析师，平均60％的员工每天在俄罗斯办公室打印11至50页。当然，在这种情况下，他们中的大多数人都不在乎雇主必须打印的费用数额。顺便说一句，它们（成本）可以覆盖到每位员工每年印刷数百美元的公司。

这些成本可以减少并正确分配。首先，必须避免滥用打印设备（“护照调查表”），遗忘打印机托盘中的打印输出，不适当的彩色打印，可能的盗窃（重新装填或使用兼容的墨盒而不是原始墨盒），可能的欺诈以及外包公司的大量账单如果您使用他们的服务。

我们不愿提及员工无良的话题，以免在Geektimes引起负面读者，但这是一个非常普遍的情况，如果没有它，今天的帖子将像没有酸奶油的罗宋汤。怎么样了负责打印设备的办公室工作人员说，耗材已用完。该应用程序上的公司购买原始墨盒。收到墨盒后，员工求助于一家专门购买原始（已读，被盗）墨盒的公司。根据我们的数据，例如，2016年原装惠普硒鼓的平均成本为10,500卢布。员工以很大的折扣出售原始墨盒，然后以更低的价格购买兼容墨盒或订购笔芯。



但是，仍然存在诸如设备的技术控制和消耗品的状态，组织打印机使用等任务。如果没有特殊工具，这简直是不可能的。

在我们的实践中，我们根据客户要解决的任务和打印基础架构的组成为客户提供各种方式。通常，领先的供应商在其产品组合中都有自己的发展。但是它们仅适用于仅使用一个制造商的设备的稀有公司。只有少数几个。打印机和MFP的队伍已经形成了很多年，在大多数情况下，都有来自不同制造商的设备和不同程度的“新鲜度”。因此，讨论通用解决方案是合乎逻辑的。

FollowMe（林代尔）




其中之一，FollowMe，可一次提供多个任务的解决方案，例如打印记帐和对设备的访问控制，以及根据设置进行任务路由。考虑到企业中所有打印机和MFP的使用，跟踪单个用户或工作组的活动以及交互式打印-FollowMe最适合此类任务。

MegaTrack（字体软件）




该应用程序监视打印设备并收集统计信息，以用于后续报告和分析。该解决方案的优点是不仅可以与网络一起使用，而且还可以与本地打印机一起使用。在每台打印机的设置中，您可以输入一张打印的费用，系统还将为每台设备发布经济计算。该解决方案支持Windows Cluster Server 2000/2003的群集配置，支持多种语言，并具有用于远程访问的Web界面。

Pcounter（AND技术公司）




Pcounter的一个功能是非常简单地集成到现有基础架构中，并随后进行扩展。该系统考虑了发送到设备的所有打印作业，收集了有关其使用情况的统计信息，并且作为一项附加功能，它具有重定向作业的能力。除打印外，Pcounter还可以组织和扫描。可选的可下载应用程序收集统计信息，以便使用公司MFP复制，扫描和发送传真。

JetAdvice 2.0（EuroForm A / S）




另一个易于使用的产品，用于收集有关公司打印设备和耗材的信息。JetAdvice 2.0监视连接到公司网络的设备中耗材的状态，并且当墨粉量下降到严重水平时，它会向负责支持或联系外部服务部门的员工发送电子邮件通知。对拥有各种各样“包括非标准设备”的各种供应商的打印机和MFP的公司来说，对众多品牌的支持将特别有价值。

Equitrac（Nuance公司）




这项决定的价值在于其开展竞选活动的能力，有助于提高公司员工对新闻界的负责任态度。当文档跟随用户时，Equitrac会创建一个虚拟打印队列，并且可以在连接到公司网络的任何打印机上进行打印。这也与用户有意识地选择最适合特定任务的设备有关。Equitrac不需要单独的服务器，可以轻松地安装在任何网络打印机或MFP上。

所有这些都是简单的解决方案，提供了不同的可能性。但是，还有一些打印管理工具可以解决各种问题，从监视整个基础结构到根据公司策略和权利提供对打印机和MFP的访问权限。

PrintStat（Nicotech）




PrintStat是用于监视市场上所有知名制造商的打印机和MFP的软件解决方案。该解决方案是真正的通用解决方案-模型库总计超过1000个点，并且随着制造商对其生产线的更新，该模型库得到了补充。这使您甚至可以在运行着从旧到最旧的所有设备的公司中使用PrintStat。

PrintStat的操作原理很简单-数据是从打印设备的内置传感器收集的，由系统合并并使用可自定义的界面进行可视化。可以通过Web界面从网络上的任何计算机上访问系统，并且您可以配置通过电子邮件定期发送通知。

PrintStat获取什么数据？它们的组成几乎完全满足了公司印刷监控所提供的需求。第一组数据是监视耗材状态。该系统评估已安装墨盒的寿命，根据打印机或MFP上的负载预测其消耗量，甚至可以将应用程序发送给耗材供应商。办公室走廊里臭名昭著的仓库与装满墨盒的箱子变得无关紧要。除了在没有消耗品及其存储的情况下优化成本外，还释放了人力资源。



自然地，仅监视PrinStat消耗品是不受限制的。该解决方案不断监视来自打印页面计数器和资源详细信息的数据。因此，可以监视设备及其资源上的当前负载。此信息对于优化办公室打印至关重要。通过比较两个功能相同或相似的MFP的负载，例如可以优化它们在房间中的位置，以便员工平均使用两个设备。

有许多示例，其中一台网络打印机处于闲置状态，而另一台网络打印机被过度使用，这种负载不均衡的原因是需要对其中一台设备采取一些额外的步骤。

在很大程度上管理办公室打印和监视打印以预测资源零件的故障并计划更换时，计划例行维护是同等重要的任务。这种维护方法使得可以提前进行支出预算，而不是“实际上”进行细分。结果也很明显-与紧急情况相比，任何公司的系统和计划费用总是更加简单和方便。

PrintStat的明显优势是能够将信息的收集和分析过程提供给从事设备维护的外包公司。为了预测服务，对于他们来说，跟踪电子邮件通知就足够了，而无需访问客户的公司基础结构。
PrintStat的另一个优势是在扩展或更新公司设备系列时自动缩放。系统独立检测新安装的打印机并开始跟踪其使用。

Print Manager Plus（打印管理器）




第二种最受欢迎​​且用途最广泛的产品，用于监视企业打印基础结构。 Print Manager Plus（PMP）具有比PrintStat更多的功能。可以使用它，包括拥有大量多样的打印机和MFP的大型公司。支持所有主要制造商的设备，从而确保在扩展或更新机队时不会遇到困难。 PMP不仅可以收集统计数据和控制消耗品，还可以通过跟踪用户使用情况和访问策略来帮助组织对打印机和MFP的访问。

例如，PMP允许您跟踪谁打印了什么以及什么打印机，并计算了特定用户或部门的打印成本。甚至根据给定条件在设备之间分配任务。例如，最好使用更高效的设备来打印大量文档，并且PMP允许您配置它们以将其自动发送到功能强大的打印机，并且按文件类型进行分发将可以满足办公室内部的机密性要求。

一个明显的优势是它自己的统计模块，该模块使您可以直接在PMP中分析打印基础结构的状态。同时，还有一个基于Crystal Reports的报表生成器，以及一百多个可视化选项。尽管PMP非常复杂，但它易于使用，并且几乎与任何平台软件兼容-从UNIX到Windows服务器系统（顺便说一句，该系统具有Microsoft的兼容性证书）。它会在一分钟内安装到服务器上，并且需要另外五个配置。

原则上，后两种解决方案能够满足与打印监控相关的所有需求，并同时解决其他任务，例如确保对打印机和MFP的访问权限。其中一个更适合大型公司，另一个更适合中小型企业市场的需求。但是，在部署其中任何一个之前，都值得对基础架构进行初步审核- 由专业集成商提出的另一项开发很有可能会是最好的。
