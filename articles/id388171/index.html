Mengikuti pembaruan Windows desktop untuk orang dalam seluler

Mengikuti desktop Windows , pembaruan dirilis untuk orang dalam seluler - build baru 10586.36



Kali ini pesan pembaruan akan lebih seperti habr-tweet.
Ini adalah pembaruan kumulatif kedua untuk Windows 10 Mobile.
Dari hal-hal menarik, perbaikan ada di bawah kap. Tidak ada daftar perubahan resmi.
Sementara itu, kami menunggu majelis baru dengan fungsi baru dan memperbaiki bug lama.
Juga dilaporkan bahwa distribusi Windows 10 Mobile ke perangkat yang tidak berpartisipasi dalam program orang dalam akan dimulai tidak lebih awal dari tahun depan.

Baca lebih lanjut tentang pembaruan di blog Microsoft (dalam bahasa Inggris) dan di blog saya (dalam bahasa Rusia)
UPD: Pada saat yang sama di Store, termasuk Store itu sendiri, banyak aplikasi sistem diperbarui, Store, misalnya, sangat didesain ulang.
UPD2:Bersamaan dengan ini, pembaruan kumulatif dirilis untuk Windows 10 non-orang dalam pada PC dengan perbaikan "umum".
