Chrome-Erweiterungen für die Webentwicklung und die Arbeit mit GitHub



Hallo an alle. Ich möchte Ihnen einige der Erweiterungen für Chrome vorstellen, die ich beim Entwickeln, Analysieren von Websites, Arbeiten mit GitHub und im Alltag verwende. Ich bin sicher, dass die Liste dieser Erweiterungen für die meisten Leser keine Öffnung sein wird, aber diejenigen, die zum ersten Mal lernen und sie installieren, werden zweifellos Spaß daran haben, mit neuen Tools zu arbeiten.

Ich denke, dass es sich nicht lohnt, ein langes Fußtuch mit einer ausführlichen Beschreibung eines Werkzeugs zu erstellen. Sie sind recht einfach und verständlich. Daher werde ich versuchen, kurz über die Vorteile ihrer Verwendung zu sprechen.

Mit GitHub arbeiten


Octotree ist meiner Meinung nach die nützlichste und bequemste Erweiterung. Sie können den Quellcode in Form eines Baums anzeigen, der auf der linken Seite angezeigt wird. Neue Navigation ist viel bequemer als Standard. Octotree ist besonders nützlich, wenn Sie an einem Großprojekt mit einer großen Anzahl verschachtelter Verzeichnisse arbeiten.

GitHub Repository Size ist ein praktisches und einfaches Plugin, das sowohl die Größe eines einzelnen Repositorys als auch die Größe jeder darin enthaltenen Datei anzeigt.

OctoLinker ist eine Erweiterung, die eine schnelle Code-Navigation bietet, indem einfache Codeelemente in anklickbare Links umgewandelt werden. Wenn Sie diesem Link folgen, können Sie die Quelle schnell finden. Abhängig von der Programmiersprache zeigt die Erweiterung die Quelldatei an oder führt den Benutzer zu einer externen Site. Elemente, die in einen Link konvertiert werden, sind mit rosa Punkten markiert, sodass Sie sofort sehen können, wo sich ein interaktiver Link befindet.

GitHub Hovercard bietet sofortigen Zugriff auf Benutzerinformationen, Repositorys, Fragen und Commits von überall auf der GitHub-Website. Bewegen Sie den Mauszeiger über die Informationen.

npmhub Wenn Sie ein Repository auf github.com mit einer Datei package.json anzeigen , analysiert diese Erweiterung die Abhängigkeiten im Paket und zeigt Links mit einer Beschreibung direkt unter der README-Datei an.

github npm stats zeigt ggf. Statistiken zum Herunterladen von Paketen auf der NPM-Site an.

Refined Github zeigt kürzlich veröffentlichte Brunchs an und ermöglicht es Ihnen, mit Anfragen zu arbeiten und Avatare von Benutzern anzuzeigen, die auf Kommentare geantwortet haben.

Isometrische Beiträge - diese Erweiterung ist nicht so nützlich wie die vorherigen, aber sehr interessant). Sie können zwischen einem regulären Beitragsdiagramm auf GitHub und einer Version der isometrischen Pixelkunst wechseln.

Mit Websites arbeiten


Eye Dropper ist ein großartiges Tool zum Anzeigen der auf Websites verwendeten Farben und zum Erstellen eigener Paletten. Ich benutze es fast jeden Tag)).

Vollseiten-Bildschirmaufnahme . Stellen Sie sich vor, Sie haben eine Landung auf 3-5 Bildschirmen entwickelt und wissen nicht, wie Sie daraus einen Vollbildmodus machen sollen. Drücken Sie fünfmal auf den Druckbildschirm und machen Sie Teile im Editor untereinander? Ach was! FPSC ist ein großartiges Tool, das in wenigen Sekunden einen Screenshot der aktuellen Seite erstellt. Dies ist sehr nützlich, wenn Sie die Site-Optimierung überprüfen müssen, ohne spezielle Services herunterzuladen, und die von Ihnen erstellte Site vollständig erfassen müssen.

WhatFont - Mithilfe dieser Erweiterung können Sie leicht herausfinden, welche Schriftarten auf der Site verwendet werden, indem Sie einfach auf die Erweiterung klicken und den Cursor an die gewünschte Stelle bewegen. Es ist einfach und elegant. Deaktiviert durch Drücken der "Esc" -Taste. Zeigt auch Textfarbe, Höhe, Schriftgröße und andere Parameter an.

PerfectPixel - Mit dieser Erweiterung können Sie Ihre Website pixelweise entwickeln.

CSS Peeper ist ein CSS-Analysetool, das speziell für Designer entwickelt wurde. Ziel der Erweiterung ist es, Designern die Möglichkeit zu geben, sich auf das Design zu konzentrieren und so wenig Zeit wie möglich mit dem Code zu verbringen. Probieren Sie es einfach in Aktion aus und Sie werden alles verstehen.

Für die tägliche Arbeit


Google Übersetzer ist ein weiteres Tool, auf das fast keiner meiner Arbeitstage verzichten kann. Dies ist keine Seite, dies ist eine Erweiterung. Damit können Sie den ausgewählten Text schnell direkt auf der Seite übersetzen, ohne zur Seite des Übersetzers zu wechseln. P.S. übersetzt keine langen Texte, aber es spielt keine Rolle.
