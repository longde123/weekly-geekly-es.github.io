Windows 95已使用20年



Windows 95操作系统于1995年8月出现。它标志着PC在世界范围内的迅速普及-因为从字面上的字面意义上来说，现在不仅计算机系统可供极客，而且还可供家庭主妇使用。当时（无论如何在美国），平均PC配置包括320 MB硬盘驱动器，8 MB RAM，386DX处理器或更高版本（建议安装486以便操作系统正常运行）。

系统的初始发行版在13个磁盘上交付。在此OS中，进行了一些创新，例如“开始”菜单，该菜单设计为左下角的按钮（在Windows 10中，按钮和菜单的返回形式当然有所变化）。使用菜单的简单性使得可以快速有效地使用机器和应用程序的文件系统。



此外，Microsoft开发人员还增加了对长文件名（最多256个字符）和即插即用系统的支持。 Wikipedia告诉我们：“ Windows 95的主要创新之处在于能够运行基于Win32 API的32位应用程序。该功能首次出现在Windows NT中，但是该家族的系统对硬件的要求更高，因此无法与“常规” Windows系列（在Windows 95发行之前以Windows 3.x系列为代表）进行比较。

此外，Internet Explorer与Windows界面（Explorer.exe）集成在一起，从而消除了Internet Explorer文件的完全删除，而不会破坏系统的功能。

自Windows 95问世以来，发生了许多变化，例如，计算机和电子设备无处不在。在许多方面，正是Windows 95的出现以及带有图形界面的此操作系统和其他OS的后续版本影响了这种动态。此外，现在许多屏幕-触摸屏，书籍和杂志都在平板电脑和智能手机上阅读（当然，纸张版本还没有消失），而Internet在Windows 95问世时仍然很少见，现在已经成为数十亿人生活中不可或缺的一部分。

顺便说一句，电子设备发生了很大变化，以至于Windows 95现在甚至可以在智能手表上启动，更不用说智能手机或平板电脑了。
