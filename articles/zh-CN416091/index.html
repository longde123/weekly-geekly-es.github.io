我如何将小屋自动化90％



一位乡村电工是一个众所周知的问题；在大多数情况下，即使不是更糟的话，对此的态度也不会令人发指。 如果可以连接错误的东西而不是在那儿-当然可以做到这一点。

当我和我的妻子开始建造一栋新的避暑别墅而不是旧的避暑别墅时（电工被放置了好几代人，电线和接线盒是一团糟），我们首先决定让电工胜任，而避暑别墅将变得精明。

Wiren Board营销人员的注释：文章的作者以前是Wiren Board的常规用户，现在是我们团队的工程师。 在加入我们的团队之前，他自动化了自己的避暑别墅。 除结尾处的注释外，所有文字均由作者撰写。


右图：联系科学-实际意义。

第一只实验兔是霍兹布鲁克犬。 电气化完成后，人们几乎可以清楚地了解到住宅建筑中尚待完成的工作规模，如何铺设电缆以及如何解决其他问题。 除了一种之外，所有就是自动化。

我花了很多个冬天的夜晚准备购物车，然后选择系统作为未来“智能夏季住宅”整个管理的基础。 我仔细观察了西门子，白羊座，树莓派，但最终选择了Wiren Board。 它正在运行Linux吗？ 对于真正的“管理员”来说，这只是灵魂的香脂！

第一个培训实例是Wiren Board版本4。它存在很多问题，并且其编程没有一开始时那么明显。 但是所有的陷阱都经过了深入的研究，每个缝隙中的海鳗都被命名了，到第5版出现时，我确定控制器可以应付我的任务。



我想要的是：首先，使在旧房子中必须手动完成的所有过程尽可能地自动化-例如，从我们与邻居共享的水井向房子供水。 其次，自动控制气候。 第三，要远程管理房屋并能够监视所有事物。 第四，使照明变得方便且真正智能。
我真的想要不间断的电源，可靠的接地和人性化的互联网。

电缆布线


智能家庭需要从配电盘中引出数百米的电缆，并在所有最偏远的地方分叉。
房子是木制的（框架），在这样的房子中铺设15至20条电缆的线束已经是一个实际的问题。 在地板和墙壁中使用隐藏式布线时，只能将电缆放置在厚壁的金属管中。 如果有几十个从盾牌中出来？ 如果某个地方没有联系？ 如果您需要铺设多条意外电缆？
简而言之，决定使用电缆通道。 商量之后，我和妻子选择了带有马赛克安装产品的Legrand DLP 80 mm。 怀疑使夏季小屋变得有点像办公室，但并没有实现。 DLP通道在外观上非常令人愉悦，它们安装方便，在其中铺设电缆也很方便，最重要的是，在其中铺设许多电缆也很方便！

盾牌选择



连接前已完成带有EDF配置文件和EDF模块的ABB TwinLine机柜

盾牌不大。 特别是如果它们安装在整个房屋的自动化区域。 如果您突然觉得盾牌太大了，而您的朋友却在笑：“您在给整个乡村供餐吗？！” -随意将盾牌抬高一倍半！ 如果您认为十根电缆足以满足您的需要，请再铺设五根。 每间客房十个营业网点-破产？ 相同数量绝对不会造成伤害。 如果在第二层安装第二个屏蔽罩更方便-那就这样吧！

自动化需要什么


当然， Wiren Board 5控制器本身 （请参阅最后的注释1） 。 主执行器是继电器模块，控制器通过RS-485与之通信。 选择的通信电缆是专用的，具有较粗的导体和屏蔽层-Belden RS-232 / RS-485 2x2x28 （请参阅结尾处的注2） ，因为它将被放置在不同的地方，包括电源线旁边。 屏蔽层在屏蔽层中接地，在输入端，我将TN-CS接地，并用房屋附近的埋入式金属焊接接地回路作为接地电极，并将输入屏蔽层中的输入中性线重新接地。 接地电阻是通过特殊的UNI-T UT521测试仪测量的，我当时是以折扣价购买的。

灯饰


配电盘使用WB-MR14和WB-MR11继电器模块；功能强大的用户通过ABB模块化接触器连接（请参见最后的注释3） 。


继电器模块和接触器

通过WB-MR14继电器，宜家的2至3个Ledare LED灯组被切换。
将几乎每个光源连接到其入口，就可以实现最大胆的愿望。 与WB-MD2调光器一起使用时，照明变得非常智能。 通过规则引擎，我实现了天文接力-黄昏开始时，入口处立柱上的聚光灯打开。 一间房间的灯可以单独打开，并具有必要的亮度。 编程的“陪伴灯”（伴有背光模式，“我们都离开”功能）：所有居民离开小屋后的某个时间，晚上的室外和花园照明都熄灭了-您可以关闭所有物品，检查一下，再次关闭，然后再次检查:)

代替普通开关的是，房屋内各处都安装有无需固定的按钮：它们提供更多选择，使您可以长时间按下：例如，同一按钮用于开启伴随的背光模式，该背光模式开启房屋门廊上的光，但是当延迟按下时，伴随的照明模式。 这样的规则是通过两个计时器来完成的（一个计时器确定长按，第二个计时器关闭照明）：
护送灯规则（功能“我们都离开”） 
var fronton_timer_1_timeout_ms = 2000;
var fronton_timer_1_id = null;

var fronton_timer_2_timeout_ms = 60000;
var fronton_timer_2_id = null;

var lightmode = 0;

defineRule("frontonButtonPressed", {
    whenChanged: "wb-mr14_143/Input 13",
    then: function(newValue, devName, cellName) {

        if (newValue) {
            log("The fronton button is on")
            if (dev["wb-mr14_143"]["K13"] == false) {

                fronton_timer_1_id = setTimeout(function() {
                        log("fronton timer 1 complete");
                        lightmode = 1;
                        dev["wb-mr14_143"]["K13"] = true;
                        dev["wb-mr14_143"]["K14"] = true;
                        dev["wb-mr14_141"]["K8"] = true;
                    },
                    fronton_timer_1_timeout_ms);
                fronton_timer_2_id = setTimeout(function() {
                    dev["wb-mr14_143"]["K13"] = false;
                    dev["wb-mr14_141"]["K8"] = false;
                    dev["wb-mr14_143"]["K14"] = false;
                    log("fronton timer 2 complete");
                }, fronton_timer_2_timeout_ms)
            };
        };
    }
});
