GPD Win - menjelajah notebook mini dengan diagonal 5,5 ", yang dirancang untuk game dan emulator

Saya suka perangkat keyboard mini. Saya sudah memiliki cinta ini sejak zaman Psion 5MX pertama. Sejak itu masih ada Nokia E90 dan sebagainya. Sayangnya, seiring waktu, ada semakin sedikit perangkat kecil dengan keyboard, tapi saya tidak berhenti mencintai mereka (mungkin patut juga dikatakan, sayangnya). Karena itu, saya tidak bisa melewati GPD Win. Ini terlihat seperti ini.



Diagonal layarnya hanya 5,5 ", pas di saku bagian dalam jaket, tetapi di dalamnya adalah netbook paling lengkap di Windows.

Layar: 5,5 inci, HD 1280 x 720 (267 ppi), 16: 9, kapasitif, multi-touch, Corning Gorilla Glass 3, Teknologi laminasi penuh dalam sel
Prosesor: Intel Atom X7 Z8700, 64 bit Quad Core, 1.6GHz ~ 2.4GHz;
Grafik: Intel HD Graphics 405 (200MHz ~ 600MHz)
RAM: LPDDR3-1600, 4GB
Memori bawaan : eMMC 4.51, 64GB
Dukungan Kartu Memori: Micro SD hingga 128GB
Antarmuka nirkabel: Wi-Fi: 802.11 a / ac / b / g / n WLAN, Bluetooth: V4.1
Konektor: 1 x USB Type-C 3.0, 1 x USB 3.0, 1 x slot Kartu TF, 1 x tipe HDMI C, 1 x 3.5mm
Gamepad internal: dua joystick analog + tombol A / B / X / Y + dua tombol shift pada wajah atas di setiap sisi + tombol L3 / R3 tambahan pada keyboard + tombol Start / Select
Keyboard: 67 tombol PC standar + 10 tambahan
Tombol lain: level volume ±; tombol daya; sakelar mode gamepad (Dinput / Mouse / Xinput)
Sistem Suara: Realtek ALC5645
Speaker / Mikrofon: Speaker Mono dan Mikrofon Internal
Bahan Tubuh: ABS + PC
Sensor: G-Sensor dan Sensor Hall
Baterai: 6700mAh, waktu kerja sekitar 6 ~ 8 jam
Dimensi dan berat: 155 * 96 * 20 mm, 300 gram


Seperti yang Anda lihat, spesifikasi teknis (kecuali untuk ukurannya) sangat mirip dengan tablet China di Windows atau netbook mereka di Cherry Trail. Secara terpisah, ada pengontrol permainan dengan stik analog, yang belum saya pelajari dengan tekun, karena saya tidak suka permainan komputer. Sebagai gantinya, saya menguji GPD Win dalam segala macam mode yang berbeda. Tapi pertama-tama, mari kita lihat lebih dekat.

Lingkup Pengiriman


Datang dalam kotak kardus hitam.


Dalam paket pengiriman dari yang berguna ada charger, kabel, microbook itu sendiri, serta headphone.



Mereka juga meletakkan stiker khusus di layar (dengan mempertimbangkan Gorilla Glass ada di dalamnya dan menutup - aneh), serta instruksi tidak berguna yang berisi segala macam rekomendasi penyetelan, yang karena alasan tertentu memberikan tautan yang rusak.



Saya tidak mengerti mengapa headphone dimasukkan ke dalam kotak, tetapi mereka tidak meletakkan kasingnya. Jujur saja, itu akan jauh lebih bermanfaat.

Desain, ergonomi


Ketika ditutup, terlihat membosankan dan formal, tidak ada logo di luar. Plastik grey matte mirip dengan yang ada di Psion 5MX, plastik ini benar-benar tidak menandai dan tidak tergores.



Layar dengan sudut pandang yang bagus, silau cukup.



Reproduksi warna tidak buruk, kecerahan maksimum cukup untuk bekerja di luar ruangan, tetapi tidak di bawah sinar matahari yang cerah.



Rangka-rantainya lebar, sebagian, saya kira, karena bentuk perangkat yang "sisa" bulat.



Tampak bawah laptop mikro. Ada kaki plastik mini, tetapi perangkat tidak sangat stabil di atas meja, itu hanya tidak dimaksudkan untuk pekerjaan seperti itu.

Benda di tengah penutup bawah ini adalah sakelar mode kipas yang menarik. Ada opsi untuk operasi pasif, dengan penyesuaian otomatis, dan maksimal.



Penandaan juga terlihat pada tutupnya. Seperti yang Anda lihat, GPD tidak menyembunyikan asal-usulnya, semuanya dibuat di kota metropolitan teknologi Shenzhen.


Di sebelah kanan, hanya radiator radiator, tidak lebih.



Demikian pula, hal-hal di sebelah kiri.



Di depan ada gril radiator lain, di sebelah kanan ada dioda yang menunjukkan status sistem.



Dalam mode tidur, menyala biru. Tidak terlalu cerah, tidak mengganggu.



Untuk memberi Anda gambaran tentang perangkat mana yang ukurannya. Tangan saya tidak terlalu besar, tetapi cukup nyaman bagi saya untuk menjangkau semua sudut, joystick, keyboard.



Ketebalan perangkat ini cukup besar, dapat dilihat bahwa itu dibuat untuk bekerja, dan bukan untuk menyombongkan diri. Ya, bukan untuk bekerja, untuk game.



Semua port, konektor, dan kunci tambahan terkonsentrasi di bagian belakang. Di tepi adalah pemicu (kanan dan kiri memainkan peran tombol mouse dalam mode "mouse"), ada USB Type-C, yang digunakan untuk mengisi daya, USB 3.0 berukuran penuh, slot microSD, jack headset, dan miniHDMI penuh 1.4. Katakan apa yang Anda suka, tetapi dengan konektor bayi semuanya baik-baik saja, pada banyak laptop ada lebih sedikit dari mereka.



Tampilan pelatuk lebih dekat. Mereka nyaman, jangan menekan secara spontan.



Dan di sisi lain.



Speaker terletak di kanan bawah, di sisi belakang. Dia adalah satu, dan monofonik. Para pengembang mengatakan bahwa mereka masih tidak berharap untuk membuat suara normal pada bayi seperti itu, dan di jalan lebih baik untuk menonton film di headphone dan bermain juga.



Keyboard, tentu saja, tidak berukuran penuh, tetapi cukup nyaman. Secara khusus, itu menyenangkan bahwa ia memiliki jumlah kunci yang cukup untuk memasukkan semua huruf Rusia. Tidak menganjurkan bahwa tutsnya tidak terlalu besar, jadi, kecuali untuk "tumbing" (yaitu, dengan ibu jari), pencetakan tidak akan berfungsi.



Lihat lebih dekat ke sisi kiri keyboard.



Di sebelah kanan keyboard terdapat tombol tambahan - tombol on / off dan dimulainya layanan game Microsoft, kontrol volume, L3 dan R3 (tombol "kanan" dan "kiri" tambahan), pilih, mulai, jeda, dan kunci gulir.



Tombol X, Y, A, B terletak di sebelah stik analog kanan. Dalam mode "mouse", secara default mereka tidak melakukan apa-apa, tetapi jika Anda mau, Anda dapat menetapkan makro apa pun kepada mereka menggunakan berbagai macam program. Jangan lupa bahwa ia menggunakan Windows 10 penuh.



Tanda silang terletak di sebelah stik analog kiri seperti pada gamepad konsol awal. Dalam mode mouse, digunakan sebagai "panah" pada keyboard.



Di bawah layar adalah saklar fisik yang menentukan operasi tongkat: mereka dapat berfungsi sebagai gamepad (D-Input), seperti mouse, dan juga berfungsi dalam mode X-Input (seperti dalam X-Box).



Perangkat diisi dari pengisian “smartphone” biasa, tetapi dengan arus yang besar, hingga 2,5 ampere (pada 5V). Sekitar muatan yang sama, misalnya, Chuwi Lapbook 15.6 , serta banyak tablet Cina.



Total - perangkat ini sangat tidak biasa dalam penampilan, bijaksana, tetapi sepertinya menarik bagi saya. Dapat dilihat bahwa pengembang mendekati desain dengan penuh cinta dan mencoba membuat semuanya senyaman mungkin, meskipun ukurannya kecil.

Pengujian kinerja


Saya tidak menguji Win GPD di banyak game, pada akhirnya, banyak yang sudah melakukannya. Tabel kompatibilitas dapat ditemukan di Internet, untuk memahami apa yang akan terjadi, apa yang tidak, well, pada kenyataannya, kita dapat mengasumsikan bahwa segala sesuatu yang bekerja pada tablet Cherry Trail berfungsi di sini adalah beberapa label kompatibilitas ( 1 , 2 ).

Diharapkan kinerja tiga dimensi - tidak ada overclocking tambahan.



Hasilnya logis - pada dasarnya hanya Ice Storm yang dapat melakukan laptop ini.



Tetapi tes PCMark sudah lebih menarik. Pada prinsipnya, karena jumlah memori yang besar, ternyata pilihan yang sangat baik untuk pekerjaan biasa.



Harap dicatat bahwa dalam uji Kreatif yang agak berat, suhunya sangat stabil, dan frekuensinya terletak di langit-langit, ini menunjukkan pengaturan sistem pendinginan yang baik.



Dalam tes Kerja, di mana ada banyak tugas yang lebih ringan, prosesor benar-benar dingin sama sekali.



Penyimpanan, mmm, eMMC, baik, apa yang harus diambil darinya. Sayangnya, tidak ada cara untuk mencapai kecepatan SSD, kinerja keseluruhan setara.



Nah, dan, seperti yang saya katakan di atas, konsol tidak kepanasan (tentu saja, dengan pendinginan maksimum). Jika Anda menguranginya, maka itu menjadi pelambatan, dan pada kecepatan tinggi ia duduk dalam dorongan turbo. Jarang sekali, tapi kipasnya cukup berisik.



Masa pakai baterai perangkat juga mengesankan saya. Rata-rata kecerahan Internet, saya dapat sekitar 6 jam. Tes Baterai Rumah PCMark menunjukkan hal yang sama.

Perangkat mengisi daya dalam 3 jam, waktu yang lama, tetapi Anda dapat bekerja dengan baterai apa pun atau pada bank daya.

Total


Ini adalah perangkat yang sangat menarik, sekarang dijual di Gearbest dengan diskon dan biaya sekitar $ 360. Coba juga kupon kupon GBGPD ($ 317,99). Ya, tentu saja, ini hampir 2 kali lebih mahal daripada tablet dengan kinerja yang sama, tetapi faktor bentuknya unik di sini. Selain itu, karena rangkaian port yang kaya, perangkat ini juga dapat digunakan di rumah sakit dengan menghubungkan ke monitor. Saya tidak punya keluhan khusus (selain yang timbul langsung dari faktor bentuk).

UPD: Review pemilik dari komentar

Pemilik perangkat sejak Maret.
Untuk game hingga ~ 2013 dan mereka yang tidak meremehkan pengaturan grafis minimum - ini adalah yang paling, terutama mengingat ukuran layar. Tidak semuanya berjalan dengan nyaman, tentu saja, tetapi secara keseluruhan itu kuat, sangat bahagia - dan tentu saja, itu sangat tergantung pada permainan dan pengoptimalannya. Sebagai contoh, RE5, DmC dan Metal Gear Rising berjalan dengan baik atau dengan drawdowns yang sangat langka (pendingin dalam posisi menengah), tetapi Darksiders (yang pertama, setidaknya yang asli, setidaknya remaster) melorot hampir di mana-mana. Game indie dan non-serakah (atau hanya dioptimalkan dengan baik) terasa seperti di rumah.

Persaingan lebih menarik. Saya tidak menyalakan platform old-school, mereka ditiru oleh semua orang yang tidak malas pada Symbian.

  • PS1 (ePSXe), semua judul yang dapat dimainkan dapat dimainkan, banyak game dapat diluncurkan dengan shader dan grafik yang ditingkatkan, tidak ada yang mengejutkan di sini.
  • PS2 (PCSX2) pada pengaturan agresif dengan satu set peretasan kecepatan dapat menarik sejumlah game 3D serius (God of War 1.2, seluruh seri DMC) di beberapa tempat bahkan dengan kecepatan penuh (tergantung pada lokasi), biasanya 80% adalah maksimum yang diperas. Mereka yang akrab dengan emulasi dan PS2 secara keseluruhan, tahu bahwa ada permainan yang membuat konsolnya sendiri melambat, Anda dapat menjalankannya di GPD Win, tetapi Anda bahkan tidak dapat memainkannya dengan air mata - Saya berbicara tentang Shadow of the Colossus dan MGS2 yang sangat berat. MGS3. Omong-omong, versi PC MGS2 adalah pilihan yang sangat baik untuk bermain di GPD Win. ICO, misalnya, dimainkan dengan nyaman, meskipun bukan tanpa drawdowns.
  • PSP (PPSSPP) - situasinya sedikit lebih baik daripada dengan PCSX2, tetapi rentang permainan yang menuntut / tidak menuntut lebih luas, ditambah daftar pengaturan saat ini memungkinkan pengorbanan transparansi tekstur untuk sejumlah game atas nama kecepatan penuh. Keseimbangan playability / view di sini juga perlu disesuaikan secara individual untuk setiap game, semuanya dua dimensi dalam kecepatan penuh, tiga dimensi dengan cara yang berbeda (di suatu tempat Anda dapat mencapai kecepatan penuh tanpa hack sama sekali, di suatu tempat frameskip adalah teman terbaik kami). Saya mencoba Persona (1-2-3), Echochrome, Monster Hunter 3, MGS, keduanya GoW. Secara umum, kecuali MGS, tidak ada yang perlu dikeluhkan, kecuali bahwa GoW melorot di beberapa tempat.
  • DreamCast (NullDC) - tidak banyak tes. Shenmue, tentu saja, dan Soul Calibur - keduanya berjalan dengan kecepatan penuh.
  • Wii / Gamecube (Dolphin) - "tidak diuji." Cabang dari ishiiruka memberikan peningkatan kecepatan yang cukup nyata, master versi 5 lebih stabil, tetapi mengorbankan kecepatan demi kompatibilitas. Saya menguji versi GC dari Resident Evil Zero dan remaster, hampir di mana-mana saya menekan kecepatan penuh pada ishiiruka. Xenoblade Chronicles - setengah dari kecepatan, biasanya tidak naik di atas 75%, banyak menyakitiku :(.
Saya ingin mencoba MGS Twin Snakes, saya tidak menunggu untuk kecepatan penuh, tetapi harus dapat dimainkan.

Dalam hal zat besi:

  • baterai - seperti biasa, tergantung pada apa yang harus dilakukan. DmC di kereta, hampir 5 jam terus menerus, pendingin berada di posisi tengah (di beberapa tempat Anda dapat mematikannya sama sekali, tetapi kasingnya menjadi lebih panas). Saya tidak membawa mesin ke pelambatan, tetapi saya tidak melihat suhu di atas 72. 66-68 di bawah beban dengan pendingin dihidupkan di posisi tengah, 62 pada kecepatan maksimum. Ngomong-ngomong, saya sudah mengeceknya sekarang - udaranya tersedot dari depan, diterbangkan dari bawah dan ke kiri. Kisi-kisi yang tepat adalah speaker mono (output stereo ke headphone). PS1 Games (diuji di Silent Hill pertama) - lebih dari 6 jam. Saya memeriksa dengan 99% dari biaya, Wi-Fi tidak aktif, USB3.0 flash drive terhubung, kecerahannya 50%;
  • memori: 64 GB, tentu saja, tidak cukup. Saya masih memiliki flash drive 64-GB dan kartu memori 16-GB, masih agak sempit. Dukungan yang dinyatakan secara resmi untuk flash drive hingga 128 GB, tetapi para penggemar mengkonfirmasi kerja kartu dengan 200 GB;
  • mengikuti rake: port USB, pembaca kartu, dan daya berbagi modul Wi-Fi. Jika Anda memasang flash drive USB dan kartu memori, dengan I / O intensif, yang kedua akan jatuh pada salah satunya. Jika Anda mulai mengunduh sesuatu melalui Wi-Fi, sampah seperti itu. Moral: berhati-hatilah dengan apa yang Anda muat pada sepotong besi;
  • tampilan telepon. Yaitu begitu telepon sehingga driver grafis menganggap mode lansekap diputar 90 derajat. Hal ini menyebabkan kebingungan (dan kudeta) dalam game menggunakan DirectX8 dan lebih rendah, tetapi diselesaikan dengan melemparkan lib yang ditambal ke permainan.


Linux: semuanya buruk di sini, untuk saat ini. Ya, versi Ubuntu untuk Cherry Trail meningkat, tetapi tidak ramah dengan baterai dan modul nirkabel (setidaknya saya belum belajar cara menggunakan berita terbaru), ditambah kecerahan layar tidak dapat disesuaikan, ditambah dengan pengontrol petualangan dan dengan rotasi layar (tergantung pada DE) . Pilihan saya adalah Xubuntu 16.04 di VirtualBox.

Sebenarnya, tidak ada yang perlu dibicarakan tentang Windows 10. Terjepit di sana-sini, dan untuk konsol game biasanya cocok untukku. Layar sentuh sebagian besar bermanfaat daripada berbahaya, tetapi tanpa pengaturan penskalaan font tidak nyaman.

Tidak ada begitu banyak keluhan tentang kegunaan, tetapi yang ada dapat menjadi penentu bagi seseorang dalam hal "take-not take". Pengontrolnya cukup nyaman - tongkat dengan stroke yang lembut, jika Anda menggunakan PSP / PS Vita, sensasinya sama. Kekurangan mereka dalam "nonpressibilitas" - L3 / R3 ditempatkan di blok kanan bawah keyboard, dan beberapa game (termasuk, misalnya, Gears of War) kehilangan beberapa kenyamanan karena hal ini. Tombol pundak (L1 / L2, R1 / R2) membuat kecanduan dan pada awalnya tampak lemah (well, mereka gemetar, mereka gemetar!), Tapi itu tidak buruk. Jauh dari ideal, tetapi demi faktor bentuk, Anda dapat bertahan dan terbiasa dengannya.

Tutupnya diperbaiki dalam dua posisi (di suatu tempat 120 dan 180), tetapi juga tidak tergantung pada yang menengah, meskipun, mungkin, aku hanya tidak cukup perusak. Keyboard, menurut saya, adalah titik kelemahan utama perangkat bagi mereka yang membutuhkannya tidak hanya untuk game. Secara khusus, setelah Nokia N900, terlepas dari ukuran tombol yang lebih besar, jauh lebih tidak nyaman untuk mengetik dengan ibu jari saya - menggeser keyboard ke kiri sangat merusak kenyamanan mengetik dan menggunakan tombol pada umumnya. Tombol-tombolnya cukup ketat, dengan klik yang mencolok. Untuk satu set frasa pendek atau untuk memasukkan kata sandi login, itu akan dilakukan, untuk penyelamatan server darurat melalui SSH, itu lebih buruk. Yaitu Anda bisa bertahan, kalau saja tidak lama.
