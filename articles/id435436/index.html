5 tren dalam infrastruktur TI: perkiraan untuk 2019

Kami memutuskan untuk berbicara tentang arah pengembangan infrastruktur cloud, yang oleh para pakar IT dan publikasi teknologi besar dianggap paling menjanjikan di tahun 2019. Di bawah potongan - tentang komputasi tanpa server, komputer kuantum "pribadi" dan sistem AI.


/ foto 16: 9 petunjuk CC BY

1. Permintaan untuk komputasi tanpa server akan meningkat


Arsitektur serverless adalah pendekatan di mana penyedia pihak ketiga mengambil tugas memelihara server. Dengan demikian, pengembang dan administrator sistem tidak perlu menghabiskan sumber daya untuk perawatan besi.

Sistem tanpa server menawarkan semua fungsionalitas yang diperlukan dalam bentuk layanan. Mereka dapat berupa solusi terpisah (seperti Proyek Fn ), atau seluruh ekosistem (misalnya, Apache OpenWhisk atau Azure yang dipatenkan).

Arsitektur tanpa server, menurut Gartner, akan menjadi salah satu tren terpanas tahun ini. Sekarang kurang dari 5% perusahaan di seluruh dunia menggunakan pendekatan serverless, dan pada tahun 2020 angka ini akan meningkat menjadi 20%. Literatur khusus , kerangka kerja sudah muncul, konferensi sedang diadakan ( termasuk di Rusia ).

Salah satu penggerak semakin populernya sistem tanpa server adalah mengurangi biaya konfigurasi dan pemeliharaan besi. Dipercaya bahwa administrator sistem menghabiskan sekitar 30-40% dari waktu kerja mereka untuk menyiapkan peralatan. Arsitektur tanpa server melepaskan sumber daya ini - upaya para insinyur dapat diarahkan pada pengembangan produk dan layanan perusahaan.

Diharapkan juga bahwa popularitas pendekatan tanpa server akan dipengaruhi oleh “booming” dalam algoritma pembelajaran mesin. Vendor cloud bekerja keras untuk memastikan ketersediaan dan skalabilitas infrastruktur TI. Dan ini adalah dua komponen terpenting dalam pengembangan sistem cerdas.

2. Semakin banyak perusahaan akan mulai menggunakan multi-cloud


Menurut sebuah studi 2018 oleh Rightscale, sekitar 80% perusahaan (dari ribuan responden) telah beralih ke infrastruktur IT multi-cloud. Tahun ini angka ini akan tumbuh - di Gartner mereka mengatakan bahwa pada tahun 2020 akan mencapai 90%.

Alasan popularitas multi-cloud adalah penghematan biaya dan fleksibilitas dalam penggunaan infrastruktur. Perusahaan dapat memilih layanan dari beberapa penyedia cloud yang layanannya sesuai dengan harganya. Pada saat yang sama, wadah semakin digunakan untuk menyederhanakan migrasi aplikasi antara lingkungan yang berbeda. Menurut ESG Research, pada tahun 2020, 33% dari semua beban kerja di cloud akan di kemas (hari ini jumlah ini adalah 19%).

Salah satu teknologi containerisasi yang paling populer disebut analytics oleh Kubernetes. Diharapkan pada 2019 lebih banyak solusi akan muncul di pasar yang menyederhanakan implementasi alat orkestrasi ini. Sebagai contoh, pada akhir tahun lalu, VMware mengakuisisi startup Heptio, yang mengembangkan solusi untuk penyebaran Kubernetes. IBM bermaksud untuk mengembangkan Kubernetes sebagai alat untuk menciptakan solusi cloud hybrid, yang untuk tujuan ini (juga pada 2018) mengakuisisi Red Hat.

3. Awan akan membantu operator telekomunikasi 5G


Menurut perkiraan Deloitte, pada tahun 2019 jaringan 5G komersial pertama akan muncul. Pada tahun 2020, jumlah pengguna jaringan generasi baru akan 15-20 juta orang. 5G direncanakan akan digunakan untuk mendukung pekerjaan perangkat IoT dari sistem industri dan " kota pintar ". Pada saat yang sama, jumlah gadget IoT akan tumbuh secara eksponensial di tahun-tahun mendatang.

Untuk mengatasi beban seperti itu, perusahaan telekomunikasi harus menyediakan skalabilitas infrastruktur tingkat tinggi. Di sini, teknologi cloud akan membantu mereka.


/ foto Derek Finch CC BY (dipangkas gambar)

VMware sedang mengembangkan solusi virtual yang akan menyederhanakan bekerja dengan inti arsitektur jaringan nirkabel dan menawarkan pelanggan fitur baru. Contohnya adalah VMware vCloud NFV untuk bekerja dengan layanan dan aplikasi di lingkungan cloud pribadi dan publik. Nokia menciptakan platform serupa khusus untuk operator - layanan ini menggunakan teknologi OpenStack untuk mengelola kontainer dan mesin virtual.

Pada saat yang sama, tidak hanya cloud yang akan membantu penyebaran 5G, tetapi, sebaliknya, jaringan generasi baru akan membuat platform cloud lebih populer. Di Deloitte, mereka mengatakan bahwa 5G akan memungkinkan penghuni daerah terpencil di mana tidak ada Internet berkecepatan tinggi untuk menggunakan awan.

4. Mesin Quantum akan menjadi lebih mudah diakses oleh pengguna


Perusahaan IT sedang mengembangkan prosesor kuantum yang semakin kuat. Intel memperkenalkan chip 49-qubit tahun lalu, dan Google chip 72-qubit.

Beberapa perangkat berbasis qubit sudah dapat diakses melalui layanan cloud seperti Alibaba Cloud dan IBM Q. Mesin-mesin kuantum dari raksasa-raksasa TI ini membantu organisasi-organisasi riset dan universitas untuk menciptakan model-model rumit yang sulit untuk komputer "klasik", misalnya, untuk memodelkan perilaku sistem kekebalan manusia.


/ foto IBM Research CC BY

Pada saat yang sama, 2019 mungkin merupakan tahun mesin kuantum "pribadi" . Pada awal Januari, IBM memamerkan Q System One di CES 2019, sebuah prototipe komputer kuantum komersial untuk bisnis. Ini adalah mesin 20-qubit yang, tidak seperti komputer kuantum lainnya, dapat beroperasi di luar laboratorium. Kondisi yang diperlukan untuk berfungsinya komputer dibuat dalam wadah khusus yang tertutup rapat.

Sangat mungkin bahwa pada 2019 "keunggulan kuantum" akhirnya akan tercapai. Ini adalah saat di mana mesin dengan qubit mulai memecahkan sejumlah masalah komputasi lebih cepat dari komputer klasik. Tugas-tugas tersebut, misalnya, termasuk enumerasi kunci SHA-256. Untuk membuktikan hipotesis superioritas kuantum, Google berencana mengadakan kompetisi kinerja dengan superkomputer NASA awal tahun ini. Hasil percobaan akan diketahui di musim panas.

5. Sistem cerdas akan mengelola cloud


Pembelajaran mesin akan lebih sering digunakan dalam pengelolaan sistem dan peralatan multi-cloud di pusat data. Teknologi ini disebut AIOps - AI untuk Operasi TI. Semakin populernya AIOps diprediksi oleh Network World dan Gartner. Pada 2019, 25% organisasi akan menggunakan algoritma pintar untuk mengelola infrastruktur TI mereka.

Sistem AI untuk bekerja dengan cloud sudah tersedia di layanan Oracle dan Microsoft . Mereka membantu memprediksi kemungkinan kegagalan sistem dan mengoptimalkan beban kerja. Ada kasus yang diketahui di mana algoritma pembelajaran mesin di cloud Azure memperkirakan kegagalan salah satu disk 20 menit sebelum terjadi. Ini memungkinkan untuk memigrasikan beban dengan cepat dari server yang salah.

Platform untuk mengelola peralatan di pusat data juga sedang berkembang - layanan ini dibuat oleh HPE. Menurut perwakilan perusahaan, platform membantu pelanggan mengurangi biaya operasional sebesar 79%, dan jumlah kegagalan server sebesar 86%.
