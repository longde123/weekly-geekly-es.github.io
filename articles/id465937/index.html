Technostream: pilihan baru video pelatihan untuk awal tahun ajaran


September bagi banyak orang sudah dikaitkan dengan akhir musim liburan, tetapi untuk sebagian besar - dengan belajar. Pada awal tahun ajaran baru, kami menawarkan kepada Anda pilihan video dari proyek-proyek pendidikan kami yang diposting di saluran Youtube Technostream. Pilihannya terdiri dari tiga bagian: kursus baru di saluran untuk tahun akademik 2018-2019, kursus yang paling banyak dilihat, dan video yang paling banyak dilihat.

Kursus baru di saluran Technostream untuk tahun akademik 2018-2019



Database (Technosphere)



Tujuan kursus adalah untuk mempelajari topologi, variasi dan prinsip-prinsip dasar dari fungsi penyimpanan dan sistem data, serta algoritma yang ditetapkan berdasarkan sistem terpusat dan terdistribusi, untuk menunjukkan kompromi mendasar yang melekat dalam satu atau solusi lain.

Kursus ini mengungkapkan berbagai solusi untuk menyimpan data dalam proyek-proyek Internet dalam tiga dimensi:

  • kontinum model data;
  • kesinambungan konsistensi data;
  • sebuah kontinum dari algoritma penyimpanan data.

Program kursus ditujukan baik untuk pemrogram sistem, pengembang DBMS, dan pemrogram terapan, pencipta sistem antrian di Internet.

Python Terapan (Technopark)



Kursus ini memperkenalkan bahasa Python - salah satu bahasa yang paling populer dan populer di pasar TI saat ini. Permintaan bahasa tidak lahir dari awal: kesederhanaan input dan sintaksis, pilihan alat terkaya untuk menyelesaikan semua jenis masalah - ini dan banyak lagi telah mengarah pada fakta bahwa Python banyak digunakan di seluruh dunia. Berkat kursus ini, Anda dapat bergabung dengan ekosistem bahasa.

Anda akan belajar:

  • Program dengan Python
  • Tulis kode yang didukung dan berkualitas tinggi;
  • Membangun proses pengembangan produk perangkat lunak;
  • Berinteraksi dengan layanan dan basis data online.

Pemrograman C / C ++ Tingkat Lanjut (Technosphere)



Anda akan berkenalan dengan alat dan praktik yang digunakan dalam pengembangan modern, dapatkan keterampilan untuk menulis kode C ++ yang benar dan fleksibel. Kursus ini akan membantu untuk memperoleh keterampilan yang diperlukan untuk spesialis dalam pengembangan perangkat lunak untuk berpartisipasi dalam proyek pengembangan industri dalam bahasa C ++, termasuk untuk mengisi posisi peserta pelatihan untuk pengembang sisi server dari aplikasi yang sangat dimuat.

Setiap pelajaran terdiri dari kuliah (2 jam) dan tugas praktis.

Pemrograman sistem | Laboratorium Tarantool (Technosphere)




Kursus ini menyoroti sistem operasi perangkat berdasarkan pada kernel GNU / Linux, arsitektur kernel dan subsistemnya. Cara interaksi dengan OS diberikan dan dijelaskan. Materi kursus sedekat mungkin dengan kenyataan, diisi dengan contoh-contoh.

Proyek IT dan manajemen produk (Technosphere)



Tujuan kursus adalah untuk mendapatkan pengetahuan di bidang produk dan manajemen proyek menggunakan contoh Mail.ru Group, untuk memahami peran produk dan manajer proyek, untuk mempelajari prospek pengembangan dan fitur-fitur produk dan manajemen proyek di sebuah perusahaan besar.

Kursus ini akan memeriksa teori dan praktik mengelola produk dan segala sesuatu yang ada di dalamnya (atau di sebelahnya): proses, persyaratan, metrik, tenggat waktu, peluncuran dan, tentu saja, tentang orang-orang dan cara berkomunikasi dengan mereka.

Pengembangan Android (Technopolis)



Kursus ini akan membantu untuk menguasai pengetahuan dan keterampilan yang diperlukan untuk mengembangkan perangkat lunak untuk Android. Anda akan belajar API Android, Alat Pengembang (SDK), perpustakaan populer, dan banyak lagi. Selain itu, dalam proses pembelajaran, Anda akan belajar tidak hanya bagaimana mengembangkan aplikasi, tetapi juga memberikan toleransi kesalahan. Setelah itu, Anda akan dapat secara mandiri membuat aplikasi dan mengontrol (secara teknis - di tingkat manajer) pengembangannya.

Pengantar Java (Technopolis)



Kursus ini dikhususkan untuk mempelajari dasar-dasar Java 11, bekerja dengan Git, mengenal beberapa praktik pengujian dan pola desain sistem. Dirancang untuk orang-orang dengan pengetahuan pemrograman dasar minimal dalam bahasa apa pun. Selama kursus, Anda akan dapat menguasai Java dan membuat aplikasi lengkap.

Menggunakan Database (Technopolis)



Anda akan menerima pengetahuan komprehensif tentang bekerja dengan basis data. Pelajari cara memilih jenis database yang paling cocok untuk proyek Anda, membuat pertanyaan, memodifikasi data, menguasai dasar-dasar SQL dan banyak lagi.

Kursus yang paling banyak dilihat di saluran Technostream untuk tahun akademik 2018-2019


Pengujian kualitas dan perangkat lunak (Technosphere, 2015)



Semua tentang pengujian saat ini dan metodologi jaminan kualitas untuk aplikasi web modern: fondasi teoritis, pengujian manual, dokumentasi, cakupan kode dengan pengujian, pelacakan bug, alat, otomatisasi pengujian, dan banyak lagi.

Pembangunan Jawa (Technosphere, 2018)



Kursus ini memiliki semua kebutuhan pemula di dunia Jawa. Kami tidak akan masuk ke detail sintaks, tetapi segera mengambil Java dan melakukan hal-hal menarik darinya. Kami berasumsi bahwa Anda tidak tahu Java, tetapi diprogram dalam bahasa pemrograman modern apa pun dan terbiasa dengan dasar-dasar OOP. Penekanannya adalah pada penggunaan tumpukan pertempuran teknologi (ya, inilah yang digunakan banyak perusahaan). Beberapa kata kunci: Java-stack (Jersey, Hibernate, WebSockets) dan toolchain (Docker, Gradle, Git, GitHub).

Administrasi Linux (Technotrack, 2017)



Kursus ini meneliti dasar-dasar administrasi sistem layanan Internet, memastikan toleransi kesalahan mereka, kinerja dan keamanan, serta fitur-fitur perangkat OS Linux, yang paling banyak digunakan dalam proyek-proyek tersebut. Sebagai contoh, kami menggunakan distribusi keluarga RHEL 7 (CentOS 7), server web nginx, DBMS MySQL, sistem pencadangan bacula, sistem pemantauan Zabbix, sistem virtualisasi oVirt, dan penyeimbang beban yang disimpan oleh ipvs +.

Teknologi web. Pengembangan di DJANGO (Technopark, 2016)



Kursus ini dikhususkan untuk pengembangan sisi server aplikasi web, arsitektur mereka dan protokol HTTP. Di akhir kursus, Anda akan belajar cara: mengembangkan aplikasi dengan Python, menggunakan kerangka kerja MVC, mempelajari tata letak halaman HTML, membenamkan diri dalam subjek pengembangan web dan Anda dapat memilih teknologi tertentu.

Go Programming (Technosphere, 2017)



Tujuan kursus adalah untuk memberikan ide-ide dasar tentang bahasa pemrograman Go (golang) dan ekosistemnya. Dengan menggunakan contoh permainan teks sederhana, kami akan mempertimbangkan semua tugas utama yang dihadapi oleh pengembang aplikasi web modern dalam proyek-proyek besar, dengan penerapannya pada Go. Kursus tidak mengatur tugas untuk mengajar pemrograman dari awal, keterampilan pemrograman dasar akan diperlukan untuk pelatihan.

Video yang paling banyak dilihat di saluran Technostream untuk tahun ajaran 2018-2019


Administrasi Linux. Pengantar (Technopark, 2015)



Video ini berbicara tentang sejarah Linux, tantangan yang dihadapi administrator OS ini, serta tantangan yang menanti Anda ketika beralih dari Windows ke Linux dan bagaimana beradaptasi.

Go Programming. Pengantar (Technosphere, 2017)



Video ini didedikasikan untuk sejarah bahasa Go, deskripsi ide-ide kunci yang tertanam dalam bahasa, dan dasar-dasar dasar: cara menginstal dan mengkonfigurasi lingkungan Go, cara membuat program pertama, cara bekerja dengan variabel dan struktur kontrol.

Video promosi yang menginspirasi tentang mereka yang pergi ke TI, apa pun yang terjadi



Ini adalah video promosi yang didedikasikan untuk merekrut siswa untuk program pendidikan kami di universitas.

Linux The Basics (Tech Track, 2017)



Video ini berbicara tentang perangkat Linux, tentang menggunakan shell perintah, tentang hak akses untuk pengguna yang berbeda. Anda akan mengetahui proses dan kondisi apa yang ada di Linux, protokol apa yang digunakan, dan bagaimana mengelola lingkungan pengguna.

Pengembangan Android. Pendahuluan (Technotrek, 2017)



Pelajaran pengantar ini menjelaskan fitur pengembangan ponsel dan siklus hidup aplikasi seluler. Anda akan mengetahui dengan tepat bagaimana aplikasi seluler ada di OS, apa yang diperlukan untuk mengembangkan aplikasi, cara mengatur lingkungan pengembangan dan membuat sendiri "Halo, dunia!"


Ingat bahwa kuliah dan kelas master saat ini tentang pemrograman dari spesialis IT kami masih dipublikasikan di saluran Technostream . Berlangganan untuk tidak ketinggalan kuliah baru!
