VKS相机逐步指南


在上一篇文章中，我们描述了在什么情况下建议在会议室中使用会议系统来捕获会议参与者的声音。 回想一下，主要因素是会议室中的参与者人数，超过10-12人。 这么多的人会产生很多外来的噪音：椅子的移动，咳嗽，沙沙作响的声音，参与者之间的窃窃私语等。 因此，捕获具有圆形辐射图的声音的方法变得不太有效，因为它们与参与者的语音一起捕获了额外的杂散噪声。

会议控制台有一个心形声音捕获图。



因此，声音信号仅在麦克风的方向轴上被捕获，所有多余的声音和噪音都被切断。 结果-演讲者的讲话清晰，清晰易懂。

BKR会议系统的一个不错的好处是无需使用其他昂贵的控制系统即可控制摄像机的位置：当激活系统的任何麦克风远程控制时，摄像机都会自动将其悬停 。

该系统可以控制一个摄像机，而不是多个摄像机（最多4个）的位置。 当会议室的形状细长时，此功能特别有用-使用一台摄像机为会议中的所有参与者提供不完整的覆盖范围，并且您必须使用两台或更多台摄像机。



设置指向活动BKR麦克风的便携式摄录机的过程需要一点时间，并且不需要多媒体设备方面的特殊技能和知识。

在本文中，我们将详细介绍使用BKR WCS-20M无线会议系统将一台CleverMic摄像机自动指向活动麦克风的过程。

将相机连接到会议系统


首先， 会议系统的中央单元执行摄录机控制功能。 它的主要功能是将来自所有“并列”麦克风的音频信号混合并进一步传输（到音频放大器，音频混合器，计算机或硬件终端）。 为此，中央单元具有音频输出1和2。



为了控制摄像机，本机必须具有特殊的控制端口。 BKR WCS-20M单元具有控制端口3，该端口以干触点的形式实现。 摄录一体机还需要一个控制端口，照相机将通过该端口接收命令，在哪个位置平移照相机的PTZ机制以及要应用的光学变焦程度。 所有CleverMic便携式摄像机都有控制端口，实现为MINI DIN 8PIN连接器（图中第4个）。

连接器3和4必须用电缆连接以传输控制信号。 为此，任何双绞线电缆都非常适合。 在摄录机一侧，我们使用MINI DIN 8PIN连接器的插针6和8，在中央单元上，我们使用一对干触点，具体取决于分配给摄录机的逻辑编号：插针1-2-摄像头编号1，插针3-4-摄像头编号2和等 在我们的展位中，摄像机的序列号为4，因此涉及干触点7-8。



PS当然，我们使用开放式电缆的连接仅用于说明目的，我们不建议以这种形式保留连接：在摄像机侧面，使用MINI DIN 8PIN连接器



摄录机和中央设备之间允许的电缆长度取决于所用控制标准的类型：

  1. RS-232 / VISCA- 长度可达15米
  2. RS-485- 长度可达1200米

展台中使用的CleverMic 2012ws摄像机同时支持两种协议，这意味着它可以在大型会议室中使用。

配置摄像头定位系统


组件物理连接后，下一步是设备的逻辑设置。 它包括以下步骤：

  • 在摄像机设置中，选择VISCA控制协议（MEN​​U / SYSTEM / PROTOCOL-VISCA）和逻辑摄像机编号4（MENU / SYSTEM / ADDRESS）。
  • 在中央单元的设置中，将VISCA控制协议放在PTZ-Protocol菜单部分中。


  • 执行初始化麦克风控制台的过程：
  • 关闭会议系统的中央单元。
  • 打开麦克风遥控器，确保它处于StandBy模式，鹅颈上的指示灯闪烁。
  • 打开中央单元。

为了将麦克风遥控器与中央单元进行逻辑连接，使用了两个参数的组合-“ LinkIP”和“ LinkID”。 这些参数的值在“系统”菜单中定义。 在制造商的工厂中，设置被设置为相同的值，用户不需要对其进行配置。



打开中央单元后，麦克风控制单元读取“ LinkIP”和“ LinkID”参数的值，与中央单元建立通信，并准备接受以下设置。



  • 转到中央设备的“设备设置”菜单，然后输入麦克风遥控器的序列号，例如9


如果正确完成了前面的步骤，则U-ID行中的麦克风控制台将显示数字9。



参数Camera SET（麦克风遥控器上的P-ID）确定将指向活动麦克风的摄像机的序列号，其值可以从000到003（总共4个摄像机）。

  • 使用摄像机的遥控器，根据每个麦克风的位置调整其位置和光学变焦。 按住遥控器上的所需键几秒钟，将摄像机的位置分配给相应的序列号。 例如，如果遥控器的U-ID为9，则摄像机的位置号也应为9。
  • 对系统中的每个麦克风执行上述步骤。
  • 如果系统中有9个以上的遥控器，则无法用遥控器记住摄像机的位置。 您必须使用中央单元管理软件，该软件可以从我们的网站下载 。
  • 如果使用安装程序，则需要使用套件随附的电缆将会议系统的中央单元连接到计算机。



安装程序并连接中央单元后，它将自动登录系统设置界面，用户将能够根据自己的喜好配置每个参数。
