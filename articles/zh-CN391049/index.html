Moto X Force：牢不可破的屏幕-真实

朋友您好！Moto智能手机在俄罗斯的销售将很快开始-直到3月1日，它仍将遭受几天的苦难。为了让它有趣，我们为您提供该系列旗舰产品-Moto X Force的评测。这是一款出色的智能手机，具有出色的自治性和不易碎的屏幕，采用独特的ShatterShield技术制成。有意思吗 然后宁愿去砍！



知名且令人难忘


Moto X Force的设计非常严格，但令人难忘。整个秘密在于坚固而略带弯曲的铝制底座，以及在表壳背面具有非凡质感的不寻常尼龙插入物。嗯，也许还有一个带有Moto徽标的椭圆形面板，您会在每种型号的直线上找到它。从3月1日开始，在俄罗斯将只出售一种版本的X Force-与您在本材料的照片中看到的完全一样。但是，我们不排除将来出现其他修改的可能性。



顶部面板上有耳机插孔和用于安装nanoSIM和microSD的可伸缩插槽。电源和音量按钮在右侧。



通常用于充电的microUSB位于智能手机的底部。USB Type C的实验被推迟到将来。但是，这并没有阻止快速充电功能的实现，但是稍后会阻止它的实现。



沥青筛


Moto X Force的主要功能是非常非常非常耐用的屏幕。它是使用ShatterShield技术制成的，是一个如此狡猾的高科技三明治。在它的一楼有一个铝基板，一个稍高的是弯曲的pOLED矩阵，甚至更高的还有一个双触摸层，并且在最顶层有两个保护面板。所有这些都是必要的，这样即使设备从人类成长的高度坠落到沥青上，您也不必担心屏幕的安全性。



但是，尝试故意这样做仍然不值得。我们将很高兴制造出一款坚不可摧的智能手机，但无论是美观还是不方便的超厚厚度和功能强大的橡胶内衬。但是工程师们仍然有一个任务，就是制造出一款具有耐用屏幕的漂亮智能手机，以供广大观众使用。根据我们的内部调查，世界上几乎每秒钟都有用户粉碎过智能手机的屏幕，因此我们专注于此元素。



因此，金属框架可能会因撞击而变形，最好不要再次暴露内部模块。另外，请勿将小工具与钥匙和小物件放在一个口袋里。他不会因此而停止工作，但外部光泽会很快消失。

顺便说一下，所有这些保护措施都没有影响图像质量。在这方面，Moto X Force不亚于其他旗舰。5.5英寸pOLED屏幕的分辨率高达2560x1440像素，密度为540 dpi。矩阵类型自动确保出色的黑色深度，高对比度和宽视角。保护面板之间没有空气间隙，因此Moto X Force显示屏上的图片在明亮的阳光下在街道上也能清晰读取。



另一个有趣的事情是对Always Always功能的支持。即使在睡眠模式下，通知图标，时间和日期也会显示在屏幕上。实践表明，对于许多用户而言，此功能很方便，而对能耗几乎没有影响。

准备好一切


Moto X Force在Qualcomm Snapdragon 810芯片组上运行，完全成熟：支持big.LITTLE架构和两个四核内核，频率分别为1.5和2 GHz。加上集成显卡Adreno 430，频率为650 MHz。当然，相当数量的RAM在这里是3 GB。所有这些使智能手机不仅可以在基准测试中获得重要的结果，而且还可以轻松应对用户设置的任何任务，而不会出现毛刺和刹车。当然，在2016年世界移动大会上展示的配备Snapdragon 820的设备功能更强大，但您不太可能会发现810的情况会更糟。



内部存储器的容量限制为32 GB，其中约有24个可供用户使用。这足以满足所有必要的程序，并且照片，音乐和视频可以存储在存储卡中。



Moto X Force已通过俄罗斯认证，完全支持我们地区所需的所有LTE频率，但在这种情况下，美国版Droid Turbo 2的情况略有不同-请注意。此外，如果没有内置NFC并无法在802.11ac标准的Wi-Fi网络中工作，情况就不完整了，这也意味着同时支持2.4 GHz和5 GHz两个频率。



新鲜的Android



X Force是Moto智能手机产品线的旗舰产品，因此完全缺少新版本操作系统的缺憾。开箱即用，用户将收到搭载Android 5.1.1的设备，但是在连接到Wi-Fi网络后，将立即显示下载令人垂涎的“六个”的邀请。类似的邀请将吸引客户Moto X Style，Moto X Play甚至MotoG。

旗舰相机


Moto X的Android裸露，而相机应用程序的界面则很特别。在这里，我们尝试遵循理性简化的道路。如果从左向右滑动屏幕，则会出现一个带有图标的菜单，用于更改设置和选择拍摄模式。如果将手指从右框拖动到中心，这将快速转到捕获的图像库。此外，在界面中还有用于激活前置摄像头并立即开始录制视频的按钮。



没有传统的快门按钮，默认情况下，在用户要聚焦的屏幕上点击该位置后立即拍照。但是，还有另一种选择：“视线”可以在显示屏上手动移动，而不是快门按钮，请单击右侧的黑框。在设置中选择了21 MP的最大分辨率后，它将以4：3的宽高比拍摄照片时立即出现。



相位聚焦方法-192个传感器立即工作。相机会快速对焦，但是会启动。使用了专有的处理算法，其任务是使颜色看起来尽可能逼真。即使在黑暗中，使用Moto X Force拍照也很舒适。光学图像稳定可能不足以发出完整的嗡嗡声，我们希望X Force一定会在下一代中出现。


距离墙上插座超过一天


Moto X Force内部隐藏着一块容量为3760 mAh（14.2 Wh）的强大电池，使用随附的适配器，可在1.5小时内将0至100％充满电。连接至电源后的30分钟内，电池最多可充电至一半。



当然，每个人都以不同的方式使用智能手机，但是我们将分享联想莫斯科办公室工作人员的经验。对于使用X Force作为主要部件的人，收费通常为1.5天。好吧，如果我们谈论关于执行某些任务以完成放电的测试，那么数据如下。在连接Wi-Fi的情况下，在达到最大亮度的情况下，X Force的完整电池可以持续4个多小时的移动游戏。在类似条件下播放全高清视频时，效果甚至更令人印象深刻-至少12个小时。
