迷你电脑重塑经验

我想讲一个有关如何将中国半成品（MeeGoPad t02）变成我不以为耻的台式设备的故事。

背景知识


我想从一个事实开始，我是一名无线电业余爱好者（15岁，还很小），在建筑计算机和主板领域不是特别先进，我想要一个可以浏览互联网，为Arduino编写甚至为Youtube制作简单视频的设备。 。

首先想到的是组装台式电脑。在哪里得到钱？我在Windows上出售了平板电脑。他疯狂地热身，并且总是节流。钱不多，但它出现了。下一步-去阿里。在那里，我看到了一个名为MeeGoPad（型号t02）的奇迹。与Intel Core M相比，估计类似物的价格。一无所有-接了它，等了一个月，瞧！设备在手边，我旋转视频，变形，自定义窗口，然后外壳开始表现异常，程序变慢，我很紧张。安装SpeedFan并看到温度为84度。实际上，在那之后，我想稍微介绍一下该设备的开发人员。



故事本身


因此，该更改的第一个计划是：

  1. 进行适当的冷却（充分冷却或接近冷却）；
  2. 添加USB端口（机箱上只有2个）；
  3. 从“闪存驱动器”尺寸转换为“ nettop”；
  4. 呈现出令人愉悦的原始外观）。

为了生意！


设备的主要类型：



我们将主板取出，机箱放在侧面。该照片显示了它的大小（我没有对此过程做任何照片，会有成品设备中的照片）。



但是，它具有与常规计算机相同的所有功能，但体积却小得多，而且更紧凑。

我从旧的自制手表中拿走了表壳。并不是一个很好的选择，因为它的高度远远超出了我的需要，这就是为什么一个小型闪存驱动器变成了这样一个真正的上网本的原因。



在底部边缘可以看到切割，先前已在其中插入了用于设置时钟的按钮。如您所见，在背面（表盘的前面有一个插槽，我用胶粘了），切成一个圆圈，有一个直径为50毫米的冷却器，旁边是电源按钮。

第一项已经完成-50毫米冷却器完全冷却10瓦系统。

第二点是添加一对USB


不难猜测，为此使用了一个便宜的USB集线器（50格里夫纳汇率），我将它放在表盘的同一插槽中：



一切都完全固定在燃烧胶上（您知道更好吗？），所以视图不是很好。但是我们不会向内看，胶水可以完美地发挥其功能。

对于每个端口，我都切了一个固定的凹槽，但操作并不十分成功。事实是端口太靠近底部，这就是为什么将高位设备（例如，带循环的编程器）放在桌子上的原因，这将重点放在端口上，这会加剧端口的磨损。

第二项已经完成-我们增加了2个端口，并且如果您愿意，您可以输出另一个端口（第一个保留在板上，集线器占用了第二个）。

第三点是从“闪存驱动器”尺寸转换为“ nettop”


由于外壳很大，因此该物品本身就可以满足要求，但是您不能再将其放入电视中了，需要一根电缆。但是，如果闪存驱动器为“爸爸”并且您需要“母亲”插头，该如何连接？救援来了……不，买了“美元”适配器“母子”。我们将“闪存驱动器”和电缆插入其中：



第三点已经完成！独自一人给一切看。

项目4.原始视图


我能说什么这并不能很好地解决，因为九年级学生的手工工作原则上不能做到美观（嗯，除了印刷电路板一点都不差）。这就是我现在的创作：



不是让眼睛高兴，而是您必须同意，结果并没有那么糟糕。未来的计划是转移到更薄的外壳（我什至在本地商店看过一个），将集线器转移到我自己的印刷电路板上，以适应更小尺寸的外壳。

顺便说一下，关于填充和任务。普通用户使用的大多数程序都可以在此“ nettop”上正常运行，以下是我亲自检查并满意的程序：

Paint.net，Photoshop CC，BASCOM AVR，Arduino IDE，Proteus 8.1，Opera，Chrome。

可以继续进行一些操作，但是我没有运行其他程序。令我惊讶的是-“ this”能够在SONY Vegas Pro 11（x86）中编辑视频！我不能说我从编辑过程中获得了极大的乐趣（尽管过程很顺利，但是对于预览图像的质量来说却很尴尬），但是3分钟的高清视频由20个片段，两个叠加层和两个音轨在29分钟内渲染而成。同时，温度保持在50度，这对于此类任务来说是很正常的。

甚至游戏开始了，您就可以玩了！但这不是现在也不是现在。

谁在乎看这部影片。不要认为这是PR，因为写文章的目的不是在YouTube上推广博客，而是要表明大多数人不应该购买昂贵且功能过于强大的PC，而应该购买像我购买的产品并具有与之合作的技能。用烙铁把它变成合适的设备，像我这样的许多人可以代替我们大家使用的工作工具-计算机。

这篇文章写在“从和到”这个nettop上。希望读者能从中找到有用的信息，并做出正确的选择。
