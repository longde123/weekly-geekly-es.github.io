Kata pengantar


Bagian ini terutama mengungkapkan pikiran / perasaan pribadi tentang memilih OS penulis, sehingga siapa pun yang tidak tertarik dengan hal ini dapat melewatkannya.

Untuk waktu yang lama saya adalah pengguna Windows Mobile (perangkat kerasnya adalah htc hd mini, perangkat yang bagus untuk waktunya, tetapi sekarang sudah usang). Ketika datang untuk membeli telepon baru, saya mulai berpikir OS mana yang paling cocok untuk saya (saya menganggap perangkat seperti telepon menjadi perpanjangan dari diri saya sampai batas tertentu, jadi mereka harus sesuai dengan pemiliknya). Hal pertama yang terlintas dalam pikiran: Android, iOS, Win phone, ponsel Ubuntu, Firefox OS, Sailish OS. Pikiran mulai: Jadi, iOS terlalu sederhana bagi saya, ponsel Win tidak menginspirasi, Android - tampaknya menjadi platform yang baik dan relatif terbuka, Anda dapat menggali (yang untuk saya +), layanan Google, dan memang seluruh ekosistemnya secara normal, tetapi untuk beberapa alasan Google telah lama tidak simpatik, saya tidak tahu mengapa, mungkin karena Google mulai marah.

Ok, korporasi sudah berakhir. Apa yang tersisa? Ponsel Ubuntu, Firefox OS, Sailish OS.

Jendela Kejadian

sedikit tentang filosofi manajemen

Jika OS Android (saya tidak akan berbicara tentang iOS, karena saya menggunakan OS ini lebih sedikit daripada Google) lebih fokus pada “menusuk” dalam hal manajemen, maka Sailfish OS berfokus pada gesekan. Ya, tidak ada tombol untuk menutup, memperkecil atau kembali - semuanya dilakukan dengan gerakan! Dari gerakan, "poking" digunakan, mungkin, hanya untuk membuka kunci (Anda perlu "menyodok" dua kali di layar). Untuk mengunci, cukup geser jari Anda dari atas ke bawah, tetapi selalu dari titik tertinggi tampilan (jika tidak, Anda cukup menutup aplikasi).

Pada awalnya, saya tidak suka bahwa Anda tidak dapat mematikan dengan cepat, misalnya, Wi-Fi atau Bluetooth, Anda harus masuk ke pengaturan dan tekan di sana (karena ternyata itu menjadi kebiasaan). Saya ingin mendesain ulang pusat notifikasi sedikit, ini tidak buruk, tetapi mungkin lebih baik (kami sedang menunggu 2.0, mereka mengatakan bahwa ini telah diperbaiki di sana), saya juga tidak suka bahwa jumlah acara yang terlewatkan ditampilkan di pusat notifikasi, saya ingin melihat apa yang secara khusus saya lewatkan / Saya lupa.

Tentang Aplikasi Android

Fitur dari Sailfish OS adalah dukungan untuk aplikasi Android, menggunakan emulator bawaan (yang dapat dengan mudah dihapus melalui Harbour yang sama untuk membuat ponsel benar-benar bersih secara rasial), saya berhasil meluncurkan semua aplikasi yang saya butuhkan: Skype, Opera, 2Gis, kantor WPS. Aplikasi Android umumnya stabil. Ada beberapa toko pelanggan yang dapat diunduh langsung dari Harbour:

- Pasar Anzhi - tidak mencoba;
- Aptoide Store - Saya pribadi menyukai Yandex.Store yang paling, lebih stabil;
- Yandex.Store - pada awalnya saya hanya menggunakannya, tetapi ternyata sangat buggy, bagi saya sepertinya ini menempati posisi nomor 1 dalam daftar teratas bug aplikasi Android saya.

Pengembangan untuk

Aplikasi OS Sailfish ditulis dalam C ++ dan QML (bagian grafis), seperti yang ditulis di situs resmi SDK meliputi:

- Lingkungan pengembangan terintegrasi (IDE) QtCreator
- Mesin pembuat Mer untuk kompilasi silang
- Emulator SailfishOS
- Tutorial, Desain, dan Dokumentasi API
- Repositori untuk perpustakaan tambahan dan kode sumber terbuka

Informasi lebih lanjut di: sailfishos.org/develop .

Jolla Phone

Katakanlah beberapa patah kata langsung tentang media OS Sailfish. Ia memiliki satu fitur yang menentukan seluruh desainnya, ponsel ini terdiri dari dua bagian: ponsel itu sendiri dan bagian belakang setengahnya. Setengah lainnya secara teoritis dapat berupa apa saja:
- panel biasa
- keyboard
- panel surya
- ...

Pada saat yang sama, bahkan panel biasa pun tidak sesederhana itu! Mereka memiliki tag NFC, terima kasih ketika mengubah panel, latar belakang, nada dering, dll. Perubahan Tentang keyboard: seorang penggila membuat keyboard yang disebut TOHK (google it), tetapi sekarang sayangnya Anda tidak dapat membelinya. Hanya anggota perusahaan di Kickstarter yang menerimanya.

Penilaian pribadi: Saya tidak terlalu canggih dalam hal smartphone, jadi lebih baik untuk hanya melihat foto-foto salinan saya (pada saat penulisan, dia berusia 4 bulan). Saya hanya bisa mengatakan bahwa perangkat ini stylish, tetapi masih ada kemungkinan goresan pada case di beberapa tempat.









Saya ketagihan bahwa para pengembang tidak "menjulurkan" logo mereka, itu ada di panel belakang, tetapi ada cukup tidak terlihat, dan di atas di tepi smartphone.

Kesimpulan


Perangkat ini ternyata sangat menarik, ya, Sailfish OS adalah platform yang agak muda: ada bug (meskipun "Saya hampir tidak menemukan bug anak-anak), belum ada banyak aplikasi asli (walaupun saya tidak menemukan kekurangan khusus, dan jika dukungan Android selalu membantu) ), ketidaklengkapan beberapa program default, namun demikian ada potensi dan itu sangat besar!

Filosofi manajemen asli, keterbukaan platform, desain menawan.

Adapun Jolla Phone itu sendiri, ternyata tidak seperti ponsel lain: desain "dua setengah", kombinasi bahan yang menarik, kemungkinan penyesuaian berkat setengah lainnya.

Secara umum, jika Jolla berkembang pada kecepatan yang sama, maka ia memiliki setiap kesempatan untuk mengambil tempat yang semestinya di jajaran OS mobile.
