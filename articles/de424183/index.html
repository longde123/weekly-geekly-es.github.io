4. Oktober, Moskau - Backend Stories 2.0

Am Donnerstag, dem 4. Oktober, werden wir auf der Deworkacy-Website ( Bolshaya Polyanka St., 2/10 , Seite 1 ) ein Treffen für Backend-Entwickler abhalten.



Wir beginnen um 19.00 Uhr, die Details sind unter dem Schnitt.

19: 20-19: 30, Pavel Derendyaev, Eröffnungsrede.

19.30–20: 10 Uhr, Ivan Pokhodnya: „Wie wir es in Java 11 wollten und fehlgeschlagen sind.“

Wie wir Spring mit Jigsaw in Java 11 starten wollten und scheiterten. Vanya zeigt, wie Sie die Anwendung auf Spring modular gestalten, mit Gradle erstellen und ausführen.

10.20–21: 00 Uhr, Alexander Kovalev, „CI / CD as a Service“.

Sasha wird darüber sprechen, wie GoCD in einer privaten Cloud bereitgestellt, infractrusture als Code darin konfiguriert und Hunderte von Pipelines verwaltet werden.

21.00–21.20 Uhr, Pause, auch bekannt als Snack (Burger).

21.20-22.05, Ivan Rasikhin, „Kappa-Architektur am Beispiel der Online-Buchhaltung der Alfa-Bank“.

Vanya wird darüber sprechen, wie wir Kappa Architecture im Internetbanking verwenden und warum die klassische dreistufige Architektur nicht zu uns passt.

05.22 - Fertigstellung.
