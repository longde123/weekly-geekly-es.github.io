Instrumente, die die Blickrichtung verfolgen (i-trackers)


Es ist äußerst wichtig, so früh wie möglich mit solchen Geräten zu arbeiten. Dies ist notwendig für diejenigen, die planen, wie sie während der Krankheit leben werden. Es gibt eine Version mit zusätzlichen Funktionen namens Eye Mobile Plus , die einen Infrarotsender zur Steuerung verschiedener Dinge wie Fernseher, eingebauter Lautsprecher und Mikrofon enthält.

Zusätzlich zu einem solchen Gerät verwende ich das einfachste Programm zum Konvertieren von Text in stimmhafte Sprache „ Balabolka “.

Für Leute, die nicht sehr technisch versiert sind, sind die iSeries- Geräte von Tobii möglicherweise besser geeignet und sehr einfach zu bedienen.

Schlaf


Kundenspezifisches Bett


Schlaf ist ein Kampf für ALS-Patienten und kann sehr unangenehm sein. Normalerweise erhalten Sie ein individuelles Versicherungsbett . Aber ich selbst habe ein Bett gekauft, das besser zu mir passt. Sie erlaubt mir, es weiterhin mit meiner Frau zu teilen, und es gibt genug Platz für die ganze Familie, um sich hinzusetzen und einen Film anzusehen. Ich empfehle icomfort von Serta. Sie hat eine Formgedächtnismatratze, und ich empfehle die weichste, um Druckstellen zu vermeiden. Sie werden das Modell „Split King“ lieben, bei dem es sich eigentlich um zwei verschobene Doppelbetten mit separaten Bedienelementen für jede Hälfte handelt.

Anpassbare Betten sind mit einer Vielzahl von Extras ausgestattet, darunter Fernbedienungen, Massagen und Fernbedienungsanwendungen. Ich empfehle, was auf Android-Geräten funktioniert. Wenn Sie einen Computer mit einem Gerät haben, das die Richtung Ihres Blicks verfolgt, können Sie ein Programm ausführen, mit dem Sie Android-Anwendungen verwenden und das Bett mit Ihren Augen steuern können.

Mein Bett hatte keine Kontrolle über Bluetooth und ich entwickelte meinen eigenen Bettcontroller auf Arduino.

Lichter, Lüfter, Steckdosen


Wemo


Wemo ist eine einfache Option für Hausbesitzer. Sie verfügen über Lichtschalter, die vorhandene Licht- und Lüftungsschalter problemlos ersetzen können . Sie können auch jede Steckdose zusammen mit dem Wemo-Steckdosenschalter verwenden , um alles fernzusteuern, was Sie an die Steckdose anschließen können, einschließlich Lampe, Lüfter, Fernseher, Radio usw. Darüber hinaus verfügen sie über ein Maker-Gerät, mit dem Sie fast alles verbinden und steuern können . Derzeit funktionieren Wemo-Geräte nur mit Android und iPhone. Sie können diese Anwendung jedoch verwenden, um sie unter Windows zu verwalten . Es sollte auch beachtet werden, dass diese Geräte funktionieren mit Amazon Echo, das unten beschrieben wird.

Beleuchtung


Eine gute Möglichkeit, das Licht in Ihrem Zuhause zu steuern, ist die Verwendung des Phillips Hue-Beleuchtungssystems. Das System ist sehr einfach in weniger als 5 Minuten zu installieren. Es gibt verschiedene Möglichkeiten, dieses System zu kontrollieren, und es kann in jedem Stadium der Krankheit eingesetzt werden. Es kann von der Standard-Farbtonanwendung für das Telefon gesteuert werden. Das System ist auch mit Amazon Echo kompatibel, sodass Sie die Beleuchtung mit Ihrer Stimme steuern können. Das System kann mit der Huetro-Anwendung auch von nahezu jedem Gerät aus gesteuert werden. Dies ist eine kostenlose Anwendung, die auf jedem anderen Betriebssystem als Linux ausgeführt wird. Wenn Sie Linux haben, können Sie Makros mit IFTT installieren. Dies ist jedoch keine ideale Option. Sie benötigen eine „Brücke“ für das Farbtonsystem, die etwa 50 US-Dollar kostet. Danach können Sie die Lampen für 20 US-Dollar kaufen.

Der Hub kann auch zur Steuerung von Smart-Home-Geräten verwendet werden und ist in Alexa integriert. Die Konfiguration als Universalfernbedienung ist sehr einfach.
