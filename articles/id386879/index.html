Editor GIMP Graphics Merayakan 20 Tahun



Pada 21 November 1995, siswa Peter Mattis mengumumkan di tiga newsgroup Usenet rilis versi beta dari editor grafis General Image Manipulation Program (GIMP) dengan "antarmuka grafis intuitif untuk banyak operasi pengeditan grafis."

Pada tahun 1997, pengodean singkatan GIMP diubah menjadi Program Manipulasi Gambar GNU untuk memenuhi rekursi.

Ini adalah program pertama di Linux. GIMP tetap menjadi editor foto terbaik setelah dua dekade, bahkan melampaui pemimpin pasar dalam perangkat lunak berpemilik Adobe Photoshop.

GIMP dibuat sebagai tesis oleh dua mahasiswa di Universitas California di Berkeley: Peter Mattis dan Spencer Kimball yang disebutkan sebelumnya. Mereka telah mengerjakan proyek sejak Juni, yaitu lima bulan.

Setelah rilis publik, program ini berkembang pesat - itu digunakan sebagai platform untuk menguji berbagai ide, karena GIMP sudah mendukung plugin dalam versi beta. Dengan demikian, dalam sejumlah besar plugin muncul untuk memproses format file baru dan menerapkan efek.

Pada 2006-2012, para pengembang bekerja sama dengan Peter Sikking untuk merumuskanesensi dari peningkatan proyek dan antarmuka. Akibatnya, GUI menjadi lebih akrab bagi para profesional, dan berbagai alat - lebih kuat dan mudah digunakan.





Dalam beberapa tahun terakhir, ada kerja keras untuk memindahkan GIMP ke mesin GEGL baru, kata pengembang . Saya harus menulis ulang atau mengubah hampir seluruh kode sumber. Sekarang pekerjaannya hampir selesai.

Untuk merayakan ulang tahun ini sedikit, para pengembang meluncurkan pembaruan GIMP 2.8.16 dengan beberapa peningkatan dalam dukungan untuk OpenRaster, file PSD dan sistem otomatisasi build untuk OS X dan berbagai perbaikan antarmuka.

Segera direncanakan untuk merilis versi 2.9.x yang tidak stabil, di mana mereka akan mendobrak mesin GEGL baru dan berbagai alat dan fungsi baru. Ini akan menjadi persiapan untuk rilis GIMP 2.10 dan pembaruan paling signifikan dalam tiga tahun terakhir. Kemudian direncanakan untuk menyelesaikan port GTK + 3 (diperlukan untuk mengembalikan dukungan untuk tablet Wacom di Windows) dan merilis GIMP 3.0.

Selain itu, peringatan antarmuka situs web resmi telah diperbarui, permalink muncul di berita dan umpan RSS / Atom . Situs ini sekarang memiliki desain karet dan versi seluler.

Secara umum, GIMP yang legendaris hidup dan berkembang, yang dengannya jutaan pengguna yang berterima kasih dapat memberi selamat kepada program favorit mereka!
