Gruppen in Slack



In Slack gibt es wie in jedem anderen Chatraum eine bequeme Möglichkeit, jemanden über seinen @userNamen oder Spitznamen zu kontaktieren . Für alle wird der Spitzname zu einem Link und wenn Sie darauf klicken, wird ein Menü für diesen Benutzer geöffnet. Der Spitzname wird dem Benutzer hervorgehoben und eine Benachrichtigung wird automatisch gesendet.

Was aber, wenn Sie mehrere Benutzer gleichzeitig kontaktieren müssen? Es gibt verschiedene Möglichkeiten, dies zu tun.

Der erste Weg: Umstellung auf Paid Slack


Vorteile von Paid Slack


  • Es gibt eine eingebaute Fähigkeit, sich in Gruppen zu unterteilen.
  • Unbegrenzte Integration.
  • Unbegrenztes Datei-Hosting.
  • Unbegrenzter Nachrichtenverlauf und Suche nach ihnen.
  • Detaillierte Statistiken.
  • Zugang für Gäste.
  • Und so weiter.

Nachteile von Paid Slack


Trotz der Vielzahl von Vorteilen gibt es auch Nachteile für die kostenpflichtige Version:

  • Teuer Der Mindesttarifplan beträgt im ersten Monat 6,67 USD pro Person und im folgenden 8 USD.
  • Die Anpassbarkeit der Benutzeroberfläche wird mit Ausnahme des Farbschemas nicht bereitgestellt.
  • Keine eingebauten Videoanrufe (in Kürze).

Der zweite Weg: Nachahmen von Gruppen durch Hervorheben von Wörtern


Zu diesem Zweck können Sie in den Benachrichtigungseinstellungen im Feld Wörter hervorheben alle erforderlichen Gruppen mit einem Komma auflisten.



Um herauszufinden, welche Gruppen existieren, oder eine neue hinzuzufügen, können Sie den integrierten Bot verwenden, indem Sie Schlüsselwörter hinzufügen, damit er auf sie reagiert, und Hilfe dazu veröffentlichen.



Die Antwort vom Bot für -groups:

Type `@` with one of these groups: `manager`, `react`, `js`. Example: `@example`. It will send notifications automatically when you add necessary groups in Highlight Words: https://slack.com/account/notifications#highlight_words_div

Die Antwort vom Bot für -channels:

Type `#` with one of channels in Slack. Example: `#example`. It will send notifications automatically when you add necessary channels in Highlight Words: https://slack.com/account/notifications#highlight_words_div

Also antwortet der Bot im Chat.



Gruppenunterschied


Gruppe beginnend mit:

  • @ - eine unabhängige Gruppe, es sieht aus wie gewöhnlicher Text, kann aber auch ein Link sein, wenn er bereits von einem Spitznamen übernommen wurde.
  • # - Eine Gruppe mit Personen aus diesem Kanal wird zu einem Link zum Kanal, der normalerweise für den Zugriff auf andere Kanäle verwendet wird.

Die Vorteile des Hervorhebens von Wörtern


  • Kostenlos.
  • Kann von Kanälen in Benutzergruppen unterteilt werden.

Nachteile beim Hervorheben von Wörtern


  • Jeder Benutzer muss alle erforderlichen Gruppen für sich selbst vorschreiben.

Der dritte Weg: die gesamte Routine an den Bot delegieren


Wenn im freien Slack etwas fehlt, kommt ein Bot zur Rettung.

Gruppen können auf bequeme Weise gespeichert werden, z. B. in einer Konfiguration oder, wenn Sie über Befehle Befehle aus dem Chat hinzufügen müssen, in der Datenbank.

Beim Zugriff auf eine bestimmte Gruppe, sofern im Wörterbuch eine vorhanden ist, sucht der Bot in der Profilbeschreibung aller Benutzer danach und sendet eine Nachricht mit einer Liste von Nicks. Alle diese Benutzer erhalten bereits eine Benachrichtigung auf der Ebene des Slacks.

Bot Vorteile


  • Kostenlos. Sie können kostenpflichtige Funktionen implementieren.
  • Verbesserte Anpassbarkeit von Nachrichten.
  • Sie können alle Funktionen implementieren und noch leistungsfähiger.

Nachteile bot


  • Es ist notwendig, einen Bot zu schreiben (auch mit vorgefertigten Lösungen).
  • Spam-Chat mit Zwischennachrichten. Sie können die Nachricht natürlich sofort löschen, aber es wird nur ärgerlich.
  • Abhängig vom Inhalt der Profilbeschreibung mit einer begrenzten Länge von 249 Zeichen.

Fazit


Wenn es in Slack viele Benutzer gibt, besteht die Notwendigkeit, sich in Gruppen für diejenigen zu unterteilen, die kompetenter sind und sich auf jedes Gebiet eng spezialisieren. Funktionen von einem kostenpflichtigen Slack können immer auf alternative Weise implementiert werden: sowohl über integrierte Funktionen als auch über einen Bot.
