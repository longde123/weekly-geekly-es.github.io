Di mana mendapatkan data untuk penelitian Anda


Semua data terbuka, diperoleh dari sumber resmi. Tahun lalu saya menggunakan gks.ru/accounting_report . Tapi tahun ini mereka masih belum memposting data baru (mungkin tersedia di API atau berdasarkan permintaan). Oleh karena itu, saya menggunakan set data terbuka dari IFTS, yang mulai diterbitkan baru-baru ini:

www.nalog.ru/opendata/7707329152-revexp
www.nalog.ru/opendata/7707329152-sshr

Setelah menggabungkan file xml dan mengonversi ke SQL, cukup menulis satu kueri SQL dengan daftar INN tahun lalu untuk dengan cepat mendapatkan data terbaru.

Berkat penggunaan data IFTS, informasi telah tersedia pada rata-rata jumlah karyawan masing-masing perusahaan untuk tahun ini. Tetapi Inspektorat Layanan Pajak Federal hanya memberikan angka-angka kering dari pendapatan dan pengeluaran, tanpa data rinci tentang laporan keuangan, seperti Komite Statistik Negara. Saya perhatikan bahwa data sering berbeda dari data dalam layanan verifikasi pihak lawan. Jika Anda membutuhkan data yang lebih akurat, lebih baik menunggu rilis dari Goskomstat.

IFTS juga memiliki satu set www.nalog.ru/opendata/7707329152-paytax , yang dengannya Anda dapat secara tidak langsung memperkirakan ukuran dana upah dan rata-rata gaji untuk organisasi.

Tahun ini, saya tidak mem-parsing data lalu lintas mereka untuk semua situs dan mempertimbangkan ECPM, karena ini membutuhkan banyak waktu, dan indikatornya sangat spesifik, tergantung pada banyak faktor. Saya menghitungnya hanya untuk beberapa situs yang menarik minat saya.

Cara menentukan TIN di situs


Untuk menemukan data pada perusahaan, Anda harus terlebih dahulu mengetahui NPWP-nya atau nama persis badan hukum tersebut. Saya menemukan hampir semua korespondensi badan hukum dan situs tahun lalu (lihat publikasi sebelumnya , ditambah saya menggambarkan kemungkinan opsi yang datanya mungkin tidak akurat / salah).

Opsi paling efektif untuk menemukan badan hukum di situs:

  • badan hukum ditunjukkan di situs itu sendiri (dalam kontak, dalam perjanjian pengguna),
  • pencarian dengan nama pendiri / pemimpin perusahaan di situs dengan informasi tentang badan hukum (lebih disukai dengan bantuan situs: operator di mesin pencari),
  • cari berdasarkan url situs di situs pencarian kerja besar, misalnya, www.google.ru/search?q=site%3Ahh.ru+tmtm.ru .

Saya menyambut inisiatif IFTS untuk membuka data, transparansi mengurangi korupsi dan meningkatkan persaingan, dan ini membuat ekonomi tumbuh lebih cepat. Saya mendesak orang-orang Khabrovsk untuk menggunakan data terbuka juga. Saya tidak yakin bahwa Anda dapat meraba-raba file SQL yang dihasilkan secara publik dari penggabungan dan konversi file xml, menulis dalam surat pribadi, saya akan memberikan tautan individual ke Yandex Disk.
