Terminal seluler dalam sistem kontrol akses

Perkembangan teknologi seluler telah mengarah pada fakta bahwa dalam sistem kontrol akses, smartphone dengan modul NFC semakin banyak digunakan tidak hanya sebagai pengenal, tetapi juga sebagai perangkat perekaman.


Bidang aplikasi


Solusi semacam itu cocok untuk fasilitas di mana tidak mungkin atau tidak menguntungkan untuk membangun terminal pendaftaran stasioner, tetapi kontrol akses dan akuntansi karyawan diperlukan. Ini bisa berupa tambang, rig minyak, situs konstruksi, bus bisnis, dan situs terpencil lainnya, termasuk tanpa koneksi internet.


Prinsip kerja


Di antara pabrikan Rusia, solusi seperti terminal akses seluler telah dipresentasikan oleh produsen terkemuka ACS: PERCo, Sigur, Parsec. Kami akan menganalisis prinsip terminal seluler menggunakan contoh solusi dari PERCo.


Ponsel cerdas dengan modul NFC dan aplikasi seluler yang diinstal digunakan sebagai terminal seluler. Aplikasi ini memungkinkan Anda untuk mendaftarkan pass karyawan dan pengunjung pada kartu akses format MIFARE.


Terminal seluler harus dimasukkan dalam konfigurasi kontrol akses dan sistem waktu kehadiran PERCo-Web.



Untuk bekerja dengan aplikasi ini, Anda harus memasukkan alamat IP dari server PERCo-Web atau memindai kode QR.



Data dapat dikirim ke server melalui jaringan Wi-Fi atau melalui jaringan seluler. Jika terminal offline, semua peristiwa akses disimpan dalam memori aplikasi dan dikirim ke server ketika koneksi terjadi.


Setelah menghubungkan terminal ke konfigurasi, dimungkinkan untuk mendaftarkan bagian-bagian karyawan dan pengunjung di dalamnya, yang akan ditampilkan dalam acara sistem.



Pendaftaran dapat dilakukan dalam beberapa mode:


  • "Pintu Masuk" - ketika kartu disajikan, pintu masuk terdaftar
  • "Keluar" - saat menunjukkan kartu, keluar terdaftar
  • "Verifikasi" - perlu untuk mengkonfirmasi izin lintas oleh operator menggunakan tombol Enter / Exit

Saat menyajikan pengidentifikasi, nama dan foto karyawan ditampilkan pada layar terminal. Layar juga menampilkan informasi apakah akses oleh pengidentifikasi ini diizinkan dalam periode waktu saat ini.


Terminal akses seluler memungkinkan Anda untuk mengatur pelacakan waktu karyawan. Berdasarkan data pada input / output terdaftar, sistem menghitung waktu kerja untuk bulan itu dan membentuk lembar waktu. Pergeseran, mingguan, dan jadwal bergulir didukung.


Terminal berguna jika terjadi keadaan darurat, misalnya kebakaran. Kemampuan untuk melacak lokasi orang dalam keadaan darurat secara signifikan meningkatkan kemungkinan penyelamatan.
