Kontes pengembang bot untuk TamTam



Bot API muncul di messenger TamTam baru-baru ini, 3 bulan lalu. Kami melakukan pengujian beta tertutup, di mana 86 pengembang menciptakan 330 bot. Bersama dengan penguji beta, kami melakukan ratusan pengeditan, merilis dua pembaruan Bot API dan satu untuk dokumentasi .

Kami memutuskan untuk mendukung pengembang dan bersama dengan Cloud Solutions Mail.ru, kami mengadakan kontes bot untuk TamTam. Dana hadiah - kupon dengan total 200.000 rubel untuk penggunaan infrastruktur cloud Mail.ru Cloud Solutions dan layanan apa pun berdasarkan platform ini. Selain pemenang, setiap peserta dalam kompetisi juga akan menerima kode promosi untuk 5.000 rubel untuk menggunakan layanan Cloud Solutions Mail.ru. Pendaftaran peserta terbuka hingga 21 Juli 2019 inklusif.

Untuk ikut serta dalam kompetisi, Anda perlu:

  • kembangkan bot untuk TamTam;
  • tulis ke bot kompetitif dan daftarkan bot Anda.

4 pemenang akan menerima kupon masing-masing 50.000 rubel selama 1 tahun menggunakan infrastruktur cloud Mail.ru Cloud Solutions. Pemenang akan dipilih dan diumumkan oleh tim TamTam di saluran Tam API Tam API Tam resmi pada 30 Juli 2019.

Dokumentasi dan perpustakaan klien:

  • Deskripsi Bot API ;
  • Skema JSAP OpenAPI ;
  • Klien Java Resmi ;
  • JavaScript
  • Kotlin DSL ;
  • Python

Aturan resmi kompetisi ada di sini .
