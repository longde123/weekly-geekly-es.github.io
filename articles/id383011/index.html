Pelontar tangan muncul dijual di AS



Orang-orang dengan Indiegogo masih belum punya waktu untuk merilis pada bulan September mereka "penyembur genggam pertama yang tersedia secara komersial" XM42 . Mereka berada di depan perusahaan XMatter LLC, yang sudah mulai menjual penyembur api X15, serta aksesori untuknya - kaleng karbon tambahan dan kaleng napalm.

Penjualan dimulai di 48 negara bagian Amerika di mana penyembur api diklasifikasikan sebagai "perangkat hiburan", kecuali California dan Maryland, penyembur api dilarang di sana.

Di sebagian besar negara bagian, penyembur api tidak dianggap sebagai senjata api (ada beberapa ironi dalam hal ini), sehingga penjualan mereka tidak diatur oleh hukum. Ada departemen senjata di supermarket Amerika: senapan dan pistol dijual di sebelah kaus kaki dan celana dalam. Tapi di sana Anda perlu mendapatkan izin khusus dari pihak berwenang, dan tidak ada izin yang diperlukan untuk membeli penyembur api.

Pabrikan menekankan kualitas produk yang tinggi. Tangki tahan tekanan 4000 psi (dari kaleng CO 2 ), yang empat kali lipat dari beban kerja. Setiap penyembur api diperiksa secara individual.



"Perangkat hiburan" bekerja langsung di luar kotak, tidak ada alat atau pengaturan tambahan yang diperlukan. Jarak yang terpengaruh tergantung pada campuran yang digunakan. Secara default, penyembur api diisi dengan campuran diesel 90% dan bensin 10% dan ketukan selama 15 meter. Kekuatan mematikan maksimum dicapai dengan 75% bensin, 25% bahan bakar diesel dengan penambahan napalm (lihat video, di mana campuran berbeda dibandingkan).



Secara umum, tekanan silinder disesuaikan. Pada tekanan maksimum, silinder 11,3 liter berlangsung selama sekitar satu menit pembakaran. Dengan napalm sedikit lagi. Hampir tidak ada yang tergantung pada jumlah oktan bensin, jadi disarankan untuk membeli yang termurah ke-87.

Para pengembang menyarankan bahwa produk $ 1.599 mereka akan digunakan dalam pertanian untuk membersihkan tanah, untuk menghancurkan gulma dan serangga berbahaya, untuk menghilangkan salju dan es dari rel, dan untuk menciptakan efek kembang api yang indah pada Hari Kemerdekaan, ketika kembang api diatur secara tradisional di seluruh Amerika. .
