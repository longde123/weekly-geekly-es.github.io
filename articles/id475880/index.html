7 Kursus Gratis untuk Arsitek Solusi Microsoft

Halo, Habr! Hari ini kita berada di garis katulistiwa dari serangkaian kursus gratis keren dari Microsoft. Pada bagian ini, kami memiliki kursus paling keren untuk arsitek solusi . Semuanya dalam bahasa Rusia, Anda bisa memulainya sekarang, dan pada akhirnya Anda akan menerima lencana. Bergabunglah sekarang!

1. Membuat bot pintar


Interaksi pengguna dengan aplikasi komputer melalui percakapan menggunakan teks, gambar atau ucapan dapat diimplementasikan menggunakan bot. Ini bisa berupa dialog "tanya jawab" sederhana atau bot kompleks yang memungkinkan orang untuk berinteraksi dengan layanan secara cerdas, menggunakan pencocokan pola, pelacakan negara, dan metode kecerdasan buatan. Dalam kursus ini, dalam 2,5 jam, Anda akan belajar cara membuat chatbot cerdas menggunakan QnA Maker dan integrasi LUIS.

Pelajari lebih lanjut dan mulai pelatihan di sini.

2. Mengembangkan dan mengonfigurasi aplikasi ASP.NET yang mengakses Azure SQL Database


Buat database untuk menyimpan data aplikasi dan konfigurasikan aplikasi ASP.NET yang meminta data dari database ini. Hanya satu jam dan Anda selesai! By the way, untuk mengambil kursus Anda perlu memiliki pemahaman umum tentang basis data relasional dan pengetahuan dasar tentang C #.

Topik-topik berikut disajikan dalam modul ini:

  • Pembuatan, konfigurasi, dan pengisian basis data terpisah di layanan "Azure SQL Database";
  • Konfigurasikan aplikasi ASP.NET yang mengakses database ini.

Detail dan mulai pelatihan

3. Menyeimbangkan Lalu Lintas Layanan Web Menggunakan Gateway Aplikasi


Dengan mempelajari modul ini, Anda akan belajar cara meningkatkan ketahanan aplikasi dengan mendistribusikan beban di beberapa server dan menggunakan perutean lalu lintas web.

Dalam modul ini Anda akan belajar cara melakukan tugas-tugas berikut:

  • Menentukan kemungkinan load balancing gateway aplikasi;
  • Membuat Gateway Aplikasi dan mengatur penyeimbangan muatan;
  • Konfigurasikan Gateway Aplikasi untuk perutean berbasis URL.

Detail dan mulai pelatihan

4. Menyebarkan dan meluncurkan aplikasi kontainer web menggunakan Layanan Aplikasi Azure


Buat gambar Docker dan simpan ke repositori Azure Container Registry. Gunakan Layanan Aplikasi Azure untuk menggunakan aplikasi web dari gambar Docker. Konfigurasikan penyebaran aplikasi web berkelanjutan dengan sniffer web yang melacak perubahan pada gambar Docker.

Dari modul ini Anda akan mempelajari yang berikut ini.

  • Buat gambar Docker dan simpan di repositori Azure Container Registry;
  • Meluncurkan aplikasi web dari gambar Docker yang disimpan di Registry Container menggunakan Azure App Service;
  • Konfigurasikan penyebaran aplikasi web secara berkelanjutan dari gambar Docker menggunakan interceptor web.

Detail dan mulai pelatihan

5. Menyebarkan situs web di Azure menggunakan Layanan Aplikasi Azure


Aplikasi web di Azure membuatnya mudah untuk menerbitkan dan mengelola situs web tanpa khawatir tentang server inti, penyimpanan, atau sumber daya jaringan. Kursus ini mencakup dasar-dasar penerbitan situs web menggunakan Azure. Ini akan memakan waktu sekitar 5 jam untuk belajar.

Modul:

  • Mempersiapkan lingkungan pengembangan Anda untuk Azure
  • Host aplikasi web menggunakan Layanan Aplikasi Azure;
  • Publikasikan aplikasi web ke Azure menggunakan Visual Studio;
  • Mempersiapkan penerapan aplikasi web untuk pengujian dan rollback menggunakan slot penerapan layanan aplikasi
  • Skala aplikasi web layanan aplikasi untuk memenuhi permintaan secara efektif dengan meningkatkan layanan aplikasi Azure secara vertikal dan horizontal;
  • Menyebarkan dan meluncurkan aplikasi web kontainer menggunakan Layanan Aplikasi Azure.

Detail dan mulai pelatihan

6. Tinjauan gaya arsitektur n-tier untuk aplikasi


Menggunakan templat Resource Manager untuk menyebarkan aplikasi dalam arsitektur n-tier, mendefinisikan konsep dasar arsitektur n-tier, kiat dan trik untuk menyebarkan aplikasi tersebut.

Dalam modul ini Anda akan belajar cara melakukan tugas-tugas berikut:

  • Identifikasi fungsi, batasan, dan aspek penting dari arsitektur n-tier;
  • Identifikasi kasus penggunaan untuk arsitektur n-tier;
  • Menyebarkan contoh arsitektur n-tier menggunakan templat Resource Manager;
  • Identifikasi metode dan sumber daya untuk meningkatkan arsitektur n-tier.

Detail dan mulai pelatihan

7. Pemrosesan dan Klasifikasi Gambar dengan Layanan Visi Azure Cognitive


Microsoft Cognitive Services menawarkan fitur bawaan untuk mengaktifkan visi komputer dalam aplikasi. Pelajari cara menggunakan Layanan Visi Kognitif untuk mendeteksi wajah, menandai dan mengklasifikasikan gambar, dan mengidentifikasi objek.

Modul:

  • Identifikasi wajah dan emosi menggunakan API visi komputer di Azure Cognitive Services;
  • Pemrosesan gambar menggunakan layanan visi komputer;
  • Klasifikasi gambar menggunakan layanan pengenalan visual khusus;
  • Penilaian persyaratan untuk implementasi Pengakuan Visual Kustom API.

Detail dan mulai pelatihan
