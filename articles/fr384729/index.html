Kibbi - le système de sécurité universel pour la maison


Kibbi avec capteurs

Indiegogo a lancé un autre projet intéressant - un système de sécurité avancé avec des capacités qui dépassent de loin les capacités de la plupart des systèmes similaires. Ce système peut être utilisé pour protéger votre appartement et pour travailler dans une maison de campagne ou un bureau. Kibbi se compose de deux parties: un moyeu central avec un grand nombre de capteurs et une caméra HD et des capteurs grand angle.

Il y a beaucoup de capteurs, et ils sont conçus pour fixer l'ouverture / fermeture des portes et fenêtres, le mouvement de divers objets (comme un téléviseur) dans l'espace. Les capteurs peuvent surveiller les vibrations, la température et certaines autres caractéristiques environnementales. Les capteurs Kibbi n'ont pas peur de l'eau, ils peuvent donc être utilisés à l'intérieur comme à l'extérieur.



Comment ça fonctionne?


En général, tout est standard ici. Si les données transmises du capteur au concentrateur principal sont quelque peu différentes de celles standard, le système émet un signal d'avertissement, qui est envoyé à l'appareil mobile de l'utilisateur.

En même temps, la caméra du système commence immédiatement à fonctionner afin de donner à l'utilisateur une idée de ce qui se passe. La vidéo est diffusée en temps réel, plus elle est enregistrée dans le "cloud", où à tout moment les fichiers vidéo peuvent être visualisés à nouveau.



Si nécessaire, l'utilisateur peut allumer l'appareil photo de l'appareil de manière indépendante, si vous souhaitez voir ce qui se passe à la maison. Dans le service cloud, vous pouvez enregistrer jusqu'à un gigaoctet de données, ce qui est suffisant pour enregistrer un clip suffisamment long. S'il y a un tel besoin, Kibbi peut enregistrer la vidéo sur un disque local (par exemple, un disque externe).

Activer et désactiver Kibbi est facile, tandis que les développeurs ont fourni un mode hors ligne. Pour ce faire, une application spéciale est installée sur le smartphone, qui suit la position de l'utilisateur. Si le hub Kibbi détecte la suppression du smartphone avec l'application, le mode de sécurité démarre automatiquement. Si le concentrateur détecte la proximité du smartphone avec l'application, le système s'éteint.

Sous le toit de ma maison : ce mode est conçu pour suivre le nombre de personnes dans la maison ou l'appartement de l'utilisateur. Si nécessaire, Kibbi indiquera combien de personnes ont quitté la maison et qui y est entrée.



Communication bidirectionnelle : Kibbi peut également être utilisé comme système de téléconférence. L'appareil est équipé d'une caméra de haute qualité et d'un excellent système de haut-parleurs.

Service IFTTT


Kibbi est compatible, entre autres, avec le service IFTTT (
If This Then That). Ce service ouvre de grandes opportunités pour connecter divers événements et créer une chaîne de cause à effet.



Par exemple, si le capteur Kibbi détecte un mouvement, la lampe intelligente s'allume automatiquement. Ou si la température ambiante descend en dessous d'un certain niveau, l'élément chauffant est allumé.

Caractéristiques


  • Caméra Full HD avec un grand angle de vision - 130 degrés;
  • Orateur
  • Microphone;
  • Module Bluetooth 4.0;
  • Module Wi-Fi;
  • Batterie de secours - le système peut fonctionner même lorsque l'électricité est coupée dans la maison;
  • Prise en charge de capteurs supplémentaires.


Conclusion


Le système est vraiment universel - et si certaines fonctions ne fonctionnent pas, vous pouvez configurer vous-même des fonctionnalités supplémentaires. Un énorme avantage est la possibilité de travailler à partir d'une batterie de secours - vous pouvez donc être sûr que Kibbi fonctionnera même lorsqu'il n'y a pas de lumière dans la maison.
