PHP Digest n ° 132 (27 mai - 10 juin 2018)


Nouvelle sélection avec des liens vers des nouvelles et des documents. En version: PHP 7.3.0 alpha 1, Yii 3.0 alpha 1 et d'autres versions, le dernier livre PHP asynchrone, des enregistrements vidéo de rapports, une partie d'outils utiles, et bien plus encore. Bonne lecture!


Nouvelles et communiqués


  • PHP 7.3.0 alpha 1 - La première version alpha de la nouvelle branche PHP est disponible pour les tests. Un cycle de test de six mois a commencé, la version finale est attendue fin novembre.
    • Parmi les changements:
      • Amélioration de la syntaxe RFC Heredoc et Nowdoc
      • Les fonctions json_encode () et json_decode () lèveront une exception en cas d'erreur et l'indicateur RFC correspondant
      • Possibilité d'utiliser une virgule après le dernier argument de la fonction RFC
      • Nouvelle fonction hrtime (), qui renvoie un temps de haute précision et ne dépend pas de l'heure système PR
      • Ajout de la fonction is_countable () pour vérifier l'énumération RFC
      • Utilisation de liens dans la liste () du RFC
      • Amélioration significative du ramasse-miettes PR
      et d'autres améliorations. Liste complète des changements .
  • Phalcon 3.4.0
  • Dernières nouvelles de PHP-FIG: avril et mai - PSR-5 (PHPDoc) est divisé en deux: formatage et catalogue de balises, PSR-9 et PSR-10 (sécurité) ont vu le jour avec un nouveau leader, PSR-14 (Event Dispatcher) dans le processus , PSR-17 (usines http) et PSR-18 (clients http) sont presque prêts à passer en revue.
  • L'état de l'écosystème des développeurs en 2018 - rapport JetBrains basé sur une enquête auprès de 6000 développeurs.


Les outils


  • z7zmey / php-parser v0.5.0 - L'analyseur PHP implémenté sur Go continue de se développer.
  • nesk / rialto - Un outil pour gérer les ressources Node.js à partir de PHP. Il peut être utilisé pour implémenter l'interaction avec les bibliothèques Node.js.
  • nesk / puphpeteer - Un exemple d'utilisation de nesk / rialto, un wrapper PHP pour travailler avec GoogleChrome / puppeteer (Headless Chrome).
  • mlambley / swagception - Génère des tests d'acceptation pour Codeception basés sur votre spécification Swagger 2.0 (Open API 2.0).
  • RobThree / TwoFactorAuth - Authentification à deux facteurs basée sur des mots de passe ou codes QR TOTP à usage unique.
  • OwenMelbz / oublier-db - Un outil simple pour anonymiser (par exemple, GDPR) les données dans la base de données.
  • gordalina / cachetool - outil CLI pour gérer apc et opcache.
  • digitaldonkey / ethereum-php - Interface typée pour l'API Ethereum JSON-RPC.
  • SocialConnect / auth - Bibliothèque pour l'authentification via les réseaux sociaux. services (OAuth1 / 2, OpenID, OpenIDConnect). Plus de 25 fournisseurs prêts à l'emploi.
  • xobotyi / rsync - Un wrapper pratique sur rsync pour PHP 7.1+. Soumis par hobotyi .
  • krakjoe / idbg - Un débogueur pour PHP implémenté en PHP, cependant, nécessite jusqu'à présent l'extension krakjoe / idbg . Poster avec une description.
