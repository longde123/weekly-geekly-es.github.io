BlueStacks 3: ikhtisar versi baru emulator Android untuk Windows



Suatu hari, BlueStacks memperkenalkan versi baru "pemain aplikasi" untuk Android pada Windows. Saya sudah menulis beberapa kali tentang platform ini (ditambah lagi saya mewawancarai kepala perusahaan), jadi kali ini saya tidak bisa melewatinya. Secara umum, saya menarik perhatian BlueStacks secara tidak sengaja: satu setengah tahun yang lalu, Bethesda merilis Fallout 4, dan dengan itu permainan untuk sistem mobile Fallout Shelter. Kemudian saya benar-benar ingin memainkan yang pertama dan yang kedua.

Tidak ada masalah dengan yang pertama - ambil konsol dan mainkan. Tetapi dengan Fallout Shelter tidak begitu sederhana. Faktanya adalah ponsel saya, Sony Experia T3, sangat jauh di belakang persyaratan game dalam spesifikasinya sehingga Google Play bahkan tidak menunjukkan kepada saya Fallout Shelter saat mencari. Masuk dari laptop, saya melihat permainan, tetapi ketika saya mencoba menginstalnya di telepon, saya menerima pesan bahwa perangkat seluler tidak mendukung permainan ini. Ada dua solusi untuk masalah ini sekaligus.

Yang pertama adalah mengambil tablet dari putri Anda (Samsung Galaxy 2) dan meletakkan mainan itu di atasnya. Masalahnya adalah putri saya tidak setuju untuk memberi saya tablet sehingga saya bisa bermain. Jadi saya mulai mencari alternatif dan menemukan emulator Android untuk Windows. Setelah mempelajari beberapa artikel, saya menyadari bahwa salah satu yang terbaik adalah BlueStacks. Setelah mengunduhnya, saya mulai terbiasa, mencoba menginstal permainan dan menyadari bahwa semuanya baik-baik saja.

Seperti disebutkan di atas, semua ini terjadi satu setengah tahun yang lalu. Saya kemudian melemparkan permainan, mengisi seluruh tempat penampungan dengan penghuni bahagia, robot, peralatan dan melakukan semua yang saya bisa. Tapi saya ingat sekarang, setelah mengetahui bahwa para pengembang telah menambahkan banyak fungsi dan produk baru yang berbeda. Yah, saya pergi sepanjang jalan yang sudah dipukuli - saya mengunduh BlueStacks dari situs resmi , ini adalah versi ketiga.



Setelah mengunduh, muncul penginstal yang mengunduh file yang diperlukan dan menginstal BlueStacks. Omong-omong, saya menguji operasi platform ini pada laptop Sony Vaio dengan karakteristik berikut:
  • RAM - 8 GB DDR 3;
  • Prosesor - Intel Pentium;
  • Grafik - Intel HD;
  • Penyimpanan - 128 GB SSD;
  • OS - Windows 10, versi 64-bit;
  • Model Notebook - Sony Vaio sve171e13v

Bukan platform perangkat keras tercepat, seperti yang Anda lihat. Sejauh yang saya ingat, BlueStacks awalnya bekerja tanpa lag. Sekitar beberapa bulan setelah instalasi, saya menghapusnya, dan setelah enam bulan, saya menginstalnya lagi. Dan untuk kedua kalinya setelah instalasi, untuk beberapa alasan itu bekerja sangat lambat. Ini berlaku untuk beberapa aplikasi, termasuk Wallapop. Perangkat lunak adalah platform jual-beli tempat penduduk setempat memajang barang, dan Anda sering dapat membeli sesuatu dengan harga yang sangat baik.

Saat memasang versi ketiga, saya ragu laptop saya akan menariknya.



Instalasi selesai dengan cerdas. Di sini, mungkin, kelebihan SSD lebih, namun semuanya bekerja lebih cepat dengannya. Setelah instalasi selesai, jendela penuh warna muncul.



Setelah itu, seperti biasa, dibiasakan dengan kemampuan perangkat lunak. Saya harus mengatakan bahwa pada tahap ini tidak ada yang melambat, yang saya terkejut mengingat versi kedua.









Dan akhirnya, pengaturan akhir, setelah itu dimungkinkan untuk mulai mempelajari kemampuan program.



Seperti pada platform Android mana pun, kami mengonfigurasikan semua data, menentukan nama, bahasa, dan opsi lain yang kami inginkan.



Jelas bahwa tanpa akun Google, tidak ada yang bisa dilakukan, jadi masukkanlah. Saya tidak membuat yang baru, saya menggunakan data akun saya yang ada.





Sejauh yang Anda mengerti, Bluestacks lebih fokus pada permainan, yang mungkin mengapa mainan paling populer dari katalog Google Play ditampilkan di "Pusat Aplikasi". Secara umum, antarmuka nyaman. Ada tiga tab "Aplikasi Saya", "Pusat Aplikasi" dan "Umpan Balik".

Ngomong-ngomong, momen lain. Segera setelah instalasi, program mulai sangat merekomendasikan mempercepat Bluestacks menggunakan virtualisasi VT-x. Ini dapat dilakukan di BIOS dengan mengaktifkan opsi yang sesuai. Saya mengunduh program untuk memeriksa ketersediaan opsi ini untuk diri saya sendiri, perangkat lunak menunjukkan bahwa ini dapat dilakukan tanpa masalah. Tapi karena semuanya bekerja dengan baik, saya tidak pergi ke BIOS.



Antarmuka telah mengalami perubahan yang signifikan, bahkan saya katakan, dramatis. Ini berlaku untuk tampilan dan fungsionalitas. Pusat aplikasi dalam prosesnya disesuaikan dengan minat pengguna. Tentu saja, jika Anda bekerja dengan Bluestacks beberapa kali dalam sebulan, tidak akan ada banyak perubahan, tetapi jika Anda sering menggunakan pemain, maka jendela akan segera berubah. Ya, nilai tambah besar adalah multi-jendela BlueStacks. Berbagai aplikasi terbuka di tab, seperti di peramban, dan masing-masing dapat digunakan kapan saja.



Segera setelah instalasi, saya mengunduh Fallout Shelter. Semuanya seperti biasa: aplikasi membutuhkan sejumlah izin, kami setuju dengan semuanya dan unduhan game.



Ya, semuanya sudah diinstal, Anda bisa bermain. Omong-omong, para pengembang mengatakan bahwa sekarang platform bekerja dengan mesin grafis baru, yang disebut Hyper-G. Ini memungkinkan Anda untuk menggunakan semua yang tersedia pada saat sumber daya game komputer untuk bekerja dengan aplikasi (baca - permainan). Komponen grafis tetap pada pengaturan semaksimal mungkin. Saat berjalan di PC, grafik gim tidak memburuk.

Ngomong-ngomong, selain Fallout, saya mengunduh aplikasi lain, Wallapop, yang disebutkan di atas. Ke depan, saya akan mengatakan bahwa bersamanya semuanya tidak semulus dengan Fallout. Kadang-kadang itu tidak berhasil sama sekali, kadang-kadang bekerja dengan sempurna, begitu program macet dengan pesan seperti itu.



Di sini kita dapat mengatakan bahwa Wallapop sendiri pada umumnya buggy. Saya menjalankannya di berbagai perangkat seluler, dan pada salah satu dari mereka aplikasi ini terkadang berperilaku sangat aneh. Entah pesan di PM mengganggu, lalu program macet, lalu pemberitahuan tidak ditampilkan. Kemungkinan besar, Wallapop sendiri lebih disalahkan untuk gangguan daripada BlueStacks.

Tapi kembali ke Fallout. Di sini, nilai tambah besar dalam permainan adalah kemampuan untuk mengendalikan mouse dan, jika diinginkan, juga keyboard. Jelas bahwa game mobile diasah untuk layar sentuh, tetapi dalam banyak kasus game yang lengkap dengan keyboard dan mouse lebih nyaman daripada layar. Ya, dan pada 17 inci Anda dapat mempertimbangkan detail lebih dari 5,5 inci.



Tidak ada masalah dengan Fallout, tidak sekali pun. Suara, kontrol, dll. bekerja dengan baik. Kecepatan unduhan hampir sama dengan pada perangkat seluler. Program prosesor dimuat dengan kokoh. Tapi saya punya prosesor lama, saya ingin mengganti semuanya dengan yang lebih kuat, tapi tangan saya tidak bisa menjangkau.



Play Store terlihat, seperti biasa, tidak ada yang muncul dan menghilang.



Nah, ini antarmuka game itu sendiri, dalam mode layar penuh (ya, Anda bisa bermain di jendela, tetapi Anda bisa di layar penuh)



Sedangkan untuk game atau aplikasi lain, mereka dapat diunduh di "Pusat Aplikasi" yang sama. Semua program di sini diberi peringkat berdasarkan popularitas di antara pengguna. Ketika Anda mengklik salah satunya, informasi tentang permainan muncul dengan tangkapan layar, ulasan, dan data lainnya.

Jika perlu, Anda dapat bekerja (atau memainkan) dari berbagai akun di Google Play. Untuk ini ada opsi khusus "Multi-jendela". Selain itu, program dapat diinstal langsung menggunakan file .apk.



Kelebihannya adalah antarmuka pemutar BlueStacks, meskipun mengandung beberapa opsi, tidak terlalu jenuh dengan detail. Semua elemen didistribusikan secara rasional, jadi Anda tidak perlu mencari-cari fungsi yang diinginkan. Tentu saja, saya tidak menggunakan semua fitur program, tetapi sepertinya mudah untuk bekerja dengannya.

Omong-omong, ada dua versi BlueStacks - berbayar dan gratis. Yang pertama berbeda dari yang kedua dengan kehadiran iklan, semua fungsi disertakan. Iklan ditampilkan sebelum mengunduh aplikasi dan terlihat seperti ini.



Di bagian pertama pemain, saya ingat, itu tentang hal yang sama, hanya iklan yang lebih sering ditampilkan, yang sangat mengganggu dalam proses bekerja dengan aplikasi apa pun. Ini jarang terjadi di sini, sehingga saraf bisa diselamatkan.

Dan akhirnya, saya memutuskan untuk mendapatkan informasi tentang kinerja "perangkat seluler" saya menggunakan program AnTuTu Benchmark.



AnTuTu membuat saya mengunduh aplikasi lain, 3DBench. Nah, unduh. Setelah itu, pengujian dimulai.

Sebagai kesimpulan


Pemain aplikasi ternyata pintar, mampu mereproduksi aplikasi "berat", termasuk game dan program lain, tanpa masalah khusus. Terkadang, jarang, masalah muncul seperti aplikasi macet selama operasi. Tapi saya hanya melihat ini dengan Wallapop, jadi sulit untuk mengatakan apakah BlueStacks yang harus disalahkan atau program itu sendiri. Saya menyukai versi ketiga lebih dari yang kedua, karena lebih cepat, antarmuka ditingkatkan, sejumlah fitur ditambahkan.
