O nascimento da matriz: redes neurais artificiais aprendidas a criar rostos realistas de pessoas e interiores de quartos



Na Matrix, em sua primeira parte, não apenas um sistema de computador inteligente poderia criar um mundo virtual, mas os computadores de Nabucodonosor, um navio rebelde, também podiam gerar um mundo virtual em miniatura. Muitos se lembram da garota de vermelho, que apareceu em uma das simulações - ela é mostrada na foto no anúncio. Antes de criarmos um modelo virtual do mundo real, nós (e nossos computadores) ainda estamos muito distantes. Mas algo já está lá agora.

Uma equipe de pesquisadores do Indico e do Facebook criou uma rede neural que pode "inventar" imagens realistas. A propósito, a evolução das redes neurais pode ser encontrada aqui . Rede Neural Artificial ( RNA)) - um modelo matemático, bem como sua implementação de software ou hardware, construído com base no princípio de organização e funcionamento de redes neurais biológicas - redes de células nervosas de um organismo vivo. Agora, as RNAs são usadas em muitas áreas, incluindo previsão, reconhecimento de padrões e trabalho com Big Data. Alec Radford, do Indico e seus colegas, decidiram prestar atenção a um tipo de NIS como a rede geradora de adversários.



Nessa rede, uma parte do sistema tenta criar uma matriz de dados falsos para "enganar" a segunda parte. A idéia era que, se você repetir constantemente esse processo, o sistema aprenderá a criar imagens melhores.

Os autores do estudo treinaram um sistema baseado em uma série de imagens dos quartos. Em seguida, o sistema foi solicitado a criar suas próprias imagens. Para garantir que o sistema realmente crie imagens originais e não as copie do banco de dados, os especialistas deram o comando para gerar uma série de modificações na mesma imagem do quarto - por exemplo, sem janela, com mesa ou TV. Essa técnica permitiu verificar se o sistema aprendeu a gerar interiores por conta própria.

Além disso, os especialistas treinaram o sistema para criar imagens, onde uma parte importante (por exemplo, uma janela) é substituída por algo. Por exemplo, um sistema substitui uma janela por uma televisão ou lareira. Isso indica que o sistema aprendeu a "entender" o que constitui uma determinada parte do interior.



Um princípio semelhante pode ser usado para outras fotos, como fotografias de retrato. Em outro experimento, os pesquisadores pediram ao sistema para destacar uma foto de uma mulher sorridente, depois adicionar uma expressão neutra e "adicionar" um homem com uma expressão neutra. O objetivo é destacar o conceito de "sorriso" e combiná-lo com o conceito de "homem".

Como resultado, os pesquisadores receberam um conjunto de imagens de homens sorridentes.

Agora, infelizmente, o tamanho das imagens é limitado a 32 * 32 pixels, o que torna o sistema mais rápido e mais eficiente. A próxima etapa do trabalho é a transferência desse esquema para vídeo e áudio.

E depois que o efeito 3D é adicionado, mais dinâmica, mais detalhes ambientais adicionais, obtemos uma "Matrix" completa. Ou não?
