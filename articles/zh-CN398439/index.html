阴谋论。三星手机爆炸


事故发生前15秒（上）和事故发生前1秒（下）的锂离子电池的高速计算机断层扫描结果。来源：科学文章“ 用的锂离子电池在-operando高速断层摄影过程中的热失控 ”，自然通信6，商品编号：6924（2015），DOI：10.1038 / ncomms7924（公开可用）

在八月份的在媒体中的第二半在Galaxy Note 7手机自燃的情况下，大众媒体开始了真正的歇斯底里

，任何电器的所有者都知道，频繁使用电池总是会发热。这是电池的完全自然属性。与此同时，工程师会告诉你说在某些情况下，任何带有锂离子电池的设备都可能爆炸。锂离子电池的高速X射线计算机断层扫描和射线照相结果表明，过热的锂离子电池会产生一种连锁反应，可能导致爆炸。这不仅适用于电话，而且通常适用于任何小工具。设计者的任务是引入安全措施，以将失火的风险降至最低，尽管不可能将其降至零。

实际上，这种锂离子电池爆炸很少见，但经常发生。根据美国消费者产品安全委员会的官方统计，自2002年以来，由于锂离子电池缺陷，已经有43次市场召回。 Galaxy Note 7的故事根本不是什么特别的故事，而是一个相当普通的技术问题。



就像富士康工厂臭名昭著的自杀一样，大数法则在这里起作用。记住“大规模自杀”的歇斯底里据说是由于工作条件差而在富士康的工厂？在2010年1月至2010年5月之间，台湾富士康工厂发生了10起自杀事件。贫穷的中国人通过组装iPhone和iPad牺牲了生命，这一点很高。例如，每天工作12个小时，每月需要150美元-这简直令人难以忍受，人们无法忍受并自杀。

但是，如果看这些数字，一切看起来就完全不同了。即，有33万人在台湾富士康工厂工作，该工厂是世界上最大的计算机组件生产工厂。。中国人的平均自杀率是每年每十万人中有十四人自杀。尽管65岁以上的人比年轻人更容易解决生活问题，因此，人们可以估计年轻人中的正常自杀率约为每年每100,000人中有10人。因此，自年初以来的过去147天中，应该发生了（330,000 / 100,000）* 10 * 147/365 = 13.3例死亡。但实际上，仅发生了10次，大大低于平均水平。

这就是整个解释。

尽管如此，公众舆论很难给出合乎逻辑的理由，因此富士康选择不争论，而是停止冲突，提高工资并改善工作条件，后来宣布通过机器人大量更换输送机上的工人。

那怎么了三星智能手机“大规模”爆炸的背后是谁的这种歇斯底里？

首先，让我们看看故事开始被媒体报道的一般背景。发布后立即使用的同一款Galaxy Note 7被认为是公司历史上最好的三星智能手机。有人称其为市场上绝对最佳的智能手机。

然后突然-一个爆炸的故事，之后该公司被迫宣布大规模召回智能手机，停止生产Note 7智能手机并停止销售。 Note 7品牌终于被埋葬了。 “世界上最好的”智能手机将永远退出市场。惨败。 “未知”敌人的绝对胜利。

我们立即注意到，三星的行动只是部分被迫。可以将它们称为“过度反应”，风险管理人员建议使用该反应，以完全并完全阻止潜在风险。与富士康的情况一样，最好是停止战斗并立即放弃，而不是修复品牌多年的损害，尤其是因为这种损害是无法弥补的。如果局势不能迅速彻底地得到纠正，那么敌人将不会在局部战斗中赢得小胜利，而会在全球战斗中赢得大胜利。在这种情况下，三星品牌可能总是被贴上“爆炸手机制造商”的标签。在未来的几年中，这可能会导致市场份额的重大损失，销售下降，并且从长远来看，甚至可能导致公司完全放弃生产智能手机。为了确保不能保证这一点，韩国风险管理者通过仅给出“一个城市”即Note 7品牌来解决该问题。

纯粹从数学上讲，最好保证损失数十亿美元，而不是获得十倍于一百倍的损失的概率。
