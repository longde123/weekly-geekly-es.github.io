PyTest Neo

Uma noite, contemplando testes sem fim e olhando para os pontos de corrida, um pensamento surgiu - como se parece com a Matrix! E assim nasceu a ideia de fazer a visualização de testes na forma de uma Matriz.



Sob animação de gato e alguns detalhes.



Se alguém estiver preocupado com a legibilidade, apenas a barra de progresso será animada e, se houver erros, um relatório padrão será exibido.

Para instalar o plugin pytest-neo , basta: 

pip install pytest-neo

O projeto está em um estágio inicial de desenvolvimento. Linux e presumivelmente MacOS são suportados. Possíveis erros. Até agora, o modo detalhado é deixado como está. Os planos nele para implementar animação confiável da Matrix.

Por favor, escreva sugestões e sugestões. Isso ajudará muito o desenvolvimento do projeto.

PS: nos testes aiohttp foi detectada uma anomalia, estudarei.
PPS: um projeto para um fã, infelizmente nem todo mundo entendeu isso e deu contras
