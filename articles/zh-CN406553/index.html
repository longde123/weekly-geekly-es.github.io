2017年家用十大CNC台式机



台式机3D打印机已经在新技术爱好者和工匠中广泛使用，他们将它们用于工作和家庭。 他们并不总是能达到期望并投入资金。 该视频包含另一类设备的演示-这些是多功能设备，不仅可以批量打印，还可以执行其他操作。 它们都相对便宜，并且占用空间很小，因此您可以在计算机旁边的桌面上使用它们。



说明：


Snapmaker




  • 主轴转速：2000-7000 rpm
  • 工作区尺寸：125x125x125 mm
  • 技术：FDM
  • 层厚度从：50微米
  • 制造商的价格：499美元起

Snapmaker是一款采用模块化设计的紧凑型机器 ，只需几分钟即可从各个模块组装而成。 Snap Maker配备有可互换的工作模块，可让您进行铣削，3D打印，刻录和激光切割。



卡扣机由钢和阳极氧化铝制成，应保证结构的可靠性和耐用性。


在我们的博客上阅读有关 Snapmaker的更详细的评论 。


XPLOTTER




  • 工作区尺寸：210x210x40 mm
  • 机械手
  • 绘图仪
  • 雷射
  • 制造商价格：$ 529

Xplotter是一个预制的家用绘图仪。 他可以使用各种工具进行专业绘图，从任何钢笔到毡尖笔。


它可以用激光燃烧和切割，也可以用吸盘把持小物体。



工具




  • 电源：100-240 V
  • 通过适配器：12V，60W
  • 连接方式：USB
  • 工作速度：20m / min
  • 工作区大小不同
  • X / Y分辨率：0.05毫米（0.002英寸）
  • 生产者价格：$ 1295起

Tooli是另一种多功能工具。 它还可以执行绘图仪，操纵器和激光切割机的功能。 此外，Tuli能够喷枪并分配糊状物质，食物和非食物-因此，使用它可以轻松制作蛋糕装饰或彩色玻璃。


Tooli的有趣之处在于其设计易于扩展-快速简便地更换导轨可让您增加工作面积。


该设备配备了自己的软件-Art 2G代码，具有最直观的界面，并且可以将模型放在一个通用的库中，然后从那里获取，这要由发达的用户社区来推动。

多博特M1



  • 激光切割和雕刻
  • 机械手
  • 3D打印
  • 锡焊
  • 价格：180000卢布

Dobot M1是具有可更换喷嘴的机械臂。 它是微型的 ，但它定位为具有3D打印，激光雕刻，焊接和组装的计算机视觉和可互换头的专业工具。

Dobot机械手精度：0.02毫米，工作区域：半径400毫米，高度250。


擎天柱




  • 铣刀
  • 激光雕刻机/切割机
  • 3D打印机类型deltabot
  • 打印机工作区尺寸：直径240毫米，高度300
  • 激光工作区：500x500 mm
  • 激光功率：0.5-2-5-10 W
  • 铣削工作区：400x400x480毫米
  • 主轴功率：450 W
  • 生产者价格：从$ 1799

Optimus是Febtop Tech变压器机器，根据当前目标，可以轻松地将其从bot 3D打印机转换为线性机器。 该机器可以进行3D打印，使用2 W激光进行燃烧和切割，以及铣削木材和塑料。 与自己的应用程序一起使用。 包括：Optimus Delta 3D打印机，铣削模块，激光，转换套件。

该机器非常耐用。



X雕刻




  • 作业面积，带边的正方形：500、750、1000毫米
  • 主轴功率：110，240 W
  • 随附软件：VCarve或Aspire
  • 制造商的价格，取决于配置：从999美元到3900美元

Inventables X-Carve是面向业余爱好者的自制CNC机床 。 用户社区专用于他，其成员撰写有关使用机器的文章并提供改进。 一些用户从事销售成品的业务。 这在学校中开始广泛传播-传统的自制蛋糕销售不再流行， 学生雕刻徽章和纪念牌以供学校销售。



许多人会做吉他，配件或直升机等事情。 所有创建的模型都将添加到Inventables集合中 ，可以随时找到它们。



每周都有针对该机器的免费软件更新。 该机器与通用程序不兼容，但是完整的软件可以轻松创建模型并将其发送给朋友和合作伙伴。 X-carve与上一代Inventables设备完全兼容-有升级的可能性。

  • 精密机械，关闭 不再：0,00015％
  • 工作区：203.2 x 282.56 x 260.35 mm

  • 喷嘴：0.4mm

食人鱼FX




  • 工作区域：304 x 330 x 76毫米
  • 制造商的价格：从$ 2500
