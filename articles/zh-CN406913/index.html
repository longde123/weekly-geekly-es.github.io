设计与人机工程学


智能手机带有一个紧凑的红色和白色框。 内部，一切都很适中：5V / 1A电源，USB电缆，用于取下托盘的夹子。





从外观上讲，智能手机可以部分称为不露面，其外观与预算定位完全一致。 背面的四角和条纹准确四舍五入，可以与您知道哪部手机紧密相关。



小米Redmi Note 5A外壳假装为金属，但实际上它仍然是塑料，制造商在其网站上随意提及。

智能手机显示屏周围的框架很小，单手使用时该设备便于拦截。

这些按钮位于右侧，一目了然，可以一目了然。 左侧是空的，只有一个内置托盘，可容纳一对SIM卡和存储卡。 最后，我们有三个完整的插槽，不再需要在第二张SIM卡和内存扩展之间进行选择。 此刻立即使为32 GB内存的小米Redmi Note 5A版本付出了过多的代价。 增大卡的容量会更明智。





在上边缘，除了第二个麦克风和3.5毫米插孔外，还发现了一个红外端口，考虑到设备的预算，该端口特别好。 他帮助我在家中控制几台电视和办公室空调。





在两个格栅下方的下边缘，隐藏了对话式麦克风和多媒体扬声器。 后者非常响亮，在高音量下不会喘息。 但是我手中的麦克风确实让我们失望了。 对话者抱怨您听得很糟，我的声音就像水桶一样。 小米对麦克风贪婪，或者我碰到一个不成功的设备。

我的版本中没有指纹扫描仪。 是否付款是个好问题。 如果您增加购买智能手机的预算，我会希望拥有功能更强大的成熟产品线-没有“ A”指数。



显示屏下方的按钮没有背光，但是经过一周的适应后，它们却变成了盲目而没有问题。 智能手机有三种颜色可供选择：灰色，金色和粉红色。

萤幕


尽管小米Redmi Note 5A仅获得了高清分辨率的显示屏，但我仍然感觉它占了设备成本的大部分。 5.5英寸IPS矩阵具有宽视角，可自定义的显色性以及主观上良好的亮度性能。 没有颗粒效果或模糊图片。



在白天，屏幕不会在阳光下褪色，而在晚上，传感器会智能地将亮度降低到令眼睛舒适的值。 显示屏上覆盖了一层具有良好疏油涂层的防护玻璃。 玻璃上几乎没有指纹。

摄影机


小米Redmi Note 5A智能手机配备了13MP主摄像头以及5MP前置自拍模块。

智能手机相机的分辨率中的数字只是带有少量营销的数字。 图片的真实质量有很多不足之处。 智能手机可以在下午在街上拍照。 自动对焦不是最快的，因此设备在拍摄过程中不会原谅手摇。

铁和软件


使用Qualcomm Snapdragon 425，您在玩具方面走得太远了，综合测试结果清楚地证明了这一点。



基本版本的智能手机具有性能限制和非常少量的RAM-仅2 GB。 我说“一切”，是因为如果在裸露的Android（Nexus 5、5X）或Motorola（Droid MAXX）上就足够了，那么在小米设备中，MIUI外壳会非常活跃地吞噬RAM，并且不会放任不管。

也就是说，应用程序运行速度很快，滚动和导航通常都很流畅。 但是，如果您仅切换到另一个程序，则旧的程序会立即在后台关闭，而下次它将再次重新启动。

周期性地，由于缺少RAM，读取大量材料时Chrome或Pocket崩溃。 在内存中阻止应用程序或在MIUI设置中禁用积极的节能措施都不能节省情况。

小米Redmi注意事项5A对于单任务处理和现场操作非常有力，请不要尝试将其挂起为多线程计算机。 使用廉价芯片会带来明显的好处-即使经过长时间的游戏或连续一个小时的观看，该设备实际上也不会发热。

智能手机带有中文MIUI 8固件。 您可以解锁引导加载程序并安装多语言非官方固件。 您可以像我一样，打开英语并将其滚动到Google Play智能手机上，从预先准备的备份中还原它。 视频中的说明和文件帮助我应对。

之后，小米Redmi Note 5A可以轻松使用了，除了在右侧的下拉菜单和几个常用应用程序中，仍然可以找到象形文字。



其他界面截图


我没有发现固件的任何功能问题，界面的响应能力和平滑度对于预算平板手机来说还算合适。

自治权


相对低分辨率的显示屏和低功耗处理器的另一个明显好处是，智能手机可以长时间依靠单电池充电。

在PC Mark测试中（模拟智能手机的不同使用场景），小米Redmi Note 5A在平均屏幕亮度下持续了超过8个小时。 这确实是1.5到2天的有效使用时间。



不可拆卸的3080 mAh电池非常坚固。 没错，我必须付一点钱。 从完整的电源设备到5V-1A，智能手机的充电时间超过三个小时。 我认为购买功能更强大的部门将有助于大大加快这一过程。

结论


小米原来有一个很好的预算平板手机，通常使用起来很愉快。 显然，从填充的角度来看，该设备是一个死胡同，并且将在一年内失去相关性。 但是，如果您需要一款价格高达$ 120的大屏幕智能手机，可以安全地考虑购买Redmi Note 5A。
