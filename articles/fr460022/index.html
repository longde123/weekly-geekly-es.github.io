    • Dépréciations pour PHP 7.4
    • Escape PDO "?" espace réservé de paramètre
    • Déprécier la syntaxe des accolades pour accéder aux éléments du tableau et aux décalages de chaîne
    • Ajoutez str_starts_with (), str_ends_with () et les fonctions associées .

    Aussi quelques nouveaux changements sans RFC séparés:
    • proc_open acceptera les arguments sous la forme d'un tableau, ils n'ont pas besoin d'être échappés et le processus démarrera directement sans shell intermédiaire: 
     proc_open(['php', '-r', 'echo "Hello World\n";'], $descriptors, $pipes);

    • En PHP 7.4, un avertissement sera émis lorsque vous tenterez d'utiliser incorrectement des types scalaires comme tableaux. Auparavant, ces appels renvoyaient simplement null : 
     $null = null; var_dump($null[0]); // NULL $int = 42; var_dump($int['foo']); // NULL var_dump(true[0] === false[0]); // true

Les outils


  • humbug / php-scoper - L'outil vous permet d'ajouter un préfixe à tous les espaces de noms dans le dossier spécifié en une seule étape. Cela peut être utile lorsque vous devez utiliser plusieurs versions d'un même package, un exemple d'utilisation pour les plugins WordPress .
  • CaptainHookPhp / captainhook - Gestionnaire de hook Git pour les projets PHP.
  • sebastianfeldmann / phpbu - Un outil pour créer des copies de sauvegarde des fichiers et des bases de données. Capable de crypter les sauvegardes et d'envoyer au stockage.
  • localheinz / http-method - Un ensemble de constantes avec les noms des méthodes HTTP.
  • spiral / roadrunner - Serveur d'applications pour PHP à Golang. Ajout de la prise en charge de FastCGI, des paramètres CORS prêts à l'emploi, des métriques intégrées sur Prometheus. Un examen détaillé dans le rapport de l'auteur .
  • leocarmo / circuit-breaker-php - Implémentation du modèle de disjoncteur pour PHP. Alternative: ackintosh / ganesha .

La sécurité


  • nbs-system / snuffleupagus - L'extension bloque l'exécution de code potentiellement dangereux lors de l'exécution. À propos de l'utilisation pour empêcher les vulnérabilités récemment publiées.
  • paragonie / argon2-refiner - L'outil vous aidera à choisir les paramètres optimaux pour le hachage en utilisant l'algorithme Argon2id ajouté en PHP 7.3 .
  • Nous utilisons les nuances de l'analyseur de requêtes en PHP pour contourner les systèmes afin de prévenir / détecter les intrusions dans le réseau.
