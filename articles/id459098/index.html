Registry Paket GitHub akan mendukung paket Swift

Pada 10 Mei, kami meluncurkan uji beta terbatas dari GitHub Package Registry, layanan manajemen paket yang memudahkan untuk mempublikasikan paket publik atau pribadi di sebelah kode sumber. Layanan saat ini mendukung alat manajemen paket yang sudah dikenal: JavaScript (npm), Java (Maven), Ruby (RubyGems), .NET (NuGet), gambar Docker, dan banyak lagi.

Kami dengan senang hati mengumumkan bahwa kami akan menambahkan dukungan paket Swift ke GitHub Package Registry. Paket Swift memudahkan Anda untuk berbagi perpustakaan dan kode sumber di proyek Anda sendiri dan dengan komunitas Swift. Kami akan mengerjakan ini dalam kemitraan dengan orang-orang dari Apple.



Artikel Blog GitHub ini

Tersedia di GitHub, Swift Package Manager adalah alat lintas-platform tunggal untuk membangun, menjalankan, menguji, dan mengemas kode Swift. Konfigurasi ditulis dalam Swift, membuatnya mudah untuk menetapkan tujuan, mengiklankan produk, dan mengelola dependensi paket. Bersama-sama, Swift Package Manager dan GitHub Package Registry memudahkan Anda untuk menerbitkan dan mengelola paket Swift Anda.

Untuk pengembang seluler, penting untuk memiliki alat yang lebih baik agar lebih produktif. Dengan pengembangan ekosistem Swift, kami senang bekerja dengan tim Apple untuk membantu menciptakan alur kerja baru bagi pengembang Swift.

Sejak peluncuran GitHub Package Registry, kami telah melihat keterlibatan masyarakat yang kuat dalam bekerja dengan alat ini. Selama periode beta, kami berusaha untuk belajar dari komunitas bagaimana Registry Paket memenuhi berbagai kebutuhan dan apa yang dapat kami lakukan untuk membuatnya lebih baik. Jika Anda belum mencoba GitHub Package Registry, Anda dapat mengajukan permohonan beta di sini .
