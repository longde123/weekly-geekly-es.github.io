Rapikan SMB Anda atau temukan HPE ProLiant DL180 Gen10 yang diperbarui dan ditingkatkan

Mengembangkan garis misi server Mission Critical , Hewlett Packard Enterprise tidak melupakan kebutuhan bisnis kecil dan menengah.

Seringkali, meskipun tidak selalu, proses menemukan daya komputasi untuk tugas-tugas baru sulit untuk diprediksi: persyaratan tumbuh, tugas mendesak baru muncul secara spontan, semua ini disertai dengan upaya untuk memahami arsitektur yang dihasilkan, dan masalah pembelian kapasitas baru berasal dari kategori membeli Rolls-Royce baru. Tapi apakah semuanya begitu menakutkan?


Server seseorang, mungkin akhir-akhir ini.

Mari kita pikirkan: server mana yang menunggu pelanggan SMB kami dan bisakah itu tersedia?

Bisnis kecil dengan kebutuhan besar


Kami dan pelanggan kami melihat peningkatan yang stabil dalam permintaan sumber daya komputasi, sementara SMB ditandai oleh "musiman", kebutuhan akan peningkatan kapasitas bersifat episodik. Dari iklan bank terkenal, kita ingat bahwa semakin cepat kita memperkenalkan "trik", semakin cepat kita akan mendapatkan pelanggan baru, itulah sebabnya usaha kecil suka bereksperimen dengan perangkat lunak dan tidak menerima pendekatan tradisional untuk berurusan dengan server - kita membutuhkan kotak "tanpa dasar". Server pelanggan seperti itu, dalam gambaran ideal dunia, adalah traktor yang berfungsi secara stabil dan yang dapat diperbaiki dengan palu godam dan obeng.
Mengapa selalu begitu sulit, tetapi menarik dengan proyek-proyek SMB? Karena kriteria untuk memilih peralatan oleh pelanggan, bukan Kecil dan Menengah: Anda perlu harga rendah, dan kemampuan server untuk mencerna tugas-tugas baru dan peningkatan kebutuhan bisnis, dan keandalan pada tingkat model "lama", ditambah kemungkinan administrasi lokal dan jarak jauh. Cherry on the cake mungkin merupakan persyaratan bagi pabrikan - kemampuan untuk membuat perkiraan sumber daya yang dikonsumsi pada peralatan saat ini dan memiliki tren seperti itu.
Kami selalu mencoba kombinasi berbeda dengan pelanggan dan mitra, tetapi sebelum memulai kembali penjualan HPE DL180 Gen10, pendahulunya cocok untuk permintaan tersebut. Itu sebabnya dikembalikan ke layanan sebagai server generasi ke-10. Mari kita lihat apa yang telah berubah.

HPE DL180 Gen10 lahir untuk SMB


Mari kita lihat server HPE ProLiant DL180 Gen10 generasi kesepuluh.
Seperti yang Anda ketahui, portofolio server HPE, bersama dengan model 2-prosesor klasik untuk seri pusat data DL300, dengan desain fleksibel dan opsi ekspansi maksimum, telah lama termasuk seri DL100 yang lebih terjangkau. Dan jika Anda mengingat artikel kami tentang Habré, yang didedikasikan untuk pengumuman generasi HPE ProLiant Gen10 , seri ini direncanakan untuk diluncurkan pada musim gugur 2017. Tetapi karena optimalisasi lini produk server, peluncuran seri ini di pasar pada tahun 2017 ditunda. Tahun ini, diputuskan untuk kembali ke pasar beberapa model seri DL100, termasuk server HPE ProLiant DL180 Gen10.


Fig. 2 Panel Depan HPE ProLiant DL180 Gen10

Tujuan utama dari server adalah untuk menutup kebutuhan perusahaan kecil dan menengah. Mereka memiliki semua yang diperlukan dan, pada saat yang sama, label harga dapat diterima yang dapat diterima oleh pelanggan kami.
Mudah beradaptasi dengan berbagai beban kerja dan lingkungan, server server HPE ProLiant DL180 2-rack-secure yang aman memberikan kinerja tinggi dengan keseimbangan skalabilitas dan skalabilitas yang tepat. Pendekatan ini telah dipertahankan dalam model baru: sekarang ini adalah server dengan semua "barang" Gen10, yang dirancang untuk keserbagunaan maksimum dan toleransi kesalahan, dengan keseimbangan keandalan, pengelolaan, dan kinerja yang tepat.

Apa yang ada di bawah tenda?


Dalam kasus 2U, dimungkinkan untuk menginstal dua prosesor Intel Xeon Bronze 3106 atau Intel Xeon Silver 4110 - ini adalah kompromi yang sama ketika kami mengorbankan variabilitas untuk mempertahankan harga server yang rendah. Tetapi karakteristik prosesor ini cukup untuk semua tugas khas usaha kecil dan menengah. Di panel depan ada delapan SFF disk faktor bentuk kecil dengan kemampuan hot swap.
Server itu sendiri berisi 16 modul RAM RDDRM DDR4-2666 dengan koreksi kesalahan, serta hingga enam adapter ekspansi tambahan dengan antarmuka PCIe Gen3.
Jumlah slot PCIe adalah sakit kepala bagi pelanggan SMB, karena sering kali perlu menginstal kartu khusus untuk perangkat lunak, kartu ekspansi, dan menghubungkan berbagai antarmuka. Sekarang tidak perlu menginstal server tambahan, bahkan membeli konfigurasi server awal.


Fitur khas HPE ProLiant DL180 Gen10 - sejumlah besar slot ekspansi

Untuk meningkatkan toleransi kesalahan server, ia, seperti dalam model server yang lebih lama, menggunakan redundansi kipas (N + 1), dan juga dimungkinkan untuk menginstal pengontrol disk tambahan yang mendukung tingkat RAID perangkat keras 0, 1, 5, dan 10. Ada juga kemungkinan memasang catu daya. dengan redundansi dan "hot swap".


Fig. 4 Sasis HPE ProLiant DL180 Gen10, tampilan atas

Fitur khas dari server HPE DL180 Gen10 adalah kemampuan untuk menginstal sejumlah besar disk, dari berbagai jenis, baik SAS dan SATA, tetapi, tidak seperti model server lama, tidak ada cara untuk menghubungkan media format NVMe yang baru.
Drive meninggalkan yang paling populer: 300GB, 600GB, 1.2TB dalam bentuk mekanik dan 400GB, 480GB dalam bentuk SSD. Segera kami mengharapkan pengumuman keranjang LFF.


Fig. 4 Keranjang Cakram HPE ProLiant DL180 Gen10
Meskipun HPE DL180 Gen10 menargetkan segmen server rak yang terjangkau, HPE tidak berkompromi dalam hal pengelolaan dan keamanan. Server sudah dilengkapi dengan pengendali jarak jauh HPE iLO 5 yang sama dengan seri lama, dan, yang penting bagi banyak pelanggan, port RJ-45 khusus untuk menghubungkan iLO ke jaringan Ethernet pada kecepatan 1 Gbit / s segera disolder. Anda dapat membaca lebih lanjut tentang kemampuan kontroler ini, yang memberikan perlindungan server terbaik di industri, dalam artikel yang telah disebutkan di atas dengan pengumuman generasi HPE ProLiant Gen10 .

Analitik prediktif tipe baru untuk bisnis kecil dan menengah.


Seperti semua server generasi Gen10 lainnya, model ini memungkinkan pembaruan driver dan mikrokode baik offline maupun online menggunakan perangkat lunak HPE SPP dan HPE SUM (Smart Update Manager), dan juga didukung oleh platform manajemen Paket Amplifier HPE iLO.

Ingat bahwa Paket Amplifier HPE iLO (Integrated Lights-Out) adalah inventaris skala besar dan alat manajemen pembaruan yang memungkinkan pemilik infrastruktur server Hewlett Packard Enterprise Gen8, Gen9, dan Gen10 yang besar untuk secara cepat menginventarisir dan memperbarui firmware dan driver. Alat ini juga membantu pemulihan sistem secara manual dan otomatis dengan firmware yang rusak.


Fig. 5 HPE InfoSight. Kecerdasan buatan untuk infrastruktur platform.

Ini memberi pelanggan kami kemampuan untuk melakukan analisis prediktif seluruh infrastruktur server dengan platform InfoEight HPE untuk mengoptimalkan kinerja serta secara proaktif mengidentifikasi dan mencegah masalah. InfoSight HP untuk Server hampir dapat sepenuhnya menyelesaikan masalah dan mengurangi biaya waktu dengan mengubah proses manajemen dan mendukung infrastruktur. InfoSight untuk Server HPE menganalisis data telemetri yang diterima oleh "perekam penerbangan" HPE AHS di semua server untuk memberikan tip pemecahan masalah dan peningkatan kinerja. Jika masalah terdeteksi pada satu server, HPE InfoSight for Servers belajar untuk memprediksi masalah dan merekomendasikan solusi untuk semua server yang diinstal.

Dukungan Perusahaan


Untuk memenuhi keinginan pelanggan, perusahaan juga meningkatkan kondisi garansi untuk model ini dibandingkan dengan model HPE DL180 Gen9 generasi sebelumnya: jika pada generasi sebelumnya garansi standar mencakup pekerjaan seorang insinyur layanan dan perbaikan server di situs pelanggan (tergantung pada kondisi tertentu) hanya selama tahun pertama setelah membeli server (selain garansi standar 3-tahun untuk suku cadang), kemudian dalam model HPE DL180 Gen10, garansi 3 tahun sudah termasuk dalam pengiriman dasar server (3/3/3 - tiga tahun untuk setiap set tion, operasi dan pemeliharaan di tempat). Pada saat yang sama, desain server sedemikian rupa sehingga sebagian besar dalam hal kerusakan digantikan oleh pengguna, dan hanya sebagian kecil dari pekerjaan penggantian yang memerlukan partisipasi dari teknisi servis HPE.

Jika kita membandingkan model ini dengan "kakaknya" dalam bentuk HPE DL380 Gen10, maka kita dapat mencatat poin-poin kunci berikut:

- HPE DL380 Gen10 mendukung hampir seluruh jajaran prosesor dari keluarga Intel Xeon Scalable terhadap hanya dua model dalam HPE DL180 Gen10;
- kemampuan untuk menginstal 24 modul memori di seri 300 versus 16 di 100;
- pada seri ke-100, kemampuan untuk menginstal keranjang disk tambahan tidak disediakan;
- seri ke-100 menawarkan serangkaian opsi yang jauh lebih kecil (pengontrol, disk, modul memori);
- Pada seri ke-100, pemasangan drive mendapatkan popularitas dengan antarmuka NVMe tidak didukung.

Terlepas dari keterbatasan ini, server HPE ProLiant DL180 Gen10 adalah salah satu produk terbaik di pasar untuk solusi server untuk usaha kecil, serta untuk perusahaan besar yang membutuhkan kuda pekerja yang terjangkau untuk kebutuhan yang berkembang dari pusat data, yang memiliki fitur keamanan terbaik di industri. dan garansi yang telah teruji waktu dan dukungan layanan dari pemimpin dunia.
