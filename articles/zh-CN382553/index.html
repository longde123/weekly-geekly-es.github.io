NASA谷神星视频之旅



根据Dawn探测器发送的照片，NASA专家创建了谷神星的“视频之旅”。该视频显示了小行星最引人注目的位置-例如Occator陨石坑，其底部有同样的白色斑点。让我提醒您，不久前，科学家在其中一张照片中发现了火山口上方的薄雾，这可能表明这些斑点是冰向表面的出口，而不是矿物盐。

此外，在谷神星上发现了一座金字塔形的高山。专家说，目前，这是在谷神星上发现的所有高山中的最高峰。但是，目前尚不完全清楚这样的山丘是如何在小行星上出现的。

为了编辑视频，我们使用了地面图像和谷神星表面的地形图，这些图像已于上周传输到地球。基于所有这些数据，专家们能够创建三维模型的谷神星。

同样有趣的是，白色斑点的反照率略低于同一水冰的反照率，因此无法认为物体起源的问题得到了彻底解决。尽管如此，所有这些问题都可以在本月中旬变得清楚，届时该站将与谷神星表面相距1.5千公里。



黎明于今年3月6日到达谷神星，并立即开始研究天体。探头在谷神星的总停留时间约为16个月。同时，在接近小行星的过程中，航天器出现问题-探针停止响应控制中心的命令。此后，Dawn计算机立即进入安全模式，并在一段时间后再次开始工作。

最近，天文学家还发布了谷神星地图以及另一段视频：单击该图像时，该图像将以完整尺寸打开（NASA / JPL-Caltech / UCLA / MPS / DLR / IDA）






除了上面提到的Occator陨石坑外，还有更深的陨石坑，例如Dantu（直径120公里，深度5深度）和Ezinu（类似大小）。较大的物体是直径160公里，深度6公里的Urvara火山口。此外，在火山口的中部有一个高3公里的山峰。

谷神星本身的直径为940公里，这是位于火星和木星之间的小行星带中最大的物体。实际上，谷神星的大小约为冥王星大小的40％。
