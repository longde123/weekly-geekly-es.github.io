3rd Turing Test: syarat untuk berpartisipasi dan hadiah untuk pemenang

Sebulan yang lalu, kami berbicara tentang bagaimana kami akan melakukan Tes Turing berikutnya dalam bahasa Rusia di konferensi kecerdasan buatan Opentalks.AI pada 14-16 Februari. Anda masih dapat mendaftar untuk kompetisi hari ini, 12 Februari, hingga 23:59 waktu Moskow. Kami mengundang semua orang untuk ikut serta dalam Tes!



Di bawah cut - condition untuk para peserta dan deskripsi hadiah untuk para pemenang.

Ketentuan


Partisipasi dalam Tes untuk tim dengan bot obrolan gratis. Pendaftaran untuk kompetisi berarti Anda menyetujui Perjanjian Penawaran publik, yang tersedia di situs web kompetisi, tempat kami juga mengunggah API. Pengujian awal bot akan diadakan mulai 12 hingga 13 Februari. Dan sudah selama konferensi, para ahli dan semua orang akan mengujinya. Pada 16 Februari, panitia akan mengumumkan pemenang yang akan diberikan hadiah uang tunai.

Tentang API


Peserta mendaftar di akun Anda dan mulai bot obrolan baru. Untuk menyederhanakan implementasi, API dibuat sinkron untuk para peserta. Server mengakses chatbot melalui HTTP dan menunggu respons. Protokol komunikasi bot obrolan adalah JSON kecil: 

{ "chat_id": "uuid", "request_id": "uuid", "text": "text" }

chat_id dan request_id masing-masing adalah UUID unik dari obrolan dan permintaan. teks adalah pesan dari pengguna.

Respons sederhana dengan teks diharapkan dari bot: 

 { "text": "text" }

Ada bot Telegram terpisah di mana Anda dapat men-debug momen teknis komunikasi dengan bot obrolan. Dalam bot Telegram kompetitif dari sisi server akan ada penundaan dalam respons sehingga tidak mungkin untuk menentukan pada saat jawaban apakah ini bot atau seseorang.

Hadiah


Berkat dukungan MTS, pemenang akan menerima hadiah.

Bot obrolan terbaik akan menjadi yang paling keliru dikira orang.

Hadiah:

  • Posisi Pertama - 200.000 rubel.
  • Posisi Kedua - 150.000 rubel.
  • Posisi Ketiga - 100.000 rubel.

Hadiah untuk relawan yang memainkan peran bot lebih baik daripada yang lain:

  • Posisi Pertama - 20.000 rubel.
  • Posisi Kedua - 15.000 rubel.
  • Posisi Ketiga - 10.000 rubel.

Hadiah untuk para ahli dengan hasil tebakan terbaik:

  • Posisi Pertama - 20.000 rubel.
  • Posisi Kedua - 15.000 rubel.
  • Posisi Ketiga - 10.000 rubel.

Selain para ahli, siapa pun dapat mengobrol dengan bot obrolan melalui bot Telegram @TuringTestRussiaBot.

Pada akhir hari terakhir konferensi, tidak hanya pemenang akan diumumkan, tetapi juga hasil umum kompetisi, dan dialog terbaik dan paling menarik akan diperagakan, yang akan dipilih oleh tim moderator terlebih dahulu.

Kami menunggu semua orang di Opentalks.AI!
