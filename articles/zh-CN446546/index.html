Microsoft通过Azure IoT创新者和初创公司的新IP优势扩展了Azure IP优势



今天，我们很高兴宣布Microsoft Azure IP 优势计划的扩展，其中包括Azure IoT创新者和初创公司的新优势。 我们于2017年2月首次宣布了Azure IP Advantage。 它旨在为我们的云客户提供针对知识产权（IP）风险的全面保护。 我们当时以及现在所看到的趋势是，由于专利申请的原因，云技术创新的风险在不断增长 。 去年，我们加入了开放发明网络（OIN）和转让许可（LOT）网络，以帮助消除客户和合作伙伴的专利被盗风险。

感谢您翻译我们的MSP Lev Bulanov

但是我们相信我们可以做得更多。 2013年至2018年期间，美国的IoT专利数量增加了400％以上 。 因此，在与客户讨论并考虑如何进一步提高Azure IP Advantage计划的价值之后，我们决定通过专注于Azure物联网（IoT）和初创企业的新优势来扩展该计划，以帮助Azure客户避免法律费用。 。 今天宣布的新功能包括：

  • Microsoft Azure Sphere和Windows IoT的无限补偿范围。 赔偿有助于保护客户免受因使用产品或服务而对客户提出的侵犯知识产权的索赔。 今天的扩展为Azure Sphere和Windows IoT提供了无限的补偿，包括Microsoft已将其集成到这些产品中的开源软件。
  • 使用Azure for IoT设备的客户访问10,000项专利，以保护自己免受IP诉讼的侵害。 此好处可以帮助防止针对Azure客户的工作负载和在Azure或IoT设备上运行的应用程序针对Azure客户提起专利诉讼，因为合格的客户可以选择专利用于诉讼辩护。
  • Azure上的初创企业有机会获得Microsoft专利，以帮助其发展业务。 也加入LOT Network的熟练创业公司可以通过LOT获得技术，包括AI，多媒体和安全性方面的Microsoft专利。 专利对于初创企业的成长起着重要的作用，它可以找到保护其创新的方法并吸引关键的资本支持。

许多初创公司向我们介绍了他们在小型公司中面临的特殊漏洞，以及他们在寻求筹集资金和建立未来的早期阶段对建立自己的IP产品组合的重视。 我们在LOT Network的合作伙伴已开发出可以满足这些需求的程序。 微软很高兴向LOT网络捐赠了500项专利，LOT网络现在将免费提供给合格的初创公司 。

综合考虑所有这些因素，我们相信Microsoft提供了最全面的环境来消除云中的IP风险。

最初的客户评论是正面的：

“对于像蒂森克虏伯这样的国际工程公司来说，Microsoft Azure IP优势计划是一个很好的工具，可以减轻向工业4.0迁移时的IP风险，”知识产权（IP）负责人Stephan Wolke博士说。和蒂森克虏伯有限公司的服务。

“应用大脑研究正在构建受大脑启发的AI系统，”在不断增长的神经形态计算市场中一家初创公司董事长兼联合首席执行官Peter Suma说。 “ Azure IP优势可以帮助我们专注于我们的下一个创新，并减少花费在担心IP风险上的时间。”

我们的目标是创建一个专注于创新的社区，而不用担心保护其创新。 我们希望此步骤扩展Azure IP Advantage计划在此方面将是有用的。
