3D打印机Raise3D N2 Dual

俄罗斯的3D打印机市场已经很成熟-有流行的型号，有特定品牌迷的“独家产品”。 现在有很多类似设备的制造商。 每个月都会出现几种新型号。 但是，只有市场领导者的新产品才能吸引真正的注意力。 其余的，如果它们没有任何明显的优势，则保留在阴影中。 不论价格和外观。

2016年底，俄罗斯市场上的新产品之一是Raise3D N2 Dual 3D打印机。 这对我们国家来说是新事物，他的出现引起了人们的兴趣。 打印机外观醒目，将其与其他类似技术区分开。 外观肯定不错，但是与其他类似机器相比，流行起来有什么优势吗？



制造商向我们提供什么？
首先，较大的打印区域引起了人们的注意：305 x 305 x 305毫米。 这比增材制造的大多数其他代表所拥有的标准200 x 200 x 200 mm区域要大。

从“ Dual”名称中可以看出，Raise3D N2 Dual模型具有两个挤出机，这使得可以打印由可溶材料（例如PVA）制成的支撑物的复杂模型。 这种组合-大的打印区域和两个挤出机，并不常见。

规格：


  • 用于3D打印的材料：PLA，ABS，Filaflex，PVA，PC，臀部，尼龙，橡胶，Flex（丝直径1.75毫米）。
  • 印刷面积：305 x 305 x 305毫米;
  • 沿Z轴的层高度：10到200微米（0.01-0.2毫米）
  • 喷嘴直径：0.4毫米；
  • 封闭式相机的3D打印功能：是；
  • 加热桌：是的，最高110;
  • 挤出机温度：最高300；
  • 打印头数量：2；
  • 触摸屏控制的可用性：是

原始设计和施工解决方案。 我们没有在其他打印机上看到过类似的东西。

同时，打印机具有由铝型材制成的刚性框架，这对打印质量有积极影响。

工艺和组装是最重要的。 可以在下面的照片中看到。

让我们更详细地介绍3D打印机的力学特性。

外壳采用全封闭设计。 这使您可以为高质量的印刷和收缩率高的塑料创造一个有利的温度。 例如，对于高质量印刷而言，最流行的材料是ABS，密闭盒是加热工作台之后最重要的条件之一。

打印机中也有加热台。 近来，许多制造商倾向于拒绝加热。 但这会从可用材料列表中删除流行的ABS，并在其他材料上造成打印问题。 Raise 3D的创建者决定不将其客户限制于任何类型的框架。 因此，打印机可以使用任何可用的打印介质。



工作表面上的玻璃是可移动的，这使工作非常舒适。 桌子本身可以加热到110℃。 这足以用于3D打印机中的所有材料。 将工作台加热到工作温度只需很少的时间。
从字面上看5-7分钟。 与许多其他打印机的10-15分钟相比，这似乎是一个主要优势。

表格校准尽可能自动化。 3D打印的绝对初学者将可以应对。 不需要其他知识。 打印机是为广泛的用户设计的。 可以看出，制造商遵循了“开箱即用的3D打印”的原则。



沿着3D打印机的内边缘，安装了工作区的LED照明。 这使您无需打开工作室就可以监视和控制打印过程。

挤出机设计


与Ultimaker不同，Raise3D的打印机使用具有两个喷嘴的原始设计的直接挤出机。



为了保持打印区域的大小，通常不将挤出机驱动器紧挨着放置。 在此3D打印机中，电动机垂直放置在另一个之上。 这样就可以创建一个非常紧凑的设计。 与其他3D打印机不同，Raise3D 3D打印机的两个挤出机和一个挤出机的修改具有相同的打印区域。 另外，我想指出直接挤出机的一个重要优点-这是使用柔性材料进行无故障打印的可能性。
