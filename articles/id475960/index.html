Pendahuluan


AHURATUS Smart Home Voice Assistant adalah perangkat IOT yang dikembangkan untuk mengontrol perangkat rumah lainnya dengan deteksi suara. Catatan: Perangkat ini HANYA dibuat untuk tujuan akademik.


Pendekatan


Deskripsi


"AHURATUS Smart Home Voice Assistant" menggunakan proses ARM Cortex-M3 untuk menjalankan instruksi serta beberapa perangkat periferal untuk mengurangi kompleksitas bus data dan perhitungan RF-Circuit.


Bill of material


# Nama komponen Peran Tautan Dokumen Teknis
1 STM32F103ZET6 Proses dan kontrol Lembar Data
2 Modul Bluetooth HC-05 Koneksi radio Bluetooth Lembar Data
3 Adaptor AC-DC 220-5V Memberi daya pada sirkuit Lembar Data
4 LED atau MOSFET atau Relay Untuk output sistem Lembar Data

Meja kerja dan perangkat lunak


# Nama Perangkat Lunak Peran Tautan dan refrensi
1 STM32CubeIDE IDE untuk Pengodean instruksi kami Tautan
2 GNU-ARM C-Compiler untuk prosesor ARM Tautan
3 Jlink Programmer dan debugger Tautan
4 HERCULES Perangkat Lunak Pemantauan SerialPort Tautan
5 Kontrol Suara Arduino Aplikasi Android untuk mengirim data ke perangkat kami Tautan


Detail


Kami menggunakan driver HAL yang disediakan oleh pabrik prosesor untuk mengurangi kompleksitas pengecekan kesalahan dan penggunaan periferal yang lurus ke depan. Kami menggunakan HC-05 untuk menerima data dari pengguna kami melalui koneksi nirkabel dengan mengatur pengaturan modul melalui perintah AT berikut :


Mengirimkan: AT + UART = 115200, 0, 0
Tanggapan: Oke
_ Catatan: Untuk menerapkan Perintah AT Anda harus menghubungkan pin KUNCI ke tingkat TINGGI.

Kesimpulan


Proyek "AHURATUS Smart Home Voice Assistant" dapat digunakan untuk memiliki perangkat rumah pintar fungsional yang sederhana dan murah dengan beberapa modifikasi sederhana di sirkuit; seperti:


  • Menggunakan MOSFET dan Sirkuit Isolasi AC untuk mengontrol perangkat dengan voltase kerja AC
  • Menggunakan Pulse Width Modulation (PWM) agar dapat lebih mengontrol output
  • Gunakan TRIAC untuk memiliki setter level lampu yang dapat disesuaikan
  • Rancang aplikasi yang lebih baik untuk berinteraksi lebih baik dengan pengguna
  • menggunakan Kinect atau Perangkat Deteksi Gerakan lainnya untuk mengukur membuat perangkat tersebut kompatibel dengan gerakan pengguna.
