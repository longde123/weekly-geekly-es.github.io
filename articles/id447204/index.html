17 April: Kuliah terbuka "Lintasan pengembang game: dari ide ke peluncuran" dan perpustakaan game di Sekolah Tinggi Hukum

Pada tanggal 17 April, Sekolah Tinggi Informatika Bisnis dari Universitas Riset Nasional Sekolah Tinggi Ekonomi mengundang semua orang untuk menghadiri kuliah terbuka oleh Sergey Golubkin, "Jalur Pengembang Game: Dari Gagasan Meluncurkan," dan berpartisipasi dalam studio game Game Dev Sim: simulator game studio .



Program malam itu:

18:30 - koleksi tamu

19:00 - 20:00 - kuliah terbuka: "Jalur pengembang game: dari ide hingga peluncuran"
Pembicara: Sergey Golubkin - mantan produser Ubisoft, Nival, Flexis dan pemilik GEMENOT Publishing House. Sergey telah mengelola tim di industri game selama 12 tahun. Dia akan memberi tahu apa yang sedang menunggu pencipta game komputer di semua tahap pengembangan, pendekatan apa yang membantu membawa game untuk dirilis, berbagi pengalaman pribadinya di industri game.

20:00 - 21:30 - Playhouse
Para tamu dari acara itu sendiri dapat mencoba untuk pergi jalan pengembang game sebagai bagian dari permainan papan Game Dev Sim: simulator studio game . Mereka akan belajar dengan cara main-main bagaimana mengumpulkan tim yang efektif, setuju dengan investor dan tidak melanggar tanggal rilis.
