محطة ويندوز الجديدة: إجابات لأسئلتك

في التعليقات على مقالة حديثة ، قمت بطرح الكثير من الأسئلة حول الإصدار الجديد من Windows Terminal الخاص بنا. اليوم سنحاول الإجابة على بعضهم.

يوجد بعض الأسئلة الأكثر شيوعًا التي سمعناها (ولا نزال نسمعها) ، بالإضافة إلى الإجابات الرسمية: بما في ذلك استبدال PowerShell وكيفية البدء في استخدام المنتج الجديد اليوم.



متى وأين يمكنني الحصول على Windows Terminal جديد؟


  1. يمكنك استنساخ الكود المصدري الطرفي من GitHub على github.com/microsoft/terminal وبنائه على جهاز الكمبيوتر الخاص بك.
    ملاحظة : تأكد من قراءة واتباع الإرشادات الموجودة على صفحة README في المستودع قبل محاولة إنشاء المشروع - لإنشاء المشروع تحتاج إلى اتباع بعض الشروط المسبقة وخطوات التهيئة!
  2. ستتوفر نسخة المعاينة من الجهاز للتحميل من متجر Microsoft في صيف عام 2019.

نحن نسعى جاهدين لإصدار Windows Terminal v1.0 بنهاية عام 2019 ، ولكن في نفس الوقت سنعمل مع المجتمع لإصدار هذا الإصدار بحيث تكون المحطة عالية الجودة.

هل تعد Windows Terminal بديلاً عن موجه الأوامر و / أو PowerShell؟


للإجابة على هذا السؤال ، دعونا نوضح بعض المصطلحات والمفاهيم:

  • Command Prompt و PowerShell (على سبيل المثال WSL / bash / وما إلى ذلك في * NIX) عبارة عن أصداف وليست أطراف طرفية ولا تملك واجهة مستخدم خاصة بها
  • عند تشغيل أداة shell / application / سطر الأوامر ، يقوم Windows تلقائيًا بتشغيلها وتوصيلها بمثيلات Windows Console (إذا لزم الأمر)
  • Windows Console هو تطبيق UI قياسي "يشبه الجهاز الطرفي" يأتي مع Windows وقد استخدمه المستخدمون على مدار الثلاثين عامًا الماضية لتشغيل أدوات سطر الأوامر في أنظمة التشغيل Windows NT و 2000 و XP و Vista و 7 و 8 و 10



وبالتالي ، ينبغي على الأرجح إعادة صياغة السؤال كـ "Windows Terminal - هل هو بديل لوحدة التحكم في Windows؟"

الجواب هو "لا":

  • سيستمر Windows Console في الشحن إلى Windows لعقود لتوفير التوافق مع العديد من ملايين البرامج النصية والتطبيقات وأدوات سطر الأوامر الحالية / القديمة
  • يعمل Windows Terminal مع Windows Console ، ولكن من المرجح أن يصبح الأداة المفضلة للمستخدمين الذين يرغبون في تشغيل أدوات سطر الأوامر على Windows
  • يمكن لـ Windows Terminal الاتصال بـ Command Prompt و PowerShell ، وكذلك أي تطبيق سطر / أداة / سطر أوامر آخر. يمكنك فتح علامات تبويب مستقلة متصلة Command Prompt أو PowerShell أو bash (عبر WSL أو ssh) وأي قذائف / أدوات أخرى من اختيارك
