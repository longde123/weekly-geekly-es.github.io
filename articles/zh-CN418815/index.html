开发人员在培训期间犯的典型错误-以及如何避免这些错误


在颠覆性技术和框架统治时代。

培训，技能开发和跟上最新技术和趋势的能力-任何程序员的头等大事。 这些不是奖金因素，而是至关重要的能力，没有这些因素，您将无法实现职业目标。 但是，在培训过程中，开发人员会犯一些常见错误-无论职业处于什么阶段。 今天让我们讨论这些错误。 我希望这里提供两三个技巧，您会立即开始更有效地学习。

翻译成Alconost

没有课程



Rawpixel Snapshot Unsplash网站

他们说，如果您计划失败，那么您计划失败。 这也适用于培训。 有时开发人员不能等待，他立即进入主动学习状态，完全没有准备。 假设您想学习Java语言的React框架并开始阅读React教程，甚至不知道要关注哪些领域，如何实现目标，即学习如何在React中编写应用程序。 在这种情况下，最好以这种方式学习：我们查看官方文档，逐步研究概念，从基础到高级，熟悉API帮助，在React上编写试点应用程序，了解其他可用示例，并在每个培训阶段设定SMART目标。 SMART目标之所以好是因为它们是特定的，可测量的，可实现的，切合实际的且有时间限制的。

没有明确的最终目标



来自Unsplash的 Patrick Fore快照

软件开发行业中有大量的新技术和新方法。 不要误会我的意思：我相信终身学习是好的。 如果您准备学习，则只需扩展技能范围并开发新知识。 但是，碰巧开发人员太容易受到SUV（“失去机会综合症”）的影响，并且正在尝试学习我所听说的所有新事物。 这样一来，他将没有时间进行生产性工作，也没有时间将知识和技能应用于商业。 在学习某些东西之前，我建议您考虑一下为什么要教它，在什么地方以及如何应用新学到的知识或技能。 例如，如果您正在开发数据库，​​但是正在学习Javascript，则意味着您将切换到全栈开发。 或者，如果您从事DevOps并学习AWS，那么也许您会被抽水以在使用AWS的公司中找到一份新工作。

选择一个太宽泛的话题



Unsplash网站Yuri Kovalev 摄

每个人都听到过“推车和小推车”这一说法吗？ 有时，开发人员过于急于尝试彻底研究选定的主题，不知道如何澄清自己的计划，并在培训期间设定了过于广泛的目标。 您问：“什么时候也来？” 我认为，如果您无法用简短的语言向其他开发人员描述培训计划的结果，那么该主题就太广泛了。 计划的学习成果可以判断开发人员通过学习特定主题应从我那里获得哪些知识。 当然，任何话题都可以永远研究，但是必须有一个限制，之后您必须对自己说：“到此为止。” 对实现最终目标的方式充满信心。 例如，这是一些使用React Javascript框架计划的学习成果： 

      ,  : -       React,  : --      JSON  REST API --     x  --     -- ,       (  ) --


请注意：这里给出的结果没有非常具体或详细地描述（例如，我们不编写“使用JSX语法创建组件”），因为您尚不了解React框架的所有细微差别。 它们不太宽：为每个项目描述一个特定的结果，而不是冗长而笼统的结果，例如“创建React应用程序”。

阅读和观看过多的书籍，手册，视频



Unsplash的Chuttersnap快照

来自手册的疲劳是现实。 如果您认为自己正在通过学​​习无穷的书籍，视频和教程来真正学习一些东西，那就浪费时间。 选择几种来源（书籍，视频，其他任何格式），然后使用它们。 您从视频中学到的信息可能会与同一主题的其他一些手册的内容强烈重叠。 另一个例子：亚马逊在React Javascript框架上的快速搜索产生了大约700本书，其中一些书名几乎相同，例如Learning React。 处理所有这些书是不现实的，即使您可以做到，也不会成为React专家或浪费您的时间。

不追踪进度



来自Unsplash的 Lindsay Henwood快照

好吧，如果您有一个清晰的计划和特定的目标，但是如果您不跟踪自己的进度和行动进度，它们就没有多大用处。 在极端情况下，每两周监测一次您的成就-每月检查一次您的前进方向是否正确。 我建议在日历上提醒您，以定期评估从教育过程开始就取得的进展。 如果您迟迟没有实现目标，那么您自己可能会明白停顿的确切位置-并调整计划。 另一方面，如果目标得以实现，则值得庆祝并继续保持动力。

有效学习



Unsplash上的思想目录快照

如今，有效学习的能力已成为一种奖励，也是取得专业成功的关键。 我坚信，如果您花时间了解流程的本质并希望改善自己，您就会精通任何事物。 这也适用于培训。 训练是一种在实践中发展起来的技能。 有时，知道不该做的事与开始训练时知道该做的事同样重要。 难怪中国社会哲学家孔子曾说过：
“没有思想的学习是浪费，没有学习的思想是危险的。”
-孔子
