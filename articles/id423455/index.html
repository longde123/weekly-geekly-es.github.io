Tablet yang benar

Saya tidak memiliki kesempatan untuk sepenuhnya membaca literatur khusus dalam PDF. Jika kita berbicara tentang pembaca, ini adalah ukuran halaman yang tidak mencukupi dan teks hitam putih (contoh kode harus berwarna). Di desktop - juga bukan pilihan. Jelas bahwa bahannya jauh lebih baik diserap berbaring di sofa. Butuh tablet dengan layar seluas mungkin.



Saya menjilat di ujung atas Samsung Note Pro untuk waktu yang lama - 12 inci didambakan di QHD dan bonus stylus. Saya memeriksa pasar loak - cukup mahal, jika dibandingkan dengan harga awal. Mengapa Nah, dalam beberapa kasus, baterai tidak lagi mengisi daya (baru dari Tiongkok - $ 25). Tetapi sebagian besar, karena memperbarui firmware resmi berhenti beberapa tahun yang lalu pada versi 5.0.2, dan itu sangat melambat. Jelas bahwa perangkat ini sudah ketinggalan zaman, dan Anda perlu mendorong seseorang entah bagaimana tanpa permintaan.

Cara memasang LineageOS
  • Gali komputer dengan Windows.
  • Disampaikan Samsung USB Android Driver untuk Windows.
  • Samsung Kies dan antivirus dinonaktifkan.
  • Memberikan utilitas Odin3 1.85.
  • Saya mengunduh file firmware TWRP untuk v1awifi (SM-P900) / v1a3gxx (SM-901) / viennaltexx (SM-905).
  • Saya mengunduh firmware LineageOS (pilih yang sesuai dengan model, saya punya v1awifi ) dan meletakkannya di drive SD eksternal di tablet.
  • Saya mengunduh aplikasi OpenGApps (platform ARM, Android 7.1) dan juga meletakkannya di media SD eksternal di tablet.
  • Dia membuka utilitas Odin3 "sebagai administrator", menekan tombol PDA dan memilih file firmware TWRP, mematikan kotak centang "Auto Reboot" dan meninggalkan tanda centang "F. Reset Time" dihidupkan.
  • Saya menghubungkan tablet dengan kabel USB3 Micro-B standar.
  • Dia menyalakan daya tablet dengan menekan tombol volume minus sampai screensaver Samsung Note Pro muncul; kemudian saya menjawab permintaan dengan tombol volume-plus yang ingin saya alihkan ke mode "Mengunduh ...".
  • Saya menekan tombol Start di utilitas Odin3, firmware selesai dalam beberapa detik.
  • Sambil memegang tombol volume-plus pada tablet bersama dengan tombol daya, saya menunggu sampai tablet mati dan dihidupkan lagi; merilis tombol hanya setelah screensaver Samsung Note Pro muncul - sehingga memuat aplikasi TWRP.
  • Di dalam TWRP, saya mengubah bahasa ke Rusia dan melakukan beberapa kali cadangan ke drive SD eksternal
  • Di dalam, TWRP melakukan pembersihan lengkap tablet dan melakukan pemulihan dari cadangan, seperti dalam lelucon tentang seorang pria yang jatuh dari gedung pencakar langit: - sejauh ini semuanya berjalan dengan baik.
  • Di dalam TWRP, saya mencoba menginstal firmware LineageOS - mengecewakan. Anda perlu memperbarui bootloader melalui Odin3, instruksi ada di sini .
  • Di dalam TWRP, saya kembali menginstal firmware LineageOS - dan sudah diinstal dan diperbarui (sementara data dan pengaturan disimpan, diverifikasi).
  • Di dalam TWRP, saya menginstal firmware OpenGApps - Saya mendapat akses ke Google Play Market.

Sedikit kesedihan, LineageOS tidak melihat saluran ke-13 pada WiFi - memperbaiki pengaturan di router.
