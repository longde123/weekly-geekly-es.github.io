Rilis


PostgreSQL 11 Beta 2 Dirilis
Rilis rilis ini juga terkait dengan masalah yang terdeteksi dengan SCRAM. Beta 1 memperkenalkan fitur utama, Saluran Binding untuk Otentikasi SCRAM, untuk mencegah serangan man-in-the-middle. Tetapi ketika pengujian ternyata libpq tidak dapat menjamin pengikatan saluran. Dalam Beta 2, fitur itu sendiri tidak dihapus, tetapi dokumentasi untuk itu telah dihapus, dan fitur kehilangan status utamanya. Anda dapat membaca lebih lanjut tentang rilis di sini , dan unduh Beta 2 di sini .

PgAdmin 4 versi 3.1 telah dirilis . Ini memperbaiki lebih dari 30 bug. Di antara fitur-fitur baru:
  • menambahkan dukungan untuk terowongan SSH,
  • MENDENGAR / PEMBERITAHUAN dukungan di Alat Kueri,
  • dukungan untuk fungsi dan prosedur di PostgreSQL 11
  • Peningkatan dukungan untuk Greenplum
Detail dapat ditemukan di sini .

Unduh paket untuk Windows, macOS, Python Wheel, Docker Container atau kode sumber tarball di sini . Dijanjikan bahwa RPM dan DEB akan tersedia dalam beberapa hari.

Richardson Software Merilis Editor SQL dan Alat Database Database RazorSQL 8
Rezor mendukung kerja dengan lebih dari 40 DBMS. Versi 8 memperkenalkan dukungan untuk mengkonversi dari Firebird ke 9 DBMSs, termasuk PostgreSQL. Unduh (untuk $ 99,95) di sini.
