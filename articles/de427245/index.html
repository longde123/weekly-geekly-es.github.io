Gruppen-Voicemail- und Anrufaufzeichnungsverwaltung in 3CX PBX

Voicemail der 3CX-Gruppe


Mit PBX 3CX v15.5 können Sie eine „allgemeine“ Sprachnachricht für eine Gruppe von Benutzern hinterlassen, die in der Gruppen- oder Anrufwarteschlange enthalten sind. Jetzt ist in der Anrufweiterleitungsregel Mit der Voicemail des Benutzers verbinden die Option Warteschlangen oder Gruppen verfügbar - jede Warteschlange / Gruppe verfügt über eine eigene Sprachbox. Gleichzeitig erhalten alle Benutzer in der Gruppe eine Benachrichtigung, die die Audiodatei der Sprachnachricht enthält. Natürlich müssen die Benutzerparameter eine E-Mail-Adresse haben.

Gruppieren Sie Voicemail im Sprachmenü / IVR




Die Weiterleitung an eine Gruppenbox über das Sprachmenü ist sehr praktisch, wenn Sie Kundenanrufe außerhalb der Geschäftszeiten sicher aufzeichnen müssen. In diesem Fall ist eine Gruppen-Voicemailbox vorzuziehen, da nicht nur ein engagierter Mitarbeiter, der möglicherweise nicht am Arbeitsplatz ist, sondern auch ein anderer Mitarbeiter aus der Gruppe den Einspruch bearbeiten kann.

Gruppen-Voicemail in Gruppen- und Anrufwarteschlangen




In der Anrufgruppe können wir einfach eine gemeinsame Sprachbox für alle Mitglieder der Gruppe einrichten. Wenn beispielsweise ein Anruf eingeht, während alle Techniker des technischen Supports beschäftigt sind, erhalten sie eine aufgezeichnete Sprachnachricht vom Benutzer.

Bitte beachten Sie, dass in der Anrufwarteschlange eine Benachrichtigung sowohl an Warteschlangenbetreiber als auch an Warteschlangenmanager gesendet wird, die möglicherweise keine Agenten sind.

Gruppen-Voicemail




Das Konfigurieren des Gruppenfelds eines Benutzers ist praktisch, wenn ein wichtiger Mitarbeiter „versichert“ werden muss. Wenn der Lagerverwalter beispielsweise weggezogen ist und den Anruf nicht annehmen kann, erhalten alle seine Untergebenen eine Sprachbenachrichtigung.

Zusätzlich zu den Weiterleitungsregeln können Sie der Gruppe eine Sprachnachricht hinterlassen, indem Sie die Vorwahl * 4 + Gruppennummer am Telefon eingeben. Beispiel: * 480 - hinterlässt im obigen Screenshot eine Sprachnachricht für alle Mitglieder der Gruppe 80 Warehouse.

Funktionen von Gruppenbenachrichtigungen


Beim Betreten einer Gruppen-Sprachbox hört der Teilnehmer die Standard-3CX-Systemnachricht für Voicemail in der Sprache, die standardmäßig auf 3CX eingestellt ist. Diese Regel gilt nicht für Anrufwarteschlangen. In diesen wird die Nachricht in der für diese Warteschlange festgelegten Sprache abgespielt.



Darüber hinaus gelten für Gruppen-Voicemail die folgenden Einschränkungen:

  • Die MWI-Benachrichtigung wird auf IP-Telefonen von Gruppen- / Warteschlangenmitgliedern nicht angezeigt (Sprachnachrichtenanzeige blinkt nicht)
  • Voicemail wird in 3CX-Clients, einschließlich Webclient, nicht angezeigt
  • Die Voicemail-Einstellungen einzelner Benutzer werden ignoriert und die folgenden Parameter werden verwendet: Nachrichtenaufzeichnung - Senden von Anhängen an Teilnehmer - Löschen der Voicemail-Datei auf dem Server

3CX Conversation Record Management


3CX v15.5 implementiert mehrere wichtige Technologien, die die Verwaltung von Anrufaufzeichnungen, einschließlich Voicemail, so bequem und funktional wie möglich machen. Sie können Datensätze über die 3CX-Oberfläche verwalten: Abhören, Löschen, Festlegen von Kontingenten und Festlegen des Zeitraums für das Löschen alter Datensätze. Darüber hinaus können Sie in den neuesten Updates auswählen, welche Anrufe aufgezeichnet werden sollen und in welchem ​​Format Datensätze gespeichert werden sollen.

Gespräche posten


Aufzeichnungen von Gesprächen können nur auf der lokalen Festplatte des Servers gespeichert werden. Dies bietet eine Hochleistungs-Telefonanlage bei gleichzeitiger Verwaltung mehrerer Datensätze. Der Speicherort für die Aufzeichnungen wird in der 3CX-Schnittstelle festgelegt (Sie können den Speicherort für Voicemail jedoch nicht neu zuweisen). Geben Sie hier nicht das Stammverzeichnis der Festplatte an!



Nachdem Sie eine Aufnahme ausgewählt haben, können Sie sie herunterladen, um sie anzuhören oder zu löschen.

Richtlinien zur Aufbewahrung von Kontingenten und Konversationsaufzeichnungen


Durch Klicken auf die Schaltfläche "Kontingent" gelangen Sie zu den Benutzeroberflächen für Kontingenteinstellungen und Richtlinien zur Aufbewahrung von Datensätzen. In den neuesten Updates können Sie auch ein neues Format zum Komprimieren von ADPCM-Datensätzen angeben. Sie können diese jedoch nur im Player, nicht jedoch im Browser anhören. Das Standardaufzeichnungsformat benötigt 1 MB für eine Gesprächsminute, während das ADPCM-Format nur 256 KB benötigt.


Wenn das Kontingent erreicht ist, wird die Aufzeichnung von Benutzerkonversationen beendet (derzeit kann nur der Gesamtwert des Kontingents für alle Benutzer festgelegt werden). Es wird empfohlen, den Administrator in den Einstellungen der TK-Anlage zu aktivieren, damit er Maßnahmen ergreifen kann. Darüber hinaus wird die Konversationsaufzeichnung immer beendet, wenn weniger als 500 MB freier Speicherplatz auf der Serverfestplatte verbleiben. Dies geschieht, um den Zugriff auf die PBX-Steuerschnittstelle nicht aufgrund eines Festplattenüberlaufs zu verlieren.

Wählen Sie Gespräche für die Aufzeichnung aus


Ab 3CX v15.5 Update 5 können Sie festlegen, welche Benutzerkonversationen aufgezeichnet werden sollen : alle oder nur externe.


In vielen Organisationen ist es logisch, nur Benutzerkonversationen mit externen Abonnenten aufzuzeichnen, beispielsweise mit Kunden oder Partnern. Verhandlungen mit Kollegen sind möglicherweise nicht so wichtig, daher sollten Sie mit diesen Datensätzen keinen wertvollen Speicherplatz belegen, insbesondere wenn die TK-Anlage auf einem Cloud-Hosting gehostet wird.

In der 3CX Enterprise Edition können Sie dem Benutzer auch erlauben, die Aufzeichnung eines Gesprächs unabhängig zu deaktivieren (oder zu aktivieren), indem Sie eine Taste auf dem IP-Telefon drücken.

Anzeigen von Anruflisten in 3CX Web Client


Oft wird uns die Frage gestellt: Wie kann ich meine eigenen Aufzeichnungen von Gesprächen im 3CX-Webclient anzeigen?



Im Moment befindet sich diese Einstellung in den Benutzereinstellungen auf der Registerkarte Telefone automatisch einstellen - 3CX Client.


Sie können dem Benutzer erlauben, nur seine eigenen Konversationen anzuhören und sie zu löschen. Diese Einstellungen gelten für alle 3CX-Clients: Webclient und Anwendungen für Windows, Mac, Android und iOS.
