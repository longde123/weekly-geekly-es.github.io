So erstellen Sie eine Finanzanwendung: 5 APIs, die dem Entwickler helfen



Hallo allerseits! Ich setze meine Forschung im Bereich Finanzen aus technologischer Sicht fort. Im vorherigen Artikel haben wir die vorhandenen Handelsterminals und ihre Funktionen untersucht, und heute schlage ich vor, tatsächlich über die Entwicklung von Finanzanwendungen zu sprechen.

Für das neue Thema habe ich fünf APIs verschiedener Finanzdienstleistungen ausgewählt, mit denen Sie sowohl eine Vielzahl von Daten empfangen als auch mit schriftlichen Robotern direkt an der Börse handeln können.

Yahoo Finanzen


Vielleicht eine der bekanntesten Finanzdienstleistungen der Welt. Mithilfe der kostenlosen API können Entwickler die Servicedaten zu Börsenkursen, Kursänderungen und Marktbewegungen verwenden. Informationen sind zu einer Vielzahl von Aktien sowie zu anderen Instrumenten wie Währungspaaren verfügbar.

Mit dem Rapid API-Dienst können Sie ganz bequem eine Verbindung zur API herstellen.



SMARTcom


API zum Erstellen einer Anwendung für den Handel an russischen Börsen. Entwickler können die API als Schicht zwischen dem Börsenhandelssystem und ihrem vorgefertigten Handelssystem (unabhängig erstellt oder gekauft) verwenden, einen eigenen Handelsroboter oder sogar ein gesamtes Handelsterminal mit einer „benutzerdefinierten“ Schnittstelle erstellen.

Der Zugriff auf die API ist kostenlos. Sie müssen lediglich ein Konto im System des Maklerunternehmens erstellen, das sie entwickelt hat. Es besteht die Möglichkeit, die Arbeit Ihrer Software mithilfe des Testzugriffs im virtuellen Handel zu debuggen.



Screenshot: SMARTcom- Dokumentation

Synchronisierung per Paybook


Ein Service, der Entwicklern von Finanzanwendungen für ausländische Märkte nützlich ist. Sie können an einem Ort Daten von Banken, Finanz- und Versorgungsunternehmen sowie Regierungsbehörden verschiedener Länder sammeln. Dieser Service wird bezahlt, Zugangskosten ab 50 US-Dollar. Die Liste der für die Integration verfügbaren Dienste finden Sie hier .



Currencylayer-API


Ein nützlicher Service, um Informationen über Wechselkurse und den Wert von 168 Weltwährungen zu erhalten. Daten werden aus vielen Quellen wie Bank-Websites und kommerziellen Informationsdiensten gesammelt. Entwickler versprechen Kompatibilität mit Anwendungen aller Art und Programmiersprachen.

Es gibt eine kostenlose, eingeschränkte Version der API (250 Anfragen pro Monat), eine angemessenere Version beginnt bei 9,99 USD (10.000 Anfragen pro Monat). Ihre auf der Currencylayer-API basierenden Lösungen werden von Unternehmen wie Lyft, Blizzard und Intel entwickelt.



Einfache Finanzierung


Der bekannte Dienst für die Buchhaltung persönlicher Finanzen in Runet bietet eine API für Drittentwickler. Wie auf der Projektwebsite angegeben, sind Anwendungen von Drittanbietern für soziale Netzwerke, mobile Anwendungen und Plug-Ins für Browser bei Benutzern der Ressource am beliebtesten.

Bewerbungen werden vom Serviceteam getestet, genehmigte werden auf einen speziellen Marktplatz gebracht und im Abonnement verteilt. Laut Statistiken aus der Verwaltung von EasyFinance können Anwendungsentwickler 15-50 Rubel pro Benutzer und Monat verdienen. Diese Einnahmen setzen sich aus Einnahmen aus dem Abonnieren der Tarife des Dienstes selbst, Integrationszahlungen von Plattformpartnern und Werbung zusammen.
