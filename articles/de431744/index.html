Was wird das Ergebnis sein?

Antwort + Analyse

a) 0
Der Trick dabei ist, dass {} nicht als Objekt, sondern als leerer Codeblock interpretiert wird. Daher wird der Ausdruck in +[] konvertiert, und dies ist 0 , weil + bevor der Wert ihn in eine Zahl umwandelt. Ich möchte darauf hinweisen, dass, wenn der Code als console.log({} + []); , dann wird es [object Object] , weil hier wird {} als Objekt interpretiert.
