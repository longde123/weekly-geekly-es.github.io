3CX v16 Update 1, 3CX App für iOS Beta und die neue Version von 3CX Call Flow Designer

Einführung in die neueste 3CX-Produktbewertung. Es wird viele interessante Dinge geben - nicht wechseln!

3CX v16 Update 1


Wir haben kürzlich das nächste 3CX v16-Update 1 veröffentlicht. Das Update enthält neue Chat-Funktionen und ein aktualisiertes Kommunikations-Widget für Ihre 3CX Live Chat & Talk-Site. Mit Update 1 wurde außerdem ein neuer Anrufverlaufsdienst eingeführt, der der Telefonanlage eine Skriptschnittstelle für die Anrufsteuerung hinzufügt. Die Skript-Engine arbeitet mit der Call Flow Designer-Entwicklungsumgebung zusammen und ermöglicht Ihnen das Erstellen von Anrufverarbeitungsskripten beliebiger Komplexität.

Chat im Webclient aktualisiert




Mit dem aktualisierten Chat können Sie Ihre Konversationen jetzt flexibel verwalten. Darüber hinaus interagiert es gut mit dem Kommunikations-Widget 3CX Live Chat & Talk .

  • Ein Site-Besucher kann einen Chat mit der 3CX Operator Queue starten. Dadurch wird eine Chat-Gruppe erstellt, die alle Warteschlangenbetreiber und diesen Besucher umfasst.
  • In Zukunft kann der Warteschlangenbetreiber den Gruppenchat auf sich selbst umstellen und die persönliche Kommunikation mit dem Besucher fortsetzen.
  • Außerdem kann der Betreiber den Chat bei Bedarf an einen anderen Betreiber oder einen normalen PBX-Benutzer übertragen.
  • Zum Entladen der Webclient-Oberfläche können die ausgewählten Konversationen in das Archiv übertragen werden (sie werden jedoch nicht gelöscht).
  • Verschiedene Arten von Chats (von der Site, Gruppe usw.) haben jetzt verschiedene Symbole zur einfachen Identifizierung.
  • Jetzt können Sie schnell eine E-Mail an einen Website-Besucher senden, indem Sie im Chat-Fenster auf eine E-Mail klicken.

Neue Funktionen werden im Chat-Handbuch und im Video ausführlicher beschrieben.

Das 3CX Live Chat & Talk-Widget wurde aktualisiert




Das aktualisierte 3CX Live Chat & Talk-Widget bietet zusätzliche Anpassungen der Benutzeroberfläche und eine verbesserte Integration in Websites, die sowohl auf WordPress CMS als auch auf anderen Technologien erstellt wurden.

  • Festlegen des Chatfenstersymbols - Sie können ein geeignetes Bild für den Titel des Chatfensters festlegen. Dies kann beispielsweise Ihr Firmenlogo sein.
  • Festlegen des Bedienersymbols - Sie können auch das Chat-Bedienersymbol festlegen, z. B. sein Foto.
  • Widget-Platzierung - Der Parameter „Position“ bestimmt die Platzierung des Widgets auf Webseiten - unten rechts (Standard) oder unten links.
  • Die Ansicht im mobilen Browser ist eine kleine, aber wichtige Ergänzung. Wenn Sie jetzt von einem mobilen Gerät aus auf die Site zugreifen, wird das Chat-Fenster standardmäßig minimiert.
  • Chat-Popup-Fenster - In 3CX v16 Update 1 wird das 3CX Live Chat & Talk-Widget-Fenster in einem separaten Fenster "eingeblendet", sodass sich der Besucher frei auf der Website bewegen und gleichzeitig den Chat kontaktieren kann.

Skriptschnittstelle Call Flow Service


Mit 3CX v16 Update 1 wird eine neue Skriptoberfläche für den Call Flow Apps Service eingeführt. Es unterstützt Sprachanwendungen 3CX neuen Standard. Bestehende Anwendungen können jedoch in der neuen Version von Call Flow Designer konvertiert oder geändert werden (siehe unten). Die Architektur des Call Flow Apps Service ist heute fertiggestellt. Der Anwendungsserver läuft unter 3CX für Debian / Raspbian Linux und Windows.

Neue 3CX App für iOS Beta


Wir haben unsere proprietäre 3CX iOS-App schon lange nicht mehr aktualisiert. Einige Benutzer haben sich sogar über eine falsche Dateiübertragung beschwert. Aber im nächsten Update sind alle Probleme behoben! Dieses Mal liegt der Schwerpunkt auf den Funktionen des integrierten Chats. Jetzt ist der Chat in der mobilen Anwendung dem Chat im 3CX-Webclient praktisch nicht unterlegen.



Die Anwendung kann Gruppenchats erstellen und ihnen Namen zuweisen. Außerdem wurde die Chat-Archivierung hinzugefügt. Wischen Sie zum Archivieren einer Konversation nach links (Sie können eine Konversation auf dieselbe Weise aus dem Archiv wiederherstellen).

Außerdem gab es in der Anwendung einen Chat mit Besuchern der Website über das 3CX Live Chat & Talk-Kommunikations-Widget. So wird es jetzt implementiert:

  • Wischen Sie nach links, um zu chatten und weitere Optionen zu entdecken: Nehmen Sie sich selbst, Übertragen, Beenden und Löschen.
  • Die Chat-Symbole auf der Website unterscheiden sich von den Symbolen für reguläre Chats, sodass Sie leicht zwischen ihnen unterscheiden können.
  • PUSH-Benachrichtigungen von Nachrichten auf der Website zeigen den Namen und den Nachrichteninhalt des Besuchers an.
  • An die Agentenwarteschlange gesendete Chats enthalten zur Vereinfachung den Namen der Warteschlange.

Probieren Sie die neue 3CX-App für iOS Beta über TestFlight aus !

Neuer Release 3CX Call Flow Designer


Erst diese Woche haben wir eine neue Version der Entwicklungsumgebung für 3CX CFD- Sprachanwendungen veröffentlicht. Es wurden neue C # -Komponenten, eine verfeinerte Benutzeroberfläche, eine verbesserte Fehlerbehandlung sowie eine automatische Aktualisierung der Entwicklungsumgebung eingeführt. Ein neuer CFD wird benötigt, um neue Arten von Sprachanwendungen für 3CX v16 Update 1 und höher zu erstellen.


Die aktualisierte Entwicklungsumgebung (IDE) CFD bietet Entwicklern zusätzliche Tools:

  • Neue Komponenten Führen Sie C # -Datei und -Code aus. Sie ersetzen die ältere Komponente "Externes Skript starten". Komponenten können C # -Codedateien oder Inline-Codefragmente direkt aus CFD-Anwendungen ausführen.
  • Die neue Komponente "Erweiterungsstatus festlegen" implementiert eine einfache Einstellung von Erweiterungsparametern aus der CFD-Anwendung.
  • Verbesserte Fehlerbehandlung. Der neue Ausdruckseditor überprüft zusätzlich die Werte und erkennt Fehler bei der Kompilierung.

Zusätzlich zu Verbesserungen im Entwicklungsprozess haben wir einige Funktionen hinzugefügt, die die Benutzerfreundlichkeit der Anwendung selbst verbessern:

  • Anwendung automatisch aktualisieren. CFD sucht jetzt automatisch nach neuen Versionen und installiert Updates, sobald diese veröffentlicht werden.
  • Mit einem neuen Menüpunkt „Projekt speichern unter“ können Sie ein CFD-Projekt unter einem neuen Namen oder an einem anderen Ort speichern.
  • Neues Kontextmenü "Audioordner öffnen" für Komponenten, die Audiodateien verwenden. Der Explorer wird geöffnet, um den Audioordner des Projekts bequem zu durchsuchen.
  • Bequeme Anzeige deaktivierter Komponenten. Sie sind jetzt abgeblendet, um sich von den aktiven Komponenten des CFD zu unterscheiden.

Die neue CFD-Version beinhaltet die Verwendung von 3CX V16 Update 1. Laden Sie die CFD herunter und installieren Sie sie aus dem Call Flow Designer-Installationshandbuch .
