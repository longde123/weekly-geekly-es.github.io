适用于macOS的Microsoft Edge

上个月，我们宣布了Windows 10下一个版本的Microsoft Edge的第一个预览版本。今天，我们很高兴地宣布适用于macOS的Microsoft Edge Canary通道的可用性。 现在，您可以从Microsoft Edge Insider站点为macOS或Windows 10 PC 安装预览版本，不久将提供更多Windows版本支持。



为Microsoft Edge建立“类似于Mac”的用户体验


适用于macOS的Microsoft Edge将提供与我们在Windows上预览的相同的新浏览体验，并进行了用户体验优化，以使其在Mac上有宾至如归的感觉。 我们正在调整总体外观，以匹配macOS用户对该平台上的应用程序的期望。



我们致力于通过差异化的用户体验功能和连接的服务，使用Microsoft Edge构建世界一流的浏览器。 在此初始版本中，我们对用户界面进行了几处更改，以与Microsoft设计语言保持一致，同时使它在macOS上显得自然。


这方面的示例包括许多调整，以匹配macOS的字体，菜单，键盘快捷键，标题框和其他区域的约定。 随着我们继续试验，迭代和倾听客户反馈，您将继续看到浏览器的外观在将来的版本中不断发展。 我们鼓励您使用“发送反馈”笑脸与我们分享您的反馈。



此外，我们通过利用Mac上可用的特定硬件功能来设计macOS独有的用户体验。 例如，通过触摸栏提供有用的上下文操作，例如网站快捷方式，标签切换和视频控件，以及使用触控板手势进行熟悉的导航。



推出适用于macOS的Microsoft Edge Insider频道


可通过我们称为“ Microsoft Edge Insider渠道”的预览渠道获得适用于macOS的新Microsoft Edge预览版本。 我们首先启动Microsoft Edge Insider Canary Channel，您可以下载该通道并在Microsoft Edge Insider网站上尝试。 此频道从今天开始在macOS 10.12及更高版本上可用。 开发频道将很快发布，一旦可用，您就可以与Canary频道并排下载和安装。 您可以从上个月的博客文章中了解有关我们的方法以及对不同渠道的期望的更多信息。



适用于Web开发人员的一致平台和工具


使用我们新的Chromium基础，您可以期望在Windows和macOS版本的Microsoft Edge上获得一致的渲染体验，以及与Windows相同的强大开发人员工具。


Web开发人员现在首次可以在macOS上的Microsoft Edge中测试站点和Web应用程序，并确信这些体验将在所有平台的下一版Microsoft Edge中相同。 （请注意，应继续检测特定于平台的功能，例如Windows 10上的PlayReady内容解密，以在这些平台上获得最佳体验。）


与我们的Windows预览版本一样，我们的新macOS版本还包括对可安装的，基于标准的Progressive Web Apps的支持，您可以使用浏览器开发人员工具进行检查和调试。 我们正在努力使PWA与您的本机应用程序相媲美，因此，安装后，它们将像本机应用程序一样出现在Dock，应用程序切换器和Spotlight中。


分享您的反馈


我们很高兴与您分享我们的第一个适用于macOS的Microsoft Edge Canary版本！ 获得您的反馈是帮助我们改进浏览器的重要一步-我们认为在macOS上创造最佳浏览体验至关重要。 我们希望您今天尝试预览 ，我们期待您的反馈和参与Microsoft Edge Insider社区。


如果您遇到任何问题，并想给团队提供反馈或分享建议，请转到Microsoft Edge Insider社区论坛 ， 在Twitter上与我们联系，或者仅使用Microsoft Edge菜单中的“发送反馈”选项即可。让我们知道您的想法。


对于Web开发人员，如果遇到在Chromium中重现的问题，最好提交Chromium错误 。 对于现有版本的Microsoft Edge中的问题，请继续使用EdgeHTML问题跟踪程序 。
