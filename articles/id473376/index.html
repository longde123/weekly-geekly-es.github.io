Dokumentasi pada Windows 10X bocor ke jaringan - versi baru OS untuk perangkat dengan dua layar


Kode sistem operasi Windows 10X direncanakan akan selesai pada awal 2020, dan peluncuran komersial akan berlangsung pada musim gugur

Microsoft terus merahasiakan detail Windows 10X - versi Windows 10 yang diadaptasi untuk laptop dan tablet dengan dua layar. Tapi sekarang kebocoran besar pertama memberikan pemahaman umum tentang tujuan desain dan pengembangan OS ini, di mana aplikasi Windows 10 asli dan aplikasi web (PWA) akan bekerja pada istilah yang sama.

Screenshot dan deskripsi dipublikasikan di situs santorini-os.azurewebsites.net . Tautan tersebut pertama kali diperhatikan oleh pengguna WalkingCat alias @ h0x0d . Informasi dari situs telah dihapus, tetapi pengguna lain, Boris alias @_milincic_, telah menyimpan arsip dengan tangkapan layar halaman .

Dilihat oleh catatan kaki, situs ini diluncurkan oleh kelompok kerja Santorini Design dari Microsoft.



Dokumentasi menyebutkan desainer utama Regina Son. Profil LinkedIn menyatakan bahwa ia telah bekerja pada Windows dan perangkat generasi berikutnya sejak Juli 2019.



Pedoman desain antarmuka Windows 10X: kegunaan, efisiensi, dan kesederhanaan. Arsitek UI mengatakan desain tersebut didasarkan pada penelitian, pengujian, dan dokumentasi teknis.

Sistem operasi pada dasarnya harus memberikan kenyamanan maksimal, sementara tidak membatasi pengguna. Contohnya adalah menu Pengaturan Cepat dengan daftar pengaturan yang paling umum digunakan.

Menu Mulai Baru (Peluncur)


Menu Mulai telah diubah namanya menjadi Peluncur. Ini termasuk garis untuk mencari web (plus mencari dokumen dan aplikasi pada komputer dengan dukungan input suara), area aplikasi dan halaman web statis, serta daftar dokumen yang direkomendasikan. Ini tampilannya.


Peluncur Santorini

"Peluncur membantu pengguna dengan cepat memulai dan melanjutkan tugas," kata dokumentasi. - Cari dengan mudah diintegrasikan dengan hasil web, aplikasi yang tersedia, dan file spesifik pada perangkat Anda. Selanjutnya dalam hierarki konten adalah kisi aplikasi default, yang dapat Anda sesuaikan dan ubah sesuai dengan kebutuhan dan preferensi Anda. Peluncur terbuka dan selanjutnya membantu mempercepat tindakan pengguna, yang seiring berjalannya waktu masuk ke memori otot. Dan akhirnya, konten yang disarankan diperbarui secara dinamis berdasarkan aplikasi, file, dan situs web yang paling sering digunakan. Anda dapat kembali ke alur kerja pribadi Anda tanpa memulainya dari awal. "

Aplikasi Bawaan: Tepi, Surat, Kalender, Word, Excel, PowerPoint, OneNote, Tim, Papan Tulis, Aktivitas, Foto, Toko, File Explorer, Spotify, Netflix, Kamera, Solitaire, Kalkulator, Alarm & Jam, Film & TV, Kantor, Catatan Tempel, Cat, Hub Belajar, Pengaturan, Cuaca, Snip & Sketsa, Perekam Suara, Musik Groove, People, Notepad, Hub Umpan Balik, Rencana Media, Perpesanan, dan hingga empat aplikasi OEM.

Grid dokumen terbuka. Ikon dari sana dapat diseret ke atas sehingga mereka segera muncul di layar pertama.

Dalam menu ini, Anda dapat mengelompokkan aplikasi ke dalam folder dan menghapusnya dari kisi (dan dari perangkat).

Kunci layar


Layar kunci pada dasarnya tidak ada, dan pengguna dapat dengan cepat mengotentikasi segera setelah perangkat dihidupkan (model Kunci Tertahan). Ini berbeda dari model Windows 10, di mana sebelum otentikasi Anda perlu menghapus layar kunci "pendahuluan" ("tirai") dari layar.


Layar untuk otorisasi dalam sistem pada September 2019


Layar otorisasi dalam sistem memberikan akses ke desktop, layar kunci pada dasarnya tidak ada

Pengenalan wajah Windows Hello didukung, serta Windows Hello PIN. Setelah bangun dari mode tidur, Windows Hello Face langsung mengenali pengguna dan segera membuka desktop-nya.

Jendela Pengaturan Cepat


Ada dua level di bagian pengaturan: yang pertama menunjukkan hingga sepuluh pengaturan yang paling populer, dan yang kedua menunjukkan sisanya. Daftar pertama yang diedit. Nilai default adalah:

  • Wi-fi
  • Data Sel (tampaknya data jaringan seluler).
  • Bahasa input
  • Mode Menulis
  • Bluetooth
  • Mode penerbangan.
  • Rotasi layar.
  • Proyek

Bilah tugas




Akses ke aplikasi dan situs yang sering digunakan, serta pengalihan tugas. Tab aktif dapat di latar depan atau di latar belakang. Pengelompokan didukung ketika beberapa contoh aplikasi dikelompokkan di bawah ikon yang sama. Saat melayang, Ul muncul dengan thumbnail untuk berbagai contoh aplikasi ini. Bilah tugas menampilkan status "status aktif" (aplikasi dalam fokus) dan "status eksekusi" (ikon digarisbawahi saat aplikasi sedang berjalan, terlepas dari apakah itu dalam fokus atau tidak).

Desainer mengatakan bahwa mereka terus bereksperimen dengan pengelompokan ikon yang berbeda (terpusat atau disejajarkan dengan kiri), dengan jumlah dan ukuran elemen, jumlah peristiwa terkini yang ditampilkan, ada / tidaknya separator, dan sebagainya.

Anda tidak dapat mengatakan dengan pasti bahwa konsep ini telah diadopsi dan disetujui oleh manajemen. Tetapi bagaimanapun juga, kita dapat menyimpulkan dengan kerangka acuan apa yang digunakan oleh para perancang, yaitu ke arah mana Microsoft ingin mengembangkan UI sistem operasi.

Windows 10X


Microsoft pertama kali menyebutkan pengembangan sistem operasi baru pada awal Oktober sebagai "OS untuk Perangkat Neo-Screen Surface," dan mengumumkan tanggal rilis tentatif untuk 2020.

OS mungkin akan muncul bersamaan dengan perangkat ini sendiri. Tampaknya "Windows 10 Lite" dapat diinstal pada perangkat lain, termasuk laptop.

Orang dalam di Microsoft melaporkan bahwa kode Windows 10X dijadwalkan berakhir pada awal 2020, bersama dengan Windows 10 20H1, dan Surface Neo dan perangkat lain dengan sistem operasi baru akan memasuki pasar pada musim gugur, sebelum dimulainya musim Natal.

Microsoft belum secara resmi mengomentari kebocoran dokumentasi.
