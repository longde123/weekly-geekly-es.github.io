3CX v15.5 Pembaruan 5 Beta dan integrasi REST dengan AmoCRM dirilis

Minggu ini kami memperkenalkan Pembaruan 5 BETA 3CX v15.5 Pembaruan 5. Pembaruan memperkenalkan jenis-jenis laporan baru, rekaman percakapan yang ditingkatkan, ditambahkan modul integrasi CRM baru (termasuk AmoCRM Rusia) dan peningkatan lainnya. Pertimbangkan apa yang muncul dalam rilis ini.

Jenis-jenis laporan baru


Laporan panggilan trunk yang baru memungkinkan Anda melihat jumlah panggilan masuk ke trunk ini (atau saluran DID).


Laporan ini sangat mudah untuk melacak jumlah panggilan yang diterima pada nomor tertentu. Informasi dapat digunakan, misalnya, untuk menganalisis efektivitas kampanye iklan atau jumlah panggilan ke berbagai departemen dalam suatu organisasi.

Dimungkinkan juga untuk mengecualikan panggilan yang berlangsung kurang dari waktu tertentu. Ini menghilangkan distorsi dalam statistik yang timbul dari panggilan terlalu pendek. Anda dapat memfilter panggilan berdasarkan durasi dalam laporan apa pun.

Dalam Pembaruan 5 Beta 3CX v15.5, Anda sekarang dapat membuat laporan ringkasan (konsolidasi) untuk beberapa Antrian Panggilan dalam sistem. Lihat videonya .


Peluang ini sering diminta oleh manajer call center. Sekarang pengawas dalam satu laporan dapat menganalisis pekerjaan beberapa kelompok operator. Jika call center melayani beberapa proyek, laporan seperti itu mutlak diperlukan! ..

Peningkatan rekaman percakapan


Pembaruan 5 telah memperluas opsi perekaman panggilan. Sekarang, administrator dapat mengaktifkan atau menonaktifkan pengguna untuk berhenti merekam percakapan. Selain itu, administrator dapat menentukan panggilan mana yang akan direkam: semua atau hanya eksternal. Seringkali, panggilan internal tidak menarik dan tidak dapat ditulis. Ini sangat mengurangi beban pada server dan menghemat ruang disk. Rekaman percakapan sekarang dapat secara opsional disimpan dalam format ADPCM. Format ini mengurangi ukuran file rekaman sekitar 4 kali, tetapi tidak dapat diputar dengan pemutar musik dan browser standar (tetapi dapat dimainkan oleh sebagian besar pemain pihak ketiga). Lihat videonya .

Kami berencana untuk menambahkan lebih banyak fitur rekaman panggilan dalam pembaruan 3CX di masa mendatang.

Modul integrasi CRM server baru


Kami bekerja keras pada integrasi REST dengan sistem CRM. Dalam Pembaruan 5, dukungan untuk sistem CRM telah diperluas dan, di samping itu, panduan integrasi yang dirancang ulang. Selain itu, menjadi mungkin untuk memuat template integrasi CRM CRM langsung dari antarmuka 3CX.



Dukungan server tambahan untuk CRM berikut:

  • AmoCRM (lebih lanjut tentang ini di bawah)
  • Exactonline
  • Zohov2
  • Opencnam
  • Vtiger
  • Template yang diperbarui untuk Salesforce, Freshdesk, dan Connectwise.

Panggilan darurat dan obrolan perusahaan


Dalam Pembaruan 5 Beta 3CX v.15.5, untuk grup / rentang ekstensi, Anda sekarang dapat menentukan jalur mana untuk merutekan panggilan darurat.


Juga, jumlah panggilan simultan dalam lisensi Anda tidak lagi memengaruhi panggilan darurat. Anda dapat melakukan panggilan darurat pada lisensi apa pun jika ada saluran telepon operator gratis. Lihat video pengaturan.

Inovasi lain - administrator sekarang dapat menonaktifkan obrolan perusahaan untuk semua pengguna.

Memulihkan kata sandi administrator dan mengganti kunci lisensi


Terkadang, kata sandi untuk antarmuka manajemen 3CX hilang atau terlupakan. Sebelumnya, untuk memulihkan kata sandi, perlu menghubungi dukungan teknis 3CX, tetapi sekarang Anda dapat mengirim kata sandi ke alamat email yang ditentukan sebagai alamat administrator di sistem.

Sekarang Anda dapat mengganti kunci lisensi dengan cepat, yang akan menghemat banyak waktu bagi pengguna yang ingin beralih dari versi gratis ke versi komersial sistem. Sebelumnya, jika kunci Anda tidak dapat diperbarui secara langsung (tanpa penggantian), diperlukan pemasangan ulang server 3CX. Sekarang Anda cukup menentukan kunci baru dan bukan yang lama, dan sistem terus bekerja. Lihat videonya .

Pemberitahuan Admin Terperinci


Untuk memastikan keamanan, 3CX memblokir alamat IP dari mana upaya dilakukan untuk memasukkan login atau kata sandi yang salah. Namun, sebelumnya sulit untuk menentukan sumber pasti masalah. Sekarang, pemberitahuan terperinci akan membantu administrator menemukan alasan pemblokiran IP atau memperbaiki masalah di perangkat VoIP yang mengirim data otorisasi yang salah.

Peningkatan lainnya v15.5 Perbarui 5 Beta


  • Cadangkan ke Google Drive mendukung Google API v1.32, yang meningkatkan kinerja dan stabilitas.
  • Dukungan untuk .NET Core to 2.0.7 dengan perbaikan untuk prosesor yang tidak mendukung ekstensi AVX (Intel NUC, Celeron dan lainnya dari daftar ini) ...
  • Jumlah slot parkir sistem telah ditingkatkan menjadi 250.
  • Konfigurasi default sekarang memungkinkan hingga 1024 panggilan simultan.
  • Memperbaiki bug dalam laporan yang terkait dengan durasi transfer panggilan yang diikuti.
  • Peningkatan pengiriman pesan suara.
  • Peningkatan dukungan untuk ekstensi SIP P-Asserted-Identity.
  • Dukungan untuk TLS 1.2 (3CX sekarang sepenuhnya kompatibel dengan PCI). Namun, konfigurasi perangkat dan klien lawas tidak mungkin dilakukan. Dalam hal ini, Anda disarankan untuk beralih ke TLS 1.0.
  • Cadangan FTP sekarang mendukung EPSV (Extended Passive Mode).
  • Template untuk Yealink T4X sekarang mendukung beberapa modul ekspansi EXP40.
  • Kecepatan pemberitahuan PUSH telah ditingkatkan (perbaikan lebih lanjut direncanakan di Pembaruan 6).

3CX Server Integration dengan AmoCRM


Dalam pembaruan ini, kami "mengembalikan" integrasi server dengan sistem AmoCRM Rusia yang populer. Integrasi menunjukkan data kontak atau perusahaan pada panggilan masuk, memungkinkan Anda membuat kontak baru, serta menyimpan riwayat panggilan dalam kartu kontak.
Panggilan keluar dari AmoCRM dilakukan menggunakan plugin browser 3CX Click2Call kami.

Untuk mengaktifkan integrasi, di portal AmoCRM, buka Pengaturan> API dan salin kunci API.



Di antarmuka 3CX, buka Pengaturan> Integrasi CRM> Sisi Server (Opsi> Integrasi CRM> tab Sisi-Server) dan pilih amocrm.



Tentukan tiga parameter:

  • Alamat Portal AmoCRM Anda
  • Alamat email pengguna tempat portal AmoCRM terdaftar
  • Kunci API disalin di atas

Ini melengkapi integrasi 3CX dan AmoCRM CRM!
