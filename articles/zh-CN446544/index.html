Microsoft通过Azure IoT创新者和初创公司的新IP优势扩展了Azure IP优势计划


在Microsoft，我们正在投资帮助客户迁移到云中。 我们看到了一个机会，可以利用我们的安全性，隐私权，合规性以及知识产权资产和专业知识来帮助企业在不断变化的环境中发展，从而帮助他们取得更大的成功。 我们很高兴现在可以采取进一步措施来扩展创新保护。

今天，我们很高兴宣布Microsoft Azure IP优势计划的扩展，其中包括Azure IoT创新者和初创公司的新优势 。 我们于2017年2月首次宣布了 Azure IP Advantage，旨在为我们的云客户提供全面的知识产权保护（IP）风险。 我们当时看到的一种趋势-一直持续到今天-专利诉讼对云技术创新的风险越来越大 。 去年，我们加入了开放发明网络（OIN）和转让许可（LOT）网络，以帮助解决客户和合作伙伴的专利主张风险。


本文在我们的博客中 。

但是我们相信我们可以做得更多。 从2013年到2018年，美国受IoT诉讼的专利数量增长了400％以上 。这就是为什么在与客户交谈并思考了我们如何为Azure IP优势计划增加更多价值之后，我们决定通过以Azure驱动的物联网（IoT）生态系统和初创公司为重点的新优势扩展该计划，以帮助阻止针对Azure客户的诉讼。 今天宣布的新功能包括：


  • Microsoft的Azure Sphere和Windows IoT的无上限赔付范围。 赔偿有助于保护客户免受因使用产品或服务而对客户提出的IP侵权索赔。 如今的扩展为Azure Sphere和Windows IoT带来了无上限的赔偿范围，包括Microsoft在这些产品中并入的开源软件。
  • 使用Azure为IoT设备提供动力以抵御IP诉讼的客户获得10,000项专利。 这项好处可以帮助阻止针对Azure客户在Azure或IoT设备上运行的工作负载和应用程序针对Azure客户提起专利诉讼，因为合格的客户可以选择专利来为诉讼辩护。
  • Azure上的初创企业获得微软专利以帮助促进其业务的能力。 也可以加入LOT Network的合格初创公司可以通过LOT获得包括人工智能，多媒体和安全性在内的技术领域的Microsoft专利。 专利对于初创企业的成长起着重要的作用，他们寻求保护其创新的方法并吸引关键的资本支持。

许多初创公司向我们介绍了他们作为小公司面临的特殊漏洞，以及他们在寻求筹集资金并为未来发展时，重视发展自己的早期IP投资组合的重要性。 我们在LOT网络的合作伙伴已经建立了一个解决这些需求的计划。 微软很高兴向LOT网络捐赠500项专利，LOT网络现在将作为免费会员的一部分提供给合格的初创公司 。


综上所述，我们认为Microsoft提供了最全面的端到端框架来应对云中的IP风险。


客户的初步反馈是积极的：


“对于像蒂森克虏伯这样的国际工程公司来说，Microsoft的Azure IP优势计划是降低向工业4.0过渡的IP风险的好工具。” 蒂森克虏伯有限公司知识产权与服务主管斯蒂芬·沃尔克（Stephan Wolke）。


该公司在不断增长的神经形态计算市场中担任董事长兼联合首席执行官彼得·苏马（Peter Suma）表示：“应用大脑研究部建立了受大脑启发的AI系统。” “ Azure IP优势可以帮助我们专注于我们的下一个创新，并减少花费在担心IP风险上的时间。”


我们的目标是建立一个专注于创新而不是担心保护其创新的社区。 我们希望这一扩展Azure IP Advantage计划的步骤将对此有所帮助。 若要了解更多信息，请访问Microsoft Azure IP Advantage网站。
