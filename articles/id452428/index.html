Kamus Yandex cepat

Mungkin semua orang tahu bahwa Yandex memiliki layanan Yandex.Slovar gratis yang luar biasa di mana Anda dapat melihat terjemahan kata yang Anda butuhkan, tetapi bagi saya itu tidak nyaman karena saya terus-menerus harus beralih ke Chrome, secara manual memasukkan teks atau tempel dari clipboard, bahkan jika Anda membaca teks dalam bahasa Inggris di browser yang sama, terus-menerus beralih di antara tab yang macet.

Sampai saat ini, saya menggunakan kamus ABBYY Lingvo

Jadi ada tertulis dalam uraian mereka di situs, dari pengalaman saya sendiri, saya bisa mengatakan itu, sangat nyaman, minus harga, kamus Inggris-Rusia ABBYY untuk Windows harganya 2.390 rubel, ada juga Babel NG dengan fungsi terjemahan konteks yang sama dan banyak kamus. yang harganya dua kali lipat.

Ada Lingoes gratis , tidak mendukung semua browser, dan itu menolak untuk bekerja di Windows 10 (setidaknya untuk saya).

Semua orang tahu bahwa Yandex memiliki layanan Yandex.Slovar, tetapi tidak semua orang tahu bahwa ada juga API Kamus untuk pengembang.

Saya agak suka programmer Delphi, oleh karena itu, memutuskan untuk menulis kamus dengan fungsi terjemahan konteks seperti ABBYY Lingvo dan Babylon NG, di sini kamus Yandex API sangat berguna.

Keajaiban pemrograman ini terlihat seperti ini, di sebelah terjemahan di browser, arah terjemahannya adalah Inggris-Rusia, Rusia-Inggris.



Untuk menerjemahkan kata atau frasa dalam aplikasi apa pun, tahan Ctrl dan arahkan kursor ke kata yang kami minati, Anda dapat mengubah tombol pintas jika Anda tidak suka Ctrl + arahkan kursor ke kursor default.



Jika Anda terus menerjemahkan dari satu bahasa saja, misalnya, dari bahasa Inggris ke bahasa Rusia, atur arah terjemahan di menu, tambah kecepatan dan ketepatan pengenalan teks.



Ketentuan penggunaan layanan Yandex.Slovar API melarang penyimpanan terjemahan ke disk, oleh karena itu koneksi permanen ke Internet diperlukan untuk terjemahan.

Program ini benar-benar gratis, berjalan pada Windows 7, 8, 10, dan tesseract ocr + OpenCV digunakan untuk mengenali kata-kata di bawah kursor mouse.
