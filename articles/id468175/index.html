Blog pengembangan Frontend layak dibaca

Setiap orang yang menyebut dirinya seorang pengembang web harus mengetahui tren dan peristiwa terkini yang terjadi di industri. Atas dasar ini, tangkap pilihan blog yang menarik pada pengembangan frontend. Perlu dicatat bahwa sebagian besar pilihan adalah blog dalam bahasa Inggris.

Blog berbahasa Inggris


Feed berita dan akun tematik

  • Smashing Magazine adalah majalah online untuk para profesional desain dan pengembangan web.
  • CSS-Tricks adalah sumber resmi CSS-Tricks.com untuk perancang web.
  • CSS {IRL} - tips, lifehacks, trik, dan tautan terkait CSS.
  • Web Platform News - berita pengembangan web harian dari sumber dengan nama yang sama .
  • Apart Book adalah penerbit buku dengan topik desain web responsif, Git dan JavaScript, strategi konten, prinsip desain, berbicara di depan umum, dan banyak lagi.



Orang yang menarik


Pilihan blog oleh orang-orang terkenal dari dunia pengembangan, peserta dan penyelenggara pertemuan dan konferensi yang memberikan kontribusi besar bagi pengembangan web, misalnya, pengembang browser Chrome, programmer Google terkemuka, dan sebagainya.

  • Mathias Bynens adalah pengembang V8, mesin sumber terbuka berkinerja tinggi JavaScript dan WebAssembly dari Google.
  • Jake Archibald adalah penginjil web dan perwakilan dari browser Google Chrome, salah satu pakar terbaik perusahaan, penulis banyak laporan tentang topik unik.
  • Phil Walton , seorang insinyur di Google yang bekerja pada browser Chrome, telah berkembang sejak tahun 1998 dan memiliki blog pribadi .
  • Monica Dinculescu - "emojineer", bekerja dengan Google di proyek Magenta , menciptakan musik dan seni melalui pembelajaran mesin.
  • Tim Kadlec adalah konsultan produktivitas teknis yang membuat web lebih cepat, penulis beberapa buku, dan peserta konferensi.
  • Léonie adalah spesialis aksesibilitas dan salah satu pendiri W3C Web Platform WG .
  • Eric Meyer adalah seorang ahli dalam HTML dan CSS, penulis banyak artikel dan buku, pencipta beberapa alat dan sumber daya yang berguna. Lebih detail - di situs pribadinya .
  • Una adalah penginjil web dan direktur desain produk untuk Bustle Digital Group.
  • Harry Roberts adalah konsultan produktivitas berbasis web independen yang bekerja dengan perusahaan internasional besar. Mengundang pakar Google, pengembang pemenang penghargaan, pembicara internasional, dan duta besar kinerja untuk SHIFT Commerce .
  • Alex Russell adalah tim teknologi oleh standar tim Chrome, anggota kelompok arsitektur teknis W3C dan pengembang ECMA TC39 (standar JavaScript).
  • Surma adalah penginjil web Google.
  • Manuel Matuzović adalah pengembang web profesional sejak 2008 dan freelancer sejak 2010, spesialis dalam HTML, aksesibilitas, tata letak CSS, dan arsitektur.
  • HJ Chen adalah penginjil web di Nexmo .
  • Jen Simmons adalah desainer penginjil di Mozilla , seorang anggota kelompok kerja CSS yang tahu bagaimana CSS Grid mengubah desain web.
  • Martin Splitt adalah penginjil OSS1 dan pengembang di Google.
  • Maximiliano Firtman adalah pengembang aplikasi seluler dan web yang independen, seorang mentor dan pembicara yang didedikasikan untuk aplikasi web yang inovatif, produktivitas, mobilitas, dan platform web. Penulis banyak artikel teknis dan buku tentang pemrograman.
  • Rachel Andrew adalah pengembang web, anggota kelompok kerja CSS, dan ketua redaksi Smashing Magazine .
  • Roma Komarov adalah pengembang CSS frontend, pencipta Hayaku, komponen bemto, bemto untuk Pug.js dan sebagainya .
  • Addy Osmani adalah pemimpin Tim Kecepatan di Google Chrome, yang membuat web lebih cepat. Addy juga bekerja pada beberapa proyek sumber terbuka - lebih terinci di situs web pribadi .



Akun resmi browser, editor



  • Chrome DevTools - Berita, pembaruan terbaru, dan ekstensi berguna untuk pengembang Chrome .
  • Pengembang Chrome - Berita untuk pengembang dari tim Hubungan Pengembang Chrome.
  • WebKit adalah blog engine tampilan halaman web WebKit resmi.
  • Mozilla Hacks adalah akun browser Mozilla Firefox resmi untuk pengembang.
  • Samsung Internet Dev adalah platform untuk mendukung pengembang peramban seluler dan VR Samsung.
  • Intent To Ship adalah blog yang secara otomatis membuat tweet ketika browser mengumumkan penghapusan, modifikasi, atau tampilan fitur di mesin mereka.
  • Visual Studio Code adalah akun resmi salah satu editor kode populer.
  • Atom Editor - Akun editor kode atom.
  • Sublime HQ - Sublime berita dan pembaruan editor teks eksklusif.
  • JetBrains WebStorm adalah blog pengembangan online untuk JavaScript frontend dan sisi server.
