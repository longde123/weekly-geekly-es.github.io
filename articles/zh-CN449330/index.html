相干CFP2-ACO / CFP2-DCO WDM光模块（100G / 200G）及其在DWDM系统中的应用



关于相干CFP模块（相干CFP光学可插拔模块）的外观的第一批新闻稿大约在5-6年前开始出现。 因此，它们在光学密封系统中的使用是新的，本质上是一种小众解决方案。 如今，经过六年的发展，这些模块已紧密进入电信领域，并继续得到普及。 它们是什么，它们之间的区别以及基于它们的解决方案（当然还有扰流板下的图片）-所有这些都在削减之下。 要阅读本文，您需要了解DWDM系统的基本原理。

模块组件


他们的第一个（Acacia）模块是CFP-ACO。 下面简要介绍了相干的CFP模块之间的实际差异。 但是为此，您首先需要做一个小小的题外话，并介绍一下什么是DSP，这在很大程度上是这项技术的核心。

关于模块和DSP的一些知识
一个模块通常由几个组件组成



  1. 窄带可调激光器
  2. 双极性相干调制器
  3. 数模转换器（DAC / ADC）-将数字信号转换为光信号的DAC，反之亦然。
  4. 数字信号处理器（DSP）-从信号中恢复有用信息，从中消除传输过程中对有用信号施加的影响。 特别是：

  • 色散补偿（CMD）。 此外，其数学补偿的存量实际上是无限的。 这是很了不起的，因为CMD的物理补偿总是会引起很多问题，因为这会增加光纤中的非线性效应。 您可以在Internet或书籍中阅读有关非线性效应的更多信息。
  • 偏振模色散（PMD）的补偿。 补偿也以数学方式发生，但是由于PMD本质的复杂性，这是一个更为复杂的过程，而PMD现在是限制光学系统范围的主要原因之一（除了衰减和非线性效应之外）。

DSP以非常高的符号速度工作，在最新系统中，这些速度约为69 Gbaud。

那么它们有何不同？


相干光模块通过DSP的位置彼此区分：

  • FP-ACO-仅光学部件位于模块上。 所有电子设备均位于插入此模块的设备的卡（板；板）上。 当时，根本没有将DSP放置在光模块内的技术。 实际上，这些是第一代模块。
  • CFP-DCO-在这种情况下，DSP位于光模块本身中。 该模块是一个完整的“盒装解决方案”。 这些是第二代模块。

在外部，这些模块具有完全相同的外形尺寸。 但是它们具有不同的填充物，消耗量（DCO大约是其两倍）和发热量。 因此，解决方案制造商具有一定的灵活性-ACO提供了更深入的解决方案集成，DCO允许您使用乐高魔方等光学模块构建解决方案，从而“开箱即用”地获得解决方案。 一个单独的问题-在大多数情况下，一对DSP的工作只能由同一制造商提供。 这强加了某些限制，并可能使DCO模块对互操作性更具吸引力 。
