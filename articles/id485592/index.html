PHP internal


  • [RFC] Tambah antarmuka Stringable - Diusulkan dalam PHP 8 untuk menambahkan antarmuka Stringable baru yang dapat ditambahkan ke kelas yang menerapkan metode __toString() . Dan kemudian gunakan tipe string gabungan | Stringable di mana string seharusnya. 
     interface Stringable { public function __toString(): string; }

Alat-alatnya


  • cycle / orm 1.2 - ORM, yang dapat digunakan baik sebagai DataMapper dan gaya ActiveRecord. Dalam rilis terbaru, kecepatan kerja meningkat sebesar 33% dan dilihat dari tolok ukurnya , sekarang ini adalah salah satu ORM tercepat.
  • BenMorel / weakmap-polyfill - Polyfill WeakMap untuk PHP 7.4.
  • lisachenko / z-engine - Perpustakaan eksperimental yang memungkinkan menggunakan FFI untuk mengakses struktur internal PHP itu sendiri. Contoh penggunaan yang keren telah ditambahkan, misalnya, menerapkan operator kelebihan beban dalam PHP dan bahkan memodifikasi AST saat itu juga.
