Perbarui alat Web dan Azure di Visual Studio 2019

Kemungkinan besar Anda sudah melihat bahwa Visual Studio 2019 dirilis . Seperti yang Anda harapkan, kami telah menambahkan peningkatan untuk pengembangan dan pengembangan web dengan Azure. Sebagai titik awal, Visual Studio 2019 menyediakan fitur baru untuk memulai dengan kode Anda , dan kami juga memperbarui pengalaman dalam membuat proyek-proyek ASP.NET dan ASP.NET Core untuk memenuhi persyaratan berikut:



Jika Anda menerbitkan aplikasi Anda ke Azure, sekarang Anda dapat mengonfigurasi layanan aplikasi Azure untuk menggunakan contoh Azure Storage dan Azure SQL Database langsung dari halaman ringkasan di profil terbitkan tanpa meninggalkan Visual Studio. Ini berarti bahwa untuk semua aplikasi web yang ada yang berjalan dalam layanan aplikasi, Anda dapat menambahkan SQL dan penyimpanan, karena tidak lagi terbatas hanya pada waktu pembuatan.



Dengan mengklik tombol Tambah, Anda dapat memilih antara repositori Azure dan database Azure SQL (lebih banyak layanan Azure akan didukung di masa mendatang):



dan kemudian Anda dapat memilih antara menggunakan instance penyimpanan Azure yang sudah Anda siapkan sebelumnya, atau mulai menyiapkan yang baru sekarang:



Saat Anda mengonfigurasi layanan aplikasi Azure melalui profil publikasikan, seperti yang ditunjukkan di atas, Visual Studio akan memperbarui pengaturan aplikasi di Layanan Aplikasi Azure untuk menyertakan string koneksi yang Anda konfigurasi (misalnya, dalam hal ini daftar huruf). Studio juga akan menerapkan tag tersembunyi ke instance di Azure tentang bagaimana mereka dikonfigurasikan untuk bekerja bersama sehingga informasi ini tidak hilang dan nantinya dapat ditemukan kembali oleh instance lain dari Visual Studio.
