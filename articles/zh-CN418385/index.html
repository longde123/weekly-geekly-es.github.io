我如何组装旧游戏计算机


哈Ha 在本文中，我想谈谈我如何为灵魂和旧游戏收集（购买）旧计算机。 以前，我指的是90年代下半场的比赛，也有些是零开始的比赛。 尽管没有人禁止运行较旧的游戏，但在这样的计算机上使用DOS总是有好处。

它被选为基础...总的来说，它是选择在我的城市中发现的，而无需通过俄罗斯和SMS进行转移。 不久前，我开始寻找这种计算机。 我已经很久没有固定计算机了，但是在这里，对年轻人的怀旧之情和哈布雷关于复古PC的一些文章使我寻找了一些东西。 关于将真正的486DX2与CRT显示器和保护屏组装在一起的故事，不会有任何美丽的故事，所有一切都比较平淡（但实用）。


选择和搜索


最初，我没有任何特殊标准，我只是想让某种东西“像我年轻时一样”（嗯，那里的草还更绿了，你知道）。 我的第一台计算机是（在遥远的第98危机中）奔腾120 / 8Mb / 210Mb / CD-ROM / SB，我不希望找到这样的东西并活着（但没有找到）。 但是我只希望Intel Pentium，AMD和其他人不考虑。 即 在486日，无论是AMD还是Cyrix，我都会很高兴，但是这也是一个幻想（要么什么都没有，要么他们像新的一样要钱，我不抱怨价格，也许他们有理由保存完好的副本，我只是还没有准备好就是花了这么多钱。金额很容易用Google搜索，我不会给出）。 我特别希望90年代末期的Pentium II盒式盒无法实现的梦想，而i440BX \ LX芯片组已达到了愿望清单中的处理器。 几次我继续做广告出售一台“旧电脑”，遇到了第一个插槽370的AMD Duron，Athlon和Celerons，这一切都不对。 然后，我立即通过拍摄外壳背面来切断这些选项，所有这些选项均通过彩色LPT / COM端口和内置声音（带有彩色连接器）发出。



这是我正在寻找的典型示例，旧港口尚未涂漆

尽管向前看，就在昨天，我看到了i440BX和插槽1上的“标准”主板的照片，并且该主板的背面已经有彩色连接器，因此当时有很多选择。 但是，回到我文章的英雄。 一次又一次地查看有关计算机销售的最新广告，我遇到了引起人们兴趣的相同选择。 与往常一样，在这种情况下，卖方无法提供有关填充的任何详细信息；剩下的只是依靠封闭系统单元的照片。 背面板说，这里不是奔腾II插槽，那就非常接近：两个标准PS / 2端口，两个USB，两个COM，一个LPT，所有这些都是经典的黑色。 看来这将是带有插槽1的主板，并且很可能是i440芯片组。 宾果！ 我当天安排了午餐会，坐了半天。 考虑可能的处理器（奔腾II插槽还是第一个赛扬处理器？）关于另一个填充物：什么声音和视频卡？ 照片中可以看到PCI调制解调器，由于某种原因，它突然变得有趣而重要-而不是“柔软”，是吗？ 好吧，总的来说，我和约会一样担心。 这种选择的异常价格加剧了这种情况：带有15英寸CRT显示器的系统单元的价格为3,000卢布。通常卖这种旧计算机的人只知道它们，任何人都不再需要它们，如果他们花500卢布来取货，那就太好了。考虑到这一切，我参加了一次会议，电脑在车库里，一个英俊的男子用一辆吉普车来展示它。事不宜迟，我取下了机壳（记住，这是用四个螺丝钉完成的，并从机壳的两侧完全拆下了）...它！ATrend ATC 6220主板（ATX，插槽1，AGP，4 PC I，3 ISA，3 DIMM）处理器被电源关闭了……再拧四个螺丝，然后……我看到了令人垂涎的黑色大处理器墨盒上的铭文：采用MMX技术的Intel Pentium II处理器。我几乎没有拘束自己的情绪，所以我开始交易。他以2000卢布的价格买下了我（不是漫画中的“战斗”）。



期待已久的奔腾II

第一个问题


不好意思，但是我买时甚至都没有打开它，对外部检查感到满意（电容器似乎完好无损，这个年龄段几乎没有灰尘，还不错），而且卖方保证计算机处于良好状态。 首次加入后立即后悔的是。 首先，计算机立即启动，而无需等待按下电源按钮，其次，我看到的是黑屏和扬声器的长哔哔声，而不是预期的短暂吱吱声和“能源之星”徽标。 Internet已建议Award BIOS这样报告内存问题。 顺便说一下，我的同事们有几个SDRAM内存管芯：每个两个128Mb和每个256Mb。 考虑到第98年的规模，我被如此庞大的数字吓坏了，只将其中一个放在128MB上。 我只是不再举手了（天哪，不是服务器，天哪）。 电脑发出短暂的吱吱声，发出一声三英寸的驱动器（是的，350Mhz处理器，好的，很多很多，顺其自然）。


我什至没有等待操作系统加载。 他将其关闭，然后逐渐（因工作而分散注意力），完全分解，吹干，清洗，更换（在可能的情况下）导热硅脂，小心地将所有电线放在盒中并仅在晚上打开。 为感谢我的努力，Windows 98以其迷人的启动旋律迎接了我。




一点构建过程

彻底清除电源后，计算机自行打开的问题已解决。


完整的硬盘驱动器为6Gb。 在那个时代可能是一种奢侈，但是现在我只准备了要安装的游戏光盘映像，它们占据了50 GB以上。 Victoria表示硬盘驱动器可以正常运行，但是速度很慢。 在读取测试中，速度约为每秒5兆字节。 我在带有“不必要”硬盘的架子上闲逛，发现一张没有贴纸的IDE光盘说它“死了”。 一台140Gb的Maxtor顺利通过了维多利亚测试，并且在测试中显示的平均读取速度约为每秒40-50兆字节（这是在UDMA-33上）。 但是安装此硬盘驱动器并不是那么简单。 如果您只是通过现代计算机上的“磁盘管理”来粉碎它，则Windows 98安装程序会发誓，并用fdisk粉碎它并没有给BIOS限制8Gb硬盘的容量。 在此计算机上运行的Acronis for DOS也遇到了相同的限制。 我将磁盘移到了较新的计算机上，fdisk开始看到25Gb，但仍然无法永久损坏，尝试了很多选择之后，我在那台计算机上启动了Akronis，他看到了完整的卷并可以正常分割磁盘。 结果，在BIOS中，驱动器仍定义为8 GB，但这似乎并不影响Windows性能。


显卡给我带来了下一个惊喜。 最初，我的计算机上安装了AGP视频卡Intel i740（老实说，我不认为Intel在那几年尝试加入视频加速器竞赛），我尽快将其更改为GeForce 4 MX440。 大约在同一年，仍然有Radeons，但即使对于他们，它们也没有从过时的AGP x2上开始。 安装Windows之后，我开始安装游戏，一些游戏开始正常，但是在期待已久的主题医院（对我而言）上，视频卡跌落了。 在游戏切换到320x240的那些地方出现黑屏。 漫长而失败的理解。 真可惜，似乎选择了计算机来运行此类游戏而没有不必要的手势。 我决定检查Intel i740，在这里一切都开始没有问题。 结果，她离开了她。


总结


我还要注意什么。 甚至在更换驱动器之前，在旧而缓慢的HDD上，Office XP软件包中的Word和Excel的启动速度都比带有i5 2.6Ghz和SSD的笔记本电脑的启动速度快。 有了新光盘，一切都将“飞起来”。


目前的配置是：i440BX /奔腾II 350Mhz / 128Mb RAM / 140Gb HDD /视频AGP 2x Intel i740上的主板


该系统通常非常通用。 主板由ATX供电，任何现代电源设备和机箱都可以使用（但是机箱需要一个大机箱，主板为全尺寸）。 键盘和鼠标PS / 2或USB（我没有尝试过USB键盘，我敢肯定BIOS无法理解）。 直到2000年的游戏 运行起来非常有信心，而且总是有机会启动DOS，以供需要它的游戏/程序使用。 那些年值得关注的游戏库非常庞大。 那些年中大多数最有趣的游戏都是2D制作的，因此很长一段时间以来我一直没有寻求好的3d加速器，但是我真正想要改变的是这种情况。 塑料变成黄色，看起来非常糟糕。 我正在慢慢地寻找一种情况，主要标准是白色（当时黑色情况没有时尚）。 我没有写任何关于显示器的内容，但是“我几乎把眼睛都移开了”，我几乎立即拆下了它，并安装了一个旧的方形LCD 17”。目前，所有的技术难题已经解决，并且我仍在安装游戏（我的清单很大）感谢您阅读到最后。
