IBM akan menawarkan kepada komunitas Open Compute Project suatu alternatif untuk prosesor x86


Server Rendering dari Google dan Rackspace

Perusahaan kami berencana untuk memulai pengembangan server sendiri berdasarkan pada prosesor Power dan arsitektur Open Compute Projec t (OCP). Manajer umum divisi Sistem Tenaga IBM, Doug Ballog, yakin bahwa akan segera ada permintaan untuk prosesor tersebut, dan banyak sekali. Server daya akan memiliki arsitektur terbuka, ini akan menjadi alternatif yang cukup serius untuk sistem terintegrasi IBM. Sistem ini termasuk, misalnya, Kekuatan Murni. Karena keterbukaan, pengguna server semacam itu bisa mendapatkan lebih banyak pilihan komponen internal sistem server.

Server OCP juga dapat dianggap sebagai pelengkap arsitektur server terbuka yang sudah terkenal berdasarkan chip x86. Pengguna mesin jenis ini akan dapat membuat sistem hyper-scalable, alternatif untuk platform dengan prosesor Intel.

Masih sulit untuk menyebutkan tanggal spesifik untuk kemunculan server-Power dengan arsitektur terbuka. Namun, Google dan Rackspace sudah mengembangkan server Zaius berdasarkan pada prosesor IBM Power 9. Mungkin atas dasar itulah server untuk komunitas Open Compute Project akan dibuat. Rackspace sebelumnya membuat dan mengusulkan server Barreleye baru dengan prosesor Power 8. POWER.- Arsitektur berbasis mikroprosesor dengan set instruksi terbatas (RISC), dikembangkan dan dikembangkan oleh IBM. Nama itu kemudian diterjemahkan sebagai Optimasi Kinerja Dengan Enhanced RISC (Optimasi Kinerja Berbasis pada Arsitektur RISC yang Ditingkatkan). Kata ini juga disebut serangkaian mikroprosesor menggunakan set instruksi yang ditentukan. Mereka digunakan sebagai prosesor sentral di banyak mikrokomputer, embedded system, workstation, mainframe, dan superkomputer.

Proyek Open Compute diumumkan pada tahun 2011 oleh Facebook. Tujuan dari proyek ini adalah untuk mendorong perusahaan dan pengembang individu untuk membuat dan menggunakan arsitektur peralatan standar untuk membuat pusat data yang hemat energi dan ekonomis. Proyek dimulai dengan pengembangan peralatan untuk pusat data Facebook di Praineville (Oregon, AS). Perusahaan memutuskan untuk membuat arsitektur server semacam itu benar-benar terbuka. Ini juga berlaku untuk papan server, catu daya, sasis server, dan rak. Secara umum, ini adalah analog "perangkat keras" dengan proyek Open Source. Sekarang OCP memiliki komunitas besar, yang menawarkan perkembangannya sendiri, serta menggunakan ide yang diajukan oleh orang lain.



Salah satu keuntungan utama dari server terbuka dan pusat data yang bekerja dengan peralatan tersebut adalah penghematan biaya. Akibatnya, kesempatan untuk menyediakan pelanggan dengan layanan lebih murah. Ini adalah keuntungan yang kuat dalam pasar yang sangat kompetitif seperti pusat data dan layanan cloud. Pusat data Facebook yang sama di Oregon mengkonsumsi energi 38% lebih sedikit daripada pusat data yang sebanding dengan perangkat lunak dan perangkat keras berpemilik.

Sebagian besar server OCP menggunakan prosesor Intel. Kami ingin menawarkan alternatif - Pemroses daya. Dinamika mitra yang menggunakan server IBM berdasarkan pada prosesor tersebut adalah positif. Upaya meningkatkan konsep yang diusulkan sedang berlangsung, termasuk melalui Open Power Foundation. Asosiasi ini dibuat oleh IBM, sekarang termasuk perusahaan seperti Google, Samsung dan NVidia. Beberapa produsen perangkat keras server mulai menawarkan versi sistem mereka sendiri dengan prosesor Power.

Perusahaan kami berencana untuk mengembangkan dan menawarkan server pelanggan bernilai sekitar $ 6000 atau lebih. Peralatan yang biayanya lebih murah akan diproduksi oleh perusahaan lain, termasuk Tyan, Supermicro dan lainnya. Omong-omong, sekarang server yang disesuaikan dengan arsitektur OCP diproduksi berdasarkan arsitektur Power8. Adapun server dengan prosesor Power9 baru, mereka akan muncul kemudian - sudah di paruh kedua tahun depan. Prosesor baru mendukung teknologi NVLink, di mana throughput lima kali lebih tinggi dari PCI-Express 3.0.
