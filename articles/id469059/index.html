Apa yang baru di ML.NET dan Model Builder

Kami sangat senang mengumumkan pembaruan untuk Model Builder dan peningkatan di ML.NET . Anda dapat mempelajari lebih lanjut di "Apa yang baru di ML.NET ?" sesi di .NET Conf .

ML.NET adalah kerangka pembelajaran mesin sumber terbuka dan lintas platform (Windows, Linux, macOS) untuk pengembang .NET.

ML.NET menawarkan Model Builder (alat UI sederhana) dan CLI untuk membuatnya sangat mudah untuk membangun Model ML kustom menggunakan AutoML.

Dengan menggunakan ML.NET , pengembang dapat memanfaatkan alat dan keterampilan yang ada untuk mengembangkan dan menanamkan AI kustom ke dalam aplikasi mereka dengan membuat model pembelajaran mesin kustom untuk skenario umum seperti Analisis Sentimen, Rekomendasi, Klasifikasi Gambar, dan banyak lagi! ..



Artikel ini di blog kami .

Berikut ini adalah highlights utama:

Pembaruan Model Builder


Rilis Pembuat Model ini menambahkan dukungan untuk skenario baru dan mengatasi banyak masalah yang dilaporkan pelanggan.



Rekayasa fitur: Di versi sebelumnya dari Model Builder, setelah memilih dataset Anda, baik dari file atau dari SQL Server, Anda hanya memiliki pilihan untuk memilih kolom yang akan diprediksi (Label). Setiap kolom lain dalam dataset secara otomatis digunakan untuk membuat prediksi (Fitur). Setiap kolom yang tidak ingin Anda sertakan, Anda harus memanipulasi dataset Anda di luar Model Builder dan kemudian mengunggah dataset yang dimodifikasi.



Konsumsi model menjadi mudah!: Di versi sebelumnya dari Model Builder, ada banyak langkah yang harus Anda ambil setelah pembuat Model dan pembuatan model untuk menggunakan model yang terlatih di aplikasi Anda, termasuk menambahkan referensi ke proyek perpustakaan yang dihasilkan, mengatur model Salin ke properti Output ke "Salin Jika Baru," dan menambahkan paket Microsoft.ML NuGet ke aplikasi Anda.

Ini semua telah disederhanakan dan otomatis, jadi sekarang yang harus Anda lakukan adalah menyalin + menempelkan kode dari Langkah Berikutnya di Pembuat Model, dan kemudian Anda dapat menjalankan aplikasi dan mulai membuat prediksi!

Alamat umpan balik pelanggan: Rilis ini juga menangani banyak masalah yang dilaporkan pelanggan tentang kesalahan pemasangan, umpan balik kegunaan dan peningkatan stabilitas, dan banyak lagi. Pelajari lebih lanjut di sini .

Pembaruan ML.NET


Ini adalah ringkasan singkat dari fitur dan perangkat tambahan yang ditambahkan ke ML.NET selama beberapa bulan terakhir.

  • Dukungan untuk .NET Core 3
  • Dukungan untuk skenario baru seperti Perkiraan Penjualan, Deteksi Anomali
  • Pratinjau: Pemuat basis data asli yang memungkinkan pelatihan secara langsung terhadap basis data relasional
  • Pratinjau: Buat model pembelajaran mendalam khusus untuk klasifikasi Gambar menggunakan TensorFlow.
