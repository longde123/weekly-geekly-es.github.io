在任何Openstack兼容主机上创建基于3CX云的PBX

您通常需要在云中安装3CX PBX，但是您选择的云提供商未包含在3CX 支持的提供商的列表中（例如Mail.ru Cloud Solutions）。 不用担心！ 做到这一点并不难，您只需找出提供者是否支持Openstack基础架构。 3CX等公司赞助了 Openstack 的开发，并支持Openstack API和用于监视和管理的标准Horizo​​n接口。

因此，要安装3CX，您需要：

  • 帐户已激活的符合Openstack API的云提供商
  • PBX Express服务的API访问凭据，包括Openstack API URL

考虑使用Dreamhost提供程序的示例安装3CX云。

创建一个OpenStack托管帐户


1.转到https://www.dreamhost.com/，然后转到“主机和服务器”>“云”部分。 在打开的页面上，单击“ DreamCompute入门”。



2.填写您的凭据。 Compute API需要一个单独的密码，PBX Express服务使用该密码与Dreamst Openstack平台交换数据。 任何Openstack提供程序都将请求大约相同的信息。



3.注册后，您将收到一封电子邮件，其中包含验证和激活帐户所需的信息。 通常，运营商会要求您扫描护照或信用卡信息。

3CX PBX Express的OpenStack帐户配置


4.验证帐户后，转到控制面板，转到“云服务”部分，然后单击“ DreamCompute”。



5.在这里，您将看到租户ID和用户名（租户ID和用户名）。 修复它们-通过PBX Express服务安装3CX时将需要此数据。 然后单击查看仪表板。



6.在控制面板中，转到“访问和安全性”>“访问API”页面。 在这里，在“身份”行中修复URL-在该URL上，PBX Express访问Dreamhost OpenStack云。


7.在“访问和安全性”部分中，转到“密钥对”选项卡，然后单击“创建密钥对”。



8.在出现的窗口中，输入密钥对的名称，然后单击创建密钥对。 密钥将创建并下载到磁盘。



通过PBX Express安装3CX Cloud


9.现在，您可以通过3CX PBX Express服务安装基于3CX云的PBX。 在PBX Express向导的一个阶段，将显示对Openstack托管参数的请求。 使用这些参数，PBX Express会自动创建安装了3CX的VPS服务器。 为Dreamhost指定了以下参数：

  • 选择托管-VPS云提供商（符合OpenStack API v2）
  • Cloud Provider API URL-步骤6中的URL
  • 用户名-步骤5中的用户名
  • 密码-步骤5中的房客ID



10.在下一页上，如果您不知道要指示什么，只需单击“下一步”。 如果遵循这些参数，则可以指定服务器位置，服务器容量和使用的SSH密钥对。 要安装的映像只能是Debian-9。


然后单击下一步并输入您的个人数据。 片刻之后，基于云的PBX将准备就绪-您将通过电子邮件收到适当的通知。 保存它-它包含重要信息。

现在，您可以继续配置PBX-连接用户 ， 外部线路等。
