Alat-alatnya


  • z7zmey / php-parser v0.5.0 - Parser PHP yang diimplementasikan pada Go terus berkembang.
  • nesk / rialto - Alat untuk mengelola sumber daya Node.js dari PHP. Ini dapat digunakan untuk mengimplementasikan interaksi dengan perpustakaan Node.js.
  • nesk / puphpeteer - Contoh menggunakan nesk / rialto, pembungkus PHP untuk bekerja dengan GoogleChrome / dalang (Headless Chrome).
  • mlambley / swagception - Menghasilkan tes penerimaan untuk Codeception berdasarkan spesifikasi Swagger 2.0 (Open API 2.0) Anda.
  • RobThree / TwoFactorAuth - Otentikasi dua faktor berdasarkan kata sandi TOTP satu kali atau kode QR.
  • OwenMelbz / forget-db - Alat sederhana untuk menganonimkan (misalnya, GDPR) data dalam database.
  • gordalina / cachetool - alat CLI untuk mengelola apc dan opcache.
  • digitaldonkey / ethereum-php - Antarmuka yang diketik untuk Ethereum JSON-RPC API.
  • SocialConnect / auth - Perpustakaan untuk otentikasi melalui sosial. layanan (OAuth1 / 2, OpenID, OpenIDConnect). 25+ penyedia di luar kotak.
  • xobotyi / rsync - Pembungkus berguna di atas rsync untuk PHP 7.1+. Dikirim oleh hobotyi .
  • krakjoe / idbg - Seorang debugger untuk PHP yang diimplementasikan dalam PHP, sejauh ini, memerlukan ekstensi krakjoe / idbg . Posting dengan deskripsi.
