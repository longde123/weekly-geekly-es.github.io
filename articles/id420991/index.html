Menggunakan Ctrl / ⌘ + Klik untuk Menavigasi Riwayat Penelusuran


Semuanya terlihat membingungkan di sini, tetapi dalam kenyataannya semuanya sederhana. Sekarang, saat menavigasi riwayat penjelajahan suatu situs web, Anda dapat membuka halaman sebelumnya atau berikutnya di tab baru. Ini dilakukan seperti ini.

Saat melihat situs dan menavigasi halaman-halamannya, Anda ingin kembali ke halaman sebelumnya, tetapi tidak ingin menutup yang sekarang. Untuk melakukan ini, cukup gerakkan kursor mouse ke tombol "Kembali" di antarmuka browser dan tekan sambil menahan tombol Ctrl (atau ⌘ untuk MacOSX). Ini bekerja dengan cara yang sama untuk meningkatkan riwayat perambanan. Ini fitur yang nyaman.
