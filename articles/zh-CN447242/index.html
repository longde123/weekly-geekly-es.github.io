现在我们可以使用Data.SqlClient.SqlBulkCopy向它写入跟踪信息 -上面已经给出了一个示例。 是的，最好屏蔽TextData中的常量： 

 # mask data foreach ($Row in $Result) { $v = $Row["TextData"] $v = $v -replace "'([^']{2,})'", "'str'" -replace "[0-9][0-9]+", '999' $Row["TextData"] = $v }

我们用多个字符替换长度为999的数字，并用'str'替换长度超过一个字符的字符串。 从0到9的数字通常用作标志，我们不使用它们，也不使用空字符和单字符字符串-其中经常出现'Y'，'N'等。

在平板电脑中，您通常要突出显示需要注意的单元格。 例如，失败，高级别的碎片等。 当然，这也可以在裸露的SQL上完成，使用PRINT形成HTML，在Jenkins中，您可以设置HTML文件类型：
