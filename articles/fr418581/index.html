React Basics (manuel, 2e édition)

2,5 ans se sont écoulés depuis la publication de la première version du tutoriel détaillé sur les bases de React. Pendant ce temps, gitbook a compté plus de 250 000 visiteurs uniques.



Sous le chat plus sur le tutoriel et une liste de changements.

Deuxièmement - c'est la même avare sur le manuel "eau". Texte, images, code source.


Dans le processus d'apprentissage, un scénario simple est décrit: un fil d'actualités (à partir d'un fichier json) + la possibilité d'ajouter des nouvelles. Pas de flirt avec le backend. La plupart du développement va directement à index.html, et seulement à la fin nous nous familiarisons avec create-react-app .


Le but de ce tutoriel est de présenter React à ceux qui sont déjà familiers avec html / css et ont une compréhension des bases de js. La bibliothèque est ajoutée en tant que script normal. C'était pratique au moment de jQuery et je n'ai pas commencé à refaire la formule d'entraînement.
