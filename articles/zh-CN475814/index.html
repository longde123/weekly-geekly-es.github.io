Docker转型：在Mirantis中销售Docker Enterprise和更新路径

昨天，与Docker Inc公司进行了一系列转型，其背后是同名的流行容器解决方案。 可以肯定地说，他们已经等待了一段时间。 毕竟，随着Docker的广泛传播，用于容器化的其他技术的发展以及Kubernetes的迅速普及，对Docker Inc.的产品和业务整体的质疑也越来越多。



答案是什么？ 正如其中一则信息标题上的标题所言，“ Unicorn [一家市值超过10亿美元的公司]倒闭了：Docker放弃了企业。” 这就是做出这样声明的原因...

Mirantis收购Docker Enterprise业务


昨晚的主要事件是Mirantis 宣布该公司将从Docker Inc.收购其关键业务Docker Enterprise Platform：

“ Docker Enterprise是允许开发人员无缝构建任何应用程序，共享它并在任何地方安全运行的唯一平台：从公共云到混合云。 财富100强企业中有三分之一使用Docker Enterprise作为创新平台。”

同一篇新闻稿还说，Mirantis收购的Docker Enterprise团队将继续开发和支持该平台 ，并实现企业客户期望的新功能。 顺便说一句，Mirantis包括后者作为一种免维护的即服务方法， 与Mirantis Kubernetes和其他云技术集成 ，以及企业部门公认的业务模型。


摘自今年四月底发布的Docker Enterprise 3.0

Mirantis在2013年宣布了其流行的云平台OpenStack的发行，此后（直到最近）在专业社区中已将此产品与之关联。 但是，在2016年底，该公司推出了Kubernetes培训和认证计划，随后采取了其他步骤（例如， 宣布推出基于 K8s的Mirantis Cloud Platform CaaS-容器即服务-），这清楚地表明了公司的重点转移了朝K8s 。 迄今为止，Mirantis一直是 Kubernetes代码库中贡献最大的20家公司之一。


Kubernetes for MCP （Mirantis Cloud Platform）-Mirantis当前CaaS解决方案的继承人

Mirantis首席执行官兼联合创始人Adrian Ionel的评论：

“ Mirantis Kubernetes技术与Docker Enterprise Container Platform一起为基于云的公司带来了简便性和选择。 它作为服务交付，是针对新应用程序和现有应用程序的最简单，最快的云基础架构方式。 Docker Enterprise员工是全球最有才华的云专家之一，他们可以为自己的成就感到自豪。 我们非常感谢有机会共同创造一个令人兴奋的未来，并欢迎Docker Enterprise团队，客户，合作伙伴和社区。”

如果到目前为止Mirantis大约有450名员工，那么新的收购将使员工人数大幅增加 -300人。 但是，根据Adrian所说，Docker的营销和销售团队将首次独立运作，因为Mirantis的目标是使所有客户的过渡尽可能顺利。

尽管Mirantis和Docker Enterprise的客户基础有所重叠，但两家公司之间的交易将为Mirantis带来约700个新的企业客户 。

在11月21日的网络研讨会上将详细了解Mirantis对产品未来的愿景-Docker Enterprise平台将如何与该公司现有的解决方案结合使用。

Docker Inc本身将 Docker Enterprise 的出售称为针对开发人员的公司生命中的新篇章 。

“ Docker的未来重点是利用现有基础为专注于现代应用程序的开发人员改善工作流程。”

“基础”下的是指在公司生命周期内创建的解决方案，例如实际的Docker CLI实用程序，Docker桌面和Docker Hub。 简而言之， Docker Inc.现在将专注于直接针对开发人员的产品开发 （ Docker Desktop和Docker Hub ）。

IT资深人士和开放源代码记者Matt Asay 在此声明中的评论如下：

“我不理解“出售我们的企业业务以专注于开发人员”的论点，因为开发人员是企业中的关键购买者/影响力，但我只希望Mirantis和Docker取得最佳发展。”

得益于其管理层的评论，Docker Inc.的行为变得更加易于理解。 这些变化影响了他。

重组Docker Inc和新任CEO


因此，出售Docker Enterprise并不是昨天Docker Inc.生命中的唯一事件。 与此同时，该公司宣布追加投资并任命新的首席执行官。

从Benchmark Capital和Insight Partners那里获得了3500万美元的投资，他们之前已经投资了该公司。 这个数量非常重要：

  • 自Docker公司成立以来（2010年）的总投资额约为2.8亿美元;
  • Docker最近在吸引新投资方面遇到了问题。

该公司已变更为首席执行官，这是今年第二次。 直到昨天，Docker公司仍由Rob Bearden（Hortonworks的前首席执行官）领导，他于5月被任命为该职位。 已经重组的公司的新负责人是斯科特·约翰斯顿 （ Scott Johnston） ，自2014年以来一直在Docker Inc.工作。 他先前的职位是CPO （首席采购官）。


Docker Inc.新任首席执行官Scott Johnston， GeekWire摄

公司前任首席执行官（Rob Bearden）对近期事件的评论：
“我加入了Docker，领导公司的下一个开发阶段。 经过与管理团队和董事会的全面分析，我们发现Docker有两个非常独特且本质上不同的业务：针对开发人员的活跃业务和针对企业的成长型业务。 我们还发现产品和财务模型非常不同。 这导致我们决定重组公司并拆分两个业务，这对客户而言应该是最好的解决方案，并使Docker成功发展成为市场上的领先技术。

开发人员正在积极使用Docker遗留物，因此，分析之后，自然的决定是将Docker的关注点重返该社区，这对我们来说至关重要。 做出决定后，我意识到Scott Johnston是重组公司首席执行官职位的理想人选。 斯科特（Scott）在初创公司早期阶段的产品开发中的强大经验正是Docker领导者所需要的。 感谢Scott同意担任新职务。 我们与他合作，以确保软转型。”
