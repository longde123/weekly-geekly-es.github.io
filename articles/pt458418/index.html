5 extensões de navegador para trabalhar com texto em inglês



Olá pessoal! Continuo compartilhando minha experiência profissional no tópico de trabalhar com textos em inglês. Hoje falaremos sobre quais extensões de navegador podem ser usadas para facilitar a criação de materiais de alta qualidade e aprender a escrever melhor em inglês. Vamos lá!

Nota : o texto contém extensões para o Google Chrome e Mozilla Firefox, porque eu uso apenas esses navegadores. Nem todos os produtos mencionados têm uma versão para os dois navegadores.

Linguix


Este serviço é um pouco semelhante ao Grammarly - também encontra erros de digitação e gramaticais nos textos, mas custa quase metade do preço. Também existem diferenças - por exemplo, um banco de dados de modelos de conteúdo que pode ser usado para escrever textos de categorias diferentes mais rapidamente (por exemplo, press releases ou notícias corporativas).



Dos recursos que eu gosto - o módulo de cálculo estatístico. O sistema analisa a “legibilidade” do texto e informa quão complexo é. Útil para combater a tentação de tornar o texto muito volumoso.



Outra coisa conveniente é o modo de edição de pasta na versão web, que funciona como um bate-papo secreto no messenger. Após escrever e editar o texto, ele é completamente excluído.

Versão para Chrome | Versão do Firefox

Escritor


Um aplicativo de cabeçalho para todos que trabalham com textos em inglês usando um navegador. Esta extensão permite abrir no Chrome um editor extremamente conveniente, altamente personalizável e que não distrai. Parece muito legal:



O Linguix funciona no editor - para que você possa procurar imediatamente erros

De decisões bem-sucedidas - a extensão salva o texto automaticamente de tempos em tempos. Você pode definir metas, obter facilmente estatísticas sobre o número de caracteres no texto, etc. Apesar de o aplicativo ter uma página separada no repositório de extensões do Chrome, ele é redirecionado para a versão web do serviço e você pode usá-lo em qualquer navegador.

Versão do navegador Writer

Dicionário do Google


Um dicionário do Google permite traduzir uma palavra desconhecida - pode ser útil ao trabalhar com materiais em inglês para preparar texto. Com esta extensão, você pode aprender facilmente o significado de qualquer palavra que encontrar em uma página da web. Você só precisa destacar uma palavra e um pop-up com uma tradução será exibido.



Uma coisa conveniente - o programa pode pronunciar palavras em voz alta, para que você possa puxar a pronúncia. Acontece que o dicionário do Google é útil não apenas para o trabalho direto no texto, mas também como uma ferramenta para aprender o idioma.

Versão do Chrome

Leitor escuro


Para ser sincero, muitos de nós estão constantemente sentados no computador até a noite, inclusive na escuridão total. Alguns, por exemplo eu, podem até trabalhar nesse formato, inclusive em textos.

Escusado será dizer que a maioria dos editores de texto da Web são páginas brancas brilhantes com letras pretas. No escuro, essa é uma fadiga ocular extremamente grave, então eu uso a extensão DarkReader. Escurece o plano de fundo nas páginas da Web, por isso é muito mais agradável trabalhar com eles à noite. Mantenha o foco


A procrastinação é um dos principais inimigos de todos que pelo menos uma vez tentaram trabalhar em textos. Escrever um texto particularmente longo sem se distrair com as redes sociais, correio e notícias é uma tarefa extremamente difícil. A extensão StayFocusd me ajuda a resolvê-lo.



Com ele, você pode impedir-se de sair de um site específico por um tempo. Como resultado, se você precisar escrever uma postagem no blog, até que você escreva, não irá ao facebook. Com o início do uso dessa extensão, minha produtividade aumentou acentuadamente. Eu recomendo.
