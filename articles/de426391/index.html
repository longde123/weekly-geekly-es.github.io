PHP-Interna


  • RFC: Spread Operator in Array - In einem Entwurf wird die Möglichkeit des Hinzufügens eines Operators ... für Arrays erörtert.
    Versteckter Text 
     $parts = ['apple', 'pear']; $fruits = ['banana', 'orange', ...$parts, 'watermelon']; // ['banana', 'orange', 'apple', 'pear', 'watermelon'];
