Kontrol suara di rumah pintar Z-Wave dan sekitarnya. Homekit + Siri. Instruksi manual


Untuk membangun kenyamanan maksimal di apartemen setiap bulan menjadi lebih murah dan lebih mudah. Dalam panduan langkah demi langkah ini, saya akan menunjukkan kepada Anda betapa mudahnya menambahkan kontrol suara ke rumah Anda! Untuk kontrol suara kita perlu:
  1. Pengontrol RaZberry atau komputer apa pun dengan perangkat lunak Z-Way
  2. Sepasang perangkat Z-Wave, serta bola lampu WiFi apa pun
  3. iPhone / iPad (lebih disukai di dinding)

Saya menggunakan Raspberry Pi B +, jadi deskripsi pengaturan kontrol suara di atasnya akan dijelaskan nanti. Pekerjaan video di akhir artikel.

1) Instal perangkat lunak versi Z-Way yang lebih lama dari 2.2.2 :
wget -O - razberry.z-wave.me/install/v2.2.2-test-new-webserver-8-ga9cd319 | sudo bash
Z-Way adalah Hub untuk Z-Wave, perangkat Enocean, berbagai bola lampu WiFi, soket, kunci, dan IoT lainnya yang dikendalikan oleh permintaan HTTP dan TCP. Antarmuka web tersedia pada port 8083.


2) Instal versi terbaru Node.js
wget https://nodejs.org/dist/v4.4.5/node-v4.4.5-linux-armv6l.tar.xz
tar -xvf node-v4.4.5-linux-armv6l.tar.xz
cd node-v4.4.5-linux-armv6l
sudo cp -R * /usr/local/
Node.js adalah platform untuk menjalankan kode JavaScript pada server berbasis mesin Google V8.


3) Instal plugin Homebridge dan Homebridge-zway
sudo npm install -g homebridge
sudo npm install -g homebridge-zway
Homebrige mengimplementasikan dukungan Homekit. Homebrige-zway adalah sebuah plugin yang menambahkan peningkatan dukungan Homekit ke Z-Way.


4) Buat konfigurasi Homebrige /usr/local/lib/node_modules/homebridge/config.json
{
    "bridge": {
        "name": "Homebridge",
        "username": "CC:22:3D:E3:CE:31",
        "port": 51826,
        "pin": "031-45-154"
    },
    "platforms": [
    {
            "platform": "ZWayServer",
            "url": "http://localhost:8083/",
            "login": "admin",
            "password": "admin"
    }
    ]
}
Potongan terakhir di url sangat penting, tanpa itu tidak akan berhasil.

5) Luncurkan Z-Way dan Homebridge
sudo /etc/init.d/z-way-server start
sudo homebridge -U /usr/local/lib/node_modules/homebridge &
Jika semuanya benar, maka dalam log Anda akan melihat perangkat yang dapat Anda kontrol dan kode PIN.


6) Menggunakan aplikasi Rumah di ponsel atau yang lain ( MyTouchHome , MyHome App , dll.) Kami menambahkan perangkat Homekit.


7) Dilakukan! Sekarang rumah bisa dikontrol dengan suara! Kontrol yang nyaman!


PS Terakhir kali saya menggunakan modul Homekit asli untuk Z-Way untuk kontrol suara , tetapi saya lebih menyukai Homebridge, karena memungkinkan bekerja tidak hanya dengan Z-Way, tetapi juga dengan sistem otomasi lainnya, seperti Philips Hue , Nest dan banyak lainnya.
