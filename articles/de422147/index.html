Erhöhen Sie Ihren Speedtest-Knoten

Wenn Sie die Geschwindigkeit Ihrer Internetverbindung messen müssen, öffnen Sie höchstwahrscheinlich speedtest.net. Mit Speedtest können Sie Ihre eigenen Messknoten erstellen, die andere Benutzer verwenden können. Dies ist beispielsweise für Internetprovider sehr wichtig (der Haupt-Speedtest-Server sollte hinter dem Gerät platziert werden, wodurch die Geschwindigkeit für Abonnenten verringert wird).

Unter katom ausführliches Installationshandbuch.

Systemanforderungen Entwicklerangebot ^



Speedtest unterstützt viele Betriebssysteme:

  • Windows Server (2008 oder 2012 mit IIS 6, 7, 7.5 oder 8)
  • Linux (2.6.18 Kernel oder neuer)
  • Mac OS X (basierend auf 10.8, frühere Versionen funktionieren möglicherweise, werden jedoch nicht vollständig unterstützt)
  • FreeBSD (Kernel 7.3 oder neuer)

Ich habe Ubuntu 18.04 verwendet.

Also lass uns in Ordnung gehen. Zunächst müssen Sie die Anwendung für HTTP Legacy Fallback und den Hauptserver für Speedtest herunterladen.

Für HTTP Legacy Fallback ist ein Webserver erforderlich. Ich habe Apache + PHP verwendet. Entpacken Sie den Inhalt von http_legacy_fallback.zip in / var / www / html .

In meinem Fall benötigte ich aus dem Archiv OoklaServer.tgz das Verzeichnis OoklaServer-linux64 . Packen Sie es an den Ort aus, an dem Sie Programme von Drittanbietern haben, z. B. / opt / speedtest .

Als Nächstes müssen Sie OoklaServer starten und sicherstellen, dass es beim Systemstart gestartet wird.
Überprüfen Sie, ob es funktioniert hat. Sie müssen http: // <Ihre IP>: 8080 im Browser öffnen

Wenn alles in Ordnung ist, wird die folgende Meldung angezeigt:


Schauen Sie sich jetzt das Legacy Fallback HTTP an. Öffnen Sie http: // <Ihre IP> /speedtest/upload.php

Es sollte so etwas wie size=500

Danach muss der Knoten in Speedtest registriert werden. Dazu müssen Sie sich unter https://account.ookla.com/ registrieren und zum Abschnitt "Speedtest-Server" gehen. Danach können Sie auf "Server hinzufügen" klicken. Ein Fenster mit einem Fragebogen wird geöffnet:



Sie müssen einen Fragebogen ausfüllen und warten, bis der Server in der allgemeinen Liste angezeigt wird.

Ungefähr einen Tag später wurde er bei Abonnenten unseres Netzwerks bereits am Eingang zu speedtest.net automatisch ausgewählt.
