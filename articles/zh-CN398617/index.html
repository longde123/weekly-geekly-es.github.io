由内而外的笔记本电脑：华硕ZenBook Flip笔记本电脑评测



您还记得，不久前在Zen系列中，出现了许多华硕的新智能手机和笔记本电脑。这次我建议结识ZenBook Flip（UX360UA）笔记本电脑。通过将盖子旋转360度并将其变成功能强大的平板电脑，可以将其“内翻”。

外表


外壳和盖子均完全由铝制成。在盖子的整个区域，都应用了Zen系列的常用企业图案。颜色-纯铝，但具有浅金色。该版本将在销售中并显示为深灰色，如标题图片所示。



在铰链侧，盖子的厚度异常大，没有比主体本身薄很多：



但是，这种厚度仅在底部与主体的交界处。在其他三个侧面，盖子要薄得多。





颜色大而凸的抛光环引人注目。铰链的坚固性激发了人们希望，它们确实能够承受比传统笔记本电脑更高的重复打开和关闭负荷：在测试过程中，铰链至少可持续20,000次循环。如果您同意所有者每天打开盖子20次，那么您将获得大约三年的保证运行。

在右侧位于：

  • 读卡器
  • 音频插孔
  • USB Type C 2.0端口
  • HDMI端口
  • USB端口
  • 电源连接器。



左侧是：

  • USB端口
  • 强大的通风系统格栅
  • 声音调整和电源按钮。

后者看起来很不寻常，但是直到将UX360UA护盖变成平板电脑模式之前。





笔记本电脑打开360度时，底部的橡胶垫脚也可以用作保险杠。





在我看来，这些环比普通笔记本电脑更紧密地工作。考虑到需要在整个打开范围内可靠地固定盖子，这是合乎逻辑的。

UX360UA的显示屏覆盖有光滑的玻璃。在这种情况下，无法逃脱：平板电脑模式涉及触摸控制。



无论施加力的点如何，按键都非常顺畅地按下，不会变形。尽管触摸板上的按钮对我来说似乎很苛刻。当机盖切换到平板电脑模式时，键盘和触摸板将关闭，因此您不必担心会意外点击。







展示架


将会发售三种版本的UX360UA，其中包括显示器：两个版本的色域为NTSC的72％，分辨率为3200x1800和1920x1080，第三个版本的分辨率为1920x1080和色域为45％。在所有三种情况下，都使用IPS矩阵。我的版本具有72％NTSC覆盖率和最大分辨率。不幸的是，我没有准确评估色彩再现性的设备，因此我只能凭主观判断进行操作。我认为，UX360UA的显色性很高。对于平板电脑来说，宽视角比普通笔记本电脑重要得多，因为在平板电脑模式下使用时，设备的倾斜度变化更大且在更宽的范围内。从这个角度来看，UX360UA显示矩阵也没有任何抱怨。

但是在我看来，这种对角线的分辨率过高。是的，图像非常平滑-276 ppi-并非每只眼睛都会注意到像素。但是，如此高的分辨率仍然会给集成显卡和电池带来额外的负担。因此，如果图像的平滑度和清晰度对您来说比对可能带来的不便更为重要（例如，如果您处理和分析照片），那么分辨率为3200x1800的UX360UA值得购买。如果您不需要图片具有“模拟”平滑度，即使您使用放大镜观看，也请选择具有传统Full HD显示屏的型号。

音效


在最近审阅过的ZenBook 3超级本中，我提到了Harman Kardon工程师可以手工创建的声音系统，如键盘下方的雕刻所示。这是相同的题词，只涂在上面。小型扬声器的尺寸听起来不错，但最好的选择当然是通过优质的耳机收听UX360UA。在这种情况下，声音更具说服力和令人愉悦。



绩效和工作经验


UX360UA使用Core i7 6500U处理器（还将提供带有i5 6200U的版本）。RAM的最大容量为16 GB（1866 MHz）。



对于图形符合集成芯片Intel HD Graphics 520。



一个驱动器SSD，容量为256或512 GB。SATA连接3。





硬件生产效率很高，UX360UA可以智能地工作，并且可以轻松地处理大量应用程序。为了进行验证，我在其上启动了Adobe Lightroom，并对几张大照片应用了一堆滤镜。尽管内置了视频卡，但一切工作都很快，就像在体面的台式计算机上一样。但是在UX360UA上玩沉重的游戏不太可能成功，图形也不会消失。是的，并非出于此目的。

华硕声称UX360UA可以使用电池供电长达12小时。当然，为此，您将必须立即打开省电模式，将屏幕调至最小亮度，并仅读取文本文件。但是，即使是通常的用例-上网，听音乐，定期观看YouTube上的视频-笔记本电脑也可以工作约8个小时。

结论


尽管UX360UA正式指的是笔记本电脑，但实际上它是一个人中的变压器，笔记本电脑和平板电脑。也就是说，该设备将主要吸引那些经常需要操作平板电脑的便利性和笔记本电脑的舒适性来处理文本的人。而且不会影响性能。在我看来，基本上这款笔记本电脑将用于公司领域，需要在旅行中使用文本和演示文稿进行积极工作。对于个人（家庭）使用，您还可以选择合适的方案。例如，在旅途中，在平板电脑上而不是在笔记本电脑上观看电影会更加方便，尤其是在汽车或飞机上的可用空间不足时。使用平板电脑躺在沙发上也更方便。如果您想与社交网络上的某人聊天，那么您不必戳虚拟键盘或使用笔记本电脑或台式机，只需将机盖扩展到笔记本电脑模式即可。
