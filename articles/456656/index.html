Lecciones sobre SDL 2: Lección 4 - Estirando PNG

Hola a todos! Esta es la cuarta lección en SDL 2. Decidí combinar las dos lecciones en una sola, ya que en el original son pequeñas. Pero se pueden encontrar aquí y aquí . Bueno, comencemos la lección.

Primero, decidiremos qué haremos. Hoy aprenderemos cómo cargar imágenes de otra extensión (no BMP), a saber: PNG. Vuelva a hacer la imagen en un formato diferente para acelerar el trabajo y cambiar el tamaño de la imagen. Trabajaremos con esta imagen:


Instalación


Entonces, primero aprendemos cómo descargar PNG. El conjunto de características estándar de SDL 2 no carga imágenes que no sean de formato BMP. Para descargar un formato diferente, debe instalar la extensión.

Aquí están las instrucciones:

Ventanas
Vaya a esta página y descargue los archivos ZIP o TAR.GZ de la sección Bibliotecas de desarrollo . Desempaquete y tírelos a la carpeta con el compilador.

Linux
Si su sistema usa Debian, entonces necesita ingresar estos comandos: 

apt-cache search libsdl2-image apt-get install libsdl2-image-dev

Si se utiliza Yellowdog Updater, Modified, entonces ingresa estos comandos: 

 yum search SDL2_image-devel yum  SDL2_image-devel


Mac OS
Descargue el archivo DMG y lea el archivo Léame.

Escribir un código


Después de configurar SDL_image, comenzamos a escribir código. Primero, como siempre, conectaremos las bibliotecas, estableceremos los tamaños de las ventanas y crearemos 3 variables globales. 

 #include <SDL2/SDL.h> #include <SDL2/SDL_image.h> #include <iostream> const int SCREEN_WIDTH = 640; const int SCREEN_HEIGHT = 480; SDL_Window *win = NULL; SDL_Surface *scr = NULL; SDL_Surface *flower = NULL;

Tenga en cuenta que conecté SDL_image.h por separado, ya que es una biblioteca separada.

A continuación, escribimos 3 funciones, conocidas por nosotros, en las que agregaremos y cambiaremos algo, como de costumbre.

Init () 
 bool init() { if (SDL_Init(SDL_INIT_VIDEO) != 0) { std::cout << "Can't init video: " << SDL_GetError() << std::endl; return false; } int flags = IMG_INIT_PNG; if ( !( IMG_Init( flags ) & flags ) ) { std::cout << "Can't init image: " << IMG_GetError() << std::endl; return false; } win = SDL_CreateWindow(" PNG", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN); if (win == NULL) { std::cout << "Can't create window: " << SDL_GetError() << std::endl; return false; } scr = SDL_GetWindowSurface(win); return true; }


Aquí agregamos la inicialización del módulo SDL_image. Para inicializarlo, primero creamos una variable int , tenemos banderas y guardamos allí todas las banderas que desea usar para la inicialización.

Lista de todas las banderas
SDL_INIT_PNG : inicializa el descargador de imágenes PNG;
SDL_INIT_JPG : inicializa el descargador de imágenes JPG;
SDL_INIT_TIF : inicializa el descargador de imágenes TIF;

A continuación, escribimos esta condición: 

 if ( !( IMG_Init( flags ) & flags ) )

La función IMG_Init () devuelve los indicadores que pudo inicializar. Si ella devolvió las banderas, pero faltaba la que queríamos, devolvemos un error. Muy simple Además, tenga en cuenta que uso IMG_GetError en lugar de SDL_GetError , ya que estamos buscando un error en la función del módulo SDL_image , no SDL .

También quiero señalar que decidí usar el tipo bool para devolver a - esta es una forma más racional.

Continuemos escribiendo nuestro programa, escribamos la función Load () , en la que cargamos una imagen PNG y la convertimos.

Carga 
 bool load() { flower = IMG_Load("flower.png"); if (flower == NULL) { std::cout << "Can't load: " << IMG_GetError() << std::endl; return false; } flower = SDL_ConvertSurface(flower, scr->format, NULL); if (flower == NULL) { std::cout << "Can't convert: " << SDL_GetError() << std::endl; return false; } return true; }


Aquí cargamos la imagen con la nueva función IMG_Load . Su función es la misma que la de SDL_LoadBMP y el resultado es el mismo: devuelve una instancia de la clase SDL_Surface . Para encontrar errores durante la carga, también utilizamos IMG_GetError .

Es hora de convertir. Primero, una pequeña teoría. Pocas personas lo saben, pero al descargar una imagen, se descarga en formato de 24 bits, y la mayoría de las pantallas modernas están en formato de 32 bits. Y cada vez que mostramos la imagen en la pantalla, primero se rehizo en formato de 32 bits. Para programas simples, esto no importa, pero al crear algún tipo de proyecto grande, afectará en gran medida el rendimiento, por lo que tomaremos y en la etapa de carga de imágenes las rehaceremos en el formato de visualización. La función SDL_ConvertSurface ayudará. Toma 3 valores: la superficie que queremos formatear, el formato en el que queremos formatear y las banderas. queremos formatear la imagen de la flor , pasarle el primer parámetro. El segundo parámetro es el formato de la superficie de la ventana, no utilizaremos banderas. Esta función devuelve una copia de la superficie, que asignamos inmediatamente a flower . Para encontrar errores, tomamos la función SDL_GetError () , porque ya estamos trabajando con superficies, y son de SDL , y no de SDL_image .

A continuación, escribimos la función Salir, agregamos solo una función.

Salir 
 void quit() { SDL_FreeSurface(flower); SDL_FreeSurface(scr); SDL_DestroyWindow(win); SDL_Quit(); IMG_Quit(); }


Agregamos IMG_Quit para no inicializar el módulo SDL_image.

Principal


Ahora queda lo más simple: la función principal . 

 int main (int argc, char ** args) { if (!init()) { system("pause"); quit(); return 1; } if (!load()) { system("pause"); quit(); return 1; }

Para comenzar, inicialice y cree todo lo que necesita y cargue los archivos necesarios. No nos detendremos aquí. 

  SDL_Rect bg_flower; bg_flower.w = SCREEN_WIDTH; bg_flower.h = SCREEN_HEIGHT; bg_flower.x = 0; bg_flower.y = 0;

Después de eso, cree un objeto de la clase SDL_Rect . Lo necesitaremos para estirar la imagen. Si observa, la imagen de la flor es 4 veces más pequeña que la ventana (2 veces más delgada y 2 veces más baja), y debemos estirarla hasta la ventana. Entonces, en el ancho del rectángulo bg_flower, escriba el valor del ancho de la ventana, y en altura, la altura de la ventana. establezca las coordenadas en 0, de modo que la imagen se muestre en la esquina superior izquierda.

Para mostrar imágenes con cambio de tamaño, hay una función especial. Se ve así: 

  SDL_BlitScaled(flower, NULL, scr, &bg_flower);

Se necesitan 4 significados. El primero es la imagen que queremos mostrar, el segundo es el Rectángulo que queremos cortar de esta imagen (si queremos tomar la imagen completa, escribimos NULL), el tercero es la superficie en la que queremos mostrar la imagen, y el cuarto es el mismo rectángulo, dimensiones y las coordenadas de las cuales tomamos para estirar o comprimir y mostrar la imagen.

A continuación, solo actualice la superficie de la pantalla, configure el retraso, salga y regrese 0. 

  SDL_UpdateWindowSurface(win); SDL_Delay(2000); quit(); return 0; };

Aquí es donde termina nuestro programa. Para compilar este milagro, debe agregar -lSDL2_image, como otra opción de compilación.

Y en esto terminaremos por hoy. Aquí está el código que obtuvimos: 

 #include <SDL2/SDL.h> #include <SDL2/SDL_image.h> #include <iostream> const int SCREEN_WIDTH = 640; const int SCREEN_HEIGHT = 480; SDL_Window *win = NULL; SDL_Surface *scr = NULL; SDL_Surface *flower = NULL; bool init() { if (SDL_Init(SDL_INIT_VIDEO) != 0) { std::cout << "Can't init video: " << SDL_GetError() << std::endl; return false; } int flags = IMG_INIT_PNG; if ( !( IMG_Init( flags ) & flags ) ) { std::cout << "Can't init image: " << IMG_GetError() << std::endl; return false; } win = SDL_CreateWindow(" PNG", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN); if (win == NULL) { std::cout << "Can't create window: " << SDL_GetError() << std::endl; return false; } scr = SDL_GetWindowSurface(win); return true; } bool load() { flower = IMG_Load("flower.png"); if (flower == NULL) { std::cout << "Can't load: " << IMG_GetError() << std::endl; return false; } flower = SDL_ConvertSurface(flower, scr->format, NULL); if (flower == NULL) { std::cout << "Can't convert: " << SDL_GetError() << std::endl; return false; } return true; } void quit() { SDL_FreeSurface(flower); SDL_FreeSurface(scr); SDL_DestroyWindow(win); SDL_Quit(); IMG_Quit(); } int main (int argc, char ** args) { if (!init()) { system("pause"); quit(); return 1; } if (!load()) { system("pause"); quit(); return 1; } SDL_Rect bg_flower; bg_flower.w = SCREEN_WIDTH; bg_flower.h = SCREEN_HEIGHT; bg_flower.x = 0; bg_flower.y = 0; SDL_BlitScaled(flower, NULL, scr, &bg_flower); SDL_UpdateWindowSurface(win); SDL_Delay(2000); quit(); return 0; };
