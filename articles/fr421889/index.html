Webinaires Skillbox Friday: Développement pour les débutants



Aujourd'hui, nous publions une sélection de webinaires directement liés au développement de logiciels. Les webinaires ont différents sujets, plus précisément, ils parlent de différents domaines de développement. Entre autres sujets - un dictionnaire d'un développeur Web, des hacks de vie pour les programmeurs débutants, le développement d'applications en PHP et pour Android.

Nous vous rappelons: pour tous les lecteurs de «Habr» - une remise de 10 000 roubles lors de l'inscription à un cours Skillbox en utilisant le code promo «Habr».

Dictionnaire des développeurs Web



L'ancien directeur technique de l'université en ligne Skillbox Nikolay Baryshnikov dirige une master class dans laquelle il parle des termes importants que les développeurs Web utilisent. La plupart de ces termes ne peuvent tout simplement pas être supprimés parmi les programmeurs: vous ne pouvez pas comprendre vos collègues.

Le webinaire est principalement destiné aux débutants qui connaissent peu le domaine dans lequel ils vont travailler.

Cours pratique annuel "Profession Web Developer" .

Comment développer une application PHP



Le principal programmeur Badoo Mikhail Ovchinnikov partage son expérience dans ce webinaire. Il explique comment développer une application en PHP. Il s'agit du fil conducteur du webinaire. Sujets intermédiaires - la popularité de PHP en tant que langage pour le développement backend, les raisons de la popularité, le bagage nécessaire de connaissances et de compétences pour un professionnel dans ce domaine. En outre, Mikhail parle également de la situation sur le marché des développeurs PHP, y compris des entreprises où un tel programmeur peut travailler, des niveaux de salaire et des exigences des employeurs.

Cours en ligne "Développeur PHP de 0 à PRO"
.

Comment devenir développeur Android



Webinaire, dont le thème principal est de devenir un développeur Android débutant. Il révèle des sujets sur le minimum de compétences et de connaissances nécessaires pour créer votre première application mobile, des outils de développement, ainsi que le temps et les efforts nécessaires pour passer d'un programmeur stagiaire à un chef d'équipe dans une entreprise internationale.

Cours en ligne "Développeur mobile à partir de zéro . "
.

Outils d'automatisation du développement mobile



Le webinaire d'auteur du chef du développement mobile AGIMA Dmitry Shashlov sera intéressant non seulement pour les débutants, mais aussi pour les développeurs relativement expérimentés. Dmitry nous dit ce qu'est l'intégration continue et la livraison continue, pourquoi les autotests sont nécessaires, comment travailler avec le code dans une équipe en utilisant gitflow, pull request et code review.

Cours pratique en ligne "Mobile Developer PRO" .

Lifehacks pour les programmeurs débutants



L'auteur du webinaire est Gleb Mikheev, directeur technique de Beta Digital Production. Il partage des hacks de vie pour les programmeurs débutants, qu'il a réussi à collecter tout le temps. Les principaux sujets du webinaire sont les caractéristiques du parcours du programmeur, un ensemble de qualités personnelles dont un spécialiste a besoin, les mythes sur les développeurs et les détails d'une étude de cas sur le travail et la vie d'un développeur frontend dans l'une des agences de Moscou.

Le webinaire sera intéressant pour les programmeurs débutants, mais de nombreux sujets peuvent être utiles aux professionnels. Soit dit en passant, si vous avez quelque chose à partager avec d'autres lecteurs sur les sujets des webinaires, désabonnez-vous dans les commentaires.
