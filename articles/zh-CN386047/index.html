素描Arduino RC步进电机控制

#include <TimerOne.h>

setup .
  Timer1.initialize(500000);         // initialize timer1, and set a 1/2 second period
  Timer1.pwm(9, 512,300);                // setup pwm on pin 9, 50% duty cycle
  Timer1.attachInterrupt(callback); 



void callback()
{
digitalWrite(STEP_PIN1, digitalRead(STEP_PIN1) ^ 1);
delayMicroseconds(10);
digitalWrite(STEP_PIN1, digitalRead(STEP_PIN1) ^ 1);
}

在我看来，当使用焊接时，Arduino 2560的电缆碰到外壳时，记录在该瞬间的下方。结果，我们的UART烧坏了，无法再通过USB进行刷新。最快的解决方案是切换到临近的Arduino Uno。他们放弃了第二个操作员的控制台，正好有足够的端口。
