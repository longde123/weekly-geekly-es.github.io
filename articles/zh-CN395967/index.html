刷新Radeon RX 480会将4 GB的版本转换为8 GB的版本

GPU卡的频率可以超频至1500 MHz。



上个月初，AMD推出了AMD Radeon RX 480显卡，这是第一家使用Polaris架构运行图形处理器的视频卡公司。该设备的性能为5.5 TFLOPS，对于价格为199美元的卡（最低配置）而言，还不错。此外，这里还增加了对虚拟现实技术的支持-现在这是许多制造商所指导的重要领域。 Polaris 10 GPU使用14nm FinFET技术制成。它包含2034个流处理器，并且使用256位总线与内存交换数据。该适配器配有一个六针电源连接器和四个视频输出。

如我们所见，该卡的特性还不错，但事实证明，它们可以改进。超频者能够提高GPU的频率，并且通过一个棘手的技巧，视频内存量可以从4 GB增加到8 GB。

对GPU超频


该卡进入市场后，发烧友立即对其超频潜力产生了兴趣。超频者能够将AMD Radeon RX 480显卡的图形处理器的工作频率提高到1425 MHz。在标准/增强模式下，GPU卡的标称频率为1120/1266 MHz。在3DMark Firestrike中检查超频卡时，结果为14646点。这比出厂时已超频的Radeon R9 390X和Radeon R9 Nano等卡的性能要高。



完成这些实验后，发烧友决定使用液体冷却系统测试卡。在这种情况下，使用了一个额外的风扇来冷却内存芯片和电源系统。此外，我还必须安装一个额外的外部电源控制器。使用该控制器，GPU电压增加到1.35V。超频结果令人印象深刻：视频芯片的频率增加到1500 MHz，存储芯片增加到9000 MHz。 3DMark Firestrike的结果增加到15363分。没错，电池加热到105摄氏度，存储卡开始消耗260瓦。



由于该卡使用6 + 1相电源系统，因此与NVIDIA GeForce GTX 1080创始人版（此处安装5 + 1相电源系统）相比，它能够提供58％的电流。如果制造商为其配备了8针电源连接器，我们可以谈论该设备的更大超频潜力。 AMD报告称，改进的卡型号-Sapphire RX 480 Nitro，Sapphire RX 480 Gaming，Powercolor RX 480 Devil和ASUS RX 480 Strix Gaming将获得一个8针附加电源连接器。华硕RX 480 Strix Gaming将获得第二个连接器-8针或6针。这种卡应该在本月开始销售。

在六月，它变得众所周知关于超过Radeon RX 480卡规格中指定的能耗参数的问题，一个150瓦的适配器在满载时消耗168W。一旦安装在PCI-E插槽中，此适配器将消耗90瓦。这超过了75瓦插槽的标准。在发现问题之后，该公司通过发布软件更新-Radeon Software 16.7.1来解决此问题。

增加记忆


制造商发布两种版本的AMD Radeon RX 480卡：带有4 GB和8 GB的视频内存。事实证明，较年轻的型号AMD Radeon RX 480 4G实际上具有8 GB的视频内存。直接检查4 GB的版本后，结果发现该主板上有8个三星存储芯片，总容量为8 GB。



但是制造商禁用了一半的内存，“额外”的内存被锁定在BIOS级别。以这种方式减少设备的内存容量并不是最合理的决定，但是在进入市场的首批AMD Radeon RX 480卡中，是以这种方式进行关机的。





事实证明，可以将其移除。为此，您需要ATIFlash 2.74程序。在它的帮助下，TechPowerUp网站的代表   拍摄了具有8 GB内存的BIOS卡的图像。图片可用在此链接。之后，将生成的图像转移到较年轻的模型。此后，较年轻的模型立即显示出视频内存最多增加了8 GB。在《使命召唤：黑色行动》 III游戏中测试了修改后的卡的操作。玩具在4K Ultra HD（3840 x 2160）中消耗了超过4 GB的视频内存。在卡的8 GB版本上进行了游戏的测试。两张牌都显示几乎相同的结果。原始的8 GB卡提供24.6 fps，修改后的4 GB卡显示24.7 fps，原始4 GB卡显示23.3 fps。



AMD Radeon RX 480的4 GB和8 GB版本之间的价格差为30美元。

刷新较年轻的卡之前，请确保板上有8个Samsung内存芯片。制造商很可能会实际减少视频内存的数量，然后，当您尝试使用较旧型号的BIOS刷新卡的较新版本时，会出现“砖块”。



以类似的方式，在2011年，他们学习了如何将ATI Radeon HD 6950转换为HD6970。年轻的BIOS技工将其刷新为旧的BIOS，从而获得了完整的HD 6970功能并节省了约100美元（当时）。已经提到的TechPowerUp网站的记者首次报道了这种固件的可能性。

说到图形卡的图形内存，值得回顾一下问题使用NVIDIA卡。即-型号为GTX970。卡的制造商在卡的规格中指出了4 GB的视频内存量。的确如此，但是该卷的八分之一没有正常工作。当访问最新的500-700 MB GTX 970内存时，性能会大大下降。下降非常明显-从150 GB / s降低到19.88 GB / s。

卡的内存分为两个部分。这是3.5 GB的主要快速内存段，其余为0.5 GB。最后500 MB的问题主要表现在4K显示器或具有多台显示器的系统的所有者中。

虚拟现实


HTC Vive和Oculus Rift等虚拟现实头盔的系统要求表明了功能强大的图形卡，价格从300美元起。 AMD以199美元的价格提供其卡。因此，VR系统成本的下限逐渐降低。

专门针对VR模式，已发布了一组名为LiquidVR的库和SDK（支持DirectX 12和Vulkan API）。

此外，还添加了可变速率着色（VRS）技术，旨在节省VR环境中的像素填充速率资源。此功能允许驾驶员通过填充多余区域的像素来减轻来自GPU的负担，以补偿头盔镜片比例的失真。

NVIDIA GTX 1080和1070卡还支持虚拟现实模式，但它们属于中高端价格段。例如，NVIDIA GTX 1070的成本为380美元。旗舰型号NVIDIA GTX 1080的价格为600美元。

软体类


现在，该公司承诺比往年更频繁地发布其产品的驱动程序的新版本。例如，在2015年，仅发布了三个具有WHQL认证的Radeon软件版本，以及九个针对某些游戏进行了优化的侧面版本。

AMD计划发布六个主要驱动程序更新，这些更新具有WHQL认证，并且是游戏玩家针对特定游戏进行优化所必需的中间版本。
