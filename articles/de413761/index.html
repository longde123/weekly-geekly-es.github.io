So verhindern Sie, dass Windows 10 nach Updates neu gestartet wird

Hallo an alle. Überraschenderweise ist die Antwort auf diese Frage (genauer gesagt, "wie man alles so macht, wie es war") in Runet schlecht beleuchtet. Im Allgemeinen muss noch nach einer vorgefertigten Lösung gesucht werden.
Das Problem muss jedoch gelöst werden, weil Das System stellt nach einem Neustart nicht alle Anwendungen wieder her.

Also:

  1. Gehen Sie zu den Richtlinieneinstellungen. Dies funktioniert auch auf einem lokalen Computer ohne Domäne.
    Gehen Sie wie folgt vor, um den Richtlinieneditor zu starten: Start → Ausführen (oder Win + R) → gpedit.msc
  2. Wählen Sie links in der Baumstruktur Folgendes aus: Computerkonfiguration, Administrative Vorlagen, Windows-Komponenten, Windows Update.
  3. Hier interessieren uns zwei Parameter:

    1. Führen Sie während der automatischen Installation von Updates keinen automatischen Neustart durch, wenn das System Benutzer verwendet (kein automatischer Neustart mit angemeldeten Benutzern für geplante automatische Update-Installationen). (Aktiviert). Hurra, es wird keinen Neustart mehr geben (wenn es aktive Sitzungen gibt).
    2. Beachten Sie auch den Parameter Configure Automatic Updates. Wenn Sie es in den Status "Deaktiviert" versetzen, lädt das System keine Updates mehr herunter und installiert sie nicht mehr.

      Wenn Sie möchten, dass das System niemals spontan neu gestartet wird (auch ohne aktive Sitzungen), müssen Sie dies tun.

Nach dem Deaktivieren des Neustarts zeigt das System regelmäßig ein schönes Fenster an, in dem Sie aufgefordert werden, den Computer neu zu starten (wie zuvor):


Dieses Fenster ist übrigens noch besser: Sie können den Neustart "bis zum Abend" verschieben, was bedeutet, dass Sie nur einmal am Tag und nicht wie zuvor alle 4 Stunden daran erinnert werden.

PS: Wie in den Kommentaren korrekt angegeben, ist der Richtlinieneditor nur für Versionen verfügbar, die steiler als Professional sind. Bei jüngeren Versionen müssen Sie Änderungen direkt an der Registrierung vornehmen.
