Penerimaan pembayaran dari kartu tanpa jur. wajah di Yandex.Money


Layanan sumbangan uang yandex sumber terbuka


Menerima pembayaran pada Yandex.Money fisik. wajah

  • pembayaran online dari kartu, Yandex, telepon
  • tambahkan tag ke pembayaran
  • menerima dan memeriksa peringatan di server Anda
  • Contoh otomatisasi bot Telegram

Implementasi dan deskripsi di bawah potongan

Proyek python



Modul:


  • money.yandex.ru - dompet khusus untuknya kami menerima uang
  • httpsserver - server python menghasilkan halaman pembayaran dan menerima peringatan Yandex
  • billingservice - prosedur jarak jauh, dipanggil setelah menerima pemberitahuan pembayaran, berfungsi dengan database
  • redis-server - digunakan sebagai server antrian untuk panggilan prosedur jarak jauh
  • example / telegrambot - contoh menghasilkan tautan untuk membayar bot Telegram

Tombol untuk mengumpulkan uang


Yandex.Money memiliki kemampuan untuk membuat tombol dan formulir untuk mengumpulkan uang di mana Anda dapat menanamkan komentar dan tag pembayaran Anda.



Pemberitahuan Pembayaran


Selain itu - dimungkinkan untuk mengkonfigurasi penerimaan notifikasi pembayaran di server Anda.



Tambahkan penerimaan pembayaran


  1. Daftarkan, tambah status dompet ke nominal
  2. PENGATURAN-> SEMUA YANG LAIN-> PEMBERITAHUAN HTTP
    • Tambahkan domain dan rahasia Anda
    • Nyalakan notifikasi
  3. Naikkan redis-server , tambahkan parameter akses ke rq_access.py
  4. Kami menerima sertifikat SSL untuk domain, misalnya melalui letsencrypt
  5. Naikkan server https dengan parameternya, jangan lupa tentang firewall jika server tidak tersedia
  6. Kami mendaftarkan id dompet dan rahasia untuk pemberitahuan dari pengaturan di httpsserver / .secret / ymsecret.py
  7. Edit httpserver-ympayment.service , mulai server di bawah pengawas
  8. Pada mesin dengan pangkalan, edit billing.service dan jalankan systemd untuk layanan ini pekerja RQ akan mulai
  9. Memeriksa prosedur mulai dari jarak jauh melalui antrian Redis
  10. Berdasarkan contoh, kami menambahkan generasi tautan pembayaran ke fungsi bot Anda
