拦截Windows 10上Firefox和Chrome的安装


截图thurrott.com

Windows 10 v.1809的内部版本具有新功能 。 当用户尝试安装Firefox或Chrome时，系统现在会显示一条消息，说明Microsoft Edge已在计算机上-并建议启动此“更快，更安全的浏览器”。

操作系统截获了未经批准可安装的替代软件的安装，并试图诱使用户避免不必要的操作。

此外，应用程序的设置在最新版本中已更改。 以前，默认情况下，允许安装任何程序- 允许在任何地方使用应用程序（默认） 。 提供的其他选项包括“警告我不要从商店安装应用程序”和“允许仅从商店安装应用程序”。

新的默认设置为“ 向我显示应用推荐内容” 。 其他选项是“禁用对应用程序的建议”，“有关不从商店安装应用程序的警告”和“仅允许从商店安装应用程序”。


截图thurrott.com

显然，根据“ 应用程序和功能”部分中的新默认设置，即“显示应用程序建议”，拦截了Windows 10中Firefox和Chrome的安装。 Windows 10中的建议包括有关安装“不必要的”浏览器（例如Firefox和Chrome）的警告。

此功能的出现引起了某些用户的愤怒。 许多人仍然记得Windows 7中显示的安装Windows 10的烦人提议。拒绝这些提议是一项艰巨的任务，因此有时Windows 10的安装违背了用户的意愿。

据评论员称，此类行为表明浏览器市场的斗争日趋激烈，并且微软希望保持/增加其份额。 现在，Chrome占据了这一市场的主导地位，这归功于Google积极在其网站上投放广告的政策。 许多Firefox和Edge用户必须碰到过要约，才能在Google拥有的网站上安装Chrome。 还有其他方法：例如，最近的YouTube重新设计使用了奇怪的Shadow DOM v0 API，这些API仅在Chrome中实现。 因此， Chrome浏览器中的 YouTube页面加载速度比其他浏览器要快得多 。


此外，在安装一些无关的免费实用程序期间，Chrome可能会“偶然地”成为默认浏览器。 这种做法非常普遍，以至于甚至有一些特殊的工具可以在安装任何程序时删除所有不必要的检查标记。

苹果还试图为MacOS用户提供其Safari浏览器。 与竞争对手不同，Firefox不会以这种方式发展：Mozilla没有可用于推广的平台。

目前尚不清楚具有替代浏览器设置拦截功能的新功能是否会进入Windows 10的十月重大更新。如果发生这种情况，此类动作可能会引起欧盟反垄断委员会的反对。 根据欧盟过去的反托拉斯程序的经验，Microsoft可以将此类Edge浏览器建议的显示对象向欧盟以外的用户，以避免出现问题。

Windows 10中的“应用程序准则”实际上可能对某些用户有用。 例如，许多人不知道没有必要安装Acrobat Reader来查看PDF文件，因此会向系统添加新的潜在漏洞。 也就是说，Microsoft默认情况下相当合理地启用了此选项。 唯一的问题是建议本身的内容。
