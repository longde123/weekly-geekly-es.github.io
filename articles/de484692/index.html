Vitamin C - Muss ich Nahrungsergänzungsmittel einnehmen oder handelt es sich um einen kommerziellen Schritt?

Einleitung


Meine Spezialität und Arbeit hat nichts mit Vitaminen und Chemie im Allgemeinen zu tun, aber es stellte sich heraus, dass mich das Thema Vitamine und Ernährung interessiert. Ich beschloss, es so gut wie möglich herauszufinden und herauszufinden, ob es sich lohnt, Vitamin C einzunehmen.


Der Schutz Ihrer Gesundheit ist gerade heute das vordringlichste Thema. Ich bin auch nicht an ihre Seite gegangen und werde im Folgenden beschreiben, wie ich Ergänzungsmittel in Form von Vitamin C kennengelernt habe.


Einmal fand ich einen interessanten Artikel mit einer Empfehlung zur Einnahme von Vitamin C zusätzlich zur Diät. Ein Instagram-Bericht eines Ernährungswissenschaftlers lag vor.
Der Ernährungsberater empfiehlt, Ihrer Ernährung organisches Vitamin C hinzuzufügen. Es wird behauptet, dass organisches Vitamin C besser absorbiert und überschüssiges Vitamin C leicht aus dem Körper ausgeschieden wird. Natürlich ist es teurer, 1000mg kosten 13 Dollar. Warum hinzufügen? Weil es viel braucht, aber wir bekommen es nicht aus Essen.


Fast immer wird die Information über Vitamin C von einer Beschreibung seines Nutzens begleitet [ 1 ]. Es kommt zu dem Punkt, dass sie versucht haben, Vitamin C zur Krebsbekämpfung einzusetzen. Als nächstes werde ich versuchen, kurz das Material zu skizzieren, das ich studiert habe.


Vitamin C


Vitamin C (L-Ascorbinsäure) ist ein starkes Antioxidans, das die Immunität stärkt, die Aufnahme im Darm und die Aufnahme von Eisen durch den Körper erleichtert und für die normale Funktion des Bindegewebes und des Knochengewebes notwendig ist. Unser Körper kann Vitamin C nicht speichern, daher müssen Sie es ständig zusätzlich erhalten.


Beziehen von Bio-Vitamin C aus Hagebutten


Im Internet fand ich einen Kursartikel über die Gewinnung von organischem Vitamin C aus Hagebutten [ 2 ]. Hagebutten werden in einem Diffusor (Temperatur 70–75 ° C) serviert. Der Diffusionssaft wird durch eine Filterpresse zu einem Dreikörper-Verdampfer geschickt, wo der Saft auf eine Dichte von 60% konzentriert und dann in einen Sprühtrockner gepumpt wird, um ein pulverförmiges Konzentrat mit Vitamin C zum Tablettieren zu erhalten.


Erhalt von synthetischem Vitamin C


Im industriellen Maßstab wird Vitamin C synthetisiert. Vitamin C wird in der Industrie aus D-Glucose nach der Reichstein-Grüssner-Methode hergestellt. D-Glucose (19) wird mit Wasserstoff unter Druck katalytisch hydriert, um D-Sorbit (30) zu erhalten. D-Sorbit oxidiert enzymatisch (Acetobacter suboxydans) zu L-Sorbose (23). Die cyclische Form von α-L-Sorbofuranose wird durch Einwirkung von Aceton und Schwefelsäure in Diisopropylidensorbose umgewandelt (62). Die Acetonierung von Sorbose bei niedrigen Temperaturen (–5 ° C) führt zu einer signifikanten Steigerung der Ausbeute des Di-O-Isopropylidenderivats. Die weitere Oxidation von Diacetonsorbose zu Säure (63) wird üblicherweise unter Verwendung von Natriumhypochlorit in Gegenwart von Nickelsulfat durchgeführt. Isopropyliden-Schutzgruppen werden durch Einwirkung von Salzsäure in einem wasserfreien Medium (Ethanol-Chloroform, 65 ° C, 50 h) entfernt. Die resultierende cyclische Form wandelt sich in 2-Oxogulonsäure (64) um, unter deren Einwirkung eine saure Lactonisierung stattfindet; Die Zwischenverbindung wird dann enolisiert, um Ascorbinsäure (65) zu ergeben.


L-Ascorbinsäure ist eine kristalline Substanz weißer Farbe mit ausreichend stark sauren und reduzierenden (Oxidationspotential beträgt +127 mV) Eigenschaften.



Vitamin C Tagesdosis


In den meisten Quellen beträgt die empfohlene tägliche Zufuhr von Vitamin C für Männer 90 mg und für Frauen 75 mg. Jugendliche benötigen 65 bis 75 mg Vitamin C pro Tag und Kinder etwa 35-50 mg. Raucher haben ein erhöhtes Bedürfnis nach Vitamin C, weil eine Zigarette 25 mg Vitamin C zerstört.


Auf Instagram und auf seiner Website behauptet der zuvor erwähnte Ernährungswissenschaftler, dass wir viel mehr brauchen, etwa 5 Gramm organisches Vitamin C, aber alle einzeln.


Es gibt nur wenige Studien auf der WHO-Website, die sich auf Vitamin C beziehen. Studien auf der WHO-Website: „Einnahme von Vitamin C-Nahrungsergänzungsmitteln während der Schwangerschaft“ [ 3 ].


Schwerer Vitamin C-Mangel führt zu Skorbut. Skorbutsymptome treten auf, wenn der Vitamin C-Gehalt im Blutplasma unter 11 µg / l fällt oder die Gesamtkörperversorgung unter 300 mg liegt.


Übermäßiges Vitamin C, das mit Vitaminzusätzen ergänzt wird, wird von gesunden Menschen normalerweise gut vertragen, obwohl Entzündungen des Darmtrakts und Durchfall möglich sind. Eine weitere mögliche, aber seltene Nebenwirkung ist eine erhöhte Oxalatausscheidung im Urin mit einem erhöhten Risiko für Nierensteine.


Organisches und synthetisches Vitamin C


Dieses Thema wird zunehmend kontrovers diskutiert. Das zum Kauf empfohlene Bio-Vitamin C besteht zu 100% aus Mais (siehe Produktbeschreibung auf der Website des Geschäfts). Es wird auch empfohlen, synthetisches Vitamin C abzulehnen es kann nicht nur nutzlos, sondern auch schädlich sein [ 4 ]. Eine vom Zentrum für Radikalforschung durchgeführte Untersuchung der Unterschiede zwischen synthetischem und organischem Vitamin C ergab jedoch, dass organisches und synthetisches Vitamin C den gleichen Effekt auf den menschlichen Körper haben [ 5 ].


Vitamin C als Krebsmittel


Das Interesse an Vitamin C als Heilmittel gegen Krebs entstand vor etwa 30 Jahren mit der Einreichung des Nobelpreisträgers Linus Pauling. Paulings Name ist auch in der Öffentlichkeit bekannt, da er nach einem persönlichen Beispiel propagandistisch die Verwendung großer Dosen von Ascorbinsäure (Vitamin C) als Nahrungsergänzungsmittel zur Verbesserung der allgemeinen Gesundheit und zur Vorbeugung (oder zumindest Verringerung des Schweregrads des Verlaufs) von Krankheiten wie Erkältungen und Krebs vorsieht. So tauchte der Begriff "Orthomolekulare Medizin" auf. Die orthomolekulare Medizin ist eine der Varianten der Alternativmedizin. Methoden der orthomolekularen Medizin umfassen die Behandlung und Vorbeugung von Krankheiten mit verschiedenen Vitaminen, Aminosäuren, Nährstoffen in Form von biologisch aktiven Zusatzstoffen (BAA) und Lebensmitteln. Vitamin C ist auch für die Bildung von Kollagenfasern notwendig, um das Körpergewebe vor freien Radikalen zu schützen. Pauling schlug vor, die tägliche Dosis von Vitamin C um das 100-200-fache zu erhöhen. Zur Bildung von Kollagenfasern, also zur Ausübung der Funktion eines Coenzyms, genügten 10 mg Ascorbinsäure. Pauling argumentierte jedoch, dass Ascorbinsäure als Antioxidans viele andere Funktionen erfüllen und Zellen und Gewebe vor Schäden durch freie Sauerstoffradikale schützen kann.


Für die erste klinische Studie über die heilenden Eigenschaften von Vitamin-C-Megadosen hat das Linus-Pauling-Institut eine äußerst schwierige Aufgabe ausgewählt - den Nachweis zu erbringen, dass Krebspatienten, die sich als hoffnungslos erwiesen und von einer Bestrahlung oder Chemotherapie nicht mehr profitierten, und die entwickelten Metastasen chirurgische Eingriffe ausschließen. Paulings Gruppe schlug vor, sie mit Ascorbinsäure in Dosen von 10 g pro Tag zu behandeln. Das positive Ergebnis war, dass sie mit Megadosen Vitamin C angeblich viel länger lebten als die "Hoffnungslosen", die dieses Vitamin nicht erhielten. Daher wurde der Schluss gezogen, dass mit seiner Hilfe eine "Lebensverlängerung" möglich ist.


Linus Pauling starb am 19. August 1994 im Alter von 93 Jahren. Die Förderung der Heilwirkung hoher Vitamin C-Dosen nach Paulings Tod hat sich deutlich abgeschwächt. In der wissenschaftlichen Literatur erschienen die Ergebnisse von Studien zu den negativen Folgen der Einnahme von Ascorbinsäure in hohen Dosen.


Der Europäische Gerichtshof hat seit dem 1. August 2005 entschieden, die Dosierung von Vitamin-C-Präparaten in der EU zu begrenzen. Der Wortlaut der Empfehlungen wurde geändert (die Wörter "behandelt", "heilt", "verlängert" usw. werden ersetzt durch "fördert die Erhaltung", "schützt"). )


Ich wurde gewarnt, dass Vitamin C ein starkes Antioxidans ist, aber nach jüngsten Studien können Antioxidantien Krebs beschleunigen, obwohl Vitamin C nicht erwähnt wird [ 6 ]. Daher ändert sich mit jeder neuen Studie die Einstellung zum Einsatz von Vitamin C bei onkologischen Erkrankungen.


Fazit


Vitamin C ist definitiv notwendig für unseren Körper. Die besten Quellen für Vitamin C sind Gemüse und Obst, Beeren. Im Gegensatz zu Vitamin C-Nahrungsergänzungsmitteln können Sie durch den Verzehr von Gemüse, Obst und Beeren auch andere Vitamine erhalten.


Lebensmittel mit hohem Vitamin C-Gehalt:


  • Hagebutte (trocken - 1200 mg / 100 g, frisch - 650 mg / 100 g)
  • Gemüsepaprika (bulgarisch) (200 mg / 100 g)
  • Schwarze Johannisbeere (200 mg / 100 g)
  • Sanddorn (200 mg / 100 g)
  • Petersilie (grün) (150 mg / 100 g)

In Ermangelung von Gemüse und Obst in der Ernährung ist es notwendig, eine zusätzliche Vitaminquelle zu finden. Um die Ascorbinsäurekonzentration von Vitamin C zu überprüfen, können Sie eine Vitamin C-Analyse im Blutplasma durchführen.


Zu wenig Informationen über die Produktion von organischem Vitamin C im industriellen Maßstab und dessen Unterschied zu synthetischer, beispielsweise Ascorbinsäure aus einer Apotheke. Vielleicht ist dies ein kommerzieller Schachzug, für den Ernährungswissenschaftler auf ihren Instagram-Konten aktiv werben und mit Werbung Geld verdienen.


Die positive Wirkung von Vitamin C bei bereits entdeckter Onkologie ist deshalb sehr umstritten zeigen jedes Jahr andere Ergebnisse. Vor dem Gebrauch ist es besser, einen Arzt zu konsultieren.


Hohe Dosen können sich nachteilig auf den Körper auswirken [ 7 ]. Wenn Sie sich daher entschließen, weiterhin Vitamine als Zusatzstoff zu verwenden, müssen Sie einen Arzt konsultieren und mithilfe von Tests sicherstellen, dass die Ergänzung erforderlich ist.
