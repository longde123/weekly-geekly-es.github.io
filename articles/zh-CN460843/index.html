引入新的3CX呼叫流程设计器和3CX CRM模板生成器

带有可视表达式编辑器的新型3CX呼叫流程设计器




3CX坚持原则-我们的产品必须简单明了。 因此，我们再次更新了3CX呼叫流程设计器（CFD）语音应用程序开发环境。 新版本具有现代化的用户界面（新图标）和可视化编辑器，以及用于创建脚本的表达式编辑器。



新版本的CFD提供以下功能：

  • 视觉表达创建。 您可以从“表达式元素”部分拖放内置函数，变量和常量。 您可以直观地创建复杂的表达式，而无需打开多个编辑器窗口。
  • 快速功能选择。 现在，内置功能分为方便的类别，您可以从中快速选择所需的功能。
  • 动态包含常量。 当将适当的组件添加到CFD工作区时，可用常量将动态包含在“表达式元素”面板中。

请注意，只有在3CX v16 Update 2中，新的“转移”组件才能将呼叫转移到分机的语音邮件。

下载新的3CX CFD或通过更新菜单进行升级。 之后，您可以在3CX v16 Update 2上运行语音应用程序。

此版本的CFD中的完整变更日志 。

新的3CX CRM模板生成器，用于连接CRM


我们还引入了新版本的模板生成实用程序，用于将您自己的CRM系统连接到3CX。 在新版本的CRM模板生成器中，出现了XML模板创建向导，该向导将指导您完成所有阶段，直到您完成现成的有效集成。



首先，请查看CRM的RESTful API文档。 然后运行模板生成向导。

  • 按照向导提示，从CRM API的文档中指示适当的参数。
  • 创建XML模板需要具备基本的编程技能，并需要对RESTful API CRM系统的支持。
  • 当向导起作用时，模板的每个部分都正确地形成，其中包括所有必需的参数。

查看模板生成器演练 。 您也可以观看英语视频教程。
