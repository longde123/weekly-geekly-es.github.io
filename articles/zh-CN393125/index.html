Chrome 50不再支持Windows XP，Vista和旧版OS X



Google已经完成了很久以前的承诺-不再在新版的Chrome浏览器中支持旧的Windows和OS X操作系统。停止支持意味着无法接收Windows XP，Windows Vista，OS X Snow Leopard，OS X Lion和OS X Mountain Lion用户的更新。对于他们来说，最新版本的Chrome浏览器是49th。浏览器的旧版本将在这些版本的OS上运行，但用户将不再收到安全更新。无法保证在旧版的Chrome上身份验证将继续适用于Google服务。

Google于2015年4月（即一年前）宣布，将于2016年停止对这些操作系统的支持。同时，原计划于去年年底停止对旧版Windows和OS X的支持，但随后Google决定将对操作系统的支持延长至2016年4月。

发生这种情况的原因是这些操作系统的制造商本身不再支持它们（有一些例外，因为仍针对Windows Vista发布了安全更新）。

如果要更新，有两种解决方法。首先是更新操作系统，这是合乎逻辑的。第二种是安装Firefox 45，它接收Windows XP和OS X 10.6的更新，或Opera，后者也可以在XP和OS X 10.7上运行。



至于新版本的Chrome中的更新，其中有很多。主要创新涉及推送通知。因此，现在借助这些通知，不仅可以发出警报用户，还可以通过对话框与他们进行交互。此外，网站现在将能够在其消息中伴随振动或声音。

出现了另一个有趣的功能-声明性预加载，它允许您下载批量内容，以便用户长时间不希望页面内容完全加载。开发人员可以通过在HTML代码中添加link rel =“ preload”属性来提供这种类型的资源加载。
