如何在成长中的初创企业中保持一席之地

如果您是第一批初创公司员工之一，请做好准备，好一天，近几年来您日以继夜地工作的一切不再至关重要。 您的职位不会那么重要，新的经理和老板，商务会议，文书工作将出现在公司本身中。 而且，最重要的是，您将了解您在项目中的角色必须完全改变。


本文的作者能够从投资者，董事会成员和总经理的角度观察到这种变化。 他指出，有时此过程在管理方面非常痛苦且困难。 在其职业生涯的曙光中，作为一名雇员的作者本人也经历了类似的经历，并“以最糟糕的方式摆脱了这种状况”。

在此材料中，作者分享了他当时想拥有的知识。

我加入了我的职业生涯中第二家半导体设备制造商MIPS Computers，担任市场总监，还担任过代理销售总监的职务。 在公司成立的第一年，我经常工作，抓住任何机会并创造新的机会。 如果我们的项目成功了，我随时会毫不犹豫地乘飞机飞往任何地方。 我与工程师紧密合作以使该产品流行，并且我试图在创建功能强大的工程工作站的公司的项目中使用微电路。 同时，我一直在寻找合适的市场，客户和销售流程。 当然，我无法入睡，但这是我一生中最美好的时光之一。

而现在，一年之后-好消息。 临时将军最终被一名常任将军取代。 我们的芯片快要完成了，我说服了重要的客户在他们的计算机中使用它。 由于几乎没有资源，我取得了令人印象深刻的成果，并使该公司在大型技术出版物上受到关注。 这还不包括像我们这样的梦usually以求的项目所产生的交易。 我满月幸福……直到新任首席执行官叫我讲话。

我忘记了所有细节，但是我很清楚地记得导演第一次说他对我的成就印象深刻。 当董事刚宣布公司需要扩大规模时，我感到非常震惊和惊讶，但是我对这个角色并不满意……停下来！ 什么

第二分钟，我几乎无法呼吸，好像我被肚子打中了一样。 “不合适”是什么意思？ 性别自己没有列出我的成就吗？ 他承认进步是显而易见的，但将其描述为一套无关紧要的策略，而核心策略却没有任何连贯性。 没有人知道我到底在做什么，而且我无法解释为什么这样做。 “您可以连续尝试所有内容。 这种方法无法扩展。” 我没话说 那不应该是创业的第一年吗？

当无语者回来时，我问将军，我是否可以尝试成为将公司提升到新水平的人。 而且，我们必须向主任表示敬意（几年后才对此表示赞赏），尽管他说他将开始寻找合适的候选人，但我也可以要求这个职位。 此外，他找到了一位教练，他会向我解释“达到新水平”的含义。 我记得如何为变革做准备，买了我能找到的所有管理书籍，并阅读了有关将小公司转变为大公司的所有可用文献。

然后事件就这样发展了...


我隐约记得与一位愉快的老人在一个新的环境中教我必要的技能的教练共进午餐。 问题是我对他的话充耳不闻。 关于改变我的角色，职位和职位的想法使我感到痛苦。 “我不明白。 我做得很好，一切都很好。 为什么什么都应该改变？ “但是教练从来没有发现这些经历。

我很ham愧承认，但我完全不记得这个人在为期数周的午餐会上试图教给我什么。 然后我只是在想自己，现在对我来说有多困难。 愤怒使我免于尝试帮助我。

一个月后，该首席执行官表示，教练将我的成绩评为“仍然有很多东西需要学习”，而我再次感到不愉快。 该公司即将聘请新的市场总监。

我感到an没，离开了公司。

这不是变化，而是损失


如果几年后他们会问我为什么我把事情拖到这么糟糕的境地，我会回答
a）抵制变化，
b）在这种情况下将自己放在首位，甚至不认为新任首席执行官是正确的。

是的，这些言论是正确的，但它们并不能完全说明情况。 仅仅十年后，我意识到如果说实话，那就不是要与变革作斗争。 在我们的创业公司，每天都有新的事情发生。 而且我足够灵活，可以适应无数次更改，并且我做了很多更改。 原因很深，但是我还没有准备好承认这一点。 问题是这些变化使我害怕失去当时的财产。

  • 我失去了地位和地位的丧失 。 我被认为不适合担任目前的职位，这会降低我的技能并严重损害我的自尊心。
  • 我对未来失去了信心。 我开始参与竞争公司的一席之地，正如我认为的那样，这将永远存在。 我的名片上的文字恰恰说明了这一点。 但是我正处在十字路口，不知道未来对我来说会怎样。
  • 我失去了自主权 。 在转折之前，我自己确定了工作的前沿，做了我想做的事情和想做的事情。 我对在旅途中从无关策略中组合策略的需求感到完全满意。 突然间，公司有了具体计划。
  • 我失去了团队合作精神 。 我们是一个小组，经过艰苦奋斗，团结一致并取得了令人难以置信的成果。 后来，新人出现在公司中，他们对我们的道路一无所知，几乎无法欣赏。 团队不再像以前那样拥有相同的信任和同情心。
  • 我认为变革的过程并不十分公平 。 没有人警告我工作会随着时间的推移而改变，因此我需要什么新技能。

到底发生了什么


科学家已经确定了大脑中的一种连接，可以解释由于社交关系而导致的身体不适。 饥饿感和社会隔离会激活类似的神经反应，因为社交关系对于生存至关重要。 尽管经常将工作视为一种经济交流，但大脑主要将其视为一种社会系统。”

回到几十年后的情况，我知道首席执行官是对的。 尽管事实上，社会损失唤醒了我一些原始的东西，但我仍然确实需要学习纪律，时间管理，观察模式的能力，将重要事物与次要事物分开并了解战术与策略之间的差异。 我必须摆脱一个有效但孤立的员工的外表，首先成为经理，然后成为领导者。 但是我只是试图学习所有这些东西而已。

这一步使我的职业发展减慢了五年。

我应该做的


在成长过程中，所有初创公司都会经历某些变形。 他们从为生存而奋斗开始，找到市场利基，开发有需求的产品，然后转向创建庞大的可扩展业务模型和利润。 至于人，一个人的社会关系的数量是有限的 。 人的思想定义了组织发展的边界：在一定的门槛下，将需要更改管理系统。 员工所需的技能在不同阶段也有所不同。



最重要的是， 对公司而言，当公司发展到新的水平时，与早期阶段相关的技能不太可能会被保留 ，这对我很有用。 这项提议应重新阅读几次，因为没有人（不是雇用您的人，风险投资人或您的创业同事）都不会说公司将使您脱胎换骨。 他们中的某些人（例如同事或什至创始人）对此一无所知，而其他人（例如风险投资人）则不以自己的利益为出发点。

严酷的现实是产品，人员和策略正在发生变化。 该公司将不得不转型，以保持业务发展。

CEO如何进行对话


当我的性别谈论公司的必要变化和发展时，他以事实为依据，而我则经历了个人经历。 我认为我在组织中的角色变化是一种损失。 当人们感到自己将要失去一些有价值的东西时，他们会非常情绪化，因为这被视为威胁。 当然，这并不能证明我的适得其反的行为，但是可以部分地解释它。

对于创业领导者而言，重要的是要预见到这种情况，并从一开始就考虑选择有助于克服早期员工经历的真正损失感的选择 。

身份丧失。 如果您从一个人那里拿走一个职位并将其交给其他人，那么实际上将员工留在公司中几乎是不可能的。 在介绍技术总监，市场营销总监，销售总监等正式职位之前，请考虑一下公司在市场中寻找自己的位置并且没有雇用至少几十个人时是否值得做。 可以肯定地说，在旅途开始时在这些职位上工作的人们肯定会在过渡到下一阶段的过程中失去他们。

对未来失去信心。 从历史上看，初创公司和风险投资商一直在推迟讨论，直到公司成长时会发生什么。 几乎没人想成为一个狂热者，因为他知道在成长阶段他很可能会失去自己的位置。 我称它为摩西的问题。 您已经为将您的人民带到应许之地而努力了多年，但是当您想留在这片土地上时却被拒绝了。 有必要对那些能够使初创企业取得理想结果的团队成员的优点进行正式认可。

失去自治权。 管理层应与员工讨论职业步骤。 他们想通过自己的工作受益吗？ 还是想控制人员和流程？ 特殊项目？ 管理层需要为员工提供具有多种选择的特定计划，而不是任意任命。

集体精神的丧失 。 公司的第一批员工奠定了其文化基础。 但是，如果您不要求“老人”修复其独特方面，那么随着新人们的到来，这种遗产很容易消失。 向他们宣告文化共同创始人 。 帮助他们了解团队正在成长，他们应该扮演特别代表的角色。 雇用新员工时，让他们帮助研究公司的文化特征。 而且，非常重要的是，请确保始终记住第一批团队成员是帮助公司发展成为现在的人。

过程中的不公感 。 仅仅告诉员工“变革即将到来”是不够的。 当公司开始进入新阶段时，他们需要什么新技能？ 或何时将其扩展到成千上万的人？ 现有员工如何获得这些技能？

结论


1.风险投资者，创始人和董事了解到，所有现代创业公司都经历几个阶段：寻找，加强和成长。

  • 他们认识到员工在每个阶段都需要不同的技能。
  • 一些最初的初创员工将无法升级。

2.尽管对于首席执行官和董事会来说，这种情况看起来是合乎逻辑的和可以理解的，但早期员工仍将这些变化视为真实而有形的损失。

  • 地位和独特地位的丧失。
  • 失去集体精神。
  • 失去自治权。
  • 我会对未来失去信心。
  • 不公正感的变化。

3.管理层应识别这种情况并向员工提供帮助。

  • 因此，早期员工将更有动力并留在公司中。
  • 公司本身将变得更强大。

4.员工应了解，对这种情况的正确态度会严重影响他们的未来职业以及他们可以为公司带来的利益。
