Wie können wir Ihnen helfen? Wie können Sie uns helfen?

Öffnen Sie die API und die Fingerabdruck-Anmeldung. Was ist neu in der Cloud-Clout-Sicherheitsanwendung?

Cloud-Clout-Anwendung für die sichere Speicherung und den Datenaustausch in den Clouds öffnet seine API.


Aufnahme aus der Serie "Silicon Valley"

Guten Tag, Habr!

Zunächst möchten sich die Entwickler der Anwendung bei allen Entwicklern für die Antwort auf die erste Veröffentlichung im Cloud-Clout-Blog bedanken. Sie haben alle Kommentare sorgfältig gelesen, beantwortet und sich selbst ein Bild von der Weiterentwicklung des Produktes gemacht.

Die Hauptfrage der Leser aller Ressourcen, in denen die Anwendungsüberprüfung veröffentlicht wurde, betraf die Offenheit und Sicherheit von Cloud-Clout selbst. Benutzer möchten wissen, welche Daten die Anwendung sammelt und ob sie wirklich vertrauenswürdig sind. Das ist fair. Daher wurde Schritt für Schritt entschieden, die Details der Anwendung offenzulegen, um die Benutzer von der vollständigen Sicherheit der Verwendung des Produkts zu überzeugen.

Für diejenigen, die nicht mit der sichersten Methode zum Speichern von Dateien in öffentlichen Clouds sowie zum Teilen mit anderen Benutzern vertraut sind, empfehlen wir, die vorherige Veröffentlichung zu Cloud-Clout on Habré zu lesen.

Das erste, was geöffnet wird, ist die Anwendungs-API. Warum hast du damit angefangen? Denn jede Anfrage in der API erklärt genau, was Cloud-Clout zwischen dem Client und dem Server austauscht und welche Informationen es auf seinem Server speichert. Nachfolgend finden Sie einen Link zur API sowie ein "Kampf" -Beispiel für eine Registrierungsanfrage.

api-staging.cloud-clout.com/swagger-ui.html

Darüber hinaus entwickeln Entwickler das Produkt selbst weiter. Letzte Woche haben sie die Möglichkeit hinzugefügt, die Anwendung per Fingerabdruck einzugeben - viele Benutzer haben dies in Kommentaren und Briefen verlangt. Jetzt können Sie sich viel einfacher und schneller bei Cloud-Clout anmelden.



Die Filesharing-Funktion in der Anwendung wird immer beliebter, daher wird sie auch aktiv weiterentwickelt. Die Möglichkeit, Dateien per QR-Code für andere Benutzer der Anwendung freizugeben, wird jetzt aktiv abgeschlossen, wenn Benutzer nebeneinander sind (das Senden von IDs wird vermieden).



Darüber hinaus werden laufend Feedbacks gesammelt, und die Entwickler möchten Sie zu einem Dialog über die zukünftige Entwicklung unseres Produkts einladen. Auf der Grundlage von Kommentaren zu Artikeln und Briefen von Benutzern wurde eine Liste empfohlener Verbesserungen und potenzieller neuer Funktionen zusammengestellt. Wir empfehlen Ihnen, in Umfragen über Optionen abzustimmen, die Sie für am vielversprechendsten oder am wichtigsten halten.

Und in den Kommentaren bitten die Autoren der Anwendung, über die folgenden Fragen nachzudenken:

  1. Premium Premium Features in der App implementieren - für welche zusätzlichen Vorteile wären Sie bereit zu zahlen?
  2. Versionsentwicklung für Firmenanwender - Welche Produkteigenschaften könnten im B2B-Bereich gefragt sein?
