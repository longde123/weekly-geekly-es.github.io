34 Open Source Python-Bibliotheken (2019)


Wir haben uns 10.000 Open Source-Bibliotheken für Python angesehen und verglichen und die 34 nützlichsten ausgewählt.


Wir haben diese Bibliotheken in 8 Kategorien eingeteilt.


Dieser Artikel wurde von EDISON Software unterstützt, einem Unternehmen für Suchmaschinenoptimierung und SEO , das auch mobile Android- und iOS-Anwendungen entwickelt .

Python-Toolkit


1. Pipenv : Python-Entwicklungsworkflow für Menschen.


2. Pyxel : Entwicklung von Retro-Spielen in Python.


3. PyTest v3.5 : Ein Framework, das beim Schreiben kurzer Tests hilft und für komplexe Funktionstests skaliert.



4. Poesie : Vereinfacht das Abhängigkeitsmanagement und das Verpacken.


5. Loguru : Vereinfacht die Protokolle.


6. Faust : Bibliothek für Streaming / Streaming-Anwendungen.


7. Pampy : Das Muster-Matching, von dem Sie geträumt haben.


8. Pyre-Check : Führende Typprüfung.


9. Delorean , eine Bibliothek zum Arbeiten mit Zeit und Datum.


10. Cirq : Eine NISQ-Schemabibliothek (Noisy Intermediate Scale Quantum).


11. Python-Nubia : Ein Befehlszeilen-Framework.


Web


12. Requests-HTML : HTML-Analyse für Menschen.


13. Bokeh : Interaktive Datenvisualisierung in modernen Browsern.


14. Vibora : schnell, cool, asynchron.


15. Pywebview v2.0 : HTML-Inhalt in einem separaten Fenster anzeigen.


16. WhatWaf : Erkennung und Umgehung von Firewalls und Sicherheitssystemen für Webanwendungen.


17. Geschmolzen : Ein minimalistisches und schnelles Framework zum Erstellen einer HTTP-API.


Terminal


18. Termtosvg : Wir zeichnen Sitzungen der Arbeit mit dem Terminal als SVG-Animation auf.


19. Asciinema v2.0 : Wir zeichnen Terminalsitzungen auf.


20. Termgraph : Werkzeuge für die Kommandozeile, zeichnet Grafiken.


Code-Editor


21. Schwarz : kompromissloser Code-Formatierer.


22. Algojammer : Ein experimenteller Code-Editor zum Schreiben von Algorithmen.


23. Bowler : Ein Refactoring-Tool auf Syntaxbaumebene.


Debuggen


24. Py-spy : Ermöglicht Ihnen die Visualisierung der Zeitverschwendung Ihres Programms, ohne das Programm neu zu starten.


25. Birdseye : Ein grafischer Debugger mit AST.


26. Eis : ein süßer Debugger.


Compiler


27. Transcrypt : Übersetzer Python 3.7 nach JavaScript.


28. Pyodide : Datascience-Stack im Browser.


Daten bezogen


29. Voluptuous : Datenvalidierung.


30. Botflow : Datengesteuertes Programmierframework für die Arbeit mit Datenpipelines (Web Crawler, Machine Learning, Quantitative Trading.etc).


31. Fast-Pandas : Vergleichende Benchmarks bei Pandas-Operationen.


Diagramm


32. Eine Tour im Wunderland der Mathematik mit Python : Eine Sammlung von Skripten zum Zeichnen von wunderschönen Formen und zum Animieren von Algorithmen.


33. Chartify : Hilft Datenwissenschaftlern beim Erstellen von Diagrammen.


34. Hypertools v0.5 : Ein Werkzeug zur geometrischen Darstellung mehrdimensionaler Daten.
