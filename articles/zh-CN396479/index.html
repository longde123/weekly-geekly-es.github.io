科学：为什么膝盖受伤经常无法he愈？


资料来源

在瑞士一个山区国家生活了数年之后，冬天和夏天都在山上滑雪滑雪，令人惊讶的是我的膝关节韧带和软骨仍然完好无损。虽然，为了损坏韧带或弯液面，足够摆在办公椅上或不正确地踢足球是足够的（有先例）。

科学期刊《科学》上发表了一篇文章，解释了为什么膝关节受伤有时甚至终生无法治愈。有关软骨的同位素分析的详细信息，欢迎访问。

前言


在发现所谓的化石遗骸的放射性碳定年法之后，同位素分析方法就广为人知。顾名思义，此方法处理材料和任何其他物质的同位素组成。换句话说，该方法显示了给定原子的主要，主要同位素与“类似物”之间的浓度差：对于碳，主要原子为12 C，13 C和14 C-“类似物”，对于碱性氧为16 O和15 O，17 O，18 O-“类似物”。

每个元素都有自己的同位素系列，包括寿命相对较长的同位素，可以测量其浓度并将其用于任何有用的目的。

情节


如果您，我亲爱的读者，严重摔倒在滑雪坡上或在冰上滑倒，那么很可能您会希望破坏某些东西（例如，手臂或腿），而不是损坏膝盖关节，尤其是弯液面或韧带。一项新的研究表明，与骨骼相对恢复较快的骨骼不同，软骨组织在一定年龄后将永远无法恢复（也许很少有例外）。奇怪的是，已有60年历史的核爆炸使这项研究得以曝光。

运动员，极端的人以及只是膝关节受伤的普通人都可以证实，膝关节受伤往往很长一段时间都不会he愈，他们必须忍受痛苦的感觉，并限制自己的余生。但是，由于很难控制生物体内软骨组织的循环，因此科学家和医生无法完全确定愈合过程是如何实际发生的（以及是否发生）。

为了找到答案，迈克尔·基尔（Michael Kier）和他的同事们使用了放射性同位素方法，通过测量组织中14 C 的浓度来找出分子的年龄。碳量14自核时代开始和1950年代进行积极的核试验以来，大气中的C急剧增加。但是，由于禁止地面核爆炸，1963年以后，它在大气中的数量迅速减少。

当然，一部分碳与食物和空气一起进入了人类有机体。如果该分子是“旧的”分子，并且在一生中没有被自然取代，那么14 C 同位素的含量将对应于他还是儿童或少年时的大气水平。也就是说，在60-70岁的患者中，应观察到组织中同位素水平的升高。相反，如果在恢复软骨，新的“原料”用于新分子的构建，耗尽了同位素的14C，即其浓度将接近当前的大气水平。

因此，借助这种方法，科学家研究了2000年之前出生的22例因医学原因进行手术以更换软骨组织的患者的膝关节中的软骨中14 C 同位素的水平。我们检查了高负载水平（在关节中心）和小负载（沿边缘）的两个地方。

事实证明，在8至13岁的患者中，胶原蛋白中14 C 同位素的水平（虽然检查了其他具有相似结果的有机分子，但软骨组织的主要塑性成分）却接近大气水平。在1935年核时代之前出生的一名患者中，胶原蛋白实际上不含这种同位素。相反，在1950年代出生的几名患者中，组织中14 C 的水平显着升高，证实了最初的假设。

这种直接测量软骨中胶原蛋白转换的方法（不幸的是，是侵入性的）与先前使用间接观察方法的研究相违背。即使在关节的那些承受重负荷的区域中，成年人身体也无法再生胶原质，即软骨。

科学家还尝试了多种刺激软骨生长的方法，包括干细胞注射法。然而，事实证明，它们对软骨的恢复没有任何显着影响。事实证明，这一发现实际上终结了医学的整个分支，该分支使用这些方法“自我修复”了膝关节的组织。

这项研究再次强调保护膝盖关节和韧带免受极端压力和损害的重要性。北卡罗来纳大学的生物学家理查德·洛瑟（Richard Loeser）同意这一结论：“年轻时必须照顾好关节。在您损伤了半月板或韧带后，[喝宝露美为时已晚]，因为它们将无法恢复自我。”

原始文章“放射性碳测年显示出健康和骨关节炎人类软骨中的胶原蛋白转换最少 ”，发表于《科学转化医学》（DOI：10.1126 / scitranslmed.aad8335）。
