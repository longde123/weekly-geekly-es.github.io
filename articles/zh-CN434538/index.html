太空之战：卫星互联网市场正在发生什么

12月中旬，美国联邦通信委员会（FCC） 批准了LeoSat卫星互联网项目。 该网络将于2019年开始部署。 SpaceX Ilona Mask还将其设备发射入轨道。 我们谈论公司的计划，并谈论为什么在轨的大量航天器不是一个好主意。


/ Flickr / NOAA / PD

卫星互联网LeoSat


LeoSat网络将由 78颗卫星组成 。 它们将位于距地球1400公里的高度，这比其他卫星系统（例如GLONASS或GPS）的轨道低十倍。 根据公司代表的说法，设备与地球的接近将使数据在地球上不同点之间的传输比通过光纤快一倍半。 例如，从伦敦到新加坡，LeoSat信号将在119毫秒内到达，而光纤信号仅在181毫秒内到达。

明年，LeoSat将把前两颗卫星发射到太空，并将立即向用户提供。 他们计划在2022年之前全面部署该网络。

每个LeoSat航天器都配备有四个激光器，通过它们与相邻设备进行通信。 另外，卫星有十个天线。 使用Ka波段无线电波与地球终端通信时需要使用它们。 数据传输速率预计为1.6 Gb / s 。

LeoSat的客户将是5G运营商。 根据电信行业的代表，新一代的地面通信将不足以用于网络-所需的5G信号传输速度是世界平均Internet连接速度的15倍。 卫星可以补充通信系统，并显着加速数据传输。

还有谁将卫星发射到太空


除LeoSat外，还有几个卫星互联网项目已获得FCC批准。 其中，最著名的是Starlink和OneWeb。

Starlink是Elon Musk和SpaceX的项目，我们已经在前面进行了更详细的讨论。 SpaceX已将两颗卫星原型机送入轨道，并准备在2019–2020年部署第一批1,600架航天器。 现在，该公司正在完成一笔额外的5亿美元投资，用于网络的开发。

许多公司已经投资了 OneWeb卫星技术，包括空中客车，维珍集团，高通等。 航天器原型计划于2019年发射。 2月7日，搭载十枚OneWeb卫星的联盟号火箭将从库鲁国际宇宙飞船发射入轨道。 设备总数将为 600件。

另一个获得FCC批准的项目是TeleSat Canada。 该公司将向轨道发射117颗卫星。 该网络计划于2021年部署。


/ PNG / 免费照片 / PD

Facebook还在为发展中国家开发互联网。 在2018年7月，该公司确认正在制造自己的卫星，并且已经向FCC提交了申请。 另一个项目是LinkSure。 这是一家中国公司，正在开发全球免费卫星互联网系统。 首架航天器将于2019年发射，整个LinkSure网络计划于2026年部署。

轨道堵塞


关于那么多卫星系统是否会受益的观点截然不同。 IT行业的一些代表认为，技术将为生活在农村地区或发展中国家的人们提供帮助。 从事太空项目的非营利组织安全世界基金会（Secure World Foundation）的克里斯塔威尔逊（Krystal Wilson） 表示 ，在这种情况下，找到可以服务于地面网络基础设施的专家将非常重要。

有些人坚信卫星互联网更可能不是针对穷人，而是针对富人。 甚至由此类解决方案的开发人员也指出了这一点。 LeoSat首席执行官表示，他的公司正在为大型公司提供优质服务。

伦敦大学学院的马克·汉德利教授对Starlink的工作进行了建模，也表达了相同的观点。 据他介绍，仅发射一颗卫星就要花费数千万美元。 因此，SpaceX的客户将是愿意为最快的Internet连接速度付出高昂代价的国际公司。

一些技术专家甚至怀疑卫星互联网的概念。 根据斯坦福大学教授扎克·曼彻斯特（Zac Manchester）的说法，在1990年代后期，几家公司试图用低轨道卫星网络代替蜂窝通信。 空间系统的发展比地面系统慢，这就是为什么卫星项目无法吸引客户并封闭的原因。

曼彻斯特认为，卫星互联网的现状与电话运营商的经历没有什么不同，因此，SpaceX和OneWeb项目可能无法满足创建者和投资者的期望。

专家指出，卫星互联网公司将不仅要解决其服务成本。 有两个重大的技术困难。 首先是空间碎片。

根据FCC的说法，每隔六年，一个人就会掉落45％的SpaceX设备碎片。

鉴于在近地轨道上将有成千上万其他公司的卫星，对地球居民的危险只会增加。 在这种情况下，残骸只会堵塞轨道。 鉴于Starlink卫星大约每五年需要更换一次，因此空间碎片的数量将非常巨大。

11月下旬，联邦贸易委员会甚至提出了一些监管文件，以解决空间碎片问题。 欧美国家的政府正在开发跟踪碎片和“清理”轨道的系统。


/ Flickr /美国航空航天局 / PD

第二个困难是系统的信息安全。 如果发给卫星之一的信号被攻击者截获，它们将能够影响整个网络的运行。 这不仅会导致Internet连接失败，还会导致航天器碰撞。 尽管他们已经在解决此问题。 选项是对无线电信号进行加密或对来自地面站的命令进行强制认证 。

接下来是什么


卫星互联网的竞争将继续增长。 已经有一些公司急于赶超对手，以至于他们在未经国家机构批准的情况下发射了航天器。 例如，美国初创公司Swamp Technologies 将卫星从印度航天中心送入轨道，却没有等待FCC确认申请。

但是，该技术的潜力仍不清楚。 专家估计，卫星互联网将在未来4-8年内真正展现自己。 只有到那时，两家公司是否设法找到光纤连接的替代产品还是卫星的期望值过高，这才变得很清楚。
