电子服务及其位置的传奇。 第2部分。电子柜

在上一部分， 第1部分。电子服务中，我们研究了什么是整体电子服务。 现在让我们谈谈其提供的地方。

三，综合电子服务组织

在我领导的高度集中化组织的最低层某个地方，有些人在做我的工作，事情进展顺利，而无需我付出很多努力。 我相信这是因为我是一个好工人。
约瑟夫·霍勒
因此，我们确定：

  1. 电子服务是一种具有计算机或电子形式的供需以满足用户需求的服务；
  2. 为了高质量地提供电子服务，该服务必须从其消费者那里获得完整而可靠的信息。 为此，他必须提供一种格式，该格式定义用户期望的数据的形式和组成。
  3. 电子服务可以彼此交互，而无需人工干预。

1.电子服务互动的组织


在生活中，经常发生仅获得一项简单服务是不够的。

首先，正如我们之前所说，要提供服务，您可能需要只有其他电子服务才能提供的信息。

其次，可能有必要与服务的客户明确其提供服务的各种细微差别，提供服务的各种条件等。

第三，经验丰富的人可能需要获得一系列相互联系的服务。

例如，某个公民需要在某个地址进行注册。 但是为此，他应该首先从先前的注册地点签出。 理想情况下，他应该仅提交一份全面的“注册提取”服务申请，该申请将指出两项操作所需的完整信息。 它有什么作用？

除了在不同权限之间转移应用程序和对它们的答复来摆脱不必要的手势之外，操作仍然可以事务性地进行。 即，如果其中一个操作由于任何原因都不可行，则不会执行两个操作。 因此，在我们的示例中，如果某人无法在新地址注册，那么他将不会在旧地址被解雇。 而且您知道，它消除了很多问题，请检查一下，我是第一手知道的。

为了组织这样的活动，有必要根据所提供信息的质量和完整性以及服务的可集成性，清楚地介绍不同服务的应用程序处理算法。 为此，应制定技术路线，并通过该技术路线，将服务使用者的要求转换为他感兴趣的结果。

因此，除了电子服务本身之外，还必须有一种将其组织为执行路径的机制。 在网站上填写电子服务申请的选项是组织这种机制的最简单示例。

2.电子柜


我们再次回到服务位置的问题，因为这个主题恰恰是本研究客户的主要愿望。 刚才，让我们看一下提供服务的地方，以及作为上述组织提供电子服务的途径的机制。

关于橱柜的思考。

根据上述内容，可以得出结论，如果电子服务是完全自动化的，则不需要任何办公室。 她在虚拟现实中需要一个小窗口，允许一个人提交申请并获得答案。

但是，如果在提供此电子服务的过程中需要人工干预，则可以考虑使用虚拟办公室作为与该主题进行交互交互的一种方式。 而且，在这种情况下的干预，您不仅可以感知服务提供商的影响，还可以感知其消费者的影响。

从这个意义上讲，服务交付办公室在某种程度上类似于WEB门户。 这是他的定义：
Web门户网站-计算机网络上的一个站点，可为用户提供在该站点内运行的各种交互式Internet服务。 一个Web门户可以包含多个站点。

Web门户的概念是在一个地方提供最大数量的Internet服务，以吸引最大数量的用户。
这给了我们什么？

首先，机柜可以提供用于操纵电子服务的各种工具。 即，消费者不仅可以构成应用程序，而且可以例如为其处理建立路径，选择用于获得结果的选项等。

其次，在这样的办公室中，重要文件的装订者可以再次存储为电子的，该文件是在服务的接收者与办公室负责人通信期间生成的。 不一定是办公室的负责人，这是某个人。 负责人随时可能变更。 它更像是一个帖子。 从这个意义上说，最主要的是代表一个人代表内阁行事-一个有权代表他的授权人。 即，电子服务的消费者可以相信结果，并希望不分发他提供的机密数据。 因此，第三，办公室可以识别（识别）服务提供商。

在检查的材料中，橱柜具有三个重要功能：

  1. 根据某些规则提供对某些电子服务的访问；
  2. 收集和提供有关与办公室提供的服务的消费者互动的信息；
  3. 代表办公室提供或协助提供服务的主题（人员，组织，职位等）的标识；

让我们用一个具体的例子来检验这些假设。

电子办公室运作的最成功例证也许是建议医生办公室，当然是虚拟的。 还有一些服务对于使整体结果负责的人进行电子化和强制性干预要更加有效。 在这种情况下，从字面上最真实的意义上讲，结果有时是至关重要的。

例如，服务1-从患者那里获取读数，以确定应该首先发送到的办公室（纯电子）。 接下来-队列，至少是电子队列，允许医生和患者协调计划，这是服务2。 结果，将在医生的电子办公室提供的患者电子卡已经包含了专家开始预约所需的初步信息。

该技术本身可以在诊所中某个地方的客观现实中发生：触觉，视觉，嗅觉等。 在下文中，医生可以在患者的电子卡中进行某些输入后，请求服务（当然是电子的）在药房的庞大电子空间中搜索药物，以便从字面上理解最佳选择。 原则上，这些动作可以由护士代表执行，该护士也代表同一电子办公室，但只能作为总包装中包含的辅助服务的执行者。

考虑到办公室（主管人员）为他确认的收益，患者可以根据需要使用电子服务预订他喜欢的商品，甚至可以使用电子钱包付款。

服务的使用者可以使用提供给他的漂亮工具在这个美丽的电子办公室中执行所有这些操作，但是已经不需要医护人员了，从而为坐在电子生产线中的其他患者节省了宝贵的时间。

值得关注的事实是，所描述示例中的电子办公室不仅需要医生（服务提供商），而且需要患者（消费者）。 就患者而言，同样需要办公室的所有相同功能：在服务和福利申请中确定自己的身份，并存储其提供的数据。 这样的服务接收者的办公室通常称为“个人”。

因此，考虑一个示例，我们发现了第四点，必要时可以识别服务的使用者。

3.电子机构


事实证明，服务的消费者可以在提供各种电子服务的各种电子机构中拥有许多个人帐户。 例如，诸如电子商店，公共服务站点，MFC网站等。

另一方面，每个此类机构除了建立自己的办公室（WEB门户）以提供电子服务外，还为客户提供单独的个人帐户，以供他们在此电子教育框架内进行授权的交互。


图4.电子柜的组织

在图4中，除了（由注册用户）授权接收服务外，还提供了一个选项供随机用户（无个人帐户）未经授权使用。 这样的机会并不代表所有电子机构，也不代表所有服务。 但是，至少必须有一项这样的服务，因为新用户需要以某种方式进行注册，获得自己的个人帐户并成为“选择的帐户”之一。

4.总结


总结以上所有内容。

电子服务必须具有：

  1. 解决访问问题的能力；
  2. 该机制为其提供条件形成条件。 某种格式的应用程序和用于将应用程序传递到其执行设备的接口；
  3. 渲染机制；
  4. 提供其提供结果的机制；

电子账户很有意义，主要是关于电子服务的提供/消费，并且此服务是主要的。

电子柜可用作：

  1. 根据规则，某些电子服务的接入点；
  2. 展示电子办公室代表及其提供的电子服务的公共信息存储场所；
  3. 用于存储有关实体在与办公室互动的框架内进行的活动的公共信息的场所；
  4. 代表内阁提供或协助提供服务的实体的标识符（真实性检查）；
  5. 申请向他提供服务的对象的标识符（真实性检查）。 包括，用于组织反馈；
