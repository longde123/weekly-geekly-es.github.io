橙色pi零视频监控-便宜且一点也不生气

最近，我为自己制定了具有以下特征的视频监控任务：

  1. 最低功耗
  2. 从摄像机（或多台摄像机）接收视频的设备的最小尺寸。 如此小巧的尺寸使其可以塞入10-10厘米的安装盒中或隐藏在宽阔的电缆槽中（！）
  3. IP摄像机将通过wi-fi连接，也就是说，无法确定谁在观看/写入视频。 您可以从普通网络为它们供电，无论是访问照明还是办公室的插座
  4. 必须有一个视频记录（在我的情况下-在microsd卡上），必须有可能通过Internet从世界任何地方以及通过移动应用程序观看视频
  5. 当然，应该可以远程配置整个系统-添加新摄像机，复制删除录像，可以访问所有设置
  6. 最低费用！ 但是呢？ 没有它，无处不在

为了实现此想法，选择取决于以下组件：

1）最便宜的带有Wi-Fi的单板计算机，您可以在其上滚动一些arm linux发行版-例如，我最喜欢的debian。 Malinka很贵，但是在ebay上的橙色pi零仅需900卢布（几个月前，我以700卢布的价格全部购买了）。 这是到ebay的链接- 在这里。



2）实际上，wifi ip摄像机-没有它们就无处不在。 小巧的圆顶天花板。 至少720p-通常不再需要在办公室门廊中进行视频监控。 每台相机的发行价约为1,500卢布。 有多远，在速卖通上- 在这里



3）32 GB microSD卡。 为什么是32，而不是更多或更少？ 因为就我而言，您需要能够几天前从两台摄像机倒带视频。 好吧，那么，另一个300-400卢布。

4）单板和摄像机的电源适配器。

那么已经做了什么。 收到邮件中的珍贵包裹后，我从那儿拿出一个橙色的单付款人，检查了一下，首先想到的是-该死，但没有视频输出。 如何安装debian和视频监控软件并进行初始设置？ 我立即想起，在树莓上总是有一个普通的复合AV，但在这里却不是。 好吧，这不可能是制造商没有留下通过复合材料进行连接的机会-我想，他去了Google观察GPIO引脚排列。 是的，第二针是大地，第九针是电视输出。

焊接AV插座。 现在，我们将使用AV-> VGA适配器（您猜想它也可以在速卖通上购买，这很有用）并连接到我们的显示器。 但是我们还不会提供任何东西-毕竟，我们还没有准备好microSD卡。

我们访问网站www.orangepi.org并下载Debian xfce桌面发行版，使用win32磁盘映像器或蚀刻机将其写入卡中。 我们将USB闪存驱动器插入橙色，打开显示器，连接电源适配器，然后…什么都没有。 这是新手最常遇到的问题-单板计算机不会以任何方式启动，而只会发出发亮的黄色二极管。 这是什么 错误记录的卡（可能是卡），无法正常工作的单板卡（不太可能，因为我们刚刚拆箱了），视频输出中没有信号吗？ 最常见的问题是营养。 橙色pi像覆盆子和其他单板的一样，非常耗电-适配器应为5 V / 2 A，如果适配器较弱并且在打开单板的最初几秒钟内电源下降，则它将无法工作，并且将无法工作。 因此，请使用功能强大的强大电源，不要从旧的android进行充电。

我们连接了一个好的电源适配器，我们看到了debian下载是如何进行的。 我们看到了桌面-太棒了！ 现在，您需要配置设备，使其可以通过Wi-Fi运行。 我们进入网络连接，创建新的Wi-Fi连接，然后从家庭办公室Wi-Fi输入我们的用户名和密码-现在该设备位于我们的本地网络上并且可以访问Internet！ 现在我们需要摆脱监视器。

为了能够远程查看桌面，对于单板vnc服务器，vnc Viewer程序将为我们提供帮助。 通常，这个东西已经内置在由这些单板产品的制造商分发的debian中，您只需要配置它-输入授权参数，等等。 确保我们现在可以访问本地网络中的桌面之后，我们卸下显示器，将设备隐藏在盒子/电缆通道/其他地方，并通过vnc查看器配置其他所有设备，同时坐在普通计算机上。

作为视频监控软件，选择了zoneminder。 为什么是他？ 免费，灵活，易于配置，可在带debian的单板手臂版本中使用。 该软件可在php / mysql上运行，因此首先需要安装必要的软件包： 

sudo apt-get install php5 mysql-server php-pear php5mysql php5-gd

接下来，我们需要在/etc/apt/sources.list中添加该行： 

 deb http://ftp.debian.org/debian jessie-backports main

我们进行更新： 

 sudo apt-get update

最后，从我们所有人的出发： 

 sudo apt-get install zoneminder

当控制台通知我们已安装zoneminder时，让我们继续其配置。 配置过程已在zoneminder官方网站上进行了详细描述，在本文​​中没有列出所有要点，因此，我们决定由亲爱的读者来完成此操作，现在，如果我们在单用户/ zm的计算机地址上输入浏览器，我们将转到基于Web的视频监视管理面板。 在这里，我们需要添加摄像机的IP地址，设置视频录制的参数，等等。



现在，我们可以通过内部网络，有线网络和Wi-Fi通过浏览器从摄像机观看视频。



但是任务是从世界任何地方通过Internet观看视频-我们尚未实现这一目标。 因此，让我们实施吧！ 至少要想到两个选择：

  2. 我们有一个专用的IP，转到路由器设置，转发必要的端口，配置虚拟服务器，现在我们可以看到zoneminder管理面板，例如，在our_white_ip ：123。 很好，但是如果没有专用的IP怎么办？
  3. 如果没有专用的ip，请重新调用vpn和dyndns。 有关此问题的大量文章，包括关于哈布雷的文章；查找和查找信息
