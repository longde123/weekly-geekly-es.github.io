Veeam可用性控制台2.0 Update 1中有哪些新增功能？

您还记得，在2017年底，针对Veeam可用性控制台服务提供商的新免费解决方案已发布，我们在博客中对此进行了讨论 。 使用此控制台，服务提供商可以远程管理和监视Veeam解决方案所在的虚拟，物理和云用户基础架构的安全性。 新颖性很快得到认可，然后发布了第二个版本，但是我们的工程师并没有为此感到吃惊，并且在6月底为Veeam Availability Console 2.0准备了U1的第一个更新。 关于他的是我今天的故事，欢迎您关注。

那些计划在不授予员工足够权限的情况下委派对Veeam可用性控制台的访问权限的人员（例如，本地管理员）现在可以为员工分配“ 门户网站操作员” 操作员的角色。 该角色使您可以在Veeam可用性控制台中执行基础结构的管理和监视，但不包括对解决方案配置的访问。 在此处阅读有关门户操作员角色的设置的更多信息。

与ConnectWise Manage集成


ConnectWise Manage用户现在可以访问在Veeam可用性控制台中实现的管理，监视和计费功能。 集成由ConnectWize Manage插件提供，可以在Veeam可用性控制台的“ 插件库”选项卡上看到。 该插件使您可以使用所谓的集成功能在两种产品之间传输数据-您可以将它们表征为要同步的某些类型的数据的出入口。 （也许我称它们为功能），特别是因为此名称出现在文档中。）稍后再对其进行介绍，但现在我们将弄清楚如何启用与ConnectWise Manage的集成。



步骤1：产生API金钥


  1. 启动ConnectWise Manager桌面客户端。
    注意：您将用来登录的帐户必须具有必要的权限，如此处所示。
  2. 选择右上角的我的帐户 。
  3. 在API密钥标签上，点击新建项目 。
  4. 在“ 描述”字段中输入新密钥的描述 ，然后单击“ 保存” 。
  5. 将显示新密钥（公共密钥和私有密钥），必须将其复制并存储在安全的地方。

步骤2：配置插件连接


  1. 启动Veeam可用性控制台； 您将用来登录的帐户必须具有Portal Administrator角色。
  2. 点击右上角的配置 。
  3. 在左侧面板中，选择插件库 ，然后单击ConnectWise Manage 。
  4. 在打开的窗口中，输入连接参数：
    • ConnectWise网站 -输入网站地址
    • ConnectWise公司 -指定组织名称
    • 公钥，私钥 -输入在步骤1中创建的密钥。



  5. 单击连接 。
  6. 在“ ConnectWise管理集成”对话框中，确认状态显示为“ 健康”图标。

步骤3：启用整合功能


  1. 启动Veeam可用性控制台； 您将用来登录的帐户必须具有Portal Administrator角色。
  2. 点击右上角的配置 。
  3. 在左侧菜单中，选择插件库 ，然后单击ConnectWise Manage 。
  4. 在“ 集成设置”部分中，将必要的开关设置为“ 开” （可以使用“ 全部启用”选项）。 有关它们的更多详细信息，请参见下文。



使用功能进行数据同步


以下是此版本中与ConnectWise Manage插件一起使用的功能（集成功能）：

  • 公司 -允许您在服务消费者公司中选择要在Veeam可用性控制台和ConnectWise Manage之间进行数据同步的公司。 激活此功能后，Veeam可用性控制台将从ConnectWise Manage接收消费者公司的列表，然后您可以配置映射以同步所需公司的数据。 可以在此处找到更多详细信息（英文）。



  • 配置 -帮助您在ConnectWise Manage中为运行Veeam可用性控制台的计算机创建配置文件。 这些服务器可以是Veeam Backup＆Replication服务器，也可以是安装了Veeam Availability Console代理并且包含在具有配置映射的公司的用户基础结构中的虚拟机和物理机。 激活此功能后，Veeam可用性控制台会为每台此类计算机创建一组设置，并为其分配Veeam受管计算机配置类型。
  • 票证（创建和处理服务请求）-允许您在ConnectWise Manage中创建票证。 订单基于警报，警报是在某些条件下在具有自定义映射的公司的Veeam Availability Console中触发的。 例如，这可能是备份操作失败，超过了存储库的配额等。 每个应用程序都包含与触发的警报关联的计算机的配置。

    激活此功能后，您可以在Veeam可用性控制台中配置新创建的请求的参数。

    有用的 ：处理完应用程序并在ConnectWise Manage中将其关闭后，Veeam可用性控制台中相应的问题通知也将自动转移到已解决状态，也就是说，不需要其他手动操作。



  • 计费 -通过此集成选项，提供商将能够在ConnectWise Manage生成的帐户中包含有关基于Veeam解决方案提供的服务的信息。 激活此功能后，Veeam可用性控制台将从ConnectWise Manage目录中接收产品列表以及与消费者公司签订的合同的必要数据。 然后，您可以配置服务和产品的映射，并根据发生应计费用来指定合同。

客户证实了集成解决方案的有效性-Vertisys总裁Matt Baldwin说：“集成使我们的备份和DRaaS软件包更具吸引力。 从我们的角度来看，优点包括简单，方便的界面以及最佳功能。 我们计划在一年内解决方案将帮助节省50-60个工时。”
