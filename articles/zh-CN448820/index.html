锈+里昂=爱



哈Ha！ 总的来说，正如我在这里所承诺的，这是一个有关如何在MacOS上将CLion用作Rust的IDE的故事。 为什么不使用IntelliJ IDEA？ 因为CLion可以亮相。 为什么选择LLDB而不选择GDB？ 因为LLDB是随Xcode一起安装的。 我喜欢这个IDE，因此决定向您介绍。 如果您有兴趣，那就欢迎猫。

防锈安装


只需在终端中执行命令： 

curl https://sh.rustup.rs -sSf | sh

并按照说明进行操作。

您还可以安装fmt-格式化实用程序和clippy-静态代码分析实用程序。 如果操作系统支持。 它们在Windows和Ubuntu上对我有效，但在Mac上不起作用。 尽管情况可能已经改变。 

 rustup component add rustfmt 

 rustup component add clippy

安装LLDB


如果您没有安装Xcode，请在终端中运行命令 

 lldb

并按照说明进行操作。

CLion安装


在官方网站上下载。 链接： Tyk 。 好吧，像往常一样，按照说明进行操作。

插件安装

  2. 转到插件


  3. 安装插件



工具链设置


  1. 创建一个新项目


  2. 通过单击链接或指定安装路径来添加标准库


  3. 选择LLDB作为调试器


  4. 选择Rust的首选设置



添加配置以运行


  1. 点击添加配置


  2. 选择货运命令





侦错


  1. 放置一个断点，然后单击带有错误的按钮

  2. 在底部，您将看到一个带有调用堆栈和变量值的窗口

个性化
