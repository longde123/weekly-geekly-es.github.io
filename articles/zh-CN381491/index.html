Weather Point 2.0：钥匙扣形式的通用气象站



气象站是一种非常流行的电子设备类别。该设备可以确定房屋内部和外部温度和湿度的平均功能，有时-室外风速加上压力。但是很显然，这些都是固定装置，不太可能在路上用来确定亚马逊丛林中的气象条件（为什么不呢？）。

一个有趣的项目现已出现在Kickstarter上-一个以钥匙串形式出现的气象站Weather Point 2.0。该设备可以检测温度，湿度，紫外线强度和大气压。而所有这些-大小只有半笔的小工具。

Weather Point 2.0-车站可以做什么？



开发人员将该设备称为“个人气象站”，该气象站一直在您身边。如上所述，该设备可以测量：
  • 紫外线辐射强度
  • 气温
  • 湿度;
  • 压力。


同时，所有数据都通过无线方式传输到手机或平板电脑（这无关紧要，该小工具可与iOS和Android兼容）。

随时随地，Weather Point 2.0的所有者将能够获取有关当地天气状况的详细信息。

小工具的外壳是金属（尽管您可以选择塑料外壳），但可以防止水和冲击。



开始工作


为此，只需下载Weather Point 2.0应用程序，然后执行与移动设备配对的过程。此外，Weather Point 2.0具有音频连接选项，即标准连接器。小工具从手机电池接收能量。



顺便说一下，小工具还可以根据压力数据来计算海拔高度。

不使用设备时，可以将其与钥匙一起佩戴；气象站没有丢失。



如上所述，水和震动不会使设备感到恐惧。



开发人员还提供了对应用程序API的访问，该API允许您在任何程序中使用设备传输的数据。



好了，您可以在许多情况下使用Weather Point 2.0。这是体育，钓鱼，狩猎，旅行。



它要多少钱？


现在，您可以以29美元的价格预订Kickstarter。开发人员还提供带有2-3和许多Weather Point 2.0的预购软件包。
