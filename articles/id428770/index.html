Makro keyboard untuk tugas sehari-hari

Halo semuanya.

Di antara masyarakat, entah bagaimana terjadi bahwa merekam makro pada keyboard adalah murni kebutuhan gaming dan seringkali hanya ditemukan pada keyboard gaming.

Tapi sebenarnya itu hanya catatan dari urutan tombol yang ditekan.

Jadi mengapa tidak menggunakannya untuk kegiatan biasa. Dan terutama di lingkungan pengguna tunanetra, ketika menggunakan mouse sangat merepotkan, dan semua navigasi di windows dilakukan hanya dengan menggunakan satu keyboard.


Contohnya


Saya akan mencoba memberikan beberapa contoh untuk pemahaman.

Varian dari urutan yang diperlukan tentu dapat muncul dengan banyak sekali.


1. Membuka konsol di folder explorer saat ini


Tentunya banyak orang tahu bahwa jika Anda mengklik kanan pada sebuah explorer dari awal dengan tombol shift ditekan, maka dalam menu konteks yang terbuka, Anda dapat memilih opsi "Buka jendela perintah".

Untuk memanggil menu konteks pada keyboard ada juga tombol menu khusus.

Hal utama adalah menggunakannya dari awal, tanpa memilih objek apa pun.

Untuk melakukan ini, pertama-tama Anda dapat menggunakan metode sederhana untuk menavigasi ke folder induk dan kembali.

Maka makro akan terlihat seperti ini:


Alt+_
Enter
Shift+__
_
Enter

2. Luncurkan verifikasi kode untuk kepatuhan dengan gaya kode


Saat memprogram, penting untuk memeriksa kode Anda untuk kesesuaian dengan gaya desain kode yang sudah ada dan utilitas flake8 menarik untuk ini.

Kemudian, untuk memverifikasi kode dengan bantuannya, di konsol yang dibuka oleh makro sebelumnya, Anda dapat menulis di memori:


flake8 %CD% > flake8.log
Enter

Setelah memeriksa, file log akan direkam di folder, setelah itu Anda dapat dengan cepat memperbaiki kekurangan Anda.


Kesimpulan


Artikel ini bukan panduan.

Tujuan artikel ini adalah untuk membahas kegunaan dan kerugian dari macro keyboard yang dapat diprogram, bukan untuk game, tetapi untuk tugas-tugas lain.

Dari sisi positif, saya dapat mencatat kenyamanan melakukan tugas yang sering diulang dengan menekan hanya satu tombol.

Di antara kekurangannya: keyboard memainkan urutan dengan kecepatan yang sama dan penundaan antara klik saat Anda memasukkan saat merekam.

Dari makro, saya ingin bermain dengan kecepatan lebih cepat. Mungkin seseorang di komentar akan memberi tahu apakah mungkin untuk mengatasi nuansa ini.
