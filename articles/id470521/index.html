Dirilis 3CX V16 Pembaruan 3 dan aplikasi seluler 3CX baru untuk Android

Pekan lalu, kami menyelesaikan tahap pekerjaan yang besar dan merilis rilis terakhir Pembaruan 3CX V16 3. Berisi teknologi keamanan baru, modul integrasi dengan HubSpot CRM dan item baru yang menarik lainnya. Mari kita bicarakan semuanya secara berurutan.

Teknologi keamanan


Dalam Pembaruan 3, kami fokus pada dukungan yang lebih lengkap untuk protokol TLS di berbagai modul sistem.

  • Level protokol TLS - parameter baru algoritma transport / enkripsi SSL / SecureSIP "di bagian" Pengaturan "â†’" Keamanan "menetapkan kompatibilitas server PBX dengan TLS v1.2. Dalam Pembaruan 3, parameter ini diaktifkan secara default - ini menonaktifkan kompatibilitas dengan TLS v1. 0. Nonaktifkan opsi ini jika Anda memiliki masalah dalam menghubungkan perangkat SIP lawas.
  • Menghubungkan batang SIP melalui TLS adalah opsi baru dalam parameter trunk - "Transport Protocol" - TLS (Transport Layer Security). Untuk menghubungkan trunk terenkripsi melalui TLS, aktifkan dan unggah sertifikat keamanan operator SIP (.pem) ke PBX. Seringkali, Anda juga perlu mengaktifkan dukungan SRTP di bagasi. Setelah itu, saluran komunikasi terenkripsi antara PBX dan penyedia akan berfungsi.



Widget yang diperbarui untuk Obrolan & Obrolan Langsung 3CX


3CX V16 Update 3 hadir dengan widget versi baru untuk 3CX Live Chat & Talk . Itu menambahkan opsi tambahan, misalnya, menetapkan tautan ke akun Facebook dan Twitter. Selain itu, sekarang Anda dapat secara otomatis membuat kode widget untuk penempatan di situs (jika situs Anda tidak berfungsi di WordPress CMS).



Seperti yang Anda lihat, sekarang Anda tidak perlu membuat widget kode HTML secara manual. Itu dihasilkan di bagian "Opsi" â†’ "Integrasi dengan situs / WordPress". Parameter widget dibahas secara lebih rinci dalam dokumentasi .

Integrasi HubSpot CRM




Pembaruan 3 memperkenalkan integrasi dengan sistem CRM terkenal lainnya - HubSpot CRM. Seperti halnya CRM lainnya, integrasi mendukung fitur-fitur ini:

  • Click-to-Call - Panggilan langsung dari antarmuka CRM melalui 3CX Diver .
  • Membuka kartu kontak - kartu kontak atau timah di CRM terbuka saat ada panggilan masuk.
  • Log interaksi - semua percakapan dengan klien dicatat dalam riwayat interaksi CRM.
  • Jika tidak ada nomor penelepon yang ditemukan, sistem dapat membuat kontak baru dalam CRM.

Panduan komprehensif untuk berintegrasi dengan HubSpot .

Meningkatkan pengalaman pengguna


  • Inisialisasi server web PBX - saat memperbarui sertifikat SSL server web PBX (jika Anda memiliki FQDN server yang dikeluarkan oleh 3CX), server nginx tidak memulai kembali seperti sebelumnya. PBX hanya mengunduh dan menginisialisasi sertifikat baru. Penting bahwa panggilan aktif tidak terganggu.
  • Koneksi ulang otomatis - di aplikasi seluler 3CX Android, koneksi ulang muncul ketika koneksi terputus, misalnya, ketika pengguna beralih dari Wi-Fi ke jaringan 3G / 4G. Menghubungkan kembali hanya akan berfungsi jika versi terbaru aplikasi Android 3CX diinstal (lihat di bawah).
  • Pemberitahuan PUSH untuk status - sekarang Anda dapat secara individu mengaktifkan atau menonaktifkan pemberitahuan PUSH untuk setiap status pengguna. Selain aplikasi yang sebenarnya, pemberitahuan dapat dikonfigurasi untuk pengguna dalam antarmuka manajemen 3CX.

Fitur Klien Web Baru




  • Nama untuk percakapan grup - sekarang Anda dapat menentukan nama percakapan grup, dan itu akan ditampilkan kepada semua peserta obrolan di klien web dan aplikasi untuk Android dan iOS.
  • Seret dan lepas lampiran ke dalam obrolan - Anda sekarang dapat menarik dan melepaskan file dari tipe yang didukung ke jendela obrolan, dan itu akan dikirim ke peserta lain.
  • Penyesuaian otomatis ponsel cerdas - kode QR pribadi telah muncul di antarmuka web-klien untuk pengaturan cepat aplikasi seluler 3CX.

Parameter SIP Trunk Tambahan


  • Proxy Standby SIP - opsi baru "Standby proxy" memungkinkan Anda untuk menambahkan server SIP siaga, jika opsi ini disediakan oleh penyedia VoIP. Ini menyederhanakan konfigurasi batang SIP yang tahan terhadap kesalahan, menghilangkan kebutuhan untuk batang cadangan tambahan.
  • Pekerjaan yang ditingkatkan dengan DNS - Deteksi Otomatis, Protokol Transport, dan parameter mode IP secara otomatis menyesuaikan dengan berbagai persyaratan operator VoIP, menerima informasi dari zona DNS.
  • Menggabungkan 3CX jembatan dan konfigurasi batang - untuk menyederhanakan antarmuka manajemen, tombol konfigurasi untuk Jembatan, batang SIP dan gateway VoIP sekarang terletak di satu bagian.

Aplikasi Android 3CX Baru


Bersama dengan 3CX v16 Update 3, kami merilis aplikasi 3CX baru untuk Android. Ini sudah dioptimalkan untuk Android 10 (Android 7 Nougat, Android 8 Oreo dan Android 9 Pie juga didukung) dan dirancang untuk bekerja dengan Pembaruan 3CX v16 3 dan lebih tinggi. Aplikasi ini menggantikan klien saat ini untuk Android.

Aplikasi ini memiliki antarmuka baru yang menyediakan kecepatan tinggi dan fungsionalitas yang dapat diperluas. Fungsi-fungsi lanjutan telah ditambahkan, misalnya, pemberitahuan PUSH tergantung pada status pengguna, prioritas panggilan GSM atas panggilan VoIP dan enkripsi percakapan secara default.



Pendekatan baru untuk merancang antarmuka aplikasi memastikan kompatibilitas dengan versi Android terbaru - tanpa menyulitkan desain. Antarmuka menjadi diperluas, layar kendali panggilan berisi lebih banyak fungsi, dan pengaturan status menjadi lebih mudah.

Seperti disebutkan di atas, aplikasi secara otomatis menghubungkan kembali percakapan telepon ketika koneksi terputus, misalnya, ketika beralih antara kantor Wi-Fi dan jaringan 4G publik. Ini terjadi dengan mulus - Anda tidak akan melihat apa pun atau mendengar jeda singkat.

3CX untuk Android mengintegrasikan terowongan baru yang muncul di server 3CX v16. Ini menyediakan enkripsi lalu lintas suara dari aplikasi ke server. Selama percakapan, kunci kuning di layar menunjukkan bahwa percakapan itu dienkripsi.


Mengatur status Anda saat ini (Tersedia, Hilang, dll.) Sekarang dilakukan dengan satu klik. Anda juga dapat menentukan apakah Anda ingin menerima pemberitahuan PUSH. Misalnya, Anda dapat mengonfigurasi bahwa ketika statusnya Tersedia, panggilan harus diterima hanya di telepon meja, tetapi tidak di aplikasi seluler.

Kami secara singkat mencantumkan peningkatan kecil namun penting lainnya dalam versi ini:

  • Menu obrolan baru - Anda dapat mentransfer obrolan sendiri atau menyembunyikannya dari antarmuka.
  • Pengunduhan percakapan dan riwayat kontak yang lebih cepat.
  • Semua lampiran yang ditransfer disimpan dalam folder "3CXPhone \ 3CX" pada perangkat.
  • Cari kontak dengan nama perusahaan.
  • Panggilan GSM selalu diutamakan daripada panggilan VoIP.
  • Ada panggilan putuskan cepat (Mute) dengan panggilan masuk.

Jika Anda bekerja dengan versi 3CX sebelumnya, disarankan untuk memutakhirkan ke v16 - lebih aman dan memiliki banyak fitur baru. Pembaruan ditawarkan secara gratis jika Anda memiliki langganan pembaruan yang valid atau langganan tahunan . Jika Anda tidak berencana memperbarui 3CX, nonaktifkan aplikasi pembaruan otomatis pada perangkat Anda .

Jika Anda menggunakan versi Android yang lebih lama (sebelum Android 7 Nougat) atau Anda tidak berencana untuk beralih dari 3CX v15.5, gunakan versi aplikasi seluler sebelumnya . Harap dicatat bahwa aplikasi lawas disediakan "apa adanya" dan tidak lagi didukung oleh 3CX.

Instal Pembaruan


Di antarmuka manajemen 3CX, buka bagian "Pembaruan", pilih "v16 Pembaruan 3" dan klik "Unduh Terpilih" atau instal distribusi:

  • Windows: Pembaruan 3CX v16 3 .
  • Linux: 3CX v16 pada Debian 9 ISO - Dari layar Saluran Pembaruan 3CX, pilih Stabil.
  • Aplikasi 3CX Android baru sudah tersedia di Google Play Store .
