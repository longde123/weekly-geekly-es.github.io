DCIM状况：近年来数据中心基础架构管理的变化

长期以来，用于管理数据中心的最流行的工具之一就是Excel电子表格，管理人员在其中保存有关基础结构状态的所有信息并跟踪其任务。 在大多数情况下，这是由于现有DCIM解决方案（用于监视数据中心组件运行的软件产品）的效率低下和笨拙所致。

但是，在过去的几年中，云基础架构的日益普及导致了新一代DCIM解决方案的出现，该解决方案正越来越多地用于商业项目中。 我们正在谈论推动DCIM发展的趋势，以及正在削减的行业专家的预测。


/ Flickr / 罗伯特·斯科布尔 / CC

DCIM的情况


在实现成熟的过程中，任何技术创新都要经历多个阶段，每个阶段的特征都是社区和相关行业专家对技术的兴趣程度不同。 为了评估这些创新的成熟程度，他们使用了Gartner提出的“炒作周期”。

在其帮助下，Gartner分析师追踪了 DCIM解决方案的开发水平-这些工具可评估和控制数据中心的运行，并监控设施中所有IT设备（服务器，存储，路由器）和基础设施组件（配电，空调系统）的能耗。

事实证明，近年来，用于管理数据中心基础结构的解决方案增强了它们在市场中的地位：它们已越来越广泛地用于商业项目中。

在此之前，由于使用困难，效率低和部署速度快，它们没有得到广泛的欢迎。 但是，随着第二代DCIM的到来，情况发生了变化。

现在，DCIM解决方案（如果您遵循Gartner的术语）正在“启蒙运动”上“争先恐后”，也就是说，他们的开发人员正在逐渐克服困难并消除软件和硬件的主要缺点。

为什么会发生


由于多种因素，DCIM解决方案过渡到炒作周期的第四阶段（即“启发的坡度”）成为可能。 以下是其中一些：

  1. 公司正在将越来越多的工作负载迁移到云中。 去年年底，LogicMonitor的研究人员采访了大约300位云中的分析师和顾问以及云服务提供商的代表。 受访者表示，到2020年，83％的工作负载将在云中工作：41％在公共场所，20％在私有云中，22％在混合云中。 同时，在内部部署的应用程序和服务的数量将减少10％，从37％减少到27％。
  2. 虚拟化市场的成本正在增加：根据Research and Markets的预测 ，到2022年，该市场将达到80亿美元的水平，比2017年增加45亿美元。因此，年均增长率将达到16.5％。
  3. 数据中心的基础架构正在变得越来越复杂。 许多公司正在开发边缘生态系统，以在公司数据中心之外尽可能接近其来源来处理数据。 根据Gartner的说法，到2022年，一半的数据将使用“外围计算”（边缘计算）技术进行处理。 如今，在公司数据中心之外，仅处理了10％的生成数据。
  4. 混合基础设施越来越受欢迎。 Gartner预测，到2020年，所有组织中的90％将使用混合解决方案的功能。 根据IDC CloudView对已经使用云技术的公司进行的一项调查 ，有87％的企业正在关注混合基础架构。

这些趋势以及对管理大型IT基础架构的分析工具的需求不断增长，导致了第二代DCIM解决方案的出现。

Sunbird Software公司总裁（开发和实施DCIM解决方案）Herman Chan表示，数据中心所有者已经获得了使用数据中心基础架构管理产品的经验，并且了解如何将其用于特定的数据中心。 基于这种经验，人们可以清楚地了解他们希望从数据中心的软件中获得什么。

因此，DCIM开发人员现在正在解决第一代解决方案的固有弱点，并提供新的，改进的平台来管理IT基础架构。

新一代DCIM解决方案


制造商自己声称，新的数据中心基础架构管理软件已获得许多改进。 例如，与第一个平台相比，减少了部署时间，并且“提升”了扩展能力和与其他产品的兼容性。

新的解决方案还可以通过仪表板和个性化的信息表示来简化团队合作，消除了来自不同制造商的设备不兼容的问题，并且可以在一个软件产品中合并有关管理其基础架构所需的数据中心状态的信息。


/ Flickr / 丹尼斯·范·祖伊莱科姆 / CC

已经实施新的DCIM解决方案的公司为他们提供了许多优势。 例如，Nu Skin（一家通过网络营销销售商品和服务的美国公司） 指出 ，“智能监控”和3D可视化系统使您可以实时跟踪多个数据中心的性能（能耗，湿度），而无论物体之间的距离。

在美国，通信运营商Comcast赞赏DCIM能够在一处提供有关数据中心状态的所有信息。 现在，基础架构各个组成部分上的数据不再像上一代的决策那样分为单独的模块。 制造航空航天和军事工业设备的洛克希德·马丁公司（Lockheed Martin）指出，将新解决方案与其他系统（特别是事件和变更管理工具）集成的便利性。

接下来是什么


由于云的普及，虚拟数据中心市场的增长，新方法和技术的出现，DCIM行业在过去的十年中迈出了一大步 。 与以前的数据中心基础架构管理解决方案相比，新的DCIM更加轻松，快捷地工作。

尽管这些系统还不是很普及，但是Sunbird软件总裁Herman Chan建议，在该技术克服了“教育倾向”并完全消除了过时的功能之后，公司将开始在其数据中心积极实施该系统。 因此，市场上将出现满足现代数据中心需求的价格合理且有效的新解决方案。
