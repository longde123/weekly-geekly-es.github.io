Nvidia Neural Network verwandelt einfache Skizzen in wunderschöne Landschaften


Raucherwasserfall und gesunder Mannwasserfall

Wir alle wissen, wie man eine Eule zeichnet. Zuerst müssen Sie ein Oval zeichnen, dann einen weiteren Kreis, und dann erhalten Sie eine wunderschöne Eule. Natürlich ist dies ein Witz und es ist sehr alt, aber die Nvidia-Ingenieure haben versucht, die Fantasie Wirklichkeit werden zu lassen.

Die neue Entwicklung namens GauGAN schafft wunderschöne Landschaften aus sehr einfachen Skizzen (wirklich einfache - Kreise, Linien und alles). Natürlich basiert diese Entwicklung auf modernen Technologien - nämlich generativen kontradiktorischen neuronalen Netzen.

Mit GauGAN können Sie farbenfrohe virtuelle Welten erstellen - und das nicht nur zur Unterhaltung, sondern auch zur Arbeit. Also, Architekten, Landschaftsarchitekten, Spieleentwickler - alle können etwas Nützliches lernen. Künstliche Intelligenz „versteht“ sofort, was eine Person will, und ergänzt die ursprüngliche Idee mit einer Vielzahl von Details.

„Mit Hilfe von GauGAN wird das Brainstorming in Bezug auf die Designentwicklung erheblich vereinfacht, da ein intelligenter Pinsel die ursprüngliche Skizze durch Hinzufügen hochwertiger Bilder ergänzen kann“, sagte einer der Entwickler von GauGAN.

Benutzer dieses Tools können die ursprüngliche Idee ändern, die Landschaft oder ein anderes Bild ändern, den Himmel, den Sand, das Meer usw. hinzufügen. Alles, was Ihr Herz begehrt, und die Hinzufügung erfolgt in nur wenigen Sekunden.

Das neuronale Netzwerk wurde unter Verwendung einer Datenbank mit Millionen von Bildern trainiert. Dank dessen kann das System verstehen, was eine Person will und wie sie erreichen kann, was sie will. Darüber hinaus vergisst das neuronale Netzwerk nicht die kleinsten Details. Wenn Sie also den Teich und einige Bäume daneben skizzieren, werden nach der Wiederbelebung der Landschaft alle Objekte in der Nähe im Spiegel des Teichwassers reflektiert.

Das System kann festgelegt werden, wie die sichtbare Oberfläche aussehen soll - es kann mit Gras, Schnee, Wasser oder Sand bedeckt sein. All dies kann in einer Sekunde umgewandelt werden, so dass der Schnee zu Sand wird und der Künstler anstelle eines schneebedeckten Ödlands eine Wüstenlandschaft erhält.

„Es ist wie in einem Malbuch, in dem steht, wo man einen Baum platziert, wo die Sonne ist und wo der Himmel. Nach der ersten Aufgabe belebt das neuronale Netzwerk das Bild, fügt die erforderlichen Details und Texturen hinzu und zeichnet Reflexionen. All dies basiert auf realen Bildern “, sagt einer der Entwickler.


Trotz der Tatsache, dass dem System ein „Verständnis“ der realen Welt fehlt, schafft das System beeindruckende Landschaften. Dies liegt daran, dass hier zwei neuronale Netze verwendet werden, ein Generator und ein Diskriminator. Der Generator erstellt ein Bild und zeigt es dem Diskriminator. Basierend auf Millionen von zuvor gesehenen Bildern werden die realistischsten Optionen ausgewählt.

Deshalb „weiß“ der Generator, wo die Reflexionen sein sollen. Es ist erwähnenswert, dass das Tool sehr flexibel und mit einer Vielzahl von Einstellungen ausgestattet ist. Mit seiner Hilfe können Sie also Bilder malen, sich an den Stil eines bestimmten Künstlers anpassen oder sich einfach dem schnellen Hinzufügen von Sonnenaufgang oder Sonnenuntergang hingeben.

Die Entwickler behaupten, dass das System nicht nur Bilder von irgendwoher aufnimmt, sondern sie zusammenfügt und das Ergebnis erhält. Nein, alle empfangenen "Bilder" werden generiert. Das heißt, das neuronale Netzwerk "erschafft" als echter Künstler (oder noch besser).

Bisher ist das Programm nicht gemeinfrei, kann aber bald in der Arbeit getestet werden. Dies kann auf der GPU Technology Conference 2019 geschehen, die derzeit in Kalifornien stattfindet. Die Glücklichen, die die Ausstellung besuchen konnten, können GauGAN bereits testen.

Neuronale Netze haben seit langem gelernt, am kreativen Prozess teilzunehmen. Zum Beispiel konnten einige von ihnen im letzten Jahr 3D-Modelle erstellen . Darüber hinaus haben Entwickler von DeepMind das neuronale Netzwerk geschult, um dreidimensionale Räume und Objekte aus Zeichnungen, Fotografien und Skizzen wiederherzustellen. Um eine einfache Figur wiederherzustellen, reicht ein Bild für das neuronale Netzwerk aus. Um komplexere Objekte zu erstellen, werden fünf Bilder zum „Ziehen“ benötigt.

Was GauGAN betrifft, wird dieses Tool offensichtlich eine kommerzielle Anwendung finden - viele Bereiche in Wirtschaft und Wissenschaft benötigen solche Dienste.
