Pebble Smartwatch Melepaskan Cloud Sync

Berita singkat tapi bagus untuk mereka yang suka jam tangan pintar Pebble!

Seperti yang diingat banyak orang, perusahaan ini baru saja dibeli oleh Fitbit . Sejak itu hanya ada sedikit berita dari mereka. Tetapi sehari sebelum kemarin, perusahaan mengumumkan rencana blognya untuk mendukung dan memajukan nasib perangkatnya. Pembaruan terbaru benar-benar melepaskan Pebble dari layanan cloud sehingga perangkat dapat digunakan bahkan setelah Fitbit menutup server.

Ketika aplikasi diluncurkan pada smartphone, langkah otentikasi akan dilewati, tetapi masih mungkin untuk menginstal firmware dan program melalui itu. Dalam aplikasi, akan dimungkinkan untuk mengkonfigurasi server alternatif untuk mengunduh program. Atau, seperti sebelumnya, program dapat diunduh melalui browser dan diinstal secara offline.

Aplikasi iOS dapat diperbarui di App Store, dan pengguna Android akan dapat memperbarui melalui saluran beta aplikasi. Rilis untuk Android di Google Play akan sedikit lebih lambat.
