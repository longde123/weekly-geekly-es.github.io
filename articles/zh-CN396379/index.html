问第98号Ethan：为什么E = mc2？

爱因斯坦最著名的方程式的计算比预期的还要精美。



从相对论的特殊理论出发，质量和能量是同一事物的不同表现形式-一个普通人不熟悉的概念。
-爱因斯坦


一些科学概念正在改变着世界，影响如此之深，以至于几乎每个人都知道它们，即使他们没有完全理解。为什么不一起努力呢？每周您发送您的问题和建议，而本周我从马克·利尤夫（Mark Liyuv）那里选择了一个问题，他问：

爱因斯坦得出方程E = mc 2。但是在爱因斯坦之前就已经知道了能量，质量，时间，长度的单位。那么结果如何如此美丽？为什么长度或时间没有常数？为什么不是E = amc 2，其中a是一些常数？

如果我们的宇宙还没有像现在这样安排，那么一切都可能会有所不同。让我们看看我的意思。



一方面，我们的物体具有质量：从星系，恒星和行星到最小的分子，原子和基本粒子。尽管它们很小，但我们以物质的名义知道的每个组成部分都具有质量的基本属性，这意味着即使我们排除其运动，即使将其放慢到完全停止的速度，它也将影响所有其他宇宙的物体。



具体地说，无论遥远的物体有多远，它都会对宇宙中的所有其他物体施加引力。他将所有东西都吸引到自己身上，被其他东西吸引，并且还拥有自己的内在能量。

最后一个说法是违反直觉的，因为他们至少在物理学中将能量描述为做某事的可能性-关于工作的能力。如果你坐着不动，该怎么办？

在回答之前，让我们看一下硬币的另一面-无质量的东西。



另一方面，有些东西没有质量-例如，光。这些粒子具有一定的能量，通过观察它们与其他事物的相互作用很容易理解-吸收时，光会将其能量传递给它们。具有足够能量的光可以加热物质，增加动能（和速度），将电子敲成更高的能级，甚至电离，具体取决于能量。



此外，无质量粒子中包含的能量仅取决于其频率和波长，其乘积始终等于粒子的速度：光速。这意味着较长的波具有较低的频率和较低的能量，而较短的波具有较高的频率和较高的能量。可以减慢大质量粒子的速度，尝试从无质量粒子中获取能量只会导致其波的扩展，而不会导致速度的变化。



牢记这一点，让我们考虑质量能如何等效于工作？是的，您可以获取一个物质粒子和一个反物质粒子（电子和正电子），将它们推到一起，得到无质量的粒子（两个光子）。但是，为什么两个光子的能量等于电子和正电子的质量乘以光速的平方？为什么没有其他因素，为什么方程式恰好等于E和mc 2？



根据SRT，有趣的是，方程式必须看起来像E = mc 2，没有任何偏差。让我们谈谈其原因。首先，假设您在空间中有一个盒子。它是静止的，并且在两侧都有镜子，而内部是一个飞向镜子之一的光子。



最初，盒子不移动，但是由于光子具有能量（和动量），因此当光子与盒子一侧的镜子碰撞并反弹时，盒子将开始沿光子最初移动的方向移动。当光子到达另一侧时，它将从另一侧的反射镜上反弹，从而将盒子的动量更改回零。并且它将继续以这种方式反映出来，而盒子将在一个方向上移动一半，而另一半将保持静止。

平均而言，盒子会移动，因此，由于它具有质量，由于光子的能量，它将具有一定的动能。但是记住物体的冲动和动量也很重要。光子动量非常简单地与它们的能量和波长有关：波越短，能量越高，动量就越高。



让我们考虑一下这意味着什么，为此，我们将进行另一个实验。想象一下，只有光子本身最初移动时会发生什么。他将有一定的精力和动力。必须保留这两个属性，因此在初始时刻，光子能量由其波长决定，盒子中只有其余能量-无论它是什么-且光子具有系统的整个动量，盒子的动量为零。



然后，光子与盒子碰撞并被暂时吸收。必须保持动量和能量-这是宇宙守恒的基本定律。如果光子被吸收，则只有一种方法可以保持动量-盒子应该以一定的速度沿与光子移动相同的方向移动。

到目前为止，一切都很好。只有现在，我们才能问自己盒子的能量是多少。事实证明，如果我们去我们平时动能公式，K个ê =½mv 2，我们应该知道箱子的质量和概念的脉冲，其速度的基础上。但是，如果我们将盒子的能量与碰撞前他拥有的光子的能量进行比较，我们会发现盒子的能量不足。

有问题吗不，解决起来很简单。盒子/光子系统的能量等于盒子的剩余质量加上盒子的动能再加上光子的能量。当盒子吸收光子时，它的大部分能量都进入了增加盒子质量的过程。当盒子吸收光子后，它的质量与碰撞前的质量相比发生变化（增加）。

当盒子再次向另一个方向发射光子时，它会收到更大的动量和速度（由相反方向上的光子的负动量补偿），甚至更多的动能（并且光子有能量），但会损失一部分剩余质量。如果您计算所有东西（有三种不同的方法，下面是描述），您会发现，唯一可以节省能量和动量的质量转换是E = mc2。



如果添加任何常数，则方程将不再平衡，并且每次发射或吸收光子时，都会损失或获得能量。在1930年代发现了反物质之后，我们直接看到证实能量可以转化为质量，反之亦然，并且转化的结果与E = mc 2完全吻合，但是通过心理实验，这个公式可以在观察到几十年后得出。只有将光子与等效于m = E / c 2的有效质量对齐，我们才能确保能量和动量的守恒。尽管我们说E = mc 2，但爱因斯坦首先以不同的方式写出了该公式，将等价于能量的质量分配给了无质量的粒子。
