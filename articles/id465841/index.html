Pelacak GPS BOXY

Halo, Habr!

Anda tidak akan mengejutkan siapa pun dengan pelacak GPS - ini adalah fakta. Saya memutuskan untuk menulis ulasan singkat ini ketika saya pertama kali mengambil "sumbu" ini. Nah, dalam urutan: ada tugas - untuk melengkapi hewan peliharaan dengan pelacak. Bukannya dia akan melarikan diri sepanjang waktu, tetapi untuk ketenangan umum, sesuatu harus diperbaiki di kerahnya. "Tech Assignment" tampak seperti ini:

  • pelacak GPS paling ringkas dengan kemampuan untuk melacak posisi online;
  • Baterai harus bertahan selama sehari atau lebih.



Akibatnya, pencarian pelacak kompak menyebabkan penulisan ulasan ini.

Jadi - mainan baru untuk gadgetman - pelacak mikro-GPS BOXY. Ditemukan di Dadget untuk 3490r. Jelas bahwa ada produsen asli, tetapi tidak dapat ditemukan. Pakaian bertemu. Saya menarik perhatian pada gadget ini justru karena ukurannya. Sungguh - ukuran kotak setengah pertandingan! SIM - kartu - nano-sim. Di luar, tidak ada tombol atau indikator. Menurut petunjuk, ada satu tombol di bawah penutup. Ditekan - dengan menekan penutup. Terlampir adalah instruksi sederhana dan tautan ke program pelacakan di Google Play. Saya mengunduhnya sebelum pembelian untuk perbandingan dengan program-program yang saya gunakan. Eh yoyo ... program terjemahan mesin Cina dalam bahasa Rusia. Setengah dari fungsinya tidak bisa dipahami atau perlu dipahami ... Secara umum, semuanya bersama-sama terlihat sangat menarik, jelas bahwa produk tersebut benar-benar baru dan ada sesuatu yang bisa dipecahkan. Secara umum - anugerah untuk gadget nyata. Diputuskan - saya ambil.



Termasuk - hanya tali telepon



Ukuran mata-mata

Karena instruksi sama sekali bukan apa-apa, saya akan berurusan dengan pemahaman tentang bagaimana ini harus bekerja.

Perangkat keras itu sendiri sangat mikroskopis. Tutupnya tidak disegel, itu terkunci. Di tengah bagian dalam adalah tombol yang ditekan ketika penutup ditekan. Antena GPS terlihat - kotak kecil dengan kode QR yang dilampirkan dengan angka (mungkin IMEI). Ada juga LED status: daya, status modul GSM, status modul GPS. Biasanya, ketika GSM dihidupkan, jaringan pertama kali dicari dan kemudian masuk ke mode operasi. Baterai jelas tidak terlihat. Tapi itu pasti kecil. Dan GPS suka memakan baterai, coba lihat. Nah, ada lubang untuk memasang kabel biasa. Berat totalnya sedikit di atas 20 g. Sumbu yang benar-benar mikroskopis, ke mana GSM dan GPS dan baterai didorong. Saya ingin tahu bagaimana cara kerjanya.



Kartu sim - berlaku, termasuk. dan pergi - satu tombol di tengah, ditekan melalui penutup



Ada tiga LED di bawah penutup: status GSM, status GPS dan indikasi daya

Programnya


Seperti yang saya katakan di atas - program pasti membawa senyum. Dirancang secara eksplisit untuk perangkat lain. Bahkan, hanya sebagian fungsi yang berfungsi. Ya, terjemahan ke dalam bahasa Rusia dari beberapa nama adalah cerita rakyat Tiongkok yang tidak dapat diterjemahkan. Agar: saya belajar apa yang berhasil di sini? Registrasi sangat cepat. Menawarkan untuk memindai kode QR atau memasukkan IMEI (dicetak pada label di bawah penutup). Ya, seperti yang lainnya - identifikasi IMEI. Setelah pendaftaran - beberapa nomor pendek diberikan. Ketentuan menyarankan menunjukkan di peta Google atau Cina. Tidak ada pilihan lain.

Hal pertama yang ingin saya periksa adalah alamat server atau data teknis lainnya, yang biasanya dimasukkan saat pendaftaran. Kemudian kegagalan pertama ditunggu. Ini tidak secara eksplisit dinyatakan di mana pun. Program ini tidak memungkinkan Anda untuk melihat informasi layanan, dan karenanya, dan ini penting - Anda tidak dapat menghubungkan pelacak ke layanan pemantauan lainnya dengan cepat. Oke, mari kita lihat apa lagi yang tersedia untuk dipetik.

Di pengaturan ada pengaturan untuk kecepatan refresh. Yah, setidaknya sesuatu ... Pelacak apa pun biasanya bekerja sesuai dengan algoritma berikut: pelacak mengirimkan koordinatnya ke server pada interval tertentu. Semakin sering - semakin akurat posisi akan ditampilkan. Dan biasanya ini dapat diatur satu atau lain cara dalam pengaturan. Dan kemudian ada pengaturan kecepatan refresh peta: seberapa sering program melihat posisi server. Secara umum, kedua pengaturan frekuensi ini ada dalam program, tetapi tampaknya keduanya tidak berfungsi. Seperti yang ditunjukkan oleh tes lapangan, pelacak mengirimkan koordinatnya 1 kali per menit. Ini bisa dilihat jika Anda kemudian mengklik tanda yang sesuai di peta, tetapi tidak menanggapi perubahan interval, maaf.

Data teknis pelacak sangat sederhana


  • Simcard - melamar
  • Standar komunikasi adalah 2G. Perhatian khusus diberikan untuk ini.
  • Kapasitas baterai tidak ditunjukkan, instruksi mengatakan hingga 72 jam
  • Koordinat diperbaiki oleh GPS atau (jika tidak ada data GPS) oleh LBS (oleh menara sel)
  • Tech. karakteristik modul GSM dan GPS tidak ditentukan
  • Sensitivitas - tidak dapat dipahami

Di satu sisi, ini mungkin mengindikasikan sedikit penurunan dalam kecepatan transfer data dari pelacak ke server. Di sisi lain, volume data setiap paket kecil. 2G (atau standar EDGE) - mendukung kartu SIM modern (jika saya tidak salah, kecuali untuk Tele 2). Tidak ada kemampuan kontrol yang sebenarnya disediakan. Operasi LED - tidak dijelaskan dalam manual. Meskipun pengalaman menunjukkan bahwa frekuensi berkedip dari indikator yang sesuai dapat menunjukkan tingkat sinyal, status, dan karakteristik lainnya. Seperti yang sudah ditulis, program konfigurasi tidak ditawarkan. Biasanya, pemrograman perangkat semacam itu dilakukan menggunakan program khusus dan menghubungkan ke PC melalui kabel, tetapi opsi ini tidak ditawarkan di sini. Instruksi menyebutkan kemungkinan pemrograman menggunakan perintah SMS, sementara tidak ada perintah yang diberikan.

Hasil dari bagian teknis: untuk pekerjaan "di luar kotak" Saya mengerti, menyalakannya, bekerja.

Untuk seorang pria gadget sejati:

  • ada pengaturan dasar, hanya Anda yang perlu memahami yang mana;
  • kemungkinan pemrograman belum tersedia atau tersedia terbatas;
  • kemampuan teknis - Anda perlu mengerti.


Tes pertama


Untuk menguji dan memahami mode operasi, saya menguji dengan penutup dilepas, sehingga pengoperasian LED terlihat.

Pelacak Aktif



- Siapkan kartu SIM nano yang kompatibel dengan jaringan 2G dan dengan layanan Internet GPRS terbuka (ini sangat penting karena pelacak GPS hanya bekerja dengan 2G). Jika kartu SIM baru, dibutuhkan 24 jam untuk aktivasi yang benar, menonaktifkan kode PIN kartu SIM, itu sangat penting!
- Masukkan kartu SIM nano ke dalam slot untuk pelacak (ada sudut penyejajaran bulat untuk kartu SIM dan slot)
- Tekan tombol daya selama tiga detik, tunggu LED kuning berkedip perlahan - Unduh aplikasi 365GPS, ID masuk: 15 digit IMEI pada pelacak, kata sandi: 123456
- Melakukan tes pelacak di luar ruangan untuk menerima sinyal satelit dan menentukan keakuratan lokasi

Untuk menghidupkan perangkat, Anda perlu menekan tombol tengah, tahan selama tiga detik, semua dioda menyala. Untuk mematikan - tekan dan hitung hingga 10.

Indikator GSM mulai berkedip cepat - ini adalah pencarian jaringan, dan setelah mendaftar di jaringan, ia mulai berkedip lebih lambat. Jangan lupa bahwa mengirim data ke server adalah melalui Internet seluler. Jika tidak ada Internet seluler, koordinatnya tidak akan ditampilkan. Pada saat yang sama, koordinat harus diakumulasikan dan dikirim setelah kemunculan komunikasi. Cara kerjanya di sini belum jelas.

Algoritma dasar kerja segera menjadi jelas: perangkat ini memiliki accelerometer (sensor gerak). Dengan tidak adanya gerakan - perangkat "tertidur" (semua dioda padam). Layak untuk diambil - segera bangun. Dan ini menjelaskan "umur" panjang dari satu tuduhan.

Pengamatan lebih lanjut menunjukkan: ketika sensor gerak dipicu, BOXY memberikan koordinat ke server 1 kali per menit. Ini adalah interval pembaruan koordinat default. Setiap kali, pelacak membaca koordinat dengan GPS atau menara (jika GPS tidak tersedia) dan mengirimkannya ke server. Jadi, ketika memantau posisi pelacak, kita akan melihat bukan pergerakan yang mulus, tetapi titik dari posisi saat ini. Jika pelacak bergerak, kami akan melihat pembaruan setiap menit. Jika pelacak diam, koordinat juga dikirim, tetapi pada interval yang jauh lebih besar.

Menjadi jelas logika umum perangkat (saya beralasan sebagai produsen):

konsumen utama bukanlah teknolog yang akan repot dengan pengaturan;
tombol dan LED disembunyikan. Untuk menyalakannya, tekan bagian tengah tutup selama 3 detik;
setelah dinyalakan, kami menempatkan BOXY di saku anak, kencangkan di kerah anjing atau letakkan di tas tangan pasangan;
pada aplikasi smartphone, pada kenyataannya, hanya satu fungsi yang diperlukan: melihat posisi pada peta. Jika pelacak bergerak - kita akan melihat posisi di peta Google dengan pembaruan setiap menit, jika pelacak tidak bergerak - kita akan melihat lagi - posisi pelacak saat ini;
nilai-nilai utama terlihat di header: - waktu pembaruan koordinat terakhir dan tingkat baterai pelacak.

Dan ini adalah hal utama - yang diperlukan darinya.

Program untuk bekerja dengan BOXY


Program ini menggunakan 365GPS dari Google Play. Program ini jelas "diasah" untuk bekerja dengan perangkat lain, seperti sebagian besar fungsi tidak berfungsi atau tidak jelas apakah berfungsi. Agar tidak kesal dengan hal ini, kami selalu mengingatkan diri sendiri bahwa tugas utama pelacak adalah melacak posisi BOXY.

Opsi program utama yang tersedia




Foto-foto kawan Tiongkok yang puas menginspirasi kepercayaan diri



Item dengan peta, dalam pengaturan, pilih peta Google atau BaIDU



Pada layar di atas Anda dapat melihat "status", tingkat baterai pelacak, tanggal dan waktu posisi terakhir

Di sudut kanan atas adalah ikon pembaruan. Jika ikon berwarna abu-abu, artinya tidak mungkin mendapatkan data terbaru. Jika ikonnya berwarna, Anda dapat mengkliknya dan mengunduh lebih banyak data yang relevan dari server. Di sudut kiri atas layar ada ikon yang menunjukkan status koordinat saat ini yang diterima dari GPS atau LBS.

Sejauh ini, belum mungkin untuk memahami apakah server memiliki umpan balik dengan pelacak atau pelacak hanya mengirim koordinat, tetapi tidak dapat memperoleh pengaturan. Menu layar ini memiliki item "sensor getaran" (mungkin untuk menonaktifkan operasi oleh sensor getaran) dan "interval pembaruan" (mungkin untuk memperbarui pembacaan data dari server).







Ada pengaturan, tetapi tidak semua berfungsi

Cara kerja pelacak


1. Ada server eksternal. Misalnya di Cina. Pelacak awalnya memasukkan alamat dan nomor port server.
2. Ketika pelacak menentukan posisinya dengan GPS, ia membentuk paket data (koordinat, waktu, nomor) dan mengirimkan paket ini melalui GSM ke server.
3. Kami terhubung ke server melalui aplikasi dan melihat posisi pelacak.

Apa masalahnya? Pelacak mengirim data dalam bentuk semi-terenkripsi. T.O. setelah pembelian - sistem berfungsi seperti yang dijelaskan dalam artikelnya.
Jika Anda ingin melampirkan pelacak ke server yang lebih nyaman, tentu saja Anda dapat melakukan ini dengan menulis alamat server baru menggunakan SMS. TAPI. Ternyata, bagian paket dienkripsi. Dan server pihak ketiga - tidak bisa mengenali koordinat. TETAPI kesedihannya adalah bahwa alamat lama server Cina - tidak lagi berkontribusi! karena awalnya - itu tidak terdaftar di mana saja, dan mencoba mengubah pengaturan, pengaturan lama - tidak lagi kembali - karena Alamat server Cina untuk data tidak diketahui.

Biasanya ini bisa dilihat - jika terhubung ke pelacak, tetapi dalam hal ini pelacak hanya dikendalikan oleh perintah SMS.

Sejauh yang saya mengerti, cukup "me-reboot" pelacak seperti ponsel - untuk mendapatkan kembali pengaturan pabrik - itu tidak akan berfungsi lagi - Anda perlu memasukkan kembali semua pengaturan secara manual, tetapi mereka tidak diketahui secara pasti.

Ini adalah sebuah cerita.

Merekam trek


Dengan memilih tanggal dan interval, Anda dapat melihat bagaimana pelacak bergerak. Setelah memulai, Anda tidak dapat menghentikan atau memundurkan pemutaran. Anda dapat memperkecil secara paralel untuk melihat semua titik yang ditandai dalam garis yang sama. Dengan mengklik suatu titik, Anda dapat melihat waktu memperoleh koordinat. Sebenarnya, itulah cara kami memahami bahwa koordinat dalam mode gerak dikirim ke server setiap menit. Bongkar, simpan lintasan tidak disediakan. Waktu penyimpanan trek di server tidak jelas. Mungkin selamanya?







Jika Anda mengklik pada setiap titik, waktu pendaftaran akan terlihat.

Apa lagi yang bisa berguna dan apa yang bisa Anda mainkan

Paragraf "Peraturan"


Anda dapat mengatur nomor ponsel, dan ketika Anda menekan dan menahan tombol kontrol selama lebih dari 5 detik, pelacak membunyikan nomor ini. Saat Anda mengangkat handset, panggilan diatur ulang. Jenis fungsi SOS. Tidak mungkin untuk memanggil pelacak: selalu sibuk.

Item "Pesan"


Dengan pengaturan yang sesuai, pemberitahuan tentang gerakan sensor gerak akan muncul di item ini.

Gunakan


Itu melakukan fungsinya. Dan ini mungkin merupakan hasil utama dari awal operasi. Sorotnya adalah daya tahan baterai. Anehnya, hari itu tenang. Dia memasukkannya ke sakunya untuk verifikasi dan berjalan sepanjang hari, mencegah pelacak dari "tertidur". Berhasil "hidup untuk melihat malam hari.

Pekerjaan pelacak untuk menentukan koordinat diamati selama perjalanan. Letakkan saja di dashboard mobil dan pergi. Dan pada akhirnya saya menonton trek. Ternyata kartun yang menghibur: Moscow-Voronezh. Dan dalam sehari: Voronezh - Dzhubga. Anda tidak dapat berhenti memainkan trek, tetapi setelah berhenti Anda dapat mengklik pada setiap titik, dan menurut data di header layar, pertimbangkan waktu setiap titik.

Kesimpulan pertama


Pro:

  • ukuran mini di mana mereka memasukkan GSM + GPS + baterai. Sebenarnya - mata-mata. Sangat menarik bahwa kotak tersebut secara terpisah menyatakan bahwa perangkat disertifikasi dan tidak termasuk dalam konsep "barang mata-mata" $;
  • solusi yang sangat indah ketika Anda harus menyelesaikan tugas microtracker untuk tugas yang berbeda: dimasukkan ke dalam saku, kait di ikat pinggang, kerah. Anda dapat menagih sekali setiap 1-2 hari;
  • Adalah nyaman bahwa fungsi-fungsi utama bekerja di luar kotak. Pendaftaran sederhana dalam aplikasi, dan tidak lagi memerlukan pengaturan apa pun. Aplikasi setidaknya melengkung, tetapi posisi pelacak menunjukkan;
  • Interval fiksasi koordinat - 1 menit, kompromi yang cukup masuk akal antara keakuratan menentukan posisi saat ini dan masa pakai baterai, dengan mempertimbangkan ukuran miniatur.

Cons:
Bagi kami, maniak tekno biasa, tentu saja, tidak jelas:

  • Tidak ada aplikasi untuk mengkonfigurasi pelacak;
  • mengoordinasikan waktu pembaruan, sensitivitas sensor gerak. Kawan-kawan Cina mungkin tidak tahu bahwa tanpa program pengaturan, kami tidak ada di mana-mana;
  • tidak ada cara untuk memasukkan alamat server tampilan koordinat lainnya. Karenanya, sejauh ini kami hanya menggunakan aplikasi China.

PS: Setelah kenalan pertama dengan perangkat, beberapa pertanyaan teknis distributor diajukan. Jawaban utamanya adalah "diharapkan untuk menulis program terpisah atau adaptasi" normal "dari yang sudah ada". Menunggu

... Dan untuk mobil?


Pelacak tidak memiliki port eksternal untuk antena jarak jauh. Daya: melalui konektor micro-USB. Pada prinsipnya, Anda dapat bersembunyi di tempat terpencil di suatu tempat di bawah torpedo atau panel belakang. Kami mendapatkan pelacak GPS / GSM tambahan dengan kemampuan untuk bekerja offline selama satu hari ketika baterai dimatikan. Anda hanya perlu memeriksa sensitivitas antena GPS saat dipasang di dalam mesin.

Apa lagi yang bisa Anda gunakan pada prinsipnya: ada fungsi notifikasi ketika sensor gerak dipicu. Anda dapat menyalakannya sehingga menjadi peringatan untuk ponsel cerdas Anda, tetapi jika Anda tidak segera menyadarinya, maka Anda hanya dapat melihatnya dengan melihat layar.

Total - dapat muncul sebagai pelacak GPS tambahan untuk mobil.
