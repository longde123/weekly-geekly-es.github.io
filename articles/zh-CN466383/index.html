Cryptocurrency Ocean：使用CoinMarketCap审查前50个项目

当您在CoinMarketCap上看到无穷无尽的加密货币列表时，想到的第一个问题是：它们有何不同？ 为什么会有那么多？

观众的第一个答案就是宣传和行销！ 但是有意识的人不会屈服于简单答案的魅力。 要回答这个问题，您需要研究每个项目的本质。

阅读2,000多种白皮书对于大学生大军来说是一项令人振奋的任务。 我们中只有两个人和一个朋友在一起，因此我们将自己限制在前50名（+对我们感兴趣的人）的列表中。

分类


作为比较的基础，我们选择了比特币作为所有加密货币的先驱。 他们探索了每种加密货币从技术上给行业带来的好处。

我们提请读者注意以下事实：我们正在分析加密货币及其使用的技术 ，而不是总体上投影在区块链上。

从业务角度来看，我们确定了加密货币演进的四个领域：

  1. 去中心化是加密货币的最重要属性。 如果不是，则不需要其他所有内容。 如果加密货币由信誉良好的中心管理，那么我们会得到一个“中央银行”，其中已经有很多。 同时，可伸缩性问题已得到解决，但是隐私是不可能的。
    • 安全 -权力下放可以通过技术手段（代码）和/或经济动机（社会工程学，博弈论）来提供。 我们认为第二种选择不太安全。
    • 治理机制是用于管理项目开发的内置机制。 或者至少是一种方便的方法，用于收集加密货币用户的意见，这些意见首先涉及在代码中实现哪些功能。

  2. 智能合约 （脚本系统）是区块链上的任何程序，包括区块链本身的内置代码。 由于数字货币需要适当水平的自动化，因此IC为加密货币带来了优势。
  3. 可扩展性 如果一个项目无法在全球范围内扩展，那么成功的机会就更少。 这通常是由于网络可以维持的每秒最大事务数。
  4. 隐私是真正的隐私（不像比特币那样虚构），可以满足黑市的需求。

分权化


该小组包括的项目与比特币的主要区别在于去中心化方面的变化。 在我们看来，我们看到的是，在进步的推销下，倒退并没有取得进展。

大多数项目已转向PoS，这有几个缺点。 例如，在经典的PoS中，所有网络参与者无休止地分叉变得无济于事（没有任何风险攻击）。 一开始很难诚实地将硬币分发给广泛的用户，以避免一只手集中精力。

以太坊提供了一笔保证金，并允许任何人（证明这笔存款的所有者在刺激分叉）注销（向网络发送有效交易）。 但是这里又有一个问题：您可以注销自己。

委托PoS-问题与常规PoS相同：如果大量硬币集中在一只手中，则可以用相同方式控制委托人。 所有权证明提供了更多的动手来积in资金，这可以导致网络集中化，而没有任何发现此问题的迹象。 没有人能证明任何事情。

根据以太坊开发人员的说法，PoS很难实现（“微不足道”）。 仍然没有经过时间检验的实现。

PoS容易受到放血攻击：重播故事可以使您积累硬币（放样），但要付出佣金。 在某个时候，这完全可以替代替代链上的共识。 之后，您可以将其添加到主网络。

PoA是一种更加集中的算法。 我们希望您不必解释原因。
诚然，PoW还存在一些问题：作为ASIC芯片的主要制造商，比特大陆可能会对矿商的交易处理政策产生重大影响。

总的来说，还没有人知道如何替换PoW或实现PoS，同时至少提供了与比特币相同的去中心化水平。

  • Tron-使用PoS，通过投票硬币选择确认节点。 60％的投票币在开发人员手中。
  • EOS-使用PoS，50％的硬币位于10个地址。
  • 卡尔达诺-实际使用PoA：所有交易均由开发人员确认。
  • 破折号-记录是通过PoW完成的，但是所有交易都通过特殊的主节点进行，这些主节点是通过投票选择的。 他们还验证交易。
  • Decred-PoW + PoS混合算法，功能上有链上投票。 事实证明，该方案类似于Dash。
  • NEO-使用PoS，开发者拥有7个节点中的5个。
  • Qtum-纯PoS，ICO后90％的代币仍留给开发商。
  • NEM-PoS +交易历史记录，启动时会分配100％的令牌。
  • Lisk-委派的PoS +基于JavaScript的脚本+Sidehains（例如，以太坊中的Plasma；或者甚至可以是常规数据库）。
  • Waves-PoS + VM。 （ 首先按当前的代码提交数量）
  • Tezos是一种经过正式验证的Michelson编程语言（可以用数学方法证明没有错误；就核反应堆而言）。 也是自我修正，链上治理。 Tezos靠近Waves和EOS。
  • 纳米-没有这样的采矿。 所有硬币都通过创世块分配。 最后，如果您需要从同一链高的多个区块中选择一个区块，那么一切都取决于PoS投票。

纵观顶级加密货币中PoS的趋势，我们可以说“诚实”的去中心化并没有真正打扰市场。 或它的参与者没有预见到缺少它的后果。

智能合约


我们在这里包含了项目，这些项目与比特币之间的主要区别在于，我们考虑了智能合约的扩展系统（在比特币中也存在，但更为原始）。 例如，浪潮并未到来，因为我们认为PoS的实施否定了智能合约的优势。

  • 以太坊-引入了图灵，一种完整的智能合约编程语言。
  • Metaverse ETP-通过附加属性扩展了UTXO模型，并添加了一些内置的智能合约来处理数字资产。 对开发者的开放程度不及以太坊。 他们计划在通过挖矿分配所有硬币之后切换到DPoS。 顺便说一句，这看起来像是解决初始分配问题的好方法。

结垢，rash撞和烧伤


可伸缩性问题的实质是优化三个参数：

  • 网络带宽（TPS），
  • 交易确认延迟
  • 交易成本。

扩展问题不仅可以通过增加区块和提高挖掘速度（例如，每分钟记录1个区块）来解决，还可以通过引入Lightning和SegWit来解决。

一个小题外话。 对等网络（p2p）中的PoW原则之一是在相同高度达成共识。 收到新块后，所有节点都必须接受并切换到该块。 否则会有叉子。 重要的是，网络上块分配的延迟不影响此协议（影响最小）。 因此，平均块出现时间（在比特币中为10分钟）越长，网络在派生出现方面越稳定。

对于朝其他方向发展的区块链（请参阅上文和下文），交易速度问题是次要的，因为解决方法不同。 在PoS中，TPS几乎能够以PoS共识协议本身的最低成本在分布式网络中记录数据。 在PoA中，根本没有共识协议（去中心化）。 网络越集中，工作速度就越快。

除了专注于扩展的项目外，在同一部分中，我们还添加了那些在技术方面没有带来任何收益的项目。

  • 莱特币，比特币现金，比特币SV，比特币钻石-只是以各种原始技术方式增加了每秒最大交易数。 例如，调整块的大小。
  • 以太坊经典-由于智能合约的重新进入而入侵了DAO之后，一些以太坊开发人员不同意为了节省被盗资金而分叉，并继续维护带有漏洞的代码版本。 以太坊经典版开始得到另一个团队的支持，该团队放弃了PoS并取消了定时炸弹。
  • DigiByte-阻止时间15秒。 5种挖掘算法（其想法是改善分散性，由于51％的攻击而导致相反的效果）。
  • Dogecoin-Litecoin的分支，更改了发行顺序和阻止时间（每块1分钟）。
  • 比特币黄金-更改了哈希功能以对抗ASIC芯片。 有趣的是，最近公众的意见已经转向特定芯片很好的事实。

私隐权


这里首先要说的是：默认情况下，比特币不像大多数其他加密货币那样提供隐私交易。 有多种解决隐私问题的机制（例如混合器）。 但是它们是附加组件，而不是项目的功能。 用户需要采取一些措施来保护自己的隐私。

在本节中，我们收集了通过设计提供隐私的加密货币。

  • Monero，ByteCoin-资金转账程序中内置了混合器。
  • Zcoin-使用零知识证明完全断开消费硬币的交易及其代币的产生。 密码学的基础是RSA-2048竞争任务，它已经25年没有破解了。
  • Zcash-使用零知识证明的概念，但使用尚未得到充分验证的密码算法zk-SNARKs。 叉比特币。
  • Zclassic与Zcash相同，但没有得到开发人员的奖励（特定地址的20％）。 叉Zcash。 开发团队停止了支持。
  • Bitcoin Private是Zclassic和Bitcoin的分支。 由前Zclassic团队开发。
  • Verge是狗狗币的一个分支，具有内置的匿名性（使用TOR网络），并同时支持5个哈希函数。 已遭受成功攻击的51％。

Dapps


（稳定币，实用程序令牌）
以下列表中的几乎所有代币都是在以太坊网络上发行的，不能视为独立的加密货币。 我们认为，值得将它们视为实现基于以太坊功能的任何功能的应用程序。

  • Tether是一种稳定的硬币，以美元和欧元部署在比特币内的Omni Layer网络以及以太坊网络上。 所有价值完全取决于发行人帐户上的代币发行量中是否存在法定货币。
  • TrueUSD，USD硬币，Paxos，Gemini Dollar类似于以太坊网络上的Tether代币。 此类标记的完整列表。
  • Maker + Dai是以太坊上的一种稳定币，提供了保证金交易的可能性。
  • Binance Coin-实用程序令牌交换。 主要思想是在交易平台内部的应用。 就像苏联的食品券一样。
  • OmiseGO，0x-去中心化交易所的开源软件的以太坊令牌，用于向开发人员收取使用软件的费用。
  • 基本注意令牌-分散广告平台成员的实用程序令牌。
  • Aeternity是一个将现实世界的数据传输到区块链的Oracle网络。
  • Chainlink是最好的预言家。
  • Pundi X是基于区块链的销售点系统。 在NEM和以太坊网络中使用令牌来为其网络上的操作收取费用。
  • 人口众多是全球保理市场的象征。
  • Augur是预测市场的代币。
  • Golem-全球计算市场，以太坊上的代币。
  • 状态-在6个项目（信使，硬件钱包，开发框架）中使用，以普及以太坊平台。 当然建立在以太坊上。
  • Petro是由石油和委内瑞拉政府担保的委内瑞拉俄罗斯加密货币。

不是加密货币


该项目使用区块链技术和其自己的硬币这一事实根本并不意味着它是一种加密货币。 正如在您的Steam帐户中存钱并不意味着Valve是银行或支付系统。 一些项目声明了其目的，与金融工具无关。

  • XRP，Stellar-用于在区块链上记录交易记录并进行付款的技术平台，由一个排放中心管理。
  • IOTA是一个物联网数据收集系统。
  • VeChain是一个区块链，用于存储满足物流需求的物联网标识符。
  • Steem是具有集成货币化的内容发布平台。
  • 本体-中国区块链凌乱软件套件。 它仅与其他区块链（NEO，以太坊）（包括其自己的（本体分类帐）和其他）相比，与Metaverse ETP（区块链即服务）非常相似。
  • ICON是一个团结其他区块链网络的网络。 对于每个第三方区块链，都使用了ICON区块链的门户。 在ICON区块链网络内部，使用了基于BFT（完全集中化）的共识协议。 其中有一个ICX令牌（ICON Exchange令牌）以及某种智能合约。
  • BitShares是常规的DEX +许多营销功能。
  • Siacoin是带支付的基于区块链的分散式数据存储系统。
  • Holo-没有共识协议！ 只需将事务写入分布式表（DHT）。 分发不可靠的信息与货币的概念矛盾：-D
  • Git-使用带有PoA的区块链的源代码控制技术:-)

结论


  • 加密货币不是炒作。 在厚厚的不道德行销手段下，贪婪倍增了这种炒作。
  • 以太坊几乎是唯一为业界带来新事物的项目。
  • 并不是所有写在区块链上的加密货币！
