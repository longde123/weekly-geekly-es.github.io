Microsoft a appelé les prix internationaux de Windows 10

La ressource Neowin rapporte que des utilisateurs d'Amérique, du Royaume-Uni, d'Espagne et d'autres pays de l'UE ont remarqué une modification du programme Get Windows 10, qui a récemment été affiché dans la barre d'état système pour les utilisateurs de Windows 7 et Windows 8.1. La fenêtre du programme a commencé à afficher les prix du nouveau Windows 10 Home, dont la sortie est attendue le 29 juillet de cette année.

Par exemple, pour les utilisateurs américains, le coût du système d'exploitation sera de 119 $. Les prix sont barrés pour souligner les futures mises à jour gratuites. Certes, Microsoft précise toujours que le coût est approximatif et peut changer.
