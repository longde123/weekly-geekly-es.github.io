Sistem Kolaborasi Dokumen untuk Zimbra Open-Source Edition

Pentingnya mengedit dokumen bersama dalam bisnis modern sulit ditaksir terlalu tinggi. Kemampuan untuk menyusun kontrak dan perjanjian dengan partisipasi karyawan dari departemen hukum, menulis penawaran komersial di bawah pengawasan atasan online dan sebagainya, memungkinkan perusahaan untuk menghemat ribuan jam kerja yang sebelumnya dihabiskan untuk berbagai persetujuan. Itulah sebabnya salah satu inovasi utama dalam Zextras Suite 3.0 adalah kemunculan Zextras Docs - solusi yang memungkinkan Anda untuk mengatur kolaborasi penuh dengan dokumen secara langsung di klien web Open-Source Edition Zimbra Collaboration Suite.

Saat ini, Zextras Docs mendukung kolaborasi dengan dokumen teks, tabel dan presentasi, dan juga tahu bagaimana memproses sejumlah besar format file. Antarmuka solusi tidak jauh berbeda dari antarmuka editor teks mana pun, sehingga karyawan perusahaan tidak perlu menghabiskan banyak waktu pelatihan untuk beralih menggunakan Zextras Docs. Tetapi yang paling menarik, seperti biasa, adalah "di bawah tenda." Mari kita simak bersama bagaimana Zextras Docs bekerja dan apa keuntungan yang bisa ditawarkan solusi kolaborasi dokumen ini.



Yang terpenting, Zextras Docs akan menarik bagi mereka yang sudah menggunakan Zimbra OSE dan Zextras Suite di perusahaan mereka. Dengan solusi ini, dimungkinkan untuk memperkenalkan layanan baru dalam produksi tanpa meningkatkan jumlah sistem informasi dan, sebagai hasilnya, dapat dilakukan tanpa meningkatkan biaya kepemilikan infrastruktur TI. Perhatikan bahwa Zextras Docs hanya kompatibel dengan Zimbra OSE versi 8.8.12 dan yang lebih baru. Itu sebabnya jika Anda masih menggunakan Zimbra versi lama, kami sangat menyarankan untuk meningkatkan ke versi 8.8.15 LTS. Karena dukungan jangka panjang, versi ini akan tetap terbaru dan aman untuk beberapa tahun lagi, dan juga akan kompatibel dengan semua penambahan saat ini.

Keuntungan dari Zextras Docs juga dapat mencakup kemungkinan penyebaran penuhnya pada infrastruktur perusahaan. Ini menghindari transfer informasi ke pihak ketiga, seperti yang sering terjadi ketika menggunakan layanan hybrid atau cloud. Itulah sebabnya Zextras Docs ideal untuk perusahaan dengan kebijakan keamanan informasi yang ketat dan bagi administrator sistem yang lebih suka untuk sepenuhnya mengontrol aliran data dan proses yang terjadi di perusahaan. Selain itu, layanan kolaborasi dokumen yang dikerahkan secara lokal memungkinkan Anda untuk menghindari risiko yang terkait dengan tidak dapat diaksesnya layanan cloud ketika masalah timbul dengan penyedia layanan, atau ketika masalah muncul dengan akses Internet berkecepatan tinggi.

Zextras Docs terdiri dari tiga bagian: server yang berdiri sendiri, ekstensi, dan musim dingin. Masing-masing dari ketiga bagian ini melakukan bagiannya:

  • Server Zextras Docs adalah mesin LibreOffice, dipertajam untuk kolaborasi dan integrasi dengan Zimbra OSE. Di server Zextras Documents semua dokumen yang diakses oleh pengguna dibuka, diproses, dan disimpan. Itu harus diinstal pada node komputasi khusus yang menjalankan Ubuntu 16.04, Ubuntu 18.04, atau CentOS 7. Jika beban pada layanan Zextras Docs cukup besar, Anda dapat memilih beberapa server sekaligus.
  • Ekstensi Zextras Documents tidak perlu diinstal, karena ekstensi ini dibangun ke dalam Zextras Suite. Berkat ekstensi ini, pengguna terhubung ke server Zextras Docs, serta load balancing saat menggunakan beberapa server. Selain itu, melalui ekstensi Zextras Docs, beberapa pengguna secara bersamaan terhubung ke satu dokumen, serta file diunduh dari penyimpanan lokal ke server.
  • Zextras Docs Zimlet, pada gilirannya, diperlukan untuk mengintegrasikan layanan ke klien web. Berkatnya, klien web Zimbra memiliki kesempatan untuk membuat, serta fungsi pratinjau dokumen Zextras Docs.



Untuk menggunakan Zextras Docs di perusahaan, Anda harus memilih satu atau lebih server fisik atau virtual untuknya. Setelah itu, Anda perlu mengunduh distribusi aplikasi server untuk Ubuntu 16.04 , Ubuntu 18.04 atau CentOS 7 dari situs web Zextras, kemudian unzip dan instal. Pada tahap akhir instalasi, server akan meminta Anda untuk menentukan alamat IP server LDAP, serta pasangan nama pengguna / kata sandi yang akan digunakan untuk memasukkan data tentang server baru ke dalam LDAP. Perhatikan bahwa setelah instalasi selesai, setiap server Zextras Documents akan terlihat oleh semua node infrastruktur lainnya.

Karena ekstensi Zextras Docs sudah terkandung dalam Zextras Suite, itu akan cukup untuk memungkinkan penggunaannya bagi para pengguna dan grup yang harus memiliki akses ke alat kolaborasi dokumen. Zextras Docs Zymelet dapat diaktifkan dari konsol admin Zimbra. Perhatikan bahwa setelah menambahkan server Zextras Docs ke infrastruktur Zimbra OSE, Anda perlu memperbarui konfigurasi server proxy Zimbra. Untuk melakukan ini, jalankan file / opt / zimbra / libexec / zmproxyconfgen sebagai pengguna Zimbra, dan kemudian jalankan perintah restart zmproxyctl untuk memulai kembali layanan proxy.

Untuk semua pertanyaan yang terkait dengan Zextras Suite, Anda dapat menghubungi perwakilan Zextras Katerina Triandafilidi melalui email katerina@zextras.com
