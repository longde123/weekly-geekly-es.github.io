Connectez l'adaptateur WiFi WN727N à Ubuntu / Mint

J'ai un problème pour connecter l'adaptateur wifi wn727n à ubuntu / mint. J'ai longtemps cherché sur Google, mais je n'ai pas trouvé de solution. Ayant résolu le problème, j'ai décidé de l'écrire moi-même. Tout ce qui est écrit ci-dessous est conçu pour les débutants.

ATTENTION! L'AUTEUR DE L'ARTICLE N'EST PAS RESPONSABLE DES DOMMAGES!

Mais, si vous faites tout correctement, il n'y aura aucune conséquence. Même si quelque chose tourne mal, rien de mauvais ne se produira. Commençons.

Tout d'abord, ouvrez le terminal à l'aide des touches Ctrl + Alt + T et entrez la commande suivante: 

lsusb



Nous voyons notre adaptateur Ralink RT7601 (en surbrillance). Vous pouvez avoir un adaptateur Ralink RT5370. Les pilotes sur différents adaptateurs sont installés différemment. Je décrirai comment procéder pour deux cas.

Manuel pour Ralink RT5370


Suivez le lien et sélectionnez RT8070 / RT3070 / RT3370 / RT3572 / RT5370 / RT5372 / RT5572 USB USB. Téléchargez l'archive avec le pilote.

Ouvrez le dossier où le pilote a été enregistré et décompressez l'archive bz2. Pour ce faire, faites un clic droit sur le fichier et cliquez sur «Extraire ici».

Après cela, l'archive tar apparaîtra. Déballez-le à nouveau. Faites un clic droit sur le fichier et cliquez sur «Extraire ici».

Ensuite, changez le nom du dossier en quelque chose de plus court, car nous devons encore écrire son chemin vers la console. Par exemple, je l'ai appelé Driver.

Nous allons dans le dossier décompressé et ouvrons le fichier /os/linux/config.mk dans un éditeur de texte

Recherchez les lignes suivantes et remplacez la lettre n par y:

# Support Wpa_Supplicant
HAS_WPA_SUPPLICANT = y

# Support natif WpaSupplicant pour Network Maganger
HAS_NATIVE_WPA_SUPPLICANT_SUPPORT = y

Après cela, enregistrez le fichier. Ouvrez le terminal et accédez au dossier décompressé. Attention! J'ai un nom d'utilisateur sergey. Vous entrez votre nom d'utilisateur! À l'avenir, changez sergey en votre nom d'utilisateur. 

 cd /home/sergey//driver/

Ensuite, exécutez les commandes: 

 sudo make sudo make install sudo modprobe rt5370sta

Et c'est tout! Oh miracle! Le WIFI fonctionne, profitez de votre santé.

Manuel pour Ralink RT7601


Pour obtenir cet adaptateur (Ralink RT7601), vous devez avoir une version de noyau 3.19 ou supérieure. si nécessaire - mettez à jour le noyau (si vous ne savez pas comment - Google vous aidera).

Ensuite, suivez le lien et téléchargez le pilote:



Ensuite, nous déplaçons l'archive téléchargée dans le dossier d'accueil et la décompressons (clic droit, «extraire ici»). Renommez simplement le dossier mt7601-master résultant en mt7601.

Après cela, entrez la commande: 

 cd mt7601/src

Maintenant, nous sommes dans le bon répertoire. Vous pouvez créer le pilote, exécutez la commande: 

 sudo make

Le système vous demandera un mot de passe - entrez (le mot de passe n'est pas affiché).

Ensuite, entrez les commandes: 

 sudo mkdir -p /etc/Wireless/RT2870STA/ cp RT2870STA.dat /etc/Wireless/RT2870STA/

Et la dernière commande qui permettra à notre adaptateur: 

 insmod os/linux/mt7601Usta.ko

Tout !!! Ubuntu voit maintenant le wifi.

Mais ce n’est pas tout! Maintenant, après chaque redémarrage, vous devez entrer la dernière commande, sinon le système ne verra pas l'adaptateur (spécifiquement pour Ralink RT7601). Mais il y a une issue! Vous pouvez créer un script et l'ajouter au démarrage. Ci-dessous est écrit comment le faire.

Tout d'abord, nous devons nous assurer que le système ne demande pas de mot de passe lors de l'utilisation de sudo. Pour ce faire, entrez la commande: 

 sudo gedit /etc/sudoers

Cette fenêtre s'ouvrira:



Nous recherchons une ligne:

% sudo ALL = (ALL: ALL) ALL

Et nous le changeons en:

% sudo ALL = (ALL: ALL) NOPASSWD: ALL

Enregistrez les modifications - cliquez sur "Enregistrer".

Entrez la commande: 

 sudo cp -R mt7601 /etc/Wireless/RT2870STA/

Après cela, entrez la commande: 

 sudo gedit /etc/Wireless/RT2870STA/autowifi.sh

Un éditeur de texte vide s'ouvre. Nous y écrivons ou copions:

#! / bin / bash
insmod /etc/Wireless/RT2870STA/mt7601/src/os/linux/mt7601Usta.ko

Cliquez sur "Enregistrer" et fermez.

Nous entrons les commandes: 

 cd /etc/Wireless/RT2870STA/ sudo chmod +x autowifi.sh

Ensuite, allez dans le menu Dash et recherchez le programme comme dans l'image ci-dessous:



Nous l'ouvrons. Cliquez sur "Ajouter".



Une fenêtre s'ouvrira. En face du champ «Nom» nous écrivons:

autowifi

En face du champ «Team» nous écrivons:

sudo sh /etc/Wireless/RT2870STA/autowifi.sh

Cliquez sur le bouton "Ajouter" et fermez le programme. Redémarrez. Après le redémarrage, tout fonctionne. Vous pouvez maintenant sélectionner un réseau dans le bac.



Ceci termine les «petites» instructions pour l'adaptateur Ralink RT7601.
