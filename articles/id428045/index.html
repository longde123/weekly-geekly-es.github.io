PHP internal


  • [RFC] Preloading - Konsep dari Dmitry Stogov dibingkai dalam RFC yang lengkap. Diusulkan untuk memungkinkan mengunggah file sebelum memproses permintaan. Semua kelas dan fungsi yang dimuat dengan cara ini akan tersedia secara permanen dalam konteks permintaan berikutnya tanpa perlu dimasukkan. Selain keuntungan produktivitas, peluang seperti itu membuka jalan bagi pengoptimalan lebih lanjut.
  • [RFC] Tingkatkan openssl_random_pseudo_bytes () - Fungsi openssl_random_pseudo_bytes() dapat dengan cara yang tidak terlihat menghasilkan urutan acak yang tidak aman. Dalam hal ini diusulkan untuk melemparkan pengecualian.

  • Async php

    • concurrent-php / psr-http - Klien HTTP asinkron yang memenuhi standar PSR. Dikirim oleh @dmitrybalabka
      Teks tersembunyi
      Sebagai bagian dari proyek concurrent-php / ext-async untuk mengimplementasikan dukungan untuk operasi IO asinkron dalam PHP, klien HTTP yang memenuhi standar PSR diimplementasikan. Ini menunjukkan kemampuan untuk mengintegrasikan dukungan concurrent-php / ext-async ke pustaka PHP yang ada sambil mempertahankan kompatibilitas. Penting untuk dicatat bahwa ketika mengimplementasikan coroutine menggunakan generator, fungsi / metode asinkron mulai mengembalikan objek Generator, yang mencegah antarmuka menjadi kompatibel ke belakang.
    • amphp / parallel 1.0.0 - Komponen dari keluarga Amp untuk eksekusi paralel proses PHP dengan dukungan untuk sinkronisasi, memori bersama dan fitur lainnya.
    • clue / reactphp-buzz 2.5 - Mudah digunakan klien HTTP asinkron. Pos baru dalam dukungan .
    • M6Web / Tornado - Mesin asinkron lain untuk PHP dengan penerapan Event Loop dan dengan adaptor untuk ReactPHP dan Amp .
