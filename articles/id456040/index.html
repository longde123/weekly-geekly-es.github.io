Tangkap aku jika kau bisa. Kelahiran Raja

Tangkap aku jika kau bisa. Jadi mereka saling berkata. Direktur menangkap wakil mereka, karyawan biasa itu, satu sama lain, tetapi tidak ada yang bisa menangkap siapa pun. Ya, mereka tidak mencoba. Saya akan segera menjelaskan asal nama panggilan saya sehingga Anda tidak berpikir bahwa saya memiliki megalomania. Putri kecil saya suka memainkan permainan seperti itu - dia bangun di ambang pintu, menutupnya dengan tangannya, dan tidak membiarkannya lewat, meminta kata sandi. Saya berpura-pura tidak tahu kata sandi, dan dia berkata: kata sandi - raja duduk di pot. Jadi, pertimbangkan bahwa saya adalah Raja di atas kendi, dengan ironi diri yang normal, memahami kekurangan saya dan keunggulan Anda di atas saya.

Baiklah, ayo pergi. Saya akan secara singkat memberi tahu Anda tentang diri saya - ini akan membuatnya lebih mudah untuk memahami alat yang saya gunakan dalam bisnis dan kesimpulan atas dasar yang saya bangun hanya skema seperti itu.

Kebetulan saya sangat awal menjadi direktur sebuah perusahaan besar. Lebih tepatnya, itu adalah peternakan unggas. Saya berumur 25 tahun saat itu. Dan sebelum itu, saya memimpin agen pemasaran selama tiga tahun.
