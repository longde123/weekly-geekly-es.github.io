什么是Redmi注意事项


像往常一样，小米催生了如此众多的设备，恶魔会摔断他的腿。 在市场上，您可以找到Redmi 5 Note，Redmi 5 Note Pro，Redmi 5 Plus。


让我们尝试找出：


  • Redmi Note 5 Pro目前仅在印度发布。 它具有一个20兆像素的前置模块，该手机没有全局固件，并且不支持LTE B7网络，这对我们的市场至关重要。
  • Redmi Note 5配备了配对的垂直相机（例如iPhone X），可以使用全局固件或中文。 支持我们的LTE频率。
  • 在某些地区，Redmi Note 5实际上是在销售带一个主相机和Snapdragon 625处理器的Redmi 5 Plus。

如果希望进一步深入研究该主题，我找到了一个说明 ，该说明指示特定的模型索引 。 我对3/32 GB基本版本中的Redmi Note 5进行了评估，并预装了全局固件。

设计与人机工程学


我不会争辩，小米以前从未否认复制苹果产品的设计。 在小米A1的外观上可以很容易地在整个MIUI外壳中看到这一点，小米A1已经与较小的iPhone 7 Plus非常相似。


在Redmi Note 5的情况下，与iPhone X的外观相似几乎是完全荒谬的。 从苹果的小工具上舔过的垂直相机模块没有任何实际好处，并且看起来像是在嘲笑。




如果小米智能手机的早期拥有者被喻为愿意以负担得起的价格选择平衡设备的实际人，那么新“笔记本电脑”的拥有者通常会露出讽刺的笑容。


是的，我知道类似的设计失真了，其中包括华硕（Zenfone 5），华为（P20系列）和另外十二个不太知名的中国品牌。 如此漫不经心的复制会损害用户体验，对此我感到非常难过。


如果您关闭无聊的侦听模式，则设备的外观将非常有趣且新鲜。 与以前围绕显示屏的Redmi Note框架相比，该框架减少了，并且新的宽高比为18：9，因此可以包装6英寸的屏幕而不是5.5英寸的屏幕。


外壳由阳极氧化铝制成，控制装置以经典方式布置。




平板手机的上面板和下面板类似于金属，但实际上它是一种遮盖良好的塑料。 该解决方案很简单，但是可以应付其任务，并且外观不会受到特别破坏。




指纹扫描仪位于设备背面。 当您经常从口袋里拿出智能手机时，这非常方便，但是当您需要解锁躺在桌上的设备时，这是非常不愉快的。 扫描仪可以瞬间工作，没有什么可抱怨的。




对于6英寸设备，小米Redmi Note 5的平衡性很好。 它很容易握在手中，您可以截获它而不必冒险掉落它。 整个身体上控制元件的位置是熟悉的。 所有按钮都在右侧。 SIM卡和存储卡的插槽在左侧。 最后一个是混合的，也就是说，您必须在第二张SIM卡和microSD卡之间进行选择。


该设备摆脱了显示屏下方的物理按钮-另一个显着改进。 但是Redmi Note 5并没有摆脱的是旧的microUSB端口和3.5毫米耳机插孔。 那个……该死的好！ 毕竟，每个家庭都可以找到充电线，而有线耳机的音质尚未被取消。 那些想要在没有电线的情况下获得高质量声音的人将不得不打扰。 要启用对apt-X技术的支持，您需要安装几个拐杖 。




小米用户熟悉的红外端口尚未消失，这意味着您仍然可以忘记用于控制家用电器的遥控器。


智能手机的底部是唯一的多媒体扬声器。 我的声音质量是平均水平，但声音足够大，不会错过繁忙街道上的电话。

铁和软件


智能手机的铁含量是一个很好的平均水平。 高通八核Qualcomm Snapdragon 636与Adreno 509视频加速器搭配使用时，最高频率可达到1.8 GHz，新的串联可在日常任务中发挥出色的性能，但在重装玩具（例如WoT，Dead Trigger 2，PUBG）中，最大图形设置下的帧频不适合舒适的游戏。 我们必须将纹理的质量降低到中等。

结论
