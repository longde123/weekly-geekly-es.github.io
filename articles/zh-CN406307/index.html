Galaxy Note 8：什么是新旗舰三星

事实证明，Galaxy Note 7的故事令人非常难过。 该设备看起来很有希望，受到评论家的选择，买家以创纪录的速度购买了设备……但随后它开始爆炸。 事实证明，电池中的元素太紧-导致过热和自燃。 该智能手机被禁止在飞机上运输，被认为具有生命危险，所有参与者都必须删除。 三星损失了170亿美元。

为了不重蹈覆辙，并让仍受到粉丝欢迎的Note系列重新命名，他们特别努力开发了新的Note 8。 而在8月23日，这款智能手机终于发布了-具有有趣的三星首个功能，对Android 8.0的支持以及令人印象深刻的功能。 包裹正在看着它来了。


主要特点


该设备最引人注目的功能是背面的双摄像头。 这是三星首次露面。 两个相机模块-分辨率均为12兆像素，并且具有光学变焦。 一个是广角镜头（f / 1.7光圈），并具有双像素自动对焦系统。 另一个是远摄镜头（f / 2.4光圈），具有双倍光学变焦。 这样的设置使您可以比iPhone 7更好地拍摄照片，并拍摄背景模糊的壮观肖像。


新旗舰配备了Bixby的智能个人助理。 他比Siri或Alexa弱得多，此外，他只能以英语或韩语来感知命令。 为了避免用户抱怨， 三星在Note 8中谨慎地安装了传统的Android助手Google助手 。

像Galaxy S8一样，智能手机的显示屏是弯曲且无边框的 。 他走到左右两端，看起来比黑色塑料骰子好，并为小按钮和小部件打开了空间。 屏幕很长，长宽比为18.5：9，这在顶部和底部留出了额外的空间。 手机可以使用指纹传感器（在背面，靠近相机）和视网膜扫描仪识别所有者。

Note系列与传统三星智能手机的区别之一是手写笔 （在公司中被称为“电子笔”）。 笔存放在设备外壳内，如有必要可将其取下。 手写笔可帮助您绘制，制作草图或自己的gif以便发送给对话者。 三星还认为，这种笔非常适合快速手工书写笔记，为文档创建签名以及处理屏幕上的小元素。 在新的智能手机中，触控笔变得略微更小，并且精度有所提高：现在，它可以识别4,096种不同的按压力，这应该会影响画质和易画性。


而且-我不敢相信现在作为一项功能值得一提-Galaxy Note8带有一个用于耳机的迷你插孔。 与一周前发布的iPhone 7或相同的Essential Essential炒作智能手机不同 。 由于设备的大小并不弱，因此3.5毫米插孔可安静地安装在USB Type-C旁边，用于充电和将数据传输到PC。

Note 8令人印象深刻的规格使三星可以将其智能手机与某些计算机进行比较，甚至声称新旗舰产品将更加强大。 但是想要购买新设备的用户还有另一个问题：Galaxy Note 8与六个月前发布的Galaxy S8有何根本不同？ 即使设备属于不同的生产线，其特征和设计也非常相似。 智能手机具有相同的Snapdragon 835处理器（俄罗斯的Exynos 8895），圆形的屏幕，并且前面板上没有按钮。 一个没有经验的人不太可能区分两个设备。

但是与Galaxy S8仍然存在差异。 新的智能手机具有更多的RAM，它具有双摄像头（非常令人印象深刻）和手写笔。 而且，该设备已经适用于Android 8.0（它将在深秋的某个地方收到此操作系统，并且在Note 8中发布时将有Android 7.1.1）。 在屏幕底部的一行上，有三个传统的界面按钮，现在已经出现了一个几乎不明显的点。 您可以按它-并删除这些相同的按钮，以增加显示屏上的可用区域。


智能手机清楚地演示了多任务处理：其中的两个应用程序可以同时使用（每个应用程序将占据屏幕的一半）。 例如，观看视频剪辑并用手指戳游戏，阅读书籍并在日记中写下重要位置，在导航仪中观看，并在汽车中滚动浏览音乐。

三星Galaxy Note 8将于2017年9月14日开始提供黑色，黄色和蓝色，但其预订已在美国三星现场和亚马逊开放 。 如果您决定在该公司在美国的官方网站上订购， 三星将为您提供Gear 360相机（原价：230美元）或Galaxy Foundation套件（190美元） ，其中包括128 GB的存储卡和免费的快速无线充电。 美国的Note 8售价为929美元。 在俄罗斯，官方价格为69,990卢布（1,192美元）。 购买包裹后，您可以节省约14,000卢布。

专家怎么说



美国观察家已经收到了他们手中的第一个Note 8，我们已经收集了一些意见。 因此， CNET记者说，智能手机符合所有标准。 它很大，有一个明亮的屏幕，两个带人像模式的相机。 它具有市场上最好的处理器，无线充电和防水功能。 同时，Note 8的电池要比2016年的电池小-也许是为了避免重复出现致命的“爆炸性”错误。 结果，该设备无法维持我们想要的时间。

另外，Note 8看起来并没有迈出很大的一步：与Galaxy Note 7和Galaxy S8相比，它有太多的优势。 受信任的评论也提到了同样的问题：除了S笔以外，至少有明显的变化，尽管价格已经大幅上涨。 许多人指责指纹扫描仪的位置：如果您的手掌很小，则可能无法够到它。


唯一真正重要的新功能是双后置摄像头。 有了它，您可以拍摄很多有趣的照片，创造出美丽的自然效果。 这两款相机都具有光学稳定性，并且该系统比iPhone 7更好（后者仅在广角相机上起作用）。 即使在弱光条件下进行变焦，也可以获得不错的镜头。

对于美国市场，新的Galaxy Note8基于今年春天推出的功能最强大的Qualcomm Snapdragon 835处理器和Adreno 540图形芯片，根据AnTuTu测试，其得分为18万点。 该模型将在俄罗斯和欧洲销售，在Exynos 8895处理器和Mali G-71芯片上运行。

总的来说，Galaxy Note8获得了该功能集，该功能集使该系列的粉丝可以忘记出现问题的Note7。 智能手机的显示屏变得更大，设计更现代，出现了第二个摄像头并且手写笔变得更加准确。 到目前为止，唯一可能不吸引Note 8的是它的自治性和成本。 但是，如果您在美国购买 ，则成本会明显降低，并且内置的处理器功能更强大。


价钱


三星已经在俄罗斯开放了对预购Note 8的访问权限，因此我们知道该设备的价格：69,990卢布（按当前汇率计算为1,192美元）。 这使其成为市场上最昂贵的Android智能手机。 但是， 像往常一样 ，有机会节省开支：在美国，该设备更便宜。 三星提供的价格为930美元。 您可以以相同的价格在Amazon上订购。 节省-260美元，即14000卢布，考虑到运费。 除了购买外，您还可以免费获得360°相机或128 GB microSD卡+无线充电设备。

顺便说一下，其他流行的智能手机在穿越俄罗斯边境时也有同样的价格上涨趋势。 俄罗斯的Galaxy S8 Plus售价为59,990卢布（1020美元），在美国， 亚马逊为575美元 ，便宜26,000卢布。 好了，iPhone 7 Parcel通常能够以711美元的价格从柏林交付，这也低于俄罗斯的价格。
