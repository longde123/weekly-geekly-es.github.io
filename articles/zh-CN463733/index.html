Mesh VS WiFi：无线选择什么？



当我仍住在公寓楼时，我面临着远离路由器的房间中低速的问题。 毕竟，许多人在走廊上有一个路由器，供应商将光纤或UTP拉到了那里，并在那里安装了标准设备。 当所有者将路由器更改为自己的路由器时，这也很好，而提供商提供的典型设备通常是最廉价或最简单的型号。 您不应该期望它们具有高性能-可以，可以。 但是我安装了具有千兆位端口的路由器，并带有一个支持2.4 GHz和5 GHz运行的无线电模块。 公寓内部，尤其是远处房间的Internet连接速度完全令人沮丧。 这部分归因于嘈杂的2.4 GHz频段，部分归因于穿过钢筋混凝土结构时信号的消光和多次反射。 然后，我决定使用其他设备扩展网络。 问题出现了：Wi-Fi网络还是Mesh系统？ 我决定进行整理，进行测试并分享我的经验。 不客气

Wi-Fi和网状理论


对于通过Wi-Fi连接到网络并观看YouTube视频的普通用户而言，使用哪种系统没有什么区别。 但是从组织常规Wi-Fi覆盖的角度来看，这些系统在根本上是不同的，并且各有利弊。 让我们从Wi-Fi系统开始。

Wi-Fi系统




这是一个可以独立工作的普通路由器网络。 在这样的系统中，分配了一个主路由器，其余的成为从属。 同时，路由器之间的过渡对于客户端仍然是不可见的，从路由器本身的角度来看，客户端将从一个单元混合到另一个单元。 可以将这种系统与蜂窝通信进行比较，因为形成了带有路由器转换器的单个局域网。 该系统的优势显而易见：网络可以逐步扩展，并根据需要添加新设备。 而且，购买支持该技术的廉价路由器就足够了。 这里有一个负号，但有一个重要的负号：必须将以太网电缆和电源连接到每个路由器。 也就是说，如果您已经进行了维修并且没有铺设UTP电缆，那么您将不得不在可能的情况下沿着底板延伸，或者考虑使用其他系统。

网格系统




这是一个专用设备网络，它也形成了多个设备的网络，从而形成了连续的Wi-Fi信号覆盖范围。 这些点通常是双频的，因此您可以在2.4 GHz网络和5 GHz网络中工作。 最大的优点是，无需连接电缆即可连接每个新设备-它们通过单独的发送器进行通信，创建自己的网络，并通过该设备传输数据。 随后，此数据被传输到常规Wi-Fi适配器，并到达用户手中。 优点是显而易见的：您不需要任何其他电线-只需将新点的适配器插入插座，将其连接到主路由器即可使用。 但是也有缺点。 例如价格。 主路由器的成本比传统路由器的成本高出几倍，另外一个适配器的成本也很高。 但是，无需重新进行修理，拔出电缆并考虑电线。

让我们继续练习




我已经从钢筋混凝土公寓搬到了自己的房子，并且面临无线网络速度下降的问题。 如果早先的空气噪声受到相邻的Wi-Fi路由器的影响很大（毕竟，每个人都在努力最大程度地松开电源以“淹没”邻居并提高其速度），现在距离和重叠部分已经开始产生影响。 我没有使用45平方米的公寓，而是搬到了200平方米的两层楼房屋中。 您可以谈论很多关于房子的生活，甚至说邻居的Wi-Fi点仅偶尔出现在智能手机的菜单中，并且没有检测到更多的无线网络这一事实已经说了很多。 尽我所能，我试图将路由器放置在房屋的地理中心，并以2.4 GHz的频率在任何地方提供通信，但是在该区域中，覆盖范围已经很差。 但是，当您从离路由器最远的房间的便携式计算机上的家庭服务器看电影时，有时会褪色。 事实证明，5 GHz网络不稳定，有几面墙壁，天花板，笔记本电脑更喜欢使用2.4 GHz网络，该网络具有更高的稳定性和更低的数据传输速度。 “我们需要更高的速度！”-正如杰里米·克拉克森（Jeremy Clarkson）喜欢说的。 我一直在寻找一种扩展和加速无线通信的方法。 我决定直接比较两个系统：Keenetic的Wi-Fi系统和Zyxel的Mesh系统。



从Keenetic方面，参加了Keenetic Giga和Keenetic Viva router刨机。 其中一个是网络的组织者，第二个是从站。 两台路由器均具有千兆以太网和双频无线电模块。 此外，它们具有USB端口和非常广泛的固件设置。 测试时，安装了最新的可用固件，领先者是Keenetic Giga。 它们通过千兆位有线以太网电缆互连。



在Zyxel一侧，将有一个由Multy X和Multi mini组成的Mesh系统。 最早的站点Multy X已连接到Internet，而“最年轻的” Multi mini安装在房屋的最远角落。 主导点连接到网络，附加功能是无线和有线分布网络。 也就是说，附加的连接点还可以用作不具有Wi-Fi模块但具有以太网端口的设备的无线适配器。

功能性




制造商经常在新闻稿中声明其设备的无线网络覆盖范围异常广泛。 但是它可以在没有墙壁，反射面和无线电干扰的开放区域中工作。 实际上，许多人面临着公寓速度降低和数据包丢失的问题，在智能手机上可以看到一个半到两个打的无线网络。 因此包括在内，使用不太嘈杂的5 GHz频段更为有效。

为简单起见，我将主机称为Wi-Fi和Mesh系统路由器。 每个路由器都可以只是一个无线设备。 但是我不知道路由器可以提供多少设备以及以什么速度提供对网络的访问。 关于第一个问题，情况如下。 支持的设备数量取决于Wi-Fi模块。 对于Zyxel Multy X和Multy mini，每个频带（2.4 + 5 GHz）将是64 + 64个设备，也就是说，如果有两个点，则可以连接128个2.4 GHz的设备和128个5 GHz的设备。
创建网状网络变得尽可能简单和直接：对于工作来说，拥有一部智能手机并在那里安装Zyxel Multi应用程序就足够了。 而且，无论您拥有iO还是Android设备都没有关系。 按照安装向导的提示，将创建一个网络并连接所有后续设备。 出乎意料的是，对于最初的网络创建，您必须启用地理位置并具有Internet连接。 因此，您至少必须从智能手机访问网络。

您可以从计算机和智能手机配置路由器。 如果在本地网络上可以轻松地通过基于Web的界面实现此功能，那么智能手机将有一个特殊的应用程序，该应用程序将来将可以使用附加功能，例如洪流下载或通过USB访问连接的驱动器上的文件。 Keenetic具有出色的功能-KeenDNS，如果IP地址为灰色，则可以从外部网络连接到已发布服务的Web服务。 也就是说，您可以连接到NAT的路由器接口，也可以连接到DVR或Web服务器的NAT接口。 但是由于该材料仍然是关于网络的，因此应注意，Wi-Fi网络的组织也非常简单：主导路由器成为主设备，而从适配器的模式在其他路由器上被激活。 同时，从属路由器可以创建VLAN，可以在单个地址空间中工作，可以以10％的增量在其上设置每个无线适配器的电源。 因此，网络可以扩展很多倍。 但是，有一件事：要组织Wi-Fi网络，必须使用以太网连接所有路由器。

测试方法


由于客户端的无线网络没有区别，并且从网络的技术组织的角度来看，它们根本不同，因此针对用户选择了一种技术。 Zyxel Multy X + Multiy mini和Keenetic Giga + Keenetic Viva设备分别进行了测试。 为了避免提供者的影响，在本地网络中，在主机前面增加了一个服务器。 并在用户设备上组织了一个客户端。 结果，拓扑如下：服务器主导的路由器-接入点-客户端。

所有测试均使用Iperf实用程序执行，该实用程序可模拟连续数据传输。 每次都针对1、10和100个线程进行测试，这使您可以评估不同负载下无线网络的性能。 它既模拟了单线程数据传输（例如在Youtube上观看视频），又模拟了多线程（例如洪流摇摆）。 通过2.4 GHz和5 GHz网络连接时，分别进行测试。

另外，由于Zyxel Multy和Zyxel mini设备可以充当适配器，因此它们通过以太网接口以1000 Mbps的速度连接到用户计算机，并且还进行了三项速度测试。 在类似的测试中，Keenetic Vivo路由器作为Wi-Fi适配器参加，通过跳线连接到笔记本电脑。

点之间的距离约为10米，钢筋混凝土地板和两堵墙。 从笔记本电脑到最终访问点的距离为1米。

图片显示，所有设备在2.4 GHz频率下的效率均低于5 GHz频率。 尽管存在这样的事实，但相邻干扰网络周围都没有噪声，因为如果在2.4 GHz频率中存在噪声，结果将变得更加糟糕。 但是，可以清楚地看到，在5 GHz频率下的数据传输速率几乎是2.4 GHz时的两倍。 另外，值得注意的是，同时加载的流的数量也有一些影响，即，随着流的数量的增加，虽然差异并不那么显着，但是数据信道的使用更加密集。

当Keenetic路由器充当中继器时，可以很清楚地看到传输速度被一分为二，因此，如果您想高速传输大量信息，而不仅仅是扩大Wi-Fi网络的覆盖范围，则应考虑这一点。

在上一个测试中，Zyxel Multy X和Zyxel Multy mini充当了用于连接远程设备的适配器（基本Zyxel Multy X与接收设备之间的连接是无线的），展示了Multy X的优势，尤其是在多线程数据传输中。 它影响了Zyxel Multy X中的大量天线：9个天线，而Zyxel Multy mini中的6个天线。

结论


因此，很明显，即使需要以2.4 GHz的频率卸载空气，当您需要足够快地传输大量信息时，切换到5 GHz也很有意义。 同时，即使在2.4 GHz的频率下，也可以使用路由器作为中继器以FullHD质量观看电影。 但是具有正常比特率的4K电影已经开始变得不流畅，因此路由器和播放设备应该能够以5 GHz的频率工作。 在这种情况下，如果一组两个Zyxel Multy X或Zyxel Multi X + Multy mini用作无线适配器，则可以实现最高速度。

现在关于价格。 一对经过测试的Keenetic Giga + Keenetic Viva路由器的价格为14,800卢布。 一套Zyxel Multy X + Multy mini-21900卢布。



Zyxel的网格系统可以以非常不错的速度提供广泛的覆盖范围，而无需铺设额外的电线。 当已经完成维修但没有放置其他双绞线电缆时，尤其如此。 此外，通过智能手机上的应用程序，这种网络的组织尽可能简单。 应该补充的是，Mesh网络可以包含6个设备，并且具有星形拓扑和树形拓扑。 也就是说，最终设备可能与连接到Internet的启动路由器相距很远。



同时，基于Keenetic路由器的Wi-Fi系统功能更强大，并且提供了更便宜的网络。 但这需要电缆连接。 路由器之间的距离可以达到100米，并且由于通过千兆有线连接进行传输，速度根本不会降低。 此外，这样的网络中可以有6个以上的设备，并且移动时漫游Wi-Fi设备将是无缝的。

因此，每个人都可以自己决定选择什么：功能和是否需要铺设网络线路，或者以几笔大钱来扩展无线网络的简便性。
