Notepad unter Windows erhielt das erste bedeutende Update seit zwei Jahrzehnten

Der Grund, warum Notepad (auch bekannt als Notepad) das Schicksal eines hässlichen Entlein erlitten hat, war für niemanden lange von Interesse. Bis das Unerwartete passierte ...

Es ist kaum zu glauben, aber Notepad - eine der bekanntesten, ältesten und gleichzeitig einfachsten Microsoft-Anwendungen - wartet auf ein sehr umfangreiches Update . Es ist wichtig anzumerken, dass dieses Programm (aus dem Basisliefersatz der meisten Windows-Versionen) fast ein Vierteljahrhundert lang praktisch ohne Änderungen von einer Version des Betriebssystems zur nächsten wanderte.

Es ist schwer zu sagen, was bei Microsoft passiert ist, ob die Wölfe im persönlichen Zoo eines der Top-Manager gestorben sind oder jemand versehentlich auf dem falschen Fuß aufgestanden ist ... Aber nach der kürzlichen plötzlichen Einführung der Unterstützung für Unix-Line-Feeds hat das Unternehmen offenbar beschlossen, hier nicht aufzuhören ...



Der geänderte Editor wird im Hauptupdate von Redstone 5 für Windows 10 angezeigt, kann jedoch bereits im Build unter der Nummer 17713 getestet werden.

Liste der Änderungen:


  • Suchen Sie den Text in einem Kreis. Wenn nun beim Suchen oder Ersetzen von Wörtern das Ende des Dokuments erreicht ist, wird der Vorgang ab dem Anfang des Textes wiederholt. Setzen Sie dazu das Flag im Popup-Menü. In früheren Versionen war es erforderlich, die Richtung der Suche anzugeben und anzugeben, wo sie begonnen hat.
  • Das vom Cursor hervorgehobene Wort wird automatisch in das Suchfeld eingefügt
  • Die Pfeile auf der Tastatur heben die Textauswahl korrekt auf und bewegen erst dann den Cursor
  • Mit der Tastenkombination Strg + Rücktaste können Sie das gesamte vorherige Wort löschen.
  • Die lang erwartete Skalierung des Textes. Sie können den Text auf standardmäßige Weise vergrößern oder verkleinern: indem Sie die Strg-Taste auf der Tastatur lange drücken und dann mit dem Mausrad oder den Tasten „+“ und „-“ scrollen. Diese Funktion wird seit langem in den meisten Texteditoren und Browsern verwendet, im Editor jedoch erst 2018.




  • Die Statusleiste ist jetzt standardmäßig aktiviert. Falls gewünscht, kann es im Menü "Ansicht" ausgeblendet werden. Benutzer können die Koordinaten von Zeichen sehen, auch wenn der Zeilenumbruch aktiviert ist.
  • Erhöhte Produktivität. Große Dokumente werden viel schneller geöffnet
  • Eingebaute externe Suchmaschine. In Notepad können Sie in der Bing-Suchmaschine nach der Bedeutung eines Wortes suchen. Wenn Sie die Funktion " Sets" aktiviert und Microsoft Edge als Standard-Webbrowser festgelegt haben, wird im Editor eine neue Registerkarte geöffnet, die die Bing-Suchergebnisseite enthält. Andernfalls wird die Bing-Suchergebnisseite standardmäßig in Ihrem Browser geöffnet.



Warum hat Microsoft Notepad so viele Jahre lang sehr nützliche Funktionen vorenthalten, und jetzt haben Sie plötzlich damit begonnen, es zu aktualisieren. Was denken Sie?
