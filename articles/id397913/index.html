Microsoft mematenkan cara untuk melacak pengguna menggunakan AI dan mendaftarkan semua tindakan di Windows


"Perantara" adalah komponen Windows 10 baru dari Microsoft. Mungkin akan terlihat seperti itu.

Sistem operasi terbaru Microsoft, Windows 10, menempati pangsa pasar OS yang semakin meningkat. Ini sebagian disebabkan oleh fakta bahwa Windows 10 telah menerima persetujuan dari banyak pengguna. Dan sebagian - fakta bahwa Microsoft berusaha untuk secara aktif mempromosikan produknya. Selama setahun terakhir, metode mempromosikan Windows 10 oleh penciptanya telah berkali-kali menjadi bahan diskusi di media.

Nuansa lain adalah transfer data pengguna Windows 10 dalam jumlah besar ke server perusahaan. Telemetri dalam "sepuluh besar" mengkhawatirkan sejumlah besar pengguna yang mencoba mencari tahu mengapa Microsoft memiliki begitu banyak data. Korporasi, sebagai tanggapan, menyatakan bahwa informasi yang diterima adalah anonim dan hanya berfungsi untuk meningkatkan kualitas produk perangkat lunaknya. Tetapi pengguna tidak terlalu menyukainya. Sekarang perusahaan Redmond menghadapi risiko menyebabkan gelombang ketidakpuasan baru. Faktanya adalah bahwa Microsoft telah mengajukan aplikasi paten untuk cara melacak pengguna menggunakan komponen OS baru.

Paten disebut " Query Formulation Via Task Continuum ." Ini menggambarkan metode yang menyederhanakan pertukaran data antara aplikasi individu secara real time. Menurut para pengembang teknologi ini, itu akan meningkatkan efisiensi kerja pengguna dengan aplikasi dan seluruh sistem operasi.

Microsoft percaya bahwa model saat ini operasi perangkat lunak, ketika masing-masing aplikasi dibiarkan ke perangkatnya sendiri dan sedikit berinteraksi dengan lingkungannya, tidak terlalu efektif. "Karena fakta bahwa masing-masing program secara praktis tidak terhubung satu sama lain ugom, sistem komputasi tidak bahwa dibutuhkan pengguna untuk titik tertentu dalam waktu", - dinyatakan dalam aplikasi paten.



Sebagai contoh, sebuah perusahaan mengutip kasus ketika pengguna membuat catatan tentang OneNote tentang topik menari. Pada saat yang sama, pengguna ini mencari di Internet Explorer untuk informasi tentang gaya dansa di Web. Kedua aplikasi ini biasanya tidak terhubung satu sama lain dan tidak bertukar informasi, tetapi perusahaan mengusulkan untuk memperbaikinya.

Pengembang perusahaan mengusulkan untuk meningkatkan interaksi aplikasi individual menggunakan "komponen perantara" atau "perantara" (Mediator). Mereka berencana menjadikan komponen ini program terpisah yang akan memantau tindakan pengguna, mentransfer data yang dikumpulkan ke aplikasi dan perangkat lain dari orang yang sama. "Komponen" tidak hanya akan memonitor teks yang dimasukkan oleh pengguna, tetapi juga tindakan lainnya. Misalnya, mencari lagu di Web atau bekerja dengan gambar yang disimpan di hard drive Anda. Selain itu, "komponen perantara" akan mengumpulkan data dari aplikasi seperti Word, Skype, Edge.

Microsoft menyarankan menjadikan "Perantara" sebagai modul opsional yang dapat dipasang di lingkungan Window 10 atau diintegrasikan ke dalam struktur sistem operasi.

Fakta bahwa Windows 10 mengumpulkan data terperinci tentang penggunanya telah lama dikenal. Pada tahun 2014, Microsoft mengeluarkan pernyataan privasi baru yang menyatakan bahwa perusahaan dapat mengumpulkan informasi tentang program PC dan laptop bekas dengan Windows 10 diinstal, serta perangkat dan jaringan tempat mereka bekerja. Data tersebut dapat digabungkan dengan ID pengguna. Informasi juga dikumpulkan tentang preferensi orang tertentu, lokasi, minatnya.



Paling aktif mengumpulkan asisten suara Cortana. Daftar server yang datanya dikirim sangat luas. Namun, jika ada kebutuhan seperti itu, maka Anda dapat menonaktifkan telemetri di Windows 10.
