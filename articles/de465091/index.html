Verlust personenbezogener Daten mutmaßlicher Mitarbeiter der Russischen Eisenbahnen

Unbekannt, öffentlich zugängliche personenbezogene Daten von 703.000 Personen, vermutlich Mitarbeiter der Russischen Eisenbahnen.



Auf der Website (ich gebe keinen speziellen Link zur Website an) befindet sich die Aufschrift: " Vielen Dank an Russian Railways für die bereitgestellten Informationen durch sorgfältigen Umgang mit den personenbezogenen Daten ihrer Mitarbeiter. "


Unter den frei verfügbaren Daten:


  • Vollständiger Name
  • Geburtsdatum
  • die Adresse
  • SNILS Nummer
  • Position
  • Fotografie
  • Telefon


Es ist nicht bekannt, wo das Leck aufgetreten ist, aber es gibt Spekulationen, dass es sich um eine Sicherheitsdienstdatenbank handelt. Nach dem Format der Fotos zu urteilen, sind dies Bilder auf dem Pass.


Nach meinen Prognosen wird diese „Datenbank der Russischen Eisenbahnen“ innerhalb von ein bis zwei Tagen in Form einer Excel-Datei und / oder im Kronos-Format öffentlich zugänglich sein.


Das heißt, Selbst wenn Roskomnadzor den Zugriff auf eine Website schließt, die Seiten mit persönlichen Daten enthält, hat dies keinerlei Auswirkungen auf die Verfügbarkeit von durchgesickerten Daten.


Nachrichten über Informationslecks und Insider finden Sie immer auf meinem Telegrammkanal „ Informationslecks “.
