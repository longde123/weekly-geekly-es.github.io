DIYorDIE聚会7月1日：选择发言人



大家好！ 7月1日，我们邀请所有DIY-kov和他们的粉丝参加我们在Mail.Ru集团办公室举行的大型DIYorDIE会议！ 这次不仅会有有趣的报道，而且还会有您的炫酷作品和利益相关者的展览。

谁会在7月1日的会议上谈论他们的手工艺品？ 选择你！ 我们建议对最有趣的演示文稿进行投票，并邀请演讲者在DIYorDIE上发言。

因此，已经向我们发送了哪些主题：

1. Nadezhdin Kirill， “ Arduino上的MIDI鼓套件”。 从想法和前提到实施-尤其是关于压电传感器，多路复用器，用于从鼓中拾取信号的粉扑海绵，MIDI消息，用于处理来自传感器的“振动数据”的算法，峰值识别等。

2. Lilekov Yuri， “ Megadron：在7 km的高度上有2.5米的泡沫 。 ” 关于使用翼展为2.5米的“飞翼”外形从头开始建造无人机的经验，以及其功能和应用前景。

3. 普罗宁·弗拉基米尔（Pronin Vladimir）， “一辆15美元的经济型鸡蛋自制船” 。 设备的历史，设计工具，实际使用情况，使用设备的项目示例。

4. Kashkanov Artem， “ O Brave Relay World” 。 关于创建“ Because-I-can_”中继项目的前提，历史和方面，例如中继计算机，其祖先RCM-1和自动灌溉控制器。

5. Autushenko Igor， “ Arduino的陷阱” 。 在设备和现有库中规避它们的缺点和方法，以及开发正确库的方法。

6. Chumakova Anastasia， “如何选择DIY项目” 。 如何选择一个自己动手做的项目，如何弄清楚案例，与听众建立沟通，以及下一步该怎么做？ 这种活动提供的机会以及如何实施。

7. Atroshkin Kirill， “我们组装了Raspberry Pi的热像仪，诺基亚的屏幕和Seek热像仪 。 ” 一般而言，有关热成像仪的一些知识，将SPI显示器连接到Raspberry并通过剪贴板输出图片，使用Python和OpenCV快速（突然）处理和显示图像，与相机交换数据的非官方协议以及一些未记录的功能，证明了原型。

8. Matveev Ruslan， “使用Arduino模拟LCD显示屏”完成了标准BC Pajero Pinin / IO。

9. Minko Vitaliy， “一种用于平板电脑的微型键盘，专为用一根手指使用而设计 。 ” 概念，组件选择，PCB设计，机壳3D模型的创建，编写固件等。

10. Larchenko Ilya， “移动机器人的开发”。 关于已经实现的内容，即：以3个自由度从摄像机传输立体声图像； 使用Neato激光雷达和ROS导航套件的Indore导航，基于8个声纳的防撞系统，基本运动和遥测功能。

11. Ivanov Mikhail， “延时摄影者的运动控制 。 ” 演示了从奇怪的原始工艺到有吸引力的设备的开发过程。 该报告的技术细节不会太过饱和，但将涉及arduino，3D打印机，焊接，机壳，在中国的订单以及来自圣彼得堡的车工。

12. Basov Alexander， “家庭仓库存储系统”。 关于物品存储的问题，仓库存储系统的服务器部分，移动应用程序，条形码扫描仪从USB转换为RS232以及系统的适用性。

13. Ovchinnikov Kirill， “智能视频保姆” 。 如何在RPi上制作智能视频保姆。 具有培训和美丽！

14. Oleg， “智能家居，z-wave设备的自动化和对HomeKit的支持” 。

15. Timofeev Grigory， “在家中制造刀具的主要方面 。 ”

16. Andrey Molchanov， “智能闹钟” 。 包括Arduino +蓝牙，2个LED灯泡，驱动器破损，笔记本电脑的散热器。 报告3.3级与5级，PWM，LED的原理，与蓝牙的配合以及膝上移动应用程序的协调性。

17. Andrei Molchanov， “植物生命支持系统”。 如何为Arduino + ESP8266的植物和垃圾场的水族馆装配生命支持系统：如何制作光学传感器，关于DHT11 \ DHT22的不可靠性，关于Arduino和ESP8266的绑定，关于珀尔帖元件，关于磁滞和温度控制，关于管理大电流的信息。

18. Andrey Molchanov， “电容表” 。 由Arduino + FET晶体管组装而成。 这个故事只涉及电路，蚀刻板，电池理论，FET晶体管和导线截面的重要性。

19. Andrey Molchanov， “空气质量传感器（WIP）” 。 如何通过Arduino + DHT22 + MQ135 +蓝牙制作空气质量传感器。 关于电路和蚀刻板的一些知识，关于电池的理论的一些知识，关于FET晶体管和导线横截面的重要性的知识。

20. Andrey Molchanov， “ Rapiro机器人” 。 关于伺服器，关于管理大电流，关于3D打印零件的可靠性。

21. Zotov Igor， “为这个家伙喝鸡尾酒！” 。 简要介绍主要内容：如何制作调酒机器人。
