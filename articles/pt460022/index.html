  • PHP 7.4.0 alpha 3 - A última versão alpha do planejado . O congelamento de recursos deve ser 22 de julho, para que a lista atual de alterações possa ser complementada apenas por aquelas para as quais a votação tenha começado até 8 de julho. Total agora na votação:
    Deprecações para o PHP 7.4
    • Escape da DOP "?" marcador de posição do parâmetro
    • Preterir a sintaxe de chaves para acessar elementos de matriz e deslocamentos de cadeia de caracteres
    • Adicione str_starts_with (), str_ends_with () e funções relacionadas .

    Também algumas alterações novas sem RFCs separados:
    • proc_open aceitará argumentos na forma de uma matriz, eles não precisam ser escapados e o processo será iniciado diretamente sem o shell intermediário: 
     proc_open(['php', '-r', 'echo "Hello World\n";'], $descriptors, $pipes);

    • No PHP 7.4, um aviso será lançado ao tentar usar tipos escalares como matrizes incorretamente. Anteriormente, essas chamadas simplesmente retornavam null : 
     $null = null; var_dump($null[0]); // NULL $int = 42; var_dump($int['foo']); // NULL var_dump(true[0] === false[0]); // true

As ferramentas


  • humbug / php-scoper - A ferramenta permite adicionar um prefixo a todos os espaços para nome na pasta especificada em uma etapa. Pode ser útil quando você precisar usar várias versões de um pacote, um exemplo de uso para plugins do WordPress .
  • CaptainHookPhp / captainhook - gerenciador de ganchos do Git para projetos PHP.
  • sebastianfeldmann / phpbu - Uma ferramenta para criar cópias de backup de arquivos e bancos de dados. Capaz de criptografar backups e enviar para armazenamento.
  • localheinz / http-method - Um conjunto de constantes com os nomes dos métodos HTTP.
  • spiral / roadrunner - Servidor de aplicativos para PHP em Golang. Adicionado suporte às configurações FastCGI, CORS prontas para uso, métricas integradas no Prometheus. Uma revisão detalhada no relatório do autor .
  • leocarmo / circuit-breaker-php - Implementação do padrão de disjuntor para PHP. Alternativa: ackintosh / ganesha .
