Kursus Final Spesialisasi Python dari Mail.ru Group



Python adalah bahasa yang sederhana dan fleksibel yang digunakan di banyak bidang perkembangan modern. Ini menciptakan aplikasi web, menulis game, menganalisis data, dan melakukan banyak tugas lainnya. Grup Mail.ru meluncurkan spesialisasi Pemrograman Python pada platform Coursera, yang terdiri dari empat program - mulai dari perendaman dalam bahasa hingga Ilmu Data. Kami ingin pemula, siswa, dan pengembang berpengalaman untuk mempelajari dan memperdalam pengetahuan mereka tentang bahasa.

Yang pertama dari empat bagian, " Immersion in Python ", dimulai kembali pada tahun 2017. Pada bulan Agustus, bagian kedua dan ketiga menjadi tersedia untuk pendengar: " OOP dan pola desain dengan Python " dan " Membuat layanan Web dengan Python ". Dan pada bulan Desember, bagian keempat dan terakhir dari Python untuk spesialisasi Analisis Data dimulai.

Semua program disiapkan dengan dukungan Institut Fisika dan Teknologi Moskow, penulisnya adalah karyawan Institut Fisika dan Teknologi Moskow dan Grup Mail.ru.

Kami sudah membicarakan bagian pertama , kedua , dan ketiga dari program ini. Artikel ini adalah tentang bagian keempat, di mana siswa belajar bagaimana menerapkan keterampilan pemrograman mereka untuk membangun model prediksi, memvisualisasikan data, dan bekerja dengan jaringan saraf.

Untuk siapa


Kursus ini dirancang untuk siswa tingkat menengah: cocok untuk siswa sekolah menengah, siswa, pengembang, dan semua orang yang ingin mendapatkan keterampilan dalam analisis data dan membangun model dengan Python.

Berhasil menyelesaikan kursus ini membutuhkan keterampilan pemrograman Python yang solid. Nilai plus akan menjadi pengetahuan dasar analisis matematika, aljabar linier, dan analisis statistik. Mereka yang belum menyelesaikan tiga bagian pertama dari spesialisasi Pemrograman Python juga dapat mengambil kursus, tetapi pengetahuan bahasa diperlukan.

Durasi dan format


Durasi kursus - 6 minggu. Anda harus menghabiskan sekitar 7-9 jam seminggu untuk melihat kuliah dan mengerjakan pekerjaan rumah, tergantung pada level awalnya.

Kursus ini dikembangkan bersama dengan spesialis dari Institut Fisika dan Teknologi Moskow dan Dana Pengembangan Pendidikan Online .

Penulis


Andrey Shestakov, programmer riset, Mail.ru Group.

Saya bekerja di grup analitik prediktif dari departemen teknologi periklanan Grup Mail.ru, jika lebih mudah - MyTarget. Secara total, saya telah menganalisis data sejak 2010. Saya juga mengajar di proyek-proyek pendidikan Grup Mail.ru di Sekolah Tinggi Ekonomi.

Saya diundang untuk menggantikan guru lain. Tampak bagi saya bahwa berpartisipasi dalam penciptaan kursus online terbuka besar akan menjadi pengalaman yang menarik dan bermanfaat. Dan ternyata. Menyimpan informasi dalam bentuk yang ringkas, memantau pidato Anda, menjelaskan materi kepada khalayak luas - semua ini membantu mengembangkan keterampilan mengajar saya. Saya berharap para siswa akan menyukainya dan mereka akan belajar sesuatu yang bermanfaat dari kelas.

Elena Shirokova, Ilmuwan Data, MegaFon.

Saya adalah bagian dari tim data besar megafon, sebelum itu saya terlibat dalam analisis data kota dalam startup Habidatum. Dia adalah salah satu guru dari kursus analisis data Urban Design Lanjutan, bersama dengan Sekolah Tinggi Ekonomi dan Strelka.

Saya ingin mencoba format pengajaran baru, berbeda dari kuliah dan seminar, dan mencoba memberikan basis yang dapat berguna bahkan setelah menyelesaikan kursus.

Zebrak Alexander, CTO Insilico Medicine.

Dia bekerja selama lima tahun di Mail.ru Group di departemen pengembangan internal dan otomatisasi. Dua tahun lalu, saya bergabung dengan tim Insiliko, di mana kami terlibat dalam penggunaan pembelajaran mesin untuk mencari obat baru dan penelitian tentang penuaan. Dia juga mengajar kursus Python di Universitas Teknik Negeri Moskow. Bauman di Technopark. Salah satu penulis kursus spesialisasi pertama adalah "Immersion in Python".

Saya memutuskan untuk mengambil bagian dalam pembuatan kursus baru, karena saya menyukai hasil kerja pada kursus pertama dan umpan balik dari siswa. Seperti banyak orang, saya pikir setiap programmer harus mengetahui metode analisis data dan membangun model pembelajaran mesin pada tingkat tertentu, sehingga kursus baru ini sangat cocok dengan spesialisasi kami. Merekam kursus online adalah tugas yang sulit, jadi pengalaman dengan kursus pertama tentu saja membantu.

Pestrov Nikita, Pimpinan Ilmu Data, Habidatum.

Pimpinan Sains Data di Habidatum, sebuah startup tentang big data dan urbanisme. Membuat layanan data dan mengubah data menjadi jawaban atas pertanyaan dari perencana kota. Pendiri CocoaHeads Moscow - komunitas pengembang iOS pertama di Rusia. Co-pendiri aplikasi mobile pendidikan terlaris untuk belajar bahasa asing mudah sepuluh di Rusia. Jalan menuju Python dimulai di magistrasi Skoltech, menciptakan sistem untuk menganalisis suasana hati pengguna Twitter.
