PyTest Neo

Suatu malam, merenungkan tes tak berujung dan melihat titik berlari, sebuah pemikiran menyelinap - bagaimana itu tampak seperti Matrix! Maka lahirlah ide, untuk membuat visualisasi tes dalam bentuk Matriks.



Di bawah animasi kucing dan beberapa detail.



Jika seseorang khawatir tentang keterbacaan, maka hanya bilah kemajuan yang dianimasikan, dan jika ada kesalahan, laporan standar ditampilkan.

Untuk menginstal plugin pytest-neo , cukup lakukan: 

pip install pytest-neo

Proyek ini masih dalam tahap awal pengembangan. Linux dan mungkin MacOS didukung. Kemungkinan bug. Sejauh ini, mode verbose dibiarkan apa adanya. Rencana di dalamnya untuk mengimplementasikan animasi yang andal dari Matrix.

Silakan tulis saran dan saran. Ini akan sangat membantu pengembangan proyek.

PS: dalam tes aiohttp terdeteksi ada anomali, saya akan belajar.
PPS: sebuah proyek untuk penggemar, sayangnya tidak semua orang mengerti ini dan melemparkan kontra
