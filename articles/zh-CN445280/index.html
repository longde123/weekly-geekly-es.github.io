网站和服务的盈利能力

对于任何想在网站和服务上创建自己的业务的人来说，这篇文章都会很有用。 以下是运营一年后的获利能力和平均网站流量的示例。 本文的主要目的是为那些决定首次创业的人大致了解项目开发的复杂性。

本文假定读者已经对iT项目的获利能力进行了深入研究，并介绍了如何以及如何赚钱。


您肯定从这个故事的朋友那里听说他们很容易从他们的网站上赚到您所在国家的平均薪水（以下简称为薪水*）。 同时，网站的生存期以周为单位，与您的朋友一起创建此类项目的经验往往为零。

正是从这种情况下，我开始从事站点和服务。 在阅读了几十个论坛之后，我发现您可以在网站上赚钱。 此外，我喜欢过程自动化的主题以及为大量人员创建服务的想法。 论坛上的评论承诺收入为“ 1位访客带来1卢布”。

我已经工作了7年，我可以说关于神话般的财富的故事是不正确的。 也许，曾经有一段时间互联网上出现了上下文广告带来的超高收益，但现在已经不复存在了。 也许从前，网站在创建后的几周内便轻松超过了每天100位访问者的目标。 但是现在没有爆炸性的增长和超级利润了。 当然，有机会进入黄金分割区并制作超级受欢迎的资源。 但是这种可能性趋于零。

此外，我将仅介绍您可以赚钱的两种类型的网站。 首先是内容网站。 顾名思义，访问者来到此类站点接收内容（文本，图片，视频，音频）。 通常，在此类网站上，主要收入来源是广告（通常来自Google / Yandex的内容相关广告）。 内容网站获得私人部分付费访问的机会更少。 第二类网站是服务。 例如，用于生成域名的网站服务，SEO页面分析器服务等。此类网站可能需要付费才能使用。 我不会单独考虑商店的站点，因为从一开始，关于服务站点的所有说明都适用于它们。 很少有人会在第一次创业时就开设在线商店。

按顺序考虑以下语句：

  1. 一个每天有100人的站点是一个很大的站点。
  2. 内容网站不会产生收入。
  3. 进行廉价的启动是不可能的。
  4. 自己动手省钱。

这些要点中的每一个都充满了苛刻的现实，并经过实验验证。

一个每天有100个人的网站是一个大网站


为了使内容网站每天增长到100个访问者，至少需要工作1年才能使内容网站每周增加4到6篇新文章。

网站流量取决于主题。 如果您制作一个主题为“ hodgepodge”的博客，那么该站点将在2-3年内增加数百个。 选择一个受欢迎的话题可以将这段时间减少到1-2年。

为了使一切看起来都不那么令人沮丧，我想说有一种方法可以使内容网站具有爆炸性的普及。 您只需要了解趋势，并希望没有竞争对手（但这太棒了）。 例如，您可以将一个受欢迎的新制作的视频游戏制作为区域粉丝站点。 将会有很多粉丝，该网站将很快克服每天100位访客的障碍。 由于还没有竞争对手，这种情况可能会在数周至数月内发生。 但是这种方法一直有效到2010年。 现在，有一大批现场交易商在追赶潮流。 老实说，让自己对网站失去兴趣只是为了赚钱而已。

由于缺乏内容，网站服务的访问量比博客的访问量增长了近一个数量级，因为搜索引擎无话可说-网站上的文字很少。 内容网站的访问者中有80％来自搜索引擎。 因此，服务站点需要不断地向广告投放现金，并希望有机分配服务链接。

内容网站不会产生收入


由于网站流量低且增长缓慢，您不应依赖网站生命周期的前2-3年的收入。 即使每天有500人参加，也不会带来任何利润（您甚至无法为这个月利润购买馅饼）。

我最受欢迎的项目每天接受4000名独立用户。 在6年的工作中，他从内容相关广告中获得的收入不到1 s / n的一半。 值得一提的是，我没有尝试在网站上加载广告。 外观仍然比收入更重要。

现在，来自内容网站的1 s / n的可比收入开始于每天有2.5万人参加。 要成长为这样的指标，您需要很多年的工作，数千篇文章和广告。 要么该死的幸运。

无法进行廉价的创业


以前，网站创建者甚至没有考虑过广告其资源，因为没有竞争对手。 在这种情况下，出勤率得到了突飞猛进的增长。 现在情况正好相反。 交通争夺激烈。 现在，任何没有广告的创业公司都将蒙受巨大的金钱损失。 广告预算应该比开发初创公司的成本高出几倍。 而此金额仅足以了解网站的投资回收期。
从生活。 有一次，朋友决定将在线服务作为他的第一个创业公司。 推出几个月后，他问我，这种项目的“推广”需要多少费用。 我决定不伤害他，并呼吁削减2倍。 他说，他将不得不支付大约10至12的广告薪酬。 这足以触动市场。 但是错误地减轻苛刻的现实并没有帮助。 几天后，一个朋友关闭了这家创业公司，甚至从社交网络上删除了他的个人帐户。
迄今为止，启动广告的预算应比开发预算大几倍。 而且，此广告仅足以评估一家初创公司的盈利能力，以吸引第一批客户并了解投资回收期。

我有一家初创公司，其中所有利润仅来自B2B（企业对企业）交互。 在此启动中，吸引一个客户的成本为1 s / n 。 您准备花费1个平均月薪只是为了吸引一位客户吗？ 但是以另一种方式，启动一家初创公司将行不通。
提示。 如果您认为广告成本过高，则可以随时将其与其他企业的盈利能力进行比较。 例如，如果您租用商业地产，在最佳情况下其价值将在大约8年内收回。 如果宣传您的iT业务的成本能比8年内更快地实现，那么这样的创业公司很有可能存在。

自己动手保存


如果您完全没有创建站点/服务的经验，那么：

  1. 转到具有职位空缺和候选人的网站。
  2. 为该项目找到一名程序员。
  3. 看他的工作成本。
  4. 关闭该站点并尝试保持镇定。

中级程序员的工资是该国平均工资的3倍。 可以雇用一个吗？ 对于大多数初创公司来说，这是完全不现实的。 因为所有的钱都只会流给一名员工，广告上将一无所有。 正如他早先写道：“广告预算应该比开发初创公司的成本高出几倍。”

为了让初创企业焕发生命，您必须是一名程序员。 设计和布局可在侧面订购。 但是编程占用了大部分预算，因此别无选择。

结论


刚开始涉足网站时，我找不到关于出勤水平和收入的真实信息。 因此，我在论坛上发表了一些表面价值声明，并对结果感到非常失望。 我希望本文将帮助初学者评估预算，时间和精力，以开始他们的业务。

我感谢所有阅读这篇文章的人。 请在评论中以以下格式写出您每月在网站上能赚多少钱：“内容网站/在线服务-每天的访问者数量-每月利润，以薪金表示”。 您可以写一些站点/服务。 我先开始：

内容网站-每天250位访问者-每月可赚取0.0005工资
内容网站-每天4,000位访问者-每月可赚取0.002工资
在线服务-每天160位访问者-每月可赚取0.015工资


* 我决定以该国的平均工资来衡量收益率，以免与货币挂钩。 我希望这能扩展本文中示例的相关性。
