Aumente o nó Speedtest

Se você precisar medir a velocidade da sua conexão com a Internet, provavelmente abrirá o speedtest.net. O Speedtest permite criar seus próprios nós de medição, que outros usuários podem usar, por exemplo, será muito importante para os provedores de Internet (o servidor Speedtest principal deve ser colocado após o dispositivo, o que reduz a velocidade dos assinantes).

Sob o manual de instalação detalhado da katom.

Requisitos de sistema que os desenvolvedores oferecem ^



O Speedtest suporta muitos sistemas operacionais:

  • Windows Server (2008 ou 2012 com IIS 6, 7, 7.5 ou 8)
  • Linux (kernel 2.6.18 ou mais recente)
  • Mac OS X (construído em 10.8, as versões anteriores podem funcionar, mas não são totalmente suportadas)
  • FreeBSD (kernel 7.3 ou mais recente)

Eu usei o Ubuntu 18.04.

Então vamos em ordem. Primeiro, você precisa fazer o download do aplicativo para HTTP Legacy Fallback e do servidor principal para que o Speedtest funcione.

O HTTP Legacy Fallback requer um servidor da Web para funcionar, usei apache + php. Descompacte o conteúdo de http_legacy_fallback.zip em / var / www / html .

No arquivo OoklaServer.tgz, no meu caso, eu precisava do diretório OoklaServer-linux64 . Descompacte-o no local em que você possui programas de terceiros, por exemplo, / opt / speedtest .

Em seguida, você precisa iniciar o OoklaServer e verificar se ele inicia na inicialização do sistema.
Verifique se funcionou. Você precisa abrir http: // <seu ip>: 8080 no navegador

Se tudo estiver bem, a seguinte mensagem será exibida:


Agora confira o HTTP de fallback herdado. Abra http: // <seu ip> /speedtest/upload.php

Deve haver algo como size=500

Depois disso, o nó deve ser registrado no Speedtest. Para fazer isso, você deve se registrar em https://account.ookla.com/ e acessar a seção "Speedtest Servers". Depois disso, você pode clicar em "Adicionar servidor", uma janela com um perfil será aberta:



É necessário preencher um questionário e aguardar até que o servidor apareça na lista geral.

Cerca de um dia depois, nos assinantes da nossa rede, ele já começou a ser selecionado automaticamente na entrada do speedtest.net.

Ficarei feliz em responder suas perguntas nos comentários.
