认知矫正2：学习错觉和失真


（ c ）

数百年来，研究人员一直在讨论眼睛，大脑或人体外部未知过程导致幻觉的原因。 由于很难测量这种生理反应，因此直到最近，才可能彻底研究错觉与其生理前身之间的相关性。

近年来，由于技术的进步，我们在研究上取得了重大进展，甚至揭示了形成光学错觉的几种神经机制（有关更多详细信息，请参见第一部分 ）。 我们今天将看到更多的发现-对于某些视觉上的错觉，尚未提出一种经过确认并正确描述的科学解释。

莫伦·伯罗斯的错觉与变化



（ c ）

每个环的颜色均一，但内部区域被白色细圆圈隔开，看上去比外部区域浅。

这种错觉有很多变化。

看看下面的图片。 在您看来，在每个矩形中，落入三角形区域的右侧部分比左侧更暗，尽管实际上亮度是相同的。


（ c ）

早在1839年的米歇尔·尤金·谢夫勒（Michel Eugene Chevreul）表示，同一颜色在深色背景下会被感知为更亮，而在浅色背景下会被感知为更黑，因为该颜色为相邻的颜色提供了额外的阴影。


（ c ）

上图中所示的每个矩形的亮度均一，但是在观察者看来，靠近边界的亮度是不同的。

根据一种假设，对这些类型的错觉的责任在于用于检测特征的局部能量模型（Morrone＆Burr，1988），该模型基于以下假设：对于人脑，位于区域边界的点的权重应比权重更大内部要点。 平均图像亮度的值不重要。

另一个假设说，这种错觉是基于对光增加和减少信号进行独立的瞬态两相处理，类似于认为在巨细胞路径内部发生的那些信号，它提供了物体运动的配准，空间深度的感知以及物体之间空间关系的确定（Burr＆Morrone ，1993； Marrocco，1976； Saito＆Fukada，1986； Swanson，Ueno，Smith，1987）。

深度感知的错觉



（ c ）

深度感知的错觉是对感知对象及其属性的不充分反映。 在此示例中，由于阴影的正确排列，因此该图看起来很深。

感知我们周围世界深度的一种方法是使用所谓的单眼信号。 如果您闭上一只眼睛，评估深度会更加困难，但是仍然可以根据您的位置确定物体的近距或远距。

物体的绝对或实际大小也有助于感知深度。 甚至不知道其确切大小的较小物体的位置似乎比放置在同一位置的较大物体的位置更远。

我们对物体的了解会影响对大小和距离的感知。 因此，了解您驾驶的汽车的典型尺寸有助于您确定其他车辆的近距或远距。


透视是一种幻觉，其中两条平行的道路在某个点处以一定距离会聚。 （ c ）

有些幻觉已经成为我们生活中不可或缺的一部分，因此我们将其视为理所当然。 例如，照片中显示的道路似乎随着其远离观察点而变窄，并且小物体被认为比大物体更远。 但是，此规则也朝相反的方向起作用：在不知道对象大小的情况下，您可以看到一个对象较小，因为它距离较远。

自体运动错觉



（ c ）

运动的错觉，以跳动的心脏（或其他深色人物）的形式出现。
由于外部环境，黑色的心脏“运动”，看起来像模糊的光环。 当大脑以条纹背景“比较”中央图像时，就会出现这种效果。
颜色还可以增强运动效果。


（ c ）

但是有时候幻想会发生一件有趣的事情。 检查：


（ c ）

如果一个人以前没有被告知上述错觉上的模式应该旋转，那么他可能不会注意到认知作用。 1983年进行的科学研究“运动，位置和立体深度的视觉分离：一些现象”也有许多类似的错觉。


幻觉效果是基于正方形的一个面的细“高光”和另一个面的“阴影”的浅色和深色条纹的帮助，它们分别产生正方形在动态背景下的运动感。 如果将具有“突出显示”和“阴影”边缘的真实图形放在动态屏幕上，也会发生相同的情况。

运动假象的确切神经生物学解释尚不存在。

月亮幻觉



（ c ）

这种错觉很容易被误认为是神秘的大气现象的假象：位于地平线上方的满月看上去要比头顶更大。 这种幻想的原因仍在讨论中。 一种解释是基于对前景物体的感知，因此，地平线上方的月亮似乎是一个遥远的物体。 正如我们已经发现的，遥远意味着小。

如果您通过在上面打了个孔的纸片看，位于地平线上方的月亮看起来不会太大，从而只能看到月亮，而看不到地平线。 如果您用两只眼睛看，那么两只眼睛都会看到大小不同的月亮！

但是，有一个有趣的替代版本，它是围绕着马里奥·庞佐（Mario Ponzo）和赫尔曼·埃宾格豪斯（Hermann Ebbinghaus）的幻想而建立的。


（ c ）

意大利心理学家马里奥·庞佐（Mario Ponzo）能够通过减小中间水平线段和单调背景之间的距离来“胜过”深度感知的错觉。 结果，穿过会聚线的上段看起来更大。


（ c ）

Hermann Ebbinghaus的错觉也与对大小的感知有关：如果周围的圆靠近中心圆，则看起来较大；反之亦然，如果距离较大，则中心圆看起来较小。


（ c ）

因此，神秘月球的影响可能是由于一种颜色的天空而发生的，我们的大脑认为该颜色位于我们的头顶上方-远离地平线，更接近天顶。


（ c ）
当月球出现在地平线上时，大脑会将其解释为更遥远的物体，同时认为月球在物理上应该更大，这与埃宾豪斯和庞佐的观察一致。

结论


在体验视觉幻觉时，我们可以看到不存在的东西，或者看不到实际的东西，甚至看不到与现实不同的东西。 由于感知和现实之间的这种脱节，幻觉展示了人脑处理物理世界信息的方法。

幻觉通常被视为智能系统的缺陷和危险的大脑限制，但实际上，它们有助于创建一种工具来扩展意识和潜意识的能力。 通过探索大脑的缺陷，我们可以了解有关大脑用来形成现实感的计算方法的更多信息。
