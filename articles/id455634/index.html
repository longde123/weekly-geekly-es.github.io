Matematika dan game "Set"


Siapa pun yang menemukan "set" di sini akan mendapatkan cokelat dari saya.

Set adalah permainan yang brilian di mana kami memotong sekitar 5 tahun yang lalu. Berteriak, menjerit, memotret kombinasi.

Aturan permainan mengatakan bahwa itu diciptakan pada tahun 1991 oleh ahli genetika Marsha Falco, membuat catatan selama studi epilepsi pada gembala Jerman pada tahun 1974. Bagi mereka yang otaknya cukup lelah oleh matematika, setelah beberapa waktu ada kecurigaan bahwa ada beberapa gema di sini dengan planimetri dan menggambar garis lurus melalui titik-titik. (Untuk dua kartu yang diberikan, ada satu dan hanya satu kartu yang menyertainya dalam set yang sama.)


Marsha Falco bertanya: "Ya, tidak menemukan" set "?"

Ingat aturannya



Seth adalah permainan kartu. Semua kartu memiliki empat parameter, yang masing-masing mengambil tiga nilai (total 3 x 3 Ã— 3 Ã— 3 = 81 kartu).



Jenis dan nilai parameter adalah sebagai berikut:

  • gambar :: = elips | belah ketupat | "Ingus"
  • warna :: = merah | hijau | ungu
  • isi :: = putih | bergaris | padat
  • number :: = 1 | 2 | 3

Tujuan permainan ini adalah untuk menemukan kombinasi khusus dari tiga kartu. Tiga kartu disebut "set" jika untuk masing-masing dari empat atribut kartu baik semuanya sama atau semua berbeda.



Dengan kata lain, kita dapat mengatakan bahwa tiga kartu tidak akan membentuk set jika dua kartu memiliki satu nilai parameter dan yang ketiga berbeda. Anda dapat melihat bahwa untuk setiap dua kartu akan selalu ada kartu ketiga (dan hanya satu) yang dengannya mereka akan menjadi set.

Kemajuan permainan: pemimpin menempatkan di meja 12 kartu. Ketika seseorang menemukan satu set, dia berteriak "Set!" dan kemudian dengan tenang mengambil komponen dari kartu yang ditetapkan. Jika tidak ada set pada kartu yang diletakkan (kemungkinan besar, sepertinya tidak ada), tuan rumah mengeluarkan tiga kartu lagi.

Jumlah maksimum kartu tanpa set adalah 20. Putaran berlanjut hingga dek habis. Orang yang mengumpulkan lebih banyak set akan menang.

Matematikawan sibuk dan menyajikan kombinasi 20 kartu. Yang menganggap dirinya Chuck Norris, bisa melupakan gambar ini dan mencoba mengumpulkan "solitaire" tanpa harus mengatur sendiri.
