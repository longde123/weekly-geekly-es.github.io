Microsoft Ignite 2019大会的主要公告



在最大的Ignite 2019年度会议上， Microsoft推出了各种新工具和服务，用于业务发展以及组织和员工的更富有成效的工作。 在这篇文章中，我们收集了主要公告。 看猫下。

顺便说一句，有一个有趣的表格，其中包含会议的所有报告。 在这里学习。

蔚蓝弧




启动用于创建集成混合环境的解决方案的初步版本。 Azure Arc提供了在任何基础架构上利用Azure统一服务，管理工具和安全工具的能力，这些基础架构包括多供应商云（例如Amazon和Google），本地服务器和边缘计算设备。

新的Microsoft Edge




适用于Windows和macOS的Chromium引擎上的Microsoft Edge浏览器将于2020年1月15日以90多种语言公开提供。 今天，它可供评估下载 。 在此处阅读详细信息。

办公通用移动应用




通用的Office移动应用程序结合了Word，Excel和PowerPoint，从而无需安装单独的应用程序。 使用它，您可以创建和编辑文档，包括使用将照片转换为文本或表格的功能。 该应用程序现已作为Android和iOS用户的公共评估版提供。

二氧化硅项目




微软和华纳兄弟。 用石英玻璃记录了1978年的超人电影 。 两家公司首次确认了Microsoft Research Project Silica项目概念。 它利用激光光学和人工智能领域的最新进展来存储数据。

Azure区块链令牌




该解决方案允许您标记物理或数字资产，对其进行管理并在各种类型的区块链平台上使用它们。 使用Azure区块链令牌，区块链开发人员将可以访问令牌分类标准（TTI）标准或使用模板创建四种最受欢迎​​的令牌类型的能力。 该服务提供对用于与令牌交互的一组开放API的访问。 在企业中使用此类令牌的场景包括：供应链数字化，忠诚度计划，工作流程等等。 目前，该解决方案有一个初步版本。

皮质项目


自Microsoft Teams推出以来的第一个新Microsoft 365服务 。 该AI工具是公司内容分类器和数字助理之间的交叉。 一项智能服务能够自动查找和分发有关主题的信息，例如，在Office，Outlook和Microsoft Teams应用程序中，识别各种主题的专家，并创建组织的“知识网络”。 该解决方案将在2020年上半年公开发售，目前正在进行封闭测试。

Microsoft 365内部人员风险管理




为您的组织提供的智能安全解决方案 。 它能够使用机器学习算法匿名分析和检测非典型和可疑用户行为（例如，文件操作）的情况，从而可以防止许多标准工具和方法无法检测到的风险。 解决方案在封闭测试模式下可用。

UI流程




Power Automate （以前称为Microsoft Flow）中过程自动化服务的公共预览。 新的解决方案创建了用于任何应用程序自动化的界面机器人，包括那些不支持通过API集成的机器人。 因此，UI Flows可以大大简化组织中流程的自动化，因为它不需要编程知识即可对其进行配置：用户只需要演示他想自动化的动作的执行即可。

流语音录音改进


Microsoft Stream视频流服务现在使用机器学习算法，只需单击一下鼠标即可检测并消除任何视频中不需要的背景噪音。

Outlook中的团队聊天按钮




使您可以方便地聊天，在Outlook中的对话和团队中的聊天之间切换。

Azure Synapse分析




一项新服务 ，扩展了Azure SQL数据仓库的功能。 Azure Synapse Analytics汇总来自所有数据源，存储和大数据分析系统的信息。 这将使公司能够更快，更安全地使用其数据。
