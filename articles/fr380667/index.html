Matrix Flare - un jouet pour initier les enfants au monde de l'électronique



Le projet Matrix Flare a commencé comme un jouet personnalisé pour un enfant, et s'est progressivement développé en un concepteur électronique d'enseignement pour tout le monde. L'auteur a lancé une campagne sur Kickstarter pour collecter des fonds pour le projet, dans l'espoir d'augmenter le montant pour la production de masse de jouets.

Il est peu probable que le projet génère la bonne somme d'argent, mais il est suffisamment intéressant, nous avons donc décidé de le décrire sur le blog Medgadgets. L'auteur de l'idée espère que Matrix Flare aidera les enfants (et pas seulement eux) à comprendre les bases de la programmation et de l'électronique.

Qu'est-ce que Matrix Flare?


Ce sont des cubes où les visages sont constitués d'éléments individuels éclairés par des LED de différentes couleurs. Pour contrôler chaque cube, un logiciel spécial fonctionnant sur Visual.net est utilisé; le code pour Arduino est utilisé.

Les données d'entrée sont enregistrées à partir de l'application Pixel Maker dans le fichier anim.h utilisé par l'Arduino IDE Trinket Jewelry Sketch d'Adafruit.com. Des croquis et des instructions de programme peuvent être utilisés pour chaque «matrice». Dans le même temps, l'IDE Arduino est un outil logiciel gratuit.



Si vous le souhaitez, vous pouvez commander un programme prêt à l'emploi avec votre cube Matrix Flare.

Toutes les séquences animées peuvent être modifiées et chargées d'une nouvelle manière. Après la fin de la campagne, un grand nombre d'animations pour les cubes seront publiées sur le site Web du fabricant.



Matrix Flare Assembly


Par défaut, une animation avec un cheval est livrée avec chaque dé, donc tout fonctionne hors de la boîte. Les étuis Matrix Flare sont fabriqués sur une imprimante 3D, il existe différentes options.

Quant aux caractéristiques de Matrix Flare, elles sont les suivantes:
  • Interface USB Micro type B utilisant Arduino IDE;
  • Batterie 3.7 VDC 150mAH, la charge passe par le port indiqué ci-dessus;
  • Si vous le souhaitez, les cubes peuvent être connectés à l'USB;
  • Lorsqu'un cube simple est désactivé, et pour lire l'animation, vous devez cliquer sur le bouton Réinitialiser;




Les étuis à dés sont imprimés sur une imprimante 3D, alors qu'il existe plusieurs modèles d'étuis.
