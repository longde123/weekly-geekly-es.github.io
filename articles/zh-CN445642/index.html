Windows Virtual Desktop现在可以在Azure上公开预览

我们最近在Azure上共享了Windows虚拟桌面服务的公开预览。 现在，客户可以访问唯一的服务，该服务提供简化的管理，多会话Windows 10，对Office 365 ProPlus的优化以及对Windows Server远程桌面服务（RDS）桌面和应用程序的支持。 使用Windows虚拟桌面，您可以在几分钟内在Azure上部署和扩展Windows桌面和应用程序，同时享受内置的安全性和合规性。



这意味着客户现在可以使用多会话Windows 10，Windows 7和Windows Server桌面和应用程序（RDS）到Windows Virtual Desktop进行虚拟化，以简化Azure的管理和部署体验。 我们还为合作伙伴（包括Citrix，三星和Microsoft云解决方案提供商（CSP））将Windows虚拟桌面构建为可扩展的解决方案。


可通过适用的RDS和Windows Enterprise许可证访问Windows虚拟桌面。 有了适当的许可证，您只需设置一个Azure订阅即可立即开始。 您可以选择要适合您的环境的虚拟机和存储的类型。 您可以利用预留实例的优势（最多可享受72％的折扣）并使用多会话Windows 10来优化成本。


您可以在今天由Julia White和Brad Anderson发布的Microsoft 365博客中阅读有关Windows虚拟桌面的更多详细信息。


立即开始公开预览。
