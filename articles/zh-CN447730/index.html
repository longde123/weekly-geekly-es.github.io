电子邮件营销的演变：从QWERTYUIOP到GDPR



自美国工程师雷·汤姆林森（Ray Tomlinson）发送世界上第一封电子邮件以来，已经过去了47年。 是他在我们今天毫不犹豫地将地址中使用@ （ 符号 ）符号与整个技术乃至电子邮件文化相关联的想法。 然后，他不太可能猜测这将导致什么，当然也没有想到，四十年后，他将以那个时代的主要创新者身份进入互联网名人堂。

在过去的几十年中，电子邮件已从一种通信方式发展成为直接营销的主要形式之一。 这种方法为跟踪和响应消费者的行为提供了巨大的机会，有助于向全世界成千上万的营销人员推广和销售产品和服务。

Mail.ru Sendbox服务团队研究了电子邮件营销如何从无情的垃圾邮件演变为与受众互动的巧妙工具。

1969年 。 美国国防部启动了计算机网络的第一个发展，这开始了Internet发展的历史 。 1969年，来自加利福尼亚大学的学生通过第一个计算机网络Arpanet连接了两台计算机。 学生们通过一条5米长的电缆相互发送了一个代码字。 可以相信，这是通过计算机网络进行的第一次信息传输。

1971年 。 麻省理工学院（MIT）毕业生雷蒙德·汤姆林森（Raymond Tomlinson）发送了第一封电子邮件 。 据汤姆林森本人说，他不记得这封信的内容。



他后来说：“ 最有可能的是，第一个信息是QWERTYUIOP精神的体现。 ” 从那时起，每年10月2日，全世界都会庆祝“ 电子邮件日” 。

1971年 。 创建了第一个程序来接收和发送电子邮件 。 它由Arpanet员工Larry Roberts撰写。 一年后，熟悉的@符号出现在电子邮件地址中。

1976年 。 英国女王伊丽莎白二世在其他国家元首之前发送了她的第一封电子邮件。



1978年 。 后来被称为“垃圾邮件之父”的Gary Twerk向Arpanet用户发送了400封电子邮件，宣传DEC计算机。 加里（Gary）在信中谈到了新设备，并应邀参加了在洛杉矶举行的计算机演示展览。 有史以来第一本新闻通讯给作者带来了1300万美元的利润。

1986年 。 第一次使用“垃圾邮件”一词 ：他们开始呼叫通过电子邮件发送的强迫性广告消息。 这个名字来自罐头肉，当时在所有可能的渠道上都在积极宣传。 演唱SPAM的迷恋旋律成为嘲笑的对象，另外一个草图“ Monty Python”甚至专门用于罐头食品的普及。

1988年 。 Microsoft Mail的第一个版本出现在Mac OS操作系统上。 用户使用AppleTalk网络交换消息。 1991年，针对其他平台（包括DOS和Windows）推出了第二个版本。 它用作后续Microsoft邮件系统Outlook和Exchange的基础。

1991年 。 从太空发送了第一封电子邮件 。 它是由STS-43 Atlantis的机组人员于1991年发送的。 宇航员当时使用的是Macintosh Portable上最新的AppleLink程序来发送以下消息：“您好，地球！ 欢迎来到STS-43船员。 这是来自太空的第一个AppleLink。 这里很酷，很遗憾您不在身边...我们正在发送低温和RCS！ 阿斯塔拉维斯塔，宝贝……我会回来的。”

1992年 。 Microsoft已经发布了Outlook电子邮件客户端的第一个版本 。 同时发布了三个版本：MS-DOS的Outlook，Windows 3.1x的Outlook，Macintosh的Outlook。 Microsoft的应用程序的主要竞争对手是The Bat!。 基希讷乌的Ritlabs，SRL公司在五年后的1997年3月问世。 后来，其他开发人员也发布了他们的电子邮件客户端。 因此，有Mozilla Thunderbird和Opera Mail 。 但是它们都没有达到Outlook和The Bat的普及程度！

1994年 。 IBM Simon推出-第一款通过移动Internet进行电子邮件访问的电话 。 因此，电子邮件获得了到达收件人的其他机会。



1994年 。 颁布了总统法令“保护消费者免受不正当广告侵害” ，为我国对广告的监管奠定了基础。 尤其是对银行，金融，保险和投资服务的广告进行了限制，但互联网上的广告发行不受监管。

1995年 。 通过了一项联邦法律，该法律已经部分影响了通过网络发布广告的情况 。 例如，给出了“广告”一词的法律定义，其中提到了“以任何形式，以任何方式分发”这一短语。 发布和分发广告（包括通过提供或使用“沟通渠道”）的每个人都被视为广告分发者。 因此，对广告的一般限制出现在Internet和电子邮件上：可靠性和道德规范，对酒精，烟草，药品，金融服务等广告的限制。

1996年 。 出现了第一个免费的hotmail服务 。 多亏了他，电子邮件所有者的数量显着增加，这意味着广告邮件的潜在受众已大大增加。



1997年 。 Hotmail推出一年后，雅虎从Four11购买了RocketMail服务，并在此基础上推出了Yahoo Mail服务 。

1998年 。 在RuNet中，他们已经获得了免费邮件服务Mail.ru和subscription.ru主题邮件服务 。 两种服务都在各自的行业中树立了标准。 同时，美国更新了《数据保护法》的规定，以便所有营销邮件都有机会拒绝接收它们。

2000年 。 创建了臭名昭著的“会话英语中心” ，两年后，由于所谓的“垃圾邮件业务”，该网站在全国范围内风靡一时。 该公司每天用其中心的电话号码发送数百万封广告信。 后来，垃圾邮件成为反垄断政策部关注的焦点。

2000年 。 AOL开始将用户反馈信息广播到某些电子邮件服务，很快，它与Hotmail和Yahoo一起引入了从接收者那里接收反馈的方案。 因此， 营销人员能够找出人们对信件的看法，并使用垃圾邮件投诉作为衡量标准 。 后来，Internet服务提供商引入了保护客户免受“垃圾邮件”电子邮件攻击的方法，并且大多数垃圾邮件不再传递给收件人。

2003年 。 在美国，Can Spam Law出现了 ，它首先解决了商业电子邮件。 同年在欧洲通过了《隐私和电子通信条例》法，该法定义了允许进行市场传播的规则。

2004年 。 出现了“发件人策略框架”（SPF） ，它使您可以验证发件人的真实性并通过控制发件人的IP地址来防止垃圾邮件。

2007年 iPhone-第一台能够从移动设备查看邮件的Apple智能手机。 到目前为止，对自定义智能手机的电子邮件支持通常受到限制。 同时，RIM的BlackBerry商业智能手机的所有者在2003年获得了使用电子邮件工作的机会。



2009年 。 Return Path报告说，发送的商业信函中有将近30％没有到达邮箱。 在同一年，默克尔说，相关性低是用户删除此类电子邮件而不阅读的主要原因。 在此方面，免费电子邮件服务的提供者和创建者以各种可能的方式帮助了他们。 在2000年代后期，Internet服务提供商还引入了一些方法来保护客户免受“垃圾邮件”的侵害。 由于采取了这些措施， 垃圾邮件已逐渐被成熟的电子邮件营销所取代 ：营销人员意识到必须保持邮件的声誉，否则用户会将其列入黑名单或简单地将其放入购物篮中。 信件变得及时且与接收者相关。

2012年 。 现在，有40％的营销电子邮件在移动设备上打开 。 移动互联网的发展促进了这种发展-现在可以随时随地获得对网络的访问。 结果，营销人员开始特别注意他们的字母在小屏幕上的显示方式。

2012年 。 迈克尔·珀索德（Michael Persod）案：2012年至2015年，一名嫌疑人使用几个IP地址和域代表总部位于加利福尼亚的Impact Media LLC发送垃圾邮件，并以迈克尔·普雷斯科特，迈克尔·皮尔森和杰夫·马丁内斯的身份介绍自己。 欺诈者提供各种商品和服务，并从卖方那里获得佣金以成功交易 。 报复行动取代了这位加利福尼亚居民，于2016年在他自己的住所中被联邦调查局逮捕。 一名联邦法官下令Persod赔偿490,000美元，并赔偿非金钱损失54,000多美元。

2013年 。 Dropbox收购了Mailbox，这是一个有前途的初创企业，由于其原始界面和当时通过滑动的异常控制，它似乎将永远改变人们使用电子邮件的方式 。 两年后，该项目已关闭-事实证明，当时用户不习惯刷卡，也不准备更改其常规邮件程序。

2014年 。 关闭另一个启动。 这次，发件箱没有达到开发人员的期望。 假定每月仅5美元，特殊人员就可以将您的所有脱机邮件数字化并通过网络发送给您 。 公众非常谨慎地接受了这个不寻常的想法，并且没有得到发展。

2018年 通用数据保护条例 （GDPR）已在欧洲生效 。 欧盟法规旨在加强对欧盟国家公民的个人数据的保护。 处理此类数据的公司的责任已大大增加，违反法规可能面临巨额罚款。 这项措施还有助于简化电子邮件营销，并将其置于更加文明的基础上。

结论


电子邮件是最有效，最便捷的通信方式之一。 尽管这项技术已有相当长的历史，但它并没有过时，但已发展成为一种有用的，有时对于用户来说必不可少的工具，以及一种有趣的营销工具。 即时通讯工具和社交网络都可以极大地帮助营销人员实现其目标，但是电子邮件在消费品牌信息和可靠通信方面仍然是领导者。 在接下来的十年中，这种情况不太可能改变，因为邮件服务也在发展：新技术的出现，提出了越来越多的“芯片”，这些芯片将在将来帮助电子邮件保持竞争性的通信方式。
