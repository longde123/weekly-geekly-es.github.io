Apa yang baru di konsol web 2019

Pada 2016, kami menerbitkan artikel terjemahan, “ Panduan Lengkap untuk Konsol Web 2016: cPanel, Plesk, ISPmanager, dan lainnya .” Saatnya memperbarui informasi pada 17 panel kontrol ini. Baca deskripsi singkat panel itu sendiri dan fitur-fitur baru mereka.


cpanel


Konsol web multifungsi paling populer pertama di Dunia, standar industri. Ini digunakan oleh pemilik situs (sebagai panel kontrol) dan penyedia hosting (sebagai alat untuk mengelola Web Host Manager, WHM). Antarmuka intuitif, tidak memerlukan pelatihan, multibahasa. Ada instruksi video.

Basis Jawa: Perl, PHP
OS yang didukung: Red Hat Enterprise Linux (RHEL), CentOS, CloudLinux. Dimungkinkan untuk mendukung Windows melalui virtualisasi atau melalui panel Enkompass dari pengembang yang sama.


cpanel


Whm

Baru

Pengembang terus berusaha mempercepat panel dan secara umum bekerja aktif untuk memperbaikinya, berdasarkan permintaan pelanggan. Jadi, dalam versi ke-82 saat ini, Instalasi selesai dalam 3 menit. Waktu pembaruan CPanel & WHM telah ditingkatkan: dari kedua dari belakang, ke-80, berakhir dalam tiga menit, dan dari yang sebelumnya dalam delapan. Pada tahun 2019, persyaratan ruang disk untuk pemasang cPanel & WHM berkurang 10%. Ada: kompatibilitas dengan PCI; backup dan restore otomatis; alat yang memungkinkan Anda untuk memasukkan daftar hitam dan akun daftar putih, alamat IP dan seluruh negara; Sertifikat SSL gratis untuk setiap situs web. Sekarang Anda dapat memasukkan satu file ke file lain (tambahkan konfigurasi). Secara umum, selama waktu ini, pekerjaan cPanel & WHM dipercepat sebesar 90%, sumber daya server yang diperlukan berkurang sebesar 30%.

Pada bulan April 2019, para pengembang mengumumkan dengan lembut, mungkin permintaan pembaruan fitur cPanel yang paling banyak diminta - menambahkan server web NGINX sebagai alternatif dari Apache. Fungsi ini berfungsi dalam format eksperimental. Dokumentasi pembaruan resmi.

Harga

Tergantung pada tingkat akun: Solo $ 15, Admin $ 20, Profesional $ 30, Premier $ 45 per bulan. Masa percobaan gratis. Ada program afiliasi. Dukungan teknis prioritas insiden $ 65.

Plesk


Panel kontrol favorit oleh penyedia hosting besar, yang dapat dimengerti bahkan bagi pemula. Antarmuka tunggal yang nyaman dengan mana Anda dapat secara terpusat mengelola semua layanan sistem. Ini dikirim dalam publikasi yang berbeda untuk hosting dan skenario penggunaan tertentu.

→ Tentang Plesk di situs web resmi

Basis Jawa: PHP, C, C ++
OS yang didukung: berbagai versi Linux, Windows


Plesk

Baru

Fitur panel baru datang dalam bentuk ekstensi yang dikompilasi ke dalam direktori di situs. Antarmuka telah ditingkatkan secara signifikan: desain adaptif, kemampuan untuk secara otomatis login klien ke Plesk dari sumber daya eksternal tanpa re-otentikasi (misalnya, dari panel penyedia hosting mereka), kemampuan untuk berbagi tautan langsung ke layar; Antarmuka pengguna yang dirancang ulang untuk penjadwal tugas. Peningkatan manajemen basis data; ada dukungan untuk beberapa versi PHP, serta Ruby, Python dan NodeJS; dukungan penuh untuk Git; Integrasi Docker Toolkit SEO. Alat pemulihan Plesk, utilitas baris perintah yang dapat digunakan untuk secara otomatis mendeteksi dan memperbaiki berbagai masalah, telah muncul. Setiap instance Plesk sekarang secara otomatis diamankan menggunakan SSL / TLS. Anda dapat mengurangi waktu respons situs dan memuat di server menggunakan Nginx Caching. Fitur yang disebut Pembaruan Cerdas telah ditambahkan ke ekstensi WordPress Toolkit yang populer, yang menganalisis pembaruan WordPress dengan kecerdasan buatan untuk menentukan apakah menginstal pembaruan dapat merusak sesuatu.

Harga

RUVDS juga menyediakan panel Plesk untuk pelanggannya, harga 1 lisensi adalah 650 rubel per bulan.

Directadmin


Pengembang memposisikan panel sebagai yang termudah untuk dikelola di dunia. Mereka mencoba mengikuti perkembangan zaman dan menggunakan teknologi canggih, sementara tidak ada yang supernatural di panel - hanya fungsi dasar . Tidak ada skrip yang telah ditentukan, tetapi Anda dapat membuat skrip Anda sendiri (API terbuka). Antarmuka multibahasa, tetapi tanpa dukungan Rusia (Anda dapat menggunakan skin tidak resmi). Filter antispam yang lemah. Tapi - tidak menuntut sumber daya server dan kecepatan tinggi. Akses bertingkat.

Basis PL: PHP
OS yang didukung: FreeBSD, GNU / Linux (Fedora, CentOS, Debian, distribusi Red Hat)

Baru

Admin inti


Manajemen terpusat dari beberapa server, gambaran global dari keseluruhan sistem. Ada banyak aplikasi untuk melakukan tugas-tugas umum setiap hari: mulai dari menganalisis log secara real time hingga sistem pemblokiran ip terintegrasi, dari melihat semua proses dan layanan hingga pemeriksaan eksternal. Sistem delegasi izin yang mudah. Platform ini dapat diperluas, multibahasa.

→ Lebih lanjut tentang fitur

Basis PL: PHP
OS yang didukung: Linux


Admin inti

Baru

Sekarang Anda dapat menghubungkan pengguna akhir ke server dalam beberapa klik dan langsung mengelola setiap server yang terhubung. Aplikasi Core-Admin Web Edition dan Core-Admin Free Web Edition menyediakan solusi khusus yang dirancang untuk kenyamanan pemrosesan server: mail, server web, FTP dan DNS. Pemantauan file web tertentu telah muncul untuk mendeteksi peretasan umum. Ada pemblokiran otomatis alamat IP ketika login gagal di berbagai layanan dan melacak pengiriman IP-mail untuk mendeteksi penggunaan server yang tidak sah. Tampilan majalah yang terintegrasi secara real time.

Harga

"Free Web Edition" 10 domain - gratis, "Micro" 15 domain - 5 € / bulan, "Starter" 20 domain - 7 € / bulan, "Base" 35 domain - 11 € / bulan, "Standar" 60 domain - 16 € / bulan, “Profesional” 100 domain - 21 € / bulan, “Premium” - jumlah domain tidak terbatas - 29 € / bulan.

Interwork


Ini terdiri dari dua modul: Nodeworx untuk mengelola server dan Siteworx untuk mengelola domain dan situs. Antarmuka pengguna sederhana dan intuitif. Panel beratnya sedikit. Aplikasi diinstal dengan cepat, sistem templat yang nyaman. Administrasi dilakukan melalui Shell, ada antarmuka baris perintah. Komunitas pengguna aktif.

Basis PL: PHP
OS yang didukung: Linux


Interworx

Baru

Nodeworx memperkenalkan pengelompokan beberapa server secara bersamaan, yang memungkinkan Anda untuk menskala cluster sesuai dengan persyaratan keandalan dan aksesibilitas aplikasi web modern. Lebih detail di menu pengelompokan . Siteworx memiliki statistik yang baik dan cadangan satu-klik.

ISPmanager


Panel pengembang Rusia tersedia dalam dua versi: ISPmanager Lite untuk mengelola VPS dan server khusus, ISPmanager Bisnis untuk menjual shared hosting (terintegrasi dengan platform penagihan BILLmanager).

Delegasi hak akses yang mudah digunakan (pengguna, pengguna FTP, administrator) dan menetapkan batas pada sumber daya (kotak surat, disk, domain, dll.). Mengkonfigurasi dan mengelola ekstensi Python, PERL, dan PHP. Pengelola file bawaan yang nyaman. Panel tidak memerlukan keterampilan pelatihan dan administrasi server virtual.
