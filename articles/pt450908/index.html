Redes negras para o Asterisk

Após instalar o Asterisk e colocar o gateway na Internet, os ataques começam imediatamente com as tentativas de conexão e realização de chamadas. Para facilitar o trabalho de outros administradores de sistema, eu publico uma lista negra de redes.

Após a última rede, com a qual o ataque foi realizado, já havia um mês de silêncio. Obviamente, com o tempo, os invasores usarão novas redes, mas não haverá problemas para adicionar novas a esta lista no futuro. Para quem não sabe onde esta lista está armazenada (eu tenho uma interface web no FreePBX 14.0.11):

  1. vá para a interface da web
  2. Conexão-> Firewall
  3. Serviços-> Lista negra

Se esta lista contiver a rede que você está usando, basta removê-la. No futuro, adicione à lista negra apenas o endereço a partir do qual os ataques serão. As informações sobre a instalação de fogo no asterisco na Internet estão completas.
