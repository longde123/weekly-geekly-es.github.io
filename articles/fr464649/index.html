Système de collaboration de documents pour Zimbra Open-Source Edition

L'importance de la coédition de documents dans les entreprises modernes est difficile à surestimer. La possibilité de rédiger des contrats et des accords avec la participation d'employés du service juridique, de rédiger des offres commerciales sous la supervision de supérieurs en ligne, etc., permet à l'entreprise d'économiser des milliers d'heures de travail auparavant consacrées à de nombreuses approbations. C'est pourquoi l'une des principales innovations de Zextras Suite 3.0 a été l'apparition de Zextras Docs - une solution qui vous permet d'organiser une collaboration à part entière avec des documents directement dans le client Web Zimbra Collaboration Suite Open-Source Edition.

Actuellement, Zextras Docs prend en charge la collaboration avec des documents texte, des tableaux et des présentations, et sait également comment traiter un grand nombre de formats de fichiers. L'interface de la solution n'est pas très différente de l'interface de n'importe quel éditeur de texte, de sorte que les employés de l'entreprise n'ont pas à consacrer beaucoup de temps à la formation pour passer à l'utilisation de Zextras Docs. Mais le plus intéressant, comme toujours, est «sous le capot». Voyons ensemble comment fonctionne Zextras Docs et quels avantages cette solution de collaboration de documents peut offrir.



Surtout, Zextras Docs plaira à ceux qui utilisent déjà Zimbra OSE et Zextras Suite dans leur entreprise. Avec cette solution, il est possible d'introduire un nouveau service en production sans augmenter le nombre de systèmes d'information et, par conséquent, sans augmenter le coût de possession de l'infrastructure informatique. Notez que Zextras Docs est uniquement compatible avec Zimbra OSE version 8.8.12 et versions ultérieures. C'est pourquoi si vous utilisez toujours des versions obsolètes de Zimbra, nous vous recommandons fortement de passer à la version 8.8.15 LTS. En raison du support à long terme, cette version restera à jour et sûre pendant plusieurs années, et sera également compatible avec tous les ajouts actuels.

Les avantages de Zextras Docs peuvent également inclure la possibilité de son déploiement complet sur l'infrastructure de l'entreprise. Cela évite le transfert d'informations à des tiers, comme c'est souvent le cas lors de l'utilisation de services hybrides ou cloud. C'est pourquoi Zextras Docs est idéal pour les entreprises ayant une politique de sécurité des informations stricte et pour les administrateurs système qui préfèrent contrôler complètement les flux de données et les processus se produisant dans l'entreprise. De plus, un service de collaboration de documents déployé localement vous permet d'éviter les risques associés à l'inaccessibilité du service cloud lorsque des problèmes surviennent avec le fournisseur de services ou lorsque des problèmes surviennent avec un accès Internet haut débit.

Zextras Docs se compose de trois parties: un serveur autonome, une extension et un hiver. Chacune de ces trois parties fait sa part:

  • Le serveur Zextras Docs est un moteur LibreOffice, optimisé pour la collaboration et l'intégration avec Zimbra OSE. C'est sur le serveur Zextras Docs que tous les documents consultés par les utilisateurs sont ouverts, traités et stockés. Il doit être installé sur un nœud informatique dédié exécutant Ubuntu 16.04, Ubuntu 18.04 ou CentOS 7. Si la charge sur le service Zextras Docs est suffisamment importante, vous pouvez sélectionner plusieurs serveurs à la fois.
  • L'extension Zextras Docs n'a pas besoin d'être installée, car elle est intégrée à la suite Zextras. Grâce à cette extension, l'utilisateur est connecté au serveur Zextras Docs, ainsi qu'à l'équilibrage de charge lors de l'utilisation de plusieurs serveurs. De plus, grâce à l'extension Zextras Docs, plusieurs utilisateurs sont connectés simultanément à un même document, et les fichiers sont téléchargés du stockage local vers le serveur.
  • Zextras Docs Zimlet, à son tour, est nécessaire pour intégrer le service dans le client Web. C'est grâce à lui que le client web Zimbra a la possibilité de créer, ainsi que la fonction de prévisualisation des documents Zextras Docs.



Afin de déployer Zextras Docs dans l'entreprise, vous devez d'abord sélectionner un ou plusieurs serveurs physiques ou virtuels pour celui-ci. Après cela, vous devez télécharger les distributions d'applications serveur pour Ubuntu 16.04 , Ubuntu 18.04 ou CentOS 7 à partir du site Web Zextras, puis décompressez-le et installez-le. À l'étape finale de l'installation, le serveur vous demandera de spécifier l'adresse IP du serveur LDAP, ainsi que la paire nom d'utilisateur / mot de passe qui sera utilisée pour entrer les données sur le nouveau serveur dans LDAP. Notez qu'une fois l'installation terminée, chaque serveur Zextras Docs sera visible par tous les autres nœuds d'infrastructure.

Étant donné que l'extension Zextras Docs est déjà contenue dans la suite Zextras, il suffira simplement de permettre son utilisation pour les utilisateurs et les groupes qui devraient avoir accès aux outils de collaboration de documents. Le Zextras Docs Zymelet peut être activé à partir de la console d'administration Zimbra. Notez qu'après avoir ajouté le serveur Zextras Docs à l'infrastructure Zimbra OSE, vous devez mettre à jour la configuration du serveur proxy Zimbra. Pour ce faire, exécutez simplement le fichier / opt / zimbra / libexec / zmproxyconfgen en tant qu'utilisateur Zimbra, puis exécutez la commande zmproxyctl restart pour redémarrer le service proxy.
