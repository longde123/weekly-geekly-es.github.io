25 perpustakaan Android menghibur. Musim Semi 2018

Ini adalah daftar 25 pustaka, proyek, dan alat Android terbaru yang dirilis dalam 3 bulan terakhir. Tidak ada tempat pertama dan terakhir, ini hanya daftar perpustakaan yang menarik. Mereka pasti layak mendapatkan perhatian Anda. Selamat menikmati!



1. scrcpy


Alat ini dikembangkan oleh tim Genymotion dan melakukan hal yang sama seperti Vysor. Ini memberikan tampilan dan manajemen perangkat Android yang terhubung melalui USB. Tidak diperlukan hak root untuk operasinya. Ini bekerja di Linux, Windows dan MacOS. Dan gratis!



Readme di sini sangat bagus. Di dalamnya Anda akan menemukan deskripsi lengkap tentang instalasi dan peluncuran proyek. Untuk pengguna Mac, proyek ini juga tersedia melalui homebrew.

2. Bayangan Panjang


Perpustakaan ini untuk dengan mudah membuat bayangan panjang yang indah di Android.



Anda akan menemukan banyak fitur menarik di perpustakaan ini:

  • dukungan untuk segala jenis komponen tampilan,
  • kontrol penuh atas tampilan bayangan,
  • kemampuan untuk mengatur beberapa bayangan untuk komponen View,
  • animasi bayangan
  • komputasi asinkron
  • dan masih banyak lagi.

README penuh dengan contoh-contoh hebat dan ditulis dengan sangat baik. Ini mungkin salah satu README terbaik yang pernah saya baca. Proyek ini dirilis di bawah lisensi MIT.

3. TextPathView


Ini adalah komponen View yang menghidupkan rendering teks. Lebih baik melihat gif di bawah sekali!



README asli ditulis dalam bahasa Cina, tetapi ada juga versi bahasa Inggris. Di sana Anda akan menemukan informasi tentang cara menggunakan perpustakaan dan pembaruan apa yang sedang disiapkan untuk rilis. Perpustakaan masih pada tahap awal pengembangan (versi 0.1.2), jadi bersiaplah untuk masalah yang muncul. Proyek ini dirilis di bawah lisensi MIT.

4. ShapeOfView


Ini adalah perpustakaan yang memungkinkan Anda untuk membuat bentuk kustom dalam bentuk yang tidak biasa.



Berikut adalah contoh lain dari apa yang dapat Anda lakukan dengan perpustakaan:



Mengesankan, ya? Anda dapat menemukan lebih banyak contoh dalam dokumentasi. Perpustakaan itu sendiri dilisensikan di bawah Apache 2.0.

5. ModalBottomSheetDialogFragment


Perpustakaan ini dirancang untuk menggunakan kotak dialog sembulan materi di bawah ini.



Dokumentasi sudah cukup untuk mulai menggunakan dialog. Proyek itu sendiri dirilis di bawah lisensi Apache 2.0.

6. PhotoEditor


Perpustakaan Photo Editor dirancang hanya untuk mengedit gambar menggunakan cat, teks, emotikon dan stiker.



README sangat bermanfaat dan bagus. Proyek ini berada pada tahap pengembangan yang agak awal (versi 0.1.1) dan dirilis di bawah lisensi Apache 2.0.

7. Hyperion-Android


Hyperion adalah alat untuk debugging dan memverifikasi aplikasi. Itu dapat dengan mudah diintegrasikan ke dalam aplikasi apa pun.



Anda dapat memperoleh informasi lebih lanjut dengan membaca README atau blog pengembang. Proyek ini dirilis di bawah lisensi MIT, dan saat ini di versi 0.9.22.

8. KotlinSilahkan Animate


Ini adalah perpustakaan untuk membuat animasi indah yang ditulis di Kotlin.



Dokumentasi di lapangan sudah cukup untuk mulai bekerja dengannya. Proyek ini dirilis di bawah lisensi Apache 2.0.

9. ElevationImageView


Berkat pustaka ini, Anda dapat menambahkan bayangan ke ImageView Anda.



Dokumentasinya cukup singkat, tetapi cukup untuk mulai menggunakan perpustakaan. Versi perpustakaan saat ini 1.1, dan proyek dirilis di bawah lisensi Apache 2.0.

10. Tombol Cepat Mengambang Dial Cepat


Perpustakaan ini menerapkan tombol mengambang dari desain material.


Dokumentasi, selain teori, termasuk contoh aplikasi yang tersedia di Google Play.
Proyek ini dirilis di bawah lisensi Apache 2.0.

11. RadiusTransition


Ini adalah aplikasi yang memungkinkan Anda untuk mengubah gambar lingkaran menjadi kotak. Gambar dalam contoh ini ditampilkan melalui Glide.



README singkat, tetapi Anda dapat membaca kode jika mau, karena cukup sederhana. Tidak ada lisensi yang disebutkan.

12. hiroaki


Tujuan Hiroaki adalah untuk membantu menguji integrasi API menggunakan kekuatan Kotlin.

Pustaka menyediakan MockWebServer untuk membuat stub server sebagai target permintaan HTTP Anda yang akan Anda kirim ke backend Anda.
Ini akan memungkinkan Anda untuk memeriksa bagaimana program Anda akan menanggapi perilaku server dan API tertentu.



Dokumentasi ini patut diintip. Proyek ini ditulis di bawah lisensi Apache 2.0.

13. Kincir Ria


Banyak dari Anda mungkin pernah mendengar salah satu roda Ferris tertinggi di dunia - London Eye. Sekarang dalam aplikasi Anda, Anda dapat membuat Ferris wheel sendiri berkat perpustakaan Ferris-Wheel.



Di README, Anda dapat membaca cara menyambungkan dan menggunakannya dalam proyek Anda. Ini dirilis di bawah lisensi Apache 2.0.

14. gradle-dependency-graph-generator-plugin


Ini adalah plugin Gradle yang memungkinkan Anda untuk memvisualisasikan dependensi Anda pada bagan.



Dalam dokumentasi Anda akan menemukan cara mengkonfigurasi plugin dan cara menggunakannya. Proyek ini dibuat di bawah lisensi Apache 2.0.

15. Izin Runtime


Perpustakaan ini, menurut Florent Champigny, adalah cara termudah untuk meminta izin dinamis di Android versi baru.



Lihat dokumentasi untuk informasi lebih lanjut. Proyek ini dirilis di bawah lisensi Apache 2.0.

16. Prefekt


Prefekt adalah Preferensi Bersama Android untuk Kotlin. Mudah digunakan dan efisien berkat cache dalam memori. Anda dapat berlangganan pembaruan nilai di SharedPreference, dan langganan akan berfungsi meskipun nilainya tidak diubah melalui fungsi Prefekt.



Ada banyak fitur menarik lainnya yang dijelaskan dalam dokumentasi, sehingga proyek ini patut Anda perhatikan. Dirilis di bawah lisensi Apache 2.0.

17. ColorPickerPreference


Perpustakaan yang memungkinkan Anda untuk mengimplementasikan ColorPickerView, ColorPickerDialog, dan ColorPickerPreference. Dengan itu, Anda bisa mendapatkan warna HSV, nilai RGB, kode warna HTML untuk gambar dari galeri.


Dokumentasinya komprehensif dan bahkan berisi deskripsi metode. Jelas perpustakaan yang bermanfaat. Lisensi: Apache 2.0

18. Visualisasi Musik


Menggunakan perpustakaan ini Anda dapat memvisualisasikan file audio Anda. Untuk ini, perpustakaan menggunakan fungsionalitas Android OpenGL ES.



Di bagian rilis perpustakaan di github Anda dapat menemukan file .apk untuk operasi perpustakaan contoh. Didistribusikan di bawah lisensi Apache 2.0.

19. TreeView


Android TreeView digunakan untuk menampilkan data dalam struktur pohon.



Dokumentasi ini dirinci dan menjelaskan semua langkah yang diperlukan untuk menghubungkan perpustakaan ke proyek Android Anda. Lisensi: Apache 2.0

20. inline-youtube-view


Utilitas ini dirancang untuk menyematkan komponen YouTube di aplikasi Android Anda. Komponen YouTube dirancang untuk bekerja dengan Android, iOS, dan React Native.



Perpustakaan menjelaskan beberapa masalah yang mungkin Anda temui, misalnya:
  • SDK YouTube tidak berfungsi di semua perangkat
  • Anda tidak dapat menjalankan lebih dari satu instance komponen YouTube sekaligus


21. WhatsNew



WhatsNew secara otomatis menampilkan deskripsi singkat tentang fitur-fitur baru aplikasi Anda setelah pengguna memperbaruinya.



Di README, Anda dapat membaca cara menggunakan perpustakaan. Proyek ini dirilis di bawah lisensi MIT.

22. ColoredShadowImageView


Ini adalah perpustakaan kedua dalam daftar ini yang memungkinkan Anda untuk membuat bayangan indah di sekitar gambar berdasarkan skema warnanya.



Dalam dokumentasi Anda akan menemukan contoh file .apk dan deskripsi pengaturan dan penggunaan. Proyek ini dirilis di bawah lisensi Apache 2.0.

23. easylauncher-gradle-plugin


Pustaka ini akan membantu Anda mengubah ikon aplikasi untuk masing-masing buildVarian Anda, menggunakan aturan Gradle sederhana. Anda dapat menambahkan potongan warna apa saja, melapisi gambar Anda sendiri atau mengubah warna ikon.



Dokumentasi berisi deskripsi penggunaan perpustakaan, cara mengkonfigurasinya, parameter yang tersedia dan deskripsi struktur proyek. Proyek ini dirilis di bawah lisensi Apache 2.0.

24. ConcealerNestedScrollView


Ini adalah pustaka yang memudahkan untuk menyembunyikan komponen tampilan di bagian atas dan bawah layar saat menggulir melalui NestedScrollView kustom.



Dalam dokumentasi Anda akan menemukan semua yang Anda butuhkan untuk mencapai efek ini.

25. Teater


Ini adalah proyek yang menggunakan arsitektur bersih + MVVM + ekstensi reaktif + komponen arsitektur Android. Tujuan utamanya adalah menggunakan praktik dan perpustakaan terbaru.



Proyek ini menggunakan perpustakaan seperti: Android KTX, komponen arsitektur Android, Dagger 2, RxJava, RxKotlin, RxAndroid, ConstraintLayout, Glide dan banyak lainnya. Itu benar-benar perlu dilihat! Lisensi: Apache 2.0.
