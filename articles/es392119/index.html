Proyecto Reflecty: un espejo como parte de una casa inteligente



DerVonDenBergen decidió hacer que el espejo sea inteligente. Él razonó que hay espejos en cada casa, ocupan mucho espacio, pero los espejos comunes tienen pocas funciones, para ser sincero. ¿Por qué no mejorar ligeramente el espejo?

Después de una planificación detallada, el proyecto para crear un espejo inteligente comenzó a implementarse. Al principio, apareció un concepto de video, y ahora, el primer prototipo, que funciona bastante bien. Según el autor, el espejo inteligente Reflecty permite al usuario controlar el hogar inteligente, además de proporcionar a la persona la información necesaria a pedido. Para rastrear las acciones del usuario, el espejo está equipado con un detector de movimiento y un módulo de reconocimiento de gestos Leap Motion. La distancia efectiva de trabajo para el espejo es de unos 20 centímetros.









Capacidades:
  • Widgets en la "pantalla principal", que muestra información meteorológica, fecha / hora, últimas noticias. Todo esto se actualiza en tiempo real.
  • Aplicación meteorológica que se puede abrir utilizando el widget correspondiente en la pantalla principal. La aplicación muestra el clima actual y el pronóstico con varios días de anticipación.
  • Información sobre rutas de transporte público, incluidas paradas cercanas.
  • También se abrió una aplicación de noticias con un widget.

Detrás del espejo de doble cara, un monitor que muestra toda la información necesaria "Brain" de todo el sistema es una mini PC Shuttle XPC. Según el autor, la Rasberry Pi sería una solución económica, pero como ya había una mini PC mencionada anteriormente, se decidió instalarla. Lamentablemente, no se dice nada sobre el precio del proyecto.
