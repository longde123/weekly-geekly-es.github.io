Bagaimana mengatur 120.000 foto sehingga tidak ada tryndets, dengan tingkat akses yang berbeda, untuk tim



Saya memutuskan untuk mencari karya seniman Sci-Fi dari masa lalu dan sekarang, yang melukis jetpack. Saya menemukan grup di VK "Museum Masa Depan", dan di sana - 16.000 foto. Saya harus mempertimbangkan kembali semuanya secara manual, saya mengambil beberapa lusin dengan perangkat seperti jetpack ( lihat pilihan di bawah potongan ), dan kemudian saya berpikir, bagaimana dengan fotografer miskin? Atau kawan-kawan lain yang bekerja dengan banyak foto / audio / video, dan tidak hanya sendiri, tetapi oleh tim, jarak jauh, dll., Dll.

Saya bertanya kepada teman-teman saya, fotografer - mereka menggunakan semacam campuran lightroom, google drive, dan hard drive yang dapat dilepas. Mereka mengatakan neraka total.

Saya bertanya kepada seorang teman yang adalah manajer komunitas komunitas game yang berbahasa Rusia dari sebuah perusahaan besar di industri game dengan huruf "B". Mereka perlu membuat katalog dan mencari di antara sejumlah besar materi audio dan video. Buat banyak trailer untuk video, tangkapan game, efek suara, musik. Banyak file terkait dengan berbagai proyek yang perlu diatasi untuk pencarian lintas proyek lebih lanjut. Tidak terlalu penting untuk menemukan dengan cepat, tetapi untuk menemukan secara prinsip.

Saya juga bertanya kepada arsitek, yang memiliki 120.000 foto objek dalam database, bagaimana mereka hidup dengannya.

Ternyata seluruh area ini disebut manajemen aset digital. Berikut adalah daftar peringkat dari 30+ solusi manajemen sumber daya digital .

Aset digital dapat menyertakan file digital apa pun, dan biasanya digunakan oleh tim kreatif dan pemasaran. Perangkat lunak manajemen sumber daya digital membantu mengatur, mengklasifikasikan, mencari, mengambil, dan mengarsipkan file digital seperti gambar, dokumen, file kreatif, klip audio dan video.

Manajer sumber daya digital dapat mendukung proses kompleks untuk menerima, mendokumentasikan, membuat katalog, mengedit, mencari, memproses, dan mengelola konten digital dan multimedia.

Apa itu perangkat lunak manajemen sumber daya digital?


Fitur khas dari perangkat lunak ini adalah penyimpanan cloud, pencarian, berbagi sumber daya dan manajemen merek. Serta pengarsipan dan pencadangan, otomatisasi alur kerja, pelacakan penggunaan dan e-commerce. Anda dapat menempatkan tanda air pada sumber daya digital Anda, seperti gambar, untuk perlindungan hak cipta - masalah umum di industri kreatif. Program ini juga memulai dan memungkinkan Anda untuk mengatur parameter yang menyediakan akses hanya untuk pengguna tertentu.

Ada beberapa jenis program ini berdasarkan fungsionalitas:

  • sebuah sistem yang mengelola sumber daya perpustakaan,
  • sebuah sistem yang mengelola sumber daya merek,
  • sebuah sistem yang mengelola sumber daya di cloud,
  • dan sistem yang mengelola sumber daya produktif.

Perangkat lunak ini memfasilitasi perencanaan manajemen semua tahap siklus hidup sumber daya digital.

Jenis Perangkat Lunak Manajemen Sumber Daya Digital


Masing-masing jenis ini berorientasi pada penggunaan operasional atau produksi kreatif. Jenis produksi kreatif berteknologi tinggi dan biasanya sulit digunakan, sedangkan jenis kerjanya ramah pengguna dan memungkinkan mereka mengakses sumber daya kapan saja.

  • Sistem Manajemen Sumber Daya Perpustakaan : Ini berfokus pada penyimpanan dan pengambilan sejumlah besar sumber daya media yang tidak sering mengubah ukuran.
  • Sistem Manajemen Sumber Daya Merek : Ini berfokus pada penggunaan kembali konten yang digunakan dalam penjualan dan pemasaran bahan. Ini termasuk gambar produk, materi promosi, dan logo.
  • Sistem Manajemen Sumber Daya : berfokus pada penyimpanan, organisasi, dan tinjauan kontrol media digital yang sering berubah.
  • Sistem Manajemen Sumber Daya Pemasaran : Ini mengelola semua sumber daya pemasaran, seperti rencana pemasaran, file digital, pedoman merek dan sumber daya pemasaran lainnya.
  • Sistem manajemen konten digital : ia mengelola semua sumber daya digital dan memiliki kemampuan untuk menyematkan konten di situs web.
  • Sistem Manajemen Sumber Daya Media : Ini mengelola file besar konten audio dan video.

Fungsi utama


  • Unggahan sumber daya: Fitur ini menyediakan platform yang luar biasa bagi pengguna untuk mengunduh file individual atau bahkan ribuan file dalam mode batch. Fungsi ini juga memiliki kemampuan untuk mengambil metadata selama proses pengunduhan.
  • Manajemen Sumber Daya: Ini terdiri dari beberapa alat yang cocok untuk mengelola sumber daya digital, mengonversi file, mengedit metadata, membuat versi, dan mengelola hak digital. Alat yang disediakan oleh fitur ini juga membantu meningkatkan keamanan sumber daya, serta menyederhanakan pencarian mereka dan memungkinkan Anda untuk membagikannya dengan pengguna lain.
  • Dasbor: Menampilkan berbagai jenis data tentang sumber daya digital, tampilan, jumlah sumber daya, serta unggahan atau unduhan, serta tindakan pengguna lain. Dasbor juga menawarkan pratinjau dan pemberitahuan sumber daya yang baru ditambahkan.
  • Pencarian: memberi pengguna platform yang luar biasa untuk mencari konten menggunakan kata kunci, tanggal unduh, jenis file, metadata, atau menggunakan kotak centang untuk menyaring sumber daya. Fitur pencarian juga memungkinkan Anda untuk mengotomatiskan pemberian tag menggunakan nilai metadata yang terkontrol dan sumber daya target melalui koleksi dan pencarian.
  • Berbagi Sumber Daya: Fitur ini memfasilitasi pembagian sumber daya dalam beberapa cara. Cara termasuk kode sematan, email, bagikan melalui jejaring sosial atau tautan unduhan langsung. Fitur berbagi sumber daya juga memudahkan berbagi sumber daya melalui halaman unduhan dengan pratinjau.
  • Resource Versioning: Menyediakan alat untuk mempublikasikan konten online. Alat memungkinkan Anda untuk memotong dan mengubah ukuran sumber daya digital secara otomatis dan menempelkannya ke halaman web mana pun, mendapatkan gagasan tentang bagaimana sumber daya Anda digunakan menggunakan peta lokasi geografis dan URL dengan kode yang disematkan.
  • Manajemen nama file: menyediakan platform yang sangat baik untuk mengidentifikasi file duplikat, menyimpan nilai metadata untuk setiap versi sumber daya, dan kontrol versi sumber daya menggunakan kode bawaan.
  • Peran dan manajemen izin : memungkinkan Anda untuk mengontrol akses ke sumber daya digital Anda dengan mengatur peran pengguna dan tingkat izin.

Manfaat Menggunakan Perangkat Lunak Manajemen Aset Data


  • Meningkatkan kolaborasi dan akses ke sumber daya kritis: menyediakan alat untuk memfasilitasi interaksi antara pengguna. Ini juga menyederhanakan akses ke sumber daya kritis, karena semua pengguna dengan izin memiliki kemampuan untuk langsung mengunduh materi berharga segera setelah tersedia.
  • Menghilangkan redundansi alur kerja: program ini memberi Anda kontrol pusat, yang membantu menyederhanakan komunikasi dan memastikan bahwa semua anggota tim berada di halaman yang sama pada waktu yang bersamaan. Berkat ini, perangkat lunak mengurangi inefisiensi dan secara signifikan mengurangi kemungkinan duplikasi alur kerja yang mahal.
  • Mempercepat siklus kreatif dan produksi. Alat yang disediakan oleh perangkat lunak ini membantu tim Anda dengan cepat membuat dan menggunakan kembali sumber daya digital, serta memberikan alur kerja dan persetujuan yang lebih halus. Perangkat lunak ini juga memberi Anda keunggulan kompetitif dalam siklus produksi Anda di atas perusahaan lain.
  • Menghilangkan biaya pekerjaan yang hilang atau tidak pantas. Salah satu masalah utama ketika bekerja dengan sumber daya digital adalah kerugian mereka. Ini adalah hal yang biasa, karena menjadi lebih mudah untuk merancang sumber daya digital berkualitas tinggi. Dengan sistem ini, perusahaan dalam industri kreatif memiliki platform yang sangat baik untuk menghilangkan biaya yang terkait dengan hilangnya sumber daya yang berharga, karena fakta bahwa mereka mudah ditemukan.

Tren Aset Data


  • Inteligensi Buatan: AI menangani semua aspek bisnis. Dalam perangkat lunak ini, kecerdasan buatan akan menggantikan tenaga kerja manual karyawan. Teknologi yang disediakan oleh kecerdasan buatan akan memungkinkan perangkat lunak untuk secara otomatis memindai konten dan membuat tag metadata yang sesuai. Dengan demikian, ini akan memungkinkan bisnis untuk dengan cepat menemukan apa yang dibutuhkannya, menghemat waktu mengatur sumber daya.
  • Otomatisasi dan penggunaan algoritme pembelajaran mesin: ini akan memungkinkan perangkat lunak manajemen sumber daya digital untuk merekomendasikan tugas-tugas tertentu dalam suatu tindakan atau kampanye. Algoritma pembelajaran mesin juga akan memungkinkan perangkat lunak untuk mengekstrak sumber daya yang lebih tepat dari sistem, mengambil pelajaran dari data dan pola penggunaan.
  • Manajemen metadata: berkat teknologi terbaru, perangkat lunak manajemen sumber daya digital akan dapat memprediksi hubungan antara sumber daya digital dan, oleh karena itu, akan memungkinkan perangkat lunak untuk secara otomatis menyarankan opsi metadata.

Kriteria pemilihan


  • Perangkat Lunak sebagai Layanan (SaaS): Anda mungkin memiliki komputer yang tidak memiliki cukup ruang untuk menyediakan ruang bagi file yang terus bertambah. Oleh karena itu, pertimbangkan perangkat lunak yang memungkinkan Anda untuk menyimpan file secara online, bukan secara lokal (SaaS). Perangkat lunak biasanya memberi Anda penyimpanan cloud.
  • Kolaborasi dan berbagi. Perangkat lunak yang Anda pilih harus memungkinkan pengguna untuk melihat dan mengomentari proyek Anda. Mengizinkan pengguna melihat dokumen Anda dan memberikan komentar pada mereka sangat penting untuk kolaborasi dan peningkatan.
  • Fungsi yang mudah: Pilihan ideal Anda untuk program ini harus ramah-pengguna, karena ini berarti akan mudah bagi Anda untuk mengajari orang lain cara menggunakan perangkat lunak dengan mudah dan efektif. Perangkat lunak integrasi pihak ketiga juga membuat hidup Anda lebih mudah.
  • Sistem pendukung ketersediaan. Perangkat lunak yang Anda pilih harus memiliki layanan dukungan. Help desk sangat ideal untuk menjawab pertanyaan mendesak pengguna Anda. Layanan dukungan juga menawarkan pelatihan khusus bagi karyawan Anda untuk memperluas pengetahuan mereka tentang penggunaan perangkat lunak.

Kasus perusahaan arsitektur


Organisasi arsip digital perusahaan arsitektur

Masalah:

  • 120.000 foto disimpan di server file dan diatur dalam folder
  • Menemukan file-file yang diperlukan untuk proyek-proyek pemasaran membutuhkan banyak waktu
  • Pemasar mencari file dengan mengklik folder. Beberapa file diberikan kata kunci dalam Windows Explorer, tetapi pencarian tag di Windows Explorer tidak efektif
  • Pemasar menyalin file dari repositori bersama ke diri mereka sendiri di PC agar tidak mencari dari berulang kali, sehingga membuat duplikat dan mengacaukan PC mereka

Tantangan:

  • Buat lokasi terpusat untuk menyimpan dan mencari file
  • Temukan perangkat lunak yang dapat mengatur arsip secara efektif dengan menetapkan berbagai tag (kata kunci, tempat, proyek, dll.)
  • Perangkat lunak harus mengenali tag yang sudah ada dalam file
  • Impor file dengan tautan (= jangan menyimpan file asli dalam program), agar tidak membuat duplikat dan tidak mengalokasikan ruang penyimpanan tambahan untuk file 120K +
  • Kenali File Duplikat yang Ada
  • Akses multi-pengguna

Solusi: Daminion

  • Mengimpor file dengan tautan, menjaga struktur file. Jika perlu, selama atau setelah impor, file dapat ditata ulang - diganti namanya, dipindahkan ke folder lain, buat hierarki folder baru, misalnya, mengatur file berdasarkan tahun, bulan dan hari.
  • Program ini memiliki sekitar 50 tag default untuk mengatur arsip (kata kunci, kategori, lokasi, proyek, hak cipta, nama, deskripsi, dll.), Tetapi juga memungkinkan untuk membuat tag khusus untuk organisasi yang lebih terperinci. Misalnya, kami membuat tag Jenis Proyek, Jenis Bangunan, dan Sektor Bangunan khusus.



    Manajemen Tag di daminion.net
  • Program mengenali tag yang sudah terkandung dalam file. Sejak hari pertama, Anda dapat menggunakan tag ini untuk mencari file, melanjutkan proses penandaan arsip. Sangat mudah untuk mencari file berdasarkan tanggal pembuatan, karena ditarik dari metadata file. Jika pemasar membutuhkan file dalam resolusi tertentu atau ukuran tertentu, sangat mudah menemukannya di Daminion (misalnya, cari foto apartemen yang lebih besar dari 1000 Kb)
  • Akses multi-pengguna - sementara beberapa orang menandai arsip, yang lain dapat masuk ke direktori dan mencari file
  • Mengenali file duplikat
  • Sepenuhnya Terikat. Jika diinginkan, Anda dapat mengubah bahasa ke bahasa Rusia
