GeekBrains começa a preparar desenvolvedores de Python full-stack


Nosso portal educacional GeekBrains anuncia o lançamento de um novo curso de treinamento para desenvolvedores web Python (Django) . Um curso avançado com um estudo aprofundado da estrutura do Django permitirá que você se candidate à posição de desenvolvedor Full-stack - um especialista com um grande conjunto de competências que pode criar um aplicativo Web a partir do zero.


O programa de treinamento com duração de 10 meses consiste em 11 cursos. Você começará com conceitos básicos, aprenderá o básico do Python e bancos de dados e depois passará para tópicos mais complexos, dominá-los o ajudará a resolver com êxito os problemas enfrentados pelos desenvolvedores do Full-stack.


Você dominará as ferramentas Front-end para o desenvolvimento de HTML, CSS e JavaScript, bem como a estrutura moderna para o desenvolvimento Back-end do Django, que o ajudará a escrever código mais rapidamente e a traduzir a idéia em um projeto de TI em funcionamento. Durante dois cursos interativos - básico e avançado - você receberá um profundo conhecimento do Django e poderá colocá-los em prática imediatamente.


As lições são ensinadas pela prática de programadores Python com educação especializada e experiência de trabalho na especialidade de 5 anos. Aulas no formato de webinars várias vezes por semana à noite. Para consolidar habilidades, após cada lição, é feita a lição de casa, com a qual um mentor pessoal dentre os melhores graduados ajudará a lidar. Ao final do treinamento, cada aluno desenvolverá independentemente uma loja on-line completa com funções de registro e autorização, que será o primeiro projeto de um portfólio.


Após a conclusão dos estudos, todos são garantidos dois meses de estágio, durante os quais você participará do desenvolvimento da equipe do projeto, um tradutor de sites com funcionalidade avançada, sob a orientação do líder da equipe. Em caso de conclusão bem sucedida da tarefa de teste, os melhores alunos poderão começar a trabalhar com os parceiros do GeekBrains ou ingressar em uma startup fundada por formandos de cursos anteriores.


As aulas começarão no dia 24 de dezembro. A mensalidade é paga, você deve se registrar para participar.
