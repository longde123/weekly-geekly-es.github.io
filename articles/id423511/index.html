Mencegah instalasi Firefox dan Chrome di Windows 10


Tangkapan layar thurrott.com

Insider build Windows 10 v.1809 memiliki fitur baru . Ketika pengguna mencoba menginstal Firefox atau Chrome, sistem sekarang menampilkan pesan bahwa Microsoft Edge sudah ada di komputer - dan menawarkan untuk meluncurkan "browser yang lebih cepat dan lebih aman" ini.

OS memotong instalasi perangkat lunak alternatif, yang tidak disetujui untuk instalasi, dan mencoba untuk mencegah pengguna dari tindakan yang tidak diinginkan.

Selain itu, pengaturan untuk aplikasi telah berubah dalam versi terbaru. Sebelumnya, secara default, instalasi semua program diizinkan - Bolehkan aplikasi dari mana saja (Default) . Pilihan lain yang ditawarkan adalah "Peringatkan saya tentang menginstal aplikasi bukan dari Store" dan "Izinkan aplikasi diinstal hanya dari Store."

Pengaturan default baru adalah Tunjukkan rekomendasi aplikasi . Pilihan lain adalah "Nonaktifkan rekomendasi untuk aplikasi", "Peringatkan tentang menginstal aplikasi bukan dari Store" dan "Izinkan aplikasi diinstal hanya dari Store".


Tangkapan layar thurrott.com

Rupanya, instalasi Firefox dan Chrome di Windows 10 dicegat sesuai dengan pengaturan default baru di bagian Aplikasi & fitur , yaitu, "Tampilkan rekomendasi untuk aplikasi." Rekomendasi di Windows 10 termasuk peringatan tentang menginstal browser "yang tidak perlu" seperti Firefox dan Chrome.

Kemunculan fungsi ini menyebabkan kemarahan di antara beberapa pengguna. Banyak yang masih ingat tawaran menjengkelkan untuk menginstal Windows 10, yang ditampilkan di Windows 7. Menolak tawaran ini adalah tugas yang tidak sepele, jadi kadang-kadang instalasi Windows 10 dimulai melawan kehendak pengguna.

Menurut komentator, tindakan tersebut menunjukkan peningkatan perjuangan di pasar browser dan keinginan Microsoft untuk mempertahankan / meningkatkan pangsa pasarnya. Sekarang Chrome mendominasi pasar ini, termasuk karena kebijakan agresif Google dalam mengiklankan browser ini di situsnya. Banyak pengguna Firefox dan Edge pasti menemukan penawaran untuk memasang Chrome di situs yang dimiliki oleh Google. Ada metode lain: misalnya, desain ulang YouTube baru-baru ini menggunakan API Shadow DOM v0 yang eksotis, yang hanya diterapkan di Chrome. Akibatnya, halaman YouTube memuat lebih cepat di Chrome daripada di browser lain .


Selain itu, Chrome dapat "secara tidak sengaja" menjadi peramban default selama pemasangan beberapa utilitas gratis yang tidak tersedia. Praktik ini begitu meluas sehingga bahkan ada alat khusus yang menghapus semua tanda centang yang tidak perlu selama instalasi program apa pun.

Apple juga mencoba menawarkan pengguna Safari browser MacOS. Tidak seperti pesaing, Firefox tidak maju dengan cara seperti itu: Mozilla tidak memiliki platform untuk promosi semacam itu.

Belum diketahui apakah fungsi baru dengan intersepsi pengaturan browser alternatif akan memasuki pembaruan besar Windows 10. Oktober. Jika ini terjadi, tindakan tersebut dapat menyebabkan oposisi dari komite antimonopoli Uni Eropa. Berdasarkan pengalaman proses antimonopoli sebelumnya di UE, Microsoft dapat menargetkan tampilan rekomendasi browser Edge tersebut kepada pengguna di luar UE untuk menghindari masalah.

"Pedoman aplikasi" pada Windows 10 mungkin sebenarnya bermanfaat bagi beberapa pengguna. Misalnya, banyak yang tidak tahu bahwa tidak perlu menginstal Acrobat Reader untuk melihat file PDF dan, dengan demikian, menambah kerentanan potensial baru ke sistem. Artinya, Microsoft secara wajar mengaktifkan opsi ini secara default. Satu-satunya pertanyaan adalah isi dari rekomendasi itu sendiri.
