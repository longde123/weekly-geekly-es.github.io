Laut Microsoft wird der Upgrade-Prozess auf Windows 10 vollständig von den Benutzern gesteuert



Trotz der Empörung der Benutzer über die Tatsache, dass ihre Computer auf Windows 10 aktualisiert werden, sagt Microsoft, dass der Win10-Aktualisierungsmechanismus vollständig von den Benutzern gesteuert wird, sobald sie sich abwenden.

Microsoft gab die folgende Erklärung zu „ungeplanten“ Updates ab:

„Wir möchten, dass Windows 7 und Windows 8.1 so einfach wie möglich auf Windows 10 aktualisiert werden. Benutzer können Aktualisierungszeiten planen, dh sie haben die vollständige Kontrolle über ihre Geräte. Es ist auch möglich, Windows 10 nicht zu installieren, indem Sie das Update aus Windows Update (WU) entfernen, indem Sie die Einstellungen ändern “, sagt hothardware.com.

Der Kotaku- Reporter Patrick Klipik ist einer von vielen , die gegen ihren Willen Windows 7 gegen seinen Willen auf Windows 10 aktualisiert haben :

„Ich habe Windows 7 mehrere Jahre lang verwendet, um die Installation von Windows 8 zu vermeiden, weil mir alle gesagt haben, dass dies nur der Fall ist Probleme. Windows 7 war ein praktisches Betriebssystem. Ich wusste was und wo es lag. Trotzdem verstehe ich, dass Updates wichtig und notwendig sind, insbesondere im Hinblick auf die Sicherheit. Ich bin in der Regel die Person, die Updates so schnell wie möglich „rollt“. Aber der Computer ist ein großer Teil meines Lebens. Einige PC-Probleme sind schwer zu beheben, und ein neues Betriebssystem könnte ein solches Problem sein.

Letzte Woche habe ich darüber nachgedacht, nicht länger ein „alter Dinosaurier“ zu sein und auf Windows 10 zu aktualisieren. Was ich nicht erwartet hatte, war am Donnerstagmorgen aufzuwachen und herauszufinden, dass Windows sich um mich gekümmert hat (mit dem Verlust aller meiner Einstellungen). Vielleicht habe ich vergessen, das Update zu verschieben, vielleicht habe ich es versehentlich geplant, indem ich auf die falsche Schaltfläche geklickt habe. Auf jeden Fall hatte ich definitiv nicht vor, Windows 10 zu installieren. "

Reddit ist immer noch voller Nachrichten von verärgerten Benutzern. Zum Beispiel schreibt MalekuaMan: "Ich verlasse mein Zuhause für zwei Tage und bei meiner Rückkehr stelle ich fest, dass ich jetzt Windows 10 habe. Was für ein *****." Ein anderer Benutzer unter dem Spitznamen rtn1797 litt ebenfalls unter Microsofts "Besorgnis": "Mein Win7-Computer wurde vor ungefähr zwei Tagen ohne meine Erlaubnis auf Win10 aktualisiert. Ich habe das Update zurückgesetzt, aber jetzt gibt es einen Hinweis, der ständig geöffnet ist und mich auffordert, den Zeitpunkt für das Upgrade auf Windows 10 zu wählen." ... WTF ... wie kann ich das loswerden? "

Niemand sagt, dass Microsoft versucht, dieses Problem herunterzuspielen, aber warum haben die Computer all dieser Leute unter ähnlichen Umständen gleichzeitig auf Windows 10 aktualisiert? Microsoft sollte eine offizielle Erklärung veröffentlichen, die Licht in dieses Problem bringt.
