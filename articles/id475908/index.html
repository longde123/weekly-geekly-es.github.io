10 kursus Microsoft paling populer di Rusia

Halo, Habr! Baru-baru ini, kami memposting bagian pertama dari serangkaian koleksi kursus pelatihan yang berguna untuk programmer. Dan bagian kelima terakhir telah nampak merayap. Di dalamnya, kami telah mendaftarkan beberapa program TI paling populer yang tersedia di platform pembelajaran Microsoft Learn. Semuanya tentu saja gratis. Detail dan tautan ke kursus di bawah cut!

Topik kursus dalam koleksi ini:

  • Python
  • Xamarin
  • Kode studio visual
  • Microsoft 365
  • Kekuatan BI
  • Azure
  • ML

1. Pengantar Python


Pelajari cara menulis kode Python sederhana, mendeklarasikan variabel, dan menggunakan input dan output konsol

Dalam modul ini Anda harus:

  • mempertimbangkan opsi untuk menjalankan aplikasi Python;
  • Gunakan juru bahasa Python untuk menjalankan pernyataan dan skrip
  • belajar mendeklarasikan variabel;
  • buat aplikasi Python sederhana yang menerima input dan menciptakan output.

Anda dapat memulai pelatihan di sini

2. Buat aplikasi seluler menggunakan Xamarin.Forms


Kursus ini sudah sepenuhnya atau hampir sepenuhnya mencakup seluruh fungsi alat dan dirancang untuk 10 jam pelatihan. Dia akan mengajari Anda cara bekerja dengan Xamarin. Bentuk dan menggunakan C # dan Visual Studio untuk membuat aplikasi yang berjalan di perangkat iOS dan Android. Oleh karena itu, untuk mulai belajar, Anda harus memiliki Visual Studio 2019 dan memiliki keterampilan untuk bekerja dengan C # dan .NET.

Modul Kursus:

  1. Membuat aplikasi seluler menggunakan Xamarin.Forms;
  2. Pengantar Xamarin. Android;
  3. Pengantar Xamarin.iOS;
  4. Membuat antarmuka pengguna di Xamarin. Aplikasi menggunakan XAML;
  5. Kustomisasi tata letak pada halaman XAML di Xamarin.Forms;
  6. Merancang XamarL Xamarin yang konsisten. Halaman menggunakan sumber daya dan gaya bersama;
  7. Mempersiapkan aplikasi Xamarin untuk publikasi;
  8. Menggunakan layanan web REST dalam aplikasi Xamarin;
  9. Menyimpan data lokal menggunakan SQLite di aplikasi Xamarin.Forms;
  10. Buat multi-halaman Xamarin. Aplikasi dengan navigasi tumpukan dan tab.

Mulai pelatihan

3. Pengembangan Aplikasi Menggunakan Visual Studio Code


Pelajari cara mengembangkan aplikasi menggunakan Visual Studio Code dan menggunakan kemampuan lingkungan ini untuk membuat dan menguji aplikasi web yang sangat sederhana.

Dalam modul ini Anda akan belajar cara melakukan tugas-tugas berikut:

  • Pelajari fitur dasar Kode Visual Studio;
  • Unduh dan pasang Kode Visual Studio;
  • Pasang ekstensi untuk pengembangan web dasar
  • menggunakan fungsi dasar editor Visual Studio Code;
  • Bangun dan uji aplikasi web sederhana.

Mulai pelatihan

4. Microsoft 365: Meningkatkan Penggunaan Perusahaan Menggunakan Windows 10 dan Office 365


Microsoft 365 membantu menciptakan lingkungan yang aman dan dapat diperbarui dengan menggunakan perangkat Windows 10 yang memiliki aplikasi Office 365 dan dikelola oleh Microsoft Enterprise Mobility + Security.

Modul ini, dirancang selama 3,5 jam, akan mengajarkan Anda cara bekerja dengan Microsoft 365, berbicara tentang dasar-dasar penggunaan alat, serta tentang keamanan dan pelatihan pengguna.

Mulai pelatihan

5. Buat dan bagikan laporan Power BI pertama


Dengan Power BI, Anda dapat membuat visual dan laporan yang mengesankan. Dalam modul ini, Anda akan belajar cara menggunakan Power BI Desktop untuk terhubung ke data, membuat elemen visual, dan laporan yang dapat Anda sediakan untuk digunakan oleh pengguna lain di organisasi Anda. Anda kemudian akan belajar bagaimana menerbitkan laporan ke Power BI dan memungkinkan orang lain untuk melihat informasi analitik Anda yang mungkin berguna dalam pekerjaan Anda.

Dalam modul ini Anda akan belajar cara melakukan tugas-tugas berikut:

  • Buat laporan di Power BI
  • Bagikan laporan dalam Power BI.

Mulai pelatihan

6. Membuat dan menggunakan laporan analitik di Power BI


Skema ini akan memperkenalkan Anda ke Power BI dalam 6-7 jam pelatihan dan mengajarkan Anda cara menggunakan dan membuat laporan intelijen bisnis. Untuk memulai studi, Anda hanya perlu memiliki pengalaman dengan Excel, mengakses Power Bi dan mengunduh aplikasi.

Modul:

  • Memulai dengan Power BI;
  • Mengambil data menggunakan Power BI Desktop;
  • Pemodelan data dalam Power BI;
  • Penggunaan elemen visual dalam Power BI;
  • Mempelajari data dalam Power BI;
  • Publikasikan dan bagikan dalam Power BI.

Mulai pelatihan

7. Dasar-Dasar Azure


Apakah Anda tertarik dengan cloud, tetapi tidak tahu manfaat apa yang bisa didapatnya bagi Anda? Anda harus mulai dengan skema pelatihan ini.

Topik-topik berikut tercakup dalam skema pelatihan ini:

  • Konsep kunci yang terkait dengan komputasi awan: ketersediaan tinggi, skalabilitas, elastisitas, fleksibilitas, ketahanan, dan pemulihan bencana;
  • Keuntungan dari komputasi awan di Azure: bagaimana cara menghemat waktu dan uang dengan bantuan mereka;
  • Perbandingan dan perbandingan strategi utama untuk transisi ke layanan cloud Azure;
  • Layanan yang tersedia di Azure, termasuk Layanan Komputer, Layanan Jaringan, Penyimpanan dan Keamanan.

Dengan menyelesaikan skema pelatihan ini, Anda akan menerima pengetahuan yang diperlukan untuk lulus ujian AZ900 Microsoft Azure Fundamentals.

Mulai pelatihan

8. Manajemen sumber daya di Azure


Hanya dalam 4-5 jam, pelajari cara membuat, mengelola, dan mengelola sumber daya cloud menggunakan baris perintah Azure dan portal web.

Modul untuk kursus ini:

  • Menyesuaikan persyaratan dengan tipe cloud dan model layanan di Azure
  • Kelola Layanan Azure menggunakan CLI
  • Mengotomatiskan tugas Azure dengan skrip PowerShell;
  • Perkiraan biaya dan optimalisasi biaya untuk Azure;
  • Mengontrol dan mengatur sumber daya Azure dengan Manajer Sumber Daya Azure.

Mulai pelatihan

9. Layanan Cloud Dasar - Pengantar Azure


Untuk memulai dengan Azure, Anda perlu membuat dan mengonfigurasi situs web pertama di cloud.

Dalam modul ini Anda akan belajar cara melakukan tugas-tugas berikut:

  • Keakraban dengan informasi tentang platform Microsoft Azure dan hubungannya dengan komputasi awan;
  • Menyebarkan situs web di Layanan Aplikasi Azure
  • Meningkatkan situs web untuk sumber daya komputasi tambahan;
  • Gunakan Azure Cloud Shell untuk berinteraksi dengan situs web.

Mulai pelatihan

10. Layanan Pembelajaran Mesin Azure


Azure menawarkan berbagai layanan untuk mengembangkan dan menggunakan model pembelajaran mesin. Pelajari cara menggunakan layanan ini untuk menganalisis data.

Modul untuk kursus ini:

  • Pengantar Pembelajaran Mesin Azure
  • Belajar model pembelajaran mesin lokal menggunakan Azure Machine Learning;
  • Mengotomatiskan pemilihan model pembelajaran mesin menggunakan layanan pembelajaran mesin Azure;
  • Daftarkan dan gunakan model ML dengan Azure Machine Learning.
