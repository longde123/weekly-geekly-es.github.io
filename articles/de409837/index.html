Smart Lampe Selbstleuchte Mini

Ein ziemlich einfaches Gerät, eine intelligente Lampe, ist an Orten und in Situationen wie einem dunklen Schrank, einem Wohnungskorridor, einem Lagerraum oder etwas anderem unverzichtbar. Diese Lampe kann in zwei Modi arbeiten - Dauerlicht und Auto. Im ersten Fall ist alles klar, im zweiten Fall sind zwei Bedingungen erforderlich, damit die Beleuchtung aufleuchtet:

  • Unzureichendes Umgebungslicht;
  • Bewegung im Abdeckungsbereich der Lampe.

Sobald die Bewegung stoppt und 30 Sekunden vergehen, erlischt die Lampe. Ein solches System ist nützlich für eine Sommerresidenz, eine Garage, eine Wohnung oder ein Haus. Das System wird mit drei AAA-Batterien betrieben.

Rabatt für die Lampe - 25%.

Das Gehäuse ist gemäß dem IPX5-Standard geschützt, der das Eindringen von Feuchtigkeit, Staub, Schmutz und Sand in das Gerät verhindert. Der Lautsprecher kann auch als Ladestation für ein Smartphone oder Tablet verwendet werden. Nach dem Aufladen des Telefons kann der Lautsprecher noch einige Stunden spielen. Falls gewünscht, kann die Säule als Lautsprecher zum Telefonieren verwendet werden.

Intelligente Waage Picooc Mini



Hierbei handelt es sich um hochpräzise Miniaturwaagen, die in nur drei Sekunden nicht nur das Gewicht, sondern auch den Body-Mass-Index, den Wasseranteil und andere biometrische Parameter bestimmen. Und dank eines empfindlichen Sensors „bemerkt“ das System die kleinsten Veränderungen im Körper.

Die Gesamtzahl der biometrischen Parameter, die das Gerät ändern kann, beträgt 12. Hier ist der Prozentsatz an Protein, Körperfett, Knochenmasse, Body-Mass-Index, Körpertyp, Muskelmasse und mehr. Das System sendet alle gesammelten Daten an die russischsprachige Anwendung Picooc, die im Anwendungskatalog für Android und iOS zum Download zur Verfügung steht. Die Synchronisation erfolgt über das eingebaute Bluetooth-Modul.

Alle Informationen werden sofort nach den Messungen an ein Mobiltelefon gesendet. Das Gerät speichert eine unbegrenzte Anzahl von Benutzern. Batterien - drei AAA-Batterien.

Rabatt auf Picooc Mini Waagen - 20%.

Wetterstation Netatmo




Wetterstation Netatmo - sagt nicht nur das Wetter an, sondern spricht auch über die Qualität der Luft, die wir atmen. Das System verfügt über mehrere Sensoren, die auf zwei Teile der Wetterstation verteilt sind. Der erste Teil wird drinnen installiert, der zweite - außerhalb des Fensters.

Das Anzeigen von Wetter- und Luftqualitätsdaten ist mit einem Smartphone möglich. Alle Daten werden auf dem Bildschirm des Mobilgeräts angezeigt, Informationen werden drahtlos übertragen. Zunächst werden sie an den Server gesendet und von dort auf das Smartphone heruntergeladen. Auf Wunsch können Sie drei zusätzliche Raumsensoren installieren - falls das Mikroklima eines großen Raums überwacht werden muss.

Die Wetterstation überwacht die folgenden Parameter - Geräuschpegel, Kohlendioxid, Luftfeuchtigkeit und Raumluftqualität. Und natürlich die Wettervorhersage für die kommende Woche. Eine Übersicht über die Station finden Sie hier .
