这是针对技术专业的初学者和学生的一系列活动。 我们谈论的是八月底，九月和十月的计划。

（c）ITMO大学

最新消息


  • 2019年招生活动结果

今年夏天，我们在哈布雷（Habré）博客上讨论了ITMO大学的课程，并分享了其毕业生职业发展的经验。 这些材料为今年的招生活动做出了贡献-超过4000人申请了治安管理官，还有超过1200人聚集在一起开始攻读本科和专业课程。

今年，来自俄罗斯不同城市的其他大学（不仅是ITMO大学）的更多毕业生进入了我们的治安管理官，在教育过程的初始阶段（本科），我们注意到平均考试分数提高到92.7。 比去年增加了2.5点。 但是，许多奥林匹克竞赛和比赛的获胜者（例如我们的ITMO.STARS ）设法规避了通过入学考试的需要，而且-从培训的第一天起，就有700多人获得了奖学金 。

  • Gennady Korotkevich在Google Code Jam上的第六次胜利

根纳季（Nennady）是我们杰出的研究生之一，曾两次获得ICPC冠军，再次成为数以万计的Google Code Jam参与者中的佼佼者。 ITMO大学的Boris Minaev和Evgeny Kap的毕业生与他一起进入了决赛 。

  • 我们在上海大学学科排名中名列前100名

这项研究是根据该评估结果而编制的，全球不同地区的四千多所大学参加了该研究。 该方法考虑了由Web of Science数据库编制索引的期刊中的科学出版物。 对于排名排名靠前的大学，科学论文涉及五十四个主题领域：从“电子和电气工程”到“自动化与控制”。 我们设法提高了我们在所介绍的知识领域中的地位，并在其他领域保持了很高的成绩。 在大多数学科领域中大学排名最高的国家中，美国和中国仍然是领导者。

专题活动


1.节日“零九月”

时间： 8月30日
什么时间：从15:00开始
地点： “ Rzhevskaya”市图书馆1号楼工业大道d.35
注册：必填（您需要通过引用填写表格 ）

我们为孩子，他们的父母和老师准备了一个特别活动。 这是一个主题节日，在该节日的框架内将举行一系列教育竞赛，测验和研讨会，设计局，机器人实验室，能源企业的信息中心和年轻的创新公司的代表将参加。

2.奉献给学生“ ITMO.GO！”

时间： 8月31日
时间：从13:00到15:00
地点：波罗的海剧院前广场四号亚历山大公园

这是我们准备参加ITMO大学社区的所有人员的主要活动-一年级学生和已从其他大学进入我们的研究生和研究生课程的人。 我们将举行露天活动，讨论学生项目，开放运动场并为“ ITMO.GO！”参与者颁发奖品。 并提供主题信息部分的访问：从“硕士研究”到“ ITMO INTERNATIONAL”和“ ALUMNI”，在这里可以讨论我们的毕业生的教育过程和职业生涯。


3.一年级本科生和研究生的“入学日”

时间： 9月2-3日
什么时间和地点： 单身汉 ， 硕士

特殊情况介绍会的目的是提供有关我们在ITMO大学提供的机会的最大信息。 为了使这样的事件真正有趣，我们在我们大学的“站点”（“ ITMO.LOVE”，“ Teambuilding”，“ Science”，“ Global”等）以交互式“问题”的形式进行了该活动。

对于一年级的本科生，我们将讨论教育过程的复杂性以及如何使其最有效。 此外，我们还为我们的科学家准备了真实的推销课程，这将帮助您确定职业发展需要哪些感兴趣的领域，并告诉您如何成为国际实验室团队的一员并进行实习。

4.关于我们的国际计划

时间： 9月6日和9月20日
注册：必填（在上面的链接上）

它将讨论学期和学术交流的可能性以及各种实习形式。 ITMO大学的专家和我们来自不同国家的合作伙伴将讨论共同资助此类旅行和培训计划的最有效选择。
