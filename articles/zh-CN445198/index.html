适用于Linux的Cmus控制台播放器

大家好

目前，我使用cmus控制台播放器，使用起来非常方便。 有鉴于此，我想写一篇简短的评论。



在新的工作站上，我终于搬到了Linux。 在这方面，需要找到适合工作相关需求的软件。 尽管有足够的linux接口播放器，但它们都有些堆积而且不是很漂亮（我的观点可能与读者的观点不一致，因为它是恕我直言）。 搜索参数中包括以下条件：

  • 简单性
  • 便利性
  • 低内存消耗
  • 播放和扫描子文件夹的能力
  • 普通玩家固有的相关包子

在此过程中，我遇到了cmus。 播放器...控制台...为什么不呢，我想到并执行了许多人的常用命令： 

sudo apt-get install cmus

第一次运行后，我们将看到以下内容：



为了将音乐添加到播放列表，只需执行命令。 在vi中按“：”并输入： 

 add ~/Music

作为此命令的结果，您将看到第一个屏幕快照中的子文件夹和轨道的列表。

要浏览该程序，可以使用箭头键，Tab（在区域之间切换），空格（以展开子文件夹），Delete（从列表中删除曲目）。 还有大量的热键。 要更改播放器中的窗口，请使用数字键从1到7。在第7个窗口中，您可以自己重新配置热键。

您当然可以通过执行以下命令来了解有关此内容的更多信息： 

 man cmus

在播放模式下，播放器“吃”了大约10 mb的RAM，我认为这是一个非常适中的大小。
官方网站声称支持更多更令人愉悦的功能（例如声音s草）以及附加组件（连接多媒体按键，从spotify或youtube下载音乐）。

总而言之，您可以放心地说，此播放器非常方便且灵活。

我同意这并不适合所有人，而只适用于控制台极客=）

为此，我请假。
