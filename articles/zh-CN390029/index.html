Torrents时间：Popcorn Time作为浏览器插件。海盗湾已经支持



Popcorn Time服务可以通过浏览器插件使用，而无需预先下载即可查看来自种子的视频文件。在特殊站点上支持开源插件，可在线观看Popcorn Time Online视频。

该插件适用于Internet Explorer，Chrome和Firefox OS Windows 7（及更高版本）和OS X 10.8（及更高版本）。开发人员承诺将在几周内增加对Safari的支持。Edge可能不会被忽略。

此外，最老旧，最受欢迎的追踪器之一，即沉没的海盗湾，也在其搜索引擎中迅速组织了插件支持。

背景知识


Popcorn Time最初是作为使用内置媒体播放器下载torrent文件的客户端而出现的。由于大多数用户对带有视频文件（电影和系列剧集）的种子感兴趣，因此期待此类软件的出现是很合逻辑的。

在同一流行的uTorrent中，支持下载洪流过程中的视频流已经实现了相当长的一段时间-但是要查看它，需要安装和配置单独的媒体播放器。在这里，所需的一切都“开箱即用”提供。

自然，这种方便的事情不会引起版权持有者的注意，他们遭受了不止一次噩梦，试图消除对他们收入的威胁。一连串的版权持有人MPAA（美国电影协会）在2014年以其残酷而闻名，设法关闭了该服务的原始版本。

但是，由于该程序的源代码可供所有人使用，因此该服务的版本会以神话般的九头蛇的头目一次又一次地周期性出现。上一个受欢迎的服务Popcorntime.io于2015年11月关闭。

来自Popcorn-Time.se域的一种“爆米花”变体的开发人员已经成长并完成了该应用程序的开发，使其支持Windows，OS X，Linux，Android，iOS，内置VPN支持，甚至包括Chromecast和Apple TV。

插件：启用并查看


但是现在，版权所有者面临着一场噩梦，他们可能无法从中醒来。Popcorn-Time.se不再提供下载和安装应用程序的权限-只需安装Torrents Time浏览器的插件，您就可以在页面上从torrent观看任何视频文件。

同时，Torrents Time插件代码可在GitHub上获得，这意味着任何人都可以帮助系统的定稿或将其安装在其主机上。

该插件可用于Windows 7（及更高版本）和OS X 10.8（及更高版本）的Internet Explorer，Chrome和Firefox。它支持Chromecast，Airplay和DLNA，并具有内置的VPN支持。开发人员提供付费服务Anonymous VPN，但没有人愿意使用其他任何服务。

海盗湾


通常，河流流入大海，但是通过进入海盗湾The Pirate Bay，您可以找到从那里流出的溪流。现在，对这些流的访问已简化为不可能- 站点管理员增加了对Torrents Time插件的支持。



现在，在接收磁铁的链接旁边，出现了“ Stream It！”链接，单击该链接可以下载该插件，然后启动它并开始观看电影。

一个简单但功能齐全的播放器支持Chromecast，Airplay和DNLA设备的字幕和投射。



其他出色的网站，例如Torrentproject.se，Videomax.is，Online.porntime.ws和KickassTorrents，宣布很快将在其页面上使用此插件。

只知道爆米花时间和种子时间是由独立的程序员组开发的。正如开发人员自己写的那样：“坚持匿名性，我们并没有躲过垄断内容所有者的胆小鬼。我们争取内容的免费分发。我们尊重法律，但是我们不相信执法机构执行法律。”
