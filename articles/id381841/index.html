Kartu itu sendiri lahir pada tahun 2000. Memori video ( 64 meter ) menarik di sini: setiap chip diberikan 32 meter untuk penggunaan eksklusif . Yaitu, ada 32 meter memori efektif :) Dan, omong-omong, Everest menegaskan ini!
Pada saat penulisan, saya benar-benar yakin akan hal ini, namun, pengujian dan studi mendalam tentang masalah ini menunjukkan:

We would like to mention that each of VSA-100 chips used in SLI has its own frame buffer for rendering lines and line successions (bands). After that the data from all these frame buffers is submitted to the general frame buffer where the complete frame is created and transferred to the monitor. Moreover, each VSA-100 chip is equipped with its own memory bus and local frame buffer and supports up to 64MB local graphics memory. A part of the available memory is used for a framebuffer (with double of even triple buffering) and the rest â€” for storing textures and Z-buffering. In case the available memory turns not enough, 3dfx suggests resorting to one of the two supported texture compression methods: DXTC or FXT1.

Omong-omong, ini mungkin satu-satunya kartu di mana memori dan chip disinkronkan. Itulah sebabnya 3dfx biasanya dikejar dengan buruk. Tetapi mengapa, jika mereka selalu menjadi pemimpin dalam kecepatan? Dan di sini ...

... Dan di sini juga keripik-keripik ini bergemuruh dengan kemampuan mulianya dalam industri yang diperlukan sebagai panduan make-up. Di sini disebut T-Buffer . Inti dari teknologi ini adalah implementasi perangkat keras Full Scene Anti-Aliasing ( FSAA ) melalui pengambilan sampel super 4x , yang melakukan hal yang persis sama: menghilangkan tangga dengan cara yang sangat kompleks di seluruh bingkai. Metode itu unik pada waktu itu, ia memberikan gambar yang sangat berkualitas tinggi, tetapi FPS mengunggulkan sehingga mereka menghidupkannya kembali hanya untuk beberapa tahun yang lalu .
Nah, berdasarkan FSAA ini, alat yang paling diperlukan untuk make-up juga diterapkan pada perangkat keras: motion blur, soft shadows, dan fokus pada objek. Aku tidak salah sekarang: itu adalah diimplementasikan.
Namun, ada FSAA ini dan perusahaan hanya di voodoo5 (diperlukan dua atau lebih chip). Yah, lebih tepatnya di voodoo4, itu juga, tetapi hanya FSAA 2x .

Also, T-buffer support was programmed into Serious Sam First/Second Encounter. It is supposed to be used for depth of field effects. To activate it, the game must run in 16-bit color depth, FSAA must be on and the following commands have to be written in the console:

ogl_iTBuffereffect=2
ogl_iTBufferSamples=4
The game is then supposed to confirm T-buffer usage in the console when starting it up. Other valid values for the second command are 2 and 8, depending on the number of VSA-100 chips on the used card. The FSAA implementation in this game is reported to be faster than in other releases. Still, given the experimental nature it is very unlikely that the DoF effects work as intended.

Multitexturing

Seperti yang kita semua ingat dengan baik :) 3dfx memiliki metode pemetaan tekstur canggung sendiri. Rumusnya sederhana: berapa banyak chip, begitu banyak tekstur per siklus. Artinya, dalam kasus kami, B5500 kami mungkin memiliki 2 tekstur per siklus. Inilah yang disebut multitexturing. Baik, dan sekarang ingat kurva tnt2 dan penyaringan trilinear-nya. Jadi, di sini semuanya persis sama (seolah-olah 3dfx memutuskan bahwa ini adalah jalur progresif). Yaitu, di dalam kayu Anda akan menemukan:
