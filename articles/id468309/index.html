Memperkenalkan Font Kode Cascadia

Kode Cascadia akhirnya tiba! Anda dapat menginstalnya langsung dari halaman rilis di repositori kami di GitHub atau secara otomatis mendapatkannya di pembaruan Terminal Windows berikutnya.



Tunggu, apa itu Kode Cascadia?


Cascadia Code diperkenalkan pada Mei ini di acara Microsoft Build. Ini adalah font monospace terbaru yang disediakan oleh Microsoft, yang menyediakan pengalaman baris perintah baru dan editor kode. Cascadia Code dikembangkan bersama dengan aplikasi Windows Terminal yang baru. Font ini direkomendasikan untuk digunakan dengan aplikasi terminal dan editor teks seperti Visual Studio dan Visual Studio Code.

Ligatur Perangkat Lunak


Cascadia Code mendukung ligatur perangkat lunak! Ligatur perangkat lunak paling berguna saat menulis kode, karena mereka membuat mesin terbang baru dengan menggabungkan karakter. Ini membantu membuat kode dapat dibaca dan â€œramahâ€ bagi sebagian orang.



Catatan: jika Anda menggunakan Visual Studio Code, maka untuk menggunakan ligatures Anda harus mengaktifkan Font Ligatures di pengaturan.



Apakah kami menyebutkan bahwa itu adalah open source?


Ya, Anda dengar benar! Cascadia Code tersedia di GitHub di bawah Lisensi SIL Open Font , sehingga Anda dapat berkontribusi jika Anda mau!

Catatan: saat ini tidak ada infrastruktur pengujian yang tepat dalam repositori, sehingga semua permintaan tarik harus diuji secara manual dan disetujui oleh Microsoft. Ini memperlambat koordinasi permintaan tarik, jadi harap dipahami.

Mengapa itu disebut Cascadia Code?


Nama Cascadia Code berasal dari proyek Terminal Windows. Sebelum dirilis, nama kode untuk Terminal Windows adalah Cascadia. Bahkan, beberapa file sumber di Terminal Windows masih menggunakan nama ini! Kami menyukai gagasan untuk memberi nama font dengan nama kode proyek. Ini merupakan penghargaan untuk Terminal Windows.

Kami menambahkan "Kode" di akhir nama font untuk menunjukkan bahwa font ini terutama untuk pemrograman. Secara khusus, ini membantu menentukan bahwa itu termasuk pengikat perangkat lunak.

Untuk memastikan bahwa Cascadia Code adalah pilihan yang tepat untuk nama font, kami melakukan survei Twitter bersama dengan opsi nama lain yang kami pertimbangkan. Kami sangat berterima kasih kepada semua orang karena berpartisipasi dan sangat senang bahwa Cascadia Code adalah pemenangnya.



Siapa perancang Cascadia Code?


Terima kasih banyak kepada Aaron Bell, desainer Cascadia Code. Kami cukup beruntung untuk bekerja dengannya selama setahun terakhir, dan dia melakukan pekerjaan yang sangat bagus. Aaron Bell, seorang desainer font dan pemilik Saja Typeworks , telah bergabung dengan Microsoft selama bertahun-tahun. Misalnya, pada tahun 2015, ia adalah desainer Selawik ! Anda dapat mengikuti aktivitasnya di Twitter ( @aaronbell ), dan di repositori kami di GitHub. Kami berencana untuk terus bekerja dengannya untuk meningkatkan Cascadia Code dan menjadikannya font yang bagus untuk semua orang!

Apa yang akan terjadi selanjutnya dengan Cascadia Code?


Jenis huruf Cascadia Code versi 1909.16 saat ini tersedia di GitHub. Ini akan mengikuti sintaks versi Windows dan akan menerima pembaruan yang terkait dengan mesin terbang baru dan lebih baik.

Tetap berkomunikasi


Untuk informasi tentang setiap pembaruan terkait dengan Cascadia Code, silakan ikuti repositori kami di GitHub atau halaman Twitter Kayla ( @cinnamon_msft ) dan Rich ( @richturn_ms ). Kami sangat senang untuk merilis font kami di domain publik dan menantikan ketika Anda mulai menggunakannya.
