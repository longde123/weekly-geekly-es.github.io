海盗的任务


海盗：A，B，C
说1
A：B有2只眼睛
B：C有2只眼睛
C：A有两只眼睛


说2
答：我们都有两只眼睛
B：我们都有三只眼睛
C：我们都有四只眼睛


每个海盗都像他的眼睛一样撒谎多次。 每个海盗有几只眼睛？


解决方案示例 
 df <- tidyr::crossing(a = 0:2, b = 0:2, c = 0:2) %>% mutate(total = a + b + c) %>% mutate(a_lie = (b != 2) + (total != 2), b_lie = (c != 2) + (total != 3), c_lie = (a != 2) + (total != 4)) %>% filter(a_lie == a, b_lie == b, c_lie == c) df
