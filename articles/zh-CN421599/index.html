英特尔深红峡谷-具有独立显卡和10纳米处理器的NUC



另一个有趣的消息来自英特尔NUC营地-那里的生活几乎如火如荼。 英特尔正式宣布了新的深红峡谷系列的两种型号，它们将基于Cannon Lake处理器。 因此，Intel Crimson Canyon成为第一批内部装有10纳米Intel Core处理器（或与之相同的Intel Core i3-8121U）的台式计算机。 但这还不是全部。 这是NUC历史上的第一次，婴儿获得了独立的图形。

所展示的两个型号NUC8i3CYSN和NUC8i3CYSM类似于两滴水，只是预装的内存量不同：前四个GB和第二个8。配备Radeon 540视频芯片（代号Lexa，Polaris体系结构），板载2 GB视频内存，英特尔深红峡谷定位为价格实惠的游戏计算机，可用于英雄联盟，TF2或CS：GO等游戏。 尽管英特尔的集成显卡在多媒体处理的某些方面具有一定优势，但离散显卡肯定会对游戏性能产生积极影响。

外观NUC8i3CYSM / NUC8i3CYSN绝对是“ nukopodobny”，不可能有人能够将它们与前几代兄弟区分开。



可以说相同的特性-针对新处理器和离散图形进行了调整。 Cannon Lake处理器之间的主要区别在于对AVX-512指令的支持，这对于小型家用计算机而言并不是很关键。
中央处理器 英特尔酷睿i3-8121U（2核，4线程，2.2-3.2 GHz，4 MB缓存，15 W TDP）
图形 AMD Radeon 540（512个流处理器，32个纹理单元，16个ROP，2 GB GDDR5内存）
记忆 4（8）GB LPDDR4-2666
贮藏 预设1 TB 2.5英寸硬盘
插槽M.2-2280，SATA / PCIe SSD或Intel Optane内存
Wi-Fi / BT 英特尔Wireless-AC 9560
802.11ac Wi-Fi + BT 5
乙太网路 英特尔千兆以太网（i219-V）
视频输出 2个HDMI 2.0a
USB接口 4×USB 3.0 Type-A（5 Gb /秒）
尺寸图 117×112×52毫米
电源供应 外部90 W
操作系统 Microsoft Windows 10家庭版x64
根据仍需澄清的非官方数据，新的NUC的成本将分别为530美元和575美元。 预定于2018年第4季度开始销售。
