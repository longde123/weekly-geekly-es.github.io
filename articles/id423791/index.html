Kami membuat konfigurator 3D tanpa pemrograman dan tata letak. Bagian dua

Dalam artikel kami sebelumnya " Membuat Konfigurasi 3D tanpa Pemrograman dan Layout " kami membahas masalah membuat toko perhiasan 3D menggunakan paket pemodelan Blender dan 3ds Max.

Hari ini kami akan memperbarui proyek ini dan menjadikannya toko yang benar-benar berfungsi. Untuk melakukan ini, kita perlu menyediakannya dengan fungsi-fungsi berikut:
  • Harga berdasarkan konfigurasi pengguna.
  • Mengirim pesanan untuk diproses lebih lanjut oleh manajer.
  • Tombol jaringan sosial (di mana tanpa mereka).

Pada akhirnya, kita akan memiliki sesuatu seperti ini:



Klik di sini untuk meluncurkan aplikasi.

Seperti biasa, Anda dapat menemukan file sumber (Max 3ds, Blender dan Teka-teki) dari konfigurator ini di versi terbaru dari distribusi kerangka Verge3D.

Rencanakan


Tugasnya terlihat agak ambisius. Kita perlu membuat toko 3D dengan konfigurator (sisi klien) dan sistem pemrosesan pesanan sisi server. Namun dalam kenyataannya, semuanya tidak begitu rumit, karena kami memiliki rencana yang jelas:

  1. Pertama, kita perlu mengkonfigurasi server untuk menerima dan memproses pesanan.
  2. Langkah selanjutnya adalah menghitung harga menggunakan editor visual Teka-teki dan menambahkan logika pembentukan pesanan dan mengirimkannya ke server.
  3. Akhirnya, kami menambahkan tombol ke aplikasi untuk membagikan aplikasi kami di Facebook, Twitter, LinkedIn dan Google+.

Mari kita lihat keseluruhan proses langkah demi langkah.

Pengaturan server


Dalam tutorial ini, kami menganggap bahwa server Anda memiliki sistem manajemen konten WordPress yang diinstal. Dimulai dengan versi 2.7, kerangka kerja Verge3D (yang terakhir kali kami gunakan untuk membuat konfigurator) dilengkapi dengan plugin WordPress. Plugin ini sudah mengandung sistem e-commerce sederhana yang dapat memproses pesanan yang ditransfer menggunakan editor logika tempat visual "pesanan" (kami akan membahas hal ini sedikit kemudian).

Jika Anda tidak memiliki server atau tidak tahu apa itu WordPress - jangan panik! Google "hosting WordPress", kami mempelajari dan memesan server yang sudah jadi (petunjuk - Anda tidak dapat memberikan lebih dari 100 rubel per bulan). Untuk mempelajari cara bekerja dengan WordPress untuk menjadi seorang programmer, administrator sistem atau perancang web tidak diperlukan.

Mari lanjutkan pengaturan toko kami. Pertama, instal dan aktifkan plugin Verge3D melalui menu "Plugins" di area admin WordPress. Plugin ini tersedia di direktori WordPress.org resmi, jadi tidak akan ada masalah instalasi.

Selanjutnya, untuk mengajari situs Anda cara memproses pesanan, tambahkan kode pendek berikut ke halaman atau posting mana saja:

[verge3d_order]

Kode ini diganti dengan formulir pesanan yang terlihat seperti ini (kurang-lebih karena penampilannya sangat bergantung pada tema WordPress yang digunakan):



Semua pesanan yang dikirim melalui formulir pemesanan ini akan tersedia melalui menu Verge3D-> E-Commerce di panel admin Anda:



Pesanan dalam daftar ini dibuat secara otomatis setelah menerima permintaan dari aplikasi konfigurator Anda. Misalnya, ketika John tertentu (pesanan pertama dalam gambar) membeli cincin perak dengan tiga batu berharga seharga $ 180, Anda dapat memanggilnya kembali atau menulis surat, mengkonfirmasi pesanan dan mencari informasi tambahan, seperti alamat pengiriman dan metode pembayaran.

Sebenarnya ini semua yang perlu Anda lakukan di server. Sekarang saatnya untuk kembali ke konfigurator 3D kami dan mengajarinya cara bekerja dengan WordPress.

Bagian ini cukup sederhana. Tambahkan puzzle "pesanan tempat" ke aplikasi Anda, tentukan URL pada formulir pemesanan, nama, deskripsi, dan jumlah pesanan. Inilah hasilnya:



Logika pekerjaannya cukup sepele: ketika pengguna mengklik tombol pesanan, kami menghitung harga cincin berdasarkan logam yang dipilih dan batu mulia. Kemudian kami menggunakan puzzle "pesanan tempat" untuk mengirim konfigurasi yang dipilih ke server.

Tombol sosial jaringan


Pernahkah Anda melihat toko tanpa tombol untuk berbagi di jejaring sosial? Itu saja, mari kita perbaiki kesalahpahaman ini dengan serangkaian teka-teki dari jenis yang sama:



Di sini logikanya bahkan lebih sederhana. Ketika pengguna mengklik tombol, kami membuka jejaring sosial yang diinginkan di tab baru, setelah sebelumnya melaporkan judul dan isi posting yang ingin kami terbitkan.

Kesimpulan


Metode yang dijelaskan dalam artikel ini dapat digunakan tidak hanya untuk konfigurator produk 3D, tetapi juga untuk aplikasi e-commerce lainnya. Misalnya, Anda dapat menggunakannya untuk membuat toko model 3D atau presentasi interaktif dari sesuatu, terutama jika Anda dapat memesan sesuatu dalam presentasi ini.

Adapun jaringan sosial, ruang lingkup untuk imajinasi sangat besar. Alih-alih tombol datar yang membosankan, Anda dapat membuat karakter animasi yang lucu.
