Contrôle vocal de la maison intelligente Z-Wave et au-delà. Homekit + Siri. Manuel d'instructions


Établir un maximum de confort dans l'appartement chaque mois devient de moins en moins cher et plus facile. Dans ce guide étape par étape, je vais vous montrer à quel point il est facile d'ajouter une commande vocale à votre maison! Pour la commande vocale, nous avons besoin de:
  1. Contrôleur RaZberry ou tout ordinateur avec logiciel Z-Way
  2. Une paire d'appareils Z-Wave, ainsi que des ampoules WiFi
  3. iPhone / iPad (de préférence dans le mur)

J'utilise Raspberry Pi B +, donc la description des paramètres de contrôle vocal sera décrite plus tard. Travail vidéo à la fin de l'article.

1) Installez le logiciel Z-Way version antérieure à 2.2.2 :
wget -O - razberry.z-wave.me/install/v2.2.2-test-new-webserver-8-ga9cd319 | sudo bash
Z-Way est un concentrateur pour Z-Wave, les appareils Enocean, diverses ampoules WiFi, des prises, des verrous et d'autres IoT contrôlés par des requêtes HTTP et TCP. L'interface Web est disponible sur le port 8083.


2) Installez la dernière version de Node.js
wget https://nodejs.org/dist/v4.4.5/node-v4.4.5-linux-armv6l.tar.xz
tar -xvf node-v4.4.5-linux-armv6l.tar.xz
cd node-v4.4.5-linux-armv6l
sudo cp -R * /usr/local/
Node.js est une plate-forme pour exécuter du code JavaScript sur un serveur basé sur le moteur Google V8.


3) Installer le plugin Homebridge et Homebridge-zway
sudo npm install -g homebridge
sudo npm install -g homebridge-zway
Homebrige implémente le support Homekit. Homebrige-zway est un plugin qui ajoute une prise en charge améliorée de Homekit à Z-Way.


4) Créez la configuration Homebrige /usr/local/lib/node_modules/homebridge/config.json
{
    "bridge": {
        "name": "Homebridge",
        "username": "CC:22:3D:E3:CE:31",
        "port": 51826,
        "pin": "031-45-154"
    },
    "platforms": [
    {
            "platform": "ZWayServer",
            "url": "http://localhost:8083/",
            "login": "admin",
            "password": "admin"
    }
    ]
}
La dernière barre oblique dans l'URL est très importante, sans elle cela ne fonctionne pas.

5) Lancer Z-Way et Homebridge
sudo /etc/init.d/z-way-server start
sudo homebridge -U /usr/local/lib/node_modules/homebridge &
Si tout est correct, dans le journal, vous verrez des appareils que vous pouvez contrôler et un code PIN.


6) En utilisant l'application Home sur le téléphone ou autres ( MyTouchHome , MyHome App , etc.), nous ajoutons l'appareil Homekit.


7) C'est fait! Maintenant, la maison peut être contrôlée par la voix! Commandes confortables!


PS La dernière fois, j'ai utilisé le module natif Homekit pour Z-Way pour la commande vocale , mais j'ai plus aimé Homebridge, car il permet de travailler non seulement avec Z-Way, mais aussi avec d'autres systèmes d'automatisation, tels que Philips Hue , Nest et bien d'autres. Plugins Homebridge sur github .
