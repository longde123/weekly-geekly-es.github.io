Raspberry Pi Compute Module 3 publié avec dix fois les performances


Module de calcul 3

En avril 2014, la Fondation Raspberry Pi a publié la première version du module de calcul (CM1) . Il était basé sur le processeur BCM2835 du mini-ordinateur Raspberry Pi d'origine. Avec le même processeur, cette carte était beaucoup plus petite: à peu près la même que la barre de RAM pour un ordinateur portable. En fait, le module est fabriqué dans le facteur de forme pour ordinateur portable DDR2 SODIMM. Tous les connecteurs et un peigne de contacts ont disparu sur la carte, seuls le processeur, la ROM et la RAM sont restés.

Près de trois ans se sont écoulés depuis la sortie du modèle d'origine. Pendant ce temps, deux générations de plus du Raspberry Pi ont vu le jour, chaque fois avec de plus en plus de performances. Il est donc temps de mettre à jour le module informatique: nous rencontrons le module de calcul 3 , rétrocompatible avec CM1.

Initialement, on supposait qu'avec l'aide d'un tel "module informatique" et d'une carte d'extension, les développeurs tiers pouvaient créer des systèmes plus fonctionnels que le Raspberry Pi lui-même, des PC à part entière. Ou vice versa, il peut être utilisé dans des appareils où la fonctionnalité et les dimensions du Raspberry Pi sont redondantes. En général, dans le développement de tout produit non standard.

Le module de calcul a acquis une reconnaissance et une popularité bien méritées. Il a été utilisé pour développer une variété d'appareils, y compris l'Internet des objets et des robots industriels. Le module informatique a même été envoyé dans l'espacedans le cadre de satellites miniatures CubeSat. Les petits ordinateurs bon marché ont prouvé qu'ils peuvent fonctionner de manière fiable dans des conditions d'exposition au rayonnement cosmique. Sous le contrôle de CM1, les satellites ont réussi à se détecter et à effectuer diverses manœuvres, notamment l'évitement des collisions et l'amarrage. Pour les manœuvres, des lidars ou des capteurs Microsoft Kinect ont été utilisés.

Compute Module 3 est basée sur le matériel du Raspberry Pi 3. Il a deux fois plus de RAM et environ dix fois les performances que le module d'origine.

Au total, deux versions de Compute Module 3 ont été lancées: standard et légère. Processeur

CM3 standard
: BCM2837 jusqu'à 1,2 GHz
RAM: 1 Go
Mémoire flash: 4 Go eMMC (intégré) Processeur

CM3L léger
: BCM2837 avec une fréquence d'horloge allant jusqu'à 1,2 GHz
RAM: 1 Go
Mémoire flash: interface pour cartes SD

Comme vous pouvez le voir, la version légère ne diffère qu'en l'absence de mémoire flash sur le module. Mais l'interface est restée, de sorte que SD ou eMMC peuvent être connectés si nécessaire. Le verso des deux modules est montré sur la photo: à gauche est CM3 standard, à droite est CM3L (Light).



Parmi les premiers, Compute Module 3 a été utilisé dans ses produits par NEC, qui l'a utilisé dans une nouvelle génération d' écrans larges . Ces écrans d'une diagonale allant jusqu'à 96 "sont destinés à être utilisés dans les lieux publics: écoles, bureaux, magasins, gares, etc.


Calculez le module 3 sur l'écran NEC

Ce n'est qu'un exemple de l'utilisation industrielle du CM3. Les modules informatiques sont susceptibles de trouver des applications dans d'autres domaines, comme les modules CM1 de première génération, espérons dans la Fondation Raspberri Pi. Tout groupe de passionnés dans le garage peut utiliser les mêmes technologies que celles disponibles pour une grande entreprise comme NEC. Dans le même temps, ils n'ont pas besoin de s'occuper des détails techniques tels que les brochages du processeur ou une interface mémoire à haute vitesse, ou l'alimentation d'un ordinateur. Lorsqu'elles sont connectées à une simple carte d'extension, toutes les interfaces externes fonctionnent immédiatement. Le module utilise le facteur de forme RAM standard pour l'ordinateur portable DDR2 SODIMM, ces connecteurs sont pris en charge par plusieurs fabricants, ils sont peu coûteux et facilement accessibles.

La taille de CM3 est presque identique à CM1, elle n'est que de 1 mm plus élevée. Le connecteur est le même. D'un point de vue matériel, la seule différence est la plus grande consommation d'énergie du processeur (VBAT). Sous forte charge, il devient très chaud.



Selon la tradition, des cartes d'extension de référence sont présentées avec le module, très probablement à des fins de démonstration et d'enseignement. Probablement, ces cartes sont pratiques à utiliser pour des expériences avec CM3 avant de concevoir et de publier leur propre carte.



La carte Compute Module IO Board V3 (CMIO3) fournit au module la puissance nécessaire et permet de programmer la mémoire flash (dans la version standard) ou d'utiliser des cartes SD (dans la version légère), facilitant ainsi l'accès aux interfaces du processeur (bloc de broches et connecteurs, comme dans RPi). Il existe des interfaces HDMI et USB.

La carte de référence CMIO3 est compatible avec les modules de première génération, ainsi qu'avec les nouveaux CM3 et CM3L.
