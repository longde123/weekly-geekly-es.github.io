三星的快速UFS存储卡：新标准有何期待？


三星UFS卡的最小体积

上周，三星电子公司发布了世界上第一个存储卡标准UFS 1.0（JEDEC通用闪存1.0卡扩展标准）。阵容包括32、64、128和256 GB的卡。

UFS存储器的开发始于2013年，当时电子设备工程联合委员会（JEDEC）通过了通用闪存2.0的标准（4月，JEDEC委员会宣布对UFS 2.1 的标准进行了更新）。 2016年，采用了JEDEC通用闪存存储1.0卡扩展标准，提供了开发新存储卡的能力。

在不久的将来，许多设备制造商将推出支持UFS标准的设备。首先，这些是动作相机，智能手机，无人机，摄影器材。三星已经开始在其设备中实现UFS内存：在Galaxy S6和S6 Edge智能手机中，主内存是UFS。除三星外，诺基亚，德州仪器，意法半导体和美光技术公司也支持拟议的规范。新标准比旧标准有什么优势？让我们弄清楚。

速度


对于现代设备，读/写速度是一个关键指标。这些卡的最大顺序数据读取速度达到530 MB / s，比UHS-1 microSD卡（最大95 MB / s）高五倍，比microSD卡高3.5倍。 UHS-II（156 MB /秒）。新卡的另一个优点是它们的高性能。



UFS 每秒提供4万次 I / O操作，是microSD卡的20倍。开发人员已经为UFS卡实现了更低的功耗。这是通过增加对多个团队的支持来确定他们的订单来实现的。



UFS读/写速度的真正优势甚至比上述更高。事实是，绝大多数设备中都安装了廉价的microSD Class 10卡。而且它们的效率远不及顶级名片。写入速度仅为10 MB / s。只有最昂贵的品牌型号（例如V90）才能提供高达90 MB / s的写入速度。但是即使在这种情况下，也只有在介质仍为空的情况下才能实现这种记录速度。 UFS在读取速度上是最快的microSD卡的五倍，在写入速度上是2-3倍。

MicroSD规格适用于处理1080p视频质量或jpeg图像。但是现在4K视频格式变得越来越普遍，照片的分辨率也在提高。许多摄影师更喜欢使用RAW照片。在这里，microSD卡已经出现故障-它们的性能还不够。新标准及时出现了。

降低功耗


通常，设备或其元素（处理器，图形适配器，驱动器）的高性能会导致大量功耗。对于UFS，情况有所不同。生产力最高的microSD卡在峰值模式下的功耗为2.88瓦，在空闲模式下的功耗为0.72瓦。根据新标准的规范，新型存储卡的功耗仅为1.54瓦。这几乎是顶级microSD卡的两倍。 UFS卡的高读写速度也意味着文件传输速度更快，并且该卡宁愿在活动模式下完成其工作，而是切换到待机模式。



闲置时，这种卡几乎不消耗能量。这意味着与使用microSD卡的类似设备相比，使用UFS卡的照相机或摄像机可以拍摄更多照片或拍摄更长的照片。具有主要和辅助UFS内存的电话也将持续更长的时间。

统一应用程序的内存


内部和附加内存属于同一标准的移动设备的出现，将使移动应用程序开发人员的生活更加轻松。现在，某些公司和个人开发人员不允许在存储卡上安装其应用程序或使之复杂化。一切都与microSD标准的性能有关，这会对应用程序的性能产生负面影响。

在许多现代小工具中，内部存储器是指eMMC标准，而其他存储器是指microSD标准。最初，有人提议用UFS接口代替过时的eMMC接口。这个接口已经使用了几年来将内部存储器连接到移动设备的芯片上。 eMMC-并行接口，UFS串行接口。这样的改变不仅可以降低功耗，还可以简化电路板上的布线。对于数据传输，只允许使用一行。具有统一内存的新移动设备上的应用程序性能应更高。



诸如统一内存扩展之类的功能允许特别大容量的应用程序在RAM耗尽的情况下使用设备的UFS存储卡。目前尚不清楚是否会在Android OS上使用此功能，但是这个主意本身是不错的。



立即引入新标准将不起作用，因为microSD和UFS卡的触点位置不同。 UFS的电接口是MIPI联盟发布的M-PHY标准。这意味着您不能将UFS卡插入microSD卡插槽，反之亦然。因此，移动设备制造商将必须安排支持UFS标准的小工具的发布。这需要时间。三星可能会在其智能手机的新型号中增加对新卡的支持。小工具和存储卡本身的全面进入UFS市场应该在2017年至2018年之间的某个时间进行。

UPD三星今天说关于一项新的开发-UFS和microSD格式的存储卡通用插槽。在评论的较早部分，许多用户抱怨新格式与旧格式不兼容，并且无法在新手机中读取microSD卡。该公司已经解决了这个问题。

三星的新设备（包括平板电脑，手机，笔记本电脑）将安装这些通用插槽。卡仍然没有向后兼容，因此无法读取带有UFS格式的microSD卡插槽的旧设备。可能首款配备新型插槽的设备将是Galaxy Note 7。

如果公司的发展将被电子设备制造商所采用，那么将不会存在兼容性问题。到目前为止，谈论microSD即将退出流通市场是不可能的。这种格式的存储卡随处可见，成千上万的存储在用户手中。另外，几乎所有现代设备都支持这种格式，因此，例如，您可以从手机中卸下microSD并将其插入平板电脑，而无需考虑两个小工具的兼容性。

无论如何，一个能够处理各种格式的套接字是一个好主意。
