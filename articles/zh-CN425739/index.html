微软宣布Project xCloud-尖端的游戏流媒体服务

微软决定宣布自己的竞争服务后，Google Project Stream宣布的风声并未平息。 发言由Kareem Choudhry主持，他自98年以来一直在Microsoft工作 ，现在担任游戏云公司副总裁。


游戏的未来是一个每个人都有选择机会的世界。 选择自己喜欢的游戏。 选择与谁一起玩。 您可以随时随地玩。 当然-在您喜欢的设备上。 我们认为游戏的发展类似于音乐和电影的历史-玩家应该始终能够按需获取所需内容，并在任何屏幕和设备上观看。 今天，我们提出了一个关键项目之一，它的任务是使我们更快地走向光明的未来：Project xCloud。



最近，我们不是随意选择游戏，而是在即将来临的铁压力下选择游戏。 Project xCloud是一种超现代的流技术，可让我们自由地在您要使用的设备上播放。 我们不再局限于特定的硬件。 这使我们的游戏玩家有能力成为自己的游戏世界的主人。


内容与社区


Project xCloud对于控制台和PC游戏玩家的最终目标是能够选择时间和地点。 移动玩家-访问此类游戏，他们的世界，角色和故事，由于硬件限制，他们到目前为止无法触摸。


我们了解，为了实现这一愿景，我们需要使开发人员尽可能轻松地将内容传输到Project xCloud。 Xbox One现已提供3000多种游戏的开发人员，以及现在制造成千上万种新产品的每个人都将能够留在这里，并认真提高各种设备上播放器的内容可用性，而这一切都不需要任何额外的工作。


什么是Project xCloud


创建和扩展Project xCloud是一个长期的过程，您可以很快加入。 我们将在2019年开始进行首次公开​​演示，由于有了这些演示，我们希望考虑在不同负载和不同地理位置使用它的经验。 我们将专注于确保Xbox上的播放器获得强大的新游戏体验，并致力于通过可扩展到成千上万新播放器的新工具扩展开发人员的功能-来自所有可用设备的统一受众。 Project xCloud旨在为所有设备的所有播放器提供高质量的游戏体验，其震撼力将与您在PC和游戏机上获得的媲美。


基于特殊的服务器硬件，它们自己的数据中心实现了与Xbox平台上现有和将来的游戏的兼容性。 多年的游戏开发不会浪费，而是会通过这些新技术得到加强。 发明了一种新的自定义刀片服务器的体系结构，该体系结构可以容纳多个Xbox One控制台的组件以及整个支持的基础结构。 基于这些自定义刀片，Azure区域将随着时间扩展。



Project xCloud当前正在测试中。 测试是在通过蓝牙与Xbox无线游戏手柄配对的设备（手机和平板电脑）上进行的，或者使用设备本身的触摸屏进行的。 主机和PC上的游戏旨在将它们深深地浸入其中，通常需要借助许多按钮，操纵杆和触发器来进行复杂的控制。 因此，正在开发一种特殊的触摸屏，按计划，该触摸屏应为想要使用而不使用游戏控制器的玩家提供最大的响应能力和最小的负载。



（图像中-由原型触摸界面控制的在Project xCloud中运行的游戏）


云游戏流传输是一项多方面的，具有挑战性的任务。 它具有大多数其他形式的数字娱乐所不具备的功能：交互性。 游戏是一种互动体验，可以根据玩家的反应而动态变化。 一次在多个设备上实现高质量体验应考虑到开发人员途中遇到的许多障碍，例如组织低延迟的视频流或支持具有许多用户的大型网络。 在游戏中，揭示创作者艺术意图的可能性或不可能性通常取决于图形的质量。 因此，除了解决延迟问题之外，还必须保持高清晰度的图形和帧速率，并且所有这些都应乘以来自玩家或玩家处置的来自控制器或触摸界面的输入的响应性。


微软基于开发游戏平台的40年经验（从PC的开发开始，对软件和硬件的开发都有深刻的了解），为下一步游戏流传输的复杂工作做好了充分的准备。 Microsoft Azure在54个地区设有数据中心，在140个国家/地区提供服务，这绝对是一个足够规模的系统，使玩家可以以最佳方式获得世界各地的一切信息，而不论他们的地理位置如何。




Microsoft Research开发人员和研究人员正在研究解决延迟问题的方法，发现了构建网络拓扑，编码和解码视频的新方法。 Project xCloud将开启通过4G网络流式游戏的可能性，并且当它们随处可见时将自动扩展到5G网络的规模。 目前，测试正在以每秒10兆比特的速度进行。 目标是在大多数现有网络上以最小比特率提供高质量的流传输，但要考虑到每个设备和网络的唯一性。


明年很快，我们将能够在公共演示中一起研究Project xCloud，您将成为这种独特游戏体验的中心。 保持联系！
