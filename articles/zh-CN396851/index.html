特斯拉Model 3：关于电动汽车模型的所有已知信息



明年，特斯拉汽车公司将推出Model 3，这是一款已经讨论了很长时间的电动汽车。这是最实惠的机型，今年4月已收到超过30万笔预订单。现在这个数字已经增加到40万。尽管有这样一个事实，即在下订单时，买方会留下1000美元的押金。

Model 3不会在2017年底之前交付给客户。购买者将会收到的有关电动汽车的已知信息是什么？

设计施工


值得强调的主要是新设计。是的，电动汽车是基于Model S平台构建的，但是大多数元素都是从头开始创建的。新的电源架构，新型的电动机，外壳设计等开发于几年前开始，直到今年春季才完成。而且，仍在进行一些改进电动汽车的工作。

该公司的工程师竭尽所能将风阻降至最低。特斯拉Model 3的阻力系数非常低-只有0.21。该系数的最小可能值对电动汽车很重要，因为它可以提高电动机的效率。

Model 3将比Model S缩短和缩小，新产品的尺寸可与BMW 3系媲美。与S型和X型一样，壳体不仅由铝制成，而且还添加了钢质元件。

此前，埃隆·马斯克（Elon Musk）表示，Model 3的初始版本对行李箱的小开口提出了许多疑问。为了解决这个问题，决定对设计进行修改。一些媒体还谈论带有透明屋顶的模型的可能性。这种屋顶的玻璃是由多层构成的。

只需额外付费，即可获得空气悬架。

汽车内饰类似于S型内饰，座椅经过了改装，触摸屏的尺寸也从17英寸减小到15英寸。使用此“电子助手”进行汽车功能管理。它的界面有所变化，导航和多媒体系统管理工具占据了大部分空间。屏幕横向安装。



在演示中，显示了一个完全没有普通仪表板的汽车版本-所有信息都显示在中央监视器上。尚不清楚是否保留此设计功能。开发人员试图使控制系统具有未来派的外观，以便所有者感觉像飞船的飞行员，而不是地面飞行器的飞行员。这有点像公司已经破产了在这方面有很多惊喜。



机舱可容纳五名成人。前排座椅稍微向前移动，后排沙发被推到船尾。据特斯拉称，由于有了这个和高高的车顶，乘客在任何车辆配置下都将感到舒适。

特斯拉Model 3的规格


电动汽车有两种版本-后轮驱动和全轮驱动。四轮驱动的附加费约为5000美元。一次充电，汽车就能行驶约350公里。也许到生产开始时，该节目甚至会得到改善。加速到100 km / h是6秒。电动汽车的基本版本同时具有自动驾驶和快速充电功能。如果需要的话，将有可能获得电动汽车的改进版本，它仅需4.5秒即可加速至100 km / h。

为了使电动汽车的拥有者对道路充满信心，计划增加增压器充电站的数量。现在大约有三千六百个，在不久的将来，这个数字将会增加两倍以上。最初，Model 3所有者可以免费使用此类工作站。但是，过了一段时间，马斯克说，很可能是要访问网络，这种费用必须在开始时就支付一次。

自动驾驶仪2.0


特斯拉不久将推出其虚拟“驾驶员助手”的新版本。这个版本将成为Tesla Model 3的一部分。在
Autopilot 2.0中，不仅软件，而且硬件都已更新。因此，根据一些报道，特斯拉电动汽车驾驶员仍将获得全自动驾驶模式。现在，Autopilot不提供自动驾驶功能，其任务是使道路上的交通更加安全。

马斯克（Elon Musk）已经声明，Autopilot 2.0可以在最困难的天气条件下“看到”周围的一切，包括雾天，降雪，雨天或多尘的空气污染。软件平台已经根据修改后的设备进行了更新。系统将能够识别并响应各种标志和交通信号。现在，自动驾驶仪无法执行此操作。

如上所述，特斯拉Model 3所有者可以免费使用Autopilot，但您必须激活自动驾驶模式才能赚钱。估计费用-2500美元。

那么什么时候可以买到呢？


首先，电动汽车将被运送到那些预先订购的客户。如果特斯拉不更改其计划中的任何内容，这将不早于2017年发生。由于订单很多，因此电动汽车的交付时间可以延长到2018年。首先，那些已经拥有Model S或Model X的客户将收到他们的新电动汽车，然后将交付给来自美国的买家。只有这样，特斯拉Model 3才能进入其他国家。



考虑到所有流行的选择，电动汽车的平均成本至少为42,000美元，

现在该公司的工厂甚至没有时间处理Tesla Model X的订单，订单不多，更不用说成千上万的Tesla Model 3了。

为了解决这个问题，该公司设定了一个目标-到2018年将电动汽车的产量提高到50万辆。早些时候，设定了类似的目标，只是日期不同-2020年。显然，为了实现这一目标，必须为所有人（包括管理人员和普通员工）做大量工作。用他自己的话说，埃隆·马斯克（Elon Musk）被这一过程吓倒了，甚至在工厂里睡觉。

他和他的团队计划成为世界上最好的机器制造商。该团队计划招募员工，并且在不久的将来将开设数十个（或数百个）新的职位空缺。
