Seminarios web de Skillbox Friday: desarrollo y todo lo relacionado



El próximo viernes, lo que significa la aparición de una nueva selección de seminarios web gratuitos de Skillbox en Habré. Los videos de hoy son sobre programación:

  • Desarrollo de aplicaciones PHP.
  • Desarrollo de aplicaciones para iOS.
  • Cómo convertirse en un desarrollador de Android.
  • Herramientas de automatización para desarrollo móvil.
  • Cómo comenzar a programar en Python.

Le recordamos: para todos los lectores de "Habr": un descuento de 10.000 rublos al registrarse en cualquier curso de Skillbox con el código de promoción "Habr".

Cómo desarrollar una aplicación PHP



Este es el seminario web del principal ingeniero de Badoo, Mikhail Ovchinnikov. Él les dice a los programadores novatos cómo desarrollar una aplicación en PHP. Se consideran las razones de la popularidad de PHP como lenguaje para el desarrollo de backend, los niveles de dificultad de las aplicaciones web básicas y lo que se requiere para convertir a un programador de PHP en un profesional.

Curso en línea "Desarrollador PHP de 0 a PRO"

Cómo desarrollar una aplicación para iOS



El CEO de Anvics, Nikita Arkhipov, habla y muestra cómo se están desarrollando las aplicaciones de iOS, demuestra las capacidades de Swift y Xcode. El caso central es la creación independiente de una aplicación de tareas pendientes: trabajar con la visualización de datos y procesar las acciones del usuario.

Cómo convertirse en un desarrollador de Android



Mikhail Ovchinnikov, un programador líder de Badoo, habla sobre cómo convertirse en un desarrollador profesional de Android. El nivel de habilidades y conocimientos para crear su primera aplicación móvil, herramientas para el desarrollo de Android y una historia sobre cómo un programador en prácticas puede convertirse en un líder de equipo en una empresa internacional: esto es lo que le espera en este seminario web.

Curso práctico "Desarrollador móvil desde cero"

Herramientas de automatización de desarrollo móvil



El gerente de desarrollo móvil de Agima, Dmitry Shashlov, habla sobre las herramientas de automatización, incluida la integración continua, la entrega continua, las pruebas automáticas y las herramientas para trabajar con código en un equipo: gitflow, solicitud de extracción y revisión de código.

Curso práctico en línea "Mobile Developer PRO"

Cómo comenzar a programar en Python



El mentor del programador en línea de Skillbox Vadim Shandrinov habla sobre cómo programar en Python. El seminario web revela las posibilidades del lenguaje, muestra cómo trabajar con funciones en Python. El caso práctico es la creación de la primera aplicación web. Además, el autor aborda el tema de la demanda de un desarrollador de Python en el mercado laboral.

Curso práctico "Desarrollador Python desde cero"

Toda la selección en su conjunto es universal y puede ser útil para todos los que trabajan en TI. Si no programa en los idiomas mencionados en los seminarios web, nada le impide aprender más sobre ellos. Y si programa, es hora de actualizar sus conocimientos.
