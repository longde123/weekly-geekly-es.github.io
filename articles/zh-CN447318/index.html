跳蚤市场上的小工具：为什么要花10欧元买一台20年历史的Packard Bell笔记本电脑



前几天，在当地的格拉纳达跳蚤市场上，我遇到了一台不寻常的笔记本电脑。 它非常厚，出售时附带了自己的文档（手册，保修单等），并带有Windows XP贴纸，表明它属于21世纪。

讨价还价，我花了10欧元买了这台笔记本电脑，然后带回家。 在周末，我决定弄清楚什么构成了技术奇迹，并在随后产生了研究结果。 提供许多图片，包括电子内部图片。

Packard Bell EasyOne Silver有两种型号：一种具有256 MB的RAM和1 GHz的Intel Celeron处理器，第二种具有128 MB的内存和1 GHz的Intel Pentium III-M处理器。 我得到了第二个模型，但是增加了RAM（256 MB）的数量。

特点


  • 型号名称：Packard Bell Easy One Silver 3100 DVD
  • 尺寸：31厘米x 26.5厘米x 4厘米
  • 重量：3.4公斤（根据感觉而定-6公斤，不少于）
  • 处理器：Intel Pentium III-M 1 GHz
  • 缓存：256 KB，二级缓存，高级传输缓存
  • 内存：256 MB / 384 MB（最大），SDRAM，133 MHz
  • 软盘驱动器：3.5英寸1.44 MB
  • 硬盘：20 GB
  • DVD光碟
  • 显示：14.1英寸TFT 1024 x 768（XGA）
  • 调制解调器：56 Kbps
  • 电池：镍，容量4400 mAh。
  • 工作时间：长达2小时
  • 操作系统：Microsoft Windows XP Mediacenter Edition

这就是我买的笔记本电脑的样子。











这是他随附的文件的照片。 这是用于快速设置的图解手册，更详细的多页手册以及有关电池保养和笔记本电脑本身使用说明。











不幸的是，我没有得到XP光盘，但是标签上有许可的OS代码。

出乎意料的是，笔记本电脑可以完全正常运行-一切都从声音到设备（可能是由于设备的厚度和扬声器的尺寸，声音非常大）到DVD驱动器。 不幸的是，事实证明电池是一个薄弱环节。 在20年的工作中，她不再承担任何责任（自主权-50秒，仅此而已）。

笔记本电脑无法立即打开。 事实证明，BIOS电池也已放电，这并不奇怪-该设备寿命很长。 Windows XP正常启动，事实证明，它与所有其他软件一样，运行速度非常快。 但是，一旦我抓到了蓝屏死机。













管理学


与当时的许多其他型号一样，Packard Bell EasyOne Silver可以使用面板上显示的专用按钮来控制某些功能。 基本上，这是音乐播放控件，尽管前面板上有一个按钮可以调用邮件客户端。 便利性令人怀疑，但在2000年代却很时尚。



车轮音量控件也显示在左侧-这是我第一次在笔记本电脑上看到它。 通常，所有这些都是通过按钮实现的，但是开发人员决定自己做。

琴键


她很舒服。 按键半透明-在塑料下，键盘组件清晰可见。 异常地。 长时间的操作并未消除按键上的油漆。 通常，笔记本电脑本身的质量很高，细节彼此紧紧固定在一起，即使20年后它们也不会吱吱作响。

驱动器


DVD在那些年代还不常见，因此直接在自己的笔记本电脑上观看电影的能力很酷。 该驱动器原来正在工作，他读取了几个磁盘。

事实证明，使用FDD驱动器会更糟。 有人把名片塞进去了。 我已经以为驱动器无法正常工作，但是决定检查一下。 他拿出所有名片，看一看驱动器中的所有内容是否完好无损，然后开始寻找软盘。 一共有三个软盘。 他们中的两个原来正在工作，并显示了已存储多年的文件。





联播网


不幸的是，笔记本电脑没有以太网端口，也没有WiFi或蓝牙。 但是，另一方面，我设法在PCMCI储藏室中找到了Wi-Fi卡，该卡安装成功。 可以连接到本地网络，但不能连接到Internet。





该卡软件允许您选择各种工作配置文件，无论是否加密，是否经过授权。 但是，可能只是该卡与现代WiFi标准不兼容，因为它是最大连接速度为11 Mbps（制造商EZ Connect）的型号。





我经历了几个配置选项，但没有成功。 家用设备ping通，但Packard Bell拒绝上网。

导热膏冷却和更换


风扇已经很吵了。 我决定拆卸，清洁和更换导热油脂。 视频中有详细的分析（通常，我最初计划用容量更大的硬盘代替HDD，但事实证明，安装的硬盘不是SATA，而是IDE。我没有这样的硬盘）。 冷却系统仍然非常简单，冷却器本身安装在覆盖处理器的金属散热器中。 如果用散热器卸下冷却器，则可以使用处理器，一切都很简单。















清洁系统后，笔记本电脑变得更安静了。 是的，主动冷却的频率比以前少了-显然，导热膏的更换很有帮助。

旧笔记本电脑有什么用处？


笔记本电脑的质量很高，如果不是新的话，就不会被过度利用。 但是，不幸的是，这仅适用于身体状况。 许多年前它已经过时了。 它可以用于教孩子们在PC上工作（如果他们摔坏了也可以）或用于打字和播放音乐。 对于其他方面，这台机器几乎不适合-既因为没有网络连接接口，又因为笔记本电脑已经非常笨重。

您还能在西班牙跳蚤市场上买什么？


任何你想要的。 我每个星期六去那里，有时会遇到很多有趣的设备。 因此，相对最近，我以10欧元的价格从Western Digital购买了一个可以正常使用的外部TB MyBook 3 TB HDD。 卖方不知道那是什么。



最近，我遇到了一台不带HP Pavilion电池，带有Intel Core i3和8 GB RAM的笔记本电脑，价格为70欧元。 我花了16欧元买了一个电池，现在农场里还有另一台笔记本电脑可以解决常见问题。

通常会遇到用于笔记本电脑和台式机的硬盘驱动器。 它们的3-5欧元售价为500 GB-1 TB。 但是已经有彩票了，有时磁盘却无法正常工作。 但是就在一周前，我花了2欧元买了一个工作正常的500 GB硬盘用于笔记本电脑。 因此，在大多数情况下，这款游戏还是值得的。
