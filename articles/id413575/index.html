BricsCAD Shape - CAD 3D gratis dari Bricsys

Baru-baru ini, produk CAD baru dari Bricsys dirilis dengan diam-diam dan hampir tanpa terasa di Linux, sehingga Linux diisi ulang dengan alat lain yang harus dimiliki untuk prototipe virtual .




Tautan untuk mereka yang ingin mencoba / mengevaluasi, versi untuk Windows dan MAC juga tersedia:
unduh halaman
Tutorial Bentuk Bricscad
pelajaran di saluran youtube resmi

Paket saya yang diunduh saat menginstal di Debian diinstal dengan kesalahan / dependensi tidak puas. Saya harus mengirimkan libwebkitgtk-1.0-0, yang kemudian menarik beberapa paket lagi.



Buka jendela



Setelah meluncurkan jendela utama, perasaan gigih dari sesuatu yang akrab tidak meninggalkan saya; di suatu tempat, saya sudah melihat "gambar" dengan seorang pria kecil di sumbu koordinat :-)



Ketika Anda memulai kembali, saya mencoba untuk membuka salah satu proyek saya dan ... itu jatuh



Perilaku ini diamati jika Anda membuka proyek pihak ketiga / selesai dari jendela peluncuran, tetapi jika Anda membuka segera setelah memulai jendela utama dari menu, maka semuanya terbuka dengan sempurna untuk mengedit dan menyimpan.





Sebagai kesimpulan, IMHO murni. Alat di Bricsys keluar sangat cocok, pada pandangan pertama, persilangan antara SketchUp dan AutoCad.

Pro:
- 146% gratis, tidak ada SMS
- Antarmuka yang sederhana dan intuitif
- Pemodelan solid 3D
- Akurasi dan dukungan perintah seperti pada CAD klasik
- keberadaan perpustakaan bahan dan komponen
- kompatibilitas penuh file output dengan AutoCad, menyimpan dalam format .dwg, dxf, .dwt

Cons:
- perangkat lunak berpemilik
- sementara tidak ada lokalisasi dalam bahasa Rusia, hanya bahasa Inggris
- terkadang jatuh
