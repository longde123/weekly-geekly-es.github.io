Fundo translúcido

Gostaria de mostrar como criar rapidamente um plano de fundo translúcido. Se você usar o comando opacity no CSS, o plano de fundo certamente se tornará transparente, mas com ele também tudo o que estiver nele.

Como corrigir este problema? Vasculhei a Internet e encontrei uma maneira maravilhosa, sobre a qual vou falar agora.

Digamos que este seja o seu site (html): 

<html> <head> <meta charset="UTF-8"> <link rel="stylesheet" href="style.css"> <title>   </title> </head> <body> <div class="mid"> <p> </p>  .  . </div> </body> </html>

A linha "" inclui uma folha de estilo CSS.

Aqui está (é chamado style.css): 

 body{ background: "   " ; * background-attachment: fixed ; background-position: top center; } .mid{ width: 1250px;** margin: 0 auto; background: rgba(161,21,207, 0.5);*** }

Na classe média 3, a linha define a cor da cor translúcida no formato RGBA. Seus 4 dígitos, neste caso 0,5, são o valor da transparência de 0 a 1.

Tão simples. Experimente, você certamente terá sucesso!

* - insira as imagens da URL aqui
** - defina a largura da página (para isso, criamos a classe no meio do código html)
*** - rgba é a cor no formato rgb. A última letra (a) indica a transparência do plano de fundo. Seu valor é um dígito de 0 a 1. 0 é um plano de fundo completamente transparente, respectivamente, 1 é o oposto.
