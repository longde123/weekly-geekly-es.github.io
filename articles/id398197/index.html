Perangkat Lunak iPad yang Sering Digunakan

Dua setengah tahun yang lalu, saya mendapatkan tablet iPad Air, yang sangat cocok dengan alur kerja.


Secara bertahap, daftar aplikasi tertentu dibentuk yang luar biasa membuat hidup lebih mudah dan memungkinkan beberapa tugas diselesaikan lebih cepat dan lebih mudah.
Tugas utama yang dipecahkan tablet ini adalah menyelamatkan saya dari membawa kertas dan laptop berat di gedung berlantai 52 yang sedang dibangun, tempat sebagian dari hari kerja saya dihabiskan.
Mungkin sesuatu dari daftar ini akan menarik bagi pengguna Geektimes.

1. Aplikasi untuk bekerja



Microsoft Word


Ketika saya membeli tablet, Microsoft office suite untuk iOS gratis. Ada beberapa saat yang tidak nyaman ketika menggunakan produk ini, dan tidak terpikir oleh saya untuk memulai dokumen baru dari awal, tetapi sesuatu dapat diperbaiki dengan cepat. Halaman dan produk dari Google juga memiliki fungsi serupa, tetapi saya menyukai Word. Saya juga menggunakan Excel, tetapi juga untuk pengeditan kecil cepat.


Foxit PDF Reader


Salah satu aplikasi yang paling banyak digunakan di tablet. Ini berisi beberapa gigabyte dokumentasi proyek dalam PDF. Anda dapat membawa album dengan gambar di situs konstruksi, atau Anda dapat melihat semuanya di layar. Tentu saja, data dapat disimpan di cloud di suatu tempat, tetapi di situs saya saat ini sangat buruk di banyak tempat dengan komunikasi seluler, dan WiFi belum berjalan. Jadi, kesempatan untuk melihat dokumen lokal membuat saya senang.

Di sana Anda dapat membuat catatan dan komentar di tempat itu, langsung pada gambar dengan rencana dan mengirimkannya ke orang yang tertarik.


Autocad 360


Kadang-kadang Anda harus melihat kode sumber gambar, aplikasi berpemilik dari Autodesk gratis, itu mengatasi tugas, tetapi meminta pendaftaran di situs web mereka dan akses ke Internet. Dan cukup banyak melambat. Ini tidak nyaman, tetapi saya tidak menemukan sesuatu yang lebih baik. Pada suatu waktu saya menggunakan TurboViewer, tetapi ia bahkan memiliki lebih banyak gangguan.


Evernote
Perangkat lunak yang mudah digunakan untuk catatan, daftar periksa, daftar tugas dan hal-hal lain untuk kolaborasi. Saya menggunakannya secara aktif sebelum acara-acara terkenal. Sayangnya, paket tarif gratis untuk dua perangkat tidak lagi mencukupi untuk saya dan akibatnya saya datang ke dua aplikasi, yang dijelaskan di bawah ini.


Kertas dropbox


Setelah Evernote mengubah kebijakan lisensi, saya menemukan Paper Dropbox tepat waktu. Itu cocok dengan semua kondisi yang saya butuhkan:

1. Kemampuan untuk membuat dan mengedit daftar besar dengan kotak centang dengan cara yang nyaman.
2. Akses dan kolaborasi web.
3. Jika memungkinkan, aplikasi untuk iOS dan Android.
4. Gratis.

Saya menggunakan semua fitur ini untuk membuat daftar belanja. Ini memiliki beberapa ratus item, posisi yang diperlukan selama seminggu dicatat oleh saya dan istri saya, dan kemudian di toko pilihan digantikan oleh pencarian. Nah, urutan daftar ini terkait dengan rute saya yang biasa melalui hypermarket.


Microsoft OneNote


Ini adalah produk kedua yang mulai saya gunakan secara aktif setelah meninggalkan Evernote. Saya pernah menggunakannya sebelumnya, tetapi sekarang semua catatan kantor dan catatan pribadi telah masuk ke dalamnya.
Dan juga sangat nyaman bagi saya selama berkeliling di sekitar fasilitas selama penerimaan pekerjaan dan hanya untuk kontrol.

Sangat nyaman untuk merekam semua komentar di salah satu catatan, melampirkan foto, dan setelah saya turun dari menara, kirim halaman ini sebagai file PDF ke semua orang yang berkepentingan melalui surat. Dan sinkronisasi dengan semua perangkat juga berguna.


DocScaner Pro


Program yang sangat mudah untuk mengambil gambar dokumen dan mengirimkannya sebagai file PDF melalui surat (atau meletakkannya di dropbox dan cloud lainnya), menggabungkan beberapa foto menjadi satu dokumen multi-halaman, dll. Sepertinya dia dulu butuh uang, tapi saya katakan padanya selama kampanye gratis.


Kamera Foto HDR


Untuk beberapa waktu saya aktif menggunakan aplikasi ini untuk memotret objek dalam kondisi pencahayaan yang sulit. Tampaknya vyrviglazno, tetapi detailnya tidak hilang dan informatif.


Microsoft photosinth


Sayangnya, saya tidak dapat menemukan aplikasi untuk ini di App Store, tetapi sangat nyaman untuk memotret panorama berbentuk bola. Sangat nyaman ketika Anda perlu mengambil gambar ruangan yang sempit, di mana Anda tidak dapat memilih titik pemotretan normal dan tidak cocok di bidang tampilan lensa.

Tentu saja, kami tidak memiliki fotografi seni, tetapi informasi visual yang terdistorsi, namun demikian.


Sekarang Anda dapat menemukan banyak analog. Dan dalam hal ini, saya pikir, dan menggunakan "Street View" dari Google. Ada kemungkinan memotret panorama 360 derajat, tapi saya masih menggunakan aplikasi ini.


Editor Foto oleh Aviary

Anehnya, saya telah bertemu beberapa editor di mana Anda dapat menyembunyikan sesuatu di foto, menambahkan teks dan menyimpan. Saya menemukan ini, saya sudah menggunakannya selama beberapa tahun.


ICQ


Saya menggunakan messenger ini dengan sangat aktif selama bertahun-tahun, tetapi kemudian saya sampai pada kesimpulan bahwa itu lebih merepotkan daripada baik dan menghapusnya.


Telegram


Ini mengakar untuk komunikasi dalam keluarga, mempertahankan saluran tentang visi saya tentang Yekaterinburg dan untuk beberapa bot yang berguna.


Kalender


Aplikasi lengkap di iOS. Kemampuannya cukup bagi saya untuk tidak melupakan pertemuan dan acara dan, yang paling mudah, itu dapat dikaitkan dengan kalender akun Google, serta mengatur kalender umum. Saya memiliki dua di antaranya: "Keluarga" dan "Bekerja". Keduanya sama dengan istrinya, jadi dia sadar akan rencana kerja saya dan, merencanakan kegiatan keluarga, dia berusaha untuk tidak tumpang tindih.


Pengingat


Sekali lagi, aplikasi standar dari iOS. Menarik bagi saya hanya dari sudut pandang pengingat yang mengikat ke daerah tersebut. Datang ke suatu tempat dan melupakan sesuatu adalah tentang saya. Dan sangat lelah dengan pengingat.


2. Aplikasi yang tidak terkait pekerjaan


FlightRadar24


Ini mungkin tampak konyol bagi banyak orang, tetapi saya suka bermeditasi di pesawat terbang ke tanah yang jauh.


Raja kolase


Aplikasi ini agak ketinggalan jaman, sekarang ada versi 2, tapi reviewnya tidak menyenangkan saya.

Hobi saya memasak, kadang-kadang saya memposting ide yang telah diwujudkan di dapur di LiveJournal atau Instagram, sementara menggunakan kolase untuk foto dan video sepertinya cocok untuk saya.


Peninjau tim


Saya pikir produk ini tidak boleh disajikan kepada siapa pun. Saya menggunakannya sangat jarang untuk bekerja, para pekerja ini berada di awan, nama yang saya tidak akan sebutkan, tetapi untuk tujuan pribadi saya menggunakannya untuk mengontrol komputer di mana pengawasan video berjalan.


Padgram


Saya sering bepergian, dan sekarang saya suka membuka peta semacam kegelapan dan melihat foto-foto dari orang biasa di Instagram. Baru-baru ini saja Instagram mengizinkan saya untuk mencari foto dengan geotag, tetapi saya sudah terbiasa dengan Padgram.


Snapseed


Editor foto lain. Pangkas, sesuaikan level, kontras kecerahan.


Foursquare


Cari restoran, kafe, objek wisata, ulasan untuk mereka dan itu saja.
Saya suka Forskquire dengan lembut dan telah menggunakannya selama beberapa tahun sekarang. Sayangnya, menurut pendapat saya, forcequire kehilangan posisinya karena facebook, google, dan tripadvisor. Juga, suasana yang berlaku di komunitas pengguna super sedikit frustasi (dalam hal permintaan paksa, itu adalah seseorang seperti moderator).


Swarm
Forskweir dulunya satu dan tak terpisahkan, tetapi kemudian entitas "Chekin" dan "Pencarian untuk pendirian" dibagi.


Vivino


Mesin pencari dan panduan anggur yang sangat baik. Mencari label foto. Proses pemilihan anggur untuk makan malam menjadi lebih lama, tetapi kualitas hasilnya telah meningkat secara dramatis.


Saya meringkas


Sejak akuisisi, tablet telah mengambil sejumlah besar tugas yang sebelumnya diselesaikan pada laptop atau smartphone. Sebagian besar surat pada proyek saat ini dibaca di tablet, dan sepertiga dari surat itu tertulis di atasnya.

Sebagian besar posting LJ juga ditulis menggunakan tablet. Saya bahkan menolak smartphone yang kuat setelah mogok dan membeli Lumiya murah di Windows Phone. Telepon berhenti duduk dalam sehari, dan saraf saya menjadi utuh. Sekarang, jika WhatsApp dapat diinstal sepenuhnya di iPad, maka saya akan membeli sendiri tombol Nokia yang biasa dalam wadah logam, tetapi sayang sekali.

Pilihan platform, tentu saja, adalah urusan semua orang, saya yakin akan ada analog dari perangkat lunak pada Android dan Windows Phone.
