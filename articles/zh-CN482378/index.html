Wacom的简史：图形输入板技术如何应用​​于电子阅读器

Wacom主要以其专业图形输入板而闻名，全世界的动画师和设计师都在使用它。 但是，该公司不仅从事此。

她还将自己的组件出售给其他技术公司-例如，生产电子阅读器的ONYX。 我们决定简短回顾过去，并说明Wacom技术为何征服了世界市场，并说明使用ONYX产品的电子书阅读器制造商如何使用该公司的解决方案。


图片： Szabo Viktor / Unsplash

Wacom技术改变了市场


最早的图形输入板出现在上世纪60年代。 它们用作将数据输入计算机的另一种方法。 用户可以使用手写笔将其绘制在平板电脑上，而不必在键盘上键入字符。 专用软件可以识别字母和数字，并将它们插入相应的输入字段。

随着时间的流逝，图形输入板的应用范围不断扩大。 1970-1980年，工程师和建筑师开始使用它们来与计算机辅助设计系统一起使用，例如AutoCAD（其第一个版本于1982年发布 ）。 该时代最著名的两种图形输入板是Intelligent Digitizer和BitPad。 两种设备均由美国公司Summagraphics发行，该公司长期以来一直是垄断者。

她甚至使用白标模型将解决方案交付给其他组织（当一家公司生产产品而另一家以自己的品牌销售产品时）。 顺便说一句，苹果公司建立了第一个基于BitPad系统的图形输入板-苹果图形输入板。

但是80年代生产的平板电脑有一个缺点-手写笔是有线的，这限制了自由度，使绘图变得困难。 更正由日本Wacom公司（1983年成立）的工程师决定的情况。 他们申请了一种新的坐标输入系统的专利，该系统可使用无线笔在计算机屏幕上控制光标。

该技术的原理基于电磁谐振现象。 工程师在平板电脑上放置了许多传感器的网格，这些传感器发出微弱的电磁信号。 该信号形成一个磁场，该磁场延伸超过工作表面五毫米。 系统通过分析此字段中的更改来记录点击。 至于触控笔，在其内部放置了一个电容器和一个专用线圈。 数位板工作表面上方的电磁波在其中产生电流，从而为笔提供必要的能量。 结果，他不需要电线或单独的电池。

基于这种新技术的第一台平板电脑是 1984年推出的 Wacom WT-460M。 他很快开始征服世界市场。 1988年，公司在德国开设了代表处，三年后在美国成立了代表处。 然后Wacom与迪士尼建立了伙伴关系-工作室使用他们的设备创作了动画电影《美女与野兽》。

大约在同一时间，Wacom无线技术进入了基于DOS和Windows的PC领域。 它构建了计算系统NCR System 3125 。 该设备具有E Ink屏幕和可识别的手写字符。 很快，这家日本公司的系统甚至在美国政府中也被使用。 1996年，比尔·克林顿（Bill Clinton）总统使用Wacom设备以数字方式签署了 1996年电信法案。

在公司成立期间，Wacom已形成多个方向。 第一个与为设计师和艺术家制作专业平板电脑有关。 Wacom产品已成为艺术行业的标准。 来自Riot Games和Blizzard的专家以及来自Pixar Studios的艺术家都在使用该公司的设备。 Wacom工作的另一个领域是商务平板电脑。 它们使您可以数字化工作流程并开始在组织内使用电子签名。 例如，一家日本制造商将智利汽车租赁公司Hertz，韩国旅馆Nine Tree Premier和美国医疗组织Sharp Healthcare用于这些目的。

专业艺术家和商业产品是该品牌的标志，并因此赢得了全球声誉。 Wacom在图形输入板市场上的份额超过 80％。 但是，日本制造商还有其他发展领域。

另一个利基市场-电子阅读器配件


该公司开发用于电气工程设计的CAD系统，并为其他公司提供组件（尤其是触摸屏和触控笔）。 他们的技术需求的原因之一是手写笔允许您在屏幕上控制光标的高精度。 在公司的整个生存过程中，Wacom的工程师已申请了许多专利，这些专利改善了电磁传感器和软件算法。 通常，他们努力使人感觉像使用笔在纸上绘画一样。

基于Wacom组件，合作伙伴公司不仅在构建图形输入板，而且还在构建包括阅读器在内的其他电子设备。 这样的公司之一就是ONYX，该公司于2009年推出了首款采用Wacom触摸技术的电子阅读器ONYX BOOX 60。 在阅读器的“板上” 是 6英寸的E Ink Vizplex显示器，带有Wacom的触摸层。 压敏部件位于阅读器的玻璃屏幕下方，并与特殊的手写笔反应 。 它可用于导航（选择设备上的菜单项）和创建手写便笺。

Wacom解决方案还用于现代ONYX阅读器。 直到现在，这家日本制造商才扩展了笔的功能：它对压力产生了更好的响应。 触控笔内置有可变元素-根据挤压强度-电阻，可让您在显示器上绘画时更改线条的粗细。 此功能已将简单的阅读器变成具有平板电脑功能的多功能小工具。


照片中： ONYX BOOX MAX 3

这种类型的第一台ONYX BOOX设备是Note Pro 。 它具有10.3英寸E Ink Mobius Carta高分辨率屏幕。 这种尺寸的显示器可以让您轻松地阅读教育或技术文献。 该设备随附的Wacom笔支持2048度的压力。 Gulliver和MAX 3阅读器附带了类似的手写笔。

使用笔，您可以直接在文档中做笔记-对于使用阅读器来处理技术文档或笔记的人，此功能将很方便。


照片中： ONYX BOOX Note 2

配备Wacom笔的最新ONYX BOOX型号为Note 2和Nova Pro设备。 它们配备了对角线分别为10.3英寸和7.8英寸的E Ink Mobius Carta显示器。 同时，与以前的阅读器不同，他们的屏幕具有两个传感器层。 第一个是电容式多点触摸显示器，用于翻开书页并用手势控制阅读器。 第二个是Wacom笔感应层。 与单独的电容式传感器相比，与手写笔配对的感应层具有更高的定位精度。 手写笔更容易在屏幕上选择要翻译的单词（例如，如果在英语文档中找到不熟悉的短语），然后按屏幕键盘上的按钮。 带有触控笔的手的位置更自然-出现隧道综合症的可能性较小。

在这种情况下，笔本身的Note 2和Nova Pro可以识别4096度的压力，从而增加了绘制线条粗细变化的范围。 因此，ONYX BOOX Note 2可用作小草图和草图的相册。 如果启用了相应的模式，则可以根据需要直接在PDF或DjVu文档上进行绘制。 阅读器将允许您将编辑的文件保存并导出到智能手机或计算机。

Wacom触摸板和笔安装在屏幕尺寸为7.8英寸或更大的大型ONYX阅读器中。 对于这种类型的小工具，记笔记和草图的能力是一项重要功能，可以大大扩展使用该设备的选项。 实际上，它结合了阅读器和基于E Ink的“数字笔记本”。 使用PDF和DjVu处理文档的能力吸引了工程师和其他技术专家-根据我们的估计，使用Wacom笔的阅读器的需求低于“小型”阅读器，但非常稳定。

Wacom的新项目和未来发展


11月下旬，日本制造商与E Ink Corporation共同推出了一种新型的彩色E Ink显示器。 该系统称为Print-Color ePaper-在这种情况下，特殊的彩色滤光片直接应用于E Ink胶片。 已经有一个带有10.3英寸屏幕的原型设备，该设备可支持4096度压力的特殊Wacom触控笔。 Sony，SuperNote，Boyue和ONYX将制作具有新屏幕的阅读器-预计将在2020年下半年发布。


请注意，ONYX已经拥有开发带有彩色屏幕的设备的经验。 年初，在CES-2019上，该公司展示了 Youngy BOOX阅读器。 它的屏幕为10.7英寸，分辨率为1280x960像素，最多可显示4096种颜色，并支持Wacom手写笔。 但是，该设备并未公开发售-只有部分中国学校将其作为教育计划的一部分收到。

未来，ONYX计划通过彩色屏幕扩大阅读器的范围。 部分产品将在明年初的CES-2020上展出。 但是，并非所有新产品都可能进入市场。 这完全取决于对彩色读取器的需求，但事实证明，该需求明显低于传统的黑白设备。

Wacom在年初也成立了一个新的财团-数字文具财团。 三星，富士通和万宝龙已经进入那里。 他们将共同寻找E Ink的新应用程序，并为基于E Ink的设备创建云服务-例如，在阅读器之间交换电子书或同步书签。 该联盟计划每年举行四次会议，以在全球市场推广电子墨水技术。
