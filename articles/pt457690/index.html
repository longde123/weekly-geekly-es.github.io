Recentemente, li um artigo sobre como aprender a programar para Android a partir do zero em meia hora. Tudo começou com as palavras "Você nem imagina, mas milhões de pessoas em todo o mundo ganham dinheiro com o desenvolvimento de aplicativos para Android". Além disso, é claro, não havia nada de bom nele: instale o Android Studio, aqui está o XML, a ativação, digite texto Java aqui. Um artigo completamente padrão, ensinarei a programar em meia hora um anúncio de cursos pagos.


A verdade é que nenhum dinheiro é suficiente se você tentar aprender tudo no mundo em cursos pagos. E a outra verdade é que no mundo existem grãos de conhecimento útil sobre desenvolvimento de bons desenvolvedores - e de graça. Este post é um desses grãos.

Kit ML: primeiro móvel para primeiro AI


Yuri Chechetkin, Yandex.Money


No ano passado, o Google I / O anunciou o ML Kit - um conjunto de ferramentas para implementar redes neurais e aprendizado de máquina em aplicativos móveis. No relatório, analisaremos o que o ML Kit pode fazer e para o que é útil.



Recursos de desenvolvimento do cliente para um sistema distribuído


Revolut Roman Yatsyna


Vamos falar sobre sistemas distribuídos em termos de design de aplicativos clientes. Digo a você o que é idempotência, por que precisamos e por que o estilo de passagem de mensagens e a API compatível com CQRS são importantes e como usá-los para projetar uma camada de dados de aplicativos.
