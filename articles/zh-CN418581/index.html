React Basics（教科书，第二版）

自从发布有关React基础的详细教程的第一版以来已经过去了2.5年。 在这段时间内，gitbook吸引了250,000多位唯一身份访问者。



在猫之下，详细介绍了该教程和更改列表。


首先是文字。 Legacy.gitbook仍可以PDF / epub / mobi格式下载。 在新版本中，我找不到此选项，但是不幸的是，新版本的gitbook在没有VPN的情况下无法正常工作。


其次-在“水”教科书上也是如此。 文字，图片，源代码。


在学习过程中，描述了一个简单的场景：新闻提要（来自json文件）+添加新闻的能力。 没有调情后端。 大部分开发工作直接进入index.html，直到最后我们才熟悉create-react-app 。


本教程的目的是向已经熟悉html / css并且了解js基础的人介绍React。 该库将作为常规脚本添加。 在使用jQuery时很方便，我没有开始重做训练公式。
