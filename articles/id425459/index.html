Memutakhirkan ke Windows 1809 (kadang-kadang) menghancurkan semua file dalam profil

Kolega di bengkel asing memperingatkan ( 1 , 2 ) bahwa ada bug serius pada penginstal Windows 1809.

Saat ini, diketahui bahwa Anda akan kehilangan file dari profil Anda (Dokumen saya, Gambar, Desktop, dll.) Jika Anda telah mengaktifkan pengalihan folder profil ke lokasi lain atau disinkronkan dengan OneDrive.

UPD # 1
Pengguna Spiceworks telah menemukan bahwa kebijakan pembersihan untuk profil lama tidak berfungsi dengan benar

UPD # 2
Saat ini, operasi yang salah dari kebijakan grup untuk membersihkan profil lama telah dikonfirmasi - situasi yang dijelaskan pada reddit atau answer.microsoft.com tampaknya disebabkan oleh bug khusus ini. Pada 23:59 pada 10/05/2018, tidak ada seorang pun dalam subjek yang menderita dari situasi ini. Namun, kasing dari upd # 1 mudah diulang di mesin virtual apa pun.

UPD # 3
Seperti yang diharapkan, majelis dipanggil kembali.

Subjek pada Jawaban Microsoft.
