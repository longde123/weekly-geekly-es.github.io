Einige Statistiken - Ubuntu-Entwickler haben zuerst Desktop-Telemetrie veröffentlicht

Im Februar dieses Jahres erschien das Dienstprogramm Ubuntu Report ( GitHub ) in der Ubuntu 18.04 LTS-Distribution. Es gab Benutzern die Möglichkeit, Daten über ihre Arbeitsumgebung an die Zentrale der Entwickler zu senden, um das Betriebssystem weiterzuentwickeln.

Und letzte Woche haben die Ubuntu-Entwickler ihre erste Telemetrie veröffentlicht. Als nächstes geben wir einige Zahlen.


/ Pixino / Hagerty Ryan / PD

Warum Telemetrie sammeln?


Laut Will Cook (Will Cooke), dem Manager der Firma Canonical, die Ubuntu entwickelt, haben sie begonnen, Telemetrie zu sammeln, um den Entwicklungszyklus des Betriebssystems und seiner neuen Funktionen zu verbessern. Will bemerkte, dass die Entwickler keine Marketingziele verfolgen.

Zu den gesammelten Informationen gehören: die verwendete Ubuntu-Version, die Grundspezifikation des PCs (Daten auf dem Prozessor und der Grafikkarte, Speicherkapazität, Bildschirmauflösung des Monitors) sowie Informationen zur installierten Software von Drittanbietern und zum Standort des Benutzers. Eine vollständige Liste finden Sie in Cooks offenem Brief vom Februar .

Im Allgemeinen begrüßte die Community die Entscheidung, Telemetrie zu sammeln, und etwa 67% der Benutzer, die kürzlich Ubuntu Desktop installiert hatten, stimmten zu, Daten mit Canonical zu teilen. Einige Benutzer haben jedoch eine Reihe von Bedenken geäußert. Insbesondere befürchten sie, dass anhand der erhobenen Daten die Identität des Eigentümers festgestellt werden kann.

Canonical stellt jedoch sicher, dass der Benutzer entscheidet, welche Daten freigegeben werden sollen, und markiert sie mit den entsprechenden Häkchen im Menü. Gleichzeitig werden IP-Adressen und Suchverlauf im Browser nicht an die Server des Unternehmens übertragen. Und was übertragen wird, wird über HTTPS gesendet.

Cook betont, dass die Telemetrie Entwicklern hilft, sich auf benutzerfreundliche Elemente des Systems zu konzentrieren. Hier können Sie die Konfiguration des Bügeleisens, die Anwendungsnutzungsparameter und deren Einstellungen untersuchen. Auf diese Weise können Entwickler die Veröffentlichung von Fixes und Patches genauer priorisieren und den Entwicklungszyklus entsprechend anpassen.

Was die gesammelten Daten zeigten


Laut Telemetrie haben drei Viertel der Benutzer Ubuntu Desktop 18.04 „von Grund auf neu“ installiert, und das verbleibende Viertel wurde von einer älteren Version aktualisiert. Die durchschnittliche Installationszeit des Betriebssystems betrug 18 Minuten (mindestens 8 Minuten). Mehr als die Hälfte der Benutzer mit einer vollständigen Neuinstallation des Systems löschte und formatierte Festplatten.

Bei Monitoren stellte sich heraus, dass die meisten Benutzer die Auflösung auf Full HD (1080p) eingestellt haben. An zweiter Stelle der Popularität stand die Auflösung von 1366x768. HiDPI und 4k waren äußerst selten.

In diesem Fall verfügen die Computer der meisten Benutzer über einen Monitor, eine Grafikkarte und eine CPU. Canonical hat jedoch keine Daten zur Anzahl der Kerne von Zentralprozessoren veröffentlicht.

Die weltweite Verbreitung von Betriebssystemen wurde basierend auf der im System festgelegten Zeitzone geschätzt. Die Daten zeigten, dass sich die meisten Ubuntu-Benutzer in den USA befinden. An Popularität folgen Brasilien, Indien, China und Russland. Canonical ist jedoch der Ansicht, dass diese Daten möglicherweise nicht vollständig zuverlässig sind, da die US-Zeitzone standardmäßig für alle neuen Betriebssystembesitzer festgelegt ist.

Noch ein paar Statistiken


Ubuntu ist nicht nur als Betriebssystem für PCs beliebt. Laut einer Cloud-Marktstudie dominiert Ubuntu 2018 die Anzahl der bereitgestellten Images in der Cloud. Auf der offiziellen Website von Ubuntu wird darauf hingewiesen, dass dieses Betriebssystem häufiger in öffentlichen Clouds "gefunden" wird (auf 60% der virtuellen Maschinen bereitgestellt).

Darüber hinaus wird Ubuntu Server häufig für die Arbeit mit OpenStack verwendet. Laut einer von OpenStack unter 70.000 Mitgliedern der OpenStack Foundation durchgeführten Umfrage wird Ubuntu in 37% der Produktionsumgebungen, in 10% der Entwicklungsumgebungen und der Qualitätssicherung sowie in 5% der experimentellen Umgebungen bereitgestellt.

In der Cloud wird dieses Betriebssystem beispielsweise von Unternehmen wie Netflix, Uber, Bloomberg , Pinterest und Instagram verwendet . Oft stellen sie ihre Arbeitstools und Anwendungen in der IaaS-Cloud bereit.

Als IaaS-Anbieter haben wir unsere Untersuchungen unter Nutzern des 1cloud- Dienstes durchgeführt. Und sie stellten fest, dass Ubuntu unter den Betriebssystemen für virtuelle Server an zweiter Stelle steht - 20,5% unserer Kunden verwenden es. Windows belegte mit 46% den ersten Platz und CentOS (17%) den dritten Platz.


Und von den 20%, die Ubuntu "nimmt", sind 14% auf der 16. Version des Betriebssystems. Es folgen Ubuntu 14 mit 5% und dann Ubuntu 12 und Ubuntu 18 (sie haben weniger als ein Prozent).

Was weiter


Will Cook sagte, dass Canonical plant, eine spezielle Website zu starten, auf der allgemeine Statistiken über die Verwendung von Ubuntu frei verfügbar sein werden. Auf diese Weise kann jeder interessierte Benutzer herausfinden, wie viele Menschen auf der ganzen Welt dieses Betriebssystem installiert haben, welche Anwendungen bei Ubuntu-Benutzern am beliebtesten sind und so weiter.

Laut Canonical stimmt eine solche Lösung gut mit dem Konzept der „Offenheit“ überein, das sie in Zukunft aktiv fördern wollen.

Die Website wird vor der Veröffentlichung der Ubuntu 18.10-Distribution (Cosmic Cuttlefish) veröffentlicht, die für Oktober 2018 geplant ist.
