专业视频会议前所未有。 思维服务器-新版本，新价格

长期以来，视频会议（VKS）思维服务器的国内解决方案不仅在俄罗斯广受欢迎。 这是用于本地网络或通过开放Internet进行会议和网络研讨会的专业系统。 Mind Server支持SIP / H.323硬件终端，WebRTC连接和移动客户端，以及会议记录，内容和桌面演示，与Skype for Business集成等。

几年前，我发表了一篇专门针对该产品的文章： 头脑视频会议是商务的最佳选择 。 该服务的功能和界面是如此完整，以至于多年来它们并未发生重大变化。 但这并不意味着该产品没有开发-在削减之后，我将描述与以前版本相比的主要改进。

现在，在继续介绍Mind Server之前，我想再次提醒一个重要事件：Habr上出现了一个专门用于视频会议解决方案的单独的中心，我邀请所有涉及这一广泛且相关主题的人都订阅视频会议中心。

10项主要变更


  1. 屏幕演示和单个应用程序的窗口的质量变得更好。
    Skype以HD（720p）画质捕获屏幕，并以全高清（1080p）画质提供Webex和Zoom等付费服务。 结果，当将桌面翻译缩放到全屏时，远程参与者会看到模糊的字体和模糊的图像，因为现代的显示器和演示面板通常具有更高的分辨率。
    在头脑中，该演示现在以4K UltraHD进行！

    与Skype的比较

  2. 全面支持WebRTC和拒绝Flash。
    以前，会议开始时的参与者必须通过浏览器明确允许使用Flash。 不幸的是，这并非总是如此。 结果，他们没有听到也没有看到发言人。 现在已经停止使用RTMP（闪存）技术。 对于会议和网络研讨会，Mind始终使用WebRTC技术。 直到2020年底，Mind平台都将支持Flash，但仅用于向后兼容。
  3. 实现了自动媒体质量调整。
    除手动控制外，网络研讨会还根据通信通道的宽度自动调整视频广播质量。 在秋季发布的版本将自动调整会议媒体的质量。 这极大地简化了参与者对服务的使用，并使您可以专注于事件。
  4. 大大改善了会议和网络研讨会的声音质量。
    由于优化和改进了用于处理音频流的算法，声音变得更加清晰。 尽管有这些改进，但不应忽略对音频设备和房间声学准备的有效选择。
  5. 人体工程学的程度和界面的易用性得到了提高。
    对界面设置进行了大量更改。 特别是：
    在“关于事件”选项卡上，添加了用于连接VVoIP客户端的地址：sip：event_id @ domain_address。
    通过单击链接“说明和支持”，增加了将用户重定向到内部客户技术支持门户的功能。
    秋季版本将在人机工程学方面进行更多更改，以简化用户开发服务并减少不必要的操作数量。
  6. 改善了平台的可靠性和安全性。
    切换到新版本的Debian增强了所有平台组件的稳定性和安全性。 新版本的Mind Server在Debian GNU / Linux 9操作系统上运行（而不是不再受支持的Debian GNU / Linux 7），直到2020年7月才受支持。 计划在2020年将Mind平台迁移到Debian GNU / Linux 10。
  7. 改进的数据传输安全性。
    在传输级别使用TLS / SRTP协议时，可以为通过SIP连接的设备提供增强的数据传输安全性。
    TLS加密增强了对通过开放Internet传输的媒体流量的保护。
    现在，Mind平台使用内置的Opensips 2.4 LTS。
  8. 简化了系统的安装和更新 。
    系统管理员可以使用带有图形界面的标准Debian安装程序，在不到30分钟的时间内独立安装或更新Mind。
    另外，由于更新了OC内核的设置，Mind的安装时间减少了30％。
  9. 改进的解决方案兼容性 。
    Mind平台除了以前支持的那些之外，还支持H.261，H.265，VP9，MPEG-4 Part 2视频编解码器和G.722.1音频编解码器。
    对现代视频和音频编解码器的支持使您可以连接更广泛的软件和设备。
  10. 前所未有的低价格 。
    更新后的资费计划中的Mind云服务价格已经比西方同类产品低2倍。 公司网络中本地安装的许可价格已经与Mind云服务的价格相等。
    现在，所有资费计划都支持免费连接领先制造商的硬件终端和SIP电话的功能。
    此外，可以下载和独立操作用于一个在线活动的Mind视频会议服务器的无限分发套件。
