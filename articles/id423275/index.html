Skillbox Friday Webinar: Segalanya untuk Pemula



Webinar adalah cara yang baik untuk mendapatkan informasi dan pengalaman di luar tembok sebuah institusi pendidikan. Untuk melihat dan mendengarkan, Anda hanya memerlukan laptop, perangkat seluler, dan, mungkin, notebook untuk merekam tesis yang menarik. Kami harap Anda memiliki semua ini, karena sudah waktunya untuk menerbitkan koleksi baru dari Skillbox. Hari ini adalah segalanya bagi pengembang.

Bahasa pemrograman mana yang harus dipilih


Teknisi perangkat lunak Badoo, Mikhail Ovchinnikov, berbicara tentang bagaimana dan bahasa apa yang dipilih untuk membangun karier pengembang. Selain itu, ia membahas pilihan arah dalam IT - dari gaming ke AI - dan menceritakan bagaimana mengembangkan dan membangun karier.


Bagaimana menjadi pengembang Java


Daniil Pilipenko, pengembang profesional dan direktur SymbioWay Center, berbagi pemikiran dan pengalaman pribadinya tentang bagaimana menjadi programmer Java.

Webinar berbicara tentang sintaksis bahasa dan praktik mengembangkan program Java dengan diskusi tentang kemungkinan kesalahan, dan secara paralel dengan ini, tentang komponen profesionalisme pengembang dan elemen-elemen kunci yang membentuk jalur pengembang Java mulai dari pemula hingga pro.
Kursus praktis "Profesi Java-developer"


Cara memprogram dalam Python


Vadim Shandrinov, programmer-mentor di Universitas Skillbox, berbicara tentang bagaimana bekerja dengan Python tanpa melupakan tentang potensi pengembang Python di pasar tenaga kerja.

Dalam prosesnya, Anda akan melakukan pekerjaan langsung dengan membuat produk Python pertama Anda.

Cara mengembangkan aplikasi untuk iOS


Nikita Arkhipov, CEO Anvics, berbicara dan menunjukkan bagaimana aplikasi iOS sedang dikembangkan. Bagian utama dari webinar adalah pembuatan aplikasi ToDo online. Menjelaskan cara melakukan ini, Nikita berbicara tentang prinsip-prinsip bekerja dengan Swift dan Xcode, memberikan gagasan tentang cara mengaitkan antarmuka dengan kode, dan mengajarkan cara menampilkan data dan memproses tindakan pengguna.
Kursus praktis "PRO Pengembang Seluler"
