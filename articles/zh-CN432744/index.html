DWDM：该解决方案比运营商便宜30-50％（企业级）



在光学市场上，过去两年发生了一些变化。 现在，您可以购买自己的DWDM设备，并将它们放在数据中心的机架中。 并且获得所有这些都比传统的运营商解决方案便宜。

谁确切需要：

  • 如果您拥有2012年之前实施的DWDM / CWDM传输网络。
  • 如果您需要提高传输网络的吞吐量和/或连接新的分支机构，那么您只是在计算预算。
  • 如果您同时拥有一个地铁网络（不是跨大陆的，而是在城市及其郊区内）。
  • 如果您的光通道过载或很快就会过载。

几年前，许多大型供应商DWDM宣布该设备将以更加企业友好的方式发展（更紧凑，更实惠，具有更多带宽）。 现在发生了，但是这种“友好”的形式有所不同。

在这篇文章中，我将解释为什么现在该切换到企业设备的原因，并且我将回顾来自多家顶级供应商的设备：华为，ADVA和Ciena。

教育计划


大公司将他们的观点与光学联系起来。 在许多银行，能源部门，交易所，大型机械工程，运输公司等中，几乎所有石油和天然气都有自己的运输网络。

首先，要连接放置“暗”光学元件的物体。 这是一种简单的光纤，可以向其中提供数字信号。 极其简化，有光-一，无光-零。 通常，在铺设的电缆中，没有一个芯线，而是几个芯线：沿着12–32–64线束。

深色光学器件的传输能力很快就耗尽了，因此您需要“画线”。 这需要光学密封，它们也是DWDM设备。 DWDM不仅可以“着色”信号，而且还可以将多个信号复用为一个信号，也就是说，由于能够在不同波长下工作，因此可以将多个载波“推”到一个光芯中。 除了物理压缩本身之外，该设备还使用OTN技术将多个低速流多路复用为一个高速流。 对于我们的教育计划，原理本身并不重要，但重要的是这是物理学和技术的特殊萨满教法。

此外，大型公司会在L1级别上构建容错网络。 通常，我们在线性信号上保留信道：第一个光信号沿着通常的（最短）路径，第二个光信号沿着独立的光线或沿着通过节点（作为分散网络的简单情况的“三角形”）传输的环形拓扑。点有独立的路线。 另外，我们通常会放置1 +1套来重复流量。 这对于创建完全冗余以及在传输网络级别创建防灾解决方案的可能性非常重要。

那么新设备呢？ 他的能力是什么？


首先，高带宽。 更详细-稍后，这就是原因。

公司的流量正在增长-这是一个明显的趋势。

如果您仍然认为对于发展中的公司来说，在80根800 Gb / s的光线路上有很多，那就出了问题。 通常，现在您已经需要更多带宽用于不同的任务和传输流量。

图1.数据中心带宽的演变



该图显示了现代数据中心中传输数据流的稳定增长。 此外，速度为100 Gbit / s的流量所占比例越来越大。

其次，成本。 到2015年，每条线100 Gbit / s的成本要比10 Gbit / s的10倍贵。 每个人都被光纤束带走了，但是现在厂商制造的设备的200 Gbit / s的价格比10倍便宜20倍，甚至比10倍便宜10倍。

第三，尺寸。 老操作员放在十字架上，新的Enterprise放在服务器房间里。 这不是像以前那样用于办公室的机架，也不是用于阁楼的机架，而是一块铁，应该放在数据中心内，冷热走廊都应如此。 现在，模块在服务器机架下方的深度不再是300毫米，而是600毫米。

我们直接进行企业设备的审查。

让我们从华为开始。 他可以在两个单元上放置六个板； 每条线路每条可以产生400 Gbit / s。

该设备紧凑，并提供巨大的带宽。 但是它没有DWDM的经典光学部分。 使用放大器和光学滤波器的各种波长的复用和解复用的光学部件（可能具有线性冗余）仍保留在操作员设备上。 也就是说，如果您将设备的一部分替换为现有的WDM传输网络，则它将通过计算一个λ扩展每行200 Gbit / s的速度来扩展网络带宽。 也就是说，当使用单个Huawei OSN902多路复用器时，线路中的最大带宽将为2.4 Tbps。

这就是ADVA的功能。

图 3.示例企业设备ADVA-FSP 3000 TeraFlex



Internet流量的增长以及向云服务的过渡，使我们能够提高数据中心之间网络的容量和速度。 太多的客户需要快速开发基础架构，同时保持简单性并满足对密度和功率不断增长的需求。 为了应对这些挑战，ADVA创建了TeraFlex终端。 现在它尚未发售。 它能够在一个波长下传输高达600 Gbit / s的数据，并在一对光纤上提供3.6 Tbit / s的总容量。
