Microsoft a breveté un moyen de suivre un utilisateur à l'aide de l'IA et d'enregistrer toutes les actions dans Windows


«L'intermédiaire» est le nouveau composant de Windows 10 de Microsoft. Il ressemblera peut-être à cela.

Le dernier système d'exploitation de Microsoft, Windows 10, occupe une part croissante du marché des systèmes d'exploitation. Cela est dû en partie au fait que Windows 10 a reçu l'approbation de nombreux utilisateurs. Et en partie - le fait que Microsoft essaie de promouvoir activement son produit. Au cours de l'année écoulée, les méthodes de promotion de Windows 10 par ses créateurs sont devenues à plusieurs reprises l'objet de discussions dans les médias.

Une autre nuance est le transfert de grandes quantités de données utilisateur de Windows 10 vers les serveurs de l'entreprise. La télémétrie dans le "top ten" inquiète un grand nombre d'utilisateurs qui ont tenté de découvrir pourquoi Microsoft possède autant de données. En réponse, la Société a déclaré que les informations reçues sont anonymisées et ne servent qu'à améliorer la qualité de ses produits logiciels. Mais les utilisateurs n'aiment pas trop ça. Maintenant, la société Redmond court le risque de provoquer une nouvelle vague de mécontentement. Le fait est que Microsoft a déposé une demande de brevet pour trouver un moyen de suivre un utilisateur à l'aide d'un nouveau composant de système d'exploitation.

Le brevet est appelé « Formulation de requête via le continuum de tâches ». Il décrit une méthode qui simplifie l'échange de données entre des applications individuelles en temps réel. Selon les développeurs de cette technologie, il augmentera l'efficacité du travail de l'utilisateur avec les applications et l'ensemble du système d'exploitation.

Microsoft estime que le modèle actuel le fonctionnement du logiciel, lorsque chaque application est laissée à ses propres appareils et interagit peu avec son environnement, n'est pas très efficace. "En raison du fait que les programmes individuels ne sont pratiquement pas connectés les uns aux autres UGOm, un système informatique n'est pas qu'il amène l'utilisateur à un moment donné dans le temps », - a déclaré dans la demande de brevet.



À titre d'exemple, une entreprise cite un cas où un utilisateur prend des notes sur OneNote sur le thème de la danse. Dans le même temps, cet utilisateur recherche dans Internet Explorer des informations sur les styles de danse sur le Web. Ces deux applications ne sont généralement pas connectées l'une à l'autre et n'échangent pas d'informations, mais la société propose de les corriger.

Les développeurs de l'entreprise proposent d'améliorer l'interaction des applications individuelles en utilisant le «composant intermédiaire» ou «intermédiaire» (Mediator). Ils prévoient de faire de ce composant un programme distinct qui surveillera les actions de l'utilisateur, transférant les données collectées vers d'autres applications et appareils de la même personne. Le «Composant» surveillera non seulement le texte saisi par l'utilisateur, mais également ses autres actions. Par exemple, rechercher des chansons sur le Web ou travailler avec des images stockées sur votre disque dur. De plus, le «composant intermédiaire» collectera des données à partir d'applications telles que Word, Skype, Edge.

«Le service de recherche utilise le classement contextuel pour ajuster la note des éléments de recherche potentiels. Cette opération est effectuée pour améliorer la pertinence de la requête de recherche souhaitée. Le système d'exploitation, jouant le rôle d'un composant intermédiaire, surveille toutes les données textuelles affichées à l'utilisateur par n'importe quelle application et effectue une analyse des données pour déterminer les intentions de l'utilisateur », indique la demande de brevet.

La société affirme que cette méthode accélérera le travail de l'utilisateur avec un PC et des services de recherche. Bien sûr, pour certains utilisateurs, cette méthode ressemblera à une tentative de Microsoft d'ajouter officiellement un "grand frère" à son système d'exploitation. L '«intermédiaire» surveillera réellement toutes les actions que vous effectuez sur votre ordinateur, collectera ces informations et transférera le vidage de données vers Bing en temps réel.

Microsoft affirme une fois de plus que le système ne transmettra que des données anonymisées, mais il est peu probable que cela calme les utilisateurs qui font attention à leur anonymat sur le réseau et pensent que les données personnelles doivent rester personnelles.



La demande de brevet décrit en détail comment tout cela fonctionnera dans l'écosystème des produits Microsoft: «Par exemple, dans l'écosystème des appareils avec Windows OS, qui peut inclure deux ou plusieurs appareils, tels que Windows Phone, tablette Surface Pro, Xbox, d'exploitation le système collectera et suivra les données des applications tierces exécutées sur les mêmes gadgets, et interagira avec les systèmes d'exploitation d'autres appareils. Ceci est fait pour que le système d'exploitation de chaque appareil particulier «sache» tout ce qui se passe avec les autres appareils de l'écosystème. Les données dispersées seront collectées en un seul ensemble. "

Microsoft suggère de faire de «l'Intermédiaire» un module optionnel qui peut être installé dans l'environnement Windows 10 ou intégré dans la structure du système d'exploitation.

Le fait que Windows 10 collecte des données détaillées sur ses utilisateurs est connu depuis longtemps. En 2014, Microsoft a publié une nouvelle déclaration de confidentialité déclarant que la société peut collecter des informations sur les PC et les ordinateurs portables utilisés avec Windows 10 installé, ainsi que sur les appareils et réseaux sur lesquels ils fonctionnent. Ces données peuvent être combinées avec un ID utilisateur. Des informations sont également collectées sur les préférences d'une personne particulière, son emplacement, ses intérêts. La



plupart activement recueille assistant vocal Cortana. La liste des serveurs auxquels les données sont envoyées est longue. Cependant, s'il existe un tel besoin, vous pouvez désactiver la télémétrie dans Windows 10.

Dans le cas de la création de «l’intermédiaire», ce sera lui qui collectera et enverra les données aux serveurs de la société. Plus d'informations seront probablement envoyées. Eh bien, les utilisateurs devront décider de travailler avec "l'intermédiaire" ou non. Très probablement, après l'avènement d'un nouveau composant du système d'exploitation, les spécialistes de la sécurité des informations trouveront rapidement un moyen de désactiver la télémétrie.
