Menginstal FPV dan telemetri pada quadrocopter

Artikel ini merupakan kelanjutan dari artikel "Quadrocopter on MultiWii SE v2.5 - dari konsep ke penerbangan pertama" , di dalamnya saya akan berbagi dengan Anda pengalaman saya memasang dan mengkonfigurasi peralatan FPV pada quadrocopter untuk penerbangan orang pertama. Tertarik, saya minta kucing.





Komponen yang Digunakan:
1. Boscam 5.8GHz 200mW 8 Channel FPV Audio Video Transmitter & Receiver TS351 + RC805 2Km Receiver / Transmitter
2. Telemetri CRIUS MAVLink-OSD v2.2
3. Modul CRIUS NEO-6 GPS v3.0 + I2C-GPS NAV v1.1
4. Monitor FPV (menggunakan TV lama)
5. kamera FPV (digunakan kamera CCTV luar ruangan)

Pemancar TS351, telemetri MAVLink-OSD dan kamera terhubung ke MultiWii SE v2.5 sesuai dengan diagram di bawah ini:




Kami menghubungkan modul GPS NEO-6 ke modul I2C-GPS di konektor GPS, dan modul I2C-GPS melalui konektor FC yang kami sambungkan ke MultiWii di konektor I2C.









Memeriksa modul GPS NEO-6 GPS untuk pengoperasian: kami menghubungkan modul melalui pemrogram FTDI (GND ke GND, RX ke TX, TX ke RX, + 5V ke VCC) ke PC; jalankan program yang terinstal "u-center" pada PC (tautan ke program di akhir artikel); pilih kecepatan transfer data otomatis -> klik "Autobauding" (nomor "1"); pilih port COM dimana modul GPS NEO-6 terhubung (angka "2"); kami menunggu ... jika semuanya dilakukan dengan benar, setelah beberapa saat data dari modul GPS NEO-6 (gambar "3") akan masuk, saya memilikinya dalam 3-4 menit. kirim data.



Firmware telemetri MAVLink-OSD: sambungkan modul melalui programmer FTDI (GND ke GND, RX ke TX, TX ke RX, + 5V ke VCC) ke PC; jalankan program Arduino IDE pada PC; buka firmware "KV_Team_OSD_2.3" (tautan ke firmware di akhir artikel) - Anda perlu menjalankan file KV_Team_OSD_2.3 / KV_Team_OSD / KV_Team_OSD.ino; flashing ... Selanjutnya, Anda perlu mem-flash font; jalankan KV_Team_OSD_2.3 / KV_Team_GUI; pilih port COM dimana modul MAVLink-OSD terhubung; di blok "FONT TOOLS", tekan "BROWSE", pilih font "KVTOSD_FMap_v1_Small.mcm", klik pada "Upload" dan tunggu font untuk memuat, indikator "TX" di blok "PORT STATUS" akan berkedip dalam proses; Sekarang di blok "Kontrol OSD", klik "BACA" (baca data dari papan), buat pengaturan yang diperlukan untuk menampilkan informasi yang Anda butuhkan pada monitor, klik "MENULIS" (tulis ke papan). PS adalah penyimpangan kecil, saya mencoba lebih dari selusin firmware untuk MAVLink-OSD, tetapi di semua font tidak ingin mem-flash, apakah ini fitur papan ini, atau membantu jika saya mem-flash font dari firmware ‚ÄúKV_Team_OSD_2.3‚Äù dari baterai (12V) ke "bagian analog" papan, dan "bagian digital" papan masing-masing ditenagai oleh 5V!





Kami menghubungkan penerima RC805 ke monitor dan, jika diinginkan, kami lampirkan semuanya ke aplikasi. Ingat bahwa untuk sinyal video yang benar, saluran pada penerima / pemancar harus identik!
