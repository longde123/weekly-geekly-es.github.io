9 extensions de navigateur utiles pour les développeurs (liste pour 2020)

Comment vous simplifier la vie avec des plugins de navigateur.

Les développeurs Web délicats peuvent utiliser le navigateur au niveau du mode divin. Les extensions (plugins, modules complémentaires) sont extrêmement utiles pour améliorer l'environnement de travail et améliorer les performances d'écriture de code.

2020 n'est pas loin. De nombreuses nouvelles extensions sont apparues. J'ai compilé les extensions les plus utiles pour les développeurs web et les ai mises sur une liste.

Pas de pièce


Il existe des sites qui obligent nos processeurs à extraire des crypto-monnaies pour eux. No Coin est une petite extension de navigateur. Il est nécessaire de bloquer les mineurs comme Coinhive, qui utilisent vos ressources informatiques sans votre consentement.


No Coin est approuvé par des milliers d'utilisateurs. Cette extension vous offre un moyen sûr et fiable d'empêcher les mineurs d'utiliser votre processeur. Parfois, vous devez saisir un captcha ou un lien court qui nécessite une extraction. No Coin permet dans ce cas d'annuler le blocage des mineurs pour une durée limitée.

Minuterie Pomodoro



Pomodoro Timer Extension

Technique de Pomodoro - méthode de gestion du temps. Cette technique originale comprend six étapes:

  1. Décidez de la tâche que vous ferez.
  2. Démarrez la minuterie Pomodoro (généralement environ 25 minutes).
  3. Travaillez sur la tâche sans être distrait par quoi que ce soit.
  4. Arrêtez de travailler lorsque la minuterie sonne et cochez la case sur un morceau de papier.
  5. Si vous avez moins de quatre coches, faites une courte pause (de 3 à 5 minutes) et passez à la deuxième étape.
  6. Après chaque 4ème "tomate", faites une longue pause (15-30 minutes). Réduisez le nombre de graduations à zéro et passez au premier point.

Pendant que vous lisez cet article, j'utilise Pomodoro. Pomodoro est un moyen pratique de mettre les choses en ordre. Il aide à apprendre à se concentrer et à gérer son temps. Si vous voulez apprendre à vous concentrer, je recommande ce livre . Elle m'a aidé.

Inspecteur Meta SEO


Habituellement, nous ne voyons pas de métadonnées lorsque nous utilisons Internet. Cette extension vous aide à découvrir et à vérifier les métadonnées que vous trouvez sur les pages Web.


Vérifiez vos balises META pour assurer un bon classement SEO

L'extension est principalement utilisée par les développeurs Web qui doivent vérifier le code HTML de leur site conformément aux recommandations de Google pour les webmasters. Il est également utile à quiconque souhaite voir le contenu d'une page, qui n'est généralement pas visible.

Les alertes sont affichées lorsque les métadonnées ne respectent pas certaines plages - lorsque la balise de description est trop courte ou trop longue.

Le référencement est une langue distincte. Je recommande de lire le livre "3 mois au n ° 1: le livre de référence SEO" No-Nonsense "2019 pour trouver votre site Web sur Google . " Il contient les meilleures pratiques et des conseils SEO.
