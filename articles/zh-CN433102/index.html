为什么科学家相信第九颗行星不存在


艺术家将第九行星看成是一个冰巨人，盖过银河系的中心，并在背景中描绘了一个类似太阳的恒星。 海王星的轨道显示为围绕太阳的小椭圆形。

自从关于我们自己的太空法庭的最有趣的假设之一出现以来，已经过去了将近三年的时间：在我们的太阳系中，海王星的轨道远远超出了海王星的轨道， 可能存在另一颗甚至比地球更大的行星。 与早先在柯伊伯带发现的微小世界（如冥王星或埃里斯）不同 ，这可能是一个比地球大的世界，其质量可能是地球大小的十倍，并负责将奇怪的物体提供给我们可见的物体。

正如康斯坦丁·巴蒂金（Konstantin Batygin）和迈克尔·布朗（Michael Brown）所建议的那样，应该证明存在它的证据更多，其中一些证据已经开始出现。 但是，大多数科学家不同意这一证据。 他们声称此数据有偏差。 如果考虑到这一点，则不需要第九颗行星 。


循环时间最长的许多海王星天体 （TNOs）的黄道在纬度和经度上的对齐可能是巧合，这是观测结果有偏差或新物理现象的迹象。

柯伊伯带是我们发现的最大的远距离物体群的所在地。 在研究它们时，人们期望它们的轨道是随机的，斜率和最小逼近点应在所有方向上均等分布。

但是，根据可用的观察，最远物体的轨道沿一个特定方向移动，并沿一个方向倾斜。 如果一个或两个对象的行为如此，则可以归因于偶然性，但是我们有六个这样的对象。 随机符合的几率约为0.0001％。 天文学家康斯坦丁·巴蒂金（Konstantin Batygin）和迈克尔·布朗（Michael Brown）提出了一种新的激进理论：存在一个超遥远的第九颗行星，它比地球重，但比天王星和海王星轻，这使这些天体进入了新轨道。


著名的sednoids的轨道，以及据称的第九行星。 在遥远的未来，即使太阳变成了红色巨人，第九行星的存在也引起了很大争议，它无法预热到足以成为宜居之地。

如果得到证实，这个惊人的想法将产生一些有趣的含义。 特别是，应执行以下操作：

  • 应当出现过多的物体，由于引力相互作用，其轨道将大大扩展。
  • 这些物体在第九行星的影响下的轨道和轨道平面将以某种方式倾斜。
  • 必须有一个很小但不为零的物体簇，其轨道与该过量人口的轨道严格相反。
  • 还必须有第九行星本身，等待发现。

随着进一步研究的到来，Batygin和Brown指出了一些对象-一个在这里，另一个在这里，在接下来的工作中还有另外两个-作为前三点的证据。 但是到目前为止，第九颗行星尚无法直接观察。


2016年发现的柯伊伯带最遥远的六个物体的轨道异常靠近，这可能表明存在第九个行星，其重力会影响其运动。

这并不奇怪！ 即使第九颗行星是真实的且很大，它也将是极其暗淡的，因为它与太阳之间的距离是预计的。 您可以决定，如果它比天王星更远十倍并且具有大致相同的大小，则它应该只比天王星少100倍，因为亮度与距离的平方成反比。 但是从我们的角度来看，太阳光遭受了这个问题的两倍：到达如此遥远世界的太阳光比到达更近世界的光要暗100倍，然后它还会反射，因此它必须进入是到达地球之前的两倍。 实际上，我们看到的光像1 / r 4一样在减少，而不是根据定律1 / r 2减少，这就是为什么如此遥远的世界很难看到的原因。


可以通过特殊观察来检测非常暗的物体，但是由于反射太阳光的问题，要在太阳系中找到一个小的，暗的和较远的物体更加困难。 如果物体的距离是另一物体的两倍，则光线必须到达该物体的距离是它的两倍，这意味着该物体将仅到达光线的1/4，然后在经过两倍的距离后必须返回，即为1 / 16原始亮度。 在这种情况下，亮度的比率1 / r 4具有灾难性的后果。

从理论的角度来看，这个想法很棒。 观察一个人如何借助单个新对象来解释一大堆没有意义的观察总是很有趣的。 但是，就像有许多绝妙的主意的情况一样，事实证明它仍然是错误的。 从六个超长对象的行为有点不正常的事实出发，并不能因此得出结论，没有六百万个超长对象能够正常工作-但我们尚未看到它们。

简而言之，我们需要确保我们观察到的证据是那里存在的物体的特征-在这一点上，想法绊倒了。


从Pan-STARRS1夏威夷天文台可见的整个天空的压缩图像是从半百万个独立帧中获得的，快门速度为45秒。 但是，从中获得第九行星数据的观测结果并非如此均匀地分布在天空中。

到目前为止，必须依靠巴蒂金和布朗提出的间接证据。 他们总共宣布了十个与他们的预测相符的天体。 这是令人印象深刻的，它代表了对原始六种方法的改进。

但是，他们没有使用来自整个天空的勘测数据来搜索这些物体：此类勘测（例如Pan-STARRS）看起来不够深入。 第九颗行星将负责的跨海王星天体及其奇怪的轨道应位于天空中的某个位置。 如果要查找这些对象，则可以在某些地方查找它们。


2015 RR245轨道与天然气巨人和其他著名的柯伊伯带天体的比较。 请注意，由于地球绕太阳运动，因此图像会因天气，季节和天空可见部分而发生变化。 结果，观察的图像可能会失真。

这是正常的，但是巴廷金-布朗理论不仅基于这些物体存在的事实，而且还基于它们存在的事实，并且它们的蓄积几乎不是偶然的。

但是发生拥塞的可能性是多少？ 它在很大程度上取决于几个因素，例如，您在何处精确地进行观测以及具有何种敏感性。 当然，如果大部分时间都在寻找可以找到对象集群的位置，那么您会发现更多的对象。 您只会在那儿呆更长的时间，并且通常能找到最多的东西。 这并不意味着正在发生异常情况，例如异常簇。

实际上，极有可能在那里没有发生异常情况。 您很可能是观察偏差的受害者。


现有的技术可以让您找到非常钝，很冷或移动缓慢的物体，但这全取决于在这些物体足够长的地方寻找它们。 在照片中-WISE任务发现了一颗罕见的超冷矮星，标有红色。 寻找第九颗行星的这种方法可能不是最佳选择。

通过不同深度的各种观测结果确定了由巴蒂金和布朗确定的十个物体，重要的是，从未评估观测偏差的可能影响。 想象一下，您有一个位于地球赤道附近的望远镜，每天晚上您都看着夜空，试图尽可能多地看到尽可能深的距离。 如果您一年365天都有清晰可见的晴朗天空，那么您同样可以探索整个天空。 但是，这不会发生。 相反：

  • 一年中的某些时期受恶劣天气的影响更大。
  • 在某些时期，大气湍流和恶劣的条件更为普遍。
  • 天空的某些部分（例如银河平面）被污染并且不允许检测TNO。

依此类推。 底线是，如果您优先考虑天空中的两个特定区域（如您期望的那样，物体应在该区域中堆积），则会在其中发现物体的堆积。 也许您只是因为正在寻找而找到它们。


来自柯伊伯带的物体的三维轨道，受到第九行星的影响。 正如迈克尔·布朗（Michael Brown）所说：“第九行星假说预测了轨道垂直于太阳系的遥远物体。 然后五分钟后找到。” 但是，只能找到它们是因为该区域中存在良好的数据。

当然，Batygin和Brown的团队今天已经打开了10个演示集群的对象。 但是它们是否证明了第九行星的存在？

有一种直接的方法可以验证效果的真实性：进行无此偏见的特殊观察，或至少对其进行数字评估。 正在对太阳系中海王星以外的世界进行主要观察： 太阳系外源调查 （ OSSOS ）。 在其框架内，已经发现了800多个物体，并且在天空的四个特定区域进行了四年的搜索。 （浪费了很多时间来寻找有形的运动并测量到目前为止离太阳很远的世界的轨道参数！）在数百个这样的天体中，有八个具有长周期性，这证明了它对第九颗行星的赞成或反对。


在OSSOS研究中发现的所有长周期海王星天体中，只有一个（蓝色标记）的参数与Batygin / Brown理论一致。

结果是肯定的和令人失望的。 在研究之前，在有或没有海王星之外的第九大行星的情况下进行了模拟，从中可以得出结论，该结果应表明该行星的存在，而该结果应反对。 这是其中八个对象的发现：

  • 在八个开放的OSSOS对象中，轨道位于不同的角度。
  • 观测到的轨道在统计上与随机轨道一致。
  • OSSOS在先前的样本中未找到模式。
  • 其中一个对象与两个建议的群集成直角移动。
  • 轨道并不太紧密。


从理论上讲，第九行星类似于巨蟹座e的系外行星 55 ，其半径大约是地球半径的两倍，质量大约是八倍。 但是一项新的研究拒绝了外部太阳系中存在这样一个世界。

最重要的是，他们的发现与没有第九颗行星的事实相符，并且有关其存在的论据被他们的研究削弱了。 特别是，在这项新的，无偏见的研究中 ，早先的研究（例如Batygin和Brown以及Trujillo和Shepard）并不存在太空轨道取向的簇（由许多变量ω和Ω定义），根本不存在。

在OSSOS样本中，我们没有找到有关ω团簇的任何证据，ω团簇是存在额外行星的假设的基础。
