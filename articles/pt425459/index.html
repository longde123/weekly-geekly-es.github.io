A atualização para o Windows 1809 (às vezes) destrói todos os arquivos em um perfil

Os colegas de uma oficina estrangeira alertam ( 1 , 2 ) que há um erro sério no instalador do Windows 1809.

No momento, sabe-se que você perderá arquivos do seu perfil (Meus documentos, Imagens, Área de trabalho etc.) se tiver ativado o redirecionamento de pasta do perfil para outro local ou se eles foram sincronizados com o OneDrive.

UPD # 1
Os usuários do Spiceworks descobriram que a política de limpeza para perfis antigos não funciona corretamente

UPD # 2
No momento, foi confirmada a operação incorreta da política de grupo para limpar perfis antigos - as situações descritas no reddit ou answers.microsoft.com são aparentemente causadas por esse bug específico. Às 23:59 de 05/10/2018, não há uma única pessoa no assunto que tenha sofrido com essa situação. No entanto, o caso da atualização nº 1 é facilmente repetido em qualquer máquina virtual.

UPD # 3
Como esperado, a assembléia foi convocada.

Assunto no Microsoft Answers.
