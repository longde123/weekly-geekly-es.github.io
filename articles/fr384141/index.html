Windows à travers les années: trente ans le système d'exploitation Microsoft

La première version du shell graphique Windows pour MS-DOS est sortie il y a près de trente ans, il ne reste que quelques mois avant l'anniversaire. Après des décennies, la part de Windows sur le marché des ordinateurs personnels dépasse les quatre-vingt-dix pour cent. Plus de la moitié de tous les ordinateurs de bureau fonctionnent sous Windows 7.

Souvenons-nous des versions de Windows que nous avons vues depuis 1985.



La première version du shell graphique Windows est sortie en 1985: le 20 novembre, elle est apparue sur les chaînes de magasins. Pour éviter des problèmes juridiques avec Mac OS, les développeurs n'ont pas utilisé le panier et les fenêtres ne pouvaient pas se chevaucher. Cette version de Windows n'avait pas de barre des tâches en bas de l'écran. "Add-on inutile sur DOS" comprenait un jeu - Reversi.



Deux ans plus tard, en 1987, Microsoft a publié une version de Windows 2.0. Ici sont apparues les applications Paint qui nous sont familières depuis l'enfance, les montres, le gestionnaire de fichiers MS-DOS et le terminal. Peu de temps après la version 2.0, la société a d'abord porté Word et Excel sur Windows.



Windows 3.0 en 1990 comprenait le Gestionnaire de programmes, la prise en charge de VGA et de la mémoire virtuelle, une nouvelle conception «3D» et le jeu «The Scarf». Le jeu était nécessaire non pas pour le divertissement des employés de bureau, mais pour former les utilisateurs aux techniques d' utilisation de la souris .



En 1992, Microsoft a publié Windows 3.1, ajoutant des polices TrueType évolutives et le jeu Démineur au système pour entraîner la précision de frapper l'écran avec le curseur. Pour la première fois, la prise en charge du contenu multimédia est apparue dans le système.



Windows for Workgroups 3.1 est sorti en 1992. Il comprenait un support natif pour les réseaux - en particulier les réseaux locaux LAN, gagnant rapidement en popularité dans les affaires de l'époque.

1993 peut être appelée l'année de naissance du système d'exploitation Windows. Avant Windows NT 3.1, ce qu'on appelait Windows était un shell graphique pour MS-DOS. Désormais, Microsoft a introduit un système d'exploitation 32 bits conçu pour les postes de travail modernes. Extérieurement, le système d'exploitation ne diffère pas de Windows 3.1.



En août 2015, Windows 95 a eu 20 ans . Les systèmes informatiques sont devenus accessibles non seulement aux geeks, mais aussi aux femmes au foyer. Le bouton Démarrer et la barre des tâches ont simplifié l'utilisation du système.



Windows NT 4.0 a combiné une interface similaire à Windows 95 avec le solide noyau Windows NT. Ce système est devenu le système d'exploitation le plus populaire de Microsoft pour les prochaines années.



Windows CE est un système d'exploitation pour les ordinateurs de poche, les smartphones et les systèmes embarqués. Il a été publié en novembre 1996, à l'époque il s'agissait d'une version "allégée" du système d'exploitation de bureau. La dernière version du système - 7.0 - est sortie en mars 2011.



Le navigateur Web Internet Explorer est devenu un élément indispensable de Windows 98, sorti en 1998. Le système dispose d'une barre d'outils pour le lancement rapide des applications et un support natif pour USB. Le système est devenu très populaire, mais pas le plus stable.



Windows 2000 est la version 5.0 du système d'exploitation Windows NT. La gamme a été suivie par XP, Vista, Windows 7 et 8. La première version bêta publique du système est apparue en 1997, la société a déployé la version finale le 17 février 2000. Le système a pris en charge la partie serveur d'Active Directory, le système de fichiers NTFS 3.0 et l'EFS chiffré, et a reçu l'interface utilisateur avec Active Desktop.

Pour utiliser Windows 2000, vous deviez avoir un processeur de 133 MHz, de 32 mégaoctets de RAM et de 2 gigaoctets de mémoire sur votre disque dur.



Sorti en 2000, Windows ME a été critiqué par les critiques et l'utilisateur comme extrêmement instable, parlant de gels et de plantages constants.



Le premier ordinateur que j'ai acheté fonctionnait sous Windows XP. J'ai obtenu un ordinateur en 2006 et Microsoft a sorti XP en 2001. Windows XP est devenu le système d'exploitation Microsoft le plus populaire de son époque.

En juin 2008, Microsoft a annoncé qu'il prendrait en charge XP jusqu'en 2014. Le soutien général au système a pris fin en 2009, prolongé - le 8 avril 2014.



En 2007, Microsoft a lancé Windows Vista en réponse au système d'exploitation Mac OS X. Des fenêtres translucides, de grandes icônes et un nouveau menu Démarrer n'ont pas impressionné les critiques.



Le système d'exploitation serveur Windows Server 2008 remplace Windows Server 2003 en tant que représentant de la nouvelle génération de la famille de systèmes Vista.



Windows 7 est l'une des versions les plus réussies de Windows. Laissant la belle interface (Aero) et les icônes, Microsoft a supprimé de nombreux bugs et a rendu le système plus fiable.

Six ans se sont écoulés entre la sortie de Windows XP et Windows Vista, et seulement trois ans entre la sortie de Vista et 7.



En 2012, Microsoft a présenté Windows 8 avec l'interface en mosaïque Metro. L'interface apparaît d'abord après le démarrage du système et est similaire au bureau. Heureusement, le système d'exploitation a continué de prendre en charge le bureau Windows classique, mais a supprimé le bouton Démarrer. Le système est conçu pour les écrans tactiles, bien que vous puissiez l'utiliser sur des ordinateurs avec des moniteurs conventionnels.



Windows 8.1 - Mise à jour pour Windows 8 à partir de 2014. Maintenant, les utilisateurs au démarrage peuvent voir l'écran de travail au lieu de Metro, le bouton "Démarrer" est apparu, bien que sans l'ancien menu de démarrage.



En 2015, Microsoft a présenté Windows 10, qui peut être mis à niveau à partir de Windows 7 et Windows 8. Microsoft a renvoyé le menu Démarrer.

Il y a eu des moments désagréables pour l'utilisateur final: en août 2015, les trackers torrent ont commencé à interdire les utilisateurs de Windows 10 sur les ordinateurs, car le système envoie une quantité excessive de données aux serveurs Microsoft.
