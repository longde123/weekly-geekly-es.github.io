Desain dan pembuatan teleskop


Mereka mengatakan bahwa jika seseorang telah memperoleh teleskop dan menggunakannya untuk pengamatan, maka setelah beberapa saat dia pasti akan menginginkan teleskop yang lebih baik. Jadi itu terjadi pada saya. Saya memiliki teleskop PowerSeeker 127mm, tetapi saya menginginkan yang terbaik. Untuk alasan ini, saya bahkan akan mempertajam cermin saya, tetapi kemudian menemukan bahwa pada AliExpress Anda dapat membeli cermin 200mm dengan harga murah. Cermin utama bersama dengan diagonal keluar sekitar 6.000 rubel dengan pengiriman. Ini sangat murah. Gambar di atas hanyalah cermin, baru saja tiba melalui pos dari Cina.

Saya membeli sesuatu, tapi itu bukan teleskop yang sudah jadi. Butuh pipa, dudukan cermin utama, diagonal, pelurusan, fokus. Bagaimana cara membuatnya sendiri? Saya memutuskan bahwa jika mungkin saya akan mencetak rincian yang diperlukan pada printer 3D. Tetapi untuk ini mereka masih perlu dirancang. Atau temukan desain di Internet.

Jika Anda mencari di Internet, tentu saja, "teleskop sudah dicetak." Berikut adalah contoh proyek:

  1. Menatap Tata Surya dengan Teleskop Raspberry Pi yang Dicetak 3D
  2. Open Source DIY Telescope Fitur Utama Raspberry Pi dan Bagian Cetak 3D

Karena berbagai alasan, saya tidak terlalu menyukai proyek ini. Saya memutuskan bahwa saya akan merancang sendiri bagian-bagian dari teleskop. Agar kehidupan tidak tampak seperti madu, untuk beberapa alasan saya memutuskan bahwa saya akan melakukan proyek di FreeCAD. Sampai saat itu, saya sudah melakukan sesuatu dalam Compass 3D (saya mengerjakan Marble-Machine ). Dan sekarang saya pikir FreeCAD seharusnya tidak lebih rumit ... Sebenarnya, saya harus menonton beberapa tutorial FreeCAD di youtube sebelum saya bisa melakukan sesuatu yang bermakna. Mendesain dalam FreeCAD bagi saya tampaknya sangat berbeda dari mendesain di Compass, ia menggunakan teknologi konstanta baru untuk saya (saya tiba-tiba tidak tahu dan di Compass itu mungkin, tetapi saya tidak tahu?). Batasan menentukan semua ukuran yang diperlukan: panjang segmen, paralelisme atau tegak lurus, sudut, jari-jari busur, secara harfiah segalanya. Dengan membatasi, pemecah FreeCAD menghitung apakah itu dapat membangun bentuk yang diberikan atau tidak. Jika tidak, maka tampaknya tidak ada cukup parameter klarifikasi yang perlu ditambahkan. Itu menarik dan tidak biasa bagi saya. Namun, secara umum, saya menyukai gagasan konstanta.

Saya melakukan perhitungan teleskop dalam kalkulator online " Perhitungan teleskop sistem Newton ."



Gambar di atas menunjukkan beberapa parameter yang diperlukan untuk membangun teleskop. Untuk setiap nilai signifikan, kalkulator menghitung nilai yang diperlukan, Anda hanya perlu mengatur yang pertama: kepadatan kaca cermin utama, diameter, bevel, diameter dan panjang pipa, dan sebagainya.

Perhitungannya membuat saya melihat lebih dekat pada tugas itu. Saya sudah membaca buku Sikoruk "Teleskop untuk Pecinta Astronomi," dan secara umum saya banyak membayangkan. Jadi, saya tahu cermin besar dan tipis dipasang dengan benar sesuai dengan skema bongkar khusus. Sebagai contoh, untuk cermin 200 mm saya yang baru dengan berat 1306 gram, ternyata seharusnya terletak pada enam titik. Program membuat perhitungan sehingga untuk setiap titik bongkar, berat yang sama dari bagian cermin jatuh, sedangkan cermin berat tidak boleh secara kritis menekuk di bawah beratnya sendiri, karena menekuk cermin adalah distorsi.



Bahkan cermin yang lebih besar dibongkar pada titik 9, 18 atau lebih. Bagaimana cara membuat enam titik buangan ini secara fisik? Dengan enam titik bongkar, "rocker" khusus dibuat, dua titik cermin didasarkan pada satu titik - sumbu rotasi rocker.

Saya harus segera mengatakan bahwa dari tautan di atas, "Teleskop DIY Sumber Terbuka .." yang sama tidak menggunakan 6 titik bongkar, tetapi hanya menggunakan tiga. Saya ingin melakukan segalanya dengan lebih benar, walaupun, tentu saja, saya sama sekali tidak memiliki kepastian bahwa saya telah berhasil. Saya berharap tes nyata akan menunjukkan apakah saya benar atau tidak.

Saya mulai mendesain dengan bagian paling besar dari pemegang cermin utama. Jujur, saya tidak suka mencetak bagian besar pada printer 3D. Ini sangat lama dan tidak selalu berhasil, misalnya, listrik mungkin tiba-tiba mati. Selain itu, bagian besar ini harus kuat, dan karena itu harus memiliki tingkat pengisian yang besar. Ini saya akan mencetak ini untuk waktu yang sangat lama. Karena itu, sejak awal, saya berpikir bahwa saya akan memesan komponen besar di pabrik dengan pemotong penggilingan CNC, dan saya akan melakukan sisanya sendiri.
