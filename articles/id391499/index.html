Remix OS untuk beta PC: cara mengatur dual-boot dengan Windows dan memulai dengan OS baru



Geektimes telah berulang kali menulis tentang Remix OS, OS desktop berbasis Android. Pada tanggal 1 Maret, versi beta dari sistem operasi ini muncul, yang sudah sangat memungkinkan untuk ditangani. Secara umum, ini adalah versi khusus Android yang terlihat dan berfungsi seperti sistem operasi lengkap untuk komputer pribadi. Ini pertama kali dirilis pada 2015, dengan dukungan minimal untuk beberapa perangkat. Pada bulan Januari, versi alpha dari Remix OS dirilis, para pengembang membuatnya tersedia untuk semua orang, membuka kemampuan untuk mem-boot sistem dari drive USB.

Dalam versi beta, dual-boot muncul, dengan kemampuan untuk beralih di antara dua OS - Remix OS itu sendiri dan Windows. Anda dapat menginstal OS di hard drive dan SSD, yang memberikan kinerja platform yang cukup tinggi. Sekarang penginstal Remix OS untuk PC mendukung bekerja dengan Windows 7 dan versi OS yang lebih baru. Versi untuk Mac akan segera tersedia. Jika OS lelah, menghapusnya bahkan lebih mudah daripada menginstalnya.



Pembaruan sekarang datang secara teratur, dukungan untuk pembaruan over-the-air sudah ada. Ada berita lain, termasuk peralihan ke kernel Linux versi 4.4 untuk kompatibilitas yang lebih baik dengan prosesor yang lebih baru, dukungan untuk video 1080p dari YouTube dan Netflix, bekerja dengan drive USB hingga 64 GB dan banyak lagi.

Kenapa tidak Mendukung Play Store. Karena Remix OS tidak disertifikasi oleh Google, tidak ada akses default ke pasar. Tetapi Anda dapat menambahkannya, lebih lanjut tentang itu di bawah ini. Pertama, mari kita bicara tentang menginstal Remix OS itu sendiri.



Tidak ingin menginstal Remix OS pada SSD atau hard drive? Tidak masalah Anda dapat bekerja dengan OS langsung dari USB flash drive, menjadikannya dapat di-boot. Namun tetap saja, kinerja OS dalam hal ini akan lebih rendah daripada jika Anda menaruhnya di HDD atau SSD.



Bagaimana cara menginstal Remix OS pada HDD atau SSD



1. Pertama, unduh versi OS terbaru dari sini . Ada build 32-bit dan 64-bit untuk PC yang sesuai.

2. Buka kemasan arsip dan dapatkan tiga file: buku teks dengan instruksi, installer dan ISO, gambar disk.

3. Buka Remix OS Installation Tool.exe dan verifikasi bahwa lokasi pemasangan sudah benar (huruf drive, dll.).



4. Klik Next dan pilih gambar disk yang baru saja dimuat. Ini akan menjadi sesuatu seperti Remix OS untuk PC_x86_64-B2016xxxxxx.iso.

5. Kami memulai instalasi dan menunggu beberapa menit hingga muncul pesan tentang perlunya me-restart PC untuk menyelesaikan instalasi.

6. Setelah reboot terjadi, Anda dapat melihat layar dengan saran untuk memilih dua sistem.



7. Kami sedang menunggu.

Langkah selanjutnya akan memakan waktu 15 hingga 30 menit, tergantung pada kinerja PC Anda. Remix OS membuat area boot sendiri 8 GB dan menginstal OS di area ini.

8. Setelah semuanya siap, Remix OS akan boot untuk pertama kalinya, dan Anda dapat memulai.

9. Jika Anda perlu kembali ke Windows, pilih ikon shutdown, lalu opsi reboot. Setelah itu, alih-alih Remix OS, pada layar pemilihan, klik Windows.

Apakah mungkin untuk mendapatkan lebih dari 8 GB?


Pemasang resmi Remix OS PC membutuhkan 8 GB untuk kebutuhannya. Ini cukup untuk menginstal berbagai aplikasi yang mungkin Anda perlukan untuk pekerjaan atau hiburan.



Tetapi jika Anda tiba-tiba membutuhkan lebih banyak ruang, maka Anda dapat menambahkannya .

Cara menghapus Remix OS


Jika tiba-tiba terjadi kesalahan, atau hanya ingin menghapus OS Remix, ini dapat dilakukan dengan sangat cepat.

Awalnya, jalankan kembali penginstal. Ini menunjukkan bahwa sistem sudah diinstal pada PC, dan menawarkan untuk menghapus OS Remix. Kami setuju, dan setelah beberapa detik OS dihapus.

Anda tidak dapat bekerja dengan penginstal, tetapi menghapus semuanya secara manual. Di bilah alamat manajer file, pilih C: \ Remix OS \ , dan jalankan file UninstallRemixOS.exe file . Kemudian semuanya terjadi dalam mode otomatis.
