Terminal Fun: 10 klassische Konsolenwitze

Die Hälfte der langen Neujahrsferien ist vorbei und jetzt ist die Zeit gekommen, sich an die bärtigen Witze der Unix-Admins zu erinnern.



Textshells in UNIX-ähnlichen Betriebssystemen eignen sich nicht nur zum Ausführen interner Befehle, zum Ausführen verschiedener Dienstprogramme und zum Schreiben von Skripten. Die Ergebnisse einiger Programme können Gäste amüsieren, die im Licht herumwandern. Die Redaktion des RUVDS-Blogs gratuliert den Lesern zu Weihnachten und bietet an, sich an die klassischen Konsolenwitze zu erinnern, die viele Generationen von Systemadministratoren begeistern.

1. Auch ein einfaches Make kann nicht nur Programme aus dem Quellcode kompilieren. Es gibt andere Optionen für die Verwendung dieses Dienstprogramms, aber zuerst müssen Sie das Paket installieren, wenn es sich nicht auf Ihrem System befindet. In Ubuntu geschieht dies mit einem einzigen Befehl: 

apt-get install make

2. Vergessen Sie nicht unser Lieblingsdienstprogramm who . Mit seiner Hilfe können Sie die FAQ des Administrators durchgehen:



3. Ein weiterer unsterblicher Klassiker ist das cowsay Programm. Es ist nicht in der Standardkonfiguration enthalten, daher müssen Sie Folgendes installieren: 

 apt-get install cowsay



4. Alle Benutzer von UNIX-Systemen kennen das Dienstprogramm ls , das Programm sl (stream locomotive) ist jedoch weniger bekannt. 

 apt-get install sl

Wenn Sie es installieren und ausführen, fährt ein Zug durch die Konsole (fast der Film der Brüder Lumiere in der ASCII-Tabelle).





5. Mit dem banalen Telnet in ASCII-Grafiken können Sie den vierten Teil von Star Wars sehen, der von Enthusiasten gezeichnet wurde. 

 telnet towel.blinkenlights.nl



6. Die nächste Nummer des Programms ist das berühmte Utility-Vermögen, mit oder ohne Quotes. Sie kann beispielsweise über das Shell-Initialisierungsskript ausgeführt werden, um kleine Vorhersagen zu erhalten, wenn sich ein Benutzer anmeldet - so etwas wie ein Konsolenhoroskop (in einigen Systemen erfolgt dies standardmäßig). Manchmal fallen Glückskurse in die Top Ten, aber zuerst müssen Sie das Paket installieren: 

 apt-get install fortune.



7. Der integrierte Befehl yes macht nichts Besonderes. Er wiederholt dasselbe in einer Endlosschleife, bis der Benutzer es satt hat: 

 yes Merry Christmas



8. Wenn Sie eine schöne Phrase in ASCII-Grafiken schreiben möchten, müssen Sie nicht viel Zeit für harte Arbeit aufwenden. Die Macher des Toilettenprogramms haben sich bereits um alles gekümmert: 

 apt-get install toilet toilet Merry Christmas



Mit Hilfe zusätzlicher Parameter kann die Beschriftung beispielsweise so schöner gestaltet werden: 

 toilet -f mono12 -F metal Merry Christmas



9. Fans der Matrix sollten ihre bevorzugten grünen Animationen in der Textkonsole starten: 

 apt-get install cmatrix cmatrix



10. Die letzte Ausgabe ist ein weiteres schönes Demo-Programm: 

 apt-get install bb bb



In einer kurzen Notiz haben wir nur über 10 klassische Konsolenspiele gesprochen. Tatsächlich gibt es noch viel mehr, und wenn Sie ein wenig Phantasie beifügen und ein Skript schreiben, leuchtet die langweilige Textoberfläche wie ein Weihnachtsbaum.
