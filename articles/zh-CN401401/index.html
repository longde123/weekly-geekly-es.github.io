购买二手MacBook时存在欺诈行为：“但是他们说，这个数字不真实！”


所有Apple计算机都具有类似的设计。 同一系列的许多型号无法通过外部符号加以区分。 欺诈者夸大发行年份或技术规格，以销售更昂贵的Mac。 我已经担任Apple电脑维修技术员很多年了，而且我也看到过此类欺诈案件。 每个对购买二手Mac时如何不被欺骗感兴趣的人，我都请您养猫。

苹果电脑识别


MacBook型号识别系统并不简单；它包含多个参数。 我将以我的Mac为例进行说明。

  • 系列：MacBook Pro
  • 型号名称：MacBook Pro Retina 15英寸
  • 模特年：2012年中
  • 型号：A1398
  • 型号标识符：MacBookPro10,1
  • 型号：MC976LL / A

在机壳的整个清单（底盖）上，只写有型号-A1398。 实际上，这仅仅是案例的模型。 该号码不包含有关Mac规格的信息。
A1398是15英寸MacBook Pro Retina，从2012年到2016年生产，具有5种不同装饰水平的型号。

生产的型号年份决定了技术特征，因此决定了二手Mac的价格。 发行年份可以在系统中查看；它与型号标识符关联。

决定价格的另一个参数是整套参数：

  • RAM容量
  • 储存容量（ssd / hdd），
  • 处理器类型和频率，
  • 图形。

工厂设备中的型号已加密（例如，MC976LL / A）。 但是这篇文章只写在盒子上。 系统中没有文章信息。 可以在Apple网站或其他资源上按序列号找到它。 序列号结合了所有识别参数。

序列号


序列号对于每个Apple设备都是唯一的。 该系列绝对可以识别任何罂粟。 它是加密的：

  • 组装工厂
  • 大会的年和周
  • 型号和设备

序列号刻在底盖上。 但是盖子可以更换，或打断号码。 您可以编造一个故事，说“封面改成用过，所以那里的数字不正确”。 可以将其更换为新的盖子，然后无需序列号。

→ 这篇文章已经描述了类似的情况。

序列号写在“关于此罂粟”系统中。 您可能已经从帖子开头的屏幕快照中了解到，您不能总是信任他。

序列号伪造


序列号在EFI固件代码中注册在EEPROM存储器芯片中。 它以一组字符的形式显式存储在此处，并且不进行任何加密。


可以拆卸MacBook，焊接微电路，将其插入编程器，然后将其改写为更“高级”的高端机顶盒。 该数字可以用几乎所有的字符集代替，它们将显示在“关于此罂粟”系统中。

实验的结果是帖子标题中的屏幕截图。

我不排除可以通过编程方式在EFI中重写序列号。 有一个服务实用程序Apple Blank Board Serializer，如果序列号为空，它会规定序列号。 Apple发送的维修板带有空序列号。 但随后在更换后，将注册特定案例的序列号。 但是此实用程序不允许更改序列号。

意见：我承认可以修改此实用程序，以便您可以更改已注册的序列号。

我进行实验的计算机是一个有趣的例子。 配备i7处理器的MD314高端配置是2011年底的MasBook Pro 13。 处理器出现故障，我将其更换，将可用的产品密封-2011年初以较年轻的配置从MacBook Pro 13获得i5 2.3 GHz。 结果是被迫降级。

如果我们在其上运行Apple Hardware Test，则此处是此类信息。



如果我们突破序列号，那么我们将看到应该有一个i7 2.8 GHz处理器。


屏幕截图取自www.appleserialnumberinfo.com/Desktop/index.php

顺便说一句，一个很好的站点，用于通过罂粟的序列号确定技术特征和工厂配置。

维修的结果是，我有了现成的欺诈选择。 年轻的模特从上往下数。 欺诈者只是做相反的事情，在最初的初级配置中，他们从顶部开始缝制序列号。 或完全更改固件。

除了序列号外，型号和生产年份（MacBook Pro 13英寸，2011年末）还注册在EFI固件中，即也可以进行修改。 带有型号标识符（MacBookPro8,1）的情况更为复杂，它已注册在SMC多控制器的固件中。 使用该芯片要困难得多。

使用编程器更改EFI固件中的序列号是一种硬性方法。 并非每个人都能做到。 它将需要焊接设备，程序员和知道如何使用它的人。 操作更为普遍。 这是Mac OS系统的修改。

计算机的所有特征都在“关于此罂粟”系统中注册。 但是可惜的是，这些信息也不能总是被信任。 有些程序和黑客可以更改此信息。 已经有关于此的帖子：

专家将立即对2014年的违规行为和Intel Hd 4000图形（2012年出现的第三代Ivy Bridge Intel Core i处理器中的集成图形）感到震惊。

实际上，这是2012年中的MacBook Air11。仅更改了发布年份。 但这足以欺骗买家约10公斤（两种型号之间的市场差额）。 可以通过破坏序列号或比较这两种模型的特征来了解欺诈行为。

我看到了一个更昂贵的骗局。 最年轻的配置的MacBook Pro Retina 15 2012在顶部购买了14年。 系统中的所有内容都已更改（制造年份，处理器频率，内存大小，存储容量，序列号）。 多付了约50吨。

购买时如何不让自己上当？


现在，让我们继续进行最主要的事情-如何处理所有这些。 您不能信任序列号，也不可以信任“关于此罂粟”系统中的信息。

那该如何生活？

在类似的文章中，我在上面放置了链接，这些作者除了按序列号打孔之外，建议运行Apple Hardware Test实用程序。

通过Apple Hardware Test中的硬件配置文件进行检查是一种可靠的方法，但不是通用方法。 让我解释一下：Apple Hardware Test是一个内置实用程序，可安装在硬盘驱动器上的所有新Mac上。 要启动它，您需要在打开时按住D按钮。 但是，如果磁盘已更改或格式化，则它将仅在支持Wi-Fi引导的2011年型号上启动。 要在某些型号上通过网络启动，请按ALT +D。 好了，您将需要网络连接。

自2014年以来，在罂粟中，该实用程序已无法用于验证。 经过简化，现在没有选项卡“设备配置文件”。 该实用程序将仅显示EFI固件中的测试结果和序列号。 我们记得，这个数字可以更改。



一种简单的通用检查方法


所有Mac识别参数都可以用一种或另一种方式替换。 仅替换铁的特性（处理器频率，RAM大小，磁盘容量，图形）是不可能的，因为此信息取自设备本身。 因此，一种解决欺诈的简单通用方法是从安装了Mac OS的外部驱动器（闪存驱动器或磁盘）启动，然后比较熨斗的特性。

Mac OS操作系统的功能是其对所有Mac型号的统一。 如果您从MacBook卸下硬盘驱动器并将其作为外部连接到任何其他Mac，它将能够从其中引导。 只有一个条件-这是系统兼容性。 新型号将无法从非常旧的版本启动。

仍然可以重新排列操作系统，但是通过网络进行安装是一个漫长的过程，您将必须像启动磁盘一样安装磁盘。

使用已安装的系统制作启动盘/闪存驱动器非常简单。 为此，您需要将驱动器连接到MacBook并重新排列其上的系统，只需选择映射的驱动器作为安装路径即可。

自2011年以来，在所有罂粟花上，您都可以通过Wi-Fi重新排列系统或制作安装盘。 我不会告诉您如何重新排列系统或如何制作启动盘。 这些信息在网络上和Apple网站上都是完整的，您可以在这里找到帖子。

仍然有一个有趣的选项；不要使用外部驱动器，而要使用任何其他Mac。 您只需要一根Thunderbolt-Thunderbolt电缆。 如果在打开时按住T按钮，则罂粟将进入外部驱动器模式，您可以像常规驱动器一样从其启动。 在Mac上，您可以安装各种测试程序。

要根据熨斗的特征识别Mac，我建议使用Mactracker程序。

我可能只想告诉他们与更改序列号或修改系统相关的欺诈风险。 但是不要忘记还有其他风险。 用手购买，您将无法获得任何技术服务保证。
