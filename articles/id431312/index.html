Sebagai orang yang serba bisa, saya sangat suka banyak musik, dengan genre / format / artis yang berbeda.

Belum lama ini saya menemukan deezer (jangan tertawa). Saya menggunakan langganan berbayar mereka, saya sangat menyukainya, langganan itu berakhir, dan membayar dari daerah saya benar-benar tugas yang sangat sulit.

Google menemukan proyek deezloader , saya katakan dengan kasar, pencurian musik yang ceroboh dari situs web Deezer. Setelah instalasi dan konfigurasi tidak memberikan login. Saya masuk lebih dalam ke Google, menemukan proyek yang sama hanya diperbarui, orang-orang memiliki cara distribusi yang menarik - melalui saluran telegram )

Proyek ini dilisensikan di bawah CC BY-NC-SA 4.0

Saya menggunakan Centos 7, simpul js 8.11 dan Deezloader Remix. Buka kemasan arsip dan lanjutkan dengan instalasi. 

curl -sL https://rpm.nodesource.com/setup_8.x | bash - sudo yum install epel-release sudo yum update sudo yum install nodejs sudo yum install gcc-c++ make cd DeezloaderRemix/app npm install node app.js

Kita pergi ke IpAddr: 1730, masuk di bawah akun deezer kami, kami mendapatkan antarmuka yang bagus untuk landak kami secara default, mengatur direktori unduhan ~ / Music / Deezloader

Banyak tangkapan layar yang sulit dipahami




















Sekarang tambahkan antrian plexmediaserver: 

 nano /etc/yum.repos.d/plex.repo 

 # Plex.repo file will allow dynamic install/update of plexmediaserver. [PlexRepo] name=PlexRepo baseurl=https://downloads.plex.tv/repo/rpm/$basearch/ enabled=1 gpgkey=https://downloads.plex.tv/plex-keys/PlexSign.key gpgcheck=1

Nah, instalasinya sendiri: 

 sudo yum -y install plexmediaserver

Nyalakan daemon dan tambahkan ke startup: 

 systemctl start plexmediaserver systemctl enable plexmediaserver

Antrian Konfigurasi Firewalld 
 nano /etc/firewalld/services/plexdeez.xml 

 <?xml version="1.0" encoding="utf-8"?> <service> <shortplexdeez</short> <description>Ports required by plexmediaserver and Deezloader.</description> <port protocol="tcp" port="32400"></port> <port protocol="udp" port="1900"></port> <port protocol="tcp" port="3005"></port> <port protocol="udp" port="5353"></port> <port protocol="tcp" port="8324"></port> <port protocol="udp" port="32410"></port> <port protocol="udp" port="32412"></port> <port protocol="udp" port="32413"></port> <port protocol="udp" port="32414"></port> <port protocol="tcp" port="32469"></port> <port protocol="tcp" port="1730"></port> </service>

Tambahkan layanan plexdeez yang baru dipanggang ke firewalld dan restart 

 sudo firewall-cmd --add-service=plexdeez --permanent sudo firewall-cmd --reload


Kami mendaftar di app.plex.tv. Kami meneruskan ip: 32400 / web kami masuk di bawah ak baru kami. Tambahkan perpustakaan Musik, tentukan direktori ~ / Music / Deezloader. Nikmati musik segar)
