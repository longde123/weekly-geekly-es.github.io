Portefeuille Wocket - Portefeuille numérique sécurisé et une carte de crédit



NXT-ID, une entreprise travaillant dans le domaine de l'authentification biométrique, a lancé un produit qui, selon les développeurs, peut remplacer un portefeuille classique. Dans le même temps, Wocket Wallet présente un certain nombre d'avantages par rapport à un portefeuille classique. À l'intérieur du portefeuille numérique, il y a aussi un petit espace pour un permis de conduire et quelques photos. Le reste du volume est occupé par l'électronique - les informations sur toutes les cartes de crédit de l'utilisateur sont stockées ici.

Les données stockées ne sont disponibles que pour l'utilisateur, qui peut recevoir des informations via une authentification combinée prenant en compte les données biométriques, le code PIN personnel ou le modèle. Une fois le processus terminé avec succès, l'utilisateur a accès à l'écran tactile et à toutes les informations stockées dans le portefeuille. La gestion des fonctions de portefeuille peut également être vocale.

Qu'y a-t-il à l'intérieur?


Les développeurs affirment que le portefeuille numérique peut stocker des informations sur le crédit, les cartes de débit, les distributeurs automatiques de billets, les cartes-cadeaux, les cartes de membre, les assurances, les billets, les données médicales, les contacts, les coupons et bien plus encore.



Pour entrer des informations sur la carte dans l'appareil, un contact physique entre la carte et l'appareil est nécessaire. Les codes-barres et le texte, ainsi que d'autres données, peuvent être numérisés ou saisis manuellement.

Les informations stockées sont cryptées avec un «sceau» biométrique et leur accès peut être obtenu après avoir passé le processus d'authentification complet, comme mentionné ci-dessus.

Caractéristiques de l'appareil


Wocket se compose de deux parties. Le premier est le stockage et un scanner. Les informations sur toutes les cartes utilisateur sont stockées ici. La seconde partie est une carte plastique à bande magnétique, sur laquelle sont enregistrées les informations de la carte utilisateur souhaitée. Il s'agit d'une sorte de double d'une vraie carte, que l'utilisateur n'a pas besoin de porter.



Travailler avec un portefeuille numérique est un peu plus compliqué que travailler avec un portefeuille conventionnel. Ici, vous devez passer par plus d'étapes que simplement obtenir une carte et payer avec quelque chose. Mais si la carte est perdue ou volée, un étranger ne pourra pas l'utiliser. Vous pouvez utiliser Wocket Wallet lors d'un paiement régulier, dans un magasin ou avec un paiement en ligne. Dans ce cas, vous devez sélectionner la carte souhaitée dans la liste, et le système affichera toutes les données qui lui sont liées, la carte.

De plus, grâce à la fonction WocketPass, un portefeuille intelligent peut mémoriser n'importe lequel de vos mots de passe et les remplir automatiquement pour accéder aux sites Web. Wocket peut même générer des mots de passe forts pour vous. C'est quelque chose comme le matériel KeePass.

Vous pouvez gérer votre portefeuille intelligent à l'aide de l'application mobile et Web Wocket, pratique et belle.

Comme le portefeuille Wocket est un appareil électronique, il fonctionne sur une batterie qui dure environ un an. Le remplacement de la batterie est une question de quelques minutes.

La qualité de construction de l'appareil est à la hauteur, l'appareil lui-même semble assez intéressant. L'écran tactile est sensible et le portefeuille n'est pas trop épais. Lorsque la carte n'est pas utilisée, elle peut être laissée dans la fente du portefeuille.



Conclusion


Wocket Wallet est un appareil intéressant et original avec un haut degré de fiabilité. Bien sûr, tout cela est plutôt inhabituel, mais, néanmoins, cela fonctionne assez bien. De plus, il n'est pas nécessaire de transporter toutes vos cartes de crédit, coupons, etc. - Maintenant, tout cela est remplacé par un petit appareil électronique.

Le processus de paiement d'un produit ou d'un service à l'aide d'une carte de crédit numérique est presque identique à ce qui doit être fait avec un paiement régulier. Parfois, il y a quelques mouvements supplémentaires, mais cela en vaut la peine - toutes les informations sont protégées de manière fiable.
