Bagaimana saya menciptakan "WildMAN" - parodi dari banyak game 8-bit dan baru-baru ini memindahkannya ke Android

Saya pikir setiap penggemar game konsol tertarik dengan topik membuat game, dan ada mimpi untuk membuat game sendiri.Di tahun-tahun mahasiswa saya, saya menjadi tertarik pada pemrograman.



Setelah belajar memprogram di Delphi, saya memutuskan untuk membuat game sendiri, pada waktu itu game untuk ponsel pada platform edisi mikro Java sangat populer, di Internet, lingkungan pengembangan Midlet Pascal sangat berguna bagi saya, yang memungkinkan saya membuat game pada platform ini tanpa mengetahui bahasa Jawa, yang pada waktu itu saya tidak tahu sama sekali.

Game j2me pertama yang berhasil saya buat adalah layar tunggal seperti arkanoid, kemudian dengan usaha keras saya menulis mesin RPG di mana Anda juga dapat membuat pencarian, mesin ini sangat tidak sempurna, tidak dioptimalkan dan terdiri dari sejumlah besar baris kode sumber, mereka dikompilasi tiga menit, pada mesin RPG ini saya membuat RPG eksperimental dengan tema kusam (Homeless Simulator)



Itu adalah tema percobaan, dibuat semata-mata untuk menunjukkan kemampuan mesin secara paralel dengan game java, konstruktor RPG sedang dikembangkan, yang saya harap akan menarik minat penggemar untuk membuat game.

Entah bagaimana, pada langkah ini, saya mulai melakukan pencarian pada topik acara favorit saya dengan Garik Kharlamov "SHOW Bulldog", khususnya "The Bruise Man", dan untuk permainan yang dikandung, saya datang dengan nama: The Bruise Man "START". Saya meninggalkan versi pertama permainan, karena itu monoton dan membosankan, di forum pemrograman Midlet Pascal, seorang programmer memberikan sepotong kode, penerjemah skrip yang mirip dengan BASIC, mengatakan bahwa permainan dapat dieksekusi berdasarkan skrip ini, dan Anda dapat memainkan permainan ini akan menanamkan minigame dan bahkan bos dengan AI sederhana. Pada awalnya, saya skeptis dengan ide ini, karena game java di ponsel pada waktu itu dan tanpa interpreter lambat dan dengan rem yang mengerikan, dia bahkan membuang screensaver dari game saya yang ditulis pada interpreter ini, saya meluncurkannya pada NOKIA N70 saya dan memastikan Kecepatan itu cukup bisa diterima.

Terinspirasi oleh bahasa scripting, saya memutuskan untuk membuat mesin permainan dari awal. Saya begitu terbawa oleh proses ini sehingga memiliki pekerjaan tetap, saya mencurahkan pemrograman untuk malam yang panjang. Setelah dua bulan bekerja dengan susah payah, saya memiliki mesin permainan di mana ada tiga mode pelaksanaan permainan: dialog, peta permainan yang dapat ditarik dalam editor khusus, dan setiap objek dalam permainan diprogram dalam mode ketiga - ini adalah mode permainan, Anda dapat membuatnya Tidak ada screensaver dan minigame puzzle yang rumit.

Butuh waktu sekitar sebulan untuk mengulang permainan darurat, tetapi perbedaannya segera terlihat, adegan skrip, teka-teki muncul dalam permainan, dan permainan menjadi beragam:


Setelah membuat game ini, ketertarikan saya pada mesin game ini tidak surut, meskipun ada ulasan negatif dan kritik, saya memutuskan untuk membuat sesuatu yang membuat ketagihan seperti BattleToads atau Black Cloak, di mana setiap level adalah mode permainan yang terpisah, mengejutkan dengan ragam permainannya, di mana kompleksitas permainannya, di mana kompleksitas permainannya sebaliknya menambah kegembiraan pada bagian itu.

Nama untuk game yang saya buat sangat sederhana: "WildMan."

Saya memiliki mesin permainan, emulator mesin ini, editor yang kuat, dengan kemungkinan tak terbatas:



Dan sedikit keterampilan dalam membuat animasi dan grafik komputer, sehingga gim ini tidak berkilau dengan grafis, dan membuat jijik setiap gamer modern.


Gim ini sering mengandung unsur-unsur dari gim 8-bit lama yang bagus seperti: Super Mario Brothers, Adventure island, Lode Runner, Baloon Fight, setiap dunia terdiri dari 3 level dan bos, dan ada 6 di antaranya: pantai, level bawah air, hutan, piramida, tingkat es dan gua terakhir.

Saya menerbitkan game WildMan pada tahun 2011, ia menerima ulasan negatif dan sedikit positif, mereka bahkan menulis tentang game buatan sendiri ini di majalah MTOG dan Mobitree, saya hanya memiliki kenangan hangat dari game ini, khususnya tentang proses pembuatan game itu sendiri, dan juga tentang menulis musik untuk itu, pada tahun 2019 saya memperoleh keterampilan dalam pemrograman di JAWA dan memindahkannya ke Android, tujuan utama ini adalah untuk memperoleh keterampilan dalam membuat aplikasi di Android, dan juga untuk menyelamatkan pekerjaan yang melelahkan ini dan hanya untuk kenangan akan hal-hal indah di Google Play siswa periode com hidup saya.
