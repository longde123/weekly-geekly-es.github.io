Pembaruan 10/15/18:
Eksploitasi PS4 muncul selama akhir pekan, di mana pesan PSN viral dapat membuat mesin Anda tidak dapat digunakan sampai Anda mengikuti solusi.

Sony mengatakan mereka mengetahui masalah ini dan sedang memperbaiki. “Kami menyadari situasi dan merencanakan pembaruan perangkat lunak sistem untuk menyelesaikan masalah ini,” Sony mengatakan kepada VG247 dalam email.

Anda disarankan untuk mengubah pengaturan konsol untuk melawan kerentanan:

- pada awalan PS4: Manajemen Akun -> Pengaturan Privasi -> Informasi Pribadi -> Pesan -> Tidak Ada (atau Teman jika Anda mempercayai mereka);

- di aplikasi seluler PlayStation Messages: Ubah profil -> Pengaturan privasi -> Informasi pribadi -> Pesan -> Tidak ada (atau Teman jika Anda mempercayai mereka).

Jika masalah tetap terjadi dan konsol terputus setelah menerima pesan, maka ada kemungkinan untuk menyelesaikan masalah - Anda perlu menghapus pesan ini dari aplikasi seluler PlayStation Messages, yang tersedia di iOS atau Android. Dianjurkan untuk merekonstruksi basis data PS4 melalui mode aman.

Jika langkah-langkah ini tidak membantu , maka hanya Reset Pabrik.
