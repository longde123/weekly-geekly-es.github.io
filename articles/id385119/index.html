Anggur untuk Android

Saya pikir Wine tidak perlu diperkenalkan . Terkadang, pengguna Linux memiliki situasi ketika Microsoft Office benar-benar diperlukan atau permainan favorit mereka tidak disajikan di Linux. Di sinilah Wine atau versi komersial CrossOver datang untuk menyelamatkan, memungkinkan aplikasi Windows untuk diluncurkan dengan relatif mudah. Sekarang akan dimungkinkan di Android.

"Wine bukan emulator," yang memungkinkan program untuk bekerja pada perangkat dengan kinerja yang sama dan hampir sebagus pada OS Windows sendiri, tetapi juga dapat bekerja buggy yang tak tertahankan.

CrossOver, produk Codeweavers yang fokus meluncurkan kantor paling populer dan aplikasi Windows lainnya, akan dirilis untuk Android akhir tahun ini. Belum jelas apa yang akan menjadi perbedaan antara versi Android CrossOver dan versi untuk Linux, tetapi tujuan utamanya adalah sama: biarkan mereka menjalankan program Windows yang populer. Karena sebagian besar program Windows tidak mendukung layar sentuh, Anda harus mempertimbangkan menghubungkan keyboard dan mouse ke tablet Anda sebelum meluncurkan aplikasi.

Tapi ada kabar buruk. Karena Wine dan CrossOver bukan emulator, sementara program Windows masih membutuhkan prosesor x86 untuk berfungsi. Sebagian besar perangkat di pasar didasarkan pada ARM dan tidak akan dapat menggunakan CrossOver. Berita baiknya adalah jumlah perangkat di Intel terus bertambah. Dan jika Anda memiliki perangkat atau rencana untuk membelinya, Anda dapat mendaftar untuk menerima pemberitahuan ketika CrossOver untuk Android "Pratinjau Teknologi" diterbitkan.
