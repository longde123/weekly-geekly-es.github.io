看完最新的《 Holivar》系列之后，我们决定继续讲述Runet的第一步以及以我们自己的方式出现在苏维埃公民家中的第一批计算机的故事，而不会陷入政治的深渊。

在上一篇文章中，有相当一部分专门介绍ZX Spectrum及其克隆，这确实淹没了苏联，并且直到90年代中期（当时与IBM兼容的计算机可供普通购买者使用）时才放弃了他们的职位。 这次，我们将尝试更认真地考虑这台计算机的历史，包括从成为runet的角度出发。



ZX Spectrum的历史是复杂且多方面的。 而且，不能说它已经结束了，按照今天的标准8位计算机的这种弱点已经成为过去。
Clive Sinclair是基于Zilog Z80处理器的训练计算机的创意，现在可以在1980-1990年代几乎或多或少的智能铁片中找到-Game Boy，带有AON的电话，家用电器-遍布世界各地，几乎形成了没有独立的文化，就是Speccy的产生。 尽管如此，我们的故事将始于更早的事件，这是围绕苏联进行光谱之旅的一种起点。

自己构建计算机



这是1986年的《无线电》杂志的摘录，第4期。 这是第一次在Radio 86RK家用计算机的装配上发表文章。 我们强烈建议您至少阅读出版商的文章序言。

假定杂志的读者可以在无线电零件存储中获得所有必要的组件后，自行组装，调试和启动该计算机。 总的来说，组装需要29颗芯片-即使在小城市中，也非常便宜。
计算机的核心是处理器KR580VM80A，它是Intel i8080A的克隆，时钟频率为1.78 MHz。 同时，实际计算机速度取决于所使用的视频模式（25、30、50条可见线），从1.31 MHz到700 kHz有效时钟周期不等。

按现代标准绝对微不足道。 同时，计算机是伪图形的-文本字符的组合用于显示游戏。

但是苏联业余计算机和Speccy之间有什么联系？ 稍后，我们将讨论原始ZX Spectrum的历史。

辛克莱研究及其在苏联计算机革命中的作用


最初的汽车出现在4年前的1982年4月在英国，并立即赢得了整个欧洲消费者的关注。

尽管主要的8位计算机竞争对手的价格标签高出一个数量级，但最初的Spectrum具有16 KB RAM的最预算版本一开始的价格仅为125英镑。 例如，传奇的Commodore 64已经以595美元的价格出售。 即使没有从一种货币到另一种货币的转移，很明显，频谱赢得了这场战斗。 但是要花多少钱呢？

如您所知，用高级材料和零件来创建预算是不可能的。 Sinclair Research的创始人克莱夫·辛克莱（Clive Sinclair）完全不惧怕。 他最初不想创建用于日常任务的功能强大的计算机。 没错，过了一会儿，Sinclair QL确实发布了他们有兴趣对年轻的Linus Torvalds进行编程的bug和可怜的软件。

Spectrum几乎定位为12岁以下儿童的训练机。 因此，主要的市场优势是：低成本，紧凑型，最少的设备以及通过其他设备扩展功能的能力。 在大小为16 KB的Spectrum ROM中写入了固件，其中包括基本的方言（Sinclair basic）并提供了简单的用户界面，其功能可能会因计算机的版本而异。


/ ZX Spectrum的Amstrad版本的“开始菜单”的屏幕快照sinclair.comboios.info

光谱由Sinclair Research在1982年至1986年间生产。 然后将权利移交给了Amstrad，但这在现代Spectrum粉丝社区中是完全不同的故事 ，这些模型被认为是最不“规范”的，并且他们的仔细考虑将花费不合理的长时间。

回到Sinclair Research的产品。 最便宜，最简单的第一批模型是一个长方形的黑色盒子。 顶部有一个橡胶键盘，背面有音频和视频接口，电源插孔和用于连接外围设备的插槽。


/ 48K ZX Spectrum


/主题是在这样的盒子里交付的

Spectrum键盘值得特别注意：40个按钮中的每个按钮（取决于输入模式）可执行不同的功能。 用户可以通过按“符号移位”和“大写移位”修改键来启用所需的输入模式。 根据所选的模式，光标可能会更改为拉丁字母L，C，K，E和G，这反过来意味着：

  • L-输入小写字母
  • C-大写
  • K-输入BASIC关键字
  • E-输入其他关键字和运算符
  • G-输入伪图形，控制字符和用户定义字符（UDG）。

光谱调色板包括8种颜色，每种都有2种亮度等级（黑色除外）。 分辨率为256x192px的屏幕被分为长度和高度为8px的熟悉度，而颜色属性设置为熟悉度，而不是像素。 这意味着相同熟悉程度的像素只能以15种颜色中的2种进行绘制。 INK属性负责第一种颜色，PAPER负责第二种颜色。 在游戏中，这引起了Spectrum的“专有”图形伪像，kleshing（属性冲突），并成为许多Spectrum粉丝和计算机对手的主要论点的绊脚石。

可以通过磁带录音机，通过连接外部磁盘驱动器，也可以使用ZX Microdrive设备，将数据输入计算机。
因此，Spectrum具有相当适中的功能（到80年代中期，Amiga 1000和Atari ST进入了市场），获得了普及，并且毫不夸张地说，全世界的粉丝和粉丝人数庞大。 经典系列中最“先进”的型号可以看作是1986年的ZX Spectrum 128“ Toastrack”，顾名思义，它具有128 KB的RAM，AY音乐协处理器和许多其他优点。

该计算机的正式生命周期于1992年结束，当时最后的“合法”模型下线了。 1983年，在玛格丽特·撒切尔（Margaret Thatcher）的要求下，克莱夫·辛克莱（Clive Sinclair）爵士本人被授予“皇家勋章骑士”的称号。 1986年，他终于放弃了计算机的生产。 如您所知，现在Sinclair不再使用Internet，而是通常使用计算机，而是喜欢使用电话而不是电子邮件。 偶尔出现在电视上的辛克莱尔（Sinclair）对ZX Spectrum已经受到并且正在受到如此多的关注表示不满意，因为“程序员可以将他们的力量引向更有用的东西”。 他认为他的“训练”计算机应该保持这种状态。 他还没有玩过为ZX Spectrum制作的游戏。

ZX Spectrum来到苏联


ZX Spectrum发行仅几年后就能够成功克隆。 而且，与多莉绵羊不同，许多克隆不仅活着，而且比其母本更有力。

演讲家社区分为两个阵营：一些“抚摸”辛克莱的原件，称其他一切为“手工工艺品”，另一些同等地与原件和克隆人息息相关，更喜欢使用后者。 然而，与“其他所有东西”相比，苏联的光谱工厂可以忽略不计。 在今天的文章中，“其他所有内容”。

30年后，不再可能确切地找到ZX Spectrum如何进入我们的国家。 据推测，Spectrum是从波兰带来的，因为一些文档和第一个软件是波兰语。

当然，工程师和程序员立即希望创建自己的Spectrum版本，以便每个人都可以在苏联可用的元素库中进行组装。 但是，克隆存在两个问题：

  • 从哪里获得处理器？

在苏联，除了性别之外，也没有Zilog Z80及其类似物。 确切地说，Z80是，但它相当昂贵，不可能用苏联的东西代替它。 后来，在90年代，我们的行业掌握了Z80，T34BM1和KR1858BM1克隆的生产，但它们的效果比真实的要差一个数量级。

  • 如何模拟ULA？

原始计算机的大多数逻辑都包含在单个芯片中，这是当时无法窃取的生产技术。 通过试验，错误，逆向工程甚至是（可能）性，微电路的工作能够在苏联元素基础上进行仿真。 芯片组的结构和组成因克隆而异。

因此，在苏联的几乎每个“技术娴熟”的城市中，都有一个克隆。 列宁格勒（Leningrad）是最受欢迎的组装工具，组装或多或少简单。

现在该回想起本文开头的时候了：顺理成章地问到“黑白Radio-PK86如何与如此强大的野兽竞争？”这一问题是合乎逻辑的。 出现了以下问题：“为什么，由于我能够制造RK，所以我仍然没有Spectrum？”。

对于一个没有被计算机技术宠爱的苏联人，Spectrum看起来像是一个启示：AY芯片上的别致音乐，盒式磁带上的数百款游戏，“仅添加电视”连接模型，在短短几个月之内就使ZX Spectrum成为了该国真正的大型家用计算机。

在这里可以看到高质量的视频，包括组装过程，调试以及列宁格勒的现代化。

不久之后，合作者然后是国有植物掌握了克隆的生产。 苏联市场的字面上充斥着质量各异，功能各异的ZX Spectrum克隆。

ZX频谱在苏联和后苏联时代的演变


也许是在后苏联时期，狂热似乎使ZX Spectrum成为最后通home的家用计算机，“不会比您的IBM和Amigi差”。 讨论Spectrum的已开发克隆具有两种意义：

  • 开发新的“铁”解决方案；
  • 开发软件和游戏，将所有果汁挤出电脑。

接下来，我们将考虑这两个方面，也许会让您与国内Speccy爱好者的技能和事业感到惊讶。

频谱兼容的计算机


在90年代初期，出现了许多先进的频谱信号。 不可能考虑所有的机器，因为许多机器只有一个内存和几个“活动”板，有些机器的制造如此特殊或昂贵，以致于没有被广泛使用。 让我们关注四个（主观上）最有趣的模型。

蝎子

蝎子计算机是由同名的圣彼得堡公司开发和生产的，在ZX Spectrum场景中的面孔是谢尔盖·佐诺夫（Sergey Zonov），他是早期ZX Spectrum克隆之一-列宁格勒电路的作者。

该公司的计算机以印刷电路板的形式制造，可以由用户安装在任何合适的机柜中，例如，从IBM PC兼容计算机安装在MiniTower机柜中；相同形式的计算机由该公司自己生产。 在这里，您应该确定计算机的关键特征：

  • Z80B处理器（3.5 / 7.0 MHz）;
  • 通过编程和按钮都包括涡轮模式；
  • 来自Andrey Larchenko的影子服务监视器；
  • 音乐处理器AY-3-8910 / 12;
  • ZX总线系统总线，用于2个插槽（通过扩展器+3）。


/蝎子板，krsk.au.ru

五角大楼

基于克隆的“ Leningrad”版本。 它是由ATM公司和MicroART的创意团队于1989年开发的。 越来越受欢迎，这不仅是因为完善的计划以及许多在家改善的方法。 改进之处包括：
  • RAM扩展至4 MB；
  • 连接Covox的能力；
  • 几种其他视频模式和多色；
  • 通过Nemo IDE控制器连接硬盘驱动器的能力
  • 涡轮增压模式。


/ ATM Turbo被认为是五角大楼的精神继任者-更为复杂和复杂。

ATM Turbo 1和2

标志性的Spectrum兼容计算机之一。 它是同一开发团队创建的五角大楼克隆的直接继承人。 它于1991年发布。 第一个计算机模型包含许多错误，并且与某些较早的软件不兼容。 因此，她没有得到太多的爱和欢迎。

ATM Turbo 2的新版本（以及稍后发布的2+版本）进行了许多改进和创新。 单独谈论每个版本几乎没有道理，因此，我们列出了ATM Turbo 2最有趣，最重要的功能：

  • 处理器加速模式高达7 MHz;
  • RAM高达1 MB；
  • OS CP / M支持（内置ROM）；
  • 与原始ZX Spectrum完全兼容；
  • 64种颜色的调色板（同时最多16种）；
  • 调制解调器（在2+版中删除）；
  • 支持IDE设备（HDD，CD-ROM）。

使用调制解调器，ATM Turbo用户可以使用特殊实用程序（例如，用于拨号的ZX终端）访问网络。 但是，我们未能确认或反驳此决定的可操作性，也未获得目击者的陈述。

ATM Turbo 2+是迄今为止最热情的ZX Spectrum克隆。 我们建议您熟悉此站点， 该站点包含用于ATM Turbo的大量程序，操作系统和电路。


/ ATM Turbo 2+板，ruecm.forum2x2.ru

Profi和Profi Plus
它是由Condor公司于1991年在莫斯科开发的。 它与原始Spectrum完全兼容（使用特殊的拨动开关切换“ Pro”和“ Spectrum”模式）。 国内最先进的克隆之一。 具有以下功能（在5.02及更高版本中）：

  • Turbo模式7 MHz;
  • RAM最高1 Mb（用作电子磁盘）；
  • 在CP / M OS中以扩展图形模式512x240px工作；
  • 256种颜色的调色板（同时最多16种）；
  • 硬件多色；
  • 自己的IDE控制器，可用于硬盘驱动器；
  • ROM 64 KB;
  • Covox;
  • LPT端口和编程器端口；
  • 连接调制解调器的能力（特别是用于访问Fidonet和组织本地网络的能力）。

由Condor公司开发的高质量软件支持的非常有趣的计算机。 不幸的是，目前还没有完整的系统磁盘以及Internet上包含程序和游戏的磁盘。 尽管如此，在9月初，VKontakte还是“ Profi计算机支持小组”的成员 ，在那里，一些爱好者在90年代就参与了Profi软件的开发，他们计划将有关此克隆的信息系统化。

软件和游戏


从某种意义上讲，在俄罗斯使用Spectrum进行的所有事情都像在地球上拉猫头鹰一样。 当然，以自己的方式极其浪漫和美丽。

ZX Spectrum游戏中最常见的类型之一是具有鸟瞰视角的平台游戏和等距rpg。 2014年，专门发布Arcade Game Designer游戏引擎以创建简单的平台游戏。 假定在它的帮助下，直接在Spectrum上或在模拟器中，新手程序员将能够创建自己的游戏，然后将其以Tap格式保存到计算机或磁带上。
