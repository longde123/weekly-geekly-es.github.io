Kiat dan trik visual studio

Apakah Anda baru atau telah menggunakan Visual Studio selama bertahun-tahun, ada banyak tips dan trik yang dapat membuat Anda lebih produktif. Kami telah berbagi kiat di Twitter menggunakan tagar #vstip untuk sementara waktu, dan ini adalah koleksi yang terbaik sejauh ini.

Debugger


Menekan F10 untuk membangun, menjalankan, dan melampirkan debugger alih-alih F5 akan secara otomatis terputus saat pertama kali kode Anda dijalankan. Tidak perlu breakpoint.



Didukung dari Visual Studio 2005

Pasang kembali ke proses (Shift + Alt + P) sangat membantu ketika Anda harus melampirkan ke proses yang sama berulang kali.



Didukung dari Visual Studio 2017 v15.8

Titik biru di margin menunjukkan beralih utas saat melangkah melalui debugging.



Didukung dari Visual Studio 2013

Solusi


Meningkatkan kinerja beban solusi dan mengurangi noise visual dengan menonaktifkan pengembalian ekspansi node di Solution Explorer serta Buka kembali dokumen pada beban solusi.



Didukung dari Visual Studio 2019

Untuk navigasi keyboard yang cepat, gunakan Ctrl + T untuk menemukan apa pun dalam solusi Anda - file, kelas, dll.



Didukung dari Visual Studio 2017

Tetapkan pintasan keyboard untuk melakukan "git pull" sehingga Anda tidak perlu menggunakan CLI atau Team Explorer untuk memastikan repo Anda terbarui.



Didukung di Visual Studio 2019

Buat Solution Explorer secara otomatis memilih dokumen aktif saat ini, sehingga Anda tidak pernah kehilangan jejak lokasinya di proyek.



Didukung dari Visual Studio 2010

Editor


Mudah mengelilingi elemen HTML dengan menggunakan Shift + Alt + W. Disisipkan dipilih sehingga Anda dapat dengan mudah mengeditnya menjadi tag yang Anda inginkan, dan tag-akhir cocok secara otomatis.



Didukung dari Visual Studio 2017

Salin setiap fragmen JSON ke clipboard dan rekatkan sebagai kelas .NET yang sangat diketik ke dalam file kode C # atau VB.



Didukung dari Visual Studio 2013

Anda tidak perlu menulis tanda kutip di sekitar nama properti JSON, cukup ketik titik dua dan Visual Studio akan memasukkan tanda kutip secara otomatis.



Didukung di Visual Studio 2015

Jadikan IntelliSense dan tooltips semi-transparan selama Anda menekan dan menahan tombol Control.



Didukung dari Visual Studio 2010

Alih-alih mengetik ulang '(' untuk menampilkan info parameter dalam tanda tangan metode, gunakan Ctrl + Shift + Spasi untuk menampilkan kelebihan beban yang saat ini digunakan.



Didukung dari Visual Studio 2010

Lain-lain


Putar suara saat peristiwa tertentu terjadi dalam Visual Studio.



Didukung dari Visual Studio 2010

Buat tata letak jendela khusus untuk skenario pengembangan spesifik atau pengaturan monitor dan gantilah dengan mudah.



Didukung dari Visual Studio 2017

Tentukan komponen Visual Studio mana yang diperlukan untuk solusi apa pun, dan Visual Studio akan meminta pengguna untuk menginstalnya jika tidak ada. Baca selengkapnya di posting blog Konfigurasikan Visual Studio di seluruh organisasi Anda dengan .vsconfig .



Didukung dari Visual Studio 2019

Ekstensi


Visual Studio Pemeriksa Ejaan . Ekstensi editor yang memeriksa ejaan komentar, string, dan teks biasa saat Anda mengetik atau berinteraksi dengan jendela alat. Itu juga bisa mengeja memeriksa seluruh solusi, proyek, atau item yang dipilih. Opsi tersedia untuk menentukan beberapa bahasa untuk mengecek ejaan.



Didukung dari Visual Studio 2013

Tambahkan File Baru . Ekstensi Visual Studio untuk dengan mudah menambahkan file baru ke proyek apa pun. Cukup tekan Shift + F2 untuk membuat file kosong di folder yang dipilih atau di folder yang sama dengan file yang dipilih.



Didukung dari Visual Studio 2015

Git Diff Margin . Git Diff Margin menampilkan perubahan Git langsung dari file yang saat ini diedit pada Visual Studio margin dan scroll bar.



Didukung dari Visual Studio 2012

Ini hanya beberapa dari ribuan ekstensi yang tersedia. Untuk melihat lebih banyak ekstensi, buka Visual Studio Marketplace .

Sebagai penutup


Ini hanya beberapa tips pilihan tangan dari tagar #vstip di Twitter. Ada banyak lagi yang perlu diperiksa. Jika Anda memiliki beberapa tips hebat, silakan bagikan menggunakan tagar #vstip agar kita semua dapat menemukannya dengan mudah.
