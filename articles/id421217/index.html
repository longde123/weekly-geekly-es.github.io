Dukungan Python di Power BI

Kami sangat senang mengumumkan bahwa Python, bahasa pemrograman yang banyak digunakan oleh ahli statistik, ilmuwan, dan analis, sekarang diintegrasikan ke dalam Power BI Desktop gratis kami. Setelah Anda mengaktifkan fungsi di pengaturan, Anda dapat menggunakan Python untuk membersihkan, menganalisis, dan memvisualisasikan data. Lebih detail di bawah cut!



Fitur integrasi pertama yang kami tambahkan adalah kemampuan untuk menjalankan skrip Python langsung di Power BI Desktop untuk membuat model data. Anda dapat melakukan ini di kotak dialog Dapatkan Data.



Setelah memilih opsi ini, Anda dapat memasukkan skrip Anda.



Integrasi dengan editor kueri memungkinkan Anda untuk membersihkan data menggunakan Python dan menggunakan metode canggih untuk menganalisis data Anda, termasuk, misalnya, menghapus misi, perkiraan dan pengelompokan.

Terakhir, Anda dapat menggunakan Python untuk membuat efek visual baru dalam laporan Anda. Seperti visualisasi dalam R, efek visual Python akan diperbarui dengan penyegaran data dan penyaringan silang. Namun, visualnya sendiri belum interaktif.



Dan itu belum semuanya
