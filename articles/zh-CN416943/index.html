设计语音接口的有用材料

语音界面比尘土飞扬的机器人拥有更好的未来。 设备的销售迅速增长，主要平台正在迅速建立伙伴关系并将其集成到与电力和互联网连接的所有事物中。 到今年年底，预计将有1亿台设备 ； 美国已有20％的家庭拥有精巧的扬声器 。 好吧，前天的国内高销量排在爱丽丝的面前。 无论您是否在那儿，研究该主题都是很有用的。


主要书籍是Cathy Pearl 设计的语音接口 。 顺便说一句，她负责Google Assistant的设计，最近介绍了语音界面的主要设计功能 。



每个主要平台都有为其设计接口的准则：

  1. 亚马逊Alexa
  2. Google助手
  3. 微软Cortana
  4. 苹果Siri
  5. Yandex Alice-目前尚无设计指南，但有常规文档

Alexa生态系统开发最完善-它具有足够的工具和有趣的实验（该平台拥有超过50％的市场）。 例如：

  1. Storyline在线服务，用于培养技能。
  2. 测试它们的回声 。
  3. Upside的Rafal Cymerys展示了一个基于Alexa的StarCraft II语音实验 。 尽管通过鼠标和键盘仍可以轻松完成基本操作，但它非常适合涉及一系列原始操作的操作。
  4. 足够的语音游戏和诸如《 侏罗纪世界》之类的任务。 与台式机有一些有趣的组合 。 甚至还有讽刺的天际 。



可以在Voicebot网站上找到大量最新新闻和最新统计信息。 以下是有关该主题的几篇相对较新的文章：

  1. 几位脱口秀喜剧演员赞赏智能助手 Amazon Alexa，Apple Siri，Google Assistant和Microsoft Cortana的幽默 。 例如，谷歌有一个单独的作家团队来提供笑话。
  2. Connected Lab的Jacky Li对如何使用触摸屏和语音控制机器功能进行了一些用户研究。 触摸屏显然会分散驾驶员的注意力。 但是语音控制并没有更好（ Nielsen / Norman Group的最新文章提供了具体原因 ）。
  3. Frederik Goossens语音设计指南 。


对于设计师来说，这只是一小部分有趣且重要的作品。 随时了解最新情况，有可能很快就会在工作中派上用场。
