Der endgültige Build von Windows 1809 wurde zurückgerufen

Aufgrund bekannter Probleme beim Löschen von Dateien aus einem Profil während eines Updates hat Microsoft beschlossen, den endgültigen Build des Windows-Updates vom 10. Oktober 2018 (1809) zu widerrufen.

Seit dem 6. Oktober wurde die Assembly nicht mehr über den Windows Update-Dienst verteilt. Wenn Sie das ISO-Image selbst heruntergeladen haben, installieren Sie es nicht und warten Sie auf die neue Assembly.

Die aktuellen Builds von Windows Server 2019 und Windows 10 Enterprise (einschließlich LTSC und IOT) sind ebenfalls von diesem Problem betroffen.
