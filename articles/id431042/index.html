PHP internal


  • RFC: Password Hashing Registry - Penawaran diterima untuk versi PHP berikutnya dan memungkinkan ekstensi untuk mendaftarkan algoritma hashing mereka sendiri.
  • RFC: Sebarkan Operator di Array - Diusulkan untuk menambahkan operator ... untuk array.
    Teks tersembunyi 
     $parts = ['apple', 'pear']; $fruits = ['banana', 'orange', ...$parts, 'watermelon']; // ['banana', 'orange', 'apple', 'pear', 'watermelon'];
  • [RFC] Preloading - Suara RFC pada preloading file telah berakhir. Semua anggota tim inti dengan suara bulat mendukung proposal ini. Jadi tahun depan di 7.4, kita akan memiliki kesempatan untuk memuat file ke opkash. Semua fungsi dan kelas yang dideklarasikan dalam file-file ini akan tersedia untuk semua permintaan berikutnya, seolah-olah mereka adalah elemen bawaan seperti strlen () atau Exception.
    Ada diskusi terperinci tentang mendukung fitur ini di Composer dan Symfony (dikirim oleh @dmitrybalabka ), dan konsep plugin Ayesh / Composer-Preload juga tersedia.

Alat-alatnya


  • spiral / roadrunner - Server aplikasi berkinerja tinggi, penyeimbang beban, dan manajer proses untuk PHP yang diimplementasikan pada Go. Untuk mendukung, cerita menarik tentang pembuatan RoadRunner dan perubahan mendasar dalam pendekatan untuk mengembangkan aplikasi PHP.
  • cebe / php-openapi - Alat membaca file OpenAPI dengan deskripsi layanan dalam objek PHP yang nyaman untuk bekerja.
  • tgalopin / html-sanitizer - Perpustakaan untuk memproses, membersihkan, dan membersihkan data HTML yang diterima dari pengguna. Bundel untuk symfony juga tersedia. Dikirim oleh berezuev .
  • jeremykendall / php-domain-parser - parser Domain yang mengenali bagian terdaftar dari domain dan akhiran publik.
  • nginxconfig.io - Generator konfigurasi Nginx.
  • rryqszq4 / ngx_php7 - Penerjemah PHP dibangun ke dalam Nginx. Memungkinkan Anda membuat penangan permintaan dalam PHP, mengubah permintaan / respons, memfilter badan respons dan tajuk, dan lainnya.
  • php-desimal / ext-desimal - Ekstensi mengimplementasikan aritmatika desimal dengan presisi sewenang-wenang dan pembulatan yang benar. Posting dari penulis yang mendukung.
  • Hywan / php-ext-wasm - Ekstensi eksperimental yang memungkinkan Anda untuk menjalankan binari WebAssembly langsung dari PHP. Ini berpotensi memberikan peluang untuk memuat modul secara dinamis yang diimplementasikan dalam bahasa lain.
