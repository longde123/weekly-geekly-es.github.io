欺骗的错觉：基于逆向预测的视觉光学错觉



人脑通常被称为世界上最复杂的生物计算机。 来自世界各地的科学家继续研究这种难以置信的神秘器官，发现其越来越多的特性，功能，功能等。 众所周知，许多科学以及它们的研究是相互联系的。 因此，对人脑的研究可以使您改善与解剖学无关的技术。 今天，我们将研究对大脑的一项新研究，科学家正在其中进行测试以欺骗大脑的新方法。 我们通过大脑处理的信息来感知周围的世界。 我们看到，听到，闻到，尝到和触摸到的一切都由我们的感官感知和大脑处理的一组信号组成。 但是，如果这些信号是错误的，或者根本不存在，大脑会确定它们是正确的，那该怎么办？ 这就是今天的研究中要讨论的内容。 走吧

学习基础

我们都熟悉光学错觉的概念。 这种现象可能与物理现象（光的折射：一杯水里的汤匙）有关，也与我们的大脑对信息的感知错误有关。 让我们抛开物理学吧，尽管它无处不在并且一直存在（就像DMB电影中一样：“你看到地鼠吗？我没看见，但是他在里面”）。 对周围世界数据的错误理解可能有多种形式和类型。 当然，最常见的是，我们会听到并阅读有关视觉幻觉的信息。 研究人员说，大多数错觉的主要原因之一是回溯预测的概念，可以说是大脑预测了过去。 预测已经发生的事情听起来有些奇怪。 但是在这里，神经生物学进入了世界，向我们解释了一切。 在最近的2000年，斯坦福大学的神经科学家David Eagleman和萨克生物研究所的神经科学家Terry Seinowski提出了这样一种理论，即有时人脑在事件发生后会收集信息，然后得出结论。

您会在下面看到一个带有闪光灯的错觉的例子，这种现象更容易理解。



同样错觉的另一个版本，但是已经在视频中


仔细看一下动画。 慢慢来 你看到了什么？ 更确切地说，您在哪里看到？ 绿色方块看起来与红色方块平行还是略有偏离？ 如果您看到绿色出现的瞬间两个正方形彼此平行，那么恭喜，您来自K行星。

伊格曼（Eagleman）和塞诺夫斯基（Seinowski）认为，感知（感知，喜欢更复杂的术语）不会在事件发生之前（绿色方块的外观）发生，而不是在事件发生期间发生，而是在事件发生之后80毫秒发生。 在这种情况下，大脑必须感知两个“信号”：红色方块的位置和出现的绿色的位置。 反向掩蔽理论说，视觉刺激的感知会因在主要刺激之后立即出现的掩蔽刺激而恶化。 也就是说，绿色方块的出现会阻止我们的大脑完全确定红色的位置，因此事实证明它们彼此不平行。 这一理论证实了逆向预测现象。

正如我们已经了解的那样，这些幻觉仅基于大脑对视觉信息的感知错误。 在我们今天讨论的研究中，科学家们决定在视觉刺激之外添加声音刺激，以了解当主要刺激和掩蔽刺激不在一般类别时是否可以使用反向掩蔽。

学习准备

该研究进行了两种类型的实验。 第一种是视觉刺激的幻觉，第二种是视觉刺激的抑制。


图片编号1

上图显示了传入刺激（左）和被摄对象（右）的示意图。 灰色矩形是与特定声音信号同时发生的闪光。

选项A是第一类实验。 如您所见，有2次闪烁和3次哔声，但被摄对象却看到3次闪烁。

选项B是第二种实验。 有3次闪烁，只有2个声音信号，被摄对象仅看到2次闪烁。

作为实验兔子（犬儒主义的对不起），有13人：4名男性和9名女性。 研究人员指出，他们没有告诉受试者实验的所有细节，而只是询问他们看到多少闪光。 因此，科学家们希望提高结果的客观性和准确性。

设置实验非常简单。 受试者以57厘米的距离以60 Hz的刷新率坐在监视器前面，他们的头位于下巴支架上，以确保在监视器前面的位置相对一致。 房间内的灯光昏暗，在实验过程中，还可以更有效地感知监视器上产生的视觉刺激。


实验设置

屏幕背景和视觉刺激（灰色矩形的闪烁）之间的区别在于亮度：背景为30％，矩形为80％。 每次视觉闪烁都会在屏幕上准确显示17毫秒。

声音的刺激通过显示器侧面的两个扬声器再现。 音频信号的持续时间（800 Hz）为7毫秒。

在音频和视频信号之间建立了23秒的间隔，以消除不同主体之间感知速度的可能差异。

在他们的报告中，研究人员以NbMf的形式展示了每个实验，其中Nb是声音信号的数量，而Mf是闪烁的数量。 也就是说，2b4f是2声哔哔声和4次闪烁。

与主题的实验分为几个主要阶段：

第一阶段 ：闪光计数。

参加了7位受试者，为他们提供了6种实验选择：
  1. 0b2f-左侧闪烁，暂停，右侧闪烁；
  2. 2b2f-左侧+声音闪烁，暂停，右侧+声音闪烁；
  3. 3b2f-左侧闪烁+声音，声音，右侧闪烁+声音；
  4. 0b3f-左侧闪烁，中央闪烁，右侧闪烁；
  5. 2b3f-左侧闪烁+声音，中央闪烁，右侧闪烁+声音；
  6. 3b3f-左侧闪烁+声音，中央闪烁+声音，右侧闪烁+声音。


实验的每个选项均进行了25次。 应该让受试者记录他们在每个实验中看到的闪光次数。

第二阶段 ：确定虚幻闪光的位置。

只有8位参与者和4个实验选项：
  1. 0b2f-左侧闪烁，暂停，右侧闪烁；
  2. 2b2f-左侧+声音闪烁，暂停，右侧+声音闪烁；
  3. 3b2f-左侧闪烁+声音，声音，右侧闪烁+声音；
  4. 3b3f-左侧闪烁+声音，中央闪烁+声音，右侧闪烁+声音。


总共以随机顺序进行了100次这样的实验。 受试者还记录了闪光灯的数量，并且还指示了他们在监视器上感知到的每个闪光灯的位置。 因此，有可能确定受试者对不存在（虚幻）爆发的感知。

第三阶段 ：对刺激位置及其方向的初步了解。 此外，随机选择了8位参与者，并且又有6个实验选项。

（从左到右三个，反之亦然，从右到左）：
  1. 0b2f-左侧闪烁，暂停，右侧闪烁；
  2. 2b2f-左侧+声音闪烁，暂停，右侧+声音闪烁；
  3. 3b2f-左侧闪烁+声音，声音，右侧闪烁+声音；


在此阶段，针对每个方向进行了15个实验，而第一个闪光灯始终位于屏幕中央，而其余闪光灯则以随机顺序显示在右侧或左侧。 受试者再次记录闪烁的次数及其方向。

第四阶段 ：偏心率。 5名受试者参加了。

实验有6种变化：
  1. 0b2f-左侧闪烁，暂停，右侧闪烁；
  2. 2b2f-左侧+声音闪烁，暂停，右侧+声音闪烁；
  3. 3b2f-左侧闪烁+声音，声音，右侧闪烁+声音；
  4. 0b3f-左侧闪烁，中央闪烁，右侧闪烁；
  5. 2b3f-左侧闪烁+声音，中央闪烁，右侧闪烁+声音；
  6. 3b3f-左侧闪烁+声音，中央闪烁+声音，右侧闪烁+声音。


以上每个选项均运行25次，而偏心角（与眼睛中心的角度）不同：4°，10°和16°。 更改此参数可让您确定幻觉的发生与对象视角的相关性。

第五阶段 ：置信度。 在这一阶段，有8名参与者和6种体验选项（见上文）。 除了记录他们看到的暴发数量之外，受试者还必须从4个选项中选择一个，以确保他们对答案的信心程度。 因此，有可能确定认知失真对实验的影响。 当一个人根据自己的信仰，情感或身体状况等做出选择或给出答案时，这种现象可以等同于主观性的概念。 换句话说，您还记得起初有多少与心理分析有关的测试要求您不要思考问题和答案，而要说出想到的第一件事。 这大约是认知失真的水平。

第六阶段 ：追溯预测。

8位参与者和3个实验选项：
  1. 1b2f-左侧闪烁+声音，中间闪烁；
  2. 2b3f-左侧闪烁+声音，中央闪烁，右侧闪烁+声音；
  3. 2b3f-左侧闪烁+声音，中央闪烁+声音，右侧闪烁。


如我们所见，第二个和第三个选项的声音信号和闪烁次数相同-2b3f，但是位置不同。 所有选项均经过25次测试。 此阶段旨在确定在预测信息过程中回溯预测的重要性。

第七阶段 ：镇压。

8位参与者经过了此阶段，实验有4个选项：
  1. 0b3f-左侧闪烁，中央闪烁，右侧闪烁；
  2. 2b2f-左侧+声音闪烁，右侧+声音闪烁；
  3. 2b3f-左侧闪烁+声音，中央闪烁，右侧闪烁+声音；

在此阶段，以随机顺序进行了总共100个实验。 测试参与者记录了他们看到的爆发次数并指出了他们的位置。 为了了解其中一个闪光灯是否处于隐藏状态或是否与相邻闪光灯合并，此阶段是必需的。

实验结果


虚幻的视听信号的实验结果图。

在图A中，我们可以看到参与者参与了与阶段I相关的实验中的闪光次数。当使用3b2f模型（3次哔哔声和2次闪光）时，与使用2b2f模型（2次哔哔声和仅2次哔哔声）相比，大多数受试者看到的闪光次数更多真实的闪烁）。 这表明附加的声音信号提示被摄对象应该闪烁3次，而实际上只有2次。

研究人员还注意到，此结果不取决于进行的测试次数。 也就是说，前50个调用中的测试结果与后50个调用中的测试结果几乎没有区别。 因此，无论重复多少次，它们都无法适应测试条件。

图B显示了受试者对他们的反应的信心程度。 我们可以看到，无论源数据如何（闪烁和声音信号的数量以及它们的组合的变化），在所有情况下的置信度都大致相同。

最后，安排C。 它展示了对象如何分布爆发，即他们认为出现的位置（右或左）。 结果表明，对爆发的感知还取决于它们出现的一侧（从左到右或从右到左）。 当受试者看到三下闪烁时，大多数人表示第一和第三相距很远，而第二则相对居中。 虽然，当然，实际上根本没有第二闪光。 在实验过程中，真实对象之间的距离没有变化。 也就是说，虚幻的闪光的存在将两个真实的物体推开了。


隐藏（抑制）的视听信号的实验结果图。

上面的逆向实验结果图，当受试者没有感觉到真实的闪光时。 对于2b3f模型（2次哔哔声和3次闪烁），绝大多数对象仅看到2次闪烁，而对于0b3f模型，则所有3次闪烁。 这表明，声音信号就像掩盖刺激一样，会阻止其中之一闪烁，从而阻止大脑正常感知。 而在没有声音信号的情况下，没有任何东西可以分散大脑以完全感知所有真实的闪光。 从图A可以看出，模型2b3f，3b3f和0b3f的测试结果几乎相同。

附表B还表明受试者对回答的信心很高。

对仅看到2次闪光（而不是真实的3次闪光）的受试者中2b3f模型（2次哔哔声和3次闪光）结果的分析表明，大多数参与者看到的是现实中的第一个和第三个闪光（图C ）。

事实证明，不存在的闪光灯的存在会将真实的闪光灯从其实际位置推向相反的方向。 在抑制第二次爆发的感知的情况下，爆发的感知位置与实际位置没有差异。

结语

这项研究表明，声音信号可以充当掩盖刺激物，从而阻止对真实刺激物的感知。 同时，声音信号的使用可以导致对虚幻的视觉刺激的感知，而这在现实中是不存在的。

人脑存储和处理大量数据。 我们所看到，听到的，闻到的和触摸到的所​​有动作（从挥手到新细胞的诞生）都是需要在反应之前进行处理的信息。 有时大脑会被愚弄。 今天，研究人员展示了一种实现这一目标的新方法。 这些研究尽管乍看之下很有趣，但对于扩大一个人对自己的认识的界限非常重要。 对于今天的大脑来说，还剩下很少的研究“对象”，以及海洋的深处和宇宙的广阔空间。
