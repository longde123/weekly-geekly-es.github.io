Game Dev Sim: board game tentang pengembangan game

Baru-baru ini, sebuah game papan Rusia muncul, yang dengan jelas dan riang memberikan gagasan tentang betapa sulitnya pekerjaan pembuat video game! Sedikit mirip dengan game Startup yang terkenal, tetapi lebih banyak dalam kemampuan konten dan gameplay. Dan tentu saja, pengaturan spesifik tentang pengembangan game. Di bawah potongan, saya mengumpulkan informasi terperinci untuk mereka yang tertarik pada topik.



Tujuan permainan dan gameplaynya


Kepala studio game Anda sendiri dan buat game keren. Apakah itu mahakarya untuk komputer, konsol atau platform seluler, pemain memutuskan. Dalam prosesnya, Anda harus mencari solusi untuk lebih dari 60 kasus berdasarkan peristiwa nyata, yang dikembangkan oleh para profesional di industri game, yang telah mengabdikan diri untuk mengembangkan game selama bertahun-tahun, dan mampu menciptakan kembali suasana kekacauan, kesenangan, kengerian, dan keputusan manajemen luar biasa yang tak dapat dilepaskan yang begitu melekat dalam pembuat game.



Tugas pemain meliputi: manajemen proyek, sewa kantor, pemilihan dan motivasi staf, pemasaran, desain game, bekerja dengan kode dan seni, partisipasi dalam pameran, interaksi dengan investor dan banyak lagi. Setiap keputusan mempengaruhi gameplay dan dapat menyebabkan pemain sukses luar biasa atau gagal total.

Game Dev Sim adalah:


  • 60 kasus nyata dari studio dev game nyata.
  • Ratusan kombinasi dan strategi. Kemampuan pemutaran di atas.
  • Perubahan dinamis dalam situasi, tergantung pada tindakan para pemain.
  • Aturan sederhana, pengembangan cepat.
  • Dibuat oleh para profesional di industri game.
  • Dan sedikit humor bermerek gamedev!



Siapa penulis game ini


Golubkin Sergey - pemilik rumah penerbitan papan permainan GEMENOT, telah merilis lebih dari 30 desktop, mantan produsen game komputer dan seluler di Ubisoft dan Nival.
Vyacheslav Utochkin adalah direktur program pendidikan untuk industri game di Sekolah Tinggi Informatika Bisnis dari Sekolah Tinggi Ekonomi, bekerja sebagai produser di Mail.Ru Group on Armoured Warfare, Perfect World dan beberapa game lainnya.



Efek pendidikan


Game Dev Sim dikandung dan digunakan sebagai praktik permainan untuk pelatihan pada program pendidikan "Manajemen Proyek Game" di Sekolah Tinggi Informatika Bisnis dari Sekolah Tinggi Ekonomi. Format desktop gim ini bagus karena ada kemungkinan komunikasi langsung yang mudah di antara para pemain, Anda bisa mendapatkan respons instan untuk setiap pertanyaan yang muncul. Ini sangat penting dan membantu proses pendidikan. pada saat yang sama, dasbor juga menyenangkan dan Anda bisa bermain untuk bersenang-senang. Aturannya cukup sederhana, sehingga siapa pun yang bahkan tidak terhubung dengan industri game dapat dengan cepat mengetahuinya dan bermain. Dan inilah video singkat tentang gim ini:
