用Epson PaperLab造纸

Geektimes编辑已经谈论了Epson PaperLab办公用纸处理机的发布。 从那时起，我们收到了有兴趣的读者的许多问题，我们最终可以在这篇文章中回答。


认识我！ 爱普生PaperLab是世界上第一台办公室中的干法办公纸张回收机。


下载废纸-获取干净的床单。



废纸以及现在正在做什么


全世界平均每年生产4亿吨纸和纸板。 纸浆主要用于造纸，再生废纸占全球纸产量的38％（ 根据欧洲造纸商CEPI联合会的数据 ）。



同时，处理一吨印刷纸可节省超过2吨木材。


那么，废纸被送到加工厂后会怎样呢？


排序



原材料按质量等级分类。 十二种废物中的七种适合于制造鸡蛋托盘，包装等。其余三种废物-报纸，杂志，办公用纸-被认为是最昂贵的原材料。 由它们制成笔记本，卫生纸，墙纸，相册等。


根据GOST 10700-97的主要分类标准是密度和质量。 例如，用于打印机和信件的纸张属于“ A”组-最高等级。


溶出度


压缩后的纸包被浸泡在专用的碎浆机储罐中，该罐根据洗衣机的原理工作：滚筒产生漩涡，将纤维混合并磨碎。 然后清除纸浆中的杂质：由于滚筒中的旋转而去除了重杂质：沙子，玻璃，回形针等。 当它们通过筛子时，它们会落在集尘器和肺部。 在浸泡过程中，添加了化学药品，这使得印刷油墨和油墨颗粒不再粘附在纤维上。


在该阶段获得的所谓的“纸浆”用于生产低级纸板，软填料，鸡蛋包装以及进一步的清洁和加工。



此外，通过在研磨机中研磨将纸浆重新拉出并进行精细清洁。 顺便说一句，加工周期通常最多可以重复7次，随后的每次纤维加工都会变短，最终使它们变得不适合生产新纸。



哪些设备可用于处理


从工业机器和系统中有：


  • 每个处理操作的单独设置 ；
  • 用于多种操作的组合设备 （小型工厂）；
  • 大型工厂完成了整个过程链。

当预计每天平均废纸流量为10至50吨时，将为每个阶段购买单独的工厂。 如果大量稳定地产生废纸，那么最好使用大型加工厂。


但是在那些每天废纸供应不超过3-5吨的地方，微型工厂是有需求的，因为在这种情况下，购买功能强大的昂贵设备毫无意义。


爱普生PaperLab


最近-在2016年-出现了另一种类型的加工机-最紧凑的。 加工和生产的整个过程合而为一。 爱普生PaperLab就是这类设备。


该设备的紧凑型尺寸创纪录，仅为2.6×1.2×1.8 m。


PaperLab的主要特点是，它是世界上第一台（根据Seiko Epson截至2016年11月的数据）办公纸回收机，它通过干法将废纸回收成新纸。


与使用大量水的传统回收系统不同，PaperLab实际上不使用水（除了维持系统湿度水平所需的极少量水外，该机器不需要与供水系统连接，可以安装在任何办公室）。 。


也就是说，除了保护树木之外，新设备还节省了大量水。


爱普生的工程师于2011年开始在PaperLab上工作，并开发了干纤维技术，并继续致力于该技术。2016年，爱普生的这项技术获得了著名的《日经全球环境技术奖》。


爱普生干纤维技术



为了不使信息过多，我们将尽可能简短地讨论干纤维技术的工作原理：


  1. 载入中
    将各种尺寸的废纸装入PaperLab。
  2. 纤维溶解
    爱普生的无水解纤装置将纸张机械分解成细长的纤维。 此过程立即且不可撤消地破坏机密文档。 由于PaperLab不使用水进行循环利用，因此无需将机器连接到供水系统，这意味着可以在任何办公室中安装机器。
  3. 拼接
    纤维与PaperPlus粘合剂共混以赋予纸张强度和白度。 另外，在同一阶段，载体可以增加颜色，香气和增加成品耐火性的物质。
  4. 成型
    纤维以薄层布置并且在压力下形成为片。 在此阶段，将指定板材的强度，厚度和尺寸。
  5. 将纸张切成所需的格式
    瞧！ 全新的纸张随时可以打印！



PaperLab有什么好处


一分钟内，PaperLab能够生产约12张A4纸。 这是每小时720张或每8小时工作日5760张。 从按下开始按钮起，第一张纸退出的时间为3分钟。 除了类似的性能，该设备还具有以下优点：


  1. 办公纸回收周期

    通常，纸张回收涉及将废纸从办公室运送到工厂。 PaperLab使您可以加快此过程并将其本地化-也就是说，您可以按照他们所说的那样回收纸张，而无需离开票房。
  2. 安全销毁敏感文件

    如果使用高度机密的文档，请确保机密文档永远不会离开建筑物： PaperLab在处理过程中会完全不可逆地破坏纸张上的所有信息。 而且，这可以在不离开办公室且无需诉诸第三方组织的服务的情况下完成。 与碎纸机不同，PaperLab甚至从理论上消除了数据恢复的可能性，原因是墨水或色粉与纸纤维完全分离，并与废料一起收集在一个槽中。

  3. 纸张“定制”的广泛可能性

    创建纸张时，该设备允许您设置其属性。 您可以：
    • 改变纸的颜色；
    • 获得所需尺寸不超过A3的纸张；
    • 创建不同密度的纸张（从普通办公纸90 g / m 2到用于明信片的厚纸150-240 g / m 2 ），
    • 甚至调味纸
  4. 可持续发展-照顾未来

    直接在办公室进行回收有助于减少对环境的负面影响：
    • PaperLab有助于减少二氧化碳排放量。 产品运输点不包括在处理链中，用于发送废纸进行处理和订购新纸。 更少的汽车-更少的烟气。
    • 减少了购买新纸张的需求，这意味着我们正在节省地球上宝贵的资源-树木。
    • PaperLab无需用水即可生产纸张（不计算维持系统中一定湿度所需的少量水）。 通常，为了根据传统方案生产一张A4纸，大约需要一杯水。 因此，由于该机器使用“干式回收”， PaperLab每生产一吨纸可节省多达54,500升水 。
