Instal Metatrader 4 pada VPS di bawah Debian 8

Banyak pelanggan kami memasang terminal perdagangan di server virtual. Ini cukup nyaman dan manfaatnya jelas. Anda tidak perlu berada di depan komputer setiap saat, server VPS selalu online dan terminal perdagangan selalu menyala.


Karena kepentingan umum klien kami di terminal MT4 bersama dengan sistem Linux, kami memutuskan untuk menulis cara menginstal Metatrader 4 pada server virtual dengan akses melalui x2go.

Sisi server


Setelah terhubung ke server melalui ssh, Anda harus melakukan langkah-langkah berikut.

Tambahkan repositori dengan x2go:

$ echo "deb http://packages.x2go.org/debian jessie main" > /etc/apt/sources.list.d/x2go.list
$ echo "deb-src http://packages.x2go.org/debian jessie main" >> /etc/apt/sources.list.d/x2go.list
$ apt-key adv --recv-keys --keyserver keys.gnupg.net E1F958385BFE2B6E
$ apt-get update && apt-get install x2go-keyring && apt-get update

Instal x2go:

$ apt-get install x2goserver x2goserver-xsession

Edit file profil di direktori home Anda. Anda perlu menemukan garis "mesg n"dan menggantinya dengan "tty -s && mesg n":

$ vi.profile

Instal browser:

$ apt-get install iceweasel

Buat file untuk memulai instalasi Metatrader. Pada tahap saat ini, itu tidak akan berhasil. File ini akan diluncurkan sedikit kemudian, dari klien:

$ printf '#!/bin/bash\n\nwine \"Downloads/OBRTraderSetup.exe\"\n' > mtinstaller.sh

Perintah berikut membuat file untuk menjalankan Metatrader yang belum diinstal:

$ printf '#!/bin/bash\n\nwine \"/root/.wine/drive_c/Program Files/OBR Forex/OBR Trader/terminal.exe\"\n' > mtrunner.sh

File-file ini harus dapat dieksekusi:

$ chmod +x mtrunner.sh && chmod +x mtinstaller.sh

Sekarang masukkan anggur:

$ dpkg --add-architecture i386 && apt update && apt-get install wine wine32

Ini menyelesaikan pekerjaan pada ssh, Anda dapat melanjutkan untuk terhubung ke mesin jarak jauh Anda melalui x2goclient.

Jendela klien x2go sekarang akan terlihat seperti ini:


Kami klik pada sesi "Debian-Firefox", masukkan kata sandi, jendela browser terbuka berjalan di VPS kami, pergi ke situs forex dan masuk. Di x2to, clipboard berfungsi dengan baik, sehingga tautan dapat disalin.


Sekarang Anda perlu menginstal Metatrader 4 di VPS kami . Unduh program dari situs . Saat berayun, Anda dapat membuka "Akun Demo"; ke surat. Saat mendaftar, semua data yang diperlukan pada akun akan diterima. Browser tidak lagi diperlukan. Jika Anda tidak ingin menutupnya, tetapi ingin kembali ke tab yang terbuka nanti, di klien x2go klik tombol yang disorot dengan warna merah. Demikian pula, untuk membiarkan Metatrader terbuka, tetapi putuskan sambungan dari VPS Anda harus menggunakan tombol ini.


Kami memulai instalasi Metatrader 4. Klik pada sesi “Debian-Metatrader4-Installer”, masukkan kata sandi untuk VDS dan ikuti instruksi di layar.

Itu saja. Sekarang Anda dapat membuka sesi x2go "Debian-Metatrader4" dan Metatrader akan mulai pada mesin jarak jauh. Jika Anda ingin membiarkannya tetap berjalan, jangan tutup jendela, gunakan tombol pause di jendela x2go klien sebagai gantinya. Selamat berdagang!

PS Selain metode menyambung ke VDS dan menginstal Metatrader ini, Anda dapat menggunakan beberapa opsi lain dari artikel terbaru tentang Habr , tetapi tampaknya bagi kami ini adalah metode yang paling disukai, karena tidak ada biaya RAM untuk menjalankan program yang tidak perlu.

Untuk pengoperasian MT-4 yang stabil, konfigurasi server berikut akan memadai:
CPU 1 x2.2 GHz, RAM 1 GB, DISK 20 GB, satu alamat IP khusus
