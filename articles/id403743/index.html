Google AIY: Kit Pembuat Gadget Kontrol Suara



Google meluncurkan inisiatif AIY Projects untuk menggabungkan AI dengan DYI, yaitu, menggabungkan aplikasi perangkat lunak AI dengan kit perakitan gadget. Akibatnya, Google percaya, pembuat akan dapat membuat perangkat yang berguna yang akan membantu "memecahkan masalah nyata" dalam hidup. Gadget ini akan benar-benar berinteraksi dengan orang tersebut. Misalnya, kenali ucapan, jawab pertanyaan, dan jalankan perintah suara.

Proyek pertama adalah Voice Kit , sistem pengenalan ucapan demo dengan atau tanpa Google Assistant. Sebagaimana dinyatakan dalam deskripsi, selain semua fungsi Google Assistant yang biasa, di sini Anda juga dapat menambahkan pasangan pertanyaan dan jawaban Anda sendiri ke gadget. Perangkat ini dibuat dalam bentuk kubus kardus kecil dengan Rasberry Pi 3 di dalamnya. Merakit satu set 12 bagian akan memakan waktu sekitar 1,5 jam.

Seperangkat bagian tersedia secara gratis saat Anda berlangganan MagPi . Ngomong-ngomong, majalah ini tersedia dalam bentuk digital - ini menggambarkan banyak proyek luar biasa, termasuk dalam bidang pembelajaran mesin, yang dibuat menggunakan Rasberry Pi.

Kit ini juga dijual tanpa majalah di toko Barnes & Noble dan di toko-toko Inggris WH Smith, Tesco, Sainsburys dan Asda. Rupanya, ini adalah satu-satunya tempat di mana kit dijual. Set termasuk papan antarmuka, kabel, tempat plastik, speaker dan tombol. Papan Raspberry Pi 3 itu sendiri, kartu SD, serta obeng dan selotip yang diperlukan untuk perakitan, tidak termasuk. Komponen utamanya adalah kartu Voice HAT untuk pengenalan suara dan kartu Mikrofon Voice HAT. Ini adalah papan baru, yang belum tersedia untuk dijual secara terpisah.



Google menawarkan instruksi terperinci tentang cara membuat gadget. Dimulai dengan ikatan Raspberry Pi 3 dan papan Voice HAT,



melipat kotak ...



... dan ikat semua ini menjadi satu gadget.





Saran-saran berikut adalah untuk mengunduh gambar SD Kit Suara untuk direkam ke kartu flash.

Perangkat periferal terhubung ke konektor yang terbuka di slot kotak kardus: keyboard (1), mouse (2) dan monitor (3).



Setelah mengunduh dari kartu flash, perangkat siap terhubung ke Google Cloud Platform melalui koneksi WiFi yang dikonfigurasi. Di sana, Anda perlu mengatur proyek baru melalui Google Assistant API, dan kemudian perangkat akan mulai berfungsi secara normal, menerima permintaan suara dan mengenalinya di cloud.

Perangkat segera memahami perintah utama berbahasa Inggris:

  • Jam berapa sekarang?
  • Ceritakan lelucon
  • Buat musik lebih keras
  • Buat musik lebih tenang
  • Volume maksimum
  • Halo (jawab: "Halo untuk Anda")

Seperti yang telah disebutkan, Anda dapat menambahkan pasangan pertanyaan dan jawaban Anda sendiri, yaitu, program AI untuk tindakan yang diperlukan. Ini dilakukan dalam file ~/voice-recognizer-raspi/src/action.py . 

 # ========================================= # Makers! Implement your own actions here. # ========================================= import RPi.GPIO as GPIO class GpioWrite(object): '''Write the given value to the given GPIO.''' def __init__(self, gpio, value): GPIO.setmode(GPIO.BCM) GPIO.setup(gpio, GPIO.OUT) self.gpio = gpio self.value = value def run(self, command): GPIO.output(self.gpio, self.value)

Jika diinginkan, Anda dapat menggunakan Cloud Speech APIs lainnya, yang menerjemahkan ucapan suara ke dalam teks dan mendukung 80 bahasa, termasuk Rusia, dan pemrosesan file audio yang panjang.

Tentu saja, ini mendukung koneksi sensor tambahan. Misalnya, menghubungkan printer dan perangkat rumah tangga lainnya dengan perintah suara:

  • Hidupkan printer
  • Redupkan cahaya di dalam ruangan
  • Nyalakan ketel di dapur
  • Rebus telur (mis. Rebus air selama tepat empat menit)
  • Matikan karangan bunga meriah di pohon Natal



Proyek semacam itu tidak dapat dikalahkan secara gratis. Pertama, kit itu sendiri dijual terpisah, beberapa detail unik di sana. Kedua, Anda juga harus membayar untuk menggunakan cloud API. Meskipun yang kedua bukanlah prasyarat. Pada prinsipnya, pengenalan suara dapat dilakukan secara lokal menggunakan aplikasi TensorFlow.
