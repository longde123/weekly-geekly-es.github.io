云技术如何改变我们的治疗方式

拯救生命和保持健康的愿望是进步的最重要动力之一。 在信息时代，医疗机构和科学发展的机会越来越多。 其中许多机会是通过在该领域采用云计算而实现的。 我们将在今天的文章中向您详细介绍。


/ Flickr /国家眼科研究所/ CC BY

数据海


医疗服务的质量直接取决于在该领域研究或实践的各个阶段收集的数据的分析水平。 根据IDC 报告 ，医疗数据的年增长率为48％，这提出了新的医疗要求。 机构以大量的存储和IT基础架构。 有几个因素导致该领域指标的增长：

科学活动发展迅速 。 指导医生和蜂蜜制造商的新科学出版物的出现频率。 设备是空前的。 到2020年，医疗信息有望每70天翻一番 。

收集数据的新方法正在兴起 。 用于监视和诊断的移动设备正在成为重要统计数据的来源，这些统计数据使得可以进行正确，及时的诊断。 例如，惠普的子公司Aruba 声称 ，今年有87％的美国医疗保健提供商将使用物联网解决方案。

更多的患者-更多的数据 。 不能忘记发达国家的人口老龄化。 预期寿命正在增长-尤其是由于高质量和负担得起的医疗服务。 结果，卫生保健系统的负担（包括信息）仅在增加。

所有这些使医疗数据的处理，存储以及重要的保护工作变得复杂。 例如，去年的Thales Group研究显示，数据泄漏在医疗组织中并不罕见。

云技术将如何帮助您


云技术使医疗组织有机会重新考虑其工作流程并应对上述挑战和问题。 让我们看看由于新的基础架构，医疗保健行业正在发生什么变化。

医疗信息的交流得以简化 。 医疗组织必须存储有关患者的大量信息：病史，心电图，放射线照片，CT扫描等。为了应对如此大量的数据，越来越多的国家正在逐步引入所谓的电子病历（EMC），该技术是在云技术的基础上实现的。

所有患者数据都被数字化，并放置在IaaS提供者云中的远程服务器上。 在那里，使用加密算法和备份系统对它们进行了可靠的保护。 同时，该国任何诊所的授权医护人员都可以访问患者数据。 因此，一个人不必在所去的每个医疗机构中都有病历，而且医生更容易掌握病历。


/ Flickr / foto PD

云还促进了制药公司与其他组织的互动。 例如，Pfeizer 已经使用基于云的平台来存储来自合作伙伴实验室的敏感医学信息。 具有指标和元数据的云存储可帮助他们每年进行800次药物临床试验。

新的分析工具正在涌现 。 将基于云的分析系统与患者数据的电子目录集成在一起，将可以开展更多的研究项目。 克利夫兰诊所前主任托比·科斯格罗夫（Toby Cosgrove）在最近的一次采访中表示，云平台上数据的自动处理将成为提高医疗机构效率的关键。

一个例子就是自闭症的遗传学研究。 由Autism Speaks Foundation收集的数据（超过100 TB ）在Google Genomics云平台上进行处理。 它使用BigQuery之类的解决方案来分析Apache Spark，Cloud Dataflow或Oracle Grid Engine集群上的大数据。

研究人员认为，这样的项目将导致遗传学和医学交叉领域的技术发展。 在美国，已经出现了一个 “ 个性化医学 ”中心，该中心根据患者的遗传数据构成治疗方案。 说到治疗...

患者的诊断方法正在发生变化 。 云与移动技术齐头并进，使您可以远程完成以前需要个人参与的工作。 为此，医疗机构在云中部署了医疗数据交换系统，甚至是视频会议解决方案。 远程医疗服务通常基于后者。

现在，患者无需出门即可获得医疗建议，这对残疾人特别有用。 分析人士说，未来几年远程医疗市场将达到400亿美元。

同时，正在开发可以帮助医生进行“远程”诊断的技术。 2016年9月，以色列公司Beyond Verbal 提出了一个基于AI系统的项目，该项目可以通过患者的声音诊断疾病。 她的任务是认识到一个人无法听到的国际化的细微差别。 该解决方案接受了200万种音频录制的培训，这些语音使用40种不同的语言。

发展的另一个领域是用于物联网的智能设备。 云解决方案提供了这些创新的基础架构。 它们使医生可以远程监视患者的健康状况，从而减轻医院的负担并减少对紧急事件的响应时间。 十年前在英国进行的一项实验表明，使用远程监控设备的患者死亡率降低了45％ 。

云技术和物联网的发展导致了大型医疗设备的出现。 在2017年夏末，创可贴发布了一个“智能”补丁 ，该补丁将使医生能够获取有关农村地区和难以到达地区的人们的健康状况数据，这些地区通常是药物开发不完善的地区。

在诊断疾病时，云技术还使患者的生活更轻松。 特别是，现在要获得测试结果，您无需亲自去看医生-可以通过Internet找到它们。 面向患者的门户网站已成为一种规范，即使习惯了，但并非所有人都使用它们，在未来，我们可以期待它们的大规模应用。
