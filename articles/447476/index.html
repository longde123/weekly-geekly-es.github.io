Configuración de LaTeX en texto sublime


A continuación hay una guía para instalar el editor de texto Sublime Text , seguido de la posibilidad de usar el sistema de diseño de computadora LaTeX . Como beneficio adicional, le enseñaremos a entender el lenguaje de Julia .


Instalar MiKTeX


  1. En el sitio, haga clic en la pestaña Todas las descargas
  2. Seleccione Net Installer dependiendo de la profundidad de bits de su sistema.
  3. Haga clic en Descargar y espere a que se descargue el instalador en línea.
  4. Después de leer detenidamente el acuerdo de usuario, seleccione la casilla de verificación Descargar MiKTeX
  5. Cuando use un proxy, ingrese sus complementos en la configuración de Conexión ...
  6. En el siguiente paso, seleccione Completar MiKTeX
  7. Al seleccionar un servidor (preferiblemente uno de los principales rusos) y especificar un directorio, espere a que se carguen los archivos necesarios
  8. Siga el paso 4 nuevamente, pero esta vez seleccionando Instalar MiKTeX
  9. Una vez completada la instalación, inicie TeXworks a través de Inicio
  10. Abra cualquier documento .tex en él o cree uno nuevo
  11. Seleccione pdfLaTeX en el menú desplegable y haga clic en el triángulo adyacente en el círculo verde.

Esto compilará y creará el archivo PDF. Con TeXworks, puede comenzar un trabajo completo, por ejemplo, con LaTeX puede recolectar un diploma o disertación de manera muy efectiva ( plantillas listas para usar)


Es mucho más conveniente usar LaTeX a través de Sublime Text. Además del resaltado conveniente y la finalización automática, aquí puede ver la imagen o la fórmula escrita sin compilar, es decir, ver el resultado durante la escritura. Por cierto, en Sublime Text puede escribir (y si agrega los complementos apropiados, luego ejecutar) los códigos de programa de muchos lenguajes de programación y marcas: C / C ++, Java, MATLAB, PHP, HTML ...


Instalar Sublime Text 3


  1. Descargue e instale Sublime Text 3
  2. Descargue e instale Sumatra PDF . Este programa liviano le permite ver documentos PDF, DJVU, FB2, etc.
  3. Después de iniciar Sublime Text, abra la línea de comando presionando Ctrl + shift + p y comience a escribir Control de paquete: Instalar paquete , y cuando aparezca, haga clic y espere a que se establezca la conexión a los repositorios.
  4. Si por alguna razón no hay Control de paquete, abra la consola presionando * ctrl + ` (() y copie el hechizo en el parseltang, que se puede encontrar en la página del complemento Control de paquete , presione Entrar y reinicie Sublime Text.
  5. Ingrese LaTeXTools en la lista de complementos y, cuando aparezca en la lista, haga clic y espere a que se complete la instalación.
  6. A continuación, iniciamos la configuración predeterminada: Preferencias / Configuración del paquete / LaTeXTools / Restablecer la configuración del usuario a la predeterminada , y luego Comprobar el sistema allí
    Si todas las etiquetas son verdes, Sublime Text se hizo amigo de Sumatra y LaTeX, y ahora puede crear o editar documentos de texto. Cuando presiona ctrl + B, se realiza una traducción y, si no hay errores, se abre el PDF paginado. Ctrl + shift + B: selecciona el tipo de ensamblaje.

Más información sobre LaTeXTools


Encontrar manuales de LaTeX en línea será fácil para usted (por ejemplo: Tutorial , Wiki ). También será útil usar el editor de fórmulas en línea hasta que los comandos se almacenen en la memoria.


Julia en texto sublime


En Package Control: Install Package, escriba Julia , descargue el complemento, y ahora Julia aparece entre los otros idiomas en la pestaña Ver / Sintaxis , lo que significa que ahora puede escribir programas en este idioma con retroiluminación y letras griegas (\ delta + presionar tab ), así como guardar archivos en formato .jl


Al guardar archivos, será conveniente dividirlos en módulos dentro de: 


 module somename # functions, variables # and some things end


El código escrito en texto sublime y guardado en formato .jl se puede ejecutar en REPL 


 #     cd("C:\\Users\\User\\Desktop") #   include("MDPSO.jl") #       #      PSO.parabol([2,3,5])


Esto concluye una pequeña guía. Todo diseño conveniente y una iluminación agradable!
