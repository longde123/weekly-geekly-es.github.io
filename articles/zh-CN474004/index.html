从Norilsk到Riyadh：金士顿工业温度microSD UHS-I存储卡的真实案例

三年前，当我们对工业用存储卡进行评论时，在评论中希望不要谈论无人机和相机-他们说，这不是此类存储卡的典型应用。 好的，我们告诉自己并记录在内容计划中-使用行业案例发表出版物。 但是，碰巧的是，在金斯敦新闻出版物的背后，这个项目很长一段时间都被列入了后备名单，直到在哈布雷（Habré）遇见了俄罗斯公司DOK 。 自2016年以来，她一直在使用这些存储卡，并且使用了数百种。 顺便说一句，在她通过Yenisei的40吉比特无线电桥中，这是无线通信的世界纪录 ，所安装的是金士顿工业温度microSD UHS-I存储卡。



案例研究-毫米波宽带



在2016年年初，也就是恰逢我们评论中的金士顿工业级存储卡问世之时，电信市场上骨干无线链路的速度正在发生质的飞跃。 第二代无线电中继站以1 Gbit / s的速度在2010-2015年占主导地位，应该将警棍传递到能够以10 Gigabit以太网标准运行并以10 Gbit / s的速度传输数据的第三代无线电链路。


无线电通过Igarka的Yenisei桥接2x20 Gbit / s。 资料来源：LLC DOK

顺便说一句，要使无线信道的传输特性类似于光缆，在全球范围内至少需要做两件事：为10 Gb以太网（10GE）无线通信创建新的元素基础，并分配足够宽的频带以使其适合10千兆数据流。 该范围是频率范围为71-76 / 81-86 GHz，在美国监管机构FCC的帮助下，该频率范围于2008年在美国分配。 很快，这个例子被包括俄​​罗斯在内的世界上几乎所有国家的监管机构效仿（自2010年以来，射频通信部允许免费使用71-76 / 81-86 GHz范围）。

在2016年，能够以10 Gb / s的数据传输速率调制QAM 256无线电信号的MMIC芯片（微波单片集成电路）终于出现在世界市场上，并且比赛开始了-谁将是第一个在市场上推出此类无线电中继设备的商业模型10GE。 令人惊讶的是，此类无线电链路的第一个生产模型是在俄罗斯圣彼得堡工程公司DOK中创建的，并在2017年巴塞罗那世界移动大会（MWC 2017）上展出。 好吧，为什么不呢？ -一样，亚历山大·波波夫（Alexander Popov）正是在圣彼得堡发明了收音机（尽管这种优势有时归功于马可尼或特斯拉）。

今天，在2019年，10GE无线广播已成为事实上的行业标准。 由于其高吞吐量，单个10 Gb / s无线电中继线通常用作整个住宅社区或大型工业区的通信服务。 蜂窝运营商愿意将10GE无线电链路用于4G / LTE基站之间的骨干网，因为 它们可将基站与移动运营商数据中心的参考时钟同步，这对于将多媒体流量传输到智能手机和平板电脑非常重要。 除了通过无线10 Gb以太网频道进行数字电话和Internet访问之外，还广播了数百个数字电视频道，并且还有来自CCTV摄像机的数据流。

《哈伯》读者会说：“-这一切都是有趣的，但是金士顿存储卡与它有什么关系呢？” 但是现在我们将继续进行此操作。

无线电中继设备内部的“黑匣子”


工业温度microSD UHS-I存储卡安装在DOK生产的PPC-10G无线电中继站控制模块中，作为配置文件和设备状态记录的文件存储。 所有关键操作参数都会全天候写入卡中：通道中的数据传输速度，接收信号电平（RSL，接收信号电平），外壳温度，功率参数等等。 根据制造商的规则，卡必须在设备运行至少一年后存储此类数据，然后最新数据将被旧数据覆盖。 实践证明，要满足此要求，8 GB的存储卡就足够了，因此PKD现在只使用这种存储卡。 一组两个无线电中继站需要两个工业温度microSD UHS-I存储卡，例如 每张卡片中都放有一张卡片。


PPC-10G无线电中继盒，带金士顿工业存储卡的模块。 资料来源：LLC DOK

网络工程师或电信运营商的管理员定期通过FTP从存储卡下载日志，或在Web界面中查看日志。 因此，对信道容量，无线电中继站内部组件的稳定性进行统计评估。 在设备出现故障或过渡到低数据速率模式时，来自日志的信息尤为重要。

使用客户提供的日志中的信息，PKD的技术支持专家可以诊断问题并提出解决方法，以尽快解决该问题。 例如，看到接收信号电平（RSL）从某个时刻起发生变化，我们可以得出结论，很可能是天线“丢失”了指向对方。 有时这是在飓风吹过，冰从电信塔的上部结构掉落到天线上之后发生的。

电信运营商在购买相当昂贵的10 Gb无线中继设备后，就依靠“即插即用”原理，依靠其所有组件的高可靠性。 存储卡也不例外。 这里的一个重要因素是维修设备的设备的复杂性。 在大多数情况下，范围为71-76 / 81-86 GHz的无线电链路安装在电信塔，建筑物和建筑物的屋顶上。 显然，在俄罗斯冬季的条件下进行攀登以用冰冷的塔更换部件是一项艰巨而危险的工作。 尽管存储卡不是PPC-10G站中的关键组件，但如果发生故障，无线电中继线将继续工作，但是它将失去记录设备状态日志和通信通道的能力。 因此， 金士顿工业温度microSD UHS-I卡的可靠运行对于无线电链路的制造商和以电信运营商为代表的客户都至关重要。


特写-带有Kingston工业存储卡的PPC-10G站模块。 资料来源：LLC DOK

“我们在俄罗斯设计和制造无线电中继站已超过10年，在此期间，我们尝试了来自不同制造商的存储卡。 有些卡可以使用一年，有些则可以使用几年，但是随后必须进行远程格式化，有时甚至无法解决问题，这引起了我们设备购买者的投诉。 当10 Gb型号PPC-10G于2016年投入生产时，我们向供应商Superwave（圣彼得堡）寻求建议。 他们建议金士顿的工业存储卡说，它们绝对不会有问题。 从那以后，没有一个金士顿卡发生故障，并且我们已经安装了大约一千个。 DOK公司董事Daniil Korneev说：“尽管事实是，电信设备是在非常恶劣的条件下在露天条件下全年运营的，但事实并非如此。”

如何绕过存储卡和其他组件的温度限制


如果查看有关工业温度microSD UHS-I存储卡的技术规格的页面，则可以看到其保证的操作和存储温度范围的限制：-40°C至+ 85°C。 但是，如果无线电中继站在俄罗斯北极地区工作，而夜间在-50°C甚至更低的温度下怎么办？ 或者相反，在非洲某处？


带有金斯敦存储卡的无线电中继站PPC-10G，位于Yamalo-Nenets Autonomous Okrug的Purovsky区Tarko-Sale市。 资料来源：LLC DOK

冬季中继站为冬季条件提供了一个自动加热器，即使在严霜条件下，箱体内的温度也高于0°C。 在“冷启动”的情况下-与在北极启动汽车技术类似-加热器是第一个启动的。 它阻止包含电子组件，直到车站建筑物内的温度上升到可接受的极限为止。
现在我们要达到温度范围的上限。 考虑到每个制造商，包括俄罗斯制造商，都希望在世界范围内销售其无线电链路，因此该设备应在烈日下能正常工作。 为了进行热带改造，在DOK站中安装了扩展的散热器系统，从而在整个设备箱内分配热量。


PPC-10G无线电中继站（当然带有金斯敦存储卡）正在安装在阿联酋的一幢高层建筑中。 资料来源：LLC DOK

“作为对金士顿存储卡的评论，我要指出，其规格中的存储温度下限为-40°C，具有较大的裕量。 来自俄罗斯北部地区的客户屡屡碰到无线电中继站在较低温度下被关闭的情况，而下次打开设备时我们从未记录到存储卡故障。 至于温度上限，我们再次从金士顿存储卡获得的机箱内部温度记录显示，中东的无线链路温度未显示出+ 80°C的过量值。 因此，人们担心太阳会加热站点和它们的组件，使其超出我们在利雅得或阿治曼的客户的允许极限，这种担心完全没有根据。
