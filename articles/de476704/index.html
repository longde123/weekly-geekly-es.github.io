So automatisieren Sie das Layout von E-Mails mit denselben Elementen: Wir verwenden intelligente Objekte



Das Layout ist eine der zeitaufwändigsten Phasen bei der Erstellung eines E-Mail-Newsletters. Oft müssen Sie in Briefen eine große Anzahl ähnlicher Blöcke einfügen, z. B. Bilder mit Beschreibungen (relevant für Online-Shops mit Produktkarten). Solche Aktionen erfordern viel Zeit und Mühe.

DashaMail verfügt über ein Tool zur Automatisierung solcher Routineaufgaben. Dies sind die intelligenten Objekte eines Blockeditors. Wir werden Ihnen sagen, wie sie funktionieren.

Was sind intelligente Objekte?


Wenn Sie einen Buchstaben setzen, müssen seine Elemente in der Regel jedes Mal neu eingerichtet werden. Auch wenn Sie eine Struktur erstellt und jedes Mal kopiert haben, muss der Inhalt ständig manuell aktualisiert werden: Ersetzen von Bildern, Beschreibungen, Links usw. Bei einer großen Anzahl von Objekten wird das Layout zeitaufwändig.

Ein intelligentes Objekt ist ein Buchstabenelement mit den von Ihnen benötigten Blöcken (Bild, Text, Schaltfläche usw.), die Daten über Variablen von externen Quellen empfangen können. Das heißt, bei intelligenten Objekten müssen Sie die Integration mit einer externen Site einmal konfigurieren, und dann können Sie Objekte mit ein paar Klicks in Buchstaben einfügen - und sie sind bereits vorhanden.

Damit alles auf diese Weise funktioniert, müssen Sie einen Link zu der gewünschten Seite auf einer externen Website bereitstellen. Dies kann beispielsweise ein Produkt in einem Online-Shop sein. Das System lädt dann automatisch die erforderlichen Daten und wendet sie an den angegebenen Stellen des Briefes an.



Wie funktioniert das?


Um ein intelligentes Objekt zu verwenden, müssen Sie seine Struktur mit den erforderlichen Blöcken erstellen, z. B. einem Bild, zwei Textblöcken und einer Schaltfläche. Danach müssen Sie die Regeln konfigurieren, nach denen das System Daten in das Objekt lädt. Dies wird mithilfe von Variablen implementiert.



Regeln werden über den allgemein anerkannten Mechanismus zum Auffinden von Elementen in HTML, CSS-Selektoren und Attributen konfiguriert. Ein weiterer interessanter Punkt: Sie können intelligente Objekte mit einem Rollover-Effekt versehen (wenn Sie den Mauszeiger über das Bild halten, ändert sich dies). So können Sie beispielsweise jedes Produkt auf einmal von zwei Seiten zeigen, und das alles direkt im Brief.



Intelligente Objekte unterstützen das adaptive Layout. Die Technologie funktioniert flexibel: Sie können Parameter wie Schriftarten und Textausrichtung auf verschiedenen Geräten konfigurieren sowie Inhalte in verschiedenen Umgebungen anzeigen / ausblenden. Beispielsweise können einige Elemente nur Desktop-Benutzern angezeigt werden, während andere für diejenigen angezeigt werden können, die den Brief von einem Smartphone aus anzeigen.



Was sind die Vorteile von Smart Objects?


Der offensichtlichste Einsatz der Smart-Object-Technologie in E-Mail-Newslettern ist die E-Commerce-Branche. Die Unternehmen hier haben viele Produkte, über die Sie direkt in den Briefen sprechen können.

Dieses Tool ist auch für Content-Projekte nützlich: In Briefen können Sie auf wunderbare Weise Ankündigungen und Vorschauen von Artikeln erstellen. Im Allgemeinen können intelligente Objekte für jedes Unternehmen nützlich sein, das sofort Informationen über eine große Anzahl von Waren, Dienstleistungen oder Angeboten erhalten möchte.

Die Zeit, die Sie für das Layout benötigen, steigt mit der Anzahl der Elemente desselben Typs. Alles ist logisch. Dementsprechend ist die vorgeschlagene Lösung am effektivsten für Briefe mit einer großen Anzahl ähnlicher Objekte.

Zum Beispiel, wie unsere Tests gezeigt haben, dauert der gesamte Zyklus des Schreibens einer Mailingliste mit dem Testen und Senden, wenn eine Vorlage mit 12 Produktkarten verwendet wird, nicht länger als 20 Minuten. Ohne intelligente Objekte dauert die manuelle Codierung eines ähnlichen Briefes mindestens eine Stunde.

Es ist auch wichtig, dass die Wahrscheinlichkeit von Fehlern beim Layout von Elementen desselben Typs verringert wird, da die manuelle Arbeit durch die Automatisierung erheblich reduziert wird („Ich habe das Element kopiert und vergessen, den Link auf der Schaltfläche zu ersetzen“). Infolgedessen nimmt die Qualität von Newslettern zu, was deren Wirksamkeit bedeutet.
