这篇文章很短，但对于在命令行上花费大量时间的足球迷来说，它会派上用场

使用npm wowcup 软件包，您可以在终端上直接查看游戏结果。 信息取自api.football-data.org

我们放了一个袋子... 

sudo npm install -g wowcup

我们正在尝试...

最近24小时 

 wowcup fixtures --last




接下来的24小时 

 wowcup fixtures --next




wowcup装置-播放直播
wowcup排名 -所有游戏
wowcup排行榜--table GROUP-用于特定组（用-h替换GROUP）

要删除红色文本，请添加--quiet 

 wowcup fixtures --last --quiet

仅此而已，谢谢大家。
