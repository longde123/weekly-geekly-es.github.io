Di AS, beberapa bank terbesar telah mengambil langkah menuju model terbuka dengan meluncurkan hub pengembang. Pada bulan Februari 2016, Visa mengumumkan peluncuran Pengembang Visa, sebuah inisiatif penting untuk mengubah jaringan pembayaran ritel terbesar di dunia menjadi platform terbuka. Untuk pertama kalinya dalam hampir enam puluh tahun sejarah perusahaan, pengembang aplikasi perangkat lunak telah memperoleh akses terbuka ke teknologi pembayaran, produk dan layanan dari pemimpin industri.

Pada bulan Maret 2016, bank holding Capital One meluncurkan portal DevExchange , repositori dengan alat dan API untuk pengembang, yang memungkinkan mereka untuk menggunakan sumber daya dan pencapaian lembaga keuangan di bidang menciptakan perangkat lunak untuk meningkatkan layanan mereka sendiri.

Kemudian, pada November 2016, konglomerat finansial Citigroup meluncurkan platform hub hub pengembang global serupa yang memungkinkan perbankan terbuka. Perwakilan Citi mengatakan dalam siaran pers resmi bahwa peluncuran API ini menandai transisi evolusi teknologi Citi ke arsitektur terbuka. Langkah ini bertujuan menyederhanakan kerja sama dan membangun kemitraan dengan perusahaan fintech dan merek konsumen di seluruh dunia, yang pada akhirnya harus memengaruhi peningkatan kualitas layanan.

Menggunakan Hub Pengembang API global, Citi memberi para pengembang akses ke delapan kategori API yang berbeda, termasuk manajemen akun, pembayaran langsung, transfer uang ke lembaga, program hadiah Citi, pembelian investasi, dan otorisasi akun.

Pada Mei 2017, BBVA meluncurkan program perbankan terbuka, yang menyediakan akses komersial ke delapan API untuk pertama kalinya. Menurut perwakilan bank, peluncuran Pasar API BBVA adalah hasil dari lebih dari satu tahun kerja sama antara bank Spanyol dan pengembang dan perwakilan bisnis untuk mengoptimalkan penyediaan layanan Open API. Simple , sebuah bank digital Amerika yang diakuisisi oleh BBVA pada tahun 2014, adalah salah satu perusahaan pertama yang mendapatkan akses ke Pasar API BBVA.

Wells Fargo dan JPMorgan Chase memiliki penawaran serupa. Semua ini menunjukkan transisi seluruh industri menuju pemisahan fungsional jaringan pembayaran dan platform perbankan. Jadi kita dapat dengan aman mengatakan bahwa perbankan terbuka adalah tren baru, yang, meskipun dengan perbedaan tertentu dalam kondisi dan isi kasus, entah bagaimana akan menjadi model operasi utama di pasar Eropa dan Amerika di masa mendatang.
