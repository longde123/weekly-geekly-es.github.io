Flexbox Infografis teknologi

Saya percaya banyak orang tahu teknologi pengaturan huruf css sebagai flexbox. Tidak sedikit manual dan manual telah ditulis tentang teknologi ini. Ini sederhana, sangat memudahkan pengembangan grid proyek web dan didukung oleh banyak browser modern. Ini membuatnya menjadi teknologi yang populer di kalangan pengembang web.

Mempelajari konsep-konsep baru untuk saya, saya mencoba selingkas mungkin, tetapi sementara itu secara informal mengungkapkan apa yang sedang dipelajari dalam bentuk grafik, hubungan beberapa entitas dengan yang lain, hubungan dan perilaku mereka. Dan flexbox tidak terkecuali bagi saya. Untuk beberapa malam, saya mengumpulkan sendiri ikhtisar "lembar contekan" tentang teknologi, yang untuk waktu yang lama tergantung di dinding saya di depan mata saya, sampai saya ingat semua poin utama pada teknologi.

Kolega dan teman saya, yang profesinya terkait pengembangan web, sering diminta untuk membagikan "lembar contekan" untuk penggunaan pribadi, dan ini akhirnya membuat saya berbagi catatan saya dengan komunitas habr.

Infografis dibagi menjadi 3 lembar, untuk memudahkan pencetakan dan penskalaan. 1 lembar utama termasuk keseluruhan ikhtisar teknologi, dan 2 lembar tambahan cukup membagi lembar utama menjadi 2 bagian.
