互联网上的家庭监控和远程查看

最近在Geektimes它发表了一篇关于监视居住的一篇文章- “国内监听”。它的作者无法通过Web界面配置摄像机的查看。我决定分享我在这件事上的经验。此外，带有图片的实际文章还介绍了如何在家里轻松便捷地进行设置。我还记得在设置过程中遇到问题，因此我决定详细介绍整个设置过程。注意，很多图片。

为了避免通过重新安装破坏系统，他从一开始就在虚拟机上进行了所有操作。

您需要：

-具有Win7或更高版本的计算机；
-软件-上一篇文章（在制造商的网站上下载）和WebView模块中提到的Securos Lite。而且，如果第一个可以轻松地在制造商的网站上下载，那么这里是第二个的链接，我不记得自己是如何搜索的；
-相机 当然，在这里，每个人都应该根据图像质量/价格/目标等进行选择。一般建议：如果相机带有Onvif标签，则很可能会在任何软件中启动。如果在站点，文档或论坛上对摄像机的描述说明了如何将其纳入VLC，那么您也可以以99％的概率从此类摄像机获取视频，至少在Securos中，有一种机制可以从显示的任何摄像机接收视频图像在VLC中。HDReady和FullHD的差异会对存档产生很大影响，但是如果有几十场演出是免费的，则最好采用FullHD。

启动相机


在面板上，我们找到并单击“ IP设备管理器”按钮。出现该管理器的窗口。在下半部分有一个搜索按钮。他找到了我自己的Beward。唯一的事情是，我立即将模型从默认默认手动修复为BDSeries。如果未找到摄像机，则可以手动尝试对其进行配置，在屏幕截图中我选择了ONVIF（现在有一半的摄像机都支持它，即使摄像机包装盒上没有这个词，那么我建议首先尝试将其配置为ONVIF）。单击“添加”，在顶部我们输入用户名/密码，单击“应用”，关闭管理器。摄像机立即出现在媒体客户端中。



打开档案记录


在这里，最好立即检查她是否写了存档（在媒体客户端上记录下来，几分钟后将其删除，转到存档）。如果在“记录”按钮上出现一个警告三角形，则说明所有情况都不好，则不会写入存档。根据个人经验，我知道通常在磁盘空间不足时会发生此问题。结果，事实证明我有一个问题，因为我只为存档分配了20 GB的磁盘。而且由于我们正在设置商业产品的免费版本，因此对于严肃的系统，可能存在默认设置。在Internet上搜索，在注册表中搜索设置，甚至与它们进行一小段对话。支持人员给了我必要的设置。在32位Windows中，这是HKEY_LOCAL_MACHINE \ SOFTWARE \ ISS \ Xpress \ Niss400 \ Video分支（在64位系统上：HKEY_LOCAL_MACHINE \ SOFTWARE \ Wow6432Node \ ISS \ Xpress \ Niss400 \ Video）。我这样设置参数：MinDelMB-1700，FreeMB-1350。

最后步骤


为了通过Web界面进行授权，您不能使用超级用户Securos的帐户（由于密码不安全而被写入）。因此，要进一步工作，必须手动创建用户。再次，打开对象树，选择SecurOS Lite对象，单击“创建”，然后在菜单中选择“部门”。在它下面，我们立即创建一个用户。最好立即写出适当的名称，例如，我问webview。接下来，转到树中的“用户权限->高级用户权限”，然后在“用户”部分的“设置”窗口中，选择我们的用户并为其设置密码。

返回浏览器，输入webview /密码，单击“继续”。出现监视器选择页面。我们有一个，我们选择它。如果浏览器是InternetExplorer，则将提供ActiveX进行安装。最好将其安装后，网络上的负载急剧减少（据我了解，H.264流正在追赶网格）。通常，现在您可以从任何浏览器（包括移动浏览器）访问此基于Web的界面，并观看视频，观看使用运动检测器编写的存档。为了给相机设防/撤防，我使用了上面的宏。唯一的缺点是，如果您使用移动流量，那么会发现一笔不菲的开销，在第一个假期，我记得我只是通过WiFi在酒店看摄像机的。顺便说一句，该服务器还能够通过rtsp传递压缩的视频（消耗很少的流量，例如通过IE中的ActiveX）。如果某人更高级，并且知道如何将rtsp链接添加到VLC，那么一切都将起作用，包括在智能手机上。但这当然是仅用于观看实况视频，不存档和不进行摄像机控制的选项。

到底发生了什么


已经写了很多东西，但是当我第一次将它安装在真实系统上时，我启动起来更快（由于虚拟机，我的录制没有问题，并且没有其他问题），我只记得愉快地回忆了它的运行速度。

现在，我将描述它现在如何对我起作用。要使用所有这些功能，事实证明对于我来说，使用Windows客户端更加方便。我做了以下工作：

-我配置了访问权限以在家中通过OpenVPN工作（实际上，我已经拥有了）。对于公司中没有的用户，可以通过TeamViewer使用VPN。我也确信您可以通过Internet轻松配置它，但是那里需要一些小技巧（使用静态IP或使用无IP）。如果有问题，请联系项目经理，我将尽力提供帮助；
-在Securos的家中，他创建了另一台类型为“ Operator Workplace”的计算机。在其下创建了一个媒体客户端。我尝试了旧的Monitor，但是它们之间的差异很大。 Media Client将处理器的负载减少了一个数量级，更加令人愉悦和方便；
-在工作中将Securos置于“操作员工作场所”模式。从同一个分发工具包开始，直到现在，我们才不快速单击页面，而是找到他要求安装类型的页面，在组合框中，我们选择这是操作员的工作场所；
-接下来，运行Securos，输入IP或名称（如果此名称解析），单击“连接”。登录名可以相同：webview。出现“媒体客户端”窗口。唯一不好的是您只能在家用计算机上对其进行配置。即我首先使其全屏显示，这很不方便。我必须使用TeamViewer连接到家用计算机，并配置应在何处以及如何显示媒体客户端。

我现在的工作算法是这样。我上班时，在媒体客户端中按“布防”按钮，然后在下班前（或在家）按撤防。我快速查看4倍/ 8倍/ 16倍可疑之处。

此外，我还在笔记本电脑上为自己设置了一个“操作员的位置”（度假时，我检查了家里的一切是否良好，以及那里的花浇水如何）。这比尝试以智能方式看摄像机要方便得多。在这里，您可以思考并扭转它。同样，在假期中，因此，我设置了通过电子邮件发送有关检测到相机移动的通知（我也想拧紧旧的SMS手机，但后来我意识到，现在对SMS或Gmail的信都无关紧要，无论如何都会出现推送消息） 。有文档，F1，您可以弄清楚。我什至想出了如何编写简单的程序来取乐，但这对我来说并不方便。现在，我想添加一个摄像头以扩大覆盖范围，但是我的厨房很小，所以我正在用鱼眼镜头慢慢观察奇迹，软件中已经可以使用它们了。
