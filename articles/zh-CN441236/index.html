Habraiting 2018：2018年最佳材料

哈Ha

这篇文章是有关哈布雷（Habré）文章（对于对此问题的技术方面有兴趣的人的第一和第二部分）的生命周期的出版物的合乎逻辑的结论，因此，它成为了一个相当有趣的统计分析工具。 事实证明，该方法非常有用，它使您可以通过各种参数查找文章，例如，“质量”最高（评分与观看次数之比）的文章，评论最多，争议性最大的文章，评论最多的材料等。



现在是时候从中获得一些好处并编制2018年文章的统计评分了。 理想情况下，在新年开始之初这样做会很好，但精明的想法的确会发生，但是要迟到。 但是迟到总比没有好，它会让您重新阅读一些对一次错过的人有用的文章。 在文本末尾有一个小的“秘密奖励”，供那些会好奇的人使用。

那些对发生的事情感兴趣的人，请在猫的陪伴下。

资料收集


想要立即查看结果的用户可以跳过此部分。

与上一部分一样 ，我们将使用Python和BeautifulSoap库来收集有关文章的信息。 有了它的帮助，我们将获得有关出版时间和文章标题的信息。 所有其他数据均已从我们获得，希望的人可以参考上一部分。

结果，我们得到了以下功能： 

def get_article_data(page_link): data_html = get_as_str(page_link) soup = BeautifulSoup(data_html, 'html.parser') str_log = "" # Get datetime t = soup.find("span", {"class": "post__time"}) if t is None: return str_log += t['data-time_published'] + ',' # Get title and link url = soup.find("meta", property="og:url") url_str = url["content"] if url else "-" title = soup.find("meta", property="og:title") title_str = title["content"] if title else "-" str_log += '{},"{}",'.format(url_str, title_str.encode('utf8')) # Extract other parameters (rating, etc) str_log += extract_as_csv(data_html) print(str_log) return str_log

该函数的结果将是格式为'2007-06-12T12：58Z，https：//habr.com/en/post/35001/，“您对窃有何感想？”，投票：0，选民加分：0，选民最低：0 ，书签：0，浏览次数：161，评论：1'，这很容易保存到csv文件中。

正如他们所说，这是技术问题。 对我们来说幸运的是，所有有关哈布雷的文章都采用了端到端编号，因此使用戳戳法找到所需的日期范围并不困难（顺便说一下，那些希望进行“数字考古”研究的人可以阅读编号为https://habr.com/en/post/1的文章） /和https://habr.com/ru/post/2/ ，已于2006年发布）。 然后只需要循环启动解析器，然后将其放置一整夜（为了不加载服务器，在请求之间添加了暂停，因此该过程不是很快）。

对公司博客存有疑问，但事实证明，它们的编号相同，只是在您打开链接时才会发生重定向。 这样就可以完全下载所有数据，而不会丢失任何内容。

对于整个示例，我们将提出不同的请求，并查看发生了什么。 第二部分描述了该技术和源代码，希望的人们可以更详细地研究该技术。
