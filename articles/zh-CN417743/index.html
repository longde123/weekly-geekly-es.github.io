传奇放大器-历史趋势剖析：冷晶体管的声音

根据大众的需求，我重新打开了“传奇放大器”的周期，以延续过去七十年来放大技术领域最令人印象深刻的发展。



我的一些读者正确地指出，尽管我对所谓的 在这个周期中，“管声音”对半导体的发展几乎没有引起什么注意。 这篇文章将纠正这种公然的不公正行为。

原谅“暖灯”的爱好者，但实际上，半导体UMZCH是公认的电路杰作，工程师，音乐爱好者和发烧友，远远超过了灯泡。

Quad 303-60s半导体传奇


可以相信，在放大电路中大量使用半导体的头十年（即在1960年代），没有任何设备可以与电子管的再现精度相提并论。 尽管有这种共同的信念，但我还是熟悉几种此类设备，其中最好的就是Quad 303。

该放大器被60年代晶体管电路发展的顶峰称为。 音频工程师Peter Breuninger认为，正是Quad 303电路为成功应用了几十年奠定了基础。



UMZCH的额定功率为每声道45瓦，而不管扬声器的标称电阻（在4〜16欧姆范围内），这对于1968年来说是非常令人印象深刻的（音乐会系统可以拥有这种功率）。



在那些年里，这款UMZCH作为一种通用设备而享有盛誉，可以被专业人士和普通音乐爱好者使用。 限幅器保护占据了一个特殊的位置，这使设备在重负载下可以长时间连续工作。



Quad 303的创建者设法降低了THD和IMD，以至于当代人很难准确地测量它们。

该设备的主要特征如下：

  • 输出功率：45 W；
  • 频率范围：30 Hz〜35 kHz;
  • 总谐波失真：0.1％;
  • 输入灵敏度：0.5V；
  • 信噪比：100 dB；
  • 宽度：120毫米; 高度：159毫米; 深度：324毫米 重量：8.2公斤

用户很快就意识到了双重单声道的好处，并开始成对使用Quad 303，例如2个整体。 事实上，根据现代人的说法，该设备在立体声和单声道模式下均能稳定稳定地工作。

克雷尔KSA-100-业余无线电的杰作


到1980年代初，晶体管系统的声誉已跌至最低点。 除了Ampzilla和马克·列文森（Mark Levinson）这样的大型制造商的一些怪物，他们无法面对面交流，市场上没有可靠的三极管放大器。 当时的用户根据诸如Stereophile之类的档案杂志上的出版物来判断，抱怨晶体管放大会使声音失真的程度几乎比当时不完善的音响系统更大。



在发烧友当中，有一些人（并非毫无道理地）相信Hi-Fi中“硅”声音和半导体的未来，其中包括Dan D'Agostino。 可以说晶体管High End始于这位音频工程师。 今天，在发烧友媒体的页面上，这个人被称为是一个悲哀和权威的英雄，他是电路不折衷的最有经验的专家之一。


KSA-100S的最新版本

一切始于对几乎没有收入的热情。 业余无线电爱好者和音频工程师坚信，在客观特性和主观确定的参数上，有可能制造出超过电子管的UMZCH晶体管。 这样的放大器是Krell KSA-100。


KSA-100 mk2

该设备的开发始于1979年，而在1981年，KSA-100在消费电子展（CES）上获得了主要奖项。 电路成为西方许多业余和专业发展的起点，它将更好地说明电路的功能。



使用高质量的组件和电路解决方案在时间上可谓微不足道，从而可以实现以下特性：

  • 功率：100W（8欧姆-20 dBW）;
  • 频率范围：20Hz – 20kHz；
  • 频率响应不均匀度：+ 0.0 / –1dB（20Hz – 20kHz），+ 0.0 / –3dB（1Hz – 150kHz）；
  • 总谐波失真：<0.1％，1kHz; <0.5％，20kHz（满功率）;
  • 摆率：100V / µs；
  • 输入灵敏度：1.4V RMS;
  • 阻尼：> 60；
  • 输入阻抗：47 kOhm。

应当指出，该设备已经生产了二十多年，并且其价格已经改变了好几次，直到1994年它一直稳定增长，达到了5500美元的峰值。 来自“绝对声音”，“ Stereophile”和“ What hi-fi”之类的出版物的专家毫不犹豫地选择了各种上音词，描述了这位硅传奇作家的功绩。 在其中一种困扰中，达戈斯蒂诺被称为“一个证明半导体放大器能够再现无与伦比的音乐的人”，这通常是正确的。

TacT Millennium数字放大器-高端D类


Tane杰出的丹麦工程师，联合创始人，领先的开发人员之一Dane Peter Lingdorf（今天被称为“ Lyngdorf Audio ApS”）创造了当时传奇的TacT Millennium。 实际上，这种发展使我们对D类放大器的看法有所不同，即使是要求苛刻的哲学家的最顽固的发烧友也是如此。



事实是，除了效率高外，大多数早期的D类放大器还以令人作呕的声音而闻名。 如此低的复制质量归因于早期积分放大的一些同时代技术的不完善。 其他人则谈到不可能制造此类放大器。

大概在1997年，Lingdorf决定打破关于数字和D类可能性的常见刻板印象。他开始深入研究脉宽和脉冲编码调制的特性，进行实验。 后来，他创立了TacT公司，负责发布与当时市场上所有产品都根本不同的设备。

作为他的研究结果，Lingdorf得出的结论是，为了获得D类的高保真度再现，有必要应用一种使用增加的采样频率（即352.8 kHz）对模拟信号进行编码的方法（开发人员显然会留给当代人一些余地）。



在这种方法中，只需少量的量化即可获得足够的动态范围。 由于该频率，可能的量化误差（换句话说，数字信号的噪声和失真）被转移到了高频区域，远远超出了人类的感知范围。

实际上，该放大器是具有模拟端级联的数模编码器，因此在该设备的规格中会出现“仅数字输入”字样。



对设备特性进行控制测量的结果不免鼓吹提倡高保真再现：

  • 功率（RMS），4欧姆，2 x 300W，1 KHz，0.1％THD + N
  • 功率（RMS），8欧姆，2 x 150W，1 KHz，0.1％THD + N
  • 标称电阻4-8欧姆
  • 频率响应0.3Hz-33KHz -3dB点，8ohm
  • 频率范围20Hz-20KHz -0dB / + 0.2dB，8ohm
  • THD + N：0.01％，1W / 8欧姆
  • THD + N：0.015％，1W / 4欧姆
  • THD + N：0.06％，100W / 8欧姆
  • THD + N：0.02％，300W / 4欧姆
  • 信噪比：113 dB，参考 200/8欧姆
  • 动态范围：140 dB，参考 200瓦/ 8欧姆
  • 通道分离n 100 dB 1 KHz，200 W / 8 Ohms

基于上述特征，绝对声音及其贡献者之一罗伯特·格林（Robert Green）将TacT Millennium列入了世界十大放大器之一。

我认为，对“死亡人数”和“这个令人恶心的D类”抱怨的人会感兴趣的是，Robert E. Greene在音频工程师和技术精湛的发烧友中同样具有权威性，是公认的权威。 UMZCH电子管评论被认为是世界上最好的放大器之一-D类工作的数字放大器。

总结


在这篇文章中，我尝试收集最具标志性的晶体管放大器，这证明了遥远的60、80和零时半导体电路的可行性。 在许多方面，我想通过这篇文章来证明复制的保真度并不取决于放大器中是否存在真空器件。 真正引起人们对灯的兴趣的原因在于怀旧，当时是“阳光明媚，草更绿”，推销TLZ的神话，并在稀有情况下使用静电耳机。
我建议参加传统调查。
