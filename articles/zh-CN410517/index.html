变压器概述ASUS Transformer Mini（T103HAF）



播送-一种非常罕见的小玩意的代表的评论，一种带有可移动扩展坞的紧凑型变压器。

技术指标


处理器 ：四核Intel Atom x5-z8300（Cherry Trail系列）

作业系统 ：
Windows 10家庭版
Windows 10专业版

内存 ：最高4 GB LPDDR3，1600 MHz
显示 ：10.1英寸（16:10），1280x800、60％NTSC色域
显卡 ：Intel HD Graphics
音频 ：华硕SonicMaster技术
数据存储 ：64/128 GB，存储卡最大128 GB
网络摄像头 ：2 MP
无线连接 ：

  • 802.11交流
  • 蓝牙V4.1
  • 3G / LTE（可选）

连接器 ：

1个组合音频插孔
1个USB 3.0
1个微型HDMI
1个Micro USB
1个micro SD

保护 ：TPM（受信任的平台模块）

尺寸 ：平板电脑-261 x 172 x 10毫米（长x宽x高），底座261 x 172 x 5.7毫米（长x宽x高）

重量 ：平板电脑-620克，扩展坞-250克

中央处理器




外表


Mini Transformer T103HAF的设计与T102HA几乎没有区别。 该设备本身是一个小巧，饱满的平板电脑，按照今天的标准，它的形式相当严格，除了圆角外，没有任何平滑的过渡。







T102的徽标上方有一个圆形的指纹传感器，而T103却没有破坏背面线条的原始纯度，只是铰链式支架的缝隙很小。

盖子下方隐藏了一个Micro SD存储卡的插槽，您可以将数据存储量增加128 GB。



支架可倾斜170度，因此可以稍微倾斜将设备稳定地放在桌子或膝盖上。



在底端有用于连接扩展坞的触点和定位孔。



右边是扬声器网格，音量按钮和微型HDMI接口。



左侧是另一个扬声器，微型USB，USB 3.0和组合的音频插孔。



电源按钮离顶端的音量按钮不远。



塑料扩展坞的外部采用柔软触感的合成材料覆盖。







在扩展坞的侧面，有一个用于附带的ASUS Pen的孔眼。 请注意，扩展坞旁边的变压器有一种“折痕”，多余的部分。 借助内置的磁铁，扩展坞可以水平“展开”或以符合人体工程学的倾斜角度升高。 键入很多文本要容易得多。









键盘单元深深地插入扩展坞中，并在三个侧面形成明显的垫子。





触摸板也略微凹陷，以便您可以通过触摸其边界进行导航。 这非常重要，因为变压器非常紧凑，而且触摸板也很小。



展示架




与T102HA一样，Transformer Mini T103HAF配备了10.1英寸显示屏，最大亮度为400 cd / m2，分辨率为1280x800。 尽管在两种情况下均使用IPS矩阵，但T103的色域更广：NTSC的百分比为60％，而不是45％。 显示器是触摸感应的，因此整个前部都覆盖有光滑的保护玻璃。 此外，使用附带的有源手写笔支持输入，该手写笔配有两个按钮，最多可识别1024度的下沉状态（手写笔在这种情况下“播放”，因此我附上了另一则评论中的照片）。



音效


T103和T102之间的另一个区别是对专有技术华硕SonicMaster的支持。 特别是，这里使用了两个音量和声音质量都非常不错的5磁扬声器。 在我看来，任何一款智能手机T103都将在“外部”声音方面带来几率。

性能表现


该变压器基于四核Intel Atom x5-z8300处理器 （Cherry Trail系列）。 RAM 2或4 GB LPDDR3。 内置的64或128 GB闪存用作数据存储。



此外，您可以插入另一个128 GB的micro SD存储卡。 作为视频子系统，使用了集成的Intel HD Graphics芯片，该芯片使用主RAM进行工作。



T103HAF和T102HA之间的另一个区别是3G / LTE模块的可选存在。 不幸的是，在我的副本中没有，因此无法进行测试。

由于变压器实际上是带有可移动键盘的平板电脑，而且对角线也只有10英寸，因此布局非常紧凑，并且您无法安装“热”组件-根本没有放置主动冷却系统的位置，变压器会被动冷却。 这是由于选择了“冷”，而不是最快的硬件，尤其是处理器。 尽管如此，T103HAF在Windows 10下仍能快速运行，在有很多褶皱和蝴蝶结的地方快速翻动，并播放不带刹车的Full HD视频（尽管在1280x800的分辨率下这毫无意义）。 云端版本的Photoshop效果很好，对25万像素的照片进行了编辑。

基准测试
















结论


华硕Transformer Mini（T103HAF）是一款紧凑的“日常”计算机，可放入任何包中，包括儿童背包。 简单又可靠的扩展坞一口气被拆除，变压器变成了带有完整“桌面”操作系统的平板电脑。 设备中没有活动部件-风扇或硬盘驱动器-这提高了可靠性。 可移动的扩展坞和可折叠成大角度的支架，使您可以在任何地方使用T103，将其放在桌子上，膝盖上，背包上，草坪上，或者只是保持重量即可。 对于某些人来说，触控笔不仅可用于其预期目的-用于在各种布局和图表上绘制和标记-而且还可用于控制小工具本身。 尽管我在平板电脑模式下没有困难，但我总是使用按钮和图标，但是如果您需要选择并复制/剪切/粘贴文本片段，使用手写笔会更加方便。 此外，“电子毡尖笔”还为儿童娱乐提供了便利：您可以打开空白页或上传图像进行着色，然后让他们做。
