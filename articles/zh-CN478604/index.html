从劳动者到php程序员。 不寻常的发展事业



今天我们发布GeekBrains学生Leonid Khodyrev（ leonidhodirev ）的故事，他今年 24岁。 他的IT途径与以前发表的故事不同，列奥尼德（Leonid）在军队之后立即开始学习PHP，最终帮助他找到了一份好工作。

我的职业故事可能与其他故事不同。 我阅读了IT代表的职业故事，在大多数情况下，一个人充满信心地前进，为实现其目标而竭尽全力。 我不是这样-我完全不知道我想成为谁，也没有为未来制定计划。 或多或少地严肃，当我从军队回来时，我开始考虑这件事。 但是让我们按顺序进行。



服务员，装载员和助理律师作为职业起点


我很早就开始工作，我的第一个“专业”是散发传单。 他们给了我一堆纸，我把所有东西都分发了，但没有收到任何钱。 尽管如此，这种经历还是有用的-我开始理解我可能会遇到的事情。

然后，他担任装卸工，侍应生，在野外活动中执行了各种任务，并将其与研究相结合。 他在大学学习，同时掌握了创建网站的主题。 我在流行的CMS上创建了简单的网站，我很喜欢。 但是，尽管如此，我还是顺其自然，没有真正思考自己的生活需求。

好吧，然后他们带我参军，多亏了我，我看到了整个国家。 我已经参军了，考虑过将来要做的事情。 回想起我在网站上的经历，我认为在这一领域工作对我来说很有趣。 在军队中，我开始寻找远程学习的可能性。 GeekBrains的Web开发课程引起了我的注意，我停下来了。 据我所记得，然后我只是将关键的“编程”或“教学编程”带入了搜索，看到了课程网站，留下了要求。 一位经理打电话给我，我开始有条不紊地询问一切。

当然，不可能在部队学习，也没有特殊的手段，所以我推迟了学习的时间。

IT成果


我出院后，没有钱。 为了开始训练，我不得不回到服务员的先前工作。 当我收到薪水时，我买了一门课程并开始学习。 不幸的是，很明显，做一个专职服务员要花很多时间，这还不足以学习。 很快找到了解决方案-他开始帮助律师进行文书工作，在“旺季”里，他去当服务生。

不幸的是，学习很困难，我退出了三遍。 但是后来我意识到这种情况无法持续，服务员很好，但是IT更为重要。 因此，我超时工作，全心投入学习。 很快，我意识到我不仅喜欢它，而且非常喜欢它。 不久之后，创建站点的第一个命令开始出现，因此除了带来乐趣之外，本课还开始带来更多的收入。 我莫名其妙地以为自己在做自己喜欢的事，他们也为此付出了我的代价！ 我现在决定自己的未来。

顺便说一句，在培训期间，在实践中，我为自己制定了一个非常认真的项目-内容管理系统。 我不仅写了它，而且还可以连接多个站点。 在此处阅读有关该项目的更多信息。



简而言之，该项目为用户提供了一个方便的平台，可以通过与您可能需要开展业务的各种服务集成来轻松扩展该平台。 目标受众-企业家和网站管理员。 对于他们，我编写了“商店”扩展名，它使您可以管理商品类别，商品本身，它们的属性，流程订单。

这是我的第一个严肃项目，使用的也是同样重要的技术。 当然，当您评估它时，请不要忘记我是在培训期间开发的。

新办公室工作


上面我已经说过，在培训期间，我执行了网站开发的命令。 我真的很喜欢它-实际上，我非常不想在办公室里工作。 但是后来我开始意识到，我需要团队合作的经验，因为大多数开发人员在职业生涯中的某个阶段或某个阶段都能获得正式工作。 我决定这样做。

我记得现在，在星期一早上，我打开了hh.ru，上传了简历，添加了证书并公开了该帐户。 然后他寻找离我家最近的雇主（我住在莫斯科），并开始发送简历。

一小时后，我有兴趣的公司回答了。 那天我被邀请来接受采访。 我注意到没有“压力测试”和其他奇怪的东西，但是我还是有点紧张。 他们开始对我的知识水平，工作经验以及总体上的一切热情地询问我。

我回答了一些不是我想要的问题，但是他们带走了我。 是的，他们让我们担心-起初他们说他们会回电。 实际上，当他们不想录取候选人时，他们通常会以这种方式回答。 但我担心徒劳-几个小时后宝钟响了。 第二天，收集了所有文件后，我开始工作。

我因支持在线预订系统而立即入狱，该系统允许代理商预订酒店，转机等。 我确保一切正常，完善功能并添加各种功能（还有错误，如果没有此功能，它将在哪里）。

一个已经完成的例子：

  • 预订报告模块；
  • 改进的平台界面；
  • 与服务提供商的数据库同步；
  • 忠诚度系统（促销代码，积分）；
  • 集成wordpress。


至于工具，主要有：

  • 布局-html / css / js / jquery;
  • 数据库-pgsql;
  • 用yii2 php框架编写的应用程序；
  • 第三方库，我使用很多不同的库。


如果我们谈论收入，它比以前要高得多。 但是这里的一切都是相对的，因为在培训期间我一个月的收入约为15,000卢布。 有时什么都没有，因为我只收到需要站点的熟人的订单。

两者之间都没有可比的工作条件-很明显，它们比我作为工人或侍应生的工作条件要好得多。 上班时间仅需25分钟，这也是令人愉悦的-毕竟，许多首都居民花费更多的时间。 说到莫斯科，我从泽列诺格勒搬到了首都，和父母住在一起。 在学习期间，他从事订餐点的创建，并搬到了首都。 在这里，我喜欢一切，我不会动，但我计划去看看世界。

接下来是什么？


当我从工作中获得乐趣时，我计划继续我的发展道路-这就是我想要的。 而且，以前对我来说似乎很困难的任务现在根本没有任何困难。 因此，我承担了较大的项目，当一切都变得很高兴。

我继续学习，因为我需要独自完成一些需要工作的主题。 即使完成了主要学习课程，教师也可以帮助您弄清楚事情的发展。

在不久的将来，我想学习一种新的编程语言并学习英语。

入门技巧


我曾经读过有关IT人员职业的文章，许多人说“不必害怕”之类。 当然，这是正确的，但请不要害怕-成功了一半。 最主要的是确切知道您喜欢什么。 尝试学习某种语言的基础知识，例如，使用网络上的课程，然后编写脚本或最简单的应用程序。 如果您喜欢它，那么该开始了。

还有一个提示-不要变成卧式石头，如您所知，在该石头下水不会流动。 怎么了 最近我和一些同学一起学习了。 事实证明，并不是每个人都有工作。 我邀请了几个人进行我的工作面试，因为我的公司需要优秀的专家。 但是最后，没有人来接受采访，尽管在此之前我被问了很多问题。

这是不值得做的-如果您想找工作，请保持一致。 即使您觉得您没有足够的经验，也要尝试进行几次面试-许多公司聘请新来者，以期希望在家中培养一名专家。 如果面试失败，您将获得宝贵的经验，并且将从内部了解就业过程。
