PHP Digest No. 139 (3 - 17 September 2018)


Seleksi segar dengan tautan ke berita dan bahan. Dengan rilis: PHP 7.3 RC 1, PhpStorm 2018.3.3 EAP, Laravel 5.7 dan rilis lainnya, HHVM menghentikan dukungan untuk PHP, mengetikkan properti dalam PHP, sebagian dari alat yang berguna, laporan video, dan banyak lagi.
Selamat membaca!


Berita dan Siaran


  • PHP 7.3 RC 1 - Siklus kandidat rilis untuk cabang 7.3 telah dimulai. Sebanyak setidaknya 6 masalah diperkirakan akan stabil. Sementara itu, Anda dapat membaca Apa yang baru di PHP 7.3 .
  • PHP 7.2.10
  • PHP 7.1.22
  • PHP 7.0.32
  • PHP 5.6.38
  • HHVM tidak akan lagi mendukung PHP - Rilis HHVM v3.30, yang dijadwalkan untuk Desember, akan menjadi versi terbaru dengan dukungan PHP. Proyek direkomendasikan untuk diterjemahkan sepenuhnya ke dalam Hack atau ke dalam PHP 7. Berita yang diharapkan telah diperingatkan tentang ini setahun yang lalu , tetapi kerangka kerja populer dan proyek menolak untuk mendukung HHVM bahkan lebih awal. Terima kasih kepada tim HHVM yang mendorong PHP untuk berkembang, semoga berhasil!
  • Berita PHP-FIG Selama Musim Panas - Pabrik PSR-17 HTTP, Spesifikasi Klien HTTP PSR-18 di Tinjau, Diadopsi, Diterbitkan dan Dibahas Secara Aktif Draf PSR-14 Event Dispatcher .
  • PhpStorm 2018.3 EAP - PhpStorm Early Access Program telah diluncurkan , karena itu dimungkinkan untuk menggunakan versi baru IDE yang sepenuhnya gratis. Dari inovasi yang sudah diterapkan:
    • Dukungan untuk PHP-CS-Fixer memungkinkan Anda untuk tidak hanya menyoroti, tetapi juga memperbaiki kesalahan standar
    • Pencarian cepat untuk variabel selama debugging
    • Ctrl + Dot pelengkapan otomatis
    • Dialog pencarian kelas / file yang didesain ulang
    • Dukungan DQL
    dan peningkatan lainnya.

PHP internal


  • [RFC] Typed Properties 2.0 - Voting mulai menambahkan properti yang diketik. Komunitas mendukung proposal hampir dengan suara bulat, yang berarti bahwa sudah dalam versi berikutnya kami sedang menunggu peningkatan yang signifikan dalam PHP. 
    class User { public int $id; public string $name; // ... }


Alat-alatnya


  • thecodingmachine / safe - Perpustakaan mengimplementasikan pembungkus atas fungsi PHP standar sehingga mereka membuang pengecualian alih-alih menghasilkan false . Lebih detail di pos .
  • rambler-digital-solutions / hydrogen - Level abstraksi yang lebih cepat dan nyaman untuk Doctrine ORM.
  • theorchard / monolog-cascade - Ekstensi untuk Monolog memungkinkan Anda untuk mengonfigurasi beberapa penangan dalam satu file.
  • renatomefi / php-fpm-healthcheck - Script shell untuk memeriksa status php-fpm. Ini mungkin berguna saat bekerja dengan Docker dan Kubernetes.
  • docteurklein / json-chunks - Streaming JSON coding dengan cepat menggunakan generator. Alternatif violet-php / streaming-json-encoder
  • thecodingmachine / phpstan-strict-rules - Aturan tambahan untuk penganalisa statis PHPStan .
  • wbars / PHPFoldings - Sebuah plugin untuk PhpStorm yang memungkinkan Anda untuk "menutup" konstruksi bahasa yang membuat kode lebih bersih: getter / setters, lambdas, $this-> dan banyak lagi.
