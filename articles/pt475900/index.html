1. Administração de Contêiner no Azure


As instâncias de contêiner do Azure são a maneira mais rápida e fácil de executar contêineres no Azure. Este tutorial ajudará você a aprender como criar e gerenciar contêineres e como obter um dimensionamento flexível para o Kubernetes usando ACI.

Módulos do Curso:

  • Criando um aplicativo Web de contêiner usando o Docker;
  • Crie e armazene imagens de contêiner usando o serviço Registro de Contêiner do Azure;
  • Iniciar contêineres do Docker usando o serviço Instâncias de Contêiner do Azure;
  • Implantar e iniciar um aplicativo de contêiner da Web usando o Serviço de Aplicativo do Azure;
  • Visão geral do Serviço Kubernetes do Azure.

Saiba mais e inicie o treinamento clicando no link

2. Engenharia de dados com os Databricks do Azure


Saiba como trabalhar corretamente com os Databricks do Azure e acelerar a configuração da solução. Trabalhe com dados no SQL Data Warehouse do Azure usando os serviços de conector interno. Uma visão geral dos serviços de dados disponíveis no Azure. Crie fluxos de trabalho simplificados e trabalhe com um espaço de trabalho de análise interativa baseado no Apache Spark. A propósito, o curso deve levar cerca de 8 horas.

Módulos:

  • Visão geral dos bancos de dados do Azure;
  • Acesse instâncias do SQL Data Warehouse usando os Azure Databricks;
  • Obtendo dados com os Databricks do Azure;
  • Leitura e gravação de dados usando os Azure Databricks;
  • Transformações de dados básicos nos Azure Databricks;
  • Executar conversão avançada de dados nos Azure Databricks;
  • Criando pipelines de dados usando o Databricks Delta;
  • Trabalhar com dados de streaming nos Azure Databricks;
  • Crie visualizações de dados com os Databricks do Azure e o Power BI.

3. Desenvolvendo soluções eficazes no Azure


Aprenda a projetar e desenvolver soluções robustas, escaláveis ​​e de alto desempenho no Azure, explorando os principais recursos de uma arquitetura eficiente.

Depois de estudar este curso de 4,5 horas, você aprenderá os critérios básicos para uma arquitetura bem-sucedida do Azure, aprenderá o design com foco em segurança, desempenho e escalabilidade, eficiência operacional, alta eficiência e acessibilidade. Inscreva-se agora!

4. Trabalhando com dados NoSQL no Azure Cosmos DB


Os dados NoSQL são uma maneira eficaz de armazenar informações que não atendem aos requisitos de um banco de dados SQL relacional. Saiba como usar dados novos de qualquer lugar no portal do Azure, a extensão do Azure Cosmos DB para Visual Studio Code e o .NET Core SDK para Azure Cosmos DB para fornecer aos usuários alta disponibilidade, independentemente de onde estejam.

Participe do Learn NoSQL no Azure Cosmos DB!

5. Implementando um data warehouse com o Azure SQL Data Warehouse


O Azure SQL Data Warehouse fornece um Big Data Warehouse relacional que pode ser dimensionado para alguns petabytes de dados. Este tutorial mostrará como o Azure SQL Data Warehouse pode atingir essa escala com o uso da arquitetura MPP (Massal Processing Parallel). Crie um data warehouse em minutos e use a linguagem de consulta familiar para criar relatórios. Baixe uma grande quantidade de dados em alguns minutos e verifique se o data warehouse está seguro.

Os seguintes tópicos são abordados neste esquema de treinamento.

  • Informações sobre como criar um SQL Data Warehouse do Azure;
  • Consulta e visualização de dados do Azure SQL Data Warehouse;
  • Importe dados para o SQL Data Warehouse usando o Polybase;
  • Saiba mais sobre os controles de segurança fornecidos pelo Serviço de Armazém do Azure e pelo SQL Data Warehouse do Azure.

Detalhes e início do treinamento

6. Criando aplicativos com o DevOps do Azure [em inglês]


O Azure DevOps permite criar, testar e implantar qualquer aplicativo na nuvem ou no local. Aprenda a configurar pipelines de montagem que montam, testam e testam aplicativos continuamente.

Os seguintes tópicos são abordados neste esquema de treinamento.

  • Colabore na criação de aplicativos com os Pipelines do Azure e o GitHub;
  • Execute testes automáticos no pipeline para verificar a qualidade do código;
  • Varredura de código fonte e componentes de terceiros para possíveis vulnerabilidades;
  • Identifique vários pipelines trabalhando juntos para criar um aplicativo;
  • Crie aplicativos com agentes baseados em nuvem e seus agentes de construção.
