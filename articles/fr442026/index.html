Trojan.Multi.BroSubsc.gen - Kaspersky fonctionne

Messages des utilisateurs de l'antivirus gratuit Kaspersky tourmentés, la fenêtre d'infection Trojan.Multi.BroSubsc.gen apparaît. Il suggère de guérir par un redémarrage, puis avec une certaine probabilité, il se répète à nouveau, en cercle.



En fait, tout est simple ...

Informations du support technique:

Les utilisateurs se plaignent souvent des notifications publicitaires dans le navigateur qu'ils reçoivent à la suite d'actions irréfléchies sur des sites douteux (et pas seulement). En cas de consentement pour accepter les notifications du site, les ressources indésirables apparaissent dans les paramètres du navigateur.

Le verdict Trojan.Multi.BroSubsc.gen est émis lors de la vérification des objets de démarrage si les paramètres du navigateur (Chrome, Yandex, Opera, Vivaldi sont désormais pris en charge, la liste sera élargie à l'avenir) URL détectables qui émettent ces notifications publicitaires. Pendant le traitement, le statut de ces URL passe de «Autoriser» à «Bloquer».
La signature spécifiée a été ajoutée le 13 février 2019.

Notifications de désactivation:

  • Pour Firefox: dom.webnotifications.enabled dans about: config (merci atomlib ).
  • Pour Chrome: Paramètres / Avancé / Paramètres de contenu / Notifications
    ou chrome: // paramètres / contenu / notifications (merci XogN )
  • Pour Opera: Paramètres / Avancé / Confidentialité et sécurité / Paramètres de contenu / Notifications
  • Pour Vivaldi: chrome: // paramètres / contenu / notifications (merci XogN )

Vous pouvez également saisir «Notifications» dans les paramètres du navigateur dans la «recherche» et aller directement à l'endroit souhaité.
