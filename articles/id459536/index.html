Kembangkan plugin untuk produk JetBrains dan jual di JetBrains Marketplace

Kami meluncurkan JetBrains Marketplace , platform baru tempat pengembang dapat menjual dan melisensikan plugin mereka untuk produk JetBrains. Sekarang penulis plugin akan memiliki kesempatan untuk mencurahkan lebih banyak waktu untuk pengembangan itu sendiri - mereka tidak akan harus membuat keputusan lisensi dan terlibat dalam penjualan. Dan pengguna akan memiliki akses ke alat yang lebih nyaman yang akan membantu mereka bekerja lebih efisien.



JetBrains Marketplace - tahap baru dalam pengembangan Repositori Plugin , yang sekarang memiliki lebih dari 4.000 alat. Mereka membantu memperluas fungsionalitas produk JetBrains dan menyesuaikannya dengan kebutuhan individu pengembang. Ada berbagai plugin di sini: beberapa menyediakan dukungan untuk bahasa tambahan dan teknologi baru, lainnya berfungsi untuk integrasi dengan platform cloud atau membantu mengingat pintasan keyboard; Anda dapat memilih tema antarmuka dan bahkan mengatur bilah kemajuan dalam bentuk Nyan Cat. Ada plugin untuk hampir semua produk JetBrains, termasuk IDE berbasis IntelliJ, TeamCity, dan alat .NET. Semua plugin ini akan tetap tersedia di masa mendatang.

Dalam beberapa tahun terakhir, lebih banyak perusahaan dan pengembang telah memilih platform IntelliJ dan produk JetBrains lainnya sebagai dasar untuk mengembangkan alat mereka sendiri untuk programmer. Kami menciptakan JetBrains Marketplace sehingga pengembang dapat fokus pada plugin dan tidak membuang waktu mencari solusi manajemen lisensi dan mengembangkan saluran distribusi. Platform kami menyediakan solusi turnkey untuk perizinan perangkat lunak, infrastruktur penagihan dan penjualan, serta dukungan penjualan dan pemasaran.

Bagaimana platform JetBrains Marketplace akan bermanfaat bagi pengguna:

  • Fungsionalitas tambahan disediakan oleh plug-in berbayar dari pengembang pihak ketiga.
  • Instalasi plug-in yang mudah, pembayaran, berlangganan, dan manajemen melalui Akun JetBrains .
  • Manajemen lisensi yang mudah melalui Akun JetBrains , termasuk pengelolaan semua langganan dan lisensi pada server lisensi pribadi.
  • Validasi lisensi plug-in dan lisensi produk JetBrains (opsional).
  • Berbagai diskon (misalnya, untuk volume), penawaran khusus untuk grup pengguna, proyek sumber terbuka, serta program dukungan JetBrains lainnya (akan segera muncul).
  • Keamanan dan perlindungan data pribadi: saat membeli plug-in, Anda tidak harus memberikan data kartu bank dan informasi lainnya kepada pihak ketiga.

Sekarang plugin pihak ketiga baru hanya tersedia untuk IDE kami berdasarkan pada platform IntelliJ, namun kami berencana untuk memperluas dukungan ke alat lain.

Apa yang sudah ada di sana


Lebih dari 100 vendor berpartisipasi dalam program Marketplace Early Access kami . Kami senang memperkenalkan plugin yang sudah dijual dan dilisensikan melalui JetBrains Marketplace (deskripsi yang diberikan oleh pengembang plugin):

integrasi hybris
oleh Alexander Bartash
Menggunakan plugin Hybris Integration, Anda dapat secara otomatis mengimpor proyek Hybris ke IntelliJ IDEA. Plugin ini memungkinkan Anda untuk mengambil keuntungan penuh dari IDE ketika mengembangkan untuk Hybris dan menambahkan lebih banyak fungsi.

Iedis 2
dari CodesMagic
Iedis adalah plugin GUI yang indah dan nyaman untuk Redis (repositori struktur data penduduk yang digunakan sebagai pangkalan data, cache, dan pesan). Iedis bekerja dengan jutaan kunci tanpa memblokir server Redis.

Java Antidecompiler
dari BIS Guard & Co.
Java Antidecompiler melindungi Java bytecode dan sumber daya dari dekompilasi dan mencegah pengembangan terbalik dan plagiarisme perangkat lunak Anda. Ini menggabungkan teknik kebingungan, algoritma kriptografi Java dan teknologi BIS Guard & Co. asli.

JetForcer | IDE Smartest Force.com
dari Tim JetForcer
JetForcer adalah plugin tangguh yang memungkinkan Anda mengembangkan untuk Salesforce di JetBrains IDE. Ini sangat ideal untuk mengembangkan klien (dukungan untuk Lightning dan Visualforce), dan bagian server (Apex dan SOQL).

JFormDesigner
dari FormDev Software GmbH
JFormDesigner adalah alat desain GUI profesional untuk antarmuka pengguna Java Swing. Dukungan kelas satu untuk MigLayout, JGoodies FormLayout, GroupLayout (Desain Gratis) dan GridBagLayout memudahkan untuk membuat formulir kelas profesional.

Berjenis
dari Manifold Systems LLC
Manifold menghembuskan kehidupan baru ke Jawa: Anda dapat menggunakan metode ekstensi, tipe struktur, pola Java, dan refleksi tipe-aman. Plugin ini menyediakan akses tipe aman ke bahasa lain, seperti GraphQL, JSON Schema, YAML dan Javascript, langsung dari Jawa.

Anggrek
oleh Thomas Fröscher
OrchidE menyediakan dukungan untuk sistem otomatisasi infrastruktur TI Ansible. Plugin ini menyediakan kemampuan platform IntelliJ canggih untuk tim DevOps yang menggunakan Ansible untuk mengelola infrastruktur mereka.

RON (Notasi Objek Berkarat)
dari Vultix
Plugin bahasa RON (Rusty Object Notation) menambahkan dukungan untuk mengimpor jenis Rust, pelengkapan otomatis, validasi, dan banyak lagi.

Bahasa Wolfram
oleh Patrick Scheibe
Plugin Wolfram Language mengubah produk berbasis IntelliJ menjadi alat pengembangan yang kuat untuk bahasa Wolfram dan kode Mathematica.

Dukungan ZenUML
dari ZenUML
Plugin ZenUML membantu menghasilkan diagram urutan langsung dari metode Java atau ZenUML DSL. Plugin ini berguna untuk menganalisis kode yang ada dan untuk proses pemodelan sebelum menulis kode.

Apa yang diharapkan


Dalam waktu dekat, akan ada lebih banyak plugin. Semuanya masih tersedia di Repositori Plugin , dan plugin berbayar juga dapat ditemukan di JetBrains Store .

Jika Anda memiliki pertanyaan tentang JetBrains Marketplace , kirimkan kepada kami di marketplace@jetbrains.com atau tinggalkan komentar di bawah ini.
