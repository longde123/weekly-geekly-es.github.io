SNR-S2965管理型访问交换机



2015年底，SNR系列交换机得到了新的访问级别交换机产品线SNR-S2965的补充。 该系列包括三种设备型号：

  • SNR-S2965-8T ：6个10 / 100BaseTX端口，2个10/100 / 1000BaseT端口和2100 / 1000BaseX SFP端口
  • SNR-S2965-24T ：20个10 / 100BaseTX端口，4个10/100 / 1000BaseT端口和4100 / 1000BaseX SFP端口
  • SNR-S2965-48T ：44个10 / 100BaseTX端口，4个10/100 / 1000BaseT端口和4100 / 1000BaseX SFP端口

在猫的帮助下，简要概述了这些设备。

较新型号具有比常规FastEthernet接入交换机更多的GigabitEhternet端口。 多亏了额外的千兆位端口，运营商才有机会以较高的ARPU提供高附加费率。 S2965系列交换机是传统的FE访问和不断增长的千兆位交换机之间的一种中间解决方案。

交换机随附带有L形连接器的电源线，控制台电缆，接地电缆，19英寸机架中的安装支架，橡皮脚和用于连接RPS电源的连接器。



在具有8和24端口的型号上，连接器位于前面板上。 -这大大简化了设备的安装和维护。 48端口型号的电源连接器位于背面。

SNR-S2965-24T交换机型号具有带外部电源监控器的RPS端口。 由于端口的存在，交换机的电池寿命可以达到8到20小时。 RPS版本仅适用于24端口型号。 因此，与其他型号不同，具有24个端口的版本具有大量指示灯。 除了功率二极管，自我诊断和端口状态外，在带有RPS的版本中，还显示了备用电源状态的指示器。



仅当交换机由12V电源供电时，RPS指示灯才会点亮。 否则，没有任何指示。 在站点上工作时，这非常方便。
使用12 Ah电池，SNR-S2965-24T-RPS可持续使用超过14个小时。

“千兆”端口数量的增加从质量上提高了交换机的性能。 整个SNR S2965线都可以全端口速度运行，而无需超额订购。 确切的数据如下表所示：



8端口和24端口型号具有被动冷却系统。 为了维持所需的温度，安装了风扇。

SNR-S2965系列交换机具有几种管理设备的方式：

便捷的网络界面
经典的类似于cisco的CLI，其语法与其他SNR开关完全兼容
SNMP协议

在某些情况下，可以使用基于Web的界面方便地完成交换机的初始配置。 例如，配置用于管理的IP地址或检查端口的状态。

为了确保网络安全，SNR交换机中实现了许多功能。
首先，它是访问端口的隔离，它禁止端口之间的流量通过。 通过创建一个组来执行此功能，在组中添加需要相互隔离的端口。

RPS的好处


应该稍微注意使用RPS（冗余电源）电源系统的好处。 要从RPS连接器为开关供电，需要12V。 因此，可以在不进行其他转换的情况下为开关板供电。 这样可以提高电源效率，并减少设备内部的散热。 上面已经提到了借助RPS为交换机提供不间断电源的可能性。

RPS测试


我们决定在实验室中测试RPS。 为此，我们用铅酸电池SNR-BAT-12-12 （12V; 12Ah）和不间断电源SNR-UPS-60 / 12连接了开关电源，该电源的输出为12伏，具有充电功能，可提供高达60瓦的负载连接。 电池连接器输出端的电压使用SNR-ERD-3的远程监视和控制设备进行监视，该设备通过SNMP进行查询并通过MRTG实用程序进行可视化。 为了在实验室交换机上进行测试，创建了具有7个活动快速以太网访问端口和2个活动SFP千兆以太网端口的条件。



根据测试结果，由RPS端口供电的电路具有良好的电池寿命（约14小时）。 从图中可以看出：在A点，电池端子的输出电压为14.3V，开关由220V网络供电。 此时，系统切换为电池电源。 在这个位置上，她工作了14个小时，直到电压达到最小可接受值（11V）。 为了防止电池放电，SNR-UPS-60 / 12会自动关闭其电源。 在从B点到C点的时间间隔内，开关不起作用。 在C点，恢复了220V电源并开始了充电过程，整个过程大约需要6个小时。 在D点，我们看到电池的输出电压等于14.3V，相当于电池的充满电。

通常，SNR-S2965交换机系列的主要优点是除了FE端口外还可以使用GE端口，并且可以使用带有RPS端口的交换机线。 第一个允许您使网络在提供服务方面更加灵活，并提供以高于100 Mb / s的速度进行访问的访问权限，这将扩展创建新费率计划的能力。 第二个优点是使您可以构建容错网络并监视其状况。
