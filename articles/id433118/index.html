Saya mendorong Anda untuk membalas posting ini dalam bahasa Inggris, bahkan Anda mengalami kesulitan dengan bahasa ini. Saya tidak sedang menulis puisi sekarang dan saya pikir kita dapat menghindari rasa malu tentang kemampuan bahasa Inggris kita.
