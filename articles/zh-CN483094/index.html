当我访问微软俄罗斯办公室时，我如何实现梦想

大家好！


我和父母住在萨拉托夫。 我有自闭症。 我是一名IT自学人士，拥有10年的自我教育和工作经验。 系统管理，移动设备管理和Web开发是我学习并经常练习的内容。


从我最近实现自己的梦想这一事实来看，2019年对我的生活来说是非常有趣的-我第一次去莫斯科，在IT研讨会上遇到了一些人，我是在Facebook社交网络的广阔区域（主要是与俄罗斯Microsoft团队）第一次见面的。


当我听到并阅读有关Microsoft Autism Hiring Program的很多内容时，我弄清了该程序的原理，开始梦想着在Microsoft工作，开始寻求技巧，并经常与IT社区进行交谈。


为了实现我的梦想，我在1 .5年内建立了自己的第一个网站 。 我建立了一个慈善组织的网站，以帮助智障人士（在我父亲，编辑和自由设计师的支持下）。
我在学校没有英语。 我通过教程学习了英语。


微软员工不仅写了一篇关于我第一次去莫斯科的文章，还谈到了我对微软，Yandex和Dodo Pizza在莫斯科的IT部门的旅行。 在翻译中，您不仅会了解我，而且会了解我在IT领域的生活...


文章本身是英文的。 我一个月前在俄罗斯首都之后将文章翻译成俄文。


如果您阅读我的翻译，可以欣赏我翻译的质量！


微软Sergey Z.的忠实粉丝访问了微软俄罗斯办事处，实现了自己的梦想


Microsoft Techcommunity上的原始文章


我们很少遇到像萨拉托夫一样年轻的程序员Sergey Z.这样的忠诚和热情的Microsoft粉丝。上个月，俄罗斯Microsoft MVP社区团队和俄罗斯Microsoft区域总监帮助实现Sergey的梦想-拜访他在莫斯科的微软办公室在互联网上与他遇到的现实生活中的此类团队成员会面。


Sergey今年19岁，患有孤独症 。 一年前，谢尔盖在庞大的社交网络Facebook上会见了俄罗斯微软MVP团队，俄罗斯微软合作伙伴团队和俄罗斯代表处团队，并仍积极参与这些社交网络团队的活动。 他提出了自己的问题，并分享了他的经验和作为Web开发人员的工作。 Sergey始终以对微软产品，技术和其他东西的热爱着称，有一天，他实现了自己的梦想-他第一次访问了微软在俄罗斯的办公室。 不论是在美国的Microsoft总公司还是在公司的俄罗斯办公室，Sergey都梦想着为Microsoft工作。


2019年10月，俄罗斯IT社区成员齐聚一堂 ，实现了Sergey的宏伟梦想。在Sergey成为Internet上IT社区的热心和积极参与者之后，俄罗斯Microsoft MVP团队Alexander Surkov，Alexey Lyubko，Alexander的成员埃尔马科夫（Ermakov）和其他许多人邀请谢尔盖（Sergey）和他的母亲塔蒂亚娜（Tatyana）参观微软俄罗斯办公室。



团队成员组织了一次筹款活动，以抵消谢尔盖和他母亲的旅行和住宿费用。 这些天来，俄罗斯Microsoft MVP团队的成员一直为Sergey担任俄罗斯IT办事处和莫斯科的个人指南。 Sergey不仅访问了俄罗斯IT办事处，而且第一次将他与他的朋友在Facebook上的虚拟会议（大型俄罗斯IT社区Microsoft的成员）变成了现实生活中最有趣的会议。


微软让粉丝开心


在制定了谢尔盖前往莫斯科的计划之后，微软社区的成员联系了EMEA地区微软MVP计划的主要代表马丁·塔塔鲁（Martin Tataru），讨论了他们如何使谢尔盖在微软方面的体验更好。 碰巧的是，同一周，MVP团队的两名成员访问了微软的祖国美国西雅图市，并就此与企业高管举行了会议。


当时，亚瑟·亚辛斯基（Arthur Yasinsky）在雷德蒙德（Redhurd）与俄罗斯微软MVP团队成员亚历山大·埃玛科夫（Alexander Ermakov），阿列克谢·洛布科夫（Alexei Lobkov）和亚历山大·苏尔科夫（Alexander Surkov）以及来自国际微软学生合作伙伴团队的帕勃罗·维拉门迪（Pablo Viramendi）会面。 帕勃罗（Pablo）和亚瑟（Arthur）都准备了由斯科特·古思里（Scott Guthrie）签名的衬衫和他的礼物包装，这是谢尔盖（Sergey）的榜样。



然后，团队成员花了一些时间与Microsoft讨论在俄罗斯可用的Microsoft服务和工具，包括Microsoft Learn ， Visual Studio Code ， Microsoft Docs等。 Pablo概述了Microsoft学生合作伙伴计划，Martin要求俄罗斯MVP团队和区域总监支持Sergey，以便他可以向他们寻求有关培训计划的更多信息。 结果，他们收到了关于如何改进英语水平不佳的人的MVP计划的宝贵反馈。


谢尔盖与微软一起访问莫斯科


2019年10月24日，谢尔盖（Sergey）访问了微软在莫斯科的俄罗斯办事处 ，这对他来说是一个完全的惊喜-他直到到达车站才知道要去哪里。


他参观了位于莫斯科Krylatsky Hills商业中心的Microsoft办公室，公司员工在那儿会见了Sergei，参观了他的办公室，并从Redmond送了礼物给他。 微软云专家Dmitry Soshnikov（ shwars ）和微软俄罗斯地区总监Alexander Ermakov向微软团队的Martin介绍了Sergey，并与他进行了45分钟的交谈。 他还访问了俄罗斯颇受欢迎的比萨连锁店“ Do Do Pizza”的IT办事处，Microsoft向其提供了基于Microsoft Azure云服务的比萨店的IT基础架构，他还访问了莫斯科市的摩天大楼，从鸟瞰。


Sergey对自己的经历感到高兴，并很高兴在Facebook上与他的朋友们在现实生活中见面。 在莫斯科期间，谢尔盖（Sergey）参观了Yandex办公室，看到了该公司开发的无人驾驶工具，并聆听了有关如何在俄罗斯最大的IT公司开始工作的故事。



亚历山大说：“ 在微软的这一天真是太棒了，它带来了更多积极的情绪 。 我们谈论了很多自闭症以及微软如何对待这种选择的员工。 在会议期间，公司办公室人力资源部门的两名员工讨论了公司的企业文化，女发言人克里斯蒂娜·舒斯塔捷娃（Kristina Schasteeva）带领我们走访了办公室 。


他补充说： “我们还对布拉格公司办公室的虚拟参观印象深刻。 真是太好了。 德米特里·索什尼科夫（Dmitry Soshnikov）也支持巡回演出，讨论了DevOps主题，并专注于未来的产品和服务 。 ”


“ 谢尔盖非常高兴。 他的母亲说她有一天从未见过他那样的微笑。 ”



参与旅行计划实施的亚历山大·苏尔科夫（Alexander Sururkov）说：“ 我认为谢尔盖的旅行非常酷。 他和他的母亲都很高兴。 我很高兴一个好人的故事得到了我们所有人的大力支持。 我认为这是我们IT社区的真正实力 。”


“ 再次感谢所有帮助将谢尔盖的梦想变为现实的人！ ”


仅前进


经过如此精彩的旅行后，Sergey更加努力地实现他的梦想。 他花了一些时间在Microsoft Learn上学习IT课程，并且还致力于通过DuoLingo门户提高英语技能。


Martin和Dmitry也正在与Microsoft Accessbillity团队讨论使用Sergey在使用Microsoft产品和服务时的去向。



他的母亲说，对于像谢尔盖（Sergei）这样的人来说，俄罗斯的教育是最大的问题。 因此，像Microsoft这样的组织可以去做更严肃的事情：让他们发展并学习他们热衷的东西。 Sergey的故事不仅展示了示范性，而且还展示了Microsoft MVP的IT社区成员和Microsoft的区域总监的同情心，而且具有发展和培养像我们长篇故事的英雄这样的聪明才智的能力也很重要。


谁知道呢，也许有一天，谢尔盖（Sergey）将在微软工作，他还将为像他这样的微软大粉丝组织此类游览和旅行。
