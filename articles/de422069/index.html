Nachrichten und Veröffentlichungen


  • PHP 7.3.0 Beta 3 - Letzte Beta in der Schleife. Die nächste Veröffentlichung wird RC 1 sein, die am 13. September erwartet wird .
    • Unter den Änderungen:
      • Verbesserte Heredoc- und Nowdoc- RFC- Syntax
      • Die Funktionen json_encode () und json_decode () lösen im Fehlerfall eine Ausnahme und das entsprechende RFC- Flag aus
      • Möglichkeit, nach dem letzten Argument der RFC- Funktion ein Komma zu verwenden
      • Neue Funktion hrtime (), die eine hochpräzise Zeit zurückgibt und nicht von der Systemzeit PR abhängt
      • Die Funktion is_countable () wurde hinzugefügt, um die RFC- Aufzählung zu überprüfen
      • Verwenden von Links in list () des RFC
      • Einige Funktionen sind veraltet. RFC 1 , RFC 2
      • RFC- Unterstützung für Cookies auf derselben Site-Cookie hinzugefügt
      • Deutliche Verbesserung des PR- Garbage Collectors
      und andere Verbesserungen. Vollständige Liste der Änderungen .
      Sie können mehr über die Änderungen in einer detaillierten Analyse der Neuerungen in PHP 7.3 lesen oder schnell nach Unterschieden suchen .
  • Remotecodeausführung auf packagist.org - Sicherheitsanfälligkeit, durch die Code auf dem PHP-Repository-Server von Packagist ausgeführt werden konnte, indem dem Formular eine speziell gestaltete Zeichenfolge anstelle einer URL zum Hinzufügen eines neuen Pakets übergeben wurde. Bereits behoben .
  • Highload fwdays'18 - 15. September in Kiew wird eine groß angelegte Konferenz über Highload sein. Es gibt keine direkten Berichte über PHP, aber das Programm bietet interessante Präsentationen zu Skalierung, Architektur und Optimierung. Rabattcode : PHPDIGEST_HL .
  • php-vote.com - Was möchten Sie in der nächsten Version von PHP sehen? Auf der Website können Sie eine Idee hinzufügen oder die bereits hinzugefügten diskutieren und abstimmen.
