Allwinner H3 Orange Pi PC papan debug hanya dengan $ 15



Sebelumnya Shenzhen Xunlong Software CO. menggunakan prosesor Allwinner A20 dan Allwinner A31s di papan debug Orange Pi . Namun baru-baru ini, perusahaan memperkenalkan model PC Orange Pi berdasarkan pada prosesor Allwinner H3, yang harganya jauh lebih murah daripada yang sebelumnya, dan $ 15. Rendahnya biaya papan dicapai karena rendahnya harga prosesor Allwinner H3 itu sendiri, dan $ 6.

Anda dapat membaca lebih lanjut tentang Allwinner H3 di sini .
Allwinner H3 serta Broadcom BCM2836 ( Raspberry Pi 2 ) mencakup empat inti ARMv7 Cortex-A7. Tetapi mengingat frekuensi yang lebih tinggi yaitu 1,2 GHz versus 900 MHz (Raspberry Pi 2) dan GPU yang lebih kuat, mungkin kinerja keseluruhan PC Orange Pi akan sedikit lebih tinggi daripada Raspberry Pi 2.

Spesifikasi
 Prosesor Orange Pi PC : SoC - quad-core Allwinner H3 Cortex A7 @ 1,6 GHz (bekerja pada 1,2 GHz) dengan GPU ARM Mali-400MP2 hingga 600 MHz
 RAM: 1GB DDR3 (dibagi dengan GPU)
Memori pengguna:  slot micro SD (kartu hingga 64GB) Video: HDMI dengan dukungan untuk CEC dan HDCP, output AV (tulip) ) Ethernet: 
 
 Port Ethernet 10 / 100M
 Suara: HDMI, port AV, mikrofon on-board
 USB:  port host 3x USB 2.0, port micro OTG 1x
 kamera: CSI Interface
 Debugging:  konektor UART 3-pin untuk Serial Console
 Lain-lain: sensor IR (IR port), tombol power, indikator LED
 Header Ekspansi:  konektor 40-pin yang kompatibel dengan Raspberry Pi termasuk GPIO, UART, I2C, SPI, PWM, CAN, I2S, SPDIF, LRADC, ADC, LINE-IN, FM-IN , dan HP-IN
 Power: jack barel 5V / 2A (konektor OTG mikro USB tidak dapat digunakan untuk memberi daya pada papan).
 Ukuran:  85x55 mm
 Berat:  38 gram

Dibandingkan dengan Orange Pi 2, Orange Pi PC: lebih kecil, tidak ada port USB, tidak ada tombol reset dan tombol u-boot, tidak ada modul WiFi.

Orange Pi PC Allwinner H3


Papan PC Orange Pi baru mendukung firmware : Lubuntu, Android 4.4, Raspbian, server Debian. Android SDK tersedia untuk membentuk firmware Anda.  Shenzhen Xunlong Software CO. Orange Pi PC
