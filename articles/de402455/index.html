Windows 10 lädt einige Updates auch über Limit-Verbindungen herunter



Nach vorläufigen Daten können Benutzer mit der Veröffentlichung der endgültigen Version von Windows 10 Creators Update die obligatorische Installation von Updates mithilfe benutzerdefinierter Grenzwertverbindungen nicht mehr umgehen. Darüber hinaus kann das Betriebssystem jetzt wichtige Updates nicht nur über eine benutzerdefinierte, sondern auch standardmäßig über eine eingeschränkte Verbindung empfangen, bei der es sich beispielsweise um eine mobile Internetverbindung handelt.

Zu diesem Schluss kam einer der Teilnehmer des Windows Insider-Programms, Richard Hay, der die Zusammenstellung der vorläufigen Version der Build 15058-Updates für Windows 10 studierte, die Microsoft Anfang letzter Woche vorgestellt hatte. Während der Bekanntschaft mit dem Build entdeckte er einen Unterschied im Wortlaut der Einstellungen von Windows Update gegenüber der vorherigen Version.


Text in den Update Center-Einstellungen in Version 14393.953


Text in den Update Center-Einstellungen in Baugruppe 15058

Jedes Mitglied des Insider-Programms kann davon überzeugt sein: Wenn Sie zu Windows-Einstellungen> Update und Sicherheit> Windows Update gehen, sehen Sie unter der Überschrift die Update-Parameter, die jetzt über eine Limit-Verbindung heruntergeladen werden können.

Der Hauptunterschied zwischen diesen beiden Versionen von Windows 10 ist der zweite Satz, der wie folgt lautet: "In diesem Fall werden automatisch nur die Updates heruntergeladen, die zur Aufrechterhaltung des normalen Windows-Betriebs erforderlich sind."

Konfigurierbare Grenzwertverbindungen in Windows 10 sind eine weit verbreitete Methode zur Steuerung des automatischen Downloads und der Bypass-Installation von Updates, die in der Home-Version des Betriebssystems erforderlich sind. Jetzt können Benutzer, die eine Mobilfunkverbindung für die Datenübertragung oder einen anderen kostenpflichtigen Standardkanal verwenden, beim Herunterladen von Updates nichts mehr tun und den gesamten Rest des Datenverkehrs verbrauchen.

Als Hay die geänderte Beschreibung des Update Centers sah, beschloss er herauszufinden, was dieselben Updates bedeuten, um den normalen Betrieb des Betriebssystems sicherzustellen. Er wandte sich an das Microsoft Support Center, um eine bessere Vorstellung von den Arten von Updates zu erhalten, die in diese Kategorie fallen.

Die Antwort des Sprechers verdeutlichte die Situation nicht: „Wir planen nicht, große Aktualisierungen über Grenzwertverbindungen zu senden, werden sie jedoch bei Bedarf für kritische Korrekturen in der Zukunft verwenden.“ Es folgten keine genauen Angaben zum verbrauchten Verkehrsaufkommen, daher fordert Haye eine genauere Betrachtung.

Das Windows 10 Creators Update wird am 11. April veröffentlicht. Es ist möglich, dass sich der Text, der für die vorläufige Version geschrieben wurde, noch ändert. Andererseits ist die Wahrscheinlichkeit, dass sich ein großes endgültiges Update kurz vor der Veröffentlichung dramatisch ändert, äußerst gering. Schließlich beschränkt sich die Arbeit nicht nur auf die Bearbeitung des Textes: Die Entwickler haben die grundlegenden Netzwerkfunktionen eingerichtet, um die Installation einiger Updates auf Kanälen mit kostenpflichtigem Datenverkehr zu ermöglichen.

Wenn wir in Creators Update über Änderungen in den Nuancen beim Einrichten von Grenzwertverbindungen sprechen, können Sie auf der Seite mit den Netzwerkverbindungseinstellungen eine Kabelverbindung als Grenzwert markieren. Darüber hinaus können Benutzer die Installation bereits heruntergeladener Updates jederzeit verzögern oder im "Schlaf" -Modus automatisch drei Tage lang übertragen.

Darüber hinaus wird die neue Assembly umfangreichere Aktualisierungen enthalten. Eines der interessantesten betrifft das Multitasking: Mithilfe der Funktion "Compact Overlay" können Benutzer in den Modus "Bild in Bild" wechseln. Damit können Sie Anwendungen in einem kleinen Fenster ausführen, das über anderen geöffneten Registerkarten angezeigt wird. Es ist nicht nur für Fans nützlich, das Video anzusehen, während sie eine monotone Aufgabe ausführen, sondern auch, um beispielsweise Instant Messenger zu starten.

Eine weitere nützliche Funktion ist die automatische Sperrung des Computers beim Trennen eines über Bluetooth verbundenen Smartphones. Wenn Sie die Reichweite von Bluetooth verlassen, wechselt der Computer nach 30 Sekunden in den Sperrmodus.

Darüber hinaus werden Augmented- und Virtual-Reality-Geräte unterstützt sowie andere 3D-Funktionen integriert, z. B. das Teilen von Inhalten mit anderen Entwicklern dreidimensionaler Modelle. Microsoft-Spieler versprachen, die Unterstützung für Spiele in 4K-Auflösung und den Start von Videosendungen zu optimieren . Der Spielemodus optimiert den Computer während des Spiels, indem unnötige Hintergrundprozesse und einige andere Tricks deaktiviert werden. Anscheinend sollten Sie jedoch nicht mehr von dieser Funktion erwarten.

Windows 10 Creators Update konzentriert sich auch mehr auf die Sicherheit. Das OS-Sicherheitscenter wird um erweiterte Funktionen zur Erkennung von Bedrohungen und Analysetools erweitert.

Die Bekanntschaft der Windows 10-Benutzer mit allen Updates erfolgt am 11. April.
