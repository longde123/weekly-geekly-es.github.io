Optane DC Persistent Memory - Optane im DIMM-Format


Letzte Woche stellte das Unternehmen auf der Intel Data Center Tech Summit-Konferenz offiziell Optane-Speicher auf der Basis von 3D-XPoint- Modulen im DIMM-Format vor, der als Optane DC Persistent Memory bezeichnet wurde (bitte nicht mit Intel Optane Memory , der Consumer-Linie von Cache-Laufwerken, verwechseln).

Die Speicherleisten haben eine Kapazität von 128, 256 oder 512 GB, die Pinbelegung entspricht dem DIMM-Standard. Natürlich muss die Hardware diese Art von Speicher unterstützen - diese Unterstützung wird in der nächsten Generation von Intel Xeon-Serverplattformen verfügbar sein. Was die Softwareunterstützung für das Produkt betrifft, gibt es seit geraumer Zeit ein Open Source-Projekt Intel Persistent Memory Development Kit (PMDK, bis Ende letzten Jahres - NVML).

Leider enthält die Präsentation keine technischen Details wie Stromverbrauch, Frequenz usw. - Wir werden auf das Update in ARK warten. Es ist auch unklar, ob DRAM und Optane auf demselben Speichercontrollerkanal kombiniert werden können. Die soeben erschienene Erinnerung kann jedoch bald „berührt“ und gemessen werden, bisher jedoch nur aus der Ferne. Im Sommer wird Optane DC Persistent Memory online getestet. Sie können auch Mitglied werden, wenn Sie bei einem Intel-Partnerunternehmen arbeiten (es ist übrigens nie zu spät, einer zu werden). Zum Testen wird eine Serverfarm mit 2 Prozessorknoten, 256 GB DRAM und 1 TB persistentem Speicher bereitgestellt.

Darüber hinaus wird Ende des Jahres mit der Speicherversorgung einzelner Projekte begonnen. Nun, große Verkäufe sind für Anfang 2019 geplant.
