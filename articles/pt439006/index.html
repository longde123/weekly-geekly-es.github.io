Como a IA ajuda a aprender a linguagem de sinais



Para qualquer pessoa, é difícil aprender a língua de sinais. O fato é que não apenas são utilizados gestos, mas também expressões faciais, além de articulação, além de “linguagem corporal”. O valor da informação transmitida depende de todos esses fatores. Existem programas para aprender a linguagem de sinais, mas muitos deles não são muito bem-sucedidos, além do preço nem sempre ser humano. Num futuro próximo, tudo pode mudar - agora métodos estão sendo desenvolvidos para estudar a linguagem de sinais usando a IA.

Trata-se de automatizar o processo de aprendizado, que é completamente intuitivo. Aqui você pode . familiarize-se com os recursos da ferramenta de análise da linguagem de sinais (grupo sueco-alemão) com o fornecimento de todos os detalhes necessários. No entanto, os desenvolvedores esperam criar plataformas de IA que possam ajudar a acelerar o processo de aprendizado às vezes.

A IA foi e é usada para reconhecer, traduzir e interpretar a linguagem de sinais. Mas as possibilidades de tais projetos são incompletas, elas não cobrem todos os aspectos. Os desenvolvedores do novo sistema planejam usar a IA para permitir que o aluno saiba o que está fazendo de errado e diga como isso pode ser corrigido.


A prática da linguagem de sinais é difícil porque não é leitura nem escrita. Para aprender a usar um PC, os desenvolvedores criaram um jogo de computador. Ela gradualmente ensina gestos, mostrando qual gesto corresponde a uma letra ou conceito específico. Em seguida, o jogador deve repetir esse gesto, gravando suas ações na câmera. O programa mostrará o sucesso da repetição e o que precisa ser feito para melhorar o resultado. Além disso, nesse jogo, há uma competição social entre os participantes, o que envolve usuários no processo.

A inteligência artificial está envolvida em todas as etapas do treinamento. O primeiro passo é interagir com a rede neural convolucional - extrai informações visuais sobre as ações do usuário a partir do vídeo. Esses dados são enviados para o próximo módulo de avaliação, que compara o resultado com os padrões disponíveis no banco de dados e emite sua classificação.

Tudo isso é realizado de forma compreensível para o usuário, é claro que o aluno não vê todo o processo. Funciona apenas com as manifestações externas das ações da rede neural.

Em um futuro próximo, os desenvolvedores planejam criar sistemas que ajudarão falantes de outros grupos de idiomas a aprender a linguagem de sinais. A coisa mais difícil aqui é treinar a rede neural e gravar vídeos com uma demonstração de cada conceito e som, isso é muito tempo e leva tempo.

Além disso, o sistema será ensinado a "entender" as expressões faciais de uma pessoa com referência ao aprendizado da linguagem de sinais. No momento, o sistema funciona de maneira ideal em sala de aula. No futuro, os desenvolvedores planejam tornar o aplicativo com IA mais universal, para que possa ser usado na rua. Os criadores da plataforma esperam que seu produto possa ser usado para ensinar linguagem de sinais para pessoas de qualquer idade.
