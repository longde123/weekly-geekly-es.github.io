Bagaimana saya mewujudkan impian saya ketika saya mengunjungi kantor Microsoft di Rusia

Halo semuanya!


Saya tinggal di Saratov dengan orang tua saya. Saya menderita autisme. Saya seorang otodidak IT dengan 10 tahun pengalaman dalam pendidikan mandiri dan pekerjaan. Administrasi sistem, administrasi perangkat seluler, dan pengembangan web adalah apa yang saya pelajari dan sering praktikkan.


Tahun 2019 ternyata menarik dalam hidup saya dalam hal fakta bahwa saya baru-baru ini mewujudkan impian saya - saya mengunjungi Moskow untuk pertama kalinya dan bertemu dengan orang-orang di bengkel IT, yang pertama kali saya temui di hamparan jejaring sosial Facebook (terutama dengan tim Microsoft Rusia).


Ketika saya mendengar dan membaca banyak tentang Program Perekrutan Autisme Microsoft , saya menemukan prinsip-prinsip program dan mulai bermimpi tentang bekerja di Microsoft dan mulai meminta tip dan sering berbicara dengan komunitas TI.


Agar saya dapat memenuhi impian saya, dalam 1,5 tahun saya membuat situs web pertama saya . Saya membuat situs untuk satu organisasi amal untuk membantu orang-orang dengan cacat mental (dengan dukungan ayah saya, editor dan desainer lepas).
Saya tidak memiliki bahasa Inggris di sekolah. Saya belajar bahasa Inggris dengan tutorial.


Karyawan Microsoft menulis sebuah artikel tidak hanya tentang perjalanan pertama saya ke Moskow, tetapi juga berbicara tentang kunjungan saya ke kantor TI Microsoft, Yandex, dan Dodo Pizza di Moskow. Dalam terjemahannya, Anda akan belajar tidak hanya tentang saya, tetapi tentang kehidupan saya di TI ...


Artikel itu sendiri dalam bahasa Inggris. Dan saya menerjemahkan artikel itu sebulan yang lalu, setelah saya berada di ibukota Rusia, ke dalam bahasa Rusia.


Jika Anda membaca terjemahan saya, Anda dapat menghargai kualitas terjemahan saya!


Penggemar berat Microsoft Sergey Z. memenuhi mimpinya dengan mengunjungi kantor Microsoft di Rusia


Artikel asli di Microsoft Techcommunity


Kami jarang bertemu penggemar Microsoft yang setia dan bersemangat seperti Sergey Z., seorang programmer muda dari Saratov, ketika bulan lalu tim Komunitas Microsoft MVP Rusia dan direktur regional Microsoft di Rusia membantu memenuhi impian Sergey - kunjungan ke kantor Microsoft di Moskow, tempat ia pertama kali bertemu dengan anggota tim seperti itu di kehidupan nyata, yang dia temui di Internet.


Sergey berusia 19 tahun dan memiliki autisme . Setahun yang lalu, Sergey bertemu dengan tim Rusia dari Microsoft MVP , tim mitra Microsoft di Rusia dan tim kantor perwakilan Rusia dari korporasi dalam luasnya jejaring sosial Facebook dan masih menjadi peserta aktif dalam komunitas dari tim yang sama ini di jejaring sosial. Dia mengajukan pertanyaan dan berbagi pengalaman dan pekerjaannya sebagai pengembang web. Yang paling menonjol dari Sergey adalah kecintaannya terhadap produk, teknologi, dan hal-hal lain dari Microsoft. Dalam satu hari, ia mewujudkan mimpinya - ia pertama kali mengunjungi kantor Microsoft di Rusia. Apakah itu kantor pusat Microsoft di Amerika Serikat atau kantor perusahaan Rusia, Sergei yang bermimpi bekerja untuk Microsoft.


Pada Oktober 2019, anggota komunitas TI Rusia berkumpul untuk memenuhi impian besar Sergey, dan setelah Sergey menjadi peserta yang sangat antusias dan aktif dalam komunitas TI di Internet, anggota tim Microsoft MVP Rusia Alexander Surkov, Alexey Lyubko, Alexander Ermakov dan banyak lainnya mengundang Sergey dan ibunya, Tatyana, untuk melakukan tur kantor Microsoft di Rusia.



Anggota tim mengorganisir pengumpulan dana untuk mengimbangi biaya perjalanan dan akomodasi Sergei dan ibunya. Semua hari ini, anggota tim Microsoft MVP Rusia bertindak untuk Sergey sebagai panduan pribadinya ke kantor TI Rusia dan Moskow. Sergey tidak hanya mengunjungi kantor-kantor TI Rusia, tetapi untuk pertama kalinya mengubah pertemuan virtualnya dengan teman-temannya di Facebook-anggota komunitas TI Rusia besar Microsoft menjadi pertemuan paling menarik dalam kehidupan nyata.


Microsoft membuat penggemarnya senang


Setelah menyusun rencana perjalanan Sergey ke Moskow, anggota komunitas Microsoft menghubungi Martin Tataru, perwakilan utama program Microsoft MVP di kawasan EMEA, untuk membahas bagaimana mereka dapat membuat pengalaman Sergey lebih baik dengan Microsoft. Secara kebetulan, pada minggu yang sama, dua anggota tim MVP mengunjungi tanah air Microsoft, kota Seattle di AS, dan mengatur pertemuan dengan para eksekutif perusahaan tentang hal ini.


Pada waktu itu di Redmond, Arthur Yasinski bertemu dengan anggota tim Microsoft MVP Rusia - Alexander Ermakov, Alexei Lobkov dan Alexander Surkov, bersama dengan Pablo Viramendi dari tim Microsoft Student Partners internasional. Baik Pablo dan Arthur menyiapkan kemeja yang ditandatangani oleh Scott Guthrie dan bungkus kado darinya - panutan yang dibimbing oleh Sergey.



Kemudian anggota tim meluangkan waktu untuk berdiskusi dengan Microsoft mengenai layanan dan alat Microsoft yang tersedia di Rusia, termasuk Microsoft Learn , Visual Studio Code , Microsoft Docs , dll. Pablo memberikan gambaran umum tentang program Mitra Pelajar Microsoft, dan Martin meminta tim MVP Rusia dan direktur regional untuk mendukung Sergey sehingga ia akan meminta informasi lebih lanjut tentang rencana pelatihan tersebut kepada mereka. Sebagai hasilnya, mereka menerima umpan balik yang berharga tentang bagaimana meningkatkan program MVP bagi mereka yang tidak bisa berbahasa Inggris dengan baik.


Kunjungan Sergey ke Moskow dengan Microsoft


Pada 24 Oktober 2019, Sergey mengunjungi kantor Microsoft Rusia di Moskow , yang merupakan kejutan besar baginya - dia tidak tahu ke mana dia akan pergi sampai dia tiba di stasiun.


Dia mengunjungi kantor Microsoft di Pusat Bisnis Krylatsky Hills di Moskow, tempat karyawan perusahaan bertemu dengan Sergei, memberinya tur kantor dan mengirim hadiah dari Redmond kepadanya. Spesialis Microsoft Cloud Dmitry Soshnikov ( shwars ) dan Direktur Regional Microsoft untuk Rusia Alexander Ermakov memperkenalkan Sergey kepada Martin di Microsoft Teams dan berbicara dengannya selama 45 menit. Dia juga mengunjungi kantor IT dari rantai pizza populer di Rusia, "Do Do Pizza", kepada siapa Microsoft menyediakan infrastruktur IT dari pizzeria berdasarkan layanan cloud Microsoft Azure, dan dia mengunjungi gedung pencakar langit Kota Moskow untuk menikmati pemandangan indah Moskow dari pandangan mata burung.


Sergey senang dengan pengalamannya dan senang bertemu dalam kehidupan nyata dengan teman-temannya di Facebook. Selama tinggal di Moskow, Sergey mengunjungi kantor Yandex, melihat kendaraan tak berawak yang dikembangkan oleh perusahaan dan mendengarkan cerita tentang bagaimana mulai bekerja di perusahaan IT Rusia terbesar.

Maju saja


Setelah perjalanan yang luar biasa, Sergey bekerja lebih keras untuk mewujudkan mimpinya. Dia meluangkan waktu untuk mempelajari kursus TI di Microsoft Learn, dan juga berfokus pada peningkatan keterampilan bahasa Inggris melalui portal DuoLingo.


Martin dan Dmitry juga berdiskusi dengan tim Microsoft Accessbillity tentang ke mana Sergey dapat pergi ketika menggunakan produk dan layanan Microsoft.



Ibunya mengatakan bahwa pendidikan di Rusia adalah masalah terbesar bagi orang-orang seperti Sergei. Jadi ada organisasi seperti Microsoft yang dapat pergi dan melakukan hal-hal yang lebih serius: Biarkan mereka mengembangkan dan mempelajari apa yang mereka sukai. Kisah Sergey tidak hanya menunjukkan sifat demonstratif, tetapi juga kasih sayang dari anggota komunitas TI Microsoft MVP dan direktur regional Microsoft, tetapi juga penting untuk memiliki kemampuan untuk mengembangkan dan melatih pikiran cerdas seperti pahlawan dalam cerita panjang kami.
