Nouveau terminal Windows: réponses à vos questions

Dans les commentaires d'un article récent , vous avez posé beaucoup de questions sur la nouvelle version de notre terminal Windows. Aujourd'hui, nous allons essayer de répondre à certains d'entre eux.

Sous la coupe se trouvent certaines des questions les plus fréquemment posées que nous avons entendues (et entendons toujours), ainsi que des réponses officielles: y compris le remplacement de PowerShell et comment commencer à utiliser le nouveau produit aujourd'hui.



Quand et où puis-je me procurer un nouveau terminal Windows?


  1. Vous pouvez cloner le code source du terminal à partir de GitHub sur github.com/microsoft/terminal et le créer sur votre ordinateur.
    Remarque : assurez-vous de lire et de suivre les instructions sur la page README dans le référentiel avant d'essayer de construire le projet - pour construire le projet, vous devez suivre certaines conditions préalables et étapes d'initialisation!
  2. La version préliminaire du terminal sera disponible en téléchargement sur le Microsoft Store à l'été 2019.

Nous nous efforçons de publier Windows Terminal v1.0 d'ici la fin de 2019, mais en même temps, nous travaillerons avec la communauté pour publier cette version afin que le terminal soit de haute qualité.

Le terminal Windows remplace-t-il l'invite de commandes et / ou PowerShell?


Pour répondre à cette question, clarifions quelques termes et concepts:

  • L'invite de commandes et PowerShell (par exemple WSL / bash / etc. dans * NIX) sont des shells, pas des terminaux, et n'ont pas leur propre interface utilisateur
  • Lorsque vous exécutez l'outil shell / application / ligne de commande, Windows les lance automatiquement et les connecte aux instances de la console Windows (si nécessaire)
  • La console Windows est une application d'interface utilisateur standard de type terminal livrée avec Windows et utilisée par les utilisateurs depuis 30 ans pour exécuter des outils de ligne de commande sous Windows NT, 2000, XP, Vista, 7, 8 et 10.



Ainsi, la question devrait très probablement être reformulée comme "Terminal Windows - est-ce un remplacement de la console Windows?"

La réponse est «non»:

  • La console Windows continuera à être livrée à Windows pendant des décennies pour fournir une compatibilité descendante avec plusieurs millions de scripts, applications et outils de ligne de commande existants / hérités
  • Windows Terminal fonctionnera avec la console Windows, mais deviendra probablement l'outil préféré des utilisateurs qui souhaitent exécuter des outils de ligne de commande sur Windows
  • Windows Terminal peut se connecter à l'invite de commandes et à PowerShell, ainsi qu'à toute autre application shell / outil / ligne de commande. Vous pouvez ouvrir des onglets indépendants connectés à l'invite de commandes, PowerShell, bash (via WSL ou ssh) et tout autre shell / outil de votre choix

Quand puis-je obtenir une nouvelle police?


À venir bientôt! Nous n'avons pas de calendrier défini, mais nous travaillons activement à l'achèvement de la police. Une fois prêt à être publié, il sera ouvert et disponible dans son référentiel.
