User.js yang subyektif untuk Mozilla Firefox dan tidak hanya

Selama hampir 15 tahun sekarang saya telah menggunakan Mozilla Firefox setiap hari, dan, tentu saja, saya memiliki user.js kecil subjektif saya sendiri, yang harus diperbarui dengan setiap rilis versi baru. Saya cepat-cepat membagikannya (dan tidak hanya).

Jika ada yang tertarik, selamat datang di kucing.

user.js


File user.js menimpa nilai-nilai, mengabaikan prefs.js , setelah memulai browser dan harus ditempatkan di folder pengguna, alamatnya dapat ditemukan di halaman about: support.

Setiap tim memiliki komentar dalam bahasa Rusia dan Inggris. Saya tidak mencoba mematikan telemetri secara manual (orang lain akan melakukan ini) dan mengkonfigurasi semuanya, seperti yang diterapkan di user.js dari www.ghacks.net yang terkenal jahat . Dianjurkan untuk menerapkan file ke browser dengan pengaturan default, karena didasarkan pada ini.

user.js di github

Skrip PowerShell untuk mengatur bilah alat di tempat pertama dan, sebagai tambahan, membuka tautan ke dua add-on: uBlock Folder Asal dan Default Bookmark .

Skrip dirancang untuk situasi saat Anda perlu mengonfigurasi peramban sesering mungkin, hanya memasang ekstensi di atas. Ekstensi Folder Bookmark Default mengharuskan bookmark disimpan di folder Menu Bookmark alih-alih folder Bookmark Lainnya. Perubahan ini telah terjadi sejak Firefox 57.

Urutan ikon pada bilah alat dijawab oleh browser.uiCustomization.state . Setelah skrip dieksekusi, semua ikon akan dipindahkan ke sisi kanan:
