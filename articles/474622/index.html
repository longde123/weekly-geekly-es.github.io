34 bibliotecas Python de código abierto (2019)


Analizamos y comparamos 10,000 bibliotecas de código abierto para Python y seleccionamos las 34 más útiles.


Hemos agrupado estas bibliotecas en 8 categorías.


Este artículo fue respaldado por EDISON Software, una empresa de optimización de motores de búsqueda y SEO , que también desarrolla aplicaciones móviles Android e iOS .

Kit de herramientas de Python


1. Pipenv : flujo de trabajo de desarrollo de Python para humanos.


2. Pyxel : desarrollo de juegos retro en Python.


3. PyTest v3.5 : un marco que ayuda a escribir pruebas cortas y escalas para pruebas funcionales complejas.



4. Poesía : simplifica la gestión de dependencias y el empaquetado.


5. Loguru : simplifica los registros.


6. Faust : biblioteca para aplicaciones de transmisión / transmisión.


7. Pampy : la coincidencia de patrones con la que estabas soñando.


8. Comprobación de piratería : comprobación de tipo de rendimiento.


9. Delorean , una biblioteca para trabajar con hora y fechas.


10. Cirq : una biblioteca de esquema Quantum de escala intermedia ruidosa (NISQ).


11. Python-nubia : un marco de línea de comandos.


Web


12. Requests-HTML : análisis de HTML para humanos.


13. Bokeh : visualización interactiva de datos en navegadores modernos.


14. Vibora : rápida, fría, asíncrona.


15. Pywebview v2.0 : muestra el contenido HTML en una ventana separada.


16. WhatWaf : detección y omisión de firewalls y sistemas de seguridad de aplicaciones web.


17. Molten : un marco minimalista y rápido para crear una API HTTP.


Terminal


18. Termtosvg : grabamos sesiones de trabajo con el terminal como una animación SVG.


19. Asciinema v2.0 : grabamos sesiones de terminal.


20. Termgraph : herramientas para la línea de comando, dibuja gráficos.


Editor de código


21. Negro : formateador de código intransigente.


22. Algojammer : un editor de código experimental para escribir algoritmos.


23. Bowler : una herramienta de refactorización a nivel de árbol de sintaxis.


Depuración


24. Py-spy : le permite visualizar lo que su programa está perdiendo el tiempo sin reiniciar el programa.


25. Birdseye : un depurador gráfico usando AST.


26. Icecream : un lindo depurador.


Compilador


27. Transcrypt : traductor Python 3.7 a JavaScript.


28. Pyodide : pila de datos científicos en el navegador.


Datos relacionados


29. Voluptuoso : validación de datos.


30. Botflow : marco de programación basado en datos para el trabajo de canalización de datos (Web Crawler, Machine Learning, Quantitative Trading.etc).


31. Fast-Pandas : puntos de referencia comparativos en las operaciones de Pandas.


Chart


32. Un recorrido por el país de las maravillas de las matemáticas con Python : una colección de scripts para dibujar hermosas formas y algoritmos de animación.


33. Chartify : ayuda a los científicos de datos a crear gráficos.


34. Hypertools v0.5 : una herramienta para la representación geométrica de datos multidimensionales.
