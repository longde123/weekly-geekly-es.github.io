3CX v16 Update 3 Beta publié - appels vidéo sur Android et iOS, connexion TLS des trunks SIP

La semaine dernière, nous avons publié la mise à jour 3 de 3CX v16! Cette mise à jour inclut la prise en charge des liaisons SIP sécurisées fonctionnant sous le protocole TLS (par exemple, l'opérateur mobile ukrainien Lifecell), les appels vidéo sur les appareils mobiles et d'autres nouveaux éléments intéressants. Testez la nouvelle version bêta - aidez-nous à publier la version finale plus rapidement!



Nous listons les principales améliorations qui sont apparues dans le nouveau package de services:

  • Connexion des trunks SIP sur TLS. Une nouvelle option «Transport Protocol» est apparue dans les paramètres de jonction SIP. Pour connecter une jonction TLS, installez-la dans TLS (Transport Layer Security) et téléchargez le certificat de sécurité (.pem) fourni par l'opérateur SIP sur le serveur. Désormais, tous les messages SIP entre le PBX et votre fournisseur seront cryptés
  • Niveau de compatibilité du protocole TLS. Dans la section Paramètres> Sécurité, une nouvelle option, «Activer le transport et les chiffrements SSL / SecureSIP conformes à la norme PCI», a été ajoutée, ce qui définit le mode serveur sur TLS v1.2 (et désactive la compatibilité avec TLS v1.0). Veuillez noter que TLS v1.2 n'est pas compatible avec certains appareils SIP hérités.
  • Mettez à jour le certificat SSL sans redémarrer. Auparavant, lors de la mise à jour du certificat SSL (Let's Encrypt) pour le serveur FQDN à partir de 3CX, le PBX a redémarré le service Web. Dans le même temps, tous les appels en cours ont été interrompus. Maintenant, le PBX initialise simplement un nouveau certificat sans redémarrer nginx. Les appels actifs ne sont pas interrompus - la disponibilité du serveur de téléphonie est encore plus élevée!
  • Notifications PUSH dans différents états utilisateur. Vous pouvez désormais activer ou désactiver individuellement les notifications PUSH pour chaque état utilisateur. Dans le client Web, cela se fait dans la section "Options"> "Renvoi d'appel", l'option "Recevoir les notifications PUSH". La même option est disponible dans la version bêta de l'application 3CX pour Android. De plus, vous pouvez configurer les notifications PUSH dans l'interface de gestion du serveur 3CX.
  • Noms des conversations de groupe. Vous pouvez maintenant spécifier votre propre nom pour la conversation de groupe, et il sera affiché à tous les participants du client Web 3CX et des applications mobiles 3CX.



Nouvelle application Android Bêta 3CX


En collaboration avec 3CX v16 Update 3 Beta, nous avons introduit la nouvelle application 3CX pour Android Beta. Il a reçu un certain nombre de fonctionnalités intéressantes, principalement en raison de vos commentaires. Il s'agit d'un mode de fonctionnement silencieux, d'un travail amélioré avec les contacts et les appels vidéo (à venir bientôt)! Pour tester la nouvelle application, rejoignez le programme de test bêta de Google Play.



La nouvelle version 3CX pour Android a reçu les fonctionnalités suivantes:

  • Mode de fonctionnement silencieux. Un nouveau mode de fonctionnement désactive la «sonnerie» pour un appel entrant. L'activation et la désactivation du mode se fait par un interrupteur dans l'interface de l'application, tandis que l'état correspondant est affiché en haut de l'écran.
  • Travail correct avec les appels entrants. Désormais, les appels entrants auxquels un autre téléphone IP a répondu ont cessé de sonner correctement dans l'application Android.
  • Chat amélioré. Un menu d'action rapide est apparu dans l'application, qui permet à l'opérateur du centre d'appels de transférer le chat du site Web vers lui-même ou de le transférer à un autre employé. Accéléré également le chargement de l'historique des messages.
  • Amélioration du travail avec le carnet d'adresses. Le téléchargement de votre liste de contacts est désormais plus rapide; Une recherche d'un contact par nom d'entreprise est également apparue.

Malheureusement, dans cette version bêta, la communication vidéo n'est pas encore entièrement déboguée, mais apparaîtra lorsque la version finale sera publiée. La vidéo montre comment cela fonctionnera.

Une nouvelle application Android 3CX nécessite un minimum de 3CX V16 Update 2 avec un certificat SSL de confiance (mieux que Update 3 Beta). Prise en charge d'Android 7, 8 et 9. Android 10 Q a été publié récemment, mais nous avons déjà apporté de nombreuses améliorations pour améliorer la compatibilité avec ce système d'exploitation. Par exemple, dans Android Q, la façon de répondre aux appels est modifiée - l'interface ressemble à une réponse à une notification PUSH, au lieu de la fenêtre habituelle «Répondre / Rejeter». Nous soutenons déjà ce type de réponse.

Nouvelles applications pour smartphone et serveur PBX hérité


Certaines fonctionnalités de la mise à jour v16 3 imposent certaines exigences aux applications 3CX pour les appareils mobiles. Les applications actuelles sur vos appareils seront automatiquement (via les outils de mise à jour Android et iOS) remplacées par de nouvelles - avec une interface différente, une communication vidéo, etc. Ces applications ne fonctionneront qu'avec 3CX v16 Update 3 et supérieur. En revanche, les versions actuelles des applications ne seront plus prises en charge par la mise à jour 3 après environ un mois.

Le problème est que certains utilisateurs utilisent toujours des versions PBX obsolètes avec lesquelles les nouvelles applications mobiles cesseront de fonctionner. Et ici, vous devez décider:

  • Mettez à jour votre PBX vers v16 Update 3 et continuez à utiliser le système comme auparavant.
  • Ne mettez pas à jour le serveur, mais désactivez la mise à jour des applications mobiles.

Dans le second cas, veillez à faire attention au mode de mise à jour de l'application, car sur la plupart des appareils mobiles, la mise à jour automatique est activée par défaut.

Possibilité de mettre à jour 3CX


Les mises à jour du serveur PBX sont disponibles pour les propriétaires d'une licence perpétuelle 3CX avec un abonnement actif aux mises à jour ou une licence annuelle (dont une gratuite pour 8 appels simultanés). Si votre abonnement a expiré, renouvelez-le via votre partenaire 3CX. Vous pouvez également échanger votre licence perpétuelle avec un abonnement annuel expiré - et recevoir toutes les mises à jour gratuitement pendant un an. Cependant, si vous n'êtes pas prêt à passer à la mise à jour 3, assurez-vous de désactiver la mise à jour automatique de l'application 3CX sur votre appareil.

Android


Allez sur Google Play, sélectionnez "Mes applications et jeux", sur l'onglet "Installé", cliquez sur "3CX" et dans le menu supérieur droit supprimez "Mise à jour automatique".

Pour les utilisateurs qui installent la dernière version de 3CX, mais qui ont des utilisateurs avec Android en dessous de la version 7, nous offrirons un fichier APK spécial pour la nouvelle application 3CX. Cela doit être fait car Google Play ne met pas à jour les applications si le système d'exploitation ou le smartphone n'est pas pris en charge. Cependant, une telle application ne sera pas automatiquement mise à jour et accompagnée du support technique 3CX.

iOS


Malheureusement, afin de ne pas mettre à jour une seule application, dans la version actuelle d'iOS, vous devrez désactiver les mises à jour pour toutes les applications.

Accédez à Paramètres> iTunes et App Store et désactivez l'interrupteur en haut de l'écran Mises à jour. Comme vous pouvez le voir, l'absence d'abonnement aux mises à jour pour une seule application crée des restrictions sur le travail de toutes les autres applications iOS. C'est un autre argument pour le renouvellement ou l'utilisation de la licence annuelle 3CX.

Installer 3CX v16 Update 3 Beta et de nouvelles applications


Si la dernière version de 3CX est installée, accédez à la section «Mises à jour» de l'interface du serveur, sélectionnez «v16 Update 3 Beta» et cliquez sur «Télécharger la sélection».
Vous pouvez également installer une distribution propre pour Windows ou Linux:

  • Windows: 3CX v16 Update 3 Beta
  • Linux: 3CX v16 sur Debian 9 ISO - Dans l'écran « 3CX Update Channel », sélectionnez «Beta».
  • Installation manuelle sur Debian et Raspbian: suivez les instructions d'installation pour Debian et Raspberry Pi .

Journal des modifications complet.

Pour essayer les nouvelles applications 3CX pour les appareils mobiles, inscrivez-vous dans le programme de test bêta:

  • 3CX pour Android Beta
  • 3CX pour iOS Beta

N'installez pas ces versions bêta dans un environnement de travail! Utilisez-les uniquement pour tester de nouvelles fonctionnalités - elles ne sont pas couvertes par le support technique.
