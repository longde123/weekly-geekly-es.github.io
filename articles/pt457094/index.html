Suplemento do Excel que facilita a definição de filtros ao trabalhar com cubos (VBA)

Como você sabe, o Excel pronto para uso não permite definir filtros de acordo com a lista de valores para tabelas dinâmicas, e isso é uma coisa necessária! Como filtrar produtos por cem códigos e depois por outros cem? Existem, é claro, maneiras, mas tudo isso não é isso ...

Ou, por exemplo, defina o filtro para o cubo por medição com datas, mas não marque a cada dia / mês / ano, mas defina o intervalo de ... a ....
Ainda com bastante frequência, é necessário definir períodos para padrões fixos, como o dia / mês / ano atual . Para esses fins, você pode adicionar conjuntos (no modelo de cubo multidimensional) ou adicionar atributos especiais à dimensão (isso não é muito conveniente, mas não há conjuntos no modelo de cubo tabular).

Vimos uma extensão do VBA que adiciona uma barra de ferramentas Cube com botões mágicos à faixa de opções.


No original, há mais botões no painel, mas eles foram cortados, por razões óbvias .



O filtro por datas pode ser aplicado a várias tabelas dinâmicas ao mesmo tempo, em uma e em diferentes folhas. Isso é conveniente quando você precisa definir os mesmos períodos em várias tabelas ao mesmo tempo.



Conexão de suplemento


É conveniente colocar o suplemento em uma bola e, por esse motivo, a imagem foi rejeitada de ser copiada para o diretório local, para que as alterações pudessem ser feitas e todos tivessem a versão atual. Mas depende de você.


O suplemento é aprimorado para um calendário (tabela de datas) de uma determinada estrutura; portanto, é improvável que ele decole sem nenhuma modificação, mas eles prometem ser pequenos. Mas a instalação de filtros na lista deve funcionar sem problemas. Sua única limitação é que a instalação de filtros para a hierarquia não é suportada, mas isso não é necessário.
