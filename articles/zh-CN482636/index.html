在德国接受治安法官的经验（详细分析）

我是明斯克的一名程序员，今年我成功进入了德国的硕士课程。 在这篇文章中，我想分享我的录取经历，包括选择正确的课程，通过所有测试，申请，与德国大学交流，获得学生签证，旅馆，保险以及到达德国的行政程序。

录取过程比我预期的要麻烦得多。 我遇到了许多陷阱，并经常因缺乏许多方面的信息而遭受痛苦。 在Internet上（包括在Habré上），已经发布了许多有关该主题的文章，但是在我看来，没有一篇文章提供了足够的详细信息来理解整个过程。 在本文中，我试图逐步详细地描述我的经历，并分享提示，警告和对发生的事情的个人印象。 我希望通过阅读这篇文章可以避免我的一些错误，您对开业活动会更有信心，并且可以节省一些时间和金钱。

本文对计划或开始在德国攻读计算机科学相关专业的硕士课程的人员很有用。 在某种程度上，本文可能对其他专业的申请人有用。 不准备去任何地方的读者可能会发现，由于各种官僚主义的细节丰富且缺乏照片，这篇文章很无聊。

关于我
我的名字叫Yalchik Ilya，今年26岁，我在白俄罗斯共和国的Pastavy小镇出生和成​​长，毕业于BSUIR并获得人工智能学位，并在白俄罗斯IT公司（例如， iTechArt集团和TouchSoft。 我一直梦想着在任何领先发达国家的大学学习。 今年秋天，我到达波恩，开始在波恩大学攻读硕士学位课程“生命科学信息学”。

1.1。 我的动力


高等教育经常受到批评。 对于许多人来说，它从来没有派上用场。 从未有人收到它，但仍然成功。 当您是一名软件开发人员时，很难说服自己需要继续接受教育，而且劳动力市场到处都是职位空缺，其中有有趣的项目，舒适的工作条件和令人眼花sal乱的薪水，而无需任何文凭。 但是，我决定获得硕士学位。 我看到许多好处：

  1. 我的高等教育的第一阶段对我有很大帮助。 我对很多事情睁开了眼睛，开始思考得更好，轻松地掌握了自己作为软件开发人员的职业。 我想知道西方的教育体系能提供什么。 许多人说，如果它确实比白俄罗斯好，那么我绝对需要它。
  2. 硕士学位将使其有可能获得博士学位。 将来，这可能会为研究小组的工作和大学教学提供机会。 对我来说，这是理想的职业延续，因为财务问题将不再令人担忧。
  3. 一些世界领先的技术公司（例如Google）通常会要求将硕士学位授予空缺职位。 这些家伙必须知道他们在做什么。
  4. 这是一个绝佳的机会，可以让您休假，从商业程序设计，从例行事务中休息，度过一段有益的时光并了解下一步的工作。
  5. 这是一个掌握任何相关领域并扩大我空缺数量的机会。

当然，也有缺点：

  1. 如果两年没有稳定的薪水，却有稳定的支出，这将使您的钱花光。 幸运的是，我设法筹集了足够的资金来安静地学习，而不依赖任何人。
  2. 在过去的两年中，存在落后于当前趋势并失去商业开发灵活性的风险。
  3. 有考试不及格的风险，而过去两年没有学位，没有钱，没有工作经验，就可能重新开始职业。

对我而言，利弊多于利弊。 接下来，我决定了选择培训计划的标准：

  1. 与计算机科学，软件工程和/或人工智能相关的领域。
  2. 用英语培训。
  3. 每年学习的费用不超过5000欧元。
  4. [希望]有机会掌握相邻领域（例如，生物信息学）。
  5. [可选]经济实惠的旅馆空间。

现在选择国家：

  1. 由于培训成本高昂，大多数讲英语的国家退学了。 根据www.mastersportal.com的数据 ，在美国平均一年学习（非最佳大学）的费用为20,000美元，在英国为14,620英镑，在澳大利亚为33,400澳元。 对我来说，这些是无法承受的数额。
  2. 许多欧洲非英语国家为欧盟公民提供了不错的价格，但其他公民的英语课程价格则飙升至美国水平。 在瑞典-每年15,000欧元。 在荷兰-每年20,000欧元。 在丹麦-每年15,000欧元，在芬兰-每年16,000欧元。
  3. 据我了解，在挪威，奥斯陆大学提供免费的英语培训选项，但是我没有时间在那里提交文件。 在我收到雅思考试成绩之前，秋季学期的招生于12月结束。 同样在挪威，生活费用具有威慑力。
  4. 在德国，有大量的优秀大学和大量的英语课程。 教育几乎到处都是免费的（巴登-符腾堡州的大学除外，那里的学费为每年3000欧元，与邻国相比也有些不足）。 甚至生活成本也比许多其他欧洲国家低得多（尤其是如果您不住在慕尼黑的话）。 此外，在德国生活将是学习德语的绝佳时机，这将为在欧盟工作打下良好的职业前景。

因此，我选择了德国。

Universität或Hochschule


在德国，大学分为两种类型：

  • Universität是一所古典大学。 它拥有更多的理论学科，更多的研究，也有可能获得博士学位。
  • Hochschule（点燃的“高中”）是一所注重实践的大学。

大学学位通常评分较低（亚琛工业大学除外，后者是大学学位并且评分很高）。 建议那些计划将来获得博士学位的人入读Universität，而那些计划毕业后工作的人则建议选择Hochschule。 就我个人而言，我更专注于“Universität”，但由于其较高的评价和Hochschule Fulda作为备用计划，因此在我的名单中包括两个“ Hochschule”-亚琛工业大学。

1.3。 入学要求


入学要求在不同的大学和同一所大学的不同课程中可能会有所不同，因此要求列表必须在大学网站上的课程说明中指定。 但是，我们可以区分与任何大学相关的一组基本要求：

  1. 高等教育文凭（“学位证书”）
  2. 分数表（“记录副本”）
  3. 语言证书（雅思或托福）
  4. 激励函（“目的陈述”）
  5. 简历（CV）

一些大学提出了其他要求：

  1. 给定主题的科学论文
  2. GRE考试
  3. 推荐信
  4. 专业说明-官方文件，指出学习的每个主题和主题的小时数（适用于文凭中指定的专业）。
  5. Cirruculum分析-比较您的文凭科目和大学教授的科目，将您的科目划分为特定类别，等等。
  6. 简要说明毕业项目的本质。
  7. 学校证书。

此外，大学通常可以下载其他任何文件来确认您的成绩和资格（出版物，课程证书，专业证书等）。

1.4。 雅思


我的入门活动从雅思考试的准备和交付开始， 如果没有足够的英语水平，您将不会纯粹通过正式标准通过考试，并且不再需要其他所有内容。

雅思考试在一个特别认可的中心的礼堂举行。 在明斯克，每个月都会举行考试。 大约需要在交付前5周进行注册。 此外，录制仅进行了3天-有在我方便的日期跳过录制的风险。 记录和付款可以在雅思考试网站上在线进行。

对于大多数大学来说，只要在9分中得分6.5分就足够了。这大致相当于中高级水平。 对于某些大学（并非总是排名最新的大学，例如亚琛工业大学），5.5分就足够了。 7.0以上的水平不需要德国的一所大学。 另外，我经常碰到有人提到，语言证书的较高分数并不能保证您有更高的录取机会。 在大多数大学中，是否通过律师资格仅是重要的。

即使您的英语水平很高，也不要忽略考试本身的准备工作，因为 它需要一些技巧来通过测试本身，并了解其结构和要求。 为了准备，我报名参加了在明斯克对应的为期两个月的全日制课程，以及免费的eDX在线课程 。

在全日制课程中，他们为我提供了很多帮助，帮助我理解了写作部分（如何分析图表和撰写论文），因为 考官希望看到非常严格的结构，因为他们偏离了得分。 另外，在课程中，我理解为什么如果他们问“是或否”，为什么不应该回答“真”或“假”，为什么用大写字母填写答案行，在答案中包括文章，何时不包括答案，等等，纯属考试时机，更有利可图。 与全日制课程相比，edX课程似乎有些无聊且效果不佳，但总的来说，该课程还概述了考试的所有必要信息。 从理论上讲，如果您在edX上完成了该在线课程，然后在过去的几年中解决了3-4组测试（您可以在torrent上找到它），那么应该有足够的技能。 “检查雅思考试的词汇”和“雅思语言练习”这本书对我也有帮助。 我们还推荐了“正在使用的雅思考试词汇”，“将搭配使用自然英语”，“用于学术目的的雅思考试-练习测试”，“雅思考试高级测试”这本书，但我没有足够的时间来阅读。

通过考试2周后，您可以在雅思网站上查看结果。 这仅仅是不适合运送到您朋友以外的任何地方的信息。 官方结果是一份证书，您需要在参加考试的考试中心获得证书。 这是带有检查中心签名和盖章的表格A4。 该文档的副本可以发送给大学（无需公证，因为大学可以在IELTS网站上验证真实性）。

我的雅思成绩
就我个人而言，我通过了雅思考试，并获得了听力，8.5，阅读：8.5，写作：7.0，口语：7.0的评分。 我的总分为8.0。

1.5。 GRE


与美国大学不同，在德国大学中，对GRE成绩的要求并不普遍。 如果需要某处，则它更有可能作为您能力的一项附加指标（例如，在Kaiserslautern TU的波恩大学）。 我所审查的课程对特定GRE结果的严格要求仅在康斯坦茨大学存在。

12月中旬，当我收到雅思考试成绩时，我开始准备其余的文件，还报名参加GRE考试。 自从我花了1天的时间准备GRE以来，我以为他失败了。 我的结果如下：言语推理149分，定量分析154分，分析写作3.0分。 不过，我也将这样的结果应用于那些要求GRE结果的大学。 如实践所示，这种情况并没有恶化。

1.6。 准备文件


高等教育文凭，成绩单，学校证书必须加盖公历，翻译成英文或德文并经过公证。 所有这一切都可以在任何翻译公司完成。 如果您打算进入通过单协助系统接受文件的大学（例如，慕尼黑工业大学，柏林工业大学，德累斯顿工业大学），请立即向翻译机构索取每份文件的一份经公证的副本。 一些大学（例如，慕尼黑工业大学，汉堡大学，FAU Erlangen-Nurnberg）要求您通过书面邮件向他们发送文件副本。 在这种情况下，对于每所这样的大学，请翻译局为每份文件再提供一份经公证的副本。

与翻译公司联系后的一周，我收到了翻译，公证和公证的文件翻译。

当您准备翻译时，请务必仔细检查质量！ 就我而言，翻译人员犯了一些错误和错别字，例如“操作系统”（而不是“操作”），“意识形态”（而不是“状态”）。 不幸的是，我很晚才注意到。 幸运的是，没有一所大学对此有过错。 索取翻译文件的电子副本很有意义-您可以从那里复制名称，这将节省您填写入门表格的时间。

另外，如果德国大学要求对专业进行描述，请检查您的专业是否以英语提供。 如果您不确定和/或找不到它，请毫不犹豫地向院长办公室/行政部门发送带问题的信。 就我而言，该专业的描述是“白俄罗斯共和国的教育标准”，但没有官方译本。 在这种情况下，有两种选择：自己翻译或再次去翻译公司。 幸运的是，他不需要公证。 就我个人而言，我曾去过翻译公司，之前曾从提到的“教育标准”中删掉了任何毫无意义的文书工作。

雅思证书可以以未经身份验证的普通副本的形式提供。 大多数大学都可以使用雅思考试验证系统，可以在其中验证您证书的真实性。 请勿通过邮寄方式将证书原件（以及其他文件）发送给他们-如果您没有收到，则不太可能将它们退还给您。

GRE测试结果通常是通过组织者的网站ets.org以电子方式发送的，但是有些大学（例如TU Kaiserslautern）可以接受从您的ETS网站上的个人帐户下载的常规证书形式的结果。

2.6。 申请伊尔默瑙（TUI）


该过程是100％在线的。 有必要在他们的网站上创建一个帐户，填写表格，上传文件扫描图。

特点：有必要为我的申请考试支付25欧元，还需要通过Skype考试。

1月16日-申请了计算机与系统工程专业（RCSE）。

1月18日-我收到了25欧元的付款要求，并显示了详细信息。

1月21日-付款（SWIFT）。

1月30日-他们发送了付款确认书

2月17日-他们发送了我的文凭验证结果。 这是一个PDF文档，说明以下内容：

  • 我的大学属于H +类别（即在德国完全认可）。 还有H±（这意味着只承认某些专业/院系）和H-（这意味着该大学在德国未被承认）。
  • 我在德国评分系统中的平均分数（结果为1.5，比单助计算的平均分数低0.1点-显然大学会提供不同的科目样本进行计算）。
  • 无论是什么意思，Oberes Drittel写作的相对分数（前三分之一）。

因此，我的应用程序已移至状态C1-决策已准备就绪。

3月19日-一位大学员工发来一封信，她在信中说我的文凭获得65分。 下一步是进行Skype口试，我可以得到20分。 入学时，您必须获得70分（因此，在考试中我必须获得20分的5分）。 从理论上讲，某人可以获得70分的文凭，那么您无需参加考试。

要组织考试，有必要写信给另一位大学员工，并确认我已准备好参加考试。 如果不这样做，则两周后取消入学申请。

3月22日，第一位员工回答了我，并告知了考试涉及的主题：

  • 理论：基本算法和数据结构，复杂性。
  • 软件工程与设计：开发过程，使用UML建模。
  • 操作系统：进程和线程模型，同步，调度。
  • 数据库系统：数据库设计，查询数据库。
  • 网络：OSI，协议。

4月9日，我被告知考试的日期和时间。

4月11日，举行了一次Skype英语考试。 教授问了以下问题：

  1. 您在计算机科学中最喜欢的主题是什么？
  2. 什么是Big-O表示法？
  3. 操作系统中的进程和线程之间有什么区别？
  4. 流程如何同步？
  5. IP的作用是什么？

我简短地回答了每个问题（2-3个句子），然后教授告诉我我已经被接受了，他正在十月份等我。 考试持续了6分钟。

4月25日，他们向我发送了正式的培训报价（以电子形式）。 可以将其以pdf格式下载到TUI网站上的个人帐户中。

过了一会儿，我给他们发了一封信，拒绝了这个提议，因为 我选择了另一所大学。

5月27日，我收到一封信，通知我文件的验证被延迟，委员会需要再花几周时间做出决定。

这封信中报道的另一个有趣的细节是需要接受采访。 我只能假设在成功完成测试和面试之后，报价可能会在8月中旬出现。 向德国驻明斯克大使馆的录音花了半个月的时间（即7月18日，最接近向大使馆录音的日期是9月3日）。 因此，如果您在8月中旬至10月初在使馆签约，那么您最多将在11月获得签证。 通常在德国大学，课程从10月7日开始。 我想相信Hochschule Fulda考虑了学生迟到的可能性。 作为一种选择，可能值得在要约到来之前于8月底立即在大使馆进行注册。

由于我已经接受了另一所大学的录取通知书，因此我拒绝参加考试。

6月27日，一位大学员工发送了有关健康保险的信息，有关购买笔记本电脑的建议，以及指向Facebook上一个小组的链接，以与本课程中的其他学生讨论问题。 稍后，她还发送了有关移居德国后所需的行政程序，德语课程，时间表等信息。 信息支持令人印象深刻！

2.9。 申请慕尼黑工业大学（TUM）


TUM的录取过程最困难，其中包括在您的帐户中填写调查表，以单人协助方式接收VPD以及通过纸邮件发送文档。 此外，在进入信息学专业后，有必要执行Cirruculum分析（文凭专业与该专业研究的学科的相关性），并就以下四个主题之一撰写一篇科学论文，每篇1000字：

  • 人工智能在未来技术中的作用。
  • 社交网络对人类社会的影响。
  • 大数据平台的特征及其对数据探索的重要性。
  • 电脑可以思考吗？

上面“ Uni-assist”段落中介绍了有关获取VPD的信息。 因此，在2月5日，我准备了VPD。 它赋予了进入大学所有专业的权利。

然后一个月，我写了一篇主题为“人工智能在未来技术中的作用”的科学论文。

3月26日-我在TUMOnline的个人帐户中填写了信息学计划的申请。 然后必须打印，签名此申请表并将其附加到文件包中，以通过纸邮件发送。

3月27日-通过DHL通过纸质邮件发送了文件包。 我的文件包中包括公证的证书，文凭，成绩单和工作簿，以及经过公证的英语翻译。 此外，该文件包还包括从TUMOnline导出的普通（非认证）语言证书（雅思，歌德A1），动机信，论文，简历和签名的申请书。

3月28日，DHL收到了一条SMS消息，我的包裹已寄到该消息。

4月1日，大学确认收到了我的文件。

4月2日，我收到了一份通知，通知我我的文件符合正式标准，现在将由selection选委员会进行评估。

4月25日，拒绝进入信息学专业。 原因是“您的资格不符合所涉课程的要求”。 此外，还提到了一些巴伐利亚法律，但是我并不清楚我的资格有何差异。 例如，出于类似的原因，亚琛工业大学（RWTH Aachen University）拒绝申请“数据科学”计划，但至少他们指出了我的文凭课程中缺少的主题列表，但TUM没有提供此类信息。 就个人而言，我希望他们的网站将对我的评价从0到100。 如果我的分数很低，那么我会理解自己的科学论文和动机信薄弱。 因此，事实证明，选拔委员会的信和论文都没有阅读，但没有计分就把我过滤掉了。 真烦人。

关于进入TUM，我还有另外一个故事。 入学要求包括“健康保险”。 拥有自己保险的外国人有可能从德国保险公司收到该保险在德国认可的确认。 我没有健康保险。 对于像我这样的人，我一定会获得德国保险。 该要求本身并不令我感到惊讶，但出乎意料的是，在填写入学申请表的阶段已经需要保险。 我向保险公司（TK，AOK，Barmer）以及中介公司Coracle发送了有关此问题的信。 TK回答说，我需要德国邮政地址才能投保。 该公司的一位专家甚至给我打了个电话，问我好几次我是否真的没有至少在附近的德语地址，或者至少没有一个德国的朋友会通过邮件接受我的文件。 总的来说，对我来说这不是一个选择。 AOK写道，我可以在他们的网站上找到所有信息。 谢谢，AOK 巴默写道，他们将在几天后与我联系。 我从他们那里听到的消息越多。 Coracle回答说，是的，他们为学生安排远程保险，但是要获得此保险，您需要...前往德国大学的录取通知书。 对于我将如何收到这封信感到困惑，如果我什至没有保险也无法提交文件，他们回答说其他学生也成功地加入了保险。 最后，他们从TUM本身回答了我，并告诉我，实际上，在申请入学阶段保险是可选的，您可以跳过此项目。 当我已经有入学通知书时，在入学时需要保险。
