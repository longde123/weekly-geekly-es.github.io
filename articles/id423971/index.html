Bagaimana menjadi programmer web


Penulis webinar ini adalah Manajer Pengembangan BelkaCar Alexander Ponomarev. Dia memberi tahu cara menjadi programmer web, dengan mempertimbangkan nuansa berikut:

  • kelebihan dan kekurangan spesialisasi;
  • teknologi dan kemungkinan arah pengembangan;
  • langkah pertama dalam pemrograman;
  • pemilihan majikan pertama;
  • Kemungkinan jalur pengembangan untuk area pengembang dan pertumbuhan.

Alat Otomasi Pengembangan Seluler


Kepala pengembangan ponsel Agima, Dmitry Shashlov, berbicara tentang aspek-aspek pengembangan ponsel seperti:

Cara mengembangkan aplikasi PHP


Pimpinan insinyur perangkat lunak Badoo Mikhail Ovchinnikov menjelaskan cara mengembangkan aplikasi dalam PHP. Jika Anda tidak terbiasa dengan bahasa ini, webinar akan membantu Anda memahami alasan popularitas PHP, termasuk sebagai bahasa untuk pengembangan backend. Ini juga akan mencakup dasar-dasar pembuatan aplikasi web sederhana dan situasi di pasar PHP, termasuk gaji, persyaratan, dan poin penting lainnya.
