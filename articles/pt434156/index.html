Fractal de Gerasimov. Encontrei um padrão. Mesa preta

Encontrei esse padrão quando olhei para a postagem do usuário xcont . Tendo tropeçado nesta publicação , chamei a atenção para o fato de que os padrões são repetidos não apenas com um aumento na escala de acordo com os números de Fibonacci.



Fiquei me perguntando se existe um padrão nesses padrões. Mas, tendo apenas 2 parâmetros x e y , decidi que precisava designar outra coisa, comum entre todos os padrões obtidos. Então notei que, se fizermos os primeiros 4 quadrados no campo, em qualquer caso, teremos 3 opções para o início do padrão, se a linha for:

para cima (↑)



para baixo (↓)



ou não vai * (-)



Para designação, decidi usar esses símbolos ↑, ↓, chamando-os condicionalmente de giros (como o giro de uma partícula). E aqui comecei a criar uma tabela de dependências desses spins para x , y .

Primeiro, vamos ver se existe um padrão se apenas y , x for alterado, faça 4


E agora sobre as propriedades

Vemos uma sequência de rodadas - ↑ - ↓ - ↑ - ↓ - ↑ - ↓

O padrão se repete com uma determinada sequência
y = 3,7,11 ... ( ↑ )

y = 5,9,13 ... ( ↓ )

y = 2,4,6,8,10,12 ... ( - )


Temos o mesmo se x = 3, vemos uma sequência de giros ↑ - ↓ ↑ - ↓ ↑ - ↓ ↑ - ↓ ↑

y = 2,5,8,11 ... ( ↑ )

y = 4,7,10,13 ... ( ↓ )

y = 3,6,9,12 ... ( - )

Eu pensei que poderia sistematizar essas seqüências e compilar uma tabela dessas rotações, e foi isso que obtive.

Eu apresento a você, "Table of Black".



O mais interessante é que esta tabela tem suas dependências e propriedades.

Primeiro, derivamos algumas fórmulas: →
se x é par e y é par → -, também se x ⋮ y ou y ⋮ x → -
x = y → -
x = y + 1 → ↑
x = y-1 → ↓
E agora sobre as propriedades da tabela em si, se tomarmos x = y como ponto de referência, em qualquer direção, temos seqüências refletidas em espelho (indicadas por linhas vermelhas e verdes).








Na verdade, encontre uso e uso. Eu tenho uma mesa de até 12 anos, já que inicialmente fiz tudo isso no papel.



Mas, usando o algoritmo JavaScript, você mesmo pode verificar valores maiores.

PS : Não sei como isso pode ser usado, mas talvez esse algoritmo possa ser aplicado em um computador quântico.
