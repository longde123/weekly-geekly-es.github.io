"Baikal-T1" mulai dijual seharga 3990 rubel



Prosesor BE-T1000 (alias Baikal-T1) memasuki toko komponen dan perangkat elektronik. Distribusi dilakukan oleh rantai ritel Chip dan Dip. Sekarang di toko online diindikasikan bahwa pengiriman ke gudang di Moskow diharapkan pada 8 Juni 2018 dalam jumlah 42 buah. Prosesor sudah dapat ditambahkan ke keranjang dengan harga 3990 rubel.

Penjualan ritel Baikal dimulai pada April 2018, tetapi hanya sebagai bagian dari papan evaluasi BFK3.1 - sebenarnya komputer papan tunggal untuk mengembangkan sistem dan perangkat lunak aplikasi, merancang dan membuat prototip solusi perangkat keras, dll. Sekarang prosesor dijual terpisah.



Prosesor BE-T1000 (Baikal-T1) berisi dua core P5600 dengan frekuensi clock hingga 1,2 GHz pada arsitektur MIPS32r5. Ada pengontrol memori DDR3-1600 ECC; menyatakan dukungan untuk antarmuka Ethernet 1/10 Gb, PCIe Gen.3, SATA 3.0 dan USB 2.0. Konsumsi daya kurang dari 5 watt. Proses teknologi adalah 28 nm. Menurut deskripsi, "itu adalah prosesor hemat energi modern dengan berbagai antarmuka kecepatan tinggi, yang dirancang untuk berbagai perangkat target di segmen konsumen dan B2B."


Blok Prosesor BE-T1000

Deskripsi singkat tentang prosesor , diagram blok , tolok ukur , dan selebaran informasi diterbitkan di situs web toko. Untuk informasi lebih lanjut, lihat gambar Debian 9.0, SDK versi 4.13, dan instruksi instalasi Linux yang singkat.

Dalam benchmark Coremark, prosesor Baikal-T1 menunjukkan kinerja 12.364 poin (10,3 poin per megahertz, 5,15 poin per megahertz per inti) di Linux 4.4.44. Saat membuat tolok ukur, kami menggunakan kompiler GCC 5.3 dan Mentor GCC 4.9.1 dengan plug-in, dan menggunakan pengaturan untuk mendapatkan kinerja maksimum di Coremark.

Dalam tes Dhrystone, kinerjanya adalah 4398 MIX MIPS (1 stream), yaitu 3,66 DMIPS per megahertz.

Dalam uji Whetstone, kinerjanya adalah 1213 MWIPS (2 utas), yaitu 0,51 MWIPS per megahertz per inti. STREAM Salin: 3119,2 MB / s, Skala: 3109,9 MB / s, Tambahkan: 2466,9 MB / s, Triad: 2467,7 MB / s.

Hasil pada berbagai tolok ukur dari paket SPEC CPU2006 INT (beban kerja ref) ditunjukkan dalam diagram.



Hasil pada tolok ukur iperf (ver. 2.0.5, iperf.fr ) dirancang untuk menguji bandwidth saluran Internet (antarmuka GbE, protokol TCP) dari 940 hingga 942 Mbit / s, dalam pengujian Baikal diperiksa sebagai "server" dan sebagai "pelanggan".
