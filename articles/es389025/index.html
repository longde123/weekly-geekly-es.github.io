Expansor de puerto USB compacto para cargar



En mi tiempo libre, me gusta desarrollar algún dispositivo útil para el hogar, por ejemplo: un sensor de temperatura o humedad con pantalla, puerta de enlace WiFi-IR, etc. Todos estos dispositivos necesitan alimentación, utilizo la carga USB desde los teléfonos. Porque cada dispositivo consume una pequeña corriente, y el cargador del iPad da 2.4A, luego decidí guardar las tomas de corriente y conectar todos mis dispositivos a una carga, para esto hice un expansor de puerto USB compacto para cargar, que casi no aumenta el tamaño de la carga en sí.

Para hacer un expansor de puerto USB, necesitamos:
  1. Junta de desarrollo de soldadura china
  2. Dos conectores macho tipo USB (tengo dos conectores)
  3. Cable USB innecesario del que necesita quitar el conector USB tipo macho
  4. Publicar en negro y rojo
  5. Cinta aislante blanca azul
  6. Cinta de espuma de doble cara
  7. Cargador
  8. Equipo de soldadura
  9. 20 minutos de tiempo libre



Suelde cuidadosamente el diseño, mientras trata de no confundir GND y VCC, para no quemar la carga y los dispositivos conectados.



Después de soldar, debe obtener un escudo que se ajuste fácilmente al cargador.




Pruebas con dispositivos USB.



Si todo funciona como el mío, para mayor durabilidad, pegamos el expansor para cargarlo con cinta adhesiva de doble cara en una base de espuma y creamos una hermosa carcasa blanca brillante estilo Apple con cinta aislante blanca.
