DevOpsProdigy KubeGraf v1.1.0

Hoje, revelamos uma nova versão do nosso plug-in de monitoramento do k8s - grafana.com/grafana/plugins/devopsprodigy-kubegraf-app - sobre o qual escrevemos anteriormente .

O principal recurso da nova versão é a capacidade de autenticar usando um token de portador. Agora, através dos manifestos (que podem ser encontrados aqui ), você pode criar um usuário com acesso somente leitura, obter um token (todo o processo é descrito no leia - me ) e usá-lo na janela de configuração do seu cluster.



Também alteramos os pontos de extremidade usados ​​para se comunicar com o k8s-api para dar suporte ao kubernetes v1.16 : As implantações, os statefulsets e os daemonsets desta versão finalmente mudam para / apis / apps / v1 /

Pequenas alterações menores:

  • erros de layout no Grafana v6.3.5;
  • conexão automática ao cluster da instância padrão do Prometheus;
  • foram adicionadas instruções para gerar a chave e o certificado (o que já é especialmente desnecessário, pois existe acesso pelo token :-));
  • formatação de memória fixa na presença de Mi.

Estamos esperando por você em nosso bate-papo por telegrama , onde você pode consultar os desenvolvedores de plug-in online.

E no nosso github : estrelas, ishshui, pull-quests, como sempre, são bem-vindos!
