聆听信息噪音：没人能找到的音乐和视频

在热门系列“瑞克和莫蒂”中，经常出现一个元素-“ 跨维度电视 ”。 这是电视的特殊前缀，在英雄的帮助下，英雄们可以访问所有平行宇宙中的各种电视频道。 不幸的是，在我们的设备中不存在这种设备，但是互联网使我们可以观察到非常相似的东西。

如果您分析“信息噪声”，则可以从新的角度看待我们的世界。 而且（很可能）对您来说似乎陌生，不可预测甚至有些陌生。

我们将告诉您如何进行所有操作。

没人需要视频


YouTube用户每分钟下载超过500个小时的视频内容，因此该服务的推荐系统的算法旨在仅向观众显示他需要的内容。 因此，您可以观察到以下情况：网站上所有视频中有三分之一的观看次数少于十。 这是一个庞大的商业广告档案库，其记录是在没有专业精神的情况下进行的，主要是针对自己或亲戚。 这是没有色彩校正和编辑的普通人的生活。

但是最有趣的是，有一些特殊服务可以观看此类视频，使您可以流式传输带有少量观看次数的视频。

Astronaut.io [ GitHub ]。 此Web应用程序分析其标题具有由智能手机或相机指定的标准名称的最新下载。 通常，它们以IMG或DSC开头，以文件的序列号结尾。 可以合理地假设这些视频的所有者没有费心去更改它们的名称，但这些视频并不是针对广泛的受众，并且不会在数百万的YouTube观众中流行。 尝试作为此类视频的研究人员很容易-您只需按GO按钮。 接下来-每隔几秒钟，播放器将自动从一个剪辑切换到另一个剪辑。

值得注意的是，该服务还有其他选择。 法国PetitTube网站采用的算法不同，它不仅限于带有标准标题的新视频。 在它上面，您可以看到无用的广告，MRI视频和其他奇怪的内容。 可能看起来很奇怪，但是这类视频（如现象）拥有忠实的观众。 小型但充满活力的Reddit IMGXXXX社区致力于发现和讨论此类内容。


DeepIntoYouTube reddit社区是类似视频的另一个来源，但是很少有视频可以在宇航员上找到。 相反，subreddit的重点是出于某种原因“没有开枪”的业余内容。 在这里，您可以找到史莱克面具中的人撰写主题ASMR的视频，仍然有大量荒谬的喜剧视频，没有陌生的用户视频博客就无法做到。

Ctrl + Shift +脸部 。 几年前，这里发布的内容是不可能的-但即使到现在，它仍被认为是一种奇怪的东西。 这是一部深切的视频，其中喜剧演员，演员和其他名人的面孔被他人的面孔代替。 当然，这些内容比荒谬的视频更受欢迎。 在频道上，您可以看到类似“另类现实”的场景 ，其中吉姆·卡里（Jim Carrey）在库布里克的《闪耀》中扮演主要角色， 弗雷迪·水星 （ Freddie Mercury）亲自出演波希米亚狂想曲。


看起来有些奇怪，但是此 YouTube视频包含其自己的URL。 这怎么可能？ 如果您相信视频的创建者，那么只有一个答案-它是从将来下载的。

但是实际上，有一些方法甚至可以在下载开始之前就为视频“保留”一个ID。 但是，尽管作出了解释，但这种Web异常似乎是超自然的。 可以使用Twitter进行类似的操作-并创建包含其自己地址的递归推文 。 该算法使您可以在下载内容之前预测地址，该算法于2009年首次发布。 值得注意的是GIF动画的实验 ，其中包含自己的哈希值。

没有人听的音乐


Forgotify服务类似于前面提到的PetitTube，但不是在YouTube上播放视频，而是在Spotify上找到几乎没人听过的歌曲。 这是下载到该网站的所有内容的第五部分 ：从60年代被遗忘的艺术家到“没有给予”的说唱歌手，或者甚至是那些直接参与录音的人也最有可能忘记的曲目。
