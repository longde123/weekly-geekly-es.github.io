WebRTC sur votre site - pas de bugs et pas de budget

La technologie WebRTC existe depuis longtemps et est prise en charge par presque tous les navigateurs (97% selon notre expérience). Beaucoup de gens pensent à installer le widget WebRTC pour eux-mêmes, pour communiquer avec les clients et augmenter la conversion.



Comment ajouter des appels à votre site Web WebRTC sans frais supplémentaires et contourner les bogues avec des râteaux, nous le dirons ci-dessous.

Pourquoi un bouton avec WebRTC?


  1. Idéalement! Pas besoin de composer ou de laisser votre numéro, appuyez simplement sur le bouton.
  2. Rentable. Ni vous ni le client ne payez quoi que ce soit. La communication est gratuite pour tout le monde.

Quelles sont les solutions?


Développement personnel


Vous pouvez installer un astérisque ou freeswitch pour vous-même et essayer d'exécuter le code de téléphone webrtc gratuit dessus. Tôt ou tard, le code fonctionnera, mais pour fonctionner avec la plupart des navigateurs, vous devez faire beaucoup de chemin. Un soutien supplémentaire commencera:

  • tout d'abord, l'astérisque avec l'ip externe nécessite des connaissances en sécurité (ils aiment beaucoup les casser),
  • deuxièmement, le support de webrtc dans les navigateurs n'est pas statique, les nuances changent dans les nouvelles versions.

Exemple de problème WebRTC


À l'automne 2018, après la mise à jour de Firefox et Safari, le travail de la plupart des téléphones Webrtc est tombé en panne. Dans Firefox, il y a eu un problème au début de l'appel, dans Safari, le son vient de disparaître. Après plusieurs jours de travail des développeurs et des testeurs, les problèmes ont été corrigés (avant même que la plupart des clients soient mis à jour). Firefox a fixé un correctif temporaire pour les prochaines mises à jour, pour Safari, ils ont légèrement simplifié l'algorithme du téléphone Web. Êtes-vous sûr d'être prêt à suivre et à résoudre tous ces problèmes de manière indépendante?

Widget externe


Si vous n'avez pas le temps et le désir de traiter les problèmes WebRTC, vous pouvez installer un widget cloud. Le widget cloud a déjà été créé et pris en charge par l'opérateur, il vous suffit de passer quelques minutes sur l'installation.

Habituellement, le principal inconvénient d'une solution cloud est le prix - vous devez payer régulièrement pour utiliser le widget, et quoi de mieux alors, par exemple, le numéro 800? Le projet Zadarma propose une autre solution - un widget gratuit pour les appels du site via WebRTC. Tout ce qui est requis du côté financier est au moins parfois de reconstituer le compte (tous les 3 mois), vous pouvez utiliser les fonds pour tous les services de téléphonie, le widget lui-même est gratuit.



Fonctionnalités gratuites du widget cloud


  • Conception personnalisable (forme, couleur, police, boutons)
  • Statistiques de widget étendues
  • Intégration avec un PBX cloud gratuit et tous les principaux systèmes CRM

Installez un widget gratuit


La création d'un widget est extrêmement simple:

  • Nous créons un compte et réapprovisionnons le compte (pour tout montant)
  • Dans le service LC, allez sur le bouton «Appelez-nous» et cliquez sur «Créer un widget»
  • Configurez la conception du widget, cliquez sur «Enregistrer» et obtenez le code du widget



Beaucoup de choses sont configurées dans le widget, vous pouvez même choisir quoi garder à la maison et ce qui est dans le cloud (pour les grands sites et les fans d'AWS, vous voudrez peut-être l'héberger).

Pour stocker le widget dans le cloud (d'une manière simple), vous devez copier 2 blocs de code sur votre site.

  • En bas de la page (section Corps), copiez le code de démarrage de la bibliothèque de widgets.
  • Nous définissons le code du widget lui-même sur la partie de la page où doit se trouver le bouton «Appelez-nous».


Comme vous pouvez le voir, la conception du widget peut être modifiée à tout moment.


Si vous décidez de garder le widget à la maison, vous devez le télécharger et l'enregistrer. Ensuite, nous sélectionnons l'onglet «Installation hors ligne», téléchargeons la dernière version et insérons également 2 morceaux de code (uniquement dans la première, vous devez spécifier les chemins en tenant compte du dossier dans lequel vous avez téléchargé).

L'installation est terminée! Le bouton pour les appels du site est prêt et fonctionne. Vous pouvez cliquer, appeler, voir les statistiques.
