Revue vidéo de l'ordinateur portable Dell Alienware 17 R3



Dell Alienware 17 R3 - un ordinateur portable puissant pour les joueurs, équipé d'un écran IPS mat de 17,3 pouces avec une résolution de 1920x1080 (UHD en option). Le processeur Intel Core i7-6820HK avec un multiplicateur déverrouillé est responsable des performances (nous avons réussi à l'overclocker à une fréquence de 4090 MHz sur les quatre cœurs). Le sous-système graphique est représenté par la carte vidéo Geforce GTX 980M (4 Go GDDR5), ainsi que la possibilité de connecter une station d'amplificateur graphique Alienware spéciale sur laquelle une carte vidéo de bureau standard est installée (par exemple, vous pouvez y installer la GTX 980Ti). La quantité de RAM est jusqu'à 32 Go DDR4, deux emplacements sont disponibles. Le système de stockage de données est représenté par un disque dur de 1 To (7200 tr / min) et deux emplacements M.2 pour les SSD PCI / NVMe.
