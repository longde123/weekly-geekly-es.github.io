Orange Pi 2G-IoT-物联网的理想单板

大约一个月前，网络上出现了关于非常出色的单板Orange Pi 2G-IOT的信息，该单板旨在实现物联网领域的各种项目。


该解决方案基于SoC RDA8810PL。 该芯片包含一个1.0 GHz ARM Cortex-A5计算核心，256KB L2高速缓存，256MB LPDDR2 RAM，512MB SLC NAND闪存和Vivante GC860图形控制器。



有一个4频段GSM / GPRS调制解调器，可以通过蜂窝网络交换数据。 板上还有一个SIM卡插槽。



小型计算机在RDA5991控制器上带有Wi-Fi 802.11b / g / n和蓝牙2.1 / EDR无线适配器。 有一个完整的USB 2.0端口，一个3.5毫米音频插孔和一个FM天线输出，该插孔带有FM天线输出，并具有通过它供电的能力，一个麦克风，一个用于调试的3针UART接口，一个用于连接相机的CSI连接器高达2Mpx，能够以30fps的速度捕获和视频高达1080p，该接口用于连接分辨率高达qHD（960 x 540）的LCD显示器。

板上有一个40针的SPI，I2C，ADC和GPIO连接器，类似于Raspberry Pi B +。 主板尺寸仅为67×42 mm，重量为35 g，可使用容量高达8 GB的microSD闪存卡存储数据。

宣布支持操作系统：Android，Ubuntu，Debian和Rasberry Pi映像。

Orange Pi 2G-IoT的价格约为10美元。
