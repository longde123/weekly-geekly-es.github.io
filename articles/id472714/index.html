Dimana pekerja front-end untuk mencari pekerjaan dan tidak jatuh cinta: Telegram, Slack dan tidak hanya

Sudah ada materi di mana mereka berbicara tentang bot yang berbicara dengan perekrut dan semua jenis filter spam untuk surat. Beberapa minggu yang lalu, saya juga memutuskan untuk berganti pekerjaan. Saya punya satu tugas - terutama untuk tidak bersinar.



Dan saya ingin mencari pekerjaan dengan hati-hati karena dua alasan:

  1. Saya tidak ingin mengiklankan bahwa saya sedang mempertimbangkan proposal, karena ada tanggung jawab kepada majikan saat ini dan proyek-proyek yang belum selesai;
  2. Saya tidak ingin menangkap banyak spam dan pergi untuk wawancara di perusahaan yang mereka sendiri tidak tahu siapa yang mereka cari.

Bagaimana semuanya dimulai. Oke, pertama saya memposting resume di Hehe (di sana Anda juga bisa rapi anonim), tetapi setelah 10 menit itu masuk:



Jadi saya memutuskan untuk mengendalikan proses pencarian sendiri dan menghancurkan resume saya dari sana.

Dia mengumpulkan sendiri platform tempat ini bisa dilakukan, saluran dan obrolan Telegram, komunitas Slack. Dan untuk pecinta eksotis - Skype.

Sementara saya sedang dalam proses wawancara. Dan saya membagikan pilihan:

Platform:


Djinni - Saya banyak mendengar tentang dia, tetapi terutama ada penonton Ukraina dan saya tidak mendapatkan apa pun yang masuk akal (saya menilai dengan statistik SimilarWeb).

Arena adalah platform baru tempat Anda dapat mencari secara anonim. Sebuah lowongan muncul dan Anda mengklik "Menarik" atau "Tidak menarik". Jika yang pertama, maka perusahaan melihat kontak Anda dan menghapusnya. Saya mendapat beberapa lowongan, semua aturan. Seperti satu.

Obrolan telegram:


pro js - chat, di mana antara lain Anda dapat menemukan lowongan.

Javscript Jobs ( @javascript_jobs ) - ada banyak pembicaraan, tetapi ada juga lowongan. Paling sering mereka digandakan dengan @js_ru.

Saluran telegram:


Pekerjaan JavaScript ( @jsdevjob ) adalah saluran komunitas Proglib, ada posisi yang masuk akal, tetapi mereka jarang memposting sesuatu.

Pekerjaan JavaScript ( @js_ru ) adalah saluran terbesar dengan pekerjaan JS. Sangat keren bahwa mereka sangat dimoderasi dan pasti ada plug di mana-mana. Ada saluran untuk kerangka kerja yang berbeda: Node JS Jobs ( @nodejs_jobs ), React JS ( @react_js ), Angular JS ( @angular_ru ).

Web Freelance ( @webfrl ) - saluran tempat Anda dapat menempel pada beberapa jenis pekerjaan proyek atau situs jarak jauh.

TProger ( @tproger_official ) adalah saluran programmer Khas, di sini, di samping artikel dan segala macam hal serupa, ada juga lowongan.

Pekerjaan Pengembang Front-end ( @front_end_jobs ) adalah saluran yang sangat kecil, tetapi kadang-kadang posisi menarik datang di sini.

Lowongan TOP untuk programmer ( @jobGeeks ) - semuanya dalam satu baris, Anda bisa mendapatkan sesuatu yang masuk akal dengan tag #frontend.

Komunitas kendur


Javascript RU adalah komunitas yang baik, antara lain, di sini timlids sendiri terkadang memposting lowongan.

Bereaksi JS adalah komunitas internasional, tetapi ada juga cabang dengan lowongan.
