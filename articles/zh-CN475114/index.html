.NET Core 3.1预览版2简介

.NET Core 3.1预览版简介2 。 .NET Core 3.1将是一个小版本，着重于对Blazor和Windows桌面（.NET Core 3.0的两个重要新版本）的关键增强。 这将是一个长期支持（LTS）版本，预计最终推出日期为2019年12月。

您可以下载适用于Windows，macOS和Linux的.NET Core 3.1 Preview 2 。

  • .NET Core 3.1 Preview 2 SDK和运行时
  • Docker映像

ASP.NET Core和EF Core现在也可用。

现在还提供了Visual Studio 16.4 Preview 3和Visual Studio for Mac 8.4 Preview 3。 这些是使用.NET Core 3.1预览版2所必需的更新。VisualStudio 16.4包括.NET Core 3.1，因此，简单的Visual Studio更新将为您安装两个版本。

详细资料：

  • .NET Core 3.1发行说明
  • GitHub发行说明



增强功能


此版本中最大的改进是对C ++ / CLI（又称为“托管C ++”）的支持。 C ++ / CLI仅在Windows上可用。 C ++ / CLI的更改主要在Visual Studio中进行。 要使用C ++ / CLI，您需要设置“ C ++桌面开发”工作负载和“ C ++ / CLI支持”组件。 您可以在下图中看到该选定组件（最后显示）。



该组件添加了两种可以使用的模式：

  • CLR类库（.NET Core）
  • CLR空项目（.NET Core）

如果找不到它们，只需在“新建项目”对话框中找到它们。

结论


.NET Core 3.1的主要目标是改进我们在.NET Core 3.0中引入的功能和脚本。 .NET Core 3.1将是一个长期支持（LTS）版本 ，至少支持3年。

请安装并测试.NET Core 3.1 Preview 2并留下您的评论。 请注意，不建议将该版本用于生产。

如果您突然错过了它，请阅读.NET Core 3.0的公告。
