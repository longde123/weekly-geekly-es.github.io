Aceptación de pagos desde una tarjeta sin jur. caras en Yandex.Money


Servicio de donación de dinero yandex de código abierto


Aceptar pagos en Yandex.Dinero físico. caras

  • pagos en línea desde tarjeta, Yandex, teléfono
  • agregar etiqueta al pago
  • recibir y verificar alertas en su servidor
  • Ejemplo de automatización de bot de Telegram

Implementación y descripción bajo el corte

Proyecto Python



Módulos:


  • money.yandex.ru - billetera personalizada para la cual recibimos dinero
  • httpsserver : servidor python que genera páginas de pago y recibe alertas de Yandex
  • billingservice : procedimiento remoto, llamado después de recibir una notificación de pago, funciona con la base de datos
  • redis-server : se utiliza como servidor de cola para llamadas a procedimientos remotos
  • ejemplo / telegrambot - ejemplo de generar enlaces para pagar bot de Telegram

Botones para recoger dinero


Yandex.Money tiene la capacidad de crear botones y formularios para recolectar dinero en los que puede insertar sus comentarios y etiquetas de pago.



Notificaciones de pago


Además, es posible configurar la recepción de notificaciones de pago en su servidor.



Agregar aceptación de pago


  1. Registrarse, aumentar el estado de la billetera a un valor nominal
  2. AJUSTES-> TODO lo demás-> NOTIFICACIONES HTTP
    • Agrega tu dominio y secreto
    • Activar notificaciones
  3. Levante redis-server , agregue parámetros de acceso a rq_access.py
  4. Recibimos el certificado SSL para el dominio, por ejemplo a través de letsencrypt
  5. Levante el servidor https con sus parámetros, no se olvide del firewall si el servidor no está disponible
  6. Configuramos la identificación de la billetera y el secreto para las notificaciones desde la configuración en httpsserver / .secret / ymsecret.py
  7. Edite httpserver-ympayment.service , inicie el servidor bajo el supervisor
  8. En la máquina con la base, edite billing.service y ejecute systemd para este servicio.
  9. Comprobación de inicio remoto de procedimientos a través de la cola Redis
  10. Según el ejemplo, agregamos la generación del enlace de pago a su función bot
