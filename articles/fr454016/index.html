3CX v16 Update 1, application 3CX pour iOS Beta et la nouvelle version 3CX Call Flow Designer

Présentation de la dernière revue de produit 3CX. Il y aura beaucoup de choses intéressantes - ne changez pas!

3CX v16 Update 1


Nous avons récemment publié la prochaine mise à jour 1 de 3CX v16. La mise à jour comprend de nouvelles fonctionnalités de chat et un widget de communication mis à jour pour votre site 3CX Live Chat & Talk. La mise à jour 1 a également introduit un nouveau service de flux d'appels, qui ajoute une interface de script de contrôle des appels au PBX. Le «moteur» de script fonctionne avec l'environnement de développement de Call Flow Designer et vous permet de créer des scripts de traitement d'appels de toute complexité.

Chat mis à jour dans le client Web




Le chat mis à jour vous permet désormais de gérer de manière flexible vos conversations. De plus, il interagit bien avec le widget de communication 3CX Live Chat & Talk .

  • Un visiteur du site peut démarrer une conversation avec la file d'attente des opérateurs 3CX. Cela crée un groupe de discussion, qui comprend tous les opérateurs de file d'attente et ce visiteur.
  • À l'avenir, l'opérateur de file d'attente peut basculer le chat de groupe vers lui et poursuivre la communication personnelle avec le visiteur.
  • En outre, l'opérateur peut transférer le chat à un autre opérateur ou à un utilisateur PBX ordinaire, si le besoin s'en fait sentir.
  • Pour décharger l'interface du client Web, les conversations sélectionnées peuvent être transférées vers l'archive (cependant, elles ne sont pas supprimées).
  • Différents types de chats (sur le site, en groupe, etc.) ont désormais différentes icônes pour une identification facile.
  • Vous pouvez désormais envoyer rapidement un e-mail à un visiteur du site en cliquant sur un e-mail dans la fenêtre de discussion.

Les nouvelles fonctionnalités sont décrites plus en détail dans le guide de discussion et dans la vidéo .

Mise à jour du widget 3CX Live Chat & Talk




Le widget 3CX Live Chat & Talk mis à jour offre une personnalisation supplémentaire de l'interface et une intégration améliorée avec les sites créés à la fois sur le CMS WordPress et sur d'autres technologies.

  • Définition de l'icône de la fenêtre de discussion - vous pouvez définir une image appropriée pour le titre de la fenêtre de discussion. Il peut s'agir, par exemple, du logo de votre entreprise.
  • Définition de l'icône de l'opérateur - vous pouvez également définir l'icône de l'opérateur de chat, par exemple, sa photo.
  • Placement du widget - le paramètre «Position» détermine le placement du widget sur les pages Web - en bas à droite (par défaut) ou en bas à gauche.
  • La vue dans le navigateur mobile est un ajout petit mais important. Désormais, lorsque vous accédez au site à partir d'un appareil mobile, la fenêtre de discussion est minimisée par défaut.
  • Fenêtre contextuelle de conversation - dans 3CX v16 Update 1, la fenêtre de widget 3CX Live Chat & Talk "apparaît" dans une fenêtre séparée, permettant au visiteur de se déplacer librement sur le site, mais en même temps de contacter la conversation à tout moment.

Interface de script Service de flux d'appels


La mise à jour 1 de 3CX v16 introduit une nouvelle interface de script du service Call Flow Apps Service. Il prend en charge les applications vocales 3CX nouvelle norme. Cependant, les applications existantes peuvent être converties ou modifiées dans la nouvelle version de Call Flow Designer (voir ci-dessous). L'architecture de Call Flow Apps Service est finalisée aujourd'hui. Le serveur d'applications fonctionne sur 3CX pour Debian / Raspbian Linux et Windows.

Une vidéo sur la migration de vos applications vocales.

  • Windows: v16 Update 1
  • Linux: 3CX v16 pour Debian 9 ISO .
  • Installation manuelle sur Debian et Raspbian - suivez les guides d'installation pour Debian et Raspberry Pi .

Nouvelle application 3CX pour iOS beta


Nous n'avons pas mis à jour notre application iOS 3CX propriétaire depuis longtemps. Certains utilisateurs se sont même plaints d'un transfert de fichier incorrect. Mais dans la prochaine mise à jour, tous les problèmes sont résolus! Cette fois, l'accent est mis sur les capacités du chat intégré. Maintenant, le chat dans l'application mobile n'est pratiquement pas inférieur au chat dans le client Web 3CX.



L'application a la possibilité de créer des discussions de groupe et de leur attribuer des noms. Ajout de l'archivage de chat. Pour archiver une conversation, balayez vers la gauche (vous pouvez restaurer une conversation à partir de l'archive de la même manière).

Toujours dans l'application, il y avait un chat avec les visiteurs du site via le widget de communication Live Chat & Talk 3CX. Voici comment il est implémenté maintenant:

  • Balayez vers la gauche pour discuter pour découvrir des options supplémentaires: Prenez-vous, Transférer, Terminer et Supprimer.
  • Les icônes de discussion sur le site sont différentes des icônes de discussions régulières afin que vous puissiez facilement les distinguer.
  • Les notifications PUSH des messages du site affichent le nom du visiteur et le contenu du message.
  • Les discussions envoyées à la file d'attente d'agent incluent le nom de la file d'attente pour votre commodité.

Nouvelle version 3CX Call Flow Designer


Cette semaine, nous avons publié une nouvelle version de l'environnement de développement d'applications vocales 3CX CFD . Il a introduit de nouveaux composants C #, une interface utilisateur raffinée, une meilleure gestion des erreurs, ainsi qu'une mise à jour automatique de l'environnement de développement. Un nouveau CFD est nécessaire pour créer un nouveau type d'applications vocales pour 3CX v16 Update 1 et supérieur.


L'environnement de développement mis à jour (IDE) CFD offre des outils supplémentaires pour les développeurs:

  • Nouveaux composants Exécuter le fichier C # et le code. Ils remplacent l'ancien composant Launch External Script. Les composants peuvent exécuter des fichiers de code C # ou des extraits de code en ligne directement à partir des applications CFD.
  • Le nouveau composant «Définir le statut de l'extension» implémente un simple réglage des paramètres d'extension à partir de l'application CFD.
  • Amélioration de la gestion des erreurs. Le nouvel éditeur d'expression vérifie en outre les valeurs, détectant les erreurs au stade de la compilation.

En plus des améliorations concernant le processus de développement, nous avons ajouté plusieurs fonctions qui augmentent la convivialité de l'application elle-même:

  • Application de mise à jour automatique. CFD recherche désormais automatiquement les nouvelles versions et installe les mises à jour dès leur sortie.
  • Un nouvel élément de menu «Enregistrer le projet sous» vous permet d'enregistrer un projet CFD sous un nouveau nom ou dans un emplacement différent.
  • Nouveau menu contextuel "Open Audio Folder" pour les composants qui utilisent des fichiers audio. Il ouvre l'Explorateur pour parcourir facilement le dossier audio du projet.
  • Affichage pratique des composants désactivés. Ils sont maintenant grisés pour différer des composants actifs du CFD.

La nouvelle version de CFD implique l'utilisation de 3CX V16 Update 1. Téléchargez le CFD et installez-le à partir du Guide d'installation de Call Flow Designer .
