DJI Osmo Mobile 3最佳12项创新



DJI Osmo Mobile 2已成为许多博主和初学者的必备工具，没有它们，他们就不会离开家。 尽管没有袋子或特殊情况下携带它的问题颇为严重，但Osmo Mobile 2的功能始终胜过运输过程中的轻微不适。 DJI考虑到缺乏模型，研究了错误，并提出了一种新设备，您可以随身携带。 一年来第二次，Osmo产品阵容有了出色的配件，或者说是专为智能手机创建的真正的移动Osmo Mobile 3稳定器。

第三代创新




是的，折叠式稳定器的想法并不是什么新鲜事物，但是到目前为止，只有DJI才能在最高水平上实现它。 在此模型中，所有内容都从字面上考虑到了最小的细节。 该公司的工程师听到了Osmo Mobile 2用户的所有愿望和要求，并将其体现在新设备中。

这不仅涉及折叠设计，而且还涉及在待机模式下不将智能手机从底座上卸下的能力，使用稳定器按钮（负责各种功能的附加触发按钮）浏览移动应用程序的菜单，只需从智能手机的水平位置切换到垂直位置即可充电录制过程中使用Osmo Mobile 3的手机以及其他有用功能，我们将在后面讨论。

当然，DJI乐于遵循其移动应用程序统一的道路。 如果几个月前您有幸成为公司最新直升机之一的所有者，智能手机的稳定器以及例如Osmo Action动作相机的所有者，那么您必须在手机上安装三种不同的应用程序：DJI GO，DJI GO 4和DJI Mimo。 我们必须放心：Osmo Mobile 3支持使用DJI Mimo应用程序（顺便说一下，该应用程序最近已被俄罗斯化），并且DJI GO逐渐成为适用于旧设备的程序。

让我们快速看一下Osmo Mobile 3的主要功能：

  • 展开外观的尺寸为285×125×103 mm；
  • 折叠尺寸157×130×46 mm;
  • 电池容量2450 mAh;
  • 工作时间15小时。

机械旋转角度范围：

  • 旋转：从-162.5°到+ 170.3°;
  • 从-85.1°滚动到+ 252.2°;
  • 从-104.5°倾斜到235.7°;
  • 最大控制速度120°/ s。

Osmo Mobile 3的主要新功能




1号 折叠设计

这就是第三代Osmo Mobile的与众不同之处。 在新颖性方面，安装了L形的三轴稳定器，而不是先前模型的三部分。 此外，DJI的想法更进一步-Osmo Mobile 3不仅可以累加，还可以固定所有轴以确保安全运输。 工作原理：我们组装了稳定器，小的突起掉入了铰链上相应的凹槽中，仅此而已，您不必担心设备会在包装袋中扭曲并且敏感的电动机会损坏。

2号 附加触发按钮

手柄背面的这一有用元素使您可以非常舒适地拍摄视频。 例如，三按可激活前置摄像头，将智能手机居中两次。 按住两次可激活运动模式，如果按住它，则无论您按住稳定器的角度如何，智能手机都将锁定在一个位置。 要启动或停止ActiveTrack 3.0跟踪模式，只需扳动一次扳机即可。

3号 待机时间

最便捷的创新之一。 无需从底座上卸下智能手机即可编辑视频或观看素材。 将固定有智能手机的Osmo Mobile 3折叠起来，然后按三下M按钮即可，在此模式下，您可以使用智能手机并在适当的时间立即返回拍摄。

4号 新的管理和导航



除机械按钮外，手势也可用于控制拍摄。 露出手掌或两个手指的字母V形状，相机将开始录制视频或拍照。 顺便说一下，这适用于主摄像头和前置摄像头。 现在，您可以使用操纵杆和稳定器按钮浏览应用程序菜单。 如果您戴着手套拍摄或第二手很忙，此功能很方便。

5号 ActiveTrack 3.0

Osmo Mobile 3使用了新版本的对象跟踪模式。ActiveTrack3.0现在可以非常快速，非常流畅和尽可能准确地工作。 而且，如果较早需要自己选择跟踪对象，则在新版本中，自动化是负责的：它可以识别人脸并将其保持在框架的中央。 不信任软件算法？ 然后像以前一样使用该功能，突出显示屏幕上的对象。 顺便说一句，ActiveTrack 3.0现在不仅可以在视频拍摄模式下使用，而且可以在创建Hyperlapse和SlowMo剪辑以及拍摄照片的过程中使用。

6号 使用M按钮更改智能手机的位置

双击M按钮，您将智能手机翻转90度。 现在，从横向到纵向模式的转换仅需几秒钟。 即使在录制时也可以使用！

7号 蓝牙5.0

Osmo Mobile 3支持最新版本的Bluetooth 5.0无线标准。 这意味着稳定器的使用寿命更长，功耗更低，并且传输数据的速度是其稳定器的两倍。

8号 USB Type-C端口



DJI遵循现代趋势之路的另一个标志。 几乎所有的现代设备都在逐步向该标准迈进，大约两年后，将不再有过时的microUSB设备。 因此，值得考虑更新电汇。

9号 新稳定器-新机芯

Osmo Mobile 3稳定器的形状和设计扩展了不同相机移动的范围：您可以沿不同方向旋转它，也可以相对于框架中心扭曲它。 一种新的安装方式可以与大型手机或智能手机兼容。

10号 运动模式

Osmo Mobile 3的创意拍摄机会列表已添加到更新的运动模式中。 有了它的帮助，即使是从一个动态场景到另一个动态场景的过渡，也会在视频上显示出流畅而动感的影像。

11号 重量轻

尽管进行了所有创新并增加了尺寸，Osmo Mobile 3的重量却更轻：405克，而Osmo Mobile 2版本为485克；重量更轻-拍摄时间更长：即使录制很长的视频，您也不会厌倦保持斯坦尼康来拍摄的效果。

12号 DJI Mimo支持



如前所述，DJI逐渐停止使用DJI GO应用来控制新设备。 DJI Mimo正在用大量不同的艺术效果和实现创意的机会代替。 Osmo Mobile 3也不例外，并且也使用该软件，这非常令人愉快。 怎么了 至少有三个原因。
