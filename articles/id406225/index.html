Kami menggunakan program Androzic - peta gps offline masih relevan

Hai Geektimes! Masa liburan sudah hampir berakhir, namun, perjalanan panjang tetap relevan. Sekarang "modis untuk kaum muda" untuk menggunakan berbagai layanan online, tetapi meskipun penerima GPS besar dan kuning, peta offline lama yang baik tidak kehilangan relevansinya - saat bepergian, Internet sering tidak ada atau mahal atau tidak nyaman. Ya, bahkan di wilayah rumah Internet seluler tidak selalu terhubung, seringkali tidak diperlukan untuk membayarnya secara terpisah. Dan gps ada di hampir semua ponsel cerdas, dan Anda tidak perlu membayar apa pun untuk penggunaannya.

Saya akan memberi tahu Anda cara menyiapkan peta luring untuk program Androzic yang tidak terlupakan, program yang telah saya gunakan selama lebih dari 5 tahun, dan yang saya gunakan di semua perjalanan, dari Paris ke Solovki.

Seperti inilah tampilan peta di ponsel cerdas saya dengan Android 7.0:



Bagaimana melakukan ini, sedikit arkeologi digital di bawah jalan pintas.

Program Androzic


Pertama, Anda perlu menginstal program ini. Anda dapat mengunduhnya dari situs dengan nama yang sama , program ini sepenuhnya gratis, ada juga tautan di situs ke sumber - sumber di github . Sayangnya, penulis berhenti mengembangkan program, tetapi semuanya berjalan seperti itu - program melakukan apa yang diperlukan dan tidak melakukan apa yang tidak perlu. Saya menggunakan versi 1.7.9, karena 2.3 sesuatu tidak tumbuh bersama, dan terlalu malas untuk mengerti. Jadi cerita selanjutnya adalah tentang 1.7.9.

Jadi, unduh dari situs apk , instal di Android. Program ini menyediakan semua yang Anda butuhkan untuk navigasi offline - peta, titik, jarak tampilan, kecepatan, dll. (Tentu saja, tidak ada layanan online seperti menunjukkan kemacetan lalu lintas dalam program).

Tidak ada peta yang disertakan dengan program ini, jadi Anda harus menyiapkannya sendiri. Turun ke arkeologi digital - kita akan membutuhkan program yang digunakan sekitar 10 tahun yang lalu :)

Pemuatan peta


Pertama-tama kita perlu membuat peta dalam format OZI Explorer. Setelah mencoba berbagai program, saya akhirnya memilih SAS.Planet . Program ini gratis, dengan bantuannya Anda dapat menyimpan peta dalam format apa pun yang nyaman. Misalnya, buat peta Moskow.

Pilih area yang diinginkan, pilih zoom yang diinginkan (z15) dan klik Mulai - program mengunduh semua ubin dari peta Google (Saya tidak tahu berapa banyak ini diizinkan oleh kebijakan Google, tapi saya pikir itu mungkin untuk penggunaan pribadi).



Buka pilihan sebelumnya lagi, pilih Stitch dan unduh peta dalam format jpeg (zoom yang sama = 15 dipilih dan bendera "peta" dihidupkan untuk membuat file ozi).



Hasil - 2 file akan muncul di folder yang dipilih - Moscow.jpg (berukuran sekitar 7000x7000 piksel) dan Moscow.map. Sekarang tugas kita adalah mengonversi kartu dan memasangnya di smartphone.

Konversi kartu


Langkah selanjutnya adalah mengonversi kartu ke format yang dipahami Androzic. Sayangnya, file yang disimpan di atas tidak terbuka secara langsung, kartu harus terlebih dahulu dikonversi ke format ozfx3. Untuk konversi, saya menggunakan program Img2OZF 3.0. Versi selanjutnya tidak cocok, karena mereka sudah membuat file dalam format ozf4 lain (mereka berhasil mengunduh versi 3.0 dari sini , karena tautan resmi sudah lama mati). Jika ada yang tahu konverter lain, itu akan menarik.

Menggunakan program ini sangat sederhana - pilih folder, klik Konversi.



Program ini membuat layer peta untuk level zoom yang berbeda, setelah beberapa saat peta siap. Akibatnya, 2 file Moscow.ozfx3 dan Moscow_ozf.map dibuat di folder output.

Unduh ke smartphone


Langkah terakhir dan termudah. Kami menghubungkan smartphone melalui usb, dan menyalin kedua file ke folder / Androzic / maps. Kami me-restart Androzic - semuanya siap, kartu baru terlihat dalam daftar, dan dapat dibuka.





Jika diinginkan, di Androzic Anda juga dapat memasukkan daftar poin menarik, dan program akan menampilkannya di layar. Petunjuk - jika Anda memilih "apa itu" di google maps di menu konteks, koordinat dalam bentuk garis lintang-bujur akan ditampilkan, dan mereka harus dimasukkan ke dalam program.



Hasil


Apa yang kita dapatkan sebagai hasilnya? Kami memiliki program gratis yang menunjukkan pada smartphone lokasi kami, peta, kecepatan, koordinat, daftar titik menarik, rekaman trek gerak, dan semua ini sepenuhnya offline, tanpa menggunakan lalu lintas atau jaringan gsm pada umumnya. Tidak ada iklan, pendaftaran, dan sms. Saat bepergian dalam roaming / ke luar negeri, atau bahkan di pesawat terbang, lihat di mana kita terbang sekarang, apa yang diperintahkan dokter.
