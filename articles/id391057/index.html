Kuliah "Harta Karun Dingin Jauh"

2015 dapat disebut sebagai tahun planet kerdil - penyelidikan Fajar memasuki orbit di sekitar Ceres, New Horizons terbang melewati Pluto. Dan awal 2016 ditandai dengan pembicaraan tentang planet kesembilan yang jauh dan misterius. Sudah waktunya untuk berbicara tentang tata surya luar. Ini adalah kuliah saya berikutnya tentang "Harta Karun Dingin Jauh". Jauh dan dingin, tetapi mengandung banyak hal menarik. Sungai metana dan pantai es, lautan sub-glasial dengan kemungkinan kehidupan, detektif asteroid, penemuan di ujung pena dan "mata" senjata.


Ini lucu, tetapi cerita dengan kuliah ini agak diilustrasikan oleh pepatah "tidak akan ada kebahagiaan, tetapi kemalangan akan membantu". Faktanya adalah bahwa saya biasanya memberi kuliah di anti-kafe, tetapi ada overlay - waktu sudah diambil setelah persetujuan dan pengumuman. Saya mulai berpikir apakah akan mencari tempat lain atau mengubah waktu ceramah, tetapi tiba-tiba mereka memanggil saya dari planetarium Ufa dan mengundang saya untuk memberi ceramah kepada mereka. Untuk bagian saya, saya mengucapkan terima kasih kepada tim planetarium dan secara pribadi kepada direktur Anatoly Pavlovich Denisov untuk menyediakan platform yang luar biasa. Saya berharap kerja sama kami akan terus berlanjut.

Kali ini saya memutuskan untuk sedikit mengubah format ceramah. Pertama-tama, saya mengurangi durasinya menjadi sekitar satu jam (sebelum kuliah berlangsung dari satu setengah jam, dan sekali, ketika tidak ada yang menghentikan saya, saya berbicara selama hampir empat jam). Saya pikir format ini akan sedikit lebih mudah dibaca. Juga, teks benar-benar menghilang dari slide, yang, secara teori, harus menyederhanakan persepsi materi. Jangan ragu untuk menulis ulasan dan komentar, saya akan berterima kasih atas umpan baliknya.

Seperti biasa, presentasi tersedia dalam format ODP , gunakan untuk kesehatan.
