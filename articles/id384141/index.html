Windows selama bertahun-tahun: tiga puluh tahun sistem operasi Microsoft

Versi pertama dari shell grafis Windows untuk MS-DOS dirilis hampir tiga puluh tahun yang lalu, hanya beberapa bulan tersisa sebelum ulang tahun. Setelah beberapa dekade, pangsa pasar Windows di pasar komputer pribadi lebih dari sembilan puluh persen. Lebih dari setengah dari semua komputer desktop berjalan pada Windows 7.

Mari kita ingat versi Windows yang telah kita lihat sejak 1985.



Versi pertama dari shell grafis Windows dirilis pada tahun 1985: pada tanggal 20 November, itu muncul di rantai ritel. Untuk menghindari masalah hukum dengan Mac OS, pengembang tidak menggunakan keranjang, dan windows tidak bisa saling tumpang tindih. Versi Windows ini tidak memiliki bilah tugas di bagian bawah layar. "Add-on yang tidak berguna lebih dari DOS" termasuk satu game - Reversi.



Dua tahun kemudian, pada tahun 1987, Microsoft merilis versi Windows 2.0. Di sini muncul aplikasi Paint yang kita kenal sejak kecil, jam tangan, manajer file MS-DOS, dan terminal. Tak lama setelah rilis 2.0, perusahaan pertama mentransportasikan Word dan Excel ke Windows.



Windows 3.0 pada 1990 termasuk Program Manager, dukungan untuk VGA dan memori virtual, desain "3D" baru, dan game "The Scarf". Permainan itu dibutuhkan bukan untuk hiburan pekerja kantor, tetapi untuk melatih pengguna dalam keterampilan menggunakan mouse .



Pada tahun 1992, Microsoft merilis Windows 3.1, menambahkan font TrueType yang dapat diskalakan dan game Minesweeper ke sistem untuk melatih akurasi memukul layar dengan kursor. Untuk pertama kalinya, dukungan konten multimedia telah muncul di sistem.



Windows for Workgroups 3.1 dirilis pada tahun 1992. Ini termasuk dukungan asli untuk jaringan - khususnya jaringan area lokal LAN, dengan cepat mendapatkan popularitas dalam bisnis saat itu.

1993 dapat disebut tahun kelahiran sistem operasi Windows. Sebelum Windows NT 3.1, apa yang disebut Windows adalah shell grafis untuk MS-DOS. Sekarang Microsoft telah memperkenalkan sistem operasi 32-bit yang dirancang untuk workstation modern. Secara eksternal, OS tidak berbeda dari Windows 3.1.



Pada Agustus 2015, Windows 95 berusia 20 tahun . Sistem komputer telah tersedia tidak hanya untuk Geeks, tetapi juga untuk ibu rumah tangga. Tombol Start dan bilah tugas disederhanakan bekerja dengan sistem.



Windows NT 4.0 telah menggabungkan antarmuka yang mirip dengan Windows 95 dengan kernel Windows NT yang kuat. Sistem ini telah menjadi sistem operasi Microsoft yang paling populer selama beberapa tahun ke depan.



Windows CE adalah sistem operasi untuk komputer genggam, telepon pintar, dan sistem tertanam. Itu dirilis pada November 1996, pada waktu itu itu adalah versi "dilucuti" dari sistem operasi desktop. Versi terbaru dari sistem - 7.0 - dirilis pada Maret 2011.



Internet Explorer Web browser telah menjadi bagian tak terpisahkan dari Windows 98, dirilis pada tahun 1998. Sistem ini memiliki bilah alat untuk peluncuran cepat aplikasi dan dukungan asli untuk USB. Sistem ini menjadi sangat populer, meskipun bukan yang paling stabil.



Windows 2000 adalah versi 5.0 dari sistem operasi Windows NT. Barisan ini diikuti oleh XP, Vista, Windows 7 dan 8. Versi beta publik pertama dari sistem muncul pada tahun 1997, perusahaan meluncurkan versi final pada 17 Februari 2000. Sistem mendukung bagian server dari Active Directory, sistem file NTFS 3.0, dan EFS terenkripsi, dan menerima antarmuka pengguna dengan Active Desktop.

Untuk menggunakan Windows 2000, Anda harus memiliki prosesor dari 133 MHz, dari 32 megabyte RAM dan dari 2 gigabytes memori pada hard drive Anda.



Dirilis pada tahun 2000, Windows ME dikritik oleh kedua kritikus dan pengguna sebagai sangat tidak stabil, berbicara tentang pembekuan konstan dan crash.



Komputer pertama yang saya beli menjalankan Windows XP. Saya mendapatkan komputer pada tahun 2006, dan Microsoft merilis XP pada tahun 2001. Windows XP menjadi sistem operasi Microsoft paling populer pada masanya.

Pada Juni 2008, Microsoft mengumumkan akan mendukung XP hingga 2014. Dukungan umum untuk sistem berakhir pada 2009, diperpanjang - pada 8 April 2014.



Pada tahun 2007, Microsoft merilis Windows Vista sebagai respons terhadap sistem operasi Mac OS X. Jendela yang tembus cahaya, ikon besar, dan menu Mulai yang baru tidak mengesankan kritik.



Sistem operasi server Windows Server 2008 menggantikan Windows Server 2003 sebagai perwakilan generasi baru sistem keluarga Vista.



Windows 7 adalah salah satu rilis Windows yang paling sukses. Meninggalkan antarmuka yang indah (Aero) dan ikon, Microsoft menghapus banyak bug dan membuat sistem lebih andal.

Enam tahun berlalu antara rilis Windows XP dan Windows Vista, dan hanya tiga tahun antara rilis Vista dan 7.



Pada 2012, Microsoft memperkenalkan Windows 8 dengan antarmuka Metro tiled. Antarmuka pertama kali muncul setelah memulai sistem dan mirip dengan desktop. Untungnya, sistem operasi terus mendukung desktop Windows klasik, tetapi menghapus tombol Start. Sistem ini dirancang untuk layar sentuh, meskipun Anda dapat menggunakannya pada komputer dengan monitor konvensional.



Windows 8.1 - Pembaruan untuk Windows 8 dari 2014. Sekarang pengguna saat startup dapat melihat layar yang berfungsi sebagai pengganti Metro, tombol "Start" muncul, meskipun tanpa menu mulai yang lama.



Pada 2015, Microsoft memperkenalkan Windows 10, yang dapat ditingkatkan dari Windows 7 dan Windows 8. Microsoft mengembalikan menu mulai.

Ada beberapa momen yang tidak menyenangkan bagi pengguna akhir: pada Agustus 2015, pelacak torrent mulai melarang pengguna dari Windows 10 di komputer, karena sistem mengirimkan jumlah data yang berlebihan ke server Microsoft.
