Microsoft telah merilis produk terbuka berdasarkan GNU / Linux


Microsoft benar-benar melakukannya. Kemarin, pada konferensi Open Compute Project (OCP) di San Jose, sebuah produk yang sepenuhnya terbuka diperkenalkan, yang disebut SONiC atau Perangkat Lunak untuk Jaringan Terbuka di Cloud . Produk ini dibuat dengan partisipasi dari produsen seperti Arista, Broadcom, Dell dan Mellanox. SONiC adalah seperangkat perangkat lunak dan utilitas yang dirancang untuk mengontrol perangkat jaringan tingkat L2 / L3 dari berbagai produsen. Saat ini, dasarnya adalah distribusi Debian dengan kernel Linux 3.16, tetapi seperti yang diklaim Microsoft, distribusi lain dengan versi kernel yang dimulai dengan 3.16 dapat digunakan.

SONiC dapat bekerja pada berbagai perangkat berkat spesifikasi SAI (Switch Abstraction Interface) yang diadopsi pada musim panas 2015. Proyek OCP. Penggagas proyek SAI juga Microsoft sendiri. SAI adalah API C standar untuk pemrograman sirkuit ASIC untuk peralatan jaringan. Proyek SAI juga merupakan open source dan tersedia di Github .

SONiC adalah dasar dari produk berbasis Microsoft GNU / Linux lainnya - Azure Cloud Switch (ACS) . ACS dikembangkan oleh perusahaan untuk penggunaan internal dan berisi perangkat lunak manajemen cloud Azure tambahan.

Microsoft mengklaim bahwa SONiC v1digunakan dalam produksi dan sudah ada di sakelar pusat data Microsoft yang ada. Mereka juga berjanji untuk memposting di Github Sonic v2 selama beberapa minggu mendatang , yang akan berisi banyak perbaikan.
