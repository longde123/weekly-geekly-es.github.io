发现MacBook Pro电池胶粘剂溶剂


苹果公司非常认真地确保用户自己或第三方主机不能以任何方式维修其设备。 这是旨在最大化利润的封闭生态系统的重要组成部分。 例如，锂离子电池几年后就失去了容量，一个人无法自行更换它们。 然后新的MacBook型号的展示就到了，此人决定购买一台新的笔记本电脑，因为旧的笔记本电脑不再装有电池。 因此，苹果公司正在支持销售。

苹果将​​电池粘在MacBook Pro笔记本电脑外壳上的强力胶已成为维修人员的噩梦。 在电子回收工厂中，工人用废金属撕掉这些电池，从而破坏了外壳和“视网膜”以及其他所有东西。

自2012年以来，新胶水开始用于MacBook Pro机型。 根据iFixit的工程师Brett Hartt的说法，这是一种功能强大的工业胶-与在摩天大楼，医院等中的玻璃面板胶合所用的胶差不多。

布雷特·哈特（Brett Hartt）是第一个设法拆卸2012 MacBook Pro的人。 最初，他尝试使用spudger-一种金属双刃刀片，通常用于维修和开放Apple技术。 但是有一次，一把锋利的仪器意外地割断了电池，有毒的烟雾开始从电池中散发出去，公司不得不撤离了建筑物。

然后，iFixit尝试了其他几种方法，试图分离或切断电池，但胶水没有渗入。 最后，工程师决定加热胶水。 这是一项非常艰苦的工作，几乎不建议在家中的外行人员进行。 事实是，当加热到很高的温度时，电池实际上会变成炸弹。 而且，如果您加热外壳另一侧的胶水，则可以轻松融化触摸板和键盘。 因此，分离电池是一个耗时，艰苦而危险的过程。

至少几年来就是这种情况，但是现在iFixit仍然发现了一种合适的丙酮基化合物，它可以与胶粘剂发生反应并使之溶解。

现在，您真的可以在家中自行分离电池，而不必冒生命危险。 IFixit启动了MacBook Pro电池拆卸套件，并发布了有关如何执行此操作的分步说明 。 这些说明适用于那些使用工业胶水且可以通过这种方式取出电池的Apple笔记本电脑（列表中未包括最新型号）：

  • MacBook Pro 13“ Retina显示屏2013年初
  • MacBook Pro 13“ Retina显示屏2015年初
  • MacBook Pro 13” Retina显示屏2012年末
  • MacBook Pro 13“ Retina显示屏2013年底
  • MacBook Pro 13“ Retina显示器2014年中
  • MacBook Pro 15” Retina显示屏2013年初
  • MacBook Pro 15“ Retina显示屏2013年底
  • MacBook Pro 15“ Retina显示屏2012年中
  • MacBook Pro 15“ Retina显示器2014年中
  • MacBook Pro 15“ Retina显示器2015年中

电池更换过程的时间估计为20分钟至3个小时，其中的说明包含32至73个步骤。 难度最高。 您将需要一套标准的工具来打开盒子（这也是一项艰巨的任务）：5位五角螺丝刀，一把螺丝刀，六尖头螺丝刀T5 Torx，T6 Torx和T7 Torx，镊子。

溶剂更换电池套件本身的价格从89.95美元到109.95美元不等，具体取决于笔记本电脑的型号。



套件中-新电池，所有必需的工具，溶剂，注射器，塑料卡，溶解胶水，橡胶手套和安全眼镜后可撬起电池。

说明说，要溶解胶水，只需在电池下倒入约1毫升溶剂即可，然后等待两到三分钟，直到胶水散布到整个周边。 然后用塑料卡撬起电池。 该过程从最右边的块开始。



然后与其余重复。



幸运的是，Apple仅将胶水涂在电池的周围，而不是整个区域。



取出旧电池后，可以使用溶剂去除身上的胶水。 丙酮基iFixit溶剂。 有人怀疑他的构成很简单，工程师很可能猜想过要使用丙酮，但由于某种原因，他们花了五年时间才这样做。

安装新电池后，您可以再使用一台出色的笔记本电脑两到三年。 显然，Apple不会简化维修人员的生活，因此在更换组件时使用特殊化学品会变得司空见惯，您需要习惯一下。
