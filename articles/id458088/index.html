Distribusi Linux Terbaik untuk Komputer yang Lebih Lama

Hai

Singkatnya: jangan jatuhkan komputer lama Anda - gunakan distribusi Linux yang ringan untuk menghidupkannya kembali!

Apa yang Anda lakukan dengan komputer lama Anda? Salah satu yang pernah memiliki konfigurasi perangkat keras yang baik tetapi sekarang dianggap usang. Mengapa tidak mengembalikannya dengan Linux? Saya akan daftar distribusi Linux ringan terbaik yang dapat Anda gunakan pada PC lama Anda.

Distribusi Linux ringan terbaik untuk laptop dan desktop lama


Saya akan mencoba untuk mengatur daftar dalam urutan persyaratan perangkat keras. Ini berarti bahwa distributor Linux yang ringan di tempat pertama akan memerlukan perangkat keras minimal.

10. Peppermint




Peppermint adalah distribusi Linux berbasis cloud yang tidak memerlukan perangkat keras yang kuat untuk dijalankan. Ini didasarkan pada Lubuntu dan menggunakan lingkungan desktop LXDE untuk operasi yang lebih lancar.

Peppermint OS hadir dengan ICE , aplikasi untuk mengintegrasikan situs web apa pun sebagai aplikasi desktop.

Persyaratan sistem minimum untuk Peppermint OS:
RAM: 1 GB RAM (disarankan 2 GB)
CPU: prosesor Intel x86
Ruang hard disk: Setidaknya 4 GB ruang disk kosong
Anda dapat mengunduh distribusinya di sini.

9. Lubuntu




Distribusi berikutnya dalam daftar kami adalah Lubuntu. Seperti namanya, Lubuntu didasarkan pada Ubuntu, tetapi menggunakan LXDE sebagai lingkungan desktop. Lubuntu adalah distribusi Linux yang paling ringan, berasal dari Ubuntu, jadi ia berspesialisasi dalam kecepatan dan dukungan untuk perangkat yang lebih tua.

Secara default, aplikasi jauh lebih sedikit dikirim dengan kit distribusi ini daripada dengan Ubuntu, dan mereka terutama terdiri dari program yang tidak menuntut sistem.



GPicView untuk melihat gambar, MTPaint untuk menggambar, Evince untuk file PDF, Audacious untuk musik, Gnome-Player untuk video, guvcview untuk webcam, Chromium sebagai browser web, Sylpheed untuk email, Pidgin untuk pesan instan, Transmission as Klien BitTorrent, Gnumeric untuk tabel, Abiword sebagai alternatif untuk Microsoft Word, Xpad untuk catatan, dll.

Jika sebelumnya Anda menggunakan Ubuntu, Lubuntu tidak akan terasa asing bagi Anda. Perangkat lunak dan repositori mereka sama, sehingga Anda dapat menginstal semua perangkat lunak yang Anda gunakan pada Ubuntu di Lubuntu. Tetapi jangan lupa tentang sistem Anda saat memasang aplikasi yang diperlukan. Jangan gunakan aplikasi intensif sumber daya.

Persyaratan sistem minimum untuk Lubuntu:
RAM: RAM 512 MB (disarankan 1 GB)
CPU: Pentium 4 atau Pentium M, atau AMD K8, atau lebih tinggi
Anda dapat mengunduh distribusinya di sini.

8. Linux Lite




Sesuai namanya, Linux Lite ("ringan") adalah distribusi Linux ringan yang tidak memerlukan perangkat keras berkinerja tinggi untuk dijalankan. Linux Lite didasarkan pada rilis Ubuntu LTS (Dukungan Jangka Panjang). LTS memberikan dukungan untuk distribusi selama 5 tahun, yaitu, setelah menginstal Linux Lite di komputer Anda, pembaruan akan dirilis selama 5 tahun, tanpa gagal.

Tim pengembang Linux Lite mengatakan: "Linux Lite benar-benar siap digunakan di luar kotak - artinya Anda tidak perlu menginstal perangkat lunak tambahan saat pertama kali menjalankan distribusi." Ini adalah berita yang sangat bagus untuk pemula, karena mereka tidak perlu menghabiskan waktu mencari dan menginstal aplikasi yang diperlukan. Sebagian besar aplikasi inti sudah dibangun ke dalam OS.

Berikut adalah beberapa aplikasi pra-instal: Firefox sebagai browser web, Thunderbird untuk email, Dropbox sebagai penyimpanan cloud, VLC Media Player untuk musik, LibreOffice sebagai alternatif untuk Microsoft Office, Gimp untuk mengedit gambar dan tweak Lite untuk menyesuaikan desktop Anda.

Mengingat distribusi ini didasarkan pada Ubuntu, di Internet Anda dapat menemukan banyak tutorial, tips, forum tematik, dan sejenisnya, jika Anda mengalami kesulitan.

Persyaratan sistem minimum untuk Linux Lite:
RAM: RAM 512 MB (disarankan 1 GB)
CPU: Prosesor 700 MHz
Monitor: layar VGA, resolusi 1024 × 768 (direkomendasikan VGA, layar DVI atau HDMI, resolusi 1366 × 768)
Ruang hard disk: Setidaknya 5 GB ruang disk kosong
Anda dapat mengunduh distribusinya di sini.

7. LXLE




LXLE adalah distribusi Linux berdasarkan versi terbaru dari Lubuntu LTS. Menggunakan shell grafis LXDE.

Terlepas dari kenyataan bahwa LXLE adalah distribusi yang ringan, ia mencoba untuk menyediakan UI (antarmuka pengguna) yang intuitif dan indah. Sistem ini dioptimalkan dengan sempurna dan memiliki kinerja yang baik, hadir dengan berbagai aplikasi ringan yang sudah diinstal sebelumnya secara default.

Persyaratan sistem minimum untuk LXLE:
RAM: 512 MB (disarankan 1 GB)
CPU: Pentium 3 (disarankan Pentium 4)
Ruang hard disk: 8 GB
Anda dapat mengunduh distribusinya di sini.

6. CrunchBang ++




CrunchBang ++ juga dikenal sebagai CBPP atau #! ++ atau CrunchBang Plus Plus. CrunchBang ++ adalah klon dari distribusi mati yang disebut CrunchBang Linux, dikenal karena kesederhanaan dan bobotnya yang ringan. CrunchBang ++ mendukung perangkat keras lama dan berfungsi tanpa masalah. Ini didasarkan pada Debian 9 dan menggunakan desain antarmuka minimalis. Ini juga didasarkan pada window manager Openbox .

Proyek ini melanjutkan tujuan yang sama dengan CrunchBang Linux: untuk menyediakan pengguna dengan distribusi Linux yang mudah digunakan dan ringan dengan fungsionalitas yang baik. Itu sebabnya Crunchbang ++ termasuk desain minimalis, antarmuka yang sederhana dan elegan.

Beberapa aplikasi standar: Geany IDE, emulator Terminal Terminator, manajer file Thunar, Gimp untuk mengedit gambar, Viewnior untuk melihat gambar, VLC Media Player untuk musik, Xfburn untuk membakar disk CD / DVD, Iceweasel sebagai browser, Transmisi sebagai BitTorrent- klien, Gnumerik untuk tabel, Evince untuk file PDF, gFTP - klien untuk mentransfer file, klien Xchat - IRC, AbiWord sebagai alternatif untuk Microsoft Word.

Persyaratan sistem minimum untuk CrunchBang ++:
Tidak ada persyaratan sistem resmi untuk CrunchBang ++. Idealnya, itu harus bekerja dengan 512 MB RAM dan prosesor Pentium 4 (CPU).
Openbox tidak sepenuhnya cocok untuk pemula, tetapi ini tidak berarti bahwa Anda harus takut untuk mencobanya.

Anda dapat mengunduh distribusinya di sini.

5. Bodhi Linux




Distribusi Linux ringan lainnya yang memberi kehidupan baru pada komputer lama adalah Bodhi Linux. Bodhi Linux terkenal karena ringannya. Bodhi Linux tidak memiliki banyak perangkat lunak pra-instal, sehingga ukuran file ISO kecil. Tetapi, dalam hal ini, Anda dapat menginstal aplikasi apa pun yang Anda perlukan menggunakan apt. Pada komputer lama, ini berfungsi dengan baik, memakan sedikit memori.

Selain itu, window manager Enlightenment membuat distribusi lebih cepat. Pencerahan lebih cepat daripada pengelola jendela lainnya. Jika Anda memiliki masalah dalam menggunakan Pencerahan atau fitur Bodhi lainnya, Anda dapat membaca tutorial yang ditulis oleh tim pengembang.

Bodhi Linux memiliki aplikasi dasar seperti Ephoto untuk grafik, Midori sebagai browser web, editor teks ePad, tetapi sayangnya tidak ada aplikasi multimedia. Tapi jangan khawatir! Seperti yang saya katakan, Anda dapat menginstal aplikasi lain melalui apt. Repositori Ubuntu juga tersedia di Bodhi Linux.

Persyaratan sistem minimum untuk Bodhi Linux:
RAM: RAM 256 MB
CPU: 1,0 GHz
Ruang hard disk: 4 GB ruang disk kosong
Anda dapat mengunduh distribusinya di sini.

4. antiX Linux



antiX adalah distribusi Linux ringan berdasarkan Debian. Terkenal karena fakta bahwa sistem tidak menggunakan "systemd". Jika Anda tidak terbiasa dengan istilah "systemd", maka Anda seharusnya tidak peduli bahwa antiX tidak menggunakan "systemd".

Alih-alih lingkungan desktop grafis biasa, antiX menggunakan icewm window manager, sehingga sistem dapat bekerja tanpa masalah bahkan pada peralatan terlemah. antiX dapat digunakan sebagai sistem LiveCD / LiveUSB cepat untuk pemulihan komputer.

Persyaratan sistem minimum untuk antiX Linux:
RAM: RAM 256 MB
CPU: prosesor PIII
Ruang hard disk: 2,7 GB ruang disk kosong
Anda dapat mengunduh distribusinya di sini.

3. SparkyLinux




Tempat ketiga dalam daftar kami adalah SparkyLinux. SparkyLinux adalah distribusi Linux ringan lainnya yang, selain komputer lama, menargetkan perangkat keras modern. Itulah sebabnya distribusi ini memiliki dua versi, salah satunya hadir dengan semua alat yang diinstal, codec, plugins, dll., Sehingga sistem siap bekerja pada permulaan pertama.

SparkyLinux didasarkan pada Debian dan memiliki beberapa opsi untuk lingkungan desktop grafis, termasuk LXDE, OpenBox / JWM, e17, MATE, LXQt, Cli, dan edisi GameOver. LXQt jauh lebih cepat daripada lingkungan lain yang disebutkan sebelumnya kecuali Cli (Command Line Interface).

Persyaratan sistem minimum untuk SparkyLinux:
RAM: 256 MB RAM untuk LXDE, e17, Openbox, GameOver dan 384 MB RAM untuk MATE, LXQt
CPU: i486 / amd64
Ruang hard disk: 5 GB ruang disk kosong
Anda dapat mengunduh distribusinya di sini.

2. Puppy Linux




Rilis terbaru dari Puppy Linux adalah Puppy Linux 6.0.2 Tahrpup CE, yang hanya berukuran 199–201 MB (!). Distribusi ini dapat diunduh dari drive USB yang sangat kecil.

Puppy Linux dapat di-boot dari Live CD / DVD / USB, dan setelah boot Anda dapat mengeluarkan CD / DVD / USB dan Puppy Linux akan terus bekerja dengan lancar. Anak anjing dimuat dan dijalankan dari RAM, yang membuatnya sangat cepat. Antara lain, Anda dapat menyimpan data apa pun di USB yang sama tempat Anda mem-boot Puppy Linux.

Secara default, Puppy Linux menggunakan JWM dan window manager Openbox, yang secara signifikan mengurangi konsumsi sumber daya. Puppy Linux berisi satu set aplikasi dasar saja. Misalnya, Gnumeric untuk tabel, Abiword sebagai alternatif untuk Microsoft Word, dan berbagai program untuk mengedit grafik dan memutar multimedia.

Persyaratan sistem minimum untuk Puppy Linux:
RAM: 64 MB (disarankan 256 MB)
CPU: 333 MHz
Anda dapat mengunduh distribusinya di sini.

1. Tiny Core




Bersiaplah untuk mengalami kegembiraan Tiny Core. Saya yakin aplikasi pengeditan gambar terkecil di Windows akan lebih sulit daripada Tiny Core Linux. Ya, beratnya hanya 16 MB (!!). Mengunduh video kecil dan berkualitas rendah bahkan lebih lama daripada mengunduh Tiny Core Linux.

Sistem ini hanya hadir dengan program yang paling diperlukan dan antarmuka minimalis. Jika Anda masih memiliki komputer yang sangat lama, instal Tiny Core dan berkenalan dengan keajaiban kecil ini. Ini memuat lebih cepat daripada OS lain yang dikenal.



Tiny Core Linux didasarkan pada distribusi Damn Small, tetapi saat ini sepenuhnya independen. Secara default, Tiny Core hadir dengan GUI FLTK / FKWM dan BusyBox. Anda akan kehilangan banyak hal, tetapi jangan khawatir, jika mau, Anda dapat menginstal semua ini secara manual.

Ada tiga versi Tiny Core: Core, Tiny Core, dan CorePlus. Versi pertama hanya berisi CLI (Command Line Interface) dan hanya pengguna berpengalaman / mahir yang dapat bekerja dengannya. Versi Core saat ini hanya berbobot 9 MB.

Tiny Core adalah untuk pengguna biasa yang akrab dengan GUI (Graphical User Interface). Versi TinyCore saat ini adalah 16 MB.

Core Plus adalah gambar instalasi, bukan distribusi. Disarankan untuk pengguna baru yang hanya memiliki akses ke jaringan nirkabel atau mereka yang menggunakan tata letak keyboard non-Inggris. Distribusi mencakup Sistem Inti dasar dengan utilitas untuk mengonfigurasi sistem: pilihan 7 manajer jendela, dukungan nirkabel melalui berbagai file firmware dan ndlswrapper, dukungan untuk tata letak non-Inggris dan alat remastering. Ukuran versi CorePlus saat ini hanya 72 MB.

Persyaratan sistem minimum untuk Tiny Core Linux:
RAM: 46 MB (disarankan 64 MB)
CPU: Pentium 2
Anda dapat mengunduh distribusinya di sini.
