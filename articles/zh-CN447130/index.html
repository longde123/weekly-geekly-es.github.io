PHP内部


  • [RFC] JIT-该提议已被PHP 8.0接受，但是在PHP 7.4中以实验模式启用该选项的想法没有得到支持。
    该主题是Joe Watkins的精彩文章，内容涉及一般的JIT及其在PHP中的实现 。
  • [RFC]更改串联运算符的优先级 -当前为串联运算符. 和数学+ ， -具有相同的优先级，并且从左到右执行。 建议降低连接的优先级：
    隐藏文字 
     echo "sum: " . $a + $b; //   echo ("sum: " . $a) + $b; //   echo "sum :" . ($a + $b);
  • [RFC] Nullable Casting-强制转换为基本类型时，建议使其能够强制转换为null，即除了(int)$x之外，还可以使用(?int)$x (int)$x 。
  • [RFC]数组表达式中的扩展运算符 -再次建议为数组添加一个运算符... 更新的产品取消了对具有非数字键的数组的支持。
    隐藏文字 
     $parts = ['apple', 'pear']; $fruits = ['banana', 'orange', ...$parts, 'watermelon']; // ['banana', 'orange', 'apple', 'pear', 'watermelon'];
