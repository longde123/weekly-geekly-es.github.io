哈勃望远镜在100亿光年的距离内看到了一个蓝色的超级巨人。 这是观测历史上最遥远的恒星。


这组图像显示了星系的引力透镜簇，通过该星团发现了一颗新恒星。 右下角的照片中，蓝色超级巨人（代号为Icarus）标有白色箭头。 图片：NASA / ESA / P。 凯莉

天文学家一直在使用引力微透镜来“增加”望远镜的开孔率几千或数千倍，并观察时空距我们数十亿光年的超长空间。 以这种方式反复发现超长星系，有时甚至发现单个恒星。 但是，现在发现的代号为“伊卡洛斯”的恒星比任何先前观测到的恒星都远100倍 ，超新星爆炸除外。

一个国际研究小组发布的证据表明 ，通过引力透镜星系团发现的物体是距我们近100亿光年的蓝色超巨星。 也就是说，哈勃记录了仅在大爆炸之后44亿年的恒星发出的光。

重力微透镜是爱因斯坦的相对论一般理论所预测的效果。 当大质量的物体（行星，恒星，星系，星系团，暗物质团）随着重力场改变其电磁辐射的传播方向时，就会发生这种情况，就像普通的镜头会改变光束的方向一样。 事实证明，这是一种镜头的类似物，通过它只能看到远处的物体。

在这种情况下，星系团就是引力透镜。 它在地球和远处的背景物体之间随机徘徊，通常可提供约50倍的额外放大倍率。 如果在一个幸运的巧合中，在一个透镜状星系团中，有一个较小的，完美对准的物体，那么背景可以增加5,000倍。


通常，发现是偶然发现的。 哈勃通过观察超新星“ Refsdal”发现了伊卡洛斯。 根据计算，超新星发出的光应该很快就被MACS J1149银河星团所透镜化，该星团位于相距约50亿光年的位置。 但是，尽管超新星还没有出现在视野中，但是在观察过程中，天文学家惊讶地发现了同一领域的新光源。 恒星的光度逐渐增加。


图片：NASA / ESA / P。 凯莉

研究人员使用哈勃望远镜测量了恒星的光谱-并注意到尽管亮度增加，恒星的温度仍保持不变。 这意味着太空望远镜注意到的不是再像Refsdal这样的超新星，而是某种稳定的恒星。 此外，它发出的光不仅穿过银河星团MACS J1149的透镜，而且还被星团内部的一些小而重的物体弯曲。 结果，以大于2000的系数进行微透镜。



“我们知道微透镜是由恒星，中子星或恒星质量的黑洞引起的，”南卡罗来纳大学的史蒂芬·罗德尼在新闻稿中说 。 因此，伊卡洛斯的发现将使天文学家对银河团本身的组成有新的认识。 鉴于星系团是我们宇宙中最庞大和扩展的结构，研究其组成将不可避免地有助于增进我们对宇宙的整体理解。

科学家认为，开放恒星伊卡洛斯（MACS J1149 Lensed Star 1）将有助于更好地了解宇宙中最神秘的物质之一-暗物质。 至少淘汰了一些最奇特的理论。 明尼苏达大学天体物理学家，科学著作的主要作者帕特里克·凯利说：“如果暗物质至少部分地由最近提出的质量相对较低的黑洞组成，那么我们将在伊卡洛斯的光曲率中看到它。” “我们的观察结果并未证实大多数暗物质是由质量约为太阳质量30倍的此类黑洞构成的可能性。”

该科学文章于2018年4月2日发表在《 自然天文学 》杂志上 （doi：10.1038 / s41550-018-0430-3）。
