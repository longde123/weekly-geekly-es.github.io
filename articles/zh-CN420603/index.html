特斯拉Model S车主的统计数据

youtube频道的作者Like Tesla （拥有Tesla Model 3，Tesla Model X）分享了与她妈妈的车有关的有趣统计信息。

她的母亲决定出售她拥有的5年的2013 Tesla Model S 85 （要价39,900美元）。



汽车可以在中央屏幕上访问里程，充电等数据：



允许获得整个所有权期间的统计信息的内容。

  • 覆盖74.960英里 （120.636公里）
  • 消耗25.820 kWh

她从不给特斯拉增压器充电（她行驶的路程不超过200英里（320公里）），并且总是在车库中为汽车充电（尽管增压器是免费的）。 即 她花了0个小时进行充电。

  • 她为这5年的燃料（电力）支付了约2,000美元 （每千瓦时0.08美元的价格）
  • 液体燃料的成本约为14,000-15,000美元 （每加仑3.75美元）
  • 对于ICE，这些加油站有214个加油站（18加仑油箱（68升）
  • 对于ICE，这大约需要在加油站花费25个小时 （如果您每个时间都算7分钟）



电池老化？



  • 现在，充满电的车辆的行驶里程为258英里 （415公里）
  • 充满电的护照里程265公里 （426公里）
  • 5年内退化约3％ （里程120.636公里）

她居住的地方（美国华盛顿州西雅图） 90％的电力来自水力发电站。

她之所以出售特斯拉Model S，是因为她决定购买新车。 在新的Tesla Model S（$ 88,000）和Tesla Model 3 AWD（$ 64,000）之间选择。 结果，我选择并订购了Model 3-尺寸和它的技术更适合她，此外价格也有很大差异。
