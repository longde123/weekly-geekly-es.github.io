在流行的操作系统中“连接”了哪种音乐

现状：在院子里2003年。 您家人的某人正在通话，这是调制解调器所连接的线路。 考虑到轮到您并有机会上线，您决定再次钻研开发人员在Windows中放置的免费音乐库。 当时，她在“ 音乐样本”文件夹中。

今天，我们决定回忆起在不同版本的Windows中可以收听的内容。


照片这家伙 Unspalsh

Windows 3.1


音乐出现在90年代引入的Windows 3.1中。 大约在这个时候，首批启用MIDI的声卡进入市场。 使用这种格式的软件也已经开始出现，例如Steinberg Cubase和Finale 。

Windows 3.1包括C：\ WINDOWS \ MEDIA文件夹中的九首MIDI歌曲。 您可以在Archive.org上聆听它们。

巴赫（Bach）和维瓦尔第（Vivaldi）的经典作品摘录了两条曲目：“ 勃兰登堡音乐会 ”和“ 四季 ”。 另外两项作品展示了Passport Designs MIDI音序器的功能-PASSPORT.MID和CANYON.MID。

众所周知，第一个作曲家并不 知名 ，但第二个作曲家是爵士音乐家乔治·斯通（George Stone）。 但是他不认为音乐会与Windows一起分发（他编写音乐是为了在MAC上播发音序器）。 Passport Designs遇到了财务困难，在没有Stone知识的情况下将Microsoft的著作权出售给了Microsoft。 乔治没有从这笔交易中收到钱，但是他的工作得到了广泛认可。

YouTube上有大量的CANYON.MID 封面 。 该作曲甚至在MAGFest节日之一中进行，该节日专门致力于计算机游戏及其音乐。


Windows 95


在这里， CLOUDS.MID已添加到前面的9条轨道中。 它被认为是“复活节彩蛋”，因此它不在MEDIA目录中。 要听作曲，有必要对文件夹进行一系列操作。 该说明位于作曲家CLOUDS的博客上-布莱恩·奥尔（Brian Orr），当时他是Microsoft的一名实习生。

除了新的MIDI合成外，Windows 95中还出现了两个音乐视频。 该事件是在操作系统随附的免费版本Video for Windows Player发行时进行的。 第一个录像带是美国歌手艾迪•布里克尔（Edie Brickell）录制的歌曲“好时光”的片段。

第二个是Weezer专辑中歌曲“ Buddy Holly”的剪辑 。 作为情景喜剧“ 快乐日”中的一幕，该剪辑获得了四个 MTV 奖项 ，包括最佳另类剪辑的称号。 如今，歌曲Buddy Holly被认为是90年代最知名的作品之一，并且仍在制作封面 。

Windows 98


在Windows 98中，所有MIDI音轨都被替换为三个新文件。 正如微软代表所说，他们参与了一些乐器。

Microsoft技术支持通常将这三种曲调用作诊断多媒体扩展问题的工具。 在他们的帮助下，我们检查了声音驱动程序的可操作性。


新的曲调称为FLOURISH.MID，TOWN.MID和ONESTOP.MID。 前两首歌的作者是内森·格里格（Nathan Grigg），他现在因在《恐惧与中土》系列游戏中的配乐而闻名。 第三首歌是由大卫·雅克利（David Yackley）创作的，微软最糟糕的MIDI音乐文件之一也因此声名远播。

Windows ME和XP


Windows ME在IT社区中没有很好的声誉，但是该操作系统因美国多乐器演奏家Beck Hensen的曲目“ Beautiful Way ”而闻名。 该作品是Midnite Vultures专辑的一部分，受到评论家和乐迷的一致好评。

新曲目展示了ASF多媒体格式的功能，其中包含流音频和视频。 通常，这些文件的扩展名为.asf，但是如果该容器仅包含音乐，则将其指定为.wma。 正是以这种格式在Windows XP中显示了合成。


照片Simon Goldin CC BY-SA /照片“ Serenity” Windows XP，摄于2006年
Windows XP在Windows ME推出仅一年后就成为了首个包含著名的Sample Music目录的操作系统。 它包含了摇滚乐队Talking Heads的最新专辑中的乐曲，还有贝多芬第九交响曲的scherzo和Mark Sils的一个轻快的爵士乐编号“ Highway Blues ”。

Windows Vista


在Windows Vista中，系统附带的音乐曲目数量达到11条。 其中包括意大利打击乐二重奏Aisha Duo的作品。 选择包括他们的专辑《 Quiet Songs》中的两 首歌曲。 样本文件夹中还有巴西爵士歌手Luciana Sousa的音乐“ Mutia Bobiera ”和纽约钢琴家Aaron Goldberg的歌曲“ MAO's Blues ”。

塞内加尔吉他手Habib Koite发行了他的首张专辑中的两首歌曲，其中结合了西方吉他音乐和马里人民的传统音乐。 同样在Vista中，可能会遇到美国打击乐手Kurs Kale的作品，他被认为是亚洲地下舞蹈流派的先驱。 现在您可以在Bandcamp上听他的作品。


The Posies歌曲稀释了民族音乐和爵士乐。 乐队刚刚发行了八年来的第一张专辑。 团队可能想利用新操作系统的输出作为自我提醒的机会。

Windows 7以上


Windows 7引入了克劳德·德彪西（Claude Debussy）的作品“ 亚麻长发女孩 ”，鲍勃·阿克雷（Bob Acre）创作的“ 沉睡 ”以及他的电子曲目 。 Scruff ，一位英国节奏慢的艺术家。


从Windows 8开始，Microsoft 停止打开选定艺术家的音乐以及操作系统，并且带有“样本”的文件夹从操作系统中消失了。 部分原因是由于Microsoft开设了Groove Music数字音乐商店。 一种或另一种方式，样本音乐文件夹的内容已经过去了整个时代。


Audiomania将举办Hot AUDIOSALE 2019的传统夏季大减价。 直到6月底，耳机，便携式扬声器，户外扬声器和电子管放大器的折扣价都可高达70％ 。

这是一个为亲人购买礼物或购买您长期关注的设备的好机会。
