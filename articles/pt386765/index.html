Windows tem 30 anos


Bill Gates apresenta um novo sistema (Windows 3.0, neste caso) O

tempo é muito rápido e é especialmente bom acompanhar o progresso do tempo usando vários dispositivos e plataformas de software como exemplos. Exatamente 30 anos atrás, a Microsoft lançou a primeira versão do Windows como sucessora do MS-DOS. Aconteceu em 20 de novembro de 1985. O lançamento deste sistema operacional marcou o fim de uma era e o começo de outra. Apesar de mais de 30 anos a interface do sistema operacional Windows ter mudado muito, muitos elementos permaneceram desde a primeira versão. São caixas de diálogo, barras de rolagem, ícones, aplicativos como o bloco de notas e o Paint.

Além disso, o Windows 1.0 fez amizade com o mouse. O MS-DOS precisava digitar comandos e nem todos os usuários lidaram com isso. Como se fosse um sistema não inteiramente para donas de casa. Depois que o sistema operacional da Microsoft começou a suportar o mouse, a maneira como as pessoas e o PC interagiram mudou radicalmente. A princípio, muitos usuários do novo sistema até reclamaram que tinham que trabalhar demais com o mouse, em vez de digitar comandos do teclado. Mas então eles se acostumaram.

O primeiro sistema operacional Windows em 1985 exigiu 256 KB de memória, uma placa de vídeo e dois disquetes. Se o usuário quisesse executar vários programas simultaneamente, também seria necessário adicionar um disco rígido com 512 KB de memória. Essas especificações básicas foram apenas o começo, após o qual o rápido desenvolvimento de PCs começou, com o desenvolvimento simultâneo de plataformas de software.
