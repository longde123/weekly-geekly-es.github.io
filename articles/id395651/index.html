Akhir yang Layak: Pengembang Menghancurkan Dunia Planetside oleh Meteor Bombing

Server game terputus, setelah mengatur kiamat sebelum ini.Pada



tahun 2003, game MMO Planetside diperkenalkan. Pada saat itu itu adalah proyek multi-pengguna yang menarik, berhasil bersaing dengan game MMO lainnya. Seiring waktu, MMO menjadi lebih banyak mainan, dan Planetside semakin sulit untuk bersaing dengan proyek game lainnya. Para pemain berangsur-angsur pergi, dan satu dekade kemudian, para pengembang memutuskan untuk menghentikan pengembangan aktif dunia game. Tim lama melemparkan semua kekuatan mereka ke dalam pengembangan bagian kedua mainan, Planetside 2.

Pada 2014, versi pertama beralih ke model free-to-play, tetapi dana yang diterima dari para pemain masih tidak cukup. Bahkan, pendapatan bahkan tidak menutupi biaya pemeliharaan server. Untuk saat ini, server Planetside tidak mati, dan banyak pengguna terus menjelajahi dunia game, yang tidak terlalu kecil. Tapi semuanya berakhir, IMO ini tidak terkecuali. Studio Daybreak Games mengumumkan penutupan server permainan, mencatat bahwa keputusan untuk menyelesaikan proyek dibuat untuk pengembangan lebih lanjut dari bisnis perusahaan dan karena sejumlah keterbatasan teknis dari mainan itu sendiri.

Pada bulan Juli tahun ini, server memutuskan untuk memutuskan, terlepas dari kenyataan bahwa pada bulan April dan Mei, pengembang menambahkan beberapa elemen baru di dunia game. Tim proyek memutuskan tidak hanya mematikan semuanya sekaligus, tetapi untuk mengatur kiamat yang spektakuler. Apa yang dilakukan: sebelum mematikan server, dunia virtual dihancurkan oleh pengeboman meteorit.



Dunia Planetside terdiri dari puluhan benua, yang masing-masing terlibat dalam perang yang sedang berlangsung. Permainan ini menggunakan alat berat dan pesawat terbang, senjata paling fantastis dan bahkan pemboman orbital. Pemain dapat mengembangkan karakter mereka sebagai seorang prajurit infanteri atau penembak jitu, peretas atau petugas medis, pilot atau pengemudi - hanya sekitar 30 spesialisasi.



Planetside adalah salah satu dari banyak proyek game bagus yang telah dinonaktifkan karena kehilangan modalnya. Kadang-kadang penggemar game meluncurkan server mereka sendiri, seperti halnya versi asli WoW dan server Nostalrius. Namun para pengembang tidak menyambut manifestasi nostalgia semacam itu. Server WoW yang sama ditutup secara hukum atas permintaan Blizzard. Menurut perusahaan, kegiatan tim proyek jelas melanggar hak cipta perusahaan. Tim Nostalrius sebelumnya berencana untuk memposting kode sumber dan pangkalan pemain di jaringan untuk memungkinkan penggemar WoW untuk terus mengerjakan proyek. Sekarang keputusan ini telah dibalik .

Sedangkan untuk Planetside, sejauh ini tidak ada server pihak ketiga dengan mainan ini di jaringan.
