48个JavaScript开源资源（2019）



在20,000个JavaScript项目和库中，我们选择了48个最佳项目（按GitHub上的星标），并分为5类：

  • 用户界面（1〜9）
  • JavaScript工具（10〜25）
  • 机器学习（26〜33）
  • 研究计画（34〜39）
  • 开发人员工具（40〜48）


本文是在EDISON Software的支持下编写的，EDISON Software 开发了电子体检系统并支持该软件 。


使用者介面


  1. 粗v2.0 ：创建手绘草图图形。
  2. Driver.js ：一个轻巧，独立的原始 JavaScript引擎，可将用户的注意力集中在页面上。
  3. Sweetalert2 ：美丽，响应迅速 ，可自定义且负担得起的（WAI-ARIA）JavaScript弹出窗口替换。
  4. Tippyjs ：轻量级的香草JS工具提示库。
  5. Spritejs ：用于2D渲染的跨平台轻量级对象模型。
  6. AutoComplete.js ：用于自动完成的简单Javascript库。
  7. Glider.js ：一种快速简便的滚动轮播替代方案。
  8. LayerJS ：Javascript UI / UX库，可让您为类似于移动应用程序的Web应用程序和网站创建直观，吸引人的应用程序。
  9. Pencil.js ：用于交互式2D绘图的库。

    JavaScript工具

  10. Johnny-Five v1.0 ：JavaScript机器人技术和IoT编程环境。
  11. Babylon.js ：用于在HTML5和WebGL中创建3D游戏的JavaScript框架。
  12. Pollyjs ：记录，播放和存根HTTP交互。
  13. Omi ：下一代4kb JavaScript Web框架（Web组件+ JSX +代理+存储+路径更新）。
  14. 刺激 ：您已经拥有的不起眼的HTML HTML框架。
  15. Apexcharts.js ：一个现代的JavaScript图库，用于使用简单的API创建交互式图和可视化文件。
  16. Greenlet ：将异步函数移到其自己的线程中。
  17. JSUI ：用于管理JavaScript应用程序的功能强大的UI工具。
  18. G6 ：图形可视化框架。
  19. Workerize ：在web-worker'e中运行模块。
  20. V8n ：可定制， 可重用的验证库。
  21. ForgJs ：轻量级的JavaScript对象验证器。
  22. Minipack ：用JavaScript编写的现代打包程序的简化示例。
  23. 列表v2 ：具有无与伦比的性能和全面功能API的一致列表。
  24. JavaScript Object Explorer ：无需研究文档即可找到所需的对象方法。
  25. Awaity.js ：是bluebird.js的一种功能轻巧的替代品，它的创建考虑了异步/等待。

    机器学习

  26. Tfjs ：加速的WebGL，一个基于JavaScript浏览器的库，用于训练和部署ML模型。
  27. Guess.js ：一个工具包，用于将数据驱动的用户体验整合到Web上。
  28. Picojs ：200线人脸识别库。
  29. Tensorspace ：用于神经网络的三维可视化的基础设施，在浏览器中创建交互式直观的模型，支持TensorFlow，Keras，TensorFlow.js的预训练深度学习模型
  30. Nlp.js ：内置于Natural节点上的NLP库，具有实体提取，情绪分析，自动语言识别等功能。
  31. Magenta-js ：在浏览器中使用机器智能生成音乐和图片。
  32. JeelizWeboji ：JavaScript / WebGL库，用于实时人脸识别和表情检测。
  33. ASLT ：使用Tensorflow.js教Alexa回答手语

    专案

  34. Tui.calendar v1.0 ：日历。
  35. Webamp ：HTML 5和Javascript中的Winamp 2.9克隆。
  36. Filepond ：文件上传库
  37. OpenSC2K ：SimCity 2000的开源翻版。
  38. Jsnes ：NES仿真器。
  39. JSpaint ：复兴的经典绘画。

    开发人员工具

  40. Carbon v3.0 ：创建并共享源代码的精美图片
  41. Jarvis ：基于浏览器的非常智能的Webpack工具栏
  42. Polacode ：拍立得为您的代码
  43. Htm（超文本标记的标记） ：JSX替代，使用带有编译器支持的标准标记模板。
  44. Rete ：用于可视化编程和创建节点编辑器的JavaScript框架
  45. Size-plugin ：跟踪Webpack资源随时间推移的压缩大小。
  46. Webpackbar ：用于Webpack 3和4的优雅ProgressBar和Profiler
  47. Worker-plugin ：向Webpack添加内置的Web Worker支持。
  48. Subliminal ：JavaScript的简约VS Code主题
