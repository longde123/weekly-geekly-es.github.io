VS Code - Extraits de marionnettiste

Cette extension de code Visual Studio ajoute des extraits de code utiles prédéfinis pour Puppeteer. Consultez l' API pour plus d'informations sur les options disponibles.



‍ Comment utiliser?


Installez d'abord l' extension .


En fait, il existe deux façons d'utiliser des extraits de code:


1. Saisie directe


Commencez à taper le préfixe ou juste une partie du fragment. Recherchez ensuite les fragments dans la liste des phrases et appuyez sur TAB ou ENTER .


2. Palette de commandes


Ouvrez la palette de commandes et tapez Insert Snippet . Sélectionnez ensuite l'extrait en surbrillance.
