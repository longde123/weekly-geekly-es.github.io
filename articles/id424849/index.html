Apa yang membuat UCS C480 ML M5 baru menarik - server pembelajaran mesin dari Cisco

Pada bulan Maret tahun ini, Cisco memperkenalkan Cisco UCS B480 M5 Server Blade. Berdasarkan sistem ini, kami telah mengerahkan kumpulan sumber daya baru di St. Petersburg dan sedang bersiap untuk meluncurkan kumpulan serupa di Moskow. Tetapi hanya enam bulan kemudian (pada bulan September), raksasa TI ini merilis produk baru lainnya - UCS C480 ML M5, "diasah" untuk bekerja dengan sistem kecerdasan buatan. Kami memberi tahu apa yang ada di dalamnya dan mengapa semuanya ke pusat data.


/ Wikimedia / Tibigc / CC

Apa yang ada di dalamnya


Pengembangan baru perusahaan adalah versi modifikasi dari server C480 M5 . Untuk mempercepat pemrosesan algoritma pembelajaran mesin, dua slot untuk CPU di server itu diganti dengan slot untuk GPU. NVIDIA's Tesla V100-32G akselerator grafis ditempatkan di slot ini.

Jumlah kartu mencapai delapan. Kinerja masing-masing akselerator adalah 47 kali lebih tinggi daripada kinerja prosesor tradisional untuk tugas pembelajaran yang mendalam. Data dipertukarkan melalui antarmuka NVLink bawaan - ini adalah 200 GB / s per GPU.

C480 ML M5 juga memiliki dua prosesor Intel Xeon, yang masing-masing mendukung hingga 28 core, dan 3 TB RAM. Pada saat yang sama, hingga 24 HDD atau SSD dapat dihubungkan ke server melalui antarmuka SAS atau SATA.

Mengapa saya perlu server baru


Menurut perwakilan Cisco, server harus menyederhanakan pekerjaan dengan model pembelajaran mesin di semua tahap pengembangan mereka: dari mengumpulkan dan menganalisis data hingga menghasilkan hasil pemrosesan. Pada saat yang sama, sangat cocok untuk bekerja dengan aplikasi pipelined di lingkungan multi-cloud. Ini adalah kerangka kerja Hadoop dan Spark, serta TensorFlow dan PyTorch. Selain itu, Cisco sedang menegosiasikan kemitraan dengan platform Anaconda, dan juga bekerja sama dengan Kubeflow bekerja sama dengan Google.

"ML M5 baru akan menyederhanakan pekerjaan dengan sistem kecerdasan buatan untuk penyedia cloud dan perusahaan yang memiliki pusat data," komentar Sergey Belkin, kepala pengembangan di 1cloud . - Teknologi pembelajaran mesin dapat digunakan untuk mengevaluasi semua proses yang terjadi di pusat data. Aplikasi semacam itu sudah ada di ekosistem Cisco. Misalnya, Tetration Analytics mengumpulkan data telemetri dari sensor perangkat lunak dan perangkat keras dan memprosesnya menggunakan metode MO. "

Cisco menyarankan bahwa pada 2019, 70% serangan hacker akan menggunakan enkripsi lalu lintas, oleh karena itu, menurut pendapat mereka, kemampuan tambahan dari server baru juga akan memainkan peran penting dalam melindungi infrastruktur TI pusat data. Untuk tujuan ini, raksasa IT juga memiliki solusi sendiri: Enkripsi Lalu Lintas yang Dienkripsi. Sistem mengidentifikasi kerentanan dalam infrastruktur dan mendeteksi pola berbahaya bahkan dalam lalu lintas terenkripsi.


/ Wikimedia / Titik Akses Global / PD

Perhatikan bahwa rilis C480 ML M5 akan memengaruhi pekerjaan organisasi yang memproses sejumlah besar data - misalnya, medis dan keuangan. Akan lebih mudah bagi analis dan pengembang dalam organisasi ini untuk mengekstrak informasi yang berguna dari Big Data. Server Cisco baru akan mempercepat proses pemrosesan set besar informasi dan mencari pola (misalnya, ketika menganalisis gambar medis).

Menurut Cisco, 80% bisnis global telah mengimplementasikan layanan berbasis pada teknologi kecerdasan buatan atau akan mengimplementasikan solusi tersebut pada tahun 2020. Dan pada tahun 2035, teknologi MO akan membantu meningkatkan produktivitas organisasi sebesar 40%.

Prospek dan Rencana untuk Masa Depan


Chirag Dekate, direktur penelitian di Gartner, percaya bahwa C480 ML M5 akan diminati oleh pelanggan lama dan baru. Pada saat yang sama, Scott Mohr, Direktur Hubungan Mitra, mengatakan bahwa Cisco akan terus meningkatkan solusinya untuk layanan berdasarkan jaringan saraf, dan pengalaman pelanggan akan memainkan peran penting dalam proses ini. Karena itu, sekarang produk baru sedang diuji oleh lima puluh mitra Cisco.

Nantinya, sistem akan tersedia untuk semua mitra bersertifikat perusahaan. Meskipun tes yang sedang berlangsung, masih belum memungkinkan untuk mengatakan apa pun tentang efektivitas server baru. Pakar Cisco (tidak seperti NetApp yang sama) memutuskan untuk tidak memberikan hasil tes kinerja Resnet atau Alexnet, karena, menurut mereka, mereka tidak mencerminkan keadaan ketika bekerja pada proyek pembelajaran mesin nyata.

Pesanan C480 ML M5 akan tersedia pada akhir tahun ini.
