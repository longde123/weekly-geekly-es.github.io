Microsoft Azure Security Center Mengumumkan Fitur Baru

Semakin banyak organisasi berinovasi lebih cepat dengan memindahkan perusahaan mereka ke cloud, peningkatan keamanan menjadi penting bagi setiap industri. Azure memiliki alat manajemen keamanan bawaan untuk data, aplikasi, komputasi, jaringan, identifikasi dan perlindungan ancaman, yang memungkinkan Anda untuk mengonfigurasi keamanan dan mengintegrasikan solusi mitra.

Kami terus berinvestasi dalam keamanan, dan kami senang berbagi pembaruan keren yang diumumkan minggu lalu di Hannover Messe 2019. Ini termasuk Perlindungan Ancaman Tingkat Lanjut untuk Penyimpanan Azure, Dasbor Kepatuhan, dan Dukungan untuk Set Skala Mesin Virtual yang Dapat diskala ) (VMSS). Daftar lengkap di bawah kucing.



Fitur-fitur berikut yang diumumkan di Hannover Messe 2019 sekarang tersedia untuk Azure Security Center:

  • Perlindungan Ancaman Azure Vault Tingkat Lanjut - Tingkat perlindungan yang membantu pelanggan mengidentifikasi dan merespons potensi ancaman di akun brankas mereka saat mereka muncul - tanpa harus menjadi pakar keamanan.
  • Dasbor Kepatuhan terhadap Peraturan - Membantu pelanggan Pusat Keamanan mengoptimalkan proses kepatuhan mereka dengan memberikan informasi tentang status kepatuhan mereka untuk serangkaian standar dan peraturan yang didukung.
  • Dukungan Set Mesin Skala Virtual (VMSS) - Mudah memantau status keamanan VMSS Anda dengan rekomendasi keamanan.
  • Modul Keamanan Perangkat Keras Khusus (HSM) (tersedia di Inggris, Kanada, dan Australia) - Menyediakan penyimpanan kunci kriptografis di Azure dan memenuhi persyaratan keamanan paling ketat dan memenuhi persyaratan pelanggan.
  • Dukungan untuk Enkripsi Drive Azure untuk VMSS - Enkripsi Drive Azure sekarang dapat diaktifkan untuk Windows dan Linux VMSS di kawasan publik Azure - memberikan pelanggan kemampuan untuk melindungi dan menyimpan data VMSS saat istirahat menggunakan teknologi enkripsi standar.

Selain itu, dukungan untuk set mesin virtual sekarang tersedia sebagai bagian dari Pusat Keamanan Azure. Untuk mempelajari lebih lanjut, baca artikel kami tentang semua inovasi ini .
