苹果的AR / VR如何面对残酷的现实

虚拟或增强现实眼镜充满了诱人的机会。 但是开发困难使苹果重新考虑了期望





让我们从上下文开始。

在过去的18个月中，一直有传言称Apple正在开发AR / VR（增强现实/虚拟现实）设备。 对于上下文：我们可以使用HolosLens观看来自Microsoft的冒险，可以使用Oculus观看来自Facebook的冒险，可以使用粗糙的硬纸板工艺或来自Magic Leap的雄心勃勃的设备观看Google的冒险活动，其资本金非常高。 这一切都为苹果参加AR / VR竞赛提供了必然的光环。 评论员不禁将这些所谓的设备放入公司的下一个产品浪潮中。 从他们的观点来看，在iPhone在饱和市场中经历了数年的麻烦之后，智能眼镜最终将振兴苹果的缩资本。 以臭名昭著的Min-Chi Ko不断做出的预测为例，每个人都同意苹果的AR / VR产品将在2020年（可能在下半年）推出。

然后发生了不寻常的事情。

显然，苹果公司与参加魔术眼镜项目的所有员工举行了会议。 史蒂夫·乔布斯。 在那里，在数百人（有人说大约一千）的面前，副总统迈克·罗克韦尔提出了一个工作时间表：代号为N301的AR / VR混合头戴式耳机将于2022年发布，而精简版的N421眼镜将仅发布。在2023年。

所有员工的开会本身并不是一件难得的事情。 但是，通过考虑苹果进入AR / VR市场需要采取的所有步骤，可以更好地理解其本质，尤其是根据关于2022/2023的声明。

但是，在从软件和硬件领域深入研究所需组件的问题之前，我们需要上下文。 苹果设备是水平的，而不是垂直的-也就是说，它们针对更广泛的消费者和用例，而不像其他针对一小部分用户的高度专业化的工具。 换句话说，苹果将不会为外科医生生产VR眼镜，也不会为服务机器人的员工生产智能眼镜。

有了所有这些，您就可以想象第一个必要的组件：CPU / GPU组合，可以使我们从虚拟现实世界中获得令人信服的幻觉。 如今，最引人注目的VR工具包包括与功能强大的PC连接的眼镜或头盔。 必须以某种方式断开Apple眼镜与计算机的连接，并以Apple Watch的方式将其自己的计算引擎推到那里，与手机断开连接。

为了使该设备得到广泛认可，出现了一个众所周知的严重难题：CPU / GPU必须与一块小电池一起工作。 粗略地讲，它应具有优于当今iPhone的计算能力和图形功能，但需要整天工作。 说起来容易做起来难，仅凭这个因素就可以解释发布时间这么长。

然后我们仍然有钥匙孔问题。 您可以使用Google的智能眼镜回忆起与实验相关的社交尴尬和虐待行为 。 如果您看到我坐在桌上戴着带摄像头的智能眼镜，知道我可以录制视频和谈话的声音，您会感到舒适吗？ 可以想象使用没有摄像头的智能眼镜（但是，这将限制其用途），或者摄像头在框架上的操作指示，以及与接受或拒绝个人对话记录相关的社会习俗的演变。 过渡并不容易。 也许以前的参与者进入市场会促进这种情况。

这使我们找到了从应用程序开发人员的角度进行最佳研究的用法示例。 对于智能眼镜及其混合现实应用（将计算机生成的图像投影到半透明眼镜上并进行聚焦以便在方便距离处进行查看），开发人员将需要一种更具创造性的方法。 可以想象在一个无聊的会议上，消息或电子邮件是如何出现在眼镜上的，或者是一个街上的人在旅途中如何读书，而冒着被张贴在帖子或其他人身上的风险。 我个人想参观卢浮宫，并就伊丽莎白·维吉·勒布朗 （ Elizabeth Vigee-Lebrun）的绘画技巧发表评论。 对于一组新的VR / AR用例，开发人员将需要iOS目前没有的平台。 这些平台本身将需要一个iOS平台来处理与智能手机和手表所提供的互动完全不同的互动。

这组困难似乎是无法克服的，在AR / VR竞赛中，没有任何参与者参加过前所未有的比赛，奖品是用户的头。 微软拥有针对企业家的价值3500美元的HoloLens 2套件。 马克·扎克伯格（Mark Zuckerberg）承认，Facebook在虚拟现实上的20亿美元押注并没有像以前那样快。 Google正在逐步淘汰对DayDream VR平台的支持。 Magic Leap已经筹集了26亿美元的投资，正试图吸引更多的资金，为此她需要放下在JP Morgan Chase的现有投资组合，因为她仍然无法用2295美元的Creator Edition设备进行革命。 顺理成章的是，苹果没有与任何人共享有关其发展的信息，而是难以沿着相同的道路发展到AR / VR产品或消费者品质的产品线。 苹果首席执行官蒂姆·库克（Tim Cook）最近对即将到来的困难发表了评论：

今天，我可以告诉您，没有能够定性解决此问题的技术。 需要显示技术，以及将各种各样的东西放在头部周围的技术，这存在很大的困难。 视野，显示质量-所有这些尚未达到所需的水平。

就我个人而言，在10月的会议上，苹果似乎谈到了现有的AR / VR或混合现实设备开发计划。 今年，该公司很可能认为2020年发布设备的障碍太多，因此暂停了，这可能导致有关取消该程序的传言。 然后，重新考虑了开发高质量用户设备的计划，并且为了使公众对开发的暂停感到放心，公司管理层组织了一次软件和硬件开发团队会议。 而且由于这种信息泄漏，直到产品的代号和日期，对于此类会议来说都不是很典型，因此看来这是故意的。
