1. Microsoft 365: Meningkatkan Penggunaan Perusahaan Menggunakan Windows 10 dan Office 365


Microsoft 365 membantu menciptakan lingkungan yang aman dan dapat diperbarui dengan menggunakan perangkat Windows 10 yang memiliki aplikasi Office 365 dan dikelola oleh Microsoft Enterprise Mobility + Security.

Modul ini, dirancang selama 3,5 jam, akan mengajarkan Anda cara bekerja dengan Microsoft 365, berbicara tentang dasar-dasar penggunaan alat, serta tentang keamanan dan pelatihan pengguna.

Pelajari lebih lanjut dan mulai pelatihan di tautan ini .

2. Mengelola sumber daya infrastruktur di Azure


Pelajari cara membuat, mengelola, melindungi, dan menskala sumber daya mesin virtual di cloud Azure. Menyelesaikan seluruh kursus akan memakan waktu sekitar 10 jam.

Modul Kursus:

  • Gambaran Umum Mesin Virtual Azure
  • Membuat mesin virtual Linux di Azure;
  • Buat mesin virtual Windows di Azure;
  • Kelola mesin virtual menggunakan Azure CLI;
  • Memperbarui mesin virtual;
  • Pengaturan jaringan untuk mesin virtual;
  • Buat templat Azure Resource Manager
  • Ubah ukuran dan tambahkan disk pada mesin virtual Azure;
  • Caching dan kinerja penyimpanan azure
  • Perlindungan drive untuk mesin virtual Azure.

Detail dan mulai pelatihan

3. Manajemen sumber daya di Azure


Pelajari cara membuat, mengelola, dan mengelola sumber daya cloud melalui baris perintah Azure dan portal web. Ngomong-ngomong, dalam kursus ini, seperti di banyak lainnya, Anda bisa berolahraga di kotak pasir Azure sendiri.

Modul:

  • Menyesuaikan persyaratan dengan tipe cloud dan model layanan di Azure
  • Kelola Layanan Azure menggunakan CLI
  • Mengotomatiskan tugas Azure dengan skrip PowerShell;
  • Perkiraan biaya dan optimalisasi biaya untuk Azure;
  • Mengontrol dan mengatur sumber daya Azure dengan Manajer Sumber Daya Azure.

Detail dan mulai pelatihan

4. Microsoft 365 Basics


Microsoft 365 adalah solusi cerdas yang mencakup Office 365, Windows 10, dan Enterprise Mobility + Security, memungkinkan Anda untuk berkolaborasi secara kreatif dalam lingkungan yang aman. Dalam kursus 4 jam ini tentang segalanya-semua yang perlu Anda ketahui untuk memulai dengan Microsoft 365.

Anda akan belajar tentang apa itu Microsoft 365, informasi dasar tentang layanan dan fitur layanan, belajar tentang kerja tim, keamanan dan kemampuan cloud. Ngomong-ngomong, untuk pelatihan Anda harus memiliki setidaknya pengetahuan dangkal tentang komputasi awan.

Detail dan mulai pelatihan

5. Administrasi Kontainer di Azure


Contoh wadah Azure adalah cara tercepat dan termudah untuk menjalankan kontainer di Azure. Tutorial ini akan membantu Anda mempelajari cara membuat dan mengelola wadah dan cara mencapai penskalaan yang fleksibel untuk Kubernet menggunakan ACI.

Modul Kursus:

  • Membuat aplikasi web kontainer menggunakan Docker;
  • Membuat dan menyimpan gambar kontainer menggunakan layanan Azure Container Registry;
  • Luncurkan kontainer Docker menggunakan layanan Azure Container Instances;
  • Menyebarkan dan meluncurkan aplikasi kontainer web menggunakan Layanan Aplikasi Azure;
  • Ikhtisar Layanan Azure Kubernetes.

Pelajari lebih lanjut dan mulailah belajar
