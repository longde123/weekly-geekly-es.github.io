Sponsor GitHub: cara baru untuk berkontribusi ke open source


Kami sangat senang mengumumkan dimulainya pengujian beta GitHub Sponsors , cara baru dukungan keuangan untuk pengembang perangkat lunak open source yang kami gunakan setiap hari. Anggota komunitas open source menciptakan alat untuk kita masing-masing, dan GitHub Sponsor adalah cara baru untuk membantu mendanai proyek mereka.


Untuk meluncurkan program baru dan meningkatkan pendanaan komunitas, kami juga meluncurkan Dana Pencocokan GitHub . GitHub siap membayar pengembang hingga $ 5000 untuk tahun pertama partisipasi dalam Sponsor GitHub.


Menjadi Sponsor
Punya pertanyaan? Faq

Dunia pengembangan saat ini sangat erat dengan open source. Semua ini tidak akan mungkin terjadi tanpa tim pengembang, perancang, peneliti, guru, penulis, dan banyak lainnya yang berdedikasi dalam pengembangan teknologi. Sekarang orang-orang luar biasa ini, berkat profil GitHub mereka, dapat menerima dana dari komunitas yang terkait dengan kontribusi mereka terhadapnya.



Tidak ada komisi


Komunitas open source adalah jantung dan inti dari GitHub. Para peserta yang mendukung infrastruktur digital bersama kami membuat komunitas ini semakin kuat, dan sebagai ucapan terima kasih atas kontribusi yang berharga ini, GitHub Sponsors tidak membebankan biaya pada mereka karena menggunakan platform ini. Kami juga siap membayar semua biaya untuk memproses pembayaran selama 12 bulan pertama program, untuk menghormati perayaan peluncurannya, dan sebagai hasilnya, 100% dari uang Anda akan diberikan kepada pengembang.


Tim dunia


GitHub Sponsor mendukung pembayaran di setiap negara tempat GitHub melakukan bisnis. Kita semua adalah bagian dari tim pengembangan perangkat lunak di seluruh dunia. Meningkatkan peluang untuk berpartisipasi dalam tim ini merupakan inti dari misi kami, jadi kami bangga menyediakan alat baru ini untuk pengembang di seluruh dunia.


Setiap peserta dipersilakan.


Sejumlah besar kontribusi untuk proyek sangat penting untuk produk yang berfungsi dengan baik, tetapi sayangnya sangat sulit untuk dilacak dalam kode. GitHub Sponsor diciptakan untuk membiayai semua jenis pekerjaan pengembangan perangkat lunak sumber terbuka. Siapa pun yang berkontribusi - sebagai pengembang, penyusun, mentor proyek - berhak mendapatkan sponsor.


Cara lain untuk berkontribusi


Sponsor GitHub adalah cara lain untuk berkontribusi ke open source, dalam bentuk dukungan finansial bagi orang-orang yang membuat dan mengembangkannya. Hadiah uang orang membantu mereka terus melakukan pekerjaan penting, memperluas peluang untuk berpartisipasi, dan memberi pengembang pengakuan yang layak mereka dapatkan. Mulai hari ini, setiap pengguna GitHub dapat mensponsori pengembang program anggota.


Semuanya dalam antarmuka GitHub yang sudah dikenal


Sekarang Anda dapat mensponsori pengembang tanpa terganggu dari urusan yang biasa. Ketika seorang peserta menjawab pertanyaan, memproses masalah atau kode mergit, Anda dapat pergi ke profilnya atau cukup mengarahkan namanya untuk mengungkapkan rasa terima kasih keuangannya.



Dengan menu pop-up baru ketika Anda mengarahkan kursor ke daftar pengguna yang kontra, Anda dapat melihat orang-orang yang mengerjakan kode yang Anda minati. Dengan hanya beberapa klik, dimungkinkan untuk mentransfer uang ke pengembang yang produknya Anda gunakan setiap hari.



Proyek open source juga dapat mengekspresikan metode pendanaan mereka secara langsung di repositori mereka. Jika Anda menambahkan .github/FUNDING.yml ke cabang master , tombol Sponsor baru akan muncul di bagian atas repositori. Menekan tombol membuka menu opsi pembiayaan dari file ini.


Format YAML fleksibel, sehingga kontributor dan peserta proyek dapat menentukan sendiri metode untuk membiayai proyek. Anda dapat menentukan satu atau lebih opsi: Buka Kolektif , Community Bridge , Tidelift , Ko-fi dan Patreon, atau tautan khusus ke metode alternatif.



Langkah pertama


Pada awalnya, GitHub Sponsors dimulai dari yang kecil dan sederhana, tetapi misi kami jauh lebih ambisius - kami ingin memperluas peluang untuk berpartisipasi dan menggunakan kode sumber terbuka. Kami di sini untuk melayani komunitas pengembangan, dan kami berharap dapat menerima umpan balik Anda tentang apa lagi yang ingin Anda lihat di Sponsor GitHub.


Kami juga mengadakan kelompok penasihat untuk mempelajari masalah operasional yang dihadapi tim. Grup ini terdiri dari pemimpin sejumlah proyek sumber terbuka, dan kami berharap dapat menemukan vektor pembangunan bersama. Bagaimanapun, open source adalah olahraga tim. Jika Anda dan tim Anda ingin berada di grup penasihat ini atau jika Anda memiliki ide tentang bagaimana GitHub Sponsor dapat membantu komunitas, kirim email ke opensource@github.com .
