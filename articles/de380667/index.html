Matrix Flare - ein Spielzeug, um Kindern die Welt der Elektronik näher zu bringen



Das Matrix Flare-Projekt begann als individuelles Spielzeug für ein Kind und entwickelte sich allmählich zu einem unterrichtenden elektronischen Designer für alle. Der Autor startete eine Kampagne auf Kickstarter, um Spenden für das Projekt zu sammeln, in der Hoffnung, den Betrag für die Massenproduktion von Spielzeug zu erhöhen.

Es ist unwahrscheinlich, dass das Projekt den richtigen Geldbetrag einbringt, aber es ist interessant genug. Deshalb haben wir beschlossen, es im Medgadgets-Blog zu beschreiben. Der Autor der Idee hofft, dass Matrix Flare Kindern (und nicht nur ihnen) hilft, die Grundlagen der Programmierung und Elektronik zu verstehen.

Was ist Matrix Flare?


Dies sind Würfel, bei denen die Gesichter aus einzelnen Elementen bestehen, die von LEDs unterschiedlicher Farben beleuchtet werden. Zur Steuerung jedes Cubes wird eine spezielle Software verwendet, die auf Visual.net ausgeführt wird. Der Code für Arduino wird verwendet.

Eingabedaten werden von der Pixel Maker App in der Datei anim.h gespeichert, die von der Arduino IDE Trinket Jewelry Sketch von Adafruit.com verwendet wird. Für jede „Matrix“ können Skizzen und Programmanweisungen verwendet werden. Gleichzeitig ist die Arduino IDE ein freies Software-Tool.



Wenn Sie möchten, können Sie mit Ihrem Matrix Flare-Würfel ein fertiges Programm bestellen.

Alle animierten Sequenzen können auf neue Weise geändert und geladen werden. Nach dem Ende der Kampagne wird eine große Anzahl von Animationen für die Würfel auf der Website des Herstellers veröffentlicht.



Matrix Flare Assembly


Standardmäßig wird mit jedem Würfel eine Animation mit einem Pferd geliefert, sodass alles sofort funktioniert. Matrix Flare-Hüllen werden auf einem 3D-Drucker hergestellt. Es gibt verschiedene Optionen.

Die Eigenschaften von Matrix Flare sind wie folgt:
  • Micro Typ B USB-Schnittstelle mit Arduino IDE;
  • Batterie 3,7 VDC 150 mAH, der Ladevorgang erfolgt über den oben angegebenen Anschluss.
  • Falls gewünscht, können die Würfel an USB angeschlossen bleiben.
  • Wenn ein einfacher Würfel deaktiviert ist und die Animation abgespielt werden soll, müssen Sie auf die Schaltfläche Zurücksetzen klicken.




Würfelhüllen werden auf einem 3D-Drucker gedruckt, während es mehrere Modelle von Hüllen gibt.
