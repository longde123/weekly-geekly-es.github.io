Windows XP 15年

就在15年前，全世界看到了“ Windows XP正式投入生产”，尽管该系统仅在2001年10月才被广泛销售，但可以说是8月24日，这是有史以来最受欢迎的个人计算机OS的发布日期。自2011年发布XP仅仅十年后，另一项成功的Microsoft产品Windows 7就成功超越XP。

进入新千年之初，微软管理人员承认，完全成功的Windows 98已经过时了，Windows 2000不能满足产品所设定的要求，并且不想记住Windows ME的所有崩溃，错误和不稳定的工作，这已经成为一种危险。如何不仅对公司内部的开发人员而且在IT社区中通常不这样做的“标准”：


“鲍尔默峰”

由于Windows ME失败的原因，XP已成为其另一个前身Windows 2000 Professional的继承人。

XP与其前代产品的显着特点可以称为设计，安全性，交付方面的巨大变化，与许多Internet Explorer 6讨厌的操作系统捆绑在一起（IE系列本身最近迎来了它的21周年纪念日），而且最重要的是，XP对于不同的受众有很多变化CIS的开放空间主要是XP Professional的盗版版本。

例如，在发布主要版本的操作系统（如Home，Professional等）的几年后，Microsoft花费了很多时间并发布了Windows XP Tablet PC Edition，该Windows XP Tablet PC Edition与PDA捆绑在一起，并支持使用手写笔，公认的手写体，并且实际上可以将成为平板电脑的基本操作系统。掌握这个市场的尝试被证明是匆忙的-在iPad和其他现代设备问世之后，世界和技术还没有准备好制造舒适使用的平板电脑。一切如何结束-我们知道。微软没有为平板电脑创建能够独立工作的移动操作系统，而是以Windows 8、8.1和10为例，采用了完全“平等”的道路，并且即使使用诺基亚品牌，移动市场也没有提交，因为该操作系统本身与用户的主要应用相差甚远。但是一个生态系统应用程序和兼容性。也许这被忽略了，因为微软已经习惯了“ PC世界围绕Windows”这一事实，而关于移动设备则不能这么说。

微软高管的预言能力还有一个有趣的事实，只有Windows 7发行才能充分体现这一点。在32位处理器主导市场的时候，MS开发人员发布了Windows XP 64位版本和Windows XP Professional x64版本。第一个用于Intel Italium处理器，第二个用于基于AMD64，Opteron和Athlon 64技术的处理器以及采用Intel EM64T技术的处理器。但是，计算机园区的大规模升级和向64位体系结构的过渡仅在零末期发生，但是32位处理器的末期即将来临这一事实确实让Microsoft倍感荣幸，并且当这种情况发生时，他们已经为此做好了准备。

好吧，最可能是众所周知的使用XP的事实，这对不熟悉金融领域的人来说是令人惊讶的-Windows XP Embedded。维修自动柜员机，终端或其他设备时，您肯定会遇到这种装配，而您肯定不会遇到“ Windows”。

除了ATM和终端机中的“生命”之外，Windows XP在某些领域仍然是重要且不可替代的操作系统。例如，在WinXP之下，一家欧洲著名制造商编写了诊断软件，该软件不仅“紧密地”粘合到其安装的硬件上，而且还只能在XP下运行。

Windows XP中与Windows 2000有关的主要创新是：

  • 新的界面设计；
  • 功能“远程助手”；
  • 系统恢复；
  • 远程访问能力；
  • CD的高级工作；
  • 使用系统写入-R -RW光盘的能力；
  • 直接使用ZIP和CAB归档文件，而无需使用软件和其他软件。

早在2004年，Microsoft就报告出售了2.1亿张Windows XP许可副本。考虑到盗版，只能猜测该公司操作系统的实际用户数。

Microsoft已发布了三个XP的Service Pack。SP1于2002年9月9日发布一年后发布。这可能是最重要的更新之一，因为SP1家族的系统开始支持USB 2.0，直到今天它仍然保持其地位，并在USB 3.0来袭之前勉强退缩。两年后，该公司推出了SP2，并且在2007年1月开始销售Vista的一个月前，它对于每个人来说都是熟悉的，并且在发布SP3之后是强制性的。

说到Vista。许多人开玩笑说微软每隔一段时间就可以获得稳定的操作系统，但不幸的是，事实证明是正确的。本应以XP为荣的系统失败了。也许问题出在MS方面，很着急，也许是在开发过程中出现了错误，但是Vista过于繁琐，总是有漏洞，没有扎根。此外，设备制造商还没有为新的操作系统做好准备，而且旧硬件的驱动程序通常也不是很好，并且XP的“滚花”驱动程序不起作用。许多笔记本电脑所有者面临相同的问题，他们期望“常规Windows”会忽略外壳上的“ Vista”标签，并且在尝试安装XP时会遇到相同的问题-缺少驱动程序，但这一次是XP。

最可悲的是，微软没有考虑广泛的消费者。如果在XP的情况下，硬件要求远远超过保留，这使大多数用户可以“切换”到新操作系统，则Vista失去了这种贵族：


系统要求Windows XP（2001），维基百科


系统要求Windows Vista（2007年初） ，维基百科

显然，XP要求是可以接受的。在2001年，有很多266 MHz，很多用户甚至拥有推荐的600 MHz处理器时钟速度。与RAM相同-在2001年，对于家用或正在运行的PC，64 Mb被认为是一个很小的数字，并且许多具有128甚至256（接近零的中间值）Mb内存。

在Vista的情况下，有人陷入困境，而那些在800-1500 MHz处理器上和平而舒适地存在于XP上的用户中，他们已经要求800最小的工作量（从1000 MHz开始）以及高达2000和两个内核。有了RAM，它甚至更难受-从512 Mb到1024 Mb，这是系统“吃”的最大份额。

此外，稳定性问题以及由此导致的Vista问题被广大群众所忽视，为数不多的稳定销售渠道之一是在笔记本电脑上预安装了XP上的官方驱动程序，根本就不存在这些驱动程序（无论如何在制造商的网站上）。

Windows 7的发布挽救了这种情况，该版本理所当然地接管了Windows XP的“宝座”，现在已成为Microsoft操作系统系列中的主导产品。

在Microsoft意识到情况稳定之后，宣布了对Windows XP的支持，但是由于用户数量众多，该支持被推迟了好几次。

当然，进步发挥了作用。较早的XP是可以使用的较新软件，因此可以说Microsoft并不是试图“杀死”其最好的产品，而是只是不想花时间去做。另外，您需要销售新的操作系统，以扩大受众。

毫不奇怪，相对于其后创建的许多其他操作系统，XP仍然屹立不倒。因此，根据统计数据，全世界约有10％以上的PC用户仍在使用2001版系统。当然，最大的份额落在第三世界国家和中国，那里的人们由于无法更换设备或其他原因（例如出于生产原因）无法负担离开系统的费用，但这个数字仍然令人印象深刻：

netmarketshare.com

从图中可以看出，计划作为“ XP的女继承人”的Vista甚至没有进入主机，而被“其他”所掩盖，但是Windows 7在几乎一半的全球操作系统市场上都没有什么亮点。

出人意料的是，即使在2015年退出支持后，XP在幸免的5个操作系统中也幸存了3个-Vista，Windows 8和Windows 8.1可以被记录为“失败者”。为了捍卫最后两个，我们可以说他们在Windows 10发行后相对于旧版XP成为“局外人”，也就是说，他们通过分配给他们的角色来应对，并且将很大一部分受众转移到了新产品Windows 10中，尽管这是通过大规模的广告宣传和，有时是强制性的。总的来说，我们做了XP对于Vista所做的事情，但是并没有一起发展。

在不久的将来，只有Windows 7才能赢得XP桂冠，而且有人怀疑微软会再次计算错误-七和8.0-8.1实在太不同了（因此，它们是Windows 10的后继产品）。XP Windows 7是否会重复历史记录？这是可能的，但我们仅在2022-23时才知道，当时最流行的OS大约有15年的历史。

同时，Windows XP历史上仍是世界上最“长时间播放”和最受欢迎的操作系统的桂冠。
