用于Beta PC的Remix OS：如何使用Windows组织双启动并开始使用新OS



Geektimes反复撰写过有关Remix OS（基于Android的桌面操作系统）的文章。 3月1日，该操作系统的beta版本出现了，已经可以使用它了。一般而言，这是Android的定制版本，其外观和功能类似于个人计算机的完整操作系统。它于2015年首次发布，对多个设备的支持最少。 1月，Remix OS的Alpha版本发布了，开发人员将其提供给所有人，从而使您能够从USB驱动器引导系统。

在beta版中，出现了双重引导功能，可以在两个操作系统之间进行切换-Remix操作系统本身和Windows。您可以将操作系统安装在硬盘驱动器和SSD上，这提供了相当高的平台性能。现在，适用于PC的Remix OS安装程序支持使用Windows 7和此操作系统的更高版本。 Mac的版本应会很快面市。如果操作系统很累，则删除它比安装它还要容易。



现在定期进行更新，内置了对无线更新的支持。还有其他消息，包括切换到Linux内核版本4.4，以更好地与更新的处理器兼容，支持来自YouTube和Netflix的1080p视频，以及最大容量为64 GB的USB驱动器。

为什么不呢 支持Play商店。由于Remix OS未经Google认证，因此没有默认进入市场的权限。但是您可以添加它，更多在下面。首先，让我们谈谈安装Remix OS本身。



不想在SSD或硬盘驱动器上安装Remix OS？没问题 您可以直接通过USB闪存驱动器使用操作系统，使其可启动。但是，在这种情况下，操作系统性能会比将其放在HDD或SSD上时低。



如何在HDD或SSD上安装Remix OS



1.首先，从此处下载最新版本的操作系统。对应的PC有32位和64位版本。

2.解压缩存档文件并获取三个文件：带有说明的教科书，安装程序和ISO，磁盘映像。

3.启动Remix OS Installation Tool.exe并确认安装位置正确（驱动器号等）。



4.单击“下一步”，然后选择刚加载的磁盘映像。它将类似于PC_x86_64-B2016xxxxxx.iso的Remix OS。

5.我们开始安装并等待几分钟，直到出现消息，提示需要重新启动PC以完成安装。

6.重启后，您会看到一个屏幕，建议您选择两个系统。



7.我们正在等待。

下一步将需要15到30分钟，具体取决于您计算机的性能。Remix OS将创建自己的8 GB引导区，并将OS安装在该区域中。

8.一切准备就绪后，Remix OS将首次启动，您可以开始使用。

9.如果需要返回Windows，请选择“关机”图标，然后选择“重新启动”选项。之后，在选择屏幕上，而不是Remix OS，单击Windows。

是否可以获取超过8 GB的空间？


官方的Remix OS PC安装程序需要8 GB的存储空间。这足以安装您可能需要工作或娱乐的各种应用程序。



但是，如果您突然需要更多空间，则可以添加它。

如何删除Remix OS


如果突然出现问题，或者只是想删除Remix OS，则可以很快完成。

最初，再次运行安装程序。它表明该系统已经安装在PC上，并提供了删除Remix OS的功能。我们同意，并在几秒钟后删除了操作系统。

您不能使用安装程序，但可以手动删除所有内容。在文件管理器的地址栏中，选择C：\ Remix OS \，然后运行文件UninstallRemixOS.exe file。然后，一切都会在自动模式下发生。
