引入新的Microsoft Edge和Bing



互联网是实现我们使命的关键工具：借助我们的知识和Microsoft 365帮助人们。人们在PC上花费的时间中有60％使用Web浏览器-它们已成为我们工作，学习和娱乐的主要场所。 我们每天都需要互联网； 人们没有考虑它就一直使用它，但是在全球网络出现20年之后，我们遇到了许多障碍和困难，因此我们有以下问题：

  • 使用互联网时，我们的数据和敏感信息是否受到保护？

  • 在工作时，我们是否可以像使用Internet在互联网上查找其他任何数据一样，在浏览器中通过搜索直接接收公司内部信息（例如薪水收入或同事的办公室位置）？

  • 您如何以与Office 365相同的方式使工具无缝地在线联机，以便我们进行更多研究并与同事合作？

Microsoft认为，无论您是出于个人目的还是出于专业目的使用Internet，无论您是IT管理员还是Web开发人员，都应该花更多的时间在Internet上。 引入了完全更新的Microsoft Edge和Microsoft Bing。



我们看到了以最佳比例将现代网络搜索与集成解决方案相结合的独特机会-如今只有Microsoft可以提供此功能。 我们将Internet与Intranet网络集成到Microsoft Search和Bing中，以便您可以在单个浏览器和搜索界面中访问越来越多的重要数据。 我们将为机密数据提供强大的保护，同时保持个性化的好处。 让我们直接从网络搜索转到Microsoft Office应用程序，以提高工作效率。

帮助人们提高工作效率


员工面临的主要问题之一是难以找到公司信息并难以在公司的内部网络中访问它。 具有讽刺意味的是，比起公司内部网中的简单页面，在更大的Internet空间上查找抽象信息要容易得多，例如，带有工资单的门户，公司有关办公室内动物的公司政策或同事在工作场所的楼层平面图。

如今，许多公司的公司信息量以兆兆字节，PB甚至Zettabytes的信息来衡量，而且情况变得越来越复杂。 麦肯锡公司关于这项研究的报告解决了这个众所周知的问题：根据收到的数据，员工将近20％的工作时间用于寻找自己应该知道的信息1。 事实证明，每周您都可以恢复一整天的工作-完成更多任务或与家人在一起的时间更多。

对于IT专业人员而言，管理现代Web工具是一项复杂而昂贵的任务，因为大多数大公司至少支持两个浏览器，以最大程度地在所有主要平台上实现现代网站与旧版网站的兼容性。 根据2019年Ponemon Institute的研究，数据泄漏的平均成本为392万美元USD2。 我们看到了为IT专业人员简化安全性和可管理性的机会。

为了帮助IT专业人员和其他员工在保持安全性和生产力的同时控制数据，今天，我们介绍：

  • 适用于Windows的Microsoft Edge和macOS预览版，可立即下载； 并计划于1月15日以90多种语言向公众开放。 Microsoft Edge使用与Google Chrome浏览器相同的Chromium引擎，为您提供一流的兼容性和性能。

  • Bing中Microsoft Search的新功能将节省在公司内部查找信息的时间。 使用自然语言，直接在浏览器的地址栏中查找人员：指示职位，部门名称和办公室位置。 或寻找办公室的位置，并作为回应，获取带说明的平面图。 您还可以了解如何解密公司中使用的缩写，并且广泛的问题和答案列表将帮助您查找必要的内部信息。



  • 今天，我们与来自生态系统的三个合作伙伴-埃森哲，BA Insight和Raytion-一起推出了Microsoft Graph连接器的初步版本。 这扩展了Microsoft搜索对Microsoft 365用户的搜索功能； 我们添加了一百多个新连接器，包括Salesforce.com，ServiceNow，Box和许多其他连接器-它们将在2020年上半年上市。 连接器将为您的员工提供对Microsoft搜索中更多信息的访问。



  • 新功能可轻松在移动设备上的Bing中访问Microsoft搜索。 智力工作者即使在旅途中也将能够搜索公司信息。



  • Microsoft Edge中预先安装了新的预防跟踪功能。 我们认为您应该知道谁有权访问您的数据，并能够选择愿意共享的信息。 MicrosoftEdge具有默认的防止跟踪功能，因此您无需采取任何措施即可迁移到更私有的浏览器。

    借助SmartScreen和反跟踪功能，我们可以帮助您保护自己免受网上诱骗，恶意软件和新型攻击（如加密千斤顶）的侵害。 Microsoft Edge和Bing现在还提供了新的InPrivate模式，该模式可以持续工作，以使您的在线搜索和浏览数据无法链接到您，因此您将可以更好地控制个人数据。



  • Microsoft Edge的预览版提供了新的“收藏夹”功能。 它简化了Web内容集合的创建，在Word和Excel中搜索和导出必要内容的组织，因此您可以使用这些应用程序的功能来分析信息并进行协作。
