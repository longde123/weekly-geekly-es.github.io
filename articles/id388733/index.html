Luncurkan Windows 95 di Nintendo 3DS? Ya itu mungkin



Jika Anda ingin kembali ke waktu ketika MS Paint dan Internet Explorer 1.0 memerintah di komputer pribadi, maka Anda tidak dapat menemukan mesin waktu. Cukup menginstal Windows 95 pada konsol Nintendo 3DS portabel Anda. Dalam salah satu posting di forum Game Boy, "retronaut" memposting tangkapan layar dan video yang menunjukkan operasi OS yang disebutkan.

Ketika ditanya mengapa mereka melakukan hal yang serupa, pengguna dengan nama panggilan Shutterbug2000 menjawab bahwa ia hanya ingin melakukannya. Untuk menjalankan Windows 95, sebuah emulator x86 digunakan.



Selain tangkapan layar, pengguna memposting video di YouTube. Menunjukkan layar boot Windows 95 dan Scandisk. Video, bagaimanapun, berhenti bahkan sebelum desktop muncul.

Pengguna memposting dan kode yang ditulisnya untuk menjalankan Windows 95, kode ini dapat ditemukan di sini. Pengguna lain telah mengkonfirmasi bahwa semuanya berfungsi, proyek ini bukan tipuan.

Tentu saja ada kerugiannya. Misalnya, kinerja seluruh sistem buruk. Sekarang juga sulit untuk memahami bagaimana Anda bisa belajar mengendalikan kursor menggunakan sensor tampilan konsol.
