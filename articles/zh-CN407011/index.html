一名黑客以15万美元的价格出售被黑客入侵的Tesla Model S P90 ++版本



著名的安全专家Jason“ wk057” Hughes喜欢入侵特斯拉汽车。 一年前，他完全控制了特斯拉Model S后轮驱动器，创建了自己的控制板，并为特斯拉电池解锁了软件锁定的5 kWh。 今年1月，一名黑客入侵了电源管理系统，并拆卸了100 kWh的电池（实际上，其工作容量为98.4 kWh，缓冲区为4 kWh）。 经过他的所有发现，杰森·休斯（Jason Hughes）成为特斯拉车主社区中的真正名人。 在二月份，他的名气达到顶峰-然后他推出了Model S P85的第一个真正的改进版本，升级后称为Model S P90 ++。

Model S P90 ++的抽水版具有30％的马力（作者本人说过570马力 ），90 kWh电池，特斯拉双引擎版本的软件（可笑模式），自动安装的自动驾驶仪等。 。

升级的关键要素是以Ludicrous模式安装控制软件。



事实是，这种软件Tesla仅在具有两个电机的Model S的强大修改版本上安装。 但是，杰森仍然设法欺骗了汽车，并给人留下了拥有两个引擎的印象。 他使用自制的硬件和开发的软件模拟了第二个引擎。 黑客发布了一张他安装在汽车上而不是第二个引擎（前轮驱动器）的硬件模块的照片。


虚拟第二引擎

尽管改装者本人谈到了570 hp，但在测功机上进行的测试显示后轮的功率为553 RWHP，扭矩为480磅-英尺的RWTQ。 测功机的结果如图所示。



当杰森·休斯（Jason Hughes）告诉所有人有关改装的消息时，一些Model S车主表达了对自己的汽车进行同样改装的愿望。 有人只是想要一个假的前轮驱动器，有人想要购买一个带有电池的完整包装。

尚不清楚杰森·休斯本人是否从事商业活动，是否将“抽水工具包”卖给了想要驾车的人。 但是现在，黑客肯定决定赚钱了。 他以15万美元的均价出售了他独特的Tesla Model S P90 ++ 。 这比串行Model S贵十万。利润似乎不切实际，但您需要了解这是全世界唯一的这样的汽车，而人类社会的独特性是最重要的。 每个人都想拥有别人没有的东西，让它变得琐碎，最重要的是独一无二。 但是，特斯拉Model S P90 ++根本不是一件小事，您可以乘坐它到城市上，在3.37 s内加速至100（甚至更快）。 您能想象飞行员在这种加速过程中会遇到什么样的超载吗？ 实际上，他像火箭上的宇航员一样挤在椅子上。



除了上述所有对特斯拉Model S P85的升级外，黑客还制造了另一个。 他安装了自动驾驶仪设备 。 事实是，特斯拉于2014年10月推出了自动驾驶仪，同时宣布现在所有汽车都将配备自动驾驶仪的“硬件”，即摄像头，雷达和360°超声波传感器。 在过去三年中，特斯拉不断更新软件，推出新功能。 当然，旧版汽车的所有者会问一个问题：是否可以在旧版汽车上安装自动驾驶仪的硬件和软件？ 特斯拉迅速降温了他们的热情，并明确表示没有修改是不可能的。 就像，旧车的设备会花费太多，因此购买带有自动驾驶仪的新车会更容易。 好吧，杰森·休斯（Jason Hughes）证明了这仍然是可能的。

要安装自动驾驶设备，黑客必须更换方向盘和转向柱，仪表板，制动助力器，ABS / ESP（牵引力控制模块），所有四个车轮上的速度传感器，挡风玻璃，后视镜，ECU驻车传感器，所有驻车传感器及其接线，所有后保险杠盖和温度控制器模块。 他添加了具有适当安装座和前置摄像头组件的雷达。 还必须修改前保险杠盖和轮罩，以适应超声波传感器。

总的来说，这家伙努力工作。 有可能在他的“燕子”上有一个15万美元的买家，如果我有钱，我会买的。
