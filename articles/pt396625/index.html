Especificações técnicas


O apogeu BK-01 é uma versão do lendário computador Radio 86-RK, que é um pouco aprimorado. O clássico Radio 86-RK tinha 16K de RAM, expansível para 32K. No Apogee, a quantidade de RAM é 56,25K.
Posteriormente, uma versão com uma imagem colorida foi lançada, Apogee BK-01Ts (mas eu só tenho BK-01).
A cópia que recebi foi lançada em abril de 1991.
