Le Pirate Bay a de nouveau activé le crypto-mineur pour les utilisateurs, vous ne pouvez pas le désactiver

Le 17 septembre, Geektimes a annoncé que The Pirate Bay avait testé le mineur comme une alternative aux bannières publicitaires sur le site. En d'autres termes, l'administration de la ressource a décidé de changer la méthode d'obtention des fonds pour l'entretien du site. Cela a été fait sans avertissement, personne n'a demandé le consentement de l'utilisateur.

Le mineur n'a été découvert que parce que l'ordinateur a commencé à fonctionner beaucoup plus lentement que d'habitude. «L'exploitation minière pirate» n'a pas duré longtemps, et les utilisateurs ont trouvé plusieurs façons de désactiver le crypto-mineur, qui consommait les ressources de l'ordinateur sur lequel le navigateur avec la page de suivi chargée était ouvert. Au final, la ressource a supprimé le code mineur Coinhive de ses pages. Mais maintenant, le code a été ajouté à nouveau et il ne peut plus être désactivé.

Pour la première fois, le code a été injecté non pas directement dans le code de la page, mais dans le code publicitaire. De nombreux bloqueurs et outils antivirus ont pu désactiver le script, qui était utilisé par les visiteurs du site qui ne voulaient pas que quelqu'un utilise les ressources de leur ordinateur. Après un certain temps, l'administration de The Pirate Bay a signalé que les tests avaient été menés sur une courte période (24 heures).

Soit dit en passant, l'argent reçu avec l'aide du mineur devrait apporter à l'administration de The Pirate Bay un bon revenu. Les experts ont calculé que s'ils n'arrêtent pas le mineur, alors dans un mois, cela aidera à gagner environ 12 000 $ en dollars. Le script du mineur lui-même est «affiné» sous Monero - une crypto-monnaie dont les transactions sont extrêmement difficiles à suivre. Peut-être que le montant réel est plus ou moins que l'estimation, mais peu importe - l'essentiel est que les fonds soient toujours décents.

Maintenant, le mineur est configuré de sorte qu'il ne charge pas trop les ressources système, de sorte que les utilisateurs ne peuvent rien suspecter.

La semaine dernière, le fournisseur CDN de Cloudflare a temporairement bloqué un autre compte de suivi de torrent ProxyBunker. L'administration de Cloudflare considérait le mineur comme un malware, donc cette décision a été prise. Dans le même temps, les utilisateurs de ProxyBunker ont la possibilité de désactiver le mineur - les propriétaires des ressources ont fourni une telle option. Mais les utilisateurs de The Pirate Bay ne peuvent pas se vanter de la même chose.

Fait intéressant, les créateurs du mineur Coinhive ne sont pas satisfaits des moyens similaires de gagner leurs clients. La société a promis de prévoir la possibilité de déconnecter le mineur de son côté, et il est prévu de réaliser cette opportunité de telle manière qu'elle ne puisse pas être cachée / rendue inactive.

Les mineurs de crypto sont maintenant devenus une bonne alternative pour les dons d'utilisateurs pour les trackers torrent et les sites pornographiques (à condition que presque personne ne transfère des fonds volontairement). La bannière publicitaire + le mineur peuvent apporter de bons revenus, à condition, bien sûr, que le trafic du site soit élevé.


Selon Adguard, plus de 220 sites de grande et moyenne taille - principalement des ressources pornographiques et des trackers torrent - ont commencé à utiliser des mineurs de différents types. Parfois, les mineurs sont également utilisés sur les ressources blanches. Par exemple, récemment Showtime et le site officiel de Cristiano Ronaldo sont tombés sur cela.

Il est difficile de comprendre si le code a été publié par des crackers ou si le mineur a publié l'équipe de ressources officielle. Le dernier site de ce type vu dans l'utilisation du mineur était le clone Dropbox uptobox.com, qui est inclus dans le top 1000 des sites les plus visités par Alexa. Son audience est d'environ 60 millions de visiteurs uniques par mois.

L'équipe CoinHive encourage ceux qui utilisent leur code sur leur site à en informer les utilisateurs. Progressivement, les antivirus et les bloqueurs de publicités commencent à bloquer les scripts des mineurs de crypto. Dans ce cas, l'utilisateur a généralement le choix - bloquer le mineur ou lui permettre de fonctionner. Le même AdGuard ne bloque pas les mineurs de crypto si l'utilisateur ne le veut pas.

Récemment, une société d'analyse a publié les résultats d'une étude sur l'utilisation des mineurs sur diverses ressources.
