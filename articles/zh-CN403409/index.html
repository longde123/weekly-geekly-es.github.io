SNR-ERD：从pingovalka到设备生产线



千分之二的开始是电信网络的飞速发展。 安装人员整天都在屋顶上，将电缆铺设到最终用户，因此急需设备来检查电缆是否与连接的端口匹配。 设备的“冻结”问题也很严重。 安装程序必须经常在通信节点之间进行连接，然后手动重新启动它们。 因此出现了开发第一个SNR-ERD的想法。

结果，“ NAG”的专家提出了创建可连接至电缆并ping特定端口的设备的想法。 稍后，该设备配备了该设备的远程重启功能。 因此出现了SNR-ERD-WEB，它已成为安装人员和网络管理员的通用助手。


第一平戈沃卡

第一个模型是一个8位微控制器，具有8 KB的闪存和500字节的RAM。 该设备是通过Web界面控制的。 该设备可以与温度传感器，重新启动设备以及控制门开闭传感器配合使用。

该设备很快找到了听众。 该设备的开发和改进是定期进行的。 此外，大多数建议来自ERD买家，他们在现场使用了该设备并且了解他需要哪些改进。 在下一个修改中，设备已收到SNMP协议，远程固件更新，恒温器功能，相位传感器。

对设备的巨大需求催生了功能不断扩展的许多设备的开发。 此链的下一步是SNR-ERD-2 ，它仍然被定位为监视数据中心参数的预算工具，并且对其修改的需求很大。



市场对具有功能不断增长的廉价紧凑型仪器的需求量很大，这导致了一系列具有共同指标SNR-ERD-2的设备的开发。 这些设备仍被定位为预算工具，用于控制小型数据中心中的基本物理参数。

ERD-2被电信运营商广泛使用，以监控数据中心（DPC）的温度。 例如，Mail.Ru Group安装了800多个ERD-2。

这是Mail.Ru集团副技术总监Sergey Kubasov的评论：

“我们在数据中心“ M100”（Mail.Ru集团）和“ ICVA”（Vkontakte）中使用ERD-2进行温度监控。 在为我们选择解决方案时，重要的是：
价格-对于ERD，一个温度传感器（与控制器一起使用）的成本是最低的之一；
外围设备-对于ERD，拥有任何L2交换机就足够了，无需购买任何其他用于聚合的控制器；
实施速度-尽可能详细地描述产品API；
交货时间-无论我们要求多少数量，货物都是在“第二天”或我们可接受的时间进行的；
生产的灵活性-迅速进行了小型定制，以增加温度传感器电缆的长度为形式，而没有其他问题；
高可靠性-在整个操作过程中，我要注意的是缺陷率非常低，几乎没有故障。
支持-我们会很快找到问题的答案。

缺少什么：
PoE-您必须拉电源，这非常不方便；
湿度传感器-那么我们将能够在ERD进行全面的气候控制。”

客户的需求不会被忽视，一段时间后，“ NAG”的专家开发了第三系列的设备。 除了已知的功能外，他们还在ERD-3中添加了：

  • 串行数据传输接口RS-232和RS-485；
  • 附加相位传感器；
  • 外部端口，最多可连接五个1-Wire传感器（SNR-DTS-2）；
  • 具有安装GSM / GPRS模块以实现无线数据信道以及发送SMS通知的能力；
  • 5V电源电压的无源PoE。

设备的设计也已更改。 将该设备放在方便的金属盒中，并安装在DIN导轨上。



ERD-3是三合一设备：以太网/ RS232或以太网/ RS485接口转换器，支持MegaTec协议的UPS控制器（不间断电源）以及用于监视传感器状态，介质温度和电压的设备。
顺便说一下，最近，ERD-2也开始在金属外壳中制造，这是对准备好在ERD基础上构建其解决方案的集成商的要求做出的回应，但是该设备的外观并不能完全满足他们的需求。 因此，现在可以以两种版本购买ERD-2：热收缩或金属外壳。


进阶ERD-2

让我们回到ERD-3 。 除了供应商的广泛需求外，集成商还需要该设备用于小型自动化，以及用于查询各种资源计量器的以太网/ RS-485和以太网/ RS-232的ASKUE系统。 基于ERD设备，我们之前撰写的“ LOAD”软硬件复合体可以正常工作。

GSM / GPRS模块进一步改进了ERD-3，可用于以GSM / GPRS为主要数据传输通道的解决方案。 该设备称为ERD-GSM。


ERD-GSM耐热测试

但是，时间并没有停滞不前，而且很明显，现代要求决定了一系列ERD设备的进一步开发，为此需要新的计算和硬件资源。

意法半导体（ST）的32位STM32F407微控制器和用于以太网网络的现代KSZ8031RNL PHY控制器被选作新硬件系统的基础。 随着平台的变化，需要完全重写软件。 同时，决定在RTOS实时操作系统的控制下转移功能软件。 操作系统允许释放控制周期，并基于灵活的优先级构建函数调用。 此外，该设备还能够通过外部专用触点供电，并符合PoE标准IEEE 802.3af-2003和IEEE 802.3at-2009以及36至48 V的无源PoE。

向新平台的过渡分多个阶段进行。 首先，诞生了带有基本设备的设备，称为ERD-Pro-mini。 最初的设备是在NAG实验室中以临时方式组装的。















现在设备看起来像这样：



ERD-Pro-mini的所有优点在运行过程中已经发现的缺点的背景下有所丧失。 由于情况的特殊性，错误地向了不希望这样做的触点提供了电压，这反过来又导致了设备的不可逆损坏。

ERD-3的主要优点之一是其外壳本身，模块化扩展的可能性以及RS-232和RS-485接口的可用性，而具有32 KB闪存而不是1024 KB的8位ATmega328控制器是瓶颈。在STM32F407 如果客户要求我实现一些新功能，则为此我不得不削减其他功能。 因此，出现了将ERD-3和ERD-Pro-mini的所有优点进行整合的任务。

ERD-3电路板比ERD-PRO-mini电路板小近一倍半。 因此，电子组件的密度明显更高，这意味着对电路板的制造提出了更高的要求。 这不是一项艰巨的任务，但是通过直接开发，事实证明，新设备除了具有ERD-3和ERD-PRO-mini中已经存在的特性外，还应具有许多其他特性。
需要在设备上放置以下设备：

  • 防止极性反转；
  • 升压型DC / DC转换器12V，用于为外部传感器供电；
  • 兼容PoE标准的5V DC / DC转换器;
  • 由PoE供电并通过专用电源连接器供电的备用电路；
  • 精密DAC（DAC），用于执行PID调节功能；
  • 强大的功率继电器，用于切换电压高达250 V，电流高达10 A的负载；
  • 五个组合的输入和输出端口，即所谓的DIO（数字输入/输出）端口；
  • 可复位保险丝，用于保护DIO端口和DAC端口，
  • 额外的外部非易失性存储器，用于存储设置和测量结果。

此外，根据GOST CISPR 24-2013，该新设备本应提供75,000小时的平均MTBF，并满足信息技术设备抗电磁干扰的要求。
结果，该光被“多功能控制器SNR-ERD-4”看到或简称为ERD-4。



在开发ERD-4的过程中，有必要设计电路图，选择组件并生产经过各种实验（例如，测试8 kV电压对空气静电放电的抵抗力）的样品。



为了符合GOST CISPR 24-2013的要求，进行了针对各种干扰类型的设备抗扰度的实验。


ERD-4纳秒抗扰度测试


ERD-4高能微秒脉冲测试

该设备在气候室内进行了测试。 进行极限电压和电流的测量。 经过一遍又一遍的测试样品后，这就是印刷电路板的最终外观。



我们还应该详细介绍DIO端口和DAC输出的操作。


DIO端口原理图详细信息


其实现的特点是每个端口都可以同时用于输入（DI模式）和输出（DO模式）。 这是通过以下事实实现的：微控制器通过特殊的去耦电路D11和Q17通过两个GPIO引脚连接到每个端口。 该解决方案增加了ERD-4的灵活性。 用户现在可以确定必要的端口配置。 例如，“ 5输入+继电器”的配置将在那些需要将大量传感器与继电器输出连接（周边保护，漏水控制）的应用中很有用。 此配置中的继电器可以包括警报器或地下水泵。 如果用户需要控制大量的继电器，接触器或起动器，则可以使用“ 5个输出+继电器”配置。 在这两种极端之间，任何中间配置都是可能的：“ 2输入+ 3输出”，“ 4输入和1输出”，等等。

同样，在ERD系列产品中，ERD-4输出首次可切换电源电压高达48V且额定/脉冲电流高达0.5 / 1A的负载。 这使您可以显着扩展ERD-4与外部开关设备和负载的接口功能。 特别是可以使用广泛的12 V和24 V继电器，如果要使用SNR-SMART插座等5伏开关设备，则内置电压源可为您提供最大2 A的总电流。

特别针对ERD-4，开发了两个夹层无线电模块。 一个用于GSM / GPRS，另一个用于868 MHz ISM网络。

GSM / GPRS模块允许通过GSM / GPRS进行备份连接，并通过SMS进行控制。 ISM 868网络模块允许您连接各种无线传感器，例如脉冲计数器，热量计，相位传感器等。




现在， SNR-ERD是电信运营商，自动化系统集成商，住房和公共部门企业，银行机构以及IT公司所需的一整套设备。 这些设备可用于通讯中心，数据中心，各种机柜中，这些设备具有用于环境控制，周边控制以及解决各种负载的遥控任务的自动化的设备。
