在诺基亚贝尔实验室，他们通过光纤以1 Tbit / s的速度实现了数据传输



统一的团队诺基亚贝尔实验室的工程师，慕尼黑的德国电信T-实验室和技术大学进行测试PCS技术（概率星座整形）。它允许以每秒1 TB的记录速度通过光纤传输数据。这比在Google Fiber网络上传输数据快1000倍。

根据开发商的说法，这项新技术将使电信运营商在不久的将来能够显着增加传输的数据量。骨干网目前的速度限制是40-100 Gbit / s。根据测试参与者的说法，测试是在现场进行的。

诺基亚贝尔实验室是芬兰诺基亚公司的子公司。实际上，这是贝尔实验室，由亚历山大·格雷厄姆·贝尔（Alexander Graham Bell）于1925年创立。在诺基亚与阿尔卡特合并之前，朗讯贝尔实验室是后者的研究部门。在2016年，随着公司阿尔卡特朗讯贝尔实验室取消的结果已经搬到了芬兰诺基亚公司的控制之下。

合作伙伴进行的研究是SASER（欧洲安全可靠路由）项目的一部分。

在德国电信的光纤网络中以1 Tbit / s的速度进行数据传输。专家说，在测试该系统时，他们几乎到达了1945年建立的“香农边界”。在香农边境下（英语“香农极限”（Shannon limit））是指可以（选择信号代码设计）以给定的信噪比校正信道中的错误的最大传输速度。

科学家说：“使用正交幅度调制测试概率星座整形（PCS）技术已显示出能够最大程度地利用光通信信道频谱的能力。” “ PCS更改了信号点，数据传输字母的使用频率。通常，所有信号点都使用相同的频率。在PSC的情况下，与从振幅较低的点开始相比，振幅较高的信号点的使用频率较低。结果，发射的信号变得更抗噪声和其他干扰。所有这些打开了使数据速率适应信道的可能性。这使速度提高了大约30％。”

科学家设法以上述速度将数据从斯图加特传输到达姆施塔特，反之亦然。

开发人员声称，在普通光纤网络以1 Tbit / s的速度传输数据之前，应该花费一些时间。该项目的作者指出，自光纤通信信道发明以来已经过去了50多年。 1966年，STC实验室（STL）的Kao和Hokam 引入了普通玻璃的光学玻璃纤维，该玻璃纤维由于杂质而具有1000 dB / km的衰减（而同轴电缆中的衰减仅为5-10 dB / km），它们包含的内容，原则上可以删除。

经过深入研究，从1975年到1980年，建立了第一个商用光纤系统，该系统使用波长为0.8μm的光并使用砷化镓（GaAs）半导体激光器工作。第一代系统的比特率为45 Mbps，转发器之间的距离为10 km。 1988年，日本与美国之间完成了世界上第一条跨洋光纤通信线路的铺设，总长约一万公里。 1988年，第一条跨大西洋电话光缆（TAT-8）也投入使用。



现在，网络用户正在生成越来越多的数据，并且越来越需要更多的信道带宽和技术来传输数据。每年，都会生成100％以上的数据。 5G网络无处不在，因此光通信通道需要发展以满足时代的需求。

诺基亚贝尔实验室总裁Marcus Weldon表示：“未来的光纤网络不仅应提供快速传输大量数据的功能，而且还应动态适应所用的任何通道以及当前的流量需求。”

诺基亚还认为，有必要尽快部署5G网络。。这将使您能够快速开发可实时传输大量数据的智能汽车基础设施。高速数据传输将使医生能够使用受控机器人进行远程操作。用户将能够使用新的媒体格式并比以前更快地接收内容。计划将“ 5G纳入”的首批国家之一是澳大利亚。沃达丰在这里负责该国新网络基础设施的开发。

诺基亚的成功追随专家的成功伦敦大学学院。年初，这所大学的一组科学家成功地实现了创纪录的1.125 Tbps的数据传输速率，比一家芬兰公司要早。英国人使用先进的DSP技术（数字信号处理）创造了记录。这为优化信噪比和提高数据传输速率提供了机会。

来自多个国家的科学和商业组织的其他专家也在努力优化通信通道的吞吐量并提高数据传输速率。因此，毫无疑问-太比特通道将很快投入使用。
