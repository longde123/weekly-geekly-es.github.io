2020年UX / UI趋势



我们将在不到几个月的时间内进入21世纪的第三个十年，而2020年将为我们带来UX和UI设计方面的崭新趋势和优秀实践。 从过渡到全新的“设计领域”（例如语音用户界面或AR / VR设计），再到界面设计的趋势（例如动画，自定义图像等）。

对于UX / UI设计而言，2019年是伟大的一年。 我们看到了侧边栏，圆角以及导航选项卡/面板中的更改如何开始采用一种不太传统的方法，这很棒。 这意味着世界在变化，设计也在变化，或者在许多情况下决定这些变化。

以下是在整个称为Internet的奇迹中探索的针对移动和Web应用程序的UX / UI设计的未来趋势和实践的列表。


本文是在EDISON Software（一家开发应用程序和网站 ，还处理用户界面的公司）的支持下翻译的。


AKA VUI语音用户界面


在2019年及最近的过去，我们发现用户执行语音操作的使用显着增加。 语音助手（例如Siri，Alexa和Google助手）已经非常流行。 由99firms公布的几项统计数据提到，根据Google的说法，所有搜索查询中有20％是使用语音完成的，此外，全球31％的智能手机用户至少每周使用一次语音技术。 同一篇文章还预测“到2020年，所有在线搜索的50％将通过语音激活。” 酷吧？

作为设计师，我们必须适应市场的需求。 而且由于我们看到2019年VUI的UX逐渐增加，我们希望在2020年逐步摆脱图形用户界面（GUI）并转而使用VUI最终会打开大门，从而在2020年看到更多的方法，实践和出色的设计推动人工智能行业的技术进步，并带动这个巨大新世界的更多追随者。



AR / VR设计


现在，这已经在Twitter上引起了人们的广泛关注。 随着诸如Apple之类的大公司投资于AR和VR，并推出了流行的ARKit或Memojis，Oculus，Google等。 我们看到，从地图到交互式体验，AR / VR的使用越来越多。

2020年将是AR / VR设计的重要一年。 根据在Apple Insider上发表的一篇文章，该公司计划在不久的将来发布一对AR点。 因此，这是一个非常重要的信号，表明AR / VR将在未来十年中得到更广泛的应用。 因此，您最好为孩子们戴上AR / VR设计师带扣，因为在2020年及以后，将会出现令人惊奇和有趣的事情，而您必须这样做！



设计中的情感感知


这是UX设计的那些领域之一，使我想起了我有多热爱生活。 我们是人，我们是情感动物，有时比我们想要的更多，因此我们作为设计师应该考虑到这一点（例如，认真！）。 那是什么意思呢？ 自远古时代开始就存在我们周围，并且已经实施了很多项目。 但是，你问什么呢？

用户体验设计师面临开发具有人类情感的出色数字产品和体验的挑战。 尽管不存在这种理想的事物，但我们可以实现的最接近的目标是确保我们分析用户在使用产品时的某些时刻的行为和某些情感，并确保这些情感反映在设计中。

看一下Spotify 。 他们在分析人的情绪并为此专门设计方面做得非常出色。 除了为您专门创建主页外，它们还会走得更远，并根据一天中的特定日期和时间提供建议。 那不是所有人！ 他们会为您创建一个播放列表。 每周，而且由于我对音乐特别感兴趣，因此此播放列表是我最好的朋友。 并相信我，直到失望为止。



好的设计讲故事


如果有人问我什么是好的设计，这是我最好的答案之一。 怎么了 正如我前面提到的那样，正因为我们人类是情感生物。 我们渴望在我们所做的每一件事中都有意义。 这就是为什么出色的设计可以讲述故事的原因，而不仅仅是看起来很漂亮。

Netflix是一个很好的例子。 他们的注册过程非常好。 正如Rafael Mkrtchyan在UX Planet上的一篇精彩文章中所写：“ Netflix用户连接越性感越好。 颜色，字体，图像-一切都能说明一切。” 他们确保用户确切知道他们将要订阅的内容，从一开始就告诉您已开启的内容，并在创建帐户的每个阶段继续提示。 而且，如果您访问他们的主页，您将看到另一个很棒的设计故事和情感设计，它们带有“继续观看”或“为您发行新作品”，它们甚至可以在一部精彩的电影或剧集的结尾提供出色的建议。 正如史蒂夫·克鲁格（Steve Krug）曾经说过的那样，就在他的惊人著作《不要让我思考》（顺便建议，我建议）的封面上，“我”是最终用户。



航海元素的新时代


正如我之前提到的，在2019年，我们看到了一种不太传统的导航元素方法。 这主要是由于我们设备的显示在变化。 自iPhone X发行以来，设计师不得不寻找一种方法来适应这些新条件。 怎么了 好吧，由于我们希望切换到无边框显示，因此我们在新设备上不再拥有流行的主屏幕按钮。



对于每个设计师来说，这都是非常困难的情况。 由于用户已经熟悉特定的导航样式，因此尝试一些新的甚至破坏性的操作可能会吓到您，使您感到困惑。 有垂直导航栏和为用户提供方便导航的新方法。 因此，请仔细浏览您的网站！
