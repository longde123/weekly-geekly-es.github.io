Ekstensi browser GitHub untuk meningkatkan produktivitas Anda



Sekarang GitHub adalah layanan paling populer di kalangan pengembang untuk pengembangan perangkat lunak kolaboratif dan penempatan kode sumber di "cloud". Ini digunakan oleh pengembang independen yang memiliki 1-2 aplikasi dalam portofolio mereka, dan raksasa teknologi, termasuk Microsoft, Google dan Facebook, yang memiliki ribuan proyek.

Sebenarnya, tidak masuk akal untuk memberi tahu Habr bahwa GitHub; Posting ini didedikasikan untuk ekstensi browsernya, yang memungkinkan Anda untuk meningkatkan kecepatan dan produktivitas. Secara umum, ada banyak ekstensi ini, jadi kami memilih ekstensi yang kami gunakan dalam pekerjaan kami sendiri atau yang direkomendasikan rekan kerja.

Skillbox merekomendasikan: Pendidikan Seluler dari awal kursus pendidikan online.
Kami mengingatkan Anda: untuk semua pembaca "Habr" - diskon 10.000 rubel saat mendaftar untuk kursus Skillbox apa pun menggunakan kode promosi "Habr".

Octotree




Banyak orang tahu tentang ekstensi ini, tetapi, sayangnya, tidak semua. Dan ia menjalankan fungsi penting - memungkinkan Anda melihat kode sumber dalam bentuk pohon. Kode ditampilkan di sisi kiri halaman.

Navigasi baru jauh lebih nyaman daripada standar, tidak ada yang mengeluh tentang kecepatan ekspansi. Octotree akan sangat berguna jika Anda mempelajari proyek berskala besar dengan sejumlah besar direktori bersarang. API GitHub digunakan untuk mengambil metadata.

Repositori pribadi juga didukung ( petunjuk penggunaan ada di sini ). Juga didukung oleh GitHub Enterprise.

Versi untuk Chrome , Firefox , Opera .

Autocomplete yang luar biasa untuk GitHub




Memperkenalkan pencarian instan ke dalam kotak pencarian GitHub. Menurut pengembang, ini bekerja dengan lebih dari 15 juta repositori (terbuka). Ekstensi ini berguna karena menghemat waktu dan dengan cepat mencari apa yang Anda butuhkan.

Versi untuk Firefox , Chrome , Safari .

Dasbor Github




Dasbor memungkinkan Anda untuk memfilter acara di dasbor aktivitas github.com. Beberapa dari mereka dapat disembunyikan, beberapa dapat ditampilkan. Semuanya sederhana dan sangat efektif.

Versi untuk Chrome , unduhan "manual" .

Sourcegraph




Dan ekstensi ini membantu untuk memahami berbagai bagian kode, dan juga menunjukkan contoh lain yang mirip dengan yang ditentukan pada awalnya. Omong-omong, ini juga bekerja dengan Phabricator. Jika Anda memiliki banyak kode, ini adalah salah satu alat yang paling efektif.

Versi untuk Chrome dan untuk Firefox .

Octolinker




Menyediakan navigasi kode cepat dengan mengubah elemen kode (seperti menyertakan, mengharuskan, atau mengimpor) menjadi tautan yang dapat diklik. Mengklik tautan ini memungkinkan Anda menemukan sumbernya. Bergantung pada bahasa pemrograman, ekstensi menunjukkan file sumber atau membawa pengguna ke situs eksternal (misalnya, ke halaman bantuan). Elemen yang dikonversi ke tautan ditandai dengan titik-titik merah muda sehingga Anda dapat langsung melihat di mana ada tautan interaktif.



Versi untuk Chrome , Firefox , Opera .

Ukuran repo github




Plugin yang mudah dan sederhana yang menampilkan volume repositori tunggal dan ukuran individu file di dalamnya.

Hanya bekerja dengan Chrome .

Github halus




Ekstensi lain yang sangat berguna yang menyederhanakan antarmuka Github dan menambahkan banyak fitur berguna: misalnya, menampilkan brunch yang baru-baru ini diterbitkan, memungkinkan Anda untuk bekerja dengan permintaan dengan cara baru dan menunjukkan avatar pengguna yang telah menanggapi komentar.



Versi untuk Chrome dan Firefox .

Github npm stats




Sebenarnya, semuanya jelas dari namanya di sini - plugin menunjukkan npm-statistik unduhan.

Versi untuk Chrome .

Selain ekstensi ini, ada ekstensi lain, ada beberapa di antaranya . Di antara yang lain, kami dapat merekomendasikan:
  • GitHub Isometric Contributionions - menghitung kontribusi berbagai pengembang untuk proyek dan memvisualisasikan informasi ini dalam bentuk grafis pixel-art;
  • Github NPM Hub - membantu mempelajari dependensi npm di repositori GitHub;
  • GitHub Linker - menghubungkan ketergantungan dalam sebuah paket atau file bower dengan halaman GitHub dari elemen-elemen ini.


Secara umum, GitHub sendiri membaik dan dimodernisasi seiring waktu, ditambah pengembang pihak ketiga menambahkan alat tambahan yang memungkinkan Anda untuk bekerja dengan layanan lebih cepat dan lebih efisien. Peluang ini jangan sampai terlewatkan, karena waktu selalu habis. Selamat menikmati.
