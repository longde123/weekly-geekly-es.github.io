新的XPRO-注重细节



预期已经可以预订的新型PICASO 3D Designer X PRO将会大量销售，这个问题变得迫在眉睫-这种3D打印机与许多同事有何不同？进行处理的功能是什么？在实践中它将如何看待。 带着这些问题，我们求助于PICASO 3D的代表，他们很好地共享了信息。



打印机配备了多个新系统，这些系统旨在显着提高打印质量并提供易用性。 这些创新包括负责以下系统：塑料供应控制，关于设备运行的自动警报，第一层控制，用于处理不同材料的配置文件，网络打印和精确的自动校准。 我们将在下面更详细地讨论它们。

打印机的打印头（以下称为PG）配备了经过更新的JetSwitch技术，该技术已在先前的Designer PRO 250型号中得到证明。让我们向不熟悉该技术的人解释：该技术允许同时使用两种材料-PG设计中提供了阀门，无需冷却即可立即关闭其中一种材料的供应，这使得使用两种材料进行打印既干净又快捷。

更新的JetSwitch现在具有两种打印模式。 在其中之一（默认情况下已启用）中，使用集成的喷嘴清洁系统，您甚至可以放心地打印最薄的元素，这使其非常适合复杂的演示和详细的打印。

材料之间的切换速度为5秒。

第二打印模式被加速。 在此模式下，材料之间的切换速度小于一秒。 因此，带有支撑和打印两种颜色的零件的打印速度大大提高。 此模式更适合于打印壁厚大于一毫米的大型对象。



关于其余-进一步，为了秩序。

塑料进料控制系统



PICASO 3D Designer X PRO配备有传感器（编码器），用于控制塑料进给系统中的螺纹打滑。 打印机不断“计数”每单位时间有多少塑料通过挤出机。 它有什么作用？ 这种解决方案的使用有助于避免许多情况下的打印缺陷，从而避免不必要的材料，时间和精力损失。 现在，如果塑料用完或喷嘴开始堵塞，Designer X PRO将自动检测到问题并尝试使用内置的喷嘴清洁算法自行修复。

如果自动清洁后问题得以解决，打印机将继续工作并通知用户打印过程中遇到困难。 这将有助于注意这种情况并避免其再次发生。



如果自动清洁不起作用，则Designer X PRO会暂停打印并使用内置的警报系统通知用户有关情况。 用户可以手动清洁打印头，更换材料或更换喷嘴。 解决问题后，打印机将能够继续工作而不会丢失打印结果，材料和时间。

对进纸进行连续控制有助于避免卷筒中的塑料用完并且打印机尚未完成模型制作的情况。 Designer X PRO尚不能独立更换塑料，但是它已经能够向用户发送有关问题的信号。

警报系统



如前所述，Designer X PRO不仅可以控制其状态，还可以与用户共享此信息。 通过更改工作相机的背光灯颜色，设备将通知您：在打印的哪个阶段，工作过程中是否有任何困难以及是否需要用户干预。 如果您可以连接互联网，则即使您不在打印机附近，设备也可以将其状态通知您。 过程可控性和可预测性是新Designer X PRO的最重要功能之一。



第一层控制系统


曾经使用过3D打印机的每个人都知道第一层的重要性。 他就像一个基金会。 如果基础不均匀或易碎，则整个建筑物迟早会倒塌。 因此，在3D打印中-错误地固定在平台上或不平坦的第一层上的可能性很高，将会破坏整个打印输出。 困难在于，对于不同类型的塑料，打印平台上的压力需要不同。 借助反馈系统，在每种情况下，Designer X PRO均可实现最佳夹紧力。

当开始打印作业且喷嘴首先接触工作台时，打印机通过集成在PG中的编码器并使用针对每种材料预安装的第一层的电阻曲线来测量灯丝的速度，并几乎立即校正工作台沿Z轴的位置，以实现第一层的最佳结构。



档案系统



借助新的配置文件系统，Designer X PRO可以“知道”当前在打印机的哪个喷嘴中安装了哪种材料。 这样可以立即解决由于材料类型设置不当而引起的一系列可能的问题-在准备打印的过程中或更换材料后的重新调整过程中。 现在，将细丝装入打印机后，在准备打印作业时，无需分别设置温度，进纸速度等。 -为每种材料准备好一个配置文件并将其保存在打印机中后，就可以使用任何类似的材料进行打印，而不会浪费时间准备新任务。



网络打印



网络不仅仅是从打印机远程接收警报的能力。 此功能使您可以将多个Designer X PRO组合为一个结构，并使用更新的Polygon X程序从一台计算机进行管理，监视打印，跟踪已安装的材料，创建作业队列，将它们分配给设备，安排工作时间表并跟踪来自所有设备的警报。一个单一的界面-现在所有这些都可以快速完成，而无需离开工作场所。



采用喷气开关技术的挤出机


由Jet Switch技术驱动的最新挤出机是具有两个喷嘴的挤出机，使您可以用两种材料进行打印，并立即停止进给其中一种而不降低温度。 由于内部结构的变化，经过改进的新型挤出机具有较高的印刷质量，提高的生产率和抗堵塞性。
