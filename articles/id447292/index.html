14 produk baru di Visual Studio 2019

Halo semuanya! Dalam sentuhan tim untuk meningkatkan produktivitas di .NET (alias. Roslyn). Kami membayar banyak perhatian untuk meningkatkan pengalaman pengkodean .NET Anda. Memang, terkadang penambahan dan koreksi kecil benar-benar dapat meningkatkan alur kerja. Mungkin Anda telah melihat beberapa peningkatan dalam versi pratinjau VS2019. Tetapi kami masih ingin membicarakan beberapa di antaranya yang seharusnya membantu Anda bekerja lebih produktif.

Perhatian, banyak gif.



Perangkat tambahan


Bagaimana Anda menyukai warna-warna baru dari Roslyn? Warna Visual Studio Code diberi peringkat tinggi, jadi kami memasukkan skema warna serupa di Visual Studio. Editor kode Anda sekarang sedikit lebih berwarna. Kata kunci, metode pengguna, variabel lokal, nama parameter, dan operator kelebihan beban telah menerima warna baru. Anda bahkan dapat menyesuaikan warna untuk setiap klasifikasi sintaks dengan pergi ke Alat> Opsi> Lingkungan> Font dan Warna dan gulir ke "Anggota Pengguna".



Di bagian bawah file, di editor Anda, ada indikator kesehatan dokumen, serta ikon untuk menghapus kode. Indikator kesehatan dokumen memungkinkan Anda untuk segera mengetahui berapa banyak kesalahan dan peringatan yang ada dalam file yang baru saja Anda buka. Anda dapat mengklik ikon kode yang jelas untuk menerapkan aturan gaya kode yang ditentukan dalam Alat> Opsi, atau jika Anda memiliki file konfigurasi editor yang memerintahkan Anda untuk menggunakan gaya kode yang sama dalam perintah Anda, VS akan menerapkan gaya yang ditentukan dalam file ini.



Anda dapat mengedit file proyek seperti SDK dengan klik ganda sederhana. Anda juga dapat melihat file-file ini sebagai pratinjau dalam navigasi GoToAll (Ctrl + t) dan mencari konten untuk tautan file.



Unggah sebagian proyek ke solusi Anda dengan solusi yang difilter ! Sekarang Anda dapat mengunggah proyek dan menyimpan file .slnf, yang hanya akan membuka proyek yang Anda tentukan. Ini akan membantu Anda dengan cepat mendapatkan kode yang Anda minati tanpa harus mengunduh seluruh solusi.



Temukan semua tautan yang diklasifikasikan berdasarkan jenis tautan. Anda dapat memfilter dengan membaca / menulis di kolom "Jenis" baru di kotak pencarian untuk semua tautan.



Jalankan pemformatan gaya kode di seluruh solusi pada baris perintah menggunakan alat format dotnet global.



Intellicode adalah ekstensi yang menawarkan petunjuk cerdas berkat model pembelajaran mesin terlatih yang bekerja di lebih dari 2000 repositori .NET sumber terbuka di GitHub.
