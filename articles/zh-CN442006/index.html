选择对象


一切都始于对象的选择。 在带有选择的Windows File Explorer中，一切自然都是不好的-选择许多对象很困难，而且不会偶然丢失选择。 带有复选框的选择选项没有特别帮助。 Google云端硬盘几乎是相同的垃圾，但是至少您不能通过在图块之间随机单击来丢失选择。


我认为应该一键打开对象（毕竟，可能需要单击太多以至于手指受伤），然后长按（例如在电话上）或使用选择图标（主要是手推车）突出显示一个对象即可激活选择模式。 这是我的选择，请查看标题：
