Der Google-Ingenieur testet USB-Typ-C-Kabel und veröffentlicht Berichte bei Amazon



Einige Leute denken, dass ein Kabel eine einfache Sache ist, fast alle Kabel funktionieren gleich und können keinen Schaden anrichten. Bei USB-Typ-C-Konverterkabeln ist dies jedoch nicht der Fall. Dies geht aus dem Test hervor, den Benson Leung, Ingenieur bei Chrome OS, Chromebook Pixel und Pixel C bei Google, durchgeführt hat.

Unter anderem prüft Benson eine Vielzahl von USB-Typ-C-Konverterkabeln auf Kompatibilität mit Pixel-Laptops, Nexus 5X- und 6P-Smartphones und im Allgemeinen auf Einhaltung der USB-Spezifikationen. Was am wertvollsten ist, er verbirgt die Ergebnisse technischer Tests nicht, sondern veröffentlicht sie direkt auf der Amazon-Website, auf der Kabel verkauft werden.

Bis heute hat Benson über ein Dutzend Bewertungen von Kabelkonvertern verschiedener Hersteller veröffentlicht: Frieq, CableCreation, Belkin, Monba, Kupx, iOrange, Juiced Systems, Orzly und Techmatte. Die Ergebnisse sind, gelinde gesagt, widersprüchlich.

Es stellte sich heraus, dass von allen Kabeln nur Frieq-, Belkin- und iOrange-Kabel die USB-Spezifikationen erfüllen. Der Rest hat mindestens eines von zwei schwerwiegenden Problemen. Zum Beispiel beanspruchen CableCreation-Adapter Unterstützung für einen Strom von 3 Ampere, aber in Wirklichkeit unterstützen die Konverterkomponenten solche Parameter nicht. Anstelle eines 56-kΩ-Widerstands verfügt er über einen 10-kΩ-Widerstand, der nur mit CC-Kabeln (beidseitig Typ C) verwendet werden kann. Wenn Sie ein Gerät anschließen, das 3 Ampere ziehen kann, kann dies katastrophale Folgen haben. Kupx-, Techmatte- und Monba-Adapter haben das gleiche Problem.

Pixel-Laptops können über USB Typ C mit 3 Ampere aufgeladen werden, daher ist das Problem von Kabeln geringer Qualität für sie sehr wichtig.

Benson erklärtDie Anforderung, in den Spezifikationen einen 56-kΩ-Widerstand zu verwenden, ist so festgelegt, dass alte Geräte, einschließlich USB Typ A, die auf der anderen Seite des Kabels angeschlossen sind, nicht beschädigt werden, wenn sie an den USB-Typ C-Anschluss angeschlossen sind und 3 Ampere ziehen. Dies könnte das Ladegerät, den Computer oder andere Geräte beschädigen.

Das Traurigste ist, dass das Kabel lange Zeit normal arbeiten kann, ohne dass es Mängel aufweist. Wenn es jedoch an bestimmte Geräte angeschlossen wird, kann es einfach durchbrennen, wie bei diesem Anker-Konverter.


In seinem Blog veröffentlichte Benson Anweisungen zum Überprüfen der Spezifikationen von Kabelkonvertern von USB Typ A nach Typ C oder von USB Typ B nach Typ C (wenn Sie ein Chromebook Pixel 2015 haben).

Sie müssen den Laptop im Debug-Modus (Dev-Modus) laden und den Adapter an eine Art Ladegerät anschließen, was nicht schade ist.

Führen Sie dann den Befehl über die Konsole aus:

ectool --name=cros_pd usbpdpower 0

Gutes Kabel:

localhost ~ # ectool --name=cros_pd usbpdpower  
Port 0: SNK Charger DCP 4920mV / 2000mA, max 5000mV / 2000mA / 10000mW
Port 1: Disconnected

Schlechtes Kabel:

localhost ~ # ectool --name=cros_pd usbpdpower
Port 0: SNK Charger Type-C 5147mV / 3000mA, max 5000mV / 3000mA / 15000mW
Port 1: Disconnected

Im zweiten Fall wird Typ C mit einem maximalen Strom von 3000 mA angezeigt, während das Kabel an den Typ A-Anschluss angeschlossen ist, was auf falsche Widerstandsparameter hinweist.

Es gibt noch eine andere Testoption: Gehen Sie im normalen Startmodus zu chrome://systemund dmesgsuchen Sie im Abschnitt nach einer Zeile wie dieser:

[  121.172637] PDLOG 2015/11/04 19:58:18.074 P1 SNK Charger Type-C 5000mV max 5000mV / 3000mA
