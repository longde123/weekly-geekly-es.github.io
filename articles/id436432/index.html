SAP: Apa yang Anda butuhkan untuk mulai belajar SAPUI5?


Pendahuluan


Pada awal September 2018, beberapa orang keren juga bergabung dengan tim kami, mereka masih baru dari Universitas dan mereka benar-benar lapar untuk belajar bagaimana merancang dan mengembangkan aplikasi web yang luar biasa dengan SAPUI5.


Itu sebabnya saya mulai mengumpulkan secara internal di web beberapa tautan untuk menciptakan "Perjalanan sempurna untuk menjadi Pengembang Ninja SAPUI5".


Saya juga mulai menulis beberapa latihan (dari gampang susah) untuk menguji apa yang telah mereka pelajari tetapi saya akan membagikannya di posting blog kedua segera setelah saya menyelesaikannya.


Presentasi


Hai semuanya,
Saya Emanuele Ricci, seorang pengembang penuh yang berbasis di Lucca (sebuah kota kecil yang indah di Tuscany, Italia).


Sejak tiga tahun terakhir, saya bekerja penuh waktu untuk Techedge Group , perusahaan konsultan besar dunia yang bermitra dengan SAP. Saya biasanya bekerja di proyek yang terkait dengan SAPUI5, SCP, HANA dan di waktu luang saya, saya suka membuat konten seputar teknologi yang saya gunakan di tempat kerja dan dalam proyek pribadi saya di luar SAP. Akhir-akhir ini, saya sedikit bereksperimen dengan Android setelah rilis SAP Fiori SDK untuk Android / iOS.


Selama tiga tahun karir saya di lingkungan SAP, saya telah mendapatkan beberapa prestasi seperti:

  • Pembicara di berbagai acara seperti UI5 Con 2017, UI5 Con 2018, SAP Inside Track Rome 2018

Di mana saya mulai mengikuti “Perjalanan sempurna untuk menjadi Pengembang Ninja SAPUI5”


SAPUI5 adalah kerangka kerja MVC (Model-View-Controller) yang dikembangkan oleh SAP untuk membuat aplikasi web tingkat perusahaan yang luar biasa.


OpenUI5 adalah versi open-source SAPUI5. Lebih tepatnya, SAPUI5 hanya memperluas OpenUI5 menambahkan beberapa tambahan UI-Control seperti Viz (grafik) dan FilterBar tetapi pada akhirnya, Anda dapat melakukan hampir semua hal hanya dengan OpenUI5.


Dokumentasi dan tautan bermanfaat


  • Dokumentasi : berbagai dokumentasi tentang berbagai topik. Langkah pertama yang harus Anda lakukan adalah mengikuti keseluruhan Walkthrough yang akan menunjukkan kepada Anda cara membuat aplikasi SAPUI5 dari A hingga Z.
  • Buka Kursus SAP : di sini Anda akan menemukan banyak kursus gratis yang dibuat oleh pengembang SAP pada banyak topik. Untuk pengembangan SAPUI5, saya sarankan Anda Mengembangkan Aplikasi Web dengan SAPUI5 .
  • Referensi API : di sini Anda akan menemukan informasi yang berguna tentang model, kontrol dengan perincian tentang sifat, peristiwa, metode, agregasi dan kelas mana yang diperluas
  • Sampel : mereka penting untuk dilihat dalam tindakan setiap kontrol UI. Untuk setiap kontrol, ini akan menunjukkan kepada Anda beberapa skenario penggunaan nyata dan cara mengkonfigurasinya untuk mendapatkan hasil yang sama. Mengklik ikon kanan-atas contoh akan memungkinkan Anda untuk mengakses kode sumber contoh
  • Build.me : ini adalah layanan yang dikembangkan oleh SAP yang menawarkan seperangkat alat berbasis cloud untuk merancang dan membangun aplikasi perusahaan Anda, dari prototipe kesetiaan rendah hingga kesetiaan tinggi
  • SAP Cloud Platform Trial (untuk pengembang) : adalah platform-as-a-service (PaaS) platform terbuka yang menghadirkan kemampuan dalam memori, layanan platform inti, dan layanan microser unik untuk membangun dan memperluas aplikasi cloud yang cerdas dan mendukung seluler. Anda dapat mendaftarkan akun uji coba gratis di https://account.hanatrial.ondemand.com/
  • SCN : Jaringan komunitas SAP, ini adalah situs komunitas utama tempat Anda dapat menemukan posting Blog dan QA yang dibuat oleh pengembang, mentor, atau pengguna SAP (pikirkan seperti StackOverflow).
  • OpenUI5 Slack : ini adalah saluran Slack tidak resmi kami tempat Anda dapat mendiskusikan pembicaraan tentang SAPUI5 dan topik lain seperti WebIDE, HANA, kontrol khusus, dan sebagainya. Di sini Anda juga akan menemukan pengembang inti SAP untuk langsung diajak bicara.
  • OpenUI5 GitHub repo : ini adalah repositori GitHub resmi OpenUI5 (dalam repo SAP Anda juga akan menemukan proyek menarik lainnya seperti alat build baru atau aplikasi sampel)
  • OpenUI5.org : ini adalah situs aplikasi OpenUI5 resmi
  • UI5Con : ini adalah acara utama tentang OpenUI5 (2017/2018) di mana Anda dapat menemukan semua sesi pembicaraan / langsung yang dibuat dengan tautan ke slide, video, dan contoh.
  • UI5Lab : situs ini dibuat oleh anggota komunitas untuk anggota komunitas;) Di sinilah kami mengembangkan dan berbagi pustaka dan kontrol kustom baru untuk mengajar pendatang baru.

Pada titik ini, Anda harus memiliki konten yang cukup untuk memulai;)
