放大照片） Ken Shirriff的微处理器和充电器爱好者在电子发烧友社区中广为人知。他已经出版了的分析以及详细说明报告微小的充电器为iPhone，其他十几个充电器，其中一个很大的苹果产品甚至不是最好的。在2013年，他从自己的照片（这是Osborne 1，TRS-80和Sinclair ZX Spectrum的处理器）中的Z80处理器中进行了ALU逆向工程。

现在，Sherriff 关注了历史悠久的处理器Intel 8008-英特尔于1972年4月1日发布的第一款8位中央处理器，即将近45年前，价格为120美元。英特尔8008芯片专门用于高级计算器，但最终在第一台个人计算机中占有一席之地。

至于说维基百科，首次，英特尔是怕处理器不感兴趣的客户，但这些担心被证明是毫无根据的。处理器正在等待巨大的成功。之后，该公司发布了英特尔8080，然后发布了非常成功的英特尔x86系列。


芯片Intel 8008

基于Intel 8008的客户不仅开始收集计算器，还开始收集小型计算机。最早的商用个人计算机之一，包括美国的SCELBI，法国的Micral N和加拿大的MCM / 70，都在此处理器上工作。Sac State 8008也许是第一台真正的具有磁盘操作系统和基于PROM内置于IBM PROM中的IBM Basic Assembly Language（BAL）的微型计算机。设计于1972-1973年。这台微型计算机组装起来可能只有一个副本，配备了彩色显示屏，硬盘驱动器，键盘，调制解调器，磁带阅读器和打印机，开发人员为英特尔提供了宝贵的帮助，以帮助他们编写有关未来英特尔8080的说明。早在1973年，萨克州立8008从1975年起，Altair 8800的功能就被超越了，它为史蒂夫·乔布斯，史蒂夫·沃兹尼亚克和世界各地的许多发烧友提供了商业灵感和想法。


Sac State 8008的Art 3D渲染

从历史证据来看，Sac State 8008可以称为世界上第一台真正的个人计算机。

Ken Shirriff发布了详细分析传说中的8008处理器上有很多照片，就像一个发烧友自己打开外壳并在显微镜下拍摄了晶体的照片，您甚至可以在其中看到芯片上的触点和晶体管。芯片部分的下方放大了。



沿着微电路的周边，可以看到18个接触垫，这些接触垫通过细线与外部接触针（微电路的引脚）相连。拍摄整个晶体需要48帧。肯·希尔里夫（Ken Shirriff）使用了照亮的金相显微镜。

然后，专家使用Hugin软件粘合48个框架。为了获得美丽的高分辨率照片，他仍然扭曲了对比度。为了进行比较，这是原始照片的外观，大致与您用光学显微镜实际看到的相符。显微镜下的Intel 8008微处理器晶体的照片（请参见高分辨率照片4730×3382） 从处理器文档中，您可以准确找到特定功能区域在芯片上的位置。它们均在下图中签名。左侧是进行计算的算术逻辑单元（ALU）。








ALU使用两个临时寄存器来存储输入数据。这些寄存器在芯片上占据了很大的面积。不是因为它们很复杂，而是因为需要大型晶体管来通过ALU电路传输信号。

ALU的三角形设计看起来也不寻常。在大多数处理器上，每个位的电路都以矩形块的形式排列。但是，在8008中，八个块（每个位一个）以不稳定的方式分布在三角形区域中，以适合三角形进位生成器留给它们的区域。

该芯片的物理结构与《英特尔8008用户指南》中的框图非常吻合。芯片上的模块与电路上的模块几乎位于同一位置。



工程师提请注意以下事实：专家们没有解释为什么英特尔为这种微电路使用明显不足的18脚数量（地址的14位和数据的8位），因为这种非标准的总线架构，我不得不为此处理器使用很多其他电子设备。他说16个联系人实际上是“英特尔的宗教”，但由于对总线体系结构的棘手操作，设计人员设法将分支的数量减少到仅18个。
