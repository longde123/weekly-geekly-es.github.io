Leukemia limfoblastik akut, kisah saya dengan akhir yang bahagia

Pada waktu itu saya tinggal di desa Fevralsk, dan belajar di sekolah menengah No. 3. Pada musim panas 1992, saya bekerja di lokasi konstruksi (ayah saya mengatur) selama 1 bulan, saya belajar di kelas 9. Saya bekerja selama 4 jam, tetapi ada beberapa kelelahan, tepat setelah makan siang saya tertidur.

Pada musim gugur, sekitar bulan November, saya masuk angin, itu terjadi agak tidak terduga, karena Saya biasanya tidak merasa sakit, dan sampai saat itu, saya tidak pernah bolos sekolah karena sakit. Nodus limfa submandibular entah bagaimana meningkat tajam, dan itu meningkat sehingga bahkan tidak jelas bahwa itu dia, karena leher di satu sisi telah meningkat sangat banyak.

Pada awalnya ada kecurigaan gondong, tetapi di bangsal menular, saya ditahan hanya beberapa jam dan dikirim ke bangsal terapi. Keesokan harinya, terapis memutuskan untuk mengambil tusukan dari leher yang bengkak. Tusukan itu tidak terlalu berhasil, saya hampir pingsan, dan tusukan itu juga tidak menunjukkan apa-apa. kap tidak bisa dibuat. Hmmm, meskipun dia masih menunjukkan bahwa ini bukan abses. Pada saat yang sama, sinusitis didiagnosis untuk saya, sinus saya hilang dan pengobatan antibiotik diresepkan. Saya dengan cepat menjadi lebih baik, bahkan peradangan pada kelenjar getah bening submandibular berkurang. Seminggu kemudian mereka akan membebaskan saya, tetapi sesuatu mengatakan kepada saya untuk melakukan tes darah lengkap. Tetapi saya harus mengatakan bahwa di rumah sakit desa kami pada saat itu benar-benar dicairkan dan tes darah tidak dapat dilakukan.

Saya meminta ayah saya untuk dirawat di rumah sakit unit militer, ayah saya setuju dengan militer dan saya dirawat di rumah sakit. Terapis pria itu ternyata sangat serius, segera melakukan semua tes dan bertanya dengan baik tentang kondisi kesehatan saya, dll. Secara umum, saya langsung mendapat semacam kepercayaan padanya. Keesokan harinya saya diperiksa oleh kepala rumah sakit, bersama dengan terapis. Mereka berbicara tentang sesuatu yang tidak dapat dimengerti, mereka berbicara kepada saya, kemudian saya hanya ingat ungkapan bahwa tidak ada pertanyaan tentang perawatan untuk limpa di rumah sakit.

Sehari kemudian, mereka memanggil ayah saya untuk berbicara, menjelaskan kepadanya bahwa tes darah sangat buruk dan saya harus dikirim ke rumah sakit daerah. Mereka memberikan semua koordinat dokter dan arahan, dengan rekomendasi yang kuat, mereka akan pergi sesegera mungkin. Saya sudah menulis dengan perasaan bahwa saya sudah pulih dan memperlakukan perjalanan ke Blagoveshchensk sebagai semacam petualangan, saya tidak tahu bahwa semua ini akan berlangsung selama 5 tahun.

Di Blagoveshchensk, saya berakhir di departemen hematologi, pertama kali Sergey Nikolaevich Gorodovich memimpin saya, kemudian Valery Vladimirovich Vaytsekhovsky memimpin saya, dan kemudian (ketika mereka menyadari bahwa remisinya stabil) mereka memberi saya perawatan hematologi muda.
Sebenarnya, saya langsung didiagnosis menderita leukemia limfoblastik akut ALL (kanker darah umum).

Saya segera diresepkan pengobatan menurut rejimen D. Hoelzer , mungkin beberapa obat lain diberikan, tetapi entah bagaimana saya ingat bahwa dengan obat-obatan inilah saya terutama dirawat. Omong-omong, protokol mengacu pada iradiasi kepala, mereka tidak melakukannya kepada saya, yah, beberapa prosedur lain yang tertulis dalam protokol.

Dalam sekitar 1,5 bulan saya dibawa keluar untuk remisi, selama waktu ini saya pulih 5-10 kg, striae muncul , tetapi secara keseluruhan kesehatan saya tidak buruk. Saya mulai belajar buku teks sendiri. Saya lupa mengatakan, setelah penetes cytosar, rambut saya mulai rontok dengan cepat. Para perawat memotong rambut saya di kepala saya yang botak, dan rambut saya mulai menipis seperti bayi :).

Anda tahu, dari perasaan secara umum, situasi di departemen hematologi pada waktu itu agak tulus, banyak pasien sudah saling kenal selama lebih dari setahun. Dokter dipercaya dan umumnya dibenarkan. Tentu saja, hal tersulit adalah melihat teman-teman mati, saya telah melihat kasus seperti itu lebih dari sekali, saya melihat bagaimana dokter yang merawat khawatir tentang pasien mereka. Setiap kasus ketika saya datang dan mengetahui bahwa salah satu teman saya telah meninggal, saya sangat dekat.

Setelah saya dibawa keluar untuk remisi, mereka memutuskan untuk mengirim saya untuk pemeriksaan lebih lanjut ke Moskow, dan saya dikirim sebagai petugas penerbangan. Di Moskow, saya tidur untuk hari pertama di koridor, karena semua bangsal ditempati, dokter yang hadir (di Moskow) sangat terkejut dengan analisis saya, dan rujukan saya ke departemen metropolitan, karena Menurutnya, saya memiliki tes yang sangat bagus dan saya tidak boleh datang sama sekali.

Mereka tidak menahan saya di Moskow lama, di sana mereka juga memberi saya tusukan pinggang pertama, obat-obatan (metotreksat) tidak diberikan kepada saya saat itu. Mereka mulai melakukan ini nanti, tetapi seperti yang saya pahami sekarang murni dari alasan pencegahan. Banyak yang takut tusukan lumbal, saya hanya bisa mengatakan bahwa mereka melakukan prosedur ini untuk saya 20-30 kali. Saya tidak mendapatkan sensasi yang sangat tidak menyenangkan, tetapi itu agak tidak menyenangkan, yah, jembatan setelah prosedur berulang ini mulai diberikan kepada saya dengan susah payah, tetapi sekarang tidak berhasil lagi.

Saya sudah meninggalkan Moskow dengan kesadaran penuh bahwa sepertinya sudah lama. Tahun pertama saya menghabiskan sebagian besar di rumah sakit, kursus perawatan adalah skema Hoelzer dalam versi yang sedikit dipangkas, seperti yang saya katakan. Saya menghabiskan sekitar 1,5-2 bulan di rumah sakit, dan kemudian mereka membiarkan saya pulang dan saya mengambil terapi suportif sendiri, tergantung pada jumlah sel darah putih dalam darah. Sebenarnya, pada awalnya saya pergi ke terapis untuk memberi tahu saya berapa banyak pil yang harus saya minum, tetapi kemudian dia dan saya bosan, dan saya menerima tes secara langsung di laboratorium dan mengambil pil sesuai dengan skema yang saya berikan.

Pada tahun 1994, saya memasuki 3 institut, saya mengikuti ujian sekolah di desa Urgal (bukan di sekolah saya), di mana ujian ini sama saja dengan penerimaan masuk ke FESU, serta saya memasuki DalGAU dan AmSU. Saya tidak diterima oleh FGUPS karena alasan kesehatan, karena departemen Bridges and Tunnels, yang punya cukup poin untuk saya, tidak menerima orang dengan penyakit darah. Saya punya pilihan, antara DalGAU dan AmSU diputuskan untuk pergi ke DalGAU karena Institut Konstruksi yang saya masuki berada di seberang jalan dari departemen hematologi tempat saya dirawat. Belakangan, kebenarannya, rumah sakit regional bergerak lebih dekat ke AmSU. Tetapi pada saat ini, saya bahkan melalui skema yang telah saya lakukan di rumah sakit di rumah sakit hari.

Jadi, saya dirawat selama 5 tahun, sekarang saya tinggal dan bekerja di St. Petersburg, saya memiliki seorang putra yang sudah berusia 6 tahun. Saya tidak terdaftar dengan ahli hematologi, dan saya bahkan tidak pergi ke rumah sakit karena tidak ada alasan.

Kesehatan untuk semua orang !!!
