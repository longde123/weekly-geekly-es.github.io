摩托罗拉Moto C Plus评测-崭新亮点

朋友您好！ 今天，我们将讨论充满活力的Moto C Plus智能手机。 新颖的设计采用了公认的Moto风格，由于其圆形的形状和名称，C Plus可以与Moto另一款简单且非常受欢迎的手机-C350进行比较。

同时，Moto C Plus的价格是可以承受的，在莫斯科的商店中约为8000卢布。 价格标签很小，这意味着电话上的竞争将是巨大的，这反过来又迫使新颖性变得有趣和有吸引力。

包装是带有两个冰淇淋的友好蓝色。 设计简单，但不浮夸：C Plus被认为是经典的主力马，而不是旗舰型号。





C Plus系列的所有智能手机都具有相同的鲜艳颜色：金色，樱桃金属色，当然还有黑色。 最后一个选项非常适合经典爱好者。



交付套件非常简单：电缆，充电（2安培-您也可以为平板电脑充电）和耳机。 事实证明，后者是相当不错的-尽管尽管它们不是时髦的“笑脸”，但对于智能手机的价格类别而言，声音质量还是令人愉悦的。 带有麦克风的遥控器将永远是多余的。 简而言之，C Plus框包含您需要的所有内容，但没有多余的装饰或不必要的限制。 尽管C Plus属于预算解决方案，但包装盒中没有一根电缆。



电话原来是非常有趣的颜色-金色。 欢迎来到奢华与魅力的世界:)实际上，颜色虽然是金色的，但看起来并不庸俗。 这种颜色的模型很适合女孩和年轻人。 好吧，对于那些想要一些平静的东西的人，可以使用黑色模型。



在正面没有硬件按钮：只有触摸按钮。 这是搭载Android的智能手机的常规三组标准键。 通常，使用虚拟的屏幕按钮来实现这种级别控制的设备。 好吧，这里都是成年人。



当然，大部分正面都被对角线为5英寸的大型显示器占据。 分辨率为1280 x 720像素。 给定标记了C Plus的分段，这已经绰绰有余。 如果您不遵循规格而仅是简单的用户意见，则可以放心地说，使用屏幕不会感到不适。 图片将是多汁的，没有令人不愉快的“颗粒”。 显示屏足够明亮，尤其不会眩光，对触摸的响应稳定并且不会在阳光下褪色。 如果愿意，您可以观看YouTube上的简短视频，甚至是整部电影。



照相机，闪光灯和当然还有说话的扬声器摆在显示屏上方。 前置摄像头很有趣：2MP分辨率，LED闪光灯和HDR。 也许仅缺少自动对焦，但这是另一个价格类别。



设备的背面也呈金色。 盖子是塑料的，易于拆卸。 底部是一个扬声器。 顺便说一句，很大声。



现在介绍一下主摄像头。 当然，在这里，所有事情都比前端更有趣。 分辨率提高了8Mr，自动对焦和LED闪光灯。 如果所有者愿意，相机将能够拍摄全景照片，并且将它们粘合在一起也不错。 至于视频，它更有可能出现。 最大记录分辨率为1280 x 720（每秒30帧）。 宁可少玩又有趣：视频问候等等，就用youtube cat拍摄。



尽管具有触摸控制，智能手机还是具有机械按钮：音量控制和开/关。 这些按钮在右侧-您可以用右手的拇指轻松触及它们。 左侧没有控件。



现在打开手机。 盖子下方有一块可拆卸电池，一个用于MicroSD存储卡的插槽和两个用于SIM卡的插槽。 根据官方技术规格，支持的最大卡数量为32 GB。 C Plus可能也识别较大的卡。 具有容量的可拆卸电池是Moto C Plus的另一个不错的组成部分。 如果随着时间的推移您需要更换它，那么这将不是问题。 该过程将花费不到一分钟的时间，并且您无需将智能手机随身携带到服务中。



现在介绍一下SIM卡。 Moto C Plus可以同时使用两张SIM卡，但是3G / 4G模式仅由一个插槽支持。 这是双重模拟的标准做法。 因此，通过Internet安装SIM卡-不要弄错连接器。





顺便说一句，电池非常好-4000 mAh。 没错。 这不是一款高端时尚的智能手机，它具有许多功能，并且需要凉爽而纤薄的设计。 最好在厚度上增加几毫米，但要确保智能手机一次充电可以工作超过一天。 顺便说一下，Moto C Plus的厚度恰好是1厘米（10毫米），不是很多。


技术指标


智能手机正在运行最新的操作系统Android 7.0。 整个过程都在四核联发科MT6737处理器（取代MT6735）的容量上。 SoC是根据28 nm工艺的标准生产的，并带有4个以1300 MHz频率运行的Cortex-A53内核。 图形处理器的作用是ARM Mali-T720 MP2。

RAM的大小为1 GB，常量为16 GB。 可通过microSD卡进行扩展。 但是Moto C Plus对蜂窝网络的支持感到高兴。 智能手机支持4G LTE Cat4网络（通道1、3、5、7、8、20、38、40）以及蓝牙4.2（A2DP，LE）和Wi-Fi。 后者甚至可以通过两个钢筋混凝土墙轻松捕获来自接入点的信号。 已检查。

Moto C Plus的界面是最好的-纯Andoird。 没有装饰会降低系统速度。 好吧，如果用户想带来美感，那么Play.Market商店将为您提供各种选择，包括免费的各种贝壳。 每种口味和颜色都叫什么。

相机可以拍摄非常好的照片。 当然，这需要阳光或大量的光线，但是她仍然知道如何做。 确实，有一个警告。 相机对光非常敏感。 即 您需要选择合适的光源，否则照片将褪色。 尽管如此，对于8000卢布的智能手机，这还是非常不错的图片质量。 即使是非常复杂的物体（如自然物）也能很好地显示出来-叶子和草不会变成坚实的绿色混乱。













Moto C Plus的总体印象


事实证明，Moto C Plus是缺少从天而降的恒星的主力机，但同时可以应付其主要任务。 Moto C Plus支持4G网络，并能够安装两个SIM卡，一个5英寸的屏幕和一个四核处理器。 对于正常操作，这样的设置已绰绰有余。

当然，Moto C Plus根本不是旗舰产品，但价格便宜4-5倍。 同时，智能手机收到了用于同类设备的薄壳（1厘米厚），明亮的屏幕（分辨率为1280 x 720像素），不会在阳光下刺眼，以及可拆卸电池，容量为4,000 mAh。 结果是：电池寿命长。 可靠的预算智能手机还需要什么？
