Gestion de la messagerie vocale et de l'enregistrement des appels dans le PBX 3CX

Messagerie vocale du groupe 3CX


Le PBX 3CX v15.5 vous permet de laisser un message vocal «général» à un groupe d'utilisateurs inclus dans la file d'attente de groupe ou d'appel. Désormais, dans la règle de transfert d'appel Connectez-vous à la messagerie vocale de l'utilisateur, le choix des files d'attente ou des groupes est disponible - chaque file d'attente / groupe a une boîte vocale individuelle. Dans le même temps, tous les utilisateurs du groupe reçoivent une notification qui inclut le fichier audio du message vocal. Bien entendu, les paramètres utilisateur doivent avoir une adresse e-mail.

Messagerie vocale de groupe dans le menu vocal / RVI




Le transfert vers une zone de groupe à partir du menu vocal est très pratique si vous devez enregistrer en toute sécurité les appels des clients pendant les heures creuses. Dans ce cas, une boîte vocale vocale de groupe est préférable, car non seulement un employé dédié qui peut ne pas être sur le lieu de travail, mais aussi un autre employé du groupe peut traiter l'appel.

Messagerie vocale de groupe dans les files d'attente de groupe et d'appel




Dans le groupe d'appels, nous pouvons simplement configurer une boîte vocale commune pour tous les membres du groupe. Par exemple, si un appel arrive alors que tous les ingénieurs du support technique sont occupés, ils recevront un message vocal enregistré de l'utilisateur.

Veuillez noter que dans la file d'attente des appels, une notification est envoyée aux opérateurs de file d'attente et aux gestionnaires de file d'attente, qui peuvent ne pas être des agents.

Messagerie vocale de groupe




La configuration de la zone de groupe d'un utilisateur est pratique lorsqu'il est nécessaire «d'assurer» un employé important. Par exemple, si le responsable de l'entrepôt s'est éloigné et ne peut pas accepter l'appel, une notification vocale est reçue par tous ses subordonnés.

En plus des règles de transfert, vous pouvez laisser un message vocal au groupe en entrant le code de composition * 4 + numéro de groupe sur le téléphone. Par exemple, * 480 - laissera un message vocal pour tous les membres du groupe 80 Warehouse dans la capture d'écran ci-dessus.

Caractéristiques des notifications de groupe


Lors de l'entrée dans une boîte vocale de groupe, l'abonné entendra le message système 3CX standard pour la messagerie vocale dans la langue définie par défaut sur 3CX. Cette règle ne s'applique pas aux files d'attente d'appels - le message est lu dans la langue définie pour cette file d'attente.



De plus, les restrictions suivantes s'appliquent à la messagerie vocale de groupe:

  • La notification MWI n'apparaît pas sur les téléphones IP des membres du groupe / de la file d'attente (l'indicateur de message vocal ne clignote pas)
  • la messagerie vocale ne s'affiche pas dans les clients 3CX, y compris le client Web
  • les paramètres de messagerie vocale de chaque utilisateur sont ignorés et les paramètres suivants sont utilisés: enregistrement des messages - envoi de pièces jointes aux participants - suppression du fichier de messagerie vocale sur le serveur

Gestion des enregistrements de conversation 3CX


3CX v15.5 met en œuvre plusieurs technologies importantes qui rendent la gestion des enregistrements d'appels, y compris la messagerie vocale, aussi pratique et fonctionnelle que possible. Vous pouvez gérer les enregistrements à partir de l'interface 3CX: écouter, supprimer, définir des quotas et définir la période de suppression des anciens enregistrements. De plus, dans les dernières mises à jour, vous pouvez choisir les appels à enregistrer et le format de stockage des enregistrements.

Publier des conversations


Les enregistrements de conversations ne peuvent être stockés que sur le disque local du serveur. Cela fournit un PBX haute performance tout en conservant plusieurs enregistrements. L'emplacement de stockage des enregistrements est défini dans l'interface 3CX (cependant, vous ne pouvez pas réaffecter l'emplacement de stockage de la messagerie vocale). N'indiquez pas ici la racine du disque!



Après avoir sélectionné un enregistrement, vous pouvez le télécharger pour l'écouter ou le supprimer.

Quotas et politiques de conservation des enregistrements de conversations


En cliquant sur le bouton Quota, vous serez redirigé vers l'interface des paramètres de quota utilisateur et des politiques de conservation des enregistrements. Dans les dernières mises à jour, vous pouvez également spécifier un nouveau format de compression des enregistrements ADPCM, mais vous ne pouvez les écouter que dans le lecteur, mais pas dans le navigateur. Le format d'enregistrement standard prend 1 Mo pour 1 minute de conversation, tandis que le format ADPCM ne prend que 256 Ko.


Lorsque le quota est atteint, l'enregistrement des conversations utilisateur est terminé (pour le moment, seule la valeur totale du quota pour tous les utilisateurs peut être définie). Il est recommandé d'activer l'administrateur dans les paramètres du PBX afin qu'il puisse agir. En outre, l'enregistrement de conversation s'arrête toujours s'il reste moins de 500 Mo d'espace libre sur le disque du serveur. Ceci est fait pour ne pas perdre l'accès à l'interface de contrôle PBX en raison d'un débordement de disque.

Sélectionnez les conversations à enregistrer


À partir de 3CX v15.5 Update 5, vous pouvez définir les conversations utilisateur à enregistrer: toutes ou uniquement externes.


Dans de nombreuses organisations, il est logique d'enregistrer uniquement les conversations des utilisateurs avec des abonnés externes, par exemple, avec des clients ou des partenaires. Les négociations avec des collègues peuvent ne pas être si importantes, vous ne devez donc pas occuper un espace disque précieux avec ces enregistrements, en particulier si le PBX est hébergé sur un hébergement cloud.

Dans l'édition 3CX Enterprise, vous pouvez également autoriser l'utilisateur à désactiver (ou activer) indépendamment l'enregistrement d'une conversation en appuyant sur un bouton du téléphone IP.

Afficher les enregistrements d'appels dans 3CX Web Client


On nous pose souvent la question: comment puis-je voir mes propres enregistrements de conversations dans le client Web 3CX?



Pour le moment, ce paramètre se trouve dans les paramètres utilisateur, l'onglet Téléphones de réglage automatique - Client 3CX.


Vous ne pouvez autoriser l'utilisateur qu'à écouter ses propres conversations et à les supprimer. Ces paramètres s'appliquent à tous les clients 3CX: client Web et applications pour Windows, Mac, Android et iOS.
