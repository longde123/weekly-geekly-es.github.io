Chromebit convierte cualquier monitor o televisor en una computadora completa por $ 85



Asus y Google finalmente lanzaron la venta de Chromebit, una PC en miniatura basada en Chrome OS por solo $ 85. El gadget se puede conectar a cualquier puerto HDMI. Después de eso, solo queda tomar el teclado y el mouse Bluetooth, encender el botón de encendido en la pantalla, y obtenemos una computadora preparada que no puede extraer juegos pesados, pero puede usarse como una PC de oficina.

El dispositivo Chromebit es un poco Chromebook simplificado y en miniatura. Para trabajar aquí, se utilizan un navegador y aplicaciones web que le permiten realizar la mayoría de las tareas diarias, como consultar el correo, trabajar con mensajeros de mensajería instantánea, navegar y editar documentos. Como la computadora es muy pequeña, no hay panel táctil, pantalla ni batería, por lo que los desarrolladores pudieron reducir radicalmente el precio de su dispositivo.

Si necesita una PC económica o un televisor inteligente ( el sueño de una persona de 1996 ), entonces este dispositivo se adapta perfectamente. También se puede utilizar como elemento principal para crear una sala para celebrar conferencias de prensa. Cualquier habitación de hotel con un televisor más o menos normal puede convertirse en una habitación así. Bueno, Chrome OS podría ser una mejor opción para una clase, por ejemplo, que Windows XP.

Si aún desea comprar esta cosa, debe recordar que no habrá milagros de rendimiento. Se instala un chip económico de Rockchip como procesador. Además, solo hay 2 GB de la RAM no más rápida. Y, por supuesto, un monitor o televisor debe tener un puerto HDMI para conectar un dispositivo; de lo contrario, no funcionará. Por supuesto, puede probar el adaptador, pero no el hecho de que todo funcionará al 100%.



Google llama a Chromebit una excelente solución para los quioscos digitales: casi no se requiere mantenimiento, el precio de un dispositivo es mínimo para tales fines. En general, Chromebit puede realizar aproximadamente el 90% de las funciones de una PC "adulta". Es cierto que no hay demasiadas aplicaciones aquí, pero lo que hay es suficiente para funcionar.
