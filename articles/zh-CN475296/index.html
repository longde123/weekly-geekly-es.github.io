我们邀请您在11月21日参加DINS Java EVENING：我们将实际使用AWS Lambda并熟悉Akka框架

我们将很高兴见到所有对Java开发无动于衷的人，它们将在下一个开放的DINS Java EVENING中举行，该会议将在我们位于Staro-Petergofsky的19号办公室举行。

今天晚上，DINS的Alexander Gruzdev将演示基于AWS Lambda的无服务器应用程序的开发。 Yandex的Igor Shalaru将向我们介绍Akka框架以及使用它的好处。

删减部分–有关报告和演讲者的更多信息，一个注册参加会议的链接，以及上次会议的资料。



报告书


行动中的AWS Lambda（Alexander Gruzdev，DINS）

Alexander将讨论一种开发方法，该方法对于那些出于任何原因而厌倦了编写新的微服务的人以及不想为EC2的停机时间付费的人都将很感兴趣。 我们将使用具体示例来分析整个过程-从编写lambda到对其进行测试，再到部署和本地调试。 该报告适用于已经听说过AWS Lambda或有关一般无服务器方法的受众。

在过去的5年多的时间里，Alexander致力于Java开发，在更大程度上致力于从生命科学到电信的各种应用领域中的企业应用程序。 他坚持这样的想法，即开发人员应始终监视现成工具的市场，以便在大多数情况下他不编写自己的“自行车”。

“ Akka作为高负载系统的核心”（Igor Shalaru，Yandex）

Akka长期以来一直是Java开发人员的武器库。 这是用于开发应用程序的功能强大且方便的工具。 在报告的框架中，我们将分析参与者模型是什么，Akka的现成模块是什么。 通过示例，让我们看看如何在Akka上开始开发，并找出这将给我们带来什么优势。 任何级别的Java开发人员（已经熟悉Akka或只是想结识这些人员）都将对该报告感兴趣。

最近7年，Igor一直在用Java编程。 在此期间，他从一个简单的开发人员转到了Tech。 丽达 他在对负载和容错有严格要求的项目中拥有丰富的经验。 他喜欢编写可快速处理千兆字节/太字节数据的应用程序。
