该计划的亮点：2016年IFA的联想YOGA Book

转变为YOGA Book系列的笔记本电脑成为IFA 2016的真正开幕。这是为数不多的具有同时具有两个不同操作系统版本的Windows 10和Android设备之一。而触控键盘Halo Keyboard的首秀就引起了很多争议。



外表


联想专有的铰接式设计立即暗示了YOGA系列笔记本电脑。除了类似于表带外，它还可以将屏幕旋转360度，包括130个部件。铰链专为承受重载荷而设计，能够承受前后不断的折叠。

YOGA Book的机身很薄-笔记本电脑的厚度仅为9.6毫米，重量-690克。由于使用了镁和铝合金制成的外壳，这使之成为可能。外壳可能有三种颜色。Android上的YOGA Book呈现灰色和金色，Windows上的笔记本电脑-仅黑色。





特色功能


晕键盘是触敏的。是的，没有熟悉的键，没有那么多喜欢的键移动以及与传统键盘有关的所有内容。但是她知道如何做出触觉反应并且能够学习。按键背光仅在您需要键入文本时才起作用。其余时间，带有防反射涂层的哑光多功能面板（也称为键盘）可以用作绘图的触摸面板。



绘画可能性的展示令人印象深刻-这位卡通画家当然使用YOGA Book为所有人绘画。在实践中，它看起来像这样：在键盘面板上放了一张纸，并使用Real Pen手写笔将图像应用到它上。触控笔能够同时在纸上用墨水书写，并在屏​​幕上显示图形信息。图像传输使用Wacom的电磁共振成像（EMR）技术。面板可识别2048个笔压选项，并允许笔最大偏差100度。



笔记本电脑配备了可在Windows和Android上使用的Book UI界面。它可以在单独的窗口中一次提供多个应用程序的工作。可以以各种方式为您移动，缩放和调整Windows。

特点


两种版本均配备了具有Full HD分辨率（1920x1200像素）的10.1英寸IPS显示器。设备的核心是Intel Atom x5-Z855处理器，集成的Intel HD Graphics 400负责图形处理，LPDDR3 RAM为4 GB，主驱动器为64 GB闪存。笔记本电脑最多支持128 GB的microSDXC存储卡。

自然地，无线连接种类齐全：2G / 3G / LTE，Wi-Fi（802.11 a / b / g / n / ac），蓝牙。 YOGA Book可与纳米格式的SIM卡配合使用，并支持4G网络。

笔记本电脑配备了微型USB，HDMI，3.5毫米音频插孔。此外，在这两种版本中，都有光，振动，霍尔和加速度传感器。

在这两个版本中，都安装了容量为8500 mAh的电池，而Android上的版本将自动持续约15小时，而Windows上的版本将自动持续约13小时。

而不是结论




联想YOGA Book堪称全球最新技术的先锋。这是一种全新型号的变压器，完全能够代表笔记本电脑，平板电脑甚至图形输入板使用。不再需要笨重的第三方设备进行绘图，也无需携带笔记本电脑和平板电脑。一本紧凑的小型YOGA图书-您可以写笔记，做草图，回复邮件，上网，看视频，看书。
