Google工程师测试USB Type-C电缆并在亚马逊上发布报告



有人认为电缆很简单，几乎所有电缆都一样工作，不会造成任何伤害。但是，就USB Type-C转换器电缆而言，情况并非如此，根据Chrome OS，Chromebook Pixel和Google Pixel C的工程师Benson Leung进行的测试来判断。

Benson负责检查各种各样的USB Type-C转换器电缆，以与Pixel笔记本电脑，Nexus 5X，6P智能手机兼容，并且通常与USB规格兼容。最有价值的是，他没有隐藏技术测试的结果，而是将其直接发布在出售电缆的亚马逊网站上。

迄今为止，Benson已经发布了来自不同制造商的电缆转换器的十几条评论：Freeq，CableCreation，Belkin，Monba，Kupx，iOrange，Juiced Systems，Orzly和Techmatte。至少可以说，结果是矛盾的。

事实证明，在所有电缆中，只有Frieq，Belkin和iOrange电缆符合USB规范。其余的至少有两个严重问题之一。例如，CableCreation适配器声称支持3安培的电流，但实际上，转换器组件不支持此类参数。它具有一个10kΩ的电阻，而不是56kΩ的电阻，只能与CC型电缆（两侧均为C型）一起使用。结果，当您连接可以拉3安培的设备时，结果可能是灾难性的。 Kupx，Techmatte和Monba适配器具有相同的问题。

像素笔记本电脑可以通过USB Type-C以3安培的电流充电，因此低质量电缆的问题对他们来说非常重要。

本森解释设置规格中要求使用56kΩ电阻的要求是为了不损坏包括USB Type-A在内的旧设备，如果USB Type-C连接到USB Type-C连接器并拉3安培，则该旧设备将连接在电缆的另一侧。这可能会损坏充电器，计算机或其他设备。

最可悲的是，电缆能够长时间正常工作而不会出现任何缺陷，但是当连接到某些设备时，就像Anker转换器一样，它可以简单地烧坏。


Benson在他的博客中发布了有关如何检查从USB Type-A到Type-C或从USB Type-B到Type-C的电缆转换器的规格的说明（如果您有Chromebook Pixel 2015）。

您需要以调试模式（开发模式）加载笔记本电脑，并将适配器连接至某种充电器，这并不可惜。

然后从控制台运行命令：

ectool --name=cros_pd usbpdpower 0

好的电缆：

localhost ~ # ectool --name=cros_pd usbpdpower  
Port 0: SNK Charger DCP 4920mV / 2000mA, max 5000mV / 2000mA / 10000mW
Port 1: Disconnected

电缆不良：

localhost ~ # ectool --name=cros_pd usbpdpower
Port 0: SNK Charger Type-C 5147mV / 3000mA, max 5000mV / 3000mA / 15000mW
Port 1: Disconnected

在第二种情况下，当电缆连接到Type-A连接器时，显示Type-C的最大电流为3000 mA，这表明电阻器参数不正确。

还有另一个测试选项：在正常引导模式下，转到，chrome://system然后在该部分中dmesg搜索如下所示的行：

[  121.172637] PDLOG 2015/11/04 19:58:18.074 P1 SNK Charger Type-C 5000mV max 5000mV / 3000mA
