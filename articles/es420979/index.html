Conceptos básicos de Redux (libro de texto, segunda edición)

Han pasado 2,5 años desde la publicación de la primera versión del tutorial detallado sobre los conceptos básicos de Redux. Durante este tiempo, gitbook contó con más de 200,000 visitantes únicos.



Debajo del gato más sobre el tutorial y una lista de cambios.


El primero es el texto. Legacy.gitbook todavía se puede descargar en formato PDF / epub / mobi. En la nueva versión, no encontré esta opción y, desafortunadamente, la nueva versión de gitbook no funciona bien sin una VPN.


En segundo lugar, esto es lo mismo en el libro de texto "agua". Texto, imágenes, código fuente.


En el proceso de aprendizaje, se describe la creación de una aplicación simple que interactúa con la API de VK: sus fotos se cargan y luego se muestran en la página ordenadas por me gusta por año.



El propósito de este tutorial es presentar a Redux a aquellos que ya están familiarizados con React. Si necesita comprender los conceptos básicos de React, eche un vistazo aquí .


Donde


Descargue sin conexión o lea en línea - legacy.gitbook


Dado que el gitbook puede tener problemas de accesibilidad, la versión 2.0.0 está disponible en Yandex.Disk (PDF / ePub / mobi)


¿Qué aprenderás en el proceso de aprendizaje?


  • Los fundamentos de la creación de una aplicación SPA en React;
  • Preparar adecuadamente una aplicación Redux (flujo de datos unidireccional);
  • Realice solicitudes asíncronas (precargador, manejo de errores) utilizando el estándar redux-thunk;
  • Interactuar con API de terceros (utilizando la API de VK como ejemplo);
  • Trabajar con documentación (opcional);
  • Optimizar redibujos de componentes;

Lo que ha sido actualizado:


  • Reaccionar 16.4, Redux 4.0
  • Código fuente (completo)
  • Texto (parcialmente + nuevas secciones)
  • Capturas de pantalla (completo)

Gratis, más o menos ...


Comienza a leer, asegúrate de lo contrario. El libro de texto toma en cuenta la experiencia de la asesoría psicológica y, como ya se mencionó, la primera edición se disparó.


¿Cómo hacer un libro mejor?


Esto es de código abierto, amigos. El gitbook tiene una conveniente interfaz de envío de errores. Seleccionó el texto, envió una corrección / deseo / comentario.



Éxito en el aprendizaje. No planeo actualizar el tutorial React-Router.
