Dokumentasi Xamarin API sekarang tersedia untuk umum

Kami sangat senang mengumumkan rilis semua dokumentasi Xamarin API dalam format Open-Source! Dokumentasi API membantu meningkatkan interaksi dengan IntelliSense dan merupakan salah satu cara terbaik untuk membantu pengembang mencapai tujuan mereka.

Dokumentasi Xamarin API


Selain itu, kami juga mentransfer hosting dari situs web Xamarin ke docs.microsoft.com:

  • Xamarin. Bentuk
  • Xamarin. Android
  • Xamarin.iOS dan Xamarin.Mac
  • Skiasharp

Situs dokumentasi baru


Situs docs.microsoft.com menyediakan fungsionalitas presentasi, pencarian, penelusuran, lokalisasi, dan versi yang sama dengan yang digunakan pada platform Microsoft .NET. Sekarang Anda dapat memfilter berdasarkan platform dan versi, sambil secara bersamaan menerima hasil pencarian yang cepat dan akurat berdasarkan ruang nama, nama kelas, dan tanda tangan anggota.


Dokumentasi Xamarin API sekarang open source

Repositori GitHub


Ada repositori Github terpisah untuk:

  • Dokumentasi API Xamarin.Forms
  • Dokumentasi Xamarin.Android API
  • Dokumentasi Xamarin API untuk iOS dan Mac
  • SkiaSharp API Documents

Dokumentasi API dilisensikan di bawah Creative Commons Attribution 4.0 , dan setiap kode sumber terkait dilisensikan di bawah MIT . Repositori di Github tunduk pada " Kode Perilaku Microsoft Open Source ." Halaman dokumentasi Xamarin.Android API adalah modifikasi berdasarkan pekerjaan yang dibuat dan dibagikan oleh Android Open Source Project , dan juga digunakan sesuai dengan kondisi yang dijelaskan dalam lisensi Creative Commons Attribution 2.5 .

Dibangun dengan Mono


Dokumentasi Xamarin API dibangun menggunakan proyek mono, toolkit mdoc open source. Toolkit ini mengubah komentar kode yang disematkan ke direktori khusus file dokumentasi XML. Saat memperbarui rakitan asli, kelas dan anggota yang ditambahkan / dihapus disinkronkan dengan repositori dokumentasi. Dengan alat ini, Anda dapat dengan rapi memisahkan masalah teknik dan dokumenter.

Beri kontribusi Anda


Bantuan dengan berkontribusi pada dokumentasi Xamarin API atau menggunakan toolkit mdoc di proyek Anda sendiri hari ini! Edit data XML secara langsung atau menggunakan alat DocWriter untuk menghindari kemungkinan kesulitan dengan format XML .
