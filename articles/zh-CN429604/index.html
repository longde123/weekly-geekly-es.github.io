联想S730-13（2018）笔记本电脑评测：坚固的金属外壳，时尚的铝制外壳

哈Ha！ 联想已经发布了瑜伽S730系列的新型号。 我们今天将要讨论的修改版收到了第八代Intel Core i7处理器，一个10小时的电池，能够以高达每小时80％的速度充电，一个指纹扫描仪以及整个Yoga笔记本系列中最薄的铝制外壳。



外表


联想S730不仅非常轻巧，而且非常轻巧。 铝盒瑜伽仅重1.2公斤。 经常随身携带会很方便。 这种笔记本电脑将首先吸引那些需要速度和出色物理键盘的人。

在比较照片中-底部的Yoga 720和顶部的Yoga S730-13。 瑜伽厚度720-14.3毫米，S730-13-11.9。



但是这种方便值得一些牺牲。 首先，该模型无法像传统瑜伽一样扩展360度。 其次，这里没有单个USB端口-只有三个Type-C，其中一个是为充电而设计的。 幸运的是，还保留了3.5毫米插孔。 730版本（不带字母S，可以转换为平板电脑）只有一个USB端口，尽管该设备的厚度为14.1毫米，而我们的英雄为11.9毫米。

着色选项-铂金和钢灰色。 车身本身由铝制成，并在喷砂机上经过了精加工。

该设备看起来确实不比杂志盒厚，并且其质量略胜一筹：例如，Elle的重量约为1200克。



介面


联想Yoga S730-13开发人员不得不牺牲许多端口以实现最小的重量和厚度。 由于宽带Internet的普及和速度，他们忘记了CD驱动器很长时间了，允许Wi-Fi摆脱LAN端口，现在是时候将USB发送过去，以微型USB Type-C代替它们了。

笔记本电脑的左侧是用于为笔记本电脑充电的USB Type-C，3.5毫米耳机端口和电源指示灯。





右侧有一个用于恢复出厂设置的重置按钮和两个USB Type-C，其中一个支持对连接的设备进行连续充电。 这些都是这里的端口。 使用无线外围设备将是最方便的方法，尽管原则上不需要使用外围设备来处理文本，表格和浏览网络。 笔记本电脑中不支持存储卡，但可以选择固态硬盘最高为1 TB的型号。



展示架


Yoga S730-13配备13.3英寸显示屏，分辨率为1920 x 1080像素，并具有宽视角。 该显示器支持Dolby Vision（高级HDR格式）。

显示屏周围的细框非常醒目。 框架的厚度应该是多少？对于变压器来说，这个问题很复杂：一方面，框架越薄，则可以在外壳中放置更大的显示器。 另一方面，切换到平板电脑模式时，宽边框可让您安全地握住小工具，而无需额外点击。 由于S730-13不能折叠到平板电脑且没有触摸屏，因此其薄型框架从各个方面来说都是一个加号。



S730和730型号之间的另一个区别影响了铰链。 而不是像第一个联想Yoga 11那样薄的两个，这里一个铰链几乎占据了笔记本电脑的整个长度。



在笔记本电脑的底部，放置了三个橡胶支脚，一个在铰链的整个长度上，另一个在靠近前端的小脚上。





琴键


自2012年以来，联想Yoga键盘一直很方便，并且没有发生任何重大变化。 因此，在S730中，键盘很舒适。 当然，那些年复一年继续使用该系列笔记本电脑的人会特别高兴。 但是习惯它会很快。 琴键行程很柔和，它们本身的中央几乎没有明显的凹痕-从触觉或听觉的角度来看，使用此键盘很高兴。



键盘具有两个亮度级别的背光。 键盘上方的右侧是带有LED指示灯的电源按钮。





键盘下方是指纹扫描仪。





五金配件


第八代Intel Core i7处理器可提供高性能和不错的电池寿命-长达10小时。 也提供i5修改版本。

i7-8565U处理器是四核移动处理器，于2018年第三季度发布，采用14纳米工艺生产。 基本频率为1.8 GHz，但可以通过Turbo Boost技术超频至4.6 GHz。
