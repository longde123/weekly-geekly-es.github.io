自然的极客时间-使空间更清洁

在阅读Geektimes时，我一直想禁用编辑器，因为他们会用自由涌现的文章从自律社区中选拔下一位管理员或类似人员。

几天前，我在主页上看到“一个男生在老师的电话中分享了一张裸照，她被开除了 ” 的帖子，差点就要决定了-我再也不会来这里了，这是另一种资源。

我认为每个人都知道做出此决定的原因-很少有人喜欢强制喂食这种信息渣。主管部门不希望有机会配置和禁用编辑器-这是正确的。我有权离开资源。

但是，谢谢lexasss，有一个用于自我调整磁带的解决方案，我想分享一下。我再说一遍，这个想法和建议来自lexasss，我仅将其发布给那些像我这样想要禁用某些用户的帖子显示的人。





从2018年开始更新-我建议使用此处发布的Keyten脚本。

我们删除Chrome中的用户帖子



因此，要在Chrome中删除编辑器，您需要安装Tampermonkey BETA扩展，创建新脚本并将代码放在此处

// ==UserScript==
// @name        Habr editor blocker
// @description Hides articles that were posted by certain authors
// @match     https://geektimes.ru/*
// @version     1
// @grant       none
// ==/UserScript==
// @namespace    http://tampermonkey.net/

var authors = [
  'alizar',
  'marks',
  'ivansychev',
  'ragequit',
  'SLY_G',
];

var posts = document.querySelectorAll('.post');
for (var idx = 0; idx < posts.length; ++idx) {
  var post = posts[idx];
  for (var i = 0; i < authors.length; ++i) {
    var selector = '.post-author__link[href$="/' + authors[i] + '/"]';
    var blockedAuthor = post.querySelector(selector);
    if (blockedAuthor) {
      post.style.display = 'none';
      break;
    }
  }
}


其中
“ alizar”，
“ marks”，
“ ivansychev”，
“ ragequit”，
“ SLY_G”

是将不会显示其帖子的用户列表。您可以自己定制列表。
