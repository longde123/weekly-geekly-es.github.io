OCZ历史记录：从RAM到SSD的速度为2.7 GB /秒

嗨，极客时间！有时候休息一下压盖，固件和基准测试，合上主机盖，阅读一些有关历史或技术的有趣信息很有用。最近，我们撰写了四篇有关驱动器开发历史的文章（一个 > 2 > 三个 > 4），今天我们想花几分钟时间介绍OCZ的历史-从发布RAM开始，以能够提供最高2倍速度的重型SSD结束7 GB /秒坐吧



OCZ成立已有14年了，通常对这类公司的历史特别感兴趣。同意，很好奇地知道这一切是如何开始的，以及我们如何设法每天都在您的计算机上得到您正在观察的内容。当然，无论如何，都有有趣而无聊的时刻-我们将只告诉您最重要的事情，并像Trion 150一样简要介绍一下最近席卷市场的驱动器。

与所有成功的技术公司一样，OCZ于2002年在加利福尼亚州圣何塞成立。许多其他公司找到了他们的家，并在这里开设了总部，包括Adobe和eBay。那时该公司不叫OCZ Storage Solutions，而是OCZ Technology Group，主要致力于RAM的生产，不仅针对普通消费者，而且还针对企业目的-DDR存储器的速度从PC-2100到PC-5000，包括Macintosh计算机。该公司早在2006年6月就进行了IPO，并进入了伦敦证券交易所-同时DDR2存储产品线出现了热潮。自2007年以来，OCZ推出了DDR3内存版本，该内存仍在许多计算机和服务器中使用。


RAM当然是不错的选择，但是随着时间的流逝，许多制造商开始重新认证并投入生产SSD驱动器。值得注意的是，OCZ很快就在这个方向上取得了成功：该公司的目标是创建负担得起且可靠的SSD，几乎每个人都可以负担得起（实际上，现在您可以自己看到）。早在2004-2006年，128 GB SSD的成本就迫切需要-但是，鉴于开发的复杂性，这不足为奇。

并在2008年3月，OCZ凭借其2.5英寸外形尺寸的Solid Series驱动器和SATA II接口打入了SSD市场，在那时，全球SSD制造商的数量已扩大到70个。该SSD配备了MLC内存模块，并创下了历史记录。当时，速度指示器：读取速度为150 MB / s，写入速度为90 MB / s。但是最有趣的是，在俄罗斯，它的价格约为八千卢布-这就是您要为120 GB版本支付的费用。许多用户正是从此驱动器开始熟悉SSD世界的，因为当时它价格合理，并且具有令人印象深刻的技术特征。不用说需求是适当的。


事实证明，对于OCZ Technology Group来说，2010年是非常多的事件。4月，该公司在纳斯达克美国证券交易所上市，股票代码为“ OCZ”。对于任何公司而言，参加这次交流都是一个重要时刻。同年11月，OCZ收购了Solid Data Inc.的知识产权，交易金额为95万美元。在现代技术世界中，没有这样的“购买”，并购是不可能的。

2010年初，俄罗斯零售市场上的OCZ Z-Drive p84驱动器的销量高达TB。该接口使用PCI-Express 8x，并且SSD本身基于SLC（256 MB快速缓存）分别提供了将近900 MB / s和800 MB / s的读写速度。那是唱片的真实记录，而且价格很高。但是，期望具有这样的容量和速度指示符的情况是愚蠢的。


并在9月，OCZ推出了RevoDrive，这是一款面向发烧友的PCE-E驱动器，具有540 MB / s的读取速度和480 MB / s的写入速度。满足各种需求的超快SSD。同时，引入了带有混合PCIe / SAS接口的高速数据链路（HSDL）。


但不仅如此，OCZ还在2010年提供了这种类型的驱动器。128 GB OCZ Enyo已成为支持USB 3.0的外部SSD。读取速度为260 MB / s，写入速度为200 MB / s，价格从七到八千卢布似乎并不大。是的，容量不是1 TB，但是许多人只梦想着拥有一个外部SSD。


该公司从2011年开始进行购买交易-这次与Indilinx Co达成了交易，因为专利永远不会重复。此后是PLX Technology的收购。 2011年初，OCZ宣布将减少RAM的生产。该公司决定将所有精力投入固态驱动器，这是正确的决定。

不久之后，OCZ推出了具有SATA 6 Gb / s接口（称为Vertex 3）的驱动器。550MB / s的读写能力使其成为SSD中真正的“怪物”。新平台支持高达1 TB的容量，并具有DDR3 DRAM缓存接口，最大支持512 MB。


您不能忽视OCZ Deneva-带有SATA II接口的高速可靠的SSD，基于SandForce SF-1222控制器和MLC。很快，这种新颖性就被选择用于企业，因此，它在许多服务器中得到了解决。

2012年，OCZ接管了SANRAD Inc.。 1500万美元-后者成为该公司在以色列的办事处。

不久之后，该公司宣布出现了Vertex 4和Vector。第一个“发布”用于读取的560 MB / s和用于写入的510 MB / s，具有SATA 6 Gb / s接口，支持TRIM。 Vector的性能大致相同（分别为550 MB / s和530 MB / s的读写），界面似乎相同。但是作为控制器，Indilinx Barefoot 3已经被使用。

2013年，OCZ与东芝达成了一项协议，至今仍是“女儿”。顺便说一下，很少有人知道OCZ还生产用于将人类活动转换为光标移动的神经冲动致动器。

作为东芝旗下的OCZ存储解决方案公司，该公司似乎重生了，并开始发布新的驱动器。2014年，出现了以东芝品牌生产的具有19 nm内存的新型Vector，早些时候OCZ与AMD一起进入了市场。


企业细分市场再次对SSD Sabre 1000的出现感到满意，该产品是专门为不断读取数据并与苛刻的应用程序一起工作而创建的。应该特别注意进入具有NVMe接口的驱动器阶段，例如OCZ Z-Drive 6000。


OCZ存储解决方案从2015年开始采用新产品-在3月在汉诺威举行的CeBIT展览上，该公司推出了SSD Vector 180，该产品取代了Vector150。它具有7毫米尺寸，SATA 3接口，并且OCZ Indilinx Barefoot 3被用作控制器M00 至于速度，读取速度为550 MB / s，写入速度为530 MB / s。最大存储容量为960 GB。


在同一个展览会上，宣布了具有eMLC内存的Intrepid 3700 SSD。容量高达1920 GB，高速性能，五年保修-他立即成为企业客户中的明星。


2015年2月，OCZ开始与Levyx合作。

不久之后，在6月的2015年Computex展览会上，OCZ推出了Trion 100-带有Toshiba Alishan控制器的全新驱动器。高达960 GB的容量，东芝TLC NAND存储器和高达550 MB / s的读取速度。


顺便说一下，在同一年，用户可以使用用于管理OCZ驱动器的SSD Guru实用程序。您不仅可以监视磁盘的状态，还可以更新固件，激活TRIM等。

OCZ在2015年12月启动了ShieldPlus保修计划，该计划现已生效。如果磁盘发生故障，公司会在其框架内向您发送新驱动器，而不是有故障的驱动器。

好吧，在几个月前举行的CES 2016上，该公司推出了Trion 150-Trion 100的后继产品。我们博客上已经进行了诚实的评测（阅读速度为550 MB / s，写作速度为540 MB / s），因此所有优点您可以自己看到。


同样在CES 2016上，RevoDrive 400 NVMe驱动器亮了起来-正是他证明了2.7 GB / s的记录速度。爱好者真正的骄傲。除了面向普通消费者的驱动器外，CES还展示了针对企业目的的解决方案-Z-Drive 6300 AIC系列，Sabre 1000，Intrepid 3700等。


现在，OCZ存储解决方案中心办公室仍位于圣何塞，该公司还在全球设有数个代表处，数百人在此工作。他们致力于确保我们的驱动器更快，更可靠。
