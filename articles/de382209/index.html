Microsoft hat die offizielle Website für die Veröffentlichung von Windows 10 gestartet

Am Tag vor der geplanten weltweiten Veröffentlichung von Windows 10 startete Microsoft die offizielle Windows 10-Startseite , die diesem Ereignis gewidmet ist. Der Hauptinhalt der Website besteht aus bekannten Materialien aus Windows-Blogs, Fotos, Videos, Screenshots zuvor veröffentlichter Builds und einer weiteren Erinnerung daran, dass Sie kostenlos auf Windows 10 aktualisieren können. Darüber hinaus gibt es eine interaktive Karte, auf der Tweets von Benutzern aus der ganzen Welt angezeigt werden, die sich auf die bevorstehende Veröffentlichung von Windows 10 freuen.

Als besondere Überraschung sind Benutzer eingeladen, Animationen und Hintergrundbilder für verschiedene Geräte herunterzuladen , die eine „Ninja-Katze“ darstellen, die eine Windows-Flagge hält und einen Tyrannosaurus reitet:



Dieser Charakter erschien in den neuesten Versionen von Skype für Windows 10 Insider Preview als lustiges Osterfest: in Der Chat musste "Windows10" senden. Es gibt verschiedene Variationen dieses Bildes, die bei Insidern besonders beliebt geworden sind.





Vor einigen Tagen stellte der Chefredakteur des berühmten TheVerge-Tech-Blogs, Tom Warren, fest, dass Microsoft nicht für die Veröffentlichung des Betriebssystems bereit ist. Das von Warren gezeigte Video zeigt, dass auf seinem Laptop Probleme beim Anzeigen von Benachrichtigungen von Tray-Symbolen auftreten. Gleichzeitig testete er die RTM-Version von Windows 10.

Es tut mir leid, aber Windows 10 ist noch nicht für den Start im Juli bereit. Pic.twitter.com/0NJoVnjDWn

- Tom Warren (@tomwarren), 26. Juli 2015

Wenig später beschwerte sich derselbe Warren über einen Fehler mit verschwundenen Symbolen:

Ich habe zu früh gesprochen. Der Fehler beim Verschwinden von Symbolen in Windows 10 wurde nicht behoben. Pic.twitter.com/9GoxNbeouD

- Tom Warren (@tomwarren), 26. Juli 2015

Und er hat es geschafft, einen „blauen Bildschirm des Todes“ auf seinem Windows-Tablet zu sehen. Ihm zufolge war das Problem mit Gerätetreibern verbunden:

Oh Windows 10 pic.twitter.com/vtC4K7uTeh

- Tom Warren (@tomwarren), 25. Juli 2015

Vor nur 10 Stunden hat Microsoft ein weiteres Update für Windows 10 veröffentlicht, in dem, wie Warrren sagt, möglicherweise Fehler behoben wurden.
