PADS Professional - Schnellstart


Guten Tag, lieber Leser!

Mentor Graphics und Nanosoft haben ein kurzes Benutzerhandbuch als eine Reihe von Lektionen entwickelt, die Anfängern die grundlegenden CAD-Funktionen von PADS Professional näher bringen.

Weitere Informationen erhalten Sie bei cat.

Kurzer Rückblick


Der Zweck dieses Handbuchs besteht darin, dem Benutzer kurze Informationen zum Arbeiten in der PADS Professional-Umgebung bereitzustellen. Nach Abschluss einer Reihe von kurzen Übungen können Sie lernen, wie Sie mit PADS Professional eine einfache Leiterplatte erstellen.

Obwohl die PADS Professional-Software eine Vielzahl von PCB-Entwurfsalgorithmen unterstützt, konzentriert sich dieses Handbuch auf den vereinfachten Entwurfsprozess und behandelt die folgenden Themen:


  1. Öffnen eines neuen Projekts in DxDesigner
  2. Platzierung und Anschluss von Bauteilen
  3. Projektverpackung und Erstellung von Topologiedateien
  4. Leiterplattenlayout
  5. Leiterplattenspur
  6. Dateien für die Produktion erstellen

Stellen Sie sicher, dass Sie eine Testversion der PADS Professional VX.2.4-Software oder eine spätere Version heruntergeladen, installiert und aktiviert haben, bevor Sie sich weiter mit den Anweisungen vertraut machen.

Fordern Sie eine Testversion an

Installation und Lizenzierung


Nachdem Sie eine Anfrage für die PADS Professional-Software gestellt haben, sollten Sie eine E-Mail mit den folgenden Links erhalten haben:


  • PADS Professional Online-Software-Installationsprogramm
  • 14 Tage Lizenzdatei

  1. Doppelklicken Sie auf das Online-Installationssymbol, um die Softwareinstallation zu starten.
  2. Klicken Sie auf Weiter .
  3. Lesen und akzeptieren Sie die Bedingungen der Lizenzvereinbarung, wenn Sie damit einverstanden sind.
  4. Klicken Sie auf Weiter , um den empfohlenen Installationsort zu akzeptieren.
  5. Wenn die Installation abgeschlossen ist, klicken Sie auf Fertig .
  6. Laden Sie die Lizenzdatei mit dem Namen " PADSPro.dat" über den per E-Mail bereitgestellten Link herunter und speichern Sie diese Datei unter der folgenden Adresse: C: \ PADS_Professional_QuickStart \ .
  7. Schließen Sie alle Fenster und starten Sie Lektion 1 .

Hinweis: Sie müssen mit dem Internet verbunden sein, bevor Sie die Installation abschließen können. Die Installationszeit hängt von der Geschwindigkeit der Internetverbindung ab
