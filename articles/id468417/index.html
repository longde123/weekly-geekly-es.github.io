3CX v16 Update 3 Beta dirilis - panggilan video di Android dan iOS, koneksi TLS dari batang SIP

Minggu lalu kami merilis Pembaruan 3CX v16 3! Pembaruan ini mencakup dukungan untuk batang SIP aman yang beroperasi di bawah protokol TLS (misalnya, operator seluler Ukraina Lifecell), panggilan video pada perangkat seluler, dan item baru yang menarik lainnya. Uji beta baru - bantu kami merilis rilis final lebih cepat!



Kami mencantumkan peningkatan utama yang muncul dalam paket layanan baru:

  • Koneksi batang SIP pada TLS. Opsi baru "Transport Protocol" telah muncul di parameter trunk SIP. Untuk menghubungkan trunk TLS, instal di TLS (Transport Layer Security) dan unggah sertifikat keamanan (.pem) yang disediakan oleh operator SIP ke server. Sekarang semua pesan SIP antara PBX dan penyedia Anda akan dienkripsi
  • Tingkat kompatibilitas protokol TLS. Di bagian Pengaturan> Keamanan, opsi baru, "Aktifkan kepatuhan PCI SSL / SecureSIP Transport and Ciphers", telah ditambahkan, yang menetapkan mode server ke TLS v1.2 (dan menonaktifkan kompatibilitas dengan TLS v1.0). Harap dicatat bahwa TLS v1.2 tidak kompatibel dengan beberapa perangkat SIP lama.
  • Perbarui sertifikat SSL tanpa memulai ulang. Sebelumnya, ketika memperbarui sertifikat SSL (Let's Encrypt) untuk server FQDN dari 3CX, PBX memulai kembali layanan web. Pada saat yang sama, semua panggilan saat ini terputus. Sekarang PBX hanya menginisialisasi sertifikat baru tanpa memulai ulang nginx. Panggilan aktif tidak terganggu - uptime server telepon lebih tinggi lagi!
  • Pemberitahuan PUSH dalam berbagai status pengguna. Sekarang Anda dapat mengaktifkan atau menonaktifkan pemberitahuan PUSH secara terpisah untuk setiap status pengguna. Di klien web, ini dilakukan di bagian "Opsi"> "Penerusan Panggilan", opsi "Terima pemberitahuan PUSH". Opsi yang sama tersedia dalam versi beta dari aplikasi 3CX untuk Android. Selain itu, Anda dapat mengonfigurasi pemberitahuan PUSH di antarmuka manajemen server 3CX.
  • Nama percakapan grup. Sekarang Anda dapat menentukan nama obrolan grup Anda sendiri, dan itu akan ditampilkan kepada semua peserta di klien web 3CX dan aplikasi seluler 3CX.



Aplikasi 3CX Android Beta Baru


Bersama dengan 3CX v16 Update 3 Beta, kami memperkenalkan aplikasi 3CX baru untuk Android Beta. Ia menerima sejumlah fitur menarik, terutama karena umpan balik Anda. Ini adalah mode operasi senyap, kerja yang lebih baik dengan kontak dan panggilan video (segera hadir)! Untuk menguji aplikasi baru, bergabunglah dengan program penguji beta Google Play.



Rilis 3CX baru untuk Android menerima fitur berikut:

  • Mode operasi senyap. Mode operasi baru menonaktifkan "dering" untuk panggilan masuk. Mengaktifkan dan menonaktifkan mode dilakukan dengan sakelar di antarmuka aplikasi, sementara status yang sesuai ditampilkan di bagian atas layar.
  • Kerja yang benar dengan panggilan masuk. Sekarang, panggilan masuk yang dijawab di telepon IP lain dengan benar berhenti berdering di aplikasi Android.
  • Obrolan yang ditingkatkan. Menu tindakan cepat telah muncul di aplikasi, yang memungkinkan operator pusat panggilan untuk mentransfer obrolan dari situs web kepada dirinya sendiri atau mentransfernya ke karyawan lain. Juga mempercepat pemuatan riwayat pesan.
  • Pekerjaan yang diperbaiki dengan buku alamat. Mengunduh daftar kontak Anda sekarang lebih cepat; Pencarian untuk kontak dengan nama perusahaan juga muncul.

Sayangnya, dalam rilis beta ini, komunikasi video belum sepenuhnya ditolak, tetapi akan muncul ketika versi final dirilis. Video menunjukkan cara kerjanya.

Aplikasi 3CX Android baru membutuhkan minimal 3CX V16 Pembaruan 2 dengan sertifikat SSL tepercaya (lebih baik daripada Pembaruan 3 Beta). Android 7, 8 dan 9 didukung. Android 10 Q telah dirilis baru-baru ini, tetapi kami telah melakukan banyak perbaikan untuk meningkatkan kompatibilitas dengan OS ini. Misalnya, di Android Q, cara untuk menjawab panggilan telah diubah - antarmuka menyerupai respons terhadap pemberitahuan PUSH, alih-alih jendela â€œJawab / Tolakâ€ yang biasa. Kami sudah mendukung respons semacam ini.

Aplikasi ponsel cerdas baru dan server lama PBX


Beberapa fitur v16 Update 3 membuat tuntutan tertentu pada aplikasi 3CX untuk perangkat seluler. Aplikasi saat ini di perangkat Anda akan secara otomatis diganti (melalui perangkat pembaruan Android dan iOS) dengan yang baru - dengan antarmuka yang berbeda, komunikasi video, dll. Aplikasi ini hanya akan bekerja dengan Pembaruan 3CX v16 3 dan lebih tinggi. Di sisi lain, versi aplikasi saat ini tidak akan lagi didukung oleh Pembaruan 3 setelah sekitar satu bulan.

Masalahnya adalah bahwa beberapa pengguna masih menggunakan versi PBX yang ketinggalan jaman sehingga aplikasi seluler baru akan berhenti bekerja. Dan di sini Anda harus memutuskan:

  • Perbarui PBX ke v16 Pembaruan 3 dan terus menggunakan sistem seperti sebelumnya.
  • Jangan perbarui server, tetapi nonaktifkan pembaruan aplikasi seluler.

Dalam kasus kedua, pastikan untuk memperhatikan mode pembaruan aplikasi, karena pada sebagian besar perangkat seluler, pembaruan otomatis diaktifkan secara default.

Kemungkinan memperbarui 3CX


Pembaruan server PBX tersedia untuk pemilik lisensi abadi 3CX dengan berlangganan aktif untuk pembaruan atau lisensi tahunan (termasuk yang gratis untuk 8 panggilan simultan). Jika langganan Anda telah kedaluwarsa, perpanjanglah melalui mitra 3CX Anda. Anda juga dapat menukar lisensi abadi Anda dengan langganan tahunan yang kedaluwarsa - dan menerima semua pembaruan gratis selama satu tahun. Namun, jika Anda tidak siap untuk meningkatkan ke Pembaruan 3, pastikan untuk mematikan pembaruan otomatis aplikasi 3CX pada perangkat Anda.

Android


Buka Google Play, pilih "My Applications and Games", pada tab "Installed", klik "3CX" dan di menu kanan atas hapus "Auto Update".

Bagi pengguna yang memasang versi terbaru 3CX, tetapi memiliki pengguna dengan Android di bawah versi 7, kami akan menawarkan file APK khusus untuk aplikasi 3CX yang baru. Ini harus dilakukan karena Google Play tidak memperbarui aplikasi jika OS atau ponsel cerdas tidak didukung. Namun, aplikasi semacam itu tidak akan secara otomatis diperbarui dan disertai dengan dukungan teknis 3CX.

iOS


Sayangnya, agar tidak memperbarui hanya satu aplikasi, dalam versi iOS saat ini Anda harus menonaktifkan pembaruan untuk semua aplikasi.

Buka Pengaturan> iTunes & App Store dan matikan sakelar di bagian atas layar Pembaruan. Seperti yang Anda lihat, kurangnya berlangganan pembaruan hanya untuk satu aplikasi membuat pembatasan pada pekerjaan semua aplikasi iOS lainnya. Ini adalah argumen lain untuk memperbarui atau menggunakan lisensi tahunan 3CX.

Instal Pembaruan 3CX v16 3 Beta dan aplikasi baru


Jika Anda menginstal versi terbaru 3CX, buka bagian "Pembaruan" di antarmuka server, pilih "v16 Perbarui 3 Beta" dan klik "Unduh Terpilih".
Anda juga dapat menginstal distribusi bersih untuk Windows atau Linux:

  • Windows: 3CX v16 Pembaruan 3 Beta
  • Linux: 3CX v16 pada Debian 9 ISO - Dari layar " 3CX Update Channel ", pilih "Beta".
  • Instalasi manual pada Debian dan Raspbian: ikuti instruksi instalasi untuk Debian dan Raspberry Pi .

Untuk mencoba aplikasi 3CX baru untuk perangkat seluler, daftar di program pengujian Beta:

  • 3CX untuk Android Beta
  • 3CX untuk iOS Beta
