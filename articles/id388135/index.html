Windows 10 menjadi lebih sulit untuk ditinggalkan

Microsoft mencoba berbagai cara untuk membujuk pengguna Windows 7 dan 8.1 untuk meningkatkan ke Windows 10: 3-5 GB file instalasi diunduh di latar belakang, pesan sembulan "Pembaruan Anda siap dipasang" terus-menerus ditampilkan, program instalasi itu sendiri "secara acak" mulai berjalan . Tetapi sekarang mereka telah melampaui diri mereka sendiri.

Lihatlah screenshot ini. Tidakkah itu terasa aneh bagi Anda?



Ingin meng-upgrade ke Windows 10 sekarang atau di malam hari? Itu sekarang atau sekarang? Ini adalah penampilan pilihan yang tidak memberikan jawaban negatif.



Dialog baru ditampilkan dari hari Minggu lalu ke pengguna Windows 7 dan 8.1, InfoWorld melaporkan .

Dengan demikian, Microsoft menggunakan taktik malware yang terkenal. Satu-satunya kelemahan adalah X merah di sudut kanan atas, yang menutup jendela. Seharusnya tidak.

Di awal tahun depan, Microsoft akan mentransfer pembaruan Windows 10 ke kategori pembaruan yang disarankan untuk Windows 7 dan 8.1, membicarakan hal ini pada bulan OktoberTerry Myerson, Wakil Presiden Eksekutif, Grup Windows dan Perangkat. Tergantung pada pengaturan dalam sistem operasi, ini dapat menyebabkan proses instalasi Windows 10 secara otomatis memulai, yaitu, penginstal akan secara otomatis memulai.

"Sebelum pembaruan mengubah sistem operasi pada perangkat Anda, mereka akan dengan jelas bertanya apakah Anda ingin melanjutkan atau tidak," janji Terry Myerson. "Dan tentu saja, jika Anda memilih pembaruan (rekomendasi kami!), Maka akan ada 31 hari untuk kembali ke versi Windows sebelumnya jika Anda tidak menyukainya." Omong-omong, Anda selalu dapat mengganggu instalasi dengan tidak menyetujui persyaratan perjanjian pengguna (EULA).

Diharapkan bahwa Microsoft akan menepati janji dan akan benar-benar ada tombol di installer "Menolak untuk menginstal", dan tidak seperti sekarang.



Untuk menghilangkan pengingat upgrade yang mengganggu, para ahli menyarankan agar pengguna Windows 7 dan 8.1 menginstal program Panel Kontrol GWX gratis (ada juga Stopper GWX).



Panel Kontrol GWX secara permanen menghapus ikon "Dapatkan Windows 10" dari area notifikasi, mencegah pembaruan Windows 10 melalui Pembaruan Windows, mencegah unduhan latar belakang tersembunyi dari file instalasi Windows 10, mendeteksi dan menghapus file instalasi Windows 10 yang tersembunyi jika masih menembus komputer memantau tampilan file lain dan pengaturan sistem yang terkait dengan Windows 10.

Melindungi diri Anda dari menginstal Windows 10 sulit, tetapi secara realistis.

Microsoft akan mendukung Windows 7 dan merilis pembaruan keamanan hingga 2020, jadi sekarang tidak perlu menginstal pembaruan.
