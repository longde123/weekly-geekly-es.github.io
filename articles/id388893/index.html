Dengan membeli MacBook dari pemasok resmi, Anda dapat dibiarkan tanpa jaminan resmi Apple. UPD. Tetapi saya berhasil



Beberapa hari yang lalu, saya memutuskan untuk membeli sendiri macbook. Pada awalnya saya mencari Avito dan menyadari bahwa MacBook Air bekas dalam kondisi baik akan cukup mahal, dan tidak ada yang bisa memberikan jaminan. Anda dapat mengingat setidaknya pos ini .

Nah, kemudian saya memutuskan untuk mencari laptop baru di toko. Akibatnya, semuanya ternyata tidak sesuai dengan rencana.

Setelah membuka Yandex.Market, saya agak terkejut dengan harga, tetapi apa yang harus dilakukan, nilai tukar dolar meninggalkan banyak yang harus diinginkan. Karena saya tidak memiliki kepercayaan penuh pada Yandex.Market (ia menunjukkan MacBook Air 13 "256GB baru, Awal 2014 di Euroset dengan jumlah 50 ribu rubel, tetapi sebenarnya perangkat ini telah lama kehabisan stok), Saya memutuskan untuk mencari situs secara terpisah demi kesetiaan.

Apa yang mengejutkan saya ketika saya mengunjungi situs MediaMarkt! Di sana, selain daftar macbook dengan harga, ada beberapa model tanpa harga, tetapi dengan catatan "Produk hanya dapat dibeli di toko MediaMarkt ".



Jika Anda mengklik tautan, maka daftar toko dengan harga ditampilkan.

Jadi, MacBook Air 13" 256gb, Awal 2014 ditemukan berada di toko MediaMarkt MEGA Belaya Dacha hanya 64999 rubel!

Jadi saya menelepon toko itu, di mana mereka memberi tahu saya bahwa laptop itu adalah yang terakhir tersedia.
Saya tiba di toko yang ditentukan, di mana setelah satu jam mencari (tentang kelambanan para karyawan dari rantai toko ini, Anda dapat menulis posting terpisah, tetapi tidak tentang itu sekarang) mereka menggali laptop ini dan setuju untuk menjualnya kepada saya.
Benar, kotak itu tidak disegel, di mana penjual mengatakan bahwa, mungkin, laptop itu sebelumnya di jendela. Tetapi karena kondisinya jelas baru, saya tidak keberatan. Untuk jaga-jaga, saya mengklarifikasi apakah akan ada garansi satu tahun resmi. Saya diberitahu bahwa ya, tentu saja.

Jadi saya meremas kodok dengan tenang membeli perangkat ini.
Setelah tiba di rumah dan bermain cukup dengan banyak peralatan di luar negeri, saya memutuskan untuk tetap memeriksa jaminan. Dan di sini saya kecewa:


(Saya tidak tahu tulisan aneh apa dalam bahasa yang tidak bisa dimengerti muncul di sebelah nama ketika memeriksa serial. Laptop itu sendiri untuk Rusia, kode model MD761RU / B)

Rupanya, etalase sudah diaktifkan sebelum itu. Dan Apple percaya garansi telah berakhir. Tentu saja, saya memutuskan untuk menghubungi dukungan teknis Apple dan MediaMarkt untuk mencari tahu cara memperbaiki masalah ini.

Komunikasi dengan MediaMarkt

Saya menulis permintaan untuk menjelaskan situasinya kepada mereka. Saya diminta untuk mengirim nomor pembelian. Sehari kemudian, seorang pemuda dari dukungan menelepon saya dan mengatakan bahwa mereka tidak perlu khawatir, mereka tidak peduli dengan informasi dari situs web Apple dengan cara apa pun, karena toko mereka melakukan perbaikan selama setahun sendiri, mereka hanya perlu cek.
Perlu dicatat bahwa pada cek, untuk beberapa alasan, nomor seri laptop saya hilang. Tetapi saya diberitahu bahwa ini seharusnya tidak menjadi masalah. Yah, sementara aku percaya itu.

Semua ini, tentu saja, sedikit menenangkan saya. Tetapi jika saya kurang lebih mempercayai pusat layanan resmi Apple, yang langsung mengambil laptop saya, maka saya sudah agak takut menyerahkan MacBook untuk diperbaiki di toko.
Namun sangat tidak menyenangkan untuk membeli macbook yang agak mahal dari pemasok resmi dan tidak mendapatkan jaminan resmi. Jadi saya memutuskan untuk menghubungi

Apple

Saat memeriksa garansi di situs web resmi terdapat tombol "Konfirmasi tanggal pembelian". Saya mengikuti tautan dan mengirim permintaan ke Apple.
Hari berikutnya saya menerima surat yang meminta saya mengirim foto cek, dan setelah beberapa saat surat lain tiba. Dikatakan bahwa konfirmasi tanggal pembelian ditolak, karena menurut informasi mereka, laptop sudah dibeli dan diaktifkan sebelumnya.
Saya menjawab bahwa saya tidak setuju dengan keputusan ini, setelah itu saya diminta menelepon mereka untuk mengklarifikasi situasi.

Di telepon, setelah beberapa menit proses, saya beralih ke spesialis terpisah yang berjanji untuk mencoba memahami situasinya. Untuk melakukan ini, saya mengiriminya foto kwitansi ke email pribadinya-perusahaan.
Saya dijanjikan untuk menghubungi kantor Amerika dan mencari tahu apakah masih mungkin untuk memperbaiki tanggal pembelian yang tercantum dalam sistem mereka dan memberi saya jaminan pada laptop saya yang baru dibeli.
Saya mengingatkan Anda sekali lagi bahwa cek tidak mengandung nomor seri (terima kasih, MediaMarkt), jadi saya ragu tentang keputusan positif. Tetapi saya, jika ada, akan berusaha untuk berhasil.
Saya menunggu jawaban sejauh ini.

Apa yang ingin saya katakan. Hati-hati dengan pembelian etalase. Tidak hanya mereka bisa macet oleh pengunjung tokotidak cukup berkualitas (saya beruntung, saya bahkan memiliki jumlah siklus baterai di suatu tempat sekitar nol), dan Apple, yang memulai masa garansi dari saat aktivasi pertama, mungkin mengalami masalah seperti itu.

Karena akan ada jawaban dari Apple, saya pasti akan berhenti berlangganan di sini.
