Internes PHP


  • RFC: Spread Operator in Array - Un brouillon discute de la possibilité d'ajouter un opérateur ... pour les tableaux.
    Texte masqué 
     $parts = ['apple', 'pear']; $fruits = ['banana', 'orange', ...$parts, 'watermelon']; // ['banana', 'orange', 'apple', 'pear', 'watermelon'];
