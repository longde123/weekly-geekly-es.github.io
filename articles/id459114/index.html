14 Plugin SEO WordPress Terbaik di 2019

Plugin WordPress SEO akan membantu Anda secara otomatis menambahkan meta tag, memuat halaman lebih cepat, memperbaiki kesalahan pengalihan, mengoptimalkan gambar, dll. Versi gratis dan premium plugin WordPress paling populer untuk SEO pada tahun 2019 tercantum di bawah ini.



Mengapa Anda membutuhkan plugin WordPress SEO?


SEO situs web WordPress Anda dapat ditingkatkan sebagian atau seluruhnya dengan plugin. Ada juga solusi serupa untuk CMS lain, seperti OpenCart dan Joomla. Plugin SEO akan membantu Anda untuk:

  • Buat peta situs XML;
  • Transliterasi dan buat alias URL secara otomatis;
  • Buat AMP;
  • Atur pengalihan sisi server dengan benar;
  • Tambahkan judul, deskripsi, tag meta kata kunci;
  • Optimalkan nama dan ukuran gambar;
  • Gunakan cache untuk memuat situs web Anda lebih cepat;
  • Tambahkan atribut nofollow ke tautan eksternal untuk tidak kehilangan jus tautan apa pun.

Plugin WordPress SEO Berguna 2019


Pilihan plugin Anda bergantung pada versi WordPress situs web Anda dibangun, waktu buka, jumlah file media, dan parameter lain yang ingin Anda optimalkan. Anda dapat menggunakan versi gratis atau yang premium, dengan berbagai fitur yang lebih luas.

Plugin SEO yang kompleks untuk Wordpress pada tahun 2019


Yoast SEO (WordPress SEO by Yoast) - solusi komprehensif yang dikembangkan berdasarkan beberapa plugin.


Plugin Yoast SEO:


Plugin ini telah diinstal lebih dari 5 juta kali dan diterjemahkan ke dalam 40 bahasa.

Fitur gratis:

  • Membuat peta situs XML;
  • Menetapkan URL kanonik untuk mencegah duplikat konten;
  • Judul dan deskripsi meta templating;
  • Alat untuk menulis teks ramah SEO;
  • Pratinjau cuplikan;
  • Integrasi dengan Google Analytics.

Fitur premium:

  • SEO Berita, SEO Video, SEO Lokal, dan ekstensi SEO WooCommerce;
  • Akses gratis 1 tahun ke dukungan teknis;
  • Alat untuk mengontrol frasa kunci, pengoptimalan sinonim, dan frasa kunci terkait;
  • Saran penautan internal otomatis;
  • Pratinjau media sosial: periksa seperti apa tampilan halaman Anda saat Anda membagikannya.
