Bing tem um editor de código


Um editor de código interno acaba de aparecer no mecanismo de pesquisa do Bing , com pesquisa e execução de código (até agora só funciona se você definir a região Estados Unidos - Inglês nas configurações da conta).A ferramenta está posicionada como "uma nova maneira de aprender linguagens de programação e brincar com o código". É ainda possível comparar a aparência da mesma solução em diferentes idiomas.

Alguns ativistas de código aberto dizem que essa ferramenta deve ser usada com cautela. Porque A resposta está expressamente escrita no Contrato de Serviços Microsoft , parágrafo 2b.

Parece que a cláusula do contrato é comum e padrão para a Microsoft, mas neste caso é importante lembrá-la, porque a Microsoft recebe uma licença para nada mais que o código do programa.

Além dos métodos listados no contrato, a Microsoft tem o direito de usar seu código para outros fins. Por exemplo, para treinar uma rede neural para melhorar os resultados da pesquisa.

Mas o mais importante, absolutamente tudo é o mesmo no editor do Rosetta Code , sob a licença GNU FDL gratuita (uma adição à GPL para documentação).
