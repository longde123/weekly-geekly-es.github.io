Als vielseitige Person mag ich sehr viel Musik mit verschiedenen Genres / Formaten / Künstlern.

Vor nicht allzu langer Zeit bin ich auf einen Deezer gestoßen (nicht lachen). Ich habe das kostenpflichtige Abonnement verwendet. Es hat mir sehr gut gefallen. Das Abonnement war beendet und das Bezahlen aus meiner Region ist wirklich eine sehr schwierige Aufgabe.

Google ist auf das Deezloader- Projekt gestoßen , ich sage grob, den unverschämten Diebstahl von Musik von der Deezer-Website. Nach der Installation und Konfiguration gab es keine Anmeldung. Ich ging tiefer in Google ein und fand das gleiche Projekt nur aktualisiert. Die Jungs haben eine interessante Art der Verbreitung - über einen Telegrammkanal.

Dieses Projekt ist unter CC BY-NC-SA 4.0 lizenziert

Ich habe Centos 7, Node JS 8.11 und Deezloader Remix verwendet. Entpacken Sie das Archiv und fahren Sie mit der Installation fort. 

curl -sL https://rpm.nodesource.com/setup_8.x | bash - sudo yum install epel-release sudo yum update sudo yum install nodejs sudo yum install gcc-c++ make cd DeezloaderRemix/app npm install node app.js

Wir gehen zu IpAddr: 1730, melden uns unter unserem Deezer-Konto an, wir bekommen standardmäßig eine schöne Oberfläche für unseren Igel , legen das Download-Verzeichnis ~ / Music / Deezloader fest

Viele unverständliche Screenshots




















Fügen Sie nun die Plexmediaserver-Warteschlange hinzu: 

 nano /etc/yum.repos.d/plex.repo 

 # Plex.repo file will allow dynamic install/update of plexmediaserver. [PlexRepo] name=PlexRepo baseurl=https://downloads.plex.tv/repo/rpm/$basearch/ enabled=1 gpgkey=https://downloads.plex.tv/plex-keys/PlexSign.key gpgcheck=1

Nun, die Installation selbst: 

 sudo yum -y install plexmediaserver

Schalten Sie den Daemon ein und fügen Sie ihn zum Start hinzu: 

 systemctl start plexmediaserver systemctl enable plexmediaserver

Firewalld-Konfigurationswarteschlange 
 nano /etc/firewalld/services/plexdeez.xml 

 <?xml version="1.0" encoding="utf-8"?> <service> <shortplexdeez</short> <description>Ports required by plexmediaserver and Deezloader.</description> <port protocol="tcp" port="32400"></port> <port protocol="udp" port="1900"></port> <port protocol="tcp" port="3005"></port> <port protocol="udp" port="5353"></port> <port protocol="tcp" port="8324"></port> <port protocol="udp" port="32410"></port> <port protocol="udp" port="32412"></port> <port protocol="udp" port="32413"></port> <port protocol="udp" port="32414"></port> <port protocol="tcp" port="32469"></port> <port protocol="tcp" port="1730"></port> </service>

Fügen Sie den frisch gebackenen Plexdeez-Dienst zur Firewall hinzu und starten Sie ihn neu 

 sudo firewall-cmd --add-service=plexdeez --permanent sudo firewall-cmd --reload


Wir registrieren uns unter app.plex.tv. Wir geben ip: 32400 / web weiter, wir melden uns unter unserer brandneuen ak an. Fügen Sie die Musikbibliothek hinzu und geben Sie das Verzeichnis ~ / Music / Deezloader an. Genieße die frische Musik)
