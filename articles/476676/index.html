48 recursos de código abierto para JavaScript (2019)



De los 20,000 proyectos y bibliotecas de JavaScript, seleccionamos los 48 mejores (por estrellas en GitHub) y los agrupamos en 5 categorías:

  • Interfaz de usuario (1 ~ 9)
  • Herramientas de JavaScript (10 ~ 25)
  • Aprendizaje automático (26 ~ 33)
  • Proyectos (34 ~ 39)
  • Herramientas para desarrolladores (40 ~ 48)


Este artículo fue preparado con el soporte del software EDISON, que desarrolla un sistema de examen médico electrónico y también es compatible con el software .


Interfaz de usuario


  1. Rough v2.0 : crea gráficos de croquis dibujados a mano.
  2. Driver.js : un motor de JavaScript ligero, independiente y básico que enfoca la atención del usuario en la página.
  3. Sweetalert2 : reemplazo emergente de JavaScript hermoso, receptivo, personalizable y asequible (WAI-ARIA).
  4. Tippyjs : Biblioteca de información sobre herramientas JS ligera y vainilla.
  5. Spritejs : modelo de objeto ligero multiplataforma para renderizado 2D.
  6. AutoComplete.js : una biblioteca Javascript simple para autocompletar.
  7. Glider.js : una alternativa rápida y fácil a un carrusel de desplazamiento.
  8. LayerJS : biblioteca Javascript UI / UX que le permite crear aplicaciones intuitivas visualmente atractivas para aplicaciones web y sitios web similares a las aplicaciones móviles.
  9. Pencil.js : una biblioteca para dibujo 2D interactivo.

    Herramientas de JavaScript

  10. Johnny-Five v1.0 : JavaScript Robotics y el entorno de programación IoT.
  11. Babylon.js : marco JavaScript para crear juegos 3D en HTML5 y WebGL.
  12. Pollyjs : graba, reproduce y elimina interacciones HTTP.
  13. Omi : Marco web de próxima generación en JavaScript de 4kb (Componentes web + JSX + Proxies + Almacenamiento + Actualización de ruta).
  14. Estímulo : un modesto marco JavaScript para HTML que ya tienes.
  15. Apexcharts.js : una biblioteca moderna de diagramas de JavaScript para crear diagramas interactivos y visualizaciones con una API simple.
  16. Greenlet : mueve una función asincrónica a su propio hilo.
  17. JSUI : una poderosa herramienta de interfaz de usuario para administrar aplicaciones JavaScript.
  18. G6 : un marco de visualización gráfica.
  19. Workerize : ejecuta el módulo en web-worker'e.
  20. V8n : biblioteca de validación personalizable y reutilizable .
  21. ForgJs : un validador de objetos JavaScript ligero.
  22. Minipack : un ejemplo simplificado de un paquete moderno escrito en JavaScript.
  23. Lista v2 : una lista coherente con un rendimiento inigualable y una API funcional integral.
  24. JavaScript Object Explorer : encuentre el método de objeto deseado sin profundizar en la documentación.
  25. Awaity.js : una alternativa funcional y liviana a bluebird.js, creada teniendo en cuenta async / wait.

    Aprendizaje automático

  26. Tfjs : WebGL acelerado, una biblioteca basada en navegador JavaScript para entrenar y desplegar modelos ML.
  27. Guess.js : un juego de herramientas para incorporar experiencias de usuario basadas en datos en la Web.
  28. Picojs : biblioteca de reconocimiento facial de 200 líneas.
  29. Tensorspace : infraestructura para la visualización tridimensional de una red neuronal, creando un modelo interactivo e intuitivo en navegadores, soporte para modelos de aprendizaje profundo pre-entrenados de TensorFlow, Keras, TensorFlow.js
  30. Nlp.js : biblioteca NLP integrada en el nodo en la parte superior de Natural, con extracción de entidades, análisis de estado de ánimo, identificación automática del lenguaje, etc.
  31. Magenta-js : genera música e imágenes utilizando la inteligencia artificial en un navegador.
  32. JeelizWeboji : biblioteca JavaScript / WebGL para el reconocimiento facial y la detección de expresiones en tiempo real.
  33. ASLT : Enseñe a Alexa a responder el lenguaje de señas con Tensorflow.js

    Proyectos

  34. Tui.calendar v1.0 : calendario.
  35. Webamp : Winamp 2.9 clon en HTML5 y Javascript.
  36. Filepond : biblioteca de carga de archivos
  37. OpenSC2K : remake de código abierto de SimCity 2000.
  38. Jsnes : emulador de NES.
  39. JSpaint : pintura clásica revivida.

    Herramientas para desarrolladores

  40. Carbon v3.0 : crea y comparte bellas imágenes de tu código fuente
  41. Jarvis : una barra de herramientas Webpack muy inteligente basada en navegador
  42. Polacode : Polaroid para tu código
  43. Htm (marcado etiquetado Hyperscript) : alternativa JSX utilizando plantillas de etiquetas estándar con soporte de compilador.
  44. Rete : marco JavaScript para programación visual y creación de un editor de nodos
  45. Size-plugin : rastrea los tamaños comprimidos de los recursos de Webpack a lo largo del tiempo.
  46. Webpackbar : elegante ProgressBar y Profiler para Webpack 3 y 4
  47. Worker-plugin : agrega soporte incorporado de Web Worker a Webpack.
  48. Subliminal : tema minimalista VS Code para JavaScript
