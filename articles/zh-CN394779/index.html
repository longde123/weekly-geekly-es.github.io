Windows 10更新中断了受害者PC上的TeamViewer黑客

有时强制更新可能会产生积极的效果



，Internet现在发布了许多有关将Windows以前版本强制更新为最新Windows 10的消息。一份请愿书甚至出现在网上，其作者呼吁通过积极的更新政策调查Microsoft Corporation的工作。问题不在于操作系统本身，一切都很好，也就是公司追求的更新策略。但是有时候它会很有用。Reddit

用户，由于最近的一次黑客攻击而成为受害者。TeamViewer注意到该黑客已与系统断开连接，并没有做任何不好的事情。就在黑客入侵期间，用户的计算机操作系统决定升级到Windows10。根据Microsoft Corporation的声明，该操作系统尽可能地保护了用户免受外界干扰。

在大多数情况下，TeamViewer破解者试图找到受害者的帐户信息以获取PayPal或其他类似信息。这次，饼干失败了，事实证明这是受害者本人的fl幸。无论如何，正如Reddit用户所声称的那样，这是迄今为止所发生事件的唯一信息来源。 Windows 10安装程序启动后，由于网络连接也在自动模式下被禁用，因此破解程序已与系统断开连接。

但是，在更新操作系统并重新启动PC之后会发生什么？您可以期望黑客重新连接，对吗？不，在这种情况下情况有所不同。

在系统更新期间删除了TeamViewer，因此，在完成升级过程后，攻击者将无法再访问受害者的PC。这似乎是正确的，因为Windows 10会删除标记为与此版本的操作系统不兼容的应用程序。

当然，不可能验证所有这些，但是如果所有这些都是正确的，那么情况就很有趣。还是由破解者决定将用户的PC升级到Windows 10？尽管不太可能，但这太不可思议了。

对于TeamViewer，有关黑客服务帐户的消息以令人羡慕的规律性出现。第一条消息出现在一个月前，现在受害者人数已达数百人。据人们判断，黑客入侵大量帐户的原因是针对不同站点使用相同密码的用户的授权数据大量泄漏。受害者中甚至还有信息安全专家，其中包括 IBM威胁研究小组信息安全部的主要雇员之一尼克·布拉德利。



尼克说：“大约在下午6:30，我处于比赛中间。” -突然我失去了对鼠标的控制，并且TeamViewer消息出现在屏幕的右下角。我一意识到发生了什么，就立即杀死了该应用程序。然后它突然降临：我还安装了其他装有TeamViewer的机器！”

“我跑下楼梯，打开了另一台计算机并开始工作。从远处看，我已经看到TeamViewer窗口。在我将其删除之前，攻击者启动了浏览器并打开了新的网页。一进入键盘，我立即取消了遥控器，立即访问了TeamViewer网站，更改了密码并激活了两因素身份验证。”

TeamViewer否认所有入侵其企业基础架构的指控。她认为，原因是受害者使用了相同的授权数据。该公司建议不要这样做，并且要包括两因素身份验证以及定期更改密码。此外，TeamViewer支持引入了两个新功能。安全性，包括可信设备和数据完整性。

好吧，与此同时，微软正试图加快将用户切换到其新操作系统的过程。在免费更新期结束之前，已经一无所有，因此该公司正在尝试。例如，现在用户正在报告新的“您无法拒绝的报价”：Windows 10更新无法取消，因为...更新窗口上没有“取消”按钮。用户唯一的选择是更改升级时间或立即启动。按下X按钮右上角的不会仅仅因为此按钮没有更改任何内容。由于您无法取消更新，因此出现此窗口时，用户唯一可以做的就是更新。同时，该公司拒绝所有指控，称如果没有独立选择“升级到Windows 10”和采用EULA，就不可能获得这样的窗口。这样的窗口（如预览屏幕截图中所示）只能在所有这些操作完成后出现。



注意三星笔记本电脑所有者的更新。事实证明在三星笔记本电脑上，Windows 10不能很好地工作。三星表示“不算太好”，不建议笔记本电脑的用户升级到新的OS版本。这样做的原因很明显-该公司尚未开发出专门针对Windows 10的驱动程序。

甚至非洲的保护主义者也遭受了Windows 10更新的困扰。。由于远离大型定居点，环保主义者组织被剥夺了使用快速互联网工作的机会。取而代之的是使用慢速卫星互联网，这仍然非常昂贵。因此，其中一个组织中的一台计算机下载了大约6 GB的Windows 10安装文件，因此，此更新使非洲志愿者付出了不菲的代价。而且，所有这些可能会更加悲惨地结束。Reddit用户zambuka42说道：“如果强制更新开始，并且在我们协调武装人员在武装准军事偷猎者的火力下崩溃时，计算机确实会在您的手中。”

但无论如何，无论有无用户许可，我们的PC都将继续更新。
