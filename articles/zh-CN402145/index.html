继续主题“癌症。 该怎么办...”

我无法在“癌症。 该怎么办……”，所以我在这里写-一个重要的话题。

有一次，我与军事医学学院的人，很多医学朋友一起工作了很长时间。 我对实验室研究，临床试验，结果应用统计有一个想法。

根据我的脊椎MRI扫描结果，今年6月，地区肿瘤科医生得出结论：第4期肿瘤学未接受治疗。 两个椎骨伴有病理性骨折，另外几个受到影响。 尽管在此之前有人怀疑前列腺癌，但这并不是一个完全的意外。 背景就是这样。

他于今年三月住院。 五天的背痛，腹部肿胀，灌肠没有帮助。 在医院里，洗了肠，看了一下直肠，看了一周，然后送回家了。 我的背部仍然酸痛，但是我拒绝做脊椎检查（他们说，这是一个很长的队列，要等很长时间...）。 同时，一位生苔藓的老神经病理学家提出了以下论点：“如果出了问题，您会做手术吗？” 论据是有力的，因为您真的只需要在不得已时才进行脊柱外科手术。 背部疼痛逐渐消退。

不久，他对PSA进行了分析，该PSA是用于测量前列腺癌风险的乘员。 我这样做是因为一年前提高了PSA，然后我被派往市癌症药房进行咨询。 在那里，他们进行了超声波检查，然后放开，告诉他们要由泌尿科医师进行观察。 他能看到什么？ 今年的PSA结果为51 ng / ml，5月份进行了活检。 活检是一种创伤性手术。 他们没有发现任何犯罪……医生这样说：“直到命运……”。 我感到高兴，因为PSA 51几乎可以肯定是癌症。 在这里，它似乎已经解决了，没有什么可担心的。

但是，在那之后的三个星期，她再次抓住了她。 然后他们做了我开始的MRI。 在前列腺中未发现肿瘤，但是脊柱转移证实了它的存在。 经过第二次活检，她出现了。 同时，PSA已经达到156，首次活检显着刺激了这一过程。

PSA动态：
2015年6月-9 ng / ml
2016年4月-51 ng / ml
2016年7月-156 ng / ml

同时，取决于年龄（和阅读地点）的标准在40岁时最高达2.5，而在60岁时最高达5（我就是这种情况）。

多达10个（甚至多达15个）的指标被认为是“灰色区域”-这是一个风险增加的区域，但不一定是癌症的区域。 活检准确地触发了该过程。 我认为没有最终解决方案。 朋友有8岁-他毫不犹豫地进行了活检。 在胚胎本身中发现了一种肿瘤，并成功地将其去除，而对排尿和勃起没有任何干扰。 （但他在三星工作时在韩国）。 第一次使用PSA 51，我无法检测到肿瘤，同时也没有感觉到我在城市诊所中摸索。 就整体情况而言，在我看来，不做稍有过多的活检是正确的。 但是，如果有所增加，则需要每三到四个月不断进行PSA分析（这是一个简单的分析），并且如果指标至少增长了一个-则需要立即进行活检。

他们用激素治疗我。 Zoladex注射液和BICAN片剂。 此外，我还服用了一种顺势疗法（根据自己的血滴）和豌豆（具有某些传统成分）。 我还喝了ASD组分2的成分-我的一位朋友推荐的一种兽药。 在网上，他们对他有不同的看法，但决定尝试一下。

今天，根据细胞变性的程度，癌症分为几种类型。 单元越简单，其行为就越积极。 改变细胞的比例很重要，格里森和考虑了这一比例。 数量从2 + 2到5 + 5（最后一个最难）。 我有4 + 4。 我在一个地方发现，有了这些数字，生活五年的概率为30％。 如果这样的话会很好...

到目前为止，我很幸运（尽管明天可能会有所不同）。 四个月来，PSA降至1.7。 没有痛苦。 在模拟器上进行体育锻炼（在附近的Bubnovsky博士中心有一个小型健身房-用于加强脊柱的肌肉骨骼）-进行全部锻炼的80％。

缺点：睾丸激素减少。 体重增加（比正常人多3公斤）。 情绪波动。 拖延比以前更多。 必须放弃部分工作计划，并且计划时间不得超过六个月。

我感觉到一件事。 肿瘤学是一个高概率领域。 医生没有给出预测，因为极有可能以一个词损坏患者。 只有一般统计。 但是有一天，我遇到了一份病史清单（约一百份，是该专利申请的附录，用于肿瘤的新诊断）。 人们将他们的诊断与现有的诊断进行了比较，也就是说，他们需要一个证据基础并进行深入研究。 同时，他们也没有站在任何患者面前...在此列表中，有1-2例PSA低于正常值的癌性肿瘤！ 有3名PSA患者（从40到70）在所有尽职调查中均未发现肿瘤。

如果您遇到这种不幸，请尝试让那些无法放弃的人知道。 你甚至都不应该告诉你的朋友。 您可以不加安慰地做。 人们知道您的病情后，将很难与您进行交流（他们会始终牢记不要过多地告诉您您需要以特殊的方式行事...）。 这会干扰您和他们。

在我看来，如果除了官方方法之外，决定使用一些与这些方法没有直接矛盾的民间疗法，那么适当的健康宿命论将是适当的。 同样，因为一切都取决于概率。 此类方法有很多（果汁，完全绝食，草药...），哪种方法最适合您的情况尚不清楚。 选择永远是一个偶然的问题。

通常，具有此诊断的人进入区域，在那里他将成为跟踪者。 您需要改变自己的人生价值观，关于常识和直觉的观念。 刷新它们。 如果他能够在某个新的水平上应对自己的危急情况，那么您可以找到最佳方法来克服它……
