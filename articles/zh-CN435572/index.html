Anycubic i3 Mega：Prusa i3的高品质翻拍



大家好！ 顶级3D商店就在您身边，在接下来的评论中，我们将讨论Anycubic i3 Mega FDM打印机。

录影带



Anycubic i3 Mega 3D打印机



特点


  • 印刷技术：FDM
  • 挤出机数量：1
  • 挤出机温度：最高260˚C
  • 工作台温度：最高100˚C
  • 层厚度：50微米起
  • 定位精度：X / Y-12.5微米，Z-2微米
  • 印刷面积：210x210x205毫米
  • 打印速度：最高60毫米/秒
  • 塑料类型：ABS，PLA，HIPS，木材
  • 螺纹直径：1.75毫米
  • 接口：USB，SD
  • 操作系统：Windows，Mac OS，Linux
  • 尺寸，重量：405x410x4453 mm，11公斤
  • 价格： 24801卢布 该价格于2019年10月10日有效，请在网站上找到实际价格。

内容描述




3D打印变得越来越便宜。 在2018年，几款廉价3D打印机立即出现，这在我们眼前改变了市场，Anycubic i3 Mega就是其中之一。



这是一台FDM打印机，包含在较低价格段中。 它的价格不超过25000卢布。

组装方式




Anycubic i3 Mega可以在最少组装后即可打印-只需将框架固定在底座上，然后用电线连接即可。



组装设计具有很高的刚性。 3D打印机的底座和框架由钣金制成。 借助电线的标记，切换所有模块只需几分钟的时间-连接时不会出错。



我们建议在第一次打印之前进行校准。 此过程是手动执行的。



特色功能


特别感谢Anycubic工程师提供的触摸屏。 它位于方便使用的角度，可快速响应点击。



Anycubic i3 Mega配备有制造商开发的Ultrabase加热台。



3D打印机可与常见的塑料-ABS，HIPS一起很好地工作，但桌面盖也是PLA打印的理想选择。



Ultrabase的表面涂有微孔材料，可在打印过程中牢固地固定物体。 在该过程结束时，将工作台冷却，通过压缩来代替微孔的热膨胀，并且无需费力即可将完成的打印输出从表面去除。



3D打印机配备了一个机械灯丝末端传感器-这对于那些需要大量打印的人来说是一项有用的功能-此简单的设备可让您减少废料量，并具有断电后恢复中断打印的功能。



许多中国3D打印机都没有详尽的文档。 Anycubic i3 Mega是一个很好的例外。 可以在USB记忆棒上找到打印机的所有材料-规格，组装和操作说明以及必要的软件。



套餐捆绑


好的文档要辅以好的设备。



除了由六角扳手，钢丝钳，刮铲和一次性手套组成的传统套件外，该套件还包括：公斤PLA线圈，丙烯酸线圈支架，USB电缆，8 GB SD卡以及备用零件：附加的聚四氟乙烯管，限位开关甚至热端。



您可能会猜到，我们对Prusa i3的主题还有另一种变化。 Anycubic i3 Mega的规格在其细分市场中与其他3D打印机的参数没有区别。 但是组件的质量和性能水平是最好的。



结论


因此，在我们面前的情况是，中国制造商不仅复制了3D打印机，而且以负责任的态度进行了处理，从而改善了原始模型的许多参数。



对于初学者，第一款汽车以及希望通过廉价功能设备扩展车队的高级用户，我们都可以推荐Anycubic i3 Mega 。
