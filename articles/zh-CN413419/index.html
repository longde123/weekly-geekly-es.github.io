发布


如所承诺的那样， PostgreSQL 11的beta版本于5月25日发布。 她在10.x方面的创新已在摘要“冻结功能冻结”中列出。

Postgres Pro 10.4.1
Postgres Professional已基于PostgreSQL 10.4和Postgres Pro 10.3.2发布了新的Postgres Pro Standard / Enterprise 10.4.1版本。 其中包括PostgreSQL 10.4的所有改进，以及许多重要的改进：

  • 更新了负责查询jsonb数据类型的jsquery模块；
  • 用于管理数据库集群备份和恢复的pg_probackup实用程序已更新至2.0.17版，并增加了新功能； 例如，添加了DELTA模式，其中读取目录中的所有数据文件，并为自上一个副本以来已更改的页面创建增量副本；
  • 负责对大型数据库和分布式数据库进行分区的pg_pathman模块已更新至1.4.11版，其中已修复了一些错误；
  • mchar模块中字母“”和“»”的排序已固定，提供了其他数据类型以与Ms SQL Server兼容
  • Windows的改进版本。


奥德赛
Yandex 推出了PostgreSQL的第一个开放源码版本的多线程Odyssey Puller。 它专为多核系统中的繁重负载而设计：每台服务器超过1万个TLS连接和超过10万个TPS，与几个pgbouncers相比，与PostgreSQL的开放连接要少得多。
奥德赛可以：

  • 跨核线性扩展；
  • 在具有自定义身份验证，限制的不同池模式下工作
  • 连接断开时自动回滚事务并终止服务器进程；
  • 正确转发错误。
