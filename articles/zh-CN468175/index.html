前端开发博客值得一读

每个自称为Web开发人员的人都应该了解行业中的趋势和当前事件。 在此基础上，收集有关前端开发的一些有趣的博客。 值得注意的是，大部分的选择都是英文博客。

英文博客


新闻提要和主题账户

  • Smashing Magazine是面向网站设计和开发专业人员的在线杂志。
  • CSS-Tricks是面向Web设计人员的官方CSS-Tricks.com资源。
  • CSS {IRL} -技巧，生活技巧，技巧和与CSS相关的链接。
  • Web平台新闻 -来自同名资源的每日Web开发新闻。
  • Book Apart是有关响应式Web设计，Git和JavaScript，内容策略，设计原理，公开演讲等主题的书籍的出版商。



有趣的人


来自开发界的知名人士，会议的参加者和组织者的博客精选，这些博客对Web的发展做出了巨大的贡献，例如Chrome浏览器开发人员，领先的Google程序员等。

  • Mathias Bynens是V8的开发人员，V8是Google的高性能开源JavaScript和WebAssembly引擎。
  • 杰克·阿奇博尔德 （ Jake Archibald）是网络传播者，也是Google Chrome浏览器的代表，该公司是公司最好的专家之一，并撰写了许多有关独特主题的报告。
  • Google的Chrome浏览器研究工程师Phil Walton自1998年以来一直在开发，并且拥有个人博客 。
  • Monica Dinculescu- “ emojineer”，与Google合作开展了Magenta项目，通过机器学习创作音乐和艺术作品。
  • 蒂姆·卡德莱克 （ Tim Kadlec）是技术生产力顾问，他使网络变得更快，他写了几本书，并参加了会议。
  • Léonie是可访问性专家，并且是W3C Web平台工作组的联合创始人。
  • Eric Meyer是HTML和CSS方面的专家，是许多文章和书籍的作者，还是一些有用的工具和资源的创建者。 详细信息-在他的个人网站上 。
  • Una是Bustle Digital Group的网络宣传员和产品设计总监。
  • 哈里·罗伯茨 （ Harry Roberts）是独立的基于网络的生产力顾问，与主要的国际公司合作。 邀请Google专家，屡获殊荣的开发人员，国际演讲者和SHIFT Commerce的性能大使。
  • Alex Russell是根据Chrome团队标准建立的技术团队，是W3C技术架构小组的成员，也是ECMA TC39（JavaScript标准）的开发者。
  • Surma是Google的网络传播者。
  • ManuelMatuzović自2008年以来是一名专业Web开发人员，自2010年以来是一名自由职业者，是HTML，可访问性，CSS布局和体系结构方面的专家。
  • HH Chen是Nexmo的网络布道者。
  • Jen Simmons是Mozilla的布道设计师，他是CSS工作组的成员，他知道CSS Grid如何改变Web设计。
  • Martin Splitt是OSS1的推广者，也是Google的开发人员。
  • Maximiliano Firtman是移动和Web应用程序的独立开发人员，是致力于创新Web应用程序，生产力，移动性和Web平台的导师和演讲者。 有关编程的许多技术文章和书籍的作者。
  • 雷切尔·安德鲁 （ Rachel Andrew）是一名Web开发人员，是CSS工作组的成员，也是《 砸碎杂志》的主编。
  • Roma Komarov是CSS的前端开发人员，Hayaku，Bemto组件，Pug.js的bemto 等的创建者。
  • 阿迪·奥斯曼 （ Addy Osmani）是Google Chrome浏览器速度团队的负责人，该团队可以使网络速度更快。 Addy还在多个开源项目上工作-在个人网站上有更多详细信息 。



浏览器，编辑器的正式帐户



  • Chrome DevTools-适用于Chrome开发人员的新闻，最新更新和有用的扩展程序。
  • Chrome开发者 -Chrome开发者关系小组的开发者新闻。
  • WebKit是WebKit官方网页显示引擎博客。
  • Mozilla Hacks是开发人员的官方Mozilla Firefox浏览器帐户。
  • Samsung Internet Dev是一个支持Samsung移动和VR浏览器开发人员的平台。
  • “意图运送”是一个博客，当浏览器宣布删除，修改或出现其引擎中的功能时，会自动创建一条推文。
  • Visual Studio Code是其中一种流行的代码编辑器的正式帐户。
  • Atom编辑器 -Atom代码编辑器帐户。
  • Sublime HQ - Sublime专有文本编辑器新闻和更新。
  • JetBrains WebStorm是一个有关前端和服务器端JavaScript的在线开发博客。
