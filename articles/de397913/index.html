Microsoft hat eine Möglichkeit patentiert, einen Benutzer mithilfe von KI zu verfolgen und alle Aktionen in Windows zu registrieren


Der "Mittelsmann" ist die neue Windows 10-Komponente von Microsoft. Vielleicht sieht es auch so aus.

Das neueste Betriebssystem von Microsoft, Windows 10, nimmt einen zunehmenden Anteil am Betriebssystemmarkt ein. Dies ist teilweise auf die Tatsache zurückzuführen, dass Windows 10 die Genehmigung vieler Benutzer erhalten hat. Und teilweise - die Tatsache, dass Microsoft versucht, sein Produkt aktiv zu bewerben. Im vergangenen Jahr wurden die Methoden zur Förderung von Windows 10 durch seine Entwickler häufig in den Medien diskutiert.

Eine weitere Nuance ist die Übertragung großer Mengen von Benutzerdaten unter Windows 10 auf die Server des Unternehmens. Die Telemetrie in den "Top Ten" beunruhigt eine große Anzahl von Benutzern, die herausfinden wollten, warum Microsoft so viele Daten hat. Das Unternehmen gab daraufhin an, dass die erhaltenen Informationen anonymisiert sind und nur dazu dienen, die Qualität seiner Softwareprodukte zu verbessern. Aber die Benutzer mögen das nicht allzu sehr. Jetzt läuft die Firma Redmond Gefahr, eine neue Welle der Unzufriedenheit auszulösen. Tatsache ist, dass Microsoft eine Patentanmeldung eingereicht hat, um einen Benutzer mithilfe einer neuen Betriebssystemkomponente zu verfolgen.

Das Patent heißt " Abfrageformulierung über Task Continuum ". Es beschreibt eine Methode, die den Datenaustausch zwischen einzelnen Anwendungen in Echtzeit vereinfacht. Laut den Entwicklern dieser Technologie wird sie die Effizienz der Arbeit des Benutzers mit Anwendungen und dem gesamten Betriebssystem erhöhen.

Microsoft ist der Ansicht, dass das aktuelle Modell Der Softwarebetrieb ist nicht sehr effektiv, wenn jede Anwendung ihren eigenen Geräten überlassen bleibt und wenig mit ihrer Umgebung interagiert. "Aufgrund der Tatsache, dass einzelne Programme praktisch nicht miteinander verbunden sind UGOM, ein Computersystem ist nicht , dass es den Benutzer zu einem bestimmten Zeitpunkt“erfolgt, - in der Patentanmeldung angegeben.



Als Beispiel führt ein Unternehmen einen Fall an, in dem ein Benutzer in OneNote Notizen zum Thema Tanzen macht. Gleichzeitig sucht dieser Benutzer im Internet Explorer nach Informationen zu Tanzstilen im Web. Diese beiden Anwendungen sind normalerweise nicht miteinander verbunden und tauschen keine Informationen aus. Das Unternehmen schlägt jedoch vor, diese zu beheben.

Die Entwickler des Unternehmens schlagen vor, die Interaktion einzelner Anwendungen mithilfe der "Zwischenkomponente" oder des "Vermittlers" (Mediator) zu verbessern. Sie planen, diese Komponente zu einem separaten Programm zu machen, das die Aktionen des Benutzers überwacht und die gesammelten Daten an andere Anwendungen und Geräte derselben Person überträgt. Die „Komponente“ überwacht nicht nur den vom Benutzer eingegebenen Text, sondern auch seine anderen Aktionen. Suchen Sie beispielsweise im Web nach Titeln oder arbeiten Sie mit Bildern, die auf Ihrer Festplatte gespeichert sind. Darüber hinaus sammelt die „Zwischenkomponente“ Daten aus Anwendungen wie Word, Skype, Edge.

„Der Suchdienst verwendet das kontextbezogene Ranking, um die Bewertung potenzieller Suchelemente anzupassen. Dies geschieht, um die Relevanz der beabsichtigten Suchabfrage zu verbessern. Das Betriebssystem, das die Rolle einer Zwischenkomponente spielt, überwacht alle Textdaten, die dem Benutzer von einer beliebigen Anwendung angezeigt werden, und führt eine Datenanalyse durch, um die Absichten des Benutzers zu bestimmen “, heißt es in der Patentanmeldung.

Das Unternehmen behauptet, dass diese Methode die Arbeit des Benutzers mit einem PC und Suchdiensten beschleunigen wird. Für einige Benutzer sieht diese Methode natürlich wie ein Versuch von Microsoft aus, dem Betriebssystem offiziell einen "großen Bruder" hinzuzufügen. Der "Vermittler" überwacht wirklich alle Aktionen, die Sie auf Ihrem Computer ausführen, sammelt diese Informationen und überträgt den Datendump in Echtzeit an Bing.

Microsoft behauptet erneut, dass das System nur anonymisierte Daten überträgt. Dies wird jedoch wahrscheinlich keine Benutzer beruhigen, die auf ihre Anonymität im Netzwerk achten und der Ansicht sind, dass personenbezogene Daten personenbezogen bleiben sollten.



In der Patentanmeldung wird detailliert beschrieben, wie dies alles im Ökosystem von Microsoft-Produkten funktioniert: „Zum Beispiel im Ökosystem von Geräten mit Windows-Betriebssystem, zu denen zwei oder mehr Geräte gehören können, z. B. Windows Phone, Surface Pro-Tablet, Xbox Das System sammelt und verfolgt Daten von Anwendungen von Drittanbietern, die auf denselben Gadgets ausgeführt werden, und interagiert mit den Betriebssystemen anderer Geräte. Dies geschieht, damit das Betriebssystem auf jedem bestimmten Gerät alles über die Vorgänge mit anderen Geräten im Ökosystem „weiß“. Verstreute Daten werden zu einem Ganzen zusammengefasst. "

Microsoft schlägt vor, das "Intermediary" zu einem optionalen Modul zu machen, das in der Windows 10-Umgebung installiert oder in die Struktur des Betriebssystems integriert werden kann.

Die Tatsache, dass Windows 10 detaillierte Daten über seine Benutzer sammelt, ist seit langem bekannt. Im Jahr 2014 gab Microsoft eine neue Datenschutzerklärung heraus, in der es heißt, dass das Unternehmen möglicherweise Informationen über gebrauchte PC- und Laptop-Programme mit installiertem Windows 10 sowie über die Geräte und Netzwerke, in denen sie funktionieren, sammelt. Solche Daten können mit einer Benutzer-ID kombiniert werden. Es werden auch Informationen über die Vorlieben einer bestimmten Person, den Ort und ihre Interessen gesammelt. Die



meisten aktiv sammelt Sprachassistent Cortana. Die Liste der Server, an die Daten gesendet werden, ist umfangreich. Wenn dies jedoch erforderlich ist, können Sie die Telemetrie in Windows 10 deaktivieren.

Bei der Schaffung des „Vermittlers“ wird er Daten sammeln und an die Server des Unternehmens senden. Möglicherweise werden weitere Informationen gesendet. Nun, die Benutzer müssen sich entscheiden, ob sie mit dem "Vermittler" arbeiten wollen oder nicht. Höchstwahrscheinlich werden Informationssicherheitsspezialisten nach dem Aufkommen einer neuen Betriebssystemkomponente schnell einen Weg finden, die Telemetrie zu deaktivieren.
