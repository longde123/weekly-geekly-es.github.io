Versuchen Sie sich in der Q # -Programmierung

Das Microsoft Quantum Team hat gute Nachrichten für Liebhaber und Programmierer von Quantencomputern, die sich selbst herausfordern möchten. Wir freuen uns, den Microsoft Summer Programming Contest für Q # - Microsoft Q # Coding Contest - Sommer 2018 bekannt zu geben! Durch Ihre Teilnahme können Sie Ihre Fähigkeiten auf dem Gebiet der Quantenprogrammierung verbessern, indem Sie Probleme unterschiedlicher Komplexität mit der Quantenprogrammiersprache Q # lösen. Die Gewinner erhalten Microsoft Quantum T-Shirts!



Quantum Computing ist ein grundlegend neuer Programmieransatz, der mit klassischem Computing wenig gemein hat. Sie sind so unterschiedlich, dass einige Probleme, die in klassischen Modellen als unlösbar angesehen werden (z. B. Faktorisierung von ganzen Zahlen, Berechnung diskreter Logarithmen auf elliptischen Kurven oder Nachahmung physikalischer Systeme), auf einem Quantencomputer effektiv gelöst werden können. Microsoft hat kürzlich das Quantum Development Kit eingeführt, das die neue Programmiersprache Q # enthält, mit der klassische Programmierer eine neue Umgebung leichter erlernen können. Q # lässt sich in Visual Studio- oder Visual Studio Code- Entwicklungsumgebungen integrieren und ist als Befehlszeilentool verfügbar. Mit Visual Studio Code können Sie Anwendungen für Windows, MacOS und Linux entwickeln.

Der Wettbewerb findet vom 6. bis 9. Juli statt und bietet den Teilnehmern die Möglichkeit, Probleme unterschiedlicher Komplexität zu Einführungsthemen aus dem Bereich des Quantencomputers zu lösen: Überlagerung, Messung, Orakel und einfache Algorithmen.

Wettbewerbsregeln


  • Der Wettbewerb wird 12 Aufgaben mit verschiedenen Schwierigkeitsgraden enthalten.
  • Um jedes Problem zu lösen, muss Code in der Q # -Sprache geschrieben werden, damit wir die beschriebenen Transformationen eines bestimmten Satzes von Qubits durchführen oder den Zustand eines solchen Satzes analysieren können. Es werden nur in Q # geschriebene Entscheidungen akzeptiert.
  • Eine Entscheidung gilt als richtig, wenn sie eine vordefinierte Reihe von Tests besteht. Bald nachdem Sie die Entscheidung getroffen haben, können Sie Ihr Ergebnis herausfinden.
  • Die Teilnehmer werden anhand der Anzahl der richtig gelösten Probleme bewertet.
  • Im Falle eines Unentschieden wird der Gewinner in Abhängigkeit von der Gesamtstrafe ermittelt, die die Teilnehmer bei der Lösung aller Probleme erhalten. Die Strafzeit wird wie folgt berechnet: Für jedes gelöste Problem wird eine Strafe in Abhängigkeit von der seit Beginn des Wettbewerbs verstrichenen Zeit berechnet. Für jede falsche Entscheidung wird eine zusätzliche Strafe von 20 Minuten verhängt (wenn Sie das Problem nicht gelöst haben, wird keine Strafe verhängt).
  • Die 50 besten Teilnehmer erhalten ein Microsoft Quantum T-Shirt.
  • KEINE ZAHLUNG ERFORDERLICH. Die Teilnehmer müssen mindestens 16 Jahre alt sein. Der Wettbewerb endet am 9. Juli 2018. Weitere Informationen finden Sie in den offiziellen Regeln .

Vom 29. Juni bis 2. Juli veranstalteten wir eine Aufwärmrunde, in der einfachere Aufgaben zu denselben Themen wie im Hauptwettbewerb vorgestellt wurden. Die Teilnahme an der Aufwärmrunde ist optional. Die Aufwärmrunde ermöglichte es, sich vorab mit den Bedingungen des Wettbewerbs und des Entscheidungssystems vertraut zu machen, Wissen aufzufrischen oder die Grundlagen des Quantencomputers und der Q # zu erlernen. 24 Stunden nach dem Start haben wir Lösungen und Erläuterungen für die drei einfachsten Aufgaben veröffentlicht. Nach Abschluss der Runde haben wir auf der Wettbewerbsseite einen Artikel veröffentlicht, in dem wir die Logik des Quantencomputers hinter der Lösung sowie die Methodik für die Verwendung von Q # enthüllten.

Informationen zum Einstieg finden Sie in den Q # -Installationsanweisungen und Referenzmaterialien für diese Sprache .
