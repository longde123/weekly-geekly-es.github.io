Tabel anggaran untuk pekerjaan berdiri

Baru-baru ini membaca artikel seharga $ 29 untuk instruksi membangun meja transformator , yaitu sekitar 1.885 rubel. ($ 29 * 65 rubel) memutuskan untuk memberi tahu versinya tentang meja untuk pekerjaan tetap. Ternyata tentu saja sedikit lebih mahal, tetapi sebagai hasilnya saya mendapat desktop penuh, dan bukan hanya instruksi.



Sekali waktu saya membaca sebuah artikel tentang manfaat bekerja sambil berdiri, melihat harga solusi siap pakai, saya dengan cepat meninggalkan ide untuk membeli meja seperti itu. Tetapi saya benar-benar ingin mencoba bagaimana rasanya bekerja sambil berdiri. Saya memutuskan untuk merakit meja seperti itu, tetapi dengan kesempatan untuk kembali ke tempat duduk biasa saya jika saya bosan jika perlu.

Sebagai permulaan, saya memilih meja. Saya harus menelepon 10 perusahaan yang menjual furnitur. Ternyata beberapa dari mereka menjual sedikit demi sedikit furnitur. Jadi saya hanya perlu meja, dan saya membelinya. Saya memilih sudut, tetapi tidak terlalu besar, saya dibimbing lebih oleh ukuran ruangan.

Butuh waktu lebih lama untuk memutuskan kaki untuk meja. Dalam artikel tentang pekerjaan berdiri, gambar ini sering dilewati.



Oleh karena itu, gambar I menghitung tinggi kaki yang diperlukan berdasarkan tinggi dan ketebalan meja.

Setelah merakit meja, untuk membuat semuanya terlihat layak, saya membeli saluran kabel dan dudukan untuk unit sistem. Hasilnya adalah tabel yang Anda lihat di gambar pertama.

  • Kaki yang dapat disesuaikan (67 cm - 107 cm) - 1000 rubel. * 4 pcs (4000 gosok.)
  • Saluran untuk kabel - 599 rubel.
  • Dudukan untuk unit sistem (saya juga membelinya dari ikea, tapi saya belum menemukannya sekarang) - ~ 1000 rubel.

Total: 12.599 gosok.

Saya suka bekerja sambil berdiri, tetapi saya tidak bisa bekerja seperti itu sepanjang hari. Karena meja itu dirangkai untuk rumah dan praktis tidak digunakan untuk bekerja (Internet, surat), dalam posisi terangkat itu berdiri selama hampir setahun. Ketika saya harus bekerja untuknya selama beberapa hari berturut-turut, saya harus menurunkan ketinggian kaki untuk pekerjaan duduk yang nyaman.
