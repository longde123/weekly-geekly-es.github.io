Astra Linux "Eagle" Common Edition: ¿hay vida después de Windows?

Recibimos una revisión detallada de uno de los usuarios de nuestro sistema operativo, que nos gustaría compartir con usted.

Astra Linux es un derivado de Debian que se creó como parte de la iniciativa rusa para cambiar a software de código abierto. Hay varias versiones de Astra Linux, una de las cuales está destinada para uso general y cotidiano: Astra Linux "Eagle" Common Edition. El sistema operativo ruso para todos es interesante por definición, y quiero hablar sobre Eagle desde la perspectiva de una persona que usa tres sistemas operativos todos los días (Windows 10, Mac OS High Sierra y Fedora) y ha sido fiel a Ubuntu durante los últimos 13 años. En base a esta experiencia, consideraré el sistema en términos de instalación, interfaces, software, características básicas para desarrolladores y conveniencia desde diferentes ángulos. ¿Cómo se comparará Astra Linux con los sistemas más comunes? ¿Y puede reemplazar Windows en casa?



Ponemos Astra Linux


El instalador de Astra Linux tiene mucho en común con el instalador de Debian. Quizás el primero sea aún más simple, ya que la mayoría de los parámetros se fijan por defecto. Todo comienza con un acuerdo de licencia general en un contexto de edificios no demasiado altos. Quizás incluso en Orel.



Un punto importante en la instalación es la elección del software, que viene por defecto con el sistema. Las opciones disponibles cubren las necesidades estándar de oficina y trabajo (para no desarrolladores).



Además, la última ventana es un conjunto de configuraciones adicionales: bloqueo de los intérpretes, consola, rastreo, configuración del bit de ejecución, etc. Si estas palabras no le dicen nada, es mejor no marcar las casillas. Además, todo esto, si es necesario, se puede configurar más adelante.



El sistema se instaló dentro de un entorno virtual con recursos modestos (sistemas relativamente modernos). No hubo quejas sobre la velocidad y el rendimiento. La configuración que se probó se describe a continuación.



El procedimiento de instalación es el habitual: montamos una imagen iso , la instalamos a través del proceso de instalación del sistema estándar y quemamos el gestor de arranque GRUB.



El sistema cuando se carga es poco exigente con los recursos: alrededor de 250-300 MB de RAM al inicio para el modo de escritorio.



Opciones de inicio alternativas: modo tableta y teléfono


Al ingresar al sistema, puede elegir una de varias opciones de inicio: seguro, de escritorio, móvil o tableta.



Para trabajar en dispositivos táctiles, puede encender el teclado en pantalla.



Veamos qué es interesante en diferentes modos. El escritorio es un modo normal donde el sistema es similar a Windows.



El modo tableta es adecuado para pantallas táctiles grandes. Además de las diferencias externas obvias, que se pueden ver en la captura de pantalla a continuación, hay otras características de la interfaz. El cursor en el modo tableta es invisible, el botón de cierre de la aplicación se coloca en la barra de tareas. Las aplicaciones de pantalla completa funcionan un poco diferente, los archivos en el administrador de archivos también se seleccionan de manera diferente.



Vale la pena mencionar el modo móvil: aquí todo es casi lo mismo que en Android. Se utiliza el entorno gráfico Fly. En los modos táctiles, funciona un toque largo, mediante el cual puede llamar al menú contextual. El modo móvil consume un poco más de recursos en comparación con el escritorio y la tableta.





La presencia de diferentes modos de operación es conveniente. Por ejemplo, si usa una tableta con un teclado enchufable y, en consecuencia, casos de uso táctiles y no táctiles.

Actualización del sistema


Antes de comenzar a usar el sistema, debe actualizarlo. Básicamente, los repositorios de Astra Linux tienen 14 mil paquetes (rama estable , de prueba y experimental ). La rama experimental pronto recibirá actualizaciones inestables, por lo que probaremos la rama de prueba. Cambie el repositorio a prueba.



Comenzamos a actualizar el repositorio y al sistema. Para hacer esto, haga clic en el botón "Actualizar" en la esquina superior izquierda, luego "Marcar todas las actualizaciones", luego "Aplicar". Reiniciar

Política de usuario


Se crean nuevos usuarios en el sistema a través de la utilidad de administración de políticas de seguridad.



De manera predeterminada, se proporciona la función de inicio de sesión remoto (Panel de control - Sistema - Inicio de sesión).



Además de las sesiones habituales separadas y remotas, puede iniciar una sesión anidada (Inicio - Apagado - Sesión).



Con los dos primeros, todo está claro. Una sesión anidada es una sesión que comienza en la ventana de la sesión actual.



Las sesiones, por cierto, se pueden completar después de un tiempo retrasado: no espere el final de las operaciones prolongadas, sino que simplemente configure el apagado automático.



Interfaz Astra Linux y software estándar


Astra Linux Common Edition se parece a Debian como lo fue hace varios años. Es notable que, externamente, Astra Linux Common Edition está tratando de acercarse a Windows.



La navegación y el trabajo con el sistema de archivos está más cerca de Windows que de Linux. Con la imagen del sistema, se adjunta un conjunto estándar de software: oficina, red, gráficos, música, video. La configuración del sistema también se agrupa en el menú principal. Cuatro pantallas están disponibles por defecto.


Como puede ver, LibreOffice se instala como una suite ofimática en el sistema

El panel de control es similar a Windows / Mac / etc. y agrupa la configuración básica en un solo lugar.



El administrador de archivos tiene una interfaz de dos paneles y puede montar archivos como carpetas.





El administrador de archivos puede calcular sumas de verificación, incluidas aquellas de acuerdo con GOST R 34.11-2012 .



Mozilla Firefox se instala como un navegador estándar. Parece bastante ascético, pero al mismo tiempo bastante adecuado. Por ejemplo, abrí y hojeé un Habr nuevo. Las páginas se representan, el sistema no se bloquea y no se bloquea.



La siguiente prueba es la edición de gráficos. Descargamos la imagen del título del artículo de Habr, le pedimos al sistema que la abra en GIMP. Nada inusual aquí tampoco.



Y con un simple movimiento de muñeca agregamos la prueba de la eficacia de uno de los artículos. En principio, no hay diferencias con los sistemas Linux estándar.



Intentemos ir más allá de simples scripts y poner paquetes estándar a través de apt-get.



Después de la actualización del índice: 
sudo apt-get update

Para la prueba, instalamos python3-pip, zsh y realizamos la instalación de oh-my-zsh (con una dependencia adicional de git). El sistema funcionó normalmente.

Como puede ver, el sistema funciona bien en el marco de los escenarios cotidianos estándar del usuario promedio. Si espera ver programas familiares para Debian / Ubuntu aquí, deberá instalarlos adicionalmente con controladores (por ejemplo, si necesita paquetes como ack-grep, se instalan a través de curl / sh). Puede agregar repositorios a sources.list y usar el conocido apt-get.

Utilidades nativas de Astra Linux


Las herramientas descritas anteriormente son solo parte de lo que está disponible para los usuarios de Astra Linux. Además, los desarrolladores han creado alrededor de un centenar de utilidades adicionales que se pueden entregar a través del mismo repositorio que utilizaron para actualizar el sistema.



Para buscar utilidades, simplemente busque la palabra "volar": todas las utilidades necesarias tienen dicho prefijo.



Es difícil hablar sobre todas las aplicaciones en el marco de una revisión, por lo que elegiremos algunas útiles desde el punto de vista de un usuario simple. La aplicación meteorológica muestra el pronóstico en ciudades seleccionadas de Rusia, está optimizada para la región rusa.



También hay una sencilla utilidad gráfica con varios filtros y configuraciones para buscar archivos.



Hay su propia utilidad para monitorear la energía de la batería y varios modos, cuya transición se configura a través de un temporizador: apague el monitor, la suspensión, la hibernación.



La elección de los archivos ejecutables para los comandos también se incluyó en un shell gráfico. Por ejemplo, puede especificar qué "vi" elegirá el sistema al ejecutar el comando.



Con una utilidad de administración separada, puede configurar qué aplicaciones comienzan al inicio del sistema.



También hay monitoreo GPS / GLONASS, bastante útil en un teléfono / tableta (en el que generalmente está presente el módulo correspondiente).



También hay su propio lector de PDF simple, para pruebas lanzadas en el libro Cultura libre de Lawrence Lessig.



Puede leer sobre todas las utilidades de Fly en el recorrido virtual de Astra Linux, en la sección de Ayuda en el escritorio virtual.

Contraste con los principales sistemas.


Desde el punto de vista de la interfaz y la lógica de los controles, el sistema se parece más a Windows XP clásico y, a veces, a elementos individuales de Mac OS.

Desde el punto de vista de las utilidades, la consola y el hardware, el sistema es similar al clásico Debian, que es bastante bueno y familiar para los mismos usuarios de Ubuntu y Minted, aunque los más avanzados perderán la gama habitual de paquetes de todos los repositorios.

Si impones tu experiencia en el retrato de usuarios potenciales, tengo expectativas positivas para el nuevo sistema. Según su experiencia con Windows / Mac, los usuarios habituales pueden sentirse cómodos con Astra Linux Common Edition. Y los usuarios de Linux más avanzados que utilizan utilidades estándar de Unix configurarán todo según les convenga.

La versión actual de Astra Linux está basada en Debian 9.4, y el último núcleo de Debian 10 (4.19) también está disponible.

Por supuesto, hay versiones más nuevas de Ubuntu, pero hay un matiz pequeño pero significativo: no son LTS (Soporte a largo plazo). Las versiones de Ubuntu LTS están alineadas con Astra Linux en las versiones de paquete. El veredicto


Beneficios clave de Astra Linux Eagle Common Edition:

  • No se cae, no se cuelga, no se notan fallas críticas.
  • Imita con éxito las interfaces de Windows NT / XP.
  • Simplicidad y facilidad de instalación.
  • Bajos requerimientos de recursos.
  • El software principal está preinstalado: suite ofimática LibreOffice, editor gráfico GIMP, etc.
  • Un gran conjunto de utilidades adicionales.
  • Las versiones de los paquetes son más antiguas que las versiones recientes de Ubuntu.
  • Su repositorio es más pequeño que el de Ubuntu y Debian.

Conclusión: las últimas versiones de Ubuntu que no sean LTS para usuarios domésticos funcionarán mejor que Astra.

Sin embargo, para los usuarios domésticos, sentarse en una distribución LTS puede no ser relevante, pero para las organizaciones es una opción bastante normal. Por lo tanto, la elección de los desarrolladores de Astra Linux dirigidos al segmento corporativo es clara y lógica.

En cuanto a las deficiencias, es más probable que sean ciertas para aquellos que están acostumbrados a trabajar con Linux, ya que Astra Linux "Eagle" está mucho más cerca de Windows que de Linux.

Astra Linux "Eagle" Common Edition parece un buen reemplazo para la versión ofimática de Windows como parte de la transición al software gratuito del gobierno, y para uso doméstico puede parecer algo conservador.
