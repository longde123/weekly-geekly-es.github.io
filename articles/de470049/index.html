Einführung in NuGet Package Management auf Lösungsebene in Visual Studio für Mac

Visual Studio 2019 für Mac Version 8.3 enthält viele neue Funktionen, die in diesem Blogbeitrag zusammengefasst sind . Während die Gesamtheit dieser Version stark von Ihrem Feedback beeinflusst wurde, war die Möglichkeit, Pakete auf Lösungsebene zu verwalten, eine der Funktionen, die die meisten von Ihnen in Visual Studio für Mac als mangelhaft bezeichneten. Ein neuer NuGet Package Manager auf Lösungsebene ist eine der aufregenden neuen Funktionen von Visual Studio 2019 für Mac Version 8.3.

Wir haben Verbesserungen vorgenommen, damit Sie Pakete leichter finden können. Dies beinhaltet eine verbesserte Erfahrung bei der Suche nach neuen Paketen, ein Verständnis dafür, welche Pakete bereits in Ihrem Projekt installiert sind, und das Auffinden von Paketen, für die Updates verfügbar sind. In diesem Blog-Beitrag konzentrieren wir uns auf die Erfahrung in der Paketverwaltung für eine Lösung. Die meisten Erfahrungen, einschließlich der Installation, Aktualisierung und Anzeige installierter Pakete, haben jedoch auch auf Projektebene eine ähnliche neue Erfahrung.

Um den NuGet Package Manager für eine Lösung zu starten, können Sie im Kontextmenü der Lösung "NuGet Package verwalten ..." auswählen:

Neue Pakete hinzufügen


Wenn Sie suchen und versuchen, ein neues Paket hinzuzufügen, können Sie jetzt die Projekte auswählen, in denen Sie das Paket installieren möchten.

Sie können jederzeit zur Registerkarte Installiert gehen und eine Liste aller in Ihrer Lösung installierten Pakete anzeigen, um sie zu deinstallieren und zu aktualisieren.



Pakete aktualisieren


Auf der Registerkarte Updates werden alle Pakete in der Lösung angezeigt, für die Updates verfügbar sind (oder ein Projekt, wenn Sie den Befehl auf Projektebene aufrufen).



Pakete konsolidieren


Große Lösungen führen häufig zu Situationen, in denen unterschiedliche Projekte auf unterschiedliche Versionen eines Pakets verweisen. Um diese Versionen in einer einzigen Version des Pakets zu konsolidieren, die Sie möglicherweise für die gesamte Lösung verwenden möchten, können Sie auf der Registerkarte Konsolidierung des NuGet Package Managers, der am Lösungsknoten aufgerufen wird, die Version des Pakets auswählen, für die Sie alle Projekte wünschen in der zu verwendenden Lösung und zur Konsolidierung von Paketen:



Laden Sie noch heute herunter!


Laden Sie zum Testen dieser neuen NuGet-Funktionen noch heute die Version 8.3 von Visual Studio 2019 für Mac herunter oder aktualisieren Sie sie über den Stable-Kanal auf die neueste Version, wenn Sie Visual Studio für Mac bereits installiert haben.

Wenn Sie Probleme mit der Version 8.3 haben, verwenden Sie bitte das Menü Hilfe> Problem melden in der IDE, um uns darüber zu informieren. Sie können auch Vorschläge für zukünftige Verbesserungen von Visual Studio für Mac bereitstellen, indem Sie das Menü "Vorschlag bereitstellen" verwenden.
