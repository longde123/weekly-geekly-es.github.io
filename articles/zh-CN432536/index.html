冬季房屋的空气质量



在冬季，室内空气最差。 蓬松，干燥。 加热有效，因为您冻结了，所以通常无法通风。 当家里有小孩的时候尤其如此。
如何保持平衡？ 这可能吗？

在本文中，我将给出数字和计算。 以及他自己的问题解决方案和为此而组装的设备。 谁在乎，请在猫下...

二氧化碳含量


在哈布雷（Habré）上有许多关于二氧化碳，其标准和控制的文章。

因此，我不会写太多。 我只能说必须保持良好的通风以将CO2的水平保持在正常水平。 但是随后又出现了另一个问题：

湿度


室内干燥的空气非常不好。 人有口干，鼻c。 出现静电，衣服破裂和震动。

房子的木制东西干dry并变质。

在低温下，空气可以保留很少的水分。 温度越低越差。

这是在不同温度下空气中最大水分含量的标签：



假设现在温度为-20，相对湿度为60％。

我们考虑一下盘子，我们考虑一下。 1立方米的外部空气中包含0.88 g / m3 x 0.6 = 0.53g的水分。
现在，我们打开窗户，启动了这种舷外机房。 电池将其加热到25度，我们会得到什么？

在25度时，此空气的湿度将为0.53 / 22.8 * 100％= 2.3％。

哇！ 这个很小！ 这是因为25度时空气中的最大水分含量为22.8 g / m3（请参阅表）。

这就是为什么冬天室内空气干燥的原因。 街上越冷，它就越干燥。
如果不采取措施，则湿度会下降到7％，带有干湿温度计的普通湿度计会翻转。

而且，当然，所有的干燥症状都会显现出来。

冬季室内湿度为30-40％，夏季为40-60％。 同时，人们感觉良好。 但是为了确保最低湿度水平为30％，我们需要在每立方米的街道空气中添加：22.8 x 0.3-0.53 = 6.3 g的水。

这仍然是一个非常粗略的计算。 我没有考虑加热时空气的膨胀。 实际上，情况会更糟。

我从哪里可以得到这种失水？ 需要大量补水！ 一个人每小时呼出大约40克水，但这很小。

播出


一个男人呼吸，绝望地破坏了空气。 还有很多空气！

该空气必须通过通风代替。 我们将计算需要为房间通风的精确度。

处于平静状态的人每天呼出800-900克二氧化碳。 进行体育锻炼-最高1.5公斤。

我们假设房间里一个人安静，每天呼出800克二氧化碳，或每小时呼出800/27 = 33.3克。

房间内的二氧化碳含量最高可达1000ppm
在2000ppm之后，它已经非常闷。
在冬天保持1000ppm的标准几乎是不现实的。 怎么了

现在让我们计算：
25度时的空气密度1.18千克/立方米
假设舷外空气是新鲜的晶体，且CO2含量为400ppm。
好吧，这样的理想情况。

如果我们取最大值为1000ppm，那么我们可以“呼气” 1000-400 = 600ppm
或1180 x 600 / 1,000,000 = 0.7g / m3 CO2

如果我们最多取2000ppm，那么2000-400 = 1600
或1180 x 1600 / 1,000,000 = 1.88g / m3 CO2

之前，我们计算出一个人每小时呼出33.3克二氧化碳。 现在让我们计算一个人在一小时内无望地毁了多少立方空气：

如果最大值为1000ppm，则33.3 / 0.7 = 47.6立方米

如果最大值为2000ppm，则33.3 / 1.88 = 17.7立方米

一个普通房间的体积为20m2，高度为2.5m，则为50m3

这意味着，为了在一个只有一个人呼吸的普通房间中保持1000ppm的水平，我们需要每小时用新鲜空气替换几乎整个房间的体积！ 我们需要加热并弄湿所有这些空气，以便根据其他标准将其恢复正常。

哦好 假设电池已完全油炸。 我们计算出需要蒸发多少水，以使从街道流入的如此大量的空气正常化。
我们记得一个人每小时呼出40克水分。 我们将立即考虑到这一点。

如果最大值为1000ppm，则47.6 x 6.3-40 = 260克

如果最大值为2000ppm，则17.7 x 6.3-40 = 71 g

也就是说，为了通过这种通风提高湿度，在第一种情况下，您每天需要花费6.24升水，在第二种情况下，则需要花费1.7升水。 这是每个呼吸的人！
2000/1000 = 2 6.24 / 1.7 = 3.67

依赖性不是线性的。 我们想要的空气越新鲜，我们将不得不花费越来越多的水。 好吧，暖气。 这只是要保持最低湿度！

我认为每个人每天要蒸发掉6.24升水太多。 因此，您不得不在冬天忍受闷气。 没有什么可做的。
我正在尝试在25度的温度下保持不超过1500ppm的CO2水平和30％的湿度。 1500很多，但在此之前更糟。 让我们看看会发生什么。
