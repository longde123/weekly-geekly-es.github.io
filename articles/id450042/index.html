Tata letak keyboard diperpanjang Rusia



Artikel ini didedikasikan untuk semua pengguna keyboard Rusia yang ingin meningkatkan kecepatan mengetik.

Pembuat antarmuka asing jarang peduli tentang adaptasi mereka terhadap budaya asing. Contoh nyata dari antarmuka semacam itu adalah keyboard komputer yang banyak digunakan oleh semua orang (selanjutnya disebut sebagai standar keyboard 104 ANSI).

Layout yang dipilih dalam pengaturan Windows adalah perangkat lunak, yaitu, mereka mengartikan kode pindaian yang diterima dari keyboard ke karakter yang diinginkan. Pengguna berbahasa Inggris hanya menggunakan satu tata letak perangkat lunak dan mereka tidak memiliki masalah dengan peralihan dan kebingungan dengan tanda baca.

Sebagai pengguna yang menulis surat dan mengisi label di MS Excel, setiap kali saya istirahat, ketika saya perlu dengan cepat memperbaiki rumus atau memasukkan data baru, saya menghabiskan banyak waktu memperbaiki kesalahan yang timbul dari terus-menerus mengubah tata letak dari bahasa Rusia ke bahasa Inggris, dan saat menulis surat dengan tanda baca Rusia yang benar, saya selalu menemukan kebutuhan untuk memasukkan karakter yang sama sekali tidak ada pada keyboard (-, ",", ruang yang tidak dapat dipisahkan).

Saya mengidentifikasi sendiri tiga masalah yang memperlambat pekerjaan:

  • Ketidaknyamanan cara standar untuk beralih tata letak.
  • Kurangnya tanda baca yang sering digunakan dalam bahasa Rusia.
  • Tanda baca menandai ketidakcocokan antara tata letak.

Solusi untuk masalah ini dapat ditemukan di GOST yang ada.

GOST 6431-75 . Mesin menulis. Lokasi tombol dan karakter pada keyboard.

Menurut GOST ini, tanda baca yang sering digunakan terletak di kisaran digital blok kunci alfanumerik. Nomor-nomor tersebut dihubungi menggunakan kunci transfer register. Untuk memasukkan koma, Anda tidak perlu menerjemahkan register setiap kali. Tanda koma dan titik diketik dengan jari telunjuk. Huruf ada dalam surat, bukan rentang digital. GOST ini sesuai dengan satu-satunya tata letak Rusia alternatif di Windows - diketik.

GOST 14289-88 . Fasilitas komputer. Keyboard Pengaturan kunci dan simbol, fungsi tombol kontrol.

GOST dikembangkan pada saat IBM Model M baru saja menjadi standar.
Ini memberikan dua solusi sederhana untuk beralih masalah tata letak:

  • Dengan kunci yang menyediakan pilihan konstan dari mode input huruf-huruf alfabet Rusia (CapsLock).
  • Menggunakan tombol untuk sementara mengubah mode input karakter dari huruf Rusia dan Latin untuk kunci alfabet dan untuk sementara mengaktifkan kasus ketiga untuk kunci dengan angka (Alt kanan).

Di semua tata letak, CapsLock digunakan untuk memasukkan lapisan "huruf besar". Tetapi dimungkinkan untuk memasukkannya bukan dalam huruf besar, tetapi dalam alfabet alternatif.


Fig. 1. Tata letak AS

Banyak tata letak nasional memiliki lapisan tambahan yang untuk sementara dihidupkan dengan tombol alt. Dimungkinkan untuk membuat beberapa lapisan tambahan menggunakan alt + tombol apa saja pada blok teks.

Dengan demikian, dalam satu tata letak, Anda dapat memuat beberapa huruf dan semua jenis karakter khusus.


Fig. 2. Tata letak Rusia Windows

Terinspirasi oleh keputusan ini, saya memutuskan untuk menghidupkannya. Saya tidak dapat membuat atau menginstal utilitas khusus. Microsoft Keyboard Layout, program Microsoft resmi untuk membuat tata letak untuk bahasa apa pun, datang untuk menyelamatkan.

Itu mungkin untuk menyelesaikan semua masalah hanya dengan membuat tata letak Rusia-Inggris tunggal dengan tanda baca yang sama untuk kedua bahasa.


Fig. 3. Tata letak diperpanjang Rusia

Untuk menggabungkan tanda baca, saya harus meninggalkan angka dalam rentang digital (ada blok digital), memberi titik pada angka 8, dan koma pada angka 5. Cara termudah untuk mencapai tombol-tombol ini adalah dengan jari-jari Anda dan tidak ada karakter ketiga di dalamnya.

Untuk kemungkinan menggunakan tata letak pada laptop, saya meletakkan angka di lapisan alt.

Dia meninggalkan tanda-tanda yang tersisa di tempat mereka dan menambahkan tanda-tanda yang sering digunakan dalam tulisan Rusia.

Apa yang memberi saya:

  • Kemampuan menulis teks Rusia dengan cepat dan kompeten dari aplikasi apa pun.
  • Jangan pikirkan apakah saya berada di tata letak yang benar untuk mengetik simbol ini atau itu.
  • Ganti tata letak oleh CapsLock tanpa utilitas.
  • Beralih sementara tata letak melalui alt yang tepat.
  • Program tidak lagi mengubah tata letak sendiri. Saat memulai program apa pun, Anda akan tetap berada dalam alfabet yang Anda alihkan sebelumnya.

Bonus

Untuk satu set tanda dan huruf kekuatan dari bahasa Yunani dan huruf lainnya, saya menambahkan layer lain, untuk aktivasi yang Anda perlu mengetikkan alt + \.
