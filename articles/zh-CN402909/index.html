对于那些有品味的人：佳能，尼康和索尼相机的高级镜头

无论智能手机如何出色，物理学都不能被愚弄。 这意味着用普通相机拍摄的照片将胜过使用照相手机拍摄的照片。 特别是如果您使用正确的镜头。



据统计，尽管有大量照片计划，但仍有相当数量的新人购买了第一台可互换镜头的相机，并配上了完整的“眼镜”。 而且，这里的意思甚至不是他们对接收到的人员的素质完全满意-镜头很多，价格也相差很大，要弄清所有这些镜头并不容易，而且需要很多时间。

在这篇文章中，我们要推荐最适合特定类型且价格合理的镜片。

第一点：在声明中，相机拍摄的帧优于智能手机上的类似图像，这有点诡异。 在智能手机中使用高质量相机，晴天和在智能手机屏幕上查看照片的情况下，总体计划中某些图片的差异可能很小。 但是，一旦我们深入研究各种类型（例如，肖像）或尝试在黄昏拍摄，然后将照片显示在笔记本电脑屏幕上，物理学定律就会无情地提醒自己。

第二点：为什么我们在评论中仅限于佳能，尼康和索尼。 不需要介绍前两家公司，“ DSLR”的主要部分正是在这些标签下发布。 好吧，索尼制造的简直令人难以置信的无反光镜相机，例如， 索尼Alpha A5100被认为是最好的入门级无反光镜相机之一。 好吧， 索尼Alpha A6000在所有销售排行榜中都牢牢扎根。

初始镜头


通常，第一个值得尝试的镜头是所谓的固定镜头，即 定焦镜头。 缺少活动部件（因此，需要解决一系列工程问题）降低了镜头的价格，并使您可以在单焦距下获得最高质量的图像。

实际上，借助此修复程序，您可以看到相机如何拍摄。 好吧，第一个解决方法是采用所谓的35s，即 可以紧密模拟人类视力的镜头。 结果，物体以与您的眼睛感知它们相同的方式出现在照片中。 既不近也不远。

第三点：以下，在选择镜头时，我们将重点放在所谓的“裁剪”镜头上。 裁剪矩阵。 因为新手摄影师不太可能拥有全画幅相机。 但是，为了避免在购买时感到困惑，我们将按照镜头上指示的焦距进行书写，即 无需重述。

完善的修复程序：

  • 对于佳能：这里值得偏离建议的约35毫米，并仔细研究一下佳能EF50mm F / 1.8 （ 8,990卢布）作为最便宜的选择；
  • 对于尼康： 尼康AF-S DX尼克尔35mm f / 1.8G （14990卢布）;
  • 对于索尼： Sony E 35mm f / 1.8 OSS （32990卢布）。

索尼的镜头似乎价格过高（就其价格而言，您可以购买更快的光圈佳能EF50mm F1 / 4），但它也具有图像稳定器，可让您以较低的快门速度进行拍摄。 换句话说，在光线不足的情况下，您可能会变得无镜框。


Sony E 35mm f / 1.8 OSS（Takuritsu Kou Flickr ）上的样本照片

如果您仍然认为Sony的价格过高，可以选择Sigma 30mm F / 2.8 DN 。 它并不总是在销售中，但是它是企业修复的一种有价值的替代方法。 是的，速度不是那么快，模糊也不是那么“乳脂状”，但成本也要低得多。

第四句话。 为了准确了解如何将这些镜头与相机一起拆卸，我们建议使用著名的Pixel Peeper网站，该网站可让您查找所需的照片。 本文中所有说明相机功能的图片都是通过此算法精确获得的。

变焦镜头


所有新手摄影师都非常喜欢变焦。 如果您打算拍摄野生生物，运动比赛或孩子在露天剧场，我们建议您考虑以下型号：

  • 对于佳能： 佳能EF70-200mm f / 4 L USM （ 47990卢布）。 他看起来和射击都很好。 同时，尽管专业镜片具有白色特征，但价格并不昂贵。 这款镜头经过修改，带有内置稳定器，但价格却贵了两倍。


（卡夫卡·弗里克（M. Kafka Flickr ））

  • 对于尼康： 尼康70-300mm f / 4.5-5.6G ED IF AF-S （ 39990卢布）。 光圈不如佳能佳，但有稳定器。
  • 对于Sony：Sony E 55-210mm F4.5-6.3 OSS。 带有图像稳定器的漂亮镜头。

广角镜


通常，在变焦之后，摄影师立即开始涉足广角镜。 因为不再移开远处的物体而是将尽可能张开的全景传达给人们变得更加有趣。 在城市摄影过程中，尤其需要这样的镜头-密集的建筑物不允许您移动足够的距离，这使您可以将所需的所有物品放入框架中。 或在拍摄聚会时。

第五点：广角镜非常适合房地产经纪人和酒店业主。 多亏了这个镜头，房间看起来比实际更大。

该类别中最有趣的镜头：

  • 对于佳能： 佳能EF-S 10-18mm f / 4.5-5.6 IS STM （17,990卢布）。 在其价格类别中的绝佳选择。 从这个角度来看，大多数评论家都说，对于这样的钱，这是一个非常非常好的建议。

  • 对于尼康：出乎意料的是，但适马10-20mm f / 3.5 EX DC HSM （ 31,990）具有价格/质量的最佳组合。 为什么是他？ 事实是，大多数广角镜都有一个标准问题，即设计本身引起的问题–镜框变形。 该软件在处理照片时很容易修复。 但是对于Sigma，这些失真很小。 这意味着处理照片更容易，效果更好。 来自著名DXOmark.com项目的专家对镜头进行了测试，并同意它与尼康AF-S DX尼克尔10-24mm F / 3.5-4.5G ED相比并不逊色。



  • 对于索尼：Sony 10-18mm f / 4（62990卢布）很好。 我们付出了很多–但我们也获得了高光圈，光学稳定器和自动自动对焦。 原则上，对于广角镜头，自动对焦不是那么重要。 如果卸下它，稳定器和Sony本身的标签，则您会获得优质的Samyang 12mm F2.0 NCS CS。 但是很难找到特价商品。

人像镜头


同样，这里有必要返回到修复程序，因为很难为肖像画提供更好的东西。 相对较低的价格和始终如一的高质量，很少让我们考虑其他问题。

最有趣的模型：

  • 对于佳能：您可以在本文开头尝试使用相同的佳能EF50mm F / 1.8 ；
  • 对于尼康：尼康AF-S尼克尔85mm f / 1.8G；
  • 对于索尼： Sigma 60mm f / 2.8 DN （这种玻璃的价格令人难以置信，价格为7,990卢布）。 以下是使用Sony ILCE-3000相机拍摄的示例。



休闲镜
