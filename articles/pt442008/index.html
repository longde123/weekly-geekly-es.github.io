O k3s é um Kubernetes pequeno, mas certificado pela Rancher Labs



Requisitos:

  • Linux 3.10+
  • 512 MB de RAM por servidor
  • 75 MB de RAM por nó
  • 200 MB de espaço em disco
  • x86_64, ARMv7, ARM64

A Rancher Labs lançou um novo produto para Edge, IoT, CI com suporte a ARM: k3s

Não há suporte para Legacy / Alpha, há um substituto etcd3 para o sqlite por padrão (você pode ativar o etcd3).

Em vez do Docker, o container é usado: não há libnetwork, swarm, drivers para armazenamento do Docker.
Para reduzir o consumo de memória, o Kubelet, o kubeproxy e o agente de flanela são combinados em um único processo.

O instalador é simples, os certificados são gerados automaticamente.
