PHP internal


  • [RFC] Peta lemah - Dukungan untuk tautan lemah ditambahkan dalam PHP 7.4 melalui kelas WeakReference khusus. Namun pada kenyataannya, aplikasi membutuhkan koleksi WeakMap yang tidak dapat diimplementasikan berdasarkan WeakReference. Sebenarnya, diusulkan untuk menambahkannya.
    Teks tersembunyi 
    $map = new WeakMap(); $obj = new stdClass(); $map[$obj] = 42; var_dump($map); // object(WeakMap)#1 (1) { // [0]=> // ["key"] => object(stdClass)#2 (0) {} // ["value"] => int(42) // } // } //   , //      WeakMap unset($obj); var_dump($map); // object(WeakMap)#1 (0) { // }
  • [RFC] Menerapkan DOM Living Standard APIs di ext / dom - Proposal untuk menerapkan dukungan untuk standar DOM baru dalam ekstensi kernel bulat.
  • [RFC] Operator Backtick yang Tidak Tepat (V2) - Ditolak oleh suara.
  • [RFC] Union Types 2.0 - Proposal diterima hampir dengan suara bulat dan dalam PHP 8.0 kami menunggu jenis gabungan. Sintaks T1|T2|... dapat digunakan di mana pun jenis dapat ditentukan sekarang.
  • PHP Internals News podcast # 36 - Tentang RFC yang tidak lulus dalam PHP 7.4.
