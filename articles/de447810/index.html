Analytics für Azure DevOps Services ist jetzt öffentlich verfügbar

Berichte sind eine wichtige Gelegenheit für Azure DevOps-Benutzer, die sich auf Analytics (Azure Analytics Service) verlassen, um datengesteuerte Entscheidungen zu treffen.

Wir freuen uns, Ihnen heute mitteilen zu können, dass die folgenden Analytics-Funktionen ohne Aufpreis in Azure DevOps Services enthalten sind. In naher Zukunft werden Kunden diese Änderungen in ihren Konten sehen.



Analytics-Funktionen jetzt in Azure DevOps Services verfügbar


  • Analytics-Widgets - benutzerdefinierte Module, die Daten im Dashboard anzeigen und den Fortschritt verfolgen.
    1. Burndown und Burnup - Überwachung des Arbeitsfortschritts in bestimmten Bereichen für einen bestimmten Zeitraum.
    2. Zykluszeit und Vorlaufzeit - Visualisiert den Produktentwicklungszyklus in Ihrem Team.
    3. Kumulatives Flussdiagramm (CFD) - Verfolgen von Arbeitselementen, während sie verschiedene Zustände durchlaufen.
    4. Geschwindigkeit - Verfolgen, wie ein Team über mehrere Sprints hinweg Mehrwert schafft.
    5. Testergebnis-Trend - Verfolgen von Testtrends, Identifizieren von Fehlern und Testdauer für eine oder mehrere Pipelines (Azure-Pipelines).

  • In Produkterfahrungen - Analytics wird in DevOps Azure und außerhalb des Dashboards ausgeführt, in dem Daten und Analysen angezeigt werden.
    1. Top Failing Test Report - Machen Sie sich ein Bild von den schwerwiegendsten fehlgeschlagenen Tests in Ihrer Pipeline.


Wir werden weiterhin Power BI-Integration über Analytics Views und direkten Zugriff auf OData Endpoint im Vorschaumodus für alle Azure DevOps Services-Kunden anbieten. Weitere Informationen zum Preismodell für die Integration von Power BI und OData sollten bis Juni 2019 erwartet werden.

Aktuelle Azure DevOps Services-Kunden, auf denen die Analytics-Erweiterung auf dem Marktplatz installiert ist, können Analytics weiterhin wie zuvor verwenden und müssen keine zusätzlichen Schritte ausführen, um Analytics abzurufen. Daher werden wir die Marketplace Analytics- Erweiterung für gehostete Kunden verurteilen.

Azure DevOps Server 2019


Für Azure DevOps Server bleibt die Analytics-Funktion als installierbare Erweiterung auf dem lokalen Markt im Vorschaumodus und wird in der nächsten Hauptversion öffentlich verfügbar sein.

Azure DevOps Analytics ist die Zukunft der Berichterstellung. Wir werden weiterhin in die neuen Funktionen von Analytics investieren. Um mehr über Analytics und die derzeit angebotenen Möglichkeiten zu erfahren:
