Alat uji otomatis, integrasi Yandex Mapkit 3, desain keren, dan pendekatan UI Berbasis Server - Pengumuman mitap Android

Hai Nama saya Andrey Danilov, saya adalah pengembang android di Avito. Saya ingin mengundang Anda ke mitap Android, yang akan diadakan di kantor kami pada 28 September. Kami akan membahas pengalaman mengintegrasikan Yandex Mapkit 3 ke dalam aplikasi, alat untuk autotest, berbicara tentang cara membuat desain aplikasi yang benar-benar mengesankan, dan mempelajari cara menggunakan pendekatan UI Berbasis Server untuk meluncurkan fitur dengan cepat. Pembicara dari Revolut, Ozon, Surf dan, tentu saja, Avito akan naik ke lantai. Di bawah potongan - abstrak, tautan ke pendaftaran dan penyiaran video.



Laporan


Yandex Mapkit 3 tidak begitu menyakitkan. Andrey Danilov (Avito)



Saya akan bercerita tentang pengalaman mengintegrasikan Yandex Mapkit 3 ke dalam aplikasi Domofond tanpa menceritakan kembali dokumentasi, hanya berdasarkan pengalaman saya sendiri. Saya akan mempertimbangkan perpustakaan baik dari sisi pengembang dan perusahaan yang menggunakan MapKit secara keseluruhan, dan dari pengguna akhir.


Cara membuat aplikasi semangat untuk Android. Mikhail Zinchenko (Surf)



Semua aplikasi perusahaan besar sekarang terlihat sama bagusnya. Sama-sama Semakin lama, bisnis menentukan persyaratan templat demi kesederhanaan dan kecepatan pengembangan. Tetapi mengembangkan aplikasi seperti itu hanya membosankan. Proyek yang menjadi dasar laporan saya berfungsi sebagai bukti bahwa pengembangan ponsel tidak hanya mengecat tombol dan menyeret widget dalam tata letak. Ini adalah kasus ketika aplikasi seharusnya tidak hanya cantik. Saya akan berbicara tentang beberapa kesulitan teknis tak terduga yang mungkin Anda temui ketika mengembangkan UI untuk aplikasi sederhana (sekilas pandang) untuk melihat konten.


Cara menjalankan tes otomatis dan alat apa yang kami butuhkan. Dmitry Movchan (Revolut)



Sudah ada banyak kerangka kerja dan solusi asli untuk menjalankan autotes untuk Android, tetapi apa yang harus dipilih dan bagaimana perbedaannya? Alat apa yang akan berguna saat menulis autotest? Kami akan membahas ini dan masalah lainnya selama laporan saya.


Buka fitur dengan cepat dengan Server Driven UI. Alexander Sviridov (Ozon)



Setahun yang lalu, kami di Ozon mulai membangun seluruh UI widget di backend. Selama ini, kami membuat beberapa alat dengan pendekatan Server Driven UI. Saya akan memberi tahu Anda apa yang mereka ketahui: bagaimana kami mendukung seratus widget, bagaimana kami membangun sistem desain, sistem analitis, masalah apa yang muncul dan bagaimana kami melawan mereka secara heroik. Laporan ini akan berguna jika Anda membuat kerangka kerja sendiri atau platform e-commerce.


Meja bundar


Setelah mitap, kami akan mengadakan pesta ulang tahun di salah satu bar terdekat. Kami berencana untuk mengatur meja bundar tentang karir pengembangan pengembang android. Moderator adalah Dmitry Movchan. Tetap jika Anda tertarik dengan topik ini atau Anda siap untuk berbagi pengalaman Anda.


Kata sandi dan penampilan


Mitap akan dimulai pada 28 September pukul 12:00. Partisipasi dalam acara ini gratis, tetapi Anda harus mendaftar . Harap sebutkan nama belakang dan nama depan Anda seperti pada paspor Anda, dan jangan lupa untuk membawanya (atau SIM) bersama Anda, jika tidak, Anda tidak akan diizinkan masuk ke kantor.


Alamat: Kantor Avito, Lesnaya 7 .


Jika Anda tidak bisa datang, maka sambungkan ke siaran langsung di saluran YouTube AvitoTech .


Sampai ketemu lagi!
