Mail.Ru secara resmi membuka sumber ICQ untuk Windows dan OS X (dan mengumumkan klien untuk Linux)


Seperti berita di situs mengatakan , ICQ sepenuhnya ditulis ulang di Qt, dan karena itu mereka mengunggah kode sumber lengkap ke GitHub .

Sebelumnya, ada upaya yang gagal untuk mengeluarkan kode sumber dan kunci pribadi untuk Sparkle (sistem pembaruan untuk OS X) masuk ke domain publik.

Sekarang repositori sudah dibersihkan, dan di samping itu, rakitan baru untuk Windows sudah tersedia untuk diunduh . Versi untuk OS X dan Linux (!) Sedang dikembangkan, manfaat Qt adalah kerangka kerja lintas platform dan Anda dapat membangun hampir semua hal.

Menilai oleh fakta bahwa mereka memposting tepat satu komit, pengembangan ICQ tidak pindah ke GitHub, dan repositori hanya diperlukan untuk meletakkan kode sumber untuk versi saat ini, seperti yang disyaratkan oleh lisensi Qt gratis. Meskipun saya ingin berharap bahwa proyek ini setidaknya akan merespon permintaan dan masalah pada github.

Baiklah, mari kita berharap untuk pengembangan lebih lanjut dari arah OpenSource Mail.ru dan tunggu garpu baru dan klien alternatif!

PS: sayangnya, modul VoIP tetap tertutup dan datang dalam bentuk perpustakaan statis.


Meskipun ada tajuk dengan komentar rinci .

UPD: Versi debug VoIP telah dihapus dari repositori . Tapi sejarah ingat segalanya .

UPD2: Dan ini diaartikel resmi dari Mail.Ru. Tetapi jenis lisensi tidak pernah ditulis.
