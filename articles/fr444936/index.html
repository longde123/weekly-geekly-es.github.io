Firmware ZXHN H118N de Dom.ru sans soudure et programmeur


Bonjour

Je l'ai sorti du placard poussiéreux. J'avais vraiment besoin du ZXHN H118N de Dom.ru.

Le problème réside dans son court firmware, qui est lié au fournisseur dom.ru (ErTelecom), où vous ne pouvez entrer que le nom d'utilisateur et le mot de passe PPPOE pour vous connecter à Internet.

Cette fonctionnalité est suffisante pour une femme au foyer, mais pas pour moi.

Par conséquent, nous allons reflasher ce routeur!

La première difficulté est que vous ne pouvez pas le reflasher non plus. Dom.ru s'en est bien occupé. Il est possible uniquement sur le firmware Dom.ru. Mais ici, dans l'ancien firmware dom.ru, sur lequel vous pouvez restaurer (rétrograder), il est possible de remplir la version originale du firmware ZTE.

1. Entrez dans la barre d'adresse 192.168.0.1/html/menu.html, puis cliquez sur n'importe quel caractère - il affichera la webcam d'usine.

2. Accédez à la page "Gestion"

3. Onglet «Mise à niveau du firmware»

4. Tout d'abord, flasher le firmware de DOMRU - DOMRU__ZXHN + H118NV2.0.1d_E04_OV.img (ancien firmware pour home.ru avec possibilité de mise à niveau).

5. Après le firmware, le routeur reçoit IP - 192.168.0.1. Mais à travers le WEB ne permet pas d'entrer dans le visage. Il n'est possible que via TELNET - Utilisateur: root Pass: root (qui s'en soucie).

6. Appuyez sur le bouton pour réinitialiser, le traduire dans l'état par défaut.

6.1. Peut-être que le routeur changera l'adresse en 192.168.1.1, et également dans les paramètres de votre appareil connecté au routeur, vous devez entrer manuellement les détails
IP 192.168.1.2
Masque 255.255.255.0
Passerelle 192.168.1.1
Sans cela, au 192.168.1.1, j'ai été redirigé vers la page 404

7. Il y a accès au WEB avec l'utilisateur: pass admin: admin

8. Ensuite, à partir du point 4. Flasher uniquement le firmware d'origine extrait de l'archive - ZXHN + H118NV2.0.1d_E04_OV.img

9. accès au WEB avec l'utilisateur: pass admin: admin

PS Sur Internet, de nombreux forums sont remplis de sujets similaires et d'une surabondance de méthodes et d'informations. C'est une méthode spécifique qui m'a aidé, j'espère qu'elle vous aidera.

Mettre à jour Sur les nouvelles versions de la version matérielle V2.3 a couvert cette faille. Valable uniquement sur les anciennes versions.
