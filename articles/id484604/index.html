Tahun Baru, Peramban Baru: Microsoft Edge Keluar dari Penilaian Awal dan Sekarang Tersedia untuk Diunduh



Sedikit lebih dari setahun yang lalu, kami mengumumkan niat kami untuk membangun kembali Microsoft Edge berdasarkan proyek sumber terbuka Chromium. Ini diperlukan untuk meningkatkan kompatibilitas bagi semua pengguna, mengurangi fragmentasi untuk pengembang web dan membangun kemitraan dengan komunitas Chromium untuk meningkatkan inti Chromium itu sendiri. Di Ignite, kami mempresentasikan visi baru kami tentang teknologi dan pencarian web, logo warna-warni baru dan konsep baru untuk Microsoft Edge + Bing - browser dan mesin pencari untuk bisnis. Dan sekarang kami menantikan tanggapan Anda!

Kami melakukan pekerjaan yang luar biasa dan hari ini kami dengan senang hati mengumumkan bahwa Microsoft Edge baru sudah tersedia untuk diunduh di semua versi Windows dan macOS yang didukung dalam lebih dari 90 bahasa. Microsoft Edge juga tersedia di iOS dan Android, yang berarti dukungan lintas platform yang hebat. Microsoft Edge yang baru menawarkan kinerja kelas dunia dengan peningkatan privasi, kecepatan, dan efisiensi - dua kali lebih cepat dari Edge sebelumnya. Kebijakan privasi kami berlaku untuk browser baru, dan Anda akan menghargai fitur-fitur baru seperti pencegahan pelacakan, yang diaktifkan secara default dan memberikan tiga tingkat kontrol saat menjelajah web.

Fitur inovatif lainnya di Microsoft Edge memungkinkan Anda untuk menyesuaikan interaksi Anda dengan halaman web, misalnya, untuk memilih tata letak, desain, dan konten tab baru untuk memanfaatkan lingkungan online secara lebih baik.



Selama beberapa bulan terakhir, kami telah dengan antusias mengerjakan fitur-fitur baru untuk Microsoft Edge, termasuk dukungan AAD, mode Internet Explorer, streaming video 4K, Dolby Audio, tulisan tangan PDF, integrasi Pencarian Microsoft dengan Bing, dukungan untuk ekstensi Chrome dan lebih banyak.

Jika Anda seorang administrator TI dalam bisnis atau pendidikan dan ingin melakukan penyebaran luas di organisasi Anda, kami juga memikirkan Anda: Anda dapat mengunduh paket dan kebijakan mandiri, serta mengetahui semua detail di situs komersial baru.



Orang-orang telah mengunduh jutaan versi Microsoft Edge baru sebelum diluncurkan ke perangkat mereka, dan banyak organisasi telah mulai menyebarkan versi ini di antara para pengguna mereka. Bisnis dan sekolah yang memiliki aplikasi dan situs web lama yang kritis, tetapi yang juga membutuhkan antarmuka web dan keamanan modern, menggunakan mode Internet Explorer baru kami untuk "membunuh dua burung dengan satu batu." Dan pengguna Microsoft 365 hanya perlu memasukkan kueri yang diinginkan ke bilah alamat Microsoft Edge untuk mencari file, orang, rencana kantor dan banyak lagi menggunakan Microsoft Search di jaringan lokal organisasi mereka. Mereka yang sudah menguji fitur ini telah memuji mereka.

Beralih ke Microsoft Edge baru - apa yang diharapkan?


Sekarang kita telah mencapai tonggak ini, pengguna komputer dan gadget mungkin bertanya-tanya apa yang diharapkan. Ada dua cara untuk mendapatkan Microsoft Edge baru: Anda dapat mengunduhnya secara manual sekarang atau, jika Anda adalah pengguna biasa, tunggu sampai itu secara otomatis diinstal pada perangkat Anda menggunakan Pembaruan Windows. Ketika Anda memutuskan untuk beralih, kata sandi, favorit, informasi Anda untuk mengisi formulir dan pengaturan dasar akan ditransfer ke Microsoft Edge baru secara otomatis. Baca lebih lanjut tentang rencana penerapan kami di sini.

Jika Anda seorang administrator TI, Anda harus mengunduh paket penerapan mandiri untuk penerapan pilot di lingkungan perusahaan Anda, karena Microsoft Edge baru tidak akan secara otomatis digunakan untuk pelanggan komersial. Selain itu, tidak ada saluran Microsoft Edge Insider yang akan ditingkatkan ke Microsoft Edge baru, karena saluran ini dapat digunakan secara paralel untuk pengujian dan validasi.

Kami memahami bahwa menggunakan peramban baru bukan hanya "memutar sakelar," jadi kami ingin membuat transisi semudah mungkin. Selain menyederhanakan penyebaran dengan alat-alat seperti Intune dan Configuration Manager, kami melakukan lebih banyak untuk membantu organisasi Anda pindah ke Microsoft Edge yang baru. Di Ignite, kami mengumumkan dukungan untuk FastTrack dan App Assure untuk Microsoft Edge. FastTrack akan membantu Anda menginstal Microsoft Edge di organisasi Anda tanpa biaya tambahan, jika Anda sudah memiliki langganan yang sesuai untuk Microsoft 365, Azure atau Dynamics 365. Jika situs Anda kompatibel dengan Internet Explorer 8 dan lebih tinggi, dengan Google Chrome atau dengan Microsoft Edge lama, maka mereka akan bekerja dengan Microsoft Edge baru. Jika tidak, hubungi App Assure dan kami akan membantu Anda memperbaikinya.

Apa selanjutnya


Tentu saja, inovasi, pengujian, dan fitur-fitur baru tidak akan berhenti muncul. Rilis pertama ini hanyalah permulaan. Jika Anda ingin melihat apa yang terjadi selanjutnya, kami menganjurkan Anda untuk terus menggunakan saluran pra-penilaian Beta, Dev, dan Canary kami, yang akan tetap tersedia untuk diunduh di situs web Microsoft Edge Insider. Anda tidak hanya dapat mengenal saluran fitur kami untuk Microsoft Edge, tetapi Anda masih dapat membantu meningkatkan Microsoft Edge dengan umpan balik Anda yang berharga. Partisipasi Anda akan membantu menjadikan Microsoft Edge baru dan seluruh Internet secara keseluruhan menjadi lebih baik - untuk semua orang di dunia.

Terima kasih


Terima kasih banyak kepada komunitas Microsoft Edge Insiders kami dan para insinyur Chromium yang bekerja bersama kami untuk menciptakan Microsoft Edge baru. Kami terus berpartisipasi aktif dalam proyek Chromium sumber terbuka dan berkontribusi pada pengembangannya. Hingga saat ini, kami telah menawarkan lebih dari 1900 peningkatan dalam hal aksesibilitas, entri data modern (termasuk sentuhan, suara, digital dan tulisan tangan), dan di banyak bidang lainnya.

Terus beri tahu kami apa yang berfungsi dengan baik dan apa yang perlu diubah. Dan tentang apa yang ingin Anda lihat di versi Microsoft Edge yang akan datang.

Terima kasih kami yang tulus! Kami tidak bisa mencapai ini tanpamu!
