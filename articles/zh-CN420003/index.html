JavaFX上带有按钮的窗口：

你好

直到最近，我对创建任何图形界面的知识还是零。 因此，决定稍微摆动一下Internet并创建一个带有一些按钮的窗口，单击该按钮会发生一些事情。 也许在本文中有人会找到曾经出现的问题的答案。 同时，我希望提出严肃认真的批评。

会发生什么？


由于我的目标是处理JavaFX，因此我需要在窗口中提供某种功能。 这个想法是这样的：创建一组人（假设他们在公司工作），我们将为每个人知道姓名，年龄，薪水和婚姻状况。 我们的窗口首先显示每个人的信息，其次，更改员工的薪水，其次，可以添加新员工，最后，可以根据需要过滤员工从其属性的值。 结果应该是这样的：



开始使用


首先，我们创建公司的主类，从该类将启动应用程序： 

package company; public class Company { public static void main(String[] args) { } }

在这个类中，我们将扩展 Application类的功能。 从它开始，我们主要需要一个start（）方法，在该方法中将执行主要操作（我们将跳过此方法中发生的错误；换句话说，如果出现问题，我们的窗口将不会关闭，但是StackTrace错误将出现在控制台中） ： 
 package company; import javafx.application.Application; import javafx.stage.Stage; public class Company extends Application { public void start(Stage primaryStage) throws Exception { } public static void main(String[] args) { launch(args); } }

太好了 为了与公司员工的代表一起工作，请创建User类：此类的对象，并将充当员工。

用户类别

建立


到目前为止，当应用程序启动时，对我们来说什么都没有发生。 要至少显示一个窗口，必须添加它。 我们会做到的。 

 package company; import javafx.application.Application; import javafx.scene.Group; import javafx.scene.Scene; import javafx.stage.Stage; public class Company extends Application { public void start(Stage primaryStage) throws Exception { Scene scene = new Scene(root, WIDTH, HEIGHT); primaryStage.setTitle("Company"); primaryStage.setScene(scene); primaryStage.show(); } public static void main(String[] args) { launch(args); } Group root = new Group(); final private int WIDTH = 1000; final private int HEIGHT = 600; }

我们刚刚做了什么：我们添加了一个大小场景 宽度\乘以高度 ，给她起了“公司”的名字。 我们场景中有一个根组，它实际上是一个包含所有按钮的“容器”。

与我们的员工一起添加一个下拉列表。 让我们创建一个HBox，在其中放置组合框userBox下拉列表和一个将显示有关员工信息的按钮（将HBox放置在VBox中）：

什么是VBox，HBox和ComboBox

HBox是一组按钮，文本，输入字段，其每个对象均按水平顺序排列。 与此类似，VBox相同，但是垂直存储其对象（子代）。 我们将使用以下方案：创建一个VBox，在其中放置几个​​HBox'ov，在每个HBox'ov中放置一个按钮和字段序列。

ComboBox本身就是一个下拉列表。

现在，我们要创建一个按钮，该按钮将显示有关所选雇员的信息。 添加按钮本身（Button）和文本字段（TextField），将在其中显示文本。 我们将以下两个对象添加到HBox中：

但是到目前为止，此按钮没有任何作用。 为了让她做某事，必须为她分配一个操作： 

 buttonGetInfo.setOnAction(new EventHandler<ActionEvent>() { @Override public void handle(ActionEvent e) { User u = (User) userBox.getSelectionModel().getSelectedItem(); if (u != null) { textInfo.setText("Age is " + u.getAge() + ", " + "Salary is " + u.getSalary() + ", " + "Relationship: " + u.getMarried()); } else { textInfo.setText("User not selected"); } } });

那是什么


该程序本身没有任何意义， 仅是为了熟悉JavaFX及其工具而编写的。 例如，在某些地方，当更改薪水时，如果未选择用户或输入行，则不会考虑选项：在这些情况下，将出现错误。 这同样适用于添加新员工时在其中输入信息的字段。 最后，如果未在任何下拉列表中选择某个值，就会发生错误。 如果需要，每个人都可以添加自己的这些弯路。
