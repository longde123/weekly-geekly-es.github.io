GitHub Package Registry将支持Swift软件包

5月10日，我们宣布了GitHub Package Registry的beta版，这是一个软件包管理服务，可让您轻松在源代码旁边发布公共或私有软件包。 它目前支持熟悉的软件包管理工具：JavaScript（npm），Java（Maven），Ruby（RubyGems）、. NET（NuGet）和Docker映像，还有更多功能即将推出。

今天，我们很高兴地宣布，我们将在GitHub Package Registry中添加对Swift软件包的支持。 使用Swift软件包，可以轻松地在整个项目中以及与Swift社区共享您的库和源代码。



Swift Package Manager在GitHub上可用，是一个用于构建，运行，测试和打包Swift代码的跨平台工具。 软件包配置是用Swift编写的，可轻松配置目标，声明产品和管理软件包依赖性。 结合使用Swift Package Manager和GitHub Package Registry，您可以更轻松地发布和管理Swift软件包。

对于移动开发人员来说，拥有最好的工具以提高生产力至关重要。 随着Swift生态系统的发展，我们很高兴与App​​le团队一起为Swift开发人员创建新的工作流。

自发布以来，我们惊讶地发现您对开始使用GitHub Package Registry感到兴奋。 在这个测试版期间，我们致力于从社区和生态系统中学习如何满足您的需求以及如何使它变得更好。 如果您尚未注册，请立即注册 。
