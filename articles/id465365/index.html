Skrip pengaturan Windows 10

Saya sudah lama ingin berbagi skrip saya untuk mengotomatisasi konfigurasi Windows 10 (saat ini versi saat ini adalah 18363 ), tetapi semua tangan tidak mencapai. Mungkin itu akan bermanfaat bagi seseorang secara keseluruhan atau sebagian saja. Tentu saja, itu akan bermasalah untuk menggambarkan semua pengaturan, tetapi saya akan mencoba untuk menyoroti yang paling penting.

Jika ada yang tertarik, selamat datang di kucing.

Fungsi utama

  • Nonaktifkan layanan pelacakan diagnostik
  • Banyak Explorer yang diubah
  • Konsol Interaktif
  • Hapus semua aplikasi UWP dari semua akun kecuali yang berikut ...
  • Nonaktifkan komponen
  • Buat tugas di Penjadwal Tugas Pembaruan Pembaruan Windows
  • Buat tugas di Penjadwal Tugas untuk membersihkan folder "$ env: SystemRoot \ SoftwareDistribution \ Unduh"
  • Tambahkan Folder ke Daftar Pengecualian Pemindaian Defender Windows
  • Nonaktifkan layanan pengguna
  • Buat pintasan format lama untuk "Perangkat dan Pencetak"
  • Impor tata letak menu Mulai yang disesuaikan dari file reg yang disiapkan
  • Lepas sematan semua pintasan dari layar beranda
  • Tetapkan ulang lokasi folder "Desktop", "Dokumen", "Unduhan", "Musik", "Gambar", "Video"
  • Segarkan ikon desktop, variabel lingkungan, dan bilah tugas tanpa memulai ulang Explorer
  • Jauh lebih banyak

NB

  • PowerShell dan PowerShell ISE harus dijalankan dengan hak yang lebih tinggi
  • Tetapkan kebijakan eksekusi skrip PowerShell yang sesuai 
Set-ExecutionPolicy -ExecutionPolicy Bypass -Force
