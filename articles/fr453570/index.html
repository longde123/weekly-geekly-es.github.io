Microsoft Edge pour macOS

Le mois dernier, nous avons annoncé les premières pré-versions de la prochaine version de Microsoft Edge pour Windows 10. Nous sommes maintenant heureux d'annoncer que le canal Microsoft Edge Canary est également devenu disponible pour macOS. Maintenant, sur le site Web de Microsoft Edge Insider, vous pouvez trouver des versions pour les ordinateurs avec macOS et Windows 10, et la prise en charge de la version Windows sera bientôt étendue.  



Microsoft Edge pour Mac


Le navigateur Microsoft Edge pour macOS offrira aux utilisateurs les mêmes nouvelles fonctionnalités que dans les versions préliminaires pour Windows, mais son interface sera adaptée pour la plate-forme Mac. Nous modifions l'interface et les fonctionnalités en fonction de ce que les utilisateurs de macOS sont habitués et attendent.



Nous nous efforçons de faire de Microsoft Edge un navigateur de classe mondiale avec des fonctionnalités uniques et des services connexes. Dans cette première version, nous avons apporté quelques modifications à l'interface utilisateur pour l'aligner sur la conception de Microsoft, mais en même temps pour la rendre naturelle pour macOS.


Les modifications concernent les polices macOS, les menus, les raccourcis clavier, les en-têtes et d'autres éléments. Dans les versions futures, nous continuerons de modifier l'apparence et les fonctionnalités du navigateur, nous expérimenterons et répondrons aux commentaires des utilisateurs. Par conséquent, parlez-nous de vos impressions à l'aide de l'émoticône Envoyer des commentaires.



De plus, nous développons des moyens d'interagir avec l'utilisateur, exclusifs à macOS, basés sur les fonctionnalités matérielles spéciales de la plate-forme Mac. Ainsi, les utilisateurs peuvent effectuer des actions et accéder aux commandes de contexte à l'aide de l'écran tactile (Touch Bar). Par exemple, vous pouvez accéder aux raccourcis du site Web, basculer entre les onglets et gérer les vidéos, et utiliser une navigation familière à l'aide des gestes du pavé tactile.



Présentation des flux Microsoft Edge Insider pour macOS


Les nouvelles pré-versions de Microsoft Edge pour macOS sont disponibles via les canaux de pré-version, que nous appelons les canaux Microsoft Edge Insider. Nous commençons par lancer le canal de construction Canary, que vous pouvez obtenir sur le site Web de Microsoft Edge Insider . Ce canal est déjà disponible pour macOS 10.12 et supérieur. Bientôt, nous ouvrirons le canal Dev, et lorsqu'il sera disponible, vous pourrez télécharger et installer cet assemblage en parallèle avec Canary. Vous pouvez en savoir plus sur notre approche et à quoi s'attendre de différents canaux dans nos publications au cours du mois dernier .



Plateforme et outils cohérents pour les développeurs Web


Étant donné que nos nouvelles versions sont basées sur Chromium, vous obtenez le même rendu dans les deux versions de Microsoft Edge - pour Windows et macOS. De plus, vous bénéficiez du même accès à de puissants outils de développement.


Pour la première fois, les développeurs Web peuvent tester des sites et des applications Web dans Microsoft Edge sur macOS et s'assurer qu'ils fonctionnent de la même manière dans la prochaine version de Microsoft Edge sur toutes les plateformes. (Notez que les fonctionnalités spécifiques à la plate-forme, telles que le décryptage du contenu PlayReady dans Windows 10, doivent toujours être surveillées séparément pour une meilleure expérience avec elles sur ces plates-formes.)


Comme les pré-versions pour Windows, notre nouvelle version pour macOS prend également en charge les applications Web progressives, qui sont installées sur les normes et peuvent être vérifiées et déboguées à l'aide des outils de développement dans le navigateur. Nous nous efforçons de veiller à ce que ces applications puissent être utilisées confortablement avec d'autres applications de plate-forme.Ainsi, après l'installation, elles seront affichées dans le Dock, le sélecteur d'applications et Spotlight comme toute autre application.  


Vos commentaires sont importants pour nous


Nous sommes heureux de partager avec vous notre première version de Microsoft Edge Canary pour macOS! Nous avons besoin de vos commentaires pour améliorer le navigateur, car nous pensons que le navigateur doit être aussi pratique que possible pour les utilisateurs de macOS. Nous espérons que vous essayez l'aperçu aujourd'hui et nous nous réjouissons de vos commentaires et de votre participation à la communauté Microsoft Edge Insider.


Si vous avez des problèmes ou souhaitez laisser des commentaires ou suggérer quelque chose, vous pouvez laisser un message sur les forums de la communauté Microsoft Edge Insider , nous contacter sur Twitter ou simplement utiliser la commande "Envoyer des commentaires" dans le menu Microsoft Edge, pour partager vos pensées avec nous.


Si vous êtes un développeur Web et que vous avez rencontré un problème qui se reproduit dans Chromium, il est préférable de signaler un bogue dans la communauté Chromium . Pour signaler des problèmes dans la version actuelle de Microsoft Edge, veuillez continuer à utiliser EdgeHTML Issue Tracker .
