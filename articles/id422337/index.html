Yandex meluncurkan cloud

Hari ini, Yandex telah membuka layanan Yandex.Cloud baru.



Daya komputasi Yandex dan layanan untuk pemrosesan dan penyimpanan data. Yandex.Cloud adalah platform cloud publik tempat Anda dapat membuat dan mengembangkan proyek Anda.

Proyek ini terdiri dari 7 layanan berbeda:

  1. Yandex Compute Cloud - Mesin dan Drive Virtual
  2. Penyimpanan Objek Yandex - Gudang Data Scalable
  3. Yandex Virtual Private Cloud - Jaringan untuk pertukaran informasi antara sumber daya cloud dan Internet
  4. Identitas Yandex dan Manajemen Akses - Identifikasi dan kontrol akses ke sumber daya cloud
  5. Database yang Dikelola Yandex - ClickHouse, PostgreSQL, dan Manajemen Basis Data MongoDB
  6. Yandex SpeechKit - Seperangkat teknologi pengenalan suara dan sintesis
  7. Yandex Translate - Terjemahan mesin dengan dukungan lebih dari 90 bahasa

Saat ini, Anda dapat menggunakan layanan hanya setelah mengisi formulir dan menyetujui aplikasi dari Yandex, namun, perusahaan berjanji bahwa pada akhir 2018 setiap orang akan memiliki akses gratis ke layanan.

Sebagai pemilik proyek VPS dan VPS.today , server virtual paling menarik bagi saya . Fitur pertama yang saya perhatikan dengan Yandex adalah Anda tidak perlu membayar untuk mesin virtual dalam keadaan off.

Biaya dihitung selama penggunaan mesin virtual, mulai dari saat peluncuran (transisi ke status MENJALANKAN) dan berhenti total. Waktu mesin virtual dimatikan tidak dikenakan biaya.

Misalnya, dengan DigitalOcean atau Vultr, dana didebet dalam hal apa pun, bahkan jika server dimatikan - sumber daya tetap disediakan untuk itu, jadi cukup logis untuk mengambil uang untuk itu.

Di pintu masuk pertama, Yandex menyediakan untuk menguji layanan 4.000 rubel, yang akan tersedia dalam waktu enam bulan.

Untuk core dan RAM, harga diindikasikan untuk penggunaan 1 jam, untuk disk harga diindikasikan untuk 1 bulan. Tagihan per detik.

Harga adalah sebagai berikut:
Sumberdaya Harga untuk 1 jam Harga 30 hari
1 inti (vCPU) 0,69 RUB 496,80 gosok.
RAM 1 GB 0,24 gosok 172.80 RUB
NVID 1 GB 7.32 RUB
Drive SATA 1 GB 2,05 gosok
Server Windows 1,12 RUB 806,40 gosok.
Alamat IP Eksternal 0,15 gosok 108 r
Saat menggunakan layanan ini, hanya lalu lintas keluar yang dibayar dengan harga 1,5 rubel. per GB, lalu lintas masuk dan lalu lintas antar awan benar-benar gratis .

Menurut informasi dari media, layanan saat ini hanya berfungsi di Rusia, tetapi di masa depan Yandex berencana untuk memasuki pasar internasional dengan bersaing dengan Microsoft Azure dan Google Cloud Platform.


Lihatlah VPS.today , situs untuk mencari server virtual. 1500 tarif dari 130 host, antarmuka yang nyaman dan sejumlah besar kriteria untuk menemukan server virtual terbaik.
