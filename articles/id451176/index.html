Kartu

  • [1] Stefan Keller dari Universitas Sains Terapan Rapperswil menerbitkan Peta Kastil Swiss. Ini saat ini merupakan peta web istana dan istana paling komprehensif di Swiss, termasuk negara-negara tetangga (Jerman, Prancis, Italia) - hingga Kastil Neuschwanstein. Aplikasi web multibahasa menggunakan data dari OpenStreetMap, Wikidata, Wikimedia Commons (gambar) dan Wikipedia. Kode sumber proyek tersedia di GitLab.
  • Layanan historical.place telah diperbarui dan sekarang menampilkan peta kastil baru di Swiss berdasarkan data kastil Wikimedia .
  • Aplikasi resmi maraton London secara mengejutkan menggunakan ubin dari server OpenStreetMap, yang tidak dapat mengatasi beban (mungkin karena konfigurasi CDN mereka yang salah). Lihat juga kebijakan untuk menggunakan petak dan switch2osm.org sebagai tutorial tentang pengaturan infrastruktur Anda sendiri.
