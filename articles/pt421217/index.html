Suporte de Python no Power BI

Temos o prazer de anunciar que o Python, uma linguagem de programação amplamente usada por estatísticos, cientistas e analistas, agora está integrada ao nosso Power BI Desktop gratuito. Depois de ativar a função nas configurações, você pode usar o Python para limpar, analisar e visualizar os dados. Mais detalhes sob o corte!



O primeiro recurso de integração que adicionamos é a capacidade de executar scripts Python diretamente no Power BI Desktop para criar um modelo de dados. Você pode fazer isso na caixa de diálogo Obter dados.



Depois de selecionar esta opção, você pode inserir seu script.



A integração com o editor de consultas permite limpar dados usando o Python e usar métodos avançados para analisar seus dados, incluindo, por exemplo, excluir missões, previsões e agrupamentos.

Por fim, você pode usar o Python para criar novos efeitos visuais no seu relatório. Como a visualização no R, os efeitos visuais do Python serão atualizados com atualização de dados e filtragem cruzada. No entanto, o visual em si ainda não é interativo.
