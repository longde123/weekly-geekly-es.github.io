Snom D335 - Telepon IP Multifungsi dengan Layar Warna TFT Widescreen



Hari ini kami ingin memperkenalkan Anda ke telepon desktop IP layar lebar Snom D335 yang baru. Mengingat fungsinya yang hebat, perangkat bisa tanpa berlebihan disebut asisten kantor nyata. Namun, putuskan sendiri.

Desain


Desainer mencoba membuat ponsel terlihat tidak hanya modern dan mengesankan, tetapi juga senyaman mungkin. Saya harus mengatakan, mereka memperhitungkan setiap hal kecil, dan membuatnya mudah dan nyaman untuk menggunakan Snom D335.



Soliditas produk baru menambah warna hitam yang ketat, yang, seperti Anda ketahui, langsing. Panel depan kasing ini ditekuk dengan lancar, seperti tipikal untuk saluran D3xx.



Untuk menavigasi menu, kami menggunakan joystick melingkar yang sudah dikenal. Untuk akses cepat ke fungsi paling populer di panel depan ada tombol khusus: di sebelah kanan dan kiri joystick - "Jangan Ganggu" dan Pesan, dan kolom di sisi kanan panel - Buku Alamat, Menu, Transfer Panggilan dan Bungkam.



Tombol-tombol tersebut dialokasikan ke grup terpisah: "kursi goyang" untuk kontrol volume, speakerphone, dan headset.



Handset itu sendiri patut mendapat perhatian khusus. Dia memiliki bentuk "anatomi" fungsional. Itu terletak dengan sempurna di tangan, dan bahkan jika Anda sedikit membuka tangan Anda, handset tidak akan keluar.

Dan, tentu saja, sorotan utama dari Snom D335 adalah layar TFT warna cerah dengan resolusi 320 x 240 piksel. Di bawah layar adalah sensor jarak built-in. Saat Anda membawa tangan ke telepon, lampu latar menyala, dan tanda tangan fungsi tombol yang dapat diprogram "buka" ke ukuran penuh.



Di sisi tampilan adalah tombol fungsi yang dapat diprogram. Perhatikan bahwa ada delapan perangkat fisik secara total, tetapi secara virtual Anda dapat membuat dan menggunakan 32 tombol fungsi, ditambah empat tombol layar dengan fungsi peka konteks.

Untuk kenyamanan, tampilan dapat dinaikkan dan dikunci dalam posisi yang memberikan tampilan paling nyaman.








Kualitas suara tinggi


Mengingat bahwa tujuan utama telepon apa pun adalah untuk memastikan kualitas transmisi dan penerimaan sinyal audio terbaik, pengembang mengintegrasikan prosesor sinyal digital (DSP) ke Snom D335. Ini memberikan dukungan untuk codec seperti G.711 A-law, μ-law, G.722 (wideband), G.726, G.729AB, GSM 6.10 (FR). DSP mampu menangani berbagai gangguan dan distorsi sinyal secara efektif, menekan apa yang disebut efek lokal.

Untuk memastikan bahwa selama jeda dalam percakapan tidak ada perasaan koneksi terputus dengan lawan bicaranya, telepon memiliki penghasil kebisingan kenyamanan bawaan (CNG). Snom D335 juga mengintegrasikan teknologi Deteksi Suara Manusia (VAD), yang secara otomatis menyalakan mikrofon ketika Anda mulai berbicara.

Untuk kemudahan komunikasi, headset USB dapat dihubungkan ke perangkat (tidak termasuk dalam paket pengiriman).

Peluang komunikasi




Untuk terhubung ke jaringan komputer, Snom D335 dilengkapi dengan switch Ethernet dengan dua port RJ45 gigabit. Tetapi, jika kantor Anda tidak memiliki jaringan kabel, itu tidak masalah. Anda dapat menghubungkan adaptor Wi-Fi Snom A210 eksternal ke ponsel melalui port USB. D335 juga mendukung protokol IPv4 dan IPv6.



Ponsel ini mendukung hingga 12 SIP-akun, dan juga membuat jurnal di mana panggilan masuk, keluar dan tidak terjawab direkam. Untuk kenyamanan, ada buku alamat untuk 1000 entri. Secara signifikan memperluas kemampuan untuk mengkonfigurasi ponsel XML-browser bawaan.

Snom D335 akan bekerja tanpa masalah dengan platform IP-PBX paling populer. Untuk bekerja dengan jaringan telepon standar, perangkat mendukung semua standar saat ini, seperti DTMF In-band, DTMF out-of-band dan SIP INFO DTMF.

Fitur-fitur canggih


Fungsionalitas D335 sepenuhnya memenuhi persyaratan bisnis modern.
Untuk menghemat waktu, yang, seperti Anda ketahui, lebih mahal daripada uang, ponsel ini memiliki fungsi untuk panggilan cepat dan dukungan untuk panggilan dengan URL. Jika pihak yang dipanggil sibuk, Anda dapat menggunakan fitur redial untuk melakukan panggilan secara otomatis.

ID penelepon bawaan membantu Anda dengan cepat memahami siapa yang menelepon Anda, dan pelanggan yang tidak diinginkan dapat masuk daftar hitam dan mereka tidak akan lagi mengganggu Anda dengan panggilan mereka. "Fitur" lain dari item baru adalah nada dering hasil personalisasi.

Jika Anda sibuk, silakan nyalakan mode Do Not Disturb, dan tidak ada yang akan mengalihkan perhatian Anda dari hal-hal penting. Konferensi tiga arah akan membantu mendiskusikan masalah bisnis saat ini dengan kolega atau bawahan. Dan menggunakan mode bebas-genggam (Hands-free talk (Speakerphone)), Anda dapat berbicara di telepon dengan rekan kerja, bahkan menjauh dari meja Anda di dalam ruangan.

Snom D335 dikonfigurasi menggunakan server web yang dilindungi kata sandi bawaan. Dan untuk konfigurasi cepat sekelompok telepon di kantor, Anda dapat mengunduh pengaturan menggunakan protokol HTTP, HTTPS, dan TFTP. Berkat ini, Anda dapat dengan cepat menyebarkan komunikasi telepon bahkan di kantor besar.


Jika perlu, Anda dapat mengkonfigurasi ulang sepenuhnya fungsi tombol perangkat keras apa pun melalui komputer, bukan hanya yang dapat diprogram:





Menu ponsel ini multibahasa, dan tentu saja, dengan dukungan untuk bahasa Rusia.

Untuk kenyamanan para pekerja tuna rungu, Snom D335 kompatibel dengan alat bantu dengar (HAC).

Ponsel ini ditenagai menggunakan teknologi PoE melalui port RJ45, tetapi, jika perlu, Anda dapat menggunakan sumber daya eksternal dengan tegangan output 5 V.
