Nouveau terminal Windows désormais disponible sur le Microsoft Store

Le nouveau terminal Windows, que Microsoft a annoncé lors de MS Build 2019 , est déjà disponible en téléchargement dans la boutique, selon un blog officiel. Pour les intéressés, le référentiel du projet sur GitHub .


Terminal est une nouvelle application Windows pour un accès centralisé aux sous-systèmes de noyau PowerShell, Cmd et Linux dans le package Linux du sous-système Windows. Ce dernier est devenu disponible pour la construction de la version 18917 de Windows Insider le 20 juin.

Pour utiliser le nouveau terminal, deux conditions doivent être remplies: installez Windows 10 version 18362.0 ou supérieure et recherchez le bouton Microsoft Store. Bien sûr, vous pouvez toujours compiler Terminal à partir des tris publiés sur GitHub, mais les développeurs avertissent que dans ce cas, la «version assemblée manuellement fonctionnera en parallèle avec la version du magasin». Apparemment, il est entendu que le magasin ne récupérera pas de terminal assemblé manuellement et ne sera pas mis à jour seul.

L'une des principales caractéristiques de Terminal, qui est activement «vendue» sur le blog de l'entreprise, est un nombre considérable de profils.



Chacun des profils peut être configuré séparément en modifiant le fichier JSON correspondant.



Microsoft propose également à chaque utilisateur de choisir les raccourcis clavier et les combinaisons à utiliser et de les personnaliser à son goût.

La cerise de personnalisation était la possibilité de changer l'image d'arrière-plan de chaque profil grâce au banal pull-up de l'image depuis le disque dur. Il n'y a donc pas de limite à l'imagination.

Maintenant, soyons un peu plus sérieux


Pourquoi n'y a-t-il aucun détail technique dans le billet de blog Microsoft? Pourquoi l'accent est-il mis sur la personnalisation, les raccourcis clavier et autres cosmétiques?

Tout d'abord, tout le monde a déjà parlé de Terminal à Build 2019, et il n'y a rien de spécial à ajouter. Maintenant, la société essaie de montrer que la nouvelle application est un produit convivial et pratique qui va de pair avec le nouveau WSL. En fait, Microsoft vient de déployer ce qu'il nous a promis en mai et il n'y a rien de spécial à ajouter.

La seconde - un certain temps s'écoulera avant la sortie de la version 1.0. À en juger par le texte du blog Microsoft, Terminal ne quittera pas la phase de sciage active avant l'hiver, c'est-à-dire qu'il n'apparaîtra sur les versions stables de Windows dans le magasin que six mois plus tard.

Dans le même temps, les représentants de l'entreprise agitent activement la communauté pour fournir des commentaires sur un nouveau produit. Ainsi, Microsoft sera très reconnaissant pour les commentaires et suggestions sur Terminal dans son propre référentiel sur Github et, disons, la communauté a répondu à ce cri. Nous pensons qu'au cours de la semaine prochaine, il y aura beaucoup de commentaires sur les «problèmes».
