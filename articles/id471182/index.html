GitHub Desktop 2.2: Apa yang Baru



Siapa pun yang menggunakan Git tahu bahwa ia memiliki jalur pembelajaran yang keren. Kami belajar dari pengembang bahwa kebanyakan orang cenderung belajar dari kenalan, apakah mereka seorang kolega, profesor, teman, atau bahkan video YouTube. Di GitHub Desktop 2.2, kami menambahkan versi pertama tutorial interaktif tentang Git dan GitHub, yang dapat menjadi teman Anda dan membantu Anda memulai. Jika Anda baru menggunakan Desktop, Anda dapat mengunduh dan mencoba tutorialnya di desktop.github.com . Nah, detailnya ada di bawah potongan.

Memulai


Untuk memulai, kami akan membantu Anda melewati dua tahap utama: membuat repositori dan menghubungkan editor. Saat Anda pertama kali membuka program, halaman pembuka akan muncul dengan opsi baru "Buat repositori pelatihan." Menggunakan fitur ini, repositori pelatihan dibuat yang memandu Anda melalui konsep dasar bekerja dengan Git menggunakan GitHub Desktop.



Banyak alat yang dibutuhkan untuk memulai dengan Git dan GitHub. Yang paling penting dari ini adalah editor kode Anda. Pada tahap pertama tutorial, Anda akan diminta untuk menginstal editor jika Anda belum memilikinya.



Jelajahi GitHub


Selanjutnya, kami akan memberi tahu Anda cara menggunakan GitHub Desktop untuk mengubah kode secara lokal dan mengirimkan pekerjaan Anda ke GitHub. Anda akan membuat cabang baru, membuat perubahan pada file, komit, mengirimkannya ke GitHub dan membuka permintaan tarik pertama Anda.

Kami juga mendengar bahwa pengguna baru awalnya mengalami kebingungan antara Git, GitHub, dan GitHub Desktop. Kami akan melihat perbedaan-perbedaan ini dalam manual dan memastikan bahwa mereka mendukung penjelasan yang biasa.



Lanjutkan dengan proyek Anda sendiri.


Di GitHub Desktop 1.6, kami memperkenalkan fitur "kemungkinan langkah selanjutnya" tergantung pada kondisi repositori Anda. Sekarang setelah Anda menyelesaikan tutorial, kami juga menawarkan langkah-langkah berikut: menjelajahi proyek GitHub yang mungkin ingin Anda berkontribusi, membuat proyek baru, atau menambahkan proyek yang ada ke Desktop. Kami ingin GitHub Desktop menjadi alat yang akan menjelaskan langkah Anda selanjutnya, terlepas dari apakah Anda sedang dalam proses kerja atau Anda hanya pengembang pemula.



Apa selanjutnya


Bekerja di GitHub Desktop 2.2 kami membuat produk lebih mudah diakses oleh pemula. Kami akan meningkatkan tutorial berdasarkan umpan balik Anda dan akan terus meningkatkan koneksi antara GitHub dan komputer lokal Anda. Jika Anda ingin mulai membuat sesuatu, tetapi tidak tahu caranya, anggap GitHub Desktop sebagai teman yang akan membantu Anda memulai.
