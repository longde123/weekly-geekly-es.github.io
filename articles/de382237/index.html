Computer hacken: normaler Telefon- und EM-Strahlungsdatenbus


Demo-Shot

Israelische Sicherheitsexperten der Ben-Gurion-Universität des Negev haben erfolgreich Informationen von einem Computer auf ein normales Mobiltelefon übertragen, wobei nur elektromagnetische Strahlung verwendet wurde, die der Computer im normalen Betriebsmodus aussendet.

So haben sie ihr Ergebnis gegenüber dem Vorjahr verbessert, als ein Smartphone zum Empfang der übertragenen Daten erforderlich war. Diese Studie zeigt die Notwendigkeit einer ernsthaften Verschärfung der Arbeitsregeln in geheimen Unternehmen. Selbst Computer, die vollständig vom Internet getrennt sind, können verwendet werden, um vertrauliche und vertrauliche Daten diskret daraus zu extrahieren.

Im Jahr 2014 haben Forscher die Datenübertragung erfolgreich durchgeführtzwischen Computer und Smartphone. Für eine erfolgreiche Übertragung musste eine spezielle Software sowohl auf einem Computer als auch auf einem Smartphone installiert werden.

Eine Grafikkarte mit einem daran angeschlossenen Monitor fungierte als Generator für Funksignale und als Sender. Das modulierte Signal wurde vom im Smartphone eingebauten FM-Radioempfänger aufgenommen und von der mobilen Anwendung entschlüsselt. Den Forschern gelang es, eine Übertragungsgeschwindigkeit von 60 Bytes / Sek. Zu erreichen.

FM-Empfänger werden in der Regel nicht nur als zusätzliche Unterhaltung in Smartphones eingebaut, sondern auch als Möglichkeit, Benachrichtigungen über Naturkatastrophen zu erhalten, wenn keine GSM-Verbindung besteht.

Es ist jedoch auf jeden Fall verboten, Smartphones an besonders geheime Unternehmen zu bringen - sie verfügen weiterhin über integrierte Foto- (und Video-) Kameras, WLAN, Bluetooth und andere Mittel zur Datenübertragung, die die Sicherheit des Unternehmens beeinträchtigen könnten. Aber auf einem normalen Telefon ohne all diese neuen Dinge - zum Beispiel wie das Motorola C123, das die Forscher in ihrer Arbeit verwendet haben - kann der Verdacht kaum fallen. Es kann jedoch auch als Spionagemittel eingesetzt werden.

Diesmal verwendeten die Forscher einen Computerdatenbus, um ein moduliertes Funksignal zu senden. Computer mit Mehrkanal- Speichergerät - Wenn Sie auf beiden Kanälen gleichzeitig dieselben Daten übertragen, wird die Signalleistung verstärkt.



Dank einer Änderung der Firmware gelang es ihnen, ein normales Telefon anstelle eines Smartphones zu verwenden. Mit der erweiterten Firmware können Sie digitale Daten auf GSM-Frequenzen empfangen. Wie von den Ingenieuren geplant, wird die übliche Hintergrundstrahlung des Computers während des Betriebs als 0 angenommen und durch Übertragen spezieller Daten auf dem Bus als 1 verstärkt. Die Übertragung wird in 12-Bit-Blöcke unterteilt. Das Telefon wartet auf den Start des Blocks (Header 1010), stellt sich auf den aktuellen Signalpegel ein und empfängt dann den Rest des Blocks.

Die Forscher testeten ihr System erfolgreich in einem Raum mit mehreren funktionierenden Computern, simulierten eine Standardarbeitsumgebung und erstellten ähnliche Software für verschiedene Betriebssysteme - sowohl Windows als auch Linux.

Mit dem Telefon als Empfänger konnten Geschwindigkeiten von bis zu 2 Bit pro Sekunde erreicht werden. Die Zeichnungen des U-Bootes können nicht auf diese Weise übertragen werden, aber Passwörter, GPS-Koordinaten oder sogar ein 256-Bit-RSA-Schlüssel sind durchaus möglich. Wenn Sie einen fortschrittlicheren Empfänger verwenden, können Sie außerdem die Übertragungsgeschwindigkeit auf 1000 Bit pro Sekunde und die Empfangsreichweite erhöhen - bis zu 30 Meter, was von den Experimentatoren durchgeführt wurde. Tatsächlich ist es möglich, ein Signal von irgendwo auf dem Parkplatz neben dem Gebäude zu empfangen, in dem der mit der Malware infizierte Computer funktioniert.
