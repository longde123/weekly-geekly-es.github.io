La fonction de test de Firefox charge la page à partir d'Internet Archive au lieu d'une erreur 404



En mai de ce code, Mozilla a mis à jour son service Test Pilot Program. Ce service, selon les développeurs du navigateur Firefox, permet de «lancer» de nouvelles fonctionnalités du navigateur. Tout utilisateur qui souhaite essayer ces fonctions peut le faire en rejoignant l'équipe de «pionniers» sur la page correspondante . Vous pouvez activer ou désactiver n'importe quelle fonction de test à tout moment.

Des représentants de la société préviennent que les nouveautés présentées sur la page Test Pilot peuvent ne pas fonctionner parfaitement. Mais cela nécessite également un «rodage» - la société doit connaître le problème à l'avance afin de l'éliminer dans la version finale. Maintenant, sur la page Test Pilot, il y a des informations sur Firefox No More 404s. Si vous activez cette fonction, le navigateur n'affichera pas d'erreur 404 sur le site de la page manquante sur le serveur, mais la version de cette page sur Internet Archive. Cela ne fonctionnera que si la page a déjà existé et si une copie a été enregistrée par les araignées Wayback Machine.

Le fonctionnement de Firefox No More 404s est simple. Le navigateur avec la fonction activée analyse les codes de réponse pour l'erreur «404 introuvable». Et si la situation le permet, Firefox affichera la page stockée dans l'archive, que le serveur sur lequel le site est situé pour une raison quelconque ne peut pas l'afficher. L'erreur "404 not found" apparaît en partie car la page vient d'être supprimée. En même temps, tout le site peut être supprimé.

Si une erreur différente est définie au lieu de «404 introuvable» ou si le site redirige l'utilisateur au lieu d'une page manquante ou inexistante vers un autre endroit, la nouvelle fonction ne fonctionnera pas. Si No More 404s détermine que la page ou le site manquant se trouve dans l'archive, l'utilisateur verra le message suivant dans la fenêtre du navigateur: «Cette page semble inaccessible. Afficher la copie enregistrée dans Wayback Machine. " Ce message est également un lien vers la ressource ou la page souhaitée.

Maintenant, la fonction ne fonctionne pas toujours correctement. Dans certains cas, malgré le message vous demandant d'afficher la page d'archives, il s'avère qu'il n'y a rien dans la Wayback Machine. Soit dit en passant, il existe un excellent analogue des No More 404 sous la forme de l'extension Resurrect Pages. Cette extension prend en charge les «archives Internet», fonctionne avec le cache des moteurs de recherche et d'autres ressources d'archives.

Pour commencer avec Firefox No More 404s, vous devez installer l'extension Test Pilot. L'installation vous permettra de travailler avec d'autres nouveaux produits. C'est Activity Stream, Teb Center, Universal Search.



Le flux d'activité rend la recherche dans le journal des pages visitées plus efficace, ce qui permet aux utilisateurs de gagner du temps. La fonction permet de retrouver le site visité, il a paru important à l'utilisateur, mais l'adresse de la ressource a été oubliée. Lorsque vous ouvrez un nouvel onglet, l'utilisateur affiche une liste des sites les plus fréquemment visités et des conseils sur le journal des pages visitées.

Centre de tabulationaffiche les onglets verticalement plutôt qu'horizontalement. De nombreux utilisateurs aiment ce mode d'affichage d'onglet plus que traditionnel.

La recherche universelle combine l'historique de Awesome Bar avec le menu déroulant de recherche de Firefox. Ceci est fait pour que l'utilisateur puisse trouver rapidement les informations dont il a besoin.
