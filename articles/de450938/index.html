Linux Install Fest 05.19 in Nischni Nowgorod 18. Mai 2019

Das Linux Install Fest 05.19 findet am 18. Mai 2019 in Nischni Nowgorod statt.

Die Veranstaltung wird von NNLUG am Nizhny Novgorod College of Radio Engineering durchgeführt.

Mit dem verfügbaren Internet ist es heute nicht schwierig, eine bestimmte Linux-Distribution herunterzuladen oder die Antwort auf die meisten Fragen zu finden, die während der Arbeit unter diesem Betriebssystem auftreten.

Das Format öffentlicher Treffen bleibt jedoch bei Open-Source-Unterstützern beliebt.

Auf dem Linux Install Fest sind eine Reihe von Berichten auf verschiedenen Komplexitätsebenen und zu verschiedenen Themen geplant: ein ständiges Interesse an Blender, Spielelösungen und täglichen Arbeitsfällen.
Details finden Sie auf der Veranstaltungsseite .

Diejenigen, die einen Bericht erstellen möchten, werden zur Teilnahme eingeladen (das Hinzufügen von Berichtsthemen ist unter nnlug.ru verfügbar).

Zeit 10:00 - 15:00 Uhr
Ort: N. Novgorod, st. Student, 6 (NRTK), Aula.
Fragen und Vorschläge können unter t.me/nnlug gestellt werden
