48 sumber daya open source untuk JavaScript (2019)



Dari 20.000 proyek dan perpustakaan JavaScript, kami memilih 48 yang terbaik (berdasarkan bintang di GitHub) dan dikelompokkan ke dalam 5 kategori:

  • Antarmuka Pengguna (1 ~ 9)
  • Alat JavaScript (10 ~ 25)
  • Pembelajaran Mesin (26 ~ 33)
  • Proyek (34 ~ 39)
  • Alat Pengembang (40 ~ 48)


Artikel ini disiapkan dengan dukungan Perangkat Lunak EDISON, yang mengembangkan sistem pemeriksaan medis elektronik dan juga mendukung perangkat lunak .


Antarmuka pengguna


  1. Kasar v2.0 : Membuat gambar sketsa yang digambar tangan.
  2. Driver.js : Mesin JavaScript vanilla ringan, independen, yang memfokuskan perhatian pengguna pada halaman.
  3. Sweetalert2 : Pengganti pop-up JavaScript yang cantik, responsif, dapat disesuaikan, dan terjangkau (WAI-ARIA).
  4. Tippyjs : Ringan, pustaka tooltip vanilla JS.
  5. Spritejs : Model objek ringan lintas platform untuk rendering 2D.
  6. AutoComplete.js : Pustaka Javascript sederhana untuk pelengkapan otomatis.
  7. Glider.js : Alternatif cepat dan mudah untuk komidi putar gulir.
  8. LayerJS : Perpustakaan Javascript UI / UX yang memungkinkan Anda membuat aplikasi yang menarik secara visual intuitif untuk aplikasi web dan situs web yang mirip dengan aplikasi seluler.
  9. Pencil.js : perpustakaan untuk menggambar 2D interaktif.

    Alat JavaScript

  10. Johnny-Five v1.0 : Robotika JavaScript dan Lingkungan Pemrograman IoT.
  11. Babylon.js : Kerangka kerja JavaScript untuk membuat game 3D dalam HTML5 dan WebGL.
  12. Pollyjs : merekam, memutar, dan mematikan interaksi HTTP.
  13. Omi : Kerangka kerja web JavaScript 4kB generasi berikutnya (komponen web + JSX + proksi + penyimpanan + pembaruan jalur).
  14. Stimulus : Kerangka JavaScript sederhana untuk HTML yang sudah Anda miliki.
  15. Apexcharts.js : Perpustakaan modern diagram JavaScript untuk membuat diagram interaktif dan visualisasi dengan API sederhana.
  16. Greenlet : memindahkan fungsi asinkron ke utasnya sendiri.
  17. JSUI : Alat UI yang ampuh untuk mengelola aplikasi JavaScript.
  18. G6 : kerangka visualisasi grafik.
  19. Workerize : jalankan modul di web-worker'e.
  20. V8n : Perpustakaan validasi yang dapat disesuaikan dan dapat digunakan kembali .
  21. ForgJs : Validator objek JavaScript yang ringan.
  22. Minipack : Contoh sederhana dari bundler modern yang ditulis dalam JavaScript.
  23. Daftar v2 : Daftar yang konsisten dengan kinerja yang tak tertandingi dan API fungsional yang komprehensif.
  24. JavaScript Object Explorer : Temukan metode objek yang diinginkan tanpa mempelajari dokumentasi.
  25. Awaity.js : Alternatif fungsional dan ringan untuk bluebird.js, dibuat dengan async / menunggu dalam pikiran.

    Pembelajaran mesin

  26. Tfjs : Accelerated WebGL, pustaka berbasis browser JavaScript untuk melatih dan menggunakan model ML.
  27. Guess.js : toolkit untuk menggabungkan pengalaman pengguna berbasis data di Web.
  28. Picojs : 200 baris perpustakaan pengenalan wajah.
  29. Tensorspace : infrastruktur untuk visualisasi tiga dimensi dari jaringan saraf, menciptakan model interaktif dan intuitif di browser, dukungan untuk model pembelajaran dalam pra-dilatih dari TensorFlow, Keras, TensorFlow.js
  30. Nlp.js : Pustaka NLP dibangun ke dalam simpul di atas Natural, dengan ekstraksi entitas, analisis suasana hati, identifikasi bahasa otomatis, dll.
  31. Magenta-js : menghasilkan musik dan gambar menggunakan kecerdasan mesin di browser.
  32. JeelizWeboji : Perpustakaan JavaScript / WebGL untuk pengenalan wajah dan deteksi ekspresi real-time.
  33. ASLT : Ajarkan Alexa untuk Menjawab Bahasa Isyarat dengan Tensorflow.js

    Proyek

  34. Tui.calendar v1.0 : kalender.
  35. Webamp : Winamp 2.9 clone dalam HTML5 dan Javascript.
  36. Filepond : perpustakaan unggahan file
  37. OpenSC2K : Pembuatan ulang Sumber Terbuka dari SimCity 2000.
  38. Jsnes : emulator NES.
  39. JSpaint : Cat Klasik yang Dihidupkan Kembali.

    Alat Pengembang

  40. Carbon v3.0 : buat dan bagikan gambar-gambar indah dari kode sumber Anda
  41. Jarvis : bilah alat Webpack berbasis browser yang sangat cerdas
  42. Polacode : Polaroid untuk kode Anda
  43. Htm (Hyperscript Tagged Markup) : Alternatif JSX menggunakan templat tag standar dengan dukungan kompiler.
  44. Rete : Kerangka kerja JavaScript untuk pemrograman visual dan membuat simpul editor
  45. Ukuran-plugin : Lacak ukuran terkompresi sumber daya Webpack dari waktu ke waktu.
  46. Webpackbar : Elegant ProgressBar dan Profiler untuk Webpack 3 dan 4
  47. Worker-plugin : Menambahkan dukungan Worker Web bawaan ke Webpack.
  48. Subliminal : tema VS Code minimalis untuk JavaScript
