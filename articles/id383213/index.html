Konsekuensi dari Satu Sengketa Privasi Forum

Belum lama ini, di satu forum profil, mereka berbicara tentang privasi di Windows 10. Sudah ada standar, secara umum, debat: apakah mengirim data pribadi, apa dan mengapa. Seperti yang sering terjadi dalam hal-hal seperti itu, rompi robek, saling tuduh menghujani, studi para ilmuwan Mabritian diterbitkan, dan sebagainya. Pada titik tertentu, terganggu oleh "ketika mengaktifkan Windows 10 mengirim 30 megabyte data dari camcorder laptop" berikutnya, saya memutuskan untuk melihat apa yang terjadi di Windows 8.1, kemudian menginstal Windows 10 di dekat (lagi) dan mencoba membandingkan perilaku .

Saya harus mengatakan segera bahwa rencana tersebut tidak mencapai pemasangan puluhan. Pertanyaan muncul di delapan.

Bagaimana data dapat dikirim dari windows?

Penjadwal tugas segera muncul di pikiran, di mana Anda dapat (tiba-tiba) merencanakan tindakan apa pun berdasarkan peristiwa atau waktu. Opsi kedua adalah layanan standar tertentu, yang, selain melakukan kegiatan rutin, juga dapat melakukan kontinjensi. Bookmark, cukup cantumkan. Tetapi kualifikasi saya, sayangnya, tidak memungkinkan saya untuk dengan yakin mengidentifikasi keberadaan layanan tersebut.

Karena itu, pilihan dipersempit untuk memeriksa perencana. Saya pikir jika sesuatu dikirim dari sana, itu adalah sesuatu yang akan mati . Saya mengatur ulang delapan di bulan Januari, bukan dengan pengaturan standar, tetapi dengan menonaktifkan semua fitur yang tidak perlu bagi saya (omong-omong, pengaturan default adalah salah satu argumen dalam perselisihan). Jadi itu direncanakan sebagai sistem referensi.

Kutipan dari "Pernyataan Privasi Windows 8.1 dan Windows Server 2012 R2 (Tambahan Fitur)"

Seleksi dan manajemen
Jika opsi standar dipilih selama instalasi Windows, program peningkatan kualitas perangkat lunak Windows akan diaktifkan: Aplikasi Windows dan Microsoft dari Windows Store akan dapat mengirimkan laporan program peningkatan kualitas perangkat lunak ke semua pengguna komputer. Jika Anda telah memilih pengaturan opsi, Anda dapat mengelola program peningkatan kualitas perangkat lunak dengan memilih Kirim informasi Microsoft tentang bagaimana saya menggunakan komputer saya sebagai bagian dari program peningkatan kualitas perangkat lunak di bagian Bantuan Microsoft meningkatkan produk dan layanan. Setelah menginstal Windows, administrator dapat mengubah pengaturan ini di bagian Pusat Dukungan pada panel kontrol.

Kesimpulannya: Saya menyimpulkan sendiri bahwa Microsoft mungkin melanggar perjanjiannya sendiri dan, khususnya, mengirim data dari kamera video (jika ada keinginan dan kemampuan teknis). Sayangnya, "pesta paranoid" agak tepat.

Saya mengusulkan untuk menarik kesimpulan saya sendiri dari cerita kecil ini sendiri.
