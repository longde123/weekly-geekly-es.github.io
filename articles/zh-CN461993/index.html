如何从“正规设计师”变成杂货店

你好

我叫Alexey Svirido，我是Alfa Bank的数字产品设计师。 今天，我想谈谈如何从“正规设计师”变成杂货店。 在剪切下，您将找到以下问题的答案：

  • 谁是产品设计师，他是做什么的？
  • 这个专业适合您吗？
  • 怎样做才能成为产品设计师？
  • 如何创建第一个产品组合？

我们将与您一起绘制胜任力图，以了解您的哪些技能值得提升，哪些已经足够强大。


能力地图示例

奖励将是关于该主题的书籍和文章的推荐，这将有助于您提高工作水平。

正如我已经提到的，我在Alfa Bank工作，在这里我从事商业（即法人）数字产品的设计。 在此之前，我曾与世界各地的客户进行过各种自由职业项目。 然后，我在广播电台部门的Gazprom Media Holding开始了图形设计的旅程。 在那里，我从事户外广告，印刷，横幅设计，纪念品产品，创建品牌，标识等工作。我从平面设计到食品杂货设计，现在我将很高兴与您分享在此过渡期间学到的知识。

谁是产品设计师，他是做什么的？


产品设计师-是谁？ 有人设计牛奶，面包，饼干吗？ 还是创造家具和家庭装饰的人？ 还是约翰尼·伊夫（Johnny Ive），他离开了苹果公司，并创造了许多小工具和设备的挚爱？

大约7年前，杂货店设计师召集了创造某种物质，物体，家具，形式的家伙。 我认为，今天，这个定义完全是从创始人那里完全选择出来的。

现代产品设计是一个有关数字的故事，有关站点和应用程序，以及有关数字产品的故事。

因此，产品设计师是一个设计师，他不仅绘制项目的视觉部分，而且还考虑该项目如何工作并与用户交互。 也就是说，在产品中，您不能只画精美的东西。 这样可以很好地工作并解决某些问题，涵盖了用户，业务以及两者的需求。

这是产品设计的主要挑战之一：如何制作用户界面友好，外观美观的产品并带来业务收益。

产品设计师会影响：

  • 视觉语言。 决定与客户沟通的产品设计师是放松地，严格地，甚至是有点积极地进行了吗？ 产品设计师在界面中定义颜色，字体，形状和空间。
  • UX用户体验 。 产品设计师决定单击按钮，链接时或仅在输入文本时会发生什么。 客户端在等待下载或应用程序中断时将看到的内容。
  • 品牌品牌 是的，通过绘制界面，设计师可以影响品牌以及用户与该品牌的关系。 通过创建复杂且不舒服的交互场景，我们可以使用户对品牌的忠诚度降低，甚至放弃。 所以这很重要！
  • 指标 。 如果企业希望通过增加目标操作数量来增加利润，那么这就是产品设计人员的责任（作为专家之一，因为不仅产品设计师可以影响这一点）。 如果您需要在页面上延迟用户或提示用户采取行动-所有这些都可以在设计的帮助下完成。
  • 经营策略。 当企业朝某个方向发展并且对某件事充满信心时，产品设计可以证明先前的步骤是错误的，并且可以帮助改变策略。 例如，如果您考虑使用附带条件的贷款报价降落，那么客户不仅会去那里花钱，而且还会去花钱用于特定目的：购买汽车，公寓，考虑休假等。并且，如果您作为产品设计师可以确定真正的需求客户并向业务证明它-您能够对整个策略产生影响。

总而言之，让我们修复产品设计不仅仅在于美观的界面。 这些漂亮的界面可以工作并解决用户任务。

我真的需要这个吗？ 谁不适合呢？


为了理解为什么需要成为产品设计师，让我们看看为什么这个职业如此流行。 产品设计师的搜索到处都是，越来越多的公司。 尽管市场很拥挤，但公司仍在继续寻找产品设计师。

这是由于企业意识到设计确实会影响产品。 通过设计，您可以简单地从没人使用的膝盖上抬起产品，并且该产品无法负担其成本。 现代公司非常积极地在设计上进行投资，试图找到那些画出能为他们带来数百万美元的界面的人。

如何理解开发产品所需的知识？ 平面设计是一个非常有趣和创新的领域。 您可以借助许多工具来解决各种问题，进行实验，创建新的东西。 但是图形设计项目的寿命非常短，解决方案对最终用户的影响范围并不总是很大。 当然，有很大的项目可以改变世界，但是更多时候，他们必须满足于更少。

什么时候值得考虑转向产品设计：

  1. 当您了解到您希望影响更多的过程而不仅仅是绘制精美的图片时。
  2. 当逻辑和自由裁量权在您的设计中占优势时。
  3. 当您不仅要用手工作而且要用头工作时，请考虑产品生命周期的所有阶段。
  4. 当您认为自己对图形设计感到厌倦并且想要切换到新的东西时。
  5. 当技术和IT引起您比印刷和户外广告更多的兴趣时。

但是产品设计不适合谁：

  1. 有创造力的人，做艺术并做得很好的艺术家。 您会对产品感到不舒服。 但是，尽管如此，您可以随时尝试。
  2. 人们认为精美图片是唯一的设计产品。 不知道是否有。 长期以来，任何方向的设计都是首先解决问题的方法，然后才具有美观性。
  3. 不知道如何在团队中工作的个人。 这里的一切都清楚了。 对产品进行的工作始终是团队的工作，如果您不具备此技能，那么如果您确实想从事产品设计，则最好进行开发。

当前哪些技能尤为重要，需要提高哪些技能？


现在，让我们离开对话，草拟您的能力图。 能力图可帮助您清楚地了解自己的特长和需要提高的水平。 产品团队通常使用能力卡来了解设计师的成长点。 我们将起草您的个人卡。 首先，让我们看看它们可以是什么。

资格卡示例：





现在，让我们与您一起快速创建任何工具的胜任力图。 首先，让我们创建坐标轴并开始草绘您现在作为图形设计师所具有的能力。 这些不仅可以是基本特征，还可以是更复杂的东西，例如3D知识或运动设计。 顺便说一句，能力地图可以扩展并包括您的其他技能，例如html和css知识，某种编程语言的知识。

  • 版式
  • 色彩搭配
  • 组成
  • 书籍/杂志的布局
  • 插图
  • ...

现在，让我们考虑一下产品设计师应该具备的技能。 您可以添加评论。

  • 移情
  • 逻辑思维
  • 动画制作
  • 影像学
  • 文案写作
  • UX模式
  • 处理分析
  • ...

每种技能的总分都为10分。 有不同的评估选项，我们将尝试自己评估技能。 为了使这些数据更相关，您可以要求评估与您交流的其他设计师或与您合作的经理的技能。

现在我们有了一份胜任力图，让我们注意一下您当前掌握的技能中哪些特别重要，这将对您的产品设计有很大帮助。

因此，了解以下内容非常重要：

  • 成分 。 邻近性，空间和空间中元素相互作用的理论，大小由小到大。 在不了解构成规则的情况下，您可能会犯一些愚蠢的错误，这些错误将降低用户与产品交互的质量，并且通常会影响一些更重要的指标，例如利润。
  • 版式 。 现代界面不仅是视觉的，而且是文本。 并且文字处理应胜任工作，以便用户可以解决他们的问题，可以在不同的屏幕上阅读文字。 为界面选择和组合字体很重要。
  • 使用颜色 。 理解颜色的使用方法也很重要，因为例如，如果您在某些屏幕上违反了对比原理，则文本可能不可读，或者元素可能不可见。

换句话说，如果没有图形设计的基本原理和技能，产品将很困难，并且您可能会犯下许多荒谬的错误。

现在，要使用该产品需要泵送什么：

  • 同理心 。 这也许是主要技能之一。 不要混淆同情和同情。 同理心是指通过用户的眼睛看产品的能力。 同理心是指您可以与产品保持距离，冷静地看待产品以及用户如何使用它。 同理心的发展本身就可以帮助设计师清醒地评估产品和解决方案，以便了解可以改进的地方，而不是推动“最喜欢的决定”。
  • 逻辑思维 。 还需要提高并不断提高这项技能，因为有了它，您可以创建具有良好用户体验的真正周到的界面。
  • UX模式 。 现代产品是用户习惯的一组特定模式。 例如，我们已经知道，当您在社交网络上打开照片时，可以向下滑动以将其关闭。 或向左滑动时，我们将有其他选项或删除。 我们知道，通过在图片中轻按两次可以增加它，依此类推。 这样的模式越来越多，重要的是要了解它们的工作方式而不是破坏它们。 例如，假设滚动条在您的手机上被反转了，那么滚动条现在不再是从下往上滑动，而是从下到上工作。 作为用户，您必须在一段时间内习惯于新模式，这会引起某些不适和愤慨。
  • 文案写作 。 正如我所说，这些文本很重要。 不仅重要，而且外观也很重要。 当撰稿人与团队合作，并与设计师合作处理所有文本时，这是一个理想的故事。 但是通常情况并非如此，设计师本人需要编写文字。 而且很难。 这就是为什么要不断地运用这种技能，以改善您的文字并使它们与产品相关的原因。
  • 处理分析 。 了解什么是分析以及为什么需要分析非常重要，因为必须以某种方式验证和验证所有决策。 这在度量标准的帮助下发生。 设计人员不需要确切了解度量标准是如何挂起的，但是能够生成正确的度量标准并了解在其帮助下检查了哪些内容确实很重要。

这只是一个基本设置，对于开始创建不仅美观而且有用的界面（将解决实际问题）而言，这是必需的。 胜任力地图上可能包含更多技能，但是设置优先级仍然非常有用，这样您就只能先掌握最必要的技能，然后再提高其余技能。

我该如何抽水？


诚实对待自己并在能力图上给出真实的评分而不过高估计是非常重要的（为此，我建议您让设计人员或管理人员参与其中）。 这是突出显示那些真正需要提升技能的唯一方法。 但是，重要的是不要运用那些已经足够强大的技能。

让我们谈谈如何降低沉陷技能以及如何做：

  1. 分析您看到的所有接口。 从日常使用的运输和导航应用程序开始，到很少打开的应用程序结束。 考虑一下为什么很少打开该应用程序或该应用程序，以及如何使其更频繁地发生，这将是一个很好的考虑。 同样值得讨论的是在特定应用程序中什么是方便的，什么不是很方便。 那你会改变什么？
  2. 不断泵送观看者。 因此，您可以产生更多的决策，您的脑海中就会有更多的模式。 每天都值得抽奖，抽奖有很多方法：您可以观察兴趣，使用自己喜欢的想法创建公告板。 同样的运球也有助于提高意识。 虽然，值得注意的是，运球更多是视觉上的故事，是关于界面的外观，而不是其工作原理。 但是，您还可以在其中找到有趣的UX解决方案。
  3. 要升级UI技能，您可以从复制界面开始。 例如，您可以使用相同的Facebook，而只需重绘几个屏幕。 但是，不要轻描淡写地重新绘制它们，而是要深入研究并分析该元素或该元素位于特定位置的原因。
  4. 如果您了解复制对您很无聊，请开始重新设计界面，考虑如何改善此功能或该功能，也许值得交换元素，编写不同的文本，甚至重新粉刷所有内容并体现它。 总是尝试回答这个问题：为什么？ 为什么按钮在这里，为什么这样的颜色，为什么这是交互的方式，为什么界面如此工作等等。 试着用“为什么”尽可能跌倒。
  5. 最后一条建议是正在进行的实践。 没有这些，技能就会褪色，那些决定切换到产品的人根本就不需要它。 练习，尝试创建自己的界面。 对于那些有足够自信的人，我建议您挑战UI的100天。 www.dailyui.co挑战开始后，您每天将收到邮件的每日主题，您需要为其绘制一个界面。 例如，“发送字母”，“购物篮”，甚至“计算器”。 因此，您可以加入用户界面，如果在应对挑战时逻辑思考，您也可以加入UX。
  6. 阅读很多。 阅读文章，书籍，研究用户案例。 Behance可以在这里提供帮助，设计师经常在上面描述他们的决定，以及为什么得出一个或另一个结论。 这将有助于激发逻辑思维并在界面中建立因果关系。

现在，让我们看看新手最常犯的错误：

  1. 不规则 每周查看和解析接口模式还不够。 这必须每天进行。
  2. 复制错误的接口。 这里应该注意，您不应该连续复制所有内容。 通常，在商店中您会发现薄弱且构思不当的应用程序。 复制此类内容将导致您不会提高技能，反之亦然。
  3. 忽略泵送技巧。 最好是只有一起做所有事情。

如何创建第一个产品组合


没有投资组合的设计师找到工作不仅困难，而且几乎是不可能的。 但是，如果您拥有图形设计产品组合，但绝对没有产品项目该怎么办？

如何创建您的第一个产品组合：

  • 就像任何其他类型的设计一样，您可以为虚构的客户设计项目。 或重新设计已经知名的大型产品，例如Facebook，Instagram，邮件等。 在许多例子中，完成这项工作的人收到了他们正在为其重新设计的公司的报价。
  • 在投资组合中，您可以安全地将工作嵌入到先前所做的挑战中。 但是只有最好的工作！ 不要仅仅因为您从头到尾经历了挑战100天用户界面中插入的100张图像。 选择一些您可能引以为傲的特殊作品要好得多。
