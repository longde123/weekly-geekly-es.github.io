Kemitraan Strategis: Mengapa ServiceNow Bekerja Sama Dengan Penyedia Awan Besar

Microsoft menandatangani perjanjian kemitraan dengan ServiceNow, implementasi di mana kami terlibat dalam IT Guilds . Mari kita bicara tentang kemungkinan sasaran transaksi.


/ Unsplash / guille pozzi

Esensi dari perjanjian


Pada pertengahan Juli, ServiceNow mengumumkan bahwa beberapa solusi mereka akan digunakan di cloud Microsoft Azure. Secara khusus, ini berlaku untuk aplikasi untuk organisasi dari industri yang sangat diatur - misalnya, sektor pemerintah. Menurut ServiceNow, mereka akan dapat meningkatkan keamanan data pribadi.

Microsoft, pada gilirannya, berencana untuk menggunakan perangkat lunak ServiceNow. Secara khusus, kita berbicara tentang perangkat ITSM Teknologi Informasi dan Pengalaman Karyawan. Mereka menyederhanakan koordinasi karyawan dan mengurangi volume birokrasi dalam mengoordinasikan tugas. TI juga akan bertindak sebagai pengecer layanan ServiceNow.

Ada apa dengan pelanggan ini


Solusi baru untuk platform Sekarang ... ServiceNow mengatakan bahwa perusahaan akan dapat mengoptimalkan proses bisnis dan menawarkan lebih banyak layanan kepada pengguna. Secara khusus, teknologi mitra direncanakan untuk diterapkan dalam pengembangan produk analitis baru pada platform Now. Ini adalah solusi cloud yang cerdas untuk mengotomatisasi alur kerja, seperti otomatisasi persetujuan. ... dan banyak lagi. Alat baru direncanakan akan diterapkan di Microsoft 365 dengan Azure. Mereka akan melengkapi perangkat lunak Adobe Inc dan SAP SE, yang dengannya Microsoft juga mengadakan kemitraan.

Perluasan basis pengguna. Pasar untuk aplikasi SaaS perusahaan sangat terfragmentasi. Tetapi analis mengatakan pemimpinnya adalah Microsoft - korporasi memiliki saham 17%. Untuk ServiceNow, perjanjian kemitraan adalah kesempatan untuk menarik pelanggan baru ke ekosistem mereka dan mengembangkan produk ITSM bersama.

Analis di Bloomberg percaya bahwa masuknya pengguna baru akan membantu ServiceNow mencapai tujuannya yaitu 10 miliar pendapatan tahunan.

Lebih banyak layanan cloud di sektor pemerintah. Seperti yang kami katakan sebelumnya, ServiceNow menggunakan kekuatan Azure Cloud untuk menyediakan layanan kepada organisasi pemerintah. Omong-omong, langkah pertama ke arah ini diambil oleh perusahaan di musim gugur. Sekarang, pelanggan pemerintah akan dapat menggunakan layanan ini melalui sistem Pemerintah Azure. Ini adalah solusi cloud aman yang Microsoft harapkan untuk dikirim ke Pentagon.


/ Unsplash / Joshua Fuller

Pasar Baru untuk Layanan Sekarang - Jerman. Perwakilan ServiceNow mengatakan mereka berencana untuk mulai bekerja dengan organisasi pemerintah Jerman (dan sponsor negara di negara-negara Eropa lainnya). Azure Cloud mencakup banyak persyaratan penyimpanan. Sebagian besar, mereka didikte oleh GDPR dan hukum lain dari regulator setempat.

Tentang proyek cloud lainnya


Microsoft bukan satu-satunya organisasi besar yang bermitra dengan ServiceNow. Pada awal Mei, diketahui tentang proyek bersama perusahaan dengan Google. Cloud penyedia layanan IT Operations Management (ITOM) penyedia layanan hosting. Pelanggan dari kedua perusahaan menerima alat yang menyederhanakan penyebaran dan pengelolaan infrastruktur TI.

Organisasi juga berencana untuk bersama-sama mengembangkan kecerdasan buatan dan sistem pembelajaran mesin untuk mengoptimalkan proses kerja. Platform ITSM ServiceNow akan menggunakan teknologi Terjemahan AutoML Google. Idenya adalah menggunakannya untuk meningkatkan kualitas pengenalan suara di bot dukungan teknis obrolan.


/ Unsplash / Thomas Kelley

Tentang area ini, ServiceNow juga bekerja dengan Amazon. Layanan mereka Alexa for Business, asisten cerdas untuk perusahaan, memberi karyawan kemampuan untuk merencanakan pertemuan pribadi dan menggunakan aplikasi perusahaan menggunakan kontrol suara. Di antara aplikasi ini, ada solusi ServiceNow untuk manajemen proses TI.

ServiceNow juga bekerja dengan Adobe pada alat untuk membantu meningkatkan layanan pelanggan dan dukungan teknis. Dan juga dengan Deloitte - sistem over yang dapat meningkatkan produktivitas karyawan. Sangat mungkin bahwa dalam waktu dekat perusahaan berencana untuk menyimpulkan beberapa perjanjian dan kesepakatan kemitraan untuk mencapai tujuannya yaitu 10 miliar pendapatan tahunan.
