审查微型7“ GPD Pocket便携式计算机。系统管理员/程序员在夹克口袋中的工作场所

GPD是一家出现在IndieGoGo上的创业公司，以制造微型Windows计算机而闻名。 他们为之筹款的第一台设备是GPD Win，这是一种带有键盘和操纵杆的怪异游戏设备（在这里我进行了评论 ）。 即使在那时，他们在评论中也谈到了GPD Pocket-一种供不需要操纵杆的人使用的“口袋”笔记本电脑。 一段时间后，他们在GPD Pocket上筹集了资金，我从Gearbest订购了该产品。 让我们认识他。



当然，对于初学者来说，请看一下性能特征。
屏幕：7英寸，1920x1200（FullHD +）（323 PPI，16:10），大猩猩玻璃3
处理器：Intel Atom x7-Z8750（1.6 GHz，4核）
显卡：Intel HD Graphics 405（200 MHz基本，600 MHz-突发）
内存：8 GB LPDDR3-1600
内置闪存：128 GB eMMC三星DJNB4R
操作系统：Windows 10 Home 64 bit
声称的电池寿命：8小时视频播放，12小时使用
无线连接：Wi-Fi 802.11 a / b / g / n / ac，蓝牙4.1
插槽和接口：1 x USB Type-C，1 x HDMI D，1 x USB，1 x 3.5mm插孔
声音：Realtek ALC5645
传感器：重力，霍尔
尺寸和重量：180 x 106 x18.5毫米，0.503千克
电池：3.8V / 7000 mAh
价格：少于$ 500， 请在此处查看当前

让我们讨论这些特征。 为什么足够？为什么不呢？ 如果您将其与任何其他“上网本”（或中国人所说的超级本）进行比较，那么bash就变成了bash。 英特尔凌动Z8750是具有四个核心，8 GB内存的现代“原子”处理器（这非常重要-“上网本”中通常有一个相当慢的eMMC，因此“永远不会有太多的内存”），一个相当大的内置闪存驱动器，所有无线接口，大容量电池，耳机插孔以及全尺寸USB。 恕我直言-太好了 让我们看看它是如何生活的。

在PCMark Home Accelerated，我们可获得5.5小时的电池寿命。 这是一个很好的价值-因此，如果您只看电影，则可以节省8-9个小时的自主权。

额外的nishtyaki


由于GPD Win最初是为极客设计的，因此开发人员实现了他们的愿望。 什么样的问题首先出现在有关笔记本电脑材料的评论中？ 更正是否支持Ubuntu。 对于GPD Pocket，我们可以放心地说-是的，它受到支持。 开发人员直接声明了这一点-他们适应了该Linux系统。



好吧，当然，不是没有门槛的-尽管铁匠的开发人员忙于所有事情，但是某些功能不能很好地工作，您必须爬上Github进行修复。 但是与许多其他笔记本电脑相比，即使是带有铃鼓“ bubuntu”的舞蹈也无法倒出，这已经很好了。



开发人员甚至宣布将发布单独的Linux版本，但是显然，由于调试不太好，他们没有采取这一步骤。 而且，Windows许可证对于具有较小对角线的设备是免费的。

合计


当我查看GPD Pocket筹款活动时，我认为这是理想的选择！ 事实证明-嗯，不是真的。 因为这些人由于某种原因减少了键盘上的按钮数量，结果，用俄语输入大量内容变得非常不舒服。 好的-除非这是您的主要“印刷”笔记本电脑。 好吧，如果您不断地对其进行填充，那么您可能会习惯了（最后，我们在Psions上进行了打印，并且没有以某种方式消亡，整本书都写在了Psions上）。 这是该设备的主要缺点。

我认为其中有几个不是很重要：例如，这里没有网络摄像头和microSD卡插槽。 还有（如果您有梦想的话）-LTE调制解调器。

否则，一切都很棒。 出色的屏幕，舒适的键盘（如果您忽略俄语），足够的性能，足够的RAM和足够的eMMC，良好的电池寿命。 不，真的。

好吧，价格已经变得比GPD Win的价格高-它的售价约为22k（可以更改） ，而对于GPD Pocket，他们要求将近30k（ 可以更改，请参阅当前版本 ）。
