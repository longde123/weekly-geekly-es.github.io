Die 9 besten Open Source-Funde für November 2019

Guten Winter, meine Damen und Herren. Ich habe für Sie eine Auswahl der interessantesten Funde aus der Open Source für November 2019 zusammengestellt.


Für eine vollständige Liste neuer nützlicher Tools, Artikel und Berichte können Sie sich an meinen Telegrammkanal @OpensourceFindings wenden ( Linkspiegel, falls das Original nicht geöffnet wird).


In der heutigen Ausgabe.
Technologien im Inneren: Rust, TypeScript, JavaScript, Go, Python.
Themen: Webentwicklung, Tools für die Qualitätssicherung, Arbeiten mit Daten, Entwickler- und Administrator-Tools.


Letzte Ausgabe .


Geschichte


Eine Zeitmaschine für Github. Hier können Sie buchstäblich sehen, wie eine Datei zu einem bestimmten Zeitpunkt aussah. Komfortabel, schön.
Es ist in JavaScript geschrieben.


Link



Qawolf


Ein einziger Befehl von npx qawolf record example.com zum Aufzeichnen von Browsertests. Es sieht super einfach aus.
In TypeScript geschrieben.


Link



plynth


Und jetzt ein bisschen Raserei. Sie können schnelle, native, thread-sichere Webanwendungen in HTML / CSS / Python schreiben. Was denn Python Ja, Python.
Sie können Vue.js sogar intern verwenden. Und schreiben Sie Code in Python. Hier, schau mal!
Es ist in Python geschrieben.


Link



Lazydocker


Minimalistische TUI für diejenigen, die viel mit Docker arbeiten und gerne eine Maus benutzen.
Geschrieben in Go.


Link



tui-rs


Übrigens über TUI. Hier finden Sie eine nützliche Bibliothek zum Erstellen Ihrer Anwendungen. In ein paar vorgefertigten Widgets.
In Rust geschrieben.


Link



easydb


Datenbank mit einem Klick. Ohne Registrierung und SMS.
Clients sind für Python, JavaScript, Ruby, Bash.


Link



transform.tools


Sehr bequemer Service, mit dem Sie ein Format automatisch in ein anderes konvertieren können. Zum Beispiel:


  • HTML und SVG in jsx
  • json in Kotlin, TypeScript oder Go Struct
  • Markdown in HTML, Yaml in Json usw.

Link



Dovpanda


Linter für pandas . Findet Fehler und fordert auf. Sehr nützlich für diejenigen, die viel mit Daten arbeiten. Oder lehrt sie, wie man arbeitet.
Es ist in Python geschrieben.


Link



fließend


Die Bibliothek zum Erstellen von Flussdiagrammen wird auf dem Desktop und auf dem Mobilgerät ausgeführt. Schön!
Es ist in JavaScript geschrieben.


Link

Das ist alles für heute. Für diejenigen, die die Auswahl mögen - abonnieren Sie den Kanal. Es gibt viele andere interessante Projekte. Verbesserungsvorschläge, Links zu Projekten, Feedback - schreiben Sie in die Kommentare.
