Nexus-Abdruck: Fingerabdruck-Identifizierung für Nexus



Das Android 6.0-Betriebssystem ist das erste mobile Betriebssystem von Google, das native Unterstützung für die Identifizierung von Fingerabdrücken bietet. Um die neue Funktion des Google-Betriebssystems zu demonstrieren, musste Fingerabdruck-Hardware erstellt werden, ein System namens Nexus Imprint. Es funktioniert auf dem Nexus 5X und Nexus 6P. Mit diesem System können Sie das Gerät schnell entsperren oder andere Aktionen ausführen, für die eine Benutzerauthentifizierung erforderlich ist.

Ein Fingerabdrucksensor befindet sich auf der Rückseite des Smartphones. Sobald der Fingerabdruck im System registriert ist, können Sie das Telefon mit diesem Pad entsperren, indem Sie einfach Ihren Finger einige Sekunden lang auf dem Pad halten.

Hier müssen Sie das Telefon nicht einmal mit der Taste einschalten - alles funktioniert automatisch. Das Touchpad befindet sich ganz bequem - mit einer gewissen Fähigkeit muss nicht einmal das Telefon gedreht werden - der Finger selbst findet den richtigen Punkt. Dies ist viel praktischer als die Möglichkeit, eine solche Plattform unter dem Bildschirm zu platzieren, wenn Sie das Telefon in eine Hand nehmen und Ihren Finger auf die andere legen müssen.



Das System arbeitet so schnell, dass Sie den Netzschalter zum Aktivieren des Geräts vergessen können - eine Berührung, und das Telefon ist „aufgewickelt“. Wenn das Telefon eingeschaltet ist, tippen Sie auf das Touchpad, um es auszuschalten.



Sie können das System sogar trainieren, indem Sie es an die strukturellen Merkmale Ihres Fingers anpassen :) Mit anderen Worten, die Algorithmen passen sich dem für Sie geeigneten Fingerwinkel an.

Auf Wunsch können Sie nicht nur Ihren, sondern auch Ihre Frau, Verwandte oder Freunde mit einem Fingerabdruck versehen (normalerweise haben nur wenige Menschen einen solchen Wunsch, aber Sie wissen es nie). Außerdem können Sie den Fingerabdruck der anderen Hand registrieren, so dass die Möglichkeit einer alternativen Aktivierung des Geräts besteht.



Nexus Imprint kann auch verwendet werden, um mobile Zahlungen mithilfe des Android Pay-Systems zu autorisieren.
