映泰推出12种图形卡主板



以太坊采矿的群众热情正在增长。 多亏了这种加密货币，GPU才变得重要起来，因为ASIC不适合使用它。 顶级视频卡像热蛋糕一样飞翔 （有些根本无法使用，例如，如今无处找不到RX 470）。 制造商非常看好这一趋势-并提供专门为矿工设计的新产品。

映泰（Biostar）已发布全球首款具有12个原生PCI-e 3.0连接器的主板 。 映泰TB250-BTC Pro是一个真正的采矿场的平台，否则为什么您需要12个视频卡？



先前用于8个Biostar视频卡的主板型号在Computex 2017上展示-并受到公众的好评，早期的H81A和TB85带有六个连接器的型号也被矿工使用。 因此，公司决定继续增加动力。 目前，TB250-BTC Pro是市场上唯一具有12个PCI-e 3.0插槽的主板。

该主板支持同时连接图形，AMD和Nvidia。



该公司认为，就价格和性能而言，配备12个图形卡的主板是最赚钱的采矿解决方案。 如果较早时有必要将两台计算机与两块包含六个视频卡的主板放在一起，那么我必须为每台计算机购买一个单独的CPU，一个单独的SSD和DDR4内存。 也就是说，所有这些都必须分两部分购买，现在一个CPU，一个SSD和一个DDR4模块就足够了。 根据Biostar的计算，以相同的挖矿性能可以节省200美元，例如，在六块RX 470图形卡和六块GTX 1060卡上，尽管您仍然必须购买两个电源。 下表列出了Biostar的计算结果。



根据制造商的估计，基于当前的加密货币成本和246.6兆字节/秒的哈希值，新的视频卡将比使用相同数量的两个系统上的视频卡采矿要快18天，而成本要高出200美元。 投资回报率从97天减少到79天。 自然，这是一个理论预测，有许多未知数，现实并不像Biostar乐观。



Biostar强调，在此型号中，PCI-e插槽彼此之间的距离比其他制造商更远，这降低了短路风险并提高了稳定性。



出于相同的目的，母板使用固态电容器和刚性电源管理模块。

主板具有标准的外围连接器：两个PS / 2，DVI，四个USB 2.0端口和四个USB3.0（一个千兆位LAN）。

支持的处理器：Intel Core i7 LGA 1151，Intel Core i5 LGA 1151，Intel Core i3 LGA 1151，Intel Pentium LGA 1151，Intel Celeron LGA1151。最大TDP为95瓦。 内存模块DDR4 2400/2133/1866 MHz的两个插槽。

采矿的普及迫使视频卡制造商发布专门为加密货币采矿设计的新视频卡。 华硕最近发布了MINING系列显卡：这是具有4 GB GDDR5视频内存和926 MHz时钟频率（加速到1206 MHz）的MINING RX 470 。


采矿RX 470

具有两个风扇的卡被设计为24/7全天候运行，即不间断地全天候运行。

反过来，又通过特许经营为Nvidia发布了另一款矿机显卡MINING P106 （1506-1708 MHz，6 GB DDR5内存）。

两款视频卡均带有一个特殊的TWEAK II GPU程序，该程序可让您配置GP​​U时钟速度和内存，同时监视哈希率和功耗，以选择最佳设置。

2017年7月，映泰（Biostar）加入了采矿图形卡制造商行列，发布了RX 470D。


映泰RX 470D

某些现代采矿显卡，例如GeForce GTX 1060，通常没有用于显示器的输出连接器。 如果您不需要采矿显示器，为什么还要在板上增加额外的芯片负担？

顺便说一句， 根据传言 ，AMD打算对Radeon RX 460进行略微修改，并发布Radeon RX 560D型号，这也是矿工们的期望。

分析人士希望，所有这些新的采矿显卡将帮助市场应对标准Radeon RX 4xx和RX 5xx显卡的严重短缺，从而减少对它们的需求。 矿工立即将这些模型推向市场，这就是为什么最初为这些视频卡创建游戏的玩家遇到问题的原因。
