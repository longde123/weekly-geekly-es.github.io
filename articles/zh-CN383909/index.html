我们考虑华硕Zenfone 2，考虑到用户的意见

华硕商标一向以其独特的制成品方法而著称，这在计算机界已得到证明。首先，生产的主要领域是个人计算机，笔记本电脑，外围设备和PC的各种组件的生产。

但是不久前，该公司开始进入移动设备市场。朝着这个方向发展的第一个有力步骤是推出了Transformer系列小工具，其中将智能手机插入平板电脑并同时结合了两个功能设备。由于选择了方向，TM ASUS已提高了在移动领域的评级。 2014年初，该公司提出了另一条ZenFone智能手机系列，该系列产品因价格适中，填充效果好而在市场上扎根。今天我们将讨论它。智能手机在世界所有国家都非常受欢迎。华硕TM的兴趣最高峰是在2015年国际消费电子展上意外展示了华硕ZenFone 2型号。今天我们将与他见面。

*注意交通

该设备采用了64位四核Intel Atom Z3580处理器，从而可以与该价格段的其他制造商的智能手机平等竞争，并且成为第一款具有4 GB RAM的智能手机。这款新设备具有许多令人愉悦的功能，Android爱好者不会忽视。我注意到外观上的积极方面-这些是5.5英寸显示屏的细边框，具有全高清分辨率，因此像素密度很高。好吧，考虑到我国的汇率，今天智能手机的宣布价值似乎是允许的。

外观设计特点


智能手机的设计保留了ASUS产品标准的基本风格。华硕ZenFone 2提供五种颜色-白色，红色，灰色，黑色和金色。外观上，智能手机看起来很原始，并且吸引了人们的注意。

主要创新是“摇杆”音量控制，它位于小工具的背面，位于具有双闪光灯的主13百万像素相机下方。其他制造商的智能手机中已经使用了类似的解决方案，例如，我们在最新型号的LG智能手机中看到了类似的性能。但是与上述具有类似解决方案的小工具系列不同，华硕ZenFone 2的开/关键位于上端中央。



在用户中，此键的位置收到了模棱两可的响应。按照惯例，人们分为三类，有人声称锁和开/关钥匙应该与音量摇杆在同一位置。其他人则抱怨这两种选择带来的不便，所有按键都应完全放在一边。还有一些人说，您应该将电源键移至侧面，将“摇杆”留在背面。我个人注意到该决定的确可以引起争议，但是每年它在越来越多的制造商的智能手机中使用。在这方面，我建议您在购买前先参观有展位的商店，然后自己决定这个问题对您是否至关重要？您更喜欢哪种解决方案，经典的端部布局或其他？



得益于圆形的后盖，该设备可舒适地放置在各种肤色的人的手掌中。小工具的尺寸（重量为170克）为77 x 152 x 10.9毫米，也有助于实现这一目标。对于这种对角线，显示器侧面的距离非常窄。底部和顶部的缩进也不会超出标准框架。



制造智能手机的主要材料是塑料。盖子采用拉丝金属制成，框架为黑色光泽。选择身体的浅色调时，指纹和其他油腻标记不应明显。微小划痕或其他可能出现的瑕疵不太可能损坏小工具的外观，因为很难在厚的光亮清漆层上注意到它们。唯一需要注意的是-智能手机有些打滑，可以从您的手掌中飞出。

前面板由专有的钢化玻璃保护，并涂有疏油涂层。在大多数情况下，没有注意到有关制造质量，零件和材料的装配的投诉。就此而言，可以得出结论，智能手机在出厂时准确，可靠且符合质量标准。

显示


华硕ZenFone 2配备5.5英寸的IPS类型矩阵，分辨率为1920 x 1080像素。像素密度为414 ppi，屏幕的纵横比为16：9。这样的参数很有竞争力，并且已经被认为是这些设备的标准配置。

屏幕模块的供应商是日本的Sharp公司。如上所述，不受外部影响，显示器由耐用的钢化玻璃保护，该玻璃具有无空气间隙的olefob涂层。



显示屏上图片的质量及其细节很可能不会让任何人感到冷漠。全方位的视角最大。高水平的色彩还原-图像不失真，色彩不反转。在入射角处，没有对比度或亮度。用户没有抱怨图片或所用矩阵的质量。

传感器最多支持十个同时触摸-使用了多点触摸技术。华硕还声称已经实现了60 ms的最小显示响应。

硬件平台和性能


作为处理器，华硕ZenFone 2具有新的64位Intel Atom Z3580，时钟速度高达2.3 GHz和PowerVR G6430图形。智能手机中的RAM为4 GB，但是还有一个版本带有Z3560处理器和2 GB的RAM。内置内存-32 GB。有一个用于存储卡的插槽，最大容量为64 GB。根据未经确认的报告，智能手机可识别最大128 GB的卡。

在综合测试中，智能手机显示出良好的结果，但这并不奇怪。目前，这是最领先的现代和生产性馅料之一。 SoC没有任何问题，可确保智能手机界面的平稳运行，并实现其在苛刻应用中的潜力。至少在中等设置下，不可能找到无法使用智能手机应对的游戏。



我想强调的一点是，要突出显示两个SIM卡和两个无线电模块的支持，这种情况很少见。第一个是主要的，它支持所有网络（最高4G）（FDD-LTE和TD-LTE）中的操作。第二个仅在第二代网络（2G）的模式下工作。还有一个额外的麦克风，用于降低噪音。实际上，这样的解决方案可能会有用，因为在第一个“ SIM卡”上进行对话期间，您将可以进入第二个。还有其他标准通信接口：蓝牙4.1和Wi-Fi 802.11 b / g / n / ac，双频。另外，还支持USB-OTG和NFC。没有红外线端口，但是很遗憾会有完整的端口。



软件


该操作系统为Android 5.0，带有集成的专有Asus ZenUI外壳。它与大多数启动器的用户友好性不同，并且不会引起大多数“用户”的不满。



在国际版本的交付中，启动器不会加载广告应用程序（俄罗斯联邦的交付中可能有第三方预装的应用程序），它具有广泛的自定义功能。可以在桌面和菜单中更改应用程序网格的大小，隐藏不必要的程序，对其进行排序，以及更改图标。

如果启动器不符合您的条件，则可以将其关闭，然后将其替换为另一个。但是诸如锁屏和带有设置菜单的通知帘之类的东西很可能仍然是标准配置，例如华硕开发人员所看到的那样。

提到扩展手势支持。一个例子是关闭屏幕上的“双击”以唤醒智能手机。同样，可以通过在状态栏上单击两次或在桌面上释放可用空间来锁定智能手机。

根据评论，该固件运行正常，没有任何明显的错误。

电池


华硕ZenFone 2具有不可拆卸的锂离子电池，容量为3,000 mAh-这是此型号智能手机的合适解决方案。使用标准充电器充满电大约需要2到2.5个小时。

还宣布支持专有技术华硕BoostMaster，该技术可为智能手机提供加速充电。使用品牌充电器（18W），ZenFone 2电池可在约40分钟内充电达60％。



通过适度的使用模式，智能手机可以一直活到今天结束，并且根据评论，有时直到第二天开始。在积极使用的情况下，操作时间当然会减少数倍，并接近4-5小时。例如玩沉重的3D游戏或在要求苛刻的应用程序中工作。在通过LTE连接连续浏览Internet的情况下，情况类似。在这种模式下，智能手机不太可能工作7小时以上。您可以尝试通过调节显示屏的背光来使正负的工作时间均等，但这不会使您赢得一个半小时以上。通常，您可以指望一天的工作。在自治方面，华硕ZenFone 2显然不比同类产品差。

相机


华硕ZenFone 2有两个摄像头，主13 MP和前置5 MP。借助所使用的软件驱动程序，可以对所使用的光学器件进行积极的评估，使人们在白天和光线良好的情况下都能获得清晰的图像。但是像往常一样，有人抱怨在弱光或弱光条件下拍摄时图像质量会下降。为了使视频拍摄流畅，有一个稳定器，该稳定器旨在提供流畅的图像。声音以立体声录制。



摄像头界面很直观，但根据大多数用户的说法，它有很多设置和模式，不堪重负。但是，我认为大量的设置不是负号而是加号。人们可能不喜欢在拍摄时迷恋助手，这是另一回事。他认为，应用程序本身可以在适当的时间更改设置。例如，如果摄像机教导了强大的背光灯的存在，那么它会立即建议打开HDR。



总结


总之，我注意到华硕原来是一款非常有趣的产品，具有有趣且具有竞争力的用户特征。在竞争对手中，智能手机显示出易于使用，稳定且忠诚的价格。当然，您会发现本文中没有提到的故障，例如缺少触摸控制键的背光，这可能会通过将智能手机放在有灰尘或沙土的桌子上而损坏主摄像头的窗户。尽管它看起来有些奇怪甚至荒谬，但可能性很小，我认为它是存在的:)。
如果我们专注于旗舰店的自命不凡，那么为什么指纹扫描仪会绕过智能手机侧或使用相同的激光自动对焦功能并不清楚。但是所有这些都消除了有关设备成本的问题（根据Y. Market，按当前汇率，它还没有达到24,000卢布的门槛）。如今，在品牌的智能手机中找到具有两个独立的“ SIM卡”并具有近似特性的模拟产品是不是很困难？我认为没有太多选择。
