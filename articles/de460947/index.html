WASI: Eine API für Nicht-Web-WebAssembly


Der Bericht wird in englischer Sprache per Telefonkonferenz gelesen

Um WebAssembly außerhalb des Browsers zu platzieren, erstellen wir WASI, die WebAssembly-Systemschnittstelle. Es handelt sich um eine Standard-API für WebAssembly-Anwendungen, die außerhalb des Web verwendet werden kann. Dadurch wird die Portabilitäts- und Sandboxing-Philosophie von WebAssembly auf den API-Bereich ausgeweitet. Ich werde darüber sprechen, wie wir hierher gekommen sind, was Sie heute damit machen können und wohin wir gehen!
