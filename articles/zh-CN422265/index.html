PICASO 3D Designer X PRO 3D打印机概述



你好 今天，我们将考虑来自俄罗斯PICASO 3D公司的新产品-PICASO 3D Designer X PRO打印机。 这是PICASO 3D Designer PRO 250的更新和改进版本。它还支持同时使用两种材料进行打印，例如，允许使用水溶性PVA塑料打印支持物或使用两种颜色进行打印。




打印区域与其他PICASO 3D打印机相同-200x200x210毫米。 通常，它是配备了所有必要功能的现代个人FDM打印机。



PICASO 3D设计器X PRO




特点


  • 印刷技术：FFF（FDM）
  • 尺寸（毫米）：492 x 390 x 430
  • 重量，公斤：16
  • 软体：PICASO 3D Polygon X
  • 打印头数量：2
  • 支持的文件格式：stl，plg
  • 工作室，毫米：200 x 200 x 210
  • 框架：钢
  • 平台：铝，玻璃
  • 层厚度，微米，从：50-200
  • XY定位精度，微米：11
  • 定位精度Z，微米：1.25
  • 挤出温度，最高，°C：380
  • 平台温度，最高，°C：140
  • 支持的材料：ABS，PLA，FLEX，NYLON，ASA，ABS \ PC，PET，PC，PVA，HIPS
  • 螺纹直径，毫米：1.75±0.1
  • 喷嘴直径，毫米：0.3
  • 接口：USB，USB闪存，Micro SD，以太网
  • 打印速度，cm3 / h，最高：30
  • 擦价： 279 000

套餐捆绑


打印机随附：打印机，闪存驱动器，电缆，胶水瓶，工具套件，灯丝盘和文档：说明，数据表，保修卡。



打印机概述


层厚度可在200至50微米之间调节。



打印机使用直径为1.75毫米的标准螺纹，这种螺纹很普遍，不使用PICASO芯片，因此您可以使用自己喜欢的制造商的塑料进行打印。



玻璃被用作餐桌材料，没有胶粘在其上，因此，为了获得更好的附着力，建议使用胶水或喷涂进行3D打印。



平台被加热，这是打印所必需的，例如使用ABS塑料。 像以前的打印机一样，该挤出机由JetSwitch技术提供动力。 这种挤出机的技术使得可以快速地从一种材料切换到另一种材料。 它由两个可移动喷嘴，一个塑料进料马达和一个无效的喷嘴截止阀组成，该阀可消除塑料泄漏，并且在物料之间切换的时间不到5秒。



端口在micro SD下消失了，现在可以从USB闪存驱动器或通过以太网电缆开始打印。 顺便说一下，现在可以通过Polygon X专有软件对打印机进行联网和管理。


线圈通常安装在打印机内部的特殊手柄上。



就像Pro 250一样，打印机也只有黑色。 外观有所变化，但主体隐藏在内部-现在打印机导板不再固定在外壳上，而是固定在打印机内部的特殊钢制框架上，这增加了整体结构的刚度，并在校准和打印时提供了更高的定位精度，这也增加了打印机的重量，大约每1公斤，最多16。



机械手按照标准笛卡尔方案进行排列：平台沿Z轴上下移动，挤出机沿X和Y轴移动；运动学根据H-Bot原理进行排列，这在FDM打印机中最为常见。 外壳由铝复合材料制成，内框和钢轨也一样。 沿X和Y轴的导轨，沿Z轴的圆柱导轨。



背光已升级，现在具有功能含义：白色-打印时，绿色-最后，黄色-如果发生故障。



现在，打印机本身可以检测到塑料供应阻塞或丢失。 打印机监视塑料的进度，如果出现问题，将尝试使用清洁算法清洁喷嘴。 如果这样做没有帮助，打印机将停止并开始发出问题信号。

软件总览


许多创新与打印机固件有关。



打印机现在支持配置文件系统，这是存储在打印机内存中的某种塑料的一组设置，也就是说，现在您可以选择要直接在打印机中打印的塑料设置，而在切片机上，您仅准备切片本身。



例如，一旦下载了模型，就可以使用其他材料打印该模型，而无需更改g代码，而只需在打印机中选择所需的配置文件即可。



该打印机的切片器是PolygonX，它取代了Polygon 2.0。 它具有简单的界面和设置。 分别配置模型的切片，单独配置打印机的配置文件。 您可以通过以太网电缆控制打印机。



标定






值得一提的是控制第一层。 这是打印机内置的系统。 通过分析所选的轮廓，有关塑料进纸速度和到平台的距离的数据，打印机可以即时更改沿Z轴的工作台高度，以获得理想的第一层，这对于FDM打印非常重要。



如何开始列印


准备好要在切片器中打印的文件后，可以通过两种方式将其发送以进行打印：从USB闪存驱动器，从计算机或通过以太网直接从切片器向其传输g代码。



在打印之前，您可以立即设置质量，它与速度成反比。



例如，用于打印ABS的平台需要涂上粘合剂。 将其放在餐巾纸上并擦拭表面。



请勿在打印机内部喷胶。 它将停留在运动部件上并聚合，这将损坏设备。

列印




打印是FDM打印机的标准配置。 打印机显示有关打印进度的信息。



您可以使打印机处于活动的暂停状态，在此暂停中，挤出机将从工作台上移开，您可以更换物料。 另外，在旅途中，您可以手动更改打印速度，喷嘴温度和第一层夹具的高度。





产品清除


如果使用ABS进行打印，则冷却后成品本身会从桌子上脱落。 在其他情况下，您可能需要该套件随附的文具刀。







合计




与许多其他打印机不同，PICASO 3D Designer X PRO打印机并未屈服于增加工作量的趋势，而是专注于打印质量和易用性。 这由轮廓系统，用于控制塑料和印刷第一层的系统以及指示系统提供服务。



高精度，可使用两种材料进行打印的支撑以及较高的打印温度，使得可以使用大多数现有的细丝（包括碳填充的碳丝）用可溶性细丝创建高质量的物体，从而可以精确打印各种功能性产品。 我们建议使用PICASO 3D Designer X PRO创建功能性原型和样品，以及零件的​​零件生产。 该打印机适合所有关心打印精度的人，由于某种原因，立体光刻技术不适合。
