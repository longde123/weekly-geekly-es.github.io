更新了第三代eGrado模型



EGrado 耳机是流行的iGrado的更新版本。新颖性是否改变了声音？还是制造商仅局限于颜色和名称？

我们已经审查了便携式iGrado。这些是带有枕形弓的开放式头戴式耳机。那些不熟悉Grado产品的人通常会对字母“ i”和Apple智能手机产生疑问。实际上，该指数与苹果公司无关，新字母“ e”也与智能手机无关。 E系列-第三代耳机，对于基本型号，可以看作MK III或V3。

有趣的是，格拉多（Grado）拥有家族企业，并在2013年将管理层移交给了第三代（乔纳森·格拉多（Jonathan Grado））。下一代的第四代耳机可能不会早于10-15年。可以在网上

详细查看Grado的历史

在公司的资料中，您可以看到新的E系列的大量改进。在已宣布的创新中，与第二代“ i”（Impruved）的主要区别是使用新的扬声器。问题是它们有多好？
直接比较模型并进行详细测量变得更加有趣。

eGrado的外观





在外部，iGrado和eGrado相同，只是颜色不同。泡罩包装保持一对一。仅包括耳机。



eGrado的主要功能是戴着帽子戴耳机的能力，这在夏天非常重要。耳机的设计非常简单。从视觉上看，您可能会认为杯子连接到船首的位置有旋转接头，但实际上没有旋转机构。



该电缆看起来很耐用，并且在外观上与iGrado型号的电缆没有区别。插头倾斜，最适合移动使用。

电缆附近最痛苦的地方之一通常位于耳机罩附近-电缆弯曲经常发生在此处，从而导致断线。为了消除该问题，软凸轮被用来增加弯曲半径，但是，许多追求经济的制造商将硬凸轮本着“依此类推”的原则。值得赞扬的是iGrado / eGrado-柔软的纤维，这解释了iGrado对电缆问题缺乏反馈的原因。



正式地，eGrado不在制造商的任何型号范围内，但据说扬声器的使用与Grado SR60e完全相同，并且eGrado和iGrado的声音之间的差异可以间接预测SR60i和SR60e之间的差异。

IGrado和eGrado测量





为了测量耳机，使用了高科技设备-一种特殊的头部，HDM-X人体模型带有参考音频分析仪测量系统的柔软耳朵。在测量之前，麦克风会通过声级计进行校准，这不仅使您能够测量幅度-频率特性，而且还能够测量耳机的灵敏度（并且由于灵敏度的知识，您可以预测“播放器/智能手机会摆动耳机吗？”）。

iGrado和eGrado的频率平衡实际上是相同的，整体差异可以归因于耳机的合身性稍有不同。但是灵敏度明显不同。对于iGrado，它是106 dB / V（94 dB / mW），而对于eGrado，它是112 dB / V（100 dB / mW）。因此，获得6dB的差异，其对应于音量的两倍的差异。这至少证实了说话者确实不同。

灵敏度的提高永远是一个加号，因为要求使用功能更弱的信号源（在便携式领域中，这意味着能耗更低，从而保证了更长的智能手机/播放器操作时间）。

有趣的是，Grado为iGrado和eGrado提供了相同的特性。缺乏严格的标准为在完全不同的条件下进行测量（有时会获得冲突的数据）提供了极大的灵活性。仅进行那些在相同且优选接近实际操作条件下进行的测量。在这种情况下，人体头部形状的人体模型，自然佩戴和经过校准的麦克风会捕获可信任的数据。如果我们谈论巧合，那么eGrado模型的敏感度数据就重合了。



电阻在32欧姆时保持不变。但同时，共振峰从50 Hz变为100 Hz，变得更加明显。当使用具有高输出阻抗的放大器时，eGrado将具有更明显的低频，但放大器阻抗为20欧姆（常见）时只有2 dB，不是很多。

技术人员通常认为阻抗越相等，效果越好，并且所有共振峰都是“邪恶的”。实际上，这种方法是单方面的。为了获得一个理想的特征，有必要牺牲一些其他的特征，一个好的营销广告商的任务就是展示“好的特征”并隐藏“牺牲的”。从主观上讲，一般声音具有一组特征，而主观评估是决定性的。从单个特性的观点来看，仅在改变放大器电阻的频率响应的方向上才应考虑阻抗不均匀。

eGrado的声音





要评估声音，请使用播放器Hidizs AP100。

eGrado的声音与架子扬声器的声音非常相似，但低频会逐渐降低。那些。声音在中高频时可以说是平滑的，而不会产生过多的低音。因此，耳机更适合于旋律音乐。由于采用开放式设计，因此可以注意到舞台宽阔且完全缺乏隔音效果。

缺乏隔音性是周围声音的可听性好（音乐本身音量适中）。在街道条件下，这会自动提供更高的安全性。但是，相反，在嘈杂的交通工具中，这对于听音乐来说是一个问题，必须大声地将其打开，这既是听觉的下降，又是周围人们非常相似地听到这种音乐的不满。

iGrado和eGrado之间的音量差异非常好，并且eGrado明显更大。但是数量和质量不是一回事。从详细阐述各个位置的角度来看，eGrado在整个频率范围内都提供了更高的分辨率。iGrado感觉不太详细，但在某些情况下，当连接到简单的智能手机时，它们可能是更可取的-细节更少，因此从mp3压缩和低质量的放大器中听到的问题也更少。

结论





eGrado 的新颖性可以被认为是成功的，保留了专有的声音，提高了灵敏度并改善了声音细节。耳机，适合那些对音质和自然音质了解很多的人。

由于没有隔音的开放式设计，耳机非常适合在户外使用。eGrado主要是夏季型号，最适合与帽子搭配使用。
