Sejarah Singkat Layar Biru Kematian



Ini adalah hal yang kultus, meskipun Microsoft tidak menginginkannya. Pada tahun 90-an, itu adalah bagian yang sama dari pengalaman dasar menggunakan Windows sebagai Paint dan Solitaire, tetapi hari ini tidak begitu sering terlihat.

Tentu saja, saya berbicara tentang " layar biru kematian ", BSoD. Pengguna PC yang lebih muda tidak tahu seberapa luas layar penyebaran panik ini dulu, atau apa artinya. Segala sesuatu yang Anda kerjakan hilang, komputer Anda memerlukan reboot - maka itu bisa memakan waktu sepuluh menit.

Kita yang mengingatnya berusaha melupakannya - tetapi ini tidak mudah dilakukan.

Sampai hari ini, BSoD adalah simbol yang dikenali dari sesuatu yang tidak berfungsi, tetapi mengapa itu bahkan muncul? Ini adalah perjalanan singkat di sepanjang bagian jalan kenangan yang sulit dibedakan, bahaya yang orang tua Anda peringatkan kepada Anda.

Windows 3.1: layar Ctrl + Alt + Delete




Windows 3.1 tidak memiliki BSoD: ketika crash, Anda melihat layar hitam. Jika Anda beruntung, bidang input perintah DOS muncul di layar ini, dan Anda dapat memulai Windows lagi. Jika tidak, waktu boot ulang telah tiba.

Namun, itu memiliki layar biru yang dipanggil dengan menekan Ctrl + Alt + Delete. Di masa depan, ia akan menginspirasi desain BSoD.

Menariknya, sebagaimana dicatat pada posting blog oleh Raymond Chen, teks untuk layar biru ditulis oleh direktur perusahaan masa depan, Steve Ballmer, ketika ia mengelola divisi sistem Microsoft.

Windows 95 dan 98: layar biru pertama kematian


Sulit untuk melebih-lebihkan pentingnya Windows 95: bayangkan hype di sekitar model iPhone awal, tetapi hanya tentang sistem operasi desktop. Orang-orang berbaris di toko-toko. Ya, tahun 90-an itu aneh: orang-orang senang dengan fitur-fitur baru komputer desktop. Tapi tidak ada yang antusias dengan layar ini.



BSDD muncul setiap kali sebuah program atau driver crash. Dia menawarkan semacam informasi terenkripsi tentang penyebab masalah, dan kemudian memberikan saran yang lebih jelas tentang apa yang harus dilakukan selanjutnya.

Secara teori, menekan tombol apa saja harus menutup program ini dan mengembalikan Anda ke desktop Windows, tetapi dalam praktiknya jarang berhasil. Menurut Raymond Chen, versi sebelumnya dari pesan ini mengatakan bahwa "Dimungkinkan untuk terus bekerja dalam mode normal," tetapi kemudian baris ini dihapus karena optimisme yang berlebihan.

Windows 2000: mungkin diperiksa untuk virus?




Dengan rilis Windows 2000, Microsoft telah memperluas serangkaian kiat layar biru. Semua referensi tentang kemungkinan kembali ke desktop menghilang, dan pengguna segera disarankan untuk mematikan komputer. Ada juga serangkaian ide untuk melacak kesalahan jika masalah ini muncul lagi, dari pemindaian virus hingga memeriksa integritas hard drive.

Windows XP, Vista, dan 7: Lebih Banyak Tip




Windows XP melanjutkan tren peningkatan jumlah tips di layar biru. Informasi tentang program mana yang menyebabkan masalah itu masih kabur, tetapi setidaknya memberikan beberapa kode yang dapat dicari di Google. Sisa layar diisi dengan segala macam tips. Pengguna masih disarankan untuk mematikan komputer, tetapi mereka juga disarankan untuk memeriksa apakah semua program diinstal dengan benar, dan mereka juga memberikan serangkaian ide untuk menemukan alasannya.

Di Vista, layar biru tidak banyak berubah, tetapi mulai muncul lebih sering. Windows 7 mengurangi jumlah penampilannya, tetapi tidak mengubah penampilannya.

Windows 8: emoticon sedih tanpa detail




Windows 8 mengubah BSoD sepenuhnya. Layar teks, mirip dengan terminal, menghilang, digantikan oleh font sistem modern dan senyum sedih yang sangat besar di ASCII. Hal yang paling menarik adalah bahwa hampir semua informasi tentang penyebab masalah telah hilang, demikian pula kiat untuk menemukan solusinya.

Ini bukan masalah serius seperti sebelumnya, karena layar biru jauh kurang umum saat ini. Anda dapat mengetahui penyebab crash PC dengan memeriksa log atau menggunakan program pihak ketiga yang mengumpulkan informasi tersebut.

Pada Windows 10, layarnya terlihat sama.

Warisan layar biru


Microsoft ingin hal-hal menjadi berbeda, tetapi hingga hari ini layar biru kematian tetap menjadi simbol Windows. Fakta ini menginspirasi salah satu game office terhebat sepanjang masa - screen saver BSoD. Itu dibuat oleh Sysinternals (yang kemudian dibeli Microsoft), dan itu membuat komputer apa pun tampak seperti membeku sampai Anda menekan tombol atau menggerakkan mouse Anda. Sangat lucu.
