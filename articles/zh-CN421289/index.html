一年半的时间里，俄罗斯的电动汽车数量从920增加到2500


2014年7月9日，莫斯科首批特斯拉汽车之一。 照片： 谢尔盖·罗多夫尼坚科（Sergey Rodovnichenko）

在一年半的时间里，在俄罗斯注册的电动汽车数量增加了2.5倍以上，即从920辆增加到2500辆。 据Avtostat机构称，电动汽车的绝大多数是日产聆风，三菱i-MiEV和特斯拉Model S的滞后时间位居第二和第三。

以下是2018年7月1日的统计数据：

  1. 日产Leaf-1800个
  2. 三菱i-MiEV-294个
  3. Tesla Model S-202个


三菱i-MiEV

这些模型中的三种占据了俄罗斯电动汽车总数的90％以上。 它包含更多奇特的汽车，包括该国唯一的特斯拉Model 3：

  • 拉达·埃拉达（Lada Ellada）-93件
  • Tesla Model X-88个
  • 雷诺Twizy-27件
  • 宝马i3-11个。
  • 特斯拉Model 3-1-pc

俄罗斯最“电气化”的地区根本不是莫斯科或圣彼得堡，而是滨海边疆区，那里注册了约25％的俄罗斯带有电厂的汽车（586辆）。 相比之下，莫斯科只有369个，莫斯科地区只有98个，圣彼得堡有73个。 大量的电动汽车在哈巴罗夫斯克地区，克拉斯诺达尔地区，伊尔库茨克州和阿穆尔州的道路上行驶。

奇怪的是，尽管在中国组装了非常便宜和负担得起的车型，但在俄罗斯却没有一个由中国组装的电动汽车。 它们可能进口到俄罗斯可能有些复杂。

同时，俄罗斯公司卡拉什尼科夫（Kalashnikov）最近推出了一种新型CV-1电动汽车，该汽车已被称为特斯拉杀手 。


基于苏联掀背式IZH-Combi的电动超级跑车的概念

“这项技术将使我们能够与特斯拉等全球电动汽车制造商相提并论，并与它们竞争。 在创建我们的概念时，我们受到了世界市场领导者的经验的启发，”卡拉什尼科夫关注组织说。

“电动超级跑车的电池由革命性的逆变器提供动力，尺寸为50x50x100厘米，重量为50千克，使您能够开发功率高达1.2兆瓦的负载。 电池容量为90 kW / h。 提出的概念基于ILZ-21252“ Combi”巡洋舰，是开发复杂系统的试验台。 它们的状况由卡拉什尼科夫公司自己开发的通用车载计算机进行调节，它使您可以监视电动机，逆变器和电池的状况。 功率CV-1-220 kW，峰值-500 kW。 汽车的动力储备为350公里。 卡拉什尼科夫在一份声明中说，从0到100 km / h的加速时间为6秒。

除了基于IZH-21252 Combi的CV-1超级跑车之外，卡拉什尼科夫关注的对象还在Army-2018国际论坛上为民用市场推出了UV-4电动汽车 。


电动车UV-4民用市场

规格UV-4：

  • 电机类型：无刷直流电机
  • 功率：最高50 kW
  • 最高时速：80 km / h
  • 续航里程：长达150公里
  • 重量：650公斤
  • 电池类型：LiFePO4，LiPO
  • 尺寸（LHV），毫米：3440×1530×1730

据报道，电动汽车“具有事故发生时的高度平滑性和低着火和爆炸危险的特征”。 此外，该车“易于使用且需要最少的维护”。

先前对俄罗斯电动汽车市场的研究是由Avtostat机构于2017年1月进行的。 当时，俄罗斯的车队只有920辆汽车 ，而日产Leaf的份额则要少得多：只有37％，而不是现在的70％。

以下是2017年1月1日的统计信息：

  1. 日产叶子-340个
  2. 三菱i-MiEV-263个
  3. Tesla Model S-177个
  4. 拉达·埃拉达（Lada Ellada）-93件

雷诺·特威齐，特斯拉Model X和宝马i3的少于20个。

根据统计数据，一年半以来，俄罗斯特斯拉Model S车队增加了25辆，三菱i-MiEV增长了31辆，拉达Ellada增长了0辆，日产Leaf增长了1460辆。 也许某种错误潜入了对Autostat的研究。
