引入AudioCodes路由器VoIP路由器

灵活性和移动性为当今的小型企业提供了竞争优势。而且，新的通信技术为此类企业提供了大量昂贵且复杂的工具，因此，只有拥有适当专业人员的大型组织才能使用它们。互联网已经与信使和电子邮件一起牢固地进入了我们的生活，但是电话技术正以非常严重的速度发展，但是电话仍然是主要且最便捷的通信方式之一。与传统电话相比，IP电话正在找到越来越多的支持者，它提供了更加方便的功能：答录机，“语音秘书”服务，可以连接任何号码，包括8-800个“热线”以及更多功能。



制造商试图跟上趋势并寻求使用户的生活更轻松，这就是为什么有趣的解决方案开始出现在以SOHO领域为导向的市场上，并允许在一个地方组织所有办公通信而没有任何问题的原因。我们将讨论一种可以替代办公室Internet中心（或对其进行补充）的设备。

为办公室配备电话的问题可以通过多种方式解决：连接固定电话，为员工配备移动电话和IP电话。通常，第一种方法是最不灵活的：适度的功能和移动时丢失数字；第二种是作为辅助工具，第三种是IP电话，尽管具有所有优点，但许多人都不知道“如何做饭”。的确，如果您连接有条件的无线路由器并建立Internet是大多数对本质有基本了解的人将要完成的任务，那么对于没有准备的人而言，建立办公室电话的任务似乎将很困难。这是因为直到最近，这种设备都是考虑到专业人士定制的，但是市场趋势却使制造商打破了普遍的定型观念。当然，“精密”设备并没有消失，但这仍然是具有扩展功能需求的大公司的特权。但是小型企业市场SOHO / SMB需要设计简单易用的解决方案。



其中之一是多服务VoIP路由器的AudioCodes路由器系列，在我们的示例中为MP-272DB模型，该模型也以Megafon R2品牌进行分发。首先，它旨在全面替换或增加办公室的Internet中心。

就像他们说的那样，一次查看就更容易了：



从技术上讲，这是经典无线路由器与PBX（带有集成的DECT基座）和两个用于连接普通电话的模拟端口的结合。该设备可以通过三种方式连接到Internet：通过ADSL2 +或VDSL线路，通过经典以太网以及用于3G / 4G网络的USB调制解调器。并非所有的SOHO路由器都提供如此丰富的功能。如果通过主要提供商突然缺少Internet连接，则设备将通过3G / 4G网络自动切换到备用频道。

从结构上讲，该设备采用白色塑料制成的略微倾斜的垂直版本，侧面带有黑色插件，在其前面板上有一个用于提供的DECT管的充电杯。





许多状态指示器位于此处，包括系统，网络连接，WiFi，DECT等的状态。设备背面的底部有一个电源端口，以太网端口（一个用于连接Internet，四个用于LAN设备），xDSL，带有RJ-11连接器的FXS和USB。所有天线都在内部，没有外部。该设备看起来非常吸引人，并且外观不会破坏内部。



该设备为全天候工作提供了支持，该工作是通过单独购买的外接电池进行的，这样，如果房间突然中断，您就不会失去沟通。为此，在后面板上有一个特殊的连接器。



办公室用户可以使用有线连接访问Internet，有线连接使用设备上的四个快速或千兆以太网端口（分别为每秒100或1000兆位，具体取决于型号），并使用无线WiFi 802.11b / g / n，最高速度为300 Mbps（2x2）。为了用户的方便和安全，支持所有常见的网络协议和解决方案：SPI防火墙，UPnP，802.1p / q（VLAN和QoS），多播（IGMP代理，监听），内置的WiFi接入点可让您组织多个网络（多个SSID）和支持WPA / WPA2，WPS，以及通过浏览器的授权。

但这仅是典型SOHO路由器的描述，虽然不错，但仍是附带功能。从历史上看，AudioCodes专门研究IP电话设备。制造商走得更远：除了具有两个模拟FXS端口的经典VoIP网关外，该设备还具有完善的无线DECT / GAP基础，最多可支持五部手机（套件中包含一个）。



电话协议使用广泛的TCP / UDP上的SIP及其带有TLS加密的安全版本。语音编解码器由以下列表表示：G.711μ/ a-law，G.729A / B，G.723.1，G.722（用于HD VoIP），G722.2（WB-AMR）以及传真支持。如上所述，DECT最多支持五部手机。内置PBX支持5个DECT用户，两个通过FXS端口的模拟用户以及最多10个注册的IP电话。还提供了对拨号计划，保持，转移，呼叫转移以及最多两个独立的三方会议的全面支持。

自动传真支持T30，T.38协议，此外还有有用的Fax2Mail功能，可以自动将传真发送到电子邮件和Mail2Fax，工作方向相反，并且可以与各种图像的TXT，PDF，DOC，Excel，Power Point格式附件一起使用。

随附的DECT手机也是白色的，具有彩色显示屏和扬声器。它具有两节可更换AAA电池，充电杯直接位于路由器上。如果不需要在设备上为听筒充电，充电杯很容易与主体分离，这使路由器更紧凑。手机功能与类似的DECT手机没有太大区别，但此处已组织了便捷的呼叫操作功能：保持，转接，电话会议。



在其他有用的功能中，还支持用于组织文件服务器的外部USB驱动器，对USB打印机的支持以及在本地网络（LPD / IPP）上发布它们的支持。可以在以FAT32或NTFS格式格式化的外部驱动器上组织文件服务器。对于高级管理功能，除了Web界面外，还有一个通过Telnet和SNMP代理的命令行。



IP电话在我们的生活中变得越来越牢固，很明显，该技术将以非常快的速度进入大众市场。AudioCodes路由器多服务路由器证明了组织具有所有属性的小型办公室非常容易：一台设备取代了Internet上的路由器PBX，使您可以通过3G / 4G来组织Internet上的备份通道，并进行文件存储和旧打印机网络的建立，以及内置的DECT支持将使摆脱一些不必要的电线成为可能。不要忘记IP电话中AudioCodes的经验，即使在非常低廉的成本下，也可以保证高度的兼容性。
