Caixa de ferramentas para pesquisadores - terceira edição: localizando e trabalhando com fontes

O trabalho em qualquer projeto de pesquisa envolve a pesquisa e o estudo de muitas fontes de informação. A organização deste processo não é uma tarefa fácil. Hoje falaremos sobre ferramentas projetadas para otimizar seus vários componentes.

  • Caixa de Ferramentas do Pesquisador # 2: Uma seleção de 15 bancos de dados temáticos
  • Caixa de ferramentas dos pesquisadores nº 1: auto-organização e visualização de dados

Os fornecedores de software acadêmico geralmente trabalham com o princípio da colaboração com instituições de ensino. Se sua organização não adquiriu o produto do seu interesse, pode ser difícil organizar o acesso a ele. Se existirem planos individuais, nem todos estão prontos para pagar por eles do próprio bolso. Por outro lado, produtos abertos ou gratuitos criados por acadêmicos geralmente sofrem com design inconveniente e falta de suporte.

Você pode experimentar muitas ferramentas diferentes antes de optar por uma versão funcional. Decidimos falar sobre o mais notável nas coleções Hacker News e GitHub.

Sourcing


DOAJ é um catálogo com curadoria de revistas científicas abertas. Em sua base - mais de 4 milhões de artigos de 13 mil publicações representando 130 países. Num futuro próximo, o site planeja aumentar significativamente o número de revistas humanitárias, altamente especializadas e em inglês. Curiosamente, os DOAJs fornecem uma API aberta para automatizar pesquisas em vários parâmetros.

Peerus - ajuda a acompanhar as publicações tópicas recentes. Todos os dias, o arquivo é atualizado com uma média de 7 mil novos materiais. O Peerus fornece integração de proxy universitário para downloads com um clique.

Crossref é um arquivo aberto para publicação e pesquisa de artigos científicos. O objetivo do projeto é reunir cientistas de todo o mundo em um local para que eles possam resolver problemas de pesquisa em conjunto. O site já coletou cem milhões de entradas. Você pode encontrar o conteúdo necessário por metadados e palavras-chave.

SciRate é um aplicativo da web de pesquisa em arXiv.org . Existem filtros por campo científico e data de publicação. Além disso, um sistema de classificações de usuários foi implementado, com base no qual cada artigo recebe uma classificação.

ArXiv Sanity Preserver - acelera sua pesquisa no arXiv.org usando algoritmos de aprendizado de máquina. A página inicial exibe os materiais mais recentes, juntamente com a anotação. Você pode procurar publicações por palavras-chave - na barra de pesquisa e por sinais semelhantes - usando o link show similar, localizado no canto superior direito, próximo a cada documento.

OSF é um centro aberto para pesquisar e criar projetos científicos. Aqui você pode estruturar seu trabalho de pesquisa - para estabelecer controle de versão e adicionar tags para que outras pessoas possam encontrar a publicação. A plataforma envolve o trabalho na nuvem, o que facilita o acesso aos seus projetos. O trabalho pode ser feito de forma independente e compartilhado com os membros da comunidade.

Organização e citação


doi2bib é um mecanismo de pesquisa para trabalhar com listas de fontes. Um identificador de objeto digital (DOI) é inserido na barra de pesquisa e o sistema exibe metadados para o material encontrado para organizar a lista bibliográfica no formato BibTeX .

O JabRef é um aplicativo que otimiza o trabalho com bibliografias no formato BibTeX. A lista gerada é exportada para HTML, Docbook, BibTeXML, MODS, RTF, Referência / Nota de fim, OpenOffice e LibreOffice. Você pode executar o utilitário na Java VM ( versão 8 ), em Win, Linux e OS X.

Mendeley é um gerenciador de listas bibliográficas e uma plataforma em nuvem que permite acessar arquivos do projeto a partir de qualquer dispositivo. Mendeley também oferece oportunidades de consultoria e ferramentas de comunicação.

O Docear é uma ferramenta aberta para organizar fontes e criar trabalhos científicos com base. O usuário precisa especificar uma pasta que desempenhará o papel de uma "biblioteca" de materiais para um projeto específico. O programa irá varrê-lo automaticamente e adicionar os arquivos encontrados ao seu banco de dados. O banco de dados pode ser estruturado na forma de uma "árvore", combinando trabalho em uma direção em um ramo. Comentários e destaques em arquivos pdf são importados automaticamente e também exibidos na interface. Quando o trabalho com as fontes terminar, o Docear ajudará a formar o "mapa mental" final e fornecerá as citações necessárias.

Duecredit - você pode citar não apenas texto, mas também código. Se o seu programa emprestar algoritmos ou métodos de trabalhos científicos, esta biblioteca o ajudará a consultá-los corretamente. A biblioteca está atualmente disponível em Python.

Trabalhar com documentos


Sphinx é uma ferramenta para “compilar” documentos no reStructuredText em HTML, ePub, Texinfo, páginas de manual e texto sem formatação. Ele suporta 50 extensões , indexação automática de componentes de software, criação de links para funções, classes, cotações, termos e assim por diante. Para iniciantes, os desenvolvedores prepararam um tutorial e um guia para um início rápido . Além disso, sugerimos que você analise os seguintes tópicos para trabalhar no Sphinx: rtd - suporta Sphinx e Read the Docs ; O Bootstrap é um tema em que modelos CSS e extensões JavaScript da estrutura do Bootstrap se integram à funcionalidade do Sphinx: navegação, menus hierárquicos e assim por diante.

O Git-scribe é uma ferramenta de código aberto para a criação de livros eletrônicos. Fornece funcionalidade para estruturar texto, edição, revisão, correção, colaboração, além de publicar e exportar para PDF, mobi e epub.

O Jupyter Notebook é um editor de texto com um ambiente embutido para programação interativa em Python. Anteriormente conhecido como iPython Notebook. Ele pode ser instalado localmente e no VPS, existem versões em nuvem do produto.

Textura - a interface deste editor fornece acesso a uma ampla gama de funções especializadas - como a criação de gráficos, fórmulas e vários métodos de citação. O programa usa um formato de arquivo especial, que, se desejado, pode ser "descompactado" e desmontado em componentes. Segundo os criadores do Texture, isso permite que os editores simplifiquem o processo de edição e revisão.
