Un moteur de recherche visuel est apparu sur Pinterest: les objets de la photo peuvent désormais être identifiés



Il arrive souvent qu'en regardant une photo d'un appartement ou d'une maison, un utilisateur de Pinterest aime une bibliothèque, un lustre ou un gadget. Mais qu'est-ce que c'est et où vous pouvez acheter, seules les personnes qui connaissent les tendances modernes de la décoration intérieure le savent. Que faire le reste? Utilisez le service de moteur de recherche visuel , les développeurs l'ont ajouté pour la commodité des utilisateurs.

Et en effet, tout a été fait de manière très pratique - si vous sélectionnez un objet séparé et effectuez une recherche, le service affichera des photos et une description d'objets similaires. La même lampe peut être mise en évidence sur la photo, après avoir reçu toute une liste de produits similaires.



Un lien vers le moteur de recherche visuel est situé dans le coin. Vous pouvez également affiner votre recherche si les résultats sont trop nombreux. Vous pouvez filtrer les résultats, par exemple, par balises et catégories. Dans ce cas, lors de la recherche d'une ampoule, les autres catégories ne seront pas affichées, dont les objets peuvent se révéler quelque peu similaires à l'objet de recherche.

Le même lustre peut être porté dans les catégories «antique» ou «design moderne». Bien sûr, la recherche n'est pas effectuée sur le Web, mais sur la base de données Pinterest, mais la base de services est désormais énorme.

Un service de recherche visuelle sera disponible pour les utilisateurs de Pinterest, y compris une application Web mobile.
