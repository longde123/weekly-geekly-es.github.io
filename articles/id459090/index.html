Bawa pengalaman pengembangan Linux Anda di Windows ke level berikutnya dengan WSL dan Visual Studio Code Remote

Menggunakan VS Code Remote dan Windows Subsystem untuk Linux (WSL) memberi Anda lingkungan pengembangan Linux berfitur lengkap pada laptop atau desktop Windows. Mari kita lihat bagaimana menggunakan alat-alat ini sepenuhnya akan mengubah cara Anda mengembangkan dengan alat-alat Linux di Windows.



Hal pertama yang pertama: Menginstal apa yang Anda butuhkan


Anda akan membutuhkan WSL dan VS Code dengan VS Code Remote - ekstensi WSL diinstal untuk melengkapi konten dalam posting blog ini. Kami akan dengan cepat membahas keduanya, dan bagaimana cara menginstalnya.

Subsistem Windows untuk Linux


Biasanya disingkat menjadi WSL, alat ini memungkinkan Anda menjalankan binari Linux yang tidak dimodifikasi langsung di Windows. Ini berarti Anda dapat menggunakan alat Linux favorit Anda, aplikasi baris perintah, dan alur kerja di mesin Windows Anda tanpa harus menggunakan pengaturan rumit seperti boot ganda. Yang terbaik dari semuanya, WSL adalah solusi terintegrasi yang erat antara Windows dan Linux. Jadi, jika Anda menyukai Linux, Anda dapat mengakses semua file Windows dan file yang dapat dieksekusi langsung dari bash shell, dan jika Anda menyukai Windows, Anda dapat dengan mudah mengakses binari Linux dan file dari PowerShell atau CMD.

Jika Anda baru mengenal WSL dan ingin mempelajari lebih lanjut, Anda dapat menyelesaikan kursus lengkap kami: Mulai dengan Windows Subsystem untuk Linux , atau Anda dapat menemukan hanya petunjuk instalasi pada dokumen WSL .

Kode studio visual


Visual Studio Code adalah editor kode sumber yang ringan namun kuat yang berjalan di desktop Anda dan tersedia untuk Windows, macOS dan Linux. Muncul dengan dukungan bawaan untuk JavaScript, TypeScript dan Node.js dan memiliki ekosistem ekstensi yang kaya untuk bahasa lain (seperti C ++, C #, Java, Python, PHP, Go) dan runtimes (seperti .NET dan Unity) . Unduh dan mulai perjalanan Anda dengan VS Code .

Anda juga akan memerlukan VS Code Remote - ekstensi WSL, yang memungkinkan Anda untuk menggunakan wadah, mesin jarak jauh, atau Windows Subsystem untuk Linux (WSL) sebagai lingkungan pengembangan berfitur lengkap. Untuk mempelajari lebih lanjut dan memulai silakan kunjungi halaman dokumentasi Remote VS Code .

Siap, siap, Linux!


Alih-alih menulis bagian panjang tentang bagaimana Anda memulai dengan VS Code Remote di WSL, kami telah mengumpulkan video cepat ini yang menunjukkan bagaimana melakukan pengembangan python di WSL menggunakan VS Code Remote.


Ini bukan hanya terisolasi ke python, Anda dapat menggunakan VSCode Remote dan WSL pada proyek Linux apa pun


Saat menggunakan VS Code Remote di WSL, perintah dan ekstensi dijalankan langsung di distro Linux, jadi Anda tidak perlu khawatir tentang masalah jalur, kompatibilitas biner, atau tantangan lintas-OS lainnya. Anda dapat menggunakan Kode VS di WSL seperti yang Anda lakukan dari Windows.

VS Code Remote dan WSL adalah pengubah game dalam hal pengembangan dengan Linux di Windows. Alat-alat ini memungkinkan Anda mengembangkan, menjalankan, dan men-debug banyak aplikasi Linux secara langsung di mesin Windows Anda, memberi Anda yang terbaik dari lingkungan Windows dan Linux.

Jika Anda ingin mempelajari lebih lanjut tentang Subsistem Windows untuk Linux dan semua baris perintah di Windows, tetap ikuti perkembangan blog ini! Dan jika Anda penasaran untuk mempelajari lebih lanjut tentang VS Code Remote, Anda dapat membaca posting blog pengumuman di sini dan dokumentasi lengkap yang membahas arsitektur di sini .

Kami senang menghadirkan kombinasi VS Code Remote dan WSL yang tangguh!

Selamat coding!
