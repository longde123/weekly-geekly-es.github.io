¿Quién agregó Python a la última actualización de Windows?

Hace unos días, el equipo de Windows anunció la actualización de mayo de 2019 para Windows 10 . En esta publicación, veremos lo que nosotros, el equipo de Python, hemos hecho para facilitar la instalación de Python en Windows. En particular, hablaremos sobre Microsoft Store y la adición del comando predeterminado "python.exe" para facilitar la búsqueda (en colaboración con Windows). Es posible que ya haya escuchado sobre esto en el podcast Python Bytes , en PyCon US o en Twitter .




A medida que el software pasa de una PC a la nube, el navegador y la Internet de las cosas, los flujos de trabajo de desarrollo cambian. Aunque Visual Studio sigue siendo un excelente punto de partida para cualquier carga de trabajo en Windows, muchos desarrolladores ahora prefieren usar las herramientas individualmente y bajo demanda.


Para otros sistemas operativos, el administrador de paquetes compatible con la plataforma es el lugar tradicional para encontrar herramientas individuales que se hayan configurado, probado y probado para el sistema. En Windows, estamos explorando formas de proporcionar experiencias similares a los desarrolladores sin afectar a los usuarios que no son desarrolladores y sin comprometer la capacidad de los editores para administrar sus lanzamientos. El Subsistema de Windows para Linux es un enfoque que ofrece a los desarrolladores consistencia entre sus entornos de compilación e implementación. Pero hay otras herramientas para desarrolladores que también importan.


Una de esas herramientas es Python. Microsoft ha estado trabajando con la comunidad Python durante más de doce años y actualmente está interactuando directamente con los cuatro contribuyentes clave para el lenguaje y el tiempo de ejecución principal. El desarrollo de Python es increíble, ya que es utilizado por analistas de datos, desarrolladores web, administradores de sistemas y estudiantes, y al menos la mitad de sus tareas ya se pueden realizar en Windows . Pero mientras que los desarrolladores de Python en Windows se enfrentan a más preguntas que en otras plataformas.


Instalar Python en Windows




Durante muchos años, se sabe que Windows es el único sistema operativo importante que no tiene un intérprete de Python incorporado. Para muchos usuarios que nunca necesitarán esto, esto ayuda a reducir el tamaño y aumentar la seguridad del sistema operativo. Pero para aquellos de nosotros que lo necesitamos, la falta de Python se siente mucho.


Una vez que descubra que necesita obtener Python, rápidamente encontrará muchas opciones. ¿Descargarás el instalador desde python.org? ¿O tal vez una distribución como Anaconda? El instalador de Visual Studio también es una opción. ¿Y qué versión se necesita? ¿Cómo obtendrá acceso después de la instalación? Encontrará más respuestas de las que necesita, y dependiendo de su situación, cualquiera de ellas puede ser correcta.


Nos tomamos el tiempo para entender por qué alguien encuentra el error mencionado anteriormente y qué tipo de ayuda necesita. Si ya es un experto en Python con necesidades complejas, probablemente sepa cómo instalarlo y usarlo. Es mucho más probable que alguien encuentre este problema por primera vez cuando intente usar Python. Muchos de los maestros con los que hablamos confirmaron esta hipótesis: los estudiantes se enfrentan a esto con mucha más frecuencia que los desarrolladores experimentados.


Y lo hicimos todo más fácil.



Primero, ayudamos a la comunidad a lanzar Python en Microsoft Store. Esta versión de Python está totalmente preparada por la comunidad, es fácil de instalar en Windows 10 y automáticamente pone a disposición los comandos estándar de python , pip e idle (como sus equivalentes en python3 y python3.7 , para todos los comandos, como en Linux).



Finalmente, con la actualización de mayo de Windows 2019, estamos terminando la imagen. Aunque Python continúa siendo completamente independiente del sistema operativo, cada instalación de Windows incluirá los comandos python y python3 que lo llevarán directamente a la página de Python . Creemos que Microsoft Store es ideal para los usuarios que comienzan a trabajar con Python y, dada nuestra experiencia y participación en la comunidad de Python, nos complace apoyarlo como la opción predeterminada.




Esperamos que todos estén tan felices como Scott Hanselman cuando descubra esto . Con el tiempo, planeamos expandir una integración similar con otras herramientas para desarrolladores y resolver problemas para comenzar.
