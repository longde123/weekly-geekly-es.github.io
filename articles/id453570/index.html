Microsoft Edge untuk macOS

Bulan lalu, kami mengumumkan pra-build pertama dari versi Microsoft Edge untuk Windows 10. Sekarang kami dengan senang hati mengumumkan bahwa saluran Microsoft Edge Canary juga telah tersedia untuk macOS. Sekarang di situs web Microsoft Edge Insider Anda dapat menemukan build untuk komputer dengan macOS dan Windows 10, dan segera dukungan untuk versi Windows akan diperluas.  



Microsoft Edge untuk Mac


Browser Microsoft Edge untuk macOS akan menawarkan pengguna fitur baru yang sama seperti pada versi awal untuk Windows, tetapi antarmuka akan disesuaikan untuk platform Mac. Kami mengubah antarmuka dan fungsionalitas sesuai dengan apa yang digunakan dan diharapkan pengguna macOS.



Kami berupaya menjadikan Microsoft Edge sebagai peramban kelas dunia dengan fitur unik dan layanan terkait. Dalam rilis pertama ini, kami membuat beberapa perubahan pada antarmuka pengguna untuk menyelaraskannya dengan desain Microsoft, tetapi pada saat yang sama membuatnya alami untuk macOS.


Perubahan tersebut menyangkut font macOS, menu, pintasan keyboard, heading, dan elemen lainnya. Dalam rilis mendatang, kami akan terus mengubah tampilan dan fungsionalitas browser, kami akan bereksperimen dan menanggapi umpan balik pengguna. Karenanya, beri tahu kami tentang tayangan Anda menggunakan emotikon Kirim umpan balik.



Selain itu, kami mengembangkan cara untuk berinteraksi dengan pengguna, eksklusif untuk macOS, berdasarkan fitur perangkat keras khusus dari platform Mac. Jadi, pengguna dapat melakukan tindakan dan mengakses perintah konteks menggunakan panel sentuh (Touch Bar). Misalnya, Anda dapat mengakses pintasan situs web, beralih di antara tab dan mengontrol video, dan menggunakan navigasi yang familier menggunakan gerakan trackpad.



Memperkenalkan Microsoft Edge Insider feed untuk macOS


Microsoft Edge pre-builds baru untuk macOS tersedia melalui saluran pra-rilis, yang kami sebut saluran Microsoft Edge Insider. Kami mulai dengan meluncurkan saluran saluran Canary, yang bisa Anda dapatkan di situs web Microsoft Edge Insider . Saluran ini sudah tersedia untuk macOS 10.12 dan lebih tinggi. Kami akan segera membuka saluran Dev, dan ketika tersedia, Anda dapat mengunduh dan menginstal unit ini secara paralel dengan Canary. Anda dapat mengetahui lebih lanjut tentang pendekatan kami dan apa yang diharapkan dari berbagai saluran di pos kami selama sebulan terakhir .



Platform dan alat yang konsisten untuk pengembang web


Karena fakta bahwa build baru kami didasarkan pada Chromium, Anda mendapatkan rendering yang sama di kedua versi Microsoft Edge - untuk Windows dan macOS. Selain itu, Anda mendapatkan akses yang sama ke alat pengembang yang kuat.


Untuk pertama kalinya, pengembang web dapat menguji situs dan aplikasi web di Microsoft Edge pada macOS dan memastikan bahwa mereka akan bekerja sama di versi Microsoft Edge berikutnya pada semua platform. (Perhatikan bahwa fitur khusus platform, seperti dekripsi konten PlayReady di Windows 10, masih harus dipantau secara terpisah untuk interaksi terbaik dengan mereka pada platform ini).


Seperti pre-builds untuk Windows, versi baru kami untuk macOS juga mendukung Progressive Web Apps, yang diinstal pada standar dan dapat diperiksa dan didebug menggunakan alat pengembang di browser. Kami sedang bekerja untuk memastikan bahwa aplikasi tersebut dapat digunakan dengan nyaman bersama dengan aplikasi platform lainnya, jadi setelah instalasi mereka akan ditampilkan di Dock, App Switcher dan Spotlight seperti aplikasi lainnya.  


Umpan balik Anda penting bagi kami


Kami senang berbagi dengan Anda build pertama kami Microsoft Edge Canary untuk macOS! Kami membutuhkan tanggapan Anda untuk menjadikan peramban lebih baik, karena kami percaya bahwa peramban semudah mungkin bagi pengguna macOS. Kami harap Anda mencoba pratinjau hari ini dan menantikan tanggapan dan partisipasi Anda di komunitas Microsoft Edge Insider.


Jika Anda memiliki masalah atau ingin meninggalkan umpan balik atau menyarankan sesuatu, Anda dapat meninggalkan pesan di forum komunitas Microsoft Edge Insider , hubungi kami di Twitter, atau cukup gunakan perintah "Kirim umpan balik" di menu Microsoft Edge, untuk berbagi pemikiran Anda dengan kami.


Jika Anda seorang pengembang web dan mengalami masalah yang mereproduksi di Chromium, yang terbaik adalah melaporkan bug di komunitas Chromium . Untuk melaporkan masalah dalam versi Microsoft Edge saat ini, silakan terus menggunakan Pelacak Isu EdgeHTML .
