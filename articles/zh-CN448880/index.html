PHP内部


  • [RFC]箭头功能2.0-对短lambda的投票已经开始，投票过程令人鼓舞。 语法保持不变： fn($x) => $x * $y 。
  • [RFC]弃用左关联三元运算符 -在几乎所有语言中，三元运算符都是从右向左求值的，但在PHP中则相反。 建议完全废除左联想，如果有必要使用多个三元运算符，则用括号显式指示该顺序。 
     1 ? 2 : 3 ? 4 : 5; // deprecated (1 ? 2 : 3) ? 4 : 5; // ok 1 ? 2 : (3 ? 4 : 5); // ok
