Diumumkan Keterampilan Visi Windows (Pratinjau)

Baru-baru ini, versi pratinjau Keterampilan Visi Windows diumumkan, yang merupakan bagian dari paket NuGet paket yang membuatnya lebih mudah bagi pengembang aplikasi untuk menyelesaikan berbagai masalah penglihatan komputer dengan satu set API sederhana.


Gambar 1 - Dari kiri ke kanan Anda melihat aksi: Detektor Objek, Detektor Kerangka, dan Pengenal Emosi.

Dengan Keterampilan Windows Vision:

  • Pengembang aplikasi dapat menggunakan API WinRT yang out-of-the-box untuk menambah keterampilan penglihatan di luar rak, seperti deteksi objek, deteksi kerangka, dan analisis sentimen wajah, ke aplikasi Windows mereka (.NET, Win32, dan UWP).
  • Pengembang visi komputer dapat menggunakan kerangka kerja akselerasi perangkat keras pada perangkat Windows, mengemas solusinya sebagai Keterampilan Visi dan tidak mengkhawatirkan API tingkat rendah.



Semua Windows Vision Skill mewarisi kelas dasar dan antarmuka dari Microsoft.AI.Skills.SkillInterfacePreview . Kerangka kerja terbuka ini dapat dengan mudah diperluas untuk bekerja dengan kerangka pembelajaran pustaka dan perpustakaan yang ada seperti OpenCV.

Keterampilan Windows Vision melengkapi dukungan Windows yang ada untuk mengeluarkan model ONNX menggunakan WinML untuk keluaran lokal. Kerangka kerja ini memungkinkan Anda untuk membuat aplikasi cerdas menggunakan kemampuan optimasi platform.
