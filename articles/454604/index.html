Generando una aplicación React con un backend GraphQL en minutos

Hemos agregado una nueva función experimental en la CLI 8base (interfaz de línea de comandos), que genera una aplicación React lista para usar a partir de su modelo de datos. Esto es muy conveniente para comenzar con GraphQL y React, así como para crear rápidamente una interfaz de usuario de la aplicación. Parece un andamio en Ruby-on-Rails.

La aplicación creada se basa en la aplicación Create React , utiliza el cliente Apollo para consultas GraphQL, incluye autenticación basada en Auth0 , así como la capacidad de descargar archivos.

Como funciona


Debe seguir algunos pasos simples:
  1. Defina su modelo de datos en 8base ,

  2. Instale la CLI de 8base y use npm y ejecute el inicio de sesión de 8base ,
  3. Ejecute el comando 8base generate app $ {YOU_APP_NAME} para generar la plantilla de la aplicación,
  4. Ejecute 8base generate scaffold $ {TABLE_NAME} para generar la IU CRUD de la tabla de datos seleccionada (puede repetir este paso para cada tabla que desee agregar),
  5. Finalmente, ejecute yarn install && yarn start para iniciar la aplicación.
