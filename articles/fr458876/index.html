Vendredi Mini CTF

Salutations à tous les habrachitatel. J'aime vraiment étirer mon cerveau avec différentes tâches - il peut s'agir de problèmes mathématiques, d'énigmes intéressantes liées à la programmation ou de certaines tâches ctf.

Pour ceux qui veulent diversifier les puzzles de vendredi, je veux proposer mon mini-ctf, dans lequel il n'y a que trois tâches.

XT_CTF_01

XT_TASK_101

Lien vers la tâche: xt_task_101.zip
Format d' indicateur : XTGFLAG {...}
Drapeau SHA1: 6980D76C3C5246A101820A77031CA0EBE84A4A0E

XT_TASK_102

Lien vers la tâche: xt_task_102.zip
Format de l' indicateur : XT {...}
Drapeau SHA1: 16448BF22750DE7E575A5A86E739FA9018ACFE52

XT_QUEST_103

Lien vers la tâche: xt_quest_103.zip
Nombre de niveaux: 15 + bonus

Chaque fichier est une tâche distincte (enfin, ou dans le cas de 3 tâches - une quête), ils ne sont pas interconnectés. Le but des deux premiers est d'obtenir un drapeau dont le format est spécifié. Comme vérification supplémentaire, j'ai indiqué des hachages de drapeau afin que vous puissiez facilement vérifier si vous avez résolu le problème correctement ou non. La quête se compose de 15,5 tâches consécutives, où une nouvelle s'ouvre après avoir résolu la précédente, la réponse à la tâche en cours est le mot de passe de l'archive avec la suivante, donc un hachage séparé n'est pas requis pour la vérification.

Dans les tâches, il n'y a pas de fichiers exécutables, des dépendances sur certains services externes. Vous pouvez utiliser tout ce que vous voulez - Google, les services en ligne, vos langages de programmation préférés.

J'évalue la complexité des tâches comme moyen pour les tâches et facile pour les quêtes. Ce serait formidable si les commentateurs s'abstiennent de demander des commentaires (ou du moins les cachent sous un spoiler) afin de ne pas gâcher le processus de décision pour ceux qui décideront plus tard.

Une page sur github , si quelqu'un est plus facile d'y enregistrer un signet.
