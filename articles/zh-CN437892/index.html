3D扫描仪概述闪亮的3D FreeScan X7 / X7 +



Shining 3D的 FreeScan X7工业3D激光扫描仪是一种便携式设备，可以用一只手轻松握住。 扫描仪的重量不超过一公斤，这使您可以在有限的空间内以及三脚架和安装架无法接近的地方使用它。

技术指标


  • 测量精度：0.03毫米
  • 体积精度：0.02毫米+ 0.06毫米/米
  • DigiMetric的体积精度：0.020毫米+ 0.025毫米/米
  • 分辨率：0.05毫米
  • 扫描速度：480,000次扫描/秒
  • 单次扫描区域：300×275 mm
  • 扫描范围：100-8000毫米
  • 工作距离：300毫米
  • 扫描深度：250毫米
  • 支持的额外测量程序：DigiMetric
  • 光源：十四波段激光
  • 激光类别：II类（人眼安全）
  • 软体：3DScan
  • 输出文件格式：STL，ASC，DGM，OBJ等，并与测量程序兼容
  • 支持的操作系统：Win10 64bit
  • 工作温度：-10-40°C
  • 数据传输：USB 3.0
  • 重量：0.95公斤
  • 尺寸：130 x 90 x 310毫米

内容描述


FreeScan X7能够在广泛的行业中运行，包括逆向工程和3D质量控制。 该设备使您可以扫描几乎任何大小，纹理和颜色的对象。



FreeScan X7 + 3D扫描仪的无线版本通过增加设备的移动性进一步简化了对象扫描。 扫描仪配有便携式AirMaster模块，该模块配备有处理器，用于处理数据并通过Wi-Fi以至少600 Mbps的速度传输数据。



接收到的数据可以在智能手机或平板电脑上实时显示，这使使用3D扫描仪更加实用。 新奇产品可以离线工作长达3个小时，等待时间约为10个小时。 FreeScan X7 +允许使用可更换电池



FreeScan X7和X7 +型号均支持结合光学和惯性技术的原始HOI混合扫描系统。



制造商声称，与上一代模型相比，这可以显着提高构建点云的准确性和可靠性，并使扫描速度提高2倍以上。



您可以在室内和室外或其他困难条件下扫描物体。 3D扫描仪使用人眼安全的14行激光阵列作为光源。



扫描精度为30微米，符合德国标准VDI / VDE2634。扫描分辨率为100微米，速度-每秒480,000点。 该设备与DigiMetric摄影测量系统兼容。



3D扫描仪由位于设备背面的四个按钮控制。

闪亮的3D FreeScan X7和X7 +带有保护套。 该套件包括连接电缆，电源，校准面板，一组标签和外壳。



Shining 3D的两个版本的扫描仪都可以与用于3D计量的通用软件一起使用。 支持的软件包括Geomagic Design X和Control X，PointShape Inspector，Verisurf，Rhinoceros和Mesh2Surface。
