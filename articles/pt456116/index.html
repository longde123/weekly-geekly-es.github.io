Como criar um belo esquema de banco de dados no yEd Graph Editor

Quero mostrar como, com a ajuda do agora popular aplicativo yEd, desenvolvido para criar vários diagramas e esquemas, você pode criar um esquema de banco de dados completo (com toda a notação).

Portanto, para iniciantes, temos um contêiner padrão com elementos para a construção de um ERD (diagrama de relacionamento de entidades). Suas designações são mostradas na figura.



Uma pequena digressão, se você precisar inserir apenas texto no diagrama, ou seja, sem nenhum bloco, as etapas simples a seguir de um comentário de jaiprakash resolvem rapidamente esse problema. "Selecione qualquer link -> adicione rótulo -> insira texto -> converta em nó".

Portanto, colocamos o problema condicional de construir o esquema a seguir.



Crie uma estrutura de elementos básicos.



Para adicionar a notação de chaves primárias e estrangeiras, você precisa inventar um pouco. Selecionando o elemento desejado, adicione um novo rótulo, que chamaremos de KEY. Porque Ele estará localizado sob a unidade principal e selecione-a usando o Ctr preso.





Defina seu posicionamento (posicionamento) selecionando Interno: superior esquerdo (interno, superior esquerdo), cor conforme desejado, Distância (distância das bordas) como zero, Inserções (inserção em pixels) como 20 2 0 0 para que fique oposto ao campo de atributo com uma pequena distância, a Família de fontes (fonte) define o Segoe UI Symbol, porque Ele suporta com precisão os caracteres necessários.



Os símbolos para cópia podem ser encontrados, por exemplo, na tabela de símbolos do Windows.





Faça o mesmo com as setas.



Para combinar várias tabelas em um bloco, você pode usar o Agrupamento.
