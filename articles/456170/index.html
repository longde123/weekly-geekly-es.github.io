Cómo crear una aplicación de finanzas: 5 API para ayudar al desarrollador



Hola a todos! Continúo mi investigación de la esfera de las finanzas desde el punto de vista de la tecnología. En el artículo anterior, estudiamos los terminales comerciales existentes y sus características, y hoy propongo hablar, de hecho, sobre el desarrollo de aplicaciones financieras.

Para el nuevo tema, recogí cinco API de varios servicios financieros que le permiten recibir una amplia variedad de datos y comerciar directamente en el intercambio utilizando robots escritos.

Yahoo finanzas


Quizás uno de los servicios financieros más famosos del mundo. Con la API gratuita, los desarrolladores pueden usar los datos del servicio en cotizaciones de bolsa, cambios de precios y movimientos del mercado. La información está disponible en una variedad de acciones, así como en otros instrumentos, como los pares de divisas.

Puede conectarse de manera bastante conveniente a la API utilizando el servicio Rapid API.



SMARTcom


API para crear una aplicación para operar en bolsas rusas. Los desarrolladores pueden usar la API como una capa entre el sistema de comercio de intercambio y su sistema de comercio prefabricado (creado independientemente o comprado), crear su propio robot de comercio o incluso un terminal de comercio completo con una interfaz "personalizada".

El acceso a la API es gratuito, solo necesita crear una cuenta en el sistema de la compañía de corretaje que lo desarrolló. Existe la oportunidad de depurar el trabajo de su software utilizando el acceso de prueba en el comercio virtual.



Captura de pantalla: documentación de SMARTcom

Sincronización por libro de pagos


Un servicio útil para desarrolladores de aplicaciones financieras para mercados extranjeros. Le permite recopilar en un solo lugar datos de bancos, compañías financieras y de servicios públicos, agencias gubernamentales de diferentes países. Este servicio es de pago, el acceso cuesta desde $ 50. La lista de servicios disponibles para la integración se puede encontrar aquí .



API de Currencylayer


Un servicio útil para obtener información sobre los tipos de cambio y el valor de 168 monedas mundiales. Los datos se recopilan de muchas fuentes, como sitios de bancos, servicios de información comercial. Los desarrolladores prometen compatibilidad con aplicaciones de cualquier tipo y cualquier lenguaje de programación.

Hay una versión limitada gratuita de la API (250 solicitudes por mes), una más adecuada comienza en $ 9.99 (10,000 solicitudes por mes). Sus soluciones basadas en la API de Currencylayer son desarrolladas por compañías como Lyft, Blizzard, Intel.



Finanzas fáciles


El conocido servicio de contabilidad para finanzas personales en Runet proporciona una API para desarrolladores externos. Como se indica en el sitio web del proyecto, las aplicaciones de terceros para redes sociales, aplicaciones móviles y complementos para navegadores son las más populares entre los usuarios del recurso.

El equipo de servicio prueba las aplicaciones, las aprobadas entran en un mercado especial y se distribuyen por suscripción. Según las estadísticas de la administración de EasyFinance, los desarrolladores de aplicaciones pueden ganar entre 15 y 50 rublos por usuario al mes. Estos ingresos consisten en ingresos por suscribirse a las tarifas del servicio en sí, pagos de integración de socios de plataforma y publicidad.
