* Intensimeter - dosimeter dari kepadatan fluks energi partikel pengion.

Dosimeter dibangun di atas empat penghitung Geiger-Muller (selanjutnya disebut sebagai "tabung" atau penghitung tidak sepenuhnya benar) - tabung SBM-20 yang populer dan terjangkau. Saat membeli, Anda harus memperhatikan tanggal pembuatan.

Tabung sensitif terhadap y dan terbatas pada β , dan tidak sensitif terhadap radiasi α .

Karakteristik SBM-20


SBM-20 dibuat dalam bentuk tabung logam bergelombang berdinding tipis yang darinya udara dipompa keluar, dan gas inert ditambahkan alih-alih pada tekanan ringan, dengan penambahan pengotor (Ne + Br 2+ Ar). Sebuah kawat tipis direntangkan di sepanjang sumbu tabung, dan sebuah silinder logam terletak di sampingnya. Baik tabung dan kawat adalah elektroda: tabung adalah katoda, dan kawat adalah anoda. Minus dari sumber tegangan konstan terhubung ke katoda, dan melalui resistansi konstan yang sangat besar - plus dari sumber tegangan konstan ke anoda. Ketika partikel bermuatan memasuki konter, sejumlah gas terionisasi, dan di bawah pengaruh tegangan antara katoda dan anoda, ion dan elektron mulai bergerak - arus jangka pendek muncul dalam tabung. Tegangan di anoda tabung turun sebentar - kita mendapatkan pulsa terbalik. SBM-20 memiliki kontak untuk koneksi basis. Jangan pernah menyolder mereka.

Persyaratan utama saya untuk sirkuit ini adalah:

  • dalam penerapan tegangan yang digunakan di sirkuit dengan mikrokontroler - 5 volt atau lebih rendah;

  • skalabilitas dan kemungkinan menggunakan penghitung Geiger-Muller lainnya dengan mengatur tegangan setidaknya 200-460 volt;
  • terdiri dari blok fungsional terpisah yang dihubungkan secara seri;
  • Desainnya dapat dengan mudah diperbaiki.

Koneksi ke Arduino


Dalam waktu dekat, tujuan kami adalah menyelesaikan pembuatan alat ukur yang nyaman dengan tampilan, dengan perhitungan ulang dosis paparan radiasi selama pengamatan jangka panjang, dengan tampilan grafis atau kontrol tingkat intensitas radiasi yang telah ditetapkan dan sinyal alarm ketika level terlampaui. Sementara itu, kami akan berkonsentrasi pada tampilan grafis sederhana. Sensitivitas tinggi dan penyaringan noise yang lebih tinggi akan memungkinkan kami untuk bereksperimen dengan sumber radiasi radioaktif yang lebih lemah.

Maka hubungkan output perangkat ke Arduino Uno pada pin D2. Pulsa tunggal dijumlahkan dalam suatu variabel melalui pemrosesan terputus, dan jumlah pulsa per menit ditampilkan secara grafis. Untuk memulai percobaan program semacam itu sudah cukup bagi kami. Bahkan satu tabung dapat mengukur cukup akurat, tetapi akan membutuhkan banyak waktu untuk melakukan pengukuran. Puluhan menit perlu dihabiskan untuk siklus dan satu pengukuran beberapa siklus bisa memakan waktu beberapa jam. Cara lain untuk melakukan hal yang sama dapat kita amati dalam perangkat produksi serial - ini dilakukan dengan meningkatkan jumlah penghitung Geiger-Muller yang terhubung secara paralel, yang akan meningkatkan jumlah partikel yang ditangkap. Cara menghubungkan beberapa tabung menunjukkan diagram ini:


Koneksi paralel beberapa tabung

//   / 
int pocet;                            //    
unsigned long time;                   //    

void setup() {
  pinMode(2, INPUT);                  // pin 2     
  attachInterrupt(0, nacti, RISING);  //  
  Serial.begin(9600);                 //       
  Serial.println(" ");                //    
}

void nacti() {
  pocet = pocet++;                    //  int0
}

void loop() {
  pocet = 0;                          //  
  time = millis() + 60000;            //   
  while (time > millis()) {}          //   1 
  if (pocet < 10) Serial.print(" ");  //    
  if (pocet < 100) Serial.print(" ");
  if (pocet < 1000) Serial.print(" ");
  Serial.print(pocet);                //   /
  Serial.print(" ");
  for (int i = 0; i < pocet; i++) {   //  
    Serial.print("#");
  }
  Serial.println(" ");                //  
}
