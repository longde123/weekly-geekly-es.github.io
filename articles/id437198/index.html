Global Game Jam 2019 (pengumuman)

Setiap tahun, pada akhir pekan terakhir bulan Januari, Jam Game Global tahunan berlangsung.
Tahun ini acara akan diadakan pada 26 dan 27 Januari dan itu akan menjadi hari terpanas bulan ini!
Global Game Jam adalah acara yang didedikasikan untuk membuat game (jika seseorang tiba-tiba tidak tahu apa itu game jam). Acara ini offline.

Daftar lengkap situs dapat ditemukan di sini.

Di Rusia, ada situs di kota-kota berikut:

  • Moskow, IT Hub College
  • St. Petersburg, Dial di Rumah Kebudayaan
  • Krasnodar, rekan kerja CO-Place
  • Innopolis, Universitas Innopolis
  • Chelyabinsk, SUSU
  • Yakutsk, IT Park Yakutsk
  • Chita Boiling Point

Acara akan dimulai di masing-masing dari mereka pada tanggal 25 jam 5 sore waktu setempat. Semua proyek dilisensikan di bawah CC-NC-SA (Creative Common NonCommercial ShareAlike). Mereka harus mencakup: file yang dapat dieksekusi, kode sumber dan sumber daya, serta semua file yang diperlukan untuk kompilasi. Acara ini tidak membatasi Anda, Anda dapat menggunakan mesin apa pun. Sumber daya yang disiapkan diizinkan. Game bisa mendapatkan poin bonus untuk menggunakan penyelam, daftar lengkap dengan deskripsi di sini , yang dibagi ke dalam kategori:

  • Dari sponsor: Kekuatan Komunitas, Maafkan dan Ukrepis, Selalu ada tempat untuk yang lain, Gunakan sumber, Luke, Party Creator, Menguji desain teka-teki, Kemandirian bahasa
  • Aksesibilitas: Biarkan saya menunjukkan kepada Anda bagaimana ini dilakukan!, Tuhan, Lebih mudah, Pergi sendiri itu berbahaya, ambil!
  • Seni: Thomas tidak sendirian, Dalam tinta, Seni berbagi
  • Audio: Mind over Matter, 4'33 '', Tingkatkan dengan Lagu
  • Kode: Tanpa Aset, Media Campuran, Di Balik Kap
  • Desain: Super Secret Stash, Ini tempat asal kami, kan?, Bolter stuck!, Sticky finger, Di laman web, Panduan yang saya cari, Ephemeral
  • Narration: Sementara itu, Passion for travelling, Russell's Kettle, 20-XIX, Ancient People
  • Meta: Domain Publik, Daur Ulang!, Selamat Ulang Tahun, Kerangka Identitas ADL, Lindungi Lautan

Saat mengirim game, Anda dapat menentukan hingga 4 penyelam. Semoga beruntung untuk semuanya!

PS: Saya sangat menyesal, saya mengacaukan pendaftaran, kursi kosong di situs Chelyabinsk (SUSU) sudah selesai, jadi saya tidak akan pergi ke acara ini = (
