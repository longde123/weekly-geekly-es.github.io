Blazor 0.9.0 lanzado

¡Blazor 0.9.0 ya está disponible! Esta versión agrega mejoras del componente Blazor Razor a .NET Core 3.0 Preview 3.


Las nuevas mejoras de Razor Component ahora están disponibles para aplicaciones Blazor:


  • Manejo mejorado de eventos
  • Formas y Validación


¡Esperamos ver sus comentarios!

Consulte el anuncio de ASP.NET Core 3.0 Preview 3 para conocer los detalles de estas mejoras. Consulte también la documentación de la versión Blazor 0.9.0 para obtener más detalles.


Nota: Las plantillas de Blazor no se han actualizado para usar la nueva extensión de archivo .razor para los componentes de Razor en esta versión. Esta actualización se realizará en una versión futura.


Instalar Blazor 0.9.0


Para comenzar con Blazor 0.9.0, instale lo siguiente:


  1. .NET Core 3.0 Preview 3 SDK (3.0.100-preview3-010431)
  2. Visual Studio 2019 (Vista previa 4 o posterior) con ASP.NET.
  3. La última extensión de Blazor de Visual Studio Marketplace.

  4. Plantillas Blazor en la línea de comando: 


    dotnet new -i Microsoft.AspNetCore.Blazor.Templates::0.9.0-preview3-19154-02

Las instrucciones de inicio, documentos y tutoriales para Blazor se pueden encontrar en blazor.net .


Actualizar a Blazor 0.9.0


Para actualizar las aplicaciones Blazor existentes a Blazor 0.9.0, primero asegúrese de haber hecho todo lo anterior.


Para actualizar su proyecto Blazor 0.8.0 a 0.9.0:


  • Actualice los paquetes de Blazor y los enlaces de la herramienta .NET CLI a la versión 0.9.0-preview3-19154-02.
  • Actualice los paquetes restantes de Microsoft.AspNetCore. * Hasta la versión 3.0.0-preview3-19153-02.
  • Elimine cualquier uso de JSRuntime.Current y use la inyección de dependencia en su lugar para obtener la instancia actual de IJSRuntime y transferirla a donde sea necesario.
