13 game baru diperkenalkan sebagai bagian dari program ID @ Xbox

Satu minggu sebelum pembukaan konferensi GDC 2019, tim Xbox memperkenalkan lini baru permainan dari pengembang independen, yang akan tersedia di gerai Microsoft sebagai bagian dari acara mendatang. Pemain dari seluruh dunia akan menjadi yang pertama untuk berkenalan dengan 30 proyek berbeda untuk Xbox One dan Windows 10, 13 di antaranya akan disajikan untuk pertama kalinya sebagai bagian dari program ID @ Xbox.




ID program Xbox memungkinkan pengembang game apa pun untuk mengungkapkan potensi kreatif mereka dengan menerbitkan proyek untuk Xbox One dan Windows 10 di Xbox Live. Studio dilengkapi dengan alat dan dukungan yang diperlukan untuk memaksimalkan kesuksesan mereka.

Artikel ini ada di situs berita kami .
