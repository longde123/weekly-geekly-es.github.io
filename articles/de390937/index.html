Windows 98 - jetzt auch im Browser



Enthusiasten führen alte Spiele, Programme und Betriebssysteme im Browser aus. Gleichzeitig ist die Arbeit, die zur Umsetzung einiger Ideen erforderlich ist, sehr mühsam und zeitaufwändig, ganz zu schweigen davon, wie viel Zeit in einigen Fällen benötigt wird. Entwickler brauchten vier Jahre, um Windows 98 in einem Browser zu starten. Aber es hat sich gelohnt - jetzt wird eine voll funktionsfähige Version dieses Betriebssystems im Browser gestartet.

Die Arbeit wurde mit Javascript durchgeführt, der Quellcode enthält mehrere tausend Codezeilen. Jetzt können Benutzer, die sich für alte Versionen des Betriebssystems interessieren, gerne mit Programmen der alten Schule, einschließlich Spielen, arbeiten.



Es ist möglich, von einer CD oder Diskette zu booten (Images sind erforderlich). Dies bedeutet, dass Sie in Tribes oder Half Life 1 im Browser spielen können! Die Anzahl der Spiele, die tatsächlich im Browser ausgeführt werden, ist zwar begrenzt. Der gleiche Achterbahn-Tycoon konnte nicht gestartet werden (aber Sie können es versuchen).



Der Browser funktioniert auch, wenn auch mit bekannten Kompatibilitätsproblemen. Natürlich kann es nicht ohne Probleme gehen, aber es ist erwähnenswert, dass dies ein Emulator ist. Trotzdem viel Arbeit, daher empfehle ich, das System in der Praxis auszuprobieren.
