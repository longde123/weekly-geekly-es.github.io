Windows akan memesan 7 GB untuk pembaruan sistem untuk menghindari kehabisan ruang hard disk



Microsoft telah menemukan jalan keluar ketika Pembaruan Windows keliru dalam memperkirakan ruang kosong pada hard drive komputer dan sebagai akibatnya kesalahan terjadi selama instalasi sistem operasi. Sekarang Windows akan memesan 7 gigabytes untuk kebutuhan pembaruan, sehingga masalah kekurangannya hilang begitu saja.

Dimulai dengan Windows 1803, Microsoft mulai memperingatkan pengguna bahwa jika mereka tidak membebaskan sejumlah ruang file, kesalahan dapat terjadi selama pembaruan OS. Alasan - Pembaruan Windows tidak memeriksa sistem untuk ruang kosong yang cukup pada hard disk sebelum inisialisasi. Oleh karena itu, solusi yang dapat digunakan oleh pemilik komputer dengan OS Windows adalah membersihkan disk dari file sementara. Tetapi, tentu saja, tidak semua pengguna dapat atau ingin melacak ketersediaan ruang kosong pada HDD.

Untuk Windows 10 32-bit, diperlukan 16 GB, untuk 64-bit - 32 GB. Ini adalah ruang file yang diperlukan untuk menginstal OS itu sendiri. Upgrade membutuhkan sedikit lebih sedikit. Dimulai dengan rilis Windows 19H1 / 1903, OS akan mulai secara mandiri mencadangkan ruang pada hard disk untuk kebutuhan pembaruan. Minimal 7 GB akan dipesan, mungkin sistem akan mengalokasikan lebih banyak, itu semua tergantung pada karakteristik komputer itu sendiri. Dan ini adalah "penawaran yang tidak dapat Anda tolak" - tidak ada kemungkinan memblokir fungsi reservasi. Benar, Anda dapat mengurangi jumlah HDD yang dipesan oleh sistem.



“Bagian tertentu dari hard drive akan dialokasikan untuk pembaruan, aplikasi, file sementara dan cache sistem. Tujuan kami adalah untuk meningkatkan penggunaan harian PC Anda, memastikan berfungsinya fungsi-fungsi penting dari sistem operasi. Jika tidak ada ruang disk, pengoperasian beberapa layanan dan aplikasi menjadi tidak dapat diandalkan. Windows dan program mungkin tidak berfungsi seperti yang diharapkan jika tidak ada ruang kosong. Dengan memesan kursi, kami meminimalkan risiko kegagalan. Ruang disk akan dialokasikan secara otomatis ketika versi baru diinstal, ”kata Microsoft dalam sebuah pernyataan.

Pada dasarnya, tidak ada yang mengerikan dalam tindakan Microsoft. Pengguna biasa dengan PC yang cukup kuat, tempat media penyimpanan yang luas dipasang, tidak akan melihat apa pun. Tetapi Windows sering diinstal pada sistem entry-level dengan ruang file minimum. Dan di sini mungkin ada masalah. Mungkin langkah ini oleh perusahaan akan memaksa produsen peralatan untuk menempatkan disk dengan kapasitas yang lebih tinggi di perangkat mereka, yang, tentu saja, akan mempengaruhi harga akhir mereka.

Sebelumnya, korporasi berbicara tentang hal baru lainnya - fitur Windows 10, yang memungkinkan Anda untuk menjalankan file yang dapat dieksekusi dengan aman. Kita berbicara tentang Windows Sandbox, "sandbox", yang akan memberi Anda kesempatan untuk memeriksa aplikasi atau dokumen apa pun sebelum membukanya. Untuk kotak pasir, virtualisasi digunakan. Ketika Anda mencoba untuk memulai file yang tidak dikenal, Windows 10 secara otomatis memulai mesin virtual "ringan", di lingkungan di mana aplikasi dimulai atau dokumen terbuka.



Mesin virtual adalah perusahaan berpemilik, ini adalah Hyper-V. Itu menciptakan kernel baru yang bekerja secara mandiri, terlepas dari inti. Segera setelah file yang diperiksa berhenti berfungsi, lingkungan virtual dihancurkan. Semua ini dimaksudkan terutama untuk pengguna korporat. Fitur ini akan muncul di Windows 10 Pro dan Windows 10 Enterprise.

Hal baru lainnya adalah sistem pemecahan masalah otomatis. Dia bekerja di latar belakang, secara teratur melakukan diagnosa peralatan dan perangkat lunak. Salah satu contohnya adalah lokalisasi masalah saat melepaskan PC dari Internet. Menurut perusahaan, fungsi (yang tidak dapat dinonaktifkan) dapat mengembalikan pengaturan default layanan kritis kritis, serta mengkonfigurasi sistem operasi sesuai dengan konfigurasi perangkat keras dan membuat perubahan lain yang diperlukan untuk operasi normal Windows.

Bagian dari OS yang diperbarui adalah log pendeteksi malware dan fungsi Tamper Protection - memberikan perlindungan tambahan terhadap gangguan eksternal dengan pengoperasian fungsi keamanan sistem yang kritis.
