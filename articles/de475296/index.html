Wir laden Sie am 21. November zu DINS Java EVENING ein: Wir sehen uns AWS Lambda in Aktion an und machen uns mit dem Akka-Framework vertraut

Wir werden uns freuen, alle, denen Java-Entwicklung nicht gleichgültig ist, beim nächsten offenen DINS Java EVENING zu sehen, das in unserem Büro bei 19 Staro-Petergofsky stattfinden wird.

Heute Abend wird Alexander Gruzdev von DINS die Entwicklung von serverlosen Anwendungen auf der Basis von AWS Lambda demonstrieren. Igor Shalaru von Yandex wird uns das Akka-Framework und die Vorteile seiner Verwendung vorstellen.

Unter der Rubrik - Weitere Informationen zu Berichten und Referenten finden Sie einen Link zur Anmeldung für die Teilnahme am Meeting sowie Materialien vom letzten Meeting.



Berichte


AWS Lambda in Aktion (Alexander Gruzdev, DINS)

Alexander wird über einen Entwicklungsansatz sprechen, der für diejenigen interessant sein wird, die es aus irgendeinem Grund leid sind, einen neuen Microservice zu schreiben, und für diejenigen, die die EC2-Ausfallzeit nicht bezahlen möchten. Anhand konkreter Beispiele analysieren wir den gesamten Prozess - vom Schreiben eines Lambda über das Testen bis hin zur Bereitstellung und zum lokalen Debuggen. Der Bericht richtet sich an eine Zielgruppe, die bereits etwas über AWS Lambda oder über serverlose Ansätze im Allgemeinen gehört hat.

Alexander widmet sich seit mehr als 5 Jahren verstärkt der Java-Entwicklung von Unternehmensanwendungen in einer Vielzahl von Anwendungsbereichen, von Life Science bis Telekommunikation. Er hält an der Idee fest, dass der Entwickler immer den Markt für fertige Werkzeuge überwachen sollte, damit er in den meisten Fällen keine eigenen „Fahrräder“ schreibt.

"Akka als Kern hochbelasteter Systeme" (Igor Shalaru, Yandex)

Akka ist seit langem im Arsenal der Java-Entwickler. Dies ist ein leistungsstarkes und praktisches Tool zum Entwickeln von Anwendungen. Im Rahmen des Berichts werden wir analysieren, was das Akteurmodell ist, was die vorgefertigten Module für Akka sind. Lassen Sie uns anhand eines Beispiels sehen, wie wir mit der Entwicklung von Akka beginnen und welchen Vorteil dies für die Zukunft hat. Der Bericht ist für Java-Entwickler aller Niveaus interessant, für diejenigen, die bereits mit Akka vertraut sind oder sich nur kennenlernen möchten.

Die letzten 7 Jahre hat Igor in Java programmiert. In dieser Zeit wechselte er von einem einfachen Entwickler zu Tech. Lida. Er verfügt über umfangreiche Erfahrung in Projekten mit hohen Anforderungen an Belastbarkeit und Fehlertoleranz. Er schreibt gerne Anwendungen, die schnell Gigabyte / Terabyte an Daten verarbeiten.
