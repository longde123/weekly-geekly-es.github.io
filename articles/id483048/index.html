Terminal Fun: 10 lelucon klasik

Setengah dari liburan Tahun Baru yang panjang telah berakhir dan sekarang adalah waktu untuk mengingat kembali lelucon berjanggut dari admin Unix.



Kerang teks dalam sistem operasi mirip UNIX tidak hanya cocok untuk mengeksekusi perintah internal, menjalankan berbagai utilitas, dan menulis skrip. Hasil beberapa program dapat menghibur tamu yang berkeliaran dalam cahaya. Dewan editorial blog RUVDS mengucapkan selamat kepada pembaca pada Natal dan menawarkan untuk mengingat kembali lelucon konsol klasik yang menyenangkan banyak generasi administrator sistem.

1. Bahkan make sepele tidak bisa hanya mengkompilasi program dari sumber. Ada opsi lain untuk menggunakan utilitas ini, tetapi pertama-tama Anda harus menginstal paket jika tidak ada di sistem Anda. Di Ubuntu, ini dilakukan dengan satu perintah: 

apt-get install make

Maka semuanya sederhana:



2. Jangan lupa tentang utilitas favorit kami who . Dengan bantuannya, Anda dapat menggores FAQ admin:



3. Klasik abadi lainnya adalah utilitas cowsay . Itu tidak termasuk dalam konfigurasi default, jadi Anda perlu menginstal: 

 apt-get install cowsay



4. Semua pengguna sistem UNIX tahu tentang utilitas ls , tetapi program sl (stream lokomotif) kurang dikenal. 

 apt-get install sl

Jika Anda menginstal dan menjalankannya, kereta akan melewati konsol (hampir film Lumiere brothers di ASCII chart).





5. Menggunakan telnet dangkal dalam grafik ASCII, Anda dapat menonton bagian keempat Star Wars yang digambar oleh para penggemar. 

 telnet towel.blinkenlights.nl



6. Jumlah berikutnya dari program ini adalah kekayaan utilitas yang terkenal, mengeluarkan kutipan dengan atau tanpa. Sebagai contoh, ia dapat dijalankan dari skrip inisialisasi shell untuk mendapatkan prediksi kecil ketika pengguna masuk ke sistem - sesuatu seperti horoskop konsol (dalam beberapa sistem ini dilakukan secara default). Terkadang kutipan keberuntungan masuk dalam sepuluh besar, tetapi pertama-tama Anda perlu menginstal paket: 

 apt-get install fortune.



7. Perintah built-in yes tidak melakukan sesuatu yang spesial, ia mengulangi hal yang sama dalam loop yang tak terbatas sampai pengguna bosan dengan itu: 

 yes Merry Christmas



8. Jika Anda ingin menulis beberapa frase dengan indah dalam ASCII-graphics, Anda tidak perlu menghabiskan banyak waktu untuk kerja keras. Pembuat utilitas toilet sudah mengurus semuanya: 

 apt-get install toilet toilet Merry Christmas



Dengan bantuan parameter tambahan, prasasti dapat dibuat lebih menarik, misalnya seperti ini: 

 toilet -f mono12 -F metal Merry Christmas



9. Penggemar matriks harus meluncurkan animasi hijau favorit mereka di konsol teks: 

 apt-get install cmatrix cmatrix



10. Masalah terakhir adalah program demo cantik lainnya: 

 apt-get install bb bb



Dalam catatan singkat, kami hanya berbicara tentang 10 kesenangan konsol klasik. Bahkan, ada banyak lagi, dan jika Anda melampirkan sedikit imajinasi dan menulis naskah, antarmuka teks yang membosankan akan bersinar seperti pohon Natal.
