Bagaimana cara kerja emulator 3D game NES 8-bit?



Kembali pada bulan Maret, salah satu pengamat sumber daya Kotaku menemukan versi beta berbasis browser dari emulator NES . Emulator ini dapat mengubah game 8-bit “datar” menjadi tiga dimensi. Emulator ini dilaporkan ditulis dalam JavaScript. Ini bekerja di browser Firefox. Program ini dapat menambahkan bayangan dan permukaan horizontal ke yang vertikal. Emulator "tiga dimensi" bekerja dengan gambar game (ROM) - seperti halnya emulator biasa.

Sekarang versi final program telah dirilis sebagai aplikasi untuk Windows. Pengembang telah mempostingversi aplikasi gratis dan berbayar ($ 4,99). Mungkin, pencipta perangkat lunak ini mengandalkan penggemar game NES klasik, siap untuk berpisah dengan $ 30 sebagai ganti sensasi baru ketika memainkan Duck Tales, Super Mario Bros. atau Mega Man. Beberapa hari yang lalu, pembuat emulator mengklarifikasi prinsip operasinya dan menunjukkan bagaimana memungkinkan untuk mengkonfigurasi sejumlah elemen gim jika algoritma emulator tidak mengatasi gim mana pun dengan sendirinya.



Komponen utama dari emulator baru adalah algoritma yang secara otomatis menentukan bentuk elemen grafis dari game yang diunduh. Program ini mencoba untuk mengelompokkan masing-masing sprite karakter permainan dan lanskap latar belakang menjadi objek 3D dasar (terutama kubus dan silinder). Selain itu, algoritma ini menganalisis bagaimana elemen tiga dimensi harus ditampilkan di layar, dengan mempertimbangkan kedalaman dan ukuran masing-masing objek dan tata letak keseluruhan elemen di layar.

Untuk banyak game lama, termasuk Super Mario Bros. dan Tetris, proses ini hampir sempurna, tidak diperlukan pengaturan tambahan. Setiap frame dengan game dapat diputar atau ditingkatkan / dikurangi. Terlihat sangat menarik di Tetris. Dengan beberapa karakter game atau objek latar belakang dalam dua game ini, insiden terjadi ketika garis besarnya kabur atau berubah menjadi sesuatu yang tidak dapat dikenali dengan baik. Tetapi ini jarang terjadi.

Jika gim tidak cocok dengan algoritma, maka konfigurasi tambahan sudah diperlukan di sini. Pengembang emulator telah memberikan kesempatan seperti itu. Benar, Anda harus mengonfigurasi setiap layar gim. Anda perlu mengklik mouse di mana saja di layar, dan mengatur karakteristik objek 3D. Anda dapat mentransfer objek dari latar depan ke belakang, mengubah ukuran dan skalanya, mengubah operasi algoritme itu sendiri (misalnya, menunjukkan perlunya menerjemahkan objek 2D ke dalam kelompok silinder volumetrik, bukan kubus).

Keuntungan dari emulator adalah bahwa penyesuaian satu elemen secara otomatis akan mengarah pada penggantian penampilannya kapan saja dalam permainan. Artinya, jika Anda mengubah garis besar Mario yang sama, ia akan berjalan dalam bentuk saat ini di semua level lainnya. Pengaturan dapat disimpan ke file. File ini kemudian dapat digunakan saat berikutnya Anda mengunduh gambar yang sama dari gim atau memasang jaring untuk pemain lain.



Kelemahannya adalah waktu dan biaya tenaga kerja yang signifikan - namun, bekerja dengan grafik bukan masalah beberapa menit. Mengubah setiap kerangka berikutnya adalah kesenangan yang meragukan jika Anda bukan perfeksionis dengan komitmen pada pekerjaan rutin. Namun dalam beberapa game, Anda tidak dapat melakukannya tanpa pengaturan tambahan. Ini berlaku untuk Super C, serta Chip dan Dale's Rescue Rangers, Castlevania dan beberapa game lainnya. Tanpa pengaturan tambahan, Anda tidak akan dapat memainkan mainan ini.



Hampir semua game NES dapat dikonfigurasikan untuk bermain dengan emulator, kecuali yang sudah banyak (artinya pseudo-3D). Ini adalah Super Dodge Ball dan Teenage Mutant Ninja Turtles 2: The Arcade Game. Anda tidak dapat mengonfigurasinya untuk pemutaran normal dalam emulator 3D. Apa pun pengaturan yang diterapkan, karakter terlihat sangat aneh, sering berubah menjadi bagian dari latar belakang. Masalah serupa muncul ketika mencoba memuat game dengan level isometrik, seperti RC Pro-Am. Juga tidak mungkin memainkannya.



Namun, sejumlah besar game dua dimensi masih kompatibel dengan emulator. Dan jika Anda menganggap bahwa ada banyak game NES, maka seorang gamer bisa mendapatkan berjam-jam game yang menyenangkan di masa lalu, dipelajari di sepanjang dan di seluruh mainan. Saya juga ingin berharap untuk menurunkan harga emulator, atau menerjemahkannya ke dalam model free-to-play. Emulator dapat bekerja dengan PC dan laptop yang cukup lama tanpa masalah.

Kemungkinan
  • Mengubah game dua dimensi menjadi voluminous;
  • Menjelajahi kemungkinan teknologi 3D;
  • Game dengan resolusi apa pun yang mendukung PC atau laptop, dalam mode layar penuh atau jendela;
  • Suara mono atau stereo;
  • Mengubah pengaturan grafis game;
  • Kemampuan untuk menyimpan pengaturan;
  • Kemampuan untuk berbagi pengaturan yang disimpan dengan seluruh komunitas.
