Cmus console player untuk Linux

Hari yang baik

Saat ini saya menggunakan pemain konsol cmus, yang sangat nyaman digunakan. Mengingat hal ini, saya ingin menulis ulasan singkat.



Di workstation baru, saya akhirnya pindah ke Linux. Dalam hal ini, ada kebutuhan untuk menemukan perangkat lunak yang cocok untuk kebutuhan terkait pekerjaan. Meskipun ada cukup banyak pemain antarmuka untuk linux, mereka semua agak bertumpuk dan tidak terlalu indah (pendapat saya mungkin tidak bertepatan dengan pendapat pembaca karena itu adalah IMHO). Kriteria berikut dimasukkan dalam parameter pencarian:

  • Kesederhanaan
  • Kenyamanan
  • Konsumsi RAM rendah
  • Kemampuan untuk memutar dan memindai subfolder
  • Roti terkait melekat pada pemain reguler

Dalam prosesnya, saya menemukan cmus. Pemain ... konsol ... dan mengapa tidak, saya pikir dan menjalankan perintah yang biasa bagi banyak orang: 

sudo apt-get install cmus

Setelah lari pertama, kita melihat yang berikut:



Untuk menambahkan musik ke daftar putar, cukup jalankan perintah. Tekan ":" seperti dalam vi dan tulis: 

 add ~/Music

Sebagai hasil dari perintah ini, Anda akan melihat daftar subfolder dan trek seperti pada tangkapan layar pertama.

Untuk menavigasi program, Anda dapat menggunakan tombol panah, tab (untuk beralih antar area), spasi (untuk memperluas subfolder), menghapus (untuk menghapus trek dari daftar). Ada juga sejumlah besar kunci panas. Untuk mengubah jendela di pemutar, tombol angka digunakan dari 1 menjadi 7. Di jendela ke-7, Anda dapat mengkonfigurasi ulang tombol pintas untuk diri sendiri.

Anda dapat membaca lebih lanjut tentang ini tentu saja dengan menjalankan perintah: 

 man cmus

Dalam mode pemutaran, pemain "makan" sekitar 10 mb RAM, yang saya anggap ukuran yang sangat sederhana.
Situs web resmi mengklaim dukungan untuk beberapa fitur yang lebih menyenangkan (seperti audio scrobling), serta add-on (menghubungkan tombol multimedia, mengunduh musik dari spotify atau youtube).

Untuk meringkas, Anda dapat dengan aman mengatakan bahwa pemain ini sangat nyaman dan fleksibel.

Saya setuju bahwa ini bukan untuk semua orang, tetapi hanya untuk Geeks konsol =)

Atas hal ini saya mengambil cuti saya.
