如何创建财务应用程序：5个可帮助开发人员的API



大家好！ 我将从技术的角度继续对金融领域的研究。 在上一篇文章中，我们研究了现有的交易终端及其功能，今天，我提议讨论实际上是关于金融应用程序的开发。

对于新主题，我选择了五种用于各种金融服务的API，使您既可以接收各种各样的数据，又可以使用书面机器人直接在交易所进行交易。

雅虎财经


也许是世界上最著名的金融服务之一。 使用免费的API，开发人员可以将服务数据用于股票报价，价格变动和市场走势。 可获得有关各种股票以及其他工具（例如货币对）的信息。

您可以使用Rapid API服务很方便地连接到API。



SMARTcom


用于创建在俄罗斯交易所进行交易的应用程序的API。 开发人员可以将API用作交易所交易系统和现成的交易系统（独立创建或购买）之间的层，创建自己的交易机器人，甚至是带有“自定义”界面的整个交易终端。

该API是免费的，您只需要在开发该API的经纪公司的系统中创建一个帐户即可。 有机会在虚拟交易中使用测试访问来调试软件的工作。



屏幕截图：SMARTcom 文档

通过存折同步


对国外市场金融应用程序开发人员有用的服务。 它使您可以在一处收集来自不同国家的银行，金融和公用事业公司，政府机构的数据。 这项服务是有偿的，接入费用从50美元起。 可用于集成的服务列表可在此处找到。



Currencylayer API


一个有用的服务，用于获取有关汇率和168种世界货币的价值的信息。 数据是从许多来源收集的，例如银行站点，商业信息服务。 开发人员保证与任何类型和任何编程语言的应用程序兼容。

有一个免费的有限版本的API（每月250个请求），一个更合适的API起价为9.99美元（每月10,000个请求）。 他们基于Currencylayer API的解决方案是由Lyft，Blizzard，Intel等公司开发的。



轻松理财


Runet中著名的个人财务会计服务为第三方开发人员提供了API。 如该项目的网站上所示，社交网络的第三方应用程序，移动应用程序和浏览器的插件在该资源的用户中最受欢迎。

服务团队将对应用程序进行测试，批准的应用程序进入一个特殊的市场并通过订阅进行分发。 根据EasyFinance管理部门的统计，应用程序开发人员每位用户每月可获得15至50卢布的收入。 该收入包括订阅服务本身的资费，平台合作伙伴的集成付款和广告的收入。
