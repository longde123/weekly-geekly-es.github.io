1. Microsoft 365: ترقية نشر المؤسسة باستخدام Windows 10 و Office 365


يساعد Microsoft 365 في إنشاء بيئة آمنة وقابلة للتحديث من خلال الاستفادة من أجهزة Windows 10 التي تقوم بتشغيل تطبيقات Office 365 التي تتم إدارتها باستخدام Microsoft Enterprise Mobility + Security.

ستعلمك هذه الوحدة ، المصممة لمدة 3.5 ساعات ، كيفية العمل مع Microsoft 365 ، والتحدث عن أساسيات استخدام الأداة ، وكذلك عن الأمان وتدريب المستخدم.

تعرف على المزيد وابدأ التدريب على هذا الرابط .

2. إدارة موارد البنية التحتية في أزور


تعرف على كيفية إنشاء موارد الجهاز الظاهري وإدارتها وحمايتها وتوسيع نطاقها في سحابة Azure. سيستغرق إكمال الدورة التدريبية بالكامل حوالي 10 ساعات.

وحدات المقرر:

  • نظرة عامة على الأجهزة الظاهرية Azure
  • إنشاء جهاز Linux افتراضي في Azure ؛
  • إنشاء جهاز ظاهري Windows في Azure ؛
  • إدارة الأجهزة الافتراضية باستخدام Azure CLI ؛
  • تحديث الأجهزة الافتراضية ؛
  • إعدادات الشبكة للأجهزة الافتراضية ؛
  • إنشاء قوالب Azure Resource Manager
  • تغيير حجم وإضافة الأقراص على الأجهزة الظاهرية أزور.
  • التخزين المؤقت أزور التخزين والأداء
  • حماية محرك الأقراص لأجهزة Azure الافتراضية.

3. إدارة الموارد في أزور


تعرّف على كيفية إنشاء الموارد السحابية وإدارتها والتحكم فيها باستخدام سطر أوامر Azure وبوابة الويب. بالمناسبة ، في هذه الدورة ، كما هو الحال في العديد من الدورات الأخرى ، يمكنك ممارسة الرياضة في صندوق رمل Azure بنفسك.

وحدات:

  • مطابقة المتطلبات مع أنواع السحابة ونماذج الخدمة في أزور
  • إدارة خدمات Azure باستخدام CLI
  • أتمتة مهام Azure باستخدام البرامج النصية لـ PowerShell ؛
  • التنبؤ بالتكلفة وتحسين التكلفة لـ Azure ؛
  • التحكم في موارد Azure وتنظيمها من خلال Azure Resource Manager.

4. أساسيات Microsoft 365


يعد Microsoft 365 حلاً ذكيًا يتضمن Office 365 و Windows 10 و Enterprise Mobility + Security ، مما يسمح لك بالتعاون الإبداعي في بيئة آمنة. في هذه الدورة التدريبية التي تستغرق 4 ساعات حول كل شيء - كل ما تحتاج إلى معرفته لبدء استخدام Microsoft 365.

سوف تتعرف على ماهية Microsoft 365 ، المعلومات الأساسية حول خدمات وميزات الخدمة ، والتعرف على العمل الجماعي ، والأمان ، والقدرات السحابية. بالمناسبة ، للتدريب تحتاج إلى معرفة سطحية على الأقل للحوسبة السحابية.

5. إدارة الحاويات في أزور


مثيلات حاوية Azure هي أسرع وأسهل طريقة لتشغيل الحاويات على Azure. سيساعدك نظام التدريب هذا على تعلم كيفية إنشاء وإدارة الحاويات وكيفية تحقيق تحجيم مرن لـ Kubernetes باستخدام ACI.

وحدات المقرر:

  • إنشاء تطبيق ويب حاوية باستخدام Docker ؛
  • إنشاء وتخزين صور الحاوية باستخدام خدمة سجل حاوية Azure ؛
  • تشغيل حاويات Docker باستخدام خدمة مثيلات حاوية Azure ؛
  • نشر وإطلاق تطبيق حاوية الويب باستخدام Azure Application Service ؛
  • نظرة عامة على خدمة Azure Kubernetes.
