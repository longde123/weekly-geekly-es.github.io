WebRTC di situs Anda - tanpa bug dan tanpa anggaran

Teknologi WebRTC telah ada sejak lama dan didukung oleh hampir semua browser (97% dalam pengalaman kami). Banyak orang berpikir untuk menginstal widget WebRTC untuk diri mereka sendiri, untuk berkomunikasi dengan pelanggan dan meningkatkan konversi.



Cara menambahkan panggilan ke situs web WebRTC Anda tanpa biaya tambahan dan memotong bug dengan garu, kami akan kirim di bawah ini.

Mengapa tombol dengan WebRTC?


  1. Dengan nyaman! Tidak perlu memanggil atau meninggalkan nomor Anda, cukup tekan tombol.
  2. Menguntungkan Baik Anda maupun klien tidak membayar apa pun. Komunikasi gratis untuk semua orang.

Apa solusinya?


Pengembangan sendiri


Anda dapat menginstal asterisk atau freeswitch untuk Anda sendiri dan mencoba menjalankan kode telepon webrtc gratis di atasnya. Cepat atau lambat, kodenya akan berfungsi, tetapi untuk bekerja dengan sebagian besar peramban Anda harus melangkah jauh. Dukungan lebih lanjut akan dimulai:

  • pertama, asterisk dengan ip eksternal membutuhkan pengetahuan keamanan (mereka sangat ingin merusaknya),
  • kedua, dukungan webrtc di browser tidak statis, nuansa berubah dalam versi baru.

Contoh Masalah WebRTC


Pada musim gugur 2018, setelah memperbarui Firefox dan Safari, sebagian besar ponsel Webrtc rusak di dalamnya. Di Firefox, ada masalah saat dimulainya panggilan, di Safari, suaranya menghilang begitu saja. Setelah beberapa hari bekerja dengan pengembang dan penguji, masalah telah diperbaiki (bahkan sebelum sebagian besar klien diperbarui). Firefox menetapkan perbaikan sementara untuk pembaruan berikutnya, untuk Safari mereka sedikit menyederhanakan algoritma telepon web. Anda yakin siap melacak dan memperbaiki semua masalah seperti itu secara mandiri?

Widget eksternal


Jika Anda tidak punya waktu dan keinginan untuk berurusan dengan masalah WebRTC, Anda dapat menginstal widget cloud. Widget cloud telah dibuat dan didukung oleh operator, Anda hanya perlu menghabiskan beberapa menit pada instalasi.

Biasanya, kerugian utama dari solusi cloud adalah harganya - Anda harus membayar secara teratur untuk menggunakan widget, dan apa yang lebih baik dari itu, misalnya, nomor 800? Proyek Zadarma menawarkan solusi lain - widget gratis untuk panggilan dari situs melalui WebRTC. Semua yang diperlukan dari sisi keuangan setidaknya kadang-kadang untuk mengisi akun (setiap 3 bulan), Anda dapat menggunakan dana untuk layanan telepon apa pun, widget itu sendiri gratis.



Fitur Cloud Widget Gratis


  • Desain yang dapat disesuaikan (bentuk, warna, font, tombol)
  • Statistik widget yang diperluas
  • Integrasi dengan cloud PBX gratis dan semua sistem CRM utama

Pasang widget gratis


Membuat widget sangat sederhana:

  • Kami membuat akun dan mengisi akun (berapa pun jumlahnya)
  • Di LC layanan, buka tombol "Hubungi kami" dan klik "Buat Widget"
  • Siapkan desain widget, klik "Simpan" dan dapatkan kode widget



Banyak hal yang diatur dalam widget, Anda bahkan dapat memilih apa yang harus disimpan di rumah dan apa yang ada di cloud (untuk situs besar dan amatir, AWS mungkin ingin meng-host-nya).

Untuk menyimpan widget di cloud (cara mudah), Anda perlu menyalin 2 blok kode ke situs Anda.

  • Di bagian bawah halaman (bagian tubuh), salin kode boot perpustakaan perpustakaan.
  • Kami mengatur kode widget itu sendiri ke bagian halaman di mana tombol "Panggil kami" seharusnya.


Seperti yang Anda lihat, desain widget dapat diubah kapan saja.


Jika Anda memutuskan untuk menyimpan widget di rumah, Anda harus mengunduh dan menyimpannya. Kemudian pilih tab "Instalasi offline", unduh versi terbaru dan juga masukkan 2 buah kode (hanya di awal Anda perlu menentukan jalur dengan mempertimbangkan folder tempat Anda mengunduh).

Instalasi selesai! Tombol untuk panggilan dari situs sudah siap dan berfungsi. Anda dapat mengklik, menelepon, melihat statistik.
