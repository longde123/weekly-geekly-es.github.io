Pengumuman Azure IoT Edge Tools (Pratinjau)

Kami senang mengumumkan ketersediaan pratinjau Ekstensi Azure IoT Edge Tools (Pratinjau) baru untuk Visual Studio 2019 . Ekstensi menyediakan serangkaian fungsi yang kaya untuk mendukung pengembangan solusi IoT Edge dengan Visual Studio 2019:



  • Proyek Azure IoT Edge baru yang menargetkan berbagai platform (Linux amd64, Linux arm32v7, Windows amd64)
  • Tambahkan modul IoT Edge baru (C # / C) ke solusi
  • Edit, bangun, dan debug modul IoT Edge secara lokal di mesin Visual Studio Anda
  • Buat dan dorong gambar docker dari modul IoT Edge
  • Jalankan modul IoT Edge di simulator lokal atau jauh
  • Menyebarkan solusi IoT ke perangkat IoT Edge (dengan Cloud Explorer )

Untuk mengatur lingkungan pengembangan lokal untuk debug, menjalankan, dan menguji solusi IoT Edge Anda, Anda memerlukan Azure IoT EdgeHub Dev Tool . Instal Python (2.7 / 3.6) , lalu instal iotedgehubdev dengan menjalankan perintah di bawah ini di terminal Anda. Pastikan versi Azure IoT EdgeHub Dev Tool Anda lebih besar dari 0.8.0. 

pip install --upgrade iotedgehubdev

Instalasi


Ada dua opsi untuk menginstal ekstensi baru:


  • Unduh dan instal ekstensi baru dari Visual Studio Marketplace .
  • Atau, Anda dapat menginstal ekstensi langsung dari dalam Visual Studio 2019 menggunakan menu Extensions -> Manage Extensions . Di jendela Kelola Ekstensi, pilih Online dari panel kiri dan input edge di kotak pencarian di kanan atas untuk mencari dan mengunduh "Azure IoT Edge Tools untuk VS 2019 [Pratinjau]".
