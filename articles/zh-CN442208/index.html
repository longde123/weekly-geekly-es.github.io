TL; DR

进入剪切区，选择一份出版物，然后等待CTRL / CMD + Enter-然后您将自己了解所有内容。

我们计划在全球范围内解决出版物中的错误和错别字的问题，并在将来加以改进，但是为此，我们需要重做很多事情，包括出版物编辑器。 因此，就目前而言，我们决定提供您要求我们提供的选项-通过kotkey发送有关出版物中错误的信息的功能。

操作算法如下：

1.在出版物中选择有错误的文本片段（可自行决定是否使用语法或标点符号）；

2.按热键CTRL + Enter（或CMD + Enter）；

3.屏幕底部将出现一个表格，其中引用了前面（第1点）突出显示的文本，以及一个可选说明的字段。

屏幕截图

4.单击“提交”。 该出版物的作者是一封私人消息中的信件（代表发送错字的人），看起来像这样：

屏幕截图

5.我们不希望作者很快注意到邮件并更正所发送的错字。

细微差别：

  1. 该功能仅在出版物中有效；
  2. 移动版将没有任何功能；
  3. 您可以至少选择整个出版物，但是引号中只能包含220个字符，因此最好用错字来指明单词。 在随附的注释中，您最多可以驱动500个字符；
  4. 1个错字= 1条消息（与出版物作者对话）；
  5. 如果您选择某人已经发送给出版物作者的片段，则会收到有关以下内容的通知：

  6. ReadOnly和Read＆Comment用户将在发送邮件的服务器场中添加验证码（仅适用于发送给每个新作者的第一条消息）。
