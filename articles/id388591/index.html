Mozilla sedang mempertimbangkan untuk merilis perangkat di Firefox OS daripada smartphone yang baru saja meninggal

Komunitas Mozilla akan mengembangkan beberapa perangkat berbasis Firefox OS untuk menggantikan proyek gagal dari ponsel cerdas mereka. Informasi tentang perangkat mendatang bocor ke jaringan melalui blog komputer Spanyol Hipertextual , setelah itu kepemimpinan komunitas mengkonfirmasi keandalan mereka. Di antara perangkat ini adalah tablet, komputer stick, router pintar, dan komputer di faktor bentuk keyboard.

Awal bulan ini, Mozilla mengumumkan penghentian rilis dan penjualan.smartphone berbasis Firefox OS - sistem operasi online di mana aplikasi tidak perlu diunduh. OS pada dasarnya adalah browser di mana semua aplikasi adalah aplikasi web. Prinsip serupa digunakan di Chrome OS - gagasan Google, yang mencoba secara agresif mempromosikan laptop dengan OS semacam itu, tetapi juga tidak mencapai banyak keberhasilan - OS ini akan segera menjadi bagian dari OS seluler Android .

Namun, menurut wakil presiden Mozilla untuk perangkat seluler, OS itu sendiri terus berkembang, dan perusahaan, tanpa kehilangan harapan, berencana untuk mencoba menggunakannya di perangkat lain. Dan seperti apa perangkat mereka nantinya - sesuai dengan konsep, deskripsi yang telah bocor ke jaringan.

Firefox Pad



Tablet 10 inci dengan sistem operasi online. Menariknya, sebenarnya, tablet dengan Firefox OS sudah ada - Vixen fox 7 inci dan InFocus 10,1 inci.

Stick Computer Firefox Stick



Komputer "pada tongkat", dongle PC, flash drive komputer, dll. Komputer yang terlihat seperti flash drive eksternal yang terhubung ke input HDMI dari setiap TV yang dilengkapi dengannya. Keyboard dan mouse bersifat nirkabel. Dukungan Wi-fi. Hasilnya adalah komputer di saku Anda. Apakah komputer seperti itu yang berbasis pada Firefox OS dapat bersaing dengan pesaing masih belum jelas.

Smart Firefox Hub Router



Tujuan utamanya adalah keamanan. Alat bawaan untuk kontrol privasi, kontrol orang tua, organisasi kolaborasi antara berbagai perangkat seluler, dan pembaruan perangkat lunak otomatis.

Firefox Pi Keyboard Komputer



Keyboard dengan trackpad dan komputer terintegrasi. Awalan Pi mengisyaratkan penggunaan komputer papan tunggal Raspberry Pi. Dilihat dari deskripsi, itu dapat dihubungkan ke TV atau monitor melalui VGA.

Sejauh ini, ini hanya konsep perangkat: tidak mungkin untuk mengatakan perangkat mana, dalam bentuk apa, kapan tepatnya akan muncul di pasar (dan apakah mereka akan muncul). Tetapi Mozilla akan dapat, jika dia mau, untuk mendapatkan umpan balik dari pengguna dan pendapat mereka tentang relevansi perangkat ini dari komentar pada berita ini.
