让我们详细介绍新型3D打印机PICASO Designer X的主要技术方面：

3D打印机PICASO Designer X的运动图

像PICASO 3D的所有机器一样，新型3D打印机的运动学是该公司自己开发的。 机械原理基于直径为10和8 mm的双圆柱轴。





在Z轴上-安装了12 mm的双圆柱轴。 桌面沿Z轴的移动-通过滚珠丝杠进行。 沿X和Y轴的驱动-通过步进电机和同步带进行。 一切都很熟悉。



PICASO Designer X 3D打印机挤出机

这是一台全新的挤出机，与以前型号的挤出机无关。 总的来说，Picaso Designer X挤出机使用了PICASO Designer X PRO业已证明的技术解决方案。 挤出机设计用于较宽的温度范围。 没有聚四氟乙烯插入物。 他们不愿意使用工程塑料。 塑料进料机构也发生了变化（向挤出机供应塑料的角度不像以前的PICASO DESIGNER那样尖锐）。 这使您可以无缝地使用易碎材料，例如WOOD。 从Designer X PRO的打印头上取下来的特殊塑料进纸导轨使使用柔性材料成为可能。





挤出机的最高温度为410°C。 这是所有使用FDM技术的个人3D打印机中最酷的指示。 在这些温度下工作的能力使您可以将Picaso Designer X用于具有高强度和耐火特性的工程塑料的印刷。 以前，只有在工业3D打印机上才能使用这种材料进行打印，起价为20,000美元。



PICASO Designer X 3D打印机控制电子设备

3D打印机PICASO Designer X收到了一个全新的控制板-这是PICASO3D自己开发的。 与Arduino业余板无关-不。 新板可在ARM CORTEX M4处理器上运行。 动力储备使您能够-处理复杂的3D打印算法，而不会出现毛刺和制动。 对于连续3D打印尤其如此。 同样，不要忘记中央处理器的强大功能使您能够不断扩展3D打印机的辅助功能，以使工作自动化。 请注意，制造商PICASO 3D-不断更新“ X”系列所有打印机的软件，并将其添加到耗材清单，3D打印机辅助系统的改进和现代化中。



PICASO Designer X 3D打印机工作平台

桌面是铝制平台，磁铁上带有可移动玻璃。 最高加热温度可达140°。 对于市场上出售的几乎所有塑料而言，足以保持工作室内部的粘附力和温度平衡。





挤出机和打印平台的最高工作温度范围（最高410°C）（最高140°C）允许您在选择打印材料时不受限制。 制作3D打印机PICASO DESIGNER X-适用于所有塑料的通用机器。

PICASO Designer X 3D打印机打印自动化系统

在新的PICASO Designer X 3D打印机中，出现了在X平台上工作的PICASO 3D打印机中使用的所有自动化系统。

即：

自动校准工作平台 -购买打印机后，您无需担心是否正确校准了3D打印机。 连接到网络后，按照屏幕上的说明进行操作，打印机将进行所有配置。





用于各种塑料的型材系统 -不再需要每次为某种类型的材料准备打印作业，安装塑料时，系统将与您检查使用哪种材料并自动更改所有必要的参数。



控制物质流动避免了婚姻。 当您的塑料用完并且打印机空闲时，将不再有其他情况。 系统将提前通知您，因此您将始终为此做好准备，并且不会浪费很多时间。 但是，塑料破损不是很常见的事件。 但是，例如由于螺纹直径不稳定，堵塞可能会更频繁地发生。 塑料进料控制系统将立即注意到进料的变化并开始自动清洁算法。 如果这样做没有帮助，打印机将暂停打印并等待操作员干预。

但这还不是全部。 新的控制系统可以跟踪塑料的任何问题。 包括不稳定的直径和线圈缠结。 如果没有控制系统，此类问题将导致打印缺陷和停机。 这可能需要很多时间。 现在，打印机在检测到问题后将暂停，并在排除故障原因后，可以继续工作。

例如：

如果您故意折断塑料-Designer X将立即出错，并且背光将变为黄色，屏幕上将出现相应的题词。

如果强行握住塑料，打印机将暂停，背光将变为黄色，屏幕将显示“喷嘴清洁”，然后将执行自动清洁过程，背光将再次变为白色，并且3D打印将继续。

零件位置控制系统。 现在，在新的Designer X中，如果对工作台施加过大压力，则该零件已松开并压在工作台上。 打印机暂停，并要求您检查零件。 通过特殊的传感器进行控制。
