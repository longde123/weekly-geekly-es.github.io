考虑税收和生活费用，开发商在哪些国家和城市能赚更多钱？



如果我们比较莫斯科，洛杉矶和旧金山的中级软件资格开发人员的薪水，以开发人员自己留给专业薪水监控服务的薪水数据为例，我们将看到：

  • 在莫斯科，这样的开发商在2019年底的薪水为130,000卢布。 每月（ 根据 moikrug.ru上的工资服务）
  • 在旧金山-每月9,404美元，大约等于564,000卢布。 每月（ 根据 glassdoor.com上的工资服务）。

乍看之下，旧金山的一名开发商的薪水是其4倍以上。 他们经常对此进行比较，对薪水水平的巨大差距得出可悲的结论，并回想起小猪彼得。

但同时，至少有两件事被忽略了：

  1. 在俄罗斯，薪水是在扣除13％的所得税后显示的;在美国，薪水是在扣除这种累进税之前的，取决于收入，婚姻状况和工作人员，范围为10％至60％。
  2. 此外，莫斯科和旧金山的当地商品和服务成本也大不相同。 根据 numbeo.com服务，旧金山的日常用品和租赁住房的成本几乎是莫斯科的三倍。

因此，如果考虑到税收，事实证明您需要比较130,000卢布的薪水。 在莫斯科的薪水为248,000卢布。 在旧金山（从工资中扣除28％的联邦税和28％的州所得税）。 并且，如果您还考虑生活费用-那么从92,000卢布起。 （我们将薪水除以2.7-生活费用比莫斯科高的次数）。

事实证明，与旧金山同行相比，莫斯科的一名中级软件开发人员可以用自己的薪水负担更多的本地商品和服务。

对于曾经获得的计算结果感到惊讶，我们决定将莫斯科的中间商薪资与世界其他城市的中间商薪资进行比较，而世界上其他城市的中间商薪资通常在最适合开发商的最佳城市中找到。 结果是列出了45个城市以及人口超过100万的12个俄罗斯城市。 您认为莫斯科在哪里？


计算方法


源数据


薪水

  • 俄罗斯城市开发人员的薪水来自薪资计算器moikrug.ru （2019年下半年的数据），基辅的开发人员薪水-dou.ua计算器的薪水（2019年6月至7月的数据），明斯克的开发人员薪水-的计算器dev.by （2019年的薪水），其他城市的薪水-来自glassdoor.com计算器。 所有薪水均按19/11/08的汇率转换为卢布。
  • 在上述所有服务上，用户自己说明他们的专长，资格，居住地以及他们目前收到的薪水
  • 为了搜索glassdoor，dou.ua和dev.by上的薪水，我们使用了请求“软件开发人员”（对应于俄罗斯的中层人员），如果缺少数据，我们使用了请求“软件工程师”。

生活费用

  • 为了计算世界各地城市的生活成本，我们使用了numbeo.com服务所考虑的生活成本加租金指数，将包括租金在内的消费品价格与纽约的类似价格进行了比较。

税收

  • 我们从各种开放源代码中获取了世界城市的税费，并链接到了我们编制的税费目录及其缩略表版本 。 每个人都可以仔细检查信息或提出更正建议。
  • 在某些国家/地区，税率大不相同，这不仅取决于收入额，还取决于许多其他因素：家庭成员，子女的存在，共同申报的声明，宗教派别等。 因此，为简单起见，我们认为该工人是单身，没有孩子并且不属于任何宗教派别。
  • 我们认为，俄罗斯，乌克兰和白俄罗斯的所有薪水都应在缴纳税款后以及其他国家（在缴纳税款之前）标出。



我们算什么


了解了每个城市的税金以及相对于莫斯科的中位数工资和平均生活成本，我们能够比较每个城市可以购买的商品和服务与莫斯科的类似商品和服务的数量。

就我们自己而言，我们将其称为商品，服务和租赁房屋的安全性指数 ，或者简称为安全性指数 。

例如，如果对于该城市而言，该指数为1.5，那么对于该工资，加上该城市中的价格和税收，您可以购买的商品是莫斯科的一半以上。

一点数学：

  • 令Sm为莫斯科（薪金）的工资中位数，令Cm为莫斯科（成本）的商品，服务和出租公寓的成本。 然后Qm = Sm / Cm-您可以在莫斯科以薪水购买的商品数量（数量）。
  • 令Sx为城市X的工资中位数，Cx为城市X的商品，服务和租赁公寓的成本。那么Qx = Sx / Cx是您可以在城市X中为薪水购买的商品数量。
  • Qx / Qm-这是我们需要的安全性指标 。

仅计算numbeo的生活成本和租金的指数，如何计算该指数？ 方法如下：

  • Im = Cx / Cm-X城市的生活成本指数与莫斯科相比：显示X城市的商品，服务和出租公寓成本多少倍于莫斯科的同等价值。 在源数据中，我们具有类似的Numbeo指数，该指数将所有城市与纽约进行了比较。 我们很容易将其转换为一个索引，该索引可以将所有城市与莫斯科进行比较。 （Im = In / Imn * 100，其中In是城市生活成本的指数，Imn是nambueo在莫斯科的生活成本的指数）。
  • Qx / Qm =（Sx / Cx）/（Sm / Cm）=（Sx / Sm）/（Cx / Cm）=（Sx / Sm）/ Im

也就是说，要获得一个城市的商品，服务和出租住房的可及性指数，您需要将该城市的薪资中位数除以莫斯科的薪资中位数，然后再将该城市的生活成本除以莫斯科。
