CES 2016上的7架最佳无人机



2016年国际消费电子展已  成为众多新型和改良无人机的家园，这些无人机将在  今年争取在空域中的地位。
 

1.杭航184


这架无人机可以搭载一名乘客，无需控制-只需在平板电脑上指示目的地即可。

Ehang 184的成本为200-300,000美元，可以达到的速度可达100 km / h，可以在高达3500米的高度飞行23分钟，承载100公斤的重量。
 

 
让我们看看Ehang 184是否合法，但该公司声称他们在此问题上与政府密切合作。
 

 

2. Yuneec的台风H


Yuneec在CES 2016上推出了采用英特尔实感技术的Typhoon H，以避免碰撞。

台风H配备了360度12万像素吊坠相机，可以拍摄4K视频。Yuneec说，台风H的价格为1799美元，英特尔实感技术将作为附加选项提供。
 

 

 

3.百合相机


Lily坚持认为这是一台摄像机，而不是无人驾驶飞机，尽管如此，Lily Camera还是赢得了CES 2016最佳创新奖。在发射1.3千克无人机之后，视频录制开始于1080p 60 fps。另外，它是防水的。

莉莉可以在高达15 m的高度飞行，并且会紧紧跟随您。使用特殊的手镯，您可以控制其飞行的速度，角度和高度，以获得最佳的拍摄效果。
 

 
在预购阶段，2015年底，该公司筹集了3400万美元，目前已经售出6万份。今年2月，预计发布Lily相机无人机的价格为800美元。
 

 

4.鹦鹉迪斯科舞厅


派诺特的新型Disco无人机将标准的四轴飞行器和飞机设计与机翼几何形状保持不变。要起飞，您只需要将它扔到空中并使其达到所需的高度即可。飞行时间-45分钟，最高时速-80 km / h。飞机前部装有摄像头。
 

 

 

5.大疆Phantom 3 4K


DJI Phantom 3 4K可以拍摄4K视频（30帧/秒，数据速率-60 Mb / s），性能与Phantom 3 Professional相同。但是，Phantom 3 4K的价格为999美元，而Phantom 3 Professional的价格为1200美元。

两种型号之间的主要区别在于Phantom 3 4K使用Wi-Fi以有限的分辨率（480p）和流速度将视频传输到控制面板。Phantom 3 Professional和其他DJI无人机使用LightBridge系统将视频流传输到平板电脑或智能手机的屏幕上。
 

 

 

6. Autel的X-Star Premium


Autel的X-Star Premium 无人机将于2016年2月上市，价格为999美元，它具有三轴底盘，可提高相机的稳定性。它以30帧/秒的速度拍摄4K分辨率的视频，或者以120帧/秒的速度拍摄1080p的视频，还可以进行在线广播。
 

 
无人机配备了GPS和GLONASS系统，可进行自动飞行和定位。该套件包括一个64 GB的microSD卡和一个用于运输的背包。
 

 

 

7.弗莱


Fleye的 主要创新是无人机独特的球形设计，所有活动部件均被隐藏和保护。反过来，这意味着您可以握住设备，触摸设备，推动设备并将其撞入Fleye无人机，而不必担心会伤害自己或他人。而且无人机本身将变得更具抵抗力。

相机可以30帧/秒的速度拍摄1080p分辨率的视频。
 

 
Fleye项目在Kickstarter上展示并成功完成，筹集了314,080欧元，预计2016年将以1,365美元的价格交付无人机。
