Vortrag "Apollo Programm: wie es war"

Letzte Woche gurgelte laut eine Geschichte von einer gefälschten Nachrichtenseite, dass der Trump-Wissenschaftsberater angeblich erklärte, die Amerikaner seien nicht auf dem Mond. Das Thema des Apollo-Programms ist fast ewig, die Analyse der Fehler von Verschwörungstheoretikern ist beliebt, aber meiner Meinung nach ist es wichtiger, über das Programm selbst zu sprechen, damit eine Person eine bestimmte Grundlage hat, auf der sie bereits Inkonsistenzen (und oft banalen Analphabetismus) in Verschwörungstheorien feststellen kann. Dies war das Thema meines letzten Vortrags im Ufa-Planetarium.
