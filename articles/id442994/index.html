Mengumumkan Sumber Terbuka Kalkulator Windows

Hari ini, kami senang mengumumkan bahwa kami membuka sumber Kalkulator Windows di GitHub di bawah Lisensi MIT. Ini termasuk kode sumber, sistem bangun, tes unit, dan peta jalan produk. Tujuan kami adalah untuk membangun pengalaman pengguna yang lebih baik dalam kemitraan dengan komunitas. Kami mendorong perspektif baru Anda dan peningkatan partisipasi untuk membantu menentukan masa depan Kalkulator.



Sebagai pengembang, jika Anda ingin tahu bagaimana berbagai bagian aplikasi Kalkulator bekerja, mudah mengintegrasikan logika Kalkulator atau UI ke dalam aplikasi Anda sendiri, atau berkontribusi langsung ke sesuatu yang dikirimkan di Windows, sekarang Anda bisa. Kalkulator akan terus menjalani semua pengujian, kepatuhan, keamanan, proses kualitas, dan penerbangan orang dalam seperti biasa, seperti yang kami lakukan untuk aplikasi kami yang lain. Anda dapat mempelajari lebih lanjut tentang detail ini di dokumentasi kami di GitHub.



Buka contrib


Selain menggunakan kembali dan mengadaptasi kode di aplikasi Anda sendiri, siapa pun dapat berpartisipasi dalam pengembangan Kalkulator Windows . Terlibat sederhana. Belajar


Meninjau kode Kalkulator adalah cara yang bagus untuk mempelajari tentang teknologi Microsoft terbaru seperti Universal Windows Platform, XAML, dan Pipa Azure . Melalui proyek ini, pengembang dapat belajar dari siklus pengembangan penuh Microsoft, serta menggunakan kembali kode untuk membangun pengalaman mereka sendiri. Ini juga merupakan contoh yang bagus dari desain aplikasi Fluent . Untuk membuat ini lebih mudah, kami akan berkontribusi kontrol kustom dan ekstensi API yang kami gunakan dalam Kalkulator dan aplikasi lain, untuk proyek-proyek seperti Windows Community Toolkit dan Windows UI Library .
