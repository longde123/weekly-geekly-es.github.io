Mainan kayu - tulisan

Ini adalah rilis luar biasa dari "mainan kayu" - itu awalnya tidak direncanakan, tetapi entah bagaimana itu muncul dengan sendirinya.

Saat menulis retrospektif game di bawah DOS, secara berkala di komentar mereka berbicara tentang topik "mengapa tidak disebutkan <game> untuk <platform>". Jadi karena tidak disebutkan untuk <platform>. Secara umum, saya memiliki sedikit pengalaman bermain di platform lain (jika konsol game tidak dihitung, tetapi ini adalah topik yang terpisah, yang saya tidak melihat banyak poin). Dan - beberapa tahun di MSX dan Spectrum di klub-klub game di akhir tahun delapan puluhan - awal tahun sembilan puluhan. Karena saya ingat sedikit dari sana. Tetapi dengan selusin mainan masih diketik, jadi ada cukup untuk masalah yang terpisah. Ini bukan beberapa game terbaik, mereka hanya game yang ada dalam ingatan saya. Bagi sebagian orang, bahkan nama-nama baru saja ditetapkan.

Emulator saya adalah konsol retro. Ada Spectrum dan MSX di dalamnya, meskipun saya mengendarai segala macam boneka dan pengawas lebih. Dan begitu menarik game di suatu tempat sampai playstation pertama.

ZX Spectrum


Baik kenalan saya maupun kenalan saya tidak memiliki spektrum rumah - di kota kami, PC menjadi tersedia sejak dini. Ya, atau saya berputar di lingkaran di mana PC menjadi tersedia lebih awal. Karena Spectrum hanya bertemu di klub-klub game dan game-game dari yang paling populer diingat.

Penyabot

Seorang ninja dengan semacam tujuan ninja berkeliaran di sekitar sebuah kompleks besar. Secara umum, saya bahkan berpikir bahwa saya tidak melihat bagian pertama, tetapi saya ingat trailer dari gambar dengan jelas, di bagian kedua tidak.





Saboteur 2

Bagian kedua, dia bermain lebih banyak. Tiba di hang glider dan berlari melalui beberapa kompleks lainnya. Tetapi juga dengan target ninja.



Rex

Dari semua pejalan kaki, yang ini diingat. Terutama bidang pelindung dan titik respawn.



Serangan sungai

Di sini saya tidak yakin apa yang saya mainkan dalam versi Spectrum - mungkin ada Atari 2600, untuk beberapa alasan saya ingat kotak tanpa keyboard. Di sisi lain, saya melihat pesawat ini di banyak tempat sehingga setidaknya di suatu tempat Spectrum.



Star Raiders II

Tapi saya ingat game ini untuk waktu yang sangat lama. Biasanya, menurut deskripsi - sistem planet multi-warna, hyperspace, pysch-pyshchik - mereka menyelipkan saya sebuah Elite.



Cara Tinju Meledak

Game pertarungan. Saya juga tidak berpikir bahwa saya ingat, tetapi tampilan kesehatan dengan simbol yin-yang tenggelam, saya kenali dari gambar.

Adam & Hawa baru

Ditambah lagi, secara berkala peristiwa dengan penghitung waktu muncul ketika Anda perlu melakukan sesuatu dengan cepat, jika tidak semuanya.
