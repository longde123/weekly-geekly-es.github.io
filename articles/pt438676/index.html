Plug-in do webinar 3CX para o seu site WordPress

Apresentando o novo plug - in de webinars da 3CX para WordPress!

O plug-in permite que você publique em seu site uma lista de seminários on-line criados no cliente web 3CX . Os visitantes do site registram-se em seminários on-line através do formulário on-line do plugin. A aparência do plugin pode ser personalizada através de modelos (Javascript e HTML). Os seminários on-line são realizados na plataforma WebMeeting, integrada ao PBX 3CX.

Listamos os benefícios do plugin:

  • Registro rápido - os visitantes do site visualizam a lista de seminários on-line e registram-se rapidamente usando o nome e o email ao qual o link para a conexão é fornecido.
  • Conexão fácil - os participantes convidados se conectam ao webinar usando o link que abre uma janela do navegador com a interface da conferência. A tecnologia de conferência WebRTC é integrada ao navegador, para que os participantes não precisem instalar aplicativos adicionais.
  • Conferências Wide WebMeeting : Gravação de conversas, flipcharts, compartilhamento de tela, apresentações, etc.



É suficiente para o organizador do webinar agendar um webinar na interface do cliente da web 3CX . E os visitantes do seu recurso podem se registrar, receber lembretes e se conectar ao evento de qualquer lugar do mundo. No entanto, você pode oferecer determinados seminários on-line apenas para usuários pré-registrados do site. O organizador também pode acompanhar o registro dos participantes na seção "Agendar uma conferência" do 3CX Web Client.

Benefícios do 3CX WebMeeting for Webinars


Videoconferência e webinars se estabeleceram como uma ferramenta de e-learning. O uso do 3CX WebMeeting para treinamentos técnicos, corporativos e outros oferece economia significativa de custos, como Não requer nenhuma assinatura e compra de equipamento, exceto webcams e fones de ouvido. É suficiente criar um webinar no cliente da web 3CX v16 Pro e as informações sobre ele serão exibidas imediatamente em seu site. A gravação em conferência integrada aprimora ainda mais o valor da solução, permitindo que você volte aos webinars anteriores e arquive-os para uso futuro.



Instale e configure o plug-in


Para instalar o plug-in, vá para a interface de gerenciamento do WordPress CMS e vá para a seção Plug - ins> Adicionar novo . Procure o plug-in dos seminários on-line 3CX e clique em Instalar agora .



Em seguida, acesse a interface de gerenciamento 3CX (v16 e superior) em Configurações> WordPress> Plug-in para Webmeeting / webinars e clique em Gerar para criar um novo token da API Webmeeting para autorizar as solicitações de plug-in para publicar a lista de webinars no site. Clique em Mostrar para ver o token da API e Copiar para copiá-lo para a área de transferência.

Criando um objeto de formulário de Webinar


Após instalar o plug-in no site, crie objetos de formulário do Webinar. Eles são usados ​​em páginas específicas e entradas de blog ou em todo o site para colocar uma tabela com os seminários on-line agendados.

Para criar um objeto, no painel de administração do WordPress, vá para a seção Webinars da 3CX , clique em Adicionar novo e especifique o nome do objeto.



No campo FQDN do WebMeeting, especifique o FQDN do servidor WebMeeting conforme indicado no lado direito da interface de gerenciamento 3CX na seção Início.

Os seguintes parâmetros são usados ​​para personalizar a aparência do formulário:

  • Token da API do WebMeeting - insira o token gerado na seção Interface da 3CX, Configurações> WordPress> Plug-in para Webmeeting / webinars .
  • Número de ramal 3CX - especifique o número do ramal do usuário 3CX que publicará a lista de webinars.
  • País / idioma do webinar - selecione o idioma / país do webinar.
  • Limitar assinantes a - o número máximo de participantes (locais) que podem se registrar no webinar.

Como postar um formulário de webinar em um site WordPress


Para colocar o formulário do webinar no lugar certo no site, copie o código de acesso correspondente da lista de objetos de formulário do webinar.

Posicionamento em todas as páginas do site: vá para Aparência> Widgets e arraste o widget Texto para a área do widget que aparece em todo o site, por exemplo, na parte inferior (rodapé) de um tema do WordPress. Em seguida, adicione um código de acesso, por exemplo, [3cx-webinar id="9" title=" "] na área de conteúdo e clique em Salvar.



Postagem em páginas selecionadas: crie uma postagem ou página de blog e adicione um código de acesso à área de conteúdo. Clique em Visualizar alterações para ver uma página com um formulário de webinar incorporado.
