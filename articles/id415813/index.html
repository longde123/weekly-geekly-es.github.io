Beberapa catatan tentang kondisi Cloud Gaming saat ini


Setelah membaca artikel oleh Dell dan DROVA: cara bermain game yang menuntut bahkan pada laptop yang lemah , saya ingat, antara lain, artikel Tinjauan pasar untuk layanan cloud gaming dengan topik streaming game.

Parsec disebutkan secara singkat dalam komentar yang satu itu (meskipun untuk beberapa alasan penulis memutuskan bahwa Parsec dibuat oleh Amazon, mungkin berdasarkan artikel blog AWS Talking with Parsec, Game-Changer of Gaming .



Jadi, fungsi Parsec tidak terbatas pada Amazon saja. Ini adalah solusi universal untuk streaming game (dan tidak hanya>.
Termasuk dari komputer lain (ada beberapa persyaratan, misalnya, tuan rumah harus mendukung pengkodean video perangkat keras (NVENC (lebih baik), VCE, Intel QuickSync), tuan rumah harus memiliki Windows 8.1 / Windows 10 atau versi Windows Server yang sesuai. Windows 7 tidak didukung dan tidak akan).

Klien - Windows / macOS / Linux, Android (tetapi membutuhkan gamepad), rPI3. Versi iOS no.

Joystick, tablet, dan semua itu


Jika permainan membutuhkan setir dan pedal atau joystick yang baik atau kita tidak perlu bermain, tetapi menggambar dan perlu menghubungkan, misalnya, tablet grafik Wacom, maka ada juga solusi - aplikasi VirtualHere. Meneruskan USB melalui jaringan (percobaan mendukung satu perangkat). Biayanya 49 USD per server (dalam hal ini, server adalah perangkat tempat perangkat USB macet)

Parsec Cloud Marketplace dan Paperspace


Parsec memiliki Cloud Marketplace sendiri, dengan kemampuan untuk menyewa mesin virtual (dan menghidupkan / mematikannya secara dinamis), Amazon AWS dan Paperspace didukung. Harganya sama dengan penyedia sumber. Frodchek jauh lebih tangguh (dan tampaknya dia tidak mengerti apa yang harus ditanyakan Negara dalam beberapa kasus - tidak pantas untuk ditanyakan, tetapi masuk akal untuk bertanya kepada Negara), situs dukungan teknis secara eksplisit mengatakan bahwa ya - mereka mengerti bahwa akan ada Positif Palsu. Tapi lebih baik dari penipuan.
Pada saat yang sama, Paperspace yang sama bahkan memiliki template virtual siap pakai untuk Parsec. Baca selengkapnya Game dengan Paperspace .
Harga untuk mesin virtual Paperspace dengan 30 Gb RAM dan Quadro P4000 dengan 8 Gb RAM - 0,4 USD / jam (0,51 USD / jam dari 6 Juli 2018) (pembayaran bulanan tersedia) + biaya penyimpanan. Tidak perlu membayar untuk lalu lintas. Tetapi IP Publik (2 US $ per bulan) harus membayar untuk menggunakan Parsec.
Jika Anda bermain lebih dari 20-30 jam sebulan, itu tidak berhasil - ternyata cukup menguntungkan. Omong-omong, ini lebih menguntungkan daripada harga yang diumumkan DROVA.
Sebenarnya, Paperspace memiliki tujuan lain - server untuk pembelajaran mesin, ada kerangka kerja khusus untuk ini, mungkin saya akan memberi tahu Anda waktu lain.

Jika Anda menginginkan server yang lebih dekat


Seperti yang telah disebutkan, Parsec tidak terikat dengan hosting.
Temukan saja server dengan GPU dengan harga yang sesuai untuk Anda. Mereka, termasuk di Moskow.

Namun, Anda masih dapat mengumpulkan mesin dan tempat yang nyaman di dekatnya.
Anda tentu saja dapat juga merakit mesin Anda sendiri, meletakkannya di tempat yang nyaman dan menggunakannya. Tapi ini investasi dalam mesin.

Kekurangan


  • Dari sudut pandang layanan yang memeriksa pengguna dengan geoip - Paperspace adalah Amerika Serikat (bahkan pusat data di Amsterdam).
  • Menambang di server setidaknya dengan Papeprspace secara tegas dilarang. Jika kita tidak memerlukan server sepanjang waktu, tetapi harus mengambilnya tanpa biaya per jam, itu tidak akan selalu berfungsi untuk membayar kembali setidaknya sebagian. Terutama di Rusia, dengan kebingungan mengenai status cryptocurrency.
  • Anda masih harus membeli gim sendiri. Selain itu, dalam kasus Paperspace - Anda perlu menentukan server Steam CDN yang lebih optimal, karena dengan geoip - pilih California (!). tetapi ini juga merupakan keuntungan, karena Anda dapat benar-benar memakai busana apa pun.
  • Unggah file ke server dari Paperspace melalui Google Drive jika Anda mengunggahnya -
    tidak akan berfungsi, ada versi server Windows. Saya menggunakan Yandex.Disk (42 Gb gratis berguna).
  • Saat membayar pogigabyte akses Internet - lebih baik tidak menggunakannya.
  • Jika saluran Internet tidak dapat memberikan setidaknya 15 Mbps TANPA bintang, tayangan tidak akan menjadi yang terbaik.


Jadi, sekarang di Rusia, sangat mungkin untuk menggunakan layanan cloud untuk menjalankan game yang intensif sumber daya, menggunakan mod atau peralatan tambahan tidak menjadi masalah. Dan harganya pun cukup kompetitif jika Anda menggunakan layanan asing.
