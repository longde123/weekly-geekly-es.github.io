Kamp Pengembang Microsoft Azure Rusia



23 Mei | 9:30 - 19:00
Microsoft Office, BP Krylatsky Hills, ul. Krylatskaya 171, Ruang pertemuan Uranus, Jupiter, Saturnus
Moskow, Rusia


Proliferasi teknologi cloud secara bertahap mengubah metode pengembangan aplikasi, dan tantangan baru bagi programmer. Aplikasi tidak dikonsolidasikan, operasi dilakukan secara paralel dan asinkron. Kontainer sebenarnya menjadi standar dalam hal peningkatan aplikasi di cloud.


Ambil bagian dalam Perkemahan Pengembang Microsoft Azure, acara teknis gratis untuk pengembang yang akan diadakan 23 Mei di Moskow. Sebagai bagian dari pelatihan, Anda akan memperoleh pengetahuan tentang kemampuan Azure dan keterampilan praktis dalam migrasi dan modernisasi aplikasi dan infrastruktur.


Selama kelas, kita harus mempelajari kontainer secara detail, bekerja dengan seperangkat besar layanan cloud Azure untuk mengimplementasikan solusi berbasis kontainer, dan juga menguasai berbagai teknik untuk memperluas fungsi kontainer tanpa mengubah kode yang ada.
