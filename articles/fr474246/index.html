JavaScript Meetup SuperJob: rapport vidéo

Le 10 octobre, nous avons eu un SuperJob Meetup JavaScript . Nous racontons ce que les participants ont discuté, partageons des vidéos et des présentations.



Alexander Kukhta , développeur principal de SuperJob, a parlé du sujet "Expérience avec JSON: API sur le client" .

Alexander représente l'équipe Platform, qui est responsable de l'architecture et des performances de l'application JavaScript de l'entreprise.

En tant que premier orateur, Alexander a partagé le fonctionnement de l'interface dans SuperJob. Il a ensuite expliqué comment les gars stockent environ 1 500 entités brutes sur le client, comment ils travaillent avec eux dans les composants React et comment ils l'ont fait pour que l'application ne ralentisse pas.


→ Présentation d'Alexandre
→ Démo de la performance


Pyotr Solopov , chef de l'équipe de développement de SuperJob, a parlé de "Type Systems and Frontend Typing" .

Dans le rapport de Peter, vous pouvez trouver des réponses aux questions:

  • Quels systèmes de types existent et comment ont-ils évolué?
  • Quelles sont les principales différences entre Flow et TypeScript?
  • Comment Flow est-il utilisé dans un grand projet comme SuperJob?

Et là, vous pouvez trouver beaucoup de bonnes mathématiques, que beaucoup ont probablement oubliées du banc des élèves.


→ Présentation de Peter


Alexander Savin , responsable du développement frontend de SuperJob, est intervenu sur le sujet «Création d'un serveur JSON: API sur Node.js et PostgreSQL» .

Alexander a partagé son expérience dans la création d'un serveur selon l'API JSON: JavaScript.
Il a expliqué quels outils peuvent vous aider à obtenir rapidement l'API d'une base de données existante et comment créer vous-même un tel outil à l'aide de Node.js et de PostgreSQL.
Alexander a également abordé le sujet des difficultés possibles lors de l'utilisation de l'approche «première base de données» et a expliqué ce qui doit être prévu dans l'architecture de l'application pour les minimiser.


→ Présentation d'Alexandre
