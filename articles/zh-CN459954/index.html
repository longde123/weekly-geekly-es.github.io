为什么最大的IT公司之一加入CNCF-开发云基础架构的基金

一个月前，苹果加入了云原生计算基金会。 我们了解这意味着什么。


照片- 莫里茨·金德尔-Unsplash

为什么选择CNCF


Cloud Native Computing Foundation（CNCF）支持Linux Foundation。 其目标是开发和推广云技术。 该基金由主要的IaaS和SaaS提供商，IT公司和网络设备制造商-Google，Red Hat，VMware，Cisco，Intel，Docker等成立于2015年。

今天，甚至包括阿迪达斯，GitHub和《纽约时报》在内的组织也成为该基金会的成员。 一个月前，苹果加入了他们的行列-它获得了白金身份， 并将每年为开放项目的开发支付 37万美元。

苹果和开源项目历史悠久。 该公司是最早在产品开发中积极使用开源软件的公司之一 。 一个示例就是OSX。另一个操作系统Darwin的组件是该操作系统的核心。 它将Apple自己编写的代码与NeXTSTEP和FreeBSD收到的代码结合在一起 。

CNCF和Linux基金会的代表表示 ，通过加入开放基金，苹果公司希望分享其专业知识。 工程师希望为他们的工作回报开源社区，并为云IT基础架构的发展做出贡献。 苹果公司代表通常不会对公司的决定发表评论。

这会影响什么


云开发将更快。 来自CNCF的项目包括Kubernetes容器编排系统，Prometheus基础结构监视工具，CoreDNS服务器和Envoy代理服务。 甚至在加入CNCF之前，苹果就一直积极参与其开发（特别是Kubernetes）。

通过成为Cloud Native Computing Foundation的成员，该公司将能够与同事进行更紧密的沟通。 由于白金的地位，在确定用于开发云工具的载体时将考虑苹果代表的意见。 CNCF当前正在从事其他15个项目，以保护生产环境，云中的文件以及消息传递。 考试苹果电脑可能会加速他们的发展。


照片- 莫里茨·金德尔-Unsplash

将会有更多的开放项目。 苹果将​​帮助现有项目的开发并带来新的项目。 该公司已经将上述的Darwin的组件XNU和Swift编程语言转移到了开源内核中 ，该语言今天在TIOBE排名中排名第13位 。

一年前，苹果公司披露了分布式NoSQL数据库FoundationDB 的源代码。 与其他类似系统不同，FoundationDB中的操作符合ACID的原则：原子性，一致性，隔离性和数据持久性。

几个星期以来，超过7,000名开发人员对该项目表现出兴趣 ，并在论坛上开设了一百个新线程。 该公司计划与社区进一步开发新的开放工具。

还有谁最近加入CNCF


今年3月，CNCF代表宣布 59个新组织加入了社区。 到5月底，参与基金的人数超过了 400家公司。 其中既有小型创业公司，也有大型IT公司。

例如，将致力于在云中开发人工智能系统的Nvidia已成为该基金的新成员。 值得注意的是Elastic-由Elasticsearch，Kibana，Beats和Logstash组成的堆栈的开发人员-以及电信设备制造商爱立信。

除了这些组织之外，还列出了一些云提供商，Internet服务提供商，咨询机构，集成商和信息安全公司。

云原生计算基金会坚信，新进入者及其技术将为云市场的发展做出贡献，并为开源生态系统带来宝贵的专业知识。
