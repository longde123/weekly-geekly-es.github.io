Keindahan atau kepraktisan di ruang server

Manajemen Kabel Rack Server




Suatu hari, kami melakukan peletakan kabel berikutnya di ruang server. Mereka telah melakukan ini berkali-kali, dan hanya sekarang saya punya perasaan kontroversial. Di rak ada port Lan, ada telepon analog. Tidak ada binding pada port switch, tetapi panel telepon memiliki ikatan ke nomor dan administrator kadang-kadang beralih pelanggan untuk menyimpan nomor ketika pindah ke kantor lain. Sebelum dikemas ulang, nyaman untuk melakukan pekerjaan port switching. Kabel patch mudah diputuskan dan memungkinkan untuk membunyikan port panel pelanggan dan port switch atau panel telepon. Setelah pemasangan ulang, ketika semua kabel patch diletakkan pada kabel organizer, itu menjadi jauh lebih rumit. Dengan telepon, situasinya sedemikian rupa sehingga lebih mudah untuk menonaktifkan kabel patch dan meninggalkannya di rak, menghubungkan yang baru, daripada melepas dan menyambung kembali. Untuk melepaskan kabel patch lama dan merekam koneksi port-port, untuk ini Anda perlu membunyikan kabel patch di rak, kami menghabiskan beberapa jam, setelah meletakkan kembali di rak, saya mengerti bahwa prosedur ini akan memakan waktu dua kali lebih banyak.

Jadi saya berpikir tentang bagaimana menggabungkan kemudahan perawatan dan "keindahan" visual.

Cara membuat rak server praktis


Dari solusi yang saya tahu, tetapi sayangnya, belum digunakan di fasilitas ini, ini adalah pihak penyelenggara untuk kabel . Saya yakin bahwa jika ada pengatur lateral, pekerjaan "dering" kabel patch akan menjadi lebih mudah. Dudukan memungkinkan Anda untuk memperbaikinya bahkan di samping, meskipun kami biasanya memasang di bagian depan. Dengan panggilan, Anda dapat membuka penutup penyelenggara dan melacak rute kabel di penyelenggara samping. Penampilan akan menjadi sedikit lebih buruk, tetapi kepraktisan akan meningkat secara signifikan.

Solusi lain adalah spidol untuk kabel patch . Kami tidak melakukan ini sebelumnya, mungkin kami tidak melihat alasan untuk menggunakan penandaan, tetapi dalam karya selanjutnya kami akan mencoba memperkenalkannya. Saya merasakan kemudahan penggunaan ketika saya perlu mencari tahu port koneksi di lemari kabel besar, dan sejumlah besar kabel patch membuat tugas ini sulit, karena ada spidol pada kabel dari kedua ujungnya, yang memungkinkan untuk "cincin" kabel patch.
Dan solusi terakhir adalah log kabel dengan port switching. Ada tata letak ruangan dengan lokasi outlet di fasilitas, tetapi tidak ada majalah dengan port untuk menghubungkan ke peralatan, dan kami sayangnya tidak melakukan ini.

Saya bertanya-tanya solusi apa yang ada untuk menciptakan kepraktisan dan penampilan lemari server yang dapat diterima?
