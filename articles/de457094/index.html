Excel-Add-In, mit dem Filter bei der Arbeit mit Cubes (VBA) einfach festgelegt werden können

Wie Sie wissen, können Sie in Excel standardmäßig keine Filter gemäß der Werteliste für Pivot-Tabellen festlegen, und dies ist eine so notwendige Sache! Wie filtere ich Produkte nach hundert Codes und dann nach weiteren hundert? Es gibt natürlich Wege, aber das alles ist nicht das ...

Oder stellen Sie beispielsweise den Filter für den Würfel durch Messung mit Datumsangaben ein, kreuzen Sie jedoch nicht jeden Tag / Monat / Jahr an, sondern stellen Sie den Bereich von ... bis ... ein.
Es ist immer noch recht häufig erforderlich, Zeiträume für feste Muster festzulegen, z. B. den aktuellen Tag / Monat / Jahr . Zu diesem Zweck können Sie Sätze hinzufügen (im mehrdimensionalen Würfelmodell) oder der Dimension spezielle Attribute hinzufügen (dies ist nicht sehr praktisch, aber das tabellarische Würfelmodell enthält keine Sätze).

Wir haben eine VBA-Erweiterung gesehen, die dem Menüband eine Cube- Symbolleiste mit magischen Schaltflächen hinzufügt.


Im Original befinden sich mehr Schaltflächen auf dem Bedienfeld, die jedoch aus offensichtlichen Gründen ausgeschnitten wurden .



Der Filter nach Datum kann auf mehrere Pivot-Tabellen gleichzeitig angewendet werden, sowohl auf einem als auch auf verschiedenen Blättern. Dies ist praktisch, wenn Sie dieselben Zeiträume in mehreren Tabellen gleichzeitig festlegen müssen.



Add-In-Verbindung


Es ist praktisch, das Add-In in einer Kugel zu platzieren. Aus diesem Grund wurde das Kopieren des Bilds in das lokale Verzeichnis abgelehnt, damit Änderungen vorgenommen werden konnten und jeder die aktuelle Version hatte. Aber es liegt an dir.


Das Add-In wird für einen Kalender (Datentabelle) einer bestimmten Struktur geschärft, sodass es unwahrscheinlich ist, dass es ohne Änderungen startet, aber es verspricht klein zu sein. Die Installation von Filtern in der Liste sollte jedoch problemlos funktionieren. Die einzige Einschränkung besteht darin, dass die Installation von Filtern für die Hierarchie nicht unterstützt wird, dies ist jedoch nicht erforderlich.
