Script de configuración de Windows 10

Hace tiempo que quería compartir mi script para automatizar la configuración de Windows 10 (actualmente la versión actual es 18363 ), pero no llegaron todas las manos. Quizás sea útil para alguien total o parcialmente. Por supuesto, será problemático describir todas las configuraciones, pero intentaré resaltar las más importantes.

Si alguien está interesado, bienvenido a cat.

Funciones principales

  • Deshabilitar los servicios de seguimiento de diagnóstico
  • Muchos ajustes de Explorer
  • Consola interactiva
  • Elimine todas las aplicaciones UWP de todas las cuentas, excepto las siguientes ...
  • Deshabilitar componentes
  • Crear una tarea en el Programador de tareas de limpieza de Windows Update
  • Cree una tarea en el Programador de tareas para limpiar la carpeta "$ env: SystemRoot \ SoftwareDistribution \ Download"
  • Agregar carpeta a la lista de exclusión de escaneo de Windows Defender
  • Deshabilitar servicios de usuario
  • Cree un acceso directo de formato antiguo para "Dispositivos e impresoras"
  • Importe el diseño personalizado del menú Inicio desde un archivo de registro preparado
  • Desanclar todos los accesos directos desde la pantalla de inicio
  • Redefina la ubicación de las carpetas "Escritorio", "Documentos", "Descargas", "Música", "Imágenes", "Video"
  • Actualice los iconos del escritorio, las variables de entorno y la barra de tareas sin reiniciar Explorer
  • Mucho mas

NB

  • PowerShell y PowerShell ISE deben ejecutarse con privilegios elevados
  • Establecer la política de ejecución de script de PowerShell adecuada 
Set-ExecutionPolicy -ExecutionPolicy Bypass -Force
