1. Microsoft 365: Atualizando a implantação corporativa usando o Windows 10 e o Office 365


O Microsoft 365 ajuda a criar um ambiente seguro e atualizável, aproveitando os dispositivos Windows 10 que executam aplicativos do Office 365 gerenciados pelo Microsoft Enterprise Mobility + Security.

Este módulo, projetado para 3,5 horas, ensina como trabalhar com o Microsoft 365, fala sobre o básico do uso da ferramenta, bem como sobre segurança e treinamento do usuário.

Saiba mais e comece a treinar neste link .

2. Administrando recursos de infraestrutura no Azure


Saiba como criar, administrar, proteger e dimensionar recursos da máquina virtual na nuvem do Azure. A conclusão de todo o curso leva cerca de 10 horas.

Módulos do Curso:

  • Visão geral das máquinas virtuais do Azure
  • Criando uma máquina virtual Linux no Azure;
  • Crie uma máquina virtual do Windows no Azure;
  • Gerenciar máquinas virtuais usando a CLI do Azure;
  • Atualizando máquinas virtuais;
  • Configurações de rede para máquinas virtuais;
  • Criar modelos do Azure Resource Manager
  • Redimensione e adicione discos nas máquinas virtuais do Azure;
  • Cache e desempenho do armazenamento do Azure
  • Impulsione a proteção para máquinas virtuais do Azure.

Detalhes e início do treinamento

3. Gerenciamento de recursos no Azure


Aprenda a criar, administrar e gerenciar recursos de nuvem por meio da linha de comando e do portal da Web do Azure. A propósito, neste curso, como em muitos outros, você pode se exercitar na caixa de proteção do Azure.

Módulos:

  • Correspondência de requisitos com tipos de nuvem e modelos de serviço no Azure
  • Gerenciar serviços do Azure usando a CLI
  • Automatize tarefas do Azure com scripts do PowerShell;
  • Previsão de custos e otimização de custos para o Azure;
  • Controle e organize os recursos do Azure com o Azure Resource Manager.

Detalhes e início do treinamento

4. Noções básicas do Microsoft 365


O Microsoft 365 é uma solução inteligente que inclui o Office 365, Windows 10 e Enterprise Mobility + Security, permitindo que você colabore de forma criativa em um ambiente seguro. Neste curso de 4 horas sobre tudo, tudo o que você precisa saber para começar a usar o Microsoft 365.

Você aprenderá sobre o que é o Microsoft 365, informações básicas sobre os serviços e recursos do serviço, sobre trabalho em equipe, segurança e recursos em nuvem. A propósito, para o treinamento, você precisa ter pelo menos um conhecimento superficial da computação em nuvem.

Detalhes e início do treinamento

5. Administração de Contêiner no Azure


As instâncias de contêiner do Azure são a maneira mais rápida e fácil de executar contêineres no Azure. Este tutorial ajudará você a aprender como criar e gerenciar contêineres e como obter um dimensionamento flexível para o Kubernetes usando ACI.

Módulos do Curso:

  • Criando um aplicativo Web de contêiner usando o Docker;
  • Crie e armazene imagens de contêiner usando o serviço Registro de Contêiner do Azure;
  • Iniciar contêineres do Docker usando o serviço Instâncias de Contêiner do Azure;
  • Implantar e iniciar um aplicativo de contêiner da Web usando o Serviço de Aplicativo do Azure;
  • Visão geral do Serviço Kubernetes do Azure.

Saiba mais e comece a aprender
