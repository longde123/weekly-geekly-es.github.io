Microsoft Kaizala memungkinkan Indian Railways menghubungkan tiga juta karyawannya dengan layanan kesehatan

Majikan terbesar India, Indian Railways, akan menggunakan Microsoft Kaizala untuk menghubungkan karyawannya di seluruh negeri dengan fasilitas kesehatan berkualitas. Aplikasi Microsoft Kaizala akan memungkinkan melayani dan pensiunan karyawan kereta api untuk memanfaatkan layanan kesehatan dari 125 kereta api dan 133 rumah sakit swasta yang diakui. Kelompok Kaizala, yang dikelola oleh dokter dari South Central Railways akan dilengkapi dengan kelompok dokter yang terfokus, staf paramedis dan perawat.


Pada mendaftar untuk layanan kesehatan, karyawan Kereta Api India akan dapat mencari di Microsoft Kaizala, rumah sakit dan dokter terdekat, daftar pusat diagnostik empaneled dan unit kesehatan. Karyawan dapat memesan janji temu dokter, berbagi laporan lab diagnostik langsung dengan dokter mereka dan menyimpan catatan digital di 'Me Chat' dari Microsoft Kaizala. Mereka juga akan dapat mengakses pengumuman utama, membagikan umpan balik mereka untuk meningkatkan kualitas layanan medis dengan kartu tindakan bawaan.




Kaizala memungkinkan Indian Railways untuk memungkinkan komunikasi langsung dengan karyawan mereka termasuk buletin pencapaian, buletin edukatif dan buletin informatif untuk menyebarkan kesadaran di sekitar CPR, pertolongan pertama umum, imunisasi dan vaksinasi, termasuk yang lain. Para dokter di kelompok Kaizala ini dapat melihat riwayat kesehatan seorang karyawan, mengeluarkan lembar kasus dan bantuan dalam mengambil keputusan tepat waktu. Microsoft Kaizala adalah produk khusus seluler yang memudahkan organisasi untuk berkomunikasi secara mulus, berkolaborasi, dan menyelesaikan tugas, menyatukan pengguna desktop dan pengguna seluler saja.
