参加RAM超频竞赛-HyperX Memory OC Competition 2019

哈Ha！ 我们正在寻找人才，或更确切地说，是最有才能的超频者-对于他们来说，HyperX与超频实验室OCLab.ru一起发起了HyperX Memory OC ompetition 2019竞赛 。 如果您是初学者超频者，并且CAS Latency从RAS到CAS Delay没有什么不同，那么就可以了。 根据我们的指示，您也可以在比赛中尝试自己的运气。 竞争的本质非常简单：HyperX Memory OC ompetition 2019致力于对Kingston和HyperX RAM超频。



比赛目标


由于比赛是超频的，因此参赛者将参加超频RAM。 频率加速的最佳结果将作为标称值的百分比进行比较。 时间，电压和性能无关紧要，主要是从Kingston或HyperX模块中压缩最大频率增加。

如何选择获奖者


根据比赛结果，将对相对于面值的百分比表示的最大加速度的前15个结果进行汇总。

如何参加


我们认为，良好的竞争应完全透明，并防止欺诈。 因此，必须验证超频结果并使用系统的屏幕截图和照片进行备份。

为了进行验证，请下载CPU-Z 1.88和Thaiphoon Burner （免费版本）。

1.首先，相对于标称值实现RAM频率的最大超频。 不要急于发送第一个结果-也许在一两天内您就可以创下新的记录。
2.启动四个CPU-Z副本，并排放置程序窗口，并在每个窗口中打开不同的选项卡：CPU，主板，内存，SPD。 在它们旁边，将Thaiphoon Burner窗口与明信片的“读取”选项卡一起放置。 所有五个窗口应在桌面上清晰可见。 拍摄一般屏幕截图，这是前提条件。



3.验证CPU-Z中的超频结果。 填写参与者的申请表时，需要一个链接。

4.拍摄计算机或测试台的照片，以便可以看到照片中超频的金士顿或HyperX主板。

5.在此处填写参加者的申请表。 除了参考经过验证的超频结果外，您还必须在调查表中指定您的记忆文章。 可以在SPD选项卡上的CPU-Z中找到它-在“零件号”列中指出。 在问卷中，我们不要求提供系统的屏幕截图和照片，但如果您获得其中一个奖励名额，稍后将需要它们。

还将CPU-Z和Thaiphoon Burner的屏幕截图以及计算机的照片附加到调查表中。

6.在OCLab.ru上等待比赛结果！



规则和日期


HyperX Memory OC竞争2019将在2019年4月24日至5月24日（含）期间举行，仅在此期间接受结果。 结果将于2019年5月27日在OCLab.ru网站上发布。

我们的比赛具有必须遵守的规则和条件，才有资格获得奖品。

  • 只有通过对金士顿或HyperX品牌的任何DDR3或DDR4内存进行超频获得的结果才可以参加比赛。 系统中模块的数量无关紧要。 其他品牌的内存以及第一代DDR2和DDR都不参与竞争。
  • 随附的XMP配置文件被视为模块的额定频率。
  • 允许使用任何处理器和母板，更改计时，电压并进行任何冷却。
  • 一位参与者考虑到了一台PC的一项结果。 您可以发送新结果，上一个结果将被取消。
  • 来自俄罗斯和EAEU CU国家的居民可以参加比赛：白俄罗斯，哈萨克斯坦，亚美尼亚或吉尔吉斯斯坦。
  • 您必须遵守“如何参与”部分中列出的所有条件；如果没有带有RAM的计算机的屏幕截图和照片，即使进行了CPU-Z验证，我们也将无法接受您的结果。

如何在CPU-Z中验证结果


选项1
在CPU-Z的“关于”选项卡上，单击“验证”按钮。 在打开的窗口的“提交”字段中，输入您的姓名和电子邮件，然后单击“提交”。 一封带有验证结果链接的信件将发送到您的邮件-这是您需要在参与者的个人资料中指定的内容。



选项2
在CPU-Z的“关于”选项卡上，单击“验证”按钮。 在打开的窗口的“选项”部分中，单击“保存验证文件”。 创建一个名称（仅英文字母）并将文件保存到您的计算机。 打开valid.x86.fr/statistics.php ，创建一个帐户并将结果上传到站点。 参与者的个人资料中需要指向结果的链接。

注意！！！
如果您无法访问valid.x86.fr/statistics.php验证站点并注册结果，则可以通过邮件将验证文件发送到icealex.oclab@oclab.ru。 这封信必须注明你的名字。
