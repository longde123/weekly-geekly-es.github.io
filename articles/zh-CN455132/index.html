文物：飞利浦Velo 1，技术爱好者的噩梦

用英语来说，就是“早期采用者”。 我不确定是否可以将其翻译成俄语，唯一想到的就是“快点-让人们发笑”。 早期采用者是一名技术狂热者，他们希望尽早对自己进行测试，而不必等待解决许多问题和生产初期无法避免的问题。 2007年，从美国带来的此类发烧友锁定了运营商，并使用了非俄罗斯化的首款iPhone。 现在，他们正在购买具有柔性屏幕，掌握3D打印和VR技术的智能手机。


通常，他们会参与对实际以现成和经过测试的设备出售的产品进行Beta测试。 但是没有 首先，您遇到的故障率很高，如果您不走运，请按担保三下的条件更改购买。 然后，您尝试解决致命的缺陷：要么在一个小时内耗尽了电池，要么应用程序出现故障，然后再进行其他操作。 一年后，第二代设备问世，而在一夜之间发生的变化使昂贵的进步顶峰变成了过时的，不合适的越野车共享。 但是你没有抱怨。 也就是说，您抱怨，但仍然很高兴：毕竟，几乎没有人拥有这样的设备，而您却只有一个。

1996年，微软对可穿戴设备市场进行了大规模攻击。 1997年，第一代设备问世-键盘PDA。 其中，在性能和功能以及可能的升级方面脱颖而出。 掌上电脑Philips Velo 1未能逃脱先驱者的诅咒，很快就过时了，坏了很多，理应被遗忘。 由于种种个人原因，我购买了这款PDA，并决定亲自体验一下22年前的移动Windows。

我用电报实时地记录了一个旧铁器收藏家的日记。

打开时间机器


手持式计算机发展的顶峰可以被认为是现代智能手机：仅在21世纪第二个十年，它们才开始以数亿份的价格出售。 上世纪90年代，手机和PDA尚未集成到一个设备中，充其量是可以将其彼此连接的。 PDA最多每年销售不超过数百万。


在1997年，两个设备确实是杰出的。 这是Psion Series 5，这是一种PDA，带有方便的键盘，EPOC32快速专有操作系统和用于开展业务（联系，日历，任务管理）的智能工具。 Palm掌上电脑（当时称为PalmPilot）是没有键盘的设备的第一个成功实现，其使用手写笔输入文本。 一种设备，其功能显然受到限制，是具有最少内存（2兆字节）的计算机的前缀，价格便宜并且可以很好地执行一组有限的任务。 在照片中-我收藏的最早的Palm PDA是1999年的预算模型IIIe，与1997年的模型没有太大不同。


据微软称，当时在台式机操作系统市场上占据着主导地位 ，一台手持计算机应该与大型计算机几乎相同。 因此，对设备制造商的要求是：分辨率为480x240的大屏幕，串行和红外端口，PCMCIA插槽（！），完整的键盘。 Windows 95用户体验几乎已完全重新创建：带有图标，任务栏，“开始”按钮，资源管理器和角落时钟的同一桌面。


据目击者称，Palm（在90年代末-无可争议的市场领导者）和Psion都害怕微软进入市场。 凭借她的营销预算和寻找生产合作伙伴的能力，她可以轻松地使相对较小的竞争对手破产。 两家公司都响应威胁，开始将其操作系统许可给其他公司。 在这两种情况下，许可过程都破坏了公司内部软件和硬件开发人员之间的流畅交互，但这是另一篇文章的主题。 微软于1996年秋季推出了Windows CE 1.0，该技术使便携式炸弹轰动的感觉和令人难以置信的多功能性广受粉丝欢迎。 实际上，它的效果并不理想：到1998年底，Windows CE平台上的所有设备制造商合计仅占据了手持式计算机市场的25％ 。

但这是微观市场的竞争。 如果在1997年，您选择了一种用于处理数据的新设备，那么您可以在Palm（基于Microsoft平台的设备）之间进行选择，您可以购买Psion。 您可以购买便宜的电子记事本，甚至是一分钱的纸质笔记本，而在并非每个人都买得起普通电脑的情况下，大多数人选择了最后一个选择。 CCP-那些愿意为前景不明的人付出很多代价的发烧友的选择。 但是，让我们不要猜测，而是看看设备。 怎么了


是的，很好！ 最初，我购买了不带附件的设备，仅包括扩展坞，没有扩展坞，则无法将PDA与专有端口同步。 但是这个实例几乎是新的。 PDA规格如下：

  • 处理器：Philips MP3910，36.8 MHz，MIPS架构
  • 内存：4 MB RAM，8 MB ROM
  • 屏幕：5.1英寸，单色，带背光，触摸屏，4种灰度，480x240像素
  • 接口和通讯：用于19200 b / c的内置调制解调器的RS232，IrDA，RJ-11，用于扩展卡的两个插槽，用于PCMCIA卡的适配器的V-Module插槽
  • 电池：2节AA电池或电池，CR2032电池用于在电池更换期间进行数据存储，电池寿命长达12小时
  • 尺寸：172x95x32毫米
  • 重量：374克
  • 初始价格：700美元（根据通胀因素调整后为1,100美元）

Microsoft没有成功地强迫制造商严格遵守规范。 例如，由于屏幕更大（600像素而不是水平480像素），HP决定分配其PDA。 飞利浦拒绝将PCMCIA插槽集成到Velo 1中，而是将其移动到底部插件模块中。 由于处理器功能更强大，并且内存可能更快，Velo 1在1997年初赢得了PC Magazine的测试。 没错，那么第一波的所有设备都没有被纳入研究范围。


飞利浦PDA的另一个独特功能是内置调制解调器。 在早期版本中，屏幕附近有一个棘手的连接器，直接将电话线插入其中。 如果您有电话，则可以上网或连接BBS，而无需其他设备。 其他制造商则指望用户在PCMCIA插槽中安装调制解调器，或销售用于连接手机的适配器。


与Psion的第五系列相比，Velo 1主要区别在于“计算器”键盘。 在这些橡胶按钮上打字不能很快完成，但是布局与笔记本电脑完全一致。 因此，例如，在输入西里尔字母时，将字母的一部分放置在陌生和意外的位置没有问题。


飞利浦放弃了内置的PCMCIA之后，为Velo 1提供了两个扩展卡插槽：一个插槽用于增加RAM的数量，第二个插槽可以插入非易失性闪存驱动器。 两种卡的标准是相同的-现在鲜为人知的微型卡 ，它是CF卡和SD / MMC卡的竞争对手。


坦率地说，当我第一次见面时，还不清楚这些设备出了什么问题，为什么它们的销售相对较差？ 出色的台式机MS Word和Excel副本，电子邮件客户端，内置浏览器（Psion和Palm都没有，必须单独购买）。 接龙！ 在90年代后期，Palm试图抵制微软的营销努力。 与合作伙伴和行业专家会面后，他们比较了一些实际任务的设备，例如，我多快能在电话簿中找到一个号码？ 在Palm需要使用触笔进行几次笔触的情况下，Microsoft由于其笨重的界面而需要进行特殊的操作。


飞利浦Velo 1的多功能性很有趣：几乎就像一台笔记本电脑，只有更小巧。 直接的Internet连接，具有处理邮件和Web的能力。 但是请不要忘记，我们谈论的是一个亚盲单色屏幕，内存只有4 MB，它既用于运行程序，又用于存储数据。 从理论上讲，这种设备几乎可以成为主计算机。 实际上-并非如此。


购买老式PDA几周后，我的运气就好了：我在盒子里订购了一个便宜的第二设备，里面有全套配件，但被漏液的电池弄坏了。 必须将其拆解并清除氧化物，否则它将打开一次。 现在该讨论这些早期PDA的硬件可靠性。


我建议您看一下Philips Velo 1所有者网站的存档副本。他在相当程度上描述了该设备的折磨，但是请不要忘记赞美他的新功能和便利性。 由于屏幕“悬空”，他几次不得不在保修期内更换PDA。 显示器的金属支架不可靠，从外壳的塑料部分飞出。 后来这个问题解决了，但又出现了另一个问题-随着时间的流逝，整个机制崩溃了。 因此，我的Velo 1（较新的Velo 1）根本没有固定在打开位置，而是完全折叠了。 从YouTube上的评论来看，这是一个常见问题。 但这还不是全部：触摸屏死机和数据丢失。 “标准”功能是调制解调器或PCMCIA模块的操作模式-它只需几分钟即可吃完电池，而不是规定的10-12小时。

Windows CE 2.0


第二个Velo 1最重要的工件是Windows CE 2.0的升级套件。 飞利浦Velo 1相对较晚地进入广泛销售-在1997年9月。 微软已经在同年11月推出了一个更新。 新版本的OS（ 此处有更多信息）引入了对彩色屏幕的支持，TrueType字体，Microsoft Powerpoint的便携式版本，并扩展了对PCMCIA卡的支持。 首次出现了对无键盘PDA的支持：它们被称为Palm PC，经过激烈的抗议，Palm改名为Palm-size。 第一代设备仅在六个月内就过时了，并不是每个人都可以更新到新版本。 有了新功能，系统要求急剧增加：现在只有2 MB的RAM还不够。


多少钱够了？ 最初，飞利浦似乎不知道该问题的答案。 建议Velo 1所有者购买4 MB的RAM扩展卡。 经过很多次之后，事实证明，需要一个8 MB的卡（考虑到焊接在板上，则为12 MB）。 结果，Windows CE 2.0的升级套件售价为99美元，其中包括两个扩展卡：8 MB RAM和OS本身的ROM。 经过简单的萨满教法（您需要一个我没有的外部电源，并且我不得不以狡猾的方式插入和取出电池才能从外部ROM卡加载），您收到了更新的操作系统。 但不是彩色屏幕。


升级到Windows CE 2.0可使您进入了解PDA的最后阶段：安装破解程序。 WinCE 1.0上的设备在市场上的生存时间非常短，以至于很难在2019年为它们找到兼容的软件。 对于Windows CE 2.0及更高版本上的PDA，McCenter网站仍然可以免费破解 （我要再次感谢他们保存存档）。


要与计算机同步并安装软件，需要H / PC Explorer。 在1999年，它获得了熟悉的名称Microsoft Activesync。 我使用iPaq hx4700进行实验时，已经安装了该程序的第三个版本（最新版本，该版本支持Windows 98）。 使用旧的Velo 1可以正常工作，但是在此之前，我必须阅读说明并在PDA本身的设置中“创建连接”（这就是为什么这样做了！！）。 由于连接通过串行端口进行，因此同步非常慢。 但是，在基于Palm OS的设备上，用西里尔字母进行文档编码时不会出现此类问题。

第一个版本和第二个版本中的Windows CE在Velo 1上运行的速度都相当慢。 通常，启动程序时会延迟几秒钟。 相比之下，当时的Palm OS的运行速度非常快，而经过优化的EPOC32上的Psion则具有快速响应用户命令的良好效果。 Windows CE在最新一代的键盘上感觉最好-处理器低于200兆赫和64兆RAM。 Velo 1是典型的先驱，奠定了通用移动OS的基础。 从理论上讲，他具有很多能力，但由于具备业务所需的特定功能，因此他的管理能力要比当时的竞争对手差一些。 但没有比这更糟的是：现在，所有90年代的手持式计算机都被视为移动的中世纪。


我计划进一步探索Velo 1的可能性：通过调制解调器，甚至可能通过以太网连接到Internet，以选择合适的软件。 我想将其复制成博物馆，以适合有用的东西，尽管以20年前的标准为准。 否则，从各种各样的可能性中，仅剩下一个不变的单人纸牌“ Kosinka”。 同时，在1997年以来我十五个愿望清单上，我在最后一个项目的前面加上了对号。 那时我可以梦想任何事情，因为我仍然买不起任何东西。 但是出于某种原因，我最想获得的科学技术成就是： 笔记本电脑 ，多功能手持计算机， 迷你光盘播放器和数码相机 。 当时照片中的文物的总成本约为3,700美元，而现代货币则接近六千美元。 梦想成真，您只需要稍等一下！
