7 extensões do Chrome para aprender inglês



Eu sou um usuário ativo do Google Chrome. É por isso que elaborei esta lista de extensões úteis que ajudarão os alunos de inglês a aprimorar a escrita, a ortografia, as habilidades de escuta e a ampliar seu vocabulário.

Textly.AI


Assistente de escrita on-line gratuito, uma versão premium estendida também está disponível. Os usuários podem verificar sua redação usando o aplicativo Web com estatísticas avançadas ou usar a extensão do navegador. Após a instalação, a ferramenta o seguirá na Web e sugerirá correções na sua escrita, detectando problemas de gramática, ortografia e estilo. Para aplicar a correção proposta, clique nela.

Sempre ter um exemplo de redação correta é crucial para dominar as habilidades de escrita em inglês, para que a ferramenta seja útil para quem deseja escrever melhor.



O Textly.AI suporta sites populares de redes sociais como Twitter e Facebook, serviços de email como Gmail e plataformas de blog como Medium e WordPress.

Ler e escrever


Uma das melhores extensões para alunos de idiomas, que não é focada apenas no inglês. Principais recursos: tradução de texto para fala e vice-versa, dicionários de texto e imagem, recomendações contextuais. Se você está confuso sobre como a palavra específica soa ou o que ela significa, esta é a ferramenta perfeita para encontrar a resposta.



Duolingo


Esta extensão é útil para aprimoramento de vocabulário. Após o registro, você precisa escolher o idioma de destino, definir suas metas semanais e começar a resolver tarefas. O Duolingo oferece cursos divididos em módulos agrupados para dominar habilidades específicas.



Verificador de Pronúncia


Caso você queira dominar sua pronúncia, além de escrever, ler e vocabulário, essa extensão será muito útil. Sua tecnologia de reconhecimento de voz é um bom anúncio para detectar erros de ortografia. Os usuários podem adicionar seus arquivos de áudio preferidos ou aprender outros idiomas, além do inglês.



ReadLang


Esta extensão alterna a palavra estrangeira com a alternativa em idioma nativo em um clique. Além disso, os usuários podem criar cartões para melhor memorização de frases. O sistema suporta 40 idiomas.



Speakit


Outra ferramenta de treinamento de escuta. Você precisa destacar o texto na página da Web para convertê-lo em fala. A ferramenta suporta mais de 50 idiomas e extremamente útil para o aprendizado de inglês.



Google translate


Surpreendentemente, os desenvolvedores do Google criaram extensões úteis para o serviço Translate. Pode ser aplicado para tradução instantânea de palavras desconhecidas em qualquer lugar da web.
