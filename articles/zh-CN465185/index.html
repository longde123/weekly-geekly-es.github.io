新的视频监控标准：夜间彩色，抗噪，ePoE



Axis在23年前发布了第一台IP摄像机，彻底改变了缓慢发展的监视市场。 Axis Neteye 200无法取代传统的模拟系统，无法以0.1 MP的速度以CIF格式每秒录制一帧或以4CIF格式每17秒录制一帧，但为新型系统-网络视频监控摄像机奠定了基础。

随着云技术的发展，对视频监控的下一波兴趣也随之而来。 Amazon Web Services和Google App Engine在零开始时推出了弹性计算云，这是云计算开发的一个基本里程碑。 已经开始开发用于管理，记录和分析远程数据中心中的视频流的软件。

最后，到2020年，云技术已大大定义了硬件功能，包括具有集成分析功能的摄像头。 有时云还不足以解决业务问题。 客户需要具有更高计算能力的摄像机以及用于嵌入现有系统并进行后续扩展的工具。

此类相机是在我们的参与下创建的。

夜间射击的优点




视频监控系统不应在夜间“褪色”，因此20多年来，摄像机中一直使用透射红外辐射的LED-这并不是事实上的标准。

首先，黑白图像中存在很多像素噪声，一些红外照明的摄像机还广播了光环，该光环在图像周围显示为白色圆形。

其次，即使具有高分辨率的夜视摄像机也无法应付流面部识别和细节传输。 为了打破技术僵局，工程师们开发了可以在弱光下捕捉色彩的传感器。

在装有Ivideon固件的新相机中，出现了星光矩阵，即使在晚上也可以记录彩色图像。 要了解在人工照明下的拍摄质量，请观看视频：



该视频主要介绍了我们的设计模型NBLC-P * 201F-ASD-这些摄像机不会出现在在线商店中，但在为企业客户部署视频监视系统时，我们将使用它们。

在其他装有Starlight的型号（NBLC-6231F，NBLC-3231F，NBLC-3261Z-SD，NBLC-2231Z-SD，NBLC- * 2 * 1Z-ASD）中，与上一代相机相比，照片也会更好。



Starlight矩阵使用带有背光的Starvis（Starlight可见性）技术。 CMOS传感器的灵敏度为每1μm²2000 mV或更高（使用706 cd /m²的光源拍摄的彩色图像，F5.6），在可见光和弱光条件下都可以提供高质量的图像。

传感器的高光敏性归因于光电池背面的导体网格位置。 因此，在光电二极管上收集了更宽范围的光，结果，减少了颗粒感和噪声，并且提高了图像清晰度。

除了矩阵外，传感器的灵敏度和光量，处理器，光圈和图像处理算法还会影响相机在弱光条件下的操作。 因此，这些相机配备了功能强大的处理器Ambarella S2LM（分别为1和3百万像素）和Ambarella S3LM（2和4百万像素）。



通过光圈入射的光越多，图像就越明亮和充满活力。 根据规格，这些型号将使用F1.0，F1.4或F2.0的光圈，以便最大可能的光量通过镜头并落在传感器上。

我们的新设备中的图像亮度是传统相机的4倍，并且显色性得到了显着改善，这在具有人工或自然夜（星，月）照明的物体中尤为明显。

相机将无法在完全黑暗的环境中工作-该线以0.1 lux的亮度通过，这大致相当于满月开放空间中的最低照度。

清晰的声音解决方案




过去，安全摄像机根本无法录制声音。 警卫人员仍无法实时跟踪对话，并且麦克风的质量无法在众多噪声源的情况下进行录音。

为了记录声音质量，尤其是在困难条件下，我们与麦克风制造商Stelberry开发了一种联合解决方案。 现在，Nobelic摄像机不仅配备了外部音频输入，而且还支持AAC编解码器，并且可以根据规格以16位或16 kHz的频率工作，并且还立即进行了与Stelberry麦克风兼容性的测试。

对于面积为50-100平方米的房间，安装两个到四个M-70麦克风和一个MX-300混音器就足够了，总信号将从该信号进入摄像机输入。 通过调节M-70麦克风的灵敏度（增益），可以将聆听区域限制在房间的必要区域，从而减少录音中的噪音量。

M-70麦克风的通带受到270 ... 4000 Hz的语音频率范围的限制，信噪比为63 dB。 这样的声音似乎沉闷不自然。 如果需要更大范围的80 ... 16000 Hz感知频率和自然声音，则使用M-60麦克风-灵敏度也可针对特定聆听区域进行调节。

在.wav中测试M-60和M-70：使用耳机收听。

正确放置麦克风




购买相机和麦克风仅能解决一半难题。 接下来，您需要考虑房间的特点和录音目的的细节，进行设置和适当的安装。

在某些情况下，为了正确记录，建议禁用麦克风中的自动增益控制（AGC）。 在嘈杂的房间中，打开了AGC的麦克风会根据环境噪声进行调整，并将其视为主要的声音来源。 在放大的环境干扰的背景下的通话信号是无法区分的。

吊顶是安装录音设备的不成功之处，因为它会感觉到振动和墙壁声音的多次反射。 在天花板上安装时，需要留出2-4毫米的间隙并安装麦克风，以使麦克风盒的孔与面板上的孔重合。

更糟的是，将麦克风放在吊顶后面。 在这种情况下，设备只能很好地听到天花板下的声音，有用的信号仍然沉闷且难以辨认。

将麦克风放在离地面1.5米的墙壁上可以实现最佳音质。 从麦克风到调音台的信号不会损失质量，可以通过最长300米的电缆传输，也可以通过调音台到DVR或摄像机的电缆传输，而另一条电缆的最长距离为300米。

具有ePoE的扩展系统




借助ePoE（以太网扩展供电）技术，信号可以通过同轴电缆或光纤对电缆的全双工10兆比特通道传输，传输距离超过800米，或者通过100兆比特通道传输，传输距离可达300米。

新的Nobelic型号在本地ePoE网络的所有线路上支持的电源标准显着增加了IP摄像机和交换机之间的距离，并允许您快速构建庞大的视频监控网络。

新设备比当前的PoE标准要好得多，后者仅适用于100米长的线路。 此外，ePoE支持通过网络自动配置设备，并且适合从模拟系统迁移。

Nobelic的ePoE解决方案适合将系统快速部署到大型建筑工地，公园，工厂和其他大型设施。

我们的主要型号已被具有符合整个视频监控行业新标准的功能的Nobelic摄像机所取代：
