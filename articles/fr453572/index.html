Microsoft Edge pour macOS

Le mois dernier, nous avons annoncé la première version préliminaire de la prochaine version de Microsoft Edge pour Windows 10. Aujourd'hui, nous sommes heureux d'annoncer la disponibilité du canal Microsoft Edge Canary pour macOS. Vous pouvez maintenant installer des versions d'aperçu à partir du site Microsoft Edge Insider pour votre PC macOS ou Windows 10, avec plus de prise en charge de la version Windows à venir.



Créer une expérience utilisateur «Mac» pour Microsoft Edge


Microsoft Edge pour macOS offrira la même nouvelle expérience de navigation que nous prévisualisons sur Windows, avec des optimisations de l'expérience utilisateur pour que vous vous sentiez chez vous sur Mac. Nous adaptons l'apparence générale pour correspondre à ce que les utilisateurs de macOS attendent des applications sur cette plate-forme.



Nous nous engageons à créer un navigateur de classe mondiale avec Microsoft Edge grâce à des fonctionnalités d'expérience utilisateur différenciées et des services connectés. Avec cette version initiale, nous avons apporté plusieurs modifications à l'interface utilisateur pour l'aligner sur le langage de conception Microsoft tout en la rendant naturelle sur macOS.


Des exemples de cela incluent un certain nombre de réglages pour correspondre aux conventions macOS pour les polices, les menus, les raccourcis clavier, la casse des titres et d'autres domaines. Vous continuerez de voir l'apparence et la convivialité du navigateur évoluer dans les prochaines versions alors que nous continuons à expérimenter, à itérer et à écouter les commentaires des clients. Nous vous encourageons à partager vos commentaires avec nous en utilisant le smiley «Envoyer des commentaires».



De plus, nous concevons des expériences utilisateur exclusives à macOS, en tirant parti des fonctionnalités matérielles spécifiques disponibles sur Mac. Par exemple, fournir des actions utiles et contextuelles via la barre tactile comme les raccourcis de sites Web, le changement d'onglet et les commandes vidéo, ainsi que permettre une navigation familière avec les gestes du pavé tactile.



Présentation des canaux Microsoft Edge Insider pour macOS


Les nouvelles versions de prévisualisation Microsoft Edge pour macOS sont disponibles via des canaux de prévisualisation que nous appelons «canaux Microsoft Edge Insider». Nous commençons par lancer la chaîne canarienne Microsoft Edge Insider, que vous pouvez télécharger et essayer sur le site Microsoft Edge Insider . Cette chaîne est disponible à partir d'aujourd'hui sur macOS 10.12 et supérieur. Le Dev Channel sortira très bientôt, et une fois disponible, vous pourrez le télécharger et l'installer côte à côte avec le Canary Channel. Vous pouvez en savoir plus sur notre approche et à quoi s'attendre des différents canaux dans notre article de blog du mois dernier .



Une plateforme et des outils cohérents pour les développeurs web


Avec notre nouvelle fondation Chromium, vous pouvez vous attendre à une expérience de rendu cohérente sur les versions Windows et macOS de Microsoft Edge, ainsi que les mêmes outils de développement puissants que vous trouverez sur Windows.


Pour la première fois, les développeurs Web peuvent désormais tester des sites et des applications Web dans Microsoft Edge sur macOS et être convaincus que ces expériences fonctionneront de la même manière dans la prochaine version de Microsoft Edge sur toutes les plateformes. (Notez que les fonctionnalités spécifiques à la plate-forme, comme le déchiffrement de contenu PlayReady sur Windows 10, devraient continuer à être détectées pour une meilleure expérience sur ces plates-formes.)


Comme avec nos versions de prévisualisation Windows, notre nouvelle version de macOS inclut également la prise en charge des applications Web progressives installables basées sur des normes que vous pouvez inspecter et déboguer à l'aide des outils de développement du navigateur. Nous nous efforçons de faire en sorte que les PWA se sentent chez eux aux côtés de vos applications natives.Ainsi, une fois installées, elles apparaîtront dans votre Dock, votre sélecteur d'application et Spotlight comme une application native.


Partager vos commentaires


Nous sommes ravis de partager avec vous notre première version Microsoft Edge Canary pour macOS! Obtenir vos commentaires est une étape importante pour nous aider à améliorer notre navigateur - nous considérons qu'il est essentiel de créer la meilleure expérience de navigation possible sur macOS. Nous espérons que vous allez essayer l'aperçu aujourd'hui , et nous attendons avec impatience vos commentaires et votre participation à la communauté Microsoft Edge Insider.


Si vous rencontrez des problèmes et si vous souhaitez donner votre avis ou partager des suggestions avec l'équipe, rendez-vous sur les forums de la communauté Microsoft Edge Insider , contactez-nous sur Twitter ou utilisez simplement l'option «Envoyer des commentaires» dans le menu Microsoft Edge pour dites-nous ce que vous en pensez.


Pour les développeurs Web, si vous rencontrez un problème qui se reproduit dans Chromium, il est préférable de signaler un bogue Chromium . Pour les problèmes dans la version existante de Microsoft Edge, veuillez continuer à utiliser EdgeHTML Issue Tracker .
