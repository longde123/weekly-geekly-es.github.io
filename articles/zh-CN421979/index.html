德古拉主题-几乎所有事物的通用主题


引言


大家好，曾几何时，我在各种操作系统上处理了大量不同的文本编辑器和IDE。 那时我很生气，因为他们每个人都有自己的代码设计主题，还有其他几个可供选择。 好的和坏的都有，但它们都是不同的，对我来说这是个问题，必须不断适应新的设计。


我开始在Internet上寻找解决此问题的方法。 对于所要寻找的东西，我只有两个条件。


  1. 适用于所有流行的文本编辑器和IDE的单一配色方案。
  2. 这是一个黑暗的话题，因为就我个人而言，在这种设计中编写文本或代码要方便得多。

吸血鬼主题


简短搜索后，我发现了网站draculatheme.com 。

该站点是针对不同应用程序的通用主题-Dracula主题（ MIT许可证 ）。

在这个站点上，我找到了我真正需要的东西。 关于我几乎需要或理论上将来可能需要的所有内容，都有一些主题。 有超过50种受支持的应用程序，因此我认为没有理由列出此出版物中的所有内容，您可以在网站上看到所有这些应用程序。


结果，我在不同的设备和不同的操作系统上安装了Dracula Theme，尤其是在Sublime Text，Notepad ++，Terminal，Vim上，也许我已经忘记了一些东西，但这不是重点。 为了娱乐起见，我在Telegram和Slack上安装了相同的主题，尽管仅更改了侧边栏。 以下是Dracula主题在Sublime Text上的屏幕截图。



安装方式


安装Dracula主题很简单，在网站上针对每个应用程序都有安装主题的说明。 您还可以为GitHub开发人员为每个应用程序克隆或派生具有主题的文件。 下面是克隆Sublime Text主题的示例。


$ git clone https://github.com/dracula/sublime.git "Dracula Color Scheme"


如果您不想从GitHub克隆，则只需从开发者网站上的链接下载存档。
