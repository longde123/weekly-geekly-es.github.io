小米Redmi注意事项4：价格合理的智能板

Redmi Note 4是小米今年的第十一款智能手机。制造商再次遵循“非常好，非常便宜”的原则：在中国，Redmi Note 4以135美元的价格开始销售。Remdi Note系列的第四代与上一代类似，但总体上模型得到了大幅改进：新颖之处在于它配备了更新的相机，全金属机身和新平台-Redmi Note 4内部使用了十核处理器。

从外观上看，在小米Redmi Note 4中很难识别低价类别的设备（内部也是如此，但稍后会更多）：智能手机被装入没有塑料插件的一体式金属外壳中，后面板的边缘弯曲。它上面有用于天线的窄盖，以及通常位于中央的圆形摄像头，闪光灯和指纹扫描仪。

智能手机的设计包括清晰边缘和平滑线条的非常优雅的组合。尽管尺寸很大，但箱子背面的圆形使您可以舒适地握在手中。由于紧密的组装和高品质的材料，小工具被认为是昂贵和重要的物品。



覆盖智能手机前面板的玻璃在边缘处略微弯曲，从而增强了坚固感。屏幕下方是硬件按钮“菜单”，“后退”和“主页”。它们配备了柔和的背光和振动反馈。



小米Redmi Note 4的情况是不可分割的。用专用回形针从左侧边缘取下SIM卡插槽。结合使用，可以安装microSIM和nanoSIM或microSIM和存储卡。



指纹扫描仪


位于小米Redmi Note 4后面板上的指纹扫描仪的特点是响应速度快：从用手指触摸圆形触摸面板到显示主屏幕几乎需要一秒钟的时间。屏幕锁定时，扫描仪也可以工作-要打开智能手机，只需将手指放在上面即可。指纹注册不需要花费很多时间-要记住一根手指，系统需要大约七到十次触摸。通常，该技术已很好地实施，可以在使用小工具时大大节省时间。考虑到设备的大对角线，唯一的事情是，始终不能始终将手指清晰地放在扫描仪上（事实证明它是用在平板的侧面），这有时会导致操作错误。



萤幕


小米Redmi Note 4的显示屏周围有一个黑色的小边框，但绝对不会干扰感知-它的宽度只有几毫米，屏幕本身很大，只有5.5英寸。分辨率FulHD可提供非常清晰的图像，没有易碎性和明显的个别问题。最大亮度足以在街道上工作，此外还有一个快速且适当触发的自动调整功能。



不能将视角称为理想角度：倾斜时图像会褪色，但仍然可以区分。在设置中，可以调整屏幕的色温，以及选择三种显色模式之一-标准，协定和自动（涉及根据照明优化图像）。



介面


小米Redmi Note 4界面的基础是带有已安装的MIUI 7.3固件的Android 6.0。没有提供俄语，只有中文和英文。简单的实用程序会按英语进行排序-时钟，计算器，便笺，录音机，播放器等，只要使用最少的英语即可毫无问题地使用它们。但是主题和应用程序商店完全是中文的，约占预装软件的三分之一。可以在设置中删除所有不需要的（或至少不可理解的），并可以通过专有的App Store下载Google Play，以安装所有必需的应用程序。



设备正在运行


小米Redmi Note 4收到了一个平台，尽管它不是来自顶级制造商的，但功能强大-这是十核SoC MediaTek Helio X20。内核的最大时钟频率为2.1 GHz。在我们的测试中，我们有一个具有64 GB驱动器和3 GB RAM 的实例，另外还有16 GB和2 GB版本出售。在AnTuTu中，智能手机“敲响”了一个相当严重的成绩-超过83000点。结果不是旗舰产品，而是大大高于平均价格段。


基准AnTuTu和PCMark

为了测试游戏中的性能，我们使用了《坦克大战突击世界》坦克大战模拟器。其中，在较低的图形设置下，fps值从40到55-60不等，外壳略微发热。在最大设置下，fps下降到15-17，因此动画看起来像撕裂了。最有可能的是，除非始终具有最大的图形效果，否则大多数玩具都会充分使用Redmi Note 4。



小米Redmi Note 4可以应付电话功能：语音传输质量没有问题，通话过程中扬声器的声音很大。智能手机支持LTE，数据传输速度高。GPS在短短几秒钟内启动并连接到卫星。



摄影机


小米Redmi Note 4收到的相机分辨率为13 MP-在Redmi 3 Pro中小于16 MP，但效果更好。



基本上，相机会快速对焦，但是在微距摄影过程中，自动对焦的速度会降低几倍。图像很酷：背景模糊，物体清晰，在大多数情况下，阴影正确。也许，小米Redmi Note 4不仅可以“竞争”该细分市场的竞争对手，而且可以“造价”更昂贵的设备。在人工照明下，颜色有时会失真，但总的来说，一切都很好。夜间拍摄要充分暴露，并且不会受到噪音的影响。最后，该应用程序具有手动模式，可让您调整ISO和白平衡。
























开启和关闭HDR拍摄同一场景。此模式有助于消除高光和太深的阴影，并扩展图像的动态范围。

离线工作


小米Redmi Note 4的电池容量非常出色-4100 mAh-但耗能高的铁不允许设置自主记录。尽管如此，新奇事物还是慢慢消失了：Redmi Note 4持续了七个半小时，以最大的屏幕亮度连续播放视频。根据PCMark电池测试，该电池可连续工作六个半小时。这是高于市场平均水平的令人满意的结果。如果您愿意，全时电池就足够了，您可以将其“拉伸”两天或更长时间（取决于使用情况）。


总结


小米Redmi Note 4与其前身Redmi Note 3 Pro非常相似。为什么要购买新型号？原因有很多：这是更强大的处理器，更新的系统（第六代Android而不是第五代），内存增加和显着改进的相机。最终，Redmi Note 4的设计变得更加平衡，具有圆形玻璃和没有塑料内衬的表壳。同时，Redmi Note 4继承了过去模型的所有优点-这是一个整体式的组件，因此小工具给人留下了昂贵而高品质的感觉；良好的自主性和出色的屏幕，具有大对角线和高像素密度。毫无疑问，这款智能手机值得一试。造成不便的原因仅在于缺少俄语，但首先，将来会出现本地固件，其次，以如此低的成本，没有什么可以阻止您自己刷新设备的。
