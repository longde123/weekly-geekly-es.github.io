Soporte de Python en Power BI

Nos complace anunciar que Python, un lenguaje de programación ampliamente utilizado por estadísticos, científicos y analistas, ahora está integrado en nuestro escritorio Power BI gratuito. Después de habilitar la función en la configuración, puede usar Python para limpiar, analizar y visualizar los datos. Más detalles debajo del corte!



La primera característica de integración que agregamos es la capacidad de ejecutar scripts de Python directamente en Power BI Desktop para crear un modelo de datos. Puede hacerlo en el cuadro de diálogo Obtener datos.



Después de seleccionar esta opción, puede insertar su secuencia de comandos.



La integración con el editor de consultas le permite limpiar datos utilizando Python y utilizar métodos avanzados para analizar sus datos, que incluyen, por ejemplo, eliminar misiones, pronósticos y agrupaciones.

Finalmente, puede usar Python para crear nuevos efectos visuales en su informe. Al igual que la visualización en R, los efectos visuales de Python se actualizarán con actualización de datos y filtrado cruzado. Sin embargo, lo visual en sí aún no es interactivo.
