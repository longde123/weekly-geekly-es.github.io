In England werden die "elektrischen Straßen" mit kabellosen Ladeautos erlebt



Die britische Highway England Highway Agency plant, das drahtlose Laden von Fahrzeugen über die Straße zu testen. Die entsprechende Pressemitteilung wird auf der Website der Agentur veröffentlicht.

Wenn das Experiment erfolgreich ist, werden die Elektroden auf den größten Autobahnen der Klasse A montiert.

Die Agentur hat bereits eine Machbarkeitsstudie zu den Technologien des „dynamischen drahtlosen Energietransfers“ durchgeführt. Jetzt ist die Zeit für Feldversuche. Es wurde eine Ausschreibung angekündigt, an der Handelsunternehmen teilnehmen können.

Großbritannien ist nicht das erste Land, das elektrische Kabel für das drahtlose Laden von Elektrofahrzeugen auf dem Bürgersteig verlegt. Im Jahr 2013 wurde in der südkoreanischen Stadt Kumi eine 12 km lange Buslinie auf diese Weise mit der SMFIR-Technologie ( Shaped Magnetic Field In Resonance ) ausgestattet, schreibt BBC.


Geformtes Magnetfeld in Resonanz (SMFIR)



Elektrische Kabel unter dem Asphalt erzeugen ein elektromagnetisches Feld, das auf die Spule im Fahrzeug wirkt, in der Strom erzeugt wird. Für den effektiven Betrieb solcher Punktladeplattformen ist es jedoch wünschenswert, dass der Bus oder das Auto einige Minuten hier bleibt. Dies ist eine gute Option für Bushaltestellen. Mit einer Strecke können wir die Position der Standorte an Kreuzungen mit Ampeln annehmen.

Anscheinend werden sie in England eine andere Methode der drahtlosen Übertragung von Elektrizität verwenden, damit die Batterien des Autos unterwegs aufgeladen werden können. Aus praktischer Sicht ist dies bequemer.

Vertreter von Highways England erwarten, 2016 oder 2017 Feldtests zum drahtlosen Laden durchzuführen, jedoch nicht auf allgemeinen Straßen, sondern vorerst auf einem separaten Trainingsgelände. Der Testort muss noch ausgewählt werden. Das Experiment wird ungefähr anderthalb Jahre dauern, bevor die Behörden eine endgültige Entscheidung über die Möglichkeit von Tests auf realen Straßen treffen.

"Die Möglichkeit, Elektroautos unterwegs aufzuladen, bietet aufregende Perspektiven", sagte Andrew Jones, britischer Verkehrsminister, auf Initiative.

Das Projekt hat zwar auch Kritiker. Sie sagen, dass die Kosten für die Verlegung der elektrischen Infrastruktur und die Kosten für Strom die wirtschaftlichen Vorteile von Elektrofahrzeugen zunichte machen. Aber auch Kritiker geben zu: Es ist immer noch sinnvoll, die Technologie zu testen.

Auch wenn das Experiment mit drahtlosem Laden wirtschaftliche Unpraktikabilität zeigt, wird Highways England den Plan zur Installation von „Tankstellen“ für Elektrofahrzeuge auf Autobahnen alle 32,1 km nicht aufgeben.
