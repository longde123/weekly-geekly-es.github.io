6. Menyegel sistem file root Raspberry Pi hanya-baca

Sekarang kami akhirnya sampai pada hal utama.

Di bawah ini adalah skrip yang harus ditempatkan dengan nama mount_unionfsdi suatu tempat di /usr/local/bin. Ingatlah untuk mengaktifkan bit kelayakan + x ( chmod755 atau 555). Perhatikan akhiran .origdan .rw, mereka harus bertepatan dengan persiapan (lebih lanjut):

#!/bin/sh
DIR=$1
ROOT_MOUNT=$(awk '$2=="/" {print substr($4,1,2)}' < /etc/fstab)
if [ $ROOT_MOUNT = "rw" ]; then
	/bin/mount --bind ${DIR}.orig ${DIR}
else
	/bin/mount -t tmpfs ramdisk ${DIR}.rw
	/usr/bin/unionfs-fuse -o cow,allow_other,suid,dev,nonempty ${DIR}.rw=RW:${DIR}.orig=RO ${DIR}
fi
