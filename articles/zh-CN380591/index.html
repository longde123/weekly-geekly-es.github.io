红外线温度计是为聪明的人准备的设备。第1部分

温度是用于测量的最重要的物理量，已经设计了许多方法。本文介绍了一种使用MT4004红外温度计测量温度的非接触式方法。
一方面，该设备的操作非常简单：将温度计对准对象，然后按按钮-您将获得结果，梦想成真！因此唱了小组“技术”。
但是你为什么不开心？



这是因为温度计的所有者必须具有一定的知识才能正确确定（甚至更好-比较）温度并熟练使用该设备。温度计将有助于降低其成本，甚至可以挽救生命。

响尾蛇有两种受体，其红外敏感性高于人类，这使它们能够在完全黑暗的环境中狩猎。

为了扩展其远程测量温度的能力，一个人发明了红外温度计，其中之一就是MT4004型号，该型号可以快速测量表面温度。



要进行测量，请按指示器旁边的“ ON”电源按钮。
短按一下，温度计将进行测量并记录结果15秒钟-直到设备关闭为止，这对于确定难以到达的地方的温度非常方便。在关闭之前，指示器会显示铭文“ oFF”。
如果持续按下“ ON”按钮，温度计将以每秒两次的频率切换到连续测量模式。选定的速度可以轻松读取更新的读数。
通常，设备有两个按钮。第二个是“ C / F”，位于外壳背面，隐藏在设备的肠子中。可以通过带有牙签或稍微尖头的火柴的身体上的孔来访问它。该按钮允许您以摄氏度或华氏度显示温度。要切换显示模式，请先按一下“ ON”按钮打开温度计，然后再按“ C / F”按钮。在我们的国家中，实际上不使用华氏度。
温度计的工作范围是-27.4 ... +428华氏度（-33 ... +220摄氏度），因此无法拍摄电影“ 451华氏度 ”。对于纸燃烧器，仪器会写上“ Hi”，表示超出了测量上限。
在南极，设备将写入“ Lo”。
在两次测试中，发现“ Er2”字样-温度计切断了温度为650摄氏度的空气流，而温度计离气体舒适灯太近。也许这是一个过热的传感器。题词“ Er1”必须存在是合乎逻辑的，但对我而言似乎并不存在（也许更好）。
