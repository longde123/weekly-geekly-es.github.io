是什么使新的UCS C480 ML M5变得有趣-思科的机器学习服务器

今年三月，思科推出了思科UCS B480 M5服务器刀片。 基于此系统，我们在圣彼得堡部署了一个新的资源池，并准备在莫斯科启动一个类似的资源池。 但是仅六个月后（9月），这家IT巨头发布了另一项新产品-UCS C480 ML M5，该产品“锐化”了用于人工智能系统的工作。 我们告诉内部内容，为什么要全部发送给数据中心。


/维基媒体/ Tibigc / CC

里面有什么


该公司的新发展是C480 M5服务器的修改版本。 为了加快机器学习算法的处理速度，该服务器中 CPU的两个插槽已替换为GPU的插槽。 NVIDIA的Tesla V100-32G图形加速器放置在这些插槽中。

卡总数达到八张。 每个加速器的性能比用于深度学习任务的传统处理器的性能高 47倍。 数据通过内置的NVLink接口进行交换-每个GPU为200 GB / s。

C480 ML M5 还具有两个Intel Xeon处理器，每个处理器最多支持28个内核和3 TB RAM。 同时，最多可以通过SAS或SATA接口将24个HDD或SSD连接到服务器。

为什么需要新服务器


根据思科代表的说法，服务器应在机器学习模型开发的所有阶段简化工作：从收集和分析数据到输出处理结果。 同时，它适合在多云环境中使用流水线应用程序。 这些是Hadoop和Spark框架，以及TensorFlow和PyTorch。 此外，思科正在与Anaconda平台协商合作伙伴关系，并与Kubeflow和Google合作。

“新的ML M5将简化针对云提供商和拥有数据中心的公司的人工智能系统的工作，” 1cloud的开发主管Sergey Belkin 说道 。 -机器学习技术可用于评估数据中心内发生的所有过程。 这些应用程序已经存在于思科生态系统中。 例如，Tetration Analytics从软件和硬件传感器收集遥测数据，并使用MO方法对其进行处理。”

思科建议到2019年，将有70％的黑客攻击将使用流量加密，因此，他们认为，新服务器的附加功能也将在保护数据中心的IT基础架构方面发挥重要作用。 为此，IT巨头还拥有自己的解决方案：加密流量分析。 该系统识别基础结构中的漏洞，甚至检测加密流量中的恶意模式。


/维基媒体/ 全球访问点 / PD

请注意，C480 ML M5的发布将影响处理大量数据（例如医疗和财务）的组织的工作。 这些组织中的分析师和开发人员从大数据中提取有用的信息将更加容易。 新的思科服务器将加快处理大量信息和查找模式的过程（例如，在分析医学图像时）。

根据思科的说法，全球80％的企业已经实现了基于人工智能技术的服务，或者将在2020年实现此类解决方案。 到2035年，MO技术将有助于将组织的生产率提高40％。

未来的前景和计划


Gartner研究总监Chirag Dekate认为，新老客户将对C480 ML M5有所需求。 同时，合作伙伴关系总监Scott Mohr 表示 ，思科将继续改善其神经网络服务解决方案，而客户体验将在此过程中发挥重要作用。 因此，现在新产品正在由五十个思科合作伙伴进行测试。

之后，该系统将提供给公司的所有认证合作伙伴。 尽管正在进行测试，但尚无法说明新服务器的有效性。 思科专家（与同一个NetApp不同）决定不提供Resnet或Alexnet性能测试的结果，因为据他们称，它们无法反映从事真正的机器学习项目时的状态。

订单C480 ML M5将于今年年底上市。
