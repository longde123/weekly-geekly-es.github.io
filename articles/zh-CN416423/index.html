俄罗斯人如何想到美国处理器，或对AdvoCam-FD8 Gold-II DVR（GPS + GLONASS）的评论

去年，我在这里谈到了用于autoreg Ambarella A12的先进处理器。 更确切地说，没有尴尬阴影的中国DVR制造商正在欺骗国内驾车者，并且在具有最新规格的酷炫技术旗舰的支持下，他们提供质量低劣的错误DVR。 没什么新鲜的-没有稳定的，完善的固件和直接的工程师，这种“硬件”不会带来任何优势。 中国欺诈始于2015年底，目前仍在持续。 俄罗斯品牌AdvoCam一直以来都与Ambarella紧密合作，并出于对被践踏的中国人的不满而小心翼翼地咬牙切齿， “完成”了自己的产品。 经过约四年的开发，于2018年5月发布了AdvoCam-FD8 Gold-II（GPS + GLONASS） 。



这是世界上第一个完全释放Ambarella A12A55处理器潜力的DVR。 该模型的价格为10,000卢布，在竞争对手的水平上为20,000卢布，即两倍。 此外，AdvoCam型号还以其Wi-Fi支持，高达256 GB的存储卡以及先进的Speedcam（用于高速摄像机和其他重要物体的警告系统）而著称，并具有独特的“跟踪”和“城市”模式分隔。

套餐捆绑


我没有发现带有AdvoCam-FD8 Gold-II（GPS + GLONASS）的全新产品。 在这里，您可以看到与AdvoCam通常相同的miniUSB电缆，电缆钩，电源适配器，视频CD播放器（不是软盘-谢谢您），驾车人士法律保护学院院长，著名电视和广播节目主持人Viktor Travin的说明和手册。 我建议不要将宣传册放在夹层中，而是将其放在乘客舱中。 在其中，Travin非常容易地解释了发生事故时如何处理注册服务商的记录。 怎么了 事件发生后，如果您不慌不忙，则以简单易懂的语言列出的提示将非常有帮助。







固定与安装


AdvoCam-FD8 Gold-II（GPS + GLONASS）使用带有真空吸盘的支架安装在挡风玻璃上。 整个安装过程不会超过一分钟：对玻璃杯和吸盘表面进行脱脂（理想情况下，尽管要直言不讳-通常没人这样做）-将带有吸盘的支架压到玻璃上-旋转锁直到听到喀嗒声-将录音机连接到支架。 旋转安装座使您不仅可以将摄像机旋转到侧面，还可以将其引导到乘客舱中。 您永远不知道将需要删除的内容-例如，如果您是出租车司机，则乘客不足的行为。



GLONASS / GPS接收器内置在支架中，而不是记录器盒中。 一种有效的解决方案，可以将敏感的无线电接收机与录音机“内部”造成的干扰隔离开来。 否则，您将不得不屏蔽外壳中的电子设备，这无疑会增加注册服务商的价格。



该设备另一个无可争议的优点是，电源不是直接而是通过安装架提供给记录仪。 由于不需要每次都连接/拔下电缆，因此极大地简化了摄像机的安装和拆卸过程。 顺便说一下，在这种情况下，miniUSB插座的使用寿命会更长，因为不会因频繁的机械影响而松动。

AdvoCam-FD8 Gold-II（GPS + GLONASS）的电源通过三米长的电缆供电（即使在卡车或公共汽车的驾驶室中也足够铺设），该电缆配有点烟器的适配器。 后者带有制造商的轻巧手，配有USB连接器，可为智能手机或平板电脑充电。 不应期望从1 A的任何iPad快速充电，但至少要稍微增加智能手机电池适配器能够充电的百分比。



外表


AdvoCam并未对该设计进行试验-这是同一品牌的带有圆形边缘的加长表壳，是FD8系列所有型号的传统表壳。 顺便说一句，金色的正面插入物-金系列录音机的标志-也存在。 让我提醒您，在FD8系列中，还有AdvoCam-FD8黑色GPS（镜头周围的黑色插页）和AdvoCam-FD8 Red-II（GPS + GLONASS）（红色插页）。 也有AdvoCam-FD8 Profi-GPS Blue（蓝色插件），但由于销售不佳，很快就停产了。 原因是平庸的-注册商的国内购买者（其中大多数是男性）对这种装置带有负面的色彩印象，即同性恋。



带有金色装饰的表壳设计也不是每个人都喜欢的-有些用户认为金色镶嵌完全是吉普赛人，只能看到路易威登图案的模型。 认真地说，我已经在评论中看到了这张照片。



但是，与流行的看法相反，颜色叠加根本不是装饰或热爱所有闪闪发光的金色元素。 假设检查员和运动中希望与您对话的其他参与者都会注意到这种生动的“沉默的见证人”，这也许会在某种程度上减轻他们的热情。 当然，最好不要在没有汽车时将“炫彩”设备留在车内，以免引起不负责任的市民窃取设备的愿望。

设备背面是一个大型2.7英寸显示屏。 作为替代，Wi-Fi不会受到伤害，尤其是因为它不会占用额外的空间。 由于长宽比为16：9，因此显示屏（及其外壳）被拉长并且高度很小。 屏幕左右两侧是双控制键。 左侧的按钮可让您打开和关闭屏幕和麦克风，以及浏览菜单项。 右侧的按钮为“确定”，开始/停止记录以及手动紧急文件标记（以防止覆盖和删除）。 通常，该控制通常适用于AdvoCam设备，甚至适用于按键配置相似的注册商。



顶部有一些按钮，用于打开/关闭电源，背光和即时照片，以及AV输出，miniUSB端口和安装垫。





镜头右侧是microSD插槽。 支持容量高达256 GB的卡，这在注册服务商市场上很少见（最大-64 GB）。 256 GB的内存容量足以以2560 x 1440像素的最大分辨率连续一天进行记录。



左侧是一个HDMI连接器，用于在计算机或电视屏幕上观看录像。 顺便说一句，我想从读者那里得知-您或您的朋友是否至少在录音机中使用过此连接器一次？ 老实说，在我看来，HDMI的真正优势大致在LED背光方面。 就是没有



设置和高级功能


电池容量-250 mAh，足以续航15分钟。 像其前身AdvoCam-FD Black-II GPS + GLONASS和AdvoCam-FD8 RED-II一样，新产品具有一项非常有趣的功能。 在负温度下，电池自动充电被阻止，从而避免了温度变化对电池的破坏作用。 尤其是在冬天，当驾驶员将摄像机留在机舱中过夜，以及在寒冷的夜晚之后，他们会在温暖的机舱中充电，对记录仪进行“热烈欢迎”。 在这里，我们成为了自动注册服务商拥有者转向服务的首要原因的主要见证者-电池的“死亡”。


因温差而膨胀的电池昂贵的台湾DVR Mio

DVR的另一个功能是速度，而今天，您不会感到惊讶。 只有在新的AdvoCam中，它才得到认真的开发。 该设备仍然警告接近固定交通警察雷达，行人过路处，红灯摄像机，危险十字路口等，总共支持12种物体。 但是，所有这一切都增加了划分为两种模式：“城市”和“跟踪”。
这就是为什么在速度解释中需要它。 在城市道路和城市间高速公路上行驶时，汽车的速度变化很大。 显然，要在城市外平稳地将速度降低到特定站点所允许的速度，将需要更多时间。 在设置中，您可以设置“城市”的通知距离-300、500或700米，以及“路线”的通知距离-1000、1200、1500米。 长期以来，对于任何混合动力或雷达探测器来说，这种分离都是必须的。 通常，出于相同的目的-提供不同的警告距离。 但这是我第一次看到“单独”注册服务商，它是在新的AdvoCam中。



这篇评论的英雄是第一个支持Wi-Fi的AdvoCam型号（已停产的混合录像机和运动相机除外-AdvoCam-FD Sport）。 要使用无线通信，您首先需要下载WiFi-CarDV移动应用程序（可在Google Play和App Store上找到，注册商的书面说明中甚至还有QR码）。 然后，从智能手机的网络列表中选择项目ADVOCAM-XXXX，输入密码“ 123456789”并连接。 设备默认情况下未启用Wi-Fi，需要手动激活。 顺便说一下，价格为15,000卢布的高级韩国Thinkware注册商也可以根据该计划工作。 显然，开发人员正确地认为不需要经常保持连接。 而且，如果用户紧急需要远程访问记录，那么将智能手机与注册服务商“连接”起来只需数十秒。

使用该应用程序应该不会造成任何困难-它尽可能简单。 摄像机的广播直接进入主屏幕，下部有三个红色按钮-“视频表”（查看照片和视频材料），“屏幕截图”和“开始/停止拍摄”。





录像机设置列表按钮位于屏幕的右上角。 但是在它的帮助下，仅可以配置基本参数，例如视频录制的分辨率或视频剪辑的持续时间（1、3或5分钟）。 但是我对远程观看视频感到很满意，它几乎立即开始播放，剪辑之间的切换也很快。

开发人员找到了窍门-实际上，注册服务商可以在两个视频流中工作。 第一种是在设置中指定的原始分辨率。 第二个流是720p，保证此类视频无需任何现代智能手机上的Wi-Fi就可以在没有Wi-Fi的情况下开始播放。 而且它们不会减慢速度，包括因为比特率非常低-只有2500 Kbps。 比重型1440p滚筒低十倍。



但是，将视频资料（不仅仅是观看，即传输）转移到智能手机的内存中将花费一些时间-将一分钟的视频复制到1440p最多需要60秒（这对于几乎所有启用了Wi-Fi的录像机都是典型的）。 我不能说直接从注册商卡上流式传输视频是新产品的独特功能，但在Wi-Fi刻录机中，甚至在高级韩国人中，这种情况确实很少见。

当然，新产品中还包含G传感器（冲击传感器）。 并且开发人员提供了新的AdvoCam和LDWS功能。 借助此系统，注册服务商可以通过发出有关车辆离开当前车道的声音信号来节省驾驶员不小心睡在方向盘上的情况。 只是不要忘记，对于此功能的正常运行，您需要一个清洁，平坦，光线充足且带有清晰标记的道路。 没有雪，水和污垢。 因此，实践证明，LDWS主要在夏季和白天工作。

铁和视频拍摄


AdvoCam系列的新代表的视频部分由六元素透镜和带着色玻璃制成的玻璃透镜代表，旗舰产品OmniVision OV4689 CMOS传感器（美国）和上述美国顶级处理器Ambarella A12A55代表。 我再说一遍，在弗拉基米尔州亚历山大市自己的AdvoCam生产中，国内AdvoCam工程师很好地“融合”了注册服务商的“内容”。 结果，他们充分揭示了先进的Ambarella A12芯片组的所有功能。 视角为对角线135度，在帧中可见四个带，沿着图像的周边没有任何失真。

制造商提供以下视频分辨率选项：

2560 x 1440、30 fps；
2560 x 1440，30 fps（HDR）;
2304 x 1296，30k / s;
1920 x 1080、30 fps；
1920 x 1080、60 fps；
1920 x 1080、30 fps（HDR）;
1280 x 720、60帧/秒

由于采用了新处理器，因此无需在使用HDR的1296p和1080p的超清晰拍摄之间进行选择（与Ambarella A7注册商一样）-分辨率最高的Quad Full HD 2560 x 1440像素也支持HDR拍摄。 在恶劣的条件下（例如光线不足或阳光直射）录制视频时，这是一个重要的优势。 HDR将具有不同曝光度的两个帧合为一体，此后即使在黑暗和阳光直射下也能获得更好的细节。 随着照明和汽车号码照明的变化，我们今天监控的汽车小工具也可以“完美地”应对。







最重要的是新的AdvoCam-FD8 Gold-II（GPS + GLONASS）。 在拍摄质量方面，以我相当有经验的观点，AdvoCam-FD8 Gold-II（GPS + GLONASS）在价格范围高达20,000卢布的情况下胜过任何竞争对手。 在设备本身的价格，我想提醒您10,000卢布。 只有高级韩国人才能平等竞争或绕过新的俄罗斯产品。

下午，这张照片令人赞叹不已-清晰，明亮，色彩丰富，光线最少，整个视场的对比度均匀。 车牌的可读性也是完整的，至少可以说-在白天拍摄时，汽车的车牌与16-17米之间是可区分的，在夜间-与10-12米之间是可以区分的。 其他品牌的注册商通常会以至少3-5米的低价售出10,000卢布。

在夜间拍摄中，AdvoCam-FD8 Gold-II（GPS + GLONASS）也被证明是值得的。 正如我已经指出的，车牌的可见度沿直线方向最长可达12米。 图片清晰，清晰，照亮的牌照号码最少。







当然，我不能没有比较测试。 上次，我清楚地表明，即使是基于上一代Ambarella A7LA70处理器的高端AdvoCam，其拍摄效果也比膝盖上的中国Dome DAB202更好（我记得-Ambarella A12上的首批注册商之一）。 这次，对手将再次成为我们东部邻国的产品-Deshman RayBerry E2，除了规格中的Ambarella A12A55外，它还吸引了高速（F / 1.8）七镜头。



顺便说一句，应该对中国人的这种特征持怀疑态度-他们真的很喜欢说谎。 实际上，谁将以及如何验证这些特性的准确性？ 展望未来，我会说这话让我分心，只是因为最后我看不到与镜头的外观特征相符的图片。 另一件事也是如此：由于缺乏工程师在固件上的工作，并且没有设置整个“处理器矩阵镜头”视频路径，因此即使是高质量的快速“眼睛”与一些廉价的塑料镜头类似物之间，也没有明显的特别区别。

我使用HDR以1440p的分辨率进行了所有比较。 让我们看一下结果。
