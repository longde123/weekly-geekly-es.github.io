Expressões regulares


Uma expressão regular é usada para verificar se uma sequência corresponde a um padrão. Uma expressão regular (expressão regular ou regex ou regexp) é uma sequência de caracteres que define um padrão. Um padrão pode consistir em literais, números, caracteres, operadores ou construções. O padrão é usado para encontrar correspondências em uma sequência ou arquivo.
Expressões regulares são frequentemente usadas ao verificar entradas, analisar e localizar cadeias. Por exemplo, verificando uma data de nascimento válida, número de segurança social, nome completo, no qual o nome e o sobrenome são separados por vírgula, pesquisando o número de ocorrências de uma substring, substituindo substrings, formatos de data, formatos de email válidos, formato de moeda e assim por diante.

No .NET, a classe Regex representa um mecanismo de regex. Ele pode ser usado para analisar rapidamente grandes quantidades de texto para encontrar padrões de caracteres específicos, extrair, editar, substituir ou excluir substrings de texto e adicionar as strings extraídas à coleção para gerar um relatório.

Expressões regulares em c #


Expressões regulares são o padrão de correspondência de padrões para analisar e alterar cadeias de caracteres e permitem ao usuário expressar como um programa de computador deve procurar o padrão especificado no texto e, em seguida, o que deve fazer quando cada correspondência for encontrada com o padrão fornecido. Às vezes, eles são abreviados como "regex". Eles são uma maneira poderosa de encontrar e modificar seqüências de caracteres que assumem um formato específico.

Regex para verificação de e-mail


Para testar vários endereços de email, podemos usar as seguintes expressões regulares. Separamos os endereços usando o delimitador ';'

^((\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*)\s*[;]{0,1}\s*)+$

Se você deseja usar o delimitador ',' use o seguinte

^((\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*)\s*[,]{0,1}\s*)+$

e se você deseja usar os dois delimitadores ',' e ';' então use isso

^((\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*)\s*[;,.]{0,1}\s*)+$

Assim, usando a expressão regular acima, você pode verificar um ou vários endereços ao mesmo tempo.

Substituir caracteres especiais da cadeia de caracteres por regex


Nesta parte, mostrarei como substituir caracteres especiais por expressões regulares em C #.

Se você possui uma sequência com caracteres especiais e deseja removê-los ou substituí-los, pode usar uma expressão regular para isso.

Use o seguinte código: 

 Regex.Replace(your String, @"[^0-9a-zA-Z]+", "")


Este código excluirá todos os caracteres especiais, mas se você não desejar excluir alguns caracteres especiais, por exemplo, a vírgula "," e os dois pontos ":" - faça as seguintes alterações: 

 Regex.Replace(Your String, @"[^0-9a-zA-Z:,]+", "")


Da mesma forma, você pode fazer alterações de acordo com seus requisitos.

Nota:

Deve-se notar que expressões regulares não são uma panacéia para toda pequena manipulação de string. Se você precisar de uma análise simples fornecida pela classe String ou por outras classes, tente usá-lo.
