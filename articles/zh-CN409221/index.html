Advercut 1212数控铣床概述

今天，在我们的评论中，我们将介绍和展示Advercut K30MT / 1212数控铣床。

谁不想阅读文本评论，请观看视频评论，其中包含有关机器的许多其他可视信息：



Advercut K30MT / 1212是一台数控铣床。 Advercut机器有30多种型号，可满足各种业务和生产任务。 该工厂位于中国，生产通过了ISO标准认证，这表明在国际水平上具有很高的装配和生产控制水平。

在我们的审查中，我们将重点放在Advercut型号K30MT / 1212上。 这是专业系列中最实惠的机器。 该机器具有非常严格的技术特征，并在专家中倍受欢迎。



Advercut K30MT / 1212是门式数控铣床，工作范围为1200 x 1200 x 200 mm。

机器的重量达到900公斤，具有良好的基础，因此，保证了良好的抗振性和刚性。

工作区域的大小为1200毫米乘1200毫米。 从机器K30MT / 1212的名称可以理解这一点。 沿Z轴的加工高度为200毫米。 可以选择订购加工高度不超过300毫米的机器。 该机器主要专注于木材，复合材料和轻金属的加工。

Advercut K30MT / 1212铣床配备了带T型槽的真空工作台，这是任何任务的通用解决方案。 无论是切割板材还是使用毛坯，都必须通过使用T型槽将机械固定到工作台上来进一步加强。



在许多情况下，真空夹钳可加快并简化生产，因为 使您节省操作员更换工件的时间，并有助于在复制元件时更好地进行切割。

要使用真空工作台，您将需要一个真空泵，该真空泵通过工作台上的技术开口从工件和工作台之间的腔中吸入空气，从而产生压降，从而将工件牢固地压在工作台上。 真空夹具的有效性取决于所用真空泵的特性，接头的密封性和工件本身的特性。



机器的机械原理建立在导轨上。 这使您可以实现切刀的高精度定位和加工质量。 目前声称的精度为0.05毫米。 二手台湾导轨



作为沿X和Y轴的驱动器，使用了带齿轮传动的步进电机。 该解决方案使您可以实现显着的门户速度，并且比滚珠丝杠上的类似型号快2-3倍。



二手螺旋台湾制造台湾。

该机器标配集中润滑系统，大大简化了维护。 无需手动润滑机器的每个节点，因为所有机构都通过一个公用的供应系统进行集中润滑。

Advercut K30MT / 1212配备一个3000 W的主轴，最高可产生24,000 rpm。 它足以用于木材和塑料，轻质材料的高质量加工。 主轴具有水冷却功能，可避免过热，并提供几乎全天候的运行，并且会中断服务和维护。 还值得注意的是，水冷式主轴比空冷时要安静得多。 可选地，可以在此机器上安装功能更强大的主轴。



机器配备有特殊的主轴喷嘴，可有效去除切削区域中的切屑。 仅将波纹从抽吸单元连接到抽吸单元即可。



主轴配有ER20夹头卡盘。 根据所使用的夹头，这允许您安装柄直径最大为13 mm的切削工具（铣刀）。 机器随附三种夹头尺寸3.175、6、8，以及铣刀入门套件，可让您立即开始使用机器。



机器标准配备了工件高度传感器。

使用RichAuto A11 DSP控制器进行管理。 A11控制器是一个紧凑的独立硬件和软件复合体，基于DSP处理器，可提供与机器的完整工作周期：驱动器控制，程序执行控制以及对传感器和机器节点的其他条件的反应。



控制器从内部存储器或外部USB驱动器读取文件，并将程序代码转换为电机驱动器的控制信号。 A11 DSP控制器已经走了很长一段路，今天它已成为工业机器上最常用的控制器。

借助自主控制器，不需要在生产室中使用单独的计算机来控制机器，操作员可以从常规USB闪存驱动器开始执行任务。 由于其紧凑性，操作员可以在机器的工作区域内移动，在定位和加工产品时提供控制和直接的视觉控制。

用于处理的程序本身可以用几乎所有兼容软件编写。 例如Artcam，MasterCAM，VisualCAM，SprutCAM等。

该机器允许您对木材，复合材料和轻金属的毛坯进行铣削，钻孔，切割，雕刻。

6）集中润滑系统
7）紧凑型独立DSP控制器，真空工作台，标配工件高度传感器，吸尘器喷嘴（顺便说一下，可以在视频中单独查看工件高度传感器，以显示其工作原理）
8）机器的多种其他选择
