Sensor inalámbrico de temperatura, humedad y presión atmosférica en nRF52832

¡Saludos a todos los lectores de Habr! El artículo de hoy tratará sobre un sensor de temperatura, humedad y presión atmosférica con una batería de larga duración. El sensor funciona en el microcontrolador nRF52832 ( hoja de datos ). Para obtener temperatura, humedad y presión atmosférica, se utilizó el sensor BME280 - hoja de datos . El sensor funciona con baterías CR2430 / CR2450 / CR2477. El consumo en modo de transmisión es de 8 mA, en modo de reposo de 5 mA. Entonces, sobre el orden.


Este es un proyecto Arduino, el programa está escrito en el IDE Arduino. La biblioteca de Adafruit Industries se utiliza para operar el sensor BME280 - sensores github | github BME280 . Para trabajar con placas nRF52832, el IDE de Ardeino utilizó el proyecto Sandeep Mistry: github . La transferencia de datos al controlador de Smart House se realiza de acuerdo con el protocolo Mysensors - placas github | protocolo | biblioteca .

La placa del sensor se desarrolló en el programa Diptrace. Las dimensiones de la placa son 36.8 mm X 25 mm.
