Sound Kit untuk Prototipe


Ini adalah seperangkat suara interaktif untuk antarmuka grafis dari aplikasi seluler. Ini memiliki suara tombol, kesalahan, serta peringatan dan notifikasi. Anda dapat mendengarkan mereka secara online di halaman proyek. Para penulis perpustakaan adalah desainer dari Facebook. Mereka mengatakan bahwa sampel dari koleksi tersebut terlibat dalam pengembangan aplikasi sketsa Origami dan Framer .

Lisensi: Non-eksklusif - memungkinkan Anda untuk menggunakan suara saat mengembangkan prototipe aplikasi dan maket. File audio dari perpustakaan juga dapat digunakan dalam produk perangkat lunak komersial - tetapi dalam hal ini, atribusi, lisensi dan tautan ke sana diperlukan. Namun, jika aplikasi Anda adalah alat untuk desainer seperti Origami atau Framer, Anda harus terlebih dahulu mendapatkan izin dari tim Desain Facebook.

Fitur: Sound Kit berisi dua folder - Volume Penuh dan Volume Rendah. Yang pertama berisi file audio dengan volume -5 dB. Mereka cocok untuk menunjukkan prototipe di komputer. Volume suara di folder kedua berkurang dan -20 dB. Para penulis mengatakan bahwa mereka cocok untuk mendemonstrasikan aplikasi pada ponsel - dalam hal ini, speaker tidak akan mengganggu suara. Perpustakaan mencakup 39 file audio, tetapi penulis belum memperbarui set ini sejak 2017.


Dermaga tonone


150 sampel audio untuk mencetak antarmuka pada game dan proyek lainnya.

Lisensi: Bebas royalti - sampel dapat dimodifikasi dan digunakan untuk tujuan komersial.

Fitur: Situs ini memiliki koleksi lain - dengan sampel Sci-Fi dan EDM . Ada pencarian berdasarkan format file (WAV, MP3, OGG, AIFF dan FLAC), serta laju sampel (misalnya, 44,1 atau 96 kHz). Untuk mengunduh sampel audio, diperlukan pendaftaran.



Foto Hal Gatewood / Unsplash


Oktaf


Kumpulan suara UI gratis untuk iOS. Penulisnya adalah Fred Showell, seorang desainer produk di LiveTiles, sebuah perusahaan perangkat lunak Australia. Fred mengatakan dia membangun Octave dari perpustakaan sampelnya sendiri, yang telah dia kumpulkan selama sepuluh tahun terakhir.

Lisensi: Suara diizinkan untuk digunakan dalam proyek apa pun - baik pribadi maupun komersial. Karangan tidak diperlukan, tetapi diinginkan. Tentu saja, perpustakaan itu sendiri tidak dapat dijual.

Fitur: Pilihannya meliputi 48 suara - bip, menampar, dll. - dalam format AIFF dengan frekuensi pengambilan sampel 44,1 kHz. File dengan Kategori UIButton segera dilampirkan padanya, yang menyederhanakan penugasan suara untuk acara di iOS dan OS X. Cukup untuk menulis dalam kode aplikasi: 

[myButton setSoundNamed:@"mySound.aif" forControlEvent: UIControlEventTouchUpInside]

Situs ini memiliki rekomendasi teknis untuk bekerja dengan suara dalam aplikasi di iPad dan iPhone dan merancang UI. Penulis juga menerima pesanan untuk pengembangan suara khusus.


Suara antarmuka


Perpustakaan berisi 500 suara untuk antarmuka grafis dalam aplikasi dan game. Itu disusun oleh seorang desainer audio dari Jerman - Martin Straka. Dia telah menulis sampel untuk aplikasi dan soundtrack game selama sepuluh tahun. Misalnya, dia adalah seorang komposer untuk para Spirit dan Mr. Terpental


Jejaknya bahkan dinominasikan untuk hadiah Festival Game Independen - ini adalah konferensi untuk pengembang game komputer independen.

Lisensi: Semua sampel bebas royalti, mereka dapat digunakan untuk tujuan apa pun - baik komersial maupun non-komersial. Karangan tidak diperlukan.

Fitur: Sayangnya, hanya 11 file audio yang didistribusikan secara gratis. Untuk mendapatkannya, cukup minta demo di situs. Untuk sisanya, Anda harus membayar $ 29. Juga, penulis dapat merekam suara unik berdasarkan permintaan.


Opengameart


Ini bukan perpustakaan sampel sederhana. Ini adalah arsip media dengan suara, musik, serta 2D, seni 3D dan tekstur. Ini dikembangkan oleh insinyur Bart Kelsey pada tahun 2009. Sejak itu, portal telah memperoleh komunitas yang luas - konten baru diunggah di sana kira-kira setiap 20 menit.

Lisensi: Konten yang disajikan pada OpenGameArt dilindungi oleh lisensi gratis. Diantaranya adalah Lisensi Publik Umum GNU , LGPL , dan Creative Commons, termasuk Attribution, Atribution Share Alike, dan CC0. Semua file diizinkan untuk digunakan untuk tujuan komersial.

Fitur: Jumlah koleksi dengan suara (termasuk untuk UI) melebihi seribu. Untuk mempermudah navigasi dalam varietas ini, situs menjalankan pencarian - berdasarkan nama, tag, atau lisensi.
