Flightradar24 - wie geht das?

Wahrscheinlich hat jeder, der jemals Verwandte oder Freunde in einem Flugzeug getroffen oder begleitet hat, den kostenlosen Flightradar24-Service genutzt. Dies ist eine sehr bequeme Möglichkeit, die Position des Flugzeugs in Echtzeit zu verfolgen.



Wie eine Umfrage unter Bekannten ergab, wissen nur sehr wenige, dass dieser Dienst nur von Enthusiasten unterstützt wird, die Daten an den Server senden. Noch weniger wissen, dass jeder dem Projekt beitreten kann.

Füllen Sie diese Lücke. Wie es funktioniert, fahren Sie unter dem Schnitt fort.

Eisen


Die allererste Frage ist, woher die Daten stammen.

Jede zivile Seite ist mit einem speziellen Sender ausgestattet, dem sogenannten ADS-B-Transponder :


(Foto von der Garmin-Website)

Dieses Gerät sendet spezielle Übertragungen mit einer Frequenz von 1090 MHz, die die grundlegenden Flugparameter, den Flugzeugtyp, seine Koordinaten, das Ziel usw. enthalten.

Das Signal im Spektrum sieht ungefähr so ​​aus:



Und was für uns am wichtigsten ist, die Dekodierung dieser Daten ist nicht schwierig, nur ein RTL-SDR-Empfänger, der für etwa 8 US-Dollar an einen Computer angeschlossen ist. Eigentlich ist die Idee wahrscheinlich schon klar - wenn wir Daten von vielen Empfängern sammeln, erhalten wir ein Bild, ungefähr so:



Das ist aber noch nicht alles. Einige Flugzeuge senden Datenpakete, haben jedoch keine GPS-Koordinaten. Wenn Sie gleichzeitig Daten von mehreren Empfängern empfangen, deren Koordinaten bekannt sind, können Sie durch Analyse der Verzögerungen an verschiedenen Empfängern den Standort des Flugzeugs ermitteln. Diese Technologie wird als Multilateration (MLAT) bezeichnet. Weitere Informationen finden Sie auf der Website .

Und schließlich ist die letzte Frage, wie Daten empfangen werden. Die Antwort ist einfach, jeder kann den Empfänger hosten, der Preis der Frage liegt zwischen 0 und 50 US-Dollar. So sieht beispielsweise die Flightaware-Empfängerkarte aus:



Wie Sie sehen können, lässt die Berichterstattung über den russischen Teil zu wünschen übrig, so dass einige gute Chancen haben, der Erste zu sein.

Um Daten zu erhalten, benötigen Sie:

- Raspberry Pi2 / 3 mit Internetverbindung (Preis ab 20 USD)
- RTL-SDR-Dongle (Preis ab 10 USD)
- mindestens die einfachste Antenne ihrer 2 jeweils 6,8 cm langen Drahtstücke (1/4 Dipol bei 1090 MHz)

Wünschenswert ist auch ein Ort mit einem mehr oder weniger offenen Überblick über mindestens einen Teil des Himmels. GHz-Signale werden von Hindernissen wie Häusern oder Bäumen absorbiert.

Ich hatte bereits Raspberry Pi und RTL-SDR, also musste ich nichts kaufen. Jeder kann jedoch versuchen, einen vorgefertigten Empfänger kostenlos zu erhalten - sowohl Flightradar als auch Flightaware senden kostenlos Kits an Benutzer aus Gebieten, in denen die Abdeckung derzeit nicht ausreicht. Sie können eine Anfrage über den Link ( Flightradar oder Flightaware ) senden. Es ist natürlich ratsam, diejenigen auszuprobieren, die in einer ziemlich abgelegenen Gegend leben.

Ich habe ein Kit von RTL-SDR V3 verwendet, das zusammen mit einer Antenne und einem Ministativ 27 US-Dollar mit kostenlosem Versand aus China kostet:



Als zweiten Bonus bieten übrigens alle, die Daten an den Server senden, sowie Flightradar24 und Flightaware uneingeschränkten Zugriff auf das Premium-Konto und alle Serverfunktionen (separat kostet dieser Zugriff ca. 50 USD / Monat). Wenn sich jemand für die Luftfahrt interessiert und einen erweiterten Zugriff auf ein solches Konto haben möchte, ist die Installation eines RTLSDR-Empfängers eine sehr kostengünstige Alternative.

sudo bash -c "$(wget -O - http://repo.feed.flightradar24.com/install_fr24_rpi.sh)"
