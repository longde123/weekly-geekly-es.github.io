Canon Mobile Print: imprimez de n'importe où dans le monde

Hier, nous avons expliqué comment imprimer directement à partir d'un appareil mobile, simplement en se connectant à un réseau sans fil.



Nous allons maintenant couvrir certaines des fonctionnalités de uniFLOW Mobile Printing qui vous permettent de créer des travaux d'impression de n'importe où dans le monde.

Applications mobiles UniFLOW


Les applications mobiles Canon uniFLOW disponibles pour les appareils fonctionnant sur iOS , Android et Windows offrent plus d'espace pour l'impression mobile.



Un employé en voyage d'affaires et les autorités exigent un rapport de toute urgence? Si auparavant, il était nécessaire d'envoyer des documents à des collègues, puis de leur demander d'imprimer, vous pouvez maintenant tout envoyer à imprimer de n'importe où dans le monde. Cette fonctionnalité, en passant, simplifie le flux de travail avec les employés indépendants ou les travailleurs à distance, qui deviennent de plus en plus dans les réalités modernes.

Dans le même temps, les administrateurs système peuvent contrôler l'accès aux appareils via des mots de passe, des codes PIN, des identifiants de tâches et des appareils mobiles, ainsi que différencier les droits et définir des budgets pour chaque utilisateur

Dans le même temps, les utilisateurs peuvent définir indépendamment les paramètres d'impression (format de feuille, qualité d'impression, couleur) et même terminer le traitement (agrafage, livret).

Pilote d'imprimante Google Cloud Print / Browser Download / Web Printer


La prise en charge du protocole Google Cloud Print vous permettra également d'imprimer un document dans votre bureau à partir de n'importe où dans le monde. Et tout ce dont vous avez besoin est un appareil qui prend en charge Google Chrome.


De plus, les administrateurs système pourront configurer une page d'impression spéciale pour les employés, ce qui vous permet d'envoyer des travaux d'impression simplement via le navigateur. Pour éviter les erreurs au stade de l'impression, le système convertit automatiquement les fichiers au format PDF - une bonne fonctionnalité qui peut être utile pour les entreprises, par exemple, avec un intranet ou un portail d'entreprise développé.



Et enfin, toutes les tâches peuvent être envoyées pour impression d'une manière familière aux utilisateurs (indépendamment du fait qu'ils soient à l'intérieur du réseau d'entreprise ou non) - via le menu du programme de bureau. Cependant, cette option sera légèrement plus coûteuse, car les administrateurs système devront installer des pilotes d'imprimante Web sur chaque «machine».

Email et impression d'invité


Dans les entreprises où l'impression dite d'invité est souvent utilisée, par exemple dans le secteur des services, cette fonction peut également être configurée. Grâce à l'utilisation d'uniFLOW, l'accès invité non seulement ne nécessite aucune configuration spéciale, mais n'affecte pas non plus la sécurité du réseau de travail.

D'un point de vue technique, tout est mis en œuvre simplement et commodément. Il suffit que «l'invité» envoie un e-mail à une adresse e-mail spéciale, et la tâche sera immédiatement imprimée. De plus, certains nouveaux modèles d'équipement de bureau Canon sont dotés d'une connectivité directe. Les administrateurs système de n'importe laquelle de ces méthodes pourront également configurer diverses règles d'accès: de l'attribution d'un identifiant unique à chaque lettre et se terminant par l'accès par le code QR. Dans le même temps, les employés des services informatiques ou, par exemple, les services de sécurité peuvent modérer les tâches afin de filtrer le spam et les documents indésirables.
