ProLiant系列100-失落的弟弟

Hewlett Packard Enterprise服务器产品组合的更新标志着2019年第二季度的开始。 同时，此更新将“丢失的小兄弟”归还给我们-一系列HPE ProLiant DL100服务器。 多年来，由于许多人已经忘记了它的存在，因此我在这篇简短的文章中建议刷新我们的记忆。



长期以来，“百分百”系列一直是不涉及爆炸性增长和扩展的架构的预算解决方案。 100系列服务器的成本相对较低，非常适合预算。 但是在第7代之后，HPE决定审查其服务器产品组合以优化生产成本。 结果就是第100个系列的消失，因此，在HPE解决方案上设计预算架构时遇到了困难。 到现在为止，我们只有第300个系列可供选择，它具有出色的性能和配置灵活性，但对预算限制却没有那么宽容。

由于竞争激烈，HPE决定将第100系列恢复其产品组合，从当前的Gen10开始，有数百种产品返回俄罗斯市场。 从4月初开始，可以订购HPE ProLiant DL180 Gen10，并在夏天出现ProLiant DL160 Gen10。 自从新的DL180落入我的手中以来，我决定回顾一下它的主要优缺点。 由于第100个系列最初是第300个系列的更简单，成本更低的版本，因此任何评论都将不可避免地导致比较。 我将通过比较当前市场上提供的DL380和DL180 Gen10来做到这一点。

两种型号都是双处理器，双单元（2U 2P）通用服务器，几乎适用于所有用例。 这是“兄弟”的唯一共同特征。

如前所述，“数百个”在支持的选项数量的限制以及总体上系统配置的灵活性方面有所不同。 DL180服务器（以及将来的DL160）将仅以BTO（按订单制造）的形式提供。

这意味着已为其分配了特定的CPU和RAM型号的一组预先准备的文章。 更具体地说，目前只有两种变体：基于Intel Xeon-Bronze 3106和Xeon-Silver 4110 CPU的单处理器配置，均具有预安装的16Gb PC4-2666V-R RAM和用于8个SFF磁盘的存储篮。
与DL380中的24个插槽相比，RAM的插槽数量减少到16个。 在受支持的内存模块列表中，除了基本配置中安装的内存模块外，所有内容都丢失了：HPE 16GB（1x16GB）Single Rank x4 DDR4-2666 CAS-19-19-19 Registered Smart Memory Kit。 当前未提供具有双列或降低负载的DIMM的选件。

如果我们谈论数据存储，那么百分之一系列将损失三分之一：

  • 一个8 SFF的磁盘篮
  • 集成式S100i控制器
  • 可选的E208i / E208e和P408i控制器

将来，计划增加额外的用于8 SFF的可选存储篮（每个机箱最多2个）和一个用于LFF驱动器的新机箱。

要访问网络，机箱配备了两个1 GE端口，可以使用可选的FlexibleLOM适配器将其扩展为两个10 / 25Gb端口。
PCI-E模块的插槽数量未更改；以下选项可用（带有双处理器配置）：

  • 3 + 3 PCI-E x8（使用FlexibleLOM需要一个特殊的提升板模块）
  • 1个PCE-E x16 + 4个PCI-E x8

由于所发布模型的新颖性，因此文档中有些混乱。 因此，根据QuickSpecs，仅显示具有SAS接口（300/600/1200 Gb 10k）的硬盘驱动器。 但是，仅支持SATA驱动器的内置Smart Array S100i RAID控制器的存在导致人们对文档中的不准确性产生了想法。

与以前一样，很可能支持其他服务器型号的所有Gen10 SATA驱动器。 而且，如果您安装了离散RAID控制器HPE Smart Array E208i，则可以使用SAS驱动器。

由于该版本的更新（我提醒您它发生在2019年4月上旬，即距本文发表不到3周前），因此尚不完整的受支持选项列表，但我们可以假定没有NVMe驱动器和图形加速器，因为该功能强大电源的限制为500W。

最重要的是，我们获得了自信的“平均性能”，HPE拥有足够的容量和不变的“优点”，不需要太多介绍。
尽管，或者说，甚至归功于有限的选件数量，对于预算有限的项目，100系列机型还是一个不错的解决方案。 如果您的任务需要DL380 Gen10的可扩展性和性能，但在经济上您负担不起，那么DL180 Gen10就是专门为您创建的。 剩下的只是等待完整的选件清单和LFF机箱以及DL160 Gen10一起出现在俄罗斯市场。
