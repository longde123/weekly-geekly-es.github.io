Alcootest DrinkMate





Nous avons déjà écrit sur ce capteur utile - avec son aide, il est possible de mesurer le niveau d'alcool dans le sang avec un niveau de précision assez élevé. Comment ça marche? Voici ce que vous devez faire pour faire fonctionner l'appareil:

1) Téléchargez l'application DrinkMate gratuite

2) Activez l'application DrinkMate sur votre téléphone

3) Suivez les instructions de l'application sur la façon de souffler dans DrinkMate

4) Affichez les résultats de la mesure. DrinkMate est

connecté via micro-USB, l'alimentation est la batterie du smartphone, l'alcootest n'a pas sa propre batterie.

Avantree PowerHouse - station de charge USB de bureau





Avantree PowerHouse USB Station est une excellente solution pour les utilisateurs qui ont constamment besoin de charger toute la masse de leurs appareils. La station peut charger des smartphones, tablettes, téléphones ordinaires. Le nombre maximum de gadgets pouvant être chargés simultanément est de quatre.

Cette solution peut être utilisée à la fois pour la maison et le bureau - est venu, mettez le gadget, chargé.

Luminous BT Smart Bulb - une ampoule intelligente





Une solution intéressante est l'ampoule intelligente LED, qui se connecte au smartphone, à travers laquelle l'ampoule intelligente Luminous BT est contrôlée. Une application spéciale vous permet de contrôler la luminosité de la luminosité de l'ampoule, de changer de couleur, de regrouper les sources lumineuses pour un contrôle simultané. Luminous BT Smart Bulb est

contrôlé à distance à l' aide de la technologie sans fil Bluetooth version 4.0 et ultérieure. La portée du signal radio est d'environ 40 mètres.

Ces ampoules sont parfaites pour la maison et le bureau.

ASWY Air Speaker WA-102 Enceinte à lévitation sans fil





Le haut-parleur ASWY Air Speaker WA-102 reproduit un son de haute qualité, bien qu'il semble assez intéressant - pendant le fonctionnement, le haut-parleur plane au-dessus d'un support magnétique. Ce support maintient le haut-parleur en l'air.

Le haut-parleur est contrôlé à l'aide d'un smartphone - via Bluetooth 4.0. La durée de vie de la batterie de l'enceinte est d'environ 6 heures. Dans le même temps, le son se propage uniformément dans toutes les directions, les ondes sonores ne sont pas absorbées par la surface sur laquelle se trouve le haut-parleur ordinaire.

Si vous le souhaitez , les fonctions de l' ASWY Air Speaker peuvent être contrôlées à l'aide de boutons - le panneau de commande des boutons est situé sur l'appareil. L'enceinte est chargée "over the air", recharge sans fil.

Rooti Labs CliMate - la plus petite station météo sans fil au monde





Le climat est une station météorologique qui vous permet de suivre l'humidité de l'air, la température et le niveau de rayonnement ultraviolet. Cette dernière fonction est très importante dans les régions où le soleil est très actif. La station montre combien vous pouvez être au soleil avec le niveau actuel de rayonnement UV.

Toutes les données sont enregistrées, puis ces informations peuvent être affichées sur l'écran d'un téléphone mobile. Les données sont transmises via Bluetooth. CliMate est un appareil sans fil, portable, petit, léger et très intelligent. Ce minuscule tracker capture l'emplacement et l'heure de la collecte des données. Dans des conditions normales d'utilisation, le tracker peut fonctionner jusqu'à 6 mois sur une seule batterie ronde.
