Windows 98 - désormais également dans le navigateur



Les amateurs exécutent d'anciens jeux, programmes et systèmes d'exploitation dans le navigateur. Dans le même temps, le travail requis pour mettre en œuvre certaines idées est très laborieux et prend du temps, sans parler du temps nécessaire dans certains cas. Il a fallu quatre ans aux développeurs pour lancer Windows 98 dans un navigateur. Mais cela en valait la peine - maintenant une version complètement fonctionnelle de ce système d'exploitation est lancée dans le navigateur.

Le travail a été effectué en Javascript, le code source contient plusieurs milliers de lignes de code. Désormais, les utilisateurs qui sont nostalgiques des anciennes versions du système d'exploitation peuvent profiter de travailler avec des programmes à l'ancienne, y compris des jeux.



Il est possible de démarrer à partir d'un CD ou d'une disquette (des images sont requises). Cela signifie que dans Tribes ou Half Life 1, vous pouvez jouer dans le navigateur! Certes, le nombre de jeux qui s'exécutent réellement dans le navigateur est limité. Le même Rollercoaster Tycoon n'a pas pu être lancé (mais vous pouvez essayer).



Le navigateur fonctionne également, mais avec des problèmes de compatibilité connus. Bien sûr, il ne peut pas se passer de problèmes, mais il convient de rappeler qu'il s'agit d'un émulateur. Néanmoins, beaucoup de travail, je recommande donc d'essayer le système dans la pratique.
