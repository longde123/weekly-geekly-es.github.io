Windows Phone - SEMUANYA, Ini Lagi atau Lagi

Di sini baru-baru ini, Microsoft "senang semua orang" bahwa dukungan untuk perangkat berbasis Windows Phone akan segera berakhir. Sebagai pengguna jangka panjang sistem operasi seluler Microsoft, berita itu tampak tidak begitu menggembirakan bagi saya. Namun, muncul pertanyaan: apakah Windows Phone RIP lagi atau lagi ?, karena, kami telah melewatinya, tahun iklan pada 2011 , lalu ada penghentian layanan Windows Phone 7 .... lalu Windows Phone 8 (saya masih menggunakannya), dan sekarang Windows Phone 10.


Tentu saja saya merasa kasihan pada platform, sepertinya sudah asli dan sangat akrab, tetapi saya mengerti bahwa kemajuan harus berjalan, dan saya sangat perlu lari ke toko untuk perangkat keras baru di Android atau iOS ... Namun ... well, sangat malas untuk membeli sesuatu yang baru ... katakanlah dukungan saya Lumia 925 berakhir pada tahun 2016, tetapi aplikasi terus diperbarui dan sepertinya semuanya berfungsi. Nah, itu menjadi menarik berapa banyak kemajuan telah melangkah maju, atau bukan kemajuan, tetapi hanya perang pemasaran. Kemudian saya ingat bahwa pada suatu waktu saya ingin mengubah Acer N30 saya ke Acer N311 yang lebih baru, tetapi saya membeli Eten X-800 sebagai gantinya. Dan seperti yang mereka katakan, inilah saatnya untuk memenuhi impian tersebut, mengingat fakta bahwa di pasar loak online perangkat ini dapat dengan senang hati dibeli untuk beberapa ribu ...

Kembali ke masa lalu ...


Secara umum, potongan besi dikirim cukup cepat, dan mengambilnya menjadi jelas ... setelah semua, mereka digunakan untuk membuat perangkat yang lebih nyaman, tetapi mereka lebih tebal dan lebih berat, lumia "tipis" dan "bulat" semua mencoba untuk melompat keluar dari tangan saya karena Lumia saya punya banyak berbeda, karena mereka bertempur tanpa dewa.

Kebetulan saya mendapatkan perangkat dengan firmware masih Windows Mobile 5.1, dan ternyata untuk meng-upgrade ke versi Windows Mobile 6.1 hanya kartu SD yang diperlukan, bootloader akan melakukan sisanya. Ngomong-ngomong, pada perangkat WP, bahkan saya tidak ingat beragam firmware "artisanal", dan posting panjang di forum yang menggambarkan peningkatan dari "orang pintar", mungkin Microsoft masih belajar cara membuat OS seluler atau "orang pintar" sudah selesai ...

Saya juga ingin mencatat bahwa kehadiran stylus memberi peluang besar untuk input yang akurat, dan nyaman untuk menggunakan keyboard, saya diam tentang pengenalan tulisan tangan (well, ini berfungsi dan bekerja dengan baik). Yah, layar Today pasti nyaman, mis. kalender, ringkasan tugas yang ditampilkan dengan gembira, dan entah bagaimana perangkat modern dari Microsoft pergi dari ini karena alasan tertentu, dan berubah menjadi telepon untuk "shkolota" yang membutuhkan dua atau tiga lusin pintasan di layar untuk meluncurkan beberapa yang "sangat diperlukan" aplikasi. Misalnya, dalam perangkat WP modern tugas tidak dikelompokkan ke dalam kategori, baik, dari kata sama sekali, meskipun sangat nyaman. Sebagai contoh, untuk memakai daftar belanja dan hanya pada perangkat Anda dapat menampilkan semua tugas dalam kategori "pembelian" dengan tepat ... ya, tentu saja, mereka akan mulai meneriakkan "unduh aplikasi Anda" dan berbahagialah, tetapi mengapa saya memerlukan banyak aplikasi ketika ada satu antarmuka terpadu dan dapat melakukan semua ini . Secara umum, sebagai organizer yang berfungsi, WM masih lebih baik daripada keturunannya yang menjalankan WP.

Masih ketinggalan zaman ...


Tetapi untuk menggunakan semua kenyamanan penyelenggara, Anda pasti perlu sinkronisasi dengan Exchange, apa pun yang dikatakan, tapi saya suka 17 inci lebih dari 3,7 dan tidak peduli seberapa sesat, 100 tombol lebih nyaman daripada papan ketik di layar. Karena ada Wi-Fi pada hewan itu, direncanakan untuk mengatur sinkronisasi dengan Exchange secara langsung dan bersenang-senang. Namun, penyiapan tidak berhasil, mis. sama sekali, aneh salah dengan sertifikat ... secara umum, menari dengan rebana tidak membantu, berhenti pada tahap ketika semua sertifikat yang diperlukan muncul di root dan masih tidak berfungsi, dengan cara Internet meninjau dan mengatur surat Internet tidak berjalan karena alasan yang sama ... Saya benar-benar tidak melakukannya Saya mengerti, tetapi enkripsi tidak bekerja dengan baik, atau perangkat tidak mendukung algoritma, tangan saya berada di tempat yang salah (yang sangat mungkin), secara umum, baik google maupun outlook tidak terbuka, lebih tepatnya mereka membuka tetapi mereka menulis bahwa kesalahan sertifikat, lebih tepatnya dengan sertifikat seperti semuanya baik-baik saja tetapi sertifikat yang dikeluarkan tidak dipahami di antaranya ...

Karena tidak mungkin untuk melakukan sinkronisasi melalui udara, saya memutuskan untuk menggunakan cara lama yang baik dan terhubung melalui "selang hitam". Kemudian saya menyadari sedikit bahwa Win10 tidak sepenuhnya berteman dengan Windows Mobile Device Center, tetapi saya dapat mengalahkan masalah ini (well, saya sedikit menurunkan registri, meluncurkan layanan, dan secara umum itu berhasil).

Tetapi wasir tidak berakhir di sana, pusat perangkat mulai, dan bahkan menjalin kemitraan dengan hewan, tetapi menolak untuk melihat Outlook untuk dipotong, mis. Dia menjelaskan kepada saya bahwa program surat tidak diinstal atau default tidak diatur ... secara umum, google untuk waktu yang lama, tetapi fakta bahwa pusat perangkat tidak melihat Outlook 2016 x64 point-blank, menginstal versi 32-bit tidak membantu.

Katakan apa yang Anda suka, tetapi untuk menyinkronkan, Anda harus mengambil langkah mundur, secara umum, mengangkat mesin virtual, menginstal Win7 dan kantor model 2007 di atasnya. Kantor dengan senang hati terhubung ke Exchange saya, tetapi hewan itu mulai menyinkronkan dengan Outlook. Ya, tentu saja, semua ini tidak terlalu efektif dan bahkan "sangat bengkok", tetapi berhasil !!!

Dunia aplikasi yang menarik ...


Secara umum, mengatur sinkronisasi dengan kesedihan menjadi dua, saya memutuskan untuk memperluas kemampuan hewan dengan menginstal aplikasi. Ya, perangkat benar-benar tidak memiliki "toko" yang sudah cukup digunakan dan aplikasi perlu diunduh di suatu tempat, dan kemudian entah bagaimana diinstal, dan ini tidak selalu berhasil. Dibandingkan dengan toko aplikasi modern di OS mobile, pilihannya tentu tidak hebat, tetapi di sisi lain, aplikasi tersebut cukup berkualitas tinggi dan bijaksana. Secara umum, saya memasang PocketMusic dan saya mendapatkan seorang pemain (pemain standar tidak memiliki equalizer) - begitulah perluasan kemampuan dikatakan, tidak hanya Anda harus menggantungkan kulit pada pemain standar. Dari plugin layar, saya menolak hari ini karena dibandingkan dengan yang modern, standar "ketinggalan jaman" terlihat cukup mengesankan. Saya juga mengatur SPB Time dan mendapatkan arloji cantik di layar penuh ... secara umum, dok terlihat sangat asli.
Saya tidak tahu mengapa, saya menambahkan SPB Finance (pada saat yang sama Microsoft Money ke mesin virtual untuk sinkronisasi), saya juga mengunduh analog MathCAD di PDA (well, saya mengalami kesulitan ketika menghitung dengan unit) dan tentu saja Adobe Reader. Pada prinsipnya, kami telah memperoleh perangkat yang diperlukan yang memungkinkan kami untuk menyelesaikan masalah yang mendesak.

Dan yang paling tampak liar di dunia modern ... aplikasi tidak perlu pembaruan, mis. mereka bekerja dan bekerja dengan baik selama bertahun-tahun, dan aplikasi bekerja bahkan ketika tidak ada internet. Khususnya, pemain tidak memerlukan izin untuk mengelola jaringan (tidak memerlukannya), dan pembaca PDF tidak merangkak ke dalam surat atau kontak karena hanya menunjukkan PDF ... Tampaknya pendekatan terhadap fungsi aplikasi ini mulai dilupakan, karena aplikasi apa pun yang paling tidak berharga dari PlayMarket Ini memerlukan daftar panjang izin, termasuk navigasi, manajemen notifikasi, akses ke buku telepon, SMS, dll.

Pekerjaan offline


Mengingat kruk, tempat saya mengatur sinkronisasi, bekerja dengan beast ternyata hanya off-line dengan sinkronisasi selanjutnya. Dan kemudian pikiran gila muncul di benak saya, dan cara yang paling dapat diandalkan untuk melindungi terhadap penjahat cyber di jaringan hanyalah kurangnya jaringan ... Dan kemudian saya ingat semua jenis skandal yang melibatkan raksasa perangkat lunak ... mereka mengatakan mereka mengawasi pengguna ... dan di sini ada cara yang dijamin untuk keluar dari pengawasan ... . Seperti yang mereka katakan, jika Anda menggunakan file data Outlook lokal, file dokumen lokal, maka orang jahat tidak akan mengenali rencana Anda dari kalender atau apa yang secara jahat Anda ciptakan dan gambarkan dalam dokumen teks, atau pembukuan hitam macam apa yang Anda lakukan. Dan Anda dapat dengan senang hati membawa data yang diperlukan di saku Anda ... dan seolah-olah ketersediaan data di cloud dan bahkan ILV yang mengerikan sama sekali tidak menarik bagi Anda ...

Dan mengapa saya melakukan semua ini?


Ya, saya tidak sengaja membeli sendiri perangkat lama baru dan memutuskan untuk membual. Saya tidak tahu ada apa dengan RIP atau TIDAK Windows Phone ... tetapi Windows Mobile masih hidup ...
