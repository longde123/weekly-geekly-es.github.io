3 livres sur C ++ 17 (en anglais)

Aujourd'hui, nous vous parlons de trois livres écrits par notre MVP Microsoft, en C ++ 17. Ils peuvent être de bons points de départ pour découvrir la nouvelle version de ce langage de programmation. Tous sont vendus sous forme électronique et papier sur des plateformes en ligne (de 8,64 € à 42,99 $ pour les versions électroniques).



Professional C ++ Fourth Edition par Marc Gregoire


«Professional C ++» est un guide de programmation C ++ avancé. Conçu pour aider les programmeurs expérimentés avec la dernière mise à jour, ce livre passe en revue les bases et plonge directement dans la description de toutes les fonctionnalités de C ++ 17. Chaque fonctionnalité est complétée par un exemple et un morceau de code que vous pouvez utiliser dans vos projets.

Les exemples illustrés comprennent du code testé sur Windows et Linux, ainsi que des conseils et des secrets sur les droits d'auteur qui peuvent vous aider à améliorer votre flux de travail. Même de nombreux développeurs expérimentés n'ont jamais pleinement exploré les limites des capacités linguistiques; Ce livre parle de fonctionnalités que vous ne connaissiez pas. Son objectif principal est de vous montrer comment transformer ces fonctions en véritables solutions.

Prix: 42,99 $ (livre électronique)


Le défi C ++ moderne: devenez un expert en résolution de problèmes réels par Marius Bancila


C ++ est l'un des langages de programmation les plus utilisés et propose des exemples d'utilisation dans divers domaines, tels que les jeux, la programmation graphique et les systèmes d'exploitation. Au fil des ans, C ++ est resté l'une des meilleures solutions pour les développeurs de logiciels dans le monde entier.

Ce livre vous montrera quelques fonctionnalités C ++ importantes et comment les implémenter pour répondre à vos besoins. Chaque problème est unique et ne teste pas seulement votre connaissance de la langue; elle teste vos capacités de réflexion stratégique et votre capacité à trouver le meilleur moyen de sortir de toute situation. Mais il existe divers problèmes qui, parfois, déroutent les développeurs. Ce livre vient à la rescousse précisément dans de telles situations.

Ce livre plaira aux développeurs C ++ à tous les niveaux.

Prix: 8,64 € (e-book)


C ++ 17 en détail par Bartłomiej Filipek


Bien que C ++ 17 ne soit pas aussi volumineux que C ++ 11, il est plus grand que C ++ 14.

De nombreux modules attendus, routines conjointes, concepts et autres fonctionnalités puissantes, mais les développeurs n'ont pas pu tout préparer à temps. C ++ 17 est-il mauvais à cause de cela? Pas du tout. Et ce livre montrera pourquoi.

Le livre vous donne des connaissances exclusives sur C ++ 17 et s'appuie sur l'expérience de nombreux articles parus sur bfilipek.com. Les chapitres ont été écrits à partir de zéro et mis à jour avec les dernières informations. Vous trouverez également de nombreux exemples et conseils d'utilisation pratiques. De plus, le livre donne une idée de l'état actuel de l'implémentation, du support du compilateur, des problèmes de performances et d'autres connaissances connexes qui vous aideront à améliorer les projets en cours.

Si vous avez une idée de C ++ 11/14 et que vous souhaitez passer aux dernières méthodes C ++, ce livre est pour vous.

Prix: 14,99 $ (livre électronique)
