Masuk ke Azure DevOps menggunakan kredensial GitHub

Di Microsoft, kami fokus pada ide, yaitu untuk memberi pengembang kesempatan untuk dengan cepat membuat aplikasi berkualitas tinggi. Salah satu cara untuk mencapai ini adalah dengan menyediakan berbagai produk dan layanan yang mencakup semua tahap siklus hidup pengembangan perangkat lunak. Ini termasuk alat IDE dan DevOps, aplikasi berbasis cloud dan platform data, serta sistem operasi, kecerdasan buatan, solusi IoT dan banyak lagi. Semuanya terkonsentrasi di sekitar pengembang, sebagai individu yang bekerja dalam tim dan organisasi, serta anggota komunitas pengembang.

GitHub adalah salah satu komunitas pengembang terbesar, dan bagi jutaan pengembang di seluruh dunia, identifikasi mereka pada GitHub telah menjadi aspek penting dari kehidupan digital mereka. Menyadari hal ini, kami senang mengumumkan peningkatan yang akan membantu pengguna GitHub memulai dengan layanan pengembang kami, termasuk Azure DevOps dan Azure.


Kredensial GitHub Anda sekarang dapat digunakan untuk mengakses layanan Microsoft.


Sekarang kami memberi pengembang kesempatan untuk masuk ke layanan online Microsoft menggunakan akun GitHub yang ada di halaman masuk Microsoft apa pun. Menggunakan kredensial GitHub Anda, sekarang Anda dapat masuk melalui OAuth ke layanan Microsoft apa pun, termasuk Azure DevOps dan Azure.

Anda akan melihat peluang untuk masuk ke akun Anda dengan mengklik "Masuk dengan GitHub".

Setelah masuk melalui GitHub dan mengotorisasi aplikasi Microsoft, Anda akan menerima akun Microsoft baru yang terkait dengan kredensial GitHub Anda. Selama proses ini, Anda juga memiliki opsi untuk menautkannya ke akun Microsoft yang ada, jika Anda sudah memilikinya.

Masuk ke Azure DevOps


Azure DevOps menawarkan serangkaian layanan bagi pengembang untuk membantu mereka merencanakan, membuat, dan mengirimkan aplikasi apa pun. Dan dengan dukungan otentikasi GitHub, kami dapat menyederhanakan bekerja dengan layanan Azure DevOps seperti Integrasi Berkelanjutan dan Pengiriman Kontinu (Saluran Pipa Azure); Perencanaan Agile (Papan Azure); dan penyimpanan paket pribadi, seperti modul untuk NuGet, npm, PyPi, dll. (Azure Artifacts). Azure DevOps gratis untuk individu dan grup kecil hingga lima orang.

Untuk memulai dengan Azure DevOps menggunakan akun GitHub Anda, klik "Mulai gratis menggunakan GitHub" di halaman Azure DevOps .



Setelah proses masuk selesai, Anda akan dibawa langsung ke organisasi terakhir yang Anda kunjungi di Azure DevOps. Jika Anda baru di Azure DevOps, Anda akan dibawa ke organisasi baru yang dibuat untuk Anda.

Akses ke semua layanan online Microsoft


Selain mengakses layanan pengembang seperti Azure DevOps dan Azure, akun GitHub Anda dapat digunakan untuk mengakses semua layanan online Microsoft, dari Excel Online hingga Xbox.

Saat mengautentikasi dengan layanan ini, Anda dapat memilih akun GitHub Anda setelah mengklik "Opsi masuk".



Komitmen kami terhadap privasi Anda.


Ketika Anda pertama kali menggunakan akun GitHub Anda untuk mengakses layanan Microsoft, GitHub akan meminta izin untuk menggunakan informasi profil Anda.

Jika Anda memberikan persetujuan, GitHub akan memberikan alamat email akun GitHub Anda (baik publik maupun pribadi), serta informasi profil, seperti nama Anda. Kami akan menggunakan data ini untuk menentukan apakah Anda memiliki akun di sistem kami, atau jika Anda perlu membuat akun baru jika tidak. Menghubungkan ID GitHub Anda ke Microsoft tidak memberikan Microsoft akses ke repositori Anda di GitHub. Aplikasi seperti Azure DevOps atau Visual Studio akan meminta akses ke repositori Anda secara terpisah jika mereka perlu bekerja dengan kode Anda, dan Anda harus memberikan persetujuan secara terpisah.

Meskipun akun GitHub Anda digunakan untuk masuk ke akun Microsoft Anda, mereka tetap terpisah - yang satu hanya menggunakan yang lain sebagai metode login. Perubahan yang Anda lakukan pada akun GitHub Anda (misalnya, mengubah kata sandi atau mengaktifkan otentikasi dua faktor) tidak akan mengubah akun Microsoft Anda, dan sebaliknya. Anda dapat mengelola koneksi antara GitHub dan identitas Microsoft Anda pada halaman manajemen akun pada tab Keamanan.

Mulai Belajar Azure DevOps Sekarang


Buka halaman Azure DevOps dan klik "Mulai Gratis dengan GitHub" untuk memulai.

Jika Anda memiliki pertanyaan, kunjungi halaman dukungan. Seperti biasa, kami akan senang mendengar umpan balik atau saran Anda, jadi beri tahu kami pendapat Anda di komentar di bawah.
