Comment empêcher Windows 10 de redémarrer après les mises à jour

Bonjour à tous. Étonnamment, la réponse à cette question (plus précisément, "comment tout faire pour que ce soit comme avant") est mal éclairée dans Runet. Et en général, une solution toute faite doit encore être recherchée.
Cependant, le problème doit être résolu car le système ne récupère pas toutes les applications après un redémarrage.

Donc:

  1. Accédez aux paramètres de stratégie. Cela fonctionnera même sur une machine locale sans domaine.
    Pour démarrer l'éditeur de stratégie, procédez comme suit: Démarrer → Exécuter (ou Win + R) → gpedit.msc
  2. À gauche dans l'arborescence, sélectionnez: Configuration ordinateur, Modèles d'administration, Composants Windows, Windows Update.
  3. Nous nous intéressons ici à deux paramètres:

    1. N'effectuez pas de redémarrage automatique lors de l'installation automatique des mises à jour si le système utilise des utilisateurs (pas de redémarrage automatique avec des utilisateurs connectés pour les installations de mises à jour automatiques planifiées) Fuh, allumez-le. (Activé). Hourra, il n'y aura plus de redémarrage (s'il y a des sessions actives).
    2. Faites également attention au paramètre Configurer les mises à jour automatiques. Si vous le mettez dans l'état Désactivé, le système cessera de télécharger et d'installer secrètement les mises à jour.

      Si vous voulez que le système ne redémarre jamais spontanément (même en l'absence de sessions actives), c'est ce que vous devez faire.

Après avoir désactivé le redémarrage, le système vous donnera périodiquement une belle fenêtre vous demandant de redémarrer l'ordinateur (comme c'était le cas auparavant):


Soit dit en passant, cette fenêtre est encore meilleure: elle vous permet de reporter le redémarrage "jusqu'au soir", ce qui signifie un rappel seulement une fois par jour, et pas toutes les 4 heures, comme c'était le cas auparavant.

PS: Comme indiqué correctement dans les commentaires, l'éditeur de stratégie n'est disponible que sur les versions plus raides que Professional. Sur les versions plus récentes, vous devrez apporter des modifications directement au registre.
Fichier .reg pour désactiver le redémarrage automatique.
