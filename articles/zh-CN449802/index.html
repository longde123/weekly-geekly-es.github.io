职业程序员。 第1部分。第一个程序

尊敬的Habr读者，我向您介绍一系列我计划在以后的书中结合的帖子。 我想探究过去，并讲述我如何成为一名开发人员并继续成为一名开发人员的故事。

关于进入IT的先决条件，反复试验的路径，自我教育和幼稚的天真。 我将从童年开始我的故事，直到今天结束。 我希望这本书对于那些正在学习IT专业的人特别有用。

而那些已经从事IT工作的人可能会走自己的路。

在这本书中，您会提到我读过的文献，以及与我在培训，工作和创办初创公司时遇到困难的人进行交流的经验。

从大学的老师开始，到大型风险投资人和数百万美元公司的所有者结束。

迄今为止，本书的3.5章已准备就绪，可能只有8-10章。 如果第一章得到读者的积极回应，我将把整本书整体出版。

关于我自己


我不是John Carmack，Nikolay Durov或Richard Matthew Stallman。 我没有在Yandex，VKontakte或Mail.ru这样的公司工作。

尽管我曾在一家大型公司工作，但我一定会告诉您的。 但是，我认为这一点与其说是通向开发人员的历史，倒不如说是大名鼎鼎，更不用说在我从事商业开发的12年职业生涯中所取得的胜利和失败。 当然，你们中的一些人具有更多的IT经验。 但是我相信我目前的职业生涯中的戏剧和胜利值得一提。 有很多事件，而且它们都是各种各样的。

我今天作为开发人员是谁
-参与了70多个商业项目，其中许多是他从头开始撰写的
-在自己的十几个项目中：开源，初创公司
-从事IT工作12年。 17年前-编写了第一个程序
- 微软最有价值人 2016
-微软认证专家
-认证的Scrum Master
-总体而言，我拥有C＃/ C ++ / Java / Python / JS
-今天的主要工作场所是Upwork Freelance服务。 通过她，我为一家处理NLP / AI / ML的公司工作。 拥有一百万用户的数据库
-在AppStore和GooglePlay中发布了3个应用程序
-我准备围绕我现在正在开发的项目建立自己的IT公司

除了开发之外，我还在流行的博客上写文章，学习新技术，在会议上发表演讲。 我在健身俱乐部和家人一起放松身心。

关于这本书的主题，大概就是关于我的。 接下来是我的故事。


故事。 开始


我最初在7岁的时候就了解了一台计算机。 我刚上一年级，在美术课上，我们得到了作业，用纸板，泡沫橡胶和毡尖的笔来制作计算机。 当然，我父母帮了我。 妈妈在80年代初就读于一所技术大学，不知道是什么计算机。 在学习过程中，她甚至设法将打孔卡打孔并将其加载到巨大的苏维埃机器中，该机器占据了培训室的最大份额。

我们在5点完成了功课，因为我们竭尽所能。 我们发现了一张厚厚的A4纸板。 从旧玩具上切下泡沫圈，并用毡尖笔绘制用户界面。 我们的设备上的按钮很少，但是我和我妈妈为他们分配了必要的功能，并且在课程中，我向老师展示了如何通过按下“开”按钮在“屏幕”的一角点亮灯，同时用毡尖的笔完成一个红色圆圈。

与计算机技术的下一个交集，我发生在同一年龄的某个地方。 在周末，我经常和我的祖母和祖父一起探访，他们以各种垃圾进行交易，并乐意以一分钱的价格买下它。 旧表，茶炊，锅炉，徽章，13世纪战士的剑等。 在各种各样的事情中，有人给他带来了一台由电视和录音机供电的计算机。 幸运的是，我祖母兼有。 苏联生产当然。 电视电子带有八个按钮来切换频道。 还有Vega的两盒录音机，它甚至可以重新录制电影录音带。


苏联计算机“搜索”和外围设备：电视“电子”，磁带录音机“ Vega”和使用BASIC语言的录音带

我们开始了解整个系统是如何工作的。 配有一台计算机，里面有几个录音带，一本破烂不堪的说明书以及另一本标题为“ BASIC编程语言”的小册子。 尽管是我的童年，但我还是积极参与将电缆连接到录音机和电视的过程。 然后，我们将其中一盘录像带插入录像机隔间，单击“前进”按钮（即开始播放），电视屏幕上会出现文字和破折号的难以理解的伪图形。

主机本身就像一台打字机，只有很泛黄的重量。 我带着孩子的兴奋推了所有钥匙，没有看到明显的结果，便跑去散步。 尽管在我之前已经写过关于BASIC语言的手册，其中包含一些程序示例，但由于年龄原因，我根本无法重写这些程序。

从童年的回忆中，我当然记得我父母与其他亲戚结成的所有小玩意儿。 第一个吵闹声是著名的游戏“狼捉鸡蛋”。 我很快浏览了一下，最后看到了期待已久的动画片，并想要更多东西。 然后是俄罗斯方块。 当时，它的价值为1,000,000张优惠券。 是的，那是在90年代初在乌克兰，为了我的学术成就，他们给了我一百万。 我当之无愧地感觉像个百万富翁，我命令我的父母参加了这个更加复杂的游戏，在该游戏中，必须正确地布置从上方掉落的不同形状的人物。 购买当天，俄罗斯方块完全被我的父母带走，他们自己两天都无法将其取走。


著名的“狼捉鸡蛋和俄罗斯方块”

然后是游戏机。 我们一家人住在一间小房子里，我的叔叔和姨妈也住在隔壁的房间里。 我的叔叔是一名军方飞行员，他穿越了热点地区，因此尽管谦虚谦虚，他还是非常顽强并且在真正的敌对行动之后不惧怕。 像90年代的许多人一样，我的叔叔开始经商，他的收入相当可观。 因此，在他的房间里出现了进口电视，VCR，然后是Subor前缀（类似于Dendy）。 当我看着他玩超级马里奥（Super Mario），TopGun，终结者（Terminator）和其他游戏时，这真是惊人。 当他把操纵杆放到我手中时，我的幸福无止境。


八位前缀“ Subor”和传奇的“超级马里奥”

是的，就像所有90年代长大的普通孩子一样，我在院子里呆了一整天。 现在先打先驱球，然后打羽毛球，然后在花园里爬树，那里种了许多不同的水果。
但是，当您可以控制Mario，越过障碍物并救出公主时，这种新颖性比任何盲人的眼镜，典当和经典之作都要有趣得多。 因此，看到我对控制台的真正兴趣，我的父母给了我学习乘法表的任务。 然后他们将实现我的梦想。 她是二年级的老师，而我刚完成一年级。 但是，据说-完成了。

没有比它自己的游戏机更强大的动力了。 一周后，我轻松回答了“九口之家”，“六个三口之家”之类的问题。 测试通过了，我买了一份令人垂涎的礼物。 您将在后面学到，控制台和计算机游戏在使我对编程产生兴趣方面发挥了重要作用。

如此年复一年地过去了。 下一代游戏机问世。 首先是Sega 16位，然后是Panasonic，然后是Sony PlayStation。 当我表现良好时，游戏是我的消遣。 在学校或家里有门框时，操纵杆是从我手里拿走的，当然我不能玩。 当然，赶上您放学回来，父亲没有下班回家接电视的那一刻，也是一种运气。 因此，不可能说我是游戏玩家或整天都在玩游戏。 没有这种可能性。 我宁愿在院子里呆一整天，那里也可以找到一些有趣的东西。 例如，一个完全荒野的游戏-用气枪进行小规模冲突。 在我们这个时代，您不会在院子里遇到这个问题，然后-那是一场真正的战争。 与我们组织的大屠杀相比，彩弹射击对孩子们来说很有趣。 空气中装有厚实的塑料子弹。 在另一个孩子的枪口空白处射击时，他的手臂或腹部地板上有一块瘀伤。 这样他们就住了。


童年的玩具枪

提到电影“黑客”并不是多余的。 他在1995年出演，由20岁的安吉丽娜·朱莉（Angelina Jolie）主演。 如果说这部电影给我留下了深刻的印象，那就什么也没说。 毕竟，孩子们的思维能感知所有事物的面值。

这些家伙如何著名地清理自动取款机，关闭交通信号灯并在整个城市玩电-对我而言，这是不可思议的。 然后我想到，变得像黑客一样无所不能。

几年后，尽管我仍然没有Internet访问权限，但我还是购买了每一期《 Hacker》杂志并试图入侵五角大楼。


电影《黑客》中的英雄

对我来说，真正的发现已经是一台真正的PC，它带有15英寸的电子管监视器和基于Intel Pentium II处理器的系统单元。 当然，他是由他的叔叔买的，到90年代末，他的身高已经足够高，可以放下自己的玩具了。

我第一次打开某种游戏，这并不太令人兴奋。 但是有一天，审判的日子到了，星星连续排在一起，我们来探望了一个不在家的叔叔。 我问：

“我可以打开计算机吗？”
“是的，和他一起做你想做的一切，”亲爱的姨妈回答。

当然，我做了他想做的事。 Windows 98桌面上有不同的图标。 WinRar，Word，FAR，围巾，游戏。 单击所有图标后，我的注意力转向了FAR Manager。 它看起来像一个难以理解的蓝屏，但是有一长串可以运行的（文件）。

通过交替单击每个按钮，我捕捉到了正在发生的结果。 有些东西起作用了，有些却没有。 一段时间后，我意识到以“ .exe”结尾的文件是最有趣的。 他们会发布不同的炫酷图片，您还可以在其中隐藏自己。 所以我可能在叔叔的计算机上启动了所有可用的exe文件，然后它们几乎把我从一个超级有趣的玩具中拉了过来，然后带我回家。


相同的FAR经理

然后有电脑俱乐部。 我和我的朋友经常去找他们在网络上玩《反恐精英》和《雷神之锤》，这在家里是做不到的。 我经常问我的父母在俱乐部里玩半小时。 他们看到我像史瑞克的猫一样，给了我又一份丰厚的合同。 我完成学业没有三倍，他们给我买了一台电脑。 该合同于今年9月初签订，令人垂涎的PC应该已经在6月到达，并且要遵守协议。

我尽力了 他甚至卖掉了自己心爱的Sony Playstation，以减少对学习的注意力。 尽管我马马虎虎学习，但是九年级对我来说意义重大。 鼻子里有血，只需要取得好成绩。

早在春天，就打算购买一台PC，这可能是我一生中最重要的事件。 我试图超前考虑，因此有一天对父亲说：

“爸爸，我不知道如何使用计算机。” 让我们注册课程。

言归正传。 我父亲打开报纸刊登广告，发现一个小字样的标题为“计算机课程” 。 我给老师打了电话，过了几天，我已经参加了这些课程。 这些课程在城市的另一面，三楼的赫鲁晓夫旧面板上举行。 在一个房间里，连续有三台PC，实际上，对感兴趣的人进行了培训。

我记得我的第一堂课。 Windows 98加载了很长时间，然后老师发言：

-是的 这是Windows桌面。 上面有程序图标。 在开始按钮的底部。 记住！ 所有工作都从“开始”按钮开始。 左键单击它。
他继续。
-在这里-您会看到已安装的程序。 计算器，记事本，Word，Excel。 您也可以通过单击“关机”按钮关闭计算机。 试试看。
最终，他当时对我来说是最困难的部分。
-老师说，在桌面上，您还可以查看可以通过双击启动的程序。
-加倍！ -那怎么样？
-试试吧。 用鼠标左键双击以启动记事本。

是的，现在。 那时最困难的事情是将鼠标放在一个位置，同时快速双击。 在第二次单击时，鼠标略微抽搐，并且快捷键随之改变。 但是，尽管如此，我们仍在本课程中成功地完成了这一无法克服的任务。

然后进行了Word，Excel的培训。 有一天，它们只是让我浏览自然和建筑遗迹的图片。 这是我记忆中最有趣的活动。 比学习在Word中格式化文本要有趣得多。

在我的电脑附近，其他学生也在学习。 有几次我遇到了写程序的家伙，同时还热烈地讨论了这个过程。 它也使我感兴趣。 记得电影《黑客》和无聊的MS Office，我要求转学编程课程。 就像生活中所有重大事件一样，这是自发发生的，出于兴趣。

我和妈妈一起上了第一堂编程课。 我不记得为什么。 显然，她应该已经同意新课程并获得报酬。 院子里是春天，天已经黑了。 我们乘坐小巴瞪羚穿越整个城市，到达郊区，
小组赫鲁晓夫，上楼让我们进去。

他们把我放在一台极端的计算机后面，并打开了一个完全是蓝色屏幕和黄色字母的程序。

-这是Turbo Pascal。 老师对他的行为发表了评论。
-看，我在这里写了文档，如何工作。 阅读。

在我之前是一幅黄色的，绝对难以理解的文字。 我试图为自己找到一些东西，但我做不到。 中国语法就是这样。

最后，一段时间后，课程负责人递给我一张A4打印纸。 这是我以前在编程课程的人的监视器上瞥见的某种古怪的东西。
-重写这里写的内容。 他命令老师离开了。
我开始写：
程序摘要；

我在键盘上寻找英文字母时写的。 在Word中，我甚至接受过俄语培训，但是在这里我需要学习其他字母。 该程序是用一根手指打字的，但是非常勤奋。
开始，结束，var，整数 -这是什么？ 尽管我从一年级开始学习英语，并且知道许多单词的含义，但我无法将它们完全融合在一起。 像骑自行车的受过训练的熊一样，我继续踩踏板。 最后是一些熟悉的东西：
writeln（'输入第一个数字'）;
然后-writeln（'输入第二个数字'）;
然后-writeln（'Result ='，c）;


第一个Turbo Pascal程序

h，写道。 他从键盘上移开手，等待上师出现，以获取进一步的说明。 最终，他站起来，将视线移过屏幕，并说要按F9键。
-大师说，现在程序已编译并检查错误。
没有错误。 然后他说要按Ctrl + F9，这也是第一次需要逐步向我解释。 您需要按住Ctrl，然后按F9。 屏幕变黑，最后一个清晰的输入出现在屏幕上：“输入第一个数字”。
在老师的命令下，我输入了7。然后输入第二个数字。 我输入3，然后按Enter。

屏幕上立即出现一行“结果= 10”。 那是欣喜若狂，我以前从未经历过类似的经历。 好像整个宇宙都在我面前打开了，我最终进入了某种门户。 温暖通过身体，我的脸上露出笑容，在我的潜意识深处的某个地方，我意识到那是我的 。 非常直观地，在情感层面上，我开始感觉到桌子底下这个嗡嗡作响的盒子具有巨大的潜力。 您可以用自己的双手做多少，她一定会做！

那是一种魔术。 我的头根本不合身，就像蓝屏上的黄色不易理解的文字变成了一个方便易懂的程序。 哪个仍然重要！ 计数本身并不令我感到惊讶，而是书面象形文字变成了计算器。 在这两个事件之间，那一刻有一个深渊。 但是凭直觉，我觉得这块铁几乎可以做所有的事情。

我几乎乘坐小巴回家，就像在太空一样。 这张照片上刻着“结果”的字样在脑海中旋转，这台机器仍然可以运转的情况如何，我能不用一张纸就自己写点东西吗。 数以千计的我感兴趣的问题同时激发了我的灵感。 我当时14岁。 那天，专业选择了我。
