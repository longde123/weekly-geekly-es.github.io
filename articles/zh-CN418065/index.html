2018 MBLT DEV会议新闻：Android Track

哈Ha！ 我是Miguil Barannikov，e-Legion的Android团队负责人，也是MBLT DEV 2018程序委员会的成员。 当您享受假期或在办公室里有空调时，我们为即将举行的会议选择报告。

顺带一提，我将向您介绍一些已经确认参加的出色演讲者，我认为他们将分享去年报告中最好的演讲者。



机器学习，UI，Google助手


只有懒惰的人不会谈论机器学习，我们还会谈论：Harshit Dwivedi将教参加者如何使用Firebase ML Kit中可用的API。

Artyom Rudoy将讲述他们如何在Badoo中引入统一的设计系统并加速UI的开发，同时达到开发人员和设计师之间的最佳理解。

Google的Dina Sidorova将继续I / O 2018的趋势，并谈论Google Assistant的工作。 顺便提一提，狄娜（Dina）向您提出了一个问题：您对听什么特别感兴趣？ 也许有人对在Google上使用Actions有疑问？ 发送问题 ，Dina在准备报告时将考虑这些问题 。



在会议上，我们还将学习如何在生产项目中应用协程，以及如何将当前应用程序重构为功能强大且稳定的体系结构中的新应用程序。

不仅会有报告


我们和会议的合作伙伴也进行测验和询问。 那年，我本人准备了关于Android开发的测验，我也计划重复这一测验。 奖品真棒。 例如，每年订购所有JetBrains产品。 每个人都可以参加-机灵的技巧和一点点知识。

最佳报告2017


去年，我和参加会议的人一起评估了报告。 最重要的是，我喜欢Uber开发人员的演讲。 Zac Sweers谈到了自动化和代码生成。 我共享视频。


您可以在有关Habré的文章的帮助下研究Zack的报告。 解密是由我的同事Marat准备的。

iOS在哪里？


但是他不是！ 但认真的是，同事Ilya Kulebyakin在另一篇文章中谈到了iOS技术。 由于iOS扬声器的发布越来越多，请参阅会议网站上的完整列表。
