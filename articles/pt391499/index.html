Remix OS para PC beta: como organizar a inicialização dupla com o Windows e começar com o novo sistema operacional



O Geektimes escreveu várias vezes sobre o Remix OS, um sistema operacional para desktop baseado no Android. Em 1º de março, apareceu uma versão beta deste sistema operacional, que já é bem possível de ser resolvida. De um modo geral, esta é uma versão personalizada do Android que parece e funciona como um sistema operacional completo para um computador pessoal. Foi lançado em 2015, com suporte mínimo para vários dispositivos. Em janeiro, a versão alfa do Remix OS foi lançada, os desenvolvedores a disponibilizaram para todos, abrindo a capacidade de inicializar o sistema a partir de um drive USB.

Na versão beta, o dual-boot apareceu, com a capacidade de alternar entre dois sistemas operacionais - o próprio Remix OS e o Windows. Você pode instalar o sistema operacional no disco rígido e no SSD, o que fornece um desempenho bastante alto da plataforma. Agora, o instalador do Remix OS para PC suporta o trabalho com o Windows 7 e versões posteriores deste sistema operacional. Uma versão para Mac deve estar disponível em breve. Se o sistema operacional estiver cansado, removê-lo é ainda mais fácil do que instalá-lo.



As atualizações agora vêm regularmente, o suporte para atualizações sem fio é incorporado. Há outras novidades, incluindo a mudança para o kernel Linux versão 4.4 para melhor compatibilidade com processadores mais recentes, suporte para vídeo 1080p do YouTube e Netflix, trabalho com unidades USB de até 64 GB e mais.

Porque não Suporte Play Store. Como o Remix OS não é certificado pelo Google, não há acesso padrão ao mercado. Mas você pode adicioná-lo, mais sobre isso abaixo. Primeiro, vamos falar sobre a instalação do próprio Remix OS.



Não deseja instalar o Remix OS em um SSD ou disco rígido? Não é um problema. Você pode trabalhar com o sistema operacional diretamente de uma unidade flash USB, tornando-o inicializável. Mas, ainda assim, o desempenho do SO nesse caso será menor do que se você colocá-lo em um HDD ou SSD.



Como instalar o Remix OS no HDD ou SSD



1. Primeiro, baixe a versão mais recente do sistema operacional aqui . Existem compilações de 32 e 64 bits para os PCs correspondentes.

2. Descompacte o arquivo e obtenha três arquivos: um livro com instruções, um instalador e ISO, uma imagem de disco.

3. Inicie o Remix OS Installation Tool.exe e verifique se o local da instalação está correto (letra da unidade, etc.).



4. Clique em Avançar e selecione a imagem de disco que acabou de ser carregada. Será algo como o Remix OS para PC_x86_64-B2016xxxxxx.iso.

5. Iniciamos a instalação e esperamos alguns minutos até que apareça uma mensagem sobre a necessidade de reiniciar o PC para concluir a instalação.

6. Depois que a reinicialização ocorrer, você poderá ver uma tela com uma sugestão de escolha de dois sistemas.



7. Estamos esperando.

O próximo passo levará de 15 a 30 minutos, dependendo do desempenho do seu PC. O Remix OS cria sua própria área de inicialização de 8 GB e instala o sistema operacional nessa área.

8. Depois que tudo estiver pronto, o Remix OS será inicializado pela primeira vez e você poderá começar.

9. Se você precisar retornar ao Windows, selecione o ícone de desligamento e a opção de reinicialização. Depois disso, em vez do Remix OS, na tela de seleção, clique no Windows.

É possível obter mais de 8 GB?


O instalador oficial do Remix OS PC leva 8 GB para suas necessidades. Isso é suficiente para instalar vários aplicativos que você pode precisar para trabalhar ou se divertir.



Mas se de repente você precisar de mais espaço, poderá adicioná-lo .

Como remover Remix OS


Se de repente algo der errado, ou apenas quiser remover o Remix OS, isso pode ser feito muito rapidamente.

Inicialmente, execute o instalador novamente. Isso mostra que o sistema já está instalado no PC e oferece a remoção do Remix OS. Concordamos e, após alguns segundos, o sistema operacional é excluído.

Você pode trabalhar não com o instalador, mas excluir tudo manualmente. Na barra de endereço do gerenciador de arquivos, selecione C: \ Remix OS \ e execute o arquivo UninstallRemixOS.exe . Então tudo acontece no modo automático.
