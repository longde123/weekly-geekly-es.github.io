Selamat datang di JavaScript Meetup SuperJob 10 Oktober

Pada 10 Oktober, pertemuan para pengembang JavaScript akan diadakan di kantor SuperJob.

Pakar SuperJob akan berbagi pengalaman mereka mengoptimalkan aplikasi pada React / Redux, menggunakan tipe statis dengan Flow, dan bekerja dengan Node.js.
Secara tradisional, program ini mencakup burger dan komunikasi informal.

Tanggal dan waktu : 10 Oktober dari pukul 19:00 hingga 21:30
Alamat : Moskow, Malaya Dmitrovka, rumah 20
Pendaftaran : superjob-meetup.timepad.ru/event/1063170/



Pembicara:

1. Alexander Kukhta , pemimpin pengembang SuperJob dengan tema "Pengalaman dengan JSON: API pada klien" :

  • Bagaimana kami menyimpan 1.500 entitas mentah pada klien
  • Bagaimana kami bekerja dengannya dalam komponen Bereaksi
  • Dan cara membuatnya tidak melambat

2. Pyotr Solopov , pemimpin kelompok pengembangan SuperJob dengan tema "Type Systems and Frontend Typing" :

  • Jenis sistem apa yang ada dan bagaimana mereka dikembangkan
  • Mengalir vs TypeScript
  • Bagaimana kami menggunakan Flow dalam proyek besar

3. Alexander Savin , manajer pengembangan frontend SuperJob dengan tema "Membuat server JSON: API di Node.js dan PostgreSQL" :

  • Alat apa yang akan membantu Anda dengan cepat mendapatkan API untuk basis data yang ada
  • Cara membuat alat seperti itu sendiri menggunakan Node.js dan PostgreSQL
  • Kesulitan apa yang mungkin timbul ketika menggunakan pendekatan "DB pertama" dan apa yang dapat membantu mengatasinya

Penerimaan gratis , tetapi agar Anda memiliki ruang yang cukup, silakan mendaftar .

Pada hari acara, harap ingat untuk membawa paspor atau SIM Anda. Ini diperlukan untuk akses ke pusat bisnis kami.
