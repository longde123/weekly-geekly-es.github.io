Anatomi Platform Hosting RUVDS Huawei FusionServer

Huawei Technologies, yang dikenal terutama sebagai penyedia peralatan jaringan dan telekomunikasi, lebih dari lima tahun yang lalu meluncurkan serangan hebat di pasar server global, yang bertujuan untuk meningkatkan kesadaran merek di kalangan pelanggan server korporat. Selain itu, perusahaan berusaha untuk menciptakan keunggulannya sendiri melalui investasi yang mengesankan dalam penelitian dan pengembangan.

Subjek tinjauan hari ini adalah server rak dual-socket 2U Huawei RH2288 V3 . Menurut pabrikan, tugas utamanya adalah pemrosesan dan penyimpanan data aplikasi penting dalam ruang rak kecil. Server Huawei RH2288 V3 cocok untuk bekerja dengan aplikasi Internet dan sistem korporat penting, untuk pemrosesan dan penyimpanan data besar, komputasi awan, dan penyediaan layanan telekomunikasi. Mereka digunakan oleh hosting dan penyedia layanan VPS, misalnya, RUVDS, mitra resmi Huawei.



Secara khusus, sebagai bagian dari proyek bersama, vendor melengkapi pusat data RUVDS di Rusia, Inggris dan Swiss dengan peralatannya. Saat ini, lebih dari 200 server RH2288 V3 telah diinstal di pusat data perusahaan. Ini memastikan operasi server virtual yang cepat dan stabil, memungkinkan Anda untuk menawarkan kepada pelanggan layanan berkualitas tinggi dan modern. Selain itu, platform server yang andal ini memungkinkan Anda untuk mengoptimalkan biaya layanan yang diberikan kepada pelanggan. Platform hosting RH2288 V3 ideal untuk hosting dan karenanya telah menjadi pilihan logis untuk melengkapi pusat data RUVDS. Kita akan mengenal server-server ini dalam ulasan ini.

Para pengembang RH2288 V3 mencoba untuk mencapai rasio kinerja dan kepadatan komponen komponen yang optimal. Server menyediakan kapasitas penyimpanan yang besar, opsi ekspansi, dan daya komputasi yang cukup tinggi. Pesaing terdekatnya adalah HP DL380p Gen9, IBM x3650 M5, Dell R730xd dan Cisco C240 ‚Äã‚ÄãM4.

Untuk meningkatkan efisiensi energi, ia mengimplementasikan teknologi manajemen daya cerdas: manajemen energi dinamis; penyediaan sumber daya catu daya dan pendinginan jika perlu; regulasi diferensial-proporsional-integral-untuk mengontrol proses perpindahan panas dan mengontrol kecepatan putaran kipas. Kecerdasan server tidak terbatas pada hal ini: sistem pencatatan data membantu untuk menemukan masalah yang menyebabkan kegagalan.

Menggunakan panel LED, Anda dapat melakukan operasi kontrol lokal, dan melalui modul iBMC independen, Anda dapat melakukan tugas jarak jauh menggunakan konsol serial (Serial over LAN, SoL), KVM untuk akses jarak jauh, serta fungsi lainnya. Pengontrol kontrol iBMC dikembangkan pada chip Huawei sendiri dan dilengkapi dengan server secara gratis.

Seperti yang Anda lihat, tata letaknya cukup ketat. RH2288 V3 menyediakan dua slot PCIe 3.0 x8 untuk kartu PCIe standar dan mendukung berbagai jenis riser:

1. Riser 1 dengan tiga slot PCIe3.0 x8 untuk perangkat berikut:

- Satu kartu PCIe 3.0 x16 panjang penuh (bandwidth: PCIe 3.0x8).
- Satu kartu PCIe 3.0 x8 ukuran penuh.
- Satu kartu PCIe 3.0 x8 ukuran penuh.

2. Riser 2 dengan dua slot PCIe3.0 x16 untuk perangkat berikut:

- Satu kartu PCIe 3.0 x16 tinggi-tinggi penuh.
- Satu papan PCIe 3.0 x16 panjang penuh (bandwidth: PCIe 3.0x8).

Modul I / O mendukung konfigurasi berikut:

  • Dua slot PCIe 3.0 x8 ukuran tinggi penuh ukuran penuh dan satu slot PCIe3.0 x8 tinggi penuh ukuran standar.
  • Dua slot PCIe 3.0 x16 ukuran penuh ukuran penuh (bandwidth salah satunya: PCIe 3.0 x8).
  • Dua slot 2,5 inci untuk hard drive dan satu slot PCIe 3.0 x16 panjang penuh standar dengan panjang penuh.
  • Dua slot 3,5 inci untuk hard drive.



Prosesor terintegrasi dengan pengontrol memori dan pengontrol PCIe. Server mendukung prosesor berikut:

  • Intel Haswell-EPE5-2600 v3 8-, 10-, 12- atau 14-core prosesor dengan paket termal maksimum (TDP) dari 135 watt.
  • Intel Broadwell-EP E5-2600 v4 8-, 10-, 12-, 14-, 16- atau 18-core dengan TDP hingga 135 watt.

Anda dapat menggunakan akselerator grafis GPU.

Fitur server adalah bahwa ia mengandung riser reguler untuk memasang papan tambahan.


Di sebelah kiri adalah baterai pengontrol. Pengontrol memiliki cache dengan kapasitas 2 GB. Melalui SAS expander, itu terhubung ke drive.


Di dalam server, Anda dapat menginstal 16 modul memori.


Server memiliki modul memori 32 GB yang diproduksi oleh SK Hynix.


Di tengah adalah pengontrol kepemilikan iBMC yang sama.


Super kapasitor digunakan untuk melindungi data cache RAID dari kegagalan daya.

Sebagai kesimpulan, kami mencatat trik vendor yang menarik: dua kartu SD 32 GB dapat digabungkan menjadi RAID 0. Jika satu kartu gagal, yang kedua akan terus berfungsi. Kartu-kartu ini menyimpan informasi yang dicatat oleh pengontrol kontrol iBMC, yaitu, jika terjadi kegagalan server, informasi tersebut diatur ulang ke kartu, dan Anda dapat melihat apa alasan kegagalannya. Pesaing tidak memiliki solusi seperti itu, atau sangat mahal. Secara umum, server Huawei setidaknya berfungsi seperti pesaing.

Solusi pusat data Huawei membantu penyedia hosting, pemilik pusat data menumbuhkan bisnis mereka dengan biaya optimal, penyebaran cepat, manajemen cerdas, dan keandalan. Jadi, misalnya, dalam RUVDS selama ini bekerja dengan peralatan ini, tidak ada server atau komponennya yang gagal, menciptakan situasi darurat atau, bahkan lebih lagi, kehilangan data. Hanya kasus terisolasi dari masalah dengan RAM dan hard drive yang dicatat, yang diselesaikan dengan sangat cepat berkat jaminan Hari Kerja Berikutnya. Hard drive dalam array RAID adalah hot swappable, dan memori utama diganti sekali dan dengan downtime minimal bagi klien untuk me-reboot server.
