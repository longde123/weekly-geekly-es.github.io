三星领导层谈到了Galaxy Note 7自燃的原因



三星的最新旗舰产品Galaxy Note 7 失败了。此设备进入市场后，一切都出了问题。购买电池的用户开始在网络上发布有关电池自燃的信息。在仅一周的时间内，仅在美国，就记录了五起电话晒黑事件，并且一架智能手机在起飞前在西南航空的一架飞机上起火。发生此事件后，许多航空公司在飞机起火时安装了特殊的集装箱。

经过短暂的试用后，三星意识到了问题所在，称其原因是缺乏电池保护和生产过程中的某些缺陷。因此，提供了Galaxy Note 7拥有者以更新的手机替换他们的手机，并增强了电池保护功能。但是，经过一段时间后，“受保护”手机的所有者开始抱怨电池同样会自燃。

该公司最终从市场上召回了这些设备，并开始研究问题。这是由韩国公司的内部专家和相关专家共同完成的。显然，许多媒体试图猜测原因，做出自己的假设。主要版本是绝缘不良或电池生产过程中的缺陷。

当然，售出的数百万个设备中有少量点亮了设备-并不是很多。但是谁愿意赢得“俄罗斯轮盘”游戏呢？ 9月初，记录到35起银河7起火，韩国17起，美国17起，台湾发生1起。之后，该公司宣布将免费出售所有已售出的手机（约250万部）。该公司在正式销售这些小工具的国家/地区停止了生产，减少了销售并召回了智能手机。

设备的所有者收到了带有手套和袋子的耐火盒。假定必须先关闭设备，然后将其放入防静电袋中，再放入常规盒中，再将其放入防火盒中。说明指出，由于存在着火的危险，请勿将盒子中的物品携带在飞机上。 UPS交付服务已连接到转发设备。

显然，整个故事对公司业务产生了极大的负面影响。2016年10月11日，在得知召回出售的智能手机后，三星股价立即下跌了9％。结果，该公司的市值减少了170亿美元，而在2017年这一下降趋势仍在继续，该公司仍然遭受亏损。目前，如果用户退回以前购买的所有智能手机，三星将损失约22亿美元，加上资本损失，总计将达到190亿美元。所有用户退回智能手机的可能性不大，但无论如何，损失都是巨大的。

情况有所好转


如上所述，一个庞大的专家团队一直在寻找导致问题的可能原因。 1月23日，在首尔举行了有关Galaxy Note 7问题的新闻发布会。在这次新闻发布会上，三星电子移动部门负责人Dzhondin Koch 举行了记者招待会。他谈到了工作进展情况。

该公司的工程师与第三方专家共同研究了20万多种设备和3万个电池。我们检查了Galaxy Note 7的硬件和软件部分。此外，系统组装的所有阶段都经过了仔细验证，对工艺流程和交付至实施点的交付过程均进行了质量控制。

最后的原因是能够确定的。事实证明这是“电池设计和小工具组装过程中的缺陷”。该公司表示，这应归咎于这一事件。为了确保以后不再发生此类情况，三星推出了多级安全协议和电池验证程序。

作为预期，问题是机身太薄，在设计智能手机组件的位置时会犯错误。 Note 7电池是扁平的锂离子电池，由钴镍氧化物制成的带正电的层和石墨的带负电的层组成。浸渍有电解质的聚合物材料将两层分开。与任何其他电池一样，聚合物层允许离子自由移动，在此期间会产生电流。但是，如果带正电和带负电的层接触，则会发生电解液的明显发热，甚至释放更多的能量，电池发热，并发生着火爆炸。如果按下Galaxy Note 7中使用的电池，则有时可能会发生上述情况。而且您不需要太费力。

聚合物非常薄，因此即使用户将智能手机放在口袋中然后坐在智能手机上，电池也会变得很热。在某些情况下，这足以实现自燃。三星工程师试图在安全性与使设备外壳尽可能薄的愿望之间找到界限。事故发生前15秒（上）和事故发生前1秒（下）的锂离子电池
