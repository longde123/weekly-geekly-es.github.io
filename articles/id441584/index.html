PHP Digest No. 150 (11 - 25 Februari 2019)


Seleksi segar dengan tautan ke berita dan bahan. Dalam masalah ini: aturan untuk memilih RFC di PHP Internal diubah, penerimaan aplikasi untuk laporan untuk Rusia Rusia 2019, ekstensi baru untuk mengimplementasikan eksekusi kode paralel, materi baru untuk pelatihan, video, sebagian alat yang berguna, dan banyak lagi yang dimulai.

Selamat membaca!



Berita dan Siaran


  • PHP Rusia 2019 - Pembicara bintang pertama yang akan diumumkan segera. Sementara itu, penerimaan aplikasi untuk laporan berjalan lancar. Hingga 25 Februari, tiket konferensi dapat dibeli dengan harga terbaik.
  • 16 Maret Badoo PHP Meetup: Tes dan kualitas kode. Pendaftaran terbuka

PHP internal


  • Proposal untuk menghapus ambang 50% untuk pemungutan suara diterima dan sekarang semua RFC membutuhkan mayoritas 2/3.
  • PEAR tidak akan diinstal secara default sejak PHP 7.4.
  • Tinjauan yang sangat baik tentang RFC yang tidak dapat diterima - Mengapa obat generik, lambda pendek, kelebihan metode, anotasi, dll. tidak pernah terealisasi dan apa ramalannya.
  • Xdebug Derick Rethans penulis berbicara tentang bug kecil di PHP <7.3.2

Alat-alatnya


  • krakjoe / parallel - Ekstensi baru dari pembuat pthreads dengan antarmuka yang disederhanakan untuk eksekusi kode paralel (multi-threaded). Baca lebih lanjut tentang motif untuk membuat ekstensi di pos .
  • willdurand / hateoas 3.0 - Perpustakaan dan bundel populer untuk mengimplementasikan HATEOAS dalam layanan REST.
  • liip / serializer - Paket untuk (de) serialisasi data. Kompatibel dengan jms / serializer dan lebih cepat dari yang terakhir. Baca lebih lanjut tentang kinerja di pos .
  • TBPixel / xml-streamer - Paket untuk streaming membaca data XML, kompatibel dengan PSR-7.
  • phpstan / phpstan-doctrine - Ekstensi PHPStan untuk mendukung ORM Doktrin, termasuk analisis DQL .
  • spatie / enum - Implementasi tipe enumerated dan mendukung postingan tentang enum di PHP .
  • kriswallsmith / Buzz - Klien HTTP ringan.
  • thecodingmachine / graphqlite - Kerangka GraphQL lain untuk PHP. Posting pengantar dengan ulasan dalam dukungan.
