8 cloud log analyzer untuk mengevaluasi lingkungan kerja

Peningkatan jumlah perangkat, peningkatan persyaratan kualifikasi, dan optimalisasi proses pengumpulan data diperlukan untuk semua jenis bisnis. Analisis log dapat memberi Anda gambaran nyata tentang apa yang terjadi di lingkungan informasi Anda. Berikut ini beberapa contoh waktu nyata:

  • perencanaan kinerja
  • deteksi dini masalah
  • pelaporan saat ini
  • manajemen ketersediaan

Jika Anda memiliki beberapa megabytes file log, maka mereka dapat dilihat secara manual, tetapi ketika Anda bekerja di bisnis menengah dan perusahaan, di mana volume log dihitung dalam gigabytes, maka analisis manual berubah menjadi mimpi buruk.



Jika Anda mencari solusi untuk memantau dan menganalisis log aplikasi web Anda, termasuk infrastrukturnya, Anda dapat mempertimbangkan alat berikut untuk bekerja dengan log. Sebagian besar dari mereka memiliki versi gratis yang sedikit terpotong, di mana Anda dapat mencoba untuk bekerja.

Loggly


Loggly adalah salah satu solusi manajemen log paling populer, cocok untuk menganalisis data dari hampir semua sumber, termasuk yang berikut:

  • aplikasi web - Apache , Nginx, PHP, Node.js, .NET, Java, JavaScript
  • sistem operasi - Linux, Windows

Dengan menggunakan Loggly, Anda dapat mengumpulkan data (log) untuk dianalisis dan membuat dasbor informatif untuk memantau indikator yang diperlukan. Tentu saja, Anda dapat mencari log dari antarmuka Loggly.



Loggly memiliki mesin pencari yang kuat di mana Anda dapat mengonfigurasi dan mengatur filter untuk semua teks, satu bidang, dan nilai Boolean. Anda juga dapat mengikat email, PageDuty, Slack, HipChat, dll. Untuk menerima peringatan.

Sumologis


Dapatkan analitik waktu nyata dengan SumoLogic untuk memantau, menganalisis, memecahkan masalah, dan melindungi aplikasi dan infrastruktur bisnis.

SumoLogic memiliki agen pengumpul / pengumpul cloud yang harus diinstal di server Anda untuk mentransfer log ke SumoLogic. Anda dapat mengonversi semua jenis log, membuat gambar visual tentang apa yang terjadi, diperbarui secara waktu nyata, untuk memantau dan memberi tahu Anda ketika batas yang ditentukan tercapai.



SumoLogic memiliki beberapa aplikasi yang telah dikonfigurasikan berdasarkan pada aplikasi yang paling umum untuk mempercepat tugas dan mengeksplorasi masalah.

Contoh aplikasi untuk:

  • mongoDB - pelacakan kondisi umum, optimisasi kueri, pemantauan keamanan;
  • AWS Lambda - pemantauan proaktif, tinjauan biaya, studi mendalam;
  • AWS - identifikasi masalah kinerja, kontrol keamanan aktif, analitik canggih;
  • Salesforce - peningkatan adaptasi untuk pengguna, audit, perbandingan data;
  • Trend Micro - analisis kerentanan , riwayat acara, peningkatan visualisasi serangan;
  • Github - visualisasi kinerja tim, pembuatan indikator penggunaan dasar;
  • Docker - analitik lanjutan, pemecahan masalah;
  • Linux - pemecahan masalah dan optimalisasi OS, peringatan tentang akses / verifikasi dan peristiwa keamanan, pelacakan indikator kinerja;
  • Nginx - ulasan, lokasi pengunjung, waktu respons , identifikasi hambatan dalam kinerja;
  • Apache - interaksi dengan pengunjung, optimalisasi kinerja, visualisasi pengunjung;
  • IIS - informasi komprehensif, statistik lalu lintas, mencari halaman lambat;
  • MySQL - keadaan umum, kesalahan utama, pemantauan kinerja permintaan.

Dalam versi gratis Anda akan menerima 500 megabyte per hari dengan penyimpanan data 7 hari dengan semua jenis sumber data yang dikumpulkan.

Splunk


Splunk Cloud tersedia di 10 wilayah AWS di seluruh dunia untuk mengumpulkan dan mengindeks semua jenis informasi. Splunk adalah cara cepat dan mudah untuk menganalisis dan memvisualisasikan data yang dibuat pada perangkat fisik, cloud atau virtual.



Dengan Splunk, Anda dapat melacak dan memvisualisasikan data dan menetapkan ambang batas untuk peringatan. Saya belum melihat versi gratis, tetapi mereka memiliki versi percobaan yang memungkinkan Anda untuk menganalisis dan memvisualisasikan data 5 GB selama 15 hari.

Papertrail


Papertrail dapat mengumpulkan data dari syslog, teks, Apache, MySQL, aplikasi Heroku, log peristiwa Windows, router, firewall, RubyOnRails dan banyak lagi.

Anda dapat mencari secara real time menggunakan browser, baris perintah atau API dan menerima peringatan melalui email atau Slack.



Logz.io


Logz.io menyediakan akomodasi ELK sebagai layanan. ELK (Pencarian Elastis, Logstash, Kibana) adalah salah satu platform analisis open source yang paling populer. Jika Anda mencari platform yang fleksibel untuk memproses log, maka Logz.io akan bermanfaat bagi Anda.

Logz.io dapat mengindeks data dari hampir semua teknologi modern dan memberi Anda analisis terperinci dengan visualisasi dan peringatan data.



Beberapa teknologi yang didukung:

  • Apache HTTP / Tomcat
  • Nginx
  • Hadoop
  • Heroku
  • Node.js
  • mongoDB
  • postgreSQL
  • Windows
  • Linux
  • Aws

Versi gratisnya menawarkan penyimpanan tiga hari dengan data 3 GB per hari.

Kayu


Pelacakan waktu-nyata, sistem penyaringan mutakhir, pencarian cepat, dan kemampuan untuk mengintegrasikan Kayu dengan aplikasi atau platform Anda.



Perpustakaan tersedia untuk Node, Ruby, dan Elixir dan mendukung banyak platform, termasuk yang berikut:

  • Logstash
  • Zeit (Sekarang)
  • AWS Lambda
  • Docker
  • Linux
  • Heroku
  • Kubernetes

Anda dapat memulai dengan versi gratis dan mencoba platform ini.

Logentries


Kirim data apa pun ke Logentries untuk pengambilan data langsung dan visualisasi. Anda dapat memantau log di antarmuka pengguna Logentries dan segera menerima peringatan tentang nilai yang diperlukan.

Lansiran dapat dikirim ke Slack, Hipchat, Pagerduty atau melalui webhooks. Logentries menyediakan REST API untuk menanyakan log Anda untuk visualisasi.



Dalam versi gratis, Anda dapat mengirim data hingga 5 GB, dan periode penyimpanan adalah tujuh hari.

Logsene


Sematext Logsene adalah ELK Stack Anda yang berbasis cloud untuk mengelola dan menganalisis log. Anda dapat mengirim log melalui saluran terenkripsi dari sumber apa pun, termasuk Syslog.

Ini berjalan pada AWS dan mendukung SOC, SSAE, FISMA, DIACAP, HIPPA, dll.



Coba demo untuk memahami cara kerjanya.

Saya harap analisa log berbasis cloud ini membantu Anda mencari, menganalisis, dan memvisualisasikan data Anda dan membuatnya lebih mudah untuk analitik TI.
