Skillbox Friday Webinar: Pengembangan dan Segalanya Tentang Ini



Jumat depan, yang berarti munculnya pilihan baru webinar gratis dari Skillbox on Habré. Video hari ini adalah tentang pemrograman:

  • Pengembangan aplikasi PHP.
  • Pengembangan aplikasi untuk iOS.
  • Bagaimana menjadi pengembang Android.
  • Alat otomatisasi untuk pengembangan ponsel.
  • Cara memulai pemrograman dengan Python.

Kami mengingatkan Anda: untuk semua pembaca "Habr" - diskon 10.000 rubel saat mendaftar untuk kursus Skillbox apa pun menggunakan kode promosi "Habr".

Cara mengembangkan aplikasi PHP



Ini adalah webinar dari insinyur Badoo terkemuka Mikhail Ovchinnikov. Dia memberi tahu programmer pemula bagaimana mengembangkan aplikasi dalam PHP. Artikel ini membahas alasan popularitas PHP sebagai bahasa untuk pengembangan backend, tingkat kesulitan aplikasi web dasar dan apa yang diperlukan untuk mengubah seorang programmer PHP menjadi seorang profesional.

Kursus online "Pengembang PHP dari 0 hingga PRO"

Cara mengembangkan aplikasi untuk iOS



CEO Anvics Nikita Arkhipov berbicara dan menunjukkan bagaimana aplikasi iOS sedang dikembangkan, menunjukkan kemampuan Swift dan Xcode. Kasus sentral adalah pembuatan aplikasi todo yang independen: bekerja dengan tampilan data dan memproses tindakan pengguna.

Bagaimana menjadi pengembang Android



Mikhail Ovchinnikov, seorang programmer Badoo terkemuka, berbicara tentang bagaimana menjadi pengembang Android profesional. Tingkat keterampilan dan pengetahuan untuk membuat aplikasi seluler pertama Anda, alat untuk pengembangan Android dan cerita tentang bagaimana seorang programmer peserta pelatihan dapat menjadi pemimpin tim di sebuah perusahaan internasional - inilah yang menanti Anda di webinar ini.

Kursus praktis "Pengembang seluler dari awal"

Alat Otomasi Pengembangan Seluler



Agima Mobile Development Manager Dmitry Shashlov berbicara tentang alat otomatisasi, termasuk integrasi terus-menerus, pengiriman berkelanjutan, pengujian otomatis dan alat untuk bekerja dengan kode dalam tim: gitflow, permintaan tarik dan tinjauan kode.

Kursus online praktis “PRO Pengembang Seluler”

Cara memulai pemrograman dengan Python



Mentor Programmer Skillbox Online, Vadim Shandrinov, berbicara tentang cara memprogram dengan Python. Webinar ini mengungkapkan kemungkinan bahasa, menunjukkan cara bekerja dengan fungsi dalam Python. Kasus praktisnya adalah pembuatan aplikasi web pertama. Selain itu, penulis membahas topik permintaan untuk pengembang Python di pasar tenaga kerja.

Kursus praktis "Pengembang Python dari awal"

Seluruh pilihan secara keseluruhan bersifat universal dan dapat bermanfaat bagi semua orang yang bekerja di bidang TI. Jika Anda tidak memprogram dalam bahasa yang disebutkan dalam webinar, tidak ada yang menghalangi Anda untuk belajar lebih banyak tentang mereka. Dan jika Anda memprogram - inilah saatnya untuk menyegarkan kembali pengetahuan Anda.
