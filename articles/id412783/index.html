Menginstal proksi MTProto Telegram dari sumber pada Centos 7


jalankan saja 'make'


Halo semuanya. Saya akan mengelola tanpa pengantar tambahan - banyak yang menunggu untuk rilis resmi, dan sekarang - mereka menunggu. Pada habr sudah ada beberapa artikel cara menginstal MTProxy dari buruh pelabuhan.

Tetapi dengan instalasi dari sumbernya, banyak yang terhenti - setelah hanya menjalankan kesalahan 'buat' dikeluarkan. Cara menginstal MTProxy dari sumber tanpa kesalahan pada Centos 7 - di bawah cut

Sumber unduhan 


cd ~ git clone https://github.com/TelegramMessenger/MTProxy

Memeriksa versi openssl 


 openssl version

Kemungkinan besar hasilnya akan seperti: 

 OpenSSL 1.0.2k-fips 26 Jan 2017

Memperbarui openssl ke 1.1.0


Hyde diambil dari sini

Instal alat bantu 

 sudo yum install libtool perl-core zlib-devel -y

Unduh kode sumber versi 1.1.0 

 curl -O -L https://github.com/openssl/openssl/archive/OpenSSL_1_1_0g.tar.gz

Buka kemasan dan buka folder 

 tar -zxvf OpenSSL_1_1_0g.tar.gz cd openssl-OpenSSL_1_1_0g

Kami merakit dan menginstal 

 ./config --prefix=/usr/local/openssl --openssldir=/usr/local/openssl shared zlib make make test sudo make install

Tambahkan versi baru ke PATH:

Buat file 

 sudo vi /etc/profile.d/openssl.sh

Konten: 

 # /etc/profile.d/openssl.sh pathmunge /usr/local/openssl/bin

> Nyalakan kembali shell

Buat file 

 sudo vi /etc/ld.so.conf.d/openssl-1.1.0g.conf

Konten: 

 # /etc/ld.so/conf.d/openssl-1.1.0g.conf /usr/local/openssl/lib

Terapkan 

 sudo ldconfig -v

Memeriksa versi openssl 


 openssl version 

 OpenSSL 1.1.0h 27 Mar 2018

Buka Makefile dan tambahkan
CFLAGS: -I / usr / local / openssl / include
LDFLAGS: -L / usr / local / openssl / lib 

 CFLAGS = -I/usr/local/openssl/include -m64 -O3 -std=gnu11 -Wall -mpclmul -march=core2 -mfpmath=sse -mssse3 -fno-strict-aliasing -fno-strict-overflow -fwrapv -DAES=1 -DCOMMIT=\"${COMMIT}\" -D_GNU_SOURCE=1 -D_FILE_OFFSET_BITS=64 LDFLAGS = -L /usr/local/openssl/lib -m64 -ggdb -rdynamic -lm -lrt -lcrypto -lz -lpthread -lcrypto

Simpan

Dan sekarang cukup membuatnya . 

 make clean make
