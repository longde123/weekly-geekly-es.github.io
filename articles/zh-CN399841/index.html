激进的设计让Galaxy Note 7电池爆炸


三星漂亮而危险的智能手机

三星已经投入了大量的精力和金钱来开发公司的最新旗舰产品三星Galaxy Note 7以及推广该产品。对于公司来说不幸的是，手机进入市场后，一切都按计划出了问题。在八月底在社交网络出现爆炸或几个报告火灾电池的手机。该公司承认该问题，并说这是由于电池制造过程中的某些缺陷所致。提供了所有Galaxy Note 7拥有者更换手机的功能，从而加强了电池保护功能。但这并不能解决火灾问题-已经修改过的智能手机开始爆炸。

对于任何技术公司来说，发生的都是一场噩梦。为了防止再次发生这种情况，三星继续分析发生了什么。第三方专家也研究情况。直到11月，才立即就可能起火的原因提出了一些建议。特别是，该公司的工程师建议在制造过程中将电池的各层压缩得太多。另一个假设是绝缘不良或电池生产过程中的缺陷。但似乎这里的问题不仅在于电池，还在于外壳和电话组件的设计。

到目前为止，该公司没有对此发表任何评论，但是大多数专家确信问题不仅仅在于电池，还不如手机壳的设计和元件放置的特征。为了确认这一点，Instrumental.ai员工购买了手机并决定将其拆解。为了确保他们自己的安全（并引起注意照片），在工作场所附近安装了灭火器。

展望未来，值得一提的是，这个问题已被人们确定。由于结构上的计算错误，在正常使用电话期间有时会挤压电池。

为什么重要呢？事实是，Note 7电池是扁平锂离子电池，由钴镍氧化物制成的带正电的层和带负电的石墨层组成。浸渍有电解质的聚合物材料将两层分开。与任何其他电池一样，聚合物层允许离子自由移动，在此期间会产生电流。但是，如果带正电和带负电的层接触，则会发生电解液的明显发热，释放出更多的能量，电池发热，并发生着火爆炸。如果按下Galaxy Note 7中使用的电池，则有时可能会发生上述情况。而且您不需要太费力。



为了使手机更薄，聚合物被制成很薄，看起来太薄了。结果，即使用户将手机放在裤子的侧袋中并坐下，电池也可能开始变热，更不用说如果用户将手机放在后袋中并直接坐在设备上会产生压力。在某些情况下，这足以加热和点燃电池。在某些情况下，电池在使用过程中会略微膨胀。但是外壳没有放置空间，增加的电池不断受到外壳元件的压力。

这里出现一个有趣的问题-制造商似乎已经接近边界，而现在已经很难在安全性要求和市场营销之间找到中间地带。为了满足喜欢“最薄，最高效，最...”的用户，技术人员必须做出让步。为了确保电池的安全性，该公司的工程师在外壳内部创建了一个特殊的耐用“口袋”。根据开发人员的说法，如果暴露于诸如压力或变形之类的电话因素下，该口袋应保持电池完好无损。但是由于表壳制造得太薄，因此该结构元素在所有情况下都无济于事。

很可能是在开发新手机的过程中，该公司决定更新电池制造过程。该手机的原型已使用以前的电池进行了测试，而该公司根本没有时间对新电池进行全面测试（为此需要大约一年和一千个电池）。这是导致当前令人遗憾的结果的方案之一。

让我提醒您，9月2日，公司宣布召回公司收到的所有Galaxy Note 7，“安全”方的设备在飞机上爆炸后，公司必须完全停止生产并召回所有设备。
