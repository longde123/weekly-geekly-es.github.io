Annonce de l'extension des outils Azure IoT Edge (préversion)

Nous sommes heureux d'annoncer la disponibilité de la version d'aperçu de la nouvelle extension pour Visual Studio 2019 - Azure IoT Edge Tools . L'extension fournit un ensemble complet de fonctionnalités pour prendre en charge le développement de solutions IoT Edge à l'aide de Visual Studio 2019:



  • Nouveau projet Azure IoT Edge conçu pour diverses plates-formes (Linux amd64, Linux arm32v7, Windows amd64)
  • Nouveau module IoT Edge (C # / C) ajouté à la solution
  • Modifier, créer et déboguer des modules IoT Edge localement sur votre ordinateur Visual Studio
  • Créer et télécharger des images Docker pour les modules IoT Edge
  • Exécuter des modules IoT Edge dans un simulateur local ou distant
  • Déployer des solutions IoT sur des appareils IoT Edge (à l'aide de Cloud Explorer )



Prérequis


  • Visual Studio 2019 : les charges de travail «Développement de bureau .NET» et «Charge de travail de développement Azure» doivent être installées; si vous prévoyez de développer des modules C, vous devez également installer «Développement de bureau Windows avec C ++»
  • Docker Desktop . Vous devez configurer correctement Docker CE en mode conteneur Linux ou conteneur Windows.
  • Pour configurer votre environnement de développement local pour le débogage, l'exécution et le test de la solution IoT Edge, vous avez besoin de l' outil de développement Azure IoT EdgeHub . Pour ce faire, installez d'abord Python (2.7 / 3.6) , puis installez iotedgehubdev en exécutant la commande ci-dessous dans votre terminal. Vérifiez que la version de votre outil de développement Azure IoT EdgeHub est supérieure à 0.8.0.

pip install --upgrade iotedgehubdev

L'installation


Il existe deux options pour installer une nouvelle extension:

  1. Téléchargez et installez la nouvelle extension à partir de Visual Studio Marketplace.
  2. Vous pouvez également installer l'extension directement à partir de Visual Studio 2019 à l'aide du menu Extensions -> Gérer les extensions . Dans la fenêtre Gérer les extensions, sélectionnez En ligne dans le volet gauche et entrez edge dans le champ de recherche dans le coin supérieur droit pour rechercher et télécharger «Azure IoT Edge Tools for VS 2019 [Preview]».



Comment utiliser cette extension?


Pour commencer, nous vous recommandons de vous référer aux ressources suivantes:

→ Utilisez Visual Studio 2019 pour développer et déboguer des modules pour Azure IoT Edge (préversion)
→ Développer et déboguer facilement des modules Azure IoT Edge C avec les outils Azure IoT Edge
→ Dépôt de documents Visual Studio Azure IoT Edge Tools
