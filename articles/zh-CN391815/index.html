小米Mi5-第一次接触中国工业

欢迎大家访问iCover博客。小米的新奇历史开始。自上一个旗舰产品问世以来，已经过去了不到一年半的时间...在这段时间里，电子巨头小米设法不仅仅一次吸引了粉丝，还吸引了使用多种不同设备的用户，从预算新颖到几乎是顶级解决方案。甚至有一些移动主题之外的设备都很好地进入了市场，并吸引了媒体和简单的“用户”（他们的Mi Band）的广泛关注，它们已经无处不在。而现在，终于到了在CES 2016上展示创新的新王冠的时刻-小米Mi5。



通常情况下，在展览会前不久，新产品的几乎所有内容都在互联网上广为人知……包括特征：

•表壳材料：金属和玻璃（大猩猩玻璃4）；
•显示：5.2英寸英寸IPS矩阵，分辨率为1920 x 1080像素，ppi430。
•平台：Qualcomm Snapdragon 820，视频芯片-Adreno 530；
•RAM：3或4 GB；
•操作系统：Android 6.0 + MIUI 7；
•用于存储数据的内存：32/64/128 GB（存储卡没有插槽）；
•网络：GSM / EDGE，UMTS / HSDPA，LTE（TD / FDD-LTE）（DualSIM）
•主摄像机：16 MP，f / 2.0，相位检测自动对焦，光学图像稳定，双LED闪光灯，视频以4k录制；
•前置摄像头：4 MP，f / 2.0，无自动对焦，视频以1080p录制；
•接口：Wi-Fi（ac / a / b / g / n）双频，蓝牙4.2（A2DP，LE），NFC，红外，C型USB（USB 3.0），AUX，GPS / GLONASS，指纹扫描仪
•电池：3,000 mAh（不可拆卸）；
•尺寸：144.5 x 69.2 x 7.3毫米；
•重量：129克（陶瓷背面版本为139克）。

先进的功能，漂亮的外观和一般的背景相当诱人，价格约为440美元。习惯于将低价和高质量相结合，人们总是开始期盼更多，并且要求过高。我们将更详细地了解新产品，该新产品不仅功能强大，而且还将在2016年与A品牌的旗舰店积极竞争。不幸的是，只有两部电话被带到了CES展览会上，无数的记者聚集在那里。而且由于无法进入笔鲨行列，我建议考虑一个来自中国的零售样本，并成功落入我们手中一天。

设计施工



几乎所有小工具的出现都经常受到舆论的批评。某些设备的爱好者坚持认为，发布智能手机的“公司”使其与您喜欢的小工具极为相似，反之亦然。这些战争在论坛上，您几乎总是可以在文章下的评论中看到，当然小米Mi5也不例外。但是，官方图片确实给人一种感觉，即小工具的外观清楚地重现了三星近年来臭名昭著的系列产品Galaxy Edge的设计功能。但是，拿起智能手机后，小米Mi Note会迅速清晰地被记住。例如，他具有相同的圆形背景。是的，并且在前面它们非常相似，只有Mi5更紧凑。而且，有趣的是，小米Mi Note的推出时间比三星Galaxy S6 Edge早了两个月...



开箱和配件详细













可以说小米没有明确地但统一其产品设计吗？它们都以一种或另一种形状相互接触，但同时它们在架子，桌子或生活中彼此看起来很和谐。奇怪的是，事实本身并没有特别令人讨厌。



考虑到所有功能，Mi5设计符合我的喜好。智能手机看起来简洁，静态和外观比其他智能手机更具代表性。另一方面，该解决方案具有其缺点。为了获得良好的外部和触感，您将不得不牺牲对摔倒的抵抗力。由于玻璃是前后使用的，实际上，没有什么可以保护它的。沿铝边的边缘。顺便说一句，与背面不同，前面板的制作没有2.5D舍入效果-顺便说一下，这是第四代康宁大猩猩玻璃。还值得一提的是，在最大配置下，后玻璃盖将由陶瓷代替。仅有的两个很容易弄脏，特别是在黑色版本的智能手机中。在白色版本中，污染不太明显。嗯，可能值得注意的是智能手机的整体滑行性-尽管在整天的测试中，他从未尝试过从我的手中滑向地板。的确，从书上的桌子上，它无需外部帮助就可以完美滚动。



也许对手的这种信心可以带来尺寸和舒适的抓地力。尽管对角线尺寸为5.15英寸，但该智能手机非常紧凑，非常适合您的手掌。用一只手玩这个小工具很高兴，我什至都不会隐藏它。也许是我最后一次遇到类似的感觉时，手里拿着苹果iPhone 5S，但是屏幕明显更小，侧面的空白更大。就屏幕而言，智能手机使用分辨率为1920 x 1080像素的IPS矩阵。很好的情况是，制造商没有跟随营销人员的领导，也没有为智能手机配备分辨率为2K或4K的矩阵-毕竟，全高清几乎是一部五英寸的智能手机，但会对性能和运行时间产生明显影响。结合新的高通芯片组，这种解决方案将在未来几年内提供一定的性能优势。至于基质的质量-极佳，颜色明亮多汁。有时甚至看起来太汁多了……几乎就像AMOLED。



控件和其他所有人一样，都位于标准位置。右边是电源/锁定键和音量键。左侧是SIM卡插槽。在AUX插孔的顶部，还有一个附加的麦克风和红外线。底部有新的USB C型连接器，侧面有整齐的孔，用于放置麦克风和扬声器。相机的背面，冷热双LED闪光灯和下面的MI标志。从上方看，玻璃凹口中有一块光线和接近传感器，一个前置摄像头和一个扬声器格栅。



细节续













下面，我们正在等待第二个最引人注目的创新。首先，将侧面触摸按钮“ menu”和“ back”的熟悉的象形图更改为两个发光LED。主页按钮现在已成为硬件按钮，但现在还包含指纹扫描器。此外，可以按下或敲击该键，然后传感器将工作。按下操作清晰明了，进度小，没有按钮播放，没有明显的单击。网络上的许多人对此决定做出了模棱两可的反应，但是乍一看，它似乎生活中很方便，而且看起来很有机。就个人而言，我很高兴公司最终摆脱了代表按键操作的通常的象形图，因此更具吸引力。
最后，我想提一下可用的颜色解决方案是白色，黑色和金色。

硬件潜力和软件



毫不夸张的说，硬件平台很好。这是Qualcomm Snapdragon 820四核芯片组，3或4 GB的RAM，32/64或128 GB的主闪存。而Adreno 530芯片负责图形处理……这就是所有乐趣的开始。



事实证明，将有三个版本的智能手机。Mi5 basic的第一个版本是UFS 2.0标准的32 GB主内存，3 GB双通道可操作LPDDR4内存，其工作频率为1333 MHz。Snapdragon 820处理器的核心频率高达1.8 GHz，Adreno 530图形处理器的工作频率为510 MHz。

第二个版本，即所谓的高级版本，已经具有相同UFS 2.0标准的64 GB闪存，以及相同的3 GB RAM，但已经以1866 MHz的频率运行。四核Qualcomm Snapdragon 820的最高时钟频率为2.15 GHz，而Adreno 530显卡的最高频率为624 MHz。

还有最新，最“高端”的版本，顺便说一句，它带有陶瓷后盖，并且完全是黑色的。它包含：四个超频至2.15 GHz的Qualcomm Snapdragon 820内核和未更改的Adreno 530显卡，时钟速度高达624 MHz。 128 GB的主闪存驱动器标准UFS 2.0和4 GB的RAM LPDDR4，其工作频率为1866 MHz。

正如我在上面的部分中所述，此类特征很可能足以在一年多的时间内保持领先地位。不幸的是，我们没有在综合测试中测试智能手机，因为没有足够的时间余地并且安装Play商店存在一些问题。但是，我认为这些值会很高。无论如何，您都可以随时返回查看未来的智能手机。



智能手机还有什么？ Wi-Fi模块802.11ac，4.2蓝牙版本，NFC和红外，支持A-GPS技术的GPS，GLONASS和北斗系统，在中国以外的地区并不普及。智能手机既可以在2G和3G网络中运行，又可以在4G网络中使用-支持本地频率。减号-不支持存储卡，但智能手机可立即与两张SIM卡配合使用...但是仍然有一个无线电模块。对于2015年和2016年的大多数旗舰店，使用的电池是最常见的电池，容量为3,000 mAh。从原则上讲，这样的电池足以满足一天中度使用智能手机的需要。在某些功能强大的3D玩具中，可以肯定可以在6个小时内将其放电，但是，如果可能，我们将检查此信息以返回智能手机更长的测试时间，但目前这只是一个假设。此外，还可以注意到快速充电功能的存在。



该智能手机运行最新的可用Android 6.0版本，以及熟悉的品牌外壳MIUI 7版本。如果您以前已经熟悉MIUI系统，那么很难说出一些新的东西，那么从这些创新中您会看到一些小的但有用的改进。该系统稳定，平稳地工作，所有动作都可以快速解决-这是智能手机优化和整体性能的优点。该外壳看起来简洁，有点类似于iOS的意识形态。所有标准应用程序均以与系统相同的样式进行制作，并且其数量不超出合理范围。有中文服务，但单击两次即可轻松删除。可能令人失望的是缺少开箱即用的Google服务，俄语以及可能受阻的引导程序。仅此而已。

说说相机？





让我们从规格开始。总共有两个摄像头。带有光圈2.0和相位自动对焦的主要16.0 MP是Sony IMX298传感器。使用超像素技术制成的Front 4.0 MP，像素大小为2 µm。光圈值相同-f / 2.0。记得小米Note出现在市场上的时间，该型号可以说是拥有出色相机的三款中国最佳智能手机之一。最好通过看下面的示例图片自己了解并得出一些结论。在我看来，Mi5绝对值得一提。当然，存在一些问题……在很大程度上，它们与自动对焦的工作以及在黑暗中拍摄时的画框本身的质量有关。下午，一切都很好。视频由智能手机使用主摄像头以4K录制，并以Full HD正面质量录制。

样品图片















































相机应用程序本身并未发生任何重大变化。它与公司其他设备的区别仅在于存在各种拍摄模式。轻拍曝光调整可方便地手动调整焦点周围的圆周运动。另一个优点是大量可用设置，并且可以快速将帧保存在设备的内存中。

干主观残渣



全新的小米Mi5经过一天的生活可以得出什么结论？我喜欢智能手机吗？是的，我喜欢它。可以认为本文完全客观吗？不确定，意见不一。我注意到，在过去的一年中，我不想与智能手机见面，这是我第一次放弃。我还要说的是，有一种希望以主机的形式保留它，以代替个人HTC One M8。

当然，智能手机有缺陷，也有一些缺陷，但是经过长时间的测试，它们还是可以被发现的，或者被搜索所迷惑。缺乏俄语和全面的Google服务是否应被视为不利？是的，但可以“笔”交付，一段时间后，将发布本地工匠提供的固件的本地化版本。一个想要购买智能手机，开箱即用并使用它的人，几乎不会考虑购买这种选择。但是，对于有信心的用户来说，这肯定会更容易，因为设置设备对他来说不是问题。

小米Mi5是否标志着技术圈的明显进步？他会展示任何新技术吗？否定可能性更大。我们已经或将很快从其他许多制造商那里看到Mi5中的所有内容。但是，正是由于产品的整体复杂性和细节的周到性，才产生了整体而积极的意见。但是，就像小米的许多其他设备一样。但是，对于完美智能手机的搜索并不仅限于此。我们很可能会找到他，而这样的智能手机将会表现出来。同时，Mi5在申请人中处于领先地位。
