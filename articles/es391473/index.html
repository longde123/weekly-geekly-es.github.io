Microsoft ha lanzado un producto abierto basado en GNU / Linux


Microsoft realmente lo hizo. Ayer, en la conferencia Open Compute Project (OCP) en San José, se presentó un producto completamente abierto, llamado SONiC o Software para Open Networking in the Cloud . El producto fue creado con la participación de fabricantes como Arista, Broadcom, Dell y Mellanox. SONiC es un conjunto de software y utilidades diseñado para controlar dispositivos de red de niveles L2 / L3 de diferentes fabricantes. Actualmente, la base es la distribución de Debian con el kernel Linux 3.16, pero como Microsoft afirma, se puede usar cualquier otra distribución con una versión de kernel que comience con 3.16.

SONiC puede funcionar en varios dispositivos gracias a la especificación SAI (Switch Abstraction Interface) adoptada en el verano de 2015. Proyecto OCP. El iniciador del proyecto SAI también fue el propio Microsoft. SAI es una API C estandarizada para programar circuitos ASIC para equipos de red. El proyecto SAI también es de código abierto y está disponible en Github .

SONiC es la base de otro producto basado en Microsoft GNU / Linux: Azure Cloud Switch (ACS) . ACS fue desarrollado por la compañía para uso interno y contiene un conjunto adicional de software de administración en la nube de Azure.

Microsoft afirma que SONiC v1se usan en producción y ya está presente en los conmutadores existentes de los centros de datos de Microsoft. También prometen publicar en Github Sonic v2 en las próximas semanas , que contendrá muchas mejoras.
