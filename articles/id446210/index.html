ADAM-3600 - pengontrol industri multifungsi



Pengontrol ADAM-3600 adalah perangkat universal untuk kendali jarak jauh fasilitas industri, pengumpulan data, pemantauan, dan otomatisasi proses.
Pada saat yang sama, ia dapat bertindak sebagai perangkat komunikasi dengan objek ( USO ), perangkat akuisisi dan transmisi data ( USPD ), dan pengontrol logika yang dapat diprogram ( PLC ).
Ini memiliki modul ekspansi untuk port I / O, serta modul untuk antarmuka nirkabel Wi-Fi, 4G / 3G / GPRS, ZigBee, GPS.

Perangkat ini berjalan pada sistem operasi RealTime Linux dan prosesor Cortex A8. Mendukung protokol industri terbuka ModBus, DNP3, MQTT, HTTP REST API. Itu dapat diintegrasikan ke dalam sistem SCADA . Ini juga mendukung integrasi ke sistem cloud IoT, seperti Microsoft Azure IoT Hub , Amazon AWS IoT , dan lainnya.

Pada artikel ini kita akan menganalisis fungsi utama perangkat dan berkenalan dengan skenario menggunakan pengontrol dalam pembangunan sistem kontrol industri .

Spesifikasi teknis


Prosesor: Cortex A8 600MHz
RAM: DDR3L 256MB
Host USB 2.0: 1
VGA: 1
Port Serial: 1x RS232 / 485, 2x RS485
Port Ethernet: 2
Input Digital: 8 (dapat diperluas hingga 40)
Output digital: 4 (dapat diperluas hingga 20)
Input Analog: 8 (dapat diperluas hingga 24)
Output Analog: 0 (meluas ke 16)
slot mPCIe: 1 x Ukuran Setengah, 1 x Ukuran Penuh
Slot ekspansi I / O: 4
Antarmuka nirkabel: Wi-Fi, 3G, 4G GPRS, ZigBee (opsional)
Jam Real Time Bertenaga Otomatis: Ya
Kartu SD: ya
Kisaran Suhu: -40 hingga 70 ¬∞ C
Catu Daya: 8 hingga 30V

Modul ekspansi port


Jumlah pin I / O dapat bertambah tergantung pada kebutuhan pelanggan. Hingga 4 modul ekspansi antarmuka dapat diinstal pada pengontrol ADAM-3600. Ada 6 jenis modul: input / output digital, input / output analog, input untuk termokopel (TC) dan sensor resistor (RTD).

Modul ekspansi antarmuka dipasang di panel depan

Antarmuka nirkabel


Berkat kehadiran port mPCIe, Zigbee, Wi-Fi, 3G / 4G / GPRS, antarmuka nirkabel GPS dapat dengan mudah ditambahkan ke controller. Dua modul mPCIe dapat diinstal pada saat yang sama: satu ukuran penuh dan satu ukuran setengah. Kedua modul Advantech bermerek didukung, serta yang diproduksi oleh pihak ketiga oleh Huawei, Quectel dan lainnya.


Tergantung pada jenis antena, konektor yang sesuai dipasang: SMA atau RP-SMA. Terminal antena terhubung ke papan melalui konektor U.FL standar.

Fungsi jaringan


ADAM-3600 memiliki dua port Ethernet independen yang dapat digunakan dalam mode switch, atau dapat dibagi menjadi WAN dan LAN. Ini nyaman untuk menghubungkan beberapa pengontrol secara seri dalam topologi rantai daisy, dan tidak memerlukan pemasangan sakelar tambahan pada saluran yang lebih panjang dari 100 meter.


Dua port LAN memungkinkan Anda menghubungkan perangkat secara seri

Mode Failover dan Multi WAN


Untuk fasilitas infrastruktur kritis yang terpencil, kehilangan komunikasi dapat menimbulkan kerugian yang signifikan. Oleh karena itu, untuk meningkatkan keandalan komunikasi, ADAM-3600 memiliki fungsi redundansi saluran dalam mode failover. Jika koneksi terputus pada saluran utama, controller akan secara otomatis beralih ke cadangan, misalnya, dari Internet kabel ke 3G. Prioritas saluran dan alamat untuk memeriksa koneksi dikonfigurasikan di EdgeLink Studio.


Mengatur Prioritas Saluran dan Mode Gagal

Dukungan VPN


Dukungan untuk versi OpenVPN 2.4 saat ini memungkinkan Anda untuk mengintegrasikan perangkat ke dalam jaringan virtual perusahaan. Terowongan UDP dan TCP, otentikasi sertifikat, dan algoritma enkripsi canggih seperti AES-256-CBC tersedia.


Konfigurasikan OpenVPN Tunnel di EdgeLink Studio

ID Perangkat Keras Perangkat


Konfigurasi awal yang cepat dapat diatur menggunakan sakelar DIP pada panel depan. NodeID menetapkan pengidentifikasi perangkat dan memulai alamat IP. Ini memungkinkan Anda untuk melakukan pemasangan sejumlah besar perangkat tanpa konfigurasi awal, dan konfigurasi lebih lanjut dilakukan dari jarak jauh melalui jaringan.


Device ID dapat diatur menggunakan jumper

Mode operasi


Fitur utama dari pengontrol ADAM-3600 adalah keserbagunaannya. Ini dapat berupa perangkat akuisisi data (merekam dan mentransmisikan telemetri), dan perangkat komunikasi dengan objek (mentransmisikan perintah kontrol dari pusat kontrol), dan pengontrol yang dapat diprogram (kontrol otomatisasi).
Setiap use case memiliki lingkungan pengembangannya sendiri.

Unit Akuisisi Data dan Terminal Jarak Jauh


ADAM-3600 dapat beroperasi sebagai perangkat akuisisi dan transmisi data (DTD) klasik. Salah satu fitur tambahan adalah pemrosesan data sebelum mengirim. Sebagai contoh, standar deviasi atau pembacaan sensor arus mentah atau resistansi dapat dihitung, dan data jadi dikonversi ke format yang sesuai untuk berbagai database, misalnya, MS SQL Server. Menyimpan data dapat dilakukan pada kartu-SD, dalam kasus ketika koneksi internet tidak tersedia. Setelah koneksi dipulihkan, data akan ditransfer ke server tanpa kehilangan.



Menggunakan protokol MQTT, Anda dapat mengatur pengumpulan data langsung ke layanan cloud seperti Microsoft Azure IoT Hub, Amazon AWS IoT, dll.

Perangkat ini dikonfigurasi dalam lingkungan grafis EdgeLink Studio, yang memungkinkan Anda untuk mengonfigurasikan perangkat dalam mode pengumpulan data atau di terminal jarak jauh tanpa menulis kode apa pun. Ini memungkinkan Anda untuk menyederhanakan implementasi secara signifikan.

Penyedia Cloud yang Terjangkau di EdgeLink Studio

Pengontrol logika yang dapat diprogram


Firmware ADAM-3600 didasarkan pada RT Linux - versi khusus dari sistem operasi real-time yang keras . Selain itu, platform benar-benar terbuka, pengembang memiliki akses root ke sistem. Ini memungkinkan Anda untuk mengimplementasikan solusi non-standar menggunakan bahasa pemrograman sistem. Advantech menyediakan SDK untuk bahasa C / C ++ , serta lingkungan pengembangan dan Toolchain untuk membangun proyek untuk Linux.

Skenario penggunaan: pemantauan real-time dari sumur minyak


Di situs web kami, di bagian materi pelatihan, kami menerbitkan skenario penggunaan produk www.untunganech.ru/resources/ . Salah satu skenario umum untuk menggunakan pengontrol ADAM-3600 adalah pemantauan dan kontrol stasiun pengeboran dan pompa. Advantech menawarkan berbagai solusi untuk membangun sistem kontrol otomatis, khususnya, sistem SCADA dengan antarmuka web - Advantech WebAccess.

Tugas dasar sistem:

  • Pemantauan suhu oli dan "selongsong", tekanan oli, laju aliran, dan parameter lain dari unit pompa; serta membaca dan menyimpan ke database.
  • Menghubungkan sensor melalui kabel (RS-232/485, LAN) dan bus nirkabel (ZigBee)
  • Menghubungkan sensor dan relay langsung ke antarmuka pengontrol digital dan analog
  • Kumpulan pembacaan sensor peralatan periferal bersama dengan statistik tentang produksi minyak dan operasi peralatan. Setelah pemrosesan awal, data dikirim ke server SCADA.



Arsitektur sistem pemantauan sumur minyak

Kesimpulan


Berkat desain modularnya, pengontrol ADAM-3600 memungkinkan Anda memilih port antarmuka dan protokol nirkabel berdasarkan kebutuhan pelanggan, sehingga Anda dapat melakukannya dengan satu perangkat untuk sejumlah besar tugas. Fitur pengaturan cepat disediakan, yang membuatnya mudah untuk menginstal dan menginventarisir sejumlah besar perangkat. Kisaran suhu pengoperasian dari -40 hingga 70 ¬∞ C memungkinkan untuk menggunakan pengontrol dalam kondisi yang keras. Dukungan untuk standar industri dan protokol IoT modern memungkinkan ADAM-3600 untuk diintegrasikan ke dalam sistem industri apa pun.
