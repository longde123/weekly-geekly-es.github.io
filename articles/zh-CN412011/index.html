Google I / O 2018上显示的内容

Sundar Pichai昨天于莫斯科时间晚上8点来到山景城的海岸线露天剧场，并开放了Google I / O2018。会议将持续5天（从5月8日至10日），为期三天。 Android P在iPhone X上的iOS 11中学习了手势，Google Assistant现在以六种不同的声音讲话，并且几乎通过了Turing测试，并且将向开发人员提供ML Kit，以在其应用程序中实现机器学习。



Google助手


助手是该程序的亮点。 自宣布以来的两年中，他决定使用5亿种不同的设备。 除了智能手机，这些还包括智能扬声器，耳机，电视和手表。 它已集成到40个不同制造商的汽车中，并支持与5000多种家用电器一起使用，例如洗碗机和电话。

现在，他被教导用六种不同的声音说话，并且一次就能理解几个命令。 到今年年底，Google助手将在80个国家/地区工作，并支持30种语言，包括俄语。 是的，您不再需要不断说“ OK Google”。 也就是说，您可以说：“好吧Google，早上7点设置闹钟，并提醒我在超市附近时买酸奶。”


以前，例如，从助理那里，我们主要期望语音响应，例如智能扬声器。 但是在此I / O上，Google展示了智能显示器类别中的多种设备-这本质上是一个智能列，仅带有触摸屏。 这样的屏幕列可以说“ OK Google，打开Jimmy Kimmel”，她将在YouTube上找到他的节目并启动它。 根据Google的说法，在同一设备上搜索食谱非常方便。 所谓的“智能显示器”与传统平板电脑在根本上的区别还不是很清楚。



在美国，还指导Google Assistant在取货和送货计划下工作，并与星巴克，7-11，Dunkin'Donuts等流行网络绑定。 现在，在美国，您可以与助手交谈，并要求订购一杯带有焦糖的拿铁咖啡，然后在一家方便的咖啡馆将其拿起。



您告诉助理的所有内容，他都会记住，并且会逐渐开始提供提示，并像Google即时一样准确地记录下一天的情况。 所有这些芯片将于2018年夏季出现在新版本的Android中。

Android P


Android P测试版将于5月8日发布，并将在11款智能手机上提供：

  • 索尼Xperia XZ2
  • 小米Mi Mix 2S
  • 诺基亚7 Plus
  • OPPO R15 Pro
  • 体内x21
  • 一加6
  • 基本PH-1
  • Google Pixel / XL
  • Google Pixel 2/2 XL

新版操作系统已修订了导航和手势方法。 特别是，他们以与iPhone X相同的方式对打开的应用程序列表进行了调用-从下至上滑动。 后台的应用程序调用按钮已被完全放弃，只有在打开应用程序时才会显示“后退”按钮。 此外，一些界面元素的设计将根据Material Design 2进行更新。

Pichai分别谈到了几个新功能。 例如，“ Pashboard”出现在Android P中，其中显示了智能手机的使用情况统计信息：您一天解锁了多少次，阅读了多少条通知。



在Android版YouTube应用程序的设置中，您可以选择一种模式，在这种模式下，如果您长时间连续观看视频，会提示您暂停。 此外，您可以从每日订阅中（而不是单独）接收来自您订阅的频道的新视频的通知。

家庭链接-儿童设备的控制面板，有助于限制在屏幕前所花费的时间。

谷歌新闻


在Google I / O上，Sundar Pichai谈到了Google新闻的重生。 现在，“新闻”类似于Yandex.Zen-它是一个智能的新闻聚合器，它知道您通常阅读的内容以及所信任的资源。 它过滤掉不必要的内容，并在最近几个小时内在“为您”选项卡上显示五个重要的故事。



机器学习


通过机器学习，新的Android设备可以通过预测下一个将启动的应用程序来显着节省电池电量，概率约为60％。

但是最主要的-现在有了机器学习套件，所有开发人员现在都可以使用API​​来访问智能手机功能，例如识别文本中的文字和面部，或者例如扫描条形码。 此外，他们从舞台上说开发人员将可以使用Google的云神经网络，但他们没有提供详细信息。
