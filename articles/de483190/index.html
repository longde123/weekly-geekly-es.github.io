Antworten des technischen Supports von 3CX: Führen Sie ein Upgrade von früheren Versionen auf 3CX v16 durch

Im neuen Jahr mit einer neuen Telefonzentrale! Es stimmt, es gibt nicht immer Zeit oder den Wunsch, die Feinheiten des Übergangs zwischen den Versionen zu verstehen und Informationen aus verschiedenen Quellen zu sammeln. In diesem Artikel haben wir alle Informationen zusammengestellt, die für ein einfaches und schnelles Upgrade auf 3CX v16 Update 4 von älteren Versionen erforderlich sind.

Es gibt viele Gründe für ein Update - Sie können alle Funktionen, die in Version 16 erschienen sind, aus dem Schulungskurs entnehmen. Hier stellen wir die wichtigsten Verbesserungen fest, die sich für normale Benutzer bemerkbar machen - neue mobile Anwendungen , ein Kommunikations-Widget für die Site und VoIP-Softphone im Browser .

Vor dem Upgrade Lizenz prüfen


Beachten Sie zunächst, dass für die Umstellung auf die neueste Version von 3CX eine Lizenz mit einem Jahresabonnement oder eine unbefristete Lizenz mit einem aktiven Abonnement für Updates erforderlich ist. Ohne ein aktives Abonnement für Updates wird Ihr neues System einfach nicht aktiviert. Sie können den aktuellen Status des Abonnements im Abschnitt Einstellungen> Lizenz überprüfen. Sie können auch im 3CX-Benutzerportal nach Updates suchen.

Bitte beachten Sie, dass die korrekten Anmeldeinformationen in der PBX-Benutzeroberfläche erst ab Version 15.5 Update 6 und höher verfügbar sind.



Wenn das Abonnement abgelaufen ist


Wenn Sie eine unbefristete Lizenz haben, müssen Sie herausfinden, ob Sie sich in einem Kulanzzeitraum befinden, in dem Sie Ihr Abonnement verlängern können. Wenden Sie sich hierzu an den 3CX-Partner, der Ihnen dient (oder an den ausgewählten Partner in Ihrer Nähe ), oder wenden Sie sich direkt an die Benutzer-Support-Abteilung . Übrigens können Sie Ihr Abonnement für Updates jederzeit erneuern, und zwar nicht nur, wenn es bereits abgelaufen ist. Darüber hinaus erhalten Sie 10% Rabatt beim Kauf von Updates für 3 Jahre und 15% beim Kauf für 5 Jahre (es handelt sich um das Abonnieren von Updates für unbefristete Lizenzen).

Wenn sich herausstellt, dass die Nachfrist bereits abgelaufen ist, können Sie Ihre unbefristete Lizenz gegen eine Lizenz mit einem kostenlosen Jahresabonnement eintauschen. Danach erhalten Sie ab dem Zeitpunkt des Umtauschs ein Jahr lang die kostenlose Nutzung der konvertierten Lizenz. Nach einem Jahr erwerben Sie einfach eine Jahreslizenz für das nächste Jahr.
Der Tausch erfolgt in Ihrem Benutzerportal im Bereich Kaufen> Inzahlungnahme (Kaufen> Tausch .



Bitte beachten Sie, dass Sie während des Austauschs keinen neuen Schlüssel erhalten, sondern nur Ihren bestehenden Schlüssel jährlich. Das System muss nicht verändert werden! Die einzige Aktion: Nachdem Sie die E-Mail mit der Bestätigung des Austauschs erhalten haben, rufen Sie die 3CX-Benutzeroberfläche auf und klicken Sie im Abschnitt Einstellungen> Lizenz auf die Schaltfläche Lizenzinformationen aktualisieren (dies funktioniert jedoch nur in 3CX v15.5 und höher). Wenn Sie eine ältere Version installiert haben, siehe unten.

Update von v15.X auf v15.5 SP6


Vor dem Upgrade auf Version 16 müssen Sie Ihr System der Version 15.X (und höher) auf Version 15.5 SP6 aktualisieren. Nur in diesem Fall ist die korrekte Übertragung der PBX-Konfiguration aus dem Backup gewährleistet. Die einfachste Möglichkeit zum Upgrade besteht darin, diese Anleitung zu befolgen . Wenn Sie jedoch eine noch ältere Version von 3CX haben, müssen Sie den gesamten Pfad der Updates nacheinander durchgehen und diese nacheinander installieren.

Stellen Sie sicher, dass Sie in jeder Phase des Updates Sicherungskopien erstellen!

Update von v15.5 SP6 auf v16.X


Hierbei ist zu beachten, dass sich der 3CX-Aktualisierungsprozess für Windows und Linux aufgrund der Architekturmerkmale dieser Betriebssysteme geringfügig unterscheidet.

Windows


Leider kann 3CX v15.5 SP6 nicht "direkt" auf v16 aktualisiert werden, da dies unter Linux möglich ist. Sie müssen die TK-Anlage sichern und während der Installation des v16-Verteilungskits wiederherstellen.

Wechseln Sie in der 3CX-Benutzeroberfläche zum Abschnitt Sichern, klicken Sie auf + Sichern, geben Sie einen Sicherungsnamen und Optionen an.



Warten Sie auf die Benachrichtigung des PBX-Administrators per E-Mail, dass die Sicherung abgeschlossen ist, und laden Sie dann die Sicherungsdatei herunter.



Bitte beachten Sie - Sie können das erstellte Backup bei der Installation der TK-Anlage unter Windows und Linux verwenden - die Backup-Datei wird für beide Betriebssysteme problemlos verwendet!
Deinstallieren Sie nach dem Backup 3CX, laden Sie 3CX v16 herunter und starten Sie die Installation. Geben Sie auf dem ersten Bildschirm des Konfigurationsassistenten die Sicherungsdatei an und setzen Sie die Installation gemäß den Anweisungen fort .


Linux


3CX-Update "an Ort und Stelle", d. H. Eine sofortige Aktualisierung der vorhandenen Installation ist nur verfügbar, wenn die TK-Anlage unter Debian 9 Stretch installiert ist (Debian 8 und 10 werden in v16 nicht unterstützt). Wenn die Verfügbarkeit von Updates in der 3CX-Benutzeroberfläche nicht angezeigt wird, überprüfen Sie die Linux-Version im SSH-Terminal (Befehl sudo lsb_release -a ).

Debian 9


Das Update hier zu installieren ist sehr einfach. Wechseln Sie in der 3CX-Benutzeroberfläche zum Abschnitt Updates und installieren Sie alle verfügbaren Updates. Warten Sie auf die E-Mail mit dem Abschluss des Updates. Kehren Sie danach zu Updates zurück und installieren Sie alle verfügbaren Updates erneut - und so weiter. bis es keine Benachrichtigungen mehr gibt.

Debian 8


3CX v16 ist nicht kompatibel mit Debian 8, auf dem v15.X lief. Daher müssen Sie die Konfiguration sichern und die neue Installation vom ISO-Image von Debian für 3CX bereitstellen .

Beachten Sie, dass Sie mit Ihrer Sicherungskopie und dem Cloud-Installationsassistenten von 3CX PBX Express von einer lokalen Installation zu einer Cloud-PBX migrieren können.
