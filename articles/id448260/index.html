Konfigurasikan Visual Studio di organisasi Anda dengan .vsconfig

Dalam Visual Studio 2017 Pembaruan 15.9, kami menambahkan kemampuan untuk mengekspor dan mengimpor beban kerja dan memilih komponen dalam file konfigurasi instalasi Visual Studio. Pengembang dapat mengimpor file-file ini ke instalasi baru atau yang sudah ada. Memeriksa file-file ini di repositori sumber Anda membuatnya mudah untuk dibagikan. Namun, pengembang masih perlu mengimpornya untuk mendapatkan fitur yang diperlukan.

Dalam Visual Studio 2019, instalasi otomatis komponen yang hilang telah ditambahkan. Detail di bawah potongan!

Pemasangan otomatis komponen yang hilang


Baru di Visual Studio 2019 : Anda dapat menyimpan file-file ini sebagai file .vsconfig di direktori root dari solusi, dan ketika solusi (atau direktori solusi) dibuka, Visual Studio akan secara otomatis mendeteksi komponen yang hilang dan meminta Anda untuk menginstalnya.



Anda dapat menemukan contohnya di repositori vswhere di GitHub. Ketika Anda mengklik Instal , Anda diminta untuk menginstal semua komponen yang hilang. Anda dapat mengklik Lihat detail instalasi lengkap jika Anda ingin memilih komponen tambahan.





Ekspor konfigurasi Anda


Di Visual Studio 2019, Anda bisa membuat file .vsconfig langsung dari Solution Explorer:

  1. Klik kanan pada solusi Anda.
  2. Klik Tambah> File Konfigurasi Instalasi .
  3. Konfirmasikan tempat Anda ingin menyimpan file .vsconfig (secara default, ini adalah direktori root dari solusi Anda).
  4. Klik Tinjau detail .
  5. Pilih atau buang perubahan yang ingin Anda lakukan dan klik Ekspor

Untuk meminimalkan biaya pemasangan, ekspor hanya komponen-komponen yang diperlukan untuk membangun, menguji, dan menerbitkan solusinya. Salah satu cara untuk melakukan ini adalah menginstal Visual Studio contoh kedua atau menginstalnya pada mesin virtual, menambahkan beban kerja dan komponen tambahan yang diperlukan untuk membangun dan menguji solusi. Tambahkan komponen sesuai kebutuhan hingga solusi berhasil dibangun, lalu ekspor konfigurasi Anda.

Umpan balik
