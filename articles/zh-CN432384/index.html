研究：22％的Tumblr用户访问了色情网站，尤其是年轻女孩


Tumblr（上）和Flickr（下）上色情内容的不同生产者和消费者群体的年龄分布。 显示了平均年龄，中位年龄，标准差和未成年人使用比例（18岁以下）。 资料来源：发表在《第十届国际AAAI网络和社交媒体会议论文集》上的科学文章“社交媒体上的色情消费”。 ICWSM 2016， arXiv：1612.08157v2

Tumblr社交网络的许多用户已经知道，自2018年12月17日以来，该网站已禁止在其平台上发布色情内容 。 该网站的所有者认为，这样他们将获得更好的内容：请参阅博客文章“更好，更积极的图姆勒” 。 但是一些专家认为，通过这样的行为，Tumblr实际上削减了它的分支机构-这可能是传统上为草莓爱好者提供服务并在很大程度上被色情识别的社交网络结束的开始。

tumblr上有多少色情内容？ 去年，来自卢卡（IMCA）的IMT职业学校，意大利比萨的信息技术研究所和贝尔实验室的一组研究人员试图做到这一点。 他们的科学文章于2017年1月发表 。 事实证明，大约22％的Tumlr用户浏览了该网站上的大部分色情内容 。 其中，年轻女孩的比例特别高。

这项研究基于1.3亿用户的行为，约占整个Tumblr用户基础的一半。 该研究的作者之一，现在是诺基亚贝尔实验室的资深研究员卢卡·艾洛（ Luca Aiello） 表示： “大约有3000万个活跃帐户正在积极消费成人内容，或者再次共享它，或者被折叠成色情内容生产者的帐户。” “我希望（在禁止色情内容之后），这些观众的参与度会显着下降：有些人只会离开，而许多人可能会大大减少在该网站上花费的时间。”

除了22％的活跃消费者外，另有28％（约4000万用户）在Tumblr上偶然遇到了色情内容。 这意味着他们不是在寻找色情内容，而是关注发布色情内容的人。 这些“随机观看者”之一可能抱怨他所看到的。 也许大量投诉是Tumblr做出此决定的原因之一。

该研究表明，Tumblr用户中的女性人数超过了许多社交网络。 根据科学家的估计，用户的平均年龄为26岁，其中女性占72％。 此外，最活跃的色情消费者群体是20-25岁的年轻女性。 例如，女性杂志《魅力》 将其读者图姆勒（Tumlr） 推荐为“在互联网上找到不错的色情内容的安全场所”。

科学文章“社交媒体上的色情消费”于2017年1月20日在预印本网站arXiv.org（arXiv：1612.08157v2）上发表，然后在《 第十届国际AAAI网络和社交媒体国际会议论文集 》上发表。 ICWSM 2016。

Reddit发烧友拼命拯救Tumblr色情


在Tumblr悲惨地宣布之后，互联网反应出了可预见的愤怒， The Next Web 写道 。

在著名的Reddit网站上，组织了特别小组来保存平台中的色情内容。 从昨天开始，他们开展了一项充满活力的活动。 该活动由用户u / itdnhr发起：他一手保存了约67,000个Tumblr色情帐户以及所有内容-并组成了一个庞大的清单。 然后，他在r / Datasets社区中共享了收集的数据。 在这里，志趣相投的人帮助过滤掉了不活跃的帐户，使43,000个帐户获得了真正的最新信息。

保存来自Tumblr色情博客的内容并非易事，更不用说潜在的违法行为了。 现在，色情档案库中已积累了约25 TB的数据。 最简单的解决方案是将其作为种子下载，但是要下载这种大小的文件并不容易。 目前尚不清楚将其保存到哪种媒体：大多数用户没有足够格式的存储。 或者，建议将文件拆分为多个部分，或通过专用服务器上的单个链接或一系列链接分发文件。 但是，即使将色情档案分成25个部分，下载和保存1 TB的文件也不是每个人的乐趣。 此外，在世界上许多国家，档案中可能会包含禁止使用的材料，以供存储和分发，而这种行为有刑事责任。

但是，尽管法律风险并没有消除最大Reddit内容归档社区中的热情。 用户仍在顽固地寻找有关如何最好地保存濒临灭绝的Tumblr独特内容的解决方案。
