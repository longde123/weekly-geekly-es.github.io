Longread ini akan menjadi teman dan cukup jujur, tetapi untuk beberapa alasan saya belum melihat artikel seperti itu. Ada banyak orang yang berpengalaman dalam hal toko online (pengembangan dan promosi), tetapi tidak ada yang menulis bagaimana membuat toko keren dengan nominal $ 250 (atau mungkin $ 70) yang akan terlihat hebat dan bekerja hebat (jual!). Dan Anda dapat melakukan semua ini sendiri tanpa seorang programmer. Yah, umumnya baik untuk memiliki programmer di sebelah Anda yang akan bernapas lembut di belakang kepala Anda dan memperbaiki tangan Anda yang canggung, tetapi karena Saya sendiri, bukan menjadi programmer, membuat toko online, saya tahu apa yang saya bicarakan. Sekali lagi - programmer tidak pernah membantu saya untuk toko ini.

Jadi, ayo pergi. Toko ini - kami secara cerdik menjual basis spam. Ya Untuk spam Saya akan menulis artikel terpisah tentang bagaimana kami melakukan basis data ini ... kami sendiri tidak melakukan spam, tetapi menjual basis data (ngomong-ngomong, jika seseorang berpikir bahwa cukup untuk menguraikan 2GIS dan saya akan kecewa dengan topi, semuanya jauh lebih rumit dan bekerja untuk kita 3 programmer dalam arah ini fulltime by the way). Artikel sedang dikerjakan dan akan menyenangkan :). Kami juga akan membagikan hasil spam dari pelanggan kami di sana - ngomong-ngomong, semua spam yang paling tidak disukai berhasil.

Apa yang membuat saya menulis artikel ini? Hari ini, sebuah publikasi diterbitkan di VC, di mana orang-orang membandingkan Bitrix dan beberapa lingkungan pengembangan lainnya. Orang-orang menulis di komentar mengatakan mereka lupa WordPress? Jawabannya menarik - well, ada banyak lubang di sana seperti pada keju Swiss. Dan mereka membawa tautan ke laporan ... (di sana, temukan sendiri di artikel). Dan ya, saya melihat laporan ini dari produsen sebuah plugin yang bagus untuk perlindungan Wordpress :) mereka terkenal takut untuk membeli plugin mereka. Tetapi jika Anda bekerja sedikit (sedikit, tanpa berkeringat) (versi terbaru, pembaruan + plug-in gratis untuk perlindungan seperti Wordfence) maka kemungkinan peretasan harus diminimalkan. Faktanya adalah bahwa 80% situs Wordpress dibuat berlutut - ada jutaan di antaranya dan, tentu saja, tingkat peretasannya sangat besar dibandingkan dengan magenta, yang sudah dilaksanakan oleh spesialis yang lebih berpengalaman.

Tugas kami adalah membuat pangkalan penjualan toko online. Ngomong-ngomong, ini sedikit lebih rumit daripada menjual barang fisik (walaupun mungkin ada nuansa tertentu karena kalkulator pengiriman, saya akui saya tidak mengambilnya di sini). Mengapa Basis perusahaan kami disimpan di Amazon S3 (mengapa - saya akan menulis nanti) dan harus mengotak-atik banyak. Jika Anda memiliki biaya pengiriman yang sama untuk semua wilayah dan barang (yah, atau logikanya sangat sederhana) - maka bahkan akan lebih mudah bagi Anda untuk memulai semuanya di luar kotak.

Baiklah, mari kita ambil langkah-langkahnya, akan lebih mudah bagi semua orang untuk memahami cara membuat toko Internet yang bagus seharga $ 200 (dan saya akan memberi Anda contoh bagaimana melakukan hal yang sama dengan $ 50 tanpa kehilangan kualitas).

Hosting


Hosting - kami memiliki hostland.ru. Tautan ini bukan rujukan, saya tidak kenal siapa pun di sana. Ya, hanya orang normal, melakukan pekerjaan normal. Ah ... jika Anda mendengar saya di sana - jatuhkan beberapa permen pada saldo kami atau sesuatu - iklannya adalah asli :) Hosting Wordpress harganya sekitar 300 rubel ditambah 50 rubel, saya bahkan tidak ingin memeriksa ulang). Akun pribadi Anda sederhana, semuanya jelas, pertama kali dalam beberapa jam Anda akan membuat WordPress khas.

Nama domain


Dan teman-teman saya lupa - nama domain! :) Ini adalah langkah # 1 - well, kami membeli di nic.ru (saya biasanya merekomendasikan orang-orang ini), meskipun sekarang hoster kami memberi kami bonus domain gratis dan kami mengambil domain baru dari mereka. Bisakah saya tidak mempertimbangkan biaya untuk membeli domain di zona RU? Mungkin biayanya sama untuk sampai ke sana-sini di metro Moskow, hanya ada pembayaran di sana selama setahun :) Langkah 2 - Anda perlu mengirim nama domain ke situs web Wordpress yang Anda buat dengan hoster. Nah untuk keperluan ini kami menggunakan Yandex.Connect. Jujur, Anda perlu mengotak-atik, tetapi jika Anda memiliki kepala di bahu Anda, maka Anda dapat melakukan banyak hal berikut:

nic.ru -> Yandex.Connect -> DNS editor dan ada untuk mendaftarkan alamat IP -> host.

Tetapi jika Anda terlalu malas untuk melakukan ini, maka hoster memberikan domain gratis (kadang-kadang, saya masih tidak mengerti seluruh logika bonus) di zona .RU dan itu menentukan segalanya, atau dukungan akan membantu.

Nah, apa yang kita miliki seimbang? 300 rubel untuk hosting (Saya akan melanjutkan dalam dolar, lebih mudah - sekitar $ 5) + 0 rubel (Saya tidak ingin menghitung) untuk pendaftaran nama domain. Nah, sementara bisnis menarik biaya ini :) kita berderit, tetapi tarik.

Dan ini adalah plugin yang memungkinkan Anda membuat toko online dari awal. Dia bebas.

Setelah instalasi, Anda sudah dapat memulai di woocommerce untuk mengedit produk Anda. Hapus produk demo, tambahkan produk Anda. Tidak ada yang bisa ditulis sama sekali, ada sejuta panduan cara untuk melakukan ini, semuanya sederhana dan jelas.
