Memperkenalkan font Kode Cascadia

Kode Cascadia akhirnya tiba! Anda dapat menginstalnya langsung dari halaman rilis repositori GitHub atau secara otomatis menerimanya di pembaruan Windows Terminal berikutnya.



Tunggu, apa itu Cascadia Code?


Cascadia Code diumumkan pada Mei lalu di acara Microsoft's Build. Ini adalah font monospace terbaru yang dikirim dari Microsoft dan memberikan pengalaman baru untuk pengalaman baris perintah dan editor kode. Cascadia Code dikembangkan bersama dengan aplikasi Windows Terminal yang baru. Font ini paling direkomendasikan untuk digunakan dengan aplikasi terminal dan editor teks seperti Visual Studio dan Visual Studio Code.

Artikel ini di blog kami .

Pemrograman ligatur


Cascadia Code mendukung ligatur pemrograman! Pemrograman ligatur paling berguna saat menulis kode, karena mereka membuat mesin terbang baru dengan menggabungkan karakter. Ini membantu membuat kode lebih mudah dibaca dan ramah bagi sebagian orang.



Catatan: Jika Anda menggunakan Visual Studio Code, Anda harus mengaktifkan ligatur font di pengaturan Anda untuk melihatnya.



Apakah kami menyebutkan itu open source?


Ya, Anda dengar benar! Cascadia Code dilisensikan di bawah lisensi SIL Open Font di GitHub , jadi silakan berkontribusi seperti yang Anda inginkan!

Catatan: Saat ini tidak ada kerangka pengujian yang tepat dibangun ke dalam repositori, sehingga semua permintaan tarik harus diuji secara manual dan divalidasi. Ini akan memperlambat persetujuan permintaan tarik, jadi harap ikut kami.

Mengapa dinamai Cascadia Code?


Nama Cascadia Code berasal dari proyek Terminal Windows. Sebelum dirilis, nama kode untuk Windows Terminal adalah Cascadia. Bahkan, beberapa file sumber di dalam Terminal masih menggunakan nama ini! Sebagai penghormatan ke Terminal, kami menyukai gagasan memberi nama font setelah nama kode-nya.
Kami menambahkan Kode di akhir nama font untuk membantu menunjukkan bahwa font ini dimaksudkan untuk pemrograman. Secara khusus, ini membantu mengidentifikasi bahwa itu termasuk ligatur pemrograman.

Untuk memastikan Cascadia Code adalah pilihan yang tepat untuk nama font, kami mengadakan polling di Twitter bersama dengan nama lain yang kami pertimbangkan. Kami sangat berterima kasih atas partisipasi semua orang dan sangat senang Cascadia Code keluar sebagai pemenang.



Siapa yang Merancang Kode Cascadia?


Terima kasih banyak untuk Aaron Bell, perancang Cascadia Code. Kami cukup beruntung untuk bekerja dengannya selama setahun terakhir dan dia telah melakukan beberapa pekerjaan yang benar-benar hebat. Aaron Bell adalah desainer font dengan perusahaannya sendiri, Saja Typeworks , dan telah bekerja dengan Microsoft selama bertahun-tahun. Dia adalah desainer Selawik pada tahun 2015! Anda dapat mengikutinya di Twitter di @aaronbell dan dia juga akan aktif di repositori GitHub kami. Kami berencana untuk terus bekerja dengannya untuk membantu meningkatkan Cascadia Code dan menjadikannya font yang bagus untuk semua orang!

Apa Selanjutnya untuk Cascadia Code?


Sampai hari ini, Cascadia Code versi 1909.16 tersedia untuk umum di GitHub. Ini akan mengikuti sintaks versi Windows karena font menerima pembaruan dengan mesin terbang baru dan halus.

Tetap terhubung


Untuk setiap pembaruan yang datang ke Cascadia Code, jangan ragu untuk mengawasi repo atau ikuti Kayla ( @cinnamon_msft ) dan Rich ( @richturn_ms ) di Twitter. Kami sangat senang dapat melepaskan font kami ke alam bebas dan kami tidak sabar menunggu Anda untuk menggunakannya!
