Tinjauan Java Track Reports dari RigaDevDays Conference

Halo semuanya! Dari 29 Mei hingga 31 Mei, konferensi RigaDevDays diadakan di Riga. Itu adalah tahun peringatan 5 tahun konferensi. Lebih dari 600 peserta berkumpul di bioskop Forum Cinemas untuk mendengarkan laporan teknologi - C #, Java, MachineLearning, Microservices, SQL.


Konferensi diadakan di bioskop. Awalnya ini membuat saya skeptis, tetapi segera setelah saya tenggelam di ruang bioskop yang nyaman dengan pemandangan layar kode besar, saya menyadari bahwa bioskop adalah tempat yang sempurna untuk konferensi.



Video


Video dari semua laporan akan tersedia dalam sebulan, tetapi untuk saat ini Anda dapat melihat rekaman "mentah" dari Hall 3, di mana ada laporan tentang Java, dan dari Hall 11, di mana ada laporan tentang layanan-layanan mikro.


Pembaruan: Video sudah tersedia.

Sonic PI Livecoding - Sam Aaron


Video


Keynotes selalu terbagi secara mental menjadi beberapa jenis:


  • inspirasional (seperti itu)
  • pengamat (tentang bagaimana)
  • filosofis (bagaimana begitu)
  • berpikir (seperti itu)
  • luar biasa (wow)

Sam Aaron sangat menginspirasi. Sam menawarkan untuk mengajar pemrograman anak-anak melalui musik. Sendiri mengklaim bahwa operasi bit, rekursi, tidak menyenangkan seperti membuat bagian DJ menggunakan kode mereka sendiri. Pemrograman tidak hanya tentang menulis logika bisnis, tetapi juga cara mengekspresikan diri. Hidup akan berubah jika Anda menemukan tempat untuk menerapkan keterampilan Anda untuk membuat dunia menjadi tempat yang lebih baik. Keynote yang menginspirasi, setelah itu Anda ingin pergi atau bahkan lari ke kode. Tapi tidak secepat itu! Menjelang akhir pembicaraan, Sam menemukan remote DJ IDE-nya. Suara sinematik melakukan tugasnya - setiap nada Sonic-Pi terdengar seperti trailer untuk film superhero.



Hack hidup! Bagi mereka yang suka mengkodekan ke musik, saya sarankan saluran YouTube Sam
Anda dapat menggabungkan melihat live coding dan set DJ. Sam juga memiliki papan ketik DIY yang keren, yang berhasil saya ambil setelah dia pindah dari tempat kejadian.


Java.Next: Mengikuti Java - Sander Mak


Video


Sander adalah pakar modul di Jawa. Bahkan sebelum dirilis, ia sudah menyampaikan presentasi tentang modul dan bahkan menulis buku Java Modularity . Dalam ceramahnya, ia menulis aplikasi modular sederhana, dan kemudian berbicara tentang jshell dan var. Hack hidup! Jika Anda tertarik untuk mempelajari lebih lanjut tentang modul Java, maka di YouTube ada beberapa laporannya. Setelah meninjau semuanya, Anda praktis akan mengunjungi bengkel berbayarnya.


Spring Boot the Ripper - Bagian 1, 2 - Evgeny Borisov


Video Bagian 1 Bagian 2


Laporan ganda Evgeny Borisov, seperti biasa, mengumpulkan aula penuh. Dia biasa membaca laporan ini bersama dengan Kiril Tolkachev, jadi kali ini dia harus menulis dan bercanda kodenya sendiri. "Teman Lama" - Lebah BeanFactory, cheburashka, spiderman, karakter terkenal dari laporan sebelumnya kembali ke bisnis. Eugene dalam bentuk live coding dari awal hingga selesai menulis beberapa aplikasi di Spring Booth. Menambahkan profil khusus, "AC" kustom, starter kustom. Dia berbicara tentang bagian dalam Spring Booth. Sebagai contoh, Anda tahu bahwa "Halo dunia" pada booting musim semi memiliki sekitar empat ratus kacang dalam daftar kacang, dan ini adalah default! Bagi semua orang yang memiliki Spring Booth on prod, laporan pasti akan bermanfaat.


IoT didukung oleh Microprofile - Layanan Mikro dalam praktiknya - Rustam Mehmandarov, Mads Opheim


Video


Rustam dan Mads mengatakan bagaimana mereka membuat pencahayaan di kantor baru menggunakan layanan microser pada kerangka Microprofile. Bola lampu di kantor sekarang merespons cuaca dan waktu. Tentu saja, semua ini diluncurkan di docker dan docker-compose. Seluruh infrastruktur proyek sepenuhnya dibangun menggunakan spesifikasi Microprofile. Sebagai bonus, Rustam menunjukkan bagaimana Anda dapat mengubah satu server web ke server lain menggunakan satu perintah. Dan semuanya bekerja tanpa kesalahan, karena server web menerapkan satu spesifikasi Microprofile.


Supersonik, Jawa Subatomik dengan Quarkus - Burr Sutter


Video


Boer Sutter menulis daftar tugas yang harus dilakukan tepat pada ceramahnya menggunakan kerangka Quarkus. Semua ini dalam mode live coding. Dia menghentikan server sepenuhnya hanya beberapa kali, dan dengan cepat menambahkan entitas - pengendali, dependensi. Quarkus mengambil semuanya dan bekerja sangat cepat sehingga saya ingin menemukan tangkapan di suatu tempat - itu tidak bisa begitu sempurna. Dan di akhir laporan, Boer menyusun kode menjadi aplikasi asli yang dimulai dalam 0,025 detik. Setengah jam lagi dari laporan dan pengembang Java akan membawanya keluar dari kamar di tangan mereka. Dan sepertinya dia telah mematahkan tepuk tangan paling keras dari speaker.


Sam Aaron menerima tepuk tangan meriah sebelumnya ketika pemberitahuan muncul di laporannya "Komputer akan restart dalam 59 detik, 58 detik, 57 detik ...". Pada saat ini, Sam tidak berhenti menceritakan laporan itu, tetapi bergegas berlari karena mengisi ranselnya.


RSocket - Protokol lintas jaringan Reaktif baru - Oleh Dokuka


Video


Pada laporan Oleg, seluruh audiens bisa saja dipotong ke Pakman untuk memenangkan buku. Mereka juga bisa melempar permen untuk menjelaskan apa itu backpreassure. Oleg memperkenalkan protokol RSocket, yang mendukung backpreassure di luar kotak. Pada demo, ada beberapa implementasi server Pakman di SocksJS, Grpc, RSocket. Ternyata masuknya sejumlah besar pengguna bertahan hanya server RSocket.


Panduan Pengembang Sibuk untuk Rekayasa Politeknik - Ted Neward


Video


Ted berpendapat bahwa pengembang harus:


  • "Polyglot", tahu beberapa bahasa pemrograman
  • "Poly-repositori", tahu beberapa database
  • "Multi-platform", kenal beberapa platform
  • pelajari bahasa pemrograman tercanggih yang mungkin
  • tidak hanya belajar bahasa, tetapi membuat proyek kesayangan di atasnya
