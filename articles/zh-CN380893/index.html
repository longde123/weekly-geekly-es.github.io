黎明继续探索谷神星的白点

黎明星际探测器在矮行星谷神星的表面发现了越来越多的新细节，给研究人员带来了新的谜团。

自2015年6月以来，“黎明”探测器一直在所谓的谷神星研究轨道上。目前，他正在以400米的分辨率拍摄矮行星的表面。从获得的图像中，将编译具有浮雕高度数据的第一张详细地图。探测器研究的一些天体图像已经使科学家感到惊讶。


©NASA / JPL-Caltech / UCLA / MPS / DLR / IDA

在一张详细的图片中，您可以区分一个不寻常的小山，它看起来像没有锥坑的火山锥。这座山比周围的风景要轻一些。圆锥体从覆盖着环形山的表面急剧突出，而不是撞击环形山的中央丘陵。据推测，它可能是由于矮行星的地质活动而形成的。

自从2004年在模糊的哈勃影像中发现谷神星的亮点以来，天文学家一直在努力弄清它们的起源。

黎明提供了其中一个斑点的详细图像，在此基础上可以得出结论，这个亮点是由于另一个较小的天体的撞击而发生的。

轻物质既位于保存完好的且最有可能是最近形成的火山口内，又位于周围的喷出物质中。被击打的物体很可能刺穿了谷神星外壳的上部，并暴露出成分不同的材料。预计将使用位于黎明号上的VIR红外光谱仪获得更准确的信息。


©NASA / JPL-Caltech / UCLA / MPS / DLR / IDA

轻质物质属于堕落物体的可能性较小。它以极快的速度撞向谷神星，并在一瞬间被抑制。在这种情况下，能量突然转化为热量，身体蒸发并在谷神星上留下了一个爆炸性的环形山。通常，当小行星撞击时，几乎立即达到数万度的温度。

到今年年底，黎明号将进入400公里的轨道，它将探测直径约35米的地表物体。这将涉及第三个Dawn工具-中子和伽马射线探测器GRaND。

谷神星的表面经常受到宇宙射线的轰击。在这种情况下，二次辐射会在表面材料的上部微米中发生，而GRaND可以检测到。这将创建主要成岩化学元素分布表：氧气，镁，铝，硅，钛和铁。还可以确定长寿命放射性元素（例如40钾，th 232和铀238）的辐射。另外，入射在表面上的中子将检测出气态物质的存在，例如氢，碳和氮。

这些研究将最终阐明谷神星表面是否有冰。
