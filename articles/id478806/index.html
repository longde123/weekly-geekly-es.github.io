Secara otomatis menambahkan sumber JSON ke REFL.ME untuk menghasilkan pesan dan peringatan

Setelah menerbitkan artikel " Pesan dan pemberitahuan di Android melalui JSON " di sini dan masuk ke 5 berita teratas hari ini di Hacker News, banyak surat tanggapan datang. Berkat ini, fitur baru yang bermanfaat telah ditambahkan.

Salah satu fitur baru yang ditambahkan atas permintaan pengguna adalah kemampuan untuk menambah dan menghapus sumber data secara terpusat untuk menghasilkan pesan dan pemberitahuan pada perangkat pengguna.

Saya memberikan contoh sederhana dalam format JSON yang memungkinkan Anda menggunakan fitur ini: 

{ "refl.me":true, "hub":true, "sources":[{"url":"https:\/\/refl.me\/\/test"},{"url":"https:\/\/refl.me\/\/test_secret"}] }

"Refl.me" - menunjukkan benar tentu.
"Hub" - menunjukkan tentu benar.
"Sumber" - daftar sumber untuk diproses. Penciptaan, pembaruan, dan penghapusan akan dilakukan secara otomatis sesuai dengan daftar ini.

Semuanya bekerja mirip dengan sumber reguler, tetapi diproses bukan oleh pesan, tetapi oleh sumber. Menurut alamat yang diterima dari "sumber", aplikasi membentuk daftar sumber data aktual untuk menghasilkan pesan dan pemberitahuan.

Dokumentasi lengkap dan semua fitur dapat dilihat di sini .
