亚历山大·别洛克里洛夫（Alexander Belokrylov）和德米特里·楚科（Dmitry Chuyko）关于julia.msk.ru上的Liberica JDK

在2019年2月14日， 今年莫斯科Java开发人员的jug.msk.ru社区的第一次会议上，来自BellSoft的 Alexander Belokrylov和Dmitry Chuiko谈到了Liberica JDK分发套件的开发功能。



关于演讲者


两位发言人均为BellSoft的员工（亚历山大也是公司的首席执行官 ）。 BellSoft由Java工程师于2017年在圣彼得堡的Oracle开发中心成立。 旗舰产品是Liberica JDK ，它是基于Open JDK的发行版 ，其独特功能是支持ARM体系结构 。 除了更详细地查看有关公司发展的报告之外，您还可以阅读Alexey Voitilov在最近发行的Java Magazine上的文章“ Java on Arm Processors”。

亚历山大和德米特里第二次在jug.msk.ru上讲话-亚历山大先前于2014年1月30日 ，德米特里于2017年5月22日做了报告。

亚历山大设法在Sun Microsystems和Oracle上工作 。 在Oracle工作期间，他参与了组织Java Day和JavaOne Russia会议，并极大地影响了JUG.ru的创建。

关于报告


Andrey Kogun回顾即将举行的JPoint 2019大会，介绍演讲者并开始会议。



在会议的第一部分中，Alexander讲述了（说明演示文稿 ） BellSoft的创建历史，其员工对Open JDK的创建所做的贡献， Liberica JDK分发工具包的背景和功能。



双方都已经与德米特里（Dmitry）一起回答了观众的许多问题，并用另一个演讲的幻灯片补充了答案。



休息后，Dmitry继续使用Liberica JDK 11的Docker映像的具体示例演示了针对容器环境的Java 11功能。 该报告的最终版本（包括幻灯片）将在一个月半的JPoint会议上进行演示。



由于时间不足，“亲爱的，让我们尝试一下ARM？ “理论，应用和工作量”以简化形式显示。 亚历山大·别洛克里洛夫（Alexander Belokrylov）和阿列克谢·沃伊特洛夫（Alexey Voitylov）表演的手表版本可以从Joker 2018的录音中看到（上面提供了报告的链接）。
