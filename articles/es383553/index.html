GameKid - Clone Game Boy de Raspberry Pi



Muchos amantes de la electrónica y los viejos juguetes de consola han usado durante mucho tiempo la Raspberry Pi para lanzar un tipo diferente de emulador de consola. Los emuladores, especialmente los universales, le permiten jugar juegos de varias consolas, incluidas las muy antiguas y raras. Entre las plataformas de hardware populares que se emulan se encuentra Game Boy. Fueron precisamente los artesanos quienes decidieron recrearlo, utilizando tanto la Raspberry Pi como la carcasa impresa en una impresora 3D.

La idea de recrear un dispositivo retro a partir de nuevos componentes electrónicos surgió con dos hermanos justo a tiempo para Navidad. Después de varias semanas de trabajo, nació GameKid. Para lanzar juguetes bajo Game Boy en esta consola, los hermanos decidieron usar el entorno de software RetroPie .

El caso GameKid es bastante similar al precursor. Hay cuatro botones frontales para controlar el juego, D-Pad, iniciar y seleccionar botones. Además, hay altavoces incorporados y un conector para auriculares. Si lo desea, el usuario de una consola de juegos portátil de este tipo puede conectar un televisor a través de un puerto HDMI. Además, hay cuatro puertos USB que le permiten conectar controladores de juegos estándar. Cuando se conecta a un televisor o monitor, esta función será muy útil.

En el interior, además de la Raspberry Pi 2 con RetroPie, hay una tarjeta de memoria de 16 GB, ROM con pistas de audio y MP3. Los hermanos ahora están recaudando fondos para este proyecto en el sitio de crowdfunding de Kickstarter., y si lo desea, puede pedir el dispositivo por $ 139. En principio, un poco caro, pero las capacidades del kit son mucho más avanzadas que las de GameBoy habitual. Sí, los cartuchos no se usan aquí, para el juego necesitas cargar cientos de juguetes en la memoria de la consola, y puedes disfrutarlo.
