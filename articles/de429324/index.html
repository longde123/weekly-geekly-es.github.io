Unreal Engine 4.21 Release

Epic Games hat Unreal Engine 4.21 veröffentlicht, das Entwicklern laut Hersteller mehr Effizienz, Produktivität und Stabilität für jedes Projekt auf jeder Plattform bietet. Dank der neuesten Version können Benutzer mit Unreal Engine erstklassige Ergebnisse erzielen.


Unreal Engine 4.21 basiert auf bewährten Tools, damit Sie so effizient und kreativ wie möglich sind und noch mehr mobile Optimierung für Android und iOS, bis zu 60% schnellere Inhaltsvorbereitung sowie mehr Leistung und Flexibilität in branchenführenden Niagara-Tools für hinzufügen können VFX in Echtzeit. Darüber hinaus können Entwickler mit dem neuen Replication Graph-Plugin, das zur Veröffentlichung bereit ist, Multiplayer-Spiele in einem bisher unmöglichen Maßstab erstellen. Mit Pixel Streaming können Sie interaktive Inhalte ohne Kompromisse bei der Renderqualität direkt auf Remote-Geräte übertragen.


Aktualisierungen in Unreal Studio 4.21 bieten Benutzern in großen Projekten, einschließlich Architektur, Fertigung, Produktdesign und anderen Bereichen der professionellen Visualisierung, neue Funktionen und eine höhere Produktivität. Das Datasmith-Toolkit von Unreal Studio unterstützt jetzt Autodesk Revit und erweiterte Materialübersetzungen für Autodesk 3ds Max, wodurch das Design effizienter angezeigt und iteriert werden kann.



Zu den wichtigsten Änderungen an Unreal Engine 4.21 gehören:
  • Replikationsdiagramm-Plugin
  • Niagara Verbesserungen
  • Sequenzer Verbesserungen
  • Pixel-Streaming (früher Zugriff)
  • Optimierung mobiler Geräte
  • Beschleunigte Kochzeiten
  • Gauntlet Automation Framework (früher Zugriff)
  • Optimierung und Verbesserung des Animationssystems
  • Unterstützung für Blackmagic-Grafikkarten
  • Verbesserte Medien-E / A.
  • Windows Mixed Reality
  • Magic Leap Verbesserungen
  • Oculus Avatare
  • Datasmith für Revit (Unreal Studio)
  • Projektvorlage für Mehrbenutzer-Viewer (Unreal Studio)
  • beschleunigte Automatisierung mit Jacketing and Defeaturing (Unreal Studio)
  • Überlegene 3ds Max-Materialübersetzung (Unreal Studio)
