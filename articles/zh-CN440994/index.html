什么是精益创业/精益创业？


精益创业方法论是由埃里克·里斯（Eric Rees）开发的，埃里克·里斯（Eric Rees）在俄罗斯以他的《从零开始》一书而闻名。 通常，此方法旨在在开始创建产品并开始业务之前了解客户的偏好。 您可以使用最少的迭代和方法来制作产品。 主要口号是“直到创造出完美的产品为止，进行创造，衡量和研究”，众所周知，理想是无法实现的。 因此，精益创业是一种整体方法论，是一种创建真正满足消费者需求的产品的整体哲学。

成功的重要标准之一是摘下粉红色眼镜。 很多时候，当检验指导公司创建产品的主要假设时，源数据是不正确的。 对于商人来说，重要的是要有力量接受真相，以便了解基本假设最初是不正确的。 此后，可以进行突破性的更改。 最主要的是要冷静地看待您的产品，并根据客户的需求与客户合作。



埃里克·里斯（Eric Rhys）的4个常识

  4. 成功的启动应该在“创建-评估-学习”模型上进行。 在创建阶段，形成了最低限度可行的产品。 然后，评估消费者的反应，并在下一阶段确定是按照预期的路线还是改变经商的方向。

基本概念


精益启动方法基于六个循环互连的元素。

  1. 想法和分析。 在创建产品时，请记住该产品旨在解决特定人群的问题。 分析现有解决方案以及您创建的产品将如何发挥作用非常重要。
  2. 研究观众。 与潜在客户建立联系。 您可能有以前没有考虑过的想法。
  3. 产品创造。 从最冒险的想法开始，创建了基本的最低可行产品。 大规模试用您的产品。 此时，创建一个近似的财务模型。 在此模型中，将确定成本和潜在的收入来源。
  4. 测试。 在此阶段，您将从听众那里获得反馈。 重要的是要理解数据必须客观，才能创造出可行的产品。
  5. 处理数据。 处理数据时，您应该回答几个问题。 您的产品有用吗？ 有什么意想不到的问题吗？ 产品价格对客户来说太高了吗？ 如果测试数据表明该想法无效，那么您需要放弃它并开始考虑一个新项目。
  6. 缩放比例。 结果很好-很好。 使用数据并继续创建完整的产品。


假设检验


为了检验假设，应准备一个或多个业务模型选项。 您需要根据金钱和时间成本选择最有吸引力的模型。 突出显示最危险的假设，例如“客户”和“问题”，然后进行实验。 建立成功的标准以及指标和术语的各种限制。 实验本身可以采取访谈，观察，广告活动等形式。 在实验阶段结束时，将对数据进行分析，并再次开始进行假设检验。 必须执行此验证周期，直到您确信该产品适合市场。

初创企业成长和成功转化为成功的，可扩展的业务的理想模型应包括四个要点。

  1. 搜索想法和客户。
  2. 确定问题与解决方案之间的相关性。
  3. 客户验证和产品符合市场要求。
  4. 业务扩展。



精益创业公司的出现-他们自己错误的结果


这个概念的创始人是美国商人埃里克·赖斯。 他创建了一家为游戏玩家制作虚拟化身的公司。 该公司获得了相当不错的资金，但是在他们花了很多时间并获得了与他们投资一样多的钱之后，他们意识到用户不喜欢他们的产品并且不会射击。

然后，公司决定进行第一次面试研究，以了解出了什么问题。 人们发现许多原始的假设都是错误的。 如果莱斯的公司首先对其进行测试，那么将会有很多不同。

最初的错误假设是他们决定用户邀请他们的朋友。 研究发现，游戏玩家不想邀请朋友，因为这会威胁到自己的声誉。 青少年们这样说：“如果我邀请一个朋友，他会告诉我：这些化身是垃圾，那我最好不要这样做。”

结果，赖斯创造了一种没有邀请其他用户就没有意义的产品。 如果他最初检查了自己的想法，那么也许会获得对独立使用有价值的产品。
对于赖斯来说，这是一次可悲的经历，但是由于此，该公司改变了商业模式，并在市场上取得了成功。
