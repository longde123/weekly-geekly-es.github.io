Hanya satu tahun yang tersisa sampai akhir dukungan gratis untuk Windows 7



Berapa tahun yang lalu, Microsoft mengumumkan pengenalan dukungan lima tahun untuk sistem operasi Windows 7. Periode ini akan berakhir pada 14 Januari 2020 - tepat satu tahun kemudian.

Tetapi Windows 7 tidak akan kehilangan dukungan bahkan setelah itu. Yaitu, hanya Microsoft yang akan terus memperbaiki sistem operasinya untuk mendapatkan uang. Banyak perusahaan yang menggunakan sistem operasi khusus ini pada komputer kerja, laptop dan peralatan khusus akan membayar untuk terus menerima pembaruan.

Extended Security Updates (ESU), sebagaimana jenis dukungan ini disebut, juga terbatas waktunya. Durasi - tiga tahun, dengan setiap tahun harga untuk layanan Microsoft akan menjadi lebih tinggi.

Para pengguna yang menjadi anggota Windows Virtual Desktop (WVD) akan menerima dukungan secara gratis. Layanan ini menawarkan mesin virtual yang menjalankan "di cloud" di mana Windows 7 diinstal plus aplikasi yang diperlukan. Mesin virtual ini akan menerima pembaruan secara gratis, sampai akhir periode dukungan yang diperpanjang.

Situasi serupa berkembang dengan Windows Server 2008 dan Windows Server 2008 R2 - setelah satu tahun, dukungan untuk OS ini juga sampai pada kesimpulan logisnya.

Omong-omong, beberapa hari yang lalu, pembaruan baru untuk Windows 7 "merusak" sistem operasi ini. Alih-alih meningkatkan tingkat perlindungan komputer dengan "tujuh", pembaruan membawa masalah baru. Jadi, misalnya, pengguna lokal yang merupakan anggota grup Administrator lokal memiliki masalah dengan akses jarak jauh pada komputer yang menjalankan Windows Server 2008 R2 dan Windows 7.

Sekarang masalahnya telah dilokalisasi sebagian. Microsoft telah merilis pembaruan KB4487345, yang menghilangkan beberapa kesalahan. Pada saat yang sama, paket tidak tersedia untuk pembaruan otomatis, untuk menginstalnya, Anda harus mengunduh tambalan secara manual.

Pada awal pekan lalu, para pengembang merilis fix KB4480960 dan KB4480970 untuk Windows 7 SP1 dan Windows Server 2008 R2 SP1. Merekalah yang menyebabkan sejumlah besar kegagalan dalam sistem operasi. Pengguna sistem mengalami berbagai kesalahan, termasuk kesalahan Aktivasi KMS, Not Genuine, dan 0xc004f200.

Ternyata , tambalan KB971033 yang sebelumnya diinstal mulai bertentangan dengan KB4480960 dan KB4480970. Patch ini tidak dimaksudkan untuk dipasang di lingkungan Layanan Manajemen Kunci (KMS), meskipun sejauh ini tidak ada masalah yang diamati dengan mereka. Untuk mengatasi masalah tersebut, korporasi mengusulkan untuk menghapus KB971033, menghapus cache KMS dan menghapus data aktivasi sistem operasi dari PC. Setelah itu, sistem harus diaktifkan kembali melalui KMS. Selain itu, Anda tidak dapat menggunakan pembaruan KB4480960 dan KB4480970.

Perlu dicatat bahwa pada tahun baru, Windows 7 untuk pertama kalinya dalam beberapa tahun telah kehilangan tempatnya sebagai pemimpin dalam daftar OS desktop paling populer. "Tujuh" digantikan oleh "sepuluh", yang memimpin puncak. Pada saat yang sama, Microsoft percaya bahwa pergantian pemimpin akan terjadi jauh lebih awal. Jadi, pada bulan-bulan pertama setelah peluncuran Windows 10, jumlah penggunanya tumbuh sangat cepat.

Pada saat itu, perusahaan berasumsi bahwa hanya dalam beberapa tahun, lebih dari satu miliar perangkat akan bekerja pada Windows 10. Namun seiring berjalannya waktu, proses pengalihan pengguna ke Windows 10 telah berlangsung lebih lambat. Dan sistem operasi menerima 500 juta penggunanya hanya pada Mei 2017. Ini memperhitungkan tidak hanya laptop dan desktop, tetapi juga Xbox, serta kacamata augmented reality Hololens.

Sudah pada September 2018, Windows 10 berjalan di 700 juta perangkat.


Struktur pasar sistem operasi desktop pada Desember 2018 menurut VentureBeat

Pada Desember 2018, Windows 10 menyumbang sekitar 39,22% dari pasar sistem operasi desktop. Windows 7 mengambil 36,9%. Pada musim panas 2015, Windows 7 diinstal pada lebih dari 60% komputer. Nah, G8 menyumbang 5,33% dari pasar sistem operasi.

macOS, pada gilirannya, diinstal pada sekitar 10% perangkat. Versi desktop Linux memperoleh 2,78% dari pasar. Menurut Venture Beat, sekitar 86,2% pasar dikendalikan oleh sistem operasi Microsoft.
