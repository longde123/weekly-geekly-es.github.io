Auf Mikrowellen ins All und ein bisschen Matan

Es gibt so ein amerikanisches Startup Escape Dynamics, die eine Trägerrakete eines eher ungewöhnlichen Typs entwickelt. Ihre Nachkommen fliegen nicht auf einer internen chemischen Quelle (z. B. beim Verbrennen von Kerosin oder Wasserstoff in Sauerstoff), sondern auf einer externen Mikrowelle. Die Reaktionskraft wird durch Ausstoßen von erhitztem Wasserstoff erhalten und durch einen Mikrowellenstrahl von einer Bodenquelle erwärmt. Die Macher dieses Startups glauben, dass es auf dieser Basis möglich ist, ein einfaches, billiges wiederverwendbares Raumschiff zu entwickeln, um (bisher kleine) Serienlasten in den Weltraum zu bringen. In der Tat bildet dieses Konzept sozusagen eine Brücke zwischen Einwegmedien, die die Zerstörung von High-Tech-Eisen im Wert von jeweils mehreren zehn Millionen Dollar implizieren, und Astroengineering-Projekten (wie einer Startschleife oder einem Weltraumaufzug), die eine billige Produktion, aber auch eine unerschwingliche Menge an Investitionen implizieren.




Am interessantesten ist, dass eine Mikrowellenquelle, die mit einer Frequenz von 92,3 Gigahertz arbeitet, ein meinen Lesern bekanntes Gyrotron ist , das für die ECRH- Plasmaerwärmung in ITER und anderen thermonuklearen Maschinen verwendet wird.
