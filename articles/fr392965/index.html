Oui, vous pouvez exécuter des applications Linux de bureau sous Windows 10



Un message de Microsoft à propos de support natif pour Bash sur Ubuntu sous Windows 10 a causé tout un émoi. Dans le même temps, la société a décidé de ne pas étirer le plaisir d'attendre les utilisateurs de son système d'exploitation et n'a publié Windows 10 Insider Preview Build 14316 avec Ubuntu intégré que quelques jours après l'annonce. Une mise à jour majeure à part entière de Windows 10 avec une nouvelle fonctionnalité sera publiée au cours de l'été, mais pour l'instant, vous pouvez explorer les capacités du système dans la version d'aperçu.

Les utilisateurs qui ont testé le nouveau produit indiquent que tout fonctionne vraiment. Dans le même temps, la question s'est posée - pouvez-vous travailler uniquement sur la ligne de commande, ou existe-t-il un moyen de lancer une application de bureau avec une interface graphique? Après vérification, un certain nombre d'utilisateurs ont réussi à lancer certaines applications Linux de bureau. Il est officiellement déclaré que ce n'est pas possible, mais un utilisateur de Reddit avec le surnom w2qw a trouvé un moyen d'exécuter Xs dans Windows 10, ce qui ouvre la possibilité d'installer et de travailler avec VIM et Firefox.



Bien sûr, le même Firefox fonctionne également sous Windows (enfin, cela ne fonctionnerait pas), mais si vous devez exécuter des applications qui ne fonctionnent que sous Linux, cette méthode peut vous aider. Donc, vous devez utiliser Xming X Server pour Windows , cela vous permettra d'exécuter "X" en dehors de bash. Et puis nous venons de lancer l'application:

DISPLAY =: 0 firefox.

Tout cela fonctionnera un peu plus lentement que l'application native, mais les performances sont tout de même acceptables.

Un autre utilisateur de Reddit, starlig-ht, a déclaré qu'il était capable de lancer Xfce.

Permettez-moi de vous rappeler que pour activer Bash sur Ubuntu dans l'environnement Windows 10 Insider Preview Build 14316, vous devez activer le mode développeur dans les paramètres de mise à jour et de sécurité. Dès que l'utilisateur fait cela, vous pouvez appeler l'invite de commande Windows, enregistrer «bash» et démarrer Ubuntu (ce processus est illustré dans la capture d'écran de l'annonce).
