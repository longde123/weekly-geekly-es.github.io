OMEGA-体育赛事时间安排的自动化

大家好，我叫谢尔盖

我是“ SERVODROID ”教育项目的负责人 。 我的项目是为儿童和成人开发易于重复使用的机器人。 我本人在圣彼得堡教机器人技术。 但是我也是一名运动员，我最喜欢的运动领域之一是冬季运动-激流回旋，这将在后面讨论。



在我国辽阔的土地上，有许多普通人甚至不敢怀疑存在的美好地方。 但是由于我来自这样的地方，所以我想在这个项目中谈论他。 在俄罗斯北部的摩尔曼斯克地区，有一个不起眼但联系紧密的城市，或者说是一个定居点，叫做镍 -以该城市开采的元素周期表中的元素命名。 全市人口约10,000。 谁会想到，在这样一个人烟稀少的城市中，可能会有一个真正的滑雪胜地，就像他们在大城市中所说的那样。 对于10k的人口来说，这是一个真正的滑雪胜地，这里设有运动场，儿童和成人训练和学习，拥有自己的滑雪缆车，可以滑雪，甚至可以单板滑雪！
简要的历史背景
1950年代，耐克斯基（Nikelsky）爱好者阿列克谢·科罗温（Alexei Korovin）和他的朋友们开始在当时茂密的草丛繁茂的Lysaya山上砍伐一块空地。 1966年，他开发了相同设计的牵引绳，并招收了第一批上学的小学生。 从这个场景来看，尼古拉·萨尔蒂科夫和瓦莱里·扎巴拉现在正在滑冰。

在70-80年间，该团队参加了区域比赛，并自信地进入了15支队伍中的前三名。 奥列格·祖耶夫（Oleg Zuev）的镍滑雪部门主管的学生曾多次获得北方节日的冠军，苏联冠军的冠军和得奖者。 随后，他在基洛夫斯克担任教练。 他的女儿是激流回旋比赛的俄罗斯冠军，他的儿子史蒂芬（Stepan）是俄罗斯国家队的成员，俄罗斯的冠军，也是欧洲锦标赛许多比赛的冠军。 新库兹涅茨克的教练A.诺维科夫培养了俄罗斯冠军埃琳娜·普罗斯特娃（Elena Prosteva）。 他还是镍滑雪的学生。

现在，滑雪板将继续致力于训练俄罗斯的新冠军！ 对于一个城市，甚至是一个与我们一样小的城市-运动员在这座山上的任何胜利-为整个国家的胜利-我们都为我们的运动员加油助威。

这个项目的实质


由于我本人是一名运动员，而且我也是一名老师，所以我了解年轻一代有机会让自己发展并展示其知识和技能的重要性是多么重要-毕竟，体育比赛是一个巨大的动力！

在我们的滑雪坡上，没有现代系统可以让您记录参与者走了多长时间，并且基本上，测量是以旧的方式进行的-使用裁判秒表宣布参与者在对讲机上的开始和结束。 是的，该方法已被证明是稳定的，但我一直想为法官和参与者本身增加新颖性和便利性，以便他们可以更有效地进行培训并独立跟踪结果。 为此，我想出了一个项目，在该项目中，我想创建一个简单而可靠的选择，我不怕说，这是注册服务商的国内类似产品。

OMEGA-自动完成记录系统


从运动中可以知道，参与者的任务是在尽可能短的时间内从“ A”点到达“ B”点。 并且为了考虑到该时间，有必要使计时器完全从开始标记的交点开始并在与终点线相交时停止的过程自动化。 早在2016年，我就想到了组装这种设备的想法。 应当指出，这一想法得到了支持，一个月后，我在山上测试了一个演示设备。 该设备是完全有线的-一条电缆沿着山延伸了300米，信号被发送到法官的计算机。 您可以在这里看到它的外观。 该系统有效，但是使用电线会导致其永久性损坏（折叠和展开到机架中时会断裂）并冻结法官的计算机，这将不得不放在街上的寒冷中。

下面的视频演示了无线工作的新版本系统的工作版本，使法官可以在舒适的条件下工作，并在几分钟内即可完成系统的部署和运行：

发生了什么变化，还有什么要做


在2019年，我决定重新设计硬件，并用无线电模块替换电线。 研究了市场上现有的无线电模块选件之后，我选择了相当有前途的NRF24无线电通信模块。 这些通信模块将使您可以在最远1000公里的距离内传输数据，在我的情况下，则可以在整个2000公里内传输数据！

系统由什么组成


1.发射台模块是一种设备，其中包含Arduino板载微型计算机，NRF24L01通信模块，以及记录参与者运动开始的特殊传感器。 除传感器外，该模块还配备了特殊的声音和灯光指示器，使参与者清楚知道是允许还是禁止启动。 这样，您就可以卸下法官和管制员的工作，使他们能够专注于参与者的出发质量和他在高速公路上的旅行。


（在照片中：“开始”表的模块已安装开始标志，并且要安装在支架上的安装座被挖入雪中）

2.完工台模块是与上述模块结合使用相同配件的设备，但有一些区别：强大的激光束用于记录完工段的相交部分，该相交部分可以在相当远的距离发光，使您可以使用系统不仅在滑雪条件下。 该模块具有天线和固定机构，通过它可以将其固定在任何圆形物体上，甚至可以将其固定在雪地上的铲子上！


（在照片中：完成工作台的模块，已安装激光模块和用于测量激光束的10米电缆-光电探测器）

3.无线电通信模块 -这些小盒子能够在最远1000-2000米的距离处接收和发送信号（由于两个具有远程天线和放大器的相同型号，因此距离较大），这意味着法官可以在司法机构中感到舒适温暖，安静，温暖的屋子里，喝杯咖啡来记录每个参与者的比赛。 在有线系统的情况下-这是一个很大的不适-我自己就知道了，因为我一直坐在终点控制工作=）这些模块通过USB电缆连接到计算机，并由笔记本电脑供电。 一个合理，简单，有效的解决方案，可解决法官的流动性和系统部署的简便性。


（图：连接到将运行计时器软件的计算机的接收器模块）

4.软件是系统中最重要的部分。
该软件是用C＃和C / C ++开发的。 该程序的主要任务是捆绑熨斗，并为法官提供方便，清晰（简单）的工作界面。 该程序可以在手动和自动模式下工作，可以在参与者开始时独立启动计时器，并在越过终点线时独立停止计时器（在表格中输入时间）。 了解比赛的经验-经常会发生参与者没有时间遵守其订单的情况，因此，该程序已经提供了这样的选择。 该程序还确保法官不会意外关闭/关闭具有结果的程序-强制保存数据并在Excel中显示它们。


（在照片中：法官工作所在的程序界面记录时间并控制熨斗的工作）


该系统由什么组成


由于最初的计算是基于在一个小城市（如我的村庄）中使用的无线电组件的容易获得性，因此，我尝试选择最流行且易于访问的组件。

  • 无线电模块NRF24L01 + PA + LNA 1100m-4个
  • 4毫米胶合板-用于切割盒子和必要的元件
  • Arduino UNO开发板-4个
  • 用于NRF24L01的适配器-如果没有它，这些通信模块的行为将不正常
  • 胶合板的盒子和系统设计（用激光切割机切割出所有东西）
  • 可以进行更新的软件开发（主要针对Arduino和C＃中的裁判程序进行更新，以补充和改进功能）
  • 10米电缆连接完成表细节
  • 激光模块实质上是任何激光指示器。 我之所以选择这款，是因为它在商店中备有各种娱乐性装饰品，非常受欢迎，如果有的话，您可以通过购买新的装饰品来快速替换它。
  • 便携式模块的可充电电池
  • 消耗品，胶水，电线，指示器，按钮，连接器。
