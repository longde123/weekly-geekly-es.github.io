Tinjau Google IoT Platform

Banyak perusahaan terkemuka di dunia bereksperimen dan menerapkan solusi IoT untuk beragam wilayah operasional. Pertumbuhan permintaan untuk IoT dalam penggunaan konsumen dan industri jelas.
Internet of Things (IoT) menyediakan ruang lingkup raksasa aplikasi yang mungkin untuk:

  • Mengoptimalkan proses industri
  • Menghemat sumber daya energi
  • Kelola faktor lingkungan seperti udara, kelembaban, suhu, dan kualitas air, yang secara langsung memengaruhi kehidupan manusia

Pengenalan IoT mengubah dunia, menjadikannya lebih baik bagi kita masing-masing.

Semakin banyak solusi IoT diimplementasikan, semakin banyak data membutuhkan analitik akut dan bahkan kecerdasan buatan untuk dengan cepat menerima informasi yang relevan. Info tersebut memungkinkan perusahaan menentukan penyimpangan dan gangguan teknis secara real time. Misalnya, industri otomotif meningkatkan keselamatan lalu lintas menggunakan teknologi pintar seperti pencegahan kecelakaan, lalu lintas lalu lintas, dan sistem deteksi mata-off-the-road.

Dengan semua yang disebutkan di atas, pengambilan keputusan waktu-nyata dalam sistem IoT masih terbatas dengan biaya, keterbatasan faktor bentuk, keterlambatan transfer data, dan kriteria lainnya. Google Cloud IoT mencoba mengubah situasi yang ada menggunakan pembelajaran mesin pada perangkat akhir dan dengan memperkenalkan layanan seperti Cloud IoT Core, Cloud IoT Edge, Edge TPU.



Cloud IoT Core


Cloud IoT Core adalah layanan yang sepenuhnya dapat dikelola yang memungkinkan untuk terhubung, mengkonfigurasi, dan menerima data dari banyak perangkat dengan cepat dan aman. Menggunakan Cloud Pub / Sub, Core dapat menggabungkan data dari perangkat desentralisasi ke dalam sistem global tunggal. Jika dikombinasikan dengan layanan Google Cloud IoT lainnya, Core menyarankan solusi yang kompleks untuk mengumpulkan, menganalisis, dan memvisualisasikan data IoT secara real time. Ini, pada gilirannya, memungkinkan menciptakan model multifungsi yang mengoptimalkan atau memprediksi data berharga untuk bisnis Anda.

Cloud IoT Core mendukung protokol MQTT dan HTTP konvensional, yang memungkinkan pelanggan memanfaatkan banyak perangkat yang ada untuk membangun sistem IoT mereka. Layanan ini berfungsi di infrastruktur Google yang secara otomatis berskala dalam waktu nyata sambil memungkinkan koneksi satu juta perangkat jika diperlukan.

Inti berisi:

Pengelola perangkat
Atur dan kelola perangkat terpisah melalui konsol atau perangkat lunak. Gunakan mekanisme identifikasi dan otentikasi perangkat pada koneksi. Anda juga dapat mengkonfigurasi logika setiap perangkat dan mengontrolnya dari jarak jauh menggunakan cloud.

Jembatan protokol
Titik akhir koneksi untuk menyeimbangkan muatan komponen, dukungan asli MQTT dan HTTP, menyimpan telemetri ke Pub / Sub adalah fitur terbaik yang disediakannya.

Clout IoT Core mengumpulkan data yang diterbitkan di Cloud Pub / Sub untuk analisis lebih lanjut. Gunakan Google BigQuery untuk melakukan analisis spesifik atau Cloud Machine Learning Engine untuk menerapkan pembelajaran mesin. Setelah itu, Anda dapat memvisualisasikan hasil menggunakan berbagai laporan dan dasbor informasi di Google Data Studio.

Selain itu, untuk menyederhanakan dan mempercepat koneksi berbagai perangkat ke Core, Anda dapat memanfaatkan layanan penyediaan tambahan dari Google IoT. Itu membuat orientasi pelanggan dan produsen. Layanan ini menggunakan keamanan tingkat perangkat keras yang dapat diandalkan (alias chip crypto atau Elemen Aman (SE)) untuk menyediakan perangkat dengan aman. Ini memungkinkan penyediaan jutaan perangkat ke Core yang tepat tanpa campur tangan manusia.

Cloud IoT Edge


Alat yang berguna untuk memperluas pemrosesan data Google Cloud dan menggunakan pembelajaran mesin pada perangkat tepi seperti jalur pengangkutan otomatis, mesin turbo, dll. Dengan ini, perangkat dapat berfungsi secara real time menggunakan data sensor mereka dan memprediksi hasil secara lokal. Edge dapat bekerja pada sistem operasi berbasis Linux. Ini berisi dua komponen runtime (Connect dan ML) dan ASIC chip custom hardware accelerator (TPU).

  • Edge Connect - Untuk koneksi aman perangkat tepi ke cloud, pembaruan perangkat lunak dan program mikro, manajemen transfer data Core.
  • Edge ML - Untuk in-perangkat inferensi pembelajaran mesin model TensorFlow Lite. Manfaat: kekuatan pemrosesan meningkat, latensi rendah, fleksibilitas, dan banyak lagi.
  • Edge TPU - AI chip untuk menjalankan model TensorFlow Lite di bagian tepi. Sangat efektif, dioptimalkan, dan memiliki beragam kegunaan. Dapat menjadi nilai tambah yang besar jika digunakan dengan Cloud TPU.

Manfaat:

  1. Pemrosesan tepi-lokal dari gambar, video, gerakan, suara, dan gerakan. Lebih efektif daripada mengirim data mentah ke cloud.
  2. Menghilangkan kebutuhan untuk mengirim sejumlah besar - data yang berpotensi rahasia -.
  3. Otentikasi perangkat tepi menggunakan JSON Web Token. Lebih efisien daripada TLS menumpuk otentikasi bersama.

Lebih sedikit fasilitas Edge TPU


Untuk model pembelajaran mesin TensorFlow Lite, rangkaian terintegrasi chip Edge TPU khusus adalah keharusan baru. Itu dibangun khusus untuk:

  • Persiapan ML lebih cepat di cloud.
  • Analisis ML lebih cepat di perangkat tepi.
  • Mengaktifkan sensor untuk membuat keputusan cerdas real-time lokal.
  • Meluncurkan AI yang tepat di tepi dengan efisiensi tinggi.

Dengan meningkatnya kebutuhan akan perangkat yang saling berhubungan yang membutuhkan kerahasiaan, waktu tunda yang rendah, dan kapasitas tinggi, peluncuran model AI pada perangkat tepi menjadi hal yang umum. Edge TPU sangat cocok untuk itu karena memberikan efisiensi tinggi dengan konsumsi sumber daya fisik dan energi yang rendah.

Edge TPU Dev Board


Papan dasar yang menyediakan semua koneksi tepi yang dibutuhkan. Anda dapat memasang atau melepas sirkuit System on Module (SOM) dan mengintegrasikan modul Edge TPU ke perangkat Anda.





Akselerator Tepi TPU


Perangkat USB yang menambahkan coprocessor Edge TPU ke sistem. Dilengkapi dengan konektor Tipe-C, Accelerator dapat dihubungkan ke sistem berbasis Linux apa pun untuk hasil ML yang cepat. Kasing ini berisi Rapsberry Pi seperti lubang konektor papan.



Dengan cara ini, Anda dapat membuat platform intelektual, yang akan memantau berbagai komponen dan efisiensi produksi secara keseluruhan secara real time, menggunakan sensor Anda dengan cepat.
