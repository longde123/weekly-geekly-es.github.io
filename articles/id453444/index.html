100 repositori GitHub paling berharga [sesuai dengan algoritma UOS]

Halo, Habr! Saya mempersembahkan kepada Anda suatu pilihan - sebuah terjemahan dari artikel dari Hackernoon "100 Top Repositori Paling Berharga dari 96 Juta" dari GitHub . Dan artikel itu sendiri ditulis oleh orang-orang yang menggunakan algoritma Jaringan U ° OS untuk mengidentifikasi proyek-proyek opsource paling berharga di Github.



Akan ada wahyu yang luar biasa sekarang, mungkin, tetapi GitHub bukan hanya hosting kode - ini adalah sebuah komunitas.

Secara total, lebih dari 30 juta akun, lebih dari 2 juta perusahaan, dan lebih dari 96 juta repositori sekarang terdaftar di Github. Semua ini bersama-sama adalah komunitas Github.

Tetapi bagaimana Anda menilai dampak komunitas ini? Apakah ada, katakanlah, repositori teratas?

[di sini orang-orang mengatakan bahwa mereka menciptakan [algoritma reputasi] mereka di OS U ° (https://github.com/UOSnetwork/uos.docs/blob/master/yellow_paper/uos_yellow_paper.eng.pdf) dan melewati seluruh GitHub untuk menemukan repositori komunitas paling berharga]

Hasilnya cukup aneh, tetapi juga beraneka ragam, tampaknya itu mencerminkan minat pada berbagai teknologi dan menunjukkan ke mana kita bergerak.

Ada pemain besar dengan proyek sumber terbuka - Google, Apple, Microsoft, Facebook, bahkan Baidu. Dan pada saat yang sama ada, misalnya, alat untuk memerangi sensor di Tiongkok.

Ada bitcoin dan cryptocurrency.

Ada detektor partikel untuk Large Hadron Collider.

Ada proyek game, misalnya, Stasiun Luar Angkasa 13 dan Cataclysm: Dark Days Ahead, serta mesin game Godot.

Ada proyek-proyek pendidikan - freeCodeCamp, Open edX, Oppia dan [Code.org] (http://code.org/).

Ada proyek web dan proyek untuk membuat aplikasi seluler, misalnya, WordPress, Joomla dan Flutter untuk menerbitkan konten Anda.

Ada basis data untuk menyimpan konten Anda, seperti Ceph dan CockroachDB.

Dan ada mesin yang membantu kami menavigasi konten - Elasticsearch.

Juga, tidak mengejutkan, ada proyek jailbreak - manajer kompatibilitas Cydia untuk iOS dan firmware khusus 3DS Nintendo.

Dan ada sistem "rumah pintar" - Asisten Rumah.

Secara umum, ini adalah ramalan keren untuk seluruh dunia teknologi: kami belajar, membuat platform untuk mengekspresikan pendapat, kami menggunakan kriptografi, kami menyingkirkan perangkat lunak berpemilik di perangkat kami, dan di waktu luang kami memotong game di "rumah pintar" yang kami buat . Penting juga bahwa perusahaan besar dengan tenang membuka kode sehingga Anda juga dapat berkontribusi untuk semua ini.

Nah, sebelum Anda masuk ke daftar, kami akan membagikan hal menarik lainnya. Dalam prosesnya, algoritme juga memberi peringkat setiap kontributor GitHub. Jika Anda memiliki akun di sana dan Anda tertarik - Anda dapat melihat reputasi atau "peringkat" Anda di [https://u.community/github{(https://u.community/github) dan mengubahnya menjadi Universal Reputasi portabel.
