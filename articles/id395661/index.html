Instal Pengaya Google Chrome di Mozilla Firefox

Pengembang Firefox sedang berupaya memperkenalkan API add-ons baru yang disebut WebExtensions, yang digunakan di browser Chrome dan Opera. Versi stabil pertama API memulai debutnya di Firefox 48, yang sudah tersedia untuk diunduh (pengumuman rilis resmi diharapkan dalam beberapa jam mendatang).

Untuk menginstal add-on dari Chrome di Firefox, Anda harus:

1) Instal Chrome Store Foxified : ini menggantikan tombol "tambahkan ke Chrome" dengan "add to Firefox", dan juga menangani semua pekerjaan mengunduh, mengonversi, dan menandatangani add-ons yang diinstal.



2) Instal add-on yang diinginkan (aplikasi dan tema belum didukung) dari direktori web Google Chrome : itu akan diunduh dan dikonversi dari format crx ke format xpi standar untuk Firefox . Akun Google tidak diperlukan. Pengguna memiliki pilihan: instal add-on sementara (sebelum memulai kembali browser, yang nyaman untuk penilaian kinerja awal) atau menandatangani dan menginstalnya secara berkelanjutan.



Untuk mendapatkan tanda tangan, Anda harus terlebih dahulu membuat akundi repositori Mozilla . Pemeriksaan otomatis biasanya memakan waktu beberapa detik. Selain itu, dimungkinkan untuk menyimpan add-on yang tidak ditandatangani ke hard drive (misalnya, untuk menandatanganinya nanti secara manual di komputer lain).



Setelah instalasi berhasil, ikon add-on (jika disediakan untuk) akan muncul pada panel browser, dan pengaturan akan tersedia melalui mekanisme manajemen add-on Firefox standar. Namun, harus diingat bahwa belum ada pembicaraan tentang kompatibilitas 100%. Bekerja di WebExtensions berlanjut. Selain itu, beberapa add-on menggunakan fitur khusus Chrome yang tidak ditemukan di Firefox.
