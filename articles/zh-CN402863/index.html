诚实的示范合同作为公共关系自我调节的机制

美好的一天！ 我想与社区分享有关IT中使用的方法如何帮助人们解决日常任务的想法。 并提出一种特定的解决方法，其优势仅在于广泛的人们使用它。 它将涉及以下方法和工具：

  • 代码重用
  • 语义版本控制
  • Github
  • 乳胶

作为主题领域，将考虑“购买汽车”，但是，所提出的方法适用于解决其他问题。 谁在乎-欢迎猫。

为了节省时间和精力，我决定购买汽车，只求助于官方经销商，但仍然遇到了俄罗斯汽车市场的问题。 下面，在描述与经销商的联系时，我会积极使用“一些”一词，以免冒犯任何人（因为即使是同一品牌的官方经销商也存在差异）。

问题的简短描述


通常，打电话时，他们会尝试以各种可能的方式诱使您进入沙龙（有些甚至声称有汽车，尽管事实并非如此）。 在机舱中，没有一个管理人员着急，时间流逝：要么没有打印合同，授权书到期，就没有签发汽车销售合同，等等。 等

有了这些经验，我决定以后不要在正式部分（与合同有关的所有事情）上浪费时间，而是在第一次打电话时就加以澄清。 但是，事实证明，合同问题（非常奇怪）是许多官方经销商的特征：

  1. 一些公司拒绝发送示范销售合同。
  2. 对于某些人而言，卖方的名称与授权的官方经销商无关。 只是一家不同的公司。
  3. 对于某些人来说，该协议是“不”。 或者他们甚至建议与许多未填写的字段签订合同（包括VIN！即输入了错误的VIN，并以新价格购买了！）。
  4. 对于某些人而言，合同在形式上是足够的，但买方的权利却微不足道。 或卖方的权利很高。
  5. 在尝试讨论协议中的可疑条款时，通常是经过专门培训的人员进行交谈，而交谈的唯一目的是向您解释，而不是一个小时的骆驼吗？ 并且他们有有效的合同，而且不是市场上的第一年，等等。

问题陈述和建议的解决方案


因此，作为买家的我的任务是节省时间并购买汽车 。

厌倦了解释下一个经销商的合同中的荒谬之处，我意识到我什至不想阅读他们的合同，因为我已经理解了我所需要的东西，并且“我猜他们会怎样欺骗我”。 因此，我意识到了解决问题的第一步。

解决的第一步：代码重用


我需要我知道的合同版本，不需要重新阅读 。 该协议是此类交易的典型代表，这意味着我可以与任何经销商联系。 此版本的合同可以保护我（法律上）免受不洁经销商的欺诈之害，但不会使诚实的经销商处于不舒服的境地。 因此，该协议是公平的。

因此， 解决方案的第一步是诚实的模型合同 。 显然，这种方法不会“正面”起作用：他们只会简单地将其与合同一并忽略。 第一步自然要紧随第二步。

解决方案的第二步：与社区共享解决方案。


语义版本控制的功能只有在社区接受的情况下才能体现出来。 诚实的模型合同也可以使用相同的机制，但效果更大。 使用所提出的方法，人自己可以获得优势，同时增强了已经使用它的人，并将优势提供给以后使用的人。 实际上，我们得到了一个具有正反馈的系统，该系统本身是封闭的。

在语义版本控制的情况下， “明确定义并制定了版本控制的概念，就可以更轻松地向软件用户传达意图。” 对于诚实的示范协议，更容易理解经销商的意图：问一个问题：“您是否按照诚实的示范协议工作？” （可能带有版本号）。

但是，仅仅讲述找到的解决方案是不够的。 还必须组织关于《诚实信用示范条约》的联合工作。 这是第三步。

解决方案的第三步：组织联合工作。


为了在诚实模型合同上进行合作， 在Github上创建了一个存储库 ，其中合同的源文本为.TeX格式 。

任何人都可以帮助完成合同。 以下是主要领域：

  • 合同文本的定稿。 特别是如果您是律师或经销商代表 。 合同应考虑卖方的风险，但买方不应分担这些风险。 了解卖方的业务流程将有助于改善合同，甚至可以根据特定条件准备多个版本。

    示例 ：如果汽车已经在机舱中（有库存），则交货时间不能为1.5个月。 否则，在经销商不道德的情况下，买方已经承担了从全额预付款中“扭曲”该百分比的风险。 而且，相反，如果汽车仅在交付状态，甚至从国外交付，则1.5个月就足够了。
  • 合同的样式。 在当前版本中，使用LaTeX标记并不是一帆风顺的。
  • 改进的可用性。
    添加无需更改合同来源即可创建pdf文件的功能，该合同包括卖方和买方的数据（例如，以两个json文件形式显示）。 创建有关如何从源代码汇编现成的pdf文档的说明（例如，我使用了在线服务，但我认为必须有一个脱机选项）。
  • 提倡诚实示范合同的想法。
  • 诚实示范合同的实现。 例如，在法律变更的情况下。

可行性研究


我想相信，随着《诚实信用示范协定》的执行，人与人之间的关系将会改善。 并且服务质量将提高。

此外，如果您在每个主题上都提供评论，则合同可能是研究主题领域的一个很好的例证：重要的是什么，一个人在更改措辞或删除此项目时承担什么风险。

但是，使用诚实的模型合同可以获得的主要好处是节省时间。 在以下活动领域中可以节省时间：

  • 对这个主题领域的陷阱的研究。
  • 无需分别阅读每个经销商的合同（如果问题出在一个或两个经销商中，则该文章将根本不存在）。
  • 不必提议并协调每个经销商的合同更正。
  • 如果出现问题，请使用正确的合同（并有明确的期限），您可以快速开始试用并提前退还预付款。

在破坏者的领导下，我评估了大量使用诚实模型合同所节省的时间。 该评估非常具有指示性 ，因此，如果有人可以提供更充分的评估，请在评论中给出。

  • 05分钟 -请求和接收合同。
  • 30分钟 -合同的校对和分析。
