Mengubah saklar radio tombol menjadi sentuhan



Saya telah lama bermimpi memasang saklar sentuh di rumah, mimpi yang menjadi kenyataan ketika saya dihadirkan dengan saklar sentuh FD Electronics yang murah. Itu terlihat indah, sangat bagus. Tapi ini hanya saklar, dan saya memiliki rumah pintar berdasarkan Z-Wave dan tentu saja saya ingin mengintegrasikannya ke dalam sistem saya.

FD Electronics, Livolo dan DeLumo masing-masing memiliki saklar radio yang beroperasi pada frekuensi 433 MHz dan 868 MHz, tetapi mereka tidak dapat diintegrasikan ke dalam rumah pintar. Sakelar ini hanya berfungsi dengan remote dari pabrikan yang sama, tidak ada kontrol dari komputer, tidak ada umpan balik, mis. jika seseorang menyalakan lampu, saya tidak akan tahu tentang itu.

Diputuskan untuk menyeberang landak dengan ularsaklar radio tombol-tekan saya adalah Z-Wave. Saya Dimmer dengan saklar sentuh FD Electronics. Setelah mempelajari sirkuit kedua sakelar, saya memutuskan untuk membuang bagian daya sakelar sentuh dan hanya menggunakan bodi dan panel sentuh, dan mengambil bagian daya dan radio dari sakelar Z-Wave saya. Inti dari perubahan adalah penggantian tombol mekanis dengan sentuhan, metode ini cocok untuk setiap saklar teknologi.


Chip yang paling populer untuk membuat tombol sentuh adalah TTP-223 . Fitur utama dari TTP-223 adalah kemampuan untuk melakukan kalibrasi otomatis. Ketika daya diterapkan, rangkaian mikro mengukur kapasitansi pada kaki sensor I dan mengambilnya untuk 0, ini nyaman karena Anda tidak perlu mengkonfigurasi apa pun. Saat menyentuh sensor, tegangan muncul pada kaki Q , kami menghubungkan beban di sana (LED, relai, optocoupler).


Kaki TOG dan AHLB menyesuaikan respons untuk menyentuh sensor. Saya mengatur mode hidup dengan menghubungkan TOG dan AHLB ke tanah, yang berarti bahwa ketika saya menyentuh sensor pada kaki QTegangan muncul, ketika saya melepaskan sensor, tegangan menghilang. Anda juga dapat mengonfigurasikan mode pengalihan, lalu setiap sentuhan menempatkan kaki Q dalam kondisi sebaliknya.
Untuk tes TTP-223 pada papan tempat memotong roti, saya membuat sirkuit kontrol relay untuk Arduino. Ini bekerja dengan baik.


Untuk mensimulasikan penekanan tombol fisik, saya menggunakan optocoupler PC817C . Saya menyolder optocoupler alih-alih tombol pada dimmer saya. Ketika tegangan diterapkan pada optocoupler, itu menutup, untuk peredup itu seolah-olah mereka menekan tombol. Menggunakan LUT, papan adaptor dibuat, di satu sisi, rangkaian sensor, di sisi lain optocouplers.


Setelah menghubungkan semua sandwich ini, saya mendapat sentuhan Z-Wave redup. Semua tes dan perakitan memakan waktu 2 siang. Saklar dipasang di koridor, di mana ia dikendalikan oleh sensor gerak, dan jika Anda perlu mematikan atau menyalakan tingkat kecerahan tertentu secara paksa, Anda dapat dengan mudah menyentuh keindahan!


PS. Saya memutuskan untuk menggunakan pengalaman yang diperoleh untuk membuat sakelar sentuh bertenaga baterai, dan sudah memesan panel sentuh dari sakelar Livolo dengan aliexpress, tetapi lebih banyak tentang itu nanti.
