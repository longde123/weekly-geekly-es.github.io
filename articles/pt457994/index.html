Dicas e truques do Visual Studio

Se você é novo ou usa o Visual Studio há anos, há várias dicas e truques que podem torná-lo mais produtivo. Estamos compartilhando dicas no Twitter usando a hashtag #vstip há um tempo, e essa é uma coleção das melhores até agora.

Depurador


Pressionar F10 para criar, executar e anexar depurador em vez de F5 será interrompido automaticamente na primeira vez que seu próprio código estiver sendo executado. Não são necessários pontos de interrupção.



Compatível com o Visual Studio 2005

A recolocação no processo (Shift + Alt + P) é extremamente útil quando você precisa se conectar ao mesmo processo repetidamente.



Compatível com o Visual Studio 2017 v15.8

Um ponto azul na margem indica uma troca de threads durante a depuração.



Compatível com o Visual Studio 2013

Solução


Melhore o desempenho da carga da solução e reduza o ruído visual, desativando a restauração de expansões de nós no Solution Explorer e reabrindo documentos sobre a carga da solução.



Com suporte do Visual Studio 2019

Para uma navegação rápida pelo teclado, use Ctrl + T para encontrar qualquer coisa em sua solução - arquivos, classes etc.



Compatível com o Visual Studio 2017

Atribua um atalho de teclado para executar um "git pull" para que você não precise usar a CLI ou o Team Explorer para garantir que seu repo esteja atualizado.



Com suporte no Visual Studio 2019

Faça com que o Solution Explorer selecione automaticamente o documento ativo atual, para você nunca perder o controle de sua localização no projeto.



Compatível com o Visual Studio 2010

Editor


Coloque elementos HTML com facilidade usando Shift + Alt + W. O inserido é selecionado para que você possa editá-lo facilmente como qualquer tag que desejar, e a tag final corresponde automaticamente.



Compatível com o Visual Studio 2017

Copie qualquer fragmento JSON para a área de transferência e cole-o como classes .NET fortemente tipadas em qualquer arquivo de código C # ou VB.



Compatível com o Visual Studio 2013

Você não precisa escrever aspas nos nomes das propriedades JSON, basta digitar dois pontos e o Visual Studio inserirá as aspas automaticamente.



Suportado no Visual Studio 2015

Torne o IntelliSense e as dicas de ferramentas semi-transparentes enquanto você pressiona e mantém pressionada a tecla Control.



Compatível com o Visual Studio 2010

Em vez de redigitar '(' para mostrar informações de parâmetro nas assinaturas de métodos, use Ctrl + Shift + Space para exibir a sobrecarga usada atualmente.



Compatível com o Visual Studio 2010

Diversos


Tocar um som quando certos eventos ocorrerem no Visual Studio.



Compatível com o Visual Studio 2010

Crie layouts de janela personalizados para cenários de desenvolvimento específicos ou monitore configurações e alterne entre eles facilmente.



Compatível com o Visual Studio 2017

Especifique quais componentes do Visual Studio são necessários para qualquer solução e o Visual Studio solicitará que o usuário os instale se estiver ausente. Leia mais na postagem do blog Configure o Visual Studio na sua organização com .vsconfig .



Com suporte do Visual Studio 2019

Extensões


Verificador ortográfico do Visual Studio . Uma extensão do editor que verifica a ortografia dos comentários, seqüências de caracteres e texto sem formatação enquanto você digita ou interativamente com uma janela da ferramenta. Também pode verificar a ortografia de uma solução inteira, projeto ou itens selecionados. Estão disponíveis opções para definir vários idiomas para verificação ortográfica.



Compatível com o Visual Studio 2013

Adicionar novo arquivo . Uma extensão do Visual Studio para adicionar facilmente novos arquivos a qualquer projeto. Basta pressionar Shift + F2 para criar um arquivo vazio na pasta selecionada ou na mesma pasta que o arquivo selecionado.



Compatível com o Visual Studio 2015

Margem Git Diff . Git Diff Margin exibe alterações Git ao vivo do arquivo atualmente editado na margem e barra de rolagem do Visual Studio.



Compatível com o Visual Studio 2012

Esta foi apenas algumas das milhares de extensões disponíveis. Para ver mais extensões, acesse o Visual Studio Marketplace .
