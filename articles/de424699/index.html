Willkommen zum praktischen Workshop API Jam

Softline und Apigee (Google Cloud) laden Entwickler und Architekten API zum ersten praktischen Workshop API Jam in Russland ein, einem großen Workshop zur Arbeit mit der Apigee Edge-Plattform.



Wir zeigen Ihnen, wie Sie mithilfe der API schnell und sicher von außen auf das Backend zugreifen können. Es wird ein wenig Theorie geben, aber meistens üben, üben und wieder üben. Die Teilnehmer sammeln echte Erfahrungen mit der Apigee Edge-Plattform. Unter der Anleitung eines Trainers gibt es 5 Labore, in denen Sie Ihre bevorzugten APIs entwerfen, sichern und analysieren können.

3. Oktober, 09: 30-16: 00 Uhr
Google Moskau Büro


Programm:

Apigee Essentials

• Erkunden Sie die Apigee Edge-Benutzeroberfläche
• Verwalten von Organisationen, Umgebungen, Plattformbenutzern, Flows und Richtlinien
• Verstehen Sie die Beziehungen zwischen Entwicklern, Apps, Proxys und APIs
• Erfahren Sie mehr über API-Produkte und Entwicklerportale

Praktische Laborübungen

• Übung 1: Entwerfen und Erstellen eines API-Proxys
• Übung 2: Sichern Sie Ihre API mit einem API-Schlüssel
• Übung 3: Erstellen und Veröffentlichen eines API-Produkts
• Übung 4: Helfen Sie App-Entwicklern, Ihr API-Produkt zu konsumieren
• Übung 5: Drosseln des API-Verkehrs

Die Veranstaltung findet in englischer Sprache statt.


Um teilnehmen zu können, benötigen Sie:

Laptop - erforderlich; Google Chrome- oder Firefox-Browser werden bevorzugt.
Apigee Account. Sie müssen im Voraus ein Konto gemäß den Anweisungen erstellen.

Die Veranstaltung ist kostenlos, wir haben jedoch nur 20 Sitzplätze, sodass wir nicht alle akzeptieren können. Um sich zu registrieren, müssen Sie das Formular ausfüllen. Teilnehmer mit realen Fällen erhalten Vorrang.
