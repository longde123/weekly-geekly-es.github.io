Une caractéristique de la technologie CMOS à balayage progressif est le balayage progressif de l'image, c'est-à-dire qu'en un seul passage du faisceau, l'image est lue ligne par ligne et affichée à l'écran. Une telle caractéristique de la matrice permet d'éviter les «décalages» lors de la prise de vue d'objets en mouvement. De plus, le mouvement vertical du sujet est parfaitement transmis, ce qui est également important pour certains types de prise de vue.

Passerelle vers le cloud




Le prochain modèle, pour lequel nous offrons une remise, Ivideon Bridge est un appareil qui vous permet de connecter jusqu'à 16 caméras (ou canaux DVR) à notre service. Son objectif principal est de permettre aux utilisateurs de systèmes locaux d'accéder au cloud à un coût minimal et sans remplacer l'équipement installé.

En termes de rapport prix / canal, Bridge devient le moyen le plus rentable de se connecter à Ivideon: un canal avec Bridge avec stockage d'archives de base payé coûtera 375 roubles (en prix avant le Black Friday).

L'appareil sera nécessaire dans quatre cas:

  1. Vous avez décidé de faire évoluer le système de vidéosurveillance, acheté des caméras avec accès à Internet et les avez connectées au service cloud. Ivideon Bridge combinera d'anciennes caméras avec de nouvelles et vous permettra de contrôler l'ensemble du système via une seule interface.
  2. L'installation dispose de caméras à l'ancienne sans accès à distance via Internet. Dans ce cas, Bridge vous aidera à accéder à la diffusion en direct, aux archives et à configurer les caméras.
  3. À plusieurs reprises, les systèmes de vidéosurveillance de différents fabricants sont installés avec différentes applications de gestion, mais vous souhaitez combiner tous les systèmes et les contrôler via une seule interface.
  4. Vous souhaitez réduire les coûts de maintenance du système. Bridge connecte les caméras au cloud, ouvrant la possibilité de contrôler à distance les appareils. Vous pouvez ajouter des caméras via votre compte personnel - le client ne connecte que les appareils et le reste des actions sont effectuées à distance par un spécialiste de l'organisation de service.

Le compteur stocke les statistiques dans le cloud

La contre- caméra Ivideon Counter 3D coûte désormais 26 000 ₽.

Caractéristiques clés:

  • l'analyse intégrée considère les personnes avec une précision de 95%;
  • couverture de la largeur de passage jusqu'à 3 mètres;
  • microphone intégré;
  • Alimentation par paire torsadée (PoE) ou DC 12V;
  • détection de mouvement dans les directions d'entrée / sortie;
  • configuration et maintenance à distance;
  • ajustement flexible des zones de détection;
  • stockage des statistiques dans les archives cloud Ivideon, stockage vidéo - dans le NAS, localement sur un PC ou une carte microSD jusqu'à 128 Go.

Le compteur est conçu pour évaluer avec précision le trafic du point de vente: combien de visiteurs sont entrés, combien sont sortis, quelles heures et jours de la semaine - les données aident à prendre des décisions managériales dans le domaine du marketing, du développement et des RH.

Dans le compte personnel d'Ivideon, le compteur est ajouté comme n'importe quelle caméra avec notre firmware. De plus, il est possible d'ajouter un nombre quelconque de compteurs - cela est nécessaire si plusieurs points distants sont contrôlés. Dans ce cas, il est pratique de regarder toutes les statistiques dans un seul bureau.
