Microsoft hat offiziell bestätigt, dass Edge auf die Chromium-Engine umschaltet

Vor einigen Tagen gab es Gerüchte, dass Microsoft einen auf Chromium basierenden Browser entwickelt, der standardmäßig anstelle von Edge verwendet wird. Der Grund war, dass Microsoft-Mitarbeiter plötzlich anfingen, sich zu einem kostenlosen Projekt zu verpflichten. Jetzt hat Microsoft offiziell angekündigt, dass es seine eigene EdgeHTML-Engine zugunsten von Chromium in der Desktop-Version des Browsers wirklich aufgibt.

Für Webentwickler ist dies eine große Erleichterung: Während des Testens wird eine Engine kleiner und eine Reihe von Originalfunktionen von IE / Edge werden in die Geschichte eingehen. Laut Mozilla stellt eine solche Entscheidung von Kollegen von Microsoft jedoch eine Bedrohung für das Internet dar, da die Dominanz einer einzelnen Plattform von Google noch vollständiger wird. Es besteht die Gefahr, dass Firefox in eine so marginale Nische gerät, dass Entwickler sogar aufhören, Websites darauf zu testen. Der Anteil von Firefox nimmt weiter ab. Im Dezember 2018 fiel er auf den niedrigsten Stand seit vielen Jahren: weniger als 9% auf Desktops .

Microsoft argumentiert seine Entscheidung mit guten Absichten: „In den letzten Jahren hat Microsoft seine Teilnahme an der Open Source Software Community (OSS) erheblich ausgeweitet und ist zu einem der größten Unterstützer von OSS-Projekten weltweit geworden. Heute geben wir bekannt, dass wir beabsichtigen, mit dem Open-Source-Projekt Chromium eine Desktop-Version von Microsoft Edge auf dem Desktop zu entwickeln, um unseren Kunden eine bessere Webkompatibilität zu bieten und die Internetfragmentierung für alle Webentwickler zu verringern “, heißt es in der offiziellen Pressemitteilung. .

Microsoft ist davon überzeugt, dass die Entwickler nicht nur Microsoft Edge, sondern auch andere Browser auf Desktops und Mobilgeräten verbessern können, indem sie ihren Entwicklern einen Beitrag zum Chromium-Projekt leisten. Infolgedessen wird das Internet für viele Zielgruppen besser: „Microsoft Edge-Benutzer (und möglicherweise andere Browser) erhalten eine verbesserte Kompatibilität mit allen Websites sowie eine bessere Akkulaufzeit und Hardware-Integration auf allen Windows-Geräten. "Webentwickler erhalten eine weniger fragmentierte Webplattform zum Testen von Websites, die weniger Probleme garantiert und die Benutzerzufriedenheit erhöht."

Auf Wiedersehen EdgeHTML


"Goodbye EdgeHTML" ist der sympathische Mozilla-Blogbeitrag. Und das ist überhaupt keine Ironie: Kollegen bedauern wirklich, dass Microsoft sich weigert, eine eigene Engine zu entwickeln: „Nachdem Microsoft Chromium für die Verwendung übernommen hat, gibt Microsoft Google noch mehr Kontrolle über das Online-Leben“, schreiben sie. - Es mag melodramatisch klingen, ist es aber nicht. „Browser-Engines - Googles Chromium und Mozillas Gecko Quantum - sind interne Softwareteile, die tatsächlich definieren, was jeder von uns im Internet tun kann. Sie bestimmen die Hauptmerkmale: Welche Inhalte können wir als Verbraucher sehen, unsere Sicherheit beim Anzeigen von Inhalten und wie stark wir Websites und Dienste kontrollieren. Die Entscheidung von Microsoft bietet Google mehr Möglichkeiten, im Alleingang zu entscheiden, welche Optionen jedem von uns zur Verfügung stehen.

Mozilla ist der Ansicht, dass Microsoft diese Entscheidung aufgrund der Vorteile des Geschäfts getroffen hat: „Google ist so nah an der fast vollständigen Kontrolle über die Infrastruktur unseres Online-Lebens, dass es möglicherweise unrentabel ist, weiterhin damit zu kämpfen. Die Interessen der Microsoft-Aktionäre können durchaus dazu dienen, die Freiheit und Vielfalt abzulehnen, die uns das Internet einst geboten hat. Google ist ein starker Konkurrent mit sehr talentierten Mitarbeitern und exklusivem Besitz einzigartiger Vermögenswerte. Die Dominanz von Google in den Bereichen Suche, Werbung, Smartphones und Datenerfassung schafft ein sehr unfaires Spielfeld, das gegen uns alle wirkt. “

Aus sozialer, bürgerlicher und individueller Sicht ist es schrecklich, die Kontrolle über die grundlegende Online-Infrastruktur eines Unternehmens zu übertragen, sagte Mozilla. Sie betonen, dass die Aufrechterhaltung der Macht in den Händen der Benutzer der Grund ist, warum Mozilla als gemeinnützige Organisation existiert. Sie betrachten die Konkurrenz von Google nicht in Bezug auf das Geschäft, sondern in Bezug auf das Wohl des gesamten Internets. In diesem Sinne ist es ziemlich traurig, dass Firefox kürzlich den Tiefpunkt erreicht hat: Sein Anteil an Desktops fiel unter 9% . Dies geschah zum ersten Mal seit vielen Jahren, und der Anteil von Firefox nimmt weiter ab, was zur totalen Dominanz von Chrome führt.



Mozilla räumt ein, dass die Dominanz von Chromium Entwicklern und Unternehmen das Leben erleichtert: Beim Testen können Sie sich darauf beschränken, zu überprüfen, ob die Anwendung nur unter Chromium gut funktioniert: „Dies geschah, als Microsoft Anfang der 2000er Jahre vor der Veröffentlichung ein Monopol auf Browser hatte Firefox Und das könnte wieder passieren. “
