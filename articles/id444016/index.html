Raih lebih banyak dengan Microsoft Game Stack


Microsoft dibangun di atas kepercayaan untuk memberdayakan orang dan organisasi untuk mencapai lebih - itu adalah DNA perusahaan kami. Kami mengumumkan inisiatif baru, Microsoft Game Stack, di mana kami berkomitmen untuk menyatukan alat dan layanan Microsoft yang akan memberdayakan pengembang game seperti Anda, apakah Anda seorang pengembang indie yang baru memulai atau studio AAA, untuk mencapai lebih banyak.


Ini adalah awal dari perjalanan baru, dan hari ini kami hanya mengambil langkah pertama. Kami percaya Microsoft secara unik cocok untuk memenuhi komitmen itu. Perusahaan kami memiliki warisan panjang dalam permainan - dan dalam membangun platform yang berfokus pada pengembang.


Ada 2 miliar gamer di dunia saat ini, memainkan berbagai macam permainan, pada berbagai perangkat. Fokus pada streaming video, menonton, dan berbagi dalam komunitas sama banyaknya dengan bermain atau bersaing. Sebagai pembuat game, Anda berjuang setiap hari untuk terus melibatkan pemain, memicu imajinasi mereka, dan menginspirasi mereka, di mana pun mereka berada, atau perangkat apa yang mereka gunakan. Kami memperkenalkan Microsoft Game Stack, untuk membantu Anda melakukan hal itu.


Artikel ini di blog kami.

Apa sebenarnya Microsoft Game Stack?


Game Stack menyatukan semua platform, alat, dan layanan pengembangan game kami - seperti Azure, PlayFab, DirectX, Visual Studio, Xbox Live, App Center, dan Havok - ke dalam ekosistem yang kuat yang dapat digunakan oleh pengembang game mana pun. Tujuan dari Game Stack adalah untuk membantu Anda dengan mudah menemukan alat dan layanan yang Anda butuhkan untuk membuat dan mengoperasikan game Anda.


Awan memainkan peran penting dalam Game Stack, dan Azure memenuhi kebutuhan vital ini. Azure menyediakan blok bangunan seperti komputasi dan penyimpanan, serta layanan cloud-asli dari pembelajaran mesin dan AI, untuk mendorong pemberitahuan dan jangkar spasial realitas campuran. Azure sudah tersedia di 54 wilayah secara global, termasuk Cina, dan terus berinvestasi dalam membangun infrastruktur cloud yang sangat aman dan berkelanjutan serta layanan tambahan untuk pengembang game. Skala global Azure adalah apa yang akan memberikan teknologi streaming Project xCloud skala untuk memberikan pengalaman bermain game yang hebat bagi para pemain di seluruh dunia, terlepas dari perangkat dan lokasi mereka.


Sudah dengan Azure, perusahaan seperti Rare, Ubisoft, dan Wizards of the Coast menjadi tuan rumah server game multi-pemain, secara aman dan aman menyimpan data pemain, menganalisis telemetri game, melindungi game mereka dari serangan DDOS, dan melatih AI untuk membuat gameplay yang lebih mendalam.


Meskipun Azure adalah bagian dari Game Stack, penting untuk menyebutkan bahwa Game Stack adalah cloud, jaringan, dan perangkat agnostik. Dan kita tidak berhenti di sini.


Apa yang baru?


Bagian berikutnya dari Game Stack adalah PlayFab, layanan backend lengkap untuk membangun dan mengoperasikan game langsung. Setahun yang lalu, kami menyambut PlayFab ke Microsoft melalui akuisisi. Hari ini kami senang mengumumkan bahwa kami membawa PlayFab ke dalam keluarga Azure. Bersama-sama, Azure dan PlayFab adalah kombinasi yang kuat: Azure menghadirkan keandalan, skala global, dan keamanan tingkat perusahaan; PlayFab menyediakan Game Stack dengan layanan pengembangan game yang dikelola, analitik waktu nyata, dan kemampuan LiveOps. Musim gugur yang lalu, kami melihat apa yang dapat dilakukan kedua platform ini bersama dengan PlayFab Multiplayer Server, yang memungkinkan Anda untuk meluncurkan dan meningkatkan game multiplayer dengan aman dengan hosting server Anda secara dinamis dengan komputasi awan Azure.

Keberhasilan jangka panjang membutuhkan pemain yang terlibat dalam siklus penciptaan, eksperimen, dan operasi yang berkelanjutan. Tidak mungkin lagi hanya mengirimkan game Anda dan melanjutkan. " Inilah sebabnya mengapa setahun yang lalu, kami menyambut PlayFab ke Microsoft melalui akuisisi. PlayFab mendukung semua perangkat utama, dari iOS dan Android, ke PC dan Web, ke Xbox, Sony PlayStation, dan Nintendo Switch; dan semua mesin gim utama, termasuk Unity dan Unreal. PlayFab juga akan terus mendukung semua awan besar ke depan.


Hari ini kami juga bersemangat untuk mengumumkan lima layanan PlayFab baru dalam pratinjau.


Dalam pratinjau publik hari ini:


  • PlayFab Matchmaking : Matchmaking kuat untuk game multi-pemain, diadaptasi dari perjodohan Xbox Live, tetapi sekarang tersedia untuk semua game dan semua perangkat.

Dalam pratinjau pribadi hari ini ( hubungi kami untuk bergabung dengan pratinjau ):


  • PlayFab Party: Layanan suara dan obrolan, diadaptasi dari Xbox Party Chat, tetapi sekarang tersedia untuk semua game dan untuk semua perangkat. Party memanfaatkan Layanan Kognitif Azure untuk terjemahan dan transkripsi waktu nyata untuk membuat game dapat diakses oleh lebih banyak pemain.
  • PlayFab Insights: Menggabungkan telemetri game real-time yang kuat dengan data game dari berbagai sumber lain untuk mengukur kinerja game Anda dan membuat wawasan yang bisa ditindaklanjuti. Didukung oleh Azure Data Explorer, Game Insights akan menawarkan konektor ke sumber data pihak pertama dan ketiga yang ada termasuk Xbox Live.
  • PlayFab PubSub: Berlangganan klien game Anda ke pesan-pesan yang didorong dari server PlayFab melalui koneksi persisten, diberdayakan oleh Azure SignalR. Ini memungkinkan skenario seperti pembaruan konten waktu-nyata, pemberitahuan perjodohan, dan gameplay multi-pemain sederhana.
  • Konten Buatan Pengguna PlayFab: Libatkan komunitas Anda dengan mengizinkan pemain membuat dan berbagi konten yang dibuat pengguna dengan aman dengan pemain lain. Teknologi ini awalnya dibangun untuk mendukung pasar Minecraft.

Menumbuhkan komunitas Xbox Live


Komponen utama lain dari Game Stack adalah Xbox Live. Selama 16 tahun terakhir, Xbox Live telah menjadi salah satu komunitas game paling bersemangat dan aktif di dunia. Ini juga merupakan jaringan yang aman dan inklusif yang telah menguraikan batas-batas dalam cara gamer terhubung di seluruh perangkat.


Hari ini, kami senang Xbox Live menjadi bagian dari Microsoft Game Stack, menyediakan layanan identitas dan komunitas. Di bawah Game Stack, Xbox Live akan memperluas kemampuan lintas platformnya, saat kami memperkenalkan SDK baru yang membawa komunitas ini ke perangkat iOS dan Android.



Pengembang seluler sekarang akan dapat menjangkau beberapa gamer yang paling bersemangat dan bersemangat di planet ini dengan Xbox Live. Ini hanya beberapa manfaat untuk pengembang seluler:


  • Identitas Game Tepercaya: Dengan Xbox Live SDK baru, pengembang dapat fokus pada pembuatan game hebat dan memanfaatkan jaringan identitas tepercaya Microsoft untuk mendukung login, privasi, keamanan online, dan akun anak.
  • Integrasi Tanpa Gesekan: Tawaran layanan a la carte baru dan tidak ada sertifikasi Xbox Live memberikan fleksibilitas bagi pengembang seluler dalam cara mereka membangun dan memperbarui game mereka. Pengembang hanya menggunakan layanan yang paling sesuai dengan kebutuhan mereka.
  • Vibrant Gaming Community: Jangkau komunitas Xbox Live yang berkembang dan hubungkan gamer di banyak platform. Temukan cara-cara kreatif untuk memungkinkan pencapaian, Gamerscore, dan statistik "pahlawan", yang memiliki pengalaman sendiri di luar permainan, untuk membuat gamer tetap terlibat.

Komponen Game Stack lainnya


Komponen lain dari Game Stack termasuk Visual Studio, Mixer, DirectX, Azure App Center, Visual Studio, Kode Visual Studio, dan Havok. Dalam beberapa bulan mendatang, saat kami berupaya meningkatkan dan menumbuhkan Game Stack, Anda akan melihat koneksi yang lebih dalam di antara layanan-layanan ini karena kami menyatukan mereka untuk bekerja lebih mulus bersama.


Sebagai contoh bagaimana integrasi ini sedang berlangsung, hari ini kami menyatukan PlayFab dan komponen-komponen Game Stack ini:


  • App Center: Crash log data dari App Center sekarang terhubung ke PlayFab, memungkinkan Anda untuk lebih memahami dan merespons masalah dalam game Anda secara real-time dengan mengikat log crash kembali ke profil masing-masing pemain.
  • Visual Studio Code: Dengan plug-in baru PlayFab untuk Visual Studio Code, mengedit dan memperbarui Cloud Script menjadi jauh lebih mudah.

Ciptakan dunia Anda hari ini dan raih lebih banyak


Saat kami memperluas fokus kami ke cloud, sifat platform mungkin berubah, tetapi komitmen kami untuk memberdayakan pengembang game seperti Anda tidak tergoyahkan, dan kami menantikan perjalanan ke depan dengan Microsoft Game Stack. Tim kami terinspirasi dan bersemangat dengan kemungkinan saat kami mulai mengumpulkan semua layanan dan teknologi hebat ini. Pastikan untuk berbagi umpan balik Anda dengan kami saat ini, sehingga kami dapat membantu Anda mencapai lebih banyak. Jika Anda berada di GDC, mampirlah ke stan Microsoft di Aula Selatan Pusat Moscone untuk mencoba banyak layanan baru, dan untuk mempelajari lebih lanjut tentang peluang menarik di masa depan.
