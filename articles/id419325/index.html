Pindahkan siswa ke Prancis

Selamat siang, dalam posting ini saya akan memberi tahu Anda bagaimana, sebagai siswa matematika, saya berangkat ke Prancis, dan sekarang saya tinggal dan bekerja di sini, dan saya bahagia seperti gajah.




Entri


Secara singkat tentang kondisi awal: 2014, saya Ukraina, 18 tahun, saya telah tinggal di Dnepropetrovsk sepanjang hidup saya, mahasiswa 3 tahun (DNU, fakultas matematika terapan, ilmu komputer khusus). Teman-teman saya yang lebih tua perlahan-lahan pergi untuk belajar di luar negeri, saya mulai memikirkan hal yang sama, tetapi sangat menakutkan. Saya mendengar dari telinga saya tentang "pertukaran" antara DNU dan Le Mans University (selanjutnya Le Mans) - mengapa tidak mencobanya?


Kenapa "ditukar"?
