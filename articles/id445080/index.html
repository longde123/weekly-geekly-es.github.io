Di Rusia, akan membuat "kereta api digital"

Suatu hari, Perdana Menteri Rusia Dmitry Medvedev menyetujui program digitalisasi enam tahun untuk Kereta Api Rusia. Di antara titik-titik lain dari rencana itu, proyek "kereta api digital" termasuk di dalamnya.

Sayangnya, dokumen itu tidak menunjukkan jumlah yang akan diinvestasikan di bagian "digital" dari modernisasi Kereta Api Rusia. Tetapi jumlah total investasi dari investor swasta yang direncanakan akan ditarik untuk periode 2019-2025 adalah 4 triliun rubel. Volume dukungan negara adalah sekitar 200 miliar rubel, seperti dilansir Cnews.

Sekitar 132 miliar rubel dialokasikan untuk program informasi perusahaan dari proses transportasi hanya dari 2019 hingga 2025. Untuk pembuatan sistem kontrol intelektual terpadu dan otomatisasi proses produksi dalam transportasi kereta api untuk periode yang sama, 3,5 miliar rubel dialokasikan. Nah, sekitar 78 miliar rubel dalam enam tahun seharusnya diinvestasikan untuk memperbarui peralatan dan perangkat komunikasi.

"Kereta Api Digital" mulai dikembangkan setelah keputusan Presiden Federasi Rusia 7 Mei tahun lalu "Tentang tujuan nasional dan tujuan strategis pengembangan Federasi Rusia untuk periode hingga 2024".

Dalam paragraf "Digitalisasi dan Teknologi Informasi" dicatat bahwa daftar bidang utama untuk pengembangan sistem informasi Kereta Api Rusia akan mencakup pembentukan ruang informasi tunggal untuk transportasi barang dan logistik, ditambah ruang yang sama untuk kompleks penumpang untuk meningkatkan keuntungan.

Platform digital untuk mengelola dan memantau transportasi akan menyediakan pelacakan lokasi dan kondisi barang, termasuk layanan transportasi kargo dari pintu ke pintu. Jadi, direncanakan untuk menyediakan "pertukaran yang signifikan secara hukum" dari dokumen elektronik dengan peserta transportasi dan penggunaan data transportasi digital ujung-ke-ujung.

Alat manajemen transportasi penumpang digital yang sedang dibuat meliputi multimodality, perencanaan dan dukungan perjalanan dari pintu ke pintu, layanan penumpang yang dipersonalisasi, layanan pembayaran tarif elektronik, menu tarif yang fleksibel dan program loyalitas.

Kereta Api Digital akan memberikan peluang untuk membentuk teknologi digital ujung-ke-ujung dari proses transportasi untuk meningkatkan efisiensi transportasi dan infrastruktur. Juga diasumsikan bahwa dalam enam tahun ke depan akan dibuat sistem otomatis terpadu terpadu untuk mengelola perusahaan negara, sistem manajemen perusahaan akan dioptimalkan, dan analisis serta pelaporan bisnis asing akan dikembangkan.

Sistem kontrol lalu lintas yang cerdas akan dibentuk, yang akan menyediakan akses ke data transportasi dan infrastruktur yang relevan, pemodelan proses konstruksi, operasi dan perbaikan dengan mengacu pada waktu dan penganggaran.

Proyek ini akan mengintegrasikan Internet hal-hal, pemrosesan data besar, registri terdistribusi, pemodelan digital dan sistem kecerdasan buatan. Direncanakan untuk menciptakan generasi baru tempat kerja mobile, ditambah pengenalan manajemen dokumen elektronik.

Dengan demikian, pengembang memodifikasi infrastruktur komputasi dan telekomunikasi Kereta Api Rusia untuk memastikan tingkat aksesibilitas layanan informasi yang terjamin.

Sekarang di Rusia, program Ekonomi Digital sedang dilaksanakan, untuk implementasi yang direncanakan menghabiskan 1,8 triliun rubel dalam lima tahun ke depan. Lebih dari 1 triliun - dana anggaran federal. Sebelumnya dilaporkan bahwa bagian "Infrastruktur Informasi" dapat menjadi program termahal dari proyek nasional "Ekonomi Digital". Lebih dari 413,4 miliar rubel akan dialokasikan untuk bagian ini. Lain sekitar 282 miliar rubel. dihabiskan untuk teknologi digital, 226,4 miliar rubel - untuk administrasi publik digital, 139 miliar rubel. - untuk personel untuk ekonomi digital dan 18 miliar rubel untuk keamanan informasi.

Dari jumlah yang diumumkan, 820 miliar rubel ditemukan yang akan digunakan untuk melaksanakan program-program informasi badan-badan negara. Dana tersebut akan digunakan untuk mengorganisir pekerjaan bagi karyawan dari berbagai kementerian dan departemen, mengembangkan sistem TI mereka sendiri dan mendukung GIS.
