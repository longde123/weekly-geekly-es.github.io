Como desativar completamente o Windows Defender no Windows 10



O Windows 10 possui um antivírus interno do Windows Defender (“Windows Defender”) que protege seu computador e dados de programas indesejados: vírus, spyware, ransomware e muitos outros tipos de malware e hackers.

Embora a solução de segurança interna seja suficiente para a maioria dos usuários, há situações em que você pode não querer usar este programa. Por exemplo, se você estiver configurando um dispositivo que não ficará online; se você precisar concluir uma tarefa bloqueada por este programa; Se você precisar atender aos requisitos da política de segurança da sua organização.

O único problema é que você não poderá remover ou desativar completamente o Windows Defender - este sistema está profundamente integrado ao Windows 10. No entanto, existem várias soluções alternativas para desativar o antivírus - usando a diretiva de grupo local, o registro ou as configurações do Windows na seção Segurança (temporariamente).

Como desativar o Windows Defender através das configurações de segurança do Windows


Se você precisar executar uma tarefa específica e não precisar desativar completamente o Defender, poderá fazê-lo temporariamente. Para fazer isso, usando a pesquisa no botão "Iniciar", localize a seção "Windows Defender Security Center" e selecione "Proteção contra vírus e ameaças" nela.



Lá, vá para a seção "Proteção contra vírus e outras ameaças" e clique no botão "Proteção em tempo real".



Depois disso, o antivírus desativará a proteção do computador em tempo real, o que permitirá instalar aplicativos ou executar uma tarefa específica que não estava disponível porque o antivírus bloqueou a ação necessária.

Para ativar a proteção em tempo real novamente, reinicie o computador ou siga todas as etapas de configuração novamente, mas ligue a chave na última etapa.

Esta solução não é permanente, mas é mais adequada para desativar o Windows 10 Antivirus para uma tarefa específica.

Como desabilitar o Windows Defender por meio de diretivas de grupo


Nas versões do Windows 10 Pro e Enterprise, você pode usar o editor de diretivas de grupo local, onde é possível desativar permanentemente o Defender da seguinte maneira:

Através do botão "Iniciar", execute o script executável gpedit.msc. O editor de políticas é aberto. Vá para o seguinte caminho: Configuração do computador> Modelos administrativos> Componentes do Windows> Windows Defender Antivirus.



Clique duas vezes para abrir a opção "Desativar o programa antivírus do Windows Defender". Selecione a configuração "Ativado" para ativar esta opção e, em conformidade, desative o Defender.



Clique em "OK" e reinicie o computador.

Depois disso, o antivírus será permanentemente desativado no seu dispositivo. Mas você notará que o ícone com o escudo permanecerá na barra de tarefas - deve ser assim, porque esse ícone pertence ao aplicativo de segurança do Windows e não ao próprio antivírus.

Se você mudar de idéia, sempre poderá reativar o Defender repetindo essas etapas e selecionando “Não definido” na última etapa, após o qual será necessário reiniciar o computador novamente.

Como desativar o Windows Defender através do registro


Se você não tiver acesso ao editor de políticas ou tiver instalado o Windows 10 Home, poderá editar o registro do Windows, desativando o Defender.

Lembro que a edição do registro é arriscada e que erros nesse assunto podem causar danos irreparáveis ​​à cópia do Windows atualmente instalada. É melhor fazer backup do sistema antes de começar a editar.

Para desativar completamente o Defender através do registro, inicie o programa regedit através do botão "Iniciar" e siga-o pelo seguinte caminho:

HKEY_LOCAL_MACHINE \ SOFTWARE \ Policies \ Microsoft \ Windows Defender

Dica: esse caminho pode ser copiado e colado na barra de endereços do editor de registro.



Em seguida, clique com o botão direito do mouse na chave (diretório) do Windows Defender, selecione "Novo" e Valor DWORD (32 bits). Nomeie a nova chave DisableAntiSpyware e pressione Enter. Em seguida, clique duas vezes para abrir o editor de chaves e defina-o como 1.



Clique em OK e reinicie o computador.

Depois disso, o Windows Defender não protegerá mais o seu sistema. Se você quiser desfazer essas alterações, repita todas as etapas, mas no final exclua essa chave ou defina-a como 0.

Recomendações


Apesar da presença de vários métodos para desativar o Windows Defender, não recomendamos o uso de um computador sem programas antivírus. No entanto, você pode encontrar situações em que desativar esse recurso seria uma opção melhor. E se você instalar um programa antivírus de terceiros, não precisará desativar o Defender manualmente, pois durante a instalação, ele será desligado automaticamente.
