Browser Edge baru Microsoft akan mendukung ekstensi Chrome



Microsoft akan membuat peramban Edge berbasis Chromium yang baru kompatibel dengan ekstensi Chrome. Fitur ini secara signifikan akan memperluas fungsionalitas Edge, meskipun di sisi lain itu akan membuat pengguna browser rentan terhadap ekstensi berbahaya, yang juga banyak.

Belum lama berselang, Microsoft mengumumkan niatnya untuk membangun kembali Microsoft Edge agar lebih nyaman dan aman bagi pengguna. Solusi untuk masalah ini agak tidak terduga - dinyatakan bahwa Edge yang diperbarui akan didasarkan pada Chromium. Segera setelah pengumuman ini, banyak yang memiliki pertanyaan tentang dukungan ekstensi Chrome oleh browser Edge. Dan sekarang dia memutuskan, dan secara positif.

Pengumuman ini dibuat oleh Kyle Pflag , Manajer Produk Senior Produk Edge. Dia berbicara tentang semuanya di Reddit , menekankan bahwa maksud tim adalah untuk menambahkan dukungan untuk ekstensi.



Karena banyak pengguna Chrome menggunakan ekstensi, solusi Microsoft cukup logis - karena mereka yang memutuskan untuk beralih ke Edge akan dapat merasakan di lingkungan asli mereka. Tetapi itu juga berarti kemungkinan bahwa pengguna Edge akan berisiko terinfeksi dari ekstensi Chrome yang berbahaya. Lusinan malware baru muncul setiap hari.

Tidak banyak orang yang ingin menyerang Edge - saat ini, pangsa pasar browser semakin kecil. Itu sebabnya jumlah penjahat cyber yang merilis malware di bawah Edgr juga minim. Tetapi pangsa pasar Chrome melebihi 60%, yang berarti ada banyak orang yang bersedia merilis malware untuk itu.

Rentang kemungkinan ekstensi tersebut sangat luas. Ini adalah tampilan iklan di halaman web mana pun, dan pencurian data pribadi, cryptomining menggunakan skrip dan banyak lagi.

Google tidak berhasil mengekang penyerang - ada beberapa yang baru untuk satu ekstensi jarak jauh. Ada seluruh tim "spesialis" yang bekerja secara eksklusif pada ekstensi untuk Chrome, memuat yang baru alih-alih yang telah dihilangkan.

Mungkin Microsoft akan dapat menangani ekstensi jahat dengan lebih baik. Mungkin ini dapat dilakukan jika Microsoft mengunduh ekstensi ke direktori sendiri. Untuk pengembang, langkah ini akan menyebabkan kesulitan tambahan, tetapi mungkin berhasil.

Patut diingat bahwa browser baru dimaksudkan terutama untuk sistem-ARM. Jika peramban itu bukan asli, maka Anda harus menjalankannya dalam mode emulasi, yang memengaruhi kinerja sistem dan peramban itu sendiri. Suatu hari, perwakilan Microsoft mengatakan bahwa browser baru akan diinstal pada semua perangkat Microsoft. Saya ingin tahu apakah browser akan bekerja pada kacamata Microsoft Hololens.
