三星正式停止销售并停止生产Galaxy Note 7



三星无法应对其新旗舰Galaxy Note 7电池的爆炸和着火，在过去的24小时内，三星一直宣布终止智能手机的生产以及已在全球范围内发布的设备的销售。

在公司的官方博客中，要求用户关闭所有大小（主要或“安全”）的设备，然后将其以全价退还给卖方或通过更换另一部智能手机退还给卖方。

“我们正在与有关监管机构合作，调查最近报告的涉及Glaxy Note 7的案件。由于消费者安全仍然是我们的重中之重，因此三星将通知全球所有运营商和零售零售商有关停止销售和交易的信息（这意味着首先要停止Galaxy Note的交易该公司的官方博客说，在调查过程中，“安全”的Galaxy Note 7摇了摇。

显然，韩国人希望这种果断的措施能够阻止人们对他们产品的讨论，但是，三星的“欺负”飞轮已经解开了。 9月中旬，在调查Galaxy Note 7电池爆炸的原因时，美国媒体上出现了文章和笔记，不仅该公司的智能手机，甚至洗衣机也在爆炸。

当应用于洗衣机时，媒体已经不正确，并坦率地“毒化”了三星。爆炸是由于某种物质的即时化学分解和高温气体的形成，伴随着强烈的声音而点燃的东西。实际上，这家韩国制造商的洗衣机由于滚筒的不平衡和强烈的振动而分崩离析，导致机构的损坏和部件的扩展，严重损坏了身体。


三星洗衣机有问题的顶装型号，主要在美国市场销售

为了证明三星的合理性，值得注意的是，在高速洗涤大型或防水物品时，机器机构的破坏是在负载下发生的。然而，问题仍然在于，为什么在洗涤和旋转过程中，机器会“磨损”且无法追踪危险的振动和滚筒内部的负载不平衡。

在发布“安全” Galaxy Note 7并在新闻界发布其用户之后，几乎立即有报道称他们的电池被点燃并爆炸。 “安全”派对的一种设备在飞机上起火，此后，一些航空公司开始要求Galaxy Note 7的所有者在飞行过程中完全关闭该设备。也有其他事件涉及最新的三星旗舰店。

在公众的压力下，为了制止声誉损失，三星采取了彻底的行动，彻底停止生产并召回所有设备。在真正消除导致电池着火的缺陷之后，这家韩国巨头可能会尝试恢复其旗舰产品的销售。但是到那个时候，将失去很多时间，这将削弱三星相对于其主要竞争对手苹果的地位，而且未知的是消费者将如何对易燃智能手机的“重启”做出反应。
