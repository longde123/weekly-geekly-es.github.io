Nouvelle application 3CX pour Android - Réponses aux questions et recommandations

La semaine dernière, nous avons publié la mise à jour 3CX v16 Update 3 et la nouvelle application 3CX (softphone mobile) pour Android . Le softphone est conçu pour fonctionner uniquement avec 3CX v16 Update 3 et supérieur. De nombreux utilisateurs ont des questions supplémentaires sur l'application. Dans cet article, nous y répondrons, ainsi que parler davantage des nouvelles fonctionnalités de l'application.

Fonctionne uniquement avec 3CX v16


Au démarrage de l'application, certains utilisateurs voient un message indiquant que le programme ne fonctionne qu'avec 3CX V16. Ceci, bien sûr, concerne la version du serveur. Vous pouvez résoudre le problème en mettant à jour le serveur PBX vers la dernière version de 3CX v16 . Mais si vous ne pouvez pas passer à la version 16 pour le moment, installez la version précédente de l' application Android . Cela vous permettra de continuer à utiliser 3CX jusqu'à ce que l'administrateur système mette à jour le serveur. Gardez à l'esprit que cette application n'est pas prise en charge ou mise à jour par 3CX, et n'est pas non plus compatible avec Android 10.

Messagerie vocale


Les utilisateurs se plaignent de la façon dont la messagerie vocale est lue dans la nouvelle application. Dans la prochaine version, nous prévoyons de revenir à la méthode de lecture précédente, qui vous permet d'écouter un message vocal sans composer le numéro de messagerie vocale du système.

Accès au carnet d'adresses


Pour le moment, l'application doit accéder à la liste des contacts de l'appareil pour fusionner le carnet d'adresses d'entreprise 3CX, les contacts personnels de l'utilisateur 3CX (numéro de poste) et le carnet d'adresses de l'appareil. Par conséquent, chaque fois que vous accédez au carnet d'adresses de l'application, une demande apparaît pour accéder aux contacts de l'appareil, même si l'utilisateur ne l'a pas autorisé par le passé. Cependant, veuillez noter que l'application ne transfère jamais les contacts de votre appareil vers le système 3CX.

Mais certains utilisateurs ne veulent toujours pas mélanger les contacts personnels du téléphone et les contacts professionnels téléchargés depuis 3CX. Dans la prochaine version, nous bloquerons par défaut l'accès au carnet d'adresses de l'appareil. Si l'utilisateur, au contraire, souhaite combiner des contacts, il leur ouvrira l'accès indépendamment dans les paramètres d'autorisations de l'application 3CX.



Affichage de groupe


L'écran de présence n'affiche plus les groupes d'utilisateurs organisationnels. Cela est fait pour décharger l'interface, car les mêmes utilisateurs peuvent être affichés dans différents groupes (après tout, un utilisateur peut être simultanément dans plusieurs groupes). Nous prévoyons de conserver ce changement.

Recevoir des notifications PUSH


L'option «Quitter - Ignorer PUSH», qui se trouvait dans l'ancienne application, a été supprimée. Au lieu de cela, des moyens plus pratiques de gérer les notifications PUSH dans différents statuts sont apparus.
Vous pouvez spécifier si vous souhaitez ou non recevoir les notifications PUSH dans l'un ou l'autre état. Ce qui suit montre comment procéder pour le statut «Ne pas déranger». Il suffit d'en configurer un pour recevoir PUSH pour chaque état.



L'administrateur PBX peut également configurer la réception de PUSH pour l'utilisateur dans l'interface de gestion 3CX, et des opérations d'édition de groupe sont disponibles.

Rappelez-vous que si l'utilisateur a un horaire de travail fixe, il est préférable de configurer le changement d'état automatique. L'horaire (heures de travail) est défini par l'administrateur du PBX. Vous pouvez utiliser les heures d'ouverture générales de l'organisation ou les heures d'ouverture individuelles de cet utilisateur. En savoir plus à ce sujet dans le didacticiel 3CX .



Mode silencieux


Le mode d'application silencieux peut être activé quel que soit l'état si vous souhaitez recevoir des notifications sur les appels et les messages sans créer trop de bruit. Le mode est activé en appuyant longuement sur l'icône 3CX sur le bureau Android.



Notifications dans Android 10


Dans Android 10, un appel entrant sur un écran déverrouillé apparaît comme une notification. Ceci est implémenté dans le style des autres notifications Android 10. Comparez les notifications sur Android 9 et Android 10.



Certains utilisateurs d'Android 10 signalent qu'un appel est entendu et qu'une notification d'appel ne s'affiche pas. Dans ce cas, il est recommandé de désinstaller et réinstaller l'application. Dans la prochaine version, nous ferons une révision pour afficher de manière fiable les notifications.

Démarrage au démarrage


Dans différents appareils, malheureusement, l'application 3CX se comporte différemment, selon la façon dont Android a été redémarré - manuellement ou se bloque (par exemple, quand il se bloque). Nous avons testé plusieurs appareils et avons découvert que l'application démarre correctement après le redémarrage du téléphone.

Changer ou désactiver les comptes SIP


Dans la nouvelle application, l'interface de gestion (commutation, déconnexion) des comptes SIP a changé. Dans le menu supérieur gauche:

  • Cliquez sur l'icône de votre profil (1)
  • Appuyez et maintenez le compte actuel pour sélectionner l'action: "Désactiver", "Modifier" ou "Supprimer"
  • Cliquez sur un autre compte pour y accéder (2)
  • Cliquez sur "Ajouter un compte" et scannez le code QR (à partir de l'e-mail ou du client Web 3CX pour ajouter un nouveau compte SIP à l'application.



Les notifications PUSH ne viennent pas dans 3CX pour Android


Après avoir mis à jour 3CX vers la version v16 Update 3 et mis à jour l'application pour Android, certains utilisateurs ont cessé de recevoir des notifications d'appel PUSH sur leurs téléphones. Nous avons remarqué ce problème sur les installations 3CX, qui utilisent leur propre compte pour le compte PUSH.



Dans ce cas, il est recommandé de basculer vers le compte 3CX intégré. Pour ce faire, cliquez simplement sur la ligne "Compte utilisateur", puis supprimez vos paramètres PUSH de l'interface 3CX, cliquez sur OK et redémarrez le serveur.



Après cela, vérifiez les paramètres des notifications PUSH dans l'interface.



Vous devez maintenant reconfigurer les applications 3CX pour les utilisateurs qui ont du mal à obtenir PUSH.

Nous espérons donc que ces explications et recommandations vous seront utiles ainsi qu'à vos utilisateurs!
