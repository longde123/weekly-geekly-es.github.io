Selamat datang di lokakarya langsung API Jam

Softline dan Apigee (Google Cloud) mengundang pengembang dan arsitek API ke bengkel langsung API Jam di Rusia, bengkel besar tentang bekerja dengan platform Apigee Edge.



Kami akan mengajari Anda cara memberikan akses eksternal ke backend dengan cepat dan aman menggunakan API. Akan ada sedikit teori, tetapi sebagian besar latihan, latihan, dan latihan lagi. Peserta akan mendapatkan pengalaman nyata menggunakan platform Apigee Edge. Di bawah bimbingan seorang pelatih, akan ada 5 lab yang akan memungkinkan Anda untuk merancang, mengamankan dan menganalisis API favorit Anda.

3 Oktober 09: 30-16: 00
Google Moscow Office


Program:

Apigee penting

• Jelajahi UI Apigee Edge
• Kelola organisasi, lingkungan, pengguna platform, alur, dan kebijakan
• Memahami hubungan antara pengembang, aplikasi, proksi, dan API
• Pelajari tentang produk API dan portal pengembang

Latihan Lab Praktek

• Lab 1: Desain dan buat proxy API
• Lab 2: Amankan API Anda dengan kunci API
• Lab 3: Membuat dan menerbitkan produk API
• Lab 4: Bantu pengembang aplikasi mengonsumsi produk API Anda
• Lab 5: Melambatkan lalu lintas API

Acara ini diadakan dalam bahasa Inggris.


Untuk berpartisipasi Anda perlu:

Diperlukan laptop: Google Chrome atau Firefox browser lebih disukai.
Akun Apigee. Anda harus membuat akun terlebih dahulu sesuai dengan instruksi .

Acara ini gratis, namun, kami hanya memiliki 20 kursi, jadi kami tidak akan dapat menerima semua orang. Untuk mendaftar, Anda harus mengisi formulir , prioritas akan diberikan kepada peserta dengan kasus nyata.
