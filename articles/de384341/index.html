Cloudready erweckt alte PCs mit Chromium OS zum Leben



Das Unternehmen Neverware hat ein Softwarepaket entwickelt, mit dem Sie fast jeden alten PC in ein Chromebook oder eine Chromebox verwandeln können. Für die Arbeit müssen Sie lediglich ein USB-Flash-Laufwerk an den Anschluss anschließen und einige Minuten auf die Installation der Software warten.

Die Software heißt Cloudready, ein Cloud-basiertes Betriebssystem, das auf Chromium OS basiert. Cloud OS wurde mit Google entwickelt. Es ist überflüssig zu sagen, dass die Belastung der lokalen Hardware nicht zu hoch ist, da das Betriebssystem trübe ist. Infolgedessen kann auch sehr schwache Hardware als funktionierender PC verwendet werden. Die Hauptbedingung ist die Verfügbarkeit des Netzwerks, damit Cloudready funktioniert.

Laut Neverwhere verwandelt Cloudready fast jeden PC, der nicht älter als 8 Jahre ist, in einen neuen. Natürlich bleibt das Erscheinungsbild eines solchen Computers gleich, aber die Software funktioniert sehr schnell. Neverwhere hat sein Betriebssystem hauptsächlich für Schulen entwickelt, obwohl es eine Version für reguläre Benutzer gibt . Das Betriebssystem für Schulen hat zwar einen Vorteil: Diese Version wird vom Unternehmen zusätzlich technisch unterstützt, und die gesamte Software wird automatisch aktualisiert.

Der Support ist nicht kostenlos - dieser Service kostet Schulen 25 USD pro Auto und Jahr. Oder Sie können ein Vierjahresabonnement für 59 US-Dollar abschließen. Mehr als 100 Schulen haben sich bereits für das Servicepaket von Neverware angemeldet. So hatten Bildungseinrichtungen die Möglichkeit, eine veraltete Maschinenflotte als praktisch neue arbeitsbereite Systeme einzusetzen.
