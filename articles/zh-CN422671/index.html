5G可以代替电缆-但并非到处都可以

花旗银行的分析师建议（ PDF ）5G可能将有线互联网提供商推向市场。 让我们谈谈这种“威胁”的真实性：新一代网络将能够替换电缆，还有很长的路要走。


/ Flickr / kahunapulej / CC

5G可以替代电缆的地方


在大城市

5G很有可能在大城市取代有线互联网。 该技术提供了很高的数据传输速率-高达10,000 Mbit / s。 这比有线连接快十倍，但是您无需解决与在密集的城市基础设施中铺设电缆相关的问题。

当一家英国公司将光缆铺设在下水道系统中时，情况就为人所知。 它比经典方法（80％）更快。 很明显，5G完全消除了“接地”布线的需要。

在美国，与有线互联网的战争已经开始。 正如金融机构Cowen的分析师在其季度报告（ PDF ）中指出的那样，Verizon和T-Mobile运营商的5G服务是美国有线互联网的最大威胁。

T-Mobile宣布，到2024年，他们计划“夺回”来自有线电视和互联网行业的大约1000万用户（这是其中很大一部分）。 Verizon计划今年向休斯顿，印第安纳波利斯，洛杉矶和萨克拉曼多的居民提供5G。

该公司将“ 诱使 ”用户免费使用YouTube TV和Apple TV。 Verizon首席执行官汉斯·韦斯特伯格（Hans Vestberg）在接受CNBC采访时表示，一切将很快实现无线化，而5G网络将把电缆推向市场。

请注意，在大城市中实施5G仍必须解决许多困难，尤其是覆盖问题 。 由于5G 使用 28 GHz及更高​​频率的高频信号，因此该信号不会长距离传输，并且无法以壁的形式穿过障碍物（这是高数据传输速率的代价）。 但是他们已经在努力解决该问题，例如，正在创建特殊的分布式天线系统 （DAS）。

在数据中心

今年6月，佐治亚理工学院的研究人员推出了所谓的异相发射机。 系统开发人员将天线模块和无线电组件组合在一个微处理器芯片中。

他们使用了具有双电源的闭环天线，这使得在正常操作和峰值负载期间都可以提高解决方案的能效。 这也使使用更复杂的调制协议成为可能。

据研究人员称，该解决方案将能够将数据传输到数据中心的速度提高6到10倍。 5G可以在毫米波上工作。 公平地说，我们注意到需要提高安全性的数据中心仍将使用电缆。


/ Flickr / christoph scholz / cc

电缆将长时间放置的地方


在郊区

由于5G使用高频这一事实，因此信号无法长距离传播。 在这种情况下，需要大量的塔。 由此产生了另一个问题-这些塔仍应与光纤电缆连接。 但是，即使在发达国家，仍然有一些地方没有“光学”。

例如，英格兰只有3％的家庭连接到光纤网络。 在大多数情况下，英国人使用几十年前铺设的过时的铜缆上网。 根据2016年的数据，在美国，偏远地区和郊区的2000万居民通常无法访问互联网 。

在成本方面，将电缆延伸到这些地方较为便宜。 这正是英格兰现在正在做的事情-控制主要互联网网络的BT供应商子公司将在2020年之前将300万座建筑物连接到光纤网络（FTTP）。 如果放了一座塔，那么用户覆盖面将很小。 因此，提供商的服务成本将增加。

因此，无需谈论在郊区和乡村大规模引入5G的问题。 电缆将在那里保留很长时间。

请注意，在某些情况下，似乎可以在郊区使用5G。 例如，在智能农场中 ，因为像LTE-U这样的技术在空旷地区比Wi-Fi更好地工作。 但同样，一切都取决于基础架构的成本。

洲际联系

连接整个大洲的海底电缆也将放置很长时间。 实际上，它们提供了地球上整个IT基础架构的功能。 现在，世界被三百个电缆系统所环绕，并且每年都有越来越多的电缆系统。

从理论上讲，世界上正在开发一些举措，这些举措可以将Internet传送到地球的偏远角落以及从大陆到大陆。 例如，前段时间，谷歌从事了 SkyBender 项目 ，该项目开发了带有太阳能电池板的无人机。 飞机向下方的所有设备广播了5G信号。 现在，出于这些目的，公司使用气球。 但是该项目离全球实施还很遥远。

在许多5G用户案例中，有可能抓住主动，例如，下一代网络将成为连接大城市中IoT设备的主要技术。 但是，电缆不会完全消失（至少在不久的将来）。 海底电缆的历史已经持续了167年 ，并且很长一段时间以来，它们将成为全球IT基础架构的基础。
