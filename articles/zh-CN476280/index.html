从荆棘到DOS：改变世界的四张软盘

不可能会有至少一个从未曾启动过传奇的MS-DOS操作系统的阅读器。 也许你们中的一个开始使用它开发个人计算机：Spectrum已经退休，俄罗斯的Amiga和Atari汽车仍然是“专业玩具”，Apple技术尚未受到如此疯狂的欢迎，IBM PC已经牢牢地占领了市场。

今天，我们想继续讨论这个话题，并讨论俄罗斯的MS-DOS：记住OS的历史，看看90年代的用户如何工作并在计算机上玩得开心。

一点历史


20世纪下半叶，计算机技术的发展是非常非线性的，甚至是混乱的。 真正没有统一世界各地制造商的统一标准，每个公司都为他们“如何呼吸”开发了计算机和软件。 因此，DOS的故事并不是像许多人所想的那样在微软开始的，而是早一点开始的。

1974年，CP / M操作系统亮起，它代表“控制程序/监视器”或“微型计算机的控制程序”。 它是由才华横溢的西雅图程序员，Digital Research Inc.的创建者Gary Kildall用自己的语言PL / M（微型计算机编程语言）编写的。


/ Gary Kildall，daviddilworth.com

最初，CP / M是为8位计算机（尤其是Intel 8080处理器）设计的，并且是第一个可以与磁盘驱动器进行交互的微型计算机操作系统。 几年后，CP / M适应于与Zilog Z80处理器（来自苏联和俄罗斯ZX Spectrum克隆的Sinclair爵士问好）和16位的Motorola 68000以及Intel 8088和Intel 8086处理器一起使用。

操作系统的正式开发以1983年的3.1版发布告终，目前CP / M及其所有派生产品都是开源软件，任何人都可以根据自己的利益使用，分发和修改。
但是，在并行世界中的某个地方，Rick Sanchez没有听到有关Microsoft的任何消息，而是在运行CP / M的计算机上学习编程的。
在1980年，当IBM为IBM PC选择“主要”操作系统时，主要的利益是专门针对Kildall的开发。 但由于当时Digital Research尚没有16位版本的CP / M，以及其母亲的女友比尔·盖茨（他的母亲的确是联合之路国际公司执行委员会主席以及两名颇有影响力的IBM成员）的儿子，因此当时并没有16位版本的事实，这笔交易并未成立默默无闻的微软，获得了“本世纪合同”。


比尔·盖茨（3dnews.ru）


/与妈妈，微软的未来创造者besage.ru

具有讽刺意味的是，当时该公司还没有针对Intel 8086的自己的解决方案，因此盖茨的同伙从Seattle Computer Products购买了CP / M克隆产品之一，用于16位处理器的86-DOS OS。 该操作系统以熟悉的名称MS-DOS出现在市场上。

2014年，Mountain View的计算机历史博物馆在获得Microsoft许可的情况下发布了MS-DOS的1.1和2.0版本的源代码，并在2018年以免费的MIT许可证发布了OS版本1.25和2.0的源代码。

您可以在这篇出色的文章中更详细地研究MS-DOS的世界历史，并且我们主要对当时俄罗斯正在发生的事情以及DOS在广大家庭用户的生活中将扮演的角色感兴趣。

我们有什么


正如我们在前几篇文章中已经写过的那样，鉴于苏联的许多经济和政治特点，苏联正以自己的特殊方式向计算机化方向发展。 事实证明，直到2000年代初，几乎每个家庭，甚至距离技术都不远的地方，都出现了个人计算机。 如果您的青年或童年在90年代过去了，您可能还记得，如果第286名男子没有自己的“汽车”，也会引起什么喜悦。 那些幸运地拥有一台现代化PC的人组织了一些小型兴趣俱乐部：交换软盘，讨论计算机世界新闻以及与软件销售商共享联系。 但这是稍后。 要查看苏联最早的家用IBM PC兼容计算机，您应该回到80年代。

保加利亚Pravets是苏联公民相对可以使用的最早的IBM兼容计算机之一。 在一个“品牌”下，生产了诸如Apple II和Orion Atmos之类的机器的克隆。 特别感兴趣的是Pravets 16模型，它是IBM PC / XT的克隆。 根据测试，由于硬盘速度慢和其他一些设计缺陷，Pravets 16的性能大约是原始IBM PC的60％。


/ Pravets 16，pokazuha.ru

尽管如此，还是有了一个开始，IBM逐渐开始接管苏联。 继保加利亚的克隆之后，第一批基于Intel 80286的原始计算机泄漏到该国，作者可以无条件声明在1988年的莫斯科Leninsky Prospekt（电子商店）上已经展示了真正的286号计算机。 它花费了相当多的空间钱（就更坚硬的货币而言-在莫斯科维奇和伏尔加河之间），站在商店的中央，在一个小的讲台上，并且被厚厚的玻璃杯的观众（在联盟中很少有买主的观众）封闭住。

观众研究了这具器具，懒洋洋地叹了口气，但听不到。 在那些年代，类似级别的计算机非常昂贵，并且主要是由大型组织和研究机构根据工作需要购买的。 随着时间的流逝，这个问题得以解决：私营企业的迅猛发展和西方逐渐渗透到该国的工作已完成。 到1992-1993年，任何人都可以以更高的忠诚度，但仍然是很高的价格自由地购买相同的286号。 在某些情况下，人们可以“拿走”在工作场所注销的旧计算机，用于家庭和游戏。

第一批计算机是在西方购买的，并以我们的“原样”出售：实际上，市场上首次没有零配件或改进的组件。 不久之后，大公司（戴尔，惠普）的代表处出现在销售“品牌”计算机的俄罗斯领土上，以及大量销售自组装计算机的当地办事处。

每年，个人计算机的可用性（以及坦率地说，简单性和易用性）都在增加。 配件以合理的价格出现在国内市场，到90年代中期，兼容IBM的计算机已不再是稀罕物，只有富人才能使用。

游戏，互联网，多媒体


在4K，多核处理器和逼真的计算机图形时代，第一批IBM的多媒体功能显得荒谬而幼稚。 例如，这里平均第386位的特征：

  • CPU时钟速度：40 MHz。
  • RAM：1至3 MB。
  • 硬盘：〜300 MB。

有趣的是，从那时起，计算机（以及程序的胃口）的功能增长了多少，不是吗？

但是，PC用户一直都想听音乐，看图片和演示（由于DOS中缺少视频播放器，不需要用户干预的所有动画内容都是通过编程方式创建的），看书并与远方的人交流。

  • 音乐：听和写；
  • 游戏：视频模式，热门游戏及其系统要求；
  • 互联网：连接选项和功能。

先生们的软件集


档案管理员


可以认为，我国最流行的文件管理器是Norton Commander，即它的版本从2.0到4.0，最后一个发布于1992年。 后来，它被功能更强大的Volkov Commander，PIE Commander和DOS Navigator取代。 所有列出的指挥官在逻辑和外观上都具有相似的界面，他们使用F1 ... F10键进行基本操作（帮助，设置，打开，编辑和其他文件操作，退出）。 支持的鼠标工作。 在3.0版中，第三方开发人员通过自己的插件，补丁程序和查看器积极地改进了管理器。 造成Norton Commander死亡的两个因素是：RAM占用了过多的内存，并且出现了大量错误。

您可以在此处阅读有关Norton Commander，其功能和主要竞争对手的更多信息 。

处理磁盘


AnaDisk是一种用于处理软盘的多功能实用程序。 允许读取，复制，格式化和恢复损坏的磁盘。

TeleDisk是一个用于复制和记录各种容量的3.5英寸和5.25英寸磁盘映像的程序。 这可能与今天有关，因为ZX Spectrum的许多图像都以.TD0文件格式出现。

CANDLE-允许您创建磁盘目录。 这不是最受欢迎的程序，但是如果您选择的磁盘有数十甚至数百个，它可能会派上用场。

各种编辑器和查看器的流行主要是由于这些程序的存在与否：简单来说，人们就是在使用现有的东西。 那些年里的道德问题（盗版，版权）仅严重困扰着版权所有者和官方软件发行者。 最终用户需要程序才能运行，但程序的来源并不重要。 就像他们说的那样， 比尔·盖茨（Bill Gates）很远，那里有软盘 。

乐曲


-莎拉，他也唱歌！

除了游戏中的音乐（我们将在后面讨论）之外，在MS-DOS中还可以播放MIDI和WAV文件。 最后，一切都取决于“铁”配置，即声卡。

通常，要在MS-DOS中输出至少某种声音，您可以使用内置计算机扬声器，PC扬声器（音频质量最低的预算最多的选件）， SoundBlaster声卡（非常昂贵，尽管是高质量的选件）和Covox （妥协选项，它可以产生良好的声音，但不如SoundBlaster常见，并且不受所有游戏的支持）。


/“理想” Covox示意图，phantom.sannata.org

与图形软件一样，对于DOS，有一个巨大的软件数据库可用于播放和创作音乐。 特别是，可以将MIDI键盘连接到计算机并使用特殊程序记录样本。

游戏


-如果计算机无法启动Doom，则这不是计算机，而是呵护。
MS-DOS应用程序软件几乎失去了它的实用性：一切都一样，在运行Windows，Linux和MacOS的现代计算机上，只有好很多倍且更方便。 也许，只有旧的DOS上的游戏仍然非常生动有趣。

可以无限枚举MS-DOS在游戏中的可能性：第一人称射击游戏类型的先驱（Wolfenstein 3D，Doom等），在SCUMM上进行了大量的任务，而不仅仅是出色的Rayman ...

您可以无限次列出这些游戏。 其中有好有坏。 当今的个人计算机标准使游戏从弱势中脱颖而出，并事实上推动了计算机行业的发展。

MS-DOS成为了需要大量生产性工作马力和刺激性游戏马的大量PC用户的统一点。 正如我们在本文开头所写的那样，Commodore，Amiga和Atari都无法征服家用计算机市场并保持领先地位，而IBM与微软合作取得了成功。 迄今为止，我们正在收获这种合作的成果。

让我们说几句有关复古游戏爱好者的话：现在，习惯上来说，是要了解那些重视优质游戏的人，而不论其创造的年份和目标平台如何。 出于客观原因，在MS-DOS下发行的很大一部分游戏无法在Windows上运行。 特别是对于那些想怀旧的人，有一个DOSBox仿真器 。 目前，它是一种免费软件，已从Sony PSP移植到了几乎所有平台上，并已移植到Windows 10和MacOSX。除了启动游戏外，DOSBox还可以正确运行许多应用程序：数据库，图形和音乐编辑器等。

网站old-games.ru展示了许多DOSBox程序集，包括预装Windows 3.1的程序集，该程序已完全可以使用。

互联网


在上一篇文章中 ，我们已经讨论了从各种设备访问Internet的方法，从以前的NIIshny操纵杆到与ZX Spectrum兼容的机器。

由于90年代中期IBM兼容PC的计算机在俄罗斯乃至全世界都获得了极大的普及，并且其容量逐年增加，因此向PC用户开放了万维网。

如果您打算在有条件的96年上线，则需要购买网卡，调制解调器，并可以使用电话线。 当然，购买后，需要花费一些时间来配置和初始化系统中的新硬件，以及将网络访问权提供给选定的提供商。

蛛网


Arachne浏览器于1996年发布，尽管有许多假设，但仍可以用于在装有MS-DOS的计算机上访问Internet。 对HTML和CSS的支持非常有限。 通常，浏览器没有经受时间的考验：它的运行速度非常慢，但是其中没有UTF-8支持，这就是为什么大多数俄语网站显示不正确的原因。


/ Arachne Browser，Dmitry Moiseev的视频截图

山猫


文本浏览器 ，是DOS最早的浏览器之一。 它于1992年发行。 它不支持JavaScript，某些标签和样式，但运行速度非常快。 使用光标键和PgUp / PgDn在浏览器中进行管理。


/ Lynx Browser，Dmitry Moiseev的视频截图

除了专门浏览MS-DOS中的网页外，还可以连接以与FTP服务器交换文件，通过电子邮件客户端读取和发送邮件，以及将计算机集成到本地网络中。

如果您对使用MS-DOS功能的现代实验感兴趣，建议您关注Dmitry Moiseev的又名Newbilius YouTube频道，例如， 在MS-DOS上观看有关Internet的视频 。
通常，在这些浏览器中，现代访问Internet的功能的特征已经非常明显。

窗户


彼得一世在欧洲开了一个窗口，比尔·盖茨开了一个窗口，一个窗口又一个窗口。
在版本6.22之前，MS-DOS是一个独立的操作系统。 7.0版于1995年发布，已经作为新Windows 95的一部分发布（Microsoft首次在该版本中使用台式机，该台式机成为该公司产品的经典之作），并且在将来，DOS本身在用户体验中的作用越来越低。 Windows Me于2000年发布了最新的Microsoft操作系统，其中包括DOS更新。

早期版本的Microsoft Windows 3.1尚未在用户中流行，因为它们包含许多错误和缺点。 实际上，这些是公司迈向窗口界面的第一步。 经过认真重新设计的1993年的3.1版（以及一年后发布的工作组3.11版，包括对局域网的支持）经过认真地重新设计，成为了第一个“严肃的”大规模Windows。
