WASI: Uma API para WebAssembly não Web


o relatório será lido em inglês por teleconferência

Para ajudar a levar o WebAssembly para fora do navegador, estamos criando o WASI, a WebAssembly System Interface. É uma API padrão para aplicativos WebAssembly usar fora da Web, que estende a filosofia de portabilidade e sandbox do WebAssembly para o espaço da API.
