5 extensions de navigateur pour travailler avec du texte anglais



Bonjour à tous! Je continue de partager mon expérience professionnelle dans le domaine du travail avec des textes anglais. Aujourd'hui, nous parlerons des extensions de navigateur pouvant être utilisées pour faciliter la création de documents de haute qualité et apprendre à mieux écrire en anglais. C'est parti!

Remarque : le texte contient des extensions pour Google Chrome et Mozilla Firefox, car j'utilise uniquement ces navigateurs. Tous les produits mentionnés n'ont pas de version pour les deux navigateurs.

Linguix


Ce service est quelque peu similaire à Grammarly - il trouve également des fautes de frappe et des erreurs grammaticales dans les textes, mais il coûte presque la moitié du prix. Il existe également des différences - par exemple, une base de données de modèles de contenu qui peuvent être utilisés pour rédiger plus rapidement des textes de différentes catégories (par exemple, des communiqués de presse ou des actualités d'entreprise).



Des fonctionnalités que j'aime - le module de calcul des statistiques. Le système analyse la «lisibilité» du texte et indique sa complexité. Utile pour lutter contre la tentation de rendre le texte trop volumineux.



Une autre chose pratique est le mode d'édition de pâte dans la version Web, qui fonctionne comme un chat secret dans le messager. Après avoir écrit et édité le texte, il est complètement supprimé.

Écrivain


Une application Masthead pour tous ceux qui travaillent avec des textes anglais à l'aide d'un navigateur. Cette extension vous permet d'ouvrir dans Chrome un éditeur extrêmement pratique, hautement personnalisable et sans distraction. Ça a l'air très cool:



Linguix fonctionne dans l'éditeur - vous pouvez donc immédiatement rechercher les erreurs

Des décisions réussies - l'extension enregistre automatiquement le texte de temps en temps. Vous pouvez définir des objectifs, obtenir facilement des statistiques sur le nombre de caractères dans le texte, etc. Malgré le fait que l'application dispose d'une page distincte dans le magasin d'extensions Chrome, en fait, elle redirige vers la version Web du service, et vous pouvez l'utiliser à partir de n'importe quel navigateur.

Version du navigateur Writer

Dictionnaire Google


Un dictionnaire de Google vous permet de traduire un mot inconnu - il peut être utile lorsque vous travaillez avec des documents en anglais pour préparer du texte. Avec cette extension, vous pouvez facilement apprendre la signification de n'importe quel mot que vous rencontrez sur une page Web. Il vous suffit de mettre un mot en surbrillance et une fenêtre contextuelle avec une traduction apparaîtra.



Une chose pratique - le programme peut prononcer des mots à haute voix, afin que vous puissiez tirer la prononciation. Il s'avère que le dictionnaire Google est utile non seulement pour un travail direct sur le texte, mais aussi comme outil d'apprentissage de la langue.

Version Chrome

Lecteur sombre


Pour être honnête, beaucoup d'entre nous sont constamment assis devant l'ordinateur jusqu'à la nuit, y compris dans l'obscurité totale. Certains, par exemple moi, peuvent même travailler dans ce format, y compris sur des textes.

Inutile de dire que la plupart des éditeurs de texte Web sont des pages blanches lumineuses avec des lettres noires. Dans l'obscurité, c'est une fatigue oculaire extrêmement grave, donc j'utilise l'extension DarkReader. Il assombrit l'arrière-plan des pages Web, il est donc beaucoup plus agréable de travailler avec elles le soir. Restez concentré


La procrastination est l'un des principaux ennemis de tous ceux qui ont essayé au moins une fois de travailler sur des textes. Écrire un texte particulièrement long sans être distrait par les réseaux sociaux, le courrier et les actualités est une tâche extrêmement difficile. L'extension StayFocusd m'aide à le résoudre.



Avec cela, vous pouvez vous empêcher de quitter un site particulier pendant un certain temps. Par conséquent, si vous devez écrire un article de blog, jusqu'à ce que vous écriviez, vous n'allez pas sur Facebook. Avec le début de l'utilisation de cette extension, ma productivité a considérablement augmenté. Je le recommande.
