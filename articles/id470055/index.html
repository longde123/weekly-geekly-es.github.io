Memperkenalkan Level Solusi Manajemen Paket NuGet di Visual Studio untuk Mac

Visual Studio 2019 untuk Mac versi 8.3 hadir dengan banyak fitur baru yang dirangkum dalam posting blog ini . Kemampuan untuk mengelola paket pada tingkat solusi adalah salah satu fitur yang sebagian besar dari Anda kekurangan dalam Visual Studio untuk Mac. Manajer paket tingkat solusi NuGet baru adalah salah satu fitur baru yang keren di Visual Studio 2019 untuk Mac versi 8.3.

Kami telah melakukan peningkatan untuk membantu Anda menemukan paket dengan lebih mudah. Ini termasuk peningkatan fungsionalitas saat mencari paket baru, memahami paket mana yang sudah diinstal di proyek Anda, dan mencari paket dengan pembaruan yang tersedia. Dalam posting ini, kami akan fokus pada fungsionalitas manajemen paket untuk solusi. Namun, sebagian besar fungsi, termasuk menginstal, memperbarui dan melihat paket yang diinstal, memiliki fungsionalitas baru yang serupa di tingkat proyek.



Untuk memulai Manajer Paket NuGet untuk Solusi, Anda dapat pergi ke menu konteks Solusi dan pilih "Kelola Paket NuGet ...":



Menambahkan Paket Baru


Ketika Anda mencari dan mencoba untuk menambahkan paket baru, Anda dapat memilih proyek di mana Anda ingin menginstal paket.

Kapan saja, Anda dapat pergi ke tab " Terpasang " dan melihat daftar semua paket yang diinstal dalam solusi Anda, yang memungkinkan Anda menghapusnya untuk memperbarui.



Pembaruan Paket


Tab Pembaruan menampilkan semua paket dalam solusi yang pembaruannya tersedia (atau dalam proyek jika Anda menjalankan perintah di tingkat proyek).



Paket konsolidasi


Seringkali keputusan besar ditemukan dalam situasi di mana proyek yang berbeda merujuk ke versi paket yang berbeda. Untuk menggabungkan versi-versi ini ke dalam satu versi paket, yang mungkin ingin Anda gunakan dalam solusi, Anda bisa pergi ke tab konsolidasi manajer paket NuGet yang disebut pada node solusi, pilih versi paket yang ingin Anda gunakan di semua proyek solusi, dan pilih konsolidasi paket (paket konsolidasi) :



Coba sekarang!


Untuk mencoba fitur-fitur baru NuGet ini, unduh Visual Studio 2019 untuk Mac versi 8.3 atau tingkatkan ke versi terbaru menggunakan saluran stabil jika Anda sudah memasang VS untuk Mac.

Jika Anda menemukan masalah dalam rilis v8.3, gunakan menu Bantuan> Laporkan Masalah . Anda juga dapat membuat saran untuk peningkatan lebih lanjut menggunakan menu Menyediakan Saran.



Nah, ikuti kami di Twitter di @VisualStudioMac untuk mengikuti berita terbaru Studio Visual untuk Mac dan beri tahu kami tentang pengalaman Anda. Looking forward untuk melihat Anda!
