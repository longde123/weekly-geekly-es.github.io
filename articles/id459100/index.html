Registry Paket GitHub akan mendukung paket Swift

Pada 10 Mei, kami mengumumkan beta terbatas dari GitHub Package Registry, layanan manajemen paket yang memudahkan untuk mempublikasikan paket publik atau pribadi di sebelah kode sumber Anda. Saat ini mendukung alat manajemen paket yang sudah dikenal: JavaScript (npm), Java (Maven), Ruby (RubyGems), .NET (NuGet), dan gambar Docker, dengan lebih banyak lagi yang akan datang.

Hari ini kami dengan gembira mengumumkan bahwa kami akan menambahkan dukungan untuk paket Swift ke GitHub Package Registry. Paket Swift memudahkan Anda untuk berbagi perpustakaan dan kode sumber di seluruh proyek Anda dan dengan komunitas Swift.



Tersedia di GitHub, Swift Package Manager adalah alat lintas-platform tunggal untuk membangun, menjalankan, menguji, dan mengemas kode Swift Anda. Konfigurasi paket ditulis dalam Swift, memudahkan untuk mengkonfigurasi target, mendeklarasikan produk, dan mengelola dependensi paket. Bersama-sama, Manajer Paket Swift dan Registri Paket GitHub akan membuatnya lebih mudah bagi Anda untuk menerbitkan dan mengelola paket-paket Swift Anda.

Sangat penting bagi pengembang seluler untuk memiliki alat terbaik agar lebih produktif. Dengan pertumbuhan ekosistem Swift, kami senang dapat bekerja sama dengan tim di Apple untuk membantu menciptakan alur kerja baru bagi pengembang Swift.

Sejak diluncurkan, kami kagum melihat kegembiraan Anda untuk memulai dengan GitHub Package Registry. Selama periode beta ini, kami berkomitmen untuk belajar dari komunitas dan ekosistem tentang bagaimana memenuhi kebutuhan Anda dan apa yang dapat kami lakukan untuk membuatnya lebih baik. Jika Anda belum melakukannya, Anda dapat mendaftar untuk beta terbatas sekarang .
