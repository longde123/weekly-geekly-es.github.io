Penyimpanan Multipath LUN untuk Debian GNU / Linux

Agak rumit:

Pada tahap awal instalasi Debian GNU / Linux, kita mungkin mengalami masalah ketidakmampuan untuk mendeteksi sistem firmware ql2400_fw.bin. Itu dipecahkan secara sederhana:
Pada sistem Linux yang berfungsi, unduh paket firmware-qlogic, buka paketnya, tulis ke gambar dan pasang melalui ILO (tindakan dilakukan pada server HP Proliant). Itu terlihat seperti ini: 

#apt-get --download-only install firmware-qlogic #cp /var/cache/apt/archives/firmware-qlogic_* . #ar x firmware-qlogic* #tar cJpfv data.tar.xz #dd if=/dev/zero of=qlfw.raw bs=1M count=50 #mkdir fw #mount -o loop qlfw.raw fw #cp -r lib/firmware/* fw #umount fw

Kami menghubungkan qlfw.raw melalui menu Virtual Device-> Image File Removable Media. Jika installer masih tidak dapat menemukan firmware, ini dapat dilakukan secara manual dengan memasang gambar di direktori / lib / firmware dan me-reboot modul qla2xxx. Beralih ke konsol teks (tindakan berikut dilakukan di ILO. Keyboard-> menu CTRL-ALT-Fn-> CTRL-ALT-F2): 

 #fdisk -l| grep 50 Disk /dev/sdr: 50 MiB, 52428800 bytes, 102400 sectors #mkdir /lib/firmware #mount /dev/sdr /lib/firmware #rmmod qla2xxx #modprobe qla2xxx

Setelah itu, kita kembali ke installer (Menu Keyboard-> CTRL-ALT-Fn-> CTRL-ALT-F5), dan instal ulang sistem dalam mode normal.
