Ekstensi untuk Visual Studio Code yang membantu Anda bekerja dengan Git

Visual Studio Code memiliki git secara default, tetapi fungsi bawaan tidak selalu cukup untuk pekerjaan yang mudah dan cepat. Kerja efektif dengan git juga membutuhkan visualisasi proses.

Di bawah ini adalah ekstensi yang akan membuat bekerja dengan git nyaman dan menyenangkan:

Gitlens


GitLens memperluas kemampuan git dari Visual Studio Code bawaan. Ekstensi ini memiliki sejumlah besar fungsi dan pengaturan yang fleksibel. Dengan satu klik Anda dapat memperoleh informasi tentang penulis, anotasi, dan lainnya baris demi baris.



Riwayat git


Git History membantu Anda melihat perubahan riwayat. Ini memiliki antarmuka yang cukup ramah pengguna dan sangat cepat.



Grafik git


Ekstensi membantu meramban cabang repositori dan juga melakukan pencarian riwayat. Untuk pengembang yang sering menerima permintaan tarik dan melakukan tinjauan kode, ekstensi ini akan menjadi sangat diperlukan.



Indikator git


Indikator sederhana aktivitas git di panel bawah Visual Studio Code.
