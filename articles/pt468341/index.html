Shell de nuvem do Azure no terminal do Windows

Agora você pode se conectar ao Azure Cloud Shell pelo Terminal do Windows!



Para fazer isso, adicionamos um novo perfil padrão - Azure Cloud Shell, que permite acessar diretórios / clientes do Azure por meio do aplicativo Terminal do Windows.

Se você já possui o Windows Terminal instalado


Suas configurações do Terminal do Windows não serão atualizadas automaticamente para o novo perfil (como o arquivo não é atualizado toda vez que você abre o Terminal do Windows), então veja como você pode adicioná-lo manualmente:

  1. Iniciar o Windows Terminal
  2. Abra Configurações (usando a lista suspensa)
  3. Adicione este perfil à sua lista de perfis (perfis de itens): 

{"acrylicOpacity" : 0.6, "closeOnExit" : false, "colorScheme" : "Vintage", "commandline" : "Azure", "connectionType" : "{d9fcfdfa-a479-412c-83b7-c5640e61cd62}", "cursorColor" : "#FFFFFF", "cursorShape" : "bar", "fontFace" : "Consolas", "fontSize" : 10, "guid" : "{b453ae62-4e3d-5e58-b989-0a998ec441b8}", "historySize" : 9001, "icon" : "ms-appx:///ProfileIcons/{b453ae62-4e3d-5e58-b989-0a998ec441b8}.png", "name" : "Azure Cloud Shell", "padding" : "0, 0, 0, 0", "snapOnInput" : true, "startingDirectory" : "%USERPROFILE%", "useAcrylic" : true}

Depois de fazer isso, a guia Shell do Azure Cloud estará disponível para você.

Configuração básica


  1. Clique na guia "Azure Cloud Shell".
  2. Você será solicitado a ir para "microsoft.com/devicelogin" e inserir o código exibido.
  3. Depois de inserir o código no navegador, você precisará fazer login na sua conta. Além disso, verifique se você está conectado com uma conta que tenha um diretório / cliente do Azure ativo.
  4. Retorne ao terminal. Após alguns segundos, você verá a mensagem "Autenticado".
  5. Somente em alguns casos : se sua conta tiver vários clientes, você será solicitado a selecionar um deles. Basta digitar o número do cliente ao qual você deseja se conectar.
  6. Você será solicitado a salvar essas configurações de conexão. Salvar os parâmetros de conexão permitirá que você entre no sistema no futuro sem seguir as etapas de 1 a 5.
  7. Em seguida, o aplicativo começará a estabelecer uma conexão com o Cloud Shell (isso pode levar algum tempo, apenas seja paciente).
  8. Agora você está conectado ao Azure Cloud Shell!

É assim que a saída de login completa se parece (pressionei '0' para o número do cliente e 'y' para escolher salvar os parâmetros de conexão):



Agora que salvei os parâmetros de conexão, eis como será o processo de login no futuro (pressione '0' para acessar os parâmetros de conexão salvos):



Você também pode fazer login usando outra conta / cliente pressionando 'n' ou excluir conexões salvas pressionando 'r'. Esses parâmetros serão salvos durante as sessões; portanto, mesmo que você inicie o terminal após alguns dias, ainda poderá fazer login com os parâmetros de conexão salvos sem precisar abrir o navegador novamente.

Espero que você aproveite a capacidade de acessar seus recursos do Azure através do Terminal do Windows. Além disso, como sempre, relate quaisquer bugs / problemas que você notou no nosso repositório Github .
