铜对万维网的贡献

当谈到数据中心的电缆组件时，光纤肯定比模拟具有许多显着的优势，这使其成为广泛使用的理想之选。同时，如果我们分析大多数现有数据中心的结构组件，则铜线仍将在其中占据主导地位。铜缆并未完全屈服于新材料，而是紧密地编织到了数据中心基础设施中，据最严重的估计，铜缆具有很好的前景，可以继续在数据中心中占据关键位置。在本文的进一步内容中，我们将尝试理解这种情况（有些模棱两可），为什么铜仍然具有如此竞争力？



基于网络的网络设备提供商Emulex负责企业营销和开发的高级副总裁Shine Walsh表示：“多年来，进步一直在试图杀死铜，但到目前为止效果并不理想。”如Cisco，EMC，HP，HDS，华为。

铜检查


“最近的现有研究为我们提供了有关1 Gigabit / s主机端口之间超过20％的连接是铜缆的统计数据。当在短数据传输距离上使用时，标准CAT 6类电缆的价格质量比对大多数数据中心而言比光纤更具吸引力。通常，对于数据中心中紧凑放置的网络设备，铜线特性足以将连接速度保持在所需水平，”沃尔什续说。

在进行了一项大规模研究后，Dell`Oro Group的员工得出的结论是，2018年，数据中心中80％的高速连接（至少10 Gigabit / s）将使用铜缆以一种或另一种形式实现。

根据研究，它可以是双轴10GBase-CR电缆，可以通过标准SFP插槽直接连接到DAC（数字模拟转换器），也可以是使用双绞线CAT 6或CAT 6a的10GBase-T，它们可以轻松地提供数据传输此外，它们以10 Gb / s的速度在长达100 m的距离上向后兼容，可提供宽度为1 Gigabit / s的连接电缆。


支持10GBase-T的网卡和电缆已经在IT设备市场中广泛使用，并且由于竞争激烈，它们的价格非常合理。根据Dell`Oro Group员工的说法，该技术已经成熟，并且在未来几年中，它将越来越多地参与数据中心的功能。根据该公司的估计，到2017年，此类设备的供应量将比现在增加两倍，而主要原因仍然是，在两年内，现有的1 Gigabit / s信道将无处可去。

UTP连接性机架服务器


尽管构建数据中心的原理相似，但是，每个特定的数据中心都有独特的内容，在这种情况下，不可能为它们的升级制定单一，准确的“路线图”。此外，在一个大型数据中心内，可以存在不同区域，这些区域对网络连接有不同的本地要求。不同带宽的信道连接到不同的站点，并且可接受的网络延迟要求也有所不同。此外，不要忘记，即使机架本身的形状因素在这里也发挥了作用，这自然也不尽相同。这种状态对大多数现代建筑中铜的使用产生了积极影响，并且可以进行现代化改造。正如实践所示，通过结合基于光纤和铜缆的技术，可以实现最大的经济效益。

Mellanox的员工Arlon Martin指出：“我们将铜作为具有高达100 Gb / s连接性的服务器机架中的主要粘合剂，如果我们看一下连接这些机架的通道的情况，那么在大型数据中心，成功的公司，我们正在目睹光学领域的明显趋势。一方面，光学器件具有使数据中心中的所有设备现代化的巨大潜力，另一方面，这些IT巨头并不认为在这些区域中使用铜是节省成本的好方法。”



分析这两种化合物的竞争力后，结论本身就表明，光纤是未来的技术，没有它，就不可能实施现代高科技项目：云服务，高负荷的电信解决方案等。“如果您查看数据中心提供的服务市场，则不应关注亚马逊，谷歌，Facebook之类的巨头，那里有众多组织：银行，医院，博物馆，其可靠性和获利能力排在首位。Siemon专业网络部门负责人Keri Higby说，铜是基于这些消费者的。

价格因素


显然，在支持特定技术的最终选择中，已实施解决方案的价格起着关键作用。升级现有数据中心的成本与此处已使用的设备密切相关，网络客户端之间的距离，能耗和传输的数据量也都起着作用。顺便说一下，铜线连接的最大通道宽度并没有停滞不前，现在，根据IEEE数据传输标准，它可以使用八芯铜缆以40 Gigabit / s的速度支持数据传输。但是，现在该技术还没有得到广泛使用，只要其应用的价格不允许大规模引入这种电缆。



Higby说：“我认为，在一年半的时间里，设备的可用性（40GBase-T）不会改变，至少要花三年的时间才能将其用于商业用途。” “尽管这项技术本身很有希望，但同时它也很有可能失败。毕竟，现在已经有一个现成的解决方案，可以使用40 Gigabit / s通道，这当然是光学！在重新配置现有数据中心方面投入巨额资金不是很有希望，使用八核连接建立新的数据中心也不可取，因为该技术的质量不足以承受可承受的价格。”

同时，在设备市场上，现在在机架安装的DAC上有许多25-50 Gigabit / s SFP +的通用解决方案，通过连接到通道宽度为100-200 Gigabit / s的其余光学器件，您可以释放多个端口，或者将它们完全用于双绞线。

“通过分析我的活动中的技术组件，我可以合理地说，在大多数现有通道宽度为每秒100 Gigabit的机架中，一根8米长的铜缆电缆的特性足以连接该机架的所有组件。”

“应用的灵活性以及管理方面的一些优势使我们有可能断言铜仍然有未来。就像现在很难想象一个完全基于“光学”的成熟数据中心一样，没有它就不可能存在一个数据中心。在这方面，光学和铜缆并没有成为有效数据中心的补充组件那样多的竞争对手，” Shine Walsh总结道。
