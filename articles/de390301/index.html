Internet Archive hat über 1500 Programme für Windows 3.1 hochgeladen, Sie können direkt im Browser arbeiten



Vor einigen Tagen hat Internet Archive Dutzende alter Viren in das Web hochgeladen, die noch unter DOS ausgeführt wurden, und so das Malware Museum erstellt . Das „Museum“ hat ungefähr 80 klassische Viren, denen viele Benutzer dieser Zeit begegnet sind.

Jetzt gibt es ein weiteres Museum - diesmal mit alten Programmen, die unter Windows 3.1 funktionierten. Die Entwickler veröffentlichten zu dieser Zeit die beliebtesten Anwendungen, darunter sowohl rein funktionierende Tools als auch Spiele. All dies finden Sie im neuen Abschnitt des Windows 3.X Showcase . Insgesamt werden hier 1523 Programme veröffentlicht, deren Anzahl weiter zunimmt. Im Browser funktioniert dies alles dank des Windows 3.1 JavaScript-Emulators.

Die Entwickler sagten, dass einige Teile dieses Projekts immer in ihren Händen waren. Fast die gesamte Software wurde von Vertretern des Projektteams auf verschiedenen Medien gespeichert. Darüber hinaus wurden einige Programme verschickt und Enthusiasten, die von der Vorbereitung eines neuen Abschnitts im Internetarchiv hörten.

Nachdem die Anzahl der gesammelten Programme eine bestimmte Zahl erreicht hatte, begannen die Entwickler, einen Win 3.1-Emulator zu erstellen. Dies stellte sich als schwierige Aufgabe heraus, aber die Freiwilligen kamen damit klar.



Außerdem gibt es eine Windows 95-Vorschau-Emulation - diejenigen, die dies alles auf einmal gefunden haben, werden sich freuen, das "alte neue" Betriebssystem wieder zu sehen.





Natürlich ist eine Menge Software urheberrechtlich geschützt, und Entwickler mussten diesen Punkt berücksichtigen. Daher wurden einige Programme durch freie Gegenstücke ersetzt - und einige mussten gerade aus diesem Grund für lange Zeit beiseite gelegt werden.









Im Allgemeinen ist es an der Zeit, sich an die Vergangenheit zu erinnern.
