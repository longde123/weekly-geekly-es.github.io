Pusat Data Luar Angkasa. Terjemahan teks dari peluncuran server ke stratosfer

Hari ini kami meluncurkan server ke stratosfer. Selama penerbangan, stratostat akan mendistribusikan Internet, merekam dan mengirimkan data video dan telemetri ke darat (tetapi ini tidak akurat)). Anda dapat melihat pergerakan server dan data telemetri di situs web proyek.

Kami menyiarkan langsung dari situs peluncuran di lapangan dekat Pereslavl-Zalessky, jadi kami mohon maaf sebelumnya untuk kemungkinan efek khusus selama siaran.

Juga, sebuah kontes sedang berlangsung di situs, di mana lebih dari 500 penjaja memberi tanda pada peta untuk menebak di mana server mengalami crash. Hadiah utama bagi orang yang paling menebak adalah perjalanan musim panas ke Baikonur untuk meluncurkan pesawat ruang angkasa berawak Soyuz MS-13.




Ayo pergi!

Sedikit latar belakang dan tautan bermanfaat, bagi mereka yang melewatkan segalanya:

  1. Posting tentang bagaimana mengoordinasikan penerbangan probe ke stratosfer (apa yang akan kita temui dalam praktik saat peluncuran)

  3. Situs proyek tempat Anda dapat memantau pergerakan probe dan telemetri secara real time
  4. Perbandingan sistem komunikasi luar angkasa yang akan kita gunakan saat peluncuran
