Sprachsteuerung des Z-Wave Smart Home und darüber hinaus. Homekit + Siri. Bedienungsanleitung


Jeden Monat maximalen Komfort in der Wohnung zu schaffen, wird billiger und einfacher. In dieser Schritt-für-Schritt-Anleitung zeige ich Ihnen, wie einfach es ist, Ihrem Zuhause Sprachsteuerung hinzuzufügen! Für die Sprachsteuerung benötigen wir:
  1. RaZberry- Controller oder jeder Computer mit Z-Way- Software
  2. Ein Paar Z-Wave-Geräte sowie alle WiFi-Glühbirnen
  3. iPhone / iPad (vorzugsweise in der Wand)

Ich verwende Raspberry Pi B +, daher wird die Beschreibung der Sprachsteuerungseinstellungen später beschrieben. Videoarbeit am Ende des Artikels.

1) Installieren Sie die Software Z-Way Version älter als 2.2.2 :
wget -O - razberry.z-wave.me/install/v2.2.2-test-new-webserver-8-ga9cd319 | sudo bash
Z-Way ist ein Hub für Z-Wave-, Enocean-Geräte, verschiedene WiFi-Glühbirnen, Sockel, Schlösser und anderes IoT, das von HTTP- und TCP-Anforderungen gesteuert wird. Die Weboberfläche ist auf Port 8083 verfügbar.


2) Installieren Sie die neueste Version von Node.js.
wget https://nodejs.org/dist/v4.4.5/node-v4.4.5-linux-armv6l.tar.xz
tar -xvf node-v4.4.5-linux-armv6l.tar.xz
cd node-v4.4.5-linux-armv6l
sudo cp -R * /usr/local/
Node.js ist eine Plattform zum Ausführen von JavaScript-Code auf einem Server, der auf der Google V8-Engine basiert.


3) Installieren Sie das Homebridge- und Homebridge-zway-Plugin
sudo npm install -g homebridge
sudo npm install -g homebridge-zway
Homebrige implementiert Homekit-Unterstützung. Homebrige-zway ist ein Plugin, das Z-Way um eine verbesserte Homekit-Unterstützung erweitert.


4) Erstellen Sie die Konfiguration Homebrige /usr/local/lib/node_modules/homebridge/config.json
{
    "bridge": {
        "name": "Homebridge",
        "username": "CC:22:3D:E3:CE:31",
        "port": 51826,
        "pin": "031-45-154"
    },
    "platforms": [
    {
            "platform": "ZWayServer",
            "url": "http://localhost:8083/",
            "login": "admin",
            "password": "admin"
    }
    ]
}
Der letzte Schrägstrich in der URL ist sehr wichtig, ohne ihn funktioniert es nicht.

5) Starten Sie Z-Way und Homebridge
sudo /etc/init.d/z-way-server start
sudo homebridge -U /usr/local/lib/node_modules/homebridge &
Wenn alles korrekt ist, sehen Sie im Protokoll Geräte, die Sie steuern können, und einen PIN-Code.


6) Mit der Home-Anwendung auf dem Telefon oder anderen ( MyTouchHome , MyHome App usw.) fügen wir das Homekit-Gerät hinzu.


7) Fertig! Jetzt kann das Haus per Stimme gesteuert werden! Komfortable Steuerung!


PS Das letzte Mal habe ich das native Homekit-Modul für Z-Way zur Sprachsteuerung verwendet , aber Homebridge hat mir besser gefallen, da es nicht nur mit Z-Way, sondern auch mit anderen Automatisierungssystemen wie Philips Hue , Nest und vielen anderen funktioniert .
