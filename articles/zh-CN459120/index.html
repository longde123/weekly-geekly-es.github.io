UNO-1000 / 2000系列模块化嵌入式计算机



UNO 1000/2000是一台成熟的X86计算机，用于嵌入式系统，具有现成模块的形状，可以安装在桌子，墙壁，DIN导轨或VESA上。 该平台易于使用，可用于任何特定任务，这归功于众多专有的iDoor扩展模块，这些模块使您可以向设备添加必要的接口。

该行的主要特点：

  • 完全被动冷却 -没有活动部件，风扇和散热器的通风孔。 灰尘不会积聚在设备内部；它们不太可能需要维护。
  • 完善的X86架构 -无需移植旧程序，就像MIPS或ARM架构一样。 可以安装Windows的桌面版本。
  • 集成I / O端口 -集成数字I / O端口，用于连接外围设备
  • 内置RS-232 / 485/422接口-无需连接外部串行适配器
  • IDoor扩展模块 -超过30种品牌扩展卡可让您组装正确的配置
  • 9V至36V DC的电源电压 -宽范围的工作电压允许您将设备安装在现有系统中，而无需使用转换器。

UNO-1000系列




UNO-1000系列设计用于安装在DIN导轨上。 在阵容中，有两种基于ARM体系结构的带有Cortex A8处理器的模型。 本系列的主要目的是作为紧凑型服务器安装在配线柜中，以进行本地数据处理或自动化。

UNO-1251G




基于ARM体系结构的产品系列中最紧凑的设备。 它具有用于显示任意状态消息的可编程OLED屏幕，用于连接到汽车设备的CAN总线以及2个LAN端口和3个串行端口。

UNO- 1251G / 1252G可以配备Wi-Fi / 3G / 4G无线iDoor模块，并具有可从外部访问的SIM卡端口。

UNO-1372G



电脑外观UNO-1372G
该计算机有两个电源输入，用于主连接和备用连接。 如果发生电源故障，将立即过渡到备份，而无需重新启动。 支持将多个LAN端口组合到一个分组接口中，以提高容错能力。 如果其中一个接口发生故障，网络将继续正常运行。

I / O端口



数字量输入和输出端口，用于连接继电器，传感器和其他外围设备。
UNO-1372G具有4个数字输入和输出，用于控制外围设备，例如传感器，继电器，簧片开关。 您可以在用户程序级别使用来自操作系统的I / O端口。

UNO-2000系列




UNO-2000系列设计用于安装在嵌入式系统，移动物体，汽车，火车中。 外壳设计用于使用固定螺钉安装在平坦的表面上。 与UNO-1000系列产品相比，计算机具有更强大的处理器。 某些型号具有内置的硬件加密和TPM密钥存储模块以保护关键应用程序。

FRAM备份存储器


FRAM备份存储器使您可以存储关键数据，以防止崩溃或死机时丢失数据。 使用内存的方式与使用RAM的方式相同，但是在断电或其他故障的情况下不会丢失其值。 FRAM可以存储重要的计数器。 恢复系统后，该程序将继续使用发生故障之前的数据。

可信平台模块（TPM）2.0


UNO-2000系列计算机（例如， UNO-2484G型号 ）具有集成的TPM 2.0模块，用于对关键数据进行硬件保护。 TPM可用于受信任的OS引导（受信任的引导），加密密钥的存储以及随机数生成器。

看门狗防止冻结


为了防止程序和操作系统冻结，UNO计算机具有硬件可编程计时器（看门狗）。 计时器一直希望系统报告其运行正常。 每条新消息都会重置计时器。 如果计时器在一定时间内未收到消息，则认为系统已冻结，并且可能会自动重新引导。

已安装COM端口扩展模块的UNO-1251G
