Penipuan Macbook Pro yang digunakan

Hari ini, seorang teman meminta saya untuk membantu membeli Macbook Pro bekas. Saya segera memperingatkan bahwa kualifikasi saya hampir nol, tetapi saya dapat membantu dengan apa yang saya bisa. Saya belum pernah bekerja sama dengan PC selama bertahun-tahun, dan saya memiliki Mac di tingkat teko teratas. 2. Saya tidak mementingkan frasa penulis berikut: â€œdalam tangkapan layar ini jelas bahwa dalam informasi terperinci tidak ada informasi tentang prosesor, memori, dan disk. Saya tidak memperhatikan ini - ini adalah kesalahan saya nomor 1. " Ini benar, tanda penipuan pertama dan paling pasti.

3. Untuk memeriksa hard drive, saya mencoba menjalankan utilitas Disk, tetapi tidak bisa. Setelah memasuki folder program, saya melihat bahwa utilitas Disk dicoret dan saat startup itu menampilkan pesan tentang ketidakmungkinan bekerja - yaitu, Anda harus menginstal ulang sistem operasi. Anda hanya perlu pergi ke root di Finder dan melihat properti disk, tetapi saya tidak menyadarinya. Label volume adalah HDD 750 GB dan membingungkan. Properti disk ternyata 500 GB.

4. Penulis menunjukkan kesalahan perintah konsol "system_profiler", tetapi pada salinan laptop saya, saya tidak melihat kesalahan seperti itu. Atau mungkin dia tidak melihat ke sana.

5. Salah satu penyergapan utama - Tes Perangkat Keras Apple tidak berfungsi dengan menekan D saat Anda menghidupkan perangkat. Saya benar-benar tidak siap untuk skenario ini, sementara saya tidak tahu cara pintas keyboard dasar lainnya untuk mengendalikan komputer sebelum memuat OS. Kemudian, saya mencoba di rumah saya iMac pertengahan 2010 - hal yang sama, ketika Anda menekan D, tidak ada yang terjadi begitu saja. Berada di rumah, saya mengambil keuntungan dari opsi Uji Perangkat Keras Apple melalui Internet - untuk melakukan ini, tahan Alt + D setelah dinyalakan. Tetapi opsi ini mungkin juga tidak berfungsi. Misalnya, di Internet rumah saya, tes Internet memberikan kesalahan 3404D pada laptop dan iMac. Dinilai oleh dokumentasi support.apple.com/en-us/HT201257Anda harus membawa Aplikasi Instal Disc 2 atau USB flash drive dengan hati-hati dari Drive Instal Ulang Perangkat Lunak MacBook Air dan unduh Tes Perangkat Keras Apple dari mereka.

Tes Perangkat Keras Apple dapat diunduh di sini github.com/upekkha/AppleHardwareTest
Ada juga instruksi tentang cara membuat USB flash drive. Opsi ini secara signifikan lebih cepat dalam hal tenaga daripada membuat flash drive USB yang dapat di-boot dengan OS X, tetapi kurang universal. Jika Anda menggunakan tes dari model yang salah, maka pada saat startup akan memberikan kesalahan bahwa komputer ini tidak didukung.



Dan dengan USB flash drive yang dapat di-boot dengan OS X, dengan menjalankan system_profiler Anda dapat mengetahui dengan tepat model mana yang ada di depan Anda.
