As 9 melhores descobertas de código aberto para novembro de 2019

Bom inverno, senhoras e senhores. Eu preparei para você uma seleção dos achados mais interessantes do código aberto para novembro de 2019.


Para obter uma lista completa de novas ferramentas, artigos e relatórios úteis, entre em contato com meu canal de telegrama @OpensourceFindings ( espelho de link, se o original não abrir).


Na edição de hoje.
Tecnologias internas: Rust, TypeScript, JavaScript, Go, Python.
Tópicos: desenvolvimento web, ferramentas para controle de qualidade, trabalhando com dados, ferramentas de desenvolvedor e administrador.


Última edição .


histórico


Uma máquina do tempo para o github. Permite que você veja literalmente a aparência de um arquivo a qualquer momento. Confortável, linda.
Está escrito em JavaScript.


Link



qawolf


Um único npx qawolf record example.com para gravar testes do navegador. Parece super fácil.
Escrito em TypeScript.


Link



plynth


E agora um pouco de frenesi. Você pode escrever aplicativos da Web rápidos, nativos e seguros para threads em HTML / CSS / Python. O que? Python Sim, Python.
Você pode até usar o Vue.js internamente. E escreva código em Python. Aqui, dê uma olhada!
Está escrito em Python.


Link



lazydocker


TUI minimalista para quem trabalha muito com o Docker e gosta de usar o mouse.
Escrito em Go.


Link



tui-rs


By the way, sobre TUI. Aqui está uma biblioteca útil para criar seus aplicativos. Dentro de um monte de widgets prontos.
Escrito em Rust.


Link



easydb


Banco de dados em um clique. Sem registro e SMS.
Os clientes são para Python, JavaScript, Ruby, Bash.


Link



transform.tools


Serviço muito conveniente, permite converter automaticamente um formato para outro. Por exemplo:


  • HTML e SVG em jsx
  • json no Kotlin, TypeScript ou Go Struct
  • Markdown em HTML, yaml em json, etc.

Link



Dovpanda


Linter para pandas . Localiza erros e avisos. Muito útil para quem trabalha muito com dados. Ou ensina-os a trabalhar.
Está escrito em Python.


Link



flowy


A biblioteca para criar fluxogramas é executada no computador e no celular. Nice!
Está escrito em JavaScript.


Link
