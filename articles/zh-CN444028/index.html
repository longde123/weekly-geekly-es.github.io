微软游戏堆栈介绍


我们宣布了新的Microsoft Game Stack计划，该计划将结合Microsoft工具和服务，使所有游戏开发人员（无论他们是独立开发人员还是AAA工作室）都可以实现更多目标。


当今世界上，有20亿玩家在各种设备上玩各种游戏。 社区对视频流，观看和共享的关注与对游戏或比赛的关注一样。 作为游戏的创作者，无论他们身在何处或使用何种设备，您每天都在尝试吸引他们的玩家，激发他们的想象力并激发他们的灵感。 我们介绍Microsoft Game Stack来帮助您做到这一点。


本文为英文。

什么是Microsoft Game Stack？


Game Stack将我们所有的游戏开发平台，工具和服务（例如Azure，PlayFab，DirectX，Visual Studio，Xbox Live，App Center和Havok）集成到任何游戏开发人员都可以使用的强大生态系统中。 Game Stack的目标是帮助您轻松找到创建和管理游戏所需的工具和服务。


云在游戏堆栈中扮演着至关重要的角色，而Azure满足了这一至关重要的需求。 Azure提供了基本部分，例如计算和存储，以及基于云的机器学习和人工智能服务，用于发送通知和混合现实空间参考。


诸如Rare，Ubisoft和Coasts of Coast之类的公司目前与Azure合作。 他们托管用于多人游戏的服务器，安全可靠地存储玩家数据，分析游戏遥测，保护其游戏免受DDOS攻击，并训练AI创造更令人兴奋的游戏玩法。


尽管Azure是游戏堆栈的一部分，但必须注意，游戏堆栈独立于云，网络和设备。 我们不止于此。


最新消息


Game Stack的下一个组件是PlayFab，这是用于创建和操作游戏的完整后端服务。 一年前，PlayFab和微软合并。 今天，我们很高兴地宣布，我们将PlayFab添加到Azure系列中。 Azure和PlayFab在一起是一个强大的组合：Azure提供可靠性，全局规模和企业级安全性； PlayFab提供由游戏栈驱动的游戏开发服务，实时分析和LiveOps功能。


PlayFab联合创始人James Gwerzman表示：“现代游戏的创造者越来越不像电影导演。 长期的成功需要玩家参与不断的创造，试验和开发周期。 您不能只是推出游戏并继续前进。” PlayFab支持从iOS和Android到PC和Web，Xbox，Sony PlayStation和Nintendo Switch的所有主要设备； 以及所有主要游戏引擎，包括Unity和Unreal。 PlayFab将来还将支持所有主要的云服务。


今天，我们也很高兴地宣布了五项新的PlayFab服务。


在今天的公开预览访问中：


  • PlayFab对接会 ：一种功能强大的对接会搜索，适用于多人游戏，改编自Xbox Live，但现在可用于所有游戏和所有设备。

在今天的私人预览访问中（ 写信给我们以获得访问权限 ）：


  • PlayFab Party：语音和聊天服务改编自Xbox Party Chat，但现在可用于所有游戏和所有设备。 Party使用Azure认知服务进行实时翻译和转录，以使游戏可供更多玩家使用。
  • PlayFab Insights：将可靠的实时游戏遥测与来自其他多个来源的游戏数据相结合，以衡量游戏的性能并产生有用的想法。 Game Insights基于Azure数据资源管理器，将提供与现有第三方数据源（包括Xbox Live）的连接器。
  • PlayFab PubSub：通过具有Azure SignalR支持的永久连接向您的游戏客户端订阅从PlayFab服务器发送的消息。 这允许进行诸如实时内容更新，配对通知和简单的多人游戏玩法等场景。
  • PlayFab用户生成的内容：通过允许玩家创建并与其他玩家安全共享用户生成的内容来吸引社区。 这项技术最初是为支持Minecraft市场而开发的。

不断发展的Xbox Live社区


游戏堆栈的另一个重要组件是Xbox Live。 在过去的16年中，Xbox Live已成为世界上最活跃，最活跃的游戏社区之一。 这也是一个安全且包容的网络，扩大了游戏的范围：玩家现在正在使用不同的设备进行合作。


我们很高兴Xbox Live将成为Microsoft Game Stack的一部分，提供身份和社区服务。 作为游戏堆栈的一部分，Xbox Live将扩展其跨平台功能，因为我们将展示将这个社区引入iOS和Android设备的新SDK。


借助Xbox Live，移动应用程序开发人员将能够与地球上最热情和最参与的玩家聊天。 以下是针对移动开发人员的一些好处：


  • 受信任的游戏身份：使用新的Xbox Live SDK，开发人员可以专注于创建出色的游戏，并使用Microsoft的受信任的身份网络来支持登录，隐私，在线安全性和子帐户。
  • 无摩擦集成：新的自定义选项和没有Xbox Live证书的移动应用程序开发人员可以灵活地创建和更新他们的游戏。 开发人员只是使用最适合他们需求的服务。
  • 充满活力的游戏社区：加入不断发展的Xbox Live社区，并将游戏玩家连接到多个平台。 寻找实现成就系统，Gamerscore和英雄统计数据的创新方法。

其他游戏堆栈组件


其他游戏堆栈组件包括Visual Studio，混合器，DirectX，Azure应用程序中心，Visual Studio，Visual Studio代码和Havok。 在接下来的几个月中，随着我们努力改善和扩展游戏堆栈，您将看到这些服务之间的更深层次的联系，因为我们将它们组合在一起以使它们更有效地协同工作。


作为这种集成如何进行的一个示例，今天我们将PlayFab和以下游戏堆栈组件链接在一起：


  • App Center：来自App Center的崩溃日志数据现在已连接到PlayFab，使您可以实时更好地理解和响应游戏中的问题，并将单个崩溃与孤立的玩家相关联。
  • Visual Studio代码：使用适用于Visual Studio Code的新PlayFab插件，编辑和更新Cloud Script变得更加容易。
