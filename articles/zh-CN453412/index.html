顶级3D商店成为UFactory的独家分销商



顶级3D商店已成为UFactory产品在俄罗斯和其他EAEU国家的授权独家分销商，已准备好销售并降低了价格。 已终止通过其他渠道向这些领土的运送。



UFactory是来自中国工业中心深圳市的教育型协作机器人制造商。 该公司的产品线包括uArm Swift Pro机械臂（57,000卢布），它是与Dobot Magician Basic （156,000卢布）最接近，更实惠的竞争对手，它具有相同的优势：两种设备都是教育性的操纵器，可与Blocky软件环境一起使用。 / Python / C ++，都可以通过PC或智能手机进行控制，并且还提供可选的喷嘴和附件，它们的最大起重量均为500克，定位精度为0.2毫米。 主要区别在于成本。 当然，除了不同的制造商，设计和设计上的细微差别。 功能相似。

该公司还生产uArm的附件和组件，例如手柄，用于激光雕刻和3D打印的喷嘴，用于精确定位和反馈的摄像头，架子和传送带。







机器人易于学习和编程，适用于各种教育和娱乐目的。







顶级3D商店很高兴推出对俄罗斯和独联体教育如此重要的产品。
