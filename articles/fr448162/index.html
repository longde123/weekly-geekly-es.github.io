Utilitaires Solarwinds gratuits pour la surveillance, la gestion de l'infrastructure informatique et la sécurité



Nous connaissons bien Solarwinds et travaillons avec lui depuis longtemps, beaucoup connaissent également leurs produits pour la surveillance du réseau (et pas seulement). Mais il n'est pas si connu qu'ils vous permettent de télécharger à partir de leur site Web une bonne quarantaine d'utilitaires gratuits qui aideront à contrôler les périphériques réseau, à gérer l'infrastructure, les bases de données et même à gérer les incidents. En fait, ce logiciel est des fragments séparés de leurs produits payants. Tous les utilitaires sont 100% gratuits, pas des versions d'essai. Sous les liens de chat vers la description et le téléchargement.

Dans cette revue, les utilitaires pour:

  • gestion de l'infrastructure du réseau;
  • Gestion de l'infrastructure informatique;
  • Gestion de la sécurité informatique
  • gestion de bases de données;
  • Organisation du service d'assistance.

Gestion de l'infrastructure réseau


ipMonitor Free Edition




Un outil simple. L'interface définit une plage d'adresses, elle les interroge et donne les nœuds disponibles pour la surveillance à distance. Parmi les outils de surveillance possibles: ping, WMI, SSH. Sur les serveurs, il peut effectuer des vérifications standard telles que CPU, mémoire, disque. Il s'agit de la version gratuite du frère aîné ipMonitor (non gratuite) et prend en charge jusqu'à 50 nœuds. Pour un petit bureau ou des tâches de test, il n'y a rien du tout. Il y a une courte vidéo avec une description .

Ensemble d'outils de flux




Réplication, test et configuration des appareils pour envoyer le trafic de flux. L'outil se compose de trois parties: un réplicateur, un générateur et un configurateur. Le réplicateur peut recevoir et transmettre le trafic netflow à un ou plusieurs récepteurs. Générateur - génère netflow pour les tests, la vérification des paramètres des pare-feu et à d'autres fins. Le configurateur configure automatiquement les périphériques réseau pour envoyer netflow au périphérique cible. Il y a une courte vidéo avec une description .

Traceroute NG




Cet utilitaire est un module complémentaire pour le système tracert. Effectue une analyse de l'itinéraire réseau, peut exécuter des paquets à l'aide des protocoles TCP et ICMP. Il y a une courte vidéo avec une description .

Scanner de port




Génère une liste de ports ouverts, fermés et filtrés pour chaque adresse IP analysée. Il peut numériser via TCP et UDP, et après la numérisation, vous pouvez télécharger toutes les données collectées dans un fichier. Fonctionne notamment via la ligne de commande.
Il y a une courte vidéo avec une description .

Moniteur de périphérique réseau




Surveillance des périphériques réseau. L'utilitaire est livré avec de nombreux modèles différents pour la surveillance via SNMP, vous pouvez configurer des rapports et des alertes. Mais il y a une limitation: la surveillance d'un seul appareil est prise en charge. Une courte vidéo avec une description .

Émulateur de réseau GNS3




Émulateur d'environnement réseau. L'utilitaire fonctionne sur Windows, Mac et Linux. Plus de 20 fournisseurs d'équipements réseau sont pris en charge. Vous pouvez émuler différentes topologies de réseau et voir comment tout cela fonctionne. Il y a une vidéo avec une description .

Visionneuse de temps de réponse pour Wireshark




Module complémentaire Wireshark pour une analyse facile des paquets par temps de réponse. Il détecte le trafic pour environ 1200 applications. Fonctionne comme une calculatrice pour le temps de réponse de l'application. Il y a une vidéo avec une description .

Ensemble d'analyseur de réseau et de surveillance de la bande passante




L'utilitaire est conçu pour surveiller le trafic NetFlow, J-Flow et sFlow. L'interface a des représentations sous forme de différentes tranches: sur l'interaction entre les appareils, sur les applications, sur les domaines et les appareils finaux. De plus, il est possible d'enregistrer des données jusqu'à 60 minutes.

Serveur TFTP




Serveur TFTP avec une interface graphique. Fonctionne sous Windows en tant que service. Il y a une vidéo avec une description .

Traqueur d'adresses IP




Analyse, validation des modifications, identification des conflits d'adresses IP. Il s'agit du frère cadet d'un gestionnaire d'adresses IP senior - peut gérer jusqu'à 254 adresses, contrairement à 2 millions dans une solution payante.

Moniteur de bande passante en temps réel




Surveillance de la bande passante des interfaces réseau et visualisation des métriques. Également une solution junior pour Network Performance Monitor (NPM).

Suivi des enregistrements des détails des appels




Recherchez, filtrez et triez les journaux CDR CCM.

Moniteur SLA IP




Collectez les données IP SLA des appareils Cisco et visualisez-les.

Calculateur de sous-réseau avancé




Calculatrice IP pour la conception de l'espace d'adressage.

Kiwi Syslog Server Free Edition




Analyse, réception et archivage de syslog et SNMP. Il y a une vidéo avec une description .

FTP Voyager FTP Client pour Windows




Client FTP avancé pour Windows. Prend en charge FTP, FTPS et SFTP. Grâce à ce client, vous pouvez effectuer des transferts de fichiers planifiés.

Serveur SFTP / SCP




L'utilitaire fonctionne comme un service Windows et prend en charge le transfert simultané de fichiers vers plusieurs appareils.

Gestion de l'infrastructure informatique


Calculateur de coût pour azur




L'utilitaire peut afficher simultanément les informations financières de plusieurs comptes. L'interface présente une ventilation des coûts pour l'année, le trimestre, le mois et, en outre, les ressources payées non utilisées sont affichées. Il fonctionne sur les plates-formes Windows et Mac.

Redirection du journal des événements pour Windows




Regroupe les événements du journal Windows et les envoie en tant que syslog où vous le souhaitez. Il existe un filtrage flexible. Quels événements des journaux Windows doivent être contrôlés - nous avons écrit dans le dernier article .

Solar-PuTTY




Mastic avancé. Il a une interface multi-onglets et une recherche pratique des sessions enregistrées. Il y a une vidéo avec une description .

Moniteur Vm




Surveillance des performances de VMware et Hyper-V. L'interface affiche des mesures de performances de base et l'état des machines virtuelles.

Moniteur d'intégrité du serveur




Conçu pour surveiller les serveurs physiques. Prêt à l'emploi peut fonctionner avec les serveurs Dell PowerEdge, HP ProLiant, IBM eServer xSeries et les hyperviseurs VMware ESX / ESXi. Prend en charge les protocoles d'interopérabilité SNMP, WMI et CIM.

Moniteur de performances de stockage




Surveillance des baies de données Dell EMC, NetApp, IBM, Pure Storage. Il y a une vidéo avec une description .

Moniteur Exchange




Surveillance des principaux services Exchange. Il y a une vidéo avec une description .

Activateur d'exécution à distance pour PowerShell




Utilitaire pour la configuration automatique du service WinRM local et distant.

Pack d'administration pour Active Directory




Un ensemble de trois utilitaires pour: détecter les comptes inutilisés, détecter les postes de travail inutilisés et importer des comptes d'utilisateurs dans AD. Il y a une vidéo avec une description .

Outil de diagnostic pour l'agent WSUS




Un utilitaire pour vérifier la disponibilité des services Windows Update (WSUS) sur des serveurs distants.

Transaction de transaction Web




Un utilitaire pour effectuer des tests synthétiques. Une transaction est prise en charge pour les applications Web.

Moniteur Wmi




Un utilitaire pour contrôler un serveur Windows via WMI. Il s'agit du serveur, car un seul périphérique est pris en charge.

Activateur SNMP pour Windows




Configurez les compteurs SNMP sur les serveurs Windows distants.

Gestion de la sécurité informatique


Consolidateur du journal des événements




Consolidateur de journaux Windows avec recherche d'événements pratique et paramètres d'alerte conditionnels. Il y a une vidéo avec une description .

Navigateur de pare-feu




Un utilitaire pour tester les règles de pare-feu. Prend en charge l'importation de configuration à partir de périphériques Cisco, Check Point et NetScreen.

Analyseur d'autorisations pour Active Directory




Un analyseur des droits d'accès pour les utilisateurs, les membres du groupe et les droits de gestion des boules de réseau.

Gestion de base de données


Database Performance Analyzer Gratuit




Analyseur de requêtes en temps réel pour les bases de données SQL Server, Oracle, DB2 et SAP ASE. Affiche les requêtes lentes. Il y a une vidéo avec une description .

Avertissements du plan SQL




Utilitaire d'analyse du plan d'exécution dans SQL Server. Surveille l'utilisation des ressources à mesure que le plan progresse. Il y a une vidéo avec une description .

Organisation du service d'assistance


Web Help Desk Free Edition




Gérez les incidents, les problèmes et les niveaux de service. Il y a intégration avec AD et rapports. Il s'agit du petit frère d'une version entièrement fonctionnelle.

La plupart de ces utilitaires sont adaptés à la résolution de problèmes locaux, certains peuvent être utilisés comme systèmes de production. Au cours des 2-3 dernières années, Solarwinds a activement développé ses produits et maintenant ils peuvent déjà être considérés comme un système universel de surveillance du niveau de l'organisation. Si vous recherchez un système de surveillance pour ce rôle, pensez à inclure Solarwinds.
