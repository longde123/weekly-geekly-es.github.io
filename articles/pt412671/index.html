Console BLE: uma maneira completamente nova de interagir com dispositivos BLE

O primeiro aplicativo de console do mundo na Microsoft Store.



A idéia desse singular (não tenho medo dessa palavra) surgiu literalmente, espontaneamente. Eu não "espalharei uma árvore sobre uma árvore" - se você é um desenvolvedor que lida com dispositivos BLE (BlueTooth Low Energy), tente encontrar um análogo funcional desse aplicativo em código aberto ou aplicativos pagos, sem mencionar lojas especializadas para determinados sistemas operacionais. Admito plenamente a possibilidade de que, no Linux (bem, ou uma das múltiplas implementações), exista algo semelhante, mas eu não vi isso. Sim, se você viu algo semelhante publicado anteriormente - me avise.

Tenho o prazer de apresentar a comunidade BLEConsole , o primeiro aplicativo do mundo para profissionais (e amadores) a trabalhar com dispositivos BLE no console padrão do Windows 10!

Provavelmente, alguns "estetas" (que não confundiram uma única linha de código para o público em geral na vida) acharão o código fonte desse aplicativo "terrível"; isso não me incomoda nada :) O principal é que esse aplicativo se torne útil e familiar para você. E se alguém obtiver algum conhecimento com o código-fonte, melhor ainda!

E depois disso: o BLEConsole se tornou o primeiro aplicativo de console na história do Windows 10 publicado ... na Microsoft Store ;) Isso foi feito "apenas uma prova de conceito" em vez de buscar um objetivo específico, mas se você quisesse me comprar cerveja - de nada ! ;) Naturalmente, a versão "trial" não é diferente da "full" e, mais ainda, do nabo auto-montado no github;)
[ATUALIZAÇÃO] Os comentários mencionaram o aplicativo do console Ubuntu (dos criadores deste sistema operacional), que estava à frente do console BLE. Mas, no entanto, o "BLE Console" ainda é o único aplicativo de console para o Windows 10 que funciona com dispositivos BLE!
