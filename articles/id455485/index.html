Check Point Scripts - jalankan skrip langsung dari Smart Console



Kami sebelumnya menulis bahwa semua pengaturan Titik Periksa dapat dibagi menjadi dua kelompok: Pengaturan Sistem dan Pengaturan Keamanan. Kita dapat mengontrol pengaturan sistem melalui WebUI, baik melalui CLI, atau melalui Gaia REST API (mulai dari 80.10). Pengaturan keamanan sebagian besar diedit melalui SmartConsole, atau sekali lagi, melalui API. Dalam 95% kasus, administrator menggunakan SmartConsole. Namun, kadang-kadang ada kebutuhan untuk mengakses parameter atau fungsi yang lebih maju, yang hanya dimungkinkan melalui CLI. Sebelumnya, Anda harus terhubung melalui SSH untuk ini, kemudian ingat perintah yang diperlukan (misalnya, status cphaprob untuk memeriksa status cluster). Sangat tidak nyaman dan tidak begitu cepat. Semuanya telah berubah dengan munculnya Gudang Script . Kami akan mencurahkan artikel ini untuk hal ini.
