Die Chinesen und nicht nur sie verwenden seit Tausenden von Jahren „unruhige Steine“ zur Entspannung. Bobby - ein Rucksack für Geeks

Die oberste Schicht besteht aus einem eleganten Polyestergewebe, gefolgt von einer 6-mm-Schicht aus Anti-Schock-Material (das vor Stößen schützt). Danach gibt es mehrere Stoffschichten, die zusätzlichen Schutz bieten. Im Inneren - eine Reihe von Abteilungen aller Art, in denen alle Geräte (Laptop, Tablets, Telefon), Thermoskanne, Schreibwaren und vieles mehr untergebracht werden können. Alles ist bis ins kleinste Detail durchdacht, es wird sofort klar, wenn Sie den Rucksack in die Hand nehmen. Die Abmessungen des Aquariums sind klein - 30,5 x 29,5 cm. Es passt sowohl zu Hause als auch im Büro. Sie können eine Wasserfarm in Russland von Madrobots kaufen .

Nach dem Zusammenbau erhalten Sie einen Holztresor mit Zahlenschloss, drehbaren Griffen und anderen Eigenschaften eines echten Safes. Und ja, Sie können den Code auf das Schloss setzen. Das Set besteht mit einer programmierbaren Lasermaschine aus Sperrholz: Teile lassen sich leicht aus den Tabletten herausdrücken, es entstehen fast keine Grate, alle Beschriftungen (Fahrmodi, Symbole, Herstellerlogo) werden ebenfalls mit einem Laser aufgebracht.
