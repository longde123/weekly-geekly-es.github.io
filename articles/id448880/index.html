PHP Digest No. 154 (9-21 April, 2019)


Seleksi segar dengan tautan ke berita dan bahan. Dalam rilisnya: Zend Framework berada di bawah naungan Linux Foundation, berita dari PHP Internals, sebagian dari alat yang berguna, dan banyak lagi.

Selamat membaca!



Berita dan Siaran


  • Zend Framework berada di bawah sayap Linux Foundation - dan mengganti nama Laminas untuk menyingkirkan merek komersial Zend.
    Ekspresif , Kemampuan , dan ZF sendiri akan masuk ke organisasi baru. Sejauh ini, repositori hanya memiliki paket bridge - laminas / laminas-zendframework-bridge dengan autoloader khusus. Mereka berjanji akan menyelesaikan langkah penuh dalam beberapa bulan mendatang.

PHP internal


  • [RFC] Arrow Functions 2.0 - Voting pada lambdas pendek telah dimulai dan proses pemungutan suara menggembirakan. Sintaksnya tetap sama: fn($x) => $x * $y .
  • [RFC] Operator terner asosiatif kiri - Di hampir semua bahasa, operator ternary dievaluasi dari kanan ke kiri, tetapi dalam PHP, sebaliknya. Diusulkan untuk menghapus asosiasi kiri sama sekali, dan jika perlu menggunakan beberapa operator ternary, maka secara eksplisit menunjukkan urutan dengan tanda kurung. 
     1 ? 2 : 3 ? 4 : 5; // deprecated (1 ? 2 : 3) ? 4 : 5; // ok 1 ? 2 : (3 ? 4 : 5); // ok
  • PHP Internals News # 5 - Inklusi Daftar dengan penulis RFC Larry Garfield.
  • PHP Internals News # 6 - Wawancara dengan Remi Collet, rilis manajer PHP 7.2 tentang jaminan kualitas dan ekstensi PHP.

Alat-alatnya


  • pinba-server / pinba-server - Server pinba sederhana dalam PHP murni yang menyimpan semua data di Clickhouse. Ada dasbor siap untuk grafana. Instalasi mudah dan persyaratan sistem rendah, dan data terkompresi membutuhkan ruang disk yang sangat sedikit. Dikirim oleh morozovsk .
  • cycle / orm - ORM untuk aplikasi berumur panjang dari RoadRunner , Anton Titov. Dokumentasi terperinci dalam proses, ada perbandingan dengan Ajaran dan Eloquent .
  • php-kawan / kawan-dev - Penjadwal dan pengelola tugas dalam PHP.
  • jwage / changelog-generator - Generator daftar perubahan marketdown berdasarkan informasi dari GitHub.
  • moneyphp / money - Implementasi pola Fowler Uang. String digunakan di bawah kap.
  • mamuz / PhpDependencyAnalysis - Alat ini membangun grafik dependensi tipe abstrak dan menunjukkan dependensi yang buruk.
  • robinvdvleuten / php-ulid - Alih-alih UUID biasa, Anda dapat menggunakan ULID sebagai pengenal, yang memiliki beberapa keuntungan, khususnya, secara leksikal meningkat secara monoton, yang berarti dapat dengan mudah disortir.
