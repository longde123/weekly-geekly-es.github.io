Windows 10 Mobile - Build 10549 für Insider verfügbar

Beschäftigte Tage für Insider. Gabe Aul hat gestern Abend auf Twitter einen neuen Build für Windows 10 Mobile angekündigt.



Um einen neuen Build zu installieren, müssen Sie ein Rollback mit dem Windows-Gerätewiederherstellungstool auf Windows Phone 8.1 durchführen und anschließend die Anwendung neu installieren. Nehmen Sie am vorläufigen Evaluierungsprogramm Windows -> Wählen Sie Fast Ring und installieren Sie eine neue Assembly.
Tatsache ist, dass unmittelbar vor der Update-Verteilung ein Fehler gefunden wurde, der ein Update vom vorherigen Build unmöglich macht.

Der vorherige Build hat sich nach einem Hard-Reset recht gut verhalten. Ich hoffe, dieser Build wird sich auch als stabil erweisen. Die einzige Überraschung ist die Notwendigkeit eines Zurücksetzens, um auf den neuesten Build zu aktualisieren.

UPD: Für diejenigen, die kein Rollback durchführen möchten - warten Sie auf den nächsten Build. Gabe auf Twitter sagte, dass ein Upgrade von 536 nur für den nächsten Build möglich sein wird.
