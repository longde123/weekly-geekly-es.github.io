电动汽车的消费数据被制造商低估了

丹麦技术大学运输研究所（Danmarks Tekniske Universitet）进行的为期两年的研究表明，在声称的制造商上方行驶时，电动汽车的电力消耗几乎为50％。

这项研究，题为«治理大数据为估算能量消耗和电动汽车的练习场»（“ 利用大数据来确定的电动汽车能源动力储备 ” 。链接到研究（英文）的文章）有一个覆盖面广： 741名驾驶员在丹麦道路上测试了200辆雪铁龙C-Zero，三菱i-MiEV和标致iOn汽车。每辆车被移交给丹麦居民三到六个月。然后，联合分析了GPS和用于充电的功耗数据。

结果表明，实际消耗量为183 Wh / km，而不是制造商宣布的消耗量为125 Wh / km。这些数据是从略大于230万公里的结果中获得的。超出部分为46.4％。

根据进行测试的一位工程师的说法：“雪铁龙C-Zero电池容量为16 kWh，意味着平均射程为87 km。这比制造商宣布的130或150公里要小得多，后者是根据新的欧洲行驶周期的规定实现的。” （Wikipedia上的新欧洲行驶周期）

因此，与带有ICE的汽车一样，真实特性和声明特性之间存在显着差异。只有四分之一的行程显示出较低的消耗量或达到了宣布的125 Wh / km的水平。当然，在其他旅行中，消耗量更高，有时甚至更高（在某些旅行中，达到400 Wh / km）。

在研究期间，在规划城市旅行和购买电动汽车方面也获得了重要的结果。就范围而言，电动汽车以30 km / h的速度进行的悠闲运动并不是最佳选择。最大的储备也是最经济的模式是平均速度为52 km / h的机芯。这也很重要-随着速度降低到30 km / h以下，每公里的能耗成倍增加。
与装有内燃机的汽车不同，在速度上的消耗增加相对较小。因此，平均速度为25 km / h的消耗量等于100 km / h的消耗量 [ 这里，“消耗量”是指 ECR- 能耗率 -idiv]。

对于驾驶员来说，以下几点很重要-慢速行驶并不意味着更经济，更进一步。对于负责城市发展规划的人来说，这一点很重要，如果他们想减少市中心的二氧化碳排放量，则倡导在同一中心的速度不超过30公里/小时。

交通运输研究所的卡洛·普拉托（Carlo Prato）表示，这项研究的另一个结果是：“人们想给汽车充电，尽管仍然至少要充电一半。这绝对表明人们担心在路上不收费。”

平均而言，每个驾驶员每天进行3.4次出行，其中近一半短于5公里[ 根据研究的作者，该值是由于丹麦社区规模小和对电动汽车 -idiv的不信任 ]。只有百分之一的行程超过50公里。此外，温度也起着重要作用：在温暖季节，平均消耗量为168 Wh / km，在寒冷季节为225 Wh / km（几乎高出三分之一）。根据测试，在14°C的温度下实现了最小的消耗。
