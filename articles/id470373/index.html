PHP Digest No. 165 (23 September - 7 Oktober 2019)


Seleksi segar dengan tautan ke berita dan bahan. Dalam rilisnya: PHP 7.4.0 RC3, dukungan untuk ketergantungan Composer pada GitHub, memperbarui standar PSR, penawaran RFC dari PHP Internals dan tentang mengoordinasikan proyek PHP, sebagian dari alat yang bermanfaat, podcast, mitaps, dan banyak lagi.

Selamat membaca!



Berita dan Siaran


  • GitHub menambahkan dukungan untuk grafik dependensi PHP - File Composer.json akan dianalisis secara otomatis untuk dependensi yang tidak aman. dan repositori untuk menerima permintaan kumpulan yang sesuai dengan pembaruan.
  • PHP-FIG membutuhkan bantuan Anda memperbarui standar PSR! - Grup PHP-FIG menawarkan penambahan deklarasi tipe secara bertahap untuk nilai dan parameter pengembalian dalam standar PSR yang telah diterima. Sehingga dalam aplikasi modern Anda dapat memanfaatkan jenisnya, tetapi pada saat yang sama meninggalkan standar yang tersedia untuk aplikasi yang lebih lama.
  • PHP 7.4.0 RC3 - Kandidat rilis lain dikirim sesuai jadwal . Hanya 3 RC yang tersisa sebelum PHP 7.4. Sementara itu, panduan untuk migrasi dari PHP 7.3.x ke PHP 7.4.x - terima kasih lex111 untuk terjemahannya.
  • PHP 7.3.10 , PHP 7.2.23 - Bug keamanan diperbaiki, semua pengguna didorong untuk memutakhirkan.

PHP internal


  • [RFC] Object Initializer - Diusulkan untuk mengimplementasikan sintaksis untuk inisialisasi cepat objek:
    Teks tersembunyi 
    class Car { public int $yearOfProduction; public string $vin; } $car = new Car { yearOfProduction = 2019, vin = "1FTFW1CVXAFD54385", }; $car = new Car { yearOfProduction = 2019, }; // throws RuntimeException: Initialization of Car class object failed due to missing required properties
    Secara singkat tentang proposal di pos penulis dan lebih banyak di podcast PHP Internals News # 30 ada bersamanya.
  • [RFC] Peringatan klasifikasi mesin - Pemberitahuan dan penilaian yang dilemparkan ke dalam mesin PHP direvisi dan sesuai dengan hasil pemungutan suara di PHP 8.0, kami memiliki:

    ‚Ä¢ Pembagian dengan nol: Pengecualian, dan fungsi fdiv() akan ditambahkan , memungkinkan pembagian dengan nol;
    ‚Ä¢ Argumen tidak valid di foreach (): Peringatan;

    Pelajari lebih lanjut di podcast. PHP Internals News # 29 c Nikita Popov.
  • Koordinasi Proyek PHP - Selama kami bersemangat di Internal, ada sesuatu yang baik juga. Dan Ackroyd membuat dokumen yang menjelaskan tugas dan diskusi proyek saat ini. Jika Anda ingin membantu PHP tetapi tidak tahu caranya, dokumen ini adalah titik awal yang baik.
