Di toko kami, penjualan perwakilan router pertama di bawah merek SNR sedang berjalan lancar. Router SNR-CPE-W4N (analog DIR-615) telah mendapatkan popularitas di antara para pelanggan kami dan telah banyak digunakan oleh penyedia komunikasi. Harga murah ditambah dengan fungsionalitas melakukan pekerjaan mereka. Baca lebih lanjut tentang SNR-CPE-W4N di ulasan kami. Di bawah potongan, saya ingin berbicara tentang kotak yang menggantikannya.

Mari kita mulai memperkenalkan mereka yang mengganti SNR-CPE-W4N dengan "kerabat" terdekat dengan nama yang sama. Namun, versi baru dari router SNR-CPE-W4N sangat berbeda dari pendahulunya, baik dalam penampilan maupun dalam pengisian.


Kasing perangkat yang ringkas (173x118x175 mm) tidak akan menyebabkan ketidaknyamanan dalam penempatan, baik di rumah maupun selama pembangunan jaringan kantor. Penutup case atas terbuat dari plastik putih dengan lapisan pernis. Tidak ada sidik jari atau goresan pada case, dan kotoran mudah terhapus. Logo SNR memamerkan di tengah.

SNR-CPE-W4N dilengkapi dengan dua antena eksternal dengan gain 5dBi, yang memastikan area jangkauan sinyal Wi-Fi cukup besar dengan transmisi data yang stabil di dalamnya. Router mendukung standar IEEE 802.11b / g / n dan teknologi MIMO2x2. Ini memungkinkan Anda untuk mencapai transfer data dengan kecepatan 300 Mbps. Pengisian perangkat didasarkan pada chipset MT7620N yang digunakan di banyak router Wi-Fi modern.


Router sangat cocok untuk pelanggan yang terhubung ke layanan IPTV. Mendukung Multicast, IGMP Snooping, serta Multicast-to-Unicast. Isolasi lalu lintas multicast / broadcast memungkinkan Anda untuk mencapai televisi IP berkualitas tinggi, bahkan pada klien Wi-Fi. Jika perangkat yang terhubung ke IPTV tidak mendukung Multicast, server DLNA xupnpd telah ditambahkan ke perangkat lunak router, yang memungkinkan Anda untuk mengunduh daftar putar penyedia langsung ke router dan menggunakannya sebagai server media.

Fitur Utama:

- Chip Mediatek MT7620N
- Wi-FI: 802.11b / g / n 2T2R
- 8MB Flash / 64MB RAM
- Dukungan IPv6
- Built-in xupnpd (DLNA)
- Perangkat lunak WIve-NG-mt
- Antarmuka web yang telah disertifikasi



Rekan terdekat SNR-CPE-W4N di jajaran router di bawah merek kami adalah versi dual-band SNR-CPE-MD1.



Ada lebih banyak peluang di sini, karena router dapat beroperasi dalam dua rentang frekuensi 2,4 GHz dan 5 GHz secara bersamaan pada kecepatan setinggi mungkin - hingga 733 Mbit / dtk. Kemampuan tersebut disediakan karena router yang mendukung standar IEEE 802.11b / g / n dan teknologi MIMO 2x2, serta 802.11a / ac.



Dimensi perangkat 187 x 143 x 35 mm memungkinkan Anda untuk menempatkannya tanpa mengurangi ruang kantor atau rumah. Area cakupan yang luas dan operasi sinyal yang stabil dipastikan dengan adanya dua antena 5dBi.

Dukungan untuk dua band memungkinkan tidak hanya menggunakan frekuensi 5GHz yang lebih bebas untuk transfer data, tetapi juga menerapkan skema di mana router wifi bertindak secara bersamaan sebagai klien nirkabel (mode APCli) pada satu frekuensi (misalnya, 5GHz), dan titik akses untuk perangkat akhir di lain (mis. 2.4GHz). Bekerja dengan layanan multimedia dilakukan di sini, seperti halnya "pasien" di atas.

Fitur Utama:

- Chip Mediatek MT7620A
- Chip Mediatek MT7610NE
- Wi-Fi: 802.11b / g / n 2T2R
- Wi-Fi: 802.11a / ac 1T1R
- 8MB Flash / 64MB RAM
- Dukungan IPv6
- Built-in xupnpd (DLNA)
- Perangkat lunak WIve-NG-mt
- Antarmuka web asli atau Russified



Kedua router mendukung IPv6, termasuk dual stack asli melalui IPOE / PPPOE dengan dhcp + ra, mode 6to4; layanan radvd dan dhcp6s, dll.



Perangkat lunak


Firmware kedua router didasarkan pada perangkat lunak Wive-NG-MT . Ini adalah pengembangan lebih lanjut dari perangkat lunak untuk router dari chip anggaran baru dari Ralink / Mediatek MT7620. Wive-NG-MT adalah sistem operasi tertanam yang dirancang khusus untuk NAG.







Kustomisasi


Serangkaian penawaran penyesuaian yang fleksibel telah dikembangkan secara khusus untuk operator telekomunikasi. Setiap router dapat "diasah" untuk operator tertentu. Perubahan dapat dilakukan seperti penampilan perangkat. Misalnya, siapkan kotak Anda sendiri dengan warna-warna operator.



Cara mengganti logo di perangkat





Perangkat lunak Wive-NG-mt memungkinkan Anda untuk mengubah init secara mandiri tanpa membangun kembali firmware menggunakan rwfs. Ini memungkinkan Anda untuk menyesuaikan router dengan karakteristik jaringan operator dan membuatnya mudah bagi pengguna untuk mengkonfigurasi router.

Untuk jaringan perusahaan, fungsionalitas pembentuk tersedia, roaming antar-titik yang dapat dikonfigurasi secara fleksibel, yang tidak memerlukan pengontrol eksternal, serta bekerja dalam mode Hotspot.
