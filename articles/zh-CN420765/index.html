双子座PDA的印象。 口袋双靴式收割机还是没用的玩具？



下午好，亲爱的哈布罗维特。

您是否知道二十年前遇到的现代小工具？ 您能想象2018年发布的点矩阵打印机还是CRT显示器吗？ 然而，英国公司Planet Computers的创意产生的第一个想法就是-来自过去的客人。 双子座PDA属于称为个人数字助理（PDA）的设备类别，在90年代末流行并在今天完全被人们遗忘。

如今，使用新设备令人惊讶不已，但是，具有机械qwerty键盘，两个USB-C端口，能够同时搭载Android，Linux和Sailfish OS并同时放在口袋中的功能的设备是一款真正的原始设备不是说有类似物。

小工具的描述以及两个月后的印象在切下列出。

这是怎么回事？


受经典Psion系列5启发的双子座以水平“翻盖”形式制成，其尺寸仍明显大于大多数智能手机（好于或几乎与Galaxy Note 9的尺寸相同）。


与第七代iPhone相比

Gemini有两种版本：有和没有SIM卡插槽。 两者的技术规格相同。

制造商网站上的完整技术规格

设备的右侧（顶部照片，从左到右）是麦克风，USB-C，语音助手/双启动按钮，扬声器。 左侧是扬声器，3.5毫米插孔，USB-C和麦克风。 内部-用于SIM和microSD卡的插槽。 两侧的扬声器和麦克风使您可以在音频对话期间将小工具固定在两侧。 此外，Planet Computers还为Gemini推出了许多配件，例如USB-C – HDMI适配器或带有以太网和USB-A端口的集线器。



从所有方面来看，双子座都可以归功于中端智能手机：6英寸的屏幕，移动ARM处理器，SIM卡插槽，毕竟是Android作为主要操作系统。 但是将其视为小型计算机会更正确。 我什至会这样说：电脑只要口袋大小就够了。

毕竟，如果您客观地看待事物，那么绝对适合您口袋中的所有设备专门为我们提供内容消费服务。 我们通过电话浏览Instagram提要，或者扭曲播放器上的曲目，或者在阅读器上翻转您喜欢的书的页面。 反过来，双子座最初是为那些创作内容的人构想的。 谁撰写博客，编辑文档，答复信件……或在上班途中通过电动火车远程连接到服务器。 在这里出现一个合理的问题...

如何实施？


双子座最初被认为是开源产品。 搭载Android 7.1作为标准操作系统，它正式支持Debian和Sailfish。 Planet Computers还提供了开源的引导加载程序代码，因此从理论上讲，将来受支持的操作系统集可能会更加广泛。 互联网上的魔术师已经看到了具有更具体任务的Gemini系统，例如Kali。

标准的Android也进行了一些更改。 其中最明显的是工具栏，Mac OS对此我们非常熟悉。



该面板完全灵活且可自定义：可以自由添加和删除任何应用程序，并更改其顺序。 随时可以通过按Alt键调出该面板，您可以使用标准键盘箭头或使用触摸屏浏览应用程序。



除其他事项外，键盘还可以执行许多小的“日常”任务：管理媒体播放器，更改亮度或音量。

Planet Computers还尝试确保我们所有的桌面习惯在Android上均能正常运行。 因此，例如，您可以使用Alt + Tab组合键（或相反方向的Alt + Shift + Tab键）在应用程序之间进行切换，通过按Enter键在Gmail中打开字母，然后使用Esc返回主菜单。 浏览器中的页面由箭头滚动，表单由标准选项卡切换。

但是，值得注意的是，并非所有熟悉的组合都有效。 您无法在YouTube中用空格暂停视频，就像用通常的Shift + Alt（或cmd +空格）切换字体一样。

当然是Linux。 Gemini支持双重引导，因此除了Android之外，没有什么阻止您安装Debian的。 或者代替。



不幸的是，对于Linux，我对设备最大的失望是连接。 首先，操作系统工作非常缓慢。 很好 其次，在6英寸的屏幕上使用台式机操作系统，甚至使用触摸屏也非常不方便-尝试在第一次尝试时用手指敲击正确的元素。



一方面，您可以将鼠标甚至外部显示器连接到Gemini，但是可移植性的想法完全消失了。 另一方面，可以合理地假设相当一部分用户将专门使用该终端，但是这里有一个问题。 在Android和Linux上，法语布局均打印“ µ”符号而不是字母“ m”。 如果首先通过安装最新更新解决了问题，则在Debian上无法以任何方式解决。 这是问题3的征兆-像任何新的Gemini产品一样，它并非没有出现软件失误和错误，但是开发人员似乎将所有精力都集中在一个OS（Android）上。 可以说，qwerty布局的工作没有任何缺陷。

结果，在几个晚上之后，Linux被废弃了。 幸运的是，Android上的Termux像时钟一样工作，并且可以通过物理键盘使用它。



哦，是的，键盘。

琴键


围绕物理键盘构建的设备仅需执行5次以上即可实现。 Gemini键盘优于任何一种屏幕键盘，但不幸的是，Planet Computers未能做到五分以上。

让我们从专家开始。 如果您将所有积极方面都纳入一句话，则值得一说：您可以在Gemini键盘上充分工作。 您几乎立即习惯了小巧的尺寸，并开始使用与台式机/笔记本电脑相同的打字方法（在我的情况下为六个手指）。 所有常用的文本键盘组合都可以正常工作-ctrl + c / ctrl + x / ctrl + v，移动到带有箭头和ctrl的行/段落的开头，以粗体/斜体/下划线突出显示，依此类推。 如上所述，Web表单由标准Tab切换，文档由ctrl + s保存。

由于所有这些，您可以在Gemini上编写文本和编辑文档。 不，我完全理解，只要有适当的愿望和毅力，您就可以在T9上重写《战争与和平》。 但就个人而言，无论是在屏幕键盘上还是在Blackberry键盘上，我都可以轻松键入多行内容，以进行消息，评论或回信。 而对于双子座来说，这只是肩膀上多字母的大文字。 不用说，这个帖子是完全写在上面的吗？



为什么所有这些都不能称为完美键盘？ 因为，由于某些缺点，使用它比可能要累的多。

键盘完全没有背光，因此在黑暗中使用非常困难。 我知道，我们在哈布雷（Habré）上的所有人都是盲目打字的大师，但双子座仍然采用非标准布局。 进入，尤其是空间变得很难，我个人有时不得不单击两次。 并且如上所述，非常缺少仍然不在设备中的键盘组合。

但是我对Gemini的主要抱怨是俄语布局。 当然，完全西里尔字母不适合，但这对任何人来说都不是新闻。 每个开发人员都以自己的方式解决此问题，但是Planet Computers的人员立即诉诸了几个非常有争议的决定。


yutsuken布局

1.使用可用键。

对于我来说，仍然是一个谜，为什么在缺乏物理按钮的情况下，双子座的创造者决定不使用最右边的键来放置字母。 为什么撇号是主要符号而不是“ g”？ 为什么不能将点和逗号放在同一键（空格右侧的键）上，从而为“ b”释放空间？ 这两个字母在字母中经常使用，例如，在此文本中，“ b”-178次，“ g”-115。将它们发送给其他选择，但不能这样做，是非常不合逻辑的决定。 此外，第二个问题使情况更加恶化。

2.为“替代”字母选择一个位置。

开发人员通常会采用以下两种解决方案之一，或者甚至同时采用两种解决方案：将相同的符号（“ w”和“ u”，“ b”和“ b”，“ e”和“ e”）放在同一按钮上，或者极右（例如，“ x”上的“ b”，“ f”上的“ e”等）。 两种方法都可以接受-您很快就习惯了一个按钮上的相似符号，而最右端的极右符号也不会造成太大麻烦。 至少他们没有带我去黑莓（Blackberry） 。

行星计算机公司采用了自己的特殊方式。 棘手的。 一些字母改变了这一行：“ u”上升了，“ e”下降了，“”移到了键盘的另一端。 其他人只是简单地异常移动，例如突然出现在该行中间的“ x”和“ b”。 即使使用了两个月，我有时也会按下错误的按钮。

3.一般将符号弄乱。

我很清楚，俄罗斯市场的本地化远不是初创公司的头等大事。 但是，毕竟，大多数缺陷都可以由说俄语的用户在半小时内发现。 为什么字母中根本没有使用的“七个”用大写字母“＆”代替问号？ 为什么字母“ u”重复两次？ 而正斜杠在哪里？

“为什么不创建自定义键盘？” -你合理地问。 问题在于物理键盘仅适用于Gemini布局。 标准Android或自定义布局仅适用于虚拟键盘，而物理键盘则切换为默认语言（英语）。

随着时间的流逝，我学会了或多或少地容忍缺点，或者也许我已经习惯了它们。 但是，仍然很难接受的是，用俄语写作时要实现自动操作非常困难，并且按一些组合键更像是手指上的瑜伽。

总体印象


在我看来，双子座不只是智能手机，还更是笔记本电脑。 我们所有的“笔记本电脑”功能-上网，处理文档或观看视频-都可以很轻松地完成。



使用智能手机组件，情况更加复杂。 大多数现代移动应用程序都设计为在肖像模式下工作。 在水平位置使用它们非常不方便或根本不可能。 双子座本人转过90°，看上去简直荒谬。



但是在我看来，2018年对PDA市场感兴趣的人不会将Instagram放在使用的应用程序的顶部。

双子座爆炸地应付目标。 您可以在其上连续打印两个小时（公平地说，在许多评论中，人们仍然抱怨使用该设备长期工作会感到疲劳/不便），因此Termux等应用程序的生产率显着提高。 加上非常宽容的电池寿命，而且我一天都没有成功种植它，因此该设备确实需要做大量工作。 键盘缺点不是很关键，迟早会解决软件问题。 但是，有一个陷阱...

谁需要它？


通常，当我们谈论小工具时，主要标准是：他应得的钱吗？ 但事实是，Gemini PDA是一种众筹产品，在Indiegogo筹集了将近200万美元用于其实施。 订购该设备的成千上万的人甚至在不熟悉该设备之前就给了自己这个问题的答案-值得。

在对新的小工具发表意见时，我总是会记得十年前听到的一句话：“设备需要解决应用中的问题，而不是解决设备本身。” 但是事实证明，我只是没有足够的申请给Gemini。

在我的口袋里，我已经有一部电话可以完美满足大部分日常需求。 是的，我经常旅行，写日记已经十二年了，比起飞机上的电视节目，我更喜欢看书。 但是，另一方面，如果我有一个背包，那么您已经可以在其中放一台平板电脑了。 如果我购买双子座的299美元的众筹价格似乎不太高，那么当前499美元/ 599美元的价格将与iPad处于同一价格类别。

结果，我对设备的看法是这样-坚实的四分，但是您需要确切地知道为什么需要它。 这就像厨房里的榨汁机-如果您每天早上都榨汁，那么购买绝对值得。 但是毕竟，大多数人在购买后的第一周内使用了三遍，之后她只占了一个地方并收集了灰尘。

PS：当然，我有兴趣了解有关双子座的habrasociety的观点，以及有关该设备在2018年作为一门课程的适当性的观点。 这样的事情会派上用场吗？ 你会偷偷地渗透吗？ 您是否撰写了有关工作方式的报告？ 还是不需要礼物？
