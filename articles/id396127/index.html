Microsoft telah mengumumkan model distribusi baru untuk Windows 10, sekarang dengan berlangganan

Apa yang dicurigai dan dibicarakan pada akhirnya terjadi, Microsoft pada konferensi Mitra Sedunia mengumumkan pengenalan model distribusi baru untuk Windows 10 - sekarang pengguna korporat akan bisa mendapatkan Windows 10 Enterprise E3 melalui layanan berbasis cloud Penyedia Solusi Cloud dengan berlangganan. Biaya opsi ini adalah $ 7 per bulan untuk satu tempat kerja.Selain itu, tidak hanya perusahaan besar, tetapi juga usaha kecil dan menengah akan dapat menggunakan versi Enterprise dari sistem operasi. Menurut perwakilan Microsoft, ini akan memungkinkan usaha kecil dan menengah untuk meningkatkan daya saing mereka karena kemungkinan menggunakan perkembangan keamanan yang sebelumnya hanya tersedia untuk perusahaan besar, serta mengoptimalkan biaya perangkat lunak mereka, sehingga memungkinkan untuk merespon secara fleksibel terhadap perubahan dalam struktur perusahaan, jumlah karyawan dan tidak membekukan dana untuk pembelian lisensi satu kali.
Sejauh ini, perubahan-perubahan ini telah diumumkan hanya untuk sektor korporasi, seperti untuk pengguna individu, di sana model baru tidak akan diterapkan di sana, meskipun kemungkinan besar, setelah break-in dengan klien korporat, itu juga akan diperkenalkan di segmen pasar ini.
Langkah serupa, meskipun sejauh ini hanya di segmen korporat, menunjukkan bahwa Microsoft secara bertahap bergerak menuju menjadikan sistem operasinya sebagai layanan. Solusi semacam itu sangat logis bagi perusahaan, dan memiliki sejumlah keunggulan, seperti pendapatan yang stabil dan konstan yang akan dihasilkan oleh skema berlangganan, lebih banyak kontrol atas pembaruan sistem operasi oleh perusahaan, kemampuan untuk mengurangi pembajakan, karena dalam kasus skema berlangganan Kami membutuhkan verifikasi terus-menerus tentang validitas berlangganan, dan sebagainya. Ada keuntungan untuk bisnis yang merupakan pelanggan pertama dari program ini. Di sisi lain, untuk pengguna individu, berbeda dengan pengguna korporat, kerugian dari model berlangganan persis sama dengan keuntungan untuk Microsoft itu sendiri - kebutuhan untuk terus-menerus membayar berlanggananmengikat ke server perusahaan, dan sebagainya.
Keberhasilan model ini di pasar individu akan tergantung terutama pada strategi yang akan dipilih Microsoft, tetapi kemungkinan besar langganan akan datang ke sini cepat atau lambat, strategi yang paling logis mungkin adalah penggantian bertahap lisensi OEM untuk layanan berlangganan, tetapi sejauh ini perusahaan tetap mempertahankannya. diam, jadi kemungkinan besar, penampilan berlangganan ke sistem operasi di segmen pasar ini masih tidak terlalu cepat, tetapi mungkin tidak bisa dihindari.
