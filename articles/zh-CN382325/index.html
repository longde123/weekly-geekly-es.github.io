Windows 10的第一天：1400万次安装和一些惊喜



微软报告成功：在第十版新Windows发行的第一天，它已安装在1400万台计算机上。在几周之内，所有想要升级操作系统第7版和第8版所有者的人都可以使用自动更新。同时，Microsoft发布了具有最近几天发现的错误修复程序的补丁程序。

Windows 10是Microsoft免费提供的以前版本的合法所有者的操作系统的第一个版本。这与“十大”以来的“十大”非常“终结”这一事实表明，用户更新系统的速度将比以前版本的OS快得多。

在等待系统新用户的惊喜中，可以注意到以下几点。

现在，更新（补丁）由操作系统决定下载-系统不再根据用户的要求提供下载。也许这样做是为了提高用户本身的安全性。但是经验丰富的用户将可以通过组策略（gpedit.msc）进行配置。

Windows版本的下载是使用peer2peer系统完成的，因此Microsoft 可以避免服务器超载。该系统由BitTorrent协议首次引入，长期以来已被多家公司用于分发数字内容（例如，用于购买的数字版本的游戏）。如果由于某种原因您感到对您的传出流量感到抱歉，请在设置中禁用此功能。



Mozilla基金会报告了 令人不快的意外，该基金会正在开发Firefox浏览器等产品。Windows 10引入了用于各种任务的默认程序分配系统，并且现在还需要在特殊菜单中配置默认​​浏览器。



此设置既可以在安装过程中进行，也可以在操作过程中随时进行。同时，浏览器本身不能再使其成为默认浏览器-显然出于安全原因。如果用户同意将浏览器设置为主要浏览器，则系统将打开一个用于默认程序设置的常规菜单，用户必须在该菜单中独立找到“ Web”部分并将程序更改为所需的程序。

这种情况激怒了Mozilla团队，并作为博客，并在致微软的公开信中，基金总监克里斯·比尔德（Chris Beard）称此举“具有侵略性”，并呼吁简化浏览器更改程序，以使他对用户的态度更加公平。作为响应，Microsoft概述了这种情况，即用户可以自行更改这些设置，如果公司收集的OS使用情况统计信息告诉她需要更改界面，那么她会这样做。
