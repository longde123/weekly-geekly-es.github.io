Versão final do Windows 1809 lembrada

Devido a problemas conhecidos ao excluir arquivos de um perfil durante uma atualização, a Microsoft decidiu revogar a versão final da Atualização do Windows 10 de outubro de 2018 (1809)

Desde 6 de outubro, o assembly não foi distribuído pelo serviço Windows Update. Se você baixou a imagem ISO - não a instale e aguarde a nova montagem.

As versões atuais do Windows Server 2019 e Windows 10 Enterprise (incluindo LTSC e IOT) também são afetadas por esse problema.
