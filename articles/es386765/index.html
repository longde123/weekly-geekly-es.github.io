Windows tiene 30 años


Bill Gates presenta un nuevo sistema (Windows 3.0 en este caso) El

tiempo es muy rápido, y es especialmente bueno seguir el progreso del tiempo usando varios dispositivos y plataformas de software como ejemplos. Hace exactamente 30 años, Microsoft lanzó la primera versión de Windows como sucesora de MS-DOS. Sucedió el 20 de noviembre de 1985. El lanzamiento de este sistema operativo marcó el final de una era y el comienzo de otra. A pesar del hecho de que durante 30 años la interfaz del sistema operativo Windows ha cambiado mucho, muchos elementos han permanecido desde la primera versión. Estos son cuadros de diálogo, barras de desplazamiento, iconos, aplicaciones como el bloc de notas y Paint.

Además, Windows 1.0 se hizo amigo del mouse. MS-DOS necesitaba ingresar comandos, y no todos los usuarios hicieron frente a esto. Como si fuera un sistema no enteramente para amas de casa. Después de que el sistema operativo Microsoft comenzó a admitir el mouse, la forma en que las personas y la PC interactuaron cambió radicalmente. Al principio, muchos usuarios del nuevo sistema incluso se quejaron de que tenían que trabajar demasiado con el mouse, en lugar de escribir comandos desde el teclado. Pero luego se acostumbraron.

El primer sistema operativo Windows en 1985 requería 256 KB de memoria, una tarjeta gráfica y dos disquetes. Si el usuario quería ejecutar varios programas simultáneamente, también era necesario agregar un disco duro con 512 KB de memoria. Estas especificaciones básicas fueron solo el comienzo, después de lo cual comenzó el rápido desarrollo de las PC, con el desarrollo simultáneo de plataformas de software.
