9 extensões úteis de navegador para desenvolvedores (lista para 2020)

Como simplificar sua vida com plugins de navegador.

Desenvolvedores da web complicados podem usar o navegador no nível do modo deus. Extensões (plug-ins, complementos) são extremamente úteis quando se trata de melhorar o ambiente de trabalho e melhorar o desempenho da escrita de código.

2020 não está longe. Muitas novas extensões apareceram. Compilei as extensões mais úteis para desenvolvedores da Web e as coloquei em uma lista.

Github refinado


O Github refinado simplifica a interface do GitHub e adiciona aprimoramentos úteis.


Github refinado

“Esperamos que o GitHub observe como essas melhorias são necessárias e adicione algumas delas. Portanto, se você gosta de alguma dessas melhorias, escreva para o suporte do GitHub sobre isso. "
- Os criadores do Github refinado.

Guia Hacker


A guia Hacker é útil se você quiser aprender sobre projetos populares de código aberto. Cada vez que você abre uma nova guia, você vê uma lista de todos os novos projetos. Você pode alternar entre agendas semanais, diárias e mensais, bem como alternar linguagens de programação.


Substitua a tela da nova guia do navegador pelos projetos de tendências do GitHub.

Diário 2.0 - Um recurso para desenvolvedores ocupados


Para ser um desenvolvedor de software brilhante, você precisa se esforçar para aprender mais e manter-se atualizado com as tecnologias mais recentes.


Daily 2.0

Novas e atualizadas tecnologias de desenvolvimento de software aparecem na Internet todos os dias. Manter-se atualizado é um enorme problema. Daily foi criado por desenvolvedores para desenvolvedores. Com esta extensão, podemos nos concentrar no código, em vez de procurar constantemente notícias sobre desenvolvimento na Internet.

Sem moeda


Existem sites que forçam nossos processadores a minerar criptomoedas para eles. No Coin é uma pequena extensão do navegador. É necessário bloquear mineiros como o Coinhive, que usam os recursos do seu computador sem o seu consentimento.


Nenhuma moeda é confiável por milhares de usuários. Esta extensão fornece uma maneira segura e confiável de impedir que os mineiros usem seu processador. Às vezes, você precisa inserir um captcha ou link curto que exija mineração. Nenhuma moeda permite, neste caso, cancelar o bloqueio de mineradores por um tempo limitado.

  • para firefox

Formatador JSON


Esta extensão torna o JSON mais legível. JSON Formatter é um projeto de código aberto.


Formatador JSON

O JSON Formatter adiciona a capacidade de executar ações com texto JSON estruturado. Ao contrário de outros aplicativos semelhantes, o JSON Formatter simplesmente estrutura o texto. Esta extensão não adiciona visualizadores, cores ou recursos de compactação.

Temporizador Pomodoro



Extensão do temporizador Pomodoro

Técnica Pomodoro - método de gerenciamento do tempo. Esta técnica original consiste em seis etapas:

  1. Decida a tarefa que você fará.
  2. Inicie o cronômetro Pomodoro (geralmente cerca de 25 minutos).
  3. Trabalhe na tarefa sem se distrair com nada.
  4. Pare de trabalhar quando o cronômetro tocar e marque a caixa em um pedaço de papel.
  5. Se você tiver menos de quatro marcas de verificação, faça uma pequena pausa (de 3 a 5 minutos) e prossiga para a segunda etapa.
  6. Após cada 4º “tomate” faça uma longa pausa (15 a 30 minutos). Reduza o número de ticks para zero e prossiga para o primeiro ponto.

Enquanto você lê este artigo, estou usando o Pomodoro. Pomodoro é uma maneira prática de colocar as coisas em ordem. Ele ajuda a aprender a se concentrar e gerenciar seu tempo. Se você quiser aprender a se concentrar, recomendo este livro . Ela me ajudou.

Inspetor de Meta SEO


Normalmente, não vemos metadados quando usamos a Internet. Essa extensão ajuda a descobrir e verificar os metadados encontrados nas páginas da web.


Verifique suas metatags para garantir uma boa classificação de SEO

A extensão é usada principalmente por desenvolvedores da web que precisam verificar o código HTML do site, de acordo com as recomendações do Google para webmasters. Também é útil para quem deseja ver o conteúdo de uma página, que geralmente não é visível.

Os alertas são exibidos quando os metadados não atendem a determinados intervalos - quando a tag de descrição é muito curta ou muito longa.

SEO é uma linguagem separada. Eu recomendo a leitura do livro “3 Meses a Nº 1: O Manual de SEO“ Sem disparates ”de 2019 para obter seu site encontrado no Google . ” Possui boas práticas e dicas de SEO.
