Notícias e Lançamentos


  • PHP 7.3.0 alpha 4 - O último alfa do loop. A primeira versão beta está prevista para 2 de agosto. À lista de novos recursos , será adicionada uma proposta recentemente aceita para introduzir novas funções array_key_first() / array_key_last() para trabalhar com matrizes: 

     $array = ['a' => 1, 'b' => 2, 'c' => 3]; $firstKey = array_key_first($array); // 'a' $lastKey = array_key_last($array); // 'c'

    A fase de votação passou na proposta Deprecations for PHP 7.3 , na qual vários recursos foram preteridos. Também obsoleto no PHP 7.3 serão declaradas constantes que não diferenciam maiúsculas de minúsculas .

    Quanto à proposta de propriedades digitadas , foi decidido adiá-la para a próxima versão, que provavelmente se tornará PHP 8.0.
  • Atualizações de todas as ramificações atuais do PHP com correções de erros de segurança:

    • PHP 7.2.8
    • PHP 7.1.20
    • PHP 7.0.31
    • PHP 5.6.37
  • Pré - visualização pública do PhpStorm 2018.2 - Entre as inovações: uma interface atualizada e suporte para a barra de toque, aprimoramento do preenchimento automático levando em consideração os espaços para nome, pesquisa e substituição estrutural, instalação simplificada do CodeSniffer / MessDetector e outras melhorias.

Internos em PHP


  • Visibilidade do espaço para nome [RFC] para classe, interface e característica - Propõe-se introduzir modificadores de acesso para classes, interfaces e características para limitar o uso do espaço para nome:

    Texto oculto 
     namespace Example { public class A { private $property; } protected class B { public $property; } private class C { protected $property; } } namespace OtherVendor { public class Factory { public function A() { return new \Example\A(); // Allowed by public } public function B() { return new \Example\B(); // Not allowed because // namespace is not shared } public function C() { return new \Example\C(); // Not allowed because // not from same namespace } } }
