机器人技巧＃42：微笑，人们会为您服务


无论我们是否想要，我们周围的世界都在不断变化。 技术在其中起着重要作用，技术的开发和传播每年都越来越活跃。 我们不再将汽车视为稀有商品，而是将移动通信视为魔术。 技术已经成为我们日常生活，工作和我们自己的一部分。 根据《终结者》的电影，机器人是在未来学家和科幻小说作家中最受欢迎的主题之一，它是使人的生活更轻松或对地球上所​​有生物进行灭绝种族的自动机器。 无论如何，仍然存在许多非常有趣的问题-一个人对机器的行为如何，他会相信它，他们的合作将如何有效？ 美国陆军研究实验室的科学家决定进行一系列实验，以确定自主机器的行为和“外观”的哪些方面改善了与人类的协作。 我们从他们的报告中学到了什么。 走吧

学习基础


为了更好地理解人与机器之间的关系，您应该首先查看人与人之间的关系。 鉴于地球上的人口分为许多组（宗教，国籍，种族等），一个人总是将人们分为“我们”和“他们”。 根据研究人员的说法，这种分离过程是在潜意识水平上进行的，导致选择属于同一群体的人进行合作的可能性增加。 科学家们还注意到，这些偏好与歧视其他民族，宗教和其他事物的人的表现无关。

在自主机器的情况下，尚不清楚在人与机器的合作时是否进行这样的社会分类。 如果是的话，它将如何体现，如何受到监管以及如何进行改变？

为了测试这一点，科学家进行了一系列实验，参与者来自两种遥远文化（美国和日本）的人，以及带有对应于相同文化并表达某些情感的虚拟面孔的汽车。

来自美国的468位参与者和来自日本的477位参与者。

来自美国的一群人的人口统计学特征如下：

  • 性别：63.2％-男性和36.8％-女性；
  • 年龄：0.9％-18-21； 58.5％-22-34; 21.6％-35-44; 10.7％-45-54; 6.2％-55-64; 2.1％-64岁以上；
  • 种族：77.3％-印欧语系; 10.3％-非洲裔美国人； 1.3％-东印度人; 9.2％-西班牙裔 6.0％-亚洲（东南）。

该小组来自日本的人口统计学特征如下：

  • 性别：67.4-男性和32.6％-女性；
  • 年龄：0.6％-18-21； 15.7％-22-34; 36.4％-35-44; 34.7％-45-54; 10.4％-55-64； 2.1％-64岁以上；
  • 种族：0.6％-东印度人； 99.4％-亚洲（东南）。

人们可能会根据个人分类以不同的方式相互联系，但值得记住的是，自主机器不是人。 因此，默认情况下，此设备已经属于另一个社交组。 当然，一个人可以将机器视为社交角色（参与某种形式的交互），但是与从肉体和鲜血中看到另一个在世的人相比，他的感知方式有所不同。

一个明显的例子是在“石头剪刀布”游戏中对人脑的早期研究。 当一个人与另一个人玩耍时，内侧前额叶皮层被激活，这在心理化过程中起着重要作用* 。
心理化* -表现自己和他人心理状态的能力。
但是，在使用机器玩耍时，大脑的这一部分未被激活。

最后通game游戏的实验显示了相似的结果。
最后通game游戏*是一种经济游戏，其中第一号玩家收到一定数量的钱，并且必须与自己第一次看到的第二号玩家分享。 要约可以是任何要约，但如果2号玩家拒绝了，则两个玩家都将不会收到任何钱。
当2号玩家从1号玩家（人类）收到不公平的要约时，他激活了大脑中叶的各个部分，这些部分负责形成负面情绪。 如果玩家1是汽车，则不会观察到这种反应。

这样的实验再次证实了一个人对机器的感觉不同于另一个人的事实。 一些研究人员认为，从本能的角度来看，该机器并不代表我们的竞争，因此某些大脑区域的活动减少了（甚至根本没有）。

一切都是很合乎逻辑的：人与人之间是狼，但即使在狼之间，在一定条件下也存在一定程度的信任和渴望合作。 但是汽车就像狐狸一样，狼根本不想与之合作。 但是，这样的偏好并不能消除世界上越来越多的机器人这一事实，因此，我们需要找到改善我们与它们之间关系的方法。

为什么研究人员决定将这种类型的社会分类用作文化认同？ 首先，早先已经确定一个人对具有某些语音特征（例如强调），社会规范和种族的机器做出积极反应。

其次，在不同社会群体的代表参加实验的情况下，在第一阶段就可以追溯到合作与竞争的独特交织。 在随后的阶段中，当参与者更好地了解彼此时，便会根据经验任务划清一条清晰的行为线。 简而言之，即使经验任务与合作有关，来自不同社会群体的参与者最初也不相互信任。 但是一段时间之后，信任和完成任务的愿望超过了潜意识的不信任。

第三，先前的实验表明，如果我们诉诸更具情境意义的迹象（情绪），则可以拒绝基于社会分类进行合作的最初期望。

也就是说，研究人员为机器赋予了实验参与者（人）所属的社会群体的视觉特征以及许多情感表达，然后检查了不同的组合以识别联系。

从所有这一切，确定了两个主要假设。 首先是文化认同积极迹象的结合可以减轻人们默认对机器的不利偏见。 其次，基于文化特性的情感表达可能会超出合作/竞争的期望。 换句话说，如果您赋予汽车某个特定文化群体的视觉特征，那么来自同一群体的人们可以在下意识上更加信任它。 但是，如果表现出某种情感，表明机器不愿进行合作，那么所有文化偏见立即就不再重要。

奇怪的是，实验是基于囚犯的困境-一种合作不是成功的战术的游戏。 按照游戏理论，囚犯（玩家）将始终最大化自己的利益，而无需考虑其他玩家的利益。 因此，理性应该促使玩家背叛和不愿合作，但实际上，这种实验的结果往往远非理性，这解释了在这个数学游戏的名称中存在“困境”一词。 有一个小问题：如果一个玩家认为第二个玩家不会合作，那么他也不应该合作，但是，如果两个玩家都遵循这样的策略，那么两个玩家都会输。

实验结果



图片编号1

总共，每个实验参与者都花了20轮与机器玩游戏。 上图是增益矩阵。 游戏中获得的积分具有真正的财务利益，因为它们随后被转换为中奖金额为30美元的彩票。

该实验的主要目的是确定参与者如何与人或机器协作/竞争，以及文化认同和情感如何影响这些过程。


图片编号2

上面是表示不同情绪（中立，悲伤，遗憾和喜悦）的机器虚拟面孔的两种文化身份（美国和日本）。 值得注意的是，实验中的所有参与者都使用了虚拟面部，即 人和汽车。

情感系列并非偶然。 较早的关于囚徒困境的实验表明，某些情绪可以鼓励合作，而另一些情绪则可以竞争，具体取决于表达的时间。

在此模型中，机器的情绪以及引起情绪的行为被分为两个主要类别。

推动一个人竞争：

  • 相互合作后感到遗憾（机器错过了为自己的利益而操作人类成员的机会）；
  • 手术后的喜悦（参与者合作，但从车中得到的回报为零）；
  • 相互竞争后的悲伤/相互合作后的中立情绪。

鼓励人们合作：

  • 相互合作的喜悦；
  • 对剥削感到遗憾；
  • 相互竞争后的悲伤。

因此，在实验中，同时存在多个变量：文化身份（美国或日本），宣布的伴侣类型（人或机器）和表达的情感（竞争性或协作性）。

根据参与者对的类型，实验结果分为两类：一个文化群体的参与者和不同文化群体的参与者。


图片编号3

上图是游戏中不同文化群体聚集在一起的实验结果（方差分析，即ANOVA）的图表。 不出所料，人们与人的合作（M = 41.15，SE = 2.39）比与机器的合作（M = 34.45，SE = 2.25）表达得更多。 关于情绪，一切都不可预测-与竞争情绪（M = 29.34，SE = 2.70）相比，玩家更经常与表达合作情绪（M = 45.42，SE = 2.97）的人合作。 中性情绪（M = 38.63，SE = 2.85）也优于竞争性情绪。

令人好奇的是，激发对人类玩家信心的情绪完全抵消了他们在机器上玩游戏的事实，从而减少了不信任的程度。


图片编号4

当参与者遇到来自同一文化团体的伙伴时，情况就类似。 合作（M = 45.55，SE = 2.61）的情绪胜过中立（M = 37.42，SE = 2.81）和竞争性（M = 28.99，SE = 2.78）。

结果的整体清楚地表明，一个人与另一个人在与机器的通信中使用相同的信任和不信任心理参数。 最初，自动驾驶汽车被视为与该人无关的社会团体的成员。 但是，使用具有适合人类伴侣的某种文化特征的视觉面孔可以减轻对机器的不信任。

最重要的观察结果是，正确的（可以说）情感完全可以使文化认同和属于机器的事物变平。

要更详尽地了解这项研究的细微差别，建议您研究一下科学家的报告 。

结语


这项研究非常模棱两可，但却向我们展示了人类心理学的重要特征。 我们在不知不觉中将周围的人归为某些社会群体，对属于我们群体的人们越来越信任。 这是我们的主要反应。 但这不是唯一的，因为当与人交流时，我们开始接收到许多由情感，言语，声音等表达的信号（视觉和声音）。 这些信号补充了一个人对我们的代表的印象，在做出决定时（无论是否信任），我们都会被排斥。 换句话说，在进行生动活泼的交流时，我们头脑中发生的社会分类并不重要。 这就是古老的谚语所说的：“他们受到衣服的欢迎，但受到思想的护航”和“不要凭封面判断一本书”。

关于汽车，一个人总是在担心和不信任的情况下进行调教，而这通常并没有明显的理由。 这就是人的天性-最初拒绝外来的东西。 然而，正如知识克服无知一样，与能够表达情感的自主机器的实际交互也会克服人类的偏见。

这是否意味着未来的所有机器人都应该充满情感？ 是的，没有。 我们今天检查了他们的工作的研究人员认为，在机器人的头几场情绪中，它将是一项出色的功能，可以加速人类社会采用自动机的过程。 我要说的是和狼一起生活-无论听起来多么粗鲁，都像狼一样how叫。

此外，我们已经赋予了机器人自己的特质：我们使机器人的眼睛像我们一样，像我们的脸一样赋予他们声音和情感。 一个人将始终试图使汽车人性化，正如本研究表明的那样，这并非徒劳。
