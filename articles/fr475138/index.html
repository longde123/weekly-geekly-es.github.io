5 façons d'utiliser efficacement le Raspberry Pi Deuxième partie

Salut, Habr.

Dans la première partie , 5 façons d'utiliser le Raspberry Pi ont été discutées. Le sujet s'est avéré assez intéressant, et aujourd'hui j'examinerai plusieurs autres options sur la façon d'utiliser ce micro-ordinateur avec avantage.


Photo de learn.adafruit.com

Comme dans la partie précédente, je considérerai les méthodes qui ne nécessitent pas de programmation.
Pour ceux qui sont intéressés, continue sous la coupe.

1. Caméra de surveillance



Source: www.raspberrypi-spy.co.uk/2017/04/raspberry-pi-zero-w-cctv-camera-with-motioneyeos

Le Raspberry Pi peut être utilisé avec presque toutes les caméras de vidéosurveillance.
Avec Raspberry Pi peut fonctionner:

  • Webcams USB (par exemple Logitech C910)
  • Caméras IP (Axis, etc.) avec un injecteur PoE (une alimentation 48 V est fournie à ces caméras via un câble réseau, ce qui permet de les transporter à l'extérieur du bâtiment)
  • caméras se connectant directement au connecteur RPi (comme sur la photo ci-dessus).

Il existe plusieurs options de configuration logicielle. Vous pouvez utiliser le package Motion , qui a des paramètres assez flexibles. Vous pouvez écrire directement depuis la console via ffmpeg, vous pouvez enfin écrire votre gestionnaire en utilisant Python et OpenCV. Vous pouvez diffuser un flux vidéo, utiliser un détecteur de mouvement, envoyer des photos par e-mail, etc.

Les personnes intéressées peuvent voir les tutoriels suivants:

  • www.bouvet.no/bouvet-deler/utbrudd/building-a-motion-activated-security-camera-with-the-raspberry-pi-zero
  • hackernoon.com/spy-your-pet-with-a-raspberry-pi-camera-server-e71bb74f79ea
  • medium.com/@gonzalovazquez/raspberry-pi-and-motioneye-setting-up-your-own-video-surveilance-95444e0faad

Important : dans la partie précédente, il a déjà été mentionné, mais il vaut mieux répéter. Pour toutes les tâches gourmandes en ressources (y compris le traitement vidéo) sur le Raspberry Pi, une alimentation de marque 2,5A de haute qualité est requise et un dissipateur thermique passif sur le CPU est souhaitable (vous pouvez l'obtenir à bon marché en Chine pour 1-2 $, en tapant le dissipateur thermique Raspberry Pi dans la recherche). Sinon, l'appareil peut se figer, des erreurs de copie de fichiers, etc. peuvent apparaître.

2. Enregistrement audio




Avec un microphone USB, le Raspberry Pi peut être utilisé comme un périphérique assez compact pour l'enregistrement audio. Ici encore, il existe de nombreuses options pour configurer un logiciel - vous pouvez écrire des fichiers localement sur une carte SD, vous pouvez diffuser sur un autre PC ou faire une diffusion sur le réseau.

Quelques tutoriels à répéter:

  • iltabiai.github.io/2018/03/17/rpi-stream.html
  • blog.mutsuda.com/raspberry-pi-into-an-audio-spying-device-7a56e7a9090e
  • www.heidislab.com/tutorials/audio-streaming-with-raspberry-pi-zero-w-and-adafruits-i2s-mic

Soit dit en passant, avec un microphone, le Raspberry Pi peut être utilisé avec Amazon Alexa et utiliser l'appareil pour les commandes vocales.

3. Prof. photographie


Ne confondez pas n3 et n1. Le premier paragraphe traite des caméras de surveillance, mais le Raspberry Pi peut également contrôler des caméras professionnelles de Canon, Nikon, Sony, etc. Il suffit de connecter l'appareil photo au Raspberry Pi via USB.


Photo de www.movingelectrons.net/blog/2017/08/09/Camera-Time-lapse-Controller-with-Python-and-Raspberry-Pi.html

Les bibliothèques gphoto2 et libgphoto2 ont à la fois la possibilité de travailler à partir de la ligne de commande et des interfaces pour Python et C ++, ce qui vous permet d'utiliser le Raspberry Pi pour contrôler le «miroir», par exemple, pour la photographie en accéléré. La liste des caméras prises en charge est assez longue et couvre presque tous les modèles, du moderne à l'ancien il y a 10 ans. Libgphoto2 possède une API assez développée et peut non seulement contrôler l'obturateur, mais aussi modifier les paramètres, télécharger des fichiers, etc.

Répétez les didacticiels:

  • pimylifeup.com/raspberry-pi-dslr-camera-control
  • medium.com/@cgulabrani/controlling-your-dslr-through-raspberry-pi-ad4896f5e225

Soit dit en passant, vous pouvez écrire des images sur la carte mémoire de l'appareil photo ou directement sur le Raspberry Pi, ce qui permet, par exemple, de les télécharger automatiquement sur le «cloud». Il existe également des bibliothèques pour contrôler non seulement les appareils photo reflex, mais également les caméras astronomiques (par exemple ZWO ASI), y compris même le guidage automatique .

4. Station météo


Raspberry Pi "sait comment" non seulement exécuter des programmes Linux, mais possède également des périphériques assez développés - série, I2C, SPI, GPIO. Cela rend l'appareil presque idéal pour collecter et envoyer des données à partir de divers capteurs - des capteurs de température et d'humidité à un dosimètre basé sur un compteur Geiger.


Photo de www.raspberrypi.org/blog/build-your-own-weather-station

Soit dit en passant, si vous êtes complètement paresseux , vous pouvez prendre des données non seulement de vos capteurs mais aussi de Web-A, cette option a également le droit d'exister. Cependant, une carte avec des capteurs pour le Raspberry Pi est facile à acheter séparément .

Tutoriels pour apprendre:

  • www.raspberrypi.org/blog/build-your-own-weather-station
  • www.instructables.com/id/Build-a-Raspberry-Pi-SUPER-Weather-Station
  • www.cooking-hacks.com/documentation/tutorials/geiger-counter-radiation-sensor-board-arduino-raspberry-pi-tutorial

5. console de jeux




En utilisant le projet RetroPie , vous pouvez transformer le Raspberry Pi en un émulateur «rétro» pour différentes consoles de jeu, d'Atari à Gameboy ou ZX Spectrum. Vous pouvez également acheter divers étuis, joysticks, etc.

Je suis loin des jeux, donc je ne peux pas en dire plus, tout le monde peut l'essayer par lui-même. Quelques tutoriels pour apprendre:

  • www.pcworld.com/article/3190347/how-to-build-a-raspberry-pi-retrogaming-emulation-console.html
  • thewirecutter.com/reviews/raspberry-pi-game-console
