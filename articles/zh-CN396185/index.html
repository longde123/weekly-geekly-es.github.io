5个自动驾驶仪传感器及其（迄今为止）解决的难题



审查自动驾驶仪问题的第一部分进展顺利，时间尚可。生活并没有停滞不前，特斯拉仍在挣扎（特斯拉Model X也在努力，但到目前为止没有自动驾驶仪的帮助）。ADAS传感器有什么问题？为什么在未来几年内没有任何变化呢？

令人惊讶的是，许多人接近了他们的身体能力。可以自信地说，因为以下列出的系统不仅在汽车中使用，而且在汽车世界之外有广阔的市场。这是一笔巨款，研发费用高达数十亿美元，而且进展缓慢。

现在更详细。

传感器的类型-如手指上的五种：
  • 摄影机
  • 雷达
  • 激光雷达
  • 超音波
  • 红外摄像机。

选择该分类是为了方便而不是遵守技术真理。通常，您可以通过不同的方式共享：
  • 被动式（摄像机）和主动式（激光雷达），
  • 在可见光范围内（相机），而不是在所有其他范围内（包括红外），
  • 按频率（从超声波到激光雷达）。

所有这些传感器本质上都非常不同，并且可以为电子大脑提供各种信息。

了解整个市场重要的是什么？



估计有所不同，但顺序是相同的。余额偏向于摄像机和雷达，其中两个占据了货币市场的85％。激光雷达仍然是一个利基解决方案（仅在没有突破的情况下），而超声波传感器不能超越其利基和突破，这也是不可见的。即使在激光雷达的背景下，夜视系统也会丢失。

现在更详细。



卡梅拉


它的工作方式是：接收可见范围内的世界图片，进行处理（单个帧和序列），然后识别。

优点：
  1. 只有摄像机才能识别道路标记和标志。
  2. 摄像机具有良好的角度分辨率，也就是说，摄像机可以很好地“理解”在汽车上移动的人（行人横过马路）。
  3. 根据摄像机的图片，可以对物体进行直接分类（“狗”-“儿童”-“成人”-“自行车”-“汽车”-“卡车”）。
  4. 而且，当然，价格适中的相机-从几百美元到买主。

缺点：
  1. 摄像机的范围分辨率很差，他们几乎不了解到障碍物的距离或进近速度（立体声摄像机可以部分解决问题）。
  2. 没有直接测量物体速度的方法，可以通过一系列框架来理解，但是测量仍然是模糊的。
  3. 受天气和道路条件的强烈影响-最近发生事故的臭名昭著的“白色货车顶着明亮的天空”。
  4. 处理摄像机的数据需要时间：延迟越大，摄像机系统越复杂。

从Intel和NVIDIA到Mobileye和Magna International的大象都挤满了市场。有很多创业公司，进入门槛很低。技术的发展取决于数据处理的速度和质量，即处理器和算法。不可能大大提高处理速度，并且突破性算法还没有突破。

结论：
您仍然不能没有车载摄像头。



雷达


它的工作方式是：以超高频发射，聆听后面的声音，立即识别到障碍物的距离，障碍物的速度和角度。

优点：
  1. 天气条件对雷达运行的影响要小得多；物体和光线的颜色完全不受影响。
  2. 可以提供良好的范围和速度分辨率，可以直接测量范围和速度，并且具有较高的精度。
  3. 即时（1 / 20-1 / 50秒）数据处理，这对于高速而言尤其重要。
  4. 由于信息量少，因此可以轻松地动态累积和分析信息。出于同样的原因，适用于V2I / V2V

缺点：
  1. 不便宜，而且通常比照相机贵。远距离雷达的价格超过1,000美元。
  2. 角度分辨率差（雷达很难理解整个运动）。
  3. 没有直接的机会来分类障碍或评估其规模。
  4. 范围对视角的依赖性很大。如果我们照亮狭窄的光束，则看不到道路和侧面的干扰。我们光芒四射-我们的射程很小。

市场上有大象，只有其他名字-博世全球，德尔福和Smartmicro。几乎没有初创公司，由于微波开发设备的高昂成本，进入门槛很高。在过去的50年中，加工方法几乎没有变化，在零件生产（作为天线-馈线路径）的开发和组织方面的投资为数亿欧元。负担得起的高质量收发器，如果​​没有这些收发器，就根本没有大规模解决方案，它在全球范围内产生了五家公司。雷达在不断改进，但发展需要数十年的时间。

结论：
雷达的优点是相机的缺点，反之亦然。没有雷达，也无法做到。

不是我想出的，而是马斯克先生：
“ ...也就是说，我认为您不需要激光雷达。我认为您可以使用无源光学系统，然后再使用一个前向雷达来完成所有这一切……如果您开车驶入雨，雪或尘土飞速行驶。我认为无需使用激光雷达即可完全解决该问题。我不是激光雷达的忠实拥护者，在这种情况下，我认为这没有道理。”



激光雷达


激光测距仪：他们用激光照耀着，看着后面的东西，发出了一系列与它们有距离的点。

优点：
  1. Wunderwaffle，可以高精度查看周围的所有细节。除了他们，没有人能够而且永远不会。

缺点：
  1. 扫描激光雷达每秒只能看到两到三遍相同的路段。
  2. 成本高，无需-激光雷达的成本。
  3. 激光雷达仍然很脆弱。
  4. 简而言之，剩下的就是缺点

市场上有很酷的公司-Velodyne LiDAR，Quantergy和Sick。所有的决定都相对年轻，有一些初创公司，但规模相对较大。在汽车领域拥有数百亿美元收入的真正大型博世级别参与者已经不在市场上，也永远不会出现。随着激光雷达成本的降低和运动部件的放弃，激光雷达的功能和需求也急剧下降。每个人都在等待250美元的决定，但是缺少了一些东西。
简单易懂：激光雷达的汽车市场比雷达或照相机的市场小一百倍，而雷达和照相机的市场则充满其他应用

结论：
在革命发生之前，激光雷达将仍然是一个利基解决方案。非常适合制作原型，没有量产车的方法。是的，并且对于原型制造来说很不好：激光雷达的逻辑逻辑不能很好地适应其他传感器。



ULTRASOUND
著名的驻车传感器：发出高频信号，等到它返回，再计算时间。

优点：
  1. 非常便宜，一个多传感器系统一百或两百美元
  2. 传感器易于集成。
  3. 该技术已经建立并且易于理解，您甚至可以在家中使用arduino进行实验。

缺点：
  1. 范围限于米的单位。
  2. 无法理解与物体的角度或直接测量速度。
  3. 数据可靠性低。


现在，市场以中国姓氏为主导，林间空地早已被践踏，很少有人愿意投资新项目。很少有创业公司，但是有例如Toposens，它在超声波方面做得很酷，但是突然之间也有一些东西成为雷达。

结论：
一旦雷达和激光雷达学会很好地解决短距离问题，超声波将仅保留用于arduino。开个玩笑:)。



红外摄像头
工作原理：在红外范围内看，分析矩阵上的图像。

优点：
  1. 热人和森林动物清晰可见。

缺点：
  1. 昂贵，仅前置摄像头就需要几千美元
  2. 范围狭窄
  3. 使用温度的限制。

结论：
一个比激光雷达更利基的故事。带有激光雷达的雷达将很快将其赶出汽车市场，如果还没有的话。

现在-甜蜜。总结一下：

  1. 没有任何传感器或传感器系统可以普及。如果我们谈论低端解决方案中唯一的角色-这些就是相机。其次，雷达。
  2. 任何ADAS系统，尤其是自动驾驶仪，都将使用由多个传感器组成的系统。这称为传感器融合，它将主要结合摄像头和雷达。我怀疑他们会完成。
  3. 您想要收集的信息越多样化，花费的成本就越高（使麻烦减少一半），花费的时间也就越长。我们处理的时间越长，为了安全起见，我们就必须越慢地开车。因此，大多数自动驾驶仪驾驶者都喜欢乌龟，也热爱好天气。

不仅如此。现在让我们谈谈现实世界中的问题。
