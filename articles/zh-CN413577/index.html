[公告]在俄罗斯七个城市的CodinGame.com上进行比赛的枢纽

tl; dr：电路第二次组织离线站点，以在线竞争CodinGame上的编程游戏bot。 地点将在7个城市：叶卡捷琳堡，伊热夫斯克，因诺波利斯，新西伯利亚，彼尔姆，顿河畔罗斯托夫和圣彼得堡。

一切将在6月16日（星期六）（在Innopolis-6月18日，星期一）进行。 如果您想分心工作任务并编写一些不寻常的内容，请来。 注册链接-在帖子末尾。

大家好！ 我喜欢参加编程比赛。 例如，我是乌拉尔FU团队的一员，最近我从事体育节目制作：Kek成为ACM ICPC世界总决赛的铜牌得主。


我的另一个爱好是在CodinGame上编程游戏机器人。 我不久前一直在这样做，但是设法参与其中，现在我正在寻找志趣相投的人。 因此，我决定为中心的组织提供帮助，并邀请所有人加入。



我的个人资料在CodinGame上。


什么是CodinGame？


CodinGame是一个用于学习如何以有趣方式编程的平台。 每两个月一次，为期10天的游戏机器人编程竞赛开始。 通常，一切都井井有条：


  • 很多参与者。 您可以与来自世界各地的强大竞争对手竞争。 例如，在上一届比赛中，有2120名参与者，其中106名来自俄罗斯。
  • 适度的进入门槛。 即使是新手，也可以制造出可以玩并击败其他机器人的机器人。 您可以使用26种编程语言中的任何一种 。
  • 舒适而美丽。 有按国家/地区排列的便捷排行榜，用于查看机器人游戏的可视化工具，用于在浏览器中运行代码的沙箱。


竞赛之一的可视化器。


6月16日晚上，一场新的比赛开始了- 代码库特鲁 （ Code of Kutulu） 。 这些规则将在开始后发布，而只有涉及霍华德·洛夫克拉夫特（Howard Lovecraft）工作的名称才知道。 但是，很明显会发生什么。 最有可能的是，有必要管理世界上根据给定规则工作的一个或多个机器人。 汽车，飞机，轮船，巫师骑行，飞行，游泳，射击，扔球以收集奖励，取得检查站，消灭对手并进球。 您需要每个节拍读取一次世界的当前状态，并对每个机器人发出命令来响应。


为什么要编写机器人并竞争


总的来说，我习惯于其他任务和比赛。 在CodinGame举行的第一场比赛之前，我认为游戏机器人有些复杂。 就像现代国际象棋引擎一样，它能够计算出游戏的25步前进并击败当前的大师。


但是，事实证明，无需编写复杂的算法。 相反，您需要提出启发式的想法，在比赛中为您的机器人加油，并尝试使他摆脱愚蠢的举动。 与体育编程不同，在此处编写测试很有用-它们可以在很长的距离内提供帮助。


CodinGame的专家们学会了制作复杂的游戏，不会吓到初学者，同时又千差万别-专家们可以在思维灵活性和编程复杂逻辑的能力上相互竞争。


在所有新游戏中，都具有漂亮的可视化效果，因此，即使是儿童也可以理解游戏。 社区随时准备为策略中的界面，工具和错误提供帮助，并以幽默的态度来主持任何讨论。


什么是枢纽，它们将在哪里


CodinGame让玩家可以离线聚会，讨论策略，并在需要时成对编写代码并相互竞争。 为此的平台是集线器（编码集线器）。



Trello董事会，集线器的组织者。


众所周知，在德国，哥伦比亚，立陶宛，土耳其和法国将有13个枢纽 。 拥有Kontur开发办公室的其他7个枢纽将在俄罗斯的城市中。


这是集线器上发生的事情：


  • 收集，分析规则并讨论策略
  • 编写代码2个小时（成对可选）
  • 前往检查站：讨论成功，吃披萨，取消排行榜顶部参与者的匿名
  • 再写2个小时的代码
  • 再次去检查站

集线器将持续4-5小时。 显然，您可以提早离开，也可以坚持一两个小时。
