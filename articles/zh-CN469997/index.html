哪些标题最有可能引起注意或进行HabraHabr分析

您好，再次希望分享一些有用的东西！ 关于家庭主妇的福利的文章没有写给Habr，所以我掏出了4年前专门为habrhabr.ru写的一篇文章，以宣传一个数字代理公司，而我已经关闭了。 我希望这次会有更多的优势。

您是否曾想过，但毫无疑问，当您在Internet上搜索信息时，您会立即跳过部分标题。 通常不会读到最后。

这是因为它们根本不有趣。 因此，文章的作者，博客作者和撰稿人都在努力想出一些措辞，从一开始就抓住您（如果不是单词！）。 事实证明，情况并非总是如此，写标题没有黄金法则。 但是某些常见的头结构形式或类型是最常见的。

我们为geektimes.ru和habrahabr.ru网站的热门新闻（2008-2016年）选择了100个头条新闻，并试图强调类似的链接和逻辑联系。 原来有八个类别，下面列出。

I）某物的X，而X可以被具有定量特征（例如，多个）或具有比较特征（尤其是最高）的单词替换。

  1. 您需要停止做的8种网站类型；
  2. 几种有用的服务；
  3. 世界上最小的游戏（58字节）。

II）怎么做 ，包括过去时，例如我/他/某人做了什么。” 此外，还有个人/外来经历和标题，例如“如何/为什么/您知道什么/有必要/为什么我/他/他们...是动词”。 在大多数情况下，它们没有问号。 文章本身回答了这个问题。 此格式的上下文是可选的。 它也可以采用“课程/经验/技巧（针对...）”或“历史记录（关于...的方式）”的形式

  1. 我如何购买3D打印机；
  2. 我为何离开Goog​​le；
  3. 为什么Mail.Ru开发间谍软件？
  4. 为什么Microsoft Internet Explorer 11是最糟糕的事情？
  5. 关于今天ICQ发生了什么（第1部分）；
  6. 苹果树和苹果树不了解什么？

!!! 与该类型相关的含义：

  1. 您购买程序了吗？ 来... [我是如何购买程序的...];
  2. Internet资源的安全性-入侵rospil.info网站[我入侵时...]；
  3. 拒绝过时的浏览器（因为我拒绝...）。

III）相反的构造类型 ，取反

  1. 您没有正确地写动物=如何正确地写动物；
  2. 如何为智能手机充电=如何为智能手机充电。

但是，请务必注意以“课程”一词开头的词组，因为它们不能表示类似的建议，而只能说明事件的后果或因某事而获得的经验。 标题为“来自太空灾难的教训”。

IV）用语句代替问题 -用“动词/创造”或名词“创造/发展”等形式的动词代替“怎么做” 这也包括DIY类型-“用自己的双手/从头开始/从即兴的手段。 但是，这些词可以省略，但标题清楚地表明了它们的缺失位置。

  1. 我们使用旧的LCD显示器制作私人显示器；
  2. 18速高斯
  3. 一个晚上可以自己动手操作强大的激光；
  4. 恢复散焦和模糊的图像。 我们提高质量；
  5. 核反应堆-从头开始。

V）关于...的事实/功能/ ...的介绍/关于...的备忘录

  1. 关于甲型流感（H1N1）的编程；
  2. SIM卡简介；
  3. 给网站设计师的备忘录；
  4. 俄罗斯发展的特点。

但是可以省略介绍性短语而不会失去含义，例如标题为“半导体电子”和“太空和军事用微电子”的标题，或者例如，“苏联为挽救死亡空间站而进行的行动”。

VI）纯新闻/子站点。 通常，这些是祝贺，有关站点更改的消息，内容的显示，警告等。

  1. 服务XXXXXX.ru暂时关闭；
  2. Habrahabra官方移动应用程序；
  3. 注意！ XXX.RU遭到入侵！

VII）审查，评估，试驾，审查，传记。
标头格式：通常是对象本身。 重要提示：审查的完整性/深度，完整的结论或评估，与事件相关的上下文约束。

  1. 评论Apple iPad；
  2. 中央处理器
  3. 内部视图：CD和HDD；
  4. 我知道史蒂夫·乔布斯。

VIII）建议/积极采取行动。 这包括写在“关于情绪”，“生气”的文章

  1. 停止扭曲！
  2. 取回我的钱，ATM！;
  3. 不要取笑程序员。
  4. QIP-仇恨分钟（服务器上的历史记录）。

IX）不属于上述任何类别

  1. 沃罗涅日茨与银行签定了一份合同，自己做零钱，并将起诉2400万卢布。
  2. 没有人写过有关诺基亚，埃洛普和刻录平台的文章。
  3. Google Play上的性暴力；
  4. 从这里开始。

标题类别数：



显然，如果您不考虑不适合任何类别的独特标题，那么通常会有一些人采用第二类别（“如何做/个人经历”），第四类别（自己创建/自己做）和6的结构。 th（子站点）。

但总的来说，受欢迎程度排名第二的是具有独特结构的头条新闻，例如，“没人写过有关诺基亚，埃洛普和《燃烧的平台》的文章”或“ Mediamagiya：回家，拿起遥控器并选择从跟踪器观看”。

撰写有趣的标题，吸引人！
