Noções básicas de JavaScript


A master class para desenvolvedores web iniciantes foi preparada por Mikhail Ovchinnikov, um engenheiro de software líder do Badoo. O que é design de UX e por que é importante?


Todo mundo já ouviu falar sobre o design de UX hoje. É difícil dar uma definição precisa do conceito, mas vamos tentar. UX é um conceito multifacetado de projetar uma interface prática e amigável que proporcionará às pessoas uma experiência nova e inspiradora (Experiência do Usuário). O designer-chefe da AIC Studio, Sergei Popkov, lhe dirá mais. Ele se dedicará às principais áreas do design de UX e explicará com um exemplo simples como aplicá-lo.

7 designers de erros comuns


É melhor aprender com os erros dos outros. Mas se você se encontrar entre eles, não contaremos a ninguém ;-) O designer da web Philip Solomin, com muita experiência, mostrará erros típicos no design da web no exemplo de projetos finalizados. Você aprenderá como minimizar falhas no trabalho e obterá uma lista dos problemas mais comuns nos layouts.
