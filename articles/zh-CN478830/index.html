Plesk，cPanel或ISPmanager：选择什么？

在开始工作之前很难测试提供商提供的所有面板，因此我们在简短的评论中收集了三个最受欢迎的面板。



当客户端从操作系统管理转移到与托管相关的任务时，就会出现困难。 他必须使用不同的CMS和许多用户帐户来管理许多站点。 为了减少人工成本，值得安装控制面板，该面板允许您通过便捷的Web界面配置相应的服务。 供应商的合作伙伴也需要将其服务出售给客户。 今天，我们比较了在Linux上订购VPS和VDS时可以使用的三种流行产品。

lesPlesk


最通用的选项之一，适用于所有类型的任务。 该面板不仅适用于流行的基于deb和rpm的Linux发行版，还适用于Windows。 尽管Windows VPS / VDS客户很少需要第三方管理工具，但可以根据需要安装它们。 Plesk在众多受支持的软件方面也与竞争对手不同，包括 很少在传统的Web服务器（Docker，NodeJS，Git，Ruby等）上使用。

开发人员提供该产品的不同版本，包括具有最少功能集的精简版本。 Plesk允许您为每个站点选择PHP版本，支持PHP-fpm，具有流行CMS的内置安装程序，以及补充面板功能的大量扩展。 根据版本的不同，Plesk可能包括一个计费面板，以及创建不同的收费计划和转销商的能力-该产品主要用于托管公司和网络工作室，对于某些VPS / VDS，其功能似乎是多余的。 在此阶段发现的Plesk的主要缺点是许可证价格高昂，并且需要购买扩展。

PacPanel和WHM


该面板旨在与RedHat Enterprise Linux和某些派生发行版一起使用。 它易于使用，但功能强大：通过cPanel，您可以管理Web服务器和数据库，灵活配置对托管用户的限制，配置费率计划，创建转销商以及使用过滤器和新闻通讯管理邮件服务。 与Plesk一样，这里还有许多其他功能，并且cPanel功能已通过商业和免费插件进行了扩展。 此外，该工具还允许您选择不同的操作模式和不同版本的PHP。 在严重的缺陷中，可以注意到相当高的许可证成本，并且缺乏对流行的基于Deb的发行版的支持。

▍ISP经理


我们审查的最后一个面板与其他面板价格低廉。 此外，它不仅可以在CentOS（RHEL克隆）上运行，而且还可以在Debian / Ubuntu上运行。 该面板针对托管任务进行了优化，并会自动更新。 用户可以访问详细的俄语文档，可以为每个站点设置PHP版本，并且可以在Docker容器中同时安装多个版本的DBMS。 支持PHP-fpm，内置了流行脚本和CMS的内置安装程序，以及许多扩展模块，可扩展集成的功能。

RuVDS的价格


上表显示了Plesk，cPanel和ISPmanager许可证的价格范围（如果在开发人员站点上购买的话）。 许多托管服务提供商愿意立即为服务器配备面板，而许可成本可能会更低。 作为新年活动的一部分，RuVDS为订购VPS的客户提供在2019年12月31日之前免费使用ISPmanager Lite的机会以及在2020年1月31日之前免费使用Plesk Web管理版本的机会。 行动结束后，许可证的费用将为每月200和650卢布。 cPanel的试用版可以免费使用14天，但是您必须直接从开发者那里购买许可证。

第一印象


因为我们已经处理了这一问题，所以客户在安装和启动面板方面不会有任何问题-另一个（价格除外）通过托管服务商购买许可证的原因。 订购服务器时，只需选择以下三个可用选项之一：ISPmanager Lite，Plesk Web管理员版或cPanel＆WHM，免费试用期为14天。 请注意，尽管Plesk可以在Windows Server上运行，但没有提供此选项。 如果需要Microsoft操作系统的面板，则必须自己安装。 这是常见的做法：Windows上的VPS / VDS不包含第三方软件。 cPanel仅适用于CentOS计算机，这也是很自然的。


站点的初始设置和创建不会造成任何特定的困难，但是，每个特定面板的功能在这里已经很重要。 让我们尝试突出他们的优点和缺点。

lesPlesk


Plesk用户界面类似于WordPress管理仪表盘。 左侧是菜单（导航栏），中间是工作区。 菜单组织得很合理，所有设置都在手边。 该界面类似于WordPress管理面板：我们非常喜欢Plesk与该流行的CMS的紧密集成，在此安装非常自动化。 安装其他第三方脚本非常方便-这是一大优势。


在窗口的右侧，您可以找到其他简化面板操作的界面元素。 它们包含各种信息，使您可以快速进入设置的各个部分，还可以安装其他软件。 Plesk的主要优点是大量扩展和与用于Web托管的奇异软件的兼容性。 我们特别喜欢开箱即用的Docker支持以及丰富的现成映像集（您也可以上传自己的映像）。


最后，美中不足的是：在Plesk Web admin版本中，只有基本功能可用，在更昂贵的版本中，它们的列表更广泛。 但是，这是入门级版本的常见属性。

PacPanel和WHM


在这里，我们喜欢将帐户分为两种类型：用户和管理员/转销商。 实际上，该产品由两个不同的面板组成：cPanel本身和WebHost Manager（WHM）。 第一个是面向普通托管用户的，使用它非常方便。


可通过特殊的WHM面板使用包括为管理员和转销商创建收费计划的功能。 整个面板的界面在逻辑上进行了组织：带有搜索栏的隐藏分层菜单通常位于左侧，而工作区位于右侧。 它有很多设置，一方面这很好。 另一方面，不能将WHM菜单称为便捷菜单。 如果在Plesk中我们几乎从未使用过搜索，那么在每个部分中都有很多选项，搜索栏成为主要的管理员工具。


▍ISP经理


此控制面板与以前的控制面板之间的重要区别是最简化，最直观的界面。 左侧是导航菜单，右侧是工作区。 您可以在工作区选项卡中单独或同时打开各种菜单选项-这非常方便，因为管理员通常需要并行使用不同的面板功能。 除了与托管直接相关的功能外，管理员还可以访问一些其他功能以及系统功能，例如防病毒扫描，文件管理器，调度程序或防火墙。 捆绑软件中包含的其他应用程序列表包括Roundcube Webmail和phpMyAdmin。


我们喜欢初始设置的简单性和自动更新软件的能力，以及面板和所有相关文档的完整俄文本地化-国外的发展对此有困难。 另一方面，简化的界面并不总是具有必要的设置灵活性，并且与Plesk和cPanel的集合相比，ISPmanager可用的附加模块的数量正在减少。 此外，在最便宜的Lite版本中，您无法创建代理商和集群配置。


安全性


控制面板为管理员提供了在服务器上安装的操作系统的广泛特权，因此，其中潜在的漏洞可能很危险。 默认情况下，具有自签名证书的启用加密的HTTPS协议用于访问列出的所有面板的功能。 在这种情况下，没有人会禁止用户安装购买的证书。 另外，cPanel和ISPmanager为管理员/经销商和客户端配置两因素登录身份验证。 另外，cPanel对管理工具具有附加保护：例如，它没有通过直接链接的phpMyAdmin访问。 此外，所有三个面板均会定期更新，它们使您可以为站点（包括自签名站点）安装SSL证书，还可以向其中添加各种与安全性相关的模块，例如防病毒工具。

后备


Plesk支持对其自身存储或外部资源的完整和增量备份。 在这种情况下，您可以创建整个服务器的完整副本，也可以创建单个用户帐户的数据副本。 cPanel会创建压缩副本和未压缩副本以及增量副本-默认情况下，它们会保存在本地。 值得注意的是，有可能启动预定的复制过程，并且缺少自己的数据恢复接口。

我们认为，ISPmanager中的备份设置不够灵活，但是此面板中还具有所有主要功能：数据存储在本地目录或外部资源中，并且可以用密码保护。 默认情况下，将复制所有用户的数据，尽管可以在设置中进行更改。 另外，这些设置指示完整备份和每日备份的数量。

优缺点


所审查的所有三个面板均排在最受欢迎的面板的顶部，并且具有广泛的功能。 Plesk支持多种软件，并允许您解决最广泛的任务。 用户可以使用200,000多种不同的Docker映像，并且大量的扩展使Plesk成为不仅适用于托管的通用工具。 cPanel旨在解决与托管相关的任务，而开发人员将对不同功能的访问权限分为两个级别：为普通用户和管理员创建了单独的面板。 还值得注意的是，对计算资源的需求很高-cPanel不应安装在低功耗VPS上。 ISPmanager面板也仅用于托管管理。 它易于使用，不需要资源且价格便宜-也许这是入门级VPS或新手管理员和托管者的最佳选择。
