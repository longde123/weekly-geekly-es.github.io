Chef, ich sehe dich. Intel Unite - Professionelles Kommunikationstool



Kommunikation regiert die Welt. Es ist die Interaktion, die eine Gruppe von Menschen zu einer Mannschaft macht, zumindest für den Fußball, zumindest für die Entwicklung eines Softwareprodukts. Wie kann die Kommunikation zwischen Menschen, unabhängig von der physischen Entfernung zwischen ihnen, bequemer und natürlicher gestaltet werden und gleichzeitig ein Minimum an finanziellen und Arbeitskosten kosten? Intel bietet eine Lösung für dieses Problem. Es heißt Unite - ein Softwarepaket für professionelle Kommunikation.

Intel Unite besteht aus drei Komponenten:

  • Der Autorisierungsserver ist für die Sicherheit des gesamten Systems verantwortlich und für die Ausgabe von Kennwörtern für die Erstverbindung.
  • Ein Computer-Hub, der mit verschiedenen Multimedia-Geräten verbunden ist - für ihn sind die Kommunikationsteilnehmer verbunden;
  • Client-Computer mit Client-Software, die eine Verbindung zum Hub herstellen.

Schematisch und auf Englisch sieht es wie folgt aus


Dieses ganze System funktioniert wie folgt. Teammitglieder installieren ein Client-Programm an ihren Arbeitsplätzen. Um eine Verbindung zum Hub herzustellen, benötigen Sie das vom Sicherheitsserver generierte Kennwort. Danach stehen die Ressourcen (dh alle angeschlossenen Mediengeräte) allen Teilnehmern der Konversation zur Verfügung. Nun, die Kommunikation selbst wird noch zugänglicher, gleichzeitig können Sie eine allgemeine Diskussion und persönliche Gespräche führen. Natürlich ist es möglich, Dateien und Textnachrichten auszutauschen. Unite-Verkehr wird mit einem 256-Bit-Schlüssel verschlüsselt.

Auf den ersten Blick scheint der Unite-Hub erheblich belastet zu sein, und Sie müssen einen leistungsstarken Server installieren, um damit fertig zu werden. Ein lauter und sperriger Server hat jedoch keinen Platz im Besprechungsraum, und die Last kann mithilfe der proprietären Intel vPro- Technologie reduziert werden. Die VPro-Unterstützung ist eine Voraussetzung für die Gateway-Plattform. Bei der Größe ist es im Gegenteil besser, eine kleinere zu nehmen, um Platz zu sparen. Intel hat einen Computer, als wäre er ein Unite-Hub. Dies ist wahrscheinlich Intel NUC - finden Sie einfach die Option mit vPro-Unterstützung in der Aufstellung. Andere Hersteller bieten ihre Unite-kompatiblen Mini-Computer auch für Besprechungsräume an - auf dem Foto unten zum Beispiel das sehr hübsche HP Elite Slice für Besprechungsräume.


Eine weitere interessante Plattform für die Bereitstellung von Intel Unite ist ein Videopanel mit einem OPS-Steckplatz ( Open Pluggable Specification ) und entsprechend einem OPS-Computer. OPS ist ein offener Standard zum Koppeln eines Videoausgabegeräts und eines Computers. Das OPS-Computergehäuse hat die Form eines Parallelepipeds und ist mit einem 80-poligen Anschluss zum Anschließen an einen Monitor ausgestattet, mit dem es als einer bezeichnet werden kann, wodurch die Anzahl der Drähte verringert wird. So sieht ein OPS-kompatibler Computer aus.


Eine Besonderheit von Intel Unite ist die Möglichkeit, mit zusätzlichen Modulen von Drittanbietern zu kommunizieren. Um sie zu erstellen, gibt es ein spezielles SDK . Mithilfe von Plug-Ins können Sie beispielsweise Meeting-Management-Tools in einem Produkt eines Drittanbieters oder in einer Cloud-basierten Webkonferenzlösung implementieren, damit der Intel Unite-Host eine Verbindung zu Webkonferenzen herstellen kann. Hersteller von Geräten für Konferenzräume (Mikrofone mit Lautsprechern, Kameras usw.) können mithilfe von Plug-Ins eine praktische Schnittstelle zur Steuerung von Geräten (Lautstärke, Videoübertragung, Beleuchtung usw.) erstellen.

Die dritte Version von Intel Unite steht derzeit zum Download zur Verfügung . Kunden sind für iOS, MacOS und Windows geschrieben. Wenn Sie nach einem Meeting-Tool suchen, probieren Sie dieses aus, es könnte für Sie funktionieren.
