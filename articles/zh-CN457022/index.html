IT重定位。 从曼谷到悉尼


美好的一天，亲爱的读者。 如果您熟悉我搬到曼谷的故事 ，那么，我想听听我的另一个故事会很有趣。 在2019年4月上旬，我搬到了地球上最好的城市- 悉尼 。 坐好舒适的椅子，酿造温暖的海鸥，欢迎来到这只猫，在这里您会发现许多关于澳大利亚的事实，比较和神话。 好吧，走吧！


引言


住在曼谷很酷。 但是所有的好事都结束了。
我不知道发生了什么事，但是每天都有各种各样的琐事引起我的注意，例如没有人行道，街道上的噪音和空气污染严重。 我脑子里浮现出一个非常不清楚的想法： “五年后我会在这里得到什么？” 。


在俄罗斯之后，在泰国，每个周末都能出海旅行，温暖地生活，一年四季都可以吃水果，这真是太酷了。 但是尽管生活很好，但我还是没有宾至如归的感觉 。 我不想购买任何内饰元素，但是购买了运输工具是因为它更容易出售，等等。 我想要某种稳定和感觉，希望我可以在该国居住很长时间并且不受签证影响。 另外，我真的希望该国说英语。 选择范围是美国，加拿大，英国和澳大利亚-您可以在这些国家/地区获得居留证。


这些国家/地区各有利弊：


  • 加拿大 -有机会离开美国独立移民，但由于天气原因，这完全是一场灾难。
  • 英格兰是高度发达的文化生活，但是获得居留证的过程可能需要长达8年的时间，而且还要花很多时间。
  • 美国是程序员的圣地。 我认为绝大多数人会尽可能地前往旧金山。 但这并不像看起来那样容易。 一年前，我有H1B签证程序，但没有被抽签。 是的，是的，如果您收到了美国一家公司的录取通知，那么您将不会获得签证，但您可以每年3月申请一次。 通常，该过程是不可预测的。 但是3年后，您可以获得梦co以求的绿卡。 持L1签证也可以搬到各州，但是现在法律正在游说不能申请绿卡。 特别令我恐惧的是在美国的税收居留权。

那么，为什么您应该将澳大利亚视为非常好的移民竞争对手呢？ 让我们看一下要点：


一些事实


我一直以为澳大利亚是世界边缘的一个小大陆 ，很有可能从磁盘上掉下来。 确实，我们在地理课程中有多少次去看澳大利亚？


澳大利亚是世界第六大国家。


地图上的比较将非常清楚。 我认为从斯摩棱斯克到克拉斯诺亚尔斯克的距离非常令人印象深刻

这是塔斯马尼亚岛，可以和爱沙尼亚相比

人口约为2500万人 （平均每人2只袋鼠）。


人类发展指数（ HDI ）在世界上排名第三。
人均国内生产总值为52,373美元 。


80％的人口是第一代和第二代移民


很好的表现。 但是为什么人们不想去澳大利亚...


自然与气候


这可能是我所经历过的最佳气候条件比率 。

看来你住在泰国，一切都很好。 永恒的夏天。 +30。 海上交通便利。 看来，哪里有更好的选择？ 但是可以！


澳大利亚的空气非常干净。 是的，我亲爱的朋友，您开始真正欣赏空气了 。 空气污染指数就是一个指标。 您可以随时比较
曼谷和悉尼 。 这里的呼吸好多了。


泰国的高温很快就困扰人。 我错过了暖和的衣服。 我真的希望一年2-3个月达到+ 12-15度。


坦白说，我在温度方面感到很舒服。 夏季+25 （9个月） ，冬季+12 （3个月） 。


这里的动物真的很棒 。 袋鼠，袋熊，考拉和可爱的国鸟-在这里，您将在自然环境中与他们相遇。 什么是唯一的宜必思酒店（在普通百姓Bin Chicken中）



鹦鹉，鹦鹉和狐狸都在这里，而不是鸽子和乌鸦。 起初，狐狸真的会吓到。 特别是如果您看过足够多的关于吸血鬼的电影。 通常，这些小蝙蝠侠的翼展可达30-40厘米，但不要惊慌，它们非常可爱，此外- 素食主义者



移民


在我看来，与加拿大一起，移民到澳大利亚是世界上最便宜的移民之一。 有几种移民方法：


  • 独立 （立即获得公关）
    澳大利亚的优势在于，它立即提供了机会，帮助有职业的人们获得居留证。 您可以在“ 职业列表”中查看您的职业 。 要获得此签证，您必须至少获得65分， 其中25至32岁的年龄可获得30分。 其余的-英语知识，工作经验，教育程度等。

我有很多刚持此签证的朋友。 缺点是收据过程可能需要一年以上的时间 。


获得签证后，您将必须前往澳大利亚并在新地方定居。 这种方法的复杂性是您需要第一次拥有资金。


  • 赞助签证 （2年或4年）
    它与其他国家相似。 您需要找到一个准备为您提供赞助签证的雇主（482）。 两年的签证并没有获得居留许可的权利，而是4个月的许可（或更确切地说，是由公司担保的权利，这需要另外1-2年的工作时间） 。 因此，您可以更快地获得令人垂涎的居留证。

获得签证的整个过程大约需要一个月。


  • 学生
    您可以注册当地的机构。 假设您获得了硕士学位（Master） 。 这种方法的优点是您将拥有兼职的权利。 另外，毕业后一年，您可以在澳大利亚。 通常这足以在这里找到工作。

对于所有签证，您都必须参加英语考试。 独立的一个人在所有方面至少需要6个（雅思），而赞助人（技术专业）仅需要5个。


与美国不同，澳大利亚具有很大的优势，那就是您的伴侣将获得与您相似的签证，并拥有完全的工作权 。


求职


如何在珍爱的澳大利亚找到工作？ 有什么陷阱？


首先，请考虑以下热门资源：


  • 寻求可能是澳大利亚的主要聚集者。
  • Glassdoor-我更喜欢。 您总是可以找到一个职位的大概薪水，也可以找到很好的匿名评论。
  • LinkedIn是该类型的经典。 他们每周在这里给我写5-8个eychars。

我获得了受扶养签证，就地找工作 。 我的经验是9年的移动开发经验。 在一家大公司成立之后，我想在房子旁边找些灯，然后放松一下。
结果，在最初的3天里，我经历了3次面试。 结果如下：


  • 采访花了25分钟， 要约 （刚好高于市场）


  • 同样， 出价 25到30分钟（以市场价值为准，但之后的出价与第一次相似）


  • 2小时的面试，以“我们决定继续与回答正确问题的候选人继续前进”的风格进行拒绝，这种拒绝是刻板的 ，您不应感到沮丧。



澳大利亚有两种主要的工作类型。 它是永久性的和契约性的 。 奇怪的是，但签订合同可以多赚40％ ，老实说，我考虑过朝这个方向发展。


如果公司正在寻找承包商六个月，而您想要一份长期工作，那么他们会拒绝您，这是合乎逻辑的。


我听说人们很难找到第一份工作，因为他们没有在澳大利亚的工作经验，但是如果您是一位优秀的专家，这是个小问题。 最主要的是赶上。 六个月后，您将开始编写本地HR，这将变得更加容易。


在俄罗斯之后，很难首先接受文化契合度，而不是工程技能。


这是我两个月前发生的一次采访中的一个简短故事。 说我被烧死就像什么都没说。 所以我会把眼泪藏在伤口后面

该公司从事“需要做某事的人”和“准备 做某事的人”之间的调解。 团队相对较小-每个平台有5个人。


接下来，我将描述招聘过程中的每个项目。


  • 家庭作业 。 有必要进行“经典”处理-以显示API中的列表。 结果，该任务通过模块化，UI和UT测试以及一系列架构玩笑而完成。 我立即被邀请参加Face2Face 4小时。


  • 技术在讨论作业时，发现他们在我担任维护者（特别是Kakao ）的项目中使用库。 老实说，在技术方面完全没有问题。


  • 算法 -有关回文和回文字典的所有愚蠢行为。 一切都立即确定，而且资源最少。


  • Cultural Fit –与负责人进行了非常愉快的交谈，内容是“我如何以及为什么要去编程”



结果，我已经在等待报价，并且想如何讨价还价。 HR期待已久的电话在这里：


“不幸的是，我们被迫拒绝了你。在我们看来，你在面试中表现得过于激进了。”

老实说，当我谈论我的“侵略”时，我所有的朋友都嘲笑这一点。


因此，请注意。 在这个国家，您必须首先是“好朋友”，然后才能编写代码 。 这，该死的，该死的真气。

工作作风


这里的工作风格与我们通常的工作风格截然不同 。 准备好在头N年内，您会受到许多因素的轰炸。


在俄罗斯，我们习惯于努力工作。 一直工作到晚上9点是正常的 。 与同事聊天，完成功能到最后...他们回家， 吃晚饭，看电视，洗澡，睡觉 。 通常， 习惯上有工作偏见 。


在这里，一切都完全不同。 工作日为7.5个小时 （每周37.5个小时） 。 通常习惯早点上班（早上8-9点）。 我以9.45到达某个地方。 但是， 下午5点以后，每个人都已经回家了 。 习惯上花更多的时间与家人在一起，我认为这是更正确的。


带孩子一起工作也是惯例。 但是，陌生人是按照事情的顺序带着狗来办公室的！



下班后，我写信给设计师，说他阻止了我开发功能，我得到了答案：


康斯坦丁（Konstantin）-抱歉在这件事上受阻....我昨晚会这样做的，但这是《权力的游戏》的最后一集，我不得不权衡优先次序。

这是正常的！ 该死，我真的很喜欢我个人的时间！


每个办公室总是在冰箱里放啤酒和葡萄酒。 在这里按午餐的顺序喝啤酒。 在星期五下午4点之后，没有人在工作 。 我们决定只是在厨房里闲逛，和新鲜订购的比萨聊天。 这一切都很轻松。 星期五，打开星期六，我真的很喜欢。


但是，有一些非常有趣的时刻。 一次，在大雨中，办公室的屋顶漏水，水顺畅地流入墙上的电视。 电视被发现无法正常工作，并换了一台新电视。 猜猜大雨过了三个月后发生了什么？

住哪里



寻找住房可能是搬家时最大的担忧之一 。 有人告诉我们，有必要收集大量文件以确认收入，经验，信用记录等。 实际上，这些都不是必需的。 在我们选择的三间公寓中，有两间被我们批准 。 最后，我们自己不想打电话。


当您寻找住房时，请准备将价格定为每周 。 在选择地区之前，我建议您先阅读一下，因为住在悉尼市中心（中央商务区）的想法不是最好的主意（对我而言，这太嘈杂且拥挤，但每个人都自己选择） 。 从中环站2至3个站后，您会发现自己处于睡眠区，氛围轻松。


一居室的平均价格是2200-2500澳元/月 。 如果您在没有停车位的情况下搜索，可能会发现更便宜。 我的许多朋友在中心租用两居室，价格可能高出一半甚至两倍。 这完全取决于您的需求。 是的，与俄罗斯不同-一间卧室将由一名客人和一间单独的卧室组成。


大多数公寓的租金都是不带家具的 ，但是如果您尝试尝试，您会发现家具齐全（与我们一样）。 查看公寓始终是一个小组。 分配了日期和时间，大约有10-20人到达，每个人都在看公寓。 在您确认或拒绝的站点上进一步。 而且您的房东已经选择向谁租房。


可以在Domain.com上查看房地产市场。


食物


我认为在悉尼找到符合您口味的食物也就不足为奇了。 毕竟，第一代和第二代移民如此之多。 我在亚历山大市工作，在我的办公室附近有两家泰国咖啡馆，也有约四家中国和日本人。 最有趣的是，来自这些国家的移民在所有这些咖啡馆工作，因此您不必担心食物的质量。


我和我妻子有一个小传统-周末去鱼市 。 在这里，您总能找到最新鲜的牡蛎（大牡蛎12片-约21澳元）和美味的鲑鱼，每250克约15澳元。 最重要的是，您可以立即购买香槟或美酒作为开胃酒。



对我而言，澳大利亚人无法理解的一件事。 在这里，每个人都在乎健康饮食，因此面包不含麸质和有机食品，但是在办公室吃午餐时，每个人都喜欢磨玉米饼或汉堡。 一个非常受欢迎的套餐是炸鱼薯条 ，几乎在快餐店的任何地方都可以找到它。 我认为，就这组的“健康”而言，一切都很清楚-“连击”。


澳大利亚牛排 -许多人发现世界各地的当地美味。 一块好的牛排在餐厅要花费25-50澳元 。 在商店中，您可以花10到15欧元购买，然后在家中或在公园的烤架上烹饪（免费） 。


如果您是奶酪或香肠的爱好者，在这里您将拥有一个天堂。 也许我只在欧洲看到过类似的不同杂货选择。 价格是非常忠诚的，对于200克Brie型煤，您将支付约5 AUD。


交通运输


在悉尼拥有自己的交通是必要的 。 所有娱乐活动，例如海滩，露营地，国家公园 -只能开车。 巴士或地铁票价平均为3澳元。 最重要的是，这会浪费等待时间。 出租车非常昂贵 -平均行程15分钟将花费25澳元。


汽车价格低得惊人 。 我们经常骑单板滑雪和尾流，所以我们只需要有一辆顶部有行李箱的汽车。 我们认为理想的是2002 RAV4 。 这是我买过的最好的价格质量的产品之一。 注意， 4500澳元 ！ 起初，我们一直在寻找收获，但是在6,000公里之后，我们以某种方式平静下来。 最重要的是，尽管行驶里程很长，这里的汽车状况仍然很好。


但是，我们也使用摩托车。 主要优点是到处都是免费停车！ 但是您应该遵守时间安排，否则可能会面临大约160澳元的罚款。


运输保险分为三种类型：


  • 每年必须进行一次，仅涵盖生理性损害。


  • 如果您想赔偿车辆的损坏，则需要支付额外的保险 ，大约300-400美元。



就在前几天，我的同事讲述了有关他的朋友赶上法拉利的令人恐惧的故事。 他没有保险， 并向业主支付了95,000澳元 。 此外，该保险还包括疏散和更换汽车的费用，否则您将自掏腰包支付。

  • 第三种类型类似于CASCO （损坏是独立支付给您的车辆的）

对我来说，很有趣的是，如果您拥有全部权利，则可以喝1-2瓶啤酒并开车，但是这里的超额罚款只是空间而已


在单击它之前，最好坐下来准备氨气（或Corvalol）
