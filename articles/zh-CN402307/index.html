魅族Pro 5 Ubuntu Edition智能手机的滥用

这不是对智能手机或运行它的Ubuntu Touch操作系统的评论，而是对如何使用智能手机上设置的Linux技术完成某些任务（通常使用笔记本电脑或台式机）的描述。



大约半年前（在他的生日），以高端配置购买了魅族Pro 5智能手机。 在第一周，中文Android Flyme被拆毁，并安装了Ubuntu Touch OTA-12操作系统。 我必须立即说当时（现在也是）的操作系统仍是原始的，但是智能手机可以正常执行手机的功能。 通话，短信，通过WiFi将4G Internet分配到笔记本电脑-一切正常。

大约在同一时间，Raspberry Pi 3配置了Ubuntu Mate-并进行了将其用作家用台式计算机的实验，但不幸的是，该台式计算机没有成功-一方面，Pi 3相当脆弱，我想拥有一些东西-它就像一台个人笔记本电脑-因为工作笔记本电脑（由雇主提供，不必出门就可以旅行）和工作iPhone仅用于满足工作需要。

结果，安装了Ubuntu Mate的microSD卡迁移到了魅族Pro 5智能手机，并安装在/ home / phablet / PI文件夹中，然后使用简单的命令在chroot环境中启动了VNC服务器。

因此，您可以随时从笔记本电脑或任何其他设备（例如，iPhone）使用vncviewer或Microsoft RDP Client连接到Ubuntu Mate桌面。 而且，如果您将蓝牙键盘和鼠标连接到魅族Pro 5，并从Libertine容器运行vncviewer，则可以直接在智能手机上使用Ubuntu Mate桌面（非常小的图片，但是非常适合紧急情况）。

智能手机的资源大约是Raspberry Pi 3的两倍-8核，4 Gb RAM –这就是所有“办公”应用程序都能完美运行的原因。 LibreOffice（本文现在使用Writer编写），Chromium，Firefox，GIMP，Inkscape，Keepass2等。
当然，终端应用程序全都存在-mate-terminal，konsole，xterm等。 在chroot环境中，具有两因素身份验证（openconnect）的VPN也可以使用。

Windows应用程序。 事实证明，这也是可能的。

PL / SQL Developer（+ Oracle Instant Client）和AutoCAD-通过wine和Exagear（仿真i686）工作。 而且很快-这张GT图是在AutoCAD中在智能手机上创建的，只需5分钟即可完成-绘制舒适！

Microsoft Office（2007）-如果Libre Office无法应付，则在紧急情况下启动并运行安装。



操作系统更新-Ubuntu Touch和Ubuntu Mate均使用apt-get完成。

下面是对一些方便的东西的描述。

SMS重定向（如有必要）（带有过滤器和排序）


由于短信存储在sqlite中，因此您可以使用SQL选择短信，例如： 

phablet@ubuntu-phablet$ echo "select timestamp, message from text_events where message like 'Example%' order by timestamp; " | sqlite3 /home/phablet/.localhost/share/history-service/history.sqlite

然后输入脚本以通过管道发送： 

 /usr/share/ofono/scripts/send-sms/ril_1 555555555 $message_string 0

通过sftp复制照片，视频和其他文件（从手机到手机）


您可能不知道智能手机位于哪个口袋中，或者配偶的笔记本电脑位于哪个房间，但是如果他们位于同一网络中，则可以通过ssh访问电话并将图片复制到笔记本电脑中： 

 phablet@ubuntu-phablet$ scp -p *.jpg laptop:/Users/wife_login/Pictures/

文件加密


特别有价值的文档和带有密码的Keepass位于已安装的Cryptkeeper文件夹中。

连接外接显示器/电视


与Microsoft无线显示适配器（Miracast）配合使用-USB接收器连接到电视或显示器。 同时，Ubuntu Touch进入桌面模式，所有电话应用程序成为桌面上的单独窗口。 它很少使用，因为图像质量仅适合观看视频，并且使用小文本非常不愉快-为此，从笔记本电脑中使用vncviewer。

通过智能手机进行互联网分发


它工作正常，有时用于连接iPhone以在Skype上通话，或用于通过WiFi将笔记本电脑连接到Internet。

个人和工作互联网流量的分离


如果希望不花费雇主的互联网流量（在工作时间内，在本地网络上），则可以在电话上删除通往Guest WiFi的默认网关，然后所有网络活动都将在4G网络上进行，并通过vnc显示在笔记本电脑屏幕上（或xrdp）。



连接键盘，鼠标，USB硬盘驱动器，USB驱动器，打印机


使用USB-C转USB适配器进行连接，您可以将USB集线器连接到该适配器上，例如鼠标，键盘等。

在家庭WiFi网络中发现了Samsung彩色激光打印机，并使用CUPS自动安装了三星彩色激光打印机，文档打印正确。

终端中的控制台mplayer，用于收听音乐


谁喜欢mplayer，您可以在控制台中开始播放音乐： 

 user@ubuntu-phablet$ cd Music; mplayer *.mp3

在控制台的另一个标签中-调​​音台（音量控制） 

 phablet@ubuntu-phablet$ padsp aumix

汞库


它使用TortoiseHg，主要存储库位于智能手机上，该存储库的副本位于多台家用计算机/笔记本电脑上。 要在智能手机上进行同步，Web服务器将开始使用TortoiseHg工作台。

从魅族Pro 5为另一部手机充电


智能手机一次充电即可工作几天，比iPhone长约2倍（我的主观看法是我并未精确测量），这非常令人满意。 此外，他可以通过USB-C转USB适配器与另一部手机共享费用。

总结


十年前，智能手机和计算机之间的性能差距正在逐渐缩小，因此，现在可以在智能手机上执行在个人计算机上执行的大多数任务。
