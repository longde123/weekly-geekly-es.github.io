Reparando em tempo real: como a Internet das coisas está mudando as regras do jogo para os fabricantes

É importante saber para quem ligar quando o equipamento avariar. Mas é melhor que essa pessoa esteja pronta para reparo com antecedência. A abordagem tradicional não é mais tão eficaz: podemos saber que o petróleo precisa ser trocado a cada cinco mil quilômetros, mas, na realidade, esse indicador não pode ser completamente diferente. E isso está mudando graças à combinação de big data e a Internet das coisas. Fabricantes armados com sensores e software para olhar dentro dos dispositivos e prever o que está acontecendo com eles. Eles podem diagnosticar o problema remotamente e fornecer o melhor serviço ao cliente.



A General Electric investe em sistemas inteligentes para seus clientes industriais - fornecedores de eletricidade, fabricantes de trens e aviões. Os especialistas criam modelos de equipamento virtual, como um motor de avião ou turbina a gás, usando a tecnologia Digital Twin . Essa ferramenta ajuda os engenheiros a avaliar as condições de trabalho atuais e a planejar um cronograma de manutenção, a pensar em possíveis opções para melhorar a eficiência do equipamento com base nos dados recebidos.

O mesmo sistema permite o uso de sensores integrados no parque eólico para criar cronogramas de manutenção para cada turbina eólica individualmente. Essa abordagem aumenta o ciclo de vida do dispositivo.

As versões mais recentes do software Internet of Things são poderosas o suficiente para analisar o fluxo de eventos e antecipar problemas antes que eles aconteçam. Isso se aplica ao conjunto de equipamentos - por exemplo, várias turbinas eólicas - e a cada um dos dispositivos individualmente.

Apanhando tendências, o fabricante do equipamento deve respondê-las - e o software também ajuda. Uma peça quebrada pode ser facilmente reparada, mas a empresa precisa obter informações precisas sobre a reparação, descobrir o que aconteceu com a peça antiga se ela foi substituída e - o mais importante - entender por que a falha ocorreu.

Do ponto de vista financeiro, um serviço personalizado baseado na IoT pode aumentar a receita, conforme confirmado pelo relatório PWC IoT. A Internet das coisas permite que você obtenha a fidelidade do cliente porque o fabricante do equipamento faz as previsões corretas para manutenção e repara o equipamento antes que ele pare de funcionar como resultado de uma falha.

Esta tecnologia foi introduzida como um serviço para os proprietários de automóveis. O sistema OnStar nos EUA para motoristas é um botão para pedir ajuda, mas quando pressionado, todas as informações dos sensores instalados no carro são transmitidas aos serviços - pelo menos a velocidade.



Os sensores coletam uma quantidade enorme de informações, mas a maioria delas não é usada. Em uma plataforma de petróleo com trinta mil sensores, de acordo com um relatório do McKinsey Global Instituteapenas 1% de todos os dados são usados ​​para otimização e previsão de processos. Os sensores restantes são usados ​​principalmente para rastrear anomalias que já ocorreram. Isso abre oportunidades para empresas que podem usar o máximo de dados possível para análises precisas: "Se você puder dizer ao cliente como economiza seu dinheiro, ele estará interessado", diz Colin Parris, vice-presidente da GE Software Research.



Os fabricantes de equipamentos podem usar os dados coletados para melhorar produtos futuros ou suas várias versões. Várias versões de modificações podem ser usadas para construir o motor de um avião, projetado para voar em distâncias curtas, ou para uma locomotiva que precisa puxar uma carga enorme.

"Este é um círculo vicioso", acrescenta o vice-presidente da GE Software Research, "quanto mais dados obtemos dos clientes, mais valor podemos oferecer a eles".
