Microsoft Azure Developer Camp Russland



23. Mai | 9:30 - 19:00 Uhr
Microsoft Office, BP Krylatsky Hills, Ul. Krylatskaya 171, Tagungsräume Uranus, Jupiter, Saturn
Moskau, Russland


Die Verbreitung der Cloud-Technologie verändert allmählich die Methoden der Anwendungsentwicklung und stellt Programmierer vor neue Herausforderungen. Anwendungen werden nicht konsolidiert, Vorgänge werden parallel und asynchron ausgeführt. Container werden zum Standard, wenn es darum geht, Anwendungen in der Cloud zu aktualisieren.


Nehmen Sie am Microsoft Azure Developer Camp teil, einer kostenlosen technischen Veranstaltung für Entwickler, die am 23. Mai in Moskau stattfindet. Im Rahmen der Schulung erwerben Sie Kenntnisse über die Funktionen von Azure und praktische Kenntnisse bei der Migration und Modernisierung von Anwendungen und Infrastruktur.


Während des Unterrichts müssen wir Container im Detail untersuchen, mit einer Vielzahl von Azure-Clouddiensten arbeiten, um containergestützte Lösungen zu implementieren, und verschiedene Techniken zum Erweitern von Containerfunktionen beherrschen, ohne den vorhandenen Code zu ändern.
