开始预订Pyra游戏机

Open Pandora的后继者（例如，他们在此处进行了介绍）达到了预购阶段。



实际上，它是基于ARM的计算机，配有游戏控制台控件，最初是根据开放软件和硬件的原理开发的。这是一种“黑客友好”的设备，也就是说，可以完全自由地进行任何形式的修改和改进，没有锁定的下载器，也可能只有公共领域的OpenSource驱动程序，图表等。



主要技术指标：
  • 5英寸720p触摸屏
  • TI OMAP 5双核ARM处理器Cortex-A15
  • 2GB RAM
  • Wi-Fi 802.11a / b / g / n，蓝牙4.0（3G / 4G / UMTS的单独版本）
  • 电池下方有32 GB的内部存储器，两个SDXC插槽和microSDXC
  • 6,000mAh可更换电​​池
  • 端口：HDMI视频输出，2个USB 2端口，1个Micro USB 3.0、1个Micro USB（正在充电），SIM卡插槽


一切如何开始


2007年，有3个人与社区一起销售GP2x Linux游戏机，他们积累了许多改进产品的想法。但是，制造商对此并不感兴趣。之后，他们提出了制作自己的控制台的想法。很快，在社区中发现了能够设计印刷电路板和设备外壳的人们。



主要思想是获得一种能够至少全速模拟Amiga的设备。从硬件和软件的角度来看，它应该尽可能开放。

开发与社区密切合作。它的许多成员都为操作系统和驱动程序的开发提供了帮助。在2008年，第一个工作原型被引入。预购从9月开始，到2008年12月，计划开始向客户发送设备。但是，鉴于一些主要与一家中国公司生产案件有关的延误，很明显，无法按时完成任务，用户仅在2010年才开始接收设备。然后在生产和管理方面以及2011年时都出现了许多问题。这个项目快要死了。总共向客户交付了约2700台设备，并计划了1200台。

2012年初，Michael Mrozek（也称为EvilDragon）领导团队，并采取了一些步骤来保存该项目。其中包括：找到生产商向德国的转移（到项目的祖国），并进行了一些建设性的改进。 800 MHz版本的所有预订都已完成，有一段时间，可以在网上商店中购买Pandora。

需求潘多拉。尽管当时（2010-2012）硬件已经过时，但由于平台的开放性以及强大的社区的影响，它很好地应对了主要任务，模拟了各种游戏平台。在2012年底，推出了带有1 GHz处理器的版本，该版本当然不会从根本上改变现状，但它使我们“更加轻松”。

到2013年底，停产的主要原因是一些零件和微电路的生产已停止。尽管如此，您仍可以在官方网站上购买许多备用零件，例如LCD屏幕，其电缆，电池，外壳零件等。

为什么现在一切都应该顺利通过


目前，在上一个项目的工作过程中，有一个人在经历过足够的痛苦和仇恨。所有技术和组织解决方案均受到社区的仔细监视和讨论。Pyra的主要目标是保留在Pandora中实施的所有好的解决方案，并修复坏的解决方案。因此，这是一个错误修复。
