USB Lightning Transcend JetDrive Go 300K闪存驱动器概述

嗨，GeekTimes！关于Apple设备需要扩展内存的问题已经提出了多少次？外部硬盘驱动器无法帮助一千名iMac和MacBook用户使用，但iPhone和iPad的用户却继续遭受煎熬。传统上，创见对这一细分市场表现出兴趣，并再次对新产品JetDrive Go 300K感到满意。 Lightning + USB 3.1-没有比这更新鲜的性能和可用性了吗？同时，我已经有一个来自Leef的16 GB闪存驱动器-Bridge-值得吗？阅读下切。



对于百万分之一的时间，我要写一个俗套：“剧院始于衣架，产品始于包装。” Transcend的产品在一系列简单的塑料水泡中脱颖而出-纸板包装带有透明窗口，在其中可以看到闪存驱动器。我不会写下纸板上确切印刷的内容-自己购买和阅读。盒子的背面有一个QR码-已扫描-AppStore中已打开一个页面-已下载的专有软件。没有它，什么都不会起作用。但是，以下更多内容。

Leef当时的设计既方便又紧凑，并且是整体的，但是进步并没有停滞不前。恕我直言，将手机放在支架上或在喇叭不从背后伸出时，将其握在手中更加方便，就像神话中动物的最佳代表一样。

认知信息：顺便说一下，可以与Lightning配合使用的SiliconMotion SM3269控制器内部是其兄弟SM3268，它完全用于带有USB Type-C连接器的双向USB闪存驱动器中。三星的内存也是最新的，由16纳米制程技术生产。



备用基准测试比较-Crystal Disk Mark。读取小块记录时，两个驱动器都会出现悲伤和悲伤，但是对于新玩具来说，线性读取/写入要好得多。



闪存驱动器有什么作用？更正用于复制文件。因此，主要测试将是将几部电影复制到USB闪存驱动器。使用大型BDRip进行验证是没有意义的，因为Transcend JetDrive Go 300（以及任何其他闪存驱动器）的目的是带给您更多的内容。

TeraCopy实用程序用于测量时间；它用于测量从正常工作的SSD到机器人的影片存储设备（第4部分）的复制质量相当好的时间。文件大小为3117881344字节，约2.9 GB。


创见赢得了明显的优势（现在我很清楚为什么我购买了新的闪存驱动器）。

行动应用程式


要在手机，播放器或平板电脑上使用驱动器，您需要安装Transcend JetDrive Go实用程序。您可以通过扫描包装上的QR码（我在一开始就提到过）或在AppStore中找到应用程序来执行此操作。 功能非常多样：您可以从闪存驱动器中打开文件，包括播放视频。应该指出的是，读取2.9 gig时的撕裂没有问题，但是使用BDRip时会刹车。另一方面，为什么BDRip电话需要12场或更多演出？ 您可以立即从设备中复制所有照片，视频和联系人，并“以防万一是消防员”保存它们（这是如果所有者不喜欢iCloud）。









您可以使用密码来保护应用程序的启动，但不能分配TouchID进行解锁，只能在屏幕上的按钮上拨老式按钮。您可以找到有关可用空间的信息，并阅读大量不需要的信息。

其实，为什么我要写所有这些？我只是吹嘘要升级便携式存储设备，可以很方便地随身携带电视节目到全国（它们适合整个季节）。当然，要说明一个事实：进步不会停滞不前。两年前，我的老好朋友Leef iBridge非常有趣，但是时间却要付出代价……现在，在Transcend JetDrive Go 300K的口袋里，只要他能活得更长一点，而且不输掉帽子……
