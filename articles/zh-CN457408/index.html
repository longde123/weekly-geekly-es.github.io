6月26日欢迎来到DINS Java EVENING

我们很高兴邀请对Java开发感兴趣的每个人参加下一次开放的DINS Java EVENING，该会议将在我们19岁的Staro-Petergofsky的办公室举行。

在会议上，我们将熟悉TextBack的经验，并了解它们如何从Vert.x迁移到Spring。 Genestack的一位同事将分享他在Gradle中从头开始在大型项目中构建构建系统的经验。

删减部分–有关报告和演讲者的更多信息，一个注册参加会议的链接。




报告书


“异步微服务-Vert.x还是Spring？” （Alexander Fedorov，TextBack的CTO）


亚历山大将谈论TextBack服务。 关于他们如何从Vert.x迁移到Spring，他们遇到什么困难以及如何生存。 那些想开始使用异步服务并希望为此选择框架的人将对该报告感兴趣。

Alexander在Java方面拥有8年以上的工业开发经验，现在他喜欢在初创公司工作。 他毕业于理工大学研究生院，在圣彼得堡理工大学的酥皮机上向学生教授了“神经信息学和新技术”课程，而Dell EMC教授了DevOps课程。

高级Gradle构建（Nikita Tukkel，Genestack后端团队负责人）


Nikita将描述针对大型和超大型建筑所特有的特定问题的解决方案。 该报告包含有关Gradle基础知识的很少信息，对于完全不熟悉Gradle的人来说，其某些部分可能无法充分理解。 对于那些担心在项目中构建有效的构建系统的问题（那些模块的数量肯定超过一百个）的人来说，该报告将引起他们的兴趣。

Nikita从事编程已有30多年，其中最近15年是使用Java。 电子工作流程，金融科技以及现在-生物信息学。 他喜欢Gradle，Kotlin和所有看起来都不像J2EE的东西。
