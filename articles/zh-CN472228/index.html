使用Visual Studio Code的9个技巧

哈Ha！ 今天，我们选择了9种最有趣的生活技巧来使用Visual Studio Code。 其中包括有用的键盘快捷键和功能小部件。 细节剪下！



上一部分： 18技巧

1.互动游乐场


在“ 欢迎”页面的右下角，有一个指向Interactive Playground的链接，您可以在其中交互式地试用VS Code的功能。 或者，您可以通过“ 帮助”>“交互式游乐场”去那里。



2.转到文件中的字符


键盘快捷键： Ctrl + Shift + O



您可以通过添加冒号@：来按类型对字符进行分组。



3.上下移动一条线


键盘快捷键： Alt +向上或Alt +向下



4.并排编辑


键盘快捷键： Ctrl + \

如果需要，您可以创建新的编辑器组并在它们之间自由移动。



5.在多个编辑器之间切换


键盘快捷键： Ctrl + 1 ， Ctrl + 2 ， Ctrl + 3



6.转到文件的开头/结尾


键盘快捷键： Ctrl + Home和Ctrl + End

7.开关侧栏


键盘快捷键： Ctrl + B



8.智能感知


键盘快捷键： Ctrl +空格键

使用“提示”小部件，您可以查看可用的方法，参数提示，简要文档等。
