爆米花时间离开了，但答应回来。哦，不，已经回来了



该程序用于观看电影和电视节目的用户Popcorn Time注意到该程序在过去几周内一直无效。 “爆米花时间”和该程序中最流行的fork也不起作用。这是因为加拿大的几位主要开发商都受到了版权所有者的法律压力。 MPAA已经设法宣布，这是该协会在其整个生存历史中对海盗的主要胜利之一。不过，只要固定几个点就足够了-前叉将再次正常工作。这样的工作已经在进行中，不是由第三方开发人员进行，而是由软件的创建者之一进行。

由于服务器关闭，绑定到PopcornTime.io域的主分支在10月下旬停止工作。一段时间后，MPAA的代表宣布了他们打算起诉加拿大开发商的意图。这行得通，关键的项目参与者拒绝了。但是后来PopcornTimes的几个用户决定还原该服务，他们完成了所有工作，现在客户端的工作副本转到Reddit。

固定版本可与新API配合使用，以便再次显示电影和节目。现在不再使用YTS作为电影提供者，而是使用TorrentsAPI。在演出中，决定开发自定义API。

现在恢复服务/程序的工作仍在进行中。特别是，即将更新的通知与爆米花时间相关的VPN服务VPN.ht的用户。该消息说：“已安装VPN.ht Movies API FIX。”这表明PopcornTime客户端也要重新启动。 VPN提供商似乎进行了一些更改，以允许该服务的用户访问该站点和Popcorn Time客户端。



现在一切正常，好像什么都没发生。的确，旧客户端无法在未安装更新的情况下运行，因此大多数旧用户还无法使用该程序。 Popcorn Time Wally的开发者，也是VPN.ht的创始人，宣布即将发布一个功能全面的客户端。

在此之前，开发人员要确保该软件涉及的新域不是新的MPAA攻击的主题。这是一个问题，因为诉讼仍与加拿大有关。没有在其中提到沃利，但在索赔中提到了他的名字。此外，在索赔和诉讼中都提到了VPN.ht。

Wally认为MPAA不应与“爆米花时间”作斗争，而应注意其自身的概念和技术。

Wally说：“ Popcorn Time的流行应该是MPA​​A如何建立对世界开放的流媒体平台的一个例子。”

不管怎样，似乎Popcorn Time的开发人员和接近客户的人们以及服务无法停止-技术正在发展，开发中包括了新的人。

储存爆米花（是的，那样脆），故事还远没有结束。
