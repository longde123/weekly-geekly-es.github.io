Pembelian GitHub selesai. Apa yang akan terjadi selanjutnya?

Pembelian GitHub dari Microsoft kini telah selesai. Baru-baru ini, CEO baru GitHub, Nat Friedman, membagikan pengumuman pengantar yang kami posting di sini. Rincian tentang layanan masa depan di bawah potongan.



Saya senang berbagi bahwa akuisisi Microsoft GitHub telah selesai. Senin 29.10 adalah hari pertamaku sebagai CEO. Jutaan orang mengandalkan GitHub setiap hari, dan saya bangga bisa memimpin perusahaan ini.

Ketika kami mengumumkan pembelian pada bulan Juni, saya membagikan dua prinsip untuk GitHub, yang harus diperbaiki dalam memahami:

  • GitHub akan beroperasi dalam tiga paradigma independen: sebagai komunitas, platform, dan bisnis. Ini berarti bahwa GitHub akan mempertahankan nilainya bagi pengembang, semangat dan keterbukaan komunitas yang khas. Kami akan selalu mendukung pengembang dalam memilih bahasa, lisensi, alat, platform, atau cloud apa pun.
  • GitHub akan mempertahankan filosofinya. Kami menyukai GitHub karena merawat para pengembang. Saya memahami dan menghargai ini, dan saya tahu pasti bahwa kami akan terus menciptakan alat yang indah, cepat, dan dapat diandalkan yang disukai pengembang.

Pada akhirnya, tugas saya adalah membuat GitHub lebih baik untuk Anda.

Saya menghabiskan beberapa bulan terakhir bertemu dengan ratusan pengembang. Gairah untuk GitHub luar biasa - baik di area di mana kami berhasil maupun di area di mana Anda ingin kami melakukan lebih banyak. Saya belajar banyak dari interaksi ini dengan para pengembang. Komunikasi dengan pengguna kami akan menjadi bagian utama dari pekerjaan GitHub.

Tiga gol akan menjadi yang utama bagi kami:

  • Jadikan GitHub tempat yang lebih baik untuk menciptakan komunitas dan tim yang produktif;
  • Jadikan GitHub tersedia untuk pengembang di seluruh dunia;
  • Keandalan, keamanan, dan kinerja.

Mari kita mulai dengan berfokus pada peningkatan praktik sehari-hari menggunakan GitHub, dan juga memperluas proyek pemotongan kertas kita bersama. Kami akan meningkatkan tugas-tugas dasar seperti pencarian, pemberitahuan, masalah / proyek dan platform pada perangkat seluler. Dan, tentu saja, Tindakan GitHub. Kami benar-benar ingin membuat mereka tersebar luas.

Kami percaya pada kekuatan komunitas. Sebagai komunitas pengembang terbesar di dunia, GitHub menyatukan lebih dari 31 juta pengembang yang dapat membuat, bekerja, berbagi, dan mengintegrasikan.
