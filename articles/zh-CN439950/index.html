第三届图灵测试：获奖者的参与条件和奖品

一个月前，我们在2月14日至16日的Opentalks.AI人工智能会议上讨论了如何进行俄语下一次图灵测试。 您仍然可以今天2月12日报名参加比赛，直到莫斯科时间23:59。 我们邀请所有人参加测试！



在削减条件下-参加者的条件和优胜者奖品的描述。

条件


参加带有聊天机器人的团队的测试是免费的。 报名参加比赛即表示您同意公开发售协议，该协议可在比赛网站上找到，我们也已在该网站上发布了API。 僵尸程序的初步测试将于2月12日至13日进行。 在会议期间，专家和所有人都将对其进行测试。 2月16日，组织者将宣布获胜者，他们将获得现金奖励。

关于API


参与者注册您的帐户并启动新的聊天机器人。 为了简化实现，使参与者可以同步API。 服务器通过HTTP访问聊天机器人并等待响应。 聊天机器人通信协议是一个小的JSON： 

{ "chat_id": "uuid", "request_id": "uuid", "text": "text" }

chat_id和request_id分别是聊天和请求的唯一UUID。 文字是来自用户的消息。

该漫游器应提供带有文本的简单响应： 

 { "text": "text" }

有一个单独的Telegram机器人，您可以在其中调试与聊天机器人通信的技术时刻。 在服务器端的竞争性Telegram机器人中，响应会有所延迟，因此无法在回答时确定这是机器人还是人。

奖品


在MTS的支持下，获奖者将获得奖品。

最好的聊天机器人将是最多专家误会一个人的机器人。

奖品：

  • 第一名-200,000卢布。
  • 第二名-150,000卢布。
  • 第三名-100,000卢布。

为发挥机器人作用的志愿者提供的奖励比其他人好：

  • 第一名-20,000卢布。
  • 第二名-15,000卢布。
  • 第三名-10,000卢布。

最佳猜想专家奖：

  • 第一名-20,000卢布。
  • 第二名-15,000卢布。
  • 第三名-10,000卢布。

除专家外，任何人都可以通过Telegram机器人@TuringTestRussiaBot与聊天机器人聊天。

在会议的最后一天结束时，不仅会宣布获胜者，还将宣布比赛的总体结果，还将展示最佳和最有趣的对话，主持人团队将提前选择。

我们在Opentalks.AI等着大家！
