Azure IoT Edge Tools (Vorschau) -Erweiterung angekündigt

Wir freuen uns, die Verfügbarkeit der Vorschau-Version der neuen Erweiterung für Visual Studio 2019 - Azure IoT Edge Tools bekannt zu geben . Die Erweiterung bietet umfangreiche Funktionen zur Unterstützung der Entwicklung von IoT Edge-Lösungen mit Visual Studio 2019:



  • Neues Azure IoT Edge-Projekt für verschiedene Plattformen (Linux amd64, Linux arm32v7, Windows amd64)
  • Neues IoT-Edge-Modul (C # / C) zur Lösung hinzugefügt
  • Bearbeiten, erstellen und debuggen Sie IoT Edge-Module lokal auf Ihrem Visual Studio-Computer
  • Erstellen und Herunterladen von Docker-Images für IoT Edge-Module
  • Führen Sie IoT Edge-Module in Local oder Remote Simulator aus
  • Bereitstellen von IoT-Lösungen auf IoT-Edge-Geräten (mithilfe von Cloud Explorer )



Anforderungen


  • Visual Studio 2019 : Die Workloads ".NET Desktop Development" und "Azure Development Workload" müssen installiert sein. Wenn Sie C-Module entwickeln möchten, müssen Sie auch "Windows-Desktop-Entwicklung mit C ++" installieren.
  • Docker Desktop . Sie müssen Docker CE ordnungsgemäß konfigurieren, das im Linux-Container- oder Windows-Container-Modus ausgeführt wird.
  • Zum Einrichten Ihrer lokalen Entwicklungsumgebung zum Debuggen, Ausführen und Testen der IoT Edge-Lösung benötigen Sie das Azure IoT EdgeHub-Entwicklungstool . Installieren Sie dazu zuerst Python (2.7 / 3.6) und dann iotedgehubdev, indem Sie den folgenden Befehl in Ihrem Terminal ausführen. Stellen Sie sicher, dass die Version Ihres Azure IoT EdgeHub-Entwicklungstools größer als 0.8.0 ist.

pip install --upgrade iotedgehubdev

Installation


Es gibt zwei Möglichkeiten, eine neue Erweiterung zu installieren:

  1. Laden Sie die neue Erweiterung vom Visual Studio Marketplace herunter und installieren Sie sie.
  2. Alternativ können Sie die Erweiterung direkt aus Visual Studio 2019 über das Menü Erweiterungen -> Erweiterungen verwalten installieren. Wählen Sie im Fenster "Erweiterungen verwalten" im linken Bereich " Online" aus und geben Sie " Kante" in das Suchfeld in der oberen rechten Ecke ein, um "Azure IoT Edge Tools für VS 2019 [Vorschau]" zu suchen und herunterzuladen.



Wie benutze ich diese Erweiterung?


Wir empfehlen, dass Sie sich zunächst auf die folgenden Ressourcen beziehen:

→ Verwenden Sie Visual Studio 2019, um Module für Azure IoT Edge zu entwickeln und zu debuggen (Vorschau)
→ Einfaches Entwickeln und Debuggen von Azure IoT Edge C-Modulen mit Azure IoT Edge-Tools
→ Dokumentrepository von Visual Studio Azure IoT Edge Tools

Ihr Feedback und Ihre Vorschläge sind für uns sehr wichtig, da sie es uns ermöglichen, uns zu verbessern. Teilen Sie uns Ihre Gedanken mit, indem Sie beispielsweise eine neue Funktion vorschlagen oder ein Problem melden. Sie können dies in unserem Visual Studio Azure IoT Edge-Tools-Repository tun.
