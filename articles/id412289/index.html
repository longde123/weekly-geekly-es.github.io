Pritunl - Server VPN untuk Anda dan teman dalam 10 menit


Sehubungan dengan acara terbaru di Rusia, saya ingin memberi tahu Anda tentang server VPN mandiri yang keren. Setelah instalasi sederhana, tanpa banyak kerja dan pengetahuan yang mendalam, Anda dapat membuat pengguna, menghasilkan profil VPN dan menonton grafik penggunaan jaringan.


Dan setelah mengamankan koneksi VPN untuk Anda dan orang yang Anda cintai, Anda dapat mencoba berbagi biaya untuk server virtual dengan teman dan kenalan Anda.


Akibatnya, VPS lima dolar dapat berubah menjadi alat yang ampuh untuk memotong kunci


Ulasan


Faktanya, Pritunl adalah add-on untuk OpenVPN dengan moncong web yang indah yang mudah dipasang dan langsung berfungsi. Untuk koneksi, klien openvpn akan melakukannya, dan klien reguler terpisah disediakan untuk desktop.

Instalasi


Instalasi sangat sederhana dan bahkan setiap penggemar pemula dapat menanganinya:


  1. Pertama, Anda perlu membeli server VPS di suatu tempat di luar negara Anda.
  2. Pada VPS ini, cukup menjalankan perintah yang dijelaskan dalam manual .
  3. Setelah itu, Anda dapat terus mengkonfigurasi melalui antarmuka web.
  4. Jika Anda memiliki nama domain dengan A-record yang dikonfigurasi, Anda dapat menentukannya di pengaturan, sertifikat Letsencrypt akan secara otomatis diterima untuk itu.

Ini adalah bagaimana Anda dapat dengan mudah dan tanpa kendala mendapatkan server VPN Anda sendiri.


Koneksi


  1. Untuk setiap pengguna, Anda perlu membuat profil terpisah di Pritunl.
  2. Tautan ke profil (atau profil itu sendiri) harus dikirim ke pengguna.
  3. Pengguna hanya perlu membuka profil ini dengan aplikasi dan klik connect.
