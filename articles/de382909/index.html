LightBlue Bean +: Bluetooth Arduino für das mobile Zeitalter



LightBlue Bean + ist ein Arduino-kompatibles Board aus der LightBlue Bean-Familie - eine Reihe benutzerfreundlicher, drahtloser Arduino-Boards. Es gibt lötfreie BLE-16-GPIO-Anschlüsse. Die Entwickler bieten auch einen Akku und eine breite Palette an Software an.

Sie können unter Bean + auf jeder Plattform programmieren - OS X, Windows, iOS, Android. Alle Steuerfunktionen werden drahtlos ausgeführt. Keine Drähte, keine Probleme.

Hauptmerkmale Bean +


Dies beinhaltet mehrere neue BLE-Funktionen: Beacon, MIDI, HID, ANCS und Observer.



Basierend auf der Karte können Sie einen intelligenten Assistenten erstellen, der Benachrichtigungen von gekoppelten Geräten, einem MIDI-Instrument, einem Keyboard, einer Maus oder etwas anderem anzeigt.

Eisenkonfiguration




Bean + verfügt über mehrere integrierte Module - zum Beispiel einen Beschleunigungsmesser und einen Temperatursensor. Der Beschleunigungsmesser kann verwendet werden, um MIDI-Maracas zu erstellen oder um eine Benachrichtigung zu erhalten, wenn es im zu Hause installierten Terarium zu heiß wird.

Der vergrößerte Aktionsradius


Mit dem neuen HF-Verstärker kann Bean + unter idealen Bedingungen mit jedem anderen Bean + in einem Radius von 400 Metern interagieren. Oder sind es 250 Meter, um mit einem Smartphone zu interagieren? Somit reicht die Reichweite des Geräts aus, um mit jedem Gerät in der Wohnung oder im Haus zu interagieren. Sie können beispielsweise die Musik in der Lobby einschalten, ohne aus dem Bett zu kommen.



Batterie


Für jede Bean + wird ein 600-mAh-Akku mitgeliefert, dessen Ladung für etwa ein Jahr Betrieb der Karte im Standalone-Modus ausreicht. Über den zusätzlichen Micro-USB-Anschluss können Sie das Gerät überall dort aufladen, wo Zugang zu Elektrizität besteht. Sie können auch das Solarpanel zum Laden verwenden, das das "Dadget" bereits auf Geektimes beschrieben hat.



Bean + eignet sich auch für Projekte, die einen stärkeren Strom benötigen, z. B. für Systeme mit Servoantrieben oder Hochleistungs-LEDs.

Optionales Zubehör


Bean + wird mit zwei Arten von Anschlüssen geliefert, einschließlich Grove-Anschlüssen, die von Seeed Studio entwickelt wurden.



Übrigens können Sie die Betriebsspannung wählen - Sie können zwischen 3,3 V und 5 V wählen.

BLE-Funktionen


Hier haben wir drahtlose Steuerung und Programmierung:



Wie oben erwähnt, können Sie Software für das Modul von jedem Betriebssystem aus schreiben, einschließlich Android. Sie können sowohl den Arduino IDE-Standard als auch on the fly verwenden.

Im Prinzip ist Bean + eine ideale Option für Projekte, die mit mobilen Geräten verbunden sind. Wenn Sie keine Lust haben, Code zu schreiben, können Sie mit LightBlue arbeiten, einer Softwareplattform, die es wert ist, hier verwendet zu werden.

Für Entwickler bietet das Unternehmen auch das offizielle iOS-, OS X- und Android-SDK an.

Die Interaktion zwischen Bean +


Ja, Boards können mit anderen Bean + Boards „sprechen“. Dies ist dank der BLE und des Beobachtermoduls möglich.
