Anton Arkhipov tentang kerja efektif dengan IntelliJ IDEA dan TeamCity di jug.msk.ru

Pada tanggal 1 November 2018, pada pertemuan komunitas jug.msk.ru dari pengembang Java Moscow, Anton Arkhipov berbicara tentang metode penggunaan dan fitur baru dari produk JetBrains - IntelliJ IDEA dan TeamCity .



Tentang Pembicara


Anton hingga saat ini bekerja di ZeroTurnaround , termasuk banyak artikel menarik di blog perusahaan . Setahun yang lalu, saya pindah ke JetBrains (yang produknya kami semua suka, menggunakannya setiap hari), sekarang melakukan TeamCity di sana.

Co-founder dan co-organizer komunitas DevClub , sudah ada di Tallinn sejak 2008. Dia memiliki pengaruh besar pada penciptaan JUG.ru ( cerita rinci tentang ini ), berbicara pada pertemuan pertama pada 25 Februari 2012 ( pengumuman , video ).

Tuan rumah permanen dari Pembekalan podcast. Dalam edisi 18, Alexey Abashev mewawancarai Anton selama satu jam pada 17 April 2012 di konferensi JavaOne Russia , dan setelah beberapa saat Anton menjadi tuan rumah bagi hampir setiap masalah.

Seorang pembicara yang sering hadir di konferensi, khususnya yang diadakan oleh Grup JUG.ru. Beberapa laporannya dalam urutan kronologis:

  • Apache Camel ( DevClub-2009 : video )
  • Pemrograman Ping-Pong ( DevClub-2009 : video )
  • "Pimp My IDEA!" ( DevClub 2011 : video )
  • Script dalam Aplikasi Java ( Joker 2013 : Presentasi )
  • “Loader kelas di Jawa: koleksi rake” ( JUG.ru-2014 : video )
  • "Java Agents, API Instrumentasi, dan Javassist" ( JUG.ru-2014 : video )
  • “Di kepala manajer produk” ( DevClub-2014 : video )
  • Java Agents ( Øredev 2015 : video )
  • “Berbicara tentang IDE” ( DevClub-2015 : video )
  • “Bekerja efektif dengan IDE (A)” ( Joker 2016 : presentasi , video )
  • “Javassist dalam layanan pengembang Java” ( JPoint 2015 : presentasi , video )
  • "Pergi!" ( DevClub-2015 : video )
  • “Buttel alat perakitan - Maven vs Gradle vs SBT” ( JPoint 2016 : video )
  • “Cara membuat profiler dari stik ... dan alat improvisasi lainnya” ( JPoint 2016 : video )
  • Bytecode for the Curious ( JPoint 2016 Student Day : video )
  • Wawancara Java yang Tidak Memadai ( DevClub-2017 : Video )

Tentang laporan itu


Andrey Kogun membuka pertemuan dan mewakili Anton.



Bagian pertama dari pertemuan itu, Anton mengabdikan diri untuk menunjukkan penggunaan efektif alat sehari-hari Java-developer - lingkungan pengembangan IntelliJ IDEA . Akselerasi dan optimalisasi kegiatan harian dan rutin sangat mengesankan. Terlepas dari kenyataan bahwa saya terus menggunakan IntelliJ IDEA setiap hari, saya tidak tahu banyak - pastikan untuk merevisi video.



Pada bagian kedua, perhatian diberikan kepada TeamCity : tinjauan dibuat dari kemampuan produk, produk baru baru-baru ini, pengumuman peningkatan dan perbaikan di masa depan. Kisah ini diilustrasikan oleh demonstrasi yang hidup.



Seperti biasa, ada kesempatan untuk berbicara dan bertanya kepada pembicara. Selain itu, pertemuan seperti itu selalu disertai dengan komunikasi para peserta satu sama lain, yang juga menyenangkan dan bermanfaat.



Presentasi mini bagian pertama dan presentasi bagian kedua . Foto-foto dari rapat akan segera datang. Video akan muncul di YouTube (dengan pengumuman di VK dan Google+ ). Anda dapat berlangganan buletin dengan pengumuman pertemuan berikut jug.msk.ru.
