对于VNA或日文中文设备的问题

“那么有可能吗？”



在杰克·汉斯利（Jack Hansley）的一篇极端文章中，提到了最近出现在互联网上的一种名为NanoVNA的设备，即价格在50 kHz至900 MHz范围内的小型便携式矢量网络分析仪（VAC），价格为50美元。

“五十美元，卡尔！”

正如杰克所写的，没有理由不以这样的价格购买这样的设备，我完全同意他的看法。 这并不是我一直在使用无线电设备，而是我在Ali上订购了该设备，并等待直到确定参数。 尽管有些细微差别，但已经设法获得它的幸运者通常会感到满意。 如果您相信这些信息，那么它是由某位日本人在很久以前（三年前）开发的，并在Git上列出了源代码，而今年中国人突然开始复制和出售它（该设备当然不是日本人）。

主题中的人员会立即了解发生了什么，其余的我将向您解释VAC的价格从$ 3,000起，除了R＆S在哪里结束，没人知道。 您可以购买相对便宜的VAC，其动态范围为120dB，频率最高为1500MHz，价格为176,000卢布，这是一个典型的数字。 因此，许多人会理解我不愿等待设备到货，并立即了解它的排列方式和工作原理，特别是因为所有源代码都是开放的。

我转到开发人员的git，查看该图，并开始感到有些震惊。



我们省略了微控制器，触摸屏和按钮（尽管有一把有趣的摇椅），这完全是没有意思的，我们看一下仪表。

首先，我们观察到可编程发电机（很难想象VAC），然后观察到众所周知（且价格便宜）的m / s Si5351A。 但毕竟，她的输出频率范围显然不是900 MHz。 我以这样的方式看待日期，输出频率高达200 MHz，此外，输出信号基本上是直接的Coal（这就是我们所说的话），没有正弦。

好的，让我们继续看一下m / cx SA612AD（显式本地振荡器）上的三个平衡混频器。 同样，在一个输入上，允许的频率最高为500 MHz，在第二个输入上的最高频率为200 MHz，因此显然不会扩展900 MHz的范围。

然后，由于某些原因，混频器的输出被馈送到m / c TLV320AIC3204（音频编解码器）的输入，尽管MK包括一个在速度和分辨率上均不差的ADC。 通常，不是电路，而是持续的误解，以及IT如何提供所要求的频率范围仍然是一个谜。

我邀请读者思考和回答这个问题。

原则上，已经有足够的数据，但是我（作为一个诚实的人）将给出另外两个提示：

  1. 乐器的动态范围不均匀-70 dB（50 kHz-300 MHz），60 dB（300 m-600 MHz），50 dB（600 m-900 MHz），
  2. 在论坛上，一只无线电猫问某人是否可以将范围扩大到1200，他们合理地（附有频谱图）回答说，第七谐波已经太弱了。

扰流板方向
关键是该设备在发电机方波的谐波上运行。 自然，在奇次谐波下，因为信号占空比为1/2。
然后，要达到900 MHz的五次谐波，就需要在发电机和混频器的范围内（尽管后者并不明显），要求基本频率900/5 = 180 MHz。

好吧，即使这正是我们在研究中的设备上应用频率混合的方式，但是要获得特性，我们也需要将某些频率与其他频率分开并分别测量，这意味着在本机振荡器之后，我们需要低通，窄带，可调滤波器（或带通和可调高频滤波器）他）虽然可以实现，但不是一项基本任务。 但是，电路上根本没有滤波器，匹配电路中的几个RC链无法执行此功能。 而且，尽管如此，该设备仍在运行，如何运行？

我提出了一个最初似乎很巧妙的想法-我们在混频器的频率中引入了一个小的失谐。 然后，在混频器的第一个输入处，我们有f + 3 * f + 5 * f + ...，在第二个输入处f1（f-Δf）+ 3 * f1 + 5 * f1 + ...，在输出处，我们得到了频率的混合
（f + 3 * f + 5 * f + ...）*（f1 + 3 * f1 + 5 * f1 + ...）=（f * f1）+（3 * f * f1）+（f * 3 * f1 ）+（3 * f * 3 * f1）+ ...，则每个乘积变成（f±f1）+（3 * f±f1）+（f±3 * f1）+（3 * f±3 * f1 ）... =（Δf+（2 *f-Δf））+（（2 * f +Δf）+（4 * f-Δf））+（（4 * f +Δf）+（2 * f-Δf））+（ 3 *Δf+ 6 * f-3 *Δf）+...。 然后，如果您放置了一个从0到Δf的传输带，截止频率最高为3 *Δf的带通滤波器，则仅保留取决于一次谐波的分量，而所有其他分量都被抑制并且可以安全地进行测量。 编解码器中包含一个可调滤波器，唯一的问题是它的技术能力，它们应该足够了。

这个想法确实很好，它允许您使用曲折曲线代替正弦曲线，但是它有一个很大的缺陷-如何处理较高的频率（三次和五次谐波）。 毕竟，如果您拾起失谐信号，以使三次谐波的拍频落入滤波器频段，则一次谐波的拍频会更多。 我初步看到了三种可能的解决方案：

  1. 制作一个下限可调的滤波器，以使Δf落入频带，但Δf/ 2和Δf* 3/2不会。 任务并不简单，但是从根本上可以使用足够的设备来解决。
  2. 进行失谐，以便测量信号的三次谐波，飞行员的第五次谐波与之抗衡。 然后滤波器保持简单，但是不清楚信号的第五谐波如何处理，导频的三次谐波显然没有达到。
  3. 进行顺序测量-首先使用Δf，我们测量一次谐波的拍频，将Δf减小3（2）倍，并测量一次和三次的和，对于第五次也是如此。 减去一个就得到您需要的一切。 在这里，我对DSP领域的了解还不够，我无法评估这种选择的实际可行性。

可以从对编解码器功能的深入研究和对程序文本的分析中确定在此设备中实现哪个选项，但是从原则上讲，这不是那么有趣-已经概述了可能的解决方案。 就个人而言，我将争取第三，如果从根本上讲是可行的，我会在稍后解决。
