半透明背景

我想展示如何快速制作半透明的背景。 如果在CSS中使用opacity命令，则背景一定会变得透明，但背景上的所有其他内容也会变得透明。

如何解决这个问题？ 我在互联网上翻遍，发现了一种绝妙的方式，现在我将告诉您。

假设这是您的网站（html）： 

<html> <head> <meta charset="UTF-8"> <link rel="stylesheet" href="style.css"> <title>   </title> </head> <body> <div class="mid"> <p> </p>  .  . </div> </body> </html>

“”行包含CSS样式表。

在这里（称为style.css）： 

 body{ background: "   " ; * background-attachment: fixed ; background-position: top center; } .mid{ width: 1250px;** margin: 0 auto; background: rgba(161,21,207, 0.5);*** }

在中三类中，该行以RGBA格式设置半透明颜色的颜色。 它的4位数字，在这种情况下为0.5，是0到1之间的透明度值。

好简单 试试吧，您一定会成功！

*-在此处插入网址图片
**-设置页面宽度（为此，我们在html代码中创建了类）
***-rgba是rgb格式的颜色。 最后一个字母（a）表示背景的透明度。 其值是从0到1的数字。0是完全透明的背景，而1则相反。
