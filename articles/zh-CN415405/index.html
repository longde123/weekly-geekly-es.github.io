富士通PRIMERGY RX2540 M4服务器审查和测试

新技术的迅速发展通常会导致所使用的设备无法应对公司任务的情况。 您必须升级现有服务器系统群，或者将其更改为生产力更高的服务器系统。 在这种情况下，有必要充分评估这种更换的选项，以了解设备的所有优缺点。 我们决定测试Fujitsu PRIMERGY RX2540 M4服务器，并告诉其为用户提供的功能。 在本文中，我们将讨论服务器的组成，服务器的体系结构，并描述测试的进度和结果。


因此，我们研究的主题是安装在19英寸机架中的2单元服务器（约1单元= 4.445厘米或1.75英寸）。

值得详细介绍每种类型的模块。

RDX（英文，可移动磁盘交换）是一项提高驱动器的容错能力和强度的技术。 此模块是在特殊情况下安装的硬盘，无法牢固固定磁盘。 该技术避免了硬盘驱动器因振动或从小高度跌落而导致的过早故障。

LTO（英语：Linear Tape-Open）-在磁带上记录数据的技术。 如今在磁带上记录信息似乎有些奇怪。 然而，LTO显示出良好的寿命结果。 根据制造商的说法，磁带最多可以存储30年的信息（铁路最长可以存储10年）。 另外，与从皱折的铁路板相比，从撕裂的磁带中恢复数据更容易。 当前支持的LTO-7标准。 它允许使用存储容量高达6 Tb的盒带，写入速度为300 Mb / s和750 Mb / s，并具有信息压缩功能。 尽管该计划规定了标准的LTO-10。 它将允许您安装存储容量高达48 Tb且写入速度为2750 Mb / s的盒带。

测试中


为了使结果看起来不是抽象值，为了进行比较，我们介绍了其他中级服务器的测试结果。 让我们从处理器测试开始：

1）CPU女王。 测试的本质是计算整数。 这使您可以测试处理器分支和预测错误的能力。 为了解决这些问题，使用了皇后区问题算法。 您可以在此处找到有关此算法的更多信息。


测试显示每秒62,251个值。 结果表明，该处理器可以同时很好地处理多个任务，在虚拟化中使用时可以大大提高收益。

2）CPU PhotoWorxx-测试算术运算的性能并处理RGB图像。 其实质如下：

  • 用随机颜色的像素填充图像；
  • 图像旋转90度和180度；
  • 图像区分；
  • 色彩空间转换（在转换为JPEG格式时使用）。



测试结果显示每秒364.43亿像素。 这表明处理器在处理图像时处理得很好。
3）CPU ZLib-使用整数运算对处理器和内存子系统的性能进行测试。 通过创建ZIP存档来完成测试。 为此，请使用打开的zlib库（ 有关该库的更多信息 ）。 通常，它用于测试多核系统。


该指标保持在525.2 MB / s。
4）FPU VP8-通过使用Google VP8或WebM 1.1.0编解码器压缩视频来测试处理器。 对扩展为1280x720且速度为8192 kbit / s的视频流进行1遍编码（考虑到最大配置质量）。 框架本身是使用多种Julia分形生成的（ 更多内容 ）。 称为CPU和FPU的处理器测试之间有什么区别？ 事实是，第一个处理器可以执行严格定义的指令集。 为了扩展处理器的功能，开发了执行特定数学运算并将结果返回给主处理器的协处理器。 这样可以提高计算机的整体速度。 今天，协处理器已集成到中央处理器中，边界开始模糊。 FPU测试是对处理器中“协处理器”的测试。


测试显示6493个单位。 尽管许多协处理器已集成到CPU中，但它们并没有显示出最高的结果。 要使用视频流，应使用单独的视频卡。 在我们的示例中，CPU将部分处理器能力专用于处理视频图形。

5）内存读取-测试从RAM到处理器的数据传输速度。 那就是处理器从RAM读取数据的速度。


测试表明，处理器以71320 MB / s的速度从内存中读取数据。

6）内存写入-测试将数据从处理器传输到RAM的速度。 换句话说，处理器以什么速度将数据写入存储单元。


测试表明，处理器以51299 MB / s的速度将数据写入RAM。

7）内存复制-测试通过处理器缓存将数据从一个存储单元传输到另一个存储单元的速度。 也就是说，处理器将以何种速度将数据从一个存储位置复制到另一个存储位置。


测试表明，处理器以61469 MB / s的速度将数据从一个单元复制到另一个单元。

8）内存延迟-测试处理器从RAM读取数据的平均时间。 如果先前从内存读取的测试显示读取速度，则此测试显示平均延迟时间。


该测试表明，读取时的平均延迟时间为85.7 ns。

权衡利弊


注意优点：

1）紧凑的服务器尺寸

2）HDD / SSD有16个托架，可让您构建磁盘阵列故障保护子系统

3）光盘驱动器是一个重要的优势。 通常，有时需要从光盘还原系统。 原因是系统始终检测到光盘驱动器，并且各种总线（例如USB 3.0）都需要驱动程序，否则在系统启动时初始化可能会失败。

4）连接RDX和LTO模块。 它们使您可以比普通的HDD和SSD驱动器更长地存储信息。

5）存在2个电源，在电源出现故障或其中一个单元发生故障时，可以提高服务器的弹性。 这是一个非常重要的优势，因为许多服务器使用一个电源即可释放。

6）具有广泛连接性的主板：

  • 支持2个处理器。
  • 24个用于RAM的插槽，总容量高达3 TB。
  • 6个PCIe插槽，可用于添加视频适配器，连接其他网络适配器等等。

7）存在以10 Gbit / s的速度运行的网络控制器。 应该注意的是，控制器包含2个端口，这增加了它的容错能力，并可以使用流量平衡。

8）在复杂的任务和高速内存访问方面，令人印象深刻的处理器功能在测试Queen CPU，内存读取/写入/复制中得到了证明。

现在让我们来谈谈缺点：

1）内置RAID控制器。 它的负号体现在少数几个端口上。 如果托架带有16个磁盘，则必须购买一个额外的控制器才能同时使用所有可用磁盘。

2）视频流和视频加速测试中的性能较差。 所有的负载都落在中央处理器上。

总而言之，值得注意的是在已安装的处理器能力和RAM以及对其进行高速访问方面的良好特性。 大量的磁盘空间也是一个优点。 所有这些使您可以将服务器用作托管具有大量数据（数据库，公司邮件，云服务），虚拟化等服务的主机。

缺点包括用于8个端口（带有16个驱动器托架）的内置RAID控制器，以及缺少视频卡，无法与使用3D模型和视频加速的现代图形应用程序一起使用。 通常，如果需要，主板可让您安装其他RAID控制器，视频卡，网络控制器和其他设备。 “开箱即用”的服务器显示出非常好的效果，并且在用于特定任务时可以进行升级和升级。
