Hentikan etsa papan sirkuit tercetak di rumah - pesanlah di pabrik

Dalam artikel terakhir saya tentang Interkom dengan MQTT, saya melakukan survei tentang artikel mana yang akan saya tulis selanjutnya. Pilihannya ada pada urutan pembuatan papan sirkuit tercetak, jadi saya akan memberi tahu Anda sedikit tentang itu. Jika artikelnya masuk, saya akan menulis tentang topik berikutnya dari pemungutan suara.

Saya tidak pernah memaksa Anda untuk segera menuangkan besi klorida / hidrogen peroksida Anda, biarkan mereka untuk membuat prototipe. Saya hanya ingin menunjukkan bahwa sama sekali tidak sulit untuk memesan papan dalam produksi hari ini, seperti yang terlihat oleh amatir radio pemula. Ada sesuatu yang ajaib tentang ini - untuk memegang di tangan Anda motherboard yang indah dari produksi kami sendiri.

Bagi saya, elektronik bertindak sebagai hobi, dan ketika saya membiakkan papan lain, saya hanya mengirimkannya ke produksi dan kembali ke sana hanya setelah menerima papan yang sudah jadi. Dengan demikian, saya dapat memutar beberapa proyek sekaligus. Saya tidak meracuni papan dan prototipe pada kabel hanya sebagai pilihan terakhir.

Dalam artikel ini saya akan membahas persiapan minimum untuk produksi papan dalam program CAD Eagle. Saya tidak menggunakan program lain, tapi saya pikir artinya akan seperti ini.

Yah, untuk mulai dengan, Eagle CAD (kadang-kadang saya kadang-kadang akan menyebutnya Elang) adalah program berbayar sendiri, tetapi ada versi gratis dengan batasan. Untuk satu proyek, diizinkan untuk menggambar tidak lebih dari 2 lembar sirkuit dan berkembang biak tidak lebih dari 2 lapisan papan dengan luas hingga 80 cm2. Saya punya cukup untuk saat ini. Tapi saya selalu menggunakan versi saat ini dari situs web produsen, dan bukan sampah yang rusak.

Saya tidak akan belajar cara bekerja dengan program ini, ada artikel bagus dari DiHalt untuk ini (tautannya ada di akhir artikel), tetapi saya akan membahas apa yang perlu dilakukan untuk memesan papan.

Tata letak papan dan kontrol DRC

Sebelum memulai pengkabelan, perlu untuk membiasakan diri dengan standar yang memungkinkan pabrikan dalam pembuatan - ini adalah ukuran minimum dan celah yang diizinkan. Tergantung pada mereka berapa banyak Anda mendapatkan biaya yang benar. Untuk memulai pengkabelan, hal utama di antara mereka adalah ukuran minimum trek dan vias. Pabrikan selalu menunjukkan kemampuan teknologinya di situs dan hampir semua orang sekarang membuat trek dari 6 mil (0,150 mm). Tentu saja mungkin dan lebih sedikit, tetapi lebih sering akan lebih mahal. Saya menyarankan Anda untuk tidak menggiling dan berkembang biak lebih besar - Anda akan lebih nyaman.

Katakanlah Anda telah, setidaknya, memisah papan sirkuit. Sekarang Anda perlu memeriksa celah dalam mode otomatis di seluruh papan. Pabrikan terkadang melampirkan file untuk mengontrolnya di berbagai program. Misalnya, OSH Park menyiapkan standarnya dalam file untuk diunduh . Pada prinsipnya, standar-standar ini dapat dipesan dari produsen lain, di mana pun kapabilitas teknologinya kira-kira sama.


* Buka file yang diunduh dalam item DRC atau konfigurasikan celahnya sendiri

Klik Periksa dan dapatkan banyak kesalahan. Untuk setiap kesalahan, Anda dapat melihat tempat penyebabnya, dan memutuskan: apakah benar, atau skor dan tidak melakukan apa pun. Urutan papan adalah prosedur sederhana: apa yang Anda gambar adalah apa yang Anda dapatkan, tugas program hanya untuk memberi tahu Anda kelemahannya.



Unggah ke situs web produsen

Semua produsen menerima papan dalam bentuk file gerber , tetapi ada pengecualian yang menyenangkan. Misalnya, Taman OSH yang sama memungkinkan Anda untuk langsung mengunduh file * .brd dari Orel atau * .kicad_pcb dari KiCAD dan kemudian melihat hasilnya.

Prosesor cam

Kembali ke file gerber. Untuk menghasilkan file-file ini dari Orel, Anda memerlukan prosesor CAM. Ini adalah file yang dikonfigurasi lapisan mana file untuk output.

Tentu saja, Anda dapat menyesuaikan sendiri outputnya sesuai dengan persyaratan yang sama dari situs web produsen, atau Anda dapat menggunakan yang siap jika Anda menemukannya.


* Elang akan menampilkan bagian atas, bantalan, lapisan vias ke file * .toplayer.ger

Tapi di sini Anda perlu berhati-hati, karena prosesor CAM ini akan menampilkan file dengan nama yang dapat dimengerti oleh satu produsen, dan yang lainnya akan bingung di dalamnya. Jadi kami melihat persyaratan untuk nama file gerber di situs web pabrikan dan memberi nama sesuai.


* persyaratan dari situs salah satu produsen

Pratinjau file gerber

Sebelum membayar pesanan, tidak akan berlebihan untuk melihat bagaimana file Anda dihasilkan.

Mungkin lapisan sablon sutra telah pindah ke sana atau font sudah hilang (masalah yang paling umum, selalu menulis dalam font vektor).

Ada berbagai layanan atau program online gratis untuk ini. File diunduh dalam arsip zip . Kadang-kadang fungsi tampilan dibangun tepat ke dalam formulir pesanan papan. Jadi, misalnya, EaseEDA memiliki pemirsa sendiri.
