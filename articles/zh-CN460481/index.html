无风扇性能计算机MIC-7000


MIC-7000系列计算机是紧凑型的，同时具有完全被动冷却的高性能工业计算机，同时配备了第八代的成熟英特尔处理器和现代芯片组。 MIC-7000计算机的一个显着特征是巨大的散热器，它占据了机箱的整个上部，并且无需使用通风格栅和风扇即可散热。

为了解决各种艰巨的任务，可以扩展计算机的配置。 专有的i-Module扩展模块允许您安装任何PCI-e x16板卡，包括全尺寸的GPU，以构建机器学习系统和启动神经网络。

MIC-7000在无人驾驶汽车中用作中央数据处理服务器，同时使用视频采集卡处理高分辨率视频，执行神经网络计算以及使用I / O端口和工业接口控制外围设备。


无风扇冷却系统



工业计算机通常用于恶劣的环境：极端温度，污染和灰尘。 在这种情况下，传统的主动空气冷却系统会迅速堵塞并失效，通风孔成为垃圾和大颗粒渗透的源头。 在MIC-7000系列计算机中，使用了完全被动的冷却系统，从而消除了由于冷却器故障而导致的设备故障。 这样的系统几乎不需要维护，并且完全安静。

工业接口和I / O端口


为了与经典工业协议兼容， MIC-7000计算机具有内置的工业RS-232 / 422/485接口和通用数字I / O端口。 还支持iDoor模块，我们在“ UNO-1000 / 2000系列的模块化嵌入式计算机 ”一文中对此进行了讨论，您可以在其中添加对专有协议（如PROFIBUS，EtherCAT，PROFINET等）的支持。


Flex I / O扩展模块


这些模块使您可以将其他连接器添加到前面板，例如HDMI，DVI，RS-232，LAN等。 使用连接到主板的无源延长线。
通用I / O模块（GPIO）可以使用AIIS-DIO32-00A1E和98R1750090E。

Flex I / O扩展模块表（可单击）

可信平台模块（TPM）



对于没有集成TPM模块的主板，有一个远程TPM模块PCA-TPM-00B1E，它使您可以将此功能集成到大多数MIC-7000计算机中。 该模块适用于对硬件保护有更高要求的系统，用于检查OS的引导程序并存储私钥。

CFast卡


MIC-7700支持带有高速热插拔SATA接口的CFast卡 。 这种类型的卡用于现代相机Blackmagick，佳能，Arri，并且可以用作SSD驱动器的紧凑替代品。


专有的i-Module扩展模块


工业系统通常需要PCI-e卡形式的高度专业化的硬件。 i-Module扩展模块是特殊情况下的外部PCI-e总线扩展，可连接到MIC-7000系列计算机以形成单个系统单元。

有形式为1.2.4 PCI-e x16 / x8 / x4端口的模块，以及用于连接旧板的常规PCI，这些旧板可在机床和自动化系统中找到。 要连接高功耗的板卡（例如生产性图形卡），请使用带有附加电源线的模块，例如MIC-75G20 。
模块还可以具有用于安装其他HDD / SSD磁盘的托盘。



带MIC-7700的i-Module MIC-75G20扩展模块支持总功率高达600W的PCI-e卡。 有两个驱动器托盘和一个可选电源

案例：制动盘质量测试系统


在MIC-7500计算机和MIC-75M20模块的基础上 ，启动了一个串行站，用于测试汽车制动盘的质量。 磁盘安装在旋转轴上，并且未扭转，速度数据由PCIE-1802 ADC卡控制。 特殊的锤子（如留声机针）使用具有16个模拟输入，1MS / s采样频率和16位分辨率的PCIE-1816卡读取光盘表面上的最小振动。 收集的数据被传输到分析系统，该系统生成有关磁盘质量的报告。


案例：无人机控制器


任务是从自动驾驶汽车系统的不同传感器构建一个集成的数据处理系统。 为了以最小的延迟实时处理视频流，使用了通过DVP-7640E捕获板通过SDI连接的摄像机，因为使用IP摄像机会带来不可接受的高延时-在400-500毫秒范围内。 来自摄像机的视频由“机器视觉”系统处理，为加速使用了NVIDIA Tesla P4图形处理器。

基于MIC-7700Q计算机的无人驾驶汽车系统的组件示意图

可以通过CAN总线访问汽车电子系统，例如发动机控制器（ECU），各种传感器和驾驶员辅助系统（ADAS）。 要将计算机连接到CAN总线，请使用iDoor模块PCM-26D2CA。

系统的主要组成部分：

  • MIC-7700Q-具有第7代Intel处理器，Q170 / H110芯片组的无风扇计算机
  • MIC-75S20-具有1个PCIe x16、1个 PCIe x4、2个 HDD / SSD插槽的i-Module扩展模块
  • PCM-26D2CA-具有CAN总线接口的iDoor扩展模块
  • DVP-7640E-用于8个SDI通道的视频捕获卡，带有H.264硬件编解码器
  • TESLA-P4 -NVIDIA Tesla P4加速器


结论


MIC-7000是适用于广泛应用的紧凑型解决方案。 得益于被动冷却和扩展的工作温度范围，它们可以在不利条件下使用。 通过i-Module，iDoor和Flex I / O扩展模块，您可以为各种任务配置任何选项，与大多数扩展潜力通常受限的嵌入式系统相比，这是一种更加灵活的解决方案。
