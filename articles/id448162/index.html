Utilitas Solarwinds Gratis untuk Memantau, Mengelola Infrastruktur dan Keamanan TI



Kami mengenal Solarwinds dengan baik dan telah bekerja dengannya sejak lama, banyak juga yang mengetahui produk mereka untuk pemantauan jaringan (dan tidak hanya). Tetapi tidak diketahui secara luas bahwa mereka memungkinkan Anda mengunduh empat lusin utilitas gratis yang baik dari situs web mereka yang akan membantu mengendalikan perangkat jaringan, mengelola infrastruktur, database, dan bahkan menangani insiden. Bahkan, perangkat lunak ini terpisah dari produk berbayar mereka. Semua utilitas 100% gratis, bukan versi percobaan. Di bawah tautan kucing ke deskripsi dan unduh.

Dalam ulasan ini, utilitas untuk:

  • manajemen infrastruktur jaringan;
  • Manajemen infrastruktur TI;
  • Manajemen keamanan TI
  • manajemen basis data;
  • Organisasi layanan Help Desk.

Manajemen Infrastruktur Jaringan


Edisi Gratis ipMonitor




Alat sederhana. Antarmuka menetapkan berbagai alamat, itu polling mereka dan memberikan node yang tersedia untuk pemantauan jarak jauh. Alat pemantauan yang mungkin: ping, WMI, SSH. Di server, dapat melakukan pemeriksaan standar seperti CPU, Memori, Disk. Ini adalah versi gratis dari kakak ipMonitor (tidak gratis) dan mendukung hingga 50 node. Untuk tugas kantor atau tes kecil, tidak ada apa-apa sama sekali. Ada video pendek dengan deskripsi .

Bundel alat aliran




Replikasi, pengujian, dan konfigurasi perangkat untuk mengirim arus lalu lintas. Alat ini terdiri dari tiga bagian: replikator, generator, dan konfigurator. Replikator dapat menerima dan meneruskan lalu lintas netflow ke satu atau lebih penerima. Generator - menghasilkan netflow untuk pengujian, memeriksa pengaturan firewall dan tujuan lain. Configurator secara otomatis mengkonfigurasi perangkat jaringan untuk mengirim netflow ke perangkat target. Ada video pendek dengan deskripsi .

Traceroute NG




Utilitas ini merupakan add-on untuk tracert sistem. Melakukan analisis rute jaringan, dapat menjalankan paket menggunakan protokol TCP dan ICMP. Ada video pendek dengan deskripsi .

Pemindai port




Menghasilkan daftar port yang terbuka, tertutup, dan difilter untuk setiap alamat IP yang dipindai. Itu dapat memindai melalui TCP dan UDP, dan setelah pemindaian, Anda dapat mengunggah semua data yang dikumpulkan ke file. Bekerja termasuk melalui baris perintah.
Ada video pendek dengan deskripsi .

Monitor perangkat jaringan




Pemantauan perangkat jaringan. Utilitas ini dilengkapi dengan berbagai templat untuk pemantauan melalui SNMP, Anda dapat mengonfigurasi laporan dan peringatan. Tetapi ada batasan: pemantauan hanya satu perangkat didukung. Video pendek dengan deskripsi .

GNS3 Network Emulator




Emulator lingkungan jaringan. Utilitas berfungsi pada Windows, Mac dan Linux. Lebih dari 20 vendor peralatan jaringan didukung. Anda dapat meniru berbagai topologi jaringan dan melihat cara kerjanya. Ada video dengan deskripsi .

Response Time Viewer untuk Wireshark




Wireshark add-on untuk analisis paket yang mudah berdasarkan waktu respons. Ini mendeteksi lalu lintas untuk sekitar 1.200 aplikasi. Bekerja seperti kalkulator untuk waktu respons aplikasi. Ada video dengan deskripsi .

Bundel Penganalisis Jaringan & Bandwidth




Utilitas ini dirancang untuk memantau lalu lintas NetFlow, J-Flow dan sFlow. Antarmuka memiliki representasi dalam bentuk berbagai irisan: pada interaksi antara perangkat, pada aplikasi, pada domain dan perangkat akhir. Selain itu, dimungkinkan untuk merekam data hingga 60 menit.

Server TFTP




Server TFTP dengan antarmuka grafis. Berjalan di Windows sebagai layanan. Ada video dengan deskripsi .

Pelacak Alamat IP




Memindai, melakukan perubahan, mengidentifikasi konflik alamat IP. Ini adalah adik dari Manajer Alamat IP senior - dapat mengelola hingga 254 alamat, tidak seperti 2 juta dalam solusi berbayar.

Monitor Bandwidth Real-Time




Pemantauan bandwidth antarmuka jaringan dan visualisasi metrik. Juga solusi junior untuk Network Performance Monitor (NPM).

Detail Panggilan Rekam Pelacak




Cari, filter, dan urutkan log CDM CCM.

Monitor IP SLA




Kumpulkan data IP SLA dari perangkat Cisco dan visualisasikan.

Kalkulator subnet canggih




Kalkulator IP untuk desain ruang alamat.

Kiwi Syslog Server Edisi Gratis




Analisis, penerimaan dan pengarsipan syslog dan SNMP. Ada video dengan deskripsi .

FTP Voyager FTP Client untuk Windows




Klien FTP tingkat lanjut untuk Windows. Mendukung FTP, FTPS, dan SFTP. Melalui klien ini, Anda dapat melakukan transfer file terjadwal.

SFTP / SCP Server




Utilitas berfungsi sebagai layanan Windows dan mendukung transfer file secara simultan ke beberapa perangkat.

Manajemen Infrastruktur TI


Kalkulator biaya untuk biru




Utilitas dapat menampilkan informasi keuangan untuk beberapa akun secara bersamaan. Antarmuka memiliki rincian biaya untuk tahun, kuartal, bulan, dan di samping itu, sumber daya berbayar yang tidak digunakan ditampilkan. Ini bekerja pada platform Windows dan Mac.

Event Log Forwarder untuk Windows




Paket peristiwa dari log Windows dan mengirimkannya sebagai syslog di mana pun Anda inginkan. Ada penyaringan yang fleksibel. Kejadian apa dari log Windows yang perlu dikontrol - kami menulis di artikel terakhir .

Solar-PutTY




Putty Lanjut. Ini memiliki antarmuka multi-tab dan pencarian yang nyaman untuk sesi yang disimpan. Ada video dengan deskripsi .

Vm monitor




Memantau kinerja VMware dan Hyper-V. Antarmuka menampilkan metrik kinerja dasar dan status mesin virtual.

Monitor kesehatan server




Dirancang untuk memonitor server fisik. Out of the box dapat bekerja dengan Dell PowerEdge, HP ProLiant, server IBM eServer xSeries dan hyperware VMware ESX / ESXi. Mendukung protokol interoperabilitas SNMP, WMI, dan CIM.

Monitor kinerja penyimpanan




Memantau susunan data Dell EMC, NetApp, IBM, Pure Storage. Ada video dengan deskripsi .

Pertukaran monitor




Memantau layanan Exchange utama. Ada video dengan deskripsi .

Enabler Eksekusi Jarak Jauh untuk PowerShell




Utilitas untuk konfigurasi otomatis layanan WinRM lokal dan jarak jauh.

Bundel admin untuk direktori aktif




Paket tiga utilitas untuk: mendeteksi akun yang tidak digunakan, mendeteksi workstation yang tidak digunakan dan mengimpor akun pengguna dalam AD. Ada video dengan deskripsi .

Alat Diagnostik untuk Agen WSUS




Utilitas untuk memeriksa ketersediaan layanan Pembaruan Windows (WSUS) di server jarak jauh.

Transaksi transaksi web




Utilitas untuk melakukan tes sintetis. Satu transaksi didukung untuk aplikasi web.

Wmi monitor




Utilitas untuk mengendalikan server Windows melalui WMI. Ini adalah server, karena hanya satu perangkat yang didukung.

SNMP Enabler untuk Windows




Konfigurasikan penghitung SNMP pada server Windows jarak jauh.

Manajemen Keamanan TI


Konsolidator Log Kejadian




Windows Log Consolidator dengan pencarian acara yang mudah dan pengaturan tanda bersyarat. Ada video dengan deskripsi .

Browser firewall




Utilitas untuk menguji aturan firewall. Mendukung impor konfigurasi dari perangkat Cisco, Check Point, dan NetScreen.

Penganalisa Izin untuk Direktori Aktif




Penganalisa hak akses untuk pengguna, anggota grup, dan hak untuk mengelola bola jaringan.

Manajemen basis data


Penganalisa Kinerja Basis Data Gratis




Penganalisa permintaan waktu-nyata untuk database SQL Server, Oracle, DB2, dan SAP ASE. Menampilkan kueri lambat. Ada video dengan deskripsi .

Peringatan Rencana SQL




Utilitas untuk menganalisis Rencana Eksekusi di SQL Server. Pantau pemanfaatan sumber daya seiring rencana berlangsung. Ada video dengan deskripsi .

Organisasi Help Desk


Meja Bantuan Web Edisi Gratis




Kelola insiden, masalah, dan tingkat layanan. Ada integrasi dengan AD dan laporan. Ini adalah adik dari versi yang berfungsi penuh.

Sebagian besar utilitas ini cocok untuk memecahkan masalah lokal, beberapa dapat digunakan sebagai sistem produksi. Dalam 2-3 tahun terakhir, Solarwinds telah secara aktif mengembangkan produk-produknya dan sekarang mereka sudah dapat dianggap sebagai sistem universal untuk memantau tingkat organisasi. Jika Anda mencari sistem pemantauan untuk peran ini, pertimbangkan untuk memasukkan Solarwinds.
