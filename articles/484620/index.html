Escribí esta publicación en relación con el reemplazo del procesador de gráficos de software para PlayStation 1 con un procesador de gráficos con DirectX11.

Bienvenido a Kat


El artículo presenta el resultado de mi investigación para mejorar el componente gráfico del emulador de PlayStation 1. La solución anterior se implementó en la rasterización de software de polígonos y texturas y, como resultado, al intentar mostrar en una pantalla con una resolución más alta que la resolución original de TV, artefactos de pixelación de polígonos y texturas La solución obvia es reemplazar el procesador de software por uno de hardware capaz de rasterizar con mayor precisión en una amplia gama de resoluciones de destino. No encontré las soluciones de rasterización de hardware existentes para PCSX: usan el controlador OpenGL para la representación gráfica, lo que crea un problema de compatibilidad con el código base del emulador Omega Red en WPF, DirectX9 y DirectX11. Solo había una solución: escribir un renderizador gráfico con DirectX11, lo cual hice.
