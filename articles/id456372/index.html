SimbirSoft mengundang para profesional TI ke Intensive Summer 2019

Perusahaan IT SimbirSoft sekali lagi menyelenggarakan program pendidikan dua minggu untuk para spesialis dan mahasiswa di bidang teknologi informasi. Kelas akan diadakan di Ulyanovsk, Dimitrovgrad dan Kazan.

Peserta akan dapat secara praktis berkenalan dengan proses pengembangan dan pengujian produk perangkat lunak, untuk bekerja dalam tim sebagai programmer, penguji, analis dan manajer proyek.
