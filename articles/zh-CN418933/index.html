OnePlus评论6.当中国人所做的一切与上次相同时

在俄罗斯，许多人喜欢中国的智能手机。 通常，在购买它们时，用户还会以无人看管的软件和不平衡的配置的形式获得许多问题。 但是，也有一些例外-当然，这些是小米和OnePlus手机。 而且，如果前者的解决方案价格相差很大，那么OnePlus设备在发布之时总是“顶级”。



OnePlus 6智能手机继承了整个OnePlus系列的传统-其特点确实是一流的。 让我们从回顾其主要技术特征开始。

屏幕 -6.28英寸，2280x1080（19：9），AMOLED，康宁大猩猩玻璃
平台 -高通Snapdragon 845（4 x Kryo 385 @ 2.8 GHz + 4 x Kryo 385 @ 1.8 GHz）
GPU -Adreno 630
内存 -6/8 GB
内置闪光灯 -128/256 GB
MicroSD支持 -否
SIM卡 -2个nanoSIM卡
相机 -主双16（f / 1.7，OIS）+20（f / 2.0）Mpix，前置16（f / 2.0）Mpix
网络支持 -LTE Cat。 16，乐队1/2/3/4/5/7/8/12/17/18/19/20/25/26/28/29/30/32/34/38/39/40/41/66 / 70
无线连接 -Wi-Fi 802.11 a / b / g / n / ac（2.4和5 GHz），蓝牙5.0，NFC
定位 -GPS，A-GPS，Glonass，BDS，伽利略
传感器 -近似值，照明，磁场，加速度计，陀螺仪，指纹
电池 -3300 mAh，支持短跑充电
尺寸和重量 -155.7 x 75.4 x 7.8毫米，178克
价格 -请参阅6/64 GB ，8/ 128 GB ，8/ 256 GB的链接

如您所见，这些特性非常出色。 时尚的格式屏幕和AMOLED（有人不喜欢这种技术，但是它很经济，并且可以提供出色的色彩），快速的处理器和GPU，大量的内存，许多内置的闪光灯，支持LTE频段20，许多定位选项。

在缺点中，值得一提的是缺乏对microSD的支持（尽管尚不清楚手机上可以使用的128 GB以上），以及Dash充电，因为智能手机使用USB 2.0标准与计算机进行通信（尽管是Type格式） C，这很好）。

外观与人机工程学


智能手机看起来符合创建此类设备的现代标准。



顶部，底部和边缘的细框架-屏幕几乎占据了所有可用空间。



在这种情况下，不可避免的邪恶是“单眉”出现，即在屏幕上方的摄像头，扬声器和传感器的切口。



这些按钮是触摸式的，如果需要，可以更改其位置。



如果从侧面看，设备非常薄。 左侧是音量摇杆。



插槽旁边是SIM卡插槽。 您可以扔掉两张nano-SIM卡，但是microSD支持，,，没有。



甚至令人惊讶的是，通过这种厚度，可以保留耳机插孔。 它位于下方，用于充电和同步的Type-C连接器以及一个扬声器。 如您所见，扬声器是单声道的，没有其他插头。



右侧是开/关按钮，以及OnePlus智能手机的标志-用于静音模式的特殊开关。 三位置开关-“正常模式”，“振动”和“请勿打扰”。 方便-有人开始响起时，我忘了在报告过程中关闭声音-无需爬上菜单就淹没了设备。



顶部只有一个麦克风插孔，用于降低噪音。



我的设备是Mirror Black版本，我想说的是它确实像镜子一样反映了周围的所有事物。



老实说，这样的“辉煌”甚至让我烦恼，但总的来说，我最喜欢OnePlus 3T设计。 这是一种很好的拉丝金属，很棒。



后面有两个摄像头，一个在另一个下面，在它们下面-一个闪光灯，在它下面-一个指纹传感器。 传感器的作用只是一颗子弹。



如果没有任何反映，这就是智能手机从背面看的样子。

电池寿命


与OnePlus 5T相比，第六版具有更大的屏幕和硬件平台，但电池容量仍保持不变-3300 mAh。 而且，尽管像Oukitel这样的公司的“砖头”粉丝会扭曲他们的鼻子，但实际上，由于屏幕经济，这足以正常运行。

我的智能手机在平衡模式下工作了大约一天，这大致相当于OnePlus 5T的电池寿命。 根据iXBT.com标准电池测试，读取模式下的电池寿命为23小时30分钟，在线视频模式下的电池寿命为14小时30分钟，而3D游戏模式下的电池寿命仅为3小时。



充电短跑充电可让您在1小时20分钟内为完全放电的智能手机充满电。
