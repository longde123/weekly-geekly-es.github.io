Configurando o LaTeX em texto sublime


Abaixo está um guia para instalar o editor de texto Sublime Text , seguido pela adição da capacidade de usar o sistema de layout de computador LaTeX . Como bônus, vamos ensiná-lo a entender o idioma de Julia .


Instale o MiKTeX


  1. No site, clique na guia Todos os downloads
  2. Selecione Net Installer, dependendo da profundidade de bits do seu sistema.
  3. Clique em Download e aguarde o download do instalador online.
  4. Depois de ler atentamente o contrato do usuário, marque a caixa de seleção Download MiKTeX
  5. Ao usar um proxy, insira seus complementos nas configurações de conexão ...
  6. Na próxima etapa, selecione MiKTeX completo
  7. Selecionando um servidor (de preferência um dos principais russos) e especificando um diretório, aguarde o carregamento dos arquivos necessários
  8. Siga a etapa 4 novamente, mas desta vez selecionando Instalar o MiKTeX
  9. Após a conclusão da instalação, inicie o TeXworks através de Iniciar
  10. Abra qualquer documento .tex ou crie um novo.
  11. Selecione pdfLaTeX no menu suspenso e clique no triângulo adjacente no círculo verde

Isso irá compilar e criar o arquivo PDF. Usando o TeXworks, você pode começar um trabalho completo; por exemplo, usando o LaTeX, você pode efetivamente coletar um diploma ou dissertação ( modelos prontos )


É muito mais conveniente usar o LaTeX através de Sublime Text. Além do realce conveniente e do preenchimento automático, aqui você pode ver a imagem ou a fórmula digitada sem compilar, ou seja, ver o resultado durante a digitação. A propósito, em Sublime Text, você pode digitar (e se você adicionar os plugins apropriados, em seguida, executar) os códigos de programa de muitas linguagens de programação e marcações: C / C ++, Java, MATLAB, PHP, HTML ...


Instalar texto sublime 3


  1. Baixe e instale o Sublime Text 3
  2. Baixe e instale o Sumatra PDF . Este programa leve permite visualizar documentos PDF, DJVU, FB2, etc.
  3. Após iniciar o Sublime Text, abra a linha de comando pressionando Ctrl + shift + p e comece a digitar Package Control: Install Package e, quando aparecer, clique e aguarde a conexão com os repositórios.
  4. Se, por algum motivo, não houver Controle de Pacote, abra o console pressionando * ctrl + ` (() e copie o feitiço no parseltang, que pode ser encontrado na página do plug-in de Controle de Pacote , pressione Enter e reinicie o Sublime Text.
  5. Na lista de plug-ins, insira LaTeXTools e, quando aparecer na lista, clique e aguarde a conclusão da instalação.
  6. Em seguida, iniciamos as configurações padrão: Preferências / Configurações do pacote / LaTeXTools / Redefinimos as configurações do usuário para o padrão e, em seguida, o sistema de verificação lá
    Se todas as etiquetas estiverem verdes, o Sublime Text fez amizade com Sumatra e LaTeX, e agora você pode criar ou editar documentos tex. Quando você pressiona ctrl + B, ocorre uma tradução e, se não houver erros, o PDF paginado é aberto. Ctrl + shift + B - selecione o tipo de montagem.

Mais sobre LaTeXTools


Encontrar online os manuais do LaTeX será fácil para você (por exemplo: Tutorial , Wiki ). Também será útil usar o editor de fórmulas on - line até que os comandos sejam armazenados na memória.


Julia em texto sublime


Em Package Control: Install Package, digite Julia , faça o download do plug-in e agora Julia aparece entre os outros idiomas na guia View / Syntax , o que significa que agora você pode digitar programas nesse idioma com luz de fundo e letras gregas (\ delta + press tab ), além de salvar arquivos no formato .jl .


Ao salvar arquivos, será conveniente dividir em módulos dentro: 


 module somename # functions, variables # and some things end


O código digitado no texto sublime e salvo no formato .jl pode ser executado no REPL 


 #     cd("C:\\Users\\User\\Desktop") #   include("MDPSO.jl") #       #      PSO.parabol([2,3,5])


Isso conclui um pequeno guia. Todo o layout conveniente e iluminação agradável!
