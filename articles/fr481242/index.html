Nouveaux cours et guides vidéo gratuits pour C #, .NET et ASP.NET pour les débutants

Si vous pensiez commencer à apprendre le C #, c'est le moment de commencer! Notre collègue Scott Hanselman travaille sur ce projet depuis plusieurs mois et a le plaisir de vous présenter la page http://dot.net/videos

Vous y trouverez une centaine de courtes vidéos (dans un avenir proche, il y en aura encore plus) qui vous apprendront différents sujets, par exemple C # 101, .NET, créer des applications de bureau, créer des applications Web ASP.NET, utiliser des conteneurs et Docker, et même bases de l'apprentissage automatique. Vous trouverez tellement de vidéos sympas et mesurées pour les débutants. La plupart d'entre eux durent moins de 10 minutes et vous pouvez les trouver sur YouTube!

Avez-vous décidé? Je recommande de commencer par ces trois dans l'ordre suivant - C #, .NET, puis ASP.NET. Eh bien, après eux, choisissez les sujets que vous aimez le plus.



Si soudain vous n'aimez pas YouTube ou si vous avez des problèmes avec lui, toutes ces vidéos sont également téléchargées sur Channel 9 et peuvent être téléchargées localement via RSS! https://channel9.msdn.com/Browse/Series



Soit dit en passant, si vous aimez la vidéo, dites-nous quels autres sujets vous aimeriez voir!
