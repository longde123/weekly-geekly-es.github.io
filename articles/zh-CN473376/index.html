Windows 10X上的文档泄漏到网络上-具有两个屏幕的设备的新版OS


Windows 10X操作系统的代码计划于2020年初完成，并将于今年秋天进行商业发布。

微软继续保密Windows 10X的细节-Windows 10版本适用于具有两个屏幕的笔记本电脑和平板电脑。 但是现在，第一个重大漏洞使人们对该操作系统的设计和开发目标有了大致的了解，本机Windows 10应用程序和Web应用程序（PWA）将在相同的条件下工作。

屏幕快照和描述发布在santorini-os.azurewebsites.net上 。 该链接最早由用户WalkingCat aka @ h0x0d注意到 。 来自该站点的信息已被删除，但是另一个用户Boris aka @__milincic_已保存了包含页面截图的存档 。

从页脚来看，该站点是由Microsoft的Santorini Design工作组启动的。



该文档提到首席设计师Regina Son。 LinkedIn的个人资料显示 ，自2019年7月以来，她一直在Windows和下一代设备上工作。



Windows 10X界面设计准则：可用性，效率和简便性。 UI架构师说，该设计基于研究，测试和技术文档。

操作系统基本上应该提供最大的便利，而不限制用户。 例如快速设置菜单，其中列出了最常用的设置。

新的开始菜单（启动器）


“开始”菜单已重命名为“启动器”。 它包括用于搜索Web的行（以及在具有语音输入支持的计算机上搜索文档和应用程序），应用程序和网页的静态区域以及推荐文档的列表。 这是它的外观。


圣托里尼发射器

该文档说：“ Launcher帮助用户快速启动和恢复任务。” -搜索可轻松与Web结果，可用的应用程序和设备上的特定文件集成。 内容层次结构中的下一个是默认的应用程序网格，您可以对其进行自定义和更改以适合您的个人需求和偏好。 启动器会打开，并进一步帮助加速用户的动作，随着时间的推移，这些动作会进入肌肉记忆。 最后，根据最常用和最近使用的应用程序，文件和网站，动态更新推荐内容。 您可以返回个人工作流程，而无需从头开始。”

默认应用程序：边缘，邮件，日历，Word，Excel，PowerPoint，OneNote，团队，白板，待办事项，照片，存储，文件资源管理器，Spotify，Netflix，相机，纸牌，计算器，闹钟和闹钟，电影和电视，办公室，便笺，绘画，学习中心，设置，天气，剪裁和素描，录音机，Groove音乐，人，记事本，反馈中心，媒体计划，消息传递以及最多四个OEM应用程序。

在此菜单中，您可以将应用程序分组到文件夹中，然后将它们从网格（和从设备）中删除。

锁屏


基本上没有锁定屏幕，一旦打开设备，用户就可以快速进行身份验证（Restrained Lock型号）。 这与Windows 10型号不同，在Windows 10型号中，在身份验证之前，您需要从屏幕上删除“初步”锁定屏幕（“窗帘”）。


截至2019年9月，系统中的授权屏幕


系统中的授权屏幕可访问桌面，基本上没有锁定屏幕

支持Windows Hello脸部识别以及Windows Hello PIN。 从睡眠模式唤醒后，Windows Hello Face会立即识别用户并立即打开其桌面。

快速设置窗口


设置部分有两个级别：第一个显示最多十个最受欢迎的设置，第二个显示其余的设置。 第一个要编辑的列表。 默认值为：

  • 无线网络
  • 小区数据（显然是蜂窝网络数据）。
  • 输入语言
  • 撰写模式
  • 蓝牙功能
  • 飞行模式。
  • 屏幕旋转。
  • 专案

任务栏




访问常用的应用程序和站点，以及任务切换。 活动选项卡可以在前台或后台。 当应用程序的多个实例分组在同一图标下时，支持分组。 悬停时，Ul会显示此应用程序各种实例的缩略图。 任务栏显示状态“活动状态”（应用程序处于焦点状态）和“执行状态”（在应用程序运行时，带下划线的图标，无论它是否处于焦点状态）。

设计师说，他们将继续尝试使用不同的图标组（居中或左对齐），元素的数量和大小，显示的最近事件的数量，是否存在分隔符等等。

您不能肯定地说这个概念已经被管理层采用和认可。 但是无论如何，我们都可以通过设计人员的工作职责范围（即，Microsoft要开发操作系统的UI的方向）来得出结论。

Windows 10X


微软在10月初首先提到了新操作系统的开发，即“ Surface Neo Dual-Screen设备的OS”，并宣布了2020年的暂定发布日期。

操作系统可能会与这些设备本身同时出现。 似乎“ Windows 10 Lite”可以安装在其他设备上，包括笔记本电脑。

微软内部人士报告称，Windows 10X代码计划与Windows 10 20H1一起在2020年初结束，Surface Neo和其他具有新操作系统的设备将在圣诞节季节开始之前的秋天进入市场。

Microsoft尚未对文档泄漏进行正式评论。
