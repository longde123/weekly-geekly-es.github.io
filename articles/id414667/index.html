[Mini fan theme] Tabel kejuaraan FIFA di terminal



Halo

Artikel ini sangat singkat, tetapi akan berguna bagi penggemar sepak bola yang menghabiskan banyak waktu di baris perintah

Menggunakan paket npm wowcup , Anda dapat melihat hasil permainan tepat di terminal. Informasi diambil dari api.football-data.org

Kami menaruh tas ... 

sudo npm install -g wowcup

Kami sedang berusaha ...

24 jam terakhir 

 wowcup fixtures --last




24 jam berikutnya 

 wowcup fixtures --next




perlengkapan wowcup - memainkan streaming langsung
peringkat wowcup - semua game
kedudukan wowcup --tabel GROUP - untuk grup tertentu (ganti GROUP dengan a - h )

Untuk menghapus teks merah, tambahkan --quiet 

 wowcup fixtures --last --quiet

Itu saja, terima kasih semuanya.
