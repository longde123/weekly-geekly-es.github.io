在谷神星的亮点之上看到一种奇怪的雾霾


NASA / JPL-Caltech / UCLA / MPS / DLR / IDA

谷神星（Ceres）是太阳系中最大的小行星之一，继续使天文学家和业余爱好者感到惊奇。事实是，在火山口上方（该物体最明显的亮点位于其中）（还有其他一些尺寸较小的亮点），发现有雾霾。黎明探测器发现的这一现象“可能是谷神星这一地区大气存在的标志”。无论如何，加州大学的科学家克里斯托弗·罗素（Christopher Russell）如此说。

从今年3月开始，黎明进入谷神星的轨道（距地面4000多公里）。首次在近距离观察到类似的现象（实际上，黎明或新视野每天都在做或发现他们无法或更早发现的事情）。欧洲航天局的科学家在2014年使用赫歇尔太空天文台观察到了同样的现象。然后，科学家发现霾是水蒸气，这表明谷神星有1/4的水构成地质活动。

现在，越来越多的专家倾向于将斑点视为表面上水冰的出口，并且观察到的雾度证实了这一观点。谷神星上的斑点是一些盐的理论并不流行，目前还没有证实。


谷神星的表面覆盖着撞击坑（照片：NASA / JPL-Caltech / UCLA / MPS / DLR / IDA）

在黎明上，有一个红外光谱仪，可以轻松地区分矿物盐和水冰。到目前为止，该工具无法使用，因为探头距离小行星的距离相对较远，而且光谱仪有时会工作。

目前，该设备已经发现谷神星比以前想象的要小，但是比科学家声称的密度高4％。但是在谷神星上发现了一座5公里高的山，该山已经被称为“金字塔”，到目前为止，这是一个非正式名称。

现在黎明位于距谷神星表面约4000公里的位置，但下个月它将在约1500公里的距离处接近小行星，并将继续收敛。现在，星际站黎明继续向谷神星移动。几天前，该设备打开了引擎，开始执行操纵以接近谷神星。操纵成功了，现在该台站以螺旋形运动，距小行星表面仅3.9万公里。
