云解决方案架构。 OTUS的新课程

注意！ 本文不是工程知识，而是针对对云解决方案的开发和支持领域的教育感兴趣的读者。 最有可能的是，如果您对学习不感兴趣，那么您将不会对本材料感兴趣。



直到最近，当使用“云”一词时，每个人都在思考大气现象，现在大多数人已经与云存储相关联。 当前，一些最受追捧和高薪的专家是在敏捷开发和云解决方案体系结构维护方面知识丰富的专家。



Otus推出了“云解决方案体系结构”课程-基于实际组织转型项目和精心设计的框架建议，开发和支持云解决方案的最佳实践。 该课程主要面向架构师和开发人员，但也确保将以下配置文件开发到Cloud Native级别：

  • IT /软件架构师
  • 开发人员和DevOps工程师
  • 网络和系统管理员
  • 信息安全专家
  • 经理和团队负责人

几天前，该课程举行了一个公开课程，学生在其中学习Cloud Landing Zone域的体系结构设计并检查了主要域的体系结构模板。 您可以在记录中查看它，以了解培训的形式并结识老师。


12月18日20:00 将举行开放日 ，老师Vladimir Gutorov将回答有关Cloud Solution Architecture课程的所有问题，更详细地介绍该课程计划以及学生毕业后将学习的技能和能力。 Vladimir Gutorov-云架构师，Nordcloud的顾问。 他领导瑞典Husqvarna集团的CI / CD团队，在开发和实施复杂的端到端IT解决方案方面拥有20多年的经验。

为了上云解决方案架构课程，您必须具有以下知识：

  • 具有开发和/或维护应用程序的经验，最好具有DevOps Agile
  • 至少有一个云提供商的经验-Azure，GCP，AWS等

您可以通过入学考试，以了解您的知识和技能是否足以进行培训。

该课程基于一个真实的项目，该项目旨在通过从多云的环境（AWS + Azure + GCP）和分布式的Cloud Native微服务和无服务器应用程序的过渡，将公司部门从传统的瀑布模型转变为在自己的数据中心中开发整体应用程序的传统瀑布模型，转变为敏捷的DevOps模型。

在本课程的最后，您将学习如何进行Agile SCRUM项目以开发和发展云解决方案体系结构，并且您将能够创建符合结构完善框架原则的云解决方案体系结构（基础架构为代码）-优化业务流程，安全性，可靠性，高性能和成本优化。 此外，当然，您还将获得本课程的结业证书，最成功的学生将收到最受欢迎的IT公司的面试邀请。
