Elemental OS 0.4 Loki Beta lanzado

Hola a todos!
Dio la casualidad de que sigo activamente el desarrollo de Elementary OS y periódicamente publico noticias sobre esto a medida que las versiones están disponibles . Esta vez, el equipo de Elementary OS nos complació con una versión beta de la nueva versión de "elementary OS 0.4 Loki" basada en Ubuntu 16.04 LTS.

Los desarrolladores nos prometen un montón de nuevas características y errores para usuarios y desarrolladores, pero advierten que beta es beta y no debe usarse en ninguna producción. El sistema operativo (¿o distribución?) Está en desarrollo activo y gradualmente se volverá más y más nuevo.
¿Qué hay de nuevo para nosotros?

  1. Ubuntu 16.04 LTS Base Repository (Gtk 3.18, Vala 0.32, Linux 4.4)
  2. Usando GtkPopoverMenu - hermosas barras de desplazamiento nativas
  3. Notificaciones y tablero mejorados
  4. Negarse a usar repositorios inseguros (por defecto no hay utilidades para trabajar con paquetes deb * y agregar repositorios de terceros) y la transición al modelo de aplicaciones aisladas en contenedores.
  5. ¡Nuevos iconos y estilos hermosos para GTK!

¡Pruébalo, escribe un rastreador de errores y desarrolla software libre! Enlace de descarga https://elementary.io/en/developer
