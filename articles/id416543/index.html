PHP Digest No. 134 (24 Juni - 8 Juli 2018)


Dalam rilis: PHP 7.3.0 alpha 3, PHPStan 0.10, Yii 1.1.20 dan rilis lainnya, tawaran segar dari PHP Internals untuk membandingkan objek, sebagian dari alat yang bermanfaat, dan banyak lagi.
Selamat membaca!


Berita dan Siaran


  • PHP 7.3.0 alpha 3 - Rilis alpha terbaru sebelum fitur beku. Beta 1 dijadwalkan untuk 19 Juli. Namun, tanggal dapat dipindahkan jika diputuskan untuk memasukkan properti yang diketik dalam 7.3.
  • PHP 7.1.19
  • PHP ODESSA CONF 2018 - Pada 11 Agustus, konferensi PHP skala besar akan diadakan di Odessa. Untuk pembaca PHP Digest, kode promosi dengan diskon 10% tersedia: PHPDIGEST

PHP internal


  • PHP 8: apa yang diharapkan. Surat Zeev Surasky
    Demo kinerja PHP 7.0 vs JIT PoC untuk pembuatan fraktal
  • [RFC] Persahabatan Kelas - Voting mulai menambahkan kelas teman. Kelas ramah memiliki akses ke bidang pribadi dan kelas yang dilindungi yang dinyatakan ramah.
  • [RFC] Perbandingan objek yang ditentukan pengguna - Dokumen ini membahas masalah membandingkan objek dalam PHP. Diusulkan untuk menambahkan dua metode sihir baru: __compareTo dan __equals .
    Teks tersembunyi 
     class Example { /** * Returns: < 0 if $this is less than $other, * > 0 if $this is greater than $other, * == 0 if $this is equal to $other. */ public function __compareTo($other): int { ... } /** * @returns bool TRUE if $this is equal to $other, FALSE otherwise. */ public function __equals($other): bool { ... } }
  • dstogov / php-tensorflow - Berkat implementasi FFI , dimungkinkan untuk membuat pengikat ringan untuk perpustakaan pihak ketiga. Sebagai contoh, Dmitry Stogov menunjukkan pembungkus di atas TensorFlow .


Alat-alatnya


  • Infection 0.9.0 - Pembaruan kerangka kerja untuk pengujian mutasi AST.
  • PHPStan 0.10 - Analisis statis kuat untuk PHP. Kotak pasir online untuk verifikasi kode.
  • sebastianfeldmann / phpbu - Alat untuk membuat salinan cadangan file dan basis data. Mampu mengenkripsi cadangan dan mengirim ke penyimpanan.
  • retribusi / rilis-alat - alat CLI untuk merilis versi produk baru.
  • mediact / dependency-guard - Alat akan menampilkan dependensi komposer yang tidak digunakan. Serta yang digunakan dalam kode, tetapi diinstal sebagai dependensi dev.
  • ackintosh / snidel - Memungkinkan Anda untuk dengan mudah mengeksekusi kode secara paralel di persimpangan proses. Alternatif spasi / async
  • php-lock / lock - Implementasi mutex untuk PHP dengan dukungan untuk berbagai adapter (flock, memcache, mysql, redis, ...). Alternatif untuk symfony / lock .
  • Imangazaliev / DiDOM - Pustaka sederhana dan cepat untuk parsing HTML. Tutorial baru dalam dukungan.
