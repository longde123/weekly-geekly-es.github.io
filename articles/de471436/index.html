Neue 3CX App für Android - Antworten auf Fragen und Empfehlungen

Letzte Woche haben wir das 3CX v16 Update 3-Update und die neue 3CX-Anwendung (Mobile Softphone) für Android veröffentlicht . Das Softphone funktioniert nur mit 3CX v16 Update 3 und höher. Viele Benutzer haben zusätzliche Fragen zur Anwendung. In diesem Artikel werden wir sie beantworten und mehr über die neuen Funktionen der Anwendung sprechen.

Funktioniert nur mit 3CX v16


Beim Starten der Anwendung wird einigen Benutzern die Meldung angezeigt, dass das Programm nur mit 3CX V16 funktioniert. Hier geht es natürlich um die Serverversion. Sie können das Problem lösen, indem Sie den PBX-Server auf die neueste Version von 3CX v16 aktualisieren . Wenn Sie derzeit jedoch kein Upgrade auf Version 16 durchführen können, installieren Sie die vorherige Version der Android-App . Auf diese Weise können Sie 3CX weiter verwenden, bis der Systemadministrator den Server aktualisiert. Beachten Sie, dass diese Anwendung von 3CX nicht unterstützt oder aktualisiert wird und auch nicht mit Android 10 kompatibel ist.

Voicemail


Benutzer beschweren sich über die Art und Weise, wie Voicemail in der neuen Anwendung abgespielt wird. In der nächsten Version planen wir, zur vorherigen Wiedergabemethode zurückzukehren, mit der Sie eine Sprachnachricht abhören können, ohne die Voicemail-Nummer des Systems zu wählen.

Adressbuchzugriff


Derzeit benötigt die Anwendung Zugriff auf die Kontaktliste des Geräts, um das 3CX-Unternehmensadressbuch, die persönlichen Kontakte des 3CX-Benutzers (Nebenstellennummer) und das Adressbuch des Geräts zusammenzuführen. Daher wird jetzt jedes Mal, wenn Sie auf das Adressbuch der Anwendung zugreifen, eine Anforderung zum Zugriff auf die Kontakte des Geräts angezeigt, auch wenn der Benutzer dies in der Vergangenheit nicht zugelassen hat. Beachten Sie jedoch, dass die Anwendung niemals Kontakte von Ihrem Gerät zum 3CX-System überträgt.

Einige Benutzer möchten jedoch immer noch keine persönlichen Kontakte vom Telefon und keine von 3CX heruntergeladenen Arbeitskontakte mischen. In der nächsten Version blockieren wir standardmäßig den Zugriff auf das Adressbuch des Geräts. Wenn der Benutzer im Gegensatz dazu Kontakte kombinieren möchte, öffnet er in den Berechtigungseinstellungen der 3CX-Anwendung unabhängig den Zugriff darauf.



Gruppenanzeige


Auf dem Anwesenheitsbildschirm werden keine organisatorischen Benutzergruppen mehr angezeigt. Dies geschieht zum Auslagern der Schnittstelle, da dieselben Benutzer in verschiedenen Gruppen angezeigt werden können (schließlich kann ein Benutzer gleichzeitig in mehreren Gruppen sein). Wir planen, diese Änderung beizubehalten.

PUSH-Benachrichtigungen erhalten


Die Option "Beenden - PUSH ignorieren", die in der alten Anwendung enthalten war, wurde entfernt. Stattdessen wurden bequemere Möglichkeiten zum Verwalten von PUSH-Benachrichtigungen in verschiedenen Status angezeigt.
Sie können angeben, ob PUSH-Benachrichtigungen in dem einen oder anderen Status empfangen werden sollen oder nicht. Im Folgenden wird gezeigt, wie Sie dies für den Status „Nicht stören“ tun. Konfigurieren Sie einfach einen, um PUSH für jeden Status zu erhalten.



Der PBX-Administrator kann auch den Empfang von PUSH für den Benutzer in der 3CX-Verwaltungsoberfläche konfigurieren, und Gruppenbearbeitungsvorgänge sind verfügbar.

Denken Sie daran, dass es besser ist, die automatische Statusumschaltung einzurichten, wenn der Benutzer einen festen Arbeitsplan hat. Der Zeitplan (Arbeitszeit) wird vom Administrator der TK-Anlage festgelegt. Sie können die allgemeinen Geschäftszeiten der Organisation oder die einzelnen Geschäftszeiten dieses Benutzers verwenden. Weitere Informationen hierzu finden Sie im 3CX-Tutorial .



Stummschaltung


Der stille Anwendungsmodus kann unabhängig vom Status aktiviert werden, wenn Sie Benachrichtigungen über Anrufe und Nachrichten erhalten möchten, ohne zu viel Lärm zu verursachen. Der Modus wird durch langes Drücken des 3CX-Symbols auf dem Android-Desktop aktiviert.



Benachrichtigungen in Android 10


In Android 10 wird ein eingehender Anruf auf einem entsperrten Bildschirm als Benachrichtigung angezeigt. Dies ist im Stil anderer Android 10-Benachrichtigungen implementiert. Vergleichen Sie Benachrichtigungen auf Android 9 und Android 10.



Einige Android 10-Benutzer berichten, dass ein Anruf gehört wird und keine Anrufbenachrichtigung angezeigt wird. In diesem Fall wird empfohlen, die Anwendung zu deinstallieren und neu zu installieren. In der nächsten Version werden wir eine Überarbeitung vornehmen, um Benachrichtigungen zuverlässig anzuzeigen.

Start beim Start


Auf verschiedenen Geräten verhält sich die 3CX-Anwendung leider unterschiedlich, je nachdem, wie Android neu gestartet wurde - manuell oder abstürzt (z. B. wenn es hängt). SIP-Konten wechseln oder deaktivieren


In der neuen Anwendung hat sich die Schnittstelle zum Verwalten (Umschalten, Trennen) von SIP-Konten geändert. Im Menü oben links:

  • Klicke auf dein Profilsymbol (1)
  • Halten Sie das Girokonto gedrückt, um die Aktion "Deaktivieren", "Ändern" oder "Löschen" auszuwählen.
  • Klicken Sie auf ein anderes Konto, um zu diesem zu wechseln (2).
  • Klicken Sie auf "Konto hinzufügen" und scannen Sie den QR-Code (per E-Mail oder 3CX-Webclient, um der Anwendung ein neues SIP-Konto hinzuzufügen.



PUSH-Benachrichtigungen sind in 3CX für Android nicht verfügbar


Nach dem Update von 3CX auf Version v16 Update 3 und der Aktualisierung der Anwendung für Android erhalten einige Benutzer keine PUSH-Anrufbenachrichtigungen mehr auf ihren Telefonen. Wir haben dieses Problem bei 3CX-Installationen festgestellt, die ein eigenes Konto für das PUSH-Konto verwenden.



In diesem Fall wird empfohlen, zum integrierten 3CX-Konto zu wechseln. Klicken Sie dazu einfach auf die Zeile "Benutzerkonto", löschen Sie Ihre PUSH-Einstellungen von der 3CX-Oberfläche, klicken Sie auf OK und starten Sie den Server neu.



Überprüfen Sie anschließend die Einstellungen für PUSH-Benachrichtigungen in der Benutzeroberfläche.



Jetzt sollten Sie die 3CX-Anwendungen für Benutzer neu konfigurieren, die Schwierigkeiten haben, PUSH zu erhalten.

Wir hoffen, dass diese Erklärungen und Empfehlungen für Sie und Ihre Benutzer nützlich sind!
