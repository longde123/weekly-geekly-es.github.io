Cientistas lançam projeto para prever expectativa de vida



Programadores, médicos e estatísticos da East Anglia University estão lançando um projeto em larga escala que usará o banco de dados de saúde de cidadãos britânicos para prever a expectativa de vida. As previsões levarão em conta doenças crônicas e seus métodos de tratamento.

Na Internet, você pode encontrar sites de entretenimento que “prevêem” a data da morte, mediante solicitação. Os cientistas se esforçam para construir suas previsões com base em dados estritamente científicos. E, embora longe de todas as pessoas possam ousar perguntar quanto elas ainda restam para viver (mesmo que seja uma estimativa aproximada), os cientistas têm certeza de que o projeto trará benefícios práticos, financeiros e médicos ao estado.

Após analisar uma grande quantidade de dados, será possível tentar avaliar o efeito de certos medicamentos utilizados no tratamento de certas doenças na expectativa de vida. O projeto também ajudará a planejar a aposentadoria com mais eficiência.

A tarefa é muito difícil - uma enorme quantidade de dados foi coletada, eles são bastante complexos e difíceis de analisar. Mas os cientistas não perdem o otimismo e esperam destacar os principais fatores que afetam a mortalidade e a expectativa de vida, incluindo fatores médicos e de estilo de vida.

A pesquisadora principal, professora Elena Kulinskaya, diz: “No mundo todo, a vida das pessoas está aumentando. "Queremos desenvolver ferramentas de software adequadas para prever a expectativa de vida, usando big data que é coletado rotineiramente pelas instituições médicas".

Na Grã-Bretanha, as recentes reformas previdenciárias permitiram aos cidadãos administrar suas aposentadorias mais livremente, mas para aumentar a eficácia desse processo, seria muito útil estimar a expectativa de vida projetada.

O projeto de cinco anos foi possível graças ao financiamento de £ 800.000 do Instituto e da Faculdade de Atuários. Este é um dos três programas de pesquisa que receberam financiamento do instituto.
