PHP internal


  • RFC: Sebarkan Operator di Array - Konsep membahas kemungkinan penambahan operator ... untuk array.
    Teks tersembunyi 
     $parts = ['apple', 'pear']; $fruits = ['banana', 'orange', ...$parts, 'watermelon']; // ['banana', 'orange', 'apple', 'pear', 'watermelon'];
