Adicionar automaticamente fontes JSON ao REFL.ME para gerar mensagens e alertas

Depois de publicar o artigo “ Mensagens e alertas no Android via JSON ” aqui e entrar nas 5 principais notícias do dia no Hacker News, muitas cartas de feedback chegaram. Graças a isso, novos recursos úteis foram adicionados.

Um dos novos recursos adicionados a pedido dos usuários foi a capacidade de adicionar e remover centralmente fontes de dados para gerar mensagens e notificações no dispositivo do usuário.

Dou um exemplo simples no formato JSON que permitirá que você use esse recurso: 

{ "refl.me":true, "hub":true, "sources":[{"url":"https:\/\/refl.me\/\/test"},{"url":"https:\/\/refl.me\/\/test_secret"}] }

"Refl.me" - indica necessariamente verdade.
"Hub" - indica necessariamente verdade.
"Fontes" - uma lista de fontes a serem processadas. A criação, atualização e exclusão serão realizadas automaticamente de acordo com esta lista.

Tudo funciona de maneira semelhante a uma fonte regular, mas é processado não por mensagem, mas por fontes. De acordo com os endereços recebidos de "fontes", o aplicativo forma uma lista real de fontes de dados para gerar mensagens e notificações.
