Ode to one programmer's day

Halo, Habr!

Sekali lagi, menggelar proyek lain pada hari Jumat pagi yang meriah, saya tiba-tiba mengunjungi muse, yang saya bagikan dengan Anda.

Ketahuilah, ini bukan omong kosong komedian!
Ode to one programmer's day.
(catatan penulis)

Pertama, dan ini adalah hukum!
Hafalkan sebuah tim 

git clone

Lalu, agar tidak pindah ke parit
Kombo dari 

 composer install

dan 

 composer update

Jika Anda bukan "scorcher terampil"
Lihat pertunjukan apa 

 git branch

Mengikuti, dan ini bukan hanya kata-kata:
Lihat apa yang akan ditampilkan 

 git branch

dengan bendera 

 -a

Jika Anda membuat tempat baru 

 git checkout

lakukan tes sesuai tradisi
Jangan lupa melakukan pengintaian,
Untuk subjek penamaan cabang
Maka cobalah keberuntungan Anda
Dengan memberi makan nama cabang git
Tidak terjadi Cuming Out :
Jangan lupa untuk segera checkout
Dan kemudian menghentikan kegagalan
Lihat lagi apa yang akan diberikan cabang git
Setelah duduk di atas tugas malam
Tambahkan semua: 

 git add . #

Klarifikasi poin utama: 

 git commit -m "[ ]"

Sehingga komit Anda tidak membungkuk
Uji tim 

 git pull

Jika kode Anda tidak lengkap omong kosong
Kami sedang menunggu perintah server 

 git push
...
Pencarian yang ganas akan segera berakhir.
Repositori. Kemudian gabungkan permintaan.
Dengan hati nurani yang bersih, tuangkan kopi
Hanya ketika ulasan kode selesai.
Dan jika tes lulus,
Selamat datang di Prod.
Ini hari yang menyenangkan,
Jika setiap langkah ok .

Saya mengundang semua orang untuk bergabung dengan pekerjaan.
