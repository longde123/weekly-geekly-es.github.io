Editor.js adalah editor yang sangat baik yang menyimpan kode sumber dalam format JSON

CodeX adalah orang-orang keren yang berbasis di ITMO University. Atas dasar universitas ini, siswa dan lulusan membuat proyek yang menarik dan bermanfaat: Hawk.so , Capella , AR-tester dan lainnya.

Satu proyek banyak membantu bisnis kami. Editor.js editor menyimpan kode sumber di JSON.



Keuntungan utama editor adalah penyimpanan dan penyajian data dalam format JSON. Mereka yang memiliki versi web dari produk dan aplikasi seluler akan mengerti secara khusus. Untuk menulis artikel, manajer konten suka menggunakan editor WYSIWYG, tetapi setelah itu kita mendapatkan "spageti":



Spaghetti hanya membunuh seluruh tata letak dan mengintegrasikannya ke dalam aplikasi seluler hanya menyusahkan . Saya harus menulis artikel secara terpisah untuk versi web dan aplikasi seluler. Itu panjang dan sulit untuk dirawat.

Trade-off adalah format JSON. Sederhana, memiliki struktur yang ketat dan fleksibel. Tambahkan editor Editor.js ke panel admin, tulis kontennya dan simpan sebagai JSON. Dengan API, kami mengirim JSON ke aplikasi seluler dan mengaturnya lebih lanjut sesuai kebutuhan, dan mengonversinya ke versi web JSON dalam HTML melalui Editor.js. Dan semua kebahagiaan.



Para lelaki menyediakan API untuk membuat plugin mereka. Situs ini memiliki dokumentasi yang baik dengan contoh-contoh. Gunakan!
