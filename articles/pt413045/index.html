macOS + Windows 10 em uma unidade externa

Condição . Há um MacBook Pro no início de 2015 sem uma unidade interna. É necessário organizar nele o macOS e o Windows 10 na versão de inicialização dupla a partir de uma unidade externa.


Relevância . A unidade interna dos macbooks modernos é apresentada pelo PCI-E SSD no formato de propriedade da Apple. 128–256 GB não originais podem ser comprados a um preço na faixa de 5 a 10 mil rublos. [ 1,2 ] Por outro lado, se você tiver uma unidade externa razoavelmente rápida (ou com baixa demanda de velocidade), poderá inicializá-la.


Obstáculos . Você não poderá instalar Macos sem (a) uma unidade adicional para o instalador e (b) uma segunda papoila para gravar este instalador nesta unidade adicional (além do arquivo com o próprio instalador) - uma instrução da Apple .


Na ausência de uma segunda papoila, pode ser escrita no Windows.

Softina TransMac (existe um teste) de acordo com as instruções dos entusiastas que usam imagens prontas ( 10.11 El Capitan , 10.12 Sierra ).


Mac OS 10.13 High Sierra Recusei-me a instalar em uma unidade externa. Por outro lado, 10.10 Yosemite e mais velhos não se sustentam nas papoilas modernas.


Quando o MacOS é instalado, a maneira padrão de instalar o Windows a partir dele é o Boot Camp. No entanto, ele não permite a instalação em uma unidade externa. Seria possível gravar o instalador do Windows em uma unidade adicional (pelas ferramentas padrão da Makosi, usando UNetbootin, Rufus ou outros meios), mas também não permitirá a instalação em uma unidade externa.


Solução . (a) Em vez de um bootkamp com as mãos, preparamos uma seção adequada na unidade [ 3 , 4 ], após a qual (b) em vez do instalador do Windows, use o WAIK para preencher o gerenciador de inicialização e os arquivos de instalação na unidade [ 5 ]; (c) instale o Windows carregando através da opção fixada no item de inicialização EFI; (d) instale o pacote de driver a partir do diretório WindowsSupport (que é obtido ao reiniciar o macOS, abrir o bootkamp e encontrar no menu “Baixar o software de suporte mais recente da Apple Windows”). Eu testei - funciona.
