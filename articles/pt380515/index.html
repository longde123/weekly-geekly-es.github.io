Que impressões você pode obter do Google

Muitos provavelmente conhecem o projeto Google Cardboard. Talvez essa seja a solução mais barata para a realidade virtual que existe atualmente. Como todos os engenhosos - este é o dispositivo mais simples. É uma caixa de papelão para um smartphone com duas lentes, que permite olhar para a tela do seu telefone como duas telas enormes mostrando uma imagem que pode ser reunida pelo cérebro. O smartphone lida com a tarefa de processar rotações da cabeça e, portanto, o efeito da realidade virtual é criado.

Este ano, o Google em sua conferência anual de desenvolvedores prestou muita atenção à imersão no mundo virtual em 360 graus:
- O Jump foi introduzido - uma série de 16 GoPro para gravação de vídeo estéreo esférico
- foi anunciado

indicava ambiguamente a importância da direção VR para o Google - e, é claro, o CardBoard apresentado naquele ano foi atualizado na conferência .



Na semana passada, exatamente esse papelão caiu em minhas mãos direto de São Francisco.

Das principais inovações da nova versão do Cardboard, se você pode chamá-las assim, foi feito:

- suporte para smartphones com pá de até 6 polegadas inclusive
- um botão universal que não requer interação com um ímã e, portanto, suporte para qualquer smartphone
- agora o sdk está disponível para os proprietários de dispositivos com maçã na parte de trás
