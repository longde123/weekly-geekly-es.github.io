Windows 10 persis berumur satu tahun

Hari ini kedaluwarsa kemungkinan upgrade gratis ke OS ini untuk pengguna Windows 7 dan Windows 8




Setahun yang lalu, Microsoft merilis sistem operasi baru, Windows 10 . Platform ini dirancang sebagai sistem tunggal untuk berbagai perangkat, termasuk smartphone, tablet, komputer desktop, konsol game, dan lainnya. Perusahaan berhasil mengimplementasikan rencananya, dan pengguna menerima platform terpadu dengan katalog tunggal aplikasi universal.

Segera setelah rilis sistem operasi, diumumkan bahwa pembaruan dari Windows 7, Windows Phone 8 dan Windows 8.1 c Windows 9hingga "puluhan" - gratis jika versi OS sebelumnya adalah salinan legal. Cara yang menarik untuk mendistribusikan sistem operasi ini dan pembaruannya. Microsoft Corporation mengatakan bahwa distribusi dan pembaruan didistribusikan tidak hanya dari server perusahaan, tetapi juga dari PC pengguna - yaitu, teknologi peer-to-peer digunakan.

Sistem ini tidak terlalu menuntut. Banyak pengguna mengklaim bahwa ia berfungsi tanpa masalah pada laptop dan komputer yang lemah, dan lebih cepat daripada versi OS sebelumnya bekerja dengan perangkat keras yang sama.

Sebagian besar pengguna sistem operasi baru puas dengannya. Benar, ada beberapa masalah. Secara khusus, OS ini sering dikritik karena banyaknya informasi pengguna yang dikumpulkan. Korporasi, katanya, sedang mengumpulkan data untuk meningkatkan dan meningkatkan produk dan layanannya. Tetapi pengumpulan informasi sangat luas. E-mail ini, dan log browser, riwayat pencarian, log panggilan dan pesan SMS, informasi tentang pengaturan perangkat dan aplikasi. Itu juga mengumpulkan (tidak selalu) data yang dimasukkan melalui keyboard. Kunci enkripsi hard disk juga ditransfer ke perusahaan, dan Anda dapat menonaktifkan opsi ini hanya dalam versi Pro.



Ketidakpuasan pengguna disebabkan oleh teknologi Wi-Fi Sense, yang secara default menganggap kontak dari jaringan sosial, Skype dan Outlook aman. Sistem "berbagi" akses Wi-Fi dengan orang-orang ini, sehingga semua orang ini, secara teoritis, dapat mengakses jaringan nirkabel Anda. Plus, jika tamu Anda dengan perangkat di Windows 10 atau Windows Phone terhubung ke jaringan nirkabel Anda, kontaknya akan dapat terhubung ke jaringan yang sama tanpa kata sandi. Untuk menghindari hal ini, Anda perlu menambahkan poin _optout ke nama Wi-Fi Anda.

Anda juga dapat menonaktifkan pengintai. Aplikasi DisableWinTracking open source akan membantu . Ada aplikasi dalam bahasa Rusia.

Sering kali mereka mengkritik Microsoft karena memberi terlalu banyak tekanan pada pengguna OS sebelumnya untuk meningkatkan ke Windows 10. Pengguna hampir dipaksa untuk memperbarui sistem, menonaktifkan kemampuan untuk menghapus pengingat pembaruan, menginstal paket layanan yang mengubah pengaturan pengguna , dll. Perusahaan bahkan harus membayar $ 10.000 untuk masalah yang terjadi ketika memperbarui PC dari salah satu pengguna Windows 7 ke Windows 10.

Hari ini diketahui bahwabahwa perusahaan menerima dua tuntutan hukum lagi. Masalahnya sama - taktik agresif mempromosikan peningkatan OS sebelumnya pada Windows 10. Gugatan pertama diajukan di Florida. Penggugat mengklaim bahwa Microsoft melanggar hukum negara untuk membatasi iklan elektronik. Selain itu, gugatan tersebut menuduh pelanggaran aturan Komisi Perdagangan Federal AS.

Gugatan kedua telah diajukan di Haifa, Israel. Penggugat mengklaim bahwa perusahaan memperbarui OS di komputernya tanpa sepengetahuannya. Dalam gugatan serupa, seperti disebutkan di atas, perusahaan membayar $ 10.000, mengaku bersalah.

Pada saat yang sama, cerita tentang jendela pembaruan yang tidak menutup belum berakhir. Sejumlah pengguna mengklaim bahwa mengklik salib tidak menutup jendela saat membatalkan pembaruan, tetapi dianggap oleh sistem sebagai konfirmasi oleh pengguna tentang niat untuk menginstal Windows 10.

Semua ini harus dihentikan besok, 30 Juli. Setelah 29 Juli, aplikasi Get Windows 10 akan dihapus dari PC pengguna di seluruh dunia. Dengan demikian, notifikasi akan hilang. Benar, mungkin perlu waktu hingga aplikasi dihapus dari komputer semua pengguna OS Windows: mulai 30 Juli, Microsoft akan mulai menghapus aplikasi secara bertahap dari PC di Windows 7 dan Windows 8.1.

Kemampuan untuk meningkatkan ke Windows 10 untuk pengguna OS Microsoft dari versi sebelumnya tidak terbatas. Korporasi segera mengumumkan bahwa peningkatan gratis ke Windows 10 tersedia tepat satu tahun - dari 29 Juli 2015 hingga 29 Juli 2016. Hari ini adalah hari terakhir Windows 10 tersedia gratis. Besok, pengguna yang ingin mendapatkan Windows 10 harus membayar $ 119. Pembaruan akan dikenakan biaya pengguna Rusia 7900 rubel.

Apa selanjutnya




Pada tanggal 2 Agustus, pembaruan paling penting untuk Windows 10 akan datang, yang disebut Pembaruan Ulang Tahun Windows 10. Ini akan membawa banyak ke sistem operasi. Ini adalah integrasi dengan Linux, dan wallpaper membosankan, dan Windows 10 di Xbox One. Baca lebih lanjut tentang pembaruan ini di sini . Omong-omong, perusahaan berencana untuk mendistribusikan Windows sebagai aplikasi sebagai layanan di masa depan. Artinya, dimungkinkan untuk tidak membayar biaya lisensi satu kali, tetapi membayar setiap bulan. Sekarang, paket kantor perusahaan didistribusikan dengan model yang sama.
