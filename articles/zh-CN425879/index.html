Google在演讲中显示的内容



几个小时前，谷歌在纽约完成了一项大型活动，在那里她宣布了自己的新产品。 这恰逢Android十周年庆典，当然，主要的注意力还是集中在新的Pixel智能手机上（尽管之前有很多泄密事件，甚至正式在香港开始销售）。


演示的持续时间比Apple更长，并且所有内容都以白色背景为框架，与史蒂夫·乔布斯剧院的黑色舞台形成鲜明对比。 老实说，在苹果公司，一切看上去都更加昂贵和专业。 尽管Google推出了很多新设备。 最重要的是，它们来自不同的领域：在这里，您拥有智能手机，平板电脑，家庭集线器，电视媒体播放器，甚至还有使用人工智能的新方法（这是出乎意料的）。


Google Pixel 3和Google Pixel 3 XL


谷歌在2017年售出了约400万部智能手机，比去年同期增长了两倍。 但是要获得大众认可还有很长的路要走。 相比之下，苹果去年售出了约2.17亿部iPhone。 但是，谷歌表示，这里的重点不是利润，而是竞争。 在公司新智能手机发布前后就形成了最大的炒作。 与上次一样，它们有两个：Pixel 3和较大的Pixel 3 XL。 除了软件的所有更改之外，主要升级是无线充电和广角前置摄像头（例如，如果要在照片中放置很多人，则可以拍摄更酷的自拍照）。




两款智能手机的规格基本相同。 后置摄像头是12兆像素，前摄像头是8兆像素。 防护等级IP68，防尘，防潮。 Snapdragon处理器为845，内存为6 GB，驱动器为64或128 GB（在此设备中，丢失的是最后一部iPhone ，它同时具有256 GB和512 GB）。 没有提供microSD卡插槽。 模型将预装Android 9.0 Pie。


不同版本的大小，显示和电池有所不同。 Pixel 3配备5.5英寸OLED屏幕，分辨率为2160x1080（443 ppi）。 屏幕像素3 XL-6.3英寸（2880x1440像素，523 ppi）。 较小的版本看起来是标准的，而较大的版本在屏幕上方具有特征性的“眉毛”，例如最新的iPhone。


Pixel 3的电池容量为2915 mA∙h，而XL版本的电池容量为3430 mA∙h。 为了开辟无线充电的可能性，两款智能手机的背面都是玻璃，表面经过磨砂处理，受到好评（令人愉悦，无指纹，比市场上所有其他玻璃智能手机弄脏得更慢等）。 那里，在背面-指纹扫描仪。 Google还不相信解锁像苹果一样的智能手机。 设备的边缘容易受到挤压：将智能手机挤压在手中，即可致电语音助手Google助手。




小工具收到了许多软件更新，尤其是相机软件。 根据许多评论家的说法，Pixel 2在iPhone知道的操作质量上处于领先地位，而新的Google模式全能。 他们甚至获得了传奇摄影师安妮·莱博维茨（Annie Leibovitz）的支持，这是他们之间的首次合作。 莱博维茨将带着Pixel 3前往美国进行一个尚未命名的项目，并拍摄“启发人和地方”的肖像。




相机的功能包括“最高拍摄”模式，该模式使用AI来解决常见的照片问题，例如照片模糊或某一观众的眼睛意外眨眼（通过在拍照前后固定更多的帧来解决问题，然后再解决这些问题）。布局）。 在同一个储钱罐中-超级分辨率变焦，当相机处于变焦模式时，有助于清晰显示图像。 谷歌表示，借助Night Sight技术，即使在弱光条件下也无需使用闪光灯。


当然，并非没有AR创新。 他们或多或少地与Marvel合作，并在Pixel 3相机应用程序中添加了不同超级英雄的贴纸（然后他们承诺将其转移到Pixel 2 / Pixel 1）。 如果有的话，这些Pixel中的贴纸可让您在屏幕上放置3D角色，然后在背景中拍照或拍摄视频。 有些人，例如钢铁侠或帝国星球大战士兵，看起来很现实。 如果您在房间的角落与他合影，那么很多人都会相信您确实有这样的身材。 Google在演示文稿上显示它的方式-好像您是他们的朋友，并且实际上是在您的怀里拍照-看起来仍然不太自然。




新的智能手机有三种颜色-白色，黑色和“非粉红色”。 Pixel 3的价格为799美元，Pixel 3 XL的价格为899美元。 这比新iPhone便宜得多（分别为999美元和1099美元）。 设备将在12个国家（美国，日本，爱尔兰，印度，法国...）销售。 俄罗斯不在名单上，但您可以像往常一样在美国地址购买它，没有人会问您。


两款智能手机均可直接从Google预订，10月18日开始发货。 免费提供6个月的YouTube音乐，而且所有者可以无限期使用Google相册服务，直到2022年1月。


Google家庭中心


带有内置语音助手的七英寸屏幕。 实际上，同一列是Google Home，仅具有连接的触摸屏。 它可以播放视频，输出语音查询结果，控制房屋中的其他智能设备。 Google认为该设备几乎是房子的基本设备，所有其他系统（最好还有Google助手）都将被启动。 为了可以安全地将其放置在卧室中，该公司没有专门为其配备摄像头，并且屏幕会自动调整为环境照明水平。




已针对Google Home Hub重做了Google Photos和YouTube服务界面，以便可以通过语音和触摸屏方便地对其进行控制。 与Google Maps和所有Nest产品（恒温器，外部相机）集成在一起，后者也是Google的一部分。 设备本身的设计非常有趣：它是Google Home的一个软圆柱，似乎已被切开，并且屏幕固定在顶部。 如果未使用该设备，它将自动变成数码相框，您在其中旋转来自Google相册的照片。 在这种情况下，模糊（和ob亵！）的照片会被自动滤除。


主要的乐趣在于，由于两家公司之间的敌对态度 ，Google仍在阻止Amazon扬声器与其YouTube的连接。 对于带有屏幕的列，这实际上是最重要和最受欢迎的服务。 因此，他们可以不用半个小时的演讲，而可以出去玩，说：“我们喜欢Echo Show，只有我们的产品可以在YouTube上使用，打招呼”，放下麦克风离开。 当锁定生效且AmazonTube仍在项目中时，带有屏幕的Google扬声器将永远是一个优势。




预购开始，专栏售价149美元，运输将于10月22日开始。 作为奖励，赠送6个月的YouTube Premium。
