Mozilla sortira Firefox 64 bits pour Windows le 3 novembre

Et d'ici la fin de 2016, le support NPAPI sera supprimé de Firefox


Firefox 42, la prochaine version du navigateur de Mozilla, sera disponible à la fois en application 32 bits et en programme 64 bits. La dernière version, disponible uniquement sur le serveur FTP, bénéficiera de l'architecture x86-64 (AMD64 / Intel 64). Par exemple, il pourra adresser plus de 4 Go de mémoire. Actuellement, sous Windows, il n'y a qu'une version 32 bits de Firefox. Pour les utilisateurs de Linux et Mac OS, Firefox 64 bits est disponible depuis longtemps. Firefox 42 sortira le 3 novembre de cette année.

Au cours des cinq dernières années, Mozilla a commencé le processus de développement de Firefox 64 bits, puis l'a abandonné , puis a recommencé . Depuis 2014, après avoir privilégié le processus de développement de la version 64 bits de Firefox pour Windows, la sortie a été retardée plusieurs fois. Comme d'habitude, il est préférable de tirer les premières nouvelles du traqueur de bogues , car le processus de développement est ouvert.

Le 3 novembre, la prochaine version de Firefox 42 sortira, mais aucune option ne sera proposée pour installer la version 64 bits. De plus, il n'y aura aucun lien même sur la page de téléchargement. L'assemblée seraObtenez exclusivement sur le serveur FTP comme test. Un porte-parole de Mozilla dit qu'une sorte de «changement par rapport aux partenaires» est attendu, donc il n'y aura pas d'annonce à part entière. La version 64 bits ne prendra pas en charge les plugins utilisant NPAPI.

64 bits se réfère ici aux systèmes avec l'architecture du processeur central x86-64sur lequel les systèmes d'exploitation 64 bits sont installés. x86-64 est une extension 64 bits de x86 32 bits avec compatibilité descendante. Les processeurs de presque tous les ordinateurs personnels modernes utilisent l'architecture x86-64. Les systèmes d'exploitation 32 bits ne permettent pas d'adresser plus de 4 Go de RAM. De plus, ce nombre comprend à la fois la RAM elle-même et la mémoire de l'accélérateur vidéo. Les systèmes 64 bits ont des restrictions plus élevées, par exemple, Windows 10 peut utiliser jusqu'à 2 To de RAM. Les systèmes d'exploitation 64 bits vous permettent d'exécuter des applications 32 bits. Les systèmes d'exploitation 32 bits ne peuvent pas exécuter de programmes 64 bits. Les avantages du code 64 bits ne se limitent pas à la quantité de RAM adressable: les programmes peuvent être plus rapides.

Contrairement à Google Chrome, qui utilise une architecture multithread, Firefox utilise un seul processus, bien que des travaux sur l'introduction de l'architecture multithread aient eu lieu . Maintenant, la version finale de Firefox pour Windows est de 32 bits et est incluse dans un processus, qui ne peut pas traiter plus de 3 Gio dans un système 32 bits et 4 Gio en 64 bits. Il existe des versions nocturnes 64 bits pour Windows. Le 3 novembre, pour la première fois, une version finale de Firefox 64 bits sera publiée pour Windows.

Les rivaux de Firefox ont depuis longtemps des versions 64 bits dans leur arsenal. La voracité de Chrome, qui lui permet d'aller au-delà de 32 bits de l'espace d'adressage et d'utiliser facilement toutes les ressources de l'ordinateur, est bien connue des utilisateurs et est devenue l'objet de blagues, bien qu'elle ait ensuite été optimisée par Googletravailler avec la mémoire. Même les projets basés sur le code source gratuit de Firefox dépassent l'ancêtre dans ce domaine: Pale Moon et Waterfox sont depuis longtemps disponibles dans les versions 64 bits pour Windows.

La version Win64 de Firefox 42, qui sortira le 3 novembre, ne prendra pas en charge les plugins NPAPI. En général, Mozilla supprimeraprise en charge de NPAPI par Firefox d'ici la fin de l'année prochaine. Cela signifie que les plugins populaires tels que Java ou le lecteur Web Unity ne fonctionneront pas. La décision de supprimer le support NPAPI est dictée par le fait que les fonctions de plug-in sont déjà prises en charge nativement par le navigateur lui-même: dans Firefox, vous pouvez regarder des vidéos grâce à HTML5 et la présence des codecs nécessaires et jouer à des jeux 3D en utilisant WebGL. Les plugins NPAPI montrent de plus en plus l'âge de la technologie sous la forme de problèmes de performances (qui affectent la durée de vie de la batterie sur les appareils mobiles), de plantages et d'incidents de sécurité de l'information.

La prise en charge des plug-ins NPAPI a déjà été supprimée de Chrome. Pour Flash, les développeurs de Mozilla feront une exception car il est devenu une partie importante de la vie quotidienne des internautes. Probablement, à une certaine période de développement, il sera également retiré. Oracle comprend le rejet de Java et propose ses propres solutions. Unity a abandonné le travail sur Unity Web Player. Au lieu de cela, il est proposé d'utiliser l'exportation vers WebGL. Dans la version d'Unity 5.4 qui sortira en mars de l'année prochaine, la prise en charge des plug-ins de navigateur sera supprimée.

Après la sortie (3 novembre), la version 64 bits de Firefox 42 pour Windows sera disponible sur https://ftp.mozilla.org/pub/mozilla.org/firefox/releases/latest/. Le cycle de développement de Firefox consiste à changer séquentiellement le statut des versions nocturnes en une version pour les développeurs, une version bêta, puis une version. Une version bêta du navigateur 64 bits est déjà disponible .
