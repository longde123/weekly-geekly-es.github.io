Kalkulator Windows Sekarang Open-Source

Kami senang mengatakan bahwa minggu lalu Kalkulator Windows kami menjadi OpenSource. Sekarang tersedia di GitHub di bawah lisensi MIT. Ini termasuk kode sumber, struktur rakitan, pengujian unit, dan struktur produk. Tujuan kami adalah menciptakan produk yang lebih baik dalam kemitraan dengan komunitas. Kami mendorong ide-ide segar Anda dan partisipasi aktif dalam menentukan masa depan Kalkulator Windows.



Sebagai pengembang, jika Anda ingin tahu cara kerja berbagai bagian aplikasi Kalkulator, mudah untuk mengintegrasikan logika atau antarmuka pengguna Kalkulator ke dalam aplikasi Anda sendiri atau berkontribusi langsung ke sesuatu yang tersedia di Windows, sekarang Anda dapat melakukannya. Kalkulator akan terus melalui semua proses pengujian yang biasa, memeriksa kepatuhan, keamanan, kualitas dan memeriksa data internal, seperti halnya semua aplikasi lainnya. Anda dapat mempelajari lebih lanjut tentang ini di dokumentasi di GitHub.



Pengembangan bersama


Selain menggunakan kembali dan mengadaptasi kode dalam aplikasi Anda sendiri, semua orang dapat mengambil bagian dalam pengembangan kalkulator Windows . Sangat mudah untuk terlibat. Proyek clone-and-go, dan pengembangannya akan dilakukan sesuai dengan proses GitHub standar. Di semua tahap, pengembang dapat berkontribusi:


  • Partisipasi dalam diskusi
  • Laporkan atau perbaiki bug
  • Tawarkan Fitur Baru
  • Prototip fitur baru
  • Mendesain dan menciptakan produk dengan teknisi kami

Pelatihan


Mempelajari kode kalkulator adalah cara terbaik untuk belajar tentang teknologi Microsoft terbaru, seperti platform Windows universal, XAML, dan Pipa Azure . Berkat proyek ini, pengembang dapat belajar dari siklus pengembangan penuh Microsoft, serta menggunakan kembali kode untuk membuat produk mereka sendiri. Ini juga merupakan contoh yang bagus untuk membuat desain aplikasi Fluent . Untuk menjadikan ini lebih mudah, kami akan memberikan kontrol khusus dan ekstensi API yang kami gunakan dalam kalkulator dan aplikasi lain untuk proyek-proyek seperti Windows Community Toolkit dan Windows UI Library .


Kami senang mengundang Anda semua ke tim Kalkulator Windows! Untuk memulai, buka proyek Kalkulator Windows di GitHub.
