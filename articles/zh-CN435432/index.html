新年，新的GitHub：无限的免费私人存储库

今天，我们宣布了GitHub上的两项重要创新，这些创新将使开发人员可以更轻松地访问它：无限的免费私人存储库和对公司更方便的产品。 细节剪下！



  • GitHub Free现在包括无限的私有存储库。 开发人员首次可以将GitHub用于其项目，并免费向存储库中添加多达三个协作者。 许多开发人员希望使用私有存储库来处理第三方项目，或者在公开发布之前私下尝试某些操作。 从今天开始，这些脚本和许多其他脚本可在GitHub上免费获得。 公共存储库仍然是免费的，并且包含无限数量的合作者。
  • GitHub Enterprise是针对Enterprise Cloud（以前称为GitHub Business Cloud）和Enterprise Server（以前称为GitHub Enterprise）的新的统一产品。 需要灵活地在云或独立配置中使用GitHub的组织现在可以以相同的价格访问这两者。 借助GitHub Connect，这些产品可以可靠地互连，从而提供了一个混合选项，使开发人员可以在任何环境中无缝工作。

了解更多

GitHub Pro（以前为GitHub Developer）和GitHub Team也可用于需要专业编码和协作功能的开发人员和团队。 而且，当然，开源项目将具有在公共存储库上一起工作所需的一切，包括我们的GitHub Team的免费版本。

无论您是要编写第一行代码的学生，还是拥有遍布全球的团队的企业负责人，我们都希望GitHub成为您编写代码，与全球开发人员社区进行协作和交流的最佳场所。 今天的变化是GitHub未来的重大投资，对于您在2019年构思和创建的所有项目，我们将感到非常高兴。
