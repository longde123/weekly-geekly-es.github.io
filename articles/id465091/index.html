Kebocoran data pribadi dugaan karyawan Kereta Api Rusia

Tidak diketahui, dipublikasikan dalam data pribadi domain publik 703.000 orang, mungkin karyawan Russian Railways.



Di situs (saya tidak memberikan tautan ke situs secara khusus) ada tulisan: " Terima kasih kepada Russian Railways untuk informasi yang diberikan, dengan menangani data pribadi karyawannya dengan cermat. "


Di antara data yang tersedia secara bebas:


  • Nama lengkap
  • tanggal lahir
  • alamatnya
  • Nomor GILA
  • posisi
  • fotografi
  • telepon


Tidak diketahui di mana kebocoran itu terjadi, tetapi ada spekulasi bahwa ini adalah basis data layanan keamanan. Dilihat dari format foto, ini adalah gambar yang pas.


Menurut perkiraan saya, "database Kereta Api Rusia" ini akan muncul di domain publik dalam bentuk file Excel dan / atau dalam format Kronos dalam satu hingga dua hari.


Yaitu bahkan jika Roskomnadzor menutup akses ke situs yang berisi halaman dengan data pribadi, ini tidak akan mempengaruhi ketersediaan data yang bocor.


Berita tentang kebocoran informasi dan orang dalam selalu dapat ditemukan di saluran Telegram saya " Kebocoran informasi ".
