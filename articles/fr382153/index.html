Plasma Mobile: un autre OS mobile des développeurs de KDE



Un autre acteur est apparu sur le marché des systèmes d'exploitation pour appareils mobiles - le système d'exploitation mobile Plasma Mobile . Il est basé sur Linux, et les créateurs font partie de l'équipe qui travaille sur le shell graphique KDE.

Plasma Mobile est la version mobile de Plasma pour ordinateurs de bureau. C'est un système d'exploitation gratuit et ouvert. Jusqu'à présent, aucun téléphone n'est fourni avec ce système d'exploitation, mais vous pouvez le télécharger et l'installer sur Google Nexus 5. Si vous le souhaitez, les mêmes astuces sont utilisées que pour l'installation d'Ubuntu Touch.



Le logiciel est basé sur le système d'exploitation Kubuntu avec un pilote graphique d'Android, kwin / Wayland. Plasma Mobile n'est pas lié au système d'exploitation Kubuntu et peut fonctionner avec d'autres systèmes d'exploitation. Nexus 5 / Kubuntu est la version qui le rend assez facile à utiliser.



Le logiciel prend en charge la mise en œuvre des fonctions de base d'un smartphone, y compris les appels téléphoniques, les notifications, le lancement personnalisé d'applications.

Il existe un support pour les widgets KDE Plasma. Cela vous permet d'exécuter des applications de bureau sur un téléphone mobile. En principe, cela n'a pas beaucoup de sens lorsque vous travaillez avec un smartphone de 5 pouces. Mais tout cela peut être très utile lors de la connexion d'un écran externe à un smartphone.



En plus des applications pour Plasma, cette plate-forme logicielle prend en charge les applications pour smartphones Ubuntu, pour les appareils sous Sailfish OS, ainsi que la prise en charge des applications Android.
