Memperluas kemampuan sistem audio mobil pada contoh Mercedes W203

Aplikasi lain dari modul MP3862BT di mobil:

Saya adalah pemilik Mercedes W203, 2000 rilis. Saya mendapatkannya dengan pemutar kaset. Mengapa pemilik pertama, orang Italia, memesan sistem audio seperti itu di mobil yang cukup modern tidak sepenuhnya jelas. Karena itu, saya sudah lama merasa perlu mendengarkan musik dari smartphone dan tablet.

Tidak ada keinginan atau peluang untuk mengubah rekaman radio menjadi kontingensi "pintar". Selain itu, 15 ribu rubel tidak ada dalam anggaran saya. Dan yang paling penting, ketika memasang radio freelance, saya akan kehilangan kendali sistem audio di setir. Oleh karena itu, saya telah lama mencari opsi untuk perbaikan murah dari sistem audio standar ke tingkat modern. Untuk memperbaiki radio, diputuskan untuk menggunakan modul MP3862BT, dari Master KIT perusahaan.



MP8036BT adalah penerima Bluetooth dengan berbagai daya dan output linier disesuaikan untuk koneksi ke input AUX. Ada tombol kontrol pemutaran di papan, yang, jika perlu, dapat ditampilkan untuk kontrol eksternal. Dari penjelasan di atas dapat dilihat bahwa papan ini dirancang khusus untuk pemasangan cepat di sistem suara apa pun.



Instalasi tidak lama datang.
Di Internet, saya menemukan deskripsi radio dan tempat untuk menghubungkan input AUX, di radio standar.



Karena ketika menghubungkan modul ke telepon pintar, melodi yang menyenangkan dihasilkan di speaker, diputuskan untuk tidak menampilkan indikasi pada panel radio, yang juga menyederhanakan koneksi.



Jika perlu, dari terminal board MP3862BT, Anda dapat mengeluarkan input AUX untuk menggunakan perangkat (pemain) ortodoks pada jack mini. Tetapi ada sesuatu yang mengatakan kepada saya bahwa ini tidak diperlukan.

Tidak ada tempat untuk memasang modul <a href=" MP3862BT "> MP3862BT tanpa banyak kesulitan.



Sebelum memasang radio, kami memeriksa semuanya untuk operabilitas.



Sayangnya, pada tahap ini, pengoperasian dek kaset tidak dapat dipulihkan. Ketika Anda mematikan mekanisme tape drive cartridge, pengontrol merasakan ada sesuatu yang salah dan memotong elektronik radio. Untuk beralih ke pemutaran dari modul bluetooth, Anda harus memasang kaset di penerima kaset. Oleh karena itu, agar semuanya baik-baik saja, Anda perlu memahat sobekan yang akan mengirim sinyal ke unit perlindungan, mengaktifkan input AUX output. Tapi, pada tahap ini, selama dia puas, dia bahagia seperti gajah. Idealnya, Anda perlu menghubungkan tombol kontrol dari tape recorder ke tombol kontrol pada modul. Setelah penyempurnaan seperti itu, akan dimungkinkan untuk mengontrol aliran dari panel radio.

Proses menghapus radio dapat ditemukan di forum, membutuhkan waktu 5-10 menit.

Manual untuk radio saya dapat diunduh di sini .

Pertanyaannya adalah, mengapa pabrikan radio tidak membuat AUX, meskipun faktanya ada semua kemungkinan? Ini mungkin akan tetap menjadi misteri selamanya.

Ini dia substitusi impor. Terima kasih Guru Kit untuk kehidupan kedua radio saya!
