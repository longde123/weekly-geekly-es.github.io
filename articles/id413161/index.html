Git puisi

Pada hari ulang tahun "matahari puisi Rusia", mari kita bayangkan seperti apa kehidupannya sekarang: sistem kontrol versi untuk ayat, sensor Benckendorf sebagai manajer proyek dan membakar tenggat waktu.



Pushkin duduk, menulis puisi. Menjalankan Benckendorf.
- Dengar, Pushkin, kita punya tenggat waktu! Harus dibebaskan. Investor bertanya: di mana isinya?
"Tidak, well, aku tidak bisa melakukan itu!" Benckendorf, saya katakan berkali-kali sehingga saya tidak bisa melakukan ini.
- Alexander! Mereka menuntut dari saya, saya tidak bisa berbuat apa-apa. Ayo keluar. Apa yang ada di sana
- Di sini ...

Pushkin menunjukkan dua bait dan awal yang ketiga.

Saya ingat visi yang indah:
Anda muncul di hadapanku
Seperti momen singkat
Seperti seorang jenius kecantikan putih.

Dalam kesedihan putus asa putus asa
Di alarm dari keributan yang bising
Suara bersalju panjang bagiku
Dan saya memimpikan fitur lucu.

Ta-ta-ta. Badai
Mimpi-mimpi lama yang tersebar
Tra-ta-ta-ta lembut
Fitur purum pum pum.

- Hebat! Sebarkan dua bait pertama dalam makanan.
- Tapi ...
- Lalu semuanya "tetapi." Kami perlu menunjukkan kontennya!
Lari.
Pushkin menghela nafas, memotong bait yang belum selesai, membuat git init, git menambahkan poem.txt, git melakukan dan git mendorong ke server. Dia mengembalikan bait terakhir dan melihat ke kejauhan, memutar pena bulu di tangannya ... Setelah sekitar 10 menit, sebuah ide datang kepadanya, dia meletakkan pena dan mulai mengetuk kunci.

Tara-ta. Badai
Mimpi-mimpi lama yang tersebar
Tra-ta-ta-ta-ta lembut

Pushkin bangkit dan menuang kopi untuk dirinya sendiri. Di koridor, ia menemukan Benckendorf yang bergegas.
- Pushkin, apa yang kamu lakukan! Ada bug di produksi!
- Ayo! Bug? Dengan saya? Tidak mungkin.
- Lihatlah dirimu.
Benckendorf membuka laptop, menunjukkan Pushkin sesuatu. Pushkin meletakkan tangannya di rambutnya dan menggaruknya dengan serius.
- Ya, tochnik, lihat Anda ... Tapi saya memiliki semua yang ada dalam kode sekarang ... Saya baru saja mulai memikirkan bait keempat ... ada beberapa pemikiran, tetapi bukan fakta bahwa itu cepat diimplementasikan ...
- Kita harus segera memperbaikinya!
Benckendorf melarikan diri. Pushkin, setelah lupa tentang kopi, pergi ke tempat kerja. Di sana ia melakukan git simpanan, memerintah "bersalju" menjadi "lembut", git melakukan, git mendorong, git simpanan pop. Tampak serius pada cangkir yang kosong. Lalu ke langit-langit. Di matanya adalah upaya untuk berkonsentrasi. Lima menit kemudian dia berhasil, dan dia mulai mengetuk kunci.

Di layar Pushkin, pekerjaan yang belum selesai:

Tahun-tahun berlalu. Badai
Mimpi-mimpi lama yang tersebar
Dan aku lupa suara lembutmu
Dan memimpikan fitur yang baik.

Di padang belantara, dalam kegelapan trem-ta-ta-tam
Hari-hariku tenang
Tanpa tram-ta-tam, tanpa pam-pa-ra-ram,
Tidak ada air mata, tidak ada kehidupan, tidak ada ta-tam.

- Dengar, Pushkin! Kemudian investor melihat kontennya, semuanya baik-baik saja, tetapi kita perlu hak cipta dan nama.
Di samping pushkin:
- Blah ...
Benckendorf, mengunyah bigmac, kacamata.
- Baiklah, Pushkin, untuk terakhir kalinya! Biarkan aku mati!
Pushkin menghela nafas, melakukan git simpanan lagi, menambahkan sederet hak cipta. Setelah berpikir sebentar, dia menyebut puisi itu "K ***".
"Kalau begitu aku akan mencari tahu! ..." bisiknya pada dirinya sendiri.
Git melakukan, git mendorong. Benckendorf telah berdiri di belakangnya selama ini dan mengunyah bigmack dengan saksama.
Pushkin:
- Ya, benar! Sangat bahagia? Apakah tidak ada lagi pengeditan? Biarkan itu bekerja dengan tenang!
"Dia tidak akan membiarkanku gagal di tempat ini!" - meyakinkan Benckendorf. - Terima kasih, saya berhutang budi! - dan lari, menjatuhkan remah-remah.
Pushkin sedang menatap monitor.
- Agar Anda kosong, manajer itu efektif!
Itu mengembangkan cabang git, pop git simpanan, git melakukan. Melirik mug kosong, memakai headphone dan duduk selama sekitar 10 menit, menggelengkan kepalanya mengikuti irama musik. Lalu dia mulai menggedor keyboard dengan keras.

Di padang belantara, dalam kegelapan trem-ta-ta-tam
Hari-hariku tenang
Tanpa trem-ta-tam, tanpa penyesalan

Benckendorf mengambil headphone dari kepala Pushkin.

- Apa ?! - Pushkin merobek headphone dan beralih ke Benckendorff.
- Ya, tidak benar-benar ... Mereka mengatakan itu tidak nyaman untuk membaca, "momen singkat" entah bagaimana memotong mata ... Bisakah Anda mengulanginya? Saya punya gelembung!
"Yah, bagaimana aku akan mengulang ini?" Bagaimana? Saya memiliki semua kode dalam keadaan dibongkar, tes tidak lulus sama sekali!
- Nah, sayang, baiklah, pikirkan sebuah utas!
Pushkin membuat git commit, git checkout master, dengan penuh perhatian mengambil pena di telinganya, lalu menukar "visi" dan "instan".
- Begitukah?
- Masuk! Yang Anda butuhkan, nishtyak, lepaskan! - Benckendorf melarikan diri.
Pushkin memberinya tampilan yang tidak baik, membuat git melakukan, mendorong git, mengembangkan checkout. Memimpin dengan pena karena perubahan yang tidak dikomit, mengetuk cangkir kopi kosong. Sekitar sepuluh menit kemudian dia sadar, dia memakai keyboard.

- Apa lagi? - Pushkin meletakkan pena dan dengan gugup mengetukkan jari-jarinya di atas meja.
- Tidak ada, saya hanya datang untuk melihat: apakah versi baru siap?

Jiwa telah tiba pum-purum-pum:

Seperti visi pum-puram-pam
Seperti tram-taram-sana-sana param.

- Bagaimana mengatakannya ... - Pushkin meraba-raba sebuah pena. - Ada beberapa baris, sepertinya tidak ada ... Yah, bait kelima tampaknya mematuk ... kita perlu memikirkan banyak dengan komponen lain, karena ...

Tidak akan ada uang - mereka akan membubarkan semua orang ke neraka dengan seekor anjing. Jangan menulis puisi Anda kalau begitu. Roll out, Anda perlu menyinari kemajuan. Dan ya, ada bug lagi - tidak ada cukup koma di sini.

Menghela nafas, mengambil cangkir dan pergi untuk minum kopi.
