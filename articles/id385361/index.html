Windows 10 Mobile - build 10549 tersedia untuk orang dalam

Hari-hari yang sibuk untuk orang dalam. Gabe Aul mengumumkan versi baru untuk Windows 10 Mobile di Twitter semalam.



Untuk menginstal build baru, Anda perlu memutar kembali menggunakan Windows Device Recovery Tool ke Windows Phone 8.1, dan kemudian instal ulang aplikasi tersebut. Peserta dalam program evaluasi pendahuluan WIndows -> pilih Fast Ring dan instal perakitan baru.
Faktanya adalah bahwa tepat sebelum distribusi pembaruan, kesalahan ditemukan yang membuatnya tidak mungkin untuk memperbarui dari versi sebelumnya.
