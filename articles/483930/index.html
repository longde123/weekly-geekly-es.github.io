6 proyectos de desarrollo web de GitHub que vale la pena ver

Hola Habr! Te presento una traducción del artículo 6 de Github Repos para desarrolladores web que deberías echar un vistazo por lampewebdev .

Una vez que hojeé un desarrollador para alimentarme y me topé con una publicación en 6 proyectos de GitHub para bombear conocimiento rápidamente .

Estoy de acuerdo con el autor en que la lista es realmente impresionante, pero a veces la lista de enlaces en estos proyectos es muy grande.

Por lo tanto, aquí hay una lista de mis proyectos favoritos de GitHub para bombear conocimiento rápidamente, pero creado específicamente para desarrolladores web.

Lista de preguntas de JavaScript (avanzado)




Después de todo, ¿todos amamos las entrevistas? ¿Resolver problemas que nunca tendremos que resolver en el mundo real? ¿Te gusta escribir algoritmos en la pizarra? O mirando el código, responda la pregunta: "¿Cuál será la conclusión?". A lo que debes responder: "¡Este terrible código necesita ser reescrito con urgencia!"

¡Pero debemos estar preparados para esto! ¡Por lo tanto, en este repositorio encontrará preguntas y soluciones en muchos idiomas!

Pasarlos todos puede llevar días, horas o meses, dependiendo de sus habilidades y experiencia, por lo que no diré exactamente cuánto.

Algoritmos JavaScript y estructuras de datos




Una nueva tendencia en el mundo de TI es hablar mucho sobre algoritmos y conocerlos bien.

¡Este increíble repositorio contiene la mayoría de ellos!

Es realmente genial que cada algoritmo esté escrito en JavaScript y probado. ¡Entonces será aún más fácil para usted entenderlos!

Agregaré de mí mismo: por favor no intentes estudiarlos con el objetivo de que puedas escribirlos directamente desde tu cabeza. En mi experiencia y opinión, el 99.5% de los desarrolladores nunca tendrán que escribir o implementar algoritmos. Solo necesita comprender la complejidad temporal de estos algoritmos para decidir cuándo cargar una biblioteca usando este algoritmo y usarlo para el propósito previsto.

Conceptos básicos de seguridad de red




La seguridad siempre ha sido un punto importante, pero cuanto más se desarrollan las tecnologías en la nube, más prioridad se vuelve para muchas empresas.

También significa que para usted, como desarrollador web, conocer los conceptos básicos de la seguridad de la red lo ayudará a obtener un mayor sueldo, ya que tendrá una ventaja sobre aquellos desarrolladores que piensan que la seguridad no es importante.

Este repositorio contiene los conceptos básicos de seguridad básicos necesarios incluso para los desarrolladores junior de hoy.

JavaScript a la luz de las funciones




La programación funcional se está convirtiendo en una tendencia no solo en el desarrollo de JavaScript, sino también en todo el desarrollo.

Además, debe saber más que solo POO.

No tengo nada más que decir sobre el desarrollo funcional. Creo que al menos debería saberlo un poco, esto lo convertirá en un desarrollador más experimentado.

Mejores prácticas Node.js




¡No bloquees el bucle de eventos! ¡Por favor no bloquee el bucle de eventos! ¡Utilice las mejores prácticas cuando trabaje con nodejs!

Si está bloqueando el bucle de eventos cuando trabaja con nodejs, ¡entonces no debe hacer esto!

Es por eso que necesita leer esta guía de mejores prácticas para nodejs.
Y para los desarrolladores de nodejs, esta guía es una lectura obligada.
¡Así que comienza ahora!

JavaScript: código limpio




Este repositorio está inspirado en un libro del mismo nombre del tío Bob.

Si no te gusta leer libros, entonces este repositorio es para ti.

Sabemos que el código incorrecto puede funcionar porque todos a veces escribimos código incorrecto.

¡Y esto es normal! Al tener un manual que muestre qué es un código incorrecto, será más fácil para nosotros escribir un buen código.

De vez en cuando miro a través de este repositorio para actualizar mis conocimientos.


¿Tienes algún repositorio favorito en GitHub? ¡Compártelos en los comentarios!
