上古产品：ThinkPad 380E，90年代经济舱和Windows 95

收集旧设备可能对您尝试学习的时间有些不诚实。 在销售开始时，大多数老式笔记本要么昂贵（ThinkPad 600 ，ThinkPad T22 ），要么非常昂贵（ThinkPad X301 ）。 如果那时候我不是很粗心，我可能不会买这样的设备。 我们始终试图在价格和质量之间找到折衷方案，并且不花钱去尝试会很有趣的技术，但根本没有必要。


我今天要谈论的笔记本电脑是1997年的经济舱。 功能中等，移动性中等，但明显不及旗舰产品。 去年，我以原始状态购买了带活动硬盘和电池的IBM ThinkPad 380E。 似乎它从未被打开过，并且仍然带有所有实用程序的原始Windows 95。 让我们看一下笔记本电脑和第一个真正的现代Windows。 我们将决定其可靠性是否与所讲的所有笑话相对应。

我用电报实时地记录了一个旧铁器收藏家的日记。 在接下来的几周中，我将分享我对Windows CE上第一批PDA之一的印象-飞利浦Velo 1。

IBM ThinkPad 380笔记本电脑系列于1997年推出。 1998年3月，阵容进行了更新。 至少从处理器的角度来看，这是一个巨大的进步时期。 在我自己的坐标系中，即使是1996年发布的Pentium MMX似乎也非常强大，并于1997年5月推出了Pentium II，直到2000年代初一直如此。 但是，对1998年IBM笔记本电脑产品线的更新仍然使用频率为200或233兆赫兹的Pentium MMX移动处理器。 但是今天我们有一个流氓经济的职位，所以150兆赫就足够了。 1997年笔记本电脑的价格是多少？


笔记本电脑很贵。 这是1997年11月来自PC Magazine的广告。 最受关注的是ThinkPad 380ED笔记本电脑，该笔记本电脑是我多年前开始从事老式笔记本电脑的爱好。 在脚注中，您可以看到本文英雄的规格和价格。


我的笔记本电脑上的规格与广告完全一致：

处理器 ：Intel Pentium MMX 150 MHz
内存 ：16 MB EDO SDRAM（后来我将容量增加到32 MB，最大为64）
硬盘 ：2.1 GB
显示屏 ：12英寸，800x600像素，FRSTN技术，65k色
显卡 ：Neomagic Magicgraph 128ZV，内部存储器1,125兆字节
连接器和通讯 ：IrDA 1.1，PCMCIA 16位，串行和并行端口，VGA，端口复制器
电池 ：镍氢电池，续航时间长达两个小时
尺寸和重量： 300x233x60毫米，3公斤


在美国，这种笔记本电脑的价格为2099美元，折合通胀后为3342美元。 在笔记本电脑上增加CD-ROM驱动器和功能更强大的处理器的小升级将花费500美元。 但是该系列的高端设备-带TFT矩阵和5G硬盘驱动器的380ED-已经花费4,000美元（按通胀调整后为6,367美元）。 在2019年，您可以以2-3千美元的价格购买一台非常好的笔记本电脑，但是在那些日子里，只有预算选项可以提供这种钱。 不要忘记，ThinkPad 3xx系列通常由入门级便携式计算机组成。 更轻薄的第五系列笔记本电脑和7xx台式电脑替代品的价格甚至更高：同一则广告提到了ThinkPad 760XD，带有入门级配置，价格为4700美元（现代价格为7500）。


我想知道他们最初提出的3-5-7数字序列是如何将预算，普通和高级设备分开的吗？ 在宝马？ 低于2,000美元，仅是由于性能进一步恶化，或者如果您寻找鲜为人知的品牌，便可能导致价格下降。 1998年，IBM将发布一款起价为1600的ThinkPad 310E笔记本电脑。它与本届新闻发布会的英雄相差无几，只是屏幕会变得更糟，尽管看起来会更糟？

当屏幕很糟糕时


我能说什么，显示视频更容易。


在回顾2008年旗舰产品ThinkPad X301时，我已经抱怨过：它似乎是一台不错的笔记本电脑，相对适合现代任务，但显示屏却很合适。 今天，我们完全被具有大视角的IPS屏幕所宠坏了，我们对普通的TFT显示器存有疑问：值得对屏幕进行一些不同的改变，并且颜色立即开始变形。 在1997年，选择是在新型的昂贵TFT和采用超扭曲向列技术的廉价显示器之间进行选择。 根据STN技术，制造了第一台笔记本电脑的黑白屏幕，然后出现了彩色屏幕。 与通常的TFT屏幕（在当时，它们被称为“有源矩阵”，意味着能够分别控制每个像素的开和关）相比，它们的特点是响应时间极高，对比度低，工作视角小。


ThinkPad 380E的笔记本电脑显示器使用快速响应STN技术制成，但是此快速转换可转换为150毫秒的护照响应时间（前两个TFT屏幕为75毫秒）。 对比度是35：1（过去的TFT显示器是100：1，现代的是1000：1）。 通常，可以通过两个控件来识别具有CSTN显示屏的老式笔记本电脑：除了显示屏通常的亮度之外，您还可以调整对比度。 380E不遵守该规则：亮度根本无法调节，并且一个滑块控制对比度。 这是一个重要的调节器，它使您可以将显示从“什么都看不见”状态调整为“可以读取但有困难的状态”。 老实说，在使用笔记本电脑时，无论是色彩再现还是响应时间都不会给我带来麻烦。 但是视角太糟糕了。 但是要付出代价，CSTN和TFT上的笔记本电脑与采用常规IPS屏幕的笔记本电脑和采用OLED显示屏的高级机型的方式有所不同。


预算笔记本电脑的角度来看


为什么在九十年代后期需要笔记本电脑？ 1998年1月，在 PC Magazine的社论中，杂志编辑Michael Miller明确定义了目标受众：需要访问Internet，需要访问大量数据并能够随时与他们合作的经常出差的专业人士。 只是顺便提及一类提到的人，他们只携带笔记本电脑上下班。 尚未形成一种情况，其中给新员工提供笔记本电脑要比安装带有显示器和电线的台式机容易。 我认为现在没有像现在这样，对所有员工，直到保安员和秘书，都放下了计算机。 甚至预算模型也不是每个人都可以使用的：1998年，我在商店货架和电影中看到了笔记本电脑。


380E缺少员工进行商务旅行六个月的主要任务-内置调制解调器。 通常，这是我的收藏集中遇到的唯一困难的设备：我没有光驱，没有USB（一年后会出现），如何向其中上传大量数据？ 在软盘上？ 但是，通过安装PCMCIA扩展卡可以毫无问题地添加调制解调器，就像将便携式计算机连接到本地网络或添加USB闪存驱动器一样。 这台笔记本电脑不支持32位PCMCIA卡，但是我有几个合适的旧适配器。 但是总的来说，由于我们前面有这样一个故意限制的设备，因此通过串行端口模拟调制解调器连接将很有趣。


奔腾II及更高版本上的设备仍可以尝试以某种方式吸引现代性，但是这里的情况有所不同-为什么呢？ 我什至没有弄清楚这个博物馆的展览，而是开始研究安装的软件。


97-99年笔记本电脑上的BIOS极为有限，您只能在此处设置启动顺序。 所有其他设置（打开和关闭内置设备，分配中断）都是在IBM专有实用程序中进行的，该实用程序在Windows和DOS的版本中均可用。 在Windows下，当然更方便。

Windows 95现代评估


在90年代，俄罗斯用户（不仅是用户）对Windows形成了如此复杂的爱与恨结合。 几乎每个人都使用Windows，以OS / 2甚至Unix / Linux形式出现的替代品非常少见。 同时，Windows每天都是关于不稳定和故障的笑话。 我对Windows 95和Windows 95的印象一直留在我的记忆中，因此很难弄清楚哪里有更多问题，哪里有更多好处。


可以肯定地说Windows 95是一个突破性的操作系统。 对于没有经验的现代用户而言，以前的版本Windows 3.1将会很复杂：位置和启动方式尚不清楚。 Windows 95列出了到目前为止使用的用户界面概念：带有相同名称菜单的“开始”按钮，控制面板，带有时钟的任务栏和驻留程序的图标。 它是越野车还是不可靠？ 从某种意义上讲，是的，它确实做到了：部分原因是技术的根本变化，虚拟设备驱动程序的出现以及对长名称的支持。 并非所有软件和设备驱动程序开发人员都准备好进行这些创新。 有大量的旧程序可能在新环境中无法正常运行。

也许俄罗斯市场的特点与Windows竞争：安装在来源不明的计算机上的盗版副本，是由任何东西组装而成的，在这种气氛下，任何东西都是越野车。 从心理上讲，不可能立即从DOS键盘文本界面重建为鼠标锐化的Windows。 在某些地方，它根本不起作用-许多仍然使用面板文件管理器。 我一直将Windows 95视为Windows 98的测试版。似乎在第95版发布三年后，我们修复了这些错误，添加了缺少的功能，并最终变得正常。 是这样，但这是另一个角度。


Windows 95的原始版本甚至都不支持TCP / IP网络。 此时此刻，微软公司高管对互联网前景的评估（以现在的形式存在）严重错误地估算了出来。 我以自己的方式尝试以Microsoft Network的形式提交自己的专有Internet。


在Windows 98发行版中，该错误已得到解决，但如何解决：如何将它们放置在似乎根本不需要推动网络的地方。 最著名的例子是不断下降的活动桌面 ，实际上是浏览器而不是桌面墙纸。 最后，此Windows 98崩溃成了实时演示，而不是95：


从这个角度来看，Windows 95不是beta 98​​，它是Microsoft最新的纯脱机操作系统。 您可以将它与适合所有人的Windows 7以及第98位（与Windows 8相比）进行比较，在Windows 8中，最大的移动性和在触摸屏上的锐度被认为是不合理的。 当然，已对时间进行了调整：Windows 95中根本没有很多熟悉的功能，这就是为什么挫败感会定期发生的原因。 但是我可以确认，在专门为Windows 95创建的功能相当强大的ThinkPad 380E计算机上，该系统运行稳定。

奇迹宝箱


与1998年更昂贵的ThinkPad 600相比，ThinkPad 380E看起来还很大。 更不用说现代笔记本电脑了（上面的照片是Thinkpad X1 Carbon，在这种情况下它甚至看起来都不是笔记本电脑）。


但是，尺寸（厚度只有6厘米！）和重量似乎对完全预算的屏幕而言并不是一个严重的缺点。 大约10年前，我通过在Linux上安装Linux修改了稍先进的ThinkPad 380ED来下载种子。 我不会使用廉价笔记本电脑做任何事情-让它保持历史上可靠的“时间胶囊”，并使用原始软件，桌面墙纸和专有实用程序。 除非需要及时拆卸，否则请卸下硬盘驱动器并将映像保存在某处。 在我看来，除了硬盘驱动器之外，这款笔记本电脑中的其他所有东西都相当可靠，并且可以使用很多年。


1997年，ThinkPad 380E是一台很酷的笔记本电脑，但您仍然想要一款功能更强大，更薄，更时尚的笔记本电脑。 现在，与昂贵的IBM模型相比，它具有一项重要的优势。 由于经济原因，没有在380th系列的情况下使用品牌橡胶涂层，最终使它完全无法使用。 这种廉价型号的第一任所有者很可能进行了升级，而没有那么积极地使用笔记本电脑。 这可能不是我收藏中最出色的笔记本电脑，但绝对是保存最完好的笔记本电脑。
