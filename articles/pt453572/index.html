Microsoft Edge para macOS

No mês passado, anunciamos as primeiras versões da próxima versão do Microsoft Edge para Windows 10. Hoje, temos o prazer de anunciar a disponibilidade do canal Microsoft Edge Canary para o macOS. Agora você pode instalar compilações de visualização no site Microsoft Edge Insider para o seu macOS ou PC com Windows 10, com mais suporte à versão do Windows em breve.



Criando uma experiência de usuário “semelhante a Mac” para o Microsoft Edge


O Microsoft Edge para macOS oferecerá a mesma nova experiência de navegação que estamos visualizando no Windows, com otimizações de experiência do usuário para que você se sinta em casa em um Mac. Estamos adaptando a aparência geral para corresponder ao que os usuários do macOS esperam dos aplicativos nesta plataforma.



Temos o compromisso de construir um navegador de classe mundial com o Microsoft Edge por meio de recursos diferenciados de experiência do usuário e serviços conectados. Com esta versão inicial, fizemos várias alterações na interface do usuário para alinhar com a linguagem de design da Microsoft e, ao mesmo tempo, torná-la natural no macOS.


Exemplos disso incluem vários ajustes para corresponder às convenções do macOS para fontes, menus, atalhos de teclado, capa de título e outras áreas. Você continuará vendo a aparência do navegador evoluir em versões futuras à medida que continuarmos experimentando, iterando e ouvindo os comentários dos clientes. Recomendamos que você compartilhe seus comentários usando o smiley "Enviar feedback".



Além disso, estamos projetando experiências de usuário exclusivas do macOS, aproveitando os recursos específicos de hardware disponíveis no Mac. Por exemplo, fornecendo ações úteis e contextuais por meio da Touch Bar, como atalhos do site, alternância de guias e controles de vídeo, além de permitir uma navegação familiar com os gestos do trackpad.



Apresentando os Canais Microsoft Edge Insider para macOS


As novas compilações de visualização do Microsoft Edge para macOS estão disponíveis nos canais de visualização que chamamos de "Microsoft Edge Insider Channels". Começamos lançando o Canal Canário do Microsoft Edge Insider, que você pode baixar e experimentar no site do Microsoft Edge Insider . Este canal está disponível a partir de hoje no macOS 10.12 e superior. O Dev Channel será lançado muito em breve e, uma vez disponível, você poderá baixá-lo e instalá-lo lado a lado com o Canary Channel. Você pode aprender mais sobre nossa abordagem e o que esperar dos diferentes canais em nosso blog desde o mês passado .



Uma plataforma consistente e ferramentas para desenvolvedores web


Com nossa nova base Chromium, você pode esperar uma experiência de renderização consistente nas versões Windows e macOS do Microsoft Edge, bem como as mesmas poderosas ferramentas de desenvolvedor encontradas no Windows.


Pela primeira vez, os desenvolvedores da Web agora podem testar sites e aplicativos da Web no Microsoft Edge no macOS e ter certeza de que essas experiências funcionarão da mesma maneira na próxima versão do Microsoft Edge em todas as plataformas. (Observe que recursos específicos da plataforma, como descriptografia de conteúdo do PlayReady no Windows 10, devem continuar sendo detectados pelos recursos para obter a melhor experiência nessas plataformas.)


Assim como nas versões de visualização do Windows, nossa nova versão do macOS também inclui suporte para Progressive Web Apps instaláveis ​​e baseados em padrões, que você pode inspecionar e depurar usando as ferramentas de desenvolvedor do navegador. Estamos trabalhando para fazer com que os PWAs se sintam em casa ao lado de seus aplicativos nativos; portanto, quando instalados, eles aparecerão no Dock, no alternador de aplicativos e no Spotlight como um aplicativo nativo.


Compartilhando seus comentários


Temos o prazer de compartilhar com você nossa primeira compilação do Microsoft Edge Canary para macOS! Obter seus comentários é uma etapa importante para nos ajudar a criar um navegador melhor - consideramos essencial criar a melhor experiência de navegação possível no macOS. Esperamos que você experimente a visualização hoje e aguardamos seus comentários e participação na comunidade Microsoft Edge Insider.


Se você encontrar algum problema, e para enviar comentários ou compartilhar sugestões com a equipe, vá para os fóruns da comunidade Microsoft Edge Insider , entre em contato conosco no Twitter ou use a opção "Enviar comentários" no menu Microsoft Edge para deixe-nos saber o que você pensa.


Para desenvolvedores da Web, se você encontrar um problema reproduzido no Chromium, é melhor registrar um bug do Chromium . Para problemas na versão existente do Microsoft Edge, continue usando o EdgeHTML Issue Tracker .
