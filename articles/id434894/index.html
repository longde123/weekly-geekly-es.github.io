Seni perdukunan atau firmware khusus untuk Olinuxino. Bagian 1

Setelah di kotak lama, papan Olimex Olinuxino ditemukan pada prosesor Allwinner A13, dengan papan 512MB. Setelah mencari-cari, konverter USB-to-ui / spi buatan rumah lama pada drive FT ditemukan di sana (yang pernah dilakukan untuk firmware AVRok) dan catu daya dari beberapa jenis telepon radio, semua ekonomi ini ada pada foto di bawah ini.


Sangat disayangkan untuk dibuang, saya tidak ingin menyimpan terlalu banyak sampah, jadi diputuskan untuk mengumpulkan server web sederhana dari ini, untuk dimainkan.




Set ini hanya memiliki satu masalah .... tidak ada ethernet, tetapi ada konektor di bawahnya dengan catu daya 3.3V. Tetap untuk yang kecil, lepaskan modul wifi dari tablet yang rusak lama dan kumpulkan semuanya. Modul Wifi di tablet di sudut kanan atas disegel. Itu harus dilepas sebagai kabel dengan antena di ujungnya.




Modul pada chip RTL8188 dari realtek bukanlah pilihan terbaik, tetapi ada sesuatu. Itu harus diinstal pada papan perantara untuk pemasangan lebih lanjut di papan. Hasilnya, kami mendapatkan sesuatu seperti ini, modul wifi dapat dilepas.



Selanjutnya, kami menyatukan semuanya dan mendapatkan lampu hijau dan koneksi ke wifi, tapi itu nanti, dan dalam perjalanan ke ini Anda masih perlu mengumpulkan firmware untuk ini semua.



Pada artikel selanjutnya, kami akan mengumpulkan firmware untuk board dari awal, dimulai dengan bootloader dan diakhiri dengan ubuntu 18.04. karena Tidak ada memori nand di papan tulis, maka kami akan mengumpulkannya untuk opsi boot dari kartu SD. Dan sejak itu Jika kami tidak menggunakan monitor, maka Linux akan menjadi kecil tanpa adanya xorg.

Secara umum, untuk dilanjutkan.
