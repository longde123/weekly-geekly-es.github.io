Bingkai baja printer 3D Anet A8

Meskipun harganya murah, Anet A8, bahkan tanpa modifikasi, memungkinkan Anda mencetak model dengan kualitas yang baik.

Contoh pencetakan 3D tanpa modifikasi pada Anet A8 (model dari jaringan)


Tapi, tidak ada batas untuk kesempurnaan, pemilik menambahkan penegang sabuk, mengubah bantalan linier dan meningkatkan kekakuan frame , memperbaikinya pada chipboard / lembaran kayu lapis dan memasang spacer.

Pada artikel ini saya akan menjelaskan nuansa yang saya temui dalam proses perpindahan dari bingkai akrilik ke kerangka baja.



Sedikit latar belakang
Menurut saran para ahli dari jaringan, saya mengumpulkan Anet A8 saya menggunakan kunci benang, biru, dari toko mobil terdekat. Sayangnya, komposisi fiksatif bereaksi dengan akrilik dan menyebabkan keretakan. Karena Saya memperbaiki printer pada lembar chipboard, ini tidak menimbulkan konsekuensi serius. Namun demikian, perfeksionisme tidak memberi saya istirahat, dan pada titik tertentu saya tidak bisa menolak dan memesan kasing / rangka / rangka baja.


Paket bundel


Dalam kit datang cincang, baja lembaran dicat jauh mirip dengan akrilik asli dan satu set sekrup, mur, ring.

Pabrikan lambung kapal percaya bahwa untuk modernisasi kita perlu membeli:

* Poros yang dipoles dengan panjang L = 396 mm x 2 pcs (saya buka tanpa itu, detailnya di bawah)
* Linear bearing LM8UU x 4 pcs
* Timing belt GT2 - 1 meter x 1 pc
* Bantalan 608zz x 2 lembar (mis. Dari spiner)

Majelis


Instruksi yang cukup rinci ada di situs web produsen. Saya akan menggambarkan kesulitan yang saya hadapi secara pribadi.

Lubang untuk mur di rangka baja lebih besar daripada di akrilik, dan mur pengunci ditempatkan di dalamnya tanpa masalah. Jadi kunci ulir tidak harus digunakan. Tetapi mur dengan grover memiliki ukuran eksternal lebih kecil dari yang asli dan untuk mengencangkan saya harus memegangnya dengan kunci kobra . Saya sangat beruntung memiliki bantalan linier dan konvensional, dan juga sabuk. Tetapi poros yang dipoles untuk sumbu Y (meja printer) dari kit asli ternyata lebih pendek satu sentimeter. Saya meningkatkan poros dengan baut M8, mur dan mesin cuci (poros lama bersandar pada mur), dan saya mengasah bagian yang menonjol, yang mencegah meja bergerak.



Kit tidak memiliki sisipan pemasangan motor plastik. Idealnya, itu harus dicetak terlebih dahulu. Saya menggunakan dudukan kuningan sebagai ganti sisipan.



Saya juga harus mengotak-atik limit switch, sesuai dengan instruksi diasumsikan bahwa mereka digabungkan dengan papan kecil dan sakelar batas sumbu Y harus diletakkan di rak. Dalam kasus saya, sakelar batas tidak memiliki papan, karena sumbu Y harus mengotak-atik. Saya memotong dudukan limit switch lama dan memperbaikinya pada sekrup M4.



Juga, trailer sumbu-X dalam instruksi perakitan untuk badan baja ada di sebelah kanan, meskipun di akriliknya ada di sebelah kiri. Tanpa mengubah firmware, homing tidak akan berfungsi. Saya meningkatkan batas kabel sakelar dan meletakkannya di sebelah kiri.



Beberapa lagi gambar elemen printer







Kesan umum


Tujuan utama tercapai, frame sangat tahan lama. Kekakuan lebih dari cukup. "Rak" yang nyaman untuk gulungan dengan plastik. Sumbu X seimbang dengan baik, motor penggerak digeser ke sisi lain sebagai lawan dari extruder. Bantalan linear diimbangi sedekat mungkin dengan rel vertikal. Menambah tinggi cetak maksimum dengan menurunkan meja (bantalan linear dapat dipasang langsung ke meja tanpa menggunakan pemegang bantalan). Serangan kereta ekstruder hampir "hilang."

Tapi bukan tanpa lalat di salep.

Bagi saya, masalah nomor satu adalah kebutuhan untuk membeli poros dan bantalan tambahan (lihat set lengkap). Yang kurang penting, sabuk sumbu-X meluncur dari tensioner ke kanan, saya harus memagari desain mesin cuci untuk melawan efek ini.


Tidak ada tensioner sabuk x-axis.

Jika Anda tidak memiliki Anet A8, maka alih-alih sekelompok koper baja Anet A8 +, lebih baik untuk melihat model lain segera dalam wadah logam.

Saya menganggap pembelian sasis baja yang dibenarkan untuk pemilik Anet A8 yang tidak puas dengan:

* kekakuan struktural (termasuk setelah modifikasi)
* Penampilan / komponen estetika
* harga tidak menakut-nakuti
