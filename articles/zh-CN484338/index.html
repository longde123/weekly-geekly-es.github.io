三星的霓虹灯项目：数字银行家，电视节目主持人，同伴


Neon项目中的超现实数字人是真实的。 好吧，有点。 是的，不要问他们天气预报...

这家神秘的公司源于三星技术和高级研究实验室（STAR Labs），于1月7日至10日在拉斯维加斯举行的CES 2020上展出 。 他们介绍了霓虹灯技术-“一种看起来像人的虚拟生物，表现出情感和智慧。” 实际上，Neon创建了看起来像真实人物的视频聊天机器人（以下简称Neons）； 这些不是全知的助手，机器人，人员的替代品或副本，它们不会告诉您天气或亚伯拉罕·林肯去世的时间。 霓虹灯不是人工智能助手，它们更像我们。 这些都是独立的（尽管是虚拟的）生物，可以从自己的经验中学到东西。 他们说，他们对世界一无所知，也不是访问互联网来澄清外界天气或打开自己喜欢的音乐的界面。

相反，这些聊天机器人是为交流和类似人的行为而设计的。 他们保留记忆和学习技能，但没有身体上的体现（至少目前如此）。 霓虹灯可以帮助实现目标或适合需要“人性化”的课程-成为教师，财务顾问，医务人员，门房，演员，新闻秘书和电视节目主持人。 每个霓虹灯都是独特的，具有自己的个性。 尽管这些生物可以借用真实人物的特征并具有相似的外表和声音，但它们仍不能完全复制现有人物。

Neon首席执行官兼Star LABS首席执行官Pranav Mistry在新闻稿中说：“地球上有数百万个物种，我们希望再增加一个。” “霓虹灯将成为我们的朋友，伙伴和同伴，他们将根据与世界的互动不断学习，发展和形成记忆。”

霓虹灯甚至在CES 2020大会之前就沙沙作响 ，因为没人真正了解即将发生的事情。 自12月中旬以来，该公司已在Twitter上发布了几则预告片，使用了“人工”一词； 唯一可以肯定知道的是Pranav Mistry的名字，他是三星的长期研究员，自10月起担任STAR Labs的首席执行官。

尽管现实的数字人可以成为某人的伴侣，但问题在于他们是否会过于现实？ 所展示的技术似乎在“ 险恶的山谷 ”中调情。这种效果使人们在遇到与人非常相似但与人非常相似的物体时感到不喜欢和/或厌恶。 例如，2004年的电影《 极地特快 》没有克服不祥之谷：动画人物看上去逼真，但不足以与真实人物混淆。 这种真实感在影片上起到了欺骗作用，因为它使一些观众感到不舒服。

炒作


1月5日，Mistry在推特上发布了两张“人造人”照片，他称之为CORE A3。 然后将完全不同的Reddit视频放在一起并上传到YouTube：


这些材料引起了人们的兴趣，他们开始怀疑霓虹灯是什么。 也许这是Bixby智能助手的替代品？ （Neon迅速驳斥了这一假设）它将仅在三星智能手机上可用吗？ 我们是否将以“ 荒野西部世界 ”系列的形象不久地和平生活？

尽管Neon由三星赞助，并且是其自己的STAR Labs的一部分，但人造人并不是Samsung的产品。 这个发展也与Bixby无关，后者于2017年出现在Galaxy S8智能手机上。 STAR Labs将自己描述为“未来的完全独立的工厂，其任务是将科幻小说变为现实。”

正如谷歌，亚马逊和三星突然发现的那样，为了让智能设备真正有用，您需要以语音助手的形式在其中实现AI。 所有主要参与者都在语音助手上进行投资：相信它们是我们与机器交互的未来。 智能技术的最终目标是在用户要求某项东西之前就预测他们的需求，或者以一种使用户忘记他没有与人交谈的方式行事。

在12月，Mistry宣布，在童话成真之前，人工智能正在等待多年的开发和实施，似乎暗示着霓虹灯将在不久的将来面世。 他说：“只要电影干扰了我们的现实感，“虚拟人”就将成为现实。” “作为新闻主播，秘书甚至电影明星，数字人很可能成为我们生活的一部分。”

为什么选择R3？


数字人使用Neon的两项专有技术。 第一个称为Core R3（真实，实时和响应-实时，实时和响应）。 这就是霓虹灯可以快速，人性地响应的原因。 第二个叫做Spectra，它负责收集信息，学习，情感和记忆。

Neon将Core R3描述为行为神经网络，进化生成系统和计算现实领域的一项先进技术。 该公司表示，Core R3“受到大自然的节奏复杂性的启发，并接受了有关人们的外观，行为和互动方式的精心培训”。 延迟是几毫秒，这使霓虹灯可以实时响应和响应。 该公司的常见问题解答说：“ Core R3可以创造出可以超越人类感知的与真实世界区分开的世界。”

聊天机器人可以采取医生或其他专家的形式。 他们也打算成为合作伙伴，而不是助手。

同时，Spectra仍在开发中。 该公司承诺将在今年的Neonworld 2020某个活动上进行预演。 没有指定什么类型的事件以及将在何处举行。 Neon还在努力改进Core R3引擎。

隐私权与安全性


霓虹灯与生活中的人们的相似之处引发了一个问题：是否可以使用技术来制造声音-视频中真实的人做着他们实际上没有做过的事情。 Neon认为，尽管数字人的外观可以基于真实的人，但是该技术与Deepfake和其他面部修复技术有根本的不同。

“ Core R3不会操纵捕获的场景或场景序列； 相反，实时引擎会创建前所未有的独特行为和交互。 因此，Core R3创造了一个新的现实。”

Neon补充说，他们设计Core R3的部分原因是出于保密和信任的道德考量。 Core R3使用高级安全协议保护数据。 只有最终用户和制造商才能访问数据。 数字人绝不会未经允许就共享个人数据。 Mistra保证：“隐私是我们的血液。” “我们认为技术奇迹不应损害我们的数据。”

该公司及其合作伙伴正计划在今年内发布Neon的Beta版。 期望可以购买例如霓虹灯的许可证或霓虹灯订阅。 关于数字人在电影院和新闻中可能出现的现象：“人们一直梦想着这种虚拟生物，将其描述在书籍中并在电影中进行展示。 霓虹灯将进入我们的世界，以创造更美好的未来。 霓虹灯还可以是瑜伽教练，银行家，K-pop明星，时装模特等。 -至少出于技术作者的意图。
