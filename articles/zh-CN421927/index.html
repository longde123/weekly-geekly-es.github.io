智能手机General Mobile GM8 Go的概述：具有Android 8.1 Oreo Go Edition的土耳其预算雇员

问候大家！

大约一周前，价格低廉的具有Android 8.1 Oreo Go Edition操作系统的General Mobile GM8 Go智能手机进入了我的坟墓，它仍然是我们地区“绿色机器人”的一种罕见版本。 我很好地驾驶了智能手机，以这种方式对其进行了测试，然后，我与自己进行了讨论，现在我准备与您分享我的感受。



因此，该设备由土耳其公司General Mobile发布。 她更喜欢把自己当作美国人。 这正是乌克兰运营商Lifecell网站上确认的内容（请参见下面的屏幕）。 后者负责将通用移动GM8 Go推向乌克兰市场。



所有这些“美国故事”都不足为奇。 从中国工厂订购设备的本地小品牌非常喜欢说它们实际上是美国，韩国，欧洲或其他一些国家。 如果仅“注册”在一个或多或少是认真的国家/地区。 例如，具有白俄罗斯血统的Prestigio向塞浦路斯点头，而Highscreen有时会提到其所谓的德国血统。 通用移动公司也是如此。

通用移动GM8 Go的价格为3,000格里夫纳（110美元，7,200卢布）。 事实证明，该设备不是很便宜。 Google，我提醒您，在发布Android 8.1 Oreo Go Edition时，它承诺使用此版本操作系统的智能手机的价格将低于100美元。 如您所见，本文的主人公超出了指定的框架。 无论如何，在乌克兰市场。

通用移动GM8 Go有什么有趣的地方？ 该设备首先吸引了我的软件。 正如我所说，该模型可在Android 8.1 Oreo Go Edition上运行。 这是Android的轻量级版本，主要设计用于硬件较弱，RAM（1 GB）和内部内存（8-16 GB）少的智能手机。 中兴通讯，阿尔卡特，HMD（诺基亚）和许多其他品牌中也有类似的设备。 在俄罗斯，在乌克兰，在其他国家，使用Go Edition的后苏联空间模型几乎尚未普及。 但是，也许不久之后，我们将大量使用它们。 因此，熟悉通用移动GM8 Go并了解此Go Edition是什么很有趣。



为了秩序起见，让我提醒您，General Mobile和Lifecell（以前称为“运营商”）之间的合作起源于2000年末。 然后在乌克兰，您可以在生活沙龙中购买General Mobile DST-01，这有点奇怪，朴实无比，并且让人回想起当时的Fly手机。 但是，此后发生了很多变化。 如今，通用移动产品不再受质量或设计的困扰。 智能手机就像智能手机。 时而有趣，时而平凡。 通用移动GM8 Go属于什么类别-我在下面告诉。


通用移动DST-01

供货范围


该套件异常丰富。 废纸，2A充电，MicroUSB-USB电缆-这些都是标准的基本组件。



我将耳机和保险杠盖放到非标准位置。 这些配件很少随智能手机（特别是预算款）一起提供。

保险杠的质量正常-很明显它很便宜，但是它坐得很紧。



耳机质量不是很高，但是我认为90％的GM8 Go客户会满意。 他们还会说谢谢-完全将它们放入套件中。





设计，建造，房屋特征


通用移动公司的中国合作伙伴在设计方面没有创造任何新东西。 今天，数十种智能手机都有类似的情况。 只需在GM8 Go和例如Xiaomi Mi A1旁边放置，就可以了解它们有很多共同点。





“土耳其”智能手机的开发者试图弥补像Mi A1这类不可分割的模型的不足。 但是，实际上，通用移动GM8 Go的后盖是可移动的。 只需用指甲撬面板即可使用特殊的插座...



...拉一点-面板将保留在您的手中。 请注意，面板实际上是白色的，金色层是油漆。 （在我的情况下，该层只是金色的，但在自然界中也有此智能手机的灰色版本。）由于某种原因，这种感觉并不会消失，因为将它戴在口袋里一两个月后，这种油漆会开始磨损。



通用移动GM8 Go机壳的设计中没有金属。 可以看出，他们试图使它在视觉上更加昂贵，以某种方式对其进行了修饰，但是……事实如此。 告诉我，同志们在窃笑吗？ 我回答：最近我从事全新的婆婆小米Redmi Note 5A-我安装了应用程序，设置了所有东西，等等。 它也是一种没有任何抱怨的设备，但是它的触感更加舒适且规格稍好，并且乌克兰的价格与美土耳其模式的价格完全相同。 但是，与小米相比，我将在下面返回。



GM8 Go中的电池是可移动的，可能是一个加号。 “可能”，因为购买另一块电池再与完整电池一起使用并不是那么简单。 无论如何，我在Lifecell网站上找不到该智能手机的电池。 但是，如果您通过速卖通设定目标并进行朗读，则可能会找到合适的电池。



卡插槽是分开的。 在GM8 Go中，您可以同时插入两个MicroSIM格式的SIM卡和一个microSD卡。 这是智能手机的绝对优势，因为今天有一半以上的新型号配备了笨拙的混合插座。 在这种情况下，您可以将两张SIM卡或一张SIM卡和一个USB闪存驱动器插入设备。





关于按钮-只需看照片并了解位置。 使用人体工程学，一切都很好，所以我不会就此止步。



接口连接器是MicroUSB。 毫不奇怪：在如此高价位的设备中，我从未见过Orthodox USB Type-C。



就车身和设计而言，结果是这样的：次要的（虽然现在已经足够好了，但现在却怪罪于智能手机，因为它们几乎都是次要的，这是愚蠢的）和相当便宜的价格（这已经是一个更严肃和合理的收费）了。 智能手机并不丑陋，也不漂亮，而只是中性的。 它不会在我内部引起生动的情绪，也不会引起拒绝。 优点是可移动电池和独立的卡插槽。

萤幕


GM8 Go中的显示器很时尚-纵横比为18：9（对角线5.5英寸，分辨率为1440 x 720像素，IPS矩阵）。 在此基础上，其优势将基本消失。 不，对于这样的钱（3,000格里夫纳，110美元，7,200卢布），屏幕对于智能手机来说是正常的。 但也并不出色。 视角广，图像略带黄色，在阳光下几乎看不见，黑色不是全黑，根本没有设置（包括色温）。 通常，屏幕就像屏幕。 见得更好。



显示范围-大猩猩玻璃（未指定代号）。 显然，没有疏油涂层。 好吧，或者它很弱。 因为显示屏表面一次变脏。

按照现代标准，联发科MT6739芯片组非常简单。 它具有四个频率高达1.5 GHz的ARM Cortex-A53内核，一个PowerVR GE8100图形加速器，一个LTE调制解调器（类别4），双频Wi-Fi 802.11n，蓝牙4.2和GPS / GLONASS接收器。



所有这一切都补充了1 GB RAM和16 GB内部存储器。 另外，您可以将存储卡最大存储为128 GB，因为该级别的设备通常仅支持32 GB。

令人高兴的是，我注意到通常实现的导航（支持GPS和GLONASS）。 对于基于联发科技芯片组的智能手机，这并非总是可以实现的。 而且这里的卫星搜索很灵巧，准确性很高。



再次是Wi-Fi，是双频。 在此级别的智能手机中并不总是可以找到此功能。

自治权


智能手机为我工作了大约两天。 对于3500 mAh的电池来说，这并不是很多，也不小，也不是最经济的28纳米芯片组。 简而言之，这是非常典型的结果。 是的，我在《沥青8》中玩了GM8 Go 5小时30分钟。 之后，智能手机就消失了。 更准确地说，他没有昏倒，但说仍然要收取3％的费用。 我自己把它淘汰了。

其他很小的时刻


在本小节中，我将讨论一些您特别无法坚持的要点。 但是我认为这也很重要，值得一提。

首先：GM8 Go具有LED指示灯，可以发出不同颜色的光。 充电时-橙色，如果有通知-绿色。 这是一个积极的观点，因为在预算设备中通常根本没有指标，或者它是单音的。



第二：GM8 Go具有杜比移动（Dolby Mobile）声音增强功能。 它可以与连接的耳机配合使用。 针对不同场景（电影，游戏，音乐等）有很多模式。 声音确实发生了变化，尽管我认为，在激活特定模式后，声音并不总是适合于特定情况。 例如，我更喜欢在“电影”模式下听音乐。 好吧，整个声音还不错。 不完美，不。 例如，实际上没有低音。 但也不错。 在这里，我要说的是扬声器（位于机箱底部）很大声。 从牛仔裤的口袋里，旋律可以完美地听到。



第三：GM8 Go具有基于两个麦克风的降噪系统。 一个在外壳的底部，第二个在后置摄像头附近。 该系统正常工作，当对话者上风时，您会正常听到，他会为您做同样的事情。 在平静的天气中，声音传输也很正常。



第四：GM8 Go具有指纹扫描仪。 效果一般（10个正确的手指识别中有7个）。 但总的来说，可以在此处使用它很好，因为在这种级别的智能手机上，开发人员通常更喜欢不使用指纹。

软体类


正如我所说，关于GM8 Go的最有趣的事情是该软件，即Android 8.1 Oreo Go Edition。 Google收到了Go Edition，因此采用了Oreo的常规版本并从中删除了一些内容。 这个版本a）有助于节省Internet流量，b）据说可以在较弱的硬件上更快地运行，并且c）占用智能手机内存的空间要少得多-这不是应该的，但实际上是。 假设智能手机具有内置的16 GB驱动器，就像General Mobile GM8 Go一样。 普通的Oreo会“吃掉”大约5-6 GB的内存，而Go Edition则要少于3 GB。这对于发展中国家来说非常重要，因为发展中国家的居民不仅买不起智能手机，而且买不起USB闪存盘。 由于Go Edition的“重量轻”，用户将获得更多存储照片，软件，音乐等的空间。

对于Android 8.1 Oreo Go Edition，可以使用特殊版本的Google应用程序（实际上是预装的）-Gmail Go，YouTube Go，Google Maps Go，Google Assistant Go。 同样，它们占用的内存比普通版本少得多，平均为50％。 另外，这些精简的应用程序在弱硬件上的运行速度应比常规Google服务更快。



这是一个好主意还是不好？我无法判断。 我没有发现Gmail和Gmail Go之间有任何特殊的视觉差异，但是更糟的是Google Maps Go与通常的Google Maps应用程序不同。 该界面更简单，使用起来更不方便，选项更少，但是它的工作速度...图速度-我会直言不讳。 放大时会出现抽搐和延迟，菜单会缓慢打开。



左侧-常用的Google Maps应用程序，右侧-Google Maps Go

同时，Oreo Go Edition上智能手机的所有者可以安装完全普通的Google服务版本。



有趣的是，GM8 Go上的常规版Google Maps不如Google Maps Go愚蠢。 正如他们所说，这是一个陷阱。 为什么有必要用轻量级的应用程序围栏花园-特别是在GM8 Go的示例上，这还不是很清楚。 更少的内存空间-是的，是的。 提高工作速度-不，我没有注意到这一点。 一切恰恰相反。 也许是智能手机的特定型号及其不完整的固件，也许-Google本身尚未对所有内容进行优化。 我不知道

GM8 Go上的Android界面通常运行速度很快，但是如果没有周期性的突然延迟就无法做到。 也就是说，是的，Go Edition可以完美清晰地运行，最终没有刹车，您不能这样做。 顺便说一下，从Go Edition版本的视觉差异来看，我注意到方形图标。





谷歌表示，作为Go Edition计划的一部分而创建的智能手机将很快收到安全补丁和最新的OS版本。 好吧，我不知道。 在日期为2018年3月的测试时，通用移动GM8 Go的安全级别。 而且智能手机未找到任何更新。

至于快速收到新版本的Android，我也不会roll之以鼻。 参加相似的Google程序Android One的同一个小米Mi A1仅在7月收到了Android 8.1 Oreo。 尽管即将推出该版本的操作系统。 简而言之，就接收更新的速度而言，Android One（带有Go Edition的型号）与Nexus相距甚远。 更新不会立即到达。

总的来说，我还是不明白-这叫Go Edition是什么样的花园。 谷歌报道说，配备此版本操作系统的智能手机（我记得，最大内存为1 GB的型号）的价格最高为100美元。 但是GM8 Go的价格是110美元。 并为此花钱，真的有可能找到一个带有2 GB RAM的模型，该模型通常可以（与GM8 Go无关）与普通的Android兼容。 和他的游戏。 以及几乎所有软件。 一个典型的例子是相同的小米Redmi Note 5A。 在这种背景下，为什么购买随Go Edition发布而购买的GM8 Go，有时也会减慢速度-我个人不明白。 为了快速更新？ 好吧，也许吧。 如果他们真的很快。 这远非事实。

但是，如果有人提供的智能手机价格在50-70美元之间，这要归功于Go Edition，它将以与GM8 Go相同的速度工作，那么，是的，在这样的设备中它是有意义的。 但是只要花110美元……总之，让我们继续下一个小节。

通用移动GM8 Go和小米Redmi Note 5A的比较


首先-我们研究特征。



有些点不能包含在表格中，因此应在其上涂上文字。 小米的塑料机身更舒适一些。 当然，这是主观的。 尽管如此，我相信Redmi Note 5A使用了更昂贵和更高质量的塑料。

尽管General Mobile如今已安装了最新版本的Android，甚至在Go Edition版本中也已针对硬件能力较弱的设备（1 GB或更少的RAM）进行了优化，但在界面周围移动和使用应用程序时有时会感到延迟。 我已经在上面谈到了这一点。 为了进行比较：在小米Redmi Note 5A中是Orthodox Qualcomm芯片组（针对GM8 Go中的MediaTek）和2 GB的RAM。 而且设备在速度上的感觉有所不同，它更快。

通用移动设备的电池更大，但是智能手机可以一次充电就可以工作。 相机可以与小米更好地拍摄更好的照片。

总的来说，尽管没有指纹扫描仪和第二个Wi-Fi范围，更微薄的包装和16：9格式（不是以非常时尚的方式），但小米模型似乎......好一点，更高档一点。 更有趣。 而且，当然，小米品牌已为我们所熟知和喜爱。 与通用移动不同。 我记得，与此同时，智能手机的价格是相同的。

合计


就像我说的那样，就目前的形式和乌克兰的实际情况而言，Android 8.1 Oreo Go Edition的概念对我来说并不明确。 通用移动GM8 Go或应该便宜得多（例如，2500格里夫纳，90美元或6000卢布），或者它应该有2 GB的RAM。 实际上，这将不必要使用Android Go Edition。



智能手机本身也许还不错。 但是竞争对手并不差，并且在某些方面要好得多。 有关所有这些软件优化和快速更新（如果有）的故事可能只对狂热的极客感兴趣。 如果他们占潜在购买者总数的一半，那就很好了。 总的来说，我希望Lifecell祝你好运。 对于销售商来说，这肯定会派上用场，因为要出售大量此类设备（甚至来自未知品牌）会非常困难。 如果可能的话。 即使考虑到货架，采用适当的方法，货架也能够出售非常不成功的产品。 GM8 Go不能归因于它们的数量，但我什至无法预测命运。
