关于牛顿第一定律在现代物理学中的曲解

在二十世纪初，出现了一些聪明的小前教育者，他们没有意识到科学创造力的方法论，并宣布了一些“局限性”，从而缩小了牛顿力学的适用范围（ I. Newton ）。

让我们从现代物理学教科书中牛顿定律的解释开始。 例如，在A.V. Peryshkina和E.M. 在第42页的Gutnik [1]上写道：
<...>从现代概念的角度来看，牛顿第一定律的公式如下：

如果其他物体不对其进行作用或补偿了其他物体的作用，这些物体可以保持速度不变的参考系统 。
顺便说一句，将此脏话与牛顿本人[2]的表述进行比较，如同一本书前面所述：
每个身体继续保持静止或匀速直线运动的状态，直到并且由于施加力迫使它改变这种状态。
如您所见，没有任何参考系的问题。 那又怎样 牛顿是否比“改善”他的法律的现代“智者”更“笨”？ 不，不！ 牛顿详细研究了……的问题：“ 明显的 ”和“ 真实的 ”运动！ 这是他在“定义”部分的“教学”中写的内容（请参见书[2]）：
区分真实运动和表观运动的起因是 产生这些运动必须施加到身体的力 。 真正的绝对运动不能发生，也不能改变，除非直接施加在运动物体本身上的力作用，而物体的相对运动可以在不对物体施加力的情况下产生和改变。 将力施加到与该运动有关的物体上就足够了。
也就是说，牛顿非常谨慎地考虑了参考系统的问题，并设法从明显的 （ 相对 ）运动中抽象出来，这些运动干扰了对所有运动和运动变化的根本原因-力的作用的理解。

牛顿和他的前辈天才哥白尼人拒绝托勒密的行星运动理论（ Πτολεμαΐος ）， 仅基于相对的，明显的运动的描述 ，因此，在确定导致这种抽象能力从本质上变形的原因的无能为力时， 才是天才。这样的动作。 事实证明，“矫正”牛顿正试图在托勒密和亚里士多德时代（ Aριστοτέλης ）抛弃我们...

顺便说一句，爱因斯坦的整个相对论都精确地考虑了相对的，明显的运动，而且这种理论的后果是无法解决的悖论也就不足为奇了。

还应该指出，在爱因斯坦的理论中，静止状态是绝对的，这又是对亚里士多德和托勒密时代的回归。 牛顿根据他的第一定律特别强调与v = const > 0相比， v = 0 = const没有优势。

惯性参考系在制定第一定律时是不可接受的，因为无论我们是否观察过程，是否进行测量，都必须执行物理定律。而且，惯性参考系的定义是通过牛顿第一定律（参见，例如[3]的第13页），然后使用“惯性参考系统”的概念来重新制定相同的定律...一种非常“科学”的方法...

因此，牛顿第一定律的主要思想之一是`` 静止状态 '' 不是特殊情况 ，而是相反， 这是 速度为零（即零时）时的``直线匀速运动状态''的特殊情况。 任何其他数字，在各个方面都等于零！

这种思想是对亚里士多德理论的主要打击，它与普通的运动思想相距不远（更确切地说，它没有任何地方），当时人们认为为了使一个物理身体运动并在将来保持这种运动，这是必需的。努力（力量）。 当然，那些运动示例，例如“抛弃石头的飞行”，“弓弦终止后的箭头飞行”，在其中看不到“支撑运动”，这些力量使这些关于运动的想法混淆了。

第二要点 。 他们声称牛顿“搜索”某种绝对参考点，最后在太阳中心“完成”了这些搜索。 虽然如此，但是第一定律只是否认零速度的绝对性 ，因此也否认了任何参考点的绝对性……这些“搜索”（如果实际上确实是……）很有可能-牛顿本人的另一种表现，不是作为物理学家，而是神学家。 确实，“ 有这样的参考系统 ”这一说法根本是错误的！ 通过精确的测量，我们将始终找到加速度。 即使是一艘以完全平静的恒速行驶的船，也不是惯性参考系，因为地球是球形的，并且船沿着海的曲面运动，更不用说地球本身在旋转……

还有更多 。 不管“量子力学”怎么说： 没有一个实验可以证明在微世界中没有实现这一定律！。相反，布朗运动证明了“普通”（即，没有任何进一步的解释）定律的实现：微粒从一个直线以恒定的速度互相碰撞。

也就是说，在上面关于缺少惯性参考系统的争论中，我感到有些兴奋吗？ 它们存在于微世界中吗？ 好像是这样！

但是，与此同时，在微观世界中没有加速……（！），因为没有像宏观宇宙中那样的“场”，并且所有的相互作用仅在很短的时间内通过“碰撞”发生在很小的空间区域中。为了证实这一论点，我将参考理查德·费曼的权威，他在[4]中声称：
