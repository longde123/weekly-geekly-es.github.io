Comment les stations de base explosent



De 2012 à 2014, des explosions dans les stations de base des opérateurs mobiles se sont produites dans plusieurs régions de la Russie. Ils n'avaient rien à voir avec les attaques, il n'y avait aucun reportage dans les médias. Une brève mention de cela ne peut être trouvée que sur les forums de l'industrie et les réseaux sociaux.

Un de nos lecteurs a dû travailler pour éliminer de tels accidents. Selon lui, les armoires de télécommunications climatiques ont explosé lors de tous les incidents. Il nous a envoyé des photos des lieux de l'explosion.

Les armoires de température où se trouvaient les batteries semblaient être déchirées de l'intérieur.

















Il est à noter que dans tous les cas, les armoires de télécommunications Ericsson SSC-02 ont explosé. Après plusieurs explosions, les installateurs partant pour l'entretien des stations de base ont été avertis qu'ils pourraient «bombarder». Ensuite, il a été considéré que les explosions étaient dues à une mauvaise installation des piles, le tuyau d'échappement de gaz était plié, ce qui a provoqué une accumulation de gaz à l'intérieur de l'armoire.

Voici une autre série de photos de l'explosion dans le territoire de Krasnodar. La vue des batteries qui éclatent de l'intérieur est horrible.





















Le blâme a été attribué au facteur humain. Comme, les installateurs montaient de travers, serraient un tuyau de gaz et voici le résultat.







Voici une photo apparue devant l'équipe de réparation de Penza.















Selon un spécialiste qui a travaillé sur les lieux de l'accident, la cause de l'explosion ne pouvait pas du tout être une installation «courbe», mais une erreur de fabrication. Il s'est avéré qu'après plusieurs explosions (il y en avait quatre au total), le vendeur a remplacé les cartes de commande des redresseurs de batterie. De plus, le système de ventilation et d'évacuation des gaz a été modernisé. Le vendeur a consacré six mois à ces travaux, mais seulement dans le territoire de Krasnodar, environ 2400 armoires de ce type ont été installées.
