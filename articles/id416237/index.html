Geekly articles weekly

Menjadi seorang desainer: dari lepas dari sebuah asrama hingga bekerja dengan perusahaan-perusahaan top dan meluncurkan produk Anda



Artikel ini telah menjadi yang paling banyak, penting, tetapi pada saat yang sama termudah untuk menulis, dan semua karena saya tidak membuat plot, tidak mencari fakta dan tidak membuat kesimpulan. Segala sesuatu di dalamnya adalah seperti itu, mulai dari kedatangan saya di Moskow dari sebuah desa kecil hingga hari ini.

(~ 17 menit untuk membaca)

Mulai



Saya, Alexey Makarov, Alexey Petrushin, Vladimir Kokhanov di Apps4All Hackathan (2013)

Saya berumur 18 ketika saya pertama kali naik panggung untuk mempresentasikan proyek. Ini adalah hasil dari hackathon 24 jam, di mana kami mengembangkan jejaring sosial yang terkait dengan geolokasi pengguna. Kemudian kami tidak memenangkan apa-apa, tetapi pengalaman praktis pertama saya dalam menerapkan desain tidak sia-sia.

Pemenang hackathon itu menemukan saya di LinkedIn dan mengundang saya ke tim mereka sebagai desainer, meskipun pada kenyataannya saya bukan desainer pada waktu itu.

Saya ingat bagaimana kami berkumpul di "Lingkaran" di Kursk dan mendiskusikan proyek-proyek yang, jika dilaksanakan, dapat menghasilkan uang yang baik. Saya terlalu muda untuk serius terlibat dalam percakapan, jadi saya hanya mendengarkan.


Di tengah, Artem Keydunov, kemudian pengembang di Uchi.Ru, dan hari ini pendiri Statsbot

Harus disebutkan bahwa saya datang ke Moskow pada usia 17 tahun dari desa kecil Ustye, tempat 5 orang belajar di kelas saya.

Saya sama sekali tidak siap untuk pindah dan tahun pertama di ibukota membutuhkan banyak upaya psikologis. Saya tidak akan bersembunyi, pikiran untuk kembali secara teratur mengunjungi saya, tetapi di dalam saya mengerti bahwa jika saya ingin mencapai sesuatu dalam hidup saya, tidak ada jalan untuk kembali.

Saya ingat betul saat itu - bulan-bulan musim gugur pertama yang hujan di tahun sekolah, ketika saya terbangun di asrama abu-abu dan mendengarkan hari yang baru, saya memaksakan diri untuk tersenyum melihat ke cermin, yang berada di sudut ruang bersama kecil, di mana pintu masuk, dapur dan kamar mandi digabungkan. Kebiasaan tersenyum, bangun dari tempat tidur, tetap bersamaku sampai hari ini.


Hostel "Persahabatan" (m. Stasiun River)

Saya belajar dengan tekun, tetapi saya tidak memiliki bakat khusus. Satu-satunya hal yang berhasil dengan baik bagi saya adalah pemrograman. Saya adalah yang terbaik dalam kelompok itu, meskipun dalam banyak mata pelajaran lain saya jauh tertinggal. Setelah menerima 3 poin untuk kursus pertama, dalam dua minggu saya lulus ujian dengan sangat baik dalam 45 menit. Dan ujian selanjutnya juga.

Meskipun berhasil dalam subjek khusus, saya merasa bahwa pengetahuan yang didapat tidak akan cukup bagi saya untuk membangun karier. Bahasa pemrograman yang kami pelajari sudah ketinggalan zaman. Dan langkahnya terlalu lambat, jadi musim panas setelah tahun pertama saya mengabdikan diri saya untuk pendidikan mandiri di bidang pengembangan web.

Setelah mempelajari dasar-dasar HTML / CSS, saya ingin mulai berlatih, tetapi tidak tahu di mana menemukan contoh desain situs web di mana saya dapat mengasah keterampilan saya. Sebagai hasilnya, saya mulai menggambar situs-situs paling sederhana di Photoshop, dan setelah itu saya mencoba membuatnya dengan tepat. Pada tugas-tugas seperti itu, saya pertama kali berkenalan dengan sesuatu yang mirip dengan desain (saya tidak tahu apa-apa tentang UX / UI saat itu).

Saya tidak ingat mengapa, tetapi pada hackathon yang disebutkan di awal, saya memutuskan untuk mendaftar sebagai desainer. Saya tidak memiliki portofolio, tetapi tim orang-orang berbakat mengundang saya ke tempat saya, dan pengalaman ini tak terlupakan: sehari tanpa tidur, hitung mundur, banyak kopi, dan pekerjaan yang berkelanjutan.

Menjelang akhir acara, ketika tugas saya sudah selesai, saya mendengar dari Alexei Makarov: "Ilyukh, akankah Anda membuat proyek?" Saya mencoba menolak, tetapi bahkan di dalam, saya mengerti bahwa kesempatan ini tidak boleh hilang dan, setelah beberapa alasan konyol, saya setuju. Sejak saat itu, kebutuhan telah hidup bersama saya untuk pergi sendiri dan meninggalkan zona nyaman yang akrab.

Pekerjaan pertama


Saya tidak mengambil kesuksesan pertama dalam desain dengan serius (saya tidak tahu mengapa), jadi saya memulai karir profesional saya sebagai desainer tata letak. Pada tahun pertama saya, saya membuat resume di hh.ru, LinkedIn, Superjob, dan pada akhir 2013 saya datang ke wawancara pertama saya di startup connect2me (saya tidak ingat, saya merespons atau diundang).

Saya tidak punya apa-apa untuk ditunjukkan, kecuali kaus oblong dengan Steve Jobs, tempat saya datang. Akibatnya, mereka percaya pada potensi saya dan mengambilnya untuk masa uji 10 ribu rubel sebulan (2,5 hari seminggu). Saya benar-benar tidak tahu, karena itu, untuk sebagian besar, ambisi atau kaus. Saya senang, karena usaha saya dihabiskan tidak sia-sia, tetapi cerita ini bisa berakhir dalam seminggu.

Setelah beberapa hari, para kolega menjelaskan bahwa level saya masih rendah untuk pekerjaan seperti itu. Saya mengerti segalanya, meskipun tidak menyenangkan melihat kepala yang gemetar, setelah pertanyaan direktur, "Yah, bagaimana dia?"

Selama liburan musim dingin, aku diberi kesempatan untuk mempelajari garis depan dengan lebih baik, lalu kembali dan mencoba tanganku lagi. Saya melakukannya, tetapi ketika saya kembali, tidak ada banyak kemajuan, dan saya mulai mengerti ke mana semuanya berjalan.

Saya tidak khawatir, karena saya mengerti bahwa karir saya baru saja dimulai, dan semuanya ada di depan, jadi saya hanya berusaha melakukan pekerjaan saya semampu saya, tidak memperhatikan pikiran orang lain. Dan seperti yang ditunjukkan oleh kenyataan, ini mungkin cukup.

Minggu depan, semua pengembang front-end datang bersama untuk memberi saya umpan balik dan memutuskan apa yang harus dilakukan dengan saya di dalam perusahaan. Yang mengejutkan saya (dan kejutan direktur), rekan-rekan saya mencatat kemajuan yang signifikan selama beberapa hari terakhir dan diputuskan untuk meninggalkan saya di perusahaan.

Setelah beberapa bulan, proyek mulai mengalami kesulitan keuangan, suasana terasa memburuk dan saya memutuskan untuk pergi. Cukup cepat, saya berhasil menemukan pekerjaan paruh waktu baru, tetapi sudah selama 20 ribu rubel di pusat bisnis yang baik, tempat saya bekerja di lantai atas, yang masih belum diketahui, Yuri Dud, yang secara berkala kami temui di lift (saya mengenalinya sejak sepak bola aktif ditonton).

Di tempat baru, saya berkembang dengan cepat dan, di samping pengaturan huruf, dengan cepat beralih ke bekerja dengan JavaScript. Anehnya, saya bisa menyelesaikan masalah apa pun, bahkan jika sebelumnya saya belum pernah mengalami hal seperti ini, yang terjadi setiap saat.

Saya terus mempertimbangkan solusi yang mungkin, membalik-balik Stack Overflow dan, setelah beberapa waktu, menemukan solusi. Mungkin tidak begitu indah, tetapi selalu berhasil, yang mengejutkan saya tidak hanya saya sendiri, tetapi juga manajer teknis.

Bagi saya, 20 ribu rubel yang pertama merupakan kekayaan. Dalam pekerjaan saya, kemajuan terlihat, tetapi meskipun demikian, saya mulai berpikir tentang desain.

Konferensi


Pada awal karir saya, konferensi yang saya hadiri secara teratur dalam 1-2 tahun institut sangat membantu saya. Perendaman dalam suasana profesional untuk spesialis pemula sangat penting, jadi saya akan menyarankan semua orang muda untuk lebih sering meninggalkan rumah.

Di salah satu konferensi, yang ternyata merupakan yang paling berbahaya, saya bertemu dengan tipe orang yang aneh dengan siapa kami berbicara sedikit. Dia mengatakan bahwa hari lain, acara UX-Rabu akan diadakan di kantor Mail.Ru Group (masih kantor lama). Gagasan mengunjungi kantor baru membuat saya tertarik, dan hari berikutnya saya pertama kali berkenalan dengan desain.

Orang pertama dari industri baru adalah Yuri Vetrov untuk saya, dan sejak itu, selama bertahun-tahun, ia telah memberikan dukungan yang signifikan tidak hanya dengan contoh, tetapi juga dengan saran yang masuk akal.

Sejak musim semi 2014, saya mulai berlatih sedikit dalam desain, dan setelah kunjungan pertama ke Dribbble Meetup, saya dengan tegas memutuskan untuk mengubah profesi saya.

Desain, buku, dan bentrok dengan kenyataan


Selain resume dari perancang, saya menerbitkan resume dari manajer proyek dan pada bulan Oktober 2014 Trood, yang terlibat dalam penciptaan sistem manajemen bisnis, menanggapinya. Tentu saja, pada saat itu saya tidak tahu apa pekerjaan manajer itu, meskipun pada kenyataannya di tempat kerja pertama saya sudah menemukan menggunakan Scrum / Agile dan istilah asing populer lainnya.

Selain manajemen produk, di antara tugas saya adalah desain halaman layanan, dan pekerjaan ini saya lakukan lebih baik. Akibatnya, saya hampir sepenuhnya dilatih kembali dari manajer ke desainer. Tapi tetap saja, pada usia 19, saya tidak memiliki keterampilan untuk merancang sistem semacam ini, jadi di suatu tempat dalam enam bulan kami memutuskan untuk bubar.

Terlepas dari kenyataan bahwa saya tidak memiliki proyek nyata untuk portofolio, saya sudah mulai menerbitkan kasus palsu di Behance, dan, yang mengejutkan saya, pesanan menarik datang kepada saya beberapa kali.

Proyek terbesar adalah sistem otomasi klinik Swiss, yang mencakup aplikasi untuk banyak platform. Kemudian saya mendapatkan penghasilan terbesar saya, pada saat itu, - 50 ribu rubel selama sebulan bekerja di malam hari dan pergi beristirahat di Spanyol.

Selain urutan yang disebutkan, proyek-proyek lain jarang muncul, dan kemudian saya duduk di buku-buku dan membacanya dalam jumlah besar. Semuanya dimulai dengan buku "Nomor 1", disumbangkan oleh pacar pertama saya. Berkat Igor Mann, saya membangun minat dalam pengembangan diri. Saya membaca 2-4 buku seminggu dan, setelah mendapatkan teori, memutuskan untuk memulai sebuah blog.


Foto-foto dari Apple Store di Barcelona

Mungkin seseorang mengingat blog saya di mana saya menulis catatan tentang pengembangan diri, yang penuh di Internet saat ini. Saya banyak membaca dan berinvestasi dalam buku 2-5 ribu per bulan. Tetangga asrama memutar jari-jari mereka di kuil (ketika mereka berpaling dari monitor), dan saya senang membaca.

Lalu saya berpikir, mengapa tidak mengundang penerbit untuk mengirimi saya buku gratis dengan imbalan ulasan. Saya memutuskan untuk mencoba: Saya menulis surat ke kotak umum Penerbit Alpina dan menerima respons positif. Akibatnya, meja saya dipenuhi dengan literatur bisnis gratis, yang sangat saya sukai.

Pada 2015, saya menerima tantangan membaca 100 buku dalam setahun dan sudah lebih cepat dari jadwal di kuartal pertama. Setelah menyelesaikan maraton ini, saya sangat muak dengan literatur bisnis sehingga untuk beberapa waktu saya tidak bisa membaca dan beristirahat. Saya terus menulis blog, tetapi tahun terakhir "studi" saya adalah di sekolah menengah dan saya harus memutuskan apa yang harus saya lakukan selanjutnya. Saya tidak ingin pergi ke magistrasi, jadi saya perlu mencari penghasilan tetap yang memungkinkan saya menyediakan perumahan bagi saya.

Saya ingat betul bagaimana saya duduk di Starbucks di Mayakovskaya dan memikirkan arah mana yang harus dipilih. Saya memutuskan untuk menulis semua hobi saya dan meletakkan 1-3 tanda rubel di sebelah masing-masing, yang menunjukkan potensi keuntungan selama satu tahun. Dan berikut ini terjadi:

Orang tua saya kesal, karena itu saya juga tidak dalam kondisi terbaik, tetapi di dalam saya tahu bahwa saya telah melakukan hal yang benar. Kemudian saya berjanji kepada mereka bahwa semuanya akan baik-baik saja. Apa yang saya hasilkan untuk hidup dan mendapatkan pekerjaan dalam hidup. Lebih dari dua tahun telah berlalu dan sepertinya saya tidak menipu mereka.

Ketika Anda mengatakan tidak, Anda perlu mendengar ya


Saya diminta untuk mengusir dari asrama seminggu kemudian, tetapi saya tidak punya uang atau bekerja. Setelah berbicara dengan kepala asrama, saya berhasil memperpanjang periode ini, tetapi hanya ada sedikit waktu.

Saya tidak menggunakan metode klasik pencarian kerja (walaupun mungkin yang paling klasik adalah berharap dan tidak melakukan apa pun) dan menulis kepada direktur Trood, Sergey Zverev. Hubungan pribadi kami cukup bagus, jadi saya memutuskan bahwa ini adalah pilihan yang baik.

Sergey menyukai gagasan ini dan dia menyerahkan kontak saya kepada perancang, yang mempercayakan saya pada tugas tes, tetapi agak cepat dia menghubungi saya lagi dan mengatakan bahwa pencarian seorang perancang tidak lagi relevan bagi mereka.

Meskipun ada penolakan, saya memutuskan untuk tidak keluar dari tugas (yang logis), tetapi untuk menyelesaikannya sebaik mungkin. Setelah menyelesaikannya, saya melewati desainer, mengirim layout ke sutradara. Sergey menyukai desain yang dihasilkan dan dia memutuskan untuk mengundang saya untuk bekerja. Sebulan kemudian, desainer itu dipecat, dan saya menggantikan desainer kepala produk.

Hanya beberapa tahun kemudian, saya menyadari gagasan bahwa Arnold Schwarzenegger menyampaikan dalam biografinya: "Ketika Anda diberitahu tidak, Anda perlu mendengar ya."

Terlepas dari kenyataan bahwa saya berhasil membuktikan diri, di tempat baru, tidak semuanya berjalan lancar. Sistem ERP adalah proyek besar dan pada saat itu pengetahuan saya sangat kurang. Saya tidak tahu bagaimana menganalisis industri, pesaing, dan menarik kesimpulan berdasarkan ini.

Akibatnya, bagian visual dari pekerjaan saya diputuskan untuk melakukan outsourcing studio desain, dan saya hanya dipercayakan dengan desain perkiraan dengan daftar komponen yang diperlukan. Itu adalah pukulan kuat bagi kompetensi saya, tetapi kedua kalinya saya tidak bisa menyerah begitu saja.

Saya memutuskan untuk membuat prototipe yang akan mempertanyakan perlunya mentransfer layout saya ke studio lain. Saya memutuskan untuk tidak mengatakan apa-apa, tetapi menunjukkan semuanya dengan perbuatan. Melihat mock-up, para pemimpin berunding, tanpa petunjuk saya, mereka menyadari bahwa tidak ada gunanya memindahkan pekerjaan ini lebih lanjut. Akibatnya, saya berhasil menyelamatkan ratusan ribu perusahaan dan membuktikan diri sebagai seorang profesional yang dapat diandalkan.

New york


Bulan-bulan berikutnya di tempat kerja, semuanya berjalan dengan baik. Saya bahkan berhasil meningkatkan gaji saya dari 80.000 menjadi 120.000 rubel per bulan, yang membuat pekerjaan semakin sulit. Saya tidak berpikir bahwa tingkat pekerjaan saya layak mendapat nilai yang sama, tetapi selama karier singkat saya, saya menyadari bahwa penghasilan tidak hanya bergantung pada keterampilan khusus, tetapi juga pada pendekatan untuk bekerja, prospek perusahaan dan kualitas pribadi.

Kemudian saya ingin membuat terobosan dalam karir saya dan saya mulai bekerja dari jam 8 pagi sampai 8 malam (minus 1,5 jam untuk pelatihan). Setelah melakukan tugas dasar saya, saya mencurahkan 1-2 jam untuk pengembangan profesional dan pribadi, dan setelah beberapa bulan hasilnya membuahkan hasil. Upaya saya mulai dirayakan dan menjelang akhir 2015, saya mendapat julukan yang menyanjung - Iron Man.

Pada bulan Desember tahun itu, kami mulai bekerja pada ide layanan baru, yang seharusnya mengotomatisasi pasar untuk pembeli misteri dan mengirim permintaan ke Starta Accelerator di New York. Setelah wawancara Skype, proyek kami diterima dan diundang selama 4 bulan di Amerika Serikat. Pada saat itu, upaya saya terpengaruh. Orang pertama dari tim tersebut terbang ke New York bersama dengan direktur umum perusahaan baru Ivan Terekhin.



Skala New York menular dan masalahnya tidak hanya pada gedung pencakar langit, tetapi juga dalam suasana kerja. Kami berada di tim orang yang ingin mencapai sesuatu dan percaya bahwa itu mungkin. Dan tidak hanya dalam kerangka New York atau AS, tetapi juga seluruh dunia. Setelah terbang melintasi lautan, saya dipenuhi dengan sensasi ini dan merasa bahwa itu tetap bersama saya sejauh ini.

Bahasa


Banyak yang bertanya di mana dan bagaimana saya belajar bahasa itu. Secara singkat - tidak ada tempat: baik di sekolah maupun di universitas. Metode yang paling efektif bagi saya adalah minat sederhana.

Sebagai pengembang, saya harus membaca Stack Overflow, saat blogging, saya sering melihat inc.com, menjadi desainer di Medium. Sangat menarik yang selalu membuat saya menerjemahkan kata-kata baru dan, setelah berbulan-bulan berlatih seperti itu, hasilnya datang.

Mentor dan pengembangan diri


Sebelum terbang ke New York, saya mendaftar untuk kursus ux-design dari AIC dan Skillbox. Formatnya menarik: pekerjaan rumah dengan memeriksa staf studio, grup tertutup di Facebook dan umpan balik yang bermanfaat. Saya memutuskan untuk aktif sejak hari-hari pertama dan secara teratur mengomentari publikasi orang lain dan membuat publikasi saya sendiri.

Kemudian saya bertemu dengan mentor kedua saya, Sergei Popkov. Pada beberapa topik, kami menemukan bahasa yang sama dan bahkan merekam wawancara video . Pengalamannya telah membantu saya lebih memahami bagaimana industri ini bekerja.

Kursus ini ternyata bermanfaat, tetapi saya selalu mengerjakan pekerjaan rumah saya dengan terburu-buru dan, hanya mendapatkan mereka untuk direvisi, saya memberikan perhatian khusus kepada mereka. Akibatnya, saya sering menerima komentar seperti "Seolah-olah orang lain melakukannya!".

Ketika tiba saatnya untuk proyek kelulusan untuk pelanggan nyata, saya memutuskan untuk bekerja keras dan melakukan pekerjaan dengan baik. Kami tidak memiliki peringkat siswa, tetapi saya tahu bahwa sebagian besar jatuh dan merasa bahwa saya bisa melakukan pekerjaan yang lebih baik. Itu adalah awal musim panas, dan kemudian untuk pertama kalinya saya mengatakan bahwa saya akan membuat proyek yang hebat dan saya akan diundang untuk merancang studio # 1.

Saya melewati tahapan analitik dan desain visual dengan cukup cepat. Saya redid pekerjaan akhir 3-4 kali, sampai saya puas. Setelah mengirim proyek kelulusan saya untuk verifikasi pada awal musim gugur, yang dilakukan oleh Sergey Popkov, saya menerima jawaban yang menyanjung dan secara harfiah beberapa jam kemudian tawaran pekerjaan.

AIC memiliki atmosfer yang hebat dan saya mengenal banyak orang dari studio ini, tetapi saya lebih tertarik (dan masih tertarik) untuk menciptakan layanan kami sendiri, jadi kami cepat memahami satu sama lain.

Presentasi kelulusan saya

Selama bulan-bulan berikutnya, berkat proyek kelulusan, portofolio, aktivitas media, dan rekomendasi, saya mulai menerima tawaran dari perusahaan-perusahaan seperti Alfa Bank, Yandex, Mail.Ru Group, VTB, kontraktor Sberbank, dan startup kecil. Sulit untuk mengatakan mengapa tepatnya, karena setiap kemajuan ditentukan oleh serangkaian faktor.

Tingkatkan Penghasilan Dan Freelance



Kantor baru Corporate BINO CX (BC Manhattan di Baumanskaya)

Saya ingin meluncurkan dan mengembangkan produk saya, yang saya lakukan pada startup BINO CX.Tetapi, meskipun pertumbuhannya bagus, karena kurangnya dana, kami harus memindahkan sebagian staf ke waktu pesta. Termasuk saya. Ini berarti saya memiliki beberapa jam ekstra untuk lepas, dan kemudian saya menghasilkan beberapa proyek yang cukup besar.

Agen desain yang merupakan bagian dari perusahaan periklanan besar meminta bantuan saya dalam mendesain layanan keuangan untuk beberapa bank besar (semua proyek ini berada di bawah NDA yang ketat). Dan dalam hal ini saya menghasilkan uang.

Proyek pertama, yang memakan waktu sekitar 5 minggu, membawa saya 210 ribu di atas gaji utama, yang turun dari 180 ribu rubel menjadi 120 ribu. Yang kedua membawa saya 90 ribu dalam 4 hari kerja, 50 ketiga.

Tetapi Anda perlu memahami bahwa itu bukan uang mudah. Selain fakta bahwa saya bekerja secara berkala di malam hari, banyak negosiasi dilakukan dalam bahasa Inggris, karena tim dari Dubai berpartisipasi dalam beberapa proyek.

Selain uang, saya dapat menguji diri saya di tingkat yang baru dan membuktikan profesionalisme saya. Sebelumnya, saya selalu bekerja sendiri, jadi tantangan baru membantu saya mendapatkan kepercayaan dan pengalaman profesional.

Selain keterampilan desain, kualitas pribadi memainkan peran yang sangat kuat dalam proyek-proyek ini: toleransi stres (terutama setelah malam tanpa tidur) dan keberanian (ketika mempresentasikan proyek kepada manajemen bank). Berkat kualitas pribadi saya, dan bukan hanya yang profesional, mereka siap berpaling kepada saya untuk proyek berulang.


Bengkel

Seperti yang dikatakan Steve Jobs: "terkadang hidup memukul kepala Anda dengan batu bata ...".

Musim gugur dan awal yang baru


"Terkadang hidup memukul kepala Anda dengan batu bata, tetapi jangan sampai kehilangan kepercayaan ..." - Steve Jobs

Proyek BINO CX kami tumbuh setiap bulan sebesar 100% dan selama setengah tahun pendapatannya mencapai sejuta rubel. Saya duduk di sebuah kafe di lantai dasar sebuah pusat bisnis terkemuka dan bekerja, terkubur di monitor. Dari sudut mataku, aku memperhatikan bagaimana para pemimpin dengan wajah yang sedikit bingung mendekatiku.

Ivan (CEO), seperti biasa, membawa saya ke titik untuk waktu yang lama, tetapi saya sudah mengerti segalanya. Kami tidak menerima investasi besar yang dijanjikan, dan karena kurangnya sumber daya perusahaan harus berpisah dengan sebagian besar staf.

Saya tidak akan mengatakan bahwa saya sangat sedih. Sebaliknya, saya merasakan keinginan untuk melakukan sesuatu yang bahkan lebih bermakna. Selain itu, saya masih memiliki saham kecil di perusahaan ini, tetapi kejutan tidak terbatas pada pekerjaan.

Saya pergi ke kantor kami dan duduk di tempat kerja saya dalam suasana hati yang baik, dan kemudian saya melihat serangkaian pesan yang menunjukkan bahwa badai serius sedang mendekat di depan pribadi. Saya berhasil melewati hanya dengan upaya ke-15 dan percakapan tidak berakhir dengan sesuatu yang baik (saya akan menghilangkan detail pribadi). Untuk pertama kalinya, dua pilar fundamental hidup saya runtuh, dan saya duduk dengan perasaan kaget baru. Tetapi, terlepas dari tragedi situasinya, untuk beberapa alasan saya memiliki pemikiran ironis di kepala saya:

"Sial, betapa kerennya itu akan terlihat dalam biografi saya!"

Dan semua karena saya yakin bahwa setiap musim gugur adalah kesempatan untuk memulai dari awal dan melakukan yang lebih baik lagi. Tidak hanya itu, saya merasa bahwa pukulan ini perlu bagi saya untuk merobek dari skenario yang biasa di mana saya terjebak. Saya membutuhkan tantangan baru dan saya menemukannya.

Setiap musim gugur adalah peluang baru


Saya tahu bahwa saya pasti akan menemukan pekerjaan dengan penghasilan 150 ribu, tetapi saya menginginkan bukan hanya pekerjaan yang stabil, tetapi juga tempat di mana saya dapat mempengaruhi sesuatu.

Untuk pertama kalinya dalam dua tahun, saya memperbarui resume saya, bertemu dengan beberapa tim desain terkenal, dan mempertimbangkan proposal untuk startup baru. Akibatnya, saya memiliki dua opsi yang sesuai dengan tujuan saya untuk melakukan sesuatu sendiri.

Kedua perusahaan menawari saya posisi kepala desainer dengan gaji 170 ribu rubel + saham di perusahaan dan bonus. Orang-orang baik di sana-sini, satu-satunya hal yang membingungkan saya adalah bahwa startup pertama, yang hidup dengan uang orang lain, menyewa kantor yang mahal. Dan startup kedua baru mulai dan jelas bahwa di dalamnya saya bisa memainkan peran yang lebih signifikan.

Dengan pemikiran inilah saya memilih perusahaan kedua dan tidak rugi. Di tempat baru, saya tidak hanya melakukan desain, tetapi, bersama dengan rekan saya, saya membuat produk dari awal untuk mengubah pasar pengiriman.

Di mana saya akan berada dalam tiga tahun jika saya terus melakukan apa yang saya lakukan sekarang?

Saya tidak ingin menjadi desainer yang hebat, membawa jutaan ke orang lain dengan gaji yang rendah juga. Itulah sebabnya saya memilih perusahaan baru yang menjanjikan di mana saya dapat memengaruhi proses apa pun dan, jika berhasil, saya akan mendapatkan uang banyak, dan jika gagal, saya akan mendapatkan pengalaman luar biasa dalam menciptakan produk saya.

Seseorang akan mengatakan bahwa ini berisiko, tetapi setelah berpikir Anda akan menyadari bahwa jauh lebih berisiko untuk tetap berada di satu tempat yang stabil selama bertahun-tahun jika Anda mengusahakan sesuatu yang lebih.

Saya percaya bahwa kaum muda adalah peluang besar untuk menetapkan tujuan yang ambisius tanpa melihat ke belakang. Adalah mungkin untuk mendapatkan perusahaan besar di setiap tahap karir Anda, dan lebih baik memperhatikan proyek berisiko sedini mungkin.
