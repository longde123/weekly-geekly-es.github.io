Prueba de la unidad de estado sólido OCZ Vector 180

Hola geektimes Hace aproximadamente tres meses hubo una publicación en la que OCZ introdujo un nuevo disco SSD: Vector 180. Quizás la diferencia clave entre Vector 180 y su predecesor Vector 150 es la disponibilidad de la función PFM . Supervisa cualquier sobretensión y elimina el "síndrome de giro del disco". ¿Qué más tiene de bueno un nuevo disco? Te lo diré debajo del corte.

OCZ Vector 180




El juego de entrega no es rico, además de la unidad, contiene un sustrato de aluminio para instalar el disco en la bahía de 3.5 ", un manual de usuario y 4 tornillos para la instalación. El manual tiene una pegatina con una clave de activación para Acronis True Image 2013. La



caja del disco de aluminio facilita la fácil disipación del calor de La altura de la unidad es de solo 7 mm, lo que permite su instalación en cualquier sistema ultrabook o de escritorio.



El controlador OCZ Barefoot 3 es exactamente el mismo que el predecesor del OCZ Vector 150. A nivel de hardware, el Vector 180 presenta una fuente de alimentación rediseñada. El objetivo de la tecnología PFM es que, a intervalos cortos, se toman instantáneas del estado de la tabla de retransmisión y se guardan en una matriz de memoria flash en lugar de con datos regulares. Esto no protege los datos que están "en vuelo" cuando se apaga la alimentación, pero le permitirá mantener la unidad en funcionamiento con una pérdida repentina de voltaje.



El OCZ Vector 180 utiliza la memoria Toshiba MLC, fabricada con la tecnología de proceso de 19 nm. Unos 32 gigabytes de memoria están reservados para el área oculta reservada, el usuario tiene acceso al volumen de 223.57 gigabytes. Utilidad patentada OCZ SSD Guru



le permite controlar la capacidad, la temperatura y el estado de la unidad. Además, puede controlar el exceso de aprovisionamiento, actualizar el firmware y realizar muchas otras funciones útiles, como forzar TRIM o ejecutar el comando de borrado seguro.

Metodología de prueba


La metodología de prueba asume las siguientes condiciones: el sistema operativo Windows 8.1 Professional funciona adecuadamente con los SSD modernos; esto significa que, tanto durante las pruebas como en la vida cotidiana, el equipo TRIM trabaja constantemente y se usa activamente. Para evitar el efecto del "nuevo SSD", antes de realizar las pruebas, se escribe un volumen de datos igual al volumen SSD duplicado en cada disco.



Entre cada ejecución de prueba, se realiza una pausa para que el comando TRIM funcione correctamente.
