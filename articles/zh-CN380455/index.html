黎明探测器从4,400公里的距离拍摄了谷神星上的白色斑点的照片


照片：IDA / DLR / MPS / UCLA / JPL-Caltech / NASA

目前正在研究小行星谷神星的 Dawn行星际探测器拍摄了质量最高的矮行星白点的照片。这些照片是今年6月6日在距小行星4.4千公里的地方拍摄的。

这些景点本身位于直径约90公里的环形山中。尽管照片的质量越来越高，而且探头正接近谷神星，但没人能猜出这些斑点的性质。基本理论是斑点是水冰到地表的出口。

此外，还有其他一些理论，其作者认为斑点是盐向表面的出口或冰冷狂热的表现。自然，会有更多的外来理论，但是到目前为止，还没有人考虑到它们。


照片：IDA / DLR / MPS / UCLA / JPL-Caltech / NASA

“这种配置的亮点使谷神星成为了一个独特的物体，与我们在太阳系中看到的一切不同。一组科学家现在正在努力弄清斑点的性质。冰反射光是一个基本理论，但研究小组继续考虑其他理论，包括斑点的盐分来源。加利福尼亚大学洛杉矶分校的克里斯·拉塞尔评论说，越接近塞雷斯，就越能发现这种神秘现象的本质。


照片：IDA / DLR / MPS / UCLA / JPL-Caltech / NASA

除了斑点之外，探测器还拍摄了谷神星的其他区域，所有这些图像现在正在由专家进行研究。

不久前，美国宇航局的专家从单个照片中整理了一段视频：
