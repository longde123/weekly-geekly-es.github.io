Internet Archive mengunggah 1500+ program untuk Windows 3.1, Anda dapat bekerja secara langsung di browser



Beberapa hari yang lalu, Internet Archive mengunggah lusinan virus lama ke Web yang masih berjalan di bawah DOS, menciptakan Malware Museum . "Museum" memiliki sekitar 80 virus klasik yang banyak ditemui pengguna saat itu.

Sekarang ada museum lain - kali ini program lama yang bekerja di bawah Windows 3.1. Para pengembang memposting aplikasi paling populer pada waktu itu, termasuk alat dan game yang berfungsi murni. Cari semua ini adalah mungkin di bagian baru dari yang Windows 3.x Showcase . Secara total, 1.523 program diposting di sini, jumlah mereka terus meningkat. Di browser, semua ini bekerja berkat emulator JavaScript Windows 3.1.

Para pengembang mengatakan bahwa beberapa bagian dari proyek ini selalu ada di tangan mereka. Hampir semua perangkat lunak disimpan oleh perwakilan tim proyek di berbagai media. Selain itu, beberapa program dikirim dan penggemar yang mendengar tentang persiapan bagian baru di Internet Archive.

Setelah jumlah program yang dikumpulkan mencapai angka tertentu, para pengembang mulai membuat emulator Win 3.1. Ini ternyata menjadi tugas yang sulit, tetapi sukarelawan mengatasinya.



Plus, ada emulasi pratinjau Windows 95 - mereka yang menemukan semua ini pada waktunya akan senang melihat sistem operasi "lama baru" lagi.





Secara alami, banyak perangkat lunak yang memiliki hak cipta, dan pengembang harus mempertimbangkan hal ini. Oleh karena itu, beberapa program digantikan oleh rekanan gratis - dan beberapa harus disisihkan untuk waktu yang lama karena alasan ini.









Secara umum, waktu untuk mulai mengingat masa lalu.
