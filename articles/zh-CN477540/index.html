人工智能与人们进行了辩论，并坚信这并不危险



剑桥大学就人工智能的潜在危害举行了两个小时的辩论。 保护者是IBM Project Debater。 通过运用自己的观点并打破对手的观点，她设法向听众证明，最终，人工智能将给社会带来的好处多于伤害。


讨论在拥有200年历史的辩论协会剑桥联盟（Cambridge Union）进行。 它接待了数百名世界领导人和著名学者，就政治，历史或文化进行辩论。 但是，这样的客人第一次出现在社会上：一个两米长的黑色面板用机器人的女性声音说话。


Project Debater模型接受了30,000多个论点的培训，并获得了审阅者的可信度评估。 因此，她学会了识别哪些论点比其他论点更有效，以及如何建立有力的证据。 IBM项目经理Noam Slonim表示，现在需要进一步研究的想法之一是分解这种模型，并理解为什么人们发现某些论点比其他论点更具说服力。


现在，Project Debater存储了一个包含4亿篇文章的数据库，他可以用自己的话来解释这些文章，讨论大约100个不同的主题。 在辩论之前，该机器查看了活动一周前发送给IBM的1,100多个论点和AI问题。 她在“ For”类别中确定了570个，在“ Against”类别中确定了511个。 一些论点被认为是无关紧要的。




机器人立即对两个位置讲话，另外双方各有两个人帮助他，补充了论点或提出了问题。 该机器使用语音识别AI来聆听和分析对手的论点。 每次都花几分钟，但是Project Debater实际上直接回答了提出的问题。


具有讽刺意味的是这台机器的开幕词。 她从“反对”方面开始：


人工智能会严重危害人类。 在新情况下，他将无法做出道德上正确的决定。 因为道德的存在是人的独特特征。

机器人说话清晰而始终如一，但仍然发生了一些错误。 她几次重复了同样的论点-例如，谈论AI执行无聊和重复性任务的能力。 在某些情况下，该机器根本没有为支持其理论而争论，就好像将其话语视为事实一样。


尽管如此，在描述AI的潜在好处时，这台机器似乎感到困惑，它代表着哪一方面，并​​开始自相矛盾：


人工智能将创造新的工作机会，为人们提供更多的创新活动，提高工作效率。 在患者护理或儿童教育领域，这里将不再需要人们。

结果，“ For AI”派对获胜。 在观众的300人中，有51.22％的人投了赞成票。


在此之前，2月，Debater项目已经与人们发生了纠纷-与记录保持者和辩论者Harish Natardzhan竞争，他被认为是辩论领域世界上最好的人之一，也是2016年世界杯决赛选手。 AI断开了与互联网的连接，花了15分钟进行准备，很多注意力都集中在驳斥对手的论点上。 然后汽车被打败了。 听众认为，尽管AI中信息的准确性和丰富性更高，但该人的说话更加火热，而且总体上听起来更令人信服。




在Watson计算机击败Jeopardy测验游戏（美国“自己的游戏”）的参与者之后，IBM团队开始开发Project Debater。 现在该系统已有六年历史了。 根据他们的说法，在过去的六个月中，这款车表现出了非常好的增长，现在可以与任何人讨论。


Project Debater是第一个可以使人们在最棘手的问题上说服的AI系统。 她摘录了大量文本，针对给定主题进行了结构合理的演讲，清楚地提交并反驳了对手的论点。 最终，这种机器将帮助人们推理，提供令人信服的，基于证据的论据并摒弃情绪。

IBM计划使用人工智能来收集大量人员的意见，例如在会议或政治集会上。 机器快速确定该人或那个人领导哪一方的论点。 政府和大公司将能够在进行重要决策之前将其用于民意调查。


这项技术可以帮助我们在经理与将受其决策影响的人们之间建立有趣而有效的沟通渠道。 最后，我们有机会以开放的心态看待这个问题，没有个人的依恋。

IBM还希望这样的AI能够帮助她创建更强大的虚拟数字助理。
