Pangsa Windows di desktop turun di bawah 90 persen


Windows 7 masih menjadi pemimpin, tetapi bukan mayoritas yang luar biasa

Sembilan bulan setelah rilis Windows 10, pangsa Windows 7 akhirnya turun di bawah 50 persen, dan Windows XP mencapai batas indikator yang jelas. Meskipun ini adalah berita baik untuk Microsoft, April bukan bulan terbaik untuk Windows, yang, menurut data Net Applications terbaru , untuk pertama kalinya menempati kurang dari 90 persen pasar.

Mari kita menganalisis angka-angka ini. Pertama, Anda harus mempertimbangkan keinginan Microsoft untuk mendapatkan 1 miliar perangkat yang menjalankan Windows 10 "dalam dua hingga tiga tahun." Windows 10 diinstal pada lebih dari 75 juta komputer dalam empat minggu pertama, 110 juta perangkat dalam 10 minggu, 200 juta dalam enam bulan, dan 270 juta dalam delapan bulan, venturebeat.com melaporkan .

Pengambilalihan ini telah menyebabkan pertumbuhan yang nyata bagi Microsoft, tetapi pertumbuhan ini melambat. Pangsa pasar Windows 10 adalah 14,15 persen pada Maret dan ditambahkan hanya 0,20 persen pada April, mencapai 14,35 persen.


Aplikasi Bersih Web Analytics

Windows 8 entah bagaimana naik 0,50 poin persentase, mencapai 2,95 persen, sedangkan Windows 8.1 tumbuh 0,40 poin menjadi 9,16 persen. Bersama-sama mereka menduduki 12,11 persen pasar pada akhir April. Diperkirakan bahwa bagian dari duo ini akan jatuh di bawah 10 persen pada akhir 2016.

Sebelum dirilisnya sistem operasi Windows 10, Windows 7 menduduki 60 persen pangsa pasar pada Juni 2015, dari awal 2016 angkanya turun menjadi 52,47 persen, dan saat ini porsinya 48,27 persen.

Meskipun pertumbuhan upgrade ke Windows 10 tidak terhindarkan, Windows 7 kemungkinan akan mempertahankan gelar "OS paling populer" untuk waktu yang lama. Menurut indikator ini, Windows 7 mengambil alih Windows XP pada September 2012.

Windows Vista menyumbang 1,34 persen, sedangkan Windows XP turun 1,24 poin menjadi 9,66 persen. Penurunan pangsa XP telah menunggu lama, mengingat bahwa Microsoft berhenti mendukung sistem operasi "kuno" pada bulan April 2014.

Secara umum, pangsa keluarga OS Windows turun 1,22 poin menjadi 89,23 persen pada April 2016. Mac OS X, sementara itu, meningkatkan bagiannya sebesar 1,43 poin menjadi 9,20 persen. Pangsa Linux adalah 1,56 persen.


Pasar OS global, data untuk April 2016

Windows tidak pernah turun di bawah angka 90 persen, setidaknya menurut Net Applications (data untuk November 2007, ketika Windows menyumbang 95,89 persen dari pasar). Meskipun Windows telah kehilangan bagiannya dengan sangat lambat selama bertahun-tahun, perlu dicatat bahwa titik drop tidak pernah melebihi 1 persen. Windows diperkirakan akan "mengembalikan" 90 persen bulan ini.

Aplikasi Net menggunakan data dari 160 juta pengunjung unik setiap bulan, pemantauan mencakup sekitar 40 ribu situs web.
