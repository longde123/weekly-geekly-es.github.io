iOS CSS kematian


Sebuah kode telah diterbitkan di github yang mengarah untuk memaksa me-restart perangkat iOS (11/12 GM) ketika mengunjungi halaman html. Juga menyebabkan Mac OS High Sierra / Mojave membeku saat menggunakan Safari.


Kode yang mengarah ke paksa restart adalah kode-html dengan sejumlah besar div bersarang dan fungsi blur latar belakang "gila": 


... div { backdrop-filter: blur(10px); -webkit-backdrop-filter: blur(10px); width:10000px; height:10000px; } </style> </head> <body> <div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div><div> ...

Juga diterbitkan html-halaman PoC yang mengimplementasikan bug ini (pengguna Safari menanggung risiko dan risiko sendiri).



Ada asumsi bahwa bug tersebut merayap pada tingkat yang lebih rendah dari webkit dan dapat menyebabkan konsekuensi yang lebih serius. Juga, penggunaan bug ini mungkin berlaku di perusahaan sosioteknik dan lelucon praktis yang konyol, jadi saya menyarankan Anda untuk sangat curiga terhadap semua tautan, terutama karena di iOS semua browser, pada kenyataannya, merupakan tambahan untuk Safari.
