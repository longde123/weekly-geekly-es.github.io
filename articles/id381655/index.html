Nintendo menuntut untuk menghapus emulator Game Boy Advance JavaScript dari github



Perusahaan Nintendo menuntut untuk menghapus dari github Emulator Game Boy Advance, yang ditulis dalam JavaScript. Rupanya, alasannya adalah hadirnya beberapa game asli dari konsol game lengkap dengan emulator gambar. Saat ini, halaman emulator tidak lagi tersedia. Permintaan itu dikirim oleh perusahaan minggu lalu.

Sejarah gim portabel elektronik Game Boy dimulai pada tahun 1989. Itu menjadi sistem permainan populer pertama, dan pesaing tidak dapat mengulangi keberhasilannya. Nintendo secara berkala merilis versi baru dari permainan - pada tahun 1996 ada versi yang dikurangi dari Game Boy Pocket, pada tahun 1998 - Game Boy Color, versi warna pertama dari permainan. Dan pada tahun 2001, Game Boy Advance muncul, game terakhir yang memiliki popularitas luas.


Game Boy Advance

Meskipun awalan terakhir dalam seri ini, Game Boy Micro, dirilis pada 2005, banyak orang masih ingat dengan nostalgia mainan favorit mereka dan menikmati kesempatan untuk memainkan game yang akrab di emulator. Dan, meskipun halaman emulator dengan game dalam kit ditutup, emulator tanpa game dapat ditemukan di alamat yang berbeda . Selain itu, github memiliki daftar emulator yang baik dari berbagai platform yang ditulis dalam JavaScript.

Ini bukan pertama kalinya Nintendo menyerang emulator Game Boy - tahun lalu nasib yang sama menimpa aplikasi mobile yang populer untuk iOS, GBA4iOS.

Anak-anak USSR berkenalan dengan Nintendo secara tidak langsung - pada awalnya kami mendapatkan game "Ya, tunggu dulu!", "Rahasia lautan", "Koki ceria", dan yang lainnya, yang ternyata adalah klon dari game Game & Watch . Menurut legenda, perancang yang menciptakan permainan ini melihat bagaimana di kereta seorang pengusaha mencoba menghibur diri dari kebosanan dengan menekan tombol pada kalkulator.


G&W game pertama bernama "Ball", 1980.

Secara resmi, NES tidak pernah dijual secara resmi di wilayah bekas Uni Soviet, dan pasar ini tampaknya tidak menarik bagi produsen dunia game dan konsol.
