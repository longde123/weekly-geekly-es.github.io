WebTotem o cómo queremos que Internet sea más seguro

Servicio gratuito para monitorear y proteger sitios web.



Idea


En 2017, nuestro equipo CARKA comenzó a desarrollar una herramienta para monitorear todo el espacio cibernético en la zona de dominio nacional .KZ, y esto fue alrededor de 140,000 sitios web.

La tarea es compleja: era necesario verificar rápidamente cada sitio para detectar signos de piratería y virus en el sitio y mostrar de manera conveniente el tablero del estado de todo el Kaznet.

Así nació WebTotem : un servicio para monitorear y proteger sitios web.





Ofrecemos nuestra solución a todos los propietarios de sitios web de forma gratuita. Ganamos dinero en soluciones para clientes corporativos que necesitan monitorear una gran cantidad de recursos (CERT, proveedores de alojamiento, etc.) y en servicios adicionales para propietarios de sitios web (pruebas de carga, auditorías, tercerización y ayuda con el tratamiento de virus, etc.).

Producto de WebTotem


Durante 2 años, nuestro sistema ha crecido hasta convertirse en una plataforma completa con 9 módulos que son interesantes para el propietario ordinario del sitio.

Simplemente se registra en el sistema y agrega su sitio para el monitoreo. Si desea instalar módulos de protección, descargue el script para el sitio (en este momento, los scripts para sitios php están disponibles en su cuenta, el módulo para nginx está bajo prueba)



Módulos de monitoreo:

Monitoreo de disponibilidad durante el día / semana / mes;
Verificar la validez y validez del certificado SSL instalado;
Recopilación de información sobre la plataforma en la que se ejecuta el sitio (CMS, versión del sistema operativo del servidor, etc.);
Comprobación de una desfiguración en el sitio;
Verificación del sitio en busca de bases de datos de reputación (VirusTotal, Spamhaus, etc.)
Recopilación de información sobre el dominio y la fecha de vencimiento de su registro;
Escáner de puerto abierto;

Módulos de protección:

Antivirus: busque shells de hackers y puertas traseras en el sitio
Firewall de aplicaciones web: un módulo diseñado para detectar y bloquear ataques a aplicaciones web

Aplicación móvil de WebTotem


Y, por supuesto, queremos alardear de nuestra aplicación móvil, que le permite monitorear y monitorear la seguridad de su sitio a través del teléfono =)
