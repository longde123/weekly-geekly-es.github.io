Menurut Microsoft, proses peningkatan ke Windows 10 sepenuhnya dikendalikan oleh pengguna



Terlepas dari kemarahan pengguna tentang fakta bahwa komputer mereka diperbarui ke Windows 10, segera setelah mereka berpaling, Microsoft mengatakan bahwa mekanisme pembaruan Win10 sepenuhnya dikendalikan oleh pengguna.

Microsoft mengeluarkan pernyataan berikut tentang pembaruan "tidak terencana":

"Kami ingin Windows 7 dan Windows 8.1 untuk ditingkatkan ke Windows 10 semudah mungkin. Pengguna dapat menjadwalkan waktu pembaruan, yang berarti mereka memiliki kendali penuh atas perangkat mereka. Mungkin juga untuk tidak menginstal Windows 10 dengan menghapus pembaruan dari Pembaruan Windows (WU) dengan mengubah pengaturan, " kata hothardware.com.

Reporter Kotaku Patrick Klipik adalah salah satu dari banyak yang , di luar kehendak mereka, Windows 7 ditingkatkan menjadi Windows 10 di luar kemauannya :

â€œSaya menggunakan Windows 7 selama beberapa tahun, menghindari menginstal Windows 8, karena semua orang mengatakan kepada saya bahwa itu hanya masalah. Windows 7 adalah OS yang nyaman. Saya tahu apa dan di mana letaknya. Namun demikian, saya mengerti bahwa pembaruan itu penting dan perlu, terutama dalam hal keamanan. Saya, sebagai aturan, adalah orang yang "menggulung" pembaruan sesegera mungkin. Tetapi komputer adalah bagian besar dari hidup saya. Beberapa masalah PC sulit diperbaiki dan sistem operasi baru bisa menjadi masalah.

Minggu lalu, saya berpikir untuk berhenti menjadi "dinosaurus kuno" dan meningkatkan ke Windows 10. Yang tidak saya duga adalah bangun pada Kamis pagi dan mengetahui bahwa Windows merawatnya untuk saya (dengan hilangnya semua pengaturan saya). Mungkin saya lupa menunda pembaruan, mungkin saya tidak sengaja menjadwalkannya dengan mengklik tombol yang salah. Bagaimanapun, saya pasti tidak berniat untuk menginstal Windows 10. "

Reddit masih dipenuhi dengan pesan dari pengguna yang marah. Sebagai contoh, MalekuaMan menulis: "Saya akan meninggalkan rumah selama dua hari dan setelah kembali, saya tahu bahwa saya sekarang memiliki Windows 10. Sialan seperti apa." Pengguna lain dengan nama panggilan rtn1797 juga menderita dari "kekhawatiran" Microsoft: "Mesin Win7 saya ditingkatkan ke Win10 tanpa izin saya sekitar dua hari yang lalu. Saya memutar kembali pembaruan, tetapi sekarang ada petunjuk yang selalu terbuka, meminta saya untuk memilih waktu untuk meningkatkan ke Windows 10 ... WTF ... bagaimana saya bisa menyingkirkan ini? "

Tidak ada yang mengatakan bahwa Microsoft mencoba mengecilkan masalah ini, tetapi mengapa komputer semua orang ini ditingkatkan ke Windows 10 pada saat yang sama dalam keadaan yang sama? Microsoft harus menerbitkan pernyataan resmi yang akan menjelaskan masalah ini.
