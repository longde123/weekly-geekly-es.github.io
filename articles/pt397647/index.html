EQ-Radio: um sistema para determinar as emoções humanas usando um sinal sem fio



Descobrir o que a outra pessoa está sentindo no momento pode ser extremamente difícil. Mais ou menos com precisão, determinar o humor do interlocutor, se ele tentar esconder suas emoções, pode apenas um psicólogo profissional ou uma pessoa com vasta experiência em comunicação. E mesmo neste caso, a probabilidade de erro é alta. Ainda existem vários tipos de testes, mas trabalhar com eles exige tempo e o consentimento de outra pessoa.

Agora há outra maneira, rápida e eficaz. Cientistas do MIT (Instituto de Tecnologia de Massachusetts, Instituto de Tecnologia de Massachusetts, MIT) desenvolveramum dispositivo que pode detectar emoções humanas usando um sinal sem fio. São analisados ​​os batimentos cardíacos, a frequência respiratória e alguns outros processos que ocorrem no corpo humano. Depois disso, o sistema de computador analisa todos os dados recebidos e indica quais emoções uma pessoa experimenta.

"Nosso trabalho prova que os sinais sem fio podem conter informações sobre o humor de uma pessoa, capturando as menores alterações invisíveis a olho nu", diz a gerente de projeto Dina Katabi. "Acreditamos que nossos resultados podem ser a pedra angular de futuras tecnologias que podem detectar depressão, desespero e outras emoções humanas negativas".



Os desenvolvedores nomearam seu dispositivo EQ-Radio. Segundo eles, a precisão do dispositivo é de 87%. O dispositivo envia sinais sem fio, que são refletidos no corpo humano e são novamente capturados pelo sistema. As ondas de rádio refletidas são analisadas por software especializado projetado especificamente para este projeto. O algoritmo captura as menores alterações nos batimentos cardíacos de uma pessoa. Em seguida, os dados obtidos são comparados com as informações carregadas no sistema e uma certa conclusão é feita.

A precisão acima é alcançada se o dispositivo trabalha com o assunto por um longo tempo. Por exemplo, isso pode ser uma análise das emoções de uma pessoa enquanto assiste a cinco vídeos de dois minutos. Se você tentar determinar o humor de uma pessoa imediatamente, sem passar nos testes preparatórios, a precisão será reduzida para 70%.

Até agora, o dispositivo pode detectar apenas algumas emoções, incluindo alegria, raiva, tristeza, satisfação.

"Conhecendo várias características do pulso de uma pessoa em vários estados emocionais, podemos analisar facilmente os dados e determinar as emoções atuais de uma pessoa", diz Mingming Ziao, participante do projeto.

Para testar a integridade de seu sistema, os desenvolvedores primeiro testaram eles mesmos. Eles tocaram várias músicas ou assistiram a vídeos que tiveram um forte efeito emocional e depois ligaram o EQ-Radio.

O desafio era "ensinar" o testador emocional a distinguir entre os batimentos cardíacos e os movimentos respiratórios de uma pessoa. Agora o EQ-Radio distingue facilmente entre os vários processos que ocorrem no corpo. Os criadores do sistema dizem que o EQ-Radio determina os tempos de abertura e fechamento das válvulas dos ventrículos do coração. Devido a isso, as alterações no batimento cardíaco podem ser determinadas com muita precisão (a probabilidade de erro é de apenas 0,3%). E, de acordo com esses dados, o sistema cria uma imagem do estado emocional de uma pessoa.

Você pode determinar não apenas emoções, o EQ-Radio também deve ser usado para monitorar a saúde humana. "Podemos identificar doenças humanas como arritmia, além de outros problemas", disse Fadel Adib, outro membro da equipe do projeto EQ-Radio.

Além disso, para isso, você não pode usar métodos tradicionais de exame, todas as medições necessárias podem ser realizadas "por via aérea". Obviamente, a princípio, o dispositivo deve ser considerado um auxiliar, e não a ferramenta principal. Mas, com o tempo, os cientistas prometem finalizar o EQ-Radio para que tudo isso possa ser usado como um sistema de diagnóstico completo.



Além dos médicos, o EQ-Radio pode ser útil para cineastas ou profissionais de marketing. Trabalhando com o EQ-Radio, você pode determinar que tipo de impacto emocional uma pessoa tem em um filme ou em algum produto. O monitoramento deve ser realizado em tempo real - o sistema funciona muito rapidamente.

A técnica é única?


Até onde sabemos, até agora ninguém tentou determinar o humor de uma pessoa pelas distorções do sinal sem fio. Existem projetos que permitem determinar as emoções das pessoas pela expressão de seus rostos (até os micro-movimentos dos músculos faciais são capturados). Esse projeto, chamado de "API da emoção" , foi desenvolvido pela Microsoft.

A API do Emotion pode determinar o humor com bastante precisão, mas, como já mencionado, é necessário o contato visual do sistema com o sujeito. E, neste caso, a falta de emoções, um humor neutro, é determinada com mais facilidade do que todos os outros sentimentos. No caso de trabalhar com o EQ-Radio, o contato visual não é necessário e o sistema pode determinar uma ampla gama de emoções sem problemas.

Os cientistas conduzirão uma apresentação detalhada dos resultados de seu trabalho na conferência MobiCom (Conferência Internacional sobre Computação Móvel e Redes ).
