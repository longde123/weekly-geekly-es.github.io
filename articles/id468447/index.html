Laragon - WAMP dengan domain lokal otomatis

Bahkan aneh bahwa tentang Laragon tidak ada publikasi tunggal tentang HabrÃ©. Saya ingin mengisi celah ini dengan sangat singkat, karena alat ini layak mendapatkan popularitas di kalangan audiens target pengembang web yang mengkode untuk Windows.



Laragon adalah WAMP yang sederhana dan ringkas (Apache + MySQL + PHP untuk Windows) dalam banyak hal mirip dengan rekan-rekannya yang lebih terkenal, seperti XAMPP , OpenServer , Denwer dll. Tapi, dengan sedikit twist:

Ketika Anda membuat folder your-test-project di direktori <laragon_root>\www\ , isi folder ini secara otomatis dapat diakses dari browser di http://your-test-project.test dan template yang digunakan domain juga dapat disesuaikan.



root dokumen untuk domain lokal juga dikonfigurasi (jika perlu) dalam beberapa klik dan baris konfigurasi:

Dalam folder <laragon_root>\etc\apache2\sites-enabled\ ada konfigurasi untuk setiap situs yang dibuat secara otomatis. Ubah jalur di isi file, hapus awalan auto. dari nama konfigurasi auto. dan voila!



Rilis ini dapat diunduh dalam tiga level trim:

  • Laragon Full : Apache 2.4, Nginx, MySQL 5.7, PHP 7.2, Redis, Memcached, Node.js 11, npm, benang, git, ...
  • Laragon Lite : Jangan memasukkan Node.js 11, npm, benang, git tetapi Anda dapat menambahkannya dengan mudah
  • Laragon Portable : PHP 5.4, MySQL 5.1 - Baik untuk memulai dengan PHP, maka Anda dapat menambahkan versi PHP / MySQL yang lebih baru dengan mudah nanti

Saya menginstal Lite sendiri, karena saya sudah memiliki simpul dan lainnya dalam sistem - Saya hanya perlu WAMP sederhana untuk menggantikan XAMPP.

Itu mudah. Juga mudah untuk memulai, mengkonfigurasi, dan bekerja. Satu-satunya hal: di luar kotak tidak ada phpMyAdmin. Pertanyaannya dipecahkan sebagai berikut:

1. Unduh distribusinya dari www.phpmyadmin.net
2. Buka dan ganti nama folder file di phpMyAdmin
3. Pindahkan folder ini ke <laragon_root>\etc\apps\



Itu saja. Sekarang phpMyAdmin Anda tersedia di http://localhost/phpmyadmin/

Bagi saya - pekerja keras yang sangat nyaman dan sederhana. Saya sarankan mencoba, saya suka bekerja dengannya.
