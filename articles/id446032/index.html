Kelas atas dilakukan sesuai dengan skenario berikut:

  • Jika metode yang dipilih mendukung pekerjaan langsung dari editor video Adobe Premiere, maka semua pekerjaan dilakukan dalam editor ini, setelah itu hasilnya diekspor dalam FFV1 codec dengan parameter GOP 1 untuk menghindari kerugian dan menggunakan kompresi gambar yang dapat dibalik secara eksklusif.
  • Jika metode yang dipilih tidak mendukung pekerjaan dari editor video, maka video tersebut didekomposisi menjadi bingkai terpisah dalam format PNG menggunakan file bat sederhana:

    File BAT untuk menyimpan masing-masing frame ke folder frames 
    wmic process where name="cmd.exe" CALL setpriority 16384 @echo off :hugly if "%~1" EQU "" goto mugly ffmpeg -probesize 1000M -i "%~1" -vsync vfr frames\image-%%03d.png shift goto hugly :mugly pause

    Untuk menguraikan video menjadi bingkai, cukup tempatkan file ini di folder ke ffmpeg.exe dan buat folder bingkai, setelah itu biasa menyeret file video langsung ke file bat.

    Setelah itu, frame yang diproses dikirim ke FFMPEG untuk pencampuran, dan diekspor dengan pengaturan yang persis sama seperti pada kasus pertama.

    File BAT untuk mengubah masing-masing frame menjadi file, tanpa suara 
     ffmpeg -framerate 24 -i image-%%03d.png -vcodec ffv1 -pix_fmt yuv420p -level 3 -g 1 -r 24 photoshop.mkv

    Parameter -framerate dan -r bertanggung jawab untuk mengatur kecepatan frame video, keduanya harus ditentukan.

Dalam kedua kasus, outputnya adalah file MKV dengan codec FFV1, berukuran sekitar 700 megabyte untuk setiap 10 detik video.

Jenis video berikut ini diproses:

6. Seorang pemula yang ingin mencoba program kelas atas dalam praktek

... Saya dapat merekomendasikan Instant 4K sebagai yang paling mudah digunakan.

Anda cukup menjalankan program, mengimpor file Anda ke dalamnya, membuat urutan "Sequence from Clip" darinya, buka pengaturannya (Sequence Settings) dan ubah resolusi sesuai keinginan Anda. Setelah itu, seret efek Instant 4K ke trek video Anda, sesuaikan resolusi dan parameter tambahan, dan Anda dapat mulai mengekspor.

Sayangnya, untuk menggunakan program ini, Anda perlu membeli atau menemukan Adobe Premiere, serta plugin itu sendiri. Untuk membeli semua ini adalah investasi yang sangat serius, dan saya tidak bisa menyarankan untuk mencari program "sembuh" di Internet.


Adobe Premiere tidak puas dengan seleraku
Saya ingin menarik perhatian Anda pada fakta bahwa Premiere tidak berfungsi dengan semua format video, dan sebagian besar film dari Internet akan bersumpah dengan ketidakjujuran yang baik. Untuk kasus ini, saya hanya dapat menyarankan untuk mengkonversi film ke sesuatu seperti ProRes menggunakan ffmpeg:
 ffmpeg -hide_banner -probesize 1000M -i file.mkv -pix_fmt yuv420p -c:v prores_ks -c:a aac -b:a 128k file.mov

Baris berikutnya adalah waifu2x : lay out video menjadi bingkai menggunakan file BAT dari bagian "Mempersiapkan bahan untuk pengujian" , buka folder di waifu2x, pilih parameter (naik ke resolusi, pengurangan noise) dan mulai ekspor. Pada akhirnya, Anda hanya perlu membawa frame dan suara ke dalam video yang sudah selesai. Anda mungkin memiliki beberapa masalah dengan konfigurasi, seperti yang sudah saya tulis, saya pribadi harus mencari cuDNN, selain itu, waifu2x memerlukan kartu video yang cukup kuat.

Jika karena alasan tertentu Anda tidak dapat mengkonfigurasi waifu2x, atau Anda tidak memiliki kekuatan GPU yang cukup, lihat Photoshop . Dalam artikel ini ada petunjuk cukup baik untuk penggunaannya.

Jika Anda puas dengan percobaan gambar, atau Anda memiliki banyak waktu ekstra, gunakan uji coba Topaz : ini akan bekerja dengan semua fungsi selama 30 hari. Anda hanya punya waktu untuk menambah film favorit Anda sebelum akhir masa percobaan (tetapi ini tidak akurat). Persyaratan untuk kartu grafis waifu2x yang kuat berlaku di sini. Selain itu, ketika menggunakan Topaz atau waifu2x, Anda mungkin memiliki masalah bekerja dengan program lain, bahkan paling sederhana, karena kedua program berusaha untuk menduduki GPU sebesar 100%, membuat antarmuka sisanya sangat lambat. Prinsip operasi sama dengan waifu2x.

Dalam tiga kasus terakhir, saya sarankan Anda memisahkan trek audio dari file video asli terlebih dahulu, dengan cara apa pun yang nyaman bagi Anda. Jika ini adalah video sederhana dari Youtube atau film dengan satu lagu - yah, file BAT satu baris paling sederhana akan dilakukan. Jika Anda “beruntung” untuk bereksperimen dengan film dengan trek 5.1 dan beberapa terjemahan, maka kasus seperti itu jauh melampaui ruang lingkup artikel ini. Tetapi dia yang mencari akan selalu menemukan.

Contoh Skrip Ekstraksi Suara 
 ffmpeg -i input-video.avi -vn -acodec copy output-audio.aac

Setelah menerima audio dalam satu file dan pada akhir pemrosesan frame dengan upscaler, Anda harus menggabungkan suara dan video ke dalam satu file.

Untuk pemula, saya sarankan menggunakan skrip dari artikel tentang Photoshop: 
 ffmpeg -framerate 24 -i image-%%03d.png -i output-audio.aac -pix_fmt yuv420p -vcodec libx264 -preset veryslow -crf 15 -c:a aac -b:a 128k -r 24 test_4K.mp4

Selebihnya harus mempelajari parameter dan pengaturan H.265 (ukuran lebih kecil dan kualitas lebih tinggi) dan H.264 (lebih cepat).
