Agregue automáticamente fuentes JSON a REFL.ME para generar mensajes y alertas

Después de publicar el artículo " Mensajes y alertas en Android a través de JSON " aquí y entrar en las 5 principales noticias del día en Hacker News, llegaron muchas cartas de comentarios. Gracias a esto, se han agregado nuevas funciones útiles.

Una de las nuevas características que se agregó a pedido de los usuarios fue la capacidad de agregar y eliminar fuentes de datos de forma centralizada para generar mensajes y notificaciones en el dispositivo del usuario.

Doy un ejemplo simple en formato JSON que te permitirá usar esta función: 

{ "refl.me":true, "hub":true, "sources":[{"url":"https:\/\/refl.me\/\/test"},{"url":"https:\/\/refl.me\/\/test_secret"}] }

"Refl.me" - indica necesariamente cierto.
"Hub" - indica necesariamente cierto.
"Fuentes": una lista de fuentes para procesar. La creación, actualización y eliminación se realizarán automáticamente de acuerdo con esta lista.

Todo funciona de manera similar a una fuente normal, pero no se procesa por mensaje, sino por fuentes. Según las direcciones recibidas de "fuentes", la aplicación forma una lista real de fuentes de datos para generar mensajes y notificaciones.
