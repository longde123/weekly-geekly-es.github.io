LoRaWAN如何帮助构建现代物联网



LoRaWAN是一项在物联网解决方案领域迅速普及的技术。 同时，对于许多客户而言，它仍然缺乏了解和异国情调，因此围绕它存在着许多神话和误解。 2018年，俄罗斯通过了关于使用LoRaWAN频率的立法修正案，扩大了未经许可使用该技术的可能性。 我们认为，现在是开始使用该技术解决实际业务问题的最佳时机。

在本文中，我们将考虑LoRaWAN的基本原理，构建自己的网络和使用第三方提供商的选项，并讨论支持LoRaWAN的产品。

什么是LoRaWAN


LoRaWAN是一组协议，这些协议定义了可在远距离运行的低功率，低功率设备的物理和网络数据传输层。 缩写LoRa表示远程，即大数据传输距离，而WAN（广域网）表示协议也描述了网络层。

与众所周知的无线标准GSM / 3G / LTE / WiFi不同，LoRaWAN最初旨在同时服务于大量低功耗用户设备。 因此，主要重点是抗干扰性，能效和范围。 同时，最大数据传输速率仅限于每秒几千比特。

像蜂窝网络一样，LoRaWAN具有订户设备和基站。 用户单元与基站的通信范围可以达到10公里。 同时，订户设备通常具有自主的电池电量，并且大多数时间处于节能模式，偶尔会唤醒以与服务器进行短期数据交换。 因此，例如，水表可以每隔几天唤醒一次，并将消耗的当前水量的当前值传输到服务器，而其余时间则处于睡眠模式。 这种方法使您无需更换电池即可使用长达数年的设备。 LoRaWAN设备的任务是尽可能快地从基站传输/接收必要的数据，并为其他设备腾出空间，因此网络上的广播时间有严格的规定。 设备仅在收到基站确认后才发送数据，这使您可以控制服务器上的空中负载，并及时平均分配数据交换会话。

LoRa标准描述了物理层，在433 MHz，欧洲868 MHz，澳大利亚/美国915 MHz和亚洲923 MHz频率范围内的信号调制。 在俄罗斯，Lo685 MHz频段用于LoRaWAN。

LoRaWAN如何运作


由于LoRaWAN在非授权范围内运行，因此极大地简化了其自身与基站的网络部署，在这种情况下，您无需依赖电信运营商。 除了部署自己的网络外，您还可以使用现有运营商的网络。 LoRaWAN提供商已经存在于世界各地，并且最近开始出现在俄罗斯，例如，运营商Er-Telecom已经在许多城市提供了与其LoRaWAN网络的连接 。

在俄罗斯，LoRaWAN通常在866-869 MHz范围内运行，一个用户单元占用的最大信道宽度为125 kHz。 这就是集线器用户Ruslan ElectricFromUfa Nadyrshin在SDR的帮助下使用频谱图上的LoRaWAN协议进行数据交换的样子。



自2018年以来，俄罗斯通过了法律修正案，大大减少了对868 MHz频率使用的限制。 有关管理俄罗斯LoRaWAN频率的新法律法规的详细信息，可以在本文中找到。

基站 -在LoRaWAN标准的术语中，它称为网关或集线器。 从目的上讲，该设备类似于常规移动蜂窝网络的基站：终端设备已连接到该设备，并遵循其指令选择数据传输的信道，功率和时隙。 基站连接到集中式软件服务器，该服务器可访问整个网络的整体状态，并参与频率规划等。
通常，LoRaWAN基站连接到固定电源并具有稳定的Internet访问权限。 研华提供WISE-6610系列LoRaWAN基站的几种型号，容量分别为100和500个用户单元，并能够通过以太网和LTE连接到Internet。

订户设备 -通常具有自主电源的低功耗客户端设备。 大部分时间处于睡眠模式，节能模式。 与远程应用程序服务器进行交互以发送/接收数据。 在基站和应用程序服务器上存储用于身份验证的加密密钥。 它可能在几个基站的范围内。 严格遵守从基站接收到的空气的工作规则。 研华WISE-4610设备是模块化I / O终端，具有各种模拟和数字I / O和RS-485 / 232串行接口。


客户可以部署其LoRaWAN基站或使用现有的第三方运营商网络

LoRaWAN公共网络


在这种体系结构中，设备连接到第三方运营商的公共网络。 客户端只需要购买订户设备并与提供商签订协议，即可获得访问网络的密钥。 而且，客户完全取决于运营商的覆盖范围。
必须记住，在公共LoRaWAN网络中，设备占用的广播时间可能受到严格限制，因此对于需要更频繁地进行数据交换的应用程序，此类网络可能不合适。
在发送数据之前，设备会请求发送许可，只有在基站回答确认后，才会进行交换。


当使用第三方LoRaWAN网络时，客户端使用其他人的基站基础结构，并且取决于提供商的覆盖范围及其数据传输限制

例如，当在已经存在必要基础设施的城市地区更换用户设备时，此方法很方便。 例如，用于在住宅建筑中安装传感器并传输少量数据，用于从电表或水表收集数据。 此类设备每隔几天可以发送一次数据。

专用网LoRaWAN


在部署专用网络时，客户独立安装基站并计划覆盖范围。 当您需要对网络或在没有运营商覆盖范围的站点进行完全控制时，此方法很方便。


在专用网络中，客户可以完全控制基础架构

在这种体系结构中，客户一次购买了用于部署基站的设备，然后不再依赖服务和第三方运营商。 此选项适用于在偏远的农业设施，生产场所等建立网络。 拥有的网络使扩展现有基础架构，增加覆盖范围，增加订户设备的数量和传输的数据量变得更加容易。

WISE-4610系列是用于将各种外围设备连接到LoRaWAN网络的模块化终端。 在他们的帮助下，您可以从任何数字和模拟传感器（例如温度计，湿度计，气压计，加速度计等）收集数据，并通过RS-232 / 485接口控制其他设备。 它具有内置的4000mA电池，可以独立工作长达六个月。 与太阳能电池板集成为电池充电。 内置的GPS接收器使您能够准确确定设备的位置，从而简化了记帐和安装：无需先进入数据库即可安装设备，安装后，设备会根据接收到的坐标自动将其绑定到对象。

编程和配置通过通常的USB接口执行，不需要其他的控制器和编程器，因此可以使用笔记本电脑在现场完成。
