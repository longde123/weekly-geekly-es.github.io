3D打印机如何打印骨骼，血管和器官


在快速原型技术发展的曙光中，人们普遍认为3D打印将改变所有生产方式，刺激消费者革命，结果是打印机将出现在每个家庭中。 然而，这还没有发生，因为许多最新技术都在发生这种情况，快速原型发现其在医学的一个完全不同的领域中的应用。

Gizmodo网站上的一篇文章 。

下面我们将重点研究和项目，这是使用生物打印和使用计算机控制的机器来组装生物物质的最有趣的例子，在此期间使用了有机墨水和超强热塑性塑料。 生物印刷的应用范围非常广泛-从人类颅骨主要部位的重建到支架的印刷-支架上干细胞可以发育成新的骨骼。 阅读下面的详细信息。

骷髅头



Osteofab是牛津高性能材料公司的产品。 最初，OPM进入市场出售一种通常用于制造医疗植入物的高效聚合物，称为热塑性塑料，称为聚醚酮酮（PEKK），但在过去几年中，该公司已成为首家掌握这种材料用途的公司，尤其是在增材制造中。 因此，2013年2月， 在美国患者身上安装了头骨的一部分的3D植入物，其质量获得了FDA的批准。 我们非常仔细地进行了铸造和印刷，以匹配患者头骨的独特几何形状，其中现在有75％是植入物。

皮具



在印刷的帮助下产生新皮肤的主要问题是难以从整个可能的光谱中再现出某种阴影。 鉴于我们的皮肤独特，稀薄且容易变化，因此很难创建出精确的副本。 关于这个主题有很多有趣的研究，其实质不能简而言之。

但是，这里有两个最有趣的地方：Wyake Forest University的科学家James Yoo，使用由美国国防部资助的赠款，正在研究一种可以直接在被烧伤的人身上打印皮肤的机器。 利物浦大学的科学家进行了另一项研究，他们使用经过仔细校准的3D扫描仪来获取包含所有细微差别的皮肤样本，这将在将来打印出更逼真的植入物。

这项研究仍在进行中，该团队计划创建一个带有扫描样本的“皮肤样本库”，可以从偏远医院进行连接，因为那里没有为特定患者的皮肤扫描所需的摄像头。 [ Gizmodo ; [PhysOrg ]

鼻子和耳朵



对于患者和医生本人而言，为耳朵，鼻子和下巴制作假体通常是一个痛苦，昂贵且耗时的过程。 在过去的五年中，英国工业设计师汤姆·弗里普（Tom Fripp）与谢菲尔德大学的科学家们一直在开发一种更便宜，更容易制造的面部假体，可以使用3D打印获得该假体。 制作这种假体的过程包括对患者面部进行3D扫描（与铸造相比，疼痛要小得多），对被替换的部分进行建模并使用颜料，淀粉和医用硅树脂进行打印。


这样的假体还有一个额外的好处：当它用完（无论如何最终都会发生）时，可以再次打印，从经济上讲，它将非常便宜。 [ 监护人 ]

假牙



Fripp和谢菲尔德大学的研究小组发表了测试眼睛假体制造过程的结果。 眼假体价格昂贵，并且由于它们是手动签名的，因此可能需要几个月的时间才能制作出来。 Fripp Designs打印机可以在一个小时内制作150个眼部假体，虹膜的颜色，血管的大小和数量等细节可以根据患者的需求轻松更改。 [ PhysOrg ]

功能植入物




随着电子设备-从无人机到医疗植入物-变得越来越小，科学家正努力为它们制造足够小的电池，但同时又可以提供必要的电荷。 但是，哈佛大学的一个工程师团队已经在使用3D打印机打印出一粒沙大小的微型电池。 这是新闻稿所说的：

“ ...研究人员使用一种氧化锂化合物的纳米粒子制造了用于阳极的墨水，并使用了另一种这种化合物的纳米粒子制造了用于阴极的墨水。 打印机将墨水施加到两个金顶的牙齿上，从而形成阳极和阴极的紧密结合结构。 然后，科学家将电极放在一个很小的容器中，并在其中填充电解质溶液以获取电池。”

随着时间的流逝，他们将能够为医疗植入物充电，其使用受到充电现有问题的限制。 [ 哈佛 ]

骨头




3D打印的植入物（如颌骨）已经存在了好几年。 但是，一小组研究人员正在进行一项旨在打印真实骨骼的实验。 例如，诺丁汉大学的科学家Kevin Shakeshuff发明了一种生物打印机，该打印机由聚乳酸和藻酸明胶制成基质，然后将基质涂上干细胞。



植入的基质将逐渐溶解，并被新生长的骨头所取代：大约需要三个月才能完全使骨头变形。 [ 福布斯 ]

血管和细胞



我们已经可以打印器官了 ，但是创建一个正常的循环系统存在一个严重的问题。

弗劳恩霍夫相间工程与生物技术研究所所长，德国科学家Gunter Tovar参与了一个名为BioRap的项目。 它的任务是使用3D打印机打印血管，在此期间使用合成聚合物和生物分子的混合物。 印刷循环系统已在动物身上进行了测试-尚未准备好将其引入人体。 但是，最终它们将使印刷媒体的移植成为可能。 [ 弗劳恩霍夫研究所 ]
