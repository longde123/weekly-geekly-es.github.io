Microsoft menyembunyikan data dari intelijen AS di pusat data Jerman



Microsoft akan membuka pusat data baru di Jerman , di mana pelanggan Eropa akan dapat menyimpan data mereka di luar jangkauan layanan intelijen AS. Pusat data akan dibuka pada akhir 2016 dan akan beroperasi di bawah manajemen T-Systems, anak perusahaan dari Deutsche Telekom Jerman.

CEO Microsoft Satya Nadella mengatakan Microsoft Cloud baru di pusat data Deutschland "akan memberi perusahaan Jerman kontrol atas data mereka sendiri." Namun, Anda harus membayar untuk keamanan data. Menurut FT , layanan pusat data "nasional" akan lebih mahal daripada layanan cloud standar.

Pengumuman dari Microsoft ini adalah langkah selanjutnya dalam pertarungan antara perusahaan IT AS dan agen intelijen AS yang mencari kerja sama. Perusahaan-perusahaan seperti Microsoft dan Google berusaha untuk mencegah pengguna: untuk ini, mereka perlu dilindungi. Dokumen Snowden menunjukkan bahwa NSA telah menciptakan infrastruktur yang kuat untuk mencegat informasi dan menyadap saluran komunikasi digital, bahkan berkompromisistem kriptografi umum dan mencegat lalu lintas antar pusat data .

Saat ini, pengadilan di New York melanjutkan prosesnya. Pada Desember 2013, pengadilan memerintahkan Microsoft untuk memberikan informasi dari kotak surat tertentu sebagai bagian dari penyelidikan kriminal. Microsoft hanya menyediakan sebagian dari buku alamat dan mengatakan bahwa semua yang lain ada di pusat data di Dublin (Irlandia), sehingga lembaga penegak hukum AS tidak dapat mengklaim untuk mengambil informasi dari wilayah negara asing.

Ini adalah pertama kalinya dalam sejarah bahwa sebuah perusahaan Amerika menolak untuk mematuhi perintah pengadilan untuk melindungi pengguna asingnya.

Banyak perusahaan IT sekarang beralih dari model untuk menjual perangkat lunak ke layanan cloud. Karenanya, dalam peran baru ini, sangat penting baginya bahwa pelanggan yakin akan keamanan penyimpanan data di cloud hosting.

Meskipun Microsoft dapat kehilangan gugatan spesifik, penempatan pusat data di Jerman akan memastikan keamanan informasi di masa depan. Microsoft bertindak sangat bijaksana, memberi mereka di bawah kendali perusahaan Jerman, sehingga pihak berwenang Amerika tidak akan memiliki kesempatan hukum untuk menerima data dari layanan Microsoft dari Microsoft sendiri .

Pendekatan serupa dalam perang melawan pihak berwenang menggunakan Apple. Dia menggunakan kriptografi kunci publik di perangkat seluler, dan dalam skema seperti itu, perusahaan itu sendiri, bahkan secara teoritis, tidak dapat memperoleh akses ke informasi pengguna.
