到目前为止，谈到主题时，我们暗含自己是具有思维能力和自由意志的智人生物物种的代表。 但是，如果您仔细查看所涵盖的材料，您会发现，由于主题必须是人而没有任何推理。 一个系统可以是任何东西（主要是系统性 ）。 信号和上下文的组合是一种抽象结构，不包含必须以人脑形式体现的要求。 即使在系统中产生自己的自由意志的“两个主人的仆人”的情况也没有义务只与人有关。

显然，提出自由意志寻找我们的兄弟的问题时想到的第一件事就是动物。 但这对我们来说太简单了。 让我们尝试寻找更多的外来生物。

老鼠


我允许自己进行一些语言研究。

人称代词（“我”，“您”，“他”，“她”和“它”）具有复数形式（“我们”，“您”和“他们”），可以用两种根本不同的方式使用：

  1. 描述人们（或不一定是人们）在指定集合中拥有的个人财产。 例如，“ 我们人类通常有两只胳膊和两条腿”或“ 在这里 ，男人与女人不同，随着年龄的增长，他们的耳朵开始长发”。 我们考虑许多具有某些属性的对象，当我们谈论这些属性时，我们使用代词来表示集合。 通过这种用法，代词“我们”是“我”的诚实复数形式。 没什么有趣的。

  2. 表示集体行动。 “ 我们进行了协商并决定了……”，“ 我们表现出了偏爱”，“ 我们有了孩子”。 指定行动或事件，并强调共同参与。 如果您说“我咨询过”，那么马上就会出现问题，“与谁一起？” 不能与自己协商。 如果您自己咨询，则不再是“我咨询过”，而是“我想过”。 这些“我们”小组有一个有趣的细微差别，即说话者不必亲自参与他归因于“我们”的行动。 例如，我可以自豪地宣布“我们是第一个将人送入太空的人”，用“我们”一词指代曾经存在的“苏联”社区。 同时，在尤里·加加林（Yuri Gagarin）飞翔的那一天，我不在那儿，没有人对包括我自己在内的人感到困惑。

从我们经常必须指定一个团体实体这一事实开始，甚至没有描述事件的开始，就可以使用一种依赖于情境的理由技术 ，得出存在团体实体的结论。 并且由于它们存在，因此能够对它们进行推理是有意义的。

关于复合生物，我们可以用“我们”，“您”和“他们”来表示的第一件事就是它们是系统。 可分解性是显而易见的，因此会产生全身性影响。 有时情况只是参与者的贡献被简单地总结。 例如，伊凡（Ivan）和瓦西（Vasily）将船从水上拖到绳子上的岸上。 假设一个参与者的力量不足以开展这项业务，但他们已经共同应对了。 我们从数量到质量都是平庸的过渡，但是仔细研究，情况并不像看上去那样简单。 您至少可以从以下事实开始：如果Ivan不知道Vasily也要走，他就不会上岸拉船。 如果瓦西里不知道伊万即将来临，他将不会离开。 参加者必须具有彼此同意的能力。 指定一个共同的目标，分配角色，计划活动。 事实证明，最初观察到的力向量F Ivan和F Vasily的简单求和只是冰山一角，在冰山下隐藏了非常重要的东西，包括谈判能力，关系的背景以及进一步使用船的计划。

1 +1拓扑是复合实体的最简单拓扑。 假设埃琳娜（Elena）和伊凡（Ivan）和瓦西里（Vasily）一起加入，他们不会拉船，但会在男人们忙于辛苦工作的同时准备食物。 第三方的出现立即使情况变得极为复杂。 我们拥有I + B + E公司，但是I + B系统并没有消失（是她拉船的）。 另外，如果Ivan和Elena彼此同情，那么谈论“ I + E”系统是有意义的。 我也不想错过“ B + E”系统，因为任何事情都会发生。 另外，由于发生了系统“ I + E”和“ B”，因此它们之间的关系很可能会建立起来，从而引起它们自身的系统性。 也就是，构造“（I + E）+ B”也发生了。 特别是如果有三角恋。 好吧，为了对称起见，我们添加系统“ And +（B + E）”和“（And + B）+ E”。 总计，在三个要野餐的朋友的陪伴下，我们观察到存在十个系统：

并且，B，E，I + B，I + E，B + E，I +（B + E），（I + B）+ E，B +（I + E），I + B + E

假设玛丽突然加入朋友。 现在，我将不承诺写出所有得到的组合。 然后尼古拉来了，我们得到了组合爆炸。 共有五个科目。 不要忘记“（I + B + H）+（E + M）”（这是我们综合科目“所有男孩”与综合科目“所有女孩”之间的关系）等选项也是没有意义的。

举了一个有五个朋友的例子，仅是为了说明复杂的雪崩的出现，即使只有很少量的与外界仔细隔离的元素也会出现。 当然，现实甚至更糟。 在讨论系统性时，事实证明，对于一个复合实体的存在，一个时空区域中所有参与者的存在都是不必要的。 例如，诸如“妈妈会说什么？”之类的恋爱关系可能会对Elena与男孩的恋爱关系产生重大影响。 也就是说，综合情况“ Elena +她的母亲”也确实出现在了情况中。 如果您认为参加野餐的所有参与者都对Elena的母亲很熟悉，那就很难过。 现在该是可能要在公司中增加人员了。 除非您忘记考虑Nikolai是穆斯林，Vasily是基督徒，Ivan是Spartak俱乐部的粉丝以及Maria是CSKA的粉丝，否则请记住。 不知何故，大型组成实体“穆斯林”，“基督徒”，“斯巴达克迷”和“ CSKA迷”也加入了友好的公司。

现在，我们的朋友们完成了所有紧急的野餐事宜，坐在火炉旁谈论一切。 尽管“什么都没有”是什么意思？ 实际上，我们在直接互动的过程中进行了一次协调目标，计划和建设性决策的会议，甚至无法用自发形成的锅炉“ I + B + E + M + H”中的组成实体数量来计算。

可能会（可能应该）出现一个问题，但是如果没有复合实体，是否有可能做到？ 启用组合功能时，一切都太复杂了。 也许考虑一下每个人就足够了，并根据他们的个人特征和动机来了解正在发生的事情？ 还原主义坚持这样做。 这是可能的，但是随后将出现大量无法解释的影响。 想象一下，您了解发条的工作原理，但决定单独考虑每个部分的工作，而不考虑它们之间如何相互联系。 从复合实体的考虑中排除将立即破坏整个图片，使其变成莫名其妙的行为，成为单个片段。

在这种情况下，最惊人的莫名其妙的现象之一可能是幽默。 围在火堆旁的朋友讲有趣的故事，并一起嘲笑他们。 从一个人的角度来看，幽默是绝对荒谬的。 为什么我们内部存在一种机制，可以促使我们寻找非常具体的信息，收到这些信息后，我们便开始进行特定的呼吸运动并体验独特的愉悦感？ 如果我们的身体需要呼吸运动，则会向我们发出深深的叹息，打ic或打哈欠。 简单实用。 笑声完全是另外一回事。 他高举，羞辱，清洗，倾倒在泥土中。 笑声既可以用作致命武器，也可以用作药物。 成功的小丑成为当下的王者，而失败的小丑希望立即被淘汰。 显然，事情不仅比呼吸练习还精细和复杂。

关于笑声实际上是最合理的假设，我认为笑声是我们与生俱来的自然交流机制之一的假设。 牛mo，猫叫，狗吠，人们大笑。 如果亚马逊丛林的居民对我（欧洲人）说些什么，我什么都不懂，如果他笑了，那将意味着与我的文化的任何代表完全相同。 这将意味着他很有趣。 我们对“有趣”的含义都有清晰的隐含（理所当然）理解，但是尝试用语言来描述它是有意义的。 有趣-这是“哈哈哈”，“哦，我不能”，“现在我要撕破肚子”的时候。 通过对笑声含义的明确描述，结果真是有趣。 说到我们与生俱来的自然交流机制，我们还可以回想起哭泣，惊恐，痛苦和哭泣。 这些机制的目的是可以理解的，发送和接收信号的含义很明确（这意味着我们对它们有自己的上下文）。 在这种背景下，出于某种原因而设计的笑声有所不同。 如果我们不明白为什么需要笑声（“开心”不是答案），那么我们不是正在发生的事情的直接受益者。 但是，如果不是我们，那又是谁？ 我们的身体已经实现了这些特定信号的传输，但是我们的个性只是其真正的发送者和接收者的一部分。

一再，不同的作者表达了这样的观点，即笑的含义完全不存在于个人层面，而是存在于集体层面。 集体必须在成员之间达成共识后才能存在。 例如，关于该团队中的哪些行为应被认为是不可接受的。 每次成功的笑话，公司都会获得越来越多的正直，我们的内部自然机制认识到我们正在参与非常正确的事情，并以良好的心情回馈我们。 积极加固。 一个不成功的笑话分裂了公司，这是一个坏主意，您想杀死一个脱口而出的人。 如果公司开玩笑并且乐在其中，那么这意味着该公司充满活力，健康且不断发展。 如果公司完全停止开玩笑，那么这意味着某种可怕的事情正在成为常态，并且该公司很可能注定要失败。

在此叙述的背景下，对“为什么需要笑声”这一问题的考虑本身并不十分有趣，而是作为系统形成机制的一个例子。 在构建系统组件时实现的机制，但其目的仅在整个系统级别上公开。

一般来说，金钱也是一样。 上面已经说过，金钱是信息对象，可以通过互联网进行传输这一事实始终向我们暗示。 但同时，我们不将金钱视为信息，而仅视为物质（？）资源。 如果银行帐户中有一千美元，我们认为这个事实是可以交换某些东西的特定数量的东西。 但是钱没关系。 这是信息。

金钱的全部信息内容似乎在于有关其数量的信息，但是在仔细考虑该问题后，事实证明物品的数量和物品本身都是不同的东西。 您可以获取有关水量的信息，甚至可以通过Internet传输此信息，但是水本身无法通过Internet传输。 金钱是可能的。

如果金钱是信息，那么应该有一个上下文来揭示信号的语义。 我们自己无法将金钱视为信息（我再次强调，我们将金钱视为物质资源），因此，这不是我们的信息。 与笑声一样，可以假定金钱的信息本质是在超人结构的层次上揭示出来的。 实际上，金钱是市场经济的内部信息。 当我们扮演企业实体的角色时，我们将它们视为信息而不是信息，这是我们玩游戏的一个非常重要的因素。

当然，除了这里的笑声和金钱，您肯定还需要记住关于爱情的事情。 将一个人作为一个整体吸收一个整体的令人愉快的事情是形成两个人的超人系统，能够创造出生和新生命的奇迹。 任务的复杂性，重要性和规模如此之大，以至于我们本性就应该使用最强大，最积极和最消极的增援。 在上一章谈到外部目标设定时，可以考虑“爱”的主题，但那时我们仍然没有一个重要的组成部分。 在这种情况下，什么可以成为外部目标设定的来源，却一无所知。 恋爱中的男孩似乎正在成为他所爱的人的奴隶，恋爱中的女孩是他所选择的人的奴隶。 但这是不正确的，因为在这种情况下，最成功的方案（完全互惠）变成了荒谬且适得其反的闭环。 仅当我们假设目标设定的来源是第三位时，所有一切都准备就绪。 显然，在这种情况下，只有复合主题“ guy + girl”可以成为第三名。 家庭 只有一个新生，但已经是一个家庭。 当我们看着一个冷酷的夫妇时，我们只看到他们组成的人的眼睛，在我们看来，除了他们之外，没有其他人。 但这不过是个骗局。 该复合物是“家庭”，尽管它没有单独的物理实施方式（我们记得“改革”的主题），但是当然它是作为一个系统存在的。

在正常模式下，我们每个人都可以被视为不可思议数量的多样化“我们”的身体体现。 即使进入完全与外部世界疏远的自我思考中，我们也不会将意识缩小到一定程度，而是尽可能地扩大内心的视线，以涵盖可以被称为我们的世界的宏伟事物，同时，我们自己的“我” 。 毕竟，我们不仅是这些有趣的复合生物的一部分（我们称它们为MYS），而且在我们内部相撞后，它们本身也成为我们用字母“ I”表示的系统的一部分。

在继续之前的简短说明：我们不一定只用我们自己的种类进入MOUSES的组成。 种间共生是野生动植物中普遍存在的现象。 令人严重怀疑的是，MICE的很大一部分隐藏在我们自身生物学的肠子中，我们甚至不知道它们的存在。

头脑


人类自豪感的一个特殊主题是我们宝贵的人类思想。 观察到自己身上的这一奇妙事物，我们大胆地将自己置于宇宙之巅。 我们称自己为“合理的人”。 尽管我们虚弱，缓慢，笨拙和皮肤黝黑，但我们有思想，这使我们成为自然界的大师。 全部 无一例外。

让我们冷静地解决这个问题，这是我们任命自己为大师的那本性吗？

想象一下宇航员飞入遥远的行星系统。 降落在一颗行星上。风景如画的沙漠，在奇特的彩色天空下，被有毒液体的海水冲洗。风。奇形怪状的石头。我们正在寻找理性活动的痕迹。没有建筑物的残留物，没有不寻常的物体，可以说有人专门为某些东西而建造它们。我们在日志中写道“没有发现合理活动的痕迹”。我们飞得更远。在下一个星球上，一切都完全不同。密集发展。各种自动生产。在采石场中，精心设计的集料矿山，惊人的复杂植物从中产生了一些东西。无论是在“地球”的表面，在其下方还是在空中，都有一些令人难以置信的运输系统。到处都有东西在建造。残旧的残骸被分解成零件并送去加工。尚不清楚，您只能猜测所有事情。所有这些光辉的主人（大头绿色的男人）都不可见，没有人要问。我们将在日志中写些什么？再次“没有智能活动的证据？”

当然，您不必飞到任何地方都可以看到。例如，您可以进入森林，然后从眼睛里脱下自满的眼神，环顾四周。您可以通过显微镜看一滴沼泽水。您可以通过显微镜看到一滴自己的血液。无处不在-复杂性难以想象的机制，显然每种机制都可以执行某些功能。我们要去哪里寻找大头绿色的人，因为草which将自己推入大地？为了谁的缘故，这种惊人的工程飞行奇迹-蚊子是从难以想象的垃圾中收集自己的？谁负责构建一个令人惊讶的复杂结构，我们在镜子中将其视为需要剃毛的头发？

如果我是虔诚的，那么“上帝”将回答所有问题，并引用圣经中的几句话，这可能是讨论的结尾。 “上帝”是对所有问题“如何？”，“为什么？”的普遍回答。和“为什么？”答案是没有意义的，但却是普遍的。如果思想最终陷入僵局，但您需要给出答案，请随意将所有内容撇弃给上帝的旨意。您可能已经猜到了，我们不会这么简单。
