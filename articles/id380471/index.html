Mengunggah gambar ke IBM Simon

Halo Giktayms! Saya telah mengumpulkan berbagai perangkat kuno selama beberapa tahun sekarang dan memutuskan untuk berbicara sedikit tentang salah satu "pameran" saya. Ini akan menjadi perangkat pertama yang bisa disebut smartphone - IBM Simon, yang, sebentar saja, baru saja berusia 20 tahun. Sepanjang jalan, saya akan memberi tahu Anda bagaimana saya mencari cara untuk mengunduh dan melihat beberapa perangkat di perangkat ini, karena karena alasan tertentu fungsi yang diperlukan ini hilang dari kotak.




Jadi, smartphone IBM Simon muncul pada tahun 1994, sebelum kata "smartphone" secara umum diucapkan. Prototipe pertama didemonstrasikan kembali pada tahun 1992 di pameran COMDEX, tetapi saya tidak dapat menemukan satu pun menyebutkan perangkat ini dalam laporan dari musim gugur COMDEX 1992. Pada saat itu semua orang berbicara tentang multimedia, tentang komputer dengan layar sentuh, tentang Windows NT 32-bit, itu juga Windows 3.2, dan, tampaknya, di antara teknologi profil tinggi ini, smartphone pertama tampak agak pudar.

Di dalamnya terdapat prosesor VADEM 16-MHz x86 yang kompatibel dan 1 MB RAM. Perangkat, seperti banyak perangkat portabel lainnya di awal 90-an, berjalan di bawah DOS yang berjalan di atas shell grafis. Shell terdiri dari dua layar: telepon dan daftar aplikasi.


Antarmuka Telepon


Daftar aplikasi. Ini namanya "Mobile office"

Antarmuka shell sangat sederhana, saya akan mengatakan primitif. Simon memiliki beberapa program bawaan yang khas dari penyelenggara mana pun, serta aplikasi untuk membuat catatan tulisan tangan dan permainan "Lima Belas." Anda tidak akan menemukan apa pun di perangkat ini. Secara teori, fungsionalitas Simon dapat diperluas dengan menginstal aplikasi pihak ketiga. Namun, dalam seluruh riwayat keberadaan perangkat, hanya satu aplikasi yang ditulis untuknya: klien akses jarak jauh ke PC DispatchIt. Biaya perangkat lunak $ 299 untuk klien untuk Simon dan $ 2999 untuk server untuk PC.
