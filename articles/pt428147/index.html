Script para coletar citações e reconhecer texto de vídeo em Python

Uma postagem sobre um script que baixa vídeos do youtube e reconhece texto nele. Decidi começar imediatamente com a implementação prática. "Vdudictionary" - Colecionador de scripts da coleção de citações de heróis "Vdud" em Python. Yuri Dud e seu projeto "Vdud" não precisam ser apresentados. As entrevistas mais quentes que são interessantes de assistir. Yuri Alexandrovich sabe como fazer um show interessante, independentemente de você conhecer o herói de uma questão específica, ser fã dele ou ouvir pela primeira vez esse nome.

Quantos cm você tem? O que você diz a Putin? Você está ouvindo OXY?


Essas e muitas outras perguntas estão agora associadas ao Dudem. Quando o entrevistado pronuncia uma frase cheia de sabedoria, os editores atenciosos a exibem cuidadosamente na tela de nossos monitores com você para nos transmitir o ponto inteiro. Meu objetivo era cristalizar essa sabedoria de gerações e criar o dicionário "Vdudexicon" ou "Vdudictionary".

Naturalmente, uma pessoa, mesmo que não seja sobrecarregada com um histórico específico de TI, não precisa coletar essas palavras com as mãos. Para isso, desenhei um script python.



Primeiro de tudo, precisamos de um arquivo que iremos processar. Para baixar vídeo do Youtube, usei o módulo pytube. 

pip install pytube

Um exemplo de download de um arquivo do youtube 

 from pytube import YouTube a=YouTube('https://www.youtube.com/watch?v=RNbXm8WKmow') a.streams.first().download()

O arquivo foi baixado. Agora começaremos a determinar metodicamente molduras com um significado concentrado, com a sabedoria dos contemporâneos, com as palavras dos heróis de nosso tempo.



Não havia placa retangular em versões mais antigas, portanto, podemos procurar texto na parte inferior da tela. Em novos lançamentos, você pode usar o mesmo e antigo OpenCV para procurar um retângulo, o que permitirá obter um quadro de um vídeo. 

 a,contours,h = cv2.findContours(gray3, cv2.RETR_TREE, cv2.CHAIN_APPROX_SIMPLE) for i in contours: cv2.drawContours(gray3,[i],0,(0,0,255),1)

Colocamos o tesseract - este é um mecanismo de OCR necessário para o reconhecimento óptico de texto. 

 sudo apt-get install tesseract-ocr sudo pip3 install pytesseract sudo pip3 install tesseract

Os lançamentos usam um fone de ouvido muito específico, o que dificulta o reconhecimento. Em geral, instalamos um dicionário para o alfabeto cirílico. Faça o download e transfira para / usr / share / tesseract-ocr / tessdata.



O script na entrada recebe o endereço do vídeo no youtube. Ele baixa um arquivo, começa a processar um quadro em 5 segundos. Se houver um quadrado no quadro, corte-o, descolora-o, aumente o contraste e o reconheça. Se a cadeia tiver menos de 15 caracteres, não a considere. Você pode, é claro, usar uma sequência de menos de 15 caracteres, mas como uma das heroínas do programa disse:
-Não sei, rapazes, como você vive com pequenas falas.

Colocamos no texto do arquivo de log, hora e um link para o momento no youtube. Ignoramos 5 segundos (não pergunte por que esse número veio à mente, a verificação não encontrou a sobreposição de duas aspas nesse período). Você pode excluir o arquivo de vídeo e prosseguir com a próxima versão.

Código de script completo:

Script Python 3 
 import cv2 import pytesseract import numpy as np from pytube import YouTube import os nameofvideo="RNbXm8WKmow" a=YouTube('https://www.youtube.com/watch?v='+nameofvideo) a.streams.first().download() title=a.title title2=title.replace("/","").replace(",","").replace(".","")+".mp4" os.rename(title2, "youtubefile.mp4") print(title) f=open('/var/www/python/'+str(nameofvideo)+'.txt','w') f.write(title+"<br>") f.write('<table><tr><td></td><td></td><td></td></tr>') spisoksimvolovpodudalenie=["*","/","|","\\",")","(","}","{","+","`","~","№","",":","$","#","@","%","[","]","&","^","' "] def udaleniesimvolov(stroka): for element in spisoksimvolovpodudalenie: stroka=stroka.replace(element,"") return stroka vidcap = cv2.VideoCapture('youtubefile.mp4') vidcap.set(cv2.CAP_PROP_POS_AVI_RATIO,1) durationsec=int(vidcap.get(cv2.CAP_PROP_POS_MSEC)/1000) print("duration: "+str(durationsec)+" sec") for thissec in range(0,durationsec,5): vidcap.set(cv2.CAP_PROP_POS_MSEC,thissec*1000) success,image = vidcap.read() gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY) gray2 = cv2.addWeighted( gray, 1.5, gray, 0, 0.5) gray3 = gray2[450:670,0:1280] if success: print(str(thissec)+" sec.") text = udaleniesimvolov(pytesseract.image_to_string(gray3, lang='rus')) if len(text)>15: print (text) f.write('<tr><td>'+str(thissec)+'</td><td>'+text+'</td><td><a href="https://www.youtube.com/watch?v='+nameofvideo+"&t="+str(thissec)+'"></a></td></tr>') print("----") f.write('</table>') f.close()
