Microsoft Azure Security Center kündigt neue Funktionen an

Da immer mehr Unternehmen schneller innovieren, indem sie ihre Unternehmen in die Cloud verlagern, wird eine erhöhte Sicherheit für jede Branche von entscheidender Bedeutung. Azure verfügt über integrierte Sicherheitsmanagement-Tools für Daten, Anwendungen, Computer, Netzwerke, Identifikation und Bedrohungsschutz, mit denen Sie die Sicherheit konfigurieren und Partnerlösungen integrieren können.

Wir investieren weiterhin in Sicherheit und freuen uns, die coolen Updates zu teilen, die letzte Woche auf der Hannover Messe 2019 angekündigt wurden. Dazu gehören der erweiterte Bedrohungsschutz für Azure-Speicher, das Compliance-Dashboard und die Unterstützung für skalierbare Skalierungssätze für virtuelle Maschinen ) (VMSS). Vollständige Liste unter Kat.



Die folgenden auf der Hannover Messe 2019 angekündigten Funktionen sind jetzt für das Azure Security Center verfügbar:

  • Erweiterter Azure Vault- Bedrohungsschutz - Die Schutzstufe, mit der Kunden potenzielle Bedrohungen in ihrem Tresorkonto erkennen und darauf reagieren können, sobald sie auftreten - ohne Sicherheitsexperte sein zu müssen.
  • Dashboard zur Einhaltung gesetzlicher Vorschriften - Hilft Security Center-Kunden, ihren Compliance-Prozess zu optimieren, indem Informationen zu ihrem Compliance-Status für eine Reihe unterstützter Standards und Regeln bereitgestellt werden.
  • Unterstützung für skalierbare Virtual Machine Scale Sets (VMSS) - Überwachen Sie den Sicherheitsstatus Ihres VMSS mithilfe von Sicherheitsempfehlungen.
  • Dedicated Hardware Security Module (HSM) (verfügbar in Großbritannien, Kanada und Australien) - Bietet Speicher für kryptografische Schlüssel in Azure und erfüllt die strengsten Sicherheitsanforderungen und Kundenanforderungen.
  • Unterstützung für Azure Drive Encryption für VMSS - Azure Drive Encryption kann jetzt für Windows und Linux VMSS in öffentlichen Azure-Regionen aktiviert werden - bietet Kunden die Möglichkeit, VMSS-Daten im Ruhezustand mithilfe der Standardverschlüsselungstechnologie zu schützen und zu speichern.

Darüber hinaus ist die Unterstützung für Sätze virtueller Maschinen jetzt als Teil des Azure Security Centers verfügbar. Um mehr zu erfahren, lesen Sie unseren Artikel über all diese Innovationen .
