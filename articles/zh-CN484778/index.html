英特尔Parallel Studio XE 2020：面向开发人员的最佳选择



在新年前夜，发布了新版本的英特尔主要开发人员工具英特尔Parallel Studio XE 2020 。 让我们借此机会，看看著名的“工作室”现在生活在什么地方以及它朝着什么方向发展。

首先，我们注意到英特尔Studio的新版本每年不会一次落在开发人员的头上-上一个版本（2019年）在其存在过程中获得了5次更新。 不幸的是，我们没有机会宣布每个更新，因此我们正在等待大型活动-现在已经到来了。

现在，了解英特尔Parallel Studio XE 2020的组成。没有特殊更改。 仍然提供三种产品选项和三种软件类别：开发工具，用于在单独的计算机和群集中分析和优化性能。 表格中显示了所有软件包的完整组成-您不太可能在其中找到不熟悉的单词。



支持语言和平台：

  1. 完整的C ++ 2017和部分的C ++ 20;
  2. 完整的Fortran 2008和扩展的Fortran 2018;
  3. Python 3.7
  4. 完整的OpenMP 4.5和OpenMP 5.0初稿；
  5. 新的操作系统是Amazon Linux 2。

当然，IPS XE 2020包括其所有组件的最新版本，以及用于* nix系统（conda *，PIP *，Docker Hub *，APT GET，YUM，NuGet）的许多方法，并支持与Visual Studio * 2019集成窗户

最新版本的组件有哪些改进？ 以下内容不一定适用于最新更新；而是列出了最近所做的操作。

  • 使用Intel Advanced Vector Extensions 512（Intel AVX-512）指令在最新的Intel Xeon和Intel Core处理器上的性能改进；
  • 使用新版本的编译器，库和分析工具加速AI推理；
  • 支持英特尔VTune Profiler中各种平台的高级“粗糙”和精确配置文件；
  • 借助Intel Advisor中的L1，L2，L3和DRAM的边界分析，预测将缓存用于矢量化的情况；
  • 支持Intel Cluster Checker多个数据库，并能够找到使用持久内存Intel Optane DC的配置和功能；
  • 增强了Intel TBB中的流控制灵活性
  • 改进分类和回归算法（各种Boosting类型）的功能Intel Data Analytics Acceleration Library（Intel DAAL）
  • 使用AWS Parallel Cluster和AWS Elastic Fabric Adapter支持云HPC，以使用Intel MPI库在MPI应用程序之间进行高性能数据交换。

现在让我们谈谈对Intel Parallel Studio XE 2020的技术支持。购买任何许可证之前，从购买之日起至少一年提供优先支持； 它的期限可以打折延长（也可以选择获得几年支持）。 支持包括免费访问产品更新以及对先前版本的访问和支持。

此外，客户还将获得以下好处：

  • 能够在完整的Intel硬件和软件解决方案堆栈上测试客户项目，包括服务器处理器，离散图形卡和现代计算加速器；
  • 与英特尔工程师互动以解决技术问题时，确保机密性（俄语支持）；
  • 协助解决技术问题-使解决方案适应Intel配置；
  • 免费访问所有产品更新和以前的版本；
  • 对技术人员的教育和实践培训，包括优化客户软件性能，代码分析，系统负载效率分析。 针对AI，ML，TEC等使用优化应用程序的实践。

可在英特尔网站上下载30天的试用版。 在此期限之后，您可以购买付费许可证，为您提供一年的产品支持。 学生和老师可以免费获得功能齐全的英特尔Parallel Studio XE 2020。
