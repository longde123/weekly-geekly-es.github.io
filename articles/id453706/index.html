Bagaimana Saya Lulus dari Ujian Sertifikasi Insinyur Data Google Cloud Professional

Tanpa direkomendasikan 3 tahun pengalaman praktis


Untuk mengantisipasi dimulainya kelas pada kursus Data Engineer , kami ingin berbagi dengan Anda terjemahan dari satu cerita yang sangat menarik yang mungkin akan berguna bagi para insinyur masa depan. Ayo pergi!


Google Hoody: Memakai. Ekspresi wajah yang bekerja serius: ada. Foto dari versi video artikel ini di YouTube .
Catatan Artikel ini adalah tentang ujian sertifikasi Google Cloud Professional Engineer hingga 29 Maret 2019. Ada beberapa perubahan setelah tanggal ini. Saya memasukkan mereka di bagian "Advanced".

Jadi, Anda ingin hoodie baru, seperti di sampul saya? Atau apakah Anda berencana untuk mendapatkan sertifikat Google Cloud Professional Data Engineer dan bertanya-tanya bagaimana cara melakukannya.

Dalam beberapa bulan terakhir, saya mengikuti kursus menggunakan Google Cloud untuk mempersiapkan ujian bagi seorang insinyur data profesional. Kemudian saya mencoba untuk lulus dan lulus. Dan beberapa minggu kemudian hoodie saya dikirim. Sertifikat datang lebih cepat.

Artikel ini akan mencantumkan beberapa hal yang mungkin ingin Anda pelajari dan langkah-langkah yang saya ambil untuk mendapatkan sertifikat Google Cloud Professional Data Engineer saya.

Mengapa Anda ingin memenuhi syarat untuk Google Cloud Professional Data Engineer?


Data ada di mana-mana. Dan pengetahuan tentang cara membuat sistem yang dapat memproses dan menggunakan data sangat dibutuhkan. Google Cloud menyediakan infrastruktur untuk membangun sistem ini.

Anda mungkin sudah memiliki keterampilan untuk menggunakan Google Cloud, tetapi bagaimana Anda menunjukkan ini kepada majikan atau klien masa depan Anda? Ada dua cara: portofolio proyek atau sertifikasi.

Sertifikat tersebut memberi tahu calon klien dan pemberi kerja: "Saya memiliki keterampilan, dan saya berupaya untuk mendapatkan akreditasi."

Penjelasan singkat dari Google diringkas.

Tunjukkan keahlian Anda dalam merancang dan membuat sistem pemrosesan data, serta membuat model pembelajaran mesin di Google Cloud Platform.

Jika Anda belum memiliki keterampilan, berkenalan dengan materi pelatihan sertifikasi berarti Anda akan mempelajari semua tentang cara membuat sistem pemrosesan data kelas dunia di Google Cloud.

Siapa yang ingin memenuhi syarat untuk Google Cloud Professional Data Engineer?


Anda melihat angkanya. Awan tumbuh. Itu sudah ada di sini dan tidak akan pergi ke mana pun. Jika Anda belum melihat angkanya, percayalah, awan tumbuh.

Jika Anda sudah menjadi spesialis data, insinyur data, analis data, insinyur pembelajaran mesin, atau mencari peluang karier di dunia data, sertifikasi Google Cloud Professional Data Engineer adalah untuk Anda.

Kemampuan untuk menggunakan komputasi awan menjadi persyaratan untuk posisi berorientasi data apa pun.

Apakah Anda memerlukan sertifikat untuk menjadi insinyur data / set data / insinyur pembelajaran mesin yang baik?


Tidak.

Anda masih dapat menggunakan Google Cloud untuk bekerja dengan solusi transfer data tanpa sertifikat.

Sertifikat hanyalah salah satu cara untuk memvalidasi keterampilan yang ada.

Berapa biayanya?


Biaya ujian $ 200. Jika gagal, Anda harus membayar lagi untuk upaya baru.

Kemungkinan biaya yang terkait dengan kursus persiapan dan penggunaan platform itu sendiri.

Biaya platform adalah biaya untuk menggunakan layanan Google Cloud. Jika Anda adalah pengguna yang canggih, Anda sudah mengetahui hal ini. Jika tidak, dan Anda hanya berkenalan dengan materi pelatihan yang dijelaskan dalam artikel ini, Anda dapat membuat akun Google Cloud baru dan memenuhi batas 300 dolar yang ditawarkan Google saat pendaftaran.

Kami akan beralih ke biaya kursus dalam satu detik.

Berapa lama sertifikasi ini berlaku?


2 tahun Setelah itu, Anda harus mengikuti ujian lagi.

Dan karena Google Cloud berkembang setiap hari, kemungkinan apa yang diperlukan untuk sertifikat akan berubah (seperti yang saya ketahui, sudah berubah pada saat saya mulai menulis artikel ini).

Apa yang perlu Anda siapkan untuk ujian?


Google merekomendasikan lebih dari 3 tahun pengalaman industri dan lebih dari 1 tahun mengembangkan dan mengelola solusi menggunakan GCP untuk sertifikasi profesional.

Saya tidak punya yang di atas.

Dari kekuatan hingga 6 bulan pengalaman yang relevan. Untuk menebus kekurangan itu, saya menggunakan kombinasi sumber daya pelatihan online.

Kursus apa yang telah saya ikuti?


Jika Anda seperti saya dan Anda tidak memiliki persyaratan yang disarankan, Anda dapat mengambil beberapa kursus berikut untuk meningkatkan keterampilan Anda.

Kursus-kursus berikut adalah apa yang saya gunakan untuk persiapan sertifikasi. Mereka terdaftar dalam urutan penyelesaian.

Saya telah menunjukkan biaya, waktu dan manfaat untuk lulus ujian sertifikasi untuk masing-masing.



Beberapa sumber online hebat yang saya gunakan untuk pelatihan ujian. Dalam urutan: Cloud Guru , Linux Academy, dan Coursera .

Biaya : $ 49 per bulan (setelah uji coba gratis 7 hari)
Waktu : 1-2 bulan, 10+ jam per minggu
Khasiat : 8/10

Ini dibagi menjadi lima sub-kursus, yang masing-masing membutuhkan waktu sekitar 10 jam seminggu waktu belajar.

Anda akan melewati serangkaian latihan praktis menggunakan platform berulang yang disebut QwikLabs. Sebelum ini, kuliah oleh praktisi Google Cloud tentang cara menggunakan berbagai layanan seperti Google BigQuery, Cloud Dataproc, Dataflow dan Bigtable akan diadakan.

Biaya : Gratis
Waktu : 1 minggu, 4-6 jam
Kegunaan : 4/10

Jangan menganggap skor utilitas rendah sebagai indikator tidak berharga. Ini jauh dari kasus. Satu-satunya alasan dia mendapat skor lebih rendah adalah karena dia tidak fokus pada sertifikasi seorang insinyur data profesional (seperti namanya).

Setelah menyelesaikan spesialisasi Coursera, saya mengambil kursus ini sebagai kursus penyegaran karena saya hanya menggunakan Google Cloud untuk beberapa kasus pengguna khusus.

Jika Anda berasal dari penyedia layanan cloud lain atau belum pernah menggunakan Google Cloud sebelumnya, Anda mungkin perlu mengikuti kursus ini. Ini adalah pengantar yang bagus untuk Google Cloud Platform secara keseluruhan.

Insinyur Data Profesional Bersertifikasi Google dari Linux Academy

Biaya : $ 49 per bulan (setelah uji coba gratis 7 hari)
Waktu : 1-4 minggu, 4+ jam per minggu
Kegunaan : 10/10

Setelah menyelesaikan ujian dan berpikir tentang kursus yang saya ambil, Insinyur Data Profesional Bersertifikat Google dari Linux Academy ternyata menjadi yang paling berguna.

Video, serta e-book Data Dossier (sumber belajar gratis yang hebat yang disertakan dengan kursus) dan ujian praktik telah menjadikan kursus ini salah satu sumber belajar terbaik yang pernah saya gunakan.

Saya bahkan merekomendasikannya sebagai referensi dalam beberapa catatan Slack untuk tim setelah ujian.

Catatan dalam Slack


  • Beberapa hal pada ujian itu bukan ujian di Linux Academy, atau Cloud Guru, atau Google Cloud Practice (diharapkan)
  • 1 pertanyaan dengan grafik titik data tentang persamaan mana yang harus Anda kelompokkan (misalnya, cos (X) atau X² + Y²)
  • Mengetahui perbedaan antara Dataflow, Dataproc, Datastore, Bigtable, BigQuery, Pub / Sub dan bagaimana mereka dapat digunakan adalah suatu keharusan.
  • Dua contoh studi ujian yang bekerja persis sama dengan di kelas praktis, meskipun saya tidak membahas studi ini sama sekali selama ujian (pertanyaan memberi pemahaman yang cukup).
  • Mengetahui sintaks dasar query SQL sangat berguna, terutama untuk pertanyaan BigQuery.
  • Ujian praktik yang diberikan oleh Linux Academy dan GCP sangat mirip dengan pertanyaan ujian, dan saya akan mengerjakannya masing-masing beberapa kali dan akan menggunakannya untuk mencari tahu kelemahan Anda.
  • Tip kecil untuk membantu dengan Dataproc: “ Dataproc buaya dan Hadoop rencana gajah untuk menyalakan api dan memasak Sarang Babi ” {The buaya Dataproc dan gajah Hadoop berencana untuk membuat api ( Percikan - percikan, percikan api - buat api) dan memasak segerombolan ( Sarang ) babi ( Babi )} (penawaran Dataproc dengan Hadoop, Spark, Sarang dan Babi)
  • “ Dataflow adalah sinar cahaya yang mengalir” { Dataflow adalah sinar cahaya saat ini} (Dataflow berurusan dengan Apache Beam)
  • “Semua orang di seluruh dunia dapat berhubungan dengan Spanner yang dicuci dengan ACID yang dibuat dengan baik ” {Siapa saja di seluruh dunia dapat berurusan dengan pembersih asam ( ACID ) dengan Spanner} (Cloud Spanner adalah basis data yang dirancang untuk mengangkat awan dengan awal, kompatibel dengan ACID dan tersedia di seluruh dunia)
  • Mengetahui nama versi klasik dari basis data relasional dan non-relasional (mis. MongoDB, Cassandra) mungkin berguna.
  • Peran IAM sedikit berbeda untuk setiap layanan, tetapi bermanfaat untuk memahami cara memisahkan pengguna agar tidak dapat melihat data tanpa membuat mustahil untuk mendesain alur kerja (misalnya, peran Pekerja Dataflow dapat merancang alur kerja tetapi tidak melihat data)

Ini mungkin cukup untuk saat ini. Kilometer cenderung berbeda dari ujian ke ujian. Kursus Linux Academy akan memberikan 80% pengetahuan.

  6. Visualisasi data dan dukungan kebijakan
