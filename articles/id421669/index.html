Pengadilan melarang distribusi file CAD tingkat senjata gratis, sehingga pencipta mulai menjualnya


File untuk mencetak senapan AR-15 dari situs web Terdistribusi Pertahanan. Flash drive dengan file dijual dengan harga gratis yang ditetapkan oleh pembeli

Aktivis Pembebasan Distribusi 3D Cody Wilson dari Defense Distributed menemukan jalan keluar asli dari situasi di mana pengadilan AS memutuskan untuk membatasi distribusi file secara gratis untuk mencetak komponen plastik untuk pistol dan senapan. Sebagai warga negara yang taat hukum, Cody berhenti mendistribusikan file dengan lisensi gratis - dan sekarang akan menjualnya . "Jika saya tidak bisa menjadi Napster di dunia pencetakan senjata 3D, saya dengan senang hati akan setuju dengan peran iTunes, " katanya pada konferensi pers pada 28 Agustus.


Cody Wilson (kanan) pada konferensi pers

Ingatlah bahwa peristiwa bersejarah terjadi pada 10 Juli 2018: setelah persidangan lima tahun , gugatan antara Departemen Kehakiman AS dan organisasi hak asasi manusia, Amandemen Kedua, yang berbicara atas nama Cody Wilson dan Defense Distributed, yang mendistribusikan file untuk pencetakan pistol dan senapan 3D melalui Internet, berakhir. , termasuk "Liberator" yang terkenal - pistol pertama di dunia, sepenuhnya dicetak pada printer 3D.

Aktivis hak asasi manusia berhasil mempertahankan kebebasan berbicara dan hak untuk secara bebas mempublikasikan file di Internet.

Yayasan Amandemen Kedua dan Pertahanan Terdistribusi menggugat Departemen Luar Negeri AS ketika pemerintahan Obama melarang distribusi file cetak senjata pada Mei 2013, mengutip undang-undang Perang Dingin lama yang melarang ekspor senjata dari Peraturan Lalu Lintas Senjata Internasional (ITAR).

Pada Juli 2018, dengan persetujuan para pihak, larangan itu dicabut. Selain itu, Departemen Kehakiman AS setuju untuk membayar semua biaya hukum penggugat dan lebih lanjut mengkompensasi biaya $ 10.000 yang harus Dibayarkan Pertahanan kepada Departemen Negara sebagai akibat dari pembatasan sebelumnya.

Sebagai hasil dari perjanjian tersebut, negara mengakui bahwa senjata api non-otomatis hingga kaliber 0,50, termasuk senapan olahraga semi-otomatis modern, seperti senapan AR-15 yang populer dan senjata api sejenis, "tidak bersifat militer". Aktivis hak asasi manusia menganggap ini bukan hanya kemenangan penting kebebasan konstitusional dan hak atas kebebasan berbicara, tetapi juga merupakan pukulan serius terhadap lobi "kiri", yang berusaha untuk melarang warga negara memiliki senjata.

Namun, pada 30 Juli, dua puluh negara bagian AS mengajukan permintaan pengadilan federal Seattle untuk perintah penahanan sementara terhadap Defense Distributed, yang telah membuka situs web DEFCAD (hanya IP A.S) dan kembali mulai mendistribusikan file untuk mencetak senjata. Selama beberapa jam, pengadilan mengeluarkan perintah ini, yang melarang Pertahanan Terdistribusi secara bebas mendistribusikan file senjata di Internet.

Dalam pernyataan bersama, perwakilan dari 20 negara bagian AS menyatakan pandangan bahwa perjanjian antara Departemen Luar Negeri dan pembela hak asasi manusia melanggar Undang-Undang Prosedur Administratif, serta Amandemen Kesepuluh Konstitusi AS, yang memungkinkan setiap negara untuk mengatur senjata api di dalam perbatasannya.

Tapi Cody Wilson menemukan jalan keluar. Dia menafsirkan larangan secara harfiah: dokumen itu melarang distribusi gratis model digital. Untuk mengatasi keterbatasan ini, ia mulai menjual flash drive dengan informasi yang sama. Harga barang ditentukan oleh pembeli (Nama Harga Anda).
