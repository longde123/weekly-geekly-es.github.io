Windows 10警告Chrome和Firefox用户有关能耗增加

操作系统会显示电池快速消耗的通知，并建议切换到Edge





微软正在努力推广其Edge浏览器。使用的方法是不寻常的。现在，在Windows 10中，有一些针对用户的新提示。许多技巧与第三方浏览器的功耗有关。这些是Google Ghrome和Mozilla Firefox。 Windows 10警告用户，使用这些浏览器时，笔记本电脑或平板电脑的电池电量消耗要比使用“本机” Edge浏览器时快。还提出了一种解决方案-切换到Microsoft Edge。

“ Microsoft Edge是专门为Windows 10创建的。浏览器功能...提供了轻松的数据共享，读取和搜索，” 他说。在Microsoft的声明中。Windows Phone应用程序开发人员Rudy Huyn发现了Windows 10用户的新提示。在使用Chrome浏览器的Windows 10环境中工作时，他获得了提示之一。微软发言人表示，Windows 10上的Firefox用户会收到类似的通知，新的提示已于7月初添加。

“ Windows提示通知的创建是为了向用户提供快速而有用的信息，以帮助他们改善使用Windows 10的体验，其中包括有助于用户延长电池寿命的信息。但是，在Windows 10中，您可以自己选择浏览器和搜索服务，”微软发言人说。

此前，该公司发布了一个视频。，这表明Microsoft Edge与其他浏览器相比具有较低的功耗。该视频的创作者将四台Surface Book设备与Microsoft Edge，Google Chrome，Mozilla Firefox和Opera一起使用。在每个设备上启动了高分辨率的循环视频，视频流是通过Wi-Fi从网络下载的。显然，这次“竞争”的赢家是搭载Microsoft Edge的设备。



该公司很快将发布Windows 10周年更新，Edge将获得许多改进。其中许多真的很有趣。例如，这是用户无法使用的浏览器选项卡上的JavaScript停止。这些选项卡将在后台接收数据，之后，如有必要，可以快速将“睡眠”选项卡退出此模式。在这种情况下，能耗可以降低到90％。此外，Flash将作为单独的过程工作。如果Flash发生问题，用户可以继续工作而不会出现问题。此外，浏览器的开发人员将重新设计许多按钮，这也将有助于节省能源。动画界面元素的帧数将减少，绘制此类元素时的负担将落在图形适配器上。加上一切开发人员已更改了某些网络功能。更改影响了TCP快速打开，尾丢失探测和最近的确认，并且初始拥塞窗口已在TCP堆栈中进行了优化。

尽管这样，但是Microsoft的一些技巧使人们对其使用的合法性提出了怀疑。这不是公司第一次尝试影响用户以将其转移到他们自己的产品上。 2015年，该公司的搜索服务-Microsoft Bing 使用了类似的方法。如果Edge用户在Bing搜索服务中输入了诸如“ chrome”和“ firefox”之类的查询，则会向他显示所有必要的搜索结果。但是在最上方，该公司显示了一条通知，指出“ Microsoft建议将Microsoft Edge用于Windows 10”，并带有“了解原因”按钮。当用户单击按钮时，他被重定向到该站点，在那里他获得了有关Edge优势的全面信息。



对于Chrome请求和Firefox请求，都显示了类似的通知。



在Windows 10上（主要在刚购买的设备上）开始使用Edge时，仅向用户显示一次此信息。随后对类似查询的搜索未引起Edge优势的通知。

微软在1990年代面临指控由于Windows中包含Internet Explorer，因此违反了反托拉斯法。现在情况有所不同，Edge不再拥有Internet Explorer曾经占领过的浏览器市场的一小部分。但是，微软显然正在寻找新方法来吸引新用户-通过将IE粉丝转移到新的浏览器，以及使用Chrome或Firefox。
