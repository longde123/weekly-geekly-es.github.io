Victor Gamov tentang Kafka Streaming IQ di jug.msk.ru

Pada 8 Mei 2019, pada pertemuan komunitas Moskow pengembang Java jug.msk.ru, Victor Gamov berbicara tentang Kafka Streams IQ ( Pertanyaan Interaktif ).



Tentang Pembicara


Victor berbicara untuk keempat kalinya di jug.msk.ru , laporan sebelumnya tentang Nashorn ( 13 November 2014 ), Hazelcast ( 4 Mei 2016 ), Apache Kafka ( 23 April 2018 ). Seperti setahun yang lalu, dia saat ini bekerja di Confluent , tetapi sekarang sebagai Advokat Pengembang .

Salah satu pendiri dan pembawa acara Podcast Debriefing , penulis bersama buku Enterprise Web Development , pembicara dari banyak konferensi (termasuk yang diselenggarakan oleh JUG.ru Group ). Situs web Victor (dengan blog teknis di sana), laporan video di YouTube , file presentasi di Speaker Deck dan Notist , Habr , Twitter , dan GitHub .

Beberapa laporan dibuat sejak pidato terakhir di jug.msk.ru :

  • “Kafka di Kubernetes: Kisah Misterius tentang Menciptakan Platform Pemrosesan Data Streaming Awan” ( DevOops 2018 : presentasi )
  • "Java Puzzlers NG S03: Apa? Dari mana datangnya mereka?! " (dengan Baruch Sadogursky di Oracle Code One 2018 : video )
  • “Fight Crime with Kafka Streams dan Bintray Firehose API” (dengan Baruch Sadogursky di Oracle Code One 2018 : video , presentasi )
  • “Crossing the Streams: Memikirkan Kembali Pemrosesan Streaming dengan KStreams dan KSQL” ( DevNexus 2019 : presentasi )
  • "Kafka di Kubernetes: Apakah itu benar-benar harus menjadi" The Hard Way "?" ( Kafka Summit 2019 : presentasi )

Tentang laporan itu


Secara tradisional, sebelum presentasi pembicara dan awal pertemuan, Andrei Kogun mengingatkan tentang acara yang akan datang - aplikasi untuk konferensi Joker Java (St. Petersburg, 25-26 Oktober) dibuka dan festival IT TechTrain diharapkan akan diadakan untuk kedua kalinya (St. Petersburg, 24-25) Agustus).



Laporan ini dikhususkan untuk Pertanyaan Interaktif , bagian dari Kafka Streams API , yang menyediakan kemampuan untuk mengakses keadaan perantara aplikasi dan hasil yang diperoleh tanpa menggunakan database dan repositori tradisional.



Tampilan slide presentasi diselingi dengan contoh kode. Sejumlah besar pertanyaan diajukan, di mana Victor segera memberikan jawaban komprehensif, yang juga dikonfirmasi oleh ulasan di Twitter .



Diskusi berlanjut baik selama istirahat pertemuan dan setelahnya. Pertemuan tersebut dihadiri oleh salah satu pembicara di konferensi JUG.ru, Ivan Ponomarev, yang laporannya baru-baru ini di konferensi JPoint 2019 "Kafka Streams API: selangkah lebih jauh dari Hello World" ( presentasi , video sejauh ini hanya tersedia untuk peserta konferensi) dapat melengkapi cerita Victor.



Penyajian laporan , kode contoh , video akan segera muncul (lihat tautan di bawah).
