Sailfish OS Unobvious Chips

Hallo Leser!

Heute möchte ich einige der nicht offensichtlichen Chips Sailfish OS hervorheben. Der Artikel richtet sich mehr an Benutzer von Sailfish OS, aber auch an die „Anhänger“ anderer Plattformen. Ich hoffe, es wird auch interessant sein, sie können den Grad an Komfort und Bereitschaft des Systems für den Endbenutzer schätzen. Wenn Sie bereit sind, gehen wir unter den Schnitt!



Wussten Sie, dass:


1. Wenn Sie Ihren Finger auf die Anwendung halten und das Smartphone umdrehen, ändert die Anwendung NICHT die Ausrichtung.
Stellen wir uns ein Beispiel vor: Sie haben im Stehen von Ihrem Smartphone aus im Internet surft und sich dann hingelegt (während die Ausrichtung des Telefons ausgeschaltet ist), damit sich die Ausrichtung nicht ändert. Halten Sie einfach Ihren Finger auf dem Bildschirm, während das Smartphone seine Position im Raum ändert.

2. Durch Doppelklicken auf die Hardware-Sperrschaltfläche wird Ihr Gerät entsperrt und Sie gelangen sofort zum Startbildschirm, wobei Sie den Sperrbildschirm umgehen.
Bitte beachten Sie, dass ein Doppelklick nicht funktioniert, wenn Sie beim Entsperren ein Passwort haben.

3. Seit Version 2.0.2 ist in Sailfish OS eine Funktion zum Aufzeichnen von Anrufen verfügbar.
Um es zu aktivieren, gehen Sie zu Einstellungen (Anwendungen) - "Anrufaufzeichnung".

Hinweis:
Im Allgemeinen können Sie im Abschnitt "Einstellungen" des Abschnitts "Apps" viele von Jolla selbst erstellte Anwendungen konfigurieren und einige Aktionen für Android-Anwendungen ausführen (Cache leeren, Programm stoppen usw.).

4. Wenn Sie nicht auf die Sperrtaste doppelklicken, wird das Telefon durch anschließende Vibration gesperrt, was darauf hinweist, dass das Telefon gesperrt ist.
Anwendungsbeispiel: Das Telefon befindet sich in Ihrer Tasche und Sie sind sich nicht sicher, ob es gesperrt ist, möchten es aber nicht erhalten. Dann müssen Sie nur noch den Sperrknopf fühlen und zweimal tippen, bis Sie die Vibration spüren.

5. Sie können die Tutorial-Anwendung sofort beenden. Klicken Sie dazu ganz oben mit der rechten Maustaste auf die Ecken, beginnend von oben links.

6. Ab Version 2.0.2 wurde ein einfacher Dateibrowser "out of the box" angezeigt, der zu Einstellungen (Speicher) - Externer Speicher / Speicherkarte (Benutzerdaten) - Wischen von oben nach unten - Option Dateimanager (Datei) wechselt Manager).

7. Ab Version 2.0.2 können Screenshots gemacht werden. Drücken Sie dazu eine halbe Sekunde lang auf beide Lautstärketasten.

8. Für Kenner des Semantic Web : Sailfish OS verwendet einige Technologien des Semantic Web:

[nemo@Jolla ~]$ tracker-
tracker-control  tracker-info     tracker-sparql   tracker-tag
tracker-import   tracker-search   tracker-stats


tracker-sparql!
Insbesondere wird der Tracker von einem Mediaplayer verwendet.

Sie werden zur Seite mit den Atmosphäreneinstellungen weitergeleitet, auf der Sie die Farben nach Ihrem Geschmack anpassen oder einfach so lassen können, wie sie sind, und sie genießen können.

-sagen, Lager zu setzen- Client für Openrepos. Ermöglichen Sie zunächst die Installation von Anwendungen aus nicht vertrauenswürdigen Quellen: Einstellungen - „Nicht vertrauenswürdige Software“. Laden Sie dann die RPM-Datei über den obigen Link herunter und installieren Sie sie von der Konsole oder mithilfe der Dateibrowser-Anwendung. Openrepos ist ein Repository, in dem es eine große Anzahl von Anwendungen gibt, die sich aus verschiedenen Gründen nicht im Jolla Store befinden (zum Beispiel verbietet der Jolla Store die Verwendung von Dämonen, siehe FAQ ). In Openrepos gibt es keine Qualitätssicherung, die einige Risiken birgt, aber gleichzeitig dem Entwickler völlige Freiheit gibt.
