Gambaran umum cPanel - konsol berbasis web untuk mengelola situs dan hosting

Kami melanjutkan serangkaian posting tentang hosting dan panel kontrol situs (konsol web) yang ditawarkan RUVDS kepada pelanggannya. Hari ini kita melihat salah satu panel cPanel paling populer, lisensi yang dapat diperoleh secara gratis selama 2 minggu, ketika membeli server virtual .

Posting sebelumnya dapat dilihat di sini: Tinjauan ISPmanager , Tinjauan Plesk, dan Parsing konsol web Plesk Obsidian yang baru . Ingatlah bahwa lisensi untuk Plesk dan ISPmanager dapat diperoleh untuk stok secara gratis hingga akhir tahun ini ketika Anda membeli VPS . Tentang panel lain - di pos Apa yang baru di konsol web 2019 . Dan di sini tentang hosting: Cara memilih VPS dan Hosting: opsi, perbandingan, statistik pengguna .



cPanel adalah salah satu konsol web komersial yang paling dicari di dunia, terutama di Amerika Utara dan Eropa. Perusahaan pengembang itu sendiri menulis di situs web resmi tentang produknya sebagai berikut: "Platform hosting terkemuka di industri dengan dukungan kelas dunia." Itu diakui sebagai nyaman untuk bekerja dengan situs, dan untuk menjual kembali hosting, yang mengapa sangat populer. Sebenarnya, panel terdiri dari dua modul antarmuka: panel kontrol situs grafis (cPanel) dan konsol administrasi Manajer Host Web (WHM). Lingkungan ini bekerja bersama dan Anda dapat mengelola situs di dalamnya - karena itu cPanel ideal untuk pemula dan pengembang yang lebih memilih konsol daripada desain yang indah.

Cpanel memungkinkan Anda untuk:

  • Buat situs dan domain dalam beberapa klik
  • Luncurkan dan kelola jaringan email
  • Simpan dan kelola file web



WHM memungkinkan Anda untuk:

  • Buat Akun cPanel untuk Klien
  • Monitor dan lindungi server Anda
  • Transfer dan buat cadangan data dalam beberapa klik



PL Dasar: Perl, PHP (mode PHP: suPHP, DSO (mod_php), FastCGI, CGI).
OS yang didukung: Red Hat Enterprise Linux (RHEL), CentOS, CloudLinux. Dimungkinkan untuk mendukung Windows melalui virtualisasi atau melalui panel Enkompass dari pengembang yang sama.
Dukungan server web: Apache, Nginx, Apache + Nginx, LiteSpeed.
Server DNS: Bind, MyDNS, NSD.
Server FTP: ProFTPd, PureFTPd
Server SMTP: Exim
Server IMAP dan POP3: Kurir, Dovecot

Antarmuka cPanel terdiri dari bagian-bagian berikut:



  1. Bilah navigasi
  2. Panel samping
  3. Informasi Rumah
  4. Fitur
  5. Statistik

Banyak orang menganggap cPanel agak kelebihan beban dengan fitur, yang menyulitkan pemula. Namun, pertama, cPanel adalah sistem multifungsi yang berbicara sendiri; kedua, pengembang terus berusaha mempercepat panel, secara aktif bekerja untuk memperbaikinya, berdasarkan permintaan pelanggan, khususnya pada antarmuka yang intuitif dan ramah pengguna; ketiga, situs web perusahaan memiliki video pelatihan , forum dan blog, ditambah dukungan teknis sepanjang waktu . Perhatikan bahwa tidak akan mudah bagi pemula untuk memiliki program baru untuk mereka (meskipun cPanel juga menangani ini), tetapi administrator yang berpengalaman tidak mungkin menderita dari banyaknya alat yang disediakan untuk pekerjaan yang mudah dan cepat. Kami akan menjelaskannya.
