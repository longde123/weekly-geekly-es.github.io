Kolektor meminta akses ke telepon debitur




Asosiasi Nasional Agen Pengumpul (SRO NAPKA) mengirim permohonan ke Duma Negara dengan permintaan untuk memberikan kepada kolektor nomor-nomor telepon para debitur yang ditunjukkan dalam Sistem Identifikasi dan Otentikasi Terpadu (ESIA). Menurut kolektor, menyediakan akses akan memecahkan masalah panggilan ke warga negara yang nomornya ditunjukkan dalam kuesioner secara tidak sengaja.

Para ahli tidak setuju dengan pendapat ini, mengatakan bahwa masalah panggilan ke pihak ketiga ada, tetapi di sini ada cara untuk menyelesaikannya, yang ditawarkan kolektor, membawa risiko tambahan. Secara khusus, ini akan menyebabkan melemahnya perlindungan data pribadi.

Mozilla akan menggantikan Firefox untuk Android




Pada bulan Januari tahun ini, mockup Fenix, browser mobile Mozilla baru, muncul di GitHub. Sekarang pengembang telah menyediakan akses ke pengembangannya . Ternyata, Fennec, Firefox untuk Android saat ini, tidak begitu populer di kalangan pengguna, jadi Mozilla memutuskan untuk tidak menghabiskan upaya lebih lanjut untuk bekerja dengannya.

Browser baru terutama dibuat untuk generasi muda. GitHub menyatakan bahwa "Fenix ​​bukan peramban Android untuk orang tua Anda."

Linux 5.1 rilis kernel


Komunitas Linux menghabiskan dua bulan bekerja pada kernel Linux 5.1. Versi baru menerima sejumlah besar fitur dan tambahan baru . Di antara pembaruan, ada baiknya menyoroti antarmuka baru untuk asynchronous io_uring I / O, dukungan untuk pemantauan terukur sistem file yang sangat besar melalui fanotify, solusi untuk masalah 2038, dll.

Torvalds mengatakan bahwa kernel 5.1 menerima lebih dari 13 ribu komit. Daftar perubahan terakhir sangat luas.

Kementerian Komunikasi tidak mendukung pra-instalasi aplikasi Rusia di smartphone


Kementerian Komunikasi dan Media Massa memberikan penilaian negatif terhadap gagasan FAS tentang pemasangan wajib aplikasi Rusia di perangkat seluler. Keputusan akhir belum dibuat, tetapi gagasan itu kontradiktif.

Secara khusus, Kementerian Komunikasi prihatin bahwa Apple tidak akan merilis versi khusus iOS untuk pasar Rusia, karena itu tidak terlalu penting bagi perusahaan. Tetapi telepon korporasi digunakan oleh banyak pejabat, termasuk Perdana Menteri Dmitry Medvedev.

Pesan utama dari proposal FAS adalah perlindungan dari produsen dalam negeri. Perwakilan dari Kementerian Telekomunikasi Evgeny Novikov tidak setuju dengan ini: "Pasar untuk aplikasi untuk peralatan pengguna sudah kompetitif, sehingga metode seperti itu tidak mungkin efektif di sini." Plus, jika membatasi persaingan secara artifisial, ini mengarah pada risiko mendistribusikan kembali pasar demi perusahaan besar Rusia.

Data tertutup bocor dari repositori Samsung untuk pengembang


Baru-baru ini diketahui bahwa data paling penting bocor dari repositori Samsung , termasuk kode sumber dari beberapa aplikasi, kunci rahasia untuk proyek internal, login dan kata sandi untuk akun pengembang. Beberapa proyek di repositori ternyata bersifat publik, berisi data yang berisi akses ke akun pengembang dan proyek tertutup.

Masalahnya ditemukan oleh Mossab Hussein, seorang spesialis keamanan dari SpiderSilk yang berbasis di Dubai. Karena kerentanannya, ia berhasil mendapatkan akses ke 135 proyek, akun Samsung di Amazon Web Services, log dan data analitis beberapa layanan, misalnya, SmartThings dan Bixby.
