Moto返回俄罗斯

拥有近一个世纪历史的摩托罗拉移动业务部门在出售Google后成为联想的一部分。 中断五年后，Moto返回俄罗斯市场。 该公司推出了两款新的廉价智能手机Moto C和Moto E系列，并宣布了非常薄的模块化Moto Z2 Play。

我设法参加了一次演讲，公司代表在会上做了演讲-摩托罗拉移动公司副总裁兼市场总监Jan Hackfeld，分公司负责人Oliver Ebel和联想移动公司俄罗斯和中亚地区总经理Anton Shveynov。

首先，我建议回顾一下摩托罗拉移动设备的历史，然后看一下新闻。



一点历史


摩托罗拉的历史始于汽车收音机，正如我在另一篇文章中所写的 。 第二次世界大战期间，该公司向美军提供了对讲机。 我们在电影《拯救大兵瑞恩》中看到了其中一个。



1956年，为医生设计了一个寻呼机，该寻呼机允许医院出于任何原因都不使用免提电话。 该设备没有1990年代的寻呼机用户记住的屏幕。 它本质上是对讲机。 操作员直接在耳边向医生报告必要的信息。

在“ 剩下的小工具：旧寻呼机广告 ”一文中，我收集了过去最有趣的视频。 但是，以我的名字有点狡猾-这个小工具什么都没走，它从我们的雷达中消失了，因为它再次成为一种利基设备。 例如，我们在《博士屋》的每一集中都看到他。



移动电话的历史始于1973年，当时AT＆T的Martin Cooper首次致电“请”贝尔实验室的竞争对手。 那年，该公司推出了DynaTAC电话，并于1983 年在移动网络上进行了首次商业通话 。 同年，摩托罗拉就免提汽车。



第一部手机重1公斤，重150克，具有红色LED显示屏，可以通话35分钟，待机时间为8小时。 它的不带天线的参数是22.5 x 12.5 x 3.75厘米，这个小工具售价3995美元，今天大约是9868美元 。 在DynaTAC之后是Flip Phone，然后是MicroTAC，后者是带有铰接盖的小得多的手机。

1991年以后引入了GSM标准。 他的服务包括呼叫转移，呼叫限制，等待，呼叫保持和全球漫游。



1993年，摩托罗拉推出了StarTAC，这是一部类似于《星际迷航》对讲机的电话。 翻盖式手机将在将来成为该品牌的名片，但是还会有其他形式：摩托罗拉V70可以使盖子旋转一圈，并且第一批Android智能手机将成为带有QWERTY键盘的滑盖。



2006年，旗舰产品摩托罗拉RAZR V3i在Svyaznoy的销量排名第二。 这款手机已经装有1.3百万像素的摄像头和710 mAh电池。 用





公司的第一批智能手机之一-摩托罗拉ROKR E6-正在运行Linux操作系统。 它具有2.4英寸触摸屏，2百万像素的摄像头，能够记录视频和微距摄影，支持最大32 GB的SD存储卡，以及内置的HTML浏览器和文档查看器。



摩托罗拉开始使用Android 1.5 Cupcake上的传播器CLIQ（欧洲-DEXT）开始Android时代。 这个小工具被称为“社交网络的第一部手机”，并为其添加了MOTOBLUR服务。 该手机最初是为VKontakte，Odnoklassniki，Facebook，Twitter的用户设计的。 实际上，外壳程序已成为社交网络的聚合器，使您可以查看警报并直接从屏幕发送消息。

滑盖形状的小工具装有一个3.1英寸的屏幕，分辨率为320x480像素，Qualcomm 7201a处理器，256 MB的RAM和全尺寸的QWERTY键盘。





接下来是Android 2.0上的Motorola Droid。 该设备基于德州仪器OMAP 3430构建，时钟频率为500 MHz，接收512 MB内部存储器和256 MB RAM。 这个小工具配备3.7英寸的屏幕，分辨率为480x854像素，并带有5百万像素的摄像头。



摩托罗拉于1993年在俄罗斯开设了代表处。 在2000年代，该国的雇员人数达到1000人。 自2011年1月1日起，由于公司失去了在竞争对手中的俄罗斯市场份额，摩托罗拉移动事业部关闭。 2012年，谷歌以125亿美元收购。

Moto是最早的Android智能手机制造商之一，当时是与美国三星和HTC并驾齐驱的三大领导者之一。 但首先，该公司对摩托罗拉拥有的17,000项专利感兴趣。 在此之前，Google拥有500项专利，其中大多数与搜索有关。

2014年，Google以25亿美元的价格出售了 Motorola Mobility。 在此之前，摩托罗拉之家已与其分离，并以23.5亿美元的价格出售。 但是谷歌保留了大部分专利。

新品牌拥有者联想长期以来一直在想是否将Moto品牌归还俄罗斯。 2015年，在柏林举行的IFA 2015展览会上，我与公司的高层管理人员进行了交谈，他们对这个问题非常感兴趣：我们国家对摩托罗拉的记忆是多少。 因此，2016年3月，俄罗斯出现了4款智能手机发售：Moto G，Moto Z Play，Moto Z和Moto X Force。 6月27日，莫斯科推出了新的廉价智能手机和中档设备。
