Pengembangan game kultus STALKER 2 dilanjutkan



Pengembangan game STALKER 2 dilanjutkan! Ya, ya, ya, apa yang ditunggu-tunggu oleh banyak dari kita! Banyak, tidak lagi berharap bahwa permainan STALKER 2 akan melihat cahaya, tetapi tampaknya penjualan Cossack 3, memungkinkan untuk menemukan dana yang diperlukan untuk melanjutkan pengembangan.

Dilihat oleh tenggelamnya kode sumber STALKER 2, beberapa tahun yang lalu, gim ini akan multi-platform dan memiliki versi untuk PC, Sony Play Station, dan XBox. Semoga saja penambahan multi-platform tidak akan menghancurkan atmosfer unik game ini.


Dilihat oleh situs web resmi game, game akan dirilis pada 2021, tetapi orang-orang yang tahu sejarah game STALKER sangat menyadari bahwa tanggal rilis dapat berulang kali dijadwal ulang :)

Saya mengundang Anda untuk berkomentar untuk membahas bagaimana kelanjutan dari permainan kultus itu.

UPD 1:
Ada petunjuk dari Sergei Galenkin, kepala departemen penerbitan Epic Games, bahwa game tersebut akan menggunakan mesin UE4, tetapi saat ini tidak ada konfirmasi / bantahan informasi secara resmi.
