Cara mengompres Windows 10
Pada Juli 2016, Microsoft merilis untuk Windows 10 mekanisme kompresi Compact OS baru untuk utilitas baris perintah compact.exeyang mengkompres file sistem operasi secara langsung selama operasinya. Mekanisme kompresi baru memiliki banyak fitur menarik lainnya, tetapi secara umum mirip dengan program WIMBoot di Windows 8.1 Update 1.

Compact OS membebaskan setidaknya 2 GB ruang disk. Tidak terlalu banyak, tetapi pada SSD kecil ini terlihat.

Program berjalan dari baris perintah dengan hak administrator. Ini berarti bahwa setelah menjalankan cmd.exe Anda harus menekan kombinasi tombol Ctrl + Shift + Enter.

Sebelum mengompresi Windows 10, masuk akal untuk mengetahui apakah prosedur ini akan menghasilkan hasil yang tidak nol. Sistem mungkin sudah dalam kondisi terkompresi, jadi tidak diperlukan prosedur tambahan. Untuk menentukan status binari OS saat ini, jalankan perintah berikut:

  • compact.exe / q



Perintah akan menampilkan informasi tentang keadaan saat ini: berapa banyak file dikompresi atau tidak dikompresi, tingkat kompresi, berapa byte dalam wadah file terkompresi. Jika level kompresi mendekati 1.0, maka menggunakan program akan efektif.

Dalam kondisi tertentu, sistem terkompresi dapat sedikit menurunkan kinerja, tetapi pada sebagian besar komputer itu tidak akan terlihat. Perlu diingat bahwa selalu mungkin untuk mengembalikan file ke keadaan normal jika Anda melihat ada masalah kinerja. Bagaimanapun, Anda disarankan untuk membuat cadangan sebelum mengompresi file sistem.

Untuk menjalankan prosedur, jalankan perintah berikut:

  • compact.exe / CompactOS: selalu



Mengompresi semua file harus memakan waktu beberapa menit, meskipun pada beberapa sistem mungkin butuh 20 menit atau lebih. Insider Build terbaru membebaskan lebih dari 2 GB.

Untuk mengembalikan sistem ke kondisi semula:

  • compact.exe / CompactOS: tidak pernah

Prosedur sebaliknya agak lebih cepat.

Program yang nyaman dan gratis yang mengubah file audio dan video ke dalam format yang berbeda.
Ada beberapa program yang memenuhi persyaratan dasar:

  • open source
  • penggunaan gratis;
  • dukungan untuk banyak format file;
  • lintas platform;
  • ringan.

Mungkin yang paling nyaman adalah dua: Handbrake dan ffmpeg (misalnya, dengan antarmuka grafis WinFF ). Sebagai contoh, pemutar video VLC gratis yang populer dapat mengonversi file, tetapi dalam kebanyakan kasus lebih nyaman menggunakan program khusus.

Handbrake berjalan di Windows, Mac, dan Linux. Versi terbaru untuk Windows saat ini adalah 0.10.5. Hampir semua file multimedia (atau folder dengan file), DVD atau BlueRay, jika tidak ada perlindungan salinan di sana,

diterima sebagai materi sumber . Format File Target


  • Wadah : .MP4 (.M4V) dan .MKV
  • Penyandi video : H.265 (x265 dan QuickSync), H.264 (x264 dan QuickSync), H.265 MPEG-4 dan MPEG-2, VP8, Theora
  • Pengkode Audio : AAC / HE-AAC, MP3, Flac, AC3, Vorbis
  • Audio pass-through : AC-3, E-AC3, DTS, DTS-HD, TrueHD, AAC, trek MP3



Program ini nyaman digunakan untuk mentranskode DVD atau file yang diunduh untuk dilihat di perangkat selain PC. Handbrake memiliki profil penyandian internal untuk perangkat berikut:

  • iPod
  • iPhone dan iPod touch
  • iPad
  • AppleTV
  • AppleTV 2
  • AppleTV 3
  • Android
  • Tablet Android
  • Windows Phone 8

Program ini memiliki banyak fitur tambahan.

  • Pemilihan fragmen (bagian dari file sumber untuk penyandian)
  • Pemrosesan batch dan permintaan antrian
  • Penanda bab
  • Teks (VobSub, Teks Tertutup CEA-608, SSA, SRT)
  • Bitrate konstan atau kualitas konstan (bitrate variabel)
  • Mendukung VFR, CFR dan VFR
  • Filter Video untuk Transkode: Deinterlacing, Decomb, Denoise, Detelecine, Deblock, Grayscale, Cropping, Scaling
  • Pratinjau langsung

Sebagian besar kode sumber untuk Handbrake didistribusikan di bawah GNU GPL gratis, versi 2. Bagian individu berada di bawah Klausul BSD 3.

Fungsionalitas serupa disediakan oleh program WinFF lain gratis , yang tampaknya ditargetkan pada pengguna Linux (lihat daftar distribusi ), meskipun ada versi untuk Windows dari Windows 95 ke Windows 8. Untuk itu, antarmuka dalam bahasa Rusia, Ukraina dan banyak bahasa lainnya dikembangkan.



Cara memilih VPN gratis
Untuk pekerjaan anonim di Internet dan menghindari penyensoran negara (memblokir situs), mudah untuk menggunakan VPN gratis. Salah satu opsi paling populer adalah FreeOpenVPN (melalui aplikasi klien OpenVPN).

Ngomong-ngomong, pada 31 Juli 2016 Kaspersky Internet Security 2017 (anti-virus populer di Rusia) dirilis, termasuk klien VPN. Dalam mode bebas, ini hanya berfungsi melalui server di Rusia dan memungkinkan Anda untuk mentransfer 200 MB per hari. Tentu saja, ini adalah penawaran yang agak buruk, tetapi jika Anda sudah menginstal KIS 2017, maka opsi ini harus dipikirkan. Hanya pengguna KIS yang tidak boleh mengaktifkan fitur VPN berbayar dalam program, karena ada banyak alternatif gratis.
