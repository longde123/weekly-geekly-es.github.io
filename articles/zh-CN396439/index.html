为什么褐矮星这么呆板？

哦，这些星星...核聚变的巨型火箱。它们（像我们的太阳一样）燃烧氢，将其转变为氦气（和其他元素），并在此过程中发出大量可见光和能量。



但是，如果您看着棕色的矮星，它将看起来不像正常的恒星（从主序列来看），就像我们的太阳一样。褐矮星不能给出足够的压力来将燃烧的氢气转化为氦气。他们只能将氢转化为氘。

让我们看一下这些过程之间的区别。氢原子的核只是一个质子，质量为938.272 MeV / c 2（我使用这些单位是因为它们很容易转化为能量，足以乘以c 2，因为E = mc 2）氘核（氘核）包含一个质子和一个中子（质量为939.566 MeV / c 2），但是由于它们相互连接，氘核的总质量略小于两个质子的质量1875.613 MeV / c 2。因此，当两个质子合并，结果出现氘核时，释放出0.931 MeV的能量。一秒钟-这并非完全正确...他们还需要保存量子数，例如电荷和轻子数。因此，它们需要产生一个正电子和中微子：但是不用担心，正电子会在具有电子的恒星中an灭，并为您增加0.511 MeV的能量，并将每个合成反应的总能量提高到1.442 MeV。



在大多数恒星中，这是链反应的第一步，该反应导致产生氦4，氦的总质量为3727.38 MeV / c 2。这意味着每合成4个氦4的质子，我们将获得26.73 MeV的能量（通过包括两个电子resulting灭并释放额外的能量的正电子）。这就是从氘和质子中获取氦4的方式：
