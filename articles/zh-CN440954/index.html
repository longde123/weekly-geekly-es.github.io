资料收集


这次，我们将尝试收集有关一段时间内文章浏览量的统计信息，不仅分析评分，还分析喜欢和不喜欢的次数。 首先，我们需要源数据-我们获得了文章列表。

我们有什么？ 当然，我们没有任何“内部人员”信息，我们将“从开源”获取数据。 用户访问主页时，可以看到 50页列表page1 - page50 。 我们将分析所有这一切。

像上一部分一样，可以从字符串中手动选择标签片段，但是要进行更改，我们将使用现成的BeautifulSoup库。 它提供对网页和各个组件的DOM模型的访问，即 我们需要什么。

与前面的情况一样，我们仔细查看页面的源代码，发现文章存储在“文章”块中，发布时间存储在“ post__time”类的元素中，文章的参数存储在“页脚”块中（借此机会，我向程序员表示敬意）所有数据的逻辑和可理解结构的站点：)。

一页的解析代码如下所示（在第一部分中介绍了一些缺少的功能，例如get_as_str以从URL获取python字符串）： 

locale.setlocale(locale.LC_TIME, 'ru_RU.UTF-8') def get_page_data(page_link): data_html = get_as_str(page_link) soup = BeautifulSoup(data_html, 'html.parser') values = [] articles = soup.findAll('article', attrs={'class':'post post_preview'}) for a in articles: t = a.find("span", {"class": "post__time"}) date = str_to_date(t.text) views, votes, votes_up, votes_down, comments, link = 0, 0, 0, 0, 0, "" link = a.find("a", {"class": "btn btn_x-large btn_outline_blue post__habracut-btn"}) # Remove #habracut from link link_url = link['href'].split('#')[0] footer = a.find("footer", {"class": "post__footer"}) spans = footer.findAll('span') for sp in spans: value = Str(str_to_ascii(sp.text)).to_int() if sp['class'][0] == 'post-stats__views-count': views = value if sp['class'][0] == 'voting-wjt__counter': votes = value html = Str(sp) # 'title="  12: ↑11  ↓1">+10</span' => 19, 3 votes_up = html.find_between('↑', ' ') votes_down = html.find_between('↓', '"') if sp['class'][0] == 'post-stats__comments-count': comments = value date_str = date.strftime("%Y-%m-%dT%H:%M:%S.000") print "{},{},views:{},votes:{},up:{},down:{},comments:{}".format(date_str, link_url, views, votes, votes_up, votes_down, comments) values.append([date, views, votes, comments]) return values

如您所见，一切都很简单。 我们在页面上获得文章列表，每篇文章都有发布时间，观看次数，评分，喜欢和不喜欢的次数。 足够分析。

但这还不是全部。 文章的发布时间可以以日期和时间（“ 2019年2月3日，18：27”）或“昨天在23:25”或“今天在22:46”的形式存储。 您需要将此类字符串转换为常规日期时间： 

 def str_to_date(s): s_ = s.lower().encode('utf8') str_date = "" d = None if " " in s_: # '  10:13' => 2019-02-18T10:13:00.000 date = datetime.datetime.now() time = datetime.datetime.strptime(s_.replace('  ', ''), '%H:%M') d = date.replace(hour=time.hour,minute=time.minute,second=0,microsecond=0) elif " " in s_: # '  10:13' => 2019-02-17T10:13:00.000 date = datetime.datetime.now() - datetime.timedelta(days=1) time = datetime.datetime.strptime(s_.replace('  ', ''), '%H:%M') d = date.replace(hour=time.hour,minute=time.minute,second=0,microsecond=0) else: # '14  2019  15:51' => 2019-02-14T10:15:51.000 d = datetime.datetime.strptime(s_, '%d %B %Y  %H:%M') return d

首先，将所有数据加载到数据帧中，并添加程序生成的列，以获取视图/计数，视图/注释以及日期和时间的不同表示形式。 

 def to_int(s): # "bookmarks:22" => 22 num = ''.join(i for i in s if i.isdigit()) return int(num) def to_hour(dt): return int(dt.strftime("%H")) def to_dayofweek(dt): return dt.weekday() def to_dayofweekstr(dt): return calendar.day_name[dt.weekday()] df = pd.read_csv(log_path, header=None, names=['datetime', 'link', 'views', 'votes', 'up', 'down', 'comments']) dates = pd.to_datetime(df['datetime'], format='%Y-%m-%dT%H:%M:%S.%f') day_names = dates.map(to_dayofweekstr, na_action=None) days = dates.map(to_dayofweek, na_action=None) hour = dates.map(to_hour, na_action=None) views = df["views"].map(to_int, na_action=None) votes = df["votes"].map(to_int, na_action=None) votes_up = df["up"].map(to_int, na_action=None) votes_down = df["down"].map(to_int, na_action=None) comments = df["comments"].map(to_int, na_action=None) df['datetime'] = dates df['day'] = days df['dayofweek'] = day_names df['hour'] = hour df['views'] = views df['votes'] = votes df['up'] = votes_up df['down'] = votes_down df['votesperview'] = votes/views df['commentsperview'] = comments/views df['votesratio'] = votes_up/votes_down df['comments'] = comments print("Dataframe:") print(df.head()) print()
