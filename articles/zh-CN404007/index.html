GPD Win-探索对角线为5.5英寸的微型笔记本电脑，专为游戏和模拟器设计

我喜欢微型键盘设备。 自从第一款Psion 5MX以来，我就一直有这种爱。 从那时起，仍然有诺基亚E90等。 las，随着时间的流逝，带有键盘的小型设备变得越来越少了，我一直没有停止爱他们（a，也许也可以这样说）。 因此，我无法超越GPD胜利。 看起来像这样。



它的屏幕对角线只有5.5英寸，适合放在夹克的内袋中，但它的内部是Windows上最完整的上网本。

屏幕： 5.5英寸，高清1280 x 720（267 ppi），16：9，电容式，多点触摸，康宁大猩猩玻璃3，单元内全层压技术
处理器： Intel Atom X7 Z8700，64位四核，1.6GHz〜2.4GHz;
显示卡： Intel HD Graphics 405（200MHz〜600MHz）
内存： LPDDR3-1600，4GB
内置内存： eMMC 4.51，64GB
存储卡支持：高达128GB的Micro SD
无线接口： Wi-Fi：802.11 a / ac / b / g / n WLAN，蓝牙：V4.1
连接器： 1 x USB Type-C 3.0、1 x USB 3.0、1 x TF卡插槽，1 x HDMI C型，1 x 3.5mm
内置游戏板：两个模拟操纵杆+ A / B / X / Y按钮+两侧上表面的两个Shift键+键盘上的其他L3 / R3按钮+开始/选择按钮
键盘： 67个标准PC键+ 10个附加键
其他按钮：音量±； 电源按钮； 游戏手柄模式开关（Dinput /鼠标/ Xinput）
声音系统： Realtek ALC5645
扬声器/麦克风：单声道扬声器和内置麦克风
主体材料： ABS + PC
传感器： G传感器和霍尔传感器
电池： 6700mAh，工作时间约6〜8小时
尺寸和重量： 155 * 96 * 20毫米，300克


如您所见，技术规格（大小除外）非常类似于Windows上的中文平板电脑或Cherry Trail上的上网本。 另外，还有一个带有模拟摇杆的游戏控制器，由于我不喜欢计算机游戏，因此我并未对其进行特别的研究。 相反，我以各种不同的模式测试了GPD Win。 但是首先，让我们仔细看一下。

供货范围


装在一个黑色的纸板箱中。


在有用的包装中，有充电器，电线，微型书本和耳机。



他们还在屏幕上贴了一个特殊的标签（考虑到大猩猩玻璃贴在屏幕上并且关闭了-很奇怪），以及一条无用的指令，其中包含各种调整建议，由于某种原因它们会断开链接。



我不太明白为什么将耳机放在盒子里，但他们没有把盒子放进去。 老实说，它会有用得多。

设计，人机工程学


关闭时，看起来很无聊又正式，外面没有徽标。 灰色哑光塑料与Psion 5MX中的塑料类似，完全没有标记，也不会刮花。



屏幕具有良好的视角，眩光适中。



色彩还原还不错，最大亮度足以在户外工作，但不能在明亮的阳光下工作。



我怀疑框架的宽度很大，部分原因是设备呈圆形的“残留”形状。



微型笔记本电脑的底视图。 有微型塑料支脚，但是设备在桌子上不是很稳定，根本不适合此类工作。

底盖中间的东西是一个有趣的风扇模式开关。 有一个被动操作选项，具有自动调整功能和最大值。



盖子上也可见标记。 如您所见，GPD并未隐藏其起源，一切都在中国科技大都市深圳制造。


在右侧，只有散热器格栅，仅此而已。



同样，东西在左边。



前面是另一个散热器格栅，右边是一个二极管，用于显示系统状态。



在睡眠模式下，发出蓝色光。 不是很明亮，也不烦人。



让您大致了解哪个设备是大小。 我的手不是很大，但是可以方便地伸到各个角度，操纵杆和键盘。



该设备的厚度非常大，可以看出它是为工作而制造的，并不是为了夸口而设计。 好吧，不是为了工作，而是为了游戏。



所有端口，连接器和其他键都集中在背面。 触发器的边缘（在“鼠标”模式下左右键起鼠标键的作用）有一个USB C型接口，用于充电，一个全尺寸USB 3.0，一个microSD插槽，一个耳机插孔以及一个完整的miniHDMI 1.4。 说出您喜欢的东西，但是有了婴儿的连接器，一切都会好起来的，在许多笔记本电脑上，它们的数量很少。



触发器的视图更近。 他们很舒服，不要自发地按。



另一方面。



扬声器位于右下角的背面。 他是一位，而且是单音。 开发人员说，他们仍然不希望在这样的婴儿上发出正常的声音，在旅途中最好戴着耳机看电影并玩耍。



当然，键盘不是全尺寸的，但是很舒适。 特别是，它具有足够的键数以便输入所有俄语字母，对此感到高兴。 密钥不是很大并不令人鼓舞，因此，除了“翻滚”（即用拇指）外，打印将无法进行。



靠近键盘左侧查看。



键盘右侧是其他键-“开/关”按钮和Microsoft游戏服务的启动，音量控制，L3和R3（附加的“向右”和“向左”按钮），选择，开始，暂停和滚动锁定。



X，Y，A，B按钮位于右侧的模拟摇杆旁边，在“鼠标”模式下，默认情况下它们不执行任何操作，但是如果您愿意，可以使用各种程序为它们分配任何宏。 不要忘记它使用完整的Windows 10。



如早期游戏机的游戏手柄上，十字线位于左模拟杆旁边。 在鼠标模式下，它用作键盘上的“箭头”。



屏幕下方是确定操纵杆操作的物理开关：操纵杆可以像鼠标一样充当游戏板（D-Input），也可以在X-Input模式（如X-Box）中工作。



该设备通过常规的“智能手机”充电进行充电，但电流较大，最高可达2.5安培（5V）。 同样的收费，例如Chuwi Lapbook 15.6以及许多中文平板电脑。



总计-该设备在外观上非常不寻常，谨慎，但对我来说似乎很有趣。 可以看出，尽管尺寸很小，但开发人员还是怀着极大的热情着手设计，并尽力使一切变得尽可能方便。

性能测试


我没有在众多游戏中测试GPD Win，最后，许多人已经这样做了。 兼容性表可以在Internet上找到，以了解将要发生的事情，什么不知道，实际上，我们可以假设在Cherry Trail平板电脑上可用的所有功能都带有几个兼容性标签（ 1，2 ）。

预期的三维性能-无需额外的超频。



结果是合乎逻辑的-基本上只有Ice Storm才能使用此笔记本电脑。



但是PCMark测试已经变得更加有趣了。 原则上，由于内存很大，因此它是休闲工作的很好选择。



请注意，在相当沉重的Creative测试中，温度非常稳定，频率位于天花板上，这表明冷却系统组织良好。



在工作测试中，有更多的轻型任务，处理器完全是冷的。



存储，嗯，eMMC，从中获取什么。 不幸的是，无法达到SSD的速度，整体性能是相当的。



好吧，而且，正如我在上面说的那样，控制台不会过热（当然，散热效果最好）。 如果减小它，则它会节流，并以最快的速度处于加速状态。 如此稀有，但粉丝却相当吵。



设备的电池寿命也让我印象深刻。 就互联网的平均亮度而言，我花了大约6个小时。 PCMark家庭电池测试也是如此。

设备需要3个小时的长时间充电，但是您可以使用任何电池或移动电源工作。

合计


这是一个非常有趣的设备，现在可以在Gearbest上以折扣价购买 ，价格约为360美元。 还可以尝试GBGPD优惠券（$ 317.99）。 是的，当然，它比具有类似性能的平板电脑贵将近2倍，但是尺寸在这里是独一无二的。 此外，由于端口集丰富，该设备还可以通过连接到监视器在医院中使用。 我没有任何特殊的投诉（除了直接来自外形尺寸的投诉）。

  • 电池-一如既往，取决于要做什么。 火车中的DmC连续将近5个小时，冷却器位于中间位置（在某些地方，您可以将其完全关闭，但外壳明显变热）。 我没有使机器节流，但是我没有注意到温度高于72。 66-68处于负载状态，而冷却器处于中间位置，最大速度为62。 顺便说一句，我现在已经仔细检查了一下-空气从前面吸入，从下方和左侧吹出。 右格栅是单声道扬声器（它将立体声输出到耳机）。 PS1游戏（在第一个寂静岭进行了测试）-超过6小时。 我检查了99％的电量，Wi-Fi已关闭，已连接USB3.0闪存驱动器，亮度为50％；
