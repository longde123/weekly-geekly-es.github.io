Sekolah pengembang Java di Nizhny Novgorod

Halo semuanya!

Kami membuka sekolah gratis untuk pengembang Java pemula di Nizhny Novgorod. Jika Anda seorang mahasiswa pascasarjana atau lulusan universitas, memiliki sedikit pengalaman di bidang TI atau profesi terkait, tinggal di Nizhny atau sekitarnya - selamat datang!



Pendaftaran untuk pelatihan - di sini , kami menerima aplikasi hingga 30 Oktober.

Detail - di bawah potongan.

Jadi detail yang dijanjikan


Pelatihan dimulai pada 5 November. Kelas di sekolah, seperti yang disebutkan di atas, gratis, akan diadakan dari Senin hingga Kamis 18: 00-21: 00, kursus ini dirancang selama 90 jam akademik. Mentor program akan menjadi direktur teknis perusahaan tujuan, arsitek dan pengembang Java, Spring Framework, JS, sistem React / Angular Ilnaz Gilyazov.

Proses Aplikasi


Ini sangat sederhana dan terdiri dari tiga tahap: mengisi kuesioner - wawancara di Skype - menyelesaikan tugas praktis. Setelah menyelesaikan semua tahapan, kami akan mengundang 25 orang untuk belajar.

Secara singkat tentang tingkat kurikulum


Seluruh program dapat dilihat secara rinci di situs web sekolah , dan di sini kami hanya dapat mengatakan bahwa kami akan segera menawarkan lulusan terbaik untuk melanjutkan karir mereka di Rosbank. Apa yang sedang dibicarakan ini? Fakta bahwa pada akhir kursus Anda akan menjadi spesialis yang memenuhi syarat untuk bekerja di salah satu bank terbesar di Rusia.
