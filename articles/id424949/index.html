PHP Digest No. 140 (17 - 30 September 2018)


Seleksi segar dengan tautan ke berita dan bahan. Dalam rilis: PHP 7.3 RC2, Codeception 2.5 dan rilis lainnya, mengetik properti di PHP 7.4, konsep preload kode dari Dmitry Stogov, sebagian dari alat yang berguna, dan banyak lagi.

Berita dan Siaran


  • PHP 7.3 RC2 - Kandidat rilis kedua dikirim sesuai jadwal . Rilis berikutnya diharapkan pada 11 Oktober. Ulasan lain dari inovasi PHP 7.3 .
  • PhpStorm 2018.3.3 EAP - Program akses awal sedang berlangsung, di mana Anda dapat menggunakan PhpStorm secara gratis. Build yang lebih baru telah menambahkan dukungan untuk notasi <generik> untuk koleksi dari PSR-5 PHPDoc, serta persimpangan tipe Foo&Bar . Baca lebih lanjut tentang persimpangan dan penyatuan jenis dari penulis PHPStan .
  • Codeception 2.5 - Pembaruan ini memberikan peluang untuk pengujian snapshot, dukungan untuk menghubungkan ke beberapa database, dan peningkatan lainnya.
  • doctrine / mongodb-odm 2.0 alpha 1
  • nikic / FastRoute - mencari pengelola! Beberapa tahun yang lalu, Nikita Popov membuktikan bahwa perutean bisa sangat cepat di PHP. Izinkan saya mengingatkan Anda tentang artikelnya yang bagus Fast routing yang menggunakan ekspresi reguler .
  • Pertemuan Yandex PHP, 18 Oktober 2018, Moskow - Pertemuan pengembang PHP akan diadakan di kantor Yandex di Moskow.

PHP internal


  • [RFC] Typed Properties 2.0 - Proposal untuk properti yang diketik diadopsi hampir dengan suara bulat: 70 melawan 1! Properti yang diketik akan ditambahkan dalam PHP 7.4.
    Teks tersembunyi 
     class Example { // All types with the exception of "void" and "callable" are supported public int $scalarType; protected Type $type; private ?Type $nullableType; // Types are also legal on static properties public static iterable $staticProp; // Types can also be used with the "var" notation var bool $flag; // Typed properties may have default values (more below) public string $str = "foo"; public ?string $nullableStr = null; // The type applies to all properties in one declaration public float $x, $y; // equivalent to: public float $x; public float $y; }
  • [PR] Upaya untuk mengimplementasikan kemampuan "preloading" - Dmitry Stogov adalah penulis utama proyek PHPNG, yang menjadi dasar untuk PHP 7. Kali ini, Dmitry memperkenalkan konsep yang memungkinkan aplikasi untuk memuat kode sebelum PHP memproses permintaan HTTP.
    Dmitry juga sedang mengerjakan implementasi JIT dalam PHP , yang diharapkan dalam PHP 8 .
