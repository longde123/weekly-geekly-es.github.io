最近，我对LTE路由器进行了对比测试，非常期待无线电模块的性能和灵敏度存在显着差异。 当我将天线连接到路由器时，速度增加了很多倍。 这给了我进行天线比较测试的想法，该测试不仅可以在私人住宅中提供通讯，而且还可以通过电缆连接使其不比城市公寓差。 好了，可以在下面找到结束该测试的原因。 传统上，对于那些想要观看而不是阅读的人，制作了视频。


测试方法


没有常规的结构化方法，您将无法获得高质量的结果，并且该测试的目的是为最快的Internet访问选择最佳的天线。 作为测量标准，选择了Zyxel LTE3316-M604路由器，该路由器在之前的测试中合理地排名第一。 该设备既可以与常规有线提供商合作，也可以通过3G / 4G使用备用通信信道，并且可以在3G和4G蜂窝网络中完全自主地工作。 在我的测试中，仅使用4G网络，因为仅通过该网络传输数据，并且语音流量的负载不会影响此通信通道。

在测试中，我选择了三种不同类型的天线：第一个测试是为了获得清晰的数值，路由器无需外部天线即可工作，仅使用内置天线即可工作。 第二项测试以圆形辐射图连接到天线。 在第三项测试中，使用了平板天线，该天线的辐射方向图与之前的测试中相比更窄。 好吧，第四步是测试狭窄取向的网状抛物面天线。

所有速度测量都是在白天的一个工作日进行的，因此基站的负载最小，下载速度最大。 在每个阶段，都要进行3次测试，并计算下载和上传速度的平均值。 路由器连接到同一BS；根据路由器Web界面中的信号调整天线。
我还制作了每日图表，显示了我所在区域的数据下载和上载速度，该图表完美地展示了Internet用户的工作。 我相信提供商将在BS上获得大致相同的负载图片。 有趣的是，下载速度图显着跳跃，但是返回图实际上没有变化-这表明用户下载的次数超过了提供数据的次数。

总结





对比测试表明，即使没有天线，且海拔高度（距地面10 m）也很好，Zyxel LTE3316-M604路由器仍可以提供可接受的互联网速度。 但是您不会将路由器留在街上，因此此选项适用于公寓，办公室，但不适用于无法用双筒望远镜看到塔楼的地方。

FREGAT MIMO天线适用于由于多种原因而无法在路由器安装现场接收无线电信号的那些人。 这可能是屏蔽墙，位置低，其他干扰。 单个情况下的两根天线将为MIMO技术提供支持，这将提高工作速度。

至于OMEGA 3G / 4G MIMO平板天线，它非常值得展示。 既可以处理直接信号也可以处理反射信号，安装选项很多，增益也很高。 小尺寸不会给您带来很大的损失，并且速度提升非常明显。 如果有3G / 4G信号，您可以放心地使用它，但是信号非常微弱或根本没有。
好吧，PRISMA 3G / 4G MIMO抛物线形网状天线最适合绝望的人，因为通过这种放大和微调到BS的能力，即使移动运营商的基站在几十公里半径内，即使在最偏远的村庄，您也可以建立连接。

结论


此刻，我离开了OMEGA 3G / 4G MIMO天线，开始工作。 由于天线的尺寸决定了它们的状况，因此我不得不在墙上稍微移动安装杆。 使用3米长的电缆和选定的路由器，当BS负载最小时，我看到的速度高达50 Mbit / s。 在现有的BS操作条件下，这倾向于达到75 Mbit / s的理论速度限制：Band3频率-1800 MHz，信道宽度为10 MHz。 但是最主要的是，在距离基站8公里以上的距离上，我设法使速度接近塔附近的速度。 让我举一个例子，说明使用不同频率的无线电信号的覆盖范围。
