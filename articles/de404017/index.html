Einsparungen bei Apple Services

Es ist kein Geheimnis, dass die Preismatrix von iTunes ziemlich groß ist und viele Indikatoren enthält, die unter anderem vom Land abhängen. In "teuren" Ländern sind die Preise für Anwendungen und Musik normalerweise höher. Bevor Sie ein Album in Ihrer nativen Apple Music oder iTunes kaufen, sollten Sie daher die Kosten mit denen anderer Regionen vergleichen.

Schauen wir uns an, wie Sie das kaufen, was Sie in amerikanischem iTunes benötigen.

Der Algorithmus ist etwas komplizierter als in den vorherigen Absätzen:

  1. Zuerst müssen Sie sich von Ihrer Apple ID abmelden.
  2. Stellen Sie sicher, dass Sie eine amerikanische IP-Adresse haben
  3. Finden Sie die kostenlose App im iTunes AppStore und laden Sie sie herunter. Denken Sie vor allem daran, dass Sie die Anwendung herunterladen müssen, nicht die Musik oder das Video. iTunes fragt nach Ihrer Apple ID - erstellen Sie einfach eine neue.
  4. Akzeptieren Sie die Vereinbarung und schließen Sie den Anmeldevorgang ab.
  5. Finden Sie, dass Sie eine kostenpflichtige Anwendung oder Musik benötigen. Wenn der Zahlungsbildschirm angezeigt wird, geben Sie keine Kreditkartendaten ein. Nutzen Sie den Postdienst und suchen Sie eine Adresse in Delaware, um Umsatzsteuer zu vermeiden.
  6. Verwenden Sie zum Bezahlen eine amerikanische iTunes-Geschenkkarte. Sie können es über eBay oder Amazon kaufen. Oder Sie können eine virtuelle US-Kreditkarte kaufen. Es gibt viele Dienstleistungen, die sie verkaufen.
