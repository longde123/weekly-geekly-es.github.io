Fujitsu PRIMERGY RX2540 M4 Revisão e teste do servidor

O rápido desenvolvimento de novas tecnologias geralmente envolve situações em que o equipamento utilizado não lida com as tarefas das empresas. Você precisa atualizar a frota existente de sistemas de servidor ou alterá-los para sistemas mais produtivos. Nesse caso, é necessário avaliar adequadamente as opções para tal substituição, para conhecer todos os prós e contras do equipamento. Decidimos testar o servidor Fujitsu PRIMERGY RX2540 M4 e informar quais recursos ele oferece aos usuários. Neste artigo, falaremos sobre em que consiste o servidor, sobre sua arquitetura e descreveremos o progresso e os resultados dos testes.

Vale a pena contar mais sobre cada tipo de módulo.

O RDX (do inglês. Removable Disk Exchange) é uma tecnologia que melhora a tolerância a falhas e a força da unidade. Este módulo é um disco rígido montado em um caso especial, fornecendo uma fixação fraca do disco. Essa tecnologia evita falhas prematuras do disco rígido devido a vibrações ou quedas de uma pequena altura.

LTO (do inglês. Linear Tape-Open) - tecnologia para gravação de dados em fita magnética. Hoje em dia, pode parecer estranho gravar informações em fita magnética. No entanto, o LTO mostra bons resultados de longevidade. Segundo os fabricantes, a fita magnética pode armazenar informações por até 30 anos (a ferrovia tem uma taxa máxima de 10 anos). Além disso, é mais fácil recuperar dados de uma fita magnética rasgada do que de placas ferroviárias amassadas. Padrão LTO-7 atualmente suportado. Permite o uso de cartuchos com capacidade de memória de até 6 TB, com velocidade de gravação de 300 Mb / se 750 Mb / s com compactação de informações. Embora os planos declarassem o padrão LTO-10. Isso permitirá que você instale cartuchos com capacidade de memória de até 48 TB e velocidade de gravação de 2750 Mb / s.

Teste


Para que os resultados não pareçam valores abstratos, para comparação, apresentamos os resultados dos testes de outros servidores de classe média. Vamos começar com o teste do processador:

1) CPU Queen. A essência do teste é calcular números inteiros. Isso permite que você teste a capacidade do processador de ramificar e prever erros. Para resolver os problemas, o algoritmo Queens Problem é usado. Você pode descobrir mais sobre esse algoritmo aqui .


O teste mostrou 62.251 valores por segundo. O resultado mostra que o processador lida bem com várias tarefas ao mesmo tempo, o que gera um grande ganho quando usado na virtualização.

2) CPU PhotoWorxx - testando o desempenho de operações aritméticas e trabalhando com imagens RGB. Sua essência é a seguinte:

  • Preenchendo a imagem com pixels de cores aleatórias;
  • Rotação da imagem em 90 e 180 graus;
  • Diferenciação de imagem;
  • Conversão de espaço de cores (usada na conversão para o formato JPEG).



O teste mostrou um resultado de 36.443 milhões de pixels por segundo. Isso sugere que o processador lida bem ao trabalhar com imagens.
3) CPU ZLib - um teste do desempenho do processador e do subsistema de memória usando operações inteiras. O teste é feito através da criação de arquivos ZIP. Para fazer isso, use a biblioteca zlib aberta ( mais sobre a biblioteca ). Geralmente é usado para testar sistemas com vários núcleos.


O indicador é mantido em 525,2 MB / s.
4) FPU VP8 - testando o processador compactando vídeo com o codec Google VP8 ou WebM 1.1.0. A codificação é realizada para 1 passagem de um fluxo de vídeo com uma extensão de 1280x720 e indo a uma velocidade de 8192 kbit / s (levando em consideração a qualidade máxima configurada). Os próprios quadros são gerados usando uma variedade de fractais de Julia ( mais ). Qual é a diferença entre os testes do processador chamados CPU e FPU? O fato é que os primeiros processadores podem executar um conjunto de instruções estritamente definido. Para expandir os recursos do processador, foram desenvolvidos coprocessadores que executavam operações matemáticas específicas e retornavam o resultado ao processador principal. Isso permitiu aumentar a velocidade geral do computador. Hoje, os coprocessadores estão integrados no processador central e a borda começou a ficar embaçada. Testes de FPU são testes de "coprocessadores" no processador.


O teste mostrou 6493 unidades. Apesar do fato de muitos coprocessadores serem integrados à CPU, eles não mostram o resultado mais alto. Para trabalhar com o fluxo de vídeo, você deve usar placas de vídeo separadas. Em nosso exemplo, a CPU dedica parte do poder do processador ao processamento de gráficos de vídeo.

5) Leitura da memória - teste a velocidade da transferência de dados da RAM para o processador. Essa é a rapidez com que o processador lê dados da RAM.


O teste mostra que o processador lê dados da memória a uma velocidade de 71320 MB / s.

6) Gravação de memória - um teste da velocidade de transferência de dados do processador para a RAM. Em outras palavras, a que velocidade o processador grava dados nas células da memória.


O teste mostrou que o processador grava dados na RAM a uma velocidade de 51299 MB / s.

7) Memory Copy - um teste da velocidade de transferência de dados de uma célula da memória para outra através do cache do processador. Ou seja, a que velocidade o processador copiará dados de um local de memória para outro.


O teste mostrou que o processador copia dados de uma célula para outra a uma velocidade de 61469 MB / s.

8) Latência da memória - testa o tempo médio de leitura de dados pelo processador a partir da RAM. Se o teste anterior para leitura da memória mostrou a velocidade de leitura, esse teste mostra o tempo médio de atraso.


O teste demonstra que o tempo médio de atraso na leitura é de 85,7 ns.

Pese os prós e os contras


Observe os profissionais:

1) tamanhos de servidor compactos

2) A presença de 16 compartimentos para HDD / SSD, que permite criar matrizes de disco subsistemas à prova de falhas

3) A unidade óptica é uma vantagem significativa. Muitas vezes, há casos em que é necessário restaurar o sistema a partir de discos ópticos. O motivo é que a unidade óptica é sempre detectada pelo sistema e os drivers são necessários para vários barramentos (por exemplo, USB 3.0) ou a inicialização pode falhar na inicialização do sistema.

4) Conexão dos módulos RDX e LTO. Eles permitem que você armazene informações por muito mais tempo do que as unidades normais de HDD e SSD.

5) A presença de 2 fontes de alimentação, o que aumenta a resiliência do servidor em caso de falta de energia ou falha de uma das unidades. Essa é uma vantagem muito importante, pois muitos servidores são lançados com uma única fonte de alimentação.

6) Placa-mãe com ampla conectividade:

  • Suporte para 2 processadores.
  • 24 slots para RAM com capacidade total de até 3 TB.
  • 6 slots PCIe, que permitem adicionar um adaptador de vídeo, conectar adaptadores de rede adicionais e muito mais.

7) A presença de um controlador de rede operando a uma velocidade de 10 Gbit / s. Deve-se notar que o controlador contém 2 portas, o que aumenta sua tolerância a falhas e possibilita o uso do balanceamento de tráfego.

8) Um impressionante poder do processador para tarefas complexas e acesso à memória de alta velocidade, o que é mostrado nos testes Queen CPU, Memory Read / Write / Copy.

Agora vamos falar sobre os contras:

1) Controlador RAID incorporado. Seu sinal de menos se manifesta em um pequeno número de portas. Se você possui compartimentos com 16 discos, precisará adquirir um controlador adicional para usar todos os discos disponíveis ao mesmo tempo.

2) Desempenho fraco no fluxo de vídeo e no teste de aceleração de vídeo. Toda a carga cai no processador central.

Para resumir, vale ressaltar as boas características em termos de potência do processador instalada e RAM, bem como o acesso de alta velocidade a ele. Uma grande quantidade de espaço em disco também é uma vantagem. Tudo isso permite que você use o servidor como uma hospedagem de serviços com uma grande quantidade de dados (bancos de dados, correio corporativo, serviços em nuvem), virtualização e muito mais.

As desvantagens incluem o controlador RAID embutido para 8 portas (com 16 compartimentos de unidade) e a falta de uma placa de vídeo para trabalhar com aplicativos gráficos modernos que usam modelos 3D e aceleração de vídeo. Em geral, se necessário, a placa-mãe permite instalar controladores RAID adicionais, placas de vídeo, controladores de rede e outros dispositivos. O servidor "pronto para uso" mostra resultados muito bons e, quando usado em tarefas específicas, pode ser atualizado e atualizado.
