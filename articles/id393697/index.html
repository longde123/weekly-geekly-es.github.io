Windows 95 diluncurkan di Apple Watch



Sistem operasi Windows 95 yang baru saja dimulai. Ini diluncurkan di browser, di berbagai smartphone, menggunakan emulator - dan di jam tangan pintar dengan Android Wear. Sekarang giliran Apple Watch. Bahkan, spesifikasi jam - prosesor 520 MHz, 512 MB RAM, 8 GB memori utama, beberapa kali melebihi persyaratan yang disarankan untuk menginstal OS ini. Karena itu, pengembang Nick Lee memutuskan untuk mencoba menjalankan Windows 95 pada perangkat ini.

Hasilnya, semuanya berubah, dan sistemnya interaktif, Anda dapat bekerja dengannya, dan tidak hanya menonton layar boot, seperti yang biasanya terjadi dalam kasus tersebut. Namun, untuk menginstal perangkat lunak pihak ketiga pada arloji, pengembang harus mengotak-atik, melewati batasan Apple. Akibatnya, setelah memodifikasi lingkungan pengembangan aplikasi untuk WatchOS, semuanya berjalan lancar.



Tentang bagaimana hal ini dapat dilakukan, pengembang merekomendasikan untuk melihat posting rekannya ini. Nick memposting kode sumber di GitHub .



Komplikasi lain adalah tampilan jam mati jika tidak digunakan. Nick bahkan harus datang dengan perangkat khusus yang mengaktifkan mahkota digital arloji untuk mengaktifkan tampilan. Perkembangan terakhir ternyata menarik - semua menu berfungsi, banyak fungsi sistem juga aktif. Benar, karena fakta bahwa semua ini ditiru, OS boot untuk waktu yang sangat lama - sekitar satu jam. Dan karya Windows 95 di Apple Watch tidak bisa disebut supercepat.
