HP iPaq hx4700：五分钟到现在

“很抱歉，但是为什么需要它？” 这是老式手持计算机的卖方。 我想，真的，为什么？ 怀旧？ 渴望回到草更绿的时代？ 想在Habr上发表文章吗？ 老实说，我是2005年买的，2009年又卖出了好价钱（足够用于新的诺基亚E63），对不起，我想恢复它。 很酷的设备。 是的，卖方回答，他是最好的！ “我有七个 ！”

然后我意识到，在以Stepan Plyushkin命名的收藏家的深渊中，我仍然很遥远。 七！ 怎么了 哦好 2004 Pocket PC几乎处于出厂状态。 卖完所有设备后，卖家将一个额外的支架，一个皮套，两个存储卡，两个额外的外壳盖在屏幕上，并进行充电。 我了解了所有这些，并在15年前进行了巡回演出。

Windows Mobile是现代iOS和Android可以访问的移动操作系统。 在千分之二的中间，她短暂地退出了早期智能手机市场的前三名 ，以在Symbian和Blackberry OS的冲击下输给新手。 今天是对设备的评论，我曾经在该设备上重读了很多书籍，回顾了许多旅行中用于汽车导航的电影。 我记忆中的第一台设备真正实现了通用，但同时在几乎所有方面都不如现代智能手机。



大概在2018年，已经需要进行一项重要的澄清。 Windows Mobile上的千分之二的设备被分为“手持计算机”和“智能手机”，或者由于某种原因，愚蠢的术语“通信器”被广泛使用。 可能是因为没有基于触摸屏的智能手机被称为基于Symbian的诺基亚设备而使用。 也就是说，后来已经在“燃烧平台”的泛滥过程中，但最初没有。 是的 因此，Windows Mobile不是智能手机，而是通信器，因为带有触摸屏。...不包括Windows Mobile上的智能手机：特殊版本的OS中没有触摸屏的设备。 2003年底，首款此类设备是Motorola Mpx200手机。

是的，我已经澄清了，高质量。 好吧，对于这篇文章，将智能手机和PDA分离为智能手机更为重要：后者显然被剥夺了电话模块。 我还设法在Windows Mobile上使用了该通信器，但在零的中间，它们是具有令人怀疑的消费者品质的设备。 它们还无法使它们变薄，即使按今天的标准，只有小的四英寸屏幕，它们还是很大的。 现在，您不会对使用铲铲式电话的任何人感到惊讶，而“内置电话的移动式计算机”的发展是在经典移动电话小型化的趋势之前。 简而言之，您离开家门，需要打电话并获得最新的千美元通讯员，而这些家伙很遗憾地看着您：好像您很不幸，把这样的平底锅放在耳边。



然后，我们当然可以：更高版本的通讯器，例如HTC HD2 ，已经类似于我们今天所谓的智能手机。 在某些时候，“简单的掌上电脑”并不是从市场上消失的：它们已经变成了面向初学者的廉价设备。 HP iPaq hx4700于2004年发布，是“预供电” PDA（为商人提供的昂贵设备）的最后代表之一，他们试图在当时投入最大的技术和款式。 让我们看看他，让他看看我们。



属于Hi-End类的现代Pocket PC应该具有绅士的最低标准，即：频率至少为520 MHz的处理器，RAM-至少为64 MB，两个内置无线适配器，最重要的是-扩展为640x480像素的屏幕

HP iPaq hx4700 Pocket PC的评论， Mobile-Review.com ，2005年2月4日。


当然，在2005年，我没有在商店购买这种PDA：我不再是一个贫穷的学生，但也不那么富裕。 我买了一个小beu，但它仍然不便宜。 然后，一部新书的价格为600美元，或者如其中一篇评论所写，为“约17,700卢布”。 尽管购买成本高昂，但该设备的一系列品质仍有望节省一些费用。 金属外壳暗示了耐用性。 两个存储卡插槽-很方便，您仍然可以购买体积更小的便宜卡。 屏幕为VGA，处理器为高端Intel，即使在ARM上有一行处理器，其频率为624兆赫。 在很长的时间（将近三到四年）内，此频率在移动设备上始终保持可访问性，直到具有800 MHz处理器甚至1 GHz的设备都可以使用。



总的来说，在未来许多年中，这是一个合理的购买。 按照今天的标准，该设备肯定看起来已经过时了：屏幕周围的巨大框架LCD显示器（厚度不再令人印象深刻）很厚。 同时，在大多数塑料竞争对手中，iPaq看起来不错。 该模型中最具争议的“创新”是用一个小巧但功能齐全的触摸板代替传统的导航按钮或操纵杆。 有了它，就可以滚动显示文本，特殊的轻触区域可以上下替换常用的按钮，而单独的模式可以将其变成仅一个触摸板，而鼠标光标出现在屏幕上。



即使这种设计工作正常，也几乎不可能在游戏中进行正常控制：通常使用侧面的物理按键代替触摸按钮来启动应用程序。 问题是几乎每个人都有这个触摸板越野车，传递混乱的点击声并滚动到PDA。 有人甚至剥了皮。 我必须通过仪式切断联系方式以编程方式或永久性将其关闭。 在2005年，我也很不幸，在2018年，我很幸运：在我刚购买的副本中，触摸板有效。 既然可以了，那就放一些软件。

手中的计算机感觉非常彻底，但绝不舒适。 原因是形状太简单，宽度也很大。 另外，使用PDA机械手（在这种情况下，触摸板起着作用）在我们看来不舒服。

认真地说，SPB和其他类似制造商的软件使Windows Mobile的使用更加方便。 不久之后，该公司发布了一个完整的外壳程序，该外壳程序几乎完全替代了Windows Mobile界面，关于该界面，我的邮件中也有一封信，该信件本身（对于那些知道的人）几乎没有历史价值：

导航优势
PDA的购买与我的第一辆汽车的购买同时发生。 Windows Mobile似乎是第一个适用于日常使用的制图软件的平台，而不仅仅是欣赏其进展。 在2009年，我将该设备卖给了一名越野爱好者，后者可能会将OZI Explorer放在其上，并上传了总参谋部地图以导航到从未有过道路的地方。 据我所知，该专业软件向Android的迁移被延迟了，因此对于那些正在研究中的人来说，配备或多或少强大的熨斗以及屏幕不会在阳光下刺眼的熨斗hx4700对于过时的人来说是一个不错的选择。 iPaq中没有内置的GPS，但是您可以通过蓝牙连接外部接收器。 在某些情况下，这种结合提供了更自信的接待。



我将GIS Russa程序放在PDA上-从严格意义上讲，它不是导航软件，而是向您显示地图上的位置和路径。 这已经很有用，我特别喜欢程序中的传感器模式。 但是，在长途旅行中，他有动力走得更快，以使平均速度不会下降。



在GIS Russa周围，围绕诉讼发生了某种争议，这终结了已经开始的集体工作，即绘制了我们（不仅是）国家各个角落的“民间”地图。 据我所知，该程序的部分代码后来被Navitel继承-那里肯定有类似的传感器。 我的电报中也有一些GIS Russa的屏幕截图和印象。

在铁人的指示下进行真正的导航时，那些日子我仍然使用现有的iGo。 我尝试了Garmin导航仪-在其中惊奇地发现了一张非常详细的地图，上面有乡间小路和庭院车道。 这个细节让我很久以前就去了沃罗涅日地区。 出于某种原因，导航员认为这条土路比平常的公路短半公里，非常凉爽，因此您一定要去那里。 起初它很有趣，然后又很吓人，然后我到达了村庄，当地人惊讶地看着我，好像他们第一次看到一辆自动驾驶的马车一样。 然后，我突破了杂草丛生的灌木丛，看到了半个小时前开车的那条路。

我只有五个人，所以我决定与真正拥有画笔的朋友一起测试hx4700的人机工程学。 结果令人失望：即使在大手掌上，中共也不太舒服，试图跳出地狱。

失败的
Windows Mobile遭受了Symbian和Palm OS平台的命运：这三个国家并非和平地终结了它们的存在，而是突然被丑闻和戏剧性地终结了。 有趣的是，Palm OS被Windows Mobile所取代，而Windows Mobile在那之后并没有长期存在。 他们都遭受了阻碍进步的艰难遗产。 Windows Mobile在触控笔下方经过了锐化处理，在后来的带有电容屏的型号中，其手指友好的界面坦率地歪了。 Symbian几乎一生都没有触摸屏。 发行到iPhone的顶峰（现在我们将向他们展示如何！）诺基亚N97确实是一次了不起的失败。 Palm OS甚至没有掌握多任务处理。

这三个平台都不具备当今明显的技术，例如，具有用户友好的下载和付款功能的集中式应用程序商店。 hx4700的所有操作经验都显示出了这一缺点：传统上，Windows Mobile程序作为可执行文件分发给计算机，该文件需要启动，连接到PDA等。 可以拔出移动安装程序本身，并跳过此过程的一部分，但这仍然很不方便。 购买了诺基亚智能手机来替代iPaq之后，我通常会碰到一些证书，这些证书每隔第二个星期二晚上在中国的一台特殊服务器上颁发，以设置未经授权的键盘字母切换。 啊！

但是，我不确定Windows Mobile是否需要像Microsoft一样被淘汰-强烈支持完全不兼容的Windows Phone平台。 顺便提一下，支持Windows Mobile和Symbian都是后者，现在看来她本人已被送往博物馆部门。 也许所有这些过时的平台都应该取得如此光荣的终结，但这并不意味着我支持当前的Android霸权。 在二十一世纪二十年代的交汇处提出的替代方案中，到目前为止，只有Maemo / Meego Sailfish OS得以发展。 给他们健康！



这样的动荡已经发生在移动设备市场上，我的PDA不应该受到指责。 他快15岁了，但是更换备用电池后，他的外观和工作像新的一样。 即使我原来的电池也能用，但我当然又买了一个来更换它。 借助Windows Mobile的所有功能，我不禁注意到待机模式下的电池耗电量极低：电池可持续使用数周！

我懒惰地研究了这个设备。 的确，到本世纪末，发烧友看到了用于Windows Mobile的自定义iPaq固件，最高版本为6.5。 而且您仍然可以在其中放置早期版本的Android，甚至是成熟的Linux。 但是我喜欢它的原始形式，它具有早期的OS，最少的程序和安装在通讯座中的相框模式。 最初我根本不打算这样做，我似乎已经购买了第一个真正的博物馆展览品：它并不非常适合现代世界，而是近代美学上令人愉悦的装置。
