如何在Windows 10上完全禁用Windows Defender



Windows 10具有内置的Windows Defender防病毒软件（“ Windows Defender”），可保护您的计算机和数据免受有害程序的侵害：病毒，间谍软件，勒索软件以及许多其他类型的恶意软件和黑客。

尽管内置的安全解决方案对于大多数用户来说已经足够了，但是在某些情况下，您可能不想使用此程序。 例如，如果您要设置的设备不会在线， 如果您需要完成该程序阻止的任务； 如果您需要满足组织的安全策略要求。

唯一的问题是，您将无法完全删除或禁用Windows Defender-该系统已与Windows 10深度集成。但是，可以使用几种解决方法来禁用防病毒软件-使用本地组策略，注册表或Windows设置在“安全性”部分（暂时）。

如何通过Windows安全设置禁用Windows Defender


如果您需要执行特定任务，而无需完全禁用Defender，则可以暂时执行此操作。 为此，使用“开始”按钮中的搜索，找到“ Windows Defender安全中心”部分，然后在其中选择“病毒和威胁防护”。



在那里，转到“防护病毒和其他威胁”部分，然后单击“实时保护”开关。



此后，防病毒软件将禁用实时计算机保护，这将使您能够安装应用程序或执行由于防病毒软件阻止了必要的操作而无法使用的特定任务。

要再次启用实时保护，请重新启动计算机或再次执行所有配置步骤，但在最后一步中打开开关。

该解决方案不是永久性的，但最适合于针对特定任务禁用Windows 10防病毒。

如何通过组策略禁用Windows Defender


在Windows 10 Pro和Enterprise版本中，可以使用本地组策略编辑器，在其中可以永久禁用Defender，如下所示：

通过“开始”按钮，运行gpedit.msc可执行脚本。 将打开策略编辑器。 转到以下路径：“计算机配置”>“管理模板”>“ Windows组件”>“ Windows Defender防病毒”。



双击以打开选项“关闭Windows Defender防病毒程序。” 选择“启用”设置以启用此选项，并相应地禁用防御者。



单击“确定”，然后重新启动计算机。

之后，防病毒软件将在您的设备上永久禁用。 但是您会注意到带有屏蔽的图标将保留在任务栏中-应该保留，因为该图标属于Windows安全应用程序，而不是防病毒程序本身。

如果您改变主意，则始终可以通过重复这些步骤并在最后一步中选择“未设置”来重新启用Defender，此后您将需要重新启动计算机。

如何通过注册表禁用Windows Defender


如果您无权访问策略编辑器，或者已安装Windows 10 Home，则可以编辑Windows注册表，从而禁用Defender。

我提醒您，编辑注册表是有风险的，此事中的错误可能会对当前安装的Windows副本造成无法修复的损害。 最好在开始编辑之前备份系统。

要通过注册表完全禁用Defender，请通过“开始”按钮启动regedit程序，然后按照以下路径进行操作：

HKEY_LOCAL_MACHINE \ SOFTWARE \ Policies \ Microsoft \ Windows Defender

提示：可以复制此路径并将其粘贴到注册表编辑器的地址栏中。



然后，右键单击Windows Defender键（目录），选择“新建”和DWORD（32位）值。 将新密钥命名为DisableAntiSpyware，然后按Enter。 然后双击以打开密钥编辑器并将其设置为1。



单击确定，然后重新启动计算机。

之后，Windows Defender将不再保护您的系统。 如果要撤消这些更改，请重复所有步骤，但最后删除此密钥或将其设置为0。

推荐建议


尽管存在几种禁用Windows Defender的方法，但我们不建议您使用完全没有防病毒程序的计算机。 但是，在某些情况下，禁用此功能可能是一个更好的选择。 而且，如果您安装了第三方防病毒程序，则无需手动禁用Defender，因为在安装过程中它将自动关闭。
