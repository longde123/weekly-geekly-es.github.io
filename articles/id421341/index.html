Valve ingin memastikan bahwa semua game Windows berjalan di Linux

Valve memperkenalkan proyek Proton - meluncurkan game Windows di Linux.


Ilustrasi wccftech.com

Valve memutuskan untuk tidak berhenti merilis SteamOS, versi game-nya Linux, dan mengumumkan rilis versi beta dari layanan Steam Play dengan versi Wine yang dimodifikasi, dengan nama sandi Proton. Versi baru dari perangkat lunak ini memungkinkan Anda untuk menjalankan game Windows di Linux langsung dari Steam, tidak kurang. Secara umum, proyek Proton memiliki tujuan yang sangat ambisius - untuk memberikan kemampuan untuk menjalankan semua game Windows dari direktori Steam di Linux, yang disebut "out of the box".



Saat ini, 26 game didukung penuh. Selain itu, penggemar dapat mencoba meluncurkan game apa pun dengan menggunakan opsi Aktifkan Steam Play untuk semua judul di pengaturan. Semua hasil upaya untuk meluncurkan game dicatat dalam tabel google .

  • Gim di Windows tanpa versi yang tersedia untuk Linux sekarang dapat diunduh dan dijalankan langsung dari klien Steam di Linux, yang sudah memiliki dukungan untuk Steamworks dan OpenVR.
  • DirectX 11 dan 12 sekarang didasarkan pada Vulkan, yang meningkatkan kompatibilitas game dan mengurangi dampak kinerja.
  • Dukungan untuk mode layar penuh telah ditingkatkan: game akan secara sempurna melebar ke layar dengan ukuran yang diinginkan, sementara resolusi internal monitor tidak akan terdistorsi dan tidak akan memerlukan penggunaan virtual desktop.
  • Dukungan yang ditingkatkan untuk pengontrol game: game akan secara otomatis mengenali semua pengontrol yang didukung di Steam. Kompatibilitas yang lebih besar dengan pengontrol di luar rak diharapkan dibandingkan dengan versi asli gim.
  • Kinerja game multi-utas telah meningkat secara signifikan di atas Wine standar.


Untuk menjalankan Proton, cukup instal versi klien Steam terbaru untuk Linux.
Agar versi beta berfungsi dengan benar, sistem harus memiliki driver baru untuk versi kartu video NVIDIA tidak lebih rendah dari 396.51 atau Intel / AMD dari paket Mesa 18.2.0, di samping itu, peningkatan batas pada jumlah deskriptor file terbuka akan diperlukan. Untuk Ubuntu 18.04, versi baru dari driver NVIDIA dan Mesa dapat diinstal dari repositori PPA ( NVIDIA , Mesa ).



Prestasi proyek diterbitkan di GitHub dengan lisensi kode sumber BSD. Pengembangan telah berlangsung selama lebih dari dua tahun dengan partisipasi langsung dari tim HQ Wine, dan banyak perubahan sudah ditransfer ke proyek Wine asli dan proyek terkait, seperti DXVK dan vk3d.

Pada 2010, Valve mengumumkan perilisan layanan Steam Play, yang, setelah membeli satu game, memasangnya di salah satu dari tiga keluarga OS populer: Windows, macOS dan Linux. Selama 8 tahun sejak saat itu, lebih dari 3.000 game dalam katalog Steam telah tersedia untuk pengikut Linux, dan pengenalan dukungan untuk game baru belum berhenti.

Namun, seiring dengan upaya untuk memperluas perpustakaan game di satu platform, Valve memutuskan untuk berhenti mendukung sistem yang lebih lama di platform lain. Pada Juni 2018, perusahaan mengumumkan penyelesaian dukungan untuk Windows XP dan Vista.
