Everykey：您的小工具的通用密钥



由凯斯西储大学企业家课程的毕业生于2012年创立的美国公司Everykey Inc.，已于2014年在Kickstarter上亮相，其手镯形式为通用钥匙。今天，他们展示了该设备的新工作原型。以很小的尺寸执行：



该项目在Indiegogo平台上展示，并筹集了超过130,000美元，比最初的标准高出将近7倍。顺便说一句，去年公司的面孔是传奇的约翰·迈克菲（John McAfee），他是McAfee防病毒软件的创造者，它以一种非常原始的方式移动设备。



本质是什么

使用Everykey，您无需发明和记住计算机和移动设备的密码以及访问帐户。他将设备放在口袋里，坐在桌子旁-一切都已解锁。移动距离超过3米-一切都被阻止：



根据开发人员的说法，Everykey能够生成密码，并提供通过无线通信访问所有密码锁的权限。很快计划将其用于打开房屋，汽车的入口以及自行车的现代防盗锁等。

如何运作

使用Everykey Apr应用程序为每个设备（平板电脑，智能手机等）生成初始密码，该应用程序执行Everykey集成器与设备的主要功能。另外，使用该应用程序，可以禁用每个丢失的Everykey并防止未经授权的访问。



每次更改结构时，AES 128位代码用于加密通信。因此，即使有人使用代码采集器，拦截通信也不会带来任何结果。

当然，Everykey可能会被盗或丢失-在这种情况下，您需要尽快使用该应用阻止设备。



规格

Everykey兼容BLE 4.0无线协议，与操作系统兼容：

  • Mac OS 10.9+（小牛，优胜美地和更新的版本）
  • Windows 7+
  • Ubuntu 14+（Linux）
  • Android 4.4以上版本（KitKat，Lollipop和更高版本）
  • iOS 8以上

可以使用以下浏览器配置对帐户的访问：

  • 谷歌浏览器v37 +
  • Mozilla Firefox v31 +
  • Safari 7.1以上
  • Microsoft Edge（未来版本）

Everykey可以在锂聚合物电池上运行长达30天，并通过USB连接器充电。外壳是防水的。
