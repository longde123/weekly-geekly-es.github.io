Excelsior JET suspende el desarrollo de su compilador AOT después de 18 años de trabajo

Los que siguen lo que está sucediendo con la plataforma Java probablemente se encuentren con el tema de que hay un compilador AOT muy interesante para Java desarrollado por una compañía rusa de Novosibirsk. Este es uno de esos proyectos para estar orgulloso.

Hasta hace poco, todo estaba muy bien:

  1. El estándar x86 estaba disponible de forma gratuita
  2. Los desarrolladores dieron entrevistas, de las cuales se deduce que el producto no se detiene, desarrolla: JVM siberiano severo: una gran entrevista sobre Excelsior JET
  3. En JPoint 2019, Nikita Lipsky ( pjBooms ) hizo una presentación muy interesante: JPoint 2019: desde el amanecer hasta el anochecer

Y luego, un día, parece que todo terminó. Hoy supe que el otro día los clientes recibieron una carta de felicidad con el titular: "No habrá Excelsior JET 16".

pjBooms , ¿es eso realmente cierto? Entonces, ¿el único compilador AOT que funciona después de 18 años de éxito en el mercado simplemente desaparece?
