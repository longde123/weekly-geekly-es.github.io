RPG de texte d'oeuf de Pâques dans le code du moteur de recherche Google

L'utilisateur Reddit ( try_number_1 ) a partagé une découverte très inhabituelle sur le forum - il a pu exécuter un RPG texte dans le code de google.com.



Il s'avère que personne ne sait exactement quel âge a ce secret de Pâques.





Instruction: (vérifié par moi en chrome et mozilla)

Pour entrer dans le RPG dont vous avez besoin (un lien direct ci-dessous dans le commentaire existe déjà , merci lexasoft83):
- allez sur la page "Google.com" en "mode navigation privée";
- allez dans le menu "paramètres \ paramètres", sélectionnez "Langues \ langues" et sélectionnez quelle langue les produits Google doivent-ils utiliser? - Anglais et sélectionnez «enregistrer»;







- entrez deux mots sans guillemets dans la ligne de recherche: «aventure texte»;
- une fois les résultats de la recherche reçus affichés, appuyez sur la touche F12;
- dans la fenêtre qui apparaît, sélectionnez l'onglet «Console»;
- pour démarrer le jeu, vous devez écrire «oui» dans la console.



Une liste complète des commandes ressemble à ceci:
- nord (nord),
- sud (mouvement sud),
- ouest (mouvement vers l'ouest),
- est (mouvement vers l'est),
- vers le bas (mouvement vers le bas)
- vers le haut (mouvement vers le haut),
- saisir (prendre l'objet),
- pourquoi (question pourquoi),
- inventaire (inventaire ouvert),
- utiliser (utiliser l'article),
- aide (demande d'aide),
- sorties (sortie),
- carte (carte),
- amis (amis).

Après avoir entré chaque équipe dans le texte du jeu sera signalé sur les directions possibles pour le prochain mouvement et les obstacles sur le chemin.

Dans un RPG secret, vous devez trouver des lettres du logo Google en utilisant des instructions textuelles simples comme "aller au nord" ou "aller à l'ouest", ainsi que des actions telles que "prendre", "ouvrir l'inventaire" et "utiliser".

Les utilisateurs de Reddit notent qu'un passage complet et réfléchi peut prendre de une à deux heures. L'un des joueurs a souligné que l'emplacement présenté dans le jeu rappelle quelque peu l'emplacement de l'un des bureaux de Google en Californie. Le développeur qui a créé ce secret pourrait y travailler.
