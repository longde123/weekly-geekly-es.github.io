Di mana mulai belajar Python: buku untuk pemula



Teman-teman, buat bookmark pilihan 17 buku Python.

Pilihan akan membantu Anda belajar bahasa pemrograman dari awal atau dengan pengetahuan minimal.
Sudah siap untuk mulai belajar Python?

Kalau begitu mari kita mulai!

Memulai dengan Belajar Python


Jika Anda tidak beralih ke Python dari bahasa pemrograman lain, tetapi mulai dari awal, pertama-tama pahami konsep dasar pengembangan. Untuk membaca literatur teknis dalam bahasa Inggris, naikkan level bahasa. Buku akan membantu dengan ini:

"Pemrograman Komputer untuk Pemula: Dasar-Dasar Istilah dan Konsep Pemrograman", Nathan Clark

Pemula dapat dengan mudah tenggelam dalam istilah profesional. Buku ini akan membantu Anda memahami terminologi: apa saja fungsi, fungsional dan pemrograman berorientasi objek, bagaimana menyusun dan menggunakan program, cara menyimpan, mengelola, dan berbagi data.

Bahasa Inggris untuk Teknologi Informasi, Pearson Longman

Bahkan jika Anda fasih berbahasa Inggris, terminologi profesional dapat menyebabkan jalan buntu - menghabiskan waktu mencari Google. Tutorial ini dengan buku audio dan kamus interaktif akan membantu meningkatkan level tata bahasa dan mengisi kembali kosakata dengan istilah teknis.

Bahasa Inggris Dasar untuk Komputer, Oxford

Kursus pelatihan yang akan membantu meningkatkan keterampilan tata bahasa, mengisi kembali kosakata dengan istilah teknis. Buku chip - berisi teks dan diagram untuk pemahaman yang lebih baik tentang materi, kamus istilah komputasi modern, singkatan dan simbol.

Temui Python


Mulailah belajar bahasa pemrograman dengan landasan teoretis - pahami istilah dan prinsip bahasa tersebut. Untuk melakukan ini, pelajarilah buku-buku:

"Halo Dunia. Pemrograman Menyenangkan ”, Carter Sand, Warren Sand

Sebuah buku sederhana untuk pengantar Python - penulis menjelaskan istilah dasar, fitur bahasa dalam bahasa yang dapat diakses. Publikasi ini terutama bersifat teoritis, dari praktik - pedoman untuk membuat beberapa program dasar.

"Memprogram Python," Michael Dawson

Pengembang dan guru yang berpengalaman berbicara tentang prinsip-prinsip dasar pengembangan menggunakan contoh pembuatan game sederhana. Setelah setiap bab, penulis menawarkan konsep permainan, ringkasan materi yang dibahas, dan tugas untuk konsolidasi. Setelah membaca, Anda akan belajar keterampilan pengembangan Python dasar dan belajar bagaimana mempraktikkannya.

Mari kita lanjutkan berlatih


Ketika Anda memahami dasar-dasar teoretis, buka buku dengan praktik - contoh kode nyata, tugas, panduan untuk mengembangkan proyek sederhana pertama:

Belajar Python oleh Marz Lutz

Penulis menjelaskan jenis utama objek dalam Python, urutan kreasi mereka dan bekerja lebih jauh dengannya, berbicara tentang elemen prosedural utama dari bahasa - fungsi. Setelah setiap bab ia memberikan pertanyaan kontrol untuk memperbaiki materi, dan setelah setiap bagian - latihan praktis.

"Pemrograman Python," Mark Lutz

Panduan untuk menggunakan Python di berbagai bidang - administrasi sistem, membuat aplikasi web dan antarmuka grafis. Penulis memberitahu cara bekerja dengan database, interaksi jaringan program, membuat antarmuka untuk skrip dan tugas lainnya.

"Python - ke puncak keunggulan", Luciano Ramallo

Buku pengembang Luciano Ramallo untuk mereka yang sudah belajar menulis dengan Python, tetapi masih belum menggunakan semua fitur-fiturnya. Penulis berbicara tentang alat dasar dan pustaka bahasa dan memberikan contoh ilustrasi tentang bagaimana membuat kode lebih pendek, lebih mudah dimengerti, dan lebih cepat.

Pemrograman Python: Panduan Langkah Demi Langkah Untuk Pemula, Brian Jenkins

Panduan langkah demi langkah untuk pemula yang ingin belajar bahasa pemrograman dari awal. Penulis menjelaskan sintaksis dasar, mengajarkan Anda cara bekerja dengan tipe dan variabel data, kelas dan objek, menangani file dan pengecualian.

"Pemrograman Python: Panduan Dasar, Blackhat, Perantara dan Lanjutan untuk Pemrograman Python," Richard Ozer

Buku ini adalah manual Python empat bagian yang lengkap. Pemula harus mempelajari bagian pertama buku ini. Penulis dengan sederhana dan jelas berbicara tentang mekanisme kerja Python, bagaimana memahami dasar-dasar bahasa dan menulis program pertamanya.

“Pemrograman Python Diilustrasikan Untuk Pemula & Menengah:“ Belajar Dengan Melakukan ”Pendekatan-Langkah Demi Panduan Utama Untuk Menguasai Python: Masa Depan Ada Di Sini!

Panduan langkah demi langkah untuk membantu Anda mempelajari bahasa Python dan mempraktikkannya. Penulis memberi tahu cara menjalankan program pertama, menjelaskan variabel, tipe data, pengidentifikasi dan berbagi informasi bermanfaat lainnya.

"Python dalam praktek," Mark Summerfield

Buku untuk pemula yang sudah menguasai bahasa Python dan ingin memprogram lebih baik. Penulis menceritakan bagaimana menggunakan pola pemrograman untuk meningkatkan kode, memparalelkan dan mengkompilasi program, menjelaskan prinsip-prinsip pemrograman dan grafik jaringan tingkat tinggi.

Pembelajaran Berkelanjutan: Buku Python berdasarkan Petunjuk


Python dapat digunakan untuk memecahkan banyak masalah - bekerja dengan situs, aplikasi web, pembelajaran mesin. Putuskan ke arah mana Anda ingin mengembangkan, dan mulai dengan literatur khusus:

"Aplikasi GIS - Mengembangkan Aplikasi Geo dengan Python," Eric Vestra

Penulis menjelaskan secara rinci cara menulis geo-aplikasi. Anda akan belajar cara mengakses dan memvisualisasikan geodata, membaca dan menulis data dalam format vektor dan raster, menyimpan dan mengakses, melakukan perhitungan geospasial dengan Python.

"Menggores Situs Web - Menggores Situs Web Menggunakan Python," Ryan Mitchell

Panduan tentang cara menerapkan skrip Python dan API web untuk mengumpulkan dan memproses data dari ribuan situs. Ini akan menarik bagi para programmer dan administrator web yang ingin mempelajari pekerjaan para pencakar web, menguasai analisis data mentah dan pengujian antarmuka.

“Pengantar pembelajaran mesin dengan Python. Panduan untuk Profesional Data ”, Andreas Muller, Sarah Guido

Buku untuk pemula. Penulis menjelaskan bagaimana membangun sistem pembelajaran mesin, menjelaskan secara rinci tahapan pekerjaan menggunakan Python dan scikit-learn, NumPy, dan perpustakaan matplotlib.

"Django. Panduan Lengkap ”, Adrian Golovaty, Jacob Kaplan-Moss

Buku tentang Django, kerangka kerja pengembangan aplikasi web Python. Para penulis berbicara tentang komponen kerangka kerja dan metode bekerja dengannya, memberikan contoh aplikasi dalam berbagai proyek.

"Mengembangkan Aplikasi Web Menggunakan Flask dengan Python," Miguel Greenberg

Penulis mengajarkan Anda untuk bekerja dengan kerangka kerja Flask yang populer, memberikan panduan langkah demi langkah tentang cara membuat aplikasi blogging sosial. Pelajari kemampuan kerangka kerja, pelajari cara memperluas aplikasi dengan teknologi tambahan.
