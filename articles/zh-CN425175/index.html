关于用于PC的OPL2 / 3 FM合成器的声卡的一些知识...

在我看来，声卡中的FM合成器被人们遗忘了，因此，我决定分享我对这个问题的看法。



现在，ISA声卡引起了某种兴奋-它只是在各地疯狂地购买，eBay的价格仅仅是空间...也许收藏家正在购买，也许事实是，只有FM音频合成器（OPL2 / OPL3）用于ISA声卡。 现在，FM合成器已被完全软件WaveTable技术所取代，并且在从DOS切换到Windows之后的游戏中，FM / WaveTable合成器上的音乐已不再通过DAC来支持mp3。

大多数人认为FM合成器（OPL2 / OPL3）上的音乐不好，WaveTable更好。 常见的误解是采用WaveTable技术的声卡更好。 是的，在WaveTable上，许多古典乐器听起来更逼真，古典作品更好，而Techno则更糟，因为它们失去了“宇宙”，“电”声。 可以通过在SB AWE32和AdLib上播放X-COM：UFO Defense（在Yamaha YM3812芯片上播放OPL2）或在xcom-sounds站点上收听Adlib-FLAC.zip录音站点来轻松验证这一点。 在真实的声卡上，音乐甚至比在FLAC中录制音乐更加生动（我在ISA PC上确保了SB16声音，我专门为此目的将其组合在一起）。 通过DosBox中的OPL2 / OPL3声音模拟器，您从音乐中获得的乐趣不会比从ISA声卡中获得的乐趣多。 区别与您比较听有64-96 kb \ s mp3录音和未压缩CD的音乐时的区别相同-区别非常明显。

那么，为什么PC上的FM合成器听起来不好？

首先，如果您演奏古典乐器的乐曲较慢，那么是的，OPL2 / 3听起来会很暗。 但是，如果以Techno / RAVE风格快速发展，则FM合成器将绽放光彩。 如果您在Windows下下载用于现代PC的Adlib Tracker 2 ，则可以验证这一点-播放器随附458条轨道。 这是Adlib开发的跟踪播放器/编辑器，但与MOD / S3M不同，它不使用数字样本，而是直接通过FM合成器播放。 在Dosovsky程序界面中，使用F3键打开资源管理器以从MODULES目录下载曲目，F5播放，F7停止。 仿真的质量非常好，但不如实时芯片上的管状。 在MODULES \ ben目录中，有一个有趣的音轨sonic.a2m-SEGA上同名游戏的音乐，听起来像是原始的，我没在普通的midi文件中看到这种声音。

其次， 在我看来，作为潮流引领者的AdLib并未为其声卡选择最佳芯片-Yamaha YM3812 。 在DOS游戏的历史上，我可以回想起一些带有惊人FM音乐的游戏-这就是X-COM所说的：UFO防御，Tyrian，Cyber​​sphere。 在DOOM和Duke Nukem 3D中，WaveTable听起来已经更好了（嗯，主要的不是那里的音乐）。 但是，在Yamaha YM2612芯片上的SEGA游戏机上，我已经可以命名更多具有令人赞叹的“太空”音乐的游戏-这些游戏包括Skeleton Krew，Alien Soldier，Exo Squad，BOB等。
在本文发表之后的一段时间，当我听过sonic.a2m音轨时，我意识到应该责怪的不是YM3812芯片，而是midi格式。 分布最多的

在这里，我有一个想法-而不是是否要恢复PC声卡上的FM合成器？ 没有竞争！ 没有人能生产出新的，旧的ISA声卡，不能将其插入现代PC中。 有一个单一的制造商Arstech制造了一个ISA到USB适配器 ，但价格仅为149美元。 还有另一家制造商为LPT端口制造 OPL2 / 3声卡。 但是它们没有DAC，因此不能称为成熟声卡。

仅在OPL2 / 3芯片上制作带有FM合成器的现代PCI-e声卡是没有意义的，因为格式上没有太多好的音乐，但是对于2-3 DOS，这也不是很有趣。 基于AY-3-8910或其类似物，还需要一个SSG零件。 1989年，Creative已经在早期版本的Sound Blaster 1.0 CT1310中制造了这种声卡-它安装了CMS Phillips SAA-1099s芯片，其声音类似于AY-3-8910，但是由于游戏开发人员不支持他们的游戏在下一版本的Sound Blaster 1.5 CT1320 Creative中，不再使用CMS芯片。 但是那是在1989-1990年代，并且在1990年代中期，借助AY-3-8910音乐处理器，他们开始在ZX-Spectrum上猛烈地编写音乐，现在所有这些音乐都可以在PC上收听。 仿真器可以很好地再现AY-3-8910，但是真正的微电路听起来更清晰，更丰富，更有趣，而且某些特殊效果仅在芯片上才有可能。 更有意思的是OPL3 + AY-3-8910的音乐，如果有真正的声卡，这种共生的跟踪器将很快出现。 作为DSP，您可以放置​​C-Media MI8786-对于大多数游戏玩家来说就足够了。

类似的PC项目已经存在-直到2000年代，生产的ISA CatWeasel声卡与SID 6581芯片上的AdLib兼容，无法收听SID音乐。 这样的声卡在今天将非常有用-像热汉堡一样卖完（成千上万的人听SID音乐），但是没有生产6581/8580微芯片，也没有类似物，它们仅以天文价格在eBay上单独出售。

您可能会问-我为什么要向别人抛出创业的念头？ 因为我自己不会做，但是我真的很想拥有这样的声卡，让另一个人致富！ 如果潜在的制造商阅读了此书，但您怀疑它是否会有所回报，那么您可以非常简单地做到这一点-在Kickstarter上启动一个项目。 我确信全世界至少有成千上万的人会对这个项目感兴趣，也许更多，因为PC市场上没有类似产品！

我认识很多成为程序员的人，因为他们年轻时就因为芯片调谐音乐而爱上了8位游戏。 我不知道单个DOS游戏会在WaveTable上吸引音乐！ WaveTable是一种邪恶行为，抢夺了整个计算机社区中美丽的“太空” chiptune音乐。 正如WaveTable的历史所表明的，事实证明，这在声卡的开发中是死胡同。 首先，她更换了时尚的FM合成器，然后她自己离开了舞台。 过去的许多热门游戏都只有Chiptune音乐。 顺便说一句，mp3也是邪恶的。 因为即使是320 kbps比特率的音乐听起来也比CD差！ mp3只是一个熟人，一直听mp3会让自己失去乐趣。 从Windows Vista和更高版本开始的Windows XP之后，Microsoft还削减了声卡（EAX）的硬件功能。 总的来说，各个方面的音质都在下降，是时候纠正这种情况了……
