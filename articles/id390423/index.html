Klausa kiamat zombie Amazon telah ditambahkan ke kebijakan layanan web Amazon



Menjelang Amazon Web Services meluncurkan layanan GameLift baru dan membuka akses ke mesin permainan 3D sumber terbuka gratis Lumberyard. Dalam hal ini, aturan untuk menggunakan layanan diperbarui di situs web AWS - khususnya, item yang terkait dengan mesin game ditambahkan. Dalam salah satu paragraf, pengacara perusahaan memutuskan untuk menyebutkan aturan untuk menggunakan layanan baru dalam kasus kiamat zombie.

Paragraf 57.10 menyatakan:

Rupanya, Amazon begitu menggoda dengan target audiens - pengembang game, di antaranya game, film dan seri zombie selalu populer. Sisa perjanjian penggunaan terlihat sangat ketat.

Lumberyard adalah mesin permainan yang didistribusikan oleh raksasa internet secara gratis. Perusahaan berencana untuk mendapatkan uang dari layanan webnya - mesin mengintegrasikan penggunaan AWS, Twitch dan Double Helix. Dapat dipahami bahwa pencipta game multi-pemain akan menggunakan layanan ini.

Namun, Amazon dengan murah hati memungkinkan Anda membuat game pemain tunggal di mesin ini yang tidak memerlukan koneksi Internet. Mesin Lumberyard didasarkan pada mesin permainan CryEngine terkenal lainnya, dilisensikan oleh Amazon musim panas lalu.di Crytek sebesar $ 50 juta.
