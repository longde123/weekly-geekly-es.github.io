Redmi Pro是XIAOMI推出的首款双摄像头智能手机



去年11月，小米推出了Redmi Note 3智能手机，该产品被誉为最佳金属智能手机，售价为200美元，它是该产品的第一篇评论。因此，由于不遵循传统的年度更新设备周期，XIAOMI上周宣布了Redmi Pro。

总的来说，这家公司知道如何制造智能手机-Redmi Note 2的价格是多少，它在短短几个月内就成为了真正的畅销书，并且像蛋糕一样发散了。通过命名，XIAOMI仍然更加有趣：最初预期Redmi Note 3将被称为Redmi Note 2 Pro，但是制造商仍然决定将智能手机分配给Redmi Note系列中的新一代设备。



但是在第四代公司中，该公司的行为有所不同，并添加了珍贵的前缀Pro，尽管Redmi Note 3也被某些人称为Redmi Note 3 Pro。



有很多泄漏和谣言-甚至更多，总的来说，大多数已得到证实。再加上线ultrabukov弥笔记本航XIAOMI推出了其功能最强大的智能手机，很可能会向其他制造商的许多旗舰店问胡椒。首先，值得一提的是屏幕：5.5英寸，采用OLED技术制成，全高清分辨率为401 ppi。顺便说一句，Redmi Pro是XIAOMI的第一款智能手机，该公司决定使用OLED。显示屏提供了NTSC颜色范围的100％覆盖，这些颜色确实非常明亮和饱和。所有这些都覆盖有带有圆形边缘的2.5D玻璃。





电池容量为4050 mAh（与Redmi 3S相同））这家中国制造商的工程师设法将新颖性的尺寸减小到最小-结果，我们的重量为151.5 x 76.8 x 8.15毫米，重量为174 g。 5毫米（根据传闻，同一台Apple会在iPhone 7中拒绝这种情况）。快速充电的能力是制造商储钱罐的另一个优点。





该设备的技术特性与现代旗舰产品相对应。提供三个选项供客户选择：更简单（3 GB RAM，32 GB内部存储器和Helio X20芯片组），更强大（3 GB RAM，64 GB内部存储器，但已经有Helio X25）和马拉松版本-相同的Helio X25，但4 GB RAM和128 GB内部存储器。是的，XIAOMI决定放弃Redmi Note 3之后的Helio X10（又名MT6795），转而使用10核甚至更年轻的芯片组。





Redmi Pro相机值得特别注意。后置摄像头是双摄像头-13兆像素（Sony IMX258传感器）和5兆像素（三星传感器），光圈从f / 0.95到f / 5.6，并带有双LED闪光灯。闪光灯非常有趣-就在两个相机之间。



为什么要使用双镜头？首先，正如制造商所称，在其帮助下，图片将能够与单反相机的照片竞争。但是，用户喜欢双摄像头的最重要的事情是能够快速拍照并在稍后设置焦点的能力，就像Lytro摄像头所实现的那样。一台定焦相机拍摄远处的物体，第二台直接聚焦在附近的物体上。



Redmi Pro中的前置摄像头也为5百万像素，捕获角度为85度-对于自拍照而言。它还以1080p拍摄。


Redmi Pro

的快照与Redmi Note 3 相比，新颖性有许多重要区别。首先，值得注意的是指纹扫描仪传感器-它适合现代智能手机，从后面板移至前面板。



其次-每个人都喜欢的SD卡插槽又回来了！更准确地说，存储卡将插槽用于第二张SIM卡。在Redmi Note 3中，用户确实错过了这一点，特别是对于具有16 GB内部内存版本的购买者。在Redmi Pro中，您最多可以安装256 GB的microSD，因此无需使用板载128 GB的版本。

接口集是标准的：Wi-Fi（802.11 ac / b / g / n），GPS，GLONASS和具有VoLTE支持的LTE。到目前为止，似乎很难提出新的建议，但是Bluetooth 4.2 LE出现在Redmi Pro中。

屏幕锁定按钮和音量键位于右侧，左侧仅是两个SIM卡或SIM卡和microSD卡的插槽。高于传统的是3.5毫米音频插孔。





所有这些操作都是通过带有MIUI 8外壳的Android 6.0棉花糖进行控制的，在XIAOMI Mi Max中已经可以看到后者-这是该制造商生产的首款具有此类外壳的智能手机。似乎它已经在买家中扎根了，所以Redmi Pro获得了MIUI 8支持，这意味着界面更整洁，附加功能（优化的图像查看系统，新应用程序“带有密码保护的便笺，具有高级功能的扫描仪），新的动态窗帘视图通知等。



也许对某人的唯一负面影响是不可拆卸的电池，但是当第二个制造商在不直接接触电池的情况下给智能手机加盖印章时，这将是不利的，这在某种程度上是没有道理的。
