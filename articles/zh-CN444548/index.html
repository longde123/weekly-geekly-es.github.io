个人经验：我们如何在拉丁美洲进行西班牙裔创业推广



我一直从事营销工作，在过去的几个月中，我一直在帮助一家新公司-一种纠正英文文本错误的服务。 在工作过程中，我和他的团队决定与目标受众进行实验-特别是尝试从以西班牙语为母语的人那里获得反馈。

我们在拉丁美洲没有营销预算，因此我们决定使用内容营销。 但是，事实证明，关于如何以及在何处推广您的公司在西班牙的意义不大。 我必须自己解决这个问题，并根据实验结果分享自己的经验。

步骤＃1：收集信息


我们对Internet的西班牙裔部分的设计一无所知，因此对于初学者而言，有必要至少对其进行基本的研究。 团队中没有以西班牙语为母语的人，因此我们没有地方可以从中获取任何内部信息，数据以俄语（主要是英语）收集。

我们建议说西班牙语的用户应该使用自己的流行网站类似物来发布内容。 例如，Reddit。 在Quora之类的网站上进行各种搜索查询和研究主题线索并不能提供太多信息-我们找到了受欢迎的Taringa网站，仅此而已。

我必须订购一位以西班牙语为母语的SMM专家就Upwork进行咨询。 之后，情况变得更好。

步骤2：发布内容


我们有许多有趣的“党派”英语材料，翻译起来并不难。 一个更有趣的问题是在哪里发布这些文本。

我们研究了各种博客平台，并决定不使情况复杂化，并在Medium上发布内容。 该选项具有多个优点-平台易于使用，界面熟悉，您可以在网站上以本地语言发布内容（甚至在俄语中也有许多有趣的博客），每篇文章都有统计信息，其中包括有关引荐来源的信息-本来可以帮助我们计划进一步的工作。

步骤＃3：促销


对于内容的“播种”，我们选择了两个主要资源：

  • Taringa.net是拉丁美洲最受欢迎的聚合网站，与Reddit有点相似，拥有超过2100万访问者。
  • Meneame.net -950万访客。

原始表格中的网站更多，但是，其中很大一部分却显得太小，根本没有任何流量，因此即使在“堆”模式下，我们也逐渐拒绝将其重新发布。

您可以在每个页面上发布链接和文章，没有特别困难。 一些资源（例如Meneame）施加了其他限制-例如，在发布您的帖子之前，您需要对其他用户的帖子进行7到8个赞。

借助Google Chrome浏览器中的自动翻译功能，发布内容本身也非常简单：



我们的任务是获取有关产品的反馈，因此，扩大潜在受众的地理位置非常重要。 奇怪的是，Reddit来到这里营救。

事实证明，在此平台上，有专门针对不同国家的小节-这些小节的内容以当地语言发布。 我们很幸运，我们的项目主题非常适合在此类子目录中发布-该工具对于学习英语很有用，许多西班牙裔用户对此都很感兴趣。

结果，我们将内容放置在以下子目录中：

  • r / aprendeingles / -专门研究英语的部分；
  • / r / es / -总体主题西班牙裔subreddit，1.43万名用户；
  • / r /阿根廷/-9.2万名订阅者；
  • / r / vzla / -委内瑞拉部分， 3.45万订户；
  • / r /哥伦比亚 --1.38万名订户；
  • / r / chile --48 000用户。

结果如何


总共，一个月内我们发布了4份材料-数千人阅读了这些材料，我们在注册方面获得了很好的转换。 西班牙裔观众非常积极地使用该服务并获得高级访问权限。 此外，该项目还受到拉丁美洲记者和博客作者的关注。 特别是，在对广受欢迎的阿根廷IT资源“新功能”进行了回顾之后，几天之内有数千人来找我们。

结论


当然，这样的内容营销并不是吸引大量流量的工具，但是如果您想用零预算来测试新市场，那么将中型+聚合器+ Reddit组合在一起非常有效。
