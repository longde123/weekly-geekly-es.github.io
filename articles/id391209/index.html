Slider bermotor DIY untuk memotret TimeLapse dan video



Halo semuanya!
Saya mendapatkan versi slider PhotoSnail yang berfungsi, yang ingin saya bagikan.

Apa itu PHOTOSNAIL?


PhotoSnail adalah proyek terbuka sistem untuk pemindahan otomatis kamera pemotretan (Slider) dengan pelacakan otomatis objek, untuk pemotretan foto dan video. Ada juga ide untuk menggunakannya untuk membuat Foto 360 yang tidak lengkap .

Unit kontrol sistem didasarkan pada Arduino dan sama seperti dalam proyek terbuka PhotoPizza
PhotoSnail sedang dikembangkan dan sekarang hanya gerakan platform bersama panduan dengan parameter yang dapat disesuaikan yang diimplementasikan. Dalam pengaturan program, Anda dapat mengatur panjang total gerakan kamera pemotretan, jumlah bingkai, arah, akselerasi, dan jeda untuk pemotretan. Kamera terhubung ke unit kontrol melalui kabel sinkronisasi dan setiap model kamera yang memiliki konektor untuk menghubungkan remote control berkabel didukung.

IDE UTAMA PROYEK



BIAYA RENDAH - tujuan saya adalah menciptakan solusi yang benar-benar terjangkau yang akan terjangkau bagi banyak fotografer dan bisnis amatir dari semua ukuran. Akibatnya, total biaya semua elemen secara signifikan lebih rendah daripada analog yang sebanding dalam karakteristik.

KETERSEDIAAN KOMPONEN - Saya secara khusus memilih komponen sehingga Anda tidak memiliki masalah signifikan dengan pencarian dan pembelian mereka. Misalnya, proyek ini menggunakan bantalan dari sepatu roda, yang dapat Anda temukan di toko olahraga terdekat.



PERAKITAN MUDAH - perakitan struktur tidak akan memakan banyak waktu dan tidak memerlukan pengetahuan atau alat khusus.

UNIVERSALITAS -Saya mencoba membuat peralatan yang elemennya dapat digunakan untuk memecahkan masalah tambahan.
Sebagai contoh, slider dapat digunakan sebagai drive untuk memindahkan bilah, di mana akhirnya kamera aksi diperbaiki.
Metode gerakan ini dapat digunakan untuk memecahkan masalah kreatif. Buat penerbangan kamera melalui rintangan dengan efek Time-lapse atau hanya merekam video.

DESAIN


Penggeser terdiri dari dua platform yang disatukan dan menjepit pemandu bersama-sama, bersandar pada roller tekanan.
Dengan demikian, slider itu sendiri menyelaraskan pemandu sejajar satu sama lain.
Panduan dapat ditangguhkan dari pohon, tripod dan mereka akan ditempatkan secara paralel.
Platform dapat bergerak terbalik dan ke samping, sementara biasanya memegang kamera.



Pada platform bawah ada dua konektor untuk menghubungkan mesin ke unit kontrol.



Drive roller ditekan ke salah satu pemandu, selip dengan gerakan horizontal dan sudut kecil tidak diamati.
Juga, roller roda gigi dapat diletakkan pada poros motor dan sabuk penggerak dapat digunakan untuk memindahkan platform pada sudut besar pengencang pemandu, juga pada sudut 90 derajat.



Head atau platform tripod untuk memasang kamera terpasang ke platform atas.





MANAJEMEN


Peta menu unit kontrol




Deskripsi fungsi unit kontrol


Unit kontrol memiliki 4 program yang dapat dikonfigurasi dengan set parameter yang sama. Program dapat dikonfigurasi untuk berbagai kondisi pemotretan. Misalnya, jika Anda merekam video, Anda harus mengonfigurasi program untuk memindahkan situs dengan cepat. Untuk memotret TimeLapse, Anda perlu mengonfigurasi jeda panjang dan jumlah bingkai, dll.

Nilai parameter


Jalankan - mulai program
langkah - atur total jarak yang harus dilalui kamera.
Bila diatur ke "0", mode inf infinite movement
- kecepatan
frame - jumlah frame per total jarak pergerakan kamera yang ditentukan dalam langkah - langkah
jeda parameter - penundaan setelah relai dipicu ( rana kamera). Ini diatur sesuai dengan kecepatan rana kamera (nilai dalam milidetik).
Jika jeda - tidak ada , mode pemotretan interval diaktifkan tanpa menghentikan kamera pemotretan.
accel - akselerasi setelah memulai program, mulai halus dan berhenti kamera
dir - arah gerakan juga mengubah tombol pada remote



Arti tombol pada remote


Dalam mode siaga
Navigasi dalam menu dilakukan menggunakan tombol panah.
Kanan dan kiri, sebuah program dipilih, dari 1 hingga 4 . Setiap program memiliki pengaturannya sendiri.
Gunakan tombol atas dan bawah untuk memilih opsi untuk mengedit.
Untuk menjalankan program ini, atau untuk mengedit parameter tekan tombol ¬´OK¬ª . Tombol yang sama untuk menyimpan parameter yang diedit.
Untuk memasukkan nilai baru, pilih parameter dan tekan tombol angka, tombol naik dan turun mengubah nilai langkah demi langkah atau memungkinkan untuk memilih nilai dari beberapa opsi.
Tombol "#" mengatur ulang nilai ke "0"
Tombol "*"mengubah arah gerakan dalam mode Jalankan.

Dalam mode pengoperasian program,
tombol dengan panah atas dan bawah selangkah demi selangkah mengubah kecepatan kamera.
Tombol OK menghentikan program (gerakan kamera)
