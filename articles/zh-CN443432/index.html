Blazor 0.9.0发布

Blazor 0.9.0现在可用！ 此版本在.NET Core 3.0 Preview 3中添加了Blazor Razor组件增强功能。


现在，新的剃刀组件增强功能可用于Blazor应用程序：


  • 改进的事件处理
  • 表格和验证


期待看到您的反馈！

请查看ASP.NET Core 3.0 Preview 3的公告，以了解这些增强功能的详细信息。 另请查看 Blazor 0.9.0 发行文档以了解更多详细信息。


注意：在此版本中，尚未将Blazor模板更新为对Razor组件使用新的.razor文件扩展名。 此更新将在将来的版本中进行。


安装Blazor 0.9.0


要开始使用Blazor 0.9.0，请安装以下软件：


  1. .NET Core 3.0预览版3 SDK （3.0.100-preview3-010431）
  2. 带有ASP.NET的Visual Studio 2019 （Preview 4或更高版本）。
  3. Visual Studio Marketplace中的最新Blazor扩展。

  4. 命令行上的Blazor模板： 


    dotnet new -i Microsoft.AspNetCore.Blazor.Templates::0.9.0-preview3-19154-02

可在blazor.net上找到Blazor的入门指南，文档和教程。


升级到Blazor 0.9.0


要将现有的Blazor应用程序升级到Blazor 0.9.0，首先请确保已完成上述所有操作。


要将Blazor 0.8.0项目升级到0.9.0：


  • 更新Blazor软件包和.NET CLI工具链接到版本0.9.0-preview3-19154-02。
  • 更新其余的Microsoft.AspNetCore程序包。 *最高版本为3.0.0-preview3-19153-02。
  • 删除对JSRuntime.Current任何使用，并使用依赖项注入来获取IJSRuntime的当前实例，并将其传输到需要的地方。
