Instalar complementos de Google Chrome en Mozilla Firefox

Los desarrolladores de Firefox están trabajando en la introducción de una nueva API de complementos llamada WebExtensions, utilizada en los navegadores Chrome y Opera. La primera versión estable de la API debutó en Firefox 48, que ya está disponible para descargar ( se espera un anuncio de lanzamiento oficial en las próximas horas).

Para instalar el complemento de Chrome en Firefox, debe:

1) Instale Chrome Store Foxified : reemplaza el botón "agregar a Chrome" con "agregar a Firefox", y también se encarga de todo el trabajo de descarga, conversión y firma de los complementos instalados.



2) Instale el complemento deseado (las aplicaciones y los temas aún no son compatibles) desde el directorio web de Google Chrome : se descargará y convertirá del formato crx al formato xpi estándar para Firefox . No se requiere una cuenta de Google. El usuario tiene una opción: instalar el complemento temporalmente (antes de reiniciar el navegador, lo cual es conveniente para la evaluación inicial del rendimiento) o firmarlo e instalarlo de manera continua.



Para obtener una firma, primero debe crear una cuentaen el repositorio de Mozilla . Las comprobaciones automatizadas suelen tardar unos segundos. Además, es posible guardar el complemento sin firmar en el disco duro (por ejemplo, firmarlo más tarde manualmente en otra computadora).



Después de una instalación exitosa, el ícono del complemento (si está previsto) aparecerá en el panel del navegador, y la configuración estará disponible a través del mecanismo estándar de administración de complementos de Firefox. Sin embargo, debe recordarse que todavía no se habla de compatibilidad 100%. El trabajo en WebExtensions continúa. Además, algunos complementos utilizan funciones específicas de Chrome que no se encuentran en Firefox.
