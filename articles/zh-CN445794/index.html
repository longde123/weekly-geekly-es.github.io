IT巨头推出了混合混合云部署解决方案

戴尔和VMware集成了VMware Cloud Foundation和VxRail平台。


/照片Navneet Srivastav PD

为什么有必要


根据云状态调查，已经有58％的公司使用混合云。 去年，这个数字是51％。 平均而言，一个组织将“五种不同的服务”“保存在云中”。 同时，引入混合云是45％的公司的优先事项。 在已经使用混合基础架构的组织中， 我们可以区分 SEGA，牛津大学和分析组织ING Financial。

云环境数量的增加使基础架构复杂化。 因此，现在，IT社区的主要任务是创建可简化多云工作的服务。 朝这个方向发展的公司之一是VMware。

去年年底，这家IT巨头收购了Heptio初创公司 ，该公司推广Kubernetes部署工具。 上周，人们知道VMware正在与戴尔推出联合解决方案。 我们正在谈论一种基于Dell EMC VxRail超融合组件和VMware Cloud Foundation（VCF）平台创建混合云环境的系统。

对新产品的了解


VMware将其VMware Cloud Foundation云堆栈更新为3.7版。 从今年4月开始，该解决方案将预安装在Dell VxRail超融合系统上。 新平台VxRail上的VMware Cloud Foundation将提供API，这些API可将Dell网络设备（如交换机和路由器）与VCF软件组件连接起来。

VCF体系结构包括vSphere服务器虚拟化软件和vSAN存储系统。 此外，它包括NSX数据中心技术，旨在优化和管理数据中心的虚拟网络。 NSX迁移到超融合基础架构的能力已在英格兰的Baystate Health 进行了测试 。 根据医院IT专家的说法，该系统可以实现所有软件和硬件以及驱动程序的高度集成。

VMware Cloud Foundation的另一个组件是vRealize Suite混合云管理平台。 它包括用于分析虚拟基础架构的运行，估算云资源成本，进行监视和故障排除的工具。

至于VxRail，它由Dell PowerEdge系列服务器组成。 一台设备最多可支持200个虚拟机。 如有必要，可以将服务器合并到一个群集中，并同时使用3000个VM。

将来，他们计划将解决方案开发为单个系统-为此，戴尔和VMware将同步VxRail和VMware Cloud Foundation的产品更新。

社区的想法


根据VMware的代表，更新后的集成平台显着提高了混合IT基础架构的生产率-与旧版本的VxRail相比提高了60％。 此外，基于VxRail的VMware Cloud Foundation将降低创建云基础架构的公司的成本。 五年内的运营成本将比公共云降低45％ 。

戴尔和VMware的主要优势之一是物理网络设备的配置和管理的自动化。 但是，他们看到分析师和IT巨头可能面临的潜在问题。 也许主要的是高竞争 。 近年来，公司已进入许多新市场（包括HCI，SDN和SD-WAN），这些主要市场已经开始运作。 为了进一步发展，IT巨头需要将其解决方案与竞争对手区分开来的新功能。

这样的领域之一可能是用于管理Dell和VMware已集成到其产品中的数据中心的机器学习技术。

类似系统


NetApp和Nutanix也正在开发混合云超融合系统。 第一家公司提供带有集成Data Fabric平台的私有云系统，该平台将本地基础结构与公共云服务连接在一起。 该产品还基于VMware技术，例如vRealize。

该解决方案的显着特征是用于计算和存储的单独服务器节点。 根据公司代表的说法，这种基础架构可以帮助数据中心更有效地分配资源，并且不会为不必要的设备支付过多的费用。

Nutanix还创建了一个混合云管理平台。 例如，该组织的产品组合包括一个用于配置和监视IoT系统的系统以及一个用于使用Kubernetes容器的工具。

通常，越来越多的超融合基础设施提供商进入多云系统市场。 预计这种趋势将在不久的将来继续。 特别是，戴尔和VMware之间的联合解决方案将很快成为更大的Project Dimension项目的一部分，该项目将云系统与外围设备和本地设备集成在一起。
