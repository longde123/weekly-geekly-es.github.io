Como trabaja ella? Como tenemos un botón conectado entre GND y la entrada, y también se activa una resistencia pull-up, cuando se suelta el botón, la salida de la unidad de entrada será 1, y cuando se presiona el botón, será 0. El disparador R superior selecciona el borde delantero (transición de 0 a 1) - luego hay un momento de soltar el botón y crea un breve impulso para desconectar la carga en la variable "Load shedding command 1". El gatillo R inferior, dado que su entrada está invertida, produce un borde posterior (transición de 1 a 0), es decir, el momento en que se presiona el botón. Y también forma en este momento un breve impulso en la variable "Comando para encender la carga 1".
Ahora repita este circuito para el resto de las entradas.
