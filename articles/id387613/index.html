4 port USB untuk Raspberry Pi Zero? Tidak masalah



Miniatur PC Raspberry Pi Zero keluar baru-baru ini, tetapi penggemar "raspberry" telah berhasil bekerja dengan dewan. Jadi, salah satu pengguna menambahkan modul WiFi ke mini-PC. Yang lain membuat konsol mini dari papan , mengintegrasikannya langsung ke pengontrol game. Dan meskipun proyek-proyek ini tidak dapat disebut rumit atau indah, mereka menimbulkan minat tertentu dari pemilik raspberry lainnya.

Sekarang satu lagi telah ditambahkan ke daftar proyek yang ditunjukkan. Jadi, pembeli lain dari Raspberry Pi Zero memutuskan bahwa satu port USB untuk papan tidak cukup, dan memutuskan untuk menambah beberapa lagi. Pada saat yang sama, ia memiliki hub USB LogiLink UA0160, yang ukurannya hampir cocok dengan papan.



Makanan dulu


Awalnya, itu perlu untuk menyalakan Pi Zero menggunakan hub. Untuk ini, diputuskan untuk menggunakan perangkat USB. LogiLink UA0160 menyingkirkan kasing, dan pekerjaan dimulai.

Koneksi adalah sebagai berikut:
  • Pi Zero PP1 - 5V hub;
  • Pi Zero PP6 - GND Hub;


Menggunakan multimeter, pengguna dapat menentukan bahwa semuanya dilakukan dengan benar.





Sekarang USB


Langkah selanjutnya adalah menghubungkan USB OTG dari Pi ke port USB hub. Ini dilakukan:

  • Pi Zero PP22 - D + hub (pada konektor mini USB);
  • Pi Zero PP23 - hub hub (pada konektor mini USB);






Kami menggabungkan dua papan


Untuk menghubungkan kedua papan bersama-sama, diputuskan untuk tidak repot, dan menggunakan pita elastis yang biasa. Nah, agar tidak menghubungkan kontak kedua papan, karet busa tipis digunakan.







Semua ini cukup sederhana, tetapi proyek yang kompleks, seperti biasa, akan muncul sedikit kemudian. Sekarang pengguna menawarkan solusi sederhana, cara untuk memperluas fungsionalitas papan. Opsi ini tentu saja lebih baik daripada menyolder ke "raspberry" dari modul komunikasi nirkabel . Sekarang Anda dapat menambahkan modul, dan banyak lagi.

Jika Anda memerlukan halaman penulis, maka ini dia .
