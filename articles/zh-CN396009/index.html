[评论]罗技G810 Orion Spectrum。人机工程学奇迹



罗技（Logitech）也许是唯一一家喜欢尝试所有内容的公司。还有鼠标，键盘，表格和馅料。还有谁敢冒险将独特且未经测试的开关放入经典板并声称这是玩家的最佳解决方案？仅Logitech。

很长一段时间，我对它们的键盘持怀疑态度（有些未开发的开关上有什么机械装置？），当我拿起手中的键盘时，我彻底改变了罗技在做什么的整个想法。
而且只能以积极的方式。



箱子及设备



键盘装在一个简单的黑色瓦楞纸箱中，在它的顶部是一个纸板套。套筒采用黑色和蓝色的公司配色方案设计，内容丰富，着眼于设备的游戏方向。键盘显示在套筒上，显示了开关的类型，并描述了一些芯片。


盒子内部是一个装在脆皮塑料袋中的键盘，下面没有任何东西 -使用说明书。好吧，实际上什么都没有。对于任何机械师来说，最必要的事情是用于卸下键帽的专用夹子（在下文中，将使用语术语：用于卸下键帽的拉拔器）。没有它，用手很难取下键帽。

键盘设计与人机工程学



罗技G810是全尺寸机械键盘，带有附加的按键块和音量轮。主要的104个按键建立在Romer-G开关上，此外，键盘还具有其他非机械按键，这些按键负责各种功能，例如关闭背光或打开游戏模式。

该案例以经典的形式执行，但由于存在附加控件，因此上部略有扩大。键盘尺寸（毫米）-443.5 x 153 x 34.3;键盘重量（不包括电缆）-1180克。键盘的所有按键均具有完整的RGB背光。缺少其他连接端口（USB，音频）。


键盘看起来过分时尚-这是您拿起键盘时的第一感觉。立即感觉到这不是“我曾经去过那里的另一种键盘”，而是……异常或类似的东西。也就是说，这是很经典的，但是这种魔幻效果是完整存在的。感觉就像它与许多具有类似经典设计的键盘有所不同。


案例类型-经典，严格，执行整齐，没有不必要的细节。只是一个没有接缝的矩形，并且只有一个地方有一个电缆出口。除了令人愉悦的第一印象外，还感受到了整个结构的坚固性-所有元素彼此之间都很好地契合在一起。甚至没有研究键盘，我也没什么可抱怨的。


键盘材料非常好，但是事实证明，它不是最实用的。板子的顶部覆盖着一块略微粗糙的磨砂塑料面板，该面板不会留下指纹，但仍会收集灰尘。但是整个键盘装饰条都是用光滑的塑料制成的。在业务中……当然，它看起来不错，但是随着时间的流逝，光泽的光泽会消失，会出现划痕，并且从一开始就出现第一个明显的指纹。我仍然不知道为什么普通而实用的磨砂塑料不适合罗技。显然，当他们在制作其他板子时，磨砂塑料已经结束了...


键盘布局正常，但不完全相同。它似乎是美国人，但带有欧洲色彩。也就是说，所有相同的104个标准键，一个长左移键，一个长右移键...以及两层楼的欧洲输入装置立即进入了视野。实际上，这非常好，因为如此多的用户（尤其是老人）更喜欢两层式输入，而不是标准的现代单层输入。


键盘的文本块中缩进了许多F键，同时功能键向左稍微移开，因此“ F1”从“ 2”键向左稍微移开。每个四重F键之间都有明显的缩进。有趣的是：键盘没有“ Fn”修饰符，因为所有附加功能都是单独执行的，并且没有提供Fn + F *的组合。一切都是通过软件完成的，我将在稍后讨论。


键盘的上部由于存在附加键和LED而被放大，以显示修改键（Num，Caps，Scroll）的操作。其他键：多媒体功能，游戏模式（阻止Windows键），关闭和打开背光灯，关闭声音和音量轮。


修饰符面板和其他键与所有其他键一起突出显示，背光非常柔和，根本不会打到眼睛。


键盘上的键帽具有标准配置文件，该配置文件已安装在大多数Cherry和类似键盘上。 G810键帽在其通常的圆柱形表面和受限制的设计上不同于其G410和G910（尽管最近，Logitech官方代表处发布了一条声明，指出新的G910修订版将具有圆柱形表面的标准键帽，如G810）。


尽管键帽的标准轮廓，键盘的底部却有一个不寻常的行：左侧和右侧的控制键比该行中的其他键帽长得多。这是通过减小间隙的长度来完成的，但并没有损害间隙：我只是没有注意到它已经缩短，因此在键盘上打字仍然很熟悉和方便，但是现在您绝对不会错过该控件-它确实很大。


对于那些在主题中的人：经典美式布局中的标准控件+ Windows + alt键帽的比例为1.25 x 1.25 x 1.25，而Logitech G810的左右控件尺寸为1.5，并且其余键的大小为1.25。事实证明，最下面一行的公式为1.5 x 1.25 x 1.25，间隙大小为5.75，而传统的间隙为6.25。对于这些比例的单位，将使用常规键的大小（例如，从文本块中获取）。


现在，让我们仔细看一下键帽。我一生中看到过很多按键，但是我第一次遇到了Logitech专门针对此键盘提出的解决方案。键帽由ABS塑料制成，并涂有油漆。键帽的油漆非常均匀，均匀，键帽本身严格模制而成，内部或外部均无塑料毛刺。


键帽具有出色的涂层：触感光滑，但非常坚韧。在某些方面，它们类似于柔软的覆盖层，只是未涂橡胶。底部照片显示了Logitech G810键帽，顶部显示了通常的激光雕刻键帽（MK Disco键盘）。


干燥，湿润的手指非常有把握地握住这些键帽，不会滑到任何地方。但是这些键帽就像键盘表面一样会吸尘。


这些键帽具有一个独特的功能：在键帽上涂漆后，进行激光雕刻（俄语和英语字符），并且将不寻常的颜色倒入激光雕刻后残留的凹槽中。我什至不知道它是油漆还是某种特殊的聚合物。事实是，没有背光灯，似乎我们面对的是带有普通油漆的普通键帽。因此，所施加的符号在日光下非常清晰可见。


只需打开背光灯，符号就会开始发光，并且光线会穿过键帽而没有明显的障碍物。这是我见过的键帽的最佳实现。最重要的是，键帽触手可及，而其余的键帽都被雕刻了（请再次与MK Disco一起查看照片）。愉快的时刻。好吧，我已经抱怨的最后一件事是缺少设备。 G810键帽很难用手拿掉，因此，我当然希望看到一个用于拔键帽的特殊拔钉器。


俄语和英语字符的样式和厚度都互不相同，因此您不会混淆它们。两个符号都位于中心附近，并且显示非常均匀。这是由于Romer-G开关的LED位于中央。


所有这些的唯一缺点是键帽是非标准的，因此您不能在键盘上放置不同的键帽组。这不仅是由于较大的输入，控件的大小和空间，而且还在于键帽与开关的连接类型-与相同的Cherry MX等不同，此类设置并未单独释放。另一方面-您多久更换一次键盘盖？在这里我很少。


键盘具有由罗技与欧姆龙合作开发的Romer-G开关。这些开关没有咔嗒声，有触觉反馈（按下时感觉到一个响应点）；操作前的按压力-45克，总钥匙行程-3毫米。


如果我们使用最接近的模拟，这些开关在某种程度上类似于Cherry MX Brown开关（触觉，无需点击，触发前45克，距挡块4毫米），但它们与打印体验大不相同。制造商声称其Romer-G的紧迫资源等于7000万次。一个严肃的数字，在实践中已经测试了多年。


默认情况下，键盘具有漂亮的天蓝色背光，然后可以在Logitech专有软件中进行调整。 Romer-G开关的亮点在于SMD RGB LED恰好位于中央，开关的移动部分由半透明塑料制成，这就是为什么光线从不射入眼睛，而在夜晚却非常柔和地扩散的原因。在白天，可以在键帽上绘画而不是突出显示，因此在白天不需要背光。从字面上看，所有内容都闪耀-键，附加键和修饰符。键盘中没有反光的衬背。


Cherry稳定器安装在长键（空格键，Enter键和其他键）下，并且以大多数键盘的典型方式实现-键入时，它们会嘎嘎作响，当您在相同的空间上单击时，节拍声音会在一般背景下脱颖而出。不是很关键，但是不愉快。


键盘的底部是一个单独的对话主题。人机工程学是这里的重中之重。底部面板由纹理塑料制成，是纯粹的装饰元素。键盘底部有多达五个大而柔软的橡胶支脚，可在桌面上提供出色的键盘稳定性。


在激烈的战斗中，既不需要也不采取行动。最重要的是，位于空格键下方的一条腿可防止键盘在手的压力下弯曲。


制造商经常敲打的另一个非常令人愉悦的琐事是可延伸的腿。在这里，它们是两层的，所有这些支脚都涂有橡胶，这就是为什么即使角度不同也不会失去键盘稳定性的原因。


伸缩式支腿甚至带有标志-倾斜角度显示在每一层上（4度-第一层，8度-第二层）。而且它们很容易推入和推出，所有这些动作都伴随着轻轻的点击。好厉害我无话可说了。


键盘电缆延伸到中间的右侧，并且在输出端具有弯曲保护。电缆的厚度适中（比Logitech G502厚，但至少是按顺序排列），包裹在织物编织物中，不会弯曲且没有阻力，可以记住给定的形状。电缆长度-1.8米。


键盘电缆连接器的外观与键盘本身相同-简洁，优雅。
Yusb 2.0，未镀金。


键盘给人留下了非常愉快的印象。她拥有大量的功能，但是所有这些功能都不会干扰正常的操作-相反，这些功能很快就扎根并成为键盘的名片。而且，最重要的是，所有这些芯片都可以工作。

当然有缺点。光泽仍然变脏。但是在一般性能的背景下，这些琐事只会丢失，因此您不应该专注于它们。另外，我想指出出色的制造质量，坚固的结构以及没有任何反冲，裂缝和缝隙。

软件，背光选项



像整个Logitech游戏产品线一样，键盘具有通用软件-Logitech游戏软件。安装后，建议更新键盘固件以使用其所有功能。最新的固件版本添加了G键-F1-F12键的附加功能，您可以在上面敲击任何想要的东西。要更新固件，必须断开并连接键盘，然后开始更新过程，并且此时不应该断开键盘连接。


该软件的主窗口显示一个键盘，突出显示了它的设置可能性。从F行中拨一个键-进入这些设置，在背光键中-继续配置键盘背光。


这是进行更新的G键。简要介绍一下主要内容：无法对键盘中的键进行重新编程。你去。您可以创建配置文件，可以自动运行这些配置文件，只需将配置文件绑定到特定的应用程序，就可以在每种配置文件中为每种口味，颜色和光线添加令人难以置信的背光选项...但是要敲入宏，启动程序或执行除标准点击之外的任何操作其中之一不能。除了那些相同的F1-F12。这只是胡说。

在任何个人资料中，即使是举手最多的人，也可以在任何琴键上对任何动作进行评分。或几乎任何。在这里，只有一系列F键。实际上，不是非常组合Fn + F *。完全无法解释的限制。推动CS的狂热者：带着捆绑带走，办公室工作人员拿着数字键盘上的整个办公程序包就会休息。只有G键，只有铁杆。它们只是替换了标准的F键功能。好吧，也许是最好的-如果您不小心按下“ F1”，将不会出现恼人的帮助。

所有这些都大大降低了为您自己定制键盘的能力。但这还不止于此。美中不足的是：键盘没有内部存储器，因此仅存储了最后保存的设置。是否设置了某些背光？当您关闭软件时，它将重置为“色波”效果，这是默认设置（在固件更新之前存在静态的天蓝色背光）。您只能通过创建了配置文件的软件来进行任何更改。事实证明，键盘高度依赖于软件，并且软件反过来在键盘按键的方向盘上钉上了棍棒。


但是Logitech的背光设置就可以了。共有三种不同的调整选项：效果，区域调整和自由样式。这是效果模式下的照明选项。虽然它们并不多，但是它们很漂亮，可以完美显示G810的背光效果。为了清楚起见，我将所有效果放在红色框中，当前效果显示在右侧。每个效果都有其他设置，例如方向或速度。

但是有细微差别。例如，在反应模式下（在软件中此模式称为“按键”）。在此模式下，按下的键会亮起，然后逐渐淡出。在此，按下按键后，按键会逐渐消失，具有“逐步”效果：按键衰减的等级可见（而在许多键盘中，LED会非常平滑且线性地熄灭）。我怀疑这些是软件缺陷，而不是背光源本身的限制，因为“色波”模式可以正常工作。我希望能够更加详细地了解这些时刻。


但是您应该致敬-背光设置非常好，简单明了。您可以选择通常的固定颜色并享受背光。此外，其他所有内容都可以轻松配置：在标准颜色之间进行选择，有对比度设置（上滑块）和亮度（下滑块）。同样，如果没有软件，将无法进行任何配置。


同样令人鼓舞的是，背光可以在任何地方进行调节-从“ G”徽标和附加按钮到修饰灯泡。键盘不能重现纯白色，但是不可能不需要RGB LED。罗技专门为白色赋予蓝色，这比大多数RGB键盘上常见的紫色或粉红色的杂色更好。图为夜间的标准蓝灯。


通常，键盘背光非常舒适，多汁且柔软。在白天，它不是那么引人注目，但在白天则不需要-由于键帽上的绘画，这些符号可见。到了晚上，亮度已经足够了，即使在最大亮度下，背光源的眼睛也不会疲劳。


游戏模式设置的单独选项卡。他们在屏幕上戳了一个按钮-激活游戏模式后，在软件中选择的按钮将不起作用。您只能禁用Windows键和弹出菜单，或者可以关闭至少一半的键盘。对于创建的每个配置文件，您可以创建自己的游戏布局。


最后一个是点击地图。老实说-几乎没有用的东西。当键盘始终具有完整的统计信息时，这很好。在这里，为了进行统计，必须包括此模式。为什么不总是收集统计信息？即使考虑到可以通过G键对激活的统计信息收集进行评分的事实，您也必须做出额外的手势。我希望看到一个更简单的解决方案。

测试：小费和感觉



通过USB连接时，键盘支持同时按下26个键-键盘具有良好的控制器。遗憾的是没有Full KRO，但是在实践中几乎没有人一次按下所有键。


最重要的是，通常喜欢机械键盘-小费。首先，在任何其他机制的背景下，键盘都非常安静，这立即变得显而易见。在这种背景下，稳定器键侵犯了总体的和平与安静。每次按下时，相同的空间都会明显点击。 G810很可能不适合用作静音键盘。正是由于大键上的稳定器会发出金属弹跳。


至于其余的。咔嗒声令人愉悦，柔软，低沉，几乎无声，按一下键时会沙沙作响。当钥匙放在末端时，对基板的打击几乎是无声的，不明显的。按照惯例，在Romer-G上敲击比在薄膜键盘上敲击要大一点，但比在Cherry和其他开关上轻得多。至少线性（MX黑色/ MX红色），至少非线性（MX棕色）。看起来更像是在MX Brown（称为O形圈，旨在减少噪音并减少琴键行程）上放了橡胶圈，使敲击声更加聋了。按下或返回键时，我没有感觉到任何粘滞现象，尽管在某些Cherry类似物中发现了这种现象。

但是，无论是薄膜键盘还是经典的机械键盘，其按键操作过程都仍然存在巨大差异。按压的性质仍然更接近于薄膜键盘。按键获得了开关本身的柔和敲击和静音操作，但是同时保留了机械冲程，触动开关时触觉障碍非常明显，并且感觉按键回到了原位-没有这种粘性，按下时不会产生“橡皮”。因此，按键的声音要比薄膜键盘的声音要清晰，但要比长冲程的机械声音要明显。有一个“机械”动作，但是由于缩短了按键行程（Romer-G的总行程为3毫米），这些开关被认为是机械的“快速”版本-按下速度更快，触觉屏障稍微更模糊，行程更短，但是纯机械发音按键的性质。

纯粹的感觉-我喜欢它。我真的很喜欢我真的不喜欢薄膜键盘，因为它会给小费，因此我在相当长的一段时间内一直偏爱机械手。在这里，我既能机械移动，又能对所有水龙头做出更快的反应。是的，小费仍然是不寻常的，因为即使有触觉反馈，键盘也给人完全不同的感觉，但这是一种新的体验，您很快就会习惯它。
