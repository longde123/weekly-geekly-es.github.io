黎明探测器发送了谷神星陨石坑的详细图片


库帕洛火山口，谷神星上最年轻的火山口之一。照片：NASA / JPL-Caltech / UCLA / MPS / DLR / IDA（单击图像时，图像将以全尺寸打开）

为了科学的利益，远离地球工作的航天器继续传输有价值的数据。其他世界的照片尤其有趣。前几天发布了冥王星的高分辨率照片，现在“黎明”探测器发送了同样多的高质量谷神星图像。这些照片是在2015年12月19日至23日之间，距小行星385公里的距离拍摄的。

图像分辨率为每像素35米。这样，就拍摄了小行星上最年轻的陨石坑之一库帕洛陨石坑。在这里可以看到一个图，可能是盐。同样，火山口底部平坦，可能是由谷神星与某些物体和碎屑碰撞过程中融化的岩石形成的。专家将研究图片，也许他们可以比较陨石坑Kupalo和Occator的结构和材料组成。


陨石坑。图片：NASA / JPL-Caltech / UCLA / MPS / DLR / IDA（单击图像时，图像将以完整尺寸打开）

“火山口和新形成的沉积物将是项目团队研究的主要目标，” 一位项目参与者说。

科学家在丹徒火山口发现了一个有趣的裂缝网络，其宽度为126公里。有趣的是，科学家们在月球上的第谷陨石坑中发现了类似的裂缝网络。火山口丹徒。照片NASA / JPL-Caltech / UCLA / MPS / DLR / IDA（单击图像时，图像将以完整尺寸打开）
