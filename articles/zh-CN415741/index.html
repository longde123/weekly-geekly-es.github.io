基于IP摄像机的小米MiJia Mi家庭安全360°远程监控和观察患病（老年人）

小米MiJia Mi家庭安全360°摄像机（购买时，在明斯克的价格为140卢布RB（70美元），现在已降至120卢布（60美元）。

在过去的两个三个月中，相机对我个人有很大帮助，但是如果我在不断使用后给人留下深刻的印象，我会发现其中有很多缺点，我将在本文结尾处指出。

也许使用此相机的经验和实践可以对某人有所帮助。

有关摄录机的一些信息


更详细的技术数据可以在官方网站上找到。


我喜欢半球形支架上的球形球型摄像机的设计。 尺寸-小115x78x78毫米。

摄像机的视角水平为360度，垂直为85度。
相机配有USB电源-微型USB 5V，2 A，电缆-约2 m长，以及用于安装的特殊平台。 背面的包装盒上是英语的技术规格。 包装盒中没有连接和操作说明。

为了监控患者，摄像机的主要功能如下：


  1. 可以水平和垂直旋转相机的功能以及对相机的远程控制。 我注意到该机制的静默操作。
  2. 在弱光下拍摄的能力以及背光LED的隐形性。
  3. 运动探测器。
  4. 双向音频通讯的可能性。
  5. 将视频录制到闪存或智能手机存储器中（连接到同一网络时）。

为了使摄像机正常工作，您需要使用IEEE 802.11 b / g / n 2.4GHz无线Wi-Fi标准的Internet连接，以及具有Android或IOS操作系统的智能手机，并具有Internet连接以及已安装的Mi Home程序和相机插件。

相机应用：


首先，我想指出的是，图像质量对我而言并不是最重要的，它具有图像，可以区分一个人的脸部和脸部，这对我来说已经足够了。 因此，我跳过此参数

  1. 标准的远程视频监控。 在我的情况下，对于一个生病的人（最初完全说谎），有必要组织远程监控。 即使与亲戚一起住在同一间公寓中，并且经常在附近居住，仍然经常需要离开一段时间，有时甚至更短，有时甚至更长：有很多理由去商店，药房，诊所。 而且，如果您还需要上班或封闭生活，但又分开生活（例如我的情况），那么摄像机会有所帮助。 即使您自己省心。 借助Mi Home程序，您可以在现场直播模式下远距离观看正在执行的操作，一切正常。 在这种模式下，我建议您关闭相机设置菜单中的状态指示灯，如果将其安装在显眼的地方，LED的蓝灯可能会使患者非常烦恼。 即使在几乎完全黑暗的情况下，背光LED也不明显。


    正常和夜间模式的屏幕截图

    全屏窗口
  2. 通过在安全模式下打开摄像机电源，可以将摄像机放在前门上方，并在必要时收听对话，从而组织进入/退出和安全控制。 您还可以找出病人一个人多长时间，护士到达/离开的时间等等。
  3. 如果患病的人实际上无法使用电话，甚至无法保持电话，更不用说接听电话或拨打电话了，则该设备可以用作双向音频连接，一种对讲机。 通过打开通话功能（程序主屏幕中的麦克风图标），您可以通话，让患者放心并从他那里接收信息。 （不幸的是，便携式摄像机使远程对话者的声音失真，患者需要适应它，而我认为麦克风的灵敏度很低）。
  4. 房间内的音频控制（在某些情况下，我会以最大音量打开电视，但无法降低电视音量，我认为这会给邻居带来一些问题）。
  5. 卧床病人（老人）的控制是人下床的非标准指示。 将其置于安全模式并将摄像头安装在床旁或将镜头略微指向侧面，当患者跌倒时，将触发运动检测器并在电话上收到推送通知，并记录十秒钟的短视频，以供观看。 在后续移动过程中（在设置中，您可以更改通知间隔，直到进行下一个操作：1、5、10、30分钟以及对响应灵敏度的两种调整（低和高））-新通知。
  6. 药物管理控制的组织-在相机中安装存储卡时，可以在录制视频的同时将录制内容保存到存储卡中。

    您可能会遇到这样一个事实，一个患病的亲戚不想，忽略，狡猾并试图扔掉药丸，在某个地方藏起来，只是不喝酒。 再一次，在床上或床下寻找药物，一边吃药，一边按下智能手机上的视频录制功能，我们录制了5-10分钟的视频，以后可以观看。 可以在相机的存储卡或智能手机的存储器中进行记录。
  7. 监控睡眠时间和睡眠质量：通过在晚上设置保护模式并将摄像头镜头稍微移开，可以使用电话通知日志或Mi Home程序查看患者晚上的睡眠情况。


    Mi Home程序中的智能手机通知日志和检测日志。

使用摄像机来控制和监视病人，老人或仅仅是无助（在医院，创伤，手术后）的人的可能性非常大。 他们阻止了我的所有需求。 但是，在我看来，相机的不断使用会带来许多弊端。

使用相机时的缺点：


  1. Mi Home程序的初始安装和相机与智能手机的连接花费了很长时间，因此有必要多次更改位置以退出帐户并重新登录，然后MI相机才能在智能手机上可见。
  2. 我无法通过允许他人访问设备来允许对患者进行监视。 通过将相机安装为共享设备并向另一个Mi Home用户帐户发送邀请，第二部手机上没有消息或通知。 可以在两个不同的设备上使用相同的帐户，但是在这种情况下，不可能同时进行监视。
  3. 即使智能手机可以很好地访问Internet，有时也需要很长时间才能将其连接（“准备播放”）到相机。
  4. 在智能手机上接收消息可能会延迟，您经常会在事件发生的同时收到推送通知，有时会延迟1到10分钟（甚至与相机位于同一Wi-Fi网络中）。
  5. 多次尝试连接时，Mi Home中设备的状态为脱机，进入公寓并使用相机连接到同一Wi-Fi网络时，连接已恢复。
  6. 设备的自动更新：使用摄像机几个月后，一次更新进行了2次，一次是在我接近摄像机时，更新设备后，与相机的连接断开，设备重新启动，但连接没有恢复，我等待了10分钟，最终重启了相机。 在第二种情况下，它离摄像机很远，并且没有物理访问权，因此在更新和重新启动设备后，连接断开（“设备处于脱机状态”），在30-40分钟后恢复了与摄像机的通信，而没有采取任何措施。
  7. 使用动作识别（检测器操作）在小米云服务器上录制视频的时间为10秒，我无法更改此参数。
  8. 运动检测器-并非总是正确地响应运动。
  9. 专有（封闭）通信协议的使用以及与其他设备和程序的兼容性不足。

简要结论：


如果您对硬件和软件都没有很高的要求，那么便携式摄像机的低成本以及对云服务和软件本身的免费访问将克服所有缺点。
