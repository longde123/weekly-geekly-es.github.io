PHP Digest n ° 136 (24 juillet - 6 août 2018)


Nouvelle sélection avec des liens vers des nouvelles et des documents. Dans la version: PHP 7.3.0 Beta 1, PhpStorm 2018.2, Composer 1.7 et d'autres versions, la norme PSR-17, une revue de Yii 3.0, une implémentation alternative de l'API Fibre pour les applications asynchrones, une partie des outils utiles et bien plus sont acceptés.

Bonne lecture!



Nouvelles et communiqués


  • PHP 7.3.0 Beta 1 - Avec la première version bêta, la phase de développement actif se termine, ce qui signifie que la liste des nouvelles fonctionnalités et des changements dans la branche 7.3 peut être considérée comme définitive. La prochaine version bêta est attendue le 16 août.
  • PhpStorm 2018.2
    • Amélioration de la correspondance des espaces de noms
    • Recherche structurelle et remplacement
    • Modèles de modules complémentaires postfix personnalisés
    • Interface mise à jour et prise en charge de la barre tactile
    • ainsi que de nouvelles inspections et autres améliorations .
  • PSR-17: usines HTTP - Une norme a été officiellement adoptée qui régit les interfaces d'usine pour la création d'objets HTTP compatibles avec PSR-7 .
  • PHP 7.2 disponible sur Google Cloud Platform
  • PHPUnit 7.3 - Parmi les changements, il y a la possibilité d'exécuter en premier des tests tombés lors d'une précédente exécution.
  • Compositeur 1.7.0

Internes PHP


  • [PHP]: Cookie de même site - La possibilité de passer un ensemble d'options, y compris la prise en charge de la norme de cookie de même site , a été ajoutée à setcookie() et à d'autres fonctions pour travailler avec les cookies. Déjà implémenté en PHP 7.3.
  • [RFC]: Typed Properties 2.0 - La proposition de propriétés typées a été reportée à la prochaine version majeure de PHP, tandis que Dmitry Stogov a publié des résultats de référence pour évaluer les frais généraux de cette fonctionnalité.

Les outils


  • spiral / roadrunner - Serveur d'applications hautes performances, équilibreur de charge et gestionnaire de processus pour PHP implémenté sur Go. Peut être utilisé pour exécuter PHP sur AWS Lambda .
  • codeplea / ahocorasickphp - Implémentations de l' algorithme Aho-Korasik pour rechercher plusieurs sous-chaînes dans une chaîne. Plus rapide que d'appeler plusieurs fois strpos() , et beaucoup plus rapide que d'appeler preg_match_all() .
  • chekalskiy / php-bank-db - Bibliothèque pour déterminer la banque par numéro de carte. Envoyé par Ilya Chekalsky .
  • igniphp / framework - Framework léger avec prise en charge de PSR-15, PSR-7, et fonctionnant sur Swoole .
  • spatie / phpunit-snapshot-assertions - Un outil pour implémenter le test d'instantanés sur PhpUnit. Compare les résultats du test avec l'exécution précédente.
  • elgentos / masquerade - Un outil pour anonymiser les informations dans la base de données. Prêt à l'emploi prend en charge Magento 2.
  • zendframework / zend-problem-details - Implémente la prise en charge de la norme « RFC 7807 Problem Details for HTTP API » pour les applications PSR-7, qui régule les réponses d'erreur.
  • formapro / pvm - Une bibliothèque puissante pour décrire les flux de travail, les processus métier et uniquement les machines d'état. Interface utilisateur disponible pour la visualisation.
  • Rican7 / entrant - La bibliothèque est conçue pour convertir les entrées brutes de n'importe quelle source en structures fortement typées.
  • rezozero / mixedfeed - La bibliothèque vous permet de combiner les flux de divers services sociaux en un seul flux.
  • KikApp - L'outil vous permet d'écrire des applications natives pour iOS et Android en PHP. Quelqu'un a-t-il de l'expérience à l'utiliser?
