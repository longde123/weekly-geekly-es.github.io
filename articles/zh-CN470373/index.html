PHP内部


  • [RFC]对象初始化程序 -建议实现用于对象的快速初始化的语法：
    隐藏文字 
    class Car { public int $yearOfProduction; public string $vin; } $car = new Car { yearOfProduction = 2019, vin = "1FTFW1CVXAFD54385", }; $car = new Car { yearOfProduction = 2019, }; // throws RuntimeException: Initialization of Car class object failed due to missing required properties
    简要介绍作者帖子中的提案，以及播客中的更多内容 PHP Internals News＃30随它而来。
  • [RFC]重新分类引擎警告 -修改了PHP引擎中的抛出通知和漏洞，并根据PHP 8.0中的投票结果，我们具有：
    •未定义变量：将引发警告；
    •访问数组中不存在的索引：警告；
    •被零除：例外，并且将添加 fdiv()函数，允许被零除；
    •foreach（）中的参数无效：警告；
    • 所有其他更改均通过了投票并被接受。
    在播客中了解更多信息。 PHP内部新闻＃29 c Nikita Popov。
  • PHP项目协调 -只要我们对Internals感到兴奋，那就也有好处。 Dan Ackroyd创建了一个文档 ，该文档描述了该项目的当前任务和讨论。 如果您想帮助PHP但不知道如何使用，则此文档是一个很好的起点。
