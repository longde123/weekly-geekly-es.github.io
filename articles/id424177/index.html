Menghubungkan Grandstream GXW4104 FXO Gateway ke 3CX


Pendahuluan


Pembaruan terbaru 3CX v15.5 Pembaruan 6 menambahkan dukungan untuk FXstream-gateway Grandstream. Saat ini, hanya satu model GXW4104 yang didukung - gateway dimana 4 jalur tembaga “tembaga” dari operator telepon dapat dihubungkan. Ini nyaman jika Anda beralih ke IP-telephony, tetapi karena alasan tertentu Anda tidak dapat menerima saluran melalui SIP dan dipaksa untuk menggunakan saluran analog. Kasus lain - Anda perlu memelihara saluran komunikasi cadangan jika terputusnya layanan Internet dan SIP. Dan contoh ketiga - Anda harus menggabungkan analog PBX dan 3CX lama. Kemudian, beberapa jalur ekstensi pertukaran telepon analog (Ext) dapat dihubungkan ke port gateway (proses penggabungan tidak dipertimbangkan dalam artikel ini).

Dalam artikel ini kami akan mempertimbangkan kasus yang paling umum - menghubungkan ke saluran analog 3CX dari operator telekomunikasi. Untuk ini, templat penyetelan otomatis gateway khusus disertakan dalam 3CX, yang pada dasarnya menyederhanakan tugas.

Penting untuk dicatat bahwa templat dirancang untuk menghubungkan gateway di dalam jaringan lokal atau dari jaringan yang diarahkan langsung, termasuk tautan VPN. Dengan kata lain, templat tidak dirancang untuk menghubungkan gerbang FXO ke 3CX secara langsung melalui Internet.

Mempersiapkan FXO Gateway


Hubungkan gateway ke jaringan melalui antarmuka WAN. Pastikan untuk mereset gateway ke pengaturan default - tahan tombol Reset pada sasis selama 10 detik. pada saat menyalakan perangkat. Kemudian tunggu beberapa menit untuk mengatur ulang dan memulai kembali.

Sekarang Anda perlu menentukan alamat IP dan MAC gateway. Luncurkan Grandstream IP Discovery Tool dan pilih antarmuka yang ada di jaringan dengan port WAN gateway (jika Anda memiliki beberapa antarmuka di PC) untuk mendengarkan.


Matikan gateway, klik tombol Listen, dan nyalakan gateway. Setelah beberapa detik, itu akan terdeteksi oleh utilitas.

Di antarmuka gateway (kata sandi default adalah admin), buka bagian Status - Sistem dan pastikan bahwa versi firmware tidak lebih rendah dari 1.4.1.5. Jika di bawah, Anda harus memperbarui firmware. Di bagian Maintenance - Upgrade / Provisioning, di bidang Path Firmware Server, tentukan URL HTTP untuk mengunduh firmware gateway firmware.grandstream.com. Disarankan juga untuk mengatur opsi Selalu Periksa untuk Firmware Baru.



Menginstal firmware membutuhkan beberapa menit - tunggu hingga gateway siap!

Pengaturan Gateway dalam 3CX


Di antarmuka 3CX, buka bagian batang SIP dan klik Tambahkan SIP Gateway. Tentukan produsen Grandstream dan model GXW-4104. Setel jumlah port ke 4 (kami menunjukkan jumlah port fisik perangkat, bukan jumlah port yang digunakan). Sebagai nomor utama, tunjukkan nomor yang membuat panggilan lebih dulu - biasanya ini adalah nomor terpenting dari empat.

Ketika halaman pengaturan gateway terbuka, buka tab General. Masukkan alamat IP dari server 3CX di sini dan salin URL dari konfigurasi otomatis - Anda akan membutuhkannya nanti.



Pergi ke bagian Pengaturan - Pengaturan SIP Gateway. Di sini Anda harus menunjukkan:

  • Alamat MAC gateway, yang ditunjukkan pada case perangkat (port WAN) atau terlihat di Grandstream Discovery Tool
  • Mode transmisi faks. FAX dalam T.38 Relay Mode direkomendasikan untuk transmisi yang andal. Tetapi jika pengguna memiliki mesin faks analog yang terintegrasi dengan telepon, Anda mungkin perlu menggunakan FAX dalam Mode Passthrough G.711
  • Skema ID Penelepon - parameter ini harus diperoleh dari penyedia layanan Anda atau dipasang secara eksperimental. Seringkali Caller ID ditransmisikan dalam ETSI-DTMF selama mode dering.
  • Nada Panggil, Nada Dering Kembali, Nada Sibuk, Urutkan Ulang Nada - ditentukan oleh standar negara Anda . Untuk negara b. USSR hanya parameter Nada Sibuk yang relevan, yang ditetapkan seperti ini: ch1-4: f1 = 425 @ -11, f2 = 0 @ -11, c = 35/35;



Konfigurasikan aturan masuk dan keluar di 3CX


Dari gateway FXO, server 3CX hanya menerima nomor yang dipanggil (Caller ID), tetapi bukan nomor yang dipanggil (DID). Oleh karena itu, Anda perlu secara manual menambahkan nomor DID ke pengaturan FXO gateway untuk membuat aturan untuk merutekan panggilan masuk dari port FXO tertentu ke Grup, Antrian, atau nomor ekstensi yang diinginkan.



Aturan keluar untuk panggilan melalui gateway juga dibuat sesuai dengan dokumentasi 3CX .

Pengaturan FXO Gateway


Pergi ke bagian Maintenance - Upgrade / Provisioning dan di bidang Config Server Path tentukan tautan konfigurasi otomatis gateway dari antarmuka 3CX yang disalin di atas. Atur parameter yang tersisa seperti yang ditunjukkan pada gambar.


Setelah mengklik Simpan dan Nyalakan Ulang, gateway akan reboot, unduh konfigurasi dari server dan sambungkan ke 3CX.

Ini melengkapi koneksi gateway Grandstream GXW4104 FXO ke 3CX.
