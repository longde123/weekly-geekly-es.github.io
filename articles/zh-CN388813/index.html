联想在CES2016上推出新的ThinkPad笔记本电脑

在拉斯维加斯举行的CES2016展览会上，中国联想公司带来了许多新的ThinkPad设备-笔记本电脑，模块化平板电脑和外围设备，包括外部电池和ThinkPad Stack投影仪。简要介绍一下新笔记本电脑-裁切之下。


联想ThinkPad T460p基于第六代Intel Core H处理器，电池续航时间长达12小时

联想ThinkPad 13


13英寸的ThinkPad 13专为青少年，学生和老师设计。这本超极本重1.45千克，机身厚度为19.8毫米。RAM DDR4的最大数量为16 GB。有两个版本可用-适用于Chrome操作系统和Windows，而键盘则有所不同。例如，Chrome的版本将没有跟踪点。我以前写的比较详细。



联想ThinkPad T460p


T460p建立在最新的第六代Intel Core H上。联想在柏林IFA2015上推出了该系列中的首批带有处理器的设备。客户可以选择处理器，从频率为2.3-2.8 GHz的Core i5-6300HQ到频率为2.5-3.1 GHz的Core i7-6600U。在最大配置中，RAM量将为32 GB。对于数据存储，使用最大1 TB的HDD，最大512 GB的混合HD​​D和SSD。笔记本电脑的重量为1.7公斤，两节电池可以工作12个小时。

价格从1249美元起。




ThinkPad T460p

ThinkPad T460s


ThinkPad T460s的厚度为18.8毫米，重量为1.36千克。这是去年T450的更新版本。RAM的最大数量为24 GB DDR4。笔记本电脑支持LTE-A。14英寸IPS显示器的分辨率为2560x1440像素。该设备一次充电最多可以工作十个半小时。

价格从1059美元起。



ThinkPad L460 / L560


14英寸L460和15.6英寸L560是中端价格段的入门级笔记本电脑：价格分别为750美元和780美元。顶级配置将包括第六代Intel Core i7处理器，分辨率为1920×1080像素的IPS显示器，500 GB的HDD和512 GB的SSD。RAM L460的最大容量为32 GB，L560-16 GB。

两台笔记本电脑都可选配DVD RW驱动器。




联想ThinkPad L560


ThinkPad L460

ThinkPad X1 Carbon


X1 Carbon是联想的一款超便携式笔记本电脑。14英寸显示屏的分辨率为1920×1080或2560×1440像素。设备支持LTE-A。1.18公斤，表壳厚度-1.65厘米。RAM的最大数量为16 GB。最多可提供1 TB的PCIe SSD数据。

新的X1 Carbon将于2月上市，价格为1,299美元。


ThinkPad X1 Carbon

ThinkPad X260


如果您需要更小的设备，Lenovo将提供ThinkPad X260。它被定位为面向企业用户的笔记本电脑。笔记本电脑显示屏的对角线为12.5英寸，分辨率-1920x1080像素。电池可以让您工作长达21小时而无需充电。设备的重量为1.3千克，外壳的厚度为2厘米。在最高配置下，笔记本电脑具有第六代Intel Core i7处理器。RAM的最大数量为DDR4的16 GB。内存容量最大为512 GB。

销售将于2016年2月开始，价格为929美元。
