有关如何根据时间从selevit自动打开和关闭停电的小指南：

1）将来自Github的HTML文件放在某个地方，例如，在/var/www/nginx-blackout/index.html中
2）在nginx.conf中，编辑根目录位置以得到如下内容： 

location /nginx-blackout {
    root /var/www;
    break;
}

location / {
    if ($time_iso8601 ~ ^2019-12-15T09:[0-2][0-9]:[0-9][0-9] ) {
            return 302 /nginx-blackout;
    }
    # ... usual location config
}
