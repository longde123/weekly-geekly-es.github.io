Mempopulerkan sains



Halo

Nama saya Vyacheslav Golovanov, dan selama lebih dari empat tahun saya secara teratur menerbitkan di Habré.

Saya mulai menulis artikel karena saya memutuskan bahwa saya memiliki sesuatu untuk dibagikan kepada orang-orang. Kemudian saya bergabung dengan program hadiah penulis , dan kemudian saya ditawari pekerjaan sebagai editor penuh waktu. Jadi, hampir tanpa terasa bagi diri saya sendiri, saya tiba di sini.

Pada awalnya saya melakukan terjemahan artikel tentang berbagai topik, sering kali tentang pengembangan - misalnya, saya merancang terjemahan seluruh buku web Ekspresif Javascript . Kemudian dia secara bertahap menjadi tertarik pada topik sains populer, mulai membuat artikel berita. Sebagai hasilnya, saya menemukan diri saya dalam terjemahan bacaan panjang - artikel panjang tentang topik sains populer. Saya membuat satu set situs sumber dengan bahan berkualitas, memperkenalkan pengguna Habr ke blog kosmologis Ethan Siegel , fisikawan partikel Matt Strasler , pencipta Bahasa Wolfram, Stephen Wolfram .

Selama pekerjaan saya di sini, Habr, seperti partikel yang tidak stabil, terbagi menjadi tiga bagian, memutar Megamind dan GeekTimes, kemudian bergabung kembali dengan Megamind, dan bahkan setelah beberapa waktu - dengan GeekTimes. Dan kemudian editor situs memutuskan untuk berkonsentrasi hanya pada topik pengembangan, dan tidak lagi menghabiskan sumber daya pada materi sains populer.

Dan kemudian saya tiba-tiba menyadari bahwa saya telah kehilangan sesuatu. Saya tidak hanya terbiasa menerjemahkan artikel tentang psikologi, fisika, astronomi, dan hal-hal menarik lainnya. Saya sangat menikmati bekerja dengan artikel-artikel ini. Ini terutama berlaku untuk topik yang berkaitan dengan astronomi dan kosmologi. Membaca (dan menerjemahkan) artikel tentang bintang, galaksi, Semesta, saya alami gemetar dan takjub. Lubang hitam menyeretku, pikiranku melebar sesuai dengan hukum Hubble, dan kilatan quasar menerangi jalanku untuk mempelajari rahasia dunia kita. Dan tiba-tiba semuanya berakhir.

Saya menyadari bahwa saya tidak ingin mengucapkan selamat tinggal pada ini, saya tidak dapat membantu tetapi terus bekerja dengan topik ini, dan saya harus entah bagaimana melangkah lebih jauh di jalan ini. Dilihat oleh statistik dan umpan balik Anda, topik terjemahan saya menarik bagi banyak pembaca sumber. Dan ketika saya meninggalkan catatan sedih di bawah artikel "terakhir" tentang Semesta , saya menerima banyak komentar, surat dan pesan dengan penyesalan, permintaan (dan kadang-kadang bahkan tuntutan!) Untuk melanjutkan.

Secara alami, ada orang yang menyukai keadaan ini - mereka tidak menyukai apa yang saya lakukan, mereka percaya bahwa bahan-bahan ini tidak diperlukan di sumber daya. Saya bisa memberi selamat kepada mereka atas fakta bahwa tidak akan ada lagi materi seperti itu di sini, dan semoga Anda semua membaca Habr yang baik dan menyenangkan. Selebihnya - saya akan melanjutkan.

Setelah semua ini, saya praktis tidak punya pilihan selain memulai sumber daya saya sendiri untuk terus menerjemahkan bahan-bahan menarik tentang topik sains populer, untuk mempromosikan (seperti yang saya harap sederhana) mempopulerkan sains, untuk menyenangkan pembaca reguler dengan teks, dan untuk berkembang sebagai penerjemah. . Di Habré saya akan terus diterbitkan - tetapi secara eksklusif sejalan dengan kebijakan baru, yaitu, pada topik teknis seperti pengembangan, kecerdasan buatan, robotika, dll.

Kepemimpinan Habr memutuskan untuk mendukung inisiatif ini dengan menyoroti "startup" blog saya ini, di mana saya akan secara berkala menerbitkan materi yang dipilih dari situs saya. Jadi, selamat datang di sumber saya tentang sains, golovanov.net .

Rincian teknis


Habr terutama ditujukan untuk pengembang, jadi saya akan membagikan detail peluncuran proyek.

Hosting


Saya meluncurkan situs pertama saya pada tahun 1998 - tepatnya 20 tahun yang lalu. Sejak itu, banyak terabyte telah mengalir, dan teknologi telah banyak berubah. Untuk waktu yang lama saya tidak berurusan dengan masalah membuat dan hosting situs baru, tetapi masih ada sesuatu yang terdengar, jadi saya memilih apa yang saya dengar - DigitalOcean.

Ternyata meluncurkan situs web Anda sangat sederhana hari ini. Saya mendaftarkan akun, memilih OS, beberapa klik. dan sekarang mesin virtual - setetes, seperti DigitalOcean menyebut mesin virtualnya - siap untuk bekerja (untuk sekarang semuanya bekerja dengan tarif paling sederhana, dan kemudian kita akan lihat). Untuk mengikat domain ke server adalah beberapa omong kosong.

Mematuhi tren transisi bertahap ke Internet terenkripsi, saya menginstal sistem Let's Encrypt. Saya menginstalnya melalui konsol - jika saya mengerti dengan benar, instalasi otomatis melalui panel kontrol DO untuk beberapa alasan memerlukan biaya.

Wordpress


Sebagai sistem manajemen konten, saya memilih WordPress. Sistem gila-gilaan populer, dengan sejumlah besar ekstensi, tambahan, informasi, dokumentasi ... Instal WP di server DigitalOcean? Tidak ada yang lebih mudah. Anda bahkan tidak perlu mengkonfigurasi apa pun - kompleks LAMP yang sudah jadi (Linuix, Apache, MySQL, dan PHP) + WP siap melayani Anda.

Namun, kemudian, saya harus bermain-main. Dengan pilihan tema dan penyesuaian untuk kebutuhan dan ide Anda, dengan prosedur untuk membuat tema anak. Dengan pemilihan plug-in yang diperlukan - tidak semua, tampaknya, fungsi yang diperlukan tersedia di WP "out of the box".

Apa yang akhirnya saya dapatkan:

Formulir Kontak 7 - Mereka mengatakan plugin paling populer dari semua. Hanya berupa umpan balik. Selain itu, plugin Flamingo untuk menyimpan pesan yang dikirim.

Sisipkan Header dan Footer - memungkinkan Anda untuk memasukkan teks di header dan footer halaman tanpa mengedit file tema.

Mistape adalah cara yang mudah untuk melaporkan kesalahan di situs. Pilih teks, tekan Ctrl-Enter.

Plugin Hitungan Tampilan Halaman - menunjukkan jumlah tampilan suatu posting secara langsung di dalamnya (seperti pada Habré)

SendGrid - sayangnya, untuk memerangi spam, DigitalOcean tidak mengizinkan pengiriman surat ke semua pelanggan baru yang layanannya bertahan kurang dari 60 hari. Saya harus mencari perantara. Layanan SendGrid menawarkan pengiriman surat gratis, dengan batasan tertentu - 40.000 surat di bulan pertama, dan kemudian 100 surat sehari.

Login Sosial Wordpress - Saya harus mengotak-atik plugin untuk pendaftaran pengguna yang disederhanakan melalui jejaring sosial. Siapa yang meminta banyak uang, yang tidak bekerja dengan jejaring sosial Rusia, yang sangat buggy. Akibatnya, ia memilih proyek terbuka berdasarkan paket hybridauth, yang digunakan dalam berbagai CMS. Saya menganggap plug-in ini wajib, karena tidak ada yang suka mendaftar sekali lagi. Sangat disayangkan bahwa sistem otorisasi pengguna Khabrovsk tidak memiliki API untuk digunakan dalam proyek pihak ketiga - melalui pendaftaran dalam hal ini akan berguna.

WP Statistics (Verona Labs) - tampilan statistik yang indah

WTI Suka Posting - poskan peringkat (plus / minus)

Yandex.Share - menambahkan tautan "bagikan" untuk sumber daya Rusia (dan tidak hanya).

Peluncuran situs web


Masalah teknis adalah yang paling sederhana. Situs ini aktif dan sedang berjalan, sedang diuji oleh pertempuran. Namun, sayangnya, proyek ini tidak akan bertahan sebagai hobi. Akan ada terlalu sedikit artikel, dan tidak ada yang tertarik. Proyek ini membutuhkan dukungan material.

Sementara saya berencana untuk secara bertahap mengisi situs dengan artikel, dan berpikir tentang bentuk penggalangan dana mana yang paling baik digunakan. Saya berencana untuk mengirim 10 hingga 20 artikel per bulan tentang topik ruang, otak, psikologi, ekologi, dan topik sains populer lainnya. Saya akan senang dengan semua saran dan komentarnya.
