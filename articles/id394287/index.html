Windows Phone ... semuanya? OS hanya menempati 0,7% dari pasar smartphone

Penjualan smartphone di dunia tumbuh sebesar 4%, tetapi pertumbuhan itu tidak memengaruhi perangkat di Windows Phone. Microsoft menjual 2,3 juta smartphone pada kuartal pertama tahun ini, 73% lebih rendah dari periode yang sama tahun lalu.

Jika pada kuartal pertama 2009 smartphone yang didasarkan pada sistem operasi Microsoft Windows Mobile menduduki 10,2% dari pasar smartphone, dan Android hanya 1,6%, sekarang perangkat Windows Phone menyumbang 0,7%.

Dan Microsoft ingin menjual bisnis telepon. Apakah ini berarti bahwa Windows Phone telah berakhir?



Pada tahun 2000, Microsoft memperkenalkan sistem operasi seluler untuk komputer pribadi, komunikator, dan telepon pintar - Windows Mobile. Pada Oktober 2010, ia memiliki penerus: Windows Phone. Smartphone pertama pada sistem ini di Rusia adalah HTC 7 Mozart , penjualannya dimulai pada September 2011. Namun demikian, gadget pada sistem seluler Windows kalah dari semua pesaing - pada kuartal keempat 2011, pangsa mereka adalah 1,9%.

Pada kuartal ketiga 2013, Windows Phone mengungguli beberapa pesaing dan memperoleh 3,6% saham. RIM memiliki 1,8%, Bada 0,3%, Symbian 0,2%. Sulit untuk membandingkan dengan iOS dan Android - mereka menerima masing-masing 12,1% dan 81,9%. Dan pada kuartal ke-3 tahun 2013, Microsoft membeli bisnis telepon Nokia beserta patennya.



Pada kuartal pertama 2015, sebuah perusahaan Amerika menjual 8,6 juta Lumia di Windows Phone. Tahun ini - hanya 2,3 juta . Pada saat yang sama, seluruh pasar ponsel cerdas untuk Windows Phone berjumlah 2,4 juta perangkat, yaitu Microsoft mengendalikannya sebesar 95%. Penurunan penjualan bisa disebabkan oleh kurangnya model Lumia baru. Sekarang Microsoft berencana untuk menjual bisnis teleponnya ke Chinese Foxconn dengan harga $ 350 juta.

Jika Microsoft tidak memperkenalkan model smartphone baru pada Windows Phone, proporsinya akan terus menurun. Vendor lain tidak akan dapat mempengaruhi pasar, karena mereka hanya menyumbang 5% dari semua perangkat di OS ini.
