WebTotem atau bagaimana kita ingin membuat Internet lebih aman

Layanan gratis untuk memantau dan melindungi situs web.



Ide


Pada tahun 2017, tim CARKA kami mulai mengembangkan alat untuk memantau seluruh ruang cyber di zona domain nasional .KZ, dan ini adalah sekitar 140.000 situs web.

Tugasnya rumit: penting untuk memeriksa setiap situs dengan cepat tanda-tanda peretasan dan virus di situs tersebut dan menampilkannya dengan mudah di dasbor keadaan seluruh Kaznet.

Begitulah lahirnya WebTotem - Layanan untuk memantau dan melindungi situs web.





Kami memberikan solusi kami kepada semua pemilik situs web secara gratis. Kami mendapatkan uang dari solusi untuk klien korporat yang perlu memantau sejumlah besar sumber daya (CERT, penyedia Hosting, dll.), Dan pada layanan tambahan untuk pemilik situs web (Uji beban, audit, pengalihdayaan dan bantuan untuk perawatan virus, dll.).

Produk WebTotem


Lebih dari 2 tahun, sistem kami telah berkembang menjadi platform yang lengkap dengan 9 modul yang menarik bagi pemilik situs biasa.

Anda cukup mendaftar di sistem dan menambahkan situs Anda untuk pemantauan. Jika Anda ingin memasang modul perlindungan, maka unduh skrip untuk situs tersebut (Saat ini, skrip untuk situs php tersedia di akun Anda, modul untuk nginx sedang dalam pengujian)



Modul Pemantauan:

Memantau ketersediaan selama hari / minggu / bulan;
Memeriksa validitas dan validitas sertifikat SSL yang diinstal;
Pengumpulan informasi tentang platform tempat situs berjalan (CMS, versi server OS, dll.);
Memeriksa kerusakan di situs;
Memeriksa situs untuk database reputasi (VirusTotal, Spamhaus, dll)
Pengumpulan informasi tentang domain dan tanggal kedaluwarsanya;
Buka pemindai port;

Modul Perlindungan:

Antivirus - Mencari kerang dan backdoor hacker di situs
Web Application Firewall - modul yang dirancang untuk mendeteksi dan memblokir serangan pada aplikasi web

Aplikasi Mobile WebTotem


Dan tentu saja kami ingin membual tentang aplikasi seluler kami, yang memungkinkan Anda untuk memantau dan memantau keamanan situs Anda melalui telepon =)



Versi iOS | Versi Android

Paket


Saat ini, lebih dari 1000 pengguna menggunakan sistem, dan kami memantau sekitar 150.000 situs.

Dari rencana untuk Juni:

1) Luncurkan Plugin Panel Plesk untuk Pemilik Host Web



2) Pisahkan plugin untuk Wordpress



3) Posting pada posting Habr tentang bagaimana kami mengembangkan infrastruktur untuk memantau sejumlah besar situs web, termasuk perincian teknis.

Kolaborasi


Kami akan senang bekerja dengan Anda jika Anda:

  • CERT (Layanan Respons Insiden Komputer)
  • Perwakilan platform CMS, kami memiliki API yang baik, dan kami dapat bersama-sama membuat plugin untuk CMS Anda
  • Sundar Pichai dan Anda ingin menambahkan kami ke Google Webmaster
  • Apakah Anda memiliki pengalaman sendiri yang ingin Anda bagikan dengan kami, sehingga kami akan menerapkannya dalam rilis berikutnya
  • Masukkan opsi Anda =)

Tinggalkan komentar, tulis ide dan saran Anda untuk direvisi. Dan jika Anda memiliki opsi untuk bekerja sama dengan solusi kami, kami senang mendiskusikannya.
