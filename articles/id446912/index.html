Berapa biaya keamanan aplikasi web (menggunakan Barracuda WAF-as-a-Service sebagai contoh)



Jadi, dalam memilih perangkat lunak WAF, perangkat atau layanan cloud, Anda dapat dipandu oleh karakteristik produk, kemudahan penempatan, peringkat produsen, kualitas dukungan teknis, contoh implementasi di perusahaan lain yang serupa dengan Anda, dan ulasan pelanggan yang menggunakan WAF dalam praktiknya. Kami mencoba mengungkap topik ini di posting terakhir kami. Peretas lebih buruk daripada melukis, atau bagaimana melindungi aplikasi web .

Tetapi juga faktor penting ketika memilih produk keamanan informasi, tentu saja, adalah harga.

Anggaran untuk keamanan informasi, di satu sisi, harus sesuai dengan kebutuhan dan kemampuan organisasi Anda, dan di sisi lain, memperhitungkan gambaran nyata dari pasokan pasar. Untuk menetapkan arah pencarian, kami membuat kalkulator harga yang akan membantu Anda memilih model produk yang tepat sesuai dengan parameter perusahaan Anda. Dalam semua rincian yang diperlukan, kami tahu konfigurasi produk Firewall Barracuda Web Application. Oleh karena itu, pada contohnya, kami akan memberi tahu lebih detail tentang fungsi dan parameter yang menjadi dasar harga produk.

Sebelum menghitung konfigurasi, Anda harus memilih model penempatan. Ini bisa berupa perangkat keras atau perangkat virtual, perangkat virtual untuk layanan tertentu (Amazon Web Services, Microsoft Azure atau Google Cloud Platform) atau cloud SaaS (misalnya, Barracuda WAF-as-a-Service ). Itu semua tergantung pada kebutuhan organisasi.

Perangkat keras dan perangkat virtual


Biasanya, perangkat keras lebih kuat. Dengan Barracuda, perangkat keras dapat melindungi hingga 600 server, sementara perangkat virtual hanya dapat melindungi 300 server.



Setiap paket perangkat keras harus menyertakan dukungan teknis. Barracuda juga menawarkan layanan Penggantian Instan - penggantian peralatan yang rusak pada hari kerja berikutnya.

WAF-sebagai-a-Layanan


Keuntungan dari solusi WAF-as-a-service adalah lebih fleksibel: Anda dapat memilih jumlah aplikasi web yang ingin Anda lindungi dan membeli langganan bulanan, sementara perangkat memiliki opsi untuk memilih hanya 1, 3 atau 5 tahun.
