在电池爆炸的报道中，三星召回了Galaxy Note 7智能手机

韩国巨人三星已经宣布召回其新的Galaxy Note 7设备，此前出现了许多有关设备电池爆炸的报道。相应的声明已发布在公司的官方网站上。

根据制造商的说法，他之所以采取这一措施，是因为他“寻求生产最高品质的产品”，并且“非常认真地对待来自客户的事件报告”。

据三星称，总共记录了35起Galaxy Note 7电池爆炸或起火，8月19日在美国开始销售，9月2日在欧洲开始销售。 Galaxy Note 7尚未到达俄罗斯联邦-计划在8月26日在俄罗斯推出Galaxy Note 7，但推迟到9月16日。

许多美国消费者对从移动运营商代表的经销商处获得的相关产品和订阅以及与Galaxy Note 7有关的订阅存在问题，到目前为止，在此问题上还没有一个统一的立场：有人保留了其他功能，而有人没有宣布正式职位。

设备的退货是完全免费的，没有任何其他费用可运送到制造商。

三星移动部门总裁Koh Dong-jin声称，问题的根源是“生产过程中的一个小问题”，但是，这给几十个智能手机用户带来了非常明显的后果。


Galaxy Note 7所有者之一的视频中，电池

“小制造问题” 着火了，该视频将新智能手机变成了一块烧焦的塑料，没有任何恢复的可能：



制造商称，返回将影响约10个国家。三星总共将退回一百万台可能有缺陷的设备，其在美国的零售价为850美元。

同时，Galaxy Note 7配备了与韩国主要市场中国不同的电池，并且不会受到天帝国的反应。

许多用户报告说，当他们使用USB至Type-C的适配器向智能手机充电时，即未使用Samsung原始电缆的情况下，发生了此事件。

这不是三星手机中的电池第一次出现空燃。一次，女主人家口袋里的三星S3电池爆炸的故事广为流传。该女孩严重烧伤了腿，器械变成一块烧焦的塑料。
