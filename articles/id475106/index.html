Pengumuman utama konferensi Microsoft Ignite 2019



Pada konferensi tahunan terbesar Ignite 2019, Microsoft memperkenalkan berbagai alat dan layanan baru untuk pengembangan bisnis dan pekerjaan yang lebih produktif dari organisasi dan karyawan. Dalam posting ini kami telah mengumpulkan pengumuman utama. Lihat di bawah kucing.

Ngomong-ngomong, ada meja yang menarik dengan semua laporan konferensi. Pelajari di sini .

Busur biru




Luncurkan versi awal solusi untuk menciptakan lingkungan hybrid terintegrasi. Azure Arc menyediakan kemampuan untuk memanfaatkan layanan terpadu Azure , alat manajemen, dan alat keamanan di semua infrastruktur, termasuk cloud multi-vendor seperti Amazon dan Google, server di tempat, dan perangkat komputasi tepi.

Microsoft Edge baru




Browser Microsoft Edge pada mesin Chromium untuk Windows dan macOS akan tersedia untuk umum pada 15 Januari 2020 dalam lebih dari 90 bahasa. Hari ini tersedia untuk unduhan evaluasi . Baca detailnya di sini .

Aplikasi Seluler Universal Office




Aplikasi mobile Office serbaguna menggabungkan Word, Excel, dan PowerPoint, menghilangkan kebutuhan untuk menginstal aplikasi terpisah. Dengannya, Anda dapat membuat dan mengedit dokumen, termasuk menggunakan kemampuan untuk mengkonversi foto menjadi teks atau tabel. Aplikasi ini tersedia sebagai versi evaluasi publik hari ini untuk pengguna Android dan iOS .

Silika proyek




Microsoft dan Warner Bros. merekam film Superman 1978 di kaca kuarsa . Perusahaan-perusahaan mempresentasikan konfirmasi pertama dari konsep proyek Microsoft Research Project Silica . Ini menggunakan kemajuan terbaru dalam optik laser dan kecerdasan buatan untuk menyimpan data.

Token Blockchain Azure




Solusi ini memungkinkan Anda untuk mengidentifikasi aset fisik atau digital, mengelolanya dan menggunakannya pada platform blockchain dari berbagai jenis. Menggunakan Azure Blockchain Token, pengembang blockchain akan memiliki akses ke standar Token Taxonomy Initiative (TTI) atau kemampuan untuk menggunakan templat untuk membuat empat jenis token paling populer. Layanan ini menyediakan akses ke satu set API terbuka untuk berinteraksi dengan token. Di antara skenario untuk penggunaan token tersebut dalam bisnis: digitalisasi rantai pasokan, program loyalitas, alur kerja, dan banyak lagi. Saat ini, solusinya tersedia dalam versi awal.

Korteks proyek


Layanan Microsoft 365 baru pertama sejak peluncuran Tim Microsoft. Alat AI ini adalah persilangan antara pengelompokan konten perusahaan dan asisten digital. Layanan cerdas dapat secara otomatis menemukan dan mendistribusikan informasi tentang topik, misalnya, dalam aplikasi Office, Outlook, dan Microsoft Teams, mengidentifikasi para pakar tentang berbagai topik dan membuat "jaringan pengetahuan" organisasi. Solusi ini akan tersedia untuk umum pada paruh pertama tahun 2020, saat ini sedang dalam pengujian tertutup.

Manajemen Risiko Orang Dalam untuk Microsoft 365




Solusi keamanan cerdas untuk organisasi Anda. Ia mampu menganalisis secara anonim dan mendeteksi kasus-kasus perilaku pengguna yang atipikal dan mencurigakan (misalnya, tindakan dengan file) menggunakan algoritme pembelajaran mesin, memungkinkan untuk mencegah banyak risiko yang tidak dapat dideteksi oleh alat dan metode standar. Sementara solusinya tersedia dalam mode pengujian tertutup.

Arus UI




Pratinjau publik dari layanan otomatisasi proses di Power Automate (sebelumnya dikenal sebagai Microsoft Flow). Solusi baru ini menciptakan robot antarmuka untuk otomatisasi dalam aplikasi apa pun, termasuk yang tidak mendukung integrasi melalui API. Dengan demikian, UI Flows dapat menyederhanakan proses proses dalam organisasi secara signifikan, karena tidak memerlukan pengetahuan pemrograman untuk mengkonfigurasinya: pengguna hanya perlu menunjukkan eksekusi tindakan yang ingin diotomatisasikan.
