11 PC papan tunggal terbuka-spec hebat yang ramah-Linux

Harga semua model di bawah $ 200.



HackerBoards.com telah membuat katalog komputer papan tunggal selama beberapa tahun . Sekarang 81 model tunggal terwakili di sana. Saya memilih 11 dari mereka yang memenuhi kriteria berikut: biaya papan harus di bawah $ 200, perangkat lunaknya harus terbuka (Linux atau Android). Papan harus dilengkapi dengan spesifikasi yang diperluas, termasuk diagram terperinci. Setiap dewan harus memiliki komunitas aktif dan dukungan teknis untuk pengembang individu. Papan ideal untuk pengembang dapat disebut sistem seperti itu, yang sering disebutkan di forum dan sumber daya lain yang ditujukan untuk pengembangan di bidang elektronik.

86 Duino Zero



  • Perusahaan / Sumberdaya - DM&P, 86Duino.com
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - DM&P Vortex86EX (1x x86 @ 300MHz)
  • Memori - 128MB RAM
  • Harga - $ 39

Ini adalah platform open source yang kompatibel dengan Arduino yang didasarkan pada prosesor SoC Vortex86EX 32-bit x86. Ini kompatibel dengan Windows, Linux, sistem operasi RTOS 32-bit. Papan ini dilengkapi dengan bus PCIE, DDR3, pengontrol ROM, xISA, I2C, SPI, IPC, Fast Ethernet, FIFO UART, USB2.0 dan pengontrol SD / SATA. Model $ 69 sangat mirip, tetapi sudah memiliki dukungan untuk audio HD dan lebih banyak ekstensi I / O. Ada juga versi mini-PC EduCake.

A20-OlinuXino-Lime



  • Perusahaan / Sumberdaya - Olimex, OlinuXino, Mouser
  • Halaman proyek
  • Prosesor - Allwinner A20 (2x Cortex-A7 @ 1GHz); Mali-400 GPU
  • Memori - RAM DDR3 512MB
  • Port USB-OTG;
  • Konektor SATA + 5V Konektor SATA;
  • Konektor HDMI dengan dukungan untuk FullHD (1080p);
  • Konektor Ethernet 100Mbps
  • Konektor untuk baterai LiPo;
  • Konektor LCD kompatibel dengan display 4.3 ", 7.0", 10.1 "Olimex;
  • Slot kartu microSD;
  • Soket debugging DEBUG-UART;
  • 4 soket GPIO (160 kontak);
  • Indikator LED GPIO, Baterai, Daya;
  • 2K Penyimpanan Alamat EEPROM MAC
  • Harga - $ 37


Mini-PC adalah komputer Linux papan tunggal yang cukup kompak dengan platform perangkat keras terbuka dan antarmuka periferal yang sangat baik. Versi 84 x 60mm dilengkapi dengan antarmuka SATA, yang cukup langka di motherboard jenis ini. Ada microSD, Ethernet, HDMI dan tiga port USB. Anda dapat menghubungkan layar LCD, ada dukungan untuk baterai dan 160 GPIO.

Tepi kotak Andromeda



  • Perusahaan / Sumberdaya - Marvell
  • Halaman proyek
  • Prosesor - Marvell IAP140 (4x Cortex-A53 core @ 1.2 GHz); Vivante GC7000UL GPU
  • Memori - 1GB LPDDR3 RAM; 8GB eMMC
  • Harga - $ 75

PC mini ini menjalankan Brillo OS, berbasis Android. Dewan memiliki komunitas yang cukup besar. Ada spesifikasi lengkap dari sistem, skema, dokumentasi, distribusi Brillo. Sistem ini mendukung standar jaringan Google Weave and Thread. Papan termasuk WiFi, Bluetooth, tiga port USB, port HDMI dan microSD.

Arduino Industrial 101



  • Perusahaan / Sumberdaya - Arduino Srl
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - Qualcomm Atheros AR9331 (1x MIPS24k @ hingga 400MHz)
  • Memori - 64MB DDR2 RAM; Flash SPI 16MB
  • Harga - $ 39


Papan dibuat dalam faktor bentuk sandwich dan mirip dengan Arduino Yun. Di dalamnya terdapat mikrokontroler, prosesor MIPS Qualcomm Atheros AR9331 dengan sistem operasi Linino dan IEEE 802.11b / g / n 1x1 modul WiFi 2,4 GHz. Sistem ini diaktifkan melalui mini-USB. Ada LED.

Arduino Yún Mini



  • Perusahaan / Sumberdaya - Arduino Srl
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - Qualcomm Atheros AR9331 (1x MIPS24k @ hingga 400MHz)
  • Memori - 64MB DDR2 RAM
  • Harga - $ 60


Papan adalah versi yang lebih kecil dari Yún asli. Sistem ini didasarkan pada OpenWrt (Linino), dengan prosesor 400MHz AR9331 Qualcomm. Chip - Atmel Atmega32U4. Papan ini dilengkapi dengan Ethernet, USB, dan microSD.

Tian Arduino



  • Perusahaan / Sumberdaya - Arduino Srl
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - Qualcomm Atheros AR9432 (1x MIPS24k @ hingga 560MHz); Atmel SAMD21G18 Cortex M0 + MCU (48MHz)
  • Memori - 64MB DDR2 RAM; 256KB untuk SAMD21G18
  • Harga - $ 99

Dibandingkan dengan dua papan sebelumnya, yang satu ini lebih produktif. Ini bekerja dengan WiFi SoC Atheros AR9432 yang cepat. Ada dukungan untuk Bluetooth EDR / BLE 4.0a. Modul lain termasuk port micro-USB, 20 pin DIO, 12x PWM, dan 6 pin analog.

Arndale octa




  • Perusahaan / Sumberdaya - InSignal, Pyrustek
  • Halaman proyek
  • Prosesor - Samsung Exynos 5420 Octa (4x Cortex-A15 @ 1.8 GHz dan 4x Cortex-A7 @ 1.3 GHz); Mali T628 MP6 GPU
  • Memori - 3GB LPDDR3e RAM
  • Harga - $ 199


Salah satu papan termahal dalam seleksi. Ini didasarkan pada Samsung Exynos 5420 SoC, ada port HDMI, eDP, dan MIPI DSI, plus port USB. Papan mendukung kamera MIPI-CSI, ada modul nirkabel opsional. Bahkan ada papan yang lebih produktif, ArndaleBoard-K, walaupun harganya sudah $ 259.

Pisang pi m2



  • Perusahaan / Sumberdaya - SinoVoip
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - Allwinner A31 (4x Cortex-A7 @ 1GHz); GPU PowerVR SGX544MP2
  • Memori - RAM DDR3 1GB
  • Harga - $ 58


Pada 2014, proyek Banana Pi dipecah menjadi SinoVoip dan LeMaker. Seperti kebanyakan proyek Allwinner lainnya, dukungan sumber terbuka sedikit berkurang. Dual-core Banana Pi adalah tiruan dari RPi Model B, sedangkan Banana Pi M2 adalah analog dari Model B + dengan konektor 40-pin. Papan ini dilengkapi dengan GBE, WiFi, lima port USB dan sejumlah antarmuka untuk menghubungkan layar atau kamera.

BeagleBone Black, Rev C



  • Perusahaan / Sumberdaya - Beagleboard.org
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - TI Sitara AM3358 (1x Cortex-A8 @ 1GHz) dengan chip PRU MCU; GPU PowerVR SGX530
  • Memori - RAM 512MB; 4GB eMMC
  • Harga - $ 48


Papan yang kompatibel dengan Debian ini muncul sejak lama. Ini dimaksudkan terutama untuk industri. Sangat tidak cocok untuk bekerja dengan konten media. Namun demikian, papan ini cukup populer, dan tahun lalu beberapa klonnya muncul sekaligus. Beberapa proyek bahkan dipamerkan di Indiegogo.

Beagle bone green



  • Perusahaan / Sumberdaya - SeeedStudio dari Beagleboard.org
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - TI Sitara AM3358 (1x Cortex-A8 @ 1GHz) dengan chip PRU MCU; GPU PowerVR SGX530
  • Memori - RAM 512MB; 4GB eMMC
  • Harga - $ 39


Papan diposisikan oleh pengembang sebagai mini-PC untuk Internet of things. Hampir segera setelah kemunculannya, itu menjadi populer, dan sekarang telah memperoleh komunitas besar. BeagleBone Green (BBG) - Berbasis pada platform open-source klasik BeagleBone Black (BBB), dua konektor Grove telah ditambahkan ke dalamnya. Tidak seperti BBB, platform BBG tidak memiliki port HDMI, konektor host USB telah dipindahkan, dan konektor klien USB telah diperbarui ke Micro USB. Perangkat lunak BBB sepenuhnya kompatibel dengan BBG.

Bubblegum-96



  • Perusahaan / Sumber Daya - uCRobotics
  • Penjelasan tentang LinuxGizmos / HackerBoards
  • Halaman proyek
  • Prosesor - Tindakan Tindakan Semikonduktor S900 (4x Cortex-A53 @ 1.8GHz); GPU PowerVR G6230 @ 600MHz
  • Memori - 2GB LPDDR3 RAM; 8GB eMMC
  • Harga - $ 89


Papan didasarkan pada ActDuino prototipe yang lebih. Spesifikasi Linaro 96Boards CE digunakan di sini. Sayangnya, chip GPS DragonBoard tidak ada di sini, tetapi ada SoC yang cepat dan 2 GB RAM. GPU PowerVR G6230 juga cukup kuat, sehingga board ini juga dapat digunakan untuk bekerja dengan media. Di dalamnya ada port HDMI, slot microSD, port micro-USB dan dua port USB, salah satunya adalah USB 3.0. WiFi dan Bluetooth juga tersedia.
