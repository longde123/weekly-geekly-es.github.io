Rilis versi baru plugin DevOpsProdigy KubeGraf

Teman, halo semuanya!

Akhir pekan lalu, rilis yang ditunggu-tunggu (dan adopsi PR membutuhkan waktu lebih dari sebulan) versi baru plug-in kami untuk memantau kubernetes-aplikasi DevOpsProdigy KubeGraf v1.2.0 !
Singkatnya: plugin ini dirancang untuk memonitor aplikasi dan simpul di cluster k8s, dengan itu Anda dapat memonitor sumber daya yang dikonsumsi, jumlah replika aplikasi yang tersedia, informasi tentang bagian sampel sampel liveness / readyness, serta membangun diagram visual semua aplikasi yang berjalan di cluster.

Kami menulis secara terperinci tentang plugin di sini , dan sekarang saya ingin memberi tahu Anda tentang chip-chip barunya.

1) Peningkatan blok dengan navigasi melalui node / namespace. Untuk melihat informasi tentang namespace atau node tertentu, Anda dapat mengklik kotak centang dengan Ctrl.

2) Dalam mode tampilan ikhtisar Node, ringkasan sumber daya pod yang dikonsumsi / diminta pada node kluster tertentu telah ditambahkan ke setiap blok dengan namespace.



3) Jadwal restart kontainer telah ditambahkan ke dashboard pemantauan pod, serta memantau sumber daya yang dikonsumsi sidecar kontainer (misalnya, Utusan ketika bekerja bersama dengan Istio).

4) Bagan dengan sumber daya yang dikonsumsi / diminta dari semua pod aplikasi telah ditambahkan ke dashboard pemantauan pemantauan penyebaran, pengaturan negara dan set dasmon, yang dapat memudahkan saat menyelidiki kemungkinan masalah aplikasi setelah penyebaran.



5) Memperbaiki masalah kompatibilitas dengan versi baru Grafana (6.5.x).

6) Memperbaiki masalah kompatibilitas plugin dengan berbagai instalasi Prometheus. Pada gilirannya, kami sarankan untuk menggunakan instalasi prometheus-operator, konfigurasi yang dapat ditemukan di repositori kami .

7) Memperbaiki metrik untuk penggunaan CPU dan memori pada node, sekarang lebih mirip kebenaran :-)

Anda dapat mendiskusikan operasi plugin, Anda dapat mencari bantuan dalam obrolan telegram kami dan dalam slack .

Dan, tentu saja, kami sedang menunggu bintang Anda, mengeluarkan dan menarik-permintaan di GitHub kami !
