PHP-Interna


  • [RFC] Stringable-Schnittstelle hinzufügen - In PHP 8 wird vorgeschlagen, eine neue Stringable Schnittstelle hinzuzufügen, die Klassen hinzugefügt werden kann, die die Methode __toString() implementieren. Verwenden Sie dann den kombinierten Typ string | Stringable an der Stelle, an der sich der String befinden soll. 
     interface Stringable { public function __toString(): string; }
