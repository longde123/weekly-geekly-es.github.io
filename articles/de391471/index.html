Das zweite Leben eines alten PCs mit Chrome OS



Eines der größten Paradoxe unserer Zeit ist, dass der alte PC mehr Rechenleistung hat als das Topmodell des Chromebooks . So können wir mit Chrome OS einen Computer, der lange Zeit in einem Schrank staubt , wiederbeleben , berichtet theverge .

Mit dem CloudReady- Programm , das vom New Yorker Startup Neverware entwickelt wurde, können Sie einen alten Mac oder Windows-PC in ein Chromebook konvertieren. Die Software verwendet tatsächlich eine modifizierte Version von Chromium, die Google Drittentwicklern zur Verfügung gestellt hat. Heute wird CloudReady meistens auf Schulcomputern installiert, sodass Sie alten Geräten ein zweites Leben einhauchen können. Das Startup bietet auch die Möglichkeit, die für einfache Benutzer entwickelte Software kostenlos herunterzuladen.

Die Verwendung von CloudReady führte zuvor zur Neuinstallation des Betriebssystems und zum Löschen aller auf der Festplatte gespeicherten Dateien. Im vergangenen Monat hat Neverware jedoch eine neue Funktion eingeführt, mit der Benutzer ihres Produkts das Betriebssystem "Dual Boot" verwenden können. Wenn Sie Ihren Computer einschalten, haben Sie die Wahl: Laden Sie Chrome OS oder ein altes Betriebssystem. Auf diese Weise können Sie die Geschwindigkeit von CloudReady genießen, ohne zuvor gespeicherte Dateien zu verlieren.
