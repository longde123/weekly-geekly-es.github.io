Apakah ada kehidupan di bawah Windows 98, bagian satu - tentang perangkat keras

Saya suka menggali dengan besi tua. Mengambil sepotong besi tua dan membawanya ke kondisi yang dapat digunakan memberi saya kesenangan. Saya pikir ini bukan hobi, karena tidak konsisten - saya hanya bersenang-senang dengan apa yang saya dapatkan. Biasanya saya tidak mengejar hal-hal tertentu dan tidak mengumpulkan koleksi, saya mencoba mendekati pilihan "pasien" dari sudut pandang pragmatis - server lama, laptop, dll., Yang masih dapat disesuaikan untuk beberapa tujuan praktis. Misalnya, jika Anda mengambil motherboard prosesor ganda dari paruh kedua yang nol pada Socket 771, maka Anda dapat memasukkan beberapa Xeon dan 32 gigabyte FBDIMM ke dalamnya dengan biaya murah dan mendapatkan platform berbiaya rendah untuk mesin virtual yang membutuhkan lebih banyak memori daripada prosesor.



Tapi sekarang sesuatu menggulung dan hanya ingin merakit dan menenangkan beberapa konfigurasi komputer hanya untuk cinta seni. Menurut saya, ini dimulai dengan unit sistem Pentium 3 yang sudah lama ada, yang pensiun belum lama ini - ia bekerja sebagai mesin tik dan memutar program di FoxPro di bawah DOS, tetapi 1C tidak menariknya lagi. Di sana mereka memasang emotikon Pentium4 modern di mana 1C sudah dimulai dalam waktu yang toleran, dan P3 dipasang di rak.

Dan beberapa hari yang lalu entah bagaimana bertepatan bahwa saya membaca beberapa artikel tentang merakit retroPC pada saat yang sama, menemukan beberapa saluran tematik di youtube dan ingin melihat ini.

Untuk sebagian besar, retroPC dikumpulkan untuk menjalankan game lama - jika ada sesuatu yang tidak sesuai dengan game pada emulator. Secara pribadi, emulator sejauh ini baik-baik saja dengan saya, karena permainan bukan yang saya putuskan untuk mulai dirakit. Lebih sederhana untuk cinta seni. Saya ingin. Namun game masih akan menjadi skala pengukuran majelis.

Assemblies - karena yang universal, tempat gim dari Digger hingga Quake 3 biasa digunakan, sangat diperlukan. Ada beberapa opsi yang memungkinkan penggunaan beberapa trik untuk memaksimalkan jumlah permainan yang biasanya diluncurkan dan membangun komputer yang dapat Anda mainkan baik di Maniac Mansion dan Wing Commander, serta di Unreal Tournament dan Deus Ex. Benar, saya tidak memiliki bagian yang tepat, jadi saya harus menari dari yang ada, perlahan-lahan mencari opsi yang menarik untuk masa depan.

Platform (motherboard + prosesor)


Unit sistem lama, yang dianggap sebagai titik awal. Lebih banyak produksi Vist, kalau ada yang ingat kantor semacam itu.



Saya menduga bahwa motherboard di dalamnya tidak berubah sejak saat pembelian - saya hampir tidak menariknya, itu sangat erat dijepit oleh catu daya (kasing masih salah satu di mana PSU tergantung di atas motherboard, tetapi tidak di atas). Papan ternyata chipset Intel D815EEA - 815, dukungan prosesor hingga Coppermine, memori hingga 512 megabyte (batasan chipset). AGP, 5PCI, AMR, video dan suara tertanam, tetapi tidak ada jaringan.



Menggunakannya sebagai titik awal, saya memutuskan untuk membangun komputer untuk permainan di akhir tahun sembilan puluhan - awal nol. Artinya, keterlambatan DOS dan Windows awal (9x-ME), untuk besi - sekitar 2001-2002.

XP dapat diinstal pada perangkat keras ini, tentu saja, tetapi saya memutuskan untuk berhenti pada Windows 98SE - untuk keaslian. Pada masa itu, XP baru saja muncul dan beberapa orang menggunakan NT / 2000 untuk bekerja dan Internet dan 9x untuk game. Sebagian besar pengguna rumahan tidak repot dan hanya menggunakan 9x sebelum XP. Saya juga memutuskan untuk tidak repot-repot. Plus itu menarik apakah hari ini dimungkinkan untuk menggunakan sistem pada Windows 98 sebagai komputer lengkap (tentu saja dengan mempertimbangkan perangkat keras).

Prosesor pada motherboard adalah Pentium 3 1000EB - inti Coppermine, cache 256 kilobyte, bus 133 megahertz, pengali, masing-masing 7,5. Hampir maksimum yang didukung papan ini (1100 dan 1133 jarang, sehingga dapat dianggap hanya maksimal).



Tapi saya masih ingin tetap menggunakan prosesor pada inti Tualatin di sini. Meskipun motherboard tidak mendukungnya, tetapi merobek, menekuk dan menutup beberapa kaki, Anda dapat menginstal Tualatin di papan dengan dukungan Coppermine. Ya, Anda bisa menggali besi solder di soket prosesor, tetapi prosesornya tidak begitu menyedihkan untuk memilih motherboard. Oleh karena itu, seiring dengan permainan dengan perangkat keras yang tersedia, saya memutuskan untuk mencari Celeron Tualatin. Pentium 3 Tualatin kurang umum dan hari ini mereka meminta banyak uang untuk itu, dan prosesor Celeron masih dapat dibeli seharga 1-2 dolar.

Akibatnya, Celeron ditemukan, dan bukan hanya satu - tetapi dengan motherboard Chaintech 6OJA3T, pendingin asli dan memori 512 megabita. Ada beberapa kapasitor bengkak di papan, tetapi mulai normal, karena saya memutuskan untuk menggunakannya sebagai platform.

Pertama, 6 PCI versus 5 untuk Intel. Kedua, dukungan Tualatin tanpa game dengan besi solder. Dan ketiga, dukungan untuk overclocking tanpa menekuk kaki prosesor. Tetapi bagaimana caranya memiliki Celeron dan tidak menaruhnya di bus 133? Tidak perlu mengaturnya ke 150-170, tetapi ke 133 itu hanya perlu. Smiley.



Conders dengan cepat disolder dan saya perlahan mulai berkumpul.

Memori


Ingatan itu memutuskan untuk meninggalkan yang datang bersama Chaintech. Dalam kasus ekstrim saya akan berubah, ada margin kecil.



Perumahan


Selanjutnya, perlu memilih kasing. Saya tidak ingin menggunakan yang lama, karena saya hanya memilih yang tertua dari yang nyaman untuk perakitan.

Ini ternyata menjadi salah satu kerabat dekat Inwin S500 - sudah dengan lubang di sisi berlawanan dengan prosesor (yaitu, memperhitungkan pentium4 dilakukan), tetapi masih tanpa port USB di panel depan. S506, menurut saya. Saya pikir ini cukup otentik untuk memulai nol. Tersedia lebih otentik hanya kaleng di mana itu menjijikkan untuk menggali.





Tidak ada sangkar hard drive, tetapi ini bukan masalah pada tahap ini. Akibatnya, saya mengaturnya kembali dari kasus lain, atau meletakkan disk di tempat lain - masih tidak berencana untuk menempatkan sejumlah besar disk.

Catu daya mengambil yang pertama dari para pekerja. Semacam 300 watt tanpa nama, dalam "plus" - konektor untuk menghubungkan kekuatan monitor, juga merupakan tanda era itu.

Colokan untuk motherboard tidak normal - yang ada pada case lama tidak pas dengan case baru - pemasangan berbeda. Oleh karena itu, dalam cadangan kami menggali universal, kecuali mungkin pembukaan untuk papan jaringan, tetapi ini tidak menakutkan. Lebih baik daripada meninggalkan lubang.



Kartu video


Sekarang perlu untuk memilih kartu video. Pilihan ideal adalah Voodoo4 atau Voodoo5, tetapi saya tidak memilikinya. Voodoo3, bagaimanapun juga.

Oleh karena itu, dari yang sudah ada, mungkin ada baiknya mengambil GeForce FX5200 atau Radeon 9200SE sebagai yang terbaru dan tercepat. Mungkin nanti aku akan mendatangi mereka.



Tetapi untuk sekarang, saya memutuskan untuk mengambil Matrox G400. Karena Matrox. Ini adalah opsi anggaran dengan memori 16 megabita, tetapi dengan kemampuan untuk menghubungkan dua monitor. Ini sekarang setidaknya dua output pada semua kartu video. Pada masa itu, menghubungkan monitor kedua ke kartu yang sama adalah kesempatan langka, tetapi bagi Matrox itu praktis standar. Plus, Matrox juga membuat kartu video dengan empat output video.



Dan berpasangan dengan G400 saya memutuskan untuk menempatkan 3dfx Voodoo 2. Karena Voodoo. Nah, peluncuran game dominasi 3dfx tanpa menggunakan emulator Glide API.



Siapa yang tidak ingat / tidak tahu - di paruh kedua tahun sembilan puluhan ada konsep "akselerator 3D" - papan sirkuit yang hanya terlibat dalam pemrosesan grafis tiga dimensi. Itu dimasukkan di samping kartu video yang ada. Dapat bertukar data melalui bus PCI, atau terhubung "ke lampu" - monitor terhubung ke akselerator, dan akselerator terhubung ke kartu video dengan kabel jumper.



Kartu suara


Dengan permainan untuk Windows, kartu suara tidak terlalu penting, kecuali dalam hal mendukung berbagai API suara tiga dimensi, Anda dapat memilih (EAX, A3D) dan memutar file MIDI (sintesis FM atau Wave Table, yang kedua lebih baik). Tetapi hanya di Windows, perbedaan dalam kualitas pemutaran MIDI mulai kabur, kartu mulai menggunakan tabel sampel perangkat lunak dalam RAM komputer. Saat ini, perbedaan antara sistem suara built-in dan beberapa jenis audigy, tentu saja, tetapi tidak sebesar antara sintesis FM dan kartu dengan Wave Table di DOS.

Jadi tugas utama ketika memilih kartu untuk retroPC adalah dukungan normal di DOS murni (dan bukan di sesi DOS Windows). Biasanya, kartu suara pada bus ISA diambil untuk tujuan ini - paling sering mereka mencoba untuk menemukan Sound Blaster AWE32 / 64, jika tidak mungkin, apa yang akan muncul dengan tangan, dimulai dengan ESS688.

Dalam kasus saya, tidak ada ISA pada motherboard, jadi saya harus memilih di antara kartu dengan bus PCI dan driver untuk DOS. Di sini orang dibagi menjadi dua kubu - beberapa menyukai Aureal Vortex2, yang lain menyukai Creative Sound Blaster Live!

Saya bukan milik kemah mana pun, menari dari hadapan. Saya memiliki beberapa jenis C-Media 8738 yang tersedia - kartu suara anggaran paling umum di zaman kita, kartu pada Aureal Vortex AU8810 - versi anggaran Vortex dan SB Live! 5.1 Digital (SB0220). Tentu saja, saya memilih SB Live! Ini adalah model yang agak terlambat, karena mereka mengatakan bahwa semuanya tidak baik dengan dukungan DOS, tetapi di tempat lain mereka mengklaim bahwa semuanya baik-baik saja dan bahkan menawarkan driver. Anda harus mencoba.



Kartu jaringan


Tidak ada kartu jaringan bawaan, jadi Anda harus menghabiskan satu slot pada kartu diskrit. Pada awalnya saya mencoba untuk menginstal 3COM 3C905C, tetapi driver entah bagaimana bengkok. Saya terlalu malas untuk mengetahuinya, jadi saya menggantinya dengan Realtek 8139 pertama yang saya dapatkan, yang dimulai tanpa pertanyaan. Mungkin kemudian saya akan kembali ke 3COM, tetapi untuk saat ini Realtek cukup senang.



Kartu ekspansi tambahan


Ada tiga slot gratis yang tersisa - sehingga Anda dapat menambahkan banyak hal.

Pertama-tama, port USB tambahan akan ditambahkan, dua built-in tidak cukup bahkan untuk retroPC.

Namun pada belokan kedua dan ketiga tidak ada ide, juga tidak perlu menambahkan apa pun. Anda bisa SATA-controller, tapi saya tidak mengerti intinya.

Mungkin saja suatu tempat di masa depan SCSI akan ditambahkan. Mungkin satu lagi Voodoo2 untuk mode SLI, jika Anda dapat menangkapnya di suatu tempat. Mereka mengatakan bahwa beberapa tahun yang lalu mereka menjual 300 rubel untuk mereka, tetapi sekarang mereka telah dengan jelas beralih ke status barang antik dan mulai sekitar 3000.

Dan kemudian Anda masih harus memilih standar mana yang Anda butuhkan DVD + R atau DVD-R.

Saya dapat menemukan CD-penulis untuk keaslian, di rak adalah pasangan - Teac (yang ingat) dan NEC. Tetapi dengan DVD-ROM lebih buruk. Tetapi ada drive combo Toshiba 2003. Tetapi jika Anda mengambil 2003 - yaitu, DVD-RW Pioneer pada waktu yang sama. Jadi ada sesuatu untuk dipilih, itu perlu ditentukan oleh kapasitas kerja.

Hard drive


Di sini, beberapa lebih suka menggunakan kartu memori CF atau SD melalui adaptor IDE, tetapi saya melakukan beberapa percobaan dan memutuskan bahwa hanya SSD yang dapat dibandingkan dengan hard drive normal. Tetapi rasional untuk mengatur SSD hanya untuk pekerjaan permanen di komputer, dan hard drive akan turun untuk sesekali dihidupkan.

Saya mengambil Seagate Barracuda 7200.10 80GB. Ini, tentu saja, adalah disk yang agak belakangan, tetapi setidaknya tidak menimbulkan pertanyaan dalam hal kondisi. Berhasil, SMART bersih.
