Robot untuk mengajar pemrograman anak-anak di Arduino



Lingkaran robotika pasti sukses akhir-akhir ini. Lego, arduino, goresan, dan banyak lagi tersedia untuk anak-anak modern (saya yakin banyak, seperti saya, cemburu pada anak saat ini dalam hal ini). Sebagai seorang guru robot, saya memutuskan untuk membagikan beberapa materi yang telah saya kumpulkan selama beberapa tahun kerja, yaitu robot yang cukup sederhana dan beberapa kegiatan dengannya.

Saya menggunakan unit-unit ini di ruang kelas untuk menjelaskan kepada anak-anak prinsip-prinsip pengoperasian driver mesin, sensor garis, dan range finder. Anak-anak yang dikalahkan oleh studi robot ini sudah tidak belajar selama satu tahun dan memiliki ide minimal tentang Arduino, motor dan teknik elektro pada umumnya. Ada banyak proyek di Internet dengan komponen-komponen ini, tetapi dua hal memotivasi saya untuk mengembangkan mesin saya - Saya ingin menggunakan kompartemen untuk 4 baterai AA dan dapat dengan aman memasang hampir semua sensor ke robot (ada beberapa lubang pemasangan tambahan untuk ini dalam desain).




Robot dikendalikan oleh papan Arduino Uno dengan perisai motor l293d, dari sensor saya menginstal range-finder HC-SR04 dan dua sensor garis tcrt5000 (memiliki output digital dan analog).


Badan robot terdiri dari 3 bagian (tidak termasuk dudukan opsional untuk sensor), yang dicetak pada printer 3D. Untuk perakitan, Anda akan memerlukan pengencang M3, saya menjelaskan semua elemen yang diperlukan dalam dokumen yang berisi file untuk printer 3D.


Robot diprogram, baik di lingkungan IDE Arduino standar (anak-anak yang lebih tua) dan dalam program miniBloq (untuk kelompok yang lebih muda). Untuk lingkungan standar, perpustakaan AFMotor digunakan, yang diperlukan untuk bekerja dengan pelindung driver mesin. Di miniBloq, untuk bekerja, Anda harus memilih peralatan robot "ta-bot v1" menggunakan driver yang sama di bagian ini.


Memiliki range finder dan sensor garis di papan, robot ini cocok untuk berpartisipasi dalam kompetisi di banyak disiplin ilmu, tetapi itu diciptakan hanya untuk pelatihan, itu tidak akan berhasil untuk memenangkan hadiah.


Kelas yang bisa diadakan bersamanya:


  • Bekerja dengan pengemudi mesin. Rotasi roda dalam arah yang berbeda dan pada kecepatan yang berbeda.
  • Manuver sederhana. Mengendarai dalam lingkaran, mengendarai angka delapan (dalam kelas ini, sebagai aturan, Anda harus memilih waktu dan kecepatan yang diperlukan berkali-kali untuk mendapatkan angka yang diberikan).
  • Sensor garis. Prinsip kerja. Transmisi pembacaan sensor ke komputer (monitor port).
  • Mengendarai barisan dengan satu sensor garis.
  • Mengendarai garis dengan dua sensor garis.
  • Pencari rentang. Prinsip pengintai. Transfer pengintai ke komputer.
  • Berkeliling ruangan dengan pengintai. Robot berjalan maju untuk menemui rintangan, segera setelah ia bergerak ke dinding, ternyata.
  • Cari musuh (seperti dalam robosumo). Dua robot ditempatkan dalam lingkaran dan mulai berputar di tempat, mencari musuh dengan pencari jarak jauh.
