Esta publicaci√≥n discute el an√°lisis wavelet de series de tiempo. La idea principal de la transformada wavelet corresponde a los detalles de muchas series de tiempo que demuestran la evoluci√≥n en el tiempo de sus caracter√≠sticas principales: el valor promedio, la varianza, los per√≠odos, las amplitudes y las fases de los componentes arm√≥nicos. Un poco de historia

Geof√≠sico D. Morlaix a finales de los a√±os 70 del siglo XX. Encontr√© el problema de analizar se√±ales de sensores s√≠smicos que conten√≠an un componente de alta frecuencia (actividad s√≠smica) durante un corto per√≠odo de tiempo y componentes de baja frecuencia (estado de calma de la corteza terrestre) durante un largo per√≠odo. Transformaci√≥n continua de wavelet (CWT)


Transformada wavelet de un solo nivel:

coefs, frecuencias = pywt.cwt (datos, escalas, wavelet)

donde:

datos: (matriz_) se√±al de entrada;

escalas (array_like) : - La escala wavelet a usar. Puede usar la relaci√≥n f = scale2frequency (scale, wavelet) / samples_period y determinar qu√© frecuencia f√≠sica es f. Aqu√≠, f en hertz cuando sample_period en segundos;

wavelet (objeto o nombre Wavelet ) : - Wavelet para usar;

sample_period (float): - Per√≠odo de muestreo para frecuencias de salida (par√°metro opcional). Los valores calculados para los coeficientes son independientes del per√≠odo de muestreo (es decir, las escalas no se escalan por muestreo de per√≠odo);

coefs (array_like) : - Wavelet continua: transformaci√≥n de la se√±al de entrada para una escala y wavelet determinadas;

frecuencias (tipo matriz) : - Si la unidad del per√≠odo de muestreo es la segunda y est√° configurada, las frecuencias se muestran en hercios. De lo contrario, se supone un per√≠odo de muestreo de 1.

Nota: El tama√±o de las matrices de coeficientes depende de la longitud de la matriz de entrada y las longitudes de las escalas dadas.

El fen√≥meno indicado puede ocurrir en otras wavelets al elegir C y B, por lo tanto, al trabajar con wavelets, debe usar el programa - Listado 1.

Usando el m√≥dulo de escalagrama


Scaleogram es una herramienta conveniente para analizar datos 1D con transformaci√≥n de wavelet continua basada en la biblioteca PyWavelets. El m√≥dulo est√° dise√±ado para proporcionar una herramienta confiable para el an√°lisis r√°pido de datos.

Scaleogram tiene las siguientes caracter√≠sticas:

  • firma de llamada simple para principiantes
  • ejes legibles e integraci√≥n pura matplotlib
  • muchas opciones para zoom, filtro de espectro, implementaci√≥n de barra de colores, etc.
  • soporte para frecuencia y unidades de frecuencia de acuerdo con la marca
  • velocidad, utiliza el algoritmo N * log (N) para transformaciones
  • portabilidad: probado con python2.7 y python3.7
  • mensajes de error detallados y documentaci√≥n con ejemplos
