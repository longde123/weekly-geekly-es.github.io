Layout de teclado estendido russo



O artigo é dedicado a todos os usuários do teclado russo que desejam aumentar a velocidade da digitação.

Os criadores de interfaces estrangeiras raramente se preocupam com sua adaptação a culturas estrangeiras. Um exemplo vívido dessa interface é um teclado de computador amplamente usado por todos (doravante referido como o padrão de teclado 104 ANSI).

Os layouts selecionados nas configurações do Windows são software, ou seja, eles interpretam os códigos de digitalização recebidos do teclado nos caracteres desejados. Os usuários que falam inglês usam apenas um layout de software e não têm problemas com alternância e confusão com sinais de pontuação.

Como usuário escrevendo cartas e preenchendo etiquetas no MS Excel, toda vez que faço uma pausa, quando preciso corrigir rapidamente uma fórmula ou inserir novos dados, passo muito tempo corrigindo erros decorrentes da mudança constante do layout do russo para o inglês e ao escrever letras com a pontuação russa correta, encontro constantemente a necessidade de inserir caracteres que não existem no teclado (-, ",", espaço inextricável).

Eu identifiquei para mim três problemas que atrasam o trabalho:

  • A inconveniência da maneira padrão de alternar layouts.
  • A falta de sinais de pontuação que são frequentemente usados ​​em russo.
  • Marcas de pontuação incompatíveis entre layouts.

A solução para esses problemas pode ser encontrada nos GOSTs existentes.

GOST 6431-75 . Máquinas de escrever. A localização das teclas e caracteres no teclado.

De acordo com este GOST, os sinais de pontuação usados ​​com freqüência estão localizados na faixa digital do bloco de teclas alfanuméricas. Os números são discados usando a tecla de transferência de registro. Para inserir uma vírgula, você não precisa traduzir o registro todas as vezes. A vírgula e o ponto são digitados com os dedos indicadores. A letra está na letra, não no alcance digital. Este GOST corresponde ao único layout russo alternativo no Windows - datilografado.

GOST 14289-88 . Instalações de informática. Teclados Arranjo de teclas e símbolos, funções das teclas de controle.

O GOST foi desenvolvido no momento em que o IBM Model M estava se tornando o padrão.
Ele fornece duas soluções simples para alternar problemas de layout:

  • Com uma tecla que fornece uma seleção constante do modo de entrada das letras do alfabeto russo (CapsLock).
  • Use a tecla para alterar temporariamente o modo de entrada dos caracteres dos alfabetos russo e latino para teclas alfabéticas e ativar temporariamente o terceiro caso para teclas com números (Alt à direita).

Em todos os layouts, o CapsLock é usado para incluir uma camada "maiúscula". Mas é possível incluí-los não em maiúsculas, mas em um alfabeto alternativo.


Fig. 1. Layout dos EUA

Muitos layouts nacionais têm camadas adicionais ativadas temporariamente com a tecla alt. É possível criar várias camadas adicionais usando alt + qualquer tecla de um bloco de texto.

Assim, em um layout, você pode ajustar vários alfabetos e todos os tipos de caracteres especiais.


Fig. 2. Layout russo do Windows

Inspirado por essas decisões, decidi trazê-las à vida. Não consegui criar ou instalar utilitários especiais. O Microsoft Keyboard Layout, o programa oficial da Microsoft para criar layouts para qualquer idioma, veio em socorro.

Foi possível resolver todos os problemas apenas criando um único layout russo-inglês com os mesmos sinais de pontuação para os dois idiomas.


Fig. 3. layout estendido russo

Para combinar sinais de pontuação, tive que abandonar os números no intervalo digital (existe um bloco digital), colocar um ponto no número 8 e uma vírgula no número 5. A maneira mais fácil de acessar essas teclas é com os dedos e não há terceiros caracteres neles.

Para a possibilidade de usar layouts em um laptop, coloquei os números na camada alt.

Ele deixou os sinais restantes em seus lugares e acrescentou os sinais frequentemente usados ​​na escrita russa.

O que me deu:

  • A capacidade de escrever com rapidez e competência textos em russo a partir de qualquer aplicativo.
  • Não pense se estou no layout correto para digitar este ou aquele símbolo.
  • Alterne o layout do CapsLock sem utilitários.
  • Alterne temporariamente o layout pela alt correta.
  • Os programas não alteram mais o layout. Ao iniciar qualquer programa, você permanecerá no alfabeto para o qual mudou anteriormente.

Bônus

Para um conjunto de sinais e letras de energia do grego e de outros alfabetos, adicionei outra camada, cuja ativação você precisa digitar alt + \.
