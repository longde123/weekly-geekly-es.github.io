Total Commander: Standardmäßig im aktuellen Ordner suchen

Total Commander hat ein Problem mit der Suche. Standardmäßig wird dies in allen Unterverzeichnissen durchgeführt. Sie haben beispielsweise einen Download-Ordner, in dem sich eine Reihe von Unterverzeichnissen und Dateien befinden. Sie wissen, dass sich die benötigte Datei im Ordner selbst befindet, aber TC sucht standardmäßig auch in allen Unterverzeichnissen, was einige Zeit in Anspruch nimmt. Mir ist aufgefallen, dass ich grundsätzlich nur im ausgewählten Ordner nach etwas suche. Dies bedeutet, dass dies standardmäßig effizienter ist. Das Problem ist, dass es in TC keine solche Möglichkeit gibt. Aber es gibt eine Krücke!

Sie können eine sogenannte „Suchvorlage“ erstellen und diese einer Tastenkombination anstelle der üblichen (oder einer beliebigen anderen) zuweisen.

1. Öffnen Sie die Suche (Standard Alt + F7)
2. Wählen Sie die gewünschte Verschachtelung aus und löschen Sie die Zeile "Suchort".
3. Registerkarte "Suchvorlagen"
4. Schaltfläche „Speichern“
5. Es ist besser, die Vorlage zu benennen, damit später klar wird, um was es sich handelt. Zum Beispiel ThisDir. Wir schließen.





1. Gehen Sie nun zu Extras> Befehlsliste
2. Kategorie usercmd.ini
3. Neu. Sie können es sinnvoll nennen. Ich habe es so belassen, wie es war, weil Faulheit auch andere Teams nicht gemacht hat.
4. Klicken Sie im Feld „Befehl“ auf die Lupe. Für eine schnelle Suche in der "Maske" geben Sie loadearch ein und wählen Sie den Befehl "LOADSEARCH gespeicherte Suche".
5. Geben Sie im Feld "Parameter" den Namen Ihrer Vorlage (ThisDir) an. Das ist auch alles.





1. Gehen Sie nun zu Konfiguration> Einstellungen> Verschiedenes
2. Wählen Sie im Abschnitt „Hotkeys vordefinieren“ die gewünschte Kombination aus. Zum Beispiel Strg + F.
3. Im Feld "Team" wird möglicherweise das Team angezeigt, das dieser Kombination bereits zugeordnet ist. Sie ändern es einfach zu Ihrem. Klicken Sie auf die Lupe und suchen Sie im bereits bekannten Fenster Ihren Befehl in usercmd.ini. Jetzt abhaken.



Wenn Sie jetzt Strg + F drücken, wird die Standardsuche nur im ausgewählten Ordner ausgeführt.

Ich habe diese Methode einmal im wincmd-Forum gefunden (es scheint hier ), aber ich habe beschlossen, sie verständlicher zu gestalten.
