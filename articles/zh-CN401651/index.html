八岁的MacBook如何通过SSD升级再次变得“伟大”，以及有关苹果笔记本电脑老化的一些问题

嗨％用户名％！ 毕竟，您还因为将“经过时间考验的MacBook”发送给垃圾邮件而感到生气，仅仅是因为“本季佩戴带有适配器的机型很时尚”吗？ 由于怀疑这匹老马不会破坏沟渠，仅仅是因为金融危机，我们了解了八岁的MacBook的疮痛，并赞赏“固件”在升级驱动器后变快了多少。



数字技术的运作可以从不同角度进行。 例如，从“已购买，已使用，扔掉/出售”的位置对其进行评估，以确定多付水平最低和最佳销售时间（直到二手市场价格下跌）的设备。 反之亦然，升级所有可以升级的内容，并延长计算机设备的寿命，直到它们变得异国他乡。 最好不要走极端，不要让计算机“跳到头顶”，消除故障，并在每次将老化模型“ 发送到停尸房或重症监护室”时进行权衡。

老年人需要注意


如果MacBook Pro Mid 2009过时了，那么肯定不是设计上的问题-如果笔记本电脑没有掉落，没有“下沉”或没有被部分压扁，则没有任何东西可以体现出它的“里程”。 问题由内而外激增。 有时它们会在字面意义上膨胀。

因此，旧MacBook的日常生活可以分为：



细目：

• 电池 。 MacBook几乎总是可以确定“电池”的磨损程度，并需要对其进行更换。 上帝保佑他，放电很快，但是电池自然会在铝制外壳内膨胀。 如果您进一步忽略更换电池的操作，则可能会损坏计算机。 有关在不认真开玩笑的情况下电池会发生什么情况， 请与韩国工程师联系 。

• SATA电缆 。 但是，他似乎没有任何理由失败，但即使在笔记本电脑中驱动器升级成熟之前，也只有电缆“嘎嘎作响”。

升级：

• RAM 。 曾几何时，在银河系很远的地方有一些Macbook，可以增加RAM的数量。 现在已经没有这种可能性了，因此，“钉住钉子”的16 GB引起了公众的关注 ，但是在使用八年的旧型号中，使用一对DDR3 SO-DIMM模块将RAM容量从2 GB增加到8 GB并不困难。 遗憾的是，在运行Core 2 Duo 橡树皮的芯片组中，内存以系统总线频率（即1066 MHz）运行。 但是容量不再令人满意，因为今天8 GB已成为多媒体计算机的主流。

• 驱动器 。 您还记得Core 2 Duo时代廉价版MacBook Pro拥有多少存储量吗？ 我们提醒您：160 GB，不是在SSD上，而是在具有SATA-II接口和5400 rpm主轴速度的硬盘上！ 当然，该驱动器必须先升级到1 TB HDD，然后再升级到固态驱动器，这将在下面讨论。

旧MacBook中的新SSD：值得吗？


在我们的MacBook购买了当前的RAM数量和新电池后，您能做的最好的事情就是升级驱动器。 与带有大量胶水和复杂螺丝的现代笔记本电脑不同，2009型号年度的“固件”绝对不能排除这种干预。

我们卸下盖子上的10颗螺钉，再卸下驱动器支架上的几颗，然后卸下我们的硬盘驱动器。 没有像垫圈广告中那样的自我怀疑或不适感。

我们使笔记本电脑可以在标准界面中快速工作……SATA“通常是两个，但有时一个”


如果您忘记了过去的Macbook的功能，并想知道它们在现代化方面的友好程度，那么这就是旧MacBook Pro Mid 2009的设计“精髓”，这是制造商的构想，这些型号可以根据SATA-II规范与驱动器交换信息，也就是说，速度高达3 Gbit / s。 确实如此，但有一些警告-该驱动器应该是“ Apple官方推荐的”。
否则，数据传输速率将降低到1.5 Gbit / s，即SATA-I已经具有的值。 不久之后，苹果发布了软件更新 ，消除了MacBook中非品牌组件工作中的荒谬事故。 结果，旧MacBook中驱动器的速度（速度级别SATA-I或SATA-II）开始取决于……随机性，并且在很多情况下不赞成移动计算机的所有者。


现在，可以认为运行Intel Core 2 Duo的旧MacBook升级已经完成

便宜又快速的金士顿UV400将尝试振兴旧款MacBook

今天，我们的病房正在尝试使用金士顿驱动器-UV400 。 这是该公司的最新型号之一，重点在于以合理的价格提供出色的功能，并在金士顿驱动器中使用了罕见的Marvell 88SS1074控制器。 与基于LSI-Sandforce和Phison控制器的驱动器不同，Marvell允许您非常灵活地“调整”软件部分，这对于在不希望节省闪存质量的情况下快速实现预算模型非常重要。


已有五年历史的昂贵硬盘（2012年！）

现代驱动器与新型快速PC更好地展现了它们的潜力，因此欢迎所有将UV400视为新型笔记本电脑/台式机中的“机车”的人进行详细的评论 。 但是今天，我们对最新驱动器如何与笔记本电脑一起使用感兴趣，笔记本电脑的使用寿命与二年级学生相同。

应该注意的是，根本不存在用于MacOS中测试驱动器的公认的广泛方法，根本没有像Apple生态系统中的Iometer-CrystalDiskMark-ATTO磁盘基准之类的“行业标准”出现，因此今天您将发现最少的“捉鹦鹉”和更多的实际测量值。应用程序。

首先，在BlackMagic Disk Speed Test实用程序中检查驱动器的性能。 该测试是由专业摄像机的制造商开发的，因此，除了在顺序操作中的性能外，它还演示了驱动器对于读取或写入特定视频格式的适用性。

SSD结果



应该注意的是，尽管SSD在SATA 3Gb / s模式下工作，但是顺序读写的实际速度却在SATA-I和SATA-II之间。 但是，即使在这种情况下，在顺序操作中，SSD的写入速度也比2012型号年的TB硬盘（8 MB高速缓存，5400 rpm，SATA-II）要快94％，读取信息要快150％。 因此，便携式计算机可以处理的视频格式表也正在转换。 但是，不要指望GeForce 9400的硬件加速已经扩展到现在（毕竟，笔记本电脑是在发布4K视频之前很久才发布的），而只是评估数字上的差异以说明HDD和SSD的整体性能。

最主要的不是大小，而是技巧


在现实生活中，从外部驱动器或千兆位互联网连接复制“哇”文件时，您将需要峰值带宽。 铁的其他品质也走在前列。

就像在汽车中一样，重要的不是最大速度，而是加速和控制的动力学特性，因此在驾驶中，值得注意的是“穿梭行驶”中的最大速度，但值得注意的是跑步者摔倒并进入转弯的时间。 对于固态硬盘，我们正在谈论的事实是这种驱动器能够更快地处理一系列小文件（即，主要是服务类型的系统“内脏”），并且在启动应用程序时以更大的敏捷性释放出RAM。 这是日常生活中这种差异的例证。



如您所见，更快的驱动器几乎对打开网站的速度没有影响（尽管我们没有通过大量调用外部资源和厚厚的javascript 巧克力来测试“核心”资源）。 但是该系统的发布速度提高了170％，不是百分比，而是感觉-就处理器和图形性能而言，2009 MacBook的性能仅次于基于Intel Core M的现代平板电脑和超极本，并且是过时设计中唯一的“弱链接”在响应时间和整体性能上都有一个史前的驱动器（您仍然找不到标准的160GB烂东西！）。 但是您还需要考虑的是，SSD在多线程负载中不会“崩溃”，但是单项测试并没有以任何方式说明。

升级后，MacBook再次成为“打字机”和Internet终端设备的角色变得很方便-即使在2017年，这也已经非常不错。 不相信-看一下2008年和2017年智能手机之间的明显差异。 比较相似的手机至少会被不同时代的设备所感知。

简而言之，即使对于运行Intel Core 2 Duo处理器的笔记本电脑，购买SSD也很有意义。 不，不是这样-如果将RAM和电池置于当前状态，这是唯一可能的情况。 由于笔记本电脑不再适合大量游戏或处理视频和图形阵列（它们很好，但您不想这样做），而廉价的现代适度固态硬盘非常适合将您的专业方向更改为“为所有事物提供一个家”。

当专业人员装满酷炫的现代计算机并榨干所有果汁时，旧的高端PC及其在Apple阵营中的同类计算机正在老化，在“社交网络-音乐-电影-文档”场景中，升级驱动器时将与中产机型没有区别。 不要放弃过去的“战斗车辆”，不要仅通过读取和写入海量文件的峰值速度来评估驱动器的性能。 拥有良好的升级和可靠的设备！
