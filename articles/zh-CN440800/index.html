Kali Linux 2019.1发布



Kali Linux 2019.1滚动发布，其中包含许多更新：更新的软件包，更新的内核4.19.13，更新的工具以及著名的Metasploit框架的版本5。


Kali Linux是一个发行版，其中包含许多用于进行渗透测试的实用程序-从分析Web应用程序的漏洞到入侵网络和服务并将其修复到系统中。 此发行版本以前称为Backtrack。


最新消息


当然，该版本的主要功能是支持Metasploit框架的第5版，该版本是为渗透测试而设计的框架的最新版本。 值得注意的是，该支持自去年开始就在开发分支机构中实施，但直到现在才进入稳定版本。 


root@kali:~# msfconsole , , / \ ((__---,,,---__)) (_) OO (_)_________ \ _ / |\ o_o \ MSF | \ \ _____ | * ||| WW||| ||| ||| =[ metasploit v5.0.2-dev ] + -- --=[ 1852 exploits - 1046 auxiliary - 325 post ] + -- --=[ 541 payloads - 44 encoders - 10 nops ] + -- --=[ 2 evasion ] + -- --=[ ** This is Metasploit 5 development branch ** ] msf5 >

Kali Linux 2019.1还包括针对Harvester，DBeaver等的更新软件包。 Kali Bug Tracker 更改日志中提供了更新，补丁和加载项的完整列表。


当然，流行的ARM映像并没有被遗忘：在Kali 2019.1中，ARM返回了对基于4.19内核的Banana Pi和Banana Pro的支持。 对于配备TFT LCD的用户，没有单独的Raspberry Pi图像，因此re4son kalipi-tft-config脚本现已全部可用，因此，如果要配置TFT板，请运行“ kalipi-tft-config”并按照说明进行操作。


安装与升级


您可以从官方站点下载磁盘或虚拟机的最终映像，或更新您的发行版。


首先，检查存储库的相关性： 


 root@kali:~# cat /etc/apt/sources.list deb http://http.kali.org/kali kali-rolling main non-free contrib

另外，在升级（apt -y完全升级）之前，您可能需要重新引导： 


 root@kali:~# grep VERSION /etc/os-release VERSION="2019.1" VERSION_ID="2019.1" root@kali:~# root@kali:~# uname -a Linux kali 4.19.0-kali1-amd64 #1 SMP Debian 4.19.13-1kali1 (2019-01-03) x86_64 GNU/Linux

最后一步是更新发行版： 


 root@kali:~# apt update && apt -y full-upgrade
