Dracula Theme - tema universal untuk hampir semua hal


Pendahuluan


Halo semuanya, sekali waktu saya berurusan dengan sejumlah besar editor teks dan IDE yang berbeda pada berbagai sistem operasi. Kemudian saya sangat terganggu oleh momen sedemikian rupa sehingga masing-masing memiliki tema desain kode sendiri, ditambah beberapa yang lain untuk dipilih. Ada yang baik dan buruk, tetapi semuanya berbeda, yang bagi saya pada waktu itu adalah masalah dan terus-menerus harus terbiasa dengan desain baru.


Saya mulai mencari solusi untuk masalah ini di Internet. Saya hanya memiliki dua kriteria untuk apa yang saya cari.


  1. Skema warna tunggal untuk semua editor teks dan IDE populer.
  2. Ini adalah topik yang gelap, karena secara pribadi jauh lebih mudah bagi saya untuk menulis teks atau kode dalam desain seperti itu.

Tema Dracula


Setelah pencarian singkat, saya menemukan situs draculatheme.com .

Situs ini adalah tema universal untuk berbagai aplikasi - Tema Dracula ( lisensi MIT ).

Di situs ini saya menemukan apa yang saya butuhkan. Ada topik untuk hampir semua yang saya butuhkan atau secara teori bisa dibutuhkan di masa depan. Ada lebih dari 50 aplikasi yang didukung, jadi saya tidak melihat alasan untuk mencantumkan semua yang ada dalam publikasi ini, Anda dapat melihat semuanya di situs.


Sebagai hasilnya, saya menginstal Dracula Theme pada perangkat yang berbeda, dengan sistem operasi yang berbeda, khususnya pada Sublime Text, Notepad ++, Terminal, Vim, mungkin saya sudah lupa sesuatu, tapi bukan itu intinya. Demi hiburan, saya menginstal tema yang sama di Telegram dan Slack, meskipun hanya mengubah bilah sisi. Di bawah ini adalah tangkapan layar dari apa yang tampak seperti Tema Dracula di Teks Luhur.



Instalasi


Menginstal Tema Dracula sederhana, di situs web untuk setiap aplikasi terdapat instruksi untuk menginstal tema. Anda juga dapat mengkloning atau fork file dengan tema untuk setiap aplikasi dari pengembang GitHub . Di bawah ini adalah contoh tema kloning untuk Teks Sublime.


$ git clone https://github.com/dracula/sublime.git "Dracula Color Scheme"


Jika Anda tidak ingin mengkloning dari GitHub, Anda cukup mengunduh arsip dari tautan di situs web pengembang.


Bagian visual


Adapun bagian visual dari topik, saya menyukainya, tetapi saya mengerti bahwa ini bumbu. Saya mengerti bahwa orang lain mungkin tidak menyukainya. Tetapi ada masalah karena ada beberapa analog serupa yang dibuat untuk semua aplikasi populer. Di bawah ini adalah beberapa screenshot tema untuk aplikasi populer dan pesan instan.

Kesimpulan


Saya harap seseorang seperti tema universal berguna. Walaupun saat ini saya tidak memiliki situasi di mana saya harus bekerja banyak dalam sistem operasi dan aplikasi yang berbeda, tetapi sampai hari ini saya menggunakan Tema Dracula.
