Stellen Sie sich vor, Sie sind nicht nur so angekommen, sondern mit einer sehr wichtigen Mission. Ich meine, Sie wissen, dass alles weitere für Sie eine speziell inszenierte Theateraufführung sein wird, aber auf dem Weg werden Sie durch das halbe Land reisen.

Spielmechanisch sieht es so aus:

  1. Es gibt 10 Punkte: Sehenswürdigkeiten und Übernachtungsmöglichkeiten.
  2. An jedem Punkt wird ein Theater- und Spielblock hinzugefügt: Bedingt wird der Ort für eine Mini-Präsentation vorbereitet, die Schauspieler zeigen Ihnen etwas, Sie treffen einige Entscheidungen mit Aktionen oder Untätigkeit.
  3. Dann erhalten Sie ein Quest-Puzzle, um es zu lösen, das Sie herumlaufen müssen, z. B. Fliesenstücke im Labyrinth der Höhlen aufheben oder die Koordinaten entschlüsseln und (vorzugsweise auf einem Pferd) zu einem Punkt fahren.
  4. Am Ausgang finden Sie einen zweiten Handlungsblock, meistens ein Treffen mit einem Schauspieler, der neue Informationen, eine Videobotschaft oder etwas anderes gibt.
  5. Plus "Gelegenheitsbegegnungen" auf der Straße, wie im klassischen Rollenspiel.
