PHP Digest Nr. 132 (27. Mai - 10. Juni 2018)


Neue Auswahl mit Links zu Nachrichten und Materialien. In der Version: PHP 7.3.0 Alpha 1, Yii 3.0 Alpha 1 und andere Versionen, ein neues Buch über asynchrones PHP, Videoaufzeichnungen von Berichten, einen Teil nützlicher Tools und vieles mehr. Viel Spaß beim Lesen!


Nachrichten und Veröffentlichungen


  • PHP 7.3.0 alpha 1 - Die erste Alpha-Version des neuen PHP-Zweigs steht zum Testen zur Verfügung. Ein sechsmonatiger Testzyklus hat begonnen, die endgültige Veröffentlichung wird Ende November erwartet.
    • Unter den Änderungen:
      • Verbesserte Heredoc- und Nowdoc- RFC- Syntax
      • Die Funktionen json_encode () und json_decode () lösen im Fehlerfall eine Ausnahme und das entsprechende RFC- Flag aus
      • Möglichkeit, nach dem letzten Argument der RFC- Funktion ein Komma zu verwenden
      • Neue Funktion hrtime (), die eine hochpräzise Zeit zurückgibt und nicht von der Systemzeit PR abhängt
      • Die Funktion is_countable () wurde hinzugefügt, um die RFC- Aufzählung zu überprüfen
      • Verwenden von Links in list () des RFC
      • Deutliche Verbesserung des PR- Garbage Collectors
      und andere Verbesserungen. Vollständige Liste der Änderungen .
  • Phalcon 3.4.0
  • Neueste PHP-FIG-Nachrichten: April und Mai - PSR-5 (PHPDoc) ist in zwei Bereiche unterteilt: Formatierung und Tag-Katalog, PSR-9 und PSR-10 (Sicherheit) wurden mit einem neuen Marktführer, PSR-14 (Event Dispatcher), zum Leben erweckt , PSR-17 (http-Fabriken) und PSR-18 (http-Clients) sind fast bereit, eine Überprüfung durchzuführen.
  • Der Zustand des Entwickler-Ökosystems im Jahr 2018 - JetBrains-Bericht basiert auf einer Umfrage unter 6.000 Entwicklern.


Die Werkzeuge


  • z7zmey / php-parser v0.5.0 - Der auf Go implementierte PHP-Parser wird weiterentwickelt.
  • nesk / rialto - Ein Tool zum Verwalten von Node.js-Ressourcen über PHP. Es kann verwendet werden, um die Interaktion mit Node.js-Bibliotheken zu implementieren.
  • nesk / puphpeteer - Ein Beispiel für die Verwendung von nesk / rialto, einem PHP-Wrapper für die Arbeit mit GoogleChrome / pupeteer (Headless Chrome).
  • mlambley / swagception - Generiert Abnahmetests für Codeception basierend auf Ihrer Swagger 2.0-Spezifikation (Open API 2.0).
  • RobThree / TwoFactorAuth - Zwei-Faktor-Authentifizierung basierend auf einmaligen TOTP- Passwörtern oder QR-Codes.
  • OwenMelbz / forget-db - Ein einfaches Tool zum Anonymisieren (z. B. GDPR) von Daten in der Datenbank.
  • gordalina / cachetool - CLI-Tool zur Verwaltung von apc und opcache.
  • digitaldonkey / ethereum-php - Typisierte Schnittstelle für die Ethereum JSON-RPC-API.
  • SocialConnect / auth - Bibliothek zur Authentifizierung über soziale Netzwerke . Dienste (OAuth1 / 2, OpenID, OpenIDConnect). Über 25 Anbieter sofort einsatzbereit.
  • xobotyi / rsync - Ein praktischer Wrapper über rsync für PHP 7.1+. Eingereicht von hobotyi .
  • krakjoe / idbg - Ein in PHP implementierter Debugger für PHP erfordert jedoch bisher die Erweiterung krakjoe / idbg . Post mit einer Beschreibung.
