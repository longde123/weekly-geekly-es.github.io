4. Periksa Titik Memulai R80.20. Instalasi dan inisialisasi



Selamat datang di pelajaran ke-4. Hari ini, kami akhirnya "merasakan" Titik Periksa. Secara virtual virtual. Selama pelajaran, kami akan melakukan tindakan berikut:

  1. Buat mesin virtual;
  2. Instal server manajemen (SMS) dan gateway keamanan (SG);
  3. Mari berkenalan dengan proses menandai disk;
  4. Mari inisialisasi SMS dan SG;
  5. Kita belajar apa itu SIC;
  6. Dapatkan akses ke Portal Gaia.

Selain itu, di awal pelajaran kita akan melihat bagaimana proses menginstal Gaia pada perangkat fisik Titik Periksa, mis. pada alat.
