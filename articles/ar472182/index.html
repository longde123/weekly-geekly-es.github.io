الإعلان عن .NET Core 3.1 Preview 1

نعلن عن .NET Core 3.1 Preview 1 . سيكون .NET Core 3.1 إصدارًا صغيرًا يركز على التحسينات الرئيسية في Blazor و Windows لسطح المكتب ، وهما الإضافتان الكبيرتان في .NET Core 3.0 . سيكون إصدار دعم طويل المدى (LTS) مع تاريخ شحن نهائي متوقع في ديسمبر 2019.

يمكنك تنزيل .NET Core 3.1 Preview 1 على Windows و macOS و Linux.

  • .NET Core 3.0 SDK ووقت التشغيل
  • صور عامل الميناء

ASP.NET Core و EF Core يصدران أيضًا تحديثات اليوم.

مرئي Studio 16.4 معاينة 2 ثم يتم أيضاً إصدار اليوم. إنه تحديث موصى به لاستخدام .NET Core 3.1 Preview 1. يتضمن Visual Studio 16.4 .NET Core 3.1 ، لذا فإن تحديث Visual Studio فقط يمنحك كلا الإصدارين.

تفاصيل:

  • ملاحظات الإصدار .NET Core 3.1

مشكلة معروفة: قد يقوم برنامج التثبيت Visual Studio 16.4 بإلغاء تثبيت .NET Framework 3.0 Runtime عندما يقوم بتثبيت .NET Core 3.1. نوصيك بإعادة تثبيت .NET Framework 3.0 SDK في هذه الحالة أو إصلاحه.



تحسينات


هناك العديد من التحسينات المستهدفة المخططة لـ .NET Core 3.1. التحسينات التالية ليست متاحة بعد ، لكن من المتوقع في المعاينات اللاحقة.

في .NET Core 3.0 ، لا يقوم مثبت وقت تشغيل .NET Core (يتضمن WPF و Windows Forms) بتثبيت .NET Core Runtime (بما في ذلك CoreFX و CoreCLR) لك. سنقوم بتغيير ذلك ، بحيث يكون مثبت وقت تشغيل سطح المكتب مكتفًا ذاتيًا. تعمل حزمة وقت التشغيل والاستضافة ، لسيناريوهات الخادم ، بالفعل بهذه الطريقة.

كان C ++ / CLI مكونًا موعودًا لإصدار .NET Core 3.0. تم تسليمها ، بالنظر إلى أنها تبعية مطلوبة لـ WPF ، لكن تجربة المطور في Visual Studio و SDK كانت مفقودة. سنقدم هذه التجربة مع .NET Core 3.1 و Visual Studio 16.4. C ++ / CLI معتمد فقط على Windows.

يتضمن نظام التشغيل MacOS 10.15 كاتالينا متطلبات أمنية جديدة ، ويجب توثيق التطبيقات. سنلبي هذه المتطلبات الخاصة بـ .NET Core SDK و .NET Core 3.1 وجميع إصدارات .NET Core الأخرى المدعومة. إذا كنت تستخدم .NET Core لتقديم تطبيقات macOS ، فإننا نقدر العمل معك على متطلبات التوثيق.

إغلاق


الهدف الأساسي من .NET Core 3.1 هو تلميع الميزات والسيناريوهات التي قدمناها في .NET Core 3.0. سيكون .NET Core 3.1 إصدار دعم طويل المدى (LTS) ، معتمد لمدة 3 سنوات على الأقل.

الرجاء تثبيت واختبار .NET Core 3.1 Preview 1 وتعطينا ملاحظات. غير مدعوم أو موصى به للاستخدام في الإنتاج.

إذا فاتتك ذلك ، تحقق من إعلان .NET Core 3.0 من الشهر الماضي.
