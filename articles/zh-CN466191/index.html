美国网络中立之战的主要内容是事件的时间顺序和当前的事务状态

2017年底，美国联邦通信委员会（FCC）废除了净中立的法定要求。 该决定引起了媒体的热烈讨论，即使经过了两年的政治和IT公司，也没有放弃试图影响FCC推翻其决定的尝试。 我们谈论网络中立性的形成及其“下降”，并讨论他是否有机会回归。

网络中立的第一步


网络中立是一个概念，根据该概念，禁止电信公司区分流量与不同来源的流量。 例如，Internet提供商无权阻止对站点的访问并赋予任何类型的内容优先权。

FCC互联网空间中的网络中立性问题在零点中间问到。 当时正在组建大型企业集团，例如AOL时代华纳和Comcast NBC Universal 。 这些组织拥有大规模的计算基础架构，控制着很大一部分市场，并且，如果需要，可以限制用户访问竞争对手的服务（例如，电子邮件或聊天）。

FCC试图排除后一种情况。 2005年，委员会主席Kevin Martin为电信公司和Internet服务提供商制定了四项规则。 这些规则被认为可以支持健康的市场竞争。

  1. 获得自己选择的法律内容的权利；
  2. 自由运行应用程序的权利；
  3. 将设备自由连接到网络的权利；
  4. 网络提供商，应用程序和服务提供商之间竞争的权利。

新原则无非是建议，也没有立法。 期望某些提供程序选择忽略它们。 从字面上看，2005年，本地提供商Madison River Communications关闭了其与自己的应用程序竞争的Vonage VoIP服务的用户。 2007年，康卡斯特（Comcast）也发生了类似的故事。 运营商在其网络上使用BitTorrent协议减慢了客户端文件的交换速度 。 这种行为在公司客户之间造成了不便，并且在竞争市场中造成了“扭曲”，有利于网络所有者。

电信公司法院


由于对实施没有约束力的建议并未带来预期的结果，因此FCC决定采取其他方式，并于2009年采用了正式的网络中立规则 。 委员会在2010年批准了该文件，但未能生效。

互联网提供商立即反对这一新命令。 网络中立性限制了它们的功能，并有义务为所有服务的用户（甚至是竞争服务）的用户提供相同级别的服务。 美国最大的提供商Verizon 告上法庭，该法院表示FCC无权规范运营商的工作。 不久之后，另一位运营商加入了 Verizon-MetroPCS诉讼。 当时，按订户数量计，他在美国运营商中排名前5位。

审判进行了四年，最终Verizon胜出。 尽管FCC有权就该决定提出上诉，但出于某种原因，该组织的代表决定不继续战斗 。 但是很快他们被迫改变主意。

公众回应


在判决支持Verizon之后，美国法院实际上允许提供商优先考虑流量，并从公司的网络中“高档”条件中获取收益。 反过来，这可能会影响用户付费的各种服务（例如，流媒体平台）的成本。

主要媒体- 福布斯 ， CNN和Business Insider-开始报道消除网​​络中立性的威胁， 抗议者聚集在 FCC总部。 像Google，Twitter，Facebook和Netflix这样的IT公司也加入了公民的行列。 许多站点度过了Internet减速日 -故意降低了大型Internet资源的加载页面的速度，以引起人们对这一问题的关注。 甚至总统巴拉克·奥巴马（Barack Obama）也表达了对这一问题的看法。 他主张收紧供应商的要求。


/ CC BY-SA / 新闻自由

结果，FCC 制定了一套更新的网络中立规则，其法律依据更为严格。 他禁止提供者：

  • 阻止法律申请，服务和设备；
  • 放慢服务；
  • 提供收费的优先服务。

该规则于2015年获得批准 。 但是，互联网服务提供商和共和党成员（最初反对网络中立）不会屈服。

净中性寿命短


首先，提供商尝试与FCC 进行谈判，并做出某种妥协。 但是所有的尝试都是徒劳的。 然后，一群电信公司提起新诉讼 。 听证会持续了几个月，但网络中立性“ 幸存 ”。 该决定被称为 “网络中立最终胜利”。

实际上，一切都不同。 2016年，巴拉克·奥巴马（Barack Obama）的总统任期即将结束，共和党人唐纳德·特朗普（Donald Trump）成为新的国家元首。 因此，FCC的主席被任命为与共和党-Ajit Pai（Ajit Pai）观点相同的人 。 他以前曾投票反对网络中立，并坚持他的看法。 他上任后几乎立即宣布，委员会准备审查其关于网络中立性的决定。


/ PD / 白宫

尽管社区批评了FCC选择的新发展载体，但委员会仍准备了一份文件 ，废除了2015年规则。 事实证明，网络中立仅持续了两年。

今天的网络中立情况如何


取消网络中立性后，流量歧视问题几乎立即开始出现。 在FCC宣布决定的同一天，康卡斯特（Comcast）开始提供付费服务，以优先处理网站流量（公司门户网站至少缺少未提供此类服务的信息 ）。 同时，根据东北大学工程师进行的测试，所有移动运营商都开始放慢流媒体服务的Internet连接速度。

美国众议院已经关注了这个问题。 今年4月，政客批准了一项法案 ， 该法案应从2015年起恢复网络中立性规则。 他的进一步命运掌握在参议院手中，这仍然是硬道理。 但是，该法案很可能会被拒绝。 参议院共和党多数派领袖米奇•麦康奈尔（Mitch McConnell）已经反对他-他称该文件“不可行”。

但是仍然有希望，政界人士将在公众压力下改变主意。 例如，大型音乐人- 汤姆·莫雷洛 （ Tom Morello） ， 迈克尔 ·施蒂佩 （ Michael Stipe）和塔利布 ·奎利 （ Talib Queli） 向参议院讲话，要求在联邦一级退回并巩固网络中立性规则。 地方诉求已经出版，例如《 芝加哥论坛报》 。

根据以前的经验，有理由相信，如果共和党在即将举行的选举中失败，那么FCC将再次改变其决定，而民主党领导人将取代Ajit Pai。
