Rilis eksperimental Blazor 0.9.0 sekarang tersedia

Blazor 0.9.0 sekarang tersedia! Rilis ini memperbarui Blazor dengan peningkatan Komponen Razor di .NET Core 3.0 Preview 3.


Peningkatan Komponen Pisau Cukur Baru sekarang tersedia untuk aplikasi Blazor:


  • Penanganan acara yang lebih baik
  • Formulir & validasi


Lihat pengumuman ASP.NET Core 3.0 Preview 3 untuk perincian tentang peningkatan ini. Lihat juga catatan rilis Blazor 0.9.0 untuk detail tambahan.


Catatan: Templat Blazor belum diperbarui untuk menggunakan ekstensi file .razor baru untuk Komponen Razor dalam rilis ini. Pembaruan ini akan dilakukan dalam rilis mendatang.


Dapatkan Blazor 0.9.0


Untuk memulai dengan Blazor 0.9.0 instal yang berikut ini:


  1. .NET Core 3.0 Pratinjau 3 SDK (3.0.100-preview3-010431)
  2. Visual Studio 2019 (Pratinjau 4 atau lebih baru) dengan ASP.NET dan beban kerja pengembangan web dipilih.
  3. Perpanjangan Blazor terbaru dari Visual Studio Marketplace.

  4. Templat Blazor pada baris perintah: 


    dotnet new -i Microsoft.AspNetCore.Blazor.Templates::0.9.0-preview3-19154-02

Anda dapat menemukan petunjuk memulai, dokumen, dan tutorial untuk Blazor di blazor.net .


Tingkatkan ke Blazor 0.9.0


Untuk meningkatkan aplikasi Blazor yang ada ke Blazor 0.9.0, pertama pastikan Anda telah menginstal prasyarat yang tercantum di atas.


Untuk memutakhirkan proyek Blazor 0.8.0 ke 0.9.0:


  • Perbarui paket Blazor dan referensi alat .NET CLI ke 0.9.0-preview3-19154-02.
  • Perbarui Microsoft.AspNetCore yang tersisa. * Paket ke 3.0.0-preview3-19153-02.
  • Hapus penggunaan JSRuntime.Current dan gunakan injeksi ketergantungan untuk mendapatkan instance IJSRuntime saat ini dan meneruskannya ke tempat yang diperlukan.
