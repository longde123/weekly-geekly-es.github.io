Popcorn Waktu berlalu, tetapi berjanji untuk kembali. Oh tidak, sudah kembali



Pengguna program untuk menonton film dan acara TV Popcorn Time memperhatikan bahwa program ini telah tidak beroperasi selama beberapa minggu terakhir. Waktu Popcorn dan garpu program yang paling populer juga tidak berfungsi. Ini karena beberapa pengembang utama dari Kanada mengalami tekanan hukum dari pemegang hak cipta. MPAA telah berhasil menyatakan bahwa ini adalah salah satu kemenangan utama asosiasi atas bajak laut dalam seluruh sejarah keberadaannya. Namun demikian, itu sudah cukup untuk memperbaiki beberapa poin - dan garpu akan kembali berfungsi secara normal. Dan pekerjaan semacam itu sudah berlangsung, dan bukan oleh pengembang pihak ketiga, tetapi oleh salah satu pembuat perangkat lunak.

Garpu utama, terikat ke domain PopcornTime.io, berhenti bekerja pada akhir Oktober karena penutupan server. Setelah beberapa waktu, perwakilan MPAA mengumumkan niat mereka untuk menuntut para pengembang di Kanada. Ini berhasil, dan peserta proyek utama menolak untuk bekerja. Tetapi kemudian beberapa pengguna PopcornTimes memutuskan untuk mengembalikan layanan, dan mereka melakukan semuanya, sekarang salinan klien yang berfungsi masuk ke Reddit .

Versi tetap berfungsi dengan API baru, sehingga film dan acara ditampilkan lagi. Alih-alih menggunakan YTS sebagai penyedia film, TorrentsAPI sekarang digunakan. Dan untuk pertunjukan itu, diputuskan untuk mengembangkan API khusus.

Sekarang pekerjaan untuk mengembalikan layanan / program masih berlangsung. Khususnya, pengguna VPN.ht, layanan VPN yang terkait dengan Waktu Popcorn, diberitahu tentang pembaruan yang akan segera terjadi. "VPN.ht Movies API FIX diinstal," pesan itu mengatakan, menyarankan bahwa klien PopcornTime juga dimulai kembali. Tampaknya penyedia VPN telah membuat beberapa perubahan yang memungkinkan pengguna layanan untuk mengakses situs dan klien Popcorn Time.



Sekarang semuanya berfungsi seolah-olah tidak ada yang terjadi. Benar, klien lama tidak berfungsi tanpa menginstal pembaruan, sehingga sebagian besar audiens lama tidak dapat bekerja dengan program ini. Pengembang Popcorn Time Wally, yang juga merupakan pendiri VPN.ht, mengumumkan rilis segera dari klien yang berfungsi penuh.

Sebelum itu, pengembang ingin memastikan bahwa domain baru yang terlibat untuk perangkat lunak tersebut tidak menjadi subjek serangan MPAA baru. Dan ini masalah, karena gugatan itu masih relevan untuk Kanada. Wally tidak disebutkan di dalamnya, tetapi namanya disebutkan dalam klaim. Selain itu, VPN.ht disebutkan dalam klaim dan gugatan.

Wally percaya bahwa alih-alih melawan Popcorn Time, MPAA harus memperhatikan konsep dan teknologinya sendiri.

"Popularitas Popcorn Time harus menjadi contoh bagi MPAA bagaimana membangun platform streaming yang terbuka untuk dunia," kata Wally.

Namun, tampaknya para pengembang Popcorn Time dan orang-orang yang dekat dengan klien dan layanan tidak dapat dihentikan - teknologinya berkembang, orang-orang baru dimasukkan dalam pengembangan.

Persediaan popcorn (ya, renyah seperti itu), ceritanya masih jauh dari selesai.
