发生了！ 具有CRI 90+的廉价LED灯

光线的显色指数会影响我们看物体的方式。 该指数越高，颜色越自然，阴影越明显。 日光灯和白炽灯的显色指数为100，廉价的LED灯为70，标准灯为80。直到今天，显色指数超过90的LED灯只能在宜家商店（Ledare系列）购买。

最终，第一家俄罗斯制造商发布了一系列具有高显色指数的灯泡。 先驱是Nanosvet。

我从制造商那里收到了26个型号的新灯，它们是在一周前才抵达俄罗斯的，并对其进行了测试。


在新系列中，有梨，G45球，蜡烛，GU10，GU5.3和GX53点。 所有灯都有两种色温：2700K暖白光和4000K中性白光。


我的测试结果：


所有灯的显色指数CRI（Ra）确实高于90。对于有暖光的灯，测得的显色指数为90-92，对于中性光的灯为92-94。

所有灯的测得光通量为声明值的94-113％（平均105％）。
所有灯的测量功率与声明的功率相差不超过1W。
所有灯的光波纹系数均小于1％。

所有灯泡均基于IC驱动器，因此当电源电压在较大范围内变化时，其亮度不会改变。 在包装上规定的电源电压范围内：170-250 V，所有灯都可以工作并且不会改变亮度，而即使在较低的电压下，许多灯也可以工作并且不会降低亮度（表的“最小电压”列中指示的电压是灯泡亮度降低5％）。

所有指示灯均应通过带有指示灯的开关正常工作（当开关关闭时，请勿闪烁，也不能微弱发光）。

与大多数俄罗斯品牌的灯具的测试结果相比，这些结果看起来令人惊讶：似乎Nanosvet几乎是唯一一个不会高估包装上的参数并且不会欺骗消费者的俄罗斯品牌。

这些灯是俄罗斯品牌的第一批灯，该灯在计算Lamptest.ru网站的最终评级的自动系统中的最高评级为5分。

使用两米积分球和Instrument Systems CAS 140 CT光谱仪测量光通量，色温和显色指数，使用Robiton PM-2进行功耗和功率因数测量，使用Uprtek MK350D进行脉动测量。 使用Lamptest-1设备，Stabil Instab 500稳定剂和Suntek TDGC2-0.5 LATR测量了光通量降低不超过标称值5％的最小工作电压。 在测量之前，为了稳定灯的参数，将它们加热了半小时。

我将给出具有温暖和中性光的灯的光谱。
第一个示例是“梨” 600 lm 2700K L160 LE-GLS-8 / E27 / 927，第二个示例是“球” G45 E14 680 lm 4000K L205 LE-P45-8 / E14 / 940。


顺便说一下，这只梨代替了60瓦的白炽灯，仅售149卢布。 对于具有CRI 90的灯，这非常酷。 一个灯泡的价格为219卢布，如果它像75瓦的白炽灯一样发光，也不错。

在表的类型字段中指示为60°的光点配备有一个透镜，可让您将光线集中在中心并减少令人眼花effect乱的效果（habr.com/ru/company/lamptest/blog/391349） 。


强大的GU10点具有细长的机身。


制造商对所有灯泡提供两年保修。

一小勺焦油。 Nanosvet计划将其所有灯都转移到CRI 90+，因此新灯的条形码与带有CRI 80的灯的条形码一致，并且它们的名称非常相似。 例如，新系列中600 lm的“梨子”被称为L160 LE-GLS-8 / E27 / 927，而在旧的L160 LE-GLS-8 / E27 / 827中。 一位数字的差异。 因此，商店可能会混淆新旧系列的灯具，而不会意识到它们之间的差异。

我很高兴终于出现了具有高显色指数的廉价LED灯。 据我所知，具有相同功率的CRI 80和90的相同灯具的工厂价格现在仅相差5％，但俄罗斯的主要品牌都没有订购具有较高CRI的灯具，考虑到“人们不需要这个，主要是为了使其更便宜”。 最好至少要有一个小型的Nanosvet订购这种灯并将其带到俄罗斯。 现在最主要的是，有了这些灯的可靠性，其他所有参数都将变得与众不同。
