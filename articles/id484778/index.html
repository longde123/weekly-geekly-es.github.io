Intel Parallel Studio XE 2020: semua yang terbaik untuk pengembang



Pada Malam Tahun Baru, versi baru alat pengembang utama Intel , Intel Parallel Studio XE 2020, dirilis . Mari kita mengambil kesempatan ini dan melihat apa yang "studio" terkenal hidup sekarang dan ke arah mana itu berkembang.

Pertama-tama, kami mencatat bahwa versi baru Intel Studio tidak jatuh seperti salju di kepala pengembang setahun sekali - yang sebelumnya, 2019, menerima 5 pembaruan dalam proses keberadaannya. Sayangnya, kami tidak memiliki kesempatan untuk mengumumkan setiap pembaruan, jadi kami menunggu Acara Besar - dan sekarang telah tiba.

Sekarang tentang komposisi Intel Parallel Studio XE 2020. Tidak ada perubahan khusus. Tiga pilihan produk dan tiga kelas perangkat lunak masih ditawarkan: alat pengembangan, untuk menganalisis dan mengoptimalkan kinerja, baik pada komputer yang terpisah maupun dalam satu cluster. Komposisi lengkap dari semua paket ditampilkan dalam tabel - tidak mungkin Anda akan menemukan kata-kata asing di dalamnya.



Dukungan untuk bahasa dan platform:

  1. C ++ penuh 2017 dan parsial C ++ 20;
  2. penuh Fortran 2008 dan diperpanjang Fortran 2018;
  3. Python 3.7
  4. OpenMP 4.5 lengkap dan konsep awal OpenMP 5.0;
  5. Sistem operasi baru adalah Amazon Linux 2.

Tentu saja, IPS XE 2020 mencakup versi terbaru dari semua komponennya, serta banyak cara untuk mendapatkannya untuk sistem * nix (conda *, PIP *, Docker Hub *, APT GET, YUM, NuGet) dan mendukung integrasi dengan Visual Studio * 2019 untuk Windows

Apa yang diperbaiki dalam versi komponen terbaru? Berikut ini tidak selalu berlaku untuk pembaruan terbaru, melainkan daftar apa yang telah dilakukan baru-baru ini.

  • peningkatan kinerja pada prosesor Intel Xeon dan Intel Core terbaru menggunakan instruksi Intel Advanced Vector Extensions 512 (Intel AVX-512);
  • percepatan inferensi AI dengan menggunakan versi baru dari kompiler, perpustakaan dan alat analisis;
  • lanjutan "kasar" dan profil yang tepat dengan dukungan untuk berbagai platform di Intel VTune Profiler;
  • Memprediksi penggunaan cache untuk vektorisasi dengan analisis batas untuk L1, L2, L3 dan DRAM di Intel Advisor;
  • dukungan untuk beberapa basis data Intel Cluster Checker, serta kemampuan untuk mengetahui konfigurasi dan fitur menggunakan memori persisten Intel Optane DC;
  • peningkatan fleksibilitas kontrol aliran di Intel TBB
  • meningkatkan fungsionalitas klasifikasi dan algoritma regresi (berbagai jenis Boosting) Perpustakaan Akselerasi Intel Data Analytics (Intel DAAL)
  • Dukungan untuk cloud HPC menggunakan AWS Parallel Cluster dan AWS Elastic Fabric Adapter untuk pertukaran data kinerja tinggi antara aplikasi MPI menggunakan Intel MPI Library.

Sekarang mari kita bicara tentang dukungan teknis untuk Intel Parallel Studio XE 2020. Sebelum membeli lisensi apa pun, Dukungan Prioritas diberikan setidaknya selama satu tahun sejak tanggal pembelian; jangka waktunya dapat diperpanjang dengan diskon (opsi untuk mendapatkan dukungan selama beberapa tahun juga tersedia). Dukungan mencakup akses gratis ke pembaruan produk, serta akses ke dan dukungan versi sebelumnya.

Selain itu, pelanggan menerima manfaat berikut:

  • Kemampuan untuk menguji proyek pelanggan pada setumpuk penuh solusi perangkat keras dan perangkat lunak Intel, termasuk prosesor server, kartu grafis diskrit dan akselerator komputasi generasi saat ini;
  • Memastikan kerahasiaan saat berinteraksi dengan insinyur Intel untuk menyelesaikan masalah teknis (dukungan berbahasa Rusia);
  • Bantuan dalam memecahkan masalah teknis - mengadaptasi solusi untuk konfigurasi Intel;
  • Akses gratis ke semua pembaruan produk dan versi sebelumnya;
  • Pendidikan dan pelatihan praktis untuk tenaga teknis, termasuk mengoptimalkan kinerja perangkat lunak pelanggan, analisis kode, analisis efisiensi beban sistem. Praktek menggunakan aplikasi yang dioptimalkan untuk AI, ML, TEC, dll.

Uji coba 30 hari tersedia untuk diunduh di situs web Intel . Setelah periode ini, Anda dapat membeli lisensi berbayar yang memberi Anda dukungan produk satu tahun. Siswa dan guru dapat memperoleh Intel Parallel Studio XE 2020 berfitur lengkap secara gratis.
