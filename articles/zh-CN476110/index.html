IT为什么需要个人品牌

“品牌”一词通常与物质利益相关。 人力资源品牌使您可以更有效地进行招聘，而个人品牌可以使您进行招聘。 个人品牌具有一个非显而易见的优势，它与个人品牌的质量/信息/策略没有任何关系。 不管您是认真的完美主义者还是炒作爱好者，都可以成为最佳的程序员。



在katom之下，IT专家的日常神经生理学和传统上常识性的错误。

在过去的20年中，我大部分时间都在编写代码，帮助其他人编写代码，有时从舞台上告诉我如何编写代码，甚至组织有关如何编写代码的会议 。

您可能已经注意到，在过去的几年中，“ devrel”，“ personal brand”，“ HR-brand”等词已变得很流行。 RIT ++音乐节的整个部分专门讨论devrel的工作以及如何与开发人员进行交流。 本文是在我演讲后写的，如果您喜欢听而不是看，则有一个视频 。

开发人员有什么特别之处，为什么我们需要以特殊方式与我们沟通？ 为什么我们没有单独的经理关系，建造者关系，电工关系，但是开发者关系存在吗？

当他们谈论个人品牌时，关于开发人员是否需要它的争论很多。 有必要花时间，去某个地方，说话-而不是花所有的时间。

我的爱好是神经生理学，作为一名程序员，我很难运用抽象的社会概念来进行操作，例如“灵感”，“认知”，“自我实现”。 对我来说，谈论正确的颞顶结点要容易得多。 因此，首先，我将分享有关我们意识通常如何运作的现代假设。 然后，我将尝试说明程序员的特殊之处，并解释为什么我们确实需要一个rel废的人和一个个人品牌，但是，例如，画家不需要它-他们很好。

意识假设


在过去的100年中，科学家一直没有从事人类意识的研究。 这是一个禁忌话题，因为大脑真的很复杂。 他如何认识我们的意识通常是无法理解的。 也许有一种灵魂和量子机制，也许这是高等思想者的接受者。

但是几十年前，我们先进行了MRI，然后进行了fMRI，然后这个过程开始了。 探究一个活着的人的工作大脑成为可能，以便至少了解有关他的设备的一些知识。

全球工作空间理论


1988年，提出了意识工作的第一个也是最简单的假设之一，这就是所谓的全局工作空间理论。 实际上，她什么也没解释，只是“指着向导”。 问题是，此向导是如何完成此技巧的，假设回答是该向导完成了此技巧，仅此而已。 焦点的秘密仍未公开。

简而言之，全局工作空间理论观察到大脑中有许多不同的事物是分开工作的，我们的意识是完整的。 大脑中必须存在某种机制来同步这些单独的工作部件。 从全局工作空间理论的角度来看，这种同步机制是我们的意识。

为什么这样，为什么计算机或下水道系统没有意识，那里还有许多独立运行但同步的机制，全局工作区理论并不能说明。

综合信息论


为了阐明先前的假设，2004年出现了集成信息理论。 她还用手指指着向导，但更准确一点：“向导坐在那里。” 集成信息理论的作者说，大脑不仅有许多不同的独立部分协同工作，而且所有信息都是集成在一起的 ：大脑存储一切并进行关联工作。 当我看着某人时，我与成千上万的面孔和行为相关联。 当一个复杂的系统以这种方式处理关联信息时，就会在其中产生所谓的意识。

就像上一次一样，该假设使向导指向了一个手指，但是并没有告诉向导如何完成技巧，为什么我们意识到自己。

注意图式理论


不到十年前的2009年，另一个假设出现了。 它的作者是一位非常有趣的神经生理学家：一个成功的儿童作家，一个成功的科幻小说作家，第一次接受教育的心理学家，第二次接受教育的神经生理学家以及作曲家。 他为另外两个假设做出了巨大贡献，此后科学界不喜欢他，现在他有些不高兴了。 迈克尔·格拉齐亚诺（Michael Graziano）是普林斯顿大学神经生理学实验室的负责人，在那里他对我们的意识如何运作提出了一个奇怪的假设。 与其他十几种假设不同，她没有指责巫师，而是试图解释魔术。

坦率地说，作为一名开发人员，他的假设是否正确对我来说根本不重要-我们的意识是被大脑吸引的，还是存在不朽灵魂的地方，或者大脑仅仅是更高层次的宇宙辐射的接收者。 我对实际上可以使用的任何解释都感兴趣，它为我提供了有关如何培训开发人员，如何聘请开发人员，在哪里寻找开发人员以及为什么开发人员需要个人品牌的提示。

注意图式理论简介：

  • 大脑根据人体传感器的数据建立人体模式。
  • 更改主体模式可控制主体的位置。
  • 注意力模式是大脑根据注意力数据建立的。
  • 更改注意力模式会切换注意力。
  • 大脑为其他人建立身体和注意力模式。

我将更详细地解释所有这些含义。 该假设始于观察到我们的大脑建立了多个模型 ，其中包括我们的身体模型（身体模式）。 大脑不断从肌肉和骨骼的传感器接收信息，并且每时每刻都有准确的人体模型。

该模型会不断更新，并且可以双向运行。 当我想移动手臂时，我不会对单个肌肉发出命令。 而是触发了一种奇怪的机制：我有意识地改变了手在大脑中的位置的模型，并且一个没有转化为意识的独立机械跟踪了变化并开始向肌肉发送命令，收集有关手的实际移动位置并调整其位置的信息以便使手精确地移动到我要移动模型的位置。

我们的程序以完全相同的方式工作：内部程序更改对象（例如，窗口）的状态，操作系统将看到此状态并更改绘制窗口的位置。 同样，在改变我的手在大脑中的模型之后，我的手会稍作延迟。

注意图式理论的创造者已经研究了很多年的大脑模型，包括身体模型和颜色模型。 例如，大脑以大大简化的模型的形式表示白色，因为实际上白色具有所有波长，但是从模型的角度来看，我们将其识别为没有颜色。 这是错误的模型，但对大脑来说很方便。

大脑正在建立简化的物理模型。 我们的物理思想几乎就像卡通漫画一样，因为它使我们的大脑在周围空间的简单模型中工作很方便。 科学家建议我们的意识，我们的灵魂，我们的自由意志-这也是大脑中发生的某种物理过程的模型。

哲学家们争论着人类的意识是什么，他们以思想在树上蔓延，描绘出奇妙的画作，在水上和云端的城堡。 但是，如果您科学地进行此操作，结果并非并非所有事情都是如此令人费解。

例如，有一种双眼竞争的现象：如果一个人固定好，左眼显示一张照片，右眼显示另一张照片，那么对象一次只会知道其中一张照片，并且大约每2秒改变一次，跟随着“大脑“一种或另一种方式的疲劳”。 对于这种情况，可以给出意识的以下定义：意识（意识）是决定一个人在“看到”它时可以说出的两张图片中的唯一内容。
迈克尔·格拉齐亚诺（Michael Graziano）进行了实验，并提出意识是我们大脑为注意力建立的模型。
大脑有一个生理上的注意过程。 这就是我们在众多活动神经集成中选择对我们而言优先级最高的集成的方式。 注意通常集中在一件事上。

在对我们的意识如何运作进行了大量研究之后，这位科学家建议我们的注意力模型与大脑为身体建立的模型完全相同。 同样， 该模型可以双向工作 。 当我们表现出自由意志时，例如，我选择将注意力从一个人转移到另一个人时，我改变了大脑的模型。 遵循该模型，神经集成体之间竞争的物理过程正在发生变化（不要问我-假设已经有十年了），然后大脑习惯了这样一个事实：如果注意力已经改变，则需要移动头部，将头部转到需要的位置。

对我们而言，最有趣的事情是使我们受益，并使我们能够将注意力与个人品牌和发展问题联系起来- 大脑不仅为我们所爱的人，而且为周围的人建立了身体和注意力的模型 。

人们被认为是社会人。 文明的一切成就之所以出现，是因为我们彼此交流。 同时，在为自己和他人建立模型的过程中，“母鸡和鸡蛋”的问题仍然存在：当婴儿出生并与他人见面时，他首先建立了注意力模型，然后注意到周围的生物以相同的方式表现，并开始建立简化的模型。他们注意的模型。 或者，孩子首先为他人建立注意力模型，然后才对自己使用相同的机制。

无论如何，实验和观察表明，我们不断地建立自己周围人的注意力模型，就像我们自己建立模型一样。

我们感兴趣的结论


如果我们带上干燥的残留物，那么对我们来说最重要的是： 大脑建立的注意力和身体模型不是当前的数据，而是历史数据 。 我大脑中的身体模型是我对我的身体在整个生命中通常如何运动的全部记忆。 这可以很容易地通过以下事实得到验证，例如，一个人的手失去了自己的身体模型，将会失去肢体的“幻像”很多年。

同样，注意力模型不只是“当前注意力”。 这些都是历史信息，注意力通常是如何移动的，当您从一种想法转换为另一种想法，从一种感觉转换为另一种感觉时会发生什么。

第二件事很重要，就是大脑会根据 情况行动 。 在什么情况下训练模型，在这种情况下应用模型。 一个程序员读了两年书，然后坐下来写代码，一方面，他们什么都知道。 他关注的模型包含大量印刷信息，但是为了拿起键盘并在上面键入工作代码，还没有经过训练的模型。

一个已经在体育馆里从事武术10年，走到街上并掉入gopnik的人，也许不能仅仅因为他已经习惯在体育馆里打架而对付他们。 在大厅里，他的大脑有不同的行为方式；在大街上，他的处境完全不同。

当一个人训练在镜子前讲话然后进入舞台时，他发现自己处在完全不同的社交环境中，固定的行为方式无济于事。

培训和身体模型 学习非常缓慢。 改变您的注意力模式并掌握新技能-公开演讲，触摸打字，编程-花费的时间不少于改变身体模型和学习（例如弹吉他）的时间。 这些是数月，数年-长期增强作用不只是被称为“长期”。

发展挑战


让我们看看这些知识将如何帮助编程和建立个人品牌。

顺便说一句，编程通常有什么问题？ 编程也许一切都很好，但是不需要个人品牌吗？


不幸的是，编程中存在很多问题。

我认为，主要问题是缺乏基础教育 。 我们不仅仅是超级社交动物，我们喜欢一个模型一个地建立，另一个模型地建立在我们多年来学到的基础之上，就像上图所示。 一个幼儿园的孩子学习计数，然后在学校学习数学，然后在大学里成为一名电工。 以下更复杂的知识是在先前知识的基础上构建的。

编程没有这样的基础。
编程是一种幻影工程，我们根据发明的物理定律在世界上建立不存在的工程结构。 没有依据可以拉动所有这些。
从程序员中选拔程序员似乎是合乎逻辑的：总的来说，结构是工程学，从第二年开始，您可以在某个地方教一名工程师成为程序员。 在实践中，事实证明与从头开始学习相同。

然后他们想到：在编程中，存在加，减，除，乘，有时甚至是根-让我们向程序员教数学家-这是现成的基础。 但没有-从头开始。 数学知识并不能真正帮助在已发明的物理定律世界中建立幻像的桥梁。

我们没有可以预测地毕业于程序员的高等教育机构。 没有基础教育，因此不可能以可预测的方式雇用程序员，因为可以以可预测的方式雇用电工。 没有这样的文凭可以保证其所有者是可以在React，Vue和Python / Elixir / Lua中编写后端的程序员。 我们是自学成才的。

而且，我们在批量生产方面的专业领域只有20年的历史，因此，我们仍然对如何正确开发软件尚不了解 。 中世纪的炼金术士级别的开发人员试图混合使用不同的技术，有时它会发出嘶嘶声，有时会爆炸，有时Stack Overflow示例会将整个基础结构放在一起，而对于nginx为何崩溃仍然是完全不可理解的。

不可能按预期地在固定的堆栈上训练程序员，然后说：“现在我们从这所大学招募Python + React开发人员。 我们在面试中询问期末考试的结果，我们可以确定一个人可以编写软件并解决业务问题。 一切都很好，请给20个人！”

不幸的是，没有这样的大学，您必须忍受和雇用自学成才的人，他们也处于搜索引擎和“技术栈”精心创建的信息泡沫中。

我在与Python开发相关的会议上演讲，并且有一个前端，我越来越了解Python和JavaScript，而越来越少了解Lua或Erlang。 我在信息泡沫中。 然后拖延到来，并说在这种状态下将工作推迟到明天，再将其委托给自己，将是很棒的。 “我在未来”是一个非常方便的角色；他不能拒绝。

解决这些问题的方法有很多，其中之一是个人品牌。

个人品牌


个人品牌是其他开发者了解您的时候。 对于性格内向的人和社交恐惧症，这不是很舒服。 例如，他们在会议上的一个私人会议上向您打招呼，而您几乎无法认出它们，因为大脑被代码阻塞了。 例如，我记得22年前我是用Visual Basic编写代码的，但是我不记得一半熟悉的会议参与者的名字是什么。

但是既然这样不舒服，为什么还要发展某种个人品牌呢？ 对于其他程序员了解他的事情，对程序员有用的是什么？

社会义务


要回答，请记住，我们一直在为自己和他人建立注意力模型。 这是创建我们的个性并模仿他人的个性的基本机制，我们可以使用它。

人们之所以被认为是社会存在，不是因为教科书这么说，而是因为我们大脑的基本机制吸引了我们的个性，意识和灵魂，被用来创建意识，注意力和其他人的个性的简化模型。 这是基本机制。

由于有了这种基本机制， 我们的注意力模型已经得到训练，即社会很重要 。 例如，当我们想要或不想做某事时，我们不想工作，但是我们想看YouTube，发生的事情取决于我们学到的模型。 例如，如果您承担社会义务，请在莫斯科Python Conf ++ 上演讲，那么行为模型将处于一种新的社会环境中。 该模型认为， 在履行社会义务的情况下 ， 成功或失败与我们生活中最重要的事情-社会性有关 。

这种“重要的-不重要的”在思想中并不总是被翻译出来。 您可以有意识地认为自己是一个内向的人，并认为不需要其他人并且他们的意见不重要。 但是，我们用来吸引人格的机制对我们来说并不重要，这是基本机制。

如果在会议上我们听到：“ React-Vue-React-Vue-React ...”，那么我们开始猜测已经在我们的项目中使用了多年的Angular可能不是最好的技术。 在生命的几十年中，大脑学会了建模其他人的注意力，并自动将此类信息标记为“重要”（至少有一些假设表明了这一点。但是我们不知道它“真的”是因为大脑很复杂）。
别人的注意力，他们所说的总是对我们来说比我们阅读的更为重要。
交流，与其他人互动时，当我们做出承诺时，默认情况下对我们很重要，因为这是我们大脑的基本机制。

它可以很好地应对拖延症 。 如果几年来您都无法开始学习英语（或学习Vue，Elixir，Erlang，Tarantool），您需要做的只是公开地说 ：“我将在六个月的主题会议上发言”或“我将在哈布雷杂志上发表文章”。 一切-进一步成为关注的模型，结果表明它非常重要，并且不再希望阅读VK，而是希望为报告做准备。 有一种做某事的愿望，因为对羞耻的恐惧使我们前进。

不破坏这种机制非常重要。 有些人许诺某些​​东西并合并，许诺和合并，许诺和合并。 很难打破从童年时代起就建立的机制，但是如果您许诺数百次向他人许诺，并且不遵守您的诺言，那么模型将被重建。 大脑非常灵活，我们的大脑具有适应能力-如果您信守诺言，您将成为“字母M的超级英雄”，并且失去了通过手指点击即可获得+50的渴望的机会。 顺便说一句，相反的假设是建立在这个假设之上的，根据该假设，最好不要向任何人承诺，因为在承诺之后，我们可以认为“我们已经做了一切”。 选择哪种假设取决于您，但是我更喜欢第一个假设，因为它提供了简单的工具。 第二个只悲观地说：“这无济于事。”

社会交往


个人品牌的第二个优点是社交交流。 我们的注意力模式已经被训练为与他人的交流很重要 。 交流是指我们建立另一个人的注意力模型并试图了解他从我们那里想要什么，他的目光指向何处，他现在正在经历什么情绪。 沟通的整体图景的形成是由我们大脑最复杂和最基本的机制提供的。
