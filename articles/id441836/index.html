One Cloud Story: Huawei + 3data = Cloud

Tiga tahun lalu, kami mengadopsi strategi baru, yang menurutnya tempat terdepan dalam rencana pengembangan perusahaan diberikan kepada teknologi cloud. Pada 2018, Huawei, bersama dengan 3data, meluncurkan layanan cloud di Rusia. Negara kami adalah yang pertama di luar Cina, di mana awan dikerahkan di bawah mereknya sendiri.


3data HP-26 data center adalah salah satu tempat di mana Huawei cloud hidup

Cloud Rusia jauh dari satu-satunya yang dibangun oleh Huawei menggunakan perangkat keras dan perangkat lunaknya sendiri. Di Cina, Huawei mengembangkan tiga solusi cloud yang sangat besar: cloud publik Huawei sendiri, cloud publik dari salah satu operator telekomunikasi terbesar China Mobile, dan cloud untuk lembaga pemerintah. Selain itu, di Cina, kami menawarkan pelanggan kami berbagai layanan yang sebanding dengan Big Western Three - Amazon, Microsoft dan Google. Banyak dari layanan ini sudah beroperasi di Jerman, Perancis, Spanyol dan di sejumlah negara, tetapi di bawah merek mitra lokal.

Pasar Rusia untuk Huawei selalu istimewa, dengan itu memulai kisah sukses kami di luar Cina. Tidak mengherankan bahwa awan asing pertama di bawah mereknya sendiri diluncurkan di sini. Itu harus dipertimbangkan sebagai bagian dari strategi untuk pengembangan solusi teknologi cloud yang terintegrasi ke dalam banyak produk perusahaan: untuk telekomunikasi, segmen perusahaan, dan bahkan untuk pengguna akhir.

Bagaimana cara memulai


Ketika muncul pertanyaan untuk mempertimbangkan kemungkinan peluncuran cloud publik di Rusia, hal pertama yang kami lakukan adalah melakukan analisis mendalam dan komprehensif tentang pasar layanan cloud Rusia pada 2016-2017. Hasil analisis memberi kami keyakinan bahwa pasar sangat dinamis dan dinamis, tetapi pada saat yang sama jauh dari kejenuhan. Banyak pertanyaan praktis muncul, khususnya, yang bersifat hukum dan komersial: lisensi apa yang diperlukan untuk menyediakan layanan seperti itu, cara terbaik untuk mendapatkannya, bagaimana memformalkan hubungan dengan pelanggan, dll.



Akibatnya, layanan cloud dikerahkan di situs mitra: 3data memberi kami pusat data sendiri dengan infrastruktur rekayasa dan saluran komunikasi, yang dalam karakteristiknya sesuai dengan Tier 3. Cloud itu sendiri sepenuhnya dibangun pada produk Huawei: semua komponen perangkat keras dan perangkat lunak cloud, yaitu, server , sistem penyimpanan data, peralatan komunikasi, serta perangkat lunak - semua produksinya sendiri.

Peralatan tersebut berlokasi di Moskow, dan layanan disediakan dengan nama merek Huawei 3data Cloud: mitra menyediakan lini pertama dukungan untuk cloud dan, sebagian, yang kedua. 3data juga merupakan garis depan bagi pelanggan yang menandatangani kontrak dengan 3data. Jumlah pusat data mereka sendiri mendekati dua lusin, perusahaan ini cukup mampu untuk proyek skala besar.

Perangkat lunak cloud kami didasarkan pada solusi Huawei FusionCloud, pada gilirannya, berdasarkan OpenStack. Virtualisasi KVM digunakan. Perlu dicatat bahwa Huawei adalah salah satu dari delapan sponsor platinum dari proyek OpenStack.



Kami menggunakan hypervisor Huawei FusionSphere berdasarkan KVM. Sangat penting untuk dicatat bahwa Huawei adalah salah satu perusahaan pertama yang menerima sertifikat FSTEC untuk hypervisor-nya.

Apa yang diluncurkan pertama kali


Pasar layanan cloud Rusia berbeda dari Cina atau Eropa. Volumenya jauh lebih kecil, dan bisnis Rusia masih tidak menerima teknologi cloud dan juga teknologi Barat. Juga, penelitian kami yang dilakukan pada tahun 2017 menunjukkan bahwa layanan cloud kelas PaaS tingkat lanjut tidak banyak diminati, meskipun ada tren kenaikan. Ada juga perbedaan positif (untuk penyedia): pasar masih jauh dari jenuh, dan ada sedikit kompetisi di Rusia. Yang paling sulit adalah memilih produk untuk tahap pertama penyebaran. Analisis pasar yang serius dilakukan dengan melibatkan kontraktor pihak ketiga, dan untuk tahap awal di Rusia, hanya layanan IaaS yang dipilih:

  • pusat data virtual (VPC)
  • server virtual (ECS)
  • penyimpanan blok (EVS)
  • Object Storage (OBS) yang mendukung protokol S3
  • Penskalaan Otomatis
  • load balancers (ELB)
  • VPN sebagai layanan
  • Firewall sebagai Layanan
  • kelompok keamanan
  • dedicated server (Bare Metal Service) dan beberapa lainnya.




Zona ketersediaan pertama secara resmi mulai beroperasi pada Maret 2018.

Cloud kami juga menawarkan layanan Pemantauan Infrastruktur Cloud gratis, Layanan Cloud Trace, dan layanan Pemberitahuan Pesan Sederhana yang sangat meningkatkan kemampuan keamanan dan administrasi infrastruktur cloud.

Cukup cepat, kami memiliki klien dengan permintaan untuk virtualisasi GPU. Khusus untuk mereka, layanan ini dengan cepat dimigrasikan dari cloud Cina. Pada akhir 2018, server dengan GPU Nvidia Tesla V100 tiba untuk klien ini. Pada kuartal kedua 2019, kami berencana untuk memperluas cluster server dengan GPU.

Layanan Direct Connect telah membuktikan dirinya sangat baik - koneksi optik gelap dari pusat data fisik klien dengan pusat data virtualnya di cloud Huawei. Dalam hal ini, klien menerima perutean tingkat IP dengan tingkat keamanan tinggi antara jaringan fisik dan cloud-nya.

Pada Oktober 2018, layanan kelas PaaS pertama diluncurkan dalam mode uji:

  • RDS, Layanan Database Relasional (Databases as a Service). Sekarang di cloud sudah tersedia produk seperti MySQL, PostgreSQL dan Microsoft SQL Server;
  • DCS, Layanan Cache Terdistribusi, yang mengimplementasikan sistem untuk menyimpan data dalam RAM. Anda dapat memilih layanan berdasarkan memcached atau berdasarkan redis).
  • DDS, Document Database Service: sistem manajemen basis data berbasis dokumen yang berbasis pada MongoDB.

Klien kecil biasanya puas dengan layanan kelas IaaS dasar, dan perusahaan menengah dan besar membutuhkan jangkauan layanan yang lebih luas, termasuk PaaS: manajemen kontainer, membangun gudang data, perantara pesan.

Kebutuhan pelanggan kami pada kenyataannya ternyata sedemikian rupa sehingga pada kuartal ketiga 2018, kami kehabisan kemampuan pusat data yang digunakan pada saat itu dan mulai mencari peluang untuk ekspansi lebih lanjut.

Peta jalan


Cloud Rusia Huawei menawarkan pelanggan lebih dari dua setengah lusin layanan, dan pada akhir kuartal pertama tahun 2019 jumlah mereka akan meningkat menjadi lebih dari 50. Kami merencanakan sekitar 30 layanan baru, termasuk yang berikut:

  • Cloud Container Engine - manajemen kontainer berbasis Kubernetes;
  • Layanan File Elastis - penyimpanan file bersama;
  • Layanan Nama Domain;
  • Layanan MapReduce - bekerja dengan data besar berdasarkan Apache Hadoop;
  • Data Lake Insight - pemrosesan dan analisis data besar berdasarkan Apache Spark;
  • Layanan Pesan Terdistribusi - antrian pesan berkinerja tinggi dari berbagai jenis, khususnya, Kafka, ActiveMQ, dll.

Juga, pada akhir kuartal pertama tahun 2019, jumlah zona akses akan meningkat menjadi tiga - pada bulan Maret direncanakan untuk meluncurkan tiga situs baru, di mana kami akan dengan tenang memindahkan pelanggan kami dari pusat data yang ada. Dua situs disewa di dua pusat data terbesar di Moskow, dan situs ketiga disediakan oleh mitra 3data kami. Karena sentralisasi khusus bisnis di negara kami, kami saat ini fokus pada mengambil posisi yang kuat di wilayah pusat Rusia, tetapi dalam beberapa tahun mendatang kami berencana untuk memperluas ke wilayah geografis lain di negara kami.



Atas dasar produk cloud, pelanggan akan ditawari solusi turnkey lengkap: solusi cadangan untuk cloud berdasarkan Commvault RBaaS, serta solusi untuk migrasi otomatis ke cloud Huawei dari lingkungan lain yang berbasis pada Hystax Acura sekarang sedang dipersiapkan untuk diluncurkan.

Mengembangkan proyek, pada tahun 2019 kami memberikan perhatian khusus pada saluran mitra. Di satu sisi, kami berencana untuk memperluas jaringan penjualan dan portofolio layanan integrasi bersama kami. Kedua, salah satu tujuan kami adalah meluncurkan ekosistem sedemikian rupa sehingga perusahaan teknologi Rusia dapat membuat solusi mereka sendiri berdasarkan produk cloud Huawei dan menawarkannya kepada pelanggan mereka. Area ketiga untuk kerja sama adalah mitra layanan yang akan membantu kami menyediakan beberapa layanan khusus kepada pelanggan kami, misalnya, meluncurkan dan mendukung Oracle DBMS di cloud kami.



Kecerdasan buatan


Seperti yang telah kami katakan, proyek Rusia harus dianggap sebagai bagian dari keseluruhan konteks. Strategi Huawei melibatkan pengembangan platform cloud global, dan di masa depan, layanan berbasis pembelajaran mesin dan teknologi kecerdasan buatan akan ditambahkan ke daftar layanan. Di Cina, sejumlah besar dari mereka tersedia untuk pelanggan sekarang, yaitu:

  • Pengenalan teks (OCR);
  • Membaca teks (Text-to-Speech);
  • Pengenalan ucapan
  • Pengenalan musik;
  • Sorot objek pada video dan banyak lagi.

Sesuai dengan strategi kami, Huawei berencana untuk menjadi pemain terkemuka di pasar layanan cloud untuk tugas kecerdasan buatan. Pada Oktober 2018, chip seri Ascend dioptimalkan untuk menyelesaikan tugas-tugas AI disajikan di Huawei Connect di Shanghai. Mereka dapat digunakan sebagai prosesor bantu untuk menyelesaikan tugas-tugas khusus di server Huawei baru. Dibuat menggunakan teknologi 12 nm, Ascend 310 sudah tersedia, sedangkan Ascend 910 yang lebih canggih akan dirilis pada kuartal kedua 2019. Chip yang dibuat dengan menggunakan teknologi 7 nm ini akan menjadi pesaing terdekat dengan Nvidia Tesla V100, dengan konsumsi daya yang sedikit lebih tinggi, ia akan memiliki kinerja 256 TFLOPS.

Tanggal pasti untuk kemunculan solusi server AI berbasis perangkat keras di Rusia masih belum diketahui, tetapi hal ini kira-kira akan terjadi pada paruh kedua tahun 2019. Diharapkan bahwa server dengan chip Ascend 910 akan menempati tempat penting di antara komponen perangkat keras cloud kami dan layanan yang sesuai akan bergantung pada mereka. komputasi awan.

Pesaing


Secara global, Huawei Cloud bersaing dengan Amazon Web Services, Google Cloud Platform, Microsoft Azure dan Alibaba Cloud, dan kami menetapkan tujuan kami untuk menjadi salah satu dari lima penyedia cloud terbesar di dunia. Kami menghormati pesaing kami, tetapi kami benar-benar yakin bahwa kami dapat melakukan tugas ini. Saat ini, Western Three menawarkan jangkauan layanan yang lebih luas daripada Huawei di Rusia, namun cloud Rusia kami memiliki donor yang kuat. Layanan Huawei di Cina telah beroperasi selama bertahun-tahun, menawarkan pengguna lebih dari seratus layanan beragam dan beberapa ratus solusi kemitraan yang dibangun di atasnya. Ini bernilai khusus, karena kami tidak harus membuat produk baru dari awal, proses migrasi mereka dari cloud induk sedang berlangsung. Selain itu, Amazon, Google dan Microsoft tidak memiliki pusat data Rusia, yang menyulitkan pelanggan yang memproses data pribadi dan secara signifikan membatasi peluang kemitraan.



Di antara para pesaing domestik utama, Mail.ru Cloud Solutions dan Yandex.Cloud dapat disebutkan, namun, Huawei memiliki kelebihan di sini: mencakup beberapa ribu karyawan R&D, berbagai peralatan produksi sendiri (termasuk server, peralatan jaringan, dan sistem penyimpanan) dan semua cloud Cina yang dikembangkan sama, dari mana mudah untuk mentransfer layanan baru di hadapan permintaan pelanggan. Pasar Rusia untuk layanan cloud benar-benar jauh dari kejenuhan, dan masih ada ruang yang cukup untuk semua orang, tetapi ini tidak akan berlanjut tanpa batas waktu. Munculnya pemain global lain harus mempercepat prosesnya.
