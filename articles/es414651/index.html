Ya hemos examinado este escáner en una revisión general, y ahora queremos hablar sobre el proceso de escaneo 3D en este dispositivo. Y el resultado que hemos obtenido.

PICASO Designer X PRO "Monster", conocido en publicidad para una impresora 3D, fue elegido como modelo para escanear. Por cierto, en Picaso X PRO, está impreso.



Etapa 1 Calibración de escáner 3D



Después de conectar todos los componentes y cables a la PC, iniciamos el programa de escaneo Einscan, que viene con el escáner (gratis).
Primero, ofreceremos calibrar el escáner.



El proceso es bastante simple. En cada paso, se nos pregunta qué hacer.



Todo es claro y obvio.



Unos pocos pasos consecutivos, y el escáner 3D está listo para usar.



Etapa 2 Proceso de escaneo 3D



Una vez completada la calibración, el software ofrece lo siguiente: repetir la calibración o continuar con el escaneo 3D.



Puede comenzar uno nuevo o continuar un proyecto antiguo.



Ingrese el nombre del proyecto y la ruta donde se guardarán los archivos.



A continuación, se nos ofrecerá la opción de escanear en modo captura de textura (es decir, en color) o escanear en modo monocromo (sin color)



Se propone además elegir el brillo de la iluminación. Esta es básicamente una de las dos configuraciones al escanear. Todo se simplifica al máximo, para el usuario: todo se hace mediante software o le indica dónde hacer clic a continuación.



Después de seleccionar, el número de ciclos de exploración. Cuanto más grande, mayor es el detalle y la calidad de la superficie. El tiempo de escaneo también llevará más tiempo.



Por cierto, esta es la última configuración. La calidad de escaneo, excepto este parámetro, no está establecida. Solo al guardar el modelo, puede seleccionar su tamaño y compresión.



Todo está listo para escanear: comencemos.



Así es como se ve este proceso desde el exterior.



La iluminación del modelo cambia y su relieve aparece en la pantalla.



El proceso no es rápido. Nos llevó unos 20 minutos. Esto es lo que vemos más cerca de completarse.

Etapa 3 Montaje y formación de un modelo 3D.





El modelo escaneado se puede editar. Elimine los artefactos innecesarios (ruido) que surgieron durante el escaneo.



Este proceso también es simple e intuitivo. En la pantalla siempre hay una pista de qué presionar.



Después de eso, puede postprocesar. Hay dos opciones para dejar, tal como están, inmediatamente después de escanear "sin relleno". O elija la opción cuando el software cierre las brechas en el escaneo, conecte los planos "colgantes".



Esto es lo que terminamos con:



En nuestra opinión, es lo suficientemente valioso para un escáner 3D automático económico. No cargamos el modelo, "pesa" más de 800 MB. Todo el proceso se realizó solo como un ejemplo; por lo tanto, el modelo no se optimizó.

Conclusiones



Escáner 3D Einscan SE tiene una calidad de escaneo bastante alta y un proceso completamente automático. Ideal para instituciones educativas debido a la simplicidad y claridad del proceso. Además, el escáner será un buen asistente en los negocios, donde no se requiere precisión en centésimas de milímetro (por ejemplo, hacer recuerdos o productos de un plan de arte).
La principal diferencia con los escáneres profesionales (por ejemplo, los escáneres RangeVision), además de la resolución significativamente más baja del escaneo 3D y la precisión de posicionamiento, es el software más simplificado, sin la capacidad de ajustar el proceso de escaneo para un objeto específico.

Otro inconveniente son los requisitos bastante altos para la computadora, la necesidad de una tarjeta de video de Nvidia.
