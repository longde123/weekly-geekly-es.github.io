Wanhao D9: impresora 3D asequible con un área de impresión grande


Wanghao no solo continúa actualizando las impresoras 3D existentes, sino que también repone la línea con nuevos dispositivos. Duplicator 9 no es una, sino tres impresoras, basadas en el mismo principio, pero que difieren en el tamaño del área de impresión. Antes del lanzamiento de D9, la compañía no tenía como objetivo un área de impresión grande, excepto las D5 , pero tiene una altura impresionante solamente, y esta impresora pertenece a una categoría de precio más alto.

Que es esto


Duplicator 9 se basa en el principio de portal clásico y se ensambla a partir de un perfil de aluminio estructural, que es tanto un marco como una guía para los rodillos. Extrusora - una, calefacción de mesa - en stock. El fabricante llama la atención sobre el hecho de que los soportes del motor y una serie de otros detalles críticos para la geometría de la impresora no se doblan desde una placa de acero, sino que se hacen mediante fresado.

Wanhao d9 / 500


  • Dimensiones del área de impresión, mm: 500 x 500 x 500.
  • Espesor de capa, desde, micras: 100.
  • Temperatura de impresión, hasta, ° C: 300.
  • Tabla de calentamiento: sí.
  • Velocidad de impresión, hasta, mm / s: 70.

El modelo anterior de la línea tiene un registro para el área de impresión del segmento de presupuesto. Una impresora para quienes necesitan imprimir con frecuencia piezas grandes enteras, pero no es práctico comprar una impresora 3D industrial de gran formato. El modelo más joven D9 / 300 ya tiene un área de impresión de 300 x 300 x 300 mm, más que el popular Wanhao i3 . El modelo medio D9 / 400 tiene un área de impresión de 400 x 400 x 400 mm.



En la foto de arriba, los sujetadores fresados ​​de los motores paso a paso y las tuercas de los tornillos del eje del eje Z.

Otras características


El Wanhao D9 está equipado con un controlador de pantalla táctil que admite la impresión interrumpida continua después de un corte de energía. La calibración automática de la mesa se implementa utilizando un sensor con una sonda retráctil, un análogo de BLTouch. Los carros de rodillos están equipados con una abrazadera para evitar el contragolpe.

¿Quién necesita Duplicator 9?


Las impresoras de este diseño no pueden presumir de una alta velocidad de impresión, por lo que trabajar con plástico con una contracción significativa será difícil debido a la falta de una cámara cerrada. Pero un área de trabajo grande le permitirá imprimir piezas grandes de PLA, SBS o PETG completamente sin pegar. Se necesitará un volumen de hasta 500 x 500 x 500 mm para el modelo anterior durante la creación de prototipos, para diseños de impresión, en la producción de equipos de cosplay.

Ventajas:

  • área de impresión grande;
  • manejo ergonómico;
  • Costo asequible.

Desventajas

  • falta de una vivienda cerrada;
  • baja velocidad
  • Una extrusora.
