Microsoft学习简介

Microsoft Learn网站是了解产品和服务的最简单方法。 在这里，您可以根据实际任务进行交互式培训。 它提供了大量免费材料，这些材料以23种语言进行了本地化，总学习时间超过80小时。 培训涵盖了我们的许多产品，并且会不断用新资源进行更新。 削减更多细节！



有关更多信息，请参见Microsoft学习启动博客条目。

学习网站提供：

  • 提供23种语言的免费培训材料（总学习时间-超过80小时）。
  • 使用学习沙箱免费临时访问Azure以创建和管理资源。 无需信用卡即可进行身份验证。
  • 使用演练和交互式代码练习的实践培训。
  • 跟踪培训进度，测试知识，测试脚本以及赚取积分，克服水平，获得证书和奖励的能力。

搜索和过滤内容


在评论页面上，可以方便地搜索与该主题相关的教育材料。 此功能提供全文搜索和结果过滤，还允许您使用各种搜索标签，例如：

  • 专业化：管理员，开发人员，解决方案架构师，业务分析师，业务用户。
  • 级别：初学者或中级。
  • 产品：虚拟机，Azure存储，Power Apps，Power BI等。



模块和培训计划


所有培训材料均以5-7课组成的模块形式组织。 学习一个模块（每个模块专用于特定主题）需要30-60分钟。 如您在插图中所见，为每个模块提供了注释和培训任务列表，并指出了完成该模块所需的时间以及所获得的经验值（XP）。



学习路径是涵盖广泛主题的模块的有序列表，如Azure示例中用于开发有效解决方案的学习路径所示。



使用学习沙箱免费临时访问Azure


对于初学者而言，了解云平台可能很困难，因为在检查免费帐户时，您需要提供信用卡信息和手机号码。 此外，有时注册用户会警惕将其组织的Azure工作订阅用于教育目的。 为了消除这种干扰，我们创建了“学习”沙箱。 它提供了对不需要信用卡的云订阅的免费临时访问。 现在，您可以研究和创建资源，并在高安全性的支持下管理资源，而又不会中断工作。



使用Azure Cloud Shell进行动手培训


用户经常在评论中指出，尽管文字和视频在学习中起着重要作用，但他们更喜欢实践练习。 为了实现这一愿望，我们将Azure Cloud Shell添加到了网站中。 在此Shell中，您可以直接运行Azure CLI，Azure PowerShell，.NET，Java，Node.js，Go和Python，或使用各种命令行实用程序，例如Git，Kubectl，Helm等 。 所有这些都可以直接在浏览器中工作，而无需安装或配置程序。



互动代码练习


Azure Cloud Shell还包括由code .命令启动的基于浏览器的代码编辑器code . 。 它突出显示了基本语法，使您可以在熟悉的环境中快速创建和编辑代码或配置文件。



知识测验


测试知识是一种测试，您需要从建议的答案列表中选择正确的答案以巩固所获得的知识。 知识测试的主要目标之一是关注材料的本质，而不是答案的正确性。 不对错误提供任何惩罚。 而是，向用户解释为什么他选择的答案不正确。
