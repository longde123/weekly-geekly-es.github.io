Windows XP正式死亡，现在终于


每个人都喜欢XP搜索狗，对吗？

大多数用户在5年前就已埋葬Windows XP。 但是忠实的拥护者和生态系统的人质仍然继续使用该操作系统，寻求各种技巧来维持其植物生长状态。 但是随着时间的流逝，Windows XP终于走到了尽头，因为它的最后一个仍受支持的选项POSReady 2009不再受正式支持。

没有回报的点已经过去了。


neowin.net的屏幕截图 。

顾名思义，Windows Embedded POSReady 2009主要用于启动可引起用户注意的应用程序，并带有“免费现金！”之类的惊叹号。 最终在2019年4月完全失去了官方支持，这标志着如此庞大的操作系统家族令人印象深刻的使用寿命的绝对终结。

英国零售商Boots在其位于伊斯灵顿的商店的自助终端上显示一个旧的Windows XP登录屏幕：

Windows POSready 2009 插座的照片theregister.co.uk

尽管员工将倒置的篮子放在机器前面，以使顾客不会触摸它，但由Register阅读器发现的POS终端仍可以欢乐地显示旧的XP登录页面。

Windows XP不再受支持。 但是，有些出版物在正式去世后被推迟了好几年。 Embedded Standard 2009终于在一月份被关闭，以Embedded POSReady 2009的形式提供的扩展实施支持于4月9日结束。

几天前，即2019年4月5日，Microsoft发布了KB4487990，用于更新“莫希干人的后裔”的最新更新，该更新固定了圣多美和普林西比以及哈萨克斯坦的Kyzylorda的时区。

此后，寂静无声。 公司已关闭所有生命支持系统。 病人已经死了，不会再昏迷了。

不幸的是，在Windows XP的大多数变体的全球支持下，它在2014年就结束了，当时大声尖叫和咬牙切齿，当时企业意识到他们突然不得不离开通常的平台。 XP自2001年以来就可以安装了XP，但是由于许多人错过了灾难性的Vista并设置了拒绝更新的趋势，因此至今仍有相当多的XP工作站仍然存在。

幸运的是，Windows 7如此出色，以至于大多数人都可以开始使用它。 但不是全部。 Windows XP是第一个严重遭受“太好退出”问题的操作系统，该问题导致最终用户在操作系统经历黄金时期后拒绝升级足够长的时间。

这部分是对Windows Vista的反应，部分是XP处于旗舰状态的时间异常长的结果。 当比尔·盖茨（Bill Gates）决定专注于Windows XP Service Pack 2（SP2）的安全性时，尽管进行了许多重大更改，但Microsoft并未重命名或发布该操作系统的新版本。 在XP之前，Microsoft迅速对其消费者操作系统进行了分类：在1992年-Windows 3.1，然后-Windows 95、98和Windows 2k和ME在2000年。 XP的任期变得更长了-它是Microsoft五年以上（2001-2006年）的主要消费者操作系统，并且一直在市场上很受欢迎，直到2009年Windows 7发行为止。

这种长寿有时会令人讨厌。 在软盘普及的最后几天，Windows XP受到了固有的缺陷之一的困扰-无法从软盘以外的任何其他设备手动加载磁盘控制器驱动程序，当使用带有AHCI或AHCI的控制器时，此操作系统的使用寿命已经很烦人了磁盘阵列 （是的，可以将驱动程序合并到写入磁盘的Windows XP的新副本中。但是，不断编写Windows XP的新副本是相当不愉快的。）

在XP时代，Microsoft从中吸取了一些教训，以了解如何使人们从他们喜爱的操作系统版本迁移到新版本，这是多么困难。 最有可能的是，正是这种经历使Windows 10成为了一种永恒的，不断更新的产品-尽管Microsoft还采取了一些措施来削弱该策略并将其他控制权返回给用户。

2014年之后，一些大型用户（例如英国政府）维持了垂死的Windows XP生命，向微软支付了可观的英镑以个性化地访问更新，而另一些用户则发现“掩盖”了计算机老化的操作系统在“ POSReady”下，借助注册表中的某些更改，您可以在相当长的一段时间内收到安全更新。

尽管事实上（从安全角度来看）陈旧的Windows XP计算机仍然是传播病毒的沃土，但在某些情况下，运行此操作系统的计算机实际上阻止了攻击者的计划。 至少在2017年爆发的WannaCry恶意软件之一中就是这种情况，当时发现装有XP的计算机经常掉入BSOD并“ 假装死了 ”，这阻止了病毒的传播，其利用没有奏效。 ”。

装有Windows 7的未安装修补程序的计算机成为黑客的主要攻击目标，而马库斯·哈钦斯 （ Marcus Hutchins）尤其感到困惑，后者发现了WannaCry流行病的全球“开关”。

值得记住的是，对于Windows 7，微软已将执行日期定为2020年，该日期即将来临。

尽管Microsoft很高兴建议为POSReady 2009计算机升级到Windows 10或Windows 10 Pro，但用户可用的硬件不太可能从中受益，因为由于系统需求增加，必须更换它。
