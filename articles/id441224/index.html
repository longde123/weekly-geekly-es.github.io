Buat contoh Augmented Faces ARCore di Unity



Saya harap artikel ini akan bermanfaat bagi mereka yang ingin merasakan kemungkinan augmented reality ARCore di Unity, tetapi pada saat yang sama tidak dapat menemukan instruksi yang jelas.

Kami akan mengumpulkan contoh aplikasi augmented Faces ARCore augmented reality untuk Unity, ia akan melacak wajah melalui kamera dan melapisi model 3D wajah rubah.

Kenapa di Unity?

  • Hanya untuk bersenang-senang.
  • Dalam contoh untuk studio Android di luar kotak tidak ada cara untuk bekerja di kamera belakang, hanya dengan kamera depan.
  • Saya harus berurusan dengan Unity sebagai bagian dari pekerjaan.

Untuk menyelesaikan tugas kami, kami akan membutuhkan Unity sendiri, ARCore untuk Unity SDK , perangkat Android yang kompatibel , dan Android SDK . Anda dapat melakukannya tanpa perangkat, tetapi opsi ini tidak tercermin dalam artikel ini.

Persatuan


Buat proyek baru di Unity. Tentukan Jenis 3D


Setelah membuat proyek, impor paket ARCore Augmented Faces yang sebelumnya diunduh bernama arcore-unity-sdk-1.7.0.unitypackage .

Paket impor


Unity akan menanyakan apa yang harus diimpor.

Impor semuanya


Setelah impor berhasil, di jendela "Proyek" kami menemukan dan membuka adegan dengan contoh.

Aset \ GoogleARCore \ Contoh \ AugmentedFaces \ Adegan


Tentukan lokasi Android SDK.

Edit / Preferensi / Alat eksternal


Alihkan platform ke Android, dan klik "Alihkan Platform".

Pengaturan File / Buld


Di jendela sebelumnya, klik tombol "Pengaturan pemain .." untuk mengonfigurasi publikasi.
Ubah nama aplikasi Android.

Pengaturan lainnya


Periksa apakah dukungan ARCore sepadan.

Pengaturan XR


Kami kembali ke jendela "Pengaturan Bangun" dan menunjukkan perangkat kami. Perangkat harus mengaktifkan dan menginstal debugging USB melalui USB.

Jalankan perangkat


Pekerjaan yang dilakukan harus cukup untuk membangun dan menjalankan aplikasi pada perangkat kita. Di jendela yang sama, klik "Build And Run", pada akhirnya kita mendapatkan sesuatu yang serupa.
