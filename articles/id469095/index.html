ML.NET dan Pembaruan Model Builder Dirilis: Apa yang Baru

Kami sangat senang mengumumkan rilis pembaruan untuk Model Builder dan ML.NET . ML.NET adalah sumber terbuka, lingkungan pembelajaran mesin lintas platform (Windows, Linux, macOS) untuk pengembang .NET.

ML.NET menawarkan Model Builder (alat antarmuka pengguna yang sederhana) dan antarmuka baris perintah yang dirancang untuk membuatnya lebih mudah untuk membuat model ML khusus menggunakan AutoML.

Menggunakan ML.NET, pengembang dapat memanfaatkan alat dan set keterampilan yang ada untuk mengembangkan dan mengimplementasikan AI dalam aplikasi, membuat model pembelajaran mesin kustom untuk skenario umum seperti analisis sentimen teks, rekomendasi, klasifikasi gambar, dan banyak lagi!

Pembaruan Model Builder


Rilis Pembuat Model ini menambahkan dukungan untuk skenario baru dan memperbaiki banyak masalah yang dilaporkan oleh pengguna.



Pengembangan peluang: Dalam versi Model Builder sebelumnya, setelah memilih dataset dari file atau dari SQL Server, Anda memiliki kesempatan untuk memilih hanya kolom untuk peramalan (Label). Setiap kolom lain dalam dataset secara otomatis digunakan untuk prediksi (Fitur). Setiap kolom yang tidak ingin Anda sertakan harus diubah dalam set data sendiri di luar Model Builder, dan kemudian set data yang dimodifikasi dimuat.



Di versi sebelumnya Model Builder, ada banyak langkah yang harus Anda ambil setelah membuat kode dan model Builder Model untuk menggunakan model yang terlatih dalam aplikasi Anda, termasuk menambahkan tautan ke proyek perpustakaan yang dibuat, mengatur properti Salin ke Keluaran untuk Menyalin Jika Lebih baru â€dan menambahkan paket Microsoft.ML NuGet ke aplikasi Anda.

Semua ini disederhanakan dan otomatis, jadi sekarang yang harus Anda lakukan adalah menyalin + menempelkan kode dari Langkah Berikutnya ke dalam Model Builder, dan kemudian Anda dapat menjalankan aplikasi Anda.

Pembaruan ML.NET


Ini adalah deskripsi singkat tentang fitur dan perangkat tambahan yang ditambahkan ke ML.NET selama beberapa bulan terakhir.

  • Dukungan .NET Core 3
  • Dukungan untuk skenario baru, seperti perkiraan penjualan, deteksi anomali
  • Pratinjau: pemuat basis data berpemilik yang memungkinkan pelatihan langsung pada basis data relasional
  • Pratinjau: Buat model pembelajaran mendalam khusus untuk mengklasifikasikan gambar menggunakan TensorFlow.
