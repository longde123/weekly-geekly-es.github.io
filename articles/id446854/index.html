Laboratorium teknologi Azure, 11 April di Moskow

11 April 2019 Azure Technology Lab akan diadakan, acara utama Azure musim semi ini.

Teknologi cloud baru-baru ini menarik lebih banyak perhatian. Tidak ada keraguan bahwa Azure adalah salah satu pemimpin di pasar penyedia layanan cloud. Platform ini terus berkembang. Pelajari tentang inovasi terbaru, berkenalan dengan praktik membangun arsitektur TI dan menggunakan teknologi cloud oleh perusahaan-perusahaan Rusia. Pelajari tentang manfaat yang disediakan oleh platform Microsoft Azure dan jalur terbaik bagi rekan kerja untuk mengambil ke cloud.

Pendaftaran



Di acara tersebut, Anda akan menemukan intensif cloud nyata di bawah bimbingan para pakar teknis terbaik.

Anda dapat menjawab pertanyaan Anda yang paling sulit untuk Ahli (Microsoft Valuable Professional) dan berkenalan dengan solusi mitra.

Jumlah kursi terbatas, akses ke acara hanya mungkin setelah konfirmasi pendaftaran.

Acara ini bersifat bisnis, harap ikuti kode pakaian kasual bisnis

  • Cara membuat cloud hybrid di Windows Server 2019 dengan tangan Anda sendiri, terima kasih kepada Azure Stack HCI;
  • Analisis terperinci atas layanan Azure Sentinel baru (SEIM sebagai Layanan);
  • DataBricks dari praktisi ahli Ciprian Jichici dan proyek menggunakan penambangan Pengetahuan dari Istvan Simon dari Prefixbox;
  • Apakah layak untuk menawarkan pelanggan untuk mentransfer aplikasi bisnis (SAP, 1C) ke Azure, dan dalam skenario apa;
  • Cara membuat aplikasi modern dalam siklus berkelanjutan DevOps dengan layanan Azure DevOps;
  • Cara meningkatkan aplikasi dengan wadah Kubernetes dan Linux dan masih banyak lagi.
