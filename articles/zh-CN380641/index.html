测试32 GB的SD和microSD卡

你好，Giktayms！很久以前就有类似的帖子，现在该更新不同存储卡上的信息了。在这种情况下，microSD将与经典SD一起进行测试。原因很明显-microSD比普通卡更通用，也许更流行。它不仅可以用于相机，还可以用于运动相机，智能手机和其他记录和再现设备，包括4K分辨率。在测试中，对来自三个不同制造商的七个存储卡进行了汇总测试。



测试参加者


东芝Exceria Pro SDHC 32 Gb-俄罗斯零售不提供




相较而言，最​​快的显卡。这是我手中的第一张UHS-II格式卡，为了显示它的速度功能，它使用了一种不寻常的读卡器- 金士顿FCR-MLG4，这是为数不多的理解新标准的读卡器之一。
卡的包装暗示了质量的高低-卡不是硬泡，而是硬纸板，卡内包装在一个塑料盒中，在该盒中还有两个其他存储卡的位置。
卡的关键物理区别是第二排触点，对于全格式SD卡，总数从9增加到17，对于microSD卡，总数从8增加到16。代替传统的用于SD的一比特交换，使用了四比特。



Exceria Pro卡支持SDA 4.20规范，并且根据制造商，其最大读/写速度分别为260和240 MB / s。



现实与东芝的保证不一致。该卡速度极快，略低于护照上显示的特征。

东芝Exceria SDHC 32 Gb-尤尔马特的价格-2210 卢布。-每千兆字节69卢布




东芝在其产品阵容中只有两种类型的存储卡-Exceria Pro和Exceria，并且它们都参加了测试。与旗舰产品不同，东芝Exceria用普通的纸板泡罩包装，没有塑料手提箱。



东芝承诺的读取速度为95 MB / s，写入速度为60 MB / s，这并不是该级别的记录，但完全符合UHS-I 3类规范；所述速度与测试结果相吻合，东芝的性能并不令人失望。在污点中，我提出了没有用于存储和携带的盒子的事实。

Toshiba Exceria microSDHC 32 Gb-在Yulmart的价格-1,990 卢布。-每GB 62卢布




首先在测试中订购microSDHC。与全尺寸卡片，硬纸板泡罩相同，没有用于携带的盒子。该套件随附一个microSD-SD适配器。制造商声称读卡速度为95 MB / s，写速度为60 MB / s。



难怪：一切都严格按照协议进行-遵守承诺的速度，卡工作无故障和问题。

Transcend Ultimate U3X TS32GSDU3X SDHC 32 Gb-市场平均价格为2,738卢布。-每千兆字节85卢布




卡装在一个纸板箱中，驱动器内部装在一个塑料箱中以进行携带。Transcend还提供RecoveRx照片恢复软件的免费下载。很好，但是我不想使用它，如果一切都稳定运行就更好了。声明的读写速度分别为95 MB / s和85 MB / s。



很高兴知道制造商遵守速度方面的承诺，创见也不例外，SDHC确认了所声明的性能。

超越极限microSDHC 633x-在Y.市场上的平均价格-2035 卢布。-每千兆字节63卢布




该卡包装在一个纸板泡罩中，其中有一个带有卡的塑料插件和一个microSD-SD适配器。与全尺寸SD一样，提供了独家的RecoveRx软件。读取速度声明为633x-相当于95 MB / s，写入速度为85 MB / s。



然后满足了期望，“小”卡的记录速度非常好。

金士顿SDA3 SDHC 32 Gb-在Yulmart的价格-1,520 卢布。-根据Yandex.Market，每千兆字节为47卢布-1873 卢布




该卡包装在带有塑料内衬的硬纸板泡罩中，看起来手提箱里正在变成很多昂贵的，非常昂贵的存储卡。但是，另一方面，金斯敦卡在参加测试的人中最负担得起的价格有所不同。金士顿宣布其高端存储卡的读取和写入速度分别为90 MB / s和80 MB / s。



超额完成计划！一切不仅好，甚至比陈述的要好。金士顿SDA3卡确实不错，尤其是对于钱。

金士顿microSDHC SDCA3 / 32GB-尤尔马特的价格-1,410卢布。-根据Yandex.Market，每千兆字节为44卢布-1,542卢布




包装完全类似于金士顿全尺寸卡片-塑料插入物和纸板泡罩。为了获得最好的microSD卡，制造商宣布读取速度为90 MB / s，写入速度为80 MB / s。



同样几乎与诺言相提并论，我们还没有达到声明的标准，但是滞后并不关键。
