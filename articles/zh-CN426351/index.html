Microsoft认证和考试的变更

在最近的Ignite会议上，Microsoft推出了一个新的认证计划。 该公司表示， 基于角色/专业的新认证将帮助您提高资格，向雇主展示您的技能和知识并发现新的机会。


已经宣布了六个新的专业： Azure管理员 ， 开发人员 ， 解决方案架构师 ，DevOps工程师，Microsoft 365现代桌面管理员和企业管理员。 Azure管理员认证的考试和培训已经可用，接下来的两项考试-Azure开发人员和Azure解决方案架构师处于beta版。 该列表中其余的三项认证将在年底之前发布。 “这仅仅是开始！ 我们计划在2019年宣布更多专业化项目。” -Selina Winter 在 Microsoft 博客上写文章 。




此外，除了常见的问题答案外，他们还承诺会根据活动情况，拖放变化和更接近实际情况的情况添加新问题。


下面显示的是Azure管理员认证培训课程。




重要！ 70-XXX系列的许多考试从2018年12月31日起将被视为过时，并将由该系列的新考试取代：AZ-10X，AZ-20X，AZ-30X，MS-10X。

如果您打算在不久的将来参加Microsoft考试，请仔细监视英语考试页面上的信息。 也许您的考试已经被新的考试所取代，并且比通过旧的考试更容易通过/为新考试做好准备。

现在，所有Microsoft Azure认证专家都可以通过$ 99过渡考试。 （价格因国家/地区而异，并反映在考试注册过程中。）有效期至2019年6月30日 。


过渡考试仅涵盖新材料以及当前认证与新认证中的技能和概念之间的差异。


如果您通过以下考试之一（或在2018年12月31日之前通过考试，则在该日期之后通过）


  • 70-533：实施Microsoft Azure基础结构解决方案
  • 70-532：开发Microsoft Azure解决方案
  • 70-535：Microsoft Azure解决方案体系结构

您可以通过相应的过渡考试（有效期至2019年6月30日）：


  • AZ-102：Microsoft Azure管理员。 过渡考试
  • AZ-202：Microsoft Azure开发人员。 过渡考试（测试版）
  • AZ-302：Microsoft Azure解决方案架构师。 过渡考试（测试版）

因此，有两种方法来获得新证书的两种方法。


1.通过过时的考试+过渡课程并获得新证书


70-533 + AZ-102-> Microsoft认证专家：Azure管理员
70-532 + AZ-202->微软认证专家：Azure开发人员
70-535 + AZ-302->微软认证专家：Azure解决方案架构师


或


2.通过两项新考试并获得新证书


AZ-100 + AZ-101-> Microsoft认证专家：Azure管理员
AZ-200 + AZ-201-> Microsoft认证专家：Azure开发人员
AZ-300 + AZ-301->微软认证专家：Azure解决方案架构师


我再重复一次，第一种方法将仅在2019年6月30日之前可用。


如果您当前正在获取MCSE证书：Cloud Platform and Infrastructure ，那么您将有时间直到2018年12月31日 。 从2019年1月1日开始，此证书将被新的MCSE：Basic基础结构取代，其中包括MCSA Windows Server 2012和MCSA Windows Server 2016 。
