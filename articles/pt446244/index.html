Extensões do Chrome para desenvolvimento na web e trabalho com o GitHub



Olá pessoal. Quero compartilhar com você algumas das extensões para o Chrome que eu uso ao desenvolver, analisar sites, trabalhar com o GitHub e apenas na vida cotidiana. Estou certo de que, para a maioria dos leitores, a lista dessas extensões não será uma abertura, mas quem aprender pela primeira vez e instalá-las certamente gostará de trabalhar com novas ferramentas.

Eu acho que não vale a pena criar um calçado longo com uma descrição detalhada de uma ferramenta, eles são bastante simples e compreensíveis. Portanto, tentarei falar brevemente sobre as vantagens de seu uso.

Para trabalhar com o GitHub


Octotree é, na minha opinião, a extensão mais útil e conveniente. Permite visualizar o código fonte na forma de uma árvore, que é exibida no lado esquerdo da página. Nova navegação é muito mais conveniente que o padrão. Octotree será especialmente útil se você estiver trabalhando em um projeto de larga escala com um grande número de diretórios aninhados.

O Tamanho do Repositório do GitHub é um plugin simples e conveniente que exibe o tamanho de um repositório individual e o tamanho de cada arquivo nele.

O OctoLinker é uma extensão que fornece navegação rápida por código, transformando simples elementos de código em links clicáveis. Seguir este link permite encontrar rapidamente a fonte. Dependendo da linguagem de programação, a extensão mostra o arquivo de origem ou leva o usuário a um site externo. Os elementos convertidos em um link são marcados com pontos cor-de-rosa, para que você possa ver imediatamente onde há um link interativo.

O GitHub Hovercard fornece acesso instantâneo às informações do usuário, repositórios, perguntas e confirmações de qualquer lugar no site do GitHub. Apenas passe o mouse sobre as informações.

npmhub Ao exibir um repositório no github.com que possui um arquivo package.json , esta extensão analisará as dependências no pacote e exibirá os links com uma descrição para eles logo abaixo do arquivo README.

github npm stats mostra estatísticas de download de pacotes no site do NPM, se houver.

O Github refinado exibe brunches publicados recentemente e também permite trabalhar novamente com solicitações e mostra avatares de usuários que responderam aos comentários.

Contribuições isométricas - esta extensão não é tão útil quanto as anteriores, mas muito interessante). Permite alternar entre um gráfico de contribuição regular no GitHub e uma versão do pixel art isométrico.

Para trabalhar com sites


O conta - gotas é uma ótima ferramenta para visualizar as cores usadas nos sites e criar suas próprias paletas. Eu uso quase todos os dias)).

Captura de tela de página inteira . Imagine que você desenvolveu um pouso em 3 a 5 telas e não sabe como torná-lo em tela cheia. Pressione a tela de impressão 5 vezes e faça peças no editor uma sob a outra? Oh bem! O FPSC é uma ótima ferramenta que tira uma captura de tela da página atual em alguns segundos. É muito útil se você precisar verificar a otimização do site sem fazer o download de serviços especiais, bem como capturar completamente o site que você criou.

WhatFont - com a ajuda desta extensão, você pode descobrir facilmente quais fontes são usadas no site, basta clicar na extensão e mover o cursor para o local que você precisa. É simples e elegante. Desativado pressionando o botão "Esc". Também mostra a cor do texto, altura, tamanho da fonte e outros parâmetros.

PerfectPixel - esta extensão ajudará você a desenvolver seu site com precisão pixel por pixel.

CSS Peeper é uma ferramenta de análise de CSS projetada especialmente para designers. O objetivo da extensão é permitir que os designers se concentrem no design e gastem o mínimo de tempo possível pesquisando o código. Apenas tente em ação e você entenderá tudo.

Para o trabalho diário


O Google Translator é outra ferramenta que quase nenhum dos meus dias de trabalho pode prescindir. Esta não é uma página, é uma extensão. Com ele, você pode traduzir rapidamente o texto selecionado diretamente na página sem acessar a página do tradutor. P.S. não traduz textos longos, mas isso não importa.
