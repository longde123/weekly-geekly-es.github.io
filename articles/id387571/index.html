Pembaruan Kumulatif Pertama Tersedia untuk Windows Mobile Insiders

Tadi malam, Gabe Aul mengumumkan dimulainya distribusi pembaruan kumulatif pertama untuk orang dalam menggunakan perakitan 10586. Apa yang baru dan dukungan perangkat apa yang telah ditambahkan, baca di bawah kucing.


Gabe Aul mengatakan bahwa segera semua perangkat akan perlahan-lahan meningkatkan ke Windows 10. Untuk Windows 10 sendiri, pengguna akan melihat pembaruan di Pusat Pembaruan Windows jauh lebih sering dan sekarang tidak hanya rakitan baru akan dirilis, tetapi juga pembaruan kumulatif seperti hari ini.

Di awal minggu depan, Windows Insider akan tersedia untuk beberapa perangkat baru:


Itu akan terjadi, setelah memperbarui Alat Pemulihan Perangkat Windows ke versi terbaru dengan dukungan untuk memulihkan firmware pabrik dari perangkat ini.
  • LG Lancet
  • Microsoft Lumia 550
  • Microsoft Lumia 950
  • Microsoft Lumia 950 XL

Apa yang baru dengan pembaruan 10586.29?


  • Berbagai peningkatan dalam penggunaan OS, terutama untuk perangkat dengan sedikit memori bebas (operasional, seperti yang saya mengerti), meningkatkan rendering kartu
  • Peningkatan tambahan dalam kompatibilitas mundur dengan aplikasi untuk 8.1 yang ditulis dalam cahaya perak
  • Peningkatan kinerja dan stabilitas Edge.
  • Peningkatan tambahan dalam stabilitas Bluetooth.
  • Memecahkan masalah dengan beralih ke koneksi seluler aktif pada smartphone dual sim
  • Pengaturan MMS sekarang dikembalikan dengan benar.

Pelajari lebih lanjut tentang memperbarui blog Microsoft .
