Automatize a criação de perfil em Java 10/09

Convidamos você para uma reunião aberta em 10 de setembro . Automatizamos a criação de perfil do Java, que será realizada no site Raiffeisenbank em Nagatino.

Juntos, descobriremos como testar e otimizar o desempenho dos microsserviços e também aprenderemos como sair da rotina de verificação da JVM na criação de perfis.



Vyacheslav Smirnov, Raiffeisenbank
Experiência em criação de perfil Java 8


Ao testar e otimizar o desempenho dos microsserviços, formei abordagens para criar um perfil do JDK. No relatório, falarei sobre a experiência do uso de ferramentas: SJK, Java Fligth Recorder, Java Mission Control (Oracle Java 8), Telegraf, Influxdb, Grafana. E sobre alguns erros nas bibliotecas: org.json4s, alpakka.

Alexey Ragozin, consultor independente
Automação de Perfis e Diagnósticos da JVM


Sobre o que falar:

  • Informações de diagnóstico disponíveis na JVM para coleta automática (JMX, FlightRecorder, etc.). Como obtê-lo e como é útil;
  • Análise de informações em modo automático. Como passar de uma matriz de dados de origem para métricas integrais úteis da qualidade do aplicativo em termos de desempenho.
