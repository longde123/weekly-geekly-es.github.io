9 penemuan open source terbaik untuk Oktober 2019

Selamat November, tuan dan nyonya. Saya menyiapkan bagi Anda pilihan penemuan paling menarik dari sumber terbuka untuk Oktober 2019.


Untuk daftar lengkap alat, artikel, dan laporan baru yang bermanfaat, Anda dapat menghubungi saluran telegram saya @OpensourceFindings ( tautan cermin jika yang asli tidak terbuka).


Dalam edisi hari ini.
Teknologi di dalam: Karat, Swift, TypeScript, JavaScript, Go, Scala, Python.
Topik: pengembangan web dan seluler, visualisasi data, alat pengembang, dokumentasi.


Masalah terakhir .


vue-interactive-paycard


Komponen animasi yang sangat bagus dan keren untuk kartu kredit.
Ditulis dalam JavaScript + Vue.


Tautan



rx


Editor minimalis untuk grafik piksel. Memiliki manajemen yang mirip dengan vi.
Ditulis dalam Karat.


Tautan



Bow


Perpustakaan untuk pemrograman fungsional di Swift. Mendukung "emulasi HKT", tipe data aljabar dan polimorfisme ad-hoc.
Ditulis dalam dan untuk Swift.


Tautan



rentan terhadap situs web


Aplikasi konsol untuk memeriksa kerentanan di situs mana pun: cukup masukkan alamatnya.
Ditulis dalam javascript.


Tautan



Sanddance


Perpustakaan yang sangat indah untuk visualisasi data.
Ditulis dalam TypeScript.


Tautan



spleeter


Utilitas yang memungkinkan Anda mengekstrak vokal, drum, bass, dll. Dari lagu apa pun. Jaringan saraf!
Itu ditulis dalam Python.


Tautan



grpcui


Program dengan antarmuka grafis untuk berinteraksi dengan gRPC. Seperti tukang pos hanya untuk gRPC.
Ditulis dalam Go.


Tautan
Dan juga ada analog dari curl



polynote


Editor dan runtime program dalam beberapa bahasa: Scala, Python, SQL, Vega. Posisi itu sendiri sebagai solusi untuk bekerja dengan data besar.
Ditulis dalam Scala dan Python.


Tautan



putri duyung-js


Utilitas untuk menggambar berbagai grafik dari teks. Bekerja dengan baik, terlihat cantik. Banyak digunakan untuk dokumentasi.


Tautan

Bonus!


Daftar praktik terbaik untuk * bahasa pemrograman apa pun. Mengapa "siapa saja" dengan tanda bintang? Tentu saja, semua ~ 3600 bahasa pemrograman terkenal tidak tercakup di sana. Tapi, Anda bisa menambahkan bahasa favorit Anda jika belum ada. PR dipersilakan!


Itu saja untuk hari ini. Bagi mereka yang menyukai pilihan - berlangganan saluran. Ada banyak proyek menarik lainnya. Saran untuk perbaikan, tautan ke proyek, umpan balik - tulis di komentar.
