IntelliCode sekarang juga dalam TypeScript / JavaScript

Di Build 2018, kami mengumumkan Visual Studio IntelliCode: seperangkat alat AI yang berkontribusi pada pengembangan yang lebih baik. Bekerja sama dengan tim IntelliCode, kami dengan senang hati mengumumkan bahwa IntelliCode sekarang tersedia untuk pengguna TypeScript / JavaScript melalui ekstensi IntelliCode untuk VS Code .



Apa itu IntelliCode?


IntelliCode melengkapi IntelliSense dengan merekomendasikan elemen penyelesaian yang sering digunakan untuk konteks saat ini di bagian atas daftar. Rekomendasi kontekstual ini didasarkan pada metode umum yang dipelajari oleh IntelliCode di ribuan proyek kehidupan nyata. Ini menghemat waktu Anda dengan meletakkan apa yang paling mungkin Anda gunakan di bagian atas daftar: Anda tidak perlu mencari berdasarkan abjad.

Bagaimana cara kerjanya untuk TypeScript / JavaScript?


Instal ekstensi Visual Studio IntelliCode melalui toko atau instal pada tab "Extensions" di Visual Studio Code. Kemudian cukup buka file TypeScript atau JavaScript dan mulai mengedit. Anda akan mulai melihat rekomendasi IntelliCode ditandai dan diangkat di bagian atas daftar penyelesaian Anda.



IntelliCode akan mengubah rekomendasinya tergantung pada konteks dan mengurutkan opsi berdasarkan relevansi dalam situasi khusus ini. Misalnya, jika elemen, seperti peta , digunakan dalam ekspresi kondisional atau loop, Anda akan melihat metode yang kemungkinan besar akan Anda gunakan untuk membuat kondisi. Jika Anda memeriksa penyelesaian untuk peta yang sudah ada di dalam ekspresi bersyarat, Anda akan melihat daftar kalimat lain.



IntelliCode juga akan membantu Anda dalam berbagai pustaka JavaScript, baik Anda menulis kode sisi klien menggunakan pustaka seperti React, Angular, atau Vue, atau melakukan pengembangan sisi server menggunakan Node.js dan Express.



Bagaimana cara melaporkan bug dan masalah?


IntelliCode masih dalam pratinjau. Jika Anda tiba-tiba menemukan bug di ekstensi, kami akan sangat senang jika Anda memberi tahu kami tentang hal itu di GitHub .
