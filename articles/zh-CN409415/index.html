拉斯维加斯的欢乐与诱惑：CES 2018上的最新HyperX

当该国摆脱蛋黄酱昏迷而逐渐变成东正教圣诞节时，年度消费电子产品展览会CES 2018则在地球的另一端举行。在内华达州的沙漠中部，计算机和家用电器制造商聚集在美国拉斯维加斯（以“恐惧与厌恶……”的方式）。展示即将在明年上架的新产品。 传统上，HyperX还带来了最新的游戏配件。 因此，我们有了一个无线耳机，它具有别致的声音并可以进行有线连接；一个机械键盘具有独立的RGB背光和最新的网络体育技术；还有一个新的鼠标，可以对所有颜色和阴影进行圆形照明。 并不是说我们过去的设备已经过时，但是自从我开始生产游戏外设以来，就很难停止了。


HyperX Cloud飞行耳机


您问-我们做到了。 HyperX Cloud Flight是一款带有插入式麦克风的头戴式耳机，其声音既可以通过空中传输，也可以通过电缆传输。 而且，这款耳机不是“众多产品”中的一个，而是具有完全烙印的杀手级功能：HyperX Cloud Flight具有所有无线游戏耳机中最长的电池寿命。 内置电池充电一次即可使用30个小时！ 这样的马拉松比赛不能忍受每一个韩国的MMO-shnik，但是对于普通的俄罗斯游戏玩家来说，这足以应付一周的时间。

我们继续改进耳机的设计，从Cloud Stinger开始，它已经发生了很大的变化。 Cloud Flight是一种时尚且外向中立的设备，适合任何年龄的游戏玩家。

对于声音，传统上是由50毫米扬声器负责的，并且通过不同的连接，输出频率范围将有所不同：对于电线，该频率范围为15-23 000 Hz，对于“无线”，则为20-20 000 Hz。 但是，在频率范围内的这种轻微“让步”使您可以在距计算机20 m（！）的距离内使用耳机。 这要归功于它使用Wi-Fi通信协议，而不是公认的蓝牙，蓝牙的最大范围限制在10米以内。

Cloud Flight具有红色背光，可以简单地发光或脉动-单独的按钮负责开关。 当然，使用背光可以减少耳机的工作时间-脉动最多可长达18小时，均匀发光最多可长达13小时。 因此，如果您独自一人并在晚上玩游戏，则可以关闭背光灯以节省电池。 您可以使用完整的电缆通过USB为耳机充电。

Cloud Flight已获得TeamSpeak和Discord的认证，并且还可以与PS4控制台一起使用。 不支持Xbox One的原因很简单，因为有人（我们不会指着手指并提供指向该站点的链接）已禁止在其控制台中使用第三方蓝牙加密狗。

HyperX Alloy Elite RGB键盘



我们将继续开发合金机械游戏键盘产品线！ 现在，我们为其添加了自定义的单独RGB背光！ 是的，我们已经在YouTube上为新键盘的预告片阅读了您的评论。 不，如果您没有最紧迫的需求，根本不需要将过去的Alloy Elite更改为具有RGB背光的版本-毕竟，拥有过去的而不是最新的型号并不会使您成为坏玩家。 首先，对于HyperX，为游戏玩家提供非常酷的配件非常重要！


乍一看， Alloy Elite RGB整体上似乎重复了以前的模型。 这是一种机械键盘，可根据您的演奏风格和日常需求，提供三种版本的CHERRY MX开关：蓝色，棕色或红色。 主要玩吗？ 您需要MX棕色或红色。 您经常输入文字吗？ MX Blue，当然。 但这不是唯一的重点。 HyperX NGenuity专有软件将向用户提供，使您可以从单个界面使用键盘+鼠标组合执行所有设置。 现在，它将变得非常酷！ 我们将花一些时间来详细说明。

Alloy Elite RGB具有非区域照明-每个按键都有自己的LED，其颜色可以通过NGenuity软件进行调整。 您甚至可以创建一个凉爽的杂烩，使专业水平提高+20，在黑暗房间的照明条件下增加+10。
如果没有人知道，那么钢制框架会为键盘增加相当大的坚固性和可靠性-当然，这更多是一种设计元素，但无论如何，金属材质的印象要比塑料材质更好。 在僵尸末日的情况下，钢合金Elite RGB可以用作撞击式冷武器。 但是这种情况不属于保修范围。

套件中有八个备用键-WASD和1-2-3-4，-质感，灰色，风格化的“钛”。 包括用于移除键帽的镊子。 传统上，右上角的媒体按钮和非常方便的音量旋钮会突出显示颜色。 还有一个柔软的可移动掌垫，但对于打印多于游戏的人很有用。


就像我们已经说过的那样， Alloy Elite RGB的背光可以为每个按钮分别设置，并且预设保存在键盘本身的内存中。 对于那些想一次疯狂地卡住所有按键的人，Alloy Elite RGB具有N键翻转技术，借助它，您可以同时用手指按住10个按钮，另一只鼻子，额跟和两种语言的脚跟-所有这些都将被处理并传输到游戏。 现在，请记住，您和您的朋友在幼儿期时是如何在同一键盘上弹奏Mortal Kombat的，有时您的水龙头未得到处理，因为对手一次按下了三个按钮！ 是时候报仇了-选择HyperX Alloy Elite RGB，打电话给朋友，放同样的Mortal Kombat，这一次不要输。

HyperX Pulsefire电涌鼠标




Pulsefire Surge不是“带灯泡的Pulsefire FPS”。 机械手不仅在外部而且内部都不同。 新颖性不仅是美丽，明亮和比彩虹的所有颜色略微闪烁，而且还具有一个新的传感器-它比Pulsefire FPS更准确，更快。

鼠标的设计呼应了Cloud Flight耳机-它变得更加激进，严肃和牢固。 沿着机械手的轮廓制作了一条透明条，在该条下隐藏了RGB LED。 同时，没有一个或两个；背光灯被宽大地安装在整个周围。 在“输血”模式下，彩色波从字面上穿过鼠标，实际上看起来非常酷，您无法传输照片。 当然，背光将通过HyperX NGenuity应用程序进行配置，还有一个用于在同一位置创建宏的工具。



里面有什么新东西？ 丰富的内部世界新型高精度Pixart 3389传感器（Pulsefire FPS中为3310），宣称的分辨率仅为16,000 dpi，简直不雅。 当然，我们不会参加愚蠢的dpi竞赛，但是由于市场上有这种传感器，因此不安装是一个罪过。 HyperX试图为其产品采用最酷的最新组件。


我们听到了观众的一个问题：“什么样的敏感性？” 传感器分辨率与光标速度无关。 dpi越高，鼠标在光学最困难的表面上处理的移动就越准确。 您可以在鼠标上设置最大分辨率，但可以降低光标速度-在游戏中，对于操纵器的任何猛击，都可以以最高精度的运动获得通常的灵敏度。
在按钮下方，仍安装了欧姆龙开关，所有受尊敬的鼠标中都存在。 5千万次点击意味着您的手指掉落的可能性远大于您坚持的次数。
