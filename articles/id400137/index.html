Leaf Browser Alpha - browser dengan tab pohon


Penggemar ekstensi Tab Gaya Pohon (tab mirip pohon) untuk Firefox, dan mereka yang tidak dapat hidup tanpanya, tetapi ingin mencari alternatif yang layak untuk Firefox, memiliki proyek terbuka bernama Leaf Browser.
Ini adalah browser yang didasarkan pada Google Chrome, tetapi dengan tab seperti pohon.


Browser diimplementasikan sebagai aplikasi untuk Google Chrome, dan dapat diinstal dalam beberapa klik dari Toko Web Chrome langsung ke browser Anda.

Sayangnya, penulis proyek kehilangan sedikit antusiasme dalam pengembangannya, tetapi saya pikir ini karena popularitas kecil dari proyek tersebut. Meskipun baru-baru ini dia tampaknya telah berubah pikiran menjadi lebih baik.
Namun demikian, saya benar-benar ingin proyek menjadi lebih luas dan bahwa akan ada lebih banyak pengembang dan pengguna yang tertarik pada proyek ini.


Jika Anda menyukai proyek ini, tolonglah, kirim laporan bug dan tulis komentar di Toko Web Chrome. Tunjukkan pada penulis bahwa ciptaannya tidak acuh pada Anda.
