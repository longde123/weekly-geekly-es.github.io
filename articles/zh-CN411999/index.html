拓扑优化和仿生设计的软件审查



生物设计或仿生设计方法涉及在自然界中借用的有效解决方案的发达设计中。 通常，这些是平滑的线，是连续的或管状的细结构的分散结构。 生物设计是拓扑优化的一种特例，它是通过正确计算零件中材料的分布（通常是使其结构复杂）来实现机械强度和轻度的最佳组合的一种方法。

在本文中，我们考虑可以处理这些任务的软件 。

如何运作




现代设计工具不仅可以根据给定的质量和强度框架预测零件的变形程度，还可以对其进行优化。 通常，准备一个空白模型，然后从中删除多余的材料。 此过程称为拓扑优化。 在这种情况下，经常使用的另一个术语是“生成设计”，它是指由软件算法部分执行的设计。 拓扑优化的结果是一个复杂的结构，在此基础上形成了用于生产的零件的模型。 这种部件具有所需的强度和最小的重量，但是难以通过传统方法制造，并且添加剂的生产将是最佳的。

实施实例


使用生成设计制作的产品具有非同寻常的未来派外观，通常可用来展示添加剂技术的功能。

宝马S1000RR




例如，宝马在这里展示了技术的前景，它基于S1000RR制造了概念车，但带有轻型3D打印金属框架。 如果我们谈论更多的实用性应用-宝马已经在劳斯莱斯和i8跑车的生产中使用3D打印。

MX3D桥




来自MX3D的荷兰人已经完成了桥梁支撑基座的生产，该基座计划于2019年在阿姆斯特丹安装。 在桥梁的制造中，使用了Autodesk软件和WAAM技术-通过工业机械臂沉积金属条。



俄罗斯工业




我们还没有打印桥，但是在功利领域，拓扑优化将成功应用。 例如，计算机工程中心SPbPU正在使用专有软件朝这个方向工作。 您可以在该中心的网站上阅读有关要解决的任务的更多信息，在这里我们注意到我们主要是在谈论航空航天业。

软件-集成系统


市场上有大量软件产品，通过它们可以进行拓扑优化计算。 通常，此功能由执行应变建模和强度计算的软件包提供。 让我们从机械工程中用于设计和计算的“大型”复合体开始。

西门子NX




NX是一个通用的集成系统，用于设计，工程计算和准备CNC机器的控制代码-CAD，CAE和CAM。 NX CAE模块内的拓扑优化重点在于CAD装配中零件的交互。

要优化的零件放置在工作空间中，其他不受该过程影响但参与交互的零件都附加到该工作空间中。 紧固和施加载荷的指示是在组件的框架内精确进行的。

西门子Solid Edge




Solid Edge-固态和曲面建模的CAD系统，主要设计用于与零件和装配体协同工作，但具有内置的工程分析模块。 开发人员将Solid Edge定位为快速设计和原型制作的一种手段，为增材制造“锐化”。



优化过程可分几步进行，而无需进行微调：加载“空白”，指示加载的孔和位置，设置所需的重量减轻比例和安全系数。 可以使用常规Solid Edge工具将结果零件直接发送到打印或定稿。

CATIA 3DEXPERIENCE




截至2018年初，3DEXPERIENCE平台由104个称为角色的模块组成。 这些是各种设计任务，开发过程管理，仿真，可视化的解决方案。 这些角色之一是功能驱动的生成器设计器。



该角色不仅可以根据指定的要求优化零件的拓扑结构，还可以优化最终的对象-简化，消除不规则性。

SOLIDWORKS




SOLIDWORKS 2018中添加了拓扑优化。Professional和Premium版本的SOLIDWORKS Simulation模块中提供了拓扑优化。 工程师指示载荷，优化条件-例如，最佳的刚度与质量之比，然后开始拓扑研究。 在过程结束时，在准备生产时，会优化零件的网格。

Autodesk Fusion 360




用于在CNC机床上进行设计，工程计算和生产准备的云平台-CAD / CAE / CAM。 在称为Ultimate的最大订阅中，可以使用Advanced Simulation模块。 他负责建模变形，使用各向异性材料并优化拓扑-形状优化。



优化优化的可能性不是很广泛，但是对于主要任务-以最小的重量和所需的强度来制造零件-便足够了。

ANSYS机械




该软件产品的主要功能：

-静态计算；
-动态分析；
-热计算；
-声学计算。

ANSYS Mechanical允许您模拟结构在载荷下的变形，以计算振动的影响和共振的发生。 要进行自动拓扑优化，必须为零件导入工件模型，指示连接点和载荷应用，并设置边界条件，例如最小厚度。 可以模拟传统材料和各向异性材料-复合材料，3D打印的零件。

更简单，更实惠的软件


让我们从“繁重的”工业软件过渡到功能更少，入门门槛更低且成本相对可承受的解决方案。

solidThinking启发




相比之下，例如，与Ansys Mechanical相比，它是更加用户友好的软件，旨在解决几个狭窄的问题-建模装配中零件的相互作用，简单的轮廓设计，拓扑优化。 您需要使用的对象可以在Inspire中创建，也可以从其他CAD系统导入。



在“切除多余的部分”之后，简化并优化了优化的零件，这极大地改善了外观并允许您立即将其发送到生产中。



除拓扑优化外，Inspire还能形成网格填充而不是连续填充，如果需要进一步减轻零件的重量，此功能很有用。

Autodesk Netfabb Ultimate




3D打印机的许多用户都知道该程序是一种方便的工具，用于修复网格并将模型分为多个部分。 Netfabb 2018的功能更加广泛-例如，创建了用于FDM打印的分支支撑结构，用于SLS和SLM的金属零件变形建模以及在Ultimate Edition中添加了拓扑优化和网格填充生成功能。

为什么和为谁




现在是拓扑优化工具，主要是由Siemens，DassaultSystèmes，Autodesk开发的大型CAD / CAM / CAE软件包的组件。 而且，即使不是纯粹出于演示目的，该技术也主要用于金属合金的3D打印中。 生成设计与强度分析相结合，解决了在保持强度的同时减小产品质量的问题，这不仅在航空航天工业中很重要。 在任何生产中，节省材料都会降低成本，特别是在涉及昂贵的金属粉末时。

很难将所描述的大多数软件产品称为易学且经济实惠的产品。 但是有更简单的解决方案，例如-solidThinking Inspire。 可以假定，随着金属和工程塑料印刷设备和耗材成本的降低，相对简单的拓扑优化工具将变得更加普及。
