Microsoft HoloLens 2简介

今天，在致力于在巴塞罗那举行的MWC 2019移动技术展览会的新闻发布会上 ，我们向全球介绍了HoloLens 2。



细节剪下。

对于Microsoft以及我们的合作伙伴，客户，IT行业和世界来说，这是一段美好的时光。 云几乎无限的计算能力和功能，再加上越来越多的“智能”和可接收设备，创造了我们几年前梦dream以求的实用技能。


当智能云和智能设备得到混合现实的补充时，我们就有了成就惊人并帮助更多人的环境。


今天是微软的重要里程碑。 我们展示了多年努力的结果以及我们许多团队（包括Azure，HoloLens，Dynamics 365和Microsoft设备）的艰苦而热情的工作-在这个时刻，金额超过各个组件。 从创新的设备设计到具有混合现实元素的云服务-今天的公告是许多团队共同努力的结果。


没有我们专用的客户，合作伙伴和开发人员社区，这一切都是不可能的。


我很荣幸代表每个团队成员向您介绍HoloLens 2，并讲述我们今天在巴塞罗那MWC开幕之前所做的所有公告。


HoloLens 2简介



自2016年推出HoloLens以来，我们已经看到了混合现实如何改变工作流程。 我们已经为数十万从事不同领域工作的人们提供了获得新机会的机会。 从建筑工地到生产车间，从手术室到学校课程，HoloLens都在改变我们的工作，学习，沟通和执行任务的方式。


当数字世界超越二维屏幕并陷入三维现实时，我们正在进入计算的新时代。 计算与协作的新时代将帮助我们实现更多，跨境合作，并在3D模式下更简单有效地进行协作。


今天，我们很自豪地介绍Microsoft HoloLens 2的世界。


客户要求我们专注于三个关键方面，以使HoloLens变得更好。 他们希望我们增强沉浸效果，使眼镜更舒适并减少投资回报。


由于包括视觉特性在内的大规模改进， 浸入效果得到了极大的增强，其结果是全息图变得更加令人印象深刻和逼真。 我们将HoloLens 2的视野扩大了一倍以上，同时保持了业界领先的全息密度（每视点47像素）。 HoloLens 2具有新的显示系统，使我们能够以低功耗实现显着的性能改进。 在HoloLens 2中，我们还完全改变了与全息图交互的方式。


利用我们的新型深度传感器以及内置的AI和语义理解功能，HoloLens 2可以使用与在现实世界中操作物理对象相同的本能手势直接操作全息图。 除了改进视频控制器和直接操作全息图的可能性外，HoloLens 2还引入了眼动追踪传感器，使与全息图的交互更加自然。 您可以使用企业范围的身份验证系统Windows Hello登录，该系统可以通过眼睛的虹膜识别您，从而使多个人可以更快速，更安全地开始共享设备。


重心更加平衡，使用轻质碳纤维以及无需调整即可穿上设备的新型机制，从而提高了舒适度 。 我们使用新的蒸汽回收技术改善了热管理，并针对各种尺寸和形状的人头提供了广泛的生理变化，并设计了HoloLens 2，使耳机可舒适地调节并适合几乎所有人。 新型完美的佩戴系统使您可以舒适地戴着耳机数小时，而不必摘掉眼镜，因为HoloLens 2可以适应您的需要并且可以直接戴在眼镜上。 当需要摆脱混合现实时，只需举起遮阳板，然后再执行其他任务即可。 这些改进加在一起超过三倍，提高了设备​​的舒适度和人体工程学性。


使用Microsoft混合现实应用程序（例如Dynamics 365 Remote Assist，Dynamics 365 Layout和新的Dynamics 365 Guides应用程序）可以缩短投资回收期 。 除了原始价值外，我们的混合现实合作伙伴生态系统还提供了HoloLens创建的各种解决方案，适用于各种行业和任务。 随着使用混合现实技术的新一轮企业家的涌现，合作伙伴生态系统得到了扩展，这些企业家正在认识诸如HoloLens 2和Azure服务之类的设备的潜力，这些设备提供了智能系统来分析混合现实所需的空间，语音识别和可视图像，此外用于存储，保护和收集遥测应用程序的可信云服务。


基于原始HoloLens耳机的独特功能，HoloLens 2是最先进的智能设备。 结合现有的和新的Azure服务，新的HoloLens 2具有更广泛的功能，不需要其他配置。


HoloLens 2设备将于今年上市，价格为3500美元。 包括Dynamics 365 Remote Assist在内的软件包的起价为每月125美元。 首先，HoloLens 2将出现在美国，日本，中国，德国，加拿大，英国，爱尔兰，法国，澳大利亚和新西兰。 您可以立即在https://www.microsoft.com/zh-cn/hololens/buy上订购HoloLens 2 。

Dynamics 365指南  


当我们在10月1日宣布Dynamics 365远程辅助和Dynamics 365布局时，我们将它们称为HoloLens的“第一个”混合现实应用程序。


今天，我们很高兴地宣布《 Microsoft Dynamics 365指南》。 Dynamics 365指南是一个新的混合现实应用程序，可帮助员工从实践中学习。 指南通过分步说明增强了学习效果，这些分步说明向员工展示了在现实生活中如何以及何时使用其所需工具。 除了在HoloLens上使用Guides的经验外，Guides PC应用程序还使创建交互式内容，附加照片和视频，导入3D模型和自定义材料变得容易，使您可以将理论知识转化为可复制的学习工具。


该应用程序将有助于最大程度地减少停机时间并提高关键设备和流程的效率，并将成为在上一代HoloLens设备以及新HoloLens 2耳机上运行的第三个Dynamics 365应用程序。

Azure混合现实服务


今天，我们还宣布了两个新的Azure混合现实服务。 这些服务旨在帮助每个开发人员和每个公司创建跨平台，上下文相关，混合现实的企业级应用程序。


Azure Spatial Anchors使公司和开发人员可以创建混合现实应用程序，其中全息图存储在特定的物理空间中。 后来，几个在iOS和Android上使用HoloLens，手机或平板电脑的人将能够立即访问它们。 开发人员可以激活空间定向功能，以帮助他们的客户工作和更快，更轻松地学习。 我们已经看到了这样的合作实例，以提高在生产，建筑，医学教育和其他领域的联合工作的有效性。


Azure 远程 渲染有助于保持3D内容的高质量和详细程度，从而使决策更快，更有效。 如今，为了与移动设备和混合现实头戴式耳机上的高质量3D模型进行交互，开发人员通常不得不“修剪”或简化3D模型以在必要的设备上运行。 但是对于实施项目检查和医疗计划而言，每个细节都很重要，而简化操作可能会导致丢失关键决策所需的重要数据。 该服务将在云中播放高质量3D内容，并将其实时广播到设备，同时保持细节。


Azure Spatial Anchors今天作为公共预览发布。 当前，Azure远程渲染已在封闭的初始版本中提供，稍后，该服务将以公共版本发布。


Microsoft HoloLens自定义程序  


从建筑工地，手术室到国际空间站，HoloLens都用于复杂的情况。 根据北美和欧洲使用的安全眼镜的几种标准，该设备已通过基本动态测试。 该耳机经过了测试，符合ANSI Z87.1，CSA Z94.3和EN 166的基本要求。我们与HoloLens 2一起提供了Microsoft HoloLens定制程序，以便客户和合作伙伴可以使HoloLens 2适应其环境要求。


我们的长期HoloLens合作伙伴Trimble采取了第一个HoloLens定制程序，该公司去年宣布了针对HoloLens的Trimble Connect，这是第一个提高在现场使用混合现实的效率的安全帽。 今天，他们宣布了带有Microsoft HoloLens 2的Trimble XR10，这是一种新的安全头盔，使在高风险场所工作的员工可以在不离开工作地点的情况下使用耳机。
