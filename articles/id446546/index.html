Microsoft Memperluas Azure IP Advantage dengan Manfaat IP Baru untuk Inovator dan Startup Azure IoT



Hari ini, kami dengan senang hati mengumumkan perluasan program Microsoft Azure IP Advantage untuk memasukkan manfaat baru bagi inovator dan startup Azure IoT . Kami pertama kali mengumumkan Azure IP Advantage pada Februari 2017. Ini bertujuan untuk memberikan perlindungan komprehensif terhadap risiko kekayaan intelektual (IP) untuk klien cloud kami. Tren yang kami lihat pada waktu itu, dan kami lihat sekarang, adalah risiko yang semakin besar dari inovasi cloud karena klaim paten. Tahun lalu, kami bergabung dengan Jaringan Penemuan Terbuka (OIN) dan Jaringan Lisensi untuk Transfer (LOT) untuk membantu menghilangkan risiko pencurian paten pelanggan dan mitra kami.

Terima kasih telah menerjemahkan MSP kami, Lev Bulanov

Tetapi kami percaya bahwa kami bisa berbuat lebih banyak. Jumlah paten IoT di Amerika Serikat meningkat lebih dari 400 persen antara 2013 dan 2018. Oleh karena itu, setelah berbicara dengan pelanggan dan berpikir tentang bagaimana kami dapat lebih meningkatkan nilai program Azure IP Advantage, kami memutuskan untuk memperluas program dengan manfaat baru yang difokuskan pada ekosistem Azure Internet of Things (IoT) dan startup untuk membantu pelanggan Azure menghindari biaya hukum. . Fitur-fitur baru yang diumumkan hari ini meliputi:

  • Cakupan Kompensasi Tidak Terbatas untuk Microsoft Azure Sphere dan Windows IoT. Kompensasi membantu melindungi klien dari klaim pelanggaran hak kekayaan intelektual yang diajukan terhadap klien karena menggunakan produk atau layanan. Ekspansi hari ini memberikan kompensasi tanpa batas untuk Azure Sphere dan Windows IoT, termasuk perangkat lunak open source yang telah dimasukkan Microsoft ke dalam produk-produk ini.
  • Akses 10.000 paten untuk pelanggan menggunakan Azure untuk perangkat IoT untuk melindungi diri mereka dari tuntutan hukum IP. Manfaat ini dapat membantu mencegah tuntutan hukum terhadap pelanggan Azure karena beban kerja dan aplikasi mereka yang berjalan pada Azure atau pada perangkat IoT mereka, karena pelanggan yang memenuhi syarat dapat memilih paten untuk digunakan dalam pertahanan litigasi.
  • Peluang untuk startup di Azure untuk memperoleh paten Microsoft untuk membantu mengembangkan bisnis mereka. Startup yang terampil yang juga bergabung dengan LOT Network dapat memperoleh paten Microsoft melalui LOT di bidang teknis, termasuk AI, multimedia, dan keamanan. Paten dapat memainkan peran penting bagi startup saat mereka tumbuh, menemukan cara untuk melindungi inovasi mereka, dan menarik dukungan modal yang kritis.

Banyak perusahaan pemula memberi tahu kami tentang kerentanan khusus yang mereka hadapi di perusahaan kecil, dan tentang pentingnya mereka lampirkan untuk membangun portofolio IP mereka sendiri pada tahap awal ketika mereka berusaha untuk mengumpulkan modal dan membangun masa depan. Mitra kami di LOT Network telah mengembangkan program yang memenuhi kebutuhan ini. Microsoft dengan senang hati menyumbangkan 500 paten ke LOT Network, yang sekarang LOT akan berikan kepada startup yang memenuhi syarat sebagai bagian dari keanggotaan gratis.

Mempertimbangkan semua yang dilakukan bersama, kami percaya bahwa Microsoft menawarkan lingkungan yang paling lengkap dan komprehensif untuk menghilangkan risiko IP di cloud.

Ulasan pelanggan awal positif:

"Untuk perusahaan teknik internasional seperti thyssenkrupp, program Microsoft Azure IP Advantage adalah alat yang hebat untuk mengurangi risiko IP ketika bermigrasi ke Industry 4.0," kata Dr. Stephan Wolke, Kepala Kekayaan Intelektual (IP) dan layanan thyssenkrupp GmbH.

"Applied Brain Research sedang membangun sistem AI yang terinspirasi dari otak," kata Peter Suma, ketua dan co-CEO dari sebuah startup di pasar yang berkembang untuk komputasi neuromorfik. "Azure IP Advantage membantu kami fokus pada inovasi berikutnya dan menghabiskan lebih sedikit waktu untuk mengkhawatirkan risiko IP."

Tujuan kami adalah untuk menciptakan komunitas yang dapat fokus pada inovasi, daripada khawatir tentang melindungi inovasinya. Kami berharap langkah ini untuk memperluas program Azure IP Advantage akan bermanfaat dalam hal ini.
