嗨，怪胎！在阅读了很多关于比特币的文章之后，包括在我们的博客上，您可能会想到它们的替代方案-毕竟，所有这些都可以围绕一种加密货币进行吗？好吧，不仅您考虑过。然后有人采取行动，创建了自己的加密货币。一些创造它们来赚钱和成名，其他创造它们竞争比特币，也许绕开它们，而另一些创造它们只是为了娱乐。其中一些在世界范围内越来越受欢迎，而另一些则仍处于“他们自己”的圈子中。我们选择了其中15种最受欢迎​​的货币，并以其为例说明了替代货币（altcoins）与原始货币（比特币）之间的区别。



实际上，比特币成功的原因以及随后的所有加密货币都与之进行了精确比较，这是因为比特币首次出现在2009年。但是，它们并不是唯一的：BitMakler.com服务拥有590种不同的加密货币，包括具有各种加密算法的比特币，可能的硬币数量（有些没有排放限制），目的（也有这种加密货币，其中有些从中获利转到慈善基金）和相对比特币的成本。

LiteCoin



交易代码：LTC
创始人：Charles Lee
成立：2011
算法：Scrypt
LTC / BTC：0.0094
LTC / USD：5.3

LiteCoin由Charles Lee于2011年推出，以替代比特币，并有可能替代比特币。该货币在2013年变得越来越流行，并且总体上对加密货币的兴趣也在增加。

LiteCoin的主要区别在于另一种用于确认工作已完成的哈希算法。当在比特币网络上立即使用SHA256时，Scrypt将SHA256用作子例程。

LiteCoin的区块生成速度是比特币的4倍，约2.5分钟。这意味着加快了收款到钱包的速度（大约15分钟）。

这意味着以下情况-您可能已经猜到，硬币的数量也增加了4倍（8400万个，而比特币为21个）。因此，排放和回报率将与比特币相同。

42coin



交易代码：42
创始人：hendo420
成立时间：2014
算法：Scrypt
42 / BTC：4.9149
42 / USD：n / d

42coin是LiteCoin的替代货币，不同之处在于总共可以生成42个硬币。因此，相对于比特币，这样的成本，比特币的总数可以是21,000,000，前420个区块没有奖励，下一个内部拥有0,00042 42coin，然后是420个“空”区块，等等。

创作者宣告由于块生成的速度（42秒，谁会想到！）及其稀有性，这种货币将成为革命性的货币。

DogeCoin



交易代码：DOGE
创始人：Billy Markus
成立时间：2013
算法：Scrypt
DOGE / BTC：0.00000075
DOGE / USD：0.0002

2013年推出的Dogecoin具有完全复制的LiteCoin算法，但有微小的变化。它的象征是著名的与狗的模因。最初，可以生成1000亿个硬币，并且随机设置一个区块中的硬币数量。 2014年3月，货币的行为发生了变化。奖励已经确定，并且排放量是无限的。

开发人员想要制造加密货币，并摆脱比特币的“黑历史”。 DogeCoin社区已多次为慈善事业筹集资金。 2014年1月，他们从牙买加雪橇队筹集了50,000美元，而当年3月25日，用户又为NASCAR赛车手Josh Wise筹集了55,000美元。

ZetaCoin



交易代码：ZET
创始人：Giscard
成立：2013
算法：SHA-256
ZET / BTC：0.00001803
ZET / USD：n / d

ZetaCoin是比特币的类似物。每年发行的限量发行量为1.6亿枚zetakoins，每年通货膨胀为1,000,000枚硬币。这也许是ZetaCoin和比特币之间的主要区别。此外-比比特币交易更快，并且复杂性有所变化。一区块的奖励是1000 ZetaCoin，并且每80640区块（约一个月）就会减半，并且不会低于一。

NameCoin



交易代码：NMC
创始人：Namecoin.info创立
年份：2011
算法：SHA-256
NMC / BTC：0.00275
NMC / USD：0.5

NameCoin是比特币的直接模拟，并使用相同的SHA-256算法。可能与比特币同时进行挖掘（合并挖掘）。采矿可以（像比特币一样）2,100万个Namecoin。为了计算区块，使用标准的比特币挖矿软件，将其重定向到Namecoin工作的服务器和端口。

NameCoin不仅是一种替代加密货币，而且还是DNS的.bit域区域，其灵感来自去中心化货币的想法。由于您的浏览器访问的“地址簿”位于您的计算机上，因此位于.bit区域中的所有站点均不受任何状态的控制，并且很可能永远不会受到任何状态的控制。

PeerCoin



交易代码：PPC
创始人：Sunny King，Scott Nadal
成立年份：2012年
算法：SHA-256
PPC / BTC： 0.005PPC
/ USD：1.6

PeerCoin（PPCoin）在加密货币历史上首次开始使用POS（权益证明/股份证明）生成系统。因此，它可以抵抗PoW系统易受攻击的大多数攻击。使用SHA-256函数作为哈希算法。

协议0.5的新版本将支持离线创建硬币。在下一个版本的Peercoin中，用于创建硬币的特殊密钥将允许用户创建硬币，并使他们的消费密钥完全脱机。

：EMC

BlackCoin



Trade的标志：BLK
创始人：rat4成立
年份：2014
算法：Scrypt
BLK / BTC：0.00020932
BLK / USD：0.189999

BlackCoin基于“权益证明”（英语：Proof-of-Stake / PoS）原理，是一个开源系统。 BlackCoin由俄罗斯开发商Rat4（Pavel Vasin）创建，以证明拒绝“工作量证明”原理（英语工作量证明/ PoW）对于加密货币的稳定性和安全性并不关键。

BlackCoin多池结合了BlackCoin社区成员的硬件哈希功能来挖掘其他数字货币。然后将这些山寨币兑换成比特币，然后购买BlackCoin硬币换成比特币，以支付给参与者。 BlackCoin本身无法通过采矿来生成，因为它基于“权益证明”原理。

Dash 商标



：DASH
创始人：eduffield
成立：2014
算法：X11
DASH / BTC：0.01202997
DASH / USD：2.97585

DASH在过去的DarkCoin中是基于X11算法的密码系统。这是带有官方DarkSend事务混合器的系统，几乎完全消除了发件人跟踪。根据用户的要求，您不能使用DarkSend并直接进行交易。总共可以在系统中生成22000000 DASH。块生成时间为2.5分钟。

Darksend是一项功能，可让Dash用户在进行转账时保持完全私密。这是CoinJoin技术的改进和增强版本。

除了基本的CoinJoin概念之外，他们还进行了许多改进-例如去中心化，增加匿名性（由于采用了新的链匿名化方法），面额命名机制，背景预混合程序。

Feathercoin具有与Litecoin完全相同的算法和技术部分，但有所更改。羽毛硬币的排放限制为3.36亿个硬币。找到的区块的奖励是200个硬币。官方网站上有令人印象深刻的在线商店列表，它们接受Feathercoin作为付款方式。

Vertoin



交易代码：VTC
创始人：vertoe创立
年份：2014
算法：Scrypt-N
VTC / BTC：0,00062005
VTC / USD：0,035

Vertcoin是基于Scrypt-N算法的密码系统。它与Scrypt系统的区别在于它使用变量N，该变量N负责计算中使用的内存量。根据开发人员的说法，这将使这种加密货币复杂化并排除专用微芯片（ASIC）的出现。硬币总数为84,000,000，该区块花费的时间为2.5分钟。通过每个块重新计算复杂度。集体奖励为50个硬币。
