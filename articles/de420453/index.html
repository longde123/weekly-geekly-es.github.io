Skillbox Friday Webinare: Entwicklung und alles darüber



Am kommenden Freitag erscheint eine neue Auswahl kostenloser Webinare von Skillbox auf Habré. In den heutigen Videos geht es um Programmierung:

  • Entwicklung von PHP-Anwendungen.
  • Anwendungsentwicklung für iOS.
  • Wie werde ich ein Android-Entwickler?
  • Automatisierungstools für die mobile Entwicklung.
  • So starten Sie die Programmierung in Python.

Wir erinnern Sie daran: Für alle Leser von „Habr“ - ein Rabatt von 10.000 Rubel bei der Anmeldung für einen Skillbox-Kurs mit dem Promo-Code „Habr“.

So entwickeln Sie eine PHP-Anwendung



Dies ist das Webinar des führenden Badoo-Ingenieurs Mikhail Ovchinnikov. Er erklärt unerfahrenen Programmierern, wie man eine Anwendung in PHP entwickelt. Der Artikel beschreibt die Gründe für die Beliebtheit von PHP als Sprache für die Backend-Entwicklung, die Komplexität grundlegender Webanwendungen und die Voraussetzungen, um einen PHP-Programmierer zum Profi zu machen.

Online-Kurs "PHP-Entwickler von 0 bis PRO"

So entwickeln Sie eine Anwendung für iOS



Nikita Arkhipov, CEO von Anvics, spricht und zeigt, wie iOS-Apps entwickelt werden, und demonstriert die Funktionen von Swift und Xcode. Der zentrale Fall ist die unabhängige Erstellung einer Aufgabenanwendung: Arbeiten mit der Datenanzeige und Verarbeiten von Benutzeraktionen.

Wie werde ich ein Android-Entwickler?



Mikhail Ovchinnikov, ein führender Badoo-Programmierer, spricht darüber, wie man ein professioneller Android-Entwickler wird. Das Niveau der Fähigkeiten und Kenntnisse zum Erstellen Ihrer ersten mobilen Anwendung, Tools für die Android-Entwicklung und eine Geschichte darüber, wie ein angehender Programmierer ein Teamleiter in einem internationalen Unternehmen werden kann - das erwartet Sie in diesem Webinar.

Praktikum "Mobilentwickler von Grund auf neu"

Automatisierungstools für die mobile Entwicklung



Dmitry Shashlov, Mobile Development Manager von Agima, spricht über Automatisierungstools, einschließlich kontinuierlicher Integration, kontinuierlicher Bereitstellung, automatischer Tests und Tools für die Arbeit mit Code in einem Team: Gitflow, Pull-Request und Codeüberprüfung.

Praktischer Online-Kurs „Mobile Developer PRO“

So starten Sie die Programmierung in Python



Der Mentor des Skillbox Online-Programmierers, Vadim Shandrinov, spricht über das Programmieren in Python. Das Webinar zeigt die Möglichkeiten der Sprache auf und zeigt, wie man mit Funktionen in Python arbeitet. Der praktische Fall ist die Erstellung der ersten Webanwendung. Darüber hinaus befasst sich der Autor mit dem Thema der Nachfrage nach einem Python-Entwickler auf dem Arbeitsmarkt.

Praktikum "Python-Entwickler von Grund auf neu"

Die gesamte Auswahl ist universell und kann für jeden, der in der IT arbeitet, nützlich sein. Wenn Sie nicht in den in den Webinaren genannten Sprachen programmieren, hindert Sie nichts daran, mehr darüber zu erfahren. Und wenn Sie programmieren, ist es Zeit, Ihr Wissen aufzufrischen.
