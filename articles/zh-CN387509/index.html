小米在GearBest上的新功能

11月24日，小米举行了一次正式活动，随着新设备的有趣提示，新闻发布已经升温了数周。结果，该演示文稿显示了小米Redmi Note 3平板手机系列的继任者，以及Mi Pad 2 –这是该公司近两年来的第一台平板电脑。此外，不久前，该公司还宣布了小米逸DVR车载DVR。我们决定以一种材料收集所有这些设备，并讨论每种新产品的功能。

小米Redmi Note 3宣布发布后，技术博客中的设备立即与类似设备魅族M1 Metal形成对比。不仅因为这些设备进入相同的价格段并具有相似的定位（这些设备甚至具有采用MediaTek Helio X10芯片组形式的相同硬件平台），而且还因为这些公司在本地市场上的竞争不受限制。后者很可能成为了小米的一种加速器，小米并没有通过简单更新某些组件就开始复制去年的Redmi Note 2，而是发布了一种全新的设备，可以竞争。



智能手机获得了更新的设计，该设计以金属为主体材料为重点。新奇有三种颜色可供选择：银色，黑色和黄金（稍后将提供）。与其他金属设备一样，在外壳的上部和下部都有用于天线的塑料插件。



小米仍然引以为傲的是尺寸的保留和重量的轻微增加（164克，Redmi Note 2中为160克），同时将电池增加到4000 mAh（Redmi Note 2中为3000 mAh）。 在后面板上，除了具有5个镜头的13百万像素的摄像头，相位检测自动对焦和记录FullHD视频的功能外，还有指纹扫描仪，其响应速度为0.3秒。例如，事实证明，只需用食指在背面弄个圆形凹槽，就可以解锁智能手机。就像在魅族M1 Metal中一样，这似乎比屏幕下方扫描仪的位置更方便。









在后一种情况下，您需要同时使用两只手。顺便说一下，可以在新的Nexus 5X和Nexus 6P上看到后面板的指纹扫描仪，而且对于配备5.5-5.5英寸屏幕的设备，要想找到更符合人体工程学的解决方案似乎仍然很困难。在小米Redmi Note 3中，它稍微凹进了外壳中。因此，该设备中没有臭名昭著的鼓鼓部分：所有东西都极其简约和整洁。



如上所述，尺寸与上一代平板手机保持相似。关键原因是使用分辨率为1920x1080像素的类似5.5英寸显示器。同样在前面的是一个5百万像素的摄像头模块，能够以FullHD拍摄视频。





至于硬件，Redmi Note 3有两种版本，它们的内存大小各不相同：2/3 GB的RAM和16/32 GB的用于存储数据的闪存。这里值得一提的是，新产品的主要缺点是缺乏对存储卡的支持。这就是为什么最好选择更强大的修改-在这种情况下，价格差约为2000卢布。

小米Redmi Note 3可以使用两个microSIM标准的SIM卡。相应的隔间位于设备的左侧。音量调节器和电源/锁定键占据了另一侧。

传统上，对于小米智能手机而言，MIUI外壳充当软件平台。在这种情况下，第七个最新版本可与Android 5.1 Lollipop结合使用。
