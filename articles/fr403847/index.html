Total Commander: recherche dans le dossier actuel par défaut

Total Commander a un problème avec la recherche. Par défaut, cela se fait dans tous les sous-répertoires. Par exemple, vous avez un dossier Téléchargements dans lequel un tas de sous-répertoires et de fichiers. Vous savez que le fichier dont vous avez besoin se trouve dans le dossier lui-même, mais TC cherchera par défaut également dans tous les sous-répertoires, ce qui prend du temps. J'ai remarqué que, fondamentalement, je cherche quelque chose uniquement dans le dossier sélectionné. Et cela signifie qu'il est plus efficace de le faire par défaut. Le problème est qu'il n'y a pas une telle possibilité dans TC. Mais il y a une béquille!

Vous pouvez créer un soi-disant «modèle de recherche» et l'affecter à une combinaison de touches au lieu de celle habituelle (ou à toute autre que vous souhaitez).

1. Ouvrez la recherche (Alt + F7 par défaut)
2. Sélectionnez l'imbrication souhaitée et effacez la ligne «Rechercher un emplacement»
3. Onglet "Modèles de recherche"
4. Bouton «Enregistrer»
5. Il vaut mieux nommer le modèle pour que plus tard il soit clair de quoi il s'agit. Par exemple ThisDir. Nous fermons.





1. Allez maintenant dans Outils> Liste des commandes
2. Catégorie usercmd.ini
3. Nouveau. Vous pouvez l'appeler de manière significative. Je l'ai laissé tel quel, car la paresse n'a pas fait d'autres équipes non plus.
4. Dans le champ "Commande", cliquez sur la loupe. Pour une recherche rapide dans la recherche de charges de type "Masque" et sélectionnez la commande "Recherche stockée LOADSEARCH".
5. Dans le champ "Paramètres", spécifiez le nom de votre modèle (ThisDir). C'est tout aussi.





1. Maintenant, allez dans Configuration> Paramètres> Divers
2. Dans la section "Prédéfinir les raccourcis clavier", sélectionnez la combinaison souhaitée. Par exemple, Ctrl + F
3. Dans le champ "Équipe", l'équipe déjà affectée à cette combinaison peut apparaître. Vous changez simplement le vôtre. Cliquez sur la loupe et dans la fenêtre déjà familière, trouvez votre commande dans usercmd.ini. Cochez maintenant.



Maintenant, lorsque vous appuyez sur Ctrl + F, la recherche par défaut se produit uniquement dans le dossier sélectionné.

Une fois, j'ai trouvé cette méthode sur le forum wincmd (il semble ici ), mais j'ai décidé de l'organiser de manière plus compréhensible.
