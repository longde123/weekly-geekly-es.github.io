市场新手：Snom D385和D785 IP电话概述



今天，我想向您介绍德国供应商Snom的一对IP电话旗舰机型，我们在先前的出版物中已经谈到过。 它们是在公司成立20周年的前夕出现的，而D785已经在俄罗斯积极销售，并且D385即将上市。

Snom提供两条固定电话IP电话：D3xx和D7xx。 前者具有更现代的设计和“蹲式”机壳，后者为我们所有人提供了熟悉的设计，并且外壳以大角度放置，这是许多IP电话所特有的。 就功能而言，两条线是相同的。

在第300和700行中，与面对D745或D345的“中级”类别相比，D385和D785是最先进的机型。 他们不以大公司的董事，经理，经理和他们的助手的身份为耻。 但这并不意味着D385和D785专为领导层而设计。 这两款手机极大地方便了任何积极从事“社会生活”的员工的生活，例如秘书或项目协调员。

D785的倾斜角度是通过可调节的支架设置的，该支架具有两个固定的倾斜角度：46°和28°。

D385和D785配备4.3英寸高清分辨率彩色显示器，D385显示器突出于机壳上方（这是主要的外部差异之一），并且可以更改其倾斜度。



在如此大的显示屏上，使用呼叫列表，地址簿或浏览所有菜单选项以及监视下属和同事的当前雇用状态非常容易且方便。







两种型号均具有消息，免打扰，目录，菜单，转移，保留，会议，已拨，静音，免提电话，耳机，音​​量和导航等物理键。





让我们简要地浏览一下主键。

  • 1-电话簿 。 屏幕上的信息显示非常方便，周到。 在这里，您可以输入有关联系人的所有必要信息：电话号码，电子邮件，公司名称，职位，昵称等等。
  • 2-通话记录 。 您可以查看来电，去电和未接来电的历史记录。 详细信息按钮显示其他信息。
  • 3-转发 。 几乎是最重要的按钮。 您可以将呼叫转移设置为任意号码，并通过单击此按钮禁用当前的呼叫转移。
  • 4-请勿打扰（“砖头”） 。 在这种模式下，没有人能够与您联系，就像您一直在忙一样。 通话信息不会丢失，电话上的红色LED亮起表示未接来电，数据已存储在未接来电日志中。 如果您厌倦了不断的通话并且想专心致志，那么这是一个很好的解决方案！
  • 5和6-确认（“滴答”）和取消（“交叉”） 。 Enter和Escape的奇特类似物。
  • 7-电话簿 。 完全类似于第一个按钮。
  • 8-设置（“齿轮”） 。 最重要的按钮之一。 允许您配置电话的所有功能，从音量到网络路由。

如果需要在日常工作中使用大量功能键，请注意D385有12个物理键和48个可分配功能，D785有6个物理键和24个可分配功能。





功能键很少？ 您可以将扩展模块D3（连接到D385）或D7（连接到D785）和18个物理可分配键。

将D385和D785与其他大多数Snom手机区分开来的功能之一就是所谓的“无纸化解决方案”。 这是安装的附加LCD屏幕，而不是功能键旁边的纸。 首先，您无需手动编写任何内容；屏幕上的铭文看起来整洁且一致； 其次，只需在屏幕上“滚动”即可通过选择所需的按钮为物理按钮分配更多的值。 至少每天都要更改键的分配，而不用取出纸，也无需重新签名。 任何用户都可以执行此操作，为此，您甚至无需进入网络界面：只需按住某些键，即可打开设置菜单。 此外，用于快速访问功能的按钮在另一个屏幕上已经开箱即用：

  • “电话会议”（Conference） 。
  • “智能呼叫转移”（Smart transfer） 。 您可以将当前呼叫重定向到某个会议模块或任何其他号码，无论是内部呼叫还是外部呼叫。 例如，在您的智能手机上。
  • “静音模式”（Silent mode） 。 整个显示变得极为可视。 与“请勿打扰”（DND）模式不同，当您打开它时将无法联系到您，“静音模式”只是关闭声音警报。 手机将接听所有电话并将其显示在显示屏上。 如果您需要沉默一会儿，或者只想应答大量呼叫中某个人的呼叫，这将非常方便。 请注意，如果在联系人列表中为某人分配了VIP身份，则当该人呼叫时，即使在“静音模式”下，电话也会打开声音通知。
  • 通话记录（已拨） 。 两部电话都没有通常的“重拨”按钮。 其功能由“已拨”按钮（“通话记录”）执行。 首次按下该按钮时，会立即显示所有呼叫的联系人/电话号码的列表（ 在许多其他制造商的电话上，您会进入呼叫日志，然后需要转到“发件箱”部分或在常规日志中查找所需的电话号码 ），以及在这里，您可以从列表中选择所需的电话号码/联系人； 或仅再次按Dialed（拨号）按钮，电话便会回拨到上次拨打的号码。

根据我们的经验，在其他制造商的电话中，要使用这些功能，您需要执行的动作比按单个按钮要多。





两种型号在外壳的右上角都有一个LED，当未接来电，语音消息到达时或只是在听筒摘机时，该指示灯会点亮。



功能特点


由于D385和D785是阵容中的顶级型号，它们具有IP电话可能需要的所有可能的功能：快速拨号，URL寻址，自动重拨，呼叫方ID，呼叫等待，黑名单，自动应答，呼叫保持，音乐等待时，呼叫转移，三方电话会议，内部线路监控，多播寻呼，免打扰（免打扰）模式，键盘锁定，1000人的联系人列表，您还可以从中区分朋友。 此外，您可以启用本地订阅费会计。

您可以根据需要在主屏幕上标记图像，设置背景，图标和菜单颜色。

两部电话最多可同时处理12个电话。 此外，D385和D785可以从物联网变成设备控制面板，从外部设备接收数据并控制应用程序。 也就是说，使用这些电话，您可以执行通常不与电话相关的操作：打开灯，录制对话，使用电子锁打开门，使用对讲机，访问语音信箱等等，甚至还可以用作电视的遥控器。 您可以在此处下载软件工具包： https : //snom.io/ 。


至于使用显示屏上的基于Web的软件界面，D385和D785具有“刷新”的成熟XML浏览器。 支持许多支持功能，例如转储或跟踪。

您可以使用USB和RJ9端口或内置的蓝牙模块将耳机，DECT，蓝牙和Wi-Fi加密狗连接到两个型号。 这些电话配备了双端口千兆交换机，并支持SIP（RFC3261），DHCP，NTP，HTTP / HTTPS / TFTP，LDAP（目录）和IPv4 / IPv6协议。 两种型号还与助听器-HAC兼容。



技术特点


以下所有内容均适用于D785和D385。 两种模型的技术填充几乎相同，不同之处在本节末尾进行描述。

D385和D785不是机械的，而是具有电子的结束按钮，它更加可靠和耐用。

由于使用了数字信号处理器（DSP）和编解码器G.71​​1（A律，μ律），G.722（宽带），G.726，G.729AB和GSM 6.10（FR），因此可以实现很高的声音质量。 这些电话配备了舒适噪声发生器（CNG）和语音活动检测 （VAD）机制。

对于数据传输，使用了具有输出到RJ45的两端口单千兆位以太网交换机IEEE 802.3。 支持的网络协议是SIP（RFC3261），DHCP，NTP，HTTP / HTTPS / TFTP以及LDAP（目录）和双堆栈IPv4 / IPv6。

有一个USB 2.0主机端口，类型A，高速/全速/低速。 两种型号都可以通过电源供电，也可以使用PoE技术供电 。









使用预先安装的单个SHA-2证书通过802.1X和EAPOL身份验证，通过TLS进行传输层保护，SRTP音频加密（RFC3711），通过TLS使用SIP进行信号加密，使用HTTPS服务器/客户端，两级接口（用于用户和管理员），支持VPN，VLAN（IEEE 802.1Q）和LLDP-MED，RTCP-XR。

剩下的就是添加有关通过HTTP / HTTPS / TFTP自动下载设置，使用DHCP 66/67参数，通过TR-069 / TR-111进行远程控制以及我们自己的Snom重定向和自动配置服务的内容，我们将在以后的评论中进行讨论。

我喜欢大型显示器，可以方便地显示信息。 此外，更改字体的颜色，您可以进一步关注一些信息。 例如，以某种颜色突出显示正在休假且无法接听电话的订户。

尽管从硬件的角度来看，我们没有什么可抱怨的，但是在测试过程中，我们在电话的本地化和接口方面遇到了一些困难。 例如，某些单词未翻译，例如，在电话簿中，项目“添加联系人”和“排序”以英语显示，而“编辑”和“删除”-俄语。 我们报告了所有这些支持，并承诺修复固件。

另外，上面提到了“取消”按钮（“ Cross”），以取消当前操作。 它会立即取消（这不是错误，而是一项功能）：如果您在此之前输入了重要信息而没有保存，则单击此按钮将终止数据-您将不得不重新输入所有内容。 您可以使用当前屏幕上显示的其他导航按钮返回上一级菜单。

在这两种型号中，用户都可以使用方便，简单，灵活且相当丰富的界面来使用和配置电话（顺便说一下，这两款电话都具有900多种功能）。 如果某些客户一次购买300部电话，Snom可以定制外壳，菜单，按键，固件的设计。
