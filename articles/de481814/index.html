FunCode Java / Kotlin Challenge Backend Wettbewerb

FunCorp veranstaltet einen Wettbewerb für Backend-Entwickler in der Sprache Java / Kotlin mit einem Preispool von 550.000 Rubel.

Jeder kann mitmachen. Autoren der besten Werke erhalten Geldpreise und Geschenke von FunCorp sowie die Möglichkeit, sich unserem Team anzuschließen.



Was ist zu tun?


Schreiben Sie eine Anwendung in Java und / oder Kotlin, die mit maximaler Geschwindigkeit die beliebtesten Unterhaltungsinhalte (Bilder und kurze Videos) ermittelt, und geben Sie sie als Band mit Metadaten zurück, sortiert nach Veröffentlichungszeit von neueren zu älteren.

Was werden wir bewerten?


  • Architektonische Lösung
  • Codequalität
  • Leistung
  • Ressourcenverbrauch
  • Originelle und mutige technische Lösungen

Angenommene Aufgaben werden bis zum 19. Januar 23:59 angenommen.

Die Ergebnisse und Gewinner werden bis spätestens 17. Februar bekannt gegeben.

1. Platz - 300.000 p.
2. Platz - 150.000 p.
3. Platz - 100.000 p.

Über uns


FunCorp entwickelt unterhaltsame UGC-Dienste und -Spiele.

Das Hauptprodukt ist iFunny, die zweitbeliebteste Unterhaltungsanwendung in den USA (mehr als 50 Millionen Installationen).

Was benutzen wir?


  • Microservice-Architektur
  • Java / Kotlin
  • Dropwizard, Spring, Ratpack, Ktor
  • MongoDB und Redis zur Datenspeicherung
  • Docker für die Bereitstellung
