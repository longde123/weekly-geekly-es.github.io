Cara melindungi kata sandi Anda pada tahun 2019

Setiap tahun dunia semakin terhubung dengan struktur Internet, semakin banyak layanan ditemukan di jaringan, kehidupan seseorang menjadi tergantung pada World Wide Web.

Sebagian besar data kami disimpan dan digunakan di sana, dan ada beberapa yang lebih baik untuk dijauhkan dari mata jahat.

Terkadang mata ini pergi ke tempat yang seharusnya tidak memiliki akses sama sekali!

Namun, inilah cara melindungi data Anda dari "barbar" yang andal sekarang akan saya sampaikan.

Bagaimana mereka mencuri mereka?


Untuk memahami prinsip perlindungan, Anda harus memahami prinsip serangan: untuk mengakses data di jaringan, Anda paling sering membutuhkan kata sandi. Inilah tepatnya kerentanan sistem komputer mana pun.

Saat memasukkan kata sandi di browser, kita sering diminta untuk menyimpannya:



Setelah Anda mengklik tombol "Simpan", kata sandi Anda direkam dalam file "Dienkripsi", kunci yang letaknya sangat dekat dengannya, di file lain.

Saat ini, cara populer untuk mencuri kata sandi menggunakan program virus - "Stiller".

Seorang penyerang, dengan dalih apa pun (atau tanpa dalih), meyakinkan korban untuk meluncurkan file jahat di PC-nya yang menyalin file dengan kata sandi Anda, segera mendekripsi menggunakan kunci terdekat, dan mengirimkannya ke pembuat program ini.

Sulit untuk sepenuhnya melindungi file dari pengiriman, namun kami dapat mengenkripsi file sehingga virus tidak dapat mendekripsi sebelum mengirim:



Khusus untuk ini diciptakan- " Kata Sandi Utama ".

Selain yang biasa, ia menambahkan kunci enkripsi dalam bentuk kata sandi tambahan. Dia adalah kunci kedua Anda. Biarkan penyandi program jahat menerima file, tetapi tidak akan dapat melihat kata sandi yang diidamkan, karena virus, tidak mengetahui kunci kedua, "Kata Sandi Utama", tidak dapat mendekripsi file sepenuhnya.

Saat ini, fitur ini didukung di hampir semua browser populer.
Mudah digunakan, dan hasilnya sangat menakjubkan!

Jangan abaikan keamanan, enkripsi data Anda dua kali! Dan lebih baik, jangan menyimpan kata sandi sama sekali: memori adalah perlindungan terbaik. Tidak ada yang akan naik ke sana!
