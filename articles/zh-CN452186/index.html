最稀有和最昂贵的编程语言

人们普遍认为，Rust，Erlang，Dart等编程语言以及其他一些编程语言在IT世界中是最罕见的。 由于我为公司选择IT专家，并不断与IT专家和雇主联系，因此我决定进行个人研究，以了解这是否正确。 该信息与俄罗斯IT市场有关。

资料收集


为了收集信息，我研究了您需要语言技能的需求中的空缺数量，以及具有该技能的简历数量。 我使用Amazing Hiring服务在HeadHunter的Linkedin上收集了数据。 我也有向我的代理机构提出申请的个人统计信息。

总的来说，我的研究涉及八种语言。

铁锈


世界统计：根据2018年Stackoverflow的统计数据，Rust在开发人员中最喜欢的语言列表中排名第一（连续第三年），在薪水方面排名第六（在最昂贵的语言中排名第六）（每年69,000美元）。
尽管该语言在世界上非常流行，但在俄罗斯，它仍然是最稀有的编程语言之一。

在关键技能方面，Rust在Headhunter上遇到了319位专家，在Linkedin上遇到了360位专家。 但是，只有24位开发人员将自己定位为Rust开发人员。 秘密地相信俄罗斯只有两家公司在Rust上写字。 Headhunter上的32家公司和Linkedin上的17家公司为Rust提供了工作。

我的代理商定期收到Rust开发人员的工作申请。 但是，几乎没有专家，我已经给我留下了与该国所有Rust专家熟悉的印象。 因此，在使用Rust语言的情况下，许多对职位空缺感兴趣的候选人都将该语言作为ToR的一部分。

Erlang


根据相同的统计数据， Stackoverflow Erlang与Rust相比差距不大，并且也获得了各种评级。 Erlang在开发人员最喜欢的语言列表中排在第二十一位，Erlang在薪水方面紧随Rust，排名第七（每年67,000美元）。

猎头公司为拥有Erlang知识的开发人员提供67个工作机会。 在Linkedin上-38.如果我们谈论简历的数量-猎头公司上只有55名开发人员直接了解Erlang作为一种主要语言（标题中已标明），而38位专家Erlang在Linkedin上的帖子标题中遇到了。

此外，由于有更多的人，而且薪水较低，因此有一种趋势是代替Erlang开发人员雇用拥有已开发的Google Go或Golang的人。 但是，我个人的观点（基于我机构的数据）是Go不会取代Erlang，因为对于真正高负载和复杂的项目，Erlang是必不可少的语言。

哈克斯


主要用于游戏开发。 几乎没有职位空缺（实际上是“猎头公司”的职位空缺）。 在Linkedin上，只有两家公司需要这种语言的知识。 如果我们讨论该提案-几乎有200位开发人员指出Linkedin对这种语言的知识，那么109则指向猎头公司（Headhunter），其中有10个人将Haxe知识作为简历的名称。 事实证明，Haxe编程语言在俄罗斯市场不是很流行。 供应超过需求。

飞镖


由Google发明。 语言在市场上越来越流行。 在Headhunter 10和Linkedin-8上有职位空缺，但雇主在关键技能列表中不需要此语言。 主要条件是具有丰富的Javascript背景和解决问题的有效方法。

熟悉编程语言的开发人员数量为275，但是同样只有11个人认为Dart是他们的主要技能。 在LinkedIn上，有124个人以一种或多种方式在简历中提到了该语言。

根据我的代理机构的个人经验和统计数据，大型IT公司已经在使用这种语言。 这表明它将很快从稀有编程语言列表中排除。 顺便说一句，说Dart语言的专家在市场上“昂贵”。

F＃


一种相当罕见的编程语言。 由Microsoft开发。 在俄罗斯，只有少数几家公司（HH上有12家，Linkedin上有7家）要求使用F＃程序员。 在其他情况下，语言知识是可选的。 顺便说一下，了解F＃的开发人员的数量正在逐渐增加。 该语言甚至出现在最近的Stackoverflow排名中。 在开发人员中最喜欢的语言列表中，他名列第九，就薪水而言，他是第一名（每年74,000美元）。

如果我们谈论已发布的简历数量，猎头公司有253个，但是很少有专家认为F＃是他们的主要语言。 只有三个人将F＃知识带到了简历的标题。 在Linkedin上，情况类似：272个开发人员在他们的投资组合中提到了F＃，其中只有6个F＃被指出。

统计信息如下：


猎头公司的职位空缺总数为122，而Linkedin上的职位空缺总数为72。 在所研究的语言中，最受欢迎的语言是Erlang。 超过50％的公司要求了解Erlang。 最不常见的语言是Haxe。 寻找分别对猎头公司和Linkedin拥有1％和3％公司知识的专业人士。




从已发布的简历数量来看，情况几乎相似。 在发布的1,644个“猎头公司”简历中，超过40％（688）的人是Erlang，最少的（7％）是具有Haxe开发技能的人。 从Linkedin获得的数据略有不同。 所有简历中最少的是由拥有Dart的家伙出版的。 在1894个投资组合中，只有124个与Dart开发有关。





Opa，Fantom，Zimbu


我决定将这三种语言合并在一个段落中是出于一个简单的原因-确实是最稀有的语言。 没有空缺，几乎没有简历。 您可以依靠那些在其技能中指出了这些语言中任何一种的开发人员。

由于这些语言没有包含在年度Stackoverflow年度报告中，也没有出现在空缺中，因此我将就这些语言的含义写几句话。

Opa是一种网络编程语言，可立即尝试替换HTML，CSS，JavaScript，PHP。 2011年设计。 Opa是免费的，目前仅在64位Linux和Mac OS X平台上可用。

Fantom是一种通用语言，可编译为Java Runtime Environment，JavaScript和.NET Common Language Runtime。 2005年设计。

Zimbu是一种独特的特定语言，可用于开发几乎所有东西：从GUI应用程序到OS内核。 目前，它被认为是一种实验性语言，并非其所有功能都得到开发。

除了编程语言，我还加入了网络安全专家的列表和位置。 与简历的数量相比，职位空缺的数量很少（约20个）。 事实证明，供大于求（如Haxe的情况），这对于IT部门而言是非典型的。 信息安全专家的薪水很低。 例如，在圣彼得堡，经验丰富的网络安全专家可获得80万至10万卢布的报酬。

我的小研究表明，掌握的“顶级”语言是：Rust，Erlang，Dart-有需求，薪水很高。 不太流行的语言是Haxe，Opa，Fantom，Zimbu。 F＃在国外很流行，俄罗斯的IT语言市场尚未被占领。
