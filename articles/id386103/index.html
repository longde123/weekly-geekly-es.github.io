Microsoft menurunkan batasan pada OneDrive, dan untuk Rusia menaikkan harga untuk perangkat lunak dan langganan

Beberapa menganggap kurangnya batasan penyimpanan secara harfiah


Microsoft menurunkan ruang penyimpanannya di OneDrive untuk tarif gratis dan pengguna berlangganan. Dua opsi untuk membeli ruang di OneDrive menghilang. Juga hari ini diketahui bahwa Microsoft akan menaikkan harga produk-produknya untuk pengguna dari Rusia.

OneDrive adalah jawaban Microsoft untuk penyimpanan cloud seperti Dropbox. Solusi serupa sekarang ditawarkan oleh banyak orang. Ini adalah raksasa transnasional Google dan Apple, serta perusahaan kecil yang spesialisasi hanya penyimpanan data cloud, dan bahkan Yandex domestik dan Mail.ru. Skema penyimpanan cloud kira-kira sama. Pengguna menginstal program klien yang menyinkronkan data dari folder di komputer ke cloud. Setelah itu, file disimpan dengan aman di cloud dan tidak terpengaruh oleh kegagalan drive komputer pengguna. Jika diinginkan, file dapat dibagikan dan dipompa melalui HTTP. Sisa layanan dibedakan oleh serangkaian fungsi (enkripsi, dukungan untuk sistem operasi), program rujukan, dan harga langganan.

Seperti pada layanan cloud lainnya, ruang kecil dari tarif gratis OneDrive dibeli dengan berlangganan paket berbayar. Microsoft cloud memiliki kekhasan: tempat tersebut dapat ditingkatkan dengan membeli berlangganan paket perangkat lunak Office 365. Setahun yang lalu, pengguna Office 365 terkejut oleh berita bahwa mereka menghapus batas ruang dalam penyimpanan cloud OneDrive. Sebelum ini, langganan menyediakan 1 TB ruang tambahan.

2 November, Microsoft menolakdari praktik ini dan mengembalikan batas lama 1 TB. Perusahaan didorong oleh langkah-langkah drastis karena beberapa memanfaatkan peluang ini. Sejumlah kecil mengambil tidak adanya batas secara harfiah: pengguna mulai membongkar cadangan dari beberapa komputer, koleksi film, dan salinan disk video yang tidak terkompresi. Beberapa dengan antusias melampaui 75 TB, yaitu 14 ribu kali rata-rata ruang yang dihuni. Oleh karena itu, pengguna langganan Office, keluarga, dan siswa Office 365 memiliki batas 1 TB. Microsoft berjanji untuk mengembalikan uang jika pelanggan percaya bahwa Office 365 tidak lagi memenuhi kebutuhannya.

Di awal tahun depan, beberapa perubahan lagi akan terjadi. Saat ini, ruang OneDrive dapat dibeli dalam paket 100 dan 200 GB. Mereka akan menyingkirkan mereka demi paket 50 GB untuk $ 1,99 / bulan. Harganya dua kali lebih tinggi daripada di Apple iCloud. Google Drive menawarkan 100 GB untuk uang yang sama. Ruang kosong di OneDrive dipotong dari 15 GB menjadi 5. Yang terakhir berlaku untuk semua orang - baik pengguna baru dan lama. Kamera menolak bonus 15 GB untuk sinkronisasi otomatis. Microsoft berjanji bahwa pengguna OneDrive gratis yang terpengaruh (yaitu, mereka yang memiliki lebih dari 5 GB file di akun mereka) dapat mendapatkan langganan Office 365 gratis satu tahun dengan terabyte ruang dan pembaruan otomatis yang dapat dimatikan jika Anda mengkonfirmasi kartu bank Anda.

Ruang pada akun Anda akan berkurang, tetapi file tidak akan segera dihapus. Jika pelanggan Office 365 melampaui batas baru, ia akan diberitahu tentang hal itu, tetapi akan diizinkan untuk menggunakan ruang yang meningkat selama setidaknya satu tahun. (Hitung mundur mulai 2 November 2015.) Kemudian, selama enam bulan, Anda dapat mengunduh file dalam mode hanya-baca tanpa kemampuan untuk mengunggah yang baru. Melampaui periode waktu ini berarti memblokir kemampuan untuk mengunduh file - file hanya dapat dihapus untuk kembali ke batas yang diizinkan yaitu 1 TB. Atau Anda dapat membeli lebih banyak ruang untuk meningkatkan batas penyimpanan. Setelah hanya satu tahun diblokir, Microsoft dapat memutuskan untuk menghapus semua file. Secara total, pelanggan Office 365 memiliki setidaknya 2,5 tahun untuk dipikirkan. Dengan tarif gratis, semuanya lebih ketat: akses 5 GB berarti 90 hari untuk menyelesaikan masalah.Setelah itu, pengguna akan memiliki akses ke file dalam mode baca selama 9 bulan. Kemudian, tahun Anda hanya dapat menghapus file (atau membeli ruang untuk mengunduhnya). Hanya setelah total 2 tahun ini data akan dihapus secara permanen.

Bahkan dengan perubahan ini, harga OneDrive tetap sangat menarik: Anda bisa mendapatkan ruang terabyte dengan langganan Office 365 seharga $ 6,99 / bulan. Google dan Apple dapat menawarkan terabyte hanya dengan $ 10 dan tanpa paket perangkat lunak perkantoran. Untuk harga ini, Microsoft siap memberikan langganan rumah ke Office 365, yang memberikan semua manfaat langganan pribadi ke lima orang sekaligus (hingga 5 TB).

Di Rusia, harga Office 365 sekarang hampir dua kali lebih rendah, masing-masing 229 ₽ / bulan dan 286 ₽ / bulan. Namun mulai tahun depan, jumlah ini akan bertambah. Hari ini menjadi dikenalbahwa Microsoft akan menaikkan harga untuk produk dan langganannya ke Rusia. Mulai 1 Januari tahun depan, harga untuk perangkat lunak dan produk cloud akan meningkat sebesar 19-25%. Pada bulan Februari, harga naik 15-30% karena devaluasi rubel. Menurut mitra dan pelanggan Microsoft Rusia, kali ini alasannya sama.

Perubahan tidak akan memengaruhi konsol game Xbox, Dynamics CRM, dan versi kotak Windows. Untuk pengguna biasa, kenaikan harga akan menjadi minimum 19%. Pelanggan komersial akan dipaksa untuk membayar mulai dari 19% lebih untuk perangkat lunak dan hingga 25% lebih untuk solusi cloud.
