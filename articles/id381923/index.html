Paten Model Utilitas: Langkah

Jika Anda datang dengan perangkat yang unik, Anda dapat melindunginya dengan mendapatkan paten untuk model utilitas. Anda juga dapat mencoba mendapatkan paten untuk suatu penemuan, tetapi kami akan berbicara tentang penemuan tersebut di publikasi lain, jika ada minat.

Saya sering merekomendasikan pengajuan dua aplikasi secara bersamaan untuk model utilitas dan penemuan.

Dari pengalaman kerja saya, saya dapat mengatakan bahwa paten untuk model utilitas lebih mudah diperoleh dan, biasanya, lebih cepat.

Sebelumnya, yaitu, hingga 1 Oktober 2014, Kantor Paten Federasi Rusia (Rospatent) tidak melakukan pemeriksaan sehubungan dengan model utilitas. Ini berarti Anda dapat bertemu dua perangkat yang dipatenkan identik.

Rospatent saat ini sedang melakukan pemeriksaan terkait dengan model utilitas. Karenanya, situasi ketika akan ada dua paten untuk perangkat yang sama dikecualikan. Namun demikian, dalam praktiknya ini dimungkinkan, karena pemeriksaan dilakukan oleh seorang ahli yang dapat melewatkan paten untuk perangkat yang serupa.

Untuk mendapatkan paten, Anda harus menyelesaikan serangkaian tindakan, yang akan saya daftarkan di bawah ini.

1. Identifikasi perangkat untuk dipatenkan


Saya menyoroti tahap ini, karena penting untuk mengidentifikasi dengan jelas perangkat yang ingin Anda lindungi.

Satu perangkat dapat memiliki beberapa perangkat terpisah, yang juga dapat dipatenkan. Mungkin lebih disarankan untuk mematenkan beberapa bagian kecil tapi signifikan dari solusi teknis.

Idealnya, Anda harus memiliki strategi paten yang menjawab pertanyaan "apa", "kapan", "di mana" untuk mematenkan.

2. Pencarian paten


Setelah mengidentifikasi perangkat tertentu, Anda harus melakukan pencarian paten.

Diperlukan pencarian paten:

- untuk memeriksa kebaruan perangkat.

Perangkat Anda mungkin sudah dipatenkan oleh orang lain, atau mungkin sudah dikenal sejak lama.

Perangkat hanya dapat dipatenkan sebagai model utilitas jika masih baru . Artinya, jika perangkat tidak diketahui dari informasi yang tersedia untuk umum di dunia sebelum tanggal pengajuan aplikasi untuk model utilitas.

- untuk menentukan analog terdekat dari perangkat Anda untuk membuat aplikasi untuk model utilitas.

- untuk memahami tren dalam pengembangan teknologi.

Seringkali, pencarian paten dilakukan untuk melihat paten pesaing dan menentukan bagaimana pesaing berkembang.

3. Persiapan dan pengajuan aplikasi untuk model utilitas


Untuk mengkompilasi aplikasi, saya sarankan membaca Peraturan Administratif Rospatent pada model utilitas .

Jika akan ada minat pada artikel saya, maka saya akan menulis tentang menyusun aplikasi untuk model utilitas.

4. Terima pemberitahuan aplikasi dengan tanggal prioritas dan nomor aplikasi


Tanggal prioritas, sebagai suatu peraturan, bertepatan dengan tanggal pengajuan aplikasi.

Prioritas adalah penting, karena informasi yang telah tersedia untuk umum sebelum tanggal prioritas akan "mencemarkan" kebaruan perangkat Anda.

5. Menunggu akhir pemeriksaan Rospatent


Rospatent dapat meminta materi tambahan yang dibutuhkan, atau mengajukan pertanyaan klarifikasi yang akan muncul sebagai bagian dari pemeriksaan.

Rospatent juga dapat membuat "penolakan awal" jika itu menentukan bahwa perangkat tersebut tidak baru dan / atau tidak dapat diterapkan secara industri.

Perangkat ini tidak dapat diterapkan secara industri jika tidak dapat digunakan di industri, pertanian, layanan kesehatan, sektor ekonomi lainnya atau di bidang sosial.

Misalnya, desain mesin gerak abadi tidak dapat diterapkan secara industri. Meskipun Anda akan terkejut betapa banyak aplikasi untuk desain mesin gerak abadi diajukan.

Mengenai "penolakan awal" ini, seseorang harus mengajukan argumennya, karena seringkali para ahli tidak menyelidiki secara mendalam dan menarik kesimpulan yang tidak dapat dibenarkan. Anda dapat meyakinkan seorang ahli dengan argumen Anda.

6. Memperoleh keputusan untuk memberikan paten atau menolak untuk memberikan


Jika Anda telah menerima keputusan untuk memberikan paten, Anda harus membayar biaya untuk memberikan paten, serta biaya untuk mempertahankan paten yang valid untuk tahun pertama validitas, mengirim tanda terima pembayaran biaya dengan aplikasi untuk memberikan paten dan mempertahankan validitas untuk Rospatent.

Jika Anda telah menerima keputusan untuk menolak memberikan paten untuk model utilitas, maka Anda dapat mengajukan banding atas keputusan ini untuk menolak, tetapi ini sudah ada di artikel lain.

7. Memperoleh paten untuk model utilitas


Harap dicatat bahwa paten model utilitas harus dipertahankan setiap tahun dengan membayar biaya perawatan dan memberi tahu Rospaten tentang pembayaran.

Ini penting, karena dalam praktiknya terdapat kasus di mana perusahaan memiliki beberapa paten utama yang kedaluwarsa karena tidak dibayarkannya biaya untuk mempertahankan paten yang berlaku. Perusahaan ini tidak dapat membantu lagi.

Perkiraan jangka waktu untuk mendapatkan paten untuk model utilitas dari tanggal pengajuan aplikasi adalah 7-10 bulan. Tanggal dapat ditingkatkan, karena tidak diatur.

Jika Anda memiliki pertanyaan dan saran tentang topik yang menarik minat Anda, saya akan berterima kasih.
