Unduhan berurutan UTorrent dalam 2 klik

Halo, Habr!

Setelah membaca publikasi "uTorrent berjaga-jaga melihat online" , saya terkejut bahwa cara belum dipublikasikan untuk memungkinkan kemampuan mengunduh file secara berurutan tanpa menggunakan perangkat lunak tambahan seperti Editor Kode BEn , tetapi sederhana dan cepat - hanya dalam dua klik.

Jadi:

Kami mengaktifkan unduhan berurutan dari bagian file dan unduhan berurutan file dari daftar torrent.

1. Unduh uTorrent.
2. Jepit kombinasi tombol [Shift] + [F2], jangan lepaskan tombol .
3. Klik dengan tombol [Shift] + [F2] ditekan: Opsi -> Preferensi -> Tingkat Lanjut.
Kami melihat pengaturan uTorrent dengan parameter bt.afterential_download dan bt.afterential_files sudah terbuka untuk perubahan



4. Ubah nilai parameter pertama menjadi true - dan dapatkan unduhan yang konsisten dari bagian-bagian file - ini adalah persis apa yang Anda butuhkan untuk menonton film saat sedang memuat. Jika Anda suka - ubah nilai parameter kedua menjadi true juga - dan dapatkan unduhan berurutan file dalam daftar torrent (dan ini berguna untuk mengunduh, misalnya seri - seri akan diunduh secara berurutan, mulai dari yang pertama.

Voila!

Nikmati tontonan Anda dan semua itu.

Diuji pada versi uTorrent untuk Windows 3.4.2.
Apakah itu berfungsi di bawah Unix - Anda dipersilakan untuk memberi tahu saya di komentar.
