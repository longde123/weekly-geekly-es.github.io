时间囊：HP iPaq h1940“开箱即用”。 什么是2000年代的PDA



上个周末我在格拉纳达的跳蚤市场上又买了一个小玩意。 他成为PDA HP iPaq h1940。 之所以选择这种特定的设备，是因为PDA装在一个装有所有文档的盒子里，甚至还有一个装有附加软件的磁盘上。 我上次写的时候 ，这种手持设备的价格是10欧元，就像一台笔记本电脑一样。

盒子里有什么








包装盒中是设备本身，电池，附加连接器，USB电缆，文档和软件光盘。 在磁盘上-MS ActiveSync 3.7，Pocket Outlook以及PDA的其他几个应用程序。



设计方案


该设备看起来非常现代和时尚。 它很适合手；牛仔裤或夹克可以毫无问题地进入口袋。 完美按下了按钮-时间完全不影响它们。

显示器也像新的一样-它不会褪色，并且不会失去亮度（我将其与其他PDA进行了比较，并对新的外观有自己的记忆）。 过去的几年完全没有影响此案-塑料没有降解，贴纸和铭文与以前相同。



这款掌上电脑的体积比许多其他机型都要小，例如， 我几年前写的HP iPAQ rx3715。


HP iPaq h1940，iPAQ rx3715，Acer 50n Premium

它明显更薄。 很有可能，PDA变得更加优雅，因此可以轻松地放在口袋中携带。

表壳-塑料，彩色-金属。

按钮是iPaq CRC的常用按钮-这是用于启动组织器，调用联系人，任务管理器和邮件Messenger的四个按钮，以及带有操作确认按钮的操纵杆。 上部有一个录音机控制按钮，一个电源按钮。

技术指标


  • 操作系统：MS Windows Mobile 2003 Professional
  • 处理器：Samsung S3C2410 266 MHz
  • 内存：64 MB
  • 对角线：3.5
  • 分辨率：320 x 240（QVGA）
  • 色彩再现：6.5万种颜色
  • 宽度：70毫米
  • 高度：113毫米
  • 深度：19毫米
  • 重量：124克
  • 电池：锂离子电池900 mAh

该PDA具有安全数字（SD）/多媒体卡（MMC）扩展插槽。 此外，您可以使用USB电缆处理存储在设备上的信息。 换句话说，PDA可以作为USB闪存驱动器连接。

这是第一台带有Windows Mobile 2003的HP掌上电脑，它是Windows CE之后的“ pocket OS”的下一个版本。 例如，它改进了Pocket Internet Explorer，它可以与GIF一起使用并支持SSL。

PC连接


为了使一切顺利进行，我决定将PDA连接到他的同伴-Packard Bell Easyone Silver笔记本电脑（他的评论上次发表 ）。 它具有Windows XP，并且PDA立即连接到USB，在Windows存储中找到了驱动程序，并且可以正常工作。



我从捆绑的CD安装了CD Active Sync，以同步数据并在Explorer中查看PDA上的信息。







Active Sync激活没有问题，并开始在笔记本电脑和掌上电脑上同步数据。 要进行完全同步，您需要创建一个用户配置文件，但是由于没有任何要同步的内容，因此您可以不用它。



多媒体类


HP iPaq h1940有一个用于SD卡（microSD并通过适配器工作）的卡插槽。 事实证明，掌上电脑仅接受2 GB或更小的卡。 读取卡中的所有文件都没有问题，而且我设法从2013年启动了mp3文件（原来是Louis Armstrong）。



使用2 GB的存储卡和PDA，您可以将该设备变成媒体播放器。 音质还不错，耳机一切都很好。



我借助存储卡上找到的两种旋律评估了声音。 这些是奥斯卡·本顿（Oscar Benton）的本森赫斯特·布鲁斯（Bensonhurst Blues）和路易斯·阿姆斯特朗（Louis Armstrong）的《 Go Down Moses》。 我记得在PDA上我以某种方式听过有声读物，因此该设备也适用于此。



有了视频，情况就更糟了。 首先，该设备无法播放质量高于320 x 240的视频，并且该设备将无法适应新格式。 在其他地方，仍然保留着可以转换为PDA的电影的资源，但是这些站点中的大多数站点已经关闭，如果没有，则很多下载链接就会死掉。

谁愿意以这种质量观看视频，即使廉价的中国智能手机做得更好呢？

软体类


该软件与其他装有Windows Mobile或Windows SE的PDA上安装的默认应用程序没有什么不同。 有一个“联系人”，一个带有组织者和任务列表的日历，一个邮件应用程序，一个记事本，一个录音机，一个文本编辑器，电子表格。 此外，还有iPAQ Image Viewer HP。

使用第三方软件，一切都比视频要好得多。 拥有大量PDA软件档案的站点仍然可以使用。 例如，HPC-我从中下载了著名的AIReader阅读器，该阅读器为电子书爱好者提供了许多功能。 它可以使用多种格式（epub，fb2，txt和其他格式）。 该阅读器的安装没有问题，显示了带有西里尔字母的屏幕（PDA是西班牙语，我认为西里尔字母可能有问题，但没有）。

使用其他软件则更为复杂-需要从PC安装某些程序。 由于笔记本电脑上装有Windows 10，因此无法进行安装。 为此，您需要ActiveSync，它在tens环境中不起作用。 唯一的方法是将应用程序下载到存储卡，然后将其插入PDA并从存储卡中安装软件包。 实际上，我是使用AIReader做到的。



Windows Mobile 2003 Professional有很多软件，唯一的用途是寻找exe还是cab，因为其他格式通常旨在从PC安装。

除了阅读器之外，我还安装了Sega仿真器PicoDrive 0.03版。 为了进行试用，我下载了最受欢迎的游戏《沙丘：阿拉克之战》的ROM，并将其全部发布在PDA上。 游戏运行正常，我什至玩了一点（一次在另一台PDA上的此模拟器上经历了几次）。 但是印象并不相同-最好是在OS中运行模拟器并在大屏幕上播放，而不是用PDA挡住视线。





无线连接




iPaq h1940 PDA仅配备了蓝牙和IrDa模块。 它不适用于BT-我试图与iPhone 8 Plus配对。 这两台设备相互看对没有问题，甚至发出了输入密码并进行确认的请求。 但是，当我输入密码并在另一台设备上确认后，PDA写下了连接问题。 最有可能是协议兼容性问题。





但是事实证明，它已经通过蓝牙连接到了华为Y6 Android手机，并通过蓝牙连接到了网络。 要进行连接，您需要打开蓝牙，选择与LAN的连接，在列表中找到设备并进行连接。











在电话上，您必须打开设备的检测功能和可见性，并激活蓝牙调制解调器。 只有在这种情况下，PDA才会连接到电话，并通过它进入在线状态。

通过内置的浏览器，无法使用Yandex打开Habr或Google。 正如他们在上一篇文章的评论中所写，其原因是旧浏览器无法使用新标准SSL，IPv6和其他现代技术。

我只能下载曾经很流行的网站hpc.ru，因为它经过优化可在PDA上查看。 另外，我什至设法下载了该应用程序-相同的AIReader，确信安装程序的正常运行。 该应用程序已在浏览器中下载并安装。 顺便说一句，一个有趣的事实是，在默认设置下安装后，安装程序文件将被删除。









该PDA上没有WiFi。

电瓶




令人惊讶地保存得很好。 在首次开启之前，我已对其充电100％。 在晚上和一天的一部分时间内，充电水平仅下降到50％-这是由于设备的积极使用，将其连接到网络等等。 电池状况良好。

2000年代初的PDA现在有用吗？


是的，可以。 以下是一些立即想到的应用程序（我用另一台设备HP iPAQ rx3715实现了其中一些应用程序：

  • 读者。 如果您不想浪费智能手机的电池或阻塞其内存，则HP iPaq h1940 PDA非常适合用作小型读取器。 通过安装AIReader或任何其他软件，您可以开始阅读。
  • 音频播放器。 如果由于某种原因其他设备不适合该角色，则PDA可以充当播放器。 将MP3下载到卡中-然后继续。 例如，如果您不想带手机，可以随身携带PDA进行跑步。
  • 娱乐中心。 从象棋到游戏机模拟器，都有用于Windows Mobile 2003的大量游戏。 PDA的小型复古游戏机非常成功。
  • 提醒清单，计划者。 当然，智能手机几乎是理想的日记，包括存储联系人，任务，计划。 PDA也适用于此目的。
  • 录音机-单个按钮即可开始录音。 设备写入声音，录音质量可以接受。

通常，在主设备发生故障的情况下，可以将PDA用作备用设备。 当然，您将无法使用PDA拨打电话，但是它可以复制现在赋予智能手机的许多其他功能。

这是一个带包装的视频以及该设备的概述。
