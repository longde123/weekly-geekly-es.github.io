Pertukaran token atau cara memperkuat portofolio crypto pada tahun 2019

Tahun ini, token pertukaran menunjukkan ROI tertinggi relatif terhadap pasar dan sektor lainnya: proyek infrastruktur atau IoT. Beberapa token, seperti BTMX atau BNB, telah menghasilkan hampir 7X (X) sejak awal tahun, menyalip kenaikan harga Bitcoin .


https://cryptorank.io/performance

Pada tahun 2019, token pertukaran mendapatkan popularitas besar. Hampir setiap pertukaran menganggap perlu mengeluarkan tokennya. CryptoRank saat ini melacak 53 token pertukaran.


https://cryptorank.io/screener

Apa alasan popularitas seperti itu?


Penggemar Crypto di seluruh dunia dengan bersemangat menciptakan penggunaan token cryptocurrency. Dalam banyak kategori proyek, sangat sulit untuk membuat sistem ekonomi untuk penggunaan token, dan pada proyek + 95%, ekosistem internal sedikit menggunakan token atau tidak menggunakannya sama sekali.

Sebaliknya, token pertukaran adalah kasus nyata yang dapat dibangun ke dalam karya pertukaran. Pengembangan bisnis dan peningkatan volume perdagangan di bursa dapat secara positif mempengaruhi pertumbuhan nilai token itu sendiri dengan partisipasi efektifnya dalam ekosistem pertukaran. Pengembangan proyek dari kategori lain tidak secara signifikan mempengaruhi token, yaitu, proyek dan tokennya ada secara independen. Pada tahun 2018, banyak token yang didepresiasi oleh hampir 100% ICO karena alasan ini.

Pertukaran menyerupai pekerjaan perusahaan yang menghasilkan laba tradisional. Pertukaran hasil keuangan mempengaruhi nilai tukar token karena terbakar, serta kebijakan dividen atau bonus lainnya.

Mengapa nilai tukar token meningkat



https://cryptorank.io/ru/currencies/binance-coin/price-chart

Ada beberapa model yang menggunakan pertukaran untuk mempertahankan nilai tukar token mereka atau merangsang pertumbuhannya. Inilah beberapa di antaranya:

  • Token Burning
  • Model dividen
  • Bekukan Token untuk Berpartisipasi dalam IEO
  • Penambangan transaksi
  • Diskon komisi untuk operasi perdagangan
  • Voting untuk daftar

Lebih lanjut dalam artikel ini kita akan mempertimbangkannya secara lebih rinci dan mencoba menentukan model yang paling efektif dan menguntungkan ...

Token Burning




Pertukaran menggunakan model token burning dalam jumlah bagian tertentu dari laba atau komisi. Selain itu, token burning terjadi selama acara-acara tertentu, seperti Fast Track dan Huobi Prime lite di bursa Huobi, yang sejak April 2019 telah membakar token dalam jumlah 6 832 347 HT, yaitu $ 25 143 036 dengan nilai $ 3,68. Ini adalah bagian penting dari token Huobi yang beredar. Detail dapat ditemukan di sini .

Sebagai aturan, pertukaran memberikan informasi tentang pembakaran token di domain publik. Oleh karena itu, mudah untuk menghitung seberapa jauh jumlah token dalam sirkulasi akan berkurang, dan kapan itu akan terjadi.

Berapa banyak token yang membakar pertukaran:
Okex - 30% komisi di pasar spot.
Huobi - 20% dari pendapatan kotor + penghasilan dari FastTrack .
Binance - 20% dari laba bersih.
Bitfinex - 27% dari pendapatan kotor + setidaknya 95% dari aset bersih yang dipulihkan dari Crypto Capital + setidaknya 80% dari dana yang dipulihkan dari peretasan BitFinex.

Dividen


Beberapa pertukaran menggunakan model dividen: Kucoin, Bibox, Switcheo dan lainnya. Jika pembayaran melebihi pengembalian rata-rata, misalnya, dari mempertaruhkan, mendarat atau pendapatan pasif lainnya, maka dividen dianggap menarik dan mulai mendukung harga.
Di setiap pertukaran Anda dapat menemukan kalkulator untuk menghitung dividen.

Partisipasi dalam IEO


Untuk berpartisipasi dalam IEO, Anda harus sering memiliki token pertukaran. Biaya proyek CNNS di pertukaran Gate.io adalah contoh jelas menggunakan token pertukaran untuk berpartisipasi. Proyek ini mengumpulkan $ 112.484.280 dengan hardcap yang dinyatakan $ 3.600.000. Jadi biaya CNNS menunjukkan kelebihan permintaan dari 3,124,56%.

Sejak pengumuman IEO di bursa sampai tanggal kepemilikannya, nilai platform token telah meningkat sebesar 25,3% (dari $ 0,75 menjadi $ 0,94 dengan sirkulasi 150.000.000 GT). Kapitalisasi token GT dalam kasus ini adalah sekitar $ 141.000.000.Hal ini menyebabkan penggunaan alokasi token 80% untuk melakukan biaya IEO dengan tekanan kuat dari pembeli, IEO CNNS investor, yang kemudian mendorong harga pada puncaknya menjadi $ 1,1. Meskipun demikian, Gate tidak bisa mempertahankan minat pada IEO pada tingkat tinggi dan nilai tukar token menurun, karena tidak lagi terlibat dalam pekerjaan pertukaran.

Anda dapat membaca lebih lanjut tentang berinvestasi di IEO di artikel kami sebelumnya.

Menambang dan menambahkan token ke sirkulasi


Peningkatan sirkulasi mempengaruhi nilai tukar token, oleh karena itu, analisis jadwal untuk membuka atau menerbitkan token pertukaran diperlukan. Beberapa pertukaran menggunakan penambangan transaksional, misalnya, Bitmax atau FCoin.
Harus dinilai seberapa banyak sirkulasi token akan meningkat pada periode bunga dan seberapa besar ini dapat mempengaruhi harga. Tanpa peningkatan permintaan, pelepasan token baru tidak akan dapat menyebabkan penurunan harga.

Diskon komisi


Sebagian besar pertukaran menawarkan diskon komisi ketika menggunakan token pertukaran. Penting untuk mengevaluasi bagaimana proposal ini menguntungkan, dalam permintaan dan sesuai.

Misalnya, pertukaran Binance dan Huobi telah memperkenalkan model menarik untuk memegang token pertukaran. Semakin banyak token, semakin besar diskon.

Penggunaan token pertukaran untuk diskon berjumlah jutaan dolar.

Voting, akses ke layanan dan manajemen pertukaran


Token pertukaran sering memungkinkan Anda untuk mempengaruhi ekosistem pertukaran.

Token pertukaran memberikan kesempatan untuk memilih proyek, dan dengan demikian mengelola sistem pencatatan untuk kepentingan mereka sendiri. Semakin luas dampaknya pada sistem pertukaran, semakin bernilai token itu.
Beberapa fitur token pertukaran tidak diumumkan sama sekali. Misalnya, di bursa Binance dan Huobi:

  • Untuk menjadi duta besar pertukaran, Anda harus memegang token pertukaran.
  • Kunci token pertukaran sebagai penjamin pertukaran OTC.
  • Kesempatan untuk berpartisipasi dalam promosi airdrops dan pemasaran.
  • Token pertukaran sebagai bagian dari indeks.

Aplikasi dalam proyek dan kualitas lain


Awalnya, token pertukaran dapat menemukan berbagai aplikasi di luar bursa. Ini dapat digunakan sebagai alat pembayaran di proyek lain atau sebagai token asli pada blockchainnya sendiri. Contoh yang jelas saat ini adalah token BNB dan Rantai Binance.

Semakin banyak proyek yang beralih ke Binance Chain. Oleh karena itu, indikator untuk mengevaluasi blockchain cryptocurrency, misalnya, indikator NVT dan lainnya, sudah mulai digunakan untuk mengevaluasi BNB.



Analisis pertumbuhan volume untuk periode tersebut


Dalam menilai, pertumbuhan historis volume di bursa saham penting untuk memprediksi pertumbuhan di masa depan berdasarkan data historis dan kemungkinan prospek lebih lanjut untuk meningkatkan volume. Juga penting untuk menganalisis perubahan dalam pangsa pasar yang diduduki dalam hal volume perdagangan.



Setelah menganalisis rasio volume perdagangan dengan kapitalisasi token pertukaran, kita dapat menyimpulkan berapa banyak volume ini dikapitalisasi. Mungkin bahkan token yang terlibat dalam sistem pertukaran tidak akan tumbuh jika mahal, dan volumenya tidak besar. Jadi, misalnya, di pertukaran Switcheo. Namun, dengan peningkatan volume, pertumbuhan token tidak akan memakan waktu lama jika token itu sendiri menempati tempat yang signifikan dalam ekosistem pertukaran
