Microsoft bereitet die Veröffentlichung einer eigenen SIM-Karte vor



Ein kostenloses Programm namens Cellular Data wurde im Windows Store für mobile Anwendungen veröffentlicht. Es bietet mobilen Internetzugang unabhängig von Verträgen mit Mobilfunkanbietern. Wie aus der Beschreibung hervorgeht, ist es für die Funktion des Programms erforderlich, eine spezielle SIM-Karte von Microsoft und eine mobile Version des Windows 10-Betriebssystems zu verwenden.

Es wird davon ausgegangen, dass Benutzer den Internetzugang mit ihrem Microsoft-Konto bezahlen. Es ist noch nicht festgelegt, in welchen Ländern und zu welchem ​​Preis ein ähnlicher Service verfügbar sein wird.

Anscheinend wird Microsoft ein virtueller Mobilfunkbetreiber, der Datendienste über geleaste Mobilfunknetze bereitstellt.

Der zugehörige Service wird von Apple bereits für seine Tablets angeboten. Mit der Apple SIM-Karte können die Mobilfunkbetreiber geändert werden, mit deren Diensten der Benutzer gerade arbeiten möchte.

Virtuelle Mobilfunkbetreiber, die in gemieteten Einrichtungen bestehender Anbieter, jedoch mit eigenen Tarifplänen, tätig sind, wurden im Jahr 2000 eingeführt. Der erste, der dies tat, war Tele2, basierend auf dem dänischen Anbieter Sonofon. Eine der Abteilungen von Virgin, Virgin Mobile, wurde der erste virtuelle Betreiber in Großbritannien.

In Russland gibt es derzeit drei Dutzend virtuelle Betreiber, basierend auf den Kapazitäten der Big Three und einiger regionaler Anbieter.
