Alexei Ragozin在jug.msk.ru会议上讨论OpenJDK 11中Java Flight Recorder的新功能和旧功能

12月12日，在2019年莫斯科Java开发人员社区jug.msk.ru的最后一次会议上， Alexei Ragozin举行了关于使用新版本的Java Flight Recorder和Mission Control的大师班。



关于演讲者


阿列克谢是jug.msk.ru会议的最频繁的演讲者，这是他第五次发言。 上一次 （2018年3月13日），他举办了关于相同工具以前版本的大师班。 在此之前，演出于2016年8月30日，2016年 1月20日和2015年6月4日举行 。

自上次在jug.msk.ru上发表演讲以来，Alexey的报告：

  • “高负荷的应用数学”（ HighLoad ++ Siberia 2018 ： 视频 ）
  • “我们在Linux下的Java交易系统中实现了亚毫秒级的响应时间”（ HighLoad ++ 2018 ： 视频 ）
  • Java应用程序的实用概要分析（ Yappi Days 2018 ： video ）
  • “使用BDD进行连续负载测试”（ Bugs Busters mitap ： 视频 ）

其他链接： Speaker Deck上的演讲者演示，一个包含技术文章的博客以及一个包含报告完整列表的部分，其中包含指向视频和演示文件的链接， GitHub ， Twitter ， TimePad上的事件注册。

关于报告


Andrey Kogun再次宣布会议开始，回顾了即将举行的会议（2020年2月29日在新西伯利亚召开的会议， DevOops 2020年4月29-30日举行的会议以及2020年5月15-16日在莫斯科举行的JPoint会议）并介绍了发言人。 打开笔记本电脑的学生已准备好开始参加大师班。



在参加大师班之前，建议按照以下说明进行准备工作。 在报告的第一部分中，概述了概要分析任务，并比较了Java 8和Java 11可用的工具。



除了控制台实用程序， Visual VM ，从BellSoft和GC Viewer构建的Liberica Mission Control 7外 ，Alexey还引入了专有应用程序-Swiss Java Knife（SJK） 。 在预先准备好的应用程序上演示了Java Flight Recorder的新功能。



报告伴随着与听众的热烈交流；对听众提出的问题作了详细的回答。 在休息时间和三个小时的会议结束时，可以与演讲者交谈并互相讨论大师班的印象。
