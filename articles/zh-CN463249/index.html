Game Dev Sim：有关游戏开发的棋盘游戏

最近，出现了一种俄罗斯棋盘游戏，它清晰而有趣地给出了电子游戏创作者的工作有多么困难的想法！ 有点像著名的Startup游戏，但是在内容和游戏功能上却更多。 当然，还有关于游戏开发的特定设置。 在剪切下，我收集了对该主题感兴趣的人的详细信息。



游戏目的和玩法


领导您自己的游戏工作室并创建一个很棒的游戏。 它是计算机，控制台还是移动平台的杰作，由玩家决定。 在此过程中，您将不得不根据游戏行业的专业人员开发的，基于真实事件的60多个案例寻找解决方案，这些专业人员致力于开发游戏已有多年经验，并且能够重现游戏创造者固有的那种难以形容的混乱，有趣，恐怖和非凡的管理决策氛围。



玩家的任务包括：项目管理，办公室租金，人员选择和激励，营销，游戏设计，使用代码和艺术作品，参加展览，与投资者互动等等。 每个决定都会影响游戏玩法，并可能导致玩家压倒性的成功或完全失败。

游戏开发模拟是：


  • 来自真实游戏开发工作室的60个真实案例。
  • 数百种组合和策略。 重播性居于首位。
  • 情况的动态变化，取决于玩家的行为。
  • 简单的规则，快速的发展。
  • 由游戏行业的专业人士创建。
  • 还有一些品牌的devdev幽默！



谁是游戏的作者


GEMENOT棋盘游戏出版商的所有者Golubkin Sergey已发布了30多个台式机，这是Ubisoft和Nival的计算机和手机游戏的前制作人。
维亚切斯拉夫·乌托奇金 （ Vyacheslav Utochkin）是经济学院高等商业信息学高等学校游戏业教育计划的主任，曾是装甲战争，完美世界和其他几种游戏的Mail.Ru集团的制作人。



教育效果


Game Dev Sim是在高等经济学院的商业信息学高级学院构思并用作游戏实践的培训课程，用于培训“ Game Project Management”教育计划。 游戏的桌面格式很不错，因为玩家之间可以轻松地进行实时交流，您可以立即回答出现的任何问题。 这非常重要，有助于教育过程。 同时，仪表板也很有趣，您可以尽情玩乐。 这些规则非常简单，因此，甚至与游戏行业无关的任何人都可以迅速找出并玩游戏。 以下是有关该游戏的简短视频：
