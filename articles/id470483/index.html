Laboratorium Teknologi Azure di Moskow



15 Oktober, kami mengundang Anda ke Lab Teknologi Azure musim gugur - salah satu acara teknis terpenting Microsoft di Rusia. Dua kali setahun, kami mengumpulkan profesional TI terkemuka, perwakilan bisnis, dan semua orang yang ingin belajar paling banyak tentang layanan Azure. Peserta acara tidak hanya menerima informasi yang paling relevan tentang salah satu pemimpin di pasar hiper-scaler - Microsoft Azure, tetapi juga menjalani pelatihan dalam teknologi, layanan, dan solusi modern yang membantu organisasi dari berbagai ukuran bisnis menempati posisi terdepan di pasar.

Bagaimana cara mendaftar? Untuk mendaftar ke suatu acara, Anda harus memilih dan mendaftar di salah satu laboratorium Azure yang paling menarik minat Anda.

Nah, detail dan programnya di bawah cut!

Apa yang akan terjadi


10.00 - 12.30 Sesi pleno di mana Anda akan belajar tentang pengembangan platform Azure, serta mendengar kisah nyata tentang penggunaan platform Azure dalam organisasi Rusia terkemuka, perubahan apa yang dicatat oleh kolega Anda dalam pekerjaan mereka, bagaimana mereka berencana untuk memperhitungkannya dalam strategi TI dan tentang apa akan fokus dalam waktu dekat.

13:30 - 18:30 laboratorium Azure - enam sesi langsung, di mana Anda akan menemukan intensif berbasis cloud pada skenario utama Microsoft Azure di bawah bimbingan para ahli teknis terbaik. Daftarkan lagu favorit Anda untuk mendapatkan daftar peserta untuk seluruh acara.

Apa yang akan menjadi laboratorium


  • Infrastruktur modern: awan telah datang untuk tinggal - Anda akan belajar semua tentang pendekatan modern untuk membangun infrastruktur perusahaan.
  • Manajemen infrastruktur dengan fokus pada keamanan - Pengembangan teknologi cloud dan integrasi TI dalam proses bisnis modern secara tak terelakkan meningkatkan persyaratan untuk memastikan keamanan informasi.
  • Mengembangkan dan menggunakan aplikasi di cloud - Pengembang perlu melakukan apa yang mereka lakukan terbaik - mengembangkan aplikasi hebat. Platform cloud Azure akan mengurus sisanya!
  • Platform Data untuk Aplikasi dan Analisis - Dengan menghadiri sesi ini, Anda akan belajar dari pakar Microsoft terkemuka mengapa bekerja dengan data sangat penting. Kami akan melihat lebih dekat pada platform manajemen data hybrid SQL Server dan layanan data platform cloud Microsoft Azure.
  • Teknologi Inteligensi Buatan - Sebagai bagian dari lagu ini, Anda dapat membenamkan diri dalam teknologi kecerdasan buatan berdasarkan Azure dan berkenalan dengan semua layanan yang tersedia.
  • Pengalaman langsung: pekerjaan mendalam dengan Microsoft Azure - Lagu ini akan sepenuhnya dikhususkan untuk pekerjaan laboratorium. Tidak diperlukan pendaftaran tambahan. Kunjungan ini gratis untuk semua pendatang.
