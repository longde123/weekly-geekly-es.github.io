34 pustaka Python open source (2019)


Kami melihat dan membandingkan 10.000 pustaka sumber terbuka untuk Python dan memilih 34 yang paling berguna.


Kami telah mengelompokkan perpustakaan ini ke dalam 8 kategori.


Artikel ini didukung oleh EDISON Software, sebuah mesin pencari optimasi dan perusahaan SEO , yang juga mengembangkan aplikasi mobile Android dan iOS .

Python toolkit


1. Pipenv : Alur Kerja Pengembangan Python untuk Manusia.


2. Pyxel : Mengembangkan game retro dengan Python.


3. PyTest v3.5 : Kerangka yang membantu menulis tes pendek, dan skala untuk tes fungsional yang kompleks.



4. Puisi : Menyederhanakan manajemen ketergantungan dan pengemasan.


5. Loguru : Menyederhanakan log.


6. Faust : perpustakaan untuk aplikasi streaming / streaming.


7. Pampy : Pencocokan Pola yang Anda impikan.


8. Pemeriksaan Pyre : Pemeriksaan jenis pemain .


9. Delorean , perpustakaan untuk bekerja dengan waktu dan tanggal.


10. Cirq : perpustakaan skema Noisy Intermediate Scale Quantum (NISQ).


11. Python-nubia : Kerangka baris perintah.


Web


12. Permintaan-HTML : Parsing HTML untuk Manusia.


13. Bokeh : visualisasi data interaktif di browser modern.


14. Vibora : cepat, dingin, asinkron.


15. Pywebview v2.0 : Tampilkan konten HTML di jendela terpisah.


16. WhatWaf : Deteksi dan bypass firewall dan sistem keamanan aplikasi web.


17. Molten : Kerangka kerja minimalis dan cepat untuk membuat API HTTP.


Terminal


18. Termtosvg : kami merekam sesi bekerja dengan terminal sebagai animasi SVG.


19. Asciinema v2.0 : kami merekam sesi terminal.


20. Termgraph : alat untuk baris perintah, menggambar grafik.


Editor kode


21. Hitam : pemformat kode tanpa kompromi.


22. Algojammer : editor kode eksperimental untuk algoritma penulisan.


23. Bowler : alat refactoring tingkat pohon sintaksis.


Debugging


24. Py-spy : memungkinkan Anda memvisualisasikan program apa yang Anda buang-buang waktu tanpa memulai ulang program.


25. Birdseye : Sebuah debugger grafis menggunakan AST.


26. Icecream : debugger lucu.


Kompiler


27. Transcrypt : penerjemah Python 3.7 ke JavaScript.


28. Pyodide : tumpukan datacience di browser.


Terkait data


29. Bahenol : validasi data.


30. Botflow : Kerangka kerja pemrograman berbasis data untuk pekerjaan jalur pipa data (Web Crawler, Machine Learning, Quantitative Trading.etc).


31. Fast-Pandas : tolok ukur komparatif dalam operasi Pandas.


Bagan


32. Tur di Negeri Ajaib Matematika dengan Python : kumpulan skrip untuk menggambar bentuk yang indah dan algoritma animasi.


33. Chartify : membantu ilmuwan data membuat grafik.


34. Hypertools v0.5 : Alat untuk representasi geometris data multidimensi.
