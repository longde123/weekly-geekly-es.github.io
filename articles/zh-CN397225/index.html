Lenovo在IFA 2016上的新闻指南

在IFA 2016上，联想推出了许多新设备，其中许多设备可以说是非常轰动。联想提供的有关新产品的最全面指南-已删减。



联想YOGA Book


联想YOGA Book笔记本电脑不仅在可能的范围内而且在所进行的修改中都给人留下了深刻的印象：该设备提供两种不同的操作系统-Windows 10和Android 6.0。



见面时首先引起您注意的是非常不寻常的铰链设计。不仅如此，它还允许您将笔记本电脑完全旋转360度，并且包含130个元素。从一开始的设计就清楚地表明它是可靠的，而且我不怕这个词是优质的。当然，Lenovo变压器的标准模式-“笔记本电脑”，“平板电脑”，“演示文稿”和“控制台”也可以在这里使用。

独特的Halo键盘不同于通常的版本-上面没有传统的机械按钮。触觉响应功能100％有效，您可以编辑文档并回复邮件。另外，智能键盘可以被训练并具有预想输入功能。触摸键的LED照明看起来就像是一部关于遥远未来的电影-它还不是全息图，但是已经接近了。

但是，键入不是唯一的键盘任务。该面板还可以用于使用Real Pen手写笔输入图形。手写笔可以用墨水在键盘面板顶部的纸上书写，同时可以用虚拟墨水书写。面板可以识别压紧力，通过简单的压紧操作，以最自然的方式调节线条的粗细。笔允许最大100度的偏差，电磁共振技术（EMR）可以识别多达2048个压力等级。



从技术上讲，联想YOGA Book的机身非常薄，厚度仅为9.6毫米，重量为690克。同时，技术部分没有受到影响-两种版本的笔记本电脑均配备了Intel Atom x5-Z8550处理器，10.1英寸全高清IPS显示屏。至于内存，笔记本电脑配备了4 GB的RAM和64 GB的内部闪存。顺便说一句，您可以使用microSDXC存储卡扩展内存。集成的Intel HD Graphics 400负责图形处理，

两个版本的笔记本电脑均支持2G / 3G / 4G（LTE）网络，并配备Wi-Fi（802.11 a / b / g / n / ac），蓝牙，GPS和HDMI连接器。

该设计在各个方面都得到了完善。一次接触Lenovo YOGA Book，我就不会放手-一种令人愉悦且易于使用的工作工具。

联想MiiX 510


多功能的MiiX 510变压器笔记本电脑更像平板电脑。也许这是在皮革基材上制成的键盘的优点。决定是有道理的-正如幻想所说，键盘很容易卸下，放回原处。键盘是传统键盘和全尺寸键盘-联想决定在平板电脑上键入文字应该方便。顺便说一句，正确完成。



便携式计算机正在运行Windows10。但是，它不会干扰其向平板电脑的转换。是否随身携带键盘只是用户方便的问题。如果使用不带键盘的笔记本电脑，它将变成功能强大且方便使用的平板电脑，位于背面的支架倾斜角度为150％。借助集成式支架，可将平板电脑轻松放置在桌子或其他任何表面上。

不带键盘的笔记本电脑重880克，厚度仅为9.9毫米。该设备接收了容量为39 Wh的可充电电池；声称的电池寿命为7.5小时。

MiiX 510建立在第六代Intel Core i7处理器（顶级）上，具有受大猩猩玻璃保护的12.2英寸显示屏。该图形芯片是内置的Intel HD Graphics520。RAM为8 GB，内部PCI SSD的最大容量为1 TB，具体取决于配置。标准支持无线网络：Wi-Fi 802.11ac，蓝牙4.0和LTE。

联想YOGA 910


笔记本联想YOGA 910-也是变压器。所有品牌芯片-360度旋转，功能强大且可靠的130个元素的循环系统，4种操作模式-到位。笔记本电脑非常轻薄-仅14.3毫米，重量不到1.4千克。对于这种厚度，性能似乎很高。



根据笔记本电脑的修改，您可以选择13.9英寸的屏幕分辨率-4K或Full HD。与上一代相比，屏幕框架已经变得非常薄-厚度仅为5毫米。该笔记本电脑基于第七代Intel Core i7处理器构建。内置PCIe SSD的容量为1TB。 YOGA 910最多可配备16 GB RAM。设备在Windows 10上运行。

笔记本电脑配备了指纹扫描仪-消除了记住复杂文本密码的问题。认识到所有者只需花费一秒钟，因此无需记住任何内容。该设备的电池可续航12小时。

联想P2


联想P2智能手机可谓是P系列的继任者，尽管其屏幕尺寸保持不变（5.5英寸），但与第一个版本相比，它的机身轮廓更加流畅。分辨率为FHD（1920x1080）并带有防护玻璃的Corning Gorilla Glass 3 AMOLED屏幕在点亮支架之前没有保存-屏幕保持明亮。



薄型金属外壳，底部正面的指纹扫描仪以及物理主页按钮，Android 6.0是P系列的可识别细节，智能手机可舒适地握在手中，看起来像是可靠的设备。重量-175克，厚度-仅8.5毫米。

P2拥有5100 mAh电池。这样的电池在相当密集的负载下可以持续整整三天。在负载下，我们不仅意味着通话和短信，还包括游戏，互联网，相机和其他智能手机系统的操作。如果没有快速充电就无法做到这一点-Turbo Power技术可让您在15分钟内为智能手机充电10个小时。

智能手机在具有3 GB RAM和32 GB内部存储器的Qualcomm Snapdragon 625芯片组上运行。如果需要，可以使用最大为128 GB的microSD存储卡扩展内部存储器。



主摄像头的分辨率为13兆像素，前置-5兆像素。主相机配有CCT闪光灯，自动对焦系统和稳定系统。

根据修改情况，智能手机可以与一或两个nano-SIM格式的SIM卡一起使用。该设备的所有版本均支持GPS，GLONASS，BDS和LTE网络。要说设备引起了人们的更多兴趣，什么也没说。

联想PHAB 2 Pro


一种围绕着神秘光环的装置已经形成。这完全是由联想和谷歌联合开发的Tango技术。虚拟现实技术使您能够显着扩展平板手机的功能，例如，在实际房间中布置虚拟家具或参观博物馆。



联想PHAB 2 Pro配备6.4英寸Full HD 2K显示屏，板载Qualcomm 652处理器，平板手机配备4 GB RAM和64 GB内部内存。4050 mAh电池，指纹扫描仪位于后面板上。

手持平板手机很方便。是的，它很大，在某种程度上不属于紧凑型设备，但这丝毫不妨碍使用它。虚拟现实的主要工具是相机。摄像头分辨率为16兆像素，在后面板上还有一个深度传感器，以及许多传感器，它们可以对周围现实进行正确的平板感知。

在展览会上，您可以尝试摆放家具，测量房间角落之间的距离，或者只是使用相机的AR模式。它非常简单地打开-通过在相机应用程序中按一个按钮即可。虚拟动物（例如猫或龙）将在平板屏幕上欢乐地探索现实世界。在真实的桌子上行走的彩绘猫有点吓人，但另一方面，为什么最近的宠物小精灵感到惊讶呢？

联想K6系列


K6系列智能手机一次包含三个设备。名称之间的主要区别很容易猜出：K6，K6 Power和K6 Note。整个产品线具有共同的特征-立体声扬声器和杜比全景声（Dolby Atmos）环绕声系统的存在，支持LTE网络并由Qualcomm Snapdragon 430处理器控制。所有三款智能手机中的指纹扫描仪均位于后面板的摄像头正下方。



强大的中产阶级-联想K6是运行Android 6.0的功能强大的多媒体设备。智能手机具有5英寸IPS屏幕，分辨率为FHD（1920x1080）。内存容量相当大-2 GB的RAM和16 GB的内部内存，以及对容量高达128 GB的microSD存储卡的支持。

分辨率为13兆像素的主摄像头具有相位自动对焦，前置摄像头的分辨率为5像素。根据修改情况，可能会提供一张或两张SIM卡，电池容量为3000 mAh。

在增强型电池中，K6电源不同于K6是合乎逻辑的。电池容量为4000 mAh，这不但会影响外壳的厚度-K6 Power略厚于K6。

K6 Note是最先进的，但也是最大的。该智能手机具有更大的屏幕尺寸（5.5英寸）和更严肃的相机（具有双CCT闪光灯的16兆像素）。还有更多的内存-3 GB的运行内存和32 GB的内部内存。电池与K6 Power中的电池相同-4000 mAh。



K6系列是价格适中的智能手机的一个很好的例子。您可以根据个人喜好选择三个选项中的任何一个，并在填充良好的金属外壳中获得可靠，高质量的智能手机。

联想YOGA Tab 3 Plus


联想YOGA Tab 3 Pus是以前型号的改进版本。外观没有任何变化-相同的支架，相同的圆柱电池，具有环绕声的JBL扬声器和四种操作模式-YOGA的所有荣耀。



YOGA Tab 3 Plus的2K显示屏尺寸为10.1英寸。现在，您可以通过蓝牙将音频流传输到平板电脑，或者根据Miracast标准以高质量传输视频流。

平板电脑的电池容量为9300 mAh，并通过随附的充电器支持快速充电。在最活跃的使用情况下，这样的容量大约需要18小时的电池寿命。



YOGA Tab 3 Plus配备了Qualcomm Snapdragon 652处理器，Qualcomm Adreno 510显卡，3 GB RAM和32 GB内部内存。可以使用microSD存储卡，无线模块-完整的集扩展内存：LTE，Wi-Fi 802.11ac，蓝牙4.0。

平板电脑装有两个摄像头，主要的一个是13兆像素，前面的是5兆像素。不同版本的设备的重量有所不同。Wi-Fi版本重637克，LTE版本重644克。

Moto Z Play和Moto Mods：哈苏True Zoom


当然，Moto推出了其新产品。该品牌推出了Z系列的新智能手机-Moto Z Play。他收到的增强型电池容量为3510 mAh-Z Play的厚度几乎是Moto Z的2倍。快速充电技术使您只需15分钟即可在出口处获得9个小时的电池寿命。



在具有防水涂层的情况下-智能手机不怕雨淋，喷雾和其他湿气问题。当然，您不应将其长时间浸入水中。

5.5英寸屏幕，超级AMOLED，全高清，由康宁大猩猩玻璃保护。在任何环境下都具有惊人的亮度，而且视角广-进行各种活动所需的一切-观看视频，照片，上网，玩游戏。

分辨率为16兆像素的主摄像头配备了双自动对焦-激光和相位。前置摄像头为广角镜头，带有5百万像素传感器和闪光灯。



智能手机的填充物看起来很严重：Qualcomm Snapgragon 625处理器，3 GB RAM和32 GB主内存。自然，支持使用高达2 TB的microSD卡。全套无线技术：Wi-Fi 802.11b / g / n 2.4 / 5 GHz，蓝牙4.0 LE，4G LTE，NFC。

Moto Z系列智能手机的模块化并不是什么秘密，但是在展会上，一个新的相机模块被引入了世界。与哈苏一起开发了哈苏True Zoom模块。除了免费使用哈苏的Phocus软件外，它还增加了十倍的光学变焦，使您可以RAW格式拍摄。
