Anweisungen zur Umgehung der Telegrammblockierung. So aktivieren Sie SOCKS 5

Der Telegrammkanal tnews_ru veröffentlichte Anweisungen zur Umgehung einer möglichen Blockierung von Telegram, einem beliebten Messenger, der vom Team von Pavel Durov entwickelt wurde. Der Artikel beschreibt eine Problemumgehung, die auf der Verwendung des SOCKS5-Netzwerkprotokolls basiert. Dieses Protokoll leitet Datenpakete nahtlos über einen Proxyserver vom Client an den Server weiter.

SOCKS ist ein Netzwerkprotokoll, mit dem Client-Server-Anwendungen Dienste hinter Firewalls (Firewalls) transparent (für sie unsichtbar) verwenden können. Die Option ist bereits in Telegram Desktop und Mac verfügbar, und mobile Clients erhalten sie beim nächsten Update.

Tipp: Aktivieren Sie die Option für die automatische Aktualisierung in den Einstellungen.

Wie kann der Zugriff auf Telegramm beibehalten werden, nachdem der Messenger in Russland blockiert wurde?

Schalten Sie zuerst SOCKS5 ein, bevor Sie die Sperre starten.

Wenn Sie aus irgendeinem Grund keine Zeit dafür haben, müssen Sie sich festziehen, weil Aus dem russischen Google Play und AppStore werden Telegrammkunden wahrscheinlich verschwinden. Vpn und Tor helfen hier, über die Sie Telegram in amerikanischen Läden herunterladen können. Aber schmeicheln Sie sich nicht - vor dem Gesetz steht das Verbot der Umgehung von Schlössern, das in erster Lesung in der Staatsduma der Russischen Föderation verabschiedet wurde.

Tipp: Ermöglichen Sie in den Android-Einstellungen das Herunterladen von Updates, die nicht aus dem offiziellen Store stammen, und laden Sie das Telegramm-Update von der Messenger-Site telegram.org herunter.

SOCKS konfigurieren 5


Wir geben die Anweisung in russischer Sprache:

1) Gehen Sie zu den Telegramm-Desktop-Einstellungen und klicken Sie auf "Standardmäßig (jetzt: TCP)".

2) Wechseln Sie im Menü "Verbindungstyp" zu "TCP socks5 proxy".

3) Geben Sie die Proxy-Server-Adresse (z. B. 192.254.68.37) und den entsprechenden Port (1080) ein. Es ist besser, Proxy-Adressen von Amerika, Deutschland, Schweden oder England zu verwenden. Aktivieren Sie dann "IPv6 verwenden" und speichern Sie.

Wie finde ich einen Proxyserver?


Folgen Sie einfach diesen Links:

Nehmen Sie einfach den Server für SOCKS5, verwechseln Sie nicht mit HTTP.

Sie können aus dieser kostenlosen Liste (socks - sockslist.net ) konfigurieren .

Standardmäßig stellt Ihr Telegramm jetzt eine Verbindung über den ausgewählten Proxyserver her.

Zusatz: Wenn Sie Probleme mit der Verbindung haben, müssen Sie die Windows-Firewall konfigurieren - fügen Sie den zulässigen Anwendungen Telegramm hinzu. Dies ist einfach, Sie müssen jedoch über Administratorrechte verfügen.

  • Gehen Sie zu "Systemsteuerung" und gehen Sie zu "Windows-Firewall".
  • Klicken Sie auf den Punkt "Berechtigungen zur Interaktion mit Anwendungen oder Komponenten ...". Ein Fenster mit einer großen Anzahl von Programmen wird geöffnet, aber es ist unwahrscheinlich, dass Sie dort Telegramm finden. Daher müssen Sie es dort hinzufügen.
  • Klicken Sie auf "Einstellungen ändern" und dann auf die untere Schaltfläche "Andere Anwendung zulassen".
  • Im Popup-Fenster "Anwendung hinzufügen" über die Übersicht Telegramm hinzufügen.
  • Das ist alles. Konfigurieren Sie die Verbindung zum Proxyserver erneut.
