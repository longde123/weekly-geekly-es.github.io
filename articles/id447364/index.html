Microsoft Edge on Chromium secara resmi tersedia untuk pengujian terbuka

Kemarin, kami mulai menguji browser Edge secara publik di Chromium. Sejauh ini, 2 versi pratinjau untuk Windows 10 tersedia: Dev dan Canary. Membangun Dev berisi tambahan terbaik minggu lalu. Mereka telah diuji oleh tim Microsoft Edge dan umumnya lebih stabil daripada Canary. Canary diperbarui setiap hari dan di dalamnya Anda dapat melacak kemajuan harian dalam pengembangan browser.

Unduh, coba dan tinggalkan ulasan di sini . Menunggu tanggapan Anda!



Omong-omong, kami memiliki komunitas @MSEdgeDev di Twitter. Di sana Anda dapat mengetahui semua berita terbaru dan terlibat dalam proses pengembangan browser. Bergabunglah sekarang!
