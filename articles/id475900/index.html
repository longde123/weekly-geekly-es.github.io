1. Administrasi Kontainer di Azure


Contoh wadah Azure adalah cara tercepat dan termudah untuk menjalankan kontainer di Azure. Tutorial ini akan membantu Anda mempelajari cara membuat dan mengelola wadah dan cara mencapai penskalaan yang fleksibel untuk Kubernet menggunakan ACI.

Modul Kursus:

  • Membuat aplikasi web kontainer menggunakan Docker;
  • Membuat dan menyimpan gambar kontainer menggunakan layanan Azure Container Registry;
  • Luncurkan kontainer Docker menggunakan layanan Azure Container Instances;
  • Menyebarkan dan meluncurkan aplikasi kontainer web menggunakan Layanan Aplikasi Azure;
  • Ikhtisar Layanan Azure Kubernetes.

Pelajari lebih lanjut dan mulai pelatihan dengan mengklik tautan

2. Rekayasa Data dengan Azure Databricks


Pelajari cara bekerja dengan baik dengan Azure Databricks dan mempercepat pengaturan solusi. Bekerja dengan data di Azure SQL Data Warehouse menggunakan layanan konektor bawaan. Ikhtisar layanan data yang tersedia di Azure. Buat alur kerja yang disederhanakan dan bekerja dengan ruang kerja analitik interaktif berdasarkan Apache Spark. Ngomong-ngomong, kursus akan memakan waktu sekitar 8 jam.

Modul:

  • Gambaran Umum Azure Databricks;
  • Akses contoh SQL Data Warehouse menggunakan Azure Databricks;
  • Mendapatkan data dengan Azure Databricks;
  • Membaca dan menulis data menggunakan Azure Databricks;
  • Transformasi data dasar di Azure Databricks;
  • Lakukan konversi data lanjutan di Azure Databricks;
  • Membuat jalur pipa data menggunakan Databricks Delta;
  • Bekerja dengan streaming data di Azure Databricks;
  • Buat visualisasi data dengan Azure Databricks dan Power BI.

Detail dan mulai pelatihan

3. Mengembangkan solusi yang efektif dalam Azure


Pelajari cara mendesain dan mengembangkan solusi yang kuat, terukur, dan berkinerja tinggi di Azure dengan menjelajahi fitur-fitur utama dari arsitektur yang efisien.

Setelah mempelajari kursus 4,5 jam ini, Anda akan mempelajari kriteria dasar untuk arsitektur Azure yang sukses, mempelajari desain dengan fokus pada keamanan, kinerja dan skalabilitas, efisiensi operasional, efisiensi tinggi, dan keterjangkauan. Bergabunglah sekarang!

Detail dan mulai pelatihan

4. Bekerja dengan data NoSQL di Azure Cosmos DB


Data NoSQL adalah cara yang efektif untuk menyimpan informasi yang tidak memenuhi persyaratan database SQL relasional. Pelajari cara menggunakan data Novel dari mana saja di portal Azure, ekstensi Azure Cosmos DB untuk Visual Studio Code, dan .NET Core SDK untuk Azure Cosmos DB untuk menyediakan kepada pengguna ketersediaan tinggi di mana pun mereka berada.

Bergabunglah dengan Belajar NoSQL di Azure Cosmos DB!

Detail dan mulai pelatihan

5. Mengimplementasikan data warehouse dengan Azure SQL Data Warehouse


Azure SQL Data Warehouse menyediakan Big Data Warehouse yang dapat menskala hingga beberapa petabyte data. Tutorial ini akan menunjukkan kepada Anda bagaimana Azure SQL Data Warehouse dapat mencapai skala ini dengan menggunakan arsitektur mass parallel processing (MPP). Buat gudang data dalam hitungan menit dan gunakan bahasa permintaan yang dikenal untuk membuat laporan. Unduh sejumlah besar data dalam beberapa menit dan pastikan penyimpanan data aman.

Topik-topik berikut tercakup dalam skema pelatihan ini.

  • Informasi tentang cara membuat Gudang Data Azure SQL;
  • Meminta dan memvisualisasikan data dari Azure SQL Data Warehouse;
  • Mengimpor data ke dalam SQL Data Warehouse menggunakan Polybase;
  • Pelajari tentang kontrol keamanan yang disediakan oleh Azure Warehouse Service dan Azure SQL Data Warehouse.

Detail dan mulai pelatihan

6. Membuat Aplikasi dengan Azure DevOps [dalam bahasa Inggris]


Azure DevOps memungkinkan Anda membuat, menguji, dan menggunakan aplikasi apa pun di cloud atau di tempat. Pelajari cara mengatur pipa rakitan yang secara terus menerus merakit, menguji, dan menguji aplikasi.

Topik-topik berikut tercakup dalam skema pelatihan ini.

  • Berkolaborasi dalam membangun aplikasi dengan Azure Pipelines dan GitHub;
  • Jalankan tes otomatis dalam pipa untuk memeriksa kualitas kode;
  • Memindai kode sumber dan komponen pihak ketiga untuk kerentanan potensial;
  • Identifikasi beberapa saluran pipa yang bekerja bersama untuk membangun aplikasi;
  • Buat aplikasi dengan agen berbasis cloud dan agen bangun Anda.
