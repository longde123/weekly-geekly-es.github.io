ExtremeSwitching X465交换机。 通用千兆和千兆

Extreme Networks交换机产品组合已扩展到包括ExtremeSwitching X465系列，该系列的产品以六种带有铜端口的型号为代表（预计在不久的将来会发布光学器件）。 本质上，ExtremeSwitching X465是Summit X460和Summit X460G2交换机的第三代产品，也是其逻辑上的延续。 从意义上讲，它们可以安装在网络的任何级别上，以进行访问，也可以安装在内核或分发中，因此它们是“通用的”。 之前我们写过有关“ PoE ++” ， “ Insight体系结构” ， “ Extreme Extended Edge / VPEX”的信息 -实际上，X465是第一个同时支持所有这些技术的交换机。



产品阵容包括四个千兆交换机和两个千兆交换机，所有型号均支持堆叠。 同时，只有多千兆位型号支持Insight架构，因此增加了RAM和CPU



图例：
“ T” -“ Base-T”（常规铜通道）
“ P” -PoE +（802.3at）
“ W” -PoE ++（802.3bt）
“ MU” -千兆位

高预算的PoE ++，以及多速100M / 1G / 2.5G / 5G接入端口，使您能够连接Wave 2或WiFi 6 AP，PoS终端，带旋转变焦功能的现代LED灯或IP摄像机，IoT设备或铺设连接基础设施下一代访问设备。 X465交换机还支持快速和永久PoE，某些型号的 PoE总预算高达3500W 。


X465可用作组织极端扩展边缘架构并连接V400 / V300端口扩展器的“控制桥”。 同时，第V300系列VPEX系列可以由X465 PoE ++（90W或60W）供电


例如，通过从X465交换机连接和供电48个VPEX并为其供电，并且还使用Insight架构通过防火墙+ NAT或WiFi控制器配置VM，可以显着简化网络架构，并节省电源和其他资本基础架构。


ExtremeSwitching X465的其他关键功能可以加以区分（并重复一些）：

--->安全策略边缘
1.高度可扩展的OnePolicy支持
支持2.28位/ 256位MACsec的上行链路
10/100 / 1000BaseT访问端口支持3.18位MACsec

--->物联网边缘
1.多千兆接入/ 30/60 / 90W PoE
2.扩展边缘交换控制桥
3.永久/快速PoE

--->任何校园边缘
1.面料连接->面料连接
2.通用附加->任何校园

--->可见性边缘
1.应用遥测/ sFlow
2.洞察架构

--->计时边缘
1.音频视频桥接（AVB）

--->提供者边缘
1. G.8032 / ERP
2. MPLS / VXLAN / EVPN

--->蓝牙加密狗，用于控制台访问
--->上行选项包括4 x 10Gbps，2 x 25Gbps，4 x 25Gbps和2 x 40Gbps
