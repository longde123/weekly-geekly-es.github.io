Komprimieren von Windows 10
Im Juli 2016 veröffentlichte Microsoft für Windows 10 einen neuen Komprimierungsmechanismus für Compact OS für ein Befehlszeilenprogramm compact.exe, mit dem Betriebssystemdateien direkt während des Betriebs komprimiert werden. Der neue Komprimierungsmechanismus verfügt über viele andere interessante Funktionen, ähnelt jedoch im Allgemeinen dem WIMBoot-Programm in Windows 8.1 Update 1. Das

kompakte Betriebssystem gibt mindestens 2 GB Speicherplatz frei. Nicht so sehr, aber bei kleinen SSDs macht sich dies bemerkbar.

Das Programm wird über die Befehlszeile mit Administratorrechten ausgeführt. Dies bedeutet, dass Sie nach dem Ausführen von cmd.exe die Tastenkombination Strg + Umschalt + Eingabetaste drücken sollten.

Vor dem Komprimieren von Windows 10 ist es sinnvoll herauszufinden, ob dieses Verfahren zu einem Ergebnis ungleich Null führt. Das System befindet sich möglicherweise bereits in einem komprimierten Zustand, sodass keine zusätzlichen Verfahren erforderlich sind. Führen Sie den folgenden Befehl aus, um den aktuellen Status der Betriebssystem-Binärdateien zu ermitteln:

  • compact.exe / q



Der Befehl zeigt Informationen zum aktuellen Status an: Wie viele Dateien werden komprimiert oder nicht komprimiert, wie hoch ist die Komprimierung, wie viele Bytes befinden sich im Container der komprimierten Dateien. Wenn die Komprimierungsstufe nahe bei 1,0 liegt, ist die Verwendung des Programms wirksam.

Unter bestimmten Umständen kann ein komprimiertes System die Leistung geringfügig verringern, auf den meisten Computern ist es jedoch unsichtbar. Beachten Sie, dass es immer möglich ist, Dateien in ihren normalen Zustand zurückzusetzen, wenn Sie Leistungsprobleme feststellen. In jedem Fall wird empfohlen, vor dem Komprimieren der Systemdateien ein Backup zu erstellen.

Führen Sie den folgenden Befehl aus, um die Prozedur auszuführen:

  • compact.exe / CompactOS: immer



Das Komprimieren aller Dateien sollte einige Minuten dauern, auf einigen Systemen kann es jedoch 20 oder mehr Minuten dauern. Der neueste Insider Build macht mehr als 2 GB frei.

So bringen Sie das System in den ursprünglichen Zustand zurück:

  • compact.exe / CompactOS: niemals

Der umgekehrte Vorgang ist etwas schneller.

Ein praktisches und kostenloses Programm, das Audio- und Videodateien in verschiedene Formate konvertiert.
Es gibt verschiedene Programme, die die grundlegenden Anforderungen erfüllen:

  • Open Source
  • freie Nutzung;
  • Unterstützung für viele Dateiformate;
  • plattformübergreifend;
  • Leichtigkeit.

Am bequemsten sind vielleicht zwei: Handbremse und ffmpeg (zum Beispiel mit der grafischen Oberfläche von WinFF ). Beispielsweise kann der beliebte kostenlose VLC-Videoplayer Dateien konvertieren, in den meisten Fällen ist es jedoch bequemer, ein spezielles Programm zu verwenden.

Handbremse läuft unter Windows, Mac und Linux. Die aktuellste Version für Windows ist 0.10.5. Fast jede Multimediadatei (oder jeder Ordner mit Dateien), DVD oder BlueRay, wenn dort kein Kopierschutz vorhanden ist, wird

als Quellmaterial akzeptiert . Zieldateiformate


  • Behälter : .MP4 (.M4V) und .MKV
  • Video-Encoder : H.265 (x265 und QuickSync), H.264 (x264 und QuickSync), H.265 MPEG-4 und MPEG-2, VP8, Theora
  • Audio-Encoder : AAC / HE-AAC, MP3, Flac, AC3, Vorbis
  • Audio-Pass-Through : AC-3-, E-AC3-, DTS-, DTS-HD-, TrueHD-, AAC- und MP3-Titel



Das Programm ist praktisch, um eine DVD oder eine heruntergeladene Datei für die Anzeige auf einem anderen Gerät als einem PC zu transkodieren. Handbrake verfügt über integrierte Codierungsprofile für die folgenden Geräte:

  • iPod
  • iPhone und iPod touch
  • iPad
  • AppleTV
  • AppleTV 2
  • AppleTV 3
  • Android
  • Android Tablet
  • Windows Phone 8

Das Programm hat viele zusätzliche Funktionen.

  • Auswahl von Fragmenten (Teile der Quelldatei zur Codierung)
  • Stapelverarbeitung und Anforderungswarteschlange
  • Kapitelmarkierungen
  • Untertitel (VobSub, Untertitel CEA-608, SSA, SRT)
  • Konstante Bitrate oder konstante Qualität (variable Bitrate)
  • Unterstützt VFR, CFR und VFR
  • Videofilter für die Transcodierung: Deinterlacing, Decomb, Denoise, Detelecine, Deblock, Graustufen, Zuschneiden, Skalieren
  • Live-Vorschau

Der größte Teil des Quellcodes für Handbrake wird unter der kostenlosen GNU GPL, Version 2, verteilt. Einzelne Teile unterliegen der BSD 3-Klausel.

So wählen Sie ein kostenloses VPN
Für anonyme Arbeiten im Internet und die Umgehung der staatlichen Zensur (Blockierung von Websites) ist es bequem, ein kostenloses VPN zu verwenden. Eine der beliebtesten Optionen ist FreeOpenVPN (über die OpenVPN-Clientanwendung).

Übrigens wurde am 31. Juli 2016 Kaspersky Internet Security 2017 (ein in Russland beliebtes Antivirenprogramm) veröffentlicht, das einen VPN-Client enthielt. Im kostenlosen Modus funktioniert es nur über Server in Russland und ermöglicht Ihnen die Übertragung von 200 MB pro Tag. Dies ist natürlich ein eher schlechtes Angebot, aber wenn Sie KIS 2017 bereits installiert haben, sollte diese Option berücksichtigt werden. Nur KIS-Benutzer sollten die kostenpflichtige VPN-Funktion im Programm nicht aktivieren, da es viele kostenlose Alternativen gibt.
