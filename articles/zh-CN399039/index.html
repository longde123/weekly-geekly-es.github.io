熊猫在企鹅之中。x86_64 LattePanda微型计算机

下午好，《极客时间》亲爱的读者们！出于命运的考虑，我碰巧拿到了一块非常有趣的板子-LattePanda。


她的有趣之处是什么？


至少，这是Windows x86_64，是Arduino兼容的组合（直接在一块板上），并输出到许多传感器。如果要在ARM SoC / Linux / GPIO解决方案上组装相同的套件，它们最终将看起来像是一团电线，这是现代控制领域的一种极不寻常的组合。

如果Windows是您的家，那么这里就是微型计算机世界中真正的核心。我将尝试在这个平台上写对我来说有趣的东西。

进一步的特征和我的印象：

首先-这是尺寸。这是微型计算机的标准板，可能比RPi多一点。对这种格式的期望丝毫没有减损，但是对于成熟的x86_64来说，这是完全不寻常的。没有风扇，电源，所有拿着x86_64的著名品牌的微型计算机板的人都熟悉和熟悉。


里面是Intel Atom x5-z8300。根据规格，它在1.44 GHz上具有不俗的芯片，并且跳跃时可高达1.84。另外，所有这些都带有良好的x86_64-2Mb高速缓存，2-4 GB的内存，这非常好。图形芯片是Intel HD 5300，甚至可以拉出4k（我尝试过）。速度为30赫兹时，但这是HDMI的限制，它将不再通过。通常1080p60Hz不会构成问题。铁不仅仅是固体，而且可能会令其惊讶。特别是如果您经常使用ARM解决方案，并且对它们有什么期望。

所有这一切，最简单的型号都非常实惠，仅需79美元-2Gb内存，32Gb eMMC，无需Windows许可证，最高可达139美元。

如果您喊“这很抢劫”，请注意-这是一台功能齐全的计算机。它是一整套运行Windows，arduino，wi-fi的电子设备，实际上是一台成熟的计算机，准备接受编程微控制器的初学者以及具有自己的草图和外围设备的任何专业人士的使用。

如果仔细研究一下这些特性，您很快就会得出结论，现代微型计算机（不包括许多其他东西）所需的一切都已经准备就绪。甚至存在许多人不愿意设置的Wi-Fi，并提供了多种颜色的加密狗。只需接上电源，然后单击电源（甚至还有两个硬件按钮-电源和重置，非常方便！），将其打开即可工作。


但这还不是全部。


最后一件有趣的事是，不仅在板上，GPIO已经成为标准，而且对于初学者来说，有7个用于传感器的连接器。全套也可用。场景只是机器人制造商的梦想-存在，气体，火灾，温度，震动传感器，按钮，灯泡甚至一堆东西...


最有趣的是-板上有一个带20个输出的Arduino Leonardo处理器。也就是说，无需购买任何东西，您就可以立即使用Arduino-装有梳子，甚至还有灯泡。我完成了任何新手arduino驱动程序的任务-眨眼了。

所有这些都非常能胜任包装和离婚，并在散热器下卷起。建造质量不错。尽管配备了三毫米厚的微型冷却器，但它无需风扇即可工作。好吧，我不知道，我不需要它，即使在负载很大的情况下，电路板也不会发红（我在Windows上安装了更新程序以进行检查）。最后，不是在玩玩具吗？顺便说一下，这是一个相当可行的任务，视频芯片具有DirectX 11.2的功能，还具有4k / HEVC解码功能。也就是说，有关视频的所有信息都没有任何问题。 YouTube，Netflix和YouTube的所有所有者，都被ARM媒体中心所错过。

想象一下，所有这些-这绝对是编程或微控制器，机器人，智能房屋，直升机或其他通常需要整个动物园的东西所需要的整套设备-传感器控制器，arduino，通常是附加的无线-fi dongle ...

食物？标准微型USB。甚至MacBook端口对我来说就足够了，但是我并不特别疯狂。坦率地说，这对英特尔来说是一大进步。

操作系统



天哪，是Windows（我已经九年没见过）了！版本10。我不能说它是好是坏，它与我上次看到的XP有很大不同。但是... Windows。直接交付给eMMC，打开电源后即可立即工作，无需进行任何设置。我注意到的是，设备管理器中的所有设备均已定义。没有一个身份不明的人。好像微软自己支持公司一样。也就是说，一个完全现成的套件-一切正常。所有固件，eMMC控制器，视频适配器，所有组件均已安装，所有驱动程序均已安装最新更新。通常的情况类似于Windows-x86_64，但在ARM-Linux解决方案的状态中相当出色，在这种情况下，经常会缺少某些东西，而当社区兴起时，某些事情会在以后出现。

加载后直接在桌面上显示Arduino Studio图标（已预安装Windows并已对其进行配置，请勿OEM填充到欢迎屏幕）。与Leonardo的连接-COM端口。我尝试了几个标准草图-一切正常，指示灯闪烁。即使重设也不会重设，而是继续工作。即使关闭电源，草图也不会消失，即使在操作系统开始加载之前，草图也不会消失。

结论很短，并不华丽-一切开箱即用，将其打开并开始。但是想想看，您刚刚听到这种关于开发板出现在市场上的话有多久了？我认为这无疑是Windows储钱罐的一大优点。

BIOS？可信计算？安全启动？锁定供应商？


BIOS到位。美国大趋势AMIBIOS就像大型生物一样。你可以进去，改变任何东西。没问题下载-UEFI。没有供应商锁，您可以启动进入Linux。我尝试了ubunt 16.04的USB闪存实时版本-它可以工作。这里确实有一个微妙之处：2Gb / 32Gb eMMC是32位UEFI，据说可以加载linux来尝试。我的版本是4Gb / 62Gb，一切都很好，没有问题。

Windu可以根据需要进行重新排列，图像位于制造商的网站上，没有什么可以阻止的。


Linux并非没有古怪之处。设备列表很短，显然不是所有设备都在其中。 SD卡不可见（它在BIOS中也不可见，您无法从其中引导，它根本不存在，直到Windows才存在。efi命令行也消失了，这是出乎意料的。它可以在Windows中正常工作而没有任何细微差别）。 Wi-fi不可见。不仅是没有模块的未知芯片，而且通常看不到设备。通常，在我看来，Linux在该微型计算机上的系统非常简单。它显示了桌面（也是4k，在其中我通常无法将分辨率更改为1080p），它可以在Internet上，但是仅此而已。 eMMC存储可见且可访问。也就是说，出于强烈的愿望，您可以将Linux直接放在内部磁盘上。尽管我没有尝试过，但新的Ubuntu只是隐藏了我无法识别的内容。

Windows和Linux都有细微差别-即使根本不存在MIPI-DSI屏幕，总会有一个小屏幕。此连接没有即插即用功能，因此在正常操作下始终处于活动状态。但这并没有特别麻烦；使用软件工具，您可以关闭图像输出并忽略它。

结论


尽管事实上他似乎并没有为一个不寻常的项目写任何东西，但事实证明很多。一切似乎有点。

我喜欢这个装置吗？是的，我真的很喜欢这样的事实，那就是发烧友需要一套完整的东西。您无需购买任何东西，无需用电线将其固定，连接，在桌子上造成混乱（例如，在阁楼上，您可以创建一个智能房屋）。

当然，如果您是高级怪胎（例如，我-是的），并且拥有LattePanda可以提供的所有东西，并且更便宜-RPi，Arduino和速卖通价格为150卢布，传感器为一束价格为50卢布，接线板，灯泡和伺服电机-您有权说这是一匹马的价格。但是冷静一秒钟，坐在椅子上，我也拥有所有这些。我也有一束电线，一整盒电路板，房间里没有足够的插座，但是在连接器的USB集线器中……但是如果我三年前有选择-LattePanda，或者我现在只剩下这些了，我会选择一个。因为很实用。如果有人告诉我，有时由于Amlogic我有时不得不坐在linux 3.10上，那么我现在可能会将所有硬件都改为LattePanda。但是有人站在他的道路的最开始，并且恰好问这些问题。
