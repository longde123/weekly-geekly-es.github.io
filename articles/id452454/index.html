Masuk ke Azure DevOps menggunakan kredensial GitHub Anda

Di semua Microsoft, kami fokus pada pemberdayaan pengembang untuk membangun aplikasi yang lebih baik, lebih cepat. Salah satu cara kami mencapai itu adalah dengan menyediakan berbagai produk dan layanan yang mencakup semua tahap siklus pengembangan perangkat lunak. Ini termasuk alat IDEs dan DevOps, platform aplikasi dan data di cloud, sistem operasi, solusi Inteligen Buatan dan IoT, dan banyak lagi. Semua ini berpusat di sekitar pengembang, baik sebagai individu yang bekerja dalam tim dan organisasi, dan sebagai anggota komunitas pengembang.


GitHub adalah salah satu komunitas pengembang terbesar, dan bagi jutaan pengembang di seluruh dunia identitas GitHub mereka telah menjadi aspek penting dari kehidupan digital mereka. Menyadari hal itu, kami senang mengumumkan peningkatan yang akan membantu pengguna GitHub memulai lebih mudah dengan layanan pengembang kami, termasuk Azure DevOps dan Azure.



Kredensial GitHub Anda sekarang dapat login Anda ke layanan Microsoft


Hari ini, kami memungkinkan pengembang untuk masuk dengan akun GitHub mereka yang ada ke layanan online Microsoft, pada setiap halaman login Microsoft. Menggunakan kredensial GitHub Anda, sekarang Anda dapat masuk melalui OAuth di mana pun akun Microsoft pribadi, termasuk Azure DevOps dan Azure.


Anda akan melihat opsi untuk masuk dengan akun GitHub Anda dengan mengklik "Masuk dengan GitHub".


Setelah masuk ke GitHub dan mengotorisasi aplikasi Microsoft, Anda akan mendapatkan akun Microsoft baru yang ditautkan dengan identitas GitHub Anda. Selama proses ini, Anda juga memiliki kesempatan untuk menautkannya ke akun Microsoft yang ada jika sudah memilikinya.


Masuk ke Azure DevOps


Azure DevOps menawarkan serangkaian layanan bagi pengembang untuk membantu mereka merencanakan, membangun, dan mengirimkan aplikasi apa pun. Dengan dukungan untuk otentikasi GitHub, kami membuatnya lebih mudah untuk memulai dengan layanan Integrasi Berkelanjutan dan Pengiriman Berkelanjutan (Pipa Azure); perencanaan tangkas (Papan Azure); dan penyimpanan untuk paket pribadi seperti modul untuk NuGet, npm, PyPi, dll (Azure Artifacts). Suite Azure DevOps gratis untuk individu dan tim kecil hingga lima.


Untuk memulai Azure DevOps menggunakan akun GitHub Anda, klik "Mulai gratis menggunakan GitHub" di halaman Azure DevOps .



Setelah Anda menyelesaikan proses masuk, Anda akan dibawa langsung ke organisasi Azure DevOps terakhir yang Anda kunjungi. Jika Anda baru di Azure DevOps, Anda akan mendarat di organisasi baru yang dibuat untuk Anda.


Akses semua layanan online Microsoft


Selain mengakses layanan pengembang seperti Azure DevOps dan Azure, akun GitHub Anda dapat digunakan untuk mengakses semua layanan online Microsoft, dari Excel Online hingga Xbox.


Saat mengautentikasi dengan layanan tersebut, Anda dapat menemukan akun GitHub Anda setelah mengklik "Opsi masuk".



Komitmen kami terhadap privasi Anda


Ketika Anda pertama kali menggunakan akun GitHub untuk masuk dengan Microsoft, GitHub akan meminta izin untuk merilis informasi profil Anda.


Jika Anda setuju, GitHub akan membagikan alamat email pada akun GitHub Anda (baik publik maupun pribadi) serta informasi profil, seperti nama Anda. Kami akan menggunakan data ini untuk memeriksa apakah Anda sudah memiliki akun di kami atau untuk membuat akun baru jika tidak. Menghubungkan identitas GitHub Anda ke Microsoft tidak memberikan Microsoft akses ke repositori Anda di GitHub. Aplikasi seperti Azure DevOps atau Visual Studio akan meminta akses ke repositori Anda secara terpisah jika mereka perlu bekerja dengan kode Anda, yang harus Anda setujui secara terpisah.


Meskipun akun GitHub Anda digunakan untuk masuk ke akun Microsoft Anda, mereka masih merupakan entitas yang terpisah - yang satu hanya menggunakan yang lain sebagai metode login. Perubahan yang Anda lakukan pada akun GitHub Anda (seperti mengubah kata sandi atau mengaktifkan otentikasi dua faktor) tidak akan mengubah akun Microsoft Anda, dan sebaliknya. Anda dapat mengelola koneksi antara GitHub Anda dan identitas Microsoft di halaman manajemen akun Anda, di bawah tab Security.


Mulai jelajahi Azure DevOps sekarang


Buka halaman Azure DevOps dan klik "Mulai Gratis dengan GitHub" untuk memulai.


Jika Anda memiliki pertanyaan, lihat halaman dukungan ini . Beri tahu kami pendapat Anda dalam komentar di bawah. Seperti biasa, kami akan senang mendengar tanggapan atau saran yang Anda miliki.
