[Terjemahan] Dari pembajakan hingga sumber terbuka: kisah saya

Halo, Habr! Saya membawa kepada Anda terjemahan dari artikel " Dari pembajakan ke open-source: cerita saya " oleh Jakob Attkinson.

Saya tumbuh dengan komputer sejak berusia 6 tahun. Meskipun saya dilahirkan dalam keluarga yang agak miskin, ayah saya sadar akan potensi teknologi yang kuat dan ingin memastikan bahwa putranya memiliki segalanya untuk pengembangan. Namun, sebagian besar hal yang diperlukan untuk komputer dibayar, dan orang tua tidak dapat mengizinkannya juga. Jadi, sekarang beritahu saya: mengapa PC bagus untuk anak-anak jika tidak ada cara untuk memainkan semua game dan menonton animes yang Anda inginkan?

Jadi saya masuk pembajakan, bahkan sebelum saya berusia 8 tahun. Dan ketika saya berusia 12 tahun, saya sudah memiliki seluruh koleksi aplikasi dan file bajakan, yang sebagian besar dilabeli dengan indah dengan sampul berwarna dan populer. Setiap kali, seluruh distrik menjangkau saya ketika sesuatu yang baru dirilis. (Catatan: Saya tidak pernah mencari untung atau meminta uang untuk perangkat lunak bajakan.)

Ketika saya berusia 14 tahun, game World of Warcraft dirilis. Berlangganan game berbayar, bukan? Apa yang terjadi dengan Blizzard? Saya tidak mampu membelinya, dan saya juga tidak bisa meretasnya ... Apa yang bisa saya lakukan?

Jadi saya masuk ke modding, custom server dan hal serupa lainnya. Akibatnya, saya menghabiskan waktu membaca dan mencoba menulis naskah alih-alih meminta orang tua saya untuk membayar langganan bulanan.

Singkatnya, pada saat saya memasuki universitas, aturan tidak tertulis tentang " retas atau tulis skrip " sudah mengakar kuat dalam diri saya. Tapi segera semuanya berubah ...

Pada tahun pertama saya, saya berteman dengan siswa lain yang mengambil sebagian besar tanggung jawab organisasi, tetapi dia tidak pernah pergi ke pesta yang menyenangkan yang begitu menarik dan menarik bagi siswa muda. Seorang teman baru tumbuh dalam keluarga di mana pembajakan adalah konsep yang tidak bisa mereka terima. " Jika Anda membutuhkan sesuatu yang orang lain kerjakan, beli atau tukarkan dengan layanan lain ," katanya.

Saya pikir orang hanya dapat membayangkan keterkejutannya ketika dia pertama kali datang ke rumah saya dan menyadari bahwa tidak ada yang "legal" di pondok ini, bahkan tidak ada lisensi Windows.

Butuh seorang teman sekitar satu tahun untuk membuka mata saya. Dia mengubah sikap saya terhadap perangkat lunak. Dalam waktu kurang dari satu tahun, saya menghapus semua program yang diinstal atau digunakan yang tidak memiliki lisensi yang valid. Saya menyumbangkan koleksi barang bajakan kepada seorang teman lama (ya, saya tahu ... saya harus membuangnya ...) dan mulai menjadi sukarelawan untuk afiliasi Microsoft lokal untuk mendapatkan lisensi Windows pertama saya (semua sukarelawan mendapatkannya saat itu). Saya meninggalkan suite Office dan belajar menulis materi sekolah saya di LaTeX. Dia juga berhenti bermain game yang tidak bisa dimainkan secara gratis, dan memutuskan bahwa " jika perangkat lunak tidak memiliki alternatif gratis, tetapi saya tidak mampu membayar, maka saya mungkin tidak perlu perangkat lunak seperti itu juga ."

Pada tahun-tahun berikutnya, saya melangkah lebih jauh. Setelah 1 bulan di pekerjaan pertama saya, saya memperbarui kontrak untuk memasukkan IntelliJ IDEA untuk lisensi rumah, karena perusahaan tidak lagi menggunakan Eclipse, dan bersikeras bahwa lingkungan pengembangan pribadi saya selaras dengan lingkungan kerja. Saya bahkan meluncurkan startup untuk melakukan sesuatu yang lebih terkait dengan teknologi di waktu luang saya, karena banyak perusahaan menawarkan diskon (dan bahkan lisensi gratis) dalam 1-2 tahun pertama kehidupan startup.

Selain itu, saya menghabiskan 2 tahun bekerja sebagai sukarelawan di komunitas game dan, dengan satu atau lain cara, mencoba memberikan uang untuk semua game yang saya terima sebagai anak melalui pembajakan. Dengan hati dan jiwa saya, saya mencurahkan seluruh perhatian saya untuk melakukan sebanyak mungkin selama waktu ini.

Saya suka percaya bahwa beberapa hal yang saya sentuh menjadi lebih baik dari waktu ke waktu, dan di masa depan saya akan mengubah setidaknya beberapa hal kecil menjadi lebih baik.

Apa hubungan semua teks padat ini dengan sesuatu yang berharga? , - Anda bisa bertanya pada diri sendiri apakah Anda punya kesabaran untuk membaca artikel ke tempat ini. Jawabannya sederhana: Saya akan bekerja dengan open source.

Meskipun saya belum berkontribusi pada Github, saya telah menyimpulkan bahwa sebagian besar alat yang digunakan adalah open source baru-baru ini. Ada hal-hal yang saya pikir "keren," tetapi belum ditemukan di Github. “Mengapa tidak memulai proyek?” - pemikiran ini muncul beberapa hari yang lalu. Tetapi kemudian ketakutan dan perasaan negatif lainnya muncul. “Saya bukan pengembang yang baik seperti orang lain dari github ini”, “Saya tidak tahu harus mulai dari mana” , dll. Meskipun saya masih belum sepenuhnya memahami apakah ini adalah alasan atau benar-benar masalah, saya memutuskan bahwa akan tepat untuk membagikan niat saya di sini dan secara bertahap mulai bertindak.

Semoga posting selanjutnya adalah tentang memulai dengan open source. Atau mengapa saya masih tidak.

Dan bagaimana Anda, pembaca Habr, menjauh dari pembajakan? Atau masih tidak, tidak, tidak, tidak retak? Apa yang memotivasi Anda untuk membayar perkembangan orang lain ketika mereka dapat ditemukan secara gratis?
