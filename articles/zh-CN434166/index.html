联想YogaBook C930：可一次更换四个小工具的设备

计算机是不同的：预算，游戏，尽可能的便携式，设计用于图形和视频等。 如果您事先知道该类别，那么几乎可以立即了解与之相关的设备的主要功能。 但是有时市场上会出现一些非常特别的东西，而我们今天想告诉您这种独家商品。 我们将讨论我们的YogaBook C930，它可以一次替换四个小工具：一台平板电脑，一台笔记本电脑，一本电子书，以及一本笔记本电脑和一个用于画册的混合笔记本电脑。 有意思吗 然后欢迎来到凯特！


很漂亮很轻


联想YogaBook C930的尺寸可以与普通的A4纸张相比较，后者在面积上明显不及我们的小工具。 结合9.9毫米的厚度和775克的重量，您将获得一个真正紧凑的设备，可以与拥有者一起随身携带。 您甚至不会觉得背包有这么大的负担，而且YogaBook C930可以毫无问题地适合大多数女士手提包。 要记住的主要事情是，最好避开附近有小物件的地方，并在将设备放在薄盒或单独的隔间中时存放设备。

在外观上-简洁的规则。 YogaBook C930的外壳完全由金属制成。 它的表面是哑光的，颜色是深灰色，因此即使保留指纹也几乎看不见。


小口音进一步强调了诱人的设计。 我立即要仔细观察的第一件事是品牌铰链，它由许多部分组成，与表带非常相似。


YogaBook C930顶部还有闪亮的Lenovo徽标。 它很小，位于角落，根本不引人注目。


该案并非没有一些棘手的“筹码”。 为了抬起主机盖，您无需将其拿起并尝试将一个部件与另一个部件分开。 取而代之的是，您只需要轻敲盖子两次，它就会自动抬起。 此外，将其向后倾斜并按需要的角度放置仅是一个简单的动作。 方便吗 绝对可以。 有效吗 更喜欢！

如果我们将联想YogaBook C930的尺寸与两年前发布的联想YogaBook的尺寸进行比较，则第二款产品更紧凑，更轻85克。 但是，性能尤其是功能方面的差异非常明显。 我们稍后再讨论，但是现在您可以在附近查看两个小工具-一个新的和一个旧的。


连接器很少，但是您可以扩展内存并使用SIM卡


紧凑的尺寸和小的厚度不仅影响了YogaBook C930侧面连接器的数量。 此外，从形式上讲，该设备不适用于笔记本电脑，但适用于平板电脑。 平板电脑很少设计用于连接大量外围设备。

但是，在设备的每个侧端都有一个USB C型连接器。 合理的做法是，左一个用于充电，右一个用于将耳机与相应的连接器连接。 您也可以将适配器从Type-C接到传统音频插孔，或者在耳机本身允许的情况下通过蓝牙连接。


除了USB之外，扬声器格栅还位于两端，实际上有两个格栅：YogaBook C930具有真实的立体声。 当然，在评估声音时，您需要在小巧的尺寸上打折。 这几乎不是听音乐的最佳解决方案，但是看电影还是很不错的。


电源和音量按钮位于设备的右侧。 像智能手机一样，在左侧有一个拉出式托盘。 它有两个插槽。 一个用于microSD卡，另一个用于nanoSIM卡。

因此，您不仅可以轻松地扩展内置数据存储的容量（卡的最大支持容量为512 GB），而且还可以使用移动运营商的资费在Wi-Fi网络之外的Yogabook上使用Internet。


最后，还有一件不能单独提及的事情：YogaBook C930的工作面板上是指纹扫描仪。 进入系统时，它可以方便地替代密码。 识别快速准确。


一个小工具中有两个完全不同的屏幕


因此，我们逐渐接触到了主要的“芯片”联想Yoga瑜伽C930-这是基于电子墨水的附加显示器。 在这里，通常是经典笔记本电脑中的键盘放置位置。 默认情况下，此处显示的是虚拟按钮，由于表面是触摸感应的，因此用户可以单击它们进行打印。 电子墨水屏幕的分辨率为1920×1080像素，刷新率很高，传感器准确而迅速地记录了用户手指的触摸。


您也可以切换到其他模式。 为此，在基于电子墨水的显示菜单中提供了四个项目。 第一个显示键盘上的键盘，第二个显示注释或图纸的字段，第三个将YogaBook切换到电子书阅读器模式，第四个打开“电子墨水”显示设置菜单。


因此，与第一本YogaBook相比，有许多令人印象深刻的差异。 回想一下，在其工作面板上，默认情况下已标记键盘触摸按钮。 在事先将一张普通纸放在工作区上之后，您只能在里面有墨水笔的特殊笔的帮助下在此处绘制或记录笔记。 新的小工具为想象力和创造力提供了更大的范围。


联想YogaBook C930的主屏幕也不会令人失望。 它的对角线完全相同，为10.8英寸，但分辨率更高：2560x1600像素。 细节真是太棒了。 另外，我们在此处添加了（虽然不是记录），但具有350尼特的高亮度水平和出色的色彩再现性：sRGB色彩空间的覆盖率明显高于最大值：高达144％。 视角很宽，无需抱怨。 涂层虽然有光泽，但由于在保护玻璃和基体之间没有气隙，因此不会带来很大的不便。


好吧，当然，这个显示器也是触摸感应的。 对于与其一起使用以及与E-Ink显示器进行交互，同样适用的有源笔-Lenovo Precision Pen支持4096级压下。 它不包含在我们测试配置的交付包装中，但钢笔放在带有两个较旧版本的YogaBook C930的盒子中。 如果您单独购买精密笔，则将花费6,990卢布。


四个用例


紧凑的机身，完全可倾斜的屏幕和两个触摸显示屏的结合为Lenovo YogaBook C930用户提供了许多不同的可能性。 实际上，事实证明一台设备可以完全或部分替换四个不同的小工具。 简要描述所有用例。

联想YogaBook C930作为笔记本电脑

这里的一切都很简单而且非常明显：机载Windows 10操作系统，因此您可以执行与常规PC相同的一切，并根据硬件功能进行调整。 但是，没有熟悉的键盘，而是由电子墨水显示屏上带有虚拟按钮的触摸面代替。

有两种颜色可供选择：带有白色轮廓的黑色背景上的黑色按钮或带有黑色背景的白色按钮。 在这种情况下，您可以选择带或不带触摸板的选项。 在第二种情况下，可以通过按特殊的虚拟键随时显示它。


所有这些看起来都不寻常，但是在这样的表面上打印是否方便？ 当然，与传统的按钮键盘相比，打印速度至少降低了两倍，并且由于缺乏触觉响应，您需要经常查看按钮和触摸板。 但是，我们的员工已经积极使用YogaBook C930数周，他们的经验表明，随着时间的流逝，您会逐渐习惯它。 当然，很难推荐使用虚拟键盘作为键入大文本的永久工具。 但是，作为在商务旅行，聊天和网上冲浪中回信的一种解决方案-确实如此。

联想YogaBook C930为哭

YogaBook C930的主屏幕可以随时向后折叠，从而将设备变成平板电脑。 在此，设备体积紧凑的优势就显得尤为重要。 回想一下，重量为775克，厚度不超过9.9毫米。 双手握住小工具的同时观看视频或浏览网页非常方便。


联想YogaBook C930作为电子书

使用YogaBook C930，无需随身携带单独的电子书阅读器。 在相同的数位板模式下，可以在电子墨水显示屏上阅读较长的文本，而不会伤害眼睛。 由于具有高分辨率，字母的细节非常好，单击附加显示的右侧或左侧即可翻动页面，还可以更改比例。 为了获得完全的幸福，也许仅一个背光不足以在黑暗中阅读，但其实现将不可避免地影响设备的厚度和重量。 支持的格式列表取决于软件版本。 PDF立即打开，没有任何问题，不久我们将添加所有经典的电子书格式。


联想YogaBook C930作为笔记本和相册

可能是最有趣的模式，要全面披露其功能，您将需要可选的Precision Pen。 但是，如有必要，您可以用手指在基于电子黑的显示器上进行绘图和书写。

在便笺模式下，您可以在单元格或标尺中选择具有特定背景的现成模板之一，例如笔记本。 有橡皮擦和选择笔粗细的能力，以及“智能笔刷”功能，该功能会自动为手绘形状和线条提供正确的形状。 可以选择并复制图像片段；您还可以将屏幕截图从IPS主显示屏传输到便笺，然后直接在其上做便笺。 简而言之，现在所有用于记录和绘图的基本功能都已就绪。 将来，我们将尝试扩展和改进所有这一切。


适用于所有日常任务。


联想YogaBook C930共有四种配置。 它们在处理器，SSD驱动器上的内部存储器数量，是否支持LTE以及包装中是否包含Precision Pen方面有所不同。 其余参数相同。 以下是完整列表：

  • Intel Core m3-7Y30，128 GB SSD，仅Wi-Fi
  • Intel Core m3-7Y30，SSD 128 GB，支持LTE
  • Intel Core i5-7Y54、256 GB SSD，仅Wi-Fi，包括Precision Pen
  • 英特尔酷睿i5-7Y54、256 GB固态硬盘，支持LTE，包括Precision Pen

我们的测试示例是上面的第一配置，但是Core m3-7Y30和Core i5-7Y54之间的计算能力差异并不明显。 而且就规格而言，这全部归结为基准频率（1.0与1.2 GHz）和最大频率（2.6与3.2 GHz）和TDP（3.5与3.75 W）之间的差异。 此外，i5-7Y54具有更高的集成显卡Intel HD Graphics 615：950和900 MHz。


当然，该计算机不是用于游戏，编辑，或者特别是用于处理三维图形的严肃事情。 该设备用于网络冲浪，聊天，阅读，观看视频，文档和演示文稿。 在m3的情况下，i5将同样稳定地处理其工作。

为了以防万一，我们还提供了Cinebench R15基准测试和3D Mark Sky Diver基准测试中几乎所有2018年“顶级”超极本中使用的Intel Core m3-7Y30与Intel Core i7-8550U芯片的比较。



当然，两个处理器都在很短的时间内保持Boost模式下的最大频率。 在长时间的负载下，它们会迅速降至基准水平。 但是同时，在日常工作中几乎不可能将YogaBook加热到很高的温度。 小工具底部的温度很少会超过40-42度。 这是在没有主动冷却的情况下，因此是绝对的安静。

将Megafon SIM卡安装到适当的托盘中后，Speedtest显示下载速度接近120 Mbps，上传速度约为5 Mbps。 它在莫斯科的东南郊。 实际中假设的最大速度可能更高-这完全取决于操作员的能力。 每天使用带有SIM卡的平板电脑，一切正常，信号接收的质量得到了保证。


Windows 10操作系统的速度没有重大问题，甚至4 GB的RAM也不会成为瓶颈。 当然，最好使用8 GB，但所有配置中只有4 GB。 这很可能就是在测试PC Mark 10系统的整体性能时结果未达到3000点的原因。


我们还在CrystalDisk标记测试中测量了SSD驱动器的速度，这是我们得到的结果。 同样，这些并不是破纪录的价值，但对于上面列出的所有日常工作场景来说绝对是舒适的。


它工作时间足够长，充电很快



联想YogaBook C930内部装有一块4600 mAh的电池。 根据全球团队的测试，一次充满电的电池寿命略超过8.5小时。 在现实生活中，出现这种情况的原因是，例如，连续播放具有最大亮度的Full HD视频以及禁用Wi-Fi会使设备放电7-7.5小时。 对于这样的紧凑和薄型设备非常好，而且不要忘了这里的主屏幕分辨率不是1920x1080，而是所有2560x1600像素。


如果您关闭主屏幕并仅使用黑白来阅读电子书或PDF文档，则YogaBook C930的工作时间甚至更长：长达12-13小时。 当然，独立的电子墨水“阅读器”的工作时间更长，但是没有这种多功能性。 这是从莫斯科到纽约或东京的直航的充足供应。

Lenovo YogaBook C930使用USB Type-C线从最常见的适配器充电。 在尺寸和重量上，它绝对可以与大多数智能手机充电产品相媲美。 而且，当然，其他Type-C移动设备也可以通过Yogabuk适配器充电。 是的，这个适配器上的小工具本身可以轻松充电：从0到90％电池电量指示器的路径在不到1.5个小时内通过。

多少钱，在哪里买？

如果您不需要Precision Pen，或者打算以后再购买，则可以选择具有128 GB内存和Intel Core m3的较小版本。 有了LTE支持，YogaBook C930的售价为97,990卢布，而没有访问运营商网络的最基本型号的售价为87,990卢布。 这些价格将在新年后的增值税增加时具有相关性，目前略低。

异议“是的，为了钱我会给自己买一本很棒的超极本！” 听起来很合理，但是技术的选择始终取决于个人需求。 是的，ultrabook功能更强大，具有更多的内存，但它并不那么紧凑且通用性较低。 在公司内部，我们将YogaBook C930视为主要用户，而不是主要的用户计算机。

例如，如果您不需要定期键入大文本，那么它将完全替代旅行中的常规笔记本电脑。 除了出色的视频主屏幕外，还有一个出色的附加电子墨水显示屏可供阅读。 好吧，如果您定期对某些电子布局进行更正，那么YogaBook C930可能会成为您工作的主要工具之一。 对于仍然习惯于书写，绘图和绘图的人来说，这几乎是一种唯一的解决方案。

当然，我们也知道，高昂的价格不会使这款设备变得庞大。 像第一本YogaBook一样，这也是一个小众的解决方案，而且等级很高。 同时，我们为能够向市场提供这种非标产品感到高兴和自豪，听到了用户的意见并了解了如何进一步开发该产品线。
