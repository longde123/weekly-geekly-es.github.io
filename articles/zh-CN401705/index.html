强大的孩子-Aneng AN8001 TrueRMS万用表

几年前，无法想象具有自动量程切换功能的TrueRMS万用表能够以6000美元的读数和0.5％的精度测量直流和交流电压，直流和交流电流，电阻，电容和频率，而价格不到15美元。今天，这个设备在我手中。


当我的万用表由于某种故障而不再精确后，我打算再次购买该产品，但我发现同一芯片上的小型Aneng AN8001器件也同样具有精确性。

测得的参数，范围和精度：

抵抗力：600（0.5％+ 3），6K / 60K / 600K / 6M（0.5％+ 2），60M（1.5％+ 3）;
直流电压：6V / 60V / 600V / 1000V（0.5％+ 3），600mV（1.0％+ 10）;
直流电流600uA / 6000uA，60mA / 600mA，10A（1.5％+ 3）;
交流电压：6V / 60V / 600V / 750V（1.0％+ 3），600mV（3.0％+ 3）;
交流电流600uA / 6000uA，60mA / 600mA，10A（1.5％+ 3）;
频率：5/50/500 / 5k / 50k / 500k / 5M / 10M（0.1 + 3）
电容：10nF（5％+ 20），100nF / 1uF / 10uF / 100uF（2％+ 5），1000uF / 10000uF（ 5％+ 5）；
尺寸和重量：130 * 65 * 30mm，200克;
营养：2xAAA。

实际上，万用表比所描述的要精确得多。

我使用AD584上的基准电压源测试了直流电压测量。


在第一列中，参考电压的值，在第二列中-万用表的读数。

9.99724 9.99（偏差0.07％）
7.49970 7.49（偏差0.13％）
4.99729 4.996（偏差0.03％）
2.49909 2.498（偏差0.04％）

交流电压在稳定器Calm的输出处。


精度约为0.1 V（在标准帮助下检查的设备显示比实际电压小1伏，该设备与该设备之间的差恰好为1伏）。

与同一芯片上的Holdpeak HP890CN相比，Aneng AN8001非常小。


与大型设备不同，小型万用表没有橡胶外壳，但是有一个支架。


其中包括探针，说明书和带有强烈化学气味的袋子。


探针配有盖，可以覆盖探针的几乎整个尖端。


按住保持12秒钟可打开背光灯。


在打开触点后，还有一个声音拨号盘会再发出0.5秒的吱吱声。该设备非常快-每秒3次测量。 15分钟后，将触发自动关闭电源（首先发出一声尖叫，然后发出警告，如果您不按按钮，则在一分钟后关闭）。

您不会嘲笑我，但是在仔细研究了该设备后，昨天我又订购了一个“备用”设备。

买了14.78美元，这里免费送货。卖方给出了正常的跟踪代码，万用表在不到三周的时间内就到达了。

ps仍然有一个带温度计的AN8002型号，但AN8004不值得-它在另一个芯片上。 Aneng AN8001有其他制造商的两个孪生兄弟：ZT101和RM101。

PPS 审查该设备具有大量内部照片，并演示了所有参数的测量结果。

ppps由于现在我有了稳定器和精确的万用表，因此可以通过重新测量参考灯的光通量来改善Lamptest上白炽灯的等效计算。
