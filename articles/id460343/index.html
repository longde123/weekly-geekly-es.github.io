Kisah bagaimana pengembangan game telah menjadi bagian dari hidup saya

Halo Nama saya Maxim Gazizov. Sebelumnya saya menerbitkan sebuah posting di HabrÃ© tentang keberhasilan dan kesalahan dalam game dev. Dan kemudian, setelah setahun bekerja pada gagasan saya yang disebut Perang Wasteland, saya menghilang dari udara. Itu karena prosesnya sangat menyeretku seperti sebelumnya. Ngomong-ngomong, ternyata saya kehilangan akun saya sebelumnya dan sekarang membuat yang baru.



Hidup sebelum proyek Anda sendiri


Sebelumnya, saya terlibat dalam pengembangan aplikasi mobile, proyek-VR dan sistem yang sangat dimuat. Hari ini saya benar-benar mengabdikan diri untuk mengembangkan permainan saya. Masalahnya adalah bahwa saya mendekati Wasteland Wars bukan sebagai produk pemeras gelembung, tetapi sebagai komunitas besar pemain dan bisnis favorit saya.

Biarkan saya mengingatkan Anda bahwa saya sedang mengembangkan bukan pelari pemain tunggal untuk iOS, bukan penembak, tetapi permainan multipemain pada platform Telegram. Dan, tentu saja, tergantung pada topik dan spesifik platform, nuansa mereka muncul, muncul dan akan muncul.

Ketika saya mengembangkan game pemain tunggal, seperti yang mereka katakan, saya memiliki norma: Saya menulis game, menaruhnya di App Store, melewati moderasi, membeli iklan - hanya itu, download dan beli; maka Anda cukup membandingkan pendapatan, mendistribusikan dana, dan menjalani kehidupan "direktur yang brilian". Namun, semuanya pernah berakhir, dan terkadang cenderung berubah.

Apa yang membuat saya memikirkan kembali prioritas saya


Hari ini situasinya sangat berbeda. Singkatnya, saya sepenuhnya terintegrasi ke dalam pengembangan dan dukungan permainan di tingkat backend. Ketika saya bangun dan pergi ke Telegram, saya melihat aliran data di bot bantu saya, saluran dan obrolan backend permainan: laporan tentang pekerjaan bot permainan, laporan tentang penggerebekan, online harian, pembelian dan tindakan pemain lainnya.

Saya diam tentang PM dan obrolan game - hidup saya sendiri selalu terjadi di sana. Orang-orang dari berbagai profesi, latar belakang dan hobi berjalan-jalan di Wasteland virtual, memompa keterampilan mereka, menjelajahi lokasi, berpartisipasi dalam acara dan mendapatkan larangan untuk kembar. Dan semua ini saya lihat di ponsel dan tablet saya, dan di komputer.

Jika ada yang terjadi, saya mencoba memperbaikinya. Ya, saya mencoba memperbaikinya sendiri, karena saya tidak memiliki tim pengembangan. Namun, saya tetap berpegang pada prinsip "ingin melakukannya dengan baik - lakukan sendiri." Dalam artikel saya yang pertama tentang HabrÃ©, saya memberi tahu Anda persis apa masalah yang terjadi. Mari kita kembali kepada mereka, ini dia:

  • server tidak stabil;
  • Pemblokiran telegram.

Ternyata untuk menyelesaikan dan mengatasi dua situasi ini. Dalam kasus pertama, saya hanya berinvestasi dalam kenyamanan para pemain, dan dalam situasi kedua saya harus bertarung. Akibatnya, seperti yang kita tahu, Telegram masih berfungsi terlepas dari tindakan ILV. Jadi, pada kenyataannya, situasi kedua diselesaikan dengan sendirinya dan dengan upaya para pemain yang antusias (yang banyak terima kasih kepada mereka)

Kemudian, tentu saja, para pendatang baru menambah kesulitan di pintu masuk permainan, tetapi mereka juga berhasil mengatasinya. Mungkin kisah paling tidak menyenangkan yang terjadi dari waktu ke waktu adalah kejatuhan penggerebekan. Itu terjadi beberapa kali, tetapi masih tidak keren, ketika di sisi lain layar ratusan orang memutuskan untuk melakukan serangan, dan itu tidak berhasil. Karena itu, saya masuk ke proses sendiri dan memperbaiki kesalahan.

Saya tidak menjawab pertanyaan utama - mengapa ini terjadi, apa yang mendorong saya untuk kode dari tablet di "Swift" (lol, dalam "Swallow") dan berada di dekat gagasan saya. Jawabannya sesederhana mungkin: ada permainan yang disukai orang, dan ada orang yang suka memainkannya. Akibatnya, bagian belanjaan, yang terutama merupakan kepentingan paman dalam pakaian yang disebut "pemangku kepentingan", pindah ke tempat ke-754 dalam prioritas.

Saya tidak memiliki banyak direktur komersial di belakang saya, dan para pemangku kepentingan Wasteland Wars adalah para pemain itu sendiri. Dan hari ini, tujuan saya adalah memberi komunitas pemain kesempatan untuk bersenang-senang dan bersenang-senang dari mana saja di dunia. Saya senang bahwa saya memihak pengguna, bukan ekonomi. Hari ini Wasteland Wars lebih dari sekadar permainan bagi saya.

Pembaruan Terbaru


Gim ini telah selamat dari beberapa pembaruan global. Hari ini ia bekerja di patch 2.2b, yang berhasil diunggah dan memulai serangan mengintip pemain. Dalam pembaruan ekstrem ini, saya menerapkan beberapa gizmos menarik yang begitu lama memohon untuk gim:

  • menambahkan mekanik geng, tetapi kelompok pemain (ala "pesta" dalam game online populer) disebut sedikit berbeda;
  • meluncurkan bonus sistem rujukan - pemilik tautan sekarang menerima sebagian boneka kecil dari setiap pembelian referensi yang diundang setelah pembaruan INI;
  • menambahkan sistem pembayaran Xsolla untuk pemain dari negara lain;
  • Saya banyak bekerja di UI dan memperbaikinya sehingga gameplaynya terlihat jauh lebih bagus dan dirasakan lebih mudah;
  • sementara pemain berpindah dari satu lokasi ke lokasi lain, ada momen dengan harapan. Sekarang dalam proses ada tips yang akan mencerahkan proses ini dan melemparkan beberapa ide kepada pemain;
  • Sebelum menyerang musuh, Anda dapat memeriksa avatarnya;
  • razia tetap, fokus pada geng dan mengacaukan mode serangan terpisah untuk aktivasi diri pada poin serangan;
  • piala yang diperoleh dalam pertempuran sekarang mempengaruhi peringkat faksi.


Lihat avatar profil pemain sebelum serangan

Apa yang akan terjadi?


Setahun yang lalu, saya pikir akan keren untuk melepaskan permainan dari Telegram dan membuat klien di beberapa platform. Tidak perlu untuk hari ini. Tindakan ILV mirip dengan menguji hipotesis dalam hal pemblokiran: mereka melakukan sesuatu, sesuatu tidak. Oleh karena itu, sekarang permainan diam-diam ada di platform di Telegram. Namun, ada segalanya untuk transfer, saya siapkan.

Yang terpenting, saya bermimpi membuat game mandiri dan memberikan pengembangan dunia game ke tangan para pemain. Saya ingin para pemain bersenang-senang dalam skrip mereka sendiri, dan tidak melekat pada skrip. Saya sedang mengerjakannya sekarang.

Ada monetisasi dalam gim saya, yang sering saya pakai topi. Saya ingin menjelaskan mengapa saya membutuhkan monetisasi ini dan bagaimana kira-kira tampilannya. Jika kita berbicara tentang monetisasi secara terperinci, maka topik ini adalah untuk hub yang sama sekali berbeda dan untuk artikel terpisah, tetapi saya akan mencoba. Monetisasi berfungsi seperti ini: semua item game dapat diterima tanpa sumbangan, itu hanya akan sedikit lebih lama, tetapi lebih menarik.

Saya sering ditanya apakah game devs hidup dengan baik. Pada gilirannya, saya tidak siap untuk mengatakan bahwa hari ini Anda sedang memainkan game online, dan besok itu akan membuat Anda kaya dalam donasi dan Anda akan menjadi seorang jutawan. Gamedev hidup dengan baik ketika ia tertarik untuk mengembangkan kreasinya, dan tidak menganggapnya sebagai pemeras gelembung dari sudut pandang produk.

Dan tidak, saya tidak menjadi jutawan tahun ini, karena saya menghabiskan donasi untuk pengembangan game. Paradox, tentu saja, - Saya ingin para pemain merasa nyaman, tetapi pada akhirnya saya mendapatkan kebencian sebagai respons. Aneh, tetapi semua orang berbeda, dan harapan kita cenderung tidak menjadi kenyataan.

Jika Anda akan membuat game online, pikirkan semuanya sebagaimana mestinya. Seperti yang saya katakan, MMORPG bukan cerita tentang jalur cepat menuju pengayaan, tetapi ratusan dan ribuan jam kode, pengujian, dan semua jenis debugging. Dan jika Anda siap dan memahami semua kesulitan di jalan Anda, maka pergilah!
