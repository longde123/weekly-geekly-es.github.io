开发JetBrains产品的插件并在JetBrains市场上出售

我们启动了JetBrains Marketplace ，这是一个新平台，开发人员可以在其中出售和许可JetBrains产品的插件。 现在，插件的作者将有机会为开发本身投入更多的时间-他们将不必提出许可决定并从事销售。 用户将可以使用更方便的工具，这将帮助他们更有效地工作。



JetBrains市场 - 插件存储库开发的一个新阶段，现在有4,000多种工具。 它们有助于扩展JetBrains产品的功能，并根据开发人员的个性化需求进行定制。 这里有各种各样的插件：一些提供对其他语言和新技术的支持，另外一些则用于与云平台集成或帮助记住键盘快捷键。 您可以选择界面主题，甚至可以以Nyan Cat的形式设置进度条。 几乎所有JetBrains产品都有插件，包括基于IntelliJ的IDE，TeamCity和.NET工具。 所有这些插件将来都将可用。

近年来，越来越多的公司和开发人员选择IntelliJ平台和其他JetBrains产品作为为程序员开发自己的工具的基础。 我们创建了JetBrains市场，以便开发人员可以专注于插件，而不会浪费时间搜索许可证管理解决方案和开发发行渠道。 我们的平台为软件许可，计费和销售基础架构以及销售和营销支持提供了一个统包解决方案。

JetBrains Marketplace平台如何对用户有用：

  • 第三方开发人员的付费插件提供的其他功能。
  • 通过JetBrains Account轻松进行插件安装，付款，订阅和管理。
  • 通过JetBrains帐户进行便捷的许可证管理，包括在专用许可证服务器上管理所有订阅和许可证。
  • 验证插件许可证和JetBrains产品许可证（可选）。
  • 各种折扣（例如，批量出售），针对用户组的特殊优惠，开源项目以及其他JetBrains支持计划（即将出现）。
  • 个人数据的安全性和保护：购买插件时，您不必向第三方提供银行卡数据和其他信息。

现在，新的第三方插件仅可用于基于IntelliJ平台的IDE，但是我们计划将支持扩展到其他工具。

已经有什么了


超过100个供应商参加了我们的市场抢先计划 。 我们很高兴介绍已经通过JetBrains市场出售和许可的插件（由插件开发人员提供的描述）：

杂交体整合
亚历山大·巴塔什（Alexander Bartash）
使用Hybris Integration插件，您可以自动将Hybris项目导入IntelliJ IDEA。 此插件可让您在为Hybris开发时充分利用IDE，并增加了更多功能。

爱迪斯2
来自CodesMagic
Iedis是Redis（一个常驻数据结构存储库，用作数据库，缓存和消息代理）的美观而便捷的GUI插件。 Iedis可以处理数百万个密钥，而不会阻止Redis服务器。

Java反编译器
来自BIS Guard＆Co.
Java Antidecompiler保护Java字节码和资源免于反编译，并防止软件的反向开发和窃。 它结合了混淆技术，Java密码算法和原始的BIS Guard＆Co.技术。

JetForcer | 最聪明的Force.com IDE
来自JetForcer团队
JetForcer是一个功能强大的插件，可让您在JetBrains IDE中为Salesforce开发。 它是开发客户端（支持Lightning和Visualforce）以及服务器部分（Apex和SOQL）的理想选择。

JFormDesigner
从FormDev Software GmbH
JFormDesigner是用于Java Swing用户界面的专业GUI设计工具。 对MigLayout，JGoodies FormLayout，GroupLayout（免费设计）和GridBagLayout的一流支持使创建专业级表单变得容易。

歧管
来自Manifold Systems LLC
集成块为Java注入了新的活力：您可以使用扩展方法，结构类型，Java模式和类型安全的反射。 该插件可直接从Java提供对其他语言（例如GraphQL，JSON Schema，YAML和Javascript）的类型安全访问。

兰花
由ThomasFröscher
OrchidE为Ansible的IT基础架构自动化系统提供支持。 该插件为使用Ansible管理基础架构的DevOps团队提供了高级IntelliJ平台功能。

RON（生锈对象表示法）
从Vultix
语言插件RON（生锈对象表示法）增加了对导入Rust，自动完成，验证等类型的支持。

Wolfram语言
帕特里克·谢贝（Patrick Scheibe）
Wolfram语言插件将基于IntelliJ的产品转变为功能强大的Wolfram语言和Mathematica代码开发工具。

ZenUML支持
来自ZenUML
ZenUML插件可帮助直接从Java或ZenUML DSL方法生成序列图。 该插件可用于分析现有代码并在编写代码之前对流程进行建模。
