8 Geräte zum Aufnehmen von Videos auf einem Smartphone

Jetzt wird die Erstellung von Videobewertungen immer beliebter. Jeder möchte sich auf den ersten Blick in diesem unprätentiösen Bereich versuchen. Aber was ist, wenn Sie nicht genug Geld für eine professionelle Kamera haben? Die Antworten auf all diese Fragen wurden in diesem Artikel diskutiert. Und am Ende haben wir für Sie ein kleines Geschenk vorbereitet. :) Na dann lass uns anfangen?

In der Tat, wenn Sie ein Smartphone haben, dann können Sie es wählen. Tatsächlich werden derzeit moderne Telefone mit Kameras hergestellt, die anständige Aufnahmen machen, zum Beispiel Xiaomi Mi4i ( * ) mit einer 13-MP-Kamera, die für Aufnahmen in einem ziemlich beleuchteten Raum völlig ausreicht.

In diesem Artikel werde ich einige Beispiele für Dinge geben, die Ihnen helfen können, mit Ihrem Telefon eine recht anständige Videoüberprüfung durchzuführen.

1) Um Ihr Smartphone so zu reparieren, dass Sie beim Aufnehmen eines Videos kein Zögern in Ihren Händen haben, benötigen Sie ein Stativ. Je nach Aufnahmeort kann dies unterschiedlich sein. Zum Beispiel reicht es für ein Heimshooting der Öffnung einer Verpackung eines überwachten Produkts aus, eine kleine 13-18 cm lange Trigon ( * ) zu kaufen . Es kann auf den Tisch gestellt werden und nimmt aufgrund seiner geringen Größe nicht viel Platz ein.



Eine Alternative zu einem Desktop-Stativ ist der universelle Flexibilitätshalter für Smartphones ( * ). Mit ihm kann der Schusswinkel dank seines elastischen und gebogenen Beins in jede Richtung absolut beliebig sein.



Für die Straße ein größeres Aluminiumstativ 35-104 cm ( *) Es ist mit einem Griff ausgestattet, mit dem Sie das Smartphone reibungslos in jede Richtung drehen und alles, was Sie brauchen, ohne plötzliche Bewegungen entfernen können. Im Prinzip ist dies ein Standardstativ für Foto- und Videoaufnahmen. Auch zur Befestigung einer herkömmlichen Kamera geeignet.



2) Für Aufnahmen auf der Straße benötigen Sie möglicherweise auch einen „Zoom“, wie bei leistungsstärkeren Kameras. Es ist unwahrscheinlich, dass ein gewöhnliches Smartphone damit fertig wird. Das Bild ist pixelig und verschwommen. Dieses Problem kann jedoch durch eine optische Linse mit 12-facher Vergrößerung ( * ) gelöst werden . Es wird für verschiedene Modelle geeignet sein, da es mehrere Halter hat.



Sie können sicherlich eine vielseitigere und kostengünstigere Option erwerben ( *), aber mit weniger "Zoom". Es haftet jedoch wie eine Wäscheklammer am Telefon und ist nicht an eine bestimmte Position der Kamera des Smartphones gebunden.



3) Für die Ferndrehung des Smartphones während der Aufnahme können Sie den interessanten Bluetooth-Halter ( * ) verwenden. Jetzt können Sie mit der Fernbedienung in einiger Entfernung stehen, um ein Video aufzunehmen, ein Bild aufzunehmen oder den Winkel sanft um 360 Grad zu drehen.





4) Wenn Sie sich entscheiden, Bilder von der Seite zu machen, bestimmte Aktionen auszuführen und möchten, dass sich das Smartphone hinter Ihnen bewegt und den Winkel festlegt, ist dies das Richtige für Sie. Ein fauler Halter für Ihr Telefon ( * ), der erfunden wurde, um Videos anzusehen oder Nachrichten vom Bildschirm Ihres Gadgets zu lesen. Sie können es aber auch an einer für Sie geeigneten Stelle reparieren und beispielsweise einen Bericht oder eine Überprüfung ohne die Hilfe eines Außenstehenden durchführen, um Ihr Gesicht abzunehmen.



5) Wenn Sie nicht genug Licht für die Fotografie zu Hause oder auf der Straße haben, können Sie eine tragbare LED-Hintergrundbeleuchtung ( *) erwerben) Dank seines Beines kann es an der für Sie notwendigen Stelle platziert oder aufgehängt werden. Es kostet einen Cent, also können Sie ein paar Stücke kaufen, viele Lichtquellen erzeugen und das Motiv von allen Seiten beleuchten.



6) Die Flash-Komponente ist eine wichtige Komponente zum Aufnehmen von Fotos oder Videos. Dieses Problem wurde aber auch mit einer LED-Lampe für Ihr Smartphone ( * ) gelöst . Es kann an jeder Position des Telefons befestigt werden, da es über eine praktische Halterung verfügt und im Gegensatz zu seinen Gegenstücken, die an das Gerät in der Kopfhörerbuchse angeschlossen werden müssen, autonom arbeitet.



7) Für ein bequemeres Ego-Shooting können Sie den Stirnhalter ( *) verwenden), ursprünglich für die Montage einer Action-Kamera angepasst, jedoch mit einer speziellen Halterung für ein Smartphone. Sie befestigen einfach das Smartphone vor Ihnen an einer speziellen Halterung und klicken auf rec. Und die Tatsache, dass er den Bildschirm vor Ihren Augen wiegt, können Sie sofort sehen, was gerade entfernt wird.

Sie werden zum Aufladen einfach in den Anschluss eingesteckt und haben auf der anderen Seite einen USB-Stecker.
