Caja de herramientas para investigadores - Tercera edición: Encontrar y trabajar con fuentes

El trabajo en cualquier proyecto de investigación implica la búsqueda y el estudio de muchas fuentes de información. La organización de este proceso no es una tarea fácil. Hoy hablaremos de herramientas que están diseñadas para optimizar sus diversos componentes.

  • Caja de herramientas para investigadores n. ° 1: autoorganización y visualización de datos

Los proveedores de software académico a menudo trabajan según el principio de colaboración con instituciones educativas. Si su organización no ha adquirido el producto que le interesa, organizar el acceso al mismo puede ser difícil. Si existen planes individuales, no todos están listos para pagarlos de sus propios bolsillos. Por otro lado, los productos abiertos o gratuitos creados por académicos a menudo sufren de un diseño inconveniente y falta de soporte.

Puede probar muchas herramientas diferentes antes de decidirse por una versión que funcione. Decidimos hablar sobre lo más notable en Hacker News y en las colecciones de GitHub.

Abastecimiento


DOAJ es un catálogo curado de revistas científicas abiertas. En su base, más de 4 millones de artículos de 13 mil publicaciones que representan 130 países. En un futuro cercano, el sitio planea aumentar significativamente el número de revistas humanitarias, altamente especializadas y que no están en inglés. Curiosamente, los DOAJ proporcionan una API abierta para automatizar búsquedas en varios parámetros.

Peerus : le ayuda a realizar un seguimiento de las publicaciones de actualidad recientes. Todos los días, el archivo se actualiza con un promedio de 7 mil nuevos materiales. Peerus proporciona integración de proxy universitario para descargas con un solo clic.

Crossref es un archivo abierto para publicar y buscar artículos científicos. El objetivo del proyecto es reunir a científicos de todo el mundo en un solo sitio para que puedan resolver conjuntamente problemas de investigación. El sitio ya ha recogido cien millones de entradas. Puede encontrar el contenido que necesita por metadatos y palabras clave.

SciRate es una aplicación web de investigación de investigación en arXiv.org . Existen filtros por campo científico y fecha de publicación. Además, se ha implementado un sistema de calificación de usuarios, en base al cual se asigna una calificación a cada artículo.

ArXiv Sanity Preserver : acelera su búsqueda en arXiv.org utilizando algoritmos de aprendizaje automático. La página de inicio muestra los últimos materiales junto con la anotación. Puede buscar publicaciones por palabras clave, en la barra de búsqueda, así como por signos similares, utilizando el enlace Mostrar similar, que se encuentra en la esquina superior derecha cerca de cada documento.

OSF es un centro abierto para buscar y crear proyectos científicos. Aquí puede estructurar su trabajo de investigación, para establecer el control de versiones y agregar etiquetas para que otras personas puedan encontrar la publicación. La plataforma implica trabajar en la nube, lo que facilita el acceso a sus proyectos. El trabajo se puede hacer de forma independiente y compartir con los miembros de la comunidad.

Organización y cita


doi2bib es un motor de búsqueda para trabajar con listas de origen. Se ingresa un identificador de objeto digital (DOI) en la barra de búsqueda, y el sistema muestra metadatos para el material encontrado para organizar la lista de bibliografía en formato BibTeX .

JabRef es una aplicación que optimiza el trabajo con bibliografías en formato BibTeX. La lista generada se exporta a HTML, Docbook, BibTeXML, MODS, RTF, Refer / Endnote, OpenOffice y LibreOffice. Puede ejecutar la utilidad en Java VM ( versión 8 ), en Win, Linux y OS X.

Mendeley es un administrador de listas bibliográficas y una plataforma en la nube que le permite acceder a los archivos del proyecto desde cualquier dispositivo. Mendeley también ofrece oportunidades de asesoramiento y herramientas de comunicación.

Docear es una herramienta abierta para organizar fuentes y construir trabajos científicos sobre su base. El usuario debe especificar una carpeta que desempeñará el papel de una "biblioteca" de materiales para un proyecto específico. El programa lo escaneará automáticamente y agregará los archivos encontrados a su base de datos. La base de datos puede estructurarse en forma de "árbol", combinando el trabajo en una dirección en una rama. Los comentarios y resaltados en archivos pdf se importan automáticamente y también se muestran en la interfaz. Cuando termine el trabajo con las fuentes, Docear ayudará a formar el "mapa mental" final y proporcionará las citas necesarias.

Duecredit : puede citar no solo texto, sino también código. Si su programa toma prestados algoritmos o métodos de artículos científicos, esta biblioteca lo ayudará a referirse correctamente a ellos. La biblioteca está actualmente disponible en Python.

Trabajar con documentos


Sphinx es una herramienta para "compilar" documentos en reStructuredText en HTML, ePub, Texinfo, páginas man y texto sin formato. Admite 50 extensiones , indexación automática de componentes de software, creación de enlaces para funciones, clases, citas, términos, etc. Para los principiantes, los desarrolladores han preparado un tutorial y una guía para un inicio rápido . Además, le sugerimos que consulte los siguientes temas para trabajar en Sphinx: rtd : admite tanto Sphinx como Read the Docs ; Bootstrap es un tema en el que las plantillas CSS y las extensiones JavaScript del marco Bootstrap se integran con la funcionalidad Sphinx: navegación, menús jerárquicos, etc.

Git-scribe es una herramienta de código abierto para crear libros electrónicos. Proporciona funcionalidad para estructurar texto, editar, corregir, corregir, colaborar, así como publicar y exportar a PDF, mobi y epub.

Jupyter Notebook es un editor de texto con un entorno incorporado para la programación interactiva en Python. Anteriormente conocido como iPython Notebook. Se puede instalar tanto localmente como en VPS, hay versiones en la nube del producto.

Textura : la interfaz de este editor brinda acceso a una amplia gama de funciones especializadas, como la creación de gráficos, fórmulas y varios métodos de citas. El programa utiliza un formato de archivo especial que, si lo desea, puede "desempaquetarse" y desarmarse en componentes. Según los creadores de Texture, esto permite a los editores simplificar el proceso de edición y revisión.
