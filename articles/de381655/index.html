Nintendo forderte, den Game Boy Advance JavaScript-Emulator von Github zu entfernen



Die Firma Nintendo verlangte , den in JavaScript geschriebenen Github Emulator Game Boy Advance zu entfernen . Anscheinend war der Grund das Vorhandensein mehrerer Originalspiele von der Spielekonsole mit dem Emulator von Bildern. Derzeit ist die Emulatorseite nicht mehr verfügbar. Die Anfrage wurde letzte Woche von der Firma gesendet.

Die Geschichte des kultigen tragbaren elektronischen Spiels Game Boy begann 1989. Es wurde das erste beliebte Spielesystem, und die Konkurrenz konnte seinen Erfolg nicht wiederholen. Nintendo veröffentlichte regelmäßig neue Versionen des Spiels - 1996 gab es eine reduzierte Version von Game Boy Pocket, 1998 - Game Boy Color, die erste Farbversion des Spiels. Und im Jahr 2001 erschien Game Boy Advance, das letzte Spiel, das weit verbreitet war.


Game Boy Advance

Obwohl das letzte Präfix dieser Serie, Game Boy Micro, im Jahr 2005 veröffentlicht wurde, erinnern sich viele Menschen mit Nostalgie an ihr Lieblingsspielzeug und genießen die Gelegenheit, vertraute Spiele auf dem Emulator zu spielen. Und obwohl die Seite des Emulators mit den im Kit enthaltenen Spielen geschlossen ist, befindet sich der Emulator ohne Spiele unter einer anderen Adresse . Darüber hinaus verfügt github über eine gute Liste von Emulatoren verschiedener in JavaScript geschriebener Plattformen.

Dies ist nicht das erste Mal, dass Nintendo den Game Boy-Emulator angreift - letztes Jahr ereignete sich das gleiche Schicksal für die damals beliebte mobile Anwendung für iOS, GBA4iOS.
