Applet kedua, menutupnya dan tombol transparan dalam Memproses 3

Untuk semua KU. Posting pertama saya, tolong jangan menilai dengan ketat.

Ada kebutuhan untuk pekerjaan untuk membaca secara visual status peralatan dan, jika perlu, nyalakan kembali cryocompressor dari jarak jauh. Di tangan adalah Arduino UNO, sebuah sensor cahaya sekitar dibeli untuk itu, sejauh ini ada 2 relay dan 2 solenoida pada 12 V (bukan titik). Di masa depan, bisnis ini akan ditumbuhi sensor suhu, kelembaban dan tekanan, tetapi sejauh ini belum siap.

Alasan untuk menulis posting adalah bahwa ada sangat sedikit informasi di Internet tentang kemampuan Memproses 3, terutama di bagian bahasa Rusia itu. Saya berharap bahwa beberapa teko seperti saya akan berguna suatu hari nanti agar tidak bodoh seperti saya selama seminggu terakhir.

Sketsa untuk Arduino sesederhana tiga rubel: setelah menerima nilai-nilai tertentu dari port com, dunk menentukan nilai pada pin dan menulis jawaban untuk port com, juga mengukur iluminasi dan juga melaporkannya. Di masa depan, itu akan diperluas.

Sketsa Arduino: 

#include <Wire.h> #include <BH1750.h> int val; BH1750 lightMeter; void setup() { Serial.begin(9600); lightMeter.begin(); Serial.println("Running..."); pinMode (7, OUTPUT); pinMode(6, OUTPUT); digitalWrite(7, HIGH); digitalWrite(6, HIGH); } void loop() { uint16_t lux = lightMeter.readLightLevel(); if (Serial.available()) { val = Serial.read(); if (val == '0') { digitalWrite(7, HIGH); digitalWrite(6, HIGH); Serial.println("  "); } if (val == '1') { digitalWrite(7, HIGH); digitalWrite(6, LOW); Serial.println(" 1 "); } if (val == '2') { digitalWrite(7, LOW); digitalWrite(6, HIGH); Serial.println("  2"); } if (val == '3') { digitalWrite(7, LOW); digitalWrite(6, LOW); Serial.println("  "); } if (val == '4') { if (lux >= 800) { Serial.print("Light: "); Serial.print(lux); Serial.println(" lx   "); } else { Serial.print("Light: "); Serial.print(lux); Serial.println(" lx  !!!!!!!"); } } } }

Saya pikir ini tidak masuk akal, tetapi karena kenyataan bahwa orang lain akan menggunakan fungsi Pribluda saya di masa depan, saya ingin menulis GUI yang sederhana namun dapat dimengerti. Pilihan jatuh pada Pemrosesan, karena saya sebelumnya akrab dengan IDE Arduino, yang, pada gilirannya, dibangun berdasarkan Pemrosesan.

Applet menggunakan pustaka ControlP5 untuk membuat kontrol.

Secara pribadi, saya memiliki masalah dengan penerapan dan pencarian informasi tentang hal-hal berikut: 1) meluncurkan jendela kedua dari aplikasi, 2) menutup jendela ke-2, tetapi agar jendela utama terus berfungsi, 3) demi kesenangan - pengaturan gambar sebagai latar belakang dan tombol, tombol tembus.
