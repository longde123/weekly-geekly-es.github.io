Vkontakte beginnt mit der Bekämpfung der Buchpiraterie

Das soziale Netzwerk führt ein digitales Fingerabdrucksystem ein, das das Herunterladen von Büchern verbietet, für die Rechte geltend gemacht werden.


Foto: Foto: REUTERS / Sergei Karpukhin

„Vkontakte“ beginnt aktiv mit „Piraten“ zu kämpfen. Das System eines digitalen Fingerabdrucks für Bücher wurde bereits eingeführt, das das Herunterladen von Dateien verbietet, die durch Beschwerden von Copyright-Inhabern gelöscht wurden, schreibt Izvestia. Das Verbot gilt auch für erneut heruntergeladene Kopien von Büchern, für die Rechte geltend gemacht werden. Es ist möglich, eine Datei problemlos in ein soziales Netzwerk hochzuladen, aber Benutzer können ein solches Buch nicht herunterladen.

Jetzt gibt es im Abschnitt "Dokumente" des sozialen Netzwerks eine große Anzahl von E-Books in verschiedenen Formaten, einschließlich doc, docx, fb2 und anderen. Der Benutzer kann solche Dateien entweder mit dem Gesprächspartner im Chat oder auf seiner Seite teilen. Es gibt auch Gruppen für Buchliebhaber - hier werden häufig Bücher platziert, die urheberrechtlich geschützt sind. Wenn Sie eine Datei in ein soziales Netzwerk hochladen, generiert das System einen Download-Link. Wenn Sie auf ein Dokument klicken, wird der Download einer solchen Datei automatisch gestartet. Wenn Sie nun auf ein Buch mit einem "digitalen Fingerabdruck" klicken, wird automatisch eine Meldung angezeigt, dass die Datei auf Anfrage des Copyright-Inhabers gelöscht wurde.

Nachdem die Verwaltung dieses Problem angegangen war, ging die Anzahl der illegal heruntergeladenen Bücher erheblich zurück. Die Aktionen der Vkontakte-Administration waren sehr effektiv. Der Leiter der Vereinigung zum Schutz des Urheberrechts im Internet (AZAPI), Maxim Ryabyko, sagte, dass der Überwachungsdienst dieser Organisation begonnen habe, weniger Ansprüche des sozialen Netzwerks zu senden. Jetzt verklagt AZAPI Vkontakte nur wegen der aktiven Verbreitung illegaler Kopien von Büchern durch Benutzer der Ressource.
