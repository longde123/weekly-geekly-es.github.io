Nützliche Software: 7 mobile Anwendungen für Fotografen



Hallo Geektimes! In letzter Zeit fotografiere ich gern und studiere dieses Thema umfassend. Bei der Auswahl der Werkzeuge für die Aufnahme stellte sich die Frage, ob es mobile Anwendungen gibt, mit denen Anfänger und professionelle Fotografen bessere Bilder aufnehmen können.

Im heutigen Artikel präsentiere ich Ihnen eine Liste von 7 Anwendungen, die ich interessant und zur Lösung dieses Problems geeignet fand.

PlanIt! Fotografien


Eine Anwendung, mit der natürliche Lichtquellen in Abhängigkeit von verschiedenen Faktoren bestimmt werden - Standort, Gelände usw. Mit dem Tool können Sie die optimalen Beleuchtungsparameter für Aufnahmen mit verschiedenen Geräten mit unterschiedlichen Matrizentypen berechnen.



Der Hauptbildschirm der Anwendung ist eine Karte, auf der Sie den Ort und das Motiv positionieren können. Die Karte selbst kann skizzenhaft, satellitengestützt oder im 3D-Format sein. Daten zur Entfernung zum Objekt und zu den Parametern der verfügbaren Beleuchtung werden in einem speziellen Fenster angezeigt - Informationen können nicht nur in der horizontalen Ebene, sondern auch vertikal oder relativ zum Horizont berechnet werden. Die Anwendung enthält keine integrierten Wissensdatenbanken oder Sammlungen von Tuning-Tipps, sodass sie eher für professionelle Fotografen geeignet ist.

Preis : Kostenloses

Betriebssystem : Android

Die Ephemeride des Photogaphers (TPE)


Ein Tool, mit dem Fotografen einen guten Ort zum Fotografieren im Freien finden. Die Anwendung zeigt Informationen über den Einfallswinkel von Sonnenlicht oder Mondlicht zu einem bestimmten Zeitpunkt des Tages, Daten über die Mondphasen, die Dämmerungszeit, das Schattenvolumen, bevorstehende Wetteränderungen usw. an.



Mit TPE können Sie auch einen Punkt am Himmel angeben, an dem Licht benötigt wird, und die Anwendung zeigt an, wann die Sonne dort erscheinen soll. So können Sie beim Aufnehmen von Landschaftslandschaften ein Objekt wie einen Baum auswählen und herausfinden, wann es im Morgengrauen schön entfernt werden kann, um keine vergebliche Wartezeit zu verschwenden.

Preis : 8,99 USD (iOS), 365,69 USD (Android)

Betriebssystem : Android / iOS / Web

Sonnenvermesser


Ein Werkzeug zur Berechnung der Position natürlicher Lichtquellen wie Sonne und Mond. Basierend auf GPS und Daten von einem Gyroskop und einem Beschleunigungsmesser kann der Sonnenvermesser dreidimensionale Projektionen der Bewegung von Himmelskörpern erstellen, und es werden auch Statistiken berechnet, beispielsweise über die Position von Schatten zu einem bestimmten Zeitpunkt.



Die Anwendung enthält eine interaktive Karte des Gebiets und einen 3D-Kompass, der nützlich ist, wenn Sie direkt von einem Smartphone aus fotografieren.

Preis : Kostenloses

Betriebssystem : Android

Dunkler Himmelsfinder




Um ein gutes Foto des Nachthimmels zu machen, müssen Sie noch einen unbeleuchteten Teil davon finden. Mit der Anwendung kann Dark Sky Finder die dunkelsten Orte bestimmen, indem er helle Bereiche auf einer Google-Karte überlagert (außerdem können Sie eine Satellitenansicht, eine Straßenansicht oder eine Hybridkarte auswählen).

Preis : 2,99 $

Betriebssystem : iOS

Einfache Freigabe


Bei Dreharbeiten, an denen Personen beteiligt sind, benötigen Fotografen häufig die Erlaubnis, ihre Bilder verwenden zu dürfen. Ein solches Dokument wird als Modellversion bezeichnet. Bekannte Fotobanken benötigen häufig auch die Verfügbarkeit beim Hochladen von Fotos.



Die Easy Release-Anwendung erleichtert dem Fotografen das Abrufen von Modellversionen. Mit seiner Hilfe können Sie schnell ein Dokument im PDF-Format erstellen, das die an der Aufnahme beteiligte Person mit einem einfachen Fingerdruck über den Bildschirm signieren kann. Natürlich hat das resultierende Dokument einen begrenzten Umfang, und es ist unwahrscheinlich, dass das russische Gericht es bei Problemen berücksichtigt, aber große Fotobestände wie Getty Images berücksichtigen es.

Preis : 9,99 $

Betriebssystem : Android / iOS

Posing App


Eine einfache und benutzerfreundliche Anwendung für Anfängerfotografen, mit deren Hilfe eine gute Pose für eine Person für ein Foto ausgewählt werden kann, abhängig von verschiedenen Situationen, Merkmalen einer Figur oder Kleidung. Die Anwendung enthält eine große Datenbank möglicher Posen für Gruppen- und Einzelaufnahmen, die kategorisiert sind - beispielsweise für Kinder, Frauen, Männer, Hochzeits- und Gruppenaufnahmen.



Darüber hinaus speichert die Posing-App eine Datenbank mit Tipps zur Auswahl einer Pose in bestimmten Fällen - das einzige Minus ist dass sie alle auf Englisch sind.

Preis : 89 reiben.

Betriebssystem : Android

Kamera von IFTTT machen


Gute Fotos können nicht nur mit einer professionellen Kamera aufgenommen werden, sondern auch mit einem guten Smartphone. Und um ein mobiles Gerät noch leistungsfähiger zu machen, hilft die Anwendung von der berühmten Website IFTTT. Wie der Hauptdienst verwendet die Anwendung "Rezepte", mit denen Sie bestimmte Aufgaben automatisieren können.



Sie können beispielsweise ein Rezept konfigurieren, das ein Foto aufnimmt, und das Bild dann automatisch für spätere Arbeiten in einen Cloud-Speicher wie Dropbox hochladen oder ein Bild in einem Brief an eine bestimmte Adresse senden. Auf diese Weise können Sie auch die automatische Veröffentlichung von Bildern in sozialen Netzwerken einrichten, ohne die entsprechenden Anwendungen öffnen zu müssen.

Preis : Kostenloses

Betriebssystem : Android / iOS
