隐形照片



你在说什么


朋友经常想知道：我为什么要进行隐形摄影？ 红外线，紫外线，热能。 那里真的有有趣的东西吗？

由于一次观看总比听一百遍更好，因此这里有一个小型演示。 有15个科目。 它们在可见光谱中，然后我们在其他范围内观察它们：

[可见光，400-750 nm。 F / 6.3，1 / 2500秒，ISO 200，尼克尔35mm玻璃镜头。 通过带有可见光滤镜Kolari Vision热镜UV / IR截止滤镜的带有远程内部IR / UV滤镜的改良尼康D90拍摄。]

数字表示：

1.紫外线灯（UV-A范围）
2.银条
3.美国一美分的铜钱硬币
4.铑环
5.一杯白葡萄酒
6.切菜板，据说是PVC制成的
7.草（在后台）
8.一杯红酒
9.一块黑色的塑料垃圾袋
10.一张纸上涂了一条防晒霜
11.附带的二极管手电筒
12.铅笔
13. 0.8毫米厚的硅板
14.蜡烛烧
15.由亚硒酸盐或烟熏石英制成的“心脏”

照明就是阳光。

准备好了吗 走吧

紫外线世界



[紫外线350-400 nm。 F / 6.3，1 / 5秒，ISO 3200，尼克尔35毫米玻璃镜头（确定了下限）。 在带有远程内部IR / UV滤镜的经过改进的Nikon D90上拍摄。 Kolari Vision紫外线紫外带通透射滤光片。


有什么变化？ [复制可见图像以方便比较]

1.如预期的那样，紫外线灯已经变亮。
3.但是铜已经变黑了。 众所周知，它在紫外线下的反射很差。
5.白葡萄酒变成...黑。 是的，葡萄在近紫外线下几乎是不透明的。 对于桃酒，此数字无效。
6.切菜板突然变黑。 而且所有的划痕和划痕都清晰可见。 专利理念：使用紫外线照相技术来区分新使用的电路板和长时间闲置的电路板（此处应有一个图释）。
7.草也变黑了。 在蓝色和紫外线下，这是所有植被的特征。 对于早期拍摄正色胶卷的摄影师来说，这有些困难。
10.抗晒霜。 可以看到他！ 这是另一个应用程序：在去海滩之前，先在紫外线下拍照并检测防晒涂层中的缝隙。
11.二极管手电筒怎么了？ 不，我没有将其关闭。 他只是在紫外线下不发光。 这可能是为什么有绘画博物馆试图切换到二极管照明的另一个原因。 因为紫外线涂料有时会褪色。
12.铅笔或笔上的黄色油漆也变黑了。
14.蜡烛的火焰几乎不可见。 可以预期的是：温度不是紫外线下会严重发光的温度。
15.还有“心”？ 瑞利的散射毁了他。 没有更多的透明度了；有一块乳白色的石头。

近红外750-900 nm



[在IR附近，750-900nm。 F / 6.3，1/2500秒，ISO 1600，尼克尔35mm玻璃镜头。 在带有远程内部IR / UV滤镜的经过改进的Nikon D90上拍摄。 过滤器B + W＃093（87C）。]


在这里观察到什么有趣的东西？

1.紫外线灯完全堵塞。 如预期的那样。
2.由于某种原因，银已经变黑，尽管它在红外光中应该表现得很好。 怎么了 因为在酒吧我们看到了天空的反射。 主要由蓝色可见光组成。 过滤器抑制了它。 那些用近红外光拍摄的人知道在这个范围内天空看起来有多黑。
4.然而，与铜或银相比，铑的相对亮度有所下降。 因为它确实反映了近IR的恶化 （请参见图5）
5.白葡萄酒是完全透明的。 喜欢...红色！ 两者都变得像淡水一样。
7.草地急剧变亮。 同样，这也是该范围内几乎所有植被的特征。
11.二极管手电筒无声。
14.但是，蜡烛发光，明亮地发光。 好吧，当然，在其温度下，大部分辐射都不来自可见光。
15.鹅卵石心怎么样？ 它变得像透明玻璃一样透明。
