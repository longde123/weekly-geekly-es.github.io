对于开源项目，您可以免费使用Azure Pipelines进行持续开发和部署。


为了确认我们打算为所有开发人员提供开放灵活的工具，在Azure Pipelines中，您可以免费执行无限的免费开发，并为每个开源项目同时免费部署多达十个任务。 Azure Pipelines具有Linux，MacOS和Windows云群集，因此对于任何类型的项目都是理想的选择。

Azure管道概述 。

Visual Studio团队服务（VSTS）开发


Azure DevOps是Visual Studio Team Services（VSTS）开发的结果。 VSTS用户将自动迁移到Azure DevOps。 现有的用户不会失去功能，但会获得更多选择和控制权。 保留了VSTS的独特功能，即完全透明和集成。 Azure DevOps服务可以无缝地协同工作。 我们将从今天开始转移用户，在接下来的几个月中，他们将看到所有更改。 这是什么意思？

  • URL将从abc.visualstudio.com更改为dev.azure.com/abc。 从visualstudio.com地址的重定向将被启用，因此您不会遇到断开的链接。
  • 随着服务的更改， 界面也已更新 。 我们将根据试用版的用户反馈继续改进它。 对于新用户，默认情况下将启用它。 在未来几个月中，它将成为现有用户的主要工具。
  • 本地Team Foundation Server（TFS）服务将继续更新，这些更新将类似于为Azure DevOps发布的更新。 在发布TFS的下一版本之后，该产品将获得一个新名称-Azure DevOps Server，并将以以前的模式进行更新。

了解如何立即将这些更改应用于当前的VSTS用户。
