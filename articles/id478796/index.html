Seperti yang sudah saya perhatikan, IT tidak menarik minat saya. Tidak ada komputer di sekolah menengah di Rusia tengah. Saya tidak punya komputer. Dan bunga, masing-masing, tidak muncul. Saya memenangkan Lotere Kartu Hijau dan pada tahun 19 meninggalkan tahun kedua geografi di sebuah universitas reguler Rusia dan bergegas ke Amerika Serikat. Saya tidak ingin menjadi seorang ahli geografi, tetapi melewati titik di atasnya memungkinkan saya untuk lulus dengan mudah. Setelah pindah ke AS, saya tinggal bersama teman-teman di Wisconsin. Beberapa tahun pertama itu semua merupakan kejutan budaya. Saya tahu bahasanya sangat buruk. Penekanannya masih mengerikan. Saya tidak memikirkan pendidikan dan sejenisnya. Saya hanya ingin bertahan hidup. Selama 2 tahun ia bekerja di supermarket tipe Volmart sebagai penjual kasir. Lalu aku bosan. Ada apartemen sewaan, ada mobil dan masih ada sedikit uang, tetapi saya bosan. Pada usia 21, saya memutuskan untuk mempertimbangkan opsi dengan pendidikan lokal, tetapi setelah penelitian tentang topik ini, dia membungkuk. Seperti di Rusia, universitas AS sebagian besar ditujukan untuk pelamar setelah sekolah menengah atau sekolah menengah atas. Itu menakutkan dan mengganggu saya. Saya tidak bisa membayangkan bagaimana pada usia 21 saya akan berada di kelas yang sama dengan anak berusia 18 tahun. Saya menelepon, menulis ke sekolah-sekolah lokal, dan kebanyakan dari mereka merekomendasikan memulai dengan community college. Sekarang saya berpikir bagaimana mereka benar. Saya pergi untuk berbicara dengan orang-orang dari komunitas kampus, tetapi dalam perjalanan ke kantor mereka saya menemukan seorang perekrut dari armada, yang memberi saya mie dan berada di armada. Saya mengobrol dengannya dan meninggalkan nomor telepon saya. Itu membuat saya tertarik. Perekrut memanggil saya minggu berikutnya dan saya setuju untuk mendaftar di armada. Saya lulus tes jenis aikyu, lalu tes aneh lain untuk bahasa fiksi. Setelah menerima hasil, dia mengatakan kepada saya bahwa saya lulus hasil tes untuk semua spesialisasi, tetapi karena saya tidak memiliki kewarganegaraan, ini adalah masalah. Untuk penduduk tetap, mereka memiliki spesialisasi di mana akses ke rahasia militer tidak diperlukan. Saya memilih logistik, tetapi ini tidak begitu penting.

Awalnya saya tidak suka armada, tetapi kemudian saya terbiasa. Kewarganegaraan dikeluarkan dengan cepat dan gratis. Saya tidak suka tinggal di "asrama", tapi itu tidak membosankan. Ketika mereka mulai membayar saya karena tahu bahasa Rusia dan tumbuh dalam peringkat, saya memutuskan untuk pergi untuk kontrak lain, dan kemudian yang lain. 9 tahun di Angkatan Laut, dan itu membuatku bosan juga. 3 wisata enam bulan ke Teluk Persia. Uang cocok untuk saya. Saya menyukai pantai timur. Tetapi tubuh mulai gagal. Plus, saya punya anak pada waktu itu dan seorang suami (!) Tapi seorang mantan. Stres juga lelah dan saya ingin keluarga saya di usia 30 tahun untuk melihat lebih sering. Saya memutuskan untuk meninggalkan armada dan memberi tahu komandan tentang hal itu. Dia ternyata seorang lulusan Harvard dan, setelah berbicara dengan saya, merekomendasikan pergi ke perguruan tinggi dan menggunakan tunjangan hukum saya Pasca 9/11 GI Bill. Negara memberikan uang kepada para veteran untuk pelatihan. Itu tidak terlalu penting. Komandan membantu saya dengan penyerahan kertas. Saya sudah tahu bahasa Inggris dengan cukup baik dan mendaftar ke 20 perguruan tinggi yang berbeda dari 20 besar.

Melewati SAT dua kali. Saya tidak punya waktu untuk mempersiapkan, tetapi saya harus menyerahkannya. Ditambah lagi, aku tidak perlu mengeluarkan biaya karena negara. Statistik mengatakan bahwa jika Anda mengambilnya lebih sering, maka peluangnya meningkat ke hasil yang lebih baik. Karena itu, mereka yang memiliki uang untuk tutor dan mengikuti ujian berkali-kali bisa mendapatkan hasil yang lebih baik. Yaitu WASP dengan uang memiliki peluang lebih baik daripada yang lain.

Untuk beberapa alasan saya yakin bahwa saya mungkin akan pergi ke sana karena statistik mereka. Saya tinggal di rumah saya dengan tetangga saya dengan anak-anak (mahasiswa pascasarjana perumahan). Saya tidak perlu khawatir tentang apa pun. Tidak ada yang menyadari bahwa saya 10 tahun lebih tua karena saya terlihat muda dan masih pergi ke gym. Saya bahkan tidak khawatir tentang pengalaman militer saya karena usia saya. Setelah Anda melakukan tur ke Timur Tengah, entah bagaimana semua hal kecil dalam hidup tetap menjadi hal kecil.
