Mengatur awalan untuk pemindai MINDEO CS2190 untuk bekerja dengan 1C dalam mode istirahat keyboard

Petunjuk untuk menginstal awalan untuk pemindai MINDEO CS2190 untuk bekerja dengan 1C Enterprise dalam mode istirahat keyboard.


Mari pertimbangkan untuk menyiapkan pemindai MINDEO CS2190 dengan kabel USB untuk bekerja dengan 1C Enterprise dalam mode memecahkan keyboard atau mensimulasikan keyboard.

Pemindai itu sendiri dengan mudah terhubung ke komputer di port USB apa pun, dan Windows dengan cepat dan mudah menginstal driver yang diperlukan. Kemudian pemindai siap bekerja, dan ini dapat diperiksa dengan memindai di buku catatan. Tetapi agar pemindai bekerja dengan 1C, yang masih jauh dari terminal, Anda harus sedikit mengubah pengaturan pemindai.

Pemindai MINDEO CS2190 dapat mensimulasikan koneksi selanjutnya. variasi:

  • Dalam mode keyboard HID
  • Dalam mode COM virtual USB (port COM virtual)
