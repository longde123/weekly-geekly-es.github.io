传染性应激：以狗及其主人为例的皮质醇水平在种间同步



人是社会人。 无论一个人试图分开或分离的程度如何，其他人都可能以一种或另一种方式影响他，他们自己可能不想这样做。 这种现象称为种内双向心理情感反应。 在这个长定义中，一个重要的词是“种内”。 这意味着，不仅在一群人中，而且在成群的鸟类，狮子的骄傲甚至羊群中都可以观察到类似的反应。 长期，以一种或另一种方式密切接触的有机体将相互影响，包括在心理情感状态方面。 今天，我们将与您讨论这项研究，据作者说，研究的结果成为种间双向心理情感反应的第一个重要证据。 这种现象如何精确地表现出来，成为了研究的实验对象，科学家如何支持他们的结论？ 这是我们从研究小组的报告中学到的。 走吧

学习基础


如果您问一个人从其他生物中脱颖而出的特征是什么，许多人首先会称其为智力，这是事实。 其他人会增加社交性和情感性，但是这些贴士不仅适用于人。 集体生活的任何生物都是社交的。 至于情感，与猫鼬相比，一个人可以并且具有更广泛的情感和表达方式，但是情感并不是其独特的特征。

一个人的心理情感状态与他的环境，特别是与周围的社会息息相关。 科学家称之为传染性情绪，其中最有力的“感染”是压力。 例如，发现老师持续施加压力的学生的皮质醇水平较高。
皮质醇*是一种具有生物活性的糖皮质激素，可参与机体对压力状况的反应。

即使在草原田鼠（Microtus ochrogaster）中也可以观察到类似的图像，这表明应激水平与先前接触过应激源*的伴侣之间存在相关性。
压力源* -引起压力的任何因素。
但这是种内压力相关性的例子。 尽管还没有进行太多研究，但种间也存在。 为了解决这个问题，科学家建议研究家犬及其主人的压力水平。 正如我们在上一个星期五讨论的那样 ，人与狗共存了数千年。 他们都是社会人，因此可以期望在这种关系中发生心理-情感关系。

科学家认为，在一个人和他的四足朋友之间，可能会发生种间压力荷尔蒙（皮质醇）的长期同步，这种水平可借助头发来检查。 他们决定丢弃血液和唾液，因为这些材料可以提供描述最近时期的结果，而头发可以让您评估长期的荷尔蒙背景。 头发长大，无论听起来多么陈腐，皮质醇都会在头发上留下印记，经过研究，您可以获得该激素浓度的回顾性日历。 以前在母亲和他们的孩子的参与下进行了类似的分析，他们的皮质醇水平根据结果同步。

至于狗，人与宠物交流的一个重要特征是训练和建立等级制度。 如果没有对狗进行训练，则可以玩恶作剧（例如，以不喜欢的方式给自己喜欢的花朵浇水，或者吠叫扔向路过的人）。 为了达到纪律，狗必须了解它的主人是领导者，即背包中的支配者。 狗的训练和纪律的过程不仅导致它与人之间的情感联系，而且还双向影响每个二元组的行为。
Dyad * -共生，其中存在等级依赖性。 二元组也是两个相互连接的对象（即一对）的组合。
对狗和人类皮质醇水平的研究还取决于宿主的人格特征，甚至取决于双性恋中狗/宿主性别的组合。


研究参与者：博德牧羊犬（左）和谢尔蒂（右）。

在这项工作中，科学家决定使用人狗二极管的例子来更详细地研究种间心理-情绪同步的问题。 为此，邀请了58人及其宠物（即58双）。 科学家通过特殊的项圈观看了每只狗的身体活动，该项圈将有关狗的身体状况的信息传送给他们。 在夏季和冬季，测量狗和宿主中皮质醇的水平，以分析其在生物材料（头发）摄入之间的变化。 两种类型的狗参加了这项研究：Sheltie（33个人：18名雌性和15雄性）和Border Collie（25个人：13雌性和12雄性），其中既有普通宠物又是各种比赛的常规参与者。 所有犬只的主人都是雌性，普通宠物的尾状测试对象的平均年龄为4.7±0.4岁，竞争者的平均年龄为4.7±0.7。 养狗人的平均年龄为46.3±1.7岁。

研究成果


为了分析人对狗的影响，使用了一个通用的线性模型，其中考虑了宠物的性别，品种和生活方式。

在夏季，最显着的相关性是HCC（毛发皮质醇浓度）与狗的生活方式（χ2 = 6.268，P = 0.012，2A）之间以及HCC与狗的性别（χ2 = 5.200，P = 0.023，2B）。

所有犬，无论性别和生活方式如何，都对人类HCC产生影响，但雌雄犬参加比赛的影响最大。

在冬季，该品种起着重要的作用（χ2 = 6.451，P = 0.011）：在Sheltie HCC中，其显着高于边境牧羊犬（12.905±1.417对12.069±1.203）。

除上述因素外，还有其他因素（狗的主人工作，狗的年龄，是否与其他宠物同住等）。 但是，这些因素实际上对结果没有影响，因此研究人员没有注意它们。

但是体育活动实际上可以影响人体内皮质醇的水平，因为科学家决定检查长期皮质醇水平与狗活动的比例。 通过特殊的项圈对四足动物的状况进行了1周的监测，科学家从中分配了3名工人和1天假进行分析。

狗处于中度和高度运动状态所花费的时间与狗的HCC相关。 夏季肝癌（N = 44，r = -0.213，P = 0.165）或冬季肝癌（N = 43，r = -0.239，P = 0.122）均未发现显着相关性。

对于科学家来说，影响皮质醇水平的最奇怪的因素仍然是性格，即狗及其主人的个性特征。 这项研究的参与者分别完成了2个调查表：DPQ（狗性格问卷-有关狗的性格的问卷）和The Big Five Inventory（关于一个人的个人素质的问卷）。

事实证明，宿主的性质极大地影响了宠物皮质醇的水平。 因此，人类神经质水平的提高对狗的HCC产生了负面影响（夏季：χ2 = 7.951，P = 0.005，β= -0.364；冬天：χ2 = 4.919，P = 0.027，β= -0.026）。

最后，科学家检查了季节本身如何影响皮质醇水平，因为样品是在夏季和冬季故意采集的。 事实证明，在冬天，压力荷尔蒙的水平比夏天高得多。

结语


长期生活在一起的两个或两个以上同一物种的个体的心理情感状态之间的关系一直是，而且仍然是不可否认的。 至于皮质醇的种间同步，没有特别令人惊讶的发现。 这项工作的目的不是发现；而是有必要确认先前所说的理论，即人和狗作为社会物种的代表，彼此之间具有很强的影响力。

进行这项研究的瑞典林雪平大学的科学家认为，狗是主人的情绪状态的投射。 也就是说，就皮质醇水平的长期同步而言，人对狗具有很大的影响。 此外，他们认为以前没有进行过此类研究。

这项工作的结果会让我们感到惊讶吗？ 否定可能性更大。 但是，我们再次获得了对“我们应对驯服者负责”这一短语的科学支持。
