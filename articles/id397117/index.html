Sailfish OS Keripik Tak Terlihat

Halo pembaca!

Hari ini saya ingin menyoroti beberapa chip Sailfish OS yang tidak terlihat. Artikel ini lebih fokus pada pengguna OS Sailfish, tetapi juga pada "penganut" platform lain, saya harap itu juga akan menarik, mereka dapat menghargai tingkat kenyamanan dan kesiapan sistem untuk pengguna akhir. Jika Anda siap, maka kami akan melakukan pemotongan!



Tahukah Anda bahwa:


1. Jika Anda memegang aplikasi dan membalikkan ponsel cerdas, aplikasi TIDAK akan mengubah orientasi.
Mari kita bayangkan sebuah contoh: Anda menjelajah Internet dari smartphone Anda sambil berdiri, lalu memutuskan untuk berbaring (sementara orientasi ponsel dimatikan), sehingga orientasi tidak berubah, cukup pegang jari Anda di layar sementara smartphone mengubah posisinya di ruang.

2. Mengklik dua kali pada tombol kunci perangkat keras membuka kunci perangkat Anda dan Anda akan segera dibawa ke Layar Utama, melewati Layar Kunci.
Harap dicatat bahwa mengklik dua kali tidak berfungsi jika Anda memiliki kata sandi saat membuka kunci.

3. Sejak versi 2.0.2, fungsi untuk merekam panggilan telah muncul di Sailfish OS.
Untuk mengaktifkannya, buka Pengaturan (Aplikasi) - "Perekaman Panggilan".

Catatan:
Secara umum, di bagian Pengaturan pada bagian Aplikasi, Anda dapat mengonfigurasi banyak aplikasi yang dibuat oleh Jolla sendiri, serta melakukan beberapa tindakan pada aplikasi Android (menghapus cache, menghentikan program, dan sebagainya).

4. Mengklik dua kali pada tombol kunci, jika ponsel tidak terkunci, akan mengunci ponsel dengan getaran berikutnya, yang menunjukkan bahwa ponsel terkunci.
Contoh penggunaan: telepon ada di saku Anda dan Anda tidak yakin apakah terkunci, tetapi Anda tidak ingin mendapatkannya. Maka Anda hanya perlu merasakan tombol kunci dan ketuk dua kali sampai Anda merasakan getaran.

5. Anda dapat segera keluar dari aplikasi Tutorial. Untuk melakukan ini, klik kanan sudut di bagian paling awal, mulai dari kiri atas.

6. Dari versi 2.0.2, browser file sederhana “out of the box” muncul, untuk menemukannya buka Pengaturan (Penyimpanan) - Memori eksternal / Kartu memori (Data pengguna) - geser dari atas ke bawah - opsi Pengatur file (File manajer).

7. Dari versi 2.0.2, dimungkinkan untuk mengambil tangkapan layar, untuk ini, lakukan penekanan setengah detik pada kedua tombol volume.

8. Untuk para pecinta Web Semantic : Sailfish OS menggunakan beberapa teknologi dari Web Semantic:

[nemo@Jolla ~]$ tracker-
tracker-control  tracker-info     tracker-sparql   tracker-tag
tracker-import   tracker-search   tracker-stats


tracker-sparql!
Secara khusus, pelacak digunakan oleh media player.

Beberapa hal yang relatif jelas, tetapi untuk beberapa alasan beberapa tidak tahu tentang mereka:


-di dalam aplikasi Kamera (Kamera), Anda dapat mengambil foto hanya dengan mengklik kereta volume (meskipun aplikasi tidak akan memperbarui fokus sebelum foto);

-di aplikasi Kamera (Kamera), Anda dapat mengatur zoom dengan mendekatkan dua jari satu sama lain di layar dan menjauh satu sama lain; :)

-di Pengaturan, pada tab Gerakan, senang untuk mengaktifkan "penutupan aplikasi Cepat" sehingga Anda dapat segera menutup program dengan gesek dari atas ke bawah;

-ada, dalam gerakan, Anda dapat mengaktifkan svayp dari kiri ke kanan untuk masuk ke Pita Acara (Acara), bukan Layar Beranda (Beranda);

-Jika ponsel terkunci, lalu klik dua kali pada layar Anda akan dibawa ke Lock Screen (Lock Screen);

Wallpaper -Ubah untuk gambar Anda sendiri hanya dari galeri. Untuk melakukan ini, buka gambar di galeri, geser dari atas ke bawah, lalu pilih opsi "Buat suasana". Anda akan dibawa ke halaman pengaturan suasana, di mana Anda dapat menyesuaikan warna dengan selera Anda atau membiarkannya apa adanya dan menikmatinya;

-Dalam Pengaturan (Pengaturan) dalam "Input teks" (Input teks), Anda dapat mengaktifkan pemisahan keyboard dalam mode lansekap;

-beritahu untuk menempatkan Gudang- klien untuk Openrepos. Pertama, izinkan penginstalan aplikasi dari sumber yang tidak terpercaya: Pengaturan - “Perangkat Lunak Tidak Dipercaya”, kemudian unduh file .rpm dari tautan di atas dan instal dari konsol atau menggunakan aplikasi Browser File. Openrepos adalah repositori di mana ada sejumlah besar aplikasi yang tidak ada di Jolla Store karena berbagai alasan (misalnya, Jolla Store melarang penggunaan setan, lihat FAQ ). Tidak ada QA di Openrepos, yang membawa beberapa risiko, tetapi pada saat yang sama memberikan kebebasan penuh kepada pengembang.

Bonus


Beberapa aplikasi yang mungkin bermanfaat bagi Anda (menurut saya):
- SearchNemo - mencari file / teks tertentu dalam file teks di seluruh sistem file;

- LLs vPlayer (hanya di Openrepos) - pemutar video omnivora dengan jumlah barang yang layak yang tidak ada di pemutar biasa;

- Layar-pesan - menampilkan teks yang diketik di layar (misalnya, dapat digunakan untuk catatan);

- Sidudict adalah kamus offline yang baik berdasarkan QStarDict. Minus: jika Anda menambahkan sejumlah besar kamus, aplikasi macet, atau mulai terlalu lama, karena setiap kali Anda mulai mengindeks ulang kamus dari awal;

- peramban file- Peramban file yang luar biasa untuk Sailfish OS;

- Daftar - ToDo-organizer. Menurut pendapat saya, penulis sangat menarik mengalahkan fungsi Sampul;

- Mercusuar - monitor sistem untuk Sailfish OS;

- Log Baterai - aplikasi untuk memantau konsumsi baterai, dapat membangun grafik;

- Unplayer (hanya di Openrepos) - pemutar musik yang sangat baik dengan kemampuan untuk memutar musik langsung dari direktori;

- Kat adalah klien VK yang aktif berkembang;

- Battery Overlay - jika tidak biasa bagi Anda bahwa indikator pengisian hanya ada di Layar Utama, aplikasi ini akan menyelesaikan masalah;

-Perekam - perekam suara;

- Textractor - OCR;

- Penerjemah TAO - klien untuk Google, Yandex, kamus Bing;

- WiFi Analyzer - melihat jumlah dan kekuatan jaringan WiFi;

- TinyEdit - editor teks dengan fungsionalitas minimum;

- Papocchio - memungkinkan Anda untuk menggambar di layar.
