Memperkenalkan Manajemen Paket NuGet tingkat solusi dalam Visual Studio untuk Mac

Visual Studio 2019 untuk Mac versi 8.3 hadir dengan banyak fitur baru seperti yang dirangkum dalam posting blog ini . Meskipun keseluruhan rilis ini sangat dipengaruhi oleh umpan balik Anda, memiliki kemampuan untuk mengelola paket pada tingkat solusi adalah salah satu kemampuan yang sebagian besar dari Anda dinyatakan kurang dalam Visual Studio untuk Mac. NuGet Package Manager tingkat-solusi yang baru adalah salah satu fitur baru yang menarik dari Visual Studio 2019 untuk Mac versi 8.3.

Kami telah melakukan peningkatan untuk membantu Anda menemukan paket dengan lebih mudah. Ini termasuk peningkatan pengalaman saat mencari paket baru, mendapatkan pemahaman tentang paket apa yang sudah diinstal dalam proyek Anda, dan menemukan paket yang memiliki pembaruan yang tersedia. Dalam posting blog ini, kami akan fokus pada pengalaman manajemen paket untuk Solusi. Namun, sebagian besar pengalaman termasuk menginstal, memperbarui, dan melihat paket yang diinstal juga memiliki pengalaman baru yang serupa di tingkat proyek.

Untuk meluncurkan Manajer Paket NuGet untuk Solusi, Anda dapat pergi ke menu konteks untuk Solusi dan pilih "Kelola Paket NuGet ...":



Artikel ini di blog kami .

Tambahkan paket baru


Ketika Anda mencari dan mencoba untuk menambahkan paket baru, Anda sekarang dapat memilih proyek yang ingin Anda instal paket.

Kapan saja, Anda dapat pergi ke tab Terpasang dan melihat daftar semua paket yang diinstal dalam solusi Anda, yang memungkinkan Anda menghapus instalasi untuk memperbaruinya.



Perbarui paket


Tab Pembaruan menunjukkan kepada Anda semua paket dalam solusi yang pembaruannya tersedia (atau proyek, jika Anda menjalankan perintah di tingkat proyek).



Paket konsolidasi


Seringkali, solusi besar berakhir dalam situasi di mana proyek yang berbeda merujuk ke versi paket yang berbeda. Untuk mengkonsolidasikan versi-versi ini ke dalam satu versi tunggal dari paket yang mungkin ingin Anda gunakan di seluruh solusi, Anda dapat pergi ke tab konsolidasi dari NuGet Package Manager yang dipanggil pada node solusi, pilih versi paket untuk Anda inginkan semua proyek dalam solusi untuk menggunakan dan memilih untuk menggabungkan paket:



Unduh hari ini!


Untuk mencoba kemampuan NuGet baru ini, unduh Visual Studio 2019 untuk Mac versi 8.3 hari ini atau perbarui ke rilis terbaru menggunakan saluran Stable jika Anda sudah menginstal Visual Studio untuk Mac.

Jika Anda mengalami masalah dengan rilis versi 8.3, silakan gunakan menu Bantuan> Laporkan Masalah di IDE untuk memberi tahu kami tentang hal itu. Anda juga dapat memberikan saran untuk peningkatan Visual Studio for Mac di masa mendatang dengan menggunakan menu Berikan Saran.



Akhirnya, pastikan untuk mengikuti kami di Twitter di @VisualStudioMac untuk tetap mendapatkan kabar terbaru tentang Visual Studio untuk berita Mac dan beri tahu kami seperti apa pengalaman Anda. Kami berharap dapat mendengar dari Anda!
