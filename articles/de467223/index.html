Willkommen beim JavaScript Meetup SuperJob am 10. Oktober

Am 10. Oktober findet im SuperJob-Büro ein Treffen der JavaScript- Entwickler statt.

SuperJob-Spezialisten werden ihre Erfahrungen mit der Optimierung von Anwendungen auf React / Redux, der Verwendung statischer Typen mit Flow und der Arbeit mit Node.js teilen.
Traditionell stehen Burger und informelle Kommunikation auf dem Programm.

Datum und Uhrzeit : 10. Oktober von 19:00 bis 21:30 Uhr
Adresse : Moskau, Malaya Dmitrovka, Haus 20
Registrierung : superjob-meetup.timepad.ru/event/1063170/



Referenten:

1. Alexander Kukhta , Hauptentwickler von SuperJob mit dem Thema „Erfahrung mit JSON: API auf dem Client“ :

  • Wie wir 1.500 Rohdaten auf dem Client speichern
  • Wie wir mit ihnen in React-Komponenten arbeiten
  • Und wie man es nicht langsamer macht

2. Pjotr ​​Solopov , Leiter der SuperJob- Entwicklungsgruppe mit dem Thema " Typsysteme und Frontend-Typisierung" :

  • Welche Typsysteme gibt es und wie haben sie sich entwickelt?
  • Flow vs. TypeScript
  • Wie wir Flow in einem großen Projekt verwenden

3. Alexander Savin , SuperJob- Frontend-Entwicklungsmanager mit dem Thema „Erstellen eines JSON-Servers: API auf Node.js und PostgreSQL“ :

  • Mit welchen Tools können Sie schnell die API für eine vorhandene Datenbank abrufen
  • So erstellen Sie ein solches Tool selbst mit Node.js und PostgreSQL
  • Welche Schwierigkeiten können bei der Verwendung des „First DB“ -Ansatzes auftreten und was kann dazu beitragen, diese zu bewältigen?
