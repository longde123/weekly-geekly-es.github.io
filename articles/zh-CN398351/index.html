展示架


显示对角线为6.8英寸。它基于具有全高清分辨率的IPS矩阵。像素大小很小，因此图像看起来完全平滑。色彩还原非常好，亮度裕度相当大（400 cd / m2），在晴朗的晴天，我不必劳累眼睛。如上所述，整个正面都覆盖有耐用的大猩猩玻璃4。考虑到非常大的显示区域，这是一个重要的优势。

音效


ZenFone 3 Ultra配备了两个五磁铁扬声器。尽管您仍然不必对这些孩子抱有太大期望，但它的播放效果非常好，声音很大。我对他可以提供的耳机感兴趣。仍宣布24位/ 192 kHz的声音格式。当然，我的耳朵远离发烧友-Sennheiser HD 215-但即使在它们上面，我也能感觉到音质。从主观上讲，ZenFone 3 Ultra播放的音乐比我的主板华擎Z97 Extreme 3上的集成声音系统更为细腻和清晰

。ZenFone 3 Ultra的另一个重要功能是使用DTS耳机环绕声技术：X。在这种情况下，将实现7.1声道声音。先进的实时算法可处理音乐记录，电影音轨和游戏声音路径，从而产生音量错觉或增强已经存在的环绕声效果。 DTS耳机：X甚至可以“刷新” 128 kbps比特率的最高质量的录音。当声源和耳机最初支持DTS耳机：X时，可以达到最佳效果。如果您拥有优质的入耳式耳机，则可以从此演示视频中获得关于此技术功能的非常遥远的（！）想法。。没错，要做到这一点，绝不会与“声卡/耳机”的任何组合相提并论。因此，评估DTS耳机的最佳选择：X是去商店并在ZenFone 3 Ultra本身上聆听。仅为了获得最佳效果，请务必索取随附的耳机，因为它们支持该技术。

摄影机


主摄像头使用23像素（实际为22.5）像素的Sony IMX318矩阵，尺寸为2.6英寸。光圈f / 2.0，最大快门速度-32秒。六元素镜片由Largan Precision制造。本相机使用4轴光学稳定器，可以减少4级曝光，而不会出现运动现象。

多亏了大屏幕，在ZenFone 3 Ultra上的拍摄比在传统尺寸的智能手机上拍摄更为愉快。巨大的重量有助于保持框架更加自信，甚至光学稳定也有帮助。拍摄视频时，仅使用电子稳定器。为了确保正确的色彩渲染，在闪光灯上方使用了RGB传感器（色彩校正传感器）。此外，与当今电视中使用的解决方案类似，专有的True2Life +技术可提高视频输出质量。这是一组软件和硬件工具，负责提高屏幕上对象移动的平滑度，优化亮度，对比度和色彩还原。

ZenFone 3 Ultra的一个重要功能是TriTech三重自动对焦系统。为了聚焦，同时使用激光传感器以及相位和跟踪自动聚焦。制造商声称该相机能够在0.03秒内对焦。对于安静的拍摄而言，没关系，但是当您需要捕捉瞬间或拍摄对象快速移动时，高速自动对焦可能是决定性的优势。例如，如果跟踪自动对焦将快速而准确地工作，那么在拍摄视频时，您不太可能会花时间转动智能手机，并且图像会在很大的延迟后变得清晰。

我必须说ZenFone 3 Ultra从来没有真正失败过：在所有测试镜头中，没有一个聚焦错误。而且，在拍摄时，相机反应非常灵敏：只要触摸到画面的某个部分，相机就会立即对准该位置并根据所选区域的照度调整曝光。

一项有趣的创新是“实时HDR”功能。如果将其激活，则智能手机将动态执行所有必要的计算，并实时显示HDR图像的外观图片。该死的方便：现在您不必浪费时间先拍照，再检查发生了什么。

此外，对于摄影爱好者来说，最长32秒的手动快门速度模式将很有用。在夜间拍摄时，这可让您从行驶中的汽车前照灯中获得流行的光环效果。但是，为此，您必须拥有一个三脚架并以某种方式使智能手机适应该三脚架。



同时，请注意“超分辨率”模式：相机连续拍摄多张照片并将它们组合为一张图像，其细节相当于92兆像素。

软体类


ZenFone 3 Ultra在Android 6上运行。它使用传统的专有图形外壳Zen UI。幸运的是，最近，华硕（ASUS）一直在尝试通过各种实用程序将已售出设备的负载降至最低。尽管无处不在的xLink系列应用程序（PC Link，Share Link等）仍然存在。

性能表现


该智能手机基于Qualcomm Snapdragon 652处理器（64位，8核），工作频率高达1.8 GHz。RAM-3或4 GB。程序和数据存储的内置内存可以是32、64或128 GB。您也可以将Micro SD存储卡插入其中一个插槽。请注意，您可以使用：

  • 两个纳米格式的SIM卡（不带存储卡），
  • 一张SIM卡和一张存储卡。



处理器中内置的Adreno 510图形芯片负责处理图形。无需抱怨性能-一切顺利。

另外，我对移动电源功能感到满意。由于ZenFone 3 Ultra具有相当大的4600 mAh电池容量，因此您可以根据需要从智能手机为其他小工具充电。可以这么说，慷慨地分享能量储备。为此，只需在断开连接菜单中选择“反向充电模式”项，智能手机将关闭，然后您可以将其连接到其他设备，例如普通便携式电池。此功能的充电电量至少为20％。



要返回正常模式，只需打开智能手机。

支持快速充电功能：在最初的45分钟内，电池获得约60％的电量。至于ZenFone 3 Ultra的“寿命”，经过PC标记压力测试，它可以持续将近12个小时。



结论


尽管具有所有优点，ZenFone 3 Ultra并非适合所有人。这样做有两个原因-尺寸和重量。要完全抓住它，您需要拥有一个非常大的掌心。另外，由于其尺寸，它不能适合每个口袋。是的，他大大减轻了233克。因此，此小工具更适合携带背包。

我可以给谁建议？那些不仅需要智能手机，还需要功能广泛的多媒体设备的人。强大的处理器，大量的RAM和内置内存，出色的摄像头，出色的屏幕和出色的声卡级别的声音-对于许多人来说，ZenFone 3 Ultra可以代替平板电脑，因此在尺寸上赢得了很多。此外，得益于其摄像头，智能手机将成为某人的替代品和摄像头。您是否注意到人们在街上用平板电脑拍照？尽管ZenFone 3 Ultra的屏幕较小，但是用一只手握住它更方便，并且在拍照和视频功能方面，它大大超过了任何平板电脑和绝大多数智能手机。

在6.8英寸的屏幕上观看电影，看书和玩游戏非常方便，这比5英寸的智能手机要舒适得多。ZenFone 3 Ultra的图片非常好吃，当您上网浏览时，很快就会习惯在大屏幕上显示站点。之后，您将普通的5英寸智能手机的屏幕视为邮票。至于声音，在ZenFone 3 Ultra上通常很难找到竞争对手。即使您不喜欢演奏，但新品质的音乐绝对应该符合您的口味。
