3D扫描仪高达500,000卢布



3D打印的范围在不断扩大，其制造商提供的3D打印机和软件的数量也在增长，并且很快它们将无处不在。程序专家从头开始创建的对象的使用不适合任何目的，在某些地方需要真正存在的东西的精确副本-这是3D扫描仪可以提供帮助的地方。

在这篇评论中，我们将考虑初始价格水平的3D扫描仪，价格从几万卢布到50万卢布不等。

» XYZprinting -手持式



  • 扫描面积厘米：200 x 200 x 200
  • 精度，微米：1000
  • 使用颜色：是
  • 价格：18990卢布

紧凑型手持式扫描仪，符合人体工程学设计。设计用于沿主轴扫描10厘米至2米大小的物体。与同一制造商的3D打印机兼容，并且可以使用其软件。





» BQ-Ciclop


  • 扫描面积厘米：20.5 x 20.5
  • 精度，微米：500
  • 使用颜色：是
  • 价格：23990卢布。


扫描仪使用内置激光按照三角测量原理工作。它具有转盘，并且能够执行快速自动校准，从而大大简化了使用它的过程。制造商提供必要的使用软件，并在维修时提供零件3D打印的资源。




» 3D系统-Sense下一代


  • 扫描面积厘米：200 x 200 x 200
  • 精度，微米：900
  • 使用颜色：是
  • 价格：37900卢布



在简约的情况下，方便且极其易于使用的3D扫描仪。
扫描尺寸从20厘米开始的对象。它具有最佳的价格/质量比之一。





» 闪亮的3D-EinScan-S

  • 扫描面积厘米：20 x 20 x 20
  • 精度，微米：100
  • 使用颜色：是
  • 价格：79925卢布。

一种双模式扫描仪，可让您创建质量卓越的对象的3D模型。它以两种模式扫描-自动和免费。在自动模式下，扫描对象的大小可以达到20厘米，在手动模式下，可以达到70厘米。使用完整的转盘，以自动模式进行的完整3D扫描仅需三分钟。在手动模式下，取决于操作员的技能，最长70厘米总共需要大约10分钟。生成的3D模型的质量使其可以在不进行进一步编辑的情况下进行打印。





» MakerBot-数字化仪

  • 扫描面积厘米：20.3 x 20.3 x 20.3
  • 精度，微米：200
  • 使用颜色：否
  • 价格：81900卢布



来自3D打印机MakerBot的著名制造商的模型。它是一个带有内置转盘的整体式摄像机，配有摄像头和两个激光器。





RangeVision-智能

  • 扫描面积厘米：50 x 37.5 x 37.5

  • 使用颜色：是

  • 扫描区域，厘米：最大520x390x390

配有转盘和手提箱。在转盘上拍摄对象仅需12秒。





闪亮的3D-EinScan-Pro

  • 价格：321436擦。
