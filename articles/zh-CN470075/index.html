Pacsafe推出的城市新品：Intasafe X和Intasafe X Slim背包



最喜欢的笔记本电脑-可靠的背包。 这样就可以形成柔软的衬里，并可以放置充电器，适配器和加密狗。 今天，我们将讨论两个这样的15英寸对角线的笔记本电脑背包：Pacsafe Intasafe X和Pacsafe Intasafe X Slim。 奖励：防止盗窃。

保护的工作原理


我们已经讨论过Pacsafe品牌。 简而言之，该公司由狂热的旅行者Rob Shlipper和Magnus Maglashan于1998年成立。 从那时起，它的设计师就为旅行者（不仅是）创造了带有防扒手保护的背包和包。



保护是由Pacsafe设计人员的经验决定的，并已在实际情况下成功进行了测试。 对于Intasafe X和Intasafe X Slim，它包括三个部分。

第一个是主隔层的拉链锁。 两个滑杆的舌头彼此插入，并挂在锁杆上，锁闩上。 小偷不会安静而迅速地打开背包：您必须首先了解城堡的设计。 扒手正在寻找最简单的偷东西方法。 因此，他们很可能不会与Pacsafe锁通信。 闪电可靠，两倍，不会刺破手柄。



第二个是右肩带上的带扣。 绑带缠绕在固定的物体上：咖啡馆的桌腿，街道上的篱笆柱或树木。 转动不显眼的销子即可牢固地扣紧带扣，现在背包不会退缩。 钢吊索缝在皮带上，无法切割。



第三个是带屏蔽的内部口袋。 口袋可保护塑料卡和RFID标签免遭潜在的入侵。 此类攻击仍然很少见，但Pacsafe正在夏季准备雪橇。



通常，Pacsafe背包用钢网加固。 由于Intasafe X和Intasafe X Slim是都市型背包，因此设计师选择了简单的选择：厚厚的聚酯纤维面料，带有防切割保护。 它可以使用很长时间，并且不会被擦拭。

Pacsafe Intasafe X


这是两个背包中最宽敞的。 Intasafe X的体积为27升，尺寸为47.5 x 29 x 16厘米，重量为830克。 它适合作为大多数航空公司航班上的手提行李（我们提醒您：请提前检查特定航空公司的限制）。



在外面，背包有三个口袋。 一个，前面有一个隐藏的拉链。 它用于需要快速访问的小物品（但不适合钱包）。 每侧都藏有两个用于存放瓶子和雨伞的口袋。 带按钮的可调节口袋。



内部的隔层是一个，很大。 背面有一个密密麻麻的口袋，内衬柔软的15英寸笔记本电脑。 顶部是另一个，用于10英寸平板电脑，以及一个用于存放外部电池的独立口袋。 在它的正上方是电线孔。 穿过电线，将其内部连接至移动电源，现在您可以为智能手机充电，而无需从背包中取出电池。



在背包的后半部分，口袋的“收纳袋”显得比较有弹性。 这是一个宽大的口袋，带有一个带有带钥匙扣的拉链的拉链袋，里面有用于存放钥匙的卡扣，开放的口袋用于存放卡，电线或扁平的小工具，以及笔盒。 魔术贴口袋与屏蔽相同。



Intasafe X悬挂在行李箱的手柄上，为此，它的后部有一个松紧带。 背面柔软。

Pacsafe Intasafe X Slim


紧凑的Intasafe X Slim外形小巧-43 x 29 x 11 cm，也适合手提行李，也可以放在飞机座椅下。 背包的重量为510克，容量为20升。



在外面，背包没有口袋。 这意味着没有必要将重要的东西留给扒手，并且后者没有机会掏出任何东西。



内部有与Intasafe X几乎相同的组织。背面有密集的柔软口袋，可容纳15英寸的笔记本电脑和10英寸的平板电脑。 还有一个用于移动电源的口袋和一个用于为小工具充电的电线孔。



相对侧的“整理器”的排列略有不同。 没有大的拉链口袋，钥匙扣位于背包的主隔间内。 但是带有屏蔽层的魔术贴口袋和用于存放小物件的口袋就位了。
