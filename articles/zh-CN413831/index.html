特斯拉自动驾驶仪的新版本将于8月发布，首次具有“全自动驾驶功能”



特斯拉汽车公司首席执行官伊隆·马斯克（Ilon Musk） 表示 ，下一次重大的Autopilot 2.0软件升级将在8月进行。 这将是期待已久的版本9（即Autopilot 2.0 v9）。 先前的主要更新8.0于2016年进行。

即将发布的更新的主要特征是该程序将首次包含用于全自动驾驶的功能，即真正的自动驾驶仪。 “迄今为止，Autopilot合理地只专注于安全性，” Elon Musk在推文中写道。 “使用版本9，我们开始激活自动驾驶功能。”


专家说，出于法律原因和出于安全考虑，特斯拉不太可能将这些功能用于预期目的，即在没有人驾驶的情况下直接驾驶汽车。 但是事实本身很重要，因为经过这么多年的努力，自动驾驶程序最终实际上变成了自动驾驶仪，并且开始证明其名称是合理的，该名称以前与现实并不完全一致。

这是软件中第一次出现“ 完全自驾能力”选项包，并将逐步进行补充。 新软件将花费很多：用于运行Autopilot 2.0的汽车的改进型自动驾驶仪（增强型自动驾驶仪）的价格为5000美元，而完全自动驾驶功能包的价格为3000美元。

马斯克上周还宣布，增强型自动驾驶仪的“上坡/下坡”功能将在几个月内发布。 这种流行的选择将是自动驾驶仪的重大改进。 在这种模式下，程序将在解耦之前确定应重建哪个车道以及何时重建。 大会之前，将对其进行重建，降低速度并将控制权转移给驾驶员。 在高速公路上，此功能还允许您根据每个车道的流量和拥堵情况自动在车道上进行重建。

在“死亡曲线”上测试模型X中最新的自动驾驶仪更新（全环解耦）


去年，埃隆·马斯克（Elon Musk）宣布，自动驾驶仪功能不久将分为增强型自动驾驶仪和完全自动驾驶功能包。 由于该程序的更新时间比计划表晚，因此去年没有发生。 但是现在看来一切都在朝着这样的分离迈进。

特斯拉今年已经开始对Autopilot 2.0进行重大改进。 3月，程序中的神经网络被完全取代：这是自2017年6月特斯拉邀请著名的深度教育专家安德烈·卡帕蒂（Andrei Karpaty）工作以来的最大变化。

同时，美国国家运输安全委员会（NTSB）发布了事故报告的初稿，该报告于2018年3月23日上午9:27左右在美国101号高速公路（US-101）和国家高速公路（SH-85）的交界处发生。配备2017款Tesla Model X P100D和其他几款汽车。



该报告指出，驾驶员“激活了特斯拉称之为“自动驾驶”的车道保持和巡航控制功能。 事故发生前不久，驾驶员对汽车进行了编程，使其以120 km / h的速度行驶，并将手从方向盘上移开。 接近回旋处时，特斯拉全速驶入划分屏障，该屏障设置在减少行车道之前。 这是一个经常发生事故的非常危险的地方。 38岁的司机是苹果公司的软件开发人员，当场死亡。 根据汽车的记录，他有5秒150米的距离对障碍物做出反应，但他没有。

NTSB的专家发现驾驶员有罪，并再次指责特斯拉将其车道保持和巡航控制功能定位为“自动驾驶仪”，这鼓励驾驶员对汽车的信任超过其应有的程度。

令人欣慰的是，在最近对莫斯科交通繁忙的Yandex自动驾驶仪进行测试期间，没有发生任何事故。



从Big Pirogovskaya，以Teddy昵称的Toyota Prius自动驾驶仪以超过40 km / h的速度启航，并在郊区的赛车训练场上加速至70 km / h。
