Picaso 3D Designer X Pro的首次Live 3D打印机评测



期待已久的3D打印机落入我们的手中，我们已经撰写了不止一次。 以前从制造商处获得的信息很有用，但是-任何描述都不能与个人经验进行比较。 因此，这是Picaso 3D Designer X Pro的第一张实时照片评论。

活动结束后，打印机首次在Top 3D Expo大会上首次亮相，成为了我们研究并测试的对象。



规格：
  • 印刷材料：PLA，ABS，PLA柔性，PVA，PC，臀部，尼龙，Laywood，FilaFlex，Filamentarno。
  • 螺纹直径：1.75毫米。
  • 打印区域：200 x 200 x 210毫米。
  • 表壳材料：铝（复合材料）。
  • 导轨（钢）：X和Y-导轨，Z-圆柱形。
  • 层厚：0.05至0.25 mm，可通过软件调整。
  • 定位精度：XY：11微米，Z：1.25微米。
  • 存在加热平台：是。
  • 印刷平台：铝和玻璃。
  • 打印头数量：2。
  • 自动工作台校准：是。
  • 喷嘴直径：0.3毫米。
  • 挤出机的最高工作温度：380°C。
  • 最高工作平台温度：140°C。
  • 噪音等级：55 dB。
  • 控制板：基于ARM CORTEX M4 32位STM。
  • 接口：USB，以太网，USB闪存（包括驱动器）。
  • 操作系统兼容性：Windows XP，Windows 7，Windows 8-10。
  • 打印速度：最高30 cm3 /小时。
  • 支持的格式：STL，PLG。
  • 软体：PICASO 3D Polygon X
  • 工作环境温度：15°-32°C。
  • 尺寸：492x390x430mm;
  • 重量：16kg。
  • 尺寸和包装重量：690 x 500 x 470，22公斤。
  • 功耗：220V +-15％50Hz（可选110 V + -15％60Hz）400W。
  • 质保：2年。

现在可以以特惠价格 购买 3D打印机Picaso 3D Designer X Pro， 价格仅为199,000卢布。

在打印之前，您需要填充塑料，为此，我们在菜单中选择适当的项目-第一行的中央图标。 您可以在那里选择两种材料进行打印，并且温度和速度的自动设置（配置文件系统，在下面进行更多介绍）将有助于实现产品的平滑度和质量。



接下来，选择要打印的文件并开始该过程。 导轨由不锈钢制成。 在照片中，我们看到了同一台带有两个喷嘴和即时塑料开关系统的新型挤出机。



可以打印两种材料并在五秒钟之内切换的材料。 与以前的型号相比，这可以增强已经明显的打印速度差距。



处理器现在为32位，引擎间距最大为1/256。 打印头沿导轨的路径相当柔软。 塑料也顺畅流动。 此外，塑料饲料控制系统可以监控可能出现的问题，并且在许多情况下有助于避免结婚。



带灯丝的线圈从内部位于外壳的两侧。



所有基本结构元件均由铝和钢制成。 在明显的细节中，只有一种塑料-中央把手。



更改了表格的突出显示。 模型在工作区域的整个高度内被照亮。



背光不仅起到装饰作用，而且还发出当前状态的信号。




盖子配备了磁性闩锁，与以前的型号相比，可以紧贴身体。



在工作区域中，有2个风扇根据模式工作。 第一层的控制系统可调节从挤出机的喷嘴到印刷台的距离，每种塑料都不同，这有助于形成理想的第一层-高质量印刷的关键。



型材系统也有助于实现这一点：用户只需选择已安装材料的型材，打印机就可以根据其参数控制喷嘴温度，塑料进料速度和压力。



它可以正常工作：我们选择了工作喷嘴，轮廓，然后才开始打印。



挤出机配有0.15mm的喷嘴。 挤出机散热器配备有温度传感器，并且可以防止过热。 风扇根据温度负载改变散热器气流的速度。
