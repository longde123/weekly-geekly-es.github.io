Chromium Google

Bahkan jika Anda tidak memiliki akun Google, browser Chromium gratis masih berkomunikasi dengan server Google di latar belakang. Ini agak aneh, karena orang memasang Chromium bukan Chrome hanya untuk mendapatkan program yang bersih tanpa ikatan komersial. Namun, saat membuat Chromium dalam mode normal, gumpalan biner Google masih diunduh dan diinstal.

Proyek ungoogled-chromium adalah serangkaian flag konfigurasi, patch dan skrip khusus untuk menghapus integrasi dengan Google, meningkatkan pengaturan keamanan dan manajemen.

Pengguna anonim Github meminjam pengaturan dari beberapa proyek yang ada, yang juga mencoba memaksimalkan Chromium dari fungsi yang tidak perlu untuk membuat konfigurasi minimalis bersih: ini adalah Chromium dari distribusi Debian , kit Inox patch , serta browser Iridium , yang berfokus pada perlindungan privasi.

Para pengembang dari ketiga proyek tersebut telah melakukan upaya besar untuk menyingkirkan kehadiran Google di peramban gratis. Tetapi "Chromium takut" bahkan lebih paranoid dalam upaya ini.

Misalnya, penulis mengganti dalam kode sumber Chromium banyak domain web yang disebutkan dengan alternatif yang tidak ada - untuk memastikan bahwa browser tidak dapat mengirim informasi apa pun di mana pun tanpa sepengetahuan pengguna.

Selain itu, semua gumpalan biner dihapus dari kode sumber: program, pustaka, dan segala bentuk kode mesin lainnya. Hanya ada kode mesin yang diperlukan untuk perakitan (misalnya, icudtl.datuntuk mendukung Unicode atau *_page_model.bin, yang mendefinisikan model untuk membersihkan halaman di DOM Distiller).


Contoh membersihkan halaman di DOM Distiller

Dalam browser "googled", fungsionalitas khusus untuk domain Google, misalnya, Google Host Detector, Google URL Tracker, Google Cloud Messaging, Google Hotwording, dll., Sepenuhnya dinonaktifkan.

Penulis menambahkan beberapa pengaturan lebih murni sesuai selera Anda. Mungkin tidak semua orang akan menyukainya.

  • Penyedia pencarian "No Search" telah ditambahkan ke mahakotak untuk sepenuhnya menonaktifkan pencarian dari mahakotak.
  • Nonaktifkan pemformatan otomatis URL di mahakotak ketika menghilang dari tampilan http://, dll.
  • Nonaktifkan dialog JavaScript saat menutup halaman (acara onbeforeunload)
  • Paksa semua pop-up ke tab baru.

Selain itu, penulis mengubah beberapa pengaturan standar browser Iridium dan Inox. Misalnya, itu melarang ping wajib alamat IPv6 ketika mendeteksi ketersediaan IPv6.

Secara umum, Anda dapat "google" Chromium dengan berbagai cara. Selain Iridium dan Kotak Masuk, opsi lain kini telah muncul.

Rupanya, banyak orang memiliki keinginan untuk mendapatkan browser minimalis sederhana, tanpa "lonceng dan peluit" dan fungsi intelektual, cepat dan dapat diandalkan. Membuat peramban seperti itu dari nol hampir tidak mungkin, tetapi membersihkan Chromium dari slag adalah ide yang bagus.

Satu-satunya masalah adalah browser berkembang sangat cepat. Versi baru dirilis beberapa kali dalam setahun. Dalam Chromium yang sama, versi stabil sekarang 53.0.2785.116, tetapi Chrome 1 dan Chromium 1 dirilis baru-baru ini - pada 2 September 2008.

Ini bukan hanya kecintaan pengembang terhadap angka. Setiap versi baru benar-benar memperkenalkan standar web baru, fitur HTML5 baru dan dukungan untuk teknologi baru. Prosesnya berjalan cepat. Karena "fluiditas" seperti itu sangat sulit untuk melacak kode baru.

Untuk beberapa alasan, penulis berupaya untuk merilis versi secara paralel dengan cabang utama Chromium. Tapi ide yang menarik adalah membuat peramban ultra-minimalis yang pembaruannya tidak diperlukan sama sekali. Nonaktifkan Canvas, WebGL, WebRTC, dan hal-hal modern lainnya, nonaktifkan riwayat, cookie, akses browser ke peralatan, dan lainnya. Buat versi Tor untuk penjelajahan yang aman, dengan fungsionalitas minimal, hanya menyisakan pemblokir iklan dan pembersih halaman seperti DOM Distiller yang disebutkan sebelumnya. Setidaknya untuk perangkat seluler, peramban minimalis seperti itu akan berguna.

Versi terbaru dari Chromium yang dibersihkan untuk Windows, MacOS, Debian, dan Ubuntu diterbitkan di halaman ini.. Build on Windows hanya berjalan di Windows 7 x64 atau lebih baru. Untuk membangun, Anda akan memerlukan Visual Studio 2013 Update 4 atau Visual Studio 2015 Update 1, versi lain tidak didukung.

Anda dapat memasang ekstensi dari Toko Web Chrome di peramban, tetapi tidak melalui antarmuka web. Mereka perlu diunduh dan diinstal secara manual. Untuk mengunduh ekstensi dari Chrome Webstore, Anda dapat menggunakan tautan langsung:

https://clients2.google.com/service/update2/crx?response=redirect&prodversion=48.0&x=id%3D[EXTENSION_ID]%26installsource%3Dondemand%26uc

Di sini kami hanya [EXTENSION_ID]mengganti pengenal ekstensi dari direktori. Misalnya, cjpalhdlnbpafiamejdnhcphjbkeiagmpengidentifikasi untuk ekstensi Asal uBlock.

Setelah itu, file crx dipasang sebagai ekstensi di browser dengan salah satu dari beberapa cara, untuk dipilih, setidaknya dengan hanya menyeret mouse pada tab dengan ekstensi.
