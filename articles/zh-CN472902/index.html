Windows for IoT：增强的硬件支持和新的智能设备功能



物联网世界变得越来越有趣，对世界各地的所有业务领域都产生了影响，并且这种转变正在加速。 人工智能和机器学习的进步帮助各行各业的公司通过利用智能边缘设备和智能云的功能来提高效率，减少停机时间，降低成本和错误率并提高客户满意度。 本文将详细讨论Windows for IoT。

Windows已经在嵌入式工业设备上工作了二十多年，在日本CEATEC的一次演讲中，我分享了Microsoft的战略，根据该战略，IoT for Windows使各种级别的设备开发人员能够快速创建IoT解决方案并帮助客户进行数字转换。使用智能和安全的物联网设备。 CEATEC作为Society 5.0战略（“ Society 5.0”）的一部分，专注于智能城市，智能建筑和智能基础设施的开发，我们很高兴Windows for IoT提供了可扩展的操作系统家族，作为构建智能外围设备的基础。 这个家庭包括：

  • Windows 10 IoT核心版 。 该操作系统专为小型智能设备而设计，但同时提供Windows用户正在等待的级别的可管理性和安全性。 与Windows 10 IoT核心服务一起，提供了长期的OS支持，以及用于管理无线更新（OTA）和设备运行状况的服务。 Windows 10 IoT核心版的一个很好的例子是Hourfleet ，这是一个汽车共享平台，可以解决许多城市和社区面临的城市移动性问题。 Hourfleet是基于Windows 10 IoT核心版和Azure IoT的全球部署，可靠且具有成本效益的产品。
  • Windows 10 IoT企业版 。 Windows 10 IoT的此功能齐全的版本提供企业级可管理性，并限制具有固定功能的智能设备的功能。 像City Beacon这样的公司正在使用Windows 10 IoT Enterprise构建下一代智能基础架构。 City Beacon已构建了一个智能中心Smart Hub，可以将其放置在城市街道上。 这些是带有各种IoT设备的多功能信息亭，这些IoT设备连接到快速网络，可以为市民提供对超本地应用程序和服务的访问。
  • Windows Server IoT 2019 。 运行Windows Server IoT的服务器专为最苛刻的工作负载而设计，用于连接，存储和分析来自所连接的应用程序，网络和Web服务所使用的大型数据库的信息。

更多硬件平台


10月14日，我们宣布了适用于NXP i.MX系列处理器（包括i.MX6，i.MX7，i.MX 8M和i.MX 8M Mini处理器系列）的Windows 10 IoT核心板支持包的全面上市。

恩智浦是用于连接嵌入式应用程序的高安全性解决方案的领导者，而恩智浦i.MX和Windows 10 IoT核心版的结合为开发人员提供了用于创建商业级安全连接设备的其他选项。

Windows 10 IoT核心版可让您充分利用安全，智能的设备，并优化功耗。 开发人员可以使用熟悉的功能强大的Windows开发环境快速创建新的智能设备，并使用我们的Azure IoT平台将它们商业化。 通过增加对恩智浦处理器的支持，我们为物联网项目的创建者提供了使用许多新型工业级微电路的机会。

恩智浦安全物联网设备和云产品业务开发副总裁杰夫·迈尔斯（Jeff Miles）说：“我们很高兴达到通用状态，我们将继续与微软建立合作伙伴关系。” “现在，恩智浦客户可以使用行业领先的智能设备解决方案，并且可以将具有Windows 10 IoT核心版的i.MX 6，i.MX 7和i.MX 8M处理器用于工业应用，包括IoT项目。”发现Azure对您的智能产品的强大功能。”

消费者开始使用NXP i.MX和Windows 10 IoT核心版


已经有使用NXP i.MX处理器的主板和设备。 我们的合作伙伴已经发布并正在开发15个以上的此类板。 例如，Diagramics用Windows 10 IoT Core开发了基于NXP i.MX8的解决方案，用于创建根据Azure IoT设备目录认证的多功能客户端设备，并使这些IoT设备可以在插件的基础上在Azure IoT中心设备预配服务中注册。 n播放 在此处了解更多信息。

在演讲中，我还谈到了基于Windows 10 IoT核心版，NXP和Azure IoT的Smart Tower的新解决方案。 此解决方案旨在提高可靠性并减少关键基础设施（例如基站或配电网络）的停机时间。 Smart Tower开发了一种解决方案，可通过创建连接到塔的传感器模块和地面控制单元来充分利用对边界设备进行处理的优势，并考虑到高延迟或静音信号的可能性来实时分析塔的状态。 此外，该解决方案使用Azure IoT中心来管理和连接设备，并允许对塔的整个受控网络进行历史分析。 由于设备的极端条件，Smart Tower选择了NXP，i.MX和Windows 10 IoT Core的组合，而熟悉的Windows IoT平台减少了开发成本并缩短了上市时间。

持续的边缘边界创新


今年早些时候，我们在Windows 10 IoT Enterprise和Windows Server IoT 2019中提供了Azure IoT Edge支持。这使集成IoT Edge运行时变得容易，从而使您可以将机器学习算法和其他同样复杂的计算功能从云转移到外围设备。 Windows for IoT还支持Windows机器学习和Windows视觉技能，使您可以在任何Windows IoT设备上运行在Azure中开发的高级AI算法。

业界领先的边缘学习机器学习工具：新的Windows ML容器


Insider Program成员已经可以使用的新Windows ML容器是专门为需要使用Windows ML API创建的智能工作负载的任务而设计的。 Windows ML容器为企业提供了一个快速灵活的平台来创建企业级IoT解决方案。 它将高级Windows功能与Windows 10 IoT平台安全性和Azure IoT Edge可管理性相结合。

Windows ML容器提供智能的硬件加速工作负载，并可以直接访问主机外围设备。 托管的工作负载可以使用通过USB，I2C，SPI或GPIO连接的传感器或摄像机。 为了硬件加速，您可以使用任何DirectX 12 GPU或与Microsoft Compute Driver Model兼容的主机集成ASIC。

得益于专门针对AI工作负载优化的容器，磁盘上容器的大小约为350 MB，比市场上支持GPU处理的任何其他容器小得多。 较小的尺寸有助于Windows ML容器的管理并降低成本。 在此处了解有关Windows ML容器预览版的更多信息。

SQL Server IoT 2019简介：IoT的Windows数据存储和分析


SQL Server已可用于嵌入式设备很多年了，我们很高兴向嵌入式设备开发人员介绍SQL Server 2019的最新功能。 引入SQL Server IoT 2019（SQL Server 2019的二进制等效项），它将通过OEM渠道获得许可，专门设计用于在具有应用程序的边缘设备上运行的专用服务器上。 结合Windows Server IoT 2019，Microsoft SQL Server IoT 2019提供了一个全面的数据库平台，用于关键任务数据分析，具有无与伦比的可扩展性，行业领先的性能，安全性以及智能处理任何数据的能力。 SQL Server IoT使用内存技术实现对操作数据的实时分析，这些技术通过只读内存提供更高的并发性和可伸缩性。 Microsoft SQL Server IoT 2019将于今年晚些时候上市。
