微软的新Python课程[英文]

学习任何一种新编程语言的最大障碍可能就是知道从哪里开始。 重要的是什么？ 成为专业人士需要了解什么？ 当您甚至不确定要阅读的文档时，很难检查文档。

也许您在看Python。 也许是因为它的受欢迎程度而被吸引。 也许您喜欢它的灵活性。 毕竟，借助Python，您可以创建任何形状和大小的解决方案，可以沉浸于Web开发中，可以借助自动化来简化生活，或者可以借助机器学习来创造未来。

对于现代开发人员来说，选择一种新语言是一种常见的情况。 完全专注于一种语言的职业生涯的日子已经一去不复返了。

幸运的是，从一种编程语言切换到另一种编程语言时，概念通常不会改变。 当然，语法可能有所不同，但是if语句仍然是if语句 ，即使它是使用{}或End If编写的。 因此，您不需要学习如何编程，而是需要学习如何使用新语言进行编程 。

我们的同事Susan和Chris为初学者创建了一系列有关Python的教程视频 ！ 您可以知道如何编写代码，例如，使用JavaScript，Java或C＃（或COBOL，Bash或...没关系）。 您可能上过大学，在线或读过书，而无需了解if表达式是什么。 您宁愿了解Python中的if语句是什么样的。



Susan和Chris记得他们的第一个Python课，以及他们当时想学习的内容。 他们还研究了各种简短的教程，例如Face API ，并向人们学习了成功学习该指南所需的知识。 之后，他们创建了一个课程。

尽管本课程不会涵盖您需要了解的关于Python的所有知识，但同事们还是相信，他们将从常规的日常编码和脚本开始，谈论Python编程中最重要的原理。 完成课程后，您将能够按照文档，学习指南，书籍等进行独立学习。我们承诺，我们将迅速准确地采取行动，以便您将自己沉浸在最感兴趣的领域（或感到尴尬！）。

立即加入！ 顺便说一句，您总是可以在Twitter上找到Susan和Chris： @hockeygeekgirl和@geektrainer 。
