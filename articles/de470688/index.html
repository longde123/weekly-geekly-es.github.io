Was ist über VMworld 2019 bekannt?

Es findet vom 4. bis 7. November in Barcelona statt. Wir sagen Ihnen, welche Technologien VMware-Experten und eingeladene Redner diskutieren werden: von der Virtualisierung bis zum maschinellen Lernen.


Fotos - Kostenlose Blog-Fotos - CC BY-SA

Was für eine Konferenz


VMworld ist die jährliche VMware-Konferenz, auf der das Unternehmen technologische Innovationen vorstellt. Jede Veranstaltung bringt 13.000 IT-Experten zusammen. Zum Vergleich: Bei der ersten Konferenz im Jahr 2004 waren es nur 1,5 Tausend.

Die folgenden Themenbereiche wurden im VMworld 2019-Programm identifiziert: Hybrid- und Multi-Cloud-Umgebungen, Anwendungsverwaltung, Netzwerksicherheit sowie maschinelles Lernen und IoT. In nur vier Tagen werden mehr als 600 Panels abgehalten .

Worüber werden sie reden?


Über Änderungen in der Architektur von Rechenzentren. Im Rechenzentrum ist die Datenübertragung über das Netzwerk zehnmal langsamer als der lokale Zugriff auf den Arbeitsspeicher. Für einige Aufgaben ist dieser Unterschied unbedeutend und wird teilweise durch Technologien wie RDMA (Remote Direct Memory Access) ausgeglichen. Eines der Hauptthemen der Konferenz wird daher VMware Cluster Memory sein. Damit können Rechenzentrumsbetreiber Cluster aus sogenannten Speicherservern erstellen. Diese Server bieten bei Bedarf Speicher für ESXi-Hosts und virtuelle Maschinen. Auf der Konferenz werden die VMware-Ingenieure die Prinzipien eines speziellen Seitenmechanismus vorstellen, der für die Adressierung an einen Remote-Server anstelle einer lokalen Festplatte erforderlich ist.

Vertreter von VMware werden auch über den Software-Stack zum Erstellen softwaredefinierter Rechenzentren sprechen.

Über Hybridwolken. Sie werden Tools für IaaS-Anbieter diskutieren. Zum Beispiel das vRealize Suite-System zum Verwalten verschiedener Kombinationen von Cloud-Umgebungen. Es werden Meisterkurse zum Debuggen und Verwalten erwartet. Kubernetes werden nicht vorbeikommen. Cormac Hogan, Director von VMware Technology, und Kollegen werden darüber sprechen, wie containerisierte Anwendungen auf vSphere bereitgestellt werden, damit ihre Leistung nicht beeinträchtigt wird.

Über Netzwerklösungen. Insbesondere über VMware NSX SD-WAN. Das System vereinfacht die Verteilung von Sicherheitsrichtlinien in einer Umgebung mit mehreren Clouds. VMware-Experten erklären Ihnen auch, wie Sie Netzwerke im Rechenzentrum eines Cloud-Anbieters schützen.

Informationen zum Einrichten eines digitalen Arbeitsbereichs. Neben Tools für Anbieter werden auch Tools für Benutzer von Cloud-Diensten vorgestellt. Sie werden über die Innovationen von Workspace ONE sprechen. Dies ist eine Lösung für die Verwaltung von Anwendungen auf Desktops und Mobilgeräten. Es geht um die Intelligent Hub-Funktion zur Identifizierung von Benutzern mit BYOD , die AirWatch Agent ersetzt hat. Identity Manager für die Verwaltung von Anmeldeinformationen wird nicht vergessen.

Die Referenten analysieren auch die Funktionen von VMware Horizon 7 und Lösungen von Partnerunternehmen: SAP, NetApp und Oracle. Die VMware-Ingenieure führen eine Reihe von Workshops zur Konfiguration in einer virtuellen Umgebung durch. Die Konferenz wird auch eine offene Diskussion mit Experten auf dem Gebiet des Endbenutzer-Computing beinhalten . Sie beantworten Fragen zum Betrieb von RAS-Systemen für Unternehmensressourcen.

Was noch zu warten


Konferenzthemen sind unter anderem maschinelles Lernen. Wie künstliche Intelligenzsysteme und periphere Computer die Geschäftsprozesse von Unternehmen in den nächsten 3-5 Jahren verändern werden. Die Referenten diskutieren intelligente Algorithmen, die die Qualität des technischen Supports verbessern und die Verhaltensmuster der Benutzer beeinflussen. Eine vollständige Liste der Themen, Redner und einen Zeitplan für die Reden finden Sie auf der offiziellen Website der Veranstaltung .

Während der Konferenz werden Live-Reden übertragen. Daher können Sie VMworld besuchen, ohne Ihr Zuhause zu verlassen. Die Aufzeichnungen werden später in die Videobibliothek hochgeladen, wo Präsentationen von Rednern der letzten vier Jahre gesammelt werden.
