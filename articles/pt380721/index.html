Pulso sob controle? Como e por que medir o pulso em movimento

Agora, um número muito grande de uma variedade de aparelhos portáteis para medir a freqüência cardíaca apareceu no mercado. Para entender qual é a diferença fundamental entre eles, e para qual finalidade um dispositivo é mais adequado e para qual outro, torna-se uma tarefa não trivial.



Vamos tentar descrever os principais recursos técnicos que determinam a funcionalidade do monitor de freqüência cardíaca, para que um usuário não treinado não pague demais por um dispositivo com funcionalidade excessiva,ou não esperava resolver tarefas que ele não poderia realizar com um dispositivo simples.

Portanto, apesar da grande variedade de fabricantes e modelos, praticamente todos os dispositivos para medir a frequência cardíaca podem ser divididos em algumas das principais características:
- a tecnologia usada para medir a freqüência cardíaca;
- usado protocolo de transferência de dados sem fio;

O fator determinante em termos de precisão, qualidade e conveniência da medição de pulso é a tecnologia usada para isso.
De fato, a única revisão de tecnologias para medir o pulso no runet foi encontrada apenas em tempos de nerd ( um e dois), que não surpreende mais, mas se torna a regra. Nem todas as tecnologias possíveis podem ser usadas para os dispositivos móveis descritos por nós, portanto, consideramos apenas duas:

Eletrocardiografia - registro de alterações nos campos elétricos que ocorrem durante o trabalho do coração, para identificar possíveis diferenças, é necessário medir pelo menos dois pontos do corpo, se possível distantes um do outro friend



As figuras mostram dois tipos de tais dispositivos, apesar das diferenças externas, a tecnologia de medição e, consequentemente, a precisão, estão próximas.

Pletismografia- Medição da frequência cardíaca com base no registro de alterações no volume em um órgão cheio de sangue (geralmente nos vasos sanguíneos). A onda de pulso resultante disso pode ser medida usando métodos mecânicos, ópticos ou de impedância (eletrodo). Para aparelhos móveis, a pletismografia óptica é quase exclusivamente usada - fotopletismografia.

Para garantir que a frequência cardíaca recebida não permaneça no sensor de frequência cardíaca, mas possa ser mostrada em tempo real ao usuário, é necessário fornecer transmissão de dados sem fio do sensor para o dispositivo de exibição de informações ou do dispositivo combinado sensor / relógio / rastreador de fitness para um computador para análise posterior.
Agora, praticamente todos os sensores modernos usam uma de duas ou ambas as tecnologias de transferência de dados de uma só vez:

Bluetooth / Bluetooth 4.0 BLE é um protocolo de transferência de dados sem fio usado em um grande número de dispositivos para transferir dados entre dispositivos ou sincronizá-los com um computador.

Formiga / Formiga +- Um protocolo de transferência de dados sem fio especializado, projetado especificamente para a troca de dados entre dispositivos projetados para esportes / saúde.

Depois de olhar para a mesa, é fácil fazer uma escolha com base nas finalidades para as quais o monitor de freqüência cardíaca é comprado.

Se você quiser entender, em princípio, o que é um pulso, como ele muda durante o dia ou durante os esportes, aprende a gerenciar a carga de acordo com o seu batimento cardíaco, tente a pulseira Runtastic mais simples , que fornecerá respostas para essas perguntas, e permitirá que você comece a acumular a história de suas ocupações.

Se você tiver certeza de que, com o tempo, usará cada vez mais as leituras do monitor de freqüência cardíaca e, em um futuro próximo, mudará para o uso de dispositivos de corrida especiais que mostram todas as estatísticas do treino (não apenas a frequência cardíaca, mas também o ritmo / velocidade / tempo / distância, etc.) na tela horas, sua escolha para o primeiro passo: um sensor de frequência cardíaca de um dos líderesGarmin / Polar / Suunto .
Inicialmente, ele pode ser usado com um dos que executam aplicativos para um smartphone, e, se necessário, você pode comprar um dispositivo de pleno direito para controlar o processo de formação, por exemplo, o Garmin 910XT / Polar RCX5 / Suunto Ambit 3

Devemos considerar também os fones de ouvido atualmente únicas, Jabra Esporte pulso .
Graças ao desenvolvimento da qualidade dos sensores baseados na tecnologia fotopletismográfica para medir o pulso, tornou-se possível integrar esse sensor óptico no fone de ouvido e medir o pulso diretamente dentro da aurícula.

Dada a integração sem fio dos fones de ouvido com o telefone, obtemos um sensor de freqüência cardíaca compacto e conveniente e fones de ouvido de corrida de alta qualidade, sem fios extras e dispositivos adicionais para medir a freqüência cardíaca.
Para aqueles que estão acostumados a andar com música, esta solução agora será a mais original e conveniente.
