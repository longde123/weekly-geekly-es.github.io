3CX v16 Pembaruan 1, aplikasi 3CX untuk iOS Beta dan versi baru 3CX Call Flow Designer

Memperkenalkan ulasan produk 3CX terbaru. Akan ada banyak hal menarik - jangan beralih!

Pembaruan 3CX v16 1


Kami baru-baru ini merilis Pembaruan 3CX v16 berikutnya 1. Pembaruan mencakup fitur obrolan baru dan widget komunikasi yang diperbarui untuk situs Obrolan & Bicara 3CX Anda. Pembaruan 1 juga memperkenalkan Layanan Alur Panggilan baru, yang menambahkan antarmuka skrip kontrol panggilan ke PBX. Script "engine" bekerja bersama-sama dengan lingkungan pengembangan Call Flow Designer dan memungkinkan Anda untuk membuat skrip pemrosesan panggilan dari segala kompleksitas.

Obrolan diperbarui di klien web




Obrolan yang diperbarui sekarang memungkinkan Anda untuk mengelola percakapan Anda secara fleksibel. Selain itu, ia berinteraksi dengan baik dengan widget komunikasi Live Chat & Talk 3CX .

  • Pengunjung situs dapat memulai obrolan dengan Antrian Operator 3CX. Ini membuat grup obrolan, yang mencakup semua operator antrian dan pengunjung ini.
  • Di masa mendatang, operator Antrian dapat mengalihkan obrolan grup ke dirinya sendiri dan melanjutkan komunikasi pribadi dengan pengunjung.
  • Selain itu, operator dapat mentransfer obrolan ke operator lain atau pengguna PBX biasa, jika perlu.
  • Untuk membongkar antarmuka klien web, percakapan yang dipilih dapat ditransfer ke arsip (namun, mereka tidak dihapus).
  • Berbagai jenis obrolan (dari situs, grup, dll.) Sekarang memiliki ikon berbeda untuk memudahkan identifikasi.
  • Sekarang Anda dapat dengan cepat mengirim email ke pengunjung situs dengan mengklik email di jendela obrolan.

Fitur-fitur baru dijelaskan secara lebih rinci dalam Panduan Obrolan dan dalam video .

Widget 3CX Live Chat & Talk yang diperbarui




Widget 3CX Live Chat & Talk yang diperbarui menawarkan kustomisasi tambahan antarmuka dan integrasi yang disempurnakan dengan situs-situs yang dibuat pada WordPress CMS dan pada teknologi lainnya.

  • Mengatur ikon jendela obrolan - Anda dapat mengatur gambar yang sesuai untuk judul jendela obrolan. Ini bisa berupa, misalnya, logo perusahaan Anda.
  • Mengatur ikon operator - Anda juga dapat mengatur ikon operator obrolan, misalnya fotonya.
  • Penempatan widget - parameter "Posisi" menentukan penempatan widget di halaman web - kanan bawah (default) atau kiri bawah.
  • Tampilan di browser seluler adalah tambahan kecil tapi penting. Sekarang, ketika mengakses situs dari perangkat seluler, jendela obrolan diminimalkan secara default.

Antarmuka skrip, Call Flow Service


3CX v16 Pembaruan 1 memperkenalkan antarmuka skrip Layanan Aplikasi Panggilan Aliran yang baru. Ini mendukung aplikasi suara standar baru 3CX. Namun, aplikasi yang sudah ada dapat dikonversi atau dimodifikasi dalam versi baru Call Flow Designer (lihat di bawah). Arsitektur Layanan Aplikasi Call Flow diselesaikan hari ini. Server aplikasi berjalan pada 3CX untuk Debian / Raspbian Linux dan Windows.

Aplikasi 3CX baru untuk iOS beta


Kami belum lama memperbarui aplikasi iOS 3CX milik kami. Beberapa pengguna bahkan mengeluh tentang transfer file yang salah. Tetapi dalam pembaruan berikutnya, semua masalah telah diperbaiki! Kali ini, penekanannya adalah pada kemampuan obrolan terintegrasi. Sekarang obrolan dalam aplikasi seluler praktis tidak kalah dengan obrolan di klien web 3CX.



Aplikasi ini memiliki kemampuan untuk membuat obrolan grup dan memberinya nama. Juga menambahkan pengarsipan obrolan. Untuk mengarsipkan percakapan, geser ke kiri di atasnya (Anda dapat mengembalikan percakapan dari arsip dengan cara yang sama).

Juga di aplikasi ada obrolan dengan pengunjung ke situs melalui widget komunikasi 3CX Live Chat & Talk. Begini cara penerapannya sekarang:

  • Geser ke kiri untuk mengobrol untuk menemukan opsi tambahan: Ambil sendiri, Transfer, Akhiri dan Hapus.
  • Ikon obrolan dari situs berbeda dari ikon obrolan biasa sehingga Anda dapat dengan mudah membedakannya.
  • Pemberitahuan PUSH dari pesan dari situs menunjukkan nama pengunjung dan konten pesan.
  • Obrolan yang dikirim ke Antrian Agen menyertakan nama Antrian untuk kenyamanan Anda.

Coba aplikasi 3CX baru untuk iOS beta melalui TestFlight !

Perancang Alur Panggilan Panggilan Rilis 3CX Baru


Baru minggu ini, kami merilis versi baru dari lingkungan pengembangan aplikasi suara 3CX CFD . Ini memperkenalkan komponen C # baru, antarmuka pengguna yang disempurnakan, penanganan kesalahan ditingkatkan, serta pembaruan otomatis dari lingkungan pengembangan. CFD baru diperlukan untuk membuat aplikasi suara tipe baru untuk Pembaruan 1 3CX v16 dan lebih tinggi.


Lingkungan pengembangan yang diperbarui (IDE) CFD menawarkan alat tambahan untuk pengembang:

  • Komponen baru Menjalankan C # File dan Kode. Mereka mengganti komponen Launch Script Eksternal warisan. Komponen dapat menjalankan file kode C # atau potongan kode inline langsung dari aplikasi CFD.
  • Komponen "Tetapkan status ekstensi" yang baru mengimplementasikan pengaturan sederhana parameter ekstensi dari aplikasi CFD.
  • Penanganan kesalahan yang ditingkatkan. Editor Ekspresi baru tambahan memeriksa nilai-nilai, mendeteksi kesalahan pada tahap kompilasi.

Selain peningkatan terkait proses pengembangan, kami telah menambahkan beberapa fungsi yang meningkatkan kegunaan aplikasi itu sendiri:

  • Aplikasi pembaruan otomatis. CFD sekarang secara otomatis memeriksa versi baru dan menginstal pembaruan segera setelah dirilis.
  • Item menu baru "Save Project As" memungkinkan Anda untuk menyimpan proyek CFD dengan nama baru atau di lokasi yang berbeda.
  • Menu konteks baru "Buka Folder Audio" untuk komponen yang menggunakan file audio. Ini membuka Explorer untuk meramban folder audio proyek.
  • Tampilan yang mudah dari komponen yang dinonaktifkan. Mereka sekarang berwarna abu-abu berbeda dari komponen aktif CFD.

Rilis CFD baru melibatkan penggunaan Pembaruan 3CX V16 1. Unduh CFD dan instal dari Panduan Instalasi Desainer Aliran Panggilan .
