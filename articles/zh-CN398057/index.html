[评论]利奥波德FC750R。无可挑剔的美学



机械键盘的世界不仅限于Razer和SteelSeries等著名品牌。除了这些公司生产的机械手之外，鲜为人知的制造商也提供了更多有趣的选择。

为什么对他们知之甚少？制造商将此类键盘定位为专业人士的键盘，而不是通过电子竞技为广告和促销投入金钱。即，这是一种对工艺质量有很高要求的设备。

Leopold FC750R是您可以玩耍和工作的日常工作工具。在白雪皑皑的执行。

一天有点恶意




“办公室键盘！没有RGB背光！没有功能和软件！不支持电子竞技！多少钱？ 9000卢布？是的，你在开玩笑。以及如何为“它”付出这样的钱？我坐在膜上已经N年了，我将同时坐在那里！”

经验丰富的用户也可以在线看到其他类似的陈述。在俄罗斯零售中，这款键盘的价格将近9000卢布。我会回答你的。外观，优质材料，周到的人体工程学，Cherry MX开关，耐用性。一看照片就足够了。



似乎每个人都已经看到了所有这些，并且所有这些已经很熟悉了。我也是这么想的，直到我把这个键盘掌握在自己手中。之后，我意识到以前使用的几乎所有键盘都具有中等质量。我们可以说这通常是一个怪胎设备。好吧……无论如何，这是正确的，因为并非每个用户都同意为没有功能，铃声和口哨声的键盘提供这样的金额。

简单的数学运算：键盘的总价是其创建和升级所花费的总费用。而且，如果同一Razer营地中的机械师满是可能，那么具有相同价格标签的Leopold就不会出现在广告横幅上。赶上余额？Razer在促销上花了更多的钱，因为他们有一个名字而不是最好的做工，而Leopold却有出色的做工并有自己的名字，尽管范围很窄。在我所见过的所有键盘中，Leopold FC750R也许在“从工厂”发行设备时拥有最佳的做工。不是自定义程序集，不是修订版，即“开箱即用”的键盘。

箱子及设备




键盘装在一个相当厚实的盒子中，盒子由厚纸板制成。该包装盒可保护键盘免受损坏，并将各种广告语和其他营销技巧降至最低。键盘及其功能，完全没有键盘本身的图像（很可能是您通过互联网订购的，因此该包装盒不需要代表视图），键盘的颜色和开关的类型都是全面的信息。我认为，这就足够了。与其他彩色包装盒相比，这个彩色包装盒什么都没有脱颖而出，很可能会降低设备的最终总成本。但是包装盒中的物品更为重要。



盒子内部是键盘本身，乍一看我想快速将其卸下并戳一下，还有一根用于连接的可移动电线，两个可移动键帽（用于Caps Lock键和空格键），一个用于PS / 2的适配器和一个用于移除键帽的工具。此外，键盘还有一个塑料盖，可以防止灰尘进入，完成键盘操作后，在日常模式下使用它非常方便。



一般来说-绅士的场景。有您需要的一切，甚至更多。但是，有两点不满意：硬线和用于移除键帽（拉出器）的工具。我更希望看到一个拉线器，因为一个完整的拉线器在卸下键帽时会刮擦键帽。但是，考虑到键盘上的塑料键帽较厚，这种细微的选择可以忽略不计。

外观，性能和重要细节




Leopold FC750R白色-缩短（TKL）键盘，没有数字键盘。键盘缺少软件，宏键以及游戏键盘特有的其他铃声。我的键盘基于Cherry MX Red线性开关构建。键盘还具有MX Blue，MX Brown和MX Black开关。



在其他厂商几乎没有“库存”的产品中，几乎没有人会提供有趣的功能：键盘内部的吸音基底可降低打印噪音，以及由PBT塑料制成的1.5毫米厚键帽。键盘的重量约为1千克，对于缩短的键盘，此数字非常重要。





键盘采用经典的外形-严格而整洁的矩形，没有任何不必要的细节。这种表现给我留下了最深刻的印象：它看起来简单，独创，但却吸引了眼球。此外，键盘还具有惊人的精致乳白色。这不是在某些预算模型中可以找到的那种颠簸的灰色塑料。而且，随着时间的流逝，键盘上的灰尘将被清除，但是塑料不会失去其原有的外观。





键盘盒由磨砂白色塑料制成，摸起来有些粗糙。这种情况非常实用，不会积聚灰尘或指纹。至少我没有机会看到他们。键盘布局是经典的美式（ANSI）-两下长按，一层输入。键的数目为87。一系列F键从数字“ 2”的正上方开始。各种形式的经典。下排的所有键都有相同的比例（空格除外），因此您可以在键盘上放置不同的键帽。顺带一提，许多业主都这样做。





所有键盘修改器（大写锁定，滚动锁定）在相同名称的键上都有自己的LED，这些LED会亮蓝色。当您按Fn + F5组合键时，将打开游戏模式（Windows键已关闭），并且F5键上的指示灯完全相同。在白天，这绝对不重要，但是在晚上，这些LED会引起眼睛的不适。





我们深入研究细节。键盘的键帽由PBT塑料制成。在大多数键盘上，键帽由ABS塑料制成，并且会随着时间的流逝而抛光。而且PBT塑料键帽非常耐用-它们不惧怕物理，化学或热（我说的是紫外线）效应，因此这些键帽可以使用数年而不会失去其原始性能。实际上，这意味着键帽不会变黄并且不会随着时间的推移而磨损，并且在清洁键盘时，可以轻松地用清洁剂对其进行清洗。





键帽的表面摸起来有些粗糙，因此手指感觉非常粗糙。但是，与大多数键盘上常用的油漆相比，在游戏或打字时稍微潮湿的手指可能会滑一些。灰尘和污垢几乎不会残留在键帽上，但是经过长时间使用后，仍然必须清洁它们。





模拟人生在键帽上的位置非常有趣。所有字符都位于键帽的末端，这种设计称为“正面打印”（有时将此设计称为“正面打印”）。首先，我们总是以锐角看待键盘，因此在工作时这种字符排列更加方便阅读。其次，在这种情况下，键帽表面保持不变，也就是说，手指不会感到任何粗糙，毛刺和其他东西。第三，不会删除这些符号，因为我们的手指没有接触它们。尽管专用于此键盘，但由于它们是使用激光雕刻制作的，因此无论如何都不会删除字符。键盘上没有俄语字符，但是何时停止使用盲打法的人呢？







键帽的轮廓也是非标准的，并具有自己的名称Step Sculpture2。就倾斜角度和其他功能而言，键帽的轮廓接近于大多数机械键盘中常用的Cherry轮廓。但是，这些键帽略低，这带来了两个功能：打字速度提高（通常，在低调按键上键入通常会更容易-请记住，笔记本电脑的键盘实际上是手指在键上飞来飞去的），而且感觉就像手指卡在了键上因此，抑郁症得到了很好的控制。在Leopold FC750R旁边，我安装了常用的MK Disco键盘，其按键盖具有标准的Cherry轮廓（上面是带有相同MK Disco的按键盖的宏观照片-普通绘画和ABS塑料）。





键帽的厚度为1.5毫米。卸下后，发现边缘上的键帽非常锋利，并且每个键帽内部都有一个标记。大键（例如空格或班次）可指示键的比例（例如，空格为6.25，回车为2.25）。所有这些细节都非常关注。有趣的一点：对于发光的键帽（F5，ScrLk，CapsLock），内部由透明塑料组成，用于透射来自LED的光。



利奥波德FC750R的长键下方是樱桃稳定器。这是我见过的最好的实现。它没有修订和解析键盘。让我解释一下：大量制造商选择这些稳定器作为普通人最简单的选择-轻松卸下键帽，清洁键盘等。



但是没有人给予他们应有的关注，因此这些稳定器会尽快响起，发出嘎嘎声。您按一个键，然后单击，就会有一个ter不休。这完全取决于稳定器的有效实现，而Leopold它们……就像它们应该与任何普通键盘一样。所有长键的按下力与常规键相同，当按下时，任何地方都不会发出嘎嘎声。看起来很简单。但是几乎没有人关注这一细节。真棒，利奥波德！韩国人为此做了努力。





在我的键盘版本中，是Cherry MX Red开关。这些是压力为45克的线性开关。操作前的行程为2 mm，按键的总行程为4 mm。点击资源为5000万次。线性开关的独特之处在于，它们在被触发时没有任何触觉障碍-按键刚好落下，并在特定时刻触发。我将立即说明，切换类型的选择取决于个人喜好，并且没有划分为游戏切换或工作切换。我在线性开关上工作并玩得很舒服。同样有趣的是，键盘背衬是黑色的。考虑到整个键盘是雪白色的，这有点奇怪。





键盘底部具有先进的人体工程学设计。您可能会注意到，键盘的底部有6条由柔软且有弹性的橡胶制成的白色支脚，但在正常位置，键盘仅接触其中4条。仅当将键盘放在可伸展的支脚上时，位于下边缘的另外两个支脚才变为“活动”状态。此外，键盘上的可伸缩支脚也涂有橡胶，因此可以在任何位置稳定站立。



键盘线是可移动的，通过mini-usb连接。电线不像键盘本身那样白-一根1.8米长的普通灰色电缆。我不是很喜欢它，因为它很坚固而且不太愿意记住它的格式。但是，您始终可以拾起电缆，这不是问题。键盘还具有电缆管理功能，因此电缆可以显示在左侧，右侧和中间。



通常，所有这些乍一看都与键盘性能有关。您无需多说-经典外观，所有组件都组装得非常高品质，而且声音很好，在组装过程中没有多余的零件，毛刺或侧边。如果您握住并摇动键盘，那么您将根本听不到任何声音-这是她的整体组件。

键盘控制（NKRO模式，多媒体）




键盘缺少软件，但并非没有附加功能。键盘基于即插即用原理工作，因此不再考虑配置文件和其他废话。有这样的人：有人需要软件，而有人则不需要。我从未在任何键盘上使用过它们，因此这种方法很适合我。

键盘支持通过USB同时按下所有键。加载USB总线后，没有人会使用套件随附的PS / 2适配器。键盘没有允许您更改键盘布局的DIP开关，并且键盘不适用于Mac。仅Windows和Linux。

键盘功能通过Fn + F *组合键控制：
  • Fn +主页-切换到NKRO模式
  • Fn + End-切换到6 KRO模式（仅同时按下6次）
  • Fn + F5-禁用Windows键（游戏模式）
  • Fn + F6-上一曲
  • Fn + F7-开始/暂停
  • Fn + F8-下一个组成
  • Fn + F9-停止播放
  • Fn + F10-关闭声音
  • Fn + F11-减少音量
  • Fn + F12-增加音量
  • Fn + F1 ... F4-什么都没有（尽管从理论上讲，按两次套准之间的延迟应该减少，为什么例如在x4模式下按住某个键时，字符打印速度会更快）

这些都是功能，我个人已经足够了。而且没有宏和背光，生活就很舒适。

测试：打字和键盘体验


键盘上的打字过程为您提供了难以置信的体验。这是我的印象：我为自己设置了它，并且不会将其更改为任何其他键盘。



键盘上的键很容易按下，但弹性适中。 Cherry MX Reds是轻量级的开关，在大多数常规键盘上，它们的按压没有太大阻力，但是由于键帽的轮廓短，因此可以进行更紧的按压，并且除了提供所有按键外，还提供了易于按压的功能，并且还有大量的厚键帽-它们有助于一路“失败”键。是的，厚壁PBT塑料比任何ABS键帽都要重得多。



每个按键的敲击特性可以说是油性的（粘稠的）并且略微粗糙：由于短敲击，按下时会感觉到阻力，并且开关处的所有十字都非常精确地固定在开关外壳本身上，这就是为什么在按下按键时会产生这种粘滞摩擦的原因。有一种印象是，相对而言，手指更靠近开关的“心脏”，因此可以感觉到并控制每次点击。压制的粘性性质，但压制本身又轻又有弹性-这就是印刷过程的描述方式。从常规的Cherry轮廓（我曾在这样的机械师上坐了很长的时间）切换到Step Sculpture之后，我感觉到您正坐在常规的低键盘上打字-打字时手指实际上悬停了。因此，可以毫无问题地处理和输入数以千计的文本，而且即使经过很长时间的工作，您的手指也不会感到疲劳-根本无需劳累。顺便说一句，这就是我喜欢Cherry MX Red的原因之一-按下时无需费力，因为手指碰到它时会使其失效，从而使键失效。

完全按下键时，键盘基板会受到轻微甚至钝的打击-这是由于线性开关和安装在键盘内部的吸音基板共同造成的。当然，即使隔离了噪声，任何机械手都比薄膜键盘响亮，因此夜间的按键声音会干扰与您在同一房间放松的人。在这种情况下，您可以使用安装在键帽横档上的橡胶圈“ O形圈”-它们可减少键到止动件的行程，并在按下时提供静音，但是打印的性质会发生很大变化。

在游戏中，键盘也被证明是神奇的，尽管我的语言不敢将其称为游戏设备。对于射手来说，线性开关非常方便-可以轻松进行任何动作并控制每个动作。键帽的轮廓和单击的性质都决定了-对任何游戏情况的快速反应。在配置文件正常的键盘上，即使键盘在Cherry MX Red上，做同样的事情也比较困难，因此Leopold的返回反馈速度要快一些。只有Cherry MX Speed更快（它们在1.2毫米之前有键击，而Cherry MX Red只有2毫米），但仅在一对键盘中可以找到。



如果省略整个说明，则可以按以下方式描述打印过程：轻，柔，粗糙和弹性的压制，对基材的钝化和令人愉悦的打击。玩游戏和写文字就可以了。

解析航班键盘






要拆卸键盘，您需要使用常规的十字螺丝刀来拧开三颗螺钉（两根在可延伸支脚下面，另一根在电缆固定点），以及任何塑料工具（一定程度的硬），以将键盘盖（实际上是外壳）从板上分离。上面放有开关。





拆卸后的键盘看起来也很好。这不仅适用于外观，而且适用于内部。如果仔细观察，您会发现所有内部的执行质量并不比其所有外部部分差。我真的为此感到震惊。无助焊剂痕迹，完美的焊接，所有零件的精确装配。一切都抛光到最小的细节，看起来非常整洁。即使在键盘电路中。



在我看来，还有一种隔音基材，是用毡制成的。或来自任何相关材料。她应付任务，这是主要的事情。USB连接线位于附近，该连接线连接至键盘主板。







没有主体的上半部分，键盘看起来像当今非常流行的“骨架”格式的游戏键盘。从内到外，键盘都很漂亮。

结论




因此，全彩RGB背光。堆积的软件。一堆杂散和其他功能。宏键和积极的外观。

Leopold FC750R中的所有功能都不是。

该键盘是最纯正的经典键盘，是一种工作工具，只是每天使用的愉快设备。她没有任何严重的缺陷，但并不适合所有人。只是背光灯的粉丝会不喜欢它。但是对好事了解很多的人会喜欢的。

是的，在某些细节的性能方面有些挑剔（除了缺乏Mac支持外，我不能称其为缺陷），但绝对可以肯定，这款键盘的性能和手感将胜过许多自称为游戏键盘的键盘。坚固，简洁，美观，简约-这些是Leopold FC750R与该领域其他许多键盘的区别所在。

  • 电路板和所有组件的惊人工艺
  • 美学，美学和美学
