Aktualisierung des Ansatzes für kontextbezogene Werbung: Yandex.Direct Turbo Pages

Wenn Sie Waren und Dienstleistungen im Internet verkaufen und Ihre Hände die adaptive Version der Website noch nicht erreicht haben, ist der logische Ausweg die Yandex.Direct Turbo-Seite. Eine vollwertige und einfache Zielseite löst das Problem des Verlusts von mobilem Datenverkehr und bringt Ihre Werbung auf ein neues Niveau. Wir erklären Ihnen, wie Sie die Seite im Artikel einrichten.

Was ist eine Turboseite?


Yandex.Direct Turbo Page - Die Zielseite, auf die der Benutzer von der mobilen Ausgabe gelangt. Die Yandex-Ergänzung kann als vollständiger Ersatz für die adaptive Version der Website und als neue Werbeebene in Yandex.Direct angesehen werden.

Die Seite hilft dabei, Verkehrsverluste zu reduzieren und die Conversion zu erhöhen, wenn es nicht möglich ist, eine mobile oder adaptive Version der Website zu erstellen. Zum Beispiel, wenn es für Ihren Online-Shop zu teuer ist, sich an Kunden mit Mobilgeräten anzupassen. Der Zeitaufwand für die Erstellung einer Turboseite ist ebenfalls erheblich geringer als für die Entwicklung einer Version für ein Smartphone.

Eine Turboseite wird doppelt so schnell geladen wie eine normale Seite einer Site. Es wird auf dem Yandex-Server gespeichert, was auch das Leben des Webmasters vereinfacht.
Ein weiterer Vorteil des neuen Produkts ist die Möglichkeit zum Herunterladen mit einer instabilen Internetverbindung. Selbst bei einem schwachen Signal sieht der Kunde Ihr Angebot, wodurch Sie die Konkurrenz überholen können.

Sobald der Käufer auf der Seite ist, sieht er alle Informationen über das Angebot. Danach hat er die Möglichkeit, einen Antrag direkt auf der Turboseite auszufüllen. Wenn die Informationen ausgefüllt sind, werden die Bestätigung des Sendens der Bewerbung und die Dankbarkeit an den Kunden angezeigt.

Vorteile für den Websitebesitzer


Turbo-Seiten helfen Webmastern. Ihre Vorteile:

  • Es gibt keinen Verlust des mobilen Datenverkehrs, selbst wenn die Site nur eine Desktop-Version hat.
  • Informationen zu einem Produkt oder einer Dienstleistung stehen den Kunden rund um die Uhr zur Verfügung, auch wenn die Hauptwebsite „aus dem Leben geraten“ ist.
  • individuelle Einstellungen mit der Möglichkeit, die Parameter für jede Anzeige (oder Anzeigengruppe) zu ändern.

Ein Kunde, der von einem Computer oder Tablet aus nach Waren sucht, gelangt zur Vollversion der Website. Wenn sich der Benutzer von einem Mobiltelefon aus anmeldet, wird das Angebot auf der Turboseite angezeigt.

Kundennutzen


Die Erstellung von Turboseiten kann den Kunden nur gefallen:

  • Informationen über das gewünschte Produkt oder die gewünschte Dienstleistung sind trotz möglicher Probleme mit dem Internet immer öffentlich zugänglich.
  • Die Seite wird schnell geladen.
  • Sie müssen nicht auf der Website surfen, um eine Bestellung aufzugeben.

Die Hauptsache ist, die Nerven und die Zeit des Käufers zu sparen. Alles ist einfach, klar und zugänglich.

Richtige Einrichtung der Turboseiten


Durch das Erstellen einer Turboseite wird der beste Effekt aus der Werbung erzielt. Der Aktionsalgorithmus ist unten.
Suchen Sie zunächst die Schaltfläche "Turbo-Seiten" unter den vorhandenen Kampagnen.


Nach dem Klicken wird die Bibliothek geöffnet, in der alle erstellten und zum Bearbeiten verfügbaren Turboseiten sichtbar sind.


Klicken Sie auf "Erstellen" und bestimmen Sie den Namen der Seite.

Wenn Sie einige zum Vergleich erstellen, nummerieren Sie sie im Titel.

Wählen Sie eine Vorlage aus. Wenn Sie grundlegende Informationen zum Produkt benötigen, bevorzugen Sie den Kurzfilm. Information impliziert eine detaillierte Beschreibung des Vorschlags. Auf einer leeren Seite können Sie selbst eine Vorlage erstellen.


Als nächstes kommt der Navigationsblock . Es enthält eine Liste von Elementen, die bearbeitet werden können. Das zu ändernde Feld wird im Abschnitt angezeigt.



Abschnittseditorfenster . Hier müssen Sie die Textfelder ausfüllen und eine Farbe auswählen. Wenn ein Titel oder eine Beschreibung nicht benötigt wird, löschen Sie einfach diese Elemente.



Medieninhalt Der Begriff „Medien“ bedeutet in diesem Fall ein einzelnes Bild, eine Illustrationsgalerie oder einen Schieberegler. Wenn Sie dem Bild eine Beschreibung hinzufügen müssen, seien Sie kurz. Ein gebräuchlicher Name reicht für die Galerie aus, einzelne werden für einzelne Illustrationen erstellt.



Für das Formular sind eine Überschrift, eine Unterüberschrift und ein Text vorgeschrieben, die dem Käufer erklären, was von ihm verlangt wird und was das Ergebnis sein wird.



Pflichtfelder - Telefonnummer und Benutzername . Der Rest kann nach Wunsch hinzugefügt werden. Denken Sie daran, dass eine gute Form kurz sein sollte.



Vergessen Sie nicht, eine Richtlinie für die Verarbeitung personenbezogener Daten vorzuschreiben .



Sie können einen Link zu einem erläuternden Dokument hinterlassen.



Dann öffnet sich die Dankesseite, von der aus der Käufer zur Hauptseite oder zurück zur Zielseite gelangt.



Am Ende veröffentlichen Sie die Seite oder speichern sie als Entwurf.
Dies ist die vollständige Ansicht der Turboseite.



Die ID jeder erstellten Seite ist eindeutig. Fügen Sie dem Seitenlink UTM-Tags hinzu, um weitere Informationen in der Metrik zu erhalten.


Es ist nicht erforderlich, speziell einen Zähler für eine Turboseite zu erstellen. Dieser wird automatisch angezeigt.



Die Ziele für den Conversion-Bericht in Yandex.Metrica werden ebenfalls automatisch erstellt.



Wir verfolgen:

  • Einreichung des Antragsformulars;
  • Klicken Sie auf die Schaltfläche auf der Hauptseite.
  • Übergang zur Hauptressource über den Link mit Dankesseite;
  • Eröffnungsform;
  • Senden Sie das Formular und gehen Sie zur Dankesseite.

Das Einrichten einer Turboseite nimmt wenig Zeit in Anspruch und liefert ein gutes Ergebnis. Ihre Kunden werden die mobile Landung für hohe Download-Geschwindigkeit lieben.

Die Turbo-Seite ist eine effektive Lösung für die Präsentation Ihres Vorschlags ohne eine mobile oder adaptive Version der Website.
