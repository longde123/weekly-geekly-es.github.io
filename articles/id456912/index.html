Cara membuat situs di 2019

Web berubah dengan cepat, tetap ikuti Anda!



Saya menulis artikel serupa pada tahun 2018, tetapi web berubah begitu cepat sehingga pedoman itu sudah usang.


Mari kita lihat prinsip-prinsip terpenting desain web pada tahun 2019 selangkah demi selangkah.

Ini adalah prinsip penting, dan ada alasan untuk itu. Kelompok fokus, yang telah menghabiskan banyak uang untuk kami, telah menunjukkan bahwa hal pertama yang diinginkan pengguna dalam menjangkau situs Anda adalah memasang aplikasi seluler.

Kembangkan strategi untuk Jangan Lacak


Sebagian besar browser modern (sehingga mereka!) Mendukung header HTTP DNT, yang merupakan singkatan dari Do Not Track. Header ini memberi sinyal bahwa pengguna tidak ingin dilacak dan, sayangnya, header ini diaktifkan secara default di beberapa browser seperti Brave ( browser open source dengan pemblokiran iklan dari pencipta JavaScript, Brendan Eich ).

Kiat: Jangan tanya apakah pengguna ingin menonaktifkan DNT dalam manual untuk menonaktifkan DNT, jika tidak, Anda masih akan setuju.


Kembangkan strategi untuk Cook


Tidak perlu bertanya tentang cookie jika digunakan di situs untuk alasan yang memadai, tetapi jauh lebih baik untuk meminta izin untuk semua cookie dalam jumlah besar, maka kita dapat melacak semua yang kita inginkan tanpa takut konsekuensi!



Kiat: Jangan memuat halaman tanpa cookie, meskipun itu tidak diperlukan untuk pekerjaan, karena dengan begitu Anda akan dapat melacak informasi pribadi yang jauh lebih sedikit yang kemudian dapat Anda jual.
