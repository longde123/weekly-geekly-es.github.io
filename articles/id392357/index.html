Bola pemadam api dari Thailand: Elide Fire Extinguishing Ball



Hanya sedikit dari kita yang pernah menggunakan alat pemadam api. Persentase orang yang sedikit lebih besar secara teoritis membayangkan cara menggunakannya. Jika terjadi kebakaran, sebagian besar akan membaca petunjuk penggunaannya saat api menyala. Area pengapian yang luas membutuhkan alat pemadam api yang besar, yang, karena kebutuhan untuk menyimpan campuran di bawah tekanan, biasanya sangat sulit.

Penemu Phanawatnan Kaimart selamat dari kebakaran hotel yang mengerikan pada Juli 1997Royal Jomtien Resort Hotel, yang menewaskan lebih dari 100 orang. Hotel ini terletak di Thailand, di daerah Pattaya yang disebut Jomtien. Setelah pengalaman yang tragis seperti itu, Kaymart memutuskan untuk membuat perangkat yang lebih sederhana dan efektif dalam memadamkan api daripada pemadam api biasa, yang kakek buyutnya muncul kembali pada akhir abad ke-19. Jadi penemu datang ke penciptaan perangkatnya yang disebut "Elide Fire Extinguishing Ball" ( bola pemadam api Elide ).



Saat dipanaskan, bola seukuran bola sepak meledak dan menaburkan bubuk di dalamnya hingga jarak 1,3 meter, sedangkan bola hanya memiliki berat 1,3 kilogram. Serbuk ini tidak berbahaya bagi manusia dan lingkungan, dan cocok untuk memadamkan api yang terkait dengan listrik. Umur simpan bola adalah 5 tahun. Bola Elide juga menerima sertifikat Rusia pada 2008.
www.elidefire.com/russia.htm



Startup Thailand Elide memastikan bahwa bola aman dan tidak akan meledak sendiri saat terjatuh - diperlukan suhu tinggi untuk mengaktifkannya. Selain itu, ini dapat berfungsi sebagai sistem pemadam api pasif jika dipasang pada dudukan di area yang berisiko kebakaran tinggi - misalnya, di dapur.

Jika api yang sangat kecil lebih mudah dipadamkan dengan alat improvisasi atau pemadam api, maka ketika api tumbuh dengan alat pemadam api, menjadi berbahaya untuk mendekatinya. Dalam kasus seperti itu, bola Elide bisa sangat berguna.
