Der Elektromotor des billigsten Tesla-Modells ist leistungsstärker als der BMW M3 Motor

Das Tesla Model 3 wird mit wiederaufladbaren Batterien letzter Änderung verwendet werden „Gigafabriki Tesla“



Unternehmen Tesla wird installiert werden , in ihrem neuen elektrischen Tesla Model 3 Batterien, die jetzt gemacht werden, „Gigafabrike“ von Nevada. Die neuen Aggregate werden, wie das Unternehmen verspricht, leistungsstärker und effizienter. Der Konverter wurde von Grund auf neu entwickelt, frühere Modelle, die mit demselben Tesla Model S arbeiteten, werden nicht verwendet. Hier ist alles neu, auch Halbleitersystemelemente. Den Ingenieuren des Unternehmens ist es gelungen, die Anzahl der einzigartigen Wechselrichterelemente um etwa 25% zu reduzieren, wodurch die Konstruktionskosten gesenkt werden können.

Darüber hinaus erhielt Modell 3 einen 435-PS-Elektromotor. Dies wurde vom technischen Direktor von Tesla angekündigt. Dies ist sogar mehr als der BMW M3, der über einen Drei-Liter-Sechszylinder-Doppelturbomotor (maximal - 431 PS) verfügt. Dank eines leistungsstarken Motors kann die langsamste Modifikation des Modells in nur 6 Sekunden auf 96 Stundenkilometer beschleunigen. Ein älteres Modell mit erweitertem Ludicrous-Modus benötigt nur 4 Sekunden, um auf diese Geschwindigkeit zu beschleunigen.


Elektronische Komponenten des Wechselrichters (Feldeffekttransistoren mit isoliertem Gate) Die

Ingenieure des Unternehmens arbeiten seit mehreren Monaten an der Entwicklung eines neuen Wechselrichters Modell 3 mit einer Leistung von 320 kW. Das Wechselrichterkonzept verwendet TO-247-Bipolartransistoren mit einem isolierten Gate. Diese elektronischen Komponenten wurden beim Bau des Wechselrichters für das Tesla Model X und das Tesla Model S verwendet. Die Produktion von Wechselrichtern hat bereits begonnen, Produktionslinien für andere Komponenten wurden ebenfalls in Betrieb genommen, da das Unternehmen beabsichtigt, bis 2018 etwa 500.000 Elektrofahrzeuge zu liefern.



Ohne Aufladen kann das neue Modell von 340 auf 400 Kilometer fahren, was sehr gut ist. Zunächst wird eine Version mit einer Reichweite von 340 Kilometern auf den Markt gebracht, wonach ein Modell mit einer Batteriekapazität von 80 kWh erscheint. Mit dieser Batterie kann ein Elektroauto 480 Kilometer weit fahren. Zusätzlich bekommt die Neuheit einen Autopiloten. Und obwohl er das Elektroauto nicht in ein Robomobil verwandelt, wird die Hilfe für den Autofahrer sehr ernst sein.

Jetzt testet das Unternehmen bereits sein neues Elektroauto. Beispielsweise wurde kürzlich ein solches Modell in einem der Service-Center des Unternehmens fotografiert. Im Aussehen unterscheidet es sich nicht von einem Demo-Beispiel.



Modell 3 wird erst Ende 2017 an Kunden ausgeliefert. Es gab mehrmals mehr Vorbestellungen für ein Elektroauto als geplant - derzeit mehr als 375 Tausend. Es ist unklar, ob Tesla Motors eine solche Belastung ohne Terminüberschreitung bewältigen kann. Es ist möglich, dass es zu Pannen kommt. Die Probleme mit Modell X waren noch im ersten Quartal - anstelle von 4.500 Elektrofahrzeugen konnte das Unternehmen 2.400 liefern. Dennoch verspricht Elon Musk, die Produktionskapazität schrittweise zu erhöhen, damit Kunden aller Elektrofahrzeugmodelle ihre Fahrzeuge pünktlich erhalten.
