Télégramme en tant que service

Si vous avez déjà rencontré des restrictions inhérentes à l'API Telegram Bot et admiré les fonctionnalités fournies par l'API Telegram Client, mais que vous pensiez qu'il était trop difficile d'installer TDLib, alors Taas est fait pour vous!

https://www.taas.ru/

Taas vous permet de créer un client Telegram virtuel dans le cloud, avec accès API. Vous pouvez utiliser toutes les méthodes fournies par l' API Telegram , sans rien installer.

Pourquoi est-ce mieux qu'un bot?

  1. Vous pouvez envoyer des messages même aux personnes qui n'ont pas commencé la première communication avec vous.
  2. La possibilité d'obtenir l'historique complet des discussions en utilisant la méthode getChatHistory.
  3. Et bien plus. Taas crée pour vous un véritable client Telegram contrôlé via l'API.


Comment commencer à utiliser l'API Telegram en utilisant Taas?

1. Inscrivez-vous sur le site Web https://www.taas.ru/

2. Créez un client dans le tableau de bord (bouton Nouveau client)



3. Le client est en cours d'exécution - prenez la clé API et faites des demandes. La documentation
