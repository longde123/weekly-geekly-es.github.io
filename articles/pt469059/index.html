O que há de novo no ML.NET e no Model Builder

É com satisfação que anunciamos atualizações do Model Builder e melhorias no ML.NET . Você pode aprender mais em "O que há de novo no ML.NET ?." sessão no .NET Conf .

O ML.NET é uma estrutura de aprendizado de máquina de código aberto e multiplataforma (Windows, Linux, macOS) para desenvolvedores .NET.

O ML.NET oferece o Model Builder (uma ferramenta simples de interface do usuário) e a CLI para facilitar a criação de modelos ML personalizados usando o AutoML.

Usando o ML.NET , os desenvolvedores podem aproveitar suas ferramentas e habilidades existentes para desenvolver e inserir IA personalizada em seus aplicativos, criando modelos personalizados de aprendizado de máquina para cenários comuns, como Análise de Sentimentos, Recomendação, Classificação de Imagem e muito mais! ..



Este artigo em nosso blog .

A seguir, estão os principais destaques:

Atualizações do Model Builder


Esta versão do Model Builder adiciona suporte para um novo cenário e soluciona muitos problemas relatados pelos clientes.



Engenharia de recursos: nas versões anteriores do Model Builder, depois de selecionar seu conjunto de dados, de um arquivo ou do SQL Server, você só tinha a opção de escolher a coluna a prever (o Rótulo). Quaisquer outras colunas no conjunto de dados foram usadas automaticamente para fazer a previsão (Recursos). Quaisquer colunas que você não desejasse incluir, era necessário manipular seu conjunto de dados fora do Model Builder e, em seguida, fazer upload do conjunto de dados modificado.



Consumo de modelo facilitado!: Nas versões anteriores do Model Builder, havia várias etapas que você precisava seguir após a geração do código e do modelo do Model Builder para consumir o modelo treinado no seu aplicativo, incluindo a adição de uma referência ao projeto de biblioteca gerado, definindo a propriedade Copy to Output do modelo como "Copy If Newer" e adicionando o pacote Microsoft.ML NuGet ao seu aplicativo.

Tudo isso foi simplificado e automatizado, então agora tudo o que você precisa fazer é copiar e colar o código das Próximas etapas no Model Builder e, em seguida, você pode executar seu aplicativo e começar a fazer previsões!

Abordar o feedback do cliente: Esta versão também aborda muitos problemas relatados pelo cliente sobre erros de instalação, feedback de usabilidade e melhorias de estabilidade e muito mais. Saiba mais aqui .

Atualizações do ML.NET


Este é um breve resumo dos recursos e aprimoramentos adicionados ao ML.NET nos últimos meses.

  • Suporte para .NET Core 3
  • Suporte para novos cenários, como previsão de vendas, detecção de anomalias
  • Visualização: carregador de banco de dados nativo que permite o treinamento diretamente em bancos de dados relacionais
  • Visualização: crie modelos personalizados de aprendizado profundo para classificação de imagens usando o TensorFlow.



Atualizações da documentação


Estamos trabalhando duro para adicionar mais documentação em tutoriais, guias de instruções e muito mais para o Model Builder, CLI e ML.NET Framework. Também simplificamos o índice do ML.NET Docs para que você possa descobrir facilmente o conteúdo.
