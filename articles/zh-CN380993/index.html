帮助旅行者和其他人：Microsoft便携式电池产品线的公告

正值旺季时，我们宣布了一系列智能手机的最好的朋友和助手-带有功能强大的电池和设计盒的Microsoft便携式电池。


总体而言，新系列的Microsoft便携式双充电器（DC）包括三个紧凑型电源DC-32，DC-33和DC-34。从外观上讲，它们采用相同的样式和外形制造，多少让人联想起苏联时代的3R12电池。主要区别在于容量。

最便携的Microsoft DC-32电池（容量为5,200 mAh）开启了这条新生产线。这是一种城市选择，由于其体积小巧（100 x 50 x 25毫米）并且重量不大（141克），因此不会成为衣服口袋或小包里烦人的障碍。



对于长途旅行或那些Anatoly Wasserman那些不限于日常生活中的一两个小工具的人，适合使用容量更大的Microsoft DC-33（9,000 mAh）或Microsoft DC-34（12,000 mAh）的电池。也许，更大的电池容量也意味着更大的尺寸，这不足为奇，但是两个旧型号仍然保持便携性，只是长度和重量有所增加（100 x 68.5 x 25 mm，重量：215 g和100x87x25 mm，重量：分别为275克）。这些电池可能无法放入牛仔裤的任何口袋，但它们不会使您出行...
... 像那样。


所有此类产品的传统产品都配备了LED指示器，用于确定充电水平，更有趣的是，还配备了两个USB端口，用于同时为多个小工具充电。



完全放电的DC-32将在160分钟内充电。补充旧DC-33 / -34型号的能量储备将分别需要280和380分钟。 Microsoft便携式双充电器产品线的设备可长时间保存电荷。即使经过几个月的不活动，能量储备也将高达80％。

同时，使用新电池为智能手机或平板电脑充电的强度与使用高速充电器（例如AC-60）可比。



设备系列Microsoft便携式双充电器将在几周内开始在全球市场上销售。俄罗斯价格尚未公布，但宣布的容量为5,200 mAh的Microsoft DC-32的估计成本为35美元（不含税和费用），按当前汇率约为1,900₽。容量为9000 mAh的DC-33和容量为12,000 mAh的DC-34的价格分别为45美元（约2 450₽）和55美元（约3000₽）。
