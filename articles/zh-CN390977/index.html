2016年世界移动大会的索尼新闻：Xperia X，Xperia X Performance和Xperia XA

在巴塞罗那举行的MWC 2016展览上，索尼立即发布了新系列的三种型号-Xperia X，Xperia X Performance和Xperia XA。隐藏在带有圆角的机壳中的三个初学者由Android 6.0 Marshmallow操作系统控制，并提供相同的5英寸对角线显示。





最实惠的将是Xperia XA。这款智能手机配备了5英寸高清显示屏（720x1280像素），MediaTek Helio P10 MT6755芯片组，2 GB的RAM和16 GB的内部存储器以及一对相机-主13 MP和前置8 MP。电池容量-2300 mAh，尺寸-143.6 x 66.8 x 7.9毫米，重量-137.4克。



至于Xperia X Performance和Xperia X型号，它们的相似之处多于差异。在这两种情况下，我们都在谈论全高清屏幕分辨率，3 GB的RAM和32 GB的内部存储器，一个具有预测性混合自动对焦功能的23兆像素主摄像头，该摄像头可以预测对象的轨迹。 Xperia X Performance使用比Xperia X（Qualcomm Snapdragon 650）更高级的芯片（Qualcomm Snapdragon 820）。第一个型号的电池容量为2700 mAh，第二个型号为2620 mAh。



型号的尺寸几乎相同：Xperia X为142.7 x 69.4 x 7.9毫米/ 153克，而Xperia X Performance为144.8 x 71.1 x 7.6毫米/ 165克。这些设备的侧面电源按钮内置了指纹传感器。

这三款新产品均支持可移动microSD存储卡（最大200 GB）和专有的节能技术，使它们最多可以工作两天而无需充电。智能手机将提供四种颜色：白色，石墨黑，石灰金和玫瑰金。
