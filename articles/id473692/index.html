Ennoble Phicomm K3C Wi-Fi Router

Perusahaan China, Phicomm dalam jajaran router Wi-Fi memiliki perangkat yang disebut K3C AC1900 Smart WLAN Router.

Perangkat ini menggunakan banyak Intel AnyWAN SoC GRX350 dan Intel Home Wi-Fi Chipset WAV500 ( omong - omong, perangkat keras yang sama digunakan di ASUS Blue Cave: prosesor Intel PXB4583EL yang sama dan chip Intel Wi-Fi PSB83514M / PSB83524M bukan PSB83513M / PSB83523M ).

Ada beberapa versi router ini:

  • B1 , B1G , B2 - untuk Cina;
  • A1 , C1 , S1 ( VIE1 ) - untuk negara lain ( saya dapat - C1 dengan firmware v.34.1.7.30 ).

Apa yang menarik minat saya tentang router standar IEEE 802.11ac ini?

Sehingga ada: 4 port gigabit ( 1 WAN dan 3 LAN ), band 5 GHz, dukungan untuk MU-MIMO 3x3: 3 dan USB 3.0. Baik dan tidak hanya.

1. Sedikit latar belakang


Bagian opsional
Router saya sebelumnya adalah TP-Link TL-WR941ND dengan versi perangkat keras 3.6 ( 4MB Flash dan 32MB RAM ). Firmware standar secara berkala digantung tanpa alasan terlepas dari versi ( diperbarui beberapa kali, pembaruan terakhir untuk perangkat keras saya dirilis pada akhir 2012 ).

Kecewa pada firmware asli, saya menginstal Gargoyle ( emnip, versi 1.8; firmware didasarkan pada OpenWRT, jika ada yang tidak tahu ) dan akhirnya router mulai bekerja sebagaimana mestinya.

WR941 pada saat pembelian memiliki perangkat keras yang baik untuk kebutuhan saya ( sekitar 10 tahun yang lalu ), tetapi sekarang saya sudah kekurangan kinerjanya. Port-portnya semuanya 100 Mbit / s, kecepatan Wi-Fi maksimum adalah 300 Mbit / s. Mungkin untuk Internet ini masih normal, tetapi mentransfer file di jaringan lokal antar perangkat agak lambat. Selain itu, memori Flash bawaan tidak cukup bahkan untuk Russifikasi firmware ( bahkan mengubah file melalui WinSCP, saya mencoba entah bagaimana ), tidak menginstal plug-in yang lebih luas ( tentu saja Anda dapat memperluas memori, menginstal firmware untuk memperbesar ukuran memori, tetapi saya tidak cukup langsung tangan untuk menyolder chip memori ).

Tapi, mungkin, bahkan semua hal di atas tidak akan segera memaksa saya untuk mengganti router. Saya baru saja membeli sendiri pada awal September tahun ini Xiaomi Redmi Note 5 untuk menggantikan Redmi Note 4 yang hilang sebelum waktunya ( setelah 2 tahun layanan teladan ) dan ternyata RN5 dan WR941 tidak kompatibel satu sama lain - RN5 tidak ingin menyambung kembali setelah memutuskan koneksi dari jaringan nirkabel yang dibuat Menggunakan WR941 (dan ini bukan masalah yang terisolasi, karena saya menemukan sedikit kemudian membaca topik di w3bsit3-dns.com ).


Secara umum, ada kebutuhan untuk mengubah router. Mengapa tepatnya subjek? Saya tertarik dengan isinya ( saya membacanya di SmallNetBuilder sekitar setahun yang lalu ) dan kemampuannya ( meskipun tidak mungkin bahkan setengah dari mereka akan digunakan dalam waktu dekat ). Tetapi bahkan ini tidak menjadi penentu dalam memilih persis Phicomm K3C ( saya juga melihat dengan seksama pada Xiaomi Mi WiFi Router 3G ), dan harga yang terjangkau ( dibeli seharga $ 32 pada tingkat ) dengan isian yang baik dan kemampuan untuk mengubah firmware saham ke OpenWRT penuh. Perute ini hadir dengan versi OpenWRT buatan pabrik ( saya membaca bahwa mata-mata ditambahkan di dalamnya, tetapi tidak menemukan perincian ).

Modifikasi OpenWRT untuk bekerja pada Phicomm K3C ( secara resmi OpenWRT tidak mendukung chipset Intel WAV500 ) dibuat oleh orang Cina dengan julukan Paldier ( GitHub- nya dan halaman dengan file firmware untuk router ini, tema router pada forum OpenWRT ). Ia juga membuat port firmware Asus Merlin untuk K3C ( karena menginstalnya memerlukan penggantian RAM dari 256MB menjadi 512MB, kami tidak akan mempertimbangkannya ).

3. OpenWRT firmware


  1. Kami menghubungkan router ke komputer / laptop, melalui port LAN ( salah satu dari ketiganya ) dan Internet melalui WAN ( karena Anda perlu mengunduh firmware, sedikit lebih dari 30MB ).
  2. Kami mempelajari alamat router di jaringan lokal ( kami akan membutuhkannya nanti, biasanya 192.168.2.1 ).
  3. Kami meluncurkan utilitas RoutAckPro yang sebelumnya diunduh ( beratnya 600kB dan banyak teks berbahasa Mandarin di dalamnya; saya tidak tahu di mana harus mengunggahnya, tetapi itu dapat diunduh dari forum w3bit3-dns.com setelah mendaftar ). Jika alamat berbeda dari yang di atas, maka masukkan dalam bentuk IP. Klik tombol Telnet di jendela. Jika semuanya dilakukan dengan benar, teks 打开 Telnet 完成 muncul di jendela. Sekarang utilitas dapat ditutup, mis. kami menyiapkan router untuk mengubah firmware melalui Telnet.


    Jendela RoutAckPro
  4. Menggunakan program Putty ( Smartty atau yang serupa lainnya ), kami terhubung melalui Telnet ke router ( IP sama dengan untuk RoutAckPro, port 23 ).


    Jendela Putty dengan pengaturan koneksi.
  5. Di konsol Putty, masukkan untuk pergi ke direktori tmp: 

    cd /tmp

  6. Kami menentukan jenis firmware apa yang perlu kami unduh ( versi besi dicetak pada stiker yang ditempelkan ke bagian bawah router, dalam kasus saya ini adalah " H / W C1 ", yaitu, saya perlu firmware untuk C1 ).
  7. Kami memilih di situs web Paldier versi file fullimage.img yang kami butuhkan . Bagi saya itu 
      http://k3c.paldier.com/openwrt/C1/fullimage.img

    Oleh karena itu, kami menulis yang berikut di konsol Putty: 

     wget http://k3c.paldier.com/openwrt/C1/fullimage.img

  8. Lalu masukkan perintah 

     /usr/sbin/upgrade /tmp/fullimage.img fullimage 0 1

    dan tunggu pesan tentang firmware yang berhasil.
  9. Setelah itu kami perkenalkan 

     rm -rf /overlay/* sync && sleep 10 && reboot

    dan tunggu hingga router memulai ulang ( beberapa menit ). Setelah itu, Anda dapat terhubung ke antarmuka webnya ( alamat 192.168.2.1 , kata sandi admin ).
  10. Setelah boot pertama, disarankan untuk melakukan reset (dengan tombol tersembunyi di router, sedikit di sebelah kanan soket daya, atau melalui antarmuka web ).


    Sekarang router akan memiliki antarmuka ini

Instruksi kilat dibuat oleh pengguna forum w3bsit3-dns.com , WayOutt , yang karenanya banyak terima kasih kepadanya.

Jika Anda tidak ingin segera menghubungkan K3C ke Internet dan Anda memiliki USB flash drive atau pembaca kartu USB dengan kartu flash.
Kita lewati langkah 5, dan pada langkah 7, alih-alih mengunduh file firmware ke router dengan perintah wget, unduh file itu ke PC ( jika Anda masih membutuhkannya nanti ) dan salin file ke drive flash USB dan sambungkan ke port USB router.
Dalam paragraf 8, kita memasukkan perintah berikut: 
 /usr/sbin/upgrade /mnt/usb/.run/mountd/sda1/fullimage.img fullimage 0 1

Item yang tersisa tetap tidak berubah.

* Setelah firmware OpenWRT, jalur untuk memasang flash drive akan berubah. Sudah menjadi seperti ini bagi saya: 
 /usr/sbin/upgrade /tmp/usb/.run/mountd/sda1/fullimage.img fullimage 0 1

4. Bersihkan antarmuka


Tetapi firmware dari Paldier, sayangnya, tidak mengandung terjemahan Rusia, tetapi ia memiliki daftar situs yang harus diblokir di Cina ( oleh karena itu, kami tidak dapat pergi ke github yang sama dengan pengaturan default, tetapi ini diselesaikan dengan menghapus satu tanda centang di pengaturan V2Ray ).

Oleh karena itu, kami akan menetapkan lokalisasi Rusia untuk LuCI.

Ini dilakukan cukup sederhana:

  1. Buka System ==> Perangkat Lunak ==> tab Tindakan .
  2. Di bidang Unduh dan instal paket , masukkan 

     http://downloads.openwrt.org/releases/18.06.0/packages/mips_24kc/luci/luci-i18n-base-ru_git-19.297.26179-fbefeed-1_all.ipk
    dan klik tombol Ok di sebelah kanan.

    Nah, atau unduh paket-paket ini, simpan ke flash drive USB, dan kemudian tancapkan ke port USB router dan instal melalui Putty dengan perintah
