Kami mengundang Anda ke pertemuan departemen pengembangan game GeekUniversity


Universitas online kami, GeekUniversity, akan menyelenggarakan pertemuan Departemen Pengembangan Game. Peserta akan dapat berdiskusi dengan praktisi mengapa dan bagaimana menerapkan arsitektur ECS (Entity Component System) dan apakah itu dapat ditiadakan. Rapat akan bermanfaat bagi pemula dan pengembang praktik di Unity. Acara ini akan diadakan di kantor Moskow Grup Mail.ru pada 25 Mei, mulai pukul 14:00. Program ini sedang dalam tahap pemotongan.


  • Valentin Simonov, Insinyur Lapangan di Unity Technologies, akan berbicara tentang Data-Oriented Tech Stack (DOTS) dan bagian-bagiannya: ECS, Sistem Pekerjaan C #, Burst Compiler, Unity Physics - teknologi yang memungkinkan game di Unity menggunakan semua kemampuan desktop modern dan perangkat keras seluler .
  • Sergey Klimenko, seorang programmer di studio Moonlight Mouse, akan berbicara tentang bagaimana menerapkan mekanisme gameplay di DOTS, di mana untuk memulai transisi ke DOD, bagaimana mengatur data dan sistem dalam ECS dan bagaimana mengoptimalkan waktu eksekusi sistem menggunakan C # Job System.
  • Vasily Markevich, seorang siswa di departemen pengembangan game GeekUniversity, Project Manager, penulis @ecscomrade , akan berbagi pengalaman pengembangannya di ECS dan menjawab dua pertanyaan utama pengembang pemula: â€œMengapa saya TIDAK membutuhkan ECS? Mengapa saya MEMBUTUHKAN ECS? "
