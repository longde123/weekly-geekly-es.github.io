Visual Studio Code扩展，可帮助您使用Git

Visual Studio Code默认情况下具有git，但是内置函数并不总是足够方便和快捷地工作。 git的有效工作还需要过程可视化。

以下是使使用git变得轻松愉快的扩展：

吉特伦斯


GitLens扩展了内置Visual Studio Code的git功能。 该扩展程序具有大量功能和灵活的设置。 只需单击一下，您就可以逐行获取有关作者，注释和更多信息的信息。



git历史


Git历史记录可帮助您查看更改历史记录。 它具有相当用户友好的界面，并且速度非常快。



git图


该扩展程序有助于浏览存储库分支，还可以进行历史记录搜索。 对于经常接受请求请求并进行代码审查的开发人员，此扩展将是必不可少的。



Git指标


Visual Studio Code底部窗格中的git活动的简单指示。
