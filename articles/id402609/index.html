WhatsApp, Telegram dan kamera depan: cara kerjanya, bagaimana cara kerjanya, dan sedikit lagi tentang korespondensi dengan dukungan



Semuanya dimulai dengan fakta bahwa sekitar dua bulan lalu saya menemukan perilaku aneh kamera depan di WhatsApp. Saya hanya menulis sekarang, bukan karena saya telah mempersiapkan posting begitu lama, tetapi karena selama ini saya berkorespondensi dengan dukungan teknis dari kurir. Yah, ke tumpukan - dia sendiri mulai ragu: saya menemukan bug atau fitur.

Saat mengirim selfie menggunakan WhatsApp, saya menemukan anting-anting saya tidak di telinga kiri saya, tetapi di telinga kanan saya. Penerima foto mengkonfirmasi fakta ini. Tetapi pada saat yang sama, di folder / WhatsApp Images foto itu normal, tidak dicerminkan. Kemudian saya mulai bereksperimen dengan Telegram. Dia berperilaku dengan cara yang sama (cermin dan mengirim foto dari kamera depan), tetapi setidaknya dengan jujur â€‹â€‹menyimpannya di / Telegram dalam bentuk yang sama saat dia mengirimkannya. Untuk pemahaman yang lebih baik, saya membawa hasil penelitian ke daftar, dan sedikit di bawah ini saya akan memberikan tangkapan layar.

Jadi, mulai hari ini, ketika mengirim foto dari kamera depan ...
WhatsApp: mencerminkan gambar, dalam formulir ini mengirimkannya ke penerima dan menampilkannya di dialog. Foto itu sendiri disimpan di folder program tidak berubah .

Telegram: mirror gambar, dalam bentuk ini mengirimkannya ke penerima, menampilkannya dalam dialog dan menyimpannya di folder program.

Tangkapan layar








Korespondensi yang puas secara terpisah dengan dukungan untuk WhatsApp. Saya tidak akan mengutip sepenuhnya, saya akan sampaikan dengan kata-kata saya sendiri.

Pada awalnya, saya ditolak tiket karena kehadiran perangkat Root (dalam hal dukungan "Sepertinya Anda menggunakan ponsel yang di-flash. Kami minta maaf, tapi kami tidak mendukung perangkat yang di-flash ..."). Oke, saya menemukan smartphone dengan firmware bawaan, mengulangi semua tindakan di atasnya dan mengirim pesan untuk mendukung.

Setelah itu, saya menerima surat tiga kali dengan permintaan untuk mengirim informasi debug (semua tiga kali saya melakukannya dengan rajin), dan pada akhirnya menjadi mempesona: "Kami memahami bahwa kesalahpahaman dapat muncul, tetapi ini sebenarnya adalah fungsi dari aplikasi dan fitur-fiturnya."

Sekarang saya sendiri tidak yakin apa itu. Karena itu, saya meminta bantuan dari penonton (lihat polling) ...

PS Pada dasarnya, saya bisa menebak dari mana kaki-kaki itu berasal. Ponsel cerdas dan kamera depan sering digunakan sebagai cermin saku (terutama wanita). Dalam hal ini, gambar benar-benar perlu dicerminkan. Tapi mengapa utusan mengirim gambar dari kamera depan dalam bentuk cermin - Saya tidak mengerti ini. Memang, dalam hal ini, penerima tidak akan melihat gambar asli yang ingin saya tunjukkan kepadanya, tetapi versi cerminnya ...

UPD: Saya sepertinya mengerti mengapa mereka menulis dari dukungan: "ini bukan bug, tetapi fitur". Sehingga saya tidak perlu membayar hadiah untuk kesalahan yang ditemukan :) :) :).
