Astra Linux "Eagle" Common Edition: apakah ada kehidupan setelah Windows

Kami menerima ulasan terperinci dari salah satu pengguna OS kami, yang ingin kami bagikan dengan Anda.

Astra Linux adalah turunan Debian yang dibuat sebagai bagian dari inisiatif Rusia untuk beralih ke perangkat lunak open source. Ada beberapa versi Astra Linux, salah satunya ditujukan untuk penggunaan umum, sehari-hari - Astra Linux "Eagle" Common Edition. OS Rusia untuk semua orang menurut definisi menarik, dan saya ingin berbicara tentang Eagle dari sudut pandang seseorang yang menggunakan tiga sistem operasi setiap hari (Windows 10, Mac OS High Sierra dan Fedora) dan telah setia pada Ubuntu selama 13 tahun terakhir. Berdasarkan pengalaman ini, saya akan mempertimbangkan sistem dalam hal instalasi, antarmuka, perangkat lunak, fitur dasar untuk pengembang dan kenyamanan dari berbagai sudut. Bagaimana Astra Linux dibandingkan dengan sistem yang lebih umum? Dan bisakah dia mengganti Windows di rumah?



Kami menempatkan Astra Linux


Pemasang Astra Linux memiliki banyak kesamaan dengan pemasang Debian. Mungkin yang pertama bahkan lebih sederhana, karena sebagian besar parameter diperbaiki secara default. Semuanya dimulai dengan perjanjian lisensi umum dengan latar belakang bangunan yang tidak terlalu tinggi. Mungkin bahkan di Orel.



Poin penting dalam instalasi adalah pilihan perangkat lunak, yang datang secara default dengan sistem. Opsi yang tersedia mencakup kebutuhan kantor dan pekerjaan standar (untuk yang bukan pengembang).



Selain itu, jendela terakhir adalah serangkaian pengaturan tambahan: memblokir penerjemah, konsol, pelacakan, pengaturan bit eksekusi, dll. Jika kata-kata ini tidak memberi tahu Anda apa pun, lebih baik tidak mencentang kotak. Selain itu, semua ini, jika perlu, dapat dikonfigurasi nanti.



Sistem dipasang di dalam lingkungan virtual dengan sumber daya sederhana (sistem yang relatif modern). Tidak ada keluhan tentang kecepatan dan kinerja. Konfigurasi yang diuji dijelaskan di bawah ini.



Prosedur instalasi adalah biasa: kita memasang iso-image , menginstalnya melalui proses instalasi sistem standar dan membakar bootloader GRUB.



Sistem saat memuat tidak memerlukan sumber daya - sekitar 250-300 MB RAM saat startup untuk mode desktop.



Opsi peluncuran alternatif: mode tablet dan telepon


Saat memasuki sistem, Anda dapat memilih salah satu dari beberapa opsi peluncuran: aman, desktop, seluler atau tablet.



Untuk bekerja pada perangkat sentuh, Anda dapat mengaktifkan keyboard di layar.



Mari kita lihat apa yang menarik dalam mode berbeda. Desktop adalah mode normal di mana sistemnya mirip dengan Windows.



Mode tablet cocok untuk layar sentuh besar. Selain perbedaan eksternal yang jelas, yang dapat dilihat pada tangkapan layar di bawah ini, ada fitur antarmuka lainnya. Kursor dalam mode tablet tidak terlihat, tombol tutup aplikasi ditempatkan di bilah tugas. Aplikasi layar penuh bekerja sedikit berbeda, file dalam manajer file juga dipilih secara berbeda.



Perlu disebutkan mode ponsel - di sini semuanya hampir sama dengan di Android. Lingkungan grafis Fly digunakan. Dalam mode sentuh, sentuhan panjang berfungsi, di mana Anda dapat memanggil menu konteks. Mode seluler mengonsumsi sumber daya yang sedikit lebih banyak dibandingkan desktop dan tablet.





Kehadiran mode operasi yang berbeda nyaman. Misalnya, jika Anda menggunakan tablet dengan keyboard plug-in dan, karenanya, kasus penggunaan sentuh dan non-sentuh.

Pembaruan sistem


Sebelum Anda mulai menggunakan sistem, Anda perlu memperbaruinya. Pada dasarnya, repositori Astra Linux memiliki 14 ribu paket (cabang stabil , uji dan percobaan ). Cabang eksperimental akan segera menerima pembaruan yang tidak stabil, jadi kami akan menguji cabang pengujian. Ubah repositori menjadi pengujian.



Kami mulai memperbarui repositori dan memperbarui sistem. Untuk melakukan ini, klik tombol "Perbarui" di kiri atas, lalu "Tandai semua pembaruan", lalu "Terapkan". Mulai ulang.

Kebijakan pengguna


Pengguna baru dibuat dalam sistem melalui utilitas manajemen kebijakan keamanan.



Secara default, fungsi login jarak jauh disediakan (Control Panel - System - Login).



Selain sesi terpisah dan jarak jauh yang biasa, Anda dapat memulai sesi bersarang (Mulai - Shutdown - Sesi).



Dengan dua yang pertama, semuanya jelas. Sesi bersarang adalah sesi yang dimulai di jendela sesi saat ini.



Sesi, omong-omong, dapat diselesaikan setelah waktu yang tertunda: jangan menunggu akhir dari operasi yang panjang, tetapi cukup mengkonfigurasi shutdown otomatis.



Antarmuka Astra Linux dan Perangkat Lunak Standar


Astra Linux Common Edition menyerupai Debian seperti beberapa tahun yang lalu. Terlihat bahwa secara lahiriah Astra Linux Common Edition mencoba mendekati Windows.



Navigasi dan bekerja dengan sistem file lebih dekat ke Windows daripada ke Linux. Dengan gambar sistem, satu set perangkat lunak standar terpasang: kantor, jaringan, grafik, musik, video. Pengaturan sistem juga dikelompokkan dalam menu utama. Empat layar tersedia secara default.


Seperti yang Anda lihat, LibreOffice diinstal sebagai office suite dalam sistem

Panel kontrol mirip dengan Windows / Mac / etc dan mengelompokkan pengaturan dasar di satu tempat.



Manajer file memiliki antarmuka dua-panel dan dapat memasang arsip sebagai folder.





Manajer file dapat menghitung checksum, termasuk yang sesuai dengan GOST R 34.11-2012 .



Mozilla Firefox diinstal sebagai browser standar. Terlihat sangat asketik, tetapi pada saat yang sama cukup memadai. Sebagai contoh, saya membuka dan membuka Habr yang baru. Halaman diberikan, sistem tidak macet dan tidak hang.



Tes selanjutnya adalah pengeditan grafis. Kami mengunduh gambar dari judul artikel Habr, meminta sistem untuk membukanya di GIMP. Tidak ada yang aneh di sini.



Dan dengan gerakan pergelangan tangan kami menambahkan tes untuk efisiensi salah satu artikel. Pada prinsipnya, tidak ada perbedaan dari sistem Linux standar.



Mari kita mencoba untuk melampaui skrip sederhana dan meletakkan paket standar melalui apt-get.



Setelah pembaruan indeks: 
sudo apt-get update

Untuk pengujian, kami menginstal python3-pip, zsh dan pergi melalui instalasi oh-my-zsh (dengan ketergantungan git tambahan). Sistem bekerja dengan normal.

Seperti yang Anda lihat, sistem berkinerja baik dalam kerangka skenario standar sehari-hari dari rata-rata pengguna. Jika Anda berharap melihat program yang familier dengan Debian / Ubuntu di sini, Anda harus menginstalnya tambahan dengan pegangan (misalnya, jika Anda memerlukan paket seperti ack-grep - mereka diinstal melalui curl / sh). Anda dapat menambahkan repositori ke sources.list dan menggunakan apt-get yang biasa.

Utilitas Asli Astra Linux


Alat yang dijelaskan di atas hanyalah sebagian dari apa yang tersedia untuk pengguna Astra Linux. Selain itu, para pengembang telah menciptakan sekitar seratus utilitas tambahan yang dapat dikirim melalui repositori yang sama yang mereka gunakan untuk memperbarui sistem.



Untuk menemukan utilitas, cukup cari kata "fly" - semua utilitas yang diperlukan memiliki awalan seperti itu.



Sulit untuk berbicara tentang semua aplikasi dalam kerangka satu ulasan, jadi kami akan memilih beberapa yang berguna dari sudut pandang pengguna yang sederhana. Aplikasi cuaca menampilkan perkiraan di kota-kota tertentu di Rusia, dioptimalkan untuk wilayah Rusia.



Ada juga utilitas grafis sederhana dengan beberapa filter dan pengaturan untuk mencari file.



Ada utilitas sendiri untuk memantau daya baterai dan berbagai mode, transisi yang dikonfigurasi melalui timer - matikan monitor, tidur, hibernasi.



Pilihan file yang dapat dieksekusi untuk perintah juga dibungkus dengan shell grafis. Misalnya, Anda dapat menentukan "vi" sistem yang akan dipilih saat menjalankan perintah.



Dengan utilitas admin terpisah, Anda dapat mengonfigurasi aplikasi mana yang mulai saat startup sistem.



Ada juga pemantauan GPS / GLONASS, agak berguna di ponsel / tablet (di mana modul yang sesuai biasanya hadir).



Ada juga pembaca PDF sederhana sendiri, untuk tes yang diluncurkan pada buku Free Culture dari Lawrence Lessig.



Anda dapat membaca tentang semua utilitas Fly pada tur virtual Astra Linux, di bagian Bantuan di desktop virtual.

Berbeda dengan sistem utama


Dari sudut pandang antarmuka dan logika kontrol, sistem ini lebih seperti Windows XP klasik, dan kadang-kadang - elemen individual dari Mac OS.

Dari sudut pandang utilitas, konsol, dan perangkat keras, sistem ini mirip dengan Debian klasik, yang cukup bagus dan akrab bagi pengguna Ubuntu dan Minted yang sama, meskipun yang paling canggih akan melewatkan berbagai paket yang biasanya dari semua repositori.

Jika Anda memaksakan pengalaman Anda pada potret pengguna potensial, saya memiliki harapan positif untuk sistem baru. Berdasarkan pengalaman mereka dengan Windows / Mac, pengguna biasa dapat dengan mudah merasa nyaman dengan Astra Linux Common Edition. Dan pengguna Linux yang lebih maju yang menggunakan unix-utilities standar akan mengonfigurasi semuanya sesuai keinginan mereka.

Versi Astra Linux saat ini didasarkan pada Debian 9.4, dan kernel terbaru dari Debian 10 (4.19) juga tersedia.

Tentu saja, ada versi Ubuntu yang lebih baru, tetapi ada satu nuansa kecil tapi signifikan - mereka bukan LTS (Dukungan Jangka Panjang). Versi Ubuntu LTS sama rata dengan Astra Linux dalam versi paket. Putusan


Manfaat utama dari Astra Linux Eagle Common Edition:

  • Tidak jatuh, tidak menggantung, gangguan kritis tidak diperhatikan.
  • Berhasil meniru antarmuka Windows NT / XP.
  • Kesederhanaan dan kemudahan instalasi.
  • Persyaratan sumber daya rendah.
  • Perangkat lunak utama sudah diinstal: LibreOffice office suite, editor grafis GIMP, dll.
  • Satu set besar utilitas tambahan.
  • Versi paket lebih tua dari versi terbaru Ubuntu.
  • Repositori lebih kecil dari Ubuntu dan Debian.

Kesimpulan: Ubuntu versi non-LTS terbaru untuk pengguna rumahan akan lebih baik daripada Astra.

Namun, untuk pengguna rumahan, duduk di distribusi LTS mungkin tidak relevan, tetapi untuk organisasi itu adalah opsi yang cukup normal. Oleh karena itu, pilihan pengembang Astra Linux yang ditujukan untuk segmen korporat jelas dan logis.

Adapun kekurangannya, mereka lebih cenderung benar bagi mereka yang terbiasa bekerja dengan Linux, karena secara lahiriah Astra Linux "Elang" jauh lebih dekat ke Windows daripada ke Linux.

Astra Linux "Eagle" Common Edition terlihat seperti pengganti yang baik untuk versi kantor Windows sebagai bagian dari transisi ke perangkat lunak bebas dari lembaga pemerintah, dan untuk penggunaan di rumah mungkin terlihat agak konservatif.

Dari Astra Linux: kami terus berkomunikasi dengan pengguna sistem operasi kami. Mereka secara teratur menulis kepada kami tentang kesan mereka - tidak hanya mereka yang baru saja beralih ke OS kami, tetapi juga pengguna yang telah menggunakan perangkat lunak kami untuk waktu yang lama. Jika Anda memiliki wawasan yang siap untuk dibagikan dan menggambarkan pengalaman pengguna Anda dengan Astra - tulis di komentar dan di jejaring sosial kami
