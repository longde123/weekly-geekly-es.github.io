Chef, vejo você. Intel Unite - Ferramenta de comunicação profissional



As comunicações dominam o mundo. É a interação que faz de um grupo de pessoas um time, pelo menos no futebol, pelo menos no desenvolvimento de um produto de software. Como tornar a comunicação entre as pessoas, independentemente da distância física entre elas, mais conveniente, natural, e ao mesmo tempo custar um mínimo de custos financeiros e trabalhistas? A Intel oferece uma solução para esse problema. Chama-se Unite - um pacote de software para comunicação profissional.

O Intel Unite possui três componentes:

  • O servidor de autorização é responsável pela segurança do sistema como um todo e é responsável pela emissão de senhas para a conexão inicial;
  • Um hub de computador conectado a vários dispositivos multimídia - é para ele que os participantes da comunicação estão conectados;
  • Computadores clientes com software cliente que estabelecem uma conexão com o hub.

Esquematicamente e em inglês, tem a seguinte aparência


Todo esse sistema funciona da seguinte maneira. Os membros da equipe instalam um programa cliente em seus locais de trabalho. Para se conectar ao hub, você precisará da senha gerada pelo servidor de segurança. Depois disso, seus recursos (ou seja, todos os dispositivos de mídia conectados) ficam disponíveis para todos os participantes da conversa. Bem, a própria comunicação se torna ainda mais acessível, ao mesmo tempo em que você pode conduzir uma discussão geral e conversas cara a cara. Naturalmente, é possível trocar arquivos e mensagens de texto. Unir o tráfego é criptografado com uma chave de 256 bits.

À primeira vista, parece que uma carga significativa é colocada no hub Unite e você precisa instalar um servidor poderoso para lidar com isso. No entanto, um servidor barulhento e volumoso não tem lugar na sala de reuniões e a carga pode ser reduzida usando a tecnologia Intel vPro proprietária. O suporte ao VPro é um pré-requisito para a plataforma de gateway. Mas o tamanho, pelo contrário, é melhor levar um menor para economizar espaço. A Intel tem um computador, como se tivesse nascido para ser um hub Unite. Você provavelmente adivinhou isso, Intel NUC - basta encontrar a opção com suporte a vPro na programação. Outros fabricantes também oferecem seus minicomputadores compatíveis com Unite para salas de reunião - na foto abaixo, por exemplo, a muito bonita HP Elite Slice for Meeting Rooms.


Outra plataforma interessante para implantar o Intel Unite é um painel de vídeo com um slot OPS ( Open Pluggable Specification ) e, consequentemente, um computador OPS. OPS é um padrão aberto para emparelhar um dispositivo de saída de vídeo e um computador. A caixa do computador OPS tem a forma de um paralelepípedo e está equipada com um conector de 80 pinos para conectar a um monitor, com o qual pode ser considerado um, reduzindo assim o número de fios. Veja como é um computador compatível com OPS.


Um recurso distinto do Intel Unite é a capacidade de interagir com módulos de terceiros adicionais; Para criá-los, existe um SDK especial. Usando plug-ins, por exemplo, você pode implementar ferramentas de gerenciamento de reuniões em um produto de terceiros ou em uma solução de conferência na web baseada na nuvem para permitir que o host Intel Unite se conecte à conferência na web. E os fabricantes de equipamentos para salas de conferência (microfones com alto-falantes, câmeras etc.) podem usar plug-ins para criar uma interface conveniente para controlar equipamentos (volume, transmissão de vídeo, iluminação etc.).

A terceira versão do Intel Unite está atualmente disponível para download , os clientes foram criados para iOS, MacOS e Windows. Se você estiver procurando por uma ferramenta de reunião, tente esta, ela pode funcionar para você.
