Microsoft Learning


Die erste Ressource, auf die Sie achten müssen, ist Microsoft Learning . Dies ist die von Microsoft empfohlene Hauptressource zum Erlernen von Azure. Diese Ressource, mit der Sie Azure von Grund auf lernen können, während Sie Ihre Rolle auswählen (Entwickler, Architekt, Administrator, Daten- oder KI-Spezialist (im Verständnis von Microsoft natürlich) ).
Es gibt viele relativ kleine Stücke, die 1-3 Monate lang 30-60 Minuten am Tag untersucht werden können, während eine sehr breite Materialschicht bedeckt wird.



Einer der wichtigsten Punkte ist, dass es viele kleine Laborarbeiten gibt, die beim Aufbau des Muskelgedächtnisses helfen. Nehmen wir zum Beispiel alle Abschnitte in Azure, die für Entwickler von Interesse sein könnten .

Natürlich ca. 1-3 Monate - das ist von Grund auf neu. Wenn Sie in den letzten Jahren mit Azure gearbeitet haben, sollte diese Ressource sehr schnell durchsucht werden, die Wissenslücken finden und sich auf die Prüfung vorbereiten.

Pluralsight RoleIQ


Für den Anfang können Sie Azure-Wissenstests in 3 (von 8) Bereichen kostenlos durchführen: Architektur , Entwicklung , Infrastruktur , KI-Ingenieur , Dateningenieur , DevOps-Ingenieur , Sicherheitsingenieur , Datenwissenschaftler, und auf dem Weg nach draußen 0-300 Ihres Wissens erhalten. Jede Rolle besteht aus mehreren Tests 6-9, mit denen Sie Ihr Wissen umfassend betrachten können. (Sie müssen ein Konto erstellen, Sie müssen keine Karte binden, es ist auch nicht erforderlich, eine kostenlose Testversion zu starten. Tests sind ohne dieses Konto verfügbar.) Architekturbeispiel



Meiner Meinung nach sind dies gute qualitative Tests ( es gibt eine qualitative, aber eine quantitative Methode, die einen Wert liefert ), um Ihr Wissen zu bewerten.

Für jede Frage wird ein Zeitlimit angegeben, abhängig von der Länge des Fragetextes + allen Antwortoptionen. Aber wenn Sie das System hacken wollen, wird es schwierig sein, weil es wird nicht viel Zeit gegeben.

Für all dies ist die Schätzung nichtlinear , d.h. Ich habe 17 von 20 Fragen bestanden - Sie erhalten 255 von 300 Fragen. Dies hängt von der Komplexität der Aufgaben ab und davon, an welchem ​​Punkt Sie sich geirrt haben. Persönlich ist ein solches System für mich ein wenig unverständlich (es stellt sich heraus, dass es schlecht ist, wenn Sie am Anfang einen Fehler gemacht haben, aber wenn es näher am Ende ist, ist es weniger beängstigend), aber es liefert ein qualitatives Ergebnis.



Übrigens können Sie Ihren Stolz amüsieren, weil Am Ende jedes Tests geben sie nicht nur eine Bewertung ab, sondern auch den Prozentsatz der Personen, die besser bestanden haben als Sie. Es ist schön zu sehen, dass 1-3% besser sind als Sie.

Ich sehe diesen Test als einen Weg, die Masse der Menschen, die lernen möchten, nach ihren anfänglichen Fähigkeiten in mehrere Gruppen aufzuteilen.
Nach dem Bestehen des Tests werden Ihnen Kurse empfohlen, die Wissenslücken schließen, aber diese Kurse sind bereits für das Geld.

Offizielle Dokumentation


Dies ist ein großartiger Ort zum Lernen als es allein ist die offizielle Quelle der Wahrheit.

Sie können stundenlang und sogar ganze Tage lang lesen, ansehen und sogar versuchen, die Angaben in der Dokumentation zu wiederholen. Ich sehe 3 Probleme:

  1. Mangel an Laborarbeit. Das ist ein großes Minus, weil Bei vielen Menschen ist das Muskelgedächtnis besser als das visuelle Gedächtnis.
  2. Lokal veraltete Dokumentation (oder zumindest Screenshots). Die Dokumentation ist riesig. Als ich das letzte Mal nachgesehen habe, hatte es zum Beispiel 9000 Bilder, auf die in der Dokumentation überhaupt nicht verwiesen wurde. Zwar gab es 50.000 von denen, auf die verwiesen wurde, aber viele mussten aktualisiert werden. Mit Text das Gleiche.
  3. Manchmal enthält die Dokumentation einfach keine Antworten auf die erforderlichen Fragen. Die einzige Möglichkeit, eine Lösung zu finden, besteht darin, alle Optionen auszuprobieren.

Alle diese drei Probleme überwiegen jedoch immer noch nicht die Tatsache, dass die Dokumentation die Hauptquelle für theoretisches Wissen ist .

Mein empfohlener Weg, um Prüfungen zu bestehen


Wenn Sie ein Projektmanager sind , der Azure verwendet, würde ich die folgenden Ressourcen und Lernschritte empfehlen:
  1. OpenEdx - AZ-900 Vorbereitungsmaterialien
  2. Höchstwahrscheinlich haben Sie viel mehr zu googeln.
  3. AZ-900-Prüfung (Eine Prüfung, die bestätigt, dass Sie alle wichtigen Konzepte in Azure kennen)


Wenn Sie ITPro sind
  1. OpenEdx - Ein allgemeiner Überblick über die Basisdienste. Laborarbeiten werden möglicherweise nicht bestanden, aber das Lesen des Inhalts ist korrekt
  2. OpenEdx-AZ-103 Vorbereitungsmaterialien
  3. In RoleIQ Pluralsight -> RoleIQ -> Microsoft Azure Administrator (führen Sie Tests durch, um Ihre Fähigkeiten zu überprüfen).
  4. AZ-900-Prüfung (Eine Prüfung, die bestätigt, dass Sie alle wichtigen Konzepte in Azure kennen)
  5. Prüfung AZ-103


Wenn Sie Entwickler sind (und .net / C # kennen)
  1. Kurse zu OpenEdx - Entwickler benötigen einen allgemeinen Überblick über die Basisdienste Nicht alle Projekte können mit reinem PAAS ausgeführt werden. Sie müssen jedoch die grundlegenden Infrastrukturdienste verstehen, da dieselben AKS schließlich auf VMs basieren, die mit Vnets mit Load Balanced, Public IP, Netzwerksicherheitsgruppen usw. verbunden sind. .
  2. OpenEdx - AZ-203 Vorbereitungsmaterialien
  3. In RoleIQ Pluralsight -> RoleIQ -> RoleIQ Pluralsight -> RoleIQ -> Microsoft Azure Developer (führen Sie Tests durch, um Ihre Fähigkeiten zu überprüfen).
  4. AZ-900 Optional, aber nützlich (eine Prüfung, die bestätigt, dass Sie alle Schlüsselkonzepte für Azure kennen. )
  5. Prüfung AZ-203

Wenn Sie C # /. Net nicht kennen, wird es für Sie schwieriger zu bestehen sein, aber es ist möglich, weil Alle C-ähnlichen Sprachen sind sehr ähnlich, und Sie können die API anzeigen, ohne C # zu kennen.

Wenn Sie ein Architekt sind (Lösung oder ein anderer Typ). Dieser Weg wird der längste sein, aber "Es gibt keine königlichen Wege zur Geometrie."
  1. OpenEdx - Ein allgemeiner Überblick über die Basisdienste. Laborarbeiten werden möglicherweise nicht bestanden, aber das Lesen des Inhalts ist korrekt
  2. OpenEdx - AZ-300 / AZ-301 Vorbereitungsmaterialien
    • OpenEdx - AZ-103 Vorbereitungsmaterialien (Nicht erforderlich, aber sehr nützlich, da Sie dieselben Dienste aus einem anderen Blickwinkel betrachten können)
    • OpenEdx - AZ-203 Vorbereitungsmaterialien (Nicht erforderlich, aber sehr nützlich, da Sie dieselben Services aus einem anderen Blickwinkel betrachten können)
  3. Ich empfehle dringend, die AZ-900-Prüfung abzulegen. (Eine Prüfung, die bestätigt, dass Sie alle wichtigen Konzepte in Azure kennen. Für Architekten ist dies ein Checkpoint, an dem Sie sicherstellen können, dass Sie auf dem richtigen Weg sind.)
  4. Unter RoleIQ Pluralsight -> RoleIQ -> RoleIQ Pluralsight -> RoleIQ -> RoleIQ Pluralsight -> RoleIQ -> Microsoft Azure Solution Architect (führen Sie Tests durch, um Ihre Fähigkeiten zu überprüfen.)

    • RoleIQ Pluralsight -> RoleIQ -> RoleIQ Pluralsight -> RoleIQ -> Microsoft Azure Developer (Optional, aber zusätzliche Übung schadet nicht)
    • RoleIQ Pluralsight -> RoleIQ -> Microsoft Azure Administrator (Optional, aber zusätzliche Übung schadet nicht)
  5. Erkunden Sie das Azure Architecture Center
  6. Die Prüfung AZ-300/301 ist obligatorisch Es gibt keine besondere Bedeutung, das eine ohne das andere zu übergeben.
