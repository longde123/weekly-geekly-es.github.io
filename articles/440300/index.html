10 comandos de consola para ayudar a debatir el código JavaScript como un PRO



Tradujeron un artículo de Amit Solanka sobre la depuración del código JavaScript mediante comandos de consola. Según el autor, estos comandos ayudarán a aumentar significativamente la productividad del programador al buscar errores y a ahorrar mucho tiempo.

Veamos los comandos que realmente pueden simplificar la vida de cualquier programador.

Agrupando registros usando console.group ('nombre') y console.groupEnd ('nombre')


Los comandos de consola console.group ('nombre') y console.groupEnd ('nombre') proporcionan una agrupación de varios registros dispares en un solo árbol desplegable, lo que brinda acceso rápido a cualquiera de los registros. Además, estos comandos le permiten crear árboles anidados para su posterior agrupación.

Hay tres métodos en total. El primero, console.group ('nombre'), marca el comienzo de la agrupación, el segundo, console.groupEnd ('nombre'), marca el final, y console.groupCollapsed () forma el grupo en modo de árbol colapsado.



Trace console.trace ()


Si el programador necesita una pila completa de llamadas a funciones, entonces debe usar el comando console.trace (). Un ejemplo de trabajar con ella: 

function foo() { function bar() { console.trace(); } bar(); } foo();

Y el resultado.



Contar llamadas con console.count ()


El comando console.count () le permite mostrar la cantidad de veces que se llamó. Vale la pena recordar: si cambia la línea de registro que se le da al equipo, la cuenta regresiva continuará en una nueva. Si lo desea, puede restablecer el contador con el comando console.countReset ().



Inicio y detención del temporizador con console.time () y console.timeEnd ()


Todo es simple aquí. Ambos comandos controlan el temporizador, lo que le permite iniciarlo o detenerlo. Por lo general, se usan para probar el rendimiento. Además, si lo desea, puede crear un temporizador específico; en este caso, debe pasar una cadena a cualquiera de los comandos.



Expresiones lógicas y console.assert ()


La función console.assert () es indispensable para trabajar con expresiones lógicas. Le permite verificar si alguna expresión es falsa. El resultado se registra. En principio, puede usar if, pero la consola es más conveniente. Un ejemplo de trabajo con un equipo: 

 function greaterThan(a,b) { console.assert(a > b, {"message":"a is not greater than b","a":a,"b":b}); } greaterThan(2,1);

Resultado



Perfilado con console.profile ()


El comando console.profile () le permite ejecutar perfiles sin ningún problema. El trabajo manual en este caso no es necesario, porque el equipo hace todo por sí misma. 

 function thisNeedsToBeProfiled() { console.profile("thisNeedsToBeProfiled()"); // ,     console.profileEnd(); }

Línea de tiempo y console.timeStamp ()


Otra característica útil, console.timeStamp (), agrega una marca de tiempo para ciertos eventos. Se puede usar para registrar el momento de devolver una llamada API o para registrar el tiempo de finalización del procesamiento de datos. En realidad, hay muchos casos aquí.

console.timeStamp ('marca de tiempo personalizada!');

Borrar la consola console.clear ()


Todo es simple aquí. Si desea borrar la consola, use console.clear ().

Propiedad console.memory


Le permite mostrar el tamaño del búfer. Vale la pena usarlo si las estadísticas de rendimiento no son muy claras, pero no hay tiempo para familiarizarse con los plazos.



Salida de tabla con console.table ()


La función console.table () le permite mostrar una pequeña tabla con la que el desarrollador puede interactuar. Aquí se usa una matriz como argumento; debe pasarse para llamar.
