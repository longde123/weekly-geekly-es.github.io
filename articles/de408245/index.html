Microsoft wird eine Version von Skype für Freiberufler veröffentlichen



Microsoft wird eine spezielle Version von Skype einführen, die sich an Kleinunternehmer und Freiberufler richtet. In naher Zukunft soll laut Blog Skype eine vorläufige Version in den USA veröffentlicht werden. Das sogenannte Skype Professional-Konto (Skype Professional-Konto) enthält neben Standardtelefonie und -nachrichten einen Kalender, Notizen, eine separate Unternehmenswebsite und ein integriertes Zahlungssystem.

Microsoft geht davon aus, dass die neue Funktionalität für diejenigen nützlich sein wird, die Skype für Konsultationen und Online-Unterricht verwenden. Angenommen, ein Tutor kann in einem Tool kommunizieren, Dateien senden, Termine planen, Notizen machen und Zahlungen erhalten, ohne zu Programmen von Drittanbietern wechseln zu müssen. Eine gewisse „intelligente Planung“ wird versprochen. Der Benutzer erhält sogar eine „separate Website für kleine Unternehmen“, die direkt in Skype geöffnet wird. Dort können Sie die Funktionsweise und verschiedene Sonderangebote anzeigen.

Bereits jetzt können Sie sich auf der Microsoft-Seite registrieren und Tester werden. Das Skype Professional-Konto ist als kostenlos positioniert, aber es ist schwer zu sagen, ob es nach dem offiziellen Start so bleibt oder beispielsweise Teil von Office 365 wird. Das Unternehmen hat es nicht eilig, über alle Innovationen zu berichten - es wird darauf hingewiesen, dass es weitere „unglaubliche Funktionen“ geben wird später.

Um sich zu registrieren, füllen Sie einfach einen Fragebogen mit Fragen zu Ihrem Unternehmen und Ihrer Verwendung von Skype aus. Das Anmeldeformular ist für alle offen, ich habe es ohne Probleme ausgefüllt und verschickt, ohne in den USA zu leben, aber dann heißt es einfach: „Danke für die Antworten.“ Anscheinend müssen Sie warten, bis jemand bei Microsoft entscheidet, dass Sie für die Teilnahme geeignet sind. Der Blog warnt davor, dass die Anzahl der Plätze begrenzt ist.
