Microsoft memperluas Program Azure IP Advantage dengan manfaat IP baru untuk inovator dan startup Azure IoT


Di Microsoft, kami berinvestasi dalam membantu pelanggan kami saat mereka pindah ke cloud. Kami melihat peluang untuk membantu mendukung perusahaan dalam lingkungan yang berubah ini dengan membawa keamanan, privasi, kepatuhan, serta aset dan keahlian kekayaan intelektual kami untuk membantu mereka menjadi lebih sukses. Kami senang sekarang mengambil langkah tambahan yang memperluas perlindungan inovasi.

Hari ini, kami dengan senang hati mengumumkan perluasan program Microsoft Azure IP Advantage untuk memasukkan manfaat baru bagi inovator dan startup Azure IoT . Kami pertama kali mengumumkan Azure IP Advantage pada Februari 2017, untuk memberikan perlindungan komprehensif terhadap risiko kekayaan intelektual (IP) bagi pelanggan cloud kami. Tren yang kami lihat pada saat itu - dan tren yang berlanjut hari ini - adalah risiko yang semakin besar terhadap inovasi cloud dari tuntutan hukum paten. Tahun lalu, kami bergabung dengan Jaringan Penemuan Terbuka (OIN) dan Jaringan Lisensi untuk Transfer (LOT) untuk membantu menangani risiko pernyataan paten bagi pelanggan dan mitra kami.


Artikel ini di blog kami .

Tapi kami yakin kami bisa berbuat lebih banyak. Jumlah paten IoT-litigated di AS menyaksikan peningkatan lebih dari 400 persen dari 2013 hingga 2018. Itu sebabnya, setelah berbicara dengan pelanggan dan merenungkan bagaimana kami dapat menambahkan nilai lebih banyak lagi ke program Azure IP Advantage, kami memutuskan untuk memperluas program dengan manfaat baru yang berfokus pada ekosistem Internet of Things (IoT) yang didukung dan startup untuk membantu mencegah tuntutan hukum terhadap pelanggan Azure. Fitur-fitur baru yang diumumkan hari ini meliputi:


  • Cakupan ganti rugi yang belum dikunci untuk Microsoft Azure Sphere dan Windows IoT. Ganti rugi membantu melindungi pelanggan dari klaim pelanggaran IP yang ditegaskan terhadap pelanggan untuk penggunaan produk atau layanan. Ekspansi hari ini membawa cakupan ganti rugi yang tidak dapat ditutup ke Azure Sphere dan Windows IoT, termasuk perangkat lunak open source yang dimasukkan oleh Microsoft dalam produk-produk ini.
  • Akses ke 10.000 paten untuk pelanggan yang menggunakan Azure untuk memberi daya pada perangkat IoT untuk membela diri terhadap tuntutan hukum IP. Manfaat ini dapat membantu mencegah tuntutan hukum terhadap pelanggan Azure karena beban kerja dan aplikasi mereka yang berjalan di Azure atau pada perangkat IoT mereka, karena pelanggan yang memenuhi syarat dapat memilih paten untuk digunakan dalam pembelaan mereka terhadap tuntutan hukum.
  • Kemampuan startup di Azure untuk memperoleh paten Microsoft untuk membantu meningkatkan bisnis mereka. Startup berkualitas yang juga bergabung dengan LOT Network dapat memperoleh paten Microsoft melalui LOT di bidang teknis termasuk kecerdasan buatan, multimedia, dan keamanan. Paten dapat memainkan peran penting bagi startup saat mereka tumbuh, mencari cara untuk melindungi inovasi mereka, dan menarik dukungan modal kritis.

Banyak startup telah memberi tahu kami tentang kerentanan khusus yang mereka hadapi sebagai perusahaan kecil dan pentingnya mereka menempatkan pada pertumbuhan portofolio IP tahap awal mereka sendiri ketika mereka mencari untuk meningkatkan modal dan membangun masa depan. Mitra kami di LOT Network telah membuat program yang menjawab kebutuhan ini. Microsoft dengan senang hati menyumbangkan 500 paten ke LOT Network, yang sekarang LOT akan tersedia untuk startup yang memenuhi syarat sebagai bagian dari keanggotaan gratis mereka.


Secara bersama-sama, kami percaya Microsoft menawarkan kerangka kerja komprehensif, ujung-ke-ujung untuk mengatasi risiko IP di cloud.


Umpan balik awal dari pelanggan positif:


"Untuk perusahaan teknik internasional seperti thyssenkrupp, program Azure IP Advantage dari Microsoft adalah alat yang hebat untuk mengurangi risiko IP dalam transisi ke Industry 4.0," kata Dr. Stephan Wolke, kepala IP dan Layanan, thyssenkrupp GmbH.


"Riset Otak Terapan membangun sistem AI yang terinspirasi oleh otak," kata Peter Suma, ketua dan co-CEO dari startup di pasar komputasi neuromorfik yang berkembang. "Azure IP Advantage membantu kami fokus pada inovasi berikutnya dan menghabiskan lebih sedikit waktu untuk mengkhawatirkan risiko IP."


Adalah tujuan kami untuk membangun komunitas yang dapat fokus pada inovasi daripada khawatir tentang melindungi inovasi mereka. Kami berharap langkah ini untuk memperluas program Azure IP Advantage akan membantu dalam hal ini. Untuk mempelajari lebih lanjut, kunjungi situs Microsoft Azure IP Advantage.
