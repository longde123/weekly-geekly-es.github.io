顿河畔罗斯托夫的Mitap Sbertekh

你好

我们邀请您参加12月12日18:00在顿河畔罗斯托夫举行的会议。
我们将讨论聊天机器人，移动应用程序，自动测试和关键系统中的日志记录。
关于主题和演讲者的剪下。

演讲主题


1. 使用Python和Java的聊天机器人和NLP。 演讲者Ivan Popandopulo将谈论Sberbank自动化系统中聊天机器人的介绍，NLP的功能及其发展趋势。 此外，他还将分享将@ 900集成到内部系统中的经验，如何使用现成的解决方案以及它们可以做什么。

2. React Native与Web的交互。 来自Sbertekh的Evgeny Cherenov将致力于移动应用程序与Web交互方法的演讲，并将回答为什么您应该警惕嵌入在应用程序中的浏览器的问题。 作为奖励，他将告诉您如何在移动开发中使用现有的基础架构，以及如何注入他人的基础架构。

3. 使用Java在关键业务系统中实现日志记录机制。 Sbertekh的Sergey Usanin将解释如何在工业系统中实现详细的日志记录而不影响生产率。

4. Python中的异步编程和DevOps中的案例 -Megaphone的Cyril Tabelsky。 报告并行化操作的方法及其在高负载服务中的应用。 这是一个使用异步应用程序来组织收集，汇总，指标分析和自动消除事故的示例。

5. 使用Python对警告系统进行自动测试。 MegaFon，Anton Ivashchenko和Dmitry Gusakov的同事将讨论创建自动测试警告系统的工具的经验以及基于事件的警报生成模型。 也可以与他们讨论形成参考数据存储库的概念及其形成方法，并简要讨论使用RobotFramework的实现。

12月12日（星期三）从18:00到22:00。
地址：圣顿河畔罗斯托夫 SFU主楼会议厅Bolshaya Sadovaya，105/42。

该活动是免费的，您需要注册 。
