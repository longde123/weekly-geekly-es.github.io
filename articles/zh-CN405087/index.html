如何使用2G / 3G / 4G信号放大功能连接视频监控



如果您看到有人攀爬建筑起重机而感到恐惧，请记住每天在高海拔工作的人。 起重机工人，窗户清洁工，高层安装人员-所有这些人不是选择办公室的墙壁，而是选择空气和令人眼花height乱的高度。

在这样的条件下工作需要铁腕和可靠的设备。 该设施的安全不仅受到标准行为准则的支持，还受到解决不同任务的监控摄像机的支持（我们曾经使用Ivideon摄像机帮助想要在检测区域中识别没有防护头盔的人的客户）。

在任何建筑工地上，都很难为至少一台摄像机找到正常的互联网。 而且什么时候需要十台相机？ 任务很复杂。 问题是普遍存在的：该国的房屋，地下室的维修，购物中心一楼的商店在厚厚的钢筋混凝土墙后面-到处都是摄像头，需要宽广的互联网渠道。 我们为该问题提供的解决方案是带有调制解调器和路由器的ZLT P10 +天线，用于接收和放大2G / 3G / 4G信号。

LTE天线的差异




左-Videostation M20，右-ZLT P10 +

去年秋天，我们展示了针对不带互联网的对象的一站式解决方案-Videostation M20 ，将20 dB天线结合在一个MIMO封装中（信号放大最大距离可达15 km，并且对运营商的基站没有障碍），调制解调器，路由器-PoE交换机（4个端口）和电源。

Videostation M20具有许多功能。 首先，只能将4个摄像机直接连接到设备，并且您必须通过分配器添加其他摄像机。 其次，只能通过电缆连接摄像机。 第三，19,990卢布的价格并不总是允许具有信誉的第一笔生意的人安装天线和相机。

另外，该装置很大（420×420×110 mm，重5 kg），这使得在许多地方安装困难-例如，将其安装在薄的高桅杆上非常困难。

考虑到所有的微妙之处，我们意识到我们需要寻找替代解决方案。

Videostation M20不会消失在任何地方，但是会出现具有其他功能的ZLT P10 + 。

选项和规格




盒子内部有两个不同的设备。 ZLT P10 +路由器和ZLT P11天线本身 。 为了方便起见，整个套件简称为ZLT P10 +。 选项：

•天线ZLT P10 +
•路由器
•电源
•安装支架
•说明



我们分别称天线和路由器为：外部单元和内部单元

工作温度：-20 ... + 40°C。
交流电源：100 ... 240 V，50 ... 60 Hz。
直流输出功率：15 V，1000 mA。
传输功率：23 dB（比Videostation高，但实际上信号范围几乎相同-15 km无障碍，实际情况下10 km）。
防潮：IP65（全面防尘，防喷水）。
重量：580克-几乎是VideoStation M20的十倍。
尺寸：252×142×50毫米



具有2个以太网端口和一个用于通过常规网络电缆连接到天线的端口的Wi-Fi路由器。 电源旁的黑点是重置按钮，可重置为出厂设置。 趋势路由器支持VPN。 我们的视频已被加密，但是如果您想允许通过VPN进行加密的视频数据通信-现在就可以没有不必要的麻烦。

ZLT P10 +可以做什么


ZLT P10 +天线使用任何俄罗斯移动运营商的USIM / SIM卡来增强信号接收。 资料传输率：

•4G（LTE）传入流量-100 Mb / s，传出流量-50 Mb / s；
•TD-SCDMA-传入流量-2.8 Mbit / s，传出-2.2 Mbit / s。

该摄像机可通过PoE连接（通过电缆）和Wi-Fi摄像机工作-十多个设备（CCTV摄像机）也通过无线网络连接。

安装方式


ZLT P10 +可以直接使用-您只需要连接设备即可。 除了ZLT P10 +，您还需要：SIM卡，电缆，闭路电视摄像机。



USIM卡已插入插槽。

室内机的WAN / POE端口和室外机的LAN端口通过常规的5e类网络电缆连接。 天线通过同一根电缆接收功率。



天线本身通过支架安装在街道上，前部朝向基站。 Wi-Fi路由器放置在室内。 天线和路由器之间的距离最多可以达到50米。 设备重量轻，可将其安装在任何地方。

室内机可以使用标准电缆（双绞线）或通过无线接口连接到计算机。 无需安装驱动程序。 支持的操作系统：Windows（自XP起），MAC OS，Linux。 通过计算机，不仅可以使用网络配置，而且可以使用有关设备操作的信息，包括有关已接收和已发送流量的数据。

指标




从左到右：电源，信号强度（红色，黄色或绿色不亮：无，弱，中等，强），Wi-Fi，室内和室外机之间的连接，PC连接。 室外机上的指示灯显示电源的可用性，信号强度，SM卡的操作，与PC的通信以及设备的操作。

在哪里使用




无处不在。 在大多数建筑工地中，将已经建造的建筑物和办公楼的墙壁连接到移动式起重机上，以在建筑工地巡逻（在起重机的每个转弯处，天线都会接收来自新塔的信号），并到达家中和国内的小型地下商店。

困难条件下的视频文档可增加透明度，向潜在的利益相关者展示实际的施工过程，吸引新客户，向您展示其他情况下看不到的东西。 安装和安装不需要特殊知识。 套件中有一条说明，任何人都可以在五分钟之内找出连接。


与我们以前的LTE天线相比，该设备的使用范围更广：

•重量和紧凑性较低-尺寸和重量轻，您可以将天线安装在任何地方，甚至可以安装在树冠分支上；
•连接的摄像机数量仅受Internet通道的宽度限制（通道速度和流量消耗的要求）
•支持Wi-Fi摄像头（减少了电缆数量，并增加了对该地区进行全面覆盖的可能性）
•价格下降到14,990卢布 。
