Festlegen von LaTeX in erhabenem Text


Im Folgenden finden Sie eine Anleitung zur Installation des Sublime Text- Texteditors, gefolgt von der Möglichkeit, das LaTeX- Computerlayout-System zu verwenden. Als Bonus werden wir ihm beibringen, die Sprache von Julia zu verstehen.


Installieren Sie MiKTeX


  1. Klicken Sie auf der Site auf die Registerkarte Alle Downloads
  2. Wählen Sie Net Installer abhängig von der Bittiefe Ihres Systems.
  3. Klicken Sie auf Herunterladen und warten Sie, bis das Online-Installationsprogramm heruntergeladen ist.
  4. Nachdem Sie die Benutzervereinbarung sorgfältig gelesen haben, aktivieren Sie das Kontrollkästchen MiKTeX herunterladen
  5. Wenn Sie einen Proxy verwenden, geben Sie Ihre Add-Ons in den Verbindungseinstellungen ein ...
  6. Wählen Sie im nächsten Schritt MiKTeX abschließen
  7. Wählen Sie einen Server aus (vorzugsweise einen der besten russischen) und geben Sie ein Verzeichnis an. Warten Sie, bis die erforderlichen Dateien geladen sind
  8. Befolgen Sie Schritt 4 erneut, wählen Sie diesmal jedoch MiKTeX installieren
  9. Starten Sie TeXworks nach Abschluss der Installation über Start
  10. Öffnen Sie ein beliebiges .tex-Dokument oder erstellen Sie ein neues
  11. Wählen Sie pdfLaTeX aus dem Dropdown-Menü und klicken Sie auf das benachbarte Dreieck im grünen Kreis

Dadurch wird die PDF-Datei kompiliert und erstellt. Mit TeXworks können Sie mit vollwertigen Arbeiten beginnen. Mit LaTeX können Sie beispielsweise sehr effektiv ein Diplom oder eine Dissertation sammeln ( vorgefertigte Vorlagen ).


Es ist viel bequemer, LaTeX über Sublime Text zu verwenden. Neben der praktischen Hervorhebung und automatischen Vervollständigung können Sie hier das Bild oder die eingegebene Formel sehen, ohne sie zu kompilieren, dh das Ergebnis während der Eingabe zu sehen. Übrigens können Sie in Sublime Text die Programmcodes vieler Programmiersprachen und Markups eingeben (und wenn Sie die entsprechenden Plugins hinzufügen und dann ausführen): C / C ++, Java, MATLAB, PHP, HTML ...


Installieren Sie Sublime Text 3


  1. Laden Sie Sublime Text 3 herunter und installieren Sie es
  2. Laden Sie Sumatra PDF herunter und installieren Sie es. Mit diesem einfachen Programm können Sie PDF-Dokumente, DJVU, FB2 usw. anzeigen.
  3. Öffnen Sie nach dem Start von Sublime Text die Befehlszeile, indem Sie Strg + Umschalt + p drücken, und geben Sie die Paketsteuerung ein: Paket installieren. Wenn dies angezeigt wird, klicken Sie auf und warten Sie, bis die Verbindung zu den Repositorys hergestellt ist
  4. Wenn aus irgendeinem Grund keine Paketsteuerung vorhanden ist, öffnen Sie die Konsole durch Drücken von * Strg + ` (() und kopieren Sie den Zauber auf den Parseltang, der auf der Plugin-Seite für die Paketsteuerung zu finden ist. Drücken Sie die Eingabetaste und starten Sie Sublime Text neu.
  5. Geben Sie in der Liste der Plugins LaTeXTools ein und klicken Sie, wenn es in der Liste angezeigt wird, und warten Sie, bis die Installation abgeschlossen ist.
  6. Als Nächstes initiieren wir die Standardeinstellungen: Einstellungen / Paketeinstellungen / LaTeXTools / Benutzereinstellungen auf Standard zurücksetzen und dort das System überprüfen
    Wenn alle Beschriftungen grün sind, hat sich Sublime Text mit Sumatra und LaTeX angefreundet, und jetzt können Sie Tex-Dokumente erstellen oder bearbeiten. Wenn Sie Strg + B drücken, erfolgt eine Übersetzung, und wenn keine Fehler vorliegen, wird das ausgelagerte PDF geöffnet. Strg + Umschalt + B - Wählen Sie den Baugruppentyp aus.

Mehr zu LaTeXTools


Das Online-Auffinden von LaTeX- Handbüchern ist für Sie einfach (zum Beispiel: Tutorial , Wiki ). Es ist auch nützlich, den Online-Formeleditor zu verwenden, bis die Befehle im Speicher gespeichert sind.


Julia im erhabenen Text


Geben Sie unter Paketsteuerung: Paket installieren Julia ein , laden Sie das Plugin herunter. Jetzt wird Julia unter anderem auf der Registerkarte Ansicht / Syntax angezeigt. Dies bedeutet, dass Sie jetzt Programme in dieser Sprache mit Hintergrundbeleuchtung und griechischen Buchstaben (\ delta + tab drücken) tab und Dateien speichern können im .jl- Format.


Beim Speichern von Dateien ist es zweckmäßig, diese in Module aufzuteilen: 


 module somename # functions, variables # and some things end


In Sublime-Text eingegebener und im .jl- Format gespeicherter Code kann in REPL ausgeführt werden 


 #     cd("C:\\Users\\User\\Desktop") #   include("MDPSO.jl") #       #      PSO.parabol([2,3,5])


Damit ist eine kleine Anleitung abgeschlossen. Alles praktische Layout und angenehme Beleuchtung!
