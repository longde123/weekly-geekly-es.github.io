DEXP Ixion ML 4.5''评测：寿命长的智能手机-一周无插座需要4,990卢布



近年来，在智能手机制造业中，一个主要趋势流行：屏幕的尺寸和分辨率应尽可能大，设备的主体应尽可能薄。所有具有自尊心的制造商的工程师都对如何使智能手机薄达十分之一毫米感​​到困惑。同时，很少有人认为大众消费者根本不需要微妙的记录，而是需要更多平凡的参数。是的，智能手机理想情况下应该美观，功能强大且具有现代感，但同时又要可靠，方便和实用。

实用性对大多数用户意味着什么？首先，这是不充电的设备的工作时间。根据调查，约有一半的消费者愿意为了自治而牺牲尺寸和重量-也就是说，包括臭名昭著的表壳厚度！的确，如果手机不能长时间离开插座，那么手机的意义何在？同时，在设计超薄设备时，最简单的“保存”方法就在电池上：现代技术使制造几乎任何形状和厚度的电池成为可能，设计人员乐于使用。如今，每天需要多次充电的智能手机不再让任何人感到惊讶。

但是类似的设备适合多少呢？并不是每个人都整天坐在温暖的办公室里，周围都是电源插座-许多行业都与不断的旅行和运动相关联，不仅与汽车相关，从理论上讲，您可以在这里给电池充电（如果点烟器没有被记录仪或导航仪占用，是的）。通常情况下，即使有电源插座，也没有时间等待手机电池充满电。自治不仅对远离“文明”的地质学家或钻机人员很重要，而且对快递员和建筑商，销售和保险代理人，医生和警察，军事人员和水手也很重要-清单还在不断增加。而且这些人不太可能将智能手机的额外时间换成超薄外壳。

俄罗斯品牌DEXP是为数不多的智能手机制造商之一，这些智能手机制造商不仅仅关注超薄电子好奇心的粉丝，还关注大量的听众，他们的电话主要是保持稳定不断的交流，而不仅仅是漂亮的玩具和风格属性。如今，DEXP已成为俄罗斯唯一的品牌，提供全系列配备高容量电池的Android智能手机。该生产线包括十几个不同类别和设备的设备。除旗舰店外，Ixion系列的大多数型号都属于最高10,000卢布的细分市场，也就是说，属于最受欢迎和要求最高的价格类别。此外，价格标签的起价非常适中，为4,500卢布。

与在超预算智能手机领域广泛使用的无名“中国”不同，DEXP保证了全国服务中心的质量和维护（请参阅公司网站上的庞大列表（http://dexp.club/Srvs））。就DNS网络中的零售商店数量而言，该品牌拥有多达1200个服务中心（超过三星和联想）。（事实是DEXP也是零售控股DNS的一部分。）

让我们仔细研究一下DEXP Ixion智能手机系列（具有增强型电池）的典型代表之一-Ixion ML 4.5”。这款入门级设备拥有4.5英寸的屏幕和4,000 mAh容量的电池-价格约为5,000卢布。





包装及设备


DEXP Ixion ML 4.5英寸智能手机以小纸板箱出售，该纸板箱饰有标志性的黑色和蓝色。型号名称显示在盖子上，并显示了智能手机本身，并且包装的底部是该设备的主要技术特征。



该包装包括网络充电器（5 V，1 A），微型USB电缆，简短的印刷用户手册和保修卡。预算模型解释了耳机和任何其他附件均不包括在内。通常，设备非常普通，足以使智能手机完全正常运行。

外观设计


DEXP Ixion ML 4.5''的设计非常简单：前面板完全由玻璃覆盖，触摸按钮，角部轮廓光滑。只有厚度是不寻常的：高达12.5毫米。但是正因为如此，智能手机设法放置了容量为4,000 mAh的大电池。试管的质量不足以达到149克，因此即使电池有所增强，该设备也不能称得上很重。外壳完全由塑​​料制成。

防护玻璃下方的前面板是采用IPS技术制成的4.5英寸显示屏。与廉价的TN智能手机相比，后者提供了更高的亮度和更自然的色彩再现。 DEXP使用IPS很棒，而不是便宜又惨的TN。



DEXP Ixion ML 4.5''的屏幕分辨率适中-540 x 960像素。但是，由于相对较小的对角线，图片看起来不粗糙（像素密度为240 ppi），此外，处理器和图形加速器资源在这里不是在绘制“额外”显示像素上花费，而是在执行更重要的计算任务上。视角足够宽，从锐角角度观看时，图像不会变成蓝色或黄色，这对于廉价的矩阵来说并不罕见。



显示屏下方是三个带背光的标准触摸按钮：“菜单”，“主页”和“后退”。屏幕上方是一个对话动态的网格，一个正面的0.3百万像素摄像头镜头，接近和光传感器以及一个双色LED指示器。给电池充电时，它发出蓝色光，完成后发出绿色光。此外，他可以闪烁蓝色，提醒您错过的事件。





DEXP Ixion ML 4.5''的可移动后盖具有相当大的带肋凹口，因此智能手机可以安全地固定在您的手中。后面板上部的中央是具有自动对焦功能的5百万像素主摄像头的镜头，左侧是LED闪光灯。





后面板的下部由DEXP徽标和正在振铃的扬声器的两个插槽占据，并且在此处进行了特殊的突出处理，以使通话声音不会被减弱，即使该设备躺在屏幕上且桌面朝上时也是如此。

硬件音量按钮安装在DEXP Ixion ML 4.5''的左侧，电源按钮在右侧。





但是，微型USB端口“爬到”了智能手机的顶部面板，旁边是用于连接耳机或耳机的3.5毫米插孔微型插孔。一个非常不寻常的位置-通常将USB端口从底部或侧面放置。
在底部，您可以找到麦克风孔和插槽，以方便卸下后盖。



在Ixion ML 4.5''的底盖下，有两个SIM卡插槽（micro-SIM + mini-SIM），一个microSD闪存卡插槽。另外，当然，此设备的骄傲是可移动的4000 mAh电池。与大多数便宜的双SIM卡智能手机一样，它只有一个无线电模块，因此在通话过程中只有一个SIM卡处于活动状态，并且两者都处于待机模式。说到盖子：它在压缩过程中不会挂出，也不会特别吱吱作响。因此，可以估计智能手机的构建质量为5，可能只有一个很小的负号。





操作系统


智能手机DEXP Ixion ML 4.5在“倒数第二个” Android操作系统-4.4.2 KitKat上运行。我很高兴开发人员没有忘记对“空中”更新（OTA更新）的支持。

 

 



制造商没有安装任何其他图形化外壳或启动器；我们已经拥有了标准的“无杂乱无章”的Android，用户可以根据自己的喜好和谨慎程度对其进行自定义。

在其他预安装的应用程序中，您只能命名Yandex服务，Dr.Web防病毒软件，VKontakte客户端和FBReader，但是，如果用户需要释放内存，则可以轻松删除它们。

 

 

硬件平台和性能


作为硬件平台，DEXP Ixion ML 4.5使用了经过时间考验的“片上系统” MediaTek MT6572，其中包括时钟频率为1.3 GHz的双核ARM Cortex-A7处理器和Mali-400MP图形加速器。还提供Wi-Fi 802.11 b / g / n，蓝牙4.0无线模块，GPS接收器和FM收音机。这里不支持LTE和NFC，这不足为奇-我们回想起设备的预算类别-5,000卢布。但是所有这些都是稳定运行的：两个GPS卫星都能捕捉到，并且对话者都能很好地听到您的声音，而且您也听到了它们，并且Wi-Fi穿透了墙壁。

目前，DEXP Ixion ML 4.5''中的RAM容量为1 GB是完全可以接受的，内置的4 GB驱动器可以使用容量高达32 GB的MicroSD / SDHC闪存卡进行扩展。由于设备中的内部存储器较少，因此无法安装例如无法传输到卡上的多个大容量3D游戏，但是对于各种业务应用程序和组织者而言，这已经足够了。

如今，MT6572是入门级智能手机最便宜的联发科技平台，因此您不应该期望它有性能记录。同时，DEXP Ixion ML 4.5''中的Android KitKat图形界面本身完全不会减慢速度，并且各种应用程序（包括相机应用程序）的启动没有明显的延迟。游戏也进展顺利。







此外，DEXP Ixion ML 4.5''可以应付大多数现代游戏。使用像《部落冲突》这样的简单玩具，智能手机根本没有任何问题，如果您想玩更苛刻的3D游戏，只需设置平均图形设置，就可以得到相当令人满意的fps，而无需任何特殊的“刹车”。这是一个非常有价值的指标，尤其是当您考虑到DEXP Ixion ML 4.5''远非游戏解决方案时。

摄影机


在DEXP Ixion ML 4.5中，安装了两个摄像头：前置0.3百万像素和主5百万像素。摄像机控制是通过联发科技硬件智能手机的常用界面执行的。

 

前置摄像头主要用于视频通话，因此其分辨率已经足够。对于其他所有事情，您都必须使用主摄像头：即使对于时髦的“自拍照”，VGA分辨率也太小。

至于5百万像素的主摄像头，则配备了自动对焦系统和LED闪光灯，这使您可以获得质量非常好的照片，尤其是在光线充足的情况下。但是，当然，如果您需要真正高质量的摄影，最好看看价格较高的设备-7-8千卢布。









离线工作


DEXP Ixion ML 4.5英寸的主要“诀窍”是延长电池寿命。一切都可以解决此问题：不是不是分辨率最高的最大屏幕，也不是经济的双核处理器，当然还有4000 mAh电池。因此，对于我们而言，评估电池在实际操作模式下的充满电尤为有趣。结果非常令人印象深刻。当作为主要电话使用时，加上几个小时的互联网浏览或浏览社交网络，加上拍摄5-10张照片，以及每天几个小时的游戏-DEXP Ixion ML 4.5仅在第四天收费。这是通过智能手机的各种功能的积极使用！如果Ixion ML 4.5仅用于通话和定期查看社交网络，那么这笔费用足以支付5到7天的费用，也就是说，差不多一个星期的时间“没有出口”！辉煌的结果！为此，您只需支付5000卢布。



发现

当然，在现代旗舰店的背景下，DEXP Ixion ML 4.5''的特性相当适中。不过，该模型的资产包括以下优势：高质量，明亮，清晰的IPS屏幕，两个SIM卡插槽，良好的装配，适合预算部分的具有自动对焦功能的体面摄像头，相对较新的Android KitKat操作系统，甚至具有运行3D的能力游戏。加上最主要的一点：距离销售点最多可工作一周，而且价格诱人。
