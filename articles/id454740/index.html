Joomla dan e-commerce


Di bagian ini, kita akan berbicara tentang komponen untuk perdagangan di Joomla sebulan sekali. Tujuannya untuk menunjukkan beragam solusi. Hari ini kami akan memberitahu Anda untuk menjauh dari toko online tradisional dan berbicara tentang komponen untuk agen perjalanan - JAtom-S.

Ekstensi JAtom-S mengubah situs web Joomla menjadi agen perjalanan profesional yang dapat menjual tur wisata dalam mode offline penuh.

Apa itu Atom-S?
Atom-S menghubungkan penyelenggara dan penjual tur tematis dalam satu sistem.
Berkat ini, kecepatan pengambilan keputusan dikurangi menjadi instan, dan interaksi apa pun otomatis.

Komponen ini dirancang untuk menyederhanakan proses menciptakan tur penjualan toko yang kuat. Setelah instalasi, cukup masukkan kunci api dari Atom-C. Setelah itu, komponen akan melakukan semua pekerjaan itu sendiri.

Fitur Ekstensi JAtom-S:

  • Pembalut jendela otomatis dengan tur
  • Secara otomatis membuat kartu tur
  • Pembuatan tag meta otomatis untuk mempromosikan tur, termasuk di jejaring sosial
  • Komponen ini memungkinkan Anda untuk mengkonfigurasi pengaturan caching dan pagination.

Ekstensi tersedia dalam beberapa versi.

  • Versi open source gratis
  • Versi Pro mencakup tema desain dan layanan untuk menyiapkan dan memasang ekstensi

Unduh JAtom-S .
