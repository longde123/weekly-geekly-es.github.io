12月10日的Java摘要


  • “前驱残骸”的第四个Alpha版本已发布-用Java语言编写的“猎户座大师”的免费类似物（这是1993年的游戏）。 第五个Alpha承诺在9月5日发布。 作者承诺切换到beta版后会列出源代码。 您可以从链接下载第四个Alpha。


  • NestedJ诞生了 -一个以Spring Data存储库的形式实现经过读取优化的排序树的库。 该库中的解决方法逻辑为O（n），它允许您执行诸如搜索子节点和父节点之类的操作。 GitHub上有144个提交和31个星。


  • Eclipse基金会成立了Edge Native Working Group ，该小组将帮助将Edge Computing整合到血腥的能源市场中。 他们承诺与其他相关计划（如Cloud Native Computing Foundation）紧密合作。 该公告的作者解释了边缘计算与物联网的区别：如果物联网人员是具有硬件和工业自动化背景的人，那么EC人员将解决相反的问题：他们采用现代化的云基础架构，并尝试找出如何在网络边缘解决问题。 该公告描述了范围，市场和货币的位置以及需要单击哪些链接以更详细地了解的内容。


  • 万维网联盟（W3C） 使WebAssembly成为第四种标准语言。 上一个是HTML，CSS和JS。 该工作是WebAssembly工作组的一部分 。 浏览器何时已有Java？


  • Josh Long 发布了一个 Bootiful Podcast 条目 ，其中与主要开发人员Martin Lippert一起讨论了在Language Server上的工作，将spring与VSCode集成，总体上将Pivotal和Spring团队等等。 播客的时长为一小时十分钟。


  • Josh Long 上传了最新摘要 “ 12月10日的春季”。


  • 具有Spring Cloud Stream和Apache Kafka Streams 的史诗般的Stream Processing 的第六部分已经发布。 这次我们将讨论状态存储和交互式查询。


  • Spring Cloud Data Flow 2.3.0的完整稳定版本已发布 。 对于任务，我们承诺会进行多次部署，但这例如以清单形式存储参数来表示。 Sheduler现在知道可以更新任务，并且在更新之后，所有后续启动都会立即选择新版本。 监控发生了最大的变化，它应该与在Kubernetes和Cloud Foundry上本地部署的应用程序统一工作。 特别是，他们拍摄了Prometheus RSocket Proxy ，现在您可以本地监视Spring Cloud Tasks和Spring Cloud Batch。
