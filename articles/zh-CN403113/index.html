英特尔®至强®处理器，2017年春夏系列


与移动或消费类服务器不同，服务器处理器部分是保守且可预测的。 任何人都不太可能会因此而烦恼，因为可靠性，兼容性和性能对专业人员很重要，而对有效功能却不重要。 尽管如此，这里无疑有运动。 因此，在Intel博客上有一定的周期性（虽然比我们希望的要少 ，但仍然不多 ），我们发布了有关至强处理器当前状况的评论-这是整条生产线的一种瞬时横截面。 好吧，两个有趣的消息促使我们立即进行此评论。

对感兴趣的主题的简短介绍，但以前没有关注Intel Xeon系列的开发的人。 Xeon （正确读取为“ zion”）-使用英特尔酷睿技术和以下更新核心策略的服务器处理器（以前是逐个滴答，现在又是逐个交谈） 。 也就是说，首先出现了Intel Core i3 / i7 Kaby Lake，过了一会儿-Intel Xeon E3 / E7 Kaby Lake。 处理器越复杂，世代之间的差异就越大。 假设Intel Xeon E3v6（Kaby Lake）在Intel Core i3 v7（Kaby Lake）出现8个月后才出现-这是第一个新闻。 但是英特尔至强E5v6并不存在，也不会很快出现，因为当前的一代是第四代，即Broadwell。 在房间里不困惑吗？ Core和Xeon的一代相差一个，因为第一个Zion是在Sandy Bridge的核心上制造的，即第二代Core。

在熟悉了英特尔至强处理器产品线的算法之后，我们开始进行比较研究。

英特尔至强E3


Intel Xeon E3 –入门级单路服务器的处理器，但是其性能足以解决各种各样的任务。 如前所述， 英特尔在今年3月推出了新的第六代Xeon E3v6 。 但是，这并不意味着现在只能订购它们。 服务器市场的惯性很大，如果前几代的平台更适合您的任务/预算，则可以轻松购买v5和v4。

结论


关于英特尔至强产品线计划中的变更，目前尚无确切的官方预测，但是，基于其发展的逻辑，知情的读者可以认为下一次更新将涉及E5型号-他们的第五代产品将面世。 也有可能在现有生产线中发布单独的其他模型。

最后，让我再次提醒您，自2000年代初以来，有关英特尔发布的所有硬件产品的知识储备库很大- 自动化关系知识库 （ARK）。 不幸的是，使用该存储库的便利性由于最新的创新而有所损毁，但是，对于一般的英特尔硬件，尤其是处理器，仍然没有更多的知识渊博的来源。 该知识库也可以作为Android应用程序使用 。
