Instal 3CX PBX di Amazon Lightsail Cloud

3CX baru-baru ini mengesahkan hosting baru, Amazon Lightsail, untuk menginstal entitas virtual PBX. Amazon Lightsail adalah versi Amazon AWS yang disederhanakan dan lebih murah. Memulai server virtual dengan 3CX sekarang dimungkinkan dengan sedikit usaha, lebih murah, dan mencakup semua hal. Ada beberapa konfigurasi siap pakai VPS-server yang cocok untuk memulai PBX dengan kapasitas apa pun . Misalnya, konfigurasi dengan 1 GB RAM seharga $ 5 per bulan. Cocok untuk instalasi dari 8 hingga 16 panggilan simultan. Konfigurasi $ 10 cocok untuk sistem dengan hingga 32 panggilan simultan (dengan beban rata-rata). Penggunaan serius PBX, misalnya, sebagai pusat panggilan (menghubungkan CRM, merekam percakapan, dan 35+ pengguna simultan) akan bekerja dengan baik pada mesin seharga $ 20 atau $ 40 per bulan.

Penting untuk dicatat bahwa menggunakan hosting Anda sendiri dan lisensi 3CX, Andalah yang tetap menjadi pemilik PBX (dan bukan perusahaan pihak ketiga - pemilik cloud PBX, seperti yang biasanya terjadi). Anda memiliki akses ke semua parameter sistem dan mengelola pengguna (mungkin ada pengguna tidak terbatas). Anda juga dapat menghubungkan batang SIP Anda sendiri dan telepon IP yang lebih cocok untuk Anda.

Memasang 3CX PBX di Amazon Lightsail membutuhkan waktu beberapa menit. Masuk ke akun Anda, dapatkan kunci API dan gunakan layanan PBX Express.


Menyiapkan hosting Amazon


Pertama, buat akun Amazon AWS jika Anda belum memilikinya. Berikan detail kartu kredit bahkan jika Anda berencana untuk menggunakan penawaran gratis. Setelah terdaftar, buka konsol AWS . Halaman awal akan terlihat seperti di bawah ini. Bahasa Rusia di konsol Lightsail saat ini tidak didukung.


Kunci akses API


Agar layanan 3CX PBX Express dapat membuat entitas PBX untuk Anda, Anda harus membuat kunci akses API. Untuk melakukan ini, pilih Akun> Akun di sudut kanan atas konsol. Klik tab Lanjut, dan kemudian klik tautan Buka konsol IAM.


Di jendela yang muncul, klik Lanjutkan ke Kredensial Keamanan.


Buka bagian Akses kunci (ID kunci akses dan kunci akses rahasia) dan klik Buat Kunci Akses Baru.



Kunci pas akan dibuat. Salin atau unduh file kunci. Anda akan menggunakan kunci ini di wizard PBX Express.



Instalasi 3CX: PBX Express Wizard


Instalasi dan konfigurasi PBX lebih lanjut dilakukan dengan menggunakan wizard PBX Express . Jika Anda tidak memiliki lisensi 3CX, dapatkan secara gratis . Konfirmasikan alamat email Anda, setelah itu akan menerima lisensi dan tautan ke layanan PBX Express. Ikuti tautan ini untuk mulai menyiapkan 3CX di akun Amazon Anda.


Masuk ke akun Amazon Anda di jendela browser yang sama tempat Anda meluncurkan PBX Express. Pilih layanan Amazon Lightsail dan tentukan ID Kunci Akses yang dibuat sebelumnya dan Kunci Akses Rahasia yang sesuai.


Pilih jenis dan wilayah server virtual Anda.


Setelah itu, pembuatan server VPS dengan 3CX di akun Anda akan dimulai. Ketika Wisaya PBX Express selesai bekerja di konsol Amazon, buka bagian Mesin Virtual untuk melihat server 3CX Anda.

Terhubung ke 3CX Server melalui SSH (Putty)


Saat Anda membuat server pertama di wilayah yang dipilih, kunci pas SSH akan dibuat untuk server virtual Amazon Lightsail. Anda dapat mengunduh file kunci dari bagian Akun> Akun> Kunci SSH. Server virtual dengan nama pengguna admin. Kunci SSH pihak ketiga tidak didukung oleh layanan PBX Express.
