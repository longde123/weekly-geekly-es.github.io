Microsoft telah secara resmi mengkonfirmasi bahwa Edge beralih ke mesin Chromium

Beberapa hari yang lalu, ada desas - desus bahwa Microsoft sedang mengembangkan browser berbasis Chromium, yang akan datang secara default, bukan Edge. Alasannya adalah bahwa karyawan Microsoft tiba-tiba mulai membuat komitmen untuk proyek gratis. Sekarang Microsoft telah secara resmi mengumumkan bahwa mereka benar-benar meninggalkan mesin EdgeHTML-nya sendiri demi Chromium dalam versi desktop browser.

Untuk pengembang web, ini sangat melegakan: selama pengujian, satu mesin akan menjadi lebih kecil, dan banyak fitur asli IE / Edge akan turun dalam sejarah. Namun, menurut Mozilla, keputusan seperti itu oleh kolega dari Microsoft menimbulkan ancaman bagi Internet, karena dominasi platform tunggal dari Google menjadi semakin total. Ada ancaman bahwa Firefox akan jatuh ke dalam ceruk marjinal sehingga pengembang bahkan akan berhenti menguji situs untuk itu. Pangsa Firefox terus menurun. Pada Desember 2018, ia turun ke level terendah selama bertahun-tahun: kurang dari 9% di desktop .
