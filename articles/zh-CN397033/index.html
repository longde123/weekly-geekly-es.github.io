联想NeXtScale平台上的Cineca Marconi超级计算机可用于欧洲研究


联想基于Intel的系统的安装于4月中旬开始。最高性能为每秒20 petaflops。

由Cineca参与创建并基于Lenovo NeXtScale平台构建的新型意大利研究超级计算机已于4月中旬开始部署。该设备基于下一代Intel Xeon Phi和Intel Xeon E5-2600 v4处理器。这些解决方案为科学家提供了技术先进，节能高效的超级计算机。

选定一年后开始的施工协议于3月30日签署，招标于2015年4月发布。该程序于2015年12月结束，联想是基于Intel x86架构的三大服务器制造商之一，并且连续两年是全球排名第一的个人计算机公司，已收到计算机系统的申请。

该协议是Cineca管理层提出的意大利基础设施发展计划的第一步。主要任务是发展科学研究。全球计划分两个阶段吸引5000万欧元的投资。第一个已经开始，它将为科学界提供容量为20 PB的计算机和容量为20 PB以上的数据仓库。该工程将于2017年下半年完成。第二阶段将从2019年开始，其任务是在2020年初之前将计算机的功率提高到50-60 petaflops。


Cineca总裁埃米利奥·法拉利（Emilio Ferrari）表示：“有了这项计划，Cineca重申了其提供数字计算和大数据基础设施的使命，该基础设施可用于研究和技术创新。”

超级计算机安装


新的MARCONI系统将在2016年4月至2017年7月之间完全准备就绪。按计划进行的建设工作包括以下几点：

“初步系统将于6月投入生产。它将基于最近宣布的英特尔至强E5-2600 v4处理器，该处理器基于英特尔x86架构，将产生2 petaflops。

»到今年年底，将增加一个新部分，配备下一代英特尔至强融核处理器家族（代号为Knights Landing）。该解决方案基于多核体系结构，整个配置将包含约25万个核。预计计算机功能将增加到11 petaflops。

”在不久的将来，该系统将在下一代英特尔至强处理器的帮助下达到计划的20 petaflops的容量。

超级计算机使用英特尔Omni-Path架构，该架构可提供数千个系统服务器的高性能互连，以实现高效运行。

Lenovo GSS Powerful Storage是包含IBM Spectrum Scale File System（GPFS）的子系统。它连接到英特尔Omni-Path结构并提供数据存储。

Marconi的逐步发展将使先进处理器技术的使用成为可能，其中包括具有绿色灵魂的极其高效的系统。事实是，Cineca团队设置的参数之一涉及将功率增加到50 petaflop，而所有阶段的能量限制都不会超过3兆瓦。

Cineca


Cineca是总部位于意大利Casalecchio di Reno的一个大学联盟。这个非营利性协会成立于1969年，包括70所意大利大学，5个研究机构以及教育，大学和科学研究部（MIUR）。四十多年来，Cineca一直通过超级计算机和软件为科学界提供支持。该联盟正在为大学行政部门和MIUR创建管理系统，并为工业，医疗保健和政府部门开发信息系统。
