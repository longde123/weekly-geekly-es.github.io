适用于旧计算机的最佳Linux发行版

你好

简而言之：不要丢下旧计算机-使用轻量级Linux发行版来恢复它！

您在使用旧计算机做什么？ 曾经具有良好的硬件配置，但现在已被淘汰的硬件。 为什么不使用Linux还原它？ 我将列出可以在旧PC上使用的最佳轻量级Linux发行版。

适用于较旧笔记本电脑和台式机的最佳轻量级Linux发行版


我将尝试按硬件要求的降序排列列表。 这意味着首先，轻量级Linux发行商将需要最少的硬件。

10.薄荷




Peppermint是基于云的Linux发行版，不需要强大的硬件即可运行。 它基于Lubuntu，并使用LXDE桌面环境进行更流畅的操作。

Peppermint OS随附ICE ，该应用程序可将任何网站集成为桌面应用程序。

Peppermint OS的最低系统要求：
RAM：1 GB RAM（建议2 GB）
CPU：Intel x86处理器
硬盘空间：至少4 GB的可用磁盘空间
您可以在此处下载发行版。

9.鲁本图




我们名单上的下一个发行版是Lubuntu。 顾名思义，Lubuntu基于Ubuntu，但使用LXDE作为桌面环境 。 Lubuntu是Ubuntu衍生的最轻量级的Linux发行版，因此它专注于速度和对较旧设备的支持。

默认情况下，与Ubuntu相比，此分发工具包提供的应用程序要少得多，并且它们主要由对系统不需要的程序组成。



GPicView用于查看图像，MTPaint用于绘图，Evince用于PDF文件，Audacious用于音乐，Gnome-Player用于视频，guvcview用于网络摄像头，Chromium作为网络浏览器，Sylpheed用于电子邮件，Pidgin用于即时消息，作为传输BitTorrent客户端，用于表的Gnumeric，可替代Microsoft Word的Abiword，可用于笔记的Xpad等。

如果您以前使用过Ubuntu，那么Lubuntu似乎对您并不陌生。 它们的软件和存储库相同，因此您可以在Lubuntu上安装在Ubuntu上使用的所有软件。 但是在安装必要的应用程序时，请不要忘记您的系统。 不要使用占用大量资源的应用程序。

Lubuntu的最低系统要求：
RAM：512 MB RAM（建议1 GB）
CPU：奔腾4或奔腾M或AMD K8或更高版本
您可以在此处下载发行版。

8. Linux Lite




顾名思义，Linux Lite（“轻量级”）是不需要高性能硬件即可运行的轻量级Linux发行版。 Linux Lite基于Ubuntu LTS（长期支持）的发行版。 LTS为发行提供了5年的支持，也就是说，在您的计算机上安装Linux Lite之后，更新将在5年内毫无问题地发布。

Linux Lite开发团队说：“ Linux Lite完全可以立即使用，这意味着您不必在第一次运行发行版时就安装其他软件。” 对于初学者来说，这是个好消息，因为他们不必花时间搜索和安装必要的应用程序。 大多数核心应用程序已经内置在操作系统中。

以下是一些预安装的应用程序：Firefox作为Web浏览器，Thunderbird用于电子邮件，Dropbox作为云存储，VLC Media Player用于音乐，LibreOffice作为Microsoft Office的替代品，Gimp用于图像编辑以及Lite调整以自定义您的桌面。

鉴于此发行版基于Ubuntu，如果遇到任何困难，您可以在Internet上找到许多教程，技巧，主题论坛等。

Linux Lite的最低系统要求：
RAM：512 MB RAM（建议1 GB）
CPU：700 MHz处理器
显示器：VGA屏幕，分辨率为1024×768（推荐的VGA，DVI或HDMI屏幕，分辨率为1366×768）
硬盘空间：至少5 GB的可用磁盘空间
您可以在此处下载发行版。

7. LXLE




LXLE是基于最新版本的Lubuntu LTS的Linux发行版。 使用LXDE图形外壳。

尽管LXLE是一种轻量级的发行版，但它试图提供一个直观而美观的UI（用户界面）。 该系统经过了完美的优化，并具有良好的性能，并带有默认情况下预先安装的各种轻量级应用程序。

LXLE的最低系统要求：
内存：512 MB（建议1 GB）
CPU：奔腾3（推荐奔腾4）
硬盘空间：8 GB
您可以在此处下载发行版。

6. CrunchBang ++




CrunchBang ++也称为CBPP或＃！++或CrunchBang Plus Plus。 CrunchBang ++是死发行版的克隆，称为CrunchBang Linux，该发行版以其简单和轻巧而著称。 CrunchBang ++支持较旧的硬件，并且可以正常工作。 它基于Debian 9并使用简约的界面设计。 它还基于Openbox窗口管理器。

该项目的目标与CrunchBang Linux相同：为用户提供易于使用的轻量级Linux发行版，并具有良好的功能。 这就是为什么Crunchbang ++包含简约的设计，简单而优雅的界面。

一些标准应用程序：Geany IDE，Terminal Terminator仿真器，Thunar文件管理器，用于图像编辑的Gimp，用于查看图像的Viewnior，用于音乐的VLC Media Player，用于刻录CD / DVD光盘的Xfburn，作为浏览器的Iceweasel，作为BitTorrent-的传输客户端，表的数字货币，PDF文件的Evince，gFTP-文件传输客户端，Xchat-IRC客户端，AbiWord作为Microsoft Word的替代方案。

CrunchBang ++的最低系统要求：
CrunchBang ++没有官方系统要求。 理想情况下，它应与512 MB的RAM和Pentium 4处理器（CPU）一起使用。
Openbox并不完全适合初学者，但这并不意味着您应该害怕尝试它。

您可以在此处下载发行版。

5. Bodhi Linux




Bodhi Linux是另一个使旧计算机焕发新生的轻量级Linux发行版。 Bodhi Linux主要以其轻便而闻名。 Bodhi Linux没有太多预装的软件，因此ISO文件很小。 但是，在这种情况下，您可以使用apt安装所需的任何应用程序。 在旧计算机上，它可以正常工作，并占用少量内存。

此外，Enlightenment窗口管理器使分发速度更快。 启蒙比其他窗口管理器快。 如果您在使用Enlightenment或任何其他Bodhi功能时遇到任何问题，可以阅读开发团队编写的教程。

Bodhi Linux具有基本应用程序，例如用于图形的Ephoto，作为网络浏览器的Midori，ePad文本编辑器，但是不幸的是，没有多媒体应用程序。 但是不用担心！ 如我所说，您可以通过apt安装其他应用程序。 Ubuntu存储库也可在Bodhi Linux上使用。

Bodhi Linux的最低系统要求：
内存：256 MB RAM
CPU：1.0 GHz
硬盘空间：4 GB的可用磁盘空间
您可以在此处下载发行版。

4. antiX Linux



antiX是基于Debian的轻量级Linux发行版。 众所周知，系统不使用“ systemd”。 如果您不熟悉术语“ systemd”，则不必担心antiX不使用“ systemd”。

代替通常的图形桌面环境，antiX使用icewm窗口管理器，因此即使在最弱的设备上，系统也可以正常工作。 antiX可以用作计算机恢复的快速LiveCD / LiveUSB系统。

antiX Linux的最低系统要求：
内存：256 MB RAM
CPU：PIII处理器
硬盘空间：2.7 GB可用磁盘空间
您可以在此处下载发行版。

3. SparkyLinux




我们名单上的第三名是SparkyLinux。 SparkyLinux是另一个轻量级Linux发行版，除较旧的计算机外，还针对现代硬件。 这就是为什么此发行版有两个版本，其中一个带有所有必需的已安装工具，编解码器，插件等，以便系统可以在第一次启动时就开始工作的原因。

SparkyLinux基于Debian，并具有图形桌面环境的多个选项，包括LXDE，OpenBox / JWM，e17，MATE，LXQt，Cli和GameOver版本。 LXQt比Cli（命令行界面）以外的其他环境要快得多。

SparkyLinux的最低系统要求：
内存：用于LXDE，e17，Openbox，GameOver的256 MB RAM和用于MATE，LXQt的384 MB RAM
CPU：i486 / amd64
硬盘空间：5 GB的可用磁盘空间
您可以在此处下载发行版。

2. Puppy Linux




Puppy Linux的最新版本是Puppy Linux 6.0.2 Tahrpup CE，其大小仅为199–201 MB（！）。 可以从非常小的USB驱动器下载此发行版。

可以从Live CD / DVD / USB引导Puppy Linux，并且在引导后可以弹出CD / DVD / USB，Puppy Linux将继续正常运行。 Puppy从RAM加载并运行，因此速度非常快。 除其他外，您可以将任何数据保存在启动Puppy Linux的同一USB上。

默认情况下，Puppy Linux使用JWM和Openbox窗口管理器，这大大减少了资源消耗。 Puppy Linux仅包含一组基本应用程序。 例如，用于表的Gnumeric，可替代Microsoft Word的Abiword，以及用于编辑图形和播放多媒体的各种程序。

Puppy Linux的最低系统要求：
内存：64 MB（建议256 MB）
CPU：333 MHz
您可以在此处下载发行版。

1.小核心




准备好体验Tiny Core的刺激。 我敢打赌，即使Windows上最小的图像编辑应用程序也要比Tiny Core Linux难。 是的，它的重量仅为16 MB（!!）。 下载小的，低质量的视频所需的时间甚至比下载Tiny Core Linux所需的时间还要长。

该系统仅带有最必要的程序和简约的界面。 如果您的计算机仍然很旧，请安装Tiny Core并熟悉这个小奇迹。 它的加载速度比任何其他已知操作系统都要快。



Tiny Core Linux基于Damn Small发行版，但目前它是完全独立的。 默认情况下，Tiny Core带有FLTK / FKWM和BusyBox GUI。 您会错过很多东西，但是不用担心，如果需要，您可以手动安装所有这些东西。

Tiny Core有三种版本：Core，Tiny Core和CorePlus。 第一个版本仅包含CLI（命令行界面），并且只有资深/高级用户可以使用它。 当前版本的Core仅重9 MB。

Tiny Core适用于熟悉GUI（图形用户界面）的普通用户。 TinyCore的当前版本为16 MB。

Core Plus是安装映像，而不是发行版。 推荐给仅可访问无线网络的新用户或使用非英语键盘布局的新用户。 该发行版包括一个基本的Core System，带有用于自定义系统的实用程序：7种窗口管理器的选择，通过各种固件文件和ndlswrapper的无线支持，对非英语版布局的支持以及重新制作工具。 当前版本的CorePlus的大小仅为72 MB。

Tiny Core Linux的最低系统要求：
内存：46 MB（建议64 MB）
CPU：奔腾2
您可以在此处下载发行版。
