Pomodoro技术的理论与实践
Nika Belkovsky，阿卡迪亚质量保证工程师

世界上第一个“西红柿”始于1987年。 从那时起，关于这种简单但有效的浓缩技术已有很多报道。 ，许多文章和YouTube视频的作者都只限于描述基本技术，却忽略了现代The Pomodoro Technique的许多重要方面。 许多人都可以谈论他们的个人经历。 幸运的是，这与我的报告无关！ 毕竟，我会告诉你：
•这是什么设备，
•通常忘记提及的内容
•有科学依据吗？
•我怎么来找她的，
•我如何使用它，
•它如何在测试人员的日常工作中为我提供帮助。

代码审查，或如何提高代码质量
Dena Bubnov，Arcadia开发人员

代码审查是开发的重要阶段之一。 那么什么是Code Review，在此过程中隐藏的优点和缺点是什么？ 在项目中实施代码审查可能会出现什么问题？ 什么是适得其反的《准则审查》，为什么在审查过程中会出现人际冲突？ Code Review在重构中扮演什么角色？ 我将谈谈代码审查的一些微妙之处，错误的观点和决定，并分享负面和正面的经验。

我将讨论微服务环境中的跟踪和监视。 我们的项目经验，实用建议，陷阱。
