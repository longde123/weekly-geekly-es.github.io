Wöchentlicher OpenGift Developer-Wettbewerb



Nehmen Sie am OpenGift Online Hackathon teil!


Die Aufgabe des nächsten Wettbewerbs wird am 18. August um 10:00 Uhr GMT eröffnet, dem Preispool von 10.000 US-Dollar an Plattform-Token!

Hackathon-Regeln

1. Aufgabendetails werden allen Teilnehmern gleichzeitig bekannt gegeben.
2. Die Aufgabe ist 24 Stunden gegeben
3. Die Ergebnisse werden gleichzeitig als Links zu einem offenen Repository veröffentlicht
4. Das beste Ergebnis wird von der Jury ausgewählt
5. Der Gewinner erhält den vollen Betrag des Preisgeldes

Anforderungen an Teilnehmer und Ergebnisse

1. Jeder kann am Hackathon teilnehmen
2. Das Ergebnis muss in einem offenen Repository veröffentlicht werden.
3. Das Ergebnis sollte eine Demonstration in Form einer Weboberfläche (für Aufgaben im Zusammenhang mit dem Web) oder einer Videoaufzeichnung des Ergebnisses (für andere Aufgaben) enthalten.

Mögliche Stapel: PHP + JS, Python, Ruby

Nachfolgende Hackathons finden samstags statt, der Preispool wird separat bekannt gegeben und kann sowohl Plattform-Token (sie können zu einem festen Kurs gegen andere Währungen innerhalb des Systems eingetauscht werden) als auch Fiat enthalten.

Zur Teilnahme ist eine Registrierung erforderlich .

Sie können den Ankündigungen neuer Hackathons in der Gruppe folgen.
