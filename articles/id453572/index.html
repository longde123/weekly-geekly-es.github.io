Microsoft Edge untuk macOS

Bulan lalu, kami mengumumkan versi pratinjau pertama dari Microsoft Edge versi berikutnya untuk Windows 10. Hari ini, kami dengan senang hati mengumumkan ketersediaan saluran Microsoft Edge Canary untuk macOS. Anda sekarang dapat menginstal pratinjau bangunan dari situs Microsoft Edge Insider untuk macOS atau Windows 10 PC Anda, dengan lebih banyak dukungan versi Windows segera hadir.



Membangun pengalaman pengguna "seperti Mac" untuk Microsoft Edge


Microsoft Edge untuk macOS akan menawarkan pengalaman penelusuran baru yang sama dengan yang kami pratinjau di Windows, dengan optimisasi pengalaman pengguna agar terasa seperti di rumah sendiri di Mac. Kami menyesuaikan tampilan dan nuansa keseluruhan agar sesuai dengan apa yang diharapkan pengguna macOS dari aplikasi pada platform ini.



Kami berkomitmen untuk membangun peramban kelas dunia dengan Microsoft Edge melalui fitur pengalaman pengguna yang berbeda dan layanan yang terhubung. Dengan rilis awal ini, kami telah membuat beberapa perubahan pada antarmuka pengguna untuk menyelaraskan dengan bahasa desain Microsoft sementara membuatnya terasa alami di macOS.


Contohnya termasuk sejumlah tweak untuk mencocokkan konvensi macOS untuk font, menu, pintasan keyboard, casing judul, dan area lainnya. Anda akan terus melihat tampilan dan nuansa peramban yang berevolusi dalam rilis mendatang ketika kami terus bereksperimen, beralih, dan mendengarkan umpan balik pelanggan. Kami mendorong Anda untuk berbagi umpan balik Anda dengan kami menggunakan smiley "Kirim umpan balik".



Selain itu, kami merancang pengalaman pengguna yang eksklusif untuk macOS, dengan memanfaatkan fitur perangkat keras tertentu yang tersedia di Mac. Misalnya, memberikan tindakan yang berguna dan kontekstual melalui Touch Bar seperti pintasan situs web, penggantian tab dan kontrol video, serta mengaktifkan navigasi yang akrab dengan gerakan trackpad.



Memperkenalkan Saluran Insider Microsoft Edge untuk macOS


Pembuatan pratinjau Microsoft Edge baru untuk macOS tersedia melalui saluran pratinjau yang kami sebut "Microsoft Edge Insider Channels." Kami mulai dengan meluncurkan Saluran Canary Microsoft Edge Insider, yang dapat Anda unduh dan coba di situs Microsoft Edge Insider . Saluran ini tersedia mulai hari ini di macOS 10.12 ke atas. Dev Channel akan dirilis segera, dan setelah tersedia, Anda akan dapat mengunduh dan menginstalnya berdampingan dengan Canary Channel. Anda dapat mempelajari lebih lanjut tentang pendekatan kami dan apa yang diharapkan dari berbagai saluran dalam posting blog kami dari bulan lalu .



Platform dan alat yang konsisten untuk pengembang web


Dengan dasar Chromium baru kami, Anda dapat mengharapkan pengalaman render yang konsisten di seluruh Windows dan versi macOS dari Microsoft Edge, serta alat pengembang kuat yang sama yang akan Anda temukan di Windows.


Untuk pertama kalinya, pengembang web sekarang dapat menguji situs dan aplikasi web di Microsoft Edge pada macOS dan yakin bahwa pengalaman itu akan bekerja sama di versi Microsoft Edge berikutnya di semua platform. (Perhatikan bahwa kemampuan spesifik platform, seperti dekripsi konten PlayReady pada Windows 10, harus terus dideteksi fitur untuk pengalaman terbaik pada platform tersebut.)


Seperti halnya pratinjau Windows yang kami buat, versi macOS baru kami juga mencakup dukungan untuk Aplikasi Web Progresif yang dapat diinstal dan standar yang dapat Anda periksa dan debug menggunakan alat pengembang browser. Kami sedang berupaya membuat PWA merasa betah di samping aplikasi asli Anda, jadi ketika dipasang mereka akan muncul di Dock Anda, pengalih aplikasi, dan Spotlight seperti aplikasi asli.


Bagikan umpan balik Anda


Dengan senang hati kami membagikan build Microsoft Edge Canary pertama kami untuk macOS dengan Anda! Mendapatkan umpan balik Anda adalah langkah penting dalam membantu kami membuat peramban yang lebih baik - kami menganggap penting untuk menciptakan pengalaman penelusuran terbaik di macOS. Kami harap Anda akan mencoba pratinjau hari ini , dan kami berharap umpan balik dan partisipasi Anda dalam komunitas Microsoft Edge Insider.


Jika Anda menghadapi masalah apa pun, dan untuk memberikan umpan balik atau berbagi saran dengan tim, kunjungi forum komunitas Microsoft Edge Insider , hubungi kami di Twitter , atau cukup gunakan opsi "Kirim umpan balik" di menu Microsoft Edge untuk beri tahu kami pendapat Anda.


Untuk pengembang web, jika Anda menemukan masalah yang mereproduksi di Chromium, yang terbaik adalah mengajukan bug Chromium . Untuk masalah dalam versi Microsoft Edge yang ada, silakan terus menggunakan Tracker Masalah EdgeHTML .
