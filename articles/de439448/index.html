Die Frage der Lizenzierung für das System wurde aus folgenden Gründen nicht eingereicht:
  • Das Problem der Lizenzierung tritt in vielen Disziplinen auf und ist sehr ablenkend.
  • Werden Sie mit einer Antwort zufrieden sein wie: OEM / BOX / „kommerziell“ (elektronisch, Abonnement usw.) / Testversion / Vermietung + Schulung + Free / ShareWare + OpenSource Zoo + Abonnement für Support-Services;
  • Die Lizenzierung kann in Teilen (Servern / Installationen / Clients), in Einschränkungen der verwendeten Ressourcen (Prozessorkerne / RAM / zur Emulation verfügbare Geräte), in Editionen / Versionen, in Kosten für einen Anbieter oder in Investitionen in Personal, das OpenSource implementiert, angegeben werden.
