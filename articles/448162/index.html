Utilidades gratuitas de Solarwinds para monitorear, administrar la infraestructura y seguridad de TI



Conocemos bien a Solarwinds y hemos estado trabajando con él durante mucho tiempo, muchos también conocen sus productos para el monitoreo de redes (y no solo). Pero no se sabe tan ampliamente que le permiten descargar de su sitio web unas cuatro docenas de utilidades gratuitas que ayudarán a controlar dispositivos de red, administrar infraestructura, bases de datos e incluso manejar incidentes. De hecho, este software es fragmentos separados de sus productos pagos. Todas las utilidades son 100% gratuitas, no versiones de prueba. Debajo de cat links a la descripción y descarga.

En esta revisión, utilidades para:

  • gestión de infraestructura de red;
  • Gestión de infraestructura de TI;
  • Gestión de seguridad informática
  • gestión de bases de datos;
  • Organización de servicio de mesa de ayuda.

Gestión de infraestructura de red


Edición gratuita de ipMonitor




Una herramienta simple La interfaz establece un rango de direcciones, las sondea y entrega nodos disponibles para monitoreo remoto. De las posibles herramientas de monitoreo: ping, WMI, SSH. En los servidores, puede realizar comprobaciones estándar como CPU, Memoria, Disco. Esta es la versión gratuita del hermano mayor ipMonitor (no gratuita) y admite hasta 50 nodos. Para una oficina pequeña o tareas de prueba, no hay nada en absoluto. Hay un video corto con una descripción .

Paquete de herramientas de flujo




Replicación, prueba y configuración de dispositivos para enviar tráfico de flujo. La herramienta consta de tres partes: un replicador, un generador y un configurador. El replicador puede recibir y reenviar tráfico de flujo de red a uno o más receptores. Generador: genera un flujo de red para probar, verificar la configuración de los firewalls y otros fines. El configurador configura automáticamente los dispositivos de red para enviar flujo de red al dispositivo de destino. Hay un video corto con una descripción .

Traceroute NG




Esta utilidad es un complemento para el sistema tracert. Realiza análisis de ruta de red, puede ejecutar paquetes usando protocolos TCP e ICMP. Hay un video corto con una descripción .

Escáner de puertos




Genera una lista de puertos abiertos, cerrados y filtrados para cada dirección IP escaneada. Capaz de escanear a través de TCP y UDP, y después de escanear, puede cargar todos los datos recopilados en un archivo. Funciona incluso a través de la línea de comando.
Hay un video corto con una descripción .

Monitor de dispositivo de red




Monitoreo de dispositivos de red. La utilidad viene con muchas plantillas diferentes para el monitoreo a través de SNMP, puede configurar informes y alertas. Pero hay una limitación: solo se admite la supervisión de un solo dispositivo. Un video corto con una descripción .

Emulador de red GNS3




Emulador de entorno de red. La utilidad funciona en Windows, Mac y Linux. Se admiten más de 20 proveedores de equipos de red. Puede emular varias topologías de red y ver cómo funciona todo. Hay un video con una descripción .

Visor de tiempo de respuesta para Wireshark




Complemento Wireshark para un fácil análisis de paquetes por tiempo de respuesta. Detecta tráfico para aproximadamente 1200 aplicaciones. Funciona como una calculadora para el tiempo de respuesta de la aplicación. Hay un video con una descripción .

Paquete de analizador de red y monitoreo de ancho de banda




La utilidad está diseñada para monitorear el tráfico de NetFlow, J-Flow y sFlow. La interfaz tiene representaciones en forma de varios sectores: en la interacción entre dispositivos, en aplicaciones, en dominios y dispositivos finales. Además, es posible registrar datos de hasta 60 minutos.

Servidor TFTP




Servidor TFTP con una interfaz gráfica. Se ejecuta en Windows como un servicio. Hay un video con una descripción .

Rastreador de direcciones IP




Escaneo, confirmación de cambios, identificación de conflictos de direcciones IP. Este es el hermano menor de un administrador senior de direcciones IP: puede administrar hasta 254 direcciones, a diferencia de 2 millones en una solución paga.

Monitor de ancho de banda en tiempo real




Monitoreo de ancho de banda de interfaces de red y visualización de métricas. También es una solución junior para Network Performance Monitor (NPM).

Rastreador de registros de detalles de llamadas




Busque, filtre y clasifique los registros CCM CDR.

IP SLA Monitor




Recopile datos IP SLA de dispositivos Cisco y visualícelos.

Calculadora de subred avanzada




Calculadora de IP para el diseño del espacio de direcciones.

Kiwi Syslog Server Free Edition




Análisis, recepción y archivo de syslog y SNMP. Hay un video con una descripción .

FTP Voyager FTP Client para Windows




Cliente FTP avanzado para Windows. Admite FTP, FTPS y SFTP. A través de este cliente, puede realizar transferencias de archivos programadas.

Servidor SFTP / SCP




La utilidad funciona como un servicio de Windows y admite la transferencia simultánea de archivos a múltiples dispositivos.

Gestión de infraestructura de TI


Calculadora de costos para azul




La utilidad puede mostrar información financiera para varias cuentas simultáneamente. La interfaz tiene un desglose de los costos por año, trimestre, mes y, además, se muestran los recursos pagados que no se utilizan. Funciona en plataformas Windows y Mac.

Reenviador de registro de eventos para Windows




Empaqueta eventos del registro de Windows y los envía como syslog donde quieras. Hay un filtrado flexible. Qué eventos de los registros de Windows deben controlarse: escribimos en el último artículo .

Solar-PuTTY




Masilla avanzada. Tiene una interfaz de múltiples pestañas y una conveniente búsqueda de sesiones guardadas. Hay un video con una descripción .

Monitor vm




Supervisión del rendimiento de VMware e Hyper-V. La interfaz muestra métricas básicas de rendimiento y el estado de las máquinas virtuales.

Monitor de estado del servidor




Diseñado para monitorear servidores físicos. De fábrica puede funcionar con servidores Dell PowerEdge, HP ProLiant, IBM eServer xSeries e hipervisores VMware ESX / ESXi. Admite protocolos de interoperabilidad SNMP, WMI y CIM.

Monitor de rendimiento de almacenamiento




Monitoreo de matrices de datos Dell EMC, NetApp, IBM, Pure Storage. Hay un video con una descripción .

Monitor de intercambio




Monitoreo de servicios clave de intercambio. Hay un video con una descripción .

Activador de ejecución remota para PowerShell




Utilidad para la configuración automática del servicio WinRM local y remoto.

Paquete de administrador para el directorio activo




Un paquete de tres utilidades para: detectar cuentas no utilizadas, detectar estaciones de trabajo no utilizadas e importar cuentas de usuario en AD. Hay un video con una descripción .

Herramienta de diagnóstico para el agente de WSUS




Una utilidad para verificar la disponibilidad de los servicios de Windows Update (WSUS) en servidores remotos.

Transacción de transacción web




Una utilidad para realizar pruebas sintéticas. Una transacción es compatible con aplicaciones web.

Monitor wmi




Una utilidad para controlar un servidor de Windows a través de WMI. Es el servidor, porque solo se admite un dispositivo.

SNMP Enabler para Windows




Configure contadores SNMP en servidores remotos de Windows.

Gestión de seguridad de TI


Consolidador de registro de eventos




Windows Log Consolidator con conveniente búsqueda de eventos y configuración de alertas condicionales. Hay un video con una descripción .

Navegador de firewall




Una utilidad para probar las reglas del firewall. Admite la importación de configuración desde dispositivos Cisco, Check Point y NetScreen.

Analizador de permisos para Active Directory




Un analizador de derechos de acceso para usuarios, miembros de grupos y derechos para administrar pelotas de red.

Gestión de bases de datos


Analizador de rendimiento de bases de datos gratuito




Analizador de consultas en tiempo real para bases de datos SQL Server, Oracle, DB2 y SAP ASE. Muestra consultas lentas. Hay un video con una descripción .

Advertencias del plan SQL




Utilidad para analizar el plan de ejecución en SQL Server. Supervisa la utilización de recursos a medida que avanza el plan. Hay un video con una descripción .

Organización de la mesa de ayuda


Web Help Desk Edición gratuita




Gestionar incidentes, problemas y niveles de servicio. Hay integración con AD e informes. Este es el hermano menor de una versión totalmente funcional.

La mayoría de estas utilidades son adecuadas para resolver problemas locales, algunas pueden usarse como sistemas de producción. En los últimos 2-3 años, Solarwinds ha estado desarrollando activamente sus productos y ahora ya pueden considerarse como un sistema universal para monitorear el nivel de la organización. Si está buscando un sistema de monitoreo para este rol, considere incluir Solarwinds.
