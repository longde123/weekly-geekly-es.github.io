汽车行业的3D扫描仪：4个好处和4个成功的项目



在有关在汽车工业中使用添加剂技术的材料中 ，我们详细研究了它们在俄罗斯工业中的主要优势和潜力。 尽管3D打印的引入仍然充满某些困难（例如，需要大量投资和自动化程度不足），但如今，三维扫描被视为汽车行业最可取且负担得起的3D技术。

3D扫描仪是一种旨在快速分析物理对象并创建其准确的3D计算机模型的设备。 其操作原理基于使用两个摄像头计算到物体的距离。 除照相机外，还使用背光-LED或激光。 两种类型的扫描仪均适用于汽车行业。

3D扫描仪提供了一个机会，可以在开发阶段显着减少时间和成本，提高产品质量，并最终加速产品在市场上的发布。 它们可用于产品生命周期管理的任何阶段，并将有助于优化该行业中任何企业的生产流程-从大型汽车，发动机，特殊设备和组件制造商到执行调整，维护和维修的小型公司。

3D扫描设备消除了传统测量设备的许多限制。 计量学家所熟悉的诸如模板，千分尺，卡尺之类的仪器价格便宜，但读数的主观性有所不同，因此不适用于复杂的测量。 坐标测量机比3D扫描仪更准确，但是它们更昂贵，尺寸更大，并且需要特殊的操作人员培训。

就价格和质量而言，包括3D扫描仪的光学控制系统是最佳的解决方案，因为它们提供：

  1. 测量速度
  2. 数字化复杂几何物体的精度很高，
  3. 可以自主工作
  4. 易于操作。

借助3D扫描仪，大大简化了设计师，技术人员和设计师的工作：费力的复杂测量的实施以及从头开始进行设计的创建已成为过去。

在本文中，我们将通过特定的示例演示该技术的好处，并且在下面的视频中，您可以实时跟踪3D扫描的过程。

3D扫描解决什么任务？




  1. 质量控制 ：检查任何几何参数的能力，包括输入和输出控制，零件，外壳和工业设备的计量控制。
  2. 汽车零部件的逆向工程，可迅速收到设计文档和产品现代化信息。
  3. 用于自动调整，原型设计和产品外观评估，生产车间和设备现代化的设计和建模 。
  4. 数字存档任何必要的分类（例如，停产的零件）。 存储在数字图书馆中的模型可从世界任何地方远程获得。

汽车行业3D扫描的主要优势


  1. 准确度

    当然，准确性是计量学的基本标准。 3D扫描仪可进行有效测量，测量精度为20至80微米（取决于设备和物体的尺寸）。
  2. 速度

    绩效就是绩效。 3D扫描将测量速度提高到一个新的水平：数字化和在诸如车门之类的组件中进行处理的过程大约需要20分钟，而不是在CMM上进行4个小时的测量。 您不仅节省时间，而且降低了生产成本。
  3. 可靠度

    现代专业和工业3D扫描仪的特点是稳定性和可靠性更高。 手持设备设计用于运输到遥远的物体，并且可以在工业振动条件下工作。 iQB Technologies在工作中使用的所有便携式3D扫描仪都会定期在俄罗斯和邻国旅行，而不会发生任何故障。
  4. 简单性

    使用3D扫描仪不需要技能，对于没有经验的用户来说，掌握设备-无论是OTK专家，首席工程师，设计师还是技术人员-只需20分钟即可。 处理点云需要了解软件，并且如果员工没有具有CAD技能的员工，则可以通过网络将扫描结果发送给外包。

汽车行业的质量控制


3D扫描的主要任务是控制几何形状，以便通过参考模型进行验证，这将有助于优化汽车生产。 您不仅可以控制汽车零部件，还可以控制模具和工具，以及执行工具磨损分析。

3D扫描仪的使用使得可以快速跟踪来自供应商的零件的结合情况，以及计量证书和验证方法的可用性，以获取有关声明的几何形状偏差的报告。 如果发现婚姻，则可以找到原因，将扫描结果与参考CAD模型进行比较，并向制造商或第三方组织提供报告。

逆向工程


3D扫描仪解决的第二个重要生产任务是逆向工程或逆向工程。 使用3D扫描和专用软件，您可以修改现有产品而无需图纸，包括不停产的模型，以及升级车身的各个部分以将更新后的模型快速推向市场。 例如，对于在Kalina的基础上开发的新Grants的发布，扫描仪将是完美的选择。

此外，逆向工程执行的任务是，由于陈旧，折旧或不足而更换生产设备，并评估电流，并在车间中选择生产设备和公用事业的最佳布置。



用于MX附件开发的3D拖拉机扫描

这是法国公司MX 引入3D扫描的示例，该公司生产用于农用拖拉机的装载设备。 拖拉机制造商很少共享其新产品的信息，而由于这些拖拉机具有知识产权，因此要获得这些拖拉机的CAD模型就更加困难。 但是，对于MX装载起重机等附件的开发和生产，绝对需要3D模型。 因此，该公司必须对要为其设计和生产附件的所有拖拉机执行3D测量，直到最近，借助机械手的3D测量解决了这些任务。 如今，MX使用3D扫描技术来数字化设计适应性设计所需的所有区域，包括支架连接点及其周围环境。 Creaform 3D解决方案将测量时间缩短了2倍以上，因此缩短了产品的上市时间。

创建CAD模型和数字档案


手动获取设计文档和图纸会花费很多时间。 数字存档和获取产品的CAD模型是3D扫描仪打开的机会之一。 您可以停止使用任何零件，快速扫描并获得3D模型，以进行进一步的复制和归档。 数字图书馆使用起来非常方便：设计人员或技术人员将可以随时随地访问所需的模型，并轻松对其进行校正。

如果您拥有成品的数字模型，则可以将功能测试或碰撞测试后获得的结果与原始产品或参考CAD模型进行比较。 这样可以对损坏和磨损进行分析，并对设计进行适当的改进。

通过存储有关在生产现场扫描的对象的信息，该公司有机会对其产品进行成像，预测未来的生产周期并降低成本。
