PhotoGuru切换到“暗面”和“更明智”

我继续尝试使用PhotoGuru的原型，以了解识别照片的“样式”是否可以以某种方式帮助用户。

在新版本中，界面已更新，程序仅给出一个答案。 通常，这是摄影师的名字，以及用户照片与六个摄影师之一的风格之间的“相似度”百分比。



如果百分比低于33％，则系统无法确定作者，并得出结论，在她前面是一个新的“大师”，而不是姓氏上显示了“ PHOTO GURU”字样。 

if (maxPercent <= 100/numPhotographers*2) { message = " ★ " photoGuru = "Photo Guru" }

我将我的Zhenya Otetskoy的 照片和肖像添加到系统中 ，该模型在31％的案例中开始出错。
编辑完样本后，我意识到我正在开始调整结果的答案。 需要另一种方式。 事实证明，该系统的教学时间更长。 我将训练周期数增加到20，现在该系统在分析测试样本中的照片时几乎不会出错。



总的来说，该模型正确地将她未知的肖像进行了分类，但是系统认为我的一张旧照片是Zhenya Otetskoy的照片，而且我不确定自己是否正在拍摄自画像。



但最让我印象深刻的是，仅通过肖像训练，该系统就能够以99％的精度确认带有“火车”的风景是由Evgeni Otetskiy拍摄的。



我试图找到一张与该系统已知的作者不同的照片。 我没有成功。

系统认为Tim Walker的照片与Zhenya Otetskoy的照片相似，Mike Brodie拍摄了Egor Voynov之类的照片，其抽象是由Maxim Shumilin制作的。



该模型最有可能在构图级别分析照片，从而产生这样的结果。 她无法欣赏美学和符号，并且她将白色正方形识别为其中一位作者的照片。

与PhotoGuru一起玩对于自学摄影很有用。 为此，您需要增加系统已知的摄影师数量，考虑防止“骗子”并添加“奖励”。 例如，向从所有摄影师那里收集了“ 99％”徽章的用户提供“ Videnin 99％”徽章和“大勋章”。

如果某人获得“ PHOTO GURU”的身份-独一无二的文凭。
