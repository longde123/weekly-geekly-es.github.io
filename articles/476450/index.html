Descripción general de cPanel: una consola basada en web para administrar sitios y hosting

Continuamos la serie de publicaciones sobre alojamiento y paneles de control del sitio (consolas web) que RUVDS ofrece a sus clientes. Hoy nos fijamos en uno de los paneles cPanel más populares, cuya licencia se puede obtener de forma gratuita durante 2 semanas, al comprar un servidor virtual .

Las publicaciones anteriores se pueden ver aquí: Descripción general de ISPmanager , Descripción general de Plesk y Análisis de la nueva consola web Plesk Obsidian . Recuerde que las licencias para Plesk e ISPmanager se pueden obtener de forma gratuita hasta el final de este año cuando compre VPS . Acerca de otros paneles: en la publicación Novedades de las consolas web 2019 . Y aquí sobre el alojamiento: cómo elegir VPS y alojamiento: opciones, comparaciones, estadísticas de usuario .



cPanel es una de las consolas web comerciales más buscadas en el mundo, especialmente en Norteamérica y Europa. La propia compañía desarrolladora escribe en el sitio web oficial sobre su producto de la siguiente manera: "Una plataforma de alojamiento líder en la industria con soporte de clase mundial". Es reconocido como conveniente tanto para trabajar con sitios como para revender hosting, lo cual es en gran parte por qué es tan popular. En realidad, el panel consta de dos módulos de interfaz: un panel de control de sitio gráfico (cPanel) y la consola de administración de Web Host Manager (WHM). Estos entornos funcionan juntos y puede administrar el sitio en cualquiera de ellos, por lo tanto, cPanel es ideal tanto para principiantes como para desarrolladores que prefieren una consola en lugar de un diseño hermoso.

Cpanel le permite:

  • Crea sitios y dominios con unos pocos clics
  • Lanzar y administrar redes de correo electrónico
  • Almacenar y administrar archivos web



WHM le permite:

  • Crear cuentas de cPanel para clientes
  • Monitoree y proteja su servidor
  • Transfiera y respalde datos en unos pocos clics



PL básico: Perl, PHP (modos PHP: suPHP, DSO (mod_php), FastCGI, CGI).
Sistema operativo compatible: Red Hat Enterprise Linux (RHEL), CentOS, CloudLinux. Es posible admitir Windows a través de la virtualización o mediante el panel Enkompass de los mismos desarrolladores.
Soporte de servidor web: Apache, Nginx, Apache + Nginx, LiteSpeed.
Servidores DNS: Bind, MyDNS, NSD.
Servidores FTP: ProFTPd, PureFTPd
Servidores SMTP: Exim
Servidores IMAP y POP3: Courier, Dovecot

▍ interfaz


La interfaz de cPanel consta de las siguientes secciones:



  1. Barra de navegación
  2. Panel lateral
  3. Información del hogar
  4. Caracteristicas
  5. Estadísticas

Muchas personas consideran que cPanel está algo sobrecargado de funciones, lo que dificulta a los principiantes. Sin embargo, en primer lugar, cPanel es un sistema multifuncional que habla por sí mismo; en segundo lugar, los desarrolladores están constantemente tratando de acelerar el panel, están trabajando activamente para mejorarlo, en función de las solicitudes de los clientes, en particular en la interfaz intuitiva y fácil de usar; En tercer lugar, el sitio web de la compañía tiene videos de capacitación , un foro y un blog, además de soporte técnico las 24 horas . Tenga en cuenta que no será fácil para los principiantes tener un nuevo programa para ellos (aunque cPanel también se encarga de esto), pero es poco probable que los administradores experimentados sufran la abundancia de herramientas proporcionadas para un trabajo conveniente y rápido. Los describiremos.

Características de la consola web de CPanel


▍ Archivos




Esta sección contiene herramientas para trabajar con archivos del sitio (incluida la interfaz de administración de conexión FTP), para monitorear y administrar el espacio en disco disponible de una cuenta, para realizar copias de seguridad y recuperación de archivos de la misma.

Administrador de archivos


La interfaz le permite administrar archivos: cargar, crear, eliminar y editar archivos sin la necesidad de FTP u otras aplicaciones de terceros.

Imágenes


La interfaz para trabajar con imágenes le permite ver y cambiar el tamaño de las imágenes, convertir los tipos de archivos de imágenes.

Privacidad del directorio


Esta interfaz debe usarse para restringir el acceso a ciertos recursos en el sitio, a saber, para proteger con una contraseña aquellos directorios a los que no desea permitir el acceso de los visitantes.

Uso del disco


Ayuda a administrar el uso del disco: le permite escanear el disco y obtener resultados gráficos de una revisión del espacio disponible de la cuenta.

Disco web


Una función para acceder a los archivos del sitio como si fueran una unidad local en su computadora.

Cuentas FTP


Interfaz para administrar cuentas de protocolo de transferencia de archivos (FTP).

Conexiones FTP


Interfaz para ver las conexiones actuales al sitio a través de FTP.

FTP anónimo


Use esta interfaz para que los usuarios puedan conectarse a su directorio FTP sin una contraseña.

Copia de seguridad


Interfaz para hacer una copia de seguridad del sitio. Le permite descargar una copia comprimida de los contenidos de su cuenta de cPanel (directorio de inicio, bases de datos, servidores de reenvío de correo y filtros de correo electrónico) a su computadora. Asistente de copia de seguridad : una guía paso a paso para ayudarlo a realizar copias de seguridad.

Control de versiones Git


Interfaz para alojar repositorios Git en la cuenta de cPanel.

Recuperación de archivos y directorios


Interfaz para recuperar elementos de copias de seguridad de archivos.

▍ Bases de datos




Esta sección contiene varias herramientas para administrar bases de datos y sus configuraciones, tanto internas como de terceros. Las sencillas guías paso a paso lo ayudan a comprender las diversas configuraciones.

Bases de datos MySQL


Para administrar grandes cantidades de información en los sitios.

Asistente de base de datos MySQL


Asistente de base de datos MySQL. Con el asistente, puede configurar bases de datos MySQL, cuentas de usuario y privilegios de usuario. Se recomienda que utilice este asistente para crear la primera base de datos y usuarios.

MySQL remoto


Una interfaz para configurar bases de datos a las que los usuarios pueden acceder de forma remota. Use esta función si desea que las aplicaciones (por ejemplo, tableros de anuncios, carritos de compras o libros de visitas) en otros servidores tengan acceso a sus bases de datos.

Bases de datos PostgreSQL


Interfaz para gestionar grandes cantidades de información en sitios. Para crear simultáneamente una base de datos y una cuenta que accederá a ella, use la interfaz del Asistente de base de datos PostgreSQL .

Phppgadmin


Una herramienta de terceros que se puede usar para administrar bases de datos PostgreSQL.

Phpmyadmin


Una herramienta de terceros que se puede utilizar para administrar bases de datos MySQL.

▍ Dominios




Estas son las herramientas para crear, agregar y administrar dominios de cuenta. Use este menú para asignar alias a dominios, asignar URL cortas o redirigir una página web a otra.

Editor del sitio


Interfaz para crear rápidamente sitios web simples a partir de varias plantillas.

Dominios


Para crear y administrar múltiples dominios para una cuenta.

Dominios adicionales


Interfaz para agregar dominios a la cuenta. Cada dominio adicional tiene sus propios archivos y se mostrará como un sitio separado para sus visitantes.

Subdominios


Estas son subsecciones del sitio que pueden existir como un sitio nuevo sin un nuevo nombre de dominio. La interfaz se puede utilizar para crear URL atractivas para varias áreas del contenido del sitio. Por ejemplo, puede crear un subdominio para su blog, al que los visitantes pueden acceder a través de blog.example.com y www.example.com/blog .

Alias


Los alias de dominio le permiten especificar nombres de dominio adicionales para dominios de cuenta existentes. Esto les dará a los usuarios la oportunidad de acceder al sitio web si ingresan la URL de dominio especificada en sus navegadores.

Redireccionamiento


Use la interfaz Redirigir para redirigir una página web en particular a otra y mostrar su contenido. Esto permite a los usuarios acceder a una página con una URL larga a través de una página con una URL más corta y más memorable.

Editor de zona


Use la interfaz del editor de zonas para agregar, editar y eliminar registros DNS A, AAAA, CNAME, SRV, MX y TXT.

▍ Correo electrónico




Una de las secciones más importantes del panel contiene herramientas para trabajar con dominios de correo electrónico. Incluye varios filtros, funciones antispam y configuraciones de cifrado. Aquí puede configurar calendarios, aplicaciones de contacto y listas negras.

Cuentas de correo electrónico


Interfaz para administrar cuentas de correo electrónico para dominios.

Transportadores


Interfaz de reenvío que le permite reenviar cualquier correo electrónico entrante de una dirección a otra. Es decir, configure una dirección de correo electrónico para reenviar copias de correos electrónicos entrantes a otra dirección.

Enrutamiento de correo electrónico


Use la interfaz de enrutamiento de correo electrónico para redirigir el correo entrante del dominio a un servidor específico.

Contestadores automáticos


Una interfaz para enviar correos electrónicos automáticos a cualquier persona que envíe correo a una cuenta de correo electrónico específica. Puede usar esta función si, por ejemplo, está de vacaciones, no está disponible o tiene un mensaje general que desea enviar desde la dirección de correo electrónico de su equipo de soporte.

Dirección predeterminada


Una interfaz para configurar una dirección predeterminada o una dirección universal que recibirá cualquier correo enviado por los usuarios a una dirección de correo electrónico no válida para un dominio.

Listas de correo


Le permite crear una dirección de correo electrónico para enviar correo a múltiples direcciones.

Seguimiento de entregas


Interfaz para ver los intentos de entrega de correo y rastrear la ruta de entrega. Por ejemplo, si necesita detectar problemas con la entrega de correo electrónico.

Filtros globales de correo electrónico


Use la interfaz de Global Email Filters para crear filtros de correo electrónico para todas las cuentas de correo de su cuenta de cPanel.

Filtros de correo electrónico


Le permite crear filtros de correo electrónico para una dirección de correo electrónico específica. Puede usar esta función para evitar el spam, redirigir el correo o enviar mensajes al programa.

Autenticación


La función le permite proteger su servidor de correo y ver información sobre los mensajes salientes, así como verificar si un mensaje ha sido enviado por un remitente de confianza.

Importador de direcciones


Use la interfaz del Importador de direcciones para importar archivos para crear múltiples direcciones de correo electrónico o reenviadores al mismo tiempo.

Filtros de spam


Interfaz de filtros de correo no deseado para configurar Apache SpamAssassin: plataforma antispam con la capacidad de filtrar, identificar, clasificar y bloquear correos masivos no deseados.

Archivo


Interfaz para guardar listas entrantes, salientes o de correo por un cierto período de tiempo.

Cifrado


Use la interfaz de cifrado para configurar GnuPG, un esquema de cifrado disponible públicamente que utiliza un enfoque de clave pública para cifrar mensajes que solo la clave privada puede descifrar.

Boxtrapper


La función de proteger el buzón de correo no deseado. Requiere que todos los remitentes de correo electrónico que no están en su lista blanca respondan a un correo electrónico de confirmación antes de que pueda recibir sus mensajes.

Calendarios y contactos


Función para configurar calendarios y aplicaciones de contacto. No funciona con la aplicación iOS Reminder si su servidor usa un certificado SSL autofirmado y no es compatible con Horde Free / Busy para aplicaciones de terceros.

Configuración de la lista gris


La interfaz para administrar Greylisting para cada dominio en la cuenta. Protege a los usuarios de correo electrónico del correo no deseado.

Disco de correo electrónico


Una interfaz para ver todos sus buzones y eliminar rápidamente cartas antiguas o mensajes grandes.

▍Metrica




La sección contiene interfaces para ver el ancho de banda del sitio web, el número de visitantes durante una determinada unidad de tiempo, errores del sitio web y configuraciones métricas en diferentes dominios.

Visitantes


Aquí puede ver las entradas de registro de Apache más recientes para este dominio.

Errores


Aquí puede mostrar hasta 300 de los errores más recientes del sitio. Estos datos lo ayudarán a encontrar enlaces rotos, archivos faltantes y otros problemas.

Rendimiento


Ver el uso mensual y total del ancho de banda del sitio web.

Acceso sin procesar


Ver los visitantes del sitio sin gráficos, cuadros u otros gráficos. Descargue la versión archivada del registro de acceso de su servidor para acceder rápidamente a los visitantes de su sitio.

Awstats


Visualización de métricas AWStats es un programa de terceros que produce estadísticas visuales sobre los visitantes de su sitio.

Estadísticas análogas


Ver métricas Analog Stats es un programa de métricas que proporciona un resumen simple de los visitantes del sitio.

Webalizer


Metric Viewer Webalizer es un programa de métricas que crea gráficos y cuadros sobre los visitantes del sitio.

Webalizer FTP


Vea las métricas de Webalizer para FTP, creando varios cuadros y gráficos sobre los visitantes que usaron el protocolo FTP para acceder al sitio.

Editor de medidas


Le permite seleccionar los programas de métrica que desea ejecutar en cada dominio.

▍ seguridad




La sección contiene varias herramientas para administrar claves y certificados, configurar el acceso y el número máximo de inicios de sesión, establecer prohibiciones en enlaces activos a sus sitios y otras configuraciones de seguridad. Incluso hay una función para restringir el acceso a un sitio desde un rango de direcciones IP, es decir, Puede denegar la entrada a un país entero.

Acceso ssh


Interfaz para una conexión remota segura al servidor a través de la línea de comando.

Bloqueador de IP


Para restringir el acceso al sitio desde un rango de direcciones IP. Si ingresa un nombre de dominio completo, esta función intentará traducirlo a una dirección IP.

SSL / TLS


La función de administrar claves SSL / TLS, certificados y solicitud de firma. Úselo para mejorar la seguridad del sitio web.

Protección Hotlink


Use la interfaz de Protección Hotlink para deshabilitar los enlaces activos a sus sitios. Un enlace activo ocurre cuando otro sitio incrusta contenido de su sitio y usa su ancho de banda para servir archivos.

Protección de sanguijuela


Use la interfaz para establecer el número máximo de inicios de sesión que pueden ocurrir en dos horas. Esta característica lo ayuda a administrar usuarios que desean publicar sus contraseñas en un área privada de su sitio.

Política de seguridad


Para configurar los problemas de seguridad que muestra cPanel cuando intenta iniciar sesión en su cuenta desde una dirección IP no reconocida.

Modseguridad


Para habilitar o deshabilitar ModSecurity para sus dominios.

Autenticación de dos factores


Una interfaz para configurar una medida de seguridad mejorada que requiere dos formas de autenticación para ingresar al sistema. Si deshabilita esta función, también debe eliminar todas las configuraciones personalizadas de 2FA existentes.

Asistente SSL / TLS


La función facilita la compra e instalación de certificados SSL para dominios en su cuenta. También enumera los certificados adquiridos que el proveedor de certificados SSL no proporcionó.

Estado SSL / TLS


Para ver el estado o la actualización de los certificados SSL.

▍ Software

Php


Ver información específica de configuración de PHP para su servidor. Esta interfaz solo aparece en servidores que ejecutan EasyApache 3.

Paquetes PHP PEAR

Módulos Perl

Rubygems

Ruby on Rails


Una interfaz para crear e implementar una aplicación Ruby on Rails desde su cuenta de cPanel.

Software del sitio


Interfaz para agregar software útil al sitio. El software disponible para usted depende de la configuración de su proveedor de alojamiento para su cuenta.

Optimizar sitio web


Configure cómo responde Apache y mejore el rendimiento del sitio web.

MultiPHP Manager


Gestión de configuración sencilla de cuentas PHP. Verá esta interfaz solo si EasyApache 4 se está ejecutando en su servidor.

MultiPHP INI Editor


Configuración de PHP Verá esta interfaz solo si EasyApache 4 se está ejecutando en su servidor.

Gerente de aplicaciones


El uso de la interfaz del administrador de aplicaciones le permite implementar rápidamente y admitir fácilmente las aplicaciones de Ruby. Verá esta interfaz solo si el servidor inicia EasyApache 4 con Ruby a través del perfil Mod_Passenger.
