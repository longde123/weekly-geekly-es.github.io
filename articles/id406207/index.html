Mencoba ProtonVPN Gratis di iOS

Apa itu ProtonVPN



ProtonVPN adalah layanan Jaringan Pribadi Virtual dari pengembang ProtonMail. ProtonMail adalah layanan webmail terenkripsi populer yang dibuat pada 2013 oleh CERN (Organisasi Eropa untuk Riset Nuklir). Server dan kantor pusat berlokasi di Swiss.


Kelebihan terbesar dan paling terkenal dari tim pengembangan adalah fokus mereka pada privasi dan keamanan. ProtonMail berbeda dari banyak layanan dengan kemampuan enkripsi huruf ujung-ke-ujung (dengan analogi seperti pada kurir modern). Korespondensi antara pengguna ProtonMail selalu dienkripsi secara otomatis. Secara umum, tim memiliki reputasi yang sangat baik di Internet.


Karena itu, ketika peluncuran layanan VPN dari perusahaan yang sama diumumkan pada musim semi 2017, ini adalah berita penting yang tersebar di semua portal TI. Layanan itu sendiri dimulai pada musim panas. Selain akun berbayar, dengan analogi dengan ProtonMail, layanan ini juga mengumumkan tingkat gratis, tetapi dengan pendaftaran terbatas. Itu perlu untuk mengajukan undangan. Saya telah melihat dari dekat opsi menggunakan VPN untuk waktu yang lama (terutama pada perangkat seluler dan dalam menghadapi risiko memblokir messenger favorit saya). Diajukan pada Juli. Dan kemarin, pada 24 Agustus, sebuah undangan masuk, dan saya langsung mendaftar. Pada intinya, antara permintaan untuk undangan dan kwitansi, dibutuhkan waktu lebih dari 1 bulan.


Fitur Layanan


Saat ini, layanan VPN berbayar sudah cukup, membandingkan versi berbayar tidak terlalu menarik. Menurut pendapat saya, jika Anda membutuhkan privasi dan kemandirian, yang terbaik adalah meningkatkan VPN pribadi Anda sendiri untuk penyedia cloud asing dengan harga $ 2-5.


Karena itu, saya akan menjelaskan dengan tepat apa yang ditawarkan dalam akun gratis :


  • Anda dapat memilih dari server VPN di 3 negara: Belanda, AS, Jepang
  • Batasi 1 koneksi simultan
  • Prioritas rendah pada kecepatan

Opsi berbayar sebesar $ 4 per bulan menghilangkan batasan berdasarkan negara dan memperluas batas koneksi simultan menjadi 2, tetapi secara subyektif, untuk menggunakan hanya pada ponsel yang cukup dan tingkat gratis.


Instalasi, konfigurasi, dan jebakan di iOS


ProtonVPN belum memiliki program iOS sendiri. Untuk terhubung, mereka menyarankan menggunakan OpenVPN Connect (selanjutnya hanya OpenVPN Connect) dan unggah profil koneksi dari ProtonVPN di sana. Meskipun situs tersebut memiliki petunjuk terperinci untuk pengaturan, tetapi saya menemukan beberapa jebakan. Selanjutnya, saya juga akan menjelaskan cara menyiasatinya:


  • OpenVPN mengaitkan file "* .ovpn" (file dengan profil koneksi). File dengan pengaturan profil perlu diunduh dari situs web ProtonVPN, tetapi Safari seluler saat menyimpan file memanggilnya "Tidak Dikenal" tanpa ekstensi, dan tidak jelas bagaimana cara membukanya di OpenVPN. OpenVPN tidak melihatnya. Masalahnya diselesaikan dengan mengunduh file dengan browser normal. Saya mengunduh file profil di komputer desktop dan mengirimkannya kepada saya sendiri melalui surat, meskipun metode ini dianggap kurang aman.
  • Dalam daftar profil untuk unduhan, Anda tidak dapat langsung melihat dalam daftar umum atau memfilter server dan negara yang tersedia untuk pengguna gratis. Secara pribadi, saya berakhir dengan Belanda, AS, Jepang, yang berada di bagian yang berbeda dari daftar.
  • Untuk masuk melalui OpenVPN, Anda harus memasukkan login khusus dan kata sandi yang dibuat di situs web ProtonVPN dan di sini muncul masalah. Katakanlah Anda telah berhasil menyalin dan menempelkan info masuk. Setelah beralih ke Safari untuk melihat dan menyalin kata sandi, bidang input login / kata sandi di OpenVPN akan dihapus. Reddit datang untuk menyelamatkan, di mana ada hack kehidupan super. Setelah Anda memasukkan login, Anda harus mengklik koneksi ke VPN. Program akan mencoba untuk terhubung dan, tentu saja, tanpa kata sandi itu akan terputus oleh batas waktu (30-60 detik). Yang paling penting, setelah bidang login seperti itu tidak lagi dihapus ketika beralih antar program, dan ini memungkinkan Anda untuk menyalin dan menempelkan kata sandi secara normal.

Ada banyak ulasan di AppStore bahwa OpenVPN tidak berfungsi, tapi saya tidak punya masalah selain yang dijelaskan di atas.


Kualitas kerja


Ini bekerja dengan baik. Situs "Dibutuhkan" terbuka. Kecepatan melalui Belanda adalah 11 Mbps saat diunduh dan 17 Mbps saat diunggah. Mengapa kecepatan unggah lebih cepat? Mungkin karena memuat server VPN. Saya pikir bahwa melalui AS dan Jepang semua kecepatan akan lebih rendah. Juga, saya kira seiring waktu kecepatan akan turun ketika pengguna baru tiba. Tetapi sebagai layanan untuk persidangan, menurut saya, semuanya baik-baik saja.
