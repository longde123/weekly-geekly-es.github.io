Windows Virtual Desktop sekarang dalam pratinjau publik di Azure

Kami baru-baru ini membagikan pratinjau publik dari layanan Windows Virtual Desktop di Azure. Sekarang pelanggan dapat mengakses satu-satunya layanan yang memberikan manajemen yang disederhanakan, multi-sesi Windows 10, optimisasi untuk Office 365 ProPlus, dan dukungan untuk desktop dan aplikasi Windows Server Remote Desktop Services (RDS). Dengan Windows Virtual Desktop, Anda dapat menggunakan dan menskalakan desktop Windows dan aplikasi Anda di Azure dalam hitungan menit, sambil menikmati keamanan dan kepatuhan bawaan.



Ini berarti pelanggan sekarang dapat melakukan virtualisasi menggunakan desktop dan aplikasi (10 RDS) multi-sesi Windows 10, Windows 7, dan Windows Desktop untuk Windows Virtual Desktop untuk pengalaman manajemen dan penyebaran yang disederhanakan dengan Azure. Kami juga membangun Windows Virtual Desktop sebagai solusi yang dapat dikembangkan untuk mitra kami, termasuk Citrix, Samsung, dan Microsoft Cloud Solution Providers (CSP).


Akses ke Windows Virtual Desktop tersedia melalui lisensi RDS dan Windows Enterprise yang berlaku. Dengan lisensi yang sesuai, Anda hanya perlu menyiapkan langganan Azure untuk memulai hari ini. Anda dapat memilih jenis mesin virtual dan penyimpanan yang Anda inginkan sesuai dengan lingkungan Anda. Anda dapat mengoptimalkan biaya dengan memanfaatkan Instance yang Dicadangkan dengan diskon hingga 72 persen dan menggunakan Windows 10 multi-sesi.


Anda dapat membaca lebih detail tentang Windows Virtual Desktop di blog Microsoft 365 yang diterbitkan hari ini oleh Julia White dan Brad Anderson .


Mulai dengan pratinjau publik hari ini.
