Torrents Time: Popcorn Time sebagai plugin browser. Sudah didukung di The Pirate Bay



Layanan Popcorn Time untuk melihat file video dari torrent tanpa unduhan awal telah tersedia sebagai plug- in browser. Plugin open source didukung di situs khusus untuk menonton video Popcorn Time Online secara online .

Plugin ini tersedia untuk Internet Explorer, Chrome dan Firefox untuk OS Windows 7 (dan di atas) dan OS X 10.8 (dan di atas). Pengembang berjanji untuk menambahkan dukungan untuk Safari dalam beberapa minggu. Mungkin saja Edge tidak akan ditinggalkan.

Selain itu, salah satu pelacak tertua dan paling populer, The Pirate Bay yang tidak dapat tenggelam, juga dengan cepat mengatur dukungan plug-in di mesin pencari.

Latar belakang


Popcorn Time awalnya muncul sebagai klien untuk mengunduh file torrent dengan pemutar media bawaan. Karena paling sering pengguna tertarik pada torrent dengan file video (film dan episode seri), cukup logis untuk mengharapkan kemunculan perangkat lunak tersebut.

Di uTorrent yang sama populer, dukungan untuk mengeluarkan aliran video dalam proses mengunduh torrent diimplementasikan untuk beberapa waktu - tetapi untuk melihatnya, instalasi dan konfigurasi pemutar media yang terpisah diperlukan. Di sini, semua yang dibutuhkan disediakan "di luar kotak".

Tentu saja, hal yang mudah seperti itu tidak dapat gagal untuk menarik perhatian pemegang hak cipta, yang menerima lebih dari satu mimpi buruk, mencoba menghilangkan ancaman ini terhadap pendapatan mereka. Dikenal luas karena kekejamannya, geng pemegang hak cipta MPAA (American Film Association) pada tahun 2014 berhasil menutup versi asli dari layanan ini.

Tetapi karena kode sumber dari program ini tersedia untuk semua orang, versi-versi layanan secara berkala muncul berulang-ulang, dengan cara para kepala mitos hydra. Layanan populer terakhir, Popcorntime.io, ditutup pada November 2015.

Pengembang salah satu opsi "popcorn" dari domain Popcorn-Time.se telah tumbuh dan mengembangkan aplikasi sehingga mendukung Windows, OS X, Linux, Android, iOS, dukungan VPN bawaan, dan bahkan Chromecast dan Apple TV.

Plugin: diaktifkan dan lihat


Tetapi sekarang pemegang hak cipta dihadapkan dengan mimpi buruk yang membuat mereka tidak bisa bangun. Popcorn-Time.se tidak lagi menawarkan untuk mengunduh dan menginstal aplikasi - cukup instal plug-in untuk browser Torrents Time, dan Anda dapat menonton file video apa pun dari torrent langsung di halaman.

Pada saat yang sama, kode plugin Torrents Time tersedia di GitHub , yang berarti bahwa siapa pun dapat membantu menyelesaikan sistem dan menginstalnya di hosting mereka.

Plugin ini tersedia untuk Internet Explorer, Chrome dan Firefox untuk OS Windows 7 (dan di atas) dan OS X 10.8 (dan di atas). Ini mendukung Chromecast, Airplay dan DLNA dan memiliki dukungan VPN bawaan. Pengembang menawarkan layanan berbayar VPN Anonim , tetapi tidak ada yang mengganggu untuk menggunakan yang lain.

Teluk bajak laut


Biasanya, sungai mengalir ke laut, tetapi dengan memasuki teluk bajak laut Teluk Bajak Laut, Anda dapat menemukan aliran yang mengalir dari sana. Dan sekarang akses ke aliran ini telah disederhanakan menjadi tidak mungkin - administrator situs telah menambahkan dukungan untuk plugin Torrents Time.



Sekarang, di sebelah tautan untuk menerima magnet, tautan "Stream It!" Telah muncul, dengan mengklik di mana Anda dapat mengunduh plug-in, dan meluncurkannya dan mulai menonton film.

Pemain yang sederhana namun berfungsi penuh mendukung subtitle dan casting untuk perangkat Chromecast, Airplay, dan DNLA.



Situs hebat lainnya, seperti Torrentproject.se, Videomax.is, Online.porntime.ws dan KickassTorrents, mengumumkan bahwa segera mereka juga akan memungkinkan untuk menggunakan plugin ini di halaman mereka.

Untuk mengucapkan terima kasih
