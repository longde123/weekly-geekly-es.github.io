Warum sind Programmierer laut Statistiken von Yandex und StackOverfow C # die billigsten?

Im Allgemeinen wurden kürzlich Statistiken zu Yandex-Gehältern in der IT in Russland hier veröffentlicht, und auf StackOverfow können Sie auch Statistiken zu Gehältern in den USA einsehen, und hier ist eine traurige Tatsache zu sehen. Gemessen an den Statistiken von C # aka .NET-Entwicklern ist das Gehalt viel niedriger als das von Java und niedriger als das von Python, JS und vielen anderen. Warum schätzt uns das Geschäft weniger als die anderen? Natürlich verstehe ich, dass es eine Lüge, eine offensichtliche Lüge und Statistiken gibt. Ja, ich selbst verdiene mehr als diese Statistiken. Diese Figuren fielen jedoch nicht vom Himmel. Daher wollte ich eine Umfrage zu diesem Thema erstellen.

Meine persönliche Meinung ist, dass C # mehr als notwendig ist, da in dem Bereich, in dem es hauptsächlich verwendet wird, Java benötigt wird. Darüber hinaus ist es möglich, dass eine große Schicht von C # an der Spieleentwicklung beteiligt ist, bei der die Gehälter im Allgemeinen niedriger sind als in anderen Bereichen. Willkommen. Ihre Meinung zu diesem Thema ist mir wichtig. Im Ernst. Im Allgemeinen ist die C # -Sprache (für meinen Geschmack natürlich) ziemlich gut. Und ich kenne viele Sprachen. Ja, meiner Meinung nach werden Haskell und Rust besser sein. Was ist das, F # Geschwister auch. Aber vor dem Hintergrund von Java, C ++, PHP und JavaScript und noch mehr, Gott vergib mir, geh, IMHO ist es wunderschön. Die .NET-Plattform ist mehr als ausgereift. Es gibt genug Frameworks und Bibliotheken für die Augen und für alles. + Es gibt immer noch ein paar coole proprietäre. Dies ist nicht Rust, wo, wenn Sie etwas Bestimmtes benötigen, es höchstwahrscheinlich nicht existiert oder eine Art Bibliothek in tiefem Alpha grau ist. Ja, auch für etwas Gewöhnliches. Vielleicht ist die Tatsache, dass ASP.NET in den letzten Jahren nicht sehr dauerhaft war. Hier ist die Hoffnung, dass mit der .net Core 3.0-Version alles geregelt ist. Ich mag zwar nur diese häufigen Änderungen, aber das Geschäft ist eher etwas Dauerhaftes und Unveränderliches
