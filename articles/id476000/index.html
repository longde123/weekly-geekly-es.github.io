Pembaruan Pembuat Model ML.NET

ML.NET adalah kerangka kerja pembelajaran mesin lintas platform untuk pengembang .NET. Dan Model Builder adalah alat antarmuka pengguna visual studio yang menggunakan Automated Machine Learning (AutoML) untuk dengan mudah melatih dan menggunakan model kustom ML.NET. Dengan ML.NET dan Model Builder, Anda dapat membuat model pembelajaran mesin Anda sendiri untuk skenario seperti analisis sentimen, prediksi harga, dll. Dan semua ini tanpa pengalaman dalam pembelajaran mesin.

Pembuat Model ML.NET


Rilis Model Builder ini berisi perbaikan bug dan dua fitur keren baru:

  • Skenario Klasifikasi Gambar - Latih model klasifikasi gambar secara lokal dengan gambar Anda
  • Coba model Anda - buat prediksi pada sampel data input langsung di UI



Skenario Klasifikasi Gambar


Kami mendemonstrasikan fitur ini di .NET Conf. Dengan menggunakannya, Anda dapat menentukan cuaca pada gambar sebagai cerah, berawan atau hujan, dan sekarang Anda dapat melatih model klasifikasi gambar secara lokal di Model Builder menggunakan gambar Anda sendiri.



Sebagai contoh, katakanlah Anda memiliki dataset gambar anjing dan kucing, dan Anda ingin menggunakan gambar ini untuk melatih model ML.NET, yang mengklasifikasikan gambar baru sebagai "anjing" atau "kucing".

Kumpulan data Anda harus berisi folder induk dengan subfolder bertanda untuk setiap kategori (misalnya, folder bernama Hewan, yang berisi dua subfolder: satu bernama Anjing, yang berisi gambar anjing, dan satu bernama Kucing, yang berisi gambar kucing):



Anda dapat menggunakan Langkah Berikutnya dan proyek-proyek yang dihasilkan oleh Model Builder untuk dengan mudah menerapkan model klasifikasi gambar terlatih dalam aplikasi Anda ke pengguna akhir, serta dalam skrip teks.

Coba model Anda


Setelah melatih model dalam Model Builder, Anda dapat menggunakan model untuk memperkirakan langsung di antarmuka pengguna untuk skenario teks dan gambar.
