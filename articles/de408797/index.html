Eine kurze Geschichte des Blue Screen of Death



Dies ist eine Kult-Sache, obwohl Microsoft es nicht wollte. In den 90er Jahren war es der gleiche Teil der grundlegenden Erfahrung mit der Verwendung von Windows als Paint und Solitaire, aber heute wird es nicht mehr so ​​oft gesehen.

Natürlich spreche ich vom " Blue Screen of Death ", BSoD. Jüngere PC-Benutzer haben keine Ahnung, wie weit verbreitet dieser panikausbreitende Bildschirm war oder was er bedeutete. Alles, woran Sie gearbeitet haben, ging verloren, Ihr Computer musste neu gestartet werden - dann konnte es zehn Minuten dauern.

Diejenigen von uns, die sich an ihn erinnern, versuchen ihn zu vergessen - aber das ist nicht einfach.

Bis heute ist BSoD ein erkennbares Symbol für etwas, das nicht funktioniert, aber warum ist es überhaupt aufgetaucht? Vor Ihnen ist eine kurze Reise entlang eines schlecht unterscheidbaren Teils der Straße der Erinnerungen, vor der Ihre Eltern Sie gewarnt haben.

Windows 3.1: Bildschirm Strg + Alt + Löschen




Windows 3.1 hatte kein BSoD: Als es abstürzte, sahen Sie einen schwarzen Bildschirm. Wenn Sie Glück hatten, wurde auf diesem Bildschirm das Eingabefeld für DOS-Befehle angezeigt, über das Sie Windows erneut starten konnten. Wenn nicht, ist die Neustartzeit gekommen.

Es wurde jedoch ein blauer Bildschirm angezeigt, der durch Drücken von Strg + Alt + Entf aufgerufen wurde. In Zukunft wird er das Design von BSoD inspirieren.

Interessanterweise wurde der Text für den Bluescreen, wie in einem Blogbeitrag von Raymond Chen aufgezeichnet, von keinem anderen als dem zukünftigen Direktor des Unternehmens, Steve Ballmer, geschrieben, als er die Microsoft-Systemabteilung leitete.

Windows 95 und 98: der erste blaue Bildschirm des Todes


Es ist schwer, die Bedeutung von Windows 95 zu überschätzen: Stellen Sie sich den Hype um die frühen iPhone-Modelle vor, aber nur um das Desktop-Betriebssystem. Die Leute stellten sich buchstäblich in den Läden auf. Ja, die 90er Jahre waren seltsam: Die Leute waren begeistert von den neuen Funktionen von Desktop-Computern. Aber niemand war von diesem Bildschirm begeistert.



BSoD wurde jedes Mal angezeigt, wenn ein Programm oder ein Treiber abstürzte. Er bot verschlüsselte Informationen über die Ursachen des Problems an und gab dann etwas klarere Ratschläge, was als nächstes zu tun sei.

Theoretisch sollte das Drücken einer beliebigen Taste dieses Programm schließen und Sie zum Windows-Desktop zurückkehren. In der Praxis hat dies jedoch selten funktioniert. Laut Raymond Chen hieß es in früheren Versionen dieser Nachricht: „Es ist möglicherweise möglich, im normalen Modus weiterzuarbeiten.“ Diese Zeile wurde jedoch aufgrund übermäßigen Optimismus gelöscht.

Windows 2000: Vielleicht auf Viren überprüft?




Mit der Veröffentlichung von Windows 2000 hat Microsoft seine Blue-Screen-Tipps erweitert. Alle Verweise auf die Möglichkeit, zum Desktop zurückzukehren, verschwanden, und dem Benutzer wurde sofort empfohlen, den Computer auszuschalten. Es gab auch eine Reihe von Ideen für die Nachverfolgung von Fehlern für den Fall, dass dieses Problem erneut auftrat, von der Suche nach Viren bis zur Überprüfung der Festplatte auf Integrität.

Windows XP, Vista und 7: Weitere Tipps




Windows XP setzte den Trend fort, die Anzahl der Tipps auf einem blauen Bildschirm zu erhöhen. Informationen darüber, welches der Programme das Problem verursachte, waren noch vage, aber es gab zumindest einige Codes heraus, die bei Google gesucht werden konnten. Der Rest des Bildschirms war mit allerlei Tipps gefüllt. Dem Benutzer wurde weiterhin empfohlen, den Computer auszuschalten. Es wurde ihm jedoch empfohlen, zu überprüfen, ob alle Programme korrekt installiert waren, und er gab eine Reihe von Ideen, um die Gründe zu ermitteln.

In Vista änderte sich der blaue Bildschirm nicht wesentlich, wurde jedoch häufiger angezeigt. Windows 7 reduzierte die Anzahl seiner Erscheinungsbilder, änderte jedoch nicht das Erscheinungsbild.

Windows 8: trauriges Emoticon ohne Details




Windows 8 hat BSoD komplett geändert. Der Textbildschirm, ähnlich dem Terminal, verschwand, er wurde durch moderne Systemschriftarten und einen riesigen traurigen Smiley in ASCII ersetzt. Das Interessanteste ist, dass fast alle Informationen über die Ursachen des Problems verschwunden sind, ebenso wie Tipps zur Lösung.

Dies ist kein so ernstes Problem wie zuvor, da Bluescreens heutzutage viel seltener sind. Sie können die Ursache des PC-Absturzes herausfinden, indem Sie die Protokolle überprüfen oder Programme von Drittanbietern verwenden, die solche Informationen sammeln.

Unter Windows 10 sieht der Bildschirm gleich aus.

Blue Screen Vermächtnis


Microsoft möchte, dass die Dinge anders werden, aber bis heute bleibt der blaue Bildschirm des Todes ein Windows-Symbol. Diese Tatsache inspirierte eines der größten Bürospiele aller Zeiten - den BSoD-Bildschirmschoner. Es wurde von Sysinternals (das Microsoft später gekauft hat) hergestellt und lässt jeden Computer so aussehen, als wäre er eingefroren, bis Sie eine Taste drücken oder mit der Maus zucken. Sehr lustig.
