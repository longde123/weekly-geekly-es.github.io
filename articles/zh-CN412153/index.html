爱因斯坦最著名方程的三个含义


爱因斯坦在观众面前展示STO。 1934年。

在物理学上数百年中，有一个不变的定律从未被怀疑：在宇宙中发生的任何反应中，质量都得以保留。 不论使用哪种成分，发生了什么反应，发生了什么都无关紧要-开始时的总和与结束时的总和相等。 但是根据狭义相对论的定律，质量不可能是一个守恒的量，因为各种观察者对系统具有多少能量持不同意见。 相反，在最简单，最强大的方程式之一的驱动下，爱因斯坦能够推导出我们今天仍然使用的定律：E = mc 2 。


1967年准备进行试验的核火箭。该火箭的工作是将质量转换为能量E = mc 2 。

最著名的爱因斯坦方程式只有三个部分：

  1. E或能量，完全占据等式的一部分，并代表系统的总能量。
  2. m ，通过转换因子与能量相关的质量。
  3. c 2 ，光速的平方是确保质量和能量相等的必要因素。


尼尔·玻尔（Niels Bohr）和阿尔伯特·爱因斯坦（Albert Einstein）在1925年保罗·埃伦费斯特（Paul Ehrenfest）的家中讨论了许多话题。玻尔与爱因斯坦的辩论是量子力学发展过程中最有影响力的因素。 如今，玻尔以其对量子物理学的贡献而闻名，爱因斯坦因其对相对论以及能量与质量等价物的贡献而闻名。

这个方程式的含义已经改变了整个世界。 正如爱因斯坦本人所说：
从相对论的特殊理论出发，质量和能量是一件事的不同表现形式。 这个概念对普通人来说是陌生的。
这是这个简单方程式的三个最重要的结果。


标准模型的夸克，反夸克和胶子除了具有质量和电荷之类的所有其他属性外，还具有色电荷。 只有胶子和光子没有质量。 所有其他物体，甚至中微子，其静止质量都不为零。

甚至静止的群众也有其固有的能量 。 在学校里，您研究了所有类型的能量-机械，化学，电，动能。 所有这些类型的能量都是物体在移动或反应时固有的，这些形式的能量可用于执行工作，例如，启动发动机，点燃灯泡或将谷物磨成面粉。 但是，即使是普通的休息质量也具有其固有的能量：而且数量巨大。 这导致了一个惊人的结果：牛顿宇宙中任意两个质量之间产生的引力也必须根据等式E = mc 2在等于质量的能量的基础上起作用。


由物质产生对粒子/由纯能量产生反物质（左）是完全可逆的反应（右），因为物质和反物质会消灭，从而产生纯能量。 这种创建和an灭的过程遵循方程E = mc 2 ，并且是创建和销毁物质和反物质的唯一已知方法。

物质可以转化为清洁能源 。 这是方程式的第二个值，E = mc 2精确地告诉我们通过转换质量可以获取多少能量。 每转换一千克质量的能量，您将获得9×10 16 J的能量，相当于21兆吨的TNT。 观察到原子核的放射性衰变，裂变或聚变反应，可以看出所得的质量小于初始质量。 质量守恒定律不起作用。 但是差等于释放的能量！ 从铀和原子弹的衰变到太阳核中的核聚变以及物质/反物质粒子的ni灭，这种方法适用于所有情况。 被破坏的质量转换为能量，其能量由公式E = mc 2计算 。


2014年大型强子对撞机中高能碰撞产生的微粒痕迹。复合粒子衰减成在空间中散射的成分，但由于碰撞中的可用能量，新粒子也会出现。

能量可以用来从几乎一无所有的物质中产生质量-仅仅是从纯能量中产生物质 。 公式的最后一个值是最杰出的。 如果拿两个台球并用力将它们推在一起，则输出将是两个台球。 如果您带上一个光子和一个电子，并将它们推到一起，就会得到一个光子和一个电子。 但是，如果以足够高的能量碰撞它们，则会得到光子，电子和一对新的物质/反物质粒子。 换句话说，您可以创建两个新的大型粒子：

  • 物质的粒子，例如电子，质子，中子等，
  • 反物质的粒子，例如正电子，反质子，反中子等。

只有在实验中投入足够的能量时才会显示。 这样的加速器（例如CERN的LHC）正在寻找新的不稳定的高能​​粒子（例如希格斯玻色子或上夸克）：从纯能量中产生新粒子。 产生的质量来自可用能量：m = E / c 2 。 这也意味着由于海森堡不确定性原理，粒子的寿命受到限制，其特征在于质量不确定性，因为δEδt〜ℏ，因此对应的δm遵循爱因斯坦方程。 当物理学家谈论粒子的宽度时，他们表示这种内部质量不确定性。


广义相对论世界中引力质量引起的时空曲率

能量和质量的等价关系也使爱因斯坦获得了相对论这样的伟大成就。 想象一下，您具有相同的静止质量的物质粒子和反物质粒子。 可以将它们消灭，并将完全按照公式E = mc 2变成具有一定能量的光子。 现在想象一下，这对粒子/反粒子运动很快，好像是从深空坠落到我们身上，然后在地球表面消失了。 这些光子将具有额外的能量-不仅来自E = mc 2的 E，还具有额外的E，即由于下降而获得的动能。


如果来自物质和反物质的两个物体在静止时被歼灭，它们将变成具有非常确定能量的光子。 如果这些光子落入重力场后出现，它们的能量将会更高。 这意味着必须存在一个由牛顿的重力无法预测的引力红色或蓝色位移-否则能量将不会守恒。

如果要节约能量，那么重力红色（和蓝色）位移必须是实数。 牛顿的引力无法解释这种影响，但是在爱因斯坦广义相对论中，空间的曲率意味着落入引力场会为您增加能量，而离开引力场则会使您浪费能量。 事实证明，任何运动物体的完全和一般关系不是E = mc 2 ，而是E 2 = m 2 c 4 + p 2 c 2 （其中p是动量）。 只有总结描述中的所有信息，包括能量，动量和重力，您才能真正描述宇宙。


当辐射量子离开引力场时，由于能量守恒，其频率会发生红移。 当他跌入田野时，他必须经历一次蓝变。 这只有在重力不仅与质量相关联而且与能量相关联的情况下才有意义。

爱因斯坦的最大方程E = mc 2 ，是对基本物理学的力量和简单性的一次胜利。 物质具有其固有的能量，可以将质量（在某些条件下）转换为纯能量，并且可以使用能量来创建以前不存在的大型物体。 这种反射方法使我们有机会发现构成我们宇宙的基本粒子，发明核能和核武器，并发现描述宇宙中所有物体相互作用的重力理论。 找到该方程式的关键是基于一个简单假设的适度思想实验：能量和动量守恒。 其余的是宇宙工作计划的必然结果。
