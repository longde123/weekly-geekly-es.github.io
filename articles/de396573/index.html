So aktualisieren Sie Windows 7 und 8.1 nach dem 29.07.2016 kostenlos auf Windows 10

Wie Sie wissen, gab Microsoft zunächst an, dass Windows 10, das am 29. Juli 2015 veröffentlicht wurde, ein Jahr lang - bis zum 29. Juli 2016 - ein kostenloses Update für Einzelhandels- und OEM-Installationen von Windows 7 und Windows 8.1 sein wird. Nach dem 29. Juli 2016 wird kein Update über Windows Update mehr angeboten, und Sie müssen es kaufen. Windows Update unter Windows 7 und Windows 8.1 schlug regelmäßig ein Upgrade auf Windows 10 vor.


Tatsächlich wurde ab dem 19.07.2016 auf allen aktivierten Systemen mit installierten Einzelhandelseditionen von Windows 7 / 8.1, auf denen Benutzer regelmäßig die empfohlenen Microsoft-Updates installierten, ein Zähler mit einem Countdown der verbleibenden Zeit für ein kostenloses Update aktiviert. Dies spornte natürlich viele an, die das Update in den letzten Tagen verschoben haben. Und wie versprochen verschwanden ab dem 29. Juli 2016 Angebote zur kostenlosen Aktualisierung des Betriebssystems über Windows Update. Haben diejenigen, die keine Zeit für ein Upgrade hatten, keine Möglichkeit mehr, ein neues Betriebssystem kostenlos zu erhalten? Zum Glück nicht! Die Möglichkeit eines kostenlosen Betriebssystem-Updates blieb bestehen. Mal sehen, wie Sie jetzt Windows 7 und Windows 8.1 auf Windows 10 aktualisieren können.

Ab dem 29. Juli 2016 bietet Microsoft weiterhin ein kostenloses Windows 10-Update für Benutzer mit Behinderungen an. Microsoft lädt Sie ein, selbst zu entscheiden, ob dieses Angebot für Sie gilt oder nicht. Lassen Sie diese schmerzhafte moralische Entscheidung für Sie und verstehen Sie die technische Seite des Problems.
Zu Beginn habe ich die Seite dieser Aktion aufgerufen und den Update-Assistenten heruntergeladen .


Als Beispiel für diesen Hinweis habe ich Windows 8.1 Professional in einer virtuellen Maschine installiert, vollständig aktualisiert und aktiviert. Ich habe die Methode unter Windows 7 überprüft. Das Ergebnis und der Prozess unterscheiden sich nicht. Daher werde ich Ihnen im Folgenden das Beispiel für die Aktualisierung von Windows 8.1 erläutern.


Ich starte den heruntergeladenen Update-Assistenten für mein Betriebssystem.


Ich akzeptiere die Vereinbarung und für kurze Zeit überprüft das Betriebssystem die Aktualisierungsmöglichkeiten meines PCs. Die Hauptanforderung sind 2 GB Speicher (für 64-Bit-Betriebssysteme, 1 GB für 32-Bit) und etwa 20 GB (16 GB für 32-Bit) auf der Systempartition der Festplatte.


Die Anwendung lädt sofort das zu Ihnen passende Image der Windows 10 Edition aus dem Internet herunter. Zum Zeitpunkt dieses Schreibens wurde die Version des Threshold2-Zweigs geladen, aber ich hoffe wirklich, dass in ein oder zwei Wochen derselbe Update-Assistent den Windows 10 Anniversary Update-Zweig von Redstone1 sofort installiert.


Nach einigen Minuten (Stunden? Tage ??) des Startvorgangs überprüft das System die Integrität des heruntergeladenen Images.


Und der Update-Prozess beginnt.


In den ersten 30-40 Minuten geschieht alles in einem laufenden Betriebssystem. Du kannst dein eigenes Ding machen. Als nächstes warnt Sie das System, dass ein Neustart erforderlich ist. Sie haben eine halbe Stunde Zeit, um dies zu bemerken.


Nach dem Neustart beginnt der Standardinstallationsprozess für Windows 10:


Abhängig von der Geschwindigkeit der Festplatte dauert der Vorgang 10 bis 40 Minuten und endet mit einer Meldung über den erfolgreichen Abschluss des Updates.


Wir wählen die Betriebssystemeinstellungen aus (deaktivieren Sie alles, was Microsoft zum Aktivieren anbietet), gehen zum Betriebssystem und sehen, dass unser Fenster 10 aktiviert ist!


Wie oben erläutert, gehört die Betriebssystemversion, die ich beim Schreiben dieser Notiz erhalten habe, zum Zweig Threshold2 (besser bekannt als Version 1511). Alle aktuellen kumulativen Betriebssystemupdates sind enthalten. Details zu Threshold, Redstone und allgemein zu Assemblys von Windows 10 finden Sie in meinem Artikel "Alles über Versionen von Windows 10" - ich empfehle.


Jetzt muss ich nur noch warten, bis über Windows Update das Windows 10 Anniversary Update - Version 1607 des Redstone1-Zweigs verfügbar ist. Für diejenigen, die dieses Update sofort abschließen möchten, kann ich Ihnen im Bereich Einstellungen / Updates empfehlen, erweiterte Einstellungen auszuwählen und den Empfang von Insider Preview-Assemblys zu aktivieren:

In den Update-Einstellungen gebe ich an, dass ich Assemblys des RS1_Release-Zweigs und der Option Early Access empfangen möchte. Die endgültige Version wurde bereits veröffentlicht und ist verfügbar. Vorläufige Versionen werden jetzt nur noch im RS_Prerelease-Zweig für Redstone2 verfügbar sein.


Build 14393 - die endgültige Version des Windows 10 Anniversary Updates - wird Ihnen über Windows Update zur Verfügung gestellt. Richtig, nicht sofort, sondern tagsüber. So funktioniert Insider Preview.


Nach etwa einer halben Stunde Update erhalten Sie die endgültige Version des Windows 10 Anniversary Update - 14393-Zweigs von Redstone1. Auf Wunsch können Sie jetzt den Empfang von Insider-Assemblys deaktivieren.


Für diejenigen, die nicht gerne ein Betriebssystem über einem anderen installieren und solche Installationen nicht als sehr stabil betrachten. Windows 10, das über den Update-Assistenten oder bis zum 29. Juli über Windows Update aktualisiert wurde, ist nach der Installation nicht nur ein aktiviertes Betriebssystem, sondern speichert auch Informationen darüber auf Microsoft-Servern. Was bedeutet das? Wenn Sie das Betriebssystem jetzt vollständig auf demselben PC neu installieren, obwohl Sie die Systemfestplatte formatiert haben, geben Sie die Seriennummer während der Installation nicht ein - es gibt eine unauffällige Schaltfläche "Später ausführen". Wählen Sie "Später ausführen". Das Betriebssystem selbst kontaktiert die Aktivierungsserver. Stellen Sie sicher, dass Windows 10 zuvor auf diesem bestimmten PC aktiviert wurde, und aktivieren Sie das neu installierte Betriebssystem.
