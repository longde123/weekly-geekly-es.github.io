Windows 95 auf Nintendo 3DS starten? Ja, das ist möglich



Wenn Sie zu der Zeit zurückkehren möchten, als MS Paint und Internet Explorer 1.0 auf PCs herrschten, können Sie keine Zeitmaschine erfinden. Es reicht aus, nur Windows 95 auf Ihrer tragbaren Nintendo 3DS-Konsole zu installieren. In einem der Beiträge im Game Boy-Forum veröffentlichte der „Retronaut“ Screenshots und Videos, die die Funktionsweise des genannten Betriebssystems demonstrieren.

Auf die Frage, warum sie etwas Ähnliches machten, antwortete ein Benutzer mit dem Spitznamen Shutterbug2000, dass er es einfach tun wollte. Zum Ausführen von Windows 95 wurde ein x86-Emulator verwendet.



Zusätzlich zu den Screenshots hat der Nutzer ein Video auf YouTube gepostet. Zeigt den Windows 95-Startbildschirm und Scandisk an. Das Video wird jedoch angehalten, noch bevor der Desktop angezeigt wird.

Der Benutzer hat den Code und den Code, den er für die Ausführung von Windows 95 geschrieben hat, hier gefunden . Andere Benutzer haben bereits bestätigt, dass alles funktioniert, das Projekt ist kein Scherz.

Es gibt natürlich Nachteile. Beispielsweise ist die Leistung des gesamten Systems schlecht. Es ist jetzt auch schwer zu verstehen, wie Sie lernen können, den Cursor mithilfe des Konsolenanzeigesensors zu steuern.
