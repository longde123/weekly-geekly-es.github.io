Windows在台式机上的份额下降到90％以下


Windows 7仍然是领导者，但在

Windows 10发布后的9个月中并不是绝大多数，Windows 7的份额最终跌至50％以下，Windows XP达到了明确指标的极限。尽管这对微软来说是个好消息，但四月份并不是Windows的最佳月份，根据最新的Net Applications数据，该月份首次占据了不到90％的市场。

让我们分析这些数字。首先，您需要考虑微软希望在两到三年内获得10亿台运行Windows 10的设备的愿望。venturebeat.com报道，在最初的四周内，Windows 10安装在超过7500万台计算机上，在10周内安装了1.1亿台设备，在六个月内安装了2亿台，在八个月内安装了2.7亿台。

这项收购已为Microsoft带来了显着的增长，但是这种增长正在放缓。 Windows 10的市场份额在三月份为14.15％，在四月份仅为0.20％，达到14.35％。


Net Applications Web Analytics

Windows 8以某种方式增长了0.50个百分点，达到2.95％，而Windows 8.1增长了0.40个百分点，达到9.16％。到4月底，它们在一起占领了市场的12.11％。预计到2016年底，这对二人组的份额将降至10％以下。

在Windows 10操作系统发布之前，Windows 7于2015年6月占据了60％的市场份额，从2016年初开始下降至52.47％，目前为48.27％。

尽管不可避免地会升级到Windows 10，但Windows 7可能会长期保留“最受欢迎的操作系统”的称号。根据此指标，Windows 7在2012年9月取代Windows XP。

Windows Vista占1.34％，而Windows XP下降1.24点至9.66％。考虑到Microsoft早在2014年4月就不再支持“古老的”操作系统，XP的份额下降一直等待了很长时间。

总体而言，截至2016年4月，Windows操作系统系列的份额下降了1.22点，至89.23％。同时，Mac OS X的份额增加了1.43点，达到9.20％。 Linux的份额为1.56％。


全球操作系统市场，2016年4月数据

至少根据Net Applications的数据，Windows从未跌破90％的关口（2007年11月的数据，当时Windows占据了市场的95.89％）。尽管Windows多年来一直在缓慢地失去其份额，但值得注意的是，下降点从未超过1％。Windows有望在本月“恢复”其90％的性能。

Net Applications每月使用1.6亿唯一身份访问者的数据，监视范围涵盖约4万个网站。
