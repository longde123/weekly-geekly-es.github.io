Googelte Chrom

Auch wenn Sie kein Google-Konto haben, kommuniziert der kostenlose Chromium-Browser im Hintergrund mit Google-Servern. Dies ist ziemlich seltsam, da die Leute Chromium anstelle von Chrome installieren, um ein sauberes Programm ohne kommerzielle Bindung zu erhalten. Wenn Sie Chromium jedoch im normalen Modus erstellen, werden die binären Blobs von Google weiterhin heruntergeladen und installiert.

Das Ungoogled-Chrom-Projekt besteht aus einer Reihe von Konfigurationsflags, Patches und speziellen Skripten, um die Integration mit Google zu entfernen, die Sicherheits- und Verwaltungseinstellungen zu verbessern.

Ein anonymer Benutzer von Github hat die Einstellungen aus mehreren vorhandenen Projekten ausgeliehen, in denen auch versucht wurde, Chromium aus unnötigen Funktionen heraus zu maximieren, um eine saubere, minimalistische Konfiguration zu erstellen: Dies ist Chromium aus der Debian-Distribution , das Inox- Patch-Kit sowie der Iridium- Browser mit Schwerpunkt auf Datenschutz.

Die Entwickler der drei genannten Projekte haben erhebliche Anstrengungen unternommen, um die Präsenz von Google in einem kostenlosen Browser zu beseitigen. Aber das "ängstliche Chrom" ist in diesem Bestreben noch paranoider.

Beispielsweise hat der Autor viele der im Chromium-Quellcode genannten Webdomänen durch nicht vorhandene Alternativen ersetzt, um sicherzustellen, dass der Browser ohne Wissen des Benutzers keine Informationen senden kann.

Außerdem wurden alle binären Blobs aus dem Quellcode entfernt: Programme, Bibliotheken und andere Formen von Maschinencode. Es gibt nur die Maschinencodes, die für die Montage benötigt werden (z. B. icudtl.datzur Unterstützung von Unicode oder zur *_page_model.binDefinition von Modellen zum Reinigen von Seiten im DOM Distiller).


Beispiel für das Löschen einer Seite im DOM Distiller

In einem "gegoogelten" Browser sind die für Google-Domains spezifischen Funktionen, z. B. Google Host Detector, Google URL Tracker, Google Cloud Messaging, Google Hotwording usw., vollständig deaktiviert.

Der Autor hat ein paar weitere Einstellungen hinzugefügt, die nur Ihrem Geschmack entsprechen. Wahrscheinlich wird es nicht jedem gefallen.

  • Der Suchanbieter "Keine Suche" wurde zur Omnibox hinzugefügt, um die Suche in der Omnibox vollständig zu deaktivieren.
  • Deaktivieren Sie die automatische Formatierung der URL in der Omnibox, wenn sie nicht mehr angezeigt http://wird.
  • Deaktivieren Sie JavaScript-Dialoge beim Schließen einer Seite (Ereignisse onbeforeunload)

Darüber hinaus hat der Autor einige Standardeinstellungen der Iridium- und Inox-Browser geändert. Beispielsweise wird das obligatorische Pingen von IPv6-Adressen verboten, wenn die IPv6-Verfügbarkeit erkannt wird.

Im Allgemeinen können Sie Chromium auf verschiedene Arten „googeln“. Neben Iridium und Posteingang wurde jetzt eine weitere Option angezeigt.

Anscheinend haben viele Menschen den Wunsch, einen einfachen, minimalistischen Browser ohne Schnickschnack und intellektuelle Funktionen schnell und zuverlässig zu bekommen. Es ist fast unmöglich, einen solchen Browser von Grund auf neu zu erstellen, aber es ist eine gute Idee, Chromium von Schlacke zu reinigen.

Das einzige Problem ist, dass sich Browser sehr schnell entwickeln. Neue Versionen werden mehrmals im Jahr veröffentlicht. Im selben Chromium ist die stabile Version jetzt 53.0.2785.116, aber Chrome 1 und Chromium 1 wurden kürzlich veröffentlicht - am 2. September 2008.

Dies ist nicht nur die Liebe der Entwickler zu Zahlen. Jede neue Version führt tatsächlich neue Webstandards, neue HTML5-Funktionen und Unterstützung für neue Technologien ein. Der Prozess geht schnell. Aufgrund einer solchen "Fluidität" ist es sehr schwierig, den neuen Code im Auge zu behalten.

Der Autor des "verängstigten" Chromiums warnt ehrlich, dass aufgrund der rasanten Entwicklung des Chromium-Projekts ständig neue Funktionen, die die Privatsphäre verletzen, in den Code eindringen können, sowie neue Fehler.

Aus irgendeinem Grund versucht der Autor, Versionen parallel zum Hauptzweig von Chromium zu veröffentlichen. Aber was für eine interessante Idee wäre es, einen ultra-minimalistischen Browser zu erstellen, für den überhaupt keine Updates benötigt werden. Deaktivieren Sie Canvas, WebGL, WebRTC und andere moderne Dinge, deaktivieren Sie Historien, Cookies, den Browserzugriff auf Geräte und vieles mehr. Erstellen Sie eine Version von Tor für sicheres Surfen mit minimaler Funktionalität, und lassen Sie nur einen Werbeblocker und einen Seitenreiniger wie den oben genannten DOM Distiller übrig. Zumindest für mobile Geräte wäre ein derart minimalistischer Browser praktisch.

Auf dieser Seite werden die neuesten Versionen des bereinigten Chromium für Windows, MacOS, Debian und Ubuntu veröffentlicht .. Build on Windows läuft nur unter Windows 7 x64 oder höher. Zum Erstellen benötigen Sie Visual Studio 2013 Update 4 oder Visual Studio 2015 Update 1, andere Versionen werden nicht unterstützt.

Sie können Erweiterungen aus dem Chrome Webstore im Browser installieren, jedoch nicht über die Weboberfläche. Sie müssen manuell heruntergeladen und installiert werden. Um die Erweiterung aus dem Chrome Webstore herunterzuladen, können Sie den direkten Link verwenden:

https://clients2.google.com/service/update2/crx?response=redirect&prodversion=48.0&x=id%3D[EXTENSION_ID]%26installsource%3Dondemand%26uc

Hier [EXTENSION_ID]ersetzen wir nur die Erweiterungskennung aus dem Verzeichnis. Zum Beispiel die cjpalhdlnbpafiamejdnhcphjbkeiagmKennung für die Erweiterung uBlock Origin.

Danach wird die crx-Datei auf verschiedene Arten als Erweiterung im Browser installiert. Sie können entweder durch einfaches Ziehen der Maus auf der Registerkarte mit den Erweiterungen auswählen.
