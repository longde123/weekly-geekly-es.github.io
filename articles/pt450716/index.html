Anunciando as habilidades do Windows Vision (versão prévia)

Alguns dias atrás, anunciamos a pré-visualização do Windows Vision Skills , um conjunto de pacotes NuGet que facilitam aos desenvolvedores de aplicativos a solução de problemas complexos de visão computacional usando um simples conjunto de APIs.


Figura 1- Da esquerda para a direita, você está vendo em ação as habilidades Detector de Objetos, Detector Esquelético e Reconhecedor de Emoções.


Com habilidades de visão do Windows:

  • Um desenvolvedor de aplicativos pode usar APIs WinRT prontas para adicionar habilidades de visão pré-criadas, como detecção de objetos, detecção esquelética e análise de sentimentos em seus aplicativos Windows (.NET, Win32 e UWP).
  • Um desenvolvedor de visão computacional pode alavancar estruturas de aceleração de hardware em dispositivos Windows, compactando sua solução como uma habilidade de visão - sem se preocupar com APIs de baixo nível.


Todas as habilidades do Windows Vision herdam as classes base e as interfaces no Microsoft.AI.Skills.SkillInterfacePreview . Essa estrutura aberta pode ser facilmente estendida para trabalhar com estruturas e bibliotecas de aprendizado de máquina existentes, como o OpenCV.


O Windows Vision Skills complementa o suporte existente do Windows à inferência de modelos ONNX, utilizando o WinML para inferências locais. A estrutura permite criar aplicativos inteligentes enquanto aproveita a otimização da plataforma.
