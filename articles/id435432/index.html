Tahun Baru, GitHub Baru: Repositori Pribadi Gratis Tanpa Batas

Hari ini kami mengumumkan dua inovasi penting pada GitHub yang akan membuatnya lebih mudah diakses oleh pengembang: repositori pribadi gratis tanpa batas dan produk yang lebih nyaman bagi perusahaan. Detail di bawah potongan!



  • GitHub Gratis sekarang termasuk repositori pribadi tanpa batas. Untuk pertama kalinya, pengembang dapat menggunakan GitHub untuk proyek mereka, menambahkan hingga tiga kolaborator ke repositori secara gratis. Banyak pengembang ingin menggunakan repositori pribadi untuk bekerja pada proyek pihak ketiga atau mencoba sesuatu secara pribadi sebelum menerbitkannya secara publik. Mulai hari ini, ini dan banyak skrip lainnya tersedia di GitHub secara gratis. Repositori publik tetap gratis dan menyertakan kolaborator dalam jumlah yang tidak terbatas.
  • GitHub Enterprise adalah produk terpadu yang baru untuk Enterprise Cloud (sebelumnya GitHub Business Cloud) dan Enterprise Server (sebelumnya GitHub Enterprise). Organisasi yang membutuhkan fleksibilitas untuk menggunakan GitHub dalam konfigurasi cloud atau mandiri sekarang dapat mengakses keduanya dengan harga yang sama. Dan terima kasih kepada GitHub Connect, produk-produk ini dapat saling terhubung dengan andal, menyediakan opsi hibrida yang memungkinkan pengembang untuk bekerja secara mulus di lingkungan apa pun.

Pelajari lebih lanjut

GitHub Pro (sebelumnya Pengembang GitHub) dan Tim GitHub juga tersedia untuk pengembang dan tim yang membutuhkan pengkodean profesional dan kemampuan kolaborasi. Dan, tentu saja, proyek open source akan memiliki semua yang Anda butuhkan untuk bekerja bersama dalam repositori publik, termasuk versi gratis dari Tim GitHub kami.

Apakah Anda seorang pelajar yang akan menulis baris kode pertama Anda, atau kepala perusahaan dengan tim di seluruh dunia, kami ingin GitHub menjadi tempat terbaik Anda untuk menulis kode, berkolaborasi dan berkomunikasi dengan komunitas pengembang global. Perubahan hari ini adalah investasi besar di masa depan GitHub, dan kami akan sangat senang untuk semua proyek Anda yang dibuat dan dibuat pada tahun 2019.
