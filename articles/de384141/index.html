Windows im Laufe der Jahre: 30 Jahre das Microsoft-Betriebssystem

Die erste Version der Windows-Grafikshell für MS-DOS wurde vor fast dreißig Jahren veröffentlicht, nur noch wenige Monate bis zum Jubiläum. Nach Jahrzehnten beträgt der Anteil von Windows am PC-Markt mehr als neunzig Prozent. Mehr als die Hälfte aller Desktop-Computer läuft unter Windows 7.

Erinnern wir uns, welche Windows-Versionen wir seit 1985 gesehen haben.



Die erste Version der grafischen Windows-Shell wurde 1985 veröffentlicht: Am 20. November erschien sie in Einzelhandelsketten. Um rechtliche Probleme mit Mac OS zu vermeiden, haben die Entwickler den Warenkorb nicht verwendet und die Fenster konnten sich nicht überlappen. Diese Windows-Version hatte keine Taskleiste am unteren Bildschirmrand. "Nutzloses Add-On über DOS" beinhaltete ein Spiel - Reversi.



Zwei Jahre später, 1987, veröffentlichte Microsoft eine Version von Windows 2.0. Hier erschienen die uns seit unserer Kindheit bekannten Paint-Anwendungen, Uhren, der MS-DOS-Dateimanager und das Terminal. Kurz nach Release 2.0 portierte das Unternehmen Word und Excel zunächst nach Windows.



Windows 3.0 enthielt 1990 den Programm-Manager, Unterstützung für VGA und virtuellen Speicher, ein neues 3D-Design und das Spiel „The Scarf“. Das Spiel wurde nicht zur Unterhaltung der Büroangestellten benötigt, sondern um die Benutzer in den Fähigkeiten des Mausgebrauchs zu schulen .



1992 veröffentlichte Microsoft Windows 3.1 und fügte dem System skalierbare TrueType-Schriftarten und das Minesweeper-Spiel hinzu, um die Genauigkeit des Aufpralls auf den Bildschirm mit dem Cursor zu trainieren. Zum ersten Mal wurde die Unterstützung für Multimedia-Inhalte im System angezeigt.



Windows for Workgroups 3.1 wurde 1992 veröffentlicht. Es beinhaltete native Unterstützung für Netzwerke - insbesondere für LANs lokaler Netzwerke - und gewann im damaligen Geschäft rasch an Popularität.

1993 kann als Geburtsjahr des Windows-Betriebssystems bezeichnet werden. Vor Windows NT 3.1 war Windows eine grafische Shell für MS-DOS. Jetzt hat Microsoft ein 32-Bit-Betriebssystem für moderne Workstations eingeführt. Extern unterschied sich das Betriebssystem nicht von Windows 3.1.



Im August 2015 wurde Windows 95 20 Jahre alt . Computersysteme sind nicht nur für Geeks, sondern auch für Hausfrauen verfügbar geworden. Die Schaltfläche Start und die Taskleiste vereinfachten die Arbeit mit dem System.



Windows NT 4.0 hat eine ähnliche Oberfläche wie Windows 95 mit dem robusten Windows NT-Kernel kombiniert. Dieses System hat sich in den nächsten Jahren zum beliebtesten Betriebssystem von Microsoft entwickelt.



Windows CE ist ein Betriebssystem für Handheld-Computer, Smartphones und eingebettete Systeme. Es wurde im November 1996 veröffentlicht, zu dieser Zeit war es eine "abgespeckte" Version des Desktop-Betriebssystems. Die neueste Version des Systems - 7.0 - wurde im März 2011 veröffentlicht.



Der Internet Explorer-Webbrowser ist zu einem unverzichtbaren Bestandteil von Windows 98 geworden, das 1998 veröffentlicht wurde. Das System verfügt über eine Symbolleiste zum schnellen Starten von Anwendungen und native Unterstützung für USB. Das System ist sehr beliebt geworden, obwohl es nicht das stabilste ist.



Windows 2000 ist Version 5.0 des Windows NT- Betriebssystems. Der Aufstellung folgten XP, Vista, Windows 7 und 8. Die erste öffentliche Beta-Version des Systems erschien 1997, die endgültige Version wurde am 17. Februar 2000 veröffentlicht. Das System unterstützte den Serverteil von Active Directory, das NTFS 3.0-Dateisystem und das verschlüsselte EFS und erhielt die Benutzeroberfläche mit Active Desktop.

Um Windows 2000 verwenden zu können, musste ein Prozessor mit 133 MHz, 32 Megabyte RAM und 2 Gigabyte Speicher auf Ihrer Festplatte vorhanden sein.



Windows ME wurde im Jahr 2000 veröffentlicht und sowohl von Kritikern als auch vom Benutzer als äußerst instabil kritisiert. Es handelte sich um ständige Einfrierungen und Abstürze.



Auf dem ersten Computer, den ich gekauft habe, wurde Windows XP ausgeführt. Ich bekam 2006 einen Computer und Microsoft veröffentlichte 2001 XP. Windows XP wurde zum beliebtesten Microsoft-Betriebssystem seiner Zeit.

Im Juni 2008 kündigte Microsoft an , XP bis 2014 zu unterstützen. Die allgemeine Unterstützung für das System endete 2009 und wurde am 8. April 2014 verlängert.



2007 veröffentlichte Microsoft Windows Vista als Reaktion auf das Mac OS X-Betriebssystem. Durchscheinende Fenster, große Symbole und ein neues Startmenü beeindruckten die Kritiker nicht.



Das Windows Server 2008-Serverbetriebssystem ersetzt Windows Server 2003 als Vertreter der neuen Generation der Vista-Systemfamilie.



Windows 7 ist eine der erfolgreichsten Versionen von Windows. Microsoft hat die schöne Benutzeroberfläche (Aero) und die Symbole verlassen, viele Fehler beseitigt und das System zuverlässiger gemacht.

Sechs Jahre vergingen zwischen der Veröffentlichung von Windows XP und Windows Vista und nur drei Jahre zwischen der Veröffentlichung von Vista und 7.



Im Jahr 2012 führte Microsoft Windows 8 mit der gekachelten Metro-Oberfläche ein. Die Benutzeroberfläche wird zuerst nach dem Starten des Systems angezeigt und ähnelt dem Desktop. Glücklicherweise unterstützte das Betriebssystem weiterhin den klassischen Windows-Desktop, entfernte jedoch die Schaltfläche Start. Das System ist für Touchscreens konzipiert, obwohl Sie es auf Computern mit herkömmlichen Monitoren verwenden können.



Windows 8.1 - Update für Windows 8 von 2014. Jetzt können Benutzer beim Start den Arbeitsbildschirm anstelle von Metro sehen. Die Schaltfläche „Start“ wurde angezeigt, allerdings ohne das alte Startmenü.



Im Jahr 2015 führte Microsoft Windows 10 ein, das von Windows 7 und Windows 8 aktualisiert werden kann. Microsoft gab das Startmenü zurück.

Es gab einige unangenehme Momente für den Endbenutzer: Im August 2015 begannen Torrent-Tracker , Benutzer von Windows 10 auf Computern zu verbieten , da das System eine übermäßige Datenmenge an Microsoft-Server sendet.
