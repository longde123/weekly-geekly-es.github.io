发布Azure IoT Edge工具（预览版）扩展

我们很高兴宣布Visual Studio 2019的新扩展的预览版-Azure IoT Edge Tools的可用性。 该扩展提供了丰富的功能集，以支持使用Visual Studio 2019开发IoT Edge解决方案：



  • 专为各种平台（Linux amd64，Linux arm32v7，Windows amd64）设计的新Azure IoT Edge项目
  • 解决方案中添加了新的IoT Edge模块（C＃/ C）
  • 在Visual Studio计算机上本地编辑，构建和调试IoT Edge模块
  • 创建和下载IoT边缘模块的Docker映像
  • 在本地或远程模拟器中运行IoT Edge模块
  • 在IoT Edge设备上部署IoT解决方案（使用Cloud Explorer ）



要求条件


  • Visual Studio 2019 ：必须安装“ .NET桌面开发”和“ Azure开发工作负载”工作负载； 如果计划开发C模块，则还必须安装“使用C ++进行Windows桌面开发”
  • Docker桌面 。 您需要正确配置以Linux容器或Windows容器模式运行的Docker CE。
  • 若要设置本地开发环境以调试，运行和测试IoT Edge解决方案，您需要Azure IoT EdgeHub开发工具 。 为此，请首先安装Python（2.7 / 3.6） ，然后通过在终端中运行以下命令来安装iotedgehubdev。 确认您的Azure IoT EdgeHub开发工具的版本大于0.8.0。

pip install --upgrade iotedgehubdev

安装方式


有两个安装新扩展的选项：

  1. 从Visual Studio Marketplace下载并安装新的扩展。
  2. 或者，您可以使用扩展->管理扩展菜单直接从Visual Studio 2019安装扩展 。 在“管理扩展”窗口中，在左侧面板中选择“ 在线” ，然后在右上角的搜索字段中输入edge以搜索和下载“ VS 2019的Azure IoT Edge工具[预览]”。
