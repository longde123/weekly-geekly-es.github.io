2. Fortinet Memulai v 6.0. Arsitektur Solusi



Salam! Selamat datang di pelajaran Memulai Fortinet kedua. Jika Anda tidak terbiasa dengan kursus, saya menyarankan Anda untuk melihat pelajaran pertama - ini menjelaskan tujuan utama dan struktur kursus. Pelajaran ini murni teoretis, tetapi mengandung banyak informasi berguna:

  • Tinjauan singkat konsep Kain Keamanan Fortinet dan produk-produknya;
  • Fitur Firewall FortiGate
  • Data sertifikasi FSTEC;
  • Tinjauan umum tentang lini produk FortiGate;
  • Informasi tentang mode administrasi dan pengoperasian perangkat;

Tautan ke video di bawah potongan.


Memikirkan pelajaran ini, saya mengejar beberapa tujuan - pertama, saya ingin mensistematisasikan data pada konsep Fabric Keamanan, sehingga orang-orang yang baru memulai perkenalan mereka dengan Fortinet akan merasa lebih mudah untuk menavigasi portofolio produk perusahaan. Kedua, sebagai bagian dari pelajaran kecil, saya mencoba memperkenalkan siswa pada fungsi utama firewall FortiGate dan berbicara tentang fitur-fiturnya. Apakah ternyata atau tidak, itu terserah Anda :)
