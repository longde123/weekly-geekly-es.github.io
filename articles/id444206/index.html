27 Maret, Moskow - JavaScript MeetUP

Kami mengumpulkan MeetUP JavaScript berikutnya sedini Rabu depan, 27 Maret pukul 19.00 di situs Corporate Innovations Hub (Bolshaya Polyanka St., 2/10, hlm. 1).

Program ini akan memiliki 3 laporan dari karyawan kami. Penerimaan gratis (tetapi mendaftar terlebih dahulu), akan ada siaran online.



Daftar laporan dan tautan bermanfaat ada di bawah potongan.

1. ReactiveX Redux // Dmitry Samokhvalov


Dmitry akan berbicara tentang konsep Observable dan manfaat yang dibawanya untuk bekerja dengan aplikasi asinkron. Ini akan menunjukkan cara menggunakan Observable dalam hubungannya dengan Redux dalam praktiknya, dan bagaimana itu berbeda dari Redux-Saga.

2. Penurunan tipe Action menggunakan Typcript // Dmitry Novikov


Dan lagi tentang Redux dan apa kerugian dari mengetikkan tipe tindakan adalah ketika tipe literal berubah menjadi primitif. Laporan ini tentang cara melacak proses ini, ditambah pengalaman solusi kami untuk masalah ini menggunakan Typefteks.

3. Proses peningkatan berkelanjutan // Oleg Luchansky


Ketika perlu menghabiskan waktu untuk tes, dan ketika Anda bisa melakukannya tanpa itu, mengapa perlu membangun kompetensi di semua bidang, dan tidak hanya menulis kode - ini dan banyak lagi yang bisa dibahas dalam kerangka laporan ini.

Anda dapat berbicara tanpa henti tentang tes, linters, CI / CD dan hal-hal keren lainnya, tetapi apa gunanya jika mereka tidak membantu tepat waktu atau meningkatkan kualitas penulisan kode?
