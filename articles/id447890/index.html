Terima kasih Tuhan aku bukan manajer

Saya telah bekerja DBA + ahli kinerja database + untuk beberapa waktu di sebuah perusahaan besar. Pekerjaan itu sangat nyaman, tetapi untuk beberapa waktu saya khawatir karena posisi saya menemui jalan buntu. Tumbuh tangga karier benar-benar mustahil bagi saya: Saya satu-satunya orang di kantor Rusia yang melapor langsung ke Amerika. Semua orang di sini adalah pengembangan.

Karena saya di Rusia, mereka tidak akan pernah menjadikan saya seorang manajer, karena saya jauh, karena saya berada di zona waktu yang berbeda, dan secara umum, peretas Rusia ini ... Seringkali, pertemuan mereka "macet" selama 5-6 jam. 2. Tonjolan tumbuh. Dia semakin sibuk dengan berbagai masalah. Mungkin celana formalnya sudah mengeluarkannya (pemimpin tim, senior, dll., Dll)

Saya memiliki keberuntungan untuk bekerja dengan ini (dan menyaksikan semua tahap sebelumnya) - dia mengawasi, tetapi jika itu - dapat meningkatkan segalanya dari awal - Vmware, NetApp, jaringan - apa pun



4. "Jumper" rusak. Sekarang seorang pria telah menjadi manajer 100%. Dia melakukan segalanya dengan tangan yang salah. Dalam istilah teknis, ia dapat terus mengembangkan "lantai atas" - membaca tentang versi baru produk, tetapi keterampilan praktisnya membeku pada saat jumper break. Kita dapat mengenali manajer seperti itu dengan teknologi favorit mereka, misalnya, seorang teknisi yang pergi sebagai manajer di tahun 90-an akan menjadi botak di telinga membahas transfer data antara komponen dalam bentuk file dan menunggu mereka diperiksa secara berkala untuk kehadiran mereka.


5. "Drop" muncul di lingkungan perusahaan.
