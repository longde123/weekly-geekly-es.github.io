Google deaktiviert die erste Version von reCAPTCHA


Google hat das Herunterfahren der ersten Version der beliebten reCAPTCHA-Service-API angekündigt . Ab dem 31. März 2018 funktionieren alle Anforderungen für die v1-API nicht mehr.

Ab November 2017 wurden bei einigen Anforderungen für die v1-API Benachrichtigungen über das bevorstehende Ende der Unterstützung für den Dienst ausgegeben (ein Beispiel ist in der CPDV enthalten). Laut Google haben die meisten Websites bereits auf die zweite Version der API umgestellt, für die der Benutzer keine Anstrengungen unternehmen muss, um das Captcha zu lösen. Aktivieren Sie einfach das Kontrollkästchen "Ich bin kein Roboter". Diese Version von reCAPTCHA verwendet einen fortschrittlichen Risikoanalysemechanismus, mit dem Sie Bots effektiv abschneiden und den Angreifern immer einen Schritt voraus sein können.

Es gibt auch Invisible reCAPTCHA, dessen Hauptunterschied das Fehlen einer Schnittstelle als solche ist. Der Entwickler kann die Prüfung an jede Benutzeraktion binden (z. B. um ein Formular zu senden), um Benutzer im Hintergrund zu prüfen.



Wenn Sie noch kein Upgrade auf die neue Version von reCAPTCHA durchgeführt haben, ist es Zeit, dies zu tun. Gehen Sie zum Admin-Bereich von reCAPTCHA, erstellen Sie einen neuen Schlüssel und aktualisieren Sie den Client-Teil des Captchas.
