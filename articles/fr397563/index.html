L'authenticité du Code de Grolier, le quatrième codex maya survivant, est prouvée.


Page 7 du Code de Grolier sur papier d'écorce de Ficus. Il représente une figure dans une coiffe avec des plumes. Une coiffure équivaut à une figure avec les dieux des tables de la planète Vénus dans le Codex de Dresde. Devant lui se trouve la tête barbue de Dieu C, soi-disant associée à l'étoile polaire, à partir de laquelle une plante pousse avec des bourgeons en forme de disques. Image numérisée: Codex Grolier

Au XVIe siècle, les conquistandors et les missionnaires catholiques ont travaillé dur pour détruire les vastes bibliothèques sacerdotales et gouvernementales mayas. En général, ils ont réussi. Des centaines de livres indiens à ce jour, seuls trois ont survécu (ils sont appelés "codes"). Ces artefacts uniques portent le nom des villes des musées dans lesquels ils sont stockés: codes mayas de Dresde, Paris et Madrid. Parmi ceux-ci, seul le Codex de Dresde est entièrement préservé. Les scientifiques y étudient encore des enregistrements astronomiques, compilés à partir d'observations mayas d'étoiles pendant plusieurs siècles.

En 1971, une véritable sensation s'est produite dans la communauté de l'histoire maya. Le collectionneur mexicain José Saenz a décidé de montrer au monde son principal trésor - le quatrièmele code maya conservé, qu'il aurait acheté en 1965 à un groupe de personnes inconnues. Le 20 avril 1971, l'artefact est exposé au public au gentlemen's club Grolie , d'où le code tire son nom.

Naturellement, l'histoire a suscité de grands doutes. Jose Saens a déclaré qu'il avait été emmené dans un endroit inconnu où des personnes non identifiées ont proposé d'acheter plusieurs artefacts, y compris des pages de ce code, qui auraient été récemment trouvés dans une grotte dans les montagnes mexicaines de la Sierra Madre de Chiapas.

Le Codex se compose de 11 fragments de papier endommagés (de l'écorce de différents types de ficus), chacun d'environ 18 × 12,5 cm; images sur un seul côté des feuilles. On suppose que l'original était composé d'au moins 20 feuilles.

Comme vous le savez, les Mayas avaient une passion douloureuse pour les étoiles et même l' emplacement des villes a été choisi conformément au schéma des étoiles. En termes de contenu, le Code de Grolier trouvé fait écho aux tableaux de Vénus dans le Codex de Dresde. Il contient des images des mêmes dieux et des données astrologiques correspondant aux phases de Vénus dans une datation simplifiée selon le calendrier maya.

Les experts ont immédiatement remis en question l'authenticité du Code de Groglier. Cela peut être compris si l'on tient compte du montant astronomique auquel ce document pourrait être estimé aux enchères. Nous pouvons probablement parler de dizaines ou de centaines de millions de dollars.

Une analyse au radiocarbone a permis de dater le manuscrit à 1230, mais n'a pas convaincu les sceptiques. Ils ont continué à affirmer que le manuscrit aurait pu être fait sur de véritables feuilles de papier trouvées dans les fouilles.

Un réexamen de 2007 a donné des résultats contradictoires.

Et seulement maintenant, en 2016, les scientifiques peuvent mettre un terme à ce différend (bien qu'ils ne le puissent pas). L'authenticité du manuscrit avec un haut degré de fiabilité est confirméeun groupe de scientifiques respectés de l'Université Brown, de l'Université Harvard et de l'Université de Californie à Riverside (tous des États-Unis). Les scientifiques ont effectué une analyse approfondie de tous les examens et recherches scientifiques précédemment menés sur ce sujet et, avec toute autorité, ont conclu que le manuscrit était authentique.

L'autorité des scientifiques qui se sont inscrits à ce travail nous permet de parler de l'atteinte d'un consensus scientifique sur l'authenticité du Code de Grolier. Il s'agit en effet du quatrième codex maya survivant.

Un groupe dirigé par le professeur Stephen Houston de l'Université Brown a soigneusement étudié l'origine du manuscrit, les caractéristiques et le style de l'iconographie, la nature et la signification des tableaux de Vénus, les données scientifiques, y compris les données d'analyse du radiocarbone, et les caractéristiques physiques du code lui-même - comment il a été élaboré. , à partir de quel papier, comment les inscriptions ont été appliquées. Ils ont conclu que, à tous égards, le code était probablement authentique. Les auteurs de l'ouvrage scientifique répertorient en détail tous les arguments des sceptiques concernant l'authenticité du code, et répondent raisonnablement à chaque argument.

Selon eux, les divergences avec le Code de Dresde (par exemple, un motif différent sur les pieds du dieu de la mort) ne devraient pas être considérées comme des preuves de faux. Seuls les anciens codes, y compris le Code de Dresde, ont été trouvés plus tôt, ce qui leur a conféré un statut canonique. Dans le même temps, l'analyse radiocarbone montre que le code de Grolier est écrit plus tôt que le code de Dresde.

Le papier de l'écorce de ficus, les fines lignes rouges du croquis sous les dessins, ainsi que l'encre Maya pigmentée bleue sont complètement authentiques, disent les scientifiques. Ils énumèrent la liste des connaissances qu'un employé devait posséder (ou deviner) afin de falsifier un document de manière fiable au milieu des années 1960. La liste comprenait les informations qui n'ont été connues des scientifiques qu'au cours des 45 dernières années après la découverte du code. Cette liste est si grande qu'elle peut être considérée comme "prohibitive", c'est-à-dire presque impossible du point de vue de la théorie des probabilités. Il est incroyable que le greffier ait deviné tous ces détails.

Par exemple, la synthèse de l'encre bleue, similaire à l'encre maya, n'a été réalisée par des scientifiques mexicains que dans les années 1980.

Sur les pages du Codex Grollier sont les dieux mayas de tous les jours qui les accompagnaient dans la vie de tous les jours: ce sont les dieux du Soleil, de la mort, de la foudre et autres. Bien que tous, selon la croyance maya, accomplissent la volonté des étoiles, et plus précisément, la volonté de "l'étoile" de Vénus. Les codes de Dresde et de Madrid représentent une plus grande variété de dieux, mais le Code de Groglie ne contient que l'ensemble de base.


Page 6 du Code de Grolier. Le dieu de la mort (dieu A) en simple coiffe avec un couteau à la main vient de décapiter un autre dieu, dont le cou coule du sang. Un couteau pour décapiter une telle forme chez le dieu de la mort se trouve sur l'une des fresques du Temple des Guerriers à Chichen Itza . Image numérisée: Codex Grolier

Selon les scientifiques, le Code de Grolier est de nature prédictive plutôt que narrative. Il décrit les événements qui devraient se produire à l'avenir plutôt que les événements passés. A en juger par les dates, le code pourrait être utilisé comme calendrier de bureau pour les prêtres pendant 104 ans après la date de création.

Le Code de Grolieu est maintenant conservé au Musée national d'anthropologie de Mexico. Il n'est pas exposé au public. Cela est probablement dû à l'histoire scandaleuse qui a accompagné sa découverte et aux doutes sur l'authenticité du document. Maintenant que ces doutes sont largement dissipés, un artefact unique peut attirer l'attention du public et de la communauté scientifique.
