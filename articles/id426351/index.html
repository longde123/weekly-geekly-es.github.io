Perubahan pada Sertifikasi dan Ujian Microsoft

Pada konferensi Ignite baru-baru ini, Microsoft memperkenalkan program sertifikasi baru. Perusahaan mengatakan bahwa sertifikasi baru berdasarkan peran / spesialisasi akan membantu Anda meningkatkan kualifikasi Anda, menunjukkan keahlian dan pengetahuan Anda kepada pengusaha dan menemukan peluang baru.


Enam spesialisasi baru telah diumumkan: Administrator Azure , Pengembang , Arsitek Solusi , Insinyur DevOps, Administrator Desktop Modern Microsoft 365, dan Administrator Perusahaan. Ujian dan pelatihan untuk sertifikasi Azure Administrator sudah tersedia, dan dua ujian berikutnya - Pengembang Azure dan Arsitek Solusi Azure dalam versi beta. Tiga sertifikasi tersisa dari daftar ini akan dikeluarkan sebelum akhir tahun. “Dan ini baru permulaan! Kami berencana untuk mengumumkan lebih banyak spesialisasi pada 2019. ” - Selina Winter menulis di blog Microsoft.




Selain itu, di samping pertanyaan biasa dengan beberapa jawaban, mereka berjanji untuk menambahkan yang baru berdasarkan skenario aktif, variasi seret dan lepas dan situasi yang lebih dekat dengan yang berfungsi.


Yang ditunjukkan di bawah ini adalah kursus pelatihan sertifikasi Administrator Azure.




Penting! Banyak ujian dari seri 70-XXX akan dianggap usang mulai 31 Desember 2018 dan akan digantikan oleh ujian baru seri: AZ-10X, AZ-20X, AZ-30X, MS-10X.

Jika Anda berencana untuk mengambil ujian Microsoft dalam waktu dekat, maka hati-hati pantau informasinya di halaman ujian berbahasa Inggris. Mungkin ujian Anda telah digantikan oleh yang baru dan akan lebih praktis untuk lulus / mempersiapkan segera untuk yang baru daripada lulus yang lama.

Semua Profesional Bersertifikat Microsoft Azure sekarang diberi kesempatan untuk lulus ujian transisi $ 99. (Harganya bervariasi berdasarkan negara / wilayah dan tercermin dalam proses pendaftaran ujian.) Harga akan tersedia hingga 30 Juni 2019 .


Ujian transisi hanya mencakup materi baru dan perbedaan antara sertifikasi saat ini dan keterampilan serta konsep dalam sertifikasi baru.


Jika Anda lulus dari salah satu ujian berikut (atau lulus sebelum 31 Desember 2018, setelah tanggal itu mereka akan dianggap usang):


  • 70-533: Menerapkan Solusi Infrastruktur Microsoft Azure
  • 70-532: Mengembangkan Solusi Microsoft Azure
  • 70-535: Arsitektur Solusi Microsoft Azure

Anda dapat lulus ujian transisi yang sesuai (akan tersedia hingga 30 Juni 2019):


  • AZ-102: Administrator Microsoft Azure. Ujian transisi
  • AZ-202: Pengembang Microsoft Azure. Ujian Transisi (Beta)
  • AZ-302: Arsitek Solusi Microsoft Azure. Ujian Transisi (Beta)

Jadi, ada dua cara, dua cara untuk mendapatkan sertifikat baru.


1. Lulus ujian usang + transisi dan dapatkan sertifikat baru


70-533 + AZ-102 -> Microsoft Certified Professional: Azure Administrator
70-532 + AZ-202 -> Microsoft Certified Professional: Azure Developer
70-535 + AZ-302 -> Microsoft Certified Professional: Arsitek Solusi Azure


Atau


2. Lulus dua ujian baru dan dapatkan sertifikat baru


AZ-100 + AZ-101 -> Spesialis Bersertifikat Microsoft: Administrator Azure
AZ-200 + AZ-201 -> Spesialis Tersertifikasi Microsoft: Pengembang Azure
AZ-300 + AZ-301 -> Microsoft Certified Professional: Arsitek Solusi Azure


Saya ulangi sekali lagi bahwa metode pertama hanya akan tersedia hingga 30 Juni 2019.


Jika saat ini Anda sedang berupaya untuk mendapatkan sertifikat MCSE: Platform Cloud dan Infrastruktur , maka Anda punya waktu hingga 31 Desember 2018 . Mulai 1 Januari 2019 , sertifikat ini akan diganti oleh MCSE baru: Infrastruktur dasar, yang mencakup MCSA Windows Server 2012 dan MCSA Windows Server 2016 .




Ujian dianggap usang sejak 31 Desember 2018


  • 70-496: Mengelola Visual Studio Team Foundation Server
  • 70-497: Pengujian Perangkat Lunak dengan Visual Studio
  • 70-498: Memberikan Nilai Berkelanjutan dengan Manajemen Siklus Aplikasi Aplikasi Visual Studio
  • 70-532: Pengembangan Solusi Microsoft Azure (digantikan oleh ujian: AZ-200 dan AZ-201 )
  • 70-533: Menerapkan Solusi Infrastruktur Microsoft Azure (digantikan oleh ujian: AZ-100 dan AZ-101 )
  • 70-535: Arsitektur Solusi Microsoft Azure (diganti dengan ujian: AZ-300 dan AZ-301 )

Ujian dianggap usang sejak 31 Maret 2019


  • 70-346: Office 365 identitas dan persyaratan manajemen (diganti dengan ujian: MS-100: Identifikasi dan layanan Microsoft 365 dan MS-101 Mobilitas dan keamanan Microsoft 365, ujian beta dijadwalkan untuk akhir 2018)
  • 70-347: Instalasi layanan Office 365 (diganti dengan ujian: MS-100: Identifikasi dan layanan Microsoft 365 dan MS-101 Mobilitas dan keamanan Microsoft 365, ujian beta dijadwalkan untuk akhir 2018)
  • 70-695: Penyebaran aplikasi desktop dan perusahaan Windows (digantikan oleh ujian: MD-100: Windows 10 dan MD-101: Modern Computer Management, ujian beta dijadwalkan untuk akhir 2018)
  • 70-697: Mengkonfigurasi perangkat Windows (diganti dengan ujian: MD-100: Windows 10 dan MD-101: mengelola komputer modern, ujian beta dijadwalkan pada akhir 2018)
  • 70-698: Menginstal dan mengonfigurasi Windows 10 (diganti dengan ujian: MD-100: Windows 10 dan MD-101: mengelola komputer modern, ujian beta dijadwalkan pada akhir 2018)

Ujian dianggap usang sejak 30 Juni 2019


  • AZ-102: Administrator Microsoft Azure. Ujian transisi
  • AZ-202: Pengembang Microsoft Azure. Ujian transisi
  • AZ-302: Arsitek Solusi Microsoft Azure. Ujian transisi

Ujian dianggap usang sejak 31 Juli 2019


  • 74-343: Manajemen Proyek dengan Microsoft Project 2013

Sertifikasi yang akan dianggap usang mulai 31 Desember 2018


  • MCSA: Azure Linux
  • MCSA: Platform Cloud

Semua ujian beta dikenakan diskon sementara sebesar 80% jika Anda mendaftar untuk ujian sebelum 26 Oktober 2018, dan lulus sebelum 29 Oktober 2018.
