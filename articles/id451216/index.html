Cara menonaktifkan Windows Defender sepenuhnya di Windows 10



Windows 10 memiliki built-in Windows Defender antivirus ("Windows Defender") yang melindungi komputer dan data Anda dari program yang tidak diinginkan: virus, spyware, ransomware, dan banyak jenis malware dan peretas lainnya.

Meskipun solusi keamanan bawaan cukup untuk sebagian besar pengguna, ada beberapa situasi di mana Anda mungkin tidak ingin menggunakan program ini. Misalnya, jika Anda mengatur perangkat yang tidak akan online; jika Anda perlu menyelesaikan tugas yang diblokir oleh program ini; Jika Anda harus memenuhi persyaratan kebijakan keamanan organisasi Anda.

Satu-satunya masalah adalah Anda tidak akan dapat menghapus atau menonaktifkan Windows Defender sepenuhnya - sistem ini sangat terintegrasi ke dalam Windows 10. Namun, ada beberapa solusi yang dapat digunakan untuk menonaktifkan antivirus - menggunakan kebijakan grup lokal, registri, atau pengaturan Windows di bagian Keamanan (sementara).

Cara menonaktifkan Windows Defender melalui pengaturan keamanan Windows


Jika Anda perlu melakukan tugas tertentu, dan tidak perlu sepenuhnya menonaktifkan Pembela, Anda dapat melakukan ini sementara. Untuk melakukan ini, gunakan pencarian di tombol "Start", cari bagian "Windows Defender Security Center", dan pilih "Virus and Threat Protection" di dalamnya.



Di sana, buka bagian "Perlindungan terhadap virus dan ancaman lainnya" dan klik pada saklar "Perlindungan waktu nyata".



Setelah itu, antivirus akan menonaktifkan perlindungan komputer waktu-nyata, yang memungkinkan Anda untuk menginstal aplikasi atau melakukan tugas tertentu yang tidak tersedia bagi Anda karena antivirus memblokir tindakan yang diperlukan.

Untuk mengaktifkan kembali perlindungan waktu nyata, hidupkan ulang komputer atau ulangi semua langkah konfigurasi lagi, tetapi hidupkan sakelar di langkah terakhir.

Solusi ini tidak permanen, tetapi paling cocok untuk menonaktifkan Windows 10 Antivirus untuk tugas tertentu.

Cara menonaktifkan Windows Defender melalui kebijakan grup


Dalam versi Windows 10 Pro dan Enterprise, Anda dapat menggunakan editor kebijakan grup lokal, tempat Anda dapat menonaktifkan Defender secara permanen sebagai berikut:

Melalui tombol "Start", jalankan skrip yang dapat dieksekusi gpedit.msc. Editor kebijakan terbuka. Pergi ke jalur berikut: Konfigurasi Komputer> Template Administratif> Komponen Windows> Windows Defender Antivirus.



Klik dua kali untuk membuka opsi "Matikan program antivirus Windows Defender." Pilih pengaturan "Diaktifkan" untuk mengaktifkan opsi ini, dan, karenanya, menonaktifkan Pembela.



Klik "OK" dan nyalakan kembali komputer.

Setelah itu, antivirus akan dinonaktifkan secara permanen di perangkat Anda. Tetapi Anda akan melihat bahwa ikon dengan perisai akan tetap ada di bilah tugas - seharusnya begitu, karena ikon ini milik aplikasi Keamanan Windows, dan bukan antivirus itu sendiri.

Jika Anda berubah pikiran, Anda selalu dapat mengaktifkan kembali Defender dengan mengulangi langkah-langkah ini dan memilih "Not Set" pada langkah terakhir, setelah itu Anda harus menyalakan kembali komputer.

Cara menonaktifkan Windows Defender melalui registri


Jika Anda tidak memiliki akses ke editor kebijakan, atau Anda telah menginstal Windows 10 Home, Anda dapat mengedit registri Windows, sehingga menonaktifkan Defender.

Saya mengingatkan Anda bahwa mengedit registri berisiko, dan kesalahan dalam hal ini dapat menyebabkan kerusakan yang tidak dapat diperbaiki pada salinan Windows yang saat ini diinstal. Lebih baik untuk membuat cadangan sistem sebelum Anda mulai mengedit.

Untuk sepenuhnya menonaktifkan Defender melalui registri, jalankan program regedit melalui tombol "Mulai", dan buka sepanjang jalur berikut:

HKEY_LOCAL_MACHINE \ SOFTWARE \ Policies \ Microsoft \ Windows Defender

Tip: jalur ini dapat disalin dan ditempelkan ke bilah alamat editor registri.



Kemudian, klik kanan pada kunci Windows Defender (direktori), pilih "New" dan Nilai DWORD (32-bit). Beri nama tombol baru DisableAntiSpyware dan tekan Enter. Kemudian klik dua kali untuk membuka editor kunci dan atur ke 1.



Klik OK, dan nyalakan kembali komputer.

Setelah itu, Windows Defender tidak akan lagi melindungi sistem Anda. Jika Anda ingin membatalkan perubahan ini, ulangi semua langkah, tetapi pada akhirnya hapus kunci ini atau atur ke 0.

Rekomendasi


Meskipun ada beberapa metode untuk menonaktifkan Windows Defender, kami tidak merekomendasikan menggunakan komputer tanpa program anti-virus sama sekali. Namun, Anda mungkin menghadapi situasi di mana menonaktifkan fitur ini akan menjadi pilihan yang lebih baik. Dan jika Anda menginstal program antivirus pihak ketiga, Anda tidak perlu menonaktifkan Defender secara manual, karena selama instalasi itu akan mati secara otomatis.
