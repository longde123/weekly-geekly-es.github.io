报警！ 警报！：鱼如何警告亲戚危险



友谊有麻烦了。 当涉及野生生物，即社交动物时，我们所有人都熟悉的这个短语成为生死攸关的问题。 同样在这里，该短语将是适当的-警告表示武装。 您可能已经看到猫鼬或其他成群生活的动物中的哨兵，并且担心掠食者的袭击。 遇险时，猫鼬会发出声音，他的所有同志立即躲在貂皮中。 但是，如果您像鱼一样笨呢？ 或者说，如果您是一条鱼。 当然，请使用人体分泌的化学化合物。 科学家发现，如有必要，某些种类的群鱼能够散发出某些物质，警告亲属危险。 科学家如何准确地发现他们的发现与以前的发现有何不同，以及鱼类社会中的通讯系统有多复杂？ 有关研究小组报告的答案，请参阅。

学习基础


在任何生物体中，社会性的主要表现是交流，即一组内某个物种的代表之间的交流。 通过通信，信息被传输，这是显而易见的。 这就像是逃离这里的呼吁，因为这里有很多东西，必须从中发出关于危险的警告。


鱼躲避危险。

粗略地说，不同生物之间交流的原因和传递的信息非常相似，但是传递的方法有所不同：声音，光线，手势，化合物的分配等。 而且其中一些信号无法被其他物种感知，如果您向小组警告附近的捕食者，这将非常有用。 人也没有注意到许多这些信号。 这正是导致以下事实的事实：数十年来尚未研究各种物种的许多社会方面。

研究人员注意到，在任何信号系统中，都有几个主要组成部分：上下文（a），自愿信号（b），接收方的响应（c）以及发送方和接收方的收益（d）。

首先，一个人面临某种环境（信息/情况），这可能是由于外部因素（寻找食物，附近有其他物种或掠食者存在），内部因素（饥饿，恐惧等）引起的。并且接收器（iii）附近有信号。 因此，信号上下文由外部刺激，个体的内部状态以及所生成信号（即目标信号）的目标“受众”组成。 科学家们还指出，这些信号是自愿的，是由他们自己的自由意志产生的。


雪鱼和虾之间的共生关系。 虾的天线之一一直与鱼接触，在危险情况下鱼会藏在一个洞中。 收到这样的信号后，虾也躲起来了。

这一点（信号的自愿性）非常重要，因为并非所有信号都是如此。 某些信号是根据上下文随机生成的，而没有信号源的有意识参与。 例如，一个受伤的人的气味向其他人发出信号，表明附近有危险，但该信号实际上是由掠食者而非个人受害者造成的。 换句话说，这是一个非自愿的信号。

下一个元素是主机。 信号使接收方产生一定的响应，具体取决于信号的性质，其中可能包括行为，生理甚至是遗传变化。

同样重要的是信号对于发送信号的人和接收信号的人的好处。 在这种情况下，情况可以互惠互利，也可以仅针对产生信号的个人。 互利信号可以用作重要的生存机制，因此通过遗传和/或后代训练代代相传。

正如科学家承认的那样，实际上很难测量信号的强度，信号的任意或随机性质以及其对发生器和接收方的好处。 通常，这些特征是在确定事实之后确定的，即估算了该组的存活程度，死亡率和出生率等。 简而言之，如果一个小组使用信号并且它们还活得很好，那么信号就可以发挥其优势。 但是这种方法非常不准确，无法回答许多问题：该群体的行为是否在受控条件下得以维持，该行为是否传播给后代等等。

如果简化传输危险信号的系统，我们可以区分两个主要的上下文元素：外部因素（捕食者）和听众（相同物种和/或群体的代表）。 在这种情况下，信号强度可能会根据情况而变化。 如果亲戚足够近，则信号会更强。 如果袭击的受害者是一名受害者，但亲戚距离较远，则警报信号将微弱，以免引起掠夺者的过多注意。


群中的乌鸦比单独的大胆，因此它们甚至可以攻击白头鹰。

观众的组成，也就是接收信号的观众的组成也很重要。 如果有不只是同一物种的代表，而且附近的一般物种（家庭，伴侣，后代）的代表，则某些鸟类和灵长类动物会放大危险信号。

关于海洋居民的讨论，科学家们再次回顾了区分自愿信号和非自愿信号的重要性。 例如，卡尔·冯·弗里施（Karl von Frisch）于1938年发现了化学化合物Schreckstoff，其成分之一是H 3 NO。 这种化合物是由受伤或死亡的人制造的，但不能称为自愿和故意的警报。

大多数鱼类信号的主要成分是尿液，尿液用于隔离信息素，在特定区域内显示优势，区分相关个体等。 基于此，科学家建议，尿液在焦虑症的信号传导中起重要作用。

在这项研究中，科学家进行了涉及黑色粗头（Pimephales promelas）的测试。 这种鱼过着社交生活。 万一发生危险，它们首先会迅速随机移动，然后冻结，然后聚集在一起并试图躲藏在安全的地方。

研究人员进行了一系列测试，其中掠食者攻击胖子，以确定所生成警报的自愿性，信号强度取决于宿主个体（朋友/陌生人）的听众或附近听众的有无。

测试结果


选择了69个个体作为信号发生器，选择了270个个体作为信号发生器。 所有个人都没有接受过警报方面的培训，但在测试过程中，他们与熟悉的人或陌生人一起进入了水族馆。 发生器和接收器之间没有特殊的生理差异：重量-0.6±0.2 g，体长-40±5 mm。

信号之前和之后的分组索引非常不同（ 2a ），具体取决于信号的类型和单个生成器的类型（双向交互：LRT x 2 2 = 9.94，p = 0.0069）。

通过将数据除以信号类型和发生器类型，科学家可以更详细地了解这些变量之间的关系。 三种不同类别的信号发生器的个体之间的非报警信号（正常）没有差异（LRT x 2 2 = 3.33，p = 0.19），但是相反，报警（LRT x 2 2 = 10.75，p = 0.0046）。 科学家们还注意到，熟悉单个信号发生器的个人群体对他们所产生的警报的反应更加积极（LRT x 2 1 = 16.86，p <0.0001），但是“陌生人”群体反应迟钝而缓慢（LRT x 2 1 = 1.28，p = 0.26）。

关于麻木（ 2b ），还在信号的类型与其产生器的类型之间找到关系（卡方检验： x 2 7 = 28.2，p = 0.0002）。 对于正常信号，无论信号发生器的类型如何，接收组通常只有一个麻木的人。 但是，在警报方面，与分组指示器一样存在差异。 如果警报来自一个熟悉的个人，而不是来自一个陌生人（费舍尔的精确检验：p = 0.002）或一个（孤立的）个人（p = 0.0002），那么警报的接收者就会更加麻木。 对陌生人和孤立个体的信号的反应速度非常相似，这再次证实了同一群体中个体之间社会联系的重要性。 正如他们所说，他们比陌生人更信任自己。

有了混沌运动的指示，情况就更加奇怪了（ 2秒 ）。 该小组对任何类型的发电机（熟悉的人，陌生人和孤立的人）的正常信号均无反应。 该小组对熟悉的人发出的警报的反应要强于对警报和孤立的人的反应。

结语


经过上述测试的研究人员证实了以下事实，即警报（在这种情况下为化学物质）是防止产生信号的人和接收信号的人免受天敌的重要组成部分。 还发现这些信号是自愿和故意产生的，与受伤或被杀害的人的非自愿信号相反。

此外，研究表明，与陌生人或孤立的人相比，该组更愿意响应同一组人发出的警报。 这一观察结果证实了这样一个事实，即特定群体而不是一般物种中存在更牢固的社会联系。

这项研究使您可以更好地了解水下居民的社会结构及其基本要素。 野生动物世界是一个危险的地方，特别是对于那些饥饿多的小鱼。 通过组建团体，他们增加了生存的机会。 但是，如果该团体对周围的危险视而不见，那么群众就没有多大用处了。 这就是为什么每个小组成员的环保意识在争取生存的斗争中发挥如此重要作用的原因。
