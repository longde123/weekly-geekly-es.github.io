Emulator konsol game terbaik untuk Windows



Ini adalah bagian kedua dari pilihan emulator konsol game. Pada bagian pertama , emulator yang paling populer dan bisa diterapkan untuk Android dikumpulkan. Ada juga diposting survei pada subjek seleksi berikutnya. Dengan margin lebar, opsi untuk membuat koleksi set-top box emulator untuk Windows dimenangkan.



Pada artikel ini saya mencoba mengumpulkan sejumlah besar emulator konsol permainan yang telah teruji untuk Windows. Dengan bantuan program ini, Anda dapat dengan mudah mengubah PC Anda menjadi konsol game retro, dan menikmati game klasik, yang hanya menyisakan pengalaman terbaik.

Mungkin sesuatu tidak berhasil sampai ke seleksi. Jika demikian, tunjukkan emulator yang dilompati dalam komentar, saya akan menambahkan ke bahan utama.

Prinsip kerjanya dengan emulator konsol


Emulator konsol game adalah shell perangkat lunak yang mengemulasi operasi perangkat keras perangkat itu sendiri. Mengembangkan emulator tidak sesederhana itu - ia membutuhkan pengalaman pemrograman dan pengetahuan informasi yang akurat tentang prinsip-prinsip operasi perangkat asli. Kebanyakan emulator dapat dibagi menjadi dua kelompok, termasuk program "sederhana" dan "majemuk". Banyak emulator memberikan fungsionalitas lebih banyak daripada konsol yang lebih lama (misalnya, menyimpan game untuk Dendy dan Sega).

Tidak ada masalah sederhana - Anda perlu menginstal program, mengunduh gambar gim, dan hanya itu - Anda dapat menikmati proses passing. Ada pengaturan minimum dalam program tersebut. "Compound" memerlukan komponen tambahan - ini adalah BIOS konsol (biasanya file .bin) dan berbagai plugin dll. File BIOS dilindungi oleh hak produsen, sehingga pengembang emulator tidak dapat menanamkannya, jika tidak, emulator seperti itu tidak akan masuk ke direktori program Google. BIOS perlu dicari di internet - ada banyak situs tempat file-file ini dibagikan.

Kami membutuhkan gambar permainan. Pengembang emulator mereka juga tidak memiliki hak untuk menyematkan perangkat lunak mereka. Gambar, atau ROM, tersedia di Internet. Untuk Sega atau PlayStation, jumlah ROM mencapai ratusan. Sekarang di Internet Anda dapat menemukan gambar bahkan game yang sangat langka yang sebelumnya sulit didapat di toko.

Sistem Hiburan Nintendo (NES)


NESTOPIA UE




Secara teknis, emulator Nestopia klasik tidak lagi didukung oleh pengembangnya. Tetapi ada port yang didukung - NESTOPIA UE. Dalam hal ini, UE berarti "Undead Edition", ini adalah versi tidak resmi dari NESTOPIA. Kelebihan dari program ini adalah kesederhanaannya. Ada pengaturan minimum, tetapi emulator bekerja dengan hampir semua game lama untuk NES, memberikan tampilan kualitas tertinggi. NESTOPIA UE adalah salah satu emulator terbaik dari konsol delapan-bit.

Fceux




Ini adalah salah satu emulator NES yang paling fleksibel. Aplikasi all-in-one menawarkan peluang luas untuk gamer biasa dan pemain mahir yang ingin mendapatkan kontrol maksimum atas proses yang terjadi di emulator. Ada alat debugging, kemampuan untuk merekam permainan, alat untuk bekerja dengan ROM dan alat bantu untuk membuat speedranas - proses mempercepat melalui permainan. Emulator ini memiliki port untuk Windows, Max OS X dan Linux.

Higan


Emulator ini merupakan kelanjutan dari Bsnes. Versi pertama dirilis pada 2005. Awalnya, emulator bekerja di bawah Windows, tetapi kemudian port muncul di Linux, OS X dan FreeBSD. Ini adalah emulator NES yang hebat yang kompatibel dengan sejumlah besar gim yang berbeda.

Emulator terus berkembang - penulis secara aktif bekerja untuk mempertahankan proyeknya. Pada 2008, majalah Inggris WebUser menyarankan emulator untuk semua gamer jadul. Pada tahun 2009, sebuah majalah Jepang sudah menyarankan semua penggemar game lama untuk mencoba emulator dalam pekerjaan.

Sistem Hiburan Super Nintendo (SNES)


B-merkuri




Emulator ini adalah modul SNES untuk RetroArch. Program ini adalah fork dari emulator BSnes, di mana sejumlah fungsi berguna telah ditambahkan. Versi emulator saat ini memiliki kemampuan untuk mengunduh file BIOS, yang sebelumnya tidak tersedia di SNES9X. Emulator mendukung sejumlah besar gim, dengan bekerja hampir semua gambar gim SNES yang ada di jaringan. Jika Anda memiliki emulator ini, kemungkinan besar tidak akan ada masalah yang muncul.

SNES9X




SNES9X jelas merupakan salah satu emulator SNES paling sukses. Ia bahkan dapat bekerja dengan rilis terbaru untuk Super Famicom, yang tidak dapat ditangani oleh beberapa emulator lain. Para pengembang melengkapi emulator dengan sejumlah besar fitur, termasuk video, suara, cheat dan multiplayer. Port emulator ini dirilis di bawah Windows, Max OS X, iOS, Android. SNES9X juga tersedia sebagai kernel di bawah RetroArch.

Zsnes


ZSNES adalah emulator konsol permainan SNES gratis yang ditulis sebagian besar dalam bahasa assembly untuk platform Linux, MS-DOS, dan Windows. Pengembangan ZSNES dimulai pada 3 Juli 1997, dan versi pertama dari program ini dirilis pada 14 Oktober 1997, untuk platform MS-DOS. Port resmi untuk Windows dan Linux kemudian dirilis. Emulator menjadi terbuka dan dirilis di bawah lisensi GPL pada 2 April 2001. Pada bulan Maret 2006, repositori sumber program dipindahkan dari SourceForge.net ke BountySource.

Fitur utama emulator:
  • Emulasi sebagian besar chip ekspansi SNES
  • Dukungan interpolasi gambar
  • Kemampuan merekam video
  • Dukungan curang


Nintendo 64


Proyek 64




Project 64 adalah salah satu emulator Nintendo 64 yang paling kompatibel dengan semua gim. Program ini tidak memerlukan BIOS. Plugin dikonfigurasi secara default dan semuanya berfungsi dengan baik. Emulasi suara dan video dari set-top box asli bekerja dengan sangat baik. Tidak ada banyak pengaturan agar tidak menyulitkan kehidupan pengguna. Tetapi ada dukungan untuk multipemain, dukungan untuk curang. Anda dapat menyesuaikan skala layar, memberikan kenyamanan gameplay yang maksimal bagi Anda.

Mupen64plus




Modul lain untuk RetroArch, emulator universal, yang akan kita bicarakan di akhir artikel. Selain itu, modul ini adalah satu-satunya. Ada banyak pengaturan. Jika Anda mau, Anda dapat dengan mudah mengubah banyak - dan skema warna layar, dan suara dan segala sesuatu yang umumnya mungkin berubah. Ada port untuk Windows, Mac OS X, Android, dan Linux.

Nintendo GameCube dan Wii


Lumba-lumba




Dolphin adalah satu-satunya emulator yang layak untuk Nintendo GameCube dan Wii. Ini bekerja hampir lebih baik daripada konsol game itu sendiri, yang harus ditiru oleh perangkat lunak ini. Selain perataan dan kemampuan menyimpan dengan cepat dalam game, Anda dapat bermain dengan resolusi 1080p. Fitur ini tidak ada di konsol asli. Tentu saja, ada beberapa bug, tetapi tidak begitu banyak. Selain itu, pekerjaan pada emulator dilakukan oleh para penggemar, sehingga tidak ada keluhan khusus terhadap mereka. Yang perlu Anda ketahui - emulator bekerja hampir dengan sempurna, menyediakan grafik berkualitas tinggi dan gameplay yang nyaman.

Gameboy, Gameboy Color, dan Gameboy Advance


Vba-m




Ada beberapa emulator untuk sistem Game Boy, tetapi Game Boy adalah yang terbaik. Ia dapat bekerja baik sebagai emulator mandiri, dan sebagai modul untuk RetroArch. Reproduksi warna, suara, gameplay - semuanya hampir sempurna. Pengguna yang menuntut dapat bekerja dengan filter grafik, menyesuaikan suara sesuai keinginannya, dan memodifikasi beberapa pengaturan lainnya. Emulator porting di bawah Windows, Mac OS X, Linux. Emulator memerlukan Microsoft DirectX, lebih disukai versi terbaru.

Nintendo DS (dan DSi)


Tidak ada $ GBA




Emulator ini memulai perjalanannya ke dunia game sebagai emulator dari sistem Gameboy Advanced. Tapi sekarang ini adalah emulator NIntendo DS yang paling serbaguna, dan satu-satunya yang mendukung DSi. Ini juga merupakan emulator tercepat, yang tidak begitu mudah untuk disediakan, mengingat kehadiran dua layar dan prosesor di konsol itu sendiri. Beberapa fungsi tidak ada dalam emulator (misalnya, WiFi dan multipemain), beberapa game tidak ditampilkan dengan sempurna. Tetapi dalam sebagian besar kasus, emulator ini menyediakan gim yang sepenuhnya nyaman. Ada banyak pengaturan di sini, jadi jika Anda melihat beberapa masalah, kemungkinan besar bisa diselesaikan dengan pengaturan. ada juga alat debugging untuk pengembang.

Desmume




Kernel yang bagus untuk RetroArch, walaupun ada sejumlah batasan, terutama ketika membandingkan emulator ini dengan yang berdiri sendiri. Meskipun demikian, Anda dapat memainkan sebagian besar game tanpa masalah, ada sejumlah pengaturan. Perangkat lunak ini juga mendukung emulasi GBA.

Sega


Fusi kega




Emulator Sega yang hampir sempurna - bukan hanya satu konsol Sega Mega, tetapi seluruh seri, termasuk Genesis, Game Gear, Sega CD. Dukungan Saturnus dan Dreamcast tidak ada di sini. Semua konsol lain dicontoh dengan sempurna, hampir kompatible dengan game untuk platform yang didukung. Ada dukungan untuk menipu, menyimpan cepat, merekam video dan audio, game online dan dukungan untuk berbagai gamepad. Mode layar penuh, penskalaan dan sejumlah filter video melengkapi kemampuan emulator. Port tersedia di Windows, Mac OS X, dan Linux.

Kejadian X Plus


Dan ini adalah modul untuk RetroArch. Juga mendukung persaingan platform seperti Genesis, Mega Drive, Sega / Mega CD, Master System, Game Gear & SG-1000. Sebagian besar game didukung, ada banyak pengaturan. Ada juga kemampuan untuk mem-boot BIOS, jika perlu.

Dreamcast


Inilah masalahnya. Untuk konsol ini, praktis tidak ada emulator normal untuk Windows. Ada sejumlah upaya untuk meniru konsol, termasuk DEMUL atau nullDC. Tetapi sangat sedikit game yang didukung di sini, sejumlah besar masalah kompatibilitas, itu tidak akan bekerja secara stabil. Menariknya, Android memiliki emulator Dreamcast yang sangat baik - ini adalah ReiCast, saya menulis tentangnya di pilihan terakhir. Tetapi di bawah Windows, tidak ada. Yang buruk adalah bahwa hampir tidak ada yang mengembangkan emulator pada saat ini.

Demul




DEmul adalah emulator Dreamcast yang merupakan penerus Chankast, aplikasi lain untuk meluncurkan game Dreamcast di PC, yang tidak lagi didukung. Demul adalah emulator multi-sistem konsol game dan mesin slot dari Sega, awalnya ditujukan khusus untuk meniru konsol game Sega Dreamcast. Saat ini, emulator juga mendukung emulasi mesin slot berdasarkan Sega Dreamcast: Naomi, Naomi 2 dan Atomiswave. Proses pengembangan mendukung Sega Model 3, Sega Hikaru, Sega Saturn. Namun, untuk ini Anda akan memerlukan Dreamcast BIOS, yang tidak disertakan dengan emulator ini. Saat ini, DEmul dapat dengan sempurna meniru lebih dari seratus game, termasuk Marvel VS. Capcom, Crazy Taxi, King of Fighters, Sega GT, Sonic Adventure, Star Wars Demolition, Virtua Fighter 3 dan Virtua Tennis 2.

PlayStation 1 & 2


PCSX




Tidak ada emulator PlayStation yang ideal. Tapi PCSX hampir disebut yang terbaik. Pengembang emulator ini melakukan segalanya untuk membuat pemain merasa tenggelam dalam suasana asli Play Station saat bermain dengan PCSX. Agar emulator berfungsi, diperlukan BIOS resmi konsol, serta beberapa plug-in video, plug-in suara. Ada versi emulator untuk Windows, Max OS X, Linux. Ada juga modul terpisah untuk emulator RetroArch universal, modul ini disebut "PCSX-Rearmed." PC yang sangat lemah tidak akan dapat bekerja secara normal dengan emulator ini, dibutuhkan sistem yang kurang lebih modern.

ePSXe




ePSXe - Emulator konsol game PlayStation. Program ini dirilis pada 14 Oktober 2000. ePSXe dilisensikan di bawah freeware.

Seperti kebanyakan emulator modern, ePSXe menggunakan plug-in untuk meniru prosesor grafis dan suara, serta drive optik konsol. Selain itu, untuk menjalankan ePSXe, Anda memerlukan file BIOS yang diperoleh sebagai hasil dari PlayStation dump, untuk alasan ini pengembang emulator tidak dapat dituduh melanggar hak cipta Sony, dan menggunakan emulator dianggap benar-benar sah jika pengguna secara independen menghapus BIOS dari PlayStation pribadinya atau jika ia memiliki Awalan ini mengunduhnya dari Internet.

ePSXe dapat menjalankan game dari CD atau dari berbagai gambar di hard drive. Dengan beberapa pengecualian, program ini dapat menjalankan banyak game untuk PlayStation dengan sempurna; plugin dan pengaturannya berperan.

Pada versi 1.5.x, sekitar 95% gim bekerja dengan baik, dan untuk banyak gim ada tambalan khusus dalam format .ppf yang meningkatkan kompatibilitas atau membantu menyelesaikan masalah tertentu.

PCSX2




Emulator PS2. Program ini masih didukung oleh pengembangnya yang secara aktif mengembangkan proyek. PCSX2 kompatibel dengan sebagian besar game untuk konsol asli. Seperti pada kasus sebelumnya, ini memerlukan BIOS, plug-in untuk video dan audio, drive, serta gambar permainan (ROM / ISO). Emulator dirilis untuk Windows, Mac OS X dan Linux. Tetapi dalam kasus ini, karakteristik PC harus di atas, jika tidak maka tidak akan berfungsi - emulator dan gim membutuhkan banyak sumber daya.

Playstation portable


PPSSPP




Emulator konsol PSP yang hebat yang mendukung sebagian besar game untuk perangkat ini. Jika spesifikasi PC tinggi, maka game akan terlihat lebih baik daripada di konsol. Emulator memungkinkan Anda untuk menggandakan resolusi, yang menghilangkan bagian artefak pada model poligonal. Selain itu, emulator memperbaiki dan menambah beberapa tekstur, menghaluskannya. BIOS tidak diperlukan di sini, ada pengaturan, ada banyak dari mereka, tetapi semuanya berfungsi dengan baik secara default. PPSSPP dirilis di bawah Windows, Max, Linux.

3DS & Wii U


Citra (3DS)




Pengerjaan emulator masih berlangsung, meskipun sudah berfungsi penuh. Sayangnya, beberapa game tidak akan bekerja dengan kecepatan penuh. Bahkan game-game yang mulai normal tidak akan terlihat sempurna. Mungkin di masa depan emulator ini akan bekerja lebih baik - para pengembang melakukan segalanya untuk membuatnya seperti itu. Omong-omong, untuk platform ini tidak begitu mudah untuk mendapatkan gambar dari game, ini adalah masalah lain.

CEMU (Wii U)




Seperti emulator sebelumnya, ini masih dalam tahap awal pengembangan. Sangat sedikit game untuk Wii U yang didukung. Untuk memainkan setidaknya sesuatu, komputer harus sangat kuat. Game seperti Mario Kart 8 dan Twilight Princess HD relatif baik sekarang. Lainnya memberikan layar mulai, tetapi mereka tidak memuat, dan jika mereka boot, mereka tidak akan berfungsi.

Playstation 3


RPCS3




Ini adalah proyek yang relatif muda dari emulator PlayStation 3. Sayangnya, emulator 3D ini tidak mendukung sebagian besar game 3D, dan jika ya, maka game yang kurang lebih normal membutuhkan komputer yang sangat kuat. Tapi game 2D untuk PlayStation 3 berjalan dengan baik - Sonic yang sama, misalnya.

RPCS3 dimulai pada 2013. Pada saat itu, emulator diajarkan untuk menjalankan beberapa sampel yang banyak dari SDK, serta beberapa game buatan sendiri. Semua ini tidak segera, build pertama dari emulator tidak dapat meluncurkan bahkan senjata self-propelled, hanya ada gaya kotak dan tekstur.

Sekarang emulator mereproduksi cukup banyak gim dua dimensi dan beberapa gim tiga dimensi, walaupun secara praktis mustahil untuk memainkan gim yang terakhir. Namun demikian, proyek ini terus berkembang secara aktif, dan segera para pengembang berjanji untuk menunjukkan banyak hal menarik.

Xbox 360


Xenia




Xenia Git adalah emulator desktop Xbox 360 open source. Banyak pengrajin mengambil bagian dalam pengembangannya. Emulator meluncurkan banyak game, apalagi, game 3D, tetapi, sayangnya, sejauh ini tidak akan berfungsi. Emulator ini memiliki kesalahan dan masalah yang jauh lebih sedikit daripada emulator PS3. Mungkin sebentar lagi ia akan meluncurkan banyak game.

Pengembangan emulator dimulai pada 2014, kemudian hanya versi percobaan, yang sama sekali tidak ada yang berhasil. Sekarang kemajuannya sudah jelas - beberapa game diluncurkan, dan mereka memulai tidak buruk.

Emulator universal


Retroarch




Mungkin ini adalah satu-satunya perwakilan dari keluarga emulator universal, yang patut diperhatikan. Sebaliknya, jika Anda menggunakan emulator batch ini, Anda tidak akan membutuhkan yang lain. RetroArch adalah emulator semua-dalam-satu, ia mampu menjalankan gim-gim konsol paling terkenal. Ini mendukung konsol konvensional, Sega, Playstation 1 dan lainnya, dan yang portabel - Game Boy Advance dan yang lebih lama.

Emulator bekerja dengan masing-masing inti, yang masing-masing, pada gilirannya, mengemulasi konsolnya sendiri. Ini berarti bahwa untuk GBA - satu inti, untuk NES - sangat berbeda.
