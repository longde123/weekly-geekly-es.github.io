5 Möglichkeiten, den Raspberry Pi vorteilhaft zu nutzen Teil drei

Hallo Habr.

Im zweiten Teil wurden 5 Möglichkeiten zur Verwendung des Raspberry Pi erläutert. Und wie sich herausstellt, kann dieser Mikrocomputer noch mehr. Nun werden wir, wie ich hoffe, weitere 5 nicht weniger interessante Features des Raspberry Pi betrachten.



Wie im vorherigen Teil werden Methoden betrachtet, für die keine Programmierung erforderlich ist.
Fortsetzung unter dem Schnitt.

Zunächst werde ich die Frage beantworten, die im vorherigen Teil gestellt wurde - welches Modell von Raspberry Pi ist besser zu wählen. In der Tat haben verschiedene Versionen unterschiedliche Leistung, Testergebnisse sind in Google leicht zu finden:



Wie Sie sehen, ist das Raspberry Pi 3 B + und die neueste Version des Raspberry Pi 4 von den modernen Modellen am schnellsten. Der einzige Nachteil des Pi4 ist, dass er einen selteneren Micro-HDMI-Anschluss hat. Das 3B + -Modell ist in dieser Hinsicht universeller. Wenn Sie ein ultrakompaktes Modell benötigen, können Sie in Richtung Raspberry Pi Zero W schauen, aber im Großen und Ganzen würde ich es nicht empfehlen. Erstens ist es, wie Sie aus den Tests ersehen können, merklich langsamer und zweitens hat Zero W (wieder!) Einen anderen Videoanschluss (Mini-HDMI), kein Ethernet und benötigt einen USB-Adapter zum Anschluss einer Tastatur oder Maus. Zero W sollte nur genommen werden, wenn Sie genau wissen, warum Sie ein ultrakompaktes Modell benötigen - die reguläre Version (Pi 3B + oder Pi 4) ist schneller und universeller.

Übrigens, wenn Sie einen ultrakompakten Mikro-PC benötigen, können Sie auf den Nano Pi blicken, der in Form interessanter Modelle erhältlich ist, wie zum Beispiel:



Und jetzt machen wir weiter. Also, 5 Möglichkeiten, den Raspberry Pi zu benutzen.

1. Informationskiosk



Foto von newscrewdriver.com/2019/08/10/raspberry-pi-web-kiosk-boots-faster-on-raspbian-than-ubuntu-core

Die neuesten Versionen von Raspbian OS verfügen über einen vollständig modernen Chromium-Browser, mit dem auch komplexe Webseiten korrekt angezeigt werden können. Wenn Sie also Tickets von Jira, Kurse für die Aktien Ihres Unternehmens, Fahrpläne oder zumindest eine Wettervorhersage auf einem großen Bildschirm anzeigen müssen, wird Raspberry Pi genau das tun.

Es ist ganz einfach und erfordert keine Programmierung. Sie müssen lediglich Chromium im Kiosk-Modus ausführen, den Mauszeiger ausblenden und den Bildschirmschoner ausschalten.

Für Interessierte ein paar Tutorials:

  • pimylifeup.com/raspberry-pi-kiosk
  • blockdev.io/raspberry-pi-2-and-3-chrom-in-kiosk-mode

2. LoRaWAN Gateway




Wenn Sie ein drahtloses LoRaWAN-Netzwerk benötigen, um Daten über LoRa-Funkmodems mit geringem Stromverbrauch zu empfangen, können Sie es basierend auf dem Raspberry Pi einrichten. Es gibt Optionen für verschiedene Preiskategorien, Einkanal-Gateways sind günstiger, 8-Kanal teurer. Der ungefähre Ausgabepreis liegt zwischen 50 und 500 US-Dollar, abhängig vom "Trick" des Gateways.

Eine detaillierte Studie zu LoRa würde den Rahmen dieses Textes sprengen. Materialien für eine detailliertere Studie:

  • learn.adafruit.com/lora-and-lorawan-radio-for-raspberry-pi
  • www.cooking-hacks.com/documentation/tutorials/lorawan-for-arduino-raspberry-pi-waspmote-868-900-915-433-mhz
  • www.thethingsnetwork.org/docs/gateways/start/build.html

3. Internetradio und Internet-TV




Das Thema Audiowiedergabe auf dem Raspberry Pi ist äußerst umfangreich: Sie können eine chinesische USB-Soundkarte für 3,5 USD, ein HiFi-Shield mit vergoldeten Anschlüssen für 35 USD und eine Platine mit Lautsprecher (siehe Abbildung oben) an das Gerät anschließen. Es ist auch einfach, Karten mit optischem Ausgang zu finden, die dem alten Audiosystem sogar moderne Netzwerkfunktionen hinzufügen.

Bei der Software gibt es ebenfalls viele Optionen. Sie können nur einen Internetradiosender fest codieren und wiedergeben, was für Ihre geliebte Großmutter wie ein „Radiopunkt“ aussieht. Sie können ein LCD-Display, Schaltflächen usw. hinzufügen. Mit einem Abonnement können Sie Spotify abspielen.

Interessenten können die folgenden Tutorials sehen:

  • www.jan-holst.dk/pi-radio/pi-radio.html
  • www.hackster.io/Granpino/raspberry-pi-internet-radio-and-mp3-player-with-bluetooth-1aa591
  • learn.pimoroni.com/tutorial/sandyj/streaming-spotify-to-your-pi
  • pimylifeup.com/raspberry-pi-spotify

Für IP-TV wird die Verwendung von Kodi empfohlen - einer speziell für die Verwendung als Media Center zugeschnittenen Software. Ich persönlich habe nicht versucht, IP-TV einzurichten, und ich habe kein kostenpflichtiges Abonnement, aber für diejenigen, die beispielsweise im Ausland leben, aber russischsprachiges Fernsehen sehen möchten, kann dies relevant sein.

Interessenten können auf die Tutorials verweisen:

  • pimylifeup.com/raspberry-pi-kodi
  • www.raspberrypi.org/documentation/usage/kodi

Sie können auch einen IR-Empfänger an den Raspberry Pi anschließen, mit dem Sie die Fernbedienung verwenden können.

4. FM Übermittler



Foto von circuitdigest.com/microcontroller-projects/raspberry-pi-fm-transmitter

Nicht viele Leute wissen es, aber durch programmgesteuertes Ändern der PLL-Frequenz können Sie auf dem Raspberry Pi Funksignale im Bereich bis 500 MHz ohne zusätzliche Hardware übertragen. Schließen Sie einfach ein Stück Kabel an den Pin an, wie auf dem Foto gezeigt, und starten Sie das Programm. Natürlich wird es hier keine HiFi-Qualität geben, und die Ausgangsleistung beträgt nur etwa 10 mW, aber zum Testen von Radiogeräten oder nur zum Spielen von Freunden oder Nachbarn kann dies nützlich sein. Eine Leistung von 10 mW reicht aus, um Musik über das FM-Radio im Raum abzuspielen.

Ich habe diese Technologie vor ein paar Jahren in einem Artikel über Geektimes genauer beschrieben , obwohl dieser Artikel heute schon veraltet ist, kann hier ein besseres Programm für die Übertragung gefunden werden .

5. Flugsteuerung für Drohne




Das nächste Projekt für den Raspberry Pi ist nicht weniger interessant. Viele kennen wahrscheinlich die Controller für Multicopter wie MultiWii, Arducopter oder DJI. Ein ähnlicher Controller kann basierend auf dem Raspberry Pi hergestellt werden. Das NAVIO2- Projekt ist eine spezielle Abschirmung, die Motorsteuerungen steuern kann und die erforderlichen Sensoren an Bord hat (Beschleunigungsmesser, Barometer, GPS usw.).

Es ist schwer zu sagen, wie viel besser ein solcher Haufen ist als der „normale“ Arducopter / Pixhawk, aber für eine Reihe von Aufgaben kann die Idee durchaus vielversprechend sein. Erstens können Sie Telemetrie (oder sogar ein Bild) von einem solchen Kopierer über ein 3G / 4G-Modem steuern oder übertragen, wodurch sich die Reichweite der Verbindung erheblich erhöht. Zweitens kann Raspberry Pi recht komplexe Algorithmen ausführen, wie z. B. einen Kalman-Filter oder eine andere Mathematik. Ich persönlich habe diesen Controller noch nicht ausprobiert, daher kann ich nicht genauer sagen, dass jeder auf Youtube nach Videos von Flügen suchen kann.
