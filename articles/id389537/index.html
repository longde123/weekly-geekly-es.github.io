Cody Wilson akan mengeluarkan file 3D untuk mencetak senapan AR-15



Cody Wilson, seorang anarkis berusia 27 tahun dari Texas, dan perusahaannya Defense Distributed mendapatkan ketenaran setelah merilis senjata plastik Liberator, yang dapat dicetak pada printer 3D. Pistol konyol ini memungkinkan beberapa tembakan untuk ditembakkan sebelum laras meledak. File untuk mencetak senjata Cody diterbitkan di domain publik.

Sekarang dia bermaksud untuk merilis sesuatu yang jauh lebih serius: senapan semi-otomatis AR-15 .

Di dalamnya, bagian dari rincian bagian dicetak pada printer, dan penerima terbuat dari logam pada mesin penggilingan CNC kecil, yang terlihat persis seperti printer 3D desktop. Sekali lagi, file untuk semua bagian, plastik dan logam, Cody akan posting di domain publik, dan biaya pencetakan AR-15, menurut perhitungannya, tidak akan melebihi $ 150.



Meskipun senapan itu sendiri akan murah, tetapi Cody Wilson bermaksud untuk mendapatkan penjualan mesin penggilingan Ghost Gunner seharga $ 1.500. Tetapi pembuatan komponen sepenuhnya otomatis dari file yang akan segera dipublikasikan secara online. Tidak diperlukan pengetahuan teknis atau pelatihan khusus.



Cody Wilson mengatakan dia bisa menerbitkan file paling cepat April. Benar, Anda tidak dapat melakukan beberapa hal bahkan di mesin. Cody mengatakan bahwa untuk mendapatkan peluru yang bagus atau untuk menjadi striker yang bagus, Anda harus mengambil bahan pihak ketiga. Misalnya, paku dapat digunakan sebagai striker. Semua yang lain terbuat dari aluminium dan plastik keras.

Menanggapi tuduhan bahwa teroris dapat menggunakan file-nya untuk membuat senjata, Cody mengatakan: “Ada buku di perpustakaan tentang cara membuat bom. Anda hanya harus memiliki komitmen tertentu untuk pertukaran ide secara gratis. "



Pihak berwenang Amerika tidak terlalu senang dengan aktivitas Cody Wilson. Mereka memaksanya untuk menghapus file pistol Liberator dari online dengan dalih larangan ekspor senjata. Cody menghapusnya, tetapi pada saat itu file sudah menyebar ke situs lain. Sekarang dia menggugat pemerintah, berusaha membuktikan bahwa penerbitan kode di Internet tidak termasuk dalam konsep ekspor senjata.
