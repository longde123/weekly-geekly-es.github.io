英特尔酷睿9代的数量和基准



两周前， 我们推出了新的第九代Intel Core处理器 。 现在是时候从单词转到数字了。 我们再次向Anandtech寻求无穷无尽的专有信息知识源。 他的专家对三种发布的Intel Core 9th Gen处理器进行了全面而全面的比较测试 。 对于没有时间和欲望去研究这份长达23页的基本工作的哈勃拉兹特利，我们将为大家介绍其中最重要，最有趣的部分。

如前所述，第九代英特尔酷睿是更新后的Coffee Lake系列的一部分，也就是说，在所有基本参数上，它都延续了第八代的辉煌传统，增加了许多其他内核和兆赫兹。



顶级型号Core i9-9900K在表中相距甚远。 首先，这是第一个在标准，非极端插槽中发布的i9，与年轻机型相同。 其次，存在8个核心，尽管这并不是完全独特的（发生的更多），但在Core之前，它也是具有X指数和相应的不人道价格的极端模型的属性。 现在，考虑到主板和其他所有东西，“为人们服务的8核”变得更加容易使用。

我们上次没有谈论的另一个趋势是超跑逐渐消失。 专家认为这有两个原因。 首先：按照“核心”对生产线进行排序和整理，以使消费者没有疑问，这更好：6个核心具有超跑或4个没有超跑。 第二：根据最近获得的数据，超跑的存在降低了芯片的安全性，鉴于最近发生的事件，这也很重要。

我要转移到摘要中的另一个有趣的点是按核心列出的Turbo频率。 如前所述，Core i9-9900K可以将两个内核超频到5 GHz，将所有内核超频到4.7。 Core i7-9700K的性能较差，但就“马力”而言，它看起来非常坚固。



现在，在Anandtech的专家的帮助下，我们可以回答已经问过的问题：最多消耗多少新产品，TDP与现实有何关系？ 让我们马上说：第二个问题的答案是负面的。 这是Core i9-9900K消耗量的图表，具体取决于加载的逻辑内核的数量。



如您所见，在95瓦的TDP下 ，最大核心频率下的最大功耗为166瓦。 不幸的是，由于各种核心加速指标，甚至不可能在这两个指标之间建立关系，这进一步降低了TDP对消费者的价值。



如您所见，Core i9-9900K的功耗超过了i9-7900X的功耗，而后者拥有更多的内核和TDP。 俗话说：“放眼世界，不免感到惊讶”。

让我们继续前进。 第一个铭牌已经为我们做好了以下准备：我们不会在这里看到任何令人惊讶的事实。 另一方面，每个人都知道在具有此类参数的地下室中，也不会有新项目。 原则上，我们无法继续阅读，但仍将显示更多幻灯片。

在经典的综合基准PCMark和Geekbench中，我们的新产品运行良好。





英特尔在公告中称Core i9-9900K为“最佳游戏处理器”。 是这样吗 细心的Anandtechs测试了六打游戏以得出结论：在游戏中，第九代产品看起来确实不错。 当然，请注意细微差别-在测试的完整版本中了解它们。 以下是几种具有多种图像质量选项的游戏的测试结果-通过键入而未选择获胜最多的一种。







总而言之，我们可以这样说：购买第九代Intel Core 9不太可能会让您失望。 接下来，问题进入了金融领域，每个领域都有其自己的优先级和演算系统。 好吧，我们不要争论口味。
