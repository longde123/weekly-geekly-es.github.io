世界杯数据中心


距2018年FIFA世界杯仅剩不到一年的时间，这是一个宏伟的足球假期，预计明年夏天将在俄罗斯多个城市举行。 为在世界范围内播出的大规模行动做准备，需要部署强大的IT基础架构。 总计将为世界杯的IT基础设施投资约110亿卢布。 Megafon将负责电信基础设施的建设，为此已拨款42.5亿卢布。

体育场已经在为客人的到来做准备，其中一些已经在上届联合会杯上进行了测试。 为了比较，上届比赛的“扩音器”：

  • 涉及1.1万公里的光纤通信线路（FOCL）；
  • 在体育馆中安装了1.5万多条天线；
  • 在80多个场馆组织了无线电广播，其中60个进行了现代化改造并提供室内覆盖。 其中包括：体育馆，酒店，运营中心，培训设施，售票处，机场和运输站的FIFA机架；
  • 在所有活动设施上提供通信渠道，移动数据传输服务，电话通信；
  • 部署了地理分布的DWDM网络，其元素彼此独立，从而可以从体育场到圣彼得堡国际广播中心提供高质量的电视广播；
  • 保留了三倍的网络，从而保证了其容错能力，并确保了对创建的基础架构的持续监控。 为了监控安装在工厂的设备，使用了自己的生命控制解决方案；
  • 组织了ISDN电话，使评论员可以进行报告。

我们作为NAG公司，也为这一工作海洋做出了一些贡献。 由我们的专家设计和组装的集装箱数据中心现在为过去的联合会杯和即将举行的2018年世界杯体育场之一提供服务。 我想谈谈这个项目。

因此，我们的工程师面临着设计移动数据中心的任务，该数据中心将满足保护设备免受故障影响所必需的所有现代安全和生命支持要求。 此外，有必要提供三台柴油发电机，它们可以自动为数据中心供电。

在数据中心放置设备的基础架构


为了将设备放置在数据中心中，我们使用了19英寸的MetalBox机柜。 机柜的结构包括：穿孔金属门，滚子轴承，门，接地端子和电缆，水平可调支架，安装组件，顶盖，侧面板。









服务器机柜安装在特殊的导轨系统上，可以轻松地在横向方向上移动它们，从而可以从正面和背面访问设备。



电源和不间断电源系统


数据中心内部的配电是通过以下电缆进行的：铜电缆导体由聚氯乙烯（PVC）复合材料制成，绝缘层由聚氯乙烯（PVC）复合材料制成，护套由聚氯乙烯（PVC）复合材料制成，其燃烧系统的易燃性低。





  • 引入最高20 kW的数据中心容量（在4个机架中，指示了不带工程系统的总最大功率）。
  • TDP：36 kW（包括工程系统）；
  • 电源电压：380 V，3L + N + PE;
  • 可靠性类别：第三；

模块化UPS UPS SNR-UPS-ONRM-20-HPMSAX33可为容器提供不间断的电源，并有可能分阶段增加功率。 它由3个容量为20 kVA的模块组成，构成一个冗余级别为N + 1的系统。



HPM系列的UPS（HP）设计为可与大容量外部电池一起使用，是在线构建的，具有双重电压转换功能，在该电压下，负载始终由逆变器供电，同时确保稳定且经过滤波的正弦电源。 UPS旨在保护工作站，服务器，通信系统，工业，电信，银行，网络和其他设备。



UPS上连接了32个Tesla电源电池系统，在100％负载下提供25分钟的电池寿命。 Tesla Power电池设计用于安装在19英寸，23英寸的机柜和机架中，也用于需要紧凑电池放置的其他情况。



前置端子可简化安装并减少电池维护和更换时间。 这些电池在低放电电流下的良好特性使其成为电信系统和其他关键负载的最佳选择，这些电池设计用于延长电池寿命。
