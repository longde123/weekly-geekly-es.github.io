Seleksi segar dengan tautan ke berita dan bahan. Dalam rilisnya: PHP 7.4.0 alpha 1, DevConfX, RFC yang diterima dan baru dari PHP Internals, sebagian dari alat yang berguna, dan banyak lagi.

Selamat membaca!



Berita dan Siaran


  • PHP 7.4.0 alpha 1 - Versi alpha pertama dari cabang PHP baru telah dirilis! Menjelang siklus pengujian enam bulan , rilis final diharapkan pada akhir November.

    Jumlah fitur baru sangat mengesankan: properti yang diketik , fungsi panah , FFI, preload dan banyak lagi! Daftar semua perubahan pada dokumen UPGRADING .
    Gambaran umum fitur-fitur baru dapat ditemukan dalam terjemahan Habra Baru dalam PHP 7.4 atau dalam Laporan oleh Nikita Popov di PHP Rusia .
  • DevConf 2019 (Moskow, 21 Juni) - Peringatan sepuluh tahun DevConf. Secara tradisi, laporan dipilih berdasarkan suara. Dengarkan ulasan aplikasi yang diajukan di podcast. PHP Lima Menit # 54 .

PHP internal


  • [RFC] Izinkan pengecualian melempar dari __toString () - Penawaran diterima dengan suara bulat.
  • [RFC] Pemisah Literal Numerik - Proposal melewati ambang batas untuk pemungutan suara, dan dalam PHP 7.4 akan dimungkinkan untuk menggunakan garis bawah sebagai pemisah dalam angka: 

    $i = 197_823_459; // 197823459
  • [RFC] Panggilan Nullsafe - Sebuah proposal lama sedang dibahas lagi di Internal. Ini mempertimbangkan kemungkinan menambahkan operator baru ?-> , yang akan memungkinkan membuat rantai aman panggilan metode, dalam kasus ketika nilai kembali bisa null : 

     $o?->mayFail1()?->mayFail2()?->mayFail3()?->mayFail4();

    Ada tawaran serupa di ECMAScript , tetapi di Hack fitur ini sudah diterapkan .
  • [RFC] Alternatif "penggunaan" sintaksis untuk Penutupan - Penulis proposal menganggap sintaks penggunaan dalam penutupan tidak terlalu nyaman dan menyarankan untuk mentransfernya ke badan fungsi:

    Teks tersembunyi 
     //   $closure = function ( ArgumentType $argument1, ArgumentType $argument2, ArgumentType $argument3, ArgumentType $argument4 ) use ($importVariable1, &$importVariable2, $importVariable3, &$importVariable4): ReturnType { // ... }; //    $closure = function ( ArgumentType $argument1, ArgumentType $argument2, ArgumentType $argument3, ArgumentType $argument4 ): ReturnType { use $importVariable1, &$importVariable2; use $importVariable3, &$importVariable4; // ... };
  • PHP Internals News # 13 - Sara Goleman (manajer rilis PHP 7.2) dan Derick Rethans (PHP 7.4) berbicara tentang manajemen rilis.
  • PHP Internals News # 14 - Dengan Nikita Popov tentang pengecualian di __toString ().
