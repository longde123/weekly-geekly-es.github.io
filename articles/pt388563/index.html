Red Star é um sistema operacional fechado de um país fechado. Alguns detalhes + vídeo



Na conferência 32c3 , realizada em Hamburgo, foram anunciados alguns detalhes interessantes do funcionamento do sistema operacional norte-coreano, que já estavam escritos em Habré e Geektimes. Este sistema é chamado Red Star. Como se viu, os especialistas norte-coreanos não passaram tanto tempo trabalhando neste sistema operacional (por um total de cerca de 10 anos). Apesar do sistema ser baseado em uma das distribuições Linux, foi possível torná-lo o mais fechado e opaco possível.

Esta não é apenas uma cópia pálida dos sistemas operacionais ocidentais, como muitos de vocês podem decidir. O sistema operacional em si foi analisado por especialistas de uma empresa alemã que trabalha na área de segurança da informação. Curiosamente, o sistema operacional foi baixado de um site fora da Coréia do Norte (como você sabe, não há rede dentro do país, há apenas uma intranet limitada).

A versão mais recente do Red Star foi escrita em 2013 e continua sendo usada no país. O design visual foi feito à imagem e semelhança do OSX - isso provavelmente foi feito sob as instruções de Kim Jong-un, que trabalha em Macs (geralmente esses sistemas são próximos ao líder norte-coreano).

A Wikipedia nos diz o seguinte: “O Red Star OS (Bulgynböl Saönja Yöngchehe) é um sistema operacional norte-coreano baseado em Linux. O início de seu desenvolvimento no Korean Computer Center remonta a 2002. Até aquele momento, as versões norte-coreanas do Windows eram geralmente usadas na Coréia do Norte. A partir de 2010, a versão atual do sistema era 2.0. Somente a versão coreana localizada é distribuída. Em fontes inglesas, é chamado de Red Star OS ".

Os norte-coreanos criaram seu sistema operacional, tomando o Fedora como base. Ao mesmo tempo, muito foi "concluído" por si só, incluindo um sistema de criptografia de arquivos. Isso foi feito para que as forças externas (lemos os americanos) não encontrassem brechas no sistema. Para aprimorar os controles de segurança, os norte-coreanos também desenvolveram um sistema que monitora arquivos. Se algo mudar, o sistema mostra um erro ou falha. Isso acontece, por exemplo, quando você desativa o antivírus ou o firewall - o sistema não permite que você configure parâmetros básicos do sistema. A Red Star também construiu um sistema de proteção contra arquivos de mídia de terceiros (filmes, músicas, livros). Se na Coréia do Norte, eles alteram os dados, em unidades flash, e não pela intranet. E agora, por exemplo, o uso de uma unidade flash USB com arquivos estrangeiros também falha - o sistema "vê2 e controla tudo isso".Tags Red Star ou arquivos de mídia de marcas d'água que são fáceis de rastrear.

Agora na Coréia do Norte, a maioria dos computadores é bastante antiga, com o Windows XP a bordo.



O próprio sistema, de acordo com especialistas que o estudaram, é o mais opaco possível para o usuário. Tudo aqui está configurado para controlar e rastrear ações do usuário. O conjunto de aplicativos é mínimo: um editor de texto, um calendário, um programa para criar músicas e vários outros programas.

Requisitos do sistema:
  • Pentium III 800 MHz
  • 256MB RAM
  • Disco rígido de 3 GB
