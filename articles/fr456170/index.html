Comment créer une application financière: 5 API pour aider le développeur



Bonjour à tous! Je poursuis mes recherches dans le domaine de la finance du point de vue technologique. Dans l' article précédent, nous avons étudié les terminaux de trading existants et leurs fonctionnalités, et aujourd'hui je vous propose de parler, en fait, du développement d'applications financières.

Pour le nouveau sujet, j'ai choisi cinq API de divers services financiers qui vous permettent à la fois de recevoir une grande variété de données et d'échanger directement sur l'échange à l'aide de robots écrits.

Yahoo finance


Peut-être l'un des services financiers les plus connus au monde. En utilisant l'API gratuite, les développeurs peuvent utiliser les données de service sur les cotations boursières, les changements de prix et les mouvements du marché. Des informations sont disponibles sur une variété de titres ainsi que sur d'autres instruments, tels que les paires de devises.

Vous pouvez vous connecter très facilement à l'API à l'aide du service d'API rapide.



SMARTcom


API pour créer une application de trading sur les bourses russes. Les développeurs peuvent utiliser l'API comme une couche entre le système de trading d'échange et leur système de trading prêt à l'emploi (créé indépendamment ou acheté), créer leur propre robot de trading, ou même un terminal de trading complet avec une interface «personnalisée».

L'accès à l'API est gratuit, il vous suffit de créer un compte dans le système du courtier qui l'a développé. Il est possible de déboguer le travail de votre logiciel en utilisant l' accès de test dans le trading virtuel.

Synchroniser par livre de paie


Un service utile aux développeurs d'applications financières pour les marchés étrangers. Il vous permet de collecter en un seul endroit les données des banques, des sociétés financières et des services publics, des agences gouvernementales de différents pays. Ce service est payant, les frais d'accès à partir de 50 $. La liste des services disponibles pour l'intégration est disponible ici .



API Currencylayer


Un service utile pour obtenir des informations sur les taux de change et la valeur de 168 monnaies mondiales. Les données sont collectées à partir de nombreuses sources, telles que les sites bancaires, les services d'informations commerciales. Les développeurs promettent la compatibilité avec des applications de tout type et de tout langage de programmation.

Il existe une version gratuite limitée de l'API (250 demandes par mois), une version plus adéquate commence à 9,99 $ (10 000 demandes par mois). Leurs solutions basées sur l'API Currencylayer sont développées par des sociétés telles que Lyft, Blizzard, Intel.



Finance facile


Le service bien connu pour la comptabilité des finances personnelles dans Runet fournit une API pour les développeurs tiers. Comme indiqué sur le site Web du projet, les applications tierces pour les réseaux sociaux, les applications mobiles et les plug-ins pour les navigateurs sont les plus populaires auprès des utilisateurs de la ressource.

Les applications sont testées par l'équipe de service, celles approuvées entrent dans un marché spécial et sont distribuées par abonnement. Selon les statistiques de l'administration d'EasyFinance, les développeurs d'applications peuvent gagner 15 à 50 roubles par utilisateur et par mois. Ces revenus sont constitués des revenus de l'abonnement aux tarifs du service lui-même, des paiements d'intégration des partenaires de la plateforme et de la publicité.
