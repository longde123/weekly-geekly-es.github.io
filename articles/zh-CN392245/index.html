优步正在寻找在其应用程序中使用迷你游戏的开发人员



屏幕上出现新通知时，微软工程师约书亚·德布纳（Joshua Debner）在西雅图使用了Uber。该应用程序提供了玩“ Code on the Road”游戏。该公司证实，开发商已经发现了一趟尤伯杯过程中测试他们的技能，寻找你的成长中的公司的新方式，根据商业内幕。

Debner不知道公司如何得知他是一名工程师。他的Microsoft电子邮件地址未绑定到该应用程序。优步坚持认为，它不会使用个人数据来识别候选人，而只会识别技术专家集中的地理区域。

“我们一直在寻找新方法来吸引想要加入我们团队并帮助我们解决有趣问题的潜在候选人。公司代表说：“如果您所在的地方有很多IT领域的工作人员，那么您可以看到我们面临的挑战” Code on the Road”。


“道路上的密码”游戏的屏幕截图

如果一个人参加了考试，那么Uber可以解决3个问题，每个问题需要60秒。如果答案是正确的，则应用程序会提供一个按钮来与公司代表直接联系。

戴布纳接受了提出他的优步成绩的提议，只是为了看看最终会发生什么。该公司给他发了一封信，邀请他关注该链接，以防有人有意考虑空缺。Uber的HR项目已在西雅图，奥斯丁，波士顿，丹佛和波特兰等城市启动，这些城市拥有许多技术专家。

Uber并不是第一家将测试和员工搜索机制引入其产品的公司。例如，Google还具有自己的秘密编程知识测试，该测试在人们搜索有关特定主题的信息时出现。
