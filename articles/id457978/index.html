18 Trik untuk Bekerja dengan Visual Studio

Terlepas dari apakah Anda seorang pemula atau telah menggunakan Visual Studio selama bertahun-tahun, ada banyak tips dan trik untuk membantu Anda menjadi lebih produktif. Untuk beberapa waktu kami berbagi kiat di Twitter menggunakan tagar #vstip , dan hari ini kami telah menyusun pilihan yang terbaik dari mereka hari ini.

Debugger


Menggunakan F10 alih-alih F5 untuk membangun, menjalankan, dan melampirkan debugger secara otomatis terputus saat pertama kali Anda menjalankan kode Anda sendiri. Breakpoint tidak diperlukan.



Didukung sejak Visual Studio 2005

Pasang kembali ke proses (Shift + Alt + P) adalah fitur yang sangat berguna yang membantu Anda bergabung dengan proses yang sama berulang-ulang.



Didukung sejak Visual Studio 2017 v15.8

Titik biru di lapangan menunjukkan pengalihan thread saat debugging.



Didukung sejak Visual Studio 2013

Berlanjut di bawah potongan!

Solusi


Meningkatkan kinerja pemuatan solusi dan mengurangi kebisingan visual dengan menonaktifkan pemulihan ekstensi node di Solution Explorer, serta Buka kembali dokumen pada pemuatan solusi .



Didukung sejak Visual Studio 2019

Untuk navigasi cepat, gunakan Ctrl + T untuk menemukan sesuatu dalam solusi Anda - file, kelas, dll.



Didukung sejak Visual Studio 2017

Tetapkan pintasan keyboard untuk melakukan "git pull" sehingga Anda tidak harus menggunakan CLI atau Team Explorer untuk ini.



Didukung sejak Visual Studio 2019

Berikan Solusi Explorer kemampuan untuk secara otomatis memilih dokumen yang sedang aktif sehingga Anda tidak pernah kehilangan lokasinya di proyek.



Didukung sejak Visual Studio 2010

Editor


Sorot elemen HTML dengan mudah dengan tag div menggunakan kombinasi Shift + Alt + W. Div yang dimasukkan dapat Anda edit dengan mudah ke tag apa pun yang Anda inginkan, dan tag penutup akan berubah secara otomatis.



Didukung sejak Visual Studio 2017

Salin setiap fragmen JSON ke clipboard dan rekatkan sebagai kelas .NET yang diketik dengan kuat ke dalam file C # atau VB.



Didukung sejak Visual Studio 2013

Anda tidak perlu menempatkan tanda kutip ganda di sekitar nama properti JSON sendiri - cukup masukkan titik dua dan Visual Studio akan secara otomatis menempatkannya.



Didukung sejak Visual Studio 2015

Jadikan IntelliSense dan tooltips transparan saat Anda menekan dan menahan Control.



Didukung sejak Visual Studio 2010

Alih-alih memasukkan kembali '(' untuk menampilkan informasi parameter dalam tanda tangan metode, gunakan Ctrl + Shift + Spasi untuk menampilkan kelebihan beban yang saat ini digunakan.



Didukung sejak Visual Studio 2010

Lainnya


Putar suara saat peristiwa tertentu terjadi di Visual Studio.



Didukung sejak Visual Studio 2010

Buat tata letak jendela Anda sendiri untuk skenario pengembangan spesifik atau pengaturan monitor, serta mudah beralih di antaranya.



Didukung sejak Visual Studio 2017

Tentukan komponen Visual Studio yang diperlukan untuk solusi apa pun, dan Visual Studio akan meminta pengguna untuk menginstalnya jika ada yang hilang. Untuk informasi lebih lanjut, lihat artikel " Konfigurasikan Visual Studio di organisasi Anda dengan .vsconfig ."



Didukung sejak Visual Studio 2019

Ekstensi


Visual Studio Pemeriksa Ejaan . Ekstensi editor yang memeriksa ejaan komentar, baris, dan teks biasa. Itu juga dapat memeriksa ejaan seluruh solusi, proyek, atau item yang dipilih. Opsi tersedia untuk menentukan beberapa bahasa untuk pemeriksaan ejaan.



Didukung sejak Visual Studio 2013

Tambahkan File Baru . Ekstensi untuk Visual Studio yang memungkinkan Anda untuk dengan mudah menambahkan file baru ke proyek apa pun. Cukup tekan Shift + F2 untuk membuat file kosong di folder yang dipilih atau di folder yang sama dengan file yang dipilih.



Didukung sejak Visual Studio 2015

Git Diff Margin . Git Diff Margin menampilkan perubahan Git saat ini di file yang dapat diedit di lapangan dan di bilah gulir Visual Studio.



Didukung sejak Visual Studio 2012

Ini hanya beberapa dari ribuan ekstensi yang tersedia. Untuk melihat lebih banyak, pergi ke Visual Studio Marketplace .

Kesimpulannya
