So erstellen Sie Websites im Jahr 2019

Das Web verändert sich schnell, bleiben Sie auf dem Laufenden!



Ich habe 2018 einen ähnlichen Artikel geschrieben, aber das Web ändert sich so schnell, dass diese Richtlinien hoffnungslos veraltet sind.


Werfen wir Schritt für Schritt einen Blick auf die wichtigsten Prinzipien des Webdesigns im Jahr 2019 .


Stellen Sie sicher, dass der Benutzer weiß, dass die Site über eine mobile Anwendung verfügt


Dies ist ein wesentliches Prinzip, und dafür gibt es Gründe. Die Fokusgruppen, für die wir bereits viel Geld ausgegeben haben, haben gezeigt, dass das erste, was ein Benutzer beim Erreichen Ihrer Website möchte, die Installation einer mobilen Anwendung ist.


Der kürzeste Weg dazu besteht darin, ein kräftiges Modalfenster zu zeigen:



Optional können Sie eine Schaltfläche oder einen Link hinzufügen, um dieses Fenster zu schließen. Es ist jedoch von grundlegender Bedeutung, dass der darauf enthaltene Text schwach und so formuliert ist, als ob sich der Benutzer schämen sollte, ihn abzulehnen, oder dass er die Anwendung in Zukunft installieren möchte.


Tipp: Wenn Sie keine mobile Anwendung haben, spielt das keine Rolle! Überfüllen Sie einfach die gesamte Website in Webview mit deaktivierter Sicherheit und schreiben Sie, dass dies eine mobile Anwendung ist!


Entwickeln Sie eine Strategie für Do Not Track


Die meisten modernen Browser (damit sie!) Unterstützen den DNT-HTTP-Header, der für Do Not Track steht. Dieser Header signalisiert, dass der Benutzer nicht verfolgt werden möchte. Leider ist dieser Header in einigen Browsern wie Brave ( Open-Source-Browser mit Werbeblockierung durch den JavaScript-Ersteller Brendan Eich ) standardmäßig aktiviert.


Keine Panik! Es stellte sich heraus, dass die meisten Benutzer den Header ausschalten, sobald Sie sie fragen, und damit alles wie am Schnürchen läuft, fügen Sie detaillierte Anweisungen dazu bei:



Tipp: Fragen Sie im Handbuch zum Deaktivieren von DNT nicht, ob der Benutzer DNT deaktivieren möchte. Andernfalls stimmen Sie weiterhin zu.


Entwickeln Sie eine Strategie für Cook


Es ist nicht notwendig, nach Cookies zu fragen, wenn diese aus angemessenen Gründen auf der Website verwendet werden. Es ist jedoch viel besser, die Erlaubnis für alle Cookies in großen Mengen anzufordern. Dann können wir alles verfolgen, was wir wollen, ohne Angst vor Konsequenzen zu haben!



Tipp: Laden Sie keine Seiten ohne Cookies, auch wenn diese nicht für die Arbeit benötigt werden, da Sie dann viel weniger persönliche Informationen nachverfolgen können, die Sie dann verkaufen können.


Blocker blockieren


Werbeblocker gibt es überall und sie wirken sich auf die Gewinne aus. Ohne Werbeblocker zu blockieren, nirgendwo!



Chrome selbst plant, die Blocker zu blockieren (es ist meins!). Wenn sich also jemand bei Brave anmeldet, können Sie mit Sicherheit sagen, dass sein Browser veraltet ist:



Tipp: Erzählen Sie niemandem von Brave, einem Open-Source-Chromium-Browser mit integrierter Werbeblockierung .


Erzählen Sie auch niemandem von Firefox , Opera oder Vivaldi - dem Rat des Übersetzers.


Blockieren Sie Anrufe aus anderen Ländern


Bei all diesen Gesetzen, der DSGVO , den Verbindungssteuern und dem Huawei-Verbot (in den USA - ca. übersetzt) ist es zwar immer noch nicht klar, was es bedeutet, aber es ist besser, sie einfach zu nehmen und sie alle vor Schaden zu bewahren.

Machen Sie so viel wie möglich ein Layout


Die Zeiten von 800-Pixel-Layouts sind lange vorbei. Jetzt gibt es immer mehr Breitbild-Displays. Versuchen Sie also, so viel Platz wie möglich zu nutzen.



Tipp: Studien zeigen, dass modale Fenster ohne Schließen-Taste viel seltener schließen!


Verwenden Sie Benachrichtigungen


Moderne Browser unterstützen Benachrichtigungen. Dies bedeutet, dass sie aktiviert sein müssen. Wenn dies jedoch nicht funktioniert, können Sie immer versuchen, sich standardmäßig für eine Mailingliste oder eine SMS anzumelden.



Tipp : Mit diesen Benachrichtigungen können Sie Benutzer auch über eine neue Version einer mobilen Anwendung informieren.


Benutzer daran erinnern


Manchmal vergessen Benutzer, dass sie sich registrieren können. Erinnern Sie sich daher regelmäßig daran und zeigen Sie einen großen Link oder eine Schaltfläche an:



Tipp: Vergessen Sie nicht, den Benutzer daran zu erinnern, sobald sein Cursor den Site-Bereich überschreitet.


Geben Sie dem Benutzer die Option zum Deaktivieren


Es ist sehr wichtig, benutzerfreundlich und unauffällig zu sein, was bedeutet, dass wir die Möglichkeit geben müssen, auf ständige Erinnerungen, Newsletter und Fenster zu verzichten.


In diesem Fall müssen Sie die Optionen dort platzieren, wo sie am auffälligsten sind, z. B. in einem der Unterabschnitte der „Kontoeinstellungen“:



Verwenden Sie Javascript


Es ist bekannt, dass Sie immer JavaScript verwenden sollten.


Nein, Sie könnten natürlich denken: "Aber warum, weil ich nur Text und ein paar Bilder habe?" Hast du an die Zukunft gedacht? Zögern Sie nicht, bald werden Sie eine Reihe von modalen Fenstern haben. In diesem Fall werden Sie hundertmal bedauern, dass Sie das trendige JS-Framework, das in einer Cloud ohne Server ausgeführt wird, nicht verwendet haben.



Tipp: Rendern Sie diese Seite nicht speziell mit Javascript.
