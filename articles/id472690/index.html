Apa yang Baru di Zabbix 4.4

Tim Zabbix dengan bangga mengumumkan rilis Zabbix 4.4. Versi terbaru hadir dengan agen Zabbix baru yang ditulis dalam Go, menetapkan standar untuk templat Zabbix, dan menyediakan kemampuan visualisasi tingkat lanjut.



Mari kita lihat fitur paling penting yang termasuk dalam Zabbix 4.4.

Agen Zabbix generasi baru



Zabbix 4.4 memperkenalkan jenis agen baru, zabbix_agent2, yang menawarkan berbagai fitur baru dan fitur pemantauan canggih:

  • Itu ditulis dalam Go.
  • Kerangka kerja plugin untuk memantau berbagai layanan dan aplikasi.
  • Kemampuan untuk mempertahankan keadaan di antara pemeriksaan (misalnya, untuk mempertahankan koneksi database persisten).
  • Penjadwal bawaan untuk mendukung slot waktu yang fleksibel.
  • Penggunaan jaringan yang efisien melalui transfer data dalam jumlah besar.
  • Saat ini, agen berjalan di Linux, tetapi dalam waktu dekat kami akan membuatnya tersedia untuk platform lain.

Kait web dan logika tindakan / pemberitahuan yang dapat diprogram

Integrasi dengan pemberitahuan eksternal dan sistem penerbitan tiket telah ditingkatkan secara signifikan, yang memungkinkan kami untuk mendefinisikan semua logika pemrosesan menggunakan mesin JavaScript bawaan. Fungsionalitas ini menyederhanakan integrasi dua arah dengan sistem eksternal, memungkinkan Anda untuk mengakses rekaman di sistem tiket Anda dari antarmuka pengguna Zabbix dengan satu klik, menghasilkan pesan obrolan dan banyak lagi.

Menetapkan standar untuk template Zabbix

Kami telah memperkenalkan sejumlah standar dan pedoman yang jelas untuk membuat template.

Struktur file XML / JSON telah sangat disederhanakan, yang memungkinkan Anda untuk mengedit template secara manual hanya menggunakan editor teks. Sebagian besar template yang ada telah disempurnakan agar sesuai dengan standar baru.

Dukungan resmi untuk TimescaleDB

Selain MySQL, PostgreSQL, Oracle, dan DB2, kami sekarang secara resmi mendukung TimescaleDB. TimescaleDB memberikan tingkat kinerja yang hampir linier, serta penghapusan instan otomatis dari data historis lama.

Dalam posting ini, kami membandingkan kinerja dengan PostgreSQL.

Basis Pengetahuan tentang Elemen Data dan Pemicu



Zabbix 4.4 menawarkan deskripsi yang jauh lebih visual tentang elemen dan pemicu data. Informasi ini sangat membantu para insinyur, memberikan mereka semua perincian yang mungkin tentang arti dan tujuan barang yang dikumpulkan, perincian masalah dan instruksi untuk menyelesaikannya.

Visualisasi Tingkat Lanjut



Bilah alat dan widget terkait telah ditingkatkan dengan berbagai cara, yang menyederhanakan pembuatan dan pengelolaannya, dan juga menambahkan kemampuan untuk mengubah pengaturan widget dengan satu klik. Ukuran panggangan dasbor sekarang cocok untuk mendukung layar lebar dan layar besar.

Widget tampilan masalah telah ditingkatkan untuk mendukung presentasi agregat, dan widget baru telah diperkenalkan untuk menampilkan grafik prototipe.

Selain itu, sekarang semua widget dapat ditampilkan dalam mode tanpa kepala.

Grafik batang dan agregasi data



Zabbix 4.4 mendukung histogram, dan widget grafik sekarang dapat menggabungkan data menggunakan berbagai fungsi agregat. Kombinasi dari kedua fitur ini sangat memudahkan analisis data jangka panjang dan perencanaan bandwidth.

Dukungan resmi untuk platform baru


Zabbix 4.4 sekarang berfungsi pada platform berikut:

  • SUSE Linux Enterprise Server 15
  • Debian 10
  • Raspbian 10
  • RHEL 8
  • Agen untuk Mac OS / X
  • Agen MSI untuk Windows

Semua platform yang tersedia dapat ditemukan di bagian unduhan .

Instalasi cloud satu-klik

Zabbix dapat dengan mudah diinstal sebagai wadah atau sebagai disk image yang siap digunakan pada berbagai layanan cloud:

  • Aws
  • Azure
  • Platform cloud Google
  • Lautan digital
  • Docker

Registrasi otomatis andal



Versi baru Zabbix memungkinkan penggunaan enkripsi PSK untuk pendaftaran otomatis dengan pengaturan enkripsi otomatis untuk host yang ditambahkan. Sekarang Anda dapat mengonfigurasi Zabbix untuk memungkinkan pendaftaran otomatis perangkat jaringan hanya menggunakan PSK, hanya tidak dienkripsi, atau keduanya.

JSONPath tingkat lanjut untuk preprocessing



Zabbix sekarang mendukung sintaks JSONPath canggih, yang memungkinkan preprocessing data JSON yang rumit, termasuk agregasi dan pencarian. Preprocessing juga dapat digunakan untuk deteksi tingkat rendah, menjadikannya alat yang sangat kuat untuk otomatisasi dan deteksi.

Deskripsi Makro Pengguna



Makro khusus memberikan fungsionalitas yang sangat baik yang menyederhanakan konfigurasi Zabbix dan sangat menyederhanakan perubahan konfigurasi. Dukungan untuk deskripsi makro khusus akan membantu mendokumentasikan tujuan setiap makro, yang akan sangat menyederhanakan manajemennya.

Pengumpulan data lanjutan yang lebih efisien



Pengumpulan data dan penemuan objek yang terkait dengan WMI, JMX, dan ODBC telah ditingkatkan dengan pemeriksaan baru yang mengembalikan array objek dalam format JSON. Kami juga menambahkan dukungan untuk gudang data VMWare untuk memantau VMWare dan layanan systemd untuk platform Linux, serta jenis preprocessing baru untuk mengkonversi CSV ke JSON.

Fitur Baru dan Yang Ditingkatkan Lainnya dari Zabbix 4.4

  • Memproses Data XML dari LLD
  • Jumlah maksimum item dependen telah meningkat menjadi 10 ribu.
  • Menambahkan konversi tipe otomatis ke preproses JSONPath
  • Nama host termasuk dalam file ekspor secara real time.
  • Windows Agent sekarang mendukung penghitung kinerja bahasa Inggris
  • Kemampuan untuk mengabaikan nilai dalam preprocessing on error
  • Data terbaru telah diperluas untuk memberikan akses tidak hanya ke data historis, tetapi juga data operasional.
  • Kemampuan untuk mengedit deskripsi pemicu telah dihapus, akses ke sana telah sangat disederhanakan
  • Dukungan yang dihapus untuk Jabber dan Eztexting jenis file media bawaan, menggunakan kait web atau skrip eksternal
  • Dasbor default yang diperbarui
  • Host yang terdaftar secara otomatis mendapat kesempatan untuk menentukan opsi "connect to dns" atau "connect to IP"
  • Menambahkan dukungan makro {EVENT.ID} untuk URL pemicu
  • Elemen layar "Layar" tidak lagi didukung
  • Jenis widget dasbor yang terakhir dibuat akan diingat dan digunakan kembali nanti
  • Visibilitas header widget dapat disesuaikan untuk setiap widget.

Seluruh daftar fitur baru Zabbix 4.4 dapat ditemukan di catatan untuk versi baru .
