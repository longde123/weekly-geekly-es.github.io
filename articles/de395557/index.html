# 2 Video Slider Control Unit

Hallo Habravchane! Erinnern Sie sich an meinen motorisierten Video-Schieberegler aus Möbelbeschlägen, Bauvorschriften und Plexiglas?)
Heute werde ich Ihnen eine neue Sache zeigen, die etwas einfacher ist als das vorherige Projekt. Diesmal ohne Schienen, nur der elektrische Teil.
Lesen Sie unten!

Hintergrund.
Sein motorisiertes videoslayder ich am Ende Juli letztes Jahr gesagt, wenn es nur ein Monat vor der Studie war, und es gab eine Situation „weder dort noch hier.“ Ich habe nur ein paar Sechs-Sekunden-Szenen auf meinem Slider gedreht und das wars. Dann der erste Kurs an der Universität, eine Infusion in das Studentenleben usw. Im Allgemeinen war ich dem Schieberegler nicht gewachsen. Ich bin über das Wochenende nach Hause gefahren. Wenn noch Zeit war, ziehen Sie einen Schieber heraus. Ich bin ein paar Mal in die Natur gegangen, um den Test zu machen. Aber das brauchen wir nicht. Um Schönheit zu fotografieren, müssen Sie mit dem Auto weit weg sein und nicht an einem freien Tag, sondern länger.
Herbst, Winter, Frühling ...
Mai. Irgendwie kommt eine Nachricht zu mir: „Hallo! Sieh dir deinen Artikel über einen motorisierten Video-Slider anbei Gigtime hat es mir gefallen! Ich beschäftige mich mit Videofilmen und möchte meinen Slider schon lange motorisieren, aber ich habe in Minsk keine Person gefunden , die die Laufwerke für sie herstellen würde. Wirst du helfen?) "
" Natürlich! " Ich antwortete.

Ich kam zum Kunden nach Hause, schaute auf seinen Schieberegler (Industrie, IGUS- System ), sprach darüber und diskutierte, was er vom Laufwerk will. Der Kunde Sergey wollte zunächst nur einen Gleitwagen motorisieren.
"Gut wie du willst", sage ich. - Vorauszahlung im Voraus! 2-3 Wochen, und ich werde es bringen. "

Zuerst machte ich mich an die Herstellung des Steuergerätegehäuses. Material, wie im ersten Entwurf, Plexiglas. Gut bewiesen! Langlebig, leicht, perfekt geschnitten (Bügelsäge mit kleinem Zahn) und verarbeitet.
Sie können es kleben!



Also, Eisen. Basierend auf dem Mikrocontroller Arduino Nano .



Der Motor wird von EasyDriver angetrieben . Nema 17



Schrittmotor . Wir steuern die Geschwindigkeit mit einem Potentiometer . Leistung organisiert auf Basis der Batteriegröße 18650 . Dieses Mal wurde der Anschluss zum Anschließen des Motors aktualisiert . Anstelle der sowjetischen DIN wurde MIC 4 PIN verwendet . Funktionell .
























Der Kunde bat nur um eine Betriebsart - der Schlitten fährt unendlich hin und her, die Geschwindigkeit kann jederzeit mit einem Potentiometer geändert werden. Bei Bedarf halten wir den Wagen mit dem Kippschalter an.

Skizze:
#define xy A0
int Distance = 0;  
int x = 0;
void setup() {                
  pinMode(2, OUTPUT);     
  pinMode(3, OUTPUT);
  digitalWrite(2, LOW);
  digitalWrite(3, LOW);
  pinMode(xy, INPUT);
  
}

void loop() 
{
   x = analogRead(xy); 
x = map(x, 20, 700, 10, 4000); 

  digitalWrite(3, HIGH);
  delayMicroseconds(x);          
  digitalWrite(3, LOW); 
  delayMicroseconds(x);
  Distance = Distance + 1;   
  
  
 if (Distance == 1600*13.8)
  {
  
    if (digitalRead(2) == LOW)
    {
      digitalWrite(2, HIGH);
    }
    else
    {
      digitalWrite(2, LOW);
    }

    Distance = 0;
   
    delay(2500);
  }
}
