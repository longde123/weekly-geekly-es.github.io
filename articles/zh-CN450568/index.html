值得认识到的是，如今的解毒剂已经失去了意义，通常在普通人的记忆中，解毒剂与民防课程（相应的主题（恐怖分子等）的电影）（ 旧研究机构的墙上的海报）相关联。 发生这种情况是因为急诊医学的重症监护达到了前所未有的高度。 通常，对所有中毒进行对症评估和治疗，依次阻止或消除有害因素。 然而，解毒剂的使用在减少中毒时间和减少并发症方面具有优势，增加了中毒患者康复的可能性并节省了他的重要资源。 当无法获得全面的重症监护的机会时，某些解毒剂可能成为重要药物，尤其是在偏远地区或发展中地区。

直到某个时候，所有解毒剂都没有被分类并且完全分开存在。 但是情况在1993年发生了变化，首先，在世界卫生组织国际化学品安全规划（IPCS）的框架内，对解毒剂的概念进行了新定义：

甲醇的特定解毒剂是乙醇。 由于甲醇代谢缓慢，因此从消耗甲醇之日起5天内要摄入乙醇。 乙醇剂量：每天1-2克/千克重量。 血液中乙醇的最佳最佳浓度为1‰。 乙醇以5％溶液的形式静脉注射（在无意识或呕吐的情况下）（20毫升96％溶液在400毫升5％葡萄糖溶液中的比例为100-150毫克/千克/小时）或30％口服溶液每3小时一次，而每日剂量则在各剂量之间平均分配。 为了加速甲酸的代谢，每天以50-100 mg的叶酸施用4-6次。 在乙二醇中毒的情况下，作用是相同的。 第一次机会-立即去医院，并说明所进行的操作。

我们将Lugol的溶液添加到正在研究的酒精中，将其混合，然后逐滴添加碱溶液（NaOH）。 在乙醇的情况下，溶液首先脱色，然后变浑浊，形成碘仿的黄色悬浮液，在高酒精浓度下形成黄色沉淀。 甲醇-不发生这种反应。

第二种选择可以是醇氧化与氧化铜的反应。 磨碎以发光的铜线在燃烧器火焰中煅烧直至变黑，然后将其降低到测试酒精中。 在甲醇的情况下，反应进行：

CH 3 OH + CuO => H 2 C = O + Cu + H 2 O（形成甲醛，并且导线变得发亮）

在乙醇的情况下，反应进行：

在2 H 5 OH + CuO => CH 3 -CH = O + Cu + H 2 O的情况下（形成乙醛，导线变得发亮）

测试人员必须知道纯醛的气味是很复杂的方法（醋-它使人想起漂亮的苹果的气味，有人闻到浓浓的烟味，甲醛刺激鼻粘膜，这种气味非常刺鼻，例如在分解酚醛树脂时可以感觉到）

5CH 3 OH + 3H 3 PO 4 + 2KMnO 4 => 5HCOH + 2MnHPO 4 + K 2 HPO 4 + 8H 2 O

由甲醇形成的甲醛与变色酸呈紫色。 乙醛不会干扰反应。
