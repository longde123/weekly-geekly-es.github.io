Instale o Metatrader 4 no VPS no Debian 8

Muitos de nossos clientes instalam terminais de negociação em servidores virtuais. Isso é bastante conveniente e os benefícios são óbvios. Você não precisa ficar na frente do computador o tempo todo, o servidor VPS está constantemente on-line e o terminal de negociação está sempre ativado.


Devido ao interesse geral de nossos clientes no terminal MT4 em conjunto com os sistemas Linux, decidimos escrever como instalar o Metatrader 4 em um servidor virtual com acesso via x2go.

Lado do servidor


Após conectar-se ao servidor via ssh, você deve executar as seguintes etapas.

Adicione repositórios com x2go:

$ echo "deb http://packages.x2go.org/debian jessie main" > /etc/apt/sources.list.d/x2go.list
$ echo "deb-src http://packages.x2go.org/debian jessie main" >> /etc/apt/sources.list.d/x2go.list
$ apt-key adv --recv-keys --keyserver keys.gnupg.net E1F958385BFE2B6E
$ apt-get update && apt-get install x2go-keyring && apt-get update

Instale o x2go:

$ apt-get install x2goserver x2goserver-xsession

Edite o arquivo .profile no seu diretório pessoal. Você precisa encontrar a linha "mesg n"e substituí-la por "tty -s && mesg n":

$ vi.profile

Instale o navegador:

$ apt-get install iceweasel

Crie um arquivo para iniciar a instalação do Metatrader. No estágio atual, não funcionará. Este arquivo será iniciado um pouco mais tarde, a partir do cliente:

$ printf '#!/bin/bash\n\nwine \"Downloads/OBRTraderSetup.exe\"\n' > mtinstaller.sh

O comando a seguir cria um arquivo para executar o Metatrader ainda não instalado:

$ printf '#!/bin/bash\n\nwine \"/root/.wine/drive_c/Program Files/OBR Forex/OBR Trader/terminal.exe\"\n' > mtrunner.sh

Esses arquivos devem ser executáveis:

$ chmod +x mtrunner.sh && chmod +x mtinstaller.sh

Agora coloque vinho:

$ dpkg --add-architecture i386 && apt update && apt-get install wine wine32

Isso conclui o trabalho no ssh, você pode prosseguir para se conectar à sua máquina remota através do x2goclient.

Clicamos na sessão “Debian-Firefox”, inserimos a senha, uma janela do navegador é aberta em execução no nosso VPS, acessamos o site forex e efetuamos login. No x2to, a área de transferência funciona bem, para que os links possam ser copiados.


Agora você precisa instalar o Metatrader 4 em nosso VPS . Faça o download do programa no site . Enquanto estiver balançando, você pode abrir uma "Conta Demo"; para o correio. Ao se registrar, todos os dados necessários na conta serão recebidos. O navegador não é mais necessário. Se você não deseja fechá-lo, mas deseja retornar às guias abertas posteriormente, no cliente x2go, clique no botão destacado em vermelho. Da mesma forma, para deixar o Metatrader aberto, mas desconecte-se do VPS, você precisa usar este botão.


Iniciamos a instalação do Metatrader 4. Clique na sessão “Debian-Metatrader4-Installer”, digite a senha do VDS e siga as instruções na tela.

Só isso. Agora você pode abrir a sessão x2go “Debian-Metatrader4” e o Metatrader iniciará em uma máquina remota. Se você deseja deixá-lo em execução, não feche a janela, use o botão de pausa na janela x2go do cliente. Tenha um bom comércio!

PS Além deste método de conexão com o VDS e instalação do Metatrader, você pode usar alguma outra opção de um artigo recente no Habr , mas parece-nos que esse é o método mais preferido, porque não há custos de RAM para executar programas desnecessários.

Para operação estável do MT-4, a seguinte configuração de servidor será suficiente:
CPU 1 x2,2 GHz, RAM 1 GB, DISCO 20 GB, um endereço IP dedicado
