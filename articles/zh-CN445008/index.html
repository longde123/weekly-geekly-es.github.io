Azure考试（修订和修订版）

在2018年9月的活动上， 宣布出现了AZ-***系列的新考试，而70-53 *系列则被遗忘了。

我早先的文章已经在这里进行了描述。 但此后发生了很大变化，本文包括2018年12月，2019年3月和2019年4月的更改，这些文章中也未考虑这些更改。



本文中 ，将提供准备在Azure Dev / Infra /体系结构上提交的资源列表。


从根本上改变了什么？


  • 用新考试完全取代现有考试肯定是一个重大变化，但这并没有给我留下深刻的印象。
  • 对于体系结构，有2项考试而不是1项考试；这通常不会有太大变化，因为 要获得建筑师的证书，您必须同时通过两者。 它只是更多的考试。
  • Azure DevOps （ex VSTS，ex VSO）和Azure安全技术考试
  • 最大的创新是增加了实用部分（很多但不是全部。例如，在az-203中未添加az-301），它们使您可以访问真正的Azure门户，给您真正的订阅，工作的帐户和要完成的几个任务。 这真是超级 考试开始考验一个人的实践能力。
    • 根据个人的变更经验，如果您有使用Azure的经验，那么实验室部分将使变更更容易，而不是更难，因为 您可能不会立即记住该问题的正确答案，但是在实际部分中，通过在Azure中设置环境，您会找到它。

假人Azure或AZ-900


AZ-900的发布要晚于其余的，但我相信这是要做的第一件事。 AZ-900包含所有基本的Azure问题，而与技术角色（开发人员/架构师/ itpro）无关。 示例：什么是CAPEX / OPEX，公共/私有/混合云，计费方式，谁拥有数据，您需要用两个词来解释几十种基本服务，德国/中国/中的公共数据中心和Azure数据中心有何区别？美国政府等 在输出中，您甚至可以获得基本证书。



最终，考试为那些太早而不能认真对待或根本不需要的人提供，但同时他们需要他们熟悉该主题的理智证书。 在我看来， 这项考试应该传递给所有参与Azure相关项目的经理和卖方 。 这样可以确保人们了解自己在做什么。 技术角色在其认证之路的一开始就通过，这也不是罪过 ，因为 该考试涵盖了数十种基本服务的问题（这是什么以及为什么），同时还给出了该平台的一般概述，而这些概述通常被忽略。

掌握考试后，您可以考虑是否值得“放弃”，因为 每次考试都需要认真的准备工作。

Az-103基础结构（itpro /或时髦的开发人员）


询问AZ-103如何和在Azure中完成哪些操作（创建计算机，访问资源，订阅，资源组，使用Azure动态目录对象，网络等）。 的确，删除了以下几个主题：“ Azure AD身份保护和企业状态漫游； 配置自助密码重置； 实施有条件的访问政策； 管理多个目录； 执行访问审查”，“实施Azure负载平衡器”，“监视虚拟网络并进行故障排除”，“将本地网络与Azure虚拟网络集成”，“基于角色的访问控制（RBAC）”，“实施多因素身份验证（ MFA），并略微扩展了“实施和管理混合身份”部分

AZ-203的发展


要求AZ-203考试有关Web应用程序 ， SQL数据库 ， 函数 ， AKS ， CosmosDB ， 逻辑应用程序 ， 事件网格 ， API管理等。 我认为这是反映Azure当前状态的正确服务集。

AZ-203有足够的问题
  1. 批处理服务为什么要参加考试？ 如果没有数字粉碎器，生活真的是不可能的吗（如果不需要计算密集型操作，WebJobs / Functions就足够了）。
  2. 开发人员为什么要询问通过天蓝色的密钥保管库（为VM配置Azure磁盘加密）进行虚拟磁盘加密？ 这是基础架构问题，部分是安全性。 但是对于发展来说，这有着极其薄弱的关系。
  3. 关于API Apps的部分也颇具争议。 我不明白为什么要在Azure考试中询问Swagger / OpenAPI。 如果没有Azure，这是一件有用的事情。 我会将有关Swagger / OpenAPI的问题转移到“逻辑应用程序”部分，这将结束本主题而不会造成损失，并且我认为这会更合逻辑。
  4. 我也不清楚为什么2019年微软的移动应用程序 。 Microsoft为其移动平台而战，当您没有自己的移动平台时，就不再需要为其提供支持的服务。 但是它们包含在核心考试中。

    • 我非常喜欢设置Android通知中心的说明，内容如下：“在创建Android通知中心之前，创建Firebase（可以免费将通知发送到Android / IOS）。”
    • 为什么要问有关Mobile Apps而不问有关Visual Studio Mobile Center？
  5. 基于声明的/基于角色的授权是有关Asp.Net Core的问题，自 在同一节点中，此实现的实现方式不同，但是您需要知道代码（Microsoft近年来一直在努力使其服务跨平台），并且与Azure没有直接关系。
  6. 为什么在监视部分中询问负载测试（基于Azure DevOps负载测试）是否已弃用并将在2020年退役 。
  7. 为什么在2019年需要Azure存储队列的原因也不清楚。
    举一个简单的例子：
    在Azure Service Bus中，对于100万条消息，每条256kb消息必须支付5美分（在美国东部）。
    1mln * 4kb / 1024（最大为兆字节）/ 1024（最大为千兆字节）/ 2个操作（插入一条消息，接收一条为2个操作）
    这是2GB，价格为$ 0.05。 一条消息的最大长度为256kb，这将使其更便宜或更昂贵。

    Storage Queue中2 GB的成本为0.09美元。
    因此，出现了问题，但是在2019年，使用Azure存储队列的方案是什么？ 如果它们更昂贵，那么会做出更严肃的决定吗？


现在的内容比2015年至2018年的70-532考试要好/新鲜得多 （如果不是因为那些通过AZ-200 / 201/202 的人的欺诈行为，那么一切都会非常好一点也不差）

建筑学


有两门这样的考试；要通过两项考试，必须颁发解决方案架构师证书。 我认为，一个人不让一个人通过是一件很奇怪的事情，因为 这是关于同一件事的两个观点。

  • 对于az-300 Microsoft Azure Architect Technologies，我们从特定服务（azure活动目录，azure存储，虚拟机）中获取
  • 对于az-301 Microsoft Azure架构师设计，我们从关键逻辑块（授权，存储，计算）出发。

我不会说它们之间存在一对一的镜像对应。 例如，在301，相应部分中的单词compute可能表示天蓝色批处理，aks和vmss。 同时，将300清楚地分为3个不同的部分（并行处理，容器，虚拟机）。

我认为没有理由在此列出您需要了解的服务，因为 您不仅需要知道服务，还需要知道它们如何互连以及何时使用它们。 而且还因为列表很长，而且完整版甚至没有包含考试说明的页面。 架构师必须广泛地了解该主题（开发人员也应如此）。

我再重复一次：我建议同时参加两项考试和准备考试 。

Azure Services上的Azure DevOps


AZ-400 Microsoft Azure DevOps解决方案专注于非常广泛的问题。 考试一方面需要了解流程和方法（处理工作项，计划等），另一方面需要了解如何管理项目依赖项，如何收集代码，发布，如何使用外部实用程序进行代码分析，如何获得反馈从用户通过图拉等

个人意见
就我个人而言，我会让考试看起来像AZ-30 *（分为2部分：devops工作的流程和练习/制作部分），因为 就像Scrum Master和Devops工程师之间的区别一样。 但是作者知道得更多。

若要获取Azure DevOps工程师专家证书，必须首先获取用于开发或管理的证书。

  • 如果您想要专门针对Azure / Cloud的认证，那么一切对您都很好。 我建议先移交AZ-203，因为 在其中，您将获得一些Azure服务的概述，并且在AZ-400中已经在发布上下文中完整介绍了Azure Kubernetes服务，Azure容器注册表，ARM，Azure CLI。
  • 好吧，如果您想参加考试但不发布在Azure中 ，那么您仍然必须熟悉Azure 。 您仍然别无选择，因为 2018年12月31日，Microsoft 撤回了所有3项TFS考试 ，现在撤消了Cloud版本或一无所有。


Azure安全工程师


Az-500是于2019年4月发布的一项考试，其次是Microsoft进行与云中角色匹配的认证和考试。

来自az-102的与Azure Active Directory安全性相关的一些问题（例如及时访问等）已迁移到该问题。 另一方面，与AAD日常工作相关的事情出现了，例如：用户管理，创建OAuth应用程序，将订阅转移给另一个所有者。 此外，在Azure Monitor / Log Analytics中设置安全日志的集合（单击许多不同的伺服器上的enable复选框）。
个人意见
因为 我还没有参加考试，还没有准备材料，考试有很多问题。
  • 根据问题列表，一个人应该知道哪些日志包括在哪些服务中。 有许多服务，并且不清楚服务本身需要知道什么级别才能了解哪些错误和安全风险会到达那里。 只需按一个按钮就不会有问题，但是了解其保护范围以及了解所提供的数据是完全不同的技能水平。
  • 在AZ-400（DevOps）上，有很多声称工程师应该在Azure会话外部了解很多。 在Az-500中也会发生同样的情况。 例如：“为容器实施漏洞管理”。 在Azure中，应暗示将ACR与Aqua集成。 但是Aqua不是Azure产品，它是一项外部服务。 即 考试，已经在外部产品上。



数据平台考试


  • 设计和实施Azure AI解决方案AI-100
  • 在Azure DP-100上设计和实施数据科学解决方案
  • 实施Azure数据解决方案DP-200
  • 设计Azure数据解决方案DP-201

我不能称自己是数据分析乃至人工智能领域的杰出专家，但是我对考试差异的理解如下：

  • DP-100数据科学解决方案-这项考试不涉及Azure。 这是一种一般的方法论考试。
  • DP-200 / DP-201是关于在Azure（Azure数据解决方案）中存储和处理数据的。 它包括常规和特定的SQL Azure，CosmosDB，SQL DWH等。
  • AI-100-这里的一切都差不多了，因为 AI并不是关于存储或大量存储，而是关于基于机器学习的分析方法（不仅仅是-花费了大量资源，经过清理，聚合，合并为DWH）

但这只是我的理解，作为一个有兴趣但对本主题不深的人。
