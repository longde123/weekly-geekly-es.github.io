更长的低成本3D照片聚合物打印机出现在Kickstarter上

我无法通过一个有趣的项目来筹集资金在Kickstarter网站上生产Longer光敏聚合物3D打印机。



该模型称为Orange30。此光敏聚合物3D打印机（SLA UV）是一种简单可靠的桌面模型，每个人都可以使用。 Orange 30的最大优点是它不需要基本的印刷技能。 制造商Longer具有一些教授3D打印视频基础知识的知识。 打印机噪音不是很大，没有开放的活动或加热部件，并且在家里或办公室使用时实际上都是安全的。

→ 更长的项目：Kickstarter上可靠，价格最实惠的台式机SLA 3D打印机
→ 官方商店Longer3d.com



Longer Orange 30感光聚合物打印机是一台“塔”，尺寸为20 x 20 x 39厘米，底座是巨大的金属材质，顶部有一个遮光盖。 顺便说一句，保护帽并不需要太多保护使用者，而只是保护光敏聚合物免受来自外部的假照明。 盖子下方有一个用于光敏聚合物的托盘和一个Z轴驱动器，该驱动器在创建模型时将打印台逐渐向上提起。 在基座的底部是带有触摸屏的控制板，用于照亮光敏聚合物的UV矩阵以及2K显示屏（分辨率为2560 x 1440像素的LCD显示屏）。 熔池的尺寸为120 x 68 mm，沿Z轴的最大工作行程为170 mm。 这比流行的廉价Sparkmaker SLA 3D打印机要多。



特点


品牌名称：长
型号：橙色30
类型：光敏聚合物3D打印机（SLA UV 3D打印机）
打印区域尺寸：120 x 68 x 170毫米
印刷层厚度：0.02 ... 0.1 mm
打印速度：30毫米/小时（一层0.1毫米）。
显示可用性：是的，触摸2.8英寸
照明类型：405 nm UV LED矩阵
消耗类型：405 nm光敏聚合物树脂（UV树脂）

如果您决定要成为一名面包师，请记住，当前出价为229美元（橙色30打印机+一瓶光敏聚合物）。 俄罗斯的运费为$ 110。 他们承诺将于2019年8月开始发送样品。



这是预算选项之一。 您可以等待并购买该打印机。 但是价格可能会上涨（在Kickstarter上，划掉的价格为450美元）。 可能不会上升。 以前的Longer型号的OEM克隆产品在中国商店免费销售（FDM打印机150美元及以上，SLA 250美元起）。

设计简单但有效：MGN12导轨沿单个Z轴安装。 不是弯曲的轴，而是普通的（中文）导轨。 金属托盘用于光敏聚合物。 是的，它很简单，有大头钉，有盖印，但不是塑料的。 形成模型的表也是金属的。 打印机开发人员提供了表格对齐机制。 为此，打印机菜单中有相应的操作。

该模型的一个大胆加法是触摸屏的出现，该触摸屏具有打印机管理，打印和预览模型。 我从未在任何地方看到内置文件查看器。 我认为，这是预算模型中的杀手级功能。



我们为什么需要光敏聚合物打印机-用于打印具有高细节（20 ... 50微米）的精确模型（副本）。 此类打印机可用于牙科，珠宝，模型制作等。有特殊的光敏聚合物，例如可燃（可铸造）的聚合物，可用于铸模3D模型。



Longer3D及其之前的版本作为OEM模型的制造商而出现，然后出现在其他品牌下，例如Alfawise。

在目录中，有一个有趣的模型Longer Orange 120-这是一种经过改进的Sparkmaker，带有显示屏和金属浴。 朗格（Longer）的同志重新设计了预算打印机，并固定了主要的门框。

注意模型Longer Orange 10 。 这是“最年轻”的机型Orange30。它具有类似的设计，但打印面积较小，仅为98 x 55 x 140毫米，并且矩阵的“子级”分辨率（854 x 480像素）。 这种打印机的零售价略低于300美元。 制造商提供了有关设置和使用Orange 10型号的视频，在网站上您可以找到驱动程序，切片器，固件和说明。

我喜欢鲜为人知的初创公司正在开发并想到项目。 此外，Longer还有其他有趣的模型。 其他制造商从Longer购买OEM许可并以自己的品牌生产相同的设备并非没有道理。 即使交货，该打印机的价格也仅为流行的Wanhao Duplicator 7和Anycubic Photon的一半。 当然，您可以一次购买两套（$ 550 +运费$ 195），从而获得更多节省。 实际上，这是两台具有2K矩阵的光敏聚合物打印机，价格仅为一个Anycubic Photon。



无论如何，我都会关注这个项目，因为它很有趣。
