Tesla Black Lane di Highway 101



Menurut laporan terakhir, kegagalan fatal menimpa pengemudi Tesla Model X pada pagi hari tanggal 23 Maret.
Tragedi itu terjadi di Highway 101 di Mountain View, California (Mountain View). Mobil listrik menabrak chipper pemisah, itu terlempar ke jalur kiri kedua, di mana Mazda menabraknya, bergerak di sepanjang jalur ini.



Audi, yang bergerak di paling kiri, menambahkan api dalam arti kata yang sesungguhnya.



Kerusakan sangat kuat sehingga menyebabkan api yang melimpah dari mobil listrik dan dari ledakan berikutnya itu benar-benar terbelah dua.



"Kami melihat awan besar asap, dan tiba-tiba bola api muncul di tempat itu," saksi Aiden Sanchez menunjukkan.



"Saya melihat Tesla X terbakar dan merokok, saya melihat seorang pria berlari dengan alat pemadam kebakaran," kata seorang saksi mata yang ingin tetap anonim.



Saya datang lebih dekat, lanjutnya, dan melihat seorang pria yang tertutup lapisan debu. Lelaki itu tampaknya sedang berurutan. Bagian depan mobil menghilang begitu saja!



Pengemudi mobil listrik dikirim ke Pusat Medis Universitas Stanford dalam kondisi serius.
