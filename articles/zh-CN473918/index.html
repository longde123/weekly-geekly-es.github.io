俄罗斯的互联网旅行：速度与机遇

在2019年春季，我对可以固定使用的LTE路由器进行了测试。 对于私人住宅的居民或具有有线网络访问权限的人来说，这是一个相当有趣的测试，其成本要高于无线网络。
这次，我决定检查移动运营商的能力：我移至不同的区域并测试了一个特定电信运营商的访问速度。 传统上，视频是为视频发烧友制作的，其他视频将带有图片。


这次我决定使用提供移动互联网的Wifire运营商的SIM卡。 而且，与一个知名运营商不同，Wifire资费不仅在4G（LTE）网络中有效，而且在3G中也有效，这对于信号电平不是很高的用户来说很重要，因为3G信号覆盖范围通常更大并且连接更好。 当然，对我来说，测试网络访问的速度和是否缺乏漫游很有趣，因此我为自己安排了一些休息，并开车穿越了美国的多个地区。



为了充分享受网络的易用性和访问速度，我使用了智能手机华为P30 Pro。 这是华为挖掘的智能手机系列中的高端智能手机，具有真正出色的功能。 对我而言，特别重要的是，该智能手机支持LTE信道的聚合，并且理论上能够提供此类运营商能够提供的最高400 Mbps的传入数据传输速率。 信道聚合是在基站一次合并多个信道以将数据传输到订户。 通常，在智能手机中，此模式的操作显示为4G +。

测试方法


在每个新的地方，大约在同一时间-下午进行测试。 基站上没有早上和晚上的高峰负载，并且有可能接收有关数据传输速度的真实数据。 我对传入速度更感兴趣，因为传出速度仅在传输大量数据时才重要，并且在视频会议或流传输期间，传出数据流不超过3-5 Mbit / s。 使用流行的服务Speedtest.net进行了测试。 进行多次测量，并取中值。 如果有任何峰值被高估或低估，则不考虑它们。 智能手机在自动模式下工作，但是几乎所有测量都具有LTE连接，只有一个地方只有3G连接，但它离城市很远。

为什么需要这个？


对提供者进行比较测试是荒谬的，因为无线通信的细微差别使得测试结果仅适合我一个人。 有时，向任何方向移动一百米就足够了，不同运营商的测量结果将发生巨大变化-这会影响地形，基站的负载以及客户端设备的功能。 我想知道从特定的提供商到用户（无论是工作还是休闲）移动很多并使用移动Internet的用户使用移动Internet有多方便和舒适。 在路上！



旅途的第一点是美丽的下诺夫哥罗德，遇到了低温和晴朗的天空。 这座城市理所当然地被列入最受欢迎的旅游路线之一。 风几乎敲了下来，但我还是骑着缆车-这很有趣。 然后，我来到了受到游客和当地人欢迎的地方-Chkalov梯子。 快到中午了，人不多，但走路的人就足够了，也就是说，基站的负载是标准的。 速度的测量值也有波动，但我将这个结果作为主要结果-51.9 / 26.0 Mbit / s（DL / UL）。 受欢迎和参观过的地方的体面指标。



接下来的一点是一个非常有趣的斯韦特洛亚尔湖。 它位于下诺夫哥罗德地区，据传说，基特日市就藏在这里。 如果您查看地图，该湖几乎是完美的圆形，因此湖的起源有多种形式：冰川岩溶，构造断裂和陨石。



一个朝圣者和游客都非常受欢迎的地方。 尽管这个地方人来人往，但这里的旅游基础设施却不是很好。 实际上，这些话可以说成是俄罗斯许多受欢迎的路线。 但是，应该访问网络，甚至只是作为停留地点的历史参考。 湖水被高大的森林所环绕，岸上只有3G信号被捕获。 在这样的时刻，特别容易感觉到地形的影响。 但是，此位置的速度为12.8 / 2.19 Mbit / s。 即使对于在任何Messenger上进行视频通话以及安静地冲浪，这也足够了。 YouTube视频也冻结了。



此外，我的路线位于伏尔加河上游至特维尔市。 许多人将这座城市与已故的香颂歌手联系在一起，但对我而言，这个城市首先是到达印度的亚撒纳修·尼基丁的出生地。 他的日记“走过三海”成为俄罗斯和印度之间商品，经济和政治互动的基础。



尽管两国首都之间的地理位置优越，并且与首都的领土关系更加紧密，但特维尔并没有让我感到沮丧，尤其是在下诺夫哥罗德之后。 尽管恢复凯瑟琳女皇的行宫的趋势令人非常高兴。 当然，在旅途中，我想分享照片，交谈，有时我必须工作，因此良好的沟通非常重要。 我决定在伏尔加河大堤上进行速度测量。 这是一个工作日，是一个工作日，因此基站的负载很典型。 每天早上和晚上观察到负载高峰，那时每个人都通过移动设备上线阅读新闻，浏览社交网络或共享当天的事件。 连接是通过4G进行的，因为这个地方很受欢迎，所以附近的某个地方有一个基站，因为信号电平非常好。 测量结果：60.6 / 40.8 Mbps。 无论是工作还是娱乐，都是非常值得的结果。 原则上，以这种速度，您可以在线观看4K视频，并可以在FullHD中以高比特率进行广播。



然后我前往首都，沿途收集一些交通拥堵。 我决定直接在运动中测量通讯速度。 在其中一个阶段中，以超过100 km / h的速度，我还启动了Speedtest实用程序。 结果非常有趣：12.3 / 4.9 Mbps。 有了如此高的速度，您就可以安全地观看在线电影或通过视频进行交流。 原则上，速度会在移动过程中波动，因为智能手机经常紧贴新塔，因此信号电平和通道负载的情况可能会完全不同。 途中，我看到了高于100 Mbit / s的下载结果，但在一些交通信号灯的十字路口却停了下来。



我对移动互联网的旅程和测试即将结束，Zaryadye Park将是最后一点。 美丽的景色，良好的基础设施以及一天中任何时候都有大量游客。 后者会极大地使测试复杂化，因为成千上万的照片不断从此地发送到世界所有国家。 人们不仅与其他城市的对话者交流，也与其他国家的对话者交流。 我想知道这里的测量结果如何？ 尽管可以预测，结果还是有些令人惊讶：58.4 / 3.1 Mbit / s。 正如预期的那样，用户通常从此处发送照片，从而下载传出数据通道。 但是即使是这样的速度也足以进行视频通话。 好吧，下载速度甚至足以观看在线电影院。



结论


我将在表中收集有关不同区域中数据接收和传输的所有数据。 应当注意的是，传入通道的速度没有任何地方不低于10 Mbit / s，这意味着在途中接收数据或处理文档将非常舒适。 我从未需要联系技术支持来解决通信问题。 激活SIM卡后几分钟便出现了Internet访问，并且该区域的更改并未影响访问网络的可能性。 Wifire提供程序不提供语音和SMS，而是通过2G / 3G / 4G信道通过蜂窝网络提供访问，也就是说，只要有信号连接，就始终可以通过任何类型的连接访问网络。



至于资费，即一个月的套餐，每隔一个月更新一次流量套餐。 可以在没有时间限制的情况下购买流量并进行消费。 这非常适合需要少量偶尔访问网络的各种智能设备。 例如，在平板电脑，调制解调器或任何类型的智能设备中，如果您很少在可用的Wi-Fi网络之外使用小工具，则迫切需要Internet访问。
