Terminal Windows Baru: Jawaban atas Pertanyaan Anda

Dalam komentar di artikel terbaru , Anda mengajukan banyak pertanyaan tentang versi baru Terminal Windows kami. Hari ini kami akan mencoba menjawab beberapa dari mereka.

Di bawah potongan adalah beberapa pertanyaan yang paling sering diajukan yang telah kami dengar (dan masih dengar), serta jawaban resmi: termasuk penggantian PowerShell dan cara mulai menggunakan produk baru hari ini.



Kapan dan di mana saya bisa mendapatkan Terminal Windows baru?


  1. Anda dapat mengkloning kode sumber terminal dari GitHub di github.com/microsoft/terminal dan mengkompilasinya di komputer Anda.
    Catatan : pastikan untuk membaca dan mengikuti instruksi pada halaman README di repositori sebelum Anda mencoba membangun proyek - untuk membangun proyek Anda harus mengikuti beberapa prasyarat dan langkah inisialisasi!
  2. Versi pratinjau terminal akan tersedia untuk diunduh dari Microsoft Store pada musim panas 2019.

Kami berusaha untuk merilis Terminal Windows v1.0 pada akhir 2019, tetapi pada saat yang sama kami akan bekerja dengan komunitas untuk merilis versi ini sehingga terminal berkualitas tinggi.

Apakah Windows Terminal pengganti untuk Command Prompt dan / atau PowerShell?


Untuk menjawab pertanyaan ini, mari kita perjelas beberapa istilah dan konsep:

  • Command Prompt dan PowerShell (mis. WSL / bash / etc. dalam * NIX) adalah shell, bukan terminal, dan tidak memiliki UI sendiri
  • Ketika Anda menjalankan alat shell / application / command-line, Windows secara otomatis meluncurkan dan menghubungkannya ke instance dari Windows Console (jika perlu)
  • Windows Console adalah aplikasi UI "terminal-like" standar yang dikirimkan bersama Windows dan telah digunakan oleh pengguna selama 30 tahun terakhir untuk menjalankan alat baris perintah di Windows NT, 2000, XP, Vista, 7, 8, dan 10



Dengan demikian, pertanyaan yang paling mungkin harus diformulasi ulang sebagai "Terminal Windows - apakah ini pengganti untuk Windows Console?"

Jawabannya adalah "Tidak":

  • Windows Console akan terus mengirim ke Windows selama beberapa dekade untuk memberikan kompatibilitas dengan jutaan skrip / aplikasi warisan, jutaan aplikasi yang ada, dan alat baris perintah
  • Windows Terminal akan bekerja dengan Windows Console, tetapi kemungkinan akan menjadi alat yang disukai untuk pengguna yang ingin menjalankan alat baris perintah pada Windows
  • Terminal Windows dapat terhubung ke Command Prompt dan PowerShell, serta ke aplikasi shell / tool / command line lainnya. Anda dapat membuka tab independen yang terhubung ke Command Prompt, PowerShell, bash (via WSL atau ssh) dan semua kerang / alat lain pilihan Anda
