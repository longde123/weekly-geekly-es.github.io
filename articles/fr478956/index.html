Présentation de 3CX V16 Update 4 Beta avec un client VoIP Chrome et une application vidéo pour Android

Cette semaine, nous avons présenté deux nouvelles versions - 3CX V16 Update 4 Beta et le nouveau client 3CX pour Android avec support vidéo! La mise à jour 4 Beta a introduit l'extension Chrome, qui implémente le softphone VoIP en tant qu'application de navigateur en arrière-plan. Vous pouvez recevoir des appels sans rompre avec le programme actuel et sans ouvrir le client Web 3CX. Vous pouvez répondre instantanément via une petite fenêtre pop-up dans le coin inférieur droit du bureau.



Les notifications d'appels arrivent même si le navigateur est réduit ou fermé du tout - l'extension ne nécessite pas de client Web en cours d'exécution.

Click-to-Call est désormais intégré dans la nouvelle extension. Lorsque vous parcourez une page Web ou travaillez dans CRM et que vous souhaitez composer un numéro, cliquez simplement dessus. Le numéro sera intercepté et composé directement depuis l'application active.

Pour installer l'extension, accédez au client Web 3CX et dans un autre onglet, ouvrez la page d'extension . Cliquez ensuite sur «Installer l'extension 3CX pour Chrome» et dans le client Web, cliquez sur «Activer l'extension 3CX pour Chrome».

L'extension 3CX pour Google Chrome nécessite la mise à jour 3CX V16 mise à jour 4 bêta et Chrome V78 et supérieur. Si vous avez installé l'extension Click to Call 3CX précédente, désinstallez-la avant d'installer une nouvelle extension.

Si la mise à jour 3 et une version antérieure sont installées, installez d'abord la mise à jour 4 et redémarrez le navigateur avec le client Web ouvert, afin de pouvoir activer l'extension.

La sortie de 3CX v16 Update 4 Beta a également ajouté la prise en charge de nouveaux protocoles de stockage et de sauvegarde:

  • Les protocoles FTP, FTPS, FTPES, SFTP et SMB peuvent désormais être utilisés pour sauvegarder la configuration et les enregistrements d'appels.
  • L'utilitaire de distribution 3CX comprend un utilitaire pour transférer les archives de conversation de Google Drive vers le disque local du serveur PBX sans perdre d'informations sur les fichiers d'enregistrement.
  • Résolveur DNS amélioré (traitement des requêtes "Invite / ACK" pour certains opérateurs SIP).

La mise à jour de la mise à jour 4 bêta se fait comme d'habitude dans la section "Mises à jour". Vous pouvez également installer la distribution 3CX v16 Update 4 Beta pour Windows ou Linux:

  • Windows: 3CX v16 Update 4 Beta .
  • Linux: 3CX v16 sur Debian 9 ISO - Sur l'écran " 3CX Update Channel ", sélectionnez "Beta".

3CX pour Android - appel vidéo pour les entreprises


Avec Update 4 Beta, nous avons publié la version finale de l'application Android 3CX avec appels vidéo intégrés. Nous avons également essayé de mettre en œuvre une prise en charge d'une grande variété de smartphones Android, afin que presque tous les utilisateurs puissent utiliser la nouvelle application.



Vous pouvez maintenant appeler l'abonné, puis cliquer sur le bouton "Vidéo" et passer à l'appel vidéo. Les appels vidéo fonctionnent entre la nouvelle application Android 3CX, le client Web et les visiophones ou interphones qui prennent en charge les codecs Google VP8 et VP9 (voir ci-dessous).

Le client prend également en charge l'API Google AAudio. Google AAudio est une alternative moderne à l'OpenSL (Open Sound Library) largement utilisé. Il est conçu pour fournir un son de haute qualité dans les applications qui nécessitent une latence minimale. La prise en charge de la nouvelle API est automatiquement activée pour les derniers modèles de téléphones - consultez la liste des appareils compatibles . De plus, la nouvelle application détermine automatiquement les capacités de l'appareil et désactive l'API Telecom pour certains modèles afin d'éviter l'apparition d'un écho.

Après de nombreux tests et optimisations (grâce à nos testeurs!), L'application a commencé à prendre en charge encore plus de smartphones. Les derniers modèles sont pris en charge: Pixel 4, Galaxy Note 10, S10 +, Xiaomi Mi9. Plus d' appareils seront pris en charge dans un avenir proche.

Autres changements et améliorations.


  • Correction d'un bug lors de la tentative de basculement de l'adresse IP vers le FQDN du serveur lors d'un appel.
  • Les collègues avec lesquels vous communiquez souvent peuvent désormais être ajoutés à la section Favoris pour une communication rapide.
  • Un filtre déroulant a été ajouté dans la section «Statut» pour afficher tous les groupes (locaux et d'autres échanges) et leurs membres.
  • Ajout d'un signal d'appel en attente pour les appels GSM entrants lors d'une conversation SIP. Répondre à un appel GSM met l'appel SIP en attente.
  • Pendant un appel GSM, les appels SIP entrants sont occupés et transférés selon les règles de transfert spécifiées.
  • Maintenant, vous pouvez simplement cliquer sur le message vocal pour l'écouter automatiquement via le lecteur Google Play intégré.
  • Ajout de l'option "Ne plus demander" lorsque l'application se voit refuser l'accès aux contacts. La demande ne sera pas renouvelée.
  • Les fichiers reçus d'autres utilisateurs sont désormais stockés dans un dossier spécial sur l'appareil conformément aux normes Android 10.
  • Ajout du filtre déroulant "Contacts" qui affiche tous les contacts, uniquement les contacts 3CX ou uniquement les contacts du carnet d'adresses de l'appareil.
  • Le nombre maximal de participants à la conférence sur demande est fixé à 3. Pour les grandes conférences, utilisez la section Conférence dans le menu latéral de l'application.
  • L'élément «Désactiver l'optimisation de l'alimentation» est immédiatement transféré dans la section «Exceptions au mode d'économie d'énergie» dans les paramètres Android.

Communication vidéo entre le client Web 3CX, l'application Android et l'interphone vidéo


Après la sortie de nouvelles applications 3CX avec prise en charge de la communication vidéo, il est devenu possible de les utiliser avec des téléphones vidéo et des interphones vidéo avec prise en charge des codecs Google VP8 et VP9 modernes. Le client Web 3CX et l'interphone vidéo fonctionneront ensemble - le bureau ou la maison peut être contrôlé par l'interphone de porte Fanvil iSeries et le PBX 3CX gratuit.



Le visiteur appuie sur le bouton de numérotation rapide attribué à un utilisateur / poste spécifique dans le PBX. Cet utilisateur reçoit un appel vidéo via l'interface client Web ou dans l'application Android 3CX. Vous pouvez également renvoyer l'appel vers votre mobile si vous n'êtes pas présent pour le moment (mais vous ne pouvez alors répondre que par la voix).



Si vous êtes souvent absent du lieu de travail, définissez les règles de transfert d'appel vers d'autres utilisateurs et l'appel vidéo sera transféré au prochain utilisateur / secrétaire disponible. Vous pouvez également configurer un appel vers la file d'attente prioritaire 3CX afin qu'un appel provenant de l'interphone soit toujours prioritaire parmi les opérateurs de file d'attente.

Les visiteurs à la réception du bureau ou, inversement, dans une pièce à accès limité peuvent appuyer sur le bouton de numérotation rapide de l'interphone vidéo pour communiquer via votre client Web. La même fonctionnalité peut être utilisée pour la vidéosurveillance d'un entrepôt ou d'une autre zone contrôlée.
