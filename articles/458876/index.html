Viernes Mini CTF

Saludos a todos los habrachitateli. Realmente me gusta estirar mi cerebro con diferentes tareas: pueden ser algunos problemas matemáticos, rompecabezas interesantes relacionados con la programación o algunas tareas de ctf.

Para aquellos que quieren diversificar el viernes con rompecabezas, quiero ofrecer mi propio mini-ctf, en el que solo hay tres tareas.

XT_CTF_01

XT_TASK_101

Enlace a la tarea: xt_task_101.zip
Formato de marca : XTGFLAG {...}
Bandera SHA1: 6980D76C3C5246A101820A77031CA0EBE84A4A0E

XT_TASK_102

Enlace a la tarea: xt_task_102.zip
Formato de marca : XT {...}
Bandera SHA1: 16448BF22750DE7E575A5A86E739FA9018ACFE52

XT_QUEST_103

Enlace a la tarea: xt_quest_103.zip
Número de niveles: 15 + bonificación

Cada archivo es una tarea separada (bueno, o en el caso de 3 tareas - una búsqueda), no están interconectados. El objetivo de los dos primeros es obtener una bandera cuyo formato se especifica. Como verificación adicional, indiqué hashes de marca para que pueda verificar fácilmente si resolvió el problema correctamente o no. La búsqueda consta de 15.5 tareas consecutivas, donde se abre una nueva después de resolver la anterior, la respuesta a la tarea actual es la contraseña del archivo con la siguiente, por lo que no se requiere un hash separado para la verificación.

En las tareas no hay archivos ejecutables, dependencias de algunos servicios externos. Puede usar lo que quiera: Google, servicios en línea, sus lenguajes de programación favoritos.

Evalúo la complejidad de las tareas como medio para tareas y fácil para misiones. Sería genial si los comentaristas se abstienen de incitar en los comentarios (o al menos los esconden debajo de un spoiler) para no estropear el proceso de decisión para aquellos que decidirán más adelante.
