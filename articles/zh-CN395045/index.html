11款出色的开放规格Linux友好型单板PC

所有型号的价格都低于200美元



。HackerBoards.com多年来一直在对单板计算机进行分类。现在在那里有81个单一模型。我选择了其中11个满足以下条件的产品：开发板的成本应低于200美元，其软件应开放（Linux或Android）。电路板必须提供扩展的规格，包括详细的图表。每个董事会都应该有一个活跃的社区并为单个开发人员提供技术支持。可以将开发人员的理想电路板称为这样的系统，该系统经常在论坛和致力于电子领域开发的其他资源中提及。

86 Duino零



  • 公司/资源-DM＆P，86Duino.com
  • 在LinuxGizmos / HackerBoards上的描述
  • 项目页面
  • 处理器-DM＆P Vortex86EX（1x x86 @ 300MHz）
  • 内存-128MB RAM
  • 价格-$ 39

它是基于32位x86 Vortex86EX SoC处理器的开源Arduino兼容平台。它与Windows，Linux，32位RTOS操作系统兼容。该板配备了PCIE总线，DDR3，ROM控制器，xISA，I2C，SPI，IPC，快速以太网，FIFO UART，USB2.0和SD / SATA控制器。售价69美元的型号非常相似，但是它已经支持高清音频和更多的I / O扩展。还有一个版本的EduCake mini-PC。

A20-OlinuXino-Lime



  • 公司/资源-Olimex，OlinuXino，Mouser
  • 项目页面
  • 处理器-Allwinner A20（2x Cortex-A7 @ 1GHz）; 马里400 GPU
  • 内存-512MB DDR3 RAM
  • USB-OTG端口；
  • SATA接口+ 5V SATA接口;
  • HDMI连接器，支持FullHD（1080p）；
  • 100Mbps以太网连接器
  • 锂聚合物电池连接器；
  • 与显示器4.3“，7.0”，10.1“ Olimex兼容的LCD连接器；
  • MicroSD卡插槽；
  • 调试套接字DEBUG-UART;
  • 4个GPIO插座（160个触点）;
  • LED指示灯GPIO，电池，电源；
  • 2K EEPROM MAC地址存储
  • 价格-$ 37


微型PC是一款相当紧凑的单板Linux计算机，具有开放的硬件平台和非常好的外围接口集。84 x 60mm版本配备SATA接口，在这种类型的主板中很少见。有microSD，以太网，HDMI和三个USB端口。您可以连接一个LCD显示器，它支持电池和160个GPIO。

仙女座盒子边缘



  • 公司/资源-Marvell
  • 项目页面
  • 处理器-Marvell IAP140（4个Cortex-A53内核@ 1.2 GHz）; Vivante GC7000UL GPU
  • 内存-1GB LPDDR3 RAM; 8GB eMMC
  • 价格-$ 75

该微型PC正在运行基于Android的Brillo OS。董事会拥有相当大的社区。有关系统，方案，文档，发行版Brillo的完整说明。该系统支持Google Weave和Thread网络标准。该开发板包括WiFi，蓝牙，三个USB端口，一个HDMI端口和microSD。

Arduino工业101



  • 公司/资源-Arduino Srl
  • 在LinuxGizmos / HackerBoards上的描述
  • 项目页面
  • 处理器-Qualcomm Atheros AR9331（1x MIPS24k @最高400MHz）
  • 内存-64MB DDR2 RAM; 16MB SPI闪存
  • 价格-$ 39


该板采用三明治形式制造，类似于Arduino Yun。内部是一个微控制器，具有Linino操作系统的Qualcomm Atheros AR9331 MIPS处理器和IEEE 802.11b / g / n 1x1 2.4 GHz WiFi模块。该系统通过mini-USB供电。有一个LED。

ArduinoYún迷你

  • 在LinuxGizmos / HackerBoards上的描述
  • 项目页面

该板是原始Yún的较小版本。该系统基于OpenWrt（Linino），并带有高通公司的400MHz AR9331处理器。芯片-Atmel Atmega32U4。该评估板配备了以太网，USB和microSD。

Arduino天



  • 公司/资源-Arduino Srl
  • 在LinuxGizmos / HackerBoards上的描述
  • 项目页面
  • 处理器-Qualcomm Atheros AR9432（1x MIPS24k @最高560MHz）; Atmel SAMD21G18 Cortex M0 + MCU（48MHz）
  • 内存-64MB DDR2 RAM; SAMDB21G18的256KB
  • 价格-$ 99

与前两块板相比，这块板效率更高。它可与快速的Atheros AR9432 WiFi SoC配合使用。支持蓝牙EDR / BLE 4.0a。其他模块包括一个微型USB端口，20个DIO引脚，12个PWM和6个模拟引脚。

Arndale八角




  • 公司/资源-InSignal，Pyrustek
  • 项目页面
  • 处理器-Samsung Exynos 5420 Octa（4x Cortex-A15 @ 1.8 GHz和4x Cortex-A7 @ 1.3 GHz）; 马里T628 MP6 GPU
  • 内存-3GB LPDDR3e RAM
  • 价格-199美元


选择中最昂贵的板之一。它基于三星的Exynos 5420 SoC，具有HDMI，eDP和MIPI DSI端口以及USB端口。该主板支持MIPI-CSI摄像头，还有一个可选的无线模块。还有一个生产率更高的板ArndaleBoard-K，尽管它的价格已经为259美元。

香蕉皮M2



  • 公司/资源-SinoVoip
  • 在LinuxGizmos / HackerBoards上的描述
  • 项目页面
  • 处理器-Allwinner A31（4x Cortex-A7 @ 1GHz）; PowerVR SGX544MP2 GPU
  • 内存-1GB DDR3 RAM
  • 价钱-$ 58


2014年，Banana Pi项目分为SinoVoip和LeMaker。与大多数其他Allwinner项目一样，开源支持也有所减少。双核Banana Pi是RPi Model B的克隆，而Banana Pi M2是具有40针连接器的Model B +的类似产品。该开发板配备了GBE，WiFi，五个USB端口和许多用于连接显示器或摄像机的接口。

比格犬骨绿



  • 公司/资源-Beagleboard.org的SeeedStudio
  • 在LinuxGizmos / HackerBoards上的描述
  • 项目页面
  • 处理器-TI Sitara AM3358（1x Cortex-A8 @ 1GHz），带有PRU MCU芯片；PowerVR SGX530 GPU

开发人员将开发板定位为用于物联网的微型PC。它出现后几乎立即开始流行，现在已经拥有了一个庞大的社区。BeagleBone Green（BBG）-基于经典的开源平台BeagleBone Black（BBB），已向其中添加了两个Grove连接器。与BBB不同，BBG平台没有HDMI端口，USB主机连接器已移动，USB客户端连接器已更新为Micro USB。BBB软件与BBG完全兼容。
