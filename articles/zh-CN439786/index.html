审查IP电话Snom D712（和少许D710）


今天，我们将讨论新的Snom D712 IP电话，它看起来与已发布的D710完全相同。 它们仅在一项功能上有所不同。 但是首先是第一件事。

外表


正如该指数所暗示的那样，D712是Snom D7xx系列的低成本型号。 更确切地说，这是最年轻的模型。 D712取代了现在停产的D710。 但是，由于“旧”模型的库存仍然很大，因此您有很大的机会出售它。

正如我所说，D712在外部与D710完全相同，因此只能通过底部的铭牌对其进行区分。



D712的外观严格而朴实，但不老式。





D712 / 710上带有四个功能按钮的四行显示与D725相同。 它们下方是传统的四向导航“操纵杆”，侧面有两个按钮-“确认”和“取消”。





D712 / 710具有五个可编程按键，您可以在这些按键上“挂断”对特定用户的呼叫或其他操作。 对于他们来说，没有其他显示，因此按钮的功能将必须在一张纸上手动签名。 但是，您可以使用Snom D7扩展模块来增加可编程按钮的设置。

D712 / 710的管形简单，但符合人体工程学，可舒适地握在手中。



听筒下方是一个漂亮的大型免提扬声器格栅。



显示屏的背光不仅负责电话活动的“可视化”，而且右上角的大红色LED以及不同按钮上的红色和绿色也负责。



与D7xx系列的所有型号一样，D712 / 710配有大型且稳定的快速释放支架，可以将其安装在两个位置之一中，从而使手机倾斜46°或28°。





除了耳机端口外，D712 / 710还配备了两个RJ45端口和一个100兆位以太网交换机。



该电话可以通过PoE技术供电，也就是说，工作场所的至少一根电线会更少。

可能性


D712 / 710最多支持4个SIP帐户，这足以实现大多数业务场景。 该电话配备了一个可记录1000条条目的地址簿，以及一条拨出/拨入/未接电话的日志。 借助集成的XML浏览器，D712 / 710可以完全远程配置。 具有自动重拨号码的功能，如果用户的线路繁忙，则可以激活“请勿打扰”模式，自动接听来电或打开特定号码的呼叫等待。 该电话最多可以同时处理4个电话-可以保持通话，这时音乐将播放您的对话者。 与旧型号一样，您可以使用Snom电话的标准功能-三方电话会议。

D712支持以下协议：

  • SIP（RFC3261）
  • DHCP，NTP
  • HTTP / HTTPS / TFTP
  • LDAP（目录）
  • IPv4 / IPv6

D710和D712之间的区别就在这里-仅新产品支持IPv6，D710没有这样的机会。

对于音频编解码器，两个模型都支持相同的集合：

  • G.711 A律，μ律
  • G.722（宽带）
  • G.726，G.729AB，GSM 6.10（FR）

有语音激活功能，舒适的噪音发生器以及可以分配给不同用户的内置铃声。 关于信息安全，D712 / 710支持大量协议和技术：

  • 802.1X身份验证和EAPOL
  • TLS
  • SRTP（RFC3711），SIPS，RTCP
  • Https
  • 受密码保护的Web界面。
  • VPN支持。
  • VLAN（IEEE 802.1Q）
  • LLDP-MED，RTCP-XR



Snom D712已经开始销售，并将开始逐步将D710从仓库中移出。 除非您需要IPv6支持，否则模型是完全可互换的。 否则，请在您所在城市的商店询问D712 :)
