在CentOS 7上安装Zimbra开源版

在企业中设计Zimbra实施方案期间，IT经理必须选择Zimbra基础结构节点将在其上运行的操作系统。 如今，几乎所有的Linux发行版都与Zimbra兼容，直到国内的红色OS和ROSA。 通常，在企业中安装Zimbra的选择是Ubuntu或RHEL，因为商业公司正在开发这些发行版。 但是，IT经理通常会选择Cent OS，它是Red Hat商业RHEL发行版的社区运行分支。



Zimbra的最低系统要求包括服务器上8 GB的RAM，/ opt文件夹中至少5 GB的可用空间，以及完整的域名和MX记录。 通常，刚开始的最大问题恰好出现在最后两点。 在这种情况下，CentOS 7的最大优点是它允许您在操作系统安装阶段设置服务器域名。 这样，即使对于以前没有使用Linux经验的用户，您也可以毫无问题地安装Zimbra Collaboration Suite。

在我们的例子中，将要安装Zimbra的服务器的域名是mail.company.ru。 安装完成后，仅需添加格式为192.168.0.61 mail.company.ru mail的行 ，其中必须注册服务器的静态IP地址而不是192.168.0.61。 之后，您需要安装所有软件包更新，并使用dig -t A mail.company.ru和dig -t MX company.ru命令向服务器添加A和MX记录。 因此，我们的服务器将具有完整的域名，现在您可以轻松在其上安装Zimbra。

您可以从官方网站zimbra.com下载包含Zimbra发行版当前版本的档案 。 解压缩归档文件后，它将仅保留运行名为install.sh的安装脚本。 您需要的控制台命令集如下：

mkdir斑马纹&& cd斑马纹
wget files.zimbra.com/downloads/8.8.12_GA/zcs-8.8.12_GA_3794.RHEL7_64.20190329045002.tgz --no-check-certificate
焦油zxpvf zcs-8.8.12_GA_3794.RHEL7_64.20190329045002.tgz
cd zcs-8.8.12_GA_3794.RHEL7_64.20190329045002
./install.sh



此后，Zimbra Collaboration Suite安装程序立即启动。 首先，您必须接受许可协议才能继续安装ZCS。 下一步将是选择要安装的模块。 如果您要创建一个邮件服务器，则有必要一次安装所有软件包。 如果您打算创建具有扩展能力的多服务器基础结构，则应仅选择一些建议安装的软件包，如我们先前文章之一所述。

安装完成后，Zimbra设置菜单将直接在终端中打开，如果您选择单服务器安装，则只需设置管理员密码即可。 为此，首先选择项目编号7，然后选择项目4设置管理员密码，该密码必须至少为6个字符。 设置密码后，按R按钮返回上一级菜单，然后按A按钮接受更改。

Zimbra安装完成后，使用命令firewall-cmd --permanent --add-port = {25,80,110,143,443,465,587,993,995,522,5223,9071,7071} / tcp打开在防火墙中运行该端口所需的端口 ，然后使用以下命令重新启动防火墙防火墙cmd-重新加载

现在，我们只需使用service zimbra start命令启动Zimbra即可开始。 您可以通过转到company.ru：7071/zimbraAdmin/来访问浏览器中的管理控制台。 电子邮件用户的访问权限将在mail.company.ru中提供。 请注意，如果在Zimbra操作过程中出现任何问题或错误，应在日志中搜索答案，该日志可以在/ opt / zimbra / log文件夹中找到。

Zimbra安装完成后，您还可以安装Zextras Suite扩展，它可以通过添加业务所需的功能来提高使用Zimbra的可靠性和成本效益。 为此，您需要从Zextras.com下载具有最新版本的Zextras Suite的存档，然后将其解压缩。 之后，转到解压缩的文件夹并运行安装脚本。 控制台形式的整个过程如下：

wget download.zextras.com/zextras_suite-latest.tgz
焦油xfz zextras_suite-latest.tgz
cd zextras_suite /
./install.sh全部



之后，您的Zimbra将有机会在邮件存储中归档和重复数据删除，挂载辅助卷，将管理权限委派给其他用户，直接在Zimbra Web客户端中使用在线聊天等等。

有关Zextras Suite的所有问题，您可以通过电子邮件katerina@zextras.com与Zextras Katerina Triandafilidi的代表联系。
