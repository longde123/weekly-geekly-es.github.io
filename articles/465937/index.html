Technostream: una nueva selección de videos de capacitación para el inicio del año escolar


Septiembre para muchos ya está asociado con el final de la temporada de vacaciones, pero para la mayoría, con estudio. Al comienzo del nuevo año escolar, le ofrecemos una selección de videos de nuestros proyectos educativos publicados en el canal de YouTube de Technostream. La selección consta de tres partes: nuevos cursos en el canal para el año académico 2018-2019, los cursos más vistos y los videos más vistos.

Nuevos cursos en el canal Technostream para el año académico 2018-2019



Bases de datos (tecnosfera)



El objetivo del curso es estudiar la topología, la diversidad y los principios básicos del funcionamiento de los sistemas de almacenamiento y datos, así como los algoritmos subyacentes a los sistemas centralizados y distribuidos, para demostrar los compromisos fundamentales inherentes a una u otra solución.

El curso revela una variedad de soluciones para almacenar datos en proyectos de Internet en tres dimensiones:

  • modelo de datos continuo;
  • continuidad de consistencia de datos;
  • Un continuo de algoritmos de almacenamiento de datos.

El programa del curso está destinado a programadores de sistemas, desarrolladores de DBMS y programadores aplicados, creadores de sistemas de colas en Internet.

Python Aplicado (Technopark)



El curso presenta el lenguaje Python, uno de los idiomas más populares y populares del mercado de TI en la actualidad. La demanda del lenguaje no viene desde cero: la simplicidad de entrada y sintaxis, la opción más rica de herramientas para resolver todo tipo de tareas, esto y mucho más ha llevado al hecho de que Python se usa ampliamente en todo el mundo. Gracias a este curso, puedes unirte al ecosistema del idioma.

Aprenderás:

  • Programa en Python
  • Escriba código compatible de alta calidad;
  • Construir un proceso de desarrollo de productos de software;
  • Interactúa con servicios y bases de datos en línea.

Programación avanzada de C / C ++ (Technosphere)



Se familiarizará con las herramientas y prácticas utilizadas en el desarrollo moderno, obtendrá las habilidades para escribir código C ++ correcto y flexible. El curso ayudará a adquirir las habilidades necesarias para que los especialistas en desarrollo de software participen en proyectos de desarrollo industrial en lenguajes C ++, incluso para cubrir puestos de aprendices para desarrolladores del lado del servidor de aplicaciones altamente cargadas.

Cada lección consiste en una conferencia (2 horas) y una tarea práctica.

Programacion del sistema | Laboratorio Tarantool (tecnosfera)




El curso destaca el sistema operativo del dispositivo basado en el kernel GNU / Linux, la arquitectura del kernel y sus subsistemas. Las formas de interacción con el sistema operativo se dan y describen. El material del curso es lo más cercano a la realidad posible, lleno de ejemplos.

Proyecto de TI y gestión de productos (Technosphere)



El objetivo del curso es obtener conocimiento en el campo de la gestión de productos y proyectos utilizando el ejemplo de Mail.ru Group, para comprender el papel del gerente de productos y proyectos, para conocer las perspectivas de desarrollo y las características de la gestión de productos y proyectos en una gran empresa.

El curso examinará la teoría y la práctica de administrar el producto y todo lo que está dentro (o al lado): procesos, requisitos, métricas, plazos, lanzamientos y, por supuesto, sobre las personas y cómo comunicarse con ellos.

Desarrollo de Android (Technopolis)



El curso ayudará a dominar los conocimientos y habilidades necesarios para desarrollar software para Android. Aprenderá las API de Android, las Herramientas para desarrolladores (SDK), las bibliotecas populares y más. Además, en el proceso de aprendizaje, aprenderá no solo cómo desarrollar la aplicación, sino que también proporcionará tolerancia a fallas. Después de eso, podrá crear de forma independiente aplicaciones y controlar (técnicamente, a nivel de administrador) su desarrollo.

Introducción a Java (Technopolis)



El curso está dedicado a aprender los conceptos básicos de Java 11, trabajar con Git y conocer algunas prácticas de prueba y patrones de diseño de sistemas. Diseñado para personas con un mínimo conocimiento básico de programación en cualquier idioma. Durante el curso, podrá dominar Java y crear una aplicación completa.

Uso de bases de datos (Technopolis)



Recibirá un amplio conocimiento sobre el trabajo con bases de datos. Aprenda cómo elegir los tipos de bases de datos más adecuados para su proyecto, realizar consultas, modificar datos, dominar los conceptos básicos de SQL y mucho más.

Cursos más vistos en el canal Technostream para el año académico 2018-2019


Pruebas de calidad y software (Technosphere, 2015)



Todo sobre las pruebas actuales y las metodologías de garantía de calidad para aplicaciones web modernas: fundamentos teóricos, pruebas manuales, documentación, cobertura de código con pruebas, seguimiento de errores, herramientas, automatización de pruebas y mucho más.

Desarrollo Java (Technosphere, 2018)



Este curso tiene todo lo que un principiante necesita en el mundo Java. No entraremos en los detalles de la sintaxis, sino que inmediatamente tomaremos Java y haremos cosas interesantes. Suponemos que no conoce Java, pero está programado en cualquier lenguaje de programación moderno y está familiarizado con los conceptos básicos de OOP. El énfasis está en el uso de una pila de batalla tecnológica (sí, esto es exactamente lo que usan muchas empresas). Algunas palabras de moda: Java-stack (Jersey, Hibernate, WebSockets) y toolchain (Docker, Gradle, Git, GitHub).

Administración de Linux (Technotrack, 2017)



El curso examinó los conceptos básicos de la administración del sistema de servicios de Internet, asegurando su tolerancia a fallas, rendimiento y seguridad, así como las características del dispositivo con sistema operativo Linux, que es el más utilizado en tales proyectos. Como ejemplo, utilizamos distribuciones de la familia RHEL 7 (CentOS 7), un servidor web nginx, MySQL DBMS, sistemas de respaldo de bacula, sistemas de monitoreo Zabbix, sistemas de virtualización oVirt y equilibrador de carga ipvs + keepalived.

Tecnología web. Desarrollo en DJANGO (Technopark, 2016)



El curso está dedicado al desarrollo del lado del servidor de las aplicaciones web, su arquitectura y el protocolo HTTP. Al final del curso, aprenderá cómo: desarrollar aplicaciones en Python, usar marcos MVC, estudiar el diseño de páginas HTML, sumergirse en el tema del desarrollo web y puede elegir tecnologías específicas.

Programación Go (Technosphere, 2017)



El propósito del curso es dar ideas básicas sobre el lenguaje de programación Go (golang) y su ecosistema. Usando un ejemplo de un juego de texto simple, consideraremos todas las tareas principales que enfrenta el desarrollador de aplicaciones web modernas en grandes proyectos, con su implementación en Go. El curso no establece la tarea de enseñar programación desde cero; se requerirán habilidades básicas de programación para la capacitación.

Los videos más vistos en el canal Technostream para el año escolar 2018-2019


Administración de Linux. Introducción (Technopark, 2015)



Este video habla sobre la historia de Linux, los desafíos que enfrenta el administrador de este sistema operativo, así como los desafíos que le esperan al cambiar de Windows a Linux y cómo adaptarse.

Ir a la programación. Introducción (Technosphere, 2017)



El video está dedicado a la historia del lenguaje Go, una descripción de las ideas clave integradas en el lenguaje y los conceptos básicos: cómo instalar y configurar el entorno Go, cómo crear el primer programa, cómo trabajar con variables y estructuras de control.

Video promocional inspirador sobre aquellos que acuden a TI, sin importar qué



Este es un video promocional dedicado a reclutar estudiantes para nuestros programas educativos en las universidades.

Linux The Basics (Tech Track, 2017)



Este video habla sobre el dispositivo Linux, sobre el uso del shell de comandos, sobre los derechos de acceso para diferentes usuarios. Descubrirá qué procesos y condiciones existen en Linux, qué protocolos se utilizan y cómo administrar el entorno del usuario.

Desarrollo de Android. Introducción (Technotrek, 2017)



Esta lección introductoria describe las características del desarrollo móvil y el ciclo de vida de una aplicación móvil. Aprenderá exactamente cómo existe la aplicación móvil en el sistema operativo, qué necesita para desarrollar la aplicación, cómo configurar el entorno de desarrollo y crear su propio "¡Hola, mundo!"
