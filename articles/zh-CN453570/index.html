适用于macOS的Microsoft Edge

上个月，我们宣布了适用于Windows 10的下一版本Microsoft Edge 的第一批预构建版本。现在，我们很高兴地宣布Microsoft Edge Canary频道也已可用于macOS。 现在，在Microsoft Edge Insider网站上，您可以找到具有macOS和Windows 10的计算机的内部版本，并且很快将扩展对Windows版本的支持。  



Mac版Microsoft Edge


适用于macOS的Microsoft Edge浏览器将为用户提供与Windows初始版本相同的新功能，但其界面将适用于Mac平台。 我们将根据macOS用户的习惯和期望来更改界面和功能。



我们努力使Microsoft Edge成为具有独特功能和相关服务的世界一流的浏览器。 在此第一个版本中，我们对用户界面进行了一些更改，以使其与Microsoft的设计保持一致，但同时又使其自然适用于macOS。


所做的更改涉及macOS字体，菜单，键盘快捷键，标题和其他元素。 在将来的版本中，我们将继续更改浏览器的外观和功能，我们将进行实验并响应用户的反馈。 因此，请使用“发送反馈”表情图告诉我们您的印象。



此外，基于Mac平台的特殊硬件功能，我们正在开发与macOS专有的用户交互方式。 因此，用户可以使用触摸面板（触摸栏）执行操作并访问上下文命令。 例如，您可以访问网站快捷方式，在标签和控制视频之间切换，以及使用触控板手势使用熟悉的导航。



推出适用于macOS的Microsoft Edge Insider提要


可通过预发布渠道获得新的macOS Microsoft Edge预构建版本，我们将其称为Microsoft Edge Insider渠道。 我们首先启动Canary构建渠道，您可以在Microsoft Edge Insider网站上获得该渠道。 此通道已可用于macOS 10.12及更高版本。 不久我们将打开Dev通道，当它可用时，您可以与Canary并行下载并安装此程序集。 您可以在过去一个月的帖子中找到有关我们的方法的更多信息以及不同渠道的期望。



适用于Web开发人员的一致平台和工具


由于我们的新版本基于Chromium，因此您在两个版本的Microsoft Edge中都获得了相同的渲染-适用于Windows和macOS。 此外，您还可以使用强大的开发人员工具。


Web开发人员首次可以在macOS上的Microsoft Edge中测试站点和Web应用程序，并确保它们在所有平台的下一版本的Microsoft Edge中均能正常工作。 （请注意，仍应单独监视特定于平台的功能，例如在Windows 10中解密PlayReady内容，以在这些平台上获得最佳体验。）


与Windows的预构建版本一样，我们的macOS新版本还支持渐进式Web应用程序，该应用程序已安装在标准上，并且可以使用浏览器中的开发人员工具进行检查和调试。 我们正在努力确保此类应用程序可以与其他平台应用程序轻松配合使用，因此在安装后，它们将像其他任何应用程序一样显示在Dock，App Switcher和Spotlight中。  


您的反馈对我们很重要


我们很高兴与您分享我们用于macOS的Microsoft Edge Canary的第一版！ 我们需要您的反馈意见，以使浏览器更好，因为我们认为该浏览器对于macOS用户应该尽可能地方便。 我们希望您今天尝试预览，并期待您的反馈和参与Microsoft Edge Insider社区。


如果您有任何问题，或者想发表反馈或提出建议，可以在Microsoft Edge Insider社区论坛上留言，在Twitter上与我们联系，或者只使用Microsoft Edge菜单中的“发送反馈”命令，与我们分享您的想法。


如果您是Web开发人员，并且遇到了在Chromium中重现的问题，则最好报告Chromium社区中的错误。 若要报告当前版本的Microsoft Edge中的问题，请继续使用EdgeHTML问题跟踪器 。
