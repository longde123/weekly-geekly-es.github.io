10 utilitas ApexSQL gratis untuk mengelola database Microsoft SQL Server



Halo, Habr! Kami banyak bekerja dengan Quest Software , dan tahun ini mereka mengakuisisi ApexSQL , produsen manajemen database dan solusi pemantauan Microsoft SQL Server. Di Rusia, bagi kami, sedikit yang diketahui tentang orang-orang ini. Di beranda mereka menulis "Killer tools for SQL Server". Kedengarannya mengancam. Kami punya ide untuk memperkenalkan vendor ini dan mencari tahu apa yang begitu mematikan dalam keputusan mereka. Mari kita mulai dengan produk bebas hype bagian. Setelah mereka akan ada bagian singkat tentang produk komersial. Saya harap Anda menemukan program ini bermanfaat. Saya minta kucing.

Dalam artikel ini:

  1. ApexSQL Compare adalah alat untuk membandingkan kode SQL, file, dan folder. Juga berfungsi sebagai ekstensi untuk SSMS atau Visual Studio.
  2. ApexSQL Decrypt - stream dekripsi objek dalam SQL Server: prosedur, fungsi, pemicu dan tampilan. Juga berfungsi sebagai ekstensi untuk SSMS atau Visual Studio.
  3. ApexSQL Discover - Penemuan instance SQL Server dan layanan SSRS, SSAS, dan SSIS terkait.
  4. ApexSQL Refactor adalah alat untuk refactoring dan memformat kode SQL. Berfungsi sebagai ekstensi untuk SSMS atau Visual Studio.
  5. ApexSQL Model - Membuat diagram objek SQL Server. Juga berfungsi sebagai ekstensi untuk SSMS atau Visual Studio.
  6. ApexSQL Plan adalah alat untuk mengoptimalkan rencana Eksekusi. Juga berfungsi sebagai ekstensi untuk SSMS.
  7. ApexSQL Lengkap - alat ini secara otomatis melengkapi pernyataan SQL dan memungkinkan Anda untuk menambahkan cuplikan Anda sendiri (pintasan keyboard untuk pelengkapan otomatis). Juga berfungsi sebagai ekstensi untuk SSMS atau Visual Studio.
  8. ApexSQL Propagate adalah alat untuk mengeksekusi kode SQL pada banyak basis data sekaligus.
  9. ApexSQL Search adalah utilitas untuk menemukan data dan objek dalam isi SQL Server. Berfungsi sebagai ekstensi untuk SSMS atau Visual Studio.
  10. ApexSQL DevOps Toolkit - alat untuk membuat saluran pipa CI / CD. Satu-satunya produk yang tercantum di sini memiliki konsol web.

Setiap produk adalah bagian dari fungsionalitas solusi ApexSQL berbayar. Di akhir artikel, kami menjelaskan secara singkat apa yang mereka miliki dari solusi komersial.

1. Bandingkan ApexSQL




Video pendek tentang mekanisme kerja dan peluang


Di Bandingkan, Anda bisa membandingkan file, folder, dan objek SQL Server. Perbandingan dua arah dan generasi selanjutnya dari skrip didukung untuk mencekik objek yang berbeda. Dalam antarmuka, utilitas dapat menyorot sintaksis C #, C ++, HTML, JavaScript, PostgreSQL, Python, T-SQL, Visual Basic, XML.



Fitur utama adalah integrasi ke dalam SSMS dan Visual Studio, memanggil objek dari berbagai instance database untuk perbandingan langsung dari Object Explorer dan pekerjaan pendukung melalui CLI .

2. Dekripsi ApexSQL




Video pendek tentang mekanisme kerja dan peluang


Di Dekripsi, Anda dapat mendekripsi prosedur, fungsi, pemicu, atau melihat dalam satu klik dari antarmuka SSMS atau Visual Studio.



Di antarmuka, Anda dapat melihat skrip dekripsi DDL. Mendukung koneksi simultan ke beberapa instance SQL Server.

3. Temukan ApexSQL




Identifikasi contoh SQL Server, SSRS, SSAS, dan SSIS yang ada berdasarkan rentang IP dan lihat informasi terperinci.



Dari antarmuka, Anda dapat mengonfigurasi deteksi basis data sesuai jadwal. ApexSQL Discover dapat mengirim pemberitahuan dengan hasil penemuan ke email.

4. ApexSQL Refactor




Video pendek tentang mekanisme kerja dan peluang


Anda dapat memformat kode berdasarkan profil kotak atau menyesuaikan kode Anda agar terlihat lebih akrab bagi Anda. Refactor dapat menyorot variabel yang tidak digunakan dalam kode, merangkum blok kode dalam prosedur, memperluas "*" ke daftar lengkap.



Salah satu fitur yang berguna adalah tabel partisi untuk memisahkan kolom yang paling jarang digunakan. Refactor akan melaporkan objek yang terkena dampak pemisahan ini.



Juga di antarmuka Refactor, Anda dapat dengan aman mengubah nama objek dengan perubahan semua dependensi.



Fungsi-fungsi ini dan lainnya dapat ditemukan lebih detail pada halaman deskripsi utilitas .

5. Model ApexSQL




Utilitas memvisualisasikan objek SQL Server dan hubungan di antara mereka. Di antarmuka yang sama, Anda dapat mengubah konfigurasi objek dan menghasilkan kode DDL yang sesuai. Semua visualisasi dapat diekspor sebagai gambar.

Model Fungsionalitas berkembang dengan produk komersial Diff dan Doc . Menggunakan produk pertama, Anda dapat membandingkan model dengan keadaan saat ini di database atau membandingkan model dua database yang berbeda, dan produk kedua diperlukan untuk mendokumentasikan objek database.

6. Paket ApexSQL




Video pendek tentang mekanisme kerja dan peluang


Ini adalah alat untuk mengoptimalkan kinerja. Menggunakan ApexSQL Plan, Anda juga dapat mengekspor rencana ke laporan dalam format HTML dan XML. Kasus penggunaan yang sering digunakan untuk utilitas ini membandingkan paket saat ini dan yang dimodifikasi untuk berbagai karakteristik.



ApexSQL Plan mendukung pembongkaran dari toko kueri dan menghadirkan karakteristik kueri ini sebagai dasbor.

7. Lengkap ApexSQL




Video pendek tentang mekanisme kerja dan peluang


Akselerator Penulisan Kode. Di antarmuka utilitas, Anda dapat mengonfigurasi alias, cuplikan, dan penyorotan kode.

8. ApexSQL Propagate




Di ApexSQL Propagate, Anda dapat mengkonfigurasi eksekusi kode pada berbagai database SQL Server, dan menentukan urutan eksekusi.



Hasil dari kode yang dieksekusi diuraikan oleh utilitas dan dicatat untuk kenyamanan melihat hasil eksekusi.

9. Pencarian ApexSQL




Video pendek tentang mekanisme kerja dan peluang


Alat mencari struktur database. Dari sini, Anda dapat dengan aman mengganti nama objek dan melacak hubungannya. Hasil pencarian diekspor ke HTML, CSV dan Excel.

10. ApexSQL DevOps Toolkit




Video pendek tentang mekanisme kerja dan peluang


Di bawah kap utilitas, kode PowerShell berfungsi. Build, Review, Document and Deploy tersedia untuk pengembang dari antarmuka ApexSQL DevOps Toolkit. Di sini Anda juga dapat menghasilkan data sintetis, secara otomatis menjalankan tes unit, perubahan dokumen. Ini dapat dihubungkan sebagai plug-in ke TeamCity, Jenkins dan lainnya.

Basis Pengetahuan ApexSQL memiliki artikel menarik tentang DevOps di lingkungan basis data.
