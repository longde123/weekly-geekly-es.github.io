一般情况下，如何选择Grandstream SIP电话？



我是Grandstream分销商Factor Group的工程师，并参与各种项目的售前咨询。 我们的客户不仅是先进的公司，而且还是初学者。 因此，我不仅经常谈论Grandstream设备的线路，而且谈论SIP电话的一些基本原理。 我的咨询和支持的结果就是这篇文章。 我试图告诉一般信息和有趣的细节。 我希望这些信息会很有趣-这是我第一次写文章。

因此，如果您决定切换到SIP电话，或者想用新设备替换出现故障或过时的设备，那么您肯定需要IP电话。 本文旨在帮助您了解Grandstream Networks电话线的全部范围，并为自己或您的员工选择最佳的设备。

Grandstream有6类IP电话-基本，中档，高端，高级，DECT和Wi-Fi电话。 在本文中，我将分析IP电话的三类-基本，中档和高端。 它们在价格范围，外观以及功能上都有所不同。 例如，高端类别的电话功能最强大，外观漂亮，但是价格也相应。 但是我们将按顺序进行，分别考虑每个类别。

基本IP电话


图1.基本IP电话

GXP1600系列IP电话是专门为中小型企业以及家庭用户设计的。 这些简单的设备具有友好的界面，支持IP电话的所有基本功能以及最大的易用性。

此手机类别包括六种型号：GXP1610，GXP1615，GXP1620，GXP1625，GXP1628，GXP1630。
图2. GXP1610 / GXP1615

GXP1610和GXP1615电话彼此之间的不同之处仅在于GXP1615具有PoE（以太网供电）技术支持，该技术除了可以为电源供电之外，还可以通过网络电缆为设备供电。 与我们正在考虑的其他类别的设备相比，只有这些手机没有背光并且不支持高清音频。

这些型号支持一个SIP帐户和两条线路，这将允许它们分配一个分机号码并同时处理两个呼叫。 它们还支持三路音频会议，并具有双10 / 100Mbps网络接口（LAN和PC）。 如果只有一台计算机连接到员工工作场所的本地网络，则使用PC连接器非常方便。 当您通过网络电缆将计算机连接到PC电话端口时，电话将充当交换机，并且不需要主交换机上的其他端口。 基本上，基本，中档，高端和高级类别的所有电话都支持此模式。

所有Grandstream台式电话可用的另一个功能是RJ9耳机端口。 但是，您需要做一点保留。 制造商本身已经测试了多种类型的耳机。 他们的支持得到保证，您可以在这里研究第16系列手机的清单，以及在这里的所有其他手机的清单。
图3. GXP1620 / GXP1625

GXP1620和GXP1625电话也彼此不同，仅在于GXP1625具有PoE技术支持。 而且我注意到，在所有电话系列中，只有两种型号-GXP1610和GXP1620不支持该技术。

GXP1620和GXP1625型号支持两个SIP帐户和两条线路，具有背光，支持高清音频，三路音频会议和双10 / 100Mbps网络接口。
图4. GXP1628

GXP1628型号与GXP1625型号相似，不同之处在于八个具有BLF功能的MPK（多功能键），使您可以监视同事的繁忙电话状态。 多任务MPK键可让您一键式执行最常见的操作。 通常，将这些键分配给最常拨打电话的用户快速呼叫或进行呼叫转移，这消除了将电话号码存储在内存中的需要，并且不会浪费时间在电话簿中进行搜索。 快捷键的存在极大地方便了IP电话的日常工作，特别是对于拨打许多电话的员工而言。 另外，该型号与GXP1625的不同之处在于它具有千兆位网络端口。
图5. GXP1630

和GXP1630型号相似于GXP1628，但也有自己的区别。 GXP1630支持三个SIP帐户，三个线路，并且可以组织四路音频会议。

中档IP电话


图6.中档IP电话

GXP1700系列IP电话是为中小型企业设计的。 这些时尚的设备具有成长中的组织所需的所有必要功能。 同时，GXP1700系列的设备价格合理，从而减轻了公司预算的负担。 除了功能和价格外，这些手机还具有出色的设计和周到的人体工程学设计，因此与它们一起工作是一种乐趣。

此电话类别包括四个电话：GXP1760，GXP1760w，GXP1780和GXP1782。 我将立即注意到GXP1760w手机具有内置的双频Wi-Fi模块，因此它属于Wi-Fi手机，我们将在另一篇文章中对其进行详细介绍。
图7. GXP1760

让我们从GXP1760开始。 该型号具有带背光的两色LCD屏幕，支持三个SIP帐户，六个线路以及可组织五路音频会议的功能。 显示屏侧面的六个键以及使用左右导航键在主屏幕上切换四页的功能总共提供了24个虚拟BLF键
图8. BLF键

对于个性化选项，GXP1760支持各种铃声以及天气预报等Web应用程序。 GXP1760还配备了Kensington安全插槽，以防止其从检查站，酒店大厅，接待区等公共场所被盗。 两个10/100 Mbit / s端口用于连接网络，可以通过Web界面以及各种自动调整技术来配置设备：

  • 使用Grandstream的IP-PBX时自动配置；
  • 通过DHCP选项66自动加载.xml配置文件；
  • 支持自动调整服务3CX和Broadsoft；
  • Grandstream自动化预配置系统（GAPS）是一个自动化系统，允许IT管理员和电信运营商直接安全地控制设备预配置，并从中央控制面板优化Grandstream产品的大规模部署。

图9. GXP1780 / 1782

GXP1780与GXP1760的不同之处在于，它支持四个SIP帐户和八个线路。 该模型具有八个物理BLF键，并且可以通过切换主屏幕来提供32个虚拟BLF键。 GXP1782与GXP1780的不同之处仅在于千兆位网络接口。

除了用于连接头戴式耳机的RJ9端口外，1700系列电话还具有用于外部EHS适配器的单独的EHS端口。

图10. EHS适配器的端口

高端IP电话


图11.高端IP电话

高质量的GXP2100系列IP电话是为需要高级电话功能的用户设计的。 这些设备具有丰富的功能。 高品质的高清扬声器，清晰的彩色显示屏，自定义桌面墙纸的功能，各种BLF按键组合以及在所有型号上均支持其他扩展面板，内置蓝牙，PoE，千兆网络端口的功能-所有这些都是该系列高级IP电话的显着特征GXP2100。

此类别包括5个电话型号：GXP2130，GXP2135，GXP2140，GXP2160和GXP2170。
图12. GXP2130

该系列中的第一个型号是GXP2130。 这款手机配有2.8英寸彩色LCD，分辨率为320x240像素。 它支持三条线路和三个SIP帐户，并提供安排四路音频会议的功能。 这款手机的一个独特功能是八个带有BLF功能的MPK键。 但是，类似于中档电话，高端设备还具有配置BLF虚拟密钥的能力。 显示屏左侧有3个物理键，还有4个滚动页面，除8个MPK键外，还提供了12个虚拟BLF键。

图13. BLF键

我将分别告诉您为什么该系列手机中需要蓝牙。 21XX系列的所有高端手机均支持蓝牙技术，并大大扩展了手机的功能。 首先，您可以连接外部蓝牙耳机，与使用有线耳机时一样，这将使您免于“绑定”到手机。 另外，可以将移动电话连接到IP电话。

图14.手机帐户

通过这种方案，IP电话本身将充当手机的耳机。 连接后，手机的“帐户”将自动出现在IP电话上，并且可以在不使用手机的情况下通过IP电话拨打电话，这就是所谓的免提模式。 拨入手机的电话也将显示在IP电话上。 并且，如果您需要呼叫手机上已有的联系人，则可以帮助您通过蓝牙在IP电话和手机之间同步联系人。
图15. GXP2135

下一个型号是GXP2135。 它与上一个的区别在于行数，SIP帐户，存在大量的BLF密钥以及不存在MPK密钥。 该电话支持八条线路和四个SIP帐户。 此电话上有八个BLF键，并且滚动主屏幕的功能将这些键的数量增加到32。
图16. GXP2140

GXP2140，GXP2160和GXP2170手机的三种最新型号均具有放大的屏幕。 它的尺寸：4.3英寸，分辨率为480x272像素。 GXP2140支持四线，四个SIP帐户，并且可以组织五路音频会议。 与其他型号相似，具有BLF功能的屏幕左侧四个键使创建16个虚拟BLF键成为可能。

该型号的独特之处在于您可以将GXP2200EXT扩展面板连接到它。

图17. GXP2200EXT
图18. GXP2140和GXP2200EXT

GXP2200EXT是带有图形LCD显示屏的可选键盘，为Grandstream GXP2140，GXP2170和GXV3240 IP电话提供附加功能，灵活性和可扩展性。 该模块具有一个大的128 x 384图形LCD显示屏和20个可编程键（每个键可以发出两种颜色的光），并且能够切换屏幕的功能在一个面板上提供了40个键。 总共最多可以将四个GXP2200EXT面板连接到Grandstream IP电话。 因此，当使用4个扩展模块时，最多支持160个快捷键。 扩展模块已连接，接通电源并由电话控制，在连接模块后立即提供附加功能。
图19. GXP2160

我们的下一款手机是GXP2160。 它支持六个线路，六个SIP帐户以及一个五路音频会议的组织。 屏幕左侧的六个BLF键以及滚动主屏幕的功能最多可以添加24个虚拟BLF键。 该电话不支持连接其他扩展面板，但具有类似的集成面板。 它具有24个具有BLF功能的MPK键。
图20. GXP2170

我们的高端Grandstream GXP2170 IP电话支持12条线路，6个独立的SIP帐户，并且能够组织五路音频会议。 由于屏幕侧面具有12个具有BLF功能的键并具有滚动功能，因此您最多可以配置48个虚拟BLF键。 该电话还支持连接其他扩展面板GXP2200EXT，非常适合在工作中积极使用电话通讯的员工。
