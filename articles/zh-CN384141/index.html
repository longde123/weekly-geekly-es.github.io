Windows历经岁月：Microsoft操作系统三十年

用于MS-DOS的Windows图形外壳程序的第一个版本发布于将近30年前，离周年纪念日还剩几个月。几十年后，Windows在个人计算机市场中的份额超过了90％。超过一半的台式机都在Windows 7上运行。

让我们记住自1985年以来我们所见过的Windows版本。



Windows图形外壳的第一个版本于1985年发布：11月20日，它出现在零售连锁店中。为避免Mac OS出现法律问题，开发人员未使用购物篮，并且窗口不能彼此重叠。此版本的Windows在屏幕底部没有任务栏。“在DOS上无用的附加组件”包括一个游戏-黑白棋。



两年后的1987年，微软发布了Windows 2.0版本。这里出现了我们从小就熟悉的Paint应用程序，手表，MS-DOS文件管理器和终端。在2.0版发布后不久，该公司首先将Word和Excel移植到Windows。



1990年的Windows 3.0包括程序管理器，对VGA和虚拟内存的支持，新的“ 3D”设计以及游戏“围巾”。游戏不是为了娱乐办公室工作人员，而是为了训练用户使用鼠标的技巧。



1992年，Microsoft发布了Windows 3.1，向系统中添加了可缩放的TrueType字体和Minesweeper游戏，以训练用光标敲击屏幕的准确性。系统中首次出现了对多媒体内容的支持。



Windows for Workgroups 3.1于1992年发布。它包括对网络的本地支持，尤其是局域网LAN，在当时的业务中迅速流行。

i.imgur.com/cy6jMEK.png

1993年可以称为Windows操作系统的诞生年。在Windows NT 3.1之前，所谓的Windows是MS-DOS的图形外壳。现在，Microsoft推出了专为现代工作站设计的32位操作系统。在外部，该操作系统与Windows 3.1相同。



2015年8月，Windows 95启用20周年。计算机系统不仅可以用于极客，而且还可以用于家庭主妇。 “开始”按钮和任务栏简化了系统的使用。



Windows NT 4.0将类似于Windows 95的接口与强大的Windows NT内核结合在一起。在未来几年中，该系统已成为Microsoft最受欢迎的操作系统。



Windows CE是用于手持计算机，智能手机和嵌入式系统的操作系统。它于1996年11月发布，当时是桌面操作系统的“精简版”版本。该系统的最新版本7.0于2011年3月发布。



Internet Explorer Web浏览器已成为1998年发布的Windows 98不可或缺的一部分。该系统具有一个工具栏，用于快速启动应用程序和对USB的本机支持。该系统已变得非常流行，尽管不是最稳定的。



Windows 2000是Windows NT操作系统的版本5.0。紧随其后的是XP，Vista，Windows 7和8。该系统的第一个公开测试版发布于1997年，该公司于2000年2月17日推出了最终版本。系统支持Active Directory的服务器部分，NTFS 3.0文件系统和加密的EFS，并使用Active Desktop接收用户界面。

要使用Windows 2000，您必须在硬盘驱动器上拥有133 MHz，32 MB RAM和2 GB内存的处理器。



Windows ME于2000年发布，遭到评论家和用户的批评，称其极为不稳定，谈论的是持续的死机和崩溃。



我购买的第一台计算机运行Windows XP。我在2006年买了一台计算机，微软在2001年发布了XP。WindowsXP成为当时最流行的Microsoft操作系统。

在2008年6月，微软宣布它将在2014年之前支持XP。对系统的一般支持于2009年终止，延长至2014年4月8日。



2007年，Microsoft 针对Mac OS X操作系统发布了Windows Vista，但半透明的窗口，大图标和新的“开始”菜单并没有给批评家留下深刻的印象。



Windows Server 2008服务器操作系统取代Windows Server 2003，成为新一代Vista系列系统的代表。



Windows 7是Windows最成功的发行版之一。微软保留了漂亮的界面（Aero）和图标，消除了许多错误，并使系统更加可靠。

从Windows XP和Windows Vista发行到现在已经过去了六年，而从Vista和7发行到现在只有三年。



2012年，Microsoft推出了带有Metro平铺界面的Windows 8。该界面在启动系统后首次出现，类似于桌面。幸运的是，该操作系统继续支持经典Windows桌面，但是删除了“开始”按钮。该系统是为触摸屏设计的，尽管您可以在带有常规显示器的计算机上使用它。



Windows 8.1-2014年以来Windows 8的更新。现在，启动时用户可以看到工作屏幕而不是Metro屏幕，尽管没有旧的开始菜单，但出现了“开始”按钮。



2015年，Microsoft推出了Windows 10，可以从Windows 7和Windows 8进行升级。Microsoft返回了开始菜单。

最终用户有一些不愉快的时刻：2015年8月，洪流跟踪程序开始禁止计算机上的Windows 10用户，因为该系统向Microsoft服务器发送了过多的数据。
