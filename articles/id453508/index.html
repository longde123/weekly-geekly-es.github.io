Manusia tanpa smartphone

Saya berumur 33 tahun, saya seorang programmer dari St. Petersburg dan saya tidak memiliki dan tidak pernah memiliki smartphone. Bukannya saya tidak membutuhkannya - saya membutuhkannya, bahkan sangat banyak: saya bekerja di bidang IT, semua anggota keluarga saya memilikinya (anak itu sudah ketiga), saya harus mengelola pengembangan ponsel, termasuk, saya punya situs web sendiri ( ramah ponsel 100%), dan saya bahkan pindah ke Eropa untuk bekerja. Yaitu Saya bukan semacam pertapa, tetapi orang yang cukup modern. Saya menggunakan telepon tombol biasa dan selalu hanya menggunakan itu.



Saya secara berkala menemukan artikel seperti "orang sukses tidak menggunakan smartphone" - ini omong kosong! Smartphone menggunakan segalanya: sukses dan tidak terlalu, miskin dan kaya. Saya belum melihat seorang pria modern tanpa smartphone - rasanya tidak memakai sepatu, atau tidak menggunakan mobil - Anda tentu saja bisa, tetapi mengapa?

Semuanya dimulai sebagai protes terhadap smartphoneisasi massal, dan itu berlanjut sebagai tantangan selama sekitar 10 tahun - saya bertanya-tanya berapa banyak saya dapat menahan tren modern, dan apakah itu mungkin. Ke depan, saya akan mengatakan: itu mungkin, tetapi itu tidak masuk akal.

Saya akui bahwa banyak orang berpikir untuk menolak menggunakan smartphone. Saya ingin berbicara tentang pengalaman saya di sini sehingga mereka yang berniat melakukan eksperimen seperti itu dapat mengevaluasi kelebihan dan kekurangan dari pengalaman orang lain.

Kisah ini tentu memiliki pro dan kontra, dan mereka cukup jelas.

Jadi, inilah kelebihan yang dapat saya identifikasi berdasarkan prioritas:

  • Saya tidak perlu khawatir tentang pengisian daya. Saya mengisi daya telepon sekitar sekali setiap dua minggu. Terakhir kali saya pergi berlibur, saya bahkan tidak mengambil biaya dengan saya, karena saya yakin bahwa telepon tidak akan habis selama waktu ini - itu terjadi;
  • Saya tidak fokus pada pemberitahuan konstan dan melihat pembaruan pada menit gratis yang diberikan. Ini terutama berlaku untuk pekerjaan - Anda tidak terlalu terganggu - itu berarti Anda lebih fokus pada pekerjaan;
  • Saya tidak menghabiskan uang untuk telepon baru, tidak mengikuti pembaruan, dan tidak merasa tidak nyaman ketika salah satu teman saya memiliki telepon yang lebih baik daripada milik saya, atau ketika telepon saya lebih baik dari teman-teman saya;
  • Saya tidak mengganggu kenalan yang terus-menerus menempel di telepon (di pesta, misalnya, atau hanya ketika bertemu). Tapi ini lebih tentang pengasuhan dan kesopanan;
  • Saya tidak perlu membeli Internet seluler - begitu-begitu plus, mengingat harganya cukup rendah;
  • Saya dapat mengejutkan orang dengan mengatakan bahwa saya tidak menggunakan smartphone dan tidak pernah menggunakannya - dan semakin jauh, mereka semakin terkejut. Saya harus mengatakan bahwa saya sendiri akan terkejut jika saya bertemu orang seperti itu - sejauh ini satu-satunya yang saya tahu di posisi yang sama adalah nenek saya, yang berusia 92 tahun.

Nilai tambah utama adalah saya tidak bergantung pada ketersediaan outlet terdekat. Sungguh menyedihkan melihat bagaimana orang pertama kali "menempel" pada soket, di mana pun mereka berada, atau cenderung mengambil tempat yang lebih dekat dengan mereka. Saya benar-benar tidak ingin saya memiliki ketergantungan seperti itu, dan ini adalah salah satu poin utama dalam "daftar perlawanan" saya. Ketika hanya ada satu tongkat pengisian daya yang tersisa di ponsel saya, ini berarti saya punya beberapa hari lagi sebelum duduk.

Tentang menyemprotkan perhatian juga merupakan poin penting. Itu benar-benar memakan banyak daya. Mungkin tepat untuk menyisihkan beberapa slot waktu per hari untuk memeriksa semua notifikasi dan membalas pesan. Tapi, mungkin, mudah bagi saya untuk berbicara sebagai orang dari luar.

Tetapi kontra, juga dalam urutan prioritas:

  • kekurangan kamera di tangan adalah rasa sakit. Saya sudah melewatkan seribu poin yang harus ditangkap untuk memori atau untuk berbagi dengan orang yang dicintai. Ketika Anda perlu mengambil foto dokumen, atau sebaliknya untuk mendapatkan foto, situasinya juga tidak biasa;
  • Saya bisa tersesat bahkan di kota asal saya. Ini lebih merupakan fitur memori, dan mudah diselesaikan dengan kehadiran navigator. Ketika saya harus pergi ke tempat baru dengan mobil, saya menggunakan peta kertas atau mengingat rute di rumah di laptop saya;
  • tidak ada cara untuk "mendistribusikan" Internet ke laptop - Anda harus selalu mencari Wi-Fi terbuka, atau bertanya kepada teman;
  • Saya sangat kekurangan seorang juru bahasa di saku saya jika saya di luar negeri, atau Wikipedia ketika datang untuk mempelajari sesuatu yang baru;
  • Saya bosan dalam antrian, di jalan, dan di tempat lain di mana semua orang normal membuka kaset, mendengarkan musik, memutar atau menonton video;
  • beberapa orang memandang saya dengan simpati atau tidak sehat ketika mereka mengetahui bahwa saya tidak memiliki smartphone. Saya tidak ingin menjelaskan kepada semua orang alasannya - saya sudah lelah;
  • Sulit bagi saya untuk tetap berhubungan dengan teman-teman yang mengobrol di Whatsapp, misalnya. Saya, sebagaimana layaknya seorang programmer, sedikit introvert, dan saya tidak suka ketika mereka memanggil saya dan saya sendiri tidak begitu suka menelepon. Komunikasi melalui pesan adalah cara terbaik untuk tetap berkomunikasi;
  • Baru-baru ini, layanan mulai muncul yang tidak mungkin digunakan tanpa smartphone - otentikasi dua faktor melalui push, misalnya, semua jenis berbagi mobil, dll. Di Rusia, seperti yang saya pahami, mereka masih berusaha mendukung cara lama, tetapi di Eropa mereka tidak lagi repot.

Tiga hal utama yang saya lewatkan adalah: kamera, navigator, dan internet (setidaknya sebagai titik akses). Tentu saja, Anda bisa hidup tanpa semua ini, dan saya hampir tidak merasa cacat. Dalam kehidupan sehari-hari, hampir selalu ada orang dengan smartphone di dekatnya, dan ini menyelamatkan saya dalam banyak kasus - saya menggunakan ponsel orang lain dalam situasi darurat.

Jika Anda ingin mencoba, cobalah, tentu saja, tetapi saya percaya bahwa tidak perlu membatasi diri secara artifisial. Lebih baik mempelajari cara menyaring atau memberi informasi dan aktivitas yang tidak berguna.

Saya memutuskan untuk menulis catatan ini karena saya akan menghentikan tantangan, dan segera saya akan menjadi orang modern dengan smartphone, instagram, dan kebutuhan pengisian yang konstan.
