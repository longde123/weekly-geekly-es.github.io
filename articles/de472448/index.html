4 Möglichkeiten, das Schreiben von Tests zu beenden


Referent: Albert Fazullin, FlatStack



Tests, Tests ... Alle reden über Tests: dass sie sie schreiben, dass sie nicht benötigt werden, dass nur bestimmte Arten von Tests benötigt werden. Auf dem Bericht werden Sie 4 Wahrheiten hören. Welches gehört dir?

  1. Sie schreiben keine Tests. Sie können sie schreiben, indem Sie mehr darüber erfahren, wie es funktioniert.
  2. Sie möchten mit dem Schreiben von Tests beginnen. Holen Sie sich einige nützliche Tipps, damit es nicht unendlich weh tut.
  3. Sie schreiben Tests, aber es ist immer noch schlecht. Vielleicht sehen Sie die Fehler, die Sie machen, und finden heraus, wie Sie sie lösen können.
  4. Sie schreiben Tests und sind glücklich. Sie können noch einmal sicherstellen, dass alles richtig gemacht wurde
