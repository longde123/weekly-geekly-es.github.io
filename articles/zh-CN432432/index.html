新的OcéColorWave系列技术可增强打印效果

2018年底标志着OcéColorWave 3000系列的两个新型号的发布，它们取代了OcéColorWave 500和ColorWave700。除了前代产品的改进特性之外，CW3500和CW3700还具有许多新功能。

直到2016年，Canon /Océ一直在CAD / GIS的低中量彩色和黑白打印领域保持领先地位。 我们设备的份额超过70％，2014年甚至达到83％。 但是，自2016年以来，情况开始发生变化，为了保持领先地位，该公司发布了OcéColorWave 3000系列的新型号系列。


CW3500和CW3700使用OcéCrystalPoin t技术，该技术结合了激光和喷墨打印的功能和优势。 激光打印消除了油墨的散布，使图像即时干燥并具有耐水性，但它是臭氧的来源，高能耗，墨粉粉尘，由于定影系统中的加热而导致的材料变形。 碳粉的意外溢出不仅会损坏打印件，还会损害用户的健康。 喷墨打印具有灵活的设置和精确的墨滴定位。 但是，根据使用的墨水和介质，图像可能会模糊或失去颜色和对比度。 另外，当使用溶剂墨水和水基墨水时，打印需要额外干燥，以去除蒸发的水分。

OcéCrystalPoint技术基于使用的OcéTonerPearls碳粉（圆形颗粒）。 实际上，在正常条件下，油墨是固态的。 它们不产生灰尘，并且无毒。 药丸装在透明的药盒中。 因此，用户始终可以直观地确定墨粉的剩余量。 即使在打印过程中也要更换墨盒。 为此，您不需要中断该过程，因为它会发生，例如，在激光打印期间更换墨盒时。

OcéTonerPearls墨粉盒安装在打印机顶部。 在重力的影响下，颗粒滚落到打印头中。 位于打印头中的加热元件将墨粉加热到液态。 打印时，胶滴从喷嘴中喷出，并立即在载体表面上结晶。 因此，打印件可以立即使用，并且可以应用于各种介质。 对于打印，可以使用高质量的纸张和再生纸以及各种胶片，而不会降低图像质量，从而大大降低了处理成本。

得益于OcéTonerPearls墨水在介质方面的“杂项”设备以及Cw3500和CW3700的高操作特性，CW3500和CW3700的范围可以远远超过彩色打印和用于CAD / GIS的黑白打印。 在OcéColorWave 3500上，除了图纸和地图，您还可以打印用于建筑街道装饰的背景光和产品，而OcéColorWave 3700则可以处理艺术品，地板图形，横幅和软标牌。

CW 3700的广泛功能是通过使用MediaSense技术来解释的。 在喷墨打印中，图像质量很大程度上取决于从打印头到介质的距离。 如果该距离大于最佳距离，则将难以确保液滴的精确定位，如果距离较小，则液滴将没有时间形成并且载体堵塞的可能性将大大增加。 因此，绝大多数打印设备的介质范围有限。 通常，计算将采用最常见介质的平均厚度来完成设备的设计任务。 即使不考虑载体本身的特性，载体在一个方向或另一个方向上的较小的允许偏差也会大大缩小载体的选择范围。 因此，ColorWave 3500型号可在厚度不超过0.4毫米的介质上进行打印。 ColorWave 3700型号使用MediaSense技术，该技术包括自动三阶段设置：

  • 送入介质时，打印机会测量其厚度
  • 根据内置算法确定介质和打印头之间的最佳距离
  • 根据其特性设置最佳介质进给速度。

因此，CW 3700可以使用厚度不超过0.8毫米的介质。 新的ColorWave 3000系列还使用了OcéPAINT（压电压电集成喷嘴技术）。 它由压电声学控制和喷嘴清洁组成。 打印机会不断监视来自打印头压电元件的信号，以确保它们符合正常操作。 信号变化表明相应喷嘴的操作不正确。 在这种情况下，将使用原始算法补偿空转喷嘴的间隙。 清洁后，所有喷嘴中的墨水供应将返回正常模式。 无需操作员干预即可自动维修打印头。

ColorWave 3000系列使用OcéPOWERsync控制器。 OcéPOWERsync（多内存架构）控制器的架构旨在通过同时执行各种任务来最大化性能。 有足够的计算资源可通过RIP（光栅图像处理器）的同时操作来执行扫描。 当几个打印作业到达时，将组织它们的多线程处理。 例如，接受第一个任务后，第二个开始接收，而第一个发送到缓冲区并排队。 然后，接受第三个任务，将第二个任务发送到假脱机，并且第一个任务由光栅处理器（RIP）处理并传输到内部打印文件。 即，所有工作过程均以串并联模式进行。 这比仅按顺序执行所有任务要快得多。

OcéPOWERsync还具有基于两个硬盘驱动器的多线程内存架构（双内存架构）。 通常，除了RAM外，交换文件还分配在硬盘上，该文件充当虚拟RAM。 但是，与虚拟RAM的数据交换速度远低于与物理RAM的数据交换速度。 因此，许多过程可能会严重减慢速度。 ColorWave 3000系列型号使用大容量物理RAM（8 GB）和两个硬盘驱动器。 因此，打印机仅与RAM交换数据，而RAM与发生单独处理的硬盘交换数据。 例如，在一个硬盘驱动器上输入/输出后台处理数据，在另一个硬盘驱动器上进行扫描和打印。 同时，即使在峰值负载期间，控制器的性能也不会降低。



与以前的ColorWave 500/700系列（其中OcéPOWERsync控制器基于Windows 8 Embedded一起工作）不同，在新的ColorWave 3000型号中，该控制器基于Microsoft Windows 10 IoT Enterprise。 这保证了从Microsoft接收更新的时间可能比Windows 8更长（Microsoft计划至少在2026年之前支持Windows 10）。

Adobe PDF打印引擎（APPE）已集成到POWERsync控制器中，而不是新模型中的旧可配置PostScript解释器（CPSI）PDF解释器。 该解决方案在文件中具有透明效果时效果很好，具有很高的处理速度，在处理大型PDF文件时特别方便，并且与Acrobat Reader完全兼容。

对于ColorWave 3000系列的设备，提供了带有集成扫描仪的整套设备。 专门针对这些版本的是获得专利的OcéImage Logic技术。 它的设计目的是即使从低质量的原件中也可以产生高质量的副本。 那些经常遇到复印的人知道有时候获得高质量的复印件非常困难。 除扫描外，新的OcéImage Logic技术还执行后续的图像处理。 该过程分为四个阶段：

  • 扫描原件。 扫描原始文档时，每条图像行都将转换为从0到255的阴影比例。创建的图像包含256个灰色阴影。
  • 后台自定义。 考虑到检测到的背景水平，可以通过增加像素值来消除文档的背景。 然后，为一页的所有背景水平设置相同的灰度，最后可以使您获得没有背景的图像。
  • 图像增强 。 通过特殊设计的过滤，通过数字处理可以改善不完美的细节。 例如，在过滤过程中，细铅笔线被放大，阴影区域被平滑。
  • 半色调处理。 由于错误扩散技术，灰度图像被转换为​​黑白图像。 对文档的每个像素进行处理，从而可以清楚地识别和填充弱线，纯色和文本。 然后最终处理图像-并准备进一步使用。



连同我们的设备，我们提供免费软件包。
它包括：

  • OcéPublisher Select 3
  • 奥西WPD2

OcéPublisher Select 3

该应用程序可以同时控制两台打印机的操作，并支持所有新的CAD打印语言，包括HP / GL，HP / GL2，TIFF，JPEG，CALCOMP，CALS，C4和NIRS。 另一个方便的功能是无需关闭应用程序即可在打印机之间切换和断开连接的能力。 OcéPublisher Select 3还具有以下优点：

  • 双打印机功能提高了工作流程的生产率：适用于支持OcéPlotWave，OcéColorWave和Canon imagePROGRAF的大幅面设备的任何组合；
  • 通过自动检测连接的打印机，在打印机之间快速切换，监视碳粉/墨水的当前状态以及改进的“拖曳”作业功能，节省了操作员的时间和精力；
  • 通过为打印作业或包含多个文件的打印作业中的单个文件分配专用的媒体模板，或同时分配多个“热文件夹”，来实现工作流程的标准化；
  • 通过自动预处理，在所见即所得模式（“所见即所得”）中进行即时预览以及通过按“绿色按钮”进行打印，可以防止由于第一次正确打印而造成的损坏。

奥西WPD2

该驱动程序使用户可以从Windows应用程序有效地打印到Océ宽幅打印机。 OcéWPD2具有以下功能：

  • 自动使用默认设置和自定义设置模板工作；
  • 有关打印机，介质和墨水状态的恒定信息；
  • 使用OcéConnect Assistant轻松安装并自动发现打印机
  • 使用模板和共享的用户大小轻松部署配置
  • 最佳的数据处理，可提高应用程序的可用性，从而加快您的工作恢复速度
  • 直观，现代化的用户界面，其中所有功能都可以在主屏幕上使用。

OcéPublisher手机

这个有用的应用程序使您可以在Océ绘图仪上打印文档，并通过带有iOS或Android的平板电脑或智能手机来管理它们。 实际上，这是用于移动打印的严肃且方便的产品。 使用OcéPublisher Mobile，您可以：

  • 访问基本设置，
  • 查看用户日志
  • 检查介质和墨水状态
  • 执行缩放
  • 预览PDF，JPG和TIFF格式的打印文档（要预览DWF文件，必须使用特殊的应用程序）。

该应用程序可与所有OcéTDS，TCS，PlotWave和ColorWave打印机一起使用。
ColorWave 3500和ColorWave 3700提供了更多有用的选项，可提高可用性并降低打印成本。 而且，与各种媒体一起使用的能力使服务提供商可以大大扩展范围。
