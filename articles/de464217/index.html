So visualisieren Sie Sound im Web: Eine Auswahl thematischer Materialien und Videovorträge mit Theorie und Praxis

Das letzte Mal haben wir für Sie einen Auszug mit offenen Bibliotheken zur Visualisierung von Audioinhalten erstellt und beschlossen, Materialien zu finden, wie Sie mit all dem umgehen können. Das Ergebnis ist eine Auswahl von Anleitungen zum Thema für Entwickler von Webanwendungen oder Spielen.


Foto s_keller / Pixabay
In unserem Blog über Habré -> Lesen für das Wochenende: 65 Materialien über Streaming, die Geschichte des alten "musikalischen Eisens", Audiotechnologien und die Geschichte der Sprecher

Theorie und Geschichte


  • Theorie: Methoden zur Hervorhebung des Grundtons . Ein Mitglied des Ableton- Entwicklungsteams, eines Studio-basierten Soundprogramms, analysiert mehrere gängige Algorithmen zum Extrahieren des Grundtons (Wiederholung des Schallwellenzyklus) in Musikaufnahmen. Beispiele sind: die Nulldurchgangsmethode und die Methode, die auf der Identifizierung der Grundharmonischen basiert. Der Autor zitiert auch mehrere zusätzliche Lesematerialien, in denen fortgeschrittene Ansätze zur Hervorhebung des Grundtons erörtert werden. Der Artikel kann für diejenigen von Interesse sein, die eine Musikanwendung schreiben möchten, die Diagramme von Schallwellen anzeigt.

  • Eine kurze Geschichte der Synthesizer . Dies ist ein Video einer der Reden auf der Scotland JS-Konferenz. Chris Lowis, Herausgeber von Web Audio Weekly, spricht darüber, wie sich Synthesizer entwickelt haben und wie sie funktionieren. All dies mit Beispielen zu JS.


  • Web Audio API-Grundlagen Mozilla-Entwickler sprechen über Web-Audio-API- Funktionen und Spezifikationen für die Verwaltung von Audioinhalten in Browsern. Es werden sowohl grundlegende Dinge wie die Interaktion mit Audiopuffern als auch fortgeschrittenere Dinge (z. B. die Visualisierung von Ton) beschrieben. Viele Grafiken, Diagramme und Codebeispiele. Das Material wird regelmäßig aktualisiert - es kann als „Desktop“ -Referenz bei der Entwicklung von Anwendungen verwendet werden.

  • 8-Bit-Musiktheorie . Ein Kanal auf YouTube, der sich der musiktheoretischen Analyse des Audiobereichs von Videospielen widmet. In diesem Video zeigt der Autor beispielsweise anhand von Beispielen, wie Musik dazu beiträgt, eine Atmosphäre im Hollow Knight-Spiel zu schaffen, und hier sprechen wir über eine der endgültigen Kompositionen in Dark Souls. Der Kanal kann für diejenigen nützlich sein, die ihr eigenes Spiel schreiben. Hier können Sie sich einige Ideen für ihre musikalische Begleitung einfallen lassen.

Übe


  • Entdecken Sie die Möglichkeiten der Web Audio API mit der Bibliothek D3.js. D3.js ist eine Reihe von Tools zur Visualisierung von Daten. Es enthält Module zum Erstellen geometrischer Formen, zum Verwalten des Ladens, Formatierens und Skalierens von Daten sowie mathematische Funktionen. Die Autoren des Materials erklären, wie man damit Schallwellengraphen erstellt. Ein Beispiel für die Anwendung aus dem Artikel finden Sie auf GitHub Pages . Der Code befindet sich im offiziellen Repository .

  • So erstellen Sie Musiksysteme in JavaScript . Global Web Audio Guide. Damit stellt der Autor die Werke der Komponisten Stephen Reich und Brian Eno nach - " It's Gonna Rain " bzw. " Discreet Music ". Das Befolgen des Handbuchs ist recht einfach (es gibt Beispiele, Diagramme und Screenshots), auch wenn Sie mit der Web-Audio-API nicht vertraut sind.

  • Musikinstrument mit Web Audio API . Ein Datensatz mit JSConf, in dem Steve Kinney, Gründer der Turing School, zeigt, wie ein Synthesizer in einem Browser auf der getUserMedia Web API und WebSockets erstellt wird. Er spricht auch über die Verwendung der Web-Audio-API, um ein Musikinstrument auf Arduino zu erstellen.

  • Einführung in p5.js. p5.js ist eine Visualisierungsbibliothek, mit der Sie "mit Code zeichnen" können. In dem Video zeigt die Entwicklerin Lauren McCarthy, wie mit diesem Tool künstlerische Elemente und Animationen erstellt werden. Wenn Sie die Funktionen von p5.js selbst bewerten möchten, gibt es auf der offiziellen Website des Projekts einen speziellen Editor .


  • Musik von 8-Bit-Spielen auf der Web Audio API . In diesem Artikel wird beschrieben, wie Sie 8-Bit-Musik mithilfe der Web-Audio-API und des Tone.js- Frameworks von Pac-Man, Super Mario Bros, Metroid, Kirby's Adventure und The Legend of Zelda neu erstellen und visualisieren. Hören Sie sich die resultierende Komposition an, sehen Sie sich den Code an und können Sie ihn bei Bedarf in CodePen ändern . Um das Verständnis der Quelle zu erleichtern, empfiehlt der Autor, dass Sie dieses Video zuerst auf YouTube ansehen - es erklärt, wie der Sound in alten Computern "funktioniert".

  • So erstellen Sie einen Tanztrack mit der Web-Audio-API . Aufnahmeleistung von Paul Adenot, einem Ingenieur bei Mozilla, der am Firefox-Browser arbeitet und W3C dabei hilft, die Web-Audio-API-Spezifikation zu vervollständigen. Paul zerlegt die Hauptkomponenten des Techno-Tracks und zeigt, wie man ihn auf JS neu erstellt. Wir können sagen, dass dies eine Live- Musik-Programmiersitzung ist .
