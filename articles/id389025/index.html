Compact port USB expander untuk pengisian daya



Di waktu senggang, saya ingin mengembangkan beberapa perangkat yang berguna untuk rumah, misalnya: sensor suhu atau kelembaban dengan layar, gateway WiFi-IR, dll. Semua perangkat ini membutuhkan daya, saya menggunakan pengisian daya USB dari ponsel. Karena setiap perangkat mengkonsumsi arus kecil, dan pengisi daya dari iPad memberikan 2.4A, maka saya memutuskan untuk menghemat outlet daya dan menghubungkan semua perangkat saya ke satu pengisian daya, untuk ini saya membuat expander port USB ringkas untuk pengisian daya, yang hampir tidak menambah ukuran muatan itu sendiri.

Untuk membuat expander port USB, kita perlu:
  1. Dewan Pengembangan Solder Cina
  2. Dua konektor tipe USB laki-laki (Saya memiliki konektor ganda)
  3. Kabel USB tidak perlu dari yang Anda perlu menghapus tipe konektor USB laki-laki
  4. Posting hitam dan merah
  5. Pita listrik biru putih
  6. Pita busa dua sisi
  7. Pengisi daya
  8. Peralatan solder
  9. 20 menit waktu luang



Hati-hati solder desain, sambil berusaha untuk tidak membingungkan GND dan VCC, agar tidak membakar pengisian daya dan perangkat yang terhubung.



Setelah menyolder, Anda harus mendapatkan pelindung yang pas dengan pengisi daya.




Menguji dengan perangkat USB.



Jika semuanya berfungsi seperti milik saya, agar tahan lama, kami merekatkan expander untuk mengisi daya menggunakan selotip dua sisi pada dasar busa dan membuat casing putih bergaya Apple yang indah dari pita listrik putih.
