始终处于领先地位：俄罗斯AdvoCam DVR的综合回顾

大家好！本文将重点介绍四个AdvoCam品牌的车载DVR ：FD One，FD8黑色GPS，FD8 Red II（GPS + GLONASS）和FD8 Gold GPS。这些设备具有几个关键功能，使其与俄罗斯出售的大量“黑匣子”区分开来。首先，这是高质量的组件，组件，因此是高质量的视频记录。还值得注意的是方便周到的管理和广泛的功能。最后，它贿赂了来源：AdvoCam并不像大多数品牌那样从中国制造商那里购买现成的注册商，而是设计自己的设备。拥有20年专业视频监控系统开发经验的俄罗斯工程师对此负责。




AdvoCam 拥有自己的生产设施（弗拉基米尔州的Aleksandrov市），在这里进行已发布自动调节器的开发，测试组装和生产前测试。例如，在这里，他们检查了小工具将如何承受俄罗斯严酷的操作条件。设备在振动台（模仿我们的理想道路）和温度室内进行了测试。 AdvoCam记录仪能够在-35至+50摄氏度的温度下运行。



任何中国设备都不可能承受这样的温度，即使从我们坑坑洼洼的震动中，他的生命（或者至少是他依恋的生命）也可能会短暂。值得注意的是，AdvoCam不会发布非常便宜的注册商，该品牌最实惠的型号估计约为5000卢布。可以理解，您不能制造仅花费数千美元的高质量设备-它很可能是“展示”注册商。也许他会拍摄视频，但是很糟糕。也许它将不断地“挂起”并关闭，或者崩溃。然后一次全部。谁需要这样的“工艺品”？既不是买主也不是普通品牌-声誉更昂贵。



说到声誉。自2014年以来，AdvoCam一直被列入俄罗斯最受欢迎的DVR品牌的TOP-3。这意味着购买者对卢布的高质量和功能性设备投了反对票，厌倦了价格低廉却不可靠的中国“渣”。

好吧，让我们继续考虑特定的模型。让我们从最实惠的价格开始-AdvoCam-FD One，价格为4,900卢布。由于几乎完全没有其他选择，因此达到了如此低的价格（按高质量记录器的标准）：在该型号中，没有GPS或测速仪。甚至没有LDWS。



只有一个G传感器，该传感器会在跳闸（电击，急刹车，倒车等）时自动保护视频，使其不会被覆盖。低廉的价格并未影响视频的质量，因此AdvoCam-FD One可以轻松地从该价格范围内绕过其他品牌的注册商。此外，就记录质量而言，该型号堪比成本几乎翻倍的设备-7-8千卢布。
AdvoCam-FD One使用快速CMOS矩阵，具有3x3微米的大物理像素大小，矩阵本身的大小为1 / 2.7英寸，分辨率为2兆像素。下图清楚地说明了为什么2百万像素CMOS矩阵优于4百万像素或5百万像素的矩阵：



矩阵的物理尺寸越大，表示每个像素接收的光越多，也就是说，在弱光条件下，图片保持清晰明快。

AdvoCam-FD One以1920 x 1080像素的分辨率录制视频，频率为30 fps。记录仪镜头包括5个由专业光学镀膜玻璃制成的透镜。可视角度为对角线150度。即，足够的信息（三个车道和一个路边）进入帧，但是图像不会在侧面弯曲。录制质量很高，尤其是当您回想起该型号的成本时。下午，车牌到15米外的地方都可以读取，非常好。在晚上，房间的直线可读性为8-10米。当然，这并不是最大的性能，但是对于5000卢布的分段来说，这是一个相当不错的水平-大多数具有类似价格标签的竞争对手在白天最多提供12米，在晚上最多提供5米。









AdvoCam-FD One配备了550 mAh电池，充满电可持续拍摄约40分钟，而无需使用外部电源。鉴于一般的autoreg无需充电大约5-10分钟即可工作，因此这是一个非常可观的指标。您甚至可以说一个记录值。不仅在价格为5到7千卢布的注册服务商中，而且在价格更昂贵的注册机中，很少有制造商为其产品配备电池超过15至20分钟的“自主权”。



AdvoCam-FD One的另一个功能是可以用作个人计算机的网络摄像头。所需要做的就是将记录器连接到PC。一个有用的功能，尤其是当您考虑到一个普通的2百万像素“网络摄像头”将花费您大约4000卢布，而在这里，只要4,900卢布，您就可以立即获得注册服务商和网络摄像头。

总的来说，AdvoCam-FD One 在2016年底应获得样本中最周到的预算注册商的荣誉称号。其中，除其他外，它能够以7-8千卢布的成本记录模型级别的视频。

我们正在考虑的第二个模型是AdvoCam-FD8黑色GPS，估计价格为6,500卢布。该单元也属于预算类别，但是，在拍摄质量上，它并不逊色于更昂贵的型号。 AdvoCam-FD8黑色GPS使用3兆像素Aptina CMOS传感器（1/3英寸），专门为专业视频监控系统设计。这里的镜头还包括5个由专业光学透明玻璃制成的镜头。该设备的最终成本由于使用了更便宜（但仍相当高效）的处理器而略有降低-该型号使用了Novatek，而更昂贵的AdvoCam DVR型号则使用了Ambarella解决方案。



输出的是高质量视频，最大分辨率为1920 x 1080像素，频率为30 fps。图片明亮，清晰，细腻。这也适用于夜间拍摄，您不会期望如此价格的设备。

在有趣的附加功能中，可以区分GPS接收器的存在，该接收器用于确定当前坐标和车速。这在交通事故后的诉讼中可能很有用。

AdvoCam-FD8黑色GPS不能节省建筑质量，这令人高兴。但是，包装中不包含存储卡（存储卡与更昂贵的AdvoCam录像机一起放在包装盒中）。因此，您必须自己购买此配件。好吧，这一点都不可怕，因为今天的卡片比香肠棒还贵很多。因此，将AdvoCam-FD8黑色GPS保存在琐事上，而不会影响设备的基本参数，例如记录质量，构建质量，功能等。









有问题的第三个注册商是AdvoCam-FD8 Red II（GPS + GLONASS）。第二代红色律师。该模型的价格估计为8,600卢布，也就是说，您不能再称其为预算了-这已经是普通的中产阶级了！



AdvoCam-FD8 Red II（GPS + GLONASS）与之前的两个型号之间的主要区别在于以当前最高分辨率的Super Full HD（即2304 x 1296像素）进行视频录制。该设备使用美国公司OmniVision的4兆像素矩阵，可让您以Super Full HD或Full HD（1920 x 1080像素）但以HDR模式录制视频。后一种模式使帧中的亮度均匀，不包括过亮或过暗的区域。此外，在HDR模式下，当记录上仅可见没有数字和字母的白色矩形时，汽车上数字的“耀斑”会明显减少。同样值得一提的是，专业开光玻璃的六镜头镜片和最强大的最新一代处理器-Ambarella A7。总的来说，这里是最相关且最现代的填充。

在分辨率方面，Super Full HD AdvoCam-FD8 Red II（GPS + GLONASS）表现出色。该视频显示了所有的小细节，并且全班都缺少数字“混乱”。对于DVR，汽车牌照几乎可以读取到最大距离-白天17-20米，夜间15米。这个水平大大高于市场平均水平。普通的中国模特白天“下山”-大约15米，晚上-不超过10-12米。









AdvoCam-FD8 Red II（GPS + GLONASS）支持最大128 GB的存储卡。对于DVR来说很少见-通常不超过32 GB。鉴于此型号的AdvoCam以Super Full HD录制视频，因此此选项非常有用-在此分辨率下，一个128 GB的USB闪存驱动器可容纳约16个小时的连续视频。值得注意的是，AdvoCam-FD8 Red II（GPS + GLONASS）拥有自己的内置内存256 MB。如果发生事故，可以将长达两分钟的“紧急”视频片段复制到其中。

需要说明的是，AdvoCam-FD8 Red II（GPS + GLONASS）取代了AdvoCam-FD8 Profi-GPS Red模型。它是2015年最受欢迎的DVR之一。新颖性得到了更现代的矩阵（上面已经提到过），以及两个系统的卫星导航模块-GPS + GLONASS（其前身仅与GPS一起使用）。对两个卫星系统的支持在城市密集发展的地方或例如在开车隧道时提供了更快的“冷启动”，更精确的坐标确定和更可靠的信号接收。

使用AdvoCam-FD8 Red II（GPS + GLONASS）中的内置GPS + GLONASS模块，您可以记录汽车的运动轨迹并在视频上贴上速度标记（还可以设置显示的最大阈值，以免自行解决违规问题）。此外，设备的内存还包含测速相机的坐标。当接近后者时，注册服务商会提前警告您。该数据库会定期更新，也就是说，它包含有关高速摄像机的最新信息。



AdvoCam-FD8 Red II（GPS + GLONASS）的另一个有趣功能是称为LDWS的车道控制系统。此选项旨在减少因打zing睡或分散注意力而导致的事故数量。因此，当转移到相邻车道时，设备将发出响亮的声音信号，根本无法听到（除非您戴着耳机驾驶）。但是，应该澄清的是，为了使LDWS正常运行，需要良好的道路标记和充足的照明。



第四个（也是今天最后一个）模型是AdvoCam-FD8 Gold GPS。金律师！这是AdvoCam系列的旗舰注册商，估计价格为12,500卢布。



该设备的一个特点是当今所有注册服务商都可以以30 fps的频率以超全高清（2304 x 1296像素）的最大分辨率进行视频录制。或全高清（1920 x 1080像素），但频率为60 fps。在后一种情况下，高速移动时视频的清晰度和清晰度明显提高。您也可以在HDR模式下录制。

AdvoCam-FD8 Gold GPS使用美国公司Ambarella的旗舰处理器模型-Ambarella A7LA70。记录仪矩阵的分辨率为4兆像素，这是American OmniVision OV4689。镜头由六个由优质透明玻璃制成的镜头组成，对角线视角为170度。

AdvoCam-FD8 Gold GPS的记录质量处于预期的高水平。即使在弱光条件下，图像看起来也很明亮，没有暗区。房间的可读距离是白天20米，晚上高达15米。这些几乎是现代注册服务商市场的最大价值。只有价格在14-15千卢布的韩国型号才能提供更多。但是对于这样的价格标签，“韩国人”实际上不提供任何其他功能。不同于“金律师”的12.5万卢布。









与以前的模型一样，在AdvoCam-FD8 Gold GPS中，实现了LDWS功能-车道控制系统。



还有一个GPS接收器，借助它，您可以在视频上贴上速度标记，记录运动轨迹等。记录仪的存储器不仅包含一个带有高速摄像机的数据库（例如AdvoCam-FD8 Red II（GPS + GLONASS），而且还包括一个交通警察哨所，路况不佳的路段，减速带等）。关于所有这些对象登记员警告驾驶员。



令人高兴的是，AdvoCam-FD8 Gold GPS随附了容量为8 GB的存储卡。令人惊讶的是，很少有autoreg随存储卡一起提供，尤其是这种容量的存储卡。值得一提的是，该设备拥有自己的256 MB内置内存。任何时候，您都可以在发生事故时将视频片段复制到视频片段上，然后再给卡加入协议。
