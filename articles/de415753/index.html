Mutiger Browser integriert sich in Tor



Während Firefox nur die Integration in Tor plant , hat der Brave-Browser dies bereits auf der Ebene der privaten Registerkarten durchgeführt. Die Funktion ist in der neuesten Version von Brave 0.23 implementiert.

Die Integration mit Tor hat den Beta-Status. Wie in der Beschreibung angegeben , können Sie den Datenschutz nicht nur auf der Ebene eines einzelnen Geräts, sondern auch auf Netzwerkebene gewährleisten. Private Registerkarten von Tor schützen den Benutzer vor dem Abhören durch seinen Internetprovider, einen externen Provider an Gast-WLAN-Zugangspunkten, sowie vor den besuchten Websites selbst, die Tracking-Cookies setzen, Analyseskripte ausführen und die IP-Adressen der Besucher registrieren können.

Private Registerkarten können einfach über das Menü Datei gestartet werden, in dem sich das Element Neue private Registerkarte befindet. Ein Benutzer kann jederzeit eine oder mehrere Registerkarten verschiedener Typen starten lassen: eine reguläre Registerkarte, eine private Registerkarte ohne Tor und eine private Registerkarte mit Tor. Sie arbeiten alle gleichzeitig.


Auf diese Weise erhalten Benutzer jetzt den Komfort des Brave-Browsers zusammen mit den Standard-Sicherheitsfunktionen von Tor.

Der Brave-Browser verfügt bereits über erweiterte Datenschutzfunktionen. Standardmäßig werden Anzeigen blockiert, Tracker, Crypto Miner und andere Bedrohungen im Internet verfolgt. Standardmäßige private Registerkarten speichern weder Browserverlauf noch Cookies. Wenn Sie jedoch den Tor-Modus aktivieren, wird der Schutz weiterhin verbessert. Wie oben erwähnt, macht es diese Methode für jemanden auf der Route zwischen Ihrem Browser und der endgültigen Site schwierig, den Kommunikationskanal zu infiltrieren oder herauszufinden, welche Site angefordert wird. Und die Seite selbst sieht auch nicht, von welcher Adresse der Besucher kam.

Die Standardsuchmaschine in den privaten Registerkarten von Tor ist DuckDuckGo , aber Sie können problemlos zu einer der 19 anderen Suchmaschinen wechseln. DuckDuckGo empfiehlt die anonyme Nutzung der Suchmaschine und sammelt keine Besucherinformationen.

Neben dem regulären Tor-Client trägt der Brave-Browser auch zur Funktionalität des Tor-Netzwerks bei und unterstützt den Betrieb der Relais (Repeater) von Tor. Alle Brave Repeater für Tor sind auf dieser Seite aufgelistet.



Die Integration des Browsers in Tor ist sehr lobenswert, es sollte jedoch beachtet werden, dass er sich noch im Beta-Status befindet. Entwickler warnen vor einigen nicht geschlossenen Fehlern und bekannten Lecks , die sie in zukünftigen Versionen beheben möchten. Die Hilfe aller Entwickler bei der Behebung dieser Probleme wird geschätzt: Der Code ist auf GitHub verfügbar. Die Veröffentlichung der nächsten Version mit einigen Fehlerkorrekturen ist in den nächsten Wochen geplant.

Die Entwickler planen außerdem, die Funktion zur Auswahl einer Geolokalisierung für den Tor-Ausgabeknoten einzuführen, damit Sie sich als Benutzer eines bestimmten Landes ausgeben können.

Wenn Sie jedoch einen garantierten Datenschutz benötigen, ist es besser, Tor Browser mit nachgewiesenem Schutz zu installieren.

Parallel zu Brave führen Tor-Entwickler ein FUSION- Projekt durch, um die Funktionen des Tor-Browsers in Firefox zu integrieren. Es gibt noch keinen klaren Plan, welche spezifischen Tor Browser-Funktionen in Firefox implementiert werden. Wie Sie wissen, basiert Tor Browser auf Firefox ESR mit einer Reihe von Patches, die für Tor spezifisch sind. Für Entwickler eines anonymen Browsers ist es nicht sehr praktisch, diese Patches von einem Repository in ein anderes zu verschieben. Daher wurde vor einigen Jahren zusammen mit Mozilla das Uplift-Projekt organisiert, das die automatische Aufnahme von Tor Browser-Patches in die Firefox-Codebasis beinhaltete. In den letzten anderthalb Jahren wurden gleichzeitig neue Informationssicherheitsfunktionen in Tor Browser und Firefox implementiert. Hierbei handelt es sich um eine Elementisolierung (First-Party-Isolierung, Festlegen von privacy.firstparty.isolate in Firefox 52+, sie ist auch Teil des Schlüsselsystems Cross-Origin Identifier Unlinkable , das Anonymität in Tor Browser bietet), einem System zur Bekämpfung von Fingerabdrücken , einschließlich Fingerabdrücken mit installierten Schriftarten , Anti- Fingerprinting auf HTML5- Zeichenfläche, neue Einstellung privacy.resistFingerprinting in Firefox 59+ usw.

Letztendlich wäre eine vollständige Zusammenführung der Codebasen Firefox und Tor Browser ideal. Aber es fängt klein an. Wie bei Brave wird Tor im ersten Schritt nur in die privaten Registerkarten von Firefox integriert.
