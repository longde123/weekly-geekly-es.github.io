Menyebarkan Office 2019 di lingkungan perusahaan (untuk profesional TI)

Gunakan Alat Penerapan Alat Penyebaran Office (ODT) untuk mengonfigurasi dan menggunakan versi perusahaan Office 2019 di organisasi Anda, termasuk Project dan Visio (dan program lainnya).

File instalasi untuk Office 2019 tersedia langsung dari Office CDN, mis. Paket Windows (MSI) tidak lagi digunakan.

Tetapi pada kenyataannya, ini berarti bahwa pada satu komputer Anda akan memiliki satu instalasi 2019 Office, dan itu tidak akan mungkin untuk menginstal paket Office lainnya.

Misalnya : Anda membeli Office 2019 Standard, Anda menginstalnya di komputer pengguna dan berencana untuk menginstal Skype for Business Basic menggunakan penginstal MSI - ini tidak akan berfungsi. Dan, bahkan jika Anda mengunduh paket Klik untuk Menjalankan Skype for Business dari Office 365, paket itu tidak akan diinstal juga. Maaf, saya ulangi: hanya ada satu instalasi Office 2019 di komputer, terlepas dari bagaimana produk Office didistribusikan.

Jadi, Anda perlu membuat satu paket untuk menggunakan Office 2019.


Unduh Alat Penerapan Office dari Pusat Unduhan Microsoft . Setelah menerima salinan ODT, Anda perlu membuat file konfigurasi.xml konfigurasi.

Anda dapat membuat banyak file configuration.xml yang akan digunakan dengan ODT.

Mengapa diperlukan: Anda dapat menginstal Office 2019 langsung dari jaringan Office CDN, atau Anda dapat mengunduh file instalasi dari jaringan Office CDN ke folder di jaringan lokal, misalnya, ke folder bersama dan menginstal Office 2019 dari lokasi ini. Anda dapat menggunakan metode berbeda untuk set komputer berbeda di organisasi Anda.

Saya sarankan menggunakan Alat Kustomisasi Kantor untuk membuat file konfigurasi. Menggunakan situs ini, Anda bisa membuat file jawaban untuk Office 365 / Office 2019 dengan pengaturan dasar seperti:

  • Versi arsitektur kantor
  • produk, komponen, dan bahasa;
  • opsi instalasi (Office CDN atau Sumber Lokal);
  • jenis lisensi.

Instal Office 2019 dengan menggunakan Office Deployment Tool


Setelah Anda menerima ODT dan membuat file konfigurasi.xml, Anda bisa menginstal Office 2019.

Buka prompt perintah dengan hak istimewa yang ditinggikan, navigasikan ke folder tempat Anda menyimpan ODT dan file configuration.xml, dan masukkan perintah berikut untuk mengunduh paket instalasi ke folder jaringan:

  • setup / unduh configuration.xml

lalu masukkan perintah:

  • setup / konfigurasikan configuration.xml

untuk menginstal Office dari folder jaringan atau langsung dari Office CDN. Tunggu instalasi Office selesai.

Dari pengalaman, saya akan mengatakan bahwa lebih baik untuk menguji instalasi paket dari jaringan Office CDN, dan kemudian membuat paket lokal.

Untuk jaga-jaga, berikut adalah daftar lengkap perintah ODT:

SETUP /download [path to configuration file]
SETUP /configure [path to configuration file]
SETUP /packager [path to configuration file] [output path]
SETUP /customize [path to configuration file]
SETUP /help
