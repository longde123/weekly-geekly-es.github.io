PHP摘要136号（2018年7月24日至8月6日）


带有新闻和资料链接的新鲜选择。 在发行版中：PHP 7.3.0 Beta 1，PhpStorm 2018.2，Composer 1.7和其他发行版，标准PSR-17，Yii 3.0的回顾，用于异步应用程序的Fiber API的替代实现，一部分有用的工具，以及更多其他被接受。

祝您阅读愉快！



新闻与发布


  • PHP 7.3.0 Beta 1-随着第一个beta版本的发布，积极的开发阶段结束了，这意味着7.3分支中的新功能和更改列表可以认为是最终的。 下一个测试版预计于8月16日发布。
  • PhpStorm 2018.2
    • 改进的名称空间匹配
    • 结构搜索和替换
    • 自定义postfix加载项模式
    • 更新了界面并支持触摸条
    •以及新的检查和其他改进 。
  • PSR-17：HTTP工厂 -已正式采用一种标准来管理工厂接口，以创建与PSR-7兼容的HTTP对象。
  • PHP 7.2可在Google Cloud Platform上使用
  • PHPUnit 7.3-所做的更改中包括首次运行先前运行的测试的功能。
  • 作曲者1.7.0

PHP内部


  • [PHP]：Same Site Cookie - setcookie()和其他使用cookie的功能已添加了传递一系列选项的功能，包括对Same-site Cookie标准的支持。 已在PHP 7.3中实现。
  • [RFC]：Typed Properties 2.0-类型属性的提案被推迟到PHP的下一个主要版本，而Dmitry Stogov发布了基准测试结果以评估此功能的开销。

工具


  • spiral / roadrunner-在Go上实现的用于PHP的高性能应用服务器，负载平衡器和流程管理器。 可用于在AWS Lambda上运行PHP 。
  • codeplea / ahocorasickphp - Aho-Korasik算法的实现，用于搜索字符串中的多个子字符串。 比调用strpos()快几倍，并且比调用preg_match_all()快得多。
  • chekalskiy / php-bank-db-用于通过卡号确定银行的库。 由Ilya Chekalsky发送。
  • igniphp / framework-支持PSR-15，PSR-7以及在Swoole上运行的轻型框架。
  • spatie / phpunit-snapshot-assertions-用于在PhpUnit上实施快照测试的工具。 将测试结果与上一次运行进行比较。
  • elgentos / masquerade-用于匿名化数据库中信息的工具。 开箱即用的支持Magento 2。
  • zendframework / zend-problem-details-为PSR-7应用程序实现对标准“ HTTP API的RFC 7807问题详细信息 ”的支持，该标准可调节错误响应。
  • formapro / pvm-一个强大的库，用于描述工作流，业务流程以及状态机。 可用的UI可视化。
  • Rican7 /传入 -该库旨在将来自任何来源的原始输入转换为强类型的结构。
  • rezozero / mixedfeed-该库可让您将各种社交服务的供稿合并为一个供稿。
  • KikApp-该工具允许您使用PHP编写适用于iOS和Android的本机应用程序。 有没有人有使用经验？
