Azure Cloud Shell di Terminal Windows

Sekarang Anda dapat terhubung ke Azure Cloud Shell melalui Terminal Windows!



Untuk melakukan ini, kami menambahkan profil default baru - Azure Cloud Shell, yang memungkinkan Anda mengakses direktori / klien Azure melalui aplikasi Windows Terminal.

Jika Anda sudah menginstal Terminal Windows


Pengaturan Windows Terminal Anda tidak akan secara otomatis memperbarui ke profil baru (karena file tidak diperbarui setiap kali Anda membuka Windows Terminal), jadi di sini adalah bagaimana Anda dapat menambahkannya secara manual:

  1. Luncurkan Terminal Windows
  2. Buka Pengaturan (menggunakan daftar drop-down)
  3. Tambahkan profil ini ke daftar profil Anda (profil item): 

{"acrylicOpacity" : 0.6, "closeOnExit" : false, "colorScheme" : "Vintage", "commandline" : "Azure", "connectionType" : "{d9fcfdfa-a479-412c-83b7-c5640e61cd62}", "cursorColor" : "#FFFFFF", "cursorShape" : "bar", "fontFace" : "Consolas", "fontSize" : 10, "guid" : "{b453ae62-4e3d-5e58-b989-0a998ec441b8}", "historySize" : 9001, "icon" : "ms-appx:///ProfileIcons/{b453ae62-4e3d-5e58-b989-0a998ec441b8}.png", "name" : "Azure Cloud Shell", "padding" : "0, 0, 0, 0", "snapOnInput" : true, "startingDirectory" : "%USERPROFILE%", "useAcrylic" : true}

Setelah Anda melakukan ini, tab Azure Cloud Shell akan tersedia untuk Anda.

Pengaturan dasar


  1. Klik tab "Azure Cloud Shell".
  2. Anda akan diminta untuk membuka â€œmicrosoft.com/deviceloginâ€ dan memasukkan kode yang ditampilkan.
  3. Setelah memasukkan kode di browser, Anda harus masuk di bawah akun Anda. Pastikan juga Anda masuk dengan akun yang memiliki direktori / klien Azure aktif.
  4. Kembali ke terminal. Setelah beberapa detik, Anda akan melihat pesan "Diotentikasi".
  5. Hanya dalam beberapa kasus : jika akun Anda memiliki beberapa klien, maka Anda akan diminta untuk memilih salah satu dari mereka. Cukup masukkan nomor klien yang ingin Anda hubungkan.
  6. Anda kemudian akan diminta untuk menyimpan pengaturan koneksi ini. Menyimpan parameter koneksi akan memungkinkan Anda untuk memasuki sistem di masa depan tanpa melalui langkah 1-5.
  7. Kemudian aplikasi akan mulai membuat koneksi dengan Cloud Shell (ini mungkin memakan waktu, bersabarlah).
  8. Anda sekarang terhubung ke Azure Cloud Shell!

Ini adalah apa yang tampak seperti keluaran login penuh (saya menekan '0' untuk nomor klien dan 'y' untuk memilih untuk menyimpan parameter koneksi):



Sekarang saya telah menyimpan parameter koneksi, berikut adalah proses login di masa depan (saya tekan '0' untuk mengakses parameter koneksi yang disimpan):



Anda juga dapat masuk menggunakan akun / klien lain dengan menekan 'n', atau menghapus koneksi yang disimpan dengan menekan 'r'. Parameter ini akan disimpan selama sesi, jadi meskipun Anda memulai terminal setelah beberapa hari, Anda masih bisa masuk dengan parameter koneksi yang disimpan tanpa harus membuka browser lagi.

Saya harap Anda menikmati kemampuan untuk mengakses sumber daya Azure Anda melalui Terminal Windows. Juga, seperti biasa, tolong laporkan bug / masalah apa pun yang Anda perhatikan di repositori kami di Github .
