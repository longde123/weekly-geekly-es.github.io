Kurangi downtime Zimbra selama server mogok dengan Zextras Backup

Hanya apa yang sudah rusak dan kebohongan yang tidak pecah atau jatuh. Aturan emas ini diketahui oleh setiap manajer TI, dan itulah sebabnya ketersediaan cadangan dan sistem pemulihan darurat dalam solusi TI modern apa pun telah menjadi standar industri. Ini tidak hanya menghindari hilangnya informasi penting, tetapi juga meminimalkan waktu henti perusahaan karena berbagai kecelakaan dan situasi darurat.



Administrator mana pun dari versi gratis Zimbra Collaboration Suite tahu bahwa ia tidak memiliki mekanisme pencadangan dan pemulihan bawaan, dan sebagian besar solusi pihak ketiga mengharuskan server untuk menjeda untuk pencadangan. Fungsi cadangan real-time disediakan oleh pengembang untuk pengguna Zimbra versi komersial. Namun, ini tidak berarti bahwa Anda harus membeli lisensi mahal hanya untuk dapat membuat dan menggunakan cadangan dengan cepat. Sistem cadangan yang sepenuhnya identik dapat diinstal secara langsung di Zimbra Collaboration Suite Open-Source Edition.

Ini dapat dilakukan dengan menggunakan ekstensi modular Cadangan Zextras, yang merupakan bagian dari rangkaian musim dingin Zextras Suite . Zextras Backup menyediakan alat yang sama bagi administrator untuk membuat cadangan informasi pada server dengan Zimbra sebagai versi komersial ZCS, tetapi biayanya jauh lebih murah. Khususnya, dengan bantuan Zextras Backup, dimungkinkan untuk membuat cadangan informasi yang disimpan di server Zimbra, memastikan keamanannya dalam keadaan darurat dan bahkan mengembalikan Zimbra agar berfungsi dalam beberapa klik bahkan setelah kehancuran fisik server. Namun, hal pertama yang pertama.

Inovasi utama yang terdapat dalam Zextras Backup adalah pemindai khusus yang bekerja secara real time. Prinsip operasinya adalah memindai log server dari Zimbra dan memonitor setiap perubahan yang terjadi pada file Zimbra Collaboration Suite dan langsung mencadangkan file yang diubah berkat mekanisme cadangan tambahan. Dengan demikian, setiap file yang baru dibuat atau dimodifikasi akan disalin ke media cadangan, dan oleh karena itu, bahkan jika server secara fisik dihancurkan sebagai akibat dari kebakaran, Anda masih akan memiliki salinan cadangan bahkan informasi yang dibuat di server beberapa detik sebelum bencana. Juga, mekanisme serupa untuk membuat cadangan informasi dapat digunakan untuk mengembalikan perubahan yang telah terjadi selama periode tertentu.

Pemindai real-time dihidupkan di konsol administrasi hanya dengan satu tombol. Anda juga dapat mengaktifkannya di baris perintah menggunakan perintah setProperty ZxBackup_RealTimeScanner TRUE cadangan zxsuite .



Fitur lain yang disediakan oleh ekstensi Cadangan Zextras adalah Cadangan Eksternal. Dengan menggunakan fungsi ini, Anda dapat membuat cadangan pada media yang terhubung dan menggunakannya saat bermigrasi ke server lain.

Manajer TI yang berpengalaman, bagaimanapun, dapat mencatat bahwa memiliki cadangan yang ada tidak akan menyelamatkan perusahaan dari downtime yang lama, karena untuk menyebarkannya, Anda harus terlebih dahulu membuat komisi server baru, kemudian menginstal sistem operasi di atasnya, serta Kolaborasi ZImbra Suite dan hanya kemudian mengembalikan cadangan dari server lama. Semua ini dalam waktu dapat memakan waktu lebih dari satu hari dan jumlah kerugian dari downtime yang lama dapat berkali-kali melebihi gaji manajer TI yang membuatnya.

Itulah sebabnya semua snapshot data yang dibuat oleh Cadangan Zextras bersifat konsisten, konsisten, dan siap untuk diterapkan segera. Jadi, misalnya, tanpa menghentikan Zimbra, Anda dapat menghapus cadangan darinya dan menyalinnya ke server jauh menggunakan rsync. Ini berarti bahwa data Anda akan dilindungi secara andal bahkan jika terjadi kegagalan penyimpanan cadangan utama, dan proses menempatkan server cadangan Zimbra ke dalam operasi dengan salinan cadangan data dapat dikurangi hingga beberapa menit.

Untuk semua pertanyaan yang terkait dengan Zextras Suite, Anda dapat menghubungi perwakilan perusahaan "Zextras" Katerina Triandafilidi melalui email katerina@zextras.com
