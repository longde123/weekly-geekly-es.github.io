支持5G NR网络的C-V2X：车辆之间数据交换的新范例



5G技术将使更有效地收集遥测数据和发现车辆的全新功能成为可能，这些功能可以提高道路安全性并扩大无人驾驶车辆的范围。 在V2X系统（一种用于在车辆，道路基础设施元素和其他交通参与者之间交换数据的系统）中，有可能公开将使用哪种5G NR通信。 这将大大提高驾驶员，乘客和行人的安全水平，减少油耗和旅行时间。

今年3月，标准化5G网络的3GPP组织批准了首个C-V2X规范的引入，并在下一版的5G全球NR（版本16）标准中支持5G NR。 我们相信该版本将在2020年上半年采用。 这项技术与对eMBB（超宽带移动通信）的支持（其规范已在3GPP Release 15中获得批准）的结合，将是使用5G NR基于Qualcomm Snapdragon Automotive Platform创建智能汽车的第一步。

我们不等待5G网络的全球部署使用蜂窝技术在车辆之间发起直接数据交换。 早在3GPP版本14中，就描述了V2X技术，该技术允许汽车以一定的时间间隔与其他交通参与者（例如交通信号灯）交换基本信息。 在使用我们的C-V2X芯片Qualcomm 9150进行的多次测试中证明了它们的功能。使用C-V2X技术进行的直接数据交换使机器可以“看到”周围的环境，即使在其他物体不在直视范围内的情况下，例如在盲目的情况下十字路口或恶劣的天气。 同时，新技术补充并扩展了其他无源传感器（如雷达，LIDAR和摄像头系统）引入的功能，这些无源传感器的范围和可见性受到限制。

3GPP Release 16文档和支持5G NR的C-V2X标准化将把这些功能提升到一个全新的水平，并使车辆能够接收和发送更多信息，例如，来自传感器的更详细的数据以及有关交通参与者“意图”的信息，有关道路基础设施的信息。以及行人的运动 此外，关于“意图”的信息交换将有助于更有效地计划汽车路线，这对于将来无人驾驶汽车的发展很重要。 C-V2X将从最初在第14版中用作改善基本道路安全性的一种技术发展为交通参与者之间直接交互的工具，这将有助于提高交通安全性和交通意识，并减少燃料和时间消耗。这条路。



充分利用C-V2X和5G NR


基于5G NR的C-V2X解决方案使用与4G和5G网络一起出现的创新功能。 5G网络的第一个版本将于今年春天开始发布，并在3GPP Release 15中进行了标准化，引入了可扩展的信道频率网格步骤，该步骤也用于C-V2X。 其应用的一个例子是能够根据车辆的速度来改变参考信号的密度的能力。 根据我们的估计，在这种情况下，高速频谱效率将提高3.5倍，这对于使用C-V2X的新场景至关重要，例如，在汽车和道路基础设施元素之间交换来自传感器的大量数据。

支持5G NR的C-V2X的实现涉及无线电级别的一些重大改进，这是5G NR独有的。 在版本16中，首次将“侧”通信线路添加到5G标准中-这是V2X系统的直接数据交换通道。 这项技术将成为在智能手机和其他领域（例如公共安全）中使用5G NR开发未来解决方案的基础。 其创建的基础是针对LTE Direct的高通技术的发展，实际上导致了C-V2X技术在3GPP Release 14中的出现。 此外，版本14中描述的技术将允许支持较旧版本C-V2X的汽车在路上交换数据，即使使用同时使用两个版本的C-V2X的最新型号（来自版本14和版本16并支持5G NR）。

车辆之间数据交换的新范例


在使用移动网络进行数据交换的现代范例中，设备会根据基站的信号质量来更改信号传输参数，例如其调制和编码。 对于C-V2X，由于我们谈论的是不断移动的车辆，而不是固定的基站，因此任务变得复杂。 在这种情况下，仅信号质量不足以了解每种情况下哪种车辆都适合交换数据。 想象一下，在拐角处的十字路口是一辆汽车。 信号电平很弱，但是汽车本身非常接近，也就是说，环境对于我们的汽车很重要。 因此，在这种情况下，两辆车都应该能够从传感器接收完整的信息，而不管它们是否彼此直接可见。

这又意味着我们需要一个新的范例，该范例不仅要考虑信号电平，还要考虑对象之间的距离。 因此，开发5G网络的方法与上一代网络的构建方法不同。 特别是，在5G NR的“较低”级别（物理和MAC级别），需要进行距离估计。 例如，仅当车辆与发射器相距一定距离且仅当发射的信息对该车辆有用时，车辆才会发送确认，例如自动重复请求类型ACK / NAK。 这种方法将有助于解决上述汽车形式的“隐藏节点”的问题，该信号具有弱信号电平，该信号电平即将来临。 通常，由于它，所有车辆的信息传输的可靠性得以提高，并且确保了大系统容量，因为不再为某些交通参与者将网络资源用于传输“无用的”网络资源。

基于5G NR的C-V2X-不仅仅是数据传输技术


决定在3GPP Release 16中包括具有5G NR支持的C-V2X规范，这是标准化现代数据传输技术的重要一步，该技术可以满足汽车行业对包括无人驾驶汽车在内的新车辆的要求。 除数据传输方法外，我们还在区域标准（例如SAE，ETSI ITS和C-ITS）中进行研究并标准化更高级别的协议和消息传输方法。 这些标准化消息将使不同制造商的车辆能够充分利用新的C-V2X技术。 与3GPP Release 14中描述的C-V2X一样，具有5G NR支持的C-V2X解决方案将主要使用5.9 GHz频段，该频段已分配给全球许多国家/地区的车辆，例如美国，欧洲和中国。 在这种情况下，新版本的C-V2X将使用此范围内的其他通道。
