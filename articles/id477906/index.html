Slurm: diskon 50% untuk kursus online Black Friday


Secara pribadi, saya tidak suka diskon. Diskon besar berarti bahwa harga awalnya dibebankan dengan kuat.
Jika ada cadangan, lebih baik mengurangi harga. Mega Slurm pertama berharga 75 ribu, Mega kedua - 60, dan ketiga biaya 50.


Di sisi lain, saya suka tradisi Black Friday, meskipun itu di-vulkanisir di Rusia oleh kenaikan harga kuda pada malam hari.


Kami tidak dapat memotong harga intensif (tidak ada tempat), tetapi kami dapat melanggar prinsip "harga tunggal untuk semua bentuk pelatihan: ruang, udalenka, online."


Pada tanggal 29 November, tepat satu hari, Slerm memberikan diskon 50% untuk semua kursus online. Slurm Kubernetes (Junior + Base + Mega) dapat dibeli seharga 32.500 rubel, dan Slurm Prom baru (Prometheus) seharga 7.500 rubel. Jangan lupa untuk menunjukkan kode promosi BF19.


Jika saat ini tidak ada uang, atau anggaran perlu didiskusikan dengan manajemen, tinggalkan permintaan saat ada diskon, dan kami akan menyetujui ketentuan pembayaran.
