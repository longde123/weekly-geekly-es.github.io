Pembatasan Microsoft tentang pelarangan browser asing dalam pencarian terintegrasi untuk Windows 10 dapat diatasi



Suatu hari , Microsoft mengumumkan pengenalan pembatasan bekerja dengan browser lain dan mesin pencari dalam pencarian terintegrasi untuk Windows 10. Masalahnya adalah Cortana, asisten suara Windows 10 dengan elemen AI, sekarang hanya dapat bekerja dengan browser Edge dan layanan pencarian Bing.

Sebelumnya, pembatasan ini diselesaikan oleh alat bawaan sistem operasi, tetapi sekarang batasannya menjadi ketat. Bahkan jika Anda menginstal browser pihak ketiga dan mesin pencari di pengaturan sistem Windows 10 secara default, Cortana tidak akan bekerja dengan semua ini, hanya dengan alat "asli". Keterbatasan ini sudah dapat dielakkan jika Anda menggunakan solusi dari pengembang Martin Brinkmann) Benar, solusinya hanya dapat mengubah browser Edge menjadi yang lain. Namun dengan mesin pencari sementara masalahnya masih sama Bing. Tetapi mungkin salah satu pengguna Geektimes dapat menyelesaikan ini juga?

Butuh satu hari baginya untuk menyelesaikan masalah ini. Dia menjelaskan metodenya dan mempostingnya di Web. Tetapi sebelum mulai bekerja, Brinkmann merekomendasikan untuk menyimpan semuanya sehingga di masa depan tidak akan ada masalah, bahkan jika terjadi kegagalan.

Persiapan

Awalnya, Anda perlu mengunduh arsip Cortana Search Override . Arsip 27 MB ini perlu dibongkar, mengekstrak isinya ke direktori lokal. Perangkat lunak itu sendiri didasarkan pada kode dari sini .

Sebelum melanjutkan, Anda perlu memastikan bahwa browser yang ingin Anda gunakan untuk pencarian web ditetapkan sebagai browser default. Ini perlu diperiksa di Pengaturan Windows, Sistem> Aplikasi default. Jika browser lain diinstal secara default, Anda harus memilih browser yang Anda sukai.

Sekarang buka folder tempat isi arsip diekstraksi.



1. Bergantung pada OS mana yang diinstal pada PC Anda, 32-bit atau 64-bit, dan buka folder x86 atau x64;
2. Sekarang jalankan SearchUIMonitor.exe dari folder ini. Sebuah peringatan sistem akan muncul, yang kita bypass dengan mengklik "info lebih lanjut" dan "Tetap jalankan";
3. Sebuah jendela muncul bahwa, setelah operasi selesai, ditutup dengan sendirinya;
4. Jika Anda mendapatkan pesan bahwa dll hilang, buka folder vcredist dan jalankan file eksekusi 32-bit atau 64-bit. Kemudian kita kembali ke poin 2 dan menjalankannya;
5. Jalankan EdgeAutomation.exe;
6. Jendela baru muncul, yang segera ditutup dengan sendirinya.
