Internos em PHP


  • [RFC] Add interface Stringable - É proposto no PHP 8 adicionar uma nova interface Stringable que possa ser adicionada às classes que implementam o método __toString() . E use o tipo combinado string | Stringable onde a string deveria estar. 
     interface Stringable { public function __toString(): string; }
