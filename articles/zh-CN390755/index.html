Slack中的群组



与其他聊天室一样， 在Slack中，有一个方便的机会可以通过@user姓名或昵称与他人联系。对于所有人而言，昵称都将成为一个链接，并且在单击该昵称后，将为该用户打开一个菜单。昵称会向用户突出显示，并自动发送通知。

但是，如果您需要一次联系多个用户怎么办？有几种方法可以做到这一点。

第一种方式：切换到付费Slack


付费松弛的优点


  • 有内置的分组能力。
  • 无限集成。
  • 无限的文件托管。
  • 无限的消息历史记录并对其进行搜索。
  • 详细的统计信息。
  • 访客访问。
  • 依此类推。

有偿松弛的缺点


尽管有很多优点，但付费版本也有缺点：

  • 贵的 最低关税计划在第一个月为每人6.67美元，在第二个月为8美元。
  • 除配色方案外，未提供界面的可定制性。
  • 无内置视频通话（即将推出）。

第二种方式：通过突出单词来模仿群体


为此，在突出显示字词字段的通知设置中，您可以用逗号列出所有必要的组。



要找出存在哪些组或添加一个新组，您可以通过添加关键字来使用内置机器人，以使其对它们做出响应并发布有关的帮助。



机器人的答案-groups：

Type `@` with one of these groups: `manager`, `react`, `js`. Example: `@example`. It will send notifications automatically when you add necessary groups in Highlight Words: https://slack.com/account/notifications#highlight_words_div

机器人的回答-channels：

Type `#` with one of channels in Slack. Example: `#example`. It will send notifications automatically when you add necessary channels in Highlight Words: https://slack.com/account/notifications#highlight_words_div

因此，机器人会在聊天中回答。

如果免费的Slack中缺少某些内容，则可以使用机器人进行救援。

可以以任何方便的方式来存储组，例如，存储在配置中，或者，如果您需要通过命令从聊天中添加，则可以存储在数据库中。

访问特定组时，如果词典中有一个组，则漫游器会在所有用户的个人资料描述中搜索该组，并发布带有缺刻列表的消息，所有这些用户都已经在松弛级别上收到通知。

机器人优势


  • 免费的。您可以实施付费功能。
  • 增强的消息可定制性。
  • 您可以实现任何功能，甚至更强大。

缺点机器人


  • 有必要编写一个机器人（即使使用现成的解决方案）。
  • 带有中间消息的垃圾邮件聊天。当然，您可以立即删除该消息，但这只会令人讨厌。
  • 取决于概要文件描述的内容，长度限制为249个字符。

结论


当Slack中有很多用户时，就需要将组划分为在任何领域中更有能力和高度专业的人员。付费Slack的功能始终可以通过其他方式实现：既可以通过内置功能也可以通过机器人来实现。
