Neues Windows-Terminal: Antworten auf Ihre Fragen

In den Kommentaren zu einem kürzlich erschienenen Artikel haben Sie viele Fragen zur neuen Version unseres Windows-Terminals gestellt. Heute werden wir versuchen, einige von ihnen zu beantworten.

Unter dem Schnitt finden Sie einige der am häufigsten gestellten Fragen, die wir gehört haben (und immer noch hören), sowie offizielle Antworten: einschließlich des Austauschs von PowerShell und wie Sie das neue Produkt heute einsetzen können.



Wann und wo kann ich ein neues Windows Terminal bekommen?


  1. Sie können den Terminal-Quellcode von GitHub unter github.com/microsoft/terminal klonen und auf Ihrem Computer erstellen.
    Hinweis : Lesen und befolgen Sie unbedingt die Anweisungen auf der README-Seite im Repository, bevor Sie versuchen, das Projekt zu erstellen. Um das Projekt zu erstellen, müssen Sie einige Voraussetzungen und Initialisierungsschritte befolgen!
  2. Die Vorschau-Version des Terminals wird im Sommer 2019 im Microsoft Store zum Download angeboten.

Wir bemühen uns, Windows Terminal v1.0 bis Ende 2019 zu veröffentlichen, aber gleichzeitig werden wir mit der Community zusammenarbeiten, um diese Version zu veröffentlichen, damit das Terminal von hoher Qualität ist.

Ist Windows Terminal ein Ersatz für Eingabeaufforderung und / oder PowerShell?


Um diese Frage zu beantworten, klären wir einige Begriffe und Konzepte:

  • Eingabeaufforderung und PowerShell (z. B. WSL / bash / usw. in * NIX) sind Shells, keine Terminals und haben keine eigene Benutzeroberfläche
  • Wenn Sie das Shell- / Anwendungs- / Befehlszeilentool ausführen, startet Windows diese automatisch und verbindet sie mit Instanzen der Windows-Konsole (falls erforderlich).
  • Die Windows-Konsole ist eine standardmäßige „terminalähnliche“ UI-Anwendung, die im Lieferumfang von Windows enthalten ist und seit 30 Jahren von Benutzern zum Ausführen von Befehlszeilentools unter Windows NT, 2000, XP, Vista, 7, 8 und 10 verwendet wird



Daher sollte die Frage höchstwahrscheinlich wie folgt umformuliert werden: "Windows-Terminal - ist es ein Ersatz für die Windows-Konsole?"

Die Antwort lautet "Nein":

  • Die Windows-Konsole wird noch Jahrzehnte an Windows ausgeliefert, um Abwärtskompatibilität mit vielen Millionen vorhandenen / älteren Skripten, Anwendungen und Befehlszeilentools zu gewährleisten
  • Windows Terminal funktioniert mit der Windows-Konsole, wird jedoch wahrscheinlich zum bevorzugten Tool für Benutzer, die Befehlszeilentools unter Windows ausführen möchten
  • Windows Terminal kann eine Verbindung zu Eingabeaufforderung und PowerShell sowie zu jeder anderen Shell / Tool / Befehlszeilenanwendung herstellen. Sie können unabhängige Registerkarten öffnen, die mit Eingabeaufforderung, PowerShell, Bash (über WSL oder SSH) und anderen Shells / Tools Ihrer Wahl verbunden sind

Wann kann ich eine neue Schriftart bekommen?


Bald erhältlich! Wir haben keine festgelegte Zeitleiste, arbeiten aber aktiv an der Fertigstellung der Schriftart. Sobald es zur Veröffentlichung bereit ist, ist es geöffnet und in seinem Repository verfügbar.
