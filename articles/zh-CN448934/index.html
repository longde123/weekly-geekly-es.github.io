免费链接，或您可以随意与网站所有者联系的事情

本文对那些致力于以“被动模式”自行推广网站的企业主很有用：他们在业余时间会稍微研究问题，尝试深入研究各个方面，尝试不同的方法和服务。 换句话说，他们没有设定严格的期限，也没有在SEO中花费预算和成本项目。

如果您是一名营销人员，并且通常情况下（根据您的指示）同时（当然，您自己，没有预算且自愿）对网站进行宣传，那么我建议您仔细阅读本文中的内容，其中的一些内容可能至少会对此稍作修饰情况。

我提供了一个位置列表，您可以在其中空闲时间发布主要工作的链接。 我必须马上说，您不能指望搜索结果会起飞（尽管在极少数情况下，这种情况也会发生），我们正在谈论的是链接质量的逐步发展和积累，同时其他因素的不断改善和现场访问。

产品目录


这与“ Allsubmitter的白色基础”无关，根本不使用这些目录：如果主持人仍然活着并且有朝一日将您添加，那么该网站本身很可能早已列入黑名单，并且不会带来任何好处。

我们将发布大型目录，例如Rosfirm或Pulscen。 从一开始，它们的区别就在于它们是公司的目录，而不是站点。

比有用


  1. 将权重链接到您的网站。 如果目录中有处理链接的特殊规则或附加条件，则可能不存在。 但是如果有分量，那通常也不错：到2019年“幸存”的目录是相当可信赖的网络资源。
  2. 额外的流量。 数量很少，但是由于它是免费的，为什么不吸引它呢？
  3. 商业因素。 在此类目录中提及您的联系信息可以对排名产生积极影响。
  4. 品牌提及。 对于品牌任务和职位很有用。
  5. 搜索地点。 包含您的联系信息的页面在顶部占据其他位置，并且您从导航请求中收集了更多流量。
  6. 其他功能和字段。 现在，许多大型网站的所有者都知道，简单的联系信息不再像以前那样有价值。 因此，它们可以为您的卡添加其他类型的信息。 因此，例如在Pulscen上，您可以上传产品报价。 通过这样的站点上的页面，您可以从尚未排在最前面的键中获得良好而热门的主题流量。
  7. 订单 在这些地区，这些站点甚至带来了实实在在的利润，因此，即使对SEO没有很大的好处，这些链接本身也可能是有益的。

如何容纳


取决于网站允许的内容。 抓住一切机会，不要偷懒。 我建议编译一个表文件（如果还没有的话），并在其中输入有关公司的所有数据，管理层允许在开放源代码中发布这些数据。 这大大简化了进一步的工作。 每次您在网站上遇到一些新字段时，都将相应的行添加到表中。

社交网络中的社区


我故意将其范围缩小到社区主题，因为从广义上讲，SMM是一个劳动密集型的过程，最好让专业人员信任它，或者至少雇用一名职员。 在这里，我们谈论的是不需要很长时间。

搜索主题社区。 以下类型适合您：

  • 专业。 您的同事在这里交流。
  • 消费者。 在这里讨论您的产品和服务。
  • 相邻 它讨论了适合您的商品和服务的问题和事件。

比有用


  1. 提及。 如果您不断参与社区生活，您的出版物就会开始链接。 根据您进行活动的方式，代表谁和以何种形式进行活动，您会提高对公司或个人品牌的知名度。
  2. 社会信号。 从第一段开始：排名时，搜索引擎会考虑一些收集“喜欢”和转发的参考和链接。
  3. 网站流量。 如果您有机会对网站的资料做出回应（并且有这样的机会，如果您正确地找到了社区），那么您将获得额外的访问量。 如果同时您的邮件收集了“转发”，则其数量会增加。
  4. 行为因素。 由于我们是在讨论社区参与度高，策略正确的问题，因此参与其中的用户将进一步改善网站的行为表现。

如何容纳


没有单一的配方，但不要为了吸引自己不惜一切代价“吸引耳朵”。 找到自然的动机，并以自然的方式吸引观众。 仅当答案在链接后面时，才应给出链接。

论坛


需要注意的是，社交网络本身并没有太多，但活跃的论坛就您自己的主题而言-与社交网络几乎一样，通常您不需要第二手就可以重新讲述它们。

还有许多功能值得考虑：

  • 预审核和特殊条件。 与可以访问发布材料的社交网络不同，您可以立即发布它们，论坛通常会采用某些机制来规范社区活动。 在某些地方，您的答案和主题仅在经过审核后才会发布；在某些地方，您甚至必须首先通过回答其他主题来获得一定的评分。
  • 对链接和引用的重要性增加。 请准备好删除您所询问的问题的具有逻辑性和相关性的绝对纯真的出版物，以删除文本中的链接。 您无能为力，只需要习惯即可。 了解广告的艺术。
  • 对传统和规则的敏感性。 论坛是非常惰性的社区，在这些社区中，关于允许的内容的传统方式和想法在您看来可能是密集的，或者仅仅是奇怪的。 如果收到警告，不要惊讶，请尝试研究并阅读规则。

比有用


在这一点上，一切都与社交网络相同。 您还可以添加以下内容：

  1. 论坛主题通常是不常见问题的唯一答案，也是不常见问题的解决方案。 您可能自己找到了此确认。 当在一个狭窄的主题上，至少由您以某种方式可以帮助用户的唯一材料由您编写时，这非常好。 这是额外的品牌忠诚度。
  2. 与社交网络不同，在社交网络中，访问材料可能受到其限制，论坛页面通常是开放的，可供所有读者阅读。 因此，可以将您的资料的链接放在其他网站上。

如何容纳


主要目标是获得忠实的潜在客户。 因此，重要的是您确切地写什么以及如何写。 您必须具备表达能力，用户应了解您可以就此事向您寻求专家意见。

有几种住宿选择：

  • 回答/对话。 找到您有话要说的话题，进行对话，合理地交流，提供站点链接。
  • 主题。 提出一个吸引社区的有趣话题。 在主题本身和讨论中都可以给出链接。 只要记住上述的感知功能，并尝试尽可能正确地做所有事情即可。 如果您提出一个主题，那么它应该具有新颖性，或者至少具有讨论的潜力。
  • 来自个人资料的链接。 您不能在出版物中提供链接，但要在个人资料中适当地保留链接。 这大大降低了有益效果，但增加了您作为一个人的忠诚度。 请记住，只有那些对您的性格真正感兴趣或有想法向您订购商品的人才会关注您个人资料的链接。 使用这种策略，您将不得不在讨论中不断提醒自己，至少使潜在的受众在他们眼前与个人资料本身有链接。
  • 品牌分支。 对于商业公司来说，这是一种文明的方式，但是这样的机会并不存在。 关键是一些专业论坛为品牌创建了自己的分支机构。 在那里，公司发布新闻稿并回答社区问题，从“您为什么有这种特殊工作格式”到“您有这样的模型，但尺寸为41.5”。

大型娱乐/主题门户


现在人数不多，但是从这里您可以吸引大量的游客。 只有您需要非常小心地工作并记住住宿的主要危险：

  • 由于覆盖范围很广，因此如果成功，请为站点上的负载做好准备。
  • 如果您不同意，那么请准备好进行一场真正的消极海啸，这必须加以解决。
  • 您不是第一个如此狡猾的人，因此他们会通过棱镜“他来公关吗？”预先看您。 因此，您将需要编写相关且有用的内容，否则将无法解决负面问题。

比有用


  1. 行为举止。 这是一个带有所有后果的人群链接的好例子：很多用户，很多搜索引擎信息，您可以为网站获得超频。
  2. 参考重量。 由于这些站点的质量指标良好，因此您将从中获得一些链接权重（当然，除非它们关闭链接）。
  3. 提及。 即使关闭了链接的索引编制，您仍然可以在良好的上下文环境中获得对品牌的引用，尤其是在注释中出现讨论的情况下。
  4. 快顶。 机器人通常会拉动对某个话题进行激烈讨论的页面，尤其是当该站点将自己定位为新闻时。
  5. 病毒传播的潜力。 如果您设法“踏上社区的痛处”，则可以从社交网络，其他大型门户网站和新闻服务中获得许多链接，并获得随之而来的所有好处。 只是不要忘记，负数传播的速度甚至更快。
  6. 品牌和反馈。 您向社区介绍您自己，它会记住您。 作为回应，他告诉您关于您的信息。 而且几乎可以肯定，不仅令人愉快的事情。 写下来，这样您就不会忘记任何事情，并不断改善。

如何容纳


这完全取决于门户网站本身的规则和布局格式：

  • 简单发布。 如果门户网站以实时磁带（例如Pikabu）的格式工作，那么我们将注册并发布我们要考虑的内容。 我们将链接放在个人资料中，然后自然地放入帖子中。 或者，如果您无法自然输入，我们也不会输入：最好是等到评论中要求他们时再说，不要被指控为自我宣传。
  • 专栏 碰巧该资源是按照杂志的规则工作的，它具有修订版和某些工作规则。 提供自己作为永久作者的权限，并在第一人称中公开和合法地谈论您认为重要的事情。 在这种情况下，可能只允许您在个人资料中保留链接，并要求您遵守编辑政策。
  • 留言 不要忘记，您不仅可以在帖子中证明自己，还可以在帖子中证明自己。 而且常常在那里，您对自己会更忠诚。 如果您对这个话题有话要说-在网站上撰写，辩论，链接到材料和示例。 请记住，通常评论也有评级。

评论


这些都是像Otzovik这样的网站，人们在那里发表有关商品和服务的意见。 在这里我不是指Yandex.Market和其他市场，因为对于在线商店而言，这是促销商品的主要渠道之一，并且这是全职和日常工作，而不是在您闲暇时发布链接。

要考虑的细微差别：

  • 防止虚假评论。 一些资源需要购买或身份证明。
  • 预审。 通常，您的评论只会在经过验证后才会发布。

比有用


  1. 参考重量。 大型评论站点可以归因于网络的信任资源，尽管您不应指望过多。
  2. 交通量 正如实践所示，虽然评论流量通常不会很大，但您可以从评论中获得非常稳定的结果。 但是，这一切都取决于品牌的推广，数量令人印象深刻。
  3. 行为举止。 由于涉及到流量并对其进行了预热，因此行为指标得到了改善，故障百分比降低了。
  4. 评分系统。 尚不确定将您的网站置于适当位置的“星级”数目会受到多少影响。 当您的带有评论的页面位于顶部时，我们将认为这只是发行中的可喜奖励。
  5. 商业因素：与公司目录大致相同。

如何容纳


这完全取决于网站的限制和格式。 有几种可能性：

  • 公司卡。 如果获得公司注册，请确保执行以下操作：从配置文件中获取链接。
  • 商品和服务。 有更详细的配置文件，您可以在其中添加产品项并收集有关它们的反馈。
  • 产品评论。 有时，用户留下的印象不是特定的卖家，而是特定的产品。 在这种情况下，您可以在评论中在您的网站上留下指向该产品的链接。 当然，只是不能代表我自己。
  • 根据要求提供反馈。 某些站点允许用户询问有关公司的问题（如果尚未在列表中）。 其他用户对此请求发表评论。 在这里，您既可以充当提问者，也可以充当回答者。 也不能代表我自己。
  • 对评论的回应。 您可以代表将要发表评论的公司，对其地址中的肯定和否定做出回应。 在这些答案中，请参阅网站的页面。 自然，链接的指示应该是出于动机和逻辑的。

问与答


这些是类似“ Mail.ru答案”的网站，用户可以在该网站上就所有可能的主题提出问题，并从其他用户那里获得答案。 这种服务在这里很方便：

  • 有关该主题的问题不断涌入。 您可以每周访问该网站一次，并且在此期间内或多或少受到热门话题的影响，至少会出现1-2个问题。 因此，您可以定期且绝对合理地发布带有链接的答案。
  • 非常简单的发布规则。 要发布回复，该帐户不需要特殊的“功绩”，通常无需预先审核。 写下问题或答案，它立即出现。
  • 针对特定问题的网站页面的理想优化。 大多数人用相同的方式来表达正负问题，因此当有人在搜索引擎中搜索“什么是零电阻滤波器？”时，他立即获得一个链接，该链接指向以相同的措词询问该问题的页面。 并在响应中获得指向您网站上所需部分的链接。

比有用


  1. 参考重量。 与以前的网站类似。
  2. 交通量 这些也是人群链接，大量用户来找他们。
  3. 行为举止。 如果您在页面上正确回答了问题，那么吸引的所有这些用户都会增加网站的行为指标。
  4. 提及。 它们通常对于总体推广和品牌推广很有用。

如何容纳


有两个级别，您需要同时进行两个工作：

  • 答案 我们正在寻找有关我们主题的已问问题，我们正在通过指向您网站的链接回答这些问题。
  • 问题。 尽管这些网站很受欢迎，但总有一些问题尚未被提出，或者已经问了很长时间了。 可以在工具提示，Wordstat中找到它们，也可以仅根据您网站的逻辑来找到它们。 有必要提出问题，提出问题并给出带有链接的答案。 自然，来自不同的账户，而不是他们的账户。

接下来是什么


在“放松”的情节模式下尝试了上述渠道之后，您可以大致了解它们的工作方式，对它们的期望以及征服它们的要求。

在这里，最主要的是实现一个非常简单但不十分鼓舞人心的事情：在这些领域中的认真工作需要资金投入。 无论您是想接收来自论坛和otzovik的流量，还是想通过社交网络上的社区来提高品牌知名度，都没关系-如果涉及系统工作，则需要时间。

然后，您需要确定要支付谁的时间：您的时间还是承包商的时间。 原则上你有吗？ 我知道许多例子，说明董事和员工在不吸引第三方公司的情况下成功地开展了自己的业务。 但是，我什至更了解案件以零结果告终并向该机构提出上诉的情况。

这就是为什么我总是建议您尝试动手的原因：它不需要太多时间。 您将了解主要内容：

  • 你喜欢自己做吗？
  • 您是否能按计划进行所有工作？
  • 您计划要采取多少百分比的潜在行动，因为您显然没有时间。

最后，最重要的是没有指标就不值得一开始。 如果您未设置分析系统并开始标记链接，那么您将永远无法理解和计算以上内容的有效性。 至少放置Google Analytics（分析）和Yandex.Metrica。 在任何情况下都有用。 您可以添加呼叫跟踪和跟踪呼叫。 您可以配置端到端分析。 您对数据进行的分析越深入，您就越能更好地理解哪些有效，哪些无效。
