在后台收听什么：关于声音，音频，配乐和黑胶唱片的10个主题播客

俄罗斯和世界各地的播客越来越多样化。 在订户和戏剧数量上的领导者是涵盖政治和金融等一般主题的节目。

我们对以简单的文字谈论声音和音频设备的主题和大众科学计划更感兴趣。 今天，我们继续分享此类播客。

音频播客


声音有声


[ SoundCloud ]


这是同名的英国杂志关于音频技术的播客，该杂志自1985年以来每月发布一次。 尽管播客只是主要产品的附录，但它仍然是一个完全独立的节目。 要了解播客，您可以访问杂志的网站或其YouTube频道 。

节目的口头发行不定期在SoundCloud上进行。 根据编辑的说法，这取决于作者是否有空闲时间。 当他们仍然有机会在麦克风旁见面时，讨论将涉及各种设备和事件。

该杂志是AES，NAMM，NAB，IBC和Musikmesse等音乐技术领域会议的定期参与者。


音效


[ iTunes ] [ 网络版本 ]


这是有关俄罗斯Hi-Fi世界的播客，在Audiomania的支持下已发布了4年。 最初，这些问题的主题是揭穿多年来在音频技术主题上的“空寂”所积累的神话。

播客在工作的第一年就超出了本期的范围，现在它侧重于原始主题-音频设备商店如何选择商品，如何建造电影院大厅，如何创建电影配乐。

为了更深入地了解Sound，建议您熟悉我们去年制作的42个问题的摘要 。 您可以在此处查看播客录制过程的外观。


初学者发烧友


[ iTunes ] [ Player.fm ]


名称完全描述内容时就是这种情况。 对于他们的狂躁的听众，演讲者讲述了如何选择正确的技术，同时又节省了成本，并向专家学习了昂贵的音频设备与廉价的音频设备之间的区别 。

在最近的一系列展览中，主持人分享了他们对CES所见所闻的印象。 播客只有一岁了，但是iTunes上的听众的积极反馈表明 ，主持人并没有白费力气。


家庭影院极客


[ iTunes ] [ Player.fm ]


业界领先的AVSForum家庭影院和电影门户网站的编辑Scott Wilkinson的存档笔记。 自2009年以来已发布播客。 Scott 的第一位客人是 Tomlinson Holman，他与George Lucas合作开发了THX ，这是对高质量家庭影院系统的一系列要求。 传统上，发行版中很多注意力都集中在音频设备上。

去年10月，斯科特（Scott）向该出版物的读者发出了有关中止该展览的信息。 据他说，这一决定完全是出于经济原因。 现在，可以在其中一个播客平台上缓慢收听所有373个问题。 但是，斯科特给未来的家庭影院怪胎带来了新的希望。

Beardycast


[ iTunes ] [ Podtail ]


俄语播客，不能说它完全致力于声音。 更具体地说，音频设备是此计划中很少见的客人，但鉴于俄语中的Hi-Fi鉴赏家的演出数量很少，因此Borodokast在该系列中值得一提。

它属于娱乐节目的类别。 演讲者主要讨论技术新闻，电影以及最热门的新闻提要。 有时通话时间也流向音频设备。 在最近一期中，提出了音频市场的问题。


电影配乐



如何制作游戏


[ iTunes ] [ Player.fm ]


此合集中的另一个俄罗斯播客。 游戏开发者谢尔盖·加伦金（Sergey Galenkin）和米哈伊尔·库兹敏（Mikhail Kuzmin）着重讨论了游戏开发的各个方面-从游戏设计到法律问题。 除其他事项外，该合集还提供了一系列有关游戏音乐的有趣播客 。

讨论作者涉及的作曲家和声音工程师。 例如，从这些问题中，您可以了解由俄语为母语的声音工程师，声音设计师和作曲家组成的专业协会的工作，这些协会为游戏业谋福利。


Soundworks系列


[ iTunes ] [ SoundCloud ]


该播客的出现要归功于导演迈克尔·科尔曼 （ Michael Coleman） 。 几年前，他与史蒂芬·斯皮尔伯格，乔治·卢卡斯，詹姆斯·卡梅隆和其他著名导演对电影声音进行了一系列视频采访。

播客形式的SoundWorks Collection延续了各种作品中原声带和声音角色的主题。 同时，Michael与帮助他进行第一次面试的团队相同。 一个典型的问题是有关负责现场声音的机组人员工作过程的故事。

最近的播客之一是献给吉列尔莫·德尔·托罗（Guillermo del Toro）的那幅画《水的形状》（The Shape of Water），以及怪物的呼吸声作为创造影片理想氛围的工具之一。 另一个是关于最近发行的《星球大战》的原声带。 对于至少一次由于太空爆炸声而愤慨的怀疑论者来说，这个问题将特别有趣。


超级马卡托兄弟 电子游戏音乐播客


[ iTunes ] [ Player.fm ]


播客，讨论并分析不同世代电子游戏的最佳配乐。 该节目已经积累了令人印象深刻的档案-已发行300余期。 播客之所以特别有价值，是因为在几乎每个情节中，您都能听到旋律，唤起从小就对游戏机游戏的美好回忆。
