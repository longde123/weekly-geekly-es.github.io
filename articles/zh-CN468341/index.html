Windows终端上的Azure Cloud Shell

现在，您可以通过Windows终端连接到Azure云外壳 ！



为此，我们添加了一个新的默认配置文件-Azure Cloud Shell，该配置文件使您可以通过Windows Terminal应用程序访问Azure目录/客户端。

如果您已经安装了Windows Terminal


您的Windows终端设置不会自动更新到新的配置文件（因为每次打开Windows终端时文件不会更新），因此可以通过以下方式手动添加它：

  1. 启动Windows Terminal
  2. 打开设置（使用下拉列表）
  3. 将此配置文件添加到您的配置文件列表（项目配置文件）中： 

{"acrylicOpacity" : 0.6, "closeOnExit" : false, "colorScheme" : "Vintage", "commandline" : "Azure", "connectionType" : "{d9fcfdfa-a479-412c-83b7-c5640e61cd62}", "cursorColor" : "#FFFFFF", "cursorShape" : "bar", "fontFace" : "Consolas", "fontSize" : 10, "guid" : "{b453ae62-4e3d-5e58-b989-0a998ec441b8}", "historySize" : 9001, "icon" : "ms-appx:///ProfileIcons/{b453ae62-4e3d-5e58-b989-0a998ec441b8}.png", "name" : "Azure Cloud Shell", "padding" : "0, 0, 0, 0", "snapOnInput" : true, "startingDirectory" : "%USERPROFILE%", "useAcrylic" : true}

完成此操作后，您将可以使用“ Azure Cloud Shell”选项卡。

基本设定


  1. 单击“ Azure Cloud Shell”选项卡。
  2. 系统将提示您转到“ microsoft.com/devicelogin”并输入显示的代码。
  3. 在浏览器中输入代码后，您将需要使用您的帐户登录。 另外，请确保您使用具有活动Azure目录/客户端的帐户登录。
  4. 返回终端。 几秒钟后，您将看到消息“已认证”。
  5. 仅在某些情况下 ：如果您的帐户有多个客户，则将要求您选择其中一个。 只需输入您要连接的客户端的号码即可。
  6. 然后将提示您保存这些连接设置。 保存连接参数将使您将来无需进入步骤1-5即可进入系统。
  7. 然后，应用程序将开始与Cloud Shell建立连接（这可能需要一些时间，请耐心等待）。
  8. 您现在已连接到Azure Cloud Shell！
