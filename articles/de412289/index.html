Pritunl - VPN-Server für Sie und Ihre Freunde in 10 Minuten


Im Zusammenhang mit den neuesten Ereignissen in Russland möchte ich Ihnen von einem coolen selbst gehosteten VPN-Server erzählen. Nach einer einfachen Installation können Sie ohne viel Arbeit und fundiertes Wissen Benutzer erstellen, VPN-Profile erstellen und Diagramme zur Netzwerknutzung anzeigen.


Nachdem Sie eine VPN-Verbindung für sich und Ihre Lieben hergestellt haben, können Sie versuchen, die Kosten für einen virtuellen Server mit Ihren Freunden und Bekannten zu teilen.


Infolgedessen kann der Fünf-Dollar-VPS zu einem leistungsstarken Werkzeug zur Umgehung von Schlössern werden


Rückblick


Tatsächlich ist Pritunl ein Add-On zu OpenVPN mit einer schönen Web-Mündung, die einfach zu installieren ist und sofort funktioniert. Für die Verbindung reicht jeder openvpn-Client aus, und für Desktops wird ein separater regulärer Client bereitgestellt.

Installation


Die Installation ist extrem einfach und selbst jeder Anfänger kann damit umgehen:


  1. Zuerst müssen Sie einen VPS-Server außerhalb Ihres Landes kaufen.
  2. Auf diesem VPS reicht es aus, die im Handbuch beschriebenen Befehle auszuführen.
  3. Danach können Sie die Konfiguration über die Weboberfläche fortsetzen.
  4. Wenn Sie einen Domainnamen mit einem konfigurierten A-Eintrag haben, können Sie diesen in den Einstellungen angeben. Ein Letsencrypt-Zertifikat wird automatisch dafür empfangen.

Auf diese Weise können Sie einfach und uneingeschränkt Ihren eigenen VPN-Server erhalten.


Verbindung


  1. Für jeden Benutzer müssen Sie in Pritunl ein separates Profil erstellen.
  2. Ein Link zum Profil (oder zum Profil selbst) muss an den Benutzer gesendet werden.
  3. Der Benutzer muss dieses Profil nur mit der Anwendung öffnen und auf Verbinden klicken.
