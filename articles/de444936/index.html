Firmware ZXHN H118N von Dom.ru ohne Löten und Programmierer


Hallo!

Ich holte es aus dem staubigen Schrank und brauchte wirklich den ZXHN H118N von Dom.ru.

Das Problem liegt in der kurzen Firmware, die an den Anbieter dom.ru (ErTelecom) gebunden ist, wo Sie nur den PPPOE-Login und das Passwort für die Verbindung zum Internet eingeben können.

Diese Funktionalität reicht für eine Hausfrau, aber nicht für mich.

Deshalb werden wir diesen Router neu flashen!

Die erste Schwierigkeit besteht darin, dass Sie es auch nicht erneut flashen können. Dom.ru hat sich gründlich darum gekümmert. Es ist nur auf Firmware Dom.ru möglich. Aber hier in der alten Firmware dom.ru, auf die Sie zurücksetzen können (Downgrade), ist es möglich, die Originalversion der ZTE-Firmware einzugeben.

1. Geben Sie in die Adressleiste 192.168.0.1/html/menu.html ein und klicken Sie auf ein beliebiges Zeichen. Daraufhin wird die werkseitige Webcam angezeigt.

2. Gehen Sie zur Seite "Management"

3. Registerkarte "Firmware-Aktualisierung"

4. Flashen Sie zuerst die Firmware von DOMRU - DOMRU__ZXHN + H118NV2.0.1d_E04_OV.img (alte Firmware für home.ru mit Upgrade-Fähigkeit).

5. Nach der Firmware erhält der Router die IP-Adresse 192.168.0.1. Aber über das WEB darf man nicht ins Gesicht. Es ist nur über TELNET möglich - Benutzer: root Pass: root (wen interessiert das?).

6. Durch Drücken der Taste wird zurückgesetzt und in den Standardzustand versetzt.

6.1. Möglicherweise ändert der Router die Adresse in 192.168.1.1, und auch in den Einstellungen Ihres an den Router angeschlossenen Geräts müssen Sie die Details manuell eingeben
IP 192.168.1.2
Maske 255.255.255.0
Gateway 192.168.1.1
Ohne dies wurde ich bei 192.168.1.1 auf Seite 404 umgeleitet

7. Es gibt Zugriff auf WEB mit Benutzer: admin pass: admin

8. Weiter von Punkt 4. Nur mit der Original-Firmware aus dem Archiv flashen - ZXHN + H118NV2.0.1d_E04_OV.img

9. Zugriff auf WEB mit Benutzer: admin pass: admin

PS Im Internet sind viele Foren mit ähnlichen Themen und einer Überfülle an Methoden und Informationen gefüllt. Dies ist eine spezielle Methode, die mir geholfen hat. Ich hoffe, sie wird Ihnen helfen.

Update Bei neuen Versionen von Hardware Version V2.3 wurde diese Lücke geschlossen. Gültig nur bei älteren Versionen.
