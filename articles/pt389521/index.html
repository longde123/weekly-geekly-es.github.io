Placa Zero Cluster do Raspberry Pi: Cluster de computação zero do Pi



A Idein , uma startup no Japão, está atualmente desenvolvendo um módulo de computação com o Raspberry Pi Zero. O objetivo é a criação do Actbulb, um dispositivo multifuncional para resolver vários problemas, incluindo análise de dados. Este dispositivo foi projetado para funcionar em residências "inteligentes" e deve ser muito pequeno; planeja colocá-lo em um cartucho comum para uma lâmpada comum.

Até agora, tudo está no estágio de design, e a empresa decidiu criar o primeiro protótipo baseado no 16 Raspberry Pi Zero. De acordo com a idéia, um cluster de computação real deve ser capaz de resolver muitos problemas.



Para montar todas as “framboesas” juntas, uma placa especial foi criada com 32 portas micro-USB. Cada miniplaca possui 2 portas. Um para poder - o outro para troca de dados. Além disso, existem 16 conectores Ethernet.

“Estamos criando um sensor que usa o módulo de computação Raspberry Pi. Portanto, precisamos de muitas “framboesas” para desenvolvimento e teste. Utilizamos mini-PCs para processamento de imagens, aprendizado profundo e outras tarefas. Precisamos do Pi real, mas não apenas para trabalhar com eles como máquinas Linux ”, diz o desenvolvedor do projeto.



Como você pode ver, existe apenas uma “framboesa” no quadro agora, simplesmente porque a empresa não possui mais e os painéis não são vendidos em domínio público. Agora a empresa coleta "o mundo em um encadeamento" e, depois de coletar todos os 16 mini-PCs, o cluster funcionará com força total.

O cluster já está planejado para ser vendido com a adição de um switch Ethernet, GPIO, HDMI e muito mais.



O Twitter do fundador está discutindo ativamente a ideia, para que você possa oferecer algo por conta própria, se desejar.
