2. Graphique √† bulles avec capture de groupe


Parfois, vous voulez montrer un groupe de points √† l'int√©rieur de la bordure pour souligner leur importance. Dans cet exemple, nous obtenons les enregistrements de la trame de donn√©es qui doivent √™tre allou√©s et les transmettons √† encircle () d√©crit dans le code ci-dessous.

37. (ACF) (PACF)


Le graphique ACF montre la corr√©lation d'une s√©rie chronologique avec son propre temps. Chaque ligne verticale (sur le graphique d'autocorr√©lation) repr√©sente une corr√©lation entre la s√©rie et son heure, √† partir de l'instant 0. La zone ombr√©e bleue sur le graphique est un niveau de signification. Ces moments qui se trouvent au-dessus de la ligne bleue sont significatifs.

Alors, comment interpr√©tez-vous cela?

Pour les AirPassengers, on voit qu'√† x = 14, les ¬´sucettes¬ª ont franchi la ligne bleue et sont donc d'une grande importance. Cela signifie que le trafic de passagers observ√© il y a jusqu'√† 14 ans a un impact sur le trafic observ√© aujourd'hui.
