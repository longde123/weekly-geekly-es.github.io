Faites-le vous-même: jeu alcoolique "Sea battle"



Les amateurs d'impression 3D ont téléchargé des fichiers STL de navires à deux et trois étages pour jouer à BattleShots - il s'agit d'une variante du célèbre jeu «Sea Battle». De plus, une option assez spécifique.

Dans les navires à deux niveaux, deux ouvertures pour les verres, dans les navires à trois niveaux, trois. Les navires à quatre mains ne sont pas fournis, probablement pour des raisons d'humanité.

"Sea battle" est un jeu pour deux participants dans lequel les joueurs appellent à tour de rôle les coordonnées sur la carte d'un adversaire qui leur est inconnu. Si l'adversaire a un navire à ces coordonnées (les coordonnées sont occupées), alors le navire ou une partie de celui-ci «se noie», et le coup obtient le droit de faire un autre mouvement. Le but du joueur est le premier à toucher tous les navires ennemis.

Nous avons tous joué dans le "Sea Battle" sur les feuilles dans la boîte. Dans cette version, le jeu est né quelque part pendant la Première Guerre mondiale. Plus tard, la société américaine Milton Bradley Company a sorti un jeu dans une version de bureau, avec une boîte en plastique.



En 1979, le premier jeu informatique, Sea Battle, est sorti.

BattleShots avec des verres à liqueur est une autre réincarnation du jeu culte. On peut dire que c'est une option pour les adultes.

Quatre fichiers sont actuellement disponibles pour l'impression.

Vous devez imprimer deux ensembles. Vous aurez également besoin d'une planche pour le terrain de jeu, dessinée en carrés de 68 mm de côté, de petits verres ou des gobelets en plastique et quelque chose pour indiquer les manquements, par exemple, des bouchons de bouteilles.









La logique est claire. Si l'adversaire monte dans le vaisseau (blessé), alors vous devez boire un verre de boisson forte.
