Oculus GoBlack-如何改善Oculus Go

摘自Oculus VR的创始人之一，Oculus Rift虚拟现实头盔的发明者Palmer Lucky的博客。



像许多产品一样，Oculus Go在成本，发布时间表，质量和能力上都是一个折衷方案。 显然，这是市场上最好的独立VR平台，但是只要有一点技术知识的人都可以做得更好。

在过去的几个月中，我一直在进行Go的修改，因此，最终的设备在使用方面迈出了一大步。 GoBlack之后，您不想再回去！ 这是更新列表，并对每个更新的动机进行了一些解释。

色泽


Oculus Go stock模型的外壳由ABS塑料制成，在与面部和紧固件的连接处使用了各种合成材料的混合物。 该设备的颜色是一种可爱的主流灰色，很好地掩盖了铸造过程中的小缺陷。 我是黑色VR头盔的忠实拥护者，因此黑色绘画在更新列表中居于首位。 考虑到头盔在控制对比度和其他变形方面的关键作用，这对于头盔内部的所有表面尤为重要。 显示器，鼻隙或其他间隙发出的任何光线都会从这些表面反射并入射到透镜和光学腔中-如果您不喜欢这些射线，请学习欣赏黑色的磨砂表面！

从外部喷涂外壳只会使设备更重，无法解决靠近脸部的问题，因此我决定用聚酯染料覆盖一切。 注意 ：这样做时要小心。 用染料涂覆塑料需要使用接近沸点的热液体，如果温度超过几度，塑料将变形。 如果温度太低，染料将不会浸渍塑料。 你会被烧死的。


不要喝酒！ 用冷水冲洗纺织品，不要让浓缩的塑料染料沾到皮肤上！


左侧是Oculus Go的原始前部，右侧是GoBlack的染料浸透部分。 猜猜哪一个反射的光更少。


我还更换了镜头周围的固定器，以使它们突出的次数更少。 设计头枕时，请始终尽量减少反射少量光的表面的使用。 这就是为什么Rift中的所有部件都使用哑光黑色织物（一直到镜头）完成，并且将开放塑料区域最小化的原因之一。

机重


Oculus Go很重。 从表面上看，它的重量胜过Rift，但总的来说，这主要是由于织物带所致，它不会增加太多的重量感，并且在头部运动时不会对皮肤造成压力。 设备本身放在脸上，重量略超过400克。

经过几次修改，我将其减少到280克，并且大部分凸起都落在设备的前边缘，感觉最明显。 从主观上讲，重量似乎减轻了一半，尤其是在运动过程中。 这是我所做的：

更换电池 ：我从头盔的前部卸下了电池，然后将其移动到了头部的顶部，并与魔术贴连接到了顶部皮带。 最初，它不是一个容量特别大的2600 mAh 18650电池，该设备可以使用两个小时，但我决定花3500 mAh来使用，该电池可以工作三个小时以上。 它通过弹簧插头[pogo-pin连接器]连接，当我想继续演奏时，可以轻松切换到新电池。 在游戏过程中充电更为方便，这可能导致过热。 制作电池模块使您可以整天使用头盔非常简单，但我认为我不会这样做。





散热/散热片 ：Oculus Go的冷却系统是被动的，它使用热管将热量从Qualcomm Snapdragon 821 SOC分配到铝板上。 不幸的是，该系统很重并且位于头盔的前部，从而最大程度地影响了使用舒适性。 我很难接近头盔的重量-头戴式显示器与火箭类似，因为它们似乎从轻量化中受益匪浅。

为了解决这个问题，我安装了一个全新的冷却系统。 热管更短，更轻，它可以将热量从单芯片系统带到附近的一排排小铝翅片，并由一个小风扇冷却。 该系统由主板供电，我将铝板换成了偏光塑料。 在我看来，结果看起来不错。



我还卸下了所有的盖子和屏幕 。 这节省了几克，但是我喜欢这种美学，即能够看到每个芯片上的所有外围设备。

减少塑料用量 ：可以采用更极端的方法，但是我想要可靠性，耐用性和外观像商店一样。 我设法除去了用于支撑冷却系统和前面板的大量塑料，但我的朋友GOROman在其Go修改中将这种想法发挥到了极致！

杂项


瞳距调节 ：出于很多原因，我拒绝了此选项。 这个距离在Oculus Go中是固定的，因此提高分辨率对距离太大或太小的人都无济于事。 我的距离相当大，大约70毫米，Oculus Go不太适合我-像场的不正确几何变形，比例和曲率会导致眼睛疲劳和模糊。 我怀疑大多数定期使用Go的人的瞳孔间距都接近平均水平，但是不幸的是，我们将Rift设计为适合几乎所有男人和女人，并为此感到自豪。 随着显示分辨率的提高，适当的光学调谐变得越来越重要。 无论眼睛之间有多远，每个人都应该可以访问虚拟现实革命。

移动键盘 ：我认为Oculus Go最有趣的功能之一就是支持虚拟桌面 ！ 我可以使用大型显示器在几乎所有地方使用Windows。 我在自己最喜欢的机械键盘之一Happy Hacking Keyboard Professional Type 2 S中为自己组装了一台PC，该键盘带有未编写的键（在VR中不需要它们，在那里仍然不可见！）。 全尺寸键盘+ Windows + GoBlack胜过任何笔记本电脑！



6DOF，Oculus Quest，NOLO等 ：GoBlack目前很少使用，主要是由于瞳孔间距的问题。 我对Oculus Quest的调整感到非常高兴，我真的很期待！ 如果需要，请等待我谈论过的新的前庭固定术。


最新版本的NOLO VR（用于Oculus Go的第三方跟踪系统）给我留下了深刻的印象，该固件经过多次固件更新后变得更好了。
