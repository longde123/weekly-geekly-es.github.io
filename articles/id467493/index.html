Bagaimana menjadi kurang dapat diakses oleh penyerang potensial di Internet. Pengalaman dan pengamatan pribadi

Entri


Mengapa penting untuk memikirkan keamanan online? Contoh kecil, jika penyerang mendapatkan akses ke akun Anda pada beberapa sumber daya, ini dapat menyebabkan berbagai jenis kerusakan, baik material maupun moral. Berikut ini beberapa contohnya.

Seorang penyerang mendapat akun dari jejaring sosial. Hal ini menyebabkan distribusi tanpa ampun yang kadang-kadang bukan berita terbaik untuk semua teman, pesan seperti itu dapat berdampak negatif terhadap kesehatan seseorang yang dekat dengan mereka.

Seorang penyerang mendapat akun dari situs pemesanan hotel. Dengan membatalkan reservasi, itu dapat merusak liburan, serta menyebabkan kerusakan materi.

Toko online, di sini penyerang juga dapat melakukan banyak kerusakan dengan membatalkan pembelian yang telah dibayar.

Mungkin ada banyak contoh seperti itu, dan, lebih dari itu, penyerang dapat berpikir jauh lebih licik dan halus, mengambil keuntungan dari ini. Nah, jika seorang penyerang menerima akun dari surat, ini memberikan ruang terbuka yang sama sekali tidak terbatas untuk sabotase. Anda tidak pernah tahu penyerang apa di Internet akan dapat memanfaatkan hari esok, jadi saya membuat daftar kecil item yang dapat membantu menjadi lebih aman.

Mail


Solusi standar : sumber daya gratis dengan fungsionalitas dan iklan yang luas

Solusi aman: terbatas dalam fungsionalitas, surat gratis atau surat berbayar penuh dengan enkripsi dan tanpa iklan

Detail: Ada kemungkinan mengenkripsi surat (tiba-tiba akan diperlukan untuk mengirim sesuatu yang penting, dokumen penting, informasi kartu bank, dll.), Akun rahasia (tidak memerlukan data pribadi selama pendaftaran), kode sumber terbuka (saya tidak mengerti apa-apa tentang ini) , tapi saya ingin percaya bahwa mereka yang mengerti, melihat dan menyetujui), kurangnya iklan dan koleksi metadata.

Berikut ini gambaran umum terperinci dari beberapa layanan email.

Sebagai contoh untuk dipertimbangkan, saya memilih dua layanan.
Posteo - biaya 1 euro per bulan, ada layanan kalender dan kontak, IMAP, dukungan POP3. Semuanya bekerja dengan baik.

Protonmail - IMAP dan POP3 tidak tersedia dalam versi gratis. Ada aplikasi untuk smartphone. Saya senang dengan fungsi seumur hidup pesan, tetapi saya belum menemukan tujuan untuk ini.

Drive awan


Solusi khas : top-10 drive cloud gratis

Solusi Aman: Drive Cloud Terenkripsi Berbayar

Detail: Ini adalah transisi ke penyimpanan data yang aman, seperti data di cloud dienkripsi, baik dari penyerang potensial dan dari pencipta cloud drive.
Dari seluruh baris disk cloud, saya menyukai layanan Mega, tetapi ada artikel di sini yang merinci kelemahan keamanan layanan ini. Namun demikian, saya menyukai layanan ini karena beberapa alasan. Ada aplikasi untuk menyinkronkan data antara komputer dan cloud, sehingga pencadangan ke cloud dilakukan dengan cepat. Juga dimungkinkan untuk melacak versi file, seperti disimpan hingga 5 versi sebelumnya. Jika Anda secara tidak sengaja menghapus sesuatu dari file, Anda dapat menemukan salinan lama dan memulihkan data yang hilang. Sumber terbuka. Antarmuka intuitif. Aplikasi yang nyaman untuk smartphone.

Cons: kemungkinan besar Anda ingin beralih ke versi berbayar dari sumber daya yang dipilih (versi gratis biasanya hingga 20GB). Tidak ada cara untuk melihat file menggunakan aplikasi cloud disk, file perlu diunduh untuk dilihat (ini berlaku untuk versi web).

Penyimpanan kata sandi


Solusi standar: passwords.txt, 2-3 kata sandi standar

Solusi Aman: Gunakan pengelola kata sandi di mana tidak ada kata sandi rangkap.

Detail: Penyimpanan kata sandi dalam bentuk yang aman, sinkronisasi antara komputer dan telepon pintar, pelengkapan kata sandi secara otomatis untuk browser. Sekarang browser tidak menyimpan kata sandi. Inilah artikel yang bagus dengan gambaran umum beberapa layanan penyimpanan kata sandi.

Saya menyukai pengelola kata sandi Kaspersky, tetapi ada kode sumber tertutup dan biaya 450 rubel per tahun.

Menyimpan data pada flash drive atau hard drive eksternal


Solusi khas: data pada flash disk

Solusi aman: data terenkripsi pada flash disk

Detail: Data dienkripsi oleh program dari pabrikan, tidak cukup aman, tetapi lebih baik daripada tidak sama sekali. Jika kehilangan atau pencurian USB flash drive atau hard drive eksternal, Anda dapat khawatir sedikit kurang dari jika tidak ada enkripsi.

Cons: Media penyimpanan mulai bekerja lebih lambat.

Penyimpanan data komputer


Solusi umum: data yang tidak dienkripsi pada disk komputer

Solusi Aman: Drive ini dienkripsi dengan sistem enkripsi BitLocket yang ada di Windows.

Detail: Dalam kasus kehilangan atau pencurian komputer, kemungkinan bahwa data dapat jatuh ke tangan penyerang berkurang.

Jaringan sosial dan sumber daya lainnya di Internet


Yang paling aman adalah tidak menggunakan. Yang paling nyaman adalah menggunakan semuanya. Kita perlu menemukan "jalan tengah" sehingga Anda dapat menggunakan, berbagi foto dengan teman, mendiskusikan sesuatu, tetapi tidak terbuka untuk seluruh dunia.

Misalnya, jika diskusi tentang transaksi pembelian-penjualan sering kali berisi data seperti siapa yang akan berada pada jam berapa, penyerang dapat menggunakan informasi ini untuk tujuannya sendiri. Buka akun media sosial dengan aliran liburan dan katakan kepada calon perampok: "Tidak ada orang di rumah, Anda dipersilakan!".

Bank


Solusi khas: satu kartu, satu akun.

Keputusan aman: dua kartu, tiga akun. Kartu utama + akun untuk pembayaran dengan kartu di toko (selalu ada sedikit). Pisahkan kartu + akun untuk pembelian di Internet (hanya mengisi jumlah pembelian sebelum membeli). Pisahkan akun penggajian tanpa merujuk ke kartu.
Detail: Kartu utama dapat dicuri, kode pin dapat dimata-matai / baca, jika terjadi kehilangan atau pencurian tidak ada risiko kehilangan banyak. Saat membayar di Internet, kartu-kartu ini dapat dikompromikan, mengeluarkan kartu virtual setiap kali sebelum pembayaran adalah proses yang melelahkan dan mahal, akan lebih mudah untuk memiliki kartu terpisah untuk keperluan ini.

Kekurangan: Sebelum membeli, Anda harus mengisi kembali kartu atau akun yang sesuai.

Siapa yang tahu apa yang akan dimanfaatkan penyerang besok. Lebih baik menjadi sedikit kurang mudah diakses, sambil tidak melewati ambang akal sehat dan kenyamanan.
