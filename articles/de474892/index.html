Programmierung für Kinder. Fünf der coolsten HTML- und JavaScript-Spiele

Hi, habrozhiteli! Möchten Sie ein tolles Geschenk für ein Kind machen, das lernen möchte, wie man programmiert, oder einen Erwachsenen unterrichten, der weit weg von der Welt der Codes ist? Dann passt die Heldin unseres Postens zu Ihnen. In diesem Buch erfahren Sie, wie Sie Code für Web-Spiele in HTML und JavaScript schreiben und dem Leser sogar dabei helfen, einige unterhaltsame Spiele zu schreiben. Unter dem Schnitt befindet sich eine Passage aus einem Buch.






Denken Sie, dass die Programmierung schwierig ist? Gar nicht! Lernen Sie das Programmieren, indem Sie Spiele in HTML und JavaScript erstellen. "The Adventures of Mario" und "Minecraft" werden Zyklen, Timer, Scoring, die Arbeit mit Grafiken und sogar mit künstlicher Intelligenz vorstellen. Kurze und einfache Erklärungen erleichtern die Erstellung Ihres eigenen Spiels.

Zu Beginn jedes Kapitels finden Sie eine Beschreibung der Aufgabe des Happy Cat Clubs. Jeder wird Ihnen von dem Spiel erzählen, das die Mitglieder des Clubs erstellen möchten.

Der Code jedes Spiels ist in kleine Fragmente aufgeteilt. Befolgen Sie die schrittweisen Anweisungen, um das Spiel von Anfang bis Ende selbst zu schreiben.

Am Ende jeder Mission haben Sie ein Do-it-yourself-Spiel und erworbene Fähigkeiten.
Spielen Sie, entspannen Sie sich und besuchen Sie die Website des Herausgebers Peter, um neue Bücher zum Thema Programmierung zu lesen.
