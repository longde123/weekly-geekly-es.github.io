Rilis YouTrack 2018.2: halaman proyek baru, gambar YouTrack Docker dan banyak lagi

Halo semuanya! Kami telah merilis versi baru dari sistem manajemen proyek - YouTrack 2018.2 - dan kami sedang terburu-buru untuk berbagi inovasi dengan Anda.




Apa yang baru di YouTrack 2018.2?

  • halaman proyek baru;
  • YouTrack Docker;
  • Otentikasi SAML
  • pembatasan upaya login dengan login;
  • manajemen data pribadi;
  • Menampilkan tugas dalam antrian di papan Agile sebagai daftar.
  • penyortiran kartu secara eksplisit pada papan Agile;
  • pembatasan pemain dan tanggal mulai untuk grafik Gantt;
  • Opsi pengurutan baru untuk set nilai di bidang khusus
  • kemampuan untuk melampirkan file ke komentar;
  • kemampuan untuk memperbarui tugas tanpa mengirim pemberitahuan dalam aturan tentang jadwal dalam alur kerja;
  • kata kunci baru untuk mencari versi yang dirilis dan diarsipkan;
  • kemampuan untuk membuat bidang khusus wajib.

Fungsi Eksperimental:


  • halaman pembuatan tugas baru.

Halaman proyek baru


Di versi baru YouTrack, halaman Projects menjadi lebih nyaman. Sekarang Anda dapat menandai proyek sebagai favorit, serta pergi ke pengaturan proyek apa pun langsung dari halaman proyek.




YouTrack Docker


Jika Anda menggunakan YouTrack Standalone, sekarang jauh lebih mudah untuk meningkatkan ke versi baru. Anda dapat mengunduh gambar YouTrack dari repositori dan membuat wadah Docker dengan instalasi baru, atau mentransfer instalasi yang ada (ZIP, MSI, JAR) ke wadah Docker.


Otentikasi Menggunakan SAML


Modul otentikasi ini memungkinkan pengguna untuk masuk ke YouTrack melalui layanan otorisasi SAML pihak ketiga, seperti Okta.




Login Pembatasan Login


Di YouTrack 2018.2, menjadi mungkin untuk membatasi upaya login yang gagal dengan login. Dikombinasikan dengan verifikasi reCAPTCHA, fungsi ini mencegah serangan dengan bruteforcing passwords dan tidak memblokir pengguna yang hanya lupa kata sandi mereka.




Manajemen data pribadi


Sebagai bagian dari persyaratan GDPR, kami telah menambahkan kemampuan untuk menganonimkan data kami.




Perbaikan lainnya


Menampilkan antrian tugas sebagai daftar


Di versi baru, ada dua cara untuk melihat antrian tugas di papan Agile: dalam bentuk daftar atau pohon. Sebelumnya, hanya satu tampilan yang tersedia, di mana subtugas dilampirkan ke tugas-tugas utama dan disajikan dalam hierarki. Tampilan baru memungkinkan Anda untuk hanya menampilkan tugas-tugas yang ditentukan dalam permintaan.




Penyortiran kartu secara eksplisit pada papan Agile


Sekarang Anda dapat memasukkan penyortiran kartu secara eksplisit pada papan Agile. Misalnya, jika Anda ingin tugas di papan tulis selalu diurutkan berdasarkan prioritas, masukkan "urutkan berdasarkan: prioritas". Harap perhatikan bahwa penyortiran manual tidak akan berfungsi jika penyortiran tugas eksplisit ditentukan.




Pelaku dan batas tanggal mulai untuk grafik Gantt


Kami telah menambahkan dua parameter baru untuk laporan Gantt Chart, yang memungkinkan Anda untuk menetapkan batasan tambahan pada distribusi tugas dalam bagan. Jadi, Anda dapat menetapkan batasan sesuai dengan pelaksana di proyek dan di bidang "tanggal mulai".




Opsi pengurutan baru untuk set nilai di bidang khusus


Untuk sebagian besar jenis bidang khusus, sekarang Anda dapat mengurutkan nilai berdasarkan properti sekunder. Selain mengurutkan berdasarkan nama, Anda dapat mengurutkan nilai berdasarkan nomor versi, tanggal pembuatan, dan tanggal rilis. Ketika nilai-nilai di bidang khusus diurutkan, parameter ini dapat digunakan dalam pencarian.




Lampirkan file ke komentar


Sekarang Anda dapat melampirkan file tidak hanya untuk tugas, tetapi juga langsung ke komentar di bawahnya. Inovasi ini nyaman untuk tim yang sering bertukar file dalam komentar (misalnya, desainer) - sekarang menemukan versi terbaru file terlampir menjadi lebih mudah.




Cari tugas serupa


Kami telah menambahkan permintaan pencarian baru - sepertinya : dengan itu Anda dapat menemukan tugas yang mirip dengan tugas yang ditentukan. Ini berguna untuk menemukan duplikat saat membentuk tumpukan produk.




Kemampuan untuk memperbarui tugas tanpa mengirim pemberitahuan dalam aturan tentang jadwal dalam alur kerja.


Opsi aturan jadwal baru memungkinkan Anda untuk memperbarui tugas di latar belakang sesuai dengan jadwal yang ditentukan.


Kemampuan untuk membuat bidang khusus wajib


Sebelumnya, untuk membuat bidang wajib diisi, Anda harus menggunakan aturan alur kerja. Sekarang dalam pengaturan proyek dimungkinkan untuk membuat bidang wajib diisi tanpa menetapkan nilai default. Dengan demikian, pengguna yang membuat tugas akan diminta untuk menetapkan nilai bidang ini.




Kata kunci baru untuk mencari versi yang dirilis dan diarsipkan


Saat mencari tugas, kini Anda dapat menggunakan kueri yang akan menemukan versi produk yang dirilis dan diarsipkan. Misalnya, untuk menemukan tugas dengan versi perbaikan "diarsipkan", Anda dapat menggunakan versi perbaikan: Permintaan diarsipkan .




Grup dan Tim


Kami telah memperbarui halaman Grup, yang sekarang menampilkan grup dan tim. Selain itu, Anda dapat menggabungkan grup dengan tim proyek.




Peran kloning


Mengkloning peran memungkinkan untuk membuat peran baru dengan set izin yang sama persis dengan peran yang ada. Jika diperlukan, semua nilai kemudian dapat diedit.




Pengaturan Modul Otentikasi Umum


Semua pengaturan yang terkait dengan parameter otentikasi telah ditransfer dari halaman Pengaturan Global ke halaman Pengaturan Umum. Juga di halaman ini adalah pengaturan untuk fungsionalitas baru yang membatasi upaya login dan pengaturan Google reCAPTCHA.




Mengatur Atribut untuk Penyedia Layanan SAML


Jika Anda menggunakan YouTrack sebagai penyedia identitas SAML, kini Anda dapat menentukan nama atribut untuk nama pengguna, nama depan dan nama belakang, serta alamat email pengguna yang diteruskan ke penyedia layanan SAML.




Fungsionalitas eksperimental


Halaman pembuatan tugas baru


Kami terus berupaya meningkatkan antarmuka YouTrack: dalam versi ini kami telah mengubah halaman untuk membuat tugas baru. Untuk menggunakan fungsionalitas baru, administrator harus mengaktifkannya di halaman Konfigurasi Fitur.


Selain peningkatan UI umum, perubahan berikut ini muncul di halaman baru:


  1. Pilihan proyek telah dipindahkan ke tajuk halaman.
  2. Fungsionalitas konsep yang ditingkatkan:
    • tag dan lampiran ditampilkan dalam daftar konsep;
    • sebuah tombol muncul untuk menghapus semua draft dari daftar.


Kami juga melakukan sejumlah perbaikan lainnya - lihat daftar lengkap tugas yang termasuk dalam rilis.


Coba YouTrack 2018.2 dan bagikan kesan Anda dengan kami! Anda dapat mengunduh versi gratis untuk 10 pengguna atau mendaftar di cloud untuk menggunakan versi uji coba selama 30 hari.


Jika Anda sudah menggunakan YouTrack berbasis cloud, maka server Anda akan ditingkatkan ke versi baru sesuai jadwal .
