Blazor 0.9.0 dirilis

Blazor 0.9.0 sekarang tersedia! Rilis ini menambahkan peningkatan komponen Blazor Razor ke .NET Core 3.0 Preview 3.


Peningkatan Komponen Pisau Cukur Baru sekarang tersedia untuk aplikasi Blazor:


  • Penanganan acara yang lebih baik
  • Formulir dan Validasi


Menantikan tanggapan Anda!

Lihat pengumuman ASP.NET Core 3.0 Preview 3 untuk mengetahui detail dari peningkatan ini. Lihat juga dokumentasi rilis Blazor 0.9.0 untuk lebih jelasnya.


Catatan: Templat Blazor belum diperbarui untuk menggunakan ekstensi file .razor baru untuk komponen Razor dalam rilis ini. Pembaruan ini akan dibuat dalam rilis mendatang.


Instal Blazor 0.9.0


Untuk memulai dengan Blazor 0.9.0, instal yang berikut ini:


  1. .NET Core 3.0 Pratinjau 3 SDK (3.0.100-preview3-010431)
  2. Visual Studio 2019 (Pratinjau 4 atau lebih baru) dengan ASP.NET.
  3. Perpanjangan Blazor terbaru dari Visual Studio Marketplace.

  4. Templat Blazor pada baris perintah: 


    dotnet new -i Microsoft.AspNetCore.Blazor.Templates::0.9.0-preview3-19154-02

Petunjuk memulai, dokumen, dan tutorial untuk Blazor dapat ditemukan di blazor.net .


Tingkatkan ke Blazor 0.9.0


Untuk memutakhirkan aplikasi Blazor yang ada ke Blazor 0.9.0, pertama-tama pastikan bahwa Anda telah melakukan semua hal di atas.


Untuk meningkatkan proyek Blazor 0.8.0 Anda ke 0.9.0:


  • Perbarui paket Blazor dan tautan .NET CLI tool ke versi 0.9.0-preview3-19154-02.
  • Perbarui paket Microsoft.AspNetCore yang tersisa. * Hingga versi 3.0.0-preview3-19153-02.
  • Hapus semua penggunaan JSRuntime.Current dan gunakan injeksi dependensi sebagai gantinya untuk mendapatkan instance IJSRuntime dan mentransfernya ke tempat yang diperlukan.
