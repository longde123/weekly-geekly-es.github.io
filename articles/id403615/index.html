Sakelar sentuh kaca sudah ada di studio

Pada akhir minggu lalu, pameran InterStroyExpo 2017 diadakan, di mana perusahaan Nootechnika yang terkenal ikut serta.

Saya berhasil melihat dan menyentuh prototipe sakelar sentuh model PSF-3 yang baru. Hanya dalam beberapa bulan, panel-panel cantik ini akan tersedia untuk umum:



Prototipe menggunakan kaca Livolo C6 yang dihentikan, panel Wemmon akan masuk ke seri. Dengan demikian, dimungkinkan untuk memilih aksesori listrik ke sakelar (soket, konektor audio dan video, bahkan port USB untuk mengisi daya telepon). Untuk memilih aksesori dan tata letak pemutus sirkuit, Anda dapat menggunakan perancang , di mana setelah rilis resmi pemutus sirkuit dari Nootehniki, akan dimungkinkan untuk memilih nooLite sebagai mekanisme.




Ketika Anda melihat sakelar, tampaknya lingkaran sensor menyala hijau, tetapi kenyataannya tidak demikian. Backing kertas memberikan efek ini (akan ada banyak opsi untuk kustomisasi, tetapi secara default di switch hitam akan ada backing putih, dan putih - hitam). Ketika Anda menyentuh sensor, bagian kecil dari lingkaran berkedip hijau, melambangkan transmisi perintah, dan sinyal suara juga dipancarkan.


Seperti seri PB, ada saklar di bagian belakang yang memungkinkan Anda memilih satu dari empat mode operasi. Berikut adalah mode yang disarankan:

  • semua tombol adalah sakelar *
  • semua tombol dimmer *
  • tombol tengah - sakelar, kontrol sisi - kecerahan
  • pada tombol + tombol skenario + tombol mati

* - Sebelumnya, remote Nolight mengirim perintah setelah melepaskan tombol, sekarang perintah akan dikirim setelah menyentuh. Untuk seseorang, opsi ini lebih nyaman, tetapi tidak memungkinkan Anda untuk mengatur kecerahan dengan memegang jari Anda pada tombol. Untuk ini, mode kedua disediakan di mana perintah akan dikirim dua perintah (tombol ditekan dan tombol dilepaskan) antara perintah ini unit daya akan mengubah tingkat kecerahan.

Masih belum ada keputusan akhir tentang mode operasi, jadi opsi Anda dipersilakan.



Terima kasih atas perhatian anda! Saya harap berita ini membawa Anda sebanyak emosi positif seperti yang saya lakukan :)
