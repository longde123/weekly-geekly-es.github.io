Vous pouvez maintenant ajouter un bouton Skype aux boutons sociaux sur les sites



Comme vous le savez, en juin de cette année , la version Web de Skype a été lancée , qui est disponible sur web.skype.com (c'est elle qui est restée opérationnelle pendant l' automne de l'application de bureau Skype dans le monde le 21 septembre). À cet égard, Microsoft a invité aujourd'hui les webmasters à ajouter un bouton Skype à la chaîne d'icônes de boutons sociaux sur les sites Web. La fonctionnalité du nouveau bouton est qu'après avoir cliqué dessus, la version Web de Skype démarrera, vous devrez ensuite sélectionner les contacts avec lesquels je souhaite partager le lien, saisir sa description si vous le souhaitez et, en fait, envoyer le lien.

Le générateur d'extraits de code social pour Skype se trouve sur cette page . En plus des boutons standard proposés par défaut (petits, plus grands, ronds ou carrés), vous pouvez éventuellement ajouter du code CSS personnalisé au bouton. Le script lui-même est le suivant:

<script>
// Place this code in the head section of your HTML file 
(function(r, d, s) {
	r.loadSkypeWebSdkAsync = r.loadSkypeWebSdkAsync || function(p) {
		var js, sjs = d.getElementsByTagName(s)[0];
		if (d.getElementById(p.id)) { return; }
		js = d.createElement(s);
		js.id = p.id;
		js.src = p.scriptToLoad;
		js.onload = p.callback
		sjs.parentNode.insertBefore(js, sjs);
	};
	var p = {
		scriptToLoad: 'https://swx.cdn.skype.com/shared/v/latest/skypewebsdk.js',
		id: 'skype_web_sdk'
	};
	r.loadSkypeWebSdkAsync(p);
})(window, document, 'script');
</script>
