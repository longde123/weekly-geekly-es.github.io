Melalui duri ke bintang-bintang, atau jalur dari keyboard biasa ke ergonomis

Saya menyambut semua pembaca GeekTimes dan sebelumnya saya berharap semua orang mendapatkan minggu yang menyenangkan dan produktif!

Saya didorong untuk menulis posting dengan permintaan dari teman-teman saya untuk menggambarkan pengalaman saya, karena ini mungkin akan menarik bagi banyak orang. Saya langsung mengakui bahwa saya tidak mengerti apa-apa tentang jenis-jenis keyboard, saya selalu puas dengan apa yang saya miliki, tetapi begitu pandangan saya berubah, dan saya memutuskan bahwa saya ingin bekerja pada keyboard yang ergonomis. Jadi esai singkat dan perbandingan pengguna dari tiga keyboard populer sedang menunggu Anda di bawah!

Kami semua belajar untuk mencetak kapan-kapan, dan semua orang mungkin ingat bagaimana itu terjadi. Saya belajar cara mencetak sebelum saya bahkan punya komputer di rumah - entah bagaimana, sebagai hadiah ulang tahun, saya meminta orang tua saya untuk "mainan" yang pernah saya lihat di jendela toko, dan itu disebut "organizer elektronik". Ini salah satunya:


Casio Electronic Organizer


"Apa pun yang lucu anak itu," mungkin berpikir orang tua ketika mereka memuaskan keinginan anak mereka. Ini sekarang semua "keren" dan dengan perangkat seluler yang telah lama melampaui mainan seperti itu dalam hal fungsionalitas. Saya ingat bagaimana saya membayangkan itu adalah komputer saya. ;) Mungkin, geek dalam diri saya berasal dari masa-masa yang jauh, karena saya menggunakan semua fungsi yang dapat disediakan perangkat ini. Seperti yang Anda lihat, keyboard QWERTY ada di sana, jadi untuk saat digunakan, tata letak tombol pada keyboard tertunda di kepala saya dan saya menemukan dalam praktek.

Ketika saya sudah lebih tua, komputer di rumah muncul di keluarga, dan mainan ini mulai semakin jarang saya gunakan. Bahkan jika saya bisa dengan tenang mengetik sesuatu di keyboard, metode pengetikan buta masih sangat jauh. Ngomong-ngomong, karena saya rindu untuk posting tentang keyboard, saya akan menunjukkan kepada Anda keyboard mana yang dibundel dengan komputer pertama saya (saya hampir tidak menemukan gambar di Internet):


KYE Genius EasyTouch K295


Pada awalnya saya mencoba belajar mengetik, dengan bodoh mengetik ulang sesuatu di Word, lalu saya memiliki program terkenal "Solo on the keyboard", berkat itu saya belajar tentang metode pengetikan buta. Bukan untuk mengatakan bahwa saya entah bagaimana berhasil tanpa kesalahan atau entah bagaimana mudah, tapi tetap saja ... Waktu berlalu, ada semua jenis esai, laporan, makalah, yang saya cetak dengan rajin, dan tidak terlalu mengganggu. Sampai entah bagaimana teman-teman saya menemukan keyboard dengan pukulan yang sangat pendek, yang sangat saya sukai dan saya pergi ke toko dan membeli yang sama. Menurut saya, itu adalah keyboard dari beberapa pabrikan Cina, tetapi tidak mungkin menemukan sesuatu yang relevan dalam pencarian.

Tidak mungkin untuk menyebutnya nyaman dengan standar saat ini, tombolnya terlalu dekat satu sama lain, tentu saja mereka sangat pendek, apalagi, kualitas keyboard meninggalkan banyak yang diinginkan - beberapa tombol mulai ditekan dengan sangat cepat, dan saya kembali pindah ke keyboard yang lama . Lalu ada pekerjaan dan gaji pertama yang saya beli laptop di obral di Eldorado (ngomong-ngomong, masih dalam kondisi kerja):


Acer Aspire 5050


Tentu saja, ada keyboard laptop paling umum dengan stroke pendek pada waktu itu, kualitasnya memuaskan, tapi cukup mudah untuk menutup - lagi, kunci terletak sangat dekat satu sama lain. Bertahun-tahun kemudian, saya melihat keyboard aluminium Apple, dan saya sangat menyukainya sehingga saya membelinya sendiri.


Keyboard Aluminium Apple


Di keyboard ini ada celah antara tombol yang menurut saya sangat kurang! Mengetik menjadi jauh lebih sederhana dan nyaman dari sebelumnya. Ngomong-ngomong, keyboard ini mungkin adalah perangkat pertama saya dari perusahaan “buah” yang terkenal. Saya tidak berpikir untuk alasan ini, tetapi laptop saya berikutnya dari Apple. ;)

Ngomong-ngomong, saya ingin mencatat bahwa saya tidak pernah berpikir tentang ergonomi pada keyboard sama sekali, meskipun beberapa sampel juga ditemukan di bioskop - saya tidak dapat mendaftar banyak, tetapi, misalnya, bidikan dari film "Fair Play", di mana salah satu penjahat sangat marah bahwa Saya memasukkannya di keyboard untuk menghitung di mana korban mereka:


Bidikan dari film "Fair Play"


Mungkin, ketika saya melihat sampel seperti itu, saya agak berpikir bahwa mengetik pada keyboard seperti itu mungkin sangat merepotkan. Hanya baru-baru ini, ketika saya mengunjungi seorang teman dan melihat keyboard ergonomisnya, saya berpikir tentang bagaimana saya ingin membuat mengetik dalam pekerjaan saya entah bagaimana lebih nyaman, atau sesuatu ... Secara umum, google pada topik ini memberi saya banyak berbagai pilihan yang saya pertimbangkan dengan rasa ingin tahu, tetapi tidak lebih. Namun, satu keyboard dalam hal tinjauan dan penampilan (ya, ketika saya mempelajari opsinya, saya menolak sangat banyak pada prinsip "menakutkan bagaimana menggunakannya?!"), Dan itu akan menjadi keyboard ergonomis pertama dalam posting ini.

Microsoft Sculpt Ergonomic Desktop




Microsoft Sculpt Ergonomic Desktop


Kuncinya adalah perjalanan singkat, yang saya sangat terbiasa, keyboard tidak terlalu besar karena unit digital terpisah dalam kit, sandaran tangan lunak dan bentuk cembung. Tangannya berbaring dengan nyaman, dan berkat kenyataan bahwa di tengah ada celah, ada kekurangan dalam metode panggilan buta saya: ternyata, dengan jari telunjuk saya sering menekan tombol "bukan saya", huruf "Saya" sangat terpengaruh dalam hal ini (itu juga Tata letak keyboard bahasa Inggris "B") - pada awalnya ia dengan keras kepala berusaha menekannya dengan jari telunjuk tangan kanannya, yang, tentu saja, tidak membawa banyak hasil. Saya juga ingin menyebutkan bahwa beralih tombol Fx ke fungsi media diimplementasikan dalam bentuk jumper, yang menurut saya solusi yang cukup nyaman.

Dari kekurangannya, saya dapat memberitahu Anda bahwa plastik di keyboard ini sangat mudah kotor, segera berubah menjadi "jari", dan palm rest terbuat dari bahan sedemikian rupa sehingga dengan cepat kehilangan presentasi aslinya. Saya ingin mengatakan secara terpisah tentang tombol fungsi - mereka jelas berbeda dari tombol utama dalam ukuran, bahan dan goresan. Tetapi mereka tidak dapat dikacaukan dengan kunci utama. Keyboard adalah nirkabel, yang menghilangkan kabel yang tidak perlu di atas meja, tetapi secara berkala menambah kekhawatiran tentang penggantian baterai, omong-omong, dalam hal ini, ini adalah minus khusus, karena ketiga elemen berada dalam satu set (di Federasi Rusia saya hanya melihat set dengan mouse di set) catu daya dari berbagai jenis: keyboard - 2 baterai AAA, mouse - 2 baterai AA, keypad numerik - baterai lithium CR2430. Mengapa di blok digital "tablet" saya masih mengertitapi itu sebabnya mouse memiliki jenis baterai yang berbeda - tidak. Ngomong-ngomong, mouse pada awalnya menyebabkan saya menolak tampilannya, tetapi ternyata ternyata sangat nyaman dalam penggunaan sehari-hari.

Sebenarnya, keyboard melakukan misinya dengan sempurna. Saya ingin mencoba dan memahami apa itu keyboard ergonomis, dan dia memberi saya pengertian ini. Ya, Anda benar-benar dapat membuat kehidupan sehari-hari Anda sedikit lebih mudah hanya dengan menggunakan alat yang praktis. Namun, saya masih benar-benar ingin menggunakan keyboard teman itu, dan dia setuju untuk membiarkan saya menggunakannya untuk waktu yang lama. Keyboard-nya adalah Keyboard Truly Ergonomic, dan bagian selanjutnya dari artikel ini akan dikhususkan untuk itu.

Keyboard yang Benar-Benar Ergonomis



Sudah ada artikel di keyboard ini di Geektimes, di sini , misalnya, seluruh ulasan dikhususkan untuk itu. Saya hanya akan membagikan kesan saya tentang penggunaannya dan menjelaskan perasaan saya tentang transisi ke penggunaan itu. Tapi pertama-tama, izinkan saya mengingatkan Anda bagaimana tampilannya.


Keyboard yang Benar-Benar Ergonomis.


Pada saat yang sama, teman tersebut memiliki versi yang lebih hardcore dengan tombol-tombol kosong, yang tentunya memberikan daya tarik tersendiri pada keyboard ini. =)


Keyboard Benar-benar Ergonomis dengan tata letak Kosong


Kompiler keyboard ini dipandu oleh fakta bahwa tata letak tombol, pertama, telah lama usang (yang berarti bahwa pada semua keyboard standar tombol diimbangi, menyalin tata letak mesin ketik di mana diperlukan), dan kedua, oleh bahwa Anda harus menjaga tangan Anda berdekatan satu sama lain (dan bahkan meraih mouse dari waktu ke waktu).

Jujur, Anda benar-benar harus memegang tangan Anda pada keyboard ini sedikit lebih jauh daripada di belakang keyboard biasa, tetapi pada Microsoft Sculpt yang sama, pengaturan tangan secara pribadi menurut saya lebih nyaman. Tetapi sensasi baru dari kenyataan bahwa surat-surat itu terletak satu di bawah yang lain tidak dapat diungkapkan dengan kata-kata! Ya, di sini metode sepuluh jari "sesuai dengan buku teks" ternyata dengan sendirinya! Untuk sebagian besar, posisi turun untuk menggeser jari yang sesuai sedikit lebih tinggi atau lebih rendah relatif terhadap posisi "rumah", ternyata jauh lebih nyaman untuk mencetak daripada sebelumnya.

Saya benar-benar ingin berbicara tentang tombol fungsi dan tombol pengubah. Mungkin mudah untuk mengetik teks (Backspace dalam jangkauan jari telunjuk hanyalah semacam perayaan!), Tetapi ketika dihadapkan dengan kebutuhan untuk bekerja di konsol atau mengetik kode di Vim favorit Anda, kesulitan tertentu muncul. Pertama-tama, saya memperhatikan lokasi Shift dan Ctrl (yang berubah menjadi Cmd dalam tata letak untuk Mac OS) - dimungkinkan untuk terbiasa, tetapi tidak sepadan, saya pribadi mencoba membiasakan diri dengan pengaturan ini sejauh ini, saya mengelola banyak kombinasi keyboard yang berbeda masukkan bukannya mengetik sesuatu. Selain itu, tombol Tab sangat merepotkan bagi saya, yang saya tidak bisa terbiasa dengannya, dan yang Anda tidak akan bergerak di mana pun. Lagi pula, kecuali untuk Backspace,posisi tengah tombol di keyboard ini jauh dari optimal. Saya perhatikan bahwa kadang-kadang karena fakta bahwa ketika menggunakan Backspace, tangan mungkin sedikit bergeser dan pers yang mengikuti penyesuaian mungkin tidak ada. Yah, dan sebagai hasilnya, menggunakan beberapa pintasan keyboard tidak senyaman yang kita inginkan. Saya juga memiliki rasa sakit yang terpisah terkait dengan fakta bahwa saya terbiasa beralih bahasa dengan mengklik Caps Lock. Lokasi standar ada di atas, di bawah sederet tombol fungsi, dan tidak menyenangkan untuk mencapai pergantian tata letak. Anda juga dapat mengagumi lokasi tombol Fn, menggunakannya untuk membuat suaranya lebih tenang, misalnya, ia membutuhkan "melipat" satu tangan, atau menggerakkan kedua tangan ke atas. Saya juga tidak bisa menggunakan panah secara normal, dengan tangan membabi buta, di mana Anda perlu,menolak untuk memukul.

Setelah sekian lama, saya mampu dengan kenyamanan yang cukup tidak hanya untuk mengetik teks, tetapi juga untuk bekerja sepenuhnya, tetapi butuh waktu dan usaha yang cukup. Jika ada, Anda dapat mencoba menyesuaikan tata letak untuk diri sendiri, tetapi Anda tidak dapat mengesampingkan ruang kiri atau kanan untuk hal lain, sayangnya. Keyboard adalah mekanis (yang merupakan nilai tambah terpisah untuk saya), tidak memakan banyak ruang di atas meja, cukup berat (berat sekitar 2 kg) untuk berbaring di atas meja hampir "seperti sarung tangan".

Terima kasih khusus kepada teman untuk keyboard yang disediakan, tetapi saya mulai melihat secara berbeda pada pahlawan posting berikutnya. Alih-alih, saya memiliki pikiran-pikiran yang tepat yang saya nyuarakan di awal posting, ala "betapa sialnya", "siapa yang bisa memunculkan ini", dan secara umum, apakah ada yang benar-benar menggunakan ini ?! Tetapi kesimpulan yang saya buat untuk diri saya sendiri hanya berbicara tentang satu hal - Sungguh mungkin baik dalam beberapa aspek, tetapi ada pilihan yang jauh lebih baik.

Kami akan berbicara lebih banyak tentang Kinesis Advantage.

Keuntungan Kinesis



Penampilan keyboard ini benar-benar tidak dapat meninggalkan acuh tak acuh pembaca Geektimes yang tidak berpengalaman. Meskipun ini saya, ada juga posting tentang dia di mana mereka berbicara dengan cukup detail tentang dia, tetapi tidak ada kesan praktis darinya, yang akan saya coba untuk menebusnya semampu saya. Pertama, seperti biasa, gambar:


Kinesis Advantage Keyboard


Pertama-tama, jelas bahwa tombolnya diatur dalam barisan genap, dan lokasi ruang, Backspace dan Enter pada tahap kenalan visual tampak sangat nyaman bagi saya. Jujur, harapan terpenuhi - nyaman untuk tangan, dengan jari-jari Anda, Anda dapat mencapai hampir semua kunci, yang diperlukan, tanpa secara khusus mengubah posisi sikat di ruang angkasa. Keyboard memiliki beberapa mode yang berubah dengan cepat dengan kombinasi khusus; Ada tiga mode: mode PC, mode Windows dan mode Macintosh. Mereka berbeda dalam hal lokasi dan komposisi tombol pengubah di bawah ibu jari sedikit berubah.


Berbagai Mode Keuntungan Kinesis


Beberapa waktu setelah Truly, saya harus terbiasa dengan kenyataan bahwa di bawah jempol tangan kiri bukanlah ruang, tetapi Backspace. Caps Lock, Shift, dan Tab di tempat "mereka", yang tidak bisa tidak menyenangkan saya. Setelah beberapa waktu, saya membuat beberapa penugasan ulang yang membuat bekerja dengan keyboard menjadi lebih nyaman: karena saya tidak pernah menggunakan tombol Delete, dan Home and End menyalin pintasan keyboard "asli" CMD + ↑ dan CMD + ↓, perubahan berikut terjadi:


My tata letak dengan tombol yang ditugaskan kembali


Living with Option tambahan dan Ctrl dalam hal menggunakan berbagai pintasan keyboard menjadi lebih mudah, ditambah Esc "tambahan" yang menambah kenyamanan saat bekerja di Vim.

Sayangnya, tombol fungsi jauh, Anda harus meraihnya, dan itu terbuat dari retsina. Selain itu, rentang fungsional F1-F12 tidak terbagi dua, yang menambah kesulitan tertentu saat menggunakannya (setidaknya Anda harus membidik secara visual sebelum menekan tombol). Ada mode terpisah di mana tombol fungsi berubah menjadi tombol media: F3 - Sebelumnya. track, F4 - Play / Pause, F5 - Track selanjutnya, F9 - Mute, F10 - Volume turun, F11 - Volume naik, Jeda - Stop. Saya tidak bisa memikirkan mengapa menggunakan kemampuan untuk membuat makro, saya melihat di suatu tempat saran untuk memasukkan kata sandi untuk memasukkan sistem sebagai makro, tetapi bagi saya ini sepertinya omong kosong. Ada built-in "lapisan" kunci digital (seperti pada Sungguh, pada kenyataannya), panah kursor dibagi: kiri / kanan di setengah kiri,atas / bawah di sebelah kanan (butuh sedikit waktu untuk membiasakan diri).

Hasil dari



Tentu saja, keyboard ergonomis "terlalu" tidak cocok untuk semua orang, karena jauh dari selalu perlu memegang kedua tangan di atas keyboard. Di sisi lain, Anda dapat mengadaptasi dan menetapkan ulang semua kombinasi keyboard sedemikian rupa sehingga mereka dapat dengan mudah digunakan dengan tangan yang bebas mouse. Harga juga memainkan peran penting - jika Microsoft Memahat dalam satu set dengan mouse harganya $ 130, maka Truly sudah $ 249, dan untuk Kinesis Advantage $ 299 (versi Pro dengan pedal adalah $ 359), dan ini tanpa pengiriman ke Rusia! Situs web Truly dari Federasi Rusia tidak tersedia, saya belum melihatnya di mana pun di toko kami. Pernah ada banyak di eBay untuk digunakan, dengan harga yang wajar, tetapi sudah, seperti, sudah dijual. Mencari Kinesis memberi Anda hanya satu toko online yang menjual keyboard ini bersama kami. Plus Anda dapat memonitor situsKeyboardCo untuk sesuatu yang menarik, atau cari Amazon untuk sesuatu yang dapat diterima. Kami juga ingat bahwa Anda dapat memasang keyboard sendiri di bawah tangan Anda sendiri atau melihat ke arah Catboard / ErgoDox , yang Anda diundang untuk memasang sendiri. Secara pribadi, saya ingin Keuntungan saya sendiri dengan kunci bersih untuk membuat keyboard lebih "culun", dan bahkan label tidak akan terlepas dengan penugasan kembali yang akan saya atur. Bagaimanapun, sebelum saya bertemu Kinesis, saya bahkan tidak bisa membayangkan bahwa mencetak sejumlah besar teks mungkin tidak menjadi beban, tetapi menyenangkan. Dan secara umum, ini dapat dibandingkan, mungkin, hanya dengan pembelian helikopter di kontrol radio! =)

Terima kasih atas perhatiannya!
