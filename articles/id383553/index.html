GameKid - Klon Game Boy dari Raspberry Pi



Banyak pecinta elektronik dan mainan konsol lama telah lama menggunakan Raspberry Pi untuk meluncurkan berbagai jenis emulator konsol. Emulator, terutama yang universal, memungkinkan Anda untuk bermain game dari berbagai konsol, termasuk yang sangat lama dan langka. Di antara platform perangkat keras populer yang ditiru adalah Game Boy. Justru para pengrajin yang memutuskan untuk membuatnya kembali, menggunakan Raspberry Pi dan kasing yang dicetak pada printer 3D.

Gagasan untuk membuat kembali gadget retro dari komponen elektronik baru muncul dengan dua saudara tepat pada saat Natal. Setelah beberapa minggu bekerja, GameKid lahir. Untuk meluncurkan mainan di bawah Game Boy di konsol ini, saudara-saudara memutuskan untuk menggunakan lingkungan perangkat lunak RetroPie .

Kasing GameKid sangat mirip dengan pendahulunya. Ada empat tombol depan untuk mengontrol gameplay, D-Pad, tombol start dan pilih. Selain itu, ada speaker built-in dan jack headphone. Jika diinginkan, pengguna konsol game portabel seperti itu dapat menghubungkan TV melalui port HDMI. Selain itu, ada empat port USB yang memungkinkan Anda menghubungkan pengontrol game standar. Saat terhubung ke TV atau monitor, fitur ini akan sangat berguna.

Di dalam, selain Raspberry Pi 2 dengan RetroPie, ada kartu memori 16 GB, ROM dengan trek audio dan MP3. Saudara-saudara sekarang mengumpulkan dana untuk proyek ini di situs crowdfunding Kickstarter., dan jika diinginkan, Anda dapat memesan perangkat ini seharga $ 139. Pada prinsipnya, sedikit mahal, tetapi kemampuan kit jauh lebih maju daripada GameBoy biasa. Ya, kartrid tidak digunakan di sini, untuk gim yang Anda butuhkan memuat ratusan mainan ke dalam memori konsol, dan Anda dapat menikmatinya.
