Ein Tag im Leben eines Adlers in wenigen Sekunden

Derzeit gibt es im Internet viele Websites, auf denen Sie Online-Videos von CCTV-Kameras ansehen können, die sich an öffentlichen Orten bestimmter Städte befinden. Insbesondere sind in der Stadt Orel mehrere solcher Kameras installiert. Eine der Kameras, die sich in einem der Hochhäuser der Gasse befindet. Als erstes erschien ein Postdienst für den Kinderpark. Zumindest auf einer der Websites im Frühjahr 2011 habe ich eine Seite gefunden, auf der Sie Videos von dieser Kamera „live“ (online) ansehen können. Diese Sendung war jedoch sehr interessant organisiert. Video kam mit einer sehr niedrigen Bildrate von 2 Bildern pro Minute. Tatsächlich war dies überhaupt kein Video, sondern ein Bild, das jede halbe Minute automatisch aktualisiert wurde (genauer gesagt, die gesamte Seite wurde aktualisiert).

Ich hatte die Idee, eine Reihe solcher Bilder für eine lange Zeit zu speichern und dann durch Kombinieren eine Videodatei mit einer üblichen Standard-Bildrate von 25 fps zu erstellen. Wenn Sie zählen, erhalten Sie ein beschleunigtes Video 25 / (2/60) = 750 Mal. Ich beschloss, Bilder von dieser Kamera an einem Feiertag am 9. Mai aufzunehmen, als die maximale Konzentration von Menschen vorherrschte.

Dafür habe ich folgende Programme verwendet.

  1. Opera - ein Browser zum Anzeigen der Sendung.
  2. HandyCache ist ein Proxyserver zum Zwischenspeichern von Bildern.
  3. Total Commander - zum Umbenennen von zwischengespeicherten Dateien in Gruppen nach Seriennummer.
  4. TimeToPhoto - zum Überlappen von Text und Datum / Uhrzeit des Frames mit dem Änderungsdatum der Datei für die entsprechenden Bilder auf Bildern.
  5. Ulead GIF Animator - um das endgültige Video aus den empfangenen Bildern zu erstellen.

Gleichzeitig machte ich einen Screenshot meines bisher erhaltenen Desktops, auf dem die letzten beiden Programme im Vollbildmodus bereitgestellt wurden.
