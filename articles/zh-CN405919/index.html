GeoWorks的故事，90年代的新贵和竞争对手Microsoft Windows


准将64

在90年代初期，仍然无法保证Microsoft Windows会占领市场，尽管多亏了MS-DOS，它们在许多竞争者中都具有明显的优势。

一段时间以来，美国在线（AOL）的一个邪教图形用户界面（GUI）试图将所有的东西都放在一个篮子里，并使用很少有人知道的图形用户界面平台（GeoWorks）支持其伪互联网软件的DOS版本。

从调制解调器似乎不是强制性的时代开始，这是一个操作系统。 得益于其祖先，即Commodore 64操作系统GEOS（图形环境操作系统），它的重量非常轻。

该平台是由Berkeley Softworks制作的（不要与Berkeley Systems，著名的“ flying toasters ”屏幕保护程序的作者混淆）。 一段时间以来，由于其功能和在低成本设备上工作的能力，其产品成为Commodore 64最受欢迎的程序之一。

“ GEOS并不是第一个GUI系统。 Crock Kamen在2006年为《 OS News 》撰写的文章中说 ，它的大部分功能已经在当时的大型OS中使用，例如经典Mac（尽管不是Windows）。 “ GEOS吹嘘的是在低成本和低功耗硬件上的办公应用程序的工作。 您不需要2000美元的计算机就可以写和打印一个简单的字母。”



最终，该操作系统以其更高级的形式在90年代初爬到PC上，伯克利软件公司更名为GeoWorks。

感谢我的一个儿时朋友，让我玩他的Commodore 64，我有了一些相关的经验。 我使用的是适用于PC的GeoWorks版本，因为它是与386th捆绑在一起的，那是我小时候买的。

这台计算机不是很快-它有40 MB的硬盘驱动器和1兆字节的内存-因此，他很幸运能够在其上安装来自GeoWorks的轻型面向对象的系统。 操作系统的重量不超过10 MB。 在与世界联系尚未普及的时代，格式的简单性发挥了重要作用。

在该平台有趣的功能中，人们可以回忆一下以下内容。

不同级别的用户可以使用不同的界面 。 DOS对于初学者来说并不是一个简单的操作系统，因此GeoWorks Ensemble试图使其变得更友好。 她提供了两种类型的界面：“应用程序”和“专业”界面，能够运行DOS程序的外壳程序，因此您可以毫无问题地玩Commander Keen。 对于从未接触过PC的人来说，这种策略是理想的-系统具有“训练轮”，就像儿童自行车一样。

内置办公应用程序 。 该程序包括许多与其他操作系统大致对应的应用程序-例如，Mac可能拥有文字处理器，日历和电子表格。 在OS中，还有一个用于绘制页眉Print Shop的程序，如果您拥有点矩阵打印机，该程序可能会派上用场。 通常，此优惠非常适合家庭用户，Microsoft在活动开始时并未特别注意。 该程序可能不如Microsoft Bob出色，但效果更好。


GeoWorks主页

特点严重，要求低 。 GeoWorks的最大特点是运行良好，无需过多的硬件需求。 为了使Windows 3.1正常运行，它需要486th，但是GeoWorks可以自由运行，并且在286和386上没有问题。它是稳定的，尽管事实上，与早期版本的Windows一样，它只是DOS的图形外壳，很少有她的问题。

该计划已吸引了众多支持者，尤其是德国用户，他们为维护她的记忆做出了很多努力。

另一家公司提供了对Commodore 64的早期访问权的Quantum Computer Services，使GeoWorks进入了PC市场，并启动了其首个用于IBM PS / 1计算机的在线网络。

该公司执行副总裁史蒂夫·凯斯（Steve Case）在1990年的新闻稿中写道：“ Promenade界面使所有家庭成员都可以轻松使用服务，而不必因使用复杂的命令和功能而烦恼。” “与此同时，该程序的高级程度足以满足经验丰富的在线服务用户。”

一年之内，该平台在America Online上进行了重新设计，Case公司经历了90年代，并且在十年之内，该公司将与Time Warner进行鲁merger的合并 -尽管AOL是Windows时代定义性程序之一。



GeoWorks在被认为很酷之前就拥有了自己的AOL-这是占领PC市场的绝佳机会，特别是因为首批AOL光盘包含GeoWorks的基本版本。 这使调制解调器所有者可以免费试用GeoWorks。

但这还不够。 在AOL之外，GeoWorks很少有第三方应用程序。 部分原因是您在早期需要一个Sun工作站来开发用于该平台的软件。 考虑到要开发用于廉价PC的软件，需要一台价格在7,000美元以下的计算机，这具有讽刺意味。 家族企业不可能进入这个市场。

同时，Microsoft发布了Windows的本地开发平台，例如Visual Basic，并获得了小型开发人员的认可。

但是，如果台式机操作系统的受众足够多，则可以解决这些限制。 甚至臭名昭著的GeoWorks粉丝也意识到，由于Microsoft的热情，她没有机会与Windows对抗。

一位这样的粉丝在1991年在《 PC Magazine》上写道： “我非常不高兴，因为没有人给这个真正令人赞叹的程序一个机会，IBM和Microsoft不允许开发它。” “我希望软件开发人员能够看到Ensemble的惊人潜力，并开始进行开发。 没有第三方开发人员，Ensemble无法生存。”

微软站在巨人的肩膀上，而GeoWorks却没有碰到脚踝。


带有Intel 386处理器的旧IBM PS / 2

但是，尽管GeoWorks无法赢得喜欢Windows的用户的青睐，但该操作系统尚未完全消失。 操作系统即使没有起飞，也往往能过很多日子。 它们突然出现在这里和那里，因为在某些情况下该软件仍然有用。

例如，webOS（一种用于Palm的操作系统）早已被放弃，现在可以控制 LG的智能电视。

GEOS的命运以类似的方式演变。 就像一头经过食品加工过程的母牛一样，它被分成一百万片，GEOS的某些部分出现在各种奇怪产品的成分列表中。 在可见GEOS骨骼的地方中，您可以命名：

数字助理。 在Palm Computing的创始人Jeff Hawkins发明PalmPilot之前，他首次尝试使用精简版的GEOS构建平台。 Tandy Zoomer于1993年发行，并没有引起轰动，但与GeoWorks，Tandy和Casio共同为Hawkins及其团队提供了有益的信息。 几年后，这为第一款真正成功的PDA 奠定了基础 -但他不再使用GEOS。

早期的智能手机。 GEOS在移动革命中的作用不仅限于Palm。 在90年代后期，该操作系统是Nokia 9000 Communicator的关键部分， Nokia 9000 Communicator是最早获得好评的智能手机之一。 他具有最简单的文本处理能力，具有图形浏览器，甚至可以编辑电子表格。 它并不便宜，一开始的价格是800美元，而且尺寸还很大。 “今天的用户将移动电子邮件和网络浏览作为日常工作，但诺基亚9000 Communicator是首款将所有功能整合在一起的设备，”技术创新评论的作者Richard Baguli在2013年写道 。 “它可能是一个笨重的设备，但我们仍然想念它。”


诺基亚9000 Communicator

电子打字机 。 在90年代，打字机制造商苦苦挣扎，而Brother没有为互联网革命做准备。 但是她有些袖手旁观：GEOS。 该公司已与 GeoWorks 合作 ，生产具有简单文本编辑和桌面发布等功能的打印机选件。 无论如何都是打字机，但是他们做的不仅仅是写文字，还有很多有趣的事情。

原始上网本 。 兄弟对GEOS的兴趣不仅限于打字机。 她在GEOS中看到了将“计算机带给大众”的机会，正如在新闻稿中所写的那样。 1998年，当GEOS几乎忘记了一切时，这家打字机公司推出了一个替代平台-GeoBook，价格为500美元，这是一款低功耗笔记本电脑，比上网本提前了十年。 可以使用DOS版本的GeoWorks使用浏览器和许多程序，但是它没有硬盘，这就是为什么可以将成本保持在较低水平的原因。 而且，就像上网本一样，测量师也讨厌这种设备。 “以这种设备的价格，您可以轻松地购买翻新的或二手的Windows计算机，甚至可能是一台新计算机。 1998年《纽约时报》的负面评论解释说，他将有能力做这辆车只能梦想的数百种事情。

今天，据我所知，在GEOS上如此疯狂的项目并不存在，但也许它可以在ATM上运行。

但是，尽管GeoWorks拥有所有多余的生命，但今天的前景比以往任何时候都要糟糕。 特别是，这是由于与GEOS相关的复杂的公司历史。 在90年代创建该程序的公司破产之后，该技术被出售给了NewDeal，后者基于GEOS构建了一套办公程序，该程序看起来非常类似于 Windows 95，它具有该平台的大部分独特魅力。

在某个时候，该操作系统属于Ted Turner的儿子，他正试图成立一家基于GeoWorks软件的低成本PC制造商MyTurn.com。 在2013年，特德·特纳（Ted Turner）试图竞选国会议员时，他在这家公司的工作对他不利 。



结果，该操作系统落入了Breadbox之手，该公司将GeoWorks视为一个志愿者项目，并试图将GEOS变成可与Android 结合使用的教育平台。

但是最近，面包箱已冻结活动。 2015年11月，公司创始人弗兰克·费舍尔（Frank Fisher）意外死亡，当时该公司正在为平板电脑开发各种变体。

他与Breadbox的长期合作伙伴John Howard目前正在研究下一步，并与Fisher家族以及对该平台感兴趣的其他开发人员进行了交谈。

他在2016 年在 Breadbox 网站上写道： “仍有待解决其他一些法律问题，但我敢肯定，GEOS代码中的生活仍然在变暖。”
