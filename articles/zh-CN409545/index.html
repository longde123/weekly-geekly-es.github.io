ImageRUNNER C3025系列：彩色多功能一体机，价格为黑白

SO / HO细分市场（小型办公室/家庭办公室）的用户可能会对新型彩色激光多功能一体机imageRUNNER C3025感兴趣，这不仅是因为其紧凑的尺寸和价格范围内令人印象深刻的功能。 它们的成本可与单色类似物的成本相比。 顺便说一下，这是我们的第一个紧凑型平台，结合了软件和硬件开发，可确保打印时的数据保护。 该系列由两个型号C3025和C3025i代表。 这些设备在彩色和黑白模式下为办公室MFP执行一组标准操作：
  • 正在扫描
  • 复制中
  • 激光印刷。



传真卡的可选安装。 让我们更详细地查看每个函数。

扫瞄


两种型号在所有方面都具有相同的性能。 它们之间的区别仅在于配置。 可以说C3025是基本版本，而C3025i是更高级的版本。 对于C3025i，该套件包括一个用于扫描的双向自动文件进纸器（RADF-可逆自动文件进纸器）。 也可以选择将RADF放在C3025型号上，在该型号的基本配置中，代替进纸器，安装了平板扫描仪盖。 扫描速度与进纸器的性能相对应，为每分钟55张。 扫描分辨率有两种模式：204x196和300x300 dpi。 扫描媒体的最大格式为A3。


同时提供推和拉扫描。

MF Scan Utility程序已预先安装在扫描软件包中。 借助其帮助，可以将扫描结果保存为TIFF，JPEG和PDF格式（均压缩并具有搜索功能，这在处理文本文档时非常方便）。 根据设置，可以将扫描的文件发送到电子邮件，网络文件夹，云或USB驱动器。 此外，您可以使用设备本身扫描到网络文件夹，USB驱动器，邮件和其他资源，而无需使用计算机和特殊软件。

自动文档进纸器允许双面扫描文档。 在C3025i型号中，此功能包含在基本包装中，而在C3025型号中，此功能是可选的。 在最初购买imageRUNNER C3025的基本设备，然后根据需要配备其他选项和功能（直至C3025i的完整版本）时，此类模块对于小型企业将非常有用。

在确定用于放置MFP的工作场所的大小时，有必要考虑是否存在自动送纸装置。 因此，有必要不考虑投影到水平面上的设备的整体尺寸，而是要考虑MFP正常运行所需区域的数据。 C3025的基本配置需要978 x 1088毫米，C3025i的最大配置需要1110 x 1117毫米。

复制


对于A4格式，黑白和彩色模式下的复印速度均为25 ppm，对于A3-15 ppm。 复印时，该设备可提供良好的分辨率：最高600 x 600 dpi。 ImageRUNNER C3025和C3025i设备具有内置的缩放功能，其最受欢迎的比例为25％，50％，71％，100％，141％，200％和400％。 也可以在手动模式下以1％的增量将比例从25％更改为400％。 设置允许每个周期最多999份。

列印


该设备具有许多有用的打印功能。 它们的设计既方便又保护数据。

  • 关键打印功能
    imageRUNNER MFP使用V2（生动和充满活力）技术进行彩色打印，可提供600 x 600或1200 x 1200分辨率的高质量打印。 A4文档的打印速度高达25 ppm，A3-15 ppm。
  • 国际金融机构办公室
    所有设备功能和设置均通过5英寸对角触摸屏控制。 您可以使用手势在屏幕窗口之间移动并选择所需的动作。 此外，此屏幕上还会显示有关偏离正常操作模式的系统消息以及参考信息（如果出现错误）以及建议。

  • 打印文件的来源
    通过有线和无线网络组织用于打印的文件传输。 使用Wi-Fi，您可以将移动设备直接连接到MFP，而无需连接到公司的本地网络。 为此，使用了佳能PRINT Business应用程序，该应用程序可与基于Android和IOS的设备一起使用。 预先安装的MFP软件与AirPrint（Mac OS X或iOS）和Mopria（Android和Windows 10移动版）技术兼容。 还提供从USB载体打印的功能。

  • 服务专区
    遵循“时间就是金钱”的原则，我们的开发人员致力于减少维修MFP的停机时间。 在imageRUNNER C3025 / i中，诊断可以在后台进行，而不会影响设备的主要操作。 远程设备诊断和碳粉消耗控制可实时监控设备的当前状态。 预先通知需要进行日常维护，可以在MFP的最小负载期间计划其实施。

    该设备具有良好的可维护性。 允许用户独立执行某些节点的更换，而无需借助专家的帮助并且不损失保修。 特别是针对这些情况，我们制作了一系列培训视频，有助于解决工作中出现的典型问题。

    imageRUNNER C3025 / i具有便捷的GDLS功能，该功能负责更新固件并将其保持在当前版本中。
  • 安全性
    imageRUNNER C3025 / i MFP实施了多阶段安全系统，包括硬件和软件。 开始时已经遇到了第一个障碍。 要启动设备，您必须输入PIN码。 下一步是使用带有读卡器的访问卡授权用户。 为此，ImageRUNNER C3025 / i MFP安装了通用登录管理器（ULM）软件。 顺便说一下，正是我们首先在这些模型上应用了ULM。

    用户卡不仅可以区分对设备各种功能的访问级别，还可以保留有关MFP使用情况的统计信息。 还借助ULM mono应用程序来组织受控的安全打印。 这有助于优化打印成本，减少打印机误用的可能性，并个性化对打印文档的责任。

    可以将总打印成本的比率导出为所有已打印纸张与目标的比率。 如果员工将打印机用于个人目的，则目标打印的成本最终可能会增加很多倍。 更为通用的解决方案是uniFLOW系统，该系统与imageRUNNER C3025 / i MFP集成良好。 该系统的功能之一是收集每个用户或组使用打印机的统计信息。 另外，如果系统中包括多台打印机，则在打印打印作业时，uniFLOW在屏幕上的每台打印机上显示打印成本。 可以配置程序，以便将文档发送到最经济的打印机进行打印。 在上一篇文章中，我们写了更多有关uniFLOW系统的文章。
  • 整理打印文件
    可选地，C3025和C3025i配备了内部装订器，其容量为600张纸，可以执行三种功能。 它用作装订器，用于按副本或按页面顺序对文档的打印纸进行分组或拆卸。 装订在角落，两个地方或根据要求进行。 可以在不装订的情况下连接纸张（生态装订）。

    另一种版本的200页内部装订器可以用作分页器，对打印的页面进行分组和分离，而无需装订功能。 在过程中经常使用各种功能的情况下，例如，复印，打印，传真，纸流分离是必要的。 为此，将装订器用作可选的输出托盘。

附加设备


在基本版本中，C3025和C3025i配备了一个标准纸盘，该纸盘包括两个550页进纸器和一个100页通用纸盘，用于80 g / m2的纸张。 （可选）您可以购买用于两个550页进纸器的可选纸盘或不带进纸功能的基座。
imageRUNNER C3025 / i MFP的纸张尺寸最大为SRA3（包括SRA3），重量为52至300 g / m2。 首字母缩写词SRA3代表“可选的原始A3格式”。 在打印后修剪纸张以完成A3尺寸的打印件时使用。

我们开发了iR C3025系列，主要用于专注于保持数字格式工作流程的小型企业。 在性能方面，该产品线在运营效率和安全性方面接近于公司模型。 它可以轻松应对每月最多8,000张的负载。 而且，我们认为iR C3025系列不仅在小型企业领域有需求，而且也会引起大型企业的兴趣。
