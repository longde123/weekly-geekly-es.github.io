EIZO Foris FS2434无框显示器评测

如果需要显示器怎么办？没错，首先确定您为什么需要它，然后上网阅读评论。是否要处理图形和照片？显色性很重要。玩玩具和看电影？响应速度和输入滞后是优先事项。好吧，当一台家用电脑既可用于工作又可用于娱乐时，并且通常每天工作24小时，我想保持某种平衡和图像质量以及动态特性，以使我的眼睛在明亮的晴天或黑暗中不会流血冬天的夜晚。重点不大-选择一款显示器，其特性可以完美地涵盖这些用户情况，并且不会对钱包造成重创。



有了这个功能，EIZO Foris FS2434是最有趣的显示器之一 （与游戏系列正式相关），我们今天将讨论。

如果您不熟悉艺卓，并且下意识地认为它是一家中国的一日游公司，那么您就白费了。也许这家公司实际上在俄罗斯联邦并不那么受欢迎，但是，它是一家有着悠久历史的认真的日本制造商：该公司在不久前（十几年前）获得了现名，但它成立于1968年，并且一直专注于显示器的生产。知道日本人对任何价格类别的完美产品的热爱，将所有果汁从可用技术中挤出来，并专注于真正重要的事情，很难相信这款显示器可能出了问题。让我们结识。

出现


监视器应显示图像，其他所有方法都只能达到此目的。因此，这里的所有设计元素都比设计成使显示器成为独立艺术品的功能性更大。也就是说，没有任何令人眼花bend乱的支架弯曲，铝拉丝插件，装饰性扬声器网，大师亲自钻出的孔眼，这比您在整个日本都找不到的更好....所有东西严格按照工业风格制造：清晰的边缘，实用的塑料，略带禁忌的设计。以我的理解，这正是“主力马”的样子，它是为工作而不是为了美而购买的。



令人难以置信的是，我喜欢黑色磨砂屏下方的红色条带-如果没有它，监视器看起来会很无聊，但与此完全不同。联想设备上的红色插入物，汽车黑色内饰中的红色灯都处处可见这种组合的优势。



设备控制按钮和状态LED位于显示器的正中央。数量不多，只有4个：信号（选择视频源），两个音量控制按钮和电源。问在哪里可以进入系统菜单的按钮？您会感到惊讶，但事实并非如此-只能通过遥控器访问菜单。  



前面板的边缘是隐藏的内置立体声扬声器栅格（每个0.5W），右侧是耳机插孔（微型插孔3.5mm）：



功能元素列表不止于此。在同一侧（比面板本身稍远）上有一个带有其余连接器的壁架：两个USB端口（3.0），一个USB输入（当然也是标准3.0）B型，另外两个音频连接器（也为迷你插孔）：输出和输入。



以下是两个HDMI端口，一个DVI-D，一个电源连接器和一个开关。





背面有一个公司徽标，一个运输手柄（这是另一个装饰元素）以及VESA安装孔（100x100毫米）和防盗锁。



框住面板活动部分的超薄框架为该设备增添了未来感，其厚度仅为2毫米，就像10卢布的硬币一样。



但是……不要急于欣喜-完全不涉及屏幕的另外4毫米（沿整个周边），因此“框架”的总厚度为6毫米，但是，它也比其他设备要小得多。



面板本身是哑光的，很难在上面打上烙印，但是要擦除它很容易。



支架呈略呈桶形的形状，支腿可让您升高和降低显示器（幅度为60毫米），倾斜显示器（跨度为25度）并在水平面内旋转屏幕（344度）：



这两个组件均由磨砂粗糙的塑料制成，后面也很容易照顾。可以选择将一个红色环插入支架，以进行精确的电缆管理。



显示器带有一个紧凑的遥控器，带有许多按钮，可让您调节从亮度到图像质量的所有内容。遥控器是好的，只有您可以随时丢失它，但是按钮不会重复。不知何故...不是日语。


包容与工作


设备的对角线为23.8英寸（60.4厘米），分辨率为1920x1080（16：9）。像素大小为0.275 mm 2。该矩阵是通过AH-IPS技术形成的，其最大亮度为250 cd / m 2，对比度为1000：1，响应GtG略小于5毫秒。

超薄边框意味着使用多显示器配置。并非所有游戏都可以在三个屏幕上使用，并且5760x1080的分辨率会显着影响性能，但是具有两个或三个显示器的系统不仅看起来很棒，而且可以“沉浸”在工作任务或游戏中。的确，为了在“高”设置下以这种分辨率玩游戏，您需要一块优质的显卡-最低配备NVIDIA GTX970或AMD R9-280（或者更好的是R9-280X），因为这样的分辨率会给RoP带来很高的负载，可能需要高达3 GB的视频内存。



显示器有许多“增强器”，其中值得注意的是G-Ignition实用程序（8Mb，Windows），它允许您创建各种颜色配置文件并将其分配给应用程序。这在某些应用程序中可能有用，但是这里主要的锐化是针对游戏的-无需进入显示器本身的控制菜单，您就可以更改颜色再现（例如，使某些助步器中的黑暗区域变亮），并且在游戏结束后，将设置恢复为“正常”州。在gaming.eizo.com上，您可以下载各种游戏的现成预设。

此外，显示器还具有硬件升频器，如果信号源的分辨率与矩阵的分辨率不匹配，则可以使您以与电视相同的方式拉伸三维图像（例如，对于控制台）。并不是说该功能是非常必要的，但是如果您将显示器与笔记本电脑结合使用，并且游戏中诚实的FullHD视频系统表现一般，那么这种芯片就可以提供帮助。

此外，制造商声称输入延迟为零（更精确地说为0.05c）-图像进入显示器并在屏幕上显示之间的时间。为此，重新设计了控制板和无闪烁的背光，即在任何亮度水平下都没有PWM（或没有明显的PWM影响）地工作。

菜单和图像质量


对于大多数测试人员和用户而言，监视器可以“开箱即用”地显示精美的图像，这是个好消息。在视觉上和主观上，我也非常喜欢这张照片，但是，校准器的读数并不完美。所要求保护的sRGB涂层有些爬行到黄红色光谱中，并被蓝绿色“切除”，但是可以通过校准部分纠正。



白色对于现代LED显示器来说是非常典型的-测量值在250 cd / m 2的范围内，即存在很小的余量，在所有条件下都能舒适地工作。

黑色的亮度原来很奇怪。首先，它的测量值超过1 cd / m 2，导致对比度低。为了在180 cd / m 2 SC 1的范围内获得120的最佳亮度值。起初，我认为这是“ uluchshayzerov”之一的结果，该亮度使游戏中的黑暗区域变亮了……但随后将所有值均降低为默认值，并产生了显着效果没有改变。







使用默认设置时，显示器具有较高的伽玛值和温度，但是可以通过屏幕菜单轻松灵活地调整这些值。将伽玛设置为2.2，温度设置为6500K，我们获得了接近参考值的值。图像仍然非常悦目。



尽管监视器具有哑光效果，但一般而言，图像的饱和度和丰富度没有问题。



在国外印刷机中，注意到了良好的显色特性（尤其是对于声明为游戏机的显示器），因此对于那些欣赏游戏和良好显色性的人来说，此实例可以称为良好的通用解决方案。处理照片是一种乐趣。

如上所述，只能通过遥控器访问屏幕菜单。菜单本身组织起来非常方便，您可以进行很多配置。



-有几种颜色模式：游戏（darkScene），游戏（灯光场景），电影院，Zeb / sRGB，纸张（在这种情况下，图像与扩展名F.lux相同）和两个用户预设User1和User2。
-亮度，黑电平，对比度，色调和饱和度（两个参数均可在-50至+50的范围内调节）。



-色温（4000、4500、5000、5500、6000、6500、7000、7500、8000、8500、9000、9300、9500和10000K）
-伽玛（电源，RTS，1.8、2.0、2.2、2.4、2.6、2.8、2.8 ）



-增益（用于R，G和B通道）和降噪
-对比度增强器（关闭/标准/增强）
-超速驱动（关闭/标准/增强）

遥控器上还有一个“智能”按钮，可用于访问快速用户功能配置文件（用户1和用户2）-这包括SmartDetection，Smart Resolution，肤色增强器，文本增强器，SmartInsight等。但老实说，没有这些bun头，监视器显示得很好。



从显示器本身的设置-您可以更改语言（有几种语言可供选择，没有俄语），以及罕见的参数，例如电源指示灯（开-关电源指示灯），徽标（开-关徽标）和按键锁定（开-关按键锁定）在前面板上；电源按钮除外）。

总结一下


显示器绝对不错。开箱即用的显色不会引起普通用户的抱怨，但是即使对于最反复无常的用户，校准也可以解决瑕疵。低输入延迟和良好的响应速度将使您即使在最动态的游戏中也不会感到丝毫的不便，出色的视角和广泛的亮度调节（与自动背光调节传感器配合使用）将使您无论在白天还是黑夜中都可以欣赏到图片。最后，它只是一台带有哑光矩阵的实用显示器-一样，日本人可以根据自己的意愿做事。
