WASI: API untuk WebAssembly non-Web


laporan akan dibaca dalam bahasa Inggris melalui teleconference

Untuk membantu mengambil WebAssembly di luar browser, kami sedang membangun WASI, Antarmuka Sistem WebAssembly. Ini adalah API standar untuk aplikasi WebAssembly untuk digunakan di luar Web, yang memperluas portabilitas dan filosofi sandboxing WebAssembly ke dalam ruang API. Saya akan berbicara tentang bagaimana kita sampai di sini, apa yang dapat Anda lakukan dengan hari ini, dan ke mana kita pergi!
