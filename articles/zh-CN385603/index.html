TP-Link Archer C2：面向大众的Wi-Fi AC

一年前，支持802.11ac的设备属于家庭网络设备制造商的高级级别，由于价格高昂，因此并非所有人都可以使用。但是今天，您可以购买具有两个频率范围内成熟功能且具有AC支持的完全廉价的路由器。 TP-Link在俄罗斯网络设备市场上稳固地占据了领先地位，推出了新的家用路由器产品线，其中包括四个位置。



今天将对其中一个路由器进行回顾，我们将不讨论顶级路由器，而是稳固的中农“黄金平均价”。我们迎接今天的英雄：TP-Link Archer C2。

Wi-Fi无线网络技术已在世界范围内稳固地获得了普及。很难想象，但是从字面上看，十二年前，Wi-Fi是笔记本电脑上的可选选项。现在，Wi-Fi直接与Internet关联，制造商正在将无线网络支持集成到几乎所有具有至少“大脑”的家用电器中，从笔记本电脑开始，到冰箱。



仅通过扫描空气就可以在一个公寓大楼中捕获多达数十个接入点，这种情况并非唯一。同时工作的设备数量在增加，每个人都需要最低可接受的网络访问速度，但是Wi-Fi可以工作的2.4 GHz频段的宽度与该技术的最初版本相比并没有改变。随着技术的发展（从802.11b的第一个公开版本到最新的802.11n），开发人员将最大速度挤出了范围：从11 Mbit / s到300 Mbit / s，但是该技术的速度并不能决定网络上的实际工作速度，因为无线电广播实际上是“常见的”环境”，并且在接入点的覆盖半径内同时运行的设备越多，它们之间的干扰就越多。

在价格较高的路由器和接入点中，制造商除了使用2.4 GHz频段外，还开始使用“故障”较少但较自由的5 GHz频段。也就是说，实际上，将两个射频路径嵌入到一个设备中。这个想法非常成功，以至于新版本的Wi-Fi 802.11ac在两个频段上都能充分发挥作用，使您可以达到每秒千兆以上的速度！

弓箭手C2装在标准的绿色硬纸板箱中，除了路由器的简要说明外，还提供一个比较表作为设备系列的一部分。



线路中的所有设备都支持双频操作，并支持IEEE 802.11a / b / g / n / ac标准。据制造商说，除较年轻的型号外，其他所有型号均具有千兆位端口，并且在性能上适合于多高清视频传输。



所有设备都有板载USB端口，但数量不同，具体取决于型号。各个模型之间的主要区别在于最大可能的数据传输速率。



TP-Link Archer C2路由器的外壳由黑色塑料制成，占用的空间很小，并且设计精美。顶部面板有光泽，并具有“破损”的表面，可有效区分设备与不露面的灰色框。绿色状态LED位于顶部面板的前面。



我们必须对以下事实表示敬意：该设计未使用当前流行的蓝色LED，而躺下的设备几乎看不见用过的绿色LED-路由器很容易放在卧室：不会干扰睡眠。





所有通信端口和按钮都位于设备背面：侧面有两个用于连接天线的RP-SMA螺钉连接器，组合的WPS和重置按钮，一个无线断开按钮，五个千兆RJ45端口（一个WAN和四个LAN），一个USB 2.0端口，一个电源按钮和交流适配器的连接器。







router刨机的底部和侧面几乎完全被通风孔覆盖。在工厂标签上，除了标准信息之外，还默认显示了两个范围的无线网络名称和用于访问网络的标准密码。

管理面板


路由器通过Web界面进行控制，简单而又逻辑，没有不必要的细节。首次连接到网络时，快速设置向导会自动打开，您可以在其中分几步配置Internet连接。同时，固件具有适用于不同电信运营商的大量模板。您只需要选择合适的一个，输入必要的授权数据即可，就是这样-Internet已在家里配置好了。与提供商的连接既可以直接通过DHCP获得IP地址，也可以手动分配，也可以使用隧道协议PPPoE，PPTP，L2TP进行。此外，路由器还完全支持IPv6网络中的寻址。



要保留与网络的连接，可以将USB调制解调器用于3G / 4G网络。如果主连接不可用，路由器将自动切换到该路由器。此外，固件中还有用于连接所有主要移动运营商的模板，因此通过USB调制解调器建立连接不会花费很多时间。在“夏季别墅”的情况下，您可以使通过调制解调器的访问成为主要的访问方式。



除了Internet访问之外，大多数现代提供商还提供其他服务，其中最受欢迎的是数字电视（IPTV）。在大多数情况下，此服务是使用数字电视机顶盒（机顶盒，STB）交付的，提供商将其提供给合同。通常，由于技术的特殊性，仅将机顶盒连接到路由器的空闲LAN端口是不够的-将需要其他配置。路由器支持各种兼容模式：既可以使用普通的IGMP代理，也可以将LAN端口“分开”以与WAN端口以桥接模式工作，以及与802.1q VLAN一起工作。如果使用VLAN，则最多可以分配三个单独的VlanID：用于Internet，电视和电话。



无线网络分别针对2.4 GHz和5 GHz频段进行配置，您可以为它们输入相同的访问参数以自动确定客户端使用的范围，或者完全不同以在范围内显式分配流量。在菜单的相应部分，您可以查看每个范围的关联（连接）无线客户端列表。对于那些没有足够标准设置的用户，在单独的选项卡上有无线接口的高级设置。完全支持所有现代访问控制方法，包括“家庭” WPA / WPA2-PSK以及通过RADIUS进行授权，这在企业部门中是必需的。



如果需要，可以通过选择与主家庭网络的隔离度来配置具有单独访问设置的其他来宾Wi-Fi网络。在这种情况下，您可以使用单独的密码（但不与本地网络隔离）创建单独的网络名称，也可以使用完全隔离的封闭网络或使用无密码连接打开网络。 （可选）您可以限制访客网络的带宽。



路由器管理面板的几个部分专门用于防御网络威胁，此外，您还可以配置“家长控制”，根据各种条件限制带宽以及配置计划的访问。可以这么说，一个很好的“绅士装”。



除调制解调器外，内置USB 2.0端口还支持使用打印机和外部存储设备：闪存驱动器和硬盘驱动器。路由器实现DLNA媒体服务器，并通过FTP访问媒体上的数据。如有必要，可以在路由器的基础上组织原始NAS。
