Neutralinojs - Alternatif Elektron Yang Mengkonsumsi Memori Lebih Sedikit

Neutralinojs adalah kerangka kerja open source untuk membuat aplikasi HTML / CSS / JS lintas platform. Perbedaannya dari Electron adalah Anda tidak perlu menginstal nodejs dan ratusan dependensi yang berjalan dari package.json untuk menjalankan aplikasi pada Neutralinojs. Mari kita bandingkan aplikasi Hello World di Neutralinojs dan Electron. Aplikasi di Neutralinojs dapat dijalankan di Windows atau Linux (MacOS saat ini tidak didukung, tetapi direncanakan untuk memperbaikinya dalam waktu dekat). Saya berlari di Ubuntu 17.10.


Halo Dunia di Neutralinojs


  1. Unduh arsip neutralinojs-v1.1.0.zip (dukungan untuk meluncurkan sepenuhnya dari baris perintah dalam rencana pengembangan) dan bongkar.


  2. Kami meluncurkan: 


    ./neutralino

    Di Windows: 


     neutralino.exe

    Aplikasi dimulai dan terlihat seperti ini:



RAM: tiga proses dibuat (neutralino, WebKitNetworkProcess, WebKitWebProcess) - totalnya mencapai 35 mb.
Volume folder dengan file pada disk: 5 mb.


Halo Dunia di Elektron


  1. Kami mengkloning repositori dengan file dan pergi ke folder yang dibuat: 


     git clone https://github.com/electron/electron-quick-start cd electron-quick-start

    Volume file yang dikloning: 344 kb.


  2. Kami kencangkan paket yang hilang: 


     npm install

    Folder node_modules muncul, volume file dalam folder aplikasi menjadi: 218 mb.


  3. Luncurkan aplikasi: 


     npm start

    Aplikasi dimulai dan terlihat seperti ini:



RAM: Saya menghitung enam proses (mulai elektron cepat, tiga proses elektron, simpul, npm) - totalnya menempati 83 mb.


Ringkasan


Aplikasi Sibuk di disk (mb) Memori utama yang digunakan (mb)
Halo Dunia di Neutralinojs 5 35
Halo Dunia di Elektron 218 83

Perbedaan dalam konsumsi memori, seperti yang mereka katakan, di wajah.
