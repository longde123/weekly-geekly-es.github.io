Bagaimana saya memilih pengontrol untuk sistem Smart Home

Pertanyaan pilihan muncul tidak hanya untuk pemula di bidang otomatisasi bangunan, tetapi juga untuk integrator dengan pengalaman yang luas. Masalah pilihan terkait dengan anggaran proyek yang tidak mencukupi, kurangnya protokol integrasi untuk pengontrol dengan peralatan yang sudah tersedia di fasilitas sesuai dengan keinginan pelanggan.

Ketika saya mulai mencari pengontrol, saya pertama kali menemui masalah dalam memilih karakteristik teknis - Saya tidak dapat menemukan tabel perbandingan pengontrol untuk otomatisasi. Karena itu, saya memutuskan untuk membuatnya sendiri.

Sebagai perbandingan, saya memilih produsen pengendali yang memiliki:

1) Pengalaman yang baik dalam mengembangkan solusi untuk otomatisasi dan manajemen peralatan;
2) Protokol Integrasi dengan Peralatan Audio / Video, Peralatan untuk Ventilasi / Pendingin Udara / Pemanasan
3) Kemampuan untuk menghubungkan "Internet of things."

Dari sumber-sumber Internet, saya memilih perusahaan-perusahaan berikut yang diwakili di pasar Rusia, yaitu:

Logic Machine 4:



EVE Server:



Iridium Server:

Dari tabel di atas, saya membuat kesimpulan berikut:

  • LM4 dan EVE Server memiliki harga awal yang ditetapkan - lebih banyak fitur daripada Iridium Server
  • LM4 dan EVE Server mendukung koneksi panel kontrol dan jumlah titik data yang tidak terbatas.

Dari tabel saya menarik kesimpulan:

Kemampuan untuk menginstal perangkat lunak server pada platform lain memungkinkan Anda untuk secara fleksibel mendekati masalah untuk proyek-proyek anggaran rendah. Jika ini bukan objek besar, maka memulai server di Windows, Linux, Raspberry PI akan menjadi solusi yang sangat baik. EVE Server dan Iridium Server dapat digunakan di Raspberry, Windows, Linux, VirtualBox (hanya Server EVE), MAC OSX (Server EVE saja), OneBox (hanya Iridium); Logic Machine tidak memiliki fitur ini.

Pada fasilitas skala besar, jumlah permintaan ke server meningkat secara signifikan, sehingga server memerlukan solusi perangkat keras yang lebih kuat. Untuk tujuan ini, dimungkinkan untuk menggunakan Server EVE RACK yang berjalan pada platform DELL Power Edge. Untuk integrator, perangkat keras Server yang andal adalah penting. Saat ini, platform tersebut adalah Logic Machine dan EVE Server, yang memiliki periode operasi yang panjang tidak hanya di rumah, tetapi juga dalam otomasi industri. Iridium Server masih pendatang baru di pasar controller, jadi masuk akal untuk menunggu sampai ada pengalaman dengan server ini di fasilitasnya.

Kesimpulannya, saya ingin menambahkan bahwa di dunia otomatisasi yang terus berubah, memiliki satu teknologi selalu mengarah ke jenis solusi yang sama di masa depan ...

PS: Saya belum memilih pengontrol untuk diri saya sendiri.
