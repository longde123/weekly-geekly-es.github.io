终极乐趣：10个经典的控制台笑话

新年长假的一半结束了，现在是时候回想一下Unix管理员胡须的笑话了。



类似于UNIX的操作系统中的文本外壳程序不仅适合执行内部命令，运行各种实用程序以及编写脚本。 某些程序的结果可能会逗乐客人在灯光下四处闲逛。 RUVDS博客的编辑委员会向圣诞节的读者表示祝贺，并提供回忆起许多代系统管理员满意的经典控制台笑话。

1.即使是琐碎的make也不能仅从源代码编译程序。 使用此实用程序还有其他选择，但是如果您的系统上没有该软件包，则首先需要安装该软件包。 在Ubuntu中，这是通过一个命令完成的： 

apt-get install make

3.另一个不朽的经典是cowsay实用程序。 它不包含在默认配置中，因此您需要安装： 

 apt-get install cowsay



4. UNIX系统的所有用户都知道ls实用程序，但是sl（流机车）程序却鲜为人知。 

 apt-get install sl

如果安装并运行它，一列火车将通过控制台（几乎是ASCII图表中的Lumiere兄弟的电影）。





5.使用ASCII图形中的普通telnet，您可以观看发烧友绘制的《星球大战》的第四部分。 

 telnet towel.blinkenlights.nl



6.该程序的下一个编号是著名的实用程序财富，无论是否发行报价。 例如，可以从外壳初始化脚本中运行它，以便在用户登录系统时获得较小的预测-类似于控制台星座（在某些系统中，默认情况下已完成）。 有时，财富行情排在前十位，但是首先您需要安装该软件包： 

 apt-get install fortune.



7. yes内置命令没什么特别的，它在无限循环中重复相同的事情，直到用户厌倦为止： 

 yes Merry Christmas



8.如果您想在ASCII图形中漂亮地写一些短语，则无需花费大量时间进行艰苦的工作。 厕所公用事业的创造者已经做好了一切准备： 

 apt-get install toilet toilet Merry Christmas



借助其他参数，可以使标题更漂亮，例如，如下所示： 

 toilet -f mono12 -F metal Merry Christmas



9.矩阵的粉丝应在文本控制台中启动他们喜欢的绿色动画： 

 apt-get install cmatrix cmatrix



10.最后一个问题是另一个漂亮的演示程序： 

 apt-get install bb bb
