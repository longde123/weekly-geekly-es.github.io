Das Tesla Model S P85D kann in nur 2,8 Sekunden von 0 auf ~ 100 km / h beschleunigen



Tesla hat angekündigt, dass das Modell S eine neue Funktionsweise namens „Ludicrous“ (absurd, lächerlich) haben wird. Jetzt kann das Elektroauto in nur 2,8 Sekunden von 0 auf ~ 100 km / h beschleunigen. Zuvor konnte das Elektroauto in 3,1 Sekunden auf die gleiche Geschwindigkeit beschleunigen. Die vorherige Betriebsart wurde „Wahnsinnig“ (Wahnsinnig) genannt.

Laut Elon Musk beträgt die Beschleunigung in diesem Fall 1,1 G, dh mehr als die Beschleunigung der Schwerkraft. "Es ist, als ob Sie Ihre eigene Achterbahn hatten", kommentierte Musk die Nachrichten.

Um diesen Modus zu aktivieren, muss der Besitzer des Tesla Model S nicht nur die Software aktualisieren, sondern auch die Hardware ändern, und das nicht kostenlos. Warum? Tatsache ist, dass Tesla, um ein solches Ergebnis zu erzielen, neue Geräte entwickeln musste, einschließlich einer Sicherung, die nicht schmilzt, wenn eine hohe Ampere-Last durch sie geliefert wird.

Ein Upgrade auf eine neue Konfiguration kostet für Neukunden 10.000 USD und für bestehende Besitzer dieses Elektroautos 5.000 USD.

Musk kündigte außerdem zwei weitere Updates an:

1. Modell S 70, ein neues 70-kWh-Modell für 70.000 US-Dollar;
2. Eine neue 90-kWh-Batterie für 3.000 US-Dollar.

Leider wurden nur sehr wenige Details zu Modell X vorgestellt. Das einzige, was gesagt wird - Modell X wird innerhalb von 2 Monaten in den Handel kommen. Und Modell 3 wird in zwei Jahren verfügbar sein.

Und der Bonus ist das Video, das Reportern vor der Präsentation gezeigt wurde.
