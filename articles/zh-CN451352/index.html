Visual Studio C ++模板IntelliSense基于代码中的实例化进行填充

自从我们宣布Template IntelliSense以来，大家都给了我们很好的建议。 一种非常流行的建议是让模板栏根据代码中的实例自动填充候选对象。 在Visual Studio 2019版本16.1预览2中 ，我们通过模板栏下拉菜单中的“添加所有现有实例化”选项添加了此功能。 以下示例来自SuperTux代码库。



模板栏下拉菜单现在包含一个新条目“添加所有现有实例”。


单击“添加所有现有实例”选项将填充下拉列表，因此您不再需要手动键入条目。



此外，类似于“ 查找所有引用” ，编辑器底部的窗口将向您显示找到每个实例化的位置以及其参数。


与我们交谈！


我们希望此更新能够解决我们在Template IntelliSense上收到的受欢迎的客户反馈！ 我们希望您下载Visual Studio 2019版本16.1预览版2，并让我们知道您的想法。 可以通过以下评论或通过电子邮件（visualcpp@microsoft.com）与我们联系。 如果您在使用Visual Studio时遇到其他问题或有其他建议，则可以使用Visual Studio中的“报告问题”工具或前往Visual Studio开发者社区 。 您也可以在Twitter（ @VisualC ）上找到我们。
