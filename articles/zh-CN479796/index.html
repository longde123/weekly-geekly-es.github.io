金士顿的萨诺斯级威胁

从大多数用户那里购买存储卡的样子如何？ 如果满足价格标签和驱动器容量，则采用商店中的第一个引人注目的模型。 启蒙阶段稍后开始-当照片张开一段时间后，相机将无法以最大分辨率录制视频，并且该设备完全无法在不利条件下工作。 因此，在选择存储卡时，您应注意读/写数据的速度，同时询问以下内容将很有用：这样的存储卡在剧烈震动，温度变化，接触潮气后还能幸免吗？ 金士顿Canvas Select Plus是那些经常拍摄和旅行，将设备暴露在恶劣条件下的人的绝佳选择。 下面为什么会这样。



选择存储卡时需要了解什么？


未经培训的人员可能会对存储卡包装上的标记感到困惑。 即使是Habr读者的高级用户，刷新知识也将很有用。 要选择正确的驱动器，应注意几个参数。

目前，SDXC卡格式最为相关。 与旧的SD和SDHC不同，此类卡的容量可以达到令人印象深刻的2 TB。 如果“全尺寸”版本的SDXC适用于大多数笔记本电脑和相机，则microSDXC已广泛用于智能手机，运动相机和平板电脑。



通常的速度等级由字母C和数字表示。 例如，C10保证读和写的卡性能至少为10 MB / s。

对于要录制视频并且不想受到任何限制的人来说，视频速度等级尤其重要。 在包装和卡体上，由字母V内的数字表示。它表示写入速度，单位MB / s。 如果您以超高分辨率拍摄视频，最好选择标有V30及更高版本的驱动器。

UHS速度等级（超高速）由字母U内的数字表示-一或三。 这是舒适录制视频时需要注意的另一个参数。 如果U1足以支持全高清分辨率的视频流，则U3已经可以保证以4K格式拍摄。

许多购买者都不知道，罗马数字I或II表示UHS总线的速度 。 这与上述速度等级没有直接关系，仅表示接口的最大可能速度。 在不涉及技术细节的情况下，我们注意到，对于当今的大多数情况，UHS-I参数已绰绰有余，因此，如果该卡标榜为“一个”，请不要担心。 寻找UHS-II格式的对象是确实需要100 MB / s以上速度的专业人员。 这种卡的价格更令人印象深刻。

应用程序性能等级对于SD卡不是有用的参数，但对于microSD来说很重要。 它以小块表示写入和读取信息的速度。 因此，值得扩展智能手机或平板电脑内存的每个人都应该注意。 此处区分类别A1和A2。 两种情况下的最小线性写入速度均相同-10 MB / s，但是使用标记为A2的卡进行随机数据写入和读取的速度更快-每秒读取4,000个小文件，写入2,000，而读取1,500并最多写入2,000个文件A1。 选择是可以满足小工具特定需求和功能的类别。

有了这些知识，在卡体上破译“ SDXC V30 U3 C10”之类的标记就不会困难了。 现在让我们看看测试参与者。

在2019年，基于OC Android的智能手机出现了新趋势。 制造商开始在智能手机上大量放置用于两个SIM卡和一个存储卡的完整插槽。 赢得了理性的声音-现在可以将设备的内部存储器扩展2或3倍而不会出现问题。 尽管几年前一些目光短浅的营销人员做出了巨大努力，但MicroSD存储卡已重新流行。

我们的金士顿Canvas Select Plus microSD的功能在平均价格水平上很典型：速度等级U1，V10视频速度标记和128 GB内存。 该驱动器具有A1级性能，并针对与Android设备一起使用进行了优化。 该卡非常适合扩展智能手机或平板电脑的内存-与购买具有更高内部驱动器的相同设备相比，它的成本更低。 此外，更换时可以将microSD卡轻松插入新设备中，而无需担心复制数据。



我们已经讨论过金士顿High Endurance存储卡。 在极端的工作条件下，它们具有出色的耐久性和可靠性。 当今的评论者应该证明还不错：这些卡已经过防水，防震和极限温度保护的测试。 这使得金士顿Canvas Select Plus SD和microSD适用于运动相机，DVR，街道监控相机-简而言之，可用于周期性承受重负荷的设备。 只要Canvas Select Plus硬盘的耐用性与实际情况相符，请确保稍后再进行测试。
