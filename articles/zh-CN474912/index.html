通过JSON在Android上显示消息和警报

要接收短信，可以使用电子邮件，SMS，推送通知或为Messenger创建机器人。

我建议考虑另一种简单的方法：

1.在Web服务的一侧以JSON创建数据源。 

{ "refl.me":true, "message":"Hello!", "refresh":96 }

“ Refl.me”必定表示是真实的。
“消息”设置消息的文本，忽略空消息。
“刷新”设置必须在24小时内检查源的次数（0-手动更新，最大288-）。 

 <?php header('Content-Type: application/json'); $json_array =array('refl.me'=>true,'message'=>'Hello!','refresh'=>96); echo json_encode($json_array); ?>

2. 下载该应用程序，并在其中添加指向我们创建的数据源的链接。

3.一切准备就绪。 接收警报和消息。

操作原理非常简单，应用程序将按照指定的时间间隔（或在手动模式下）检查源，如果“ message”的值已更改为与先前值不同的值，则会在应用程序中显示警报。

一切都无需第三方服务器即可运行，不需要个人数据和注册。
