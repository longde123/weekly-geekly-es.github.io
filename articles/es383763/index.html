C Sensibo cualquier acondicionador "más sabio"



Los apartamentos y oficinas están llenos de aires acondicionados de modelos antiguos. A menudo funcionan bien, pero no hay demasiadas funciones en tales dispositivos. Sí, y la mayoría de los nuevos modelos tampoco se pueden llamar "inteligentes". Pero en nuestro tiempo de rápido desarrollo de la tecnología de la información, IoT (Internet de las cosas) quiero que si no toda la casa, al menos algunos dispositivos se puedan llamar "inteligentes".

El mismo aire acondicionado: llegas a casa o al trabajo, enciendes el aire acondicionado y esperas hasta que la temperatura se calme (no importa si eso significa calentar o enfriar). Siempre me gustaría tener un microclima adecuado en una casa u oficina, ¿verdad? Y aquí un pequeño dispositivo viene al rescate, un sistema llamado Sensibo. Este sistema, entre otras cosas, ayuda a ahorrar energía: puede configurar el aire acondicionado para que funcione de tal manera que el dispositivo se apague si no hay nadie en la habitación durante un tiempo determinado. Por cierto, el dispositivo fue desarrollado por una pequeña startup israelí, y los israelíes saben cómo usar aires acondicionados como ningún otro.

Cómo se ve?




El sistema es muy pequeño, consta de dos partes: un concentrador y un sensor de control. Luego, se necesita el concentrador que, si tiene varios aires acondicionados instalados, podría administrarlos todos a la vez, y no alternativamente.

El sensor de control se instala directamente en el aire acondicionado, utilizando la base con el lado adhesivo. Este velcro es muy fuerte, por lo que el sensor no se caerá ni caerá.



El sensor interactúa constantemente con el concentrador, que, a su vez, se conecta a la red Wi-Fi del hogar u oficina.







Configuración


Interior: un conjunto completo de detectores, que incluyen un detector de temperatura, un detector de humedad, iBeacon, un detector de flujo de aire, un detector de luz, un receptor de infrarrojos.

Como plataforma, se utiliza CC2538, que funciona según el estándar de RF 802.15.4 con el protocolo 6LoWPAN. El MCU secundario es el CC2541 de Texas Instrument con Bluetooth Low Energy y soporte iBeacon.

En modo independiente (de dos baterías CR123A), Sensibo puede funcionar durante unos tres años.

¿Qué puede hacer Sensibo?




Para controlar todo el sistema, debe usar un dispositivo móvil con una aplicación preinstalada. Hay versiones de software para iOS y Android, además de que también hay un programa para Pebble. Es claro de inmediato que el sistema fue hecho por geeks.

El teléfono inteligente se puede usar como control remoto y como sistema de monitoreo: los indicadores de microclima en la habitación se muestran en la pantalla.

Además, Sensibo se puede controlar desde cualquier lugar donde haya acceso a Internet, mientras que la mayoría de las funciones funcionan automáticamente. Por ejemplo, si el propietario se acerca a la casa, Sensibo enciende el aire acondicionado y comienza a funcionar para calefacción o refrigeración, estableciendo los parámetros necesarios del microclima en la habitación.
