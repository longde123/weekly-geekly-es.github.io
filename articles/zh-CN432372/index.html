微软已经正式确认Edge会切换到Chromium引擎

几天前，有传言称微软正在开发基于Chromium的浏览器，默认情况下它将取代Edge。 原因是Microsoft员工突然开始致力于免费项目。 现在，微软已经正式宣布 ，它真的放弃了自己的EdgeHTML引擎，转而在桌面版浏览器中使用Chromium。

对于Web开发人员来说，这是一个很大的安慰：在测试过程中，一个引擎将变小，并且IE / Edge的许多原始功能将被淘汰。 但是，根据Mozilla的说法，微软同事的这一决定对互联网构成了威胁，因为来自Google的单个平台的主导地位越来越全面。 Firefox可能会陷入边缘边缘，开发人员甚至会停止对其进行测试。 Firefox的份额持续下降.2018年12月，它跌至多年来最低水平： 台式机不到9％ 。

微软怀着良好的意愿对这一决定进行了辩护：“在过去的几年中，微软大大扩展了对开源软件社区（OSS）的参与，成为世界上最大的OSS项目支持者之一。 今天，我们宣布，我们打算使用开源项目Chromium在桌面上开发Microsoft Edge的桌面版本，以为我们的客户提供更好的Web兼容性，并为所有Web开发人员减少Internet的碎片化，” 官方新闻稿说。 。

微软认为，通过允许开发人员为Chromium项目做出贡献，他们不仅可以改善Microsoft Edge，还可以改善台式机和移动设备上的其他浏览器。 结果，Internet对于许多受众来说将更好：“ Microsoft Edge用户（以及可能的其他浏览器）将与所有网站都有更好的兼容性，并在所有Windows设备上具有更长的电池寿命和硬件集成。 “ Web开发人员将获得用于测试站点的较少分散的Web平台，这可以保证减少问题并提高用户满意度。”

再见EdgeHTML


同情的Mozilla博客文章“再见EdgeHTML” 。 这一点根本不具有讽刺意味：同事们对微软拒绝开发自己的引擎感到非常遗憾：“微软采用Chromium进行使用后，便赋予了Google对在线生活的更大控制权，”他们写道。 -听起来很不错，但事实并非如此。 “浏览器引擎-Google的Chromium和Mozilla的Gecko Quantum-是内部软件，实际上定义了我们每个人在互联网上可以做的很多事情。 它们确定了主要功能：我们作为消费者可以看到哪些内容，查看内容时的安全性以及我们对网站和服务的控制程度。 微软的决定为Google提供了更多机会，让我们可以一手决定哪些选项可供我们每个人使用。”

Mozilla相信Microsoft会根据业务的优势做出此决定：“ Google几乎几乎完全控制了我们在线生活的基础架构，因此继续奋斗可能无利可图。 微软股东的利益很可能会拒绝互联网曾经为我们提供的自由和多样性。 Google的竞争对手非常强大，拥有非常有才能的员工，并且拥有专有资产的专有权。 Google在搜索，广告，智能手机和数据收集方面的主导地位造成了一个非常不公平的竞争环境，对我们所有人都不利。”

Mozilla说，从社会，公民和个人的角度来看，转移对一家公司的基本在线基础设施的控制权是很糟糕的。 他们强调，保持用户手中的权力是Mozilla作为非营利组织存在的原因。 他们不是在业务方面看待来自Google的竞争，而是在整个互联网的福利方面看待。 从这个意义上来说，Firefox最近跌至谷底感到很遗憾：其在台式机上的份额下降到9％以下 。 这是多年来第一次发生，并且Firefox的份额持续下降，从而占据了Chrome的全部主导地位。



Mozilla承认Chromium的主导地位使开发人员和企业的生活更加轻松：在测试中，您可以验证自己的应用程序仅在Chromium下运行良好：火狐浏览器 而且这可能再次发生。”
