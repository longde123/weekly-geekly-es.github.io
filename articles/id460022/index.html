  • PHP 7.4.0 alpha 3 - Rilis alpha terbaru dari yang direncanakan . Pembekuan fitur harus 22 Juli, sehingga daftar perubahan saat ini hanya dapat ditambahkan oleh mereka yang pemungutan suara dimulai selambat-lambatnya 8 Juli. Total sekarang dalam pemungutan suara:
    • Penghentian untuk PHP 7.4
    • Melarikan diri dari PDO "?" penampung parameter
    • Gunakan sintaks kurung kurawal untuk mengakses elemen larik dan offset string
    • Tambahkan str_starts_with (), str_ends_with () dan fungsi terkait .

    Juga beberapa perubahan baru tanpa RFC terpisah:
    • proc_open akan menerima argumen dalam bentuk array, mereka tidak perlu melarikan diri dan proses akan mulai secara langsung tanpa shell perantara: 
     proc_open(['php', '-r', 'echo "Hello World\n";'], $descriptors, $pipes);

    • Dalam PHP 7.4, pemberitahuan akan dilemparkan ketika mencoba menggunakan jenis skalar sebagai array yang salah. Sebelumnya, panggilan semacam itu hanya mengembalikan null : 
     $null = null; var_dump($null[0]); // NULL $int = 42; var_dump($int['foo']); // NULL var_dump(true[0] === false[0]); // true

Alat-alatnya


  • humbug / php-scoper - Alat ini memungkinkan Anda untuk menambahkan awalan ke semua ruang nama dalam folder yang ditentukan dalam satu langkah. Ini mungkin berguna ketika Anda perlu menggunakan beberapa versi dari satu paket, contoh penggunaan untuk plugin WordPress .
  • CaptainHookPhp / captainhook - Git hook manager untuk proyek PHP.
  • sebastianfeldmann / phpbu - Alat untuk membuat salinan cadangan file dan basis data. Mampu mengenkripsi cadangan dan mengirim ke penyimpanan.
  • localheinz / http-method - Satu set konstanta dengan nama-nama metode HTTP.
  • spiral / roadrunner - Server aplikasi untuk PHP di Golang. Menambahkan dukungan untuk FastCGI, pengaturan CORS di luar kotak, metrik bawaan di Prometheus. Ulasan terperinci dalam laporan dari penulis .
  • leocarmo / circuit-breaker-php - Implementasi pola Circuit Breaker untuk PHP. Alternatif: ackintosh / ganesha .
