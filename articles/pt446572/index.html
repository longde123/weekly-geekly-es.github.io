O Editor.js é um excelente editor que salva o código-fonte no formato JSON

CodeX são caras legais que são baseados na ITMO University. Com base nessa universidade, estudantes e graduados criam projetos interessantes e úteis: Hawk.so , Capella , AR-tester e outros.

Um projeto ajudou muito nossos negócios. Editor Editor.js , salvando o código-fonte em JSON.



A principal vantagem do editor é o armazenamento e a apresentação de dados no formato JSON. Aqueles que possuem a versão web do produto e o aplicativo móvel entenderão especialmente. Para escrever artigos, os gerentes de conteúdo gostam de usar o editor WYSIWYG, mas depois obtemos "espaguete":



O espaguete simplesmente matou todo o layout e integrá-los em aplicativos móveis foi apenas uma dor . Eu tive que escrever um artigo separadamente para a versão web e o aplicativo móvel. É longo e difícil de manter.

O trade-off é o formato JSON. É simples, possui uma estrutura rígida e é flexível. Adicione o editor Editor.js ao painel de administração, grave o conteúdo e salve-o como JSON. Pela API, enviamos o JSON para o aplicativo móvel, organizamos-o ainda mais, conforme necessário, e o convertemos para a versão da Web do JSON em HTML via Editor.js. E toda felicidade.



Os caras fornecem uma API para criar seus plugins. O site possui boa documentação com exemplos. Use!
