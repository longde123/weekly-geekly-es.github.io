创业营销：如何在不花费200美元的情况下吸引来自世界各地的数千名用户



今天，我将告诉您如何为启动Product Hunt的创业公司做准备，在此之前应该采取什么步骤，以及如何在发布当日和发布后激发对该项目的兴趣。

引言


在过去的两年中，我一直生活在美国，并一直在利用英语（不仅是英语）资源推广创业公司 。 今天，我将分享用最少的投资吸引IT初创公司的国际用户的经验。 最适合此的是内容营销工具。 其中许多是免费的或几乎免费的，但可以带来良好的效果。

因此，如果您想吸引国际用户，就应该在这里提倡您的创业公司。

显示hn


长期以来，Hacker News一直是利基市场中最受欢迎的技术和创业公司之一。 如果一个项目至少要花几十分钟才能到达其主页，这可能已经导致流量激增-轻松获得数百次点击即可。



此资源有一个“显示HN”部分-在这里项目创建者或普通用户可以共享有用的链接。 促销值得从此资源开始-它是免费的，如果成功，您可以获得良好的结果。 最有可能的是，您不会成功接触到主要用户，并且HN上的受众并不总是很友好，但是从知名资源到网站的链接无论如何都不会多余。

Betapage和Betalist


在类似模型上工作的两个资源。 这些是启动启动目录。 收费后，项目描述会绕过常规队列（不会快速移动）进入列表，可以在主页上进行固定，这些服务还在其邮件列表中包括指向该项目的链接。

我上次使用Betapage时的价格是：

  • 一日游首页：$ 28
  • 2天课程：$ 48
  • 3天课程：$ 68
  • 4天课程：$ 88

订阅时事通讯的费用为30美元，因此该价格应加到最终价格中。 我们在此网站上使用付费列表的经验不能称得上成功-数百名注册用户没有在主邮件列表上给我们一天。

Betalist发布费用较高-$ 129，但也有可能免费发布。 在后一种情况下，等待放置大约一个月，但是如果有要保存的任务，这是一个不错的选择。 在最近的项目中，我们选择了此选项，仅收到452个用户，而发布之日激增。



通常，主要成本是Betapage和Betalist产生的，而在我们的案例中，仅在第二种情况下它们至少部分合理。

国际内容营销


由于我们计划吸引不仅来自英语国家的用户，因此我们决定朝这个方向进行试验。 我们已经有一些不错的英文文章，以及关于Betalist和Product Hunt的描述。

在Upwork的帮助下，我们找到了西班牙裔编辑，他们不仅帮助我们进行了翻译，而且还为在何处以及如何共享与已出版资料的链接提供了建议。 我必须说，将一篇博客文章从英语翻译成西班牙语的费用通常不超过10美元。

结果，我们选择了两种主要资源来“播种”内容：

  • Taringa.net是拉丁美洲最受欢迎的聚合网站，与Reddit有点相似，拥有超过2100万访问者。
  • Meneame.net -950万访客。

此外，我们自己选择了Reddit上的多个部分，拉美裔拉丁美洲国家的居民可以在这些部分进行交流：

  • r / aprendeingles / -专门研究英语的部分；
  • / r / es / -总体主题西班牙裔subreddit，1.43万名用户；
  • / r /阿根廷/-9.2万名订阅者；
  • / r / vzla / -委内瑞拉部分， 3.45万订户；
  • / r /哥伦比亚 --1.38万名订户；
  • / r / chile --48 000用户。

结果超出了所有人的期望-我们不仅收到了Reddit的注册和评论，例如Meneame的评论，而且该项目引起了拉丁美洲记者和博客作者的注意。 特别是，在对流行的阿根廷IT资源进行了新的审查之后，几天之内有数千人来找我们。



但是，主要事件当然是在Product Hunt上的展示位置。

产品搜寻：发布指南


该网络上有很多关于如何最好地在Product Hunt上运行的指南和故事，因此我不会写太多，我只会专注于要点。

首先，您应该从PH命令本身阅读启动指南 。 这是文档的三个要点：

  • 您不能寻找发布产品的猎人 。 现在这变得没有意义了-以前，没有向受欢迎的用户的订户发送有关其发布产品的电子邮件，而现在还没有。
  • 您可以直接链接到产品发布，而不能链接到PH主页。 根据城市传说，PH会对用户单击直接URL而不在主页上找到的项目进行罚款。 这是不对的。
  • 您不能要求喜欢-他们会为此被罚款 。 因此，与发布相关的所有通信都应包含明确的消息：“我们在PH上发布，这里是链接，请进，发表评论，提出问题！”

这些是许多人仍然不知道的要点。 还有几点需要重点强调。

  • 猎人更好 。 这是更时尚的时刻，如果此人在其社交网络上的PH上发布指向您帖子的链接，则将是一个加分。 但是，如果长时间不寻求这种角色，那么您可以这样做。
  • 在PH上运行需要24小时工作 。 新的一天的倒计时从美国西海岸时间00:00开始。 您必须在线24小时才能及时回复评论。
  • 24小时后，您还需要工作 。 进入当天的顶级产品列表会带来好处-在新闻通讯中也提到了您，但是如果您没有成功，还有其他方法可以进一步使用PH。 例如，站点有机会提出问题，答案可能是特定软件的建议。 如果您选择这种方式，则会获得更多流量。

结果，我们没有进入前5个项目的名单，但是用户至少投票了一个星期，这使我们得以保留在主要项目上，因为它每天显示大约10-15个产品。 自发布之日起，时间越长，就越需要滚动查找产品，但这不会影响到网站的过渡。

长期住宿的主要秘诀是您应该对此保持兴趣。 这意味着准备并定期发布提及您的商品目录链接的文章，进行新闻通讯等。



示例：一个项目博客帖子，该帖子要求转到PH上的产品页面以及带有类似计数器的内置徽章。

结论


材料中描述的活动花费了我们大约200美元，花费了几周的时间来准备和实施它们。 结果，我们获得了数千次对该网站的点击和数百次注册。 您可以将这些技术复制给您的初创公司，我敢肯定，这将有助于获得第一批国际用户。
