Das QRL-Team hat eine enge Frist für die Token-Migration festgelegt


Figure snannerb - QRL-Community in Discord

Neulich brachte das QRL-Team erfolgreich Münzen des Hauptnetzwerks zur Bittrex-Börse und setzte eine enge Frist (28. Februar 2019) für die Token-Migration.

Wir haben in einem früheren Artikel über das QRL-Projekt (Quantum-Resistant Ledger) geschrieben.

Hintergrund


In der zweiten Jahreshälfte 2016 gründete ein britischer Wissenschaftler, Peter Waterland, eine Gruppe, um eine Blockchain zu entwickeln, die mit einem Quantencomputer gegen Angriffe resistent ist. Ende desselben Jahres wurde ein „ Weißbuch “ in Open Access veröffentlicht, auch in russischer Sprache.

Vom 1. bis 24. Mai 2017 werden Mittel für den Ausbau des Hauptnetzes gesammelt. Zu diesem Zeitpunkt funktionierte das Testnetzwerk bereits und der Quellcode des Projekts wurde auf GitHub veröffentlicht. Die Ethereum-Infrastruktur wurde als Plattform für das ICO ausgewählt, das QRL-Token des ERC-20-Standards wurde ausgestellt. Es war geplant, 2 Millionen US-Dollar anzuziehen, aber das Team konnte mehr als 4 Millionen US-Dollar sammeln. Nach dem ICO konnte jeder QRL-Token an den Börsen Bittrex, Upbit und Liqui kaufen oder verkaufen.

Ein Jahr später, Ende Juni 2018, wurde das Haupt-QRL-Netzwerk gestartet. Kurz vor dem Start des Hauptnetzwerks wurde der Beginn des Austauschprozesses (Migration) von ERC-20-Standard-Token für eine vollwertige quantenstabile (im Gegensatz zum Token) QRL-Münze im Verhältnis 1: 1 angekündigt.

Es war geplant, die QRL-Münze an der Börse freizugeben, aber das Projektteam hielt seinen Austauschpartner bis zum Ende geheim. Ende November wurde die Notierung von Bittrex bekannt gegeben. Während des Austauschs wurden zwei Wochen lang Arbeiten durchgeführt, um Token durch Netzwerkmünzen zu ersetzen. Community-Nutzer stimmten sogar über das Enddatum des Übergangs ab. Während dieser ganzen Zeit war es unmöglich, Münzen in die Börse einzugeben und von ihr abzuheben.

Unmittelbar nach Bekanntgabe der Auflistung wurde das Enddatum des Token-Migrationsprozesses angegeben: 28. Februar 2019. Nach diesem Datum wird das automatische Portal für die Migration geschlossen, und Token können nur durch Kontaktaufnahme mit dem Projektteam ausgetauscht werden.

Migrationsanweisungen


1. Erstellen und speichern Sie eine QRL-Brieftasche


Folgen Sie wallet.theqrl.org , um eine Brieftasche zu erstellen.



Erstellen Sie ein sicheres Passwort und geben Sie es ein. Sie können auch die Höhe des Signaturbaums und den Kennwort-Hashing-Algorithmus angeben, dies ist jedoch nicht erforderlich. Klicken Sie auf "Brieftasche erstellen" und die Brieftasche wird erstellt. Auf dem Bildschirm wird eine Seite mit einer QRL-Adresse und Informationen für den Zugriff auf die Brieftasche angezeigt: die Mnemonik und der Wert, der zum Generieren der Brieftasche verwendet wurde - Hexseed.



Sie sollten die auf dem Bildschirm angezeigten Informationen und die Brieftaschendatei speichern. Beim Speichern einer Brieftaschendatei wird empfohlen, "Sichere Brieftaschendatei speichern" auszuwählen.

Jetzt müssen Sie sicherstellen, dass Sie Zugriff auf die neu erstellte Brieftasche haben. Klicken Sie auf "Brieftasche öffnen". Geben Sie im folgenden Fenster den Pfad zur Brieftaschendatei und das Kennwort an, das Sie beim Erstellen der Brieftasche angegeben haben.



Wenn alles richtig gemacht wurde, wird ein Fenster mit der Adresse, dem Transaktionsverlauf und den verfügbaren Vorgängen geöffnet.



2. Erstellen Sie eine ERC-20-Token-Zerstörungsadresse


Wechseln Sie zur Token- Migrationsseite migration.theqrl.org .



Geben Sie Ihre QRL-Adresse ein. Es wird außerdem empfohlen, eine E-Mail-Adresse anzugeben. Sie erhalten an diese Adresse eine Benachrichtigung über den erfolgreichen Abschluss des Token-Migrationsprozesses. Klicken Sie dann auf die Schaltfläche "Senden". Die Token-Zerstörungsadresse wird generiert.



Halten Sie die generierte Token-Zerstörungsadresse geheim und verwenden Sie keine Fremden.

3. Senden Sie ERC-20-Token an die Zerstörungsadresse


Nachdem Sie Token an die Zerstörungsadresse gesendet haben, können Sie die Anzahl der darauf empfangenen Token überprüfen, indem Sie auf "Überprüfen" klicken.



Es wird empfohlen, zuerst einen kleinen Teil der Token zu senden. Nach einer erfolgreichen Operation können Sie den Rest senden. Nach dem Senden von Token an die Zerstörungsadresse erreichen QRL-Münzen nicht sofort Ihre Adresse. Die Anmeldung erfolgt innerhalb weniger Tage.
