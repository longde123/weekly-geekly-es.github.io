Printer 3D Funtastique EVO v1.0: pengantar pertama



Jujur, saya sedang merakit printer 3D untuk pertama kalinya. Bagi saya, itu selalu sesuatu yang rumit dan tidak bisa dipahami, sesuatu yang hanya dapat dikumpulkan di laboratorium. Tapi 2017 datang, dan printer dapat dirakit di rumah. Baca saja instruksinya. Printer 3D modular Funtastique EVO v1.0 cocok untuk peran printer rumah pertama ketika pengetahuan khusus dalam pencetakan 3D tidak diperlukan. Ini dirakit secara sederhana, sebagai desainer anak-anak, tidak ada yang perlu disolder dan diprogram. Setiap orang telah mendengarkan kami.

Kesimpulan

Saya menyukai printer ini. Jika Anda melakukan semuanya sesuai dengan instruksi, yaitu: letakkan lapisan di bawah meja, gunakan pita perekat, kalibrasi, maka tidak ada masalah. Saya merekomendasikan model ini sebagai printer 3D pertama yang mempelajari perangkat lunak, mencetak rincian yang diperlukan untuk rumah. Di masa depan saya ingin mencetak komponen 3D untuk mobil saya. Mobil itu tua, suku cadangnya tidak mudah dan murah untuk didapatkan. Dan Anda dapat menemukan perangkat lunak di Internet. Sesuatu seperti ini. Terima kasih
