卡利Linux 2020.1


有一个滚动发布的Kali Linux 2020.1，其中包含许多更新和创新：默认情况下为非root用户，放弃python 2（已弃用），将XFCE作为默认桌面环境（来自2019年的最新版本）。


TL; DR


  • 默认情况下是非Root用户（root / toor已死。kali/ kali万岁）
  • 平台/位的单个安装映像
  • Kali NetHunter无需根设备。
  • 新主题和改进的迷彩模式。
  • 新的实用程序。

新品


在Kali（及其前身BackTrack，WHAX和Whoppix）的整个历史中，默认凭据为root / oror。 在Kali 2020.1中不再使用默认的root超级用户帐户。 现在，默认用户帐户是普通的非特权用户（kali / kali-不要忘记更改密码）。




不再为每个桌面环境（DE）提供单独的映像。 相反，现在有一个映像可以在安装过程中选择DE。 这意味着Xfce（自2019.4起是默认环境），GNOME，KDE，MATE或LXDE没有下载链接。 仅一个图像即可选择任何环境。


默认情况下，该映像包含kali-desktop-xfce和kali-tools-default软件包，这些软件包允许脱机安装Kali。 选择任何非默认工具将需要网络连接。




移动渗透测试平台Kali NetHunter也进行了一些改进。 现在，尽管有一些限制，但您不再需要为运行Kali NetHunter而扎根手机。


  • NetHunter-需要具有恢复功能和修补内核的根设备。 没有限制。
  • NetHunter Light-要求具有自定义恢复功能的根设备，但没有内核修补程序。 它有较小的限制，即 不支持Wi-Fi注入或HID。
  • NetHunter Rootless-使用Termux安装在所有标准的未修改设备上。 一些限制，例如Metasploit中缺少数据库支持以及缺少root权限。

现在，已经为GNOME用户显示了一个新主题，并且还有一个额外的好处-可以在明亮和黑暗主题之间进行选择。




延续Kali 2019.4主题，Kali-undercover（在社区中受到高度肯定）现在更像Windows，可以帮助以隐身模式工作。




添加了以下新工具：云枚举，emailharvester，phpggc，sherlock，splinter。


您可以在怀旧的时刻使用一些新的（kali-社区壁纸）和旧的（kali-legacy-壁纸）墙纸。


Python 2已结束支持，所有依赖Python 2的工具都将被删除。


更新您的Kali版本：
