如何在流行的浏览器中禁用自动视频播放



随着Internet的发展和平均访问速度的提高，许多网站所有者认为在加载页面后立即自动开始播放视频是一种可接受的做法。

但是，并非所有用户都同意此政策。那些不同意这一观点的人，认为音乐播放质量不好的站点（在90年代至00年代等）很糟糕，现在他们认为视频不应自动开始播放，而只能根据用户的请求开始播放。

自动开始视频会导致流量浪费（用户可能希望避免），频道带宽的减少以及出现不适当的响亮声音。而且，即使在Firefox选项卡中也标记了当前正在发出声音的选项卡，对于某些用户来说，浏览器的这种行为似乎也不舒服。

尤其是对他们而言，博主Martin Brinkmann准备了多种方法来禁止针对不同浏览器自动播放视频。其中包括使用Flash的视频和HTML5选项。

Firefox和HTML5


很好，Firefox具有无需插件即可阻止自动运行视频的设置。对于HTML5，设置about：config将对您有所帮助。您需要加载此页面，找到设置media.autoplay.enabled并将其切换为false



Firefox和Flash


为了防止基于Flash的视频自动启动，您可以下载扩展页面，转到那里的插件页面，然后将Shockwave Flash设置更改为“要求激活”（始终在启动前询问）。



Chrome和HTML5


Chrome没有设置禁用自动运行。插件可以帮助解决此问题，但只能在桌面版本中提供帮助-由于移动Chrome浏览器不支持插件，因此其用户仍然无法摆脱这种不幸。

在桌面上，您可以使用Disable HTML5 Autoplay和Video Autoplay Blocker插件。

Chrome和Flash


要禁用Flash视频，您需要加载Chrome：//设置/内容设置，在此处找到插件部分，并将设置切换为“让我选择何时运行插件内容”（选择何时启动插件）。



Opera和HTML5


Opera是基于Chromium / Blink引擎的，因此它的情况类似于Chrome。要阻止基于HTML5的视频，您可以使用禁用HTML5自动播放和视频自动播放阻止程序插件。

歌剧和Flash


要禁用Flash视频，您需要下载Opera：//设置，在左侧菜单中切换到“网站”，然后在“插件”部分中切换到“点击播放”选项。



Internet Explorer / Microsoft Edge


在IE中，无法配置视频的自动显示-您只能完全禁用Flash。希望Edge附带插件支持时，将为其移植带有Chrome的插件。

维瓦尔第


新的Vivaldi浏览器基于Chromium / Blink引擎，因此所有设置和插件的工作方式与Chrome浏览器完全相同。

更新：不幸的是，在Firefox浏览器中，如果关闭了自动视频播放功能，它将开始使YouTube视频播放静音（在HTML5模式下）。
