Praktek mempersiapkan kata-kata asing dengan pengisi suara untuk menghafal dalam program Anki

Dalam artikel ini saya akan menceritakan tentang pengalaman pribadi saya dalam menghafal kata-kata bahasa Inggris menggunakan program luar biasa dengan antarmuka Anki yang tidak jelas. Saya akan menunjukkan kepada Anda bagaimana tidak berubah menjadi rutinitas pembuatan kartu baru dengan voice acting untuk menghafal.


Diasumsikan bahwa pembaca sudah memiliki pemahaman tentang teknik pengulangan interval dan akrab dengan Anki. Tetapi jika Anda tidak tahu, saatnya berkenalan .


Kemalasan bagi seorang spesialis TI adalah hal yang hebat: di satu sisi, memaksa seseorang untuk menyingkirkan rutinitas melalui otomatisasi, dan di sisi lain, dengan sejumlah besar rutin, kemalasan menang, menekan minat belajar mandiri.


Bagaimana tidak berubah menjadi rutinitas proses pembuatan kartu untuk menghafal kata-kata asing?


Ini resep saya:

  2. Instal Anki
  3. Instal plugin AwesomeTTS
  4. Tambahkan bookmark di browser:

  6. Sinkronkan

Daftar untuk AnkiWeb


Registrasi tidak rumit, tetapi memungkinkan untuk menggunakan Anki di perangkat yang berbeda. Saya menggunakan Anki versi Android untuk diingat dan versi PC untuk membuat kartu baru. Anda juga tidak dapat menginstal aplikasi pada ponsel cerdas Anda, karena Anda dapat menghafal kartu secara langsung di situs di bawah akun Anda.


Alamat situs web: https://ankiweb.net/

Kartu memasak


Kami menulis kata-kata yang tidak dikenal


Kami menggunakan metode "proofreading" untuk dengan cepat mengekstraksi kata-kata tidak dikenal dari teks asing untuk menghafalnya.


Inti dari metode ini:


  1. Anda membaca teks asing dengan sangat cepat.
  2. Labeli kata-kata yang artinya tidak jelas bagi Anda.
  3. Anda mengeja kata-kata ini, menerjemahkan dalam konteks dan menghafal (Anki digunakan pada tahap ini).
  4. Kemudian Anda membaca teks itu lagi, tetapi dengan pemahaman semua kata, karena lihat paragraf 3.

Mungkin metode ini entah bagaimana disebut berbeda (ceritakan dalam komentar), tapi saya pikir intinya jelas.

Akting suara


Google Cloud Text-to-speech adalah layanan khusus yang memungkinkan Anda menerjemahkan teks secara kualitatif ke versi pidato. Untuk menggunakannya di Anki, Anda perlu membuat kunci API Anda sendiri atau yang disarankan oleh penulis plugin AwesomeTTS dalam dokumentasi (lihat bagian KUNCI API ).

Sinkronkan


Saya menggunakan Anki di PC untuk membuat dan mengelompokkan kartu berdasarkan topik, seperti ini paling mudah dilakukan dalam versi ini.


Saya menghafal kartu dalam aplikasi untuk Android.


Di atas, saya telah menunjukkan cara mengkonfigurasi sinkronisasi Anki untuk PC dengan AnkiWeb.


Setelah menginstal aplikasi Android, pengaturan sinkronisasi bahkan lebih mudah.


Terkadang konflik sinkronisasi terjadi dalam suatu program. Misalnya, Anda mengubah kartu yang sama pada perangkat yang berbeda, atau karena kegagalan sinkronisasi. Dalam hal ini, aplikasi akan memperingatkan Anda sumber mana yang akan diambil sebagai dasar untuk sinkronisasi: baik AnkiWeb atau aplikasi - hal utama di sini bukan untuk membuat kesalahan, jika tidak data tentang kemajuan pembelajaran dan perubahan yang dibuat dapat dihapus.
