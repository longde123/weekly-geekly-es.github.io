Breakout auf Svelte entwickeln

Auf MDN gibt es ein Tutorial namens "2D Pure JavaScript Game" , in dem die Grundlagen der Verwendung des HTML5- Elements <canvas> erläutert werden.



In diesem Tutorial werden wir die Entwicklung dieses Spiels auf Svelte wiederholen.


1. Erstellen Sie eine Leinwand und zeichnen Sie darauf

Reines JavaScript - Svelte


In diesem Beispiel werden drei geometrische Formen angezeigt: ein Quadrat, ein Rechteck und ein Kreis.



Die Canvas-Variable wird definiert, nachdem die Komponente im DOM bereitgestellt wurde . Daher wird der gesamte Code in den onMount-Lifecycle- Handler eingefügt . Binden des Canvas-Elements an die Canvas-Variable mithilfe dieser Bindung . 


<script> import { onMount } from 'svelte'; let canvas; onMount(() => { const ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.rect(20, 40, 50, 50); ctx.fillStyle = "#FF0000"; ctx.fill(); ctx.closePath(); ctx.beginPath(); ctx.arc(240, 160, 20, 0, Math.PI*2, false); ctx.fillStyle = "green"; ctx.fill(); ctx.closePath(); ctx.beginPath(); ctx.rect(160, 10, 100, 40); ctx.strokeStyle = "rgba(0, 0, 255, 0.5)"; ctx.stroke(); ctx.closePath(); }); </script> <canvas bind:this={canvas} width={480} height={320} ></canvas> <style> canvas { background: #eee; } </style>

Außerdem übertragen wir einfach den gesamten Code ohne Änderungen an der Funktion onMount (). Kommentare werden nur bezüglich der Verwendung von Svelte gegeben.
