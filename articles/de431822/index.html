Betrachten Sie eine der letzten Episoden in der Saga der Open-Source-Branche, die keine Ahnung hat, was sie im Allgemeinen tut:

Dies ist kein Einzelfall. Höchstwahrscheinlich gibt es in Ihren Abhängigkeiten viele andere Module, die von den Autoren aufgegeben wurden. Ich habe diesen Code nicht aus Altruismus geschrieben, sondern zum Spaß. Ich habe studiert und das Lernen macht Spaß. Ich habe es veröffentlicht, weil es einfach war und weil das Teilen von Code auch beim Lernen hilft. Ich denke, die meisten kleinen Module auf npm werden aus den gleichen Gründen erstellt. Es ist jedoch sehr lange her. Seitdem bin ich von diesem Modul zu einem anderen und dann zu einem anderen gewechselt und bin gerade dabei, von diesem Modul überzugehen. Ich habe viel bessere Module geschrieben, das Internet hat einfach nicht ganz aufgeholt.
