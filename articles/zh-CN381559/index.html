抗危机超极本：便宜又开朗

你好，GT！碰巧的是，汇率和各种经济问题极大地提高了我们和我的面包收入，我们每天的生活和呼吸方式的价格。 笔记本电脑并没有摆脱这种悲惨的命运，如果早些时候花35.4万卢布就可以找到一台相当便携且可长时间使用的便携机，那么现在……总的来说，我们会弄清楚火星上是否有生命。





选择了371版的Acer Aspire V3作为测试样本，V3有一个旅行车和各种配置的小型手推车：某处有更多RAM，某处屏幕更简单，这不是重点。



最主要的是它重量轻，具有便携式设备所需的所有端口，外观良好且组装完美。我们将在演出过程中处理其余的事情：）

外观和外围连接


简而言之设计：不是MacBook。不，嗯，这是可以理解的，每个公司都有自己的准则和可识别的风格。在这种情况下，V3系列拥有绝对的不露面和出色的外观。唯一的“技巧”是铝板作为笔记本电脑护盖上的装饰元素。从触感上来说，它当然不是冷铝，但看起来不错。



其他所有内容只能用“整洁”一词来表征。整洁的触摸板，整洁的键盘（当然，Acer的典型布局和箭头块还没到任何地方）。一切都是哑光和灰色。并且容易弄脏。



USB端口，HDMI和电源连接器均位于右侧。左侧只有存储卡插槽和一个用于连接耳机的4针“移动” MiniJack。套件中有两个3针连接器的适配器，您猜对了。



在我看来，充电口太细了。当然，我了解Apple，专利和所有方面的知识，但是您可以制作一个普通的连接器（对于Lenovo来说，虽然不完美），但半年后不会松动。



顺便说一句，在这种情况下，完整的充电器又很小又很轻。关于网线还不能说。



它的重量并不轻，占用的空间通常比电源要大。显然，只有Apple和Asus可以设计正常收费。



键盘


现代笔记本电脑的祸害-键盘工程领域中的各种“设计”解决方案。要么正常的辅助键块看起来都是多余的，然后Ctrl，Fn，Alt和Win键以一种奇怪的方式定位，然后在箭头块上嘲笑它们……总的来说，谁值得呢？

在这种情况下，布局不是理想的，而是尽可能接近“您可以舒适地使用”状态。



逃离左上角？是。 Ctrl在右下角？是。 Alt靠近太空吗？是。保罗完成了。

箭头单元与Macbook的箭头单元类似，尽管按钮被垂直分割了一半，但它们仍配备了特殊的“斜面”，并且手指感觉很好，并且FN功能的组合是同类产品中最好的之一。当FN +箭头分别作为Page up / Page down和Home / End时，我也可以识别该选项，但是还有两个附加按钮，因此Acer的音量和亮度控制方法不在F1-F12区域中，而是在箭头上，在快速舒适的区域中，这是完全合理的。

同样，具有触感的键盘本身完全是“不是MacBook”，但是廉价笔记本电脑的主要缺点（中部变形和不确定的触发）被剥夺了。关键行程是正常的，不会停留在任何地方，不会停留，不会发酸等等。



LED指示完全不显眼，而且，打开和关闭的盖子都可以访问。通常，当您认为笔记本电脑价格便宜时，它与商业系列和所有程序员的需求都相去甚远-它是您自己的合适键盘解决方案。



触控板


廉价山毛榉树的第二个不幸是触摸面板。好吧，在这种情况下，结果比使用键盘要差一些。没有关于主要功能的抱怨，一切正常，点击，交换...但是这里捏到缩放和两指滚动正常工作。否则，一切都还不错。触控面板的面积不小，长宽比与显示器的长宽比相对应，以1对1模式工作，当然精度不高，但是具有自定义的光标加速和几天的练习时间（而且一周都不带笔记本电脑，所以工作将会变得非常舒适。



显示


另一个问题（仅现在，除高端市场以外的所有笔记本电脑）。我个人认为，每个营销人员都需要坐在他要“推向”市场的产品后面。因此，他本着“是的，在那儿需要正常的屏幕，我们会贴上1080p贴纸，然后人们才能播放”的精神为他的每项决定所苦。

笔记本电脑的TN矩阵的视角非常差，但是FullHD。我很乐意同意1440x900的分辨率，但要使用简单的IPS（或高质量TN）矩阵。由于不清楚的原因，制造商完全忘记了此选项。坦率地说，无论是1376×768，还是1920×1080或1680×1050，它的质量都是可悲的。

总的来说，我当然会夸大一点。您可以使用显示器，朴实的玩具，删除照片中的“垃圾”，观看互联网，随时随地观看电影……仅此而已。您甚至无需考虑任何色彩处理。它以相当不错的角度打开。也许，这是他唯一的优势。



亮度的存量也不令人印象深刻。最小值很好，晚上你的视力不会紧张。最大值（和屏幕的粗糙表面）允许您在明亮的光线下工作，但同时它们以惊人的速度消耗电池。

自治


笔记本电脑最具争议的方面之一。在“平衡”模式下，笔记本电脑可以安静地生活3.5-4小时，具体取决于活动的类型。如果您毫不客气地上网，请使用Skype，观看YouTube-将会有差不多的结果。

在“节能”模式下，我可以在火车上观看两小时的电影，该系列的三集（每次40分钟），但我仍然要多付30％的费用。就是说，作为“学生打字机”，山毛榉将很容易使用4对，如果您在休息时发现一个插座，并在其周围坐了半个小时，则足以容纳5-6个。顺便说一句，最近对GT有一个系列文章中的三件（量，作者：@dskosin 1，2，3上采用高科技技术进行抽象的主题）。阅读，这个话题很有趣。

规格和性能


一方面，我们面对制造商的期望，以节省仅出现的一切。另一方面，有必要使笔记本电脑处于需求和成功之中。



我的Acer内部有6 GB的RAM，一个Intel Core i5-5200U + Intel HD Graphics 5500和一个1 TB硬盘。制造商自豪地在所有标签上宣称5500可以“占用”高达3 GB的RAM作为视频内存。但是，尚不清楚她为什么需要那么多（这里的屏幕不是2560x1440，并且没人会在上面播放GTA 5），但是很好。



诚实地进行冷却，只有将奇迹包裹在毯子中并将其放在冬天的电池中，才可能使该奇迹过热。

处理器为双核，四线程。最大时钟频率为2.2 GHz，在短时间内，在TurboBoost模式下，它可以跳至2.7。打电话给笔记本电脑慢舌并不会转。浏览器运行，电影没有结结巴巴，简单的玩具（尤其是2D），例如无限美丽的《以撒的绑定：重生》或普通的HearthStone卡，完全没有问题。遗憾的是，更大的Aspire V3不再适用，但可以理解-笔记本电脑永远不会定位为游戏机，而“炫目”显示屏则无法使用图形。但是，后者是通过HDMI连接一些普通的显示器来处理的，在这种情况下，每次处理一张照片都非常舒适。

RAM可以扩展到16 GB（分两个模具输入，在这种情况下有4 + 2 GB的不同制造商：金士顿和三星），但这款笔记本电脑可以用头将所有六项任务全部解决。



我对音质和内置麦克风感到非常满意。通常，在廉价的笔记本电脑中，声音只是“是”，而网络摄像机只是在“起作用”，在这里音乐/电影听起来或多或少令人愉悦（尽管如此，但实际上这是一台笔记本电脑），与您在Skype上聊天的人很棒，并且我总是听到五点以上的声音。但是，Skype的认证已经包含在具有此特征的标签中，因此不足为奇，但是该死的不错。

优点缺点


如您所见，笔记本电脑颇具争议。如果我们将美元汇率变动的问题抛在一边，那么价格方面的Aspire V可能会满意：

  • 很棒的表演
  • 几乎无声的冷却；
  • 宜人的质量，身体材料；
  • 多汁生动的声音（用于笔记本电脑）；
  • 漂亮的键盘，尽管具有独特的功能。

以下要素并不十分令人鼓舞：
  • 4针音频插孔1 ;
  • 电池电量和自主权（完全取决于屏幕亮度）；

当然，最糟糕的情况是显示屏。没有视角，色彩表现很可耻，最大亮度设法耗尽了电池，同时又没有表现出出色的效果，并且在某些照明水平下，还表现出了可怕的PWM。但是vkoryachili FullHD。啊。

判决


Aspire V3-371适用于任何工作目的，它的轻便和紧凑性使其成为在道路上工作以及需要时经常进行频繁运输的理想选择。



作为一台“学生”笔记本电脑，它将轻松处理图形/ 3D模型，并从插座上记下笔记，放学后还会有一些娱乐活动-值得拥有。好吧，价格……什么也做不了，您什么也做不了，即使在今天，在Yulmart也可以花约3.4万卢布购买，但带有低分辨率显示屏（1376x768）的版本甚至更便宜。一般来说，六百美元-一切都很好。
