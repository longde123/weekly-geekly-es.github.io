Das BitCluster-Skript zeigt die Verbindungen zwischen Bitcoin-Wallets und Transaktionen an

Die Link-Analyse kann für große Veranstaltungsorte in Darkweb und im regulären



Bitcoin- Netzwerk durchgeführt werden - eine großartige Alternative zu regulärem Geld. Aber dieses System ist immer noch nicht vollständig anonym, wie viele glauben - Sie können den Benutzer identifizieren. Dafür müssen Sie mindestens einmal "aufleuchten". Es ist fast unmöglich, die Identität eines sehr vorsichtigen Benutzers preiszugeben. Sie können jedoch alle Benutzertransaktionen verfolgen, ohne ihn zu identifizieren.

Jetzt haben Bitcoin-Experten ein Tool eingeführt, mit dem Sie Transaktionen nach einer Reihe von Kriterien gruppieren können: Brieftasche, Handelsplattform oder einem bestimmten Benutzer. Dieses Tool identifiziert nicht die Identität des Benutzers, kann jedoch alle Details einer Reihe von Zahlungen anzeigen.

„Unser Ziel ist es zu zeigen, wie viele Daten über Benutzer des Bitcoin-Netzwerks gesammelt werden können und ob es möglich ist, gemeinsame Parameter für Brieftaschen zu finden, die anonym und nicht miteinander verbunden zu sein scheinen“, sagt einer der Forscher, David Décary-Hétu ) Das Team von Spezialisten, die ihr Software-Tool erstellt haben, arbeitet an der School of Criminology der University of Montreal.

Die Entwickler zeigten auf der NorthSec 2015-Konferenz die erste Version ihrer in Python geschriebenen Software. Das Tool heißt BitCluster. Es analysiert Transaktionen zwischen separaten Adressen und zeigt die erkannten Verbindungen an. Die Ergebnisse des Skripts können in Tabellenform heruntergeladen werden.


Screenshot der Transaktionsanalyseergebnisse für eines der Geschäfte in Darkweb

Anstatt Transaktionen an einer bestimmten Adresse zu verfolgen, verfolgt das Skript alle Übertragungen, die sich auf einen bestimmten Benutzer beziehen. Dies kann ein Drag-Dealer, ein Betrüger (der Ersteller einer Krypto-Ransomware) oder ein regulärer Benutzer des Systems sein. Fachleute werden auf der HOPE-Veranstaltung neue Forschungsergebnisse zur Analyse von Operationen im Darkweb mit BitCluster vorstellen.

In den meisten Fällen stellen die Entwickler von Krypto-Ransomware eine Brieftasche zur Verfügung, an die das Opfer Geld senden muss, um einen Entschlüsselungsschlüssel für seine Dateien zu erhalten. Angreifer können somit Zahlungen einzelner Benutzer verfolgen. Wenn die Ersteller der Crypto Ransomware alle empfangenen Crypto-Münzen in eine Brieftasche stecken, können Sie sehen, wie viel wann und wer die erste Transaktion gesendet hat. Sie können dies ohne Skript herausfinden, aber es ermöglicht Ihnen, die erforderlichen Vorgänge schnell auszuführen.

Darüber hinaus kann BitCluster Gruppen miteinander verbundener Brieftaschen identifizieren, die zum Ausführen von Transaktionen an bestimmten Standorten verwendet werden. Beispielsweise können Brieftaschen von Standorten identifiziert werden, auf denen Kundengelder während einer Transaktion mit einem Verkäufer oder Anbieter eines bestimmten Dienstes gespeichert werden. Die Forscher haben die Analyse für die ursprüngliche Seidenstraße bereits abgeschlossen, und es wurden auch Arbeiten durchgeführt, um neue Märkte zu erkunden, darunter AlphaBay und Nucleus. Anhand der verfügbaren Informationen zeigt das Skript an, wie viele Personen zu einem bestimmten Zeitpunkt Zahlungen leisten, wie viele Transaktionen abgeschlossen sind und für welchen Betrag. Die Größe der durchschnittlichen Transaktion wird ebenfalls angezeigt.



Frühere Studien haben auch die Möglichkeit gezeigt, Transaktionen mit bestimmten Benutzern der Website zu verknüpfen, wenn der Zeitpunkt der Veröffentlichung von Feedback zu einer Transaktion untersucht wird. Diese Arbeit wurde auch mit BitCluster durchgeführt.

In Kürze werden Entwickler ihr Skript öffentlich zugänglich machen. Das Team möchte dieses Tool nicht zu Hause hosten, da es sehr ressourcenintensiv ist und leistungsstarke Server erforderlich sind, um eine große Datenmenge zu analysieren (z. B. auf der Mt. Gox-Site).

Das Skript kann nicht nur von Forschern verwendet werden. Strafverfolgungsbeamte können mit ihm arbeiten. „Unser Ziel ist es nicht, Opfern oder Strafverfolgungsbehörden zu helfen. Dies ist ein offenes Tool, mit dem die Benutzer besser verstehen können, wie das Bitcoin-Netzwerk funktioniert und was in diesem Netzwerk geschieht “, sagt einer der Ersteller des Skripts.

Das Tool kann einige Arten von Vorgängen nicht ausführen. Beispielsweise gibt es Dienste, die eine Transaktion gemeinsam nutzen, indem sie an eine benutzerdefinierte Auswahl von Brieftaschen gesendet werden, die in keiner Weise miteinander in Beziehung stehen. Eine weitere Möglichkeit, die Nachverfolgung Ihrer Zahlungen zu vermeiden, besteht darin, für jede Transaktion eine neue Brieftasche zu erstellen. Das Skript kann solche Transaktionen nicht binden. Allgemeine Brieftaschen und Transaktionen werden nur angezeigt, wenn Zahlungen nach der Aufteilung dennoch auf ein einziges Konto gesendet werden.
