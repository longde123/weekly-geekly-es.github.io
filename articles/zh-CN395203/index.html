2016年6月发布的超级计算机500强评级，中国推出了100台PFlops超级计算机

今天，根据Linpack测试发布了第47位世界 500强超级计算机评级。与之前的6个版本不同，该榜单的领导者发生了变化，中国双威TaihuLight超级计算机（Linpack测试结果为93 petaflops，理论性能为125.4 petaflops）名列榜首。其性能是该榜单的前任中国天河2号的三倍左右。



新型超级计算机是由中国国家并行计算机工程与技术研究中心（国家并行计算机工程与技术研究中心）开发的。有在国家超级计算中心的系统无锡市中国东部的江苏省。



该超级计算机基于神威家族的新中文处理器-SW26010，具有原始的64位RISC架构，大概是使用28 nm技术制造的。每个处理器均配备260个内核，工作频率为1.45 GHz，性能为3.06 teraflops。

该处理器是在上海高性能集成电路设计中心开发的。该处理器由4个相似的核心模块组成，通过芯片上的集成网络连接。每个模块在一个8x8阵列中具有一个控制核心（管理处理元件，MPE），一个DDR3存储器控制器（128位）和64个处理核心（计算处理元件，CPE）。两种类型的内核都具有微结构，并具有出色的命令执行能力。控制MPE内核支持操作系统和用户代码的执行，使用264位向量操作，在第一级指令和数据中包含32 KB的缓存，在第二级中包含256 KB的缓存。计算CPE-内核可以执行仅与264位向量的用户代码，大多采用16K高速缓存的指令和64K临时存储器（便笺式存储器）。 4个核心块中的每一个都可以访问8 GB的DDR3-2133 RAM，因此该节点具有32 GB的RAM，总带宽高达136.5 GB / s。

MPE内核上的SIMD计算性能是每个周期16个双精度浮点操作（64位），而CPE内核上则为每个周期8个操作。 MPE内核在1.45 GHz处的总体性能可以达到23.2千兆位，而CPE内核-11.6 gflops。

超级计算机总共使用了超过1060万个内核，包括40个计算机架中的40,960个单处理器节点。每个机架有4个超级节点；该超级节点由32个模块组成，每个模块有8个节点。模块是水冷的。关于超级计算机主网络的细节很少，众所周知，每个SW26010芯片都具有到三级网络“ Sunway Network”的PCI Express 3（16x）连接。网络直径-7，“对等带宽”-70 TB / s。 Dongarra说正在使用主机通道适配器和公司交换机Mellanox，具有大约12 GB / s（100 Gb / s）的信道带宽和大约1μs的延迟。

HPL测试（Linpack）的计算效率为理论性能的74％。同时，在更复杂的HPCG测试中，系统仅显示峰值水平的0.3％（某些系统达到 1-3％），这表明内存相对较慢且网络带宽不足。对于SW26010，峰值触发器与内存带宽的比率为22.4 flops /字节（作为比较，Intel Knights Landing有7.2 flops /字节）。 Dongarra还指出，该系统的RAM相对较少，只有1.3 PB（天河2号只有1.4 PB，美国泰坦，在Top500-0.71 PB中已经排名第三。

在HPL测试期间，超级计算机的平均功耗为15.3 MW（在天河2号时略低于17 MW），最大-略低于18 MW。据Dongarra称，能源效率为每瓦6吉普托（考虑到处理器，内存和网络消耗）。新的超级计算机采取第三等级green500.org（更节能RIKEN受不得 Ç6.6 GFLOPS / W和RIKEN小月 6.2 GFLOPS / W）。

超级计算机操作系统Sunway Raise OS 2.0.5基于Linux。向用户提供了C / C ++，Fortran编译器，自动矢量化实用程序和数学库。 Sunway OpenACC实用程序提供对OpenACC 2.0 标准的支持，以简化多核处理器的编程。

制造一台超级计算机的成本为18亿元人民币，约合2.7亿美元。
