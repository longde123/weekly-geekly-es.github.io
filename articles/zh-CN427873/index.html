现代技术学院的3D建筑打印机



你好 我们谈论的是现代技术学院以苏联M.F.英雄的名字命名的建筑3D打印机的调试。 帕诺瓦。 这是一所建筑学院，从事这一领域的专家培训，并参加了莫斯科世界锦标赛俄罗斯技巧锦标赛。


什么是3D打印机



这是该学院购买的第二台Spetsavia建筑打印机，用于混凝土3D打印。 前一个被送到垃圾填埋场工作。 这是对S-2020模型的修改，已按订单生产且未组装。

打印机在框架尺寸和打印区域方面与标准型号不同。



特点

  • 工作区域：2500 x 1500 x 800毫米。
  • 外部尺寸：3000 x 2000 x 2100毫米
  • 驱动：带正齿轮的步进电机。
  • 打印头进纸类型：螺旋钻。
  • 层：10-15 x 30-40毫米。
  • 生产率：每小时最高0.5立方米。
  • 材料：具有或不具有防冻剂和增塑剂的水泥组合物，石膏混合物（包括硬化的防潮剂）。
  • 净重：560公斤。
  • 毛重：960公斤。
  • 电源：220V，50 Hz，1600瓦。
  • 工作条件：室内，在5至40°C，空气湿度不超过60％。

该打印机进入了六个抽屉。 两人卸下并转移了他们。


单独的组件重200公斤，需要吊起才能卸下它们。

3D打印机组件



打印机框架由几个组件组成。



沿垂直轴的导轨安装在200公斤的轴上，并在其上安装有用于溶液的料斗挤出机。



将混合物装入漏斗的漏斗中。 在内部，螺杆通过旋转挤出溶液。



底座的侧面是主要的承重元件，每个重80公斤。



我们用三个矩形管连接侧壁。



他们悬挂了轴；步进电机位于轴上。 该运动通过线性齿轮传动装置进行。 然后固定黄色金属面板。



站立在地板上的零件是沿Z轴的导轨框架。



他们悬挂在轨道上，并连接了一条链，该链传递了来自步进电机的力以垂直移动料斗。



我们用料斗将料斗吊起来并连接电源线。



完全组装打印机大约需要6个小时，需要两名员工。

软件，项目准备



该打印机可与Mach3一起使用，并从.dxf和.dvg文件进行打印。 您可以将任何g代码下载到其中。 文件准备工作在Sheetcam中完成。 您可以使用AutoCAD中的项目文件。

列印



混凝土混合物由特殊设计的挤出机挤出。 它像家用3D打印机中的塑料一样分层堆叠。


大学生揉搓溶液并将其装入料斗。

对于在打印机上进行打印，一个操作员就足够了，他还可以手动或使用特殊的装载机将混合物装入料斗。

如在绞肉机中那样，通过旋转螺杆进料混合物。 螺杆电动机速度在程序中设置。 将混凝土装入料斗，然后将其脱气一会儿，夯实一下即可打印。

3D打印机用于什么用途？



该打印机用于研究在建筑中使用3D打印的方法。 它用于印刷小型建筑形式-长凳，架子，浮雕元素。



打印机旁边的细节是工作台的一部分。 因此，它是完整的。



还有其他选择。







当然，不仅是商店。







打印机还可以创建整体结构，但是打印外壁更为有效，外壁的内部空间可以填充溶液或填充隔热材料。



另一个应用示例：在叶卡捷琳堡附近的Artyomovsky，Spetsavia打印机打印了《权力的游戏》中温特费尔城堡之一塔的副本。



该塔被印在当地混凝土厂的地盘上，并被设计成一个岗亭。

利弊



优点：


  • 必不可少的视觉辅助；
  • 您可以打印小型建筑形式以供实际使用。
  • 易于安装，易于配置，只需打印；
  • 比进口类似物便宜；

缺点


  • 需要特殊的设备来卸载打印机，至少需要两个人来安装它。
