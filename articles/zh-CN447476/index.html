在Sublime Text中设置LaTeX


以下是安装Sublime Text文本编辑器的指南，然后添加了使用LaTeX计算机布局系统的功能。 作为奖励，我们将教他理解朱莉娅的语言。


安装MiKTeX


  1. 在网站上，单击“ 所有下载”选项卡
  2. 根据系统的位深度选择“ Net Installer” 。
  3. 单击下载，然后等待在线安装程序下载。
  4. 仔细阅读用户协议后，选择“ 下载MiKTeX”复选框
  5. 使用代理时，在“ 连接设置”中输入您的附件...
  6. 在下一步中，选择Complete MiKTeX
  7. 选择服务器（最好是俄罗斯顶级服务器之一）并指定目录，然后等待必要的文件加载
  8. 再次执行步骤4，但这一次选择安装MiKTeX
  9. 安装完成后，通过“开始”启动TeXworks 。
  10. 打开其中的任何.tex文档或创建一个新的
  11. 从下拉菜单中选择pdfLaTeX ，然后单击绿色圆圈中的相邻三角形

这将编译并创建PDF文件。 使用TeXworks，您可以开始全面的工作，例如，使用LaTeX，您可以非常有效地收集文凭或学位论文（ 现成的模板 ）


通过Sublime Text使用LaTeX更加方便。 除了方便的突出显示和自动填充之外，您还可以在不编译的情况下查看图像或键入的公式，即在键入过程中查看结果。 顺便说一下，您可以在Sublime Text中输入（如果添加了适当的插件，然后执行）许多编程语言和标记的程序代码：C / C ++，Java，MATLAB，PHP，HTML ...


安装Sublime Text 3


  1. 下载并安装Sublime Text 3
  2. 下载并安装Sumatra PDF 。 这个轻量级程序允许您查看PDF文档，DJVU，FB2等。
  3. 启动Sublime Text之后，按Ctrl + shift + p打开命令行，然后开始键入Package Control：Install Package ，然后在出现时单击并等待与存储库的连接。
  4. 如果由于某种原因没有软件包控制，请按* ctrl +` （（））打开控制台，然后在parseltang上复制该拼写（可以在Package Control插件页面上找到），按Enter并重新启动Sublime Text。
  5. 在插件列表中，输入LaTeXTools，然后在列表中显示时单击并等待安装完成。
  6. 接下来，我们启动默认设置： 首选项/软件包设置/ LaTeXTools /将用户设置重置为默认 ，然后在此处检查系统
    如果所有标签均为绿色，则Sublime Text与Sumatra和LaTeX成为了朋友，现在您可以创建或编辑tex文档。 当您按ctrl + B时​​，将进行翻译，并且如果没有错误，则会打开分页的PDF。 Ctrl + Shift + B-选择装配类型。

有关LaTeXTools的更多信息


在线查找LaTeX手册对您来说很容易（例如： Tutorial ， Wiki ）。 在将命令存储在内存中之前，使用联机公式编辑器也将很有用。


朱莉娅的崇高文字
