Bot para VK en Python con MySQL en una hora, parte 1

Hay muchos tutoriales en Internet sobre cómo escribir un bot para Vkontakte, pero todos tienen inconvenientes importantes para los principiantes. Proporcionan una base muy mínima debido a que en el futuro será difícil para los principiantes comprender cómo funcionan ciertas cosas.

En este tutorial quiero mostrar cómo hacer el bot más simple con una funcionalidad básica bastante grande.

Entonces comencemos.

Que necesitamos


  • Python desde la versión 2.7 (Todo el código se probó en las versiones 2.7.16 y 3.7.3)
  • Módulo PyMySQL
  • Módulo vk_api
  • Módulo de solicitudes
  • Grupo VKontakte (¡Atención! El bot funcionará precisamente desde el lado del grupo, y no desde el lado de la cuenta de la persona)
  • Base de datos (incluidos los datos del usuario, etc.)

Conexión


Primero, conectaremos todas las bibliotecas: 

import vk_api from vk_api.utils import get_random_id from from vk_api.bot_longpoll import VkBotLongPoll, VkBotEventType import pymysql.cursors import requests

Ahora puede comenzar a configurar el bot y la base de datos.

En mi caso, todos los datos están en el archivo config.py. Puedes prescribirlos en cualquier lugar. No afectará la funcionalidad de ninguna manera.

Conexión de base de datos


Para hacer esto, cree una función usando el módulo PyMySQL: 

 def get_connection(): connection = pymysql.connect(host='you_host', user='you_user', password='you_password', db='you_db' charset='utf8mb4', cursorclass=mymysql.cursors.DictCursor) return connection

Esta función devolverá datos para seguir trabajando con la base de datos. En consecuencia, sustituimos nuestros datos donde vemos "usted_".

Trabajar con un bot


Agreguemos una función bot para responder a cualquiera de nuestros mensajes con nuestros mensajes. Para hacer esto, después de la última línea escribimos esto: 

 if event.type == VkBotEventType.MESSAGE_NEW: #   #       if event.obj.text != '': #       if event.from_user: vk.messages.send( user_id=event.obj.from_id, random_id=get_random_id(), message=event.obj.text)

Ahora sobre el método vk.messages.send (). Este es el método estándar del módulo vk_api. Puede leer sobre los métodos en la documentación oficial de Vkontakte (el enlace estará debajo).

Le permite enviar mensajes a los usuarios.

El argumento user_id indica a qué cuenta se debe enviar este mensaje. En nuestro caso, event.obj.from_id apunta a la cuenta que nos acaba de enviar un mensaje. Es decir, enviamos un mensaje a la persona que nos lo envió.
Este argumento es obligatorio!

El argumento random_id es necesario para que Vkontakte no envíe mensajes 2 veces. Este argumento es obligatorio!

El argumento del mensaje en realidad indica qué mensaje debemos enviar. Event.obj.text almacena información sobre el texto del mensaje que recibimos. Este argumento también es obligatorio.

Puede leer más sobre los argumentos de este método en la página oficial de Vkontakte.

Lanzamiento


Ahora nuestro bot está listo y podemos comenzarlo. Después de comenzar, puede escribir de forma segura en el bot y ver el resultado.

Resumen


Esta parte 1 ha terminado. Aprendimos a encender el bot. Configurarlo, así como enviar mensajes.
