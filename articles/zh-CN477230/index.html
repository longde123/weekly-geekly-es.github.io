教育软件的历史：第一款个人计算机，教育游戏和学生软件

上次我们讨论了自动化学习过程的尝试如何导致60年代当时非常先进的PLATO系统的出现。 对她来说，开发了许多不同学科的培训课程。 但是，PLATO有一个缺点-只有具有特殊终端的大学生才能使用培训材料。

随着个人计算机的出现，这种情况已经改变。 因此，教育软件运用于所有大学，学校和家庭。 我们继续讲故事。


照片： Matthew Pearce / CC BY

电脑革命


导致个人计算机革命的设备是基于Intel 8080微处理器的Altair 8800 ，为此计算机设计的总线已成为后来计算机的事实上的标准。 Altair由工程师Henry Henry Roberts在1975年为MITS开发。 尽管存在许多缺陷-汽车没有键盘或显示器-但该公司在第一个月就卖出了数千台设备。 Altair 8800的成功为其他PC铺平了道路。

1977年，Commodore以其Commodore PET 2001进入了市场。这台重11公斤的金属外壳计算机已经装有分辨率为40x25字符的显示器和一个输入设备。 同年，Apple Computer推出了Apple II。 他收到了彩色显示屏，它是BASIC语言的内置解释器，可以再现声音。 Apple II成为面向普通用户的PC，因此不仅大学的技术专家与之合作，而且学校的教师也与之合作。 这刺激了负担得起的教育软件的开发。

在某个时候，美国老师安·麦考密克（Ann McCormick）担心某些青少年会非常犹豫和缓慢地阅读。 因此，她决定制定一种新的教学方法。 1979年，麦考密克获得了一笔赠款，并从Apple教育基金会获得了Apple II。 她与斯坦福大学心理学博士Teri Perl和Atari程序员Joseph Warren共同创立了The Learning Company 。 他们一起开始为学生开发教育软件。

到1984年，The Learning Company已发行了15款针对儿童的教育游戏。 例如，学生在《洛矶靴》中解决了各种逻辑难题。 她在软件发行商行业协会的排名中获得第一名。 还有一个Reader Rabbit教授阅读和写作。 十年来，它已售出1400万本。


到1995年，该公司的收入达到了5,320万美元，《儿童技术评论》的编辑沃伦·巴克利特纳甚至称 The Learning Company为“学习的圣杯”。 据他说，正是安·麦考密克（Ann McCormick）团队的工作帮助教师了解了计算机的强大教育工具。

还有谁呢？


在80年代上半叶，学习公司并不是唯一的教育软件开发商。 益智游戏由 Optimum Resource，Daystar Learning Corporation，Sierra On-Line和其他小型公司发行。 但是The Learning Company的成功只能重现Brøderbund，它是由Doug和Gary Carlston兄弟（Doug和Gary Carlston）创立的。

公司曾一次开发游戏，也许他们最著名的项目是波斯王子。 但是不久，兄弟俩将注意力转向了教育产品。 他们的作品包括：詹姆斯发现了数学和数学讲习班，以教授基本的数学；惊人的写作机，用于教授阅读和语法；以及三重：日本文化的故事-一种以儿童娱乐性故事形式提供的日本历史课程。

教师参与了应用程序的开发，他们还使用该软件制定了教学计划。 该公司定期在学校举行研讨会，以普及计算机培训，为用户发布纸质手册，并为教育机构提供优惠计划。 例如，以《 Mieko：日本文化的故事》的通常价格为179.95美元，该学校版本的价格几乎是其价格的一半-89.95美元。

到1991年，Brøderbund占领了美国教育软件市场的四分之一。 公司的成功吸引了The Learning Company的注意，后者以4.2亿美元的价格收购了竞争对手。

学生软件


大学教育并没有停留在计算机革命的一边。 1982年，麻省理工学院为工程专业的学生购买了数十台PC。 一年后， 雅典娜项目在IBM的支持下在大学启动。 该公司为大学提供了价值数百万美元的计算机，并为其程序员提供了开发教育软件的机会。 所有专业的学生都可以使用新技术，并且计算机网络开始在校园中工作。

80年代末期，基于UNIX的教育基础设施出现在MIT，大学专家为其他大学开发了程序。 最成功的系统之一是公认的自然科学教学综合系统-大学工作人员不仅编写了计算机课程，而且还启动了测试学生知识的系统。

雅典娜成为该大学大规模使用计算机和软件的首次经验，并成为其他教育机构类似项目的典范。

教育生态系统发展


八十年代初期，企业家开始对教育软件表现出兴趣。 保罗·艾伦（Paul Allen）因与比尔·盖茨（Bill Gates）的分歧而于1983年离开微软，创立了Asymetrix学习系统。 他在那里开发了一个用于ToolBook学习内容的环境。 该系统允许创建各种多媒体产品：课程，测试知识和技能的应用程序，演示文稿和参考资料。 2001年，ToolBook被公认为是最好的在线电子学习工具之一。

远程学习生态系统开始发展。 先锋计划是FirstClass计划，该计划是由贝尔北部研究人员-史蒂夫·阿斯伯里（Steve Asbury），约翰·阿斯伯里（Jon Asbury）和斯科特·韦尔奇（Scott Welch）共同开发的。 该软件包包括用于处理电子邮件，文件共享，聊天，面向教师，学生和家长的会议的工具。 该系统仍在使用和更新（它是OpenTex产品组合的一部分）-全球有3000所教育机构和900万用户连接到该系统。

互联网在90年代的激增引发了下一次教育革命。 教育软件的开发继续并获得了新的发展：1997年，“交互式学习环境”（Interactive Learning Network，Interactive Learning Network）的概念诞生了。
