Estudiamos el principio del porcentaje de trabajo para la propiedad de relleno superior utilizando el ejemplo de la tarea "Diseño de marcador de posición para imágenes"

Cuando pasamos por entrevistas, a menudo nos hacen preguntas sobre el diseño. A menudo se reducen a enumerar los valores de la propiedad de visualización o cómo centrar los elementos. Esto me molestó francamente, y quería desarrollar mis propias tareas, con la ayuda de las cuales puedo probar las habilidades de diseño, y no cómo memorizó los valores de las propiedades.


Descripción de la tarea


A menudo, cuando leo artículos en Internet, me encuentro con el siguiente problema: el texto comienza a saltar cuando se carga la imagen.



Una solución a este problema es agregar un marcador de posición para la imagen, en el cual la relación de ancho y alto repetirá la relación de aspecto de la imagen.



Comenzaré su implementación con marcado: 


<div class="post"> <img src="example.jpg" class="post__img" alt="   "> </div>

Para establecer dimensiones para un marcador de posición, debe conocer la relación de aspecto de la imagen. Por ejemplo, usaré el tamaño de imagen 1920x1080px.


Pero si muestra una imagen de este tamaño, en la mayoría de las pantallas se mostrará con desplazamiento horizontal. Por lo tanto, estableceré el elemento .post en un ancho máximo de 640 px, y el ancho de la imagen se ajustará a él. 


 .post { max-width: 640px; } .post__img { max-width: 100%; }

Agregaré un marcador de posición a la página usando el pseudo-elemento anterior. También debe recordar colocar la imagen encima. 


 .post { max-width: 640px; position: relative; } .post::before { content: ""; } .post__img { max-width: 100%; position: absolute; top: 0; left: 0; }

Ahora necesita el ancho del marcador de posición para repetir el ancho de la imagen. Anteriormente, hice que la imagen ocupara el 100% del ancho del elemento padre .post. Haré lo mismo para el pseudo-elemento antes agregando display: block. 


 .post { max-width: 640px; position: relative; } .post::before { content: ""; display: block; } .post__img { max-width: 100%; position: absolute; top: 0; left: 0; }

Amigos, en esto terminé mi parte del trabajo, y la suya permaneció. Necesita calcular la altura del pseudo-elemento anterior.


Para hacer esto, use la propiedad padding-top con un valor porcentual. Para hacer esto correctamente, debe conocer la peculiaridad de calcular el interés en la propiedad padding-top. Y también recuerde que los tamaños de imagen son 1920x1080px. 


 .post { max-width: 640px; position: relative; } .post::before { content: ""; display: block; /*    padding-top    % */ } .post__img { max-width: 100%; position: absolute; top: 0; left: 0; }

Espero que intentes resolver este problema tú mismo. Pero si surgen dificultades, aquí está mi solución.


Solución


Las propiedades padding-top y padding-bottom tienen una característica muy útil. Si establece los valores en porcentaje, se calcularán a partir del ancho del bloque principal.


En consecuencia, sabiendo esto y recordando el tamaño de la imagen (1920x1080px), puede calcular la parte superior del relleno para el marcador de posición utilizando la proporción: 


 padding-top = ( * 100%) /  = (1080 * 100% ) / 1920 = 56.25%

donde Chi y Vi son el ancho y la altura de la imagen.


Queda por agregar un valor para padding-top: 


 .post { max-width: 640px; position: relative; } .post::before { content: ""; display: block; padding-top: 56.25%; } .post__img { max-width: 100%; position: absolute; top: 0; left: 0; }
