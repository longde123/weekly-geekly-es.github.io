新型Tesla Model 3的未来派用户界面

你好，Giktayms！ 我向您介绍了汤姆·约翰逊 （ Tom Johnson）撰写的特斯拉Model 3 UI对未来愿景的揭示内容。


如果您对设计了解很多，或者对汽车非常了解，那么本文适合您。 它将考虑控制面板元件的主要功能，以及新型特斯拉Model 3的用户界面-最新推出的电动汽车已为更广泛的社会提供，它配备了触摸屏。 新的用户界面设计可以说明特斯拉工程师如何看待自动驾驶汽车的未来。

如果您对此主题感兴趣， 请查看我专门为您准备的Figma文件 ，并通过链接找到我制作的原型 。



从一开始，我就关注着围绕特斯拉Model 3进行的所有活动。去年，我终于设法试用了Model S，此后，我对驾驶这款汽车的简易性以及对每个动作的反应能力感到惊讶司机。 不用说，我忍不住爱上了特斯拉，立即对它“价格适中”的版本的想法感到兴奋，该版本的特性不亚于我尝试过的模型。 像许多特斯拉车迷一样，我关注每条特斯拉Model 3新闻，并翻阅了无数网页，以寻找有关最小细节的信息，甚至连这辆汽车的最小细节也将如何工作。

计划在此模型中实施的工程师的想法真的很酷，但是我很难相信他们会完全放弃新特斯拉的传统仪表板。 这听起来很酷，但对营销人员却不那么有用，对买家而言更是如此。 然后我合理地想到：“来吧，这些只是想法，您永远都不知道自己能想到什么。”

好吧...在这里我没猜到：这辆车向公众展示了，传统的仪表板甚至都没有闻到...

还有一个速度开关...

和空调通风口...

甚至手套箱的把手...


现在，所有这些元素都位于触摸屏上，恰好位于汽车中间。 最重要的是：工程师们甚至都没有想到隐藏突出于汽车中部的尴尬屏幕，这表明中指直指整个汽车世界。 因此，所有对此方法的怀疑者，以及提倡熟悉的仪表板不可侵犯的其他汽车公司，都获得了“ F * ck you”的美誉。 最初，特斯拉放弃了这样的发动机，但这对她的工程师和设计师来说似乎还不够，他们决定继续本着同样的精神，直到整个有一个多世纪历史的汽车行业都颤抖了。

实际上，一切都有些不同


关于即将发布的新特斯拉车型的消息令我敬畏。 但是，这次，我带着许多不满，不屑一顾和偏见的感叹声支持那些怀疑论者。

Model 3具有四个车轮，外观上看起来像汽车，但这就是它与熟悉的汽车的相似之处。

我真的很想知道特斯拉工程师是如何设法将汽车的所有控件隐藏在玻璃显示屏面板后面的。 因此，我从字面上开始研究Internet，以搜索有关新模型的用户界面设计的相关信息。 值得承认的是，这花了我一些时间，但是最后我很幸运地偶然发现了设计师Andrew Goodlad提出的一个有趣的项目。 然后，我决定他能够使用网络上发布的信息以及专用于新车界面的YouTube精彩视频，重建即将进入市场的Tesla Model 3的用户界面。 这个家伙创建了一个真实的原型界面及其相关的所有内容。

Model 3 UI设计的新增功能


特斯拉Model 3是一辆汽车，其创造者最不关心驾驶员的事，而这是有意为之的。

让我们弄清楚……汽车没有任何把手，尽管它甚至有方向盘（也要感谢）。 为了打开雨刮器，您需要按下屏幕上的按钮（作者注*：为此，仍然在汽车中提供了一个操纵杆，但是仍然可以使用屏幕上的按钮来调节雨刮器的速度）。 再次，您将必须按屏幕上的按钮才能打开手套箱或施加手制动（*作者注：应自动激活手制动，但是，可以相信，使用该设置，您可以设置按下时可以施加的制动按钮）。 对司机来说不是很方便吗？ 现在，在驾驶过程中，驾驶员不再依赖于自己经过多年磨练的反应和肌肉记忆。

您可以决定新特斯拉的工程师进行了此类更改，以使汽车看起来更先进，更危险或不像其他任何人一样……但是，事实并非如此：仅仅是新特斯拉的工程师决定在可预见的将来为人们准备汽车驾驶没有司机的帮助。

这辆车是为方便驾驶而创建的。


至于用户界面，我注意到的第一件事就是显示屏的分辨率。

与Model S和Model X一样，新的Tesla可以安装垂直屏幕，但Model 3的宽高比为8：5，分辨率为1920x1200像素。 您可能会问，那么下一步呢？ 这是什么：

  • 这是HDTV家伙！
  • 底部（永久）控件的高度为120像素。
  • 1200-120 = 1080像素。
  • 1920x1080。


正如新的特斯拉所暗示的那样，高清电视首先创建了中央控制屏幕，以观看其上的东西，而不是为了方便驾驶。 因此可以肯定地说，Model 3是未来的救世主，汽车将在没有驾驶员帮助的情况下行驶。 这不仅是具有异常控制系统的汽车，还是真正的带轮客厅。 考虑到技术的逐步发展，这也许是一个过早的陈述，但是不久之后，观看电影的最佳地点可能就是您最喜欢的铁马沙龙。

结论


我最近查看了史蒂夫·乔布斯（Steve Jobs）提出的原始iPhone的演示文稿，并为iPhone与Model 3的明显相似之处感到震惊。

它们都没有按钮。

这就是我发生的事情：



然后，乔布斯向听众介绍了对于智能设备行业来说，物理按钮的存在是过去的遗物。 我们看到的是：如果现在有人想推出带有全键盘按钮的新设备，那简直就是开玩笑。

那为什么我们不与汽车行业划清界限呢？ 为什么我们（或者我个人）在我们面前看到一个触摸屏，而不是通常的控制面板，甚至都不想想象这将成为一种普遍接受的方法。 苹果不是通过放弃标准键盘来做类似的事情吗？
他们说，您可以争论：那是一部电话，这是一部汽车-范围非常不同。

但是，包括驱动软件在内的任何软件都可以更新，改进。 而对于物理按钮，几乎不可能想到任何有效的更改：用一个按钮替换另一个按钮或更新其外观似乎没有什么用处。 因此，用成熟的控制屏幕替换按钮和其他物理汽车控件可能不是一个疯狂的主意。
