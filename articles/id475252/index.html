Ambil sekarang Microsoft Office. Ini juga alat, sangat kompleks, membutuhkan pelatihan, tetapi hanya sarana untuk mencapai beberapa tujuan. Desain Klasik 2003 adalah cinta seumur hidup. Mouse itu sendiri menemukan tombol kanan. Pada 2007, saya memasang keajaiban baru dan kemudian menghancurkannya dengan sangat cepat. Mereka tidak hanya mengubah desain, tetapi juga tata letaknya. Tidak ada keinginan untuk mencari (membiasakan diri) dengan hal yang benar. Sebagian besar dari semua saya bekerja di Excel dengan fungsi lembar kerja, jadi saya melewatkan sedikit JIKA KESALAHAN, keuntungannya jelas. Saya kembali ke tab hanya pada tahun 2010. Puji Tuhan, pada tahun 2013 mereka menambahkan sebuah tombol. Apa yang ingin Anda lakukan? Saya kadang-kadang menggunakannya karena saya tidak tahu di mana yang saya butuhkan. Saya masih tidak mengerti mengapa itu perlu untuk mengubah antarmuka pengguna dan lokasi.

Baru-baru ini, diperlukan untuk membuat antarmuka yang identik pada mesin yang berbeda, dan beberapa fitur menarik terungkap. Saya ingin mengatakan bahwa saya sedang berbicara tentang akun lokal, tentang surat di mail.ru dan cloud di sana (100GB, gratis).

Pastikan untuk menghapus kantor sebelumnya dan juga menginstal program tambahan untuk itu. Belum lagi sangat menyebalkan. Tetapi mereka tidak dapat me-mount kode untuk ini di installer sendiri? Selain itu, kantor lama mungkin memiliki pengaturan pribadi. Sangat menyenangkan memilikinya secara otomatis di yang baru. Outlook juga tidak mudah. Microsoft menyediakan kemampuan untuk menyinkronkan beberapa folder, tetapi tidak semua (daftar pengaturan email sampah tidak disinkronkan) pada perangkat yang berbeda jika Anda memiliki surat Exchange atau Outlook.com. Selain itu, sejauh yang saya tahu, di VBA tidak ada cara untuk melakukan ini setidaknya dalam mode semi-otomatis: ekspor / impor ke file dengan makro.

Setelah pengalaman ini, saya membuat file dengan instruksi yang cukup rinci untuk mengatur panel, tampilan dan hal-hal lain, serta file ekspor data. Saya akan menggunakannya minggu depan. Xiaomi tidak membenarkan bepergian, saya akan menyiapkan laptop baru.
