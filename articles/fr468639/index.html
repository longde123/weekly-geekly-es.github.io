Nouveautés des consoles Web 2019

En 2016, nous avons publié l'article traduit, «Le guide complet des consoles Web 2016: cPanel, Plesk, ISPmanager et autres ». Il est temps de mettre à jour les informations sur ces 17 panneaux de contrôle. Lisez de brèves descriptions des panneaux eux-mêmes et de leurs nouvelles fonctionnalités.


cpanel


La première console Web multifonction la plus populaire au monde, une norme de l'industrie. Il est utilisé à la fois par les propriétaires de sites (en tant que panneau de contrôle) et les fournisseurs d'hébergement (en tant qu'outil d'administration de Web Host Manager, WHM). Interface intuitive, ne nécessite pas de formation, multilingue. Il y a des instructions vidéo.

Java de base: Perl, PHP
Système d'exploitation pris en charge: Red Hat Enterprise Linux (RHEL), CentOS, CloudLinux. Il est possible de prendre en charge Windows via la virtualisation ou via le panneau Enkompass des mêmes développeurs.


cpanel


Whm

Nouveau

Les développeurs tentent constamment d'accélérer le panel et, en général, travaillent activement à son amélioration, en fonction des demandes des clients. Ainsi, dans la 82e version actuelle, l'installation est terminée en 3 minutes. Le temps de mise à jour de CPanel et WHM a été amélioré: à partir de l'avant-dernier, 80e, il se termine en trois minutes et de la précédente en huit. En 2019, l'espace disque requis pour le programme d'installation cPanel et WHM a été réduit de 10%. Il y a: compatibilité avec PCI; sauvegarde et restauration automatiques; un outil qui vous permet de mettre sur liste noire et liste blanche les comptes, les adresses IP et des pays entiers; Certificat SSL gratuit pour chaque site Web. Vous pouvez maintenant inclure un fichier dans un autre (ajouter une configuration). En général, pendant ce temps, le travail de cPanel & WHM a été accéléré de 90%, les ressources serveur nécessaires ont été réduites de 30%.

En avril 2019, les développeurs ont annoncé gentiment, peut-être la demande de mise à jour de la fonctionnalité cPanel la plus demandée - l'ajout du serveur Web NGINX comme alternative à Apache. La fonction fonctionne dans un format expérimental. Documentation de mise à jour officielle.

Les prix

Dépend du niveau du compte: Solo 15 $, Admin 20 $, Professionnel 30 $, Premier 45 $ par mois. Période d'essai gratuite. Il existe des programmes d'affiliation. Incident de support technique prioritaire de 65 $.

Plesk


Un panneau de contrôle préféré des grands hébergeurs, qui sera compréhensible même pour un novice. Une seule interface pratique avec laquelle vous pouvez gérer de manière centralisée tous les services du système. Il est fourni dans différentes publications pour un scénario d'hébergement et d'utilisation spécifique.

→ À propos de Plesk sur le site officiel

Java de base: PHP, C, C ++
Système d'exploitation pris en charge: différentes versions de Linux, Windows


Plesk

Nouveau

Les nouvelles fonctionnalités du panneau se présentent sous la forme d'extensions compilées dans un répertoire du site. L'interface a été considérablement améliorée: conception adaptative, possibilité de connecter automatiquement les clients à Plesk à partir de ressources externes sans réauthentification (par exemple, à partir du panneau de leur fournisseur d'hébergement), possibilité de partager des liens directs vers des écrans; Interface utilisateur repensée pour le planificateur de tâches. Amélioration de la gestion des bases de données; il existe un support pour plusieurs versions de PHP, ainsi que Ruby, Python et NodeJS; prise en charge complète de Git; Intégration Docker Boîte à outils SEO. L'outil de récupération Plesk, un utilitaire de ligne de commande qui peut être utilisé pour détecter et résoudre automatiquement un large éventail de problèmes, est apparu. Chaque instance Plesk est désormais automatiquement sécurisée à l'aide de SSL / TLS. Vous pouvez réduire le temps de réponse des sites et la charge sur le serveur à l'aide de la mise en cache Nginx. Une fonctionnalité appelée Smart Updates a été ajoutée à l'extension WordPress Toolkit populaire, qui analyse les mises à jour WordPress avec une intelligence artificielle pour déterminer si l'installation de la mise à jour peut casser quelque chose.

Les prix

RUVDS fournit également un panneau Plesk à ses clients, le prix d'une licence est de 650 roubles par mois.

Directadmin


Les développeurs positionnent le panel comme le plus facile à gérer au monde. Ils essaient de suivre le temps et utilisent des technologies avancées, alors qu'il n'y a rien de surnaturel dans le panneau - seulement des fonctions de base . Il n'y a pas de scripts prédéfinis, mais vous pouvez créer les vôtres (API ouverte). Interface multilingue, mais sans support russe (vous pouvez utiliser des skins non officiels). Filtre antispam faible. Mais - peu exigeant pour les ressources du serveur et haute vitesse. Accès à plusieurs niveaux.

Base PL: PHP
Système d'exploitation pris en charge: FreeBSD, GNU / Linux (distributions Fedora, CentOS, Debian, Red Hat)
