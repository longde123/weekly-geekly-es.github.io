Windows 95 berumur 20 tahun



Sistem operasi Windows 95 muncul pada Agustus 1995. Ini menandai penyebaran PC yang cepat di seluruh dunia - karena sekarang sistem komputer telah tersedia tidak hanya bagi geek, tetapi juga bagi ibu rumah tangga, dalam arti kata sebenarnya. Konfigurasi PC rata-rata pada waktu itu (dalam hal apa pun, di AS) termasuk hard drive 320 MB, 8 MB RAM, prosesor 386DX atau lebih tinggi (486 direkomendasikan untuk instalasi dan operasi normal OS).

Distribusi awal sistem disampaikan pada 13 disket. Di OS ini, ada inovasi seperti menu "Start", yang dirancang sebagai tombol di sudut kiri bawah (pada Windows 10, tombol dan menu dikembalikan, tentu saja, dalam bentuk yang sangat berubah). Kesederhanaan bekerja dengan menu memungkinkan untuk bekerja dengan cepat dan efisien dengan sistem file mesin dan aplikasi.



Selain itu, pengembang Microsoft telah menambahkan dukungan untuk nama file yang panjang (hingga 256 karakter) dan sistem Plug and Play. Wikipedia memberi tahu kita bahwa â€œinovasi utama di Windows 95 adalah kemampuan untuk menjalankan aplikasi 32-bit berdasarkan Win32 API. Fitur ini pertama kali muncul di Windows NT, tetapi sistem keluarga ini memiliki persyaratan perangkat keras yang lebih tinggi dan karena itu tidak dapat membandingkan popularitasnya dengan seri Windows "biasa" (yang sebelum rilis Windows 95 diwakili oleh keluarga Windows 3.x). "

Selain itu, Internet Explorer terintegrasi dengan antarmuka Windows (Explorer.exe), yang menghilangkan penghapusan lengkap file Internet Explorer tanpa mengganggu fungsi sistem.

Sejak munculnya Windows 95, banyak yang telah berubah, misalnya, komputer dan elektronik telah menjadi mana-mana. Dalam banyak hal, itu adalah penampilan Windows 95 dan versi selanjutnya dari ini dan OS lainnya dengan antarmuka grafis yang mempengaruhi dinamika tersebut. Selain itu, sekarang banyak layar - layar sentuh, buku dan majalah dibaca di tablet dan telepon pintar (tentu saja, versi kertas belum hilang), dan Internet, yang masih langka pada saat munculnya Windows 95, kini telah menjadi bagian integral dari kehidupan miliaran orang.

Omong-omong, elektronik telah banyak berubah sehingga Windows 95 sekarang diluncurkan bahkan pada jam tangan pintar, belum lagi smartphone atau tablet.
