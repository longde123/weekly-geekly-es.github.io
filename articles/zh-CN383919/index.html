拼车将帮助找到一个女孩



最近，共享私家车的服务-所谓的“拼车”-越来越受欢迎。这是常规的搭便车，但是以现代方式进行：使用GPS导航，智能手机和社交网络。您可以提前选择一位同行旅行者，查看他的Facebook个人资料，并同意旅行的路线和时间。

有趣的是，最狡猾的公民除了省钱外，还发现了拼车的其他好处。事实证明，这是安排日期的好方法，用Re / code 编写 Carmel DeAmisis。

对于某些“接客”，该过程已成为标准程序：我们启动了UberPool和Lyft Line应用程序，指示路线并与女孩们坐在车里。这比Internet上的临时通信更为有效。熟人马上就住了，在这里您可以看到女孩的所有细节，与她聊天。在旅途中有足够的时间来增进彼此之间的了解。在某种程度上，这就像“快速约会”，这在很大程度上还取决于运气。而且无论如何，比通过Tinder或其他约会服务约会更有趣。

不幸的是，该应用程序无法对女性同伴进行预过滤，但是选择的对象太多了。周六晚上有些男人不停地坐在马路上，直到他们找到要回家的人。女孩有时也会这样做。

女孩上车后，驾驶员便问她要去哪里。如果她说她要回去和男朋友在一起，那么进一步的交流不太可能取得成果。尽管如此，成功的会议还是经常发生：一位26岁的旧金山风险投资人Mitchell在接受 Re / code 采访时说，他一直在这样做-他设法安排了四个约会，每个拼车服务UberPool安排了两个约会和Lyft线。

没有任何计划，恋爱关系就会出现。尤伯（Uber）的一位私人出租车司机说，每天，试图调情或获得电话号码的乘客都会坐下来一次或两次。

显然，Uber或Lyft汽车是一个非常合适的聚会场所。他们说，很多随机的旅行者经常彼此了解并开始轻松地交流，在深夜或晚上从聚会中回来。没错，这并不总是发生，即在城市的某些日子和某些地区。例如，星期六晚上是理想的时间。

对于喜欢在线聊天的人，拼车是在线约会网站的绝佳选择。在这里，他们感到更自然。
