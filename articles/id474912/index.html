Pesan dan peringatan di Android melalui JSON

Untuk menerima pesan singkat, Anda dapat menggunakan email, SMS, pemberitahuan push atau membuat bot untuk messenger.

Saya mengusulkan untuk mempertimbangkan cara sederhana lain:

1. Buat sumber data dalam JSON di sisi layanan web. 

{ "refl.me":true, "message":"Hello!", "refresh":96 }

"Refl.me" pasti menunjukkan benar.
"Pesan" mengatur teks pesan, pesan kosong diabaikan.
"Refresh" menetapkan berapa kali sumber harus diperiksa dalam 24 jam (0 - pembaruan manual, 288 - maksimum). 

 <?php header('Content-Type: application/json'); $json_array =array('refl.me'=>true,'message'=>'Hello!','refresh'=>96); echo json_encode($json_array); ?>

2. Unduh aplikasi dan tambahkan tautan ke sumber data kami yang dibuat di dalamnya.

3. Semuanya siap. Terima peringatan dan pesan.

Prinsip operasi sangat sederhana, aplikasi memeriksa sumber sesuai dengan interval yang ditentukan (atau dalam mode manual), jika nilai pesan telah berubah ke nilai yang berbeda dari nilai sebelumnya, peringatan ditampilkan di aplikasi.

Semuanya berfungsi tanpa server pihak ketiga, tidak memerlukan data pribadi dan pendaftaran.
