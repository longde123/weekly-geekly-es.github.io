Analisis notebook jaringan yang tidak populer

Semua orang terbiasa dengan situasi ketika, karena keadaan, Anda dipaksa untuk bekerja dengan dokumen yang sama di beberapa perangkat.

Ada banyak solusi jaringan untuk keperluan ini. Untuk tugas menangani teks, saya memutuskan untuk meninggalkan Evernote dan mencoba buku catatan Leanote, Wiznote, Turtl, dan Simplenote. Sebagai perbandingan, saya menggunakan Poison tempat Anda dapat menyimpan dok biasa, serta Notesforweb dari pengembang aplikasi Notes untuk android, Jacob Ras, di garter ke layanan Dropbox. Saya secara khusus tidak mempertimbangkan turunan dari Perusahaan Baik yang terkenal. Meskipun pertimbangan privasi tidak berlaku dalam kasus apa pun, karena semua orang tahu bahwa tidak ada anonimitas di jaringan.

Keputusan saya untuk tidak menggunakan repositori ditentukan oleh pertimbangan lain. Pertama, saya tidak ingin meletakkan semua telur dalam satu keranjang, dan kedua, saya biasanya membatasi akun tergantung pada pekerjaan yang dilakukan.

Apa yang saya butuhkan dari notebook jaringan gratis?

Pertama-tama, agar semuanya disinkronkan dengan benar antara server dan klien itu sendiri, dan kemudian antara server dan perangkat lain dalam sebuah rantai, akses ke catatan Anda secara offline, antarmuka yang nyaman, dan tidak adanya pembatasan jumlah data yang diunduh.

Cukuplah untuk mengatakan bahwa saya telah menggunakan Simplenote untuk waktu yang lama. Saya tidak tergoda oleh klien asli untuk android, meskipun terlihat hebat. Membutuhkan banyak memori dalam memori perangkat, jadi untuk menggantinya saya menemukan mNote lama, yang dikembangkan untuk Android versi 1.6 dan Glance Note yang lebih baru (hanya 22 mb). Untuk Windows ada klien Resoph notes yang sangat baik, untuk Linux - nvPY. Kedua klien itu ringan, meskipun dengan standar modern, dengan antarmuka yang ketinggalan zaman. Perlu juga disebutkan plug-in browser (Sincpad), yang tentu saja juga nyaman. Bekerja dengan 4 perangkat berbeda dalam penyandian berbeda, sinkronisasi berfungsi tanpa kesalahan. Ada akses ke data di semua perangkat. Teks dapat diunduh sebanyak yang Anda suka. Satu masalah - semua tanpa ilustrasi. Tetapi untuk ekspresi pikiran dan ide cukup cocok.

Secara umum, seseorang bisa berhenti dan tidak perlu repot-repot lagi.

Untuk kebutuhan bisnis murni, saya membuka akun di server Leanote Cina. Ada aplikasi untuk Windows, Mac, Linux, dan seluler untuk iOS dan Android. Antarmuka notebook menarik saya dengan kesederhanaannya dan kehadiran semua fungsi yang diperlukan. Di buku catatan, Anda dapat membuat catatan sederhana, menggunakan mode bahasa markup dokumen yang disederhanakan - penurunan harga, menyisipkan gambar dalam teks. Notepad memiliki semua fitur pengeditan teks dasar. Anda dapat mempublikasikan catatan sebagai blog di situs yang sama. Anda tidak dapat menerbitkan, tetapi mengekspornya dalam beberapa format umum: pdf, html, leanote, dan evernote. Jika koneksi tidak stabil atau tidak ada sama sekali, catatan tersedia secara offline. Seperti kebanyakan notebook jenis ini, frekuensi jam dapat disesuaikan. Terlepas dari kenyataan bahwa ada beberapa bahasa antarmuka (5, yang dua adalah bahasa Cina: disederhanakan dan tradisional, Korea, Inggris, Jerman), tidak ada banyak pilihan. Bahasa Inggris di notebook terlihat tidak mencolok dan tidak mungkin menakuti rata-rata pengguna berbahasa Rusia. Omong-omong, catatan dapat disimpan bukan di jaringan, tetapi Anda dapat mengonfigurasi notepad untuk disimpan di penyimpanan lokal. Ada plug-in untuk browser, tetapi, menurut saya, itu sudah memerlukan pembaruan, karena hanya berfungsi dengan benar pada Windows.

Saya kesal dengan pekerjaan sinkronisasi. Mungkin, kelemahan dalam kualitas komunikasi terpengaruh. Setelah sinkronisasi, informasi saya tetap relevan hanya pada mesin yang saya gunakan sebelumnya. Setelah saya mulai sinkronisasi di android. Informasi yang ketinggalan zaman dibanjiri ke server dari smartphone.

Omong-omong, tentang aplikasi untuk android. Anda tidak akan menemukan aplikasi Leanote di Google Play store yang terkenal. Sejauh ini hanya terletak di Github, di mana ada beberapa versi. Dari jumlah tersebut, saya menguji dua yang terakhir. Semuanya mudah dipasang dan berfungsi dengan benar. Penting bahwa aplikasi hanya membutuhkan 32 MB dalam memori ponsel cerdas. Tetapi bahkan jika Anda mengecualikan tautan telepon dalam rantai sinkronisasi, maka semuanya tampak normal hanya dalam urutan: Windows PC - Leanote server - Linux. Jika Anda melakukan hal yang sama secara terbalik, maka pada PC Windows tidak akan ada pembaruan informasi. Artinya, informasi saya sekarat di suatu tempat di sepanjang jalan, jelas, tetapi itu tidak membuat saya merasa lebih baik, jadi saya menunda penggunaan notebook ini.

Aplikasi WizNote yang lebih populer, juga dikembangkan oleh tim China untuk mencatat secara online. Aplikasi "hidup" di semua platform populer. Menurut berbagai peringkat, ia memiliki lebih banyak penggemar daripada notebook Leanote. Saya memutuskan untuk mencoba fungsinya.

Tampilannya menyerupai notebook Evernote yang sama. Rangkaian fungsi untuk mengedit teks sama dengan Leanote. Hanya ada 3 bahasa antarmuka di versi desktop WizNote: seperti biasa, dua bahasa Cina dan Inggris. Ada juga plugin browser yang bertindak sebagai clipper.

Di jaringan Anda dapat menemukan banyak ulasan tentang program ini, juga, dan evaluasi komparatifnya dengan Evernote. Aplikasi seluler "memakan" 92 mb memori perangkat. Ini sedikit, tentu saja, tetapi jika kami mempertimbangkan konsumsi seperti itu untuk setiap program yang diinstal pada smartphone, yang tanpanya Anda tidak dapat hidup tanpanya, maka kekurangan memori sangat cepat terdeteksi. Setiap orang memiliki ponsel pintar yang berbeda, tidak semua orang memiliki memori 3 GB, jadi bukan fakta bahwa akan nyaman bagi seseorang untuk menggunakan Wiznote di telepon mereka.

Mengenai sinkronisasi, saya belum melihat adanya gangguan sejauh ini. Program di baki pada Mate di LM bekerja sedikit lebih lambat, tetapi pada Windows tidak ada masalah. Jika terjadi kegagalan koneksi, Anda dapat memilih teks dari hard drive, seperti halnya dengan notebook Leanote.

Terlepas dari kenyataan bahwa WizNote yang lebih populer di kalangan pengguna, saya pribadi cenderung menyukai notebook Leanote ketika membandingkan kerajinan Cina. Menurut pendapat saya, kesederhanaan antarmuka menang, ringan dengan fungsionalitas yang lebih besar. Jika Anda tidak menggunakan perangkat sebanyak yang biasanya saya lakukan, maka mungkin tidak akan ada masalah dengan hilangnya informasi.

Jika kita melihat lebih jauh pada notebook Turtl, di sini, di satu sisi, perlindungan informasi ditingkatkan dibandingkan dengan Cina. Tetapi untuk pekerjaan, perlu menginstal klien desktop atau seluler. Layanan tidak memiliki antarmuka web, kecuali untuk plug-in clipper, yang juga tidak berfungsi tanpa klien diinstal pada PC. Tidak ada yang disimpan di disk komputer. Jika, dengan demikian, Anda lupa kata sandi atau server tiba-tiba jatuh, atau Anda tidak memiliki akses ke jaringan di seluruh dunia, maka Anda tidak akan melihat catatan seperti telinga Anda.

notesforweb.com - sejenis lotion menggunakan Dropbox. Hanya ada antarmuka web dan aplikasi seluler untuk android. Catatan disimpan di folder dropbox. Setelah menulis sesuatu di aplikasi Notes untuk ponsel cerdas, Anda dapat memodifikasinya di PC dari browser. Tampaknya nyaman. Anda dapat menambahkan gambar ke awal teks. Versi browser hanya memungkinkan Anda untuk mengedit teks dan kemudian mencetaknya. Dalam versi seluler, Anda dapat mengubah catatan Anda menjadi daftar, membagikannya, mencetaknya, atau menghapusnya. Ada kemungkinan bahwa dari catatan paling populer yang menggunakan layanan dropbox untuk sinkronisasi, ini adalah opsi yang paling nyaman.

Tapi sekarang semuanya bagiku menari nyata dengan rebana. Anda tidak dapat memilih aplikasi yang ada untuk semua jenis platform dan perangkat. Kemungkinan besar, ada baiknya mengatur ruang dengan benar di folder yang sesuai di Poison, Dropbox, atau cloud dari Mail Group. Untuk membuat catatan / catatan menggunakan aplikasi yang paling umum: kantor WPS, Libreoffice (untungnya, sekarang ada kemampuan untuk meletakkan dan mendapatkan file dari jarak jauh), Wordpad atau Notepad, dan semuanya lebih baik dalam format rtf. Lagi pula, ketersediaan server dari Yandex tidak seperti Cina. Dropbox, secara umum, juga merupakan opsi win-win, terutama jika klien diinstal pada PC atau smartphone. Berbicara tentang server dan ketersediaannya, perlu diingat bahwa sebagian besar layanan penyimpanan cloud yang terkenal adalah server Amerika (Dropbox, Turtl, Zoho, Simplenotes, Google Keep, Gnotes, OneNote). Dengan latar belakang semua peristiwa yang terjadi di dunia, Tirai Besi 2 tampaknya tidak lagi menjadi sesuatu yang lucu dan tidak realistis.

Oleh karena itu, saya mendapat kesan bahwa penyimpanan di Yandex / Mail adalah pilihan yang paling membingungkan bagi pengguna biasa daripada mengambil dan membuat server cloud rumah Anda sendiri segera.
