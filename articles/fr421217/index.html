Prise en charge de Python dans Power BI

Nous sommes heureux d'annoncer que Python, un langage de programmation largement utilisé par les statisticiens, les scientifiques et les analystes, est désormais intégré à notre bureau gratuit Power BI. Après avoir activé la fonction dans les paramètres, vous pouvez utiliser Python pour nettoyer, analyser et visualiser les données. Plus de détails sous la coupe!



La première fonctionnalité d'intégration que nous avons ajoutée est la possibilité d'exécuter des scripts Python directement dans Power BI Desktop pour créer un modèle de données. Vous pouvez le faire dans la boîte de dialogue Get Data.



Après avoir sélectionné cette option, vous pouvez insérer votre script.



L'intégration avec l'éditeur de requêtes vous permet de nettoyer les données à l'aide de Python et d'utiliser des méthodes avancées pour analyser vos données, y compris, par exemple, la suppression de missions, la prévision et le clustering.

Enfin, vous pouvez utiliser Python pour créer de nouveaux effets visuels dans votre rapport. Comme la visualisation dans R, les effets visuels de Python seront mis à jour avec l'actualisation des données et le filtrage croisé. Cependant, le visuel lui-même n'est pas encore interactif.
