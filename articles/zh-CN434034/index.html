YouTube上的优秀教程

YouTube上有许多免费教程和课程。



我运行了一个freeCodeCamp YouTube无广告频道 。 我们提供了有关许多流行编程语言和框架（包括JavaScript ， Python ， Java ， Ruby ， C ， C ++ ， Angular等）的完整视频课程和教程。

还有其他YouTube频道提供免费编程课程，这些课程并没有比付费频道更糟，有时甚至更好。

在本文中，我列出了十个YouTube频道，您可以观看它们进行编程。 通道布局是随机的。

从不同的老师那里看同一主题可能很有用，因此您可以更深入地了解问题。

有很多很棒的频道，我在此列表上没有足够的空间。 在评论中分享您的发现。

编码火车



在没有编辑可能性的情况下实时录制高质量的课程非常困难，并且需要特殊的技能。 这正是Coding Train的Daniel Schiffman所做的。 他以有趣的方式教授复杂的主题，使初学者容易理解。 看到频道的开场视频后，您就会明白为什么需要立即订阅。

遍历媒体



布拉德·特拉弗西（Brad Traversy）对卓越的热情在他的视频中得到了明显体现。 他的“胡话”风格友好而实用。 他似乎完全了解自学成才的程序员需要知道的内容。 他的频道提供有关各种Web开发框架和语言的教程。

德里克·巴纳斯（Derek banas）



Derek Banas不仅是程序员，还是广泛的专家。 他在几乎所有流行（和不那么流行）的编程语言中都有专业水平的课程。 他目前的追随者数量超过该列表中的任何其他人，这是当之无愧的。 如果您想学习编程语言，那么他的频道是您需要观看的第一件事。

LearnCode.academy



LeanCode.academy的Will Stern将近20年的Web开发经验带入了他的课程。 他的课程非常棒，涵盖了您成为Web开发人员所需的所有知识。

Dev Tips已经存在了一段时间，但最近又传到了David和MPJ的手中。 MPJ还主持了Fun Fun Function频道。 这两个渠道都非常有用，但是“开发技巧”比“ FunFunFunction”具有更多的教训。 如果您需要有关JavaScript框架的课程，DevTips是一个不错的渠道。

升级



与大多数其他编程频道相比，LevelUpTuts具有更多的教学视频。 这是学习最佳Web开发实践的好渠道。 重点是前端框架。 LevelUpTuts的Scott Tolinski还主持了最好的Web开发播客之一，语法。

忍者网



当《网络忍者》的肖恩（Sean）说他将帮助您掌握开发技能时，他并不是在开玩笑。 这些是涵盖完整堆栈开发的高质量教程。 每个人都喜欢英国的口音，对吗？

森德克斯




这是高级Python教程的渠道。 尽管即使是新手也可以理解该频道的说明，但该频道涵盖了其他频道未涵盖的更复杂的主题。 它提供了很棒的机器学习课程，甚至还展示了如何在GTA5上对具有自我管理功能的汽车进行编程。

Siraj Raval频道可能是此列表中最有趣的频道。 我认为可以肯定地说，Siraj是唯一定期在自己的视频上展示自由式说唱的YouTube程序员。 他的频道也是研究人工智能和深度神经网络的最佳场所之一。 Siraj擅长制作有趣而令人兴奋的高级技术视频。
