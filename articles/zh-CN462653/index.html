LTE是独立性的象征


夏季是外包的热门时间吗？


传统上，夏天被认为是商业活动的“死季”。 有些人正在休假，而另一些人则不急于购买某些商品，因为没有相应的态度，此时卖家和服务提供商本身更喜欢放松。


因此，对于自由职业者（例如，“即将来临的系统管理员”）的外包商或IT专业人员来说，夏天被认为是不活动的时间...


但是您可以从另一侧看。 许多人正在转移到休息的地方，有人想要在新地方建立通信，有人想要从俄罗斯任何地方（或至少从最近的郊区）获得稳定的访问权限。 使用云服务进行咨询，用于连接和配置，组织（例如）对家用计算机的远程访问的服务-所有这些可能都是需要的。


对于初学者来说，不值得立即记录整个三个夏季的损失，甚至更好，至少环顾四周，看看谁在这种环境下需要什么。 例如，LTE通信。


“魔术棒”


大城市的居民在高质量的沟通方面非常宠爱。 有许多用于通过有线方式访问Internet的选项，包括基于光纤的“专用”，可能的免费Wi-Fi以及大型移动运营商的可靠蜂窝通信。


不幸的是，离区域中心越远，获得高质量通信的机会就越少。 下面我们将探讨LTE通信将派上用场的领域。


当提供者是逆行的时候


本地服务提供商远非总是“处于技术浪潮的顶峰”。 通常，提供商的设备，基础设施和服务质量都令人印象深刻。


让我们从基础架构开始。 将光纤GPON传导到村庄中的每个公寓或村庄中的每个房屋-到目前为止达到梦想的水平。


小型提供者比大型提供者更贫穷，省级提供者比大城市提供者更穷，他们用于建立发达基础设施的资源更少。 同时，小规模居民区的购买力低于大城市（极少数例外）。 因此，就投资回报而言，“直接投资”资金通常没有前景。


在某些情况下，用户必须对具有适当速度和功能的基于ADSL的连接感到满意。 但是在这里，我们谈论的是具有完善基础设施的住区。 新建的夏季农舍，诸如仓库之类的偏远物体，工业场所，除“以太”外，通常与外界没有任何联系。


如果我们谈论设备-可能性仍然很小。 要购买新的通信设备，必须找到其他资金。 而且这并非总是可能的，特别是因为数量（取决于当前船队的过时程度）可能非常令人印象深刻。


另一个重点是服务水平。 “人员饥饿”并非如此罕见。 优秀的专家总是供不应求，大城市或“国外工作”使找到比本地服务提供商更高的薪水成为可能。


值得一提的是市场上的垄断地位。 如果整个地区只有一个Internet服务提供商，那么他不仅可以决定价格，而且可以决定服务水平。 然后，系列中的推理：“他们（客户）将在哪里远离我们？”，成为服务消费者时的主要座右铭。


不能说所有这些问题都是由于某人的贪婪，不愿做任何事情以及其他致命的罪恶而引起的。 一点也不。 仅仅出于经济，技术或其他原因，您就无法快速解决所有问题。


因此，采用LTE空中访问形式的替代方案是通过更改提供商来改善服务质量的好机会。


风滚草


有很多人的位置，活动类型和简单的生活方式与频繁的移动有关。


如果您乘汽车旅行，最好简单地忘记使用有线连接的选项。 但是正是在旅途中，有时需要对网络的高质量访问。 例如，对于建筑师，建筑商，房地产经纪人，设备维修技术人员以及博客作者，以及通常情况下在旅途中必须连接网络资源的所有人员。


您可以在每台设备上使用移动通信（并为此付费），但是在汽车上安装LTE路由器并通过Wi-Fi连接移动设备会更容易，更经济。


注意事项 经常乘车旅行的人可以推荐便携式设备，例如便携式LTE Cat.6 Wi-Fi AC1200路由器（WAH7706型）。 这样的小型路由器具有小巧的尺寸，能够为多个设备提供可靠的通信。



图1.便携式AC1200 LTE路由器（WAH7706型号）。


你带上网了吗？


但是，在大城市中，有些地方很难甚至根本无法访问网络。 建筑就是一个很好的例子。 无法实现有线互联网连接，但是现在我们需要连接，例如，用于视频监控。


有时，用于出售公寓的临时办公室在未完成的设施上运行，这需要对远程网络资源的高质量访问。


在工业区的设施中也会出现类似情况。 由于距离远且消费者数量少，拉电缆根本无利可图。 帮助LTE覆盖范围广。


当然，度假村镇也需要LTE。 服务消耗具有季节性，在夏季，别墅中有很多人，而在冬季则没有人，这使得这些设施对“提供电线的人”没有吸引力。 因此，长期以来，LTE路由器一直被认为与拖鞋或花园喷壶具有相同的“夏季别墅属性”。


无法切断的电线


通过物理电缆进行访问可提供稳定，可靠的通信（在适当的技术水平上），但有一个局限性-一切工作直到电缆损坏为止。


以视频监视系统为例。 如果您是通过Internet远程录制来自摄像机的图片，则建立独立的连接非常重要。 在这方面，有线访问不是最佳解决方案。


看一下位于住宅楼底楼公寓中的商店，美发师或其他小型企业。 如果电缆至少出现在某个地方，例如在可触及的地方至少有一点穿过电缆面板，则可以将其切断，视频监控系统将停止传输。 而且，即使内部资源有副本，例如在注册服务商的硬盘上，所有这些：摄像机和注册服务商都可以禁用或随身携带，同时保持完全隐身状态。


对于无线通信，进入房间后可以中断对网络的访问（如果您不考虑特殊的“干扰”）。 如果您至少在很短的时间内维护了自主电源，那么在大多数情况下，您可以记录入侵的时刻，然后将其呈现给警察，保险公司，安全公司等。


另一个麻烦是“普通用户”的交换机和其他设备的故障，例如，由于熟练的建筑商和简单的“工匠”的过错，他们会并且可能为邻居造成各种问题。


对于这种情况，通过LTE进行无线通信可能是必不可少的。


LTE的力量是什么


缩写LTE代表长期演进。 实际上，这甚至不是一个标准，而是一个旨在回答以下问题的开发方向：“当3G功能不再足够时，将计划什么？”。 假定LTE将在3G标准框架内运行，但随后的发展具有更广泛的特征。


最初，为3G网络设计的设备可以部分用于基于LTE技术的通信。 这样就可以节省引入新标准的成本，降低订户的“进入门槛”，并大大扩大覆盖范围。


LTE具有相当广泛的频道列表，这为最广泛的应用打开了可能性。


营销人员谈论LTE作为第四代移动通信-“ 4G”。 但是，术语上的一些混乱是值得注意的。


根据国际电信联盟（ITU）的一份文件 ，LTE-A技术获得了官方指定的IMT-Advanced。 同时也表明，IMT-Advanced被认为是“ 4G”技术。 但是，ITU并不否认术语“ 4G”没有明确的定义，并且原则上可以应用于其他技术的名称，例如LTE和WiMAX。


为避免混淆，基于LTE-A的通信被称为“ True 4G”或“ True 4G”，而较早的版本称为“营销4G”。 虽然这些名称可以被认为是相当随意的。


如今，大多数标有“ LTE”的设备都可以使用各种协议。 这对访问区域（覆盖区域）的扩展以及每次都不需要购买新设备的用户的钱包都有积极影响。


移动作为路由器-有什么缺点？


当阅读有关LTE技术的可用性时，有时会出现一个问题：“为什么要购买专用设备？ 为什么不只使用手机？” 毕竟，现在几乎可以通过任何移动设备“通过Wi-Fi分发Internet”。


当然，您可以将手机用作调制解调器，但是，温和地说，这样的解决方案会给路由器带来很多损失。 对于专用路由器，您可以选择一个户外放置选项，将其放置在有信心的地方，例如屋顶下。 另一种选择是连接专用天线。 （下面将讨论特定型号对外部天线的支持）。


为了从手机，平板电脑甚至笔记本电脑直接访问，这种可能性几乎是不可行的。



图2. LTE7460-M608室外LTE路由器非常适合夏季别墅和其他偏远站点。


当您需要同时将多个用户连接到这样的“通过手机分配”时，工作变得非常不便。 手机上的Wi-Fi发射器的功率比带有内置接入点的路由器的功率弱。 因此，您必须坐在尽可能靠近信号源的位置。 另外，移动设备的电池放电非常快。


除了硬件上的细微差别外，还有其他方面。 通常，为语音蜂窝和移动Internet的平均使用而设计的移动运营商的普遍报价具有流量限制，并且对于提供对网络的共享访问不是特别有利。 使用仅设计用于Internet的合同更加容易和便宜。 结合专用设备，可以以低廉的价格获得良好的速度。


一些实际问题


首先，建议您了解哪些任务需要访问网络。


如果您打算“逃离文明”，而只需要互联网就可以在电子书上下载另一本小说，这就是一种使用方式。


如果您需要始终保持联系，请继续工作并过着积极的在线生活-这是一种完全不同的消遣方式，并且网络上的负载也完全不同。


客户设备扮演着重要角色。 假设我们有一台IT笔记本电脑-一台旧笔记本电脑，以备不时之需。 在这种情况下，更陈旧和更现代的路由器是合适的。 最主要的是在2.4GHz频率范围内具有Wi-Fi支持。


如果我们以个人计算机的形式谈论客户端，那么它们可能根本没有Wi-Fi接口。 在这里，您需要选择带有LAN端口的型号以进行双绞线连接。


在上述情况下，您可以推荐带有4个LAN端口（型号LTE3301-M209）的N300 LTE路由器。 这是一个经过时间考验的好方法。 尽管仅在802.11 b / g / n（2.4GHz）级别的框架内支持Wi-Fi，但有线连接端口的可用性使其可以用作家庭办公室的成熟交换机。 当有网络打印机，个人计算机，用于备份的NAS（通常是小型企业的全套设备）时，这一点很重要。


LTE3301-M209路由器配有外部天线，可以接收来自基站的信号。 此外，通过2个SMA-F连接器，即使蜂窝信号减弱，您也可以连接外部功能强大的LTE天线以实现可靠的通信。



图3.具有4个LAN端口的LTE Cat.4 Wi-Fi N300路由器（LTE3301-M209）。


当一堆最新的电子设备转移到避暑别墅或暑假办公室时：移动设备，精密的笔记本电脑-最好选择支持最新创新的最现代型号，以便通过Wi-Fi，LTE和其他有用的东西进行访问。


如果有机会在户外放置，则值得一看LTE7460-M608型号。 （请参见图2）。


首先，可以将LTE路由器放置在更好的接收区域，例如屋顶下，建筑物外等等。


其次，该位置不仅可以在建筑物内部，而且可以在场地的开放区域实现可靠的Wi-Fi连接。 LTE7460-M608使用增益为8 dBi的集成天线进行通信。 另一个重要功能-PoE电源使您可以将设备放置在距离房屋100米的地方，将其放置在屋顶或桅杆上。 当高大的树木生长在房屋附近时，尤其如此，这可能会干扰来自基站的蜂窝信号。 LTE7460-M608带有PoE注入器，可提供PoE +功率高达30瓦。


但是有时在某些情况下可能无法使用外部设备。 在这种情况下，具有FXS端口（型号LTE3316-M604）的千兆级LTE Cat.6 Wi-Fi AC1200路由器将为您提供帮助。 该设备具有四个GbE RJ-45 LAN端口。 重要的是，第一个LAN1端口可以重新配置为WAN。 事实证明，这是一种通用设备，在寒冷的月份，它可以用作城市公寓中的常规路由器，以通过双绞线与提供商进行通信，而夏季则可以用作LTE路由器。 除了金钱上的收益-购买一台设备（而不是两台设备）之外，使用LTE3316-M604还可以使您无需重新配置本地网络的设置，访问设置等。 您最大需要的是转移路由器以使用另一个外部通道。


LTE3316-M604路由器还允许您连接外部功能强大的LTE天线，因为它具有2个SMA-F连接器。 例如，我们可以推荐带有系数的LTA3100天线模型。 6dBi增益。

结论


从所描述的示例可以看出，在提供Internet访问方面没有“死角”。 但是，访问网络的方法和负载的性质都会发生变化，这会影响特定技术的选择。


LTE是一种相当通用的选项，它使您可以在相当宽的覆盖范围内组织稳定的连接。


正确的设备选择使您可以更灵活地调整可用功能，以满足每个消费者的需求。
