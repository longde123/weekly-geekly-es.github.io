Konferensi Hari Azure di Moskow

Azure Day adalah konferensi tahunan yang diselenggarakan oleh komunitas Microsoft MVP (Most Valuable Professionals). Gagasan inti Azure Day adalah berbagi informasi cloud Azure terbaru dan paling berharga dengan komunitas TI. Laporan akan disajikan oleh guru pengembangan dan solusi bisnis - MVP, yang bekerja setiap hari dengan teknologi Microsoft, pakar Microsoft dari Eropa dan Rusia.



Hari Azure adalah, tanpa berlebihan, kesempatan langka untuk belajar tentang bagaimana Azure berkembang, untuk mengajukan pertanyaan kepada mereka yang terkait langsung dengan pengembangan arah ini. Azure di ujung jari!



Kami mengundang para profesional TI dan eksekutif bisnis ke acara tersebut.

Dalam program:


  • Pencelupan mendalam dalam pengembangan menggunakan layanan Azure: bot, database, IoT, AI / ML
  • Gambaran umum dan contoh bekerja dengan infrastruktur Azure: mesin virtual, jaringan, DevOps, keamanan
  • Contoh mengoptimalkan dan membangun proses bisnis menggunakan teknologi Microsoft: Office 365, Dynamics, Power BI.

Detail dan pendaftaran di situs web acara - www.azureday.ru
