PHP内部


  • [RFC]更改级联运算符的优先级 -接受的提议以降低级联运算符的优先级. 。 该行为将在PHP 8中更改，在PHP 7.4中，将表达式与一起使用时将引发警告. 在+或-之前-不带方括号。 
     echo "sum: " . $a + $b; //   echo ("sum: " . $a) + $b; //   PHP 8 echo "sum :" . ($a + $b);
  • [RFC]数字文字分隔符 -建议增加使用符号_作为数字分隔符的可能性，以提高可读性： 
     $i = 197_823_459; // 197823459
    该提案在两年前的投票中失败了 ，但是从那以后，在Python，JavaScript和TypeScript中添加了类似的功能。
  • PHP Internals News播客＃10-与Nikita Popov谈了Lisk替代原理和操作员优先级。
