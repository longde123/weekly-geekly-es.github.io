Internes PHP


  • [RFC] Ajouter une interface Stringable - Il est proposé en PHP 8 d'ajouter une nouvelle interface Stringable qui peut être ajoutée aux classes qui implémentent la __toString() . Et puis utilisez le type combiné chaîne | Stringable où la chaîne est censée être. 
     interface Stringable { public function __toString(): string; }

Les outils

  • BenMorel / faiblessemap-polyfill - Polyfill WeakMap pour PHP 7.4.
  • lisachenko / z-engine - Une bibliothèque expérimentale qui permet d'utiliser FFI pour accéder aux structures internes de PHP lui-même. De bons exemples d'utilisation ont été ajoutés, par exemple, en implémentant la surcharge des opérateurs en PHP et même en modifiant AST à la volée.
