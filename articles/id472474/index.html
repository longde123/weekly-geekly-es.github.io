Bug kosmetik lucu di Google Chrome

Baru-baru ini saya menemukan bug lucu di penginstal browser Google Chrome paling populer. Terlepas dari kenyataan bahwa bug itu kosmetik - itu membuat saya tersenyum.



Bugnya adalah penginstal membuat dua pintasan pada panel peluncuran cepat. Menghangatkan jiwa, fakta bahwa saya tidak hanya meninggalkan serangga setelah saya. Secara umum, saya bertanya-tanya bagaimana itu terjadi? Dengan rilis rilis baru, mereka melakukan pengujian parsial, tetapi tidak menyelesaikan pengujian? Bukankah itu harus diperiksa bagaimana pembaruan meletakkan di atas versi baru? Kemungkinan besar, Google memiliki tes otomatis yang tidak sepenuhnya mencakup semua hasil: misalnya, mereka memeriksa jalan pintas, dan fakta bahwa itu harus dalam satu salinan tidak diperhitungkan.

Pada saat publikasi artikel, bug tersebut relevan.
OS: Windows 7 Home Basic 32bit.

Langkah-langkah untuk bermain:
1) Unduh rilis resmi terbaru dari google
2) Instal peramban (jika sudah memilikinya, hapus dulu)
3) Kami melihat hasilnya

Harapan:
Google Chrome menginstal dan menambahkan pintasan: ke desktop, ke panel peluncuran cepat, ke menu utama tombol Mulai.

Hasil:
Google Chrome memasang dan menambahkan satu pintasan lagi ke bilah luncur cepat.

Selamat malam, teman-teman!
