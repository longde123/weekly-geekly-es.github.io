成为Positive Hack Days 10的演讲者



开始接受参加实用安全周年纪念国际论坛周年纪念日的申请 。 我们正在等待公认的专家和年轻专业人士。 最佳报告将由国际计划委员会选择，该委员会包括IT行业的独立研究人员和领先专家。 所有希望举报的人都必须在3月31日之前发送申请 。

PHDays 10的主题是“入门”。 我们邀请大家参与未来信息安全概念的共同发展。 我们希望有10,000多名来宾和参与者，其中不仅包括黑客和信息安全专家，还包括大型商人和知名政治家。 我们对各种安全领域的实践和原始研究感兴趣：

  • 搜索漏洞及其利用方法；
  • 抵制攻击的手段；
  • 现代计算系统的体系结构问题；
  • 发现事件，攻击及其调查的做法；
  • 威胁情报和威胁搜寻；
  • OSINT
  • 调查网络群体的行为；
  • 建立保护信息基础设施的过程的实践；
  • 开发安全软件的方法和技术-无服务器和云应用程序，微服务架构，人工智能系统；
  • 正式的应用程序安全模型；
  • 软件开发过程中的安全风险管理；
  • BIOS / UEFI等固件安全性；
  • 评估企业信息风险的方法；
  • 识别业务流程中“痛点”的方法；
  • 制定业务信息安全策略的方法。
  • 传统上，我们关注嵌入式系统安全，家用和工业物联网，电信网络，智能家居和视频监控（CCTV），金融技术和工具的安全性，Web和业务应用程序，以及有关反向工程的区块链缺陷和漏洞的话题。 ，应用密码术，机器学习，恶意软件和漏洞利用开发。

您可以采用以下格式之一参加会议：报告（50分钟），快速通道（15分钟）和动手实验（最多4个小时）。 要参加会议，请在页面上填写申请表： https : //cfp.phdays.com/。
