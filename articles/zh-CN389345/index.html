恐怖分子使用自己的加密信使



一群黑客Ghost Security Group宣布其打击网络空间恐怖主义表现的任务，发现了一种特殊的移动应用程序，根据GSG的说法，所谓的通信恐怖分子正在使用该应用程序 “伊斯兰国家。”根据GSG，Alrawi应用程序支持加密聊天。

自然，该应用程序无法从Google下载-它是通过​​专门的站点分发的。尚无人检查过聊天加密协议的可靠性-但聊天室已加密，因此无法禁止此应用程序。

早在2015年12月，Ghost Security Group在Telegram中进行了一次加密聊天，发现了一个移动应用程序，这是一个促进恐怖分子“成就”的新闻门户。该门户网站由Amaq新闻社小组管理，该小组的活动与ISIS相关。

在GSG表示恐怖分子在Telegram上使用加密聊天之后，Pavel Durov 的公司对其服务进行了一些更改 -特别是引入了主持群组的可能性。

现代恐怖分子利用现代交流手段所得到的信息导致了不同的观点。一些州的总统敦促开发应用程序和硬件的公司在其产品中保留用于特殊服务的特殊后门。

反对这种观点的人认为，大多数人的私密性比试图打击恐怖分子的交流手段更为重要，因为无论如何后者都将找到如何相互交流的方法-尤其是因为有人指控恐怖分子过于狡猾是毫无根据的。例如，在法国最近发生的恐怖袭击中，恐怖分子使用常规短信。
