IntelliSense für C ++ in Visual Studio funktioniert jetzt anhand von Beispielen in Ihrem Code

Seit wir IntelliSense für Vorlagen angekündigt haben, haben Sie tolle Angebote erhalten. Ein sehr beliebter Vorschlag war, dass die Vorlagenleiste automatisch eine Variante basierend auf Beispielen im Code auffüllt. In Visual Studio 2019 Version 16.1, Vorschau 2, haben wir diese Funktion der Option "Alle vorhandenen Instanziierungen hinzufügen" im Dropdown-Menü "Vorlagenleiste" hinzugefügt. Die folgenden Beispiele stammen aus der SuperTux- Codebasis.



Das Dropdown-Menü der Vorlagenleiste enthält jetzt den Eintrag "Alle vorhandenen Instanziierungen hinzufügen".



Wenn Sie auf die Option "Alle vorhandenen Instanziierungen hinzufügen" klicken, wird jetzt eine Dropdown-Liste angezeigt, sodass Sie den Text nicht mehr manuell eingeben müssen.



Wie bei " Alle Referenzen suchen" wird im Fenster unten im Editor außerdem angezeigt, wo die einzelnen Instanzen gefunden wurden und welche Argumente sie haben.
