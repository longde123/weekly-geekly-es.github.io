Juri memutuskan bahwa Android menggunakan Java API dengan itikad baik

Uji coba dua minggu antara Oracle dan Google telah berakhir. Oracle telah menghabiskan jutaan dolar untuk menyiapkan gugatan hukum dan membayar pengacara. Dia dengan hati-hati mempersiapkan proses selanjutnya, dengan mengecualikan dari juri satu - satunya orang dengan gelar di bidang TI. Menurut sebuah survei terhadap kandidat juri, ia juga satu-satunya yang, bahkan sebelum persidangan, memahami makna istilah "penggunaan yang adil" (ia belajar tentang hal itu ketika belajar di Universitas California di Berkeley). Tetapi pengecualian dari juri ini tidak membantu Oracle.

Setelah tiga hari diskusi kemarin, juri yang tersisa mencapai keputusan bulat bahwa penggunaan Java API di Android adalah contoh penggunaan yang adil berdasarkan undang-undang hak cipta.

Jika putusannya ternyata berbeda, hakim akan melanjutkan proses, melanjutkan ke tahap penilaian kerusakan. Oracle mengklaim $ 9 miliar

"Saya salut atas kerja keras Anda dalam kasus ini," Hakim Distrik William Alsup berbicara kepada juri, di bawah pengawasan siapa sidang ini telah berlangsung sejak 2010. - Pengadilan memberi Anda rasa terima kasih dan membebaskan Anda. "Aku ingin pergi ke ruang musyawarah juri dan secara pribadi berjabat tangan dengan kalian masing-masing."

William Alsup menerima ketenaran khusus selama salah satu episode sidang pada Mei 2012, ketika ia berbicara tentang kompensasi sebesar $ 150 ribu untuk "pencurian" fungsi rangeCheck (9 baris kode) dan menguji file untuk Android. Tiba-tiba ternyata hakim tahu cara memprogram .

Litigasi antara Oracle dan Google over Java API di Android telah berlangsung selama enam tahun. Gugatan pertama diajukan pada 2010, pada 2012 Hakim Alsup memutuskan bahwa API tidak tunduk pada hak cipta sama sekali, tetapi keputusan ini dibatalkan di pengadilan banding. Sekarang, juri telah menyetujui pembelaan lain terhadap klaim Oracle: antarmuka pemrograman Android telah disebut sebagai contoh penggunaan kode Java secara adil.


Programmer Dan Bornstein, yang memberikan kesaksian di persidangan, menamai mesin virtual Dalvik dengan nama sebuah desa nelayan kecil di Islandia. Foto: Dan Bornstein

Dari sudut pandang pengacara perangkat lunak bebas, ini bukan pilihan yang baik, karena penggunaan wajar masih menyiratkan bahwa undang-undang hak cipta benar-benar berlaku untuk Java API. Bahkan, mereka seharusnya tidak dikenakan hak cipta sama sekali, seperti yang diperdebatkan Hakim Alsup pada tahun 2012. Tapi ini lebih baik daripada tidak sama sekali.

Perwakilan Google menjawab bahwa bahasa pemrograman Java selalu "bebas dan terbuka" untuk digunakan - ini juga berlaku untuk implementasi ulang Java API.
