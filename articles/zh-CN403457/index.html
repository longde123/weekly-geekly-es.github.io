DNK | 总理 第一部分：设计



在为MINI-ITX标准开发新设计时，我们专注于严格的外观，增加了一些游戏玩家的攻击性，复古和优雅。 所有这些导致尺寸非常紧凑的对称线。 我们立即放弃了框架设计，使设计成为一体，其中每个零件都在CNC机器上加工而成，并且是刚性底座，紧固件和装饰元素。 因此，可以进行更有效的空气交换并获得出色的效果。 这种设计提供了出色的整体结构刚性，可防止振动和外部影响，例如液体和小的金属物体进入内部，这完全符合使用内部电源的个人计算机机箱的欧洲标准。



通过这种方法，用于制造PC机壳的标准技术变得多余了：切割，冲压，弯曲，焊接，铆接，喷漆，铁。 对于3个部分的空气交换，初始形状是通过以2500吨压力挤压铝而确定的。

在开发布局设计时，我们尝试了几种选择。 垂直（事实证明不够稳定），水平（在桌面上占用了大量空间）。 结果，当前的布局被认为是最佳的，并且机箱上部没有孔洞，可以扩展桌面空间。

技术指标


  • 材质： 6063铝
  • 外形尺寸： 280 x 140 x 200mm
  • 容量： 7.8公升
  • 净重： 2.2公斤
  • 主板： Mini-ITX
  • 图形加速器： 最大180 x 135mm
  • CPU散热器： 最高可达47mm
  • 电源： SFX，SFX-L
  • 信息存储设备： 2 x 2.5英寸+ 2 x 3.5英寸（取决于PCI-e卡）
  • USB 3.0前面板
  • 电源指示灯，硬盘指示灯
  • 肯辛顿锁

冷却方式


为了成功进行空气交换，我们采用了层状设计。 薄板设计的空气通道面积比带有许多孔的设计大一倍半。 在薄板设计中，主板和GPU前面的整个空间都参与了从顶部到底部的空气流动。 另外，没有气袋。



板条的最佳厚度和深度可产生有趣的视觉效果。 通风槽仅在直角处可见。 它们以大于15°的角度隐藏。


电源供应


机箱的整个正面都保留用于SFX，SFX-L标准电源。 推荐位置-风扇连接至主板。 这种布置将温暖的空气从箱体中抽出，并将其扔到前部上部。


信息储存


根据PCI-e扩展卡的特定配置，最多可以安装4个存储设备：

  • 双插槽卡-2 x 2.5英寸
  • 单插槽卡-2 x 2.5英寸+1 x 3.5英寸
  • 没有PCI-e卡-2 x 2.5英寸+ 2 x 3.5英寸



使用专业的声卡和视频捕获卡时，附加的3.5英寸驱动器可能很有用；在没有PCI-e卡的情况下，您可以组织相当大的服务器或NAS。

图形加速器


今天，几乎所有制造商都为小型系统生产视频卡。 您可以从Nvidia GeForce GT610到GTX 1070中选择图形。

我们将更详细地介绍一种新颖性。 去年，AMD推出了一系列用于专业用途的图形加速器：Radeon PRO WX，大大降低了专业领域的价格。 我们认为，其中最有趣的是Radeon PRO WX 5100 。



该加速器基于RX470游戏兄弟，具有256位内存通道和板载8Gb DDR5。 对于我们的第一个生产样本，我们一定会测试AMD Ryzen 7 1700 + Radeon PRO WX 5100套装。

不用螺丝刀组装


我们简化了外壳的组装/拆卸过程。 无需螺丝刀即可执行所有基本操作。 为此，使用了用于手动拧紧的弹簧螺钉和硅胶垫圈，以实现更可靠的紧固。 顺便说一下，主板，电源和硬盘驱动器的紧固件都装有橡胶防振插件。



俄罗斯收藏


对于我们的俄罗斯客户，我们计划在俄罗斯组装机壳和配置。 这将加快交货时间，降低成本，并使货到付款成为可能。 另外，对我们来说，提供技术支持和保修也将更加容易。
