Rappel de la version finale de Windows 1809

En raison de problèmes connus avec la suppression de fichiers d'un profil lors d'une mise à jour, Microsoft a décidé de révoquer la version finale de la mise à jour d'octobre 2018 de Windows 10 (1809)

Depuis le 6 octobre, l'assembly n'a pas été distribué via le service Windows Update. Si vous avez vous-même téléchargé l'image ISO - ne l'installez pas et attendez le nouvel assemblage.

Les versions actuelles de Windows Server 2019 et Windows 10 Enterprise (y compris LTSC et IOT) sont également affectées par ce problème.
