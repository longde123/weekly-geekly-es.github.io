甲骨文创始人拉里·埃里森的生平与成功

哈Ha！ 我建议翻译文章“亿万富翁甲骨文创始人和国际花花公子拉里·埃里森的生平和崛起，拉里·埃里森两次辍学，拥有680亿美元的财富 。 ”



现年74岁的甲骨文联合创始人拉里·埃里森（Larry Ellison）是技术领域最有趣的人之一。

乘游艇比赛，购买整个夏威夷或侮辱竞争对手，总是有很大的机会让我们感到惊讶的是，资本达684亿美元的拉里（Larry）下次还会做。

让我们追踪一下他从两次离开大学到国际花花公子和科技钛的道路。

注意，剪切下包含50张图像。

据《福布斯》报道，拉里·埃里森目前的身家为684亿美元。 这使他成为全球第七大富翁。



劳伦斯·约瑟夫·埃里森（Lawrence Joseph Ellison）于1944年8月17日出生于布朗克斯，以弗洛伦斯·斯派曼（Florence Spellman）的名字在一个单身母亲家庭中出生。


Oracle董事会执行主席兼CTO Larry Ellison在2014年旧金山的Oracle开放世界大会上作了演讲。

拉里（Larry）9个月大时，得了肺炎。 他的母亲将他送到芝加哥，与她的姨妈和叔叔莉莲（Lillian）和路易斯·艾里森（Louis Allison）住在一起。



他的养父路易（Louis）是俄罗斯移民，以“艾莉森（Allison）”为名，以纪念他移居美国的地点：埃利斯岛（Ellis Island）。



埃里森（Ellison）就读于芝加哥南部的一所高中，然后就读于伊利诺伊大学香槟分校（Urbana-Champaign）。



他的养母去世后，拉里退出了大学的第二年。 后来他回到芝加哥大学上大学，但在第一学期后退学。



最终，在1966年，年仅22岁的拉里（Larry）搬到了加利福尼亚州伯克利市，临近未来的硅谷，硅谷已经成为技术行业的鼎盛时期。



他乘坐豪华的蓝绿色“雷鸟”从芝加哥前往加利福尼亚，他认为这将给他的新生活留下深刻的印象。



Allison跳槽到工作，包括Wells Fargo和大型机制造商Amdahl等公司的工作。 在此过程中，他学习了计算机技能和编程。



转折点出现在Ellison加入Ampex时，Ampex是一家电子公司，该公司已为CIA创建了代号为“ Oracle”的数据库。



1977年，艾莉森（Allison）和合伙人鲍勃·米纳（Bob Miner）和埃德·奥茨（Ed Oates）成立了一家新公司-软件开发实验室。 该公司的初始资金为2,000美元，其中1200美元来自埃里森的口袋。


Oracle联合创始人

Allison及其公司受到IBM科学家Edgar F. Codd关于所谓的关系数据库的理论的启发，该数据库是一种存储和访问计算机系统信息的方法。 如今，这一理论已被视为理所当然，但在70年代，它是一个革命性的想法。



Oracle数据库的第一个版本是版本2 –没有版本1。在1979年，该公司更名为Relational Software Inc.，并在1982年正式更名为Oracle SystemsCorp。 以纪念其旗舰产品。



甲骨文已经迅速发展成为不断发展的计算机行业的主要驱动力之一。 1986年，该公司进行了首次公开发行股票，收入为5500万美元。



但是，在1990年，由于Allison后来称其为“一个令人难以置信的业务错误”，Oracle必须裁员10％，约400名员工。 该公司允许其卖方记录本季度的未来销售额，这意味着所有数字都失真了。 这导致了诉讼和监管机构的麻烦。



十年之初并不成功。 解决了这个巨大的错误之后，甲骨文宣布将要破产。 同时，诸如Sybase之类的竞争对手吸收了Oracle的市场份额。



这花了几年时间，但是到了1992年，Allison和Oracle得以使用新员工和流行的Oracle 7数据库来纠正此过程。



艾莉森以冒犯竞争对手而闻名。 在90年代的大部分时间里，他和甲骨文都与Informix竞争对手进行了PR争夺战，后者甚至将Dinosaur Crossroads广告牌放在甲骨文的硅谷办事处之间。



但是甲骨文继续与竞争对手竞争。 作为甲骨文的主要股东，数百万的埃里森（Ellison）继续增长。 他有几个昂贵的爱好，包括赛车。 这是艾莉森（Allison）在1995年的比赛中掌舵。



他还赞助了甲骨文美国帆船队，该队获得了2010年美洲杯冠军。



艾莉森甚至设法将潜在的损失变成了巨大的胜利。 1999年，埃里森（Ellison）的门生马克·贝尼奥夫（Mark Benioff）离开了甲骨文，从事新的Salesforce.com创业公司的工作。 拉里（Larry）是一位早期投资者，向他朋友的新公司投资了200万美元。



当贝尼奥夫得知甲骨文公司的埃里森正在开发与Salesforce直接竞争的产品时，他试图让他的导师离开公司董事会。 相反，艾莉森强迫贝尼奥夫解雇他，因此他保留了自己在Salesforce中的股份。



鉴于Salesforce的市值目前已达到1,150亿美元，即使竞争对手成功了，艾里森个人也能获利。 这导致了两位领导人之间复杂的关系，这种关系一直持续到今天。



实际上，除了Salesforce之外，90年代末的互联网繁荣还为Oracle带来了利润：所有这些新兴的互联网公司都需要数据库，而Oracle就在那里。



当史蒂夫·乔布斯（Steve Jobs）在1997年重返苹果公司担任首席执行官时，他邀请埃里森（Ellison）加入董事会。 艾莉森接受了要约，但过了一段时间，他觉得他无法继续。



当网络繁荣结束且价格低廉时，艾莉森的拥挤胸口使艾莉森得以通过增加成本来推动甲骨文。 例如，2004年，甲骨文以103亿美元的价格收购了PeopleSoft，这是一家人事软件提供商。



在2010年，甲骨文收购了Sun Microsystems，这是一家与甲骨文同时在1982年成立的服务器公司。 此次收购为Oracle提供了许多关键技术，包括对流行的MySQL数据库的控制。



从2010年代开始，艾莉森（Allison）开始沦为幕后黑手，将更多的职责转移给了马克·赫德（Mark Hurd）和萨夫拉·卡茨（Safra Katz）等可信赖的代表。



...但是他的花费并没有减少。 2012年，他收购了夏威夷拉奈岛98％的土地。



艾莉森（Allison）于2016年3月在拉奈（Lanai）成立了一家名为Sensei的初创公司，从事水培农业。



埃里森（Ellison）还于2014年收购了夏威夷低成本航空公司Island Air，此举是在该公司陷入财务困境两年后出售的。



艾莉森还投资了教育平台制造商Leapfrog Enterprises。



然而，并非所有埃里森的附带项目都能奏效：他是Theranos的首批投资者之一，这是一家血液检测初创公司，在创始人伊丽莎白·霍尔姆斯（Elizabeth Holmes）被指控欺诈后于2018年关闭。


Theranos创始人伊丽莎白·霍尔姆斯

...并住在加利福尼亚伍德赛德。 他在加利福尼亚州的Mirage Ranch有另一个家。



艾莉森还经营印第安维尔斯网球锦标赛。



据彭博社报道，埃里森的消费习惯是如此极端，以至于他的会计师菲利普·西蒙（Philip Simon）在2002年要求拉里“预算和计划”。



艾莉森结婚并离婚了四次。 加上他的巨大财富，这使他享有国际花花公子的声誉。 他们说他要和模特儿兼演员Nikita Kan约会。



埃里森有两个孩子。 他的女儿梅根（Megan）是奥斯卡提名的电影制片人，曾在电影《零黑暗的三十岁》和《美国喧嚣》中工作。



埃里森（Ellison）的儿子戴维（David）也从事电影业务制作电影，包括《终结者》和《碟中谍》。



2014年，拉里（Larry）正式辞去Oracle首席执行官的职务，将控制权移交给了Heard和Katz。 这时，他获得了世界第五富有的头衔。



2016年，艾莉森获得了个人奖金：1998年，他为甲骨文前首席执行官埃文·戈德堡（Evan Goldberg）的新业务投资了1.25亿美元，该公司是一家开发NetSuite业务管理软件的初创公司...



当NetSuite首席执行官Zack Nelson同意将该公司以93亿美元的价格出售给Oracle，而Allison获得其35亿美元的股份时，这最终奏效了。 NetSuite的投资者T. Row Price以埃里森的利益冲突为由，试图阻止这笔交易，但该交易于2016年11月完成。

甲骨文正在缓慢而稳步地将其业务转移到云中，这使拉里·埃里森（Larry Ellison）有能力与云计算市场的领先者亚马逊网络服务（Amazon Web Services）竞争。


Larry Ellison在Oracle OpenWorld '16上

Larry在Oracle之外还有其他事情要做。 他于2018年12月28日加入特斯拉董事会，是主要投资者。 2018年初，艾莉森称特斯拉首席执行官埃隆·马斯克为“密友”，并为他辩护。


特斯拉创始人埃隆·马斯克（Elon Musk）

艾莉森还是著名的慈善家。 2016年5月，艾莉森向南加州大学的癌症治疗中心捐赠了2亿美元。



2010年，他签署了比尔·盖茨（Bill Gates）和沃伦·巴菲特（Warren Buffet）首次提出的“礼物的承诺”，并承诺在他去世前捐出其财富的95％。



除了这些问题，Larry仍在与Oracle积极合作。 他是公司董事长兼首席技术官。



也许他不在首位，但他仍然是公司的面孔，帮助带领公司度过了业务不稳定的时光...
