Panduan Mendengarkan Jarak Jauh Samsung TV CIA Diterbitkan


Malaikat yang menangis

Penggemar seri Doctor Who sci-fi mengingat ras alien predator Weeping Angels. Ini adalah karakter dari episode "Don't Blink" - episode kesepuluh dari musim ketiga (2007) dan salah satu episode terbaik dalam sejarah Doctor Who.

Dari sudut pandang orang luar, para malaikat terlihat seperti patung batu, yang matanya sering ditutupi dengan telapak tangan. Tetapi jika Anda berkedip, melihat Malaikat, Anda akan segera melihat perubahan posisi patung yang sulit dipahami - tampaknya telah menjadi lebih dekat. Dan Tuhan melarang untuk berpaling untuk waktu yang lama.



Jika Anda berpaling, Malaikat Menangis akan mendekat. Faktanya adalah bahwa alien ini hanya bergerak ketika tidak ada yang melihat mereka, bahkan jika mereka hanya berkedip. Dalam arti tertentu, mereka mirip dengan karakter mitos Yunani, Medusa gorgon, yang membuat semua orang yang melihatnya membatu. Hanya di sinilah prinsip ini yang sebaliknya.


Gorgon Medusa, salah satu dari tiga saudara perempuan Gorgon (melukis oleh Caravaggio). Menurut legenda, dia adalah seorang gadis dengan rambut yang indah, tetapi dewa Poseidon mengambilnya di kuil Athena, di mana seorang gadis yang tak berdosa berusaha bersembunyi dari pengejaran Allah. Sebagai balas dendam atas penghujatan di pelipisnya, Athena mengubah rambutnya menjadi hydra

Mitologi Yunani dan fiksi ilmiah modern secara luar biasa bergabung di dalam laboratorium rahasia CIA dan MI5, tempat perangkat lunak dikembangkan untuk kebutuhan intelijen asing. Programmer CIA yang menyukai budaya pop modern , tentu saja, menonton Doctor Who, dan episode itu dengan Weeping Angels. Itu sebabnya mereka memberi nama seperti itu pada bookmark untuk TV Samsung. TV yang mengikuti penonton adalah Malaikat Menangis yang kecil, malaikat yang menangis, hanya mencoba untuk berkedip atau terlalu banyak bicara.

Pada 7 Maret 2017, Wikileaks mulai menerbitkan koleksi dokumen rahasia Vault 7 oleh Badan Intelijen Pusat AS. Bagian pertama dari koleksi Year Zero berisi 8761 file , termasuk daftar berbagai malware, virus, trojan, puluhan eksploitasi dan payload 0day untuk mereka, sistem kendali jarak jauh dan dokumentasi terkait.

Pilihan yang diterbitkan termasuk dokumentasi untuk penandaan Weeping Angel untuk TV Samsung dengan mikrofon internal dan pengenalan suara (kontrol suara) diaktifkan. Malware ini dibuat oleh tim pengembangan Embedded Development Branch (EDB) CIA, bersama dengan MI5 / BTSS intelijen Inggris. Program menambahkan mode 'Palsu-Off' ke TV ketika TV terlihat dimatikan, tetapi pada saat yang sama merekam percakapan di ruangan dan mengirimkannya ke server web CIA.

Pada 21 April 2017, Wikileaks menerbitkan informasi baru pada tab Weeping Angel - panduan pengguna terperinci untuk agen CIA . Dokumen ini bertanggal 28 Februari 2014 ( pdf ). Dalam dokumentasi teknis, alat ini disebut EXTENDING. Menurut Wikileaks, ini adalah versi asli dari bookmark, dibuat di MI5 / BTSS intelijen Inggris dan ditingkatkan di CIA. Kolega dari Inggris dan AS mengoordinasikan pekerjaan mereka pada alat ini dan berbagi pengalaman mereka dengan mengorganisir Lokakarya Pengembangan Bersama - lokakarya pengembangan bersama.

Tiga lusin halaman buku petunjuk menguraikan hal-hal berikut:

  • fitur-fitur utama dari bookmark MENUNGKU;
  • konfigurasi malware: Ubuntu 12.10 ISO image termasuk dalam paket untuk membuat mesin virtual Linux di mana file pengaturan terenkripsi dihasilkan (installer Oracle VM Virtual Box Windows memungkinkan Anda untuk mengatur lingkungan untuk menjalankan VM Ubuntu ini) - konfigurasi harus dibuat hanya pada komputer yang aman terputus dari Internet; ada juga skrip wlan.bat untuk mengkonfigurasi Host Virtual Network Virtual di laptop;
  • proses instalasi;
  • kompatibilitas dengan TV Samsung;
  • proses menginstal server web untuk menerima informasi dari TV - server web diinstal menggunakan installer Windows XAMPP, ada juga server web Server Android PAW, paket termasuk file .apk dan folder PAW2 yang dikonfigurasi;





  • memproses data audio yang direkam melalui mikrofon internal menggunakan program Windows ECDLive.exe;
  • prosedur untuk menghapus bookmark dari TV (dalam konfigurasi awal, "tanggal kematian" dari bookmark diatur);
  • pengujian dan kemungkinan masalah operasional;
  • Masalah yang diketahui dan pembatasan bookmark
  • dekripsi kode kesalahan (Lampiran A).

Dokumen tersebut menyatakan bahwa program ini ditujukan untuk model Samsung F Series (firmware 1111, 1112 dan 1116), dan malware tersebut telah dikonfigurasikan sebelumnya pada komputer pribadi yang menjalankan Linux. Instalasi pada TV dilakukan menggunakan stik USB. Program ini dapat bekerja dalam tiga mode: 1) perekaman audio terus menerus; 2) rekaman audio hanya ketika TV mati; 3) rekaman audio hanya ketika TV menyala.

Seperti dapat dilihat dari dokumentasi, file audio yang direkam dari TV tidak ditransmisikan melalui Internet, tetapi melalui WiFi - ke laptop atau smartphone Android yang terletak di dekatnya. Itu sudah dapat terhubung ke Internet, melalui itu Anda bisa mendapatkan arsip file yang direkam atau mendengarkan secara real time. Mungkin, bookmark ini paling cocok untuk pemasangan di TV "publik", yang terletak di gedung perkantoran, lobi hotel, kafetaria, penata rambut, dll.

Malware file sendiri dari arsip Vault 7 berjanji akan dipublikasikan setelah memeriksa dan menutup kerentanan oleh produsen.

PS Sekarang CIA dan FBI sedang melakukan penyelidikan bersama untuk mengidentifikasi karyawan yang terlibat dalam kebocoran dokumen. Anehnya, selama 1,5 bulan tidak mungkin untuk menghitungnya.
