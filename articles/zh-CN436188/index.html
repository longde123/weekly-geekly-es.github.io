DJI智能控制器让您尽在指尖



除了流行的无人机和最新的OSMO Pocket稳定器外，DJI还为拉斯维加斯国际消费电子展CES 2019带来了一个非常有趣的小工具。

DJI Smart Controller是一款内置超亮显示屏并预装Android操作系统的遥控器。 该控制器将最新的Mavic 2型号变成了Phantom 4 Pro +（PLUS）和Advanced +（PLUS）（包括带屏幕的遥控器）的替代品。 此外，在为专业拍摄选择特定的直升机模型时，DJI Smart Controller具有很多明显的优势，这些优势可能是决定性的。 今天我们将讨论它们。

人机工程学和外观




乍一看，由于遥控器的大尺寸，似乎不方便使用。 但是，如果您将小工具拿到手中，所有的疑问都会消失：DJI做得很好，而且在人机工程学方面，智能控制器确实给人留下了深刻的印象。 他非常安全地躺在他的手掌中，不会滑落，并且可以通过拇指轻松触及所有按钮和操纵杆，而无需改变手柄。 没有裂缝或间隙-车身部件尽可能紧密可靠地安装。

在前面板上，您会发现一个5.5英寸的触摸屏，两个操纵杆以及一组标准的控制和导航键：“返回”按钮，主页，飞行停止，电源，5D键，飞行模式开关等。



完整的HDMI端口，microSD卡插槽，USB-A连接器，倾斜和相机设置，开始录制按钮，快门按钮和通风格栅被带到了上边缘。

后面板上有两个通过跳线连接的天线，该跳线隐藏了一个用于存放可移动操纵杆的隔间，一个扬声器，两个可自定义的键和一个用于主动冷却内部系统的进气格栅。 顺便说一下，当您折叠天线时，操纵杆的插槽会“包裹”在小工具内部，在运输过程中丢失珍贵的配件变得更加困难。

底部有一个麦克风，一个USB-C端口和两个螺孔。

萤幕




我想马上注意到DJI智能控制器的屏幕非常亮-1000 cd / m2。 即使戴着墨镜与他一起工作，您也可以清楚地看到显示屏上发生了什么。 因此，不再需要用于智能手机和其他设备的各种遮阳板。 所有细节都将在显示屏上区分出来，并且在任何情况下图片都将保持清晰多汁。 为了方便用户进行设置，您可以根据照明或特殊的SRE功能激活亮度级别的自动调节，该功能可在阳光直射下调节屏幕上的暗区和亮区。

矩阵的对角线为5.5英寸，分辨率为1920 x 1080像素。 屏幕是触敏的；用于浏览设备菜单。 导航项目的另一种方法是外壳上的5D键，操纵杆和按钮。 这样做是有原因的：遥控器在-20度的温度下操作，并且按照惯例，触摸屏对戴手套的触摸手无响应。

软件和通讯模块




智能控制器可在Android 7.1.2上运行。 DJI对OS进行了一些外观和软件更改，但实际上许多Android对此早已熟悉。

使用microSD卡上的内置浏览器或APK文件，可以在遥控器上安装任何应用程序。 例如，您可以启动照片或视频编辑器，并在拍摄地点立即编辑新鲜的材料，然后在Facebook或Instagram上共享它们。 同时，DJI警告该公司“对使用第三方应用程序的安全性不承担任何责任”。 因此，我们建议您谨慎选择内容，而不要下载会对遥控器的操作产生不利影响的非正式程序。

设备上没有太多预安装的应用程序。 有一个DJI Pilot可以控制最新型号的工业无人机和DJI GO4。据该公司称，可用程序的列表将随固件的每个新版本而扩展。

用户具有16 GB的内部存储器，但是由于使用了microSD卡，该数量很容易增加。 该设备支持最大128 GB的介质，并且读写速度最高为UHS-1 Speed Grade 3。

此外，在DJI智能控制器上，您将找到一套蓝牙4.2，用于现代小工具的Wi-Fi 802.11a / g / n / ac标准模块以及GPS + GLONASS卫星定位系统。

DJI智能控制器正在工作




现在让我们来谈谈最有趣的事情-DJI Smart Controller可以做什么。

在控制直升机方面，完整的Mavic 2遥控器和DJI Smart Controller之间没有显着差异。 一切工作正常，无人机立即响应来自控制器的所有命令，视频被传输到屏幕而没有延迟或干扰。

遥控器能够以每秒60帧的速度播放4K视频，并通过HDMI端口将图像传输到外部设备。 此外，可以通过HDMI电缆将图像实时广播到外部屏幕。 这在有几个人参与拍摄的情况下特别有用。 假设您控制一架无人驾驶飞机，并在内置屏幕上跟踪空中发生的事情，客户或导演使用自己的监视器指导拍摄过程，并告诉您选择哪个角度。 您不会互相干扰，也没有人关闭图片。

该控制器与OcuSync 2.0无线通信系统的无人机兼容。 现在没有多少这样的设备：Mavic 2 Pro和Mavic 2 Zoom。 DJI承诺该清单将很快补充Mavic 2 Enterprise，当然还会补充将来出现的其他直升机。

OcuSync 2.0支持在2.4和5.8 GHz频率之间自动切换，以避免干扰并在遥控器和无人机之间提供稳定的通信。 得益于此，DJI智能控制器只要能够在FCC网络中，无电磁干扰，空旷且海拔约100米的FCC网络中工作，就可以接收长达八公里的信号。



内置的麦克风和扬声器将吸引视频博客和摄影师，他们在飞行中对正在发生的事情发表评论，并希望与朋友和同事保持联系。 只需连接到Wi-Fi接入点，打开广播，您不仅可以展示周围的美景，还可以谈论它。

拍摄过程中智能手机或平板电脑是否放电？ 只需通过USB-A连接器将其连接到遥控器，缩小的设备将在几分钟内恢复使用。 DJI Smart Controller本身通过附带的适配器支持快速充电。 给控制器的电池充满电大约需要两个小时。 电池续航时间为两个半小时-如果您有几个备用电池，这应该足以在Mavic 2上拍摄约两个小时的视频。 该设备可在-20°C至+ 40°C的温度范围内工作。 而且，如上所述，您可以使用遥控器，甚至可以戴着手套通过机械键浏览菜单项。

有两种方法可以在社交网络上共享素材。 首先是将应用程序直接安装在遥控器本身上。 第二种方法是将照片和视频下载到您的移动设备。 为此，请激活Go Share功能并使用智能手机上的DJI GO 4扫描生成的QR码。 一切就绪-捕获的照片和视频的预览将出现在手机屏幕上，您只需选择需要下载的文件即可。

价钱




价格，不幸的是，咬伤-在官方商店DJI Smart Controller将以649美元的价格出售。 如果您是上一代Mavic系列产品或Phantom 4（Advanced，Pro，Pro V2.0）的模型的所有者，并且长期以来希望购买用于遥控器的明亮显示器，那么我们建议您注意5.5英寸屏幕的CrystalSky。 但是Mavic 2的飞行员更幸运。 使用DJI Smart Controller，他们可以专业水平地拍摄视频，在晴朗晴朗的天气，南部或雪坡上舒适地控制无人机，在社交网络上播放视频，并使用第三方程序立即在屏幕上进行编辑。 有许多使用设备的选项。 而且，与此同时，您不必占用其他附件包中的空间。
