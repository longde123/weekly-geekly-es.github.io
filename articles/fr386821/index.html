Pour la commodité des clients, les centres sont situés dans une salle dédiée où les opérations ne sont effectuées qu'avec des colis. Cette méthode d'enregistrement permet à l'opérateur de trouver l'envoi le plus rapidement possible en lisant le code à barres sur l'avis du client avec un scanner spécial.
