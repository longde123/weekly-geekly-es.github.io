关于音频的聆听内容：15个播客

播客是我们最喜欢的格式之一，随着语音激活平台和爱丽丝（Alice）等助手的流行，他开始重新引起了观众的注意。

尤其是对于本系列，我们找到了关于声音和音频的最佳英语播客，但我们将与Audiomania的专家一起从播客的短篇故事开始摘要。


Sascha Kohlmann / CC摄影

音效


[ iTunes ] [ 网络版本 ]

这是俄语的播客，内容涉及声音，高端和Hi-Fi设备，黑胶和音频格式。 自2013年以来已发布。 迄今为止，已记录了43期。 在该计划中，专家们用简单的词来解释音频系统的技术特征，并讨论来自技术界的有争议的问题，向听众介绍新的发展和品牌的历史。

在Habré上，有播客最有趣版本的文字记录 。 它们按主题排序，这将帮助您快速浏览并确定有关“声音”的信息。


业余逻辑电视


[ iTunes ] [ 网络版本 ]

该播客由声音工程师George Thomas和开发人员Tommy Martin创建。 该节目致力于无线电技术，每月发行几次。

该程序的第一个版本发布于13年前， 卡特里娜飓风过后不久，它专门用于应急电源。 演讲者涵盖了广泛的主题-从声音文物到Raspberry Pi 。


初学者发烧友


[ iTunes ] [ 网络版本 ]

顾名思义，此播客的主要受众是初学者。 主导讨论廉价音频技术，并就设备选择提出建议。 在最近一期中，讨论了专门研究便携式音频的CanJam Show的最新消息 。 一年半以来，共发行了27期。 从iTunes上的评分和reddit上的正面评价来看，播客的受欢迎程度只会增加。


什么高保真音响？


[ iTunes ] [ SoundCloud ]

有关音频和视频的杂志什么是Hi-Fi？ 自1976年问世以来。 同名的播客是著名出版物的子公司。 该项目还比较年轻，仅在今年5月才发布了第一个版本。 演讲者已经设法讨论了最新的Hi-Fi和Apple“智能”扬声器的功能，并概述了几种耳机。


Darko.audio


[ iTunes ] [ SoundCloud ]

为那些“力求更好的音质和热爱思考的人们”提供的世界新闻音频和价格合理的Hi-Fi设备的概述。 通话时间主要讨论音频设备和音乐格式。 演示者自己考虑了专用于流音乐内容和相关服务方法的最佳问题 。


音调控制


[ iTunes ] [ 网络版本 ]

感谢播音员贾斯汀·牛顿和吉他手Derek Heidemann的支持，此播客得以存在。 有关吉他，新旧设备，录音和音乐写作的主要演讲。 播客不仅可以帮助您了解放大器和吉他踏板的特性和功能，还可以浏览乐器的选择。 吉布森“太空”吉他以及公司的破产，电视广播员和爵士乐大师芬德·迈泰奥拉的混合体-这些都是讨论的主题示例。 演讲者定期检查吉他和相关设备。 播客每周播出两次。


AES


[ iTunes ] [ 网络版本 ]

自1948年以来，美国组织AES（音频工程协会）一直在开发，修订和发布音频和相关行业的标准。 她还发行了完整的科学杂志。 播客充当该杂志的附录。 每期简短（约7分钟）描述了新期的主题。


照florantevaldez / PD

偶尔播客


[ iTunes ] [ Player.fm ]

发烧友和The Occasional Magazine 兼职编辑播客。 大多数问题均以受邀嘉宾的名字命名。 其中包括音响工程师，工程师和设计师。

最新系列讨论了音频市场成功的秘诀和耳机设计的当前趋势。 该节目的第五期直接录制在传奇的曼利实验室办公室。 公司的一位创始人领导工厂的领导者，与他们讨论了开发放大器的细微差别，并谈到了他如何组织工作过程 。


AV狂暴


[ iTunes ] [ 网络版本 ]

这个播客就像一个有趣的早晨广播节目。 但是，在每个问题中您都可以找到有用的东西。 演示者和他们的客人提供有关选择家庭影院组件，投影仪，低音炮和其他设备的建议。

该播客由前Audioholics评论家Tom Andry和专业影音设备专家Rob H.主持。 大多数情况下，问题以问答形式构建。 听众会问有关家庭音频系统的问题，并询问购买设备时要寻找的内容。 展会网站上每个问题的描述中都有时间码。

每周剧集持续约两个小时。


AV论坛


[ Player.fm ] [ 网络版本 ]

业界领先的在线音频和视频社区的播客。 演讲者讨论了家庭影院和音响设备的功能。 顾名思义，播客不仅限于谈论音频。 在最近的一集中， 对《不可能的任务：后果》进行了回顾 。

要特别注意测试。 在播客存在期间，演示者检查了JBL 耳机 ，Arcam 放大器 ，BenQ 投影仪和其他音频设备的工作质量。 此外，在某些问题上还有一些有趣的收藏，例如，最好的OLED电视。 每期持续约90分钟。


Avexcel


[ iTunes ] [ 网络版本 ]

播客的创始人称其为“选择最佳家庭影院设备的指南”。 但是，播客超出了本主题的范围。 演讲者谈论扬声器，低音炮，声音校准，接收器和市场新闻。 他们还讨论了海滩派对的吊杆盒和蓝光电影 。 每个问题都以听众提问的答案结尾。 播客每周播出一次，持续约一个小时。


欧姆定律


[ iTunes ] [ 网络版本 ]

此播客可以归类为流行科学。

PS Audio联合创始人Paul McGowan谈到了音频设备，声音，音乐及其播放。 在播客中，他分享了他在音频设备市场40多年的经验。 他的公司生产从乙烯基音频系统的组件到现代Hi-End设备的所有产品。

播客内容丰富。 每个问题都是对有关音频设备或声音调整的问题的解答。 麦克高恩说，您需要在放大器和前置放大器的规格中注意什么 ， 为什么扬声器带有后置高音扬声器并回答其他问题。 对于那些想更好地了解音频设备的技术特性的人，播客会特别有趣。


头fi


[ iTunes ] [ Player.fm ]

同名在线社区的播客，致力于个人和便携式音频。 在播客存在期间-从2007年到2016年-发行了80多集。

该计划的首批客人之一是著名的模拟录音恢复者Steve Hoffman。 除采访外，演讲者还分享行业新闻并审查音频设备。 例如，问题之一专门用于通道内监视器的设备。


家庭影院极客


[ iTunes ] [ Player.fm ]

首席播客是AVS论坛门户的专业音乐家和编辑Scott Wilkinson。 他与家庭影院和音频设备领域的领导人进行了交谈。 最新一期的来宾是便携式音响专家Till Hertsens，他回答了听众关于选择耳机的问题。

尽管该节目已不再出现，但所有373集均仍以播客格式提供。


SoundGuys播客


[ iTunes ] [ 网络版本 ]

同名在线出版物的流行科学播客，专用于耳机和扬声器。 幽默的演讲者讨论了音频的最多样化方面。

他们将告诉您MP3格式的历史，在智能手机上听音乐的功能以及音乐对我们的消费者习惯的影响。 每集持续约20分钟。
