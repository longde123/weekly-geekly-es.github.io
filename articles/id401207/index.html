Microsoft sedang mengembangkan shell adaptif untuk Windows 10



Microsoft diduga bekerja pada peningkatan signifikan ke shell Windows 10. Di dalam perusahaan, proyek ini dikenal sebagai "cshell," yaitu, Composable Shell. Pembaruan akan memungkinkan Windows 10 untuk bekerja pada berbagai perangkat yang sangat luas. Wartawan di Windows Central mengklaim bahwa Composable Shell akan menjadi shell adaptif yang dapat beradaptasi dengan gadget yang menjalankan sistem operasi Windows 10.

Sekarang Microsoft menggunakan berbagai kerang untuk Windows 10 pada PC, OS seluler dan sistem untuk Xbox. Shell Komposit menghilangkan kebutuhan untuk menggunakan shell sendiri untuk masing-masing jenis perangkat yang disebutkan. OS akan disatukan dan akan dapat bekerja, kecuali untuk perangkat yang disebutkan, pada sistem lain.

Asumsi lain adalah Microsoft's Composable Shell memainkan peran penting dalam pengembangan perangkat, yang disebut "Surface Phone." Windows 10 akan berfungsi pada perangkat ini. Kemungkinan besar, dalam waktu dekat kita akan melihat gadget kecil dari Microsoft dengan platform perangkat keras berbasis ARM. Jika demikian, tetapi kemungkinan besar ya, maka perusahaan benar-benar membutuhkan cangkang adaptif universal.

Composable Shell akan memainkan peran penting untuk Continuum, fitur kritis Windows 10 Mobile. Tahun lalu, perusahaan menunjukkan versi demo cangkangnya di acara Ignite 2016. Kemudian perusahaan mendemonstrasikan bagaimana Continuum untuk ponsel bekerja pada perangkat seluler dan terlihat sangat mirip dengan PC dan gadget seluler.

Ponsel Microsoft yang akan datang akan memiliki layar lipat, dan Shell Komposit akan menunjukkan kemampuannya. Jadi, ketika dilipat, Windows 10 akan berfungsi dalam mode seluler. Saat Anda membuka layar, Windows 10 dengan Composable Shell akan masuk ke mode desktop. Pada saat yang sama, hampir semua fitur Windows 10 akan tersedia untuk pengguna perangkat seluler dan mereka yang bekerja di desktop.

Sekarang pengguna Windows 10 (setidaknya sebagian dari mereka) mengharapkan tampilan Surface Phone dan Windows 10 pada arsitektur ARM. Sayangnya, belum ada shell universal, jadi tidak ada gadget jenis ini juga. Versi desktop Windows 10 tidak diadaptasi untuk bekerja pada perangkat dengan layar kecil dengan diagonal kurang dari 6 inci. Masalah ini hanya akan dipecahkan, dan Windows 10 akan menjadi sistem universal.


Composable Shell - shell yang dibagi menjadi beberapa modul - komponen. Sistem dapat mentransfer data antar komponen individu, jika diperlukan.

Shell akan benar-benar universal. Windows 10 Mobile, misalnya, dapat berfungsi pada PC desktop, tetapi versi desktop sistem operasi ini akan berfungsi tanpa masalah pada perangkat seluler. Windows 10 juga akan digunakan untuk bekerja di perangkat seperti Hololens dan Xbox.



Untuk Microsoft, pengembangan ini akan berguna dalam arti bahwa korporasi tidak lagi harus memelihara dan memperbarui setiap modifikasi OS-nya untuk berbagai jenis perangkat. Lingkungan Composable Shell akan hampir sama di semua perangkat. Benar, masih akan ada perbedaan antara fungsi perangkat yang berbeda. Jadi, Xbox akan "diasah" untuk bekerja dengan game, versi seluler akan disesuaikan untuk smartphone, dengan pengurangan jumlah fungsi.

Microsoft sekarang merencanakan transisi yang mulus untuk mengintegrasikan berbagai perangkat ke dalam satu Shell Windows. Pengoperasian OS akan bersifat universal, terlepas dari gadget mana OS diinstal. Saat ini, Windows Shell sangat berbeda tergantung pada edisi Windows 10. Sebagai contoh, versi mobile dari sistem seperti itu menggunakan Windows Shell bukan versi yang sama dengan sistem untuk komputer pribadi.

Korporasi saat ini bekerja pada desktop penuh untuk Windows 10 Mobile melalui Continuum. Secara umum, jika semua hal di atas benar-benar terjadi, pengguna akan menunggu pembaruan besar berikutnya untuk Windows 10. Secara umum, Microsoft masih sangat aktif mempromosikan sistem operasi Windows 10 yang baru. Kemungkinan besar, manajemen korporasi menekankan bahwa semakin banyak pengguna bekerja dengan OS ini, dan setelah bekerja dengan sistem operasi baru, semakin sedikit pengguna yang dapat meninggalkannya setelah beberapa waktu ...
