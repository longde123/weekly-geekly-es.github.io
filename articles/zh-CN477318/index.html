PHP内部


  • [RFC]弱映射 -在PHP 7.4中，通过特殊的WeakReference类添加了对弱链接的支持。 但是实际上，应用程序需要一个WeakMap集合，该集合不能基于WeakReference来实现。 实际上，建议添加它。
    隐藏文字 
    $map = new WeakMap(); $obj = new stdClass(); $map[$obj] = 42; var_dump($map); // object(WeakMap)#1 (1) { // [0]=> // ["key"] => object(stdClass)#2 (0) {} // ["value"] => int(42) // } // } //   , //      WeakMap unset($obj); var_dump($map); // object(WeakMap)#1 (0) { // }
  • [RFC]在ext / dom中实现新的DOM Living Standard API-在内核扩展中实现对新DOM标准的支持的提议是一致的。
  • [RFC]弃用Backtick运算符（V2） -被投票拒绝。
  • [RFC]联合类型2.0-提案几乎被一致接受，在PHP 8.0中，我们正在等待组合类型。 现在可以在指定类型的任何地方使用语法T1|T2|...
