macOS + Windows 10 pada drive eksternal

Ketentuan Ada MacBook Pro Awal 2015 tanpa drive internal. Diperlukan untuk mengatur di atasnya macOS dan Windows 10 dalam versi dual-boot dari satu drive eksternal.


Relevansi Drive internal macbook modern disajikan oleh PCI-E SSD dalam faktor bentuk milik Apple. 128â€“256 GB non-asli dapat dibeli dengan harga di kisaran 5-10 ribu rubel. [ 1 , 2 ]. Di sisi lain, jika Anda memiliki drive eksternal yang cukup cepat (atau tuntutan kecepatannya rendah), Anda dapat mem-boot darinya.


Rintangan . Anda tidak akan dapat menginstal Macos tanpa (a) drive tambahan untuk pemasang dan (b) opium kedua untuk menulis pemasang ini ke drive tambahan ini (ditambah file dengan pemasang sendiri) - instruksi dari Apple .


Dengan tidak adanya poppy kedua dapat ditulis di bawah Windows.

Softina TransMac (ada uji coba) sesuai dengan instruksi dari para penggemar menggunakan gambar yang sudah jadi ( 10.11 El Capitan , 10.12 Sierra ).


Mac OS 10.13 High Sierra Saya menolak untuk menginstal pada drive eksternal. Di sisi lain, 10,10 Yosemite dan lebih tua tidak akan berdiri di atas bunga poppy modern.


Ketika MacOS diinstal, cara standar untuk menginstal Windows dari sana adalah Boot Camp. Namun, itu tidak memungkinkan instalasi pada drive eksternal. Mungkin saja untuk menulis pemasang Windows ke drive tambahan (dengan alat Makosi standar, menggunakan UNetbootin, Rufus, atau cara lain), tetapi itu juga tidak akan mengizinkan instalasi pada drive eksternal.


Solusi (a) Alih-alih bootkamp dengan tangan kami, kami menyiapkan bagian yang sesuai pada drive [ 3 , 4 ], setelah itu (b) alih-alih installer Windows, gunakan WAIK untuk mengunggah bootloader dan file instalasi ke drive [ 5 ]; (c) menginstal Windows dengan memuat melalui Opsi yang dijepit ke item Boot EFI; (d) instal paket driver dari direktori WindowsSupport (yang kita dapatkan dengan me-reboot ke macOS, membuka bootkamp dan menemukan di menu "Unduh perangkat lunak dukungan Apple Windows terbaru"). Saya mengujinya - itu berhasil.
