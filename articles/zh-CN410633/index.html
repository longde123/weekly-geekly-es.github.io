万好D9：经济实惠的3D打印机，打印面积大


Wanghao不仅继续升级现有3D打印机，还用新设备补充了产品阵容。 复制器9不是一台，而是三台打印机，它们基于相同的原理，但是打印区域的大小不同。 在发布D9之前，该公司并没有针对大的打印区域，也许D5除外，但是它只有令人印象深刻的高度，并且该打印机属于价格较高的类别。

这是什么


复印机9以经典的门式原理为基础，并由铝制结构型材组装而成，该型材既是框架又是滚筒的导向装置。 挤出机-一台，餐桌加热-库存。 制造商提请注意以下事实：发动机支架和其他对打印机的几何形状至关重要的细节不是从钢板弯曲的，而是通过铣削制成的。

万好D9 / 500


  • 打印区域的尺寸，毫米：500 x 500 x 500。
  • 层厚度，以微米为单位：100。
  • 印刷温度最高，°C：300。
  • 餐桌加热：是的。
  • 打印速度，最高，mm / s：70。

该行中的较旧型号具有预算段打印区域的记录。 对于那些需要相对频繁地打印整个大型零件的打印机，但是购买工业级大幅面3D打印机不切实际。 较年轻的D9 / 300机型的打印面积已经达到300 x 300 x 300毫米-比流行的Wanhao i3更大。 中间型号D9 / 400的打印区域为400 x 400 x 400毫米。



在上图中-步进电机的铣削紧固件和Z轴轴线螺丝的螺母。

其他功能


万好D9配备了触摸屏控制器，该控制器支持断电后继续中断打印。 使用带有伸缩式探头的传感器（BLTouch的类似物）可实现工作台的自动校准。 滚轮滑车配备了一个可防止反冲的夹具。

谁需要复制器9


这种设计的打印机不能吹嘘高打印速度，由于缺少封闭的相机，很难使用具有明显收缩率的塑料来工作。 但是较大的工作区域将使您无需胶合即可完全打印PLA，SBS或PETG的大型零件。 在原型制作，cosplay设备生产中用于印刷版图的过程中，将需要最大尺寸为500 x 500 x 500 mm的旧型号。
