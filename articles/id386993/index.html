Rostelecom sangat menyukai game



Hari ini platform game Rostelecom telah dibuka. Game (Beta) dengan katalog game dari mitra dan toko kunci untuk game Steam dan GOG. Perusahaan komunikasi terbesar mengadakan presentasi tertutup proyek dan mengumumkan bahwa itu adalah distribusi yang akan menjadi fokus utama perusahaan untuk beberapa tahun ke depan.

Sejauh ini, Mail.ru Group, Datcroft, Destiny Development, dan Alawar adalah mitra, tetapi daftarnya akan berkembang. Mitra menerima 50% dari keuntungan dari permainan "setelah membayar semua komisi." Untuk memposting game di portal, pengembang harus membayar 3000 rubel.

Layanan Rostelecom digunakan oleh lebih dari 100 juta penduduk Rusia. Operator telekomunikasi bermaksud untuk secara aktif mempromosikan layanan baru di antara para penggunanya, tetapi potensi audiens dari portal game tidak terbatas pada mereka.
