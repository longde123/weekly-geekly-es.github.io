Bagaimana kamu bisa lupa? Untuk memahami hal ini, Anda perlu mengendarai mobil listrik selama beberapa waktu. Minggu pertama setelah pembelian, saya perlu membiasakan diri dengan templat baru, atau lebih tepatnya, dengan tidak adanya templat lama - agar mobil dapat dikendarai, ia harus menghidupkan mesinnya (yaitu, ia akan mengeluarkan suara dan bergetar). Jika Anda adalah pengemudi mobil dengan mesin pembakaran internal, bayangkan Anda masuk ke dalamnya dan putar kunci ke ujung (tekan tombol Start tanpa menekan rem). Anda tinggal nyalakan alatnya. Setelah akhir perjalanan, mesin pembakaran internal perlu dibungkam (sistem start / stop jauh dari mana-mana) - mesin pembakaran internal, yang tidak ada dalam mobil listrik. Setelah berhenti, Anda dengan hati nurani yang baru saja keluar dari mobil listrik (semua pengalaman bertahun-tahun memberi tahu Anda bahwa jika mesin tidak bekerja, maka Anda mematikannya dan mengambil kunci - Anda dapat menutup pintu dengan aman), tekan tombol di pintu untuk menutupnya dan mobil mulai menyala "Squeak" - memperjelas bahwa Anda lupa mematikannya dengan menekan tombol Daya. Autopilot


Mobil ini memiliki kontrol jelajah pintar dan sistem retensi jalur.

Kontrol pelayaran beroperasi dari 0 km / jam dan menjaga jarak ke kendaraan di depan. Mungkin berhenti sendiri dan kemudian terus bergerak. Jika setelah berhenti 3 detik berlalu, maka Anda harus mengaktifkannya lagi - ini dapat dilakukan dengan menekan pedal "gas" atau tombol pada roda kemudi. Secara umum, dia sangat bodoh. Tidak mengerti tanda-tandanya. Terkadang membingungkan pengendara sepeda dan mobil. Ternyata itu hanya berfungsi sesuai data dari frontal radar. Bagian terburuknya adalah bahwa ia tidak membedakan antara hanya benda berdiri di jalan. Ini mungkin dilakukan agar radar tidak membingungkan jalan itu sendiri dan kendaraan di atasnya. Misalnya, dalam situasi di mana mobil turun, dan kemudian di depannya jalan tiba-tiba naik - radar dapat mengambil ini untuk mobil di depan dan menyalakan rem darurat.Untuk mencegah hal ini terjadi, ia mengabaikan semua objek dengan kecepatan nol. Yang buruk adalah bahwa kadang-kadang situasi seperti itu muncul - saya dari jauh, atau, misalnya, dari belokan saya berkendara ke persimpangan di mana mobil sudah berdiri di barisan saya untuk mengantisipasi lampu hijau. Kontrol pelayaran saya akan mengabaikannya dan saya harus menekan pedal rem sendiri. Tetapi jika radar memperbaiki mobil saat bergerak, maka tidak ada masalah.

Memegang strip umumnya berhasil dengan baik. Secara signifikan lebih baik daripada Audi terbaru, misalnya. Namun masih perlu ditandai di kedua sisi jalan dan tidak masuk tikungan terlalu tajam. Seperti radar, ada juga tangkapan - jika saya melepaskan tangan dari kemudi, saya harus menariknya sedikit setiap 10 detik untuk memperjelas bahwa saya masih hidup, meskipun ada pekerjaan para insinyur yang mengembangkan kontrol pelayaran untuk mobil ini.

"Autopilot" yang buruk adalah salah satu alasan utama mengapa saya berpikir untuk membeli Tesla - itu jauh lebih baik di dalamnya - sulit bagi saya untuk mempercayai mata saya ketika saya meluncurkan versi terbaru dalam Model 3.
