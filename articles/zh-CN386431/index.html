简化流程：英特尔的新平台和明显的节省

你好，GT！这个话题是话题性的，所以我什至不得不转移时间表中其余的出版物。不久前，出现了新一代英特尔处理器的发售（当然还有新的主板）：消费类产品增加了1151插槽。许多购买者对“主板+处理器+内存”捆绑包做出了非常奇怪的选择，这就是我们决定发布的原因这个迷你指南。



关于新的英特尔


新一代的Intel Core i3 / i5 / i7使用新的插槽：1151，取代了“老人” 1150（与1155和1156之前一样）。一方面，这些更改仅阻止处理器升级：您还必须更换主板。另一方面，自Intel Core i5 / i7-2XXX起，CPU性能可被描述为“足以满足当前大多数任务”，此类系统不升级CPU，不更换内存和视频卡，不安装SSD。较年轻的处理器（i3 / Celeron / Pentium）可能无法满足其所有者的需求，但仅在这一细分市场中，使用新处理器升级3-4-5年的系统可能不会挽救局面：如果您确实想要更多功能，则需要进行所有更改。

根据Tick-Tack的体系结构（这已经被反复编写，包括在GT上），目前这一代产品是Tack：“调试”了14 NM工艺技术和新架构。那么，随着Skylake架构的出现，新Core发生了什么变化？如果您没有巧妙的单词，并尝试用三个简单的句子来描述所有的变化，则会发生以下情况：

  • 市场的“消费者”部分获得了DDR4内存的访问权；
  • 电源线已更改：稳压器再次位于主板上，而不是CPU上；
  • CPU / GPU体系结构和芯片组已进行了各种改进。

只有三个半“实际”的芯片组：在“经济” B150系列，“通用” H170和“超频” - Z170。还有H110，但它更多地侧重于小型办公计算机，因此您只能考虑将其作为家庭购买的最后选择：通常，这些笔记本电脑携带廉价的笔记本电脑。如此狭窄的产品线使选择变得更加容易。

英特尔B150-廉价高效


宣布的芯片组规格如下：每个通道最多两个DIMM内存模块（即只有4个插槽），8个PCI Express 3.0通道（即1个x16插槽可以在最大带宽限制为64 Gb / s的情况下运行）多达12个USB -port（其中6个可以通过USB 3.0起作用），没有内置RAID控制器。

也就是说，对于大多数家庭任务（包括编辑照片/视频，连接周围的山脉和玩具），它非常适合。此外，英特尔为“即将淘汰的” DDR 3内存标准和新的DDR 4提供了支持，因此可以从使用DDR 3的旧系统进行升级的方式非常经济：将更多的钱花在SSD驱动器上的内存上会更好。

ASUS B150M-A D3 是升级旧系统的绝佳廉价选择。多达四个DDR3内存插槽（这意味着以前流行的2x4 GB可以补充2x4或2x4），处理器视频支持（以防您不玩），后面板上的6xUSB 3.0 + 2xUSB 2.0（通常）家用计算机所必需的。



与Core i3-6100结合使用（2核，4线程，3.7 GHz），您将获得出色的升级套件，可用于以后带有DDR3或任何AMD硬件的Core2duo / Quad系统。对于集成视频，与上一代Intel HD（可以在插槽1150上的Core i5或Core i7中找到）相比，内存带宽几乎增加了一半半，并且纹理（像素和纹理填充率）的处理速度提高了三倍。 。这并不是说英特尔HD 530一次，“灭顶”少年独立显卡，不过NVIDIA GT安装任何类型的噱头720 / 730，你可以放心地忘记：英特尔HD更快，游戏和多媒体应用。

游戏中FPS的明显增加将使GTX 750Ti达到某种水平（如果您换了新的视频卡，并且没有从旧的系统单元中拖出任何老式的怪兽），最低价格起价为8.5 + 1000卢布。

完全有可能在相同的Core i3-6100，更先进的MSI B150M BAZOOKA（mATX，4 DDR4，PCI-E，幸福还需要什么吗？），GTX 960和8 GB的基础上组装一款颇具战斗力的游戏车。如果您幸运的话，金士顿（Kingston）的处理器可以略微减少时间（但总的来说，爬升到2133/2400 MHz以上是没有意义的：只有以字母K结尾的处理器才有支持）。通过这样的配置和FullHD分辨率，大多数玩具都不会出现问题。

原则上，您可以在B150上贴一些i5-6500拥有诚实的四核，但是如果资金紧张的话，我会花6000卢布购买Core i3在SSD驱动器上。这种增加将更加明显，尤其是在“家庭”任务中。

英特尔H170-要求99％


H170是Z170的弟弟，没有超频潜力。它与B150的不同之处在于PCI-Express线的数量增加了一倍（一个视频卡可以获得一个全速工作的x16插槽，两个只有2x8），比B150支持2个USB端口（3.0），可以进行RAID并包含各种品牌的产品。英特尔ARK讨论的各种技术。

原则上，基于H170，您可以使用某些技嘉GA-H170-HD3 DDR3与旧的RAM（突然从前一台计算机上剩下4个模块，每个 4-8 GB）一起使用，也可以选择DDR4。作为处理器 -所有相同的i3 / i5或i7-6700没有字母K的处理器。对于您来说，结尾没有K的处理器并不有趣-它们只有借助Z170芯片组才能发挥其潜能。



在此配置仅受您的需要限制。您是否想在未来2-3年内升级旧的或已故的系统单元-DDR3选件+新的i5 + SSD（如有必要）和某些GTX 970或R9 390：FullHD玩具的最大设置，QuadHD（2560x1440）高。使用AMD R9 390-即使处于“高”（当然没有平滑）的4k2k，也可以在可接受的FPS下工作。

在DDR4下，MSI H170A PC MATE或超紧凑型技嘉GA-H170N-WIFI将运行良好，但在后一种情况下，不会超过16 GB的RAM不发光。:(
