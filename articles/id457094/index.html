Add-in Excel yang memudahkan untuk mengatur filter saat bekerja dengan kubus (VBA)

Seperti yang Anda ketahui, di luar kotak, Excel tidak memungkinkan Anda untuk menetapkan filter sesuai dengan daftar nilai untuk tabel pivot, dan ini adalah hal yang sangat diperlukan! Bagaimana cara memfilter produk dengan seratus kode, dan kemudian dengan seratus lainnya? Tentu saja ada cara, tetapi semua ini bukan itu ...

Atau, misalnya, atur filter untuk kubus dengan pengukuran dengan tanggal, tetapi jangan mencentang setiap hari / bulan / tahun, tetapi atur rentang dari ... ke ....
Cukup sering, pengaturan periode untuk pola tetap diperlukan, seperti hari / bulan / tahun saat ini . Untuk tujuan ini, Anda dapat menambahkan set (dalam model kubus multidimensi) atau menambahkan atribut khusus ke dimensi (ini sangat tidak nyaman, tetapi tidak ada set dalam model kubus tabular).

Kami melihat ekstensi VBA yang menambahkan Cube Toolbar dengan tombol ajaib ke pita.


Dalam aslinya, ada lebih banyak tombol pada panel, tetapi mereka dipotong, karena alasan yang jelas .



Filter berdasarkan tanggal dapat diterapkan pada beberapa tabel pivot secara bersamaan, baik pada satu tabel maupun pada lembar yang berbeda. Ini nyaman ketika Anda perlu mengatur periode yang sama dalam beberapa tabel sekaligus.



Koneksi tambahan


Lebih mudah untuk menempatkan add-in di dalam bola, dan karena alasan inilah gambar ditolak dari disalin ke direktori lokal sehingga perubahan dapat dilakukan dan semua orang memiliki versi saat ini. Tapi terserah kamu.


Add-in dipertajam untuk kalender (daftar tanggal) dari struktur tertentu, sehingga tidak mungkin akan lepas landas tanpa modifikasi, tetapi mereka berjanji untuk menjadi kecil. Tetapi memasang filter pada daftar harus bekerja tanpa masalah. Satu-satunya batasan adalah pemasangan filter untuk hierarki tidak didukung, tetapi ini tidak perlu.
