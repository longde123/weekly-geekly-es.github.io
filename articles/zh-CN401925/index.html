由Pi Zero W介绍



推出了新版本的Rapberry Pi Zero单板计算机-具有Wi-Fi和蓝牙功能的Pi Zero W

天线是电路板的一部分：



新的官方案例需付费。 顶盖有多种版本-带有GPIO电缆切口，中间有一个用于摄像头模块的孔，并且完全封闭。



一些规格：

  1. 尺寸：65mm x 30mm
  2. 重量：9克
  3. 处理器：带有VC4的BCM2835
  4. 默认CPU频率：1GHz，二级缓存超频
  5. 内存：512MB RAM
  6. 无线模块：用于WiFi / BT的BCM43438芯片
  7. WiFi：802.11 b / g / n
  8. 蓝牙：4.0（带流量控制的完整UART）
