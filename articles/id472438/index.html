Bagaimana proxy digunakan dalam keamanan informasi: 6 kasus penggunaan praktis



Gambar: Unsplash

Proxy biasanya digunakan sebagai "strip" antara klien dan situs web atau layanan online. Mengarahkan lalu lintas melalui proxy membantu pengguna menyembunyikan lokasi dan alamat IP mereka yang sebenarnya. Pada saat yang sama, perusahaan juga menggunakan proxy (lihat reverse proxy) untuk menyelesaikan banyak masalah, mulai dari meningkatkan keamanan informasi hingga melakukan tugas anonim dan mengendalikan penggunaan Internet oleh karyawan.

Hari ini kita akan berbicara tentang bagaimana teknologi proxy sebenarnya digunakan oleh berbagai perusahaan untuk meningkatkan tingkat keamanan infrastruktur mereka.

Menyembunyikan informasi dari cracker


Misalnya, seringkali proxy semacam itu digunakan sebagai gateway SSL. Dalam hal ini, perusahaan dapat menanggapi permintaan HTTP eksternal tanpa takut mengungkapkan struktur jaringannya. Bertindak sebagai mediator antara klien eksternal dan backend dari jaringan internal, proksi dapat menyembunyikan perangkat nyata dari infrastruktur perusahaan. Tanpa proxy, permintaan akan dikirim langsung ke server, dan mereka akan meresponsnya - ini dapat memberikan informasi kepada penyerang tentang perangkat jaringan dan kemungkinan titik lemah dalam perlindungannya.



Gambar: linuxbabe.com

Tipe lain dari proxy yang berguna untuk tugas keamanan adalah proxy penduduk. Bahkan, ini adalah alamat yang diberikan kepada pengguna oleh penyedia Internet nyata dari kota, wilayah atau negara tertentu. Biasanya, alamat ini diberikan kepada pemilik rumah, mereka dicatat dalam database register Internet regional (RIR). Ketika digunakan dengan benar, permintaan dari alamat tersebut tidak dapat dibedakan dari panggilan dari pengguna sebenarnya.

Proxy residen sering digunakan untuk intelijen kompetitif, mengorek data dari situs dan melewati sistem antibot. Pada gilirannya, perusahaan dapat menggunakan proxy residen yang diputar untuk meningkatkan keamanan jaringan atau situs perusahaan. Jika server dari berbagai negara dan wilayah menanggapi permintaan dari penyerang potensial, informasi tentang perangkat jaringan nyata tidak mungkin diperoleh.

Kami di Infatica menyediakan layanan proxy penduduk yang dirotasi - dan banyak pelanggan kami menggunakannya secara tepat untuk memastikan keamanan infrastruktur mereka.

Enkripsi dan dekripsi lalu lintas


Paling sering, perusahaan menggunakan proksi terbalik - biasanya ini adalah server proxy di balik firewall jaringan perusahaan. Ini mengarahkan permintaan klien eksternal ke server dalam jaringan. Reverse proxy juga dapat digunakan untuk menyelesaikan tugas-tugas seperti enkripsi lalu lintas yang lewat.

Selain itu, proksi berguna untuk bekerja dengan lalu lintas yang sudah dienkripsi. Menggunakan proksi terbalik untuk koneksi HTTPS yang masuk memungkinkan Anda mendekripsi konten setelah diterima dan mengirimkan versi dekripsi dalam jaringan.

Ini memiliki sejumlah keunggulan. Misalnya, Anda tidak perlu menginstal sertifikat keamanan di server backend. Selain itu, Anda hanya akan memiliki satu titik di mana Anda perlu mengkonfigurasi SSL / TLS. Selain itu, karena server dalam jaringan tidak perlu melakukan operasi sumber dekripsi konten yang intensif sumber daya, bebannya berkurang secara signifikan.

Memantau dan mengoptimalkan biaya perlindungan


Penggunaan reverse proxy juga membantu meningkatkan tingkat keamanan, memberi perusahaan titik di mana semua lalu lintas mengalir - dan di sini paling mudah untuk memantau dan mencatat.

Selain itu, jika semua server di balik firewall pertama-tama mengirim lalu lintas ke server proxy pada batas, maka dengan melindunginya, tingkat keamanan keseluruhan akan meningkat (jika tidak, perlu mengalokasikan sumber daya untuk melindungi semua server).

Kontrol akses


Proxy tidak hanya berguna untuk menyeimbangkan dan mendistribusikan lalu lintas dan memuat di jaringan, tetapi mereka juga dapat digunakan untuk membatasi akses ke sumber daya tertentu di jaringan. Server proxy dapat bertindak sebagai gateway untuk akses ke segmen jaringan individual, server atau layanan, baik untuk pengguna eksternal dan untuk karyawan perusahaan.

Kesimpulan


Perusahaan secara aktif menggunakan berbagai jenis proxy untuk memecahkan masalah yang terkait dengan peningkatan tingkat keamanan infrastruktur mereka. Proxy dapat digunakan sebagai alat untuk kontrol akses, logging, enkripsi lalu lintas dan bahkan menyembunyikan perangkat nyata dari infrastruktur internal.
