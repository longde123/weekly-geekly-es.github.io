新闻与发布


  • PHP 7.3 RC 1-分支7.3的发布候选循环已经开始。 预计总共将至少稳定6个问题。 在此期间，您可以阅读 PHP 7.3的新功能 。
  • PHP 7.2.10
  • PHP 7.1.22
  • PHP 7.0.32
  • PHP 5.6.38
  • HHVM将不再支持 PHP-计划于12月发布的HHVM v3.30将是支持PHP的最新版本。 建议将项目完全转换为Hack或PHP7。预期的消息在一年前就已被警告过，但是流行的框架和项目甚至更早地拒绝支持HHVM。 感谢HHVM团队推动PHP开发，祝您好运！
  • 夏季PHP-FIG新闻 -PSR-17 HTTP工厂 ， PSR-18 HTTP客户端规范正在审核，采纳，发布并积极讨论了PSR-14 Event Dispatcher草案。
  • PhpStorm 2018.3 EAP-推出了PhpStorm抢先体验计划，因此可以免费使用IDE的新版本。 在已经实施的创新中：
    •支持PHP-CS-Fixer ，不仅可以突出显示，而且可以纠正标准错误
    •调试期间快速搜索变量
    •Ctrl +点自动补全
    •重新设计的类/文件搜索对话框
    •DQL支持
    和其他改进。
