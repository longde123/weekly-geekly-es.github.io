Robot menggantikan apoteker di apotek



Teknologi Obat MTS Amerika siap untuk mempresentasikan perkembangan baru mereka - robot Omnicell M5000. Tugasnya adalah menyiapkan set obat secara individual untuk setiap pasien. Robot mendistribusikannya ke "kantong" khusus dengan perhitungan selama beberapa hari minum obat.



Robot medis mengenali obat dengan parameter seperti barcode, ukuran tablet, dan juga memperhitungkan warnanya.



Dalam satu jam, robot medis berhasil mengumpulkan kit untuk 50 pasien. Kecepatan ini secara signifikan akan meningkatkan pekerjaan apotek dan memfasilitasi pekerjaan dokter .
