Annonce des compétences Windows Vision (préversion)

Il y a quelques jours, nous avons annoncé l'aperçu de Windows Vision Skills , un ensemble de packages NuGet qui permettent aux développeurs d'applications de résoudre facilement les problèmes complexes de vision par ordinateur à l'aide d'un simple ensemble d'API.


Figure 1- De gauche à droite, vous voyez en action les compétences Détecteur d'objets, Détecteur squelette et Reconnaissance d'émotion.


Avec Windows Vision Skills:

  • Un développeur d'applications peut utiliser des API WinRT prêtes à l'emploi pour ajouter des compétences de vision prédéfinies telles que la détection d'objets, la détection de squelette et l'analyse des sentiments faciaux dans leurs applications Windows (.NET, Win32 et UWP).
  • Un développeur de vision par ordinateur peut tirer parti des infrastructures d'accélération matérielle sur les appareils Windows en regroupant leur solution en tant que compétence de vision - sans se soucier des API de bas niveau.


Toutes les compétences Windows Vision héritent des classes de base et des interfaces dans Microsoft.AI.Skills.SkillInterfacePreview . Ce cadre ouvert peut facilement être étendu pour fonctionner avec les cadres et bibliothèques d'apprentissage automatique existants tels que OpenCV.


Windows Vision Skills complète la prise en charge de Windows pour l'inférence des modèles ONNX en utilisant WinML pour l'inférence locale. Le cadre vous permet de créer des applications intelligentes tout en tirant parti de l'optimisation de la plateforme.
