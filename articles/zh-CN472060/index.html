德国警察冲进了一个军事掩体，掩体容纳了新宣布的数据中心


料斗的方案。 图：德国警察

Cyber​​Bunker.com是匿名托管的先驱，该公司于1998年开始运营。 该公司将服务器放置在最不寻常的地方之一：前北约地下综合大楼内，该地下综合大楼建于1955年，以防发生核战争时用作掩体。

客户排队：尽管价格高昂，但所有服务器通常都很忙：VPS的价格从每月100欧元到200欧元（不包括安装费），而VPS计划不支持Windows。 但是，托管人成功地忽略了来自美国的任何DMCA投诉，接受了比特币，并且除了电子邮件地址之外，不需要其他任何个人信息。

但是现在“匿名的非法行为”已经结束。 2019年9月26日晚上，德国特种部队和警察的战士冲进了受保护和守卫的掩体 。 缉获是在打击儿童色情制品的借口下进行的。

攻击并不容易，因为地堡位于森林中人迹罕至的地方，而数据中心本身位于地下的几层。
约有650人参加了这次行动，其中包括执法人员，救援服务，消防员，医务人员，无人机操作员等。


在照片左上方的三座建筑物旁边，可以看到掩体的入口。 中心是一座通讯塔。 右边是数据中心的第二座大楼。 从警察无人机拍摄的照片

捕获的物体位于德国西南部的特拉本-特拉巴赫镇附近（美因茨首府莱茵兰-普法尔茨州）。 掩体的四个地下层深达25米。



检察官尤尔根·鲍尔（Juergen Bauer）告诉记者，匿名托管已被调查了好几年。 手术是经过精心准备的。 暴风雨同时，有7人被拘留在Traben-Trarbach餐厅和法兰克福附近的Schwalbach市。 主要嫌疑人是59岁的荷兰人。 他被拘留，他的三个同胞（49岁，33岁和24岁），一个德国人（23岁），一个保加利亚人和唯一的女性（德国，52岁）。

在波兰，荷兰和卢森堡也进行了搜索。 总共没收了大约200台服务器，纸质文件，大量存储介质，移动电话和大量现金（约合4100万美元）。 调查人员说，对证据的分析将需要数年时间。


料斗中操作员的工作场所

在突袭期间，德国当局还没收了至少两个域，包括荷兰ZYZTM Research公司（zyztm [。] Com）和cb3rob [。] Org的域。

根据当局的说法，上述荷兰人于2013年购买了一个前军事掩体-并将其变成了一个大型且受到良好保护的数据中心，“据我们的调查，目的是将其仅用于非法目的提供给客户，”鲍尔补充道。

在德国，除非证明托管人知道并支持非法活动，否则不应对托管人负责。

前北约设施是从联邦议院地理信息部门购买的。 当时的新闻稿称其为多层防御性建筑，面积为5500平方米。 它有两栋相邻的办公楼，面积为4300平方米，总建筑面积占地13公顷。



区域刑警负责人约翰尼斯·昆兹（Johannes Kunz）补充说，嫌疑人“涉嫌有组织犯罪”，尽管他已申请移居新加坡，但大部分时间都在该地区度过。 数据中心的所有者没有迁移，而是居住在地下掩体中。

Brower说，总共有13人在20至59岁之间接受调查，其中包括3名德国公民和7名荷兰公民。

有7人被拘留，因为他们有可能逃离该国。 他们涉嫌参与犯罪组织，违反税收规定，并涉嫌参与与毒品，洗钱和伪造文件有关的“成千上万的犯罪”，并有助于传播儿童色情制品。 当局没有给出任何名字。

调查人员将数据中心描述为“防弹主机”，旨在隐藏当局的视线。

昆茨说：“我认为这是巨大的成功……我们能够将警察部队带入掩体区，该掩体受到最高军事水平的保护。” “我们不仅要克服真实或模拟保护，还必须突破数据中心的数字保护。”


数据中心中的服务器

据称在德国数据中心内提供的非法服务包括大麻路，Flight Vamp 2.0，Orange Chemicals和华尔街市场，这是世界第二大毒品分销平台。

例如，在大麻路工地，有87个非法毒品贩子被登记。 总体而言，该平台至少处理了数千种大麻产品的销售。

华尔街市场平台处理了约25万笔毒品交易，销售额超过4100万欧元。

Flight Vamp被认为是瑞典最大的毒品贩运平台。 寻找其运营商的工作由瑞典调查机构进行。 根据调查，那里有600名卖家和大约10,000名买家。

通过奥兰治化学公司，各种类型的合成药物分布在整个欧洲。

可能现在所有列出的商店都将不得不移至Darknet上的另一个主机。

根据鲍尔（Bauer）的说法，2016年年底，德国电信公司Deutsche Telekom的僵尸网络攻击也从赛博朋克服务器中进行了，该攻击禁用了大约100万个客户路由器。

在2013年购买掩体时，买方没有立即证明自己的身份，而是说他与Cyber​​Bunker相关联，后者是冷战期间位于另一掩体中的类似荷兰数据中心的运营商。 这是世界上最古老的匿名主机之一。 他宣布所谓的“赛博邦克共和国”独立，并愿意发布除儿童色情制品和与恐怖主义有关的一切以外的任何网站。 该站点当前不可用。 在主页上，有执法机构引以为傲的题词：“没收了服务器”（DIESE SERVER WURDE BESCHLAGNAHMT）。



根据whois的历史记录 ，Zyztm [。] Com最初以荷兰的Herman Johan Xennt的名义注册。 Cb3rob [。] Org域由Cyber​​Bunker托管的组织所有，并向自封为无政府主义者的Sven Olaf Campheis注册。几年前，他因参与上述大规模攻击而被定罪，该大规模攻击曾在某些地方短暂破坏了互联网。

文学中的匿名托管



Sven Olaf Campheis的前Facebook个人资料照片。 在2013年被捕后，他与当局进行了粗略的交谈，并宣布了赛博邦克共和国的独立

赛博邦克共和国和其他离岸托管服务的故事在某种程度上让人联想到尼尔·史蒂文森（Neil Stevenson）的小说《 密码学》（Cryptonomicon）中虚构的Kinakut状态。 这部小说以“另类历史”的形式写作，显示了人类的发展在输入参数的微小变化或偶然的结果下会朝哪个方向发展。

Kinakuta苏丹国是苏禄海角的一个小岛，位于加里曼丹岛和菲律宾巴拉望岛之间的海峡中间。 第二次世界大战期间，日本人使用Kinakuta作为攻击荷兰东印度群岛和菲律宾的桥头堡。 有一个海军基地和一个飞机场。 战争结束后，得益于石油储备，金角田重新获得了包括财政在内的独立性。

由于某种原因，基纳库蒂苏丹决定在自己的州之外建立一个“信息天堂”。 一项通过的法律适用于经过Kinakuta领土的所有电信：“统治者宣布，我放弃了对该国内部及其边界的信息流的所有行政管理权”。 -在任何情况下，政府都不会对信息流flows之以鼻，也不会利用其权力限制这些信息流。 这是Kinakuta的新法律。” 之后，在Kinakuta领土上创建了地穴的虚拟状态：

地穴。 网络的“实际”资本。 黑客天堂。 公司和银行的噩梦。 世界各国政府的“敌人第一”。 网络上没有任何国家或国籍。 只有免费的人愿意为他们的自由而战！
