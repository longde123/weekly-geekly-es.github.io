对Windows 7的免费支持期仅剩一年



多少年前，微软宣布推出对其Windows 7操作系统的五年扩展支持，期限将在2020年1月14日结束，也就是一年之后。

但是Windows 7甚至不会失去支持。 它将是，只有Microsoft将继续为此付费以为其操作系统提供服务。 许多在工作计算机，便携式计算机和专用设备上使用此特定操作系统的公司将付费继续接收更新。

所谓的扩展支持更新（ESU）也受时间限制。 期限-三年，并且每年Microsoft服务的价格都会越来越高。

那些是Windows虚拟桌面（WVD）成员的用户将免费获得支持。 该服务提供了运行Windows 7的“在云中”运行的虚拟机以及任何必要的应用程序。 这些虚拟机将免费获得更新，直到扩展支持期结束。

Windows Server 2008和Windows Server 2008 R2也会出现类似的情况-一年后，对这些操作系统的支持也已得出其逻辑结论。

顺便说一下，几天前，Windows 7的新更新“破坏了”该操作系统。 更新并没有增加“七个”对计算机的保护级别，而是带来了新的问题。 例如，作为本地管理员组成员的本地用户在运行Windows Server 2008 R2和Windows 7的计算机上的远程访问有问题。

现在，问题已部分解决。 Microsoft已发布更新KB4487345，它消除了一些错误。 同时，该软件包不可用于自动更新，要安装它，您将必须手动下载补丁。

上周初，开发人员发布了Windows 7 SP1和Windows Server 2008 R2 SP1的修补程序KB4480960和KB4480970。 正是它们导致了操作系统中的大量故障。 系统用户遇到各种错误，包括KMS激活错误，非正版和0xc004f200。

事实证明 ，以前安装的补丁KB971033开始与KB4480960和KB4480970发生冲突。 虽然到目前为止，尚未发现此修补程序没有计划安装在密钥管理服务（KMS）环境中。 该公司提议解决该问题，从PC上删除KB971033，清除KMS缓存并删除操作系统的激活数据。 之后，必须通过KMS重新激活系统。 此外，您完全不能使用KB4480960和KB4480970更新。

值得注意的是，在新的一年中，Windows 7多年来已首次失去在最受欢迎的桌面OS列表中的领导者地位。 “七”被“十”所取代，后者位居榜首。 同时，Microsoft相信更换领导者的时间会更早。 因此，在Windows 10发布后的头几个月中，其用户数量增长很快。

当时，该公司假设在短短几年内，将有超过十亿台设备可在Windows 10上运行。 但是随着时间的流逝，将用户切换到Windows 10的过程进展缓慢。 到 2017年5月，该操作系统才获得 5亿用户。 这不仅考虑了笔记本电脑和台式机，还考虑了Xbox以及Hololens增强现实眼镜。

Windows 10已于2018年9月运行在7亿台设备上。


根据VentureBeat的数据，2018年12月台式机操作系统的市场结构

2018年12月，Windows 10约占台式机操作系统市场的39.22％。 Windows 7占36.9％。 在2015年夏季，Windows 7已安装在60％以上的计算机上。 好吧，G8占据了操作系统市场的5.33％。

反过来，macOS已安装在大约10％的设备上。 桌面版Linux占据了2.78％的市场份额。 根据Venture Beat的说法，大约86.2％的市场是由Microsoft操作系统控制的。
