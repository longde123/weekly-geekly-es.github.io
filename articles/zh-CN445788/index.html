我们最近发布了改进的Web SDK，配备了新的文档系统和演示应用程序，这将使我们的平台对开发人员来说更加灵活和方便。 如果您以前已经熟悉我们的SDK，您将立即注意到这些变化-现在您有了一个清晰的示例，了解如何在应用程序中嵌入API函数。

Web SDK：新功能


Ivideon不仅仅是基于云的视频监控服务和设备供应商。 在Ivideon内部，一个完整的开发周期正在进行中：从相机固件到服务的Web版本。 我们制作客户端和服务器SDK，改善LibVLC，实施WebRTC，进行视频分析，为合作伙伴开发具有White Label支持的客户端，并为SDK演示项目。

结果，我们设法成为一个平台，在此基础上，合作伙伴可以创建自己的解决方案。 现在，我们的Web SDK进行了重大升级，我们希望会有更多的集成解决方案。

为了方便起见，我们在开头添加了“快速入门”部分，这将帮助您轻松地了解设备管理。

下面的代码演示了Ivideon Web SDK的基本用法：已在页面上添加了一个播放器，并且已开始为公共摄像机播放视频。

我们还添加了一些新功能：

  • 支持一次性视频链接；
  • 按钮已添加到播放器中，以控制视频质量和存档播放速度；
  • 播放器控件可以一次打开和关闭（以前，您可以打开或隐藏所有内容）；
  • 添加了使相机声音静音的功能。
