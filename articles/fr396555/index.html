Comment compresser Windows 10
En juillet 2016, Microsoft a publié pour Windows 10 un nouveau mécanisme de compression Compact OS pour un utilitaire de ligne de commande compact.exequi compresse les fichiers du système d'exploitation directement pendant son fonctionnement. Le nouveau mécanisme de compression possède de nombreuses autres fonctionnalités intéressantes, mais en général, il est similaire au programme WIMBoot dans Windows 8.1 Update 1.

Compact OS libère au moins 2 Go d' espace disque. Pas tant que ça, mais sur les petits SSD, cela se remarque.

Le programme s'exécute à partir de la ligne de commande avec des privilèges d'administrateur. Cela signifie qu'après avoir exécuté cmd.exe, vous devez appuyer sur la combinaison de touches Ctrl + Maj + Entrée.

Avant de compresser Windows 10, il est logique de savoir si cette procédure produira un résultat différent de zéro. Le système peut déjà être dans un état compressé, donc aucune procédure supplémentaire n'est requise. Pour déterminer l'état actuel des binaires du système d'exploitation, exécutez la commande suivante:

  • compact.exe / q



La commande affichera des informations sur l'état actuel: combien de fichiers sont compressés ou non compressés, le niveau de compression, combien d'octets se trouvent dans le conteneur de fichiers compressés. Si le niveau de compression est proche de 1,0, l'utilisation du programme sera efficace.

Dans certaines conditions, un système compressé peut légèrement diminuer les performances, mais sur la plupart des ordinateurs, il sera invisible. Gardez à l'esprit qu'il est toujours possible de ramener les fichiers à leur état normal si vous constatez des problèmes de performances. Dans tous les cas, il est recommandé d'effectuer une sauvegarde avant de compresser les fichiers système.

Pour exécuter la procédure, exécutez la commande suivante:

  • compact.exe / CompactOS: toujours



La compression de tous les fichiers devrait prendre plusieurs minutes, bien que sur certains systèmes, cela puisse prendre 20 minutes ou plus. La dernière version Insider Build libère plus de 2 Go.

Pour ramener le système à son état d'origine:

  • compact.exe / CompactOS: jamais

La procédure inverse est un peu plus rapide.

Un programme pratique et gratuit qui convertit les fichiers audio et vidéo en différents formats.
Plusieurs programmes répondent aux exigences de base:

  • open source
  • utilisation gratuite;
  • prise en charge de nombreux formats de fichiers;
  • multiplateforme;
  • légèreté.

Les plus pratiques sont peut-être deux: Handbrake et ffmpeg (par exemple, avec l'interface graphique WinFF ). Par exemple, le populaire lecteur vidéo VLC gratuit peut convertir des fichiers, mais dans la plupart des cas, il est plus pratique d'utiliser un programme spécialisé.

Le frein à main fonctionne sur Windows, Mac et Linux. La dernière version pour Windows en ce moment est 0.10.5. Presque tout fichier multimédia (ou dossier contenant des fichiers), DVD ou BlueRay, s'il n'y a pas de protection contre la copie, est

accepté comme matériel source . Formats de fichiers cibles


  • Conteneurs : .MP4 (.M4V) et .MKV
  • Encodeurs vidéo : H.265 (x265 et QuickSync), H.264 (x264 et QuickSync), H.265 MPEG-4 et MPEG-2, VP8, Theora
  • Encodeurs audio : AAC / HE-AAC, MP3, Flac, AC3, Vorbis
  • Pass-through audio : AC-3, E-AC3, DTS, DTS-HD, TrueHD, AAC, pistes MP3



Le programme est pratique à utiliser pour transcoder un DVD ou un fichier téléchargé pour le visionner sur un appareil autre qu'un PC. Handbrake a des profils d'encodage intégrés pour les appareils suivants:

  • iPod
  • iPhone et iPod touch
  • iPad
  • AppleTV
  • AppleTV 2
  • AppleTV 3
  • Android
  • Tablette Android
  • Windows Phone 8

Le programme possède de nombreuses fonctionnalités supplémentaires.

  • Sélection de fragments (parties du fichier source pour l'encodage)
  • Traitement par lots et file d'attente de demandes
  • Marqueurs de chapitre
  • Sous-titres (VobSub, sous-titres codés CEA-608, SSA, SRT)
  • Débit constant ou qualité constante (débit variable)
  • Prise en charge VFR, CFR et VFR
  • Filtres vidéo pour le transcodage: désentrelacement, décombillage, débruitage, détélécine, déblocage, niveaux de gris, recadrage, mise à l'échelle
  • Aperçu en direct

La plupart du code source de Handbrake est distribué sous la GNU GPL gratuite, version 2. Les parties individuelles sont sous la clause BSD 3.

Comment choisir un VPN gratuit
Pour un travail anonyme sur Internet et pour contourner la censure de l'État (blocage des sites), il est pratique d'utiliser un VPN gratuit. L'une des options les plus populaires est FreeOpenVPN (via l'application cliente OpenVPN).

Soit dit en passant, le 31 juillet 2016, Kaspersky Internet Security 2017 (un antivirus populaire en Russie) a été lancé, qui comprenait un client VPN. En mode gratuit, il ne fonctionne que via des serveurs en Russie et vous permet de transférer 200 Mo par jour. Bien sûr, c'est une offre plutôt médiocre, mais si vous avez déjà installé KIS 2017, cette option doit être gardée à l'esprit. Seuls les utilisateurs de KIS ne doivent pas activer la fonction VPN payante dans le programme, car il existe de nombreuses alternatives gratuites.
