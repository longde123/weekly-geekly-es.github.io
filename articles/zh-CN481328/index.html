BlitzTeam关于塞浦路斯集团Wargaming的个人索赔，金额为169万美元

Wargaming依法起诉其前员工从事开源项目（开源软件）的工作，该项目正式开发至2018年春季。 Wargaming向目前在BlitzTim LLC工作的前雇员提起了个人诉讼，要求每人最高赔偿50万美元。

Wargaming在许多会议，文章，访谈和其他来源中已正式宣布支持该项目作为开源。

在引起轰动的Nginx和Rambler案之后，很明显白俄罗斯开放源代码的历史并不比俄国更糟。

开源及其在白俄罗斯的应用可能性是一个基本问题，是建设IT国家和健康的现代化IT社区的基础。

我们打算保护BlitzTim LLC员工的利益，并且要在定义和合法化我们的开源原则方面发挥积极作用。

白俄罗斯几乎每个大中型公司都使用甚至开发开源产品。 对于我们的IT国家和所有HTP居民而言，确定原则和法律非常重要，这样白俄罗斯开放源代码才能按照全球IT社区的原则开展工作。

2019年12月9日，由Blitz Team LLC开发和发布了移动射击游戏Battle Prime的全球发布。 这是工作室的第一个项目。

该游戏是在我们自己的私人游戏引擎“闪电战引擎”上开发的，该引擎是从零开始创建的。 在我们的博客上阅读更多内容。

2019年12月17日，白俄罗斯共和国最高法院知识产权事务司法委员会通知了我们的五（5）名员工有关由塞浦路斯法律公司Wargaming World Limited提起的民事诉讼的案件，该诉讼由白俄罗斯法律办公室“ Stepanovsky，Papakul＆Partners” 提交 （ spplaw.by ）。

BlitzTim LLC的上述员工先前曾在塞浦路斯Wargaming公司集团（SOOO Game Stream aka Game Stream）的明斯克开发中心工作，并直接参与了《战车世界：闪电战》和DAVA Framework / DAVA Engine的开发。

Wargaming要求其五名前雇员赔偿3 570 000白俄罗斯卢布（约合169万美元 ）。

我们不知道诉讼是否仅由现在在BlitzTim LLC工作的Game Stream前雇员，还是其他叉子的开发人员和所有者（包括Game Stream JLLC的现有雇员）提起。 此外，我们还不知道Wargaming计划从该公司的公共GitHub存储库中撤出其他开源库的计划。

目前，除其他事项外，我们的人员负责：



  • 在Game Stream上工作时，在GitHub上放置DAVA Framework源代码的伪造副本；
  • 从这些副本到GitHub上的个人帐户创建派生。

Wargaming在他的诉讼中进一步声明：


Wargaming于2014年7月22日获得了DAVA Framework / DAVA Engine的专有权。

让我们看一下开放源代码中DAVA框架/ DAVA引擎的历史：



  • 该项目最初由DAVA Consulting，LLC于2011年添加到Open Source，可从github.com/dava/dava.framework获得，后来由Wargaming开发人员将其重命名为github.com/dava/dava.engine 。
  • 在2018年，Wargaming从GitHub的公开部分删除了主要存储库github.com/dava/dava.engine ，这导致GitHub系统自动选择了当前根fork github.com/dheerendra1/dava.framework 。
  • 该前叉（dheerendra1）表明，至少从2011年起，该发动机就在Wargaming收购前很久就获得了BSD3的许可。 通过从存储库的当前公共根目录打开任何源来验证这一点很容易。 例如， github.com/dheerendra1/dava.framework/blob/master/Sources/Internal/DAVAConfig.h文件创建于2011年6月29日，其中包含BSD3许可证。
  • 在较新的fork中，您可以查看2011年至2018年之间提交的全部历史记录，例如github.com/smile4u/dava.engine/blob/development_history
  • 2016年5月17日，用于存储许可证信息的格式更改为一种更简单的格式：先前在每个存储库文件中找到的许可证信息被转移到位于存储库根目录中的单独的LICENSE.TXT文件中，受到GitHub的欢迎：( github.com/smile4u/ dava.engine / blob / development_history / LICENSE
