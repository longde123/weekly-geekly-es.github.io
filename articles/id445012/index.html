Kalkulator Windows akan mendapatkan mode grafis



Belum lama berselang, berita itu dipublikasikan di HabrÃ© tentang pengungkapan kode Kalkulator Windows , salah satu program paling terkenal di dunia. Kode sumber untuk perangkat lunak ini tersedia di GitHub .

Kemudian dikatakan bahwa pengembang program menawarkan semua orang untuk menyumbangkan keinginan dan ide mereka mengenai fungsionalitas program. Dari sejumlah besar, satu saat ini dipilih. Penulisnya menyarankan menambahkan mode grafis ke kalkulator .

Sebenarnya, semuanya jelas di sini - mode grafis akan memungkinkan untuk memvisualisasikan persamaan dan fungsi, kira-kira sama dengan Mode Plotting di Matlab. Fitur ini diusulkan oleh insinyur Microsoft Dave Grochocki. Menurutnya, mode grafis tidak akan terlalu canggih. Ini akan memungkinkan siswa untuk merencanakan grafik menggunakan persamaan aljabar.

â€œAljabar adalah jalan menuju ranah matematika yang lebih tinggi dan disiplin terkait. Namun demikian, ini adalah salah satu mata pelajaran yang paling sulit bagi siswa, banyak yang mendapat nilai buruk dalam aljabar, â€kata Grochoski. Pengembang percaya bahwa jika Anda menambahkan mode grafis ke kalkulator, akan lebih mudah bagi siswa dan guru untuk saling memahami di kelas.

â€œGrafik kalkulator bisa sangat mahal, solusi perangkat lunak memerlukan lisensi, layanan online tidak selalu merupakan solusi terbaik,â€ lanjut Grochoski.

Menurut perwakilan Microsoft, ini adalah mode grafis yang merupakan salah satu fitur yang paling sering diminta dalam aplikasi Feedback Hub, di mana pengguna produk perangkat lunak perusahaan mereset penawaran mereka.

Tujuan yang ditetapkan pengembang:
  • Menyediakan visualisasi dasar dalam Windows Calculator;
  • Dukungan untuk kurikulum matematika dasar di AS (sayangnya, sementara fungsi Kalkulator akan direncanakan berdasarkan kebutuhan siswa di negara ini), termasuk kemampuan untuk membangun dan menafsirkan fungsi, memahami model linear, kuadratik dan eksponensial, mempelajari fungsi trigonometri dengan kalkulator dan memahami konsep persamaan.

    Apa lagi yang akan didapat pengguna:
    • Kemampuan untuk memasukkan persamaan untuk membangun grafik yang sesuai.
    • Kemampuan untuk menambahkan beberapa persamaan dan memvisualisasikannya untuk membandingkan grafik satu sama lain.
    • Mode pengeditan persamaan sehingga Anda dapat melihat perubahan apa saat modifikasi tertentu dibuat ke persamaan asli.
    • Mengubah mode tampilan bagan - bagian yang berbeda dapat dilihat dengan berbagai tingkat detail (mis. Kita berbicara tentang penskalaan).
    • Kemampuan untuk mempelajari berbagai jenis grafik.
    • Kemampuan untuk mengekspor hasilnya - sekarang Anda dapat berbagi visualisasi fungsi di Office / Tim.
    • Pengguna dapat dengan mudah memanipulasi variabel sekunder dalam persamaan, yang akan memberikan kesempatan untuk memahami bagaimana perubahan dalam persamaan mempengaruhi grafik.

    Sejauh yang dapat dinilai, grafis dapat dibangun untuk fungsi yang tidak terlalu rumit.

    Sekarang pengembang Kalkulator sedang mencoba untuk menunjukkan bahwa program ini membaik seiring waktu. Dia dilahirkan sebagai asisten dasar untuk melakukan operasi aritmatika. Sekarang ini adalah kalkulator ilmiah andal yang dapat digunakan oleh berbagai pengguna untuk memecahkan masalah yang sangat serius. Di masa depan, perangkat lunak akan semakin meningkat.

    Adapun pembukaan kode sumber, ini dilakukan agar siapa pun dapat berkenalan dengan teknologi Microsoft seperti Fluent, Universal Windows Platform, Azure Pipelines dan lainnya. Berkat proyek ini, pengembang dapat mempelajari lebih lanjut tentang bagaimana pekerjaan dilakukan untuk membuat proyek tertentu di Microsoft. Analisis terperinci tentang kode sumber Kalkulator Windows dapat ditemukan di sini, tepat di Habr .

    Program ini ditulis dalam C ++ dan berisi lebih dari 35.000 baris kode. Untuk mengkompilasi proyek, pengguna memerlukan Windows 10 1803 (atau lebih baru) dan versi terbaru dari Visual Studio. Semua persyaratan tersedia di GitHub.
