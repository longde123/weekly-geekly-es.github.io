TimeWeb. Perhatian Malware

Selamat siang, kawan-kawan tercinta! Sebagai pengembang situs web dan terus-menerus menyebarkan server baru, pada 28 Juni, saya mengalami masalah. Kami menyebarkan server baru, memeriksa IP-nya (yaitu, Anda mengetikkan server ip di browser dari server, misalnya, http://92.53.120.58/ atau http://188.225.32.192/ ) - hasilnya adalah "Awas, malware."



Saya menulis tiket ke dukungan teknis TimeWeb tentang masalahnya. Dia meminta untuk memeriksa situasi, sehubungan dengan yang muncul. Saya pikir mereka akan mengambil tindakan!



Dan apa yang mereka lakukan? Mungkin tidak ada! Hari ini saya mulai menyebarkan server baru:



Semua dari mereka mengeluarkan peringatan tentang malware, berikut adalah daftar IP yang digunakan hari ini:

  • 92.53.120.58
  • 188.225.32.192
  • 188.225.35.136
  • 188.225.36.21
  • 188.225.37.52

Dan saya menggunakan ini pada 28 Juni 2017.

  • 188.225.77.106
  • 188.225.78.155

Periksa, kolega, server dengan IP.
