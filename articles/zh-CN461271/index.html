如何在2019年推广移动应用程序：4种实用方法+有用的工具



移动应用市场的竞争正在迅速发展。 今天，已经很难指望成功了，已经掌握了ASO的方法，需要更深入地浸入其中，并且有必要制定非标准的广告和营销步骤。

今天，我们将讨论从项目的一开始到高级阶段，推广移动应用程序的方法。 此外，对于每个步骤，我都将提供指向有助于推广的网站和服务的链接。 走吧

方法＃1。 创建早期用户社区


当项目在开发的一开始就获得动力时，它总是非常有用的。 如果您可以在应用程序发布后立即引起注意，那么将来会更容易吸引观众。

这可以借助专门的资源来促进初创企业在全球市场上实现。 该细分市场中的顶级平台是Product Hunt网站。 这是一个对技术感兴趣并正在探索新项目的用户社区。

在此资源上成功发布产品将吸引成千上万的用户，其中大部分来自美国和英语国家。 所有这些意味着您不仅可以指望接受Beta测试人员，还可以指望付费客户。



以下是前往产品搜索的一些提示：

  • 不要吹牛，不要乞求喜欢-因为这样可以禁止他们 。 ProductHunt团队本身的发布指南中详细描述了发布项目的规则，请务必阅读。
  • 寻找猎人 。 这是具有大量订户的受欢迎的资源用户。 他们所有人都将收到一封电子邮件通知，说明用户已经发布了一个新项目。 因此，您获得了第一个用户。
  • 在PH上运行需要24小时工作 。 新的一天的倒计时从美国西海岸时间00:00开始。 您必须24小时在线，才能及时回复评论-线程中的活动越高，一天中的活动就越高。
  • 使用该网站进行长期促销 。 例如，在热门产品的“产品搜索”中，有一些页面可供选择-平台上很多喜欢的产品都可以到达该页面。 如果您选择这种方式，则会获得更多流量。

除产品搜寻外，还有其他用于启动公告的资源，包括：

  • 黑客新闻
  • Betalist
  • Betapage

在每个项目上放置有关该项目的信息，这样您就可以在项目初期就以最少的费用吸引第一批用户。

方法＃2。 吸引竞争对手的广告流量


该项目的公告是好的，但是为了长期成功，您将需要一个完善的营销策略。 根据竞争对手活动的数据来创建它更容易。 为了从该应用程序上赚钱，您需要吸引尽可能多的流量，即将进行应用内购买或观看广告的用户。 仅靠自然交通来解决这个问题将很困难，大多数情况下必须购买。

这并不容易，它可以监视竞争对手的有效举动，从而为应用程序配置广告，从而吸引用户。 如果某些步骤适用于一个应用程序，那么它们很有可能会帮助另一个应用程序。 因此，您可以节省测试假设的费用。

Apptica服务是用于分析移动应用程序中的应用程序内活动的最佳工具之一。 有了它的帮助，您可以跟踪：

  • 不同市场中的顶级广告商和广告应用程序；
  • 最有效的广告活动，甚至使用的广告类型（针对各种广告网络（包括Facebook和Instagram）的各种格式的大型广告画廊）；
  • 最受欢迎的广告网络，其中包含有关展示次数，流量来源等的所有统计信息。



所有这些都将有助于制定促进应用程序货币化的策略-显然，这两个因素都将直接影响用户数量。

方法＃3。 使用自定义动作


这似乎令人惊讶，但是近年来，QR码的受欢迎程度一直在稳步增长 。 对于用户而言，快速下载移动应用程序最简单的方法之一就是扫描带有链接的QR码。 因此，该技术被积极地用于促进应用。

有多种使用QR码增加应用程序安装数量的方法。 首先，您需要创建代码-QR Code Generator之类的工具适合于此。

然后，代码可以连同其描述一起用作应用程序的广告元素。 加上代码-它们可以在线和离线放置。 不同的公司在名片，横幅，商店中的广告上打印QR码，将其放置在每封电子邮件的签名中，并将其用作上下文广告的元素。



方法＃4。 应用程序本地化


吸引新用户的最明显方法之一是使应用程序以不同的语言提供。 Statista表示，在未来几年中，许多新兴市场中移动应用程序的下载量将快速增长。 这意味着将应用程序从英语（例如英语）本地化为葡萄牙语，西班牙语和俄语将为数百万新用户提供访问权限。

为了解决翻译界面的任务， Localize工具很棒。
