Electric Polestar 2 - das erste Elektroauto mit nativem Android Auto OS



Polestar 2 - Entwicklungsunternehmen Volvo (oder besser gesagt eine Tochtergesellschaft von Polestar). Dies ist ein Elektroauto, das nach Angaben der Entwickler eine Reichweite von 450 km hat. Sein Preis entspricht den Kosten für Tesla Model 3. Vertreter von Polestar sagen auch , dass die technischen Eigenschaften eines Elektroautos ungefähr den Eigenschaften der besten Elektroautos einer ähnlichen Klasse entsprechen.

Jetzt ist es das erste Elektrofahrzeug, das mit Android Auto OS arbeitet. Das Betriebssystemprojekt wird nicht nur von Google, sondern auch von einer Reihe von Partnern des Unternehmens implementiert. Zum ersten Mal über dieses Betriebssystem wurde es 2014 bekannt, dann wurde berichtet, dass das Automobil-Betriebssystem autonom ist, es erfordert nicht die Verwendung eines Smartphones. Android Auto wurde dann 2017 in Google I / O eingeführt.



Im Jahr 2018 demonstrierte das Unternehmen auf seiner Veranstaltung eine funktionierende Version des Betriebssystems auf dem Volvo XC40. So erschien die Google Assistant-Schaltfläche am Lenkrad eines Elektroautos, mit der Sie den Navigator anrufen, das Mikroklima im Auto steuern, Medieninhalte starten usw. können. I / O-Teilnehmer stellten fest, dass die Arbeit mit einem Auto-Betriebssystem im Allgemeinen der Verwendung eines Android-Tablets ähnelt. Es gibt Benachrichtigungen, ein Anwendungskontrollzentrum usw. Zwar wird YouTube entfernt, um den Fahrer nicht von der Straße abzulenken.

Zuvor war das Unternehmen bereits an der Entwicklung von Software für Elektrofahrzeuge beteiligt, Android Auto ist jedoch eine vollautomatische Plattform.

Was das Elektroauto betrifft, so entwickeln seine Elektromotoren eine Leistung von 400 PS. Dies ist ein fünftüriges Auto. Polestar hatte zuvor angekündigt, dass das Auto etwas später in diesem Jahr in Produktion gehen wird. Das erste Elektroauto des Unternehmens, Polestar 1, ist bereits in Produktion gegangen, es werden jedoch nur 500 Exemplare veröffentlicht. Der Preis für das erste Modell beträgt 150.000 US-Dollar, obwohl es sich um einen Hybrid handelt, nicht um ein reines Elektroauto.
