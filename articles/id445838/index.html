Robotika untuk anak-anak: mata robot


Halo orang-orang! Saya hadir untuk Anda prototipe mata kucing yang dikembangkan oleh saya. Tentu saja, proyek ini belum ideal (sedang difinalisasi), tetapi berhasil.

Idenya adalah untuk membuat robot kucing untuk anak-anak dari komponen kayu lapis dan radio. Proyek dari Internet diambil sebagai model dasar robot .



Berdasarkan video yang saya tonton bersama robot kucing, saya mulai menggambar detail robot masa depan saya di corel. Internet aslinya berukuran kecil, dan pergerakannya didorong oleh SG90 servos. Dalam perkembangan saya, saya menggunakan servos DSS-M15S. karena ukuran model kucing saya secara signifikan lebih besar. Tapi sekarang saya ingin memberi tahu bukan tentang model robot, tetapi tentang matanya. Saya mengembangkannya sendiri, dari awal, terutama untuk proyek ini.


Dalam perkembangan saya, mata terbuka sesuai dengan prinsip diafragma, dan mereka terbuat dari kayu lapis.

Dan agar energi kehidupan bersinar di mata robot, saya menggunakan LED merah ultra-terang, mereka tetap bersamaku dari proyek aquaponik sebelumnya.

Mekanisme yang memungkinkan Anda untuk membuka dan menutup diafragma mata, saya terapkan pada SG90 servo. Semua yang saya miliki, Anda dapat menonton video.
