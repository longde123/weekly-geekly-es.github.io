Mobiles Terminal in Zugangskontrollsystemen

Die Entwicklung mobiler Technologien hat dazu geführt, dass in Zutrittskontrollsystemen ein Smartphone mit einem NFC-Modul zunehmend nicht nur als Kennung, sondern auch als Aufzeichnungsgerät eingesetzt wird.


Anwendungsgebiet


Eine solche Lösung eignet sich für Einrichtungen, in denen es unmöglich oder unrentabel ist, ein stationäres Registrierungsterminal einzurichten, die Zugangskontrolle und die Abrechnung der Mitarbeiter jedoch erforderlich sind. Dies können Minen, Ölplattformen, Baustellen, Geschäftsbusse und andere abgelegene Standorte sein, auch ohne Internetverbindung.


Arbeitsprinzip


Unter den russischen Herstellern wurde eine solche Lösung wie ein mobiles Zugangsterminal bereits von den führenden Herstellern von ACS vorgestellt: PERCo, Sigur, Parsec. Wir werden das Prinzip des mobilen Endgeräts am Beispiel einer Lösung von PERCo analysieren.


Als mobiles Endgerät wird ein Smartphone mit einem NFC-Modul und einer installierten mobilen Anwendung verwendet. Mit der Anwendung können Sie Mitarbeiter- und Besucherausweise auf Zutrittskarten im MIFARE-Format registrieren.


Das mobile Endgerät muss in die Konfiguration des Zutrittskontroll- und Zeiterfassungssystems PERCo-Web einbezogen werden.



Um mit der Anwendung zu arbeiten, müssen Sie die IP-Adresse des PERCo-Web-Servers eingeben oder den QR-Code scannen.



Daten können über ein Wi-Fi-Netzwerk oder über ein Mobilfunknetz an den Server übertragen werden. Wenn das Terminal offline ist, werden alle Zugriffsereignisse im Anwendungsspeicher gespeichert und beim Herstellen einer Verbindung an den Server gesendet.


Nachdem das Terminal mit der Konfiguration verbunden wurde, können Passagen von Mitarbeitern und Besuchern registriert werden, die in Systemereignissen angezeigt werden.



Die Registrierung kann in verschiedenen Modi durchgeführt werden:


  • "Eintritt" - Wenn eine Karte vorgelegt wird, wird der Eintritt registriert
  • "Beenden" - Bei Vorlage einer Karte wird ein Beenden registriert
  • "Verifikation" - Es ist notwendig, die Erlaubnis des Fahrers mit den Enter / Exit-Tasten zu bestätigen

Bei der Vorlage der Kennung werden Name und Foto des Mitarbeiters auf dem Terminalbildschirm angezeigt. Der Bildschirm zeigt auch Informationen darüber an, ob der Zugriff mit dieser Kennung im aktuellen Zeitraum zulässig ist.


Mit einem mobilen Zutrittsterminal können Sie die Zeiterfassung für Mitarbeiter organisieren. Basierend auf den Daten der registrierten Ein- / Ausgänge berechnet das System die Arbeitszeit für den Monat und bildet ein Arbeitszeitblatt. Schicht-, Wochen- und fortlaufende Zeitpläne werden unterstützt.


Das Terminal ist im Notfall nützlich, zum Beispiel bei einem Brand. Die Möglichkeit, im Notfall den Aufenthaltsort von Personen zu verfolgen, erhöht die Rettungswahrscheinlichkeit erheblich.
