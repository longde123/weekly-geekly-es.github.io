Google gab zu, dass es auch GitHub kaufen wollte

Diane Greene, Direktorin von Google Cloud, beantwortete die Fragen eines Moderators bei einer Veranstaltung des Fortune Magazine und gab eine interessante Erklärung zum Kauf von GitHub durch Microsoft ab: "Es würde mir nichts ausmachen, wenn wir sie kaufen würden, aber es ist in Ordnung." Ich habe nichts dagegen gehabt, sie zu kaufen, aber es ist in Ordnung), sagte sie .

So gab eine der führenden Unternehmen von Google tatsächlich zu, dass sie auch nicht abgeneigt war, eine Hosting-Plattform zu erwerben.

Früher gab es Gerüchte, dass Google ein Kaufangebot machte, aber Diana Green weigerte sich, diese Gerüchte zu bestätigen oder abzulehnen. Sie hoffte, dass GitHub unter dem Einfluss von Microsoft völlig neutral bleiben würde.

Für Redmond ist GitHub ein Schlüsselelement im hart umkämpften Cloud-Computing-Markt. Diana Green bemerkte auch, dass zwei Google-Produkte jetzt auf der Liste der beliebtesten Repositories auf GitHub stehen. Sie erwartet, dass Microsoft die Situation nicht ändern wird.

GitHub-Gründer Chris Wanstrath soll sich wegen seiner Bekanntschaft mit Microsoft-CEO Satya Nadella zwischen Google und Microsoft für Letzteres entschieden haben. Als Eigentümer hat er jedoch das Recht, sein Unternehmen an jedermann zu verkaufen und es auf Wunsch sogar kostenlos zu verschenken. Und hier ist der Deal 7,5 Milliarden Dollar wert.

Heute ist GitHub das beliebteste Quellcode-Repository im Internet. Es beherbergt 85 Millionen Repositorys, über die 28 Millionen Entwickler arbeiten. Unternehmen platzieren hier Projekte, Dokumentationen und Code, organisieren die gemeinsame Arbeit ihrer Mitarbeiter. Es gibt Beispiele, in denen GitHub persönliche Blogs und sogar Kommentar-Feeds in Blogs von Drittanbietern hostet. Diese Plattform wird von großen Unternehmen wie Apple, Amazon, Facebook, Google und demselben Microsoft verwendet.

Nach dem Kauf hat GitHub wahrscheinlich den Status einer unabhängigen Plattform verloren. Die Frage ist nun, ob Unternehmen, die Microsoft als ihren Konkurrenten betrachten, es weiterhin nutzen werden. Eine ähnliche Frage stellt sich bei normalen Entwicklern, von denen viele Microsoft in einem unansehnlichen Licht sehen. Viele erinnern sich noch daran, wie der frühere CEO des Unternehmens, Steve Ballmer, im Jahr 2001 Linux und alle freie Software als „Krebstumor“ bezeichnete und einen Stuhl auf den Mitarbeiter warf, der sagte, er würde Google verlassen. Steve Ballmer war wütend und verflucht ("Eric Schmidt ist ein Fan. Ich werde diesen Kerl begraben, ich habe es schon einmal getan, und "Ich werde es wieder tun. Ich werde Google töten", so ein Mitarbeiter. All dies in der Geschichte von Microsoft, und das Bild ist schwer zu reparieren. Werden Entwickler ihren Code weiterhin hier veröffentlichen? Diese Fragen bleiben offen.

Kurz nach dem Kauf führte The Verge eine Umfrage unter Vertretern einiger großer Unternehmen durch, ob sie GitHub weiterhin verwenden würden. Vertreter von Google lehnten einen Kommentar ab, und Facebook sagte, dass sie diese Website weiterhin nutzen würden, solange sie so nützlich und bequem bleibt wie jetzt.

Nach den Worten von Diana Green zu urteilen („Ich hoffe wirklich, dass Microsoft das Projekt völlig neutral lässt“), teilt sie die Bedenken vieler Entwickler. Die meisten von ihnen haben jetzt eine abwartende Haltung eingenommen. Sie beobachten, was als nächstes passiert. Tatsächlich ist diese Transaktion noch nicht abgeschlossen: Der Abschluss sollte innerhalb eines Jahres erfolgen.

Einige Entwickler sehen den Deal optimistisch und erwähnen das Engagement von Microsoft für die Bewegung freier Software in den letzten Jahren und die Unternehmensgeschichte mit hervorragenden Entwicklertools.

Andere bleiben pessimistisch. Zum Beispiel sagt Victor Felder , der an vielen Open-Source-Projekten beteiligt ist, dass es nicht einmal Microsoft ist, sondern die grundlegende Unabhängigkeit von GitHub. Wenn das Projekt von Google, Apple, Amazon oder Facebook gekauft worden wäre, hätte sich nichts geändert: Ein solcher Deal stärkt die Machtkonzentration in den Händen kommerzieller Technologieriesen und entzieht der freien Community die Macht.
