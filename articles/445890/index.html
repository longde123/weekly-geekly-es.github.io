PyTest Neo

Una tarde, contemplando un sinfín de pruebas y observando los puntos de carrera, se le ocurrió una idea: ¡qué aspecto tiene Matrix! Y así nació la idea, hacer la visualización de pruebas en forma de matriz.



Debajo de la animación del gato y algunos detalles.



Si alguien está preocupado por la legibilidad, solo se anima la barra de progreso y, si hay errores, se muestra un informe estándar.

Para instalar el complemento pytest-neo , simplemente haga: 

pip install pytest-neo

El proyecto se encuentra en una etapa temprana de desarrollo. Linux y presumiblemente MacOS son compatibles. Posibles errores. Hasta ahora, el modo detallado se deja como está. Los planes para implementar una animación confiable de Matrix.

Por favor escriba sugerencias y sugerencias. Esto será de gran ayuda para el desarrollo del proyecto.

PD: en las pruebas aiohttp se detectó una anomalía, estudiaré.
