Sélection: 5 outils d'analyse concurrentielle non évidents que vous ne connaissiez peut-être pas



L'analyse concurrentielle est un sujet extrêmement intéressant, mais la plupart des articles sur ce sujet ont une immersion minimale dans le problème. On nous dira de texte en texte à quel point les outils sont utiles, comme SimilarWeb, un accès payant à des coûts comme Boeing, Alexa, qui était pertinent pour la dernière fois il y a de nombreuses années, ou App Annie, qui sera presque impossible à apprendre à utiliser rapidement.

Je suis professionnellement engagé dans le marketing, y compris sur les marchés internationaux, depuis de nombreuses années et j'ai compilé une liste de cinq outils non évidents pour étudier la progression de vos concurrents.

Infatica


Ce n'est pas du tout évident, mais les services de proxy résidents peuvent être utilisés avec succès pour l'intelligence économique. L'idée est simple: aujourd'hui, les entreprises ont appris à gérer les tentatives d'analyse de leurs sites à l'aide de robots. Le plus souvent, ces logiciels utilisent IP à partir de pools appartenant à des fournisseurs de proxy. Les bloquer est assez facile.

Si vous pouvez louer des adresses qui ne diffèrent pas de celles utilisées par les utilisateurs ordinaires, vous pouvez établir une connexion et étudier le contenu dont vous avez besoin. De plus, les proxys résidents vous permettent d'analyser les résultats de recherche et la publicité dans différents pays - vous le verrez comme les utilisateurs de ces pays le voient.



J'utilise le service Infatica car ils n'ont aucune restriction sur le volume de pompage du trafic ou les sessions établies simultanément.

Apptica


Si vous travaillez dans un domaine où la publicité dite in-app est activement utilisée (c'est-à-dire les publicités dans les applications mobiles), il n'y a pas beaucoup d'options disponibles pour suivre l'activité des entreprises rivales. Un des outils fonctionnels et peu coûteux est le service Apptica.

Avec lui, vous pouvez obtenir des informations sur les types d'annonces et les formats et réseaux publicitaires utilisés par les annonceurs.



Le service est également utile pour les développeurs d'applications mobiles, car vous pouvez analyser les campagnes in-app lancées dans des applications concurrentes. Par conséquent, vous pouvez comprendre quel type de publicité doit être implémenté dans l'application.

MOAT


Un outil simple qui vous permet de voir quel type de publicité en ligne une entreprise particulière utilise. Il ressemble à un moteur de recherche simple, dans lequel vous devez conduire le nom de l'entreprise souhaitée.

En conséquence, le système publiera une sélection de publicités de différents formats («créatifs»):



Pour chacun d'eux, le rapport fournira des informations sur les sites sur lesquels ce type de publicité a été placée, le type de fichier et sa taille. Autrement dit, avec l'aide de MOAT, vous pouvez comprendre non seulement quel type de concurrents de publicité lancent, mais où exactement ils le font.

Ivalival


Nous avons déjà examiné les outils d'analyse de l'activité publicitaire des concurrents dans les applications mobiles, les sites externes, et RivalIQ aide à mener une étude des réseaux sociaux.

Le service présente un rapport agrégé sur les concurrents, qui comprend des données provenant de sites tels que Facebook, Twitter, YouTube et Instagram. Dans le même temps, vous pouvez non seulement recevoir des données sur où et quels articles les entreprises publient, quel type de réponse elles reçoivent, mais aussi faire des comparaisons.



Il est clair pourquoi cela est nécessaire - en s'appuyant sur des données sur l'activité des concurrents sur les réseaux sociaux, vous pouvez utiliser ce canal plus efficacement.

Ghostery


En général, Ghostery est initialement un outil de confidentialité en ligne. Par exemple, en utilisant l'extension Chrome, les utilisateurs peuvent trouver tous les différents trackers Web intégrés dans une page Web et les bloquer (cela empêche la collecte de données et accélère le travail):



Ces informations sont très précieuses pour un spécialiste du marketing - il est si facile de savoir quelles données les entreprises collectent sur leurs utilisateurs.
