Windows for IoT: Dukungan Perangkat Keras yang Disempurnakan dan Fitur Perangkat Cerdas Baru



Dunia IoT menjadi lebih menarik, berdampak pada semua bidang bisnis di dunia, dan transformasi ini semakin cepat. Kemajuan dalam AI dan pembelajaran mesin membantu perusahaan di semua industri meningkatkan efisiensi, mengurangi waktu henti, biaya dan tingkat kesalahan, dan meningkatkan kepuasan pelanggan menggunakan kemampuan Intelligent Edge Devices dan Intelligent Cloud. Artikel ini akan berbicara lebih banyak tentang Windows untuk IoT.

Windows telah bekerja pada perangkat industri tertanam selama lebih dari dua dekade, dan selama pidatonya di CEATEC di Jepang, saya berbagi strategi Microsoft, yang menurutnya Windows for IoT memberi pengembang perangkat berbagai tingkat kemampuan untuk dengan cepat menciptakan solusi IoT dan membantu pelanggan melakukan transformasi digital dengan menggunakan perangkat IoT yang cerdas dan aman. CEATEC berfokus pada pengembangan kota pintar, bangunan pintar, dan infrastruktur pintar sebagai bagian dari strategi Society 5.0 ("Society 5.0"), dan kami senang bahwa Windows untuk IoT menyediakan keluarga sistem operasi yang dapat diskalakan sebagai fondasi untuk membangun perangkat cerdas. Keluarga ini termasuk:

  • Windows 10 IoT Core . OS ini dirancang untuk perangkat pintar berukuran kecil, tetapi pada saat yang sama memberikan pengelolaan dan keamanan di tingkat yang ditunggu pengguna Windows. Bersama dengan Windows 10 IoT Core Services , dukungan OS jangka panjang disediakan, serta layanan untuk mengelola pembaruan nirkabel (OTA) dan kesehatan perangkat. Contoh hebat dari Windows 10 IoT Core adalah Hourfleet , sebuah platform berbagi mobil yang menangani masalah mobilitas perkotaan yang dihadapi banyak kota dan masyarakat. Platform Hourfleet adalah produk yang digunakan secara global, andal, dan hemat biaya berdasarkan Windows 10 IoT Core dan Azure IoT.
  • Windows 10 IoT Enterprise . Versi lengkap dari Windows 10 IoT ini menyediakan pengelolaan tingkat perusahaan dan membatasi kemampuan perangkat pintar dengan fitur-fitur tetap. Perusahaan seperti City Beacon sedang membangun generasi berikutnya dari infrastruktur cerdas menggunakan Windows 10 IoT Enterprise. City Beacon telah membangun hub pintar Smart Hub, yang dapat ditempatkan di jalan-jalan kota. Ini adalah kios multifungsi dengan beragam perangkat IoT yang terhubung ke jaringan cepat dan dapat memberi warga akses ke aplikasi dan layanan hiperlokal.
  • Windows Server IoT 2019 . Server yang menjalankan Windows Server IoT dirancang untuk beban kerja yang paling berat, untuk menghubungkan, menyimpan, dan menganalisis informasi dari basis data besar yang digunakan oleh aplikasi, jaringan, dan layanan web yang terhubung.

Lebih banyak platform perangkat keras


Pada 14 Oktober, kami mengumumkan ketersediaan umum Paket Dukungan Papan Inti Windows 10 IoT untuk keluarga prosesor NXP i.MX, termasuk keluarga prosesor i.MX6, i.MX7, i.MX7, i.MX 8M dan i.MX 8M Mini.

NXP adalah pemimpin dalam solusi keamanan tinggi untuk menghubungkan aplikasi yang tertanam, dan kombinasi NXP i.MX dan Windows 10 IoT Core memberi pengembang opsi tambahan untuk membuat perangkat yang aman dan terhubung pada tingkat komersial.

Windows 10 IoT Core memungkinkan Anda memanfaatkan sepenuhnya perangkat pintar yang aman dengan konsumsi daya yang dioptimalkan. Pengembang dapat dengan cepat membuat perangkat pintar baru menggunakan lingkungan pengembangan Windows yang kuat dan dikomersialkan menggunakan platform Azure IoT kami. Menambahkan dukungan untuk prosesor NXP, kami memberikan para pencipta proyek IoT kesempatan untuk menggunakan sejumlah microcircuits tingkat industri baru.

"Kami senang telah mencapai tahap ketersediaan umum, dan kami terus membangun kemitraan dengan Microsoft," kata Jeff Miles, Wakil Presiden Pengembangan Bisnis NXP, Secure IoT Devices dan Cloud Products. "Sekarang pelanggan NXP memiliki akses ke solusi perangkat pintar terkemuka di industri dan dapat menggunakan prosesor i.MX 6, i.MX 7 dan i.MX 8M dengan Windows 10 IoT Core untuk aplikasi industri, termasuk untuk proyek IoT," Menemukan kekuatan Azure untuk produk cerdas Anda. "

Konsumen Mulai Menggunakan NXP i.MX dan Windows 10 IoT Core


Sudah ada papan dan perangkat yang menggunakan prosesor NXP i.MX. Mitra kami telah merilis dan sedang mengembangkan lebih dari 15 dewan tersebut. Misalnya, Diagramics mengembangkan solusi berbasis NXP i.MX8 dengan Windows 10 IoT Core untuk membuat perangkat klien multi-fungsional yang disertifikasi sesuai dengan katalog perangkat Azure IoT dan memungkinkan untuk mendaftarkan perangkat IoT ini di Layanan Penyediaan Perangkat Hub Azure IoT pada plug-in basis n-main Pelajari lebih lanjut tentang ini di sini .

Dalam pidato saya, saya juga berbicara tentang solusi baru dari Smart Tower, berdasarkan kombinasi Windows 10 IoT Core, NXP dan Azure IoT. Solusi ini dirancang untuk meningkatkan keandalan dan mengurangi waktu henti infrastruktur kritis, seperti menara sel atau jaringan distribusi daya. Smart Tower telah mengembangkan solusi yang memungkinkan Anda untuk sepenuhnya menggunakan keuntungan dari pemrosesan pada perangkat perbatasan dengan membuat modul sensor yang terpasang pada menara, dan unit kontrol tanah yang menganalisis keadaan menara secara real time, dengan mempertimbangkan kemungkinan penundaan tinggi atau sinyal bisu. Selain itu, solusinya menggunakan Azure IoT Hub untuk mengelola dan menghubungkan perangkat dan memungkinkan analisis historis dari seluruh jaringan menara yang dikendalikan. Smart Tower memilih kombinasi NXP, i.MX dan Windows 10 IoT Core karena kondisi ekstrim dari perangkat, dan platform Windows IoT yang sudah dikenal mengurangi biaya pengembangan dan waktu untuk memasarkan.

Inovasi Perbatasan Tepi Berkelanjutan


Awal tahun ini, kami menyertakan dukungan Azure IoT Edge di Windows 10 IoT Enterprise dan Windows Server IoT 2019. Hal ini memudahkan untuk mengintegrasikan runtime IoT Edge, memungkinkan Anda untuk mentransfer algoritma pembelajaran mesin dan fungsi komputasi lain yang sama canggihnya dari cloud ke perangkat periferal. Windows untuk IoT juga mendukung Windows Machine Learning dan Windows Vision Skills, yang memungkinkan Anda untuk menjalankan algoritma AI canggih yang dikembangkan di Azure pada perangkat Windows IoT apa pun.

Alat Pembelajaran Mesin Edge Pembelajaran terdepan di industri: Windows ML Container Baru


Wadah Windows ML baru, yang sudah tersedia untuk anggota Program Orang Dalam, dirancang khusus untuk tugas yang membutuhkan beban kerja cerdas yang dibuat menggunakan Windows ML API. Wadah Windows ML memberi perusahaan platform cepat dan fleksibel untuk menciptakan solusi IoT kelas perusahaan. Ini menggabungkan fitur Windows canggih dengan keamanan platform Windows 10 IoT dan pengelolaan Azure IoT Edge.

Wadah Windows ML menyediakan beban kerja akselerasi perangkat keras yang cerdas dan akses langsung ke perangkat host. Beban kerja yang dihosting dapat menggunakan sensor atau kamera yang terhubung melalui USB, I2C, SPI, atau GPIO. Untuk akselerasi perangkat keras, Anda dapat menggunakan GPU DirectX 12 atau ASIC terintegrasi host yang kompatibel dengan Microsoft Compute Driver Model.

Berkat optimalisasi wadah khusus untuk beban kerja AI, ukuran wadah pada disk adalah sekitar 350 MB, yang jauh lebih kecil daripada wadah lain di pasar yang mendukung pemrosesan GPU. Ukuran yang lebih kecil memfasilitasi pengelolaan wadah Windows ML dan mengurangi biaya. Pelajari lebih lanjut tentang versi pratinjau wadah Windows ML di sini .

Memperkenalkan SQL Server IoT 2019: Penyimpanan dan Analisis Data Windows untuk IoT


SQL Server telah tersedia untuk digunakan dengan perangkat tertanam selama bertahun-tahun, dan kami senang memperkenalkan fitur SQL Server 2019 terbaru kepada pengembang perangkat tertanam. Memperkenalkan SQL Server IoT 2019, setara biner dari SQL Server 2019, yang akan dilisensikan melalui saluran OEM dan secara khusus dirancang untuk digunakan pada server khusus yang menjalankan perangkat tepi dengan aplikasi. Dikombinasikan dengan Windows Server IoT 2019, Microsoft SQL Server IoT 2019 menyediakan platform basis data yang komprehensif untuk analisis data yang sangat penting dengan skalabilitas yang tak tertandingi, kinerja terkemuka di industri, keamanan dan kemampuan untuk memproses data secara cerdas. SQL Server IoT memungkinkan analisis data operasional real-time menggunakan teknologi dalam-memori yang memberikan konkurensi dan skalabilitas yang lebih tinggi melalui memori hanya baca. Microsoft SQL Server IoT 2019 akan tersedia akhir tahun ini.

Microsoft banyak berinvestasi dalam teknologi cloud dan memanfaatkan kekuatan Windows untuk mempercepat transisi pelanggan ke cloud dengan perangkat pintar yang terhubung. Inovasi yang kami umumkan di CEATEC dan solusi IoT praktis yang dibuat oleh pelanggan dan mitra kami hanyalah awal dari perjalanan yang mengasyikkan yang membuka Windows untuk IoT. Pastikan untuk mengetahui solusi Windows IoT ini secara langsung atau melalui Internet .
