SPA Meetup 5: Integração Jest com controle de qualidade, UIKit poderoso, bibliotecas de componentes, DI para dimensionamento, comandos de plataforma

Oi Este é um pós-relatório do mitap SPA SPA de Moscou, uma série de reuniões para desenvolvedores de front-end interessados ​​no tópico de aplicativos da web (aplicativo de página única). Juntamente com palestrantes do Mail.Ru, Yandex, VKontakte, Tinkoff e Avito, conversamos sobre a integração do Jest com a infraestrutura de controle de qualidade, o DI como uma maneira de escalar, o crescimento das bibliotecas de componentes e uma equipe de plataforma. Eles também discutiram se o UIKit pode ser o mais flexível e adequado para tudo quanto possível.


Venha para o gato, se você estiver interessado nesses tópicos. Dentro, você encontrará materiais para reuniões: gravações em vídeo de apresentações, apresentações com oradores, comentários de convidados da reunião e links para o relatório fotográfico.
