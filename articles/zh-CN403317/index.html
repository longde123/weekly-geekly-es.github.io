评论Lenovo IdeaPad Y900-17ISK游戏笔记本电脑

功能强大的游戏笔记本电脑可能并不薄，但每个规则都有例外，联想Y900就是其中之一。 我们对长度和宽度保持沉默-尽管如此，我们谈论的是游戏的“种子”-但是关于Y900的纤薄性，值得一提。 笔记本电脑的厚度为36毫米，坦率地说，与其他游戏“种子”相比，还差一点。 具有类似技术特性和重量的Y900竞争对手不再具有这种尺寸，它们的尺寸将增加10-15毫米。 至于重量，这里没有发生奇迹，Y900游戏重约4.38公斤-很难随身携带。 做什么-对于强大的游戏“种子”来说，这样的重量是可以接受的。

笔记本电脑的盖子由金属制成，具有精美的抛光质感。 在其中心是“ Y900”游戏系列的鲜明徽标，在黑色背景下非常醒目。

在游戏配件上可以找到相同的徽标：耳机和鼠标。


关闭时，联想Y900有点让人联想到昂贵的跑车。 类似于尾灯的红色内衬增强了这种相似感。 它们位于冷却系统通风格栅的上方。 工程师的想法很可能是差不多的。

对于连接外围设备，Lenovo Y900具有两个USB 3.0连接器（左）和相同的USB 2.0（右）。 对于最现代，最快的设备，Y900带有一个USB 3.1 Type C连接器（Thunderbolt 3），焊接在左侧面板上。 DisplayPort和HDMI连接器可用于连接外部显示器，如果这还不够，则可以使用USB 3.1 Type C，因为Thunderbolt 3控制器允许您将两台分辨率为4K的外部显示器或一个显示器连接到5K显示器。


为了访问本地网络，Lenovo Y900具有一个千兆以太网连接器（Killer e2400控制器）和一个Killer Wireless-AC 1535 Wi-Fi模块，顺便说一下，无线模块上连接了两个天线，从而形成了两个接收通道和一个发射通道。数据（MIMO 2x2：2技术）。 这种方法完美地影响了无线网络的信号强度并提高了数据传输速率。


由于屏幕的对角线宽达17英寸，因此Y900拥有足够的空间来容纳带有数字块的键盘。 在它的左边是四个用于宏的键和一个用于对处理器进行超频的按钮。 并且由于后者位于Esc键的下方，因此我们避免了意外的点击。

尽管Y900属于游戏机型，但其键盘是机械键盘，并且以可识别的样式制成。 在这里，我们看到了多年来在著名的联想系列产品中使用过的相同按键：IdeaPad和ThinkPad。 除非WASD游戏按钮收到亮红色的大写字母。 使用Y900，您可以舒适地玩游戏和键入文字。 事实证明，笔记本电脑的按键行程很深，约为2.5毫米，按压的时刻感觉良好。 总的来说，我们之前使用的是相同的Lenovo键盘-一种最方便，最周到的笔记本电脑键盘，具有出色的触觉响应和耐用的键。

顺便说一句，Fn键在通常的位置-键盘的左下方，Ctrl的右侧。 如果这种安排不适合某人，则可以在BIOS中交换这些密钥。

Y900键盘具有完全可定制的背光，非常适合现代游戏笔记本电脑。 拉丁字符和俄语字符都突出显示。 至于WASD游戏键，键上没有俄语字符，因为这些按钮带有笔记本电脑随附的“游戏”盖。 但是，无法安装它们-俄语和拉丁字符应用在“本机”盖上。

值得注意的是工作区域。 它采用令人愉悦的触感，略微粗糙的塑料Soft-Touch制成，触摸板的边框很小，可自定义背光。 触摸面板的尺寸为105×70 mm，位于一个小凹处，其凹陷深度约为1 mm。

在IdeaPad Y900的背面，除了冷却系统的孔和覆盖低音炮的独立烤架之外，没有其他有趣的东西。 后者为三角形，位于下方。

最后，值得一提的是IdeaPad Y900的屏幕。 联想决定在笔记本电脑中安装17.3英寸光滑IPS面板，该面板带有抗反射涂层，分辨率为1920×1080。 当然，这里可能会出现分辨率更高的屏幕，例如，在移动领域流行的“ 2K”或通常的4K，但是，如实践所示，在这样的面板上播放并不容易。 事实是，视频卡通常无法将游戏的分辨率（例如1920x1080像素）扩展到具有更高奇异分辨率的屏幕，因此游戏将以1920x1080的分辨率开始，但以1：1的比例或以其本机的最大屏幕分辨率忽略，所有其他设置。 在两种情况下都是不方便的。 在第一种情况下，图片将太小；在第二种情况下，移动视频卡将无法在最大图形设置下适应高分辨率，这会将任何游戏变成幻灯片播放。 因此，分辨率为1920×1080像素的17英寸屏幕可以称为游戏笔记本电脑的最佳选择。

联想IdeaPad Y900的核心是四核Intel Core i7-6820HK处理器。 是的，这不是新奇的Kaby Lake，但说实话：在当前条件下，追求处理器性能绝对没有意义。 正如大量评论和测试所表明的那样，Kaby Lake在性能上并不比Skylake差很多，而且四核i7-6820HK的动力储备甚至足以应付繁重的任务，例如处理视频。

该处理器的工作频率为2.7 GHz，使用Turbo Boost技术可以将其提高到3.6 GHz。 顺便说一下，这是少数支持超频的英特尔移动处理器之一。 得益于出色的IdeaPad Y900冷却系统和自动加速键，您可以轻松快捷地向处理器添加数百兆赫兹。


使用现代DDR4内存，频率为2133 MHz，RAM容量为16 GB。 笔记本电脑总共有四个RAM插槽。 前两个已经被模块占用，位于键盘下方，另外两个位于通常的位置，位于下部服务面板的后面。 随着内存容量的增加，将不会有问题。


IdeaPad Y900的视频卡是最好的之一。 这是旗舰NVIDIA GeForce GTX 980M，并配有8 GB的GDDR5视频内存。 这种组合对于所有现代游戏都足够了。


同样有趣的是磁盘子系统，用户可以提高自己的品味。 在对IdeaPad Y900的修改中，我们安装了两个快速M.2驱动器，每个驱动器容量为128 GB，以及一个硬盘驱动器，容量为1 TB。 您可以使用两个SSD构建非常快速的RAID-0阵列，并且可以将电影，发行或最重要文件的备份副本存储在硬盘上。

我们已经讨论过IdeaPad Y900的网络功能。 Killer e2400网络适配器负责千兆位以太网连接器的运行，负责负责无线网络的Killer Wireless-AC 1535的无线通信标准是频率为2.4和5 GHz时的802.11 a / b / g / n / ac。 为了连接各种无线外围设备（例如鼠标和耳机），可以使用蓝牙4.1版。

但是事实证明，笔记本电脑的电池是不可拆卸的。 一方面，这有点不寻常。 另一方面，这样的设计可以赢得几毫米的距离，并使笔记本电脑更薄。 尺寸的减小不会损害游戏笔记本电脑。

至于硬盘的速度，那么这里的一切都是微不足道的。 其结果与任何其他2.5英寸驱动器相似，并且无需等待奇迹。



现在介绍一下IdeaPad Y900冷却系统。 她原来是所有游戏模型的标准设计。 处理器和视频卡的散热器（以及视频适配器的热组件）具有由两个涡轮机冷却的热管。 在空闲模式下，两个笔记本电脑涡轮几乎无法听见声音。 来自CO的噪声与背景噪声合并在一起，并且没有超过其水平。 在最大负载下，IdeaPad Y900开始发出更多的噪音，但是负载涡轮的音量水平并不高于正常通话水平。 总的来说，联想工程师因其安静高效的冷却系统而倍受赞誉，该公司的程序员应感谢Nerve Sense应用程序，在该应用程序中，您可以强制全功率打开CO涡轮机。 此功能在夏季以及激烈的游戏战斗中都很有用，因为铁的速度比冷却系统产生的噪音要重要得多。

同样值得注意的是，联想IdeaPad Y900的出色音质。 由于笔记本电脑尺寸适中，因此在其内部不仅可以放置扬声器，而且还可以放置紧凑的移动低音炮。 IdeaPad Y900听起来很棒。 您不仅可以玩这样的笔记本电脑，还可以看电影或听音乐。 音频部分没有任何抱怨，即使在最大音量下（事实证明它很高），我们也没有听到扬声器之间有多余的声音或喘息声。 联想IdeaPad Y900的声音也许可以称为游戏机型中最好的声音之一。 这并不奇怪，因为JBL工程师还致力于其声音。
