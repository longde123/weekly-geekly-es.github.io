Mikrotik mAP lite

Assim que vi, adquiri um novo modelo em miniatura Mikrotik RouterBoard mAP Lite sobre o qual quero compartilhar.


Eu já tinha um Mikrotik RouterBoard mAP 2n , mas o mAP lite realmente me impressionou com suas dimensões: 48x49x11mm
Imagens MAP Lite









As especificações e testes do dispositivo podem ser encontrados no site do fabricante. Resumidamente comparado ao mAP 2n:
  • Frequência do processador 650 MHz, que é mais do que a do mAP 2n - 400 MHz
  • RAM de 64 MB como mAP 2n
  • FLASH 16 MB como mAP 2n
  • Uma porta Ethernet (10/100) - o mAP 2n possui dois
  • Wi-Fi QCA9533 802.11b / g / n - no mAP 2n AR9331

nada mais que natural, exceto o tamanho 48x49x11mm versus 68x68x19mm no mAP 2n.
O preço recomendado do dispositivo no site é de US $ 25, eu consegui comprá-lo com entrega por 2700 rublos.
A primeira surpresa é o tamanho da caixa (o mesmo mAP 2n vem em uma caixa longa)
Embalagem mAP lite e mAP 2n



O mAP lite vem com uma fonte de alimentação (5V - 1A), suporte metálico, pés de borracha.
Pacote




MAP Lite









A primeira coisa que desmontei o bebê. Acabou não sendo fácil abrir. Fixadores de pressão muito apertados.
Dentro








Dentro do dispositivo existem ímãs, que permitem montar o roteador em qualquer superfície metálica, mas o mais importante no quadrado que o acompanha.
Mount




Potência verificada através de PoE (802.3af / at (+ 4.5 / -7.8 ou + 3.6 / -1.2))
PoE


No mAP Lite havia um novo firmware. RouterOS padrão e regulares. Tudo funcionou sem problemas. Está quente no trabalho (embora possa ser, se você direcionar um tráfego mais substancial, a imagem mudará).
Este dispositivo é montado na próxima montagem.
Assembléia





A essência é a seguinte. Conectei um roteador e um modem com 4G e Wi-Fi - Huawei E8372 (MTS 8211F) ao banco de potência (Romoss solo 3 PH30 6000mAh). O roteador se apega à Ethernet, se não estiver conectado, a determinadas redes Wi-Fi, se não o encontrar, ao modem. Em seguida, VPN L2TP / IPsec para o servidor doméstico para o SoftEther. Proxy da Web configurado para redirecionar para o proxy doméstico. Agora vou percorrer os scripts e, no final, os publicarei em um novo artigo, se for interessante.

Lista de desejos:
  • USB Porta separada. A julgar pelo caso, o fabricante também pensa nisso. Pode-se conectar um modem diretamente.
  • 802.11ac
  • Sinal sonoro - você realmente precisa
  • Controle de LED como no mAP 2n

Outra porta Ethernet e energia para outros dispositivos através dela, mas isso já levará a um aumento de tamanho, o que eu não quero.
