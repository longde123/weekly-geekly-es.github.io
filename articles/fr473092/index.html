AMA avec Habr, # 13: nouvelles importantes pour les utilisateurs et les entreprises

Qu'est-ce qui n'a pas remarqué? Le dernier vendredi d'octobre est arrivé, ce qui signifie que le temps pour la prochaine ligne directe avec les employés Habr. Aujourd'hui au programme: nouveaux compteurs de blogs d'entreprise, changement de type de publication et de chat dans Telegram. En échange de nouvelles, nous attendons vos questions, souhaits et suggestions sur tous les sujets okolokhabrovskie.

Modifier le type de publication


Parfois, les utilisateurs ont indiqué à tort le type de publication - «Publier» au lieu de traduction, ou «Publier» au lieu de News. Oui, et avec une langue différente. «Assez pour supporter cela», avons-nous pensé, et avons permis de changer le type de publication après sa publication - le commutateur est sur la page de post-édition.



Toutes les métriques de publication lors du changement de type quoi? C'est vrai, sauvé.

Blog d'entreprise Analytics


Si vous êtes une entreprise et que vous avez un blog sur Habré, vous devez simplement consacrer la soirée du vendredi à la connexion du compteur Yandex.Metrica ou de Google Analytics. Pour ce faire, dans le panneau d'administration du blog, vous devez vous rendre dans la section "Autre" → "Analytique" et configurer le compteur conformément aux instructions. Si vous avez plusieurs sites ou projets, mais qu'ils ont tous un blog sur Habré, un seul compteur vous suffit - toutes les informations y seront collectées. Vous pourrez voir quelles requêtes de recherche sont entrées dans votre blog sur Habré, combien de visiteurs et avec quel profil ils y vont quotidiennement. Les cartes de défilement et de clic fonctionneront probablement bientôt (le plugin iFrame Allow sera nécessaire). Il existe de nombreux rapports différents - si vous les regardez et les comparez avec les métriques de votre site, il y aura de nombreuses idées pour travailler avec du contenu - les premiers avis de clients d'entreprise en parlent avec éloquence.



En général, vous savez quoi faire. Allez-y :)
