Terminal Windows Baru Sekarang Tersedia di Microsoft Store

Terminal Windows baru, yang diumumkan Microsoft di MS Build 2019 , sudah tersedia untuk diunduh di toko, menurut blog resmi. Bagi yang berminat, repositori proyek di GitHub .


Terminal adalah aplikasi Windows baru untuk akses terpusat ke subsistem PowerShell, Cmd dan Linux dalam kemasan Windows Subsystem Linux. Yang terakhir menjadi tersedia untuk membangun Windows Insider build 18917 pada 20 Juni.

Untuk menggunakan Terminal baru, dua syarat harus dipenuhi: instal Windows 10 versi 18362.0 atau lebih tinggi dan temukan tombol Microsoft Store. Tentu saja, Anda selalu dapat mengkompilasi Terminal dari jenis yang diposting di GitHub, tetapi pengembang memperingatkan bahwa dalam kasus ini, "versi yang dirakit secara manual akan bekerja secara paralel dengan versi dari toko." Rupanya, dipahami bahwa toko tidak akan mengambil terminal yang dirakit secara manual dan tidak akan diperbarui sendiri.

Salah satu fitur utama Terminal, yang secara aktif "dijual" di blog perusahaan, adalah sejumlah besar profil.



Masing-masing profil dapat dikonfigurasi secara terpisah dengan mengedit file JSON yang sesuai.



Microsoft juga menawarkan setiap pengguna untuk memilih hotkey dan kombinasi mana yang akan digunakan, dan menyesuaikannya sesuai selera mereka.

Cherry kustomisasi adalah kemampuan untuk mengubah gambar latar belakang setiap profil melalui penarikan dangkal gambar dari hard drive. Jadi tidak ada batasan imajinasi.

Sekarang mari kita sedikit lebih serius


Mengapa tidak ada detail teknis di posting blog Microsoft? Mengapa penekanan pada kustomisasi, hotkey dan kosmetik lainnya?

Pertama, semua orang sudah mengatakan tentang Terminal di Build 2019, dan tidak ada yang istimewa untuk ditambahkan. Sekarang perusahaan sedang mencoba untuk menunjukkan bahwa aplikasi baru adalah produk yang ramah dan nyaman yang sejalan dengan WSL baru. Faktanya, Microsoft baru saja meluncurkan apa yang mereka janjikan pada bulan Mei dan tidak ada yang istimewa untuk ditambahkan.

Yang kedua - beberapa waktu akan berlalu sebelum rilis versi 1.0. Dilihat oleh teks blog Microsoft, Terminal tidak akan keluar dari tahap menggergaji aktif sebelum musim dingin, yaitu, itu akan muncul pada versi stabil Windows di toko hanya enam bulan kemudian.

Pada saat yang sama, perwakilan perusahaan secara aktif mengagitasi masyarakat untuk memberikan umpan balik pada produk baru. Jadi, Microsoft akan sangat berterima kasih atas komentar dan saran tentang Terminal dalam repositorinya sendiri di Github dan, katakanlah, masyarakat merespons teriakan ini. Kami berpikir bahwa selama minggu depan akan ada banyak komentar di "masalah".
