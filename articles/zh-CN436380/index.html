3D打印快速入门：适合初学者或大众的廉价打印机

现在已经到了，在家中拥有一台小型台式3D打印机并不会完全无利可图的情况，使用它并不需要对技术或打印经验有深入的了解。 该商店有几种价格低于$ 200（！）的好机型，适合新手用户使用。 所有3D打印机均可使用流行且安全的PLA塑料进行打印，这种塑料无味且最容易学习。
许多人已经使用了3D打印服务，或者至少听说过附加技术。

每年，这项技术都变得越来越容易获得，而Gearbest也为此做出了贡献，使个人3D打印机的价格更加实惠：更便宜，并且在俄罗斯联邦有缺货（！）。

我将从一台好的3D打印机-Geeetech的E180开始 。 当前值为$ 219。



该副本由众筹资金开发，于2018年成功发行。 3D打印机配备大触摸屏，可通过智能手机和特殊应用程序访问的Wi-Fi模块，以及可进行离线打印的microSD读卡器。 要使用E180进行操作和控制，您不需要计算机-只需智能手机或平板电脑。 在保持中间结果的同时会暂停打印。 您可以更换灯丝或关闭打印机电源。 如果发生电源故障，可以从停止点继续。

打印机使用特殊的Geeetech 0.4毫米喷嘴进行打印。 打印工作区域为130x130x130毫米。 定位精度高达25微米，最小印刷层：50微米。 紧凑，简单的设计使您可以在工作中或办公室中使用它。 请勿伸出电线或打开高温部件！

预算的下一个竞争者是3D打印机IUM1 Portable ，其打印区域为110 x 110 x 120 mm。



当前值是$ 189。 打印机具有紧凑的设计和封闭的设计。 所有运动部件都在壳体内部。 该工作台配有特殊的可移动聚合物板，以增加部件的附着力，该板本身位于磁性基座上，可在打印后快速移除。 打印机具有一个microSD读卡器，无需计算机即可进行打印，还具有暂停打印（暂停）以更换灯丝的功能。 您可以分层打印颜色部分。 喷嘴0.4毫米，最小打印层100微米。 打印机紧凑，轻巧，封闭-适用于学校或办公室的家庭使用。

另一个有趣且便宜的选择是Easythreed的3D NANO打印机 。



当前价格$ 159.99和优惠券GBACERU0117C 。 这是具有封闭运动学的立方体打印机。 桌子完全静止不动，打印头“飞过”模型，对打印质量产生积极影响。 安装的喷嘴为0.4毫米，最小层为0.1毫米。 由于有趣的设计，打印区域分为两部分。 主要的是110x110x90毫米。

注意他的“老大哥”-EasyThreed ET4000 。 这是一台更大的立方体打印机。



237.99美元的当前价格已经贵了一点，但优点是由于导板数量的增加以及较大的触摸屏显示器，使得运动学上的刚性更高。 打印区域的大小为120x120x120毫米。 打印精度提高，可用的层大小从50微米开始。 喷嘴也为0.4毫米。 制造商为窗户提供了防护玻璃，如果儿童可以接触打印机，这将非常有用。

如果您有时间和需求，可以从“设计师”那里组装3D打印机。 最受欢迎的构建套件之一是Alfawise EX8 3D打印机套件 。 今天，它是家用3D打印的最经济的选择。



当前的成本为145.99美元，考虑到送货到门，实际上是以组件的价格计算的。 Alfawise EX8是针对此设计的打印机进行的最新改进。 沿Z轴安装了两个驱动器，改进了气流模型。 根据X的规定，笔架得到了加固。Alfawise EX8配备了直接进纸挤出机驱动器，使您可以将任何塑料“进给”到打印机：PLA，ABS，PETG，SBS，Flex等。 SD卡读卡器允许您离线打印。 有一个加热的桌子。 打印机具有最大的打印面积：220 x 220 x 240毫米。 最小印刷层：100微米。 喷嘴安装为0.4毫米，但可以在MK8热端下方安装任何喷嘴。

为了进行比较，我添加了一个价格昂贵的品牌打印机：Tronxy C5 。



这是带有铝制外壳的立方打印机，采用简单的黑色设计。 当前值是$ 330.99。 打印机还具有直接进纸（MK8），打印区域为210 x 210 x 210毫米。 可用材料：ABS，HIPS，尼龙，PLA，PVC，木材。 定位精度为12微米，最小层为50微米。 喷嘴0.4毫米。 提供模型吹塑。 坚固的金属外壳可提高打印精度，并减少操作过程中的寄生振动。
