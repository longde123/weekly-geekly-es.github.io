我如何通过Google Cloud Professional数据工程师认证考试

没有推荐的3年实践经验


期待着数据工程师课程的开始，我们希望与您分享一个非常有趣的故事的翻译，这可能对将来的工程师有用。 走吧


Google Hoody：穿着。 严重的工作表情：存在。 本文的视频版本在YouTube上的照片 。
注意事项 本文适用于2019年3月29日之前的Google Cloud Professional数据工程师认证考试。 这个日期之后发生了一些变化。 我将它们包括在“高级”部分中。

所以，您想要一个新的连帽衫，比如在我的封面上吗？ 或者您打算获得Google Cloud Professional Data Engineer证书，并且想知道如何做。

在过去的几个月中，我参加了有关使用Google Cloud的课程，为专业数据工程师准备考试。 然后我试图通过它并通过它。 几周后，我的连帽衫发货了。 证书来得更快。

本文将列出您可能需要学习的一些知识，以及为获得Google Cloud Professional Data Engineer证书而采取的步骤。

您为什么要获得Google Cloud Professional数据工程师的资格？


数据无处不在。 并且需要有关如何创建可以处理和使用数据的系统的知识。 Google Cloud提供了用于构建这些系统的基础架构。

您可能已经具备使用Google Cloud的技能，但是如何向未来的雇主或客户证明这一点？ 有两种方法：项目组合或认证。

该证书告诉潜在的客户和雇主：“我有技能，并努力获得认证。”

Google的简要说明进行了总结。

证明您具有设计和构建数据处理系统以及在Google Cloud Platform上创建机器学习模型的能力。

如果您还不具备这些技能，那么熟悉认证培训材料意味着您将学习有关如何在Google Cloud上创建世界一流的数据处理系统的所有信息。

谁想获得Google Cloud Professional数据工程师的资格？


您看到了数字。 云正在增长。 它已经在这里，不会去任何地方。 相信我，如果您还没有看到这些数字，那么云正在增长。

如果您已经是数据专家，数据工程师，数据分析师，机器学习工程师，或者正在寻找数据世界中的职业道路，那么Google Cloud Professional数据工程师认证适合您。

使用云计算的能力已成为任何面向数据的职位的要求。

您是否需要证书才能成为一名优秀的数据工程师/数据集/机器学习工程师？


不行

您仍然可以在没有证书的情况下使用Google Cloud来处理数据传输解决方案。

证书只是验证现有技能的一种方法。

多少钱？


考试费用为200美元。 如果失败，您将再次需要为新的尝试付费。

与预备课程和平台本身的使用相关的可能费用。

平台费用是使用Google Cloud服务的费用。 如果您是老练的用户，那么您已经意识到这一点。 如果不是这样，并且您只了解本文介绍的培训材料，则可以创建一个新的Google Cloud帐户，并达到Google在注册时提供的300美元的限制。

我们将在一秒钟内继续介绍该课程的费用。

认证有效期为多长时间？


2年 之后，您将需要再次参加考试。

而且由于Google Cloud每天都在发展，因此证书所需的内容可能会发生变化（我发现，到我开始撰写本文时，它已经发生了变化）。

您需要准备什么考试？


Google建议使用3年以上的行业经验，以及使用GCP进行专业认证的解决方案开发和管理超过1年的建议。

我没有上述任何内容。

从实力到6个月的相关经验。 为了弥补短缺，我使用了在线培训资源的组合。

我选了什么课程？


如果您和我一样，并且没有建议的要求，则可以参加以下一些课程来提高自己的技能。

以下课程是我用来准备认证的课程。 它们按完成顺序列出。

我已经指出了通过每门认证考试的成本，时间和有用性。



我用来训练考试的一些很棒的在线资源。 顺序： Cloud Guru ， Linux Academy和Coursera 。

Coursera在Google Cloud Platform上的课程工程数据

费用 ：每月49美元（7天免费试用后）
时间 ：1-2个月，每周10个小时以上
有用的 ：8/10

Coursera的Google Cloud Platform上的数据工程是与Google Cloud的合作。

它分为五个子课程，每个子课程每周大约需要10个小时的学习时间。

如果您不熟悉Google Cloud中的数据处理，则此专业会将您的级别从0提高到1。您将使用称为QwikLabs的迭代平台通过一系列实践练习。 在此之前，将举行Google Cloud从业者关于如何使用各种服务（例如Google BigQuery，Cloud Dataproc，Dataflow和Bigtable）的讲座。

在Google Cloud Platform上介绍Cloud Guru

费用 ：免费
时间 ：1周4-6小时
有用性 ：4/10

不要将公用事业得分低视为课程毫无价值的指标。 事实并非如此。 他获得较低分数的唯一原因是因为他不专注于认证专业数据工程师（顾名思义）。

在完成Coursera专业化课程后，我将本课程作为进修课程，因为我仅在少数几个专业用例中使用了Google Cloud。

如果您来自其他云服务提供商，或者以前从未使用过Google Cloud，则可能需要学习此课程。 这是对整个Google Cloud Platform的出色介绍。

Linux Academy的Google认证的专业数据工程师

费用 ：每月49美元（7天免费试用后）
时间 ：1-4周，每周4小时以上
有用性 ：10/10

在完成考试并考虑了我参加的课程之后，事实证明， 来自Linux Academy的Google Academy认证专业数据工程师是最有用的。

视频以及Data Dossier电子书 （课程附带的大量免费学习资源）和实践考试使该课程成为我使用过的最佳学习资源之一。

考试后，我什至在一些Slack笔记中为团队推荐了它作为参考。

松弛中的注释


  • 考试中的某些内容不是Linux Academy，Cloud Guru或Google Cloud Practice的考试（预期）
  • 1个问题，带有一个数据点图形，该数据点图形应将它们分组（例如，cos（X）或X²+Y²）
  • 必须了解Dataflow，Dataproc，Datastore，Bigtable，BigQuery，Pub / Sub之间的差异以及如何使用它们。
  • 考试研究的两个实际例子与实践课完全相同，尽管我在考试期间根本没有涉及这些研究（问题给予了足够的理解）。
  • 了解SQL查询的基本语法非常有用，尤其是对于BigQuery问题。
  • Linux Academy和GCP提供的实践考试在形式上与考试题非常相似，我将对它们进行多次处理，并使用它们来找出您的弱点。
  • 帮助Dataproc的一个小技巧：“ Dataproc鳄鱼和Hadoop计划点燃火并煮一头猪的蜂房 ” {鳄鱼Dataproc和大象Hadoop计划点燃火（ Spark-火花，火花引发-火），并且煮一群猪（ Hive ）的猪（ Pig ）}（Dataproc处理Hadoop，Spark，Hive和Pig）
  • “ 数据流是流动的光束 ” { 数据流是当前的光束}（数据流处理Apache Beam）
  • “ 世界各地的每个人都可以与优质的ACID洗涤过的 Spanner相关” { 世界各地的任何人都可以使用Spanner处理酸洗（ ACID ）}（Cloud Spanner是旨在通过以下方式提升云的数据库：从头开始，与ACID兼容并在全球范围内可用）
  • 知道关系数据库和非关系数据库的经典变体的名称（例如MongoDB，Cassandra）可能会派上用场。
  • 每个服务的IAM角色略有不同，但了解如何将用户与能够查看数据的人分开，而又不会导致无法设计工作流非常有用（例如，Dataflow Worker角色可以设计工作流但看不到数据）

现在可能就足够了。 公里因考试而异。 Linux Academy课程将提供80％的知识。

选配


考试已于3月29日更新。 本文介绍的材料仍然提供了良好的基础，但重要的是要注意一些更改。

Google Cloud Professional数据工程师考试（ 第1版 ）的各个部分

  1. 数据处理系统设计
  2. 创建和支持结构和数据库。
  3. 数据分析和机器学习连接
  4. 用于分析和优化的业务流程建模
  5. 可靠性保证
  6. 数据可视化和政策支持
  7. 安全性与合规性设计

Google Cloud Professional数据工程师考试（ 第2版 ）的各个部分

  1. 数据处理系统设计
  2. 数据处理系统的构建和运行
  3. 机器学习模型的操作化（大多数更改已在此处发生） [NEW]
  4. 质量保证解决方案

第2版​​将第1版的第1、2、4和6部分合并为第1部分和第2部分。还将第1版的第5部分和第7部分合并为第4部分。第2版的第3部分进行了扩展，以涵盖Google Cloud的所有新机器学习功能。

由于这些更改是最近发生的，因此许多培训材料没有机会进行更新。

但是，熟悉本文中的材料应该足以覆盖您所需要的70％。 我将其与您对以下问题的一些研究结合在一起（在第二版考试中介绍了这些问题）。

  • Google机器学习API（ML）
  • Google Cloud Machine Learning Engine
  • Google Cloud TPU （由Google专为ML培训开发的定制设备）
  • Google词汇表词汇

如您所见，最新的考试更新侧重于Google Cloud中的ML功能。
