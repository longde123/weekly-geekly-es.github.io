Pernahkah saya harus melepaskan prinsip-prinsip yang dijelaskan dalam hidup saya? Sayangnya, ya. Ya, tanpa kesenangan dan arahan langsung. Dan itu sebelum saya mengetahui tentang kode. Tetapi saya sudah mengatakan - dia dekat dengan saya dalam roh, jadi saya mengakui bahwa saya tersandung.

Akankah saya meyakinkan pembaca untuk mengikutinya? Saya tidak punya hak seperti itu. Apakah saya akan merekomendasikan orang untuk hidup seperti ini? Jelas ya.
