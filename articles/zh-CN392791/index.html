俄罗斯初创公司正准备以199美元的价格发布5.7“厚度为5.7mm的智能手机

一家俄罗斯初创公司将于5月份开始收集5.7毫米超薄保护壳U PRO智能手机，三星的FullHD屏幕，同一家韩国制造商的相机以及8核联发科技MT6753处理器的预订。考虑到送货上门的情况，小工具的价格为199美元，与俄罗斯各地服务中心的协议将使您不必担心所购买的小工具无法维修或退回。

让我们看看制造商的承诺。尽管到目前为止，这些只是文字-您将需要实时观看设备。



技术指标


显示： 5.2英寸FHD三星AMOLED，PPI = 424
处理器：联发科MT6753八核Cortex – A53，1.3GHz
内存： 2GB + 16GB，LPDDR3
系统： Android 5.1（Lollilop）
尺寸： 148 * 72.4 * 5.7 mm
主摄像头： 13MP ，F2.2、5个Largan镜头，LED闪光灯
前置摄像头： 5MP，F2.4、4镜头
SIM卡：双SIM卡（micro-SIM + nano-SIM或microSD）
通信： GSM / WCDMA / FDD / TDD / LTE
WIFI： 802.11 a / b / g / n / ac（2.4G + 5G）
玻璃：前后康宁大猩猩玻璃3
电池： 3000 mAh
充电： 5V-2.5A快速充电（Quick Charge）

展示架


在U PRO智能手机的显示屏上，它们并不是特别省钱，而是采用了三星的FullHD Super AMOLED 5.2英寸对角线。分辨率-1920x1080，像素密度424 PPI，对比度-10001。





电瓶


智能手机将配备3000 mAh电池。该指标是平均水平，但厚度为5.7 mm似乎是可以接受的。快速充电功能可让您在10分钟内为手机充电4个小时，并在75分钟内为电池充满电。

在极端情况下，可以从手机中为另一个小工具充电。您不太可能能够以3000 mAh的电流进行散步，但足以应付来自其他智能手机的多个呼叫。

摄影机


三星再次制造了 主摄像头（带ISOCELL传感器的13百万像素），MidVCM马达和5个Largan镜头。光圈-F2.2。界面中内置了专业的拍摄模式，可以设置白平衡，光敏性，时间和曝光值。他们保证，由于夜间使用了ISOCELL传感器，因此可以拍摄出优质的照片。三星必须处理这个问题。闪烁-来自两个LED。根据开发人员，聚焦速度为0.3秒。 前置摄像头为5MP，带有OmniVision 1.4um传感器和四个广角镜。





铁


U PRO智能手机基于8核64位MediaTek MTK 6753 A53 1.3 GHz。图形为三核Mali T720 450 MHz。处理器本身专为中等价格段的上半部分而设计。

内存-2 GB DDR3。内置-16 GB。使用microSD扩展至128 GB。



案例设计


该表壳最有趣的是它的厚度为5.7毫米。例如，具有5.1英寸显示屏的三星Galaxy S5的厚度为8.1毫米，而iPhone 6S Plus的厚度为7.3 mm，尽管它具有5.5英寸的显示屏。

外壳由不锈钢和铝合金制成。该设计具有微孔结构形式的阳极氧化涂层。

在两侧，表壳均由康宁大猩猩玻璃3框起来。





该设备看起来很有趣，特性很好，价格为14,000卢布。仍然需要等待设备本身，并了解期望值与实际情况相符的程度。我一直希望俄罗斯人能重演小米现象。
