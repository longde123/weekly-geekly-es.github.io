Visual studio 2019

Baru-baru ini, di Microsoft Connect (); 2018 , kami mengumumkan rilis Visual Studio 2019 Pratinjau 1. Ini adalah versi Pratinjau pertama dari Visual Studio berikutnya. Di dalamnya, kami fokus pada beberapa bidang utama, seperti mempercepat startup dan bekerja dengan proyek-proyek yang tersimpan di repositori git, meningkatkan IntelliSense dengan kecerdasan buatan (AI) (Visual Studio IntelliCode), menyederhanakan kolaborasi dengan rekan-rekan melalui integrasi Live Share, dan banyak lagi. . Lebih detail di bawah cut!



Fokus pada pekerjaan


Segera, Anda akan melihat bahwa Visual Studio 2019 terbuka di jendela awal yang baru. Ini dirancang untuk bekerja lebih nyaman dengan repositori Git modern, apakah itu repositori lokal atau repositori Git online di GitHub, Azure Repos atau di tempat lain.



Tentu saja, Anda masih dapat membuka proyek yang sudah ada atau membuat yang baru. (Kebaruan juga akan segera tersedia di Visual Studio 2019 untuk Mac.)


VS19 untuk Windows dan untuk Mac

Juga hari ini, kami berbagi fitur pencarian baru di Visual Studio 2019 yang akan menggantikan bidang Quick Launch yang ada. Sekarang Anda dapat mencari opsi pengaturan, perintah dan instalasi. Pencarian baru juga lebih cerdas. Ini mendukung pencarian ekspresi yang mengandung kesalahan.



Visual Studio 2019 akan membantu Anda membuat kode lebih cepat. Dalam pratinjau ini, kami fokus pada kenyamanan mencapai konsistensi dan pemeliharaan kode. Kami telah menambahkan fitur refactoring baru seperti mengubah loop di kueri LINQ dan mengonversi tuple ke struktur yang dinamai untuk membuatnya lebih mudah untuk menjaga kode Anda dalam kondisi baik. Dengan indikator kesehatan dokumen baru dan fitur pembersihan kode, kini Anda dapat dengan mudah mengidentifikasi dan memperbaiki kesalahan dengan mengklik tombol.



Debugging rutin juga menjadi lebih mudah. Anda akan segera melihat bahwa kinerja ditingkatkan, dan ini membuat proses debugging lebih lancar. Kami juga menambahkan kemampuan pencarian ke Autos, Locals, dan Watch, membantu Anda melacak objek dan nilai. Dalam artikel mendatang, kami akan berbicara lebih banyak tentang pembaruan di area debugger, termasuk fitur "Time Travel Debugging", Debugger Snapshot baru untuk Layanan Azure Kubernetes (AKS) dan Set Skala Mesin Virtual (VMSS), serta kinerja yang lebih baik untuk men-debug proyek C ++ besar, berkat debugger 64-bit yang tidak dalam proses.



Membantu tim Anda bekerja bersama


Membangun berdasarkan pekerjaan yang kami mulai di Visual Studio 2017, kami meningkatkan Visual Studio IntelliCode , IntelliSense kami yang peka konteks dengan antarmuka AI, sehingga Anda dapat melatihnya dalam repositori kode Anda sendiri dan membagikan hasilnya dengan tim Anda. IntelliCode mengurangi jumlah klik yang diperlukan, menawarkan rekomendasi kontekstual untuk setiap kasus tertentu. Kami menulis lebih banyak tentang ini di sini .



Awal tahun ini, kami memperkenalkan Visual Studio Live Share , alat yang membantu Anda berinteraksi secara real time dengan siapa pun di seluruh dunia menggunakan Visual Studio atau Visual Studio Code. Live Share diinstal secara default di Visual Studio 2019, sehingga Anda dapat segera mengundang kolega Anda untuk bergabung dengan pengkodean untuk memperbaiki kesalahan atau membantu membuat perubahan dengan cepat.



Terakhir, di Visual Studio 2019, kami menyajikan cara yang benar-benar baru untuk menggunakan permintaan tarik (PR), yang memungkinkan Anda untuk memeriksa, menjalankan, dan bahkan men-debug permintaan tarik dari tim Anda tanpa meninggalkan IDE. Hari ini kami mendukung kode di Azure Repos, tetapi kami akan segera memperluas ke GitHub. Untuk memulai, Anda dapat mengunduh ekstensi Permintaan untuk Visual Studio dari Visual Studio Marketplace.



.NET Core 3 Pratinjau 1


Kami juga mengumumkan rilis .NET Core 3 Preview 1, dan Visual Studio 2019 akan menjadi produk untuk mendukung pembuatan aplikasi .NET Core 3 untuk platform apa pun. Tentu saja, kami juga terus mendukung dan meningkatkan pengembangan lintas platform di C ++, serta pengembangan aplikasi mobile .NET untuk iOS dan Android menggunakan Xamarin.




Orang-orang dari Komunitas Pengembang Microsoft @msdevru, saluran komunitas Pengembang Microsoft untuk pengembang dan semua orang yang tertarik dengan teknologi baru membantu kami menyiapkan artikel ini.
