Apple tv


O decodificador da Apple TV agora se parece com um console de jogos. O custo é de US $ 149 para a versão com 32 GB e 199 para 64 GB.

O Siri Remote com uma interface sensível ao toque, um acelerômetro e um sensor de giroscópio é conectado via Bluetooth 4.0, ou seja, não precisa ser direcionado para o console. Algumas funções se assemelham aos controles do Wii: nos jogos, o controle remoto pode ser acenado e os movimentos são gravados. O controle remoto funciona por até 3 meses com uma única carga. A pulseira Remote Loop, para que o controle remoto não voe fora de controle, é vendida separadamente.

O controle remoto recebeu um nome por um motivo. Assistente de voz SiriIntegrado profundamente: ela pode produzir filmes sob demanda, como “mostrar algo que você pode assistir com uma criança”, “mostrar filmes com um ator assim”, “mostrar algo novo”. A pesquisa é realizada em todos os principais recursos do vídeo familiares aos americanos sob demanda: Netflix, Hulu, Showtime, HBO e, é claro, iTunes.

O próprio console funciona com base no sistema operacional tvOS . Esta é uma versão do iOS, o Xcode também está aqui. Eles prometem que os desenvolvedores gostarão do sistema operacional. O tvOS já está disponível para desenvolvedores na versão beta, para os consumidores será lançado no final de outubro.

O console tem sua própria loja de aplicativos. Um grande foco está em aplicativos e conteúdo. Estas são aplicações informativas de competições esportivas, canais de TV e jogos. Nos jogos, pode haver um modo cooperativo. Qualquer proprietário de um iPhone ou iPod Touch pode ingressar como um segundo player e usar o dispositivo como um controlador. Há suporte para controladores de terceiros baseados em MFi.
