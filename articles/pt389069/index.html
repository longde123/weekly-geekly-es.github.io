O Remix OS, um sistema operacional com várias janelas baseado no Android, já está disponível para PC


Imagem da área de trabalho do Remix OS para PC (clicável) Os

desenvolvedores do Remix OS lançaram uma versão alfa deste sistema operacional para PC. Este sistema operacional compatível com Android foi originalmente projetado para o Remix Mini , uma unidade de sistema compacta. O sistema está adaptado para funcionar com mouse e teclado em vez de com uma tela sensível ao toque, suporta múltiplas janelas e barra de tarefas.

A campanha de crowdfunding para arrecadar fundos para a unidade do sistema Remix Mini não apenas aumentou com sucesso a quantidade necessária, mas também a excedeu muitas vezes : em outubro de 2015, a quantia total chegou a mais de US $ 1,6 milhão em vez dos US $ 50.000 que os criadores do Remix inicialmente pretendiam - startup chinesa Jide, fundada por um ex-funcionário do Google.



Uma imagem grátis de um sistema de 64 bits com 700 MB pode ser baixada do site do desenvolvedor. É proposto fazer o upload da imagem em um pen drive (com um volume de pelo menos 8 Gb) usando o programa incluído e, em seguida, você pode inicializá-lo. Os desenvolvedores recomendam o USB 3.0 com uma velocidade de gravação de pelo menos 20 Mb / s.

Em outubro do ano passado, soube-se que o Google planeja combinar gradualmente o Chrome OS com a próxima versão do Android. Portanto, o Remix OS para PC pode servir como um exemplo do resultado dessa combinação.
