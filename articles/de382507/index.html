Die neue universelle 2DIN Android-basierte Headunit für das Auto. Bauen Sie Ihren Traum

Hallo alle zusammen.

Derzeit gibt es eine große Anzahl von Android-Auto-Head-Units. Die meisten davon werden natürlich in China hergestellt. Trotzdem eignen sich viele von ihnen aus dem einen oder anderen Grund nicht für Endbenutzer. In diesem Zusammenhang entstand die Idee, ein neues Gerät zu schaffen, das die typischen Fehler bestehender Geräte beseitigt und die besten Eigenschaften maximiert.

Im Moment gibt es eine Vision für das zukünftige Gerät, die Designer haben die Schnittstelle fast ausgearbeitet, aber es gibt noch keine definitiven Lösungen für die Hardware. So sehen wir das neue Gerät.



Jemand wird sagen: "Sie haben das Nexus 7 in das Armaturenbrett gesteckt", und zum Teil werden sie Recht haben. Bei thematischen Ressourcen wird Nexus 7 häufig für Baugruppen dieses Typs verwendet (ich habe selbst N3 2013). Aber wir wollen über „nur ein Tablet im Auto“ hinausgehen und ein anständiges Produkt schaffen.
Aufgrund der Tatsache, dass eine der zentralen Rollen in unserem Leben jetzt ein Smartphone spielt, soll es so weit wie möglich in das zukünftige System integriert werden. Machen Sie im Wesentlichen einen Smart Server und die Headunit (im Folgenden als GU bezeichnet) zu einem Terminal. Es gibt eine ganze Reihe von Technologien und Protokollen zur Implementierung ähnlicher Funktionen: Android Auto, Apple CarPlay, MirrorLink, Miracast, MHL usw.
Wenn Sie am Schicksal dieses Geräts interessiert sind, wenn Sie an seiner Entwicklung teilnehmen möchten, Erfahrungen austauschen und interessante Lösungen vorschlagen möchten, um seine endgültige Form zu bestimmen - willkommen bei cat.

Um das Thema informativer zu gestalten, füge ich einige Informationen zu vorhandenen Geräten hinzu:
Fast alle auf dem Markt befindlichen Fahrzeuggeräte arbeiten mit dem 4.4-System. Es gibt Geräte mit 4.4.2 und 4.4.4, aber eine solche Minderheit. Von den mehr als 60 befragten Herstellern solcher Geräte planen 5, Android 5.0 innerhalb von 3 Monaten zu beherrschen, weitere 5 sind bis zum neuen Jahr möglich. Der Rest versammelt sich entweder nicht oder spricht überhaupt kein Englisch.
Das einzige Büro, das Geräte mit 4.4.4 an Bord mit einem Upgrade auf 5.0 veröffentlicht, ist NewsMy
Der Grund dafür ist einfach: Dies ist das einzige Büro unter den gefundenen, das Android vom Hauptgerät entfernt hat. Das heißt, die Headunit ist fast ein Vollzeitradio mit einem Bildschirm und einer Reihe von Ausgängen + einer Android-Box im Handschuhfach mit einer Reihe zusätzlicher Ausgänge für USB-Modems, USB-OTG, HDMI usw., die Sie aus dem Auto ziehen, mitnehmen und anschließen können zum Fernseher. Daher ist die Version von Android dort frischer.

Die überwiegende Mehrheit der Hersteller verwendet eine Reihe von MCU + SoM. Das heißt, auf der Hauptplatine befinden sich alle Ein- / Ausgänge für die Haupthardware, BT-, DAC-, Radio- und TV-Tuner usw. Chips auf einer separaten Platine (SoM - System on Module), die Android selbst dreht.
Darüber hinaus hat Android häufig keinen direkten Zugriff auf Eisen, was einige Probleme und Einschränkungen verursacht. Bei einfachen Modellen unterstützt Bluetooth beispielsweise eine begrenzte Anzahl von Protokollen, z. B. das Freisprechprofil (HFP) und das erweiterte Audioverteilungsprofil (A2DP). Das heißt, Sie können Musik und Sprache steuern. Dadurch werden Kontakte synchronisiert, aber das Anschließen einer drahtlosen Maus, Tastatur oder eines OBD-II-Scanners funktioniert nicht, da Android hat direkt keinen Zugriff auf den Chip. Teurere Modelle verfügen über Dual-Bluetooth. Mit dem zweiten können Sie nur zusätzliche Geräte anschließen.
