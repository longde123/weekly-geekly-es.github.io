回顾华硕ZenPad C 7.0平板电脑



最近，我们对ASUS ZenPad 8.0平板电脑进行了评估，该平板电脑属于“小配件”类别，即专门用于补充机主外观的设备。好吧，或者至少不与他发生冲突。这次，本次评测使用的是具有相同意识形态的平板电脑-ZenPad C 7.0（索引Z170 CG）。让我们看看这个小工具可以为我们提供什么，包括与ZenPad 8.0相比。

出现




ZenPad C 7.0的设计与以前评估过的ZenPad 8.0大致相似。防护玻璃几乎完全覆盖了设备的整个正面，并且狭窄的框架沿周边穿过。只有“七个”不是银色，而是黑色。扬声器位于非常靠近框架的边缘。



七种颜色也有四种颜色之一：黑色，白色，红色或金属色Aurora。



塑料外壳的表面质感令人愉悦。除了美观功能外，此解决方案还具有纯粹的实用价值：平板电脑更容易握在手中，不会滑落，并且没有手印，就像光滑的塑料和铝通常如此。

在厚度上，8.0和7.0几乎相同，但是一个纯粹主观的较小模型看起来更薄。可能是由于末端区域更平缓的过渡。



在右侧（如果您将设备上下颠倒，它是顶部）是音量和电源按钮。

如果ZenPad 8.0需要卸下整个后盖才能访问SIM卡插槽和MicroSD，则“七个”只需松开插头即可。



电源连接器和麦克风位于底端（左）。

显示


在此，使用基于IPS矩阵的分辨率为1024 x 600（160 dpi）的显示器。今天，它是适度的，但仅在纸上：图像很平滑，尽管可以看到像素，但是为此，您需要将自己埋在屏幕上并仔细寻找它们。没有颜色声明和视角。防护玻璃具有疏油涂层，可以很好地保留指纹。显示器的唯一缺点可能是低的最大亮度。但这可能是为了节省电池电量。

相机


可以说，“七个”中的主摄像头可以提供技术质量的图片：很难想象到两百万像素的像素。矩阵的感光度也很低，因此从拍摄的角度来看，ZenPad C 7.0与ZenPad 8.0几乎没有什么不同。这是在一个用两个60瓦灯泡照明的小房间里拍摄的照片。第一张是在夜间模式下拍摄的，第二张是在HDR中拍摄的。

快照



在这种情况下，您只能建议不要参与昏暗的拍摄。该相机需要阳光充足且光线充足的房间。

无线连接


您可能已经猜到，相对于ZenPad 8.0，“七个”在每种意义上都是最年轻的模型。这在无线模块中也很明显：它不支持LTE网络中的操作，而3G中最多。但是，与G8不同的是，如果您突然感到有打电话的欲望，并且智能手机不在手边，这款平板电脑就更像手机一样便于握住。都是一样，当然是一把“铲子”，但至少是一个工兵，而不是一个完整的刺刀。

性能


对于计算和图形，这里是64位四核Intel Atom x3-C3230四核处理器。集成显卡-MaLi-450 MP4。RAM的大小为1 GB，有16 GB的数据存储空间，但其中大约11个可供用户使用，最大支持64 GB的MicroSD存储卡。

ZenPad C 7.0和8.0之间最明显的区别就是性能。这款数位板有些周到，不会“追赶”，会对每一次按键都立即做出反应。不用担心，您不必在屏幕上拨几次，启动命令后平板电脑也不会减速。您只是觉得有些懒惰。正如他们所说，了解生命的人并不着急。

工作时间


尽管实际上在此处电池的安装电量为13 Wh *，而不是ZenPad 8.0中的15.2，但工作时间却相当可比。较低的显示分辨率（以及因此的功耗）以及更适中的处理器可能起了作用。在网上冲浪时，平板电脑持续了将近8个小时，并且在游戏上的时间甚至更长-大约为4。

结论


华硕ZenPad C 7.0与8.0具有相同的用途-随时随地可见。而且由于尺寸更紧凑，它甚至更简单，因为“八个”不能放到每个口袋里，而“七个”则有更多的机会。如果不考虑显示器周围框架的不同颜色，则设备看起来几乎相同。因此在配件7.0中的作用并不逊色于哥哥。其余的，老实说，也是如此：他有足够的生产力来上网浏览音乐，视频。您无法在其上玩3D游戏，但平板电脑目标受众中的绝大多数人都不需要它。这款平板电脑看起来优雅，舒适，朴实，可放入任何甚至很小的袋子，没有任何问题。并且它完全履行了其作为辅助设备的作用。

建议零售价：8990卢布。
