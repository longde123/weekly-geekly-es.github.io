西伯利亚人起诉贝塞斯达“辐射4”案，指控其失去妻子和工作的游戏


照片：Kai Pfaffenbach /路透社

在创建游戏时，开发人员必须非常小心。否则，游戏玩家可能会开始因为对“游戏药物”的持续依赖而责备游戏的创造者。那就是西伯利亚的玩家所做的-他指责贝塞斯达这样一个事实，即由于习惯了《辐射4》，他失去了妻子和工作。玩家认为公司对他的所有麻烦负责，并要求贝塞斯达50万卢布。 Sibiryak声称他开始在《辐射4（Fallout 4）》中踢球，并没有怀疑比赛会如此精彩，因此他在完成所有工作之前无法将自己从通道中扯开。

同时，问题并没有出现在青少年中，而是出现在一个完全成人的人中，该人是克拉斯诺亚尔斯克的一名28岁居民。开始游戏后，玩家根本无法将自己从玩具上撕下来。他白天和黑夜都坐在PC屏幕上三个星期，几乎没有起床。结果，辐射恋人被解雇了，他的妻子也离开了他。游戏玩家停止与朋友聊天。

此外，西伯利亚人几乎不睡觉也不吃东西，因此，他的健康受到了损害。

“如果我知道这个游戏会让人上瘾，那么我就不会玩。玩家说，我要么不买，要么等待新年假期开始游戏。

克拉斯诺亚尔斯克（Krasnoyarsk）居民现在想从贝塞斯达游戏工作室（Bethesda Game Studios）和将《辐射4（Fallout 4）》本地化的工作室那里获得50万卢布的罚款。

在俄罗斯，这是首批可能成为试金石的诉讼之一。如果工作室被判有罪，其他游戏玩家将开始向法院申请（“辐射”中的玩家不再是必须的）。

在世界上，这已经发生了。因此，在2010年，夏威夷群岛的一名居民起诉了Lineage II的开发商。克雷格·斯莫特伍德（Craig Smallwood）要求付款，并声称如果他知道游戏是如此令人上瘾，他将永远不会开始比赛。出乎意料的是，法官同意了他的意见，迫使“线”的开发者NCsoft支付了原告在诉讼中六个月所累积的所有法律费用。

Fallout 4是Bethesda Game Studios生产的Bethesda Softworks视频游戏，是Fallout 3的续集。该游戏是该系列的第五部分。在俄罗斯和独联体，它已由SoftKlab本地化。现在可以在3D打印机上打印游戏中的任何对象。
