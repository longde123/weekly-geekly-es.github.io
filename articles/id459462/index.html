SEO untuk Yandex. Cari semua?

Artikel Habro mendorong saya untuk menulis catatan ini: SEO tidak berfungsi pada tahun 2019?

Dan jika sampai saat ini saya memiliki beberapa kecurigaan buruk tentang optimasi SEO di Yandex, tetapi saya menghubungkan semuanya dengan pengalaman saya yang kurang dalam promosi situs web. Namun, sekarang gambar tampaknya telah berkembang dan saya akhirnya matang untuk berbagi pengamatan singkat saya tentang optimasi mesin pencari di Runet.

Subjek promosi


Pada awal tahun lalu, putri saya membuat situs web untuk toko online kecil. Barang-barang buatan tangan dan dibuat sesuai pesanan secara individual untuk setiap pembeli. Dan situs ini hanya sebuah karya contoh kerja, umpan balik untuk menempatkan pesanan dan pembayaran kacau melalui transfer bank (saya tidak akan menulis tentang situs itu sendiri, ini bukan tentang itu).

Setelah itu, saya melakukan promosi SEO. Saya mendaftarkan situs di mesin pencari, mengacaukan analitik dan melihat hasil awal lalu lintas pencarian. Pada awalnya, seperti yang diharapkan, hasilnya hampir nol.

Hit langka pada kueri frekuensi rendah yang mengenai hasil pencarian murni karena kebetulan. Oleh karena itu, saya membeli permalink, memesan deskripsi normal untuk barang, tetapi hasilnya masih tidak jauh berbeda dari nol.

Kenalan dan perceraian dari Yandex Direct


Ketika mempelajari masalah optimisasi mesin pencari, saya menemukan ungkapan di suatu tempat bahwa "jika Anda tidak di Direct, maka Anda tidak di Runet." Oleh karena itu, untuk entah bagaimana mendorong calon pembeli ke situs, saya membeli iklan dari Yandex.Direct, dan, tepuk tangan !, Lalu lintas telah berlalu. Dan tidak hanya lalu lintas berbayar, tetapi juga lalu lintas organik dari kueri penelusuran pergi.

Ini adalah panggilan pertama, tetapi saya tidak mementingkan fakta ini. Bagaimanapun, situs saya baru dengan peringkat nol, dan TiTs belum diperbarui untuk siapa pun untuk waktu yang lama (dan beberapa saat kemudian, Yandex sepenuhnya beralih ke ICS).

Tetapi karena iklan di Yandex.Direct tidak membayar untuk dirinya sendiri, saya mengikatnya dan mulai bereksperimen dengan saluran promosi lainnya. Detail tentang saluran iklan lain tidak penting saat ini, tetapi tiba-tiba saya melihat fakta aneh - lalu lintas dari pencarian organik di Yandex turun tajam begitu saya kehabisan perusahaan periklanan di Yandex.Direct.

Dan sekarang saya sudah memperhatikan ini!

Apakah uang adalah segalanya?


Apakah hasil pencarian organik yang terhubung dengan keberadaan perusahaan iklan berbayar di Yandex.Direct? Artinya, dengan tidak adanya perusahaan periklanan di Yandex.Direct, apakah situs mendapatkan nilai minus untuk hasil pencarian organik?

Tentu saja, saya bisa saja salah, tetapi mengapa lalu lintas organik dari Rusia dari pencarian Google jauh lebih banyak daripada dari pencarian Yandex, ketika tidak ada iklan berbayar di Yandex.Direct?

Awalnya saya ingin merumuskan kesimpulan, tetapi setelah beberapa pemikiran, saya memutuskan untuk tidak melakukan ini. Saya bukan Nostradamus dan saya tidak berpura-pura menjadi peramal, dan saya tidak menawarkan layanan promosi situs web, jadi buat kesimpulan sendiri ;-)

Saya berharap pengamatan saya akan bermanfaat bagi seseorang.

Terima kasih atas perhatian anda!
