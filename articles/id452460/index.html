Memperkenalkan Terminal Windows

Kami sangat senang mengumumkan Terminal Windows! Windows Terminal adalah aplikasi terminal baru, modern, cepat, efisien, kuat, dan produktif untuk pengguna alat-alat dan cangkang-perintah seperti Command Prompt, PowerShell, dan WSL.



Terminal Windows akan dikirimkan melalui Microsoft Store di Windows 10 dan akan diperbarui secara berkala, memastikan Anda selalu terbarui dan dapat menikmati fitur-fitur terbaru dan peningkatan terbaru dengan upaya minimum.



Fitur utama Terminal Windows


Banyak tab


Anda sudah bertanya dan kami sudah mendengarkan! Fitur yang paling sering diminta untuk Terminal adalah dukungan banyak tab dan kami sangat senang AKHIRNYA dapat memberikan fitur utama ini. Anda sekarang dapat membuka sejumlah tab, masing-masing terhubung ke shell baris-perintah atau aplikasi pilihan Anda, misalnya Command Prompt, PowerShell, Ubuntu di WSL, Raspberry Pi melalui SSH, dll.



Teks yang indah


Terminal Windows menggunakan mesin rendering teks berbasis DirectWrite / DirectX yang dipercepat GPU. Mesin rendering teks baru ini akan menampilkan karakter teks, mesin terbang, dan simbol yang ada dalam font pada PC Anda, termasuk CJK ideogram, emoji, simbol powerline, ikon, ligatur pemrograman, dll. Mesin ini juga membuat teks lebih cepat daripada mesin GDI Konsol sebelumnya!



Anda juga akan memiliki opsi untuk menggunakan font baru kami! Kami ingin membuat font yang menyenangkan, baru, dan monospace untuk meningkatkan tampilan dan nuansa modern Terminal. Font ini tidak hanya mencakup ligatur pemrograman, tetapi juga akan bersumber terbuka dan memiliki repositori sendiri. Tetap disini untuk informasi lebih lanjut tentang proyek font baru!



Pengaturan dan konfigurasi


Kami telah terhubung dengan begitu banyak pengguna baris perintah yang CINTA untuk menyesuaikan terminal dan aplikasi baris perintah mereka. Windows Terminal menyediakan banyak opsi pengaturan dan konfigurasi yang memberi Anda kendali besar atas penampilan Terminal dan setiap kerang / profil yang dapat Anda buka sebagai tab baru. Pengaturan disimpan dalam file teks terstruktur sehingga memudahkan pengguna dan / atau alat untuk mengkonfigurasi.


Menggunakan mekanisme konfigurasi Terminal, Anda akan dapat membuat beberapa "profil" untuk setiap shell / aplikasi / alat yang ingin Anda gunakan, apakah itu PowerShell, Command Prompt, Ubuntu, atau bahkan koneksi SSH ke perangkat Azure atau IoT. Profil ini dapat memiliki kombinasi gaya dan ukuran font, tema warna, tingkat keburaman latar belakang / transparansi, dll. Anda sekarang dapat membuat Terminal gaya khusus Anda sendiri yang disesuaikan dengan selera unik Anda!


Lagi!


Setelah kami mengirimkan Windows Terminal 1.0, kami berencana untuk memulai banyak fitur yang sudah ada dalam jaminan kami, di samping banyak fitur yang mungkin ditambahkan oleh komunitas!


Kapan saya bisa mendapatkannya?


Mulai hari ini, Terminal Windows dan Konsol Windows telah dibuat open source dan Anda dapat mengkloning, membangun, menjalankan, dan menguji kode dari repositori di GitHub: https://github.com/Microsoft/Terminal


Musim panas ini pada tahun 2019, preview Terminal Windows akan dirilis ke Microsoft Store untuk pengadopsi awal untuk menggunakan dan memberikan umpan balik.


Musim dingin ini pada tahun 2019, tujuan kami adalah meluncurkan Terminal Windows 1.0 dan kami akan bekerja dengan komunitas untuk memastikan siap sebelum kami rilis!

Tunggu ... apakah Anda mengatakan open source?


Ya kami lakukan! Kami sangat senang mengumumkan bahwa kami membuka sumber tidak hanya Terminal Windows, tetapi juga Konsol Windows yang menampung infrastruktur baris perintah di Windows dan menyediakan UX Konsol tradisional.


Kami tidak sabar untuk bekerja sama dengan Anda dalam meningkatkan dan meningkatkan pengalaman baris perintah Windows!


Ini kedengarannya luar biasa, tetapi mengapa Anda tidak bisa hanya meningkatkan Windows Console yang ada?


Tujuan utama Windows Console adalah untuk menjaga kompatibilitas dengan alat baris perintah, skrip, dll. Meskipun kami telah berhasil memperkenalkan banyak perbaikan utama pada fitur-fitur Konsol (misalnya menambahkan VT dan dukungan warna 24-bit, dll. Lihat posting blog ini ), kami tidak dapat memperkenalkan peningkatan berarti lebih lanjut pada UI Konsol tanpa â€œmenghancurkan dunia . "


Oleh karena itu, saatnya telah tiba untuk pendekatan baru yang segar.


Terminal Windows menginstal dan berjalan bersama aplikasi Konsol Windows in-box yang ada. Jika Anda menjalankan Cmd / PowerShell / dll. secara langsung, mereka akan mulai melekat pada instance Konsol tradisional dengan cara yang sama persis seperti yang mereka lakukan hari ini. Dengan cara ini, kompatibilitas ke belakang tetap utuh sambil memberi Anda pilihan untuk mengalami Terminal Windows jika / ketika Anda ingin melakukannya. Windows Console akan terus dikirimkan dalam Windows selama beberapa dekade mendatang untuk mendukung aplikasi / sistem lama yang ada.


Oke, tapi bagaimana dengan berkontribusi ke proyek open source terminal / aplikasi yang ada?


Kami dengan hati-hati mengeksplorasi opsi ini selama perencanaan dan menentukan keterlibatan kami dalam proyek yang ada akan membutuhkan perubahan persyaratan dan arsitektur proyek dengan cara yang terlalu mengganggu.


Sebagai gantinya, dengan membuat aplikasi terminal open-source baru, dan open-source Windows Console, sekarang kita dapat mengundang komunitas untuk berkolaborasi dengan kami dalam meningkatkan kode dan memanfaatkannya dalam proyek masing-masing.


Kami percaya ada banyak ruang di pasar untuk ide-ide baru / berbeda tentang apa yang dapat dan harus dilakukan terminal dan kami bertujuan untuk membantu ekosistem aplikasi terminal (dan terkait) berkembang dan tumbuh melalui pengenalan ide-ide baru, pendekatan menarik, dan inovasi menarik di ruang ini.
