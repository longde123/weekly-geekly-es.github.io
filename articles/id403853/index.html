Log dan basis data acara IoT. Mulai cepat dengan server iRidium

Cukup sering, Anda perlu melihat kapan dan apa yang terjadi dalam sistem otomasi. Untuk proyek-proyek serius, Anda perlu memiliki akses ke database peristiwa semacam itu, seringkali dalam waktu yang lama.



Kecelakaan terjadi, pintu terbuka, sensor bekerja. Temperatur turun menjadi -10, sehingga pipa pecah. Seseorang datang ke kantor pada malam hari - rekaman kamera hilang, tetapi di sini ada log sensor gerak. Secara umum, hal yang bermanfaat.

Berdasarkan iRidium, basis data semacam itu dibuat cukup sederhana, dan secara lokal, tanpa menggunakan cloud atau server SCADA. Cukup menambahkan tag protokol yang Anda butuhkan, dan kemudian menandainya untuk disimpan ke database.



Secara default, basis data disimpan di folder "_user directory_ \ Documents \ iRidium pro dokumen \ Server \ Database".

Pada saat yang sama, ada basis utama di mana Anda perlu memahami sedikit, tetapi ada basis kustom yang hanya mencatat data yang kami butuhkan dalam format yang diinginkan. Di sinilah letak basis data API , jika seseorang ingin memahami secara detail.

Saat yang menyenangkan: jika Anda menggunakan hingga 50 tag, lisensi server untuk integrator sepenuhnya gratis.
