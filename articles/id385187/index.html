Cloud rumah. Tinjauan Aplikasi Acer BYOC

Hanya orang-orang malas yang tidak menggunakan teknologi cloud sekarang - banyak dari Anda memiliki Dropbox, Google Drive atau Yandex.Disk. Awan membantu kami menyimpan data dan mengaksesnya dari mana saja di dunia. Tetapi jika Anda tidak mempercayai perusahaan ini dan tidak ingin file Anda disimpan di server mereka, ada jalan keluar lain - membuat cloud di rumah atau di kantor. Dan Acer menawarkan solusi yang cukup sederhana untuk ini dalam bentuk seperangkat aplikasi BYOC yang dapat bekerja dengan Windows, Android dan iOS. Baiklah, mari kita lihat bagaimana Anda dapat menyinkronkan semua foto dan dokumen Anda, dan jangan khawatir bahwa seseorang akan melihatnya atau mereka akan hilang sama sekali.



Saya perhatikan bahwa Acer BYOC (jangan disamakan dengan BIOS) ada dalam dua variasi: untuk bisnis dan untuk rumah. Perusahaan mana pun dapat membuat cloud sendiri berdasarkan aplikasi Acer. Di rumah, Anda hanya akan mendapatkan penyimpanan yang besar (dan volumenya hanya bergantung pada volume hard drive di komputer), yang dengannya semua perangkat akan disinkronkan.

Seperangkat program BYOC terdiri dari berbagai aplikasi: abPhoto, abMusic, abVideo, abFiles, abDocs untuk masing-masing bekerja dengan foto, musik, video, file dan dokumen. Ngomong-ngomong, sekilas, BYOC singkatan dari Build Your Own Cloud (Buat cloud Anda sendiri). Yang terpenting, platform ini mendukung berbagai OS, perangkat, dan jaringan, sehingga kemampuannya hampir tak ada habisnya.



Untuk memulai, Anda harus menginstal aplikasi di komputer dan di ponsel cerdas Anda. Omong-omong, aplikasi bekerja tidak hanya dengan perangkat Acer, tetapi juga dengan perangkat Android dan bahkan iOS, tetapi aplikasi desktop hanya tersedia untuk Windows sejauh ini. Setelah program diinstal di mana-mana, Anda harus mendaftarkan akun Acer atau masuk menggunakan Facebook atau Twitter.



Setelah itu, Anda dapat menggunakan cloud pribadi Anda seperti yang Anda inginkan. Dalam hal foto di abPhoto, hal pertama yang perlu Anda lakukan adalah mengatur cadangan sehingga semua gambar Anda dikirim ke komputer Anda sesegera mungkin.



Tentu saja, program disinkronkan di kedua arah. Jika Anda menambahkan foto di komputer Anda, itu dapat diunduh di ponsel cerdas Anda. Foto dapat dibagikan dengan teman, tetapi hanya yang Anda izinkan yang akan mendapatkan akses. Dengan pengaturan privasi, Acer serius.



Aplikasi musik abMusic dapat bekerja dengan trek yang ditemukannya dalam memori ponsel cerdas dan mengunduhnya ke cloud Anda. Ini juga akan bertindak sebagai pemain sehingga Anda tidak harus menjalankan program tambahan. Di iOS, omong-omong juga. Jadi jika Anda ingin menggunakan perpustakaan musik Anda dan tidak siap membayar untuk iTunes Match apa pun, gunakan solusi Acer.



Program ini bahkan memiliki fungsi StoreConnect, sehingga dapat dengan mudah mengambil semua file Anda dari iTunes. Cukup masukkan lagu favorit Anda ke folder yang diinginkan di komputer Anda lalu dengarkan dari ponsel Anda. Kecuali, tentu saja, memungkinkan kecepatan koneksi. Di sini, seperti halnya awan biasa, yang terbaik adalah menggunakan Wi-Fi, daripada jaringan seluler.



Jika Anda tidak ingin terus-menerus mengakses cloud, Acer BYOC memungkinkan Anda untuk mengunduh data ke telepon Anda dan menyimpannya secara lokal, tidak ada batasan. Ini berlaku untuk semua jenis file, apakah itu musik, foto, dokumen atau yang lainnya.



Ada lebih banyak aplikasi klasik di suite BYOC. Misalnya, abFiles. Sudah tidak ada tampilan pemain atau foto. Itu hanya file manager yang berfungsi dengan cloud. Untuk semua layanan cloud populer, satu program biasanya terlihat seperti itu. Di sini Anda dapat bekerja secara langsung dengan file dan folder. Unduh ke komputer, unduh kembali, salin, transfer, hapus. Secara umum, set fitur paling standar.



Dan aplikasi abDocs terlihat sangat berbeda. Ini adalah program kantor berbasis cloud yang memungkinkan Anda untuk menyinkronkan file, serta membuka dokumen, spreadsheet, dan presentasi. Pada prinsipnya, ini adalah perangkat lunak yang nyaman dan tanpa menggunakan cloud, Anda cukup membuat atau melihat file, tetapi ketika mereka juga diunduh ke komputer sendiri, ini sangat ideal. Dalam aplikasi, file Office dapat dikonversi ke PDF atau dikompresi menjadi arsip ZIP, kadang-kadang perlu dilakukan pada smartphone.



Salah satu manfaat dari Acer BYOC sangat dihargai ketika menggunakan abVideo untuk menonton film. Saya mengunggah film ke folder, dan kemudian dengan tenang menontonnya dari ponsel atau tablet Anda. Anda tidak perlu mengunduh apa pun, menghabiskan ruang, dan pada saat yang sama, Anda tidak bergantung pada kecepatan server selama menonton video online normal. Omong-omong, pada PC, satu program bertanggung jawab untuk musik dan video - abMedia.



Anda dapat menghubungkan beberapa perangkat (hingga 32 buah) ke penyimpanan cloud pribadi Anda. Data akan disinkronkan di semua gadget. Tetapi jika komputer dimatikan, maka sinkronisasi akan terjadi hanya ketika dihidupkan kembali. Acer BYOC menggunakan enkripsi AES untuk mentransfer data, sehingga penyerang harus bekerja keras untuk mendapatkan data Anda, karena akses hanya melalui akun.



Dalam kenyataan modern, ketika negara secara konstan membatasi penyimpanan data di luar negeri dan mengatakan bahwa semuanya harus disimpan di Rusia, tidak ada yang lebih dapat diandalkan untuk membuat cloud Anda sendiri di rumah. Selain itu, ini jauh dari satu-satunya plus. Anda sendiri dapat menyesuaikan ukuran penyimpanan, dan Anda hanya perlu membayar untuk hard drive, jika Anda memutuskan untuk meningkatkan kapasitas cloud Anda. Dan kemungkinan kehilangan file atau membiarkannya jatuh ke tangan yang salah jauh lebih kecil.

Pada saat yang sama, ab Apps hanyalah langkah pertama dalam menciptakan ekosistem cloud yang lengkap. Rencananya adalah untuk mengajarkan Internet of Things untuk berteman dengan cloud. Tetapi sementara ini hanya dalam rencana, di samping itu, akan ada persaingan serius dengan Google dan Amazon, yang telah maju jauh di bidang ini.
