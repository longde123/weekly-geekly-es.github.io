如何以及在哪里种植世界上最大的钻石

俄罗斯新钻石技术公司（NDT）的香港国际宝石学院（IGI）颁发了创纪录的重达10.02克拉，颜色E和纯度VS1的人造钻石证书。对于珠宝界来说，具有这种特征的宝石是相当琐碎的现象，但是从32克拉的合成钻石切割而成的人造石对合成钻石市场来说是一件大事，这是一次独特的事件。

简要回顾历史

在十八世纪末，科学家们做出了合成人造钻石的首次尝试，当时科学家得出了最后的结论，即碳是钻石的基础。自19世纪末以来，科学家们试图将廉价且负担得起的碳（煤或石墨）形式转化为坚硬而有光泽的钻石。许多人都发表了成功的声明，包括法国化学家亨利·莫桑（Henri Moissan）和英国物理学家威廉·克鲁克斯（William Crookes）等著名科学家。稍晚些时候，发现它们都不能提供有关结果的真实证据。 1954年，在通用电气的实验室获得了第一枚向世界展示的人造钻石。

有趣的是，在通用电气公司生产合成钻石的过程中，它们受到自然界本身使用的“技术”的指导。如专家所言，天然钻石是在约1300°C的温度和约50,000 atm的压力下形成的。在地球表面以下几百公里深处的行星幔厚度中。陨石，金伯利岩和其他火成岩将晶体带到地表。为了在实验室条件下模拟所描述的条件，GE专家使用挤压机挤压电池，并在其中放置用作催化剂和溶剂的石墨和铁-镍-钴熔体。

GE专家称他们的技术为HPHT（高压高温）。随着时间的流逝，正是她成为获得廉价工业钻石和钻石粉的基础。

当今钻石的生长方式如今

，合成钻石的工业生产主要基于两种技术之一，即上述的HPHT技术和CVD技术。较不常用的技术是外来技术，例如在爆炸中由石墨合成金刚石纳米晶体，或在超声空化作用下从有机溶剂中的石墨颗粒悬浮液生产微米级金刚石的实验方法。

高温高压

该技术被简化为在高温（约1500°C，具有所需的梯度）和高压（50-70000 atm。）下生长金刚石单晶的过程。水压机压缩一个特殊的容器，在该容器中有金属熔体（铁，镍，钴等）和石墨。在基质上放置一个或多个种子-小钻石晶体。电流流过腔室，将熔体加热到所需温度。在这些条件下，金属充当金刚石形式的种子上碳结晶的溶剂和催化剂。生长一个大或几个较小的晶体的过程平均持续12-13天。



今天，人造钻石和钻石粉的工业化生产每年达到数十亿克拉。在1970年代，他们还使用HPHT技术学习了制造重达1克拉的中等质量的珠宝石。

化学气相沉积

自1960年代以来，世界领先的实验室一直在开发另一种更便宜的金刚石合成技术-CVD（化学气相沉积，气相沉积）。在合成过程中，将钻石沉积在由碳氢化合物气体加热至600-700°C的基底上，然后使用微波辐射将其离子化或加热至高温。当沉积在硅或多晶金刚石上时，获得多晶晶片，其在电子和光学中的用途有限。生长速率为0.1至100μm/ h。板的厚度通常限制为2-3毫米，因此可以将切割下来的钻石用作珠宝，但是通常它们的大小不超过1克拉。



这种合成方法的功能在2000年代初期引起了小型初创公司和大型公司（例如戴比尔斯工业集团成员Element Element 6）的极大关注。



直到最近，HPHT方法的潜力仍被大大低估了。 Nikolai Khikhinashvili说：“几年前我们购买设备时，所有人都一致认为工业压机仅适用于合成金刚石粉末。”所有注意力和资源都集中在改进CVD方法上。同时，HPHT技术被认为是利基市场，因为绝大多数专家根本不相信可以将其用于生长足够大和高质量的晶体。然而，据尼古拉（Nikolai）称，无损检测专家能够提出自己的合成技术，这使他们能够获得这种质量和尺寸的钻石，到现在为止，只有通过使用天然晶体才能实现。关于切割技术，然后在实验室中生长，天然钻石的加工过程完全相同。





“当然，我们并不是唯一种植5-6克拉以上钻石的人，”尼古拉（Nikolai）表示。 -但是，今天绝对所有公司都遵循“三分之二”的原则：大型，高质量，具有商业可行性。我们公司率先掌握了以可承受的成本生产高品质钻石大晶体的技术。 32压机使我们每月可生长约3,000克拉，同时，从最纯净的IF到SI，从II型到II型，所有质量非常高的宝石都是D，E，F和纯色类别的钻石。我们的产品中有80％是重量在0.5至1.5克拉之间的珠宝钻石，尽管技术能力使我们能够种植任何尺寸的定制钻石。”为了证明所言，尼古拉向记者展示了一块10卢布硬币的水晶：“例如，这枚28克拉。切割后，我们得到了15克拉的钻石。”



自2000年代初以来，全球钻石专卖商De Beers一直对进入钻石珠宝市场的前景表示担忧，据管理层称，钻石珠宝市场可能会破坏其业务。但是，正如时间所表明的那样，人造钻石并不是天然宝石的竞争对手，因为相比之下，人造钻石在珠宝市场上所占的份额微不足道。此外，在改进生产技术的过程中，开发了研究方法，使人们可以自信而准确地识别合成钻石。合成的特征之一应被称为金属夹杂物，在有色钻石中则是易于定义的增长领域。另外，与紫外线中的天然钻石相比，使用HPHT，CVD技术获得的人造宝石具有不同的发光模式。



“消费者如何看待人造钻石？”尼古拉说：“这是非常有利的，特别是对于现代青年，对他们而言，这些钻石无冲突并且由使用高科技而又不干涉自然的人们创造出来的，这一点很重要。”好了，而且重要的是，这种石头的成本只有后者的一半左右。当然，证书应反映出宝石已生长，但很显然，带有此类钻石的戒指的所有者不需要携带证书！同时，在物理和化学特性方面，在无损检测实验室中生长的钻石与天然钻石完全相同，”专家得出结论。

金刚石和氮之间有什么共同点

根据氮含量，可以将钻石指定为两种主要类型之一。 I型钻石包含最多0.2％的氮，其原子位于（Ia）组或（Ib）组中的晶格位置。这种类型的钻石在天然钻石中占优势（98％）。通常，此类宝石并非无色。 IIa型钻石几乎不含氮（少于0.001％）。在天然矿物领域中，这种晶体非常罕见，仅占1.8％。几乎没有（占0.2％的情况）未发现掺有硼（IIb）的无氮钻石。晶格节点中的硼原子决定了它们的电导率，并赋予钻石浅蓝色。

工业生产中的钻石

在公司预算收入中，绝大部分仍由珠宝钻石提供。同时，新兴趋势表明，在未来几年中，对用于特殊光学，微电子和其他高科技工业领域的人造金刚石和金刚石板的需求将逐渐增加。

“到目前为止，工业区占我们产量的20％，但我们计划在三年内将其增加到50％，特别是因为需求正在迅速增长。现在，我们主要生产4 x 4和5 x 5 mm的印版，按顺序切割几张7 x 7和8 x 8 mm甚至10 x 10 mm的纸，但是现在谈论批量生产还为时过早。 Nikolai Khikhinashvili说，我们的下一个目标是向英制金刚石镶片制造过渡。这是“黄金尺寸”和必要的最小值，这在大众电子和光学行业中非常需要。为了得到这样的盘子，您需要种植重达一百克拉的钻石晶体。”他们计划在今年年底之前向NDT提供此类板的首批“原型”。
