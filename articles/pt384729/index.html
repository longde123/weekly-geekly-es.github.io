Kibbi - o sistema de segurança universal para o lar


Kibbi com sensores A

Indiegogo lançou outro projeto interessante - um sistema de segurança avançado com recursos que excedem em muito os recursos da maioria dos sistemas similares. Este sistema pode ser usado para proteger seu apartamento e trabalhar em uma casa de campo ou escritório. O Kibbi consiste em duas partes: um hub central com um grande número de sensores e uma câmera e sensores HD de grande angular.

Existem muitos sensores, e eles são projetados para consertar a abertura / fechamento de portas e janelas, o movimento de vários objetos (como uma TV) no espaço. Os sensores podem monitorar vibração, temperatura e algumas outras características ambientais. Os sensores Kibbi não têm medo de água, portanto podem ser usados ​​em ambientes internos e externos.



Como tudo isso funciona?


Em geral, tudo é padrão aqui. Se os dados transmitidos do sensor para o hub principal forem um pouco diferentes dos padrão, o sistema emitirá um sinal de aviso, que é enviado ao dispositivo móvel do usuário.

Ao mesmo tempo, a câmera do sistema começa a funcionar imediatamente, a fim de dar ao usuário uma idéia do que está acontecendo. O vídeo é transmitido em tempo real, além de ser gravado na "nuvem", onde a qualquer momento os arquivos de vídeo podem ser visualizados novamente.



Se necessário, o usuário pode ligar a câmera do dispositivo de forma independente, se você quiser ver o que está acontecendo em casa. No serviço de nuvem, você pode salvar até um gigabyte de dados, o suficiente para salvar um clipe suficientemente longo. Se houver essa necessidade, o Kibbi pode salvar o vídeo em uma unidade local (por exemplo, uma unidade externa).

É fácil ativar e desativar o Kibbi, enquanto os desenvolvedores fornecem um modo offline. Para fazer isso, um aplicativo especial é instalado no smartphone, que rastreia a localização do usuário. Se o hub Kibbi detectar a remoção do smartphone com o aplicativo, o modo de segurança será iniciado automaticamente. Se o hub detectar a proximidade do smartphone com o aplicativo, o sistema será desligado.

Sob o teto da minha casa : esse modo foi projetado para rastrear o número de pessoas na casa ou no apartamento do usuário. Se necessário, Kibbi notificará quantas pessoas deixaram a casa e quem entrou.



Comunicação bidirecional : o Kibbi também pode ser usado como um sistema para teleconferências. O dispositivo está equipado com uma câmera de alta qualidade e um excelente sistema de alto-falantes.

Serviço IFTTT


O Kibbi é compatível, entre outras coisas, com o serviço IFTTT (
If This Then That). Este serviço abre grandes oportunidades para conectar vários eventos e criar uma cadeia de causa-efeito.



Por exemplo, se o sensor Kibbi detectar movimento, a lâmpada inteligente acenderá automaticamente. Ou se a temperatura ambiente cair abaixo de um determinado nível, o elemento de aquecimento é ligado.

Especificações


  • Câmera Full HD com um amplo ângulo de visão - 130 graus;
  • Alto falante
  • Microfone;
  • Módulo Bluetooth 4.0;
  • Módulo Wi-Fi;
  • Bateria de reserva - o sistema pode funcionar mesmo quando a eletricidade é desligada em casa;
  • Suporte para sensores adicionais.


Conclusão


O sistema é realmente universal - e se algumas funções não funcionarem prontamente, você poderá configurar recursos adicionais. Uma grande vantagem é a capacidade de trabalhar com uma bateria de reserva - para ter certeza de que o Kibbi funcionará mesmo quando não houver luz na casa.
