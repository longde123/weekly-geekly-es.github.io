无毒伪善

作弊的程序员被教会了技术技能的重要性，而忘记了与他人进行机智的交流。 我们的竞争环境充满挑战和成就。 这引起了复杂和自大。 毫不奇怪，毒性是IT中的一个严重问题。

我对她一生中出现的频率感到有些厌倦。 很难发表意见，以免陷入另一种试图找出所说的话是否不礼貌的尝试。

简单规则


我第一次遇到这个概念是在青少年时代。 后端刷新了糟糕的API，而没有与应用程序团队讨论。 我讽刺地抱怨iOS团队缺乏确认。 后端导师立即向公司网络发送了有关我的匿名投诉。

膝盖颤抖，我参加了与人力资源部的会议，人力资源部的任务是在企业思想的梳理下梳理学生。 我记得两件事： 卡内基的书 ，稍后再说，和这句话： “我在书信中看不到这样的东西，但是你是一个聪明的人，并且你知道有人会认真对待它 。 ” 我不明白



这是复杂和变化的起点。 有一段时间我不敢在聊天室写东西，然后我开始尝试一下。 令人恐惧的是，我无法在陌生人和我的言语中都认识到同样的神秘毒性。 在发送之前，我大声重读了邮件，但没有任何帮助。 对于我对关节和咀嚼的真诚误解，他们回答说： “如果您自己不理解这一点，那么您的生活将很困难 。 ”

从那时起，我积累了一系列规则，这些规则有助于始终保持对自己沟通风格的正确性的信心：

  • 消息末尾的禁止点。 更好的表情符号或括号
  • 使用模因/贴纸/ GIF传达宽大的想法（同意/怀疑/欣赏）。 他们在对话者中建立了积极的基础。
  • 禁止未经稀释的批评。 首先对完成的工作表示问候和感激
  • 将命令式替换为以下问题： “更改” -> “您如何看待这种更改？”
  • 轻松地对事物感兴趣，祝你好运，不仅关于工作交流
  • 如果无法避免批评，请指定这仅是您的意见。 现在，介绍词“似乎”是您最好的朋友。 永远不要写给别人“没有/禁止/没有必要”
  • 禁止使用任何带有负面含义的词，而代之以以下问题： “无用” -> “您怎么看，这对我们为什么有用？” ， “不可读” -> “我们可以支持吗？”

当然，可以使用更狭窄的规则来补充此列表。 一切都会好起来的，但是……我不想！

我不想浪费时间去寻找和遵守不成文的通信标准，以白白取悦周围的人。 我不喜欢铃鼓跳舞，只是人们注意到复制品的本质，而不是看完复制品后的情感反应。

毕竟，我对同事应该不友好。 预期的最低标准是中立。 没办法 ，明白吗？ 而且，如果您设法唤起愉快的情绪，可以结交朋友-这对我来说是一个加分！

但是在我们这个时代，臭名昭著的地方并没有转移到积极的领域。 歪曲的无毒推定为： “除非有其他证明，否则你很粗鲁 。 ” 为什么会这样呢？

西方影响


斯拉夫人不是IT的先驱。 美国人已经访问过那里并建立了自己的规则。 当然，英语是最重要的编程语言。 但是没有人同意进入IT部门以将其文化转变为美国人。 毕竟，一个不需要学习英语的中国人就不需要喝牛奶茶吗？

我在山谷里住了5个月，我清楚地看到这种激进的礼节来自何方。 “你好吗？”是一个自动的问候语。 轻触袖子后，机器人“对不起” 。 毫无生气的“非常感谢您”在商店成功付款。
为什么每个程序员都应该接受这种通信标准？ 也许他是先天性强于俄罗斯的废弃地雷？ 我打赌你

您对遇到的每个人说的话毫无价值。 当我感到内时，我要道歉。 我很感谢别人！ 但是只有当他们做对我来说很重要的事情时，才是重要的。 我不想对我的同事们的工作表示感谢。 是的，我可以做到，但我不应该这样做。

这不是“基本的礼貌” 。 潜规则使我说出我不是要说的话。 我需要向对话者展示我没有的感觉。 看起来像不是我的男人。 这就是所谓的伪善 。

在异国土地上，您必须遵守房主制定的规则。 那仅仅是发展不是任何特定民族的领土，而是一个跨国社区。 在我们宽容的世纪中，我们可以回想一下，并不是每个人都有隐藏空话帷幕后面的短语真正含义的传统。

错误的观念是“做自己喜欢的事，像他们一样成功”，这种作用发挥了作用，美国公司的沟通方式也被俄罗斯公司所吸收。 外包业务害怕失去重要的客户，因此，按最后通order的顺序，向员工推了一条备忘录“如何不惹怒西方同事” 。 举行培训课程，沟通不正确是从项目转移到低绩效评估的原因 。

神化


在某人看来，我简直无法与打的猫头鹰交流柔韧的猫头鹰。 也许是这样，但是我有朋友。 我喜欢那些对我很可爱的人，与同事一起喝啤酒，并以他们开玩笑的笑容超越允许的范围。 我认识到沟通的重要性。 毕竟，是她创造了产品，使企业成功。 但是，我为尽自己最大的努力去不成为自己的身份并与同一个人打交道而感到厌倦。

工作时间


最近发生了一个故事，证明了一种礼貌的种植文化的神化 。
我发现同事A可以享受多次折扣。 他们还告诉我，有礼貌的要求没有错。 同事A友好地回答： “是的，我稍后再看:)” 。 第二天，同事B来到我身边，严厉指责我代表他写信A。 也就是说，同事A回答了我之后，就责骂同事B披露了信息。 自然，我什么也没回答，经常在厨房里微笑。

这种情况显示了伪善的无毒的主要问题。 人们不会改变内部。 他们只是尽职尽责地笑了笑，把屎拉在自己身上， 在背后胡扯 。 所有这些都造成了一种不健康的恐惧和不确定性气氛。 如果同事A简单地写了“不”而不是臭名昭著的“以后：)” ，我会在一小时内忘记。 但是结果将永远铭记在心，这将影响我在团队中的紧缩。

人们不会因为有人在附近打喷嚏而受到被动侵略的感染。 这是我们通过压制人格并强加“在任何情况下都微笑”不自然的社会规范而获得的。 蒙面的人遵守规则，但这不会减少冲突的数量。 相反，现在它们孤立地成熟，在自己的汁中煮沸。 健康气氛的盖子跳上沸腾的激情大锅。

不工作


对我来说，参与开源似乎是一件很酷的事情。 解决人们的需求，为每个人做有用的事情。 但是我只是被一种无毒的气氛扼杀了。 有问题的人对儿童，妻子的健康感兴趣，并认为这是必须遵守的准则 。

在回应客观反馈时，他到达： “我完全同意安东的观点，但所有这些都必须更加礼貌地说 。 ” 也就是说，感谢您的工作，对事物感兴趣，并且不要使用“没有任何意义”的表达。

有一种人对他们对礼貌的唯一理解充满信心。 这引起了无数的后果： “我实际上不在乎，只是经过而已，看来您是有毒的 。 ” 这样的评论自动消除了任何争议。 毕竟，现在您可以参考对手的毒性， 巧妙地贬低任何其他论点 。 粗鲁的污名燃烧着新的烙印，没有人会发现它是否真的如此。



观察明显的错误也几乎是有毒的。 在某个时候，我非常厌倦了关于“您错误地检查了工作并普遍延迟工作，这很失礼”这一话题，因此我决定眨眼不看。 较之于花一个小时思考一下消息，更容易进行修复。

顺便说一句，我确实读过“与人交流的秘密”。 我惊骇地回想起有关奉承，虚伪和通过口头trick俩进行操纵的技巧。 如果这确实是IT中推荐的行为方式，那么最好立即将其关闭。 我不想在这样的团队中工作。

主观性


礼节规则不成文的主要问题是无法检查它。 任何时候，任何人都可能会发现“需要工作”一词是有毒的，对此没有任何异议。

通常，话语的主观性证明了这一点。 而且， 在轨的茶壶也不应忘记。 但是人类的思维是灵活的，因此突然之间不需要证明毒性问题。 “你有毒，因为在我看来，这是有毒的”这句话是不争的证据。

我可以详细说明具体情况。 但是我不会因为一个简单的原因而去做：，子，这是主观的 。 任何信息都可以解释为有毒和无毒的，以赋予任何情感色彩。 在表情符号和感激之情中，您可以看到微妙的巨魔，在普通的问题骚扰中 。 但是，这样做不仅是没有必要的，而且是严格禁止的。

一个人不能被冒犯，一个人可以被冒犯。 没有无法原谅的诅咒清单，在任何情况下都是侮辱。 或者，也许您从不写“噢，您的bit子”与朋友聊天？

周围没有人知道哪个关联会生成随机的单词和表达式。 例如，我讨厌“不过”与工会联合提出的复杂建议，并认为它们傲慢自大。 你知道当有人给我写这个信时我会做什么吗？ 也许我正要告诉所有人有关令人恶心的毒性行为？ 不，我意识到自己有一种扭曲，我不再将蟑螂投射到其他人上，而是要注意所到达文本的本质 。

解决方案


如何根据每个人的生活经验来系统化每个人形成的规则？ 嗯，你知道吗？ 毕竟，程序员已经做到了。 现在，每个项目都有一个短毛猫。 关于“在我看来我需要在这里有两个空白行” ，“ 我认为这个缩进是错误的”主题的所有讨论都奇迹般地消失了。 毕竟，如果可以引入法规，每天争论和发明的目的是什么？

而且，毒性问题也已被人类解决。 Hagakure和其他代码渗透了历史。 现代版本是行为准则 。 但是在某个地方，我们走错了方向，因为部分毒性迹象仍然浮在空中，通过单击进行补充和更改。 不要那样做 只需将所有规则放在公司的Google码头中即可。 添加示例，不要在说明上略过。 我一直缺乏能够解决任何主观争议的基本文件。

只是不需要“政治是您的教育，妈妈必须学习” 。 对于一个出生在蒙古包里的男人来说，把门关在女孩面前的习惯并没有被母亲的牛奶所吸收。 一个夸张的例子，但是礼貌与人民的文化紧密地联系在一起，并且期望来自不同大陆的人们的思想中有100％的巧合是愚蠢的。

不要考虑对话者的思想和讽刺程度 。 阅读所写内容。 停止隐藏反感和吞下冲突。 一次诚实的交谈可以解决整个问题，并使态度相反。

停止路过，评估他人的沟通方式，建议您改正某些东西。 因为您很可能会分享自己的个人经验，而实验却不支持。 每个人都有这样的经历，而且很恐怖，没有一个是真实的。 这与建议您下注黑色相同，因为您玩了五次。

总是应该允许有毒性的临界案例（ “我无法表明规则有误，但我有所感触” ），有利于被告。 只是不采取任何行动，如果没有办法，您可以理性地解释对话者的粗鲁态度。

这甚至与激进的诚实并不接近，只是没有不必要风景的交流。 您需要修复的不是您周围的世界，而是您的感知。 只需停止让您的邻居预期您的心情即可。 开始提供没有金属丝的充分交流。 进行交流时，您不怕发送下一条消息，而不会用表情符号包围它。
