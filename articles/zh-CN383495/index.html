基金会律师已经发布了针对俄罗斯互联网用户的法律建议。备忘录“ 哈士奇，转发，推文：如何不违反反极端主义立法 ”包含一些技巧，这些技巧将有助于避免触犯法律并可能防范最危险的行为。

因此，互联网用户可以对任何形式的公开声明承担刑事或行政责任，包括对极客时间的讽刺性评论。

俄罗斯《刑法》和《行政犯罪法》中有几条通常被称为反极端主义立法。

刑法典


‣ 第280条。公开呼吁进行极端主义活动。
28 第280.1条。公众呼吁采取旨在破坏俄罗斯联邦领土完整的行动。
‣ 第282条。引起仇恨或仇恨，以及对人的尊严的侮辱。
28 第282.1条。组织一个极端主义社区。
28 第282.2条。极端主义组织活动的组织。
28 第282.3条。资助极端主义活动。


20. 第20.29条。生产和分配极端主义材料。
‣ 联邦法律“打击极端主义活动”。

2011年6月28日，俄罗斯联邦最高法院全体会议还裁定第11号“关于极端主义犯罪刑事案件的司法实践”。

属于“极端主义”定义的行为清单（该条规定了惩罚，包括对它们的上诉）。

1.宪政制度的基础发生剧烈变化，并违反了俄罗斯的完整。

2.恐怖主义和其他恐怖活动的公开辩护。

3.引起社会，种族，民族或宗教仇恨。

4. 宣传根据一个人的社会，种族，民族，宗教或语言背景或对宗教的态度而排他性，优越或自卑。

5.宣传和纳粹用具或象征，或用具或象征类似纳粹的用具或象征性的公开演示了混乱的地步。

6.公众呼吁采取这些行动或大规模散布明显的极端主义材料，以及为大规模分发而生产或储存这些材料。

7.组织和准备此类行动并煽动他们。

8.为此类行动或援助的组织，准备和实施提供资金。提供培训，印刷以及材料和技术基础，通讯类型或提供信息服务。

“反极端立法的规范是通过这种方式制定的，而其实践却使您常常无法事先预测自己的行为是否会被视为非法。另外，如果您用反讽的文字来写或发表评论，从而颠倒了文本的字面意思，那么执法机构就不太可能理解您的反讽，因此您可能要对出版的字面意义负责。

实用技巧


由于法院不了解您信息的讽刺和真实含义，因此您需要清楚详细地解释一切。公共判决基金会的律师提供了一些有用的技巧。

•如果您决定重新发布可疑的东西，或更确切地说是毫无疑问的东西，请在出版物的旁边发表评论，以表示对此材料的否定态度，或者自己撰写为什么要发表。

•如果您发布其他人的帖子，则可以视为发布您自己的帖子，因此，发布该帖子的责任可以相同。

• 警惕加入社交媒体团体。。社交网络中张贴有争议图片，呼吁等的团体可能包含极端主义社区的所有标志（稳定性，结社目的，组织者的存在，组织的稳定性，参与者行动的协调性）。在这里，刑法“极端主义社团组织”第282条第1款就在等待您（尽管名字，该文章不仅涵盖了组织本身，而且涵盖了这种社团的参与）。

• 不要进行可疑的捐赠。这里潜藏着艺术形式的“我的”。俄罗斯联邦《刑法》第282.3条“资助极端主义活动”。总是想好向谁汇款，为什么要汇款。

基金会的律师总结道：“记住，实施反极端主义立法的做法充斥着各种轶事，没有人可以避免令人不快的故事。” 互联网用户应始终牢记好几个好律师，或者更确切地说是熟悉反极端主义立法适用细节的律师。
