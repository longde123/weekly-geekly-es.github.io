Apa yang harus dibaca dan dilihat sebagai permulaan dalam Ilmu Data: buku, kamus, dan kursus

Pilihan sumber daya dalam matematika, statistik dan pemrograman untuk para pemula Date Scientists. Periksa bahan-bahannya jika Anda berencana untuk belajar kursus online. Jadi, Anda maju dari teman sekelas, dan pada saat yang sama memompa keterampilan yang berguna - untuk mempelajari materi tambahan sendiri.

Bahasa Inggris teknis


Sebagian besar materi dalam koleksi adalah dalam bahasa Inggris. Karena itu, pertama-tama, Anda perlu memahami kosakata teknis dan belajar memahami istilah yang rumit. Sumber daya ini akan membantu Anda menavigasi literatur teknis jika tingkat bahasa Inggris Anda menengah atau di bawah rata-rata.

  • Kamus Cambridge adalah kamus penjelasan yang akan membantu Anda memahami arti kata dan frasa.
  • Pilihan singkatan yang sering digunakan dalam Ilmu Data, dengan dekripsi.
  • Glosarium Ilmu Data dan Pembelajaran Mesin.

Pemrograman


Siswa di Data Scientist menulis kode Python. Untuk menulis kode selama pelatihan, cukup menguasai konsep dasar bahasa: operator, tipe data, variabel, loop, fungsi, kelas. Sumber daya berikut akan membantu Anda dengan cepat memahami dasar-dasar dan berlatih sendiri:

  • Tutorial Pemula Python3 tutorial tentang Udemy
  • Mempelajari Panduan Konsep Dasar Python: Dari Nol ke Pahlawan oleh FreeCodeCamp;
  • Teori dan praktik Google dari Kelas Python Google ;
  • Pilihan tugas dari Pythontutor dalam bahasa Rusia dan Inggris;
  • Tinjauan pustaka Pandas , yang akan kami gunakan untuk memproses dan menganalisis data dalam program.

Jika Anda ingin memahami Python secara lebih rinci dan di bawah bimbingan seorang mentor, Anda dapat mengikuti kursus Analisis Python untuk Data secara paralel.


Basis data


Untuk berpikir dalam konteks data, Anda perlu memahami bagaimana basis data relasional terstruktur dan bekerja. Untuk melakukan ini, cukup menguasai dasar-dasar SQL - untuk mengambil minggu ketiga kursus pada dasar-dasar analisis data untuk bisnis dari University of Colorado di Boulder. Anda dapat mempraktikkan pengetahuan dalam tugas-tugas berikut:

  • Latihan SQL
  • Latihan Query Rating Film SQL;
  • Latihan Query Rating Film-Rating SQL;
  • Latihan Query Jaringan Sosial SQL;
  • SQL Query-Jejaring Sosial Latihan Ekstra;
  • Latihan Modifikasi Peringkat Film SQL;
  • Latihan Modifikasi Jaringan Sosial SQL .

Pengetahuan mendalam tentang basis data dapat diperoleh pada kursus "SQL for analytics."

  • Pelajari atau pelajari dasar-dasar matematika, statistik, Python, dan SQL.
