Cómo deshabilitar completamente Windows Defender en Windows 10



Windows 10 tiene un antivirus integrado de Windows Defender ("Windows Defender") que protege su computadora y sus datos de programas no deseados: virus, spyware, ransomware y muchos otros tipos de malware y piratas informáticos.

Aunque la solución de seguridad integrada es suficiente para la mayoría de los usuarios, hay situaciones en las que es posible que no desee utilizar este programa. Por ejemplo, si está configurando un dispositivo que no se conectará en línea; si necesita completar una tarea bloqueada por este programa; Si necesita cumplir con los requisitos de la política de seguridad de su organización.

El único problema es que no podrá eliminar o deshabilitar por completo Windows Defender: este sistema está profundamente integrado en Windows 10. Sin embargo, existen varias soluciones alternativas para deshabilitar el antivirus, utilizando la política de grupo local, el registro o la configuración de Windows en la sección de Seguridad (temporalmente).

Cómo deshabilitar Windows Defender a través de la configuración de seguridad de Windows


Si necesita realizar una tarea específica y no necesita deshabilitar completamente Defender, puede hacerlo temporalmente. Para hacer esto, mediante la búsqueda en el botón "Inicio", busque la sección "Centro de seguridad de Windows Defender" y seleccione "Protección contra virus y amenazas".



Allí, vaya a la sección "Protección contra virus y otras amenazas" y haga clic en el interruptor "Protección en tiempo real".



Después de eso, el antivirus deshabilitará la protección de la computadora en tiempo real, lo que le permitirá instalar aplicaciones o realizar una tarea específica que no estaba disponible porque el antivirus bloqueó la acción necesaria.

Para habilitar nuevamente la protección en tiempo real, reinicie la computadora o realice todos los pasos de configuración nuevamente, pero encienda el interruptor en el último paso.

Esta solución no es permanente, pero es más adecuada para deshabilitar Windows 10 Antivirus para una tarea específica.

Cómo deshabilitar Windows Defender a través de políticas de grupo


En las versiones de Windows 10 Pro y Enterprise, puede usar el editor de políticas de grupo local, donde puede desactivar permanentemente Defender de la siguiente manera:

A través del botón "Inicio", ejecute el script ejecutable gpedit.msc. Se abre el editor de políticas. Vaya a la siguiente ruta: Configuración del equipo> Plantillas administrativas> Componentes de Windows> Antivirus de Windows Defender.



Haga doble clic para abrir la opción "Desactivar el programa antivirus de Windows Defender". Seleccione la configuración "Habilitado" para habilitar esta opción y, en consecuencia, deshabilite el Defensor.



Haga clic en "Aceptar" y reinicie la computadora.

Después de eso, el antivirus se desactivará permanentemente en su dispositivo. Pero notará que el ícono con el escudo permanecerá en la barra de tareas; debería ser así, porque este ícono pertenece a la aplicación de seguridad de Windows y no al antivirus en sí.

Si cambia de opinión, siempre puede volver a habilitar Defender repitiendo estos pasos y seleccionando "No configurado" en el último paso, después de lo cual deberá reiniciar la computadora nuevamente.

Cómo deshabilitar Windows Defender a través del registro


Si no tiene acceso al editor de políticas, o ha instalado Windows 10 Home, puede editar el registro de Windows, deshabilitando Defender.

Le recuerdo que editar el registro es arriesgado y que los errores en este asunto pueden causar daños irreparables a la copia de Windows instalada actualmente. Es mejor hacer una copia de seguridad del sistema antes de comenzar a editar.

Para deshabilitar completamente Defender a través del registro, inicie el programa regedit a través del botón "Inicio" y vaya a él a lo largo de la siguiente ruta:

HKEY_LOCAL_MACHINE \ SOFTWARE \ Policies \ Microsoft \ Windows Defender

Consejo: esta ruta se puede copiar y pegar en la barra de direcciones del editor del registro.



Luego, haga clic derecho en la clave (directorio) de Windows Defender, seleccione "Nuevo" y el valor DWORD (32 bits). Asigne un nombre a la nueva clave DisableAntiSpyware y presione Entrar. Luego haga doble clic para abrir el editor de claves y configúrelo en 1.



Haga clic en Aceptar y reinicie la computadora.

Después de eso, Windows Defender ya no protegerá su sistema. Si desea deshacer estos cambios, repita todos los pasos, pero al final elimine esta clave o configúrela en 0.

Recomendaciones


A pesar de la presencia de varios métodos para deshabilitar Windows Defender, no recomendamos usar una computadora sin ningún programa antivirus. Sin embargo, puede encontrar situaciones en las que deshabilitar esta función sería una mejor opción. Y si instala un programa antivirus de terceros, no necesita deshabilitar Defender manualmente, porque durante la instalación se apagará automáticamente.
