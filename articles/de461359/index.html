Was ist Geolokalisierung?


Laut Wikipedia

„Geolokalisierung ist die Identifizierung oder Bewertung des tatsächlichen geografischen Standorts eines Objekts, z. B. einer Radarquelle, eines Mobiltelefons, einer Basisstation oder eines mit dem Internet verbundenen Geräts. In seiner einfachsten Form umfasst die Geolokalisierung die Erzeugung eines Satzes geografischer Koordinaten und ist eng mit der Verwendung von Ortungssystemen verbunden. Ihre Nützlichkeit wird jedoch durch die Verwendung dieser Koordinaten zur Bestimmung eines signifikanten Standorts wie Straße, Postleitzahl verbessert. “

Mit anderen Worten, Geolocation ist Ihre physische Netzwerkadresse auf dem Planeten.

IPv4- und IPv6-Adressen

IP ist eine Abkürzung für Internet Protocol. Eine IP-Adresse ist ein eindeutig identifizierbares Etikett für jedes Gerät, das an ein Computernetzwerk angeschlossen ist und Verteilungen am Registrierungsort für die Kontinente (Registrare) aufweist. Oder einfach ausgedrückt: IP ist Ihre virtuelle Adresse.

Überall dort, wo Sie das Internet auf der Welt nutzen, wird Ihnen eine IP-Adresse zugewiesen.
IP-Adressen werden in eigenen Bereichen gebildet, die von regionalen Internet-Registraren für verschiedene Anbieter und Netzwerkbesitzer festgelegt werden und von einem autonomen System (ASN) reguliert werden:

Amerikanisches Register für Internetnummern (ARIN) - für Nordamerika;
RIPE Network Coordination Center (RIPE NCC) - für Europa, den Nahen Osten und Zentralasien;
Asia-Pacific Network Information Center (APNIC) - für Asien und den Pazifik;
Netzwerkinformationszentrum für Lateinamerika und die Karibik (LACNIC) - für Lateinamerika und die Karibik;
African Network Information Center (AfriNIC) - für Afrika und den Indischen Ozean.

Wenn Sie mit IP-Adressen arbeiten, werden zwei Arten von IP-Adressen angezeigt:
0.0.0.0 ist die IPv4-Adresse.
2001: db8: 85a3 :: 8a2e: 370: 7334 ist ein Beispiel für eine IPv6-Adresse.

Was ist eine API?

API ist die Abkürzung für Application Programming Interface.

Dies ist eine Reihe von Funktionen, mit denen Anwendungen miteinander kommunizieren können. Sie haben Websites gesehen, auf denen Sie sich über Facebook oder Twitter (oder eine andere externe Plattform) registrieren können. Diese Beziehung zwischen dieser Website und Facebook wird mithilfe der API erstellt.

Nachdem wir nun wissen, was Geolocation, IP und API sind, präsentiere ich Ihnen die IP-Geolocation-API.

Geolocation API in der Praxis

Die IP-Geolocation-API ist ein kleiner Code (API), mit dessen Hilfe der ungefähre physische Standort eines Benutzers anhand seiner virtuellen Adresse (IP) ermittelt werden kann. IP Geolocation ist eine Methode zum Ermitteln des ungefähren physischen Standorts eines mit dem Internet verbundenen Geräts über eine Adresse. Die IP Geolocation API ist die Verfügbarkeit dieser Methode als API.
