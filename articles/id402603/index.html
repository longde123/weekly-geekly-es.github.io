Sejarah satu penganalisa spektrum



Kadang-kadang saya bertanya-tanya mengapa saya sangat tertarik pada beberapa hal dan sama sekali tidak tertarik pada yang lain, karena minat adalah apa yang membantu menghabiskan waktu berhari-hari, tanpa merasa lelah, untuk terlibat dalam subjek yang menarik dan pada saat yang sama, jika memudar, sulit untuk memaksa diri Anda untuk melakukan sesuatu. Tidak jelas dari mana asalnya, di mana ia pergi, hanya jelas bahwa sementara dia ada di sana, Anda tidak dapat kehilangan waktu, Anda perlu melakukan apa yang menarik karena minat akan pergi cepat atau lambat, dan pengalaman akan tetap ada.

Jadi, kali ini, bekerja beberapa waktu lalu sebagai tukang listrik, saya tertarik pada pertanyaan - "bagaimana saya tahu jika konsol alarm berfungsi dan apa bedanya?"

Bagaimana kisah ini berakhir, baca di bawah potongan.

Waktu berlalu, saya berganti pekerjaan, tetapi minat dan keinginan untuk memahami tetap ada.
Sekali lagi, berjalan-jalan di Internet, saya menjumpai penjelajah rf dan menyadari bahwa inilah yang saya butuhkan, walaupun harganya membingungkan saya, tampaknya menjadi "tampilan", dan tidak terstandarisasi untuk daya, tetapi biaya $ 130.

Saya tidak suka pembelian seperti itu, lebih baik untuk menunda uang ini dan membeli analisa penuh di masa depan, dan mencoba untuk mengumpulkan ini sendiri, karena menilai dari foto dari Internet, itu dibangun hanya dengan satu keping si4313. Pada saat itu, saya sudah memiliki pengalaman dengan display dan MK, jadi saya memesan si4313 di Ali dan mulai membaca lembar data.

Setelah chip itu tiba, butuh sedikit waktu untuk memulainya. Tahap pertama dilewati, tahap berikutnya adalah deskripsi operasi perangkat dan pengembangan sirkuit, menurut TK tertulis. Itu sederhana, sehingga sirkuit itu cepat dirancang, papan ditarik dan dipasang. Dan di sini bagi saya saat yang paling sulit datang - ini adalah pengembangan antarmuka, meskipun jenis pengembangannya, perlu untuk memutuskan bagaimana mengatur elemen-elemen pada layar sehingga mereka terlihat cantik, dan juga memilih warna. Sebagai hasil dari banyak trial and error, antarmuka ini ternyata.



Setelah perangkat yang ditunjukkan dalam gambar siap, saya puas, beberapa tirai terbuka dan pengembangan berhenti. Biasanya, ketika saya melakukan sesuatu untuk diri saya sendiri, ini selalu terjadi, untuk sementara saya berkonsentrasi pada pengembangan sebanyak mungkin sampai saya mencapai hasil yang memuaskan saya dan melupakan proyek selama satu atau dua bulan.

Beberapa bulan kemudian, saya memutuskan bahwa akan tetap menyenangkan untuk memesan papan di pabrik dan mengemas semuanya dalam sebuah kotak. Kasus aluminium ditemukan pada Ali, papan digambar ulang untuk itu, kemudian kasus dan papan dipesan. Pengembangan kembali terhenti untuk sementara waktu sementara pesanan sedang berlangsung, yaitu selama satu bulan.

Ketika paket tiba, ternyata case tidak cocok untuk ketinggian, saya harus memutakhirkannya menggunakan sisipan yang dicetak pada printer 3D. Selama beberapa hari berikutnya, perangkat itu dirakit, minat dipenuhi sepenuhnya, dan pengembangan kembali ditinggalkan. Pada saat itu, ia memiliki karakteristik sebagai berikut:

  • rentang frekuensi 240 - 960 MHz
  • rentang minimum - 3.18MHz
  • rentang maksimum - 636MHz
  • bandwidth input filter disesuaikan - 10,6 kHz / 24,0 kHz / 56,2 kHz / 112,1 kHz / 225,1 kHz / 335,5 kHz / 420,2 kHz / 620,7 kHz
  • langkah pemindaian yang dapat disesuaikan - 10kHz / 50kHz / 100kHz / 500kHz / 1MHz / 1.5MHz / 2MHz
  • sensitivitas - -98 ... -118dBm
  • konsumsi saat ini - 170mA
  • menyimpan tangkapan layar ke kartu sd

Kontrol dilakukan menggunakan layar sentuh resistif 3,2 '' dan enkoder, meskipun papan memiliki tempat untuk tombol mekanis, tetapi mereka tidak cocok dengan case ini.













Setelah beberapa waktu, saya menyadari bahwa akan sangat mudah untuk dapat mengambil tangkapan layar, dan dalam beberapa minggu saya memasang kartu sd ke penganalisa, untungnya saya menyediakan kemungkinan menghubungkannya pada tahap pengkabelan PC. Ada beberapa pertanyaan sederhana namun penting, apa kesalahan dalam pembacaan kekuatan dan apakah itu bekerja di seluruh jajaran. Sebelum itu, saya menggunakannya untuk memeriksa key fobs dari alarm, yaitu pada frekuensi 315 dan 433 MHz, dan saya tidak tahu apakah itu berfungsi dengan benar di sisa rentang.

Untuk menjawab pertanyaan, generator RF dari dernuss telah dirakit, dijelaskan di sini dan diperiksa. Saya senang dengan hasilnya, penganalisa melihatnya di seluruh jajaran, tetapi saya tidak dapat menemukan kekuatannya, ternyata generator itu tidak terstandarisasi dalam daya, dan dalam pengaturannya Anda hanya dapat memilih pelemahan sinyal dari maksimum yang dapat diberikannya.

Ketika saya menerbitkan artikel terakhir tentang osiloskop, saya menulis engine9 dalam email pribadi dan menawarkan untuk bekerja pada desain antarmuka keledai, tetapi kemudian saya memiliki rencana lain dan saya menolak, mengatakan bahwa saya sedang merencanakan versi lain pada chip 303, jadi di sini kita akan mengerjakannya. Setelah beberapa waktu, kami menulis dan mulai mengerjakan keledai. Sekali dalam sebuah percakapan, saya menyebutkan apa yang dilakukan penganalisa dan ingin dia melihatnya dengan tampilan kreatifnya dan menyarankan apa yang bisa diperbaiki, karena saya jelas tidak menyukai beberapa aspek desain. Dia melihat dan mulai ... Secara umum, kami mulai bekerja pada desain analisa, itulah yang terjadi pada akhirnya.


Tentu saja, Anda tidak dapat menyebut perangkat ini sebagai penganalisis spektrum, lebih cenderung menjadi "meter tampilan" dan tidak akan pernah menjadi penganalisis, karena pengembang perangkat semacam itu hampir tidak dapat mempengaruhi pengoperasian chip si4313, yaitu, jika chip memudar dalam kisaran tertentu dan memberikan hasil yang salah , perbaiki ini tidak akan berfungsi. Untungnya, sejauh ini tidak ada bug yang diketahui, tetapi penguji itu salah))
