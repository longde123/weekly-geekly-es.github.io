Concurso de backend de desafío FunCode Java / Kotlin

FunCorp realiza un concurso para desarrolladores de back-end en el lenguaje Java / Kotlin con un premio de 550,000 rublos.

Cualquiera puede participar. Los autores de las mejores obras recibirán premios en efectivo y regalos de FunCorp, así como la oportunidad de unirse a nuestro equipo.



¿Qué hay que hacer?


Escriba una aplicación en Java y / o Kotlin, que con la máxima velocidad determinará el contenido de entretenimiento más popular (imágenes y videos cortos) y los devolverá como una cinta, acompañada de metadatos, ordenados por tiempo de publicación de más reciente a más antiguo.

¿Qué evaluaremos?


  • Solución arquitectónica
  • Calidad del código
  • Rendimiento
  • Consumo de recursos
  • Soluciones técnicas originales y audaces.

Las tareas aceptadas se aceptan hasta el 19 de enero a las 23:59.

Los resultados y los ganadores se anunciarán a más tardar el 17 de febrero.

1er lugar - 300,000 p.
2do lugar - 150,000 p.
3er lugar - 100,000 p.

Acerca de nosotros


FunCorp desarrolla entretenidos servicios y juegos de UGC.

El producto principal es iFunny, la segunda aplicación de entretenimiento más popular en los Estados Unidos (más de 50 millones de instalaciones).

Que estamos usando


  • Arquitectura de microservicios
  • Java / Kotlin
  • Dropwizard, Spring, Ratpack, Ktor
  • MongoDB y Redis para almacenamiento de datos.
  • Docker para desplegar
