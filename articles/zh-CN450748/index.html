ISTQB认证。 第1部分：存在还是不存在？


正如我们的最新研究表明：与经验和工作形式不同，教育和文凭对质量保证专家的工资水平几乎没有影响。 但这是真的吗？获得ISTQB证书的意义何在？ 为什么获得证书对雇主公司比对专家本人更有利？ ISTQB值得花时间和金钱来交付吗？ 我们将在有关ISTQB认证的文章的第一部分中尝试找到这些问题的答案。

什么是ISTQB，ISTQB认证级别，您是否需要它？


ISTQB是一个致力于软件测试开发的非营利组织，由8个国家/地区的代表成立：奥地利，丹麦，芬兰，德国，瑞典，瑞士，荷兰和英国。

ISTQB Tester Certification是一项程序，可让专家获得国际测试证书。

截至2018年12月， ISTQB进行了830,000+项考试，并颁发了605,000+份证书，在126个国家中得到认可。

听起来不错吧？ 但是，认证真的必要吗？ 拥有测试专家证书的好处是什么？它为他们提供了什么机会？

选择哪个ISTQB？


首先，我们将介绍针对测试专家的认证选项。 ISTQB根据矩阵提供3个级别的认证和每个级别的3个指导：


您需要了解的有关选择级别和方向的知识：


1. 核心方向 的基础（F） 级别是任何更高级别证书的基础。

2. 专家领域的 F级 -为他提供高度专业的认证：可用性，移动应用程序，性能，验收，基于模型的测试等。

3. 敏捷的 F级和高级（AD） 区域 -在过去两年中，对此类证书的需求增长了20％以上。

4. AD级 -由以下人员提供认证：
-测试经理；
-测试自动化工程师；
-测试分析；
-技术测试分析；
-安全测试。

5. Level Expert（EX） -涉及测试管理和改进测试过程方面的认证。

顺便说一下，在选择所需级别的认证级别时，请参阅ISTQB主站点上的信息，因为 提供者站点的描述不正确。


谈论好处


从质量保证专家的角度来看，认证是：


1.首先， 由测试领​​域的国际专家对资格和专业资格进行确认 ，从而为进入新的劳动力市场开辟了道路。 在国际上，该证书已在世界126个国家/地区得到认可-是远程工作的天堂或移动的先决条件。

2.在劳动力市场上的竞争力增强：尽管大多数雇主不需要申请人提供ISTQB认证，但约55％的测试经理表示，他们希望拥有100％的认证专家人员（研究ISTQB_Effectiveness_Survey_2016-17） 。

3.对明天充满信心。 该证书不能保证雇佣或自动升职时的最高薪水，但这是一种“防火”的金额，低于该金额您的工作将不会受到赞赏。

4.质量检查领域知识的扩展和系统化。 认证是质量检查专家建立和丰富测试知识的好方法。 如果您是一位经验丰富的测试人员，则可以通过国际标准和行业方法来更新和简化该主题领域的知识。

从公司的角度来看，认证是：


1.市场上的另一个竞争优势：拥有经过认证的专家的公司提供高质量的咨询和质量保证服务的可能性大大降低，这对新订单的声誉和流程产生了积极影响。

2.参加大型招标时的奖金：获得认证专家的资格为公司参与有关招标的竞争性选择提供了优势。

3.降低风险：该证书表明专家具有测试方法，可以降低质量较差的测试分析的风险，并可以通过优化测试方案的数量来提高测试速度。

4.在国际市场上为外国客户和外国软件提供软件测试服务的优势 。

5.通过对非认证专家进行指导和培训，使其达到测试领域公认的国际标准，从而提高公司内部的能力 。

对于公司，ISTQB提供了一些更有趣的奖励和指导：

1. ISTQB国际软件测试卓越奖


软件测试领域的国际奖项，以表彰其在软件质量，创新，研究和促进软件测试专业方面的杰出服务。

获奖者是测试和开发领域的专家，研究的作者，测试的新方法。

2.合作伙伴计划（ISTQB）

该计划旨在表彰致力于软件认证测试的组织。 该计划包括四个合作伙伴级别（银，金，白金和全球），而该组织的合作伙伴级别则取决于其获得的认证点数（资格网格）。

特点：


  1. 包含在ISTQB网站上的合作伙伴组织列表中。
  2. 在ISTQB会员全国委员会或考试提供者的网站上提及该组织。
  3. 与ISTQB相关的活动和会议的特权。
  4. 有权获得新的ISTQB Syllabi计划的Beta版，并有能力为准备工作做出贡献5。
  5. 独家ISTQB合作伙伴论坛的荣誉会员。
  6. 互认ISEB和ISTQB认证。

3.作为质量检查领域活动的组织者，您可以申请参加ISTQB会议网络。

反过来，ISTQB在官方网站上发布会议信息，参加会议网络的活动组织者可以享受以下折扣：

  • 参加 ISTQB证书的人;
  • 合作伙伴合作伙伴计划 。

4.在“ISTQ®学术研究纲要”学术研究合集中发表测试领域的研究

5.来自世界各地的最佳测试实践集。 ISTQB学术档案

测试专家如何看待ISTQB认证？


“质量实验室”专家的意见。


Angelica Pritula（ISTQB CTAL-TA认证），质量实验室公司领先的测试专家：

-是什么促使您获得此证书的？

-在国外，这是要在一家严肃的公司中担任测试员的必要条件。 然后我住在新西兰，当时我是一家为手术室生产麻醉控制系统的组织雇用的。 该系统已获得新西兰政府的批准，因此有强制性要求测试仪必须经过认证。 公司支付了我的两张证书。 我所需要的只是准备和通过。

-你准备得怎么样？

-我从官方网站下载了免费教科书，并准备了它们。 我为第一次普通考试准备了3天，为第二次高级考试准备了2周。

在这里我必须说，我的经历并不适合所有人，因为 我是受过教育的开发人员。 到那时，在进行测试之前，我已经开发软件两年了。 另外，我的英语水平几乎与说母语的人一样，因此对我来说，准备并通过英语考试不是问题。

-您个人在ISTQB认证中看到哪些优点和缺点？

-优势不可否认，申请工作时到处都需要此证书。 后来，在测试分析中使用高级证书成为了国民经济部以及微软子公司的工作准证。

这里的缺点只是价格高。 如果证书不是由公司支付的，则费用是有形的。 当我租房时，一本普通的要花300美元，而一本高级的要花450美元。

3.劳动力市场上正在发生的事情：申请工作时是否需要在测试领域进行认证？


作为基础，我们通过LinkedIn获得了空缺的公开数据，并分析了测试专家的认证要求与测试中空缺总数的比率。


来自LinkedIn劳动力市场分析的观察结果：


  1. 在大多数情况下，申请测试专家职位时不需要认证。
  2. 尽管证书是无限期授予的，但在空缺中还要求有时间限制才能获得证书（过去两年内获得ISTQB基金会认证的级别为佳）。
  3. 在专业的测试领域具有高资历的求职者必须拥有梦co以求的文件： 自我测试，测试分析，测试管理，高级质量检查。
  4. ISTQB 不是唯一的认证选项；允许等效项。

结论


认证可能是单个公司或政府项目的强制性要求。 在决定是否获得ISTQB证书时，您应该关注以下现实：

  1. 在选择测试专家职位的候选人时，决定性因素是经验和知识 ，而不是证书的可用性。 尽管在具有类似技能的情况下，将优先考虑获得认证的专家。
  2. 认证有助于职业发展（对于90％的管理人员来说，团队中有50-100％的认证测试员很重要），此外，在一些外国公司中，获得证书是增加薪水的一个机会 。
  3. 认证有助于增强您的自信心 。 它还可以帮助您培养从不同角度思考事物的能力；您可以成长为专家。
