Tesla Model X mulai dijual pada bulan September



Tesla, yang dipimpin oleh Elon Musk, mengumumkan (pdf) bahwa mobil listrik Model X baru akan tersedia pada bulan September. Pengiriman mobil listrik kepada pelanggan pertama akan dimulai pada akhir kuartal ketiga tahun ini, meskipun hanya sejumlah kecil Model X yang akan dikirim.Sekarang perusahaan sedang membangun sampel akhir untuk pengujian, memeriksa pengoperasian mobil dalam berbagai kondisi.

Perusahaan menciptakan Model X bersama dengan Model S, di sebuah perusahaan yang berbasis di Fremont, California. Menariknya, kedua model mobil ini berjalan di jalur yang sama. Baru-baru ini, menurut Mask, perusahaan telah mengumpulkan beberapa versi uji Model X. Pada saat yang sama, dimulainya produksi mobil listrik model ini ditunda beberapa kali. Sebelumnya, perusahaan berjanji untuk memulai pengiriman Model X pada akhir 2014. Sekarang perusahaan yakin bahwa peluncuran produksi Model X tidak akan ditunda, semuanya berjalan sesuai rencana.

Ini tidak berarti bahwa perusahaan tidak memiliki masalah. Jika timbul masalah dengan setidaknya satu pemasok, maka produksi Model X dapat dikurangi menjadi 800 kendaraan listrik per kuartal. Ya, dan produksi Model S akan dipengaruhi oleh masalah seperti itu - karena, sebagaimana telah disebutkan, satu baris digunakan untuk merakit kedua model.

Musk mengatakan Model X adalah mobil listrik yang cukup sulit untuk dirakit. Mungkin ini adalah mobil paling sulit untuk diproduksi di dunia. Namun demikian, perusahaan akan berusaha menyelesaikan semua pekerjaan tepat waktu dengan menyediakan kendaraan listrik baru kepada pelanggannya. Mobil listrik pertama kali diumumkan pada 2012. Tesla mulai menerima pesanan di muka pada tahun 2013, dan untuk melakukan pemesanan di muka, pembeli harus melakukan "setoran", pembayaran di muka, dalam jumlah $ 5.000.

Sudah diketahui bahwa mobil listrik akan dilengkapi dengan penggerak semua roda, di samping itu, pabrikan akan memasok pintu kupu-kupu, sistem yang digunakan oleh beberapa produsen supercar.

Dapat juga diasumsikan bahwa mobil akan dilengkapi dengan sistem penghindaran rintangan otomatis. Video yang relatif baru diposting di Web menunjukkan prototipe Tesla X, di mana, sejauh yang bisa dinilai, sistem seperti itu diuji. Tesla Model X menggunakan arsitektur dasar khas Model S, sehingga perusahaan dapat menggunakan satu baris untuk merakit Tesla X dan Tesla S.
