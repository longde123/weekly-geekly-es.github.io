Mathematik und das Spiel "Set"


Wer hier ein "Set" findet, bekommt von mir einen Schokoriegel.

Set ist ein brillantes Spiel, in dem wir vor ungefähr 5 Jahren geschnitten haben. Schreien, schreien, Kombinationen fotografieren.

Die Spielregeln besagen, dass es 1991 von der Genetikerin Marsha Falco erfunden wurde, die sich 1974 während einer Studie über Epilepsie bei deutschen Schäferhunden Notizen machte. Für diejenigen, deren Gehirn durch die Mathematik ausreichend erschöpft ist, besteht nach einiger Zeit der Verdacht, dass es hier einige Echos mit Planimetrie und geraden Linien durch die Punkte gibt. Erinnere dich an die Regeln



Seth ist ein Kartenspiel. Alle Karten haben vier Parameter, von denen jeder drei Werte annimmt (insgesamt 3 x 3 × 3 × 3 = 81 Karten).



Die Typen und Werte der Parameter sind wie folgt:

  • Abbildung :: = Ellipse | Raute | "Rotz"
  • Farbe :: = rot | grün | lila
  • füllen :: = weiß | gestreift | fest
  • Nummer :: = 1 | 2 | 3

Das Ziel des Spiels ist es, spezielle Kombinationen von drei Karten zu finden. Drei Karten werden als "Satz" bezeichnet, wenn für jedes der vier Attribute der Karte entweder alle gleich oder alle unterschiedlich sind.



Mit anderen Worten, wir können sagen, dass drei Karten keinen Satz bilden, wenn zwei Karten einen Parameterwert haben und die dritte unterschiedlich ist. Sie können sehen, dass es für zwei beliebige Karten immer eine dritte (und nur eine) gibt, mit der sie ein Satz sind.

Spielfortschritt: Der Anführer legt 12 Karten auf den Tisch. Wenn jemand ein Set findet, ruft er "Set!" und nimmt dann ruhig die Komponenten der Set-Karte auf. Wenn die ausgelegten Karten keinen Satz enthalten (vor allem scheint dies nicht der Fall zu sein), gibt der Host drei weitere Karten aus.

Die maximale Anzahl von Karten ohne Satz beträgt 20. Die Runde wird fortgesetzt, bis das Deck leer ist. Derjenige, der mehr Sätze gesammelt hat, gewinnt.

Mathematiker machten sich Sorgen und präsentierten eine Kombination von 20 Karten. Wer sich als Chuck Norris betrachtet, kann dieses Bild vergessen und versuchen, "Solitaire" ohne Set selbst zu sammeln.
