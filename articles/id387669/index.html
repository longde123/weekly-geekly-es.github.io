Microsoft membuka Chakra


Gaurav Seth berbicara tentang ChakraCore

Tidak, Microsoft tidak bekerja dengan "pengetahuan suci", apa pun itu. Pidato tentang yang lain - perusahaan berencana untuk menerbitkan elemen kunci dari mesin JavaScript Chakra di GitHub, di bawah lisensi MIT. Sebelumnya ini dikatakan secara informal, perusahaan membuat pernyataan resmi tentang ini pada 5 Desember di konferensi JSConf.

Nama asli Chakra, pengembang perusahaan telah memilih untuk teknologi JavaScript, awalnya dibuat untuk Internet Explorer 9. Mesin ini juga digunakan dalam versi baru Internet Explorer, plus browser Edge. Selain itu, mesin bekerja di layanan seperti Cortana, Outlook.com, Azure DocumentDB dan TypeScript.



Setelah kode mesin terbuka, Chakra akan digunakan secara luas oleh komunitas Open Source. Bagaimanapun, perwakilan dari Microsoft Corporation mengatakan demikian. Selain pengembang independen, raksasa seperti Intel, AMD dan NodeSource menunjukkan minat pada Chakra.

Mesin Chakra pertama kali diperkenalkan pada tahun 2008, dan sejak itu popularitasnya baru saja tumbuh - layanan cloud, IoT, dan sumber daya web yang besar bekerja dengan Chakra. Untuk pengembang, ChakraCore, mesin virtual JavaScript yang dapat diintegrasikan ke dalam berbagai produk dan aplikasi, akan dibuka.



ChakraCore akan mencakup JIT, penganalisis sintaks, penerjemah, dan banyak lagi. Microsoft berharap bahwa membuka kode produk akan membuatnya lebih universal di masa depan daripada sekarang.
