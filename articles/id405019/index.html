BlackBerry KEYone: tentang keyboard



Ya, BlackBerry KEYone yang sama dengan keyboard QWERTY fisik. Sudah ada banyak ulasan objektif di Internet, jadi saya tidak akan mengulangi lagi, tetapi ceritakan tentang kesan subjektif saya. Terutama tentang keyboard dan cara mengetik Rusia.


Jika Anda sudah tahu segalanya, segera buka bagian Keyboard .



Entri


(banyak air, Anda bisa melewatkan)


Jika Anda melewatkan semuanya, maka KEYone adalah smartphone BlackBerry baru kedua dengan keyboard QWERTY fisik.

Karakteristik singkat:

  • Snapdragon 625, 3 GB RAM
  • Layar 4,5 "IPS 1620x1080, 3: 2
  • Android 7.1.1 Nougat
  • Memori internal 32 GB, microSD hingga 2 TB
  • USB Type-C

Keyboard nostalgia
BlackBerry selalu terkenal dengan keyboard-nya, tetapi karena sistem operasinya yang unik di era iOS dan Android, mereka tidak lagi diperlukan. Dengan transisi BlackBerry ke Android, situasinya berubah menjadi lebih baik.

Tombol pada 2017 mungkin tampak seperti solusi kuno. Namun, smartphone Android pertama saya pada tahun 2010 adalah Motorola Cliq . Lalu ada Droid , Droid 2 dan Droid 4. Sebuah keyboard penuh memberi saya kebebasan luar biasa dalam teks: Saya sering mengetik dan banyak, dan tidak terbatas pada permintaan otomatis keyboard di layar.

Sayangnya, setelah Droid 4 tidak ada yang merilis smartphone dengan keyboard di Android, dan saya pergi ke Xiaomi.

Pada 2016, saya sangat senang dengan pengumuman BlackBerry Priv, smartphone Android pertama dengan keyboard dalam waktu yang lama, tetapi saya sudah memiliki Mi5 baru, dan membeli yang baru akan mahal.

Tahun ini, Xiaomi mulai membuat saya kesal - gangguan konstan dan mengabaikan privasi (trik kotor ini mengunggah foto saya ke cloud saya, bahkan ketika saya langsung menolaknya).

Singkatnya, saya sekarang memiliki KEYone, dan saya senang dengan itu, tetapi ada komentar. Dalam rangka:

Perumahan


Tentang kasus ini


Tahan lama, tidak licin, duduk dengan baik di tangan. Bingkai logam yang bagus di sekitar tepi. Tubuhnya cukup tebal. Saya sangat senang tentang itu. Saya tidak suka tren untuk smartphone ultra-tipis. Ketebalan KEYone adalah yang Anda butuhkan.

Ada klaim untuk desain: pertama, mengapa lubang untuk headphone tidak di tengah, tetapi sedikit di samping? Perfeksionis dari Internet membuka pembuluh darah mereka.

Kedua, pada bagian atas case yang sama, di sebelah kanan lubang headphone, ada colokan plastik dengan lubang mikrofon. Semua ini menyerupai baki drive CD pada komputer yang tidak bergerak, saya hanya ingin memasukkan jarum dan melihat apa yang keluar. (floppy disk? 1,3 inci hard drive?).

Layar


Tentang layar


Layarnya luar biasa, cerah dan tajam. Resolusi cukup untuk tidak melihat piksel individual. Sentuhan berfungsi dengan baik.

Dari minus: tingkat kecerahan minimum terlalu tinggi, membaca dalam gelap tidak sepenuhnya nyaman. Selain itu, rasio aspek layar berbeda dari FullHD standar, sehingga dalam beberapa program yang kurang optimal Anda harus sedikit menggulir layar. Tapi ini sangat jarang.

Keyboard




Hal utama yang patut dibeli adalah BlackBerry.

Empat baris tombol, total 35 pcs. Selain alfabet Latin AZ, ada tombol ALT, dua cipher, backspace dan enter, tombol untuk keyboard suara dan tombol karakter khusus, serta ruang di mana pemindai sidik jari terpasang di dalamnya.

Setiap tombol dapat diberi tindakan tertentu. Misalnya, meluncurkan aplikasi. Bahkan dua tindakan: pers pendek dan panjang.

Geser


Keyboard tidak sederhana, tetapi peka terhadap sentuhan: ia mengerti gesekan pada sisi dan atas dan ke bawah, serta kaset pada keyboard tanpa menekan.

Pertama, dengan svaypami Anda dapat menggulir halaman di Internet, atau menggulir layar dengan aplikasi.

Kedua, mereka diperlukan untuk bekerja dengan teks. Lihat:

Ketuk dua kali pada keyboard - kursor diaktifkan. Sekarang kita dapat memindahkannya ke kiri-kanan, pilih teks dengan menahan shift (seperti pada komputer besar), dan bahkan menggunakan pintasan keyboard Ctrl + C dan Ctrl-Vi! Bahkan Counter-Set untuk membatalkan aksi berhasil.

Saya akan menakuti anak-anak manja: di sini bahkan OpenOffice dapat diluncurkan dan dibuka dengan dokumen Word.

Bahasa rusia




Saya punya telepon dari Hong Kong, tidak ada karakter Rusia di tombol. Mereka akan berada dalam pengiriman resmi Rusia, tetapi bagi saya tampaknya akan terlihat buruk: sekarang hampir dua tombol ditandai pada setiap tombol. Tambahkan ini sepertiga, dan akan ada bubur.

Bagaimana cara mencetak dalam bahasa Rusia?

Keyboard BlackBerry bawaan tahu tata letak Rusia (yang tidak dicetak pada tombol) dan transliterasi Rusia. Dengan tata letak, ini kurang lebih jelas, tetapi transliterasi mungkin tampak baru bagi seseorang.

Sederhananya, ketika Anda perlu menulis kata "Halo", kami menekan tombol bahasa Inggris Pi, Ar, Ay, Vi, I, Ti. Huruf Rusia muncul secara otomatis.

Saya menggunakan metode serupa pada semua ponsel keyboard saya tanpa ukiran, bagi mereka saya menginstal aplikasi ruKeyboard .

Masalahnya adalah transliterasi di BlackBerry berfungsi sesuai dengan aturannya sendiri. Misalnya, untuk meletakkan huruf "I" Anda harus mengklik huruf Q, untuk huruf "Z" Anda perlu menekan tombol Z dua kali, huruf "Ha" akan muncul jika Anda mengklik "X". Dan sebagainya. Tidak ada yang bisa dilakukan dengan ini saat ini.

ruKeyboard


Mungkin seseorang menulis transliterasi tepat di nol tahun, tetapi saya lebih suka transliterasi fonetik normal. Kemudian "Ha" akan dicetak dengan huruf "H", dan "I" - dengan kombinasi tombol "Wye" dan "Hey". Jadi bisa melakukan rKeyboard .

RuKeyboard masih tersedia di Google Play. Ini belum diperbarui sejak 2012, tetapi sudah terinstal dengan sempurna dan bekerja pada KEYone baru. Begini cara kerjanya:


Namun, integrasi penuh dengan antarmuka KEYone tidak berfungsi. Pertama, jika tata letak Rusia diaktifkan, maka tombol pintas untuk program tidak berfungsi. Kedua, Anda hanya dapat beralih di antara dua bahasa. Saya sering menggunakan tata letak Cina, dan saya harus beralih melalui tirai - sebanyak tiga ketukan di layar. Di keyboard BlackBerry standar, Anda dapat memegang bilah spasi untuk memilih tata letak.


Selain itu, kursor tidak berfungsi di ruKeyboard.

Lainnya


Perlu dibiasakan dengan fakta bahwa tombol buka kunci dengan sensor sidik jari dan tombol beranda adalah dua tombol yang berbeda. Karena kebiasaan, saya menekan bilah spasi untuk pergi ke layar utama.

Secara umum, untuk membuat tiga tombol utama Android menyentuh dan langsung di bawah layar dan di antara keyboard adalah ide yang buruk. Apa yang mencegah mereka dari mekanis, tetapi peka terhadap sentuhan, jika Anda mau? Jari saya sering menggeser dan menekan salah satu tombol di jalan dari keyboard ke layar. Ini sangat umum ketika menggunakan kursor.


Pada bilah geser keyboard BlackBerry Priv sebelumnya , tombol-tombolnya benar-benar virtual dan muncul di layar. Mengapa tidak mungkin membuat tombol virtual di KEYone dan memperbesar layar juga merupakan misteri.


Ringkasan Keyboard


Ringkasnya, keyboard ini sempurna dan melebihi semua harapan jika Anda siap untuk menggunakan transliterasi yang aneh. Dalam situasi saya - penolakan input baru dan tiga tata letak yang digunakan secara aktif - Saya ingin yang terbaik. Saya sudah menulis permintaan transliterasi ke perwakilan BlackBerry di forum FourPDA dan membuat permintaan fitur di situs web resmi. Mari kita lihat apa yang terjadi.


Kamera


Tentang kamera

Untuk pemula, hanya ada jebakan untuk selfie acak! Awasi tangan Anda:


Pertama saya ingin mengunci telepon, dan karena kebiasaan saya klik tombol dari ujung kanan - saya sampai ke tombol Convinience, yang meluncurkan kamera. Saya tidak memerlukan kamera, dan saya ingin keluar dari kamera dengan menekan - lagi karena kebiasaan - tombol tengah bawah, yang ternyata merupakan ruang. Bam! Saya mengambil selfie terkejut.

Kamera mulai dan bekerja dengan cepat, gambarnya bagus, videonya juga. Benar, kamera depan tampaknya berfokus pada infinity, dan wajah di sekitarnya sedikit buram, tetapi ini agak sepele.

Sangat mengecewakan bahwa sistem tidak menggunakan Camera API 2, dan belum dimungkinkan untuk mengambil gambar di RAV.




Baterai


Tentang baterai
Setelah Mi5 - ini adalah kebahagiaan. Akhirnya, saya memiliki biaya yang cukup untuk sepanjang hari dengan penggunaan yang cukup aktif.

Performa


Hanya ada 3 GB RAM, yang tidak banyak pada 2017, jadi saya matikan semua yang tidak diperlukan. Secara umum, ponsel tidak cepat, kadang ada hang. Beralih antar aplikasi juga tidak terlalu lancar. Tapi ini bukan bencana. Saya tidak bermain-main, saya tidak akan mengujinya "demi centang".



Memori


Tentang memori
Memori internal hanya 32 GB, di mana sistem sudah mengambil sepuluh. Masuk akal untuk segera memasukkan kartu memori cepat, hampir semua jumlah yang didukung (hingga dua terabyte). Masuk akal untuk mengambil kartu yang bagus, misalnya SanDisk, pada 64-128 GB.

BlackBerry dihargai untuk aplikasinya - hub BlackBerry, surat, berbagai barang lainnya. Saya tinggal di China, di mana lantai internet diblokir, jadi saya biasanya tidak bisa menggunakan semua kebahagiaan ini. Matikan hampir semuanya sehingga memori tidak mau makan.



Total




Untuk uang Anda dapat menemukan smartphone lebih cepat, lebih produktif dan lebih menarik. Tetapi Anda tidak akan menemukan smartphone dengan keyboard. Membeli BlackBerry KEYone hanya untuk mereka yang membutuhkan keyboard fisik. Secara pribadi, saya puas.

PS: Sehubungan dengan rilis KEYone, Priv sebelumnya sekarang dapat ditemukan setengah harga. Dan dia tidak kalah keren dengan caranya sendiri.
