Yandex: há tudo ... sobre os usuários


A Yandex e as duas maiores agências de crédito russas lançaram um novo projeto para avaliar mutuários individuais: sua pontuação de pontuação pode ser estimada, entre outras coisas, usando dados coletados pela Yandex. A empresa disse que os dados anonimizados serão processados ​​por algoritmos automática e exclusivamente em um circuito Yandex fechado. A agência de crédito transmitirá à Yandex um endereço de e-mail com hash e o celular do usuário. Com base nesses dados, o sistema criará um modelo e emitirá um indicador de solvência.

O advogado do Centro de Direitos Digitais, Vladimir Ozherelyev, conta como isso afetará a segurança dos dados do usuário e se é consistente com a lei “Sobre Dados Pessoais”.

Os dados do usuário serão protegidos?


Sem uma compreensão completa dos algoritmos internos, é difícil avaliar a quantidade de dados do usuário que será protegida. Se um acesso não autorizado for obtido no banco de dados com informações sobre os mutuários, os recursos técnicos dos invasores poderão permitir que você estabeleça uma pessoa específica de acordo com o conjunto de dados, mesmo que eles sejam despersonalizados - tudo depende do nível de algoritmos usados. Apesar da grande atenção dos bancos à proteção de informações, sua transferência para o Yandex representa uma ameaça adicional. Porém, dadas as capacidades técnicas e a competência da Yandex, não há razão para acreditar que os dados dos mutuários certamente estarão em perigo. Tais projetos são inevitáveis ​​devido ao desenvolvimento contínuo da tecnologia da informação e geralmente desempenham funções socialmente úteis.

E a lei sobre dados pessoais?


Outro aspecto é a legalidade dessa transferência e processamento. A Lei Federal “Sobre Dados Pessoais” contém mecanismos que garantem a conformidade com a lei em tais operações. Os bancos podem obter facilmente o consentimento do mutuário para o processamento de dados, mas o Yandex precisará fazer esforços para que todos os usuários do Yandex concordem que seus dados possam ser usados ​​posteriormente para avaliar a solvência. É provável que a coleta e a preparação de dados para fins de avaliação de solvência sejam realizadas com antecedência pela Yandex, para que, no momento de receber uma solicitação do banco, seja possível avaliar com base nos dados existentes. Assim, a coleta de dados para fins de avaliação de solvência pode ser realizada independentemente do desejo do usuário de realizar essa análise. Infelizmente, a Lei Federal “Sobre Dados Pessoais” contém muitas lacunas que permitem abusar do direito ao processamento. Uma pequena quantidade de multas permite que você observe os riscos de violar a lei com os dedos.

Como encontrar um compromisso entre usuários e empresas que desejam usar seus dados?


Seria ideal que os usuários criassem um serviço de usuário Yandex separado que permita selecionar tipos de dados e o tempo de processamento para fins de pontuação de crédito e enviá-los aos bancos à vontade. De fato, quanto mais informações um banco puder avaliar para um tomador de empréstimo, mais rapidamente ele poderá tomar uma decisão sobre um empréstimo e, com base nas informações adicionais fornecidas pelo Yandex, o rating de crédito poderá ser aumentado, o que seria relevante para os tomadores de empréstimos que não têm um histórico de crédito.
