高速PAC控制器WISE-5580


与工业控制器的现代要求有关，即架构的开放性以及与PC兼容的计算机的快速增长，后者已越来越多地用作工业自动化任务的控制器。 这种与PC兼容的控制器称为SoftPLC。 此名称表明，常规PLC的大多数功能（在硬件级别解决）都可以使用软件解决。

PC兼容控制器的主要优点包括：开放性，可扩展性和选择编程工具的能力。

缺点之一是控制器周期长，结果导致性能指标低。

制造工业计算机已有30年历史的研华公司于2019年宣布发布具有高性能和速度的新型工业PC兼容控制器WISE-5580 。

WISE-5580是现成的硬件和软件平台（应用就绪平台），旨在解决高速数据输入输出，该数据的快速处理以及整个系统通过IoT协议与上层系统进行操作通信的任务。



新型高性能控制器具有高性能的Intel Core i7 / i5 / Celeron处理器以及具有实时功能和mini PCI Express的高速内部EtherCAT网络总线。

IT和ACS之间的桥梁


WISE-5580是为满足现代管理平台的要求而开发的，它允许办公室/ IT专家使用CoDeSys执行系统解决高层任务，而ACS专家则使用CoDeSys执行系统解决自动化问题。



一方面，控制器可以通过I / O模块通过最快，最确定的实时总线EtherCAT进行扩展，另一方面，可以通过微型PCI Express总线通过通信模块进行扩展。

一般规格：


  • Intel Core i7 / i5 / Celeron处理器（最高2.6 GHz）
  • 操作系统Windows 7/10
  • 具有警报功能的备用电源
  • 安全值I / O
  • 2个带隔离的RS-232 / 422/485串行端口
  • 2个支持POE的千兆以太网端口
  • 4个USB 3.0端口
  • 支持3G / LTE / Wi-fi / GPS标准
  • VGA和HDMI视频输出
  • Codesys RTE支持
  • 根据IEC 61131-3进行编程
  • 支持Modbus，EtherNet / IP，PROFINET，OPC / UA和MQTT代理模式
  • 控制器尺寸：139x100x80毫米



扩充模组


I / O模块允许您从各种模拟和离散传感器，热敏电阻，热电偶和控制执行器启动控制器信号。

您可以将数码摄像机连接到具有USB 3.0的通信扩展模块和具有PoE的千兆以太网端口，并在分析接收到的图像之后，通过EtherCAT模块将控制信号发送到伺服器。 使用WISE-5580的这种情况将使在其上实施技术视觉系统成为可能。



1- WISE-5580视觉系统集成负责人
2- WISE-5424V ，具有4个USB 3.0端口的扩展模块
3- WISE-5001 ，带有4个数字量输入的电源模块
4- WISE-5079 ，EtherCAT扩展模块
5- WISE-5074 ，EtherCAT总线耦合器
6-I / O模块

使用可选的EtherCAT总线耦合器（ WISE-5074 ），可以将整个系统分为2个不同的DIN导轨，甚至可以将部分模块安装在另一个机柜中。

无线接口


为了通过Wi-Fi，3G，LTE，GPS等无线通信通道与控制器进行通信， WISE-5400E通信模块从PCI Express 总线的侧面连接，可以在其中安装迷你PCIe卡并且有一个SIM卡插槽。



模块顶盖上的SMA天线连接器有一个开口。

软体类




作为WISE-5580中的现成软件解决方案，使用了 CODESYS工具软件包。 这样就可以根据IEC 61131-3对控制器进行编程，并且由于EtherCAT总线的支持，CODESYS提供了控制微秒过程的能力，因此WISE-5580可以成为具有优化性能的PAC（可编程自动化控制器）。

CODESYS具有适用于各种应用场景的内置3级执行系统（控制运行时系统）以及数据可视化功能。
还可以通过LabVIEW软件平台监视整个系统的状态，DAQ Navi软件包的现成驱动程序和库将简化软件开发。

内置的Advantech Lmsensor驱动程序提供了最大化控制器本身性能的功能，并使得在使用Microsoft Visual C ++，Embedded Visual C ++和其他编程语言在不同Windows系统平台上开发的程序中的通用Lmsensor操作变得容易。

结论



视频：WISE-5580俄语宣传视频

WISE-5580控制器-是用于收集，转换，处理数据和自动过程控制任务的交钥匙解决方案。 控制器软件允许您将I / O与上层无缝集成，从而节省了整个系统的开发成本，但同时又实现了广泛的自动化任务。

得益于高速内部接口，强大的处理器和CoDeSys支持，使用WISE-5580的开发人员和最终用户可以获得与标准PLC可以提供的功能相同的功能，但同时它们具有方便的PC兼容开放平台。
