Transisi massal ke Windows 10 tertunda, terlepas dari upaya Microsoft



Bulan lalu, Microsoft mulai memaksa pengguna untuk memutakhirkan ke Windows 10, bukannya memperbarui sesuka hati. Ini diharapkan memicu gelombang aktivasi baru, karena pengguna yang belum mengonfigurasi pemblokiran pembaruan akan menemukan Windows 10 siap dipasang di mesin mereka.

Microsoft mengatakan bahwa pengguna akan dapat menolak untuk meningkatkan ke OS baru setelah memulai instalasi, tetapi Saya tidak mengatakan apakah transisi ke Win 10 akan ditawarkan lagi setelah klien menolak untuk memperbarui.

Diasumsikan bahwa kebijakan seperti itu akan menyebabkan peningkatan yang signifikan dalam instalasi, tetapi menurut perhitungan Aplikasi Net, StatCounter dan DAP (lihat grafik di atas) ini tidak terjadi, demikian laporan Extremeetech .

Aplikasi Net melaporkan bahwa Win 10 diinstal hanya pada 14,2% perangkat. Ini berarti bahwa Microsoft baru saja melangkahi 200 juta instalasi, meskipun mengklaim telah terjadi pada bulan Januari. Perlu dicatat bahwa produk baru perusahaan saat ini lebih populer daripada gabungan Windows 8 dan Windows 8.1 (win live 7! Approx.) .

Apa bahayanya bagi Microsoft?

Beberapa pengguna akan menolak untuk memperbarui karena mereka tidak yakin apakah itu legal atau tidak. Beberapa dari mereka tidak akan melihatnya, karena sistem mereka tidak dikonfigurasi untuk menerima pembaruan otomatis. Dan sisanya hanya tidak suka Windows 10. Tetapi bahkan dengan asumsi bahwa salah satu pengguna masih berubah pikiran dan beralih ke Win 10, Microsoft masih tidak akan mendekati "OS dasar" dari pasar PC.

Sasaran satu miliar perangkat yang menjalankan Windows 10 sangat ambisius. Mungkin Microsoft harus mulai dengan menawarkan perangkat lunak gratis dari paket Office atau Xbox Live?

Plus, jika Microsoft pernah berharap untuk membuat OS seluler yang ingin digunakan orang, itu akan perlu membentuk ekosistem lengkap dengan Windows 10. Artinya, itu semua tergantung pada seberapa mudah aplikasi Windows bermigrasi dari ponsel ke versi boot platform. dan sebaliknya.
