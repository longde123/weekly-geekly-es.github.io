Versi HTML5 dari StarCraft



Game StarCraft asli (dengan tambahan Brood War) telah diangkut ke HTML5 . Sekarang karya agung dari Blizzard Entertainment ini diluncurkan tepat di peramban Anda. Benar, ini belum merupakan game yang lengkap, tetapi hanya level demo, tetapi terlihat cukup layak. Pengembang bekerja keras, porting kode dari bahasa pemrograman C, C ++ dan assembler ke Javascript.



Seorang pengguna dengan nama panggilan Ryuta menerbitkan kode sumber di Github. Jika Anda menempatkan game di komputer pribadi Anda, yang bertindak sebagai server, Anda dapat bermain offline dengan menjalankan StarCraft di localhost.

Pengembangan versi HTML5 merupakan kabar baik bagi pengguna Linux, karena sebelum permainan tidak dimulai langsung di OS gratis, tetapi hanya melalui Wine. Sekarang demo bekerja tanpa emulator. Strategi kultus secara real time keluar 18 tahun yang lalu, tetapi sejauh ini dapat disebut standar kualitas dalam genre RTS. Bagaimanapun, senang mengingat masa lalu yang indah yang kembali berkat teknologi web modern. Mungkin tidak lama. Repositori di Github berisi grafis StarCraft eksklusif, tempat Blizzard Entertainment dapat terbang ke pembuatnya.







Ngomong-ngomong, pekerjaan juga sedang berjalan pada porting Starcraft 2 ke WebGL .
