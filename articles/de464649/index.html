Dokumentenkollaborationssystem für Zimbra Open-Source Edition

Die Bedeutung der Mitbearbeitung von Dokumenten in modernen Unternehmen ist schwer zu überschätzen. Durch die Möglichkeit, Verträge und Vereinbarungen unter Beteiligung von Mitarbeitern der Rechtsabteilung zu erstellen, kommerzielle Angebote unter Aufsicht von Vorgesetzten online zu schreiben usw., kann das Unternehmen Tausende von Arbeitsstunden einsparen, die zuvor für zahlreiche Genehmigungen aufgewendet wurden. Aus diesem Grund war eine der Hauptinnovationen in Zextras Suite 3.0 das Erscheinen von Zextras Docs - einer Lösung, mit der Sie die vollständige Zusammenarbeit mit Dokumenten direkt im Open-Source Edition-Webclient der Zimbra Collaboration Suite organisieren können.

Derzeit unterstützt Zextras Docs die Zusammenarbeit mit Textdokumenten, Tabellen und Präsentationen und weiß auch, wie eine Vielzahl von Dateiformaten verarbeitet werden. Die Lösungsoberfläche unterscheidet sich nicht wesentlich von der Oberfläche eines Texteditors, sodass die Mitarbeiter des Unternehmens nicht viel Zeit für Schulungen aufwenden müssen, um auf die Verwendung von Zextras Docs umzusteigen. Aber das interessanteste ist wie immer "unter der Haube". Schauen wir uns gemeinsam an, wie Zextras Docs funktioniert und welche Vorteile diese Lösung für die Zusammenarbeit von Dokumenten bietet.



Vor allem Zextras Docs wird diejenigen ansprechen, die Zimbra OSE und Zextras Suite bereits in ihrem Unternehmen verwenden. Mit dieser Lösung ist es möglich, einen neuen Service in die Produktion einzuführen, ohne die Anzahl der Informationssysteme zu erhöhen, und kann daher auf die Erhöhung der Betriebskosten der IT-Infrastruktur verzichten. Beachten Sie, dass Zextras Docs nur mit Zimbra OSE Version 8.8.12 und höher kompatibel ist. Wenn Sie immer noch veraltete Versionen von Zimbra verwenden, empfehlen wir dringend, ein Upgrade auf Version 8.8.15 LTS durchzuführen. Aufgrund des langfristigen Supports bleibt diese Version noch einige Jahre auf dem neuesten Stand und sicher und ist auch mit allen aktuellen Ergänzungen kompatibel.

Zu den Vorteilen von Zextras Docs kann auch die Möglichkeit der vollständigen Bereitstellung in der Unternehmensinfrastruktur gehören. Dies vermeidet die Weitergabe von Informationen an Dritte, wie dies häufig bei der Verwendung von Hybrid- oder Cloud-Diensten der Fall ist. Aus diesem Grund ist Zextras Docs ideal für Unternehmen mit einer strengen Informationssicherheitsrichtlinie und für Systemadministratoren, die es vorziehen, die Datenflüsse und -prozesse im Unternehmen vollständig zu steuern. Darüber hinaus können Sie mit einem lokal bereitgestellten Dienst für die Zusammenarbeit von Dokumenten die Risiken vermeiden, die mit der Unzugänglichkeit des Cloud-Dienstes verbunden sind, wenn Probleme mit dem Dienstanbieter oder Probleme mit dem Hochgeschwindigkeits-Internetzugang auftreten.

Zextras Docs besteht aus drei Teilen: einem eigenständigen Server, einer Erweiterung und einem Winter. Jeder dieser drei Teile leistet seinen Beitrag:

  • Der Zextras Docs-Server ist eine LibreOffice-Engine, die für die Zusammenarbeit und Integration mit Zimbra OSE optimiert wurde. Auf dem Zextras Docs-Server werden alle Dokumente, auf die Benutzer zugreifen, geöffnet, verarbeitet und gespeichert. Es muss auf einem dedizierten Computerknoten installiert sein, auf dem Ubuntu 16.04, Ubuntu 18.04 oder CentOS 7 ausgeführt wird. Wenn die Belastung des Zextras Docs-Dienstes groß genug ist, können Sie mehrere Server gleichzeitig auswählen.
  • Die Zextras Docs-Erweiterung muss nicht installiert werden, da sie in die Zextras Suite integriert ist. Dank dieser Erweiterung ist der Benutzer mit dem Zextras Docs-Server verbunden und kann bei Verwendung mehrerer Server einen Lastenausgleich durchführen. Darüber hinaus werden über die Zextras Docs-Erweiterung mehrere Benutzer gleichzeitig mit einem Dokument verbunden und Dateien vom lokalen Speicher auf den Server heruntergeladen.
  • Zextras Docs Zimlet wird wiederum benötigt, um den Service in den Webclient zu integrieren. Ihm ist es zu verdanken, dass der Zimbra-Webclient die Möglichkeit hat, Zextras Docs-Dokumente zu erstellen und eine Vorschau anzuzeigen.



Um Zextras Docs im Unternehmen bereitzustellen, müssen Sie zuerst einen oder mehrere physische oder virtuelle Server dafür auswählen. Danach müssen Sie die Serveranwendungsverteilungen für Ubuntu 16.04 , Ubuntu 18.04 oder CentOS 7 von der Zextras-Website herunterladen, entpacken und installieren. In der letzten Phase der Installation werden Sie vom Server aufgefordert, die IP-Adresse des LDAP-Servers sowie das Paar aus Benutzername und Kennwort anzugeben, mit dem Daten über den neuen Server in LDAP eingegeben werden. Beachten Sie, dass nach Abschluss der Installation jeder Zextras Docs-Server für alle anderen Infrastrukturknoten sichtbar ist.

Da die Zextras Docs-Erweiterung bereits in der Zextras Suite enthalten ist, reicht es aus, sie nur für Benutzer und Gruppen zu aktivieren, die Zugriff auf Tools für die Dokumentzusammenarbeit haben sollten. Das Zextras Docs Zymelet kann über die Zimbra-Administrationskonsole aktiviert werden. Beachten Sie, dass Sie nach dem Hinzufügen des Zextras Docs-Servers zur Zimbra OSE-Infrastruktur die Konfiguration des Zimbra-Proxyservers aktualisieren müssen. Führen Sie dazu einfach die Datei / opt / zimbra / libexec / zmproxyconfgen als Zimbra-Benutzer aus und führen Sie dann den Befehl zmproxyctl restart aus , um den Proxy-Dienst neu zu starten.
