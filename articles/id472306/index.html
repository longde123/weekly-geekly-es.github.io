PHP internal


  • [RFC] Operator Backtick yang Tidak Tepat (V2) - Dalam PHP, Anda dapat menggunakan tanda kutip ` ` sebagai alias untuk shell_exec() . Artinya, baris yang tertulis di dalamnya akan dianggap sebagai perintah dan diluncurkan di shell. Proposal untuk menyatakan kesempatan ini sudah usang menyebabkan badai kontroversi , seperti proposal lainnya untuk penghinaan.
  • [RFC] Menerapkan DOM Living Standard APIs baru di ext / dom - Diusulkan untuk menerapkan dukungan untuk standar DOM baru dalam ekstensi kernel. Pelajari lebih lanjut tentang penawaran di podcast. PHP Internals News # 31 dengan penulis RFC Benjamin Eberlei .
  • Sebuah jajak pendapat dari Sarah Goleman di Twitter tentang gagasan blok uji coba di tingkat fungsi, terinspirasi oleh fitur serupa dari C ++:

Alat-alatnya


  • mougrim / php-xdebug-proxy - Proxy PHP Xdebug: ketika fitur standar Xdebug tidak cukup .
  • php-defer / php-defer - Sebuah analog dari fungsi menunda dari Go . Callback yang dilewatkan sebagai parameter fungsi ini akan berfungsi ketika utas eksekusi meninggalkan ruang lingkup fungsi / metode saat ini (eksternal). Sebuah destructor hanya digunakan di bawah tenda. Idenya sudah tua dan sudah diterapkan beberapa tahun yang lalu sebagai bagian dari phplang / scope-exit .

  • graze / parallel-process - Memungkinkan Anda untuk dengan mudah menjalankan beberapa symfony / proses secara paralel.
