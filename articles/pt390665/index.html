Relógio de anel caseiro em ATtiny85



O fabricante chinês Chen Liang (陳亮) publicou instruções passo a passo sobre como montar relógios de anel fofos no microcontrolador ATtiny85.

Comparado com o projeto ATtiny Watch Core anterior , o autor decidiu que o sistema funcionaria com uma voltagem elétrica de 2,0 V, para que o pequeno microcontrolador ATTINY85V-10SU caísse. Para ele é o minúsculo painel de depuração Digispark.

As baterias CR1220 devem durar seis meses.

Para a tela, o autor utilizou a tela OLED SSD1306, a menor tela desse tipo com resolução de 64x32 pixels.

O estojo é mais fácil de imprimir em uma impressora 3D e cubra a tela com um pedaço de plástico transparente.

Para a montagem, você precisará de mais três parafusos M2, três botões, dois resistores (mais de 1k), fios de cobre com isolamento, uma placa fina, uma placa adaptadora do SOP8 ao DIP8 e um pequeno pedaço da placa PC multiuso.



O autor descreve em detalhes a instalação e configuração do Arduino, fornece links para as fontes de todo o software necessário para o relógio funcionar, ele também postou arquivos para impressão 3D da caixa do relógio.
