Viel Spaß beim Lesen!



Nachrichten und Veröffentlichungen


  • Das Zend Framework wird von der Linux Foundation unterstützt - und in Laminas umbenannt, um die Handelsmarke Zend loszuwerden.
    Expressive , Apigility und ZF selbst werden in die neue Organisation aufgenommen. Bisher enthält das Repository nur das Bridge- Paket laminas / laminas-zendframework-bridge mit einem speziellen Autoloader. Sie versprechen, den vollständigen Umzug in den nächsten Monaten abzuschließen.

PHP-Interna


  • [RFC] Pfeilfunktionen 2.0 - Die Abstimmung für kurze Lambdas hat begonnen und der Abstimmungsprozess ist ermutigend. Die Syntax bleibt gleich: fn($x) => $x * $y .
  • [RFC] Veralteter linksassoziativer ternärer Operator - In fast allen Sprachen wird der ternäre Operator von rechts nach links ausgewertet, im Gegenteil in PHP. Es wird vorgeschlagen, die linke Assoziativität insgesamt abzuschaffen. Wenn mehrere ternäre Operatoren verwendet werden müssen, geben Sie die Reihenfolge explizit mit Hilfe von Klammern an. 
     1 ? 2 : 3 ? 4 : 5; // deprecated (1 ? 2 : 3) ? 4 : 5; // ok 1 ? 2 : (3 ? 4 : 5); // ok
  • PHP Internals News # 5 - On List Inclusions mit dem RFC- Autor Larry Garfield.
  • PHP Internals News # 6 - Interview mit Remi Collet, Release Manager von PHP 7.2, zur PHP-Qualitätssicherung und -erweiterungen.

Die Werkzeuge

Einfache Installation und geringe Systemanforderungen sowie komprimierte Daten beanspruchen nur sehr wenig Speicherplatz. Eingereicht von morozovsk .
  • Cycle / Orm - ORM für langlebige Anwendungen des RoadRunner- Autors Anton Titov. Detaillierte Dokumentation im Prozess gibt es einen Vergleich mit Doctrine und Eloquent .
  • php-comrade / comrade-dev - Scheduler und Task-Manager in PHP.
  • jwage / changelog-generator - Generator einer Liste von Änderungen am Marketdown basierend auf Informationen von GitHub.
  • moneyphp / money - Implementierung des Fowler-Musters Money. Unter der Haube werden Saiten verwendet.
  • mamuz / PhpDependencyAnalysis - Das Tool erstellt ein Abhängigkeitsdiagramm abstrakter Typen und zeigt fehlerhafte Abhängigkeiten an.
