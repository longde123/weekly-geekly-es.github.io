一个完整的噩梦：USB-C和Thunderbolt 3

您是否已经购买了新的MacBook或MacBook Pro？还是Google Pixel？您将对这些新的USB-C端口感到困惑。这个外观简单的端口充满了普遍的困惑，并且向后兼容的特性为不同的任务使用了不同的电缆。买家必须非常小心地选择电缆！



USB Type-C：端口和协议


USB Type-C端口已经足够普及，Google开始在计算机，Pixel和Nexus手机上使用它们，Apple在12英寸MacBook上使用它们，现在也在新的MacBook Pro上使用。这是24针双向连接器的物理规格，在本文中，我将使用此物理电缆和USB-C端口作为最常用的术语，Google报告称该端口被称为“ USB-C” 2,100万次，“ USB C” 1,200万次，没错，“ USB Type-C”只有850万次。


USB-C兼容性：支持多种协议，并且每个级别都向下兼容

USB-C 之下的级别，从而允许各种信号通过其自身：

•USB 2.0-奇怪的是，最早的USB-C设备（包括诺基亚N1）支持的信号和仅为USB 2.0供电。几乎所有新计算机都至少支持USB 3.0，但是某些手机和平板电脑仍然存在局限性。

•USB 3.1第1代-非常类似于“ SuperSpeed” USB 3.0，适用于从硬盘驱动器到网络适配器和扩展坞的各种外围设备的5 Gb / s串行通信。向后兼容SuperSpeed USB 3.0，Hi-Speed USB 2.0，甚至是1996年以来的原始USB 1.x！苹果在12英寸MacBook中使用此协议。

•USB 3.1 Gen 2-令人困惑的名称将USB外设的带宽增加了一倍，达到10 Gb / s。向后兼容所有以前的USB版本。仅最新的USB-C设备支持它。我不知道是谁给他起了这样的名字。

•备用模式-物理USB-C连接器支持其他非USB协议，包括DisplayPort，MHL，HDMI和Thunderbolt。但是并不是每个设备都支持Alternate Mode协议，这对客户来说非常令人困惑。

•Power Delivery不是数据协议，但是USB-C允许您提供高达100瓦的电源。但是同样，有两种不同的规格和许多不同的配置。

•音频附件模式（音频附件）-与模拟音频一起使用的规范。

USB-C的主要问题是混乱。并非每条USB-C电缆，端口，设备和电源都相互兼容，因此您必须考虑许多组合。最新最复杂的设备（例如，带Touch Bar的MacBook Pro）将支持该端口的大多数用途，但常见的旧设备仅支持USB 3.0，并且幸运的是，还支持备用模式DisplayPort。

但这还不是全部。许多USB-C外设也有其局限性。想象一下USB-C HDMI适配器。它可以通过USB 3.0实现HDMI，也可以使用本机备用模式HDMI。它也可以使用具有Thunderbolt备用模式的HDMI多路传输，甚至在理论上甚至可以使用外部图形芯片通过Thunderbolt进行HDMI！是我宣传带有集成GPU的Thunderbolt Display的想法。而且只有最新的计算机才能支持这三种模式。想象一下，购买适配器“ USB-C HDMI”的消费者发现其不能与MacBook或Pixel或其他产品一起使用时感到困惑吗？

这些电缆看起来相同，但功能却大不相同！ （在我看来，Monoprice为两根不同的电缆贴了一张照片）

电缆兼容性问题更加严重。许多公司，包括我心爱的Monoprice，都制造质量和兼容性各异的USB-C电缆。如果不小心，可能会限制功能，甚至使用错误的电缆损坏设备。严重：错误的电缆会损坏您的设备！这不应该，但是在这里。

两端的某些USB-C电缆只能传输5 Gb / s，另一些与10 Gb / s的USB 3.1 gen 2兼容。另一些不能用于供电，或者与Alternate Mode Thunderbolt不兼容。看看Monoprice 3.1 10 Gbps / 100瓦USB-C到USB-C，3.0 5 Gbps / 15瓦USB-C到USB-C和2.0 480 Mbps / 2.4 A USB-C到USB-C。为什么它们甚至存在？为什么需要仅支持2.0的USB-C至USB-C电缆？

并且电缆的末端带有不同的连接器。Monoprice销售一种很酷的USB-C适配器到USB 3.0 10 Gb / s，但它也有一个支持5 Gb / s，甚至有限的480 Mb / s USB 2.0的适配器。他们看起来几乎一样。对消费者来说真是一场噩梦！Monoprice将每条5 Gb / s电缆错误地命名为USB 3.0，而每条10 Gb / s电缆错误地命名为USB 3.1。另一方面，这种名称比正式名称更易于用户理解。

我没有遇到Monoprice。我喜欢他们的电缆。但是它们种类繁多的USB-C电缆完美地说明了不兼容的问题。几乎所有制造商和销售商都有这些问题。

雷雳3


让我们继续讨论一个更加令人困惑的话题。从2011年MacBook Pro的销售开始，Mac所有者就习惯了Mini DisplayPort连接器，该连接器可同时使用图形端口和数据端口。它们还用于将Thunderbolt电缆插入Mini DisplayPort连接器，并发现没有任何作用。

新的USB Type-C端口会给我们带来相同的体验：

•并非所有USB-C端口都具有相同的功能。许多都是纯数据的，有些知道如何使用数据和视频，很少有数据，视频和Thunderbolt 3！

•Thunderbolt 3需要专用电缆。尽管它看起来与普通的USB-C完全一样！

•具有Thunderbolt 3的设备看起来与具有USB-C的设备完全一样-具有USB-C电缆的常规设备的速度限制为5 Gb / s或更低，但是具有Thunderbolt 3的设备以40 Gb / s的速度传输PCI Express！

Thunderbolt 3端口和电缆必须与USB 3.1 Type-C电缆，端口和设备向后兼容。但是它们的工作会更慢。赞扬创建者的向后兼容性。顺便说一下，这是一个简化。实际上，就像HDMI一样，Thunderbolt 3是Type-C电缆和端口的“备用模式”。但实际上，Thunderbolt 3是USB-C的USB 3.1的超集，因为没有Thunderbolt 3实施仅支持USB 2.0。

因此，支持Thunderbolt 3的计算机的所有者在购买设备和电缆时应格外小心，以免占用带宽。 Apple当前使用的大多数USB-C附件和电缆都可以与新的MacBook Pro兼容（向后兼容），但可能无法提供全速。而且，旧的12英寸Retina MacBook的所有者甚至更糟，因为配备Thunderbolt 3的设备根本无法在其中运行！

由于Thunderbolt 3包含数据和视频，因此您很容易将其与计算机，电缆和设备的兼容性混淆。例如，一根Thunderbolt 3电缆可以支持两台4K 60 Hz显示器，甚至可以支持5K显示器，而USB-C电缆仅限于一台4K显示器。有趣的是，USB-C备用模式不像Thunderbolt 3那样具有视频兼容性。后者支持HDMI 2.0，而USB 3.1仅支持HDMI 1.4b。但是在DisplayPort的情况下，USB 3.1将具有优势，它支持1.3版本，而不仅仅是Thunderbolt 3这样的1.2版本。这一切都取决于特定机器上的实现。


苹果没有在新的MacBook Pro上做Thunderbolt图标，这使消费者更加困惑！

请注意，有40和20 Gb / s的Thunderbolt 3电缆。MacBook Pro与许多早期的Thunderbolt 3设备中使用的第一代德州仪器Thunderbolt 3控制器不兼容！

我的意见


鉴于新的USB Type-C端口具有如此疯狂的“兼容性”，客户将必须非常小心。而且，如果该行业正朝着一个简单，可靠的双向数据，视频和电源端口发展的事实，那么这种混乱的设备和电缆将使消费者不满，并需要技术人员。

另外：如果卡住了，那应该可以工作


我对该文章的原始版本提出了很多批评，其原因在于似乎并非所有事情都像我描述的那样糟糕。基本上，只要人们拥有仅支持USB等的Nexus手机，这都是对的。但是在我看来，这种通用电缆和端口的许多用例存在问题。

电子技术已不再是极客感兴趣的领域。大多数计算机，电话，平板电脑和外围设备都是由不懂技术的人购买的。它们不会将协议与接口区分开，也不需要了解“ USB Type-C”与“ Thunderbolt 3”或“ USB 3.1”的区别。他们想购买东西，连接它们，以便一切正常。他们通过连接器的形状和匹配程度而不是规格和徽标来评估兼容性。

从历史上看，该行业已对此进行处理。在经历了最初的麻烦之后，USB已成为普通用户的福音。电缆，设备和外围设备-大多数情况下都可以正常工作。尽管使用USB 3，微型USB，微型USB和大功率充电器的体验并不理想，但用户对USB的期望“先安装后可行”仍然适用。我本人现在正在使用廉价的USB电缆花环。原因是USB既是电缆又是协议。除了电源（有多少iPad正在从iPhone立方体缓慢充电？）之外，USB还能工作，因为USB是USB。

现在，有一条“通用”电缆可以成为设备上的唯一端口。数据，视频，电源-仅一个USB C型端口。英特尔选择了更高的设备，增加了一个完全独立的数据和视频支持世界，即Thunderbolt3。期望所有端口，电缆和设备都能正常工作是不现实的，特别是如果制造设备或USB 3.1 gen 1电缆便宜得多的话甚至USB 2.0。

从这一刻起（自Thunderbolt 3设备开始销售以来），我们的端口就无法满足用户的期望。电缆不兼容，尽管端口看起来相同，但设备不支持任何外围设备。这是一场噩梦：消费者将错误地从包装盒，商店或背包中拉出电缆，并确定设备或充电器在无法使用时断开。我们将面临挫折，回报和混乱的技术支持。

这是一个具有兼容性的古老故事。我们正在改善兼容性，以提高消费者对一切都会正常工作的期望。但是USB Type-C永远无法正常工作，因为USB-C一次太多了。这是一场噩梦。
