微软如何降低噪音以创建有用而美丽的声音世界

您是否想过在收到消息，电池电量耗尽或出现会议提醒时计算机发出的声音？ 马修·贝内特（Matthew Bennett）在思考。 我想了很多

Bennett在选择完美的Windows 10日历警报声音之前，撰写，执行和处理了900多个版本的Windows 10日历警报声音。

Bennett回忆说：“那是很长一段时间，但是效果很好。” Bennett在最近一次访问他位于雷德蒙德的录音棚时耸了耸肩。 在一个用蜡烛点燃的隔音房间中，有一个调音台，几台高质量的演播室监视器和大液晶屏，中央是一个全倍频程合成器键盘。

作为从事许多Microsoft应用程序和设备的作曲家和声音工程师，Bennett在公司的声音设计中一直发挥着重要作用15年。 他对声音有坚强的信念和既定的哲学，并且精通专业术语来讨论这些问题。



本文在我们的网站上 。

总结他的角色时，他反映：“首先，我们对消费者的责任不是伤害。 产生不会割伤耳朵且不会令人讨厌的声音。 此外，声音必须具有功能性和优美感。 美观和功能齐头并进。 声音越美，对我们想要唤起人们的产品整体感觉的影响就越大。”


Bennett说，一年来，他与四个帮手一起为Windows 10制作了一系列声音，以使他的工作变得完美。 “就像疯了一样，我正在整理这些声音。 我花了多长时间才感到困扰。 我着迷于控件。”

现年53岁的贝内特（Bennett）是位爵士乐迷，他会弹奏键盘并对西非音乐感兴趣，他拥有音乐人种学（音乐的社会和文化方面的科学）硕士学位。 在2000年代初期，他离开了微软10年，担任自由作曲家，创作声音设计和创作音乐。

在此期间，他为电影和《财富》 500强公司录制了几条配乐，但最终，这项工作使他不满意。

为了寻找新的灵感，他放弃作曲，开始在西非，印度，中东和印度尼西亚学习中世纪的唱歌和音乐。 渐渐地，他再次开始写作，他努力创造一种个人的音乐语言-“一个我可以生活的声音世界”，正如他本人所描述的那样。 给出的示例显示了这些搜索的结果。

回到微软，贝内特全力以赴工作。 现在，他的作品不仅可以在Windows中听到，而且可以在Xbox操作系统以及应用程序和其他产品（如Office，Cortana和Skype）中听到。 就像用户界面设计开发人员努力在公司所有产品中实现视觉和功能的一致性一样，它创建了一个声音景观，该声音景观应统一Microsoft的声音配置。

我们希望实现设备与感觉之间的和谐


此外，Microsoft创建了Fluent声音和感官设计开发环境，以影响公司外部使用的声音。

Bennett说：“使用声音，我们可以塑造用户界面的节奏和情感结构。” -声音成为与用户其他感觉一致的元素-包括触摸，图形纹理和动态效果。 我们正在改变Microsoft，甚至希望在Microsoft之外对声音设计的态度。 我们希望在设备和感觉之间实现和谐。”

微软的媒体架构师里克·塞内查尔（Rick Seneschal）在贝内特（Bennett）工作了20年。 Seneschal管理着国际音乐服务，该服务每年为各种公司和代理商的活动，视频，播客和产品选择大约4,000首歌曲。

塞内沙尔说贝内特并不着急，也很挑剔。

Seneschal说：“ Matthew是我工作过的最专注的人。” -他可以无限地完善和改进自己的工作。 他对自己的领域有着深刻而广阔的视野，并且思想很好。 他不仅发出声音，而且说：“是的，听起来不错。” 对于它产生的声音和纹理，这是逻辑和理由。”

它产生的声音和纹理的背后是逻辑和理性。


Bennett立即说微软的声音不是。

他说：“这不是声音效果，不是游戏声音，不是蜂鸣声，狗叫声和咆哮的喇叭声，不是未来派的声音，不是音乐不能充满一切，不是语音也不是警报。” “我们不会录制诸如cong，lute或小提琴之类的真实乐器声音的片段，因为它们会带您到某个文化地点和时间，有时甚至是您自己的过去，并分散您的工作注意力。”

他没有透露品牌名称，但表示主要智能手机制造商的声音在这方面完全没有道理。

“听所有这些铃声。 您是否听说过其中至少有一个是在现实生活中使用的？” 数字版的《海边》（By the Sea），琶音和弦已定下调，以一种动人的音色合成了鸟的歌声。 “起初它们很有趣，大约在15年前，它们可能很有趣，但是如今，它们还有很多不足之处。”

Bennett指出，较旧版本的Windows中的声音与Windows 10中出现的声音有很大不同。首先，它的声音要多得多。 他说：“不再需要表示成功下载的Triiumphant声音，” “我们为什么要庆祝计算机开机？” 现在我们应该已经把它视为理所当然了，总的来说，这就是我们在做的事情。”

今天的声音要短得多。 诸如Windows NT Workstation 4.0（1996）中的关闭信号之类的先前声音可能会持续长达8秒钟-按照当今的标准是无限的，需要的干扰声音更少，以几分之一秒为单位。 启动系统的声音以及关机的声音已经成为过去，现在被认为是另一个不必要的技术噪声污染源。


Microsoft Production Studios的Matthew Bennett和声音工程师Dan Sharett。

现代声音远没有它们的前辈那么宏伟。 Windows 2000中的启动声音持续了将近12秒钟：似乎有一个战斗机中队起飞，然后彩虹色的马林巴斯进入了。 “今天，我们使用最自然，最安静，最安静且不引人注意的声音，” Bennett说。 “我们发出的声音会让人保持沉默。”

专家称之为拟态的声音消失了：复制真实对象的声音，例如删除文档时弄皱的纸或打字时键入打字机的键。

这些声音帮助人们熟悉了新技术，但是不再需要它们了，它们不再有用。 现在，这些只是毫无意义的小饰品，”贝内特说。

在当今的简约设计中，不受欢迎的声音不受欢迎。

“大约五年前，视觉设计领域开始切断所有多余的东西，例如阴影，宽边框，灯光效果和大字体，” Bennett补充说。 “声音设计也会发生同样的事情。”
我们在保持沉默的同时发出声音
Windows 7中大约有40种声音。 在Windows 10中-大约八种，尽管“出于未知的技术原因”保留了旧声音。 Bennett说：“刚开始的时候，有七种不同的声音提示错误消息，没人知道它们的意思。 对于合作伙伴或我们自己都没有明确的规则。 我们摆脱了所有这些声音，将它们替换为两种：一种用于非严重错误，另一种用于更重要的错误。”

贝内特通过比较语调轮廓来帮助工作，语调轮廓是用不同语言发音相同事物的人不断变化的声音。 问题准备好了吗？ 发音时英语，法语，日语，中文，西班牙语和俄语的高度变化几乎相同：“向上，向下，向下，急剧上升”。

Bennett通过音乐手段体现了这种语调，他听到的声音持续2.5秒，当Windows 10日历警报出现时可以听到，这是他引以为傲的声音之一。

在新消息的旋律（短语“为您消息！”-“给您的消息！”）和一般通知的旋律（“一切都很好”-“一切都很好”）中可以听到语调轮廓。

Bennett认为，操作系统的声音应考虑与它们相关联的那些动作的功能。 例如，信使中的消息通常比电子邮件需要更紧急的响应，因此，根据Bennett的说法，在Windows 10中，在信使中接收消息的声音“压力更大，更鼓励用户采取行动”。 当您收到一封新信时，“您仍然想知道已经有东西来了，但是声音比较平静。”

他称自己的创作为音乐吗？

贝内特说：“从某种意义上来说，它们就像音乐一样，具有旋律，节奏和和谐感。” “它们听起来丰富而富有音乐感，而不是音乐本身。” 我们不希望用户像Windows 8那样想象音乐家演奏电子音乐并将某些情感带入其中。与Windows 7相比，这是一大进步，但还不成熟。 现在，尽管声音风格仍然是电子的，但您不会认为有一个小家伙坐在计算机内部。 同时，灵魂在计算机上被感觉到，只有在声音的一瞬间，它才能栩栩如生。 这是我们的主要想法之一。”

音乐概念在贝内特的作品中当然起着重要的作用。

他解释说：“错误消息旋律中使用了次要非和弦（m9），因为它略有不一致，所以说：确实需要注意这些词，”他解释道。

只有少数IT公司聘请Bennett等出色的作曲家。 没有专业协会或一群志趣相投的专家来交流和讨论当前问题。

Bennett说：“声音设计略落后于工业，硬件和图形。” -声音工程师与游戏以外的产品设计团队没有密切关系。 微软是最早聘请具有直接产品经验的音响工程师的公司之一。”

Bennett认为，声音设计不仅应影响微软和其他IT公司，而且应影响整个世界。 2018年2月9日，《纽约时报》发表了一篇文章，谈到了无处不在的现代电子设备产生的低音：“打雷，需要引起注意，无机噪声正成为一种规范； 分散注意力的声音信号会像巴甫洛夫的狗一样在我们体内产生反射。”

Bennett听到了。


Microsoft Production Studios的Matthew Bennett

他说：“仅Windows声音一天就发出数亿次声音。” “这些声音很多，即使持续不到一秒钟，它们也会影响许多人的生活。” 它们会影响情绪和幸福，可能是有益的，也可能是有害的。”

世界卫生组织已经认识到，意外的大声噪音会导致体内危险的皮质醇激增。 另外，不必要的声音和过大的音量是噪声污染的形式之一。

“在热带雨林中，有许多声音传达着信息-鸟的颤音和风声。 但是这些是很容易感知的自然声音。 这是一个强有力的隐喻，显示了我们应该如何与声音设计联系起来。”

在谈话结束时，我向Matthew坦白说，自1990年左右以来，我一直在静音Windows计算机上的系统声音。 我认为，它们不是必需的，只是令人讨厌。

我问他是否从中失去了一些东西-也许没有注意到使用操作系统的一些明显方面。

他回答说：“声音可以增加互动的深度和美感。 我们想告诉人们您现在可以再次打开声音。 我们会很尊重您，我们不会唤醒您的孩子，当您在图书馆或开会时，我们不会打破沉默。 这不再是随机噪声。 这些是经过精心选择的声音，可以在您的周围自然地发出声音。”

轻柔的提醒（而不是后记）


马修·贝内特（Matthew Bennett）制作Windows 10日历警报声音


许多人对日历的声音感到恼火，因为每种声音都意味着您需要做一些事情。 甚至有人认为这些声音就像火灾警报一样，因此您需要在某个地方运行，而不是一次，而是整天运行。 我们希望警报声音能使人振奋，但又不要打扰。 我们需要找到适当的乐观和精力来促使采取行动，但又不必为此担心。

应该以低音量听到这种声音，应该感觉而不是听到。 如果您聆听，您可以理解这些是相等间隔的七个音调。 旋律非常安静，平稳地开始，并且对于用户界面而言持续时间足够长-2.5秒。 但是在正常音量下，您实际上听不到一半以上的声音，因为前三个音几乎听不到。 他们听起来像呼吸，像是音乐的暗示，暗示着现在将会发生某种事情。 在那之后，音量增加并且声音变得更加可听。 最后有一个衰减的混响，当您远离声源时可以听到。



这是很长的声音，但非常开放。 它会引起相当长的但不会引起不适的感觉。 它轻盈，宜人，美丽。

通过立体声通道分离良好的耳机或扬声器，您可以听到从左到右的运动。 听起来很令人信服和自然。

这种单件声音由几层组成：有前景，中间和背景。 前景是电子弹拨和音调的打击乐声音。 它们不像真正的乐器，但看起来并不完全陌生。 它们没有被记录，而是使用数字技术建模。

在这种简短的旋律中（与Windows 10中的许多其他旋律一样），感觉到了三重奏。 所有这些似乎最和谐，最舒缓的旋律都基于三醇节奏，这是我们多年以来一直习惯的。

我们试图确保我们的声音不像计算机程序或人。 它们必须是复杂而自然的。 就像我们操作系统的呼吸。 没有这些声音，她将不会有这种和谐的感觉。
