Novo terminal do Windows: respostas às suas perguntas

Nos comentários de um artigo recente , você fez muitas perguntas sobre a nova versão do nosso Terminal Windows. Hoje vamos tentar responder a alguns deles.

Abaixo do corte estão algumas das perguntas mais frequentes que ouvimos (e ainda ouvimos), bem como respostas oficiais: incluindo a substituição do PowerShell e como começar a usar o novo produto hoje.



Quando e onde posso obter um novo Terminal do Windows?


  1. Você pode clonar o código-fonte do terminal no GitHub em github.com/microsoft/terminal e compilá-lo no seu computador.
    Nota : certifique-se de ler e seguir as instruções na página README no repositório antes de tentar construir o projeto - para construir o projeto, você precisa seguir algumas pré-condições e etapas de inicialização!
  2. A versão de pré-visualização do terminal estará disponível para download na Microsoft Store no verão de 2019.

Nós nos esforçamos para lançar o Windows Terminal v1.0 até o final de 2019, mas, ao mesmo tempo, trabalharemos com a comunidade para lançar esta versão para que o terminal seja de alta qualidade.

O Windows Terminal substitui o prompt de comando e / ou o PowerShell?


Para responder a essa pergunta, vamos esclarecer alguns termos e conceitos:

  • O prompt de comando e o PowerShell (por exemplo, WSL / bash / etc. no * NIX) são shells, não terminais e não possuem sua própria interface do usuário
  • Quando você executa a ferramenta shell / aplicativo / linha de comando, o Windows inicia e os conecta automaticamente às instâncias do console do Windows (se necessário)
  • O Windows Console é um aplicativo de interface do usuário padrão, semelhante ao terminal, fornecido com o Windows e tem sido usado pelos usuários nos últimos 30 anos para executar ferramentas de linha de comando no Windows NT, 2000, XP, Vista, 7, 8 e 10



Portanto, a pergunta provavelmente deve ser reformulada como "Terminal do Windows - substitui o Console do Windows?"

A resposta é "Não":

  • O Windows Console continuará sendo enviado ao Windows por décadas para fornecer compatibilidade com muitos milhões de scripts, aplicativos e ferramentas de linha de comando existentes / herdados
  • O Terminal do Windows funcionará com o Windows Console, mas provavelmente se tornará a ferramenta preferida para usuários que desejam executar ferramentas de linha de comando no Windows
  • O Terminal do Windows pode se conectar ao Prompt de Comando e ao PowerShell, bem como a qualquer outro aplicativo de shell / ferramenta / linha de comando. Você pode abrir guias independentes conectadas ao prompt de comando, ao PowerShell, ao bash (via WSL ou ssh) e a qualquer outro shells / ferramentas de sua escolha

Quando posso obter uma nova fonte?


Em breve! Não temos uma linha do tempo definida, mas estamos trabalhando ativamente na conclusão da fonte. Quando estiver pronto para o lançamento, ele estará aberto e disponível em seu repositório.
