Menginstal ulang Windows 8 prainstal pada akhir 2018

Artikel tentang cara mengembalikan windows 8 berlisensi pada laptop lama tempat HDD mati.

Banyak yang bisa diperoleh dari artikel 2013 .
Namun, waktu telah membuat penyesuaiannya sendiri.

Petunjuk langkah demi langkah (bagi mereka yang, seperti saya, mengatur ulang Windows terakhir kali sekitar 10 tahun yang lalu) di bawah potongan. Saya harap ini membantu seseorang menghemat beberapa jam yang saya habiskan untuk mencari 8.0 CoreSingleLanguage di torrent.


1. Anda harus mendapatkan kunci windows dari BIOS (harap jangan menilai secara ketat untuk terminologi).
Ada banyak informasi. Saya mengambil keuntungan dari ProduKey . Lainnya dengan referensi (mengingat bahwa kami akan mengunduh gambar disk, maka malware kemungkinan tidak akan lepas kendali pada kami).

2. Anda perlu memahami versi windows apa itu.
Tautan sebelumnya menyebutkan program ShowKeyPlus . Tersedia di toko janda (saya punya laptop utama dengan 10koy).
Dalam kasus saya, kuncinya adalah dari Win 8 RTM CoreSingleLanguage OEM: DM .

3. Unduh distribusi yang sesuai.
Pada 2018, ini dilakukan seperti ini . Yaitu segera unduh 8.1 Bahasa Tunggal .

4. Kami menulis gambar pada flash drive.
Saya menggunakan rufus-3.1p.exe (hanya karena di Windows ubunta tidak melihat flash drive dan dd tidak akan berfungsi).

5. (Opsional) Tulis HANYA file PID.txt (keberadaan ei.cfg menyebabkan kesalahan)
[PID]
Nilai = ZZZZZ-ZZZZZ-ZZZZZ-ZZZZZ-ZZZZZ

Mereka mengatakan bahwa rilis yang sesuai akan membaca nilai dari BIOS itu sendiri, atau meminta Anda untuk memasukkannya.

6. Boot ke flash drive dan instal OS.

Pembaruan : di kamenty merekomendasikan menggunakan alat pembuatan media sebagai gantinya. Siapa pun yang mengikuti jejak saya dapat mencoba mempersingkat jalan. Khusus menghancurkan Windows untuk memeriksa, saya belum siap :)
