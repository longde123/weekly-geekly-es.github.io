Lanzado Kali Linux 2019.1



Hubo un Kali Linux 2019.1 de lanzamiento continuo, que contiene muchas actualizaciones: paquetes actualizados, kernel actualizado 4.19.13, herramientas actualizadas, así como la versión 5 del famoso marco Metasploit.


Kali Linux es una distribución que contiene muchas herramientas para pruebas de penetración, desde analizar vulnerabilidades de aplicaciones web hasta piratear redes y servicios y repararlos en el sistema. Esta distribución se conocía anteriormente como Backtrack.


Que hay de nuevo


La característica principal del lanzamiento es, por supuesto, el soporte para la versión 5 de Metasploit Framework , la actualización más global del marco diseñado para pruebas de penetración. Vale la pena señalar que en las ramas de desarrollo, este soporte se ha implementado desde el año pasado, pero solo entró en la versión estable ahora. 


root@kali:~# msfconsole , , / \ ((__---,,,---__)) (_) OO (_)_________ \ _ / |\ o_o \ MSF | \ \ _____ | * ||| WW||| ||| ||| =[ metasploit v5.0.2-dev ] + -- --=[ 1852 exploits - 1046 auxiliary - 325 post ] + -- --=[ 541 payloads - 44 encoders - 10 nops ] + -- --=[ 2 evasion ] + -- --=[ ** This is Metasploit 5 development branch ** ] msf5 >

Kali Linux 2019.1 también incluye paquetes actualizados para Harvester, DBeaver y más. Una lista completa de actualizaciones, parches y complementos está disponible en el Registro de cambios de Kali Bug Tracker.


Por supuesto, las imágenes populares de ARM no se olvidaron: en Kali 2019.1, ARM devolvió el soporte para Banana Pi y Banana Pro, que se basan en el kernel 4.19. No hay imágenes de Raspberry Pi separadas para usuarios con pantallas TFT LCD, el script re4son kalipi-tft-config ahora está disponible en todas ellas, así que si desea configurar la placa TFT, ejecute 'kalipi-tft-config' y siga las instrucciones.


Instalación y actualización


Puede descargar la imagen final de un disco o máquina virtual desde el sitio oficial o actualizar su distribución.


Primero, verifique la relevancia de los repositorios: 


 root@kali:~# cat /etc/apt/sources.list deb http://http.kali.org/kali kali-rolling main non-free contrib

Además, antes de actualizar (apt -y full-upgrade), es posible que deba reiniciar: 


 root@kali:~# grep VERSION /etc/os-release VERSION="2019.1" VERSION_ID="2019.1" root@kali:~# root@kali:~# uname -a Linux kali 4.19.0-kali1-amd64 #1 SMP Debian 4.19.13-1kali1 (2019-01-03) x86_64 GNU/Linux

El último paso es actualizar la distribución: 


 root@kali:~# apt update && apt -y full-upgrade
