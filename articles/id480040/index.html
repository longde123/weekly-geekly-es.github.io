Saat ini, Internet Rusia benar-benar meledak dari berita pencarian di kantor Nginx Moskow, sebuah perusahaan IT terkenal di dunia dengan akar Rusia. Setelah 15 tahun, Grup Rambler tiba-tiba teringat bahwa mantan karyawan perusahaan, programmer Igor Sysoev, telah mengembangkan perangkat lunak di seluruh dunia untuk mengelola server web. Menurut berbagai sumber, Nginx diinstal pada sepertiga dari semua server web dunia, dan perusahaan itu sendiri dijual pada bulan Maret tahun ini ke American F5 Networks seharga $ 670 juta.
