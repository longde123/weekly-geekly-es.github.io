[公告] 1月底的《 Global Game Jam 2020》

每年的1月最后一个周末，每年都会举行“全球游戏大赛”。


今年的活动将从星期五晚上（1月31日）到星期日晚上（2月2日）举行，这将是本月最热的周末！

Global Game Jam是一项致力于创造游戏的活动（万一某个人突然不知道是什么游戏）。 该活动是离线进行的，只有在官方场地之一参加活动的人员才能接受工作。

果酱在许多国家发生。 完整的站点列表可以在这里找到。
