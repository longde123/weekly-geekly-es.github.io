WISE-PaaS - platform cloud untuk berbagai hal industri Internet



WISE-PaaS - (Platform sebagai Layanan) adalah platform cloud Advantech untuk hal-hal industri Internet, menggabungkan berbagai alat untuk mengumpulkan, memproses, dan memvisualisasikan data, otomasi, mengelola perangkat dan sistem kecerdasan buatan dan pembelajaran mesin. Platform ini menggabungkan beberapa set solusi perangkat lunak yang siap pakai untuk membangun sistem yang kompleks di industri, perawatan kesehatan, manufaktur, logistik, dll.

Platform WISE / PaaS dapat berjalan pada kapasitas penyedia cloud Amazon Web Services (AWS), Google Cloud Platform (GCP) dan Microsoft Azure, serta secara lokal di OpenStack.

Artikel ini membahas beberapa produk perangkat lunak WISE / PaaS yang memungkinkan Anda untuk dengan cepat mengembangkan aplikasi tanpa terganggu dengan membangun infrastruktur. Ini mendukung aplikasi yang sedang berjalan dalam bahasa populer: Java, .NET, Ruby on Rails, Node.js, Grails, Scala on Lift, Python PHP, serta mesin database MySQL utama, MongoDB, PostgreSQL, Redis, RabbitMQ. Untuk visualisasi data, kerangka Grafana tersedia. Seperangkat alat perangkat lunak terpisah untuk sistem tertanam menyederhanakan pengaturan awal perangkat, memperbarui firmware, dan mengumpulkan data dari perangkat IoT.

Aplikasi Marketplace


Platform ini merupakan konstruktor dari berbagai alat yang dapat dibeli sesuai kebutuhan di toko aplikasi . Di antara solusi yang diusulkan tersedia produk Advantech sendiri dan produk mitra. Masa percobaan gratis tersedia untuk beberapa produk Advantech.

Solusi perangkat lunak di pasar dibagi menjadi beberapa kategori besar tergantung pada tujuannya:

  • Sistem manajemen aplikasi cloud - orkestra, pemantauan, cadangan
  • Manajemen perangkat akhir - memperbarui firmware, mengumpulkan data dari perangkat, memantau, konfigurasi otomatis
  • Keamanan - sistem deteksi dan pencegahan intrusi (IPS / IDS), antivirus
  • WISE-Paas / AFS - sistem kecerdasan buatan, pembelajaran mesin dan visi mesin.



Untuk pengujian gratis, Anda harus mendaftar di portal WISE / PaaS dan mendaftar untuk berlangganan uji coba. Untuk melakukan ini, pilih produk yang diminati di pasar dan klik Mulai Uji Coba.



Alamat portal untuk memasuki sistem akan tergantung pada pusat data yang dipilih selama pendaftaran. Pusat data Azure saat ini tersedia (Hong Kong, Beijing), Alibaba Cloud (Hangzhou).

Alamat masing-masing titik masuk:

wise-paas.io (Azure HK2)
wise-paas .cn (Azure BJ)
wise-paas.cn (Alibaba)

Masuk ke panel kontrol harus dilakukan dengan rincian yang diterima melalui email setelah masa percobaan selesai.

WISE-PaaS / Dashboard


WISE-PaaS / Dashboard - satu set alat untuk memvisualisasikan data berdasarkan kerangka kerja Grafana . Biasanya digunakan untuk membuat grafik, bagan dan tampilan visual dari proses yang terjadi dalam waktu yang lama. Selain tujuan industri, dapat digunakan untuk memantau proses iklim di rumah pintar dan sistem perawatan kesehatan.


WISE-PaaS / Platform Visualisasi Data Dashboard

Widget


Kerangka kerja Grafana memiliki banyak opsi untuk menampilkan data: tabel, grafik, grafik, peta panas, dan banyak lagi. Anda dapat membuat dasbor lengkap untuk menampilkan berbagai data dengan hampir tanpa keterampilan pemrograman, Anda dapat menambahkan widget dengan mouse.


Antarmuka untuk menambahkan widget grafana ke dasbor

Selain widget bawaan, Anda dapat memasang plugin pihak ketiga untuk integrasi dengan sistem lain. Misalnya, plug-in untuk sistem pemantauan Zabbix memungkinkan Anda untuk mengimpor data darinya dan menampilkan pemberitahuan sistem pemantauan.

Dengan demikian, WISE-PaaS / Dashboard memungkinkan Anda untuk menggabungkan data dari berbagai sumber untuk menampilkannya dalam satu panel.


Data dari sistem pemantauan Zabbix dalam antarmuka Grafana

Sumber data


Dasbor dapat menerima data untuk ditampilkan dari berbagai sumber. Database yang didukung saat ini: CloudWatch, Elasticsearch, Graphite, InfluxDB, MySQL, OpenTSDB, PostgreSQL, Prometheus, RMM-SimpleJson, SCADA-SimpleJson, SimpleJson. Selain basis data ini, Anda dapat mengonfigurasi format kueri apa pun untuk mengumpulkan data dari berbagai sumber. Dataset uji juga tersedia untuk menjelajahi sistem.


Grafana mendukung berbagai sumber data

Pemberitahuan Anomali


Untuk menanggapi peristiwa tertentu, Dashboard memungkinkan Anda mengonfigurasi berbagai notifikasi. Ini bisa berupa panggilan API otomatis atau pemberitahuan operator. Ini sangat berguna saat membuat meja kontrol untuk memberi tahu operator tugas tentang perubahan tidak normal. Sebagai pemicu, kelebihan atau penurunan level tertentu, nilai rata-rata untuk periode tertentu, kurangnya data, dll. Dapat diatur.


Buat pemicu baru dan tambahkan bilah notifikasi

Untuk menampilkan pemberitahuan, ada widget terpisah "Lansiran", yang dapat menampilkannya di panel yang sama dengan bagan.

WISE-PaaS / SaaS Composer


SaaS Composer adalah kerangka kerja untuk membangun skema proses teknologi dua dan tiga dimensi. Tidak seperti skema usang klasik, ini memungkinkan Anda untuk membuat model yang lebih informatif dan visual dari proses yang sedang berlangsung. Visualisasikan seluruh jalur produksi dan bangunan dalam 3D, tampilkan peristiwa waktu nyata dalam model tiga dimensi.



Fungsi utama:

  • Antarmuka web pada HTML 5 Canvas. Performa bagus tanpa perlu menginstal perangkat lunak tambahan untuk melihat diagram.
  • Pembuatan model 2D dan 3D. Impor model 3D dalam format .OBJ + .MTL
  • Dukungan untuk gambar JPEG, PNG, SVG, OBJ, MTL. Dukungan untuk grafik vektor SVG saat membangun model 3D. Anda dapat mengimpor gambar dari skema yang ada.
  • Tambahkan animasi dan tampilkan data di atas objek 3D
  • Dukungan bahasa scripting
  • Integrasi dengan produk-produk WISE-Paas lainnya, khususnya WISE-PaaS / Dashboard

Perancang model 3D memiliki perpustakaan komponen utama: pipa, katup, kabel, motor, peralatan mesin, kisi-kisi dan banyak lagi. Di dalamnya, Anda dapat membuat model realistis objek nyata dan menambahkan widget dengan data.



Demo gedung Advantech yang menunjukkan status sistem daya waktu-nyata, konsumsi energi di lokasi berbeda, kondisi udara: tingkat CO2, partikel halus di udara, dll


Demo yang dibuat menggunakan SaaS Composer menunjukkan status berbagai parameter di gedung.

WISE-PaaS / APM


Manajemen Kinerja Aset - dirancang untuk meningkatkan kontrol atas efisiensi jalur produksi untuk peramalan, penilaian risiko, dan kontrol volume produksi yang lebih akurat.

WISE-PaaS / APM memiliki algoritme internal untuk analisis proses produksi yang memungkinkan Anda melacak mesin mana yang tidak bekerja secara efisien, untuk memprediksi volume produksi, kemungkinan masalah dan kebutuhan untuk itu. layanan.


WISE-PaaS / APM memungkinkan Anda untuk mengontrol efisiensi jalur produksi

WISE-PaaS / EnSaaS - Bekerja dengan perangkat (Edge to Cloud)


Untuk dengan mudah mengintegrasikan perangkat akhir ke dalam infrastruktur cloud, WISE-PaaS menawarkan seperangkat alat untuk bekerja dengan sistem tertanam dan IoT.

WISE-Paas / DeviceOn adalah platform untuk mengelola dan mengkonfigurasi sejumlah besar perangkat akhir, seperti sensor, terminal, sistem tertanam dan banyak lagi.


Fungsi utama:

  • Penyediaan tanpa sentuhan - konfigurasi otomatis peralatan akhir dan menambahkannya ke sistem
  • Pembatasan akses - untuk memastikan keamanan perangkat dan mencegah akses yang tidak sah
  • Perbarui (OTA) —otomatis memperbarui perangkat lunak dan firmware pada perangkat akhir
  • Pemantauan - pemantauan status peralatan dan pemberitahuan masalah melalui pemberitahuan push, SMS atau E-Mail
  • Cadangan dan pengarsipan - membuat cadangan konfigurasi perangkat dan datanya
  • Membangun peta perangkat - konstruktor untuk membangun tata letak perangkat pada rencana bangunan, dan pada peta

WISE-Paas / WISE-Agent


WISE-Agent - perangkat lunak yang diinstal pada perangkat akhir untuk berinteraksi dengan WISE-PaaS / DeviceOn. Semua sistem operasi utama didukung. Paket yang dikompilasi tersedia untuk Windows, Ubuntu, Android (RISC), OpenWRT (RISC).
Interaksi dengan platform cloud terjadi sesuai dengan protokol MQTT.

[Kasus] Menggunakan platform WISE-PaaS dalam pengembangan kapal pintar


Perusahaan SaierNico - sedang mengembangkan sistem cerdas untuk perusahaan pelayaran dan mengembangkan peralatan untuk kapal. Dengan menggunakan platform Wise-PaaS, SaierNico telah mengembangkan sistem pemantauan dan respons insiden kapal jarak jauh secara real-time.



Sensor mengumpulkan data dari berbagai komponen kapal: kecepatan mesin, tekanan, kondisi sistem pendingin udara, pompa dan lain-lain. Untuk transfer data, broker RabbitMQ digunakan, yang meningkatkan keandalan pengiriman pesan, karena komunikasi dengan kapal tidak stabil. Data mengalir ke sistem WebAccess / SCADA.

Arsitektur sistem


WISE-PaaS / APM digunakan untuk melacak kinerja komponen kapal.
Visualisasi data untuk pusat kontrol didasarkan pada WISE-PaaS / Dashboard dan WISE-PaaS / SaaS Composer .

Pembaruan firmware perangkat akhir dalam sistem kapal dilakukan menggunakan WISE-PaaS / OTA .
