LightBlue Bean +: Arduino Bluetooth pour l'ère du mobile



LightBlue Bean + est une carte compatible Arduino de la famille LightBlue Bean - une gamme de cartes Arduino sans fil faciles à utiliser. Il y a un BLE, 16 GPIO, connecteurs sans soudure. Les développeurs proposent également une batterie et une large gamme de logiciels.

Vous pouvez programmer sous Bean + sur n'importe quelle plateforme - OS X, Windows, iOS, Android. Toutes les fonctions de contrôle sont exécutées sans fil. Pas de fils, pas de problèmes.

Caractéristiques principales de Bean +


Cela inclut plusieurs nouvelles fonctionnalités BLE: balise, MIDI, HID, ANCS et observateur.



Sur la base du tableau, vous pouvez créer un assistant intelligent qui affiche les notifications des appareils couplés, d'un instrument MIDI, d'un clavier, d'une souris ou de toute autre chose.

Configuration du fer




Bean + possède plusieurs modules intégrés - par exemple, un accéléromètre et un capteur de température. L'accéléromètre peut être utilisé pour créer des maracas MIDI ou pour recevoir une notification lorsqu'il devient trop chaud dans le terarium installé à la maison.

Le rayon d'action accru


Avec le nouvel amplificateur RF, Bean + peut interagir avec tout autre Bean + dans un rayon de 400 mètres, dans des conditions idéales. Ou est-ce à 250 mètres pour interagir avec un smartphone. Ainsi, la portée de l'appareil est suffisante pour interagir avec n'importe quel appareil de l'appartement ou de la maison. Par exemple, vous pouvez activer la musique dans le hall sans sortir du lit.



Batterie


Pour chaque Bean +, une batterie de 600 mAh est fournie, dont la charge est suffisante pour environ un an de fonctionnement de la carte en mode autonome. Le port micro-USB ajouté vous permet de charger l'appareil partout où il y a accès à l'électricité. Vous pouvez également utiliser le panneau solaire pour la charge, ce que le "Dadget" a déjà décrit sur Geektimes.



Bean + convient également aux projets qui nécessitent un courant plus fort, par exemple, pour les systèmes avec servo-variateurs ou LED haute puissance.

Accessoires optionnels


Bean + est livré avec deux types de connecteurs, y compris les connecteurs Grove développés par Seeed Studio.



Soit dit en passant, vous pouvez choisir la tension de fonctionnement - vous pouvez choisir 3,3 V ou 5 V.

En principe, Bean + est une option idéale pour les projets associés aux appareils mobiles. Si vous n’avez pas envie d’écrire du code, vous pouvez travailler avec LightBlue, une plate-forme logicielle qui mérite d’être utilisée ici.

Pour la commodité des développeurs, la société fournit également les SDK officiels iOS, OS X et Android.
