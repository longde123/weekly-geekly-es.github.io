Ich denke, die meisten werden zustimmen, dass das wichtigste Kriterium für einen guten Schulungsort nur ein Indikator ist - der Wunsch, zum Standort zurückzukehren und das Training fortzusetzen.

| Wirksamkeit


Eine effektive Demonstration dessen, was genau „auf diese Weise“ und nicht anders getan werden muss.


Wenn die Aufgabe beispielsweise erklärt, warum keine Hardcodierung erforderlich ist (unter Verwendung manuell gesteuerter Werte anstelle von Variablen), wird dies nicht nur erklärt, sondern auch als Beispiel gezeigt (etwas, das fast allen anderen ähnlichen Diensten so sehr fehlt), und dies geschieht wie folgt Weg:
- Zuerst werden Sie in der ersten Aufgabe aufgefordert, einige Parameter fest zu codieren.
- Danach werden mehrere Parameter im Code manuell für andere Werte festgelegt (da sich im wirklichen Leben häufig Änderungen ergeben, z. B. eine Änderung des Produktpreises).
- Dann zeigen sie die Methode, die anstelle der Hardcodierung verwendet werden soll, damit Sie nach den Änderungen nicht manuell eine Aktion nach der anderen wiederholen müssen
„Danach müssen Sie nicht erklären, warum es genau so gemacht werden muss, wie es akzeptiert wird, weil Ihnen gerade anhand eines Beispiels gezeigt wurde, wie viel bequemer die„ richtige Methode “ist und warum sie als richtig angesehen wird.

Einfach aber genial. Meiner Meinung nach ist es sehr wichtig zu verstehen, warum wir das tun, was wir im Code tun, und warum genau dies der Weg ist und nicht anders, um sich an das Wesentliche von Handlungen zu erinnern und es zu verstehen. Daher ist Code Avengers meiner Erfahrung / Meinung nach von allen Dienstleistungen für das Programmierstudium derzeit die beste.
