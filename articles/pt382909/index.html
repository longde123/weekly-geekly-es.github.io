LightBlue Bean +: Arduino Bluetooth para a era móvel



O LightBlue Bean + é uma placa compatível com Arduino da família LightBlue Bean - uma variedade de placas Arduino sem fio fáceis de usar. Há um conector BLE, 16 GPIO, sem solda. Os desenvolvedores também oferecem uma bateria e uma ampla gama de software.

Você pode programar no Bean + em qualquer plataforma - OS X, Windows, iOS, Android. Todas as funções de controle são realizadas sem fio. Sem fios, sem problemas.

Principais Recursos Bean +


Isso inclui vários novos recursos de BLE: beacon, MIDI, HID, ANCS e observador.



Com base no quadro, você pode criar um assistente inteligente que exibe notificações de dispositivos emparelhados, instrumento MIDI, teclado, mouse ou qualquer outra coisa.

Configuração de ferro




O Bean + possui vários módulos embutidos - por exemplo, um acelerômetro e um sensor de temperatura. O acelerômetro pode ser usado para criar maracas MIDI ou para receber uma notificação quando ficar muito quente no terário instalado em casa.

O raio de ação aumentado


Com o novo amplificador de RF, o Bean + pode interagir com qualquer outro Bean + em um raio de 400 metros, sob condições ideais. Ou são 250 metros para interagir com um smartphone. Assim, o alcance do dispositivo é suficiente para interagir com qualquer dispositivo no apartamento ou na casa. Por exemplo, você pode ativar a música no lobby sem sair da cama.



Bateria


Para cada Bean +, é fornecida uma bateria de 600 mAh, cuja carga é suficiente para cerca de um ano de operação da placa no modo autônomo. A porta micro-USB adicionada permite carregar o dispositivo em qualquer lugar onde haja acesso à eletricidade. Você também pode usar o painel solar para carregar, que o "Dadget" já descreveu no Geektimes.



O Bean + também é adequado para projetos que precisam de uma corrente mais forte, por exemplo, para sistemas com servoconversores ou LEDs de alta potência.

Acessórios opcionais


O Bean + vem com dois tipos de conectores, incluindo os conectores Grove, desenvolvidos pela Seeed Studio.



A propósito, você pode escolher a tensão operacional - você pode escolher 3.3V ou 5V.

Como mencionado acima, você pode escrever software para o módulo a partir de qualquer sistema operacional, incluindo Android. Você pode usar o padrão IDE do Arduino e em tempo real.

Em princípio, o Bean + é uma opção ideal para projetos associados a dispositivos móveis. Se você não gosta de escrever código, pode trabalhar com o LightBlue, uma plataforma de software que vale a pena usar aqui.

Para conveniência dos desenvolvedores, a empresa também fornece SDK oficial para iOS, OS X e Android.

A interação entre Bean +


Sim, os painéis podem "conversar" com outros painéis do Bean +. Isso é possível graças ao BLE e ao módulo observador.
