Microsoft akan merilis versi Skype untuk freelancer



Microsoft akan memperkenalkan versi khusus Skype, yang ditargetkan untuk pemilik usaha kecil dan freelancer. Dalam waktu dekat ini direncanakan untuk merilis versi awal di Amerika Serikat, menurut blog Skype . Apa yang disebut dengan Akun Profesional Skype (Akun Profesional Skype) akan, di samping telepon dan pesan standar, termasuk kalender, catatan, situs web bisnis yang terpisah, dan semacam sistem pembayaran internal.

Microsoft berharap bahwa fungsionalitas baru akan bermanfaat bagi mereka yang menggunakan Skype untuk konsultasi dan pengajaran online. Katakanlah tutor akan dapat berkomunikasi, mengirim file, menjadwalkan janji temu, membuat catatan, dan menerima pembayaran dalam satu alat tanpa beralih ke program pihak ketiga. "Penjadwalan pintar" tertentu dijanjikan. Pengguna bahkan akan menerima "situs web terpisah untuk bisnis kecil", yang akan dibuka langsung di Skype. Di sana Anda dapat menunjukkan mode operasi dan berbagai penawaran khusus.

Sudah sekarang Anda dapat mendaftar di halaman Microsoft dan menjadi penguji. Akun Skype Professional diposisikan sebagai gratis, tetapi sulit untuk mengatakan apakah akan tetap demikian setelah peluncuran resmi atau, katakanlah, akan menjadi bagian dari Office 365. Korporasi tidak terburu-buru untuk menceritakan semua inovasi - dicatat bahwa akan ada "fitur luar biasa" lainnya yang nanti

Untuk mendaftar, cukup isi kuesioner dengan pertanyaan tentang apa bisnis Anda dan bagaimana Anda menggunakan Skype. Formulir pendaftaran terbuka untuk semua orang, saya mengisi dan mengirimkannya tanpa masalah, sama sekali tidak tinggal di AS, tetapi kemudian hanya mengatakan: "Terima kasih atas jawabannya." Tampaknya, Anda perlu menunggu sampai seseorang di Microsoft memutuskan bahwa Anda cocok untuk berpartisipasi. Blog memperingatkan bahwa jumlah tempat terbatas.
