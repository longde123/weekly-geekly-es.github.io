Microsoft meluncurkan inisiatif pendidikan skala besar di universitas-universitas Rusia

Dalam kerangka Forum Ekonomi St. Petersburg, Microsoft di Rusia mengumumkan perluasan kerja sama dengan universitas-universitas Rusia terkemuka. Perusahaan akan membuka sejumlah program master di bidang teknologi yang relevan: kecerdasan buatan, pembelajaran mesin, data besar, intelijen bisnis, dan internet. Ini akan menjadi elemen pertama dari serangkaian inisiatif pendidikan yang Microsoft rencanakan untuk diterapkan di Rusia.

Sebagai bagian dari forum, Microsoft menandatangani Perjanjian Intent dengan salah satu peserta program - Sekolah Tinggi Ekonomi.

Jadi, di Moscow Aviation Institute, fokus utamanya adalah Augmented Reality dan teknologi AI, di RUDN University, mereka akan fokus pada teknologi kembar digital , layanan kognitif seperti visi komputer dan pengenalan suara untuk robot. Di Moskow State Pedagogical University, beberapa disiplin ilmu diluncurkan sekaligus, termasuk "Teknologi Jaringan Saraf dalam Bisnis" berdasarkan Microsoft Cognitive Services, "Pengembangan Aplikasi Internet" di Microsoft Azure Web Apps, dll. Di MGIMO, di mana program master " Inteligensi Buatan " diluncurkan setahun lalu dengan dukungan ADV Group dan Microsoft, kursus baru "Microsoft Artificial Intelligence Technologies" dibuka. Selain studi mendalam tentang teknologi AI, seperti, khususnya, pembelajaran mesin, pembelajaran mendalam, layanan kognitif, obrolan bot dan asisten suara, program ini mencakup disiplin pada transformasi bisnis digital, layanan cloud, blockchain, Internet of things, augmented dan virtual reality, serta komputasi kuantum.

Siswa dari semua program master akan memiliki kesempatan untuk menjalani praktik dalam format hackathons Microsoft, yang melibatkan penciptaan proyek secara real time dengan dukungan dan bimbingan para ahli teknologi dari perusahaan. Proyek-proyek ini selanjutnya akan dapat mengklaim status pekerjaan kualifikasi akhir.
