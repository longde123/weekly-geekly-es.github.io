Mulai di situs ini di tautan .

Pilihan paling nyaman untuk menggunakan informasi tentang menyalakan starter ternyata menjadi opsi dengan optocoupler PC817.

Diagram sirkuit

Papan berisi tiga sirkuit yang identik, semuanya ditempatkan dalam kotak yang terbuat dari plastik ABS, ukuran 100x100 mm.

Optocoupler foto

Ketika terhubung ke starter dengan katup semikonduktor, arus bocornya cukup untuk membuka PC817 dan akan ada operasi penghitung semu. Untuk menghilangkan situasi ini secara berurutan satu lagi ditambahkan ke rangkaian LED optocoupler dan LED indikasi operasi. Untuk melakukan ini, jumper J1 dibuka dan LED1 tambahan disolder.

papan tempat memotong roti terhubung ke ARDUINO MEGA 2560. Untuk ini, konektor dua baris di ujungnya digunakan. Layar dengan resolusi 240x400 digunakan sebagai perangkat tampilan informasi.

HX8352B.

Selain itu, konektor ke ICSP pada papan layar dibongkar dan slot micro SD tidak digunakan. Faktanya adalah bahwa slot SD "asli" tidak dapat digunakan karena konflik pada bus SPI. Pembaca kartu terpisah dengan stabilizer 3.3V dan chip buffer dengan tiga status output 74LVS125A digunakan untuk kartu flash. Di sini garu sedang menungguku. Buffer dengan tiga status, tetapi berfungsi baik E01-ML01DP5 atau pembaca kartu. Dalam komentar di perpustakaan SdFat, saya melihat peringatan tentang ketidakcocokan dengan perangkat lain. Konverter level pada TXS0108E telah dihapus dan diganti oleh jumper, sebagai E01-ML01DP5 toleran terhadap sinyal 5V - tidak membantu. Menggunakan osiloskop, kehilangan sinyal pada jalur MISO terdeteksi ketika pembaca kartu terhubung. Setelah pemeriksaan yang teliti, ditemukan bahwa input dari sinyal yang memungkinkan OE dari 4 saluran 74LVS125A disolder secara sederhana ke kabel biasa dan tidak ada pembicaraan tentang keadaan ketiga. Chip buffer digunakan sebagai konverter tingkat primitif dari 5V ke 3.3V menggunakan 3,3 KOhm resistor yang terhubung secara seri dengan saluran sinyal. Selain jalur MISO. Kunci output rendahnya mungkin menarik sinyal ke ground. Setelah menentukan bahwa sinyal aktif dari garis MISO adalah pin 13, itu robek dari trek dan

disolder

antara output input (9) 74LVS125A dari pemilihan perangkat CS dan resistor terminasi. Sekarang, jika tidak ada akses ke kartu memori, buffer MISO dinonaktifkan dan tidak mengganggu pengoperasian perangkat lain.

 /* Recode russian fonts from UTF-8 to Windows-1251 */ String utf8rus(String source) { int i,k; String target; unsigned char n; char m[2] = { '0', '\0' }; k = source.length(); i = 0; while (i < k) { n = source[i]; i++; if (n >= 0xC0) { switch (n) { case 0xD0: { n = source[i]; i++; if (n == 0x81) { n = 0xA8; break; } if (n >= 0x90 && n <= 0xBF) n = n + 0x30;//0x2F break; } case 0xD1: { n = source[i]; i++; if (n == 0x91) { n = 0xB8; break; } if (n >= 0x80 && n <= 0x8F) n = n + 0x70;//0x6F break; } } } m[0] = n; target = target + String(m); } return target; }
