Memperkenalkan Perpustakaan Akses Database SOCI - C ++

Entri


Perpustakaan itu sendiri cukup matang - rilis pertama pada tanggal github kembali ke tahun 2004. Saya terkejut ketika Habr di mesin pencari tidak memberi saya satu pun tautan ke artikel yang menyebutkan perpustakaan yang luar biasa ini.


Diucapkan sebagai: sosi , dengan penekanan pada suku kata pertama.


SOCI mendukung ORM , melalui spesialisasi sosi :: type_conversion .


Dukungan Database (DB) (backends):


  • DB2
  • Burung api
  • MySQL
  • ODBC
  • Oracle
  • PostgreSQL
  • Sqlite

Saya tidak akan menerjemahkan manual atau memberikan kode dari contoh di sini, tetapi saya akan mencoba untuk beradaptasi (dengan perubahan dalam struktur tabel, dan penyederhanaan lainnya) kode dari proyek masa lalu saya untuk membuatnya lebih visual dan menarik.


Instalasi


Kami mengunduh bahan baku dari cabang master , membongkar, dan di dalam direktori kami menjalankan perintah:


Di jendela


$ mkdir build && cd build && cmake -G "Visual Studio 15 2017 Win64" ../ && cmake --build. --config Release

atau alih-alih perintah terakhir, Anda bisa membuka proyek yang dihasilkan di Visual Studio dan kompilasi.
( Wilk diminta tentang membangun dengan cmake di baris perintah)


Di nix


$ mkdir build && cd build && cmake ../ && sudo install
