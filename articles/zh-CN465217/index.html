Acronis True Image 2020：新的复制方案和增强的保护

Acronis True Image的新版本已发布-一种针对个人用户的数据保护系统。 我们想谈谈True Image 2020与True Image 2019的区别。欢迎所有感兴趣的人关注。



Acronis True Image是面向个人用户的个人数据保护解决方案。 True Image包括备份和灾难恢复系统，数据身份验证工具以及反恶意软件工具。 每年八月底，我们都会推出True Image的新版本，今年，其中出现了许多附加功能，这些功能可以提高解决方案的生产率。 一些芯片将受到移动设备和Mac计算机用户的赞赏。

出现在商店货架上的Acronis True Image 2020已发布，其中涉及与保护系统各个部分相关的100多项改进。 其中一些是小的修正或优化，但是在我们的文章中，我们将仅关注最重要和最有趣的更新，这些更新可以使我们产品的用户的生活变得更好。

Acronis双重保护


该版本最有趣的功能之一是备份的复制。 复制是将最新的可用数据版本从备份转移到Acronis云。 结果，用户将具有两个备份：一个备份在本地介质（例如外部驱动器）上，另一个在云中。



设置好这种“双重”备份计划后，本地备份操作完成后，数据将自动复制到云中。 因此，可以保护用户免受意外数据丢失的影响，例如，由于用户具有“快速”本地备份而导致的意外删除文件，以及由于用户可以从云归档中恢复任何数据而导致的崩溃或设备丢失。



与两个连续的常规备份（本地存储和云备份）相比，Acronis双保护方案具有许多技术优势。 首先，用于复制的系统资源更少，因为您不必分析磁盘上的数据，无需考虑要备份的内容以及归档中已存在的内容。 其次，由于到云的备份通常要比本地备份花费更长的时间，因此应用程序必须保留文件系统的快照。 这是系统本身的负担，某些快照可能要等到操作结束后才能保留下来。 如果是副本，则不需要快照。

改进了归档和重复数据删除


归档机制本身已经得到了改进和改进，包括重复数据删除功能。 用户可以通过减少创建备份的时间以及减少归档文件本身的大小来注意到这一点。 此外，改进的体系结构允许您查看本地存档和云副本的内容，而无需额外的资源，直至按需下载单个文件。

改进的存档器的最新测试显示，性能显着提高。 因此，与先前版本的Acronis True Image 2019相比，在磁盘上创建增量版本的本地备份的时间减少了45％，而云备份的增量时间减少了80％。

新的通知和信息系统


除了对图形界面的整体改进（变得更加全面）之外，我们还添加了一个通知中心。 它使您可以获取有关备份过程的最新信息，以及有关反恶意软件系统的运行情况的报告，而无需进行不必要的转换。



通知面板显示最近的30个事件，以便用户可以跟踪过程的成功完成，或者相反，可以跟踪备份和复制期间的失败和错误。 现在，您无需进入“报告”，所有问题和错误都可以在一个地方看到，并且在必要时进行修复。

先进的勒索软件对抗


顺便说一句，新的通知面板还包含有关Active Protection（一种防止勒索软件的系统）的工作的信息。 现在，通知还显示有关哪些进程被阻止以及为什么被阻止的信息（这对于开发人员了解为什么他们的软件被错误地阻止的原因很有用），以及监视单个进程的原因。



主动保护系统本身已变得更加“敏感”。 由于AI的不断训练，我们大大减少了误报次数，并提高了引擎效率。 还进行了单独的改进，提高了Acronis系统进程对加密程序尝试阻止其工作的尝试的稳定性。

便携式设备的优化


由于大多数消费者不仅将Acronis True Image安装在固定设备上，而且还安装在便携式设备上，所以我们对笔记本电脑上的数据保护系统进行了许多改进。

以前，使用电池供电时，您只能关闭备份功能，因此用户必须在笔记本电脑的电池寿命和数据安全性之间进行选择。 现在，在设置菜单中，当您不再希望将资源花费在资源密集型的复制和归档过程上时，可以将“滑块”设置为特定的电池电量，而优先选择最长的移动工作。



另一个方面是选择用于备份到云的网络。 如果固定式PC一切正常，并且通过家庭或办公室Internet通道进行下载，则移动设备将连接到各种各样的无线网络。 新版本的Acronis True Image 2020允许您配置可以在其中传输备份数据的网络。 通常，为此使用家庭和办公室网络。 通过这种方法，例如，如果您连接到咖啡馆或机场中的网络，则可以避免在“弱”以及付费Wi-Fi上失败地使备份失败。 从通道保护的角度来看，无论如何都使用加密将信息传输到Acronis云，但是一些用户仍然担心在开放网络中进行数据传输。 对于他们来说，也要求能够限制使用的Wi-Fi连接列表。

MAC的高级安全性


Acronis True Image 2020引入了Apple计算机的其他功能。 因此，现在该工具包允许您在新计算机上完全还原APFS分区。 但最重要的是，Acronis现在支持Apple Power Nap。 Mac的这种“睡眠”模式涉及节能和效率的结合-也就是说，计算机进入睡眠状态，但同时继续执行诸如邮件传递，更新新闻提要之类的过程……现在，如果计划了该过程，则还可以执行备份。
