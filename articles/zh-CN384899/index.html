回顾bb-mobile Kalash LTE平板电脑-金属，4G / LTE和Android 5.1

最近，最新版本的Android成为了旗舰店和品牌Google Nexus设备的特权。而支持LTE的通信模块似乎是一种不可思议的可选芯片。但是，如今4G网络已经覆盖了俄罗斯的79个地区，发布后立即向许多公司提供了Android 5.1的访问权限。尽管它们都针对智能手机，但俄罗斯bb-mobile品牌于2015年9月初发布了Kalash LTE，这是首款最实惠的平板电脑，结合了Android 5.1和LTE支持。其他值得注意的特性是具有64位架构和IPS显示屏的新型4核处理器。同时，这款平板电脑属于预算型产品，价格为6,990卢布。



在iPad发行之前，平板电脑甚至没有普及的迹象。是的，曾经有一次基于Windows和Linux的模型，但是“出于某种原因”的设备并未出售。现在正好相反-基于Android和iOS的平板电脑已经取代了电子书，货架上的GPS导航器，并加速了个人计算机市场的萎缩。

但是，对我来说，在第一个经过测试的型号（2011年商用）中，存在明显的缺陷-便携式设备的10英寸有点大，重量很大，并且不支持电话。如今，它们都是过去的-7英寸型号的流行性正在增长，您经常可以称呼它们，这些型号的重量从280克到350克不等。 bb-mobile KALASH LTE就是这样一种设备，因此，我将尽力向您介绍该模型的所有优缺点。



俄罗斯的bb-mobile品牌作为婴儿电话，摄像头门和其他奇特产品的制造商而闻名。但自2014年初以来，他一直在开发基于Android或Windows的平板电脑。一年半以来，该品牌约有20款车型，其中一些会定期更新。因此，去年我们的英雄bb-mobile Kalash LTE已经有了一个前身-只是bb-mobile Kalash，他们不要混淆。除了通信模块外，它们之间的差异也很大-去年的型号具有更少的内存（512 MB对比1 GB），2核处理器，TFT-TN显示屏和Android 4.4。现在，卡拉什仍然可以3990卢布的价格出售。那么在过去的一年里，卡拉什发生了什么变化？

外观bb-mobile Kalash LTE


Kalash LTE型号看起来很不寻常-坚固的金属外壳立即引起您的注意，而预算员工并不熟悉塑料。摆在我们面前的不是像其他金属片那样薄的后盖，而是一块坚固的板，它可以穿透外壳周围的边缘。这种全金属外壳旨在提高可靠性。当然，这将大大降低秋天摔坏平板电脑的风险，但您不应刻意冒险-这不是构造工具，也不是切菜板。



也许是在这种情况下，平板电脑被命名为“ Kalash”。一个响亮的名字掩盖了俄罗斯制造的最耐用设备的寓言性描述。众所周知，同名的自动机器可以在任何条件下长期可靠地工作。 （但是，几乎所有其他bb-mobile平板电脑都封装在同一个公寓金属盒中，并且它们不同时被称为“ Kalash”。）



尽管有耐用的外壳，但重量还是不大-不到300克。您可以用一只手握住此类小工具，例如在通话过程中。尽管在横向模式下使用它要方便得多-背面有两个塑料插件。它们不仅使握持非常耐用，而且还改善了通信质量（Wi-Fi，3G / 4G或GPS信号无法穿过固体金属）。

下部的塑料插件仅包含一个扬声器，并随着其振动而悦耳动听，这无与伦比。扬声器对于此类设备足够响亮，并且适合观看电影，尽管最大音量下的声音质量不是最高。



上部塑料部件是可移动的，它是存储卡插槽和SIM卡的盖。最初，在我看来只有一个SIM卡插槽-这并不奇怪，因为该型号使用双插槽，其中一个存储卡位于另一个存储卡的顶部，而不是在其侧面。





一侧的锁定和音量按钮在右侧。它们旁边有一个小孔，可以称为“过去的幽灵”-这是隐藏在外壳中的重置按钮。如果您最近几年挂有某种官方固件的Android设备，请在评论中写下。也许我很误会，但是通过这种方式，四年来我不必重新启动任何操作。

显示


显示分辨率根本不是很高，只有1,024 x 600像素，就像最早的Android平板电脑一样。这还不够，但对于7英寸而言并不重要。从手工观察时，粒度很少见，但绝对是。另一方面，选择了TFT-IPS矩阵技术，从而可以精确地透射颜色并且提供宽视角。



即使到现在，许多型号仍旧使用过时的TFT-TN矩阵，它们在可怕的视角和较低的色彩还原方面有所不同。它们最明显的问题是颜色反转-颜色随视角的变化而变化。

bb-mobile Kalash LTE平板电脑的IPS矩阵没有这样的问题。倾斜时颜色不会改变，配色方案是自然的，并且颜色不会过饱和。在这样的显示器上阅读或观看电影非常舒适。

相机


拍摄质量低于平均水平，这并不奇怪-主相机的作用为2百万像素，而前置相机仅为0.3百万像素（640 x 480像素）。但是，在框架中获得清晰的文字并不难。我认为，这是预算平板电脑中相机的主要任务。













前置摄像头非常适合视频通话，即使在路上也是如此-不错，有4G LTE。但总的来说，这款平板电脑不是自拍的最佳解决方案，并且绝对不能替代相机。即使在晴天在街上用主摄像机拍摄，也有其缺点-明亮的物体会被稍微点燃。

绩效和工作经验


如果您已经听说过Android 5.0 / 5.1（Kalash LTE具有5.1），那么您将了解增强的性能和更适度的电池消耗。据说是在Android 4.4（相对于4.2）的情况下进行的，但是对于Android 5.0 / 5.1，所有的承诺都尽可能接近现实。首先，我们完成了Java Java机器ART的定稿，并最终删除了Dalvik。这允许应用程序和系统启动更快，消耗更少的RAM和处理器资源。

此外，新界面还包含Google的Material Design意识形态的元素。因此，该系统更易于阅读且速度更快。它比整个Android 4.x系列更简单，更漂亮。







5.1版更可能修复错误，但正是在此版本中，他们引入了对系统级别的两个SIM卡的支持。对于dvuhsimochny，“卡拉什”创新很重要。

可以通过引入对64位处理器的完全支持来提高新系统的性能。这就是Kalash LTE平板电脑配备的设备。联发科技MT8735M是基于Cortex-A53内核的4核SoC（片上系统），频率高达1400 MHz。在分辨率为1024 x 600的情况下，性能非常出色-菜单立即切换，游戏加载并快速启动。



1 GB RAM，因此任何游戏都可以使用，即使是最现代的游戏也可以。令人高兴的是，处理器中的视频芯片也是新的，因此支持OpenGL ES 3.0的游戏可以显示一些图形奇迹。但是大多数时候，由于处理器频率不是很高，游戏中的图形仍然必须设置为“中等”设置。

我对平板电脑和另一个功能（支持4G LTE）感到很满意。第四代网络已经占领了俄罗斯的79个城市，技术用户仍然很少。因此，从过载的3G网络切换时，差异是显而易见的。从表面上看，增加了三倍到五倍，但实际上，与可靠的3G连接相比，值得将速度提高两倍。当您连接到无限的Internet时，您可以在旅途中轻松下载电影。

来自移动通讯程序的视频通话已成为现实。如果在3G连接上，即使使用Skype / Viber进行语音呼叫，它也是一个薄弱的链接，那么在4G的情况下，您可以看到对话者，并且语音通信现在完全无法进行。方便家庭和工作-与家人和同事进行视频通话不再需要带有Wi-Fi的笔记本电脑。

此外，借助Kalash LTE，您不仅可以通过即时通讯程序拨打电话。还支持在移动网络上进行普通呼叫，并且效果很好。有一个扬声器，蓝牙耳机效果很好。使用7英寸平板电脑作为主手机会有些奇怪，但有可能。在此型号中，即使没有耳机也可以做到这一点。



无线连接也相当现代-蓝牙4.0和Wi-Fi b / g / n。还提供了完整的GPS支持，从而使平板电脑成为导航器的便捷替代品。幸运的是，Google Play Sea上的制图和导航程序还有Mail / Yandex的免费选项。7英寸显示屏可让您将设备安装在汽车的前面板上，并以“侧视”完美看到道路。是的，扬声器适合道路行驶-导航仪中的语音注释可以毫无问题地听到。



所有商品均配备3,000 mAh电池。在3D游戏中，最长可连续玩6个小时。观看视频将在8小时20分钟内耗尽电池。平板电脑经过一天或多或少的主动使用，可以毫无问题地承受。

发现


对于Android 5.1和4G LTE，很难找到具有类似规格的设备，尤其是在价格方面。著名公司的平板电脑价格昂贵得多，坦率的“中国”并不以担保和可靠性为特色。

因此，平板电脑的唯一缺点是显示分辨率为1024 x 600像素。但是，由于有了高质量的IPS矩阵，这种麻烦很快就被忘记了。其余部分是坚固的中端，坚固的金属外壳和良好的硬件填充物。

该平板电脑非常适合作为GPS导航仪或电子书的替代品。是的，功能强大的处理器与7英寸显示屏相结合，使该设备易于游戏。
