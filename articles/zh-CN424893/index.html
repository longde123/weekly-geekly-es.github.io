工匠为1989年的型号Macintosh SE / 30创建了一个WiFi模块。



许多极客都从事过时的或什至几乎是古董的设备的改装，这些设备曾经很流行，但是由于技术进步迅速而被遗忘了。 一种这样的设备是苹果的Macintosh SE / 30系统 。 有人称它为公司有史以来最好的计算机。

实际上，该系统的功能（大约稍低一些）令当代人的想象力惊叹。 因此，在30年前，此计算机支持128 MB的RAM容量。 由于其功能强大，该系统非常受用户欢迎，以至于许多粉丝并没有放弃道德上和物理上过时的计算机，而是继续使用它。 没错，大多数现代工作任务无法在他的帮助下解决，但是您可以尝试使用熨斗。

工匠通过各种技巧扩展了系统的功能，现在Macintosh SE / 30已添加了WiFi。 这样做不是借助“树莓”，而是借助创建的以太网到WiFi桥接器。 将模块连接到Danaport以太网后，可以“教”旧计算机以查看最新时间的无线网络。

实际上，将WiF添加到旧PC并不是什么新鲜的事-尽管在Raspberry的帮助下，它以前已经做过了。 现在情况有所不同-主机已经开发了一种实用的本地WiFi模块，该模块可以在30年以上的计算机上的旧操作系统环境中正常工作。

自然，为了使WiFi出现，您需要适当的驱动程序和软件。 昵称为[ants]的用户可以同时写两个人（ GitHub ）。

Vonets WiFi卡的问题在于需要从浏览器进行配置。 但是由于旧的“罂粟花”上的现代浏览器无法正常工作，因此您必须寻找出路。 在当前情况下，已开发了可自动调整Vonets卡的扩展程序。 该扩展程序的工作原理类似于负责在现代“罂粟花”中配置WiFi的模块的原理，甚至无线网络图标也出现在工具栏中。

当然，该项目本身只是为粉丝而设计的，因为30年前无线网络可以为计算机添加哪些新功能？ 几乎没有。 也许某些页面可以打开iCab ，但是即使Google起始页也将非常缓慢地打开。 而且，HTTPS不能像Javascript一样起作用。

因此，您将只能打开最简单的页面，也许是那些旨在重现90年代互联网氛围的页面。 可以在此处找到此类页面的示例。

至于Macintosh SE / 30，这是一台由Apple Computer Inc开发并于1989年至1991年出售的个人计算机。这是原始黑白Mac电脑中最快的型号。 该型号具有黑白显示器，并支持第三方加速器，网卡和显示适配器。

正式地，它支持32 MB的RAM，但是内存量可以增加到128 MB。 当时很多。 系统中安装了40或80 MB的硬盘驱动器。 此外，此Macintosh电脑具有1.44软盘驱动器，这是第一个支持这种类型的软盘的型号。


模型的名称不是偶然选择的。 事实是，该公司使用字母“ x”表示68030处理器的存在，但是在Macintosh SE更新之后，正是这种处理器成为其配置的一部分，该型号被称为“ Macintosh SE / X”，用于当时的情况非常挑衅（即使现在，许多公司也不会以这种方式命名其设备）。 因此，决定选择名称“ SE / 30”。

该模型一直生产到1991年，之后被Macintosh Classic II取代。 顺便说一下，这是一个不太成功的模型，因为实际上配置相似，由于16位系统，它的速度比其“祖先”慢40％。 仅支持10 MB的RAM，没有扩展端口。
