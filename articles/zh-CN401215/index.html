路由器为操作员“接地”


在我们的商店中，以SNR为品牌的路由器的首批代表销售正在火热进行中。 SNR-CPE-W4N路由器 （模拟DIR-615）已在我们的客户中获得普及，并已被通信提供商广泛使用。 低价与功能相辅相成。 在我们的评论中阅读有关SNR-CPE-W4N的更多信息。 在削减之下，我想谈谈取代他的盒子。

让我们开始介绍那些用相同名称的最接近“亲戚”替换SNR-CPE-W4N的人。 但是，新版本的SNR-CPE-W4N路由器在外观和填充方面都与其前身有很大不同。


该设备的紧凑型外壳（173x118x175 mm）不会在家庭中以及在办公网络建设期间给您带来不便之处。 上壳盖由白色塑料制成，表面涂有清漆。 外壳上没有指纹或划痕，很容易清除污垢。 SNR标志位于中间。

SNR-CPE-W4N配备了两个具有5dBi增益的外部天线，从而确保了足够大的Wi-Fi信号覆盖区域，并且内部具有稳定的数据传输。 该路由器支持IEEE 802.11b / g / n标准和MIMO2x2技术。 这使您能够以300 Mbps的速度实现数据传输。 该设备的填充基于许多现代Wi-Fi路由器中使用的MT7620N芯片组。


该路由器非常适合连接到IPTV服务的订户。 支持组播，IGMP侦听以及组播到单播。 多播/广播流量的隔离使您即使在Wi-Fi客户端上也可以实现高质量的IP-TV。 如果连接到IPTV的设备不支持多播，则xupnpd DLNA服务器已添加到路由器软件中，使用该软件，您可以将提供商的播放列表直接下载到路由器并将其用作媒体服务器。

主要特点：

-芯片联发科MT7620N
-Wi-Fi：802.11b / g / n 2T2R
-8MB闪存/ 64MB RAM
-IPv6支持
-内置xupnpd（DLNA）
-WIve-NG-mt软件
-粗糙的网络界面



我们品牌下的路由器系列中最接近的SNR-CPE-W4N是其双频版本SNR-CPE-MD1。



这里有更多的机会，因为路由器可以以最高可能的速度-高达733 Mbit / s同时在2.4 GHz和5 GHz的两个频率范围内运行。 由于路由器支持IEEE 802.11b / g / n标准和MIMO 2x2技术以及802.11a / ac，因此提供了此类功能。



设备的尺寸为187 x 143 x 35 mm，可以放置它而不会影响办公室或家庭的空间。 两个5dBi天线的存在确保了较大的覆盖范围和稳定的信号运行。

对两个频段的支持不仅允许使用更自由的5GHz频率进行数据传输，还可以实现一种方案，其中wifi路由器同时在一个频率（例如5GHz）上充当无线客户端（APCli模式），并在终端上提供接入点。另一个（例如2.4GHz）。 与上面的“患者”一样，此处进行多媒体服务的工作。

主要特点：

-芯片联发科技MT7620A
-芯片联发科MT7610NE
-Wi-Fi：802.11b / g / n 2T2R
-Wi-Fi：802.11a / ac 1T1R
-8MB闪存/ 64MB RAM
-IPv6支持
-内置xupnpd（DLNA）
-WIve-NG-mt软件
-原始或粗俗的网络界面



这两款路由器均支持IPv6，包括具有dhcp + ra，6to4模式的IPOE / PPPOE上的本机双协议栈； radvd和dhcp6s服务等



软体类


两个路由器的固件均基于Wive-NG-MT软件。 这是Ralink / Mediatek MT7620的新型预算芯片路由器软件的进一步开发。 Wive-NG-MT是专门为NAG设计的嵌入式操作系统。







客制化


专门为电信运营商开发了一套灵活的定制服务。 对于特定的运营商，任何路由器都可以“锐化”。 可以根据设备的外观进行更改。 例如，用操作员的颜色准备自己的盒子。



如何在设备上更改徽标





Wive-NG-mt软件允许您独立更改init，而无需使用rwfs重建固件。 这使您可以使路由器适应运营商网络的特性，并使用户易于配置路由器。

对于公司网络，可以使用整形器功能，可以在点之间灵活配置漫游，而无需外部控制器，并且可以在“热点”模式下工作。
