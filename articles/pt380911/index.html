Lançamento de distribuição Mageia 5



Mageia é uma distribuição Linux sem fins lucrativos desenvolvida pela comunidade com o mesmo nome. Foi fundada em 2010 pelos ex-desenvolvedores do Mandriva como um fork e tem muito em comum com seu ancestral.

O principal ambiente de trabalho é o KDE. Existe uma versão com o GNOME, assim como nos repositórios e em DVD há seus garfos - Mate e Cinnamon. Existe um configurador gráfico que permite configurar o sistema sem precisar entrar no console.

Novas versões da distribuição são lançadas aproximadamente uma vez a cada 9 meses. Para aqueles que precisam de lançamento sem interrupção - existe um ramo do Caldeirão em desenvolvimento, que é constantemente atualizado com novas versões de programas.

Em 19 de junho, a quinta versão da Mageia foi lançada. Vamos ver como os desenvolvedores podem nos agradar dessa vez.

Inovações


Principais alterações:
  • UEFI.

  • 3.19.8.

  • A plataforma de jogos Lutris foi adicionada, fornecendo uma interface única para o lançamento de jogos. Steam, Desura, jogos nativos, vinho, vários emuladores para consoles são suportados. Os planos suportam jogos do Humble Bundle e do GOG.

Aparência no ambiente de trabalho do KDE:


As imagens de distribuição são híbridas e podem ser gravadas em uma unidade flash USB. As seguintes imagens estão disponíveis para download :
  • LiveDVD 1.7 GB para KDE e GNOME
  • LiveCD para KDE e GNOME
  • DVD de instalação
  • imagens para instalação em rede

informação adicional


O repositório de distribuição é dividido em três grupos:
  • Core - Software Livre
  • Não-livre - software não-livre
  • Tainted é um software livre que pode ter problemas com patentes (codecs, etc.)

Por padrão, apenas o Core está ativado. Você pode conectar os outros e atualizar o sistema, após o qual todos os codecs necessários serão instalados, versões mais completas de algumas bibliotecas serão instaladas etc. Além disso, imediatamente após a instalação, o programa Mageia Welcome será iniciado, no qual você poderá adicionar codecs e alguns softwares adicionais (os repositórios serão conectados automaticamente).
