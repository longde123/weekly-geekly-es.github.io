如何为IoT网络选择通信标准

在上一篇文章中，未考虑无线网络标准。
建立网络时，最重要的问题之一就是物联网的各个元素如何互连。这里可以有选择，当然，这完全取决于项目的目标。
考虑网络连接选项时的关键方面：
  • 范围。在办公室还是整个城市部署网络？
  • 频次 需要什么穿透力？抗干扰力是什么？
  • 数据传输速率。需要什么带宽？数据多久更新一次？
  • 电源。设备由电源或电池供电吗？
  • 安全性 设备是否参与关键任务应用程序？

LoRaWAN
LoRaWAN或长距离广域网中引入的节能网络技术研究中心，IBM研究中心和公司Semtech公司。该技术基于Semtech LoRa（™）PHY芯片。
LoRa在未许可频率的亚千兆赫ISM频段（工业，科学和医学无线电频段）中运行。网络体系结构是一个明星，终端设备无线连接到一个或多个网关，并且网关使用标准IP连接连接到网络服务器。
为了支持和传播技术，最近成立了LoRa联盟，其中包括许多公司，包括俄罗斯的花边。

LoRa的优点：
  • 开放标准
  • 远射
  • 在城市地区的高渗透
  • 低功耗，AA电池寿命估计长达10年
  • 各种未经许可的频率，例如109 MHz，433 MHz，868 MHz，915 MHz sub-GHz ISM频段
  • 自适应数据速率
  • 支持个人和社交网络
  • 全面的安全性以及内置的身份验证和身份验证

LoRa的缺点：
  • 低数据率
  • Semtech是唯一的芯片供应商
  • 没有漫游

LoRa的集线器由MultiTech等公司提供，并且已经创建了The Things Network等公共网络。

斯威夫特
由俄罗斯公司实施的系统斯威夫特 -Telematika，用自己的Marcato 2.0协议。频率可以适应ISM范围。
该技术在某种程度上类似于LoRa技术，但后者具有所有优点和缺点。根本的区别：LoRa使用宽带编码，而STRIZ使用窄带调制。据该公司称，这种调制可以更有效地利用频谱带，提高灵敏度和能效，并降低成本。
STRIZH无线网络已在莫斯科部署，覆盖率100％，部分覆盖了莫斯科地区，圣彼得堡和其他一些城市，并拥有200多个基站。生产和出售带有内置调制解调器的无线电调制解调器，基站，计数器和传感器。

SigFox
System由同名公司建造，该公司于2009年在法国成立。它使用超窄带（UNB）技术，该技术与第二次世界大战期间潜水艇之间进行通信的技术相同。该技术最初旨在用于低数据速率的通信。
西格福克斯根据特定的地区法规，目前使用的是868 MHz（由ETSI和CERT标准定义）以及在美国902 MHz（由FCC定义）上最受欢迎的欧洲ISM频段。使用现代蜂窝网络的功能来部署该系统。
该设备每天最多可以发送140条消息，每条消息最多可以包含12个字节的有用数据。 12个字节满足了传输数据的设备的需求，例如设备位置，功耗指标，警报或任何其他类型的基本感官信息。您每天还可以从8个字节的有用数据中向每台设备最多发送4条消息。为了接收消息，设备必须从服务器请求数据，必须为特定事件或特定时间进行编程。发送到设备的8字节允许您在必要时发送配置数据，可以优化电池寿命。如果不需要完整的双向通信，这就足够了。
与竞争对手不同，该网络已经在整个欧洲和北美部署，覆盖了数以万计的设备。该公司正在对SigFox Ready™设备进行认证。

SigFox的优点：
  • 覆盖面广
  • 在城市地区的高渗透
  • 超低功耗，估计使用2节AA电池可长达20年的传感器运行
  • 天线设计的灵活性
  • SigFox协议与现有收发器兼容
  • 成本低

SigFox的缺点：
  • 低数据率
  • 对蜂窝基础设施的依赖
  • 有限的抗噪能力

失重 -公共通信技术标准组LPWAN（低功耗广域网） 的基站和设备之间共享数据。标准是由Weightless SIG非营利组织开发的。当前有3种标准可用-失重N，失重P和失重W。使用未经许可的亚千兆赫频率。
Weightless-N使用超窄带（UNB）技术，是一种单向通信标准。在成本和能耗方面，这是该组中最经济的标准。
Weightless-W在当地法规允许的范围内使用TVWS频率（电视空白，未使用的电视频谱频率）。这增加了终端的成本并增加了其功耗。
Weightless-P-最后的标准，于2015年7月推出，是双向的，支持该组中效率最高的所有主要SRD / ISM范围（短距离设备/工业，科学和医学），并具有漫游，传呼等许多附加功能，自适应编码等。因此，它的范围略短，功耗更高。

减轻体重的好处：

  • 在城市地区的高渗透
  • 低功耗，估计可长达10年的传感器运行时间（Weightless-N）
  • 各种非许可频率（Weightless-P）
  • 支持个人和社交网络
  • 高安全性
  • 低成本（尤其是失重-N）

失重的缺点：
  • 低数据率

Dash7
Dash7联盟协议（或D7A）是一种开放式无线协议，其工作频率为未经许可的ISM / SRD频段的433 MHz，868 MHz和915 MHz。支持AES 128位加密和高达167 kbit / s的数据传输，最大数据包为256字节。
该协议由总部位于比利时的非营利性Dash7联盟推广。该协议基于国际标准ISO / IEC 18000-7，该标准描述了有源RFID的接口，并在美国军事后勤（NATO）中使用。DASH7协议的当前版本不再与ISO / IEC 18000-7标准兼容。

Dash7的优点：

低功耗蓝牙（BLE）
低功耗蓝牙（BLE）是使用蓝牙特殊兴趣小组开发和实施的无线个人联网技术。目前，蓝牙技术已出现在所有移动平台上，BLE配备了数百万新设备。该技术得到良好的支持，并且对于近距离通信是可靠的。通常用于智能手机和其他个人（很少是家用）电子设备之间的通信。特别是，iBeacon技术基于该技术。

Wi-Fi
Wi-Fi（或WiFi，最初来自英语Wireless Fidelity）是一种本地无线网络技术，该技术允许电子设备主要使用2.4 GHz和5 GHz ISM射频连接到网络。该技术是由Wi-Fi联盟基于IEEE（电气与电子工程师协会）802.11标准开发的。 Wi-Fi是同名联盟的注册商标，该联盟包括600多家公司。这种事实上的技术几乎已经普及；全世界每年发布数十亿个Wi-Fi设备。
Wi-Fi最初是为本地通信而设计的。带有杆状天线的现代接入点可提供长达约100 m的范围而无障碍。有使用放大器和半抛物面天线的解决方案，其范围超过20 km。
这项技术并没有停滞不前，不断发展。因此，Wi-Fi Direct技术允许Wi-Fi设备直接连接，而无需访问点和网络。设备可以相互连接，也可以同时连接到多个设备的组。通过Wi-Fi Direct认证的设备可以轻松，简单地连接：将两个兼容NFC的设备连接在一起，或者输入PIN码。此外，所有直接的Wi-Fi连接均受WPA2保护。
如果同一区域中还有其他类似设备，则Wi-Fi连接可能会断开，或者连接速度可能会降低。默认情况下，许多2.4 GHz 802.11b和802.11g接入点在初始启动时都在相同的通道上运行。在高密度区域，例如大型住宅区或具有许多Wi-Fi热点的办公楼，Wi-Fi污染可能是一个问题。此外，许多其他设备使用2.4 GHz频段：微波炉，ZigBee设备，蓝牙设备，无绳电话，婴儿监视器，这可能会引起严重的额外干扰。当市政当局或其他大型设施（例如大学）寻求提供更大的Wi-Fi覆盖区域时，这也是一个问题。
最近发布的Cisco和Apple文档“ Cisco无线LAN上Apple设备的企业最佳实践”包含有关在iOS 9.0或更高版本的操作系统上使用iPhone，iPad，iPod设备的联合建议。如本文档所述，“ 2.4 GHz频段不适合任何业务和/或关键企业应用。”对于使用Apple设备的无线网络，建议公司仅使用5 GHz频率（802.11a / n / ac标准）。尽管如此，2.4 GHz频段仍是主要频段，默认情况下已用于大多数移动设备，此外，并非所有国家/地区都允许将5 GHz频率用于Wi-Fi。
即使配置正确（由于算法的缺点），最初引入的WEP加密标准也可以相对容易地破解。较新的设备支持更高级的WPA和WPA2数据加密协议。许多组织使用附加加密来防止入侵。目前，破解WPA2的主要方法是密码猜测，因此建议使用复杂的字母数字密码，以最大程度地提高密码猜测的复杂性。此外，Wi-Fi标准不提供开放网络中传输数据的加密。这意味着攻击者可以使用嗅探器程序监听通过开放的无线连接传输的所有数据。因此，使用免费热点时，不应将关键数据传输到Internet。

Wi-Fi HaLow
最近，在2016年国际消费电子展上，Wi-Fi联盟宣布开发用于物联网的新无线标准。新标准称为HaLow，尚未得到IEEE 802.11ah规范的批准。首批兼容HaLow Wi-Fi的设备的认证将于2018年开始，但是支持该新规范的产品将在更早的市场上出现。
未经许可的900 MHz频率将用于连接HaLow Wi-Fi。这将显着增加城市地区的信号穿透力，其作用半径将比现代无线标准大得多，最大可达1公里。同时，“范围”的回报是低信号功率。 HaLow Wi-Fi带宽将大大低于最大Wi-Fi 802.11ac（7 Gb / s），估计速度为：50 kbit / s-18 Mbit / s。
根据联盟的说法，HaLow将广泛使用现有的Wi-Fi协议，这将提供高度的兼容性和安全性。

螺纹

作为Wi-Fi的补充实现的线程在安全性和功耗方面在家庭自动化中有明显的限制。该协议基于6LoWPAN标准（低功耗无线个人局域网上的IPv6）-IEEE802.15.4标准的低功耗无线个人网络上的IPv6通信标准（IP协议的新版本，地址长度为128位而不是IPv4中的32位）。对于支持IEEE802.15.4标准的现有设备，可以轻松将其升级为Thread。除了专门为家庭自动化设计的网状网络的可靠性之外，该协议还提供AES银行类安全性。您可以将250多个授权设备连接到同一网络。对“睡眠模式”的广泛支持甚至允许使用单节AA电池运行设备多年。

ZigBee
ZigBee是由IEEE 802.15.4标准（于2003年出现）规范的顶级网络协议的规范。 ZigBee和IEEE 802.15.4描述了无线个人区域网（WPAN）。 ZigBee规范专注于要求以相对较低的速度保证安全的数据传输以及需要使用独立电源长期运行网络设备的应用。 ZigBee技术不仅支持简单的网络拓扑（点对点，树形和星形），还支持具有消息中继和路由功能的自组织和自修复网状拓扑。ZigBee联盟
正在开发ZigBee，其中包括300多家公司。联盟还对设备进行认证。2015年12月16日，联盟宣布批准ZigBee 3.0，该版本考虑了当前的物联网要求，并支持所有以前的版本以及已售出的数亿个设备。

ZigBee的优点：
  • 自组织和自我修复的能力
  • 易于部署
  • 高抗扰度
  • 高安全性
  • 未经许可的频率
  • 低功耗（包括设备的睡眠模式）

ZigBee的缺点：
  • 低速
  • 大部分流量都用于传输包含地址信息，同步信息等的数据包。
  • 在城市地区渗透率低
  • 标准化水平不够高，缺乏用于开发复杂应用程序的单一软件和硬件平台


Z-Wave
Z-Wave是专用于家庭自动化的专有无线通信协议。该技术使用内置在消费电子产品和各种设备中的低功耗和微型射频模块。Z-Wave在高达1 GHz的频率范围内运行，并针对低延迟传输简单的控制命令进行了优化。Z-Wave解决方案基于自组织网状网络，其中每个节点或设备可以使用中间相邻节点接收控制信号并将其传输到其他网络设备。
Z-Wave无线电芯片由Sigma Designs和Mitsumi提供。Z-Wave的一个显着特征是所有这些产品相互兼容。兼容性通过Z-Wave或Z-Wave Plus认证过程确认。认证由Sigma Designs提供，该公司已认证1350多种Z-Wave产品。在全球范围内，该协议得到Z-Wave联盟的支持，该联盟聚集了325多家制造商。

Z-Wave的优点：
  • 专为家用电子产品开发
  • 自组织和自我修复的能力
  • 易于部署
  • 高抗扰度
  • 高安全性
  • 未经许可的频率
  • 缺乏对多个2.4 GHz设备的干扰
  • 低功耗

Z-Wave的缺点：
  • 低速
