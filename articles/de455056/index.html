Primitive Grafiken für Lazy, Oldfags und Lazy Oldfags

Hallo Habr. Vor nicht allzu langer Zeit bin ich auf das Netzwerk einer ziemlich einfachen Bibliothek für die Arbeit mit Grafiken in C / C ++ gestoßen, die für Schüler und Studenten in ihrer Programmforschung (ohne fast jedes Wissen erforderlich) sowie für einige alte Leute, die sich erinnern, nützlich sein kann uralt wie Mammutgrafiken für DOS. Insbesondere eine so primitive Grafikbibliothek wie graphics.h für Turbo C, auf die Informatiklehrer und Programmierlehrer an der Universität noch starren.

Kurz gesagt, die Grafikbibliothek stammt von Borland und wird vom Standard nicht akzeptiert. Sie kann Grafiken zeichnen. Wenn Sie ein Rechteck zeichnen müssen, sind keine monströsen Konstruktionen aus Zeigern, Vektoren und anderen für Anfänger schrecklichen Dingen erforderlich. Geben Sie einfach rechteck () ein und geben Sie die Koordinaten der oberen linken und unteren rechten Punkte des Rechtecks ​​in Klammern an. Und alle.

So konnten Handwerker diese Bibliothek auf MinGW portieren und ein wenig verbessern, indem sie die normale Arbeit mit Maus und Tastatur vermasselten, Bilddateien luden und schrieben, die gesamte RGB-Palette unterstützten (gegen die 16 Farben, die anfangs von graphics.h verfügbar waren) und ein paar weitere kleine Chips. Das Besitzen dieser alten Frau kann immer noch nützlich sein, um schnell primitive (und nicht so) Grafiken zu erstellen.

Kurz zu dieser Bibliothek:

Gewinn:

+ Für Anfänger leicht zu erlernen
+ Einfach anzuschließen
+ Keine Anforderungen an die Systemressourcen
+ Kann in allen Auflösungen und Adaptern (zumindest unter CGA schreiben)
+ Kann Bilder ohne Verzerrung hochladen
+ Kann bei Maus- / Tastaturereignissen verwendet werden
+ Kann die volle RGB-Palette verwenden

Fehler:

- Eine unbedeutende Menge von Füllstilen und Linien (es wird nicht durch Zeichnen von Grundelementen entschieden, sondern durch einfaches Laden der fertigen Bilder)
- Ein unbedeutender Satz von Schriftarten für die Anzeige von Text im grafischen Modus (behoben durch Zugriff auf die Windows-Schriftdateien, aber dort ist etwas Tanzen mit einem Tamburin erforderlich).
