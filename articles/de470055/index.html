Einführung in NuGet Package Management auf Lösungsebene in Visual Studio für Mac

Visual Studio 2019 für Mac Version 8.3 enthält viele neue Funktionen, die in diesem Blogbeitrag zusammengefasst sind . Die Möglichkeit, Pakete auf Lösungsebene zu verwalten, war eine der Funktionen, die den meisten von Ihnen in Visual Studio für Mac fehlten. Der neue Paketmanager auf NuGet-Lösungsebene ist eine der coolen neuen Funktionen in Visual Studio 2019 für Mac Version 8.3.

Wir haben Verbesserungen vorgenommen, um Ihnen die Suche nach Paketen zu erleichtern. Dies umfasst verbesserte Funktionen bei der Suche nach neuen Paketen, beim Verständnis, welche Pakete bereits in Ihrem Projekt installiert sind, und bei der Suche nach Paketen mit verfügbaren Updates. In diesem Beitrag konzentrieren wir uns auf die Paketverwaltungsfunktionen für die Lösung. Die meisten Funktionen, einschließlich der Installation, Aktualisierung und Anzeige installierter Pakete, verfügen jedoch auf Projektebene über eine ähnliche neue Funktionalität.



Um den NuGet Package Manager für die Lösung zu starten, können Sie im Kontextmenü "Lösungen" die Option "NuGet-Paket verwalten ..." auswählen:



Neue Pakete hinzufügen


Wenn Sie suchen und versuchen, ein neues Paket hinzuzufügen, können Sie die Projekte auswählen, in denen Sie das Paket installieren möchten.

Sie können jederzeit zur Registerkarte " Installiert " gehen und eine Liste aller in Ihrer Lösung installierten Pakete anzeigen, mit denen Sie sie zur Aktualisierung entfernen können.



Paketaktualisierungen


Auf der Registerkarte Updates werden alle Pakete in der Lösung angezeigt, für die Updates verfügbar sind (oder im Projekt, wenn Sie den Befehl auf Projektebene aufrufen).



Paketkonsolidierung


Oft werden große Entscheidungen in Situationen getroffen, in denen unterschiedliche Projekte auf unterschiedliche Versionen des Pakets verweisen. Um diese Versionen zu einer einzigen Version des Pakets zu kombinieren, die Sie möglicherweise in der Lösung verwenden möchten, können Sie zur Registerkarte Konsolidierung des NuGet-Paketmanagers auf dem Lösungsknoten wechseln, die Version des Pakets auswählen, die Sie in allen Lösungsprojekten verwenden möchten, und die Paketkonsolidierung auswählen (Pakete konsolidieren) :



Probieren Sie es jetzt aus!


Um diese neuen Funktionen von NuGet auszuprobieren, laden Sie Version 8.3 von Visual Studio 2019 für Mac herunter oder aktualisieren Sie auf die neueste Version über den stabilen Kanal, wenn Sie VS für Mac bereits installiert haben.

Wenn in Version 8.3 Probleme auftreten, verwenden Sie das Menü Hilfe> Problem melden. Sie können auch Vorschläge für weitere Verbesserungen über das Menü "Vorschlag bereitstellen" machen.
