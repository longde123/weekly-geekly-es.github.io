DJI Osmo Mobile 2-手机电子稳定器

如果您对Osmo一无所知，而DJI更可能是适合您的时尚DJ的名称，那么我将简要介绍所有内容。 对于本主题的那些人，可以对角线阅读下面的两个段落，或者从“让我们弄清楚什么是什么”部分开始。



中国DJI公司是著名的直升机市场领导者，也是世界上最先进的公司之一。 几年前，他们发布了DJI Osmo电子稳定器，这是一款具有智能功能的便携式相机稳定器。 有了它，您就可以在旅途中拍摄视频，并且图片仍将保持流畅，并具有许多不错的拍摄功能。

后来，该公司通过添加Osmo +扩大了稳定器的范围，Osmo +的相机获得了光学变焦，并且不停地发布了第一代DJI Osmo Mobile。 实际上，它是带有用于智能手机的稳定器的老式Osmo笔。 该产品一次“炸爆”了市场，并处于领先地位。

但是很快，竞争对手智云Smooth Q，飞宇科技，Lanparte，Swiftcam等公司出现了。 竞争对手的产品质量和功能都较差，但价格几乎是其一半。

为了恢复其地位，大疆创新发布了Osmo Mobile 2 。

让我们看看是什么


让我们从外观开始。 用肉眼无法区分新型号的机壳和旧版Osmo Mobile。 但是有一个重要的区别-材质。 新版本使用轻质和高强度尼龙，因此新产品的重量已从之前的501克降至485克。 差异似乎微不足道，但是如果您拍摄一个多小时，您会感觉得到。 Osmo Mobile 2被称为最轻的手动稳定器之一。

新模型仅以深灰色发布，而之前的模型有黑色和银色两种选择。 一方面，现在无法匹配智能手机的颜色，但另一方面，这样的步骤可能简化了生产过程，从而降低了最终价格。

Osmo Mobile 2手柄比原始稳定器的手柄长。 因此，可以提供能量消耗更大的电池。 我们将谈论他。

它能工作多长时间？


太多了，您现在就想要！

该稳定器配备了新的2600 mAh锂聚合物电池，可拍摄的视频比其前辈长三倍-15小时！ 因此，我一次听到有人决定等待新模型的发布，而没有为第一个Osmo付出代价的人感到高兴。 在这种情况下，电池不可拆卸。 尽管很难称它为这种工作时间的缺点。

这是另一个非常重要的更新-Osmo Mobile 2可用于为智能手机充电。



外壳上有一个USB连接器，您可以在上面连接智能手机，而不必担心它会很快放电，现在可以通过常规的microUSB电缆为稳定器充电。 因此，您始终可以在汽车或任何移动电源中为其充电。



套餐捆绑


Osmo Mobile 2仅随附由EPP泡沫塑料制成的外壳和microUSB电缆。

关于拍摄模式




任何积极创建社交网络内容的人都会对这样的稳定​​器感到高兴，因为用这样的东西进行现场直播是一件好事。

如果您的工作不是很专业，则可以将稳定器与三脚架一起使用以进行更稳定的拍摄。 此外，新版本支持标准的三脚架插孔，您无需购买额外的适配器垫。

简要介绍智能模式




  • ActiveTrack 相机跟踪在DJI Go应用中选择的对象。 有手动和自动模式。


  • 超间隔 。 在加速拍摄过程中，允许操作员与稳定器一起移动。 这样，您可以整天起飞。


  • 运动间隔。 使您可以沿预定路径进行加速拍摄。 例如，使用此功能，您可以通过跟随太阳的移动来拍摄日出或日落。


  • 游戏中时光倒流 简单，快速的拍摄。 设置稳定器，设置时间，然后逐帧拍摄，然后缝入视频片段。


  • 全景图 Osmo Mobile 2最多可以自动捕获9张图像并创建全景图像。


  • 长时间曝光（长时间曝光）。 可以创建稳定的长时间曝光图像。 尝试移走过往的汽车，获取长远的灯光痕迹，夜间游乐园-摩天轮，或用“牛奶”水制作海边的镜头，而无需三脚架。

  • 慢镜头 高品质的游戏中时光倒流。





您可以通过两种方式设置智能手机：纵向和横向。 现在可以垂直拍摄稳定的Instagram视频。 是的，是的-垂直视频很糟糕，但是既然每个人都可以这样做-我们就可以高质量地制作它！

您还忘了讲什么？


稳定器支持低功耗的蓝牙4.0技术。

与配备外部镜头的移动设备兼容，但有一个“ BUT”按钮-在垂直模式下使用iPhone X时，稳定器马达可能掉入框架中。 为了避免这种情况，有必要提高稳定器的高度。

Osmo Mobile 2可以与宽度为58.6-84.8 mm，厚度最大为8.4 mm的智能手机一起使用。 新的稳定器还与第三方应用程序兼容。 在iOS设备上，素材存储在DJI Go应用程序中，而在Android设备上，默认情况下使用手机本身或microSD卡-这是iOS和Android配对的典型情况。

总而言之，我们可以说大疆已经发布了一款具有竞争力的稳定器：便宜很多倍，强大三倍，同时又更轻便。 如果您附近的人喜欢填写Instagram的照片，故事和视频，那么新的Osmo Mobile 2是一款实用且便宜的生日礼物。
