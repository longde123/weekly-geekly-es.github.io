Cara membuat aplikasi keuangan: 5 API untuk membantu pengembang



Halo semuanya! Saya melanjutkan penelitian saya tentang bidang keuangan dari sudut pandang teknologi. Pada artikel sebelumnya, kami mempelajari terminal perdagangan yang ada dan fitur-fiturnya, dan hari ini saya mengusulkan untuk berbicara, pada kenyataannya, tentang pengembangan aplikasi keuangan.

Untuk topik baru, saya mengambil lima API berbagai layanan keuangan yang memungkinkan Anda untuk menerima beragam data dan langsung berdagang di bursa menggunakan robot tertulis.

Keuangan Yahoo


Mungkin salah satu layanan keuangan paling terkenal di dunia. Dengan menggunakan API gratis, pengembang dapat menggunakan data layanan tentang harga saham, perubahan harga, dan pergerakan pasar. Informasi tersedia pada berbagai saham dan juga instrumen lainnya, seperti pasangan mata uang.

Anda dapat dengan mudah terhubung ke API menggunakan layanan API Cepat.



SMARTcom


API untuk membuat aplikasi untuk perdagangan di bursa Rusia. Pengembang dapat menggunakan API sebagai lapisan antara sistem perdagangan pertukaran dan sistem perdagangan siap pakai mereka (dibuat secara mandiri atau dibeli), membuat robot perdagangan sendiri, atau bahkan seluruh terminal perdagangan dengan antarmuka "khusus".

Akses ke API gratis, Anda hanya perlu membuat akun di sistem perusahaan pialang yang mengembangkannya. Ada peluang untuk men-debug pekerjaan perangkat lunak Anda menggunakan akses tes dalam perdagangan virtual.

Sinkronkan dengan buku pembayaran


Layanan yang berguna bagi pengembang aplikasi keuangan untuk pasar luar negeri. Ini memungkinkan Anda untuk mengumpulkan data di satu tempat dari bank, perusahaan keuangan dan utilitas, lembaga pemerintah di berbagai negara. Layanan ini dibayar, biaya akses mulai dari $ 50. Daftar layanan yang tersedia untuk integrasi dapat ditemukan di sini .



API Currencylayer


Layanan yang berguna untuk mendapatkan informasi tentang nilai tukar dan nilai 168 mata uang dunia. Data dikumpulkan dari banyak sumber, seperti situs bank, layanan informasi komersial. Pengembang menjanjikan kompatibilitas dengan aplikasi jenis apa pun dan bahasa pemrograman apa pun.

Ada versi terbatas API gratis (250 permintaan per bulan), yang lebih memadai mulai dari $ 9,99 (10.000 permintaan per bulan). Solusi mereka berdasarkan API Currencylayer dikembangkan oleh perusahaan seperti Lyft, Blizzard, Intel.



Keuangan mudah


Layanan terkenal untuk akuntansi keuangan pribadi di Runet menyediakan API untuk pengembang pihak ketiga. Seperti yang ditunjukkan di situs web proyek, aplikasi pihak ketiga untuk jejaring sosial, aplikasi seluler, dan plug-in untuk peramban paling populer di kalangan pengguna sumber daya.

Aplikasi diuji oleh tim layanan, yang disetujui masuk ke pasar khusus dan didistribusikan dengan berlangganan. Menurut statistik dari administrasi EasyFinance, pengembang aplikasi dapat memperoleh 15-50 rubel per pengguna per bulan. Penghasilan ini terdiri dari pendapatan dari berlangganan tarif layanan itu sendiri, pembayaran integrasi mitra platform dan iklan.
