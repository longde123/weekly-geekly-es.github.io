Unreal Engine 4.8 veröffentlicht

Die neue Version der Unreal Engine 4.8- Game-Engine enthält 189 neue Funktionen und Verbesserungen, die von einer Community von mehreren Dutzend Entwicklern entwickelt wurden.

Die wichtigsten Innovationen sind nachstehend aufgeführt.

Graswiedergabe




Graswiedergabesysteme sind erheblich für offene Welten optimiert. Der neue Renderer generiert dynamisch Gras, Büsche und andere Bodenobjekte in zufälliger Reihenfolge um den Player herum.

Während sich der Spieler auf der Karte bewegt, werden neue Objekte hinzugefügt und alte Objekte außerhalb der Reichweite verschwinden nicht mehr, sondern bleiben in der offenen Welt.

Entwickler können die Grasdichte im Menü Landschaftsgrasausgabe entsprechend der Landschaft festlegen: spärliche Vegetation auf einem Berghang oder dicht in einem Tal.

Verbesserte Editorstabilität


In Unreal Engine 4.8 hat dieses Problem erhöhte Aufmerksamkeit erhalten. Viele der nervigsten Fehler des Motors wurden behoben und einige Funktionen hinzugefügt.

SteamVR / HTC Vive-Unterstützung




Unterstützung für neue Geräte hinzugefügt. Eine Vorschau des Spiels finden Sie im HTC Vive-Helm (Schaltfläche „VR-Vorschau“). Alle Virtual-Reality-Controller-Funktionen werden unterstützt, einschließlich Tasten / Touchpads.

Reprojektion von Playstation 4 Morpheus VR auf 120 Hz




Unterstützung für die neueste Sony Morpheus-Hardware, einschließlich 120-Hz-Rendering mit zeitlicher Neuprojektion sowie 60-Hz-Rendering. Deutlich verbesserte Unterstützung für dreidimensionalen Klang.

Hierarchische Detailebene (LOD)




Das neue hierarchische System erstellt automatisch Detailebenen für Raster, wodurch die Anzahl der Anrufe und die Anzahl der Polygone aus großer Entfernung erheblich reduziert werden. Diese Funktion ist besonders für offene Welten relevant.

Kombinationsnetze werden automatisch ausgewählt. Wenn Simplygon verfügbar ist, wird es verwendet, um ein Netz mit geringen Details zu generieren. Andernfalls werden die LOD-Versionen jedes Netzwerks kombiniert. Materialien werden nach Möglichkeit automatisch zusammengeführt.

Laubbehandlungsgenerator




Ein neues experimentelles Merkmal für die prozedurale Erzeugung großer (oder winziger oder mittelgroßer) Laubregionen in kurzer Zeit.

Die Laub-Eigenschaften werden durch die Option "Laubtyp" festgelegt.

Spielvideonavigation




Unreal Engine-Spiele haben die Funktion, das Gameplay vollständig aufzuzeichnen und wiederzugeben. Ab Version 4.8 können Sie schnell zum gewünschten Teil des Videos wechseln.

Standardmäßig speichert die Engine Videos jetzt lokal (Einstellungen können geändert und in der Cloud gespeichert werden).

Hochwertige Tiefenschärfe und Tonabbildung




Der neue Tiefenschärfealgorithmus heißt Circle DOF. Es zeichnet sich durch hohe Leistung und das Vorhandensein von Parametern aus, die den üblichen Parametern physikalischer Kameras entsprechen. Entfernungsunschärfe wird unterstützt, um den Effekt der Lichtstreuung in der Atmosphäre zu simulieren.

Das neue Farbschema entspricht dem Filmstandard des Academy Color Encoding System ( ACES ).

Aktualisierter Blatteditor




Die Benutzerfreundlichkeit des Editors wurde verbessert, und das neue Menülayout in den Fenstern erleichtert das gleichzeitige Arbeiten mit einer großen Anzahl von Blatttypen.

Neue Blatttypen werden direkt aus diesem Menü erstellt, indem Sie auf die Schaltfläche „Laubtyp hinzufügen“ klicken.

Die Schaltflächen für jeden Blatttyp geben die Gesamtzahl an, von wo aus sie gespeichert werden können. Die Auswahl mehrerer Typen und Operationen auf ihnen funktioniert gleichzeitig.

Dynamisches Navigationsraster für offene Welten




Ein völlig neues System zur Unterstützung dynamischer NavMeshes in offenen Welten. Das Raster wird automatisch an der Stelle generiert, an der es benötigt wird. Die Option muss über das "Navigationssystem" - "Navigation Invoker Component" aktiviert werden.

Neue Vorlage




Eine neue hochauflösende Version für den unwirklichen Beispielcharakter ist in den Vorlagen enthalten, die mit der Spiel-Engine geliefert werden. Es gibt auch eine niedrig aufgelöste Version für mobile Geräte (kostenlos auf dem Marktplatz erhältlich).

Kurveneditor für die Benutzeroberfläche von Unreal Motion Graphics




Mithilfe von Kurven können Sie bequem Animationseigenschaften festlegen.

Reflexionen auf durchscheinenden Oberflächen




Durchscheinende Oberflächen können jetzt andere auf dem Bildschirm vorhandene Objekte dynamisch reflektieren. Die Option wird durch "Bildschirmraumreflexionen" aktiviert.

Plugins




Das erste Plugin von Allegorithmic wurde auf dem Marktplatz veröffentlicht, und in Zukunft werden noch mehr Plugins von Drittentwicklern erwartet.

Nachbearbeitung




Verbesserte Tools und Filter für die Nachbearbeitung. Deutlich verbesserte Filtergenauigkeit für Unschärfe, einschließlich Morphing der endgültigen Animation.
