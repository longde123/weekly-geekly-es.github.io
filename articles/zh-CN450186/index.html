一点加密：神秘而传奇的谜

秘密地渗透到敌人的意图和意图中，以识别敌人的长处和短处，进行预测，并在一个小时内阻止计划-这就是构成情报的现象。 上世纪初理应获得情报和反情报艺术鼎盛时期的头衔。 这篇文章并不是对第二次世界大战德国国歌的歌颂，而是再次提醒人们在上个世纪40年代的胜利中付出了巨大的努力。 毕竟，到第二次世界大战开始时，德国已成为该领域特殊设备生产的世界领导者。 各种录音机，包括小型，微型麦克风，监听系统，被认为是世界上最好的，因为它们是根据德国在无线电工程和化学领域的最新科学成果而开发的。



希特勒帝国最复杂的加密机Enigma只有几份能够幸免于战：法西斯军队的忠实士兵投降，将其摧毁了成千上万枚。 如今，很少有人能说出传说中的“谜语”中使用的加密技术。



1943年，西南和斯大林格勒前线的部队关闭了包围圈（斯大林格勒大锅，其中有将近30万敌军），无线电营员德国人克莱德斯很幸运地留在了包围圈之外。 无线电运营商从锅炉中的同志那里收到了Enigma加密的最后一条消息。 他仍然记得它的内容：我们在狭窄的地下室里战斗X敌人拿起了掷火器X让德国生活了。 然后又出现了一条明文信息：向我的父母表达敬意。



受到机器可靠性的打击，德国政府保留了它的所有权利，并开始将其用于军队。 Enigma是二战期间德国人使用的最复杂的加密系统。 为了通过莫尔斯的无线电频道与前线部队进行通讯，士兵使用了一种更为简单的系统：双重加密。

出生于1921年的德国人Kledits和出生于1923年的约尔根·莱因霍尔德（JürgenReinhold）是能够与我们的时代的恩尼格玛谈论他们的作品的幸存的目击者之一。



在国防军的领导下，克莱德斯（Kledits）在1939年学习成为电话接线员，然后是无线电接线员。 最初，他在法国的步兵师中服役，并于1941年前往东方。 在内部，单元只能使用双重加密。 1942年初，他被调到情报部门，负责窃听和分析无线电报传输的苏联信息。 随后，该部队通过苏联领土前进，进行无线电拦截并使用先进技术，最后一点是斯大林格勒。

Kledits是一名无线电运营商。 截获消息的分析是由同一单位的其他专家（主要是数学家）进行的。 德文与他们的联系很少，对他们的秘密工作一无所知。 如今，有关第二次世界大战中德国密码学家工作的信息极为零散。

但是，Kledits可能已经猜到了他的同事在做什么。 作为一名无线电运营商，他面临着通过莫尔斯设备将消息传输到军事总部的任务。 为了防止这些严格机密的数据落入敌人手中，我们使用Enigma对其进行了加密。 就像Kledits回忆的那样，加密始终由三人负责：一个人在Enigma上输入了适当的字母，第二个人对结果进行了加密，第三个人将其写下来。



为了正确传输加密消息，无线电运营商必须设置正确的密钥（由数字组合组成）。 每个支队都有一个代码清单，每当一名军官参与其中时，代码清单就会传给无线电操作员。 这样的密钥有效期为24小时。

“我们坚信无法破解Enigma，”另一部门的专家之一尤尔根·莱因霍尔德（JürgenReinhold）说。 他于1941年被征召入伍，在坦克部队的情报部门任职，为此他学习了9个月。 同时，他研究了摩尔斯电码，学习了双重加密并与Enigma合作。 在第二次世界大战期间，德国人使用了数以万计的这种设备-不知道英国人正在监听和解密大多数消息。

于尔根·莱因霍尔德（JürgenReinhold）对此一无所知。 为了传递最重要的Enigma战略和国家重要信息，可以选择“ namber van”。 只有在了解到Enigma的漏洞之后，他们才恢复使用双重加密方法。 从雷纳德（Reinhold）故事的工作原理来看：有必要在字段A和B中寻找垂直的数字对，形成一个四边形，形成下一个四边形，然后在摩尔斯电码中以五位数的字符组输入结果。



区别在于使用Enigma和面部双重加密：密码的大小是一台打字机的大小，因此对于野营设备来说太大了，因此不适合在前线作战并徒步旅行的士兵使用。 双重加密中使用的“纸和笔”处理是此类设备的最佳选择。 此外，战线的消息经常在几个小时后就变得过时，甚至连敌人代码中最好的破解者也无法如此迅速地工作。

1942年11月，Kleditsa支队抵达斯大林格勒。 在那里，一组罗马尼亚情报人员访问了一个广播小组。 传说中的谜团给他们留下了深刻的印象。 “啊，la machina dechifrata！” -这就是他们的口中之物，因为他们也对设备的高安全性感到高兴，而不怀疑英国人早就能够闯入Enigma。

苏联和斯大林格勒锅炉在军事上取得成功后不久，克莱迪察部队就被解散了。 撤退开始了。 1945年5月9日，在捷克斯洛伐克，一个分散的团伙再次聚集，以摧毁他们的《恩尼格玛》副本。 即使在演习中，也向无线电操作员传授了一些知识：绝不应该让谜团落入敌人的手中，并且在灾难期间，应该吃掉纸上记录的加密密钥。 无线电运营商甚至练习过“吃”纸。

于尔根·莱因霍尔德（JürgenReinhold）在匈牙利战争结束后幸存下来。 即使在1945年5月8日战争结束之后，他仍然使用加密机。 他想离开匈牙利前往英国，但害怕红军。 使用在Enigma上加密的消息，在前同志的指导下，他得以到达目的地。 他的密码机模型很可靠：该设备从头到尾始终可以完美地工作。 于尔根（Jurgen）之后，用锤子摧毁了最后的谜团。

自从大多数士兵摧毁了他们的Enigma以来，迄今为止，只有两台加密机得以幸免。 收藏家愿意为传奇的Enigma支付约20,000欧元，而特殊型号的价格甚至更高。

设计的简单性，高可靠性以及最重要的是，优异的密码耐久性是使Enigma密码机享誉数百年的参数。 机电编码器第一次世界大战后首次出现。 1917年，美国人爱德华·赫本（Edward Hepburn）发明了一种旋转加密机，后来被称为“谜”。 她连接到一台电动打字机。 1923年，柏林工程师Arthur Sherbius创建了Enigma的独立工业版本。

Enigma A-Die Handelsmaschine（商业机器）是1923年由Scherbius＆Ritter在德国柏林开发的印刷加密机器。 这是第一款以Enigma品牌生产的加密设备。 据我们所知，目前尚无该模型的尚存示例。 Enigma A打印机制的不安全性导致1924年开发了新的机器模型。

Schreibende谜


新型号具有刻字杆（Typenhebel）-与传统打字机完全相同。 由于机械和制造问题，加密机的“介绍”被推迟到1926年。 1926年，它被Die Schreibende Enigma（印刷的Enigma）取代。

机器带有4个加密转子和每侧28个电触点，并带有逐步位移的齿轮机构。 另外，每个齿轮具有不同数量的齿，这确保了较长的加密时间和不规则的（即，难以预测的）步进。 机器内部安装了四个加密转子，其初始位置由4个手柄（在右侧）设定。 通过将手柄插入第五个手柄（位于最顶部），可以启动整个加密机制。 加密转子上方是一个5位计数器，借助它可以计算键盘上输入的字符数。



在加密机的前面有一个旋钮，用于选择3种模式：加密，解密和纯文本。 如果设置了“纯文本”模式，则Enigma就像一个简单的打字机一样工作。

在机器顶部，在背面，是一台可正常工作的打印机，就像那个时代的标准电动打字机一样。 它由一个纸架和一个花瓣打印机组成。 创建加密的文本时，字符以每组5个字母的形式打印，然后自动设置一个空格。 这样的一行适合十个这样的组（50个字符）。 键盘上有字母，数字，标点符号和空格。 两个Shift键用于在字母和数字之间切换。

Schreibende Enigma遇到机械问题，因此决定开发另一个模型。 它于1929年推出，后来被称为Enigma H29型或简称EnigmaH。H29是Enigma系列打印机的最新型号。 但是，可惜的是，这些模型并不可靠，尽管有改进，但实际上却收到了很多抱怨。

根据尚存文档中的描述，这正是Enigma A的外观（上图）。 它的尺寸为27 x 23 x 13厘米，重约5公斤。 键盘由26个键组成，分为两行。 每行上方是相应的灯泡。 按键是空的，任何字符都可以输入到顶部。

谜团b


机电加密旋转机Enigma B于1924年底由位于德国柏林的Cipher Machine Corporation（ChiMaAG）推出。 像其前身Enigma A一样，这里也使用了灯泡。



已知有两种版本的Enigma B：一种是早期的版本，前排有两排按键，转子上方有两排灯泡。 中央有两个固定的加密转子和一个可移动的接触反射器。



在更高版本中，一个加密转子是可移动的。 键盘的排列方式与传统打字机略有不同，字符以字母顺序排列。 灯板现在位于机器的中央，并且转子向后移动。

字母W丢失；在右下角添加了Å，Ä和Ö。 当时，除了名称，外来词和借用语外，W并未用于瑞典的官方语言，在这种情况下，也可以用VV（双V）代替。



据信，尚无早期模型B的幸存示例。 但是，至少有两个较新型号的副本（A-133和A-134），它们已成为该系列的一部分。 两个版本之间的差异是如此之大，以至于后者更可能过渡到EnigmaC。


在A-133内

A-133具有三个移动的加密转子-一个字母（AO），另两个数字（01-28）-和一个固定的反射器（FM），安装在转子的左侧。 可能的设置数量为2 891 341 824。

谜C




电动旋转加密机Enigma C于1925年中期由位于德国柏林的Cipher Machine Corporation（ChiMaAG）推出。 像它的前辈Enigma A和Enigma B一样，它使用灯泡。



Enigma C的组件：26个字母的键盘，用于输入打开的消息； 26个灯泡，当打开时会照亮所需的密文字母； 电源（3.5伏电池或等效电池）； 三个可移动的接触轮在同一轴线上旋转； 固定接触反射器； 固定输入接触轮。

军事谜


Enigma I是1927/29年为德国军队开发的机电加密机器，并于1932年投入运行。 基础是Enigma D，前面加了固定反光板和控制面板。 该机器曾在第二次世界大战期间使用，并以各种名称闻名。 自1938年12月以来，Enigma提供了五个磁盘，而不是三个磁盘：三个-在汽车中，两个远程。 德国军方坚持做出改变，从而大大提高了机器的耐用性。 其中最重要的是引入了位于键盘下方的开关。 26条开关连接器通过13条短电缆成对连接。 该设备使在输入和输出阶段交换组成该对的字母表字母成为可能。



这是在意大利发现的，当时是在第二次世界大战结束时丢失的行李中的一列火车上。

谜团I被陆军和空军使用，后来被德国海军使用，在这里被称为M1，M2，M3。 陆军版和海军版之间唯一明显的区别是，后者的轮子带有字母（AZ）而不是数字。 各种制造商生产了大约20,000辆这种军用车辆，但至今只有少数幸存下来。

直到1932年，所有Enigma商业模型都可以在国际市场上免费获得。 后来，所有商业和国际销售都获得了德国军队的批准。 Enigma H被出售给匈牙利军队，但由于价格高昂而从未受到欢迎。 此外，德国人继续向瑞士和荷兰军队出售Enigma。

第一台Enigma机器是由Scherbius和Ritter于1923年开发的，后来更名为Gewerkschaft Securitas（安全协会），后来又发展成为AG加密机器。 1929年，谢尔比乌斯（Sherbius）逝世后，公司移交给了另一个所有者的手中，1933年德国军队获得了制造Enigma机器的权利，公司名称再次更改为Heimsoeth和Rinke。

谜团的一种相当不寻常的变化。 尽管其设计显然基于Enigma D，但它只有10个按键和10个指示灯，每个指示灯由0到9的数字表示。它用于加密数字消息（即以前编码的消息），例如报告关于天气。 字母Z可能代表Ziffern或Zahlen（数字）。 加密机的正式名称为Z30。 总共有两个Enigma Z版本：标准版本和改进版本。

波兰突破（1933）


1930年，波兰密码局BiuroSzyfrów（波兰军事情报局的一个特殊部门）率先尝试破解Enigma密码。 作为德国最亲近的邻居，他们对另一个国家的危险非常了解。 商业版Enigma的研究已经开始。

波兰人是第一个被惊动的人。 1926年2月，他们注视着危险的邻居，突然无法阅读德国海军的加密资料，从1928年7月开始，他们也看不懂德国联邦国防军的资料。 很明显：他们在那里切换到机器加密。

该局聘请了波兹南大学的三位年轻的杰出数学家：玛丽安·里夫斯基，杰里·鲁基基和海因里希·齐加尔斯基。 然后，他们开始研究Enigma密码，只有几个截获的消息。

玛丽安·里夫斯基（Marian Reevsky）在1932年底解决了这个问题，几周后又取得了第一个突破：他拆除了谜团内部的秘密电线。 他与他的同事们一起开始开发各种解密德国交通的技术。


里夫斯基自行车里程表

Henry Zygalsky开发了所谓的Zigalsky床单。 后来，Zygalsky图纸方法实现了自动化-创建了传奇的Bomba。

他们的成功基于纯粹的数学分析，这得益于从德国间谍汉斯·提洛·施密特（汉斯·提洛·施密特（代号Asche））以及商业谜语（通过波兰邮件截获）获得的信息。随后，他们获得了商业之谜，并利用收集到的所有信息将机器转变为军用机器。这种经过改装的机器成为了著名的波兰谜语。

从1930年开始，德国人使用了一种非常简单的密钥管理方案，在每条消息的开头，随机选择的消息密钥以加密形式发送了两次。这样就可以使用手动方法相对轻松地恢复每日密钥。


密码学孟买是英国对德国之谜的回应。

根据波兰人提供的信息，英国数学家艾伦·图灵（Alan Turing）开发了传说中的图灵-韦尔希曼·邦贝（Turing-Welchman Bombe）。密码炸弹的更高级版本-一种用于解密Enigma代码的电子机械。

在布莱奇利公园在战时，安装了212台Bombe机器，借助它们的帮助，每天可以解密多达3千条消息。战争结束后，所有的邦贝都被摧毁。60年后，借助详细的图纸和说明，机器被修复，花了10年的时间。

由Turing开发的解码器的工作原理包括枚举密码密钥的可能选项，如果知道解密消息的结构或部分明文，则尝试对文本进行解密。


第二次世界大战之前和期间，Enigma是其他加密机（例如英国Typex和美国Sigaba）开发的灵感。甚至在第二次世界大战之后，一些密码机仍基于与Enigma相同的原理。这是美国的KL-7，俄罗斯紫罗兰和瑞士Nema。
