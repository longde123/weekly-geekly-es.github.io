Montre deux jours sur attiny2313 et ds1307

Motif


Pour commencer, chaque personne vit à une époque qui, malheureusement, ne peut être arrêtée. Et puisque nous vivons dans ce monde, alors gardons une trace du temps et ne manquons pas une minute, en le gaspillant. Pour cela, j'ai fabriqué ce merveilleux appareil, appelé une montre .


Commençons!


Le soleil s'est levé dans la rue, j'ai dessiné un circuit imprimé et je l'ai gratté sur un copieur pour imprimer le circuit.



De plus, ceux qui savent comment les circuits sont fabriqués comprendront que j'ai transféré le dessin avec un fer à repasser sur textolite, gravé, étamé, etc., et ceux qui ne le savent pas - lire sur Internet la fabrication de circuits imprimés selon la méthode LUT - (technologie de repassage laser).
J'ai oublié de tirer le résultat, mais je pense que la qualité sera visible sur la planche finie: On



peut voir des endroits qui ne sont pas remontés, mais rien de mal n'est arrivé, seule la décharge a pris du retard.
Ensuite, il a fallu souder les composants, sur lesquels je ne m'attarderai pas particulièrement, il suffit d'en laisser une liste:
  • AtTiny2313 - Microcontrôleur 8 bits
  • ds1307 - puce de compteur avec une bonne précision
  • quartz d'horloge à 32,768 kHz
  • 2 x résistances 10K
  • 7 résistances par 10 ohms
  • 1 résistance de 5,2 kΩ
  • 4 résistances pour 1 kOhm
  • 4 transistors KT315 (leurs frères modernes)
  • Buzzer 5 volts
  • 2 boutons d'horloge (montage en coin)
  • 4 indicateurs (10011-BSR) anode commune
  • 47 microfarad tantale
  • et prise de courant et support d'angle (diamètre du trou - 5,1 mm, diamètre du contact central - 1,5 mm)

Et ne laissez pas la longueur de cette liste vous déranger, car nous assemblons l'appareil depuis de nombreuses années, et il devrait être assemblé au bon niveau!

Le programme


Comme tous les appareils, une montre a besoin d'un programme. Dans ce cas, le programme est écrit en BASIC dans BASCOM AVR. Ce langage est très pratique pour la programmation, car le code est clair, efficace et ne prend pas beaucoup de place sur le microcontrôleur. Je n'ai pas commencé à écrire le programme à partir de zéro, mais j'ai pris le code source d'un article sur le site avec un fer à souder. Ceux qui sont intéressés trouveront, je laisse juste le code que j'ai finalisé ici:

Portd.0 = 1
Config Pind.1 = Input

Config Timer1 = Timer , Prescale = 64

Config Timer0 = Timer , Prescale = 64

Dig1 Alias Porta.0 : Dig2 Alias Portd.2 : Dig3 Alias Portd.3 : Dig4 Alias Portd.6

Porta.1 = 1
Waitms 100

Mine = 0

I2cstart
I2cwbyte &HD0
I2cwbyte 0

Mine = Makebcd(mine)
I2cstart
I2cwbyte &HD0
I2cwbyte 1
I2cwbyte Mine
I2cstop

Hour = Makebcd(hour)
I2cstart
I2cwbyte &HD0
I2cwbyte 2
I2cwbyte Hour
I2cstop
Waitms 1000
Porta.1 = 1
Waitms 10
Porta.1 = 0
End If

Do
I2cstart
I2cwbyte &HD0
I2cwbyte &H00
I2cstart
I2cwbyte &HD1
I2crbyte Seco , Ack
I2crbyte Mine , Ack
I2crbyte Hour , Nack
I2cstop
Seco = Makedec(seco)
Mine = Makedec(mine)
Hour = Makedec(hour)

If Seco = 80 Then
Seco = 10
Seco = Makebcd(seco)
I2cstart
I2cwbyte &HD0
I2cwbyte 0
I2cwbyte Seco
I2cstop
End If

If Hour > 9 Then
Number(1) = Hour / 10
Number(1) = Abs(number(1))
Else
Number(1) = 20
End If

Number(2) = Hour Mod 10

If Mine > 9 Then
Number(3) = Mine / 10
Number(3) = Abs(number(3))
Else
Number(3) = 0
End If

Number(4) = Mine Mod 10

If Pind.1 = 0 Then
Porta.1 = 1
Waitms 10
Porta.1 = 0
If Mine = 59 Then
Mine = 0
Else
Incr Mine
End If

Mine = Makebcd(mine)
I2cstart
I2cwbyte &HD0
I2cwbyte 1
I2cwbyte Mine
I2cstop
Else

If Pind.0 = 0 Then
Porta.1 = 1
Waitms 10
Porta.1 = 0
If Hour = 23 Then
Hour = 0
Else
Incr Hour
End If
Hour = Makebcd(hour)
I2cstart
I2cwbyte &HD0
I2cwbyte 2
I2cwbyte Hour
I2cstop
End If
End If

Waitms 250
Loop

Awake:
If Point = 1 Then
Porta.1 = 1
Waitus 10
Porta.1 = 0
End If
Toggle Point
Return

Refresh:
Reset Dig1 : Reset Dig2 : Reset Dig3 : Reset Dig4
