Platform Wave Blockchain Diinangi di Microsoft Azure Cloud

Platform blockchain kami untuk mengeluarkan token kriptografis dan kampanye crowdfunding Waves Platform di-host di cloud Microsoft Azure Blockchain sebagai Layanan (BaaS). Integrasi ke dalam BaaS membuat Waves tersedia untuk mengimplementasikan berbagai solusi blockchain di seluruh dunia.



Saat ini, Microsoft Azure adalah salah satu layanan cloud terbesar. Platform ini memberi pelanggan peluang untuk mengembangkan dan mengeksekusi aplikasi, menyimpan data di server yang berlokasi di pusat data terdistribusi, membuat blockchains pribadi dan membuat kontrak pintar yang dijalankan secara otomatis ketika tindakan tertentu dilakukan atau persyaratan dipenuhi.

BaaS adalah lingkungan berbasis Azure aman yang mencakup layanan dan alat cerdas untuk mengembangkan, menguji dan menjalankan aplikasi blockchain dan memungkinkan pengguna untuk menguji berbagai ide blockchain dalam praktiknya. Platform ini menyediakan serangkaian layanan cloud yang dapat digunakan oleh pengembang dan profesional TI untuk mengembangkan, menjalankan, dan mengelola aplikasi melalui jaringan pusat data Microsoft di seluruh dunia.

Dari sudut pandang hierarki cloud klasik, IaaS-PaaS-SaaS (IaaS - Infrastruktur sebagai Layanan - menyediakan kekuatan komputasi awan; PaaS - Platform sebagai Layanan - menyediakan tingkat abstraksi tertentu, menyediakan platform yang sudah jadi, menghilangkan kebutuhan untuk mengkonfigurasi secara langsung sumber daya komputasi; SaaS - Perangkat Lunak sebagai Layanan - memberikan konsumen layanan utama dan sepenuhnya abstrak dari infrastruktur dan platform cloud) BaaS adalah PaaS, sebuah platform untuk mengembangkan proyek-proyek blockchain.

Menambahkan Gelombang ke Azure memberikan peluang baru bagi perusahaan dan pengembang perangkat lunak yang ingin, misalnya, untuk membuat token mereka di blockchain.

Menurut Konstantin Goldstein, penginjil teknis di Microsoft Rusia, memperluas kumpulan platform di BaaS menciptakan lingkungan yang unik bagi pengembang dan perusahaan yang tertarik untuk mengimplementasikan proyek-proyek blockchain. Dalam satu klik, BaaS akan membantu untuk menyebarkan infrastruktur blockchain, serta mengintegrasikan teknologi canggih seperti kecerdasan buatan, Internet hal-hal dan lain-lain ke dalam proyek-proyek ini.

Microsoft Azure sudah memiliki sejumlah besar solusi blockchain yang tersedia, termasuk Ethereum, ChainCore, Corda, Nxt, Lisk - dan sekarang Waves. Ini membentuk ekosistem komprehensif Blockchain sebagai Layanan untuk implementasi berbagai proyek blockchain untuk segala kebutuhan dan untuk perusahaan dengan ukuran berapa pun.
