大家好

这篇短文的写作促使我就电视的选择展开争论。

如今，在这一领域以及“相机的百万像素”领域中，营销bacchanalia一直在追求许可：HD Ready早已被Full HD取代，并且4K甚至8K已经越来越流行。

让我们看看-我们真正​​需要什么？

学校的几何课程和Wikipedia的一些基础知识将在此方面为我们提供帮助。

因此，根据这个非常维基百科的内容 ，普通人的裸眼是一种独特的设备，可以同时以130°-160°的角度查看空间，并以1-2'（约0.02°-0.03°）的角度区分元素。 在这种情况下， 快速对焦发生在10厘米（年轻人）至50厘米（大多数50岁以上的人）到无限远的距离 。

看起来很酷。 其实不是那么简单。

不幸的是，人眼无法在整个平面上以宽角度提供相同的视觉质量。 是的，用两只眼睛我们可以识别出前方180°范围内的物体，但是我们只能识别110°以内的三维物体（至绿色区域），而只有约60°-70°较小范围内的全色物体（至蓝色区域）才能识别它们。 是的，在某些鸟类中，视场几乎达到360°，但是我们拥有所拥有的。

因此，我们得到一个人以60°-70°的视角接收最高质量的图像 。 如果您需要更多的覆盖范围，我们将被迫通过图像“奔跑”。

由于我们对图像的高度和宽度具有较大的参数-并且眼睛需要在整个宽度上覆盖整个屏幕-因此，考虑到如上所示，可视角度不应超过70度，因此我们计算出到屏幕的最佳距离：



也就是说： 为了使眼睛能覆盖整个屏幕，我们的距离必须不小于屏幕对角线的一半 。 同时，此距离至少应为50厘米，以使任何年龄的人都能舒适地对焦。 记住这一点。

现在，让我们计算一个人区分屏幕上像素的距离。 这是一个与角度的切线相同的三角形，在这种情况下，只有R是像素大小：



即：在大于2873.6像素大小的距离处，眼睛看不到颗粒感。 因此，考虑到上方像素边的计算，您必须与屏幕保持以下最小距离，以便图片正常：

  • HD就绪：D / 1468.60 x 2873.6 = 1.96D，即两个屏幕对角线
  • 全高清：D / 2202.91 x 2873.6 = 1.3D，即比对角线屏幕少一半左右
  • 超高清4K：D / 4405.81 x 2873.6 = 0.65D，即屏幕对角线的一半多一点


现在一切都变成了-

结论


  1. 您离屏幕的距离不应超过50厘米-眼睛将无法正常聚焦在图像上。
  2. 您不应坐在屏幕对角线的0.63附近，否则眼睛会感到疲倦，因为它必须绕过图片。
  3. 如果您打算以大于两个屏幕对角线的距离看电视-不要购买比“高清就绪”还凉的东西-您将不会注意到两者之间的区别。
  4. 如果您打算在大于屏幕对角线一半的距离处看电视-不要购买比Full HD凉爽的东西-您不会注意到两者之间的区别。
  5. 仅当您看着屏幕的对角线距离小于一半且小于对角线的一半时，才建议使用4K。 也许这是某种计算机游戏监视器或巨型面板，或者是紧靠电视的椅子。
  6. 使用更高的分辨率是没有意义的-您要么看不到4K的差异，要么您太靠近屏幕并且视角将无法覆盖整个平面（请参见上面的第2段）。 通过弯曲的屏幕可以部分解决该问题-但计算（更为复杂）表明此增益极为可疑。

现在，我建议测量您的房间，您最喜欢的沙发的位置，电视的对角线并思考：多付一点钱有意义吗？
