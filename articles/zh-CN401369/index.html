智能手机配有USB充电器和短USB-micro USB适配器。



智能手机的外壳为塑料，但拉丝铝的表面处理质量很高。有三种颜色。





相机上方是麦克风之一的孔。摄像头模块本身略微凹入外壳中。闪光灯是普通的一种颜色。相机下方是指纹扫描仪。从视觉上看，这并没有引起人们的注意，但是从触觉上可以看出，扫描仪比相机更坚固地进入了人体，因此不可能将它们混在一起并想知道为什么智能手机无法解锁。



底部有一个大型扬声器网。



ZenPhone 3 Max没有锐利的边缘或角落：外壳光滑，有衬里，甚至显示屏的保护玻璃边缘也被弄圆了。



由于所有元素的“流线型”形状和平滑度，智能手机在您的手掌中非常舒适。从战术上讲，只有音量和电源按钮高亮显示。



顶部是一个3.5毫米组合音频插孔。





下方是一个微型USB连接器和一个用于第二个麦克风的孔，左侧是一个用于存放micro和nano格式的SIM卡的托盘。







您可以将两个SIM卡（微型和纳米）插入托盘，或者将SIM卡（微型）和存储卡插入。

展示架


ZenPhone 3 Max在IPS矩阵上配备了一个显示屏，对角线为5.2英寸，分辨率为1280 x720。图像完全平滑，即使在圆形图像上也看不到任何渐变。视角大，传统上显色质量高。最大亮度为400 cd / m2。

软体类


设备上是具有专有图形外壳ZenUI 3.0的Android 6.0。它允许您下载和安装各种主题。系统按钮“后退”，“主页”和“启动的应用程序”不是以屏幕下方的单独触摸面板形式出现的，而是显示在屏幕底部。


当然，品牌应用程序已预先安装在智能手机上：ShareLink，ZenCircle，ZenTalk等。

摄影机


对于主相机，使用分辨率为13兆像素的矩阵。五元素透镜由Largan Precision制造。前置摄像头分辨率-5兆像素。

后面板上的指纹扫描仪不仅可以用于预定目的-解锁智能手机-而且可以控制相机。例如，从主摄像头切换到前置摄像头，以及释放快门。

性能表现


ZenFone 3 Max基于联发科的MT6737T平台。它配备了两个ARM处理器：中央64位四核Cortex A53（最高工作频率为1.5 GHz）和图形Mali-T720。RAM的数量-2或3 GB。

ZenPhone 3 Max的一个特色是它的电池-4130 mAh。此外，智能手机可以在为其他小工具使用外部充电器的模式下工作，为此，使用了随附的USB-micro USB适配器。



得益于高能效的硬件和大容量电池，智能手机可以长时间不充电就可以正常工作。PC标记压力测试结果：


您是否找到了一次只需一次充电就可以“居住” 4-7天，不仅躺在那里而且还非常活跃地用于通话和SMS的时代？在智能手机时代，我们已经忘记了它是什么-几天不记得充电器了。并连续使用了13个小时ZenPhone 3 Max激发了希望，不仅历史在螺旋式发展，而且技术也在不断发展。毕竟，超过半天的冲浪时间相当于将近24小时的通话时间：根据制造商的说法，通话时间为20小时。此外，据指出，智能手机在待机模式下最多可躺30天。我怀疑为此需要将其置于省电模式并关闭所有可能的功能，仅保留蜂窝功能。但是，即使他处于这种状态也不能持续两个月，而是一个月，这对智能手机的“长寿”迈出了一大步。

结论


华硕ZenPhone 3 Max（ZC520TL）是一款现代智能手机，性能可与华硕ZenFone 3 Laser媲美。甚至他们的身体都差不多。但是，它们并不是直接竞争对手：激光具有更高的屏幕分辨率和改进的相机对焦系统，而麦克斯的电池容量则多了三分之一。因此，如果您不喜欢在账单付清几秒钟的情况下拍照，如果您经常在很长一段时间内都离网点很远，那么请注意ZenFone 3 Max –就“寿命”而言，它没有很多竞争对手。
