Pemilik arsitektur MIPS bermaksud untuk membuka set instruksi (ISA) pada tahun 2019



Wave Computing pada 17 Desember 2018 mengumumkan peluncuran program MIPS Open , di mana para peserta akan memiliki akses gratis ke set instruksi MIPS bit 32 dan 64 (arsitektur set instruksi MIPS (ISA)) (untuk perintah dan penambahan itu sendiri, lihat detailnya di situs mereka). Selain itu, peserta program akan menerima lisensi untuk paten perusahaan (dibayar atau gratis - belum jelas). Program MIPS Terbuka akan dikelola oleh Wave Computing dan dewan penasihat yang diharapkan mencakup produsen peralatan, mitra, universitas, dan tokoh ilmu pengetahuan.
Tindakan ini harus memberikan gerakan kepada komunitas pengguna arsitektur MIPS , yang sangat populer sebelumnya dan tetap signifikan saat ini. Rupanya, pemilik arsitektur MIPS takut dilupakan karena penyebaran ARM berbayar di satu sisi, dan RISC-V gratis (RISC-lima) di sisi lain.
Perhatikan bahwa sementara itu, pernyataan niat telah disuarakan, dengan asumsi kewajiban diharapkan nanti. Klarifikasi masalah ini harus diharapkan pada kuartal pertama tahun 2019.
Wave Computing mengumumkan akuisisi MIPS Technologies pada 15 Juni 2018 dalam siaran persnya . Wave Computing mengumumkan fokusnya pada AI, ML, Deep Learning, dll. (Secara umum, tema berkumpul tema modis).
Melihat sekelilingnya, pembaca dapat menemukan satu atau lebih perangkat yang didasarkan pada arsitektur MIPS. Di Rusia ada banyak perkembangan menggunakannya.
Sebagian pesan tentang penjualan MIPS Technologies di situs web iXBT harus diberikan di sini:
Perhatikan bahwa kesepakatan itu tidak mengejutkan karena CEO Wave Computing Derek Meyer pernah menjadi Wakil Presiden Penjualan dan Pemasaran MIPS; Mike Uhler, Wakil Presiden Operasional Wave Computing, sebelumnya adalah MIPS CTO; dan Darren Jones, wakil presiden bidang teknik Wave Computing, sebelumnya mengepalai bisnis yang sama di MIPS.
