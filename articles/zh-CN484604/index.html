新年，新浏览器：Microsoft Edge超出初步评估，现在可以下载



一年多以前，我们宣布了基于开源项目Chromium重建Microsoft Edge的打算。 这对于提高所有用户的兼容性，减少Web开发人员的碎片并与Chromium社区建立合作关系以改善Chromium本身的核心是必要的。 在Ignite上，我们展示了我们对Web技术和搜索的新视野，新的彩色徽标以及Microsoft Edge + Bing的新概念-Microsoft Edge + Bing –商业浏览器和搜索引擎。 现在，我们期待您的反馈！

我们的工作非常出色，今天我们很高兴地宣布，新的Microsoft Edge已经可以在所有受支持的Windows和macOS版本中以90多种语言下载。 Microsoft Edge也可在iOS和Android上使用，这意味着强大的跨平台支持。 新的Microsoft Edge提供世界一流的性能，并具有更高的隐私性，速度和效率，是以前Edge的两倍。 我们的隐私权政策适用于新的浏览器，您将欣赏新功能，例如防跟踪功能，该功能默认情况下处于启用状态，并在浏览Web时提供三级控制。

Microsoft Edge的另一个创新功能允许您自定义与网页的交互，例如，选择新选项卡的布局，设计和内容，以更好地利用在线环境。



在过去的几个月中，我们一直在热情地致力于Microsoft Edge的新功能，包括AAD支持，Internet Explorer模式，4K视频流，杜比音频，PDF手写，Microsoft Search与Bing集成，对基于Chrome的扩展的支持以及还有更多。

如果您是企业或教育部门的IT管理员，并且想要在您的组织中进行广泛部署，我们也考虑过您：您可以下载独立的软件包和策略，以及在新的商业站点上找到所有详细信息。



人们已经在其设备上数百万次下载了新Microsoft Edge的预发布版本，并且许多组织已经开始在其用户中部署这些版本。 具有重要的旧应用程序和网站，但又需要现代Web界面和安全性的企业和学校，使用了我们的新Internet Explorer模式“用一块石头杀死两只鸟”。 Microsoft 365的用户只需在Microsoft Edge的地址栏中输入所需的查询，即可在其组织的本地网络中使用Microsoft Search搜索文件，人员，办公计划以及更多内容。 那些已经测试了这些功能的人对此表示赞赏。

切换到新的Microsoft Edge-会发生什么？


现在我们已经达到了这个里程碑，计算机和小工具用户可能想知道会发生什么。 有两种获取新Microsoft Edge的方法：您可以立即手动下载它，或者，如果您是普通用户，请等到使用Windows Update将它自动安装在设备上。 当您决定切换时，您的密码，收藏夹，填写表格的信息和基本设置将自动传输到新的Microsoft Edge。 在此处阅读有关我们的部署计划的更多信息。

如果您是IT管理员，则由于新的Microsoft Edge不会自动部署到商业客户，因此您需要下载一个独立的部署程序包以在企业环境中进行试用部署。 此外，所有Microsoft Edge Insider通道都不会升级到新的Microsoft Edge，因为这些通道可以并行用于测试和验证。

我们了解到，部署新的浏览器不仅仅是“转动开关”，因此我们希望使过渡尽可能容易。 除了使用Intune和Configuration Manager之类的工具简化部署之外，我们还可以做更多的工作来帮助您的组织迁移到新的Microsoft Edge。 在Ignite，我们宣布了对Microsoft Edge的FastTrack和App Assure的支持。 如果您已经有Microsoft 365，Azure或Dynamics 365的相应订阅，FastTrack将帮助您免费在组织中安装Microsoft Edge。如果您的站点与Internet Explorer 8和更高版本，Google Chrome或旧的Microsoft Edge兼容，则它们将与新的Microsoft Edge一起使用。 如果没有，请联系App Assure，我们将帮助您修复它。

接下来是什么？


当然，创新，测试和新功能将不会停止出现。 此第一个版本仅仅是开始。 如果您想知道接下来会发生什么，建议您继续使用我们的Beta，Dev和Canary预评估渠道，这些渠道仍可从Microsoft Edge Insider网站上下载。 您不仅可以了解我们的Microsoft Edge功能管道，而且仍可以通过宝贵的反馈帮助改进Microsoft Edge。 您的参与将有助于使新的Microsoft Edge和整个Internet变得更好-对于世界上的所有人。

谢谢你


非常感谢我们的Microsoft Edge Insiders社区以及与我们合作创建新的Microsoft Edge的Chromium工程师。 我们将继续积极参与开源铬项目，并为它的发展做出贡献。 迄今为止，我们在可访问性，现代数据输入（包括触摸，语音，数字和手写）以及许多其他方面提供了1900多项改进。

不断告诉我们什么有效，哪些需要更改。 关于您希望在将来的Microsoft Edge版本中看到的内容。
