Änderungen an Microsoft-Zertifizierungen und -Prüfungen

Auf einer kürzlich abgehaltenen Ignite- Konferenz stellte Microsoft ein neues Zertifizierungsprogramm vor. Das Unternehmen sagt, dass neue Zertifizierungen basierend auf Rollen / Spezialisierungen Ihnen helfen werden, Ihre Qualifikationen zu verbessern, Ihre Fähigkeiten und Kenntnisse gegenüber Arbeitgebern zu demonstrieren und neue Möglichkeiten zu entdecken.


Es wurden sechs neue Besonderheiten angekündigt: Azure Administrator , Entwickler , Lösungsarchitekt , DevOps Engineer, Microsoft 365 Modern Desktop Administrator und Enterprise Administrator. Prüfungen und Schulungen für die Azure Administrator-Zertifizierung sind bereits verfügbar, und die nächsten beiden Prüfungen - Azure Developer und Azure Solutions Architect - befinden sich in der Beta-Phase. Die verbleibenden drei Zertifizierungen aus dieser Liste werden vor Jahresende ausgestellt. „Und das ist erst der Anfang! Wir planen, 2019 weitere Spezialisierungen anzukündigen. “ - Selina Winter schreibt im Microsoft- Blog .




Zusätzlich zu den üblichen Fragen mit mehreren Antworten versprechen sie, neue Fragen hinzuzufügen, die auf aktiven Szenarien, Drag-and-Drop-Variationen und Situationen basieren, die den funktionierenden näher kommen.


Im Folgenden werden die Azure Administrator-Zertifizierungsschulungen aufgeführt.




Wichtig! Viele Prüfungen der 70-XXX-Reihe gelten ab dem 31. Dezember 2018 als veraltet und werden durch neue Prüfungen der Reihe ersetzt: AZ-10X, AZ-20X, AZ-30X, MS-10X.

Wenn Sie in naher Zukunft Microsoft-Prüfungen ablegen möchten, überwachen Sie die Informationen auf den englischsprachigen Prüfungsseiten sorgfältig. Vielleicht wurde Ihre Prüfung bereits durch eine neue ersetzt, und es ist praktischer, die neue sofort zu bestehen / vorzubereiten, als die alte zu bestehen.

Alle Microsoft Azure-zertifizierten Fachleute haben jetzt die Möglichkeit, die 99-Dollar-Übergangsprüfungen zu bestehen. (Der Preis variiert je nach Land / Region und spiegelt sich im Registrierungsprozess der Prüfung wider.) Sie sind bis zum 30. Juni 2019 verfügbar.


Übergangsprüfungen decken nur neues Material und den Unterschied zwischen der aktuellen Zertifizierung und den Fähigkeiten und Konzepten der neuen Zertifizierung ab.


Wenn Sie eine der folgenden Prüfungen bestehen (oder vor dem 31. Dezember 2018 bestehen, gelten sie nach diesem Datum als veraltet):


  • 70-533: Implementieren von Microsoft Azure Infrastructure-Lösungen
  • 70-532: Entwickeln von Microsoft Azure-Lösungen
  • 70-535: Microsoft Azure-Lösungsarchitektur

Sie können die entsprechende Übergangsprüfung bestehen (verfügbar bis 30. Juni 2019):


  • AZ-102: Microsoft Azure-Administrator. Übergangsprüfung
  • AZ-202: Entwickler von Microsoft Azure. Übergangsprüfung (Beta)
  • AZ-302: Microsoft Azure Solution Architect. Übergangsprüfung (Beta)

Es gibt also zwei Möglichkeiten, zwei Möglichkeiten, um ein neues Zertifikat zu erhalten.


1. Bestehen Sie eine veraltete Prüfung + Übergangsprüfung und erhalten Sie ein neues Zertifikat


70-533 + AZ-102 -> Microsoft Certified Professional: Azure Administrator
70-532 + AZ-202 -> Microsoft Certified Professional: Azure Developer
70-535 + AZ-302 -> Microsoft Certified Professional: Azure Solution Architect


Oder


2. Bestehen Sie zwei neue Prüfungen und erhalten Sie ein neues Zertifikat


AZ-100 + AZ-101 -> Microsoft Certified Specialist: Azure Administrator
AZ-200 + AZ-201 -> Microsoft-zertifizierter Spezialist: Azure Developer
AZ-300 + AZ-301 -> Microsoft Certified Professional: Azure Solution Architect


Ich wiederhole noch einmal, dass die erste Methode nur bis zum 30. Juni 2019 verfügbar sein wird.


Wenn Sie derzeit daran arbeiten, das MCSE- Zertifikat zu erhalten : Cloud Platform and Infrastructure , haben Sie Zeit bis zum 31. Dezember 2018 . Ab dem 1. Januar 2019 wird dieses Zertifikat durch die neue MCSE: Basic-Infrastruktur ersetzt, die MCSA Windows Server 2012 und MCSA Windows Server 2016 umfasst .




Ab dem 31. Dezember 2018 als veraltet geltende Prüfungen


  • 70-496: Verwalten von Visual Studio Team Foundation Server
  • 70-497: Softwaretests mit Visual Studio
  • 70-498: Bereitstellen eines kontinuierlichen Werts mit Visual Studio Application Lifecycle Management
  • 70-532: Microsoft Azure-Lösungsentwicklung (ersetzt durch Prüfungen: AZ-200 und AZ-201 )
  • 70-533: Implementieren von Microsoft Azure Infrastructure-Lösungen (ersetzt durch Prüfungen: AZ-100 und AZ-101 )
  • 70-535: Microsoft Azure Solution Architecture (ersetzt durch Prüfungen: AZ-300 und AZ-301 )

  • 70-346: Office 365-Identitäts- und Anforderungsmanagement (ersetzt durch Prüfungen: MS-100: Identifizierung und Dienste Microsoft 365 und MS-101 Mobilität und Sicherheit Microsoft 365, Beta-Prüfungen sind für Ende 2018 geplant)
  • 70-347: Installation von Office 365-Diensten (ersetzt durch Prüfungen: MS-100: Identifizierung und Dienste Microsoft 365 und MS-101 Mobilität und Sicherheit Microsoft 365, Beta-Prüfungen sind für Ende 2018 geplant)
  • 70-695: Bereitstellung von Windows-Desktop- und Unternehmensanwendungen (ersetzt durch Prüfungen: MD-100: Windows 10 und MD-101: Modern Computer Management, Beta-Prüfungen sind für Ende 2018 geplant)
  • 70-697: Konfigurieren von Windows-Geräten (ersetzt durch Prüfungen: MD-100: Windows 10 und MD-101: Verwaltung moderner Computer, Beta-Prüfungen sind für Ende 2018 geplant)
  • 70-698: Installieren und Konfigurieren von Windows 10 (ersetzt durch Prüfungen: MD-100: Windows 10 und MD-101: Verwaltung moderner Computer, Beta-Prüfungen sind für Ende 2018 geplant)
