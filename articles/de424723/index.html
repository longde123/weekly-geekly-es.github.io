Skillbox Friday Webinars: Nützlich für Anfänger und mehr



Hallo nochmal. Der Freitag kam unerwartet, daher ist es Zeit, eine neue Sammlung von Webinaren zu verschiedenen Programmnuancen zu veröffentlichen. In der heutigen Ausgabe finden Sie Informationen sowohl für Anfänger als auch für bereits mehr oder weniger erfahrene Programmierer.

Wir erinnern Sie daran: Für alle Leser von „Habr“ - ein Rabatt von 10.000 Rubel bei der Anmeldung für einen Skillbox-Kurs mit dem Aktionscode „Habr“.

Wie werde ich ein C # -Entwickler?


Der Autor des Webinars ist Mikhail Ovchinnikov, Lead Badoo Software Engineer. Er spricht über die Attraktivität der C # -Sprache, was in dieser Sprache entwickelt werden kann, wie man mit der Programmierung beginnt und dies so effizient wie möglich macht.


Wie man Programmierer in einer stressigen Umgebung arbeitet


Nikolai Baryshnikov, ehemaliger technischer Direktor der Skillbox Online-Universität, spricht darüber, wie man in einer stressigen Umgebung arbeitet.

Was ist eine stressige Situation und wie kann man sich darauf vorbereiten? Wie verwalte ich meine eigene Ladung? Wie kann man mehrere Projekte gleichzeitig durchführen, ohne das persönliche Leben zu vergessen?
Welche Tools stehen für Stressbewältigung und Gehirnentladung zur Verfügung und wie funktionieren sie? All diese und einige andere Fragen werden in diesem Webinar behandelt.

Online-Bildungskurs "Beruf Java-Entwickler" .


Automatisierungstools für die mobile Entwicklung


Dmitry Shashlov spricht über Automatisierung im Mobilbereich: Kontinuierliche Integration,
Continuous Delivery, Autotests und Code-Team-Lösungen - Gitflow, Pull-Request und Code-Überprüfung.

Praktikum "Mobile Developer PRO" .


So programmieren Sie in Python


Python ist eine leistungsstarke Programmiersprache, die sich hervorragend für viele Aufgaben eignet. Der Skillbox-Mentor Vadim Shandrinov spricht über die Funktionen der Sprache und erklärt, wie man in Python programmiert.

Das Ergebnis des Webinars ist das Schreiben Ihres ersten Programms in dieser Sprache.
