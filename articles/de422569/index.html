Stündliche Zeiterfassungs-App

Hourly ist eine iPhone-App, mit der Sie die für Aufgaben aufgewendete Zeit verfolgen können. Sie können alles in Betracht ziehen: Wie viel Sie aus der Zielseite machen, den Code umgestalten, eine Codeüberprüfung durchführen, den Habr lesen, Auchan umrunden, zur Arbeit gehen.



Die Anwendung ist am nützlichsten für Freiberufler, die auf Uhrbasis arbeiten - um den Arbeitsprozess zu melden und zu steuern.

Aber für diejenigen, die auf die übliche Weise arbeiten (für ein Gehalt), hilft es auch: Wenn Sie herausfinden, wie viel Arbeit erledigt wird, wird es einfacher, einen Arbeitsplan zu planen. Das habe ich schon getan, es dauert im Durchschnitt so viel, aber das ist so viel. Außerdem können Sie Ihre temporären Parasiten analysieren (z. B. eine Aufgabe zur Förderung oder Sozialisierung sozialer Netzwerke starten).

Laden Sie die Anwendung in Apstor herunter . Leider ist der Android-Entwickler noch nicht im Team, sodass der Link zu Google Play viel später erfolgt.

Schauen Sie sich auch die Promo-Seite der Anwendung an, insbesondere die Hover-Seite (aber wenn Sie den Inspektor öffnen, wundern Sie sich nicht - der Designer hat sich etwas ausgedacht).

Empfehlen Sie die Anwendung Freunden, die sie möglicherweise nützlich finden.

Team:

  • Entwickler - Evgeny Rtishchev ( katleta )
  • Designer - Mikhail Kapanaga
  • Herausgeber - Tatyana Adyakova
