Browser tepi dengan Chromium muncul di layanan hosting file



Akhir pekan lalu, build Edge Browser berbasis Chromium bocor ke jaringan. Dengan demikian, pengguna mendapat kesempatan untuk mencoba browser baru di kantor. Mereka yang telah melakukan klaim ini bahwa Edge sekarang terlihat dan berfungsi hampir sama dengan Chrome, dan fungsionalitas browser web ini juga hampir sama.

Microsoft tidak secara aktif berbagi informasi tentang browser barunya, tetapi berkat bocoran data, lebih banyak diketahui tentang hal itu. Nah, sekarang, dengan tampilan build itu sendiri di layanan hosting file, sudah menjadi jelas bagaimana browser terlihat.

Para ahli percaya bahwa ada kemungkinan yang sama sekali tidak nol bahwa Microsoft akan mengambil bagian pasar (walaupun kecil) dari Google Chrome. Setelah Microsoft beralih ke Chroimum, para ahli menyatakan keprihatinannya bahwa pasar browser secara bertahap akan mulai masuk ke tangan Google. Tetapi jika kita berbicara tentang OS desktop, situasinya mungkin agak berbeda.

Sebagian besar pengguna reguler bekerja dengan Windows. Dan ini berarti bahwa jika Edge mengubah (untuk yang lebih baik) fungsi dan antarmuka (plus ekstensi untuk Chrome muncul), maka pengguna dapat meninggalkan browser lain demi perangkat lunak Windows "asli".

Mungkin saja Microsoft pada akhirnya akan mempengaruhi pengembangan browser dan pembuatan standar web baru. Untuk Firefox dan Opera, ini mungkin tidak terlalu bagus.

Seperti Apa Peramban Microsoft Edge 75

Perangkat lunak ini didasarkan pada Chromium 75, versi saat ini adalah 75.0.107.0. Di awal pekerjaan, browser bertanya kepada pengguna apakah dia ingin mengimpor data dari Chrome, termasuk favorit, formulir lengkapi otomatis, riwayat.



Pengguna yang telah memasang peramban, percaya bahwa itu lebih dari siap untuk bekerja, meskipun faktanya ini adalah versi uji perangkat lunak.



Korporasi telah memodifikasi antarmuka browser sehingga terlihat seperti produk dari Microsoft. Misalnya, halaman pengaturan menerima bilah navigasi yang terletak di sebelah kiri, seperti kebanyakan aplikasi dari Windows 10.

Tentu saja, perusahaan telah menerapkan layanannya sendiri di browser ini. Misalnya, Penjelajahan Aman Google telah dihapus demi SmartScreen, yang dikembangkan oleh karyawan Microsoft.



Fakta bahwa Microsoft Edge mulai mendukung ekstensi Chrome bukan lagi rahasia. Untuk mendapatkan ekstensi, pengguna harus mengizinkan pemasangannya dari direktori Chrome. Selain itu, ada ekstensi dari Microsoft. Sejauh ini, peramban menampilkan ekstensi Chrome sebagai tidak diverifikasi, tetapi ini tidak memengaruhi operasinya. Tetapi pengguna Edge bisa mendapatkan semua ekstensi Chrome yang saat ini tersedia.



Selain fitur yang biasa ada di dalam Chrome 75, Microsoft telah menambahkan beberapa. Untuk menambah / menghapusnya, Anda harus mengatur bendera di tepi: // halaman bendera.



Dengan demikian, pengguna dapat menambahkan Fluent, PlayReady DRM, Edge Reading View.

Tentang niat untuk membangun kembali Microsoft Edge, agar lebih nyaman dan aman bagi pengguna, Microsoft mengumumkan pada Desember tahun lalu. Solusi untuk masalah ini ternyata sedikit tidak terduga - hanya sedikit yang dapat memprediksi transisi korporasi ke produk Google. Solusinya, pada prinsipnya, cukup logis, karena pengguna Windows sekarang dapat berhenti di browser "asli" mereka, dan tidak menggunakan solusi dari perusahaan pihak ketiga.

Perlu dicatat bahwa pangsa Firefox secara bertahap menurun. Bahkan ada ancaman bahwa browser akan menjadi sangat tidak signifikan sehingga pengembang akan berhenti menguji situs untuk itu. Pada Desember 2018, Firefox sudah memiliki kurang dari 9% pasar browser desktop , yang merupakan nilai minimum selama bertahun-tahun.

Mozilla mengatakan bahwa semakin populernya Chromium memungkinkan pengembang, swasta atau perusahaan, untuk memeriksa kemudahan servis suatu layanan atau aplikasi hanya untuk browser ini. Ini mengurangi jumlah waktu yang dibutuhkan untuk pengembangan dan pengujian. Perwakilan Mozilla percaya bahwa sekarang ada situasi yang mirip dengan awal tahun 2000-an, ketika Microsoft benar-benar memonopoli browser.
