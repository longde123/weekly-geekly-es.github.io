PHP internal


  • [RFC] Ubah prioritas operator gabungan - Proposal yang diterima untuk menurunkan prioritas operator operator gabungan . . Perilaku akan diubah dalam PHP 8, dan dalam PHP 7.4 peringatan akan dilemparkan saat menggunakan ekspresi dengan . sebelum + atau - tanpa tanda kurung. 
     echo "sum: " . $a + $b; //   echo ("sum: " . $a) + $b; //   PHP 8 echo "sum :" . ($a + $b);
  • [RFC] Pemisah Literal Numerik - Diusulkan untuk menambahkan kemungkinan menggunakan simbol _ sebagai pemisah dalam angka, untuk meningkatkan keterbacaan: 
     $i = 197_823_459; // 197823459
    Proposal sudah gagal dalam pemungutan suara dua tahun lalu, tetapi sejak itu fitur serupa telah ditambahkan dalam Python, JavaScript dan TypeScript.
  • PHP Internals News Podcast # 10 - Dengan Nikita Popov tentang Prinsip Substitusi Lisk dan Prioritas Operator.
