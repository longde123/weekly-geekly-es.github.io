Optane DC Persistent Memory - Optane dalam format DIMM


Pekan lalu, di konferensi Intel Data Center Tech Summit, perusahaan secara resmi memperkenalkan memori Optane berbasis pada modul 3D XPoint dalam format DIMM, yang disebut Memori Persisten Optane DC (harap jangan bingung dengan Intel Optane Memory , lini konsumen cache drive).

Strip memori memiliki kapasitas 128, 256 atau 512 GB, pinout sesuai dengan standar DIMM, namun, tentu saja, perangkat keras harus mendukung jenis memori ini - dukungan ini akan muncul di generasi berikutnya dari platform server Intel Xeon. Sedangkan untuk dukungan perangkat lunak untuk produk, untuk waktu yang cukup lama telah ada proyek Open Source Intel Persistent Memory Development Kit (PMDK, sampai akhir tahun lalu - NVML).

Sayangnya, tidak ada detail teknis dalam presentasi, seperti konsumsi daya, frekuensi, dll. - kami akan menunggu pembaruan di ARK . Juga tidak jelas apakah DRAM dan Optane dapat dikombinasikan pada saluran pengontrol memori yang sama. Namun, memori yang baru saja muncul akan segera dapat "disentuh" ​​dan diukur sesuatu, namun, sejauh ini hanya jarak jauh. Di musim panas, Memori Persisten Optane DC akan diuji secara online - Anda juga dapat menjadi anggota jika Anda bekerja di perusahaan mitra Intel (omong-omong, tidak pernah terlambat, menjadi satu). Untuk pengujian, server menyediakan farm dengan node 2-prosesor, DRAM 256 GB dan Memori Persisten 1 TB.

Selanjutnya, pada akhir tahun, persediaan memori untuk masing-masing proyek akan dimulai. Yah, penjualan luas direncanakan untuk awal 2019.
