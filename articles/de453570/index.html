Microsoft Edge für MacOS

Im vergangenen Monat haben wir die ersten Vorabversionen der nächsten Version von Microsoft Edge für Windows 10 angekündigt. Jetzt freuen wir uns, Ihnen mitteilen zu können, dass der Microsoft Edge Canary-Kanal auch für macOS verfügbar ist. Jetzt finden Sie auf der Microsoft Edge Insider-Website Builds für Computer mit MacOS und Windows 10, und bald wird die Unterstützung für die Windows-Version erweitert.  



Microsoft Edge für Mac


Der Microsoft Edge-Browser für macOS bietet Benutzern dieselben neuen Funktionen wie in den vorläufigen Versionen für Windows, die Benutzeroberfläche wird jedoch für die Mac-Plattform angepasst. Wir ändern die Benutzeroberfläche und Funktionalität entsprechend den gewohnten und erwarteten MacOS-Benutzern.



Wir sind bestrebt, Microsoft Edge zu einem erstklassigen Browser mit einzigartigen Funktionen und verwandten Diensten zu machen. In dieser ersten Version haben wir einige Änderungen an der Benutzeroberfläche vorgenommen, um sie an das Design von Microsoft anzupassen, aber gleichzeitig, um sie für macOS natürlich zu machen.


Die Änderungen betreffen MacOS-Schriftarten, Menüs, Tastaturkürzel, Überschriften und andere Elemente. In zukünftigen Versionen werden wir das Erscheinungsbild und die Funktionalität des Browsers weiter ändern, experimentieren und auf Benutzer-Feedback reagieren. Teilen Sie uns daher Ihre Eindrücke mit dem Emoticon "Feedback senden" mit.



Darüber hinaus entwickeln wir Möglichkeiten zur Interaktion mit dem Benutzer, exklusiv für macOS, basierend auf speziellen Hardwarefunktionen der Mac-Plattform. So können Benutzer über das Touchpanel (Touch Bar) Aktionen ausführen und auf Kontextbefehle zugreifen. Sie können beispielsweise auf Website-Verknüpfungen zugreifen, zwischen Registerkarten wechseln und Videos verwalten sowie die vertraute Navigation mithilfe von Trackpad-Gesten verwenden.



Einführung in Microsoft Edge Insider-Feeds für macOS


Neue Microsoft Edge-Pre-Builds für macOS sind über Pre-Release-Kanäle verfügbar, die wir als Microsoft Edge Insider-Kanäle bezeichnen. Wir beginnen mit dem Start des kanarischen Build-Kanals, den Sie auf der Microsoft Edge Insider- Website erhalten. Dieser Kanal ist bereits für macOS 10.12 und höher verfügbar. In Kürze werden wir den Dev-Kanal öffnen. Sobald er verfügbar ist, können Sie diese Assembly parallel zu Canary herunterladen und installieren. In unseren Beiträgen im letzten Monat erfahren Sie mehr über unseren Ansatz und was Sie von verschiedenen Kanälen erwarten können.



Konsistente Plattform und Tools für Webentwickler


Aufgrund der Tatsache, dass unsere neuen Builds auf Chromium basieren, erhalten Sie in beiden Versionen von Microsoft Edge das gleiche Rendering - sowohl für Windows als auch für MacOS. Darüber hinaus erhalten Sie denselben Zugriff auf leistungsstarke Entwicklertools.


Zum ersten Mal können Webentwickler Websites und Webanwendungen in Microsoft Edge unter macOS testen und sicherstellen, dass sie in der nächsten Version von Microsoft Edge auf allen Plattformen gleich funktionieren. (Beachten Sie, dass plattformspezifische Funktionen wie das Entschlüsseln von PlayReady-Inhalten in Windows 10 weiterhin separat überwacht werden sollten, um auf diesen Plattformen die bestmögliche Erfahrung mit ihnen zu erzielen.)


Wie die Pre-Builds für Windows unterstützt auch unsere neue Version für macOS Progressive Web Apps, die auf den Standards installiert sind und mit den Entwicklertools im Browser überprüft und debuggt werden können. Wir arbeiten daran, dass solche Anwendungen bequem zusammen mit anderen Plattformanwendungen verwendet werden können. Nach der Installation werden sie wie jede andere Anwendung im Dock, App Switcher und Spotlight angezeigt.  


Ihr Feedback ist uns wichtig


Wir freuen uns, Ihnen unseren ersten Build von Microsoft Edge Canary für macOS vorstellen zu können! Wir benötigen Ihr Feedback, um den Browser zu verbessern, da wir der Meinung sind, dass der Browser für MacOS-Benutzer so bequem wie möglich sein sollte. Wir hoffen, dass Sie die Vorschau noch heute ausprobieren und freuen uns auf Ihr Feedback und Ihre Teilnahme an der Microsoft Edge Insider-Community.


Wenn Sie Probleme haben oder Feedback hinterlassen oder etwas vorschlagen möchten, können Sie eine Nachricht in den Microsoft Edge Insider-Community-Foren hinterlassen, uns auf Twitter kontaktieren oder einfach den Befehl "Feedback senden" im Microsoft Edge-Menü verwenden. um Ihre Gedanken mit uns zu teilen.


Wenn Sie ein Webentwickler sind und auf ein Problem gestoßen sind, das in Chromium reproduziert wird, ist es am besten, einen Fehler in der Chromium- Community zu melden. Verwenden Sie weiterhin den EdgeHTML Issue Tracker , um Probleme in der aktuellen Version von Microsoft Edge zu melden.
