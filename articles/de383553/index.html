GameKid - Clone Game Boy von Raspberry Pi



Viele Liebhaber von Elektronik und altem Konsolenspielzeug haben den Raspberry Pi lange Zeit verwendet, um eine andere Art von Konsolenemulator auf den Markt zu bringen. Mit Emulatoren, insbesondere universellen, können Sie Spiele mit verschiedenen Konsolen spielen, einschließlich sehr alter und seltener. Zu den beliebten Hardware-Plattformen, die emuliert werden, gehört Game Boy. Es waren genau die Handwerker, die beschlossen, es mit dem Raspberry Pi und dem auf einem 3D-Drucker gedruckten Gehäuse nachzubauen.

Die Idee, ein Retro-Gadget aus neuen elektronischen Bauteilen nachzubauen, kam pünktlich zu Weihnachten auf zwei Brüder. Nach mehrwöchiger Arbeit wurde GameKid geboren. Um Spielzeug unter Game Boy auf dieser Konsole zu starten, entschieden sich die Brüder für die RetroPie- Softwareumgebung .

Der GameKid-Fall ist dem Vorläufer ziemlich ähnlich. Es gibt vier vordere Tasten zur Steuerung des Gameplays, des D-Pads sowie der Start- und Auswahltasten. Darüber hinaus gibt es eingebaute Lautsprecher und eine Kopfhörerbuchse. Falls gewünscht, kann der Benutzer einer solchen tragbaren Spielekonsole einen Fernseher über einen HDMI-Anschluss anschließen. Darüber hinaus gibt es vier USB-Anschlüsse, über die Sie Standard-Gamecontroller anschließen können. Bei Anschluss an ein Fernsehgerät oder einen Monitor ist diese Funktion sehr praktisch.

Im Inneren befinden sich neben dem Raspberry Pi 2 mit RetroPie eine 16-GB-Speicherkarte, ein ROM mit Audiospuren und MP3. Die Brüder sammeln jetzt Spenden für dieses Projekt auf der Crowdfunding- Website von Kickstarter.Wenn Sie möchten, können Sie das Gerät für 139 US-Dollar bestellen. Im Prinzip etwas teuer, aber die Fähigkeiten des Kits sind viel weiter fortgeschritten als beim üblichen GameBoy. Ja, hier werden keine Kassetten verwendet. Für das Spiel müssen Sie Hunderte von Spielzeugen in den Konsolenspeicher laden, und Sie können es genießen.
