亚马逊将教授其AI来定义人类情感



如果家庭中的人们能够真正信任AI，那么他们必须学会区分自己的情绪。 了解人们的需求，不仅是通过他们的要求，而且是通过他们的总体心情/福祉。 这样，即使一个人不说话或做某事，一个智能机器人也可以通过间接信号来对情况做出反应。 Affectiva初创公司的创始人Rana El Kalubi如此说道，他致力于解决这一问题。 她的目标是“教计算机理解微笑和假笑之间的区别。”


机器必须理解三个方面，才能真正理解一个人的情绪。 他的表情，声音的语气和动作的敏锐度（或放松度）。 工程师已经很好地处理了第一个因素。 任何人都可以使用相同的Microsoft API ，并教机器人实时了解您是悲伤还是快乐。 但是，对于其他两个“情感指标”的分析，直到最近才有市场决定。
