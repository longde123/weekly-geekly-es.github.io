pemantauan printer snmp di The Dude

Snmp


Ada banyak instruksi di jaringan bagaimana menginstal server pemantauan Bung dari Mikrotik. Sekarang paket server pemantauan dirilis hanya untuk RouterOS. Saya menggunakan versi 4.0 untuk Windows.



Di sini saya ingin mempertimbangkan cara memonitor printer di jaringan: memonitor level toner, jika sudah habis, tampilkan notifikasi. Kami meluncurkan:



Klik hubungkan:



Klik add device (tambah merah) dan masukkan alamat ip printer:



Pada langkah berikutnya, deteksi klik, ia menemukan semua probe yang tersedia, klik selesai:



Klik dua kali pada ikon yang muncul, pengaturan terbuka, pilih "printer", dan klik "ok":



Klik kanan pada ikon dan pilih tampilan:



Di bidang label, kami menulis OID:
[Device.Name] - nama perangkat
[oid ("1.3.6.1.2.1.43.5.1.1.16.1")] - model printer
[oid ("1.3.6.1.2.1.43.11.1.1.6.1.1")] - jenis kartrid
[oid ("1.3.6.1.2.1.43.11.1.1.9.1.1")] - level toner
Di tab gambar, Anda dapat melampirkan ikon Anda sendiri:



Kami keluar, ternyata seperti ini:



Tidak semua printer oid ("1.3.6.1.2.1.43.11.1.1.9.1.1") segera menunjukkan level toner, pada beberapa parameter ini menunjukkan berapa banyak halaman yang tersisa untuk dicetak. Untuk menghitung level toner, Anda perlu membagi berapa banyak halaman yang tersisa untuk mencetak pada total sumber daya kartrid dan kalikan dengan 100. Untuk melakukannya, pilih "Lihat" lagi, lalu Fungsinya:



Klik untuk membuat fungsi baru (merah plus):



Saya memanggil fungsi toner:



Di bidang kode, tulis rumus dan simpan: 

round(100*oid("1.3.6.1.2.1.43.11.1.1.9.1.1")/oid("1.3.6.1.2.1.43.11.1.1.8.1.1"))

Pada label, ganti [oid ("1.3.6.1.2.1.43.11.1.1.9.1.1")] dengan panggilan ke fungsi [toner ()]



Kami pergi. Ternyata seperti ini:
