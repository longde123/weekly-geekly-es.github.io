新型英特尔第9代处理器：核心减去GPU


我们将继续发布有关处理器工程的新闻，并监视第九代处理器的发布。 新部分具有英特尔酷睿不寻常的功能：缺少图形内核，型号名称中的特殊字母F强调了这一点。 但是，“标准” Core在其中占有一席之地，它在当前一代中打开了i3类。

该系列中的顶级机型Core i9-9900KF具有与之前推出的Core i9-9900K完全相同的特性，但缺少图形核心。 实际上，它实际上在那儿，但是被禁用以增加生产过程的利润。 与Core i7-9700KF Core i5-9600KF处理器相同的图片-它们与之前的GPU完全相同。

对于Core i3-9350KF，这是新类别的代表：具有超频功能的4核处理器，不存在超读和图形内核，具有高基本频率（4 GHz）和turbo（4.6 GHz）频率。 Core i5-9400F是一个六核处理器，没有超读和GPU，相反，它的基频较低。 Core i5-9400是其双胞胎兄弟，与传统的Intel9集成显卡Intel HD 630配合使用。

有趣的是，F符号曾经出现在Intel处理器的名称中，但它们是具有集成OmniPath矩阵的Xeon。 现在，它表示没有图形核心的Core的大规模模型-英特尔很长时间没有这种产品了。 显然，其目的是在计算核心正常工作且图形不符合必要的特征时减少技术拒绝。

根据官方信息，新处理器的销售预计将于今年第一季度开始。 有趣的是，缺少图形内核是否会影响超频能力，或者反过来影响能效指标。 也许您自己尝试时会告知它？
