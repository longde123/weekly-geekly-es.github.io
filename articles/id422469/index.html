Menghubungkan FXstream Gateways Grandstream GXW4216, GXW4224, GXW4232, GXW4248 ke 3CX


Pendahuluan


Pembaruan terbaru 3CX v15.5 Pembaruan 6 menambahkan dukungan untuk Grandstream FXS dan FXO. FXS-gateway adalah solusi yang sangat baik dalam situasi jika Anda sudah memiliki armada besar telepon analog dan Anda perlu dengan cepat memperbarui inti komunikasi perusahaan tanpa mengubah perangkat. Ada situasi lain - pertukaran telepon yang beroperasi di sebuah hotel. Kamar-kamar sering sudah memiliki telepon hotel khusus dengan fitur tambahan dan desain yang konsisten. Masuk akal untuk meninggalkan mereka, terutama karena mereka tidak melakukan fungsi telepon perusahaan khusus.

Keuntungan utama dukungan gateway dari 3CX adalah konfigurasi semi-otomatis, di mana cukup untuk membuat pengguna di 3CX, tentukan parameter gateway dan unggah file konfigurasi. Jadi, Anda tidak perlu, misalnya, mengisi kredensial 48 ekstensi di gateway secara manual. Dan jika ada beberapa ratus pengguna seperti itu? ..

Selain telepon analog, Anda dapat menghubungkan mesin faks, telepon analog DECT, dan, misalnya, kunci elektromekanis, ke port Grandstream FXS.

Harap dicatat - jika Anda berencana untuk menghemat uang dengan menawarkan pelanggan gateway analog dan telepon biasa - Anda tidak akan menghemat apa pun. Biaya unit port gateway FXS, biaya telepon analog, serta kabel telepon tambahan secara total lebih mahal daripada jumlah telepon IP murah yang sama. Dan fungsionalitas dari sistem telepon seperti itu akan sangat dibatasi oleh kemampuan dasar perangkat analog. Pada saat yang sama, kegunaan dan produktivitas pengguna akan berkurang.

Pengaturan Gateway dalam 3CX


Semua gateway Grandstream FXS terhubung ke 3CX dengan cara yang sama. Sebelum menghubungkan gateway, Anda harus sudah membuat nomor ekstensi (pengguna), yang harus ditugaskan ke port gateway. Jika Anda berencana untuk menghubungkan mesin faks ke gateway, buat nomor faks ekstensi di bagian antarmuka 3CX Faxes.



Ketika nomor ekstensi pengguna dan faks siap, di bagian FXS / DECT, tambahkan Grandstream gateway dari model yang diinginkan.



Dalam contoh ini, gateway GXW4224 24-port ditambahkan. Tunjukkan pabrikan, model, dan alamat MAC perangkat.

Setelah menambahkan gateway, antarmuka pengaturan akan terbuka. Pada tab Basic, pilih mode faks - FAX di T.38 atau Fax in Audio G711 (Pass Through) . Umumnya disarankan agar Anda menggunakan protokol T.38.



Pada tab Ekstensi, tetapkan ekstensi 3CX mana yang harus dikonfigurasikan pada port gateway.



Simpan pengaturan gateway, buka tab General lagi dan simpan file konfigurasi gateway XML ke disk.



Ini melengkapi konfigurasi gateway di sisi 3CX.

Pengaturan Grandstream FXS Gateway


Pertama-tama, setel ulang gateway ke pengaturan default. Untuk melakukan ini, tahan tombolReset pada gateway selama 5 detik. Setelah reset, gateway menerima alamat IP melalui DHCP. Alamat saat ini harus terlihat pada layar LCD perangkat. Jika tidak terlihat, tekan tombol panah ke bawah.



Kemudian pergi ke antarmuka gateway (kata sandi default adalah admin) di bagian Pemeliharaan - Peningkatan dan Penyediaan - Unggah Perangkat Firmware dan instal firmware terbaru dari situs web Grandstream .

Kembali ke bagian yang sama, bagian Konfigurasi Konfigurasi Perangkat, dan unduh file konfigurasi 3CX yang Anda buat sebelumnya.



Gateway akan menerapkan konfigurasi dan reboot. Ketersediaan gateway ditentukan oleh penampilan alamat IP pada layar LCD. Ini menyelesaikan pengaturan.

Di bagian Pengguna di antarmuka manajemen 3CX, Anda akan melihat bahwa semua ekstensi yang dikonfigurasi di gateway telah berhasil terdaftar. Jika gateway digunakan untuk menghubungkan mesin faks, nomor ekstensi faks akan terlihat di bagian Home - System Numbers.
