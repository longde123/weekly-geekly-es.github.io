O IntelliSense para C ++ no Visual Studio agora funciona com base em exemplos no seu código

Desde que anunciamos o IntelliSense para modelos , você recebeu ótimas ofertas. Uma sugestão muito popular foi a barra de modelos preencher automaticamente uma variante com base em exemplos no código. No Visual Studio 2019 versão 16.1 Visualização 2, adicionamos esse recurso à opção "Adicionar todas as instâncias existentes" no menu suspenso Barra de modelos. Os exemplos a seguir são retirados da base de código do SuperTux .



O menu suspenso da barra de modelos agora contém o item - "Adicionar todas as instâncias existentes".



Clicar na opção "Adicionar todas as instâncias existentes" agora exibe uma lista suspensa, para que você não precise mais inserir texto manualmente.



Além disso, como Localizar todas as referências , a janela na parte inferior do editor mostra onde cada instância foi encontrada e quais são seus argumentos.
