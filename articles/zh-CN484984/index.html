Unity游戏开发人员。 OTUS的新课程

注意！ 本文不是工程知识，仅供有兴趣在Unity上进行游戏开发领域的教育的读者使用。 最有可能的是，如果您对学习不感兴趣，那么您将不会对本材料感兴趣。



您还记得我们的母亲在童年时曾责骂我们玩计算机游戏直到早晨吗？ 我们中的许多人不仅玩游戏，还梦想着开发世界上最酷的游戏并从中赚钱。 现在，童年的梦想实现了-Otus启动了Unity游戏开发培训课程 。

该课程教授游戏开发的各个阶段，从概念本身到立即实施。 在这里，您将学习在线免费游戏的游戏设计和本地化，货币化，以及如何吸引和留住玩家。

作为课程的一部分，您最多可以创建3个游戏。 让我们分别考虑每个：

1. 一步一步的3D管家，例如RAID Shadow Legends（移动游戏） 。

  • 创建简单的动画
  • Unity UI
  • 粒子系统
  • 声音再现等

2. 2D平台游戏亚军。 （手机游戏）

  • 静音地图
  • 2D物理
  • 屏幕操纵杆和操纵杆
  • 使用Cinemachine等的摄像机控制。

3. 具有网络游戏和“ Battle Royale”（适用于PC）的机制的3D竞技场勇士。

  • 创建一个多人游戏
  • 掌握3D物理和地形
  • 导航网格
  • 光子统一网络等。

要了解什么是Unity游戏开发 ，可以观看由课程老师Nikolai Zapolnov主持的开放式网络研讨会“一小时之内的僵尸射击游戏”：


该网络研讨会讨论了实际的案例，并且老师回答了学生的问题并演示了Otus的培训形式。

Nikolai Zapolnov是一位在游戏开发领域拥有丰富经验和深厚知识的老师。 他为《割绳子2》，《最后的营地》，《凤凰游骑兵：解谜RPG》，《粉碎联盟》做出了贡献，并且是Rovio（《愤怒的小鸟》的创作者）的高级游戏开发人员。

1月27日20:00，尼古拉（Nikolay）将以“ Unity Game Developer”为主题举行开放日，在此期间，他将详细介绍课程计划，在线培训格式的功能以及课程参与者在此期间可以发展的技能和能力学习。

在Unity Games开发人员课程中学习需要什么知识？

足够了：

  • 编程基础
  • OOP原则
  • 计算机代数和线性代数的基础知识

如果您具有以下知识，那也将是很棒的：

  • C＃经验
  • Unity的经验

要了解您的技能是否足以进行培训，可以通过入学考试。

对于那些对此课程感兴趣的人，我们记得可以在1月27日举行的开放日获取更详细的信息。 待会见！
