谷神星在变化：黎明的探测器检测到小行星表面的变化


谷神星。 资料来源：ESO / L.Calcada / NASA / JPL-Caltech / UCLA / MPS / DLR / IDA / Steve Albers / N.Risinger

谷神星的观测是太阳系中最有趣的物体之一，它使我们能够记录小行星表面的变化。 事实证明，谷神星是太阳系中这一类继续进化和变化的唯一对象。 黎明探测器使研究物体表面成为可能，并向太阳系知识库中添加了新数据。

观察是通过分光光度计和目视进行的-通过拍摄物体。 值得回顾的是，相对较早的时候，Dawn 能够在谷神星上发现大量的水冰。 特别是在某些陨石坑中尤为丰富。 其中之一的直径为20公里。 2016年4月至2016年10月的新观测结果显示，曲轴箱壁上的冰量有所增加。

罗马航空物理与行星研究所的科学家安德里亚·拉波尼（Andrea Raponi）说：“这是对谷神星表面变化的首次检测。”

Raponi领导了这项研究，这项研究允许发现矮行星表面冰的数量变化。 “诸如谷神星在其轨道中运动时接近太阳以及季节性变化等因素的综合作用导致水蒸气的释放，然后水蒸气沉积在火山口的冷壁上。 这导致小行星上的冰量增加。 科学家说，变暖还会导致山体滑坡。

由于收集了有关表面化学成分，地质结构和新的地球物理数据的信息，因此，Dawn任务被证明是所有研究太阳系及其天体的项目中最成功的项目之一。 在第一项研究中，科学家发现谷神星的冰壳厚度约为40公里。 它的成分是水（冰），盐和有机化合物。

在第二项研究中，科学家获得了有关皮质动态变化的新信息。 据推测，发现了最近在表面上发现的材料。 黎明以前发现过在海水中形成的矮行星表面常见的碳酸盐。 在Occator，Oxo火山口和Ahuna Mons地区发现了大量碳酸盐。


探测器发送了小行星最高的山峰Ahuna Mons的照片。 它的高度是4公里。 基地的宽度是17公里。 山上有大量的苏打水，在彩色插图上以绿色和红色标记。 资料来源：NASA

贾科莫·卡罗佐（Giacomo Carrozo）领导的罗马航空与行星研究所的科学家进行了一项研究，结果发现12个碳酸盐岩带丰富。 此外，还发现了其他化合物。 值得注意的是，水冰在小行星表面上不稳定。 在这里，冰只有藏在阴影中才存在很长时间。 否则，冰会迅速升华并消失。

另外，在外部因素的影响下，水合碳酸盐脱水了数百万年。 在谷神星上发现了水合碳酸盐这一事实表明，地壳相对年轻，而且其形成过程仍在继续。 由于各种现象，包括冷冻齿烷，地壳的成分不均一。 最初，地壳是均匀的，因为它是由液态海洋的水形成的，但是在数百万年之后，其成分发生了变化，变得不均匀。
