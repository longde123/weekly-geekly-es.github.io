Penelitian: Linux masih OS paling populer di cloud

Kami membahas statistik penyedia IaaS asing, memberikan angka untuk cloud kami dan berbicara tentang alasan yang memengaruhi distribusi OS open source.


Foto - Ian Parker - Unsplash

Distribusi saham


Menurut IDC, pada tahun 2017, 68% server perusahaan in-house dan berbasis cloud menjalankan Linux. Sejak itu, angka ini telah meningkat - tren ini dilacak oleh banyak penyedia IaaS.

Pada 2015, pejabat Microsoft mengatakan bahwa setiap instance keempat di Azure cloud berjalan di Linux. Dua tahun kemudian, jumlahnya 40%. Tahun ini, jumlah mesin Linux telah melampaui 50% . Perusahaan IT itu sendiri juga telah menjadi pengguna aktif sistem operasi terbuka. Misalnya, jaringan yang ditentukan perangkat lunak (SDN) dari suatu organisasi dibangun atas dasar itu.

Gambaran serupa terlihat di awan penyedia IaaS lainnya. Misalnya, di cloud 1cloud.ru, Linux menjalankan 44% mesin virtual. Dalam kasus Windows, angka ini adalah 45%.


Berbagi sistem operasi pada server aktif di cloud 1cloud

“Kami berharap bahwa dalam waktu dekat, Linux dapat menjadi pemimpin dan mengatasi sistem operasi lain,” komentar Sergey Belkin, kepala departemen pengembangan proyek 1cloud.ru . "Mengingat bahwa hanya beberapa tahun yang lalu, lebih dari setengah dari mesin virtual yang digunakan di cloud kami berjalan di Windows."

Perkiraan ini dikonfirmasi oleh statistik penyedia IaaS lainnya. Misalnya, di cloud pribadi salah satu pemasok barat terbesar di Linux, lebih dari 90% contoh berfungsi .

Pada saat yang sama, sistem operasi open source tetap menjadi platform paling populer untuk hosting web. Menurut agen analitik W3Techs, 70% dari sepuluh juta situs paling populer digunakan di server Linux (menurut peringkat Alexa ). 30% sisanya dimiliki oleh Windows.

Mengapa linux


Para ahli mengidentifikasi setidaknya dua faktor yang mempengaruhi distribusi sistem operasi di cloud.

Fleksibilitas arsitektur. Faktor ini dianggap sebagai salah satu faktor penentu dalam Linux Foundation. Linux cocok untuk melakukan beragam tugas dan bekerja pada platform dengan berbagai ukuran: dari perangkat seluler hingga superkomputer. Misalnya, pada 2017, 498 superkomputer top-500 menjalankan sistem operasi open-source ini. Tetapi pada akhir tahun itu, Linux mulai bekerja 100% dari komputer di atas.

Superkomputer paling kuat hingga saat ini - IBM's Summit - dijalankan oleh Linux . Superkomputer exaflops pertama di Amerika Serikat, yang mereka rencanakan akan selesai pada tahun 2021, juga akan bekerja berdasarkan OS open source ini .

Komunitas luas. Basis kode Linux diperbarui kira - kira setiap sepuluh minggu . Sejak 2005, lebih dari 15 ribu insinyur telah berkontribusi pada pengembangan inti. Di antara mereka adalah karyawan dari 200 perusahaan besar. Hanya pada tahun 2017, 3% dari perubahan dalam basis kode dilakukan oleh pengembang dari Google dan Samsung. Lebih dari 13% perubahan adalah "bertanggung jawab" pada Intel.


Foto - Ian Parker - Unsplash

Perusahaan IT besar secara aktif terlibat dalam pengembangan Linux itu sendiri dan produk-produk open-source berdasarkan itu. Microsoft menawarkan platform Azure Sphere untuk aplikasi IoT, yang didasarkan pada kernel Linux. Intel meluncurkan proyek cloud Clear Linux , di mana para insinyur mengoptimalkan OS open source untuk berjalan pada prosesor mereka. HPE menawarkan ClearOS untuk dikirimkan bersama perangkat kerasnya. IBM mengakuisisi RedHat dan sekarang sedang mengembangkan salah satu distribusi paling populer di pasar.

Produk open source baru secara aktif digunakan di cloud, yang memiliki efek positif pada distribusi Linux.

Apa selanjutnya


Angka-angka pasti tentang popularitas sistem operasi tertentu di cloud harus diperlakukan dengan sejumlah skeptisisme. Infrastruktur TI modern penyedia cloud bersifat komprehensif. Banyak hypervisor dapat disebut "bersarang", dan situasi muncul ketika satu sistem operasi dikelilingi oleh yang lain.

Tetapi bahkan dengan fakta ini dalam pikiran, aman untuk mengatakan bahwa Linux semakin banyak digunakan di cloud.
