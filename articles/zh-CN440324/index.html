Linux Foundation的新基础-将开发边缘计算

在我们的博客中，我们已经讨论了两个“新” Linux Foundation基础-GraphQL和Ceph。 上个月，该财团建立了另一个项目-Edge，其任务是开发云和外围计算。 我们告诉谁已经成为会员。


/图片NOAA图片库 CC BY

为什么我们需要新资金


如今，边缘计算市场高度分散-所有开放式专有解决方案都没有集成，并且彼此隔离地开发。 同时，作为外围计算基础的物联网小工具的普及也在持续增长。 据分析师称，到2020年，智能设备的数量将超过 200亿。 Linux基金会认为，为了使所有这些小工具都能正常运行，人类将需要一个“整个生态系统”。 它将在新基金中创建。

Edge项目将专注于创建开源框架，以开发与专有硬件或操作系统无关的外围计算。 该基金中的公司还将处理边缘计算术语的标准化。

五十个IT巨头已经支持 Linux Foundation Edge，包括Red Hat，IBM，Intel，Arm，Qualcomm，DellEMC，HPE和百度。

已经包括哪些项目


到目前为止，新基金管理着五个项目-Akraino Edge Stack，EdgeX Foundry，Open Glossary，Home Edge和EVE。 接下来，我们将简要讨论它们中的每一个。

Akraino边缘堆栈 。 实现外围计算概念的云平台。 它为用户提供了可以实时处理数据的边缘基础结构，最大延迟为20毫秒。 同时，Akraino可以与第三方云平台一起使用，以创建边缘应用程序和容器的生态系统。

该平台由OpenStack管理。 开发人员使用Ceph存储来存储数据，IT基础架构的分布式节点由带有Open vSwitch的KVM虚拟机管理程序进行管理。 其中一些工具基于AT＆T开发的解决方案。

EdgeX铸造厂 。 一个由微服务组成的独立框架，用于与物联网设备一起使用。 EdgeX基于Dell Edge网关上使用的免费Fuse IoT堆栈。 EdgeX项目拥有超过12.5万行代码，并包括用于数据分析，处理和保护的一系列现成的微服务。 该平台为开发人员提供了在一个平台上“遍历”边缘应用程序整个生命周期（从设计到部署）的机会。

您可以在官方文档中找到该解决方案创建者的快速入门指南。

打开词汇表 。 帮助IT专业人员“说相同语言”的计划。 在边缘计算行业中，没有确定的术语和基本概念的单一翻译。 该项目的创建者打算解决此问题。 它们构成一个字典，其中字母词收集具有定义的基本词，并链接到相关概念和上下文中的使用示例。 所有信息均以英文提供。

可以在GitHub上的项目存储库中找到Open Glossary文档。

家边缘 三星电子公司的一个项目，它是一个用于控制智能家居中外围设备的平台。 它将包括一组可与不同的库和第三方生产环境一起使用的API。

EVE或Edge虚拟化引擎 。 具有用于外围计算和IoT设备的API的应用容器基础架构。 源代码是由美国初创公司Zededa编写的。 EVE Edge容器是跨平台的， 与常规Docker容器不同 。 Zededa的代表将他们的开发与Android进行了比较，后者将来自不同制造商的移动设备“结合”在一起。 EVE将在今年3月开始根据Apache开源软件许可进行分发。

其他Linux基础基金会


除了Edge，Linux基金会还开发了更多的云开发基金。 其中包括Cloud Foundry和Cloud Native Computing Foundation 。

第一种开发基于不同云基础架构，框架和服务的单一平台，用于开发，测试和扩展应用程序。 它的活动包括Cisco，Google，IBM，Microsoft等。 而且，有 60％的大公司已经在使用 Cloud Foundry解决方案-他们的代表指出，该平台在创建一个应用程序方面平均节省了十个星期和十万美元。


/图片Jasmine Nears CC BY-SA

Cloud Native Computing Foundation是Kubernetes编排工具，Prometheus监视系统，Envoy代理服务和CoreDNS DNS服务器“ 左 ”的基础。 在CNCF“孵化器”中，还有15个专为各种任务量身定制的项目：从保护生产环境到在云中存储文件和消息传递。

该基金得到了大多数与云连接的IT公司的支持：从VMware和Oracle到ZTE和SUSE。 去年，有1.8万人参加了CNCF项目的开发，参与公司的数量增加了130％。

Linux基金会总共为数千个组织提供支持，并且其数量还在不断增长。 根据去年的报告，在2018年，平均每天有一家公司加入该财团。 在2019年1月末，又有22位新成员加入了该基金-来自云技术，半导体制造，安全和能源领域。 因此，我们可以预期，Linux基金会将在不久的将来开设更多的新基金，其工作将致力于开发IT生态系统。
