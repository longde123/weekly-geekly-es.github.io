  • PHP 7.4.0 Alpha 3 - Die neueste Alpha-Version des geplanten . Das Einfrieren der Funktionen sollte am 22. Juli erfolgen, daher kann die aktuelle Liste der Änderungen nur durch diejenigen ergänzt werden, für die die Abstimmung spätestens am 8. Juli begonnen hat. Insgesamt jetzt in der Abstimmung:
    • Abschreibungen für PHP 7.4
    • PDO entkommen "?" Parameter Platzhalter
    • Veraltete Syntax für geschweifte Klammern für den Zugriff auf Array-Elemente und String-Offsets
    • Fügen Sie str_starts_with (), str_ends_with () und verwandte Funktionen hinzu .

    Auch ein paar neue Änderungen ohne separate RFCs:
    • proc_open akzeptiert Argumente in Form eines Arrays, sie müssen nicht proc_open werden und der Prozess wird direkt ohne Zwischenshell gestartet: 
     proc_open(['php', '-r', 'echo "Hello World\n";'], $descriptors, $pipes);

    • In PHP 7.4 wird ein Hinweis ausgegeben, wenn versucht wird, Skalartypen falsch als Arrays zu verwenden. Bisher gaben solche Aufrufe einfach null : 
     $null = null; var_dump($null[0]); // NULL $int = 42; var_dump($int['foo']); // NULL var_dump(true[0] === false[0]); // true
