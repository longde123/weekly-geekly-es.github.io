以NextDent为例的牙科3D打印



在加速的技术进步的刺激下，世界正在以前所未有的速度变化，这些变化中最具活力的领域之一是3D打印，而最适合牙科的医学应用之一就是牙科。在牙科领域，不仅3D打印的速度和准确性至关重要，而且其材料的特殊性能（例如安全性，生物相容性和严格的医疗参数）也至关重要。在这篇评论中，我们将讨论最新的3D打印牙科材料的特性。

例如，让我们举一个荷兰公司NextDent BV的产品为例，该公司于2012年从Vertex-Dental BV拆分出来，当时已成功从事牙科材料超过76年。荷兰和德国的医疗机构都参与了NextDent材料的开发，所有产品均经过最严格的控制和认证。几乎找不到合适的例子。

1. NextDent基地


印刷临时义齿基托。



NextDent Base是IIa类生物相容性材料，适用于印刷各种义齿基材。与义齿的标准材料相比，这种材料的收缩率低，从而可以很好地固定假牙。在后续加工后残留的少量残留单体使该材料具有更高的生物相容性。



NextDent 3D牙科材料可在1000毫升容器中使用。

规格：

  • 颜色：深浅的粉红色
  • Brookfield粘度在23°C：1.0-1.5 Pa·s
  • 弯曲强度：≥85 MPa
  • 弯曲模量：≥2,100 MPa
  • 吸水率：≤32μg/ mm3
  • 与水的溶解度：≤1.6μg/ mm 3
  • 残留单体量：≤1％
  • 邵氏硬度：D 80-90

2. NextDent SG（手术指南）


设计和打印高精度透明手术模板。



NextDent SG是经过I类生物相容性认证的材料，专门用于印刷手术模板（更精确的钻孔和更快的手术指南）。材料的高精度-保持精确尺寸和透明度的恒定性-使您可以在牙医工作期间准确定位钻头和其他手术器械。

NextDent SG耐消毒剂。此外，可以使用伽马射线和高压灭菌器对该材料进行灭菌。使用高压灭菌器不会影响尺寸稳定性，因此NextDent SG可用于任何牙科诊所。



规格：

  • 颜色：半透明橙色
  • Brookfield粘度在23°C：1.1-1.6 Pa·s
  • 弯曲强度：≥80 MPa
  • 弯曲模量：≥2,000 MPa
  • 在134°C下灭菌：最高-5分钟。
  • 邵氏硬度：D 80-90



3. NextDent C＆B


用于桥和冠的生物相容性印刷材料。



NextDent C＆B是IIa类生物相容性材料，用于印刷牙冠和牙桥（1-3齿）。该材料的性能以及其耐磨性使其成为最适合的NextDent C＆B。NextDent C＆B产品可以用牙科水泥固定。



规格：

  • 颜色：不同
  • Brookfield粘度在23°C：0.9-1.4 Pa·s
  • 弯曲强度：≥85 MPa
  • 弯曲模量：≥2,100 MPa
  • 吸水率：≤30 mcg / mm3
  • 与水的溶解度：≤5μg/ mm3
  • 邵氏硬度：D 80-90

4. NextDent C＆B MFH（微填充混合材料）


颜色和半透明性变化很大。



NextDent C＆B MFH是IIa类生物相容性材料，设计用于印刷中期胎冠和牙桥。无机填料和树脂的平衡组合使材料具有较高的强度和耐磨性。

它很容易加工和抛光材料，可以使用各种类型的复合油漆套装对产品进行油漆。由于不透明度和半透明性之间达到了完美的平衡，因此打印出的牙冠与活牙在视觉上无法区分。



规格：

  • 颜色：不同
  • Brookfield粘度在23°C：0.8-1.3 Pa·s
  • 弯曲强度：≥100 MPa
  • 弯曲模量：≥2,400 MPa
  • 吸水率：≤30 mcg / mm3
  • 与水的溶解度：≤5μg/ mm3
  • 负载强度系数：≥1.5 MPa 1/2
  • 强度：≥7,000 J / m2

5. NextDent正交清除


美观的3D打印材料。



NextDent Ortho Clear是另一种生物相容性IIa类材料，适用于所有类型的牙齿矫形器和牙线。当患者每天使用时，它是透明的并且几乎是不可见的。这种3D打印材料的特点是高弯曲强度和冲击强度以及耐磨性。适用于波长为365-385 nm的打印机。



规格：

  • 颜色：透明
  • Brookfield粘度在23°C：0.8-1.3 Pa·s
  • 弯曲强度：无限
  • 弯曲模量：≥1300 MPa
  • 吸水率：≤65 mcg / mm3
  • 与水的溶解度：≤5μg/ mm3
  • 负载强度系数：≥1.1 MPa 1/2
  • 强度：≥6,000 J / m2
  • 邵氏硬度：D 80-90

6. NextDent Ortho IBT


用于正畸的柔性材料。



NextDent Ortho IBT是经正畸使用的一流生物相容性认证材料-创建定位托盘。使用适当的软件，牙医可以为各种矫正措施计划矫正器的确切位置和形状。

由于印刷产品的特性具有灵活性，牙齿矫正医生可以轻松地一次放置所有托架，从而节省了他和病人的时间。



规格：

  • 颜色：透明
  • Brookfield粘度在23°C：1.1-1.6 Pa·s
  • 拉伸弹性：12-18％
  • 肖氏硬度：80-90

简单设计和3D打印矫形器。



NextDent Ortho Rigid是IIa类生物相容性材料，专为数字化护齿器和矫形器而设计。结合相应的软件，它可以轻松设计和打印这些产品，而无需进行其他后期打印过程。

  • 弯曲强度：≥75 MPa
  • 弯曲模量：≥1800 MPa
  • 负载强度系数：≥0.9 MPa / 2
  • 吸水率：≤32μg/ mm3
  • 与水的溶解度：≤5μg/ mm3
  • 强度：≥3,000 J / m2
  • 邵氏硬度：D 80-90

8. NextDent模型


高精度牙科模型。



NextDent模型用于在需要高精度的情况下打印详细的主模型。由于颜色和不透明性，模型可以准确传达所扫描颚的元素的位置和形状的所有细微差别，并为牙医提供理想的表面，以使您准确了解治疗对象。精确的3D打印模型是更有效地创建牙齿的理想基础。

10. NextDent托盘


适用于单个印模托盘的高质量3D打印材料。



NextDent托盘是一流的生物相容性认证材料，可用于单个义齿勺的3D打印。该材料允许高速进行大容量数字印刷。

NextDent托盘可以在几分钟内创建最复杂的表格。印刷的表格具有足够的刚性，可以与任何类型的压印材料进行进一步的加工，从而可以生产出高质量和高精度的产品。

NextDent牙龈面膜是一种柔性材料，可与模型材料结合使用，例如，用于牙龈的部分或大规模修复。在需要更大灵活性的情况下，它具有特定优势。

用于注塑模具3D打印的可燃材料。



NextDent Cast是一种完全可燃烧的金属铸造材料。NextDent Cast适用于从金属合金生产各种类型的牙科产品。只需设计，打印并将其与其他推荐的材料结合使用，即可创建模具和金属铸件。在铸造过程中燃烧无残留，NextDent Cast使您能够制造出最精确的金属正畸产品和假体。
