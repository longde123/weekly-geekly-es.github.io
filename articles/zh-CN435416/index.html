DIY项目技术。 入门部分


本文是由一位业余无线电爱好者撰写的，他在管理复杂电子设备的开发和生产项目方面经验丰富。 本文的目的：帮助发烧友组织其项目活动，并确定实施业余项目（DIY）和商业项目的方法上的差异。


业余无线电运动已经存在了一个多世纪 。 业余无线电是业余爱好 ，因为 火腿在业余时间进行工作。 业余无线电爱好者的创造力具有设计活动的所有特点：


  • 旨在创造独特的产品；
  • 家庭预算有限；
  • 对家人的时间有限制。

从爱好者的角度来看，一种独特的DIY产品既可以是无线电设计师组装的圣诞树花环（是的！），也可以是设计文档包，其数量足以组织该项目开发的原始电子设备的小规模生产。


除了创建独特的产品，业余项目的可接受结果可能是：


  • 技术创造力带来的喜悦感；
  • 对完成的项目感到自豪；
  • 加强家庭关系。

但是，业余项目的结果不应该是：


  • 浪费金钱和时间；
  • 家庭关系恶化；
  • 在项目活动过程中由于爆炸，火灾或电击对健康和财产造成损害。

简单规则


要获得业余项目的成功并最小化道德和物质上的损失，只需遵循简单的规则即可：


  1. 在创建独特项目产品的工作原型之前，请勿吸引任何外部融资。
  2. 在决定打开该项目之前，请确保成功项目的技术可行性。
  3. 请理解，在早期阶段，既无法计划业余项目的时间安排，也无法计划预算。
  4. 请记住，在您的业余项目初期，除您之外的任何人都不会感兴趣。
  5. 不要忘记，在业余项目的早期阶段，只有发烧友的家人才能从她那里得到时间和金钱。

这些简单规则的基础是业余项目和商业项目之间的根本区别：从自有资金筹集资金方面的创造自由。


在业余项目中，表演者有权独立确定项目的目的，时间和预算。 承包商可以随时启动，终止或暂停项目活动。 所有项目结果均属于承包商。


吸引外部资金需要承包商对项目结果负责。 投资者有权影响项目的目的，时间和预算。 只有得到投资者的许可，才能停止或暂停项目。 投资者完全或部分拥有项目成果。


简单的例子


业余无线电广播员在DIP-28封装中发布了基于ATmega328P-PU微控制器的设计。 这种解决方案的开发方案很简单：


  • 确定项目计划；
  • 它采用了著名的价格合理的调试板，带有“板上” ATmega328P-PU，可与带有库的简单自由开发环境兼容；
  • 从在该平台的开源网络上发布的一组项目中，选择适合计划实施的任何项目；
  • 通过将调试板“绑定”到必要的外围设备来开发解决方案的原型；
  • 要么适应所选开源项目的产品要求，要么开发原始软件；
  • 在调试了面包板上的软件和电路之后，将开发出符合开发人员技术能力水平的产品印刷电路板。

应该注意的是，这个“简单方案”与我们的“简单规则” 100％一致：


  1. 尚未吸引外部融资。
  2. 其他开发人员发布的类似项目的存在证明了实施的技术可行性。
  3. 职权范围，项目进度表及其预算尚未制定。
  4. 该项目开发的解决方案已发布。
  5. 甚至在订购印刷电路板和组件进行组装之前，都使用开源软件以及原型解决方案节省了家庭的时间和金钱。
  6. 您的孩子也可以参与到该“简单计划”中来，这显然有助于改善家庭关系。

为什么会这样呢？


作为业余项目的一部分，如果没有开发团队且无法吸引资金，几乎不可能说出新的技术词汇。 所有这些想法在理论上已经在科学文章中得到证实，或者已经由其他爱好者实施，然后以开源形式发布，或者在工业设计中得到体现。 这很可悲，但是尽管如此，它简化了实施业余项目构想的技术可行性的证明，并且消除了发挥“可怕的商业秘密”的必要性，除非您拥有某些专利，否则就不会产生这种“商业秘密”。 但这更多的是初创公司，而不是DIY项目。


通常，业余项目的目标是在已经发布了所有方案，图纸和源代码的某个地方重复或改进现有的开发或设计。 此开发或设计也可以作为自组装套件出售，也可以在工厂组装。 业余项目的另一个目标可能是为任何不寻常的用途重新制作现有的开发或设计。 例如，一名无线电爱好者正在开发一种基于用于组装无线电台的套件的天线分析仪。


确定项目计划实施的技术可行性之后，将考虑该计划的技术实施方法。 在此阶段，将从电路，设计，现成的开源软件的可用性等角度对现有解决方案进行详细分析。 在这个地方的业余项目中，建议不要制定技术任务，而只是制定项目MVP的要求。 然后，业余项目进入初步设计阶段，以创建工作布局。 在商业项目中，此时，已经有更新的技术任务和技术要求列表。 然后，绕过初步设计阶段，商业项目进入工作项目阶段，以创建原型为目标。
在此处了解有关项目阶段的更多信息。 也有对GOST的引用。


方法上的差异是由于这样一个事实，即商业项目中的承包商通常已经在该项目的主题上有基础，这可以通过跳过初步设计阶段来节省投资者的时间和金钱。 由于家庭预算有限，业余项目的执行者无力下令订购十二个原型的制造商并在调试过程中重新焊接一半的组件，从而合理地削减了电路板。


在业余项目中，为了节省金钱和时间，您必须使用建模和原型测试您的每个决定。 调试板，数据表和应用笔记在此过程中提供了帮助。 如果必要的调试板价格过高，则必须根据开放源代码提供的信息来独立开发。 有时建议以较低的价格订购“原型”印刷电路板的制造，然后在其上组装所有这些自制的调试板。 必须特别注意物流：最好从更可靠的供应商那里购买更昂贵的组件。


在一个商业项目中，对于无线电爱好者在原型制作上所花费的时间，他们通常设法调试原型并准备进行初步测试。 如果进行了大量更改，并且原型很可能无法通过机械强度或“气候”测试，则订购下一批原型，并且已经进行了修改。 经过初步测试后，对相同的原型进行验收测试。 然后，将工作文档调整为设计文档的状态，然后将产品投入生产。 赢利！


此时，最有可能在业余项目中，原型制作过程仍在进行中。 如果此过程以创建工作布局结束，则开发人员可以继续该项目。 然后，他决定创建原型（原型），编写技术任务，制定业务计划，吸引融资。
但是在一个业余项目中，开发人员已经收到初步结果，可能无法继续该项目。 这是方法上的另一个根本区别：在工作布局上实施项目设计后，您可以举行家庭议会并演示结果。 然后可以关闭该项目，并发布该项目的结果。 将有追随者-太好了！ 找不到-不是那么重要。


DIY项目成功的主要条件是让亲人了解所花的时间和金钱并支持您的工作！
