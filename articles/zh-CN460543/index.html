面向思科开发人员的新认证。 行业认证概述

思科认证计划已有26年的历史（它成立于1993年）。 许多人都非常了解CCNA，CCNP和CCIE认证的工程线。 今年，该计划还为开发人员补充了认证，包括DevNet助理，DevNet专家，DevNet专业人员，DevNet专家。


DevNet程序本身已经在公司中使用了五年以上。 本文中已经在集线器上编写了有关Cisco DevNet程序的详细信息。


因此，我们对新认证的了解：


  1. 至于工程认证，DevNet认证有四个级别-助理，专家，专业，专家。
  2. 在模块的自动化/编程方面对工程认证进行了补充。
  3. 面向开发人员的认证中有一个与网络可编程性基础相关的模块


让我们看一下每个认证，包括内容和针对的对象。


思科DevNet助理


谁专注于：
对于年轻的专业人​​员，即从程序员和SRE / DevOps到测试人员和自动化工程师的初级专业人员。


DEVASC 200-901考试将包括应用程序开发的基础知识（git的知识，python的基础知识）以及使用Cisco硬件/解决方案API的知识和技能。
如前所述，认证还包括一个基于编程网络基础的模块（占总数的15％）。



思科DevNet专家


谁专注于：
在其中一个领域拥有3至5年经验的专家。
具有在开发和支持基于Cisco平台构建的应用程序方面的实践经验的开发人员。


在此认证中，您可以选择以下一个或多个专业，为每个专业提供适当的考试。
对于程序员：


  • 核心（DEVCOR 300-901）
  • DevOps （DEVOPS 300-910）
  • 物联网（DEVIOT 300-915）
  • Webex （DEVWBX 300-920）

对于自动化专家：


  • 协作自动化（CLAUTO 300-835）
  • 数据中心自动化（DCAUTO 300-635）
  • 企业自动化（ENAUTO 300-435）
  • 安全自动化（SAUTO 300-735）
  • 服务提供商自动化（SPAUTO 300-535）

根据Core专长，DevOps将具有用于测试有关CI / CD，Docker，12要素应用程序原理，OWASP威胁的知识的模块。


Webex专门研究Cisco Webex设备和解决方案。 以前，许多统一通信解决方案以通用的Webex品牌转移，而Cisco Spark在Webex Teams中更名。 指导包括用于自动化Webex团队，定制，用于协作的设备（Webex设备）编程的模块。


物联网专业化包括用于开源物联网解决方案，可视化和解释（包括使用Freeboard，Grafana和Kibana）的模块。


DevNet专家认证考试：DevOps还包括以下主题：构建/部署工具（例如Jenkins，Drone或Travis CI）的功能和概念； 用于自动化基础设施服务（例如Ansible，Puppet，Terraform和Chef）的配置管理工具； Kubernetes（概念，集群中的应用程序部署，对象的使用）； 定义扩展应用程序或服务所需的要求（内存，I / O磁盘，网络，CPU）； 开发和测试期间保护应用程序和基础结构的方法。


下表是DevOps行业中存在的一些认证的比较表。 在您看来，表格会比较具有不同特征的对象（是）。 实际上，这里有一些IaaS服务，开源项目和面向供应商的认证。



当然，涵盖DevOps领域的一系列技能和知识还包括使用更多不同程序和工具的能力。 许多项目也有自己的认证，例如Docker认证助理，认证的Jenkins工程师，AppDynamics认证，Red Hat的Ansible认证专家等。


自动化专家认证


在自动化专业中，有一个基于编程网络基础的模块（占主题总数的10％），其中的主题包括：


  • 将Linux / macOS / Windows工作站设置为开发环境
  • Python编程语言基础
  • 吉特
  • 使用REST API
  • 解析json
  • CI / CD

思科DevNet专业人士


谁专注于：
具有3年以上应用程序开发和实施经验的专家； 具有Cisco解决方案和Python编程语言的经验。
这将很有趣：对于那些转向自动化和DevOps的开发人员； 使用思科生态系统的解决方案架构师； 经验丰富的网络工程师，希望扩展其技能，包括应用程序开发和自动化； 基础架构开发人员设计安全的生产环境。


认证包括两项考试：


  1. 基础考试，旨在确认开发人员的专业技能（DEVCOR 300-901）
  2. 在以下领域之一进行专业考试：DevOps，IoT，Webex，协作自动化，数据中心自动化，企业自动化，安全自动化，服务提供商自动化。 上面在Cisco DevNet Specialist认证说明中对它们进行了详细描述。

基础考试包括以下主题：


  • 软件开发与设计
  • 了解和使用API
  • 思科平台
  • 应用程序部署和安全性
  • 基础架构与自动化

“软件开发和设计”模块包括“网络编程基础知识”模块中的主题，并且还补充了以下主题：应用程序开发基础（架构模式，考虑到应用程序需求的数据库类型选择，诊断应用程序问题，评估应用程序架构并考虑各种因素）参数）; 与Webex Teams集成（包括Webex Teams SDK，OAuth等的知识）； Firepower管理中心中的令牌认证； 深入了解git（git服务器，分支，解决冲突等）。


基础结构和自动化模块还将具有有关使用Ansible剧本Puppet清单配置网络参数的任务和问题。
