Diodo como rectificador



El circuito rectificador convierte la corriente alterna en corriente continua. Esta conversión es crítica para los electrodomésticos. La corriente alterna proviene de los enchufes, y se necesita corriente continua para las computadoras y otros microelectrónicos. La corriente alterna alterna entre las direcciones negativa y positiva. Pero la corriente continua fluye en una dirección. Para convertir una corriente de una corriente alterna en una constante, debe evitar que fluya en la dirección negativa. Para esto hay un diodo. Se puede ensamblar un rectificador de media onda desde un diodo. He ensamblado un circuito usando yn como la fuente del transformador de corriente de la vieja radio conectado a 220V.
